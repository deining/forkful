## Що це таке і для чого це потрібно?
Командний рядок - це інструмент, який дозволяє вам взаємодіяти з операційною системою та запускати програми з використанням текстових команд. Читання аргументів командного рядка є важливим для програмістів, оскільки це дозволяє їм передавати зовнішні дані в свої програми та зробити їх більш гнучкими та корисними для користувачів.

## Як це зробити:
Програмування на Rust дозволяє легко читати аргументи командного рядка за допомогою методу ```args()``` з бібліотеки ```std::env```. Далі ви можете вибрати конкретний аргумент за допомогою методу ```nth()```, який приймає його позицію в списку аргументів. Нижче наведено приклад коду та його виведення:

```Rust
use std::env;

fn main() {
    // Отримуємо аргументи командного рядка
    let args: Vec<String> = env::args().collect();

    // Виводимо перший аргумент
    println!("Перший аргумент: {}", args.nth(1).unwrap());
}
```

Введіть наступну команду для запуску програми та передачі аргумента:

```bash
cargo run argument1
```

Виведення буде таке:

```bash
Перший аргумент: argument1
```

## Глибше дослідження:
Читання командних аргументів відіграє важливу роль в створенні корисних та зручних програм. Без цього, програми не змогли би отримувати зовнішні дані та адаптуватися до різних сценаріїв використання.

Історично, читання аргументів командного рядка було вживане у багатьох мовах програмування, включаючи C та Java. Однак, існують і інші способи читання вхідних даних, такі як стандартний ввід та файли конфігурації.

Щоб дізнатися більше про читання аргументів командного рядка в Rust, рекомендуємо переглянути [документацію](https://doc.rust-lang.org/std/env/fn.args.html) стандартної бібліотеки.
