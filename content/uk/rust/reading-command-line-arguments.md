## Чому

У програмуванні широко використовуються аргументи командного рядка для зміни поведінки програми під час її виконання. Це дозволяє користувачам передавати параметри в програму і налаштовувати її роботу. Тому читання аргументів командного рядка є важливою навичкою для кожного програміста, який працює з мовою програмування Rust.

## Як це зробити

Існує кілька способів читання аргументів командного рядка в Rust, за допомогою різних бібліотек та вбудованих функцій. Один із способів - використання структури `std::env::args`, яка представляє собою ітератор з аргументами командного рядка. Для отримання першого аргументу використовується `next()`, а для перевірки наявності наступного аргументу - `is_some()`. Нижче наведений приклад коду та вихідний результат:

```rust
use std::env;

fn main() {
    for arg in env::args().skip(1) {
        println!("{}", arg);
    }
}
```

```none
$ ./program arg1 arg2
arg1
arg2
```

## Глибоке погруження

Якщо вам потрібні більш складні зчитування аргументів командного рядка, ви можете роздивитися більш спеціалізовані бібліотеки, такі як `clap` або `getopts`. Вони надають додаткові можливості, такі як обробка прапорців та підтримка більш складних форматів аргументів. Також важливо знати, що аргументи командного рядка можна передавати не тільки при запуску програми з терміналу, але і використовувати їх у скриптах та інших програмах.
