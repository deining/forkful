## Чому

Преобразування рядка до нижнього регістру є важливою функцією програмування, яка дозволяє зручно порівнювати та фільтрувати рядки. В даному блозі матимемо можливість детальніше дізнатися про методи конвертації рядків до нижнього регістру у мові програмування Rust.

## Як?

```Rust
fn main() {
    let string = "ПРограмУВАNNЯ рУСТ";
    println!("РЯДОК ДО НИЖНЬОГО РЕГІСТРУ: {}", string.to_lowercase());
}
```
*Output: "рядок до нижнього регістру: програмування руст"*

```Rust
fn main() {
   let string = "Київ";
   println!("РЯДОК ДО НИЖНЬОГО РЕГІСТРУ: {}", string.to_lowercase());
}
```
*Output: "рядок до нижнього регістру: київ"*

## Глибшій аналіз

Метод `to_lowercase()` у мові програмування Rust використовується для перетворення всіх символів рядка до нижнього регістру. Це корисна функція при порівнянні рядків, оскільки вона дозволяє ігнорувати регістровані символи. Також, цей метод може бути використаний для форматування даних, наприклад, у випадку, коли потрібно вивести рядок у нижньому регістрі на екран.
