## Для чого
Зниження регістру рядка є частою потребою в програмуванні. Це може бути корисно для порівняння рядків, операцій над даними або просто для полегшення читання. У цій статті ми розглянемо як перетворити рядок на нижній регістр у мові програмування Rust.

## Як це зробити
Існує декілька способів конвертування рядка у нижній регістр в мові Rust. Розглянемо два з них.

```Rust
fn main() {
    let string = "Привіт, Світ!";
    let lower_string = string.to_lowercase();
    println!("Рядок у нижньому регістрі: {}", lower_string);
}
```

Виходячи з цього коду, ми маємо вивід: "рядок у нижньому регістрі: привіт, світ!". Також, у мові Rust існує функція ```to_ascii_lowercase()```, яка перетворює рядок у нижній регістр, ігноруючи будь-які не-ascii символи.

```Rust
fn main() {
    let string = "Привіт, Світ!";
    let lower_ascii_string = string.to_ascii_lowercase();
    println!("Рядок у нижньому регістрі: {}", lower_ascii_string);
}
```

Вивід буде таким же, як і у попередньому прикладі. Однак, цей метод не дозволяє робити операції над не-ascii символами.

## Глибока підготовка
Існує багато функцій та методів у мові Rust для роботи з рядками. Для зниження регістру доступні методи ```to_lowercase()``` та ```to_ascii_lowercase()```. Також, можна використовувати мета-методи, які дозволяють здійснювати інші операції з рядками, такі як розділення на символи, знаходження підрядків і багато іншого.
