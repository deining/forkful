## Чому

Звичайно, у вас можуть бути випадки, коли вам потрібно перетворити рядок на нижній регістр при програмуванні на Rust. Це може бути корисно при роботі зі строковими даними, наприклад, для забезпечення їхньої рівності або для забезпечення правильного форматування.

## Як це зробити

Конвертація рядка на нижній регістр у Rust не потребує багато зусиль. Вам просто потрібно використовувати статичний метод `.to_lowercase()` для типу `String`:

```rust
let text = "HELLO WORLD";
let lower_text = text.to_lowercase();
println!("Рядок у нижньому регістрі: {}", lower_text);
```

Виконуючи цей код, ви отримаєте наступне вихідне повідомлення:

```
Рядок у нижньому регістрі: hello world
```

Також слід зауважити, що цей метод не лише перетворює великі літери на маленькі, але і коригує всі символи, які мають різні варіанти написання (наприклад, латинські та кириличні літери).

## Глибоке поглиблення

Перетворення рядка на нижній регістр у Rust не є простим обробкою даних в один рядок коду. Він використовує Unicode-правила перетворення для забезпечення культурної свідомості та підтримки різних мов. Більше інформації про це можна знайти в офіційній документації [Unicode и перетворення рядків у Rust](https://doc.rust-lang.org/std/string/struct.String.html#method.to_lowercase).
