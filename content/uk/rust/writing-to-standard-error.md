# Чому

Якщо ви програмуєте на Rust, ви, напевно, знаєте про можливість виводити повідомлення на стандартну помилку. Це корисний інструмент, який дозволяє вам контролювати процес виконання вашої програми та виявляти наявні проблеми.

# Як користуватися стандартною помилкою

Щоб вивести повідомлення на стандартну помилку, використовуйте функцію `eprintln!`. Наприклад:

```Rust
fn main() {
    eprintln!("Це повідомлення виведено на стандартну помилку!"); 
}
```

В результаті ви отримаєте наступний вивід:

```
Це повідомлення виведено на стандартну помилку!
```

Також, ви можете використовувати `eprintln!` для виведення значень змінних або об'єктів. Наприклад:

```Rust
fn main() {
    let num = 10;
    eprintln!("Значення змінної num: {}", num); 
}
```

Результат:

```
Значення змінної num: 10
```

# Детальніше про стандартну помилку

Виведення повідомлень на стандартну помилку дозволяє вам виявляти помилки та проблеми, що виникли при виконанні вашої програми. Також, це дозволяє отримати додаткову інформацію про процес виконання та допомагає вам виявити й усунути помилки.

Важливим аспектом використання стандартної помилки є правильна обробка помилок. Не забувайте перевіряти результати викликів функцій та використовувати механізм `match` для обробки різних варіантів.
