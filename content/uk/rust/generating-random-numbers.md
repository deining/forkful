# Що це таке і навіщо нам це потрібно?

Генерація випадкових чисел - це процес отримання чисел, що не мають певної послідовності або шаблону. Програмісти використовують цей процес для створення випадкових даних для тестування програм або алгоритмів, а також для шифрування даних.

# Як це зробити:

```Rust
use rand::{thread_rng, Rng}; 

fn main() {
    // Генерація випадкового числа в діапазоні від 1 до 10
    let random_number = thread_rng().gen_range(1, 11); 

    // Генерація випадкової строки з 10 символів
    let mut rng = thread_rng();
    let random_string: String = rng.sample_iter(Alphanumeric).take(10).collect(); 

    // Виведення результату
    println!("Випадкове число: {}", random_number);
    println!("Випадкова строка: {}", random_string);
}
```

Виведення:

```
Випадкове число: 7
Випадкова строка: w4O2HsA5C9
```

# Глибоке занурення:

Генерація випадкових чисел використовувалася в програмуванні ще до появи мови Rust. Багато мов мають свої вбудовані функції для генерації випадкових чисел, але у Rust для цього використовується стороння бібліотека `rand`, яка надає більші можливості та контроль над процесом генерації.

Існують інші альтернативи для генерації випадкових чисел у Rust, такі як `openSSL` та `libc`. Однак, `rand` є основним вибором для більшості програмістів за його простотою та ефективністю.

Також варто зазначити, що випадкові числа, отримані за допомогою `rand`, не є справжньо випадковими, а просто є псевдовипадковими значеннями, що генеруються за допомогою математичного алгоритму.
