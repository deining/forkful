##Чому
Генерація випадкових чисел є важливим аспектом багатьох програм, особливо ігор та алгоритмів. Наприклад, це може бути корисно при створенні випадкового паролю або розподілу випадкових даних для тестування алгоритмів. У цій статті ми розглянемо, як генерувати випадкові числа у мові програмування Rust.

##Як
Для початку, ми повинні імпортувати бібліотеку `rand` у наше програмне забезпечення. Це забезпечить нам доступ до функцій, якими ми будемо користуватися для генерації випадкових чисел.
```Rust
use rand::Rng;
```
Далі нам потрібно створити змінну типу `Rng`, яка буде відповідати за генерацію випадкових чисел.
```Rust
let mut rng = rand::thread_rng();
```
Тепер ми готові генерувати випадкові числа за допомогою методу `gen_range`, який приймає мінімальне та максимальне значення, і повертає випадкове число з цього проміжку. Наприклад, якщо нам потрібно згенерувати випадкове число від 1 до 10, то ми можемо написати наступне:
```Rust
let random_number = rng.gen_range(1..=10);
println!("{}", random_number); // Виводить, наприклад, 7
```
Ми також можемо використовувати метод `choose` для вибору випадкового елементу з набору. Наприклад, якщо ми маємо масив чисел `[1, 2, 3, 4, 5]`, то ми можемо використати метод `choose` для вибору одного з цих чисел:
```Rust
let numbers = [1, 2, 3, 4, 5];
let random_number = rng.choose(&numbers).unwrap();
println!("{}", random_number); // Виводить, наприклад, 3
```
Для більш складних генерацій, ми також можемо використовувати методи `shuffle` та `sample` для вибору випадкового набору елементів з масиву.

##Глибокий занурення
Можливості бібліотеки `rand` у Rust є дуже розширеними. Вона дозволяє генерувати не тільки цілі числа та числа з плаваючою комою, але і більш складні типи даних, такі як рядки та байтові послідовності. Крім цього, за допомогою параметрів можна налаштувати рівномірність генерації чисел та можливість повторів. Для більш детального ознайомлення з бібліотекою `rand` рекомендується переглянути документацію на офіційному сайті.
