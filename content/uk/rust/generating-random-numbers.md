## Чому

Генерування випадкових чисел є важливою частиною багатьох програм та додатків. Завдяки випадковості, ці числа можуть бути використані для шифрування даних, створення випадкових паролів або розробки ігор. У Расті є простий та потужний спосіб генерувати випадкові числа, який ми дослідимо далі.

## Як

```rust
use rand::prelude::*;

fn main() {
    // генеруємо випадкове ціле число від 1 до 100
    let number = thread_rng().gen_range(1..=100);
    println!("Ваше випадкове число: {}", number);

    // генеруємо масив з 10 випадкових дійсних чисел
    let mut array = [0.0; 10];
    thread_rng().fill(&mut array);
    println!("Ваш масив випадкових чисел: {:?}", array);
}
```
Вихід:
```
Ваше випадкове число: 73
Ваш масив випадкових чисел: [0.9356793295736845, 0.8230984234973826, 0.5019267416920428, 0.9921200634579611, 0.13672340633044623, 0.03630243876899211, 0.346303302810954, 0.21789347890392733, 0.9091309573904278, 0.6506249917622973]
```

## Занурення

У Расті існує два основних типи генераторів випадкових чисел: `thread_rng` та `thread_rng`. Перший використовує глобальний генератор для створення випадкових чисел, а другий - власний генератор, який може бути керований та налаштований користувачем. Також, Раст має багато інших корисних функцій для генерування випадкових чисел, таких як `gen_bool` для отримання випадкового булевого значення або `gen_ascii_chars` для створення випадкових символів. Для більш детального дослідження функцій генерації випадкових чисел у Расті, рекомендую переглянути [документацію](https://docs.rs/rand/0.8.3/rand/).
