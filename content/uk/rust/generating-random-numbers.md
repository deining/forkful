## Чому

Згенерувати випадкові числа може бути корисною функціональністю для багатьох програм, які вимагають випадкових величин у своїй роботі. Наприклад, це може бути необхідно для створення унікальних ідентифікаторів, випадкових паролів або випробування алгоритмів.

## Як

```Rust
use rand::Rng;

fn main() {
    // Створюємо новий генератор випадкових чисел
    let mut rng = rand::thread_rng();

    // Генеруємо випадкове ціле число в діапазоні від 1 до 10
    let random_number = rng.gen_range(1, 11);

    println!("Випадкове число: {}", random_number);
}
```

В результаті виконання цього коду, ми отримаємо випадкове число від 1 до 10, яке буде виведене на екран. Ось ще один приклад, де ми використовуємо генератор випадкових чисел для створення випадкового положення на шаховій дошці:

```Rust
use rand::Rng;

fn main() {
    // Створюємо новий генератор випадкових чисел
    let mut rng = rand::thread_rng();

    // Генеруємо випадкове число в діапазоні від 0 до 7 для координати х
    let x = rng.gen_range(0, 8);
    
    // Генеруємо випадкове число в діапазоні від 0 до 7 для координати y
    let y = rng.gen_range(0, 8);

    println!("Випадкове положення: ({}, {})", x, y);
}
```

В результаті виконання цього коду, ми отримаємо випадкову позицію на шаховій дошці, наприклад (3, 6).

## Глибоке пірнання

За замовчуванням, Rust використовує випадковий генератор на основі безпечного імпортера для забезпечення високої якості випадкових чисел. Це означає, що згенеровані числа не будуть повторюватися і не можуть бути передбаченими.

Крім того, Rust надає можливість налаштувати і налаштувати випадковий генератор, вибравши інший імпортер або задаючи свої власні параметри. Це дозволяє вам створювати більш випадкові числа або пристосувати генератор для конкретних вимог вашої програми.
