## Що та навіщо?

Інтерполяція рядків - це метод заміни певних значень в рядку змінних, який дозволяє швидко та зручно форматувати виведені дані. Це особливо корисно для програмістів, оскільки дозволяє змінювати рядки коду без необхідності вручну перераховувати та замінювати значення змінних.

## Як це зробити?

Існує кілька способів інтерполяції рядків в мові програмування Rust. Найпростішим і найбільш дієвим є використання макросу `format!`, в якому можна використовувати спеціальні позначення, щоб показати, де значення змінних повинні бути вставлені. Давайте подивимося на приклад:

```Rust
let name = "Jane";
let age = 25;
let message = format!("Привіт! Мене звати {}, мені {} років.", name, age);
println!("{}", message);
```

Після виконання коду, ми отримаємо виведення:

```Rust
Привіт! Мене звати Jane, мені 25 років.
```

## Глибше в деталі

Інтерполяція рядків була вперше впроваджена у мові програмування Python, але з того часу стала популярною у багатьох інших мовах, включаючи Rust. Є також альтернативний метод інтерполяції рядків, використовуючи макрос `print!` або `println!`, які також дозволяють вставляти значення змінних у рядки.

Імплементація інтерполяції рядків в Rust відрізняється від інших мов за допомогою важливої функції - безпеки типів. Це означає, що типи даних обов'язково мають збігатися, інакше код не буде компілюватися.
