## Чому

Розробка програм знаходиться в постійному русі і ми постійно стикаємося з викликами, пов'язаними з обробкою дат та часу. Чи не було б чудово мати можливість легко обчислювати дату в майбутньому або минулому? Відповідь проста: це допоможе вам ефективно керувати вашим розкладом та планувати події.

## Як це зробити

Найпростіший спосіб обчислити дату в майбутньому або минулому це використовуючи метод `add` з модуля `chrono`. Ось приклад як це працює:

```Rust
use chrono::{Datelike, Duration, NaiveDate};

// задайте початкову дату
let initial_date = NaiveDate::from_ymd(2021, 7, 15);

// додайте до неї 30 днів
let result_date = initial_date.add(Duration::days(30));

// виведіть результат
println!("Дата після 30-ти днів: {}", result_date);
```

А вот результат у терміналі:

```
Дата після 30-ти днів: 2021-08-14
```

Ви також можете використовувати метод `subtract`, щоб обчислити дату в минулому. Наприклад:

```Rust
use chrono::{Datelike, Duration, NaiveDate};

// задайте початкову дату
let initial_date = NaiveDate::from_ymd(2021, 7, 15);

// відніміть 7 днів
let result_date = initial_date.subtract(Duration::days(7));

// виведіть результат
println!("Дата 7 днів назад: {}", result_date);
```

Результат у терміналі буде таким:

```
Дата 7 днів назад: 2021-07-08
```

## Розглиблений аналіз

Якщо вам потрібно обчислювати більш складні дати (наприклад, дозволяти переходи через місяці та роки), ви можете використовувати механізм `Duration` з модуля `chrono`. За допомогою цього механізму, ви можете визначити кількість років, місяців та днів, які ви хочете додати або відняти від початкової дати. Також, ви можете використовувати інші одиниці виміру часу, такі як години, хвилини та секунди.
