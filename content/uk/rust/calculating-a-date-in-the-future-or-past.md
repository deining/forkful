# Що & Чому?
Обчислення дати у майбутньому або минулому - це процес визначення конкретної дати на основі вихідної дати та заданого періоду часу. Програмісти виконують це для різних цілей, включаючи отримання відстані між двома датами, створення календарів та реалізації функцій пересування часу у програмі.

# Як?
Нижче наведений приклад коду на мові Rust, який демонструє обчислення дати 5 років в майбутньому від поточної дати та виводить результат у форматі ```YYYY/MM/DD```:

```Rust
use chrono::{Utc, Duration};

fn main() {
    let now = Utc::now();
    let future_date = now + Duration::days(1825);
    println!("{}", future_date.format("%Y/%m/%d"));
}

```
Виведе: 2025/05/30

# Deep Dive
Історичний контекст: обчислення дати є важливою складовою частиною багатьох сучасних програм та систем. Саме тому, було створено багато різних алгоритмів та інструментів для виконання таких обчислень.

Альтернативи: окрім мови Rust, існують інші мови програмування, які підтримують обчислення дат, такі як Java, Python та JavaScript. Також існує можливість використання готових бібліотек, які забезпечують широкий функціонал для роботи з датами.

Деталі реалізації: мова Rust має вбудовану бібліотеку `chrono`, яка надає інструменти для роботи з датами та часом. Вона покращена для роботи з міжнародними стандартами та має широкий функціонал для роботи з датами, включаючи обчислення, форматування та додавання/віднімання періодів часу.
