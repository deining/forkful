## Чому

Створення тимчасових файлів - це важлива частина багатьох програм, особливо для зберігання тимчасових даних, які не потрібні в довгостроковій перспективі. Це також може бути корисним для тестування або створення даних для подальшого оброблення. У цій статті ми розглянемо, як створювати тимчасовий файл за допомогою мови програмування Rust.

## Як

```Rust
use std::fs::File;
use std::io::Write;
use std::env;

fn main() {
    // Використовуємо змінну оточення для визначення шляху до тимчасової директорії
    let temp_dir = env::temp_dir();
    // Створюємо новий тимчасовий файл за допомогою змінної оточення та рандомної назви
    let mut temp_file = File::create(temp_dir.join("temp_file.txt")).expect("Не вдалося створити файл");
    // Записуємо дані у тимчасовий файл
    writeln!(temp_file, "Це тимчасовий файл у Rust!") .expect("Не вдалося записати в файл");
}
```

В результаті, у тимчасовій директорії буде створений файл з назвою "temp_file.txt", який містить рядок "Це тимчасовий файл у Rust!".

## Deep Dive

Крім створення тимчасових файлів, мова програмування Rust також надає можливість видаляти їх після виконання необхідних дій. Для цього потрібно використовувати функцію `std::fs::remove_file()` з назвою файлу, який потрібно видалити.

Також, можна використовувати бібліотеку `tempfile` для створення тимчасових файлів у безпечній та ефективній манері. Вона надає зручний інтерфейс та дозволяє автоматично видаляти тимчасові файли після завершення їх використання. Детальніше про цю бібліотеку можна дізнатися [тут](https://crates.io/crates/tempfile).
