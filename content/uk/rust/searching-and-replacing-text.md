## Чому

В даній статті ми розглянемо практичні переваги застосування Rust для пошуку та заміни тексту. Завдяки його функціональності та високої продуктивності, ця мова програмування є ідеальним вибором для розв'язання таких задач.

## Як

Для початку, нам потрібно імпортувати потрібні бібліотеки за допомогою ключового слова `use`. У цьому прикладі ми будемо використовувати бібліотеку `regex` для роботи з регулярними виразами.

```Rust
use regex::Regex;
```

Далі, ми можемо визначити текст, у якому хочемо здійснювати заміну.

```Rust
let text = "Привіт, я програміст української мови. Радий тебе познайомити!";
```

Тепер ми можемо визначити регулярний вираз, який буде шукати букву "о" і замінювати її на букву "е".

```Rust
let re = Regex::new("о").unwrap();
```

І нарешті, ми можемо використати метод `replace_all` для здійснення заміни та вивести результат на екран.

```Rust
let result = re.replace_all(text, "е");
println!("{}", result);
```

Вихідний текст буде виглядати так:

```
Привіт, я програміст української мови. Радій тебе пезнайомити!
```

## Глибока погляд

Використання регулярних виразів у поєднанні з мовою Rust дозволяє здійснювати більш потужні та складні операції з текстом. Наприклад, за допомогою регулярних виразів можна замінити одночасно кілька символів або використовувати умови для заміни. Це дає змогу більш ефективно та економно працювати з текстом та зменшити кількість потрібного коду.

Прикладом більш складної заміни може бути видалення всіх чисел з рядка тексту.

```Rust
use regex::Regex;

fn main() {
    let text = "Сьогодні 24 грудня 2021 року. Завтра 25 грудня 2021 року.";

    let re = Regex::new(r"[0-9]+").unwrap();

    let result = re.replace_all(text, "");
    println!("{}", result);
}
```

В даному випадку, ми застосували регулярний вираз `[0-9]+`, який шукає будь-яке число в тексті. Результатом буде рядок без чисел:

```
Сьогодні грудня року. Завтра грудня року.
```
