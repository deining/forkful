## Чому

Якщо ви когда-небудь втратили час ручного редагування великого кількості тексту, ви зрозумієте, чому пошукові та замінювальні оператори є незамінними інструментами для будь-якого програміста. Вони дозволяють швидко та ефективно замінити частини тексту, що допомагає економити час та зусилля.

## Як це зробити

```Rust
// створення рядка для заміни
let заміна = "Докладніше про Руст";

// створення рядка, в якому необхідно замінити текст
let текст = "Це стаття про пошук і заміну тексту в Руст.";

// заміна тексту за допомогою методу `replace`
let новий_текст = текст.replace("пошук і заміну", заміна);

// виведення результату
println!("{}", новий_текст);
```

Виходом буде:

``` 
Це стаття про Докладніше про Руст в Руст.
```

## Глибоке погруження

Пошук та заміна тексту є невід'ємною частиною багатьох програм, особливо тих, що працюють з великими обсягами даних. У Русті є декілька різних методів, які дозволяють здійснити ці операції, у тому числі методи `replace`, `replace_range` та `replacen`. Кожен з них має свої переваги та використання залежить від конкретної ситуації.
