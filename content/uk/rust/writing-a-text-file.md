# Чому

Написання текстового файлу є важливим аспектом розробки програм, особливо в Rust. Це дозволяє зберігати дані, налаштовувати параметри програми та збереження проміжних результатів обчислень.

# Як це зробити

Написання текстового файлу можна реалізувати в Rust за допомогою стандартної бібліотеки `std::fs`. Для цього потрібно використовувати функцію `write` і передавати їй вхідні дані у форматі `String`. Далі користувач має застосувати метод `unwrap`, щоб перевірити, чи відбулося записування в файл без помилок.

Приведемо приклад коду:

```Rust
use std::fs::File;
use std::io::prelude::*;

fn main() {
    let mut file = File::create("output.txt").unwrap();
    file.write_all(b"Hello, World!").unwrap();
}
```

У цьому прикладі ми створюємо файл під назвою `output.txt` і записуємо у нього рядок "Hello, World!".

# Поглиблене дослідження

Написання текстового файлу в Rust дозволяє проводити більш складні операції, такі як зміна вмісту чи додавання нових рядків до вже існуючого файлу. Також ця операція може бути комбінована з іншими функціями збереження даних, які надаються в стандартній бібліотеці Rust.
