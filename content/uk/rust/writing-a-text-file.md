# Чому

Написання текстового файлу є важливою частиною програмування на Rust. Вона дозволяє зберігати та обробляти дані, що використовуються в програмі.

# Як це зробити

Нижче показано приклад, як за допомогою Rust створити та записати дані до текстового файлу. Використовуйте кодуски ```Rust ... ``` для перегляду прикладів програми та виводу.

```Rust
use std::fs::File;
use std::io::Write;

fn main() {
    let mut file = File::create("example.txt").expect("Не вдалося створити файл!");

    file.write_all(b"Привіт, українці!").expect("Не вдалося записати до файлу!");
}
```

Отриманий файл буде містити рядок "Привіт, українці!". Метод ```File::create()``` створює новий файл, вказуючи його ім'я, а ```file.write_all()``` записує дані до файлу.

# Вдавайся глибше

Тепер, коли ви вмієте створити і записати дані до файлу, можна розглянути більш складні методи роботи з текстовими файлами на Rust. Наприклад, ви можете використовувати структуру ```File``` для читання та запису байтів або використовувати бібліотеку ```csv``` для обробки даних, що містяться у текстовому файлі. Додаткова інформація про ці теми доступна в документації Rust або у ресурсах, перелічених нижче.
