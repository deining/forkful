## Чому

У програмуванні часто необхідно поєднувати різні рядки даних, щоб отримати потрібний результат. У Rust існує спеціальний метод для цього – конкатенація рядків. Давайте дізнаємося, як це працює.

## Як це зробити

Конкатенація рядків в Rust виконується за допомогою знаку "+" або функції "format!". Давайте розглянемо приклад використання обох методів:

```rust
// Використання знаку "+"
let name = "Софія";
let greeting = "Привіт, " + name;
println!("{}", greeting); // Виведе: Привіт, Софія

// Використання функції "format!"
let surname = "Петренко";
let fullname = format!("{} {}", name, surname);
println!("{}", fullname); // Виведе: Софія Петренко
```

Ви можете замість функції "format!" використовувати метод "to_string()", який перетворює будь-який тип даних в рядок.

```rust
let age = 25;
let info = name.to_string() + " is " + &age.to_string() + " years old.";
println!("{}", info); // Виведе: Софія is 25 years old.
```

## Більш глибоке дослідження

Конкатенація рядків в Rust працює за допомогою використання функцій "Push" та "Pop", які додають та видаляють символи з рядку. Використання знаку "+" або функції "format!" просто полегшує цей процес. Також важливо пам'ятати про те, що рядки в Rust є нещільними даними, тому об'єднання багатьох рядків може призвести до витрати багато пам'яті.
