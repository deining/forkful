##Чому

Нинішня версія мови програмування Rust (версія 1.50.0) надає розширені можливості для керування рядками. Це дозволяє розробникам створювати більш ефективні та швидкодіючі програми. Конкатенація рядків - це процес об'єднання двох чи більше рядків в один, та є корисною у багатьох випадках, коли потрібно з'єднати повідомлення, шляхи файлів чи інші дані.

##Як

Будь-які рядки в Rust можна з'єднати за допомогою оператора "+" або методу "format!" зі стандартної бібліотеки. Наприклад:

```Rust
let str1 = "Привіт, ";
let str2 = "світ!";
let result = str1 + str2;
println!("{}", result); 
```

Виходом цього коду буде: "Привіт, світ!". Можна також використовувати метод "format!" для конкатенації більше ніж двох рядків:

```Rust 
let str1 = "Це";
let str2 = " дуже ";
let str3 = "корисно!";
let result = format!("{}{}{}", str1, str2, str3);
println!("{}", result); 
```

Результатом буде "Це дуже корисно!". Крім того, у Rust є можливість включати значення в середині рядка за допомогою методу "format!":

```Rust 
let num = 10;
let result = format!("Необхідна кількість: {}", num);
println!("{}", result); 
```

В результаті отримаємо "Необхідна кількість: 10". Також можна використовувати оператор "+", щоб додавати значення до кінця рядка без використання методу "format!":

```Rust 
let num = 10;
let result = "Кількість елементів: " + &num.to_string();
println!("{}", result); 
```

Це створить рядок "Кількість елементів: 10". Варто зазначити, що у Rust рядки є не змінними, тому кожен раз, коли ми з'єднуємо рядки, ми створюємо зовсім новий рядок, який займає більше пам'яті.

##Глибокий розбір

У Rust кожен рядок представлений як скалярний тип даних "str" (строчка) або "String" (рядок). Різниця між ними полягає в тому, що "str" є незмінним (immutable), тоді як "String" можна модифікувати (mutable). Оператор "+" та метод "format!" працюють з "String", тому при використанні їх з "str" потрібно перетворити його до типу "String":

```Rust 
let str1 = "Foo";
let str2 = "bar";
let result = String::from(str1) + str2;
println!("{}", result); 
```
