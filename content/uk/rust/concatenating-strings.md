## Чому: Причина використання з'єднання рядків в програмуванні

З'єднання рядків - це процес об'єднання двох або більше рядків в один. Це дуже корисний і простий метод в програмуванні, особливо в мові Rust. В деяких випадках це дозволяє зменшити кількість коду і полегшити читання програми.

## Як використовувати з'єднання рядків в Rust

```Rust
let greeting = "Привіт";
let name = "Олег";
let message = greeting.to_owned() + " " + name; // З'єднуємо рядки
println!(message); // Виводимо "Привіт Олег"
```

Ви можете використовувати оператор `+` для з'єднання рядків в Rust. Але у цьому прикладі ми також використовуємо метод `to_owned()`, який конвертує рядок у власний. Це не є обов'язковим, але дозволяє уникнути проблем з пам'яттю у деяких ситуаціях.

```Rust
let numbers = vec!["1", "2", "3"];
let result = numbers.join(", "); // З'єднуємо рядки з комою та пробілом
println!(result); // Виводимо "1, 2, 3"
```

Метод `join()` може бути корисним для з'єднання більшої кількості рядків, наприклад, всіх елементів у векторі. Він дозволяє також додати розділювач між рядками.

## Глибші деталі про з'єднання рядків

Коли ви використовуєте оператор `+`, ви створюєте новий рядок, що містить обидва рядки, які з'єднуєте. Це не надто ефективно, оскільки старі рядки все ще зберігаються у пам'яті. Тому для більш ефективного використання пам'яті, ви можете використовувати макрос `format!`, який не створює новий рядок, а просто форматує вихідні рядки у бажаний вигляд.

```Rust
let age = 25;
let person = format!("I am {} years old.", age); // Не створює нового рядка
println!(person); // Виводить "I am 25 years old."
```

Це особливо корисно при роботі з більшими рядками, такими як HTML-код або SQL-запити, де з'єднання рядків може бути недостатнім.
