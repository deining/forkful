# Чому

Існує багато ситуацій, коли потрібно витягнути певний підрядок зі строкового значення. Наприклад, може виникнути потреба видалити певну частину з назви файлу або отримати лише ім'я користувача з email адреси. У таких випадках, використання функцій для витягнення підрядків може значно спростити завдання.

# Як

Використання функції `slice()` є основним способом витягнення підрядків в Rust. Давайте розглянемо кілька прикладів використання цієї функції.

```Rust
let full_string = "Hello world!";
let substring = &full_string[2..7];
println!("{}", substring);

// Output: llo w
```

В даному прикладі, ми використовуємо функцію `slice()` для витягнення підрядка з другого символу до сьомого включно. Обов'язково використовуйте `[start..end]` для вказання початкового та кінцевого індексів. Також можливе використання `..` замість `[..end]` або `[start..]` для показу всіх символів до кінця або від початку відповідно.

```Rust
let full_string = "abcd1234";
let substring = &full_string[..4];
println!("{}", substring);

// Output: abcd
```

Більше прикладів використання функції `slice()` можна знайти в розділі "Deep Dive".

# Глибока розробка

У Rust, функція `slice()` використовується для витягнення підрядків шляхом вказання початкового та кінцевого індексів. Зверніть увагу, що початковий індекс включається, а кінцевий - ні. Також у Rust є поняття "рядків-зрізів" (`str slice`), які є іншому способом використання функції `slice()`. Рядки-зрізи є непостійними значеннями та можуть містити різну кількість символів в залежності від рядка, з якого витягнуто підрядок.
