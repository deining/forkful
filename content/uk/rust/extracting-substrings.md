## Для чого

Ця стаття призначена для тих, хто бажає вчитися видобувати підрядки в мові програмування Rust. Це може бути корисно для різних завдань, таких як обробка текстових даних чи маніпулювання рядками.

## Як це зробити

З мовою програмування Rust видобування підрядків є досить простим. Використовуючи стандартну бібліотеку, ми можемо скористатися методом `get()` для отримання осколків рядків з заданого індексу по заданий розмір.

```Rust
let s = "Привіт світ!";
let slice = &s[0..5]; // "Привіт"
```

Ми можемо також скористатися методом `find()`, щоб знайти позицію підрядка у рядку та використати її для видобування підрядків.

```Rust
let s = "Привіт світ!";
let index = s.find("світ").unwrap(); // Знаходимо позицію підрядка
let slice = &s[index..]; // "світ!"
```

Якщо ми бажаємо видобути підрядок відповідно до певного критерію, ми можемо скористатися комбінацією цих методів разом.

```Rust
let s = "a, b, c, d, e";
let first_comma = s.find(",").unwrap(); // Знаходимо позицію першого коми
let last_comma = s.rfind(",").unwrap(); // Знаходимо позицію останнього коми
let substr = &s[first_comma..last_comma]; // "b, c, d"
```

## Глибше

В процесі розгляду прикладів вище, ми використовували тип даних `&str`, який є неявним у використанні при роботі з рядками в Rust. `&str` є скороченням від *string slice*, що означає шматок рядка. Це дозволяє основному рядковому типу `String` мати більшу гнучкість та ефективність при роботі з рядками.

Також важливо враховувати, що `&str` є *незмінним* типом, тобто ми не можемо змінити окремі елементи `&str`. Це зроблено для більшої безпеки при програмуванні.
