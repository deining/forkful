## Чому

Видобування підрядків є важливою частиною багатьох програм та проектів у програмуванні. Це дозволяє нам отримувати окремі частини рядків для подальшого використання, зберігаючи при цьому поліморфізм та ефективність.

## Як це зробити

Використовуючи мову програмування Rust, ми можемо легко та швидко видобувати підрядки з рядків. Ось декілька прикладів:

```Rust
// Видобути підрядок за індексом
let str = "Привіт світ!";
let substr = &str[7..11]; // виведе "світ"

// Видобути підрядок за певним словом
let str = "Я люблю програмування";
let substr = str.split(" ").collect::<Vec<_>>();
// substr буде містити ["Я", "люблю", "програмування"]

// Видобути підрядок за певним шаблоном
let str = "Шаблоноване вираження - {0}";
let substr = format!(str, "Руст"); // substr буде містити "Шаблоноване вираження - Руст"
```

## Глибока зміна

Видобування підрядків в Rust базується на типізованих рядках та використовує функцію наперед вивчених стрічок для оптимальної швидкості та ефективності. Крім того, використання методів як `split()` та `format!()` дозволяє нам створювати більш складні шаблони для видобування підрядків.
