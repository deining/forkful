## Що і чому?

Витягання підрядків - це процес вилучення певних символів або слів з рядка, щоб отримати підрядок або частину цього рядка. Програмісти часто використовують цю техніку для здобуття доступу до певних даних або для подальшої обробки тексту.

## Як це зробити:

```Rust
// Розбити рядок на декілька підрядків за допомогою роздільника
let s = "Це рядок з декількома підрядками";
let parts: Vec<&str> = s.split(" ").collect();
println!("{:?}", parts);

// Вилучити підрядок за його позицією у рядку
let s = "Це рядок для демонстрації";
let substring = &s[0..6];
println!("{}", substring);

// Вилучити підрядок за його ключовим словом
let s = "Це рядок для демонстрації";
let keyword = "для";
let substring = s[s.find(keyword).unwrap()..];
println!("{}", substring);

// Вилучити підрядок за його довжиною
let s = "Це рядок для демонстрації";
let substring = &s[..6];
println!("{}", substring);
```

Вихідні дані для всіх прикладів будуть:

```Rust
Вектор з підрядками: ["Це", "рядок", "з", "декількома", "підрядками"]
Демонстрація роботи з позиціями у рядках
Демонстрація пошуку за ключовим словом та витягання підрядка
```

## Поглиблене дослідження:

Історичний контекст: Техніка витягання підрядків була широко використовувана в ранніх мовах програмування, таких як Сі і Паскаль. Однак, з появою багатьох нових функцій у сучасних мовах, таких як Раст, це стало більш ефективне і зручне для програмістів.

Альтернативи: Замість використання вже зазначених методів, можна використовувати бібліотеку regex, яка дозволяє виконувати більш складні дії з підрядками. Однак, використання вбудованих методів є більш швидким та простим способом.

Деталі реалізації: В Раст, індексування рядків починається з 0, а не з 1. Для вилучення підрядків за ключовим словом, потрібно використовувати метод find, але користувач повинен використовувати unwrap для перевірки наявності шуканого ключового слова у рядку.
