"## Зачем"

Поділ стрічки на підрядки є важливою операцією у багатьох програмах. Це дозволяє отримати доступ до більших даних або здійснити пошук підрядка в тексті. У даній статті ми розглянемо, як використовувати мову програмування Rust для ефективного витягування підрядків зі стрічки.

## Як використовувати

Для початку нам потрібно імпортувати бібліотеку `str` в наш проект:

```Rust
use std::str;
```

Для екстракції підрядка зі стрічки ми можемо використовувати метод `slice`:

```Rust
let my_string = "Це стрічка для прикладу.";
let my_substring = &my_string[3..8];
println!("Вичленений підрядок: {}", my_substring);
```

У цьому прикладі ми вичленяємо підрядок починаючи з третього індексу і до восьмого (не включно). Результатом буде стрічка "стріч".

## Глибокі деталі

Метод `slice` використовує два індекси для визначення початку і кінця підрядка. Другий індекс не включається у вичленений підрядок, тому останній символ підрядка завжди буде на одну позицію меншим за заданий кінець.

Також, в методі `slice` можна використовувати від'ємні індекси, які будуть відлічувати позицію символу з кінця стрічки. Наприклад, `my_string[-3..]` вичленить підрядок з трьох останніх символів стрічки.
