## Чому: 
"## Why - невеликий розділ, який пояснює, чому хтось міг би зацікавитися перетворенням рядка у великі букви.

Є багато сценаріїв, в яких потрібно перетворити текст на великі літери для полегшення його читання або для використання у програмі. Наприклад, ви можете знадобитися це під час створення користувацького інтерфейсу або парсингу даних з інтернету.

## Як:
У даному розділі ми продемонструємо, як перетворити рядок на великі букви в Rust. Використовуючи вбудовану функцію `to_uppercase()` та символьний тип даних `char`, ми зможемо зробити це за кілька рядків коду.

```Rust
let text = "привіт, світ!";
let result = text.to_uppercase();

println!("Результат: {}", result);

// Вивід: Результат: ПРИВІТ, СВІТ!
```

Ви також можете використовувати цю функцію для перетворення окремих символів або працювати з рядками, що містять кириличні символи.

```Rust
let symbol = 'й';
let result = symbol.to_uppercase();

println!("Результат: {}", result);

// Вивід: Результат: Й
```

## Глибоке занурення:
Щоб краще зрозуміти, як працює перетворення рядка на великі букви, давайте зазирнемо трохи під капот цієї функції. Коли ви викликаєте `to_uppercase()`, Rust використовує стандартну бібліотеку для перетворення рядка на нижній регістр, а потім міняє кожну букву на її велику версію відповідно до Unicode таблиці символів.

Також варто зауважити, що іноді при роботі з рядками можуть виникати проблеми з кодуванням. Для запобігання цьому, можна використовувати бібліотеку `encoding_rs` або встановити кодування вручну за допомогою `to_uppercase_with_fold()`.
