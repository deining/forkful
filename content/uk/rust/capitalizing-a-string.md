## Що це таке і чому: 
Капіталізування строки означає перетворення першої букви кожного слова у велику літеру. Це може бути корисно для покращення зовнішнього вигляду тексту або для виконання певних дій у програмі. 

## Як це зробити: 
```Rust 
let my_string = "привіт світ"; 

println!("Капіталізована строка: {}", my_string.to_uppercase()); 

// Вивід: "КАПІТАЛІЗОВАНА СТРОКА: Привіт світ" 
```

## Глибше занурення: 
1. Історичний контекст: Капіталізація тексту має коріння в друкуванні, коли використовувалися лише прописні літери. З часом, з'явилася потреба в нижній регістрі. 
2. Альтернативи: В Rust також є функція ```to_lowercase``` для перетворення всіх літер у нижній регістр. 
3. Деталі реалізації: У Rust кожен символ має своє представлення у вигляді числа, тому для капіталізації строки потрібно перетворити числа, що відповідають нижнім літерам у числа, що відповідають великим літерам.
