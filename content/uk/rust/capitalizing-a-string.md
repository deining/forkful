## Чому

Використання рядків з великою першою літерою є важливою частиною написання чистого та професійного коду. Використовуючи мову програмування Rust, можна легко зробити це за допомогою функції `to_uppercase()`, яку ми розглянемо у цій статті.

## Як

Для початку, створимо новий проект Rust та додамо до нього функцію для роботи з рядками. У цій функції ми будемо перетворювати рядок, що надійшов до нас у вигляді аргументу, на рядок з високими першими літерами за допомогою функції `to_uppercase()`. Ось приклад коду:

```rust
fn uppercase_string(string: &str) -> String {
    string.to_uppercase()
}
```

Після того, як ми визначили нашу функцію, можна викликати її з будь-яким рядком і побачити результат. Ось приклад виклику та виведення в консоль:

```rust
let string = "hello world";
println!(uppercase_string(string));
```
Вивід: `HELLO WORLD`

Також, варто зазначити, що ця функція не працює лише зі словами англійської мови, а з будь-якими символами та мовами, що підтримуються у Rust.

## Глибше

Функція `to_uppercase()` використовується для переведення рядка у верхній регістр. Вона може бути корисною при роботі з користувацьким вводом даних, наприклад, при перевірці правильності написання імен або рядків для використання в назвах файлів.
