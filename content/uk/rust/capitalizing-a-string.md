## Чому

Розміщення заголовків ваших програм є важливою частиною побудови чистого і структурованого коду. Коли ви правильно називаєте свої зазначення, це спрощує читання і редагування вашого коду. Одним з простих способів зробити ваш код більш зрозумілим є використання стандартної практики великих літер для назви змінних та функцій. У цій статті ми детально розглянемо як користуватися функцією `to_uppercase()` для перетворення малих літер у великі.

## Як зробити

```Rust
fn main() {
    let name = "rust";
    let upper_name = name.to_uppercase();

    println!("The capitalized name is {}", upper_name);
}
```

Вивід:

`The capitalized name is RUST`

У цьому коді ми спершу створюємо змінну `name`, яка містить слово `rust`. Потім ми використовуємо функцію `to_uppercase()` для перетворення значення цієї змінної у великі літери. Нарешті, ми використовуємо `println!()` для виведення коректного результату.

## Заглиблення

Для розуміння як ця функція працює, важливо знати, що тип змінної `name` у нашому прикладі є рядком. Це означає, що ми можемо використовувати всі методи рядків, такі як `to_uppercase()`, які надає мова програмування Rust.

Крім цього, функція `to_uppercase()` здатна обробляти рядки у різних кодуваннях, що робить її корисною для міжнародних програм.
