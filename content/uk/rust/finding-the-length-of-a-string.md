## Чому
Навіщо комусь шукати довжину строки? Найімовірніше, вам потрібно обробляти текст або знаєте, що довжина строки впливає на розмір використовуваних ресурсів.

## Як
В Расті є кілька способів знайти довжину строки. Найпростіший спосіб - використовуючи функцію `len()`. Давайте подивимось на приклад коду:

```Rust
let my_string = "Привіт світ!";
let length = len(my_string); // поверне значення 13
```

Ось і все! Функція `len()` приймає строки або будь-який інший тип, який реалізує інтерфейс `std::convert::AsRef<str>`, і повертає довжину у байтах.

Також можна використовувати метод `len()` безпосередньо на стрічці:

```Rust
let my_string = "Привіт світ!";
let length = my_string.len(); // поверне значення 13
```

Цей метод автоматично перетворює типи, тому функцію `len()` можна також використовувати на змінних типів `String` або `&String`.

```Rust
let my_string = String::from("Привіт світ!");
let length = my_string.len(); // поверне значення 13
```

## Глибокий занурення
Що ж, тоді як саме працює метод `len()` і функція `len()`? У Расті строки представлені як скалярні типи, які підтримують кілька операцій, таких як доступ до символів за індексом та ітерування. Метод `len()` та функція `len()` використовують ці операції для підрахунку кількості символів у стрічці.

Однак варто зазначити, що в Расті кодування строк відрізняється від більшості інших мов програмування. Раст використовує кодування UTF-8, що означає, що кожен символ може бути представлений декількома байтами. Це може вплинути на результат методу `len()`, залежно від кодування та типу символу. Також варто пам'ятати, що метод `len()` та функція `len()` повертають кількість байтів, а не кількість символів, тому це слід враховувати при роботі з різними кодуваннями.
