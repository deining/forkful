## Що & Чому?
Знаходження довжини рядка - це процес підрахунку кількості символів у заданому рядку тексту. Програмісти використовують цей підхід, щоб знайти розмір рядка або виконати інші операції з використанням цієї інформації.

## Як:
```
Rust
let string = "Привіт, світ!";
let length = string.len();
println!("Рядок {} має довжину {}", string, length);
```
Output: Рядок Привіт, світ! має довжину 13

## Глибинне вивчення:
Історично, підрахунок довжини рядка був однією з основних операцій у багатьох мовах програмування. У Rust також існують альтернативні методи знаходження довжини рядка, такі як використання ітераторів або змінна зберігання кількості символів. Реалізація функції ```len()``` у Rust ефективна та швидка, оскільки вона використовує значення байтової довжини рядка замість підрахунку символів.
