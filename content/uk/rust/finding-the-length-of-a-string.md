## Чому

Знаходження довжини рядка є дуже важливою операцією в багатьох програмах. Наприклад, це може бути корисно при роботі з введеними користувачем даними або при форматуванні виводу. Розуміння того, як знаходити довжину рядка, допоможе вам створювати більш ефективні та функціональні програми.

## Як

```Rust
let string = "Programming is fun!";
let length = string.len();
println!("The length of the string is {}", length);
```

В цьому прикладі ми створили змінну `string`, яка містить рядок "Programming is fun!". Далі ми використовуємо метод `.len()`, щоб отримати довжину цього рядка. На останньому рядку ми виводимо довжину рядка за допомогою функції `println!()`. В результаті ви побачите: `The length of the string is 19`.

## Глибоке занурення

В Rust довжина рядка обчислюється за допомогою методу `.len()`, який повертає значення типу `usize`. Ви можете також використовувати функцію `.chars()` для отримання кількості символів у рядку. Прикметною особливістю Rust є те, що довжина рядка є статичним елементом, що означає, що вона не може бути змінена під час виконання програми.
