## Чому

Конвертування дати у рядок є важливим елементом у програмуванні, адже це дозволяє зберігати та відображати дату у зручному форматі для користувача. Також це дозволяє здійснювати різні операції з датами, такі як порівняння, сортування та фільтрування, що є важливими для багатьох програм.

## Як

Для конвертування дати у рядок у мові Rust використовуються функції з бібліотеки `chrono`. Для початку, необхідно імпортувати цю бібліотеку у свій код. Після цього можна використовувати ці функції для отримання рядкового представлення дати. Наприклад:

```Rust
use chrono::{DateTime, Local, NaiveDate, Utc, offset::TimeZone};
fn main() {
  // отримуємо поточну дату та час у форматі UTC
  let utc_now: DateTime<Utc> = Utc::now();
  // конвертуємо дату у локальний часовий пояс
  let local_time = utc_now.with_timezone(&Local);
  // отримуємо рядкове представлення дати та часу у форматі "28.02.2021 15:30:00"
  let date_string = local_time.format("%d.%m.%Y %H:%M:%S").to_string();
  println!("{}", date_string);
}
```

Вищенаведений приклад показує, як за допомогою функцій з бібліотеки `chrono`, можна отримати поточну дату та час у форматі, який можна використати для подальших операцій з рядками.

## Deep Dive

У мові Rust, є різні способи конвертування дати у рядок. Наприклад, для представлення різних стандартів дат та часу, можна використовувати різні форматуючі рядки. Також, можна змінювати часовий пояс, у якому буде відображатися дата. Це дозволяє створювати більш гнучкі та універсальні рішення для конвертування дати у рядок.
