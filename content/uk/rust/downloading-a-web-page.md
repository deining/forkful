### Що & Чому?

Завантаження веб-сторінки - це процес отримання веб-сторінки з Інтернету на свій комп'ютер або інший пристрій. Програмісти часто завантажують веб-сторінки, щоб отримати інформацію, необхідну для їх програм або проектів. 

### Як це зробити:

```Rust
use reqwest::blocking::Client;

fn main() {
let client = Client::new(); // створення клієнта
let response =  client.get("https://www.example.com") // використовуємо метод get для отримання сторінки
                .header("User-Agent", "Mozilla/5.0") // встановлюємо значення заголовка, щоб уникнути блокування
                .send().unwrap(); // виконуємо запит і зберігаємо вивід у змінній response 
                
println!("Статусний код: {}", response.status()); // виводимо статусний код відповіді
println!("Вміст сторінки: {}", response.text().unwrap()); // виводимо вміст сторінки у текстовому форматі
}
```

Перед тим, як запускати цей приклад коду, необхідно додати бібліотеку `reqwest` у файл `Cargo.toml` і встановити її за допомогою команди `cargo install`.

### Глибше вдивимося:

Завантаження веб-сторінок використовується в багатьох програмах, наприклад, для отримання даних для аналізу або для здійснення запитів до API. Альтернативами до використання бібліотеки `reqwest` можуть бути `curl` або `wget`. Реалізація завантаження веб-сторінок у бібліотеці `reqwest` базується на протоколі HTTP і використовує TCP сокети для встановлення з'єднання з сервером.
