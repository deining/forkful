## Чому

Якщо ви вивчаєте мову програмування Rust, ви швидко зрозумієте, що вона має потужні функції для роботи з текстовими файлами. Чи не було б чудово, якщо б ви могли дізнатися, як читати і обробляти такі файли за допомогою Rust? Цей блог пост допоможе вам зрозуміти саме це.

## Як

Перш за все, ми повинні встановити необхідний пакет для роботи з файлами - ```std::fs```. Після цього ми можемо викликати функцію ```read_to_string```, яка дозволяє читати текстовий файл і повертати його вміст у вигляді рядка. Нижче наведений код, який показує цей процес:

```Rust
use std::fs;
let contents = fs::read_to_string("file.txt").expect("Не вдалося прочитати файл");
println!("Зміст файлу: {}", contents);
```

Ви можете також використовувати інші функції, такі як ```read``` для читання файлу в байтах або ```read_dir``` для отримання списку файлів у директорії. Результати цих функцій можна використати для подальшої обробки і аналізу даних з файлу.

## Глибокий занурення

Хоча зараз ми показали приклад читання простого текстового файлу, Rust також має можливості для роботи з більш складними форматами даних. Наприклад, ви можете використовувати бібліотеку ```serde``` для зчитування і запису даних в JSON або CSV форматі. Ви також можете використовувати бібліотеку ```regex``` для пошуку і обробки тексту за допомогою регулярних виразів.
