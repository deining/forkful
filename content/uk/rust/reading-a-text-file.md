## Чому

Вивчення мови програмування Rust може бути важким завданням для будь-якого розробника. Більшість початківців зазвичай розпочинають з вивчення основних концепцій, таких як зчитування файлів. Читання текстових файлів є однією з найпоширеніших дій у багатьох програмах, тому це важлива навичка, яку варто вивчити.

## Як

```Rust
use std::fs::File;
use std::io::{BufReader, BufRead};

fn main() {
  let file = File::open("file.txt").expect("Could not open file.");
  let reader = BufReader::new(file);

  for line in reader.lines() {
    println!("{}", line.unwrap());
  }
}
```

У цьому простому прикладі ми використовуємо файловий ввід / вивід доступний в стандартній бібліотеці Rust, щоб прочитати зміст файлу і вивести його на екран. Функція `BufReader::new()` дозволяє прочитати файл у буфер для більш ефективного оброблення. Завдяки ітератору `lines()` ми можемо зчитувати файл по рядках та виводити його вміст.

## Глибокий дайджест

При роботі з файлами, варто мати на увазі певні аспекти, такі як безпека та обробка помилок. Використання `expect()` дозволяє обробляти випадки, коли файл не може бути зчитаний. Крім того, більш ефективною альтернативою є використання методу `unwrap()` для повернення потенційної помилки. Нехтувати безпекою та обробкою помилок може призвести до проблем у ваших програмах.
