## Чому
Існує багато ситуацій, коли потрібно зчитати вміст текстового файлу в програмі, наприклад, для обробки даних або виведення інформації користувачу. У даній статті ми розглянемо, як просто і ефективно зчитати текстовий файл за допомогою мови програмування Rust.

## Як
Основним інструментом для зчитування текстових файлів в Rust є модуль `std::fs`, який надає функціонал для роботи з файловою системою. Використовуючи цей модуль, ми можемо відкрити файл за допомогою функції `File::open()`, вказавши шлях до файлу. Після цього можна використовувати методи `read_to_string()` або `read_line()` для зчитування вмісту файлу в рядок або по репозиціях відповідно. Окрім цього, важливо прибрати стиль модернізації `\r\n` або `\n` з кінця кожного рядка за допомогою методу `trim()`.

````Rust
use std::fs::File;
use std::io::{BufRead, BufReader};

let file = File::open("example.txt").expect("Не вдалося відкрити файл");
let file_reader = BufReader::new(&file);

for line in file_reader.lines() {
    let string_line = line.expect("Не вдалося зчитати рядок");
    let clean_line = string_line.trim();
    println!("{}", clean_line);
}
````

Результатом буде виведений вміст файлу у консоль.

## Глибока занурення
Якщо потрібно прочитати великі файли або ті, що не вміщуються у пам'ять, варто розглянути використання іншого модуля `std::io`, а саме `BufReader` з методом `lines()` для читання по одному рядку, або `skip()` для пропуску певної кількості рядків, які є зайвими. Також можна використати метод `seek()` для переміщення курсора у файлі на позицію, яку потрібно прочитати.
