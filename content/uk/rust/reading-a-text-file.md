Що & Чому?
Читання текстового файлу - це процес отримання інформації з файлу, який містить текстовий вміст. Програмісти читають текстові файли для отримання необхідних даних для своїх програм.

Як це зробити:
За допомогою мови програмування Rust, можна легко читати текстові файли. Для цього потрібно використати модуль "std::fs" та функцію "File:ral::new(&path)". Нижче наведено приклад коду та виведений результат.

```rust
use std::fs::File;

fn main() {
  // Відкриття та зчитування файлу
  let file = File::open("example.txt").expect("Unable to open file!");
  let contents = String::from_utf8_lossy(&file);
  
  // Виведення зчитаного тексту
  println!("{}", contents);
}
```

Глибоке занурення:
Читання текстових файлів історично пов'язане з розробкою комп'ютерних систем, де збережені дані потрібно було аналізувати та використовувати. У сучасному програмуванні існує кілька альтернативних способів читання текстових файлів, таких як використання сторонніх бібліотек чи вбудованих функцій мов програмування.

Інша корисна функція, яка допомагає читати текстові файли у Rust - це функція "std::io::BufReader" яка дозволяє ефективно зчитувати великі файли шляхом буферизації.
