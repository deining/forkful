# Що і Чому?
Виведення відлагоджувального виводу - це процес виведення додаткової інформації під час розробки програм. Це дозволяє програмістам перевіряти правильність виконання коду та виявляти помилки для подальшого виправлення.

# Як: 
```Rust
fn main() {
    let num1 = 5;
    let num2 = 7;
    println!("Додавання двох чисел: {}", num1 + num2);
    println!("Змінна num1: {}", num1);
    println!("Змінна num2: {}", num2);
}
```
**Вихід:**
```
Додавання двох чисел: 12
Змінна num1: 5
Змінна num2: 7
```

# Глибинне дослідження:
Історичний контекст:
Вимога до виведення відлагоджувального виводу зародилася в рамках здійснення діагностики і відлагоджування програм. Із появою різних мов програмування техніки та засоби для такого роду контролю програм постійно вдосконалювалися. У мові програмування Rust для виведення відлагоджувального виводу використовується макрос `println!` для друку значення змінної та `format!` для форматування тексту.

Альтернативи:
Нативна підтримка `\#debug` макросу у мові Rust дозволяє побачити значення змінних та вивести їх у консоль. Також існують бібліотеки, які надають додаткові можливості для виведення відлагоджувального виводу, наприклад, `log` та `env_logger`.

Деталі реалізації:
Макрос `println!` використовує форматний рядок для переведення значень змінних у текст та їх виведення у консоль. Ця функція відлагоджування має багато використань та може бути дуже корисною для відстеження значень змінних під час виконання програми.
