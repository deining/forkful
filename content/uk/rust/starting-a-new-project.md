## Чому

Стартувати новий проект у програмуванні може бути викликом для багатьох людей, особливо для тих, хто ще не має достатнього досвіду. Однак, створення нового проекту може бути захоплюючим та надихаючим процесом, який допоможе вам розширити свої знання та навички у програмуванні.

## Як

Щоб почати новий проект у мові Rust, спочатку вам необхідно встановити її розвиток на своєму комп'ютері. Після цього, створіть нову папку та відкрийте її у своєму обраному текстовому редакторі. Почніть з імпортування основного пакету мови Rust та створення функції, яка буде виводити текст на екран:

```Rust
use std::io;

fn main() {
    println!("Привіт, Руст!");
}
```

Збережіть файл з розширенням .rs та запустіть його за допомогою компілятора Rust у своєму терміналі. Ви повинні побачити виведення "Привіт, Руст!" на екрані.

## Глибше вдивимось

Створення нового проекту у мові Rust може бути складнішим, ніж це виглядає на перший погляд. Важливо пам'ятати, що ви не повинні бути експертом у цій мові, щоб почати проект. Розробка нового проекту - це процес навчання та пошуку відповідей на свої запитання. Не бійтеся експериментувати та робити помилки, це частина навчання!
