## Чому

Українські програмісти, особливо ті, хто вивчає Rust, можуть мати питання щодо того, чому вони повинні видаляти символи, які відповідають певному шаблону. Відповідь проста: це дуже корисна інструмент для засобів обробки тексту, особливо у випадках, коли необхідно видалити непотрібні символи зі строки.

## Як

У цьому прикладі ми будемо використовувати засіб обробки тексту в Rust, щоб видалити всі крапки з коротких речень українською мовою. Почнемо з імпорту бібліотеки `regex` та створення змінної `text`, якій будемо присвоювати речення, що потребує обробки.

```rust
use regex::Regex;
let text = "Це речення повинно бути оброблене...";
```

Далі, ми створимо регулярний вираз, щоб вибрати всі крапки, що потрібно видалити. У нашому випадку, це буде вираз `"\."`, який означає будь-яку крапку в тексті. Ми також визначимо пусту строку `""` як заміну для крапок.

```rust
let re = Regex::new(r"\.");
let result = re.replace_all(text, "");
```

Нарешті, застосуємо результат до нашого початкового тексту, використовуючи метод `replace_all`. Збережіть результат в змінну `result` та надрукуйте:

```rust
println!("{}", result); // "Це речення повинно бути оброблене"
```

## Глибоке погруження

Хоча цей приклад досить простий, у більш складних сценаріях може бути корисно знати різноманітні методи видалення символів, такі як `replace`, `replace_range` та `drain`. Також можливо використати регулярні вирази, щоб вибрати конкретні символи або набори символів для видалення. Для більш детального огляду цих методів та додаткових прикладів, перегляньте [офіційну документацію засобу обробки тексту в Rust](https://docs.rs/regex/latest/regex/).
