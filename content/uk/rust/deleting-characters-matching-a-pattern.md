## Що і чому?

Видалення символів, які відповідають заданому шаблону, - це процес, коли програмісти видаляють зі строки символи, які відповідають певному критерію. Це може бути корисним, наприклад, для фільтрування небажаних символів або для заміни певних даних. В програмуванні це часто використовується для оптимізації та полегшення роботи з даними.

## Як:

```Rust
let s = "Hello Rust!";
let pattern = 'R'; // вибираємо символ R
let result = s.replace(pattern, ""); // видаляємо з рядка всі входження символу R
println!(result); // виводимо на екран "Hello ust!"
```
## Глибинний розгляд:

Видалення символів за шаблоном використовується в багатьох мовах програмування, не просто в Rust. Альтернативним способом є використання регулярних виразів, але це може бути більш складним для новачків. У Rust, для видалення символів, використовується функція `replace()`, яка приймає 2 аргументи - шаблон та символ або рядок для заміни.
