## Чому

Іноді в коді нам потрібно видаляти певні символи, які відповідають певному шаблону. Це може бути необхідно для фільтрування даних або знаходження потрібної інформації. В цій статті ми розглянемо, як це зробити в мові програмування Rust.

## Як це зробити

Для видалення символів відповідних заданому шаблону, ми можемо використовувати метод `replace_all` зі стандартної бібліотеки `Regex`. Спочатку необхідно імпортувати цю бібліотеку:

```Rust
use regex::Regex;
```

Потім ми можемо використовувати метод `replace_all`, передаючи йому шаблон та текст, з якого ми хочемо видалити символи:

```Rust
let pattern = Regex::new(r"[A-Za-z]+").unwrap();
let text = "Hello 123 world!";
let replaced_text = pattern.replace_all(text, "");
println!("{}", replaced_text); // => " 123 !"
```

У цьому прикладі, всі символи, які відповідають шаблону `[A-Za-z]+` (літери a-z в будь-якому регістрі), будуть замінені на порожній рядок.

Можна також використовувати більш складний шаблон, щоб видаляти символи, які відповідають декільком шаблонам одночасно. Наприклад, якщо ми хочемо видалити всі літери та цифри з тексту, ми можемо використовувати шаблон `[A-Za-z0-9]+`.

## Глибоке погруження

Дуже корисною функцією у бібліотеці `Regex` є можливість застосовувати функції до знайдених шаблонів. Наприклад, ми можемо використовувати функцію `map` для заміни знайдених символів на інші значення:

```Rust
let pattern = Regex::new(r"[A-Za-z]+").unwrap();
let text = "Hello 123 world!";
let replaced_text = pattern.replace_all(text, |caps: &regex::Captures| {
    caps[0].to_uppercase()
});
println!("{}", replaced_text); // => "HELLO WORLD!"
```

У цьому прикладі, ми використовуємо функцію `to_uppercase`, щоб перетворити знайдений шаблон у верхній регістр. Таким чином, весь текст буде перетворений у верхній регістр, а цифри та спеціальні символи залишаться без змін.
