## Чому

Порівняння двох дат є важливим аспектом програмування в будь-якій мові, включаючи Rust. Воно дозволяє визначити, яка з дат більша або менша, що дозволяє з легкістю сортувати дані та створювати логіку для деяких операцій. У цій статті ми розглянемо, як порівняти дві дати в Rust та як це може бути корисним в реальних сценаріях програмування.

## Як це зробити

Для початку, нам потрібно використати бібліотеку `chrono`, яка надає зручні функції для роботи з датами та часом в Rust. Давайте розглянемо приклад, як можна проілюструвати порівняння двох дат:

```Rust
use chrono::{DateTime, Utc};

// створимо дві дати
let date1 = DateTime::parse_from_rfc3339("2021-08-01T00:00:00+00:00").unwrap();
let date2 = DateTime::parse_from_rfc3339("2021-08-15T00:00:00+00:00").unwrap();

// порівняємо їх
if date1 < date2 {
    println!("Дата 1 є ранішою за дату 2");
} else if date1 > date2 {
    println!("Дата 1 є пізнішою за дату 2");
} else {
    println!("Дата 1 та дата 2 є однаковими");
}
```

У цьому прикладі ми використали функцію `parse_from_rfc3339` для створення об'єктів типу `DateTime` з рядків у форматі RFC 3339. Після цього ми змогли здійснити порівняння за допомогою операторів `<` та `>`.

Також важливою функцією для порівняння дат є `compare`, яка повертає значення `Ordering` - енум, який представляє результат порівняння. Давайте також розглянемо такий приклад:

```Rust
use chrono::{DateTime, Utc};
use std::cmp::Ordering;

// створимо дві дати
let date1 = DateTime::parse_from_rfc3339("2021-08-01T00:00:00+00:00").unwrap();
let date2 = DateTime::parse_from_rfc3339("2021-08-15T00:00:00+00:00").unwrap();

// порівняємо їх та обробимо результат
match date1.compare(&date2) {
    Ordering::Less => println!("Дата 1 є ранішою за дату 2"),
    Ordering::Greater => println!("Дата 1 є пізнішою за дату 2"),
    Ordering::Equal => println!("Дата 1 та дата 2 є однаковими"),
}
```

Зверніть увагу, що ми використали метод `compare` об'єкта `date1`, передавши інший об'єкт в якості аргумента.
