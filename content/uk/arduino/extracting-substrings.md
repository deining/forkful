## Чому

У програмуванні Arduino часто потрібно обробляти рядки тексту. Один зі способів це зробити - це витягнути певну частину рядка, відому як підрядок. Це може стати необхідним для видалення непотрібних символів, отримання певної інформації або для інших маніпуляцій з даними.

## Як це зробити

Існує кілька способів взяти підрядок з рядка у програмі Arduino. Для початку ми створимо рядок і застосуємо до нього функцію `substring()`, яка дозволяє витягнути підрядок з заданої позиції та довжини.

```Arduino
String str = "Привіт, світ!";
String substr = str.substring(7, 12);
```

У цьому прикладі, ми створюємо рядок `str` зі значенням "Привіт, світ!" і використовуємо `substring()` для отримання підрядка з позиції 7 (починаючи з нуля) та довжини 12 символів. Результатом буде рядок "світ!".

Також можна використовувати скорочену форму `substring()` з лише одним параметром, щоб витягнути підрядок з заданої позиції до кінця рядка:

```Arduino
String str = "Привіт, світ!";
String substr = str.substring(7);
```

Існує також можливість використовувати метод `charAt()` для отримання конкретного символу з рядка і створювати підрядки з ним:

```Arduino
String str = "Привіт, світ!";
char c = str.charAt(7); // c = 'с'
String substr = str.substring(7); // substr = "світ!"
```

## Глибокий занурення

У разі потреби більш тонкої настройки витягування підрядка, можна використовувати функцію `indexOf()` для пошуку певного символу або підрядка в рядку. Метод `indexOf()` повертає позицію першого входження символу чи підрядка, а потім можна використовувати це значення для створення більш точного підрядка.

```Arduino
String str = "Привіт, світ!";
int pos = str.indexOf("світ"); // pos = 6
String substr = str.substring(pos); // substr = "світ!"
```

Також можна використовувати ці методи для витягування підрядка з рядка користувача, використовуючи функцію `Serial.readString()`.

```Arduino
String user_input = Serial.readString();
int pos = user_input.indexOf("hello"); // пошук підрядка "hello"
String substr = user_input.substring(pos); // витягнення підрядка по позиції
```
