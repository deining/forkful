## Чому

Вимірювання взаємодії між різними об'єктами часто включає розбивання тексту на частини або підрядки для подальшого аналізу. В Arduino це може бути корисно при роботі зі сенсорами, які повертають довгі рядки даних.

## Як це зробити

Для початку потрібно створити змінну типу `String` та присвоїти їй потрібний рядок. Далі, використовуючи метод `substring()`, можна вибрати ділянку рядка за допомогою індексів початку і кінця.

```Arduino
String text = "Прикладний текст";

String substring = text.substring(5, 8); // поверне "ики"
```

У цьому прикладі ми вибираємо рядок від індексу 5 до індексу 8 (не включно) зі змінної `text` і присвоюємо його змінній `substring`.

## Глибокий занурення

Крім вказання індексів початку і кінця, метод `substring()` також має другий параметр - крок. Він визначає кількість символів, які будуть пропущені між вибраними частинами рядка.

```Arduino
String text = "123456789";

String substring = text.substring(0, 9, 3); // поверне "13"
```

У цьому прикладі ми змінили значення кроку на 3, щоб вибрати кожен третій символ починаючи з першого і до кінця рядка. Це корисно, коли потрібно вибрати частину даних з інтервалом або повторюваністю.
