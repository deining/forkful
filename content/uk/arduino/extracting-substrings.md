## Чому

Підстроки - це частини рядка, які можна виокремити. Це корисний інструмент, коли нам потрібно отримати певну частину рядка, використовуючи тільки його початок і кінець. Наприклад, ми можемо використовувати підстроки, щоб отримати дату з дати та часу або видалити непотрібну частину імені або адреси.

## Як

Існує кілька способів витягнути підстроку з рядка в Ардуіно. Одним з них є використання функції `substring()`.

```Arduino
String example = "Це приклад рядка";
String substring = example.substring(4, 10);
Serial.println(substring);
```

У цьому прикладі ми створили рядок `example` і витягнули з нього підстроку, починаючи з 4-го символу і закінчуючи на 10-му. Результат буде `приклад`.

## Глибокий занурення

Функція `substring()` повертає копію підстроки із заданої початкової і кінцевої позицій. Аргументи цієї функції - це індекси символів, а не кількість символів. Тому, якщо потрібно отримати підстроку довжиною 5 символів, необхідно визначити кінцеву позицію, додавши 5 до початкової.

```Arduino
String example = "Інша рядок";
String substring = example.substring(0, 5);
Serial.println(substring);
```

У цьому прикладі результат буде `Інша`, оскільки ми витягнули підстроку із перших 5 символів рядка.

Існує також можливість використовувати від'ємні значення для початкової і кінцевої позицій. В цьому випадку, значення буде зчитуватися з кінця рядка.

```Arduino
String example = "Це рядок для прикладу";
String substring = example.substring(-14, -8);
Serial.println(substring);
```

У цьому прикладі результат буде `для`, оскільки ми витягнули підстроку із сьомого символу з кінця до третього символу з кінця.
