## Що та чому?
Читання аргументів командного рядка - це процес отримання даних, які надсилаються через командний рядок, і використання їх у програмі. Програмісти роблять це, щоб забезпечити більш гнучкий та інтерактивний спосіб взаємодії з програмою.

## Як:
```Arduino
#include <stdio.h>

int main(int argc, char *argv[])
{
  for (int i=0; i<argc; i++){
    Serial.println("Аргумент " + String(i+1) + ": " + argv[i]);
  }
  return 0;
}
```

Ввійдіть ```1 2 3``` в командному рядку та отримайте наступний вивід:

```
Аргумент 1: 1
Аргумент 2: 2
Аргумент 3: 3
```

## Глибоке занурення:
Історичний контекст: читання аргументів командного рядка було запроваджено у спільності UNIX в 1970-х роках та використовується досі.

Альтернативи: є бібліотеки, які допомагають спростити процес читання аргументів командного рядка, такі як ```getopt```, але більшість мікроконтролерів, включаючи Arduino, мають обмежену кількість пам'яті, тому читання аргументів напряму є ефективнішим.
