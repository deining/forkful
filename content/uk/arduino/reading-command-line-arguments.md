## З чого почати
Відтепер ваші проекти Arduino можуть стати ще більш інтерактивними та управляються за допомогою командної строки! Розберемось, що це таке та як це може збільшити вашу креативність та ефективність.

## Як це зробити
Для того, щоб зчитувати аргументи командної строки в Arduino, потрібно використовувати функцію ```Serial.readString()```. Перед тим, як почати читати аргументи, потрібно встановити швидкість передачі даних в Serial Monitor на ту ж саму, що і в коді (наприклад, 9600 бод).

Отже, приклад коду для зчитування аргументів командної строки виглядатиме так:
```
String inputString; // змінна для зберігання введених даних
void setup() {
  Serial.begin(9600); // встановлення швидкості передачі даних
}
void loop() {
  if (Serial.available()) { // перевірка на наявність даних для зчитування
    inputString = Serial.readString(); // зчитування даних у змінну
    Serial.println(inputString); // виведення даних у Serial Monitor
  }
}
```
Після завантаження цього коду на плату Arduino і відкриття Serial Monitor, ви зможете вводити аргументи командної строки та бачити їх виведення у Serial Monitor.

## Детальніше про зчитування аргументів
Крім ```Serial.readString()```, є ще кілька інших методів для зчитування даних з командної строки, таких як ```Serial.parseInt()``` для зчитування цілих чисел і ```Serial.parseFloat()``` для зчитування дробових чисел. Також можна використовувати цикли для зчитування декількох аргументів одночасно.

Наприклад, такий код може зчитувати два аргументи, розділені пробілом, та виводити їх у Serial Monitor:
```
String inputString;
String arg1;
String arg2;
void setup() {
  Serial.begin(9600);
}
void loop() {
  if (Serial.available()) {
    inputString = Serial.readString();
    int spaceIndex = inputString.indexOf(' '); // знаходимо індекс першого пробілу
    arg1 = inputString.substring(0, spaceIndex); // отримуємо перший аргумент
    arg2 = inputString.substring(spaceIndex + 1); // отримуємо другий аргумент
    Serial.println("Перший аргумент: " + arg1); // виводимо результат у Serial Monitor
    Serial.println("Другий аргумент: " + arg2);
  }
}
```
