## Чому

У програмуванні з використанням мови Arduino однією з найважливіших і корисних навичок є вміння читати аргументи командного рядка. Це дозволяє вашій програмі взаємодіяти з користувачем та отримувати необхідну вхідну інформацію.

## Як це зробити

Для того, щоб програма могла читати аргументи командного рядка, спочатку потрібно підключити бібліотеку "Serial". Потім можна використовувати функцію "Serial.readString()" для отримання рядка введеного з вказаного порту. Наприклад:

```Arduino
#include <Serial.h>

void setup(){
  Serial.begin(9600);  // встановлюємо швидкість портів
  while (!Serial){    // чекаємо підключення
    ;  
  }
}

void loop(){
  if (Serial.available()){  // якщо є введений рядок у порті
    String input = Serial.readString(); // зчитуємо рядок
    Serial.println(input); // виводимо його на монітор серійного порту
  }
}
```

Програма буде чекати на введення рядка у монітор серійного порту, після чого виведе його на монітор зворотнього зв'язку. Для ознайомлення з усіма можливими функціями для роботи з аргументами командного рядка, рекомендується прочитати офіційну документацію Arduino.

## Детальніше про читання аргументів командного рядка

Аргументи командного рядка - це рядки тексту, які користувач може вводити при запуску програми з командного рядка. Це дозволяє користувачеві передавати параметри налаштування для програми та взаємодіяти з нею. Надійна обробка аргументів командного рядка дозволить створити більш гнучкий та зручний інтерфейс для користувачів вашої програми.
