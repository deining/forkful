## Чому

Програмування мікроконтроллерів, зокрема Arduino, стає все більш популярним хобі серед українців. Вивчення читання аргументів командного рядка є важливим кроком у покращенні навичок програмування та розробки проектів на Arduino.

## Як це робити

Для того, щоб отримати доступ до аргументів командного рядка в Arduino, використовуйте функцію **Serial.read()**. Наприклад, якщо ви хочете прочитати перший аргумент командного рядка, використовуйте такий код:

```Arduino
char firstArg = Serial.read();
```

Зверніть увагу, що аргументи командного рядка зчитуються один за одним, тому щоб отримати доступ до наступних аргументів, потрібно знову викликати **Serial.read()**.

Для зручності можливо створити функцію, яка буде зчитувати всі аргументи командного рядка і зберігати їх у змінній типу **String**. Нижче наведено приклад такої функції:

```Arduino
String readArgs(){
  String args = "";
  while(Serial.available()){
    args += Serial.read();
  }
  return args;
}

//Виклик функції та збереження аргументів у змінній
String arguments = readArgs(); 
```

Зверніть увагу, що функція **Serial.available()** перевіряє, чи є доступні аргументи для зчитування, та повертає кількість доступних символів.

## Глибоке дослідження

Для більш детального розуміння принципу читання аргументів командного рядка, важливо звернути увагу на такі моменти:

- Якщо передача аргументів здійснюється через послідовний порт з використанням USB, обов'язково встановіть відповідну швидкість передачі даних (бодов)
- У програмі можливо використати роздільник для розпізнавання окремих аргументів
- Пам'ятайте про обмеження пам'яті мікроконтроллера і не перевантажуйте його зчитуванням великої кількості аргументів
