## Чому
Цей блог допоможе вам зрозуміти, як читати текстовий файл з вашої Arduino плати та використовувати його дані для програмування. Знання цього процесу можуть бути корисними для різних застосувань, таких як збереження даних або комунікація з іншими пристроями.

## Як
Щоб почати, створіть текстовий файл з даними, які ви хочете використовувати в своїй Arduino програмі. Наприклад, це може бути список температур, виміряних вашим датчиком, або координати для руху вашого робота. Далі нам потрібно зчитати цей файл у нашій Arduino скетчі. Для цього використовуємо функцію ```File.read()``` та інші вбудовані функції для роботи з файлами.

```Arduino
#include <SD.h> // підключення бібліотеки для роботи з SD карткою

void setup() {
  Serial.begin(9600); // ініціалізація Serial порта
  SD.begin(13); // ініціалізація SD картки з піну 13
  File file = SD.open("data.txt"); // відкриття нашого текстового файлу

  while (file.available()) { // цикл для читання всіх рядків файлу
    String data = file.readStringUntil('\n'); // читання до символу нового рядка
    Serial.println(data); // виведення рядка у Serial порт
  }

  file.close(); // закриття файлу
}

void loop() {
 // можна додати інші операції зчитування даних з файлу
}
```

Цей приклад коду допомагає зрозуміти, як отримати доступ до даних у файлі та вивести їх у Serial порт для подальшого використання у вашій програмі.

## Детальний опис
Хоча прочитання текстових файлів у Arduino може бути простою задачею, є деякі особливості, які потрібно враховувати. Наприклад, можуть виникати проблеми з кодуванням, якщо ви користуєтесь кирилицею у вашому файлі. Також потрібно відмітити, що ваша SD карта повинна бути відформатована у форматі FAT або FAT32, щоб працювати з Arduino.
