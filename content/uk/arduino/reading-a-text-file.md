## Для чого
Для чого потрібно читати текстовий файл на Ардуіно? Час від часу, ви, можливо, захочете отримати дані зовнішнього джерела, як, наприклад, файлу. Наприклад, ви можете зберігати введені користувачем дані у текстовому файлі та потрібно зчитувати їх для подальшої обробки на Ардуіно.

## Як це зробити
Для того, щоб зчитати текстовий файл на Ардуіно, вам потрібно використати функцію `Serial.read()`. Для цього з ним потрібно з'єднати вашу Ардуіно до комп'ютера через USB. Далі складається зчитування даних з файлу у байтовий масив та їх подальша обробка. Ось приклад коду:

```Arduino
void setup() {
  Serial.begin(9600); // встановлення швидкості передачі даних
}

void loop() {
  while (Serial.available()) { // перевірка наявності даних
    char data = Serial.read(); // зчитування даних у змінну
    Serial.println(data); // виведення даних у монітор порту
  }
}
```

При введенні даних у текстовий файл і їх зчитуванні на Ардуіно, ви повинні отримати такий же рядок даних, який ви ввели.

## Вдосконалене дослідження
Для більшого розуміння того, як працює зчитування текстових файлів на Ардуіно, ви можете дізнатися про різні методи та функції, які можуть бути використані, такі як `Serial.parseInt()` для зчитування числових даних, або `Serial.parseFloat()` для зчитування даних з плаваючою точкою. Також, ви можете дослідити роботу з більш складними форматами даних, наприклад, JSON.
