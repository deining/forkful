# Чому

Зчитування текстового файлу є важливою функцією для багатьох проектів, створених на Arduino. Він дозволяє користувачам зберігати та використовувати велику кількість даних для своїх пристроїв. Це може бути корисно для збереження налаштувань, текстових повідомлень або будь-яких інших даних, необхідних для вашого проекту.

# Як це зробити

Існує кілька способів зчитування текстового файлу на Arduino, але ми розглянемо найпростіший. Ви можете використовувати функцію `open()` для відкриття файлу та функції `read()` для зчитування даних з файлу.

```Arduino

// Відкриття файлу для зчитування
File myFile = SD.open("test.txt");

// Перевірка, чи файл вдало відкрито
if (myFile) {

  // Зчитування та вивід першого символу
  char firstChar = myFile.read();
  Serial.println(firstChar);

  // Зчитування та вивід решти файлу
  while (myFile.available()) {
    char charData = myFile.read();
    Serial.print(charData);
  }
  
  // Закриття файлу
  myFile.close();
} else {
  // Вивід помилки, якщо файл не може бути відкрито
  Serial.println("Error opening file!");
}

```

В даному прикладі ми відкриваємо файл з назвою "test.txt" та зчитуємо та виводимо всі дані, що містяться в ньому. Зверніть увагу, що ми закриваємо файл після того, як виконали всі необхідні операції з ним.

# Вивчення глибше

Розглянутий приклад лише базовий для розуміння основ зчитування текстового файлу. Існують багато інших функцій та методів для роботи з файлами на Arduino. Наприклад, ви можете використовувати функцію `write()` для записування даних у файл або функцію `seek()` для переміщення курсора до певної позиції у файлі.

Також, якщо ви зберігаєте більші обсяги даних у текстовому файлі, вам може знадобитися використання бібліотеки `SDfat`, яка дозволяє працювати з більшими розмірами файлів та швидше зчитувати та записувати дані.
