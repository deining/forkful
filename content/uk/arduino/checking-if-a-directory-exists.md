## Чому

Перевірка існування директорії є важливим кроком у програмуванні Arduino. Це дозволяє перевірити, чи існує потрібна нам директорія, щоб забезпечити безперебойну роботу наших пристроїв.

## Як

```Arduino
#include <SD.h> // підключення бібліотеки SD для роботи з SD-картою

void setup() {
  Serial.begin(9600); // встановлення швидкості передачі даних для монітору порту
  while (!Serial) {
    ; // очікування підключення до монітору порту
  }

  if (SD.begin()) { // перевірка наявності SD-карти
    // код для виконання, якщо карта існує
    Serial.println("SD-карта існує");
  } else {
    // код для виконання, якщо карта не існує
    Serial.println("SD-карта не існує");
  }
}

void loop() {
  // код для виконання у циклі
}
```

В цьому прикладі ми підключаємо бібліотеку SD для роботи з SD-картою і встановлюємо швидкість передачі даних для монітору порту. Далі за допомогою умовного оператора ми перевіряємо існування карти за допомогою функції `SD.begin()`. Якщо карта знайдена, на монітор виводиться повідомлення "SD-карта існує", а якщо карта не знайдена - "SD-карта не існує".

## Deep Dive

Перевірка наявності директорії на SD-карті виконується за допомогою функції `SD.exists()`. Ця функція приймає один аргумент - шлях до директорії, яку потрібно перевірити. Якщо директорія існує, то функція поверне `true`, якщо ні - `false`. 

Також варто зазначити, що перед використанням функції `SD.begin()`, необхідно підключити інтерфейсний модуль SD-карти до плати Arduino і налаштувати піни для спілкування з SD-картою.
