## Чому
Програмування мікроконтролерів Arduino може бути дуже захоплюючим заняттям, але часто виникають ситуації, коли потрібно перевірити, чи існує певна директорія. Наприклад, це може знадобитися для збереження даних або створення нових файлів. Для того, щоб програма працювала ефективно, важливо знати, як перевірити існування директорії у своєму коді. 

## Як це зробити
Для перевірки існування директорії використовується функція ```fileExists()```. Нижче наведений приклад коду, який демонструє, як перевірити, чи існує директорія з назвою "my_folder".

```Arduino
#include <SPI.h>
#include <SD.h>

void setup() {
    // ініціалізація SD карти
    if (!SD.begin(10)) {
        Serial.println("SD карта не доступна!");
        return;
    }
    // перевірка існування директорії
    if (SD.exists("my_folder")) {
        Serial.println("Директорія my_folder існує.");
    }
}

void loop() {
    // ваш код для головного використання SD карти
}
```

Якщо директорія існує, на моніторі порта серійного з'єднання виводиться повідомлення "Директорія my_folder існує."

## Глибоке дослідження
Перевірка існування директорії - це лише один з способів роботи з файловою системою на мікроконтролерах Arduino. Існують також інші функції, наприклад, ```mkdir()``` для створення нових директорій і ```rmdir()``` для видалення існуючих. Також важливо враховувати обмежені розміри інтегрованої пам'яті на більшості Arduino до 32 кілобайт. Це може обмежити кількість файлів та директорій, які можна створити.
