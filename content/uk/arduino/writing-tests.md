## Чому

Написання тестів є важливе для будь-якого ардуіно програміста. Воно допомагає перевірити правильність роботи коду та запобігти можливим помилкам.

## Як

Написники тестів у мові програмування Ардуіно прості. Давайте подивимося на приклад коду та його вивід.

```
Arduino. креслення (12,6 "--" )
роздруковувати ("Привіт з України!")
```

Вивід:

```
Привіт з України!
```

## Глибинне занурення

Написання тестів у мові програмування Ардуіно може бути розглянуто як форма документування коду. Про те, які тести були написані та яку вони мають функціональність. Це також сприяє полегшенню роботи над проектом у разі потреби провести зміни та виявлення проблем.
