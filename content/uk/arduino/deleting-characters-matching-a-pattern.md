# Що та чому?

Видалення символів, які відповідають вказаній шаблону - це процес, який використовують програмісти для того, щоб змінювати текстові дані. Наприклад, якщо вам потрібно видалити всі символи, які повторюються у тексті, ви можете використовувати цей підхід. 

Це може бути корисно, коли вам потрібно видалити небажані частини тексту або замінити певні символи на інші. Цей підхід дозволяє швидко та ефективно змінювати текст без необхідності писати складні алгоритми заміни.

# Як?

Нижче наведено приклад коду, який демонструє, як видалити всі символи, які повторюються у введеному рядку. Оберіть свій вхідний рядок і замініть ```your_input_string``` на своє значення.

```arduino
String your_input_string = "Hello World! This is an example string.";
String output_string = "";
for (int i = 0; i < your_input_string.length(); i++) { // прохожу по кожному символу у вхідному рядку
  char current_char = your_input_string[i];
  bool duplicate_found = false;
  for (int j = i + 1; j < your_input_string.length(); j++) { // прохожу по всім наступним символам
    if (current_char == your_input_string[j]) { // якщо є дублікат, встановлюю прапорець
      duplicate_found = true;
      break;
    }
  }
  if (!duplicate_found) { // якщо прапорець не встановлений, значить це унікальний символ і додаю його до вихідного рядка
    output_string += current_char;
  }
}
Serial.println(output_string); // виводжу результат у моніторі порта для перевірки
```

Вихідний рядок буде мати наступний вигляд: ```Helo Wrd! Ti sa eple strng.``` За необхідності ви можете змінити умови умовних операторів, щоб досягти потрібного результату.

# Поглиблене дослідження

Цей підхід є досить старим та добре дослідженим. Його використовують у багатьох мовах програмування, не тільки в Arduino. Також існує кілька альтернативних підходів до видалення символів за шаблоном, наприклад, використання регулярних виразів.

Окрім того, імплементація цього підходу може бути різною залежно від мови та середовища програмування. Наприклад, у Arduino це може бути зроблено за допомогою бібліотеки String або за допомогою масивів символів.

Більш детальна інформація про цей підхід та його застосування може бути знайдена в офіційній документації Arduino або в Інтернеті.
