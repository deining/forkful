## Чому

Існує багато причин, чому люди можуть бажати видаляти символи, що відповідають певному шаблону у своїх програмах на Arduino. Можливо, вам потрібно просто очистити вхідний рядок від зайвих символів або перетворити його на певний формат для подальшої обробки. Це корисний метод, який може допомогти вам зробити вашу програму більш ефективною та зменшити шанси на помилки.

## Як це зробити

```Arduino
// Приклад коду для видалення букв "a" у вхідному рядку

String input = "Arduino";
String output = ""; // буде зберігати вихідний рядок після видалення символів

for (int i = 0; i < input.length(); i++) { // проходимося по всім символам у вхідному рядку
  if (input[i] != 'a') { // перевіряємо, чи поточний символ не є буквою "a"
    output += input[i]; // якщо це так, то додаємо його до вихідного рядка
  }
}

Serial.println(output); // виводимо результат "rduno" у Serial Monitor
```

Це простий приклад того, як ви можете видаляти символи зі строки на Arduino. Ви можете змінювати умову `if` або робити більш складні фільтрації, наприклад, видаляти символи за певними умовами або використовувати більш складні шаблони.

## Глибше

Одна з популярних методів видалення символів зі строки - використання регулярних виразів. Це потужний інструмент, який дозволяє вибирати, змінювати та видаляти певні частини тексту за допомогою спеціальних шаблонів. У Arduino ви можете використовувати бібліотеку "Regex" для роботи з регулярними виразами. Ви можете ознайомитися з документацією [тут](https://www.arduino.cc/reference/en/libraries/regex/).

Також варто звернути увагу на швидкість обробки великої кількості символів на Arduino. Якщо ви збираєтеся видаляти символи з дуже довгих рядків або великої кількості тексту, може бути корисно дослідити та порівняти різні підходи до реалізації цієї задачі.
