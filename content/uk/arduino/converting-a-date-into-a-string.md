## Чому

Arduino - це потужна платформа для розробки, яка дозволяє нам створювати різноманітні проекти з використанням мікроконтролерів. Одним з часто використовуваних елементів у програмуванні на Arduino є обробка дати і часу. Перетворення дати в рядок є важливим елементом у багатьох проектах, і в цій статті ми розглянемо, як це зробити.

## Як

```Arduino
// Підключення бібліотеки для роботи з датою та часом
#include <RTClib.h>

// Ініціалізація об'єкту для роботи з годинником реального часу
RTC_DS1307 rtc;

void setup() {

  // Ініціалізація з'єднання з годинником
  Wire.begin();

  // Перевірка підключення до годинника
  if (! rtc.begin()) {
    Serial.println("Couldn't find RTC");
    while (1);
  }

  // Якщо годинник не має стартового часу, встановлюємо поточний час і дату
  if (! rtc.isrunning()) {
    rtc.adjust(DateTime(F(__DATE__), F(__TIME__)));
  }

  // Отримання поточної дати
  DateTime now = rtc.now();

  // Перетворення дати в рядок у форматі "dd.mm.yyyy hh:mm:ss"
  String date = String(now.day()) + "." + String(now.month()) + "." + String(now.year()) + " " + String(now.hour()) + ":" + String(now.minute()) + ":" + String(now.second());

  // Виведення результата у моніторі порта
  Serial.println(date);
}

void loop() {
}

```

В результаті ви отримаєте такий вивід у моніторі порта: "10.10.2021 13:25:49". Ви можете змінити формат перетворення дати, за допомогою функцій бібліотеки RTClib.

## Точніше про перетворення дати в рядок

Перетворення дати в рядок - це процес, коли ми беремо числа, що представляють рік, місяць, день, годину, хвилини і секунди, і об'єднуємо їх в один рядок у вигляді, який ми хочемо. У прикладі вище, ми використали функцію String(), щоб об'єднати числа з різних змінних у один рядок. Ви також можете використати інші методи, наприклад, функцію sprintf() для форматування рядка, або використовувати бібліотеку для роботи з датою та часом, як ми зробили у прикладі.
