## Чому

Генерація випадкових чисел - це важлива функція, яку можна використовувати для створення випадкових подій в програмі. Це може бути корисно для розваг, випробування різних сценаріїв або для криптографічних цілей.

## Як це зробити

Для початку нам потрібно імпортувати бібліотеку "Random" у наш код Arduino.

```arduino
#include <Random.h>
```

Потім ми можемо викликати функцію "random" для генерації випадкових чисел. Наприклад, щоб отримати випадкове число від 0 до 100, ми можемо використати такий код:

```arduino
int randomNumber = random(0, 100);
```

Ми також можемо використовувати генератор випадкових чисел для створення випадкових кольорів або випадкових символів.

```arduino
int randomColor = random(0, 255); // отримуємо випадкове значення для створення колірного RGB коду
char randomChar = random('a', 'z'); // отримуємо випадковий символ від a до z
```

Зазначте, що при кожному запуску програми буде генеруватися нове випадкове число.

## Глибоке занурення

Використання функції "random" може бути досить заплутаним, оскільки вона використовує різні алгоритми генерації випадкових чисел в залежності від платформи, на якій працює програма. У більшості випадків, цей алгоритм базується на годиннику вбудованого мікроконтролера, тому якість випадкових чисел може бути обмежена.

Крім того, існують спеціальні алгоритми для генерації криптографічно-стійких випадкових чисел, які можуть бути використані для більш чутливих застосувань.
