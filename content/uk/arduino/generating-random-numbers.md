## Чому
Генерація випадкових чисел є важливою частиною багатьох програм, особливо в ігровому світі. Вона може використовуватися для створення випадкової гри чи генерації унікальних значень для кожного гравця. Також вона може бути корисною при розв’язанні складних математичних задач.

## Як
Для створення випадкових чисел в Arduino можна використовувати функцію `random()` або бібліотеку `random`. Нижче наведено приклад генерації 10 випадкових чисел від 0 до 1000 за допомогою функції `random()`.

```Arduino
void setup(){
  Serial.begin(9600); // ініціалізуємо з’єднання з комп’ютером
}

void loop(){
  for (int i = 0; i<10; i++){ // повторюємо 10 разів
    int num = random(1001); // генеруємо випадкове число від 0 до 1000
    Serial.println(num); // виводимо його на монітор послідовного порту
    delay(500); // чекаємо 0.5 секунди
  }
}
```

Результат можна побачити в послідовному вікні монітора, де будуть виведені 10 випадкових чисел.

```
532
765
893
12
499
267
955
621
178
1000
```

## Глибокий погляд
Функція `random()` використовує вбудований генератор випадкових чисел Arduino, який генерує послідовні числа на основі певної формули. Ця формула використовує час від запуску програми як наступну вхідну змінну, тому якщо пристрій запускається в один і той же час, то і випадкові числа будуть повторюватися.

За допомогою бібліотеки `random` можна збільшити випадковість генерованих чисел. Наприклад, функція `randomSeed(seed)` використовується для визначення початкового значення для генератора випадкових чисел. Це значення може бути взяте з аналогового піна или використане число Unix-часу.
