## Що та чому?
Генерування випадкових чисел - це процес створення чисел, які не мають певної логічної послідовності. Це корисний інструмент для програмістів, оскільки дозволяє створювати випадкові елементи, які роблять програму більш цікавою та випадковою.

## Як це зробити:
```
Arduino.Random (min, max); // генерувати випадкове число від min до max
```

У цьому прикладі ми використали функцію Random для змінної Arduino та задали діапазон від min до max. Результат буде випадковим числом між цими двома значеннями.

```
Arduino.RandomSeed (seed); // встановити початкове значення для генерування випадкових чисел
```

Ця функція дозволяє задати початкове значення для генерації випадкових чисел. Це може бути корисно, якщо ви хочете отримувати одні й ті ж випадкові числа при кожному виконанні програми.

## Глибоке погруження:
Генерування випадкових чисел - це процес, який використовується в багатьох програмах, від ігор до криптографії. Раніше для цього використовувалися фізичні пристрої, такі як кулі на барабані або карти, але зараз це робиться за допомогою комп'ютерних алгоритмів.

У Arduino, для генерації випадкових чисел використовується алгоритм ЛЗГ (лінійний конгруентний генератор). Цей алгоритм використовує початкове значення (seed) та математичні операції для генерації послідовності чисел. Хоча цей алгоритм не є повністю випадковим, він добре підходить для більшості застосувань на мікроконтролерах.

Альтернативою алгоритму ЛЗГ є використання випадкових чисел, згенерованих за допомогою спеціальних модулів, таких як "розумний" генератор випадкових чисел (TRNG), які базуються на випадкових подіях в природі, таких як радіоактивний розпад.
