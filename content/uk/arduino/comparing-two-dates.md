## За що

Об’єднання двох дат є важливим елементом багатьох проектів. Незалежно від того, чи ви хочете відстежувати час події, обчислити різницю між двома моментами часу або створити таймер, знання про порівняння дат дозволить вам досягти більшої точності та ефективності у своїй роботі.

## Як це зробити

Щоб порівняти дві дати за допомогою Arduino, вам знадобиться використовувати вбудовану бібліотеку "DateTime". Перш за все, ви повинні оголосити змінні для зберігання двох дат:

```Arduino
DateTime date1 = DateTime(2019, 12, 31, 23, 59, 00); // Перший момент часу
DateTime date2 = DateTime(2020, 1, 1, 0, 00, 00); // Другий момент часу
```

Для порівняння цих двох дат використовуйте функцію "compare()", яка повертає одне з трьох значень: "-1", "0" або "1". Ці значення показують, чи один момент часу раніше, пізніше або рівний іншому моменту часу.

```Arduino
int result = date1.compare(date2); // Порівняти дві дати
```

Залежно від отриманого результату, ви можете виконати різні дії у вашому коді. Наприклад, якщо значення дорівнює "-1", то перший момент часу буде ранішим за другий.

## Глибше пірнання

У бібліотеці "DateTime" також є додаткові функції, які дозволяють здійснювати більш складні порівняння дат. Наприклад, функція "isBefore()" перевіряє, чи є перший момент часу ранішим за другий, і повертає значення типу "bool".

```Arduino
bool is_before = date1.isBefore(date2); // Перевірити, чи перша дата раніше за другу
```

Також можна використовувати функцію "daysTo()", щоб обчислити різницю в датах у днях.

```Arduino
int days = date1.daysTo(date2); // Обчислити різницю в датах у днях
```
