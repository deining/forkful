## Чому

Програмування на Arduino - це не тільки цікава, але й корисна заняття, яке дозволяє нам створювати різноманітні проекти з використанням мікроконтролера. Порівняння двох дат допоможе нам більше зрозуміти процес роботи з часом і датами в Arduino.

## Як

Для порівняння двох дат в Arduino необхідно використати змінні типу "дата" (Date). У наступному прикладі ми покажемо як порівняти дати 22 лютого 2021 року і 31 березня 2021 року:

```Arduino
#include <Time.h>

void setup() {
  Serial.begin(9600);
  // ініціалізуємо дати
  // поточна дата
  Date date1 = Date(22, 2, 2021);
  // інша дата
  Date date2 = Date(31, 3, 2021);
  
  // порівнюємо дати
  if (date1 < date2) {
    Serial.println("Дата 1 менша за дату 2");
  }
  else if (date1 > date2) {
    Serial.println("Дата 1 більша за дату 2");
  }
  else {
    Serial.println("Дати рівні");
  }
}

void loop() {

}
```

Виходячи з коду, ми отримаємо результат "Дата 1 менша за дату 2", оскільки 22 лютого буде перед 31 березня.

## Глибинний аналіз

У Arduino дати зберігаються у форматі UNIX часу, який представляє кількість секунд, які пройшли з 1 січня 1970 року. Порівняння двох дат полягає у порівнянні їх значень UNIX часу. Якщо значення більше, то дата є більшою, якщо менше - то дата є меншою. У разі коли значення співпадають, то дати рівні. Також важливо дотримуватися правильної формату дати при створенні змінних, оскільки від цього залежить правильність порівняння.
