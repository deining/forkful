## Чому

Якщо ви зацікавлені в програмуванні на Arduino, то ви, напевно, задумались про можливість порівняння двох дат. Такий функціонал може стати корисним при створенні різних проектів, які використовують робототехніку або інтернет речей. Також це може бути корисно для перевірки термінів гарантії та інших дедлайнів.

## Як це зробити

Існує декілька способів порівняти дві дати на Arduino, але ми розглянемо найпростіший варіант.

Спочатку нам потрібно задати дві змінні, у яких будуть зберігатися дати. Наприклад:

```Arduino
int date1 = 20210101; 
int date2 = 20201231; 
```

Далі нам потрібно перетворити ці числа в дати, щоб зробити можливим порівняння. Це можна зробити за допомогою функції `setDate()`, яку можна знайти у бібліотеці `TimeLib.h`.

```Arduino
setDate(date1 / 10000, date1 / 100 % 100, date1 % 100); 
setDate(date2 / 10000, date2 / 100 % 100, date2 % 100);
```

Останнім кроком є порівняння двох дат за допомогою функції `year()`, `month()` та `day()`.

```Arduino
if (year(date1) == year(date2) && month(date1) == month(date2) && day(date1) == day(date2)) { 
    Serial.println("Обидві дати однакові!"); 
} else { 
    Serial.println("Обидві дати різні."); 
}
```

Після завершення цих кроків, ви можете вибрати необхідну дату для порівняння та побачити результат у вбудованому моніторі Serial.

## Глибше занурення

Є багато різноманітних функцій та методів для порівняння дат на Arduino. Наприклад, ви можете застосувати цей підхід до дат з годинами, хвилинами та секундами.

Крім того, наявні також бібліотеки, які спрощують процес порівняння, такі як бібліотека `DateTime` або `Date.h`. Вони надають зручні методи для порівняння дат та часів.

Якщо ви хочете розширити свої знання про дати та час на Arduino, вам також корисно ознайомитись з документацією бібліотеки `TimeLib.h`. Тут ви знайдете більше інформації про функції та методи, які можна використовувати для роботи з датами та часами.
