# Що & Чому?
Порівняння двох дат це процес визначення як одна дата відноситься до іншої. Зазвичай програмісти порівнюють дати, щоб зрозуміти, чи події сталися раніше, пізніше або одночасно. Це допомагає зробити правильні висновки та прийняти важливі рішення.

# Як зробити?
Для порівняння дат у Arduino ми можемо використовувати функцію "compare" у бібліотеці "TimeLib.h". Давайте подивимося приклад коду:
```Arduino
#include <TimeLib.h>

void setup() {
  Serial.begin(9600);

  // Встановлюємо дати, які хочемо порівняти
  time_t date1 = makeTime(0, 0, 0, 1, 1, 2020); // 1 січня 2020 року
  time_t date2 = makeTime(0, 0, 0, 1, 1, 2021); // 1 січня 2021 року

  // Порівнюємо дати
  int result = compare(date1, date2);

  // Виводимо результат в Serial Monitor
  Serial.print("Перша дата до другої дати: ");
  Serial.println(result);
}

void loop() {
  // Порожній цикл
}
```
Ви повинні бачити в Serial Monitor результат "Перша дата до другої дати: -1", що означає, що перша дата (1 січня 2020 року) відбулася раніше другої дати (1 січня 2021 року). Якщо би ми змінили порядок дат у функції "compare", то отримали б результат "Перша дата до другої дати: 1", що було б означало, що перша дата відбулася пізніше другої.

# Детальніше
У Arduino, дати представлені у вигляді змінної типу "time_t". Якщо ви хочете отримати поточну дату та час, ви можете використовувати "now()" функцію з бібліотеки "TimeLib.h". Є також "year()", "month()", "day()" та інші функції, які дозволяють отримати окремі частини дати.

Існує також можливість порівнювати дати без використання бібліотеки "TimeLib.h", але це більш складний процес, який вимагає ручного розбору дат та обчислень. Використання "TimeLib.h" дозволяє спростити цей процес.
