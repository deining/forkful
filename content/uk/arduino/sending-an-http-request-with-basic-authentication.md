## Що і для чого?

Відправлення HTTP-запиту з основною автентифікацією - це процес взаємодії з веб-сервером, що вимагає підтвердження своїх користувачів за допомогою ім'я користувача та пароля. Це дуже важливо для програмістів, оскільки це дає можливість забезпечити безпеку віддаленого доступу до веб-сайту або послуги.

## Як це зробити:

Для виконання відправлення HTTP-запиту з основною автентифікацією на платформі Arduino потрібно виконати кілька кроків. Спочатку необхідно ініціалізувати Інтернет-підключення та створити об'єкт для взаємодії з клієнт-серверним протоколом. Потім слід вказати URL-адресу веб-ресурсу, до якого необхідно звернутися, та встановити тип запиту. Наприклад, можна відправити GET-запит для отримання даних з сервера. Для додаткової безпеки потрібно задати ім'я користувача та пароль для основної автентифікації. Нижче наведений приклад коду для виконання цих кроків:

```Arduino
#include <WiFi.h>
#include <HTTPClient.h>

WiFiClient client; // ініціалізуємо Інтернет-підключення
HTTPClient http; // створюємо об'єкт для взаємодії з сервером

http.begin(client, "http://www.example.com/resource"); // вказуємо URL та налаштовуємо запит
http.setType(HTTP_GET); // встановлюємо тип запиту
http.setAuthorization("username", "password"); // задаємо основну автентифікацію

int httpCode = http.sendRequest();

if (httpCode > 0) {
  String response = http.getString(); // отримуємо відповідь від сервера
  Serial.println(httpCode);
  Serial.println(response);
}

http.end();
```

В результаті виконання коду на моніторі виводу буде показано код відповіді сервера та отримані дані.

Відправлення HTTP-запитів з основною автентифікацією стало популярним з поширенням бездротових технологій та Інтернету речей. Існують інші методи автентифікації, такі як шифрування та цифрові підписи, але в основних випадках достатньо використовувати основну автентифікацію. Важливо забезпечувати безпеку своїх даних та доступу до веб-ресурсів, тому використання основної автентифікації є важливим елементом розробки програм для платформи Arduino.
