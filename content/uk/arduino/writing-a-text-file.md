## Чому

Написання текстового файлу є важливою частиною розробки проектів на Arduino для збереження даних або налаштування пристроїв. Це дозволяє зберігати дані навіть після вимкнення плати Arduino.

## Як це зробити

Для написання текстового файлу спершу потрібно відкрити з'єднання до карти SD або внутрішньої пам'яті плати. Далі, використовуючи функцію `File.write()`, можна додавати потрібну інформацію до файлу. Нижче представлено приклад коду для написання текстового файлу:

```Arduino
#include <SD.h> // підключення бібліотеки SD
File dataFile; // оголошення змінної для файлу

void setup() {
  // встановлення початкових налаштувань
  Serial.begin(9600); // встановлення швидкості зв'язку
  SD.begin(4); // початок роботи з модулем SD на піні 4
  dataFile = SD.open("data.txt", FILE_WRITE); // відкриття файлу для запису
}

void loop() {
  // запис даних до файлу
  dataFile.println("Це прикладні дані для запису в файл!");

  // перевірка, чи закрито файл
  if (dataFile) {
    // закриття файлу
    dataFile.close();
    // відображення повідомлення про успішний запис
    Serial.println("Файл успішно записано!");
  } else {
    // відображення повідомлення про помилку
    Serial.println("Помилка при відкритті файлу.");
  }
  // затримка для плавного запису
  delay(1000);
}
```

Після виконання цього коду, текстовий файл з назвою `data.txt` буде створений на карті SD або внутрішній пам'яті плати, а у ньому буде написано "Це прикладні дані для запису в файл!".

## Глибше занурення

Для роботи з текстовими файлами, зокрема для читання та запису, можна також використовувати функції `File.read()` та `File.seek()`. Більше інформації про ці та інші функції можна знайти у документації Arduino.
