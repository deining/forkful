## Чому

Написання текстового файлу є важливою частиною програмування Arduino. Це дозволяє зберігати дані та інформацію на мікроконтролері, що може бути корисним для подальшого аналізу і використання.

## Як

Нижче наведено приклад коду та результату використання текстового файлу в Arduino:

```Arduino
void setup() {
  // відкриття файлу для запису
  File myFile = SD.open("text.txt", FILE_WRITE);

  // записування даних у файл
  myFile.println("Привіт, світ!");
  myFile.println("Це перший файл, який я створюю за допомогою Arduino.");
  myFile.println("Дякую за читання!");

  // закриття файлу
  myFile.close();
}

void loop() {
  // порожня петля
}
```

Результат запуску цього коду буде створення файлу "text.txt" у пам'яті мікроконтролера. Відкрийте цей файл у текстовому редакторі для перегляду доданих рядків.

## Глибше копання

Крім збереження простих текстових даних, використання текстового файлу у програмуванні Arduino також дозволяє читати та змінювати вміст файлу. Це може бути корисно для роботи з більш складними данними, наприклад, читання даних з сенсорів або контролю за роботою пристроїв.
