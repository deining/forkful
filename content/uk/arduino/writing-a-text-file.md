## Що & Чому?

Записування текстового файлу - це процес створення файлу, що містить текстову інформацію. Це може бути корисно для збереження даних або використання в програмах для подальшої обробки. Програмісти використовують запис текстових файлів, щоб зберегти важливі дані та полегшити розробку програм.

## Як це зробити:

Нижче наведені приклади коду та результату виконання для записування текстового файлу в Arduino.

```
Arduino скетч для записування текстового файлу:

#include <SPI.h> // бібліотека для роботи з SD карткою
#include <SD.h> // бібліотека для роботи з SD карткою

void setup() {
  // відкриваємо з'єднання з SD карткою
  Serial.begin(9600);
  while(!Serial) {}

  // перевіряємо, чи вдалося підключити SD картку
  if (!SD.begin(4)) {
    Serial.println("Помилка підключення SD картки");
    return;
  }

  // створюємо новий файл з текстом
  File file = SD.open("text.txt", FILE_WRITE);

  if (file) {
    // записуємо текст у файл
    file.println("Привіт, світ!");
    file.println("Це перший запис у моєму новому файлі.");
    file.close();
    Serial.println("Текст успішно записано");
  } else {
    Serial.println("Не вдалося створити файл");
  }
}

void loop() {
  // нічого не робимо
}
```

Результат виконання:

```
Привіт, світ!
Це перший запис у моєму новому файлі.
```

## Глибоке погруження:

Історичний контекст: Запис текстових файлів в самому початку був істотною частиною програмування. Це було зроблено для збереження даних та забезпечення можливості використовувати їх у майбутніх програмах.

Альтернативи: Є багато інших способів збереження даних, наприклад використання баз даних або збереження даних на веб-сервері. Однак, запис текстових файлів потрібен тоді, коли потрібно зберегти швидкий запис даних без необхідності налаштовувати складні системи.

Особливості реалізації: Використання бібліотек ```SPI``` та ```SD``` необхідне для роботи з SD карткою. Також важливо вибирати правильний тип запису (наприклад, ```FILE_WRITE```), щоб мати можливість редагувати файл.
