## Чому

Саме зараз Ардуіно зробила програмування доступним для всіх, включно з новачками. Відправлення HTTP запитів з Ардуіно дозволяє взаємодіяти з Інтернетом та отримувати потрібну інформацію з веб-ресурсів, зробити дію віддалено або синхронізувати дані. Це відкриває безліч можливостей для проектів з Ардуіно.

## Як це зробити

Для початку нам необхідно підключити бібліотеку Ethernet або WiFi до нашого коду, залежно від того, який модуль ми будемо використовувати. Далі потрібно налаштувати підключення до мережі і вказати необхідні параметри веб-сторінки, до якої буде відправлений запит.

```Arduino
#include <SPI.h>
#include <Ethernet.h> // або #include <WiFi.h>

byte mac[] = { 0xDE, 0xAD, 0xBE, 0xEF, 0xFE, 0xED }; // MAC-адреса модуля
IPAddress server(192,168,1,4); // IP-адреса веб-сторінки

EthernetClient client; // або WiFiClient client;
```

Тепер можна відправляти HTTP запити до веб-сторінки, використовуючи методи `GET` або `POST`. Наприклад, для отримання даних з веб-сторінки і їх запису в змінну `response` використовується такий код:

```Arduino
if (client.connect(server, 80)) { // підключення до веб-сторінки
  client.println("GET /data HTTP/1.1"); // виклик методу GET на адресу /data
  client.println("Host: 192.168.1.4"); // IP-адреса веб-сторінки
  client.println("Connection: close");

  client.println();
}

while (client.connected() && !client.available()) {} // очікуємо відповіді з сервера

while (client.available()) { // якщо дані доступні
  char c = client.read(); // зчитуємо символи з відповіді
  response += c;
}

client.stop(); // закриваємо з'єднання
```

Докладніше про можливості відправлення HTTP запитів з Ардуіно можна дізнатися на офіційному сайті: https://www.arduino.cc/en/Reference/HttpClient

## Глибокий занурення

Щоб отримати більш розгорнуту інформацію про відправлення HTTP запитів з Ардуіно, можна прочитати документацію до бібліотеки Ethernet чи WiFi для більш детального розбору коду. Також, можна попрактикуватися та зробити кілька проектів, де необхідні HTTP запити з Ардуіно, щоб краще зрозуміти цей процес.
