## Завдання

Написання програм для мікроконтролерів може бути захоплюючим і корисним заняттям. Arduino є популярним вибором серед любителів електроніки і програмування, оскільки він пропонує широкі можливості для творчості. У цій статті ви дізнаєтеся, як відправити HTTP-запит за допомогою Arduino та деякі корисні відомості про цей процес.

## Що таке HTTP-запит та чому це важливо?

HTTP-запит - це спосіб взаємодії інтернет-послуг із серверами, де вони зберігають свої дані. Це може бути корисно для отримання даних з Інтернету або збереження інформації на віддаленому сервері. Наприклад, ви можете налаштувати Arduino для отримання інформації про погоду з Інтернету або для зберігання даних з сенсорів на сервері. 

## Як відправити HTTP-запит за допомогою Arduino

Для відправки HTTP-запиту з Arduino вам знадобиться бібліотека Ethernet. Це бібліотека, що дозволяє підключати мікроконтролер до Інтернету. Для початку вам потрібно підключити Ethernet-модуль до Arduino. Після цього встановіть бібліотеку Ethernet в Arduino IDE або завантажте її з офіційного сайту Arduino. Ви також можете використовувати Ethernet Shield для зручного підключення.

```Arduino
#include <SPI.h>
#include <Ethernet.h>
```

Далі вам потрібно налаштувати зв'язок з сервером, до якого ви хочете надіслати запит.
```Arduino
// змінні для збереження IP-адреси сервера та порту
IPAddress serverIP(192, 168, 1, 1);
int port = 80;
// створення об'єкту клієнту
EthernetClient client;
```
Після цього ви можете підключитися до сервера та відправити запит.
```Arduino
// підключення до сервера
if (client.connect(serverIP, port)) {
  Serial.println("Connected to server");
  // відправка HTTP-запиту
  client.println("GET /data HTTP/1.1");
  client.println("Host: 192.168.1.1");
  client.println("Connection: close");
  client.println();
}
```
У цьому прикладі ми використовуємо GET-запит, який дозволяє отримати інформацію з сервера. У рядку "Host" ви повинні вказати IP-адресу або ім'я хоста сервера, до якого надсилаєте запит.
