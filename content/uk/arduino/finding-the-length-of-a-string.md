## Чому

У програмуванні на Arduino дуже часто виникає необхідність знаходження довжини рядка. Це може знадобитися для обробки текстової інформації або для роботи зі строковими командами. Також, знаходження довжини рядка є важливим елементом для оптимізації пам'яті та уникнення переповнення буфера.

## Як це зробити

В Arduino існує багато способів знаходження довжини рядка, але ми розглянемо один із найпростіших - за допомогою функції `strlen()`. Для початку, створимо змінну `str` і призначимо їй значення рядка, довжину якого хочемо знайти.

```Arduino
char str[] = "Привіт, Україно!";
```

Далі, викличемо функцію `strlen()` і передамо в неї параметром нашу змінну `str`. Результатом буде число, яке відповідає довжині рядка.

```Arduino
int length = strlen(str);
```

Щоб перевірити правильність роботи, виведемо значення змінної `length` у монітор порта за допомогою функції `Serial.print()`.

```Arduino
Serial.print(length);
```
Результатом виконання програми буде:

```
16
```

## Глибше

Функція `strlen()` визначає довжину рядка, шукаючи перший символ з кодом 0 (нуль) і повертаючи його індекс у рядку. Тобто, фактично, рахується кількість символів від початку рядка до 0 (нуль-термінатора). Наприклад, для рядка `"Привіт!"` довжина буде 7, бо 0 (нуль-термінатор) знаходиться після сьомого символу.

Також варто зазначити, що дана функція працює тільки з масивом типу `char`, тобто з рядками, а інші типи даних вона не обробляє.
