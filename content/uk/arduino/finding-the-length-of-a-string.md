## Почему
Багато людей цікавляться програмуванням Arduino не тільки для того, щоб робити прості електронні пристрої, але і для створення складних проектів. Іноді вам потрібно знати довжину рядка, який ви використовуєте в своїх програмах. Це може бути корисно, наприклад, для перевірки, чи користувач правильно ввів дані або для обробки великої кількості даних.

## Як
Використовуйте функцію `strlen()` для знаходження довжини рядка у вашому коді Arduino. Нижче наведений приклад коду, який демонструє використання цієї функції.

```Arduino
char string[] = "Це приклад рядка.";
int length = strlen(string);
Serial.println(length); //Виводить "17" (довжина рядка)
```
В цьому прикладі ми зберігаємо рядок у змінну `string` та використовуємо функцію `strlen()` для знаходження його довжини, яка потім виводиться на монітор серійного порту.

## Глибше
Існує кілька інших функцій, які можна використовувати для роботи з рядками в Arduino, наприклад `strcat()` для об'єднання двох рядків, `strcmp()` для порівняння двох рядків та `strtok()` для розбиття рядка на підрядки за допомогою певного розділювача. Більше інформації про ці функції можна знайти у офіційній документації Arduino.
