## Що і Навіщо?

Знаходження довжини рядка - це процес визначення кількості символів, які містяться у заданому рядку. Це важлива операція для програмістів, оскільки дозволяє легко маніпулювати рядками та виконувати різні операції з ними.

## Як: 
Нашою метою є знаходження довжини рядка "Hello World!", тому наш код буде виглядати так:

```Arduino
char str[] = "Hello World!"; //створюємо рядок
int length = 0; // змінна, у яку зберігатиметься довжина рядка
for(int i=0; str[i]!='\0'; i++){ // проходимося циклом по кожному символу рядка
	length++; // кожного разу, коли зустрічаємо новий символ, збільшуємо змінну
}
Serial.println(length); // виводимо результат у Serial Monitor
```

Результат у Serial Monitor буде виглядати так: `12`, оскільки рядок "Hello World!" складається з 12 символів.

## Глибша Погляд:
Історичний контекст: Знаходження довжини рядка є важливою та необхідною операцією у багатьох мовах програмування, включаючи С++, Java, та інші. 
Альтернативи: У більш високорівневих мовах програмування, таких як Python, існують вбудовані функції для знаходження довжини рядка.
Деталі реалізації: Завдяки тому, що в рядках останнім символом є `'\0'`, можна використовувати цей символ як ознаку кінця рядка і перестати перевіряти кожен символ під час обходу.
