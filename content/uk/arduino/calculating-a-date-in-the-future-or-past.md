# Що і чому?

Обчислення дати у майбутньому або минулому - це здатність визначати дату у будь-якому моменті часу. Це важливий інструмент для програмістів, оскільки дозволяє відстежувати події і розраховувати час до майбутніх подій.

# Як це зробити:

Щоб зробити це за допомогою Arduino, спочатку потрібно включити бібліотеку "Time.h". Потім ми можемо використовувати функцію "now ()" для отримання поточної дати і часу. Далі використовуємо функцію "maketime ()", яка приймає значення року, місяця, дня, годин, хвилин і секунд, щоб створити бажану дату. Наприклад, якщо ми хочемо отримати дату через 10 днів, ми можемо використовувати функцію "maketime (now () + (10 * 24 * 3600))". Це дозволить нам обчислити і отримати дату у форматі Unix Timestamp.

```
#include <Time.h>

void setup() {
  Serial.begin(9600);
  setTime(8, 29, 31, 15, 11, 2020); //встановлення початкової дати для прикладу
}

void loop() {
  time_t now = now();
  time_t future = now + (10 * 24 * 3600); //обчислення дати через 10 днів
  Serial.println(now());
  Serial.println(future);
  delay(1000);
}
```

Результатом буде виведення поточної дати у форматі Unix Timestamp і дати через 10 днів у Serial Monitor.

```
1605436175
1606468175
```

# Глибокий занурення:

Історичний контекст:
Обчислення дати в майбутньому або минулому було важливою задачею у розробці комп'ютерних програм вже багато років. За допомогою цієї функціональності можна стежити за циклами та визначати час до подій, що допомагає в управлінні запланованими задачами.

Альтернативи:
Існує багато бібліотек для обчислення дати, таких як "DateTime.h", "TimeLib.h" і "NTPClient.h", які також забезпечують потужні функції для роботи з датою і часом.

Деталі реалізації:
Бібліотека "Time.h" використовує 32-бітові значення Unix Timestamp, що дозволяє відстежувати час від початку січня 1970 року. Це значення зберігається у змінній типу "time_t".
