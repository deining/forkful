## Чому
Багато людей цікавляться обчисленням дати в майбутньому або минулому для різних цілей, таких як планування подій або ведення обліку. Ця стаття допоможе вам зрозуміти, як таке обчислення можна виконати за допомогою Arduino.

## Як використати:
Будь-який розумний проект на Arduino потребує обчислень. Будь-який процес, що потрібно виконати в заздалегідь встановлений час, вимагає обчислень. Розглянемо приклад коду на Arduino, який дозволить вам обчислити дату в майбутньому або минулому в залежності від ваших потреб.

```
ArduinoDateTime myDateTime = ArduinoDateTime(2021, 9, 15, 12, 0, 0); // ініціалізуємо об'єкт дати з встановленими значеннями
int yearsToAdd = 2; // змінна, що вказує скільки років додати до поточної дати
int daysToAdd = 30; // змінна, що вказує скільки днів додати до поточної дати
myDateTime.addYears(yearsToAdd); // додавання років до поточної дати
myDateTime.addDays(daysToAdd); // додавання днів до поточної дати
```

Результатом виконання цього коду буде об'єкт дати, де значення будуть відповідно змінені в залежності від ваших вхідних параметрів. Ви можете також користуватися іншими методами, щоб обчислити дату в минулому або в майбутньому з урахуванням інших параметрів, таких як місяці, години, хвилини або секунди.
Для більш детальної інформації про методи обчислення дати дивіться [документацію](https://www.arduino.cc/en/Reference/RTC).

## Глибше
Написання коду для обчислення дати на Arduino може здатися складним завданням, але усе залежить від ваших потреб і здатностей програмування. Не забувайте про наявні методи, такі як `setDateTime()`, `subtractYears()`, `subtractDays()` та інші, які можна використовувати для обчислення більш складних дат. Крім того, ви можете використовувати бібліотеки для зручнішого обчислення дати на Arduino, як от бібліотека для [обчислення часу для ESP8266 і ESP32 мікроконтролерів](https://github.com/PaulStoffregen/Time).
