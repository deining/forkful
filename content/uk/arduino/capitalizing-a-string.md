## Що і чому?
Заголовок
Капіталізація рядка - це процес перетворення першої літери рядка на заголовну. Це часто використовується програмістами для полегшення опрацювання даних та надання більш зрозумілого вигляду. 

## Як це зробити:
```Arduino 
String str = "привіт світ";
Serial.println(str); // виводиться "привіт світ"
str.capitalize();
Serial.println(str);// виводиться "Привіт світ"
```

## Глибше поглядіть:
Історичний контекст: У ранніх версіях мови програмування, таких як C, для капіталізації рядка необхідно було використовувати власну функцію. З появою новіших мов, таких як C++, цей процес став більш спрощеним завдяки вбудованим функціям. 

Альтернативи: Іншими способами капіталізації рядка є застосування циклу та умовного оператора, або використання сторонніх бібліотек.

Деталі реалізації: Бібліотека String має вбудовану функцію capitalize(), яка на основі коду ASCII перетворює першу літеру рядка на заголовну. Ця функція також робить першу літеру параграфу та кожну літеру після знаків пунктуації заголовною.
