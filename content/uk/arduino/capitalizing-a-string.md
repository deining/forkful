## Чому

Програмування на Arduino є корисним і захопливим хобі, яке може принести багато користі у повсякденному житті. І знання про перетворення рядка є важливою навичкою, яка може бути корисною для проектів зі зчитуванням та обробкою даних.

## Як зробити

Для перетворення рядка на великі літери використовуйте функцію toUpperCase(). Наприклад, якщо ми створимо рядок "привіт", то використавши функцію toUpperCase(), він буде виглядати "ПРИВІТ". Окрім цього, можна також використовувати функцію toLowerCase() для перетворення рядка на малі літери.

```Arduino
String text = "привіт";
String upperCaseText = text.toUpperCase();
Serial.println(upperCaseText); // виведе "ПРИВІТ" у моніторі послідовного порту
```

Також варто зазначити, що функція toUpperCase() не змінює оригінальний рядок, а повертає новий, перетворений рядок.

## Поглиблене дослідження

У Arduino існує багато корисних функцій для роботи з рядками. Наприклад, функція indexOf() дозволяє знайти позицію певного символу або рядка в іншому рядку. Також можна використовувати функції для вирізання частин рядка, знаходження довжини рядка та інші.
