## Чому

Програмування на Arduino дозволяє створювати різноманітні проекти, включаючи інтерактивні іграшки, домашні автоматизаційні системи та робототехніку. Капіталізація рядків є важливою функцією, яка дозволяє змінювати вигляд тексту в програмі. Необхідність у цій функції виникає тоді, коли потрібно використати великі ініціали для назв змінних чи функцій.

## Як це зробити

Для капіталізації рядка у програмі Arduino використовується функція `toUpperCase()`. Наприклад:

```Arduino
String name = "arduino";
name.toUpperCase();
Serial.println(name);
```

В результаті виконання цього коду на моніторі виведеться "ARDUINO". Ця функція змінює всі символи рядка на великі літери.

## Вдосконалення

Інша корисна функція, пов'язана з капіталізацією рядків - `charAt()`. Вона дозволяє отримати певний символ із рядка за його індексом. Наприклад, якщо необхідно виконати капіталізацію лише першої літери в рядку:

```Arduino
String name = "arduino";
name[0] = name[0].toUpperCase();
Serial.println(name);
```

На виведенні буде "Arduino", оскільки функція `toUpperCase()` була застосована лише до першої літери рядка.
