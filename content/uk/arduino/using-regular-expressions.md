## Для чого

У програмуванні часом нам потрібно шукати певні шаблони в рядках тексту. Використання регулярних виразів дозволяє нам швидко та зручно знаходити, перевіряти та замінювати певні шаблони, що є незамінним інструментом для розробників.

## Як

Щоб використовувати регулярні вирази в Ардуіно, нам потрібно використовувати бібліотеку String, яка містить вбудовані функції для роботи з рядками. Давайте розглянемо кілька прикладів.

```Arduino
#include <String.h>

void setup(){
  Serial.begin(9600);
}

void loop(){
  String text = "Привіт, моє ім'я - Ардуіно!";
  if(text.match("Ардуіно")){
      Serial.println("Це моє ім'я!");
  }
  if(text.indexOf("Привіт") >= 0){
      Serial.println("Це перший рядок!");
  }  
}
```

У цьому прикладі ми використовуємо функцію `match`, що дозволяє перевірити, чи збігається рядок з певним виразом (у нашому випадку - "Ардуіно"). Також, ми використовуємо функцію `indexOf`, щоб знайти позицію певного слова (у нашому випадку - "Привіт") у рядку `text`.

```Arduino
#include <String.h>

void setup(){
  Serial.begin(9600);
}

void loop(){
  String text = "Ціна: $200";
  String price = text.substring(text.indexOf("$") + 1); // 200
  Serial.println(price);
}
```

У цій програмі ми використовуємо функцію `substring` для виділення частини рядка з певною позицією (у нашому випадку - після символу "$"). Це дозволяє нам отримати ціну з рядка і далі працювати з нею у програмі.

## Глибше копання

У нашому прикладі ми використовували дуже прості регулярні вирази, але вони можуть бути набагато складнішими та потужнішими. Наприклад, у нашому першому прикладі вміст рядка мінявся, але ви можете також замінювати окремі символи, розділяти рядки на частини за певним символом та багато іншого.

При роботі з регулярними виразами важливо пам'ятати, що вони можуть бути складними та запутаними для новачків. Тому, рекомендується використовувати маленькі шаблони та робити перевірку крок за кроком.
