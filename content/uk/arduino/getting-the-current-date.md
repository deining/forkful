## #Чому

Отримання поточної дати є важливою функцією для багатьох проектів, що використовують Arduino. Наприклад, ви можете створити проект, який ввімкне світло в певний час або зберігати дані з певною датою і часом. Також, використання поточної дати дає змогу створити більш точний логгер даних, що може бути корисним у багатьох сферах.

## #Як

```Arduino
#include <RTClib.h> //підключення бібліотеки для роботи з датою і часом
RTC_DS1307 rtc; //ініціалізація годинника RTC DS1307

void setup() {
  Serial.begin(9600); //ініціалізація злнчення з Serial Monitor
  rtc.begin(); //ініціалізація роботи з годинником
  if (!rtc.isrunning()) { //перевірка, чи працює годинник
    Serial.println("Час не встановлено!"); 
    rtc.adjust(DateTime(F(__DATE__), F(__TIME__))); //встановлення поточної дати і часу, якщо годинник не працює
  }
}

void loop() {
  DateTime now = rtc.now(); //отримання поточної дати та часу
  Serial.print(now.year(), DEC); //виведення на Serial Monitor року в десятковому форматі
  Serial.print('/'); 
  Serial.print(now.month(), DEC); //виведення місяця
  Serial.print('/'); 
  Serial.print(now.day(), DEC); //виведення дня
  Serial.print(' ');
  Serial.print(now.hour(), DEC); //виведення години
  Serial.print(':'); 
  Serial.print(now.minute(), DEC); //виведення хвилини
  Serial.print(':'); 
  Serial.print(now.second(), DEC); //виведення секунди
  Serial.println(); 
  delay(1000); //затримка для оновлення даних кожну секунду
}
```

## #Глибока пірка

Для отримання поточної дати та часу використовується годинник реального часу (RTC). Найпопулярнішими модулями годинників для Arduino є DS1307 і DS3231. Для роботи з датою та часом використовується бібліотека "RTClib". Для отримання поточної дати користувач має правильно встановити дату і час на годиннику. Якщо годинник не працює або його не підключено, можна використати команду "rtc.adjust()" для встановлення поточної дати і часу на основі дати комп'ютера, на якому ви працюєте. Також, в більш складних проектах, можна використовувати модуль Wi-Fi або Ethernet для синхронізації годинника з Інтернету, щоб завжди мати точний час і дату.
