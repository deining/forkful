## Чому:

Виведення дебаг-виводу на екран є важливою функцією при написанні програм для Arduino. Це дозволяє вам перевірити правильність виконання коду та виявити помилки, які можуть бути сховані в різних частинах вашої програми.

## Як:

Для виведення дебаг-виводу на екран у програмі для Arduino використовується функція `Serial.print()`. Нижче наведено приклад коду та відповідного виводу:

```Arduino
int value = 10;
Serial.print("Змінна value має значення: ");
Serial.println(value);
```

Вивід:
```
Змінна value має значення: 10
```

У цьому прикладі ми використали `Serial.print()` для виведення значення змінної `value` та `Serial.println()` для додавання символу нового рядка після виведення. Також можна використовувати інші функції, наприклад `Serial.print()` для початку нового рядка або `Serial.write()` для виведення символу.

## Deep Dive:

Для детальнішого вивчення функцій для виведення дебаг-виводу можна ознайомитися з документацією Arduino Serial. Також, використовуючи `Serial.begin()`, можна налаштувати швидкість передачі даних через порт для виведення.

Важливо зазначити, що виведення дебаг-виводу може збільшити час виконання програми та споживання пам'яті, тому рекомендується вимикати його у виробничому середовищі.
