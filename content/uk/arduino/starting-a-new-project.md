## Що і Чому?

Start-up проекту неґрунтований, це просто ваша програма або проект з Arduino, який ви хочете розпочати. Програмісти часто створюють нові проекти, щоб експериментувати з новими ідеями, вдосконалювати вже наявні проекти або ділитися своїми роботами з іншими користувачами Arduino.

## Як:

```Arduino
int main() {                // Оголошуємо функцію main
  Serial.begin(9600);       // Ініціалізуємо послідовний порт з швидкістю 9600 біт/с
  pinMode(LED_BUILTIN, OUTPUT); // Налаштовуємо вбудований LED як вихід
  digitalWrite(LED_BUILTIN, HIGH); // Включаємо вбудований LED
  Serial.println("Привіт світе!"); // Виводимо повідомлення в послідовний порт
  while(1) {                // Безкінечний цикл
    digitalWrite(LED_BUILTIN, HIGH); // Включаємо вбудований LED
    delay(1000);            // Затримка 1000мс
    digitalWrite(LED_BUILTIN, LOW); // Вимикаємо вбудований LED
    delay(1000);            // Затримка 1000мс
  }
}
```

В результаті програма буде включати та вимикати вбудований LED з інтервалом 1 секунда та виводити повідомлення "Привіт світе!" в послідовний порт.

## Ще детальніше:

В 2005 році, інженер Massimo Banzi заснував проект Arduino з метою навчання студентів простим способом для створення цифрових пристроїв. З того часу, багато людей використовують Arduino для створення різноманітних проектів, таких як роботи, носимі пристрої, автоматизація та багато іншого. Існують також альтернативні платформи, такі як Raspberry Pi, які можуть також бути використані для розробки проектів.

Коли ви починаєте новий проект з Arduino, вам необхідно спочатку отримати платформу Arduino (будь-яку сумісну модель), розробник Arduino IDE та базові знання програмування на мові C++. Ідея проекту може бути будь-яка - від простого світлофору до складнішої системи управління будинком.
