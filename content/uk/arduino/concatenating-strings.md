## Чому

Написання програм для Arduino може бути цікавою та захоплюючою заняттям, але іноді деякі завдання можуть задати проблеми. Наприклад, якщо потрібно об'єднати декілька текстових рядків в одну змінну. Це може здатися складним завданням, але насправді це дуже просто, як тільки ви зрозумієте принцип конкатенації рядків.

## Як це зробити

Для початку, вам потрібно створити змінну типу `String` і присвоїти їй перший текстовий рядок. Далі, за допомогою оператора `+` ви можете додавати до цієї змінної інші текстові рядки. Наприклад:

```Arduino
String text = "Привіт";
text = text + " усім!";
```

В результаті, змінна `text` буде містити текстовий рядок "Привіт усім!".

Але якщо ви хочете додати не тільки текстові рядки, але й числа, вам потрібно використовувати функцію `String()` для перетворення числа в текстовий рядок. Наприклад, якщо ви хочете додати значення потенціометра до текстового рядка:

```Arduino
int value = analogRead(A0);
String text = "Значення потенціометра: ";
text = text + String(value);
```

У цьому випадку, змінна `text` буде містити, наприклад, рядок "Значення потенціометра: 512".

## Глибше занурення

Конкатенація рядків - це процес з'єднання текстових рядків в один. Це може бути корисно для створення повідомлень для користувача, виведення інформації на LCD екран або збереження даних в одну змінну. Важливо пам'ятати, що конкатенація рядків відбувається за допомогою оператора `+`, але не обмінює значеннями змінних. Це означає, що якщо потрібно зберегти об'єднаний текст у нову змінну, потрібно знову присвоїти його до цієї змінної.
