## Що і чому?

Перевірка існування директорії є важливим етапом для багатьох програмістів, оскільки це дозволяє їм перевірити наявність певної директорії перед виконанням певних дій з нею. Наприклад, це може бути корисно, якщо програма потребує певної директорії для збереження даних або для отримання файлів для обробки.

## Як це зробити:

Для перевірки існування директорії у Python, можна використовувати функцію `os.path.exists()` з модуля `os`. Ця функція повертає `True`, якщо директорія існує, і `False` у противному випадку.
```Python
import os
directory = "path/to/directory"
if os.path.exists(directory):
    print("Директорія існує")
else:
    print("Директорія не існує")
```
Ви також можете використовувати функції `isdir()` або `isfile()` для перевірки конкретної директорії або файлу.

## Глибше копання:

Перевірка існування директорії виконувалась і в старших версіях Python, наприклад, у версії 2.3. Також існують інші шляхи перевірки, такі як використання умов операторів `try` і `except`, але використання `os.path.exists()` є найбільш простим і швидким способом.

Є також альтернативи, такі як використання модуля `pathlib` у Python 3, який надає зручний і більш сучасний спосіб роботи з файловою системою. Проте, для старших версій Python, `os.path.exists()` є надійнішим варіантом.
