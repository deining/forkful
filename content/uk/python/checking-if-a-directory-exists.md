## Для чого?
Перевірка, чи існує директорія, є важливою частиною програмування на Python. Це дозволить вам впевнитися в тому, що ваша програма працює з правильними шляхами до файлів і уникнути зайвих помилок.

## Як це зробити?
```Python
import os
path = "/Users/user/Folder"
if os.path.exists(path):
    print("Директорія існує")
else:
    print("Директорія не існує")
```

В цьому прикладі ми використовуємо модуль `os` для перевірки існування директорії `Folder` за шляхом `/Users/user/Folder`. Якщо директорія існує, то на екран виведеться повідомлення "Директорія існує". У противному випадку, буде виведено повідомлення "Директорія не існує".

## Глибока розробка
Перевірка існування директорії може бути корисною і при створенні нових директорій. Наприклад, ви можете використовувати цей метод, щоб перевірити, чи вже існує директорія з такою ж назвою, яку ви хочете створити. Це може допомогти уникнути перезапису вже існуючих директорій.
