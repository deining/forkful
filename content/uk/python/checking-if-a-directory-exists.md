## Чому
Часто в процесі програмування нам потрібно перевірити, чи існує певний каталог на нашому комп'ютері. Це може бути важливо для правильного виконання програми та уникнення помилок. У цій статті ми розглянемо, як перевірити існування каталогу використовуючи мову програмування Python.

## Як
Для перевірки існування каталогу використовується функція `os.path.exists()`. Нижче наведений приклад коду та виведена зразок вихідного повідомлення.

```Python
import os

# перевірка існування каталогу
if os.path.exists("my_directory"):
  print("Каталог існує.")
else:
  print("Каталог не існує.")
```

Вивід:
```
Каталог існує.
```

## Profundum
Функція `os.path.exists()` повертає `True` якщо каталог існує та `False` у протилежному випадку. Для більш детального розуміння роботи цієї функції, давайте розглянемо її реалізацію.

Спочатку, використовуючи модуль `os`, отримуємо доступ до системних функцій нашого комп'ютера. Далі, використовуючи метод `getcwd()` отримуємо поточний робочий каталог. Наступним кроком, за допомогою функції `listdir()` перебираємо всі елементи поточного робочого каталогу та порівнюємо їх з назвою каталогу, який ми шукаємо. Якщо збігається, то функція повертає `True`, в іншому випадку - `False`.
