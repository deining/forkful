## Чому

Перевірка існування директорії є важливим кроком у розробці програм, оскільки вона дозволяє перевірити наявність необхідної структури даних перед виконанням операцій з ними.

## Як це зробити

```Python
import os # імпортуємо модуль os

# перевіряємо, чи існує директорія "documents"
if os.path.isdir("documents"):
  print("Директорія documents існує.")
else:
  print("Директорії documents не існує.")
```

Вище наведений приклад дозволяє перевірити наявність директорії "documents" за допомогою використання методу `os.path.isdir()`. Якщо директорія існує, то у консолі виведеться повідомлення про це, інакше - повідомлення про її відсутність.

## Глибоке дослідження

Окрім методу `os.path.isdir()`, є ще декілька способів перевірити існування директорії. Наприклад, можна використовувати функцію `os.path.exists()`, яка дозволяє перевірити наявність будь-якого типу файлу або директорії. Також, ви можете використовувати спеціальну бібліотеку `pathlib`, яка надає зручний спосіб перевірити існування директорії та багато іншого. Головне - вибрати метод, який найкраще відповідає вашим потребам із надійністю і продуктивністю.
