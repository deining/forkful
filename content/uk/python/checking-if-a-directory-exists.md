# Зачем
Якщо ви розробник програмного забезпечення або студент, ви, напевно, знаєте, що іноді ваша програма повинна перевірити, чи існує певний каталог на вашому комп'ютері. Це може бути джерелом даних для вашої програми або просто звичайний каталог для збереження файлів. В будь-якому випадку, перед створенням нового каталогу або роботою з вже існуючим, важливо перевірити, чи він дійсно існує. Це дозволить уникнути помилок та зберегти час на пошуки непотрібних помилок у програмі.

## Як перевірити, чи існує каталог

Для перевірки існування каталогу у Python використовується функція `os.path.isdir()`. Вона приймає шлях до каталогу як її аргумент і повертає `True`, якщо каталог існує, або `False`, якщо його не існує.

```Python
import os

# Приклад перевірки існування каталогу
path = "C:/Users/Ukrainian/Documents"

if os.path.isdir(path):
    print("Каталог існує")
else:
    print("Каталог не існує")
```

У даному прикладі ми використовуємо шлях до каталогу `C:/Users/Ukrainian/Documents`, але це може бути будь-який шлях до каталогу на вашому комп'ютері. Важливо лише, щоб він був коректним.

## Детальний аналіз перевірки існування каталогу

Хоча використання функції `os.path.isdir()` досить просте, варто знати, що вона виконується не лише за допомогою перевірки, чи існує каталог, але й за допомогою перевірки прав доступу до каталогу. Якщо ваша програма не має доступу до каталогу, функція також поверне `False`, незалежно від того, чи каталог існує фізично чи ні. Це важливо враховувати при використанні функції, адже з нею може статися несподівана помилка, якщо ваша програма не має дозволу на роботу з каталогом.
