# Що і навіщо?
Робота з CSV - це зчитування та записів данних у форматі "кома-розділений текст". Цей формат зручний для збереження табличних даних, таких як імена, адреси, телефонні номери та інші деталі. Програмісти часто працюють з CSV для обробки великих обсягів даних, наприклад, при імпорті даних з баз даних або веб-скрапінгу.

## Як працювати з CSV в Python:
Щоб зчитати дані з файлу CSV, можна скористатися модулем ```csv``` у Python. Для цього необхідно відкрити файл, використовуючи функцію ```open()``` і передати його об'єкту модуля ```csv.reader()```. Потім можна прочитати дані за допомогою циклу ```for``` і вивести їх у потрібному форматі, наприклад, у вигляді списку.

```Python
import csv
with open('file.csv') as csv_file:
    csv_reader = csv.reader(csv_file, delimiter=',')
    for row in csv_reader:
        print(row)
```

Результат буде виглядати приблизно так:

```Python
['name', 'surname', 'phone number']
['John', 'Smith', '123456789']
['Jane', 'Doe', '987654321']
```

Для запису данних у файл CSV використовується модуль ```csv.writer()```. Також, можна використовувати додаткові параметри, такі як ```dialect``` для вказування окремих роздільників або ```newline``` для визначення роздільників між рядками.

```Python
import csv
with open('file.csv', 'w') as csv_file:
    csv_writer = csv.writer(csv_file, delimiter='|')
    csv_writer.writerow(['name', 'surname', 'phone number'])
    csv_writer.writerow(['John', 'Smith', '123456789'])
    csv_writer.writerow(['Jane', 'Doe', '987654321'])
```

Результат буде записано у файл у форматі:

```Python
name|surname|phone number
John|Smith|123456789
Jane|Doe|987654321
```

## Глибше вдивіться:
Формат CSV був розроблений у 1972 році компанією IBM для збереження даних у табличному форматі. Досі він є популярним серед різних програм та систем, так як дозволяє зберігати та оброблювати великі обсяги даних. Альтернативами CSV є інші формати, такі як JSON або XML, але вони не мають такого ж простого синтаксису.

При роботі з CSV варто звернути увагу на деякі нюанси, наприклад, наявність роздільників у тексті, що може призвести до помилок у зчитуванні. Також, можна використовувати спеціальні бібліотеки, які розширюють функціональність CSV, наприклад, ```csvkit``` або ```pandas```.
