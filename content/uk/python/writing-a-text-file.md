Український блог для програмування: Як написати текстовий файл у Python

## Why (Чому)

Написання текстового файлу є необхідною навиною для будь-якого програміста. Це дозволяє зберігати дані на комп'ютері і редагувати їх у зручному форматі. Також, це корисно, коли потрібно вивести велику кількість даних, наприклад, з результатів обробки файлу.

## How To (Як) 

Написання текстового файлу у Python є досить простим процесом. Ми можемо використати вбудовану функцію `open()` для відкриття файлу і `write()` для введення даних у файл.

```Python
file = open("example.txt", "w") # відкриваємо файл у режимі запису
file.write("Hello! This is a sample text file.") # вводимо дані у файл
file.close() # закриваємо файл
```
Ви можете перевірити ваш файл на комп'ютері і побачити, що він з'явився з введеними даними.

## Deep Dive (Занурення)

Коли ви використовуєте функцію `open()` для створення файлу, ви повинні вказати режим, у якому він буде відкритий. У нашому випадку, це "w" означає режим запису. Для читання файлу, ви можете використати режим "r". Також є можливість додати дані до вже існуючого файлу з режимом "a".

Також використовуючи функцію `write()`, важливо пам'ятати, що вона замінює попередні дані в файлі. Якщо ви хочете додати дані до вже існуючого, ви можете використати процедуру, як показано нижче:

```Python
file = open("example.txt", "a") # відкриваємо файл у режимі додавання
file.write(" This is an added line.") # додаємо дані
file.close()
```
