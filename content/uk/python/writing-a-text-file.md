## Чому

Написання текстового файлу є важливою навичкою для кожного програміста. Воно дозволяє зберігати та організовувати дані, що потрібні для вашої програми, а також забезпечує резервне копіювання важливої інформації.

## Як

Для створення текстового файлу використовуються стандартні методи програмування Python. На початку потрібно імпортувати модуль ```io``` для роботи з файлами і використовувати метод ```open()``` для створення та відкриття файлу.

```Python
import io

with open("text.txt", "w") as file: # Вказуємо назву файлу і режим запису "w"
    file.write("Привіт, світе!") # Записуємо текст у файл
```

Після виконання цього коду, у вашій папці з'явиться файл з назвою "text.txt" та з написом "Привіт, світе!".
