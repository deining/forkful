## Чому

Ви могли б питати себе: чому потрібно отримувати поточну дату у своїх програмах? Ну, відповідь доволі проста. Багато програм потребують використання актуальної дати для різних операцій. Наприклад, ви можете хотіти додати поточну дату до назви файлу, або перевірити срок дії певного продукту. Без зручного способу отримання поточної дати, ці завдання можуть бути дуже складними. Тож, давайте подивимося, як це зробити в мові програмування Python.

## Як

Отримання поточної дати в Python - це дуже просто. Для цього використовується модуль `datetime`. Для початку, додайте його у свій код:
```python
import datetime
```
Після цього, ви можете використовувати функцію `datetime.now()` для отримання поточної дати. Наприклад:
```python
current_date = datetime.now()
```
Це створить змінну `current_date`, яка буде містити об'єкт з поточною датою та часом. Щоб вивести цю дату на екран, скористайтеся методом `strftime()`:
```python
print(current_date.strftime("%d/%m/%Y %H:%M:%S"))
```
У цьому прикладі, ми використовуємо формат `%d/%m/%Y` для представлення дати у форматі `день/місяць/рік`, а `%H:%M:%S` - для представлення часу у форматі AM/PM. Звичайно, ви можете вибрати будь-який інший формат в залежності від вашої потреби.

## Глибоке занурення

Тепер, коли ми знаємо, як отримати поточну дату, давайте розглянемо кожен з параметрів більш детально. У модулі `datetime`, значення дати та часу представлені у вигляді окремих об'єктів. Давайте подивимося на деякі з них:
- `datetime.date()` - об'єкт, що містить лише дату (без часу);
- `datetime.time()` - об'єкт, що містить лише час (без дати);
- `datetime.datetime()` - об'єкт, що містить дату та час.

Крім того, для кожного з цих об'єктів є різні методи, які дозволяють змінювати значення часу та дати, отримувати день тижня та інше.
