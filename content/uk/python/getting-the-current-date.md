# Так, як отримати поточну дату в Python

Python - це мова програмування, яка надає безліч корисних функцій і можливостей. Однією з них є отримання поточної дати, що може бути корисним для багатьох причин. У цьому блозі ми розглянемо, як отримати поточну дату використовуючи Python, а також дослідимо деякі нюанси цієї функції.

## Чому

Отримання поточної дати може бути корисною функцією для багатьох задач. Наприклад, для відстеження часу виконання різних дій в програмі або для фіксування дати створення файлу. Без цієї функції було б дуже важко визначити поточну дату з точністю до хвилини.

## Як отримати поточну дату в Python

Отримати поточну дату в Python дуже просто. Спочатку нам потрібно імпортувати модуль `datetime` і визначити змінну, в якій буде зберігатися поточна дата. Ось приклад коду:

```Python
import datetime

current_date = datetime.datetime.now()
print(current_date)
```

Результат буде виглядати приблизно так:

`2021-08-15 16:05:17.920653`

Можна також отримати лише поточну дату без часу, використовуючи метод `date()`:

```Python
import datetime

current_date = datetime.datetime.now().date()
print(current_date)
```

Результат буде виглядати приблизно так:

`2021-08-15` 

Можна використовувати різні методи для форматування отриманої дати в зручний для нас вигляд. Наприклад, можна вивести дату в форматі дд-мм-рррр:

```Python
import datetime

current_date = datetime.datetime.now().strftime("%d-%m-%Y")
print(current_date)
```

Результат буде виглядати так:

`15-08-2021`

Щоб отримати більше інформації про можливі методи форматування дати, можна переглянути [офіційну документацію `datetime`](https://docs.python.org/3/library/datetime.html).

## Глибше дослідження

Найбільш важливе для зрозуміння в роботі з поточною датою - це формати дат і часу в Python. Основні формати, які використовуються для представлення дат, це:

- `%y` - дві останні цифри року
- `%Y` - повний рік 
- `%m` - місяць (01-12)
- `%d` - день (01-31)
- `%H` - година в 24-годинному форматі (00-23)
- `%I` - година в 12-годинному форматі (01-12)
- `%M` - хвилина (00-59)
- `%S` - с
