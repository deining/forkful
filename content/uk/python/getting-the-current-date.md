## Що і чому?

Отримання поточної дати - це важлива задача для багатьох програмістів. Поточна дата може бути необхідною для виконання різних операцій, таких як створення звітів або збереження даних. Також вона може бути потрібна для виведення на екран або для встановлення часової мітки для подальшого використання.

## Як зробити:

Для отримання поточної дати використовується функція ```Python datetime.now() ``` . Ця функція повертає об'єкт дати і часу, який можна зберегти у змінній для подальшого використання. Наприклад:
```
import datetime
current_date = datetime.now()
print(current_date)
```
Результат виводу буде подібний до ```2021-09-28 15:30:00.542089```, де перші 10 символів показують дату в форматі рррр-мм-дд, а наступні - час у форматі гг:хх:сс. 

## Поглиблене дослідження:

Функція ```datetime.now()``` була вперше введена у Python 2.4 і дозволяє отримати поточну дату на основі системного годинника.  Також існують альтернативні способи отримання поточної дати, наприклад, за допомогою модуля ```time``` або сторонніх бібліотек, таких як ```dateutil```. Крім того, поміж різними операційними системами можуть бути відмінності у форматі виведення поточної дати.
