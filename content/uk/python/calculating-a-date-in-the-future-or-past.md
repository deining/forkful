## Чому

Подорожування в часі є важливою частиною розробки програмного забезпечення та аналізу даних. Для тих, хто працює з даними або виконує розрахункові завдання, виявлення дати в майбутньому або минулому може бути корисним для прогнозування або тестування. Використання Python для здійснення цих розрахунків дозволяє легко та швидко отримати бажану інформацію.

## Як

Щоб знайти дату в майбутньому або минулому, використовуйте модуль datetime у Python. Для початку, включіть цей модуль у своєму коді за допомогою `import datetime`. Далі, створіть змінну для поточної дати за допомогою `today = datetime.datetime.today()`. Її можна використовувати як початкову точку для визначення бажаної дати.

Наприклад, якщо ми хочемо порахувати дату через 100 днів, ми можемо використовувати наступний код:

```Python
import datetime

today = datetime.datetime.today()
future_date = today + datetime.timedelta(days=100)

print(future_date)
```

Це виведе наступний результат:

```
2022-01-31 00:00:00.000000
```

Аналогічно, для пошуку дати в минулому, просто використовуйте від'ємне значення для `timedelta`. Наприклад, щоб знайти дату 50 днів тому, використовуйте `datetime.timedelta(days=-50)`.

## Поглиблене дослідження

Модуль datetime також має багато інших корисних функцій, які можна використовувати для роботи з датами та часом. Наприклад, `datetime.date` дозволяє робити операції тільки з датами (без часу), а `datetime.time` - тільки з часом (без дати).

Крім того, в модулі datetime є можливості для роботи з часовими зонами, виведення дати у різних форматах та багато інших. Рекомендую детальніше ознайомитися з документацією модулю, щоб дізнатися більше корисних функцій.
