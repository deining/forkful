## Чому
Дати - це важливі для людей. Це наш спосіб визначати час і відстежувати події. Вміти обчислювати дати в майбутньому або минулому може бути корисним для планування або аналізу подій.

## Як це зробити
Для цього ми можемо використати зручну функцію `datetime` з модуля `datetime`. Спочатку ми потрібно імпортувати цей модуль у наш код:
```Python
import datetime
```

Далі, ми можемо використати функцію `today()` з `datetime` для отримання поточної дати:
```Python
now = datetime.datetime.today()
print(now)
```
Вивід: `2021-10-20 18:30:00.125374`

Щоб обчислити дату у майбутньому, ми можемо додати `timedelta` до поточної дати і вказати, на який період часу ми хочемо відступитися:
```Python
future_date = now + datetime.timedelta(days=30)
print(future_date)
```
Вивід: `2021-11-19 18:30:00.125374`

Або, якщо нам потрібно обчислити дату у минулому, ми можемо використати від'ємне значення для `days`:
```Python
past_date = now - datetime.timedelta(days=30)
print(past_date)
```
Вивід: `2021-09-21 18:30:00.125374`

Це лише декілька прикладів використання функцій `datetime` для обчислення дат. Існує багато інших можливостей, таких як використання різних атрибутів для детальнішого визначення часу, або знаходження різниці між двома датами. Для докладнішої інформації про можливості цього модуля, рекомендуємо переглянути офіційну документацію.

## Deep Dive
Функція `today()` повертає поточну дату з часом, але тільки в об'єкті типу `datetime` з модуля `datetime`. Якщо нам потрібно вивести дату у зручному для нас форматі, ми можемо використовувати метод `strftime()` для перетворення дати у рядок. Наприклад, ми можемо вивести дату у форматі "день-місяць-рік":
```Python
print(now.strftime("%d-%m-%Y"))
```
Вивід: `20-10-2021`

Також можна використовувати `strftime()` для перетворення рядка у об'єкт типу `datetime`. Для цього потрібно передати рядок та форматування у метод `strptime()`:
```Python
date_string = "20/10/2021"
converted_date = datetime.datetime.strptime(date_string, "%d/%m/%Y")
print(converted_date)
```
Вивід: `2021-10-20 00:00:00`
