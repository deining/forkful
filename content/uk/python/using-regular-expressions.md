## Зачем

Українські читачі, ви мабуть питаєте себе: "Навіщо мені взагалі потрібні регулярні вирази в програмуванні?" Відповідь проста: це потужний інструмент для пошуку та заміни тексу в рядках. Використовуючи регулярні вирази, ви можете багато ефективніше та швидше робити операції з рядками, що значно полегшить вашу роботу.

## Як це зробити

Для початку, ми повинні встановити модуль `re` (регулярні вирази) в Python. Після цього, ми можемо використовувати функції модуля для пошуку та заміни тексту.

```Python
import re
```

Давайте спробуємо знайти всі слова у рядку, що закінчуються на букву "а":

```Python
text = "Україна, мама, собака"
matches = re.findall(r"\w+а", text)
print(matches)
```

Виведення:

```
['Україна', 'мама', 'собака']
```

Тут ми використовували функцію `findall`, яка знаходить всі збіги нашого регулярного виразу (`\w+а`) у заданому тексті. Зверніть увагу на символ `r` перед регулярним виразом – це означає, що ми використовуємо "сирі" (raw) рядки, що допомагає уникнути проблем зі спеціальними символами. Також, можна використовувати функцію `search`, яка знаходить перший збіг у рядку та повертає `Match` об'єкт, або функцію `sub`, яка заміняє всі збіги на певний текст.
