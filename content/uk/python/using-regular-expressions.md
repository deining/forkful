## Що і чому?

Використання регулярних виразів (regular expressions) в програмуванні - це спосіб обробки тексту швидко та ефективно. Регулярні вирази - це міні-мова, яка дозволяє здійснювати пошук та заміну певних шаблонів у тексті. Програмісти використовують регулярні вирази для ефективної обробки текстових даних, таких як логи, HTML-сторінки, електронні листи тощо.

## Як?

Використання регулярних виразів в Python дуже просте за допомогою вбудованої бібліотеки `re`. Давайте розглянемо приклад використання регулярних виразів для знаходження та вилучення номерів телефонів зі строкового значення:

```python
import re

# Строка з номерами телефонів
text = "Мій номер телефону: (123) 456-7890"

# Використання регулярного виразу для пошуку номерів телефонів
phone_pattern = r"\(\d{3}\) \d{3}-\d{4}"
results = re.findall(phone_pattern, text)

# Виведення результатів
print(results) # Виведе: ['(123) 456-7890']
```

Як бачите, з використанням регулярних виразів ми знайшли та вилучили номер телефону з тексту за допомогою дуже простих шаблонів. Також, за допомогою регулярних виразів можна здійснювати складніші операції, такі як пошук та заміна, перевірка наявності певних слів та багато іншого.
