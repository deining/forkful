## Чому

Регулярні вирази є потужним інструментом в програмуванні Python, який дозволяє шукати та змінювати текст за допомогою шаблонів. Вони можуть допомогти скоротити час і зусилля, які витрачаються на маніпулювання текстом в програмах.

## Як

Для використання регулярних виразів у Python необхідно спочатку імпортувати модуль ```re```. Основні функції, які вам знадобляться, це ```findall()```, ```search()``` і ```sub()```. Перша функція дозволяє знайти всі співпадіння з шаблоном у тексті і повертає список знайдених значень. Друга функція поверне об'єкт Match, який можна використовувати для отримання різних даних про знайдене співпадіння. Третя функція дозволяє замінити співпадіння з шаблоном на інший рядок.

Наприклад, якщо ми хочемо знайти всі слова, що починаються з літери "а" у реченні "Абрикоси та яблука смачні", ми можемо використати такий код:

``` Python
import re

text = "Абрикоси та яблука смачні"
matches = re.findall(r'\bа\w+', text)
print(matches)
```

Цей код поверне список ["Абрикоси", "яблука"].

## Глибоке погруження

Для того щоб розуміти і впевнено використовувати регулярні вирази, необхідно розібратися у всіх доступних спеціальних символах і практикуватися в їх застосуванні. Надалі, можна вивчити інші функції ```re```, які дозволяють більш точно знаходити шаблони, такі як ```split()```, ```match()``` та інші.
