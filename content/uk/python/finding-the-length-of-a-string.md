## Що і чому?
Пошук довжини рядка - це процес визначення кількості символів у рядку. Програмісти роблять це для того, щоб мати доступ до кожного символу у рядку та виконувати необхідні дії з цими символами.

## Як це зробити:
```
Python len () Функція використовується для знаходження довжини рядка. Наприклад:
```python
example_string = "Привіт, світ!"
print (len (example_string))
```

Результат: 12

## Глибокий занурений:
1. Історичний контекст:
Довжина рядка була впродовж довгого часу важливою операцією для обробки тексту в комп'ютерних програмах. Раніше, коли пам'ять була обмежена, важливо було ефективно використовувати довжину рядка для збереження ресурсів.
2. Альтернативи:
Одна з альтернатив для визначення довжини рядка - це використання циклу для знаходження кількості символів у рядку. Однак, це може бути більш часомістким та менш ефективним підходом.
3. Деталі реалізації:
Функція len () використовує ітерируемий об'єкт, такий як рядок, та повертає його довжину. Вона також може бути використана для знаходження довжини інших типів об'єктів, які підтримують ітерацію, такі як списки та кортежі.
