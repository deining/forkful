## З чого?

Знаходження довжини рядка є важливою та поширеною задачею у програмуванні. Вона допомагає визначити кількість символів у текстовому рядку і дозволяє робити різні маніпуляції з ним.

## Як це зробити?

Для знаходження довжини рядка використовується функція `len()`, яка приймає рядок як аргумент і повертає його довжину.

```Python
text = "Привіт, світ!"
length = len(text)
print(length)
```

Вивід: 13

Також можна використовувати цикл `for` для перебору кожного символу рядка і використання лічильника для підрахунку довжини.

```Python
text = "Привіт, світ!"
count = 0
for char in text:
    count += 1

print(count)
```

Вивід: 13

## Занурення

Знаходження довжини рядка може бути корисним у багатьох різних сценаріях програмування. Наприклад, можна використовувати його для перевірки, чи введене користувачем слово або фраза є доволі довгим, щоб бути прийнятним для обробки. Також ця функція може допомогти визначити кількість символів у файлі або документі.
