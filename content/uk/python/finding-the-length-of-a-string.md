##Чому
Найдіть довжину рядка є корисною навичкою в програмуванні, оскільки вона дозволяє вам працювати з текстовими даними та здійснювати різні операції з ними.

##Як
Використовуйте вбудовану функцію `len()` в Python, щоб знайти довжину рядка. Ось приклад коду та його вихідний результат:

```Python
my_string = "Привіт світ!"
print(len(my_string))
```
Вихід: 12

Також, ви можете використовувати цей метод для знаходження довжини будь-якого об'єкта у Python, такого як список чи кортеж.

```Python
my_list = [1, 2, 3, 4, 5]
print(len(my_list))
```
Вихід: 5

##Глибокий погляд
У Python довжина рядка визначається як кількість символів, що містяться в ньому. Це включає всі букви, цифри, символи пунктуації та пробіли. Також важливо пам'ятати, що індекси символів у рядку починаються з нуля, тому останній індекс буде на одиницю менше, ніж довжина рядка.

Наприклад, у рядку `"Привіт"`, довжина буде 5, але останній індекс буде 4. Тому, якщо ви хочете отримати останній символ у цьому рядку, ви можете скористатися індексом `4`, тобто `my_string[4]`.
