## Чому
Розуміння того, як знайти довжину рядка, є важливим аспектом програмування Python. Це дозволяє нам легко маніпулювати та обробляти дані в наших програмах. Незалежно від того, чи ви новачок у програмуванні, чи досвідчений розробник, знання цього основного концепту допоможе вам у побудові ефективних програм.

## Як
Насправді, знаходження довжини рядка в Python дуже просте. Наприклад, ми можемо використати вбудовану функцію `len()` для отримання довжини будь-якого рядка. Давайте подивимося на приклад:

```Python
my_string = "Привіт, Python"
print(len(my_string))
```

Цей код виведе `13`, оскільки рядок містить 13 символів. Також важливо зазначити, що ця функція працює не тільки зі стрічками, але й з будь-якими ітерованими об'єктами (наприклад, списками, кортежами чи словниками).

Крім того, ми також можемо використати цикл `for` для розрахунку довжини рядка. У такому випадку ми будемо перебирати кожен символ у рядку та збільшувати лічильник на 1 кожен раз, коли зустрічаємо новий символ. Наприклад:

```Python
my_string = "Привіт, Python"
count = 0

for char in my_string:
    count += 1

print(count)
```

Цей код також виведе `13`. Такий підхід може бути корисним, наприклад, якщо нам потрібно виконати якісь додаткові дії з кожним символом у рядку.

## Глибше копаємо
Якщо зазирнути під капот, то дізнаємося, що усе це можливо завдяки тому, що можна ітерувати по рядку в Python. Тобто, Python розглядає рядок як послідовність символів і може перебирати їх за допомогою циклів.

Крім того, більшість методів для рядків, таких як `strip()`, `upper()` чи `lower()`, теж використовують цю функціональність ітерування для роботи з рядками.
