## Причина

Якщо ви часто завдовжки користуєтесь Python, то вам, напевно, доводилося працювати зі стрічками. Для ефективної роботи з ними необхідно знати, як обчислити довжину стрічки. Це важливий елемент, який допоможе вам зробити ваш код більш гнучким та зрозумілим.

## Як це зробити

Існує простий спосіб обчислити довжину стрічки в Python за допомогою вбудованої функції `len()`. Подивіться на наступний приклад коду:

```Python
name = "Петро"
print(len(name))

# Вивід: 5
```

У цьому прикладі ми створили змінну `name` і призначили їй значення стрічки "Петро". Потім ми використали функцію `len()` для обчислення довжини цієї стрічки. І, як бачимо, вона складає 5 символів.

Також важливо зазначити, що функція `len()` працює не тільки зі стрічками, але й з багатьма іншими типами даних, включаючи списки, кортежі та словники.

## Глибше в деталі

Функція `len()` спрацьовує шляхом обрахунку кількості елементів у контейнері. Для стрічок це означає кількість символів, а для списків, кортежів та словників - кількість елементів, які вони містять.

Варто також зауважити, що функція `len()` приймає аргумент `obj`, який є обов'язковим. Цей аргумент може бути будь-яким об'єктом, для якого можна обчислити довжину. Інакше ви отримаєте помилку.

Наприклад, якщо ви спробуєте використати функцію `len()` для числа, отримаєте помилку:

```Python
number = 123
print(len(number))

# Вивід: TypeError: object of type 'int' has no len()
```

Отже, важливо використовувати функцію `len()` тільки для об'єктів, для яких вона призначена.
