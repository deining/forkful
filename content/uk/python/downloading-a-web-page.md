## Що і З чого? 

Завантаження веб-сторінки - це процес отримання вмісту сторінки з Інтернету на ваш комп'ютер. Програмісти часто роблять це для отримання інформації з веб-сайтів та використання її у своїх програмах. Це може бути корисно для автоматизації завдань або для отримання актуальної інформації з Інтернету.

## Як зробити:

```Python 
import requests
url = 'https://www.google.com/' 
response = requests.get(url)
print(response.text)
```

Рядок `import requests` підключає бібліотеку `requests`, яка дозволяє нам виконувати запити до Інтернету. Потім ми визначаємо змінну `url`, яка містить URL-адресу веб-сторінки, яку ми хочемо завантажити. Далі ми використовуємо функцію `get()` для отримання вмісту цієї сторінки та зберігаємо результат у змінній `response`. Нарешті, використовуючи метод `text`, ми можемо вивести зміст сторінки у терміналі за допомогою функції `print()`.

## Глибокий погляд:

Завантаження веб-сторінок стало необхідною частиною роботи багатьох програмістів. У минулому, для цього зазвичай використовувалися словники HTML та складні регулярні вирази. Однак, з появою бібліотеки `requests` виконання цієї задачі стало набагато простіше. Альтернативою для `requests` є бібліотека `urllib`, але вона часто вимагає більше коду для досягнення того ж результату.

Щоб більш детально дослідити можливості бібліотеки `requests`, рекомендуємо ознайомитись з документацією на офіційному сайті. Також ви можете пошукати додаткові істочники, які допоможуть вам освоїти цю тему.
