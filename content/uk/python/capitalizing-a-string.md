## Чому

Написання програм на мові Python може здатися складною та займати багато часу. Але іноді навіть найпростіші завдання можуть вимагати певних технічних знань. Наприклад, нещодавно мені було потрібно привести рядок до верхнього регістру. На перший погляд, це здається незначним завданням, але виявилося, що для цього потрібно використовувати спеціальну функцію у Python. У цій статті я розповім про те, як зробити це всього лише за кілька рядків коду.

## Як

```Python
# Вхідні дані
s = "привіт, питон"

# Використовуємо функцію capitalize()
new_s = s.capitalize()

# Виведення результату
print(new_s)

# Результат
Привіт, питон
```

Як бачите, для того, щоб привести рядок до верхнього регістру, потрібно всього лише використати вбудовану функцію `capitalize()`. Ця функція повертає новий рядок з першою буквою у верхньому регістрі, а всі інші букви залишаються незмінними. 

## Глибоке дослідження

Функція `capitalize()` повертає рядок з першою буквою у верхньому регістрі, але що буде, якщо рядок вже містить слова з верхнім регістром?

```Python
# Вхідні дані
s = "Привіт, Питон"

# Використовуємо функцію capitalize()
new_s = s.capitalize()

# Виведення результату
print(new_s)

# Результат
Привіт, питон
```

Як бачимо, функція `capitalize()` не перетворила інші слова у рядку на верхній регістр, вона лише змінила першу букву. Це важливо враховувати при використанні цієї функції.
