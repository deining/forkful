## Чому
Ви, як програміст, можливо зазнаєте дедаху на рутинну задачу: переконатися, що всі символи в рядку мають нижній регістр. І хоч вона може здатися простою, ця задача досить важлива для багатьох програм, що працюють зі вводом користувача або перевіркою даних.

## Як це зробити
```
string = "Це рядок з Великою Літерою"
lowercase_string = string.lower()
print(lowercase_string)
```

В результаті ви отримаєте вивід "це рядок з великою літерою". Просто, чи не так?

## Глибиная змілина
Конвертування рядка до нижнього регістру може дати нам більше контролю над нашим програмним кодом. Наприклад, при роботі зі списками або словниками, ми можемо застосувати цей метод для кожного елементу та отримати уніфікований вивід. Крім того, це також може бути корисно при порівнянні двох рядків для забезпечення відповідності.
