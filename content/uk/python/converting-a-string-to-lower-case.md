## Чому

Для чого потрібно перетворювати рядки в нижній регістр? Існує багато випадків, коли це корисно, але основною причиною є зручність. Коли рядки усією величиною з однаковими символами, легше порівнювати та обробляти їх.

## Як це зробити

Існує кілька шляхів перетворення рядків на нижній регістр у Python. Один з них - використання методу `lower()`:

```Python
sentence = "ЦЕ РЯДОК З ВЕЛИКИМИ ЛІТЕРАМИ"
lower_sentence = sentence.lower()
print(lower_sentence)
```

Вихід: `це рядок з великими літерами`

Ми також можемо застосовувати цю функцію до конкретних символів або підрядків у рядку:

```Python
word = "Об'єктно-орієнтоване програмування"
small_word = word[10:].lower()
print(small_word)
```

Вихід: `програмування`

Нарешті, ми можемо використовувати регулярні вирази з модулем `re` для перетворення всього рядка:

```Python
import re
text = "Регулярні вирази в Python"
lowercase_text = re.sub("[А-ЯЁ]", "\l&", text)
print(lowercase_text)
```

Вихід: `регулярні вирази в python`

## Глибшого погляду

Конвертування рядка в нижній регістр є частиною вбудованого механізму Python - методу `str.lower()`. Цей метод використовує таблицю перекодування символів для виконання операції. Однак, слід зазначити, що таблиці перекодування можуть бути різними залежно від версії Python та операційної системи. Тому результати перетворення можуть змінюватися, якщо ви переходите на іншу версію Python або робите це на різних ОС.
