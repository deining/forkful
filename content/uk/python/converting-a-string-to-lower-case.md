## Чому

Конвертація рядка в нижній регістр є важливою частиною багатьох програм. Використовується для зроблення рядка більш зручним для обробки, пошуку та порівняння з іншими рядками.

## Як зробити

```python
string = "Україна"
print(string.lower())
```

Вивід:
```python
україна
```

Цей код показує простий приклад того, як використовувати метод `lower()` для конвертації рядка в нижній регістр. Цей метод працює для будь-якої рядкової змінної та повертає нову змінну з нижнім регістром.

## Глибше розуміння

Існують різні способи конвертації рядка в нижній регістр. Один з них - використання побітових операцій, таких як `OR` та `AND`, для отримання відповідних символів з ASCII таблиці. Цей підхід частіше використовується в більш старих версіях Python. 

Найбільш оптимальним та простим методом є використання вбудованого методу `lower()`. Він автоматично конвертує всі літери в нижній регістр, без потреби в складних операціях. Також важливо пам'ятати, що при роботі зі строками з кирилицею, метод `lower()` не буде працювати коректно і необхідно використовувати спеціальні бібліотеки, такі як `unidecode` для правильної конвертації.
