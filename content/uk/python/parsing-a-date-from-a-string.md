Що і чому?

Розбір дати з рядка - це процес, який дозволяє програмістам отримувати дату з рядка, що зберігається в певному форматі. Це корисно, коли ми отримуємо дані з зовнішніх носіїв, таких як файли або бази даних, де дата може бути представлена у вигляді тексту.

Як це зробити?

```Python 
import datetime
def parse_date_from_string(date_string):
    return datetime.datetime.strptime(date_string, '%d-%m-%Y')
print(parse_date_from_string('01-01-2021'))
```

Вихід: `2021-01-01 00:00:00`

Вивчення глибше

Історичний контекст: Розбір дати з рядка широко використовувався на початку розвитку комп'ютерних систем для перетворення дати у зручний формат для обробки та зберігання.

Альтернативи: Інші способи отримання дати з рядка можуть включати використання регулярних виразів або власноруч написання функції для перетворення.

Деталі реалізації: У прикладі ми використали стандартний модуль ```datetime``` для розбору дати з рядка. Параметр ```'%d-%m-%Y'``` вказує формат рядка, в якому дата представлена. Більше про форматування дивіться в документації.
