## чому

Іноді вам може знадобитися виокремити частину рядка, зокрема підрядок (substring), щоб обробити або використати його окремо від інших даних. Наприклад, ви можете використовувати виокремлені підрядки для пошуку певних текстових шаблонів або для зміни формату даних. У пайтоні є кілька способів виокремлення підрядків, які допоможуть вам досягти цієї мети.

## Як

Найпростіший спосіб виокремлення підрядків - це використання оператора зрізу (slice operator). Він дозволяє вам вибрати ділянку рядка за певними індексами. Наприклад, якщо ви хочете виокремити підрядок "world" з рядка "Hello world", ви можете використати наступний код:

```Python
text = "Hello world"
substring = text[6:11]
print(substring)
```
Ви отримаєте вивід "world".

Також ви можете використати вбудований метод `find()` для пошуку певної підрядкової послідовності в рядку. Цей метод повертає індекс першого входження підрядка у рядку. Наприклад, якщо ви хочете знайти індекс слова "python" у рядку "I love coding in python", ви можете зробити так:

```Python
text = "I love coding in python"
substring_index = text.find("python")
print(substring_index)
```
Ви отримаєте вивід 17.

Ще одним способом виокремлення підрядків є використання регулярних виразів за допомогою модуля `re`. Він дозволяє вам шукати певні текстові шаблони та виокремлювати з них підрядки. Наприклад, якщо ви хочете виокремити всі числа з рядка "Today is December 31st", ви можете використати наступний код:

```Python
import re
text = "Today is December 31st"
numbers = re.findall("\d+", text)
print(numbers)
```
Ви отримаєте вивід ['31'].

## Глибокий погляд

Виокремлення підрядків є важливою операцією в роботі з рядками в Python. Ви можете використовувати її для різноманітних завдань, від простого пошуку і заміни певних цифр у тексті до складніших операцій обробки даних. Крім того, ви можете поєднувати різні методи та інструменти для досягнення необхідного результату.
