# Причина

Екстрагування підрядків (substrings) є одною з найбільш поширених операцій у програмуванні. Це корисний інструмент для вирішення багатьох завдань, зокрема, пошуку певної інформації у тексті або обробки рядків даних.

# Як це зробити

Існує кілька шляхів екстрагування підрядків у Python. Давайте розглянемо декілька прикладів з використанням методу `slice` та функції `find`.

## За допомогою методу `slice`

```Python
my_string = "Привіт, це підстрока!"
substring = my_string[7:17]
print(substring) # виводить "це підстрока"
```

У цьому прикладі ми використовуємо `slice` для вибору підрядка з позиції 7 до 17. Важливо зазначити, що позиція 17 не включається у вибірку, тому отримуємо підрядок "це підстрока".

## За допомогою функції `find`

```Python
my_string = "Сьогодні чудовий день!"
position = my_string.find("чудовий")
print(position) # виводить 9
```

У цьому прикладі ми використовуємо функцію `find` для знаходження позиції, на якій розташований заданий підрядок. У нашому випадку, функція повертає значення 9, що відповідає позиції слова "чудовий" у рядку.

# Заглиблення

В Python існує багато інших способів екстрагування підрядків, таких як використання регулярних виразів або методу `split`. Також варто зазначити, що індекси позицій у Python починаються з нуля, тому вартість `slice` позиції не включається у вибірку.
