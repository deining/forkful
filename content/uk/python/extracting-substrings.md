Що це і чому? 

Витягнення підрядків (також відоме як підстрока) - це процес отримання частини рядка тексту або його символів. Це корисний інструмент для програмістів, оскільки дозволяє працювати з окремими частинами тексту, що збільшує ефективність і гнучкість програм.

Як це зробити: 

```Python
text = "Hello World"
print(text[0:5]) # Виводить "Hello"
print(text[6:]) # Виводить "World"
```
В цьому прикладі ми використовуємо [індексування](https://www.tutorialspoint.com/python/string_index.htm) для вибору частини рядка за допомогою дужок та вказуємо початкову та кінцеву позиції символів, які хочемо отримати.

Залишилося багато варіантів для використання витягнення підрядків, наприклад, ви може додавати та змінювати символи, використовувати умови для вибору підрядків або навіть розбивати їх на дрібніше частини.

В глибину: 

Використання підрядків в програмуванні не нове. Вперше його згадують у мові Fortran у 1960-х роках, коли програмісти використовували це для роботи з числовими даними. На сьогоднішній день у більшості мов програмування підтримують витягнення підрядків.

Існують інші способи отримання підрядків, наприклад, використання функцій для розбиття рядка на частини або використання регулярних виразів для визначення патернів символів. Різноманітність цих методів дозволяє програмістам вибрати найбільш підходящий для своїх потреб.

Також важливо враховувати, що рядки у багатьох мовах програмування є [immutable](https://www.tutorialspoint.com/immutable-objects-in-python), тобто не ми змінюємо їх, а створюємо новий рядок з новими значеннями.
