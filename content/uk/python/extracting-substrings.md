## З чого почати

Якщо ви програміст або просто цікавитесь програмуванням, ви знайомі з використанням різних методів обробки тексту. Один із таких методів - це вилучення підрядків або підстрок з рядка. Це дуже корисний інструмент, який дозволяє отримати певну частину тексту, яка задовольняє певні умови.

## Як вилучати підрядки в Python

Для вилучення підрядків ми будемо використовувати методи, які надає Python. Ось приклад вилучення підрядка з рядка:

```Python
# Заданий рядок
text = "Привіт, це Python блог!"

# Вилучаємо підрядок від першого символу до 12-го
substring = text[0:12]

# Виводимо результат
print(substring)

# Виведе: Привіт, це Python
```

У цьому прикладі ми використали синтаксис `[start: end]`, щоб вказати, яку частину рядка ми хочемо отримати. Важливо, що індекси символів у Python починаються з нуля.

Також ми можемо використовувати негативні індекси, щоб вказати, яку частину рядка ми хочемо вилучити з кінця. Наприклад, `text[-12:]` вилучить підрядок з 12 повернутих символів з кінця рядка.

Є інші способи вилучення підрядків з рядків, такі як використання методів `find()` і `split()`. Якщо ви хочете дізнатися більше про ці методи, перегляньте наш Deep Dive розділ нижче.

Наприклад, метод `find()` може повертати індекс першого входження підрядка, але він також має можливість вказувати початкову та кінцеву позиції для пошуку. Також важливо знати, що рядки в Python є не змінними об'єктами, тому краще створювати новий рядок з використанням вилученого підрядка, а не змінювати початковий.
