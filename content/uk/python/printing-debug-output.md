## Why

Друкування вихідних даних для налагодження коду є важливою частиною процесу розробки програмного забезпечення. Це дозволяє досліджувати значення змінних та відстежувати проблемні місця у програмі.

## How To

Існують кілька способів друкування вихідних даних у програмі Python. Один з них - використання функції `print()`. Наприклад, якщо ми хочемо друкувати значення змінної `x`, ми можемо написати:

```python
x = 5
print(x)
```
Ви зможете побачити шлях до файлу, де ви запустили програму, і значення змінної `x` буде написано під ним. Можна додавати більше змінних і виводити їх значення разом, використовуючи кому:

```python
x = 5
y = 10
print(x, y)
```
В результаті ми отримаємо:

```
5 10
```


## Deep Dive

Крім використання функції `print()`, також можна використовувати модуль `logging` для друкування вихідних даних. Це дозволяє керувати рівнем налагодження і дозволяє легко вимикати друк вихідних даних, коли програма переходить до фінального етапу. Наприклад, якщо ми хочемо друкувати значення змінної `x` з повідомленням про те, що це значення додається до іншого числа:

```python
x = 5
import logging
logging.basicConfig(level=logging.DEBUG)
logging.debug(f"The value of x is {x}.")
```
Результатом буде запис у журналі дебагу:

```
DEBUG:root:The value of x is 5.
```

Також можна використовувати `logging` для друкування вихідних даних під час виконання умовних операцій або циклів, що допомагає відстежувати, як програма переходить через різні частини коду.
