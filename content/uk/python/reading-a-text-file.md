## Чому
Для чого потрібно читати текстовий файл? Перш за все, це дає можливість отримати доступ до інформації, збереженої в цьому файлі. Також це може бути корисно при обробці даних та аналізі.

## Як це зробити
Python має вбудовану функцію для читання текстових файлів - `open()`. Її синтаксис виглядає наступним чином:

```
file = open('text_file.txt', 'r')
```

Другим аргументом є режим відкриття файлу, який може бути `r` для читання, `w` для запису чи `a` для допису до файлу. Якщо файл знаходиться в різній папці, потрібно вказати повний шлях до нього. Після цього, ви можете прочитати файл за допомогою методу `read()`:

```
text = file.read()
print(text)
```

Це виведе вміст файлу на екран. Також можливо прочитати файл построчно за допомогою методу `readline()`:

```
line = file.readline()
print(line)
```

Це корисно, якщо ви бажаєте обробити файл по одному рядку за раз. Якщо ж вам потрібно прочитати усі рядки, ви можете використати цикл `for`:

```
for line in file:
    print(line)
```

Це надрукує усі рядки у файлі один за одним. Після закінчення роботи з файлом, не забудьте закрити його за допомогою методу `close()`:

```
file.close()
```

Також можливо використати блок `with` для автоматичного закриття файлу після завершення роботи з ним:

```
with open('text_file.txt', 'r') as file:
    text = file.read()
    print(text)
```

Це забезпечить, що файл буде закритий навіть у випадку помилки у вашому коді.

## Глибший дослід
Загрузка та читання великих текстових файлів може бути повільною та надто ресурсоємкою. Також важливо враховувати відмінності у кодуванні файлів, які можуть спричинити проблеми під час читання. Чудовою альтернативою може бути використання бібліотеки `pandas`, яка має швидкі та ефективні методи для роботи з текстовими даними.
