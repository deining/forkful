## Чому

Часто програмісти потребують прочитати великий текстовий файл, щоб отримати інформацію або зробити аналіз даних. Це може бути текстовий документ, таблиця чи інший формат даних. При цьому, використання певних засобів та знання основ програмування може значно полегшити цей процес.

## Як це зробити

Для початку, потрібно відкрити файл у програмі Python за допомогою вбудованої функції `open()`. Ця функція приймає два аргументи - назву файлу та режим, в якому він повинен бути відкритий (наприклад, `r` для читання або `w` для запису).

```python
file = open("text.txt", "r") # відкриваємо файл для читання
print(file.read()) # виводимо зміст файлу на екран
file.close() # закриваємо файл
```

У цьому прикладі ми використали метод `.read()` для прочитання всього змісту файлу. Також можна використовувати методи `.readline()` або `.readlines()` для читання по одному рядку або отримання списку рядків з файлу відповідно.

Після того, як закінчили роботу з файлом, важливо закрити його за допомогою методу `.close()`, щоб не залишати відкриті з'єднання на комп'ютері.

## Глибока занурення

Крім звичайного читання змісту файлу, в програмі Python є багато різних властивостей та методів для роботи з текстовими файлами. Наприклад, можна задавати кодування файлу за допомогою параметру `encoding` при відкритті файлу, або використовувати контекстний менеджер `with` для автоматичного закриття файлу після завершення операцій з ним.

Також, можливо використовувати цикли та умовні оператори для роботи з рядками текстового файлу та фільтрування даних.
