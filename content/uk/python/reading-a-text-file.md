Що і чому?
Читання текстового файлу - це процес, за допомогою якого програмісти отримують доступ до вмісту файлу і працюють з ним у своїй програмі. Це може бути корисно для отримання вхідних даних для обробки або для запису результатів програми.

Як це зробити:
```Python
with open('file.txt', 'r') as f: # відкриття файлу для читання
    data = f.read() # зчитування вмісту файлу
    print(data) # виведення зчитаного тексту

# Вихідний файл:
# Це текстовий файл для прикладу.
# Він містить деякі дані, які ми хочемо прочитати.
```

Глибше вникнення:
Читання текстових файлів існує вже давно, і є одним із базових методів роботи з файлами. Альтернативою може бути використання модуля ```csv``` для читання файлів з табличною структурою. Детальніше про використання цього модуля можна дізнатися за посиланням у розділі "Дивіться також".
