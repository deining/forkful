## Чому

Якщо ви цікавитеся програмуванням на Python, ви напевно знаєте, що читання текстових файлів є необхідною частиною багатьох проектів. Це дозволяє отримувати дані для подальшої обробки або зберігати результати програми. Тому вивчення цього процесу є важливим для будь-якого начинаючого програміста.

## Як зробити

Для читання текстових файлів у Python існує декілька способів. Найпростіший з них - використання вбудованих функцій open () та read (). Наприклад, якщо у вас є файл з назвою "text.txt", ви можете виконати наступний код:

```Python
file = open("text.txt", "r")
text = file.read()
print(text)
```
Результатом буде виведення всього тексту з файлу на екран.

## Занурення в деталі

Крім використання функцій open () та read (), існує ще багато інших способів читання текстових файлів у Python. Ви можете використовувати цикли для ітерації по кожному рядку у файлі, а також використовувати різноманітні методи для обробки даних. Необхідно також пам'ятати про правильне закриття файлу після завершення читання.
