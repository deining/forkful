## Чому

Відкриття і читання файлів тексту є важливою складовою частиною програмування Python. При роботі з файлами, їх дані можна зберігати, модифікувати та обробляти за допомогою програмного коду. Це дозволяє програмістам працювати з великою кількістю даних та створювати потужні програми.

## Як

Для відкриття та читання файлу тексту у Python, використовується функція `open()`. Вона приймає два аргументи - назву файлу та режим відкриття. Нижче наведений приклад коду, який відкриває файл "test.txt" у режимі читання та виводить його вміст:

```Python
file = open("test.txt", "r")
print(file.read())
```

Виходячи з прикладу вище, можна помітити, що перед використанням файлу, спочатку потрібно його відкрити за допомогою функції `open()`, а після використання - закрити за допомогою функції `close()`. Це необхідно для того, щоб уникнути зміни вмісту файлу, та вивести його коректно.

За допомогою режимів відкриття, можна читати файли та здійснювати з ними різноманітні дії, наприклад, додавати або перезаписувати дані.

## Deep Dive

Окрім базового читання файлу, у Python є багато різноманітних інструментів та функцій для роботи з ним. Наприклад, функція `readline()` дозволяє читати файл по рядках, а метод `readlines()` - по всьому файлу та повертає його вміст у вигляді списку.

Також можна використовувати цикл `for` для ітерації по кожному рядку файлу та обробки його даних. Наприклад, за допомогою циклу `for`, можна вивести всі слова, які починаються з певної літери:

```Python
file = open("test.txt", "r")
for line in file:
    words = line.split(" ")
    for word in words:
        if word[0] == "a":
            print(word)
```

Існує багато інших способів обробки та маніпулювання даними з текстових файлів у Python. Для більш детальної інформації та прикладів коду, рекомендуємо ознайомитися з документацією.
