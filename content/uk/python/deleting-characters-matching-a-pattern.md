## Чому
У програмуванні існує безліч різноманітних завдань, одним з яких є видалення символів, що відповідають заданому шаблону. Це може бути корисно при обробці текстових даних або застосуванні певних фільтрів до введених користувачем даних.

## Як це зробити
Для видалення символів, що відповідають певному шаблону, ви можете використовувати метод `sub()` зі стандартної бібліотеки `re` (регулярні вирази). Приклад коду може виглядати так:
```Python
import re

text = "Це рядок з деякими цифрами 123 та спеціальними символами!@#"
pattern = "[0-9!@#$%]"
cleaned_text = re.sub(pattern, "", text)

print(cleaned_text)
```
Результатом виконання цього коду буде рядок без цифр та спеціальних символів: "Це рядок з деякими цифрами та спеціальними символами". Ви можете змінювати шаблон та текст за своїми потребами і отримувати різні результати.

## Глибше
Щоб краще зрозуміти процес видалення символів, вам слід дізнатися більше про регулярні вирази та метод `sub()`. Регулярні вирази - це спеціальні шаблони, які використовуються для пошуку та заміни тексту. Вони є потужним інструментом у роботі з текстовими даними, оскільки дозволяють вибирати певні частини тексту за заданим шаблоном.

Метод `sub()` використовується для заміни частин тексту, що відповідають паттерну, іншим текстом. Він приймає три аргументи: шаблон для пошуку, текст для заміни та вихідний текст. Важливо пам'ятати, що `sub()` повертає новий рядок, а не змінює початковий.
