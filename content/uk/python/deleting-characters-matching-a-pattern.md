## Чому

Іноді під час програмування нам потрібно видаляти певний паттерн символів зі строки. Наприклад, при обробці даних з файлу CSV, ми можемо мати рядки з непотрібними комами або зайвими символами, які ми бажаємо видалити. У таких випадках корисно знати як видаляти символи за певним паттерном, що допоможе нам більш ефективно обробляти дані.

## Як це зробити

Написати функцію для видалення символів за паттерном може бути досить просто. Спочатку нам потрібно імпортувати модуль "re", який надає інструменти для роботи з регулярними виразами. Потім ми можемо використати метод "sub()" для видалення символів, які відповідають паттерну.

```Python
import re

def delete_characters(pattern, text):
    return re.sub(pattern, "", text)

text = "Hello, world!"
pattern = "[, \!]"
new_text = delete_characters(pattern, text)

print(new_text)
```

В цьому прикладі ми видаляємо всі коми та знаки оклику зі строки "Hello, world!". Результатом буде "Helloworld".

## Глибше дослідження

У регулярних виразах є багато спеціальних символів, які можна використовувати для вибору тих чи інших символів. Наприклад, символ "." відповідає будь-якому символу, а "*" позначає 0 або більше повторень попереднього символу. Використовуючи ці символи та багато інших, ми можемо створити складні шаблони, за якими будуть видалятися символи.
