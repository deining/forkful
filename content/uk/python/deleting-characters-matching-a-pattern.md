## Чому
Іноді при роботі з текстовими данними нам потрібно видалити певні символи за певним шаблоном. Це може бути доцільно для того, щоб очистити данні або виконати певну форматування.

## Як це зробити
```Python
# Приклад для видалення цифр з тексту
text = "Це мій текст з 123 цифрами"
pattern = r'\d' # використовуємо регулярні вирази для пошуку цифр
new_text = re.sub(pattern, "", text) # використовуємо функцію .sub() для заміни знайдених символів на порожній рядок
print(new_text) # виводимо результат: "Це мій текст з цифрами"
```

```Python
# Приклад для видалення символів певного алфавіту з тексту
text = "Це мій текст з англійською та російською мовами"
pattern = r'[a-zA-Zа-яА-Я]' # використовуємо регулярні вирази для пошуку символів латиниці та кирилиці
new_text = re.sub(pattern, "", text) # використовуємо функцію .sub() для заміни знайдених символів на порожній рядок
print(new_text) # виводимо результат: "Це мій текст з та мовами"
```

## Трохи детальніше
Для видалення символів за певним шаблоном, нам потрібно використовувати регулярні вирази. У Python для цього існує модуль re, який містить функції для роботи з регулярними виразами. Функція .sub() видаляє знайдені відповідності та заміняє їх на певний символ або стрічку. Докладніше про роботу з регулярними виразами та модулем re можна дізнатися з документації Python або з інших ресурсів, які приведені у розділі "See Also".
