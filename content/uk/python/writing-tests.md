##Чому

Написання тестів є необхідною складовою частиною процесу програмування. Використання тестування допоможе впевнитися, що ваш код працює правильно та не має помилок, забезпечити високу якість програми і полегшити її підтримку у майбутньому.

##Як

Для написання тестів ви можете використовувати вбудований модуль unittest в Python. Для початку створіть новий файл з розширенням .py і імпортуйте модуль, щоб почати його використання. Далі, визначте клас для вашого тесту і створіть методи для перевірки різних частин вашого коду за допомогою методів assert. Проходження тестів підтвердить правильність вашого коду, а помилки допоможуть вам знайти і виправити проблеми.

```Python
import unittest

# визначте свій клас, що успадковує один з класів вбудованого модуля unittest
class TestCalculator(unittest.TestCase):
  
    # метод для перевірки правильності додавання двох чисел
    def test_add(self):
        result = 1 + 2
        # перший аргумент в assert - очікуване значення, другий - фактичне
        self.assertEqual(result, 3)
        
    # метод для перевірки правильності множення двох чисел
    def test_multiply(self):
        result = 3 * 4
        self.assertEqual(result, 12)

# запустіть тести за допомогою методу main вбудованого модуля unittest
if __name__ == '__main__':
    unittest.main()
```

Вивід під час запуску файлу повинен бути таким:

```
..
----------------------------------------------------------------------
Ran 2 tests in 0.000s

OK
```

Це означає, що обидва тести пройшли успішно.

##Вглубшому

Написання тестів допоможе вам покращити якість вашого програмного коду. Вони дозволяють перевірити правильність окремих частин коду, які ви написали, та виявити проблеми до того, як ваша програма буде виконуватися в живу. Тестова звітність також може бути корисною для обговорення з командою або для демонстрації клієнту.

Інші корисні модулі для написання тестів - pytest та doctest. pytest дозволяє більш покращене тестування за допомогою різних типів асертів, а doctest дозволяє включати тести прямо в документацію вашого коду.
