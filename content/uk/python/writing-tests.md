## Чому
Це необхідно для забезпечення високої якості вашої програми та попередження можливих проблем в майбутньому. Тести дозволяють перевірити, чи ваш код працює правильно та виконує очікувані результати.

## Як
Написання тестів в Python дуже просте і дозволяє швидко перевірити різноманітні аспекти вашої програми. Для початку, вам потрібно імпортувати модуль ```unittest```, який надає функціональність для написання тестів.

```Python
import unittest
```

Тепер ви можете створити свій перший тестовий клас. Назва класу повинна починатися зі слова ```Test```, щоб дефайнити, що це тест.

```Python
class TestMathFunctions(unittest.TestCase):
```

Для створення тестових методів використовуйте ключове слово ```def```, а далі назву тесту та об'єкт ```self```.

```Python
def test_addition(self):
```

У тілі кожного тесту ви можете викликати функції з вашої програми та порівнювати результат з очікуваним за допомогою методу ```assertEqual```.

```Python
def test_addition(self):
    self.assertEqual(add(2, 3), 5)
```

Запустіть свої тести за допомогою команди ```unittest.main()```, яку ви можете виконати зі своєї IDE або терміналу.

```Python
if __name__ == '__main__':
    unittest.main()
```

## Deep Dive
Написання тестів в Python дозволяє виконати ще більш детальну перевірку вашої програми. Ви можете використовувати спеціальні методи для перевірки цілочисельних, рядкових або булевих значень, а також використовувати умови для перевірки різних випадків. Крім того, ви можете створювати власні тести та розширювати функціональність за потреби.
