## Чому

Існує багато ситуацій, коли потрібно конвертувати дату в рядок у Python. Наприклад, для відображення дати у зрозумілому для користувача форматі або для збереження дати у файлі. У цій статті ми детально розглянемо, як конвертувати дату в рядок за допомогою мови програмування Python.

## Як це зробити

Для початку, нам потрібно імпортувати з модуля `datetime` класу `datetime`:

```Python
import datetime

datetime_object = datetime.datetime.now()
```
Цей об'єкт `datetime` є представленням поточної дати та часу. Тепер, щоб конвертувати його у рядок, нам потрібно використати метод `strftime()`:

```Python
date_string = datetime_object.strftime("%d-%m-%Y")
print(date_string)
```

В результаті на екрані ви побачите згенерований рядок з поточною датою у форматі `день-місяць-рік`.

## Глибоке вивчення

У методі `strftime()` ми використали спеціальні символи для визначення бажаного формату дати. Давайте розглянемо їх детальніше:

- `%d` - день у форматі двох цифр (наприклад, 01)
- `%m` - місяць у форматі двох цифр (наприклад, 05)
- `%Y` - рік у форматі чотирьох цифр (наприклад, 2021)
- `%H` - година у форматі двох цифр (наприклад, 10)
- `%M` - хвилина у форматі двох цифр (наприклад, 30)
- `%S` - секунда у форматі двох цифр (наприклад, 45)

За допомогою цих символів ви можете створити будь-який бажаний формат для вашої дати. Зверніть увагу на те, що відрізняються великі та малі літери символів. Наприклад, у форматі `%d-%m-%Y` ми використали маленькі літери для визначення дня та місяця, щоб отримати формат `день-місяць-рік`, а не `місяць-день-рік`.
