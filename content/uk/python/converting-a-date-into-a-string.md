## Чому

В даних часах, комп'ютерні програми користуються форматом дати і часу для збереження і відображення інформації. Часом же, буває потрібно вивести ці дані у форматі рядка для зручності користувачів або для подальшого використання у програмі. У цьому пості, ми дізнаємось, як перетворити дату у рядок за допомогою мови програмування Python.

## Як

Для початку, нам потрібно імпортувати модуль datetime, який містить функції для роботи з датами і часом. Потім, ми створимо змінну з датою, яку бажаємо перетворити у рядок.

```Python
import datetime

date = datetime.date(2021, 9, 21)
```

Далі, ми можемо використовувати метод `strftime()` для перетворення дати у рядок за допомогою спеціальних символів. Наприклад, `%d` виводить день у форматі двоцифрового числа, а `%B` - повне назву місяця.

```Python
string_from_date = date.strftime("%d %B %Y")

print(string_from_date) # 21 September 2021
```

Також, ми можемо використовувати метод `str()` для перетворення дати у рядок, проте результатом буде більш простий формат.

```Python
string_from_date = str(date)

print(string_from_date) # 2021-09-21
```

## Поглиблене дослідження

Щоб детальніше розібратись у процесі перетворення дати у рядок, давайте розглянемо список найпоширеніших спеціальних символів, які можна використовувати в методі `strftime()`.

- `%d` - день місяця у форматі двоцифрового числа
- `%m` - місяць у форматі двоцифрового числа
- `%B` - повна назва місяця
- `%y` - рік у форматі двоцифрового числа
- `%Y` - рік у форматі чотирицифрового числа
- `%A` - повна назва дня тижня
- `%a` - скорочена назва дня тижня
- `%H` - година у 24-годинному форматі
- `%I` - година у 12-годинному форматі
- `%M` - хвилина
- `%S` - секунда

Додатково, до цих символів можна додавати різноманітні розділювачі, наприклад `/%/`, який розділятиме дату, місяць та рік.
