## Чому

Відправка запиту HTTP з базовою аутентифікацією є важливим елементом розробки веб-додатків. Вона дозволяє забезпечити безпеку і захист від несанкціонованого доступу до ваших даних.

## Як це зробити

```Python
import requests

# Вказуємо URL, до якого будемо відправляти запит
url = "https://www.example.com"

# Вказуємо логін і пароль для аутентифікації
username = "johnsmith"
password = "qwerty123"

# Створюємо об'єкт запиту і додаємо в нього параметри аутентифікації
r = requests.get(url, auth=(username, password))

# Виводимо код статусу запиту
print(r.status_code)
```

В результаті ви отримаєте відповідь з кодом статусу 200, що означає успішний запит з правильними обліковими даними.

## Глибоке дослідження

Аутентифікація HTTP базі була створена для надання простого та ефективного шляху для перевірки облікових даних безпеки. Вона використовує заголовок `Authorization` для передачі інформації про аутентифікацію. Базова аутентифікація передає логін та пароль у форматі Base64 через цей заголовок. Це безпечніше, ніж передавати облікові дані у відкритому тексті.
