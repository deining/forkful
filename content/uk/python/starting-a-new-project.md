## Чому

Створення нового проекту у програмуванні є важливим етапом у розвитку навичок та вдосконаленням існуючих знань. Це дозволяє вам застосовувати вміння, які ви вивчили, та вивчати нові, працюючи над цікавим і призначеним саме для вас проектом.

## Як

```Python
# Приклад програми на Python для пошуку середнього арифметичного
numbers = [5, 10, 15, 20, 25]
sum = 0
for num in numbers:
  sum += num

avg = sum / len(numbers)
print("Середнє значення: " + str(avg))
```

В цьому прикладі ми використовуємо цикл `for` для знаходження суми чисел та обчислення середнього значення. Після чого ми виводимо результат, використовуючи функцію `print()` із рядковими значеннями.

```Python
# Приклад програми на Python для роботи зі списками
foods = ["pizza", "burrito", "sushi", "taco", "ramen"]

# Додавання елементу до списку
foods.append("pasta")

# Видалення елементу зі списку
foods.remove("taco")

# Сортування списку за алфавітом
foods.sort()

print(foods)
```

У цьому прикладі ми показуємо, як працювати зі списками у Python. Використовуючи вбудовані функції `append()`, `remove()` та `sort()`, ми можемо додавати, видаляти та сортувати елементи у списку.

## Deep Dive

Завжди важливо починати з простих проектів та поступово розширювати їх складність. Почніть з того, що ви вже знаєте, і додавайте нові елементи, поступово поглиблюючи свої знання. Крім того, спробуйте створювати проекти, які вас цікавлять, це збільшить мотивацію та закріпить отримані знання.
