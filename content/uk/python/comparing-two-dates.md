## Чому

У програмуванні часто виникає необхідність порівнювати дати, наприклад, для визначення того, яка з них є більш новою або для фільтрації даних за датами. У цій статті ми розглянемо, як це можна зробити за допомогою Python.

## Як це зробити

Для порівняння дат ми можемо скористатися вбудованими функціями у Python - `date()` та `timedelta()`. Перша функція дозволяє створити новий об'єкт з датою, а друга - визначити різницю між двома датами. Для прикладу, нам потрібно порівняти 1 січня 2020 року з 1 січня 2021 року:

```Python
date_1 = date(2020, 1, 1)
date_2 = date(2021, 1, 1)

difference = date_2 - date_1
print(difference.days)

# Output: 366
```

Таким чином, ми отримали різницю в 366 днів. Також ми можемо використовувати методи `day`, `month` та `year` для отримання відповідних значень дня, місяця та року з дати.

```Python
print(date_1.day)
print(date_2.month)

# Output:
# 1
# 1
```

## Вдивимося глибше

У Python є також модуль `datetime`, який дозволяє працювати з датами та часом більш гнучко. Він дозволяє, серед іншого, визначати різницю між двома датами з точністю до мікросекунд. Також можна використовувати методи `strftime()` та `strptime()` для форматування дат та подальшої роботи з ними. Для докладнішої інформації про можливості модуля `datetime` рекомендуємо ознайомитися з офіційною документацією.
