## Чому

Нерідко у програмуванні доводиться порівнювати дати, наприклад, для перевірки часового проміжку або порівняння двох подій. У цій статті ми розглянемо, як це зробити за допомогою Python.

## Як

```Python
# Імпортуємо модуль datetime
import datetime

# Задаємо дві дати
date1 = datetime.date(2021, 10, 20)
date2 = datetime.date(2021, 10, 25)

# Порівнюємо їх за допомогою оператора ">"
if date1 > date2:
  print("Дата 1 є більшою за дату 2")
elif date1 < date2:
  print("Дата 1 є меншою за дату 2")
else:
  print("Дати рівні")

# Вивід: "Дата 1 є меншою за дату 2"
```

Також можна порівнювати дати за допомогою методу `compare`, який повертає -1, 0 або 1, в залежності від того, чи менша, рівна або більша одна дата від іншої. Наприклад:

```Python
# Задаємо ті ж дати
date1 = datetime.date(2021, 10, 20)
date2 = datetime.date(2021, 10, 25)

# Порівнюємо їх за допомогою методу compare
print(date1.compare(date2))

# Вивід: "-1"
```

## Глибокий погляд

При порівнянні дат важливо враховувати не тільки числове значення, а й формат дати (з тире, крапками або слешами), а також розташування компонентів дати (часова зона, або чи присутній часовий пояс). Також використовуються різні формати запису дати, наприклад, в Америці зазвичай першим ставиться місяць, а в Європі - день. Варто детальніше дослідити ці особливості перед роботою з датами.
