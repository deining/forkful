## Чому

Порівняння двох дат - це важлива функція у багатьох програмах, особливо тих, які працюють зі складним часовим розкладом. Воно дозволяє зрозуміти, яка з дат була раніше або пізніше, а також обрахувати різницю між ними.

## Як

Для порівняння двох дат у Python використовують модуль `datetime`. Давайте розглянемо декілька прикладів:

```Python
import datetime

# Створення об'єкту datetime для першої дати
date1 = datetime.datetime(2020, 5, 10)

# Створення об'єкту datetime для другої дати
date2 = datetime.datetime(2021, 1, 1)

# Порівняння за допомогою оператора "<"
print(date1 < date2) # Виведе True

# Обрахування різниці між датами
difference = date2 - date1
print(difference.days) # Виведе 236
```

У цьому прикладі ми використовуємо модуль `datetime` для створення двох об'єктів, які представляють дати. Потім за допомогою оператора `<` ми порівнюємо, яка з них є ранішою. Крім того, ми можемо обрахувати різницю між датами, використовуючи оператор `-` та атрибут `days` для отримання кількості днів різниці.

## Поглиблене дослідження

У модулі `datetime` є багато корисних функцій та методів для роботи з датами. Давайте розглянемо декілька з них:

### today()

Цей метод повертає об'єкт `datetime` для поточної дати.

```Python
import datetime

today = datetime.datetime.today()
print(today) # Виведе поточну дату та час
```

### strptime()

Ця функція дозволяє перетворити рядок на об'єкт `datetime`.

```Python
import datetime

date_str = "25-11-2020"
date = datetime.datetime.strptime(date_str, "%d-%m-%Y")
print(date) # Виведе об'єкт datetime для 25 листопада 2020 року
```

### strftime()

Цей метод дозволяє отримати рядок з об'єкту `datetime`.

```Python
import datetime

date = datetime.datetime(2020, 12, 25)
date_str = date.strftime("%A, %d %B %Y")
print(date_str) # Виведе "Friday, 25 December 2020"
```

Для ознайомлення з іншими функціями та методами модуля `datetime` можна переглянути [офіційну документацію](https://docs.python.org/3/library/datetime.html).
