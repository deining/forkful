# Чому

На сьогоднішній день, програмування стало необхідним навичкою у будь-якій сфері. Порівняння двох дат є важливим елементом у багатьох програмах, особливо в контексті веб-розробки та роботи з базами даних. Тому, знання, як порівнювати дві дати, є важливим для будь-якого бажаючого стати успішним програмістом.

# Як це зробити

Для порівняння двох дат у Python необхідно використовувати функцію "datetime" із стандартної бібліотеки. Спочатку, потрібно імпортувати цю бібліотеку до свого коду за допомогою команди "import datetime". Далі, створимо дві змінні, які будуть містити дати, які потрібно порівняти. Наприклад, "first_date = datetime.datetime(2020, 5, 15)" та "second_date = datetime.datetime(2021, 3, 10)". Далі, можна використовувати умовні оператори, такі як "if-else" для порівняння цих двох дат та виконання необхідних дій залежно від результату порівняння.

```Python
import datetime

first_date = datetime.datetime(2020, 5, 15)
second_date = datetime.datetime(2021, 3, 10)

if first_date > second_date:
    print("Перша дата пізніша за другу")
elif first_date < second_date:
    print("Друга дата пізніша за першу")
else:
    print("Обидві дати однакові")
```

В цьому прикладі, за допомогою функції "datetime", ми створили дві змінні, які містять різні дати. Потім, за допомогою умовних операторів, ми порівняли ці дати та вивели відповідне повідомлення. Залежно від потреб, можна додавати додаткові умови та дії для виконання.
