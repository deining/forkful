## Чому

У програмуванні часто виникає потреба з'єднати кілька рядків в один. Це може бути корисно, наприклад, для створення повідомлень або побудови складних рядків. Використання конкатенації рядків є швидким і простим способом цього досягнути.

## Як

Для з'єднання рядків у Python використовуються оператор "+" або метод "join()". Давайте подивимося на приклади з використанням обох методів:

```Python
# Використовуючи "+" оператор
str1 = "Привіт"
str2 = "Україно!"
str3 = str1 + str2
print(str3)  # Вивід: ПривітУкраїно!

# Використовуючи "join()" метод
list_of_words = ["Найкращий", "день", "для", "програмування"]
str4 = " ".join(list_of_words)
print(str4)  # Вивід: Найкращий день для програмування
```

Обидва методи дають однаковий результат, але для складних рядків краще використовувати метод "join()", оскільки він пропонує більш гнучкі можливості для вставлення роздільників між рядками.

## Глибоке погруження

У Python рядки є незмінними (immutable) об'єктами, тому з'єднання рядків призводить до створення нового об'єкта, а не змінює існуючий. Також важливо пам'ятати, що при з'єднанні рядків всі аргументи повинні бути рядками, тому, наприклад, це приведе до помилки:

```Python
num = 10
str5 = "Кількість студентів: " + num  # Помилка
```

Якщо потрібно з'єднати рядок із числовим значенням, спочатку треба сконвертувати числове значення у рядок за допомогою функції "str()".
