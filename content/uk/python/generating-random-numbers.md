# Чому
Існує багато випадків, коли програмістам потрібно генерувати випадкові числа. Це може бути необхідно для створення тестових даних, шифрування або моделювання випадкових подій у програмі.

## Як це зробити
Один з найпростіших способів зробити це використовує вбудований модуль `random` в Python. За допомогою функції `random.randint()`, ми можемо генерувати цілі числа у певному діапазоні. Наприклад:

```python
import random
print(random.randint(1, 10)) # виводить випадкове число від 1 до 10
```

Ми також можемо використовувати інші функції, такі як `random.random()` для генерування випадкового дійсного числа або `random.choice()` для вибору випадкового елемента зі списку.

## Глибоке погруження
Генерування випадкових чисел у Python базується на генераторі псевдовипадкових чисел. Це означає, що фактично немає способів зробити абсолютно випадковий вибір. Генератор псевдовипадкових чисел використовує початкове число, відоме як "насіння" (`seed`), для створення послідовності випадкових чисел. Якщо не вказати насіння, як це робилося в нашому прикладі з `random.randint()`, модуль `random` автоматично бере поточний час як насіння.
