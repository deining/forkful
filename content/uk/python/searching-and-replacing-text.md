## Чому

У пошуку та заміні тексту є важлива роль у програмуванні, оскільки це дає змогу змінювати та вдосконалювати текст, що дає можливість більш ефективно та точно працювати з кодом.

## Як

Існує багато способів швидко та легко змінювати текст у Python. Один з найпростіших способів - використання методу `.replace()`.

```Python
# Приклад 1
text = "Це є текст для прикладу."
new_text = text.replace("текст", "код")
print(new_text)
```
Результат:

```Python
Це є код для прикладу.
```

Можна також використати регулярні вирази для пошуку та заміни тексту. Наприклад:

```Python
# Приклад 2
import re
text = "1 це перший рядок, а це 2 другий рядок"
new_text = re.sub(r"\d", "Рядок", text)
print(new_text)
```
Результат:

```Python
Рядок це перший рядок, а Рядок це другий рядок
```

## Вглиб

Регулярні вирази можуть бути дуже потужним інструментом для пошуку та заміни тексту у Python. Вони дають можливість шукати не тільки окремі слова чи фрази, але й різні шаблони. Наприклад, можна використати регулярний вираз для знаходження великих букв у тексті та заміни їх на малі:

```Python
# Приклад 3
import re
text = "ЦЕ є ТЕКСТ для ПРИКЛАДУ"
new_text = re.sub(r"[A-Z]", lambda x: x.group().lower(), text)
print(new_text)
```
Результат:

```Python
це є текст для прикладу
```

Також, регулярні вирази дають можливість виконувати більш складні операції, такі як шукання та заміна у залежності від певних умов чи шаблонів.
