## Як це зробити:
```Python
import re

# Приклад рядка
text = "Hello, World! 1234"

# Видалення всіх цифр
no_digits = re.sub(r'\d', '', text)
print(no_digits)  # Вивід: "Hello, World! "

# Видалення пунктуації
no_punctuation = re.sub(r'[^\w\s]', '', text)
print(no_punctuation)  # Вивід: "Hello World 1234"

# Видалення голосних
no_vowels = re.sub(r'[aeiouAEIOU]', '', text)
print(no_vowels)  # Вивід: "Hll, Wrld! 1234"
```

### Власна функція, яку я написав

Я роблю це достатньо часто, тому я рефакторив це в функцію `delete()`. Це також гарна демонстрація [doctests](https://docs.python.org/3/library/doctest.html):

```python
def delete(string: str, regex: str) -> str:
    """
    >>> delete("Hello, world!", "l")
    'Heo, word!'

    >>> delete("Hello, world!", "[a-z]")
    'H, !'
    """
    return re.sub(regex, "", string)
```



## Поглиблений огляд
Практика видалення символів, які відповідають певному шаблону у тексті, має глибокі коріння в комп'ютерних науках, які сягають ранніх інструментів Unix, таких як `sed` і `grep`. У Python модуль `re` надає таку можливість, використовуючи регулярні вирази – потужний і універсальний інструмент для обробки тексту.

Альтернативи модулю `re` включають:
- Методи рядків, такі як `replace()`, для простих випадків.
- Сторонні бібліотеки, як-от `regex`, для складніших шаблонів та кращої підтримки Юнікоду.

За лаштунками, коли ви використовуєте `re.sub()`, інтерпретатор Python компілює шаблон у серію байт-кодів, які обробляються автоматом станів, що виконує зіставлення шаблону безпосередньо з текстом на вході. Ця операція може бути ресурсоємною для великих рядків або складних шаблонів, тому питання продуктивності є критично важливими для обробки великих даних.
