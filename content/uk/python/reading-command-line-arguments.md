## Чому

Python є гнучкою та потужною мовою програмування, і, звичайно, зручним інструментом для написання скриптів та автоматизації завдань на вашому комп'ютері. Одним із цих інструментів є зчитування аргументів командного рядка, яке дозволяє легко та ефективно взаємодіяти з вашим кодом через термінал.

Якщо ви хочете зробити свої скрипти більш універсальними та легшими в управлінні, або просто хочете ефективніше працювати з ними через термінал, зчитування аргументів командного рядка є необхідним умінням для кожного програміста Python.

## Як То

Щоб зчитати аргументи командного рядка в Python, використовуйте модуль `sys` та його атрибут `argv`. Давайте розглянемо приклад коду, який використовує цей метод:

```Python
import sys

# отримуємо аргументи командного рядка
args = sys.argv
# перший аргумент - назва скрипту
script_name = args[0]
# другий аргумент - перший аргумент, переданий скрипту
first_arg = args[1]
# третій аргумент - другий аргумент, переданий скрипту
second_arg = args[2]

print("Назва скрипту: " + script_name)
print("Перший аргумент: " + first_arg)
print("Другий аргумент: " + second_arg)
```

Використовуючи цей код, ми можемо передати аргументи під час запуску скрипта, наприклад:

```
python my_script.py Hello World
```

У результаті отримаємо наступний вивід:

```
Назва скрипту: my_script.py
Перший аргумент: Hello
Другий аргумент: World
```

Це простий приклад, який можна розширити та використовувати для різних завдань, які вимагають зчитування аргументів командного рядка.

## Глибоке Занурення

Крім розбору аргументів командного рядка, модуль `sys` також містить функції `exit()` та `stderr`, які дозволяють управляти виконанням скрипта та виводити повідомлення про помилку. Крім того, ви можете використовувати `os` для отримання інформації про поточний каталог або виконати команди в терміналі.
