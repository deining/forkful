## Чому

Розуміння та використання аргументів командного рядка є важливими навичками для будь-якого Пайтон програміста. Це дозволяє вам надавати різноманітні налаштування та параметри для вашої програми, що дає більшу гнучкість та зручність користувачам управляти вашим кодом.

## Як це зробити

Почнемо з основ. Щоб отримати доступ до командних аргументів у Пайтон, нам потрібно імпортувати модуль sys. Потім ми можемо отримати список будь-яких переданих аргументів за допомогою ```sys.argv```. Давайте переглянемо код, який демонструє це:

```Python
import sys

print("Приклад читання командних аргументів:")

# Перший аргумент - це назва нашого скрипта, тому ми його пропускаємо
print("Передані аргументи:", str(sys.argv[1:]))

# Перевіряємо, чи був переданий аргумент -h або -help, щоб дати користувачеві інструкцію
if "-h" in sys.argv or "-help" in sys.argv:
    print("Використайте цей сценарій, передавши виключно командні аргументи:")
    print("-h або -help: виводить інструкцію")
    print("-o або -output: вказує шлях до виводу")
```

Ввійдіть у такий самий формат коду у своєму улюбленому IDE або текстовому редакторі та збережіть як файл з розширенням .py. Назвіть його, наприклад, ```commandline_args.py```. Тоді ви можете викликати його з терміналу, передаючи якісь аргументи: ```python commandline_args.py -h -o output.txt```. Ви повинні побачити наступне:

```
Приклад читання командних аргументів:
Передані аргументи: ['-h', '-o', 'output.txt']
Використайте цей сценарій, передавши виключно командні аргументи:
-h або -help: виводить інструкцію
-o або -output: вказує шлях до виводу
```

І це все! Тепер ви можете читати передані аргументи та використовувати їх у своїй програмі.
