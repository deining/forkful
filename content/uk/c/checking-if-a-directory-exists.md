Що і чому?

Перевірка, чи існує папка, - це процес, коли програміст перевіряє, чи існує вказана папка в системі. Це необхідно для того, щоб програма могла коректно виконувати дії з файлами в цій папці.

Як це зробити?

```C
#include <stdio.h>
#include <stdlib.h>
#include <sys/types.h>
#include <sys/stat.h>

int main(){
    char dir_name[] = "folder"; // назва папки, яку ми хочемо перевірити
    struct stat st = {0};

    if (stat(dir_name, &st) == -1){ // виклик функції stat та перевірка її результату
        printf("Папки %s не існує\n", dir_name);
    }
    else{
        printf("Папка %s існує\n", dir_name);
    }
    return 0;
}
```
Вихідний файл:
```
Папки folder існує
```

Глибока занурення

Перевірка існування папки використовується для багатьох цілей, серед яких встановлення прав доступу до файлів, збереження даних у вказаному місці та безпечне виконання дій з файлами в цій папці. Сама перевірка може бути здійснена за допомогою функції stat або access, обидві з яких реалізовані в стандартній бібліотеці C.

Також, існують альтернативні методи перевірки існування папки, наприклад, використання командного рядка або використання бібліотеки dirent. Проте, функція stat є більш зручною та детальною для перевірки існування папки.

Див. також

Для додаткової інформації про функції stat та access рекомендується ознайомитись з документацією їх використання. Також можна прочитати про бібліотеку dirent та її застосування для перевірки існування папки у вашій програмі.
