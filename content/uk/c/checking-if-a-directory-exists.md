# З Чого Почати

У цьому блозі ми будемо досліджувати, як перевірити наявність директорії за допомогою програмування на мові "C". Ми покажемо вам, як це зробити крок за кроком та надамо глибокі знання про цей процес. Готові? Почнемо!

## Чому Знаходити Наявність Директорії

Перевірка наявності директорій є важливою частиною програмування на мові "C". Це може бути корисно в багатьох сценаріях, наприклад, якщо програма повинна зберігати дані в певній директорії або якщо потрібно перевірити, чи існує певна директорія перед створенням нової. Перевірка наявності директорії допомагає запобігти можливим помилкам та забезпечити більш ефективну роботу програми.

## Як Перевірити Наявність Директорії

Для перевірки наявності директорії ми будемо використовувати функцію "opendir" та "readdir". Спочатку нам потрібно підключити бібліотеку "dirent.h". Після цього, нам потрібно відкрити директорію за допомогою функції "opendir". Якщо ця функція поверне значення "NULL", це означає, що директорія не існує. У протилежному випадку, ми можемо використовувати функцію "readdir" для зчитування файлів у директорії. Ось код, який демонструє цей процес:

```C
#include <stdio.h>
#include <dirent.h>

int main() {
    DIR *dir = opendir("./exampleDirectory");
    if (dir == NULL) {
        printf("Директорія не існує.\n");
    } else {
        printf("Директорія існує.\n");
        struct dirent *entry;
        printf("Список файлів у директорії:\n");
        while ((entry = readdir(dir)) != NULL) {
            printf("%s\n", entry->d_name);
        }
        closedir(dir); // закриваємо директорію
    }
    return 0;
}
```

Після виконання цього коду, ви повинні побачити повідомлення про те, чи існує директорія та список файлів, які знаходяться у ній.
