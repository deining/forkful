## Простота та корисність розрахунку дати у майбутньому чи минулому 

Цікавістью програмування є можливість розв’язати різноманітні задачі, а однією з них є розрахунок дати у майбутньому чи минулому. Наприклад, вам потрібно визначити, коли була або буде певна подія, або скільки часу минуло з певної дати. У цій статті ми розглянемо, як це можна зробити за допомогою мови програмування C.

## Як це зробити

Для того, щоб розрахувати дату у майбутньому або минулому, нам необхідно використовувати структуру "tm", яка містить поля для року, місяця, дня, години, хвилини та секунди. Потім ми використовуємо функції "mktime" та "localtime" для перетворення дати у форматі "tm" в "time_t" і навпаки. 

Ось приклад коду, який розраховує дату у майбутньому на п'ять днів:

```C
#include <stdio.h>
#include <time.h>

int main()
{
  // отримуємо поточну дату та час
  time_t now = time(NULL);

  // переводимо у формат "tm"
  struct tm *date = localtime(&now);

  // додаємо 5 днів до дати
  date->tm_mday += 5;

  // переводимо знову у формат "time_t"
  time_t future = mktime(date);

  // виводимо нову дату
  printf("Дата через 5 днів: %s", ctime(&future));

  return 0;
}
```

Результат виконання коду буде виглядати так:

```
Дата через 5 днів: Sun Jul 18 04:00:00 2021
```

Цей приклад можна змінювати за своїми потребами, наприклад, для знаходження дати в минулому, додати змінну "теперішній час" до "tm_mday" замість додавання. 

## Огляд та деталі

Описана вище функціональність базується на праці з "unix timestamp", який представляє собою кількість секунд, які пройшли з 1 січня 1970 року. Такі ж функції можна знайти і у інших мовах програмування, але лише з різними назвами. 

Також слід зазначити, що при розрахунку дати в майбутньому або минулому потрібно враховувати зміну часових зон та переходи на літній час. Для більш складних задач, пов'язаних з датою та часом, також можна використовувати сторонні бібліотеки.
