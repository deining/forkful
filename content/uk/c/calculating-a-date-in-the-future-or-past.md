# Чому

Обчислення дати в майбутньому або минулому може бути корисним для багатьох програм та проектів, особливо тих, які працюють з часом, термінами або фіксованою датою.

# Як це зробити

Перед тим, як почати, ми повинні зрозуміти основні поняття, пов'язані з датами та часом в мові програмування C. Найбільш важливими з них є типи даних time_t та struct tm.

Для обчислення дати в майбутньому або минулому, ми можемо скористатися функцією localtime, яка повертає локальний час у вигляді структури struct tm. Для цього, спочатку потрібно створити змінну типу time_t, яка буде містити дату та час в секундах. Наприклад:

```C
time_t current_time;
```

Далі ми можемо використати функцію time для отримання поточного часу та присвоїти цей час нашій змінній, щоб мати базову вихідну точку для обчислень:

```C
current_time = time(NULL);
```

Для обчислення дати в майбутньому, ми можемо додати необхідну кількість секунд до нашої базової дати. Наприклад, якщо ми хочемо отримати дату через 7 днів, ми можемо використати наступний код:

```C
current_time = current_time + (7 * 24 * 60 * 60);
```

Аналогічно, для обчислення дати в минулому, ми можемо відняти необхідну кількість секунд від базової дати.

Для перетворення отриманої дати у зручний для нас формат, ми можемо скористатися функцією strftime, яка конвертує структуру struct tm у рядок за заданим форматом. Наприклад:

```C
strftime(buffer, 80, "%d.%m.%Y", localtime(&current_time));
```
