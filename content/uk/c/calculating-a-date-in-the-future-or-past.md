## Чому

Програмування може бути цікавим і захоплюючим хобі, яке дає можливість розкрити свою творчість та логічне мислення. Обчислення дати в майбутньому або у минулому може бути цікавим завданням для вивчення програмування або вирішення практичних задач.

## Як

Для обчислення дати в майбутньому або у минулому використовуються стандартні функції з бібліотеки <time.h>. Для початку, необхідно включити цю бібліотеку до свого коду, а потім використовувати функції для введення поточної дати і обчислення нової дати.

```C
#include <time.h>

int main(){
	//отримати поточну дату і час
	time_t t = time(NULL);
	struct tm * now = localtime(&t);

	//збільшити дату на 1 день
	now->tm_mday += 1;
	mktime(now);

	//вивести нову дату
	printf("%d/%d/%d", now->tm_mday, now->tm_mon + 1, now->tm_year + 1900);
	
	return 0;
}
```

В даному прикладі коду ми збільшуємо дату на 1 день. Для цього використовуємо поля структури "tm" для введення поточної дати, а потім за допомогою функції "mktime" отримуємо правильний формат дати, який далі виводимо за допомогою функції "printf".

## Поглиблене дослідження

У коді для обробки дати також можна використовувати функції, що дозволяють встановлювати дату вручну, а також проводити порівняння між двома датами. Детальніше про це можна дізнатися з документації з функцій бібліотеки <time.h>.
