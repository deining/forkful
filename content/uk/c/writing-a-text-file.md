## Що & Чому?

Записування текстового файлу - це процес збереження даних у вигляді простого тексту в комп'ютері. Програмісти використовують цей метод для збереження та організації великих об'ємів інформації, також для обміну даними між програмами та пристроями.

## Як це зробити?

Для записування текстового файлу використовується функція ```fprintf()```. Нижче наведений приклад коду, де ми записуємо текстовий файл з назвою `file.txt` та додаємо до нього рядок `Hello World!`.

```C
#include <stdio.h>

int main() {
    FILE *file = fopen("file.txt", "w");
    fprintf(file, "%s", "Hello World!");
    fclose(file);
    return 0;
}
```

В результаті виконання цього коду, ми отримаємо файл `file.txt` з написом `Hello World!`.

## Глибоке занурення

Перший текстовий редактор для запуску програм був створений у 1950-х роках. З тих пір було розроблено багато форматів для записування текстових файлів, таких як ASCII, UTF-8 та Unicode. Є також альтернативи для записування даних, наприклад, бази даних та розширені формати файлів, такі як PDF.

У програмі `fprintf()`, перший параметр є файловим покажчиком, який вказує на файл, який потрібно записати. Другим параметром є форматувана рядок, в якому можна вказувати дані для запису у файл. Цей рядок може містити спеціальні символи для форматування даних, наприклад `%d` для цілих чисел чи `%s` для рядків.
