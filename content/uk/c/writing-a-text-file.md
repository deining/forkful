## Чому

Писання текстового файлу є важливим елементом програмування, оскільки ви можете зберігати дані на вашому комп'ютері і з ними взаємодіяти за допомогою вашої програми.

## Як

Щоб створити текстовий файл, необхідно використовувати функцію `fopen()`, яка відкриє файл у потрібному режимі (запис або читання). Після відкриття, ви можете використовувати функцію `fprintf()` для запису даних у файл. Наприклад:

```C
#include <stdio.h>

int main(){
    FILE *file = fopen("example.txt", "w");
    
    if(file){
        fprintf(file, "Це приклад текстового файлу.");
        fclose(file);
    }
    
    return 0;
}
```

Цей код створить файл з назвою "example.txt" і запише в нього рядок "Це приклад текстового файлу." Після закінчення роботи з файлом, не забудьте закрити його за допомогою `fclose()`.

Щоб прочитати вміст текстового файлу, використовуйте функцію `fscanf()`. Наприклад, для виведення вмісту файлу на екран, можна використовувати цей код:

```C
#include <stdio.h>

int main(){
    FILE *file = fopen("example.txt", "r");
    
    if(file){
        char output[100];
        
        fscanf(file, "%s", output);
        printf("%s", output);
        
        fclose(file);
    }
    
    return 0;
}
```

## Глибокий погляд

Цікавий факт про текстові файли в C - їх можна редагувати за допомогою будь-якого текстового редактора, а не тільки в C програмі. Це дозволяє спростити роботу з великими об'ємами даних, а також легше відстежувати та вносити зміни до файлу.
