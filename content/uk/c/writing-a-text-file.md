## Чому

Написання текстового файлу може бути корисним у багатьох випадках, наприклад, для збереження даних або створення звіту. Також це може бути корисно для створення програм, які зберігають конфігураційні дані у текстових файлах.

## Як

Для написання текстового файлу у мові програмування C, необхідно використовувати функцію `fopen()`, яка відкриває файл і повертає вказівник на нього. Для запису даних у файл використовується функція `fprintf()`, яка приймає вказівник на файл та дані, які необхідно записати у нього.

```C
#include <stdio.h>

int main() {

    // відкриваємо файл для запису
    FILE *file = fopen("sample.txt", "w");

    // записуємо дані у файл
    fprintf(file, "Це приклад написаного тексту у файлі.");

    // закриваємо файл
    fclose(file);

    return 0;
}
```

Результатом виконання даного прикладу буде створення файлу `sample.txt` з вмістом `Це приклад написаного тексту у файлі.`

## Глибокий аналіз

При роботі з текстовими файлами важливо враховувати, що символ кінця рядка (`\n`) може відрізнятися на різних операційних системах. Наприклад, у Windows символ кінця рядка складається з двох символів - `\r\n`, тоді як у Linux - з одного символу `\n`.

Також важливо перевіряти результат виконання операцій з файлами, наприклад, чи вдалося відкрити файл або чи вдалося записати дані у файл. Для цього можна використовувати функцію `ferror()`, яка повертає `0`, якщо операція була виконана успішно, або ненульове значення у випадку помилки.
