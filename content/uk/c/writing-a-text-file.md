## Чому

Написання текстового файлу є важливою частиною програмування, особливо в мові C. Текстові файли дозволяють зберігати дані у структурованому форматі, що дозволяє легко читати та маніпулювати ними.

## Як 

Для початку, нам потрібно створити змінну типу `FILE`, яка буде використовуватися для зберігання вказівника на файл. Далі, ми можемо використовувати функцію `fopen()` для створення файлу та прив'язки до нього нашої змінної. Наприклад:

```C
FILE *fp; // створюємо змінну
fp = fopen("myfile.txt", "w"); // відкриваємо файл у режимі запису
```

Коли файл створений та прив'язаний до змінної, ми можемо використовувати функцію `fprintf()` для запису даних у наш файл. Наприклад:

```C
fprintf(fp, "Це мій перший текстовий файл"); // записуємо текст у файл
```

Щоб закрити файл та звільнити пам'ять, можемо використовувати функцію `fclose()`. Наприклад:

```C
fclose(fp); // закриваємо файл
```

В результаті, отримаємо файл з назвою "myfile.txt", який містить текст "Це мій перший текстовий файл".
