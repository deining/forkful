## Чому

Програмування – це надзвичайно цікаве та пізнавальне заняття. Серед безлічі можливостей, які надає нам цей світ, написання текстових файлів – один із найважливіших аспектів. Створення текстових файлів може бути корисним як для особистого використання, так і для професійної діяльності. У цьому блозі ми розглянемо, чому створення текстових файлів може бути корисним для вас.

## Як

Написання текстових файлів у мові програмування C є досить простим завданням. Для цього ми будемо використовувати функцію ```fprintf```, яка дозволяє нам записувати дані у заданий файл. Нижче наведені приклади коду та очікуваний результат:

Розглянемо спочатку створення нового текстового файлу:

```C
#include <stdio.h>

int main()
{
    FILE *fptr;  // оголошуємо вказівник на файл
    fptr = fopen("test.txt", "w");  // відкриваємо файл у режимі запису

    if (fptr == NULL)
    {
        printf("Помилка при відкритті файлу!");
        return 1;
    }

    // записуємо дані у файл
    fprintf(fptr, "Привіт, це тестовий файл!");

    // закриваємо файл
    fclose(fptr);

    return 0;
}
```

В результаті ми отримаємо новий текстовий файл з назвою "test.txt", у якому буде записано речення "Привіт, це тестовий файл!".

Також ми можемо зчитувати дані з існуючих текстових файлів за допомогою функції ```fscanf```:

```C
#include <stdio.h>

int main()
{
    FILE *fptr;  // оголошуємо вказівник на файл
    fptr = fopen("test.txt", "r");  // відкриваємо файл у режимі читання

    char text[20];

    // зчитуємо дані з файлу
    fscanf(fptr, "%s", text);

    // виводимо зчитаний текст
    printf("Зміст файлу: %s", text);

    // закриваємо файл
    fclose(fptr);

    return 0;
}
```

Очікуваний результат: "Зміст файлу: Привіт,".

## Глибока занурення

Створення та робота з текстовими файлами можуть бути складнішими, аніж здається на перший погляд. Наприклад, для додавання нових даних до вже існуючого файлу, можна використовувати функцію ```fputs```, а для зчитування цілого рядка – ```fgets```.
