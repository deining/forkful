# Чому

У деяких випадках програмістам необхідно видаляти символи зі строки, які відповідають певному шаблону. Наприклад, при обробці великих об'ємів даних або при використанні зовнішніх бібліотек, можуть зустрітися зайві символи, які потрібно видалити для подальшої обробки. У цьому випадку корисним буде мати знання про те, як видаляти символи за вказаним шаблоном у мові програмування C.

## Як це зробити

У мові C існує кілька способів видалення символів за певним шаблоном. Розглянемо деякі з них на прикладі використання функції `strtok()`.

```
// Приклад видалення заданого символу зі строки
#include <stdio.h>
#include <string.h>

int main()
{
    char str[] = "Привіт, світ!";

    // Видаляємо всі символи коми зі строки
    char *token = strtok(str, ",");
    while (token != NULL)
    {
        printf("%s ", token);
        token = strtok(NULL, ",");
    }
 
    return 0;
}
```

При виконанні програми в консолі ми побачимо наступний результат:

```
Привіт світ!
```

## Глибоке поглиблення

Для більш глибокого розуміння того, як відбувається процес видалення символів за шаблоном у функції `strtok()`, необхідно розібратися в механізмі роботи цієї функції. Вона шукає заданий символ у переданому їй рядку і, знаходячи його, замінює його на символ, який передається другим параметром функції. У нашому прикладі ми використовуємо кому як символ, який потрібно видалити, та передаємо в якості другого параметру порожню строку, щоб відбулася незаміненість символу.
