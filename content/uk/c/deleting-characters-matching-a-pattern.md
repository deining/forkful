## Чому

При написанні програм можуть виникати ситуації, коли необхідно видаляти символи, що відповідають певному паттерну. Наприклад, це може знадобитися для фільтрування введеної користувачем інформації або для редагування текстового файлу. У цьому пості ми розглянемо, як за допомогою мови програмування С можна видаляти символи, що відповідають певному паттерну.

## Як це зробити

Для видалення символів за певним паттерном ми можемо скористатися функцією `strchr()` з бібліотеки `<string.h>`. Ця функція шукає перший входження певного символу в рядку і повертає покажчик на цей символ. Якщо символ не знайдено, повертається покажчик на `NULL`.

Нижче наведений приклад коду, який видаляє всі літери "a" з рядку:
```C
#include <stdio.h>
#include <string.h>

int main()
{
    char string[] = "Hello World!";
    char *ptr;

    // шукаємо перше входження символу "a" в рядку
    ptr = strchr(string, 'a');

    // доки не знайдено кінець рядка
    while(ptr != NULL)
    {
        // перезаписуємо рядок, пропускаючи символ "a"
        strcpy(ptr, ptr + 1);

        // знову шукаємо перше входження символу "a" в рядку
        ptr = strchr(string, 'a');
    }

    printf("%s", string); // виводимо остаточний результат: "Hllo World!"

    return 0;
}
```
Можна помітити, що ми використовуємо цикл `while`, оскільки функція `strchr()` повертає покажчик на перший вхідний символ. Це означає, що ми повторюємо процес видалення символу "a" до тих пір, поки він знаходиться в рядку.

## Глибше копаемо

Видалення символів за певним паттерном можна реалізувати і за допомогою функцій `strlen()` та `memmove()` з бібліотеки `<string.h>`. Функція `strlen()` повертає довжину рядка, а `memmove()` копіює пам'ять з одного місця в інше, запобігаючи накладанню пам'яті.

Оскільки функція `strchr()` відшукують лише перший вхідний символ, нам необхідно буде перевіряти всі символи у рядку. Тому ми використовуємо цикл `for` для проходження через кожен символ рядка. Якщо символ відповідає заданому паттерну, ми використовуємо функцію `memmove()`
