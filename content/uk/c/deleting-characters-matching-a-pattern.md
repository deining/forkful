## Що і чому?
У програмуванні часто виникає потреба у видаленні символів, що відповідають певному шаблону. Наприклад, може знадобитись видалити усі цифри з рядка або усі спеціальні символи з файлу. Існує багато ситуацій, коли видалення символів за певними правилами є необхідним для правильної роботи програми.

## Як це зробити:
```C
#include <stdio.h>
#include <string.h>

int delete_chars(char str[], char pattern[]) {
    // клонуємо рядок str в temp
    char temp[strlen(str) + 1];
    strcpy(temp, str);

    // перевіряємо кожен символ рядка на відповідність заданому шаблону
    for (int i = 0; i < strlen(temp); i++) {
        if (strchr(pattern, temp[i]) != NULL) {
            // видаляємо символ, якщо він відповідає шаблону
            memmove(&temp[i], &temp[i + 1], strlen(temp) - i);
            i--;
        }
    }

    // копіюємо результат назад у вихідний рядок
    strcpy(str, temp);
    return 0;
}

int main() {
    // приклад видалення всіх цифр з рядка
    char str[] = "He11o w0rld";
    char pattern[] = "0123456789";
    delete_chars(str, pattern);
    printf("%s", str); // виведе "Heo wrld"
    return 0;
}
``` 

## Глибокий погляд:
Проблема видалення символів за певним шаблоном сутнісна для багатьох мов програмування. У C це зазвичай вирішується шляхом перебору символів рядка та виконання певних дій в залежності від відповідності символу заданому шаблону. У деяких мовах програмування, таких як Python, існують спеціальні методи для видалення символів за певними умовами, що спрощує завдання. Також існують стандартні функції у бібліотеках C, наприклад `fscanf()`, які можуть виконувати подібні завдання.
