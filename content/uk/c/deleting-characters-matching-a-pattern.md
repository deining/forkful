## Чому

Зазвичай, причиною видалення символів, що відповідають певному зразку, є необхідність убезпечити ввід користувача від небажаних значень. Це може бути корисно при написанні програм, які працюють з великими обсягами даних або прикладих системах, де користувачі вводять дані у визначеному форматі.

## Як це зробити

Для того, щоб видалити символи, що відповідають певному зразку, необхідно використати функцію `strpbrk()`. Нижче наведений код на мові С, що демонструє як це працює:

```C
#include <stdio.h>
#include <string.h>

int main() {
    char string[50] = "Hello, world!";
    char pattern[10] = "lo";
    char *ptr;

    ptr = strpbrk(string, pattern);

    while (ptr != NULL) {
        *ptr = '\0';
        ptr = strpbrk(string, pattern);
    }

    printf("Результат: %s", string);

    return 0;
}
```

Виведення програми буде `He, wrd!`, оскільки всі символи, що відповідають зразку `lo`, будуть замінені на пусту строку.

## Глибока розробка

Процес видалення символів можна розглянути докладніше. Функція `strpbrk()` приймає два аргументи - перший це рядок, в якому потрібно шукати зразок, а другий - рядок зі зразком. Якщо жодний з символів зі зразку не зустрічається у рядку, то функція поверне `NULL`.

У наступному кроці необхідно перевірити, чи є результат, повернутий функцією, не рівним `NULL` і продовжувати цикл, доки не буде знайдений останній символ зі зразку. Коли останній символ буде знайдений, замінимо його на пусту строку і повторимо процес пошуку і заміни. Таким чином, всі символи, що відповідають зразку, будуть вилучені з рядку.
