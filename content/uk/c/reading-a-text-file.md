## Чому:
Текстові файли є необхідною частиною кожної програми, оскільки вони дозволяють зберігати та оброблювати велику кількість даних. Якщо ви хочете покращити свої навички програмування на мові C, читання та обробка текстових файлів є важливою навичкою, яку ви повинні засвоїти.

## Як:
Для того, щоб читати текстовий файл на мові C, вам потрібно виконати кілька основних кроків. Спочатку, необхідно відкрити файл за допомогою функції `fopen()`, яка приймає два параметри - шлях до файлу та режим відкриття. Наприклад, якщо ви хочете прочитати файл з назвою "test.txt", то код буде виглядати наступним чином:

```C
FILE *ptr;
ptr = fopen("test.txt", "r");
```

Потім вам необхідно створити буфер для зберігання даних, які ви збираєтесь прочитати з файлу. Для цього використовуйте функцію `fgets()`, яка приймає три параметри - буфер, розмір буфера та вказівник на файл. Наприклад:

```C
char buffer[255];
fgets(buffer, 255, ptr);
```

Останнім кроком буде закриття файлу за допомогою функції `fclose()`, яка приймає вказівник на файл:

```C
fclose(ptr);
```

## Поглиблене дослідження:
Крім основних функцій для читання текстових файлів, на мові C є ще кілька корисних функцій, які допомагають зчитувати та оброблювати дані. Одна з них - `fgetc()`, яка дозволяє читати посимвольно з файлу та повертати цей символ. Також у мові C є функція `fscanf()`, яка дозволяє читати дані з файлу та зберігати їх у бажаних змінних. Ця функція корисна, наприклад, для зчитування чисел з файлу.
