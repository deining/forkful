## Що і навіщо?

Читання текстового файлу полягає в тому, щоб програма могла взаємодіяти з вмістом цього файлу. Перевага читання текстового файлу полягає в тому, що це дозволяє програмістам працювати зі структурованою інформацією та обробляти її за допомогою програмних засобів.

## Як:

Приклад коду для читання текстового файлу виглядає наступним чином:

```
#include <stdio.h>
#include <stdlib.h>

int main() {
    // Відкриття файлу для читання
    FILE *fp;
    fp = fopen("myfile.txt", "r");

    // Перевірка, чи файл був успішно відкритий
    if (fp == NULL) {
        printf("Помилка при відкритті файлу!");
        exit(1);
    }

    // Читання та виведення вмісту файлу за допомогою циклу
    char c;
    while((c = fgetc(fp)) != EOF) {
        printf("%c", c);
    }

    // Закриття файлу
    fclose(fp);
    return 0;
}
```

Приклад виводу програми для файлу, що містить текст "Привіт світе!":

```
Привіт світе!
```

## Глибоке поглиблення:

Історичний контекст: читання текстових файлів було однією з перших технологій в програмуванні та досі залишається важливою для багатьох застосунків.

Альтернативи: крім читання за допомогою звичайних програмних засобів, таких як у прикладі вище, існує також можливість використання бібліотек або фреймворків для більш ефективного та розширеного читання файлів.

Деталі реалізації: перед читанням файлу, програма повинна його відкрити та передати у вказівник FILE, а після закінчення використовувати функцію fclose() для закриття файлу. Читання виконується за допомогою функції fgetc(), яка послідовно зчитує один символ з файлу.
