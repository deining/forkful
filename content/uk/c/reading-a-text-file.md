# Для чого
Програмування - це не тільки про створення громіздких програм та складних алгоритмів. Іноді, просто читання текстового файлу може бути дуже корисним для вас. Наприклад, ви можете хотіти зчитати дані з файлу для подальшої обробки або відображення на екрані. Читання текстового файлу є важливою частиною багатьох програм, отож давайте розглянемо, як це зробити у мові С.

## Як це зробити
Щоб прочитати текстовий файл у мові С, нам потрібно використати функцію `fopen()` для відкриття файлу та отримання вказівника на нього. Наприклад, якщо ми хочемо відкрити файл з назвою "input.txt", ми можемо написати такий код:

```C
FILE *file; // оголошуємо змінну для вказівника на файл
file = fopen("input.txt", "r"); // відкриваємо файл для читання
```

Після цього, ми можемо використати функцію `fgets()` для поетапного читання рядків з файлу. Наприклад, ми можемо вивести на екран усі рядки з файлу за допомогою циклу `while`:

```C
char buffer[255]; // оголошуємо буфер для зберігання прочитаних рядків
while (fgets(buffer, 255, file) != NULL) { // читаємо рядки з файлу, доки не досягнемо кінця
  printf("%s", buffer); // виводимо рядок на екран
}
```

І не забудьте закрити файл за допомогою функції `fclose()` після того, як ви закінчите читати його.

## Розглиблене вивчення
Читання текстових файлів насправді може бути дуже складним іноді. Наприклад, якщо ваш файл має складну структуру з вбудованих роздільників або потребує складного алгоритму для обробки, то вам, можливо, знадобиться додаткове дослідження та експериментування для того, щоб запрограмувати правильне читання файлу.

Також, якщо ви часто працюєте з великими текстовими файлами, вам може знадобитися вивчити спеціальні техніки для роботи з ними, наприклад, читання файлу по частинах, щоб уникнути перевантаження пам'яті.
