# Чому

Програмування - це універсальний інструмент, який дозволяє нам створювати різноманітні програми і застосунки для власних потреб. Читання текстового файлу - це один із важливих аспектів програмування, особливо в мові програмування С. Це дозволяє нам зчитувати і обробляти дані з текстових файлів, що є дуже корисним у різних сценаріях програмування, таких як обробка даних або створення звітів. Тому, якщо ви зацікавлені в поглибленні своїх знань у програмуванні на С, читання текстових файлів є важливим навичкою для вас.

# Як

Для того, щоб зчитати текстовий файл у мові програмування С, нам спочатку потрібно відкрити файл за допомогою функції `fopen()`. У цій функції ми повинні вказати шлях до файлу та режим відкриття, наприклад "r" для читання файлу. Далі, ми можемо використовувати функцію `fgets()` для зчитування рядка з файлу. Приклад коду наведено нижче:

```C
#include <stdio.h>

int main()
{
   FILE *fp;
   char str[100];

   fp = fopen("text.txt", "r");
   if(fp == NULL) //Перевіряємо чи файл вдалося відкрити
   {
      perror("Помилка в операції з відкриття файлу");
      return 1;
   }

   //Зчитуємо рядок із файла та виводимо його на екран
   while(fgets(str, 100, fp) != NULL) 
   {
      printf("%s", str);
   }

   fclose(fp); //Обов'язково закриваємо файл після використання
   return 0;
}
```

Вхідним файлом для цього прикладу є `text.txt`, який містить:

```
Привіт!
Ласкаво просимо до мого блогу.
Сьогодні ми розглянемо читання текстового файлу у мові програмування С.
```

Вивід програми буде наступним:

```
Привіт!
Ласкаво просимо до мого блогу.
Сьогодні ми розглянемо читання текстового файлу у мові програмування С.
```
