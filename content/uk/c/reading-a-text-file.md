## Чому

В програмуванні мовою C, часто виникає необхідність прочитати дані з текстового файлу. Це може бути корисно для обробки великої кількості даних або для збереження інформації між запусками програми. Якщо ви цікавитесь основами програмування на C, ця стаття надасть вам необхідні знання для роботи з текстовими файлами в цій мові.

## Як

Для того, щоб прочитати текстовий файл у програмі на C, ви можете використовувати функцію `fopen()`, яка відкриває файл за заданим шляхом і повертає вказівник на файловий потік. Після цього, використовуйте функції `fgets()` або `fscanf()` для зчитування даних з файлу і збереження їх у змінній. Наприклад:

```C
#include <stdio.h>

int main() {
    FILE *file;
    char line[100];

    file = fopen("data.txt", "r"); // відкриваємо файл для читання
    
    if (file == NULL) { // перевіряємо чи відкрилось файл
        printf("Помилка при відкритті файлу.");
        return 1;
    }
    
    while (fgets(line, 100, file) != NULL) { // зчитуємо рядок з файлу і зберігаємо його у змінну line
        printf("%s", line); // виводимо рядок на екран
    }
    
    fclose(file); // закриваємо файл
    return 0;
}
```

При використанні функції `fgets()`, ви можете зчитати рядок з файлу і зберегти його у змінній `line`. У прикладі, цей рядок виводиться на екран за допомогою функції `printf()`. Крім того, ви можете використовувати функцію `fscanf()` для зчитування даних з файлу, форматування їх і збереження у вказані змінні.

## Глибоке занурення

У стандартній бібліотеці C є багато функцій для роботи з текстовими файлами. Наприклад, функція `fprintf()` дозволяє записувати дані у файл, а функція `fseek()` - переміщатись у файлі до певної позиції.

Також, важливо пам'ятати про закриття файлу за допомогою функції `fclose()`. Якщо ви не закриєте файл після завершення роботи з ним, це може призвести до проблем у подальшій роботі з файлами.
