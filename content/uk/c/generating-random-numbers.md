## Що і чому?

Генерування випадкових чисел - це процес, коли програма створює числа випадковим чином. Програмісти можуть використовувати цей процес для створення випадкових елементів в своїх програмах, наприклад, для генерації випадкових паролів або випадкового вибору елементів зі списку.

## Як це зробити:

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
  // встановлюємо генератор випадкових чисел на основі поточного часу
  srand(time(NULL));
  
  // генеруємо випадкове ціле число від 1 до 10
  int random_num = rand() % 10 + 1;
  
  printf("Випадкове число: %d\n", random_num);
  
  return 0;
}
```

**Висновок:**
`Випадкове число: 7`

## Глибше:

Генерування випадкових чисел використовується в програмуванні здавна і має свої альтернативи, такі як використання алгоритмів шифрування чи фізичних процесів (наприклад, шуму з оточуючого середовища). У мові C використовується функція `rand()`, якій передається зерно (seed), щоб забезпечити випадковість. Якщо зерно не передано, воно встановлюється за замовчуванням на 1.
