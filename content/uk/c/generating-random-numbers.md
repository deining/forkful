## Чому

Генерація випадкових чисел є важливою завданням у багатьох програмах. Випадкові числа можуть бути використані для створення унікальних ідентифікаторів, шифрування даних або проведення імітаційних експериментів. Також, випадкові числа можуть додати елемент непередбачуваності та цікавості до програми.

## Як

Генерація випадкових чисел в C зазвичай виконується за допомогою функції `rand()` та `srand()`. Ці функції використовують псевдовипадкове число, що залежить від seed (початкового значення). Щоб гарантувати унікальність випадкових чисел, необхідно встановити початкове значення залежно від поточного часу, наприклад, `srand(time(NULL))`.

Для генерації випадкових чисел у певному діапазоні, використовуйте вираз `rand() % (максимальне_значення - мінімальне_значення) + мінімальне_значення`. Наприклад, для генерації випадкового числа від 1 до 10, використовуйте `rand() % 10 + 1`.

Нижче наведений приклад коду, який генерує 5 випадкових чисел в діапазоні від 1 до 100 та виводить їх на екран:

```C
#include <stdio.h> 
#include <stdlib.h> 
#include <time.h> 
  
int main() 
{
    // встановлює початкове значення для генерації унікальних випадкових чисел
    srand(time(NULL)); 

    // генерація та виведення 5 випадкових чисел 
    for (int i = 0; i < 5; i++) { 
        int num = rand() % 100 + 1; 
        printf("%d\n", num); 
    } 
  
    return 0; 
}
```

Приклад виводу:

```
57
23
42
96
10
```

## Deep Dive

За замовчуванням, функція `rand()` генерує псевдовипадкові числа у діапазоні від 0 до `RAND_MAX` (зазвичай 32767). Невеликі значення `RAND_MAX` можуть призводити до повторення випадкових чисел після деякої кількості ітерацій.
