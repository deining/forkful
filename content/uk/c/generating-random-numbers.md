## Чому

Генерація випадкових чисел є важливою складовою у багатьох програмах та алгоритмах. Вона може бути використана для створення унікальних ідентифікаторів, шифрування даних та імітації різних сценаріїв. Тому, знання як генерувати випадкові числа є важливим для будь-якого програміста.

## Як

Для генерації випадкових чисел використовується функція `rand()` з бібліотеки `stdlib.h`. Ця функція повертає випадкове число від 0 до `RAND_MAX`, яке може бути змінене за допомогою функції `srand()`, вказавши нове початкове значення.

```C
#include <stdio.h>
#include <stdlib.h>

int main(){
    srand(42); // встановлення початкового значення
    int i;
    for(i=0; i<10; i++){
        printf("%d ", rand()); // друкуємо 10 випадкових чисел
    }
    return 0;
}

/*
Вивід:

1608637542 342073347 304797350 41456223 1566760313 1289211570
1611663981 1701803258 1652202955 1945323639
*/
```

## Глибше

Хоча за допомогою функції `rand()` можна отримати випадкові числа, вони будуть повторюватися при кожному запуску програми, оскільки початкове значення завжди буде однакове. Тому, щоб отримати більш непередбачувані результати, можна використовувати більш складні формули для генерації чисел.

Наприклад, для створення псевдовипадкових чисел за алгоритмом Лемера, можна використовувати наступну формулу:

```C
long a = 1103515245;
long c = 12345;
long m = 2147483648;
long x = 42; // початкове значення
...
x = (a*x + c)%m;
```

За допомогою цього алгоритму, можна отримати значення від 0 до `m-1`, які вже не будуть повторюватися при кожному запуску програми.
