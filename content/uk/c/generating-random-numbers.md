## Чому

Генерація випадкових чисел є важливою частиною багатьох програм. Випадкові числа можуть бути використані для створення випадкових подій, наукових досліджень або навіть для захисту інформації. У цьому блозі ми розглянемо, як згенерувати випадкові числа у мові програмування C та розберемося з тим, як вони працюють.

## Як

Для генерації випадкових чисел у мові програмування C використовуються функції з бібліотеки `stdlib.h`. Одна з найпоширеніших функцій - це `rand()`, яка повертає випадкове число від 0 до `RAND_MAX`. Для отримання цього числа необхідно спочатку ініціалізувати генератор випадкових чисел за допомогою функції `srand()`. При розділенні числа часом використовують функцію `time()` для створення стартового значення. Давайте подивимося на приклад:

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>
  
int main()
{
  // ініціалізуємо генератор випадкових чисел
  srand(time(0));
  
  // генеруємо випадкове число і виводимо його
  int random = rand() % 100; // отримаємо число від 0 до 99
  printf("Ваше випадкове число: %d\n", random);
  
  return 0;
}
```

Вихідні дані можуть виглядати приблизно так:

```
Ваше випадкове число: 73
```

Це простий приклад, але у вас є можливість експериментувати з функціями `rand()` та `srand()` та створювати власні способи генерації випадкових чисел.

## Глибоке погруження

Усе не так просто, як здається на перший погляд. Фактично, генератор випадкових чисел не може створювати повністю випадкові числа, оскільки він працює за певним алгоритмом. Це означає, що при достатньої кількості запусків генератора можна передбачити наступні випадкові числа. Тому, якщо безпечність вашої програми залежить від випадкових чисел, вам потрібно ретельно вибирати генератор та ініціалізувати його за допомогою дійсно випадкового змінного.
