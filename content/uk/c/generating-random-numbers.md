#Чому
Генерація випадкових чисел є важливою в будь-якому веб-сайті або програмі. Вона допомагає створювати різноманітні сценарії та забезпечує більш креативний та динамічний досвід для користувачів.

##Як це зробити
Для генерації випадкових чисел використовуються функції rand() та srand() в мові програмування C. Рандомні числа можна генерувати в заданому інтервалі, що дозволяє їх використання для створення різних сценаріїв та ігор.

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    int i, n;
    /* генерування seed для випадкових чисел */
    srand(time(0)); 

    /* генерування 10 випадкових чисел від 1 до 100 */
    for (i = 0; i < 10; i++) {
        n = (rand() % 100) + 1;
        printf("%d\n", n);
    }
    return 0;
}
```

Вивід програми може виглядати так:

```bash
87
41
5
63
72
11
29
14
98
23
```

##Глибокий погляд
Генерація випадкових чисел базується на алгоритмах, що використовуються в програмуванні. Ці алгоритми допомагають створити невпорядковану послідовність чисел, які виглядають випадковими для користувача. Однак, вони базуються на строгих математичних принципах та можуть бути передбачувані при достатньо великій кількості повторних генерацій.
