## Чому

Створення програм, які здатні обробляти тексти, є важливою частиною багатьох проектів. Іноді нам потрібно отримувати лише частину тексту, а не весь рядок. В таких випадках, виникає потреба в видобуванні підрядків або ж підрядків. Це корисно, коли нам потрібно отримати певну частину і використати її для подальшої обробки. В цій статті ми розглянемо, як витягувати підрядки в мові С.

## Як це зробити

У мові С є кілька способів видобування підрядків з рядка. Перший спосіб - використання функції `strcpy`, яка копіює підрядок у окремий масив символів. Ось приклад коду:

```C
char str[] = "Це прикладний рядок";
char substr[10];
strcpy(substr, str + 4); // видобуваємо підрядок починаючи з 4-го символу
printf(substr); // виводиться 'прикладний'
```

Другий спосіб - використання бібліотечної функції `strncpy`, яка копіює не більше заданої кількості символів. Приклад коду:

```C
char str[] = "Це прикладний рядок";
char substr[10];
strncpy(substr, str + 10, 5); // видобуваємо підрядок починаючи з 10-го символу довжиною 5 символів
substr[5] = '\0'; // додаємо термінатор рядка для коректного виводу
printf(substr); // виводиться 'рядок'
```

Також, ми можемо використовувати оператори зчеплення символів `[]` та `*` для доступу до підрядка. Приклад коду:

```C
char str[] = "Це прикладний рядок";
printf("%s", str + 7); // виводиться 'рий рядок'
```

## Глибоке уточнення

Виявляється, що у мові С немає вбудованого типу для представлення рядків, тому отримані результати залежать від використовуваного типу. Наприклад, при використанні `char *` для зберігання рядка, який був створений з літералів, підрядок також буде мати тип `char *`. Однак, якщо ми використовуємо масив символів `char[]`, то підрядок буде мати тип `char[]`. Тому, коли працюємо з підрядком, потрібно приділяти увагу типам, щоб уникнути непередбачуваних помилок.
