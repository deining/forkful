# Чому

У програмуванні, часом може знадобитися витягти підстрічки зі строки, наприклад, для подальшої обробки чи збереження у змінну. Ця необхідність може виникнути під час роботи з рядками, які містять багато інформації, але для потрібної функціональності нам знадобиться лише частина цієї інформації. Також, витягнення підстрічки може бути корисним для роботи зі специфічними даними, такими як номери телефонів, e-mail адреси тощо.

# Як

Для початку, нам необхідно визначити початкову строку, з якої будемо витягувати підстрічку, та індекси початку та кінця підстрічки, яку ми хочемо витягнути. Для цього використовуємо функцію `strncpy()`, яка копіює задану кількість символів з однієї строки в іншу. Приклад:

```C
char str[] = "Hello World";
char subString[6];

strncpy(subString, str + 6, 5);
printf("SubString: %s", subString);

// Output: World
```

У цьому прикладі ми витягуємо підстрічку "World" починаючи з 6-ого символу (індекс 6) зі строки "Hello World". Для деяких задач може бути корисно також використовувати функцію `strncat()`, яка додає підстрічку до існуючої строки.

# Глибше

Розглянемо докладніше функцію `strncpy()`. Вона приймає три аргументи: вказівник на строку-приймач, вказівника на джерело підстрічки та кількість символів для копіювання. Перші два аргументи потрібно зазначати відповідно до типу `char*` (вказівник на символ) або `char[]` (масив символів). Останній аргумент приймає ціле число, яке вказує, скільки символів потрібно скопіювати з джерела. Якщо кількість символів, що потрібно скопіювати, більша за довжину джерела, тоді у підстрічці будуть записані лише доступні символи, а решта буде заповнена нулевими значеннями.

Також варто зазначити, що функція `strncpy()` не додає заключного символу нуль-термінатора `\0` до результуючої строки, тому якщо ми хочемо використовувати її для створення нової строки, необхідно додати цей символ вручну
