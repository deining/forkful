## Чому

У цій статті ми зосередимось на витягуванні підрядків (substrings) в мові програмування C. Якщо ви ще не знайомі з цією технікою, то ця стаття допоможе вам зрозуміти її важливість та навчитися використовувати її в своїх програмах.

## Як

Для витягування підрядків використовується функція `strncpy`, яка копіює певну кількість символів з одного рядка в інший. Також доцільно використовувати функцію `strlen` для визначення кількості символів у рядку. Нижче наведено приклад коду, який використовує ці функції:

```C
// Приклад коду для витягування підрядка

#include <stdio.h>
#include <string.h>

int main() {
    char source_str[] = "Це прикладний текст для демонстрації";
    char dest_str[20];
    int start_point = 10; // Початкова позиція підрядка
    int length = 7; // Кількість символів у підрядку, який ми хочемо витягнути

    strncpy(dest_str, source_str + start_point, length);
    dest_str[length] = '\0'; // Додатково додаємо завершальний символ

    printf("%s", dest_str);
    
    return 0;
}

// Вивід: текст для
```

Як бачимо, ми використовуємо функцію `strncpy`, а в результаті отримуємо підрядок "текст для", починаючи з 10 символу.

## Поглиблене дослідження

Витягування підрядків є важливою технікою у програмуванні. Використання цієї функції дозволяє зробити роботу з рядками більш ефективною та здатною до автоматизації. Також за допомогою цієї техніки можна зберігати багатомовний текст у різних мовах у один рядок та з легкістю витягувати потрібну інформацію.
