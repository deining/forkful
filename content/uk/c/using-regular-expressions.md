## Чому

Іноді під час програмування значення вхідних даних може бути невідомим або незрозумілим. У таких випадках дуже корисно мати засіб, що дозволить виконувати пошук та заміну символів швидко та ефективно. Тут на допомогу можуть прийти регулярні вирази.

## Як?

```C
#include <stdio.h>
#include <regex.h>

int main() {
    char text[20] = "Hello, World!";
    regex_t regex;
    int status;

    status = regcomp(&regex, "World", 0);
    status = regexec(&regex, text, 0, NULL, 0);
    if (status == 0) {
        printf("Знайдено збіг!\n");
    } else {
        printf("Збіг не знайдено!\n");
    }

    return 0;
}
```

В результаті виконання цього фрагмента коду ми отримаємо наступний вивід:

```
Знайдено збіг!
```

Також регулярні вирази підтримують багато інших операцій, таких як заміна певних символів або вилучення певних частин тексту.

## Глибокий погляд

Регулярні вирази - це вираження, що зберігають шаблон для пошуку або заміни в тексті. Вони дозволяють виконувати пошук і заміну символів за допомогою різних спеціальних символів та символьних класів. Наприклад, символ "*" означає будь-яку кількість будь-яких символів, а клас "[0-9]" - будь-який цифровий символ.

Регулярні вирази можуть бути дуже корисними для фільтрації та обробки великих обсягів тексту. Вони підтримуються більшістю мов програмування, включаючи С, що робить їх потужним інструментом для багатьох програмістів.
