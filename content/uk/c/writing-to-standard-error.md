## Чому

Написання до стандартного помилкового потоку є важливою частиною процесу розробки програмного забезпечення. Це дозволяє вам відстежувати помилки та відлагоджувати свої програми в більш ефективний спосіб.

## Як

Для написання до стандартного помилкового потоку використовується функція `fprintf()`. Давайте розглянемо приклад коду нижче, який демонструє її використання:

```C
#include <stdio.h>

int main() {
    // Виводимо повідомлення до стандартного помилкового потоку
    fprintf(stderr, "Помилка: така змінна не існує!");
    return 0;
}
```

Виходячи з цього прикладу, рядок `"Помилка: така змінна не існує!"` буде виведено до стандартного помилкового потоку, який зазвичай відображається на екрані або записується у файл.

## Глибока поглиблення

Надалі, ви можете налаштувати направлення стандартного помилкового потоку, використовуючи функцію `freopen()`. Наприклад, якщо ви хочете записувати помилки у окремий файл замість виведення їх на екран, ви можете використати наступний код:

```C
#include <stdio.h>

int main() {
    // Вказуємо маршрут до файлу, де будуть записуватись помилки
    freopen("errors.txt", "w", stderr); 

    // Виводимо повідомлення до стандартного помилкового потоку
    fprintf(stderr, "Помилка: така змінна не існує!");
    return 0;
}
```

У цьому випадку, рядок `"Помилка: така змінна не існує!"` буде записаний у файл `errors.txt` замість виведення на екран.
