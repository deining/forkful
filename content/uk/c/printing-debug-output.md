## Чому

Якщо ви коли-небудь в програмуванні стикалися з помилками і не розуміли їх причини, то, напевно, вам доводилося використовувати дебаг-виведення для пошуку і виправлення помилок. В даній статті ми розглянемо, як саме це виконується в мові програмування C.

## Як це зробити

Для того, щоб виводити відладочну інформацію в C, необхідно використовувати функцію `printf()` зі спеціальними форматувальними рядками. Нижче наведено приклад коду, який виводить значення змінних `a` та `b`:

```C
#include <stdio.h>

int main() {
    int a = 10;
    int b = 20;
    printf("Значення a: %d\n", a);
    printf("Значення b: %d\n", b);
    return 0;
}

// Вивід:
// Значення a: 10
// Значення b: 20
```

Як бачимо, у форматуючому рядку можна вказувати тип даних та спеціальні прапорці для виведення. Більш детально про них можна дізнатися у нашій Deep Dive секції.

## Глибокий занурення

Функція `printf()` дозволяє використовувати різні прапорці для виведення різних типів даних. Наприклад, для виведення значення змінної типу `double`, можна використовувати `%f`, для рядку - `%s`, а для символу - `%c`. Також є можливість вказувати кількість десяткових знаків та розмір поля виведення.

Крім того, ви можете використовувати спеціальний прапорець `%p` для виведення адреси змінної. Це дуже корисно при відлагодженні коду, коли потрібно відстежувати зміни значень за адресами.

Детальніше про форматуючі рядки та їх можливості можна дізнатися у документації зі специфікації C.
