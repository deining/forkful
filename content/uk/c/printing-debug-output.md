## Що & Чому?

Друк виходу для налагодження - це процес виведення інформації про поточний стан програми або виконання деяких дій під час її роботи. Програмісти користуються цим методом для відстеження помилок, виконання перевірок та випробувань коду.

## Як:

```C
// Приклад коду для друку рядка відладкового повідомлення
printf("Це відладочне повідомлення зі значенням рядка: %s", "Вітаємо!");
```

Вихід: `Це відладочне повідомлення зі значенням рядка: Вітаємо!`

```C
// Приклад коду для друку цілочисельного відладкового повідомлення
int x = 10;
printf("Значення змінної x = %d", x);
```

Вихід: `Значення змінної x = 10`

## Глибокий занурення:

Друк виходу для налагодження походить ще зі старих часів, коли програмісти використовували вивід на екран або використовували спеціальні пристрої для друку на папері. Сьогодні існує багато альтернатив для друку виходу, наприклад, використання спеціальних програм для налагодження або журналів виходу. У деяких мовах програмування також є вбудовані функції для друку виходу, наприклад, `console.log()` у JavaScript. В C програмісти можуть використовувати функцію `printf()` для друку виходу на екран.
