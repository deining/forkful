## Чому
В програмуванні дуже важливо виводити різну інформацію для відладки програми. Це дозволяє нам зрозуміти, як працює наш код і виявити помилки, що допущені під час написання програми.

## Як це зробити
Для виведення дебаг-інформації використовується функція `printf()`. Нижче наведени приклад коду, де використовується ця функція для виведення змінної `x` на екран.

```C
#include <stdio.h>

int main(){
    int x = 10;
    printf("Змінна x має значення: %d", x);
    return 0;
}
```
Результат виконання програми буде наступним:

```
Змінна x має значення: 10
```

Також можна використовувати функцію `fprintf()` для виведення інформації в файл замість стандартного виводу на екран.

## Глибша розробка
Крім виведення значень змінних, `printf()` може також виводити різні формати даних, такі як рядки, символи, десяткові числа та інші. Для цього потрібно використовувати відповідні спеціфікатори формату. Наприклад, `%s` - для виведення рядків, `%c` - для символів, `%f` - для дробових чисел тощо.

Також можна використовувати різні параметри форматування, такі як вирівнювання, кількість знаків після десяткової коми та інші.
