## Чому

Написання імен змінних та функцій з великої літери є одним із стандартів доброго стилю програмування. Це полегшує читання коду і покращує зрозумілість написаного. Також це є важливим при написанні програм, які використовують різні мови.

## Як

Одним зі способів зробити всі літери у рядку великими є використання функції `toupper()`, яка входить до стандартної бібліотеки Си. Наприклад:
```C
#include <stdio.h>
#include <ctype.h>

int main() {
    char str[] = "hello world";
    for (int i = 0; str[i] != '\0'; i++) {
        str[i] = toupper(str[i]);
    }
    printf("%s", str);
    return 0;
}
```
Результат виконання цього коду буде:
```C
HELLO WORLD
```

## Глибокий пір

Функція `toupper()` базується на таблицях перетворення, тому вона працює тільки зі стандартними літерами англійського алфавіту. Якщо необхідно приводити до великих літер рядки з іншими мовами, то можна використати функцію `toupper_l()` і передати в неї потрібну локаль за допомогою функції `setlocale()`. Також можна написати свою функцію для переведення рядка до верхнього регістру, враховуючи потрібні символи і їх коди.
