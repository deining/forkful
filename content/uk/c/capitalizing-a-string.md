## Чому

Навіщо взагалі займатися перетворенням рядків до верхнього регістру? Деякі завдання, такі як пошук і порівняння рядків, вимагають, щоб всі букви в рядку були одного регістру. Також це може бути корисно для створення чіткого і однорідного вигляду при виведенні даних.

## Як

```C
#include <stdio.h>
#include <string.h>

void capitalizeString(char *str) {
    int i;

    for (i = 0; i < strlen(str); i++) {
        // перевіряємо чи символ є малою літерою
        if (str[i] >= 'a' && str[i] <= 'z') {
            // перетворюємо його до великої літери за допомогою ASCII коду
            str[i] = str[i] - 32; 
        }
    }
}

int main() {
    // створюємо рядок для тестування
    char str[] = "Hello, world!";
    
    // викликаємо функцію capitalizeString
    capitalizeString(str);

    // виводимо результат
    printf("%s\n", str); // HELLO, WORLD!

    return 0;
}
```

## Глибшої погрузки

Є багато способів для перетворення рядків до верхнього регістру. Наприклад, можна використовувати функції з бібліотеки <string.h> такі як `toupper()` або `strlwr()`. Також можливо використати цикл `while` замість циклу `for` для перевірки кожного символа рядка.
