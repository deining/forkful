## Чому

У програмуванні часто потрібно приводити рядки до великих літер, наприклад, для уніфікації формату вводу даних або для виведення користувачу зрозумілої інформації. Така дія називається капіталізацією рядка і є важливою навичкою для кожного програміста.

## Як це зробити

```C
#include <stdio.h>
#include <string.h>

int main()
{
   // оголошення та ініціалізація рядка
   char str[20] = "програмування";

   // використання стандартної функції toUpperCase для капіталізації рядка
   strupr(str);

   // виведення результату
   printf("Результат: %s\n", str);

   return 0;
}
```

## Глибше занурення

Існує кілька шляхів капіталізувати рядок в C програмі. Найбільш ефективним є використання стандартної функції `strupr`, яка змінює всі символи в рядку на великі літери. Також можна використовувати цикл, у якому перебирати кожен символ рядка і змінювати його на велику літеру, використовуючи функцію `toupper`. Важливо пам'ятати, що при використанні циклу потрібно перевіряти наявність кожного символу в рядку та враховувати особливі символи, наприклад, пробіли чи знаки пунктуації.
