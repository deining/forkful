## Чому

Привіт всім програмістам! Сьогодні ми розглянемо одну з найпростіших, але в той же час важливих операцій у C-програмуванні - насамперед, про те, як перетворити рядок так, щоб всі символи були написані великими літерами. Звісно, іноді це може здатися трохи банальним або незначним, але насправді це корисна та необхідна операція, наприклад, при роботі зі строками, зчитаними з файлу або отриманими від користувача.

## Як

Існує кілька способів написати код для перетворення рядка у верхньому регістрі, але найпростішим та найбільш зрозумілим є використання стандартної функції "toupper". Для цього нам потрібно включити заголовочний файл <stdio.h> у наш код та використати цю функцію у наступний спосіб:

```C
#include <stdio.h>

int main() 
{
    char str[100];
    int i;
    
    // отримуємо рядок від користувача
    printf("Введіть рядок: ");
    scanf("%s", str);
    
    // перетворюємо кожен символ у верхній регістр і виводимо результат
    for(i = 0; str[i] != '\0'; i++) 
    {
        printf("%c", toupper(str[i]));
    }
    
    return 0;
}
```

При введенні рядка "Програмування" ви отримаєте на виході "ПРОГРАМУВАННЯ".

## Глибше

Варто відзначити, що функція "toupper" не міняє вихідний рядок, а створює копію рядка у верхньому регістрі. Також слід звернути увагу на те, що ця функція працює тільки з літерами латинського алфавіту, тому для перетворення рядка українськими літерами потрібно використовувати інші методи.

Ще одним цікавим моментом є використання функції "tolower", яка, як можна здогадатися, перетворює всі символи у нижній регістр. Для цього потрібно замінити "toupper" на "tolower" у нашому прикладі.
