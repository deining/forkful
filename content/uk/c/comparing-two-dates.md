## Чому

Часто в програмуванні потрібно порівнювати дати. Наприклад, перевірити, який з двох кварталів завершився раніше або вивести різницю між двома датами. У цьому статті ми розглянемо, як це можна зробити за допомогою мови програмування C.

## Як

Для порівняння двох дат використовується функція `difftime()`, яка знаходить різницю між двома часовими значеннями. Для цього потрібно використати бібліотеку `time.h`.

Нижче наведено приклад коду, який порівнює дві дати та виводить різницю у днях:

```C
#include <stdio.h>
#include <time.h>

int main()
{
    // Дві дати, які будемо порівнювати
    time_t date1 = 1565844000; // 15 серпня 2019 року
    time_t date2 = 1514761200; // 1 січня 2018 року

    // Використовуємо функцію difftime() для порівняння
    double difference = difftime(date1, date2);

    // Виводимо різницю у днях
    printf("Різниця у днях: %.0f\n", difference/86400);

    return 0;
}
```

Результат виконання програми:

```
Різниця у днях: 590
```

## Глибоке дослідження

Функція `difftime()` приймає два аргументи типу `time_t`, які представляють секунди з початку епохи UNIX (1 січня 1970 року). Результатом є значення типу `double`, яке представляє різницю між переданими датами в секундах.

Якщо потрібно порівняти дати в більш точних одиницях (наприклад, хвилини, години), можна використати спеціальні константи, такі як `60` для хвилин або `3600` для годин. Наприклад:

```
// Порівняння у хвилинах
double difference = difftime(date1, date2)/60;
```

Також варто зазначити, що функція `difftime()` повертає значення від'ємного знаку, якщо перший переданий аргумент більший за другий. Для більш зручного використання можна використовувати функцію `abs()`, яка повертає значення за модулем:

```
double difference = abs(difftime(date1, date2));
```
