Вітаю, українські програмісти!

Сьогодні ми розглянемо одну з найбільш основних та корисних задач у програмуванні - порівняння двох дат. Чи знаете ви, що це необхідна навичка для будь-якої програми, яка працює з часом? Не гаймо часу та розберемося, чому це так важливо та як це зробити у нашій улюбленій мові програмування - C.

## Чому

Порівняння двох дат - це необхідний елемент багатьох програм, які використовують часові мітки. Наприклад, у фінансових програмах може знадобитися порівняти дати платежів або у бізнес-додатках здійснюється подання даних за певний проміжок часу. Також, ця навичка особливо корисна у різноманітних ділових та організаційних програмах.

## Як

Для початку, нам потрібно знати, що дата може бути представлена у форматі структури `tm` в C. Така структура містить компоненти дати, такі як рік, місяць, день, години, хвилини та секунди. Для порівняння дат нам необхідно взяти дві такі структури і скористатися функцією `difftime`, яка повертає різницю в секундах між двома датами. Ось приклад коду з використанням цієї функції:

```C
struct tm date1 = { .tm_year = 2020, .tm_mon = 6, .tm_mday = 23, .tm_hour = 18, .tm_min = 30, .tm_sec = 0 }; // 23 червня 2020 року о 18:30:00
struct tm date2 = { .tm_year = 2021, .tm_mon = 3, .tm_mday = 12, .tm_hour = 9, .tm_min = 15, .tm_sec = 0 }; // 12 березня 2021 року о 9:15:00
double difference = difftime(mktime(&date2), mktime(&date1)); // отримуємо різницю в секундах
if (difference > 0) {
  printf("Дата 2 пізніше за дату 1 на %.0lf секунд\n", difference);
} else if (difference < 0) {
  printf("Дата 1 пізніше за дату 2 на %.0lf секунд\n", -difference);
} else {
  printf("Дата 1 і дата 2 рівні\n");
}
```

У цьому прикладі ми порівнюємо дати 23 червня 2020 року о 18:30:00 та 12 березня 2021 року о 9:15:00. В результаті отримуємо різницю в 42698400 секунд (близько 495 діб). Не забувайте, що `difftime` повертає значення
