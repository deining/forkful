## Чому

Найпоширенішою причиною для з'єднання рядків в C є створення більш складних рядків за допомогою склеювання менших фрагментів. Це може бути корисно, коли потрібно створити повідомлення, відображати дані або побудувати шлях до файлу.

## Як

Існує кілька способів з'єднати рядки в C, але найпростішим і найбільш універсальним є використання функції `strcat()`.

```C
#include <stdio.h>
#include <string.h>

int main()
{
    char str1[] = "Привіт, ";
    char str2[] = "Світ!";
    strcat(str1, str2);
    printf("%s\n", str1);
    return 0;
}
```

В цьому прикладі ми використовуємо функцію `strcat()` для додавання рядка `str2` до рядка `str1`. Результатом буде рядок `Привіт, Світ!`.

## Глибше

Функція `strcat()` з'єднує два рядки шляхом додавання рядка-джерела в кінець рядка-цілі. Аргумент-джерело повинен бути масивом символів, а аргумент-ціль повинен мати достатньо місця для місткості обох рядків.

У разі, якщо місткість рядка-цілі не вистачає для обох рядків, результат може бути неочікуваним. Це може призвести до помилки розташування пам'яті або пошкодження інших даних.

Щоб уникнути цих проблем, можна використовувати функцію `strncat()`, яка також з'єднує рядки, але при цьому контролює максимальну кількість символів для з'єднання. Це допоможе запобігти виходу за межі місткості рядка-цілі.
