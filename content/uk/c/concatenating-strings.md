## Чому

C-програмування є одним з найпопулярніших мов програмування у світі і знається багатьма програмістами. Одним з найкорисніших навичок, які можуть бути вивчені в C, є злиття рядків або concatenating strings. Ця техніка дозволяє об'єднувати два або більше рядки в один великий рядок, що робить її корисною для багатьох задач програмування.

## Як

Найпростіший спосіб злиття рядків у C - використовувати функцію `strcat()`. Давайте подивимося на приклад коду:

```C
#include <stdio.h>
#include <string.h>

int main() {
  // оголошуємо два рядки
  char str1[50] = "Привіт, ";
  char str2[] = "Україна!";

  // додаємо другий рядок до першого
  strcat(str1, str2);

  // виводимо результат
  printf("%s", str1);

  return 0;
}
```

В даному випадку, `strcat()` додаємо рядок `str2` до кінця рядка `str1`. Результатом буде новий рядок `Привіт, Україна!`. Важливо пам'ятати, що рядок `str1` повинен мати достатньо місця, щоб зберегти новий результат, інакше можуть виникнути проблеми з пам'яттю.

Щоб злити більше ніж два рядки, можна використовувати цю функцію кілька разів. Також існують інші функції, такі як `strncat()` і `strcat_s()`, які дозволяють вказати максимальну кількість символів для додавання та збереження безпеки при роботі з пам'ятю.

## Deep Dive

В C існує багато різних функцій для роботи з рядками, і не всі вони є важливими для злиття рядків. Наприклад, функція `strcpy()` дозволяє копіювати значення одного рядка в інший. Порівняно з `strcat()`, вона більш проста, але не дозволяє додавати значення до існуючого рядка.

Також існує функція `sprintf()`, яка дозволяє форматувати значення і записувати їх у рядок. Наприклад, можна створити рядок, який містить числа, використовуючи цю функцію.

Вивчення різних функцій для роботи з рядками дозволить вам більш гнучко працювати з ними і робити більш складні операції в програмах.
