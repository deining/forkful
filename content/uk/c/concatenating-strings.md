## Для чого

У даний час мова С стала однією з найпопулярніших мов програмування у світі. І одним із перших етапів при вивченні цієї мови є розуміння роботи з рядками, або так званим конкатенуванням рядків. Це особливо важливо для вивчення програмування, адже така функція використовується в багатьох програмах та додатках, у тому числі й у веб-розробці. Тож, якщо ви працюєте з мовою С, то вміння конкатенувати рядки є дуже корисним.

## Як

Для початку, давайте розглянемо структуру рядків у мові С. У мові С рядок представлений як масив символів. Тобто, кожен символ має свій власний індекс у масиві. Ідея конкатенування рядків полягає у злитті двох чи кількох рядків у один цілий.

```C
#include <stdio.h>

int main(void)
{
    char str1[30] = "Це перший рядок";
    char str2[30] = " а це другий рядок";
    char result[60];
    
    // Використовуємо функцію strcat для злиття рядків
    strcat(result, str1);
    strcat(result, str2);
    
    // Виводимо результат на екран
    printf("Результат: %s\n", result);
    
    return 0;
}
```

В даному прикладі ми використовуємо функцію strcat, що дозволяє злити два рядки у новий рядок. Після виконання цієї функції, змінна `result` міститиме злитий результат рядків `str1` і `str2`. Виведемо текст на екран і переконаймось у правильності роботи нашої програми.

```
Результат: Це перший рядок а це другий рядок
```

## Глибше
