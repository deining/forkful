## Чому 

Найпопулярніше питання, яке чути при вивченні мови програмування C - це як знайти довжину рядка? Незважаючи на те, що це на перший погляд здається простим завданням, відповідь на нього може бути неочевидною для новачків у цій мові. Але не хвилюйтесь, у цій статті ми покажемо вам як це зробити в C!

## Як це зробити

Для початку, нам необхідно зрозуміти, що рядок у C зберігається як масив символів *char*. Це означає, що ми можемо використати цей масив, щоб знайти його довжину. Для цього нам необхідно скористатися функцією *strlen()*, яка приймає масив char як аргумент та повертає кількість символів у рядку. Давайте подивимося на приклад коду:

```C
#include <stdio.h>
#include <string.h>

int main(){
    char str[] = "Привіт світ!";
    int length = strlen(str);
    printf("Довжина рядка: %d\n", length);
    return 0;
}
```

Тут ми створюємо рядок *str* і застосовуємо функцію *strlen()* до нього. Потім, за допомогою функції *printf()*, виводимо довжину рядка на екран. Результатом буде 13, оскільки наш рядок містить 13 символів.

Є ще один спосіб знайти довжину рядка, але цей спосіб працює лише у випадку, коли рядок закінчується символом *'\0'* - нуль-термінальним символом. Це означає, що *strlen()* буде рахувати символи у рядку до тих пір, поки не зустріне символ *'\0'*. Для прикладу, подивимося на наступний код:

```C
#include <stdio.h>

int main(){
    char str[] = "Hello World!";
    int length = 0;
    while (str[length] != '\0'){
        length++;
    }
    printf("Довжина рядка: %d\n", length);
    return 0;
}
```

Тут ми використовуємо цикл *while* для того, щоб перевіряти кожен символ у рядку до тих пір, поки не зустрінемо символ *'\0'*. Потім, за допомогою функції *printf()*, виводимо довжину рядка на екран. Результат буде такий самий - 12.
