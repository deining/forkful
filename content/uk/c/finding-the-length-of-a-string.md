## Що і Чому?

Розрахувати довжину рядка означає визначити кількість символів у даному рядку. Програмісти часто мають потребу робити це для обробки текстових даних та визначення їх розміру.

## Як це зробити:

### Основний код:

```C
// Оголошення змінної рядка
char str[] = "Привіт, світ!";
// Розрахунок довжини рядка
int len = 0;
while (str[len] != '\0') {
    len++;
}
// Виведення результату
printf("Довжина рядка: %d", len);
```
### Результат виконання:
```
Довжина рядка: 13
```

### Використання вбудованих функцій:

В мові програмування C є також вбудована функція `strlen()`, яка розраховує довжину рядка автоматично. Це зменшує кількість коду та полегшує процес.

```C
// Оголошення змінної рядка
char str[] = "Привіт, світ!";
// Розрахунок довжини рядка
int len = strlen(str);
// Виведення результату
printf("Довжина рядка: %d", len);
```

### Результат виконання:
```
Довжина рядка: 13
```

## Глибше розуміння:

### Історичний контекст:

Концепція довжини рядка була вперше впроваджена в мові програмування C із версії 4.0. Більш ранні версії мали численні проблеми з обробкою текстових даних, тому була потреба в цій функції.

### Альтернативи:

Надто часте використання циклу для розрахунку довжини рядка може бути витратним та незручним. Замість цього, програмісти можуть використовувати вбудовану функцію або використовувати функцію `strlen()` з інших бібліотек, таких як `<string.h>`.

### Деталі реалізації:

У мові C рядки є масивами символів, які завершуються нульовим символом `'\0'`. Функції `while` та `for` використовуються для перебору кожного символу у рядку та перевірки, чи це останній символ. Якщо так, то розрахунок довжини завершується та виводиться результат.
