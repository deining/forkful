#Чому

Якщо ви працюєте з мовою програмування C, ви, напевно, стикнулися з необхідністю знаходити довжину рядка. Це важлива операція, яка дозволяє вам працювати зі строками в програмі, і це те, про що ми сьогодні поговоримо.

#Як знайти довжину рядка в С

```
#include <stdio.h>
#include <string.h>

int main()
{
    char str[50];
    int len;

    printf("Введіть рядок: ");
    fgets(str, 50, stdin);

    // Вираховуємо довжину рядка використовуючи функцію strlen
    len = strlen(str);

    printf("Довжина рядка: %d", len);
    return 0;
}

```

Введіть рядок: Привіт, Я працюю з С!
Довжина рядка: 20

Також існує інша способом знайти довжину рядка, використовуючи цикл. Такий підхід корисний, якщо ви хочете детальніше зрозуміти, як ця операція працює.

```
#include <stdio.h>

int main()
{
    char str[50];
    int len = 0; // початкова довжина рядка

    printf("Введіть рядок: ");
    fgets(str, 50, stdin);

    // перевіряємо кожен символ до того, як доходимо до кінця рядка.
    while (str[len] != '\0')
    {
        len++;
    }

    printf("Довжина рядка: %d", len);
    return 0;
}

```

#Глибші вдосконалення

Так, ми можемо отримати довжину рядка за допомогою функції `strlen` або циклу, але як це працює під капотом? Давайте розберемося.

Користуватися функцією `strlen` це найбільш простий і швидкий спосіб знайти довжину рядка. Вона приймає рядок як аргумент, а потім із знаходить і повертає його довжину.

Тепер, що стосується циклу, то він має умову `str[len] != '\0'`, що означає, що цикл буде тривати до того, поки ми не доходимо до кінця рядка (знака `\0` є кінцем рядка в С). Кожен проход циклу збільшує змінну `len` на одиницю, тим самим рахуючи кожен символ рядка.

Також важливо згадати, що функція `strlen` враховує всі символи в рядку, включаючи пробіли і знаки пунктуації, тоді як цикл рахує тільки чисті символи.
