## Чому

Не зважаючи на те, що вважається базовою задачею у більшості мов програмування, знаходження довжини рядка є важливим навичкою для кожного програміста. Все залежить від специфіки завдання, але знання довжини рядка може бути корисним для зчитування текстових даних, валідації користувацького вводу, знаходження певного паттерну тощо.

## Як це зробити

Для початку, необхідно оголосити рядок як звичайний масив символів у C. Наприклад:
```C
char str[] = "Здравствуйте!";
```
Далі, використовуючи вже відомий цикл ```for```, ми можемо пройтися по кожному елементу рядка та підрахувати його кількість:
```C
int length = 0; // змінна для збереження довжини
for(int i = 0; str[i] != '\0'; i++) {
    length++;
}
```
У цьому прикладі ми використали умову, що перевіряє наявність нуль-терміналу ```'\0'```, який завершує кожен рядок у C.

Результатом виконання цього коду буде змінна ```length```, яка міститиме довжину рядка, у нашому прикладі - 13.

## Глибокий розбір

При знаходженні довжини рядка ми можемо замінити цикл ```for``` на цикл ```while```, щоб зробити код більш зрозумілим:
```C
int length = 0;
int i = 0; // індекс елементу
while (str[i] != '\0') { // доки не зустрінемо нуль-термінал
    length++;
    i++;
}
```

Також, у більшості випадків нам не потрібно знаходити довжину рядка кожного разу заново, тому можемо написати функцію ```strlen()```, яка буде повертати довжину рядка:
```C
int strlen(char str[]) {
    int length = 0;
    int i = 0;
    while (str[i] != '\0') {
        length++;
        i++;
    }
    return length;
}
```
Її можна буде використовувати в подальшому у наших програмах, щоб зробити код більш організованим та повторюваним.
