## Чому

Перевірка довжини рядка є важливою задачею для більшості програм. Знаходження довжини допомагає визначити кількість символів, що потрібно обробити та врахувати при роботі з рядками.

## Як

```c
char str[] = "Привіт, світ!";
int length = 0;

while(str[length] != '\0')
{
  length++;
}
printf("Довжина рядка: %d", length);
```

В цьому прикладі ми створили рядок "Привіт, світ!" та за допомогою циклу while шукаємо останній символ рядка, який є нуль-термінатором (знаком кінця рядка). При кожній ітерації циклу змінна `length` збільшується на одиницю, тим самим показуючи довжину рядка. При використанні формату `%d` ми можемо вивести цю змінну на екран.

## Deep Dive

Алгоритм пошуку довжини рядка полягає в тому, що ми проходимо по всіх символах рядка до тих пір, поки не зустрінемо нуль-термінатор. Для ефективнішого пошуку можна використовувати вбудовану функцію `strlen()` з бібліотеки `string.h`. Також варто пам'ятати, що під час обробки рядків з кирилицею, кожен символ буде займати дві одиниці пам'яті.
