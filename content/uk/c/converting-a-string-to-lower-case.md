## Why

Існує багато ситуацій, коли необхідно перетворити рядок на малий регістр - можливо, для порівняння з іншими рядками або для зручності введення користувачем. В будь-якому випадку, конвертування рядка у нижній регістр є поширеною задачею в програмуванні, і знання, як цього добитися, є необхідною навичкою для будь-якого розробника C.

## How To

Для початку, нам потрібна функція, яка буде виконувати конвертацію. Давайте назвемо її `toLowercase()` і передамо їй рядок, який ми хочемо перетворити у нижній регістр. 

```C
void toLowercase(char *str) {
  int i = 0;
  // Поки символ не дорівнює кінцю рядка
  while (str[i] != '\0') {
    // Перевіряємо, чи символ є великою літерою
    if (str[i] >= 'A' && str[i] <= 'Z') {
      // Якщо так, замінюємо його на відповідний символ нижнього регістру
      str[i] = str[i] + 32;
    }
    i++;
  }
}
```

Тепер, коли у нас є функція, застосуємо її до нашого рядка і перевіримо результат.

```C
int main() {
  char str[] = "HELLO WORLD";
  // Викликаємо функцію для конвертації
  toLowercase(str);
  // Виводимо результат
  printf("%s", str);
  // Виведе "hello world"
  return 0;
}
```

## Deep Dive

Отже, як саме ця функція перетворює великі літери на малі? Для цього використовується ASCII кодування, де букви між `A` та `Z` мають значення від 65 до 90, а відповідні букви малого регістру - від 97 до 122. Тому для заміни ми просто додаємо до великої літери 32, щоб отримати відповідну малу літеру.
