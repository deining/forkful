# Чому

Стрічка дати є важливою частиною програмування, особливо коли мається справа з роботою з датами та часом. Конвертування дати в рядок є корисним навичкою, особливо при роботі з форматами дат та часу, які використовуються в різних мовах та системах. Дана стаття допоможе вам зрозуміти, чому вона потрібна та як це зробити за допомогою мови програмування C.

# Як це зробити

Існує кілька способів конвертування дати в рядок за допомогою мови програмування C. Найпростішим способом є використання функцій `sprintf` або `snprintf`, які дозволяють форматувати та виводити рядок дати. Для зручності пропонуємо розглянути кілька прикладів та виведення результатів.

```C
#include<stdio.h>
#include<stdlib.h>
#include<time.h>

int main(){
  // Установлюємо поточну дату та час
  time_t current_time = time(NULL);

  // Створюємо змінну для строки дати
  char date_string[50];

  // Використовуємо sprintf для форматування та збереження рядка дати в змінній
  sprintf(date_string, "Поточна дата та час: %s", ctime(&current_time));

  // Виводимо отриманий результат
  printf("%s", date_string);

  return 0;
}
```

Виведення:

```
Поточна дата та час: Sun Aug 29 13:54:52 2021
```

Також можна використовувати функцію `strftime`, яка дозволяє вибрати потрібний формат дати та часу. Нижче наведений приклад використання цієї функції.

```C
#include<stdio.h>
#include<stdlib.h>
#include<time.h>

int main(){
  // Установлюємо поточну дату та час
  time_t current_time = time(NULL);

  // Створюємо змінну для строки дати
  char date_string[50];

  // Використовуємо strftime для форматування та збереження рядка дати в змінній
  strftime(date_string, sizeof(date_string), "%d/%m/%Y, %H:%M:%S", localtime(&current_time));

  // Виводимо отриманий результат
  printf("Поточна дата та час: %s", date_string);

  return 0;
}
```

Виведення:

```
Поточна дата та час: 29/08/2021, 14:02:24
```

Існує також можливість використовувати готові бібліотеки для конвертування дати в рядок, такі як `strptime` або `ctime`, які дозволяють зберегти більше функціоналу та простоту в роботі з датами.
