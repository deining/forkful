## Чому

У цій статті ми розглянемо процес конвертування дати у рядок у мові програмування C. Це корисна навичка для будь-якого програміста, оскільки дати є важливою частиною багатьох програм.

## Як це зробити

Для початку нам необхідно використати структуру `struct tm` для збереження дати та функцію `strftime()` для конвертування дати у рядок. Давайте розглянемо це на прикладі:

```C
#include <stdio.h>
#include <time.h>

int main(void)
{
    // Ініціалізуємо структуру структуру struct tm з поточною датою та часом
    time_t t = time(NULL);
    struct tm *now = localtime(&t);

    // Використовуємо функцію strftime() для конвертування дати у рядок
    char str[50];
    strftime(str, sizeof(str), "%d/%m/%Y %H:%M:%S", now);

    // Виводимо результат
    printf("%s", str);

    return 0;
}
```

У цьому прикладі ми створюємо рядок `str` та використовуємо функцію `strftime()` для конвертування дати та часу у зручний для нас формат. Зверніть увагу, що перший параметр функції `strftime()` - це рядок, який визначає бажаний формат дати та часу.

Результат виконання програми буде виглядати так:

```C
05/12/2021 13:30:00
```

Єдине, що залишається - це змінити формат рядка, якщо вам потрібно інший вигляд дати або часу.

## Поглиблення

Процес конвертування дати у рядок може бути трохи складнішим, якщо вам потрібно враховувати часові зони та інші часові параметри. Структура `struct tm` містить багато корисних полів, які можуть бути корисними для подальшої обробки дати та часу.

Крім того, ви можете розширити цей приклад для обробки дати у будь-якій мові, якщо ви знаєте, як виконувати операції над рядками та здійснювати форматування дати та часу.
