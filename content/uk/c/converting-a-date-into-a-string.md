## Чому
Ви могли б замислитися, чому потрібно конвертувати дату в рядок у програмуванні на C? Одна із причин - це підготовка даних для виведення на екран або збереження у файл. Крім того, у багатьох програмах необхідно працювати з датами та часами, і конвертування дати у рядок дозволяє зручно проводити такі операції.

## Як
Для конвертації дати в рядок у C ми можемо скористатися функцією `strftime()`. Потрібно вказати формат рядка, у який ми хочемо перетворити дату, а також вказати дату, яку потрібно конвертувати. Наприклад:

```C
#include <stdio.h>
#include <time.h>

int main() {
  time_t t;
  t = time(NULL);
  struct tm *current_time = localtime(&t);
  char buffer[100];

  // Перетворення дати у рядок у форматі "дд.мм.рррр"
  strftime(buffer, sizeof(buffer), "%d.%m.%Y", current_time);
  printf("Сьогоднішня дата: %s\n", buffer);

  // Перетворення дати у рядок у форматі "години:хвилини:секунди"
  strftime(buffer, sizeof(buffer), "%H:%M:%S", current_time);
  printf("Поточний час: %s\n", buffer);

  return 0;
}
```

Вихідна дата буде використовувати поточну локальну часову зону та формат, який ми вказали у функції `strftime()`. Очікуваний вивід:

```
Сьогоднішня дата: 24.12.2021
Поточний час: 15:30:00
```

## Глибші дослідження
Щоб краще зрозуміти, як працює конвертація дати в рядок у C, важливо знати про структуру `struct tm` і функції `time()` і `localtime()`. `struct tm` містить різні поля, такі як рік, місяць, день, години, хвилини, секунди, які дозволяють представити дату та час у кодованому форматі. Функція `time()` повертає поточний час у вигляді кількості секунд, що пройшли з 01.01.1970. Функція `localtime()` приймає це значення і повертає вказівник на структуру `struct tm` зі значеннями у поточній локальній часовій зоні.
