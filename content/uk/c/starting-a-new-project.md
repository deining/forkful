## Чому

В статті ми розглянемо, як почати новий проект на мові програмування C. Це корисний навик, який допоможе вам створити потужні і ефективні програми, що виконуються на різних платформах. 

## Як почати
```C
#include <stdio.h>

int main()
{
    // Введення двох цілих чисел від користувача
    int num1, num2;
    printf("Введіть перше число: ");
    scanf("%d",&num1);
    printf("Введіть друге число: ");
    scanf("%d",&num2);

    // Обчислення суми двох чисел
    int sum = num1 + num2;

    // Виведення результату на екран 
    printf("Сума чисел %d та %d дорівнює %d", num1, num2, sum);
    return 0;
}
```
У цьому прикладі ми використовуємо стандартну бібліотеку `stdio.h`, яка надає функції для введення та виведення даних. Спершу ми оголошуємо дві змінні `num1` та `num2`, а потім просимо користувача ввести два цілих числа. За допомогою функції `scanf` ми зберігаємо введені користувачем числа у відповідні змінні. Далі ми обчислюємо суму цих чисел та виводимо результат на екран за допомогою функції `printf`., яка містить форматований рядок із вказівниками на змінні.

## Глибокий погляд
Починаючи новий проект, важливо обрати правильну структуру і організацію коду. Тут вам можуть допомогти різні інструменти та бібліотеки. Крім базової бібліотеки `stdio.h`, існують інші, такі як `stdlib.h`, `math.h`, `string.h`, які містять функції для обробки різних типів даних.

Також варто подумати про використання структур даних, наприклад масивів або структур, для зберігання і обробки даних у більш зручному форматі. Наприклад, використання структури `struct` може допомогти створити свій тип даних з більшими можливостями.
