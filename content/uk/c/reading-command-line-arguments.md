## Чому

В програмуванні на мові C командний рядок є важливою частиною. Використання командних аргументів дозволяє змінювати поведінку програми без необхідності змінювати сам код. Також це дає можливість передавати необхідну інформацію програмі під час її запуску, що робить її більш гнучкою та зрозумілою.

## Як

Щоб скористатись командними аргументами в програмі на мові C потрібно додати два аргументи до функції `main()`: `argc` та `argv[]`.

`argc` є цілочисельною змінною, яка зберігає кількість аргументів командного рядка, а `argv[]` є масивом рядків, який зберігає значення кожного з цих аргументів. Нижче подано приклад коду, який демонструє використання командних аргументів:

```C
#include <stdio.h>

int main(int argc, char *argv[]) {
    printf("Кількість аргументів: %d\n", argc);

    for (int i = 0; i < argc; i++) {
        printf("Аргумент %d: %s\n", i, argv[i]);
    }

    return 0;
}
```

Після компіляції та запуску програми з командними аргументами, ви побачите такий результат:

```
$ ./program argument1 argument2 argument3
Кількість аргументів: 4
Аргумент 0: ./program
Аргумент 1: argument1
Аргумент 2: argument2
Аргумент 3: argument3
```

## Глибинне дослідження

У мові C доступні більш специфічні механізми для роботи з командними аргументами. Наприклад, функція `getopt()` дозволяє обробляти аргументи зі спеціальними опціями (-h, -v тощо) та їх значеннями. Також необхідно звернути увагу на правильну обробку помилок, наприклад, якщо користувач не передав жодного аргументу або передав некоректний аргумент.
