## Чому
В цій статті ми розглянемо, як працювати з аргументами командного рядка в мові програмування C. Це може бути корисно для тих, хто хоче створювати програми, які можна запускати з командного рядка.

## Як це зробити
Для того, щоб отримувати аргументи командного рядка у програмі на C, вам потрібно використовувати два параметри в функції `main()` - `argc` і `argv`. Розглянемо наступний приклад коду:

```C
#include <stdio.h>

int main(int argc, char *argv[]) {
    // Перший аргумент - назва самої програми
    printf("Назва програми: %s\n", argv[0]);

    // Перебираємо усі аргументи, починаючи з другого
    for (int i = 1; i < argc; i++) {
        // Виводимо кожен аргумент на окремому рядку
        printf("Аргумент #%d: %s\n", i, argv[i]);
    }
    
    return 0;
}
```

Давайте запустимо цю програму з наступними аргументами командного рядка:

```
./program arg1 arg2 arg3
```

На виході ми отримаємо:

```
Назва програми: ./program
Аргумент #1: arg1
Аргумент #2: arg2
Аргумент #3: arg3
```

Також варто зазначити, що аргумент `argv` є масивом рядків, тобто вхідні дані в програму необхідно буде перетворити у потрібний тип даних за необхідності.

## Глибоке погруження
Цікаву можливість надає функція `getopt()`, яка дозволяє зчитувати аргументи командного рядка з більш складною логікою, наприклад, коли потрібно додатково опцій під час запуску програми. Детальніше про цю функцію можна почитати у [документації](https://www.gnu.org/software/libc/manual/html_node/Argument-Syntax.html).
