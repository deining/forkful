## Що & Чому?

Читання аргументів командного рядка - це процес, коли програми отримують вхідні дані, вказані користувачем при запуску програми в командному рядку. Це дозволяє програмі виконувати різні дії в залежності від вказаних аргументів. Програмісти використовують цей метод для створення більш гнучких та інтерактивних програм.

## Як це зробити:

```
#include <stdio.h>

int main(int argc, char *argv[])
{
    // перевіряємо кількість аргументів
    if (argc == 2)
    {
        // виводимо вказаний аргумент
        printf("Ваш параметр: %s", argv[1]);
    }
    else
    {
        // виводимо повідомлення про помилку
        printf("Вказано недостатньо аргументів!");
    }

    return 0;
}
```

Приклад виклику програми з аргументом:

```
$ ./program "Hello World!"
```

Результат:

```
Ваш параметр: Hello World!
```

## Попригляд та Альтернативи:

Відоме використання командного рядка має свої корені ще зі стародавніх операційних систем Unix та DOS. Альтернативою для читання аргументів може бути використання файлу конфігурації або інтерфейсу користувача. Проте, читання аргументів командного рядка є швидким та зручним способом передачі даних до програми.
