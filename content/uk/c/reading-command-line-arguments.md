## Чому

Командні рядки - це потужний засіб взаємодії з нашими комп'ютерами. Знання, як користуватися аргументами командного рядка в C може покращити ваші навички програмування та зробити ваші програми більш гнучкими і зручними для користувачів.

## Як користуватися аргументами командного рядка в C

Для того, щоб програма читала аргументи командного рядка, вам необхідно використовувати два параметри в функції main: `argc` та `argv`. `argc` представляє собою кількість аргументів та `argv` є масивом рядків, де кожний елемент є окремим аргументом. Ось приклад коду:

```C
#include <stdio.h>
int main(int argc, char *argv[])
{
    if (argc < 2)
    {
        printf("Usage: %s [name]\n", argv[0]);
        return 1;
    }
    else
    {
        printf("Hello %s!\n", argv[1]);
    }
    return 0;
}
```

Запустивши цей код з наступними аргументами командного рядка: `./program John`, ви отримаєте наступний результат: `Hello John!`.

## Поглиблене вивчення

Знаючи, як користуватися аргументами командного рядка, ви можете створювати більш складні та функціональні програми. Ви можете використовувати `argc` та `argv` для отримання інформації про введені користувачем параметри та використовувати їх для керування програмою. Також, ви можете перевіряти, чи користувач ввів коректні аргументи та виводити повідомлення про використання, якщо їх недостатньо чи їх введено неправильно.
