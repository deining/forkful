## Чому

Створення тимчасового файлу є важливою частиною багатьох програм на С, оскільки він дозволяє заощадити час та ресурси, шукаючи та видаляючи непотрібні файли.

## Як

Створення тимчасового файлу на C є простим процесом, який включає такі кроки:

1. Включення необхідних заголовочних файлів: `#include <stdio.h>` і `#include <stdlib.h>`
2. Використання функції `tmpfile()` для створення тимчасового файлу із спеціальною префіксною назвою.
3. Запис даних у файл за допомогою функції `fprintf()`.
4. Закриття файлу за допомогою функції `fclose()`.

Приклад коду:

```C
#include <stdio.h>
#include <stdlib.h>

int main() {

   FILE *fptr;
   int num = 10;

   // Створення тимчасового файлу
   fptr = tmpfile();

   // Запис даних у файл
   fprintf(fptr, "%d", num);

   // Закриття файлу
   fclose(fptr);

   return 0;
}
```

Запустивши цей код, ви отримаєте такий вихідний файл:

```
$ cat /tmp/tmp.zyeDmrflVY
10
```

## Глибші відомості

Тимчасові файли звичайно зберігаються в оперативній пам'яті, що робить їх доступними для програми швидше, ніж постійні файли, які зберігаються на диску. Однак, якщо пам'ять закінчується, тимчасові файли можуть бути перенаправлені на диск. Це може призвести до затримки в роботі програми, тому важливо слідкувати за доступним обсягом пам'яті для збереження тимчасових файлів.
