# Що й чому?
Створення тимчасового файлу - це процес створення файлу, який зберігає тимчасові дані, необхідні під час виконання програми. Програмісти використовують цей метод, щоб зберегти можливість повторного використання даних або зберегти проміжні результати обрахунків.

## Як створити тимчасовий файл:
```
#include <stdlib.h>
#include <stdio.h>

int main() {
  FILE *temporary_file = tmpfile(); // Створюємо тимчасовий файл
  if (temporary_file != NULL) {
    fputs("Hello, world!", temporary_file); // Записуємо дані у файл
    rewind(temporary_file); // Повертаємо позицію курсора на початок файлу
    char buffer[100];
    if (fgets(buffer, 100, temporary_file) != NULL) {
      printf("Виводимо дані з тимчасового файлу: %s", buffer); // Виводимо дані з тимчасового файлу
    }
    fclose(temporary_file); // Закриваємо файл
  }
  return 0;
}
```
Вивід: `Виводимо дані з тимчасового файлу: Hello, world!`

## Поглиблене дослідження:
Створення тимчасових файлів було вперше реалізовано в UNIX-системах для збереження тимчасових даних під час роботи з тривалими процесами. Однак, цей метод активно використовується й в сучасних мовах програмування, включаючи C. Крім використання функції `tmpfile()`, програмісти також можуть створювати тимчасові файли за допомогою функцій `mkstemp()` та `mkdtemp()`. Вони мають більш гнучкість і дозволяють встановлювати політику безпеки, наприклад, дозволяючи лише власнику файлу змінювати його вміст.
