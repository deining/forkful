## Чому

Створення тимчасового файлу є важливим аспектом програмування в C. Іноді ми хочемо створити файл, використовувати його для певних потреб, а потім видалити цей файл, щоб зберегти простір на нашому пристрої. Створення тимчасового файлу дозволяє нам робити це без створення постійних файлів.

## Як створити тимчасовий файл

Цей метод дозволяє створити тимчасовий файл у цій самій директорії, де знаходиться ваша програма. В якості прикладу, ми будемо використовувати функцію `tmpfile()`. Ця функція повертає стрім, який ми можемо використовувати для запису та читання з тимчасового файлу.

```C
#include <stdio.h>

int main(){

   FILE *fp;

   // створення тимчасового файлу
   fp = tmpfile();

   // запис до тимчасового файлу
   fputs("Це тимчасовий файл для прикладу!", fp);

   // читання з тимчасового файлу
   char buffer[255];
   fgets(buffer, 255, fp);

   printf("Вміст тимчасового файлу: %s", buffer);

   // закриття файлу
   fclose(fp);

   return 0;
}
```

Результат виконання програми:

```
Вміст тимчасового файлу: Це тимчасовий файл для прикладу!
```

## Глибокий занурення

Існує кілька інших методів створення тимчасових файлів, включаючи використання функції `mkstemp()` та вручну створюючи тимчасові імена з використанням функції `tmpnam()`. Однак, необхідно бути обережними при використанні цих методів, оскільки можуть виникнути проблеми з безпекою.
