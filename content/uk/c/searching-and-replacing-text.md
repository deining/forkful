## Чому

Пошук і заміна тексту - це базові функції, необхідні для ефективної роботи зі змінними та шаблонами в C. Вони дозволяють швидко змінювати значення на потрібні та знаходити частини строки за заданим патерном.

## Як

Для пошуку та заміни тексту в C, ви можете використати функцію `str_replace()`, яка приймає три аргументи: вихідну строку, патерн для пошуку та нове значення для заміни. Наприклад, якщо ви хочете замінити всі пробіли в рядку на підкреслення, ви можете написати наступний код:

```C
char str[] = "Це рядок з пробілами";
str_replace(str, " ", "_");

printf("%s", str); // Вивід: "Це_рядок_з_пробілами"
```

Можна також використовувати функцію `str_replace()` для знаходження і заміни певних патернів в строках. Наприклад, якщо ви хочете замінити всі літери "о" на "і" у слові "коли", використайте наступний код:

```C
char word[] = "коли";
str_replace(word, "о", "і");

printf("%s", word); // Вивід: "кілі"
```

## Глибока занурення

Функція `str_replace()` використовує найпростіший метод для заміни тексту - вона просто шукає патерн і заміняє його на нове значення в усіх випадках. Проте, існують інші можливості для більш точного і динамічного пошуку та заміни тексту в C, такі як регулярні вирази за допомогою бібліотеки `regex.h`.
