## Чому

Чому б хтось хотів займатися пошуком та заміною тексту? Існує безліч причин, чому ця задача може виникнути. Наприклад, ви хочете замінити всі входження певного слова або фрази у вашому коді на іншу, більш оновлену версію. Або ви можете бути письменником і хочете змінити певне слово на всі місця в вашому творі. Якщо ви програміст або письменник, то це корисне вміння, яке може заощадити вам час та зусилля.

## Як

```C
#include <stdio.h>

int main()
{
   char str[100] = "Привіт, світ!";
   printf("Оригінальний рядок: %s\n", str);
   
   // Заміна "світ" на "Україна"
   str[7] = "Україна";
   printf("Замінений рядок: %s\n", str);
   
   return 0;
}
```

Вище наведений приклад демонструє, як можна замінити певну частину тексту у рядку за допомогою мови програмування C. Програма спочатку створює рядок зі значенням "Привіт, світ!", а потім замінює слово "світ" на "Україна". Результатом виконання програми буде рядок "Привіт, Україна!".

## Глибші роздивлень

Існує багато більш складних способів заміни тексту у програмуванні, таких як використання регулярних виразів або функцій пошуку та заміни. Ви також можете написати власну функцію, яка буде здійснювати пошук та заміну за заданими параметрами. Не соромтеся використовувати Інтернет для пошуку додаткових підказок та прикладів коду, які допоможуть вам вирішити вашу конкретну задачу з пошуком та заміною тексту.
