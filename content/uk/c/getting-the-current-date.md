# Що і чому?

Отримати поточну дату - це процес отримання поточної дати та часу у коді програми. Це необхідно для того, щоб програми могли взаємодіяти з користувачем, зберігати дані та виконувати інші дії, що пов'язані з часом.

# Як це зробити?

```c
#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    char* c_time_string;

    current_time = time(NULL);

    c_time_string = ctime(&current_time);

    printf("Поточна дата та час: %s", c_time_string);

    return 0;
}
```

## Детальніше

### Історичний контекст

З отриманням поточної дати коду програми пов'язують початки роботи із машинним кодом. У минулому це був складний процес, але сучасні мови програмування, такі як C, мають вбудовані функції для отримання поточної дати.

### Альтернативи

Є кілька варіантів отримання поточної дати, однак найбільш поширеним є використання стандартної бібліотеки C - time.h. Інші альтернативи можуть включати в себе використання зовнішніх бібліотек або власноруч написаних функцій.

### Деталі реалізації

Функція time() повертає величину, яка представляє кількість секунд, що минули з 00:00:00 01 січня 1970р. Також, є функція ctime(), яка перетворює цю величину у рядок, який містить поточну дату та час.
