# З чого почати: Отримання поточної дати в С

Так, це знову я, Ваш товариш-програміст, готовий розкрити цікаві та корисні теми. Сьогодні, ми з вами розкажемо про те, як отримати поточну дату в С. Це надзвичайно корисна функція, яка може стати у пригоді при написанні різних програм. Продовжуйте читати, щоб дізнатись більше!

## Чому потрібна поточна дата?

Перш ніж перейти до того, як генерувати поточну дату у С, давайте розберемося, чому це може бути корисно. Одним із основних використань поточної дати є відстеження часу виконання програми. Наприклад, ви можете створити лог-файл, який буде вказувати, коли програма була запущена та якою версією вона є, на основі поточної дати. Також, це може допомогти у виконанні певних завдань, які повинні бути виконані протягом певного періоду часу. Якщо проект залучає розклад дій, то поточна дата може бути важливою для правильної організації робочого процесу.

## Як отримати поточну дату у С

Тепер, коли ми знаємо, які можуть бути використання для отримання поточної дати, давайте спробуємо це зробити самі. У С є багато способів отримання поточної дати, але ми розглянемо один з найпростіших - за допомогою функції `time()`.

```
#include <stdio.h>
#include <time.h>

int main() {
    time_t now = time(NULL);
    printf("Поточна дата і час: %s", ctime(&now));
    return 0;
}
```

У цьому прикладі ми викликаємо функцію `time()` та передаємо їй значення `NULL`. Щоб перетворити цей час в рядок, ми використовуємо функцію `ctime()`, якій передаємо отриманий час. При виконанні програми, ви побачите вивід у форматі `Поточна дата і час: Day Mon DD HH:MM:SS YYYY`, де Day - день тижня, Mon - місяць, DD - день, HH:MM:SS - години, хвилини, секунди, а YYYY - рік.
