## Зачем

Отримання поточної дати є важливою частиною багатьох програм у С. Це дозволяє користувачам і розробникам відслідковувати поточну дату та час, що може бути корисним для ведення звітності або автоматизації певних функцій.

## Як

Для отримання поточної дати в С, нам знадобиться використовувати стандартну бібліотеку `time.h`. Наприклад, для друку поточної дати на екран, можна використати функцію `time()` з цієї бібліотеки у поєднанні з функцією `ctime()` для форматування дати у зручний для читання вигляд:

```C
#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    char* c_time_string;

    // Отримання поточної дати
    current_time = time(NULL);

    // Форматування дати
    c_time_string = ctime(&current_time);

    // Друк поточної дати на екран
    printf("Поточна дата: %s", c_time_string);

    return 0;
}
```

Вищевказаний код виведе наступне:

```
Поточна дата: Mon Jul 26 14:56:25 2021
```

На цьому прикладі видно, що дата форматується у вигляді дня тижня, місяця, дня та часу. Але що, якщо ми хочемо друкувати дату у більш зручному для нас форматі? Для цього можна скористатися функцією `strftime()` з бібліотеки `time.h`.

Наприклад, для друку дати у форматі "День/Місяць/Рік":

```C
#include <stdio.h>
#include <time.h>

int main() {
    time_t current_time;
    struct tm* time_info;
    char time_string[9]; // 9 символів для збереження дати у форматі "ДД/ММ/РРРР"

    // Отримання поточної дати
    current_time = time(NULL);

    // Форматування дати
    time_info = localtime(&current_time);
    strftime(time_string, sizeof(time_string), "%d/%m/%Y", time_info);

    // Друк поточної дати на екран
    printf("Поточна дата: %s\n", time_string);

    return 0;
}
```

Тепер видрукована дата буде мати вигляд "26/07/2021".
