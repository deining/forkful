## Чому

Програмування захоплює багатьох людей, але чому би вам це потрібно? Однією з причин може бути отримання поточної дати. Це корисно для створення програм, що потребують часової мітки, таких як календарі, розклади або журнали.

## Як це зробити

Існує кілька способів отримати поточну дату в мові програмування C. Один з шляхів - це використання функції `time()`, яка повертає кількість секунд, пройдених з 1 січня 1970 року. Ця дата відома як "UNIX час". Нижче наведений код для отримання поточного дня, місяця, року та дня тижня з використанням функції `time()`:

```C
#include <stdio.h>
#include <time.h>

int main()
{
    // отримання UNIX часу
    time_t now = time(NULL);

    // створення структури для збереження дати та часу
    struct tm *current_time = localtime(&now);

    // використання функцій strftime() та tm_wday для отримання поточної дати та дня тижня
    printf("Поточна дата: %d/%d/%d\n", current_time->tm_mday, current_time->tm_mon + 1,
                                        (current_time->tm_year + 1900));

    printf("Поточний день тижня: %d\n", current_time->tm_wday);
    
    return 0;
}
```
Вивід:

```
Поточна дата: 10/03/2021
Поточний день тижня: 3
```

Інший спосіб - використання функції `localtime()`, яка повертає структуру `tm` з інформацією про поточний час. Для отримання певної інформації, наприклад, дня тижня, можна використати відповідну змінну структури `tm`:

```C
#include <stdio.h>
#include <time.h>

int main()
{
    // отримання поточного часу
    time_t now = time(NULL);

    // створення структури для збереження дати та часу
    struct tm *current_time = localtime(&now);

    // використання змінної tm_wday для отримання поточного дня тижня
    printf("Поточний день тижня: %d\n", current_time->tm_wday);

    return 0;
}
```
Вивід:
```
Поточний день тижня: 3
```
