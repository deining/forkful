## Чому

Програмісти часто стикаються з необхідністю надсилання HTTP запитів під час роботи з мережевими додатками або створення власних веб-серверів. Це дуже важливий процес, який дозволяє обмінюватися даними між різними пристроями та серверами. Тому варто знати, як правильно надсилати HTTP запити, щоб дати змогу вашому коду працювати з потрібними даними та виконувати необхідні завдання.

## Як це зробити

Для надсилання HTTP запитів у мові програмування С є кілька способів. Розглянемо найпростіший з них - за допомогою бібліотеки libcurl. Ця бібліотека дозволяє виконувати HTTP запити та отримувати відповіді в зручному для нас форматі.

Для початку вам потрібно додати бібліотеку libcurl до вашого проекту. Для цього встановіть пакет libcurl-dev на Ubuntu або libcurl-devel на CentOS.

Після цього в коді вам потрібно включити заголовочний файл <curl/curl.h>. Це дозволить використовувати функціонал бібліотеки libcurl.

Наступним кроком є створення об'єкту CURL, який представляє нам можливість виконати HTTP запити. За допомогою функції curl_easy_setopt() ми можемо встановити необхідні налаштування для нашого запиту, такі як адреса сервера та тип запиту.

Наприклад, для надсилання GET запиту на адресу "https://www.example.com" код буде виглядати так:

```C
CURL *curl;
curl = curl_easy_init();
curl_easy_setopt(curl, CURLOPT_URL, "https://www.example.com");
curl_easy_setopt(curl, CURLOPT_HTTPGET, 1);
curl_easy_perform(curl);
curl_easy_cleanup(curl);
```

Після виконання запиту, прийдеться обробити отриману відповідь, що зберігається у змінній "curl". Для цього використовуйте функцію curl_easy_getinfo(), яка дозволяє отримати доступ до різних даних про запит, наприклад, код стану HTTP або заголовки відповіді.
