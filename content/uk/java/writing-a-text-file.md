З чого почати: Чому писання текстового файлу є цінним навиною.

Як се робить: Приклади коду та прикладних результатів в блоках "```Java ...```".

Глибока іммерсія: Детальна інформація про написання текстових файлів.

Один з найбільш основних навичок, яку потрібно володіти під час програмування, це вміння працювати з файлами. І хоча можна зберігати дані в різних форматах, текстовий файл є найпростішим і найбільш поширеним способом збереження інформації. Також текстові файли є зручним форматом для обміну даними між програмами.

Якство прочитати та записати текстовий файл в Java? Це дуже просто за допомогою стандартного пакету `java.io`. Ми можемо використовувати класи `File`, `FileReader` та `FileWriter` для отримання доступу до файлу та зчитування або запису даних. Наприклад, щоб прочитати дані з текстового файлу, ми можемо використовувати такий код:

```Java
import java.io.*;

public class ReadTextFile {

    public static void main(String[] args) throws IOException {

        // Створимо об'єкт типу File, який вказує на наш файл
        File file = new File("myFile.txt");

        // Створимо об'єкт типу FileReader для зчитування даних з файлу
        FileReader fileReader = new FileReader(file);

        // Створимо буфер зчитування байтів з файлу
        BufferedReader bufferedReader = new BufferedReader(fileReader);

        // Заведемо змінну для збереження зчитаного рядка
        String line;

        // Зчитуємо рядки з файлу, доки не досягнемо кінця
        while ((line = bufferedReader.readLine()) != null) {
            // Виводимо рядок у консоль
            System.out.println(line);
        }

        // Закриваємо буфер та файл
        bufferedReader.close();
        fileReader.close();
    }
}
```

Припустимо, що у нашому текстовому файлі `myFile.txt` міститься наступний текст:

```
Привіт, світе!
Цей файл був згенерований програмою Java.
```

В результаті, отримаємо наступний вивід у консоль:

```
Привіт, світе!
Цей файл був згенерований програмою Java.
```
