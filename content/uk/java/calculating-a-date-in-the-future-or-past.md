## Чому

Порахувати дату в майбутньому або минулому може бути корисно під час програмування додатків, які використовують часові маркери. Наприклад, можна використовувати цей метод для встановлення термінів виконання завдань, за якими ви можете відстежувати роботу вашої програми або відображати поточну дату у вашій програмі. Також, знаючи часові зони, можна вирахувати необхідну дату для різних регіонів.

## Як

Використовуючи Java, існує кілька способів розрахунку дати в майбутньому або минулому. Один із способів - використовувати клас `Calendar`, який містить методи для маніпуляції датами. Наприклад, для розрахунку дати п'яти днів в майбутньому ви можете використати такий код:

```java
Calendar calendar = Calendar.getInstance();
calendar.add(Calendar.DAY_OF_MONTH, 5);
Date futureDate = calendar.getTime();
System.out.println(futureDate);
```

В результаті ви отримаєте дату через п'ять днів від поточної дати. Існує також специфічний клас `LocalDate`, який дозволяє працювати тільки з датами, без урахування часових зон. Для розрахунку майбутньої дати з використанням `LocalDate` ви можете використати такий код:

```java
LocalDate currentDate = LocalDate.now();
LocalDate futureDate = currentDate.plusDays(5);
System.out.println(futureDate);
```

Цікавим фактом є те, що метод `plusDays()` також може приймати від'ємні значення, тому з його допомогою можна розрахувати дату в минулому.

## Глибокий занурення

Більшість методів для розрахунку дати засновані на користуванні епохою Unix - кількості секунд, які пройшли з 1 січня 1970 року. Також програмісти можуть використовувати бібліотеку `java.time`, що була введена в Java 8, для більш зручного та точного роботи з датами.
