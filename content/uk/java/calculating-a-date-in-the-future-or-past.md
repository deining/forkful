# Навіщо

Розрахунок дати в майбутньому або в минулому може бути корисним для виконання різноманітних завдань, наприклад, планування подій або виконання аналітики. У цій статті покажемо, як це зробити за допомогою Java.

## Як

Попередній та майбутній розрахунок дати можна здійснити за допомогою класу `Calendar` та методів `add()` та `getTime()`. Наприклад, для визначення дати за один день вперед ми можемо використати наступний код:

```Java
Calendar cal = Calendar.getInstance(); // створення об'єкту календаря
cal.add(Calendar.DATE, 1); // додаємо 1 день до поточної дати
System.out.println(cal.getTime()); // виводимо результат
```

Результат виведення буде датою наступного дня.

## Глибокий пір

Клас `Calendar` дозволяє визначати дату в майбутньому або в минулому не тільки за днями, але і за годинами, хвилинами та іншими одиницями часу. Наприклад, для визначення дати за місяць вперед ми можемо використати метод `add()` з аргументом `Calendar.MONTH`.

```Java
cal.add(Calendar.MONTH, 1); // додаємо 1 місяць до поточної дати
```

Ви також можете встановити конкретну дату, використовуючи методи `set()` і `setTime()`. Для більш детальної інформації про ці методи та інші можливості класу `Calendar`, рекомендуємо ознайомитися з офіційною документацією.
