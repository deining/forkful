## Чому

Здоровий глузу наступаєчмо часу, коли нам доводиться обчислювати дату в майбутньому або минулому. Це може бути потрібно для розрахунку терміну доставки товару, дати виїзду чи бронювання квитків. В таких випадках необхідно володіти навиками роботи з датами в Java.

## Як

Найпростіший спосіб обчислення дати в майбутньому або минулому - це за допомогою класу `Calendar` в Java. Для цього необхідно використовувати методи `add()` або `set()`. Перший метод дозволяє додати певну кількість одиниць (наприклад, днів або місяців) до поточної дати, а другий метод встановлює конкретну дату. Наприклад:

```Java
Calendar calendar = Calendar.getInstance();
// Додати 7 днів до поточної дати
calendar.add(Calendar.DAY_OF_MONTH, 7);
// Встановити конкретну дату - 25 лютого 2021 року
calendar.set(2021, Calendar.FEBRUARY, 25);

// Вивести обчислену дату на екран
System.out.println(calendar.getTime());
```

В результаті отримаємо наступний вихід: `Sun Feb 28 15:47:10 EET 2021`.

Окрім класу `Calendar`, також існують більш прості та зручні у використанні бібліотеки для роботи з датами, наприклад, `Joda-Time` та `java.time` з Java 8. Для обчислення дати в майбутньому або минулому за допомогою `java.time` необхідно скористатися методом `plus()` або `minus()`:

```Java
// Щоб отримати дату за 7 днів уперед
LocalDate currentDate = LocalDate.now();
LocalDate futureDate = currentDate.plus(7, ChronoUnit.DAYS);
// Щоб отримати дату за 2 місяці назад
LocalDate pastDate = currentDate.minus(2, ChronoUnit.MONTHS);

// Вивести обчислені дати на екран
System.out.println(futureDate); // 2021-07-28
System.out.println(pastDate); // 2021-03-26
```

## Глибоке занурення

Існує багато факторів, які можуть впливати на обчислену дату, наприклад, різниця у часових зонах, рік високосний чи ні, зміна часу. Тому важливо ретельно перевіряти кожне обчислення дати та використовувати відповідні методи.

Також, варто звернути увагу на форматування дати - часовий пояс, розділові знаки, великі чи малі букви - це все може впливати на правильність виведення інформації про дату.
