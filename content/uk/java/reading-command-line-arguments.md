## Чому

У цій статті ми дізнаємося, чому важливо знати як читати аргументи командного рядка в Java. Це дозволить вам краще керувати вашими програмами і зробить їх більш гнучкими.

## Як

Командний рядок є корисним інструментом в розробці програм, оскільки дозволяє передавати параметри програмі під час її запуску. Давайте розглянемо декілька прикладів того, як ми можемо читати аргументи командного рядка в Java:

```Java
public class CommandLineArgs {
  public static void main(String[] args) {
    // Перший аргумент завжди є ім'ям програми
    System.out.println("Ім'я програми: " + args[0]);
    
    // Другий аргумент може бути будь-якими параметрами
    System.out.println("Перший параметр: " + args[1]);
    
    // Третій і так далі
    System.out.println("Другий параметр: " + args[2]);
  }
}
```

Якщо ми запустимо цю програму з такими аргументами: `java CommandLineArgs Привіт Світ`, то ми отримаємо такий вивід:

```
Ім'я програми: Привіт
Перший параметр: Світ
```

Якщо ви хочете передати рядок з пробілами як один аргумент, то ви можете заключити його у подвійні лапки:

```
java CommandLineArgs "Привіт Світ"
```

Більше прикладів ви можете знайти в [офіційній документації](https://docs.oracle.com/javase/tutorial/essential/environment/cmdLineArgs.html) Java.
