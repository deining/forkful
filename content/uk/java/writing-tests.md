## Чому
Написання тестів є важливою частиною програмування, оскільки воно дозволяє перевірити правильність роботи програми та впевнитися у її якості. Тестування також допомагає виявляти та усувати вади у програмі, що забезпечує більш стабільну та надійну роботу.

## Як
Для написання тестів у Java використовується бібліотека JUnit. Нижче наведено приклад написання тесту для перевірки правильності роботи методу "calculateSum".

```Java
@Test
public void testCalculateSum() {
    int result = calculateSum(2, 3);
    assertEquals(5, result);
}
```
Результат виконання тесту буде виглядати наступним чином:
```
Expected :5
Actual   :5
```

## Глибше погруження
Існують різні типи тестів, такі як одиничні, інтеграційні та функціональні тести. Кожен з цих типів має свою роль та важливість у виявленні вад програми. Також важливо враховувати покриття тестами, щоб переконатися, що всі можливі сценарії виконання програми були протестовані.
