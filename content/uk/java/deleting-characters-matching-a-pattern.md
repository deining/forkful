## Чому

У програмуванні часом виникає потреба видаляти символи, які відповідають певному шаблону. Наприклад, якщо ви хочете очистити текст від зайвих символів або перетворити вхідні дані в певний формат, вам може знадобитися даний процес.

## Як це зробити

Для виконання цієї задачі у Java існує кілька способів. Один з них - використання методу replaceAll() з класу String. Даний метод приймає два параметри: шаблон, за яким необхідно видаляти символи, і рядок, у якому треба це зробити.

```java
String input = "Привіт! Який красивий день!";
String pattern = "\\p{Punct}"; //шаблон, що видаляє всі розділові знаки
String output = input.replaceAll(pattern, ""); //видаляємо всі розділові знаки зі строки

System.out.println(output); //Виведе: Привіт Який красивий день
```

Якщо вам потрібно видалити лише певні символи, ви можете використовувати інший шаблон, наприклад, "\\d" для видалення всіх цифр.

## Детальний огляд

Шляхом використання методу replaceAll() ви можете видаляти не лише окремі символи, а й цілі слова, рядки або навіть речення. Наприклад, якщо ви хочете видалити всі слова, що містять літеру "а", ви можете використати такий шаблон: "\\w*а\\w*", де "\\w" позначає будь-який символ або цифру.

Також, щоб видаляти символи з певної позиції рядка, ви можете використовувати метод substring() з класу String. Наприклад, якщо ви хочете видалити всі символи з 3 по 6 позицію, код буде виглядати наступним чином:

```java
String input = "Привіт! Який красивий день!";
String output = input.substring(0,2) + input.substring(7); //видання всі символи, крім зайвих

System.out.println(output); //Виведе: Прив. Який красивий день
```
