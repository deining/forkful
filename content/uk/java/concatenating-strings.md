## Що & Чому?

Конкатенація рядків - це процес об'єднання двох або більше рядків значень в один рядок. Програмісти використовують цей метод для створення нових рядків з даних, які вони вже мають, або для створення смислових повідомлень.

## Як це зробити:

```Java
String str1 = "Привіт";
String str2 = "світ!";
System.out.println(str1 + " " + str2);
```

Вивід: "Привіт світ!"

Ми можемо використовувати оператор "+" для об'єднання рядків разом. Ми також можемо використовувати метод `concat ()` для злиття рядків.

## Глибока занурення:

Конкатенація рядків не завжди була простою. Завдяки введенню типу даних `StringBuilder` в Java 5, створення та оптимізація довших рядків стало значно легшим. Використання `StringBuilder` замість звичайної конкатенації забезпечує оптимальну продуктивність.

Існують також інші методи об'єднання рядків, такі як `format()`, який дозволяє вставляти значення в рядок за допомогою псевдо-коду. Також є багато сторонніх бібліотек, які надають додаткові можливості для об'єднання рядків у Java.
