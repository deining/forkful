## Для чого

Генерування випадкових чисел є важливою частиною багатьох програм та ігор, де необхідно створювати випадкові елементи для покращення реалістичності або непередбачуваності.

## Як це зробити

```Java
// Імпортуємо необхідну бібліотеку
import java.util.Random; 

// Створюємо об'єкт класу Random
Random random = new Random(); 

// Генеруємо випадкове ціле число в діапазоні від 0 до 10 
int randomNumber = random.nextInt(11); 

// Виводимо результат
System.out.println("Випадкове число: " + randomNumber); 
```

В результаті, ви отримаєте випадкове число в діапазоні від 0 до 10. Завдяки механізму випадковості, це число може бути будь-яким, тому у кожен раз, коли ви виконуєте код, результат буде різним.

## Глибоке занурення

Генерування випадкових чисел полягає в використанні алгоритмів та методів для створення послідовності чисел, що виглядають як випадкові. В Java для цього використовується клас Random, який знаходиться в стандартній бібліотеці.

Зауважте, що в генерації випадкових чисел немає повної випадковості, оскільки алгоритми базуються на початковому значенні (названому "зерном"), а потім продовжують генерувати числа з використанням певних формул та операцій. Також слід пам'ятати, що необхідно правильно використовувати генератор випадкових чисел, оскільки неправильне використання може призвести до повторюваності результатів.
