## Чому

Генерація випадкових чисел є необхідною функцією для багатьох програм. Вона дозволяє створювати випадкові дані, які можуть бути використані для тестування, симуляцій, криптографії та інших цілей.

## Як це зробити

Для генерації випадкових чисел у Java ми будемо використовувати клас `Random`. Для цього спершу буде необхідно створити його інстанцію. Потім можна викликати метод `nextInt()` для отримання випадкового цілого числа, або `nextDouble()` для отримання випадкового числа з плаваючою точкою.

```Java
Random random = new Random();
int randomNumber = random.nextInt(100); // отримуємо випадкове ціле число від 0 до 99
double randomDouble = random.nextDouble(); // отримуємо випадкове число з плаваючою точкою від 0 до 1
System.out.println(randomNumber);
System.out.println(randomDouble);
```

В результаті виконання цього коду ми отримаємо випадкове ціле число та випадкове число з плаваючою точкою в консолі.

```
58
0.867283
```

## Глибше вдаємося

Шляхом застосування алгоритмів та генерування великого числа випадкових чисел, ми можемо створити ще більш складну функцію для генерації випадкових даних у Java. Одним з популярних алгоритмів є алгоритм Лемера, який використовує лінійну конгруєнтну формулу для генерації випадкових чисел.

Незважаючи на свою популярність, важливіше враховувати безпеку при генерації випадкових чисел у програмах, що використовують криптографічні функції. Для цього рекомендується використовувати бібліотеки безпечного випадкового числа, такі як `SecureRandom` у Java.
