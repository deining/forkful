## Що & Чому?

Генерація випадкових чисел - це процес створення чисел, які не мають ніякого впливу на користувача або програму. Програмісти використовують генерацію випадкових чисел для створення різноманітності в програмах, таких як комп'ютерні ігри, криптографічні застосунки та тестування програм.

## Як?

```Java
// Генерація випадкового цілого числа в діапазоні від 0 до 10
int randomNumber = (int) (Math.random() * 10);

// Генерація випадкового числа типу double від 0 до 100
double randomDouble = Math.random() * 100;
```

**Вихідні дані:** 
```
randomNumber = 7
randomDouble = 89.106234268
```

## Глибокий пір
Історичний контекст: перші методи генерації випадкових чисел були розроблені в 19 столітті класом зірок. У сучасних програмах використовуються складні алгоритми, які враховують велику кількість факторів для забезпечення максимальної випадковості.

Альтернативи: можна використовувати генерацію псевдовипадкових чисел, які базуються на початковому числі (seed), але їх випадковість може бути прогнозована.

Деталі реалізації: генерація випадкових чисел в Java базується на класі Math, який містить методи для генерації різних типів даних. Також можна створювати спеціальні об'єкти типу Random для більш сложних операцій.
