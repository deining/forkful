## Чому

Створення тимчасового файлу є важливою складовою частиною програмування в Java. Використання тимчасових файлів ​​може бути корисним при збереженні даних проміжних результатів, які будуть видалені після закінчення програми або сесії.

## Як створити тимчасовий файл

Для створення тимчасового файлу в Java, вам потрібно використовувати клас `File` і його метод `createTempFile()`. Перший параметр методу - це префікс для назви тимчасового файлу, а другий - суфікс. Відповідно, у файла буде назва `prefixXXXXXsuffix`, де `XXXXX` - це унікальний числовий ідентифікатор. Ось приклад коду для створення тимчасового файлу та його надання імені:

```Java
File tempFile = File.createTempFile("myFile", ".txt");
System.out.println("Назва мого тимчасового файлу: " + tempFile.getName());
```

Результат:

```
Назва мого тимчасового файлу: myFile15123.txt
```

## Розглянути детальніше

Щоб розібратись з тимчасовими файлами більше, трохи варто дізнатися ще один параметр методу `createTempFile()`. Його третім параметром є батьківська директорія файлу, вказана як об'єкт класу `File`. Це дає можливість визначити каталог, де буде зберігатися тимчасовий файл. За замовчуванням, він буде зберігатися в системному тимчасовому каталозі. Ось приклад з використанням третього параметра:

```Java
File tempFile = File.createTempFile("myFile", ".txt", new File("C:/temp"));
```
