# З чого почати з використання регулярних виразів у Java

Регулярні вирази є потужним інструментом для обробки тексту та знаходження певних шаблонів у рядках. Вони дозволяють ефективно перевіряти, чи відповідає певний текст заданому шаблону, здійснювати пошук та заміну рядків та багато іншого. Це корисний інструмент для розробників, який може покращити продуктивність і якість коду.

## Як використовувати регулярні вирази у Java

Для початку роботи з регулярними виразами у Java необхідно імпортувати пакет `java.util.regex`. Далі необхідно створити об'єкт `Pattern`, який містить регулярний вираз, і використовувати його для створення об'єкту `Matcher`, який буде здійснювати пошук в певному рядку. За допомогою методів `find()` та `matches()` можна перевірити, чи відповідає рядок заданому шаблону. Також варто знати про спеціальні символи та символьні класи, що допоможуть збільшити потужність регулярних виразів.

```
import java.util.regex.*;

public class RegularExpressionsExample {
    public static void main(String[] args) {
        // Створення об'єкту Pattern з регулярним виразом
        Pattern pattern = Pattern.compile(".*\\d.*");
        
        // Створення об'єкту Matcher для рядка "Hello 123"
        Matcher matcher = pattern.matcher("Hello 123");
        
        // Перевірка, чи є в рядку числа
        System.out.println(matcher.find()); // true
        
        // Перевірка, чи відповідає рядок заданому шаблону
        System.out.println(matcher.matches()); // true
    }
}
```

## Глибше про використання регулярних виразів

Крім основних функціональностей пошуку та перевірки рядків, регулярні вирази також мають ряд додаткових можливостей. Наприклад, за допомогою групування можна організувати регулярний вираз у такий спосіб, щоб отримати бажані частини вхідного рядка. Також можна використовувати квантифікатори для визначення кількості повторень заданого символу або групи символів.

```
import java.util.regex.*;

public class RegularExpressionsExample {
    public static void main(String[] args) {
        // Створення об'єкту Pattern з регулярним виразом
        Pattern pattern = Pattern.compile("([A-Z]+)\\s+(\\d+)");
        
        // Створення об'єкту Matcher для рядка "Hello 123"
        Matcher matcher = pattern.matcher("Hello 123");
