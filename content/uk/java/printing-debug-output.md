## Що & Чому? 
Виведення дебаг-вивода є процесом виведення додаткової інформації про виконання програми з метою полегшення пошуку та усунення помилок. Програмісти зазвичай роблять це для відстеження значень змінних та визначення місця виникнення помилок у коді.

## Як зробити: 
Нижче наведено приклади коду та відповідного виведення дебаг-вивода для Java програми.

```java
// показує значення змінної типу int
int num = 5;
System.out.println("Значення num: " + num);
// виводиться: Значення num: 5 

// показує результат виклику методу
String str = "Hello";
System.out.println("Розмір рядка str: " + str.length());
// виводиться: Розмір рядка str: 5
```

## Вдетальному:
Виведення дебаг-вивода часто використовувалося у ранніх версіях мов програмування для відладки, оскільки зберігання та виведення даних не було так просто, як сьогодні. Альтернативою використанню ```System.out.println()``` є використання спеціальних інструментів для відладки, таких як дебагери. Що до реалізації виведення дебаг-вивода, він може бути здійснений за допомогою інтерфейсу ```java.lang.Object``` та методу ```toString()```.
