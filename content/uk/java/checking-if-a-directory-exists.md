Що й чому?

Перевірка наявності каталогу є процесом перевірки, чи існує певний каталог на комп'ютері. Програмісти часто використовують цей процес для забезпечення того, що певний каталог існує перед його використанням для збереження або відкриття файлів.

Як це зробити:

```Java
import java.io.File;

public class DirectoryCheck {
    public static void main(String[] args) {
        File directory = new File("C:/Folder");
        if (directory.exists()) {
            System.out.println("Каталог існує.");
        } else {
            System.out.println("Каталог не існує.");
        }
    }
}
```

Вихідні дані для цього прикладу будуть залежати від того, чи існує каталог "C:/Folder" на вашому комп'ютері.

Глибші відомості:

Перевірка наявності каталогу є основною операцією в програмуванні, що дозволяє перевірити, чи існує певний каталог, щоб уникнути помилок у випадку його відсутності. Існують інші способи перевірки наявності каталогу, наприклад, використовуючи метод `isDirectory()` класу `File`. Якщо каталог не існує, можна створити його за допомогою методу `mkdir()`, або зробити це автоматично за допомогою методу `mkdirs()`, який також створює будь-які необхідні батьківські каталоги. 

Додаткові джерела:

Для отримання більш докладної інформації про перевірку наявності каталогу дивіться документацію для класу `File` та його методів. Можна також знайти корисні приклади коду та обговорення на форумах програмістів.
