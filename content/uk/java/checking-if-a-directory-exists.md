## Чому

Перевірка наявності директорії є важливою частиною програмування в Java. Це допомагає переконатися, що необхідна директорія існує перед тим, як розпочати виконання коду, що може використовувати цю директорію.

## Як

### Перевірка наявності директорії за допомогою методу exists ()

```Java
File directory = new File("path/to/directory"); 
// Ініціалізація об'єкта директорії

if(directory.exists()){ 
    // Перевірка наявності директорії
    System.out.println("Директорія існує.");
} else{
    System.out.println("Директорія не існує.");
}
```

Вивід залежить від того, чи існує вказана директорія. Якщо так, то на екрані з'явиться повідомлення "Директорія існує". Якщо директорія не існує, то з'явиться повідомлення "Директорія не існує". 

### Перевірка наявності директорії за допомогою методу isDirectory ()

```Java
File directory = new File("path/to/directory"); 
// Ініціалізація об'єкта директорії

if(directory.isDirectory()){ 
    // Перевірка, чи є даний об'єкт директорією
    System.out.println("Директорія існує.");
} else{
    System.out.println("Директорія не існує.");
}
```

Цей метод перевіряє, чи є вказаний об'єкт директорією. Якщо так, то на екрані з'явиться повідомлення "Директорія існує". Якщо об'єкт не є директорією, то з'явиться повідомлення "Директорія не існує".

У Java, файли і директорії представлені об'єктами класу File. Клас File містить набір методів для роботи з файлами і директоріями, включаючи методи для перевірки наявності, створення, перейменування і видалення. Для перевірки наявності директорії використовуються методи exists () і isDirectory ().
