## Що і чому? 
Interpolating a string - це спосіб створення нового рядка шляхом комбінування інших рядків і змінних. Програмісти використовують цей метод для зручної роботи з рядками та створення динамічного виводу.

## Як це зробити?
Наприклад, для виведення повідомлення користувачу з ім'ям, ви можете скористатись наступним кодом:
```Java
String name = "Іван";
System.out.println("Привіт, " + name + "!");
```
Результат: Привіт, Іван!

Якщо ви хочете включити значення змінної в середині рядка, ви можете скористатись символом ```\```.
Наприклад:
```Java
int age = 30;
System.out.println("Я " + age + " років.");
```
Результат: Я 30 років.

## Глибше погляньмо
"Interpolation" - це термін, який походить з математики та означає розрахунок значень між даними точками. У програмуванні, крім конкатенації рядків, існують ще кілька способів зробити те ж саме за допомогою методів класу ```String```, наприклад ```String.format()```, ```StringBuilder```, або ```String.join()```.
