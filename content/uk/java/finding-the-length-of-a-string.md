## Зачем

Для чого ми хочемо знаходити довжину рядка? Часто в програмуванні нам потрібно знати кількість символів у рядку, щоб виконати певні дії з цими даними. Наприклад, перевірити, чи не перевищує рядок максимально допустиму кількість символів або виконати обчислення залежно від його довжини. У статті ми розглянемо як саме це зробити за допомогою мови програмування Java.

## Як виконати

Для того, щоб отримати довжину рядка в Java, ми можемо скористатися методом `length()`, що повертає кількість символів у рядку. Наприклад:

```Java
String str = "Привіт, світ!";
int length = str.length();
System.out.println(length); // виведе 13
```

Ми також можемо використовувати цей метод для порівняння довжин двох рядків:

```Java
String str1 = "Привіт";
String str2 = "До побачення";
if(str1.length() > str2.length()){
    System.out.println("Рядок str1 довший за str2");
} else {
    System.out.println("Рядок str2 довший за str1");
}
```

## Deep Dive

При роботі з рядками, ми часто стикаємося з різноманітними завданнями, пов'язаними з їх довжиною. Наприклад, нам може знадобитися видалити певну кількість символів з початку або кінця рядка. Для цього ми можемо скористатися методами `substring()` та `trim()`. Або, можливо, нам потрібно взнати кількість символів після певної позиції у рядку. У цьому випадку можна використовувати метод `indexOf()`, який повертає індекс першого входження заданого символу або рядка.

Надалі, ці знання можуть бути корисними при роботі зі стрічками у будь-якому проекті, де потрібно обробляти текстові дані.
