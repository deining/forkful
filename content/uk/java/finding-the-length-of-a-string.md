## Що & Чому?

Знаходження довжини рядка - це одна з найбільш поширених операцій, яку виконують програмісти при роботі зі стрічками в своїх програмах. Це дозволяє отримати кількість символів у рядку, що може бути корисно при обробці та маніпулюванні цими даними.

## Як?

Для знаходження довжини рядка використовується метод `length()`, який доступний в класі `String` у мові Java. Давайте подивимось на приклад коду:

```Java
String str = "Привіт, світ!";
int length = str.length();
System.out.println("Рядок '" + str + "' має довжину " + length + ".");
```

В результаті виконання цього коду ми отримаємо наступне виведення:

```
Рядок 'Привіт, світ!' має довжину 13.
```

Як бачите, метод `length()` повертає ціле число - кількість символів у рядку. Це дозволяє нам легко опрацьовувати ці дані у подальшому коді.

## Глибший погляд

Знаходження довжини рядка - це операція, яку виконують програмісти вже давно. У перших версіях мови Java, для знаходження довжини рядки використовувався метод `getSize()`. Однак, у більш нових версіях, для цього був використаний більш збільшений назву - `length()`, що дозволяє більш зручно працювати зі стрічками. Також, існують різні альтернативні способи знаходження довжини рядка, наприклад, за допомогою циклу `for` або `while`, але використання методу `length()` є найефективнішим і швидким способом.
