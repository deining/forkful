## Чому

Багато людей можуть здатися, що пошук довжини рядка є простим і не потребує особливих знань або навичок. Однак, знання цього навичка може виявитися дуже корисним у роботі зі строковими данними і покращити ефективність програми.

## Як

Для пошуку довжини рядка в Java, використовується метод `length()`, який повертає ціле число - кількість символів у рядку.

```Java
// Створення рядка з текстом
String text = "Привіт, світ!";

// Виведення довжини рядка
System.out.println(text.length());

// Виведення результата: 12
```

Зверніть увагу, що пробіли інклюзивно рахуються як символи. 

## Deep Dive

Під капотом метод `length()` використовує властивості масивів - рядок можна розглядати як масив символів. Тому, щоб отримати довжину рядка, метод просто підраховує кількість елементів в цьому масиві.

Важливо також відмітити, що з введенням нового механізму для рядків в Java 11, змінні рядки тепер зберігаються в спеціальному об’єкті `java.lang.String`. Тому змінних рядків більше не можна розглядати як масиви символів, але метод `length()` все ще працює так само.
