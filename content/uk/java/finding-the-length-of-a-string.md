## Чому

Чомусь, може трапитися ситуація, коли нам потрібно визначити довжину рядка в нашій програмі. Наприклад, ми можемо хотіти перевірити, що користувач ввів достатньо символів у формі або можемо потребувати повернути певну частину рядка в залежності від його довжини. У такому випадку, нам знадобиться знати, як знайти довжину рядка в Java.

## Як

Щоб знайти довжину рядка в Java, ми можемо скористатися методом `length()` у класі `String`. Давайте подивимося на приклад коду, який показує цей метод у дії:

```Java
String name = "John";
int length = name.length();
System.out.println("Довжина рядка: " + length);
```

Виведе: `Довжина рядка: 4`, оскільки слово "John" має 4 символи.

Можемо також використовувати цей метод разом з іншими методами, наприклад `substring()`, щоб отримати підрядок з рядка в залежності від його довжини. Подивимося на інший приклад коду:

```Java
String sentence = "Це речення має 23 символи.";
int length = sentence.length();
String trimmed = sentence.substring(0, length - 1);
System.out.println(trimmed);
```

Виведе: `Це речення має 23 символи`, оскільки ми вирізали останній символ з рядка за допомогою методу `substring()`. Як бачите, визначення довжини рядка допомагає нам працювати з ним більш ефективно і зручно.

## Глибоке дослідження

За лаштунками, метод `length()` у Java використовується для повернення значення змінної `count` у класі `String`. Коли ми створюємо новий об'єкт рядка, його значення передається до конструктора об'єкта `String`, який встановлює змінну `count`. Це значення потім використовується методом `length()` для повернення довжини рядка.

Також варто зазначити, що метод `length()` не враховує пробіли в рядку, тому якщо ви хочете врахувати їх у довжині, ми можемо використовувати метод `trim()`, який видаляє всі пробіли з початку та кінця рядка.
