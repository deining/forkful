## Чому

Іноді при написанні програм, важливо правильно оформити текстові дані, щоб вони виглядали зрозуміло та організовано. При знайомстві з Java, ви напевно почули про процес капіталізації рядка. Це простий, але дуже корисний процес, який може бути використаний для зміни форматування тексту. 

Але чому ми повинні хвилюватися про капіталізацію рядків? Давайте розберемось детальніше у цій статті.

## Як

Для початку, давайте розглянемо, які кроки потрібно виконати, щоб скапіталізувати рядок в Java.

Найпростіший спосіб - використовувати клас `String`. Він має вбудований метод `toUpperCase()`, який перетворює рядок у верхній регістр. Наприклад:

```Java
String text = "Привіт світ!";
String capitalizedText = text.toUpperCase();

System.out.println(capitalizedText); // ВИВІД: ПРИВІТ СВІТ!
```

Можемо також скапіталізувати лише перший символ рядка, використовуючи метод `substring()`:

```Java
String text = "пРиВіТ СвіТ!";
String firstLetter = text.substring(0, 1).toUpperCase();
String restOfText = text.substring(1);

String result = firstLetter + restOfText;

System.out.println(result); // ВИВІД: ПриВіт Світ!
```

Це прості, але ефективні способи капіталізації рядка. Не забувайте, щоб повертати оригінальний регістр після виконання необхідних операцій.

## Глибинне дослідження

Важливо знати, що кожен символ у рядку має свій власний ASCII-код. Ці коди обробляються при виконанні операцій капіталізації. Наприклад, ASCII-код символу "a" дорівнює 97, а "A" - 65.

Крім того, метод `toUpperCase()` використовує локалізовані дані, що означає, що він може поводитися по-різному, залежно від мови, в якій працює програма.

Якщо ви використовуєте більш складні символи, може виникнути проблема з правильною капіталізацією. Тому важливо розуміти принципи роботи методів, щоб уникнути незручностей у майбутньому.
