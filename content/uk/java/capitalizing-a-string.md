## Чому

В цій статті ми поговоримо про метод аперціювання ряду символів. Він може бути корисним для тих, хто працює з рядками і хоче зробити їх вигляд більш читабельним.

## Як це зробити

Загальна ідея застосування методу аперціювання полягає в тому, що він робить перший символ ряду великим, а всі інші - малими. Давайте подивимося на приклад коду на мові Java:

```java
public class CapitalizeString {
    public static void main(String[] args) {
        String str = "приклад стрічки для аперціювання";
        System.out.println(capitalize(str));
    }

    private static String capitalize(String str) {
        if (str == null || str.isEmpty()) {
            return str;
        }
        return Character.toTitleCase(str.charAt(0)) + str.substring(1).toLowerCase();
    }
}
```
У вихідному коді ми спочатку визначаємо рядок, який буде аперціонованим, а потім викликаємо метод capitalize, передаючи йому цей рядок. На виході ми отримуємо вже аперціонований рядок.

```bash
# Вихід:
Приклад стрічки для аперціювання
```

Цей метод також можна застосувати до рядків з більш ніж одним словом, враховуючи правила розділення слів.

## Глибше о аперціонуванні рядків

У методі capitalize використовується метод toTitleCase з класу Character, який робить перший символ рядка великим. Щоб зробити всі інші символи малими, ми використовуємо метод toLowerCase з класу String.

Важливо зазначити, що метод capitalize не змінює оригінальний рядок, а повертає новий аперціонований рядок. Також, він враховує різні формати символів, наприклад, літери з діакритичними знаками.
