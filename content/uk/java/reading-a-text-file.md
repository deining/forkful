# Як це зробити

Зчитування текстового файлу у Java просте, тому що мова програмування надає нам кілька зручних класів та методів для цього. Проте, перш ніж перейти до коду, ми повинні зрозуміти що таке текстовий файл. Такий файл є послідовністю символів, таких як букви та цифри, які зберігаються на комп'ютері. Іноді текстовий файл може також містити символи, що не мають певного значення, такі як пропуски та розділові знаки.

Отже, почнемо зі створення об'єкту для текстового файлу у Java, використовуючи клас `File`.

```Java
File file = new File("myFile.txt");
```

У цьому прикладі ми створюємо об'єкт з назвою "myFile.txt". Тепер для читання цього файлу нам потрібно використати клас `Scanner`.

```Java
Scanner scanner = new Scanner(file);
```

Зараз ми маємо об'єкт `scanner`, який ми можемо використовувати для читання текстового файлу. Наприклад, якщо ми хочемо вивести вміст нашого файлу на екран, ми можемо використати метод `nextLine()` для построчного читання тексту.

```Java
while (scanner.hasNextLine()) {
    String line = scanner.nextLine();
    System.out.println(line);
}
```

Даний код буде виводити на екран кожен рядок з файлу "myFile.txt". Також, ми можемо використати метод `next()` для читання окремих слів з файлу.

```Java
while (scanner.hasNext()) {
    String word = scanner.next();
    System.out.println(word);
}
```

Цей код виведе на екран кожне слово з файлу. Також, ми можемо скористатися методом `nextInt()` для читання цілих чисел з файлу чи `nextDouble()` для читання десяткових чисел.

# Глибоке занурення

Якщо ви хочете знати більше про цей процес, ви можете також вивчити про інші класи, які допоможуть вчити зчитувати текстовий файл у Java. Наприклад, клас `BufferedReader` зручний для читання великих файлів, оскільки він використовує буфер для ефективного читання даних з файлу.

```Java
BufferedReader reader = new BufferedReader(new FileReader(file));
String line;
while ((line = reader.readLine()) != null) {
    System.out.println(line);
}
```
