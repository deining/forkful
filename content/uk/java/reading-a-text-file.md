## Чому

Як програмістам важливо мати навички читання текстових файлів у своєму арсеналі? Текстові файли є основною формою зберігання та обміну даними в більшості програм, і вміння ефективно працювати з ними є необхідним для успіху у роботі з програмними проектами.

## Як це зробити

Щоб прочитати текстовий файл у Java, ми використовуємо клас `Scanner`. Для початку, нам потрібно імпортувати цей клас у нашому коді:

```java
import java.util.Scanner;
```

Далі, нам потрібно створити новий екземпляр класу `Scanner` з файлом, який ми хочемо прочитати:

```java
Scanner scanner = new Scanner(new File("файл.txt"));
```

Тепер, ми можемо використовувати метод `nextLine()` для читання кожного рядка у файлі та виведення його на екран:

```java
while (scanner.hasNextLine()) {
  String line = scanner.nextLine();
  System.out.println(line);
}
```

Нижче наведений приклад вхідного файлу та його відповідного виводу:

**Вхідний файл "файл.txt":**

```
Привіт, це перший рядок!
Ось другий рядок.
І нарешті, третій рядок.
```

**Вивод на екран:**

```
Привіт, це перший рядок!
Ось другий рядок.
І нарешті, третій рядок.
```

## Глибший детальний огляд

При роботі з великими текстовими файлами, може бути корисним використати відповідний метод для парсингу окремих даних, таких як числа або рядки. Наприклад, метод `nextInt()` може бути використаний для читання наступного цілого числа у файлі, а метод `next()` - для читання наступного слова.

Також, варто зазначити, що клас `Scanner` показує кращі результати при читанні файлів у форматі UTF-8, тому важливо вказувати цей формат при створенні нового екземпляру.
