## Чому

Якщо ви вивчаєте програмування, читання текстового файлу є важливою навичкою, яку ви повинні засвоїти. Це дозволить вам працювати з даними збереженими у текстовому форматі, таким як CSV або JSON. Також, ви можете використати цю навичку для створення програм, які прочитають і оброблять дані з файлів, що генеруються іншими програмами.

## Як

Для початку, нам потрібно створити об'єкт класу `File`, щоб вказати шлях до текстового файлу, який ми хочемо прочитати. Потім, ми створюємо об'єкт класу `Scanner`, якому передаємо цей файл як параметр. Робимо це, щоб використати методи класу `Scanner`, щоб прочитати дані з файлу.

```Java
File file = new File("положення_файлу.txt");
Scanner scan = new Scanner(file);
```

Тепер, ми можемо використати метод `nextLine()` об'єкта `scan`, щоб отримати наступний рядок з файлу. Цей метод повертає рядок з даними, або `null`, якщо ми досягли кінця файлу. Також, ми повинні закрити об'єкт `Scanner` після того, як ми закінчили читати файл.

```Java
String line = scan.nextLine();
System.out.println(line);
```

Ви можете повторити цей процес, щоб прочитати кожен рядок файлу. Можна також використовувати інші методи класу `Scanner`, такі як `nextInt()` або `nextDouble()`, щоб прочитати цілі або дійсні числа з файлу.

## Глибокий розбор

У наведеному прикладі ми використали метод `nextLine()` для отримання рядка з файлу. Але цей метод не розрізняє закінчення рядку, тобто символи `"\n"`, `"\r"` або їх комбінації. Якщо ви хочете отримати рядок без цих символів, використовуйте метод `next()`.

Також, можна використовувати конструкцію `while` для прочитання всього файлу до кінця.

```Java
while (scan.hasNext()) {
    String line = scan.next();
    System.out.println(line);
}
```

Крім того, об'єкт `Scanner` може працювати із різними типами даних, що знаходяться в рядках. Наприклад, ви можете використовувати метод `nextBoolean()` для отримання булевого значення, або `nextBigDecimal()` для отримання числа з плаваючою точкою.
