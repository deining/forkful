## Що і для чого?
Порівнювання двох дат - це процес визначення, яка з двох дат є ранішою або пізнішою. Програмісти часто порівнюють дати, щоб визначити порядок подій або для перевірки того, що дата пізніша, ніж певний час.

## Як це зробити:
```Java
 LocalDate firstDate = LocalDate.of(2021, 10, 10); // перша дата
 LocalDate secondDate = LocalDate.of(2021, 10, 15); // друга дата

 // порівняння за допомогою методу compareTo()
 int result = firstDate.compareTo(secondDate);
 // результат буде -1, оскільки перша дата раніша за другу

 // порівняння за допомогою методу isBefore()
 boolean isBefore = firstDate.isBefore(secondDate);
 // результат буде true, оскільки перша дата раніша за другу
```

## Глибокий погляд:
Порівнювання дат було важливим завданням ще з початку програмування. Раніше, для порівняння дат, використовувався рядок чи числове значення, яке представляло дату. Однак, з появою Java 8, була додана нова бібліотека для роботи з датами і часом - java.time. Ця бібліотека містить різноманітні методи для порівняння дат, такі як compareTo(), isBefore(), isAfter() та інші.

Існують й інші способи порівняння дат, такі як використання операторів порівняння (>, <, =), але вони працюють правильно лише для коректно встановлених дат. Використання методів класу LocalDate забезпечує коректність порівняння для будь-яких дат.
