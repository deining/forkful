## Чому

Ви можете хотіти видаляти символи, які відповідають певному шаблону, наприклад, якщо ви хочете очистити рядок від зайвих символів або застосувати правила форматування до тексту.

## Як це зробити

Програмне забезпечення Fish Shell має вбудовану важливу функцію `string match`, яка дозволяє знайти збіг між текстом та шаблоном. Використовуючи цю функцію, ми можемо видалити символи, які відповідають заданому шаблону, використовуючи наступний синтаксис:

```Fish Shell
set my_string "Welcome to Fish Shell!"

set pattern "space"

set result (string match -rv $pattern $my_string)

echo $result
```

В результаті ми отримаємо рядок "WelcometoFishShell!", де пробіли були видалені, оскільки вони відповідали заданому шаблону "space". Ми також можемо видаляти символи за допомогою інших шаблонів, таких як "digit" (цифра) або "alnum" (букви та цифри). Для отримання детальнішої інформації про використовувану команду `string match`, виконайте команду `man string match` у терміналі.

## Глибоке дослідження

Хоча `string match` є досить зручною командою для видалення символів за шаблоном, вона має деякі обмеження. Наприклад, ця команда видаляє лише перший збіг символів і не дозволяє видаляти символи у середині рядка. Також варто враховувати, що ця команда виконується відносно повільно, коли шаблон складається з великої кількості символів.

Для більшого контролю над видаленням символів за шаблоном ми можемо використовувати регулярні вирази. Наприклад, ми можемо використовувати команду `sed` для видалення всіх комбінацій букв та цифр, які потрапляють під шаблон `[[:alnum:]]`:

```Fish Shell
set my_string "Fun8#Coding!"

set result (echo $my_string | sed 's/[[:alnum:]]//g')

echo $result
```

В результаті ми отримаємо рядок "#", оскільки всі букви та цифри були видалені з оригінального рядка. Це може бути корисно, якщо ви хочете видалити лише певну категорію символів з рядка, а не всі відповідні шаблону.
