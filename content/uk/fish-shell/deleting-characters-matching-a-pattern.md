## Для чого

Існує багато причин, чому ви можете захотіти видалити символи, які відповідають певному шаблону. Наприклад, ви можете потребувати видалити всі пробіли зі свого тексту перед подальшою обробкою даних.

## Як це зробити

```Fish Shell
echo "Це текст з пробілами" | tr -d ' '
```

Результат:

```Fish Shell
Цетекстзпробілами
```

У цьому прикладі ми використовуємо команду "tr" для видалення всіх символів, що відповідають певному шаблону, у нашому випадку - пробілам. Це може бути корисно при обробці даних або під час написання скриптів.

## Глибоке занурення

Існує багато інших способів видалення символів, що відповідають певному шаблону, у Fish Shell. Наприклад, ви можете використовувати регулярні вирази або команду "sed" для більш складних випадків. Також можна використовувати символьні класи для видалення певних типів символів, наприклад, цифр або літер.
