## Чому

Деякі розробники можуть замітити, що у текстовому редакторі або іншому середовищі розробки у них з'являються знаки, які походять від смайлів або емодзі. Заборонити їх можна за допомогою видалення символів, які відповідають певному шаблону. Це може бути корисно, якщо вони заважають перегляду та редагуванню коду або потрібно виконати певні операції над текстом.

## Як це зробити

Для видалення символів, які відповідають певному шаблону в Fish Shell, використовуйте команду `substitute`. Наприклад, якщо ви хочете замінити всі емодзі на порожні рядки, можна використати такий код:

```
Fish Shell
substitute -r '*' '<>' $string
```

У результаті отримаєте наступне виведення:

```
string
<><><><>
```

Крім того, можна використовувати `substitute` для видалення символів, що не належать до певного шаблону. Наприклад, якщо потрібно видалити всі символи, крім букв, цифр та пропусків, можна використати такий код:

```
Fish Shell
substitute -r '[^a-zA-Z0-9\s]' '' $string
```

У результаті отримаєте рядок, в якому залишаться тільки букви, цифри та пропуски.

## Розглянемо глибше

Ви можете визначити свої шаблони за допомогою регулярних виразів, що робить `substitute` ще більш потужним інструментом. Також у Fish Shell є можливість використовувати іншу команду - `gsub`, яка замінює всі збіги замість першого, як робить `substitute`. При цьому, пробіли біля тексту не враховуються.
