## Що і чому?
Знаходження довжини рядка - це процес вимірювання кількості символів у рядку. Програмісти часто використовують цю функцію для обробки текстових даних та перевірки їх довжини.

## Як:
```Fish Shell``` є потужним інструментом для роботи з рядками та включає в себе функції для знаходження довжини рядків. Для того, щоб знайти довжину рядка, необхідно використовувати наступну команду:
```
set string "Привіт, світе!"
echo (string | wc -m)
```

В результаті отримаємо відповідь 13, що відповідає кількості символів у рядку.

## Глибокий дайв:
Знаходження довжини рядка є важливим компонентом для обробки текстових даних. Історично цю функцію використовували для знаходження кількості символів у телеграфних повідомленнях. На сьогоднішній день існують інші засоби для вимірювання довжини рядка, такі як метод .length в JavaScript.
