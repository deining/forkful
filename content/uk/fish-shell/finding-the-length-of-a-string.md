## Чому

Наші українські читачі скоріш за все не бачили потреби визначення довжини рядка в своєму коді. Проте, це може бути корисним, коли потрібно обробляти рядки з різною довжиною або як частина складніших програм. Давайте розберемося, як це може бути зроблено за допомогою командного рядка Fish Shell.

## Як

```Fish Shell
set str "Привіт, це рядок прикладу"
echo (string length $str)
```

Вищенаведений приклад коду використовує команду `string length` для визначення довжини рядка, який ми створили змінною `str`. Код виведе кількість символів у рядку, яка в цьому випадку буде рівна 25. Пам'ятайте, що в командному рядку Fish Shell треба використовувати дужки для отримання результату виразу.

Щоб отримати реальне число байтів у рядку, можна використовувати опцію `-b` цієї команди. Наприклад, `string length -b $str` поверне 26, оскільки цей рядок складається з 25 символів української мови та одного байту (0xC2) на кожен символ. 

Зверніть увагу, що деякі шрифти можуть використовувати більше байт для відображення кирилиці, тому результат визначення довжини рядка може бути трохи більшим, ніж очікувана кількість символів.

## Deep Dive

Команда `string length` також може бути корисною при обробці файлів та потоків даних у командному рядку. Наприклад, якщо ви хочете обробити лише першу половину рядка, використовуючи символ `-s` для вказання початкового індексу, а закінчуєте при виведенні результуючого значення командою `head`:

```Fish Shell
string length "Hello World" | string sub (head -c (math round (string length)/2))
```

Тут ми обчислюємо довжину виразу "Hello World" (11 символів), ділимо її на 2 (5,5), заокруглюємо вниз до найближчого цілого числа (5) та використовуємо результат для обрізки рядка. Сподіваємося, вам це допоможе краще зрозуміти суть команди `string length` та її потенціалу у вашому коді.
