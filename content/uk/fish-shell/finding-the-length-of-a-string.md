## Чому

Найбільш поширеним завданням при програмуванні є обрахування довжини рядка. Це важливо для багатьох різних операцій, таких як обрізання рядка або перевірка його довжини. У цій статті ми розглянемо, як за допомогою Fish Shell знаходити довжину рядка швидко та ефективно.

## Як

Для знаходження довжини рядка у Fish Shell можна використовувати вбудовану команду `string length`. Приклад використання команди виглядатиме так:

```Fish Shell

set my_string "Це приклад рядка."

echo (string length $my_string)

```

Цей код виведе на екран 21, оскільки в рядку є 21 символ.

Також можна використовувати функцію `len`:

```Fish Shell

set my_other_string "Це інший приклад рядка."

echo (len $my_other_string)

```

Цей код також виведе на екран 30, оскільки символів у рядку є 30.

## Глибше

У Fish Shell є кілька інших способів знаходження довжини рядка, таких як використання команди `wc` або використання різних функцій типу `length`. Однак, найбільш ефективним залишається використання вбудованої команди `string length`, оскільки вона працює швидше та не вимагає додаткових пакетів.

Також важливо зазначити, що якщо рядок містить utf-8 символи, то їх кількість буде врахована при використанні вбудованої команди `string length`.
