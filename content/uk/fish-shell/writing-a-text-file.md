## Чому

Наш світ насичений інформацією і її подальше оброблення часто стає важкою задачею. Писання текстових файлів може спростити цей процес, дозволяючи зберігати важливу інформацію у структурованому та автоматизованому форматі.

## Як

Кодуючі приклади та вихідні дані міститимуться в блоках коду "```Fish Shell ... ```"

```Fish Shell
# Створення нового файлу з допомогою fish shell команди
touch new_file.txt

# Додавання тексту до файлу
echo "Привіт, світ!" >> new_file.txt

# Відображення вмісту файлу
cat new_file.txt
```

В результаті ми отримаємо `Привіт, світ!` у вмісті нашого файлу `new_file.txt`.

## Глибоке занурення

При написанні текстових файлів, можна використовувати різні команди та опції для збереження та обробки інформації. Наприклад, команда `cp` дозволяє копіювати файли, а `mv` - переміщати файли в інші директорії. Також, ви можете використовувати команди для фільтрації та обробки вмісту файлів, наприклад `grep` та `sed`.
