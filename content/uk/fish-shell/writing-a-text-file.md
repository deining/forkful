## Чому

Написання текстового файлу є важливою частиною програмування, оскільки воно дозволяє зберігати дані та розробляти скрипти. Також це може бути корисно для створення звітів або расподілу інформації з іншими користувачами. 

## Як

Необхідно відкрити термінал та виконати команду `touch <ім'я файлу>` для створення нового файлу. Далі можна редагувати файл за допомогою будь-якого текстового редактора. 

```Fish Shell
touch test.txt
nano test.txt
```

Після редагування, файл потрібно зберегти натиснувши `Ctrl+O`, а потім вийти з редактора, натиснувши `Ctrl+X`. 

## Глибокий занурення

Можна використовувати команду `echo <текст> > <ім'я файлу>` для додавання тексту до файлу без потреби відкривати його у редакторі. Також можна використовувати команду `cat <ім'я файлу>` для перегляду вмісту файлу. 

```Fish Shell
echo "This is a test file." > test.txt
cat test.txt
```
