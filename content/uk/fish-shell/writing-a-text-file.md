## Чому

Написання текстового файлу - це важлива частина програмування на Fish Shell. Вона дозволяє зберігати та організовувати ваші дані для подальшого використання.

## Як

Написання текстового файлу дуже просте у Fish Shell. Для цього потрібно використовувати команду `echo`, за якою слідує текст, який ви хочете записати у файл. Наприклад:

```Fish Shell
echo "Привіт, світ!" > hello.txt
```

Цей приклад створить текстовий файл `hello.txt` зі строкою "Привіт, світ!". Також, ви можете використовувати різні ключові слова, наприклад, `>>` для додавання тексту до вже існуючого файлу.

```Fish Shell
echo "Здрастуйте, світ!" >> hello.txt
```

Цей приклад додасть строку "Здрастуйте, світ!" до вже існуючого файлу `hello.txt`.

## Глибокий занурення

Крім простого створення текстового файлу, Fish Shell також має можливість використовувати пайпи та редирект для обробки та організації даних. Наприклад, ви можете виконати запит до іншого файла та перенаправити результат до нового файлу:

```Fish Shell
cat other_file.txt > new_file.txt
```

Цей приклад виконає запит до файлу `other_file.txt` та перенаправить результат до нового файлу `new_file.txt`.

## Спробуйте на практиці

Спробуйте вищезазначені приклади для створення та обробки текстових файлів у Fish Shell. Це може бути корисне для збереження та організації різних видів даних.
