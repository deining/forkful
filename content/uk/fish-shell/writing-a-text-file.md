## Чому

Написання текстового файлу - це важлива частина програмування для збереження та передачі даних іншим користувачам.

## Як

Написання текстового файлу за допомогою Fish Shell є дуже простим. Використовуйте команду ```echo``` для створення текстового файлу та використовуйте ```>``` для збереження даних у цей файл. Наприклад:

```
echo "Привіт, світ!" > hello.txt
```
Це створить текстовий файл з назвою "hello.txt" та збереже у ньому написану фразу.

## Deep Dive

Текстові файли є важливою частиною програмування, оскільки вони дозволяють зберігати дані безпосередньо на комп'ютері та використовувати їх для подальшої обробки та аналізу. Крім того, вони є ключовими для роботи з іншими програмами та скріптами.

Також, зустрічаються різні формати текстових файлів, такі як .txt, .md або .csv, що мають свої особливості та використовуються для різних завдань. Детальніше можна дізнатися в наступних розділах:
