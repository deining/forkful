Що і чому?

Запис файлу тексту - це коли ви створюєте файл, який містить текстові дані. Програмісти часто записують текстові файли, такі як код програми, для збереження і подальшого використання.

Як це зробити?

Записати файл тексту можливо за допомогою команди ```Fish Shell echo```, яка дозволяє вам вивести текст у термінал або в файл. Наприклад, щоб створити новий файл з назвою "hello.txt" і записати в нього слово "привіт", використайте такий код:
```
Fish Shell echo "привіт" > hello.txt
```
Це створить новий файл з введеним текстом. Щоб переглянути вміст цього файлу, можете використати команду ```Fish Shell cat```, наприклад:
```
Fish Shell cat hello.txt
```
Це повинно вивести вміст файлу, тобто слово "привіт".

Докладніше

Запис текстових файлів є важливою частиною програмування, від якої залежить збереження та використання різноманітних текстових даних. У деяких інших оболонках команд, зокрема в Bash, також є команда для запису файлів, але Fish Shell використовує команду ```echo``` для цієї мети.
