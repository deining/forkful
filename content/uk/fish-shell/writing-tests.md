## Чому

Написання тестів є важливою частиною процесу програмування. Вони допомагають забезпечити якість програмного коду, виявити можливі проблеми та зберегти час і гроші в майбутньому. Розглянемо, як написати тести у Fish Shell.

## Як це зробити

Для написання тестів у Fish Shell ви можете використовувати команду `confirm`. Ця команда перевірить, чи робота вашої програми відповідає очікуваному результату. Давайте розглянемо приклад:

```Fish Shell
# Пишемо функцію для обчислення суми двох чисел
function sum_numbers
  set sum $argv[1] + $argv[2]
  echo $sum
end

# Виклик функції і перевірка результату
confirm "Результатом має бути 10" sum_numbers 5 5 
```

Результат виконання цього тесту буде виглядати так:

```
Тест пройдено успішно: Результатом має бути 10
```

Якщо результат буде відрізнятися від очікуваного, то тест не пройде, і виведеться повідомлення про помилку.

## Глибший погляд

Написання тестів у Fish Shell також можна виконувати за допомогою команди `explain`, яка забезпечує більш детальну інформацію про несправність коду. Також ви можете використовувати різні умови для перевірки роботи програми, наприклад, `contains` для перевірки наявності певного елементу в результаті.

Для написання більш складних тестів, вам може знадобитися використовувати інші інструменти, такі як програми для автоматизації тестування. Рекомендуємо ознайомитися з такими інструментами, як `Bats` і `Shellspec`.
