## Що & Навіщо? 

Перетворення дати з рядка - це процес виділення дати з текстового рядка і перетворення її в формат, що може бути використаний в програмі. Це часто використовується програмістами для обробки даних та забезпечення правильності подальшої обробки.

## Як це зробити:

Fish Shell має вбудовані засоби для розбору дат із рядків. Для того, щоб виконати це завдання, ви можете використати команду `date` разом з опціями `-f` для вказання формату дати і `-s` для передачі рядка, який потрібно розібрати. Наприклад:

```
Термінал> date -f "%d. %b %Y" -s "12. серпня 2020"
1614673073
```
В результаті ми отримали числове значення, що відповідає даті в форматі UNIX. Також, ви можете використати команду `strftime` для отримання дати у вказаному форматі. Наприклад:

```
Термінал> strftime "%Y-%m-%d" "12. серпня 2020"
2020-08-12
```

## Глибше вдивимось:
Історичний контекст: Розбір дат з рядків є дуже поширеним завданням для програмістів вже багато років. Подібна функціональність з'явилася ще у ранніх версіях мови програмування C, а згодом була перенесена в більшість сучасних мов програмування.

Альтернативи: Парсинг дат з рядків можна виконати і в інших шелах, наприклад, в Bash. Також існують спеціальні бібліотеки для парсингу дат у великому спектрі мов програмування.

Деталі реалізації: Для парсингу дат Fish Shell використовує функції `date` та `strftime`, які базуються на C-бібліотеці для роботи з часом.
