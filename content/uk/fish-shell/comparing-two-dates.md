## Чому

У цьому блозі ми будемо говорити про порівняння двох дат у Fish Shell програмуванні. Оскільки дати є важливою частиною багатьох програм, знання як порівнювати їх буде корисним для будь-якого програміста.

## Як це зробити

Для порівняння дат у Fish Shell потрібно використовувати команду `date`, яка повертає поточну дату і час. Вона має декілька флагів, які дозволяють вказати формат дати, який ви хочете отримати. Наприклад, `date +"%Y%m%d"` поверне поточну дату у форматі рік-місяць-день. Цей формат символізується як: `%Y` - повний рік, `%m` - місяць (01-12), `%d` - день (01-31).

Для порівняння дат потрібно використовувати логічні оператори `>` (більше), `<` (менше) або `=` (рівне). Наприклад, якщо ви хочете порівняти чи дата, яку ви визначили, більша за поточну дату, ви можете використовувати наступну команду:

```Fish Shell
if test (date +"%Y%m%d") ">" "20200101"
    echo "Визначена дата більша за поточну дату."
end
```

В цьому прикладі ми використовуємо `test`, щоб перевірити умову, і якщо вона виконується, ми виводимо повідомлення. Ви можете замінити флаг `>` на інший, якщо хочете порівняти дати з іншим форматом.

## Поглиблене дослідження

Для більш складних порівнянь дат у Fish Shell можна використовувати команду `set -l`, яка дозволяє зберігати значення у змінну. Наприклад, ви можете порівняти дати у форматі рік-місяць-день за допомогою наступного коду:

```Fish Shell
set -l date1 "2019-12-31"
set -l date2 "2020-01-01"

if test $date1 ">" $date2
    echo "Дата 1 більша за дату 2."
end
```

Також ви можете використовувати умови `and` та `or` для складніших порівнянь. Наприклад, якщо ви хочете порівняти, чи дата, яку ви визначили, є в межах певного періоду, ви можете записати наступну умову:

```Fish Shell
if test $date1 ">" "20190101" and $date1 "<" "20201231"
    echo "Дата 1 знаходиться у межах 2019-2020 років."
end
```
