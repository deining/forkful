## Чому

Якщо ви хочете порівняти дві дати у Fish Shell, цей структурований захід допоможе вам зрозуміти, як це зробити.

## Як це зробити

```Fish Shell``` надає кілька корисних команд для порівняння дат, таких як ```date``` і ```strftime```. Для того, щоб порівняти дві дати, ви можете використати команду ```test```, яка дозволяє перевірити умови для певного виразу.

Наприклад, якщо ви хочете перевірити, чи одна дата більша за іншу, ви можете використати такий код:

```Fish Shell
set date1 (date "+%s" -u)
set date2 (date "+%s" -u -d "2 days")

if test $date1 -gt $date2
echo "Перша дата більша за другу."
end
```

Цей код використовує команду ```set```, щоб створити дві змінні - ```date1``` і ```date2```, які містять першу і другу дати. Потім використовується команда ```test```, щоб перевірити, чи вираз ```$date1 -gt $date2``` є істинним (тобто, чи перша дата більша за другу). Якщо умова справджується, то виводиться повідомлення "Перша дата більша за другу".

## Поглиблене вивчення

Для більш складних порівнянь дат, ви можете використовувати команду ```strftime``` для перетворення дат у спеціальний формат, щоб їх можна було порівнювати. Наприклад, якщо ви хочете порівняти дві дати за місяцем, ви можете використати наступний код:

```Fish Shell
set date1 (strftime "%m" (date "+%F" -d "1 month"))
set date2 (strftime "%m" (date "+%F" -d "2 months"))

if test $date1 -eq $date2
echo "Дати мають однаковий місяць."
end
```

У цьому прикладі, спочатку командою ```date``` визначається перша дата, а потім використовується команда ```strftime``` для перетворення дати у формат місяця (```%m```). Цей формат дозволяє порівняти тільки місяці, і якщо вони співпадають, то виводиться повідомлення "Дати мають однаковий місяць".
