## Тому чому

Наші життя повні різноманітними подіями, зустрічами та термінами. Іноді нам потрібно порівняти дві дати, щоб зрозуміти, яка була спочатку або скільки часу пройшло між ними. Це особливо корисно для роботи з базами даних або для створення звітів. У цій статті ми поговоримо про те, як порівнювати дві дати за допомогою команди Fish Shell.

## Як це зробити

Існує кілька способів порівнювати дві дати у Fish Shell. Найпростіший спосіб - використовувати команду `date`. Давайте подивимося на приклад:

```Fish Shell
date -d "10 days ago"
```

Результат буде наступним:

```
Mon Oct 19 11:38:02 EEST 2020
```

У цьому прикладі ми порівнюємо поточну дату з датою, яка була 10 днів тому. Ви можете змінювати дати та форматування, використовуючи різні функції команди `date`.

## Глибші занурення

Правильне порівняння двох дат може бути складною задачею. У Fish Shell є декілька функцій, які можуть допомогти нам у цьому. Однією з них є `datef`. Ця функція дозволяє зчитувати дати з різних форматів та порівнювати їх. Ось приклад використання:

```Fish Shell
set -l date1 (datef -i "%d-%m-%Y" "30-11-2020")
set -l date2 (datef -i "%d-%m-%Y" "16-05-2020")
if test $date1 -gt $date2
  echo "$date1 було пізніше за $date2"
else if test $date2 -gt $date1
  echo "$date2 було пізніше за $date1"
else
  echo "Обидві дати рівні"
end
```

Функція `datef` може читати дати з різних форматів та перетворювати їх у числа, що дозволяє нам проводити порівняння. Зверніть увагу, що ми використовуємо флаг `-i` для вказівки вхідного формату та `-g` для вказівки, що ми хочемо отримати число. Для отримання більш детальної інформації про функцію `datef`, ви можете виконати `man datef`.
