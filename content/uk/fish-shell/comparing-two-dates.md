## Що & Чому?

Порівняння двох дат - це один з найбільш поширених завдань для програмістів. Воно полягає в тому, щоб порівняти дві дати, щоб визначити, яка з них є ранішою або пізнішою. Це корисно для багатьох програм, які потребують обробки даних за певний період часу.

## Як це зробити:

Більшість програмістів використовують вбудовану функцію для порівняння дат в Fish Shell. Для цього вам потрібно використати команду "test", а потім вказати дві дати в форматі "рррр-мм-дд" як аргументи. Наприклад:

```Fish Shell
test 2021-01-01 -gt 2020-01-01
```
Результатом буде значення "1", оскільки перша дата більша за другу. Якщо обидві дати будуть однакові, то результат буде "0". Якщо ж перша дата менша за другу, то результат буде "-1".

Також, у Fish Shell існує спеціальна функція $date1 -nt $date2, яка порівнює дати на раніше або пізніше. Наприклад:

```Fish Shell
set date1 2021-01-01
set date2 2020-01-01
if $date1 -nt $date2
	echo "Дата 2021-01-01 більша за дату 2020-01-01"
end
```
Результатом буде виведений рядок "Дата 2021-01-01 більша за дату 2020-01-01".

## Глибокий погляд:

Порівняння дат має важливе значення для багатьох алгоритмів та програм. У мові Fish Shell є різні способи порівняння дат, наприклад, за допомогою команди "test" або функції "$date1 -nt $date2". Також, є альтернативи для порівняння дат в інших шелах, наприклад, в Bash Shell. Детальніше про це можна прочитати на сторінці документації Fish Shell.
