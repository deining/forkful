## Чому

Додавання функції порівняння двох дат може значно полегшити життя програмістам, що працюють з датами у своїх проектах. Це дозволяє здійснювати швидке порівняння дат для розрахунків або умовних виразів.

## Як

```Fish Shell``` має вбудований оператор порівняння дат ```-gt``` (більше ніж) та ```-lt``` (менше ніж), що дозволяє зробити порівняння двох дат за допомогою вбудованої команди ```date```.

```fish
set date1 (date -d "20 April" +%s)
set date2 (date -d "15 April" +%s)

if test $date1 -gt $date2
    echo "Перша дата більше ніж друга дата!"
else if test $date1 -lt $date2
    echo "Перша дата менше ніж друга дата!"
else
    echo "Дати рівні!"
end
```

В результаті ви отримаєте вивід, що заголовки для порівняння «X-d-m» означають місяць, день та рік: 
```bash
Перша дата більше ніж друга дата!
```

## Deep Dive

Коли ви використовуєте функцію порівняння двох дат у Fish Shell, важливо пам'ятати, що дати повинні бути у форматі, який може бути зрозумілим для вбудованої команди ```date```. Якщо ви користуєтеся іншим форматом, можливо, вам знадобиться змінити його на потрібний для правильної роботи функції порівняння.

Також слід бути обережним з обробкою великих чисел. Якщо дати надіслати у форматі, що перевищує максимальне значення, то функція може повернути неправильний результат. Для запобігання цьому, можна скористатися умовою, щоб перевірити, чи не перевищує значення дат максимальне.
