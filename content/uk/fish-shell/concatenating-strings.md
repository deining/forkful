# Що і чому?

Конкатенація рядків - це комбінування двох або більше рядків в один рядок. Це корисний інструмент для програмістів, коли потрібно об'єднати різні рядки тексту для подальшої обробки або виводу.
Також це допомагає уникнути повторної письмової роботи, коли потрібно створити рядок, що складається з однакових частин.

# Як:

Щоб конкатенувати рядки в Fish Shell, використовуйте команду "string concat". Перший аргумент цієї команди - це рядок, який буде використовуватися як "посилення", тобто міститься в кінці створеного рядка.
Наступні аргументи - рядки, які будуть конкатеновані до першого рядка в тому ж порядку, що і були передані у команду.
Ось приклад коду:

```
Fish Shell 3.2.0
string concat I am a Fish user, and I love to use string concatenation
```

Результат виведення:

```
I am a Fish user, and I love to use string concatenation
```

# Глибока занурення:

Конкатенація рядків існує в багатьох мовах програмування, включаючи C, Java, і Python.
Альтернативою команді "string concat" в Fish Shell може бути використання оператора "+" для конкатенації рядків.
Це дає більш читабельний код, але вимагає більше роботи для розбиття рядків на окремі частини.
Реалізація конкатенації рядків в Fish Shell базується на функції "sprintf" з бібліотеки Squirrel, що дозволяє передавати змінну кількість аргументів у функцію.
