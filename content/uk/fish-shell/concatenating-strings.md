## Чому

Перетворення рядків у рядки є важливою частиною роботи зі скриптами рослинного шаблонування. Це дає можливість створювати більш динамічні та зручні скрипти, що пристосовані до різних ситуацій.

## Як це зробити

Простий спосіб склеїти два рядки - це використання оператора `+`:

```Fish Shell
set greeting "Привіт"
set name "Олена"
echo $greeting$name
```

Ви отримаєте вивід `"ПривітОлена"`.

Однак, якщо ви хочете додати додатковий роздільник між рядками, наприклад кому, ви можете використовувати команду `string join`:

```Fish Shell
set first_name "Михайло"
set last_name "Ковальчук"
echo (string join ", " $first_name $last_name)
```

Ви отримаєте `"Михайло, Ковальчук"`.

## Поглиблене дослідження

У Fish Shell є багато умовних операторів, які дозволяють вам більш гнучко працювати зі стрічками. Наприклад, ви можете використовувати умовний оператор `string length`, щоб перевірити довжину рядка перед його об'єднанням з іншими рядками.

Крім того, в Fish Shell є багато функцій для перетворення рядків, які дозволяють вам зробити більш складні операції зі стрічками, наприклад заміну підрядка або видалення символів.
