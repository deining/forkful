##Чому 

Для багатьох програмістів вивчення генерації випадкових чисел є важливою навичкою для створення різноманітних ігор, симуляцій та інших застосувань. Також, це може бути корисно для створення тестових даних або випадкових вибірок для аналізу даних. У цій статті ми розглянемо, як генерувати випадкові числа в Fish Shell.

##Як це зробити

Для генерації випадкових чисел у Fish Shell є вбудована команда `random`. Давайте спробуємо створити випадкове число від 1 до 10.

```
Fish Shell> random 1 10
8
```

Також, ми можемо використовувати цю команду для створення списку випадкових чисел. Наприклад, ми хочемо створити список з 5 випадкових чисел від 1 до 100.

```
Fish Shell> for i in (seq 1 5)
            random 1 100
61
100
31
56
93
```

Також, ми можемо скористатися командою `math` для створення складніших випадкових чисел, наприклад, створимо випадкове дробові число від 0 до 1.

```
Fish Shell> random 0 1
0.5498047
```

##Заглиблюємось

Генерація випадкових чисел відбувається за допомогою псевдовипадкового генератора, який використовується для створення послідовності чисел, які виглядають випадково. Цей генератор використовує випадкове початкове значення, яке називається "seed", яке може бути змінений для отримання різних послідовностей випадкових чисел.

Також, варто зазначити, що вбудована команда `random` використовує 31-бітний псевдовипадковий генератор, тому немає гарантії повної випадковості чисел, якщо вам потрібно використовувати їх у криптографічних цілях чи серйозних наукових дослідженнях.
