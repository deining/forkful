## Чому

Генерація випадкових чисел є важливою галуззю програмування, особливо для випадків, коли потрібні випадкові дані для створення тестових прикладів, генерації криптографічних ключів або моделювання випадкових подій.

## Як

Найпростіший спосіб згенерувати випадкове число використовуючи Fish Shell - це використати команду `math random` у поєднанні з командою `echo`:

```
Fish Shell

echo (math random)
7цб7/35а
```

Ви можете також задати межі для випадкового числа, використовуючи опції `min` і `max`:

```
Fish Shell

# генерує випадкове число від 10 до 100
echo (math random --min 10 --max 100)
35.66561
```

Щоб отримати цілі числа, щоб використовувати в коді, ви можете застосувати опцію `round`:

```
Fish Shell

# генерує випадкове ціле число від 1 до 10
echo (round (math random --min 1 --max 10) )
6
```

## Глибоке занурення

У Fish Shell, генерування випадкових чисел базується на функції `urandom`, яку використовують більшість операційних систем для створення випадкових даних. Ця функція використовує ентропію з джерела випадкових подій в системі, таких як покажчики часу і клавіші.

Більш детальну інформацію про алгоритми генерації випадкових чисел в Fish Shell можна знайти в [документації](https://fishshell.com/docs/current/cmds/math.html#math-random).
