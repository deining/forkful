## Навіщо
Програмування за допомогою оболонки Fish є швидким та ефективним способом отримати доступ до веб-сторінок за допомогою вашого терміналу. Це особливо корисно для тих, хто працює з багатьма веб-сторінками або хоче автоматизувати процес отримання даних з Інтернету.

## Як використовувати
Код для завантаження веб-сторінки за допомогою Fish виглядає наступним чином:

```Fish Shell
# Завантаження веб-сторінки за допомогою команди `curl`
curl <URL>  # URL - адреса веб-сторінки

# Завантаження веб-сторінки у файл
curl <URL> > <Назва файлу>  # Назва файлу - назва вашого файлу з розширенням, наприклад, index.html

# Завантаження веб-сторінки за допомогою команди `wget`
wget <URL>  # URL - адреса веб-сторінки

# Завантаження веб-сторінки у файл
wget -O <Назва файлу> <URL>  # Назва файлу - назва вашого файлу з розширенням, наприклад, index.html
```

Вивід цих команд буде залежати від того, який тип веб-сторінки ви завантажуєте. Зазвичай, ви побачите HTML-код сторінки, але це може бути будь-який тип даних, який доступний для завантаження з Інтернету.

## Глибше
Скачування веб-сторінки через термінал дозволяє вам отримати доступ до сторінки без використання вашого браузера. Це особливо корисно для автоматизації завантаження великої кількості сторінок або для програми, яка використовує дані з Інтернету.

Крім того, оболонка Fish має велику кількість корисних функцій, які полегшують використання команд для завантаження веб-сторінок. Наприклад, ви можете скопіювати шаблон команди і замінити URL або назву файлу, щоб швидко завантажити нову сторінку. Ви також можете використовувати змінні, що дозволяють створювати болісну сторінку мапування, яка тільки надає щонайменше метаданів для додаткового кодування.
