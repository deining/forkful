## Чому

Завантаження веб-сторінки може бути корисним для ознайомлення з новою інформацією, отримання доступу до ресурсів або перевірки стану веб-сайту.

## Як це зробити

```Fish Shell``` дозволяє нам просто і швидко завантажити веб-сторінку, використовуючи команду ```curl``` і спеціальний символ ```>```, який дозволяє передати результат у файл. Наприклад:

```
curl https://www.example.com > page.html
```

Це завантажить веб-сторінку і збереже її у файлі з назвою ```page.html```, який можна відкрити для перегляду. Також можливо використовувати команду ```wget``` для завантаження веб-сторінки. Наприклад:

```
wget https://www.example.com
```
 
Це завантажить сторінку і збереже її під назвою, яка буде автоматично створена за допомогою адреси сайту, наприклад ```index.html```. Якщо необхідно створити нову папку для збереження завантаженої сторінки, можна використовувати команду з прапорцем ```-P```, який вказує шлях до нової папки. Наприклад:

```
wget -P ./pages https://www.example.com
```
 
Це завантажить сторінку і збереже її у папці ```pages```, яка буде розміщена у поточній директорії.

## Глибоке занурення

Існує багато різних команд і прапорців, які можна використовувати для завантаження веб-сторінок, наприклад, можна указати кількість повторів завантаження за допомогою прапорця ```-r```, або ігнорувати некоректні сертифікати за допомогою прапорця ```--insecure```. Для більш детальної інформації про всі можливості команд і прапорців варто переглянути документацію або використовувати флаг ```--help```. Також є можливість використовувати різні скрипти і додаткові програми для автоматизації завантаження веб-сторінок.
