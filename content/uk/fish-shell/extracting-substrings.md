Привіт, дорогі українські читачі! Сьогодні ми поговоримо про одну корисну функцію у Fish Shell - видобування підрядків. Як саме це працює та навіщо це потрібно? Це ми розкриємо у нашому блозі, тому тримайтеся.

## Чому
Видобування підрядків - це надзвичайно зручна функція для обробки текстових даних. Вона дозволяє вибирати лише ту частину рядка, яка нам потрібна, а не працювати з усім текстом цілком. Таким чином, ми можемо ефективніше обробляти та аналізувати дані.

## Як використовувати
Давайте розглянемо декілька прикладів використання видобування підрядків у Fish Shell.

```Fish Shell
set text "Hello, world!"
set substring (echo $text | string sub --start 7 --end 11)
echo $substring
```

В цьому прикладі ми створюємо змінну `text` з текстом "Hello, world!", а потім застосовуємо функцію `string sub` для видобування підрядка від 7-го до 11-го символу. Результатом буде "worl", який ми виводимо на екран.

Далі давайте розглянемо більш складний приклад, де ми будемо використовувати регулярні вирази.

```Fish Shell
set email "example123@email.com"
set regex "^[\w.-]+@([a-z0-9]+[.])+[a-z]{2,4}$"
set substring (echo $email | psub --pattern $regex)
echo $substring
```

У цьому прикладі ми визначаємо змінну `regex` з регулярним виразом для перевірки правильності електронної адреси. Потім застосовуємо функцію `psub` для пошуку відповідного підрядка у строкі `email`. Результатом буде "email.com", який ми також виводимо на екран.

## Глибше
Видобування підрядків у Fish Shell використовує функцію `string sub` або `psub` для вибору частини рядка за допомогою індексів або регулярних виразів. Для вибору індексів можливо використовувати або додатні, або від'ємні значення, що дозволяє рахувати з кінця рядка.

Також варто знати, що при використанні функції `psub`, результатом буде масив підрядків, що відповідають регулярному виразу.
