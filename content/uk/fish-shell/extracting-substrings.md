## Для чого
Іноді при написанні скріптів або команд у Fish Shell потрібно вилучити певний фрагмент тексту зі строки. Наприклад, для подальшого використання цього фрагменту в якості змінної або для перевірки умов. У цьому випадку, витягнення підрядка допомагає зробити це швидко та ефективно.

## Як це зробити
Для вилучення підрядка зі строки в Fish Shell, ви можете використовувати команду `string sub` та спеціальний оператор `|` для подальшої обробки. Наприклад:

```
Fish Shell > set str "Hello, world!"
Fish Shell > echo $str | string sub -b 7 -e 10
worl
```

У цьому прикладі ми спочатку створюємо змінну `str` зі значенням "Hello, world!". Потім, використовуючи команду `echo`, ми передаємо це значення до команди `string sub` разом з параметрами `-b` та `-e`, які вказують, відповідно, початковий та кінцевий індекси для вилучення підрядка. У результаті, виводиться лише підрядок "worl".

Варто зазначити, що обидва параметри `-b` та `-e` є необов'язковими. Якщо не вказувати початковий та кінцевий індекси, `string sub` вилучатиме весь рядок. Також, за допомогою параметра `-r` можна вилучити підрядок зверху до початкового індексу, а `-l` - знизу до кінцевого індексу.

## Глибше

Крім стандартної команди `string sub`, Fish Shell також надає додаткові утиліти для роботи з підрядками, такі як `string split`, `string match` та `string replace`. Крім того, ви можете використовувати регулярні вирази для вилучення підрядків зі строк за допомогою команди `string match -r`. Ви можете ознайомитися з більш детальною документацією на офіційному сайті Fish Shell.
