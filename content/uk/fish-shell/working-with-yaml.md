## Чому

Робота з YAML є важливим навичком для будь-якого програміста, оскільки він дозволяє зберігати та організовувати дані в структурованому форматі. Це робить роботу з даними більш зручною та ефективною.

## Як користуватися

Щоб працювати з YAML, вам потрібно мати знання про Фіш Шел. Спочатку необхідно встановити фреймворк "PyYAML", який дозволить обробляти дані у форматі YAML в Фіш Шел. Один з варіантів це зробити це:
```Fish Shell
sudo apt-get install python-yaml
```

Після цього, ви можете використовувати наступний код для завантаження та обробки YAML даних:
```Fish Shell
set data (python -c "import yaml;print(yaml.dump({'key': 'value'}))")
echo $data
```

Ви повинні побачити, що дані були завантажені та виведені у форматі YAML.

Якщо ви хочете дізнатися більше про роботу з YAML, варто розглянути наступні питання:
