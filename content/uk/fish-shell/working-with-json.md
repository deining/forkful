## Чому

JSON - це формат для обміну даними, що зробив його популярним серед програмістів. Робота з JSON дозволяє нам легко обмінюватися даними між різними програмами та зберігати структуровану інформацію про наші дані.

## Як

```Fish Shell``` вбудував багато функцій, які допомагають полегшити роботу з JSON. Одна з них - це ```jq```, який дозволяє нам розбирати та перетворювати дані в JSON форматі.

Наприклад, якщо у нас є JSON файл з інформацією про користувачів сайту, ми можемо вивести всі імена користувачів за допомогою наступного коду:

```Fish Shell
jq '.users[].name' users.json
```

Отримаємо наступний вихід:

```Fish Shell
"John"
"Mary"
"David"
```

Це означає, що ми вибрали всіх користувачів (за допомогою ```[].```) із полем ```name```.

## Глибоке занурення

Попередній приклад був простим, але за допомогою ```jq``` ми можемо робити набагато більше. Наприклад, ми можемо використовувати умови для фільтрації даних або створювати нові об'єкти та масиви.

Код виглядатиме приблизно так:

```Fish Shell
jq '.users | map({name: .name, age: .age})' users.json
```

Отримаємо такий вихід:

```Fish Shell
[
  {
    "name": "John",
    "age": 28
  },
  {
    "name": "Mary",
    "age": 35
  },
  {
    "name": "David",
    "age": 42
  }
]
```

Тут ми створюємо новий масив, вибираючи зі старого тільки ім'я та вік користувачів.

Більше про всі можливості ```jq``` можна дізнатися в його документації.
