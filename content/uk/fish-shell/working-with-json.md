## Чому

Робота з JSON є важливою складовою частиною у програмуванні, оскільки цей формат дозволяє зручно обмінюватися даними між різними програмами та платформами. Використання Fish Shell дозволяє швидко та ефективно працювати з JSON даними без необхідності використання додаткових інструментів.

## Як

Для початку роботи з JSON в Fish Shell необхідно встановити плагін `json`. Для цього достатньо виконати наступну команду:

```Fish Shell
fisher install jorgebucaran/fish-json
```

Після встановлення плагіну, можна починати роботу з JSON. Наприклад, давайте створимо простий JSON файл з даними про користувача:

```Fish Shell
echo '{"name": "Maria", "age": 30}' > user.json
```

Щоб переглянути дані з цього файлу, скористайтеся командою `json`:

```Fish Shell
json user.json
```

Результатом буде наступне:

```JSON
{
   "name": "Maria",
   "age": 30
}
```

Для роботи з більш складними даними, наприклад, з даними вкладених об'єктів чи масивів, можна використовувати опцію `--pretty`, яка робить виведення даних більш зрозумілим:

```Fish Shell
json --pretty user.json
```

Результат буде наступним:

```JSON
{
  "name": "Maria",
  "age": 30
}
```

Детальну інформацію про всі можливості та опції плагіну можна знайти в офіційній документації [тут](https://github.com/jorgebucaran/fish-json).

## Вивчення глибоких занять

Для більш глибокої роботи з JSON в Fish Shell можна вивчити інструменти, такі як `jq` та `jtc`, які дозволяють здійснювати більш складні операції з даними, такі як фільтрація та зміна структури. Також, вивчення більш продвинутих функціональностей Fish Shell, таких як використання змінних та функцій, допоможе зручніше та ефективніше працювати з JSON.
