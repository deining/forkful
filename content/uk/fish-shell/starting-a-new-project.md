## Чому

Стартувати новий проект може бути цікавим викликом для вас. Ви можете вдосконалити свої навички програмування та створити щось нове та корисне для спільноти.

## Як

Навчаючись програмуванню у сучасних мовах, таких як Fish Shell, ви отримуєте доступ до потужних інструментів, які дають вам змогу швидко та ефективно створювати проекти. Для початку, давайте розглянемо приклад програми "Hello World" у Fish Shell:

```Fish Shell
echo Hello World
```

Ви можете запустити цей код у своєму терміналі, та побачити результат виводу:

```
Hello World
```

Це простий приклад, але він показує, як використовувати команду `echo` та виводити текст у терміналі. Ви можете експериментувати з різними командами та побачити, як вони впливають на результат.

## Глибше

Якщо ви хочете розпочати новий проект у Fish Shell, вам необхідно буде створити та налаштувати свій конфігураційний файл `config.fish`. Це дозволить вам налаштувати різні змінні та функції для вашого проекту. Ви також можете встановити різноманітні плагіни та розширення для покращення своєї продуктивності.

Крім того, ви можете використовувати Fish Shell для автоматизації рутинних задач та швидкого виконання скриптів. Це дозволить вам економити час та зусилля у розробці свого проекту.
