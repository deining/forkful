## Тому

Розпочати новий проект - це завжди захоплююче пригода! Не тільки ви можете проявити свою творчість і вміння програмування, але також можете створити щось корисне для себе і світу.

## Як

Будь-який проект потребує робочої папки, де зберігати всі файли і код. Це може бути реальна папка на вашому комп'ютері або віртуальна папка на хмарному сховищі. У Fish Shell ви можете швидко створити нову папку за допомогою команди `mkdir`:

```Fish Shell
mkdir new_project
```

Також ви можете перейти до цієї нової папки за допомогою команди `cd`:

```Fish Shell
cd new_project
```

Тепер, коли ви в папці свого нового проекту, ви можете почати створювати файли і писати код. Наприклад, ви можете створити пустий файл за допомогою команди `touch`:

```Fish Shell
touch index.html
```

Ця команда створить новий файл під назвою `index.html` в вашій робочій папці.

## Глибокий занурення

Крім створення нової папки та файлів, ви також можете використовувати Fish Shell для автоматизації деяких рутинних завдань у вашому проекті. Наприклад, створення скриптів для збирання, тестування або встановлення вашого проекту. Або ж використовувати вже готові рішення, які ви можете знайти в Fish Shell репозиторії.

Не бійтесь експериментувати і знаходити нові, більш ефективні способи для вашого проекту. Крім того, Fish Shell має велику документацію, де ви можете знайти більше корисних команд та трюків для вашого проекту.
