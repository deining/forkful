# Що & Чому?

Робота з CSV - це формат збереження табличних даних, де значення розділяються комами. Програмісти використовують його для збереження та обробки великих обсягів даних, таких як інформація з баз даних або результати запитів з API.

# Як це зробити?

Для роботи з CSV у Fish Shell існує спеціальний інструмент - пакет «csvutil», який можна завантажити зі сторінки [GitHub respository](https://github.com/fishpkg/csvutil). Використовуючи цей пакет, ви можете легко читати та записувати дані з CSV-файлів.

Наприклад, щоб прочитати дані з CSV-файлу з назвою "data.csv", використовуйте наступний код:

```
set data (csvutil read data.csv)
```

Цей код зчитує дані з файлу та зберігає їх у змінну "data". Для отримання значень звертайтесь до цієї змінної за допомогою індексів, наприклад:

```
echo $data[1][2]
```

Цей приклад виведе друге значення з першого рядка файлу "data.csv".

Щоб записати дані у CSV-файл, використовуйте наступний код:

```
csvutil write data.csv (echo "1,2,3" "4,5,6")
```

Цей приклад створить новий CSV-файл з двома рядками: "1,2,3" та "4,5,6".

# Глибше занурення

CSV виник як спрощений формат для збереження та передачі даних в електронних таблицях. Він широко використовується в програмуванні як зручний спосіб обробки табличних даних. Основний альтернативний формат - JSON, але він менш зручний для роботи з табличними даними.

Пакет "csvutil" побудований на основі Perl-модуля з тим же ім'ям та дозволяє ефективно працювати з великими CSV-файлами. Він також підтримує популярні опції для керування розділювачом та форматуванням даних.
