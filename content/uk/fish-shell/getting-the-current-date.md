## Чому

Програмістам часто потрібно отримувати поточну дату для рішення різних задач. Наприклад, для створення логів або встановлення термінів дії токенів. Фіш-шел дозволяє отримати поточну дату швидко та без зайвих складнощів.

## Як отримати поточну дату у Фіш-шел

```Fish Shell
set date (date +"%m-%d-%Y") 
echo $date
```

Вище приведений код дозволяє зберегти поточну дату у змінній "date" та вивести її на екран. Це може бути корисно, якщо вам потрібно використати цю дату у подальших розрахунках або обчисленнях.

```Fish Shell
echo (date +"Сьогодні день тижня %A, %d %B") 
```

У цьому прикладі ми використовуємо команду "date" з параметрами, щоб отримати не тільки число, а й назву дня тижня та місяця на українській мові. Ви можете змінювати параметри, щоб отримувати бажану форму дати.

## Глибинний аналіз отримання поточної дати

Функція "date" не є окремою командою у Фіш-шел, а є частиною стандартного пакету "coreutils". Вона дозволяє форматувати та виводити поточну дату у різних форматах за допомогою спеціальних параметрів.

Наприклад, у нашому прикладі ми використовуємо "%m" для виведення номера місяця у форматі 01-12, "%d" для дня місяця у форматі 01-31, "%Y" для виведення повного року, "%A" для назви дня тижня та "%B" для назви місяця на українській мові.

Якщо вам потрібна більша гнучкість, ви можете додатково ознайомитися з параметрами команди "date", щоб створювати унікальні формати дати для своїх потреб.
