## Чому

Існує багато причин, чому люди можуть хотіти отримати поточну дату в програмі. Це може бути корисним для відображення поточного часу для користувачів або для створення умов в програмі, які залежать від дати.

## Як

Для отримання поточної дати в Fish Shell потрібно використовувати команду `date`. Наприклад, якщо ви хочете вивести поточну дату у форматі "місяць/день/рік", то команда буде виглядати наступним чином:

```Fish Shell
date +"%m/%d/%y"
```

Результат буде виглядати як `03/20/21`, де "03" - це номер місяця, "20" - день та "21" - рік.

Щоб отримати більш детальну інформацію про поточну дату, можна використовувати різні параметри команди `date`. Наприклад, параметр `+%A` покаже повну назву поточного дня тижня, а параметр `+%B` - повне назва місяця. 

```Fish Shell
date +"Сьогодні %A %d лютого %B %Y року."
```
Результат буде виглядати як `Сьогодні Субота 20 лютого Березень 2021 року.`

## Глибоке осмислення

Команда `date` отримує поточну дату та час з системи та дозволяє відформатувати їх у бажаний спосіб. Щоб побачити всі доступні параметри для форматування, можна використовувати допоміжну команду `man date`.

Також, за допомогою команди `date` можна отримати не тільки поточну дату, але й дати в минулому або майбутньому. Для цього потрібно вказати кількість днів, годин, хвилин або секунд, на яку потрібно змістити дату.
