## Чому

Перевірка існування папки - це важлива операція, яку часто використовують програмісти, щоб переконатися, що папка, яку вони хочуть використовувати, справді існує. Це дозволяє уникнути помилок та забезпечити правильну роботу програми.

## Як

```Fish Shell``` має вбудовану функцію для перевірки існування папки - ```test -d```. Давайте подивимося на приклад:

```
set folder "/Users/username/Documents"
if test -d $folder
    echo "Папка існує!"
else
    echo "Папка не існує!"
end
```

Якщо папка існує, то виведеться повідомлення "Папка існує!" інакше - "Папка не існує!". Ви також можете використовувати командний рядок для перевірки існування папки:

```
if test -d "/Users/username/Desktop"
    echo "Папка існує!"
end
```

Цей приклад показує, що ви можете використовувати будь-який шлях до папки для перевірки, а також можна використовувати змінні для її збереження.

## Глибока занурення

Окрім перевірки існування папки, ви також можете використовувати команду ```test``` для перевірки наявності файлів або директорій за іншими умовами. Наприклад, ви можете перевірити, чи є файл написаним користувачем або чи є файл доступним для читання та запису:

```
if test -f $file
    echo "Це файл!"
else if test -w $file
    echo "Файл доступний для запису!"
end
```

Також ви можете поєднувати різні умови за допомогою ```and``` і ```or``` операторів. Це дозволяє вам створювати більш складні перевірки, наприклад:

```
if test -d $folder and test -w $folder
    echo "Папка існує та доступна для запису!"
end
```
