# Що і чому?

Перевірка існування директорії - це коли програміст перевіряє, чи існує певна директорія на комп'ютері. Це важливо для забезпечення валідної роботи програми, яка може використовувати дану директорію для зберігання або доступу до файлів. 

# Як це зробити:

Найпростішим способом перевірити існування директорії у Fish Shell є використання команди ```test -d directory_name```. Це поверне значення "1", якщо директорія існує, і "0", якщо ні.

Також можна використовувати умовний оператор ```if```, який дозволяє виконувати певні дії, якщо директорія існує. Наприклад:

```
if test -d directory_name 
	echo "Директорія існує"
end
```

# Глибокий погляд:

Перевірка існування директорії була важливою частиною програмування з самого початку. Це дозволяє забезпечити коректну роботу програми, яка може використовувати директорію для різних цілей. 

Існує також інший спосіб перевірки існування директорії у Fish Shell - використання змінної $status. Ця змінна повертає значення "0", якщо команда успішно завершилась, і "1", якщо була помилка. Таким чином, можна виконати такий код:

```
test -d directory_name 
if test "$status" -eq 1 
	echo "Директорія не існує"
end
```

І ще один цікавий факт - команда "dirname" виконує операцію обернену до перевірки існування директорії. Вона повертає шлях до батьківської директорії від заданого файлу або директорії.
