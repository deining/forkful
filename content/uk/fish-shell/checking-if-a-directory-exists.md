## Чому

Перевірка, чи існує директорія, може бути корисною при створенні скриптів для керування файлами та директоріями в Fish Shell. Наприклад, ви можете перевірити, чи існує потрібна директорія, перед тим як зберегти файл у ній, щоб уникнути помилок.

## Як

```Fish Shell
if test -d "my_folder"
    echo "Директорія існує!" 
else 
    echo "Директорія не існує!" 
end
```

У цьому прикладі ми використовуємо команду `test`, яка перевіряє, чи існує директорія з назвою "my_folder". Якщо так, то виконається команда `echo "Директорія існує!"`, а якщо ні, то виконається команда `echo "Директорія не існує!"`. Після цього, ви можете додати потрібні команди, які будуть виконуватись у залежності від того, чи існує директорія.

## Глибоке погруження

Перевірка наявності директорії може бути зроблена за допомогою різних команд, наприклад `test`, `if`, `mkdir`, тощо. Також, варто знати про флаги `-d`, який перевіряє, чи існує директорія, та `-f`, який перевіряє, чи існує файл з певною назвою. Крім того, можна використовувати іменовані директорії, наприклад `$HOME` для директорії користувача чи `$PWD` для поточної робочої директорії.
