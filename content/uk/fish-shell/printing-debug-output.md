## За що

Друкування дебаг-виводу може бути корисним для відстеження помилок або налагодження програми. Це дозволяє програмістам бачити значення змінних та переконатися в правильності їх обчислень.

## Як

```Fish Shell``` має убудовану функцію ```echo```, яка дозволяє виводити повідомлення прямо в терміналі. Наприклад, щоб вивести значення змінної ```num``` можна використати наступний код:

```
echo $num
```

Також можна додати текст до виводу:

```
echo "Значення змінної num:", $num
```

Можна також використовувати оператори для додавання, вирахування та інших математичних операцій:

```
set num 10
set result $num * 3
echo "Результат:", $result
```

## Глибока зануреність

Крім використання функції ```echo```, у ```Fish Shell``` є ще декілька корисних команд для друкування дебаг-виводу:

- ```printf``` може бути корисною для форматованого виводу, наприклад, використовуючи спеціальні символи для розділення значень.
- ```status``` показує поточний код виходу програми.
- ```set_color``` дозволяє змінити колір тексту, що друкується, для більшої видимості.
- ```number``` виводить слова, що відповідають цілому числу, такі як 1 = "один", 2 = "два" та ін.
