## Чому

Управління текстовими даними є невід'ємною частиною програмування. Пошук і заміна тексту є надзвичайно корисними інструментами, які дозволяють оперативно змінювати дані в різних файлах та скриптах.

## Як використовувати Fish Shell для пошуку та заміни тексту

Fish Shell має вбудований інструмент для пошуку та заміни тексту - `sed`. Щоб почати використовувати його, ви можете відкрити термінал і виконати таку команду:

```Fish Shell
sed 's/old_text/new_text/g' file.txt
```

Ця команда замінить всі входження `old_text` на `new_text` у файлі `file.txt`. Якщо ви хочете виконати заміну безпосередньо в файлі, ви можете додати ключове слово `-i`:

```Fish Shell
sed -i 's/old_text/new_text/g' file.txt
```

Також, ви можете використовувати `sed` для заміни тексту в багатьох файлах за допомогою шаблонів:

```Fish Shell
sed -i 's/old_text/new_text/g' *.txt
```

Зверніть увагу, що `sed` виконує заміну тільки у першому входженні рядка. Якщо ви хочете замінити всі входження, ви можете додати ще один ключовий флаг `-g`:

```Fish Shell
sed -i 's/old_text/new_text/g' *.txt
```

## Занурення у пошук та заміну тексту

Тепер, коли ви знаєте, як використовувати `sed` для пошуку та заміни тексту, давайте розглянемо детальніше деякі ключові функції:

### Заміна у конкретних рядках

Щоб обмежити заміну тільки на певному рядку, ви можете використовувати числовий діапазон перед шаблоном:

```Fish Shell
sed '3,5s/old_text/new_text/g' file.txt
```

Ця команда замінить текст тільки у третьому, четвертому та п'ятому рядках файлу.

### Виключення або повторення рядка

Часто необхідно виключити або повторити певну частину рядка, щоб змінити лише деякі входження. Наприклад, ви можете виключити всі слова, що починаються зі слова "apple":

```Fish Shell
sed 's/apple[^ ]*//g' file.txt
```

Або повторити кожен рядок двічі:

```Fish Shell
sed 's/.*/& &/g' file.txt
```
