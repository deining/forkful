## Чому

Перетворення рядка на малий регістр є важливою функцією для багатьох програмістів, які працюють зі строковими даними. Це дозволяє зменшити співставлення між рядками, що покращує ефективність пошуку та порівняння.

## Як

Перетворення рядка на малий регістр у Fish Shell може бути здійснено за допомогою вбудованої функції `string tolower`. Нижче наведені декілька прикладів коду та вихідного результату для демонстрації цієї функції:

```Fish Shell
set my_string "Hello World"
echo "Lowercase string:" (string tolower $my_string)
```

Виведе: `Lowercase string: hello world`

```Fish Shell
set my_num 123
echo "Lowercase number:" (string tolower $my_num)
```

Виведе: `Lowercase number: 123`

Зверніть увагу, що функція `string tolower` повертає новий рядок, а не змінює оригінальний рядок. Тому, якщо ви хочете змінити значення змінної, необхідно присвоїти їй нове значення.

## Глибокий занурення

Для розуміння того, як працює функція `string tolower`, треба знати про кодування символів використовуване в Fish Shell. Функція використовує кодування UTF-8, де знаки певної букви мають різні коди залежно від великих та малих літер. Функція перетворює кожен символ у рядку на його еквівалент у малому регістрі, зберігаючи при цьому інші символи без змін.

Наприклад, літера "A" у кодуванні UTF-8 має значення `41`, а літера "a" має значення `61`. Функція `string tolower` буде перетворювати всі "A" на "a", залишаючи решту символів без змін.
