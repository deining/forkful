## Чому

Створення тимчасових файлів є важливою частиною програмування у Fish Shell (поточна версія). Вони дозволяють зберігати тимчасові дані, що потрібні під час виконання програми, та підтримувати чистоту та ефективність вашого коду.

## Як це зробити

Існує декілька способів створення тимчасових файлів у Fish Shell. Один з них - використовувати вбудовану команду `mktemp` (створення тимчасового файлу). Для цього давайте спробуємо зберегти дані у тимчасовому файлі, використовуючи команду `echo` (виведення на екран):

```fish
# Створення тимчасового файлу із назвою "temp_file"
mktemp temp_file

# Записуємо дані у тимчасовий файл
echo "Це тимчасовий файл" > temp_file

# Перевіряємо, чи дані було успішно записано у файл
cat temp_file
>> Це тимчасовий файл
```


Ви також можете використовувати команду `touch` (створення файлу), щоб створити тимчасовий файл і надати йому потрібні вам права:

```fish
# Створення тимчасового файлу із назвою "temp_file"
touch temp_file

# Надаємо права на запис в файл
chmod u+w temp_file

# Записуємо дані у тимчасовий файл
echo "Це тимчасовий файл" > temp_file

# Перевіряємо, чи дані було успішно записано у файл
cat temp_file
>> Це тимчасовий файл
```

## Глибока розробка

Для створення тимчасового файлу у Fish Shell використовується спеціальний синтаксис з символом `>` (редирект). Перший варіант використання команди `echo` показує, як створити тимчасовий файл, не надаючи йому жодних прав доступу. Другий варіант дозволяє надати права на запис у файл, але також потребує додаткових кроків, які може бути зручно виконувати, якщо ви працюєте з більш складними програмами.
