## Що & Чому?
Створення тимчасового файлу це процес створення файлу, який тимчасово зберігає дані під час роботи програми. Програмісти часто створюють тимчасові файли, щоб зберегти проміжні результати роботи програми, або для збереження даних, які використовуються лише один раз.

## Як це зробити:
За допомогою `Fish Shell`, створення тимчасового файлу є дуже простим. Ось приклад коду, який створює тимчасовий файл і виводить його назву:

```Fish Shell
set temp_file (mktemp)
echo "Назва тимчасового файлу: $temp_file"
```

Вихідний результат буде схожий на це:
```
Назва тимчасового файлу: /tmp/tmp.oGASKAeSdS
```

## Поглиблене дослідження:
Створення тимчасового файлу є широко використовуваним процесом в програмуванні, і його історія сягає коріння UNIX систем. Алтернативами до `Fish Shell` можуть бути команди `mktemp` або `mkstemp`, які також створюють тимчасові файли, але вимагають більше зусиль і коду для використання їх в програмі.

Імплементація створення тимчасового файлу в `Fish Shell` базується на функції `mktemp` з однойменної утиліти UNIX.
