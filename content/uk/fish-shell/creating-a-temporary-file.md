## Чому

Створення тимчасового файлу може бути необхідною частиною багатьох програм, особливо тих, що працюють з багатофайловими операціями.

## Як це зробити

За допомогою Fish Shell, створення тимчасового файлу дуже просте. Просто використовуйте команду `mktemp` з потокового буфера `echo` для створення та виведення шляху до нового файлу. Наприклад:

```
Fish Shell:
echo (mktemp)

```

Результат виводу буде схожий на наступний:

```
/ tmp / tmp.87iH7s0g2G	
```

Це означає, що новий тимчасовий файл був створений у папці `/tmp/` з унікальним ім'ям `tmp.87iH7s0g2G`. Також можна вказати додаткові параметри для `mktemp`, наприклад, розширення файлу або використовувати певний шаблон для імені. Для детальнішої інформації про параметри команди `mktemp` можна переглянути документацію за допомогою команди `man mktemp`.

## Глибокий погляд

Є декілька важливих причин, чому на практиці корисно створювати тимчасові файли. По-перше, це дозволяє уникнути перенавантаження системи і встановлення необхідних прав доступу до постійно зберіганих файлів. Крім того, використання тимчасового файлу дозволяє програмі працювати більш ефективно і збільшити швидкість виконання операцій.

Це особливо важливо в сучасних веб-додатках, де багатокористувацький штурм може призвести до масштабного використання постійних файлів. Іншим важливим використанням тимчасових файлів є тестування програм. Вони дозволяють зберігати тестові дані на окремому диску та прискорювати процес тестування.
