## Чому

Створення тимчасового файлу є важливим кроком для ефективного програмування в основі командного рядка. Тимчасові файли дозволяють зберігати та обробляти дані, які необхідно для поточного виконання коду, а потім видаляються, щоб не займати зайве місце. Давайте дізнаємося, як це зробити за допомогою Fish Shell!

## Як це зробити

Найпростіший спосіб створення тимчасового файлу в Fish Shell - використовувати команду `mktemp`. Наприклад, якщо нам потрібно створити файл з розширенням `.txt`, ми можемо виконати наступний код:

```Fish Shell
set temp_file (mktemp /tmp/temporary_file_XXXXXX.txt)
echo "Це тимчасовий файл, створений Fish Shell!" > $temp_file
echo "Вміст тимчасового файлу:"
cat $temp_file
rm $temp_file
```

Виходячи з цього коду, ми створюємо тимчасовий файл з випадковим ім'ям, використовуючи команду `mktemp` та всередині цього файлу друкуємо текст. Після цього ми видаляємо цей файл за допомогою команди `rm`.

## Глибше

Крім команди `mktemp`, у Fish Shell є ще одна корисна команда для створення тимчасових файлів - `random`. Вона дозволяє створювати випадкові числа та спеціальні знаки, які можуть бути використані для генерації випадкових імен файлів. Наприклад, `random 10` створить рандомну послідовність з 10 символів, яку можна використовувати, як ім'я для тимчасового файлу. 

Крім того, команда `mktemp -d` дозволяє створювати тимчасові директорії, які також можуть бути корисними для зберігання тимчасових файлів у відокремленому місці.
