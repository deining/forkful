## Чому

Робота з текстовими файлами є необхідною для багатьох програмістів та системних адміністраторів, оскільки вони зберігають важливі дані, які використовуються для роботи програм або налаштування систем. У цій статті ми розглянемо, як читати текстовий файл за допомогою командного рядка Fish Shell, щоб ви могли ефективно використовувати цю навичку у своїй роботі.

## Як це зробити

Для читання текстового файлу в Fish Shell нам потрібно використовувати команду «cat». Вона виводить вміст файлу на екран терміналу. Наприклад, якщо ми хочемо прочитати вміст файлу з назвою «example.txt», нам потрібно ввести наступну команду:

```Fish Shell
cat example.txt
```

Це виведе на екран увесь вміст файлу, включаючи його рядки та символи переведення рядка.

Також можна використовувати команду «less», щоб читати текстовий файл поступово. Це особливо зручно для файлів, які мають великий обсяг даних. Наприклад:

```Fish Shell
less example.txt
```

Ця команда дозволить нам прокручувати вміст файлу поступово за допомогою клавіш «вверх» та «вниз».

## Глибше

Крім звичайного читання текстового файлу, Fish Shell має кілька корисних функцій, які зроблять цей процес більш продуктивним.

Наприклад, ви можете використовувати опцію «-n» за допомогою команди «cat», щоб вивести вміст файлу з номерами рядків:

```Fish Shell
cat -n example.txt
```

Також можна вказати конкретні рядки для виведення за допомогою опції «-s»:

```Fish Shell
cat -s 2,5 example.txt
```

Це виведе лише другий та п’ятий рядки з текстового файлу «example.txt».
