## Чому

Зараз багато програм і систем використовують текстові файли для зберігання інформації. Наприклад, файл налаштувань або файл журналу. Читання цих файлів безпосередньо з терміналу може бути корисним для швидкого перегляду вмісту або пошуку конкретної інформації.

## Як

Для читання текстових файлів у Fish Shell, ви можете використати команду `cat`. Наприклад, якщо ви хочете прочитати файл налаштувань з ім'ям `settings.txt`, ви можете виконати наступну команду:
```Fish Shell
cat settings.txt 
```
Це виведе вміст файлу у вашому терміналі, що дозволить вам швидко переглянути його без відкривання додаткових програм.

## Глибоке занурення

Крім команди `cat`, Fish Shell також має багато інших вбудованих команд для роботи з текстовими файлами, таких як `head`, `tail`, `grep` та багато інших. Ви можете детальніше ознайомитися з цими командами, використовуючи документацію Fish Shell або шукаючи додаткові ресурси в Інтернеті.
