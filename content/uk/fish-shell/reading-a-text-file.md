# Причина 

Написання і читання текстових файлів є важливими та корисними навичками для програмістів усіх рівнів. Це дозволяє зберігати та організовувати дані, а також дозволяє виконувати роботу з широким спектром вхідних та вихідних форматів. Це особливо важливо для програмістів, які працюють зі скриптами та автоматизованими процесами. 

## Як це зробити 

Кодування текстових файлів та читання їх є досить простою задачею з використанням **Fish Shell**. Ось короткий приклад того, як зчитати дані з файлу та вивести їх на екран: 

```Fish Shell 
set input_file "data.txt"
set file_contents (cat $input_file)
echo $file_contents
```

Цей код встановлює змінну `input_file` для вказаного файлу та використовує команду `cat` для зчитування його вмісту у змінну `file_contents`. Наступний рядок виводить зміст файлу на екран. 

Намагайтесь використовувати команду `cat` замість набагато більшого та складнішого коду, який можна було б використовувати з іншими оболонками. 

## Глибина аналізу 

Ми розглянули лише базовий приклад, як зчитати та вивести дані з текстового файлу. Але **Fish Shell** має багато інших корисних команд для читання та обробки файлів. 

Наприклад, ви можете використовувати команду `head` для виведення перших кількох рядків файлу або `tail` для виведення останніх кількох рядків. Ви також можете використовувати різні опції та фільтри для обробки та форматування даних, які ви читаєте з файлу. Більш детальну інформацію можна знайти в документації **Fish Shell** та шукати додаткові приклади в Інтернеті.
