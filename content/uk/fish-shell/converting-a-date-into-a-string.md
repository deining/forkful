## Чому

У цій статті ми розглянемо процес конвертації дати в рядок за допомогою Fish Shell. Це корисний навик для тих, хто працює з датами та потребує зручного форматування для виведення дати.

## Як

Якщо ви хочете конвертувати дату в рядок в Fish Shell, ви можете використовувати вбудовану функцію `date` разом з форматуванням за допомогою `%s`.

```Fish Shell
# Конвертація поточної дати в рядок
echo (date +%s)
# Виведення: 1591280361
```

Більш складні формати дати також можна використовувати за допомогою `%Y-%m-%d %H:%M:%S`, де `%Y` представляє рік, `%m` - місяць, `%d` - день, `%H` - година, `%M` - хвилина, `%S` - секунда.

```Fish Shell
# Конвертація поточної дати в рядок з більш складним форматом
echo (date +%Y-%m-%d_%H:%M:%S)
# Виведення: 2020-06-04_14:46:01
```

## Deep Dive

У Fish Shell є доступ до більшої кількості форматів дати, оскільки вона використовує вбудований пакет `strftime`. Це дає можливість користувачам швидко перетворювати дату у потрібний формат. Ви також можете використовувати функції `strftime` безпосередньо для додаткового контролю над форматуванням дати.

Наприклад, якщо вам потрібно вивести дату з назвою дня тижня, ви можете використовувати значення `%A` для отримання повного назви дня.

```Fish Shell
# Конвертація поточної дати у рядок з повною назвою дня тижня
echo (date +"%A, %d %B %Y")
# Виведення: Thursday, 04 June 2020
```

Перелік доступних форматів дати можна знайти у [документації Fish Shell](https://fishshell.com/docs/current/cmds/date.html).
