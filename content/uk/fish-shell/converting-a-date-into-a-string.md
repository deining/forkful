## Для чого

"Конвертування дати у рядок" - важлива задача при програмуванні, яка дозволяє відобразити дату у зручному для користувача форматі. У цьому блозі ми розглянемо як це зробити за допомогою Fish Shell та наведемо приклади коду та результатів.

## Як це зробити

```Fish Shell
set date (date -f %Y-%m-%d)
echo $date
```

Цей простий код дозволить нам конвертувати поточну дату у форматі рік-місяць-день і зберегти його у змінну `date`. Наступним кроком буде вивід дати за допомогою команди `echo`, яка виведе рядок зі значенням змінної `date`.

Результат: `2021-07-22`

Звичайно, це простий приклад, тому давайте подивимося на більш складний код, який дозволить нам сконвертувати дату з більш детальним форматом.

```Fish Shell
set date (date -f "%A, %B %d, %Y at %H:%M:%S")
echo $date
```

Цей код дозволить нам отримати дату у вигляді "День тижня, Місяць День, Рік о %Години:%Хвилини:%Секунди". Наприклад, результат може бути `Thursday, July 22, 2021 at 10:30:05`.

## Глибока пригода

Конвертування дати у рядок є важливою функцією при програмуванні, оскільки дозволяє відобразити дату в зручному для користувача форматі. У Fish Shell це можна зробити за допомогою команди `date`, яка має ряд параметрів, що дозволяють вибрати потрібний формат дати.

Наприклад, команда `date -f "%A, %B %d, %Y at %H:%M:%S"` показана вище, використовує такі параметри:

- `%A` - повна назва дня тижня
- `%B` - повна назва місяця
- `%d` - день в місяці
- `%Y` - повний рік (4 цифри)
- `%H` - години в 24-годинному форматі
- `%M` - хвилини
- `%S` - секунди

Крім того, в Fish Shell є можливість додаткового форматування дати, наприклад, вибір часової зони або вказання формату у проміжних місцях (%) в рядку формату.
