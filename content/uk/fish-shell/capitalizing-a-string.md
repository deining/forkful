## Запекти: Затамування рядка

Функція, яка змінює першу літеру кожного слова в рядку на велику, може бути корисною в різних сценаріях. Наприклад, вона може допомогти покращити вигляд тексту або створити заголовки для вашого вмісту. У цій статті ми розглянемо, як використовувати цю функцію в Fish Shell та який вплив вона має на результат.

## Як це зробити

```Fish Shell
set string "богдан сказав, що рибка яка затамуєстя"
echo (string capitalize $string)
```

Вищевказаний код використовує вбудовану функцію `capitalize`, яка приймає рядок як аргумент та повертає змінений рядок з першою літерою кожного слова у верхньому регістрі. В результаті виконання цього коду, ми отримаємо наступний вихід: "Богдан Сказав, Що Рибка Яка Затамуєстя". Якщо ви хочете зберігти оригінальний рядок і отримати лише його копію, ви можете використовувати функцію `capitalize-preserve` замість `capitalize`.

```Fish Shell
set string "зберегти оригінальний рядок!"
echo (string capitalize-preserve $string)
echo $string
```

Виходом цих команд буде "Зберегти Оригінальний Рядок!" та "зберегти оригінальний рядок!" відповідно.

## У глибину

Здавалося б, що зміна першої літери кожного слова на велику - це просто задача рядкового маніпулювання. Але насправді, це процес зводиться до розбиття рядка на окремі слова, змінення першої літери кожного слова та злиття їх разом з використанням відповідних роздільників. І хоча це може бути трохи складніше, ніж здається на перший погляд, Fish Shell покликана полегшити цю задачу для вас.
