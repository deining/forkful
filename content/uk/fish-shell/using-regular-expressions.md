## Що та чому?

Використання регулярних виразів - це швидкий і зручний спосіб знаходити та обробляти текст на основі певних шаблонів. Це особливо корисно для програмістів, оскільки можна швидко вибрати та замінити певні частини тексту без необхідності вручну перебирати кожен символ.

## Як?

Код Fish Shell дозволяє легко використовувати регулярні вирази за допомогою функції `string match`. Нижче наведено приклад вибору усіх слів з рядка, що починаються з літери "п" та закінчуються на літеру "х":

```Fish Shell
string match -rvxp '\bp\w*x\b' example.txt
```

Результат виконання цієї команди буде наступним:

```
приклад, повідомлення
```

## Глибокий погляд

Регулярні вирази були розроблені у 1950-х роках та стали популярними серед програмістів завдяки своїй короткості та універсальності. Існують також інші засоби для обробки тексту, такі як групування та заміни, але регулярні вирази залишаються одними з найшвидших та найпотужніших інструментів.

Щоб використовувати регулярні вирази належним чином, потрібно детально ознайомитись з синтаксисом та особливостями їх застосування. Більшість мов програмування підтримують регулярні вирази, тому вони можуть бути корисними для будь-якого проекту.
