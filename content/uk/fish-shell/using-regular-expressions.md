## Чому

У програмуванні, нещодавно було дуже популярним використовувати регулярні вирази, або Regex, для обробки тексту. Вони дозволяють швидко та ефективно здійснювати пошук та заміну рядків за певними шаблонами. Крім того, вони є незамінним інструментом у фільтрації та валідації введення від користувачів. Знання регулярних виразів допоможе вам дуже швидко та ефективно здійснювати обробку тексту у своїх програмах.

## Як використовувати

Для початку, потрібно зрозуміти, що складається з регулярного виразу. Він складається з символів, які утворюють шаблон, а також з визначених спеціальних символів, які допомагають здійснювати пошук за певними правилами. Давайте подивимось на приклад:

```Fish Shell
set string "Привіт, це приклад регулярного виразу!"
if string match '^Привіт' # починається з "Привіт"
    echo "Знайдено збіг!"
end
```

У цьому прикладі, ми використали спеціальний символ "^", який означає, що ми шукаємо текст, який починається з "Привіт". Якщо це вірно, то ми виведемо повідомлення "Знайдено збіг!". Ви можете експериментувати з іншими спеціальними символами та шаблонами для вивчення їх функціональності.

## Глибока розробка

Тільки вивчення синтаксису регулярних виразів ще не дає повного уявлення про їх потужність. Наприклад, ви можете використовувати групи, щоб зводити у єдиний результат рядок, який відповідає декільком шаблонам. Також, можна застосовувати квантифікатори, які визначають кількість повторень для певного символу або шаблону. Крім того, існують техніки негативного шаблону, які дозволяють ігнорувати певні символи у тексті. Це лише декілька прикладів функціональності, яку можна дістати з використанням регулярних виразів.
