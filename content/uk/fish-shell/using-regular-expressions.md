## Чому

Теперішня версія Fish Shell пропонує багато корисних інструментів, серед яких *регулярні вирази*. Вони дозволяють здійснювати швидкий та ефективний пошук та заміну шаблонів у текстових даних.

## Як використовувати регулярні вирази в Fish Shell

Для початку, потрібно відкрити термінал і набрати команду `fish`, щоб перейти до Fish Shell. Потім, увімкніть режим регулярних виразів, введіть `set -U fish_grep`, і натисніть Enter. Тепер ви можете почати використовувати регулярні вирази в командному рядку.

```Fish Shell
set -U fish_grep
echo 'Це рядок тексту для тестування' | grep '(сторінки){e}'
```

Вище наведений приклад показує, як швидко можна знайти всі входження шаблону `(сторінки)` у заданому тексті. Крім того, ви можете використовувати регулярні вирази в командах `sed` та `awk`.

```Fish Shell
echo 'Це речення потребує заміни ' | sed -e's/заміни/заміни кольору/'
```

Вище наведений приклад демонструє, як можна змінити слово `заміни` на `заміни кольору` у заданому тексті. Регулярні вирази також можна використовувати для заміни частин тексту у великих файлах.

## Поглиблене вивчення

Регулярні вирази дозволяють виконувати швидкий та точний пошук та заміну шаблонів у текстових документах чи програмах. Вони легко адаптуються для різних мов програмування та операційних систем. Для більш детального вивчення цієї теми, рекомендуємо прочитати ["Регулярні вирази: введення та основи"](https://linux.die.net/Bash-Beginners-Guide/sect_04_01.html) та ["Інтерактивний урок з використання регулярних виразів"](https://regexone.com/lesson/introduction_abcs).
