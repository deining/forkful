## Чому

Генерація випадкових чисел є важливим інструментом в багатьох областях програмування. У Haskell, це є потужною можливістю для розв'язання різноманітних задач, таких як тестування програм або створення випадкових даних для алгоритмів машинного навчання.

## Як

```Haskell
import System.Random

randInt :: IO Int
randInt = do
  gen <- newStdGen
  return $ fst $ random gen

randList :: Int -> IO [Int]
randList n = do
  gen <- newStdGen
  return $ take n $ randomRs (1, 100) gen

main = do
  -- Випадкове ціле число
  num <- randInt
  putStrLn $ "Випадкове ціле число: " ++ show num

  -- Випадковий список цілих чисел
  list <- randList 5
  putStrLn $ "Випадковий список цілих чисел: " ++ show list
  ```

Вище наведений код використовує модуль "System.Random", який надає інструменти для генерації випадкових чисел у Haskell. Функція "randInt" повертає випадкове ціле число, а функція "randList" повертає список випадкових цілих чисел заданого розміру. Використовуючи модуль "IO", ми можемо вивести результати на екран.

Випадкові числа в Haskell генеруються за допомогою "структури зі стандартним індексом", яка має властивість виписати наступне випадкове значення при кожному виклику. Тому, для отримання більш непередбачуваних результатів, необхідно генерувати новий генератор в кожній функції.

## Глибоке погруження

У Haskell, генерація випадкових числе охоплює велику кількість можливостей. Наприклад, модуль " System.Random" також має функцію "randomR" для генерації випадкових чисел в заданому діапазоні, або функцію "randomIO" для отримання випадкового значення з будь-якого типу даних.

Для більш детального вивчення генерації випадкових чисел в Haskell, рекомендуємо ознайомитися з офіційною документацією та іншими вказаними посиланнями в розділі "See Also".
