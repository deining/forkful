## Чому

Генерація випадкових чисел є важливою та корисною функцією в програмуванні, так як вона дозволяє створювати непередбачувані значення, які можуть бути використані для роботи з даними та створення вкладених алгоритмів.

## Як це зробити

Існує кілька способів генерації випадкових чисел в мові програмування Haskell. Розглянемо кілька з них:

```Haskell
import System.Random

-- Генерування випадкового цілого числа від 1 до 10
randInt :: IO Int
randInt = randomRIO (1,10)

-- Генерування списку випадкових чисел
randList :: IO [Int]
randList = sequence $ replicate 10 randInt

-- Генерування випадкової пари чисел
randPair :: IO (Int, Int)
randPair = do
  x <- randInt
  y <- randInt
  return (x,y)

-- Виконання додаткових дій з випадковими числами
main :: IO ()
main = do
  num <- randInt
  putStrLn $ "Ваше випадкове число: " ++ show num
  lst <- randList
  putStrLn $ "Ваш список випадкових чисел: " ++ show lst
  (x,y) <- randPair
  putStrLn $ "Ваша випадкова пара чисел: " ++ show (x,y)
```

Запуск скрипту `main` дозволить нам отримати у консолі випадкове число, список та пару чисел. Для кожного нового запуску скрипту, результати будуть іншими, оскільки вони генеруються випадковим чином.

## Глибше пірнання

У мові Haskell генерація випадкових чисел здійснюється за допомогою вбудованого модуля `System.Random`. Цей модуль містить багато функцій для генерації різних типів випадкових значень, таких як цілі числа, числа з плаваючою точкою, символи та інші. Також він надає можливість вказувати межі генерації чисел та враховувати різні закономірності у процесі генерації. Наприклад, функція `randomR` дозволяє вказати діапазон для генерації чисел, а `random` використовується для генерації чисел від 0 до максимального значення типу першого аргументу.
