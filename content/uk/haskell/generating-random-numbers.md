## Що & Чому?
Генерування випадкових чисел - це процес створення недетермінованих чисел у програмі. Це корисний інструмент для програмістів, оскільки випадкові числа можуть бути використані для симуляцій, тестування та алгоритмів, які вимагають випадковості.

## Як:
```Haskell
import System.Random

-- Генерування випадкового цілого числа від 1 до 100
randomNumber :: IO Int
randomNumber = randomRIO (1,100)

-- Генерування випадкового Float числа від 0 до 1
randomFloat :: IO Float
randomFloat = randomIO

-- Повторення генерування випадкового числа
main :: IO ()
main = do
    r1 <- randomNumber
    print r1
    r2 <- randomFloat
    print r2
```

Вихід:
```
72
0.5844740424654213
```

## Глибока занурення:
Генерування випадкових чисел стало значно простіше завдяки появі пакету System.Random у Haskell. Але у цьому пакеті є деякі обмеження, наприклад, можливість повторного використання вже згенерованих чисел. Існують інші альтернативи, такі як пакет random-fu, який надає більш потужні та ефективні інструменти для генерування випадкових чисел.

Імплементація генерування випадкових чисел в Haskell базується на генераторі псевдовипадкових чисел. Це означає, що при кожному запуску програми, генератор видає послідовність чисел, які насправді не є випадковими, але мають властивості випадковості. Щоб забезпечити більшу випадковість, можна також використовувати зовнішні джерела, такі як шум коливання вологості повітря або часового маркера.
