## Чому

В програмуванні, іноді потрібно знати довжину рядка, наприклад, для валідації введених користувачем даних. У цій статті ми покажемо, як знайти довжину рядка у мові Haskell.

## Як зробити це

Існує кілька способів знайти довжину рядка у Haskell. Ми покажемо вам два з них.

### Використання функції `length`

У мові Haskell існує вбудована функція `length`, яка повертає довжину списку або рядка. Для строкових значень вона повертає кількість символів у рядку.

```Haskell
length "Привіт, світе!" -- 13
```

### Використання рекурсії

Можна також написати свою функцію для знаходження довжини рядка, використовуючи рекурсію. Для цього нам потрібно створити функцію, яка буде розбирати рядок на окремі символи і додавати одиницю до результату на кожній ітерації.

```Haskell
len :: String -> Int
len [] = 0 -- базовий випадок: для пустого рядка повертаємо 0
len (x:xs) = 1 + len xs -- розбираємо рядок на символи та додаємо 1 до результату на кожній ітерації
```

Приклад використання:

```Haskell
len "Привіт, світе!" -- 13
```

## Глибше та подробніше

Функція `length` вбудована у мову Haskell у модулі `Data.List`. Вона працює для будь-яких списків, а не тільки для рядків. Якщо вам потрібно знайти довжину списка інших типів даних, наприклад, чисел, ви можете використати цю функцію.

Також є інша вбудована функція у мові Haskell, `length'`, яка працює зі скінченними і нескінченними списками шляхом використання лінивого обчислення.
