## Причина

Якщо ви хочете працювати з рядками в Haskell, важливо знати, як знаходити їх довжину. Ви можете використовувати цей метод для перевірки довжини вхідного рядка або для ітерації по його символах ліворуч-направо.

## Як

Щоб знайти довжину рядка в Haskell, ви можете використовувати функцію `length`. Нижче наведена приклад коду і його виведення:

```Haskell
str = "Привіт, світе!"
length str -- повертає 13
```

Функція `length` використовується для знаходження довжини будь-якого списку, в тому числі і рядків. Якщо ви хочете перевірити довжину введеного рядка, ви можете використовувати функцію `getLine` для зчитування рядка з консолі та подальшого його обчислення з допомогою `length`.

## Глибокий занурення

У Haskell немає вбудованої функції, яка напряму знаходитиме довжину рядка. Замість цього, використовується операція `:` для роботи з рядками. Наприклад, повернемося до прикладу з попереднього розділу, де `str` - словник, що містить рядок `Привіт, світе!`:

```Haskell
str = "Привіт, світе!"
length str -- повертає 13
```

Тут, `str` уявляється як `(П) (р) (и) (в) (і) (т) (,) ( ) (с) (в) (і) (т) (е) (!)` (де позначення у дужках вказує на елемент списку). Таким чином, довжина рядка приблизно дорівнює кількості елементів у списку.
