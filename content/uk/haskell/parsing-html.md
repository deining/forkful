## Чому

Разбір HTML є важливою навичкою для людей, які працюють з веб-сайтами або веб-аплікаціями, оскільки він дозволяє отримувати та обробляти дані з веб-сторінок. Завдяки розбиранню HTML, можна отримати доступ до різноманітних інформаційних джерел та створювати зручні інструменти для роботи з веб-даними.

## Як зробити

```Haskell
import Text.HTML.TagSoup

page = "<body><h1>Hello, world!</h1></body>"

tags = parseTags page -- перетворюємо рядок на список тегів
header = takeWhile (~/= "<h1>") tags -- вибираємо всі теги до першого <h1>
h1 = head header -- перший елемент списку - перший тег <h1>
content = fromTagText h1 -- отримуємо текст з тегу <h1>
```

В результаті, змінна `content` буде містити рядок `"Hello, world!"`. Просто змінюючи оператори та використовуючи різні функції з бібліотеки `Text.HTML.TagSoup`, можна здійснювати складніші операції з HTML-документами.

## Детальний розбір

Код в прикладі використовує бібліотеку `Text.HTML.TagSoup`, яка надає функції для розбору HTML-документів. Ця бібліотека дозволяє перевести рядок з HTML-кодом в список з тегів, з якими далі можна працювати. Наприклад, функція `parseTags` розбирає рядок на окремі елементи, які є тегами. Кожен тег представлений як рядок.

У прикладі також використовується функція `takeWhile`, яка обрізає список тегів до того моменту, поки умова не буде виконана. В даному випадку, ми обрізаємо список до першого елементу з тегом `<h1>`. Потім ми вибираємо перший елемент за допомогою функції `head` і зберігаємо його в змінну `h1`. В результаті виходить рядок з HTML-кодом для тегу `<h1>`, який можна перетворити в звичайний текст за допомогою функції `fromTagText`.
