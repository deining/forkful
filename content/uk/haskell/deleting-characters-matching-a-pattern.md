## Чому

У цій статті ми розглянемо як видаляти символи з рядків, що відповідають певному шаблону. Ця функція може бути корисною при роботі з текстовими даними, коли потрібно прибрати зайві символи або пробіли.

## Як

Існує кілька шляхів видалення символів, що відповідають певному шаблону в Haskell. Один з них - використання функції `filter`, яка отримує на вхід списки і застосовує до них певну функцію-предикат. Результатом буде новий список, в якому залишаться лише ті елементи, що задовольняють умову предиката.

```
-- Приклад видалення цифр з рядку
removeDigits :: String -> String
removeDigits str = filter (\c -> not (c `elem` ['0'..'9'])) str

-- Виклик функції
removeDigits "abc123" -- поверне "abc"
```

Ще один спосіб - використання функції `Data.List.delete`, яка приймає на вхід елемент і список і видаляє всі входження цього елемента зі списку.

```
-- Приклад видалення символу 'a' з рядку
removeChar :: String -> String
removeChar str = delete 'a' str

-- Виклик функції
removeChar "Haskell" -- поверне "Hskell"
```

## Детальний опис

Якщо потрібно видалити символи з рядку, що відповідають певному шаблону, можна використовувати більш складні функції з пакету `Text.Regex`, який дозволяє працювати з регулярними виразами. Для цього необхідно підключити цей пакет до проекту та використовувати функції `matchRegex`, `subRegex` чи `gsubRegex`, які дозволяють знайти та замінити певні символи за шаблоном.

```
-- Приклад видалення малих літер з рядку за допомогою регулярних виразів
import Text.Regex

removeLowercase :: String -> String
removeLowercase str = subRegex (mkRegex "[a-z]") str ""

removeLowercase "Haskell" -- поверне "H   "
```
