## Чому

Існує багато ситуацій, коли необхідно змінити регістр рядка на рядок нижнього регістру. Наприклад, при порівнянні рядків або при виконанні пошуку інформації в базі даних. В таких випадках корисно знати, як конвертувати рядок до нижнього регістру.

## Як

У Haskell існує функція `toLower`, яка дозволяє конвертувати рядок до нижнього регістру. Для початку потрібно імпортувати модуль `Data.Char`:

```Haskell
import Data.Char
```

Після цього, ми можемо використати `toLower` для зміни регістру рядка:

```Haskell
toLower "HELLO" -- поверне "hello"
```

Якщо ми застосуємо цю функцію до рядка, який вже має нижній регістр, то нічого не зміниться:

```Haskell
toLower "Hello" -- поверне "hello"
```

Можна також скомбінувати функції `map` та `toLower`, щоб застосувати конвертацію до кожного символу у рядку:

```Haskell
map toLower "Hello" -- поверне "hello"
```

## Глибока занурення

Так як у Haskell рядки є не змінними, то функція `toLower` повертає новий рядок зі зміненим регістром. Це важливо пам'ятати, при використанні функції в програмі.

Також варто зазначити, що при конвертації до нижнього регістру, деякі символи можуть залишитися без змін, на приклади, українська літера `Ї`.
