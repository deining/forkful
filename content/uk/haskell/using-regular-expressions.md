## Чому

У програмуванні часто зустрічаються ситуації, коли потрібно виконувати пошук та заміну певних текстових шаблонів у рядку або файлі. Для цього і використовуються такі інструменти, як регулярні вирази. Вони дозволяють швидко та ефективно виконувати розпізнавання та перетворення даних, що робить їх незамінними в програмуванні.

## Як користуватися регулярними виразами в Haskell

Для роботи з регулярними виразами в Haskell потрібно використовувати спеціальну бібліотеку `Text.Regex.TDFA`. Щоб почати користуватися нею, треба спочатку імпортувати потрібні модулі:

```Haskell
import Text.Regex.TDFA
import Text.Regex.TDFA.Text
import qualified Data.Text as T
```

Тепер можна створити регулярний вираз за допомогою функції `makeRegex`, передавши йому шаблон у вигляді рядка:

```Haskell
let regex = makeRegex "hello"
```

Далі, щоб знайти всі входження цього виразу у текстовому рядку, використовуємо функцію `matchAllText` та передаємо їй регулярний вираз та текст, в якому потрібно шукати:

```Haskell
let text = "Hello world, hello everyone!"
let matches = matchAllText regex text
```

В результаті отримаємо список з входженнями, кожне з яких є окремим об'єктом із полем `mend` - індексом знайденого тексту та `mgroup` - знайденим шаблоном:

```Haskell
-- [ (1, "hello"), (14, "hello") ]
```

Щоб замінити текст у певному рядку за допомогою регулярного виразу, використовуємо функцію `subRegex`, передаючи їй регулярний вираз, текст для заміни та сам рядок:

```Haskell
let newString = subRegex regex "bye" text
```

Отримаємо рядок `"bye world, bye everyone!"`.

## Глибокий дайв у регулярні вирази

Регулярні вирази дозволяють виконувати багато різних операцій, наприклад вибірка конкретних символів або груп символів, обмеження довжини тексту, перевірка комбінацій символів і багато іншого. Для більш детального ознайомлення з можливостями регулярних виразів в Haskell рекомендуємо звернутися до офіційної документації бібліотеки `Text.Regex.TDFA`.
