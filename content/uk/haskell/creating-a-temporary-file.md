## Чому

Створення тимчасового файлу є корисним для поділу великих задач на менші кроки або для збереження тимчасових даних між запусками програми.

## Як це зробити

```Haskell 
import System.IO

-- Створення тимчасового файлу з іменем temp.txt
tempFile <- openTempFile "" "temp.txt"

-- Запис тексту в тимчасовий файл
let text = "Це є тимчасовий файл!"
hPutStr tempFile text

-- Читання з тимчасового файлу
hGetContents tempFile >>= putStrLn

-- Закриття тимчасового файлу
hClose tempFile
```

Виведення: 
``` 
Це є тимчасовий файл!
```

## Глибокий занурення

Створення тимчасових файлів може бути корисним при роботі з більшими об'ємами даних, таких як зображення або великі текстові файли. Крім того, використання тимчасового файлу може бути допоміжним при відладці програми.
