## Навіщо

Для чого починати новий проект на Haskell? Це потенційно може бути корисним для вирішення складних завдань, які є проблематичними у традиційних мов програмування. Крім того, Haskell має сильну систему типів та підтримує функціональне програмування, що може полегшити розробку та підтримку проекту.

## Як почати

Зараз поговоримо про те, як почати новий проект на Haskell. Перш за все, вам слід встановити Haskell на свій комп'ютер. Є кілька способів це зробити, але найпростіший - використовуючи інсталятор Stack. Далі, створіть папку для вашого проекту та відкрийте її у терміналі.

```Haskell
stack new myproject
cd myproject
```

Наразі, ваша папка має основну структуру проекту, яку створив Stack. Далі, ви можете створити файл з розширенням `.hs` і почати писати код.

```Haskell
-- Функція, яка додає два числа разом
add :: Int -> Int -> Int
add x y = x + y

main = do
    print (add 2 3) -- Результат: 5
```

Як бачите, Haskell має чіткий та зрозумілий синтаксис. Ви можете використовувати функціональне програмування та композицію функцій для покращення вашого коду. Після написання коду, ви можете запустити його за допомогою Stack.

```Haskell
stack runghc mycode.hs
```

Це запустить ваш код та виведе результат у консоль.
