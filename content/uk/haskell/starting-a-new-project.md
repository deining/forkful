## Що & Чому?

Старт нового проекту - це коли програміст починає розробку нової програми або додає нові функції до вже існуючого проекту. Програмісти роблять це, щоб зробити свої програми більш ефективними та функціональними.

## Як:

```Haskell 
main = do
  putStrLn "Вітаємо! Ви починаєте новий проект."
  putStrLn "Що ви хочете зробити?"

  input <- getLine
  putStrLn ("Ви обрали " ++ input ++ " як ваш наступний крок.")

```

Вихідний код буде виводити повідомлення з привітанням та запитувати користувача, що він хоче зробити. Потім він використовує введену інформацію, щоб виконати наступний крок.

## Deep Dive:

Розпочати новий проект - це важлива частина будь-якої програмістської роботи. Він допомагає зберегти код чистим та організованим. Існує також кілька альтернативних способів розпочати новий проект, таких як використання шаблонів або створення власних рамок роботи. У Haskell є кілька інструментів для автоматизації процесу створення нового проекту, які можуть бути корисні для розробників.
