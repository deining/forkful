## Чому
Кожен програміст знає, що робота з текстовими файлами є невідємною частиною розробки будь-якого програмного продукту. Крім того, це є швидким і простим способом зберігання і об`єднання даних. Однак, написання текстового файлу може бути складною задачею для початківців. У цій статті ми поговоримо про те, як написати текстовий файл у Haskell і навчимося це робити ефективно.

## Як написати текстовий файл у Haskell
Написання текстового файлу у Haskell вимагає використання спеціальних функцій і рядків з кодами символів. Для початку, нам потрібно використати модуль `Data.Text`. Далі, створюємо змінну з текстом, який хочемо записати у файл. Наприклад, `myText = "Привіт, світе!"`.
Для створення текстового файлу, використаємо функцію `writeFile`. Перший аргумент цієї функції буде назвою файлу, а другий - змінною з текстом. Результатом роботи цієї функції буде створений файл із вказаним текстом. Виконавши цей код, ми отримаємо файл з назвою `myFile.txt` і текстом "Привіт, світе!".
```
Haskell
import Data.Text
main = do
  let myText = "Привіт, світе!"
  writeFile "myFile.txt" myText
```

## Глибокий занурення
Написання текстового файлу у Haskell може бути більш складною задачею, якщо потрібно записувати більш складний текст або дізнаватися про різні типи кодувань. Щоб уникнути проблем з кодуванням, ми можемо використати модуль `Data.Text.Encoding`. З цим модулем ми можемо визначити тип кодування нашого файлу і використати відповідну функцію для запису файлу. Наприклад, якщо ми хочемо використовувати кодування UTF-8, можемо скористатися функцією `writeFileUtf8` замість `writeFile`.

Також варто згадати, що у Haskell є можливість читання текстових файлів за допомогою функції `readFile`. Ця функція приймає шлях до файлу і повертає вміст у вигляді рядка. Це може бути корисно, якщо ми хочемо зчитати вміст із файлу для подальшої обробки.
