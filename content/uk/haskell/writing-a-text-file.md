## Чому

Написання текстового файлу - це важлива частина програмування в Haskell. Використання текстових файлів може забезпечити зручний і організований спосіб зберігання інформації, а також дозволяє легко обмінюватися даними між різними програмами.

## Як зробити

Найпростіший спосіб створити текстовий файл в Haskell - використати функцію `writeFile`. Наприклад, ми можемо створити файл з ім'ям "hello.txt" та записати в нього список рядків:

```Haskell
writeFile "hello.txt" ["Привіт", "Світ!"]
```

Вивід: файл "hello.txt" з таким вмістом:

```
Привіт
Світ!
```

Також можна використовувати функцію `appendFile`, щоб додавати дані до вже існуючого файлу.

## Глибоке занурення

Написання текстового файлу в Haskell може бути більш складним, якщо ми бажаємо працювати з більш складними типами даних, наприклад, списками чи кортежами. Також важливо пам'ятати про кодування, яке використовується для збереження тексту у файлі.
