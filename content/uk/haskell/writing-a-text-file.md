## Що і чому?

При написанні програм, розробники часто потребують зберігати дані для подальшого використання. Це можна зробити за допомогою створення текстового файлу, який зберігатиме необхідну інформацію. Такі файли можуть бути використані для зберігання конфігурацій, вхідних даних або результатів обчислень.

## Як це зробити?

У Haskell існує декілька способів створення текстового файла. Один з них - використання функції ```writeFile```. Ця функція приймає два аргументи: шлях до файлу та рядок, який потрібно записати. Наприклад, якщо ми хочемо створити файл з назвою "data.txt" і записати в нього рядок "Hello, world!", то ми можемо написати такий код:

```Haskell
writeFile "data.txt" "Hello, world!"
```

Інший спосіб створення файлу - за допомогою функції ```withFile```, яка дозволяє вказати режим відкриття файлу і обробляти його з допомогою функції. Наприклад, ми можемо створити файл "data.txt" і записати в нього рядок "Hello, world!" за допомогою наступного коду:

```Haskell
withFile "data.txt" WriteMode $ \handle -> hPutStrLn handle "Hello, world!"
```

Обидва ці способи дозволяють зробити файл доступним для подальшого читання та редагування.

## Глибоке занурення

Створення та редагування текстових файлів - це важлива частина роботи багатьох програмістів. У Haskell є багато інших функцій для роботи з текстовими файлами, наприклад, ```appendFile```, яка дозволяє додавати дані до існуючого файлу. Також існує можливість робити більш складні операції, наприклад, зчитування та обробка структур даних з файлу.

Існують інші мови програмування, які також мають свої засоби для роботи з текстовими файлами, наприклад, Python має модуль ```io```, що надає функції для запису та зчитування із файлів.

Для більш детального вивчення цієї теми, ви можете ознайомитися з офіційною документацією Haskell по роботі з файлами.
