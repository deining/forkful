## Чому

Перевірка та порівняння дат є важливою частиною програмування в багатьох проектах, де використання часових моментів є необхідністю. Наприклад, веб-додатки повинні відображати часові мітки, бази даних можуть використовувати дати для сортування, а в логічних алгоритмах може бути потреба у порівнянні двох дат. У цій статті ми розглянемо, як можна порівняти дві дати в мові програмування Haskell.

## Як
```Haskell
import Data.Time

-- Створення простих дат
let date1 = fromGregorian 2021 4 15
let date2 = fromGregorian 2021 4 20

-- Порівняння дат
date1 == date2 -- повертає False
date1 /= date2 -- повертає True
date1 < date2  -- повертає True
date2 > date1  -- повертає True

-- Порівняння більш точних часів
let time1 = UTCTime date1 (secondsToDiffTime 3600) -- додає 1 годину до дати
let time2 = UTCTime date2 (secondsToDiffTime 3600)

time1 == time2 -- повертає True
time1 < time2  -- повертає False
time2 > time1  -- повертає False

-- Порівняння за допомогою функції compare
compare date1 date2 -- повертає LT – означає що date1 менше date2

-- Порівняння за допомогою функції diffUTCTime
diffUTCTime time1 time2 -- повертає 0 – означає що часи рівні
```

В цьому прикладі ми використали модуль Data.Time, який містить функції для роботи з датами та часом. Ми створили дві прості дати за допомогою функції fromGregorian та дві більш точні дати за допомогою типу даних UTCTime. Порівнювати дати можна за допомогою операторів порівняння, але більш зручно використовувати функції compare та diffUTCTime.

## Глибинний аналіз

У мові програмування Haskell є багато інших функцій та методів для порівняння дат. Якщо вам потрібно порівняти дві дати з використанням іншого календарного формату, ви можете використовувати функцію Data.Time.Calendar.diffDays, яка повертає різницю в днях між двома датами. Також, Data.Time.Calendar.Days включає функції для порівняння дат за допомогою інших календарних систем, таких як Юліанський календар чи Григоріанський календар.
