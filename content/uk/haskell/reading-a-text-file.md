## Чому

Якщо ви хочете прочитати великий обсяг текстових даних за короткий проміжок часу, наприклад, виконати аналіз тексту або застосувати шаблони пошуку, то цей огляд про читання текстового файлу в Haskell буде корисним для вас.

## Як

Найпростішим способом для читання текстового файлу в Haskell є використання функції `readFile`, яка автоматично зчитує весь вміст файлу у вигляді строкового значення.

```Haskell
main = do
    file <- readFile "example.txt"
    putStrLn file
```

У цьому прикладі, ми спочатку присвоюємо вміст файлу змінній `file` за допомогою оператора `<-`, а потім виводимо цей зміст з допомогою функції `putStrLn`.

Також можна використовувати функцію `lines`, яка розділить вміст файла на список рядків.

```Haskell
lines file
```

Це особливо корисно, якщо ви бажаєте обробляти кожен рядок окремо.

## Вдумайся

У Haskell, всі дані мають типи, в тому числі і текстові дані. Це означає, що ви можете здійснювати різні операції на рядках, такі як розділення, злиття та зміна регістру.

Також важливо пам'ятати, що функція `readFile` повертає значення в "управляючому" контексті `IO`, тому її потрібно викликати у функції `main`, щоб отримати результат.
