Що і чому?
Читання текстового файлу - це процес отримання даних з файлу, який містить текстову інформацію. Програмісти часто використовують цей процес для отримання даних з файлів, які потрібні для їх програм.

Як це зробити:
```Haskell
main = do
  file <- readFile "myFile.txt"
  putStrLn file

-- Зміст файлу 'myFile.txt' виведеться на екран
```

Глибше дослідження:
Читання текстових файлів є важливою частиною процесу програмування, оскільки це дозволяє програмі отримати необхідну інформацію для роботи. Варто звернути увагу на альтернативи для читання файлів, такі як використання бібліотеки `System.IO` або зчитування файлу в режимі "лінії за лінією". Для реалізації читання текстового файлу ви можете використовувати функцію `readFile` з пакету `System.IO`.

Дивіться також:
Додаткову інформацію про читання файлів в Haskell ви можете знайти на офіційному сайті мови. Також можна розгорнути свої знання про роботу з текстовими файлами, вивчивши різноманітні приклади на платформі Github.
