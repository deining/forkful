## Чому 

Наш світ все більше залежить від веб-ресурсів, тому Ви, як програміст, можете бути зацікавлені у завантаженні веб-сторінок для подальшої обробки та аналізу. На щастя, ми маємо потужні інструменти, такі як Haskell, які дозволяють робити це легко та ефективно.

## Як це зробити

Завантажити веб-сторінку можна за допомогою модуля Network.HTTP в Haskell. Нижче наведено приклад функції, яка завантажує HTML-сторінку з вказаної URL-адреси та повертає його як рядок:

```Haskell
import Network.HTTP

getPage :: String -> IO String
getPage url = simpleHTTP (getRequest url) >>= getResponseBody
```

Наступним кроком є використання отриманого рядка HTML для подальшої обробки. Наприклад, можна розбити його на окремі елементи за допомогою функції `words` і виконати будь-які інші дії з цими елементами. 

```Haskell
processPage :: String -> [String]
processPage html = words html -- приклад обробки за допомогою функції words
```

## Глибоке погруження

Завантаження веб-сторінки - це процес, що потребує багато різноманітних дій. Необхідно встановити з'єднання із веб-сервером, надіслати запит, отримати відповідь та обробити її. У модулі Network.HTTP ці процеси автоматизовані, що максимально спрощує роботу з веб-сторінками.

Крім того, ви можете зрозуміти більше про будову веб-сторінок та принципи роботи веб-серверів, якщо будете вивчати пакет Network.HTTP у деталях.
