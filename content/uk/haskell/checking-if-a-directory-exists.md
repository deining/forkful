## Що та чому?

Перевірка існування директорії - це процес, який дозволяє програмі перевірити, чи існує вказана користувачем директорія на комп'ютері. Це важливий етап в програмуванні, оскільки дозволяє уникнути помилок та некоректної роботи програми.

## Як?

```Haskell
import System.Directory

main = do
    dirExists <- doesDirectoryExist "файлова система/шлях до директорії"
    if dirExists
        then putStrLn "Директорія існує!"
        else putStrLn "Директорія не існує."
```

Вивід: 
```Haskell
Директорія існує! 
```

## Занурення в глибину

Історичний контекст: Перевірка існування директорії - це сучасна функція, яка була додана до мови Haskell у версії 1.3. Це стало можливим завдяки збільшенню популярності мови та розширенню її функціональності.

Альтернативи: Існує декілька інших методів перевірки існування директорії, наприклад, використання командного рядка або вбудованих засобів операційної системи.

Деталі реалізації: Функція doesDirectoryExist викликає системний виклик, щоб перевірити наявність директорії на операційній системі. Результатом є булева змінна, яка показує, чи існує вказана директорія.
