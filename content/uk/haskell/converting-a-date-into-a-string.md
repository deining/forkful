## Чому 

Конвертування дати в рядок є важливою задачею для багатьох програмістів, оскільки це дозволяє легко відображати дату в зручному форматі для користувача. В цій статті ми розглянемо, як конвертувати дату в рядок за допомогою мови програмування Haskell.

## Як

Для конвертації дати в рядок нам знадобиться функція `show`. Давайте подивимося на приклад коду, який використовує функцію `show` для конвертації дати в рядок:

```Haskell
import Data.Time.Format

date :: UTCTime
date = read "2021-05-10 12:00:00 UTC"

stringDate :: String
stringDate = show date

main :: IO ()
main = print stringDate
```

Як бачимо, ми імпортуємо модуль `Data.Time.Format`, який містить функцію `show`. Також ми визначаємо змінну `date`, яка має тип `UTCTime`, і використовуємо функцію `read` для парсингу рядка з датою. Далі ми використовуємо функцію `show` для конвертації дати в рядок і використовуємо функцію `print` для виведення результату.

Виконавши цей код, ми отримаємо наступний результат:

```
"2021-05-10 12:00:00 UTC"
```

Тепер ми можемо використовувати цей рядок для відображення дати у зручному форматі для користувача.

## Глибоке погруження

Якщо ви хочете більш докладно дізнатися про конвертування дати в рядок в Haskell, вам слід ознайомитися з модулем `Data.Time.Format`, який надає багато корисних функцій для роботи з датами та часом. Ви також можете дослідити різні типи даних, які використовуються для роботи з датами в Haskell, такі як `UTCTime`, `LocalTime`, `Day` тощо.
