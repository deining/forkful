## Що & Чому?

Надсилання HTTP-запиту - це процес взаємодії з веб-сервером, який дозволяє отримувати ресурси з Інтернету. Програмісти використовують це для отримання необхідних даних з веб-сайтів, що є важливим елементом для розвитку додатків та збір інформації.

## Як:

```Haskell
import Network.HTTP

main = do
  response <- simpleHTTP (getRequest "https://example.com")
  getResponseBody response
```
Виконання цього коду допоможе вам отримати вміст веб-сторінки за допомогою HTTP запиту та зберегти його у змінній. Просто замініть URL на той, який ви бажаєте отримати, і виконайте код у своєму улюбленому засобі для виконання коду Haskell.

## Поглиблене дослідження:
Відправлення HTTP-запитів - це нещодавній розвиток в Інтернеті, який був створений для забезпечення зручності та ефективності обміну даними між веб-сайтами та додатками. Існують інші альтернативи для виконання цієї задачі, такі як використання спеціальних бібліотек для мов програмування, які зосереджені на веб-комунікаціях. Однак, використання вбудованих засобів Haskell дозволяє зробити процес надсилання запитів більш простим та зручним.

При відправленні HTTP-запитів за допомогою бібліотеки Network.HTTP, ви можете використовувати цілу низку функцій та методів, які дозволяють вам налаштовувати запити та обробляти отримані відповіді. Також, ви можете використовувати засоби управління іншими типами веб-запитів, такими як POST, PATCH та DELETE.
