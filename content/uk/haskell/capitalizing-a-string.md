## Чому

Є багато ситуацій, коли корисно мати рядок, який починається з великої літери. Наприклад, якщо ви створюєте унікальний ідентифікатор для об'єкта, ви можете застосувати техніку великої літери, щоб зробити його більш читабельним.

## Як

Існує багато різних способів здійснити це в Хаскелі. Один із них - використання вбудованої функції `toUpper`, яка повертає рядок із всіма буквами у верхньому регістрі. Для того, щоб ця функція працювала, потрібно підключити модуль `Data.Char`.

```Haskell
import Data.Char (toUpper)

capitalize :: String -> String
capitalize str = toUpper (head str) : tail str
```

Код вище приймає рядок і повертає його копію, де перша літера перетворена на велику. Ми користуємося функцією `head` для отримання першої літери та `tail` для отримання решти рядку, які ми потім з'єднуємо разом.

Також ми можемо використовувати рекурсивну функцію, щоб перетворити всі літери у верхній регістр. Наприклад:

```Haskell
import Data.Char (toUpper)

capitalize :: String -> String
capitalize [] = []
capitalize (x:xs) = toUpper x : capitalize xs
```

У цьому прикладі ми використовуємо паттерн-матчинг для першого та решти рядку. Якщо рядок порожній, ми повертаємо порожній рядок. У іншому випадку, ми перетворюємо першу літеру на велику та рекурсивно викликаємо функцію для решти рядку.

## Глибше

Якщо ви детальніше ознайомитеся з рекурсією та паттерн-матчингом в Хаскелі, ви можете побачити, що функція `capitalize` з останнього прикладу може бути переписана як:

```Haskell
import Data.Char (toUpper)

capitalize :: String -> String
capitalize = map toUpper
```

Тут ми використовуємо функцію `map`, яка застосовує функцію `toUpper` до кожного символу рядку. Це ще один приклад функціонального підходу до програмування, який характерний для Хаскелу.
