## Зачем

Один з найпоширеніших завдань у програмуванні - це обробка тексту. Іноді ми потребуємо змінити регістр тексту, а інколи зробити першу букву слова великою. У цій статті ми розглянемо, як зробити перетворення першої букви великою у мові програмування Haskell.

## Як

Для початку, нам потрібно імпортувати модуль `Data.Char`, щоб отримати доступ до функції `toUpper`. Потім, нам потрібно написати функцію, яка буде приймати строку, і вертати цю ж строку, але з першою буквою великою.

```Haskell
import Data.Char

capitalize :: String -> String
capitalize (x:xs) = toUpper x : xs
```

Ми використовуємо зразок зіставлення у визначенні функції `capitalize`, де `x` - перший символ у строці, а `xs` - решта стрічки. Після цього, ми використовуємо функцію `toUpper` для перетворення `x` у велику букву. Наступні символи ми додаємо до цієї першої букви.

Давайте скомпілюємо цей код і спробуємо ввести рядки для обробки:

```Haskell
main = do
  putStrLn "Введіть рядок:"
  input <- getLine
  putStrLn $ capitalize input
```

Введіть, наприклад, "привіт світе", і ви побачите виведений результат: "Привіт світе".

## Deep Dive

Якщо ви поглянете на типи функцій `toUpper` і `capitalize` у нашому коді, ви звернете увагу на те, що `toUpper` має тип `Char -> Char`, а `capitalize` - `String -> String`. Це відбувається тому, що ми використовуємо імплементацію `toUpper` для першої букви у нашій функції `capitalize`, так як тип `toUpper` дозволяє нам працювати з одним символом.

Також, варто зазначити, що функції у Haskell є чистими, тобто вони не змінюють стан програми і не мають побічних ефектів. У нашому випадку, ми не змінюємо оригінальну стрічку, а просто повертаємо нову версію з великою першою буквою.
