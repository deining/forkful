## Чому

Для чого нам потрібно з'єднувати рядки у Haskell? Деякі завдання вимагають об'єднання різних рядків в один, щоб отримати більш складну або зручнішу для обробки дані. Наприклад, створення повідомлень, шаблонів електронної пошти або сформування запитів до бази даних.

## Як

Існує декілька способів об'єднання рядків у Haskell, але найбільш зручним є використання функції "++". Давайте подивимося на приклад коду з використанням цієї функції:

```Haskell
str1 = "Привіт"
str2 = "світ!"
str3 = str1 ++ " " ++ str2
print str3
```

Вийде результат: "Привіт світ!". Тут ми використали оператор "++" для об'єднання рядків у змінній str3. Можна також використовувати цей оператор для об'єднання більшої кількості рядків, вони будуть додаватися один за одним в тому ж порядку, як у нашому прикладі.

Ще одним корисним способом об'єднання рядків є використання функції "concat", яка приймає список рядків та повертає один з'єднаний рядок:

```Haskell
list = ["це", "об'єднані", "рядки"]
str = concat list
print str
```

Вийде результат: "цеоб'єднанірядки". Такий спосіб корисний, коли у нас вже є список рядків, які потрібно об'єднати.

## Глибинне дослідження

Варто врахувати, що оператор "++" та функція "concat" не є ефективними з точки зору продуктивності, оскільки кожна нова рядок повторно створюється при кожному виклику. Для більш ефективного з'єднання великих рядків у Haskell існує тип даних "ByteString". Він займає менше місця в пам'яті та працює швидше, але потребує додаткових зусиль для роботи з ним.
