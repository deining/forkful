## Чому

З'єднання рядків є важливою технікою в багатьох програмах, особливо в програмуванні. Використання цієї техніки допомагає створити більш складні рядки та шаблони, що полегшує роботу з ними. Також вона може бути корисною при виведенні повідомлень для користувачів або при форматуванні вихідного тексту.

## Як це зробити

Є декілька різних способів з'єднувати рядки в Haskell. Один з них - використання оператора `++`, який дозволяє додавати рядки один до одного. Наприклад, `str1 ++ str2` з'єднає рядки `str1` та `str2` разом. Для цього потрібно перш ніж виконати це, використати `import Data.List` щоб підключити бібліотеку для рядків. Давайте подивимося на приклад:

```Haskell
import Data.List

str1 = "Вітаю,"
str2 = "Друг!"
str3 = str1 ++ str2

-- Вивід: "Вітаю, Друг!"
```

Можна також використати функцію `concat` для з'єднання списку рядків в один рядок. Наприклад:

```Haskell
arr = ["Вітаю,", " ", "Друг!"]
str = concat arr

-- Вивід: "Вітаю, Друг!"
```

## Глибокий занурення

Для тих, хто хоче дізнатися більше про з'єднання рядків, можна розглянути функції `intercalate` та `unwords`. Функція `intercalate` дозволяє вставляти рядок між елементами списку. Наприклад:

```Haskell
arr = ["Вітаю", "Друг"]
str = intercalate ", " arr

-- Вивід: "Вітаю, Друг"
```

Функція `unwords` з'єднує список рядків та додає пропуски між ними. Наприклад:

```Haskell
arr = ["Вітаю", "Друг"]
str = unwords arr

-- Вивід: "Вітаю Друг"
```
