Що та чому?
Друк вихідних даних - це процес виведення даних для відлагодження програми. Програмісти користуються цим для виявлення проблем та помилок у своєму коді.

Як це зробити:
Нижче наведено приклади коду та вихідних даних для продемонстрування друку вихідних даних у Haskell.

```Haskell
-- Приклад 1: Виведення значення змінної у консоль
num :: Int
num = 10
putStrLn $ "Значення num: " ++ show num

-- Вихід: Значення num: 10

-- Приклад 2: Виведення списка елементів у консоль
list :: [Int]
list = [1, 2, 3, 4, 5]
putStrLn $ "Список елементів: " ++ show list

-- Вихід: Список елементів: [1, 2, 3, 4, 5]

-- Приклад 3: Виведення результату функції у консоль
add :: Int -> Int -> Int
add x y = x + y
putStrLn $ "Результат додавання: " ++ show (add 5 7)

-- Вихід: Результат додавання: 12
```

Глибша знадоба:
Друк вихідних даних є незамінною частиною відлагодження програм. Раніше програмісти користувались відлагоджувачами, але з появою можливості друкувати вихідні дані у консоль, цей процес став більш зручним та швидшим. Існують альтернативні способи друку вихідних даних, такі як запис до файлу або використання спеціальних бібліотек, але друк до консолі є найпоширенішим та простим способом.
