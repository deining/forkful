## Чому

Ви можете замислюватися, чому знадобиться вам витягувати підрядки в TypeScript програмуванні? Це може бути корисно для безлічі завдань, наприклад, ви можете потрібні певні частини рядка для обробки або порівняння з іншими даними. В цій статті ми поглиблено розглянемо, як витягати підрядки в TypeScript та надаємо приклади коду з вихідними даними, щоб ви краще зрозуміли цей процес.

## Як

Для початку, нам потрібно мати рядок, з якого ми будемо витягувати підрядки. Наприклад, ми можемо мати рядок "Вітаємо вас!" і хочемо витягнути підрядок "вас".

```TypeScript
let str = "Вітаємо вас!";
```

Тепер, щоб витягнути підрядок, ми можемо використовувати функцію `substring()`, яка приймає два аргументи - початковий та кінцевий індекси. Давайте подивимось на приклад коду:

```TypeScript
let str = "Вітаємо вас!";
let subStr = str.substring(9, 12);
console.log(subStr); // вас
```

В цьому прикладі, ми використали метод `substring()` для витягування підрядка, що починається з 9-го символу і закінчується 12-м символом (забираючи остаточний пункт, тобто "в"). Вихідні дані показують, що ми успішно витягнули підрядок "вас".

## Поглиблене вивчення

Якщо вам потрібно витягнути підрядок з будь-якими символами, то можно використати метод `indexOf()` для знаходження початкового індексу підрядка у вихідному рядку. Приклад коду:

```TypeScript
let str = "Вітаємо вас!";
let subStr = str.substring(str.indexOf("тає")+3, str.indexOf("вас")+3);
console.log(subStr); // таємо вас
```

Тут ми використали метод `indexOf()` для знаходження початкового і кінцевого індексів підрядка "тає" та "вас" у вихідному рядку. Обидва індекси були зміщені на 3, щоб до підрядка включались всі потрібні символи. В результаті ми отримали підрядок "таємо вас".
