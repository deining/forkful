## Чому

У програмуванні часто виникає необхідність використання лише частини рядка або тексту. Це може бути корисно для пошуку певних даних або перетворення тексту. Дані дії є частиною більшого процесу, який може покращити ефективність програми та спростити завдання програміста.

## Як

В TypeScript є кілька способів виокремлення підрядків з рядка, включаючи метод `substring`, `slice` та `substr`. Давайте розглянемо приклад використання кожного з цих методів:

```TypeScript
const str: string = "Привіт, це приклад рядка.";

// Використання `substring`
console.log(str.substring(7, 12)); // виводить "це при"

// Використання `slice`
console.log(str.slice(12)); // виводить "клад рядка."

// Використання `substr`
console.log(str.substr(0, 5)); // виводить "Приві"
```

У цьому прикладі ми маємо рядок `str`, і за допомогою різних методів ми можемо виокремити різні підрядки. Для методів `substring` та `slice` першим параметром є початковий індекс, а другим - кінцевий індекс. В методі `substr` перший параметр - це початковий індекс, а другим - кількість символів, які потрібно виокремити.

## Глибоке дослідження

У TypeScript є ще один метод - `split`, який може бути використаний для розбиття рядка на масив підрядків за допомогою розділюючого символу. Наприклад, якщо ми розб'ємо наш рядок за допомогою коми, ми отримаємо наступний результат:

```TypeScript
const str: string = "цифри: 1,2,3,4,5";
console.log(str.split(",")); // виводить ["цифри: 1", "2", "3", "4", "5"]
```

Цей метод може бути корисним для обробки великих текстових файлів або даних з форматованим контентом.
