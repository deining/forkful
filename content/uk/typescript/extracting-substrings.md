## Що і чому? 
Витягування підрядків є процесом отримання підрядку з вихідного рядка тексту за допомогою програмування. Це часто використовується програмістами для отримання потрібних даних з великих текстових рядків.

## Як це зробити: 
У TypeScript цей процес дуже простий за допомогою використання методу ```substring ()```. Дана функція приймає два аргументи - початковий та кінцевий індекси, та повертає підрядок рядку між ними. Переглянемо приклад коду:

```TypeScript
let str = "Привіт світ!";
let substring = str.substring(0, 6);
console.log(substring); // виведе "Привіт"
``` 

## Глибокий аналіз: 
Ідея витягування підрядків з'явилася ще у 60-х роках з появою мови програмування BASIC. До цього процесу також можна використовувати функцію ```slice ()``` або метод ```substr ()```. У TypeScript використання методу ```substring ()``` є більш простим і зрозумілим для програмістів. Переглянемо приклад коду з використанням методу ```substr ()```:

```TypeScript
let str = "Привіт світ!";
let substring = str.substr(0, 6);
console.log(substring); // виведе "Привіт"
```

Проте, варто знати, що метод ```substr ()``` може приймати третій аргумент - кількість символів для вилучення після заданого індексу, що робить його корисним для підрядків зі складними довжинами.
