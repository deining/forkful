## Чому

Використання регулярних виразів є важливою частиною програмування в TypeScript. Вони дозволяють швидко та ефективно шукати та замінювати текстові шаблони у строках коду. Незалежно від того, чи ви новачок чи досвідчений розробник, регулярні вирази корисні для вирішення багатьох завдань.

## Як

Для використання регулярних виразів у TypeScript необхідно імпортувати клас RegExp з стандартної бібліотеки. Після цього, ви можете створити новий екземпляр регулярного виразу, передаючи йому шаблон, за яким потрібно шукати, та флаги, що визначають використання регістру та інші параметри пошуку.

```TypeScript
import { RegExp } from 'typescript';

// Створюємо регулярний вираз для пошуку усіх цифр в рядку
const regex = new RegExp(/[0-9]/, 'g');
const str = 'Це рядок з кількома 123 цифрами';

// Використовуємо метод match для пошуку відповідного тексту та виводу його в консоль
console.log(str.match(regex)); // Результат: [1,2,3]
```

## Розділ в Глибину

Регулярні вирази дають вам багато можливостей для пошуку та заміни текстових шаблонів. Наприклад, ви можете використовувати спеціальні символи для визначення певних шаблонів, таких як [\w] для пошуку будь-якого слова чи \d для пошуку цифр. Також, ви можете використовувати функції replace та test для заміни та перевірки відповідності шаблону.

Ще однією корисною можливістю регулярних виразів є використання групування, яке дозволяє знаходити та виділяти певну частину тексту у знайденому шаблоні. Це особливо корисно у разі, коли потрібно виконувати заміну лише певної частини знайденого шаблону.
