## Чому

Регулярні вирази - це потужний інструмент в розробці програм, який дозволяє здійснювати пошук і заміну тексту за певними шаблонами. Використання регулярних виразів може допомогти вам ефективніше розв'язувати завдання, пов'язані з текстовими даними.

## Як це зробити

Для початку давайте розберемося з основами використання регулярних виразів у TypeScript. Ви можете використовувати будь-яку функцію, що приймає строку, як input, і використовувати в ній регулярний вираз, обертаючи його у круглі дужки з префіксом `new RegExp`. Наприклад:

```TypeScript
let str = "Привіт, світ!";
let pattern = new RegExp("привіт", "i");
console.log(pattern.test(str)); // виведе true
```
У цьому прикладі ми створюємо регулярний вираз для пошуку слова "привіт" у будь-якому регістрі в рядку `str`. Використовуючи метод `test`, ми перевіряємо, чи збігається цей регулярний вираз з `str`.

Слід зауважити, що використовуючи `RegExp`, ви можете використовувати кілька флагів, що змінюють поведінку регулярного виразу. Серед них: `i` - ігнорувати регістр, `g` - пошук всіх співпадінь, і `m` - вважати кожен рядок окремим рядком. Зверніться до документації TypeScript для отримання повної інформації про дозволені флаги.

## Глибокий погляд

Для більш складних і потужних виразів, можна використовувати спеціальні символи та модифікатори. Наприклад, символ `.` відповідає будь-якому символу, а символ `*` - будь-якій кількості попереднього символу. Наприклад:

```TypeScript
let str = "apple, banana, orange";
let pattern = new RegExp("a.+e");
console.log(pattern.test(str)); // виведе true
```

У цьому прикладі ми перевіряємо, чи можна знайти послідовність символів, що починається з `a` і закінчується на `e` у рядку `str`. І символи `.` і `*` дозволяють нам опустити деяку частину цієї послідовності.
