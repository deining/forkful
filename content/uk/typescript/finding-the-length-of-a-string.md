## Почему

Довго працюючи зі змінними типу рядок в TypeScript, ви, напевно, неодноразово потребуватимете знати їх довжину. Це може бути корисно для валідації введених даних або для знаходження середнього значення усіх рядків в масиві. Знаючи цей функціонал, ви зможете більш ефективно працювати зі змінними типу рядок і отримати більше користі від вашого коду.

## Як

Пошук довжини рядка - дуже простий процес у TypeScript. Для цього використовується метод `length`, який повертає кількість символів у рядку. Ось приклад коду:

```Typescript
const str = "Це приклад рядка.";
console.log(str.length);

// Результат: 18 (так як у рядку 18 символів)
```

Ви також можете використовувати метод `length` для знаходження довжини рядка, який зберігається у змінній типу рядок:

```Typescript
const name = "Іван";
const surname = "Іванов";
console.log("Повне ім'я:", name + surname, "Довжина:", (name + surname).length);

// Результат: Повне ім'я: Іван Іванов Довжина: 12
```

Окрім того, ви можете використовувати метод `length` для знаходження довжини рядка, який є елементом масиву. Ось приклад коду:

```Typescript
const fruits = ["Яблуко", "Банан", "Виноград"];
console.log("Кількість фруктів:", fruits.length);

// Результат: Кількість фруктів: 3
```

Як бачите, метод `length` може бути використаний на різних типах даних у TypeScript.

## Deep Dive

За замовчуванням, метод `length` повертає кількість символів у рядку без урахування пробілів. Якщо ви хочете включити пробіли, ви можете використовувати метод `trim`, який видаляє пробіли з початку і кінця рядка, а потім застосувати метод `length` до видаленого рядка.

```Typescript
const text = "   Це текст з пробілами.   ";
console.log("Кількість символів:", text.length); // Результат: 26
console.log("Кількість символів з пробілами:", text.trim().length); // Результат: 18
```
