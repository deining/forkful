##Чому

Визначення довжини рядка є важливим елементом в програмуванні, оскільки це дає можливість перевірити, чи не перевищує рядок встановлену межу символів або які дані надійшли від користувача.

##Як

Для визначення довжини рядка в TypeScript, використовується вбудована функція `length`, яка повертає кількість символів у рядку. Нижче наведені декілька прикладів використання цієї функції та виведення результату:

```TypeScript
let str: string = "Привіт із TypeScript!";
console.log(str.length); // 20
```

```TypeScript
let emptyStr: string = "";
console.log(emptyStr.length); // 0
```

```TypeScript
let num: number = 123;
console.log(num.length); // Compile error, бо type number не має властивості length
```

В першому прикладі ми створюємо змінну `str` типу `string` зі значенням "Привіт із TypeScript!" та використовуємо функцію `length` для отримання довжини рядка. У другому прикладі ми використовуємо пустий рядок (`emptyStr`) та отримуємо 0 як результат, оскільки функція `length` повертає кількість символів у рядку, а пустий рядок не має жодного символу. У третьому прикладі, який необхідно краще пропустити біля, ми намагаємося використати функцію `length` для змінної `num`, але отримуємо помилку компіляції, оскільки тип `number` немає властивості `length`.

##Глибше ознайомлення

Функція `length` використовується для отримання довжини рядка, але важливо пам'ятати, що вона повертає кількість символів, а не кількість байтів. Тому, якщо ваш рядок містить символи, які займають більше одного байта (наприклад, кириличні символи), то результат може бути відрізнятися від кількості символів, яку ви бачите в редакторі коду.
