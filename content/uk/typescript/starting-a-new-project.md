## Чому

Запуск нового проекту є чудовим способом почати навчатися та вдосконалювати вміння у написанні програм на TypeScript. Це також дає можливість створювати корисні та інтерактивні додатки для різних платформ та пристроїв.

## Як

```TypeScript
// Приклад коду на TypeScript
function greeting(name: string) {
  console.log(`Привіт, ${name}!`); // виведе "Привіт, Василь!"
}

greeting("Василь");
```

Ви можете почати зі створення файлу з розширенням `.ts` та написанням простої функції. Потім скомпілюйте цей файл за допомогою компілятора TypeScript, щоб отримати файл з розширенням `.js`, який можна виконати на будь-якому пристрої.

## Глибоке занурення

Щоб почати новий проект на TypeScript, необхідно встановити компілятор TypeScript та бібліотеку `@types/node`. Після цього ви можете створювати складніші структури даних, використовувати класи та інші особливості мови для створення реалістичних додатків.
