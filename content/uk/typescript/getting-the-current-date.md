## Для чого

Майже кожен проект, що працює з даними, залежить від поточної дати. Незалежно від того, чи ви розробляєте веб-додаток, мобільний додаток або створюєте просту консольну програму, вам буде необхідно отримати поточну дату для подальшої обробки. Відоме що TypeScript пропонує багато корисних інструментів для маніпуляції з датами, тому ви подивимося, як його використовувати, щоб отримати поточну дату.

## Як це зробити

Для початку, нам необхідно імпортувати вбудований об'єкт `Date` з TypeScript. Давайте подивимося на приклад коду:

```TypeScript

// імпортуємо об'єкт `Date`.
import { Date } from "typescript";

// отримаємо поточну дату.
const currentDate: Date = new Date();

// використовуємо вбудовані методи для отримання необхідної інформації про дату.
console.log("Поточна дата:", currentDate.getDate()); // поверне день місяця, наприклад 22.
console.log("Поточний місяць:", currentDate.getMonth()); // поверне номер місяця, 0 відповідає січню.
console.log("Поточний рік:", currentDate.getFullYear()); // поверне чотиризначний рік, наприклад 2021.

```

Ви також можете отримати час із поточної дати, використовуючи методи `getHours()`, `getMinutes()` і` getSeconds ()`.

## Глибокий дайджест

Об'єкт `Date` має багато вбудованих методів для маніпуляції з датами, таких як встановлення дати, місяця, року, години, хвилин та секунд. Ви також можете використовувати метод `toLocaleDateString()`, щоб отримати поточну дату у бажаному форматі.

Наприклад, якщо вам потрібно отримати дату у форматі дд/мм/рррр, ви можете використовувати наступний код:

```TypeScript

// отримаємо поточну дату.
const currentDate: Date = new Date();

// використовуємо метод `toLocaleDateString()` з параметром формату.
console.log("Поточна дата:", currentDate.toLocaleDateString("uk-UA")); // поверне дату у форматі дд/мм/рррр, наприклад 22/07/2021.

```

Також не забувайте про часові зони, коли використовуєте методи з `Date`, оскільки вони можуть повертати різний результат залежно від зони, в якій працює ваш додаток.
