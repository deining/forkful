##Чому 
Зачому люди використовують текстові файли в програмуванні? На це питання є кілька причин. Спочатку, текстові файли можуть бути використані для зберігання даних, які змінюються рідко, такі як налаштування програми або список користувачів. Також ці файли можуть бути використані для обміну даних між різними програмами та платформами, оскільки вони є універсальним форматом. Нарешті, текстові файли можуть бути використані для зберігання даних, що можуть бути простими для редагування та пошуку, наприклад, список контактів або нотатки.

##Як використовувати
Для використання текстового файлу в TypeScript, спочатку потрібно імпортувати модуль "fs". Наприклад:

```TypeScript
import * as fs from 'fs';
```

Потім можливо відкрити файл для читання або запису, використовуючи метод "readFile" або "writeFile", відповідно. Приклад читання файлу та виведення його вмісту:

```TypeScript
fs.readFile('file.txt', (err: any, data: any) => {
  if (err) console.log(err);
  else console.log(data.toString());
});
```

Приклад запису даних до файлу:

```TypeScript
fs.writeFile('file.txt', 'Це приклад запису даних до файлу', (err: any) => {
  if (err) console.log(err);
  else console.log('Дані успішно записані до файлу');
});
```

##Вивчення питання детальніше
Оскільки текстові файли є важливою частиною програмування, важливо розуміти їх структуру та роботу. Текстові файли можуть бути відкриті для читання та запису у різних режимах, таких як додавання даних або перезапис поточного файлу. Також важливо зберігати файли у правильних кодуваннях для запобігання проблемам з кирилицею та іншими символами.

Також існують різні бібліотеки та утиліти для роботи з текстовими файлами, які можуть спростити процес, наприклад, "fs-extra" та "csv-parser". Вивчення цих інструментів може допомогти покращити продуктивність та ефективність при роботі з текстовими файлами.
