## Чому

Навіщо писати текстовий файл? Існує багато причин, чому це може бути корисною навичкою для вас як програміста. Перш за все, ви можете легко зберегти свій код та поділитися ним з іншими розробниками або зберегти його для майбутнього використання. Також це може бути корисним для документування вашого коду та створення звітів про ваші проекти.

## Як це зробити

Для написання текстового файлу використовується метод `writeFileSync()` з модуля `fs` в TypeScript. Ви можете використовувати будь-які типи даних, такі як рядки, числа або об'єкти, в залежності від того, що ви хочете записати. Нижче наведений приклад найпростішого шляху створення текстового файлу з рядками:

```TypeScript
// імпортуємо модуль fs
import * as fs from "fs";

// створюємо рядок, який буде записаним у файл
let text = "Це мій перший текстовий файл у TypeScript!";

// використовуємо метод writeFileSync() для створення файлу зі змістом рядка
fs.writeFileSync("myFile.txt", text);

// використовуємо метод readFileSync() для перевірки, чи був файл створений та містить правильний текст
let result = fs.readFileSync("myFile.txt", "utf-8");
console.log(result); // виводить "Це мій перший текстовий файл у TypeScript!"
```

## Глибока знахідка

Написання текстових файлів може бути корисним для багатьох завдань, включаючи обробку та збереження даних з зовнішніх джерел, створення простих баз даних або роботу зі звітами та логами. У TypeScript також є можливість використовувати асинхронні методи для запису та читання файлів, що дозволяє займатися іншими завданнями під час обробки файлу. Більш детально про роботу з файлами можна дізнатися у офіційній документації TypeScript.
