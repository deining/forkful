Що & Чому?
Створення текстового файлу - це процес створення файлу, що містить текстову інформацію. Це необхідна частина роботи програмістів, оскільки більшість програм потребують взаємодії з користувачем через текстові файли.

Як:
```TypeScript
// Приклад коду для створення текстового файлу
const fs = require('fs');

fs.writeFile('textfile.txt', 'Привіт, світе!', (error) => {
  if (error) throw error;
  // Виведе повідомлення, якщо файл успішно створений
  console.log('Текстовий файл успішно створений!');
});

// Приклад коду для читання текстового файлу
fs.readFile('textfile.txt', 'utf8', (error, data) => {
  if (error) throw error;
  // Виведе текстову інформацію з файлу
  console.log(data);
});
```

Глибоке занурення:
Створення текстових файлів давно вважається базовою операцією програмування. З часом, з'явилися більш спеціалізовані формати файлів, такі як CSV або JSON, які було зручніше використовувати для обміну даними. У TypeScript існує кілька способів створення текстового файлу, таких як використання модуля 'fs' та вбудованих засобів мови. Якщо необхідно читати великі файли, слід розглянути використання потоків замість повного завантаження файлу в оперативну пам'ять.
