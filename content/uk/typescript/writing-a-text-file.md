"## Чому"

З пильності до деталей у програмуванні, може здатися, що створення текстового файлу - це марно витрачений час. Проте, це важлива та необхідна навичка для будь-якого програміста. Знання, як правильно створювати текстові файли, є корисним не тільки для написання коду, але й для того, щоб зберігати дані та налаштування програм.

"## Як це зробити"

Створення текстових файлів - це завдання, з яким легко впоратися у TypeScript. Найпростіший спосіб створити текстовий файл - це використовуючи функцію `fs.writeFile()`, яка доступна у Node.js.

```TypeScript
fs.writeFile('text.txt', 'Це мій перший текстовий файл!', (err) => {
  if (err) throw err;
  console.log('Текстовий файл успішно створено!');
});
```

Цей код створює новий файл з назвою "text.txt" та записує в нього строку "Це мій перший текстовий файл!". Далі, за допомогою функції `console.log()`, виводиться повідомлення про успішне створення файлу.

Також, важливо знати, що є можливість додавати текст до вже існуючого файлу за допомогою функції `fs.appendFile()`.

```TypeScript
fs.appendFile('text.txt', ' Це додатковий текст.', (err) => {
  if (err) throw err;
  console.log('Текстовий файл змінено!');
});
```

Цей код додає строку "Це додатковий текст." до уже існуючого файлу "text.txt".

"## Розглянемо глибше"

В TypeScript є багато інших способів створення, редагування та читання текстових файлів: використання стандартних функцій з модуля `fs`, використання бібліотеки `fs-extra`, або навіть збереження файлу в форматі JSON. Навчитися користуватися цими методами допоможуть різні підручники та документація TypeScript.
