## Чому

Перевірка існування директорії є важливою частиною розробки програмного забезпечення. Вона дозволяє перевірити, чи існує окрема директорія на комп'ютері і при необхідності розпочати виконання певних дій з цими даними. Це є особливо важливим для програм, які використовують зовнішні файли та директорії для збереження та отримання даних.

## Як

 У TypeScript, є кілька способів перевірити існування директорії. Одним з найпростіших є використання функції `existsSync` з модуля `fs`. Вона приймає шлях до директорії як параметр і повертає `true`, якщо вона існує, або `false`, якщо вона не існує. Наприклад:

```TypeScript
import fs from 'fs'; // імпортування модуля fs

const directoryPath = 'C:/Users/username/Documents'; // шлях до директорії

if (fs.existsSync(directoryPath)) {
  console.log('Директорія існує'); // виводиться у разі успішної перевірки
} else {
  console.log('Директорія не існує'); // виводиться у разі невдалої перевірки
}
```

Також можна використовувати змінну `fs.constants.F_OK` для перевірки наявності директорії та `accessSync` для перевірки доступу до неї. Наприклад:

```TypeScript
import fs from 'fs';

const directoryPath = 'C:/Users/username/Documents';

try {
  fs.accessSync(directoryPath, fs.constants.F_OK); // перевірка наявності директорії
  console.log('Директорія існує і доступна'); // виводиться у разі успішної перевірки
} catch (err) {
  console.log('Директорія не існує або не доступна'); // виводиться у разі невдалої перевірки
}
```
