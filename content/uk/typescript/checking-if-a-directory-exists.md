## Чому

Перевірка існування директорії є важливим кроком в програмуванні для забезпечення безпеки та ефективності. Це дозволяє переконатися, що необхідний шлях існує та правильно налаштований перед використанням.

## Як

```TypeScript
import fs from 'fs';

// Перевірка за допомогою fs.existsSync()
if (fs.existsSync("/шлях/до/директорії")){
  console.log("Директорія існує");
} else {
  console.log("Директорія неіснує");
}

// Перевірка за допомогою fs.stat()
fs.stat("/шлях/до/директорії", (err, stats) => {
  if (err) {
    console.log("Директорія неіснує");
  } else {
    console.log("Директорія існує");
  }
});
```

В першому прикладі, ми використовуємо метод `existsSync()` з модулю `fs`, щоб перевірити, чи існує вказана директорія. Якщо директорія існує, ми отримуємо повідомлення "Директорія існує". У протилежному випадку, ми отримуємо повідомлення "Директорія неіснує".

У другому прикладі, ми використовуємо метод `stat()` для отримання інформації про файл. Ми передаємо шлях до нашої директорії та функцію зворотнього виклику, в якій ми перевіряємо наявність помилки. Якщо директорія існує, ми отримаємо повідомлення "Директорія існує". У разі неможливості отримати інформацію про файл, ми отримаємо повідомлення "Директорія неіснує".

## Deep Dive
