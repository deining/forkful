## Що та чому?

Перевірка існування директорії є процесом, за допомогою якого програмісти перевіряють, чи існує конкретна директорія на комп'ютері. Це важливо для забезпечення коректності роботи програми, щоб не траплялося помилок під час виконання коду.

## Як це зробити:

В TypeScript є декілька способів перевірити, чи існує потрібна директорія. Один з них - використання вбудованої функції ```existsSync ()```, яка повертає ```true``` або ```false``` в залежності від того, чи існує директорія.

```typescript
if (existsSync('./myDirectory')) {
    console.log('Директорія існує!');
} else {
    console.log('Директорія не існує!');
}
```

## Глибоке занурення:

Перевірка існування директорії є важливою частиною механізму обробки помилок. Якщо програма буде намагатися отримати доступ до неіснуючої директорії, вона може зупинитися або спричинити непередбачувані помилки. Альтернативою функції ```existsSync ()``` є використання методу ```access``` з модулем ```fs```, але це потребує додаткових кроків для перевірки дозволів доступу.
