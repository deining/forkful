## Чому

Ви можете потрапити в ситуацію, коли вам потрібно видалити певний рядок символів зі свого тексту. Наприклад, ви хочете видалити всі цифри з рядка, або небажані знаки пунктуації. Тоді ви можете використовувати TypeScript для пошуку та видалення символів, які відповідають певному шаблону.

## Як

Для початку, створіть змінну `str`, яка буде містити текст, з якого ви хочете видаляти символи. Тоді використайте метод `replace()` для заміни потрібних символів на пустий рядок:

```TypeScript
let str = "Hello, world!";

let newStr = str.replace(/[ ]/g, "");

console.log(newStr); // Output: Hello,world!
```

У цьому прикладі, ми застосували регулярний вираз `[ ]` для пошуку всіх пробілів у рядку, а флаг `g` вказує на глобальний пошук, тобто не лише перший співпадіння буде замінено.

Існує багато різних регулярних виразів, які можна використовувати для пошуку певних символів, наприклад:

- `/[0-9]/g` - для пошуку всіх цифр;
- `/[!@#$%^&*]/g` - для пошуку всіх символів пунктуації;
- `/[aeiou]/g` - для пошуку всіх голосних букв.

## Глибоке занурення

На практиці, ви можете використовувати метод `replace()` разом з будь-якими регулярними виразами, що підходять до ваших потреб. Важливо знати, що цей метод не змінює оригінальний рядок, а повертає новий рядок, в якому заміняються потрібні символи.

Також, варто зазначити, що метод `replace()` не тільки замінює символи, але й може використовуватися для вставки нових символів, видалення певного підрядка рядку та багатьох інших операцій.
