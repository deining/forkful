## Чому

Нерідко в програмуванні виникає потреба видаляти символи, які відповідають певному шаблону. Це може бути корисно для фільтрації введених даних або для перетворення рядків з певними характеристиками. У цій статті ми розглянемо, чому і як це може бути корисно для вас у програмуванні на TypeScript.

## Як це зробити

Для видалення символів, які відповідають певному шаблону, ми можемо використовувати метод `.replace()` встроєного об'єкта рядків. Цей метод приймає два аргументи: шаблон, за яким потрібно здійснювати пошук, і рядок, яким необхідно замінювати знайдені символи. Наприклад, якщо ми хочемо видалити всі цифри з рядка, ми можемо використати наступний код:

```TypeScript
let str = "Hello 123 World 456";
str = str.replace(/[0-9]/g, "");
console.log(str); // виведе "Hello World"
```

Ми використали шаблон `[0-9]`, який означає будь-яку цифру, і додали флаг `g`, щоб застосувати заміну для всіх входжень цього шаблону у рядку. Замість патерна ми також можемо використовувати строку, яка буде видаляти всі її входження. Наприклад:

```TypeScript
let str = "Hello 123 World 456";
str = str.replace("123", "");
console.log(str); // виведе "Hello World 456"
```

## Глибокий розбір

Метод `.replace()` може також приймати функцію як другий аргумент. Ця функція приймає знайдений патерн і повертає рядок, яким потрібно замінити цей патерн. В нашому прикладі з цифрами, ми можемо використати таку функцію:

```TypeScript
let str = "Hello 123 World 456";
str = str.replace(/[0-9]/g, (match) => {
  // match - знайдений патерн
  return parseInt(match) + 1; // повертаємо збільшене на 1 значення
});
console.log(str); // виведе "Hello 124 World 457"
```
