## Чому

Написання програм заснованих на TypeScript може бути викликом для багатьох, але часто це є невід'ємною частиною роботи програміста. Одним з завдань, які можуть виникнути, є видалення символів, які відповідають певному шаблону. Це може бути потрібно для фільтрації даних або для підготовки тексту для подальшої обробки. У цій статті ми поглиблено розглянемо, як це можна зробити за допомогою TypeScript.

## Як це зробити

Щоб видалити символи, які відповідають певному шаблону, нам потрібно використовувати регулярні вирази. У TypeScript, для роботи з регулярними виразами, ми можемо використовувати вбудований клас `RegExp`. Він дозволяє нам передати шаблон і здійснити пошук та зміну відповідних символів.

```TypeScript
// Приклад коду для видалення символів, що повторюються
const inputText = "Привітте"
const pattern = /(.)\1+/g;
const outputText = inputText.replace(pattern, "$1");
console.log(outputText);

// Виведе "Привте"
```

У цьому прикладі ми використовуємо регулярний вираз `(.)\1+`, який відповідає будь-якому повторюваному символу. Використовуючи метод `replace()` ми замінюємо всі повторення на перший символ, який ми знайшли за допомогою `$1`.

Ми також можемо використовувати регулярні вирази для фільтрації даних. Наприклад, якщо нам потрібно видалити всі знаки пунктуації з рядку, ми можемо використовувати наступний код:

```TypeScript
// Приклад коду для видалення знаків пунктуації
const inputText = "Hello, world!";
const pattern = /[^\w\s]/g;
const outputText = inputText.replace(pattern, "");
console.log(outputText);

// Виведе "Hello world"
```

У цьому прикладі ми використовуємо регулярний вираз `[^\w\s]`, який відповідає будь-якому символу, який не є буквою, цифрою чи пробілом. Використовуючи метод `replace()`, ми замінюємо ці символи на порожній рядок.

Нам було надзвичайно легко виконати ці дії завдяки регулярним виразам і TypeScript.
