## Чому

Багато з нас часто задумуються, як швидше та ефективніше заміняти текст у своїх програмах. Це може бути корисно під час редагування великих обсягів коду або під час вирішення проблем з орфографією чи граматикою. У TypeScript є простий та потужний спосіб для пошуку і заміни тексту за допомогою вбудованого методу `replace`. У цій статті ми розглянемо, як це працює та які є кращі практики для ефективного використання цього методу.

## Як використовувати

Для початку, ми створимо змінну `text` з деяким текстом, який ми хочемо замінити. Потім, ми викличемо метод `replace` на цій змінній та передамо два аргументи: регулярний вираз, який визначає текст, який потрібно замінити, та новий текст, який ми хочемо вставити на його місце.

```TypeScript
let text = 'Це текст з деякими словами, які ми хочемо замінити';
let newText = text.replace(/слова/g, 'рядки');
console.log(newText);
```

Виведення буде наступним: `Це текст з деякими рядками, які ми хочемо замінити`.

Ми також можемо використовувати змінні для тексту та тексту заміни, щоб наш код був більш динамічним:

```TypeScript
let word = 'слова';
let replaceWord = 'рядки';
let newText = text.replace(new RegExp(word, 'g'), replaceWord);
```

Цей приклад надає той самий результат, але дозволяє нам змінювати значення змінних та використовувати цей метод для різних текстів.

## Глибокий пір у метод `replace`

Тепер, коли ми розуміємо, як використовувати метод `replace`, давайте розглянемо деякі додаткові функції та можливості цього методу.

### Застосування функції до тексту

Замість передачі нового тексту як другого аргументу, ми також можемо передати функцію, яка буде виконуватися для кожного входження знайденого тексту. Функція повинна повертати текст, який буде використаний як заміна.

```TypeScript
let newText = text.replace(/слова/g, (match) => {
  // тут ми можемо робити будь-які додаткові операції з текстом
  return match.toUpperCase(); // повертаємо текст у верхньому регістрі
});

console.log(newText); // "ЦЕ ТЕКСТ З ДЕЯКИМИ РЯДКАМИ, ЯКІ МИ ХОЧЕМО ЗАМІНИТИ"
