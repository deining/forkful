## Чому
Успішне програмування вимагає ефективного використання часу та ресурсів. Пошук та заміна тексту є важливою вмінням, яке дозволяє здійснювати швидке та точне редагування коду. TypeScript надає можливість швидко та ефективно проводити процес пошуку та заміни тексту в програмах, що робить його незамінним інструментом для програмістів.

## Як
```TypeScript
// Створення нової змінної
let myVariable = "Привіт, світ!";

// Заміна тексту в змінній
myVariable = myVariable.replace("Привіт", "Привітання");

// Виведення змінної з заміненим текстом
console.log(myVariable); // Виведе "Привітання, світ!"
```
Щоб здійснити пошук та заміну тексту в TypeScript, потрібно використати метод `replace()`. Цей метод приймає два аргументи: шаблон (текст для пошуку) та замінюваний текст. Метод повертає новий рядок з заміненим текстом, тому ми можемо зберегти його в змінну або вивести прямо в консоль.

## Глибше
Метод `replace()` в TypeScript не тільки дозволяє замінювати один шаблон, але й майже будь-яку кількість шаблонів в одному рядку. Для цього потрібно використовувати регулярні вирази. Наприклад, ми можемо замінити всі коми на крапки у рядку:
```TypeScript
let sentence = "Це речення, яке використовує коми, а не крапки.";
let newSentence = sentence.replace(/,/g, "."); // Параметр "g" означає глобальне заміщення

console.log(newSentence); // Виведе "Це речення. Яке використовує коми. А не крапки."
```
Вираз `/,/g` є регулярним виразом, який визначає шаблон для пошуку - в нашому випадку це кома. Ми також можемо замінити тільки першу кому, використовуючи метод `replace()` без `g`:
```TypeScript
let firstComma = sentence.replace(",", "."); // Замінить тільки першу кому

console.log(firstComma); // Виведе "Це речення. Яке використовує коми, а не крапки."
```
