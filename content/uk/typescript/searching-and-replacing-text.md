## Що & чому?
Пошук та заміна тексту - це процес знаходження вказаного шаблону тексту у файлі та заміни його іншим текстом. Це корисний інструмент для програмістів, оскільки дозволяє швидко та ефективно змінювати великий обсяг текстових даних.

## Як:
```TypeScript
// Приклад коду:
let str = "Hello World";
let replaceStr = str.replace("Hello", "Привіт");
console.log(replaceStr);

// Вивід: "Привіт World"
```

## Глибокий погляд:
Історичний контекст: пошук та заміна тексту були вперше використані в популярному текстовому редакторі "vi" у 1976 році.

Альтернативи: інші інструменти для пошуку та заміни тексту, такі як функції "find and replace" у текстових редакторах та регулярні вирази у більш розширених програмах.

Деталі виконання: у TypeScript для пошуку та заміни тексту використовується метод `replace()`, який приймає два аргументи - перший - шаблон для пошуку, а другий - текст для заміни.
