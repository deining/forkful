## Чому

Нерідко під час програмування нам потрібно змінити окремі фрагменти тексту у великому обсязі коду. Це може бути зв'язано з оновленням стандартів кодування, заміною застарілого коду або іншими причинами. У таких випадках користувачам зручно мати інструмент для автоматичної заміни тексту, щоб заощадити час і уникнути помилок. У TypeScript, для цього є вбудоване рішення - функція `replace()`.

## Як замінювати текст у TypeScript

Перш за все, необхідно встановити TypeScript: `npm install -g typescript`.

Тепер створимо простий файл з кодом TypeScript:

```TypeScript
let sentence = "Це якийсь текст, який потрібно замінити.";
let newSentence = sentence.replace("текст", "пароль");
console.log(newSentence);
```

Вивід:

```
Це якийсь пароль, який потрібно замінити.
```

Якщо потрібно замінити всі входи фрагмента тексту, можна додати глобальний флаг `g`:

```TypeScript
let sentence = "Це якийсь текст, який потрібно замінити і тут ще один текст.";
let newSentence = sentence.replace(/текст/g, "пароль");
console.log(newSentence);
```

Вивід:

```
Це якийсь пароль, який потрібно замінити і тут ще один пароль.
```

## Заглиблення в пошук та заміну тексту

Функція `replace()` приймає два аргументи: пошук і заміну. Існує кілька можливих варіантів використання цих аргументів:

- String (шукає перший вхід і замінює його)
- RegExp (шукає всі входи і замінює їх)
- Function (викликається для кожного входу і повертає значення для заміни)

Детальнішу інформацію про цю функцію та її можливості можна знайти в [офіційній документації TypeScript](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-2-4.html#the-regexp-based-matching-functions).
