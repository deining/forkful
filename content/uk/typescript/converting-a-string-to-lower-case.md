## Чому

Іноді у програмуванні нам потрібно змінити регістр символів у рядку на нижній (lower case). Це може бути необхідно для правильної роботи алгоритмів або виведення даних у відповідному форматі. У цій статті я поділюся з вами, як виконати цей процес за допомогою TypeScript.

## Як це зробити

Для початку, нам потрібно створити змінну, яка буде містити наш рядок:

```TypeScript
let str: string = "HELLO WORLD";
```

Щоб змінити регістр усіх символів рядка на нижній, ми можемо використати метод `toLowerCase()`:

```TypeScript
console.log(str.toLowerCase());
```

Результат виконання такого коду буде наступним:

```
hello world
```

Зверніть увагу, що вихідний рядок залишився без змін, а метод `toLowerCase()` повернув новий рядок, перетворений до нижнього регістру. Тому, коли ви використовуєте цей метод в своїх проектах, не забувайте зберігати результат до змінної.

## Глибше

Якщо ви подивитеся на типову сигнатуру методу `toLowerCase()` у TypeScript, ви побачите, що він повертає `string`:

```TypeScript
toLowerCase(): string;
```

Це означає, що дана функція використовує типову безпеку та не може повернути нічого іншого, крім рядку. Це також дозволяє компілятору TypeScript виявляти можливі помилки в коді ще до його виконання.

Тепер давайте подивимося, як саме працює метод `toLowerCase()` у JavaScript. У браузері, його виконування залежить від використовуваної мови. Наприклад, російські символи будуть перетворені до нижнього регістру за допомогою методу `toLocaleLowerCase("ru-RU")`, а віршійські - за допомогою `toLocaleLowerCase("tr-TR")`. Деякі браузери навіть підтримують метод `toLowerCase()` для рядків з іншими алфавітами, наприклад, японським.

Хоча ці відмінності можуть стати причиною некоректної роботи програми, метод `toLowerCase()` виконується досить швидко, тому ви можете впевнено використовувати його для перетворення рядків до нижнього регістру.
