## Для чого

Читання аргументів командного рядка є важливим навичкою для кожного програміста, оскільки вона допомагає управляти програмами та передавати їм інформацію.

## Як це зробити

```TypeScript
const args = process.argv.slice(2); // отримання аргументів з командного рядка
console.log(`Перша аргумент: ${args[0]}`); // виведення першого аргументу
console.log(`Другий аргумент: ${args[1]}`); // виведення другого аргументу
```

Вище наведений приклад демонструє як отримати та вивести аргументи з командного рядка у TypeScript. Зверніть увагу, що ми використовуємо метод `.slice()` для обрізання перших двох елементів, оскільки перші два аргументи є шляхом до виконуваного файлу та назвою файлу. Ми також використовуємо індекси змінної `args` щоб вивести конкретні аргументи, але можна також використовувати цикл `for` для перебору всіх аргументів.

## Детальний аналіз

Коли ми запускаємо програму з командного рядка, ми можемо передати їй інформацію у вигляді аргументів. Наприклад, якщо ми запускаємо наш файл `app.ts` з наступним командним рядком `node app.ts hello world`, тоді `args` буде масивом, що містить першим елементом `hello` та другим `world`. Ми можемо використовувати цю інформацію для зміни поведінки нашої програми і додатково використовувати умовні операції або взаємодіяти з базою даних.
