# Чому

Порівняння дат є важливою задачею для багатьох програм, особливо для тих, що стосуються обробки даних.

# Як

Мова програмування TypeScript надає нам потужний інструмент для порівняння дат - клас Date. Давайте розглянемо декілька прикладів, як ми можемо виконати порівняння дат у TypeScript.

```typescript
// Створення двох об'єктів дати
let date1: Date = new Date(2020, 10, 12);
let date2: Date = new Date(2021, 2, 15);

// Порівняння дат
if (date1.getTime() > date2.getTime()) {
    console.log("date1 пізніше за date2");
} else if (date1.getTime() < date2.getTime()) {
    console.log("date2 пізніше за date1");
} else {
    console.log("date1 та date2 рівні");
}

// Вивід: date2 пізніше за date1
```

У прикладі вище ми створили два об'єкти дати та порівняли їх, використовуючи метод `getTime()`, який повертає кількість мілісекунд, які пройшли з 1 січня 1970 року. Це дає нам можливість легко порівняти дати за допомогою звичайних операторів порівняння.

```typescript
// Створення двох об'єктів дати
let date1: Date = new Date(2020, 10, 12);
let date2: Date = new Date(2021, 2, 15);

// Порівняння дат
if (date1 > date2) {
    console.log("date1 пізніше за date2");
} else if (date1 < date2) {
    console.log("date2 пізніше за date1");
} else {
    console.log("date1 та date2 рівні");
}

// Вивід: date2 пізніше за date1
```

У цьому прикладі ми також використали оператори порівняння для порівняння двох об'єктів дати. Ми можемо це зробити, оскільки TypeScript автоматично конвертує об'єкти дати в числа при порівнянні.
