## Чому

Це здібність, яка є необхідною для будь-якого програміста, особливо якщо ви працюєте з даними чи наводите логи у своїх проектах. Читання текстового файлу дозволяє отримувати необхідні дані та аналізувати їх у вашій програмі, що є надзвичайно корисною навичкою для розробників будь-якого рівня.

## Як це зробити

```TypeScript
import fs from 'fs';

// Зчитуємо текстовий файл
const data = fs.readFileSync('file.txt', 'utf-8');

// Виводимо зчитані дані у консоль
console.log(data);
```

Ви можете використовувати вбудовану функцію `readFileSync` з модулем `fs` для зчитування текстового файлу у вашій програмі. Відмінність між різними кодуваннями файлу може призвести до неправильного відображення тексту, тому важливо вказати правильне кодування при зчитуванні файлу. У нашому випадку, ми вказуємо `utf-8` як кодування.

## Глибокий занурення

Ви можете зчитувати не тільки цілий файл, але і окремі рядки чи символи з того ж самого файлу. Для цього використовуйте функцію `readFileSync` і вказуйте другим параметром початкову та кінцеву позиції для зчитування. Також, ви можете використовувати асинхронну функцію `readFile` для зчитування файлу у фоновому режимі.
