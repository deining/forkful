## Чому

У програмуванні існує багато ситуацій, коли нам потрібно прочитати текстовий файл. Це може бути для того, щоб отримати дані з файлу, аналізувати його або просто переглянути вміст. В цій статті ми розглянемо, як це можна зробити за допомогою TypeScript.

## Як це зробити

Для початку, давайте створимо текстовий файл з якимось вмістом. Наприклад, файл з назвою `sample.txt` з таким вмістом:

```
Це приклад тексту
для читання у програмі
з допомогою TypeScript
```

Тепер до роботи з TypeScript. Ми можемо використовувати стандартну бібліотеку `fs`, яка дозволяє працювати з файловою системою. Перш ніж читати файл, ми повинні його імпортувати за допомогою наступного коду:

```TypeScript
import * as fs from 'fs';
```

Тепер ми можемо використовувати функцію `readFileSync()` для прочитання файлу. Ця функція приймає два аргументи - шлях до файлу та кодування. Наприклад, якщо ми хочемо прочитати наш файл `sample.txt`, то код буде виглядати наступним чином:

```TypeScript
const content = fs.readFileSync('sample.txt', 'utf-8');
```

Інтерпретатор TypeScript прочитає файл та збереже його в змінну `content`. Тепер, якщо нам потрібно вивести цей вміст, то ми можемо використати функцію `console.log()`:

```TypeScript
console.log(content);
```

Це виведе на консоль вміст нашого текстового файлу.

## Глибоке дослідження

Крім `readFileSync()`, у бібліотеці `fs` є ще одна функція - `readFile()`, яка працює асинхронно. Це означає, що вона продовжить виконувати наступні рядки коду, не чекаючи завершення читання файлу. Ми можемо використовувати цю функцію, якщо нам потрібно прочитати більший обсяг даних або якщо оскільки вона є більш ефективною для використання в асинхронному коді.

Перевага використання файлів у TypeScript також в тому, що ми можемо використовувати типи даних для специфікації даних у файлі. Наприклад, якщо ми знаємо, що в нашому випадку файл містить тільки рядки, то ми можемо використовувати тип даних `string` для зіставлення типу даних читаного файлу.
