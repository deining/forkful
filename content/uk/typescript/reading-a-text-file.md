## Що і чому?
Читання текстового файлу - це процес отримання інформації з файлу, який містить текстову інформацію. Програмісти часто використовують цей процес для отримання даних з файлів, наприклад, для зчитування конфігурацій або інших налаштувань для своїх програм.

## Як?
У TypeScript, читання текстового файлу можна виконати з використанням класу FileReader. Нижче наведено приклад коду, який шукає файл з ім'ям "config.txt" у поточній директорії та читає його вміст. Після читання, вміст файлу виводиться у консолі.
```TypeScript
const fileReader = new FileReader();
fileReader.readAsText("config.txt")
fileReader.onload = () => console.log(fileReader.result);
```
Вихідні дані містять зчитаний текст з файлу "config.txt".

## Глибокий дайв
Історично, читання текстових файлів було виконано за допомогою методу readLine у мові програмування C. Цей метод читав кожен рядок файлу по черзі і повертав його в програму для обробки. Однак, в TypeScript це було спрощено з використанням класу FileReader, що робить процес читання більш зрозумілим та зручним.

Існують інші способи читання текстових файлів, наприклад, за допомогою модуля fs у Node.js або використовуючи стандартні функції у мовах програмування, таких як PHP або Python.

Щоб використовувати клас FileReader, потрібно імпортувати його з модуля "fs" у вашому коді. Після цього, можна використовувати різні методи для читання файлів, які відповідають вашим потребам.
