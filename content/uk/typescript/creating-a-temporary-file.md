## Чому

Створення тимчасового файлу може бути корисним в деяких випадках, наприклад, якщо необхідно створити файл для тимчасового збереження даних, які в подальшому будуть видалені. Це може допомогти уникнути переповнення пам'яті на пристрої або в разі потреби залишити сліди програми.

## Як створити

Використовуючи TypeScript, створення тимчасового файлу дуже просте. Для цього вам знадобиться встановлений пакет "tmp" за допомогою наступної команди:

```TypeScript
npm install tmp
```

Після цього можна створити тимчасовий файл за допомогою наступного коду:

```TypeScript
import { fileSync } from "tmp";

const tmpfile = fileSync();

console.log(tmpfile.name); // виводить шлях до тимчасового файлу
```

Ви також можете вказати певний префікс для тимчасового файлу, щоб його шлях був більш зрозумілим:

```TypeScript
const tmpfile = fileSync({ prefix: "temp" });

console.log(tmpfile.name); // виводить шлях до тимчасового файлу з префіксом "temp"
```

Після завершення роботи з тимчасовим файлом, його можна видалити за допомогою методу `.remove()`:

```TypeScript
tmpfile.remove();
```

## Глибока занурення

Хоча створення тимчасового файлу є простим, але варто також знати кілька особливостей. Наприклад, за замовчуванням пакет "tmp" створює тимчасовий файл, який автоматично видаляється при закритті програми. Якщо вам потрібно зберігати тимчасовий файл на довший час, зазначте параметр `unsafeCleanup` у `false` при створенні файлу:

```TypeScript
fileSync({ unsafeCleanup: false });
```

Також можна вказати шлях для тимчасових файлів за допомогою параметру `dir`:

```TypeScript
fileSync({ dir: "/tmp/tmpfiles" });
```

Більше докладної інформації про можливості пакету "tmp" можна знайти в його документації [тут](https://www.npmjs.com/package/tmp).
