## Чому

Найчастіше, якщо у вас є програма, яка не працює, потрібно розуміти, що саме відбувається всередині неї. Виведення даних для налагодження є корисним інструментом, який допоможе зрозуміти проблему і знайти розв'язок.

## Як досягти цього за допомогою TypeScript

JavaScript мова є динамічною, тому виведення значень для налагодження може бути складним. TypeScript справляється з цим завданням, надаючи нам типи даних та можливість використовувати інструменти для відстеження даних під час виконання. Ось декілька прикладів:

```TypeScript
let age: number = 27;
console.log("Мені " + age + " років.");
// Output: Мені 27 років.
```

У цьому прикладі ми створюємо змінну "age", яка має тип "number". Потім ми використовуємо команду "console.log", щоб вивести рядок зі значенням змінної "age". Коли програма виконається, вона виведе повідомлення "Мені 27 років.", яке допоможе нам зрозуміти, що значення було присвоєне змінній.

```TypeScript
let firstName: string = "Олександр";
let lastName: string = "Петренко";
console.log("Мене звати " + lastName + ", " + firstName + " " + lastName);
 // Output: Мене звати Петренко, Олександр Петренко
```

У цьому прикладі ми виводимо значення двох змінних "firstName" і "lastName", щоб створити рядок з нашим ім'ям і прізвищем.

### Глибше

Якщо ви використовуєте TypeScript для веб-розробки, ви можете виводити даних в консоль браузера для подальшого аналізу. Наприклад, ви можете вивести значення змінних, які містять дані отримані з API або форми, щоб перевірити, чи правильно вони були передані. Також ви можете виводити повідомлення після виконання певних дій, щоб впевнитися в їх успішному виконанні. Не бійтеся використовувати виведення даних для налагодження, це допоможе вам зрозуміти вашу програму краще.
