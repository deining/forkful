## З чого

Діагностичний вивід - це важлива частина програмування, яка дозволяє нам відстежувати та розуміти, що відбувається у нашому коді. Він дозволяє нам знайти та виправити помилки та оптимізувати наші програми. 

## Як це зробити

Щоб надрукувати діагностичний вивід в TypeScript, слід використовувати функцію `console.log()`. Наприклад, якщо ми хочемо вивести змінну `name` у консоль, ми можемо написати:

```TypeScript
console.log(name);
```

Результат буде відображений у консолі та допоможе нам перевірити, чи значення змінної `name` є правильним. Ми також можемо додавати більше інформації в діагностичний вивід, використовуючи рядки та оператор конкатенації `+`, наприклад:

```TypeScript
console.log("Привіт, " + name + ", як справи?");
```

Результат буде виглядати так:

```TypeScript
Привіт, Юра, як справи?
```
Це допоможе нам збільшити наш розуміння коду та виправити будь-які проблеми, які можуть виникнути.

## Глибока занурення

Крім простого друку змінних, функція `console.log()` також має багато інших корисних властивостей. Наприклад, ви можете додавати стилі до виведеного тексту використовуючи `%c` та назву стилю в якості аргументів, наприклад:

```TypeScript
console.log("%cЦя тема є дуже важливою!", "color: blue; font-size: 18px;");
```

Результат буде виглядати так:

```TypeScript
Ця тема є дуже важливою!
```

Також, ви можете використовувати `console.log()` для відслідковування конкретної фази виконання вашого коду, використовуючи `console.time()` та `console.timeEnd()`, наприклад:

```TypeScript
console.time("обчислення");
// Ваш код для обчислення чогось тут
console.timeEnd("обчислення");
```

Результат буде таким:

```TypeScript
обчислення: 123.456ms
```

Це дозволить нам виміряти час, необхідний для виконання певної дії та знайти проблемні ділянки коду, які потребують оптимізації.
