## Чому

Наданням великої літери рядку можна досягти більш читабельного і красивого вигляду. Це корисно для виводу інформації в користувацькому інтерфейсі.

## Як

Для отримання великої літери першого символу рядку можна скористатись методом `toUpperCase()` інтерфейсу `String`:

```TypeScript
const str: string = "привіт";
console.log(str.charAt(0).toUpperCase() + str.slice(1)); // Виведе "Привіт"
```

Якщо ж потрібно виконати таку операцію для великої кількості рядків, рекомендується створити функцію, яка буде приймати рядок і повертати його з першою великою літерою:

```TypeScript
function capitalize(str: string): string {
  return str.charAt(0).toUpperCase() + str.slice(1);
}

console.log(capitalize("привіт")); // Виведе "Привіт"
console.log(capitalize("світ")); // Виведе "Світ"
```

## Подробиці

Значення типу `string` є незмінним у TypeScript, тому ми не можемо змінити сам рядок і повернути його з великою літерою. Які ж методи ми використовуємо у функції `capitalize()`?

Метод `charAt()` повертає символ за заданим індексом в рядку. При цьому, індекси символів починаються з нуля. Тому, щоб отримати перший символ рядку, ми вибираємо символ за індексом 0.

Метод `toUpperCase()` перетворює рядок до верхнього регістру. Після цього за допомогою методу `slice()` ми видаляємо перший символ з рядку і об'єднуємо його з рештою рядка з верхнім першим символом.
