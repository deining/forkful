## Чому

Капіталізація рядків - це важливий процес у програмуванні, який дозволяє змінювати регістр букв у рядку. Це може бути необхідно для форматування тексту або для порівняння рядків. У цій статті, ми розглянемо, як здійснювати капіталізацію рядків у TypeScript.

## Як?

Для початку, нам потрібно мати строковий тип даних, до якого ми будемо застосовувати капіталізацію. Давайте розглянемо приклад коду:

```TypeScript
let str: string = "привіт світ";
console.log(str.toUpperCase());
```

На виході ми отримаємо "ПРИВІТ СВІТ", де всі букви перетворилися в верхній регістр. Для дотримання консистентності, можна також застосовувати капіталізацію до першої букви в рядку. Для цього використовуємо метод `charAt()` та `toUpperCase()`:

```TypeScript
let str: string = "hello world";
str = str.charAt(0).toUpperCase() + str.slice(1);
console.log(str);
```

На виході отримаємо "Hello world", де перша буква стала великою. Також, помітна зміна регістру стане видно у випадку, коли ми будемо порівнювати рядки. Давайте розглянемо приклад використання методу `localeCompare()`:

```TypeScript
let str1: string = "cat";
let str2: string = "dog";
if (str1.localeCompare(str2) === 0) {
  console.log("Рядки рівні");
} else {
  console.log("Рядки не рівні");
}
```

На виході отримаємо "Рядки не рівні", оскільки `cat` і `dog` мають різний регістр букв. Якщо запрацюємо з капіталізованими рядками:

```TypeScript
let str1: string = "Cat";
let str2: string = "Dog";
if (str1.localeCompare(str2) === 0) {
  console.log("Рядки рівні");
} else {
  console.log("Рядки не рівні");
}
```

То на виході ми отримаємо "Рядки рівні", оскільки тепер обидва рядки мають однаковий регістр букв.
