# Для чого: 

Конкатенація рядків - це процес поєднання двох чи більше рядків в один з їхнім наступним друком. Це дозволяє програмістам створювати більш складні та змістовні рядки, які виконують певні функції та працюють з більшим обсягом даних одночасно. 

# Як:

```TypeScript
const name = "Василь";
const age = 30;
const greeting = `Привіт, мене звати ${name} і мені ${age} років!`;
console.log(greeting); // Виведе: Привіт, мене звати Василь і мені 30 років!
```

В даному прикладі ми використали конкатенацію рядків з використанням змінної `${name}`, що дозволяє додати значення зі змінної до рядка. Також, ми можемо додавати не тільки змінні, але й будь-які інші рядки або символи, використовуючи оператор `+`.

# Глибше:

Конкатенація рядків є важливою частиною програмування, оскільки дозволяє зручний та ефективний спосіб об'єднати різні дані та створити щось більш складне з них. Крім того, використання шаблонних строк (змінні в дужках) дозволяє програмістам зберігати код більш чистим та зрозумілим для інших розробників, оскільки не потребує додаткових операцій з конкатенацією.
