## З чого починати

У TypeScript є багато корисних функцій, але одна із них, можливо, є найбільш корисною для вас - це можливість конкатенувати рядки. Люди часто використовують цю функцію для поєднання двох або більше рядків, щоб створити новий рядок з більшою значущістю, можливо, для виведення на екран або для використання в логіці програми.

## Як це зробити

```TypeScript const greeting = "Привіт"; const name = "Олександр"; const message = greeting + " " + name + "! Вітаю з успішним розвідуванням TypeScript!"; console.log(message); // Виводиться на екран "Привіт Олександр! Вітаю з успішним розвідуванням TypeScript!" ```

У цьому прикладі ми скористалися оператором "+" для об'єднання рядків "Привіт", "Олександр" і "Вітаю з успішним розвідуванням TypeScript!" в один рядок. Можете спробувати самостійно змінити рядки і побачити, як це вплине на остаточний результат. Не забувайте додавати пробіли і коми, щоб ваш вивід був більш читабельним!

## Глибоке занурення

Строковий тип даних у TypeScript називається "string". Він може містити будь-який символ, включаючи літери, цифри, пробіли і спеціальні символи. Якщо ви хочете з'єднати декілька рядків, ви можете використовувати будь-який оператор об'єднання, такий як "+", але у більш складних випадках надійніше використовувати метод "concat()" для рядків. Ви також можете використовувати шаблонні рядки (string templates) з знаком "`" для зручної конкатенації рядків.
