## Чому

Строки є невід'ємною частиною програмування, і часом нам потрібно об'єднати декілька рядків у один. Найпоширенішим способом для цього є використання конкатенації рядків (або просто додавання їх разом). В цій статті ми поговоримо про те, як виконувати цю операцію за допомогою TypeScript.

## Як

За допомогою TypeScript, ми можемо додавати рядки разом двома способами: за допомогою оператора "+" та за допомогою методу "concat()". Давайте подивимося на приклади коду, щоб краще зрозуміти ці методи.

```TypeScript
let name = "Олена";
let greeting = "Привіт, ";

let sentence = greeting + name;
console.log(sentence);
// Виведе: Привіт, Олена

let sentence2 = greeting.concat(name);
console.log(sentence2);
// Виведе: Привіт, Олена 
```

Як бачимо, результати обох методів ідентичні. Ми також можемо додавати більше ніж два рядки разом, застосовуючи цей же підхід.

```TypeScript
let language = "JavaScript";
let adjective = "цікава";
let sentence3 = "Це мова програмування " + adjective + " - " + language + "!";
console.log(sentence3);
// Виведе: Це мова програмування цікава - JavaScript!
```

## Deep Dive

У TypeScript є ще один корисний технічний прийом - шаблонні рядки. Вони дозволяють нам створювати рядки з допомогою певних виразів, застосовуючи спеціальний синтаксис з символом "`. Давайте подивимося на приклад:

```TypeScript
let age = 25;
let name = "Марія";
let sentence = `Привіт, мене звати ${name} і мені ${age} років.`;
console.log(sentence);
// Виведе: Привіт, мене звати Марія і мені 25 років.
```

Як бачимо, ми використали спеціальну синтаксичну конструкцію `${...}` для вставки змінних в рядок. Це дуже зручно, особливо якщо ми маємо багато змінних, які потрібно об'єднати в рядок.
