## Чому

Конкатенування рядків - це процес з'єднання двох або більше рядків в один, більш довгий рядок. Це корисне для будь-якої програми, що працює з текстом, оскільки дозволяє легко створювати нові рядки з уже існуючих. Наприклад, ви можете додати ім'я користувача до вітання або створити повідомлення з відповідною інформацією.

## Як виконати

```TypeScript
// Створюємо два рядки
let firstName = "Олександр";
let lastName = "Петренко";

// Конкатенуємо їх разом з допомогою "+" оператора
let fullName = firstName + lastName;

// Виводимо повне ім'я
console.log(fullName); // Виконання: Олександр Петренко
```

В даному прикладі ми створили два рядки з іменем та прізвищем, а потім об'єднали їх за допомогою оператора "+" для створення нового рядка з повним іменем. Ви можете також додавати більше рядків або інших даних, використовуючи цей же метод.

## Глибоке занурення

У TypeScript є багато різних способів конкатенування рядків, включаючи використання зворотних лапок ("` `") та методу `concat()`. У зворотних лапках ви можете вставляти значення змінних без потреби записувати їх явно. Наприклад:

```TypeScript
let firstNumber = 10;
let secondNumber = 20;

// Використання зворотних лапок
let sum = `Сума чисел ${firstNumber} та ${secondNumber} дорівнює ${firstNumber + secondNumber}`;

console.log(sum); // Виконання: Сума чисел 10 та 20 дорівнює 30
```

Метод `concat()` дозволяє об'єднувати багато рядків одночасно, що є корисним для більш складних завдань. Наприклад:

```TypeScript
let firstSentence = "Сьогодні";
let secondSentence = "дуже";
let thirdSentence = "гарний";
let fourthSentence = "день";

// Використання методу concat()
let fullSentence = firstSentence.concat(secondSentence, thirdSentence, fourthSentence);

console.log(fullSentence); // Виконання: Сьогоднідужегарнийдень
```

Зверніть увагу, що після завершення конкатенації ви отримали один рядок без пропусків між словами. Якщо вам потрібно додати пробіл між кожним словом, ви можете додати " " у кінці кожного рядка, використовуючи опцію додавання `+ " "`.
