## Чому

Генерація випадкових чисел є важливою частиною програмування, оскільки вона дозволяє вирішити різноманітні задачі, які вимагають випадковості. Наприклад, ми можемо використовувати випадкові числа для генерації унікальних ідентифікаторів, випробування алгоритмів або створення реалістичної симуляції.

## Як

```typescript
// Генерація випадкового цілого числа в заданому діапазоні
const randomNumber = Math.floor(Math.random() * (max - min + 1) + min);

// Генерація випадкового дійсного числа між 0 та 1
const randomDecimal = Math.random();

// Згенеруємо масив з 10 випадкових чисел
const randomArray = [];
for (let i = 0; i < 10; i++) {
    randomArray.push(Math.random());
}
```

В цьому прикладі ми використали вбудований об'єкт `Math`, який має метод `random()`, що повертає випадкове число між 0 та 1. Якщо нам потрібно згенерувати число в заданому діапазоні, ми можемо використовувати формулу `Math.random() * (max - min + 1) + min`. Також ми можемо виконати декілька ітерацій, щоб отримати більше випадкових значень і зберегти їх у масив.

## Глибокий занурення

У TypeScript є вбудований модуль `crypto`, який має метод `randomBytes()`, що дозволяє генерувати безпечні випадкові байтові дані. Це може бути корисно для створення криптографічних ключів чи захисту чутливих даних. Ми також можемо використовувати сторонні бібліотеки, наприклад, `uuid`, яка дозволяє швидко створювати унікальні ідентифікатори за допомогою випадкових чисел.
