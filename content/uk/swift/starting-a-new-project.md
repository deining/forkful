## Чому

Створення нового проекту - це захоплююча та стимулююча подія для будь-якого програміста. Це дає можливість використовувати свої знання та творчість для створення щось нового та корисного.

## Як це зробити

```Swift
// Створення нового проекту
let projectName = "Мій перший проект"
print("Вітаю! Ви створили новий проект під назвою \(projectName)")
```

Коли ви створюєте новий проект в Swift, вам необхідно виконати наступні кроки:

1. Відкрийте Xcode та натисніть "Create a new Xcode project".
2. Виберіть тип проекту (наприклад, "Single View App" або "Game").
3. Введіть назву проекту та виберіть місце для збереження файлів.
4. Натисніть "Next" та налаштуйте проект за потребою (наприклад, обрати мову програмування та пристрій для тестування).
5. Натисніть "Create" та почніть розробку свого проекту!

## Глибоке занурення

Створення нового проекту може бути складною задачею, але це також дуже захоплююче. Існує безліч додатків та ігор, які були створені за допомогою Swift, тому це чудова можливість перевірити свої вміння та навички. Також, не бійтеся експериментувати та використовувати різні фреймворки та бібліотеки для поліпшення свого проекту.
