# Що & Чому?
Старт нового проекту - це коли програмісти починають створювати нову програму, додаток або веб-сайт з нуля. Вони це роблять, щоб реалізувати свої ідеї, вирішити проблеми або просто покращити існуючі продукти.

# Як:
Створення нового проекту в Swift досить просте. Спочатку вам потрібно відкрити Xcode - основну інтегровану середу розробки для Swift. Потім додайте новий проект, вибравши шаблон, який найбільше підходить до вашого проекту. Ви можете вибрати з різних шаблонів, таких як одиночний вид, веб-сайт, гра або додаток для мобільного пристрою. Після цього ви можете почати кодувати і використовувати всі можливості мови Swift для досягнення своїх цілей.

```Swift
// Приклад коду Swift
var number = 5
if number >= 5 {
  print("Це число більше або дорівнює 5")
}
```

### Результат:
```Це число більше або дорівнює 5```

# Детальний розбір:
Swift був створений компанією Apple в 2014 році з метою замінити мову Objective-C. Вона була створена для максимальної продуктивності та простоти розробки, що робить її дуже популярною серед програмістів. Є також альтернативи для розробки проектів на Swift, такі як Flutter або React Native, але Swift є стандартною мовою для розробки iOS та macOS додатків.
