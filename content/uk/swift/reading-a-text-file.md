## Що і чому?
Читання текстового файлу - це процес отримання інформації з файлу, що містить текст. Це один із широко використовуваних способів отримання даних для подальшої обробки або використання в програмі. Програмісти часто читають текстові файли, щоб отримати доступ до важливих даних, які необхідні для їх програм.

## Як:
```
let fileName = "example.txt" // ім'я файлу
let fileURL = URL(fileURLWithPath: fileName) // створення URL з ім'ям файлу

do {
    let fileContent = try String(contentsOf: fileURL) // зчитування вмісту файлу у вигляді рядка
    print(fileContent) // виведення зчитаного вмісту
} catch {
    print("Помилка при прочитанні файлу: \(error)") // виведення помилки, якщо файл не може бути зчитаний
}
```
**Виод:** Вміст файлу example.txt.

## Глибинний аналіз:
Читання текстових файлів має дуже довгу історію і є одним з основних способів роботи з даними на комп'ютері. Однак, існують інші способи отримання даних, такі як робота з базами даних або використання API. У Swift, є різні способи читання текстових файлів, наприклад, використання методу `contentsOf URL` для читання з URL або отримання `Data` об'єкту з `FileManager`.
