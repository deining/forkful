## Чому

У програмуванні Swift часто потрібно витягати підрядки з великих рядків даних. Це може бути необхідно для обробки текстових даних або для подальшої роботи з підрядками окремо. Тому вивчення витягування підрядків є важливим для будь-якого програміста.

## Як це зробити

Використовуючи вбудовану функцію `substring` в Swift, можна легко витягувати підрядки з великих рядків. Приклад коду та його виведення для витягування підрядка "world" з рядка "Hello world":

```Swift
let str = "Hello world"
let index = str.index(str.startIndex, offsetBy: 6) // знаходимо індекс для початку витягуваного підрядка
let substring = str.substring(from: index) // витягуємо підрядок з вказаного індексу до кінця рядка
print(substring) // виведення: world
```

За допомогою параметру `offsetBy` можна вказати, скільки символів треба пропустити, а параметр `from` вказує з якого індексу починати витягувати підрядок. Крім того, з `substring` можна витягувати підрядки за допомогою інших параметрів, які дозволяють вказувати початковий та кінцевий індекси для витягування.

## Глибоке занурення

У Swift є також можливість використовувати оператори `..<` і `...` для створення зручних діапазонів для витягування підрядків. Наприклад, код `str[..<index]` буде витягувати підрядок з початку рядка до індексу `index` (не включаючи його).

Також варто врахувати, що вбудований тип `String.Index` в Swift дозволяє виконувати операції порівняння і обчислювати індекси, що дуже зручно для витягування підрядків.
