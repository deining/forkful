## Про що: Чому виділення підрядків є важливою навичкою для багатьох програмістів

Виділення підрядків є невід'ємною частиною роботи зі стрічками у багатьох програмах. Це дозволяє отримувати певні частини тексту, необхідні для подальшої обробки або використання. Наприклад, нам можуть знадобитися лише певні символи із введеного користувачем імені, або лише деякі слова зі статті на сайті. У даному блозі ми розглянемо, як краще використовувати функції Swift для виділення підрядків та чому це є так важливою навичкою для програмістів.

## Як: Приклади коду та вихідні дані для виділення підрядків

```Swift
let str = "Це простий текст для прикладу."
let index = str.index(str.startIndex, offsetBy: 16)
let substring = str[index...] // "текст для прикладу."

let food = "Пиріг з яблуками"
let slice = food.prefix(10) // "Пиріг з я"

let sentence = "Сьогодні гарний день для прогулянки"
let range = if let range = sentence.range(of: "гарний день") {
  let substring = sentence[range] // "гарний день"
}
```
В першому прикладі ми використовуємо функцію `index` для вирахування певного символу із стрічки, а потім створюємо підрядок з цього символу до кінця стрічки. У другому прикладі ми використовуємо метод `prefix`, який дозволяє вибрати перші N символів із стрічки. У третьому прикладі ми використовуємо метод `range(of:)`, який шукає певний підрядок у стрічці і повертає його діапазон. Потім ми використовуємо цей діапазон для виділення підрядка зі стрічки.

## Глибоке дослідження: Додаткова інформація про виділення підрядків

У Swift існує безліч функцій та методів для роботи зі стрічками. Деякі з них дозволяють виділяти підрядки за допомогою діапазонів чи позицій символів, інші - за допомогою певних шаблонів. Крім того, важливо пам'ятати про специфічні особливості роботи зі стрічками, такі як різниця між Unicode та ASCII символами. Глибоке вивчення цих можливостей допоможе вам більш ефективно використовувати функції для виділення підрядків у своїх програмах.
