## Що і чому?
Витягнення підрядків - це процес отримання деякого підрядка з рядка (текстового ланцюжка). Це корисний інструмент для програмістів, оскільки дозволяє працювати з частинами тексту замість цілого рядка.

## Як це зробити:
```Swift
let str = "Hello World"
let index = str.index(str.startIndex, offsetBy: 6) // вираховуємо, звідки почати витягування підрядка (починаємо зсунувати стартовий індекс від початку рядка на необхідну кількість)
let substring = str[index...] // вичитуємо підрядок, починаючи з вирахованого індексу до кінця вхідного рядка

print(substring) // "World"
```

## Глибоке погруження:
(1) Витягнення підрядків було вперше представлене в мові програмування C. (2) Іншими способами роботи зі строками є заміна і розділення, але вони мають свої обмеження - заміна замінить всі зустрінуті в тексті символи, а розділення поверне лише перший знайдений підрядок. (3) Swift використовує тип даних String, що імплементує протокол Collection, що подає можливість використовувати багатий функціонал для роботи з рядками, зокрема, для витягнення підрядків.
