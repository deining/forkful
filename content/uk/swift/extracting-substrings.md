## Причина

Один із найбільш корисних інструментів у Swift - здатність витягувати підрядки з рядків. Це допомагає розбивати рядки на менші частини для подальшої обробки або відображення. Якщо ви ще не використовували цю функцію, цей пост допоможе вам розібратися в її використанні.

## Як

Для початку варто зазначити, що у Swift існує кілька способів витягування підрядків з рядків. Найпростіший з них - метод `prefix()` та `suffix()`, які повертають перші або останні символи рядка відповідно. Наприклад:

```Swift
let name = "Іванова Ірина"
print(name.prefix(6))
// Виведе "Іванов"
```

Для видалення певної кількості символів на початку або в кінці рядка, можна використовувати методи `dropFirst()` та `dropLast()`. Наприклад:

```Swift
let programmingLanguage = "Swift"
print(programmingLanguage.dropFirst(2))
// Виведе "ift"
```

Також існує можливість витягувати підрядки за певним регулярним виразом, використовуючи метод `range(of:)`. Цей метод поверне діапазон індексів з вказаним регулярним виразом. Наприклад:

```Swift
let text = "Hello, world!"
if let range = text.range(of: "[a-z]{5}", options: .regularExpression) {
    print(text[range])
    // Виведе "world"
}
```

## Глибока занурення

Описані вище методи - лише маленька частина можливостей витягування підрядків в Swift. Існує багато інших методів, які можуть бути корисними в різних ситуаціях, таких як `split()`, `components(separatedBy:)` або `filter()`. Крім того, ці методи можуть бути використані не тільки для роботи зі звичайними рядками, але і в комбінації з колекціями та іншими типами даних.

Також важливо звернути увагу, що згадувані методи повертають так звані `SubSequence`, що є підрядком оригінального рядка. Це означає, що якщо ви збережете цей підрядок у змінну, він буде посилатися на оригінальний рядок. Для отримання самого підрядка потрібно використовувати метод `String()`.
