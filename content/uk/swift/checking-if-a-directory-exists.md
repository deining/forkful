# Чому
Чи ви коли-небудь шукали спосіб перевірити, чи існує певна папка на вашому пристрої? Наприклад, вам може знадобитися перевірити наявність папки для збереження фотографій або інших важливих файлів. У цій статті ми поговоримо про те, як це зробити з допомогою програмування на Swift.

# Як це зробити
Для перевірки існування папки нам знадобиться використовувати функцію `FileManager.default.fileExists(atPath:)`. Ця функція приймає рядок, який відповідає шляху до папки, і повертає булеве значення, що вказує на наявність чи відсутність цієї папки. Ось приклад коду:
```Swift
let fileManager = FileManager.default
let folderPath = "/Users/test/UserData"
if fileManager.fileExists(atPath: folderPath) {
    print("Папка існує")
} else {
    print("Папка не існує")
}
```
Якщо папка `UserData` існує у заданому шляху, то на екрані буде виведено повідомлення "Папка існує". В іншому випадку буде виведено "Папка не існує".

# Глибинний аналіз
Якщо ви бажаєте докладніше дізнатися про перевірку існування папки, то зверніть увагу на наступні моменти:

- Рядок, який передається до функції `fileExists(atPath:)`, повинен бути повним шляхом до папки, включаючи ім'я папки.
- Якщо ви працюєте з додатком для iOS чи macOS, то шлях до папки буде виглядати трохи іншим і передавати його до `fileExists(atPath:)` слід з додатковою метою, оскільки шлях до папок у цих операційних системах є динамічним.
- Будьте уважні до регістру символів у шляху до папок. Код чутливий до регістру, тобто папка "UserData" та "userdata" будуть розпізнаватися як різні папки.
