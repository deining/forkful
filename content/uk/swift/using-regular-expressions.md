## Чому
Українські програмісти часто зіткнуться з необхідністю використовувати регулярні вирази у своїх програмах. Регулярні вирази - це потужний інструмент для пошуку, заміни та перевірки тексту за певним шаблоном. Вони дозволяють ефективно обробляти інформацію, що містяться у рядках, що є необхідним для багатьох програм.

## Як це зробити
```Swift
let regex = try! NSRegularExpression(pattern: "([0-9]+)") // ініціалізація регулярного виразу
let string = "Це 52 рік, і Христині 22 років." // текст, який будемо перевіряти
let match = regex.firstMatch(in: string, range: NSRange(string.startIndex..., in: string)) // перша знахідка числа у тексті
if let matchRange = match?.range(at: 1), // визначаємо діапазон знахідки
    let range = Range(matchRange, in: string) { // перетворюємо на дійсний діапазон у рядку
    let numberString = string[range] // отримуємо рядок числа
    print(numberString) // виводимо "52"
}
```
В даному прикладі ми створюємо регулярний вираз, який шукає будь-які послідовності цифр у тексті. Потім ми перевіряємо, чи є така послідовність у заданому рядку і виводимо її на екран. В реальному використанні, можна виконати будь-які дії з отриманим числом, наприклад, зберегти його у змінну або перетворити на числовий тип даних.

## Глибоке дослідження
Регулярні вирази мають багато можливостей та функцій для ефективного використання. Наприклад, вони підтримують використання метасимволів, які дозволяють шукати широкий діапазон символів за певним шаблоном. Також, ви можете використовувати регулярні вирази для перевірки правильності введених користувачем даних, таких як електронна пошта, номер телефону чи ім'я користувача. І хоча ініціалізація та використання регулярних виразів може здатися складною на початковому етапі, вони важливий інструмент для будь-якого програміста.
