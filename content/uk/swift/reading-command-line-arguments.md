## Що & Чому?
Отримання аргументів командного рядка це спосіб для програмістів отримати інформацію, що була введена під час запуску програми. Це може бути корисно для налаштування програми або передачі вхідних даних. 

## Як:
Для отримання аргументів командного рядка використовуються змінні зі знаком "$" перед їхнім іменем. Наприклад, якщо ваша програма запускається з аргументом "Hello", ви можете отримати цей аргумент наступним чином:

```Swift
let argument = CommandLine.arguments[1]
print(argument) // Виведе "Hello"
```
Якщо ваша програма була запущена з кількома аргументами, вони будуть виведені у вигляді масиву:

```Swift
let arguments = CommandLine.arguments
print(arguments[0]) // Виведе назву програми
print(arguments[1]) // Виведе перший аргумент
print(arguments[2]) // Виведе другий аргумент та так далі
```

## Глибшим поглядом:
Отримання аргументів командного рядка було створено у далекому 1973 році для операційної системи UNIX. На сьогоднішній день, існують інші способи передавання вхідних даних у програму, такі як зчитування з файлів або вводу за допомогою користувача. Як для реалізації, отримання аргументів командного рядка використовує механізм, що називається "параметри командного рядка". Цей механізм дозволяє системі передавати дані програмі при її запуску.
