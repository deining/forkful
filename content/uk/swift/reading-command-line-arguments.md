## Чому

Консольні аргументи - важлива частина програмування на Swift, особливо для розробників, які хочуть створювати багатофункціональні програми. Ця стаття допоможе вам зрозуміти, наскільки корисними є консольні аргументи та як їх можна використовувати у вашому коді.

## Як

Найпростіше вважати консольні аргументи аргументами командного рядка. Вони дозволяють передавати інформацію до програми при запуску з терміналу. Їх можна використовувати у багатьох сценаріях, наприклад:

```Swift
// Отримання всіх переданих аргументів
let arguments = CommandLine.arguments

// Виведення першого аргумента
print(arguments[0])

// Перевірка, чи введено необхідний аргумент
if arguments.count > 1 {
    let argument = arguments[1]

    // Виконання певних операцій з введеним аргументом
    // Наприклад, виклик певної функції чи створення об'єкта з введеним ім'ям
}
else {
    // Виведення повідомлення про недостатню кількість аргументів
    print("Для програми потрібен ще хоча б один аргумент")
}
```

## Deep Dive

Надалі ви можете розширити можливості консольних аргументів, додавши їм параметри. Це дає змогу передавати до програми не тільки дані, а й вказувати опції запуску. Наприклад, команда `myProgram calculate -v` буде запускати програму `myProgram` з опцією `-v`, яка вказує на необхідність виведення детальної інформації під час виконання дії `calculate`.

```Swift
// Обробка аргументів з параметрами
for i in 1..<arguments.count {
    let argument = arguments[i]

    // Перевірка, чи даному аргументу переданий параметр
    if argument.hasPrefix("-") && i+1 < arguments.count {
        let parameter = arguments[i+1]

        // Обробка параметра за його значенням
        if argument == "-v" {
            // Виведення детальної інформації для дії calculate
            print("Розрахунки виконуються в режимі verbose")
        }
    }
}
```

Зверніть увагу, що в даному прикладі ми проходимо по всіх аргументах, починаючи з першого. Якщо аргумент має префікс `-` (що відповідає параметрам), ми перевіряємо, чи наступний аргумент є параметром. Якщо так, то обробляємо його відповідно до нашої логіки.
