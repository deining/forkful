## Чому

Стрінги - це один з найбільш використовуваних типів даних у програмуванні. Зниження регістру строк часто використовується для створення різних функцій і фільтрів. Конвертування строк до нижнього регістру може полегшити роботу з ними і зробити код більш читабельним.

## Як

Щоб сконвертувати рядок до нижнього регістру в Swift, використовуйте метод `lowercased()` для змінної типу `String`. Давайте подивимося на приклад коду:

```Swift
let string = "HELLO WORLD"
print(string.lowercased())
```

Вивід:
```
hello world
```

Як бачите, метод `lowercased()` повертає новий рядок зі зниженим регістром. Якщо бажаєте зберегти зміну у вихідній змінній, просто присвойте їй нове значення:

```Swift
var string = "HELLO WORLD"
string = string.lowercased()
print(string)
```

Ви також можете знизити регістр всіх букв у рядку, використовуючи метод `lowercased(with:)` з параметром типу `Locale`, який визначає мову та культуру письма. Для прикладу, давайте перетворимо рядок українською мовою у нижній регістр за допомогою локалі `uk_UA`:

```Swift
let string = "ПРИВІТ СВІТ"
print(string.lowercased(with: Locale(identifier: "uk_UA")))
```

Вивід:
```
привіт світ
```

## Поглиблене вивчення

Конвертування строк до нижнього регістру може бути корисною не тільки для створення функцій і фільтрів, але й для порівняння і пошуку текстової інформації. Проте, слід пам'ятати, що цей процес змінює рід створеної строки, тобто вона вже не буде відповідати оригінальних значенням.

Для збереження оригінальної строки і порівняння її зі строкою в нижньому регістрі, можна використати метод `caseInsensitiveCompare(_:)`:

```Swift
let string = "Apple"
let lowercaseString = "apple"

if string.caseInsensitiveCompare(lowercaseString) == .orderedSame {
    print("Обидва рядки містять одинакову інформацію")
}
```

Вивід:
```
Обидва рядки містять одинакову інформацію
```

Також, конвертування строк до нижнього регістру може викликати проблеми з Unicode символами, які можуть мати різні форми верхнього та нижнього регістру. У таких випадках, бажано використовувати метод `lowercased(options:, locale)` з параметром `options`, який вказує додаткові нала
