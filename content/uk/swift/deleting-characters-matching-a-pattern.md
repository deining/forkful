## Чому

Іноді при програмуванні необхідно видаляти певні символи зі строк згідно з певним шаблоном. Це може бути корисно для очищення даних або для перетворення даних в бажаний формат. У цій статі ми розглянемо, як зробити це в мові Swift.

## Як

Процес видалення символів зі строк в мові Swift можна реалізувати за допомогою методу `replacingOccurrences`, який приймає два параметри. Перший параметр - це шаблон, за яким потрібно замінювати символи, а другий - символ, на який потрібно замінювати.

Наприклад, якщо ми хочемо видалити всі пробіли зі string, то ми можемо використати такий код:

```Swift
var str = "Це речення з пробілами"

str = str.replacingOccurrences(of: " ", with: "")
print(str)

// Output: Цереченнязпробілами
```

Якщо ж ви хочете видалити символи, що повторюються підряд, ви можете використати регулярні вирази. Наприклад, такий код видалить всі символи, які повторюються більше двох разів підряд:

```Swift
let str = "лововоєбадбаджон"

let regex = "([a-zA-Z])\\1{2,}"
let newStr = str.replacingOccurrences(of: regex, with: "$1", options: .regularExpression)
print(newStr)

// Output: ловбадєбадбаджон
```

## Глибоке занурення

Видалення символів за певним шаблоном може бути корисним інструментом для обробки даних. Мова Swift має багато вбудованих методів для роботи зі строками, тому, якщо ви ще не ознайомилися з ними, радимо почитати документацію.

Окрім того, регулярні вирази є потужним інструментом для роботи зі строками, тому варто вивчити їхні можливості для більш складних задач.
