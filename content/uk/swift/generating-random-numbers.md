## Чому

Згенерувати випадкові числа є необхідною здатністю для багатьох програм на Swift. Це може бути корисно, наприклад, для генерації унікальних ідентифікаторів, створення випадкових паролів або просто для тестування програм.

## Як

Існує декілька способів згенерувати випадкові числа на Swift, але ми розглянемо два основних.

```Swift
//  Генерує випадкове число в діапазоні від 0 до 9
let randomNumber = Int.random(in: 0...9)
print(randomNumber) // Виведе, наприклад, 5
```

```Swift
// Генерує випадкове число з плаваючою точкою в діапазоні від 0 до 1
let randomDouble = Double.random(in: 0..<1)
print(randomDouble) // Виведе, наприклад, 0.3764519689
```

## Глибокий дайв

Схоже, що генерування випадкових чисел є процесом, що запускається внутрішньою алгоритмічною послідовністю, що гарантує унікальність кожного числа. Цей алгоритм може бути застосований до будь-якого типу даних, наприклад до цілих чисел, чисел з плаваючою точкою чи навіть до тексту.
