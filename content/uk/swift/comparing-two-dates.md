## Чому

В програмуванні часто потрібно порівнювати дві дати. Це може бути корисно, якщо потрібно перевірити, чи одна дата передує іншій. В цьому блозі ми розглянемо, як порівнювати дві дати в мові Swift і в чому це може бути корисно.

## Як це зробити

Для порівняння дат в мові Swift ми будемо використовувати клас Date. Цей клас містить усю необхідну інформацію про дати, таку як рік, місяць, день, година, хвилина та секунда. Для порівняння двох дат нам потрібно буде створити об'єкти типу Date за допомогою поданих значень і використовувати метод compare для їх порівняння.

```Swift
// Створення об'єктів типу Date
let firstDate = Date(year: 2021, month: 10, day: 20)
let secondDate = Date(year: 2021, month: 9, day: 15)

// Порівняння дат
let result = firstDate.compare(secondDate)

// Вивід результату
print(result) // Виводиться "orderedDescending", оскільки перша дата більше за другу
```

Як бачимо, результатом порівняння буде один з трьох варіантів: "orderedAscending", "orderedDescending" або "orderedSame". Ці значення вказують, відповідно, на порядок дат (перша дата передує другій), зворотний порядок (перша дата після другої) або рівність дат.

## Глибоке дослідження

Крім порівняння дат за допомогою методу compare, ми також можемо використовувати різні методи для отримання більш детальної інформації про порівняння дат. Наприклад, метод isEqual може використовуватися для перевірки рівності дат, а метод distance перераховує різницю між датами у заданому одиницях часу (наприклад, днях, годинах, хвилинах).

Для отримання більш докладної інформації про клас Date та його методи ви можете переглянути [документацію Apple](https://developer.apple.com/documentation/foundation/date).
