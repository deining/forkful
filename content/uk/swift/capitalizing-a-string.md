## Чому

Існує багато різних ситуацій, коли вам може знадобитися написати код, який би перетворював малі літери великими в тексті. Наприклад, якщо ви хочете вивести ім'я користувача із нижніми регістрами у заголовку сторінки. Ця функція дозволить звичайному тексту виглядати більш формально та професіонально.

## Як

```Swift
let name = "oleh"
let capitalized = name.capitalized

print(capitalized)

// Output: Oleh
```

Щоб привести рядок до великих літер, нам необхідно викликати метод `capitalized` на певному рядку. Цей метод повернеться новий рядок, де всі слова будуть починатися з великої літери, незалежно від того, якими були вони у оригінальному рядку. Щоб отримати капітализований рядок, ми можемо зберегти результат у змінну та вивести її значення у консоль.

## Глибше

У Swift існує багато методів для приведення рядків до різних форматів, включаючи `uppercased`, `lowercased` та `capitalized`. Кожен з цих методів має свої особливості та призначення. Наприклад, метод `lowercased` повертає рядок, де всі літери будуть написані з малими регістрами, тоді як `uppercased` поверне рядок з великими літерами. Є також можливість використання функцій для зміни регістра лише окремих літер, як `uppercasedFirst` та `lowercasedFirst`. Таким чином, можна вибрати оптимальний метод для кожної конкретної задачі.
