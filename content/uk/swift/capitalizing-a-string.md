## Що & Чому?

Написання програм вимагає доброї техніки та чіткого іменування змінних. Один зі способів цього є написання стрічки з великої літери, щоб її було легше розпізнати. Це також допомагає зменшити ризик помилки при програмуванні.

## Як зробити:

```Swift
let string = "привіт друзі"
let capitalizedString = string.capitalized
print(capitalizedString)
```
```
Вивід: Привіт Друзі
```

Щоб змінити стрічку на верхній регістр, ми можемо використовувати метод `capitalized`, який поверне новий рядок з першою літерою кожного слова великою.

## Глибоке занурення:

Існують інші способи для перетворення стрічки на верхній регістр, такі як `uppercased` або використання циклу `for`. Однак, метод `capitalized` надає більш зручний та читабельний підхід до цього завдання. Історично, використання заголовних літер в програмуванні було пов'язано з мовою COBOL в 1950-х роках. Це допомагало програмістам відокремлювати ключові слова від змінних та інших елементів програми.
