# Чому

Перед тим, як почати програмувати, важливо задати собі запитання: "Чому мені потрібна поточна дата?". Незважаючи на те, що на перший погляд може здатися, що ця операція є простою та неважливою, вона є важливою компонентою багатьох програм та додатків. Отже, давайте розглянемо, як ми можемо отримати поточну дату у Swift.

# Як

Для початку, нам потрібно імпортувати фреймворк Foundation у нашому проекті. Для цього використовуємо наступний код:

```Swift
import Foundation
```

Тепер, щоб дізнатися поточну дату, ми можемо використовувати клас ```Date```. Для цього ми можемо використовувати такий код:

```Swift
let currentDate = Date()
```

Це поверне нам об'єкт типу ```Date``` з поточною датою та часом.

# Поглиблене дослідження

Якщо вам потрібно отримати більш детальну інформацію про поточну дату, ви можете скористатися класом ```DateComponents```. Використовуючи його, ми можемо отримати не тільки дату, але і окремі компоненти, такі як день тижня, місяць чи рік. Ось приклад коду:

```Swift
let currentDate = Date()

let calendar = Calendar.current
let dateComponents = calendar.dateComponents([.year, .month, .day, .hour], from: currentDate)

print(dateComponents.year) // Виведе поточний рік
print(dateComponents.month) // Виведе поточний місяць
print(dateComponents.day) // Виведе поточний день
print(dateComponents.hour) // Виведе поточний годину
```

Як бачите, ми можемо отримати більш детальну інформацію про поточну дату, використовуючи клас ```DateComponents```.
