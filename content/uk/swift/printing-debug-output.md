## Що & Чому?

Друкування виводу для налагодження - це процес виведення важливої інформації на екран для з’ясування, як програма працює. Програмісти роблять це, щоб бачити значення змінних та кроки виконання коду для вирішення можливих помилок.

## Як:

```Swift
func printDebugOutput() {
    let name = "John"
    let age = 30
    let job = "developer"
    
    print("Name: \(name)") // Prints "Name: John"
    print("Age: \(age)") // Prints "Age: 30"
    print("Job: \(job)") // Prints "Job: developer"
}

// Виклик функції
printDebugOutput()
```
В результаті виконання даного коду, на екрані з'явиться такий вивід:
```
Name: John
Age: 30
Job: developer
```

## Глибокий погляд:

Друкування виводу для налагодження було дуже популярним у минулому, коли доступність інструментів для видрукування виводу була обмежена. Зараз існують інші методи налагодження, такі як використання відладчика чи логування в консоль. Проте, друкування виводу залишається одним з найпростіших та швидких способів побачити, що відбувається під час виконання програми.

Для виведення рядка на екран використовується функція ```print()```, яка може приймати будь-яку дану та виводити її значення на екран. При виведенні значень змінних, використовується спеціальний символ ```\( )```, який дозволяє вставити значення змінної в середину рядка.
