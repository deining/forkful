# Чому

Друк вихідних даних для налагодження є важливою частиною програмування. Воно дозволяє розробникам перевіряти значення, що передаються в різні частини програми та знайти можливі помилки. 

# Як зробити

Для виведення вихідних даних у Swift використовується функція `print()`. Наприклад, для виведення тексту "Hello, world!" до консолі, потрібно написати наступний код: 

```Swift 
print("Hello, world!")
``` 

Якщо ми бажаємо вивести значення змінної, можна скористатися такою ж функцією. Наприклад, якщо ми маємо змінну `name` зі значенням "Kate", то за допомогою такого коду ми можемо перевірити її значення: 

```Swift 
print(name)
```

У випадку, коли нам потрібно вивести значення декількох змінних одночасно, можна скористатися оператором `+` для їх об'єднання: 

```Swift 
let age = 25 
let city = "Kyiv" 

print("I am " + String(age) + " years old and I live in " + city + ".") 
```

У цьому випадку, до функції `print()` ми передаємо одну строку з об'єднаними значеннями змінних та тексту. Переконайтесь, що всі змінні перетворені до типу `String`, використовуючи функцію `String()`. 

# Profound Dive

Для більш детального налагодження програми, можна використовувати функцію `debugPrint()`. Ця функція виводить не лише значення змінної, а й її ім'я, тип та адресу в пам'яті. Наприклад: 

```Swift 
let number = 5 

debugPrint(number) 
```

Результатом буде: 

```Swift 
number: 5 of type Int at 0x7fd2a4c5ef80 
```

Це дозволяє розробникам більш детально вивчати програму та знаходити помилки.
