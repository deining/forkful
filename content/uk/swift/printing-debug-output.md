#Чому

У програмуванні друк виводу для відлагодження є важливою проступочотю. Він дозволяє розробникам відстежувати значення змінних та стежити за ходом виконання програми, щоб виявити та виправити будь-які помилки.

##Як

Код не завжди виконується так, як очікувалося, тож друк виводу може допомогти виправити проблеми. Для друку використовується функція print(), яка дозволяє передати значення змінних та текстові повідомлення для виводу на консоль. Наприклад:

```Swift
var name = "Oksana"
var age = 25
print("Вітаю, мене звати \(name) та мені \(age) років!")
```

Результатом буде: "Вітаю, мене звати Oksana та мені 25 років!"

Також, можливо використовувати команду debugPrint() для отримання більш детальної інформації про значення змінних.

##Глибоке дослідження

Друк виводу може бути корисним не тільки для пошуку та виправлення помилок, але й для вивчення поведінки програми. Наприклад, можна встановити точки виводу (breakpoints) в коді та переглянути значення змінних на різних етапах виконання. Також існують інструменти для візуалізації даних, які дозволяють побачити зміну значень змінних у реальному часі.
