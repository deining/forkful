## Чому
Знайти довжину рядка є важливим елементом для багатьох програмувальників, оскільки це дозволяє їм працювати з текстом та визначати його межі.

## Як
Знаходження довжини рядка в Swift дуже просте завдяки вбудованому властивості "count". Для того, щоб дізнатися довжину рядка, просто використовуйте наступний код:

```Swift
let myString = "Привіт світе!"
let length = myString.count
print(length)
// Вивід: 12 
```

Це просто 1 із багатьох способів отримати довжину рядка. Ви також можете використовувати функцію "countElements", яка знаходить кількість символів у рядку, або звертатися до властивості "characters" та використовувати функцію "count".

```Swift
let myString = "Привіт світе!"
let count = myString.characters.count
print(count)
// Вивід: 12 
```

## Глибоке дослідження
Хоча знаходження довжини рядка майже просте завдяки властивостям Swift, корисно знати, що саме відбувається під час цього процесу. Для рядка Swift зберігається доречна інформація про його довжину та ще додаткова інформація, яка допомагає працювати з рядком у програмі. Коли ми використовуємо властивість "count", Swift швидко сканує рядок та використовує цю дорожню карту для того, щоб обрати бажану довжину. 

Також, варто зазначити, що довжина рядка у Swift не завжди відповідає кількості символів. Наприклад, деякі символи можуть бути багатобайтовими, тому його довжина буде більшою, ніж кількість видимих символів.
