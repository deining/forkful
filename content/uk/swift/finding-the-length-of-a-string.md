# Що це і чому?
Функція знаходження довжини рядка - це спосіб дізнатися, скільки символів міститься в рядку. Це корисна операція для програмістів, оскільки дозволяє визначити довжину тексту та працювати з ним відповідно.

# Як це зробити:
Використання функції `count` у Swift дозволяє легко знайти довжину рядка. Наприклад:
```Swift
let text = "Привіт, світ!"
print(text.count)
```
Вивід: `13`

# Глибокий занурення:
У минулому, програмісти повинні були самі обчислювати довжину рядка, використовуючи різні методи, наприклад підрахунок кількості символів у рядку. Однак, функція `count` з'явилася у Swift 4, що спростив цей процес для програмістів. Додаткові альтернативи для знаходження довжини рядка можуть включати використання циклів та інших методів рядків у Swift. Детальніше про це можна дізнатися за посиланням у розділі "Дивись також".
