## Чому

У програмуванні часто виникає необхідність визначати довжину рядка. Це може бути корисно для перевірки валідності введених даних або для обробки даних у функціях. У цій статті ми розглянемо як знаходити довжину рядка у мові Swift.

## Як

Для знаходження довжини рядка у Swift використовується метод `count`. Для цього ви повинні передати рядок до даного методу. Давайте розглянемо пару прикладів:

```Swift
let string = "Привіт, світ!"
print(string.count) // виведе 13, оскільки рядок містить 13 символів
```

```Swift
let emptyString = ""
print(emptyString.count) // виведе 0, оскільки це пустий рядок
```

Нагадаємо, що пробіли також враховуються при підрахунку довжини рядка.

## Глибоке дослідження

Для тих, хто цікавиться більш детальною інформацією, дослідимо як саме працює метод `count` в Swift. Перш за все, щоб зрозуміти, як він підраховує довжину рядка, потрібно знати, що в основі Swift лежить мова програмування Objective-C. Отже, в Swift присутній метод `count` використовує метод `countByEnumeratingWithState` мови Objective-C, який повертає кількість елементів у колекції. При цьому у Swift рядок представлений у вигляді колекції символів, тому метод `count` просто повертає кількість цих символів у рядку.
