## Чому

У програмуванні іноді потрібно видаляти символи, що відповідають певному шаблону, наприклад, для очищення даних або заміни їх на інший символ. В цій статті ми розповімо, як це можна зробити за допомогою Bash.

## Як це зробити

Для видалення символів, що відповідають певному шаблону, використовується команда `sed`. Нижче наведені 2 приклади.

```Bash
sed 's/[0-9]//g' data.txt
```

Ця команда видалить усі цифри з тексту, що міститься у файлі `data.txt`.

```Bash
sed 's/[aeiou]//g' data.txt
```

Ця команда видалить усі голосні з тексту.

Результат виконання команди буде виведений на екран або може бути збережений у новому файлі за допомогою оператора перенаправлення виводу `>`. Наприклад, `sed 's/[0-9]//g' data.txt > output.txt` збереже результат у файлі `output.txt`.

## Глибока занурення

Команда `sed` дає велику кількість можливостей для видалення символів за певним шаблоном. Наприклад, у вище наведених прикладах ми використовували шаблон в квадратних дужках, однак можна використовувати й інші символи для позначення шаблону. Крім того, `sed` має багато опцій для більш точного вибору символів, що будуть видалені.
