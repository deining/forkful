## Що та чому?

Запуск нового проекту - це перший крок, який розпочинається до написання коду. Це процес створення нового додатку або програми, що вимагає розробки і організації коду для досягнення поставленої мети.

Програмісти починають нові проекти, оскільки вони потребують програмного забезпечення, яке не існує або не влаштовує їх в наявному вигляді. Це також може бути пов'язано з покращенням наявного коду або розробкою нових функцій.

## Як це зробити:

Найпростішим способом запустити новий проект є використання команди `mkdir` для створення нової директорії і `touch` для створення нових файлів. Наприклад:

```Bash
mkdir my_project
cd my_project
touch main.py
```

Цей код створить папку з назвою `my_project` та файл з назвою `main.py`.

Також можна використовувати шаблони або скрипти для створення вже готової структури проекту. Наприклад, `cookiecutter` - це популярний інструмент для швидкого створення нового проекту з вже налаштованими файлами та папками.

## Глибше занурення:

Запуск нового проекту може бути пов'язаним з історичними подіями, такими як пов'язані з програмуванням технології або дією ринку. Також можна використовувати альтернативні підходи для запуску нового проекту, такі як використання інших мов програмування або фреймворків.

Ще одним важливим аспектом є вибір оптимальної структури проекту та організація коду для забезпечення продуктивності та легкості змін. Це може бути досягнуто за допомогою використання патернів дизайну та правильної організації файлів та папок.
