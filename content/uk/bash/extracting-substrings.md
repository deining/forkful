# З чого почати

Цей пост буде корисним для тих, хто хоче вивчити як витягувати підрядки за допомогою Bash програмування. Якщо ви займаєтесь автоматизацією задач на вашому комп'ютері або сервері, ви, напевно, захочете знати, як витягувати певні частини тексту з файлів або рядків.

# Як

Для початку, давайте розглянемо основну команду, яка використовується для витягування підрядків в Bash.

```Bash
# Використання команди cut для витягування підрядка з рядка
cut -d ":" -f 2 file.txt
```

У цьому прикладі, команда `cut` витягує підрядок з рядка `file.txt`. Параметр `-d` визначає роздільник, а `-f` визначає номер стовпця, який ви хочете витягнути. В нашому випадку, ми використовуємо `:` як роздільник і витягуємо другу частину тексту з рядка.

Тепер давайте розглянемо ще один приклад використання команди `substr` для витягування підрядків.

```Bash
# Використання substr для витягування підрядка зі змінної
my_var="Привіт, світ!"
echo ${my_var:0:6}
```

У цьому прикладі, ми використовуємо вбудовану функцію `substr` для витягування підрядка зі змінної `my_var`. Параметр `0` визначає початковий індекс підрядка, а `6` визначає його довжину. В результаті, ми отримуємо `Привіт` як підрядок зі змінної `my_var`.

# Глибші дослідження

Хоча ці два приклади показують основи витягування підрядків, існують й інші методи для досягнення того ж самого результату. Наприклад, ви можете використовувати регулярні вирази з командами такими як `grep` або `sed` для витягування підрядків. Крім того, ви можете використовувати вбудовані функції Bash, такі як `substr` або `substring` для більшої гнучкості та контролю над вашими витягуваннями.
