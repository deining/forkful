## Тому Чому 
Існує багато ситуацій, коли вам може знадобитися виділити певну частину тексту або рядка у вашому Bash скрипті. Наприклад, ви можете бажати отримати лише ім'я користувача з його повного імені та прізвища, або виділити лише дату з повного часу. Для цього використовуються різні методи, одним з яких є використання функції вилучення підрядка в Bash.

## Як це зробити
Для виконання вилучення підрядка ви повинні використовувати команду "cut" у вашому Bash скрипті. Нижче показані приклади використання цієї команди з різними параметрами.

```Bash 
# Приклад 1 - отримання підрядка з позиції 3 до кінця рядка
text="Привіт світ!"
cut -c 3- <<< $text
# Вивід: іт світ!

# Приклад 2 - отримання підрядка з позиції 2 із загальною довжиною 4 символи
text="abcdefg"
cut -c 2-5 <<< $text
# Вивід: bcde

# Приклад 3 - отримання підрядка до символу "!"
text="Привіт всім!"
cut -d"!" -f1 <<< $text
# Вивід: Привіт всім
```

## Глибокий занурення 
Команда "cut" має багато параметрів, які можна використовувати для різноманітного виділення підрядків. Наприклад, ви можете використовувати параметр "-f" для вказівки номера поля, або "-s" для пропуску рядків, в яких немає символу, який ви шукаєте. Також, команда "cut" підтримує використання регулярних виразів, що дозволяє вам більш гнучко виконувати вилучення підрядків у вашому скрипті.
