## Що і для чого?

Витягування підрядків - це процес отримання певної частини рядка або тексту. Це корисний інструмент для програмістів, який дозволяє працювати з більшими об'ємами даних та швидше знаходити потрібну інформацію.

## Як це зробити?

Існує кілька способів витягування підрядків у Bash. Давайте розглянемо кожен з них окремо за допомогою прикладів коду та виведення результатів.

```Bash
# Витягування всього рядка
text="Привіт, світ!"
echo ${text}

# Витягування частини рядка по індексу
echo ${text:6}

# Витягування підрядка за допомогою позиції початку та довжини
echo ${text:6:3}

# Витягування підрядка з кінця рядка
echo ${text: -4}

# Витягування підрядка за допомогою шаблону (регулярний вираз)
pattern="Привіт"
if [[ $text =~ $pattern ]]; then
    echo ${BASH_REMATCH[0]}
fi
```

Виведення результатів:

```
Привіт, світ!
світ!
сві
світ!
Привіт
```

## Глибоке занурення

- Історичний контекст: витягування підрядків є основною функцією у більшості мов програмування та утиліт командного рядка, включаючи Bash.

- Альтернативи: окрім розглянутих вище методів витягування підрядків, існує багато інших варіантів, таких як використання команди `awk` або функції `cut` з допомогою каналів.

- Детальніше про реалізацію: у Bash використовується синтаксис `${variable: offset: length}` для витягування підрядків. Це відрізок змінної, який починається з позиції `offset` та має довжину `length`.
