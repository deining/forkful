## Чому

Відображення змісту рядків є важливим аспектом у програмуванні Bash і дозволяє поєднувати різні значення для створення потрібного повідомлення або виводу. Це дозволяє програмі поводитися більш гнучко та людям поліпшити зрозумілість виводу.

## Як

Для з'єднання рядків у Bash, ми використовуємо оператор `+` або просто не пишемо жодного оператора між рядками. Ось приклад:

```Bash
# З'єднаємо два рядки
$ full_greeting="Привіт, " + "Дмитро!"
$ echo $full_greeting
Привіт, Дмитро!

# Якщо ми не використовуємо жодного оператора, рядки автоматично з'єднюються
$ full_name="Марія" "Коваленко"
$ echo $full_name
МаріяКоваленко
```

## Глибокий погляд

Конкатенація рядків в Bash може бути використана не тільки для з'єднання двох рядків, але і для більш складних виразів. Наприклад, ми можемо використовувати змінну, щоб утворити потрібне повідомлення:

```Bash
# Створимо привітання з іменем користувача
$ user_name="Віктор"
$ greeting="Привіт, " $user_name"!" 
$ echo $greeting
Привіт, Віктор!

# Можемо також використовувати результати команд для конкатенації
$ disk_space="Залишилось ми скачати `du -h file.txt` гігабайт"
$ echo $disk_space
Залишилось ми скачати 500 МБ гігабайт
```
