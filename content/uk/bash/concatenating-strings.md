## Чому
Конкатенація рядків є корисним інструментом для об'єднання тексту в коді Bash. Вона дозволяє з'єднати кілька рядків в один, щоб створити більш складну та зручну структуру для роботи зі змінними та виводу даних.

## Як
Конкатенація рядків в Bash може бути здійснена за допомогою оператора `+` або за допомогою команди `echo`. Наприклад:

```Bash
# З'єднання двох рядків за допомогою оператора +
string1="Привіт, "
string2="світ!"
result=$string1$string2
echo $result

# Результат: Привіт, світ!
```

Також можна додавати значення змінних до рядків. Наприклад:

```Bash
# Додавання значення змінної до рядка
name="Оля"
result="Привіт, $name"
echo $result

# Результат: Привіт, Оля
```

Іншим способом є використання команди `cat` для об'єднання декількох рядків у один файл. Наприклад:

```Bash
# З'єднання кількох рядків та їх виведення у файл
cat << EOF > hello.txt
Привіт,
світ!
EOF

# Вміст файлу hello.txt - "Привіт, світ!"
```

## Глибше
Конкатенація рядків також може бути корисною для створення більш складних структур даних, таких як URL-адреси або запити до веб-серверів. Використовуючи цей метод, можна зручно керувати змінною тривалістю рядків та додавати та видаляти їх, не змінюючи основний код.

Також важливо враховувати, що під час з'єднання рядків можуть виникати проблеми зі спеціальними символами. Для уникнення цього, використовуйте потрібні екранування, такі як `\` або `"` залежно від вимог вашої програми.
