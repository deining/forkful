## Чому

Робота з форматом JSON є важливою навичкою для будь-якого програміста. JSON використовується для зберігання та передавання даних, що робить його незамінним у веб-розробці та в роботі з API. Вивчення роботи з JSON дозволить вам передавати та отримувати дані ефективніше та зручніше.

## Як

Для початку роботи з JSON потрібно знати, як створювати та читати дані. Розглянемо приклад створення об'єкту JSON за допомогою Bash:

```Bash
# Створення об'єкту JSON
object='{"name": "John", "age": 25, "city": "Kyiv"}'

# Виведення значень з об'єкту JSON
echo "Ім'я: $(echo $object | jq -r '.name')"
echo "Вік: $(echo $object | jq -r '.age')"
echo "Місто: $(echo $object | jq -r '.city')"
```

Отримаємо наступний вивід:

```Bash
Ім'я: John
Вік: 25
Місто: Kyiv
```

Також можна створювати масиви JSON та отримувати дані з них. Наприклад, створимо масив зі списком популярних кольорів та виведемо його вміст:

```Bash
# Створення масиву JSON
colors='["red", "blue", "green", "yellow"]'

# Виведення значень з масиву JSON
echo "Список кольорів:"
echo $(echo "$colors" | jq -c '.[]')
```

Отримаємо наступний вивід:

```Bash
Список кольорів:
"red", "blue", "green", "yellow"
```

## Глибоке занурення

Після того, як ви освоїли основи роботи з JSON, можете докладніше дослідити цей формат та його можливості. Наприклад, ви можете вчитися парсити та зберігати дані з складніших об'єктів та масивів, використовувати умови та функції для обробки даних, а також включати JSON в скрипти для автоматизації робочих процесів.

Також варто ознайомитися з різними інструментами для роботи з JSON, такими як бібліотеки для різних мов програмування та онлайн сервіси для перетворення та валідації JSON даних. Це дозволить вам більш ефективно та швидко працювати з цим форматом даних.
