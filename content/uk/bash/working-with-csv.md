## Чому

Робота з CSV файлами є важливою навичкою у сучасному світі, особливо для тих, хто працює з даними. Цей формат даних дозволяє ефективно зберігати, обробляти та обмінюватися величезною кількістю інформацією. Також, робота з CSV є необхідною у багатьох програмах та сервісах, тому вміння зчитувати та записувати дані у цьому форматі може бути корисним для будь-якого IT спеціаліста.

## Як

Для роботи з CSV файлами у Bash використовується команда `csvtool`. Ця команда дозволяє зчитувати та записувати дані у форматі CSV. Наведемо приклад зчитування даних з CSV файлу та їх виведення на екран:

```Bash
#!/bin/bash
# Приклад зчитування та виведення даних з CSV файлу

# Зчитуємо дані з CSV файлу у змінну data
data=$(csvtool -t ',' col 1 data.csv)

# Виводимо дані на екран
echo $data
```

Отриманий результат буде виглядати наступним чином:

```Bash
name1
name2
name3
```

Також, за допомогою команди `csvtool` можна виконувати різноманітні операції з даними, такі як сортування, фільтрація та об'єднання.

## Глибоке дослідження

Для більш складних операцій з CSV файлами можна використовувати Bash скрипти з використанням регулярних виразів та інших інструментів. Також, можна вивчити стандарт CSV формату та його особливості, щоб зміцнити свої знання та робити більш ефективну роботу з даними.
