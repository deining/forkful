## Чому?

Регулярні вирази є потужним інструментом у програмуванні на Bash. Вони дозволяють здійснювати швидкий і точний пошук та заміну тексту в рядках. Використання регулярних виразів може значно скоротити час виконання скриптів та полегшити роботу з текстовими даними.

## Як використовувати регулярні вирази в Bash?

Для початку, необхідно використовувати команду `grep` для пошуку тексту за допомогою регулярних виразів. Наприклад, якщо ви хочете знайти всі слова, які починаються з літери "a" у ваших документах, ви можете виконати команду `grep '^a' *`. Цей вираз буде шукати всі рядки, які починаються з літери "a" у всіх файлів поточної директорії.

```Bash
# Створимо файл з даними
echo "apple, avocado, banana" > fruits.txt

# Використовуємо регулярний вираз, щоб знайти всі слова, які починаються з "a"
grep '^a' fruits.txt

# Виводить: apple, avocado
```

Крім того, регулярні вирази можна використовувати для заміни тексту. Для цього використовується команда `sed`. Наприклад, якщо ви хочете замінити слово "авокадо" на "ананас" у файлі `fruits.txt`, можна виконати таку команду: `sed -i 's/авокадо/ананас/g' fruits.txt`. Цей вираз замінить всі входження слова "авокадо" на "ананас" у файлі `fruits.txt`.

```Bash
# Виконуємо заміну за допомогою регулярного виразу
sed -i 's/авокадо/ананас/g' fruits.txt

# Перевіряємо вміст файлу
cat fruits.txt

# Виводить: apple, ананас, banana
```

## Глибока занурення

У додатку до простих прикладів, описаних вище, регулярні вирази в Bash також дозволяють використовувати метасимволи, щоб знайти більш складні шаблони. Наприклад, при пошуку імен файлів, які містять у собі дату та час у форматі "рік-місяць-день-години", можна використовувати такий регулярний вираз: `[0-9]{4}-[0-9]{2}-[0-9]{2}-[0-9]{2}`.

Крім того, ви можете використовувати інші опції команди `grep` та `sed`, щоб зробити пошук та заміну ще більш продуктивними.
