## Why

Є багато ситуацій, коли нам потрібно порівняти дві дати в програмі, наприклад, для визначення різниці між двома подіями або перевірки умови на часовий проміжок. У цих випадках корисно знайти спосіб, як швидко та ефективно порівняти дати в Bash.

## How To

Кодування для порівняння дат в Bash може бути трохи заплутаним, тому давайте розглянемо кілька прикладів. Спочатку нам потрібно визначити дві дати, які нам потрібно порівняти:

```Bash
date1="2021-03-01"
date2="2021-04-15"
```

Тепер, коли ми маємо наші дати, ми можемо використовувати функцію `date` в Bash для перетворення їх в секунди для подальшого порівняння:

```Bash
date1=$(date -d "$date1" +"%s")
date2=$(date -d "$date2" +"%s")
```

Далі ми можемо використовувати оператори порівняння, такі як `<` або `>`, для визначення, яка дата є ранішою або пізнішою:

```Bash
if [ "$date1" -gt "$date2" ]; then
    echo "$date1 is later than $date2"
elif [ "$date1" -lt "$date2" ]; then
    echo "$date1 is earlier than $date2"
else
    echo "$date1 is the same as $date2"
fi
```

Ви також можете використовувати функцію `test` для того, щоб визначити, чи дві дати знаходяться в певному проміжку часу:

```Bash
if [ "$date1" -ge "$date2" ] && [ "$date1" -le "$date2" ]; then
    echo "$date1 is between $date2"
fi
```

Також варто зазначити, що ці приклади працюють лише з датами в форматі `YYYY-MM-DD`. Якщо вам потрібно порівнювати більш складні формати дат, вам можуть знадобитися додаткові операції з датами, такі як `date +%j`, які повертають числове представлення дати, що може бути використане для порівняння.
