## Чому

Конвертування дати у рядок – це корисна навичка, яка може бути дуже корисна для автоматизації робочих процесів або для створення зручних звітів із даних за допомогою командного рядка. Також це може бути корисно під час написання скриптів для перевірки та оформлення дат у багатьох різних форматах.

## Як

Найпростіший спосіб конвертувати дату у рядок у Bash – використання команди `date` зі специфікатором формату. Наприклад, для отримання поточної дати у форматі `ДД-ММ-РРРР` треба виконати наступну команду:

```Bash
date +"%d-%m-%Y"
```

Отриманий результат буде виглядати, наприклад, так: `28-06-2021`.

Існує багато різних специфікаторів формату, які можна використовувати для конвертації дати у різні формати. Наприклад, для виведення дати і часу у форматі `РРРР-ММ-ДД ГЦМЧ` можна використати команду:

```Bash
date +"%Y-%m-%d %H:%M:%S %Z"
```

Якщо необхідно отримати дату у користувачом вказаному форматі, то це теж можна зробити за допомогою команди `date`, вказавши потрібний специфікатор формату та задаючи дату за допомогою опції `-d`. Наприклад, щоб отримати дату у форматі `ДД MMM РРРР`, де MMM – скорочене найменування місяця, потрібно виконати наступну команду:

```Bash
date +"%d %b %Y" -d "2021-06-28"
```

Отриманий результат буде виглядати так: `28 Jun 2021`.

## Deep Dive

Навіщо потрібно так багато різних форматів дати? Якщо коротко, то кожний з них має свої плюси та мінуси. Наприклад, якщо ви працюєте з даними у форматі JSON, то можливо найкращим варіантом буде використання формату `РРРР-ММ-ДД`. В разі, якщо результат вам потрібен для подальшої обробки унікальними скриптами або програмами, краще вибрати дату у форматі Unix часу, який є числовим значенням, що показує кількість секунд, що минули з початку 1970 року. Для отримання цього значення достатньо використати команду `date +%s`.
