## Чому

Робота з командною оболонкою Bash може здатися складною та незрозумілою для багатьох користувачів, але пошук та заміна тексту може значно полегшити роботу з файлами та скриптами. В даній статті ми розглянемо простий метод пошуку та заміни тексту за допомогою командної оболонки Bash та дізнаємося, як це може заощадити час та зусилля.

## Як це зробити

Для пошуку та заміни тексту використовуються дві команди - `grep` та `sed`. Перша команда (`grep`) використовується для пошуку тексту у вказаному файлі, а друга команда (`sed`) використовується для заміни цього тексту.

Для початку створимо файл `example.txt` з довільним текстом:

```
Вітаємо усіх українців з Днем Незалежності!
```

Тепер давайте використаємо команду `grep` для пошуку слова "Незалежності" у нашому файлі:

```
grep "Незалежності" example.txt
```

В результаті ми отримаємо наступне:

```
Вітаємо усіх українців з Днем Незалежності!
```

Тепер давайте замінимо слово "Незалежності" на "Незалежності України" за допомогою команди `sed`:

```
sed -i 's/Незалежності/Незалежності України/' example.txt
```

Параметр `-i` дозволяє змінити файл `example.txt` без створення копії. В результаті ми отримаємо наступне:

```
Вітаємо усіх українців з Днем Незалежності України!
```

## Огляд

Пошук та заміна тексту є корисною та незамінною функцією в командній оболонці Bash. Цей метод є дуже ефективним, коли потрібно швидко змінити багато файлів за певною шаблоном. Крім `grep` та `sed`, існують інші команди для пошуку та заміни тексту в Bash, такі як `awk`, `tr` та інші. Більш детальну інформацію можна знайти у розділі допомоги цих команд - `man` (наприклад, `man grep`).
