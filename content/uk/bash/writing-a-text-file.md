## Що це і навіщо?
Написання текстового файлу - це процес створення файлу, який містить текстову інформацію, за допомогою команд інтерфейсу командного рядка (CLI). Це важливий процес для програмістів, оскільки дозволяє зберігати та обробляти великі обсяги текстової інформації, що становить основу програмування.

## Як?
Нижче наведені приклади коду та виведення результатів для написання текстового файлу за допомогою команди ```Bash ... ```.

Створення нового текстового файлу:
```
Bash touch example.txt
```
Видалення існуючого текстового файлу:
```
Bash rm example.txt
```
Додавання тексту до існуючого файлу:
```
Bash echo "Додатковий текст" >> example.txt
```
Перегляд вмісту файлу:
```
Bash cat example.txt
```

## Глибоке занурення
Написання текстового файлу є необхідною складовою програмування. Зараз це один з найдавніших методів збереження інформації. До появи графічного інтерфейсу користувача (GUI), програмістам доводилося використовувати лише інтерфейс командного рядка для написання та збереження файлів. Існує багато альтернативних шляхів написання текстових файлів, таких як використання програм для редагування, але написання через CLI залишається надійним та ефективним засобом.
