## З чого почати
В програмуванні Bash багато різноманітних і корисних інструментів, одним з яких є можливість зберігати дані у вигляді текстового файлу. При цьому файл можна буде використовувати для подальшої обробки або передачі до іншої програми. В цій статті ми розглянемо, як зробити це за допомогою Bash та які інші можливості є у роботі з текстовими файлами.

## Як це зробити
Для створення текстового файлу в Bash ви можете скористатися командою `touch`, яка створить пустий файл з заданою назвою. Наприклад, так: `touch example.txt`. Щоб додати до файлу текст, ви можете використати команду `echo`, наприклад `echo "Hello World!" > example.txt`.

```Bash
touch example.txt
echo "Hello World!" > example.txt
```

Для зчитування та обробки вмісту текстового файлу використовуйте команду `cat`, яка виведе вміст файлу в терміналі. Наприклад, `cat example.txt`.

```Bash
cat example.txt
```

## Розкопки на глибину
Для більш складних операцій з файлом, наприклад, додавання та видалення текстових рядків або пошук необхідних даних, ви можете скористатися командою `sed`. Ця утиліта дозволяє редагувати текстові файли, використовуючи регулярні вирази.

```Bash
sed -i 's/Hello/Hi/g' example.txt
```

У даному прикладі ми замінимо слово "Hello" на "Hi" у файлі `example.txt`. Практичне використання команди `sed` дозволяє швидко та зручно редагувати та обробляти дані у текстових файлах.
