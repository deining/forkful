## Чому

Написання текстового файлу є важливою частиною програмування в сучасному світі. Він дозволяє зберігати дані, підготувати змінні для використання та розповсюдження програмного забезпечення.

## Як це зробити

Текстові файли можуть бути створені використовуючи Bash за допомогою команди "touch". Нижче наведені приклади коду та виведення, які можна додати до вашого скрипту.

```Bash 
# Створення текстового файлу з назвою "test.txt"
touch test.txt 

# Додавання тексту до файлу
echo "Це приклад тексту" >> test.txt 

# Виведення вмісту текстового файлу
cat test.txt 
```

## Глибше входження

Написання текстового файлу вимагає знань основних команд та синтаксису Bash. Для створення файлів з більш складною структурою, ви можете використовувати оператори циклів та умовних операторів. Також важливо використовувати правильні пробіли і розділові знаки для дотримання форматування файлу.
