## Чому

Програмування на Bash - це потужний інструмент, який дозволяє робити автоматизацію процесів на вашому комп'ютері. Інколи однак, вам може знадобитися написати текстовий файл у своїй програмі. Сьогодні ми поговоримо про те, як це зробити.

## Як це зробити

Написання текстового файлу на Bash доволі просте. Вам потрібно використовувати команду `echo` та перенаправляти її вивід у файл за допомогою символу `>`. Наприклад, якщо ви хочете створити файл зі списком імен користувачів, ви можете скористатися таким кодом:

```Bash
echo "Іван
Марія
Петро" > users.txt
```

Це створить новий файл з іменами у вигляді списка. Якщо ви хочете додати нові імена до файлу, використовуйте символ `>>` замість `>`, щоб дописати в кінець файлу. Також ви можете використовувати змінні для збереження тексту, який потрібно записати у файл.

Наприклад, якщо у вас є змінна `name`, яка містить ім'я користувача, ви можете використати такий код:

```Bash
echo "$name" >> users.txt
```

Це дозволить додати нове ім'я до списку у файлі. Крім того, ви можете також створювати та редагувати файл за допомогою команди `cat`.

```Bash
cat > new_file.txt
```

Ця команда дозволяє вам ввести текст і зберегти його у вказаному файлі. Щоб закінчити ввід, натисніть `Ctrl + D`.

## Глибокий занурення

У попередньому розділі, ми розглянули основи написання текстового файлу на Bash. Однак, ви можете використовувати більш складні команди та оператори для створення та редагування файлів. Наприклад, ви можете використовувати `sed` для редагування вмісту файлу або `grep` для пошуку конкретного тексту.

Ви також можете зберігати не тільки текстові дані у файлах, але й інші типи даних, такі як каталоги, схеми та керуючі файли. Це дозволяє створювати більш складні скрипти та автоматично керувати різноманітними процесами.
