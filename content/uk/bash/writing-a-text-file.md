##Чому 
Написання текстового файлу є важливою частиною написання програм. Воно дозволяє вам зберігати, організовувати та пере віряти важливі дані для вашої програми.

##Як
Написання текстового файлу в Bash дуже просте. Використовуючи команду `echo`, ви можете додати текст до файлу або створити новий файл. Наприклад:

```Bash
echo "Привіт світ!" > hello.txt
```

Ця команда створить новий файл з назвою `hello.txt` та додасть рядок "Привіт світ!" до файлу.

Крім того, ви можете використовувати функцію `cat`, щоб додати більше тексту до файлу. Наприклад:

```Bash
cat >> hello.txt
```

Ця команда дозволяє вам введе текст у терміналі, який буде прямо доданий до файлу `hello.txt`.

Щоб переглянути вміст текстового файлу, ви можете використовувати команду `cat` або `less`. Наприклад:

```Bash
cat hello.txt
```

##Глибока погрузка
Крім створення та додавання тексту до файлу, використання текстових файлів в Bash може бути корисним для багатьох інших цілей, таких як збереження змінних серед різних сеансів в терміналі або перенаправлення вихідних даних вашої програми у файл.

Ви також можете використовувати команду `grep`, щоб шукати конкретний текст у вашому файлі або команда `sed`, щоб змінювати текст у вашому файлі.
