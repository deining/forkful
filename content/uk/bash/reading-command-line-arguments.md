## Чому

Стаття містить корисну інформацію для новачків, які хочуть навчитися читати аргументи командного рядка у Bash. Читається легко та швидко.

## Як

Щоб читати аргументи командного рядка в Bash, використовуйте спеціальні змінні `$1`, `$2`, `$3` та інші в залежності від кількості аргументів. 

```Bash
#!/bin/bash
# Приклад читання першого аргумента
# Виклик: sh script.sh argument
echo "Перший аргумент: $1" 
```

Вивід:
```
Перший аргумент: argument
```

За допомогою змінної `$#` ви можете отримати загальну кількість аргументів командного рядка. А змінна `$0` містить назву самого скрипта.

```Bash
#!/bin/bash
# Приклад читання кількості аргументів
# Виклик: sh script.sh argument1 argument2 argument3
echo "Кількість аргументів: $#"
echo "Назва скрипта: $0"
```

Вивід:
```
Кількість аргументів: 3
Назва скрипта: script.sh
```

## Глибоке поглиблене вивчення

Ви можете також використовувати прапорці аргументів, які покращують можливості читання. Наприклад, прапорець `-p` дозволяє вимагати введення значення для аргумента, якщо він не був переданий у командному рядку.

```Bash
#!/bin/bash
# Приклад читання аргумента з прапорцем
# Виклик: sh script.sh -p argument
while getopts ":p:" arg; do
  case $arg in
    p) # введіть значення для a, якщо не передано у командному рядку
      argument=$OPTARG
      ;;
  esac
done
echo "Аргумент: $argument" 
```

Виводить:
```
Аргумент: argument
```

Щоб вирізати складні аргументи з командного рядка, використовуйте команду `cut` і вказуйте роздільник знаку `=`.

```Bash
#!/bin/bash
# Приклад вирізання аргумента з прапорцем
# Виклик: sh script.sh -p argument
while getopts ":p:" arg; do
  case $arg in
    p) # введіть значення для a, якщо не передано у командному рядку
      argument=$OPTARG
      ;;
  esac
done
complex_arg=$(echo "$argument" | cut -d'=' -f2)
echo "Складний аргумент: $complex_arg" 
```

Вивід:
```
Складний аргумент: argument
```
