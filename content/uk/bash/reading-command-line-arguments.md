## Для чого

Якщо ви особливо зацікавлені в програмуванні і вже знайомі з Bash, ви, напевно, знаєте про аргументи командного рядка. Але якщо ви тільки починаєте вивчати цю мову, то вам буде дуже корисно ознайомитися з цією функцією. Чому? Тому що здебільшого практично кожна Bash-команда приймає аргументи командного рядка. Це означає, що ви зможете подавати вхідні дані відразу при запуску програми, що значно спростить вашу роботу.

## Як це зробити

У Bash для отримання аргументів командного рядка ви можете використовувати глобальну змінну `$@` або вбудований цикл `for`. Давайте подивимось на кожен з цих методів.

### Використання змінної `$@`

Перш за все, давайте створимо файл з нашим кодом. Нехай це буде файл з розширенням `.sh`. У ньому ми можемо оголосити наші змінні, які будуть містити аргументи командного рядка.

```Bash
#!/bin/bash

echo "Перший аргумент: $1"
echo "Другий аргумент: $2"
echo "Третій аргумент: $3"
```

В даному прикладі ми виводимо на екран перші три аргументи командного рядка, але ви можете зчитувати скільки завгодно аргументів, використовуючи цю ж техніку.

Тепер, запустивши цей файл з декількома аргументами, ми отримаємо наступний результат:

```Bash
$ ./my_script.sh argument1 argument2 argument3

Перший аргумент: argument1
Другий аргумент: argument2
Третій аргумент: argument3
```

### Використання циклу `for`

Крім використання глобальної змінної, ми також можемо зчитувати аргументи командного рядка з використанням циклу `for`.

```Bash
#!/bin/bash

for arg in "$@"
do
  echo "Аргумент: $arg"
done
```

Цей приклад виводить на екран усі аргументи, передані при запуску нашого файла.

## Глибоке занурення

Тепер, коли ми знаємо, як отримати аргументи командного рядка, давайте поглянемо на деякі особливі випадки.

### Переадресація вхідного потоку

У деяких випадках ви можете хотіти передавати вхідні дані у вашу програму не через аргументи командного рядка, а через переадресацію вхідного потоку. Для цього ви можете використовувати оператор `<` для передачі вмісту файлу як вхідних даних до вашої
