Що & Чому?

Читання аргументів командного рядка - це процес використання аргументів, які передаються до програми при її запуску. Це дозволяє програмі отримати вхідні дані від користувача та виконувати певні дії згідно з ними. Програмісти використовують цей процес, щоб зберігати програмний код більш гнучко та динамічно.

Як це зробити:

```
Bash ...
# Приклад читання першого аргументу командного рядка
echo "Перший аргумент: $1"
```

```
# Введення: bash script.sh Вітання!
# Вихід: Перший аргумент: Вітання!
```

Глибоке погруження:

- Історичний контекст: Читання аргументів командного рядка було введено в UNIX в 1978 році. Вони дозволяють користувачам взаємодіяти з програмами та передавати їм дані безпосередньо під час їх запуску.

- Альтернативи: Є інші мови програмування, що дозволяють читати аргументи командного рядка, такі як Perl, Python та Java. Але Bash є стандартним і широко використовуваним в UNIX-системах.

- Деталі реалізації: В Bash, аргументи командного рядка зберігаються в спеціальній змінній під назвою $0 для першого аргументу, $1 для другого і т.д. Ці змінні можуть бути використані в програмному коді.
