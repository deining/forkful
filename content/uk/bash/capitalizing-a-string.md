## Що та чому?

Капіталізація рядків - це процес перетворення першої літери великої літери в кожному слові у рядку. Програмісти часто роблять це для поліпшення зрозумілості і читабельності коду.

## Як це зробити:

```Bash
# Використовуємо команду tr для капіталізації рядка
echo "привіт світ" | tr [:lower:] [:upper:]
# Результат: ПРИВІТ СВІТ
```

```Bash
# Використовуємо вбудовану функцію Bash для капіталізації рядка
str="привіт світ"
echo ${str^}
# Результат: Привіт світ
```

## Глибока занурення:

Капіталізація рядків була популярна ще з часів стародавніх машин, коли регістри були дуже обмежені і потрібно було мінімізувати використання пам'яті. Методи реалізації зміни регістру також варіюються залежно від платформи.

Існують також інші способи капіталізувати рядки, наприклад, використання різних функцій мови, таких як awk і sed. Проте, важливо вибрати правильний метод для вашого випадку, оскільки це може вплинути на продуктивність коду.
