## Чому

Капіталізація рядка є важливою частиною розробки програмних продуктів, оскільки вона допомагає зробити вихідні дані більш зрозумілими та читабельними для користувачів.

## Як це зробити

За допомогою наступного коду в Bash можна здійснити капіталізацію рядка:

```Bash
str="привіт світ"
echo ${str^}
```

Вивід: "Привіт світ"

У цьому прикладі ми використали символ "^" для капіталізації першої літери у рядку. Також можна використовувати знаки "^^" для капіталізації всього рядка, або "^я" для переведення літер у нижній регістр.

## Глибше про капіталізацію рядка

У Bash існує багато різних способів капіталізувати рядок, включаючи використання умовних виразів та циклів. Також можна створити власну функцію для капіталізації рядка.
