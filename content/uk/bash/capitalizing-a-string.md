## Чому

Існує багато ситуацій, коли в програмуванні слід капіталізувати рядок. Наприклад, при виведенні користувацьких повідомлень або назв файлів. Капіталіція рядків зробить їх більш читабельними та полегшить роботу з ними.

## Як

Кодування капіталізації рядків досить просте в мові Bash. Для цього використовуємо команду `tr` та флаг `-s`.

```
Bash
# Приклад коду капіталізації рядка
string="привіт bash"

# Використовуємо команду `tr` та флаг `-s` для капіталізації рядка
capitalized_string=$(echo $string | tr '[:lower:]' '[:upper:]')

# Виводимо результат
echo $capitalized_string
```

Виходом з цього коду буде `ПРИВІТ BASH`, де усі літери будуть написані великими.

## Глибоке погруження

Для капіталізації можна також використати команду `awk`. Наприклад:

```
Bash
# Приклад коду капіталізації рядка з використанням awk
string="привіт bash"
capitalized_string=$(awk '{ print toupper($0) }' <<<$string)

# Виводимо результат
echo $capitalized_string
```

У цьому випадку результатом буде теж `ПРИВІТ BASH`.
