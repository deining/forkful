У цьому блозі ми подискутуємо про те, як прочитати текстовий файл за допомогою Bash програмування. Якщо ви хочете навчитися читати дані з файлів за допомогою Bash, то цей блог буде корисним для вас.

## Як

### Крок 1: Відкриття файла

Спочатку нам потрібно відкрити текстовий файл за допомогою команди `cat`:

```Bash
cat example.txt
```

Де `example.txt` - це назва вашого текстового файлу. Якщо все пройшло успішно, то ви повинні побачити вміст файлу в терміналі.

### Крок 2: Використання циклу

Тепер давайте розглянемо приклад коду, який дозволяє читати дані з файлу по рядках:

```Bash
while read line; do
  echo "$line"
done < example.txt
```

У цьому прикладі ми використовуємо `while` цикл, який читає по одному рядку з файлу та виводить його в терміналі. Для кожного рядка, який прочитується з файлу, виконується команда `echo`, яка виводить цей рядок в терміналі.

### Крок 3: Робота зі змінними

Також можна використовувати змінні для зберігання даних з файлу. Наприклад, якщо ваш файл містить список імен, ви можете зберегти його в змінну та вивести ці імена в терміналі:

```Bash
names=$(cat names.txt)
echo "$names"
```

У цьому прикладі ми зчитуємо дані з файлу `names.txt` та зберігаємо їх в змінну `names`. Потім виводимо цю змінну в терміналі.
