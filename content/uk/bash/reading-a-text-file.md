## Чому

Програмування на Bash є потужним інструментом для автоматизації процесів у багатьох сферах, таких як системне адміністрування, веб-розробка та аналіз даних. Читання текстових файлів є необхідним елементом для багатьох скриптів Bash, тому вивчення цієї навички є важливим кроком для покращення навичок у програмуванні.

## Як

Для зчитування текстових файлів у Bash використовується команда `read`. Нижче наведено приклад коду та його виведення для читання з файлу text.txt та виведення всього його вмісту:

```Bash
while read line; do
  echo $line
done < text.txt
```

Цей приклад демонструє використання циклу `while` та змінної `line`, яка містить кожен рядок з файлу. Кожен рядок потім виводиться за допомогою команди `echo`. В результаті виконання скрипта виведеться вміст файлу `text.txt`.

## Deep Dive

Команда `read` має багато опцій, які роблять читання файлів у Bash ще більш гнучким та потужним. Наприклад, можна використовувати опцію `-r`, щоб зчитувати рядки точно так, як вони є у файлі, без обробки спеціальних символів. Або опцію `-a`, яка дозволяє зчитувати рядки у масиви для подальшої обробки.

Додаткова інформація про ці опції та багато інших можливостей команди `read` доступна у офіційній документації [Bash](https://www.gnu.org/software/bash/manual/bash.html#Bash-Builtins).
