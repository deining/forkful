## Why
Цей текст перш за все призначений для людей, які зацікавлені у програмуванні на мові Bash. Якщо ви хочете навчитися, як працювати з текстовими файлами за допомогою командного рядка, то це стаття для вас.

## How To

```Bash
# Відкриття текстового файлу для читання
cat file.txt

# Виведення вмісту файлу на екран
echo "Вміст файлу:"
cat file.txt

# Використання циклу для читання кожного рядка файлу
while read line
do
  echo "Рядок: $line"
done < file.txt
```

Вихідна інформація з файлу може бути виведена на екран за допомогою команди `cat`. Також можна використовувати цикл `while` для посимвольного читання кожного рядка файлу і обробки його даних.

## Deep Dive

Для читання великих текстових файлів можна використовувати команду `less`, яка дозволяє переглядати файл поступово, а не весь в забороненому режимі. Крім того, можна використаті команди `grep`, `awk` або `sed`, які дозволяють здійснювати пошук і заміну певних даних в файлі.
