## Чому

Веб-сторінка є одним з основних інструментів для доступу до інформації в Інтернеті. Завантаження веб-сторінки з допомогою програмування Bash дозволяє автоматизувати процес отримання необхідної інформації з Інтернету.

## Як це зробити

Для завантаження веб-сторінки з використанням Bash, необхідно виконати наступні дії:

1. Встановіть необхідні програми. Для цього в терміналі введіть команду `sudo apt-get install curl`, яка встановить утиліту `curl`, необхідну для завантаження веб-сторінок.

2. Напишіть наступний код у файлі з розширенням `.sh`, наприклад `download.sh`:

```Bash
#!/bin/bash
# код для завантаження веб-сторінки
curl <URL>
```

Замість `<URL>` підставте посилання на ту веб-сторінку, яку бажаєте завантажити.

3. Збережіть файл та виконайте його з терміналу командою `./download.sh`. Процес завантаження буде запущений, а результат буде збережений у файлі з назвою, яка вказана після опції `-o`, наприклад `curl -o page.html <URL>`.

4. Щоб автоматизувати процес і завантажувати різні веб-сторінки, можна створити цикл у своєму коді, який буде змінювати посилання у команді `curl`.

## Глибоке занурення

Також можливе завантаження веб-сторінки у вигляді HTML документу у змінну у Bash. Для цього необхідно використати опцію `-s` команди `curl`, яка дозволяє вивести результат у вигляді тексту, а не зберегти у файлі. Приклад коду для цього:

```Bash
#!/bin/bash
# код для завантаження веб-сторінки
html_code=$(curl -s <URL>)
echo "$html_code"
```

Таким чином, можна далі обробляти HTML код і використовувати у своїй програмі.
