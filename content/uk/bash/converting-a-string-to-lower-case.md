## Чому

У програмуванні часто потрібно маніпулювати рядками тексту, іноді навіть змінювати їх регістр. Конвертування рядка до нижнього регістру відкриває багато можливостей для обробки та аналізу даних.

## Як

Існує кілька способів конвертування рядка до нижнього регістру у Bash. Один з них - використання вбудованої функції `tr` для заміни символів верхнього регістру на символи нижнього регістру.

```
Bash
# Введення початкового рядка
text="HeLLo WoRLd"
# Використання функції tr для заміни символів
text=$(echo $text | tr '[A-Z]' '[a-z]')
# Виведення результату
echo $text
```

Вищезазначений код відтворить "Hello world" у виведенні. Також можна використовувати функцію `sed`, щоб замінити символи верхнього регістру на символи нижнього регістру прямо в початковому рядку:

```
Bash
# Введення початкового рядка
text="HeLLo WoRLd"
# Використання функції sed для заміни символів
text=$(echo $text | sed 's/[A-Z]/\L&/g')
# Виведення результату
echo $text
```

Цей код також поверне "Hello world" у виведенні.

## Глибокодослідна інформація

Варто враховувати, що при конвертуванні рядка до нижнього регістру у Bash будуть враховані лише літери англійського алфавіту. Якщо у рядку є символи інших мов або спеціальні символи, вони залишаться без змін.

Також важливо врахувати, що функції `tr` і `sed` будуть створювати новий рядок з конвертованими символами, тому оригінальний рядок залишиться без змін. Якщо потрібно замінити символи верхнього регістру на символи нижнього регістру прямо в оригінальному рядку, можна використати функцію `tr` з опцією `-c` та функцію `sed` з опцією `-i`.
