## Чому

Конвертування рядка в нижній регістр є важливим кроком в обробці текстових даних. Він дозволяє зробити дані більш уніфікованими та зручними для подальшої обробки.

## Як це зробити

Найпростіший спосіб конвертувати рядок в нижній регістр у Bash - використати команду `tr` із використанням параметра `-s`. Наприклад:

```Bash
str="ПРИВІТ"
echo $str | tr '[:upper:]' '[:lower:]'
```
Виходом з цієї команди буде `привіт`.

## Поглиблене дослідження

Іноді, при роботі з більш складними рядками, може знадобитися використати більш продуктивний метод для конвертування в нижній регістр. В таких випадках можна використовувати вбудовану функцію `tolower` в Bash. Наприклад:

```Bash
str="Привіт, Світ!"
echo ${str,,}
```
Без проблема можна використовувати цю функцію для зміни регістру у рядках з більшим обсягом даних, також включаючи й українські символи.
