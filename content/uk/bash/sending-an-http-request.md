## Що & Чому?
Надсилання HTTP-запитів - це процес, за допомогою якого програмісти взаємодіють з веб-серверами. Вони використовують цю техніку для отримання або надсилання інформації, наприклад, запиту даних або оновлення даних на веб-сторінці.

## Як виконати:
Нижче наведено приклади використання команди ```curl``` для відправки HTTP-запитів та отримання відповіді від сервера. Для початку необхідно встановити програму ```curl```.

- Отримати информацію зі сторінки: ```Bash curl https://www.example.com```
- Відправити POST-запит з даними: ```Bash curl -X POST -d "name=John&age=25" https://www.example.com/api```

В результаті ви отримаєте відповідь від сервера з необхідною інформацією.

## Вдивись у глибину:
HTTP-запити є частиною HTTP-протоколу, який був створений близько 30 років тому. Цей протокол використовується для передачі даних між веб-серверами та веб-клієнтами. Окрім команди ```curl```, існують інші альтернативи, такі як програма ```wget``` або вбудовані засоби для виконання HTTP-запитів у мовах програмування, наприклад, Python або JavaScript. Під капотом, команда ```curl``` використовує бібліотеку libcurl для взаємодії з веб-сервером.

Також можна перевірити документацію вашої мови програмування для використання вбудованих засобів для виконання HTTP-запитів.
