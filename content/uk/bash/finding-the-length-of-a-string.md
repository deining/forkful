## Що і чому?
Знаходження довжини строки - це важлива задача для програмістів, яка дозволяє визначити кількість символів у текстовому рядку. Це може бути корисно для перевірки допустимих максимальних розмірів для введення або для обробки повідомлень користувачів.

## Як це зробити:
```Bash
# Задати змінну зі строкою
str="Це приклад строки"

# Використання вбудованого оператора ${#var} для знаходження довжини
echo "Довжина строки: ${#str}"

# Використання команди `expr` для вирахування довжини
echo "Довжина строки: $(expr length $str)"
```

Вивід:
```
Довжина строки: 17
Довжина строки: 17
```

## Поглиблене дослідження:
1. Історичний контекст: знаходження довжини строки входить до стандарту мови програмування Bash і було реалізовано в рамках проекту GNU.
2. Альтернативи: якщо ви програмуєте на інших мовах, таких як Python або Java, є власні вбудовані функції для знаходження довжини строки. Також можна використати команду `wc` для підрахунку кількості символів у файлі.
3. Деталі реалізації: вбудований оператор ${#var} використовує лічильник символів для підрахунку довжини строки, тоді як команда `expr` використовує функцію `length` та передає строку як аргумент.
