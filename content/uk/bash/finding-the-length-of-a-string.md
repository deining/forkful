## Чому

Існує багато різних випадків, коли потрібно знати довжину рядка у Bash скрипті. Наприклад, це може бути необхідно для перевірки довжини введеного користувачем пароля або для обробки текстових даних. У цій статті ми розглянемо як знайти довжину рядка у Bash та як це може бути корисно для вас.

## Як

Існує кілька способів знайти довжину рядка у Bash. Розглянемо кілька прикладів за допомогою коду та виводу результатів:

```Bash
# Використання змінної string
string="Привіт, світ!"
echo ${#string} # результат: 14

# Використання команди wc з параметром -c для підрахунку символів
echo -n "Привіт, світ!" | wc -c # результат: 14

# Зчитування рядка від користувача та відображення його довжини
echo "Введіть своє ім'я:"
read name
echo "Привіт, ${name}! Ваше ім'я складає ${#name} символів."
```

Результат:

```
14
14
Введіть своє ім'я:
John
Привіт, John! Ваше ім'я складає 4 символи.
```

## Deep Dive

У Bash існує спеціальна змінна `#`, яка повертає кількість аргументів, переданих у командний рядок. Коли використовується змінна `${#string}`, то буде повернуто довжину рядка `string`. Це дозволяє легко отримати довжину будь-якого рядка без необхідності використовувати додаткові команди.

Також можна використовувати команду `wc` з параметром `-m` для підрахунку символів у рядку. Однак, ця команда є більш ресурсозатратною, оскільки для підрахунку символів у рядку вона має обробляти весь текст, тим часом як за допомогою змінної `${#string}` необхідний результат можна отримати без зайвих операцій.
