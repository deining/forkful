## Що і навіщо?
Робота з YAML - це процес, що дозволяє програмістам працювати зі структурованим текстовим форматом даних. Це дозволяє зручно зберігати та обмінюватися інформацією між різними програмними системами.

## Як це зробити:
Один із способів роботи з YAML - використання командного рядка Bash. Для цього потрібно встановити програму `yq`, яка дозволить зчитувати та редагувати YAML файли. Далі, для взаємодії з YAML файлом, можна використовувати наступні команди:
```Bash
# Зчитати значення певного ключа
yq read file.yml key

# Додати новий ключ та значення
yq write file.yml new_key new_value

# Видалити ключ та його значення
yq delete file.yml key
```

## Глибші води:
YAML був створений у 2001 році як мова для структурованого збереження даних. Він легкий у використанні та читабельний для людей, що робить його популярним серед програмістів. Існують альтернативи для роботи зі структурованими даними, такі як JSON та XML, але YAML вважається одним з найбільш зручних та простих у використанні форматів.

Для роботи з YAML в Bash, також можна використовувати утиліту `jq`, але вона більш призначена для роботи з JSON даними та не має всіх функцій для роботи з YAML.
