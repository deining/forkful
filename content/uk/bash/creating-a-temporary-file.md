## Чому

Створення тимчасового файлу є важливою технікою в програмуванні, оскільки дозволяє зберігати тимчасові дані, що використовуються в процесі виконання програми. Це уникне ненавантаження основного файлу і збереже чистоту коду.

## Як

```Bash
# Створення тимчасового файлу
tempfile=$(mktemp)

# Запис тексту у файл
echo "Привіт, світ!" > $tempfile

# Вивід вмісту файлу
cat $tempfile
```

Вивід: 
```
Привіт, світ!
```

## Глибша аналітика

Тимчасові файли зазвичай створюються за допомогою команди `mktemp`. Ця команда автоматично створює унікальний файл зі згенерованим ім'ям, яке забезпечує його унікальність у контексті виконання програми.

Також, з допомогою параметрів `--tmpdir` та `--suffix`, можна вказати шлях для створення тимчасового файлу та додатковий суфікс у його імені. Наприклад, `mktemp --tmpdir=/tmp --suffix=.log`.

Будьте уважними при створенні тимчасового файлу, оскільки він буде автоматично видалений при завершенні програми або при виході з поточної сесії терміналу.
