## Чому

Створення тимчасових файлів є важливою частиною програмування в Bash, оскільки вони дозволяють зберігати тимчасові дані, що часто втрачаються після закінчення програми.

## Як

Для створення тимчасового файлу використовуйте команду `mktemp` з наступним синтаксисом:

```
temp_file=$(mktemp)
```

Ця команда створить унікальне ім'я для тимчасового файлу та збереже його в змінній `temp_file`.

Ви також можете створити тимчасовий файл з певним префіксом, використовуючи параметр `-p`:

```
temp_file=$(mktemp -p /tmp myprefix.XXXXXX)
```

Це створить тимчасовий файл з префіксом `myprefix` у директорії `/tmp`.

## Глибокий погляд

Команда `mktemp` використовує шаблони для створення унікального імені тимчасового файлу. Ви можете користуватися наступними шаблонами для визначення певних властивостей тимчасового файлу:

- `X` - створює унікальний рядок у імені файлу
- `u` - використовує UID поточного користувача у рядку
- `g` - використовує GID поточного користувача у рядку
- `t` - використовує час у секундах з початку Unix epoch у рядку
- `d` - використовує тимчасову директорію у назві файлу

Наприклад, команда `mktemp myprefix.XXXXXXXXXX` створить файл з іменем у вигляді `myprefix.hs4Ldm8sVH`.
