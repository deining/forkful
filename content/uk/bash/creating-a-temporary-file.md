## Чому
Створення тимчасового файлу є важливою складовою багатьох скриптів у Bash. Він дозволяє зберігати тимчасові дані та використовувати їх без забруднення основного файлового дерева.

## Як
```Bash
#!/bin/bash
# Створення тимчасового файлу з рандомним ім'ям
temp_file=$(mktemp)
# Запис до тимчасового файлу
echo "Це тимчасовий файл!" > $temp_file
# Виведення вмісту тимчасового файлу
cat $temp_file
# Видалення тимчасового файлу
rm $temp_file
```

В даному прикладі ми створюємо тимчасовий файл за допомогою команди `mktemp`, записуємо в нього дані та видаляємо його за допомогою команди `rm`. Це дуже корисні команди, які дозволяють легко створювати тимчасові файли та працювати з ними.

## Глибоке занурення
Тимчасові файли використовуються для збереження тимчасових даних, які потрібні для виконання скрипту. Однак, при створенні тимчасового файлу, важливо враховувати безпеку данних, оскільки вони можуть бути доступні для інших користувачів. Тому, бажано використовувати `mktemp` з різними параметрами, які дозволять гарантувати безпеку данних.
