## Чому

Генерація випадкових чисел є необхідною для багатьох програм та скриптів. Вона дозволяє створювати випадкові дані, що може бути корисно при тестуванні, моделюванні та інших аналітичних задачах. Також генерація випадкових чисел може додати цікавість та випадковість до вашої програми.

## Як

Для генерації випадкових чисел використовуйте утиліту ```Bash shuf```. Вона приймає список значень або файл та повертає випадкові значення.

Наприклад, якщо у вас є файл зі списком чисел ```numbers.txt```:

```Bash
1
2
3
4
5
```

Ви можете використати наступну команду для генерації випадкового числа від 1 до 5:

```Bash
shuf -n 1 numbers.txt
```

Це поверне випадкове число зі списку, наприклад, ```3```.

## Глибокий занурення

Існує декілька генераторів випадкових чисел, які ви можете використовувати в своїй утиліті ```shuf```. Наприклад, ви можете вказати параметр ```rand-source``` для використання іншого джерела, як наприклад, вихідного файлу-справжнього рандомайзера.

Також, ви можете використати команду ```Bash RANDOM```, яка згенерує випадкове число в діапазоні від 0 до 32767. Це можна використовувати для маленьких чисел або для створення простих випадкових чисел.
