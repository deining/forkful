## Чому

Створення випадкових чисел є незамінною частиною багатьох програм та скриптів. Це дозволяє забезпечити випадковість та уникнути непередбачуваних результатів при виконанні програми. Потрібно це робити, щоб ваші програми були більш надійними та захищеними.

## Як

Створення випадкових чисел в Bash може здатися складною задачею, але насправді це дуже просто. Ви можете використовувати вбудовану команду `echo` з параметром `$RANDOM`, який генерує випадкове число в діапазоні від 0 до 32767.

```Bash
echo $RANDOM
```
Вихідний результат буде виглядати приблизно так: `21744`.
Щоб отримати більші числа, можна використовувати параметр `-n` у команді `dd`, наприклад:

```Bash
dd if=/dev/urandom bs=1 count=4 status=none
```

Виходячи з діапазону значень, який ви хочете отримати, ви можете використовувати комбінацію цих двох методів для генерації випадкових чисел.

## Глибоке занурення

У Bash є багато різних методів для генерації випадкових чисел. Одним з них є використання команди `/dev/random`, яка зчитує випадкові дані з системного генератора випадкових чисел. Щоб скористатися цим методом, ви можете використовувати команду `cat` з параметром `</dev/random`, наприклад:

```Bash
cat </dev/random
```

Ви можете встановити кількість байтів, яку ви хочете отримати, за допомогою параметра `-n`, наприклад:

```Bash
cat </dev/random -n 4
```

Ви можете також використовувати зовнішні програми, які генерують випадкові числа, наприклад `shuf` або `openssl`:

```Bash
shuf -i 1-10 -n 1
```

```Bash
openssl rand -hex 8
```
