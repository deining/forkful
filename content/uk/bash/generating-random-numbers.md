# Що і чому?

Генерування випадкових чисел - це процес створення чисел, які не мають певного шаблону чи логічної послідовності. Програмісти використовують цей процес для створення різних видів даних, таких як паролі або інформація для ігрових програм.

# Як:

Використання програми ```Bash``` дозволяє легко згенерувати випадкове число за допомогою команди ```$RANDOM```. Нижче наведений приклад, як використовувати цю команду і яке буде виведення.

```Bash
#!/bin/bash

echo "Згенероване випадкове число: $RANDOM"
```

Виведення:  
```Bash
Згенероване випадкове число: 15640
```

# Глибше занурення:

Генерування випадкових чисел має довгу історію в програмуванні. Раніше для цього використовувалися складні математичні формули, але зараз існують багато вбудованих функцій, які спрощують цей процес для програмістів.

Існують також інші способи згенерувати випадкові дані, наприклад, за допомогою спеціальних програм або обладнання, такого як RNG (генератор випадкових чисел).
