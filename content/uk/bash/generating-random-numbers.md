## Чому

Генерація випадкових чисел є важливою функцією для багатьох комп'ютерних програм і задач. Вона може бути використана для створення випадкових даних, тестування програм або для шифрування інформації. Також, ця функція може бути корисною для генерації унікальних паролів або для наочного демонстрування концепцій випадковості у програмуванні.

## Як робити

Щоб повністю зрозуміти, як працює генерація випадкових чисел, ми розглянемо приклади на мові Bash та їхні вихідні дані. Для початку, нам потрібно використати команду `shuf`, що є стандартною утилітою для генерації випадкових чисел. Наприклад, якщо ми хочемо отримати п'ять випадкових цілих чисел від 1 до 10, ми можемо використати наступний код у Bash:

```
shuf -i 1-10 -n 5
```

Ця команда буде виводити різні комбінації п'яти випадкових чисел, наприклад: 4 10 6 9 1.

Можна також використовувати цю команду для генерації випадкових слів з заданого словника. Наприклад:

```
shuf -n 1 -e Сало Морозиво Сир Пиріг
```

Ця команда вибере випадкове слово зі списку та виведе його на екран.

## Занурення

Генерація випадкових чисел наводить на думку концепцію "випадковості". Проте, випадкові значення насправді генеруються на основі деякого алгоритму, що використовує початкове значення, називане "сід". Для того, щоб отримати дійсно випадкові числа, необхідно користуватися зовнішнім джерелом для сіда - наприклад, шум з мікрофону або знятий з погодного обладнання. Також варто знати, що спосіб генерації випадкових чисел може відрізнятися в залежності від операційної системи та інших факторів.
