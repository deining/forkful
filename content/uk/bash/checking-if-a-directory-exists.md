Why:

Для чого: Перевірка наявності каталогу є важливою задачею при виконанні скриптів або команд в терміналі і дозволяє забезпечити правильну роботу програми і уникнути помилок.

## Як це зробити

```Bash
if [ -d "назва каталогу" ]; then
  echo "Каталог існує"
else
  echo "Каталог не існує"
fi
```

Запуск цього коду призведе до виведення повідомлення про наявність або відсутність вказаного каталогу. Після ключового слова `-d` вказується назва каталогу, який необхідно перевірити. У разі використання цієї перевірки в скриптах, також можна виконати певні дії у залежності від результату перевірки.

## Глибші деталі

Під час перевірки наявності каталогу, Bash перевіряє, чи є файл з такою назвою і чи він є саме каталогом. У разі перевірки існування файлу, не обов'язково перевіряти його тип, але для більш точної перевірки, можна використати ключове слово `-e`, яке перевіряє наявність будь-якого файлу з вказаною назвою.
