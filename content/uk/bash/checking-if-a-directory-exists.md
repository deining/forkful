## Чому

Розробка Bash скриптів є важливою задачею для багатьох програмістів. Однак, є певні випадки, коли необхідно перевірити, чи існує директорія в системі. Це може бути корисним при побудові умовних операторів або при перевірці наявності необхідного шляху для створення нових директорій.

## Як

За допомогою наступного фрагменту коду Bash, ви можете перевірити, чи існує директорія з заданим шляхом у вашій системі:

```Bash 
if [[ -d /приклад/директорія ]]; then
  echo "Директорія існує"
else
  echo "Директорія не існує"
fi
```

Приклад виходу:

```Bash 
Директорія існує
```

Базуючись на цьому фрагменті коду, ви можете змінювати умови для перевірки існування директорії та виконувати різні дії, залежно від результатів перевірки.

## Глибокий занурення

Перевірка існування директорії базується на користувальницькій команді `test`, яка має різні параметри. Одним з них є `-d`, який перевіряє, чи існує даний шлях як директорія. Також можна використовувати інші параметри, такі як `-f` для перевірки файлу, або `-e`, який перевіряє будь-який тип файлу чи директорії.

Також варто зазначити, що якщо ви плануєте перевіряти існування директорії відносно поточної робочої директорії, то використовуйте параметр `.` у початку шляху, наприклад, `-d ./директорія`.
