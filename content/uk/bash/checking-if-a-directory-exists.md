## Чому

Уперше, перед тим, як розпочати урок, важливо зрозуміти, що таке перевірка існування директорії. Це корисна навичка у програмуванні Bash, яка дозволяє перевірити, чи існує певна директорія на комп'ютері. Така перевірка може бути корисна у багатьох сценаріях, наприклад, даний розділ дозволить вам перевірити, чи існує необхідна вам директорія перед початком виконання певної операції.

## Як

Існує кілька способів перевірки існування директорії за допомогою програмування Bash. Найпростіший з них - використання команди `test`, яка перевіряє наявність файлу або директорії і повертає значення `true` або `false`. Наприклад:

```Bash
if test -d "Ім'я директорії"; then
  echo "Директорія існує"
else
  echo "Директорія не існує"
fi
```

Іншим способом є використання команди `mkdir`, яка створює директорію і повертає значення `0` або `1` в залежності від успішності операції. Якщо директорія вже існує, команда поверне значення `1`. Наприклад:

```Bash
mkdir "Ім'я директорії"
if [[ $? -eq 1 ]]; then
  echo "Директорія вже існує"
else
  echo "Директорія не існує, створено нову"
fi
```

Можна також використовувати команду `ls`, яка виводить список файлів в директорії. Якщо файл або директорія існує, вона буде виведена у списку. Наприклад:

```Bash
if ls "Ім'я директорії"; then
  echo "Директорія існує"
else
  echo "Директорія не існує"
fi
```

## Глибоке дослідження

Розділ "How To" вже дав вам загальне уявлення про те, як перевірити наявність директорії за допомогою Bash. Але давайте розглянемо цей процес більш детально. Як ви бачите, у кожної з вищезазначених команд є свої особливості та переваги. Команда `test` є більш універсальною і може перевіряти і існування файлу, і директорії. Команда `mkdir` цілком призначена для створення директорій і повертає спеціальне значення, що показує їх наявність.
