## Зачем

Перевірять, чи існує директорія, є важливою частиною програмування в Bash. Це дозволяє контролювати доступ до файлів і виконувати певні дії в залежності від наявності цієї директорії.

## Як

Для перевірки існування директорії використовується команда "test" з флагом "-d" і назвою директорії. Наприклад, якщо нам потрібно перевірити, чи існує директорія "test", то ми можемо використати наступний код:

```Bash
if test -d test; then
    echo "Директорія test існує"
else
    echo "Директорія test не існує"
fi
```

Цей код здійснює перевірку наявності директорії "test" і, в залежності від результату, виводить відповідне повідомлення.

## Deep Dive

При перевірці наявності директорії, слід мати на увазі деякі нюанси. По-перше, "test" не може перевірити, чи є директорія символічним посиланням. Тому, якщо потрібно перевірити саме директорію, а не символічне посилання на неї, слід використовувати команду "test -e" замість "test -d". Крім того, "test" не перевіряє, чи користувач має права на читання цієї директорії. Тому при здійсненні певних дій з директорією слід перевірити це окремо.
