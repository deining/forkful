## Чому

У сучасному світі технологій, важко уявити роботу програміста без знання отримання поточної дати. Це важлива частина інфраструктури будь-якого проекту, яка допомагає в здійсненні різних обчислень, логіку та взаємодію з користувачем. У цій статті ми поглиблено дізнаємося про отримання поточної дати в Bash програмуванні та як це може бути корисним для вашої роботи.

## Як це зробити

Здобуття поточної дати в Bash дуже просте та прямолінійне завдання. Для цього ми використаємо команду `date`, яка буде повертати різні формати поточної дати залежно від наших потреб. Ось декілька прикладів, як це можна реалізувати в коді:

```Bash
# Отримання дати в форматі року-місяця-дня
CURRENT_DATE_1=$(date +"%Y-%m-%d")
echo "Поточна дата: $CURRENT_DATE_1"

# Отримання дати та часу в форматі року-місяця-дня година:хвилина:секунда
CURRENT_DATE_2=$(date +"%Y-%m-%d %T")
echo "Поточна дата та час: $CURRENT_DATE_2"

# Отримання дати з перекладом назви місяця та дня тижня
CURRENT_DATE_3=$(date +"%A, %B %d, %Y")
echo "Поточна дата: $CURRENT_DATE_3"
```

Результат виконання цього коду буде виглядати так:

```
Поточна дата: 2021-10-12
Поточна дата та час: 2021-10-12 08:00:00
Поточна дата: Tuesday, October 12, 2021
```

Одним із застосувань отримання поточної дати може бути автоматизація створення резервних копій вашого проекту. Наприклад, використовуючи цю команду, ви можете додати будь-яку з цих дат до назви архіву, щоб зробити його більш організованим та легшим для розпізнавання.

## Глибше занурення

Команда `date` використовує різні форматування для отримання поточної дати. Основні символи, що використовуються для цього, є:

- `%Y` - повертає чотиризначний рік (наприклад, 2021)
- `%m` - повертає двоцифровий місяць (наприклад, 10)
- `%d` - повертає двоцифровий день (наприклад, 12)
