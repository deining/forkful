## Зачем
Будь-який програміст, що працює з Bash, рано чи пізно потребує отримати поточну дату для виконання певних операцій, таких як резервне копіювання, логування або створення унікальних імен файлів.

## Як виконати
Існує кілька способів отримати поточну дату в Bash, але найбільш простий і поширений - використання команди `date`. Ось приклад використання:

```Bash
current_date=$(date +"%Y-%m-%d")
```
Ця команда поверне поточну дату у форматі `YYYY-MM-DD`, який можна використовувати в подальших операціях.

Також можна застосувати функцію `date` для отримання інших форматів, наприклад, тільки часу:

```Bash
current_time=$(date +"%H:%M:%S")
```

## Глибока здамовніть
Щоб зрозуміти, як саме працює команда `date`, треба знати про форматування дати і часу у Bash. Формати дати і часу задаються за допомогою спеціальних символів, таких як `%Y` - рік, `%m` - місяць, `%d` - день і т.д. Повний перелік можна знайти у документації Bash. Крім того, команда `date` може приймати додаткові аргументи, такі як `-d` для отримання дати з певною модифікацією чи `-u` для отримання універсального часу.
