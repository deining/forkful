## Що це і для чого?
Отримання поточної дати є одним з базових та важливих завдань у програмуванні. Воно дозволяє отримати поточний час, дату та часовий пояс на комп'ютері або сервері, що є необхідним для багатьох програм та скриптів.

## Як отримати поточну дату в Bash
```
date +%d-%m-%Y
```
Використання команди `date` у поєднанні зі спеціальними параметрами дозволяє отримати поточну дату в потрібному форматі. Наприклад, команда `+%d-%m-%Y` виведе дату у форматі день-місяць-рік.

Вихідні дані:
```
11-11-2021
```

## Передбачення
Існує кілька інших способів отримання поточної дати в Bash, наприклад, використання команди `eval` або перенаправлення результату команди `date` у текстовий файл.

## Детальний аналіз
Історично, команда `date` була створена у 1972 році для операційної системи Unix. Вона отримала значну популярність серед розробників та продовжує бути однією з основних команд у будь-якій системі Unix або Linux.

Для отримання повної документації та переліку параметрів команди `date`, можна запустити команду `man date` або переглянути офіційну документацію на веб-сайті Linux.
