## З чого почати

Програмування є важливою частиною сучасного світу, а Bash є однією з найпопулярніших мов командного рядка. Однією з корисних навичок, яка допоможе вам стати ефективнішим у програмуванні, є вивчення як отримати поточну дату в Баші. В цьому блозі ми розглянемо, чому це корисно і як цього досягти.

## Як це зробити

Поточне моментальне рівна цілій змінній DATE, якій можна присвоїти низ рядків командою "date +"%d/%m/%y"". Цей метод може бути корисним, якщо ви потребуюєте простої інформації про поточну дату в якомусь додатку або скрипті.

```Bash
#!/bin/bash
DATE=$(date +"%d/%m/%y")
echo "Сьогоднішня дата: $DATE"
```

```Bash Output
Сьогоднішня дата: 28/01/21
```

Також можна додати більше форматування до поточної дати, щоб виводити більше інформації. Наприклад, для виведення і часу, і часової зони, використайте команду "date +"%r %Z"".

```Bash
#!/bin/bash
CURRENT_TIME=$(date +"%r %Z")
echo "Поточний час і часова зона: $CURRENT_TIME"
```

```Bash Output
Поточний час і часова зона: 08:35:40 AM EST
```

## Глибше процідження

Існує багато різних параметрів, які можна використовувати при отриманні поточної дати в Баші. Наприклад, ви можете використовувати команду "date +"%A"" для виведення повної назви поточного дня, або "date +"%B"" для виведення повної назви поточного місяця. Також наявні інші параметри для виведення дати і часу в різних форматах.

Можна також використовувати команду "man date" для отримання більш детальної інформації про різні параметри та приклади використання.
