## Для чого
Коли починаєте вивчати мову програмування, ви, швидше за все, зіткнулися з необхідністю отримання поточної дати. Як правило, використання поточної дати це частина багатьох скриптів та програм. Це корисно для запису подій, розробки звітів, автоматизації процесів та багато чого іншого. Також, це може бути корисно просто для того, щоб познайомитися з роботою змінних та операторів у мові Bash.

## Як
Щоб отримати поточну дату у Bash, використовуйте команду `date` з опцією `+%d/%m/%y`, яка встановлює формат дати на DD/MM/RR формат. Нижче наведено приклад використання цієї команди, а також вихідний код.

```Bash
date +%d/%m/%y
```

Вихідний код буде виглядати приблизно так: `24/07/21`. Якщо хочете отримати більше інформації про поточну дату, ви можете використовувати інші опції `date`, такі як `%A` для виведення повної назви дня тижня або `%B` для виведення повної назви місяця.

## Глибокий погляд
Команда `date` є частиною утиліти GNU Coreutils та вона працює у багатьох операційних системах. Вона також може бути дуже корисною для автоматизації завдань в скриптах. Наприклад, ви можете використати `date` для створення каталогів з поточною датою у назві або для копіювання файлів у різні папки за датами.
