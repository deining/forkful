## Для чого: 

При написанні скриптів на Bash виникає потреба відстежувати проміжні значення та помилки для знаходження проблем та відладки програми. Такі виявлені помилки можна легко виправити, якщо виводити на екран значення змінних та кроки виконання коду.

## Як це зробити:

Використання команди `echo`, яка виводить передані їй аргументи на екран, дозволяє виводити потрібну інформацію для відлагодження. Наприклад:

```Bash
# оголошення змінних
name="Іван"
age=25

# виведення значень змінних
echo "Прізвище: $name"
echo "Вік: $age"

# виведення кроку виконання коду
echo "Виконується умова if"

# помилка
echo "Помилка виконання скрипту"
```

Результат виводу:

```
Прізвище: Іван
Вік: 25
Виконується умова if
Помилка виконання скрипту
```

## Глибоке погруження:

Крім просто виводу значень, також можна використовувати команди `echo` та `printf` для форматованого виводу даних. Наприклад, для виводу числа з плаваючою точкою з двома знаками після коми можна використати наступний скрипт:

```Bash
# оголошення змінної
num=25.254

# виведення числа з точністю до двох знаків після коми
printf "Число: %.2f\n" $num
```

Результат виводу:

```
Число: 25.25
```
