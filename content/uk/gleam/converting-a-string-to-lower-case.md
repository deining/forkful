## Чому

Коли ви працюєте з текстовими данними, ви, можливо, захочете перетворити їх у нижній регістр (lower case) для однорідності та простоти обробки. Наприклад, це може бути корисно, коли ви порівнюєте рядки або шукаєте конкретне слово в тексті. У цій статті ми дізнаємось, як перетворити рядок на нижній регістр у мові програмування Gleam.

## Як

Існує кілька методів для перетворення рядка на нижній регістр у мові Gleam. Ми розглянемо декілька з них нижче.

### Використання функції `to_lower`

Функція `to_lower` є вбудованою у мову Gleam та приймає рядок як аргумент та повертає його у нижньому регістрі.

```Gleam
import gleam/string

let name = "Hello World"
let lowercase_name = string.to_lower(name)

// Результат: "hello world"
```

### Використання методу `to_lower`

В Gleam також є функція `to_lower`, яка може викликатись на об'єкті типу `String`. Вона працює точно так само, як і функція з розділу вище.

```Gleam
import gleam/string

let name = "Hello World"
let lowercase_name = name.to_lower()

// Результат: "hello world"
```

## Deep Dive

Якщо ви подивилися на вторгення коду, можливо, ви побачили, що всі букви були перетворені в нижній регістр. Це включає не тільки літери, а й будь-які пунктуаційні знаки, цифри та пробіли. Це потрібно враховувати при використанні цієї функції у вашому коді.

Крім того, важливо знати, що мова Gleam використовує кодування UTF-8 для обробки символів. Це означає, що комбіновані символи, наприклад з акцентами або діакритичними знаками, будуть зачислені окремо.
