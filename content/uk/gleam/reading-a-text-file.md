# Що & чому?

Читання текстового файлу - це процес, коли програміст обробляє вміст текстового документу, щоб отримати необхідні дані для своєї програми. Це є необхідною частиною програмування, оскільки більшість програм потребують зчитування даних з файлів.

# Як це зробити?

```
Уявімо, що у нас є текстовий файл з наступним вмістом:

Hello world!
This is a sample text file for Gleam programming.

Ми можемо написати такий код, щоб зчитати цей файл у нашій програмі:
```
Gleam.file_reader("sample.txt")
```
В результаті ми отримаємо, наприклад, такий вивід:
```
["Hello world!", "This is a sample text file for Gleam programming."]
```

# Вглибину

Читання текстового файлу є стандартною функцією багатьох мов програмування, також доступну у Gleam. Цей процес дозволяє програмістам працювати зі збереженими даними, що полегшує процес створення програм.

Як альтернативу, можна використовувати бібліотеку зчитування файлів для обробки даних у спеціальному форматі, наприклад, CSV або JSON. Це може пришвидшити процес читання та обробки даних.

Щоб отримати більш точну інформацію про читання текстових файлів у Gleam, можна прочитати документацію на офіційному сайті мови програмування.
