## Чому

З часом, коли ми пишемо код, ми можемо зіткнутися із помилками та проблемами. Іноді наш код не працює так, як ми очікували і ми шукаємо способів вивести цю інформацію. Одним з інструментів, який може нам допомогти в цьому, є запис до стандартного виводу помилок. В цій статті ми розглянемо, чому це важливо та як це зробити у мові програмування Gleam.

## Як це зробити

Для запису до стандартного виводу помилок у мові програмування Gleam існує кілька можливих шляхів. Один із них - використовувати функцію `error` з модулю `io`.

```Gleam
let message = "Помилка: це неправильно"
io.error(message)
```

В результаті виведеться:

```sh
Помилка: це неправильно
```

Ми також можемо використовувати функцію `write_to_error_port` з модулю `io`:

```Gleam
let message = "Помилка: це неправильно"
io.write_to_error_port(message)
```

В результаті отримаємо:

```sh
Помилка: це неправильно
```

## Глибоке занурення

Крім функцій `error` та `write_to_error_port`, у мові програмування Gleam є інші способи запису до стандартного виведення помилок. Наприклад, ми можемо скористатися функцією `file:path` з модулю `io` для виведення помилок у вказаний файл.

```Gleam
let message = "Помилка: це неправильно"
let error_file = "error.txt"
io.file:path(error_file, message)
```

В результаті у файлі `error.txt` буде записано:

```sh
Помилка: це неправильно
```
