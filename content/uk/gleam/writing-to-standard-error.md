# Чому

Написання до стандартного виводу помилок є важливою частиною програмування, оскільки він допомагає виявляти та виправляти помилки в коді.

# Як це зробити

```Gleam
fn main() {
  let name = "Gleam";

  gleam::stdout()
    .write_line("Привіт, я " ++ name)
    .map_err(
      fn(e) { gleam::stderr().write_line(e) }
    );
}
```

Результат:

`Привіт, я Gleam`

Цей приклад показує, як використовувати `stdout()` та `stderr()` модулі з Gleam бібліотеки для писання до стандартного виводу та виводу помилок. Зверніть увагу на використання методу `map_err()` для обробки помилки, яка може виникнути при записі до стандартного виводу.

# Глибші води

Написання до стандартного виводу помилок може бути корисною інструментом для налагодження коду. Ви можете використовувати їх для виводу помилок та налагодження певних шляхів коду. Також, використання `stdout()` та `stderr()` може допомогти визначити, де саме в коді виникає помилка.
