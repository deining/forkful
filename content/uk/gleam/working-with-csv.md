## Чому

Робота з CSV файлами є неодмінною частиною багатьох проектів програмування, особливо якщо ви працюєте з даними. Використання мови програмування Gleam дає можливість легко і ефективно обробляти дані у форматі CSV.

## Як використовувати

Для початку роботи з CSV файлами у мові програмування Gleam, необхідно імпортувати бібліотеку CSV. Для цього, додайте наступний рядок коду до вашого файлу:

```Gleam
import csv
```

Після цього, ви можете використовувати функції бібліотеки CSV для зчитування та запису даних у CSV форматі.

Наприклад, для зчитування даних з CSV файлу, необхідно викликати функцію `csv.read` і передати їй шлях до файлу. Приклад коду:

```Gleam
let data = csv.read("data.csv")
```

Отримані дані будуть збережені у змінній `data` у вигляді списку рядків.

Для запису даних у CSV файл, використовуйте функцію `csv.write` і передайте їй список списків, який буде записаний у файл. Приклад коду:

```Gleam
let data = [["John", "Doe", "john@example.com"], ["Jane", "Smith", "jane@example.com"]]
csv.write("data.csv", data)
```

Цей код створить CSV файл з наступним вмістом:

```
John, Doe, john@example.com
Jane, Smith, jane@example.com
```

## Глибока підготовка

Бібліотека CSV у мові програмування Gleam підтримує багато додаткових функцій, таких як вказання роздільника між значеннями, пропуск першого рядка у файлі та інші.

Стежте за документацією бібліотеки CSV для ознайомлення з усіма можливостями та функціями, що надаються.
