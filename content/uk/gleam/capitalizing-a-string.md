## Чому

Здавало би, це може здатися простим завданням, але створення зрозумілої та читабельної розмітки - ключовий елемент роботи програміста. Один з часто використовуваних методів для цього - це капіталізація рядка, а саме, перетворення першої літери на велику. Це може бути особливо корисно для створення заголовків, маркерів часу та інших важливих елементів веб-сторінок. У цій статті ми розглянемо, як зробити це за допомогою мови програмування Gleam.

## Як

Для початку, давайте створимо новий проект Gleam та додамо необхідну залежність у файл `gleam.toml`:

```Gleam
[dependencies]
string = "0.8.0"
```

Тепер ми можемо імпортувати модуль `string` та використовувати його функцію `capitalize` для капіталізації рядка:

```Gleam
import string

capitalize("hello") // "Hello"
```

Просто так! Тепер наша функція може обробити будь-який рядок та повернути його з першою великою літерою. Але що, якщо ми хочемо капіталізувати не лише першу літеру, а всі літери слова? Для цього існує також функція `capitalize_all`, яка працює аналогічно, але для всіх літер в рядку:

```Gleam
import string

capitalize_all("hello world!") // "Hello World!"
```

Зверніть увагу, що ця функція також ігнорує будь-які спеціальні символи чи числа і залишає їх без змін. Також варто відзначити, що обидві функції можуть бути використані для капіталізації не лише англійських слів, а й будь-яких інших букв та символів.

## Глибше проникнення

Хоча функції `capitalize` та `capitalize_all` можуть здатися простими, вони дійсно базуються на складному механізмі та мають синтаксис, подібний до регулярних виразів. Для капіталізації окремих літер використовується метод `to_upper`, який повертає копію рядка з першою літерою великої, а для інших - з незміненими символами. Таким чином, кожне слово в рядку проходить обробку та отримує необхідний результат.
