## Чому

З кожним поколінням комп'ютерів та програмування стає все більш доступним і розповсюдженим в Україні. Чи не було б цікаво дізнатися про простий шлях до створення програм? З розвитком комп'ютерної технології все більше людей починають цікавитися програмуванням та зацікавлюються різними мовами програмування. Сьогодні ми поговоримо про мову програмування Gleam та про одну з базових операцій - капіталізацію рядків (*capitalizing a string*).

## Як

Капіталізація рядка - це процес перетворення першої літери кожного слова на велику. Наприклад, рядок "програмування це цікаво" повинен бути перетворений у "Програмування Це Цікаво". Давайте подивимося, як ми можемо це зробити за допомогою мови програмування Gleam.

```Gleam
офсіт = 32 // Код ASCII відмінності між великою та малою буквою
функція капіталізації(рядок) {
	новий_рядок = рядок
	новий_рядок[0] = новий_рядок[0] - офсіт
	попередні_символ = новий_рядок[0]
	для (символ в новий_рядок) {
		якщо (попередні_символ == " ") {
			новий_рядок[і] = новий_рядок[і] - офсіт
		}
		попередні_символ = символ
	}
	повертає новий_рядок
}
вивід(капіталізація("програмування це цікаво"))
```

Вивід цього коду буде "Програмування Це Цікаво". Давайте розберемося що відбувається в цій функції крок за кроком.

1. Починаємо зі створення змінної `офсіт`, яка буде використовуватися для розрахунку різниці між великою та малою буквою за таблицею ASCII.
2. Далі, ми створюємо функцію `капіталізації`, яка приймає рядок як аргумент.
3. Створюємо нову змінну `новий_рядок` та копіюємо в неї аргумент функції. Це дає нам можливість змінювати рядок, не впливаючи на вхідний аргумент.
