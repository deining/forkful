## Чому 

Почати новий проект в Gleam це чудовий шлях для розширення своїх знань у програмуванні та вивчення нових інструментів. Gleam - це сучасна мова програмування, яка пропонує багато можливостей для створення якісного і ефективного коду. 

## Як це зробити 

Давайте розберемося, як розпочати новий проект в Gleam та що для цього потрібно. 

```Gleam
import gleam/io
fn main() {
  gleam/io.print("Привіт, Україно!")
}
```

У цьому прикладі ми імпортуємо модуль "io" для використання функції "print" та виводимо повідомлення "Привіт, Україно!" у консоль. Це перший крок для створення нашого нового проекту. 

## Глибшого пірнання 

Але Gleam надає нам багато більше можливостей, ніж просто виведення повідомлень у консоль. Ми можемо створювати модулі, функції та структури даних, щоб структурувати наш код і зробити його більш організованим і ефективним. Крім того, Gleam має сильну типізацію, що дозволяє нам уникнути багатьох помилок та полегшує розробку.
