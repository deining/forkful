## Що & Чому?

Зчитування аргументів командного рядка - це процес отримання даних від користувача під час виконання програми. Це важливий інструмент для програмістів, оскільки дозволяє взаємодіяти з програмою і передавати їй необхідні дані для обробки.

## Як:

Для зчитування аргументів командного рядка в Gleam, використовуються функції з модулю `gleam/arg`. Наприклад, для отримання першого аргументу командного рядка, ми можемо використати функцію `arg.first`:

```gleam
import gleam/arg

let first_arg = arg.first
```

Якщо програма запускається з аргументами `hello world`, то змінна `first_arg` буде містити значення `hello`. Після чого ми можемо виконати різні операції з цим значенням.

## Глибоке погруження:

Зчитування аргументів командного рядка - це стандартний процес в програмуванні. Без цього, було б дуже важко передавати програмі необхідні дані для обробки. Альтернативою зчитуванню аргументів може бути написання свого власного парсера, але це займе більше часу і зусиль. У Gleam, є інші функції для зчитування аргументів, такі як `arg.rest`, яка повертає список всіх аргументів після першого, або `arg.from_env`, яка дозволяє зчитувати аргументи з оточення.
