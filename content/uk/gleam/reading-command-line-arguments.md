## Чому

Читання аргументів командного рядка є невід'ємною частиною процесу програмування та допомагає здобути більшу контроль над виконанням програми. Ця навичка є важливою для розробники, що працюють з мовою програмування Gleam. Отже, цей пост допоможе вам розібратися з читанням аргументів командного рядка та розпочати ефективне використання цієї функції в своїх програмах.

## Як

Для читання аргументів командного рядка у Gleam використовується модуль `gleam/io/args`. Цей модуль містить функцію `args/0`, яка повертає список аргументів, переданих під час запуску програми. Для отримання доступу до цих аргументів використовується звичайний цикл `for` та індексування елементів списку. Наприклад:

```Gleam
import gleam/io/args

fn main(_arguments) {
  for argument in args/0() {
    gleam/io/println(argument)
  }
}
```

При запуску програми з аргументом "hello", цей код виведе:

```
hello
```

## Глибші дослідження

Для отримання більшої контролю над читанням аргументів командного рядка в Gleam, можна використовувати функцію `gleam/io/args/option/2`. Ця функція приймає два аргументи: ім'я аргументу та тип, в який потрібно перетворити значення аргументу. Наприклад, якщо потрібно отримати ціле число із аргументу під назвою "number", то це можна зробити так:

```Gleam
import gleam/io/args

fn main(_arguments) {
  let number = args/option/2("number", int)
  gleam/io/println(number)
}
```

Якщо при запуску програми передати аргумент "123", цей код виведе:

```
123
```
