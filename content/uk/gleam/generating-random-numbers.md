# Що та чому?

Генерування випадкових чисел - це процес створення чисел, що вибираються на випадкових принципах. Це корисний інструмент для програмістів, оскільки допомагає створювати різноманітні сценарії та тести для програм.

# Як це зробити:

```Gleam
// Генеруємо випадкове ціле число у діапазоні від 1 до 10
let random_number = random.uniform(1, 10)
```
```Gleam
// Генеруємо випадковий біт (0 або 1)
let bit = random.bool()
```
```Gleam
// Генеруємо випадкову послідовність символів
let random_string = random_string(10)
// Поверне наприклад "qfTpK2gKkF"
```

# Глибоке погруження:

Історичний контекст: Генерування випадкових чисел є давнім та важливим концептом у світі програмування. Вимога до генераторів випадкових чисел почалася ще в 1940-х роках, коли були створені пристрої для шифрування комунікацій військовим. З того часу його використання стало все більш розповсюдженим у різних областях програмування.

Альтернативи: Існують різноманітні способи генерування випадкових чисел, такі як використання генераторів псевдовипадкових чисел або отримання випадкових даних з зовнішніх джерел.

Деталі реалізації: У Gleam для генерування випадкових чисел використовується функція ```random``` з модуля ```Random```. Вона використовує алгоритм Marsaglia псевдовипадкових чисел для заповнення апаратури генерування випадкових чисел.
