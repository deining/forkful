## Чому

Навіщо генерувати випадкові числа у програмуванні? Звичайно, це корисно для створення різноманітності у даних, створення випадкових тестових сценаріїв або просто для веселих випадкових експериментів.

## Як це зробити

Щоб згенерувати випадкове число у Gleam, нам потрібне використати модуль `Random` та функцію `generate`. Ось приклад коду для генерування випадкового числа від 1 до 100:

```Gleam
import Random

let random_number = Random.generate(1, 100)
```

Ви також можете використовувати `generate` для генерування випадкового булевого значення, вказавши тип `Bool` у виклику функції:

```Gleam
import Random

let random_bool = Random.generate(Bool)
```

У наших прикладах ми використовуємо базовий генератор псевдовипадкових чисел з джерелом ентропії заснованим на грубих значеннях. Однак, у Gleam також є модуль для криптографічно безпечного генерування випадкових чисел - `SecureRandom`. Проте, повідомлення про помилку може бути виведене, якщо немає достатньої ентропії, буде використано базовий генератор замість.

## Поглиблене дослідження

У Gleam існує багато різних типів даних для генерування, включаючи цілі числа, рядки, лістинги та більш складні типи, наприклад кортежі та записи. Ви можете знайти більше інформації про модуль `Random` та різні методи генерування випадкових значень у документації на офіційному сайті Gleam.
