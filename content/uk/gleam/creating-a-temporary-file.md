## Що & За що?
Створення тимчасового файлу - це процес створення файлу, який існує тимчасово, тобто не на постійному зберіганні. Програмісти часто використовують цей метод для зберігання даних під час виконання програми.

## Як: 
```Gleam
import gleam/tempfile

let temp_file = tempfile.create() // Створюємо тимчасовий файл

gleam/io.write(temp_file, "Це приклад тексту для запису в тимчасовий файл.") // Записуємо дані в файл

let contents = gleam/io.read_to_end(temp_file) // Читаємо дані з файлу

gleam/io.close(temp_file) // Закриваємо файл

```
Вивід: "Це приклад тексту для запису в тимчасовий файл"

## Глибока пірковани. 
(1) Історичний контекст: Використання тимчасових файлів є давнім практикою серед програмістів. З часом, із зростанням обсягів даних та складністю програм, цей метод став все більш важливим. (2) Альтернативи: Обговорюються інші методи збереження тимчасових даних, такі як користувацький каталог або використання бази даних. (3) Деталі імплементації: Конкретний спосіб створення тимчасового файлу може відрізнятися в залежності від мови програмування та операційної системи.
