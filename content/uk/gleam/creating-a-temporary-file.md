## Чому
Введення тимчасових файлів є корисним для створення інструментів обробки даних або для тимчасового збереження інформації, яка повинна бути видалена після виконання деяких дій.

## Як
Для створення тимчасового файлу в Gleam, використовуйте функцію `Temp.file()` і передайте їй назву файлу. Це створить файл у тимчасовій директорії і поверне шлях до нього. Наприклад:

```Gleam
let tmp_file = Temp.file("data.csv")
```

```
/tmp/data.csv
```

Далі, можна використовувати звичайні операції з файлами, такі як читання, запис або видалення даних. Після закінчення роботи з файлом, слід видалити його за допомогою функції `File.delete()`. Наприклад:

```Gleam
File.delete(tmp_file)
```

## Поглиблене вивчення
Під час створення тимчасового файлу, Gleam автоматично обробляє контроль доступу до нього. Крім того, використовується системний змінний каталог для зберігання тимчасових файлів, тому не треба хвилюватися про місце зберігання або конфлікти з іншими файлами.
