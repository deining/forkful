## Чому

Створення тимчасових файлів може бути важливою задачею для багатьох програмістів, особливо якщо вони працюють з багатьма файлами одночасно або виконують операції з файлами, що вимагають бутищчок не більше, а ніж тимчасовими.

## Як

```Gleam
import gleam/os

temp_file = os.temp_file("myfile.txt")
```

Цей код буде створювати тимчасовий файл з назвою "myfile.txt" в поточному каталозі програми. Ви також можете визначити інші параметри, наприклад, шлях та розширення файлу. 

```Gleam
import gleam/os

temp_file = os.temp_file("myfolder/myfile", file_extension=".csv")
```

Ви можете робити з цим файлом будь-які дії, як і з будь-яким іншим файлом, наприклад, записати дані у нього або прочитати інформацію з нього. Можете також перевірити існування файлу за допомогою функції `os.file_exists()`.

```Gleam
import gleam/os
import gleam/io

file_data = "This is some data to write to the temporary file"
temp_file = os.temp_file()
io.write_file(temp_file, file_data)
```

У вищепереліченому прикладі ми спочатку створили тимчасовий файл без вказаної назви, потім записали в нього дані і, нарешті, прочитали ці дані за допомогою функції `io.read_file()`. Ви також можете використовувати цей файл для тестування невеликих частин коду без використання постійних файлів.

## Глибоке дослідження

Функція `os.temp_file()` є частиною стандартної бібліотеки Gleam та вона використовується для створення тимчасових файлів у багатьох програмах на Gleam. Ця функція генерує випадкову назву для файла на основі часу та інших параметрів, тому ви можете бути впевнені в унікальності назви вашого тимчасового файлу.
