# Що & Чому?
Написання текстового файлу - це процес створення файлу, який містить текстову інформацію. Це важлива частина програмування, оскільки програмісти зберігають та обробляють багато даних у текстових файлах для різних цілей.

# Як:
Код з нижче наведеними прикладами та виходом може бути виконаний у Gleam, щоб створити текстовий файл.

```Gleam
import io

// Створення текстового файлу
let filename = "text.txt" 
// Відкриваємо файл для запису
let file_writer = io.file.open(filename, .{create, write})
// Записуємо текст у файл 
file_writer.write("Привіт світ!")
// Закриваємо файл
file_writer.close()

// Вивід вмісту текстового файлу
let file_reader = io.file.open(filename, .{read})
let file_contents = file_reader.read_to_string()
file_reader.close()
io.println(file_contents)
// Вивід: Привіт світ!
```

# Глибоке занурення:
Історичний контекст: написання текстового файлу - це одна з найстаріших та найбільш важливих технік, які використовуються програмістами для зберігання та обробки даних в їх програмах. До винайдення більш потужних систем керування базами даних, текстові файли були переважно використовуваними для зберігання даних у комп'ютерних програмах.

Альтернативи: у сучасних мовах програмування існують інші способи створення та зберігання даних, такі як бази даних, JSON і XML файлів, але написання текстового файлу залишається важливою технікою для програмістів.

Деталі реалізації: Gleam надає можливості для створення та редагування текстових файлів за допомогою вбудованих бібліотек. За допомогою цих інструментів програмісти можуть легко зберігати та обробляти дані у своїх програмах.
