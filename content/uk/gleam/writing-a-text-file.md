## Зробимо наш код більш читабельним з Gleam

Як програмісти, ми проводимо багато часу, пишучи код. Тому часто нам потрібно зробити наш код більш читабельним, щоб інші люди зрозуміли його швидше і легше його використовували. В цій статті ми розглянемо, як Gleam може допомогти нам написати текстовий файл, що зробить наш код більш організованим і легким для читання і редагування.

## Як це зробити?

Для початку нам потрібно створити новий файл з розширенням ".gleam", наприклад "text_files.gleam". Далі додамо наступний код:

```Gleam
import gleam/file

fn write_to_file() {
  let text = "Це зразок тексту, який ми хочемо записати до файлу."
  file.write("./my_text_file.txt", text)
}
```

У цьому прикладі ми імпортуємо модуль `gleam/file`, щоб мати доступ до функції `write`, яка дозволяє нам створити та записати до файлу. Використовуючи ключове слово `let`, ми створили змінну `text`, в яку записали наш текст. Далі, використовуючи функцію `write`, ми передали шлях до нашого нового файлу та змінну `text`.

Якщо ви запустите цей код, ви побачите, що в папці з програмою з'явився новий файл з назвою "my_text_file.txt", а у ньому записаний наш текст.

## Глибше вдійдемо в цю тему

Ще одна корисна функція, яка є у модулі `gleam/file`, це функція `append`. Вона дозволяє додавати новий текст до вже існуючого файлу замість того, щоб перезаписувати його повністю.

```Gleam
import gleam/file

fn append_to_file() {
  let text = "Це новий текст, який ми хочемо додати."
  file.append("./my_text_file.txt", text)
}
```

Також, ви можете використовувати функції `write` та `append` для створення текстових файлів з іншими розширеннями, наприклад ".txt", ".html", ".json" тощо. Просто змініть розширення у шляху до файлу, який передається функції.
