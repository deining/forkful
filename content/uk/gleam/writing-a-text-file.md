## Чому

Написання текстового файлу є необхідною задачею для людини, яка працює з програмуванням. Зберігання коду або документації у вигляді текстового файлу є зручним та зрозумілим способом організації інформації.

## Як

Простий спосіб створити текстовий файл у Gleam - це використовувати вбудовану функцію `file` та передати їй назву файлу та зміст, який потрібно записати. Наприклад:

```Gleam
let file_name = "hello.txt"
file.write(file_name, "Привіт світ!")
```

У цьому прикладі ми створюємо файл з назвою "hello.txt" та записуємо до нього рядок "Привіт світ!".

Якщо потрібно дописати до вже існуючого файлу, то можна використовувати функцію `file.append`. Наприклад:

```Gleam
file.append(file_name, " Я радий тебе бачити!")
```

Також, є можливість зчитувати зміст файлу за допомогою функції `file.read`. Наприклад, якщо ми хочемо вивести зміст файлу на екран, можна використати наступний код:

```Gleam
let content = file.read(file_name)
io.println(content)
```

## Глибоке занурення

Значення аргументів, які можна передавати функції `file.write`, `file.append` та `file.read` може бути різним в залежності від того, яку роль відіграє файл у проекті. Наприклад, можна передати другим аргументом не рядок, а список рядків, тоді в файл будуть записані всі елементи списку.

Також, важливо зазначити, що перед використанням цих функцій, потрібно імпортувати модуль `gleam/io/file`.
