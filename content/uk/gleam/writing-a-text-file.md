## Чому
Записувати текстовий файл може бути корисною та необхідною задачею для програмістів. Це дає можливість зберігати інформацію у структурованому форматі, що дозволяє легше зчитувати та обробляти її в майбутньому.

## Як
```Gleam
defmodule FileExample do 
  def write(file_name, content) do 
    File.write(file_name, content)

    IO.puts("Файл успішно записано!")
  end
end

FileExample.write("example.txt", "Привіт світ!")
```

В цьому прикладі ми створюємо модуль `FileExample`, який містить функцію `write`, яка приймає два аргументи: ім'я файлу та його вміст. За допомогою функції `File.write` ми записуємо вміст у файл і виводимо повідомлення про успішне завершення. 

Також, ми можемо використати функцію `File.append` для додавання даних до вже існуючого файлу. Наприклад:

```Gleam
File.append("example.txt", "Це додатковий текст.")
```

## Глибока поглядка
При записі текстових файлів, важливо враховувати кілька речей. По-перше, необхідно визначити шлях, за яким буде створюватись файл. Також, варто звернути увагу на формат зберігання даних у файлах, оскільки це може вплинути на подальшу обробку даних у програмі. 

Проте, з Gleam це дуже просто зробити. Другим аргументом `File.write` є опціональний параметр `mode`, який дозволяє визначити, як буде створюватися файл. Наприклад:

```Gleam
File.write("example.txt", "Новий текст", "w")
```

У цьому випадку, `w` вказує, що файл буде перезаписаний. Інші доступні значення для параметру `mode` можна знайти у [документації Gleam](https://gleam.run/docs/stdlib/file.html).
