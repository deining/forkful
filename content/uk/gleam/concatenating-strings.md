## Чому
Припустимо, що ви пишете програму для обробки даних і вам потрібно створити повідомлення, яке складається з декількох різних фрагментів тексту. Замість того, щоб просто вписувати кожен фрагмент окремо, ви можете скористатися конкатенацією рядків, яка дозволяє об'єднати кілька рядків в один, більш компактний фрагмент. Це може значно спростити ваш код та зробити його більш зрозумілим.

## Як це зробити
Щоб об'єднати рядки в Gleam, використовуйте символ плюс (+) між кожним рядком, який ви хочете об'єднати. Наприклад, якщо ви маєте два рядки "Привіт" та "Світе", ви можете скласти їх, написавши "Привіт" + "Світе". Результатом буде "ПривітСвіте".

```Gleam
let greeting = "Привіт"
let message = "Світе"

let fullMessage = greeting + message

// fullMessage = "ПривітСвіте"
```

Якщо ви хочете вставити пробіли між фрагментами, просто додайте їх як рядки у дужках, як показано нижче:

```Gleam
let greeting = "Привіт"
let message = "Світе"

let fullMessage = greeting + " " + message

// fullMessage = "Привіт Світе"
```

## Глибока занурення
Окрім простого об'єднання рядків, Gleam також має деякі зручні функції для роботи з рядками. Перш ніж конкатенувати рядки, ви можете перевірити їх довжину за допомогою функції `String.length`. Також ви можете використовувати функцію `String.slice` для вибору певної частини рядка.

```Gleam
let name = "Олег"
let greeting = "Привіт"

let fullMessage = greeting + " " + String.slice(name, 0, 2)

// fullMessage = "Привіт Ол"
```
