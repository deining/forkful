## Чому

З'єднання рядків є важливою частиною багатьох програм, оскільки дозволяє об'єднати різні тексти в одну, більш читабельну і зручну для обробки. Це дозволяє програмістам працювати з даними більш ефективно і зрозуміло. Використання конкатенації рядків в програмах допоможе покращити продуктивність та збільшити ефективність вашої роботи.

## Як це зробити

Щоб з'єднати рядки в Gleam, використовуйте оператор `<>`. Наприклад, якщо ви хочете об'єднати два рядки "Привіт" та "Світ!", ви можете написати:

```Gleam
"Привіт" <> "Світ!"

// Виводиться "ПривітСвіт!"
```

Якщо бажаєте додати інший рядок або змінну до рядка, використовуйте пробіл після першого рядка та перед наступним. Наприклад:

```Gleam
"Привіт" <> " " <> "Світ!"

// Виводиться "Привіт Світ!"
```

Глеам дозволяє також з'єднати більше двох рядків за допомогою багаторазового використання оператора `<>`. Наприклад:

```Gleam
Привіт" <> " " <> "Світ!" <> " " <> "Я" <> " " <> "Gleam!"

// Виводиться "Привіт Світ! Я Gleam!"
```

Також ви можете використовувати змінні разом з оператором `<>` для об'єднання рядків та значень змінних. Наприклад:

```Gleam
let слово = "Привіт"

"Привіт" <> "Світ!"

// Виводиться "ПривітСвіт!"
```

## Глибока копія

Ви можете використовувати особливий синтаксис, який називається "проміжний оператор", щоб об'єднати більше двох рядків. Це дозволяє вам з'єднати рядки в одну рядокову змінну, яку ви можете використовувати пізніше. Наприклад:

```Gleam
let речення = слово <> "Світ!" <> " " <> "Я" <> " " <> "Gleam!"
```

У цьому випадку, змінна `речення` містить рядок "Привіт Світ! Я Gleam!". Це дуже зручно, коли ви хочете з'єднати більше двох рядків і мати доступ до результату пізніше в коді.
