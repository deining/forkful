Маєте у планах порівнювати дві дати, але не знаєте, з чого почати? Дізнайтеся, як легко і просто це зробити за допомогою програмування Gleam!

## Навіщо

Порівняння двох дат може бути корисним у багатьох сценаріях, таких як знаходження різниці між датами, визначення того, яке число є більшим або меншим, або сортування даних за датою.

## Як це зробити

Нижче наведений приклад коду Gleam, який порівнює дві дати і виводить результат у консоль.

```Gleam
import gleam/time

let date1 = time.parse("2020-01-01", "%Y-%m-%d")
let date2 = time.parse("2020-02-01", "%Y-%m-%d")

if date1 > date2 {
  io.println("Перша дата пізніше за другу")
} else if date1 < date2 {
  io.println("Перша дата раніше за другу")
} else {
  io.println("Дати рівні")
}
```

Виведе:

```
Перша дата раніше за другу
```

## Глибока поглибина

Порівняння дат у Gleam базується на часовому типі даних, який зберігає дати та час у форматі єдиної точки з точністю до секунди. Це означає, що порівнювані дати повинні бути у межах одного часового поясу. Також, у коді можна використовувати функції для перетворення дат у різні формати або для отримання окремих частин дати, наприклад рік або місяць.
