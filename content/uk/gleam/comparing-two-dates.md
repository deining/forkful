Що & Чому?

Порівняння двох дат - це процес порівняння двох дат, щоб встановити, яка дата більша або менша. Це важливо для програмістів, оскільки вони часто працюють з датами та часовими мітками.

Як це зробити:

```
Gleam.compare_dates({2019, 1, 1}, {2020, 1, 1})
```

Результат: ```-1```

Щоб порівняти дві дати у Gleam, використовуйте функцію ```compare_dates```, передайте їй два параметри - перша дата та друга дата - у вигляді кортежів. Результатом буде число, що показує, яка з дат є більшою:

* ```-1``` - перша дата менша за другу
* ```0``` - дати рівні одна одній
* ```1``` - перша дата більша за другу.

Глибокий занурення:

* Історичний контекст: порівняння дат було важливим завданням для щоденного життя людей з давніх часів. У сучасному програмуванні, це дуже корисна функція для обробки дати та часу.
* Альтернативи: є багато інших мов програмування, які також мають вбудовану функцію для порівняння дат, наприклад, Java або Python.
* Деталі реалізації: у Gleam, порівняння дати використовує функцію ```erl_calendar.datetime_to_gregorian_days```, яка переводить дату в грегоріанський формат та далі порівнює їх за допомогою оператора «менше» та «більше».

Дивіться також:

Додаткову інформацію про порівняння дат у мові Gleam можна знайти в документації https://gleam.run/documentation/standard-library.html#compare_dates. Для порівняння дат у Java можна скористатися функцією ```compare```, яка повертає значення типу ```int```, де ```0``` - дати рівні, ```1``` - перша дата більша за другу, ```-1``` - перша дата менша за другу.
