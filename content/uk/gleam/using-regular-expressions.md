## Чому

Створення регулярних виразів є невід'ємною частиною розробки програмного забезпечення. Використання регулярних виразів дозволяє знаходити та змінювати певні шаблони в тексті, що робить процес обробки даних більш ефективним та швидким.

## Як

Gleam має вбудовану підтримку для регулярних виразів. Для того, щоб створити регулярний вираз, використовуйте ключове слово "regex" і заключіть шаблон у лапки. Нижче наведений приклад пошуку та заміни у тексті:

```Gleam
import gleam/regex

let text = "Привіт, світ!"

// Пошук по виразу "світ"
let result = regex.find(text, "світ")

IO.println("Результат пошуку: ${result}")

// Заміна "світ" на "Gleam"
let new_text = regex.replace(text, "світ", "Gleam")

IO.println("Новий текст: ${new_text}")
```

Результат виконання програми:

```
Результат пошуку: світ
Новий текст: Привіт, Gleam!
```

## Глибша підтримка регулярних виразів

Регулярні вирази також можуть використовуватися для більш складних завдань, як наприклад перевірка правильності введення дати або електронної адреси. Gleam має широкі можливості для роботи з регулярними виразами, зокрема можливість визначення повторень, групування шаблонів та багато іншого. Для більш детальної інформації зверніться до документації Gleam.
