## Чому

Коли ми програмуємо, часто нам потрібно замінити певний текст у нашому коді. Це може бути через те, що ми вирішили змінити назву змінної або функції, або виправити орфографічну помилку. Заміна тексту допомагає нам ефективно вносити зміни в наш код і уникати непотрібних помилок.

## Як замінювати текст

Є кілька способів заміняти текст у коді з використанням Gleam. Перший спосіб - це використовувати стандартну функцію `String.replace`, яка заміняє весь входящий текст на новий текст. Наприклад, ми можемо замінити слово "тест" на "проба" у рядку "Це тест":

``` Gleam
import gleam/string

let str = "Це тест"
let new_str = string.replace(str, "тест", "проба")

// Вихід: "Це проба"
```

Ще один спосіб - використовувати регулярні вирази для заміни певних частин тексту. Для цього можна використовувати модуль `re` з Gleam. Наприклад, ми можемо замінити всі голосні букви у рядку на "*":

``` Gleam
import gleam/re
import gleam/string

let str = "Це тест"
let new_str = re.replace(string.replace(str, "тест", "*"), "[aeiou]", "*")

// Вихід: "Ц* т*ст"
```

## Глибока аналітика

Заміна тексту в Gleam - це більше, ніж просто заміна слів у рядках. Ми також можемо шукати і заміняти текст у файловій системі з використанням пакету `gleam/filepath`. Цей пакет дозволяє нам перебирати файли і заміняти в них певні частини тексту.

Крім того, Gleam також підтримує функцію заміни у складніших структурах даних, таких як списки і кортежі. Це дозволяє нам замінювати текст у великих об'єктах і зберігати їх структуру.
