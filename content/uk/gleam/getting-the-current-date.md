## Чому

Багато програмістів, особливо тих, хто працює з базами даних або розробляє додатки, потребують точну, актуальну дату для своїх проектів. Отримання поточної дати є важливою функцією для будь-якої програми, щоб забезпечити правильну роботу та збереження даних. У цій публікації я розкажу вам про Gleam, інструмент для отримання поточної дати в програмуванні.

## Як

Отримання поточної дати у Gleam є досить простим завданням. Ви можете використовувати вбудовану функцію `Date.now()` для визначення поточного часу. Для цього нам потрібно імпортувати модуль для роботи з датами:

```Gleam
import gleam/datetime

fn main() {
  let current_date = datetime.Date.now()
  IO.print("Поточна дата: #{current_date}")
}
```

Ви можете отримати поточний час у різних форматах, використовуючи методи, такі як `datetime.Date.to_atom()`, `datetime.Date.to_iso8601()`, `datetime.Date.to_rfc3339()`, тощо. Для детальнішої інформації про різні формати дати, ви можете переглянути [документацію](https://gleam.run/modules/datetime#date). 

## Поглиблене дослідження

Отримання поточної дати у Gleam базується на стандарті Джуліанського календаря, який використовується у багатьох мов програмування. Gleam дозволяє просто виконувати різні операції з датами, такі як обчислення часу, переведення з одного формату в інший, розрахунок різниці між двома датами, тощо. Також, Gleam дозволяє працювати з часовими поясами та імпортувати зовнішні бібліотеки для роботи з датами.
