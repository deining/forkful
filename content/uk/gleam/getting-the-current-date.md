## Чому

Получення поточної дати є важливою функцією для багатьох програм, оскільки дата може впливати на різні аспекти програми, такі як терміни виконання, розклади, та інші.

## Як це зробити

Програмування на Gleam дозволяє легко отримати поточну дату за допомогою вбудованої бібліотеки `Time`.

```Gleam
import Time

pub fn main() {
  current_date = Time.Now()
  Time.format(current_date, "dd/mm/yyyy")
}
```

Вихідний код виведе поточну дату у форматі "день/місяць/рік" (наприклад, "25/05/2021").

## Глибоке занурення

Подальше дослідження функції `Time.Now()` показує, що бібліотека використовує внутрішню функцію для отримання поточного часу з операційної системи. Також, за допомогою спеціальних опцій форматування, можна додати до дати і часу іншу інформацію, наприклад, часовий пояс чи дня тижня.
