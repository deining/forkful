## Чому

Для чого взагалі потрібно конвертувати дату в рядок? Це може здатися надто складним та займати зайва відверів роботі програміста. Але насправді, перетворення дати в рядок має свої цікаві застосування, особливо, коли ми працюємо з датами у коді.

## Як це зробити

Щоб конвертувати дату в рядок у Gleam, ми використовуємо функцію `format_datetime` з модулю `gleam/time`.

```Gleam
import gleam/time

let birthday = { day: 12, month: gleam/time/December, year: 1995 }

let formatted_date =
  gleam/time/format_datetime("{day}.{month}.{year}", birthday)

IO.print(formatted_date)
```

В даному прикладі ми використовуємо патерн `{day}.{month}.{year}` для конвертації дати у формат "День.Місяць.Рік". Після виконання коду, ми отримаємо наступне виведення: `12.12.1995`.

Можливостей для форматування інших дат є безліч, їх можна дізнатись у документації Gleam.

## Заглиблення

При роботі з датами у Gleam ми працюємо з типом `gleam/time/DateTime`, який містить інформацію про день, місяць, рік та час. Цей тип доволі зручний у використанні, оскільки має багато функцій для роботи з ним.

Наприклад, для отримання поточної дати та часу, ми можемо використовувати функцію `now`:

```Gleam
let current_datetime = gleam/time/now()

IO.inspect(current_datetime) // Результат: gleam/time/DateTime
```

Крім того, у модулі `gleam/time` є функції для порівняння дат, обрахунку різниці між ними та багато інших корисних операцій.
