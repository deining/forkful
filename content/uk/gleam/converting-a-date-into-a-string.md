## Why

Використання типу даних "date" в програмуванні є дуже поширеним. Іноді нашому користувачу потрібно побачити дату в різних форматах, наприклад у вигляді рядка. Тому, вивчення того, як перетворити дату в рядок є важливим для оптимізації наших програм.

## How To

### ```Gleam``` для конвертації дати в рядок

Щоб перетворити дату у рядок використовуємо вбудовану функцію ```format``` у Gleam. Приклад коду наведено нижче, де дата буде перетворена у форматі "ДД-ММ-РРРР".

```Gleam
import gleam/date

let my_date = 2022-01-01
let converted_date = date.format(my_date, "{DD}-{MM}-{YYYY}")
```

У цьому прикладі, ми імпортуємо модуль ```date``` Gleam та створюємо змінну ```my_date```, яка відповідає початковій даті. Потім використовуємо функцію ```format```, яка приймає два аргументи: дату та формат, у який ми хочемо перетворити дату. Наступною змінною є ```converted_date```, яка буде містити дату у вказаному форматі.

### Результат

При виконанні коду з нашого прикладу, змінна ```converted_date``` буде містити рядок "01-01-2022".

## Deep Dive

### Форматування дати

У функції ```format```, ми можемо використати різні формати для змінної дати. Наприклад, у прикладі вище ми використовували ```{DD}``` для виведення числа дня в місяці з двома цифрами та ```{MM}``` для виведення числа місяця з двома цифрами. Також, можна використати ```{D}``` та ```{M}``` для виведення числа дня та місяця з одною цифрою.

Нижче наведено список деяких інших форматів, які можна використовувати для перетворення дати в рядок:

- ```YYYY``` - повний рік (наприклад, 2022)
- ```YY``` - останні дві цифри року (наприклад, 22)
- ```MMMM``` - повний назва місяця (наприклад, січень)
- ```MMM``` - скорочена назва місяця (наприклад, січ)
- ```MM``` - число місяця (наприклад, 01)
- ```DD``` - число дня в місяці (наприклад, 01)
- ```DN``` - номер дня в тижні (наприклад, 1 для понеділка, 2 для вівторка та т.д.)
