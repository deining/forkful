## Чому

Іноді, коли працюємо з текстовими даними, нам потрібно видалити певні символи, які відповідають заданому шаблону. Це може бути корисно, наприклад, для очищення тексту перед аналізом або написанням фільтрів для пошуку.

## Як це зробити

Завдання видалення символів за певним шаблоном можна легко виконати в Gleam за допомогою функції `String.replace`. Давайте подивимося на приклад:

```Gleam
let text = "АБВ123ГДЕ456ЄЖЗ";
let cleaned_text = String.replace(text, "[0-9]", "");
```

У цьому прикладі, ми задаємо змінну `text`, якій присвоюємо рядок з кирилицею та цифрами. Потім, ми використовуємо функцію `String.replace` для видалення усіх цифр з рядку. Результат зберігається у змінну `cleaned_text`, яку ми можемо вивести на екран, щоб переконатися, що символи були успішно видалені:

```Gleam
IO.print(cleaned_text); // Виводиться "АБВГДЕЄЖЗ"
```

Також, до функції `String.replace` можна додати додатковий аргумент, що вказує кількість символів, які потрібно видалити за раз. Наприклад, якщо ми хочемо видалити тільки перші три цифри з рядку, ми можемо написати наступне:

```Gleam
let text = "АБВ123ГДЕ456ЄЖЗ";
let cleaned_text = String.replace(text, "[0-9]", "", 3);
IO.print(cleaned_text); // Виводиться "АБВГДЕ456ЄЖЗ"
```

При цьому, у змінну `cleaned_text` будуть записані оригінальні символи, із набором перших трьох цифр видаленими.

## Детальніше

Функція `String.replace` приймає три обов'язкових аргументи: рядок, шаблон та замінювальний рядок. Рядок, який потрібно замінити, передається першим аргументом, шаблон - другим, а замінювальний рядок - третім. При цьому, шаблон задається як рядок з регулярним виразом.

Якщо необхідно видалити декілька символів, які відповідають шаблону, можна використовувати функцію `String.replace` у циклі. Також, ця функція підтримує використання функціонального стилю програмування за допомогою динамічних аргументів.
