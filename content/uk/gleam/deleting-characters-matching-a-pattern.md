## Чому

Існує багато ситуацій, коли необхідно видалити певні символи з тексту, відповідно до заданого шаблону. Наприклад, це може бути корисно при очищенні вхідних даних або при форматуванні тексту для подальшої обробки. У цьому блозі ми розглянемо, як за допомогою мови програмування Gleam видалити символи, які відповідають заданому шаблону.

## Як це зробити

Для початку нам потрібно імпортувати модуль `gleam/regex`, щоб мати доступ до функцій, пов'язаних з регулярними виразами.

``` Gleam
import gleam/regex
```

Представимо, що у нас є рядок з декількома цифрами та символами, і ми хочемо видалити всі символи, які не є цифрами. Спочатку нам потрібно створити регулярний вираз, який відповідає нашій потребі. Наприклад, `[^0-9]` вибере всі символи, які НЕ є цифрами. Для цього ми будемо використовувати функцію `compile_regex`, яка приймає регулярний вираз у вигляді рядка та повертає об'єкт регулярного виразу, необхідний для подальших операцій.

``` Gleam
let regex = regex.compile_regex("[^0-9]")
```

Далі, нам потрібно застосувати цей регулярний вираз до нашого рядка. Для цього ми використовуємо функцію `replace_all`, яка приймає регулярний вираз та рядок для обробки. В результаті отримаємо новий рядок без тих символів, які відповідають нашому шаблону.

``` Gleam
let input = "Текст 123 для прикладу"
let output = regex.replace_all(regex, input, "")
```

Тепер у змінній `output` ми отримаємо наступне значення: `123`

## Глибший розгляд

Регулярні вирази - це потужний інструмент для роботи з текстом, тому ми рекомендуємо ознайомитися з детальними документами та прикладами використання регулярних виразів у мові Gleam. Також варто зазначити, що функція `replace_all` може приймати додаткові аргументи, які дозволяють вказувати, наприклад, символ-заміну для видалених символів.
