Що & чому?
У програмуванні іноді виникає потреба видаляти символи, які відповідають певній шаблону. Це може бути корисно для перетворення даних або очищення вхідних даних перед подальшою обробкою. Програмісти використовують цю функцію, щоб полегшити свої завдання та зекономити час.

Як використовувати:
Нижче наведено приклад коду на мові ```Gleam```, який показує, як видалити всі голосні літери з рядка "Привіт, світ!". Результат виводу буде "Првт, свт!".

```Gleam
let str = "Привіт, світ!"
let vowels = "аеєиіоюяАЕЄИІОУЮЯ"
let filtered = str
  |> String.filter((char) -> String.contains(char, vowels) == false)

Logger.info(filtered) // Output: Првт, свт!
```

Глибинний погляд:
Історичний контекст - видалення символів, які відповідають певному шаблону, було вперше введено в мові програмування Perl. Ця функціональність також є в більшості сучасних мов програмування, таких як Java, Python та JavaScript. Альтернативою може бути використання циклів для перебору символів та їх перевірки, але це може призвести до більш складного та менш ефективного коду.

Крім того, варто зазначити, що видалення символів за шаблоном може мати різні результати, якщо вхідні дані відрізняються. Наприклад, для рядка "Привіт, світ!" результат буде "Првт, свт!", а для рядка "Hello, world!" - "Hll, wrld!".

Підсумовуючи, видалення символів за шаблоном - корисна функція в програмуванні, яка дозволяє швидко та зручно обробляти дані. Це може бути використано для різних завдань, включаючи фільтрацію даних та очищення вхідних даних перед подальшою обробкою.
