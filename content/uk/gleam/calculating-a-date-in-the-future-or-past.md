#Що і чому?

Обчислення дати в майбутньому або минулому є важливим завданням для програмістів, оскільки це дозволяє встановити потрібну для певної операції дату. Наприклад, ми можемо створити програму, що надсилає сповіщення на певну дату або виконує дії, які залежать від дати.

#Як:

```Gleam
import Time

let date = Time.add_days(Time.now(), 7)

IO.println(date)
```

Результат:

```
2020-08-21
```

```Gleam
import Time

let date = Time.subtract_days(Time.now(), 14)

IO.println(date)
```

Результат:

```
2020-08-07
```

#Поглиблене дослідження:

(1) Історичний контекст:

Непросто було колись обчислити дату в майбутньому або минулому. Перші розробники повинні були працювати зі складними календарями та формулами, що утруднювало роботу з датами. Завдяки розробкам у галузі комп'ютерних наук, нині ми можемо легко обчислювати дати.

(2) Альтернативи:

Ви можете використовувати інші мови програмування для обчислення дати в майбутньому або минулому. Наприклад, JavaScript має вбудовану функцію для роботи з датами. Однак, Gleam надає простий та чистий підхід до цієї задачі.

(3) Деталі реалізації:

Gleam використовує модуль Time для обчислення дати в майбутньому або минулому. Це вбудований модуль, що пропонує функції, такі як add_days та subtract_days для додавання або віднімання кількості днів від поточної дати.
