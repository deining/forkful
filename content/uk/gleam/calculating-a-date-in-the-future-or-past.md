# Чому

Розрахунок дати у майбутньому або минулому може бути корисним для планування подій, створення напоминалок або з'ясування проміжку часу між двома подіями.

## Як

Кодування дати у майбутньому або минулому в Gleam є простим завданням. Для розрахунку дати використовуйте вбудований модуль `DateTime`, який надає багато корисних функцій для роботи з датами. Для прикладу, для обчислення дати, що буде через 1 рік, використовуйте функцію `add`, яка додає певну кількість днів до поточної дати.

```Gleam
import DateTime

let one_year_from_now = DateTime.now() |> DateTime.add([days => 365])
```

В результаті отримаємо дату, яка буде через рік від поточного моменту.

## Глибокий захід

Хоча Gleam має багато корисних функцій для роботи з датами, проте варто пам'ятати кілька моментів, щоб уникнути помилок. Наприклад, функція `add` приймає список кортежів, де перший елемент є типом елементу, який потрібно додати (у нашому випадку "дні"), а другий - кількість цих елементів. Також важливо врахувати часовий пояс і використовувати правильні функції для конвертації дати у різні формати. Для детальнішої інформації зверніться до офіційної документації Gleam.
