## Що це таке і навіщо це потрібно?

Перевірка наявності директорії - це процес, коли програміст перевіряє, чи існує конкретна директорія на комп'ютері або сервері. Це корисний інструмент, який дозволяє впевнитися, що необхідна директорія існує перед тим, як приступати до роботи з файлами в ній.

## Як це зробити:

Використовуючи Gleam, ми можемо легко перевірити наявність директорії за допомогою функції ```exists_directory``` та задати шлях до неї як аргумент. Наприклад:

```Gleam
let directory = "/home/user/folder"
if exists_directory(directory) {
    println("Директорія існує!")
} else {
    println("Директорія не знайдена!")
}
```

В результаті наша програма виведе повідомлення про те, чи існує директорія за вказаним шляхом.

## Вдігляд у глибину:

Історичний контекст: в перших мовах програмування, таких як C та Fortran, не було зручного способу перевірити наявність директорії. З часом, було розроблено різні методи, проте вони мають свої недоліки. Завдяки розвитку сучасних мов програмування, таких як Gleam, перевірка наявності директорії стала простішою та зручною.

Альтернативи: наявність директорії можна перевірити за допомогою командного рядка або з використанням пакетів, проте використання вбудованих функцій мов програмування є більш зручним та надійним варіантом.

Деталі реалізації: у Gleam, функція ```exists_directory``` використовує системний виклик для перевірки наявності директорії та повертає булеве значення, що показує, чи існує вказана директорія.
