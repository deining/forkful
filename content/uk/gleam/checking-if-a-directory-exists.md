## Чому

У цій статті ми поговоримо про перевірку існування директорії. Це важлива задача для багатьох програм, які працюють з файловою системою. Завдяки перевірці існування директорії, програма може виконати певні дії, наприклад, створити нову директорію або перейти до вже існуючої.

## Як

```Gleam
// Приклад коду для перевірки існування директорії
import gleam/io.FileSystem.{directory_exists, list}

// Перевірка існування директорії
let directory = "/home/username/documents/"
let exists = directory_exists(directory)

// Перевірка результату
if exists {
    // Виведення списку файлів у директорії
    let files = list(directory)
    io.print("У директорії є файлів:", length(files))
} else {
    // Створення нової директорії
    io.print("Директорія не існує, створюємо її")
    io.make_directory(directory)
}
```

В даному прикладі ми використовуємо функцію `directory_exists` з модуля `FileSystem`, щоб перевірити чи існує директорія за шляхом, який ми вказали. Результат перевірки ми зберігаємо у змінну `exists`. Якщо директорія існує, ми отримуємо список файлів у цій директорії за допомогою функції `list` і виводимо кількість файлів. Якщо ж директорія не існує, ми створюємо її за допомогою функції `make_directory`.

## Deep Dive

Перевірка існування директорії є важливою задачею для багатьох програм на Gleam. У разі, якщо програма отримує неправильний шлях до директорії, може виникнути помилка. Тому, перед тим як працювати з директорією, потрібно перевірити чи існує вона.

Простий спосіб перевірки існування директорії - це використовувати функцію `directory_exists` з модуля `FileSystem`. Однак, у більш складних програмах можна використовувати інші методи перевірки, наприклад, перевіряти права доступу до директорії або перевіряти наявність піддиректорій.
