## Чому

Перевірка існування директорії є важливим елементом у програмування, оскільки вона дозволяє перевірити наявність потрібних файлів та структуру файлової системи. Це також допомагає уникнути помилок і крахів програми.

## Як використовувати

Для перевірки існування директорії використовується функція `exists` з модуля `os` в мові Gleam.

```Gleam
import os

// Шлях до директорії, яку потрібно перевірити
let dir = "home/user/documents"

// Виклик функції `exists`
let exists = os.exists(dir)

// Виведення результату
pub fn main() {
  if exists {
    // Директорія існує
    println("Директорія існує!")
  } else {
    // Директорія не існує
    println("Директорія не існує.")
  }
}
```

Вивід на екрані буде залежати від того, чи існує директорія "home/user/documents" на вашій системі.

## Глибокий аналіз

Для більш тонкого контролю над перевіркою існування директорії можна використовувати функцію `stat` з модуля `os`. Вона повертає об'єкт `Stat` з інформацією про файл або директорію. З цим об'єктом можна виконати різні перевірки, наприклад, перевірити розмір директорії.

```Gleam
import os

// Шлях до директорії, яку потрібно перевірити
let dir = "home/user/documents"

// Виклик функції `stat`
let dir_stat = os.stat(dir)

// Виведення розміру директорії
pub fn main() {
  // Перевіряємо чи існує директорія
  case dir_stat {
    Ok(stat) -> {
      // Директорія існує, виводимо її розмір
      println("Розмір директорії: {}", stat.size)
    }
    Err(error) -> {
      // Помилка, директорія не існує
      println("Директорія не існує.")
    }
  }
}
```

Результатом виконання цього коду буде розмір директорії "home/user/documents" або повідомлення про те, що директорія не існує.
