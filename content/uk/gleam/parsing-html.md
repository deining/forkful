Знаходження і витягування інформації з HTML-сторінок - це важлива задача для багатьох програмних проектів, починаючи від веб-скрапінгу і закінчуючи автоматизацією веб-контенту. У цій статті ми поговоримо про те, чому важливо вміти розбирати HTML і як це зробити за допомогою мови програмування Gleam. 
## Чому 
Розбір HTML-коду дозволяє нам здійснювати різноманітні завдання, пов'язані з отриманням даних з веб-сторінок. Наприклад, ми можемо витягнути інформацію про продукти з онлайн-магазину, отримати дані про погоду з веб-сайту або навіть автоматизувати заповнення форм на веб-сторінках. Без вміння аналізувати HTML-код, наші можливості обмежені, тому це важливий навик для будь-якого програміста. 
## Як це зробити 
Для того, щоб отримати дані з HTML-коду за допомогою Gleam, нам потрібно використовувати модуль ```Html``` і його функції. Наприклад, ми можемо використовувати функцію ```Html.parse``` для аналізу HTML-коду і отримання дерева вузлів. Потім ми можемо використовувати різні методи для навігації по дереву та вибірки потрібних елементів, наприклад, функцію ```Html.select``` для вибірки елементів за допомогою CSS-селекторів. Нижче наведений приклад коду для витягування заголовка з HTML-сторінки: 
```Gleam
content = """
<html>
  <head>
    <title>Gleam статті</title>
  </head>
  <body>
    <h1>Навчання Gleam мові програмування</h1>
    <p>Дякуємо за читання цих статей!</p>
  </body>
</html>
"""
tree = Html.parse(content)
selector = "h1"
title = tree |> Html.select(selector)
```
В результаті, змінна ```title``` буде містити елемент з заголовком сторінки. Ми також можемо отримати сам текст заголовка ще однією функцією:
```Gleam
text = Html.text(title)
```
Отриманий результат буде містити строку з текстом "Навчання Gleam мові програмування".
