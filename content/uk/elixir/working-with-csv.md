## Чому

Програмування CSV є важливою навичкою для людей, які працюють з великими об'ємами даних, такими як бізнес-аналітики, дата-інженери та розробники програмного забезпечення. Використання Elixir для обробки CSV є ефективним способом працювати з цим типом даних.

## Як це зробити

Для початку роботи з CSV файлами в Elixir, ми можемо використати модуль `File`, який надає широкий набір функцій для роботи з файлами. Для відкриття файлу CSV нам потрібно використати `File.open` та передати шлях до файлу та режим доступу `:utf8`. Наприклад:

```Elixir
File.open("data.csv", [:read, :utf8])
```

Після цього, ми можемо використати модуль `CSV` для роботи з даними в нашому файлі. Насамперед, нам потрібно імпортувати цей модуль у нашому коді:

```Elixir
import CSV, only: [parse: 2]
```

За допомогою функції `parse`, ми можемо прочитати наш CSV файл та отримати список списків з даними. Наприклад:

```Elixir
[["John", "Doe", "john.doe@example.com"],
["Jane", "Smith", "jane.smith@example.com"]]
```

## Глибоке дослідження

CSV є одним з найпоширеніших форматів для зберігання табличних даних. За допомогою Elixir, ми можемо ефективно робити операції з цим типом даних, такі як сортування та фільтрація.

Крім того, модуль `CSV` надає можливість робити асинхронні операції з даними, що дозволяє ефективно працювати з великими об'ємами даних. Ми також можемо використовувати різні налаштування, такі як роздільник та кодування, для роботи з різними типами CSV файлів.
