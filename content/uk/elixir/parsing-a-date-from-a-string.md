## Що і Для чого?

Парсинг дати з рядка - це процес перетворення інформації про дату, записану у вигляді рядка, у конкретний формат, який може бути зрозумілим для програми. Програмісти зазвичай використовують парсинг дати з рядка, щоб зробити обробку дат та часу більш зручною та ефективною для своїх програм.

## Як це зробити:

Програмувальне середовище Elixir надає декілька зручних функцій для парсингу дати з рядка. Нижче наведено приклади коду та вихідних даних для різних форматів дати.

```elixir
iex> Date.from_iso8601("2021-05-20")
{:ok, ~D[2021-05-20]}
```

```elixir
iex> Date.from_string("May 20, 2021", "{{month_name}} {{day}}, {{year}}")
{:ok, ~D[2021-05-20]}
```

```elixir
iex> DateTime.from_iso8601("2021-05-20T12:00:00Z")
{:ok, ~U[2021-05-20T12:00:00Z]}
```

```elixir
iex> Time.from_iso8601("2021-05-20T12:00:00Z")
{:ok, ~T[12:00:00Z]}
```

## Глибший погляд:

Перш ніж існували сучасні програми, користувачі звикли до запису дати за допомогою більш незрозумілих форматів, таких як "DDMMYYYY" або "MMDDYY". Парсинг дати з рядка допомагає зробити перетворення з цих форматів в нормалізований вигляд більш простим та ефективним. Можливою альтернативою для парсингу дати з рядка є використання зовнішніх бібліотек, які можуть мати додаткові функціональні можливості.
