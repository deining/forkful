## Чому

Конкатенація рядків є важливою частиною більшості програм на Elixir. Вона дозволяє об'єднувати різні рядки тексту для створення більш складних повідомлень або структур даних.

## Як

Конкатенація рядків в Elixir здійснюється за допомогою оператора `<>`, який складає два рядки разом. Наприклад, якщо ми маємо два рядки "Привіт, " і "Світ", результат конкатенації буде "Привіт, Світ". Давайте подивимося на кілька прикладів:

```elixir
greeting = "Привіт, "
name = "Світ"

full_greeting = greeting <> name
IO.puts full_greeting
```

Вивід: `Привіт, Світ`

Ми також можемо використовувати змінні, які містять рядки, як аргументи у виклику функції `<>`. Наприклад:

```elixir
message = "Сьогодні ми маємо "
weather = "хорошу погоду"

IO.puts(message <> weather <> ".")
```

Вивід: `Сьогодні ми маємо хорошу погоду.`

## Глибше

У Elixir використовується принцип "конкатенаційної оптимізації" для ефективного об'єднання рядків. Це означає, що при використанні оператора `<>` Elixir буде спробувати мінімізувати копіювання та виділення пам'яті для створення нового рядка, а замість цього буде змінювати вже існуючий. Це допомагає зберегти ресурси та покращити продуктивність.

Також варто зазначити, що використання `<>` не обмежується тільки рядками, але може бути використане для об'єднання будь-яких типів даних, які можуть бути перетворені до рядка. Наприклад, списки:

```elixir
first_list = [1, 2, 3]
second_list = [4, 5, 6]

IO.puts("Два списки разом: " <> first_list <> second_list)
```

Вивід: `Два списки разом: [1, 2, 3][4, 5, 6]`
