##Чому

Пошук довжини рядка є важливою задачею у багатьох програмах, особливо у веб-додатках. Це дозволяє нам визначити кількість символів у рядку, що може бути корисно при перевірці коректності введеного користувачем пароля, лічильників символів у повідомленнях та багатьох інших випадках. Тому, вивчення як знаходити довжину рядка є важливим для будь-якого, хто програмує.

##Як це зробити

Існує багато способів знаходження довжини рядка в Elixir, але ми розглянемо два найпоширеніших: використання функції `String.length/1` та використання методу злиття `|>`.

```Elixir
# Використання функції String.length/1
string = "Привіт, світе!"
length = String.length(string)
IO.puts("Довжина рядка: #{length}")

# Використання методу злиття
string
|> String.length()
|> IO.puts("Довжина рядка: #{&1}")
```

В обох прикладах відбувається одне й те ж: отримання довжини рядка та виведення її на екран.

##Глибокий занурення

В Elixir рядки - це список символів, тому довжина рядка визначається кількістю елементів у цьому списку. Це означає, що для знаходження довжини рядка використовується типове метод січення списку, а не обраний символ у рядку. Це є важливим для розуміння, оскільки це впливає на час виконання наших програм.
