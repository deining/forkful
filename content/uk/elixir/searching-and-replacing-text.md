## Чому 

Заміна тексту є важливою частиною програмування, оскільки дозволяє швидко та ефективно змінювати великі обсяги даних за допомогою програмного коду. При пошуку та заміні тексту можна також автоматизувати процеси та уникнути помилок в ручному режимі.

## Як

Щоб замінити текст в Elixir, використовуйте функцію `String.replace/3` та передайте у неї три аргументи: початковий рядок, шаблон для пошуку та заміщуваний текст. Наприклад, якщо ви хочете замінити всі пропуски на тире, це можна зробити за допомогою наступного коду:

```Elixir
input = "Це тестовий текст"
output = String.replace(input, " ", "-")
IO.puts(output) # Виведе "Це-тестовий-текст"
```

Більш складні шаблони пошуку можна виразити за допомогою регулярних виразів. Наприклад, якщо ви хочете замінити всі голосні літери на верхні регістри, ви можете використовувати регулярний вираз `[aeiou]` та функцію `String.upcase/1`:

```Elixir
input = "Це є тестовий текст"
output = String.replace(input, ~r/[aeiou]/, &String.upcase/1)
IO.puts(output) # Виведе "ЦЕ Є ТЄСТОВИЙ ТЄКСТ"
```

## Глибше вдивлюємося

Заміна тексту в Elixir базується на паттернах, які можна задавати за допомогою регулярних виразів або функцій-зворотнього виклику (callback functions). Також є можливість використовувати опції для більш точного вибору тексту для заміни, а також для виконання декількох замін одночасно.
