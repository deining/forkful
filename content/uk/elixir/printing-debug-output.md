## Причина
Чому б хтось хотів виводити відлагоджувальний виведення на екран? Друкування відлагоджувального виведення є корисним інструментом для отримання важливої інформації про виконання програми, що допомагає виявити і виправити можливі помилки.

## Як це зробити
Існує кілька способів виводити відлагоджувальний виведення в Еліксирі. Один з них - використання функції `IO.inspect/2` для виведення змінної або об'єкта на екран. Також можна використовувати команду `IO.puts/1` для виведення рядка тексту. Приклад коду з використанням обох функцій:

```Elixir
name = "Іван"
IO.inspect(name) # виведе "Іван"
IO.puts("Привіт, #{name}!") # виведе "Привіт, Іван!"
```

Зверніть увагу, що пропускаються крапки ком то виведених значень. Це може бути корисно для організації виведення у відповідності до умов та результатів.

## Глибоке занурення
На перший погляд, друкування відлагоджувального виведення може здаватися простим завданням. Але в Еліксирі є багато варіантів та можливостей для налаштування виведення. Наприклад, можна додати додаткові параметри до функції `IO.inspect/2`, щоб виводити лише певні властивості об'єкта або надавати назви змінних для полегшення розуміння виведення.

Для більш детального дослідження можна також ознайомитися з додатковими модулями, наприклад `Logger` та `IO.List`. Якщо ви пишете інтерактивну програму в Еліксирі, можете використати функцію `IEx.pry/0` для зупинки виконання та перегляду поточного стану програми.
