## Чому

Отримання поточної дати є важливою частиною програмування, оскільки вона дозволяє вам працювати з актуальною інформацією про час.

## Як

Існує кілька способів отримати поточну дату в Ерликсир:

```elixir
# Використання бібліотеки Date
import Date

Date.utc_today()
#=> ~U[2021-01-20]

# Використання функції System.time()
System.time(:seconds) # => 1611164728.6417778

# Використання директиви `now` в шаблоні подій GenEvent
GenEvent.now() # => {{2021, 1, 20}, {9, 18, 48}}
```

Незалежно від того, який метод ви оберете, ви отримаєте поточну дату або час в зручному форматі.

## Глибші занурення

В Ерликсир є багато інструментів для роботи з датами, таких як модуль `Calendar` для взаємодії з календарними об'єктами, та бібліотека `timex` для розширення можливостей роботи з датами. Ви можете також використовувати функції з модуля `DateTime` для отримання більш широкого спектру інформації про поточну дату.
