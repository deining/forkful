# Почему

Конвертация даты в строку является важным шагом при работе с датами в программировании. Это позволяет представить дату в удобном для обработки формате и использовать ее в различных приложениях. 

# Как это сделать?

Конвертация даты в строку в языке Elixir происходит посредством использования функции `DateTime.to_string`. Для этого необходимо указать дату, которую нужно конвертировать, и формат, в котором она будет представлена. Например, для конвертации текущей даты в формате `dd-mm-yyyy` используется следующий код:

```elixir
DateTime.to_string(DateTime.utc_now(), "{0D}-{0M}-{0Y}")
```
Результатом будет строка вида `13-12-2021`.

# Глубокое погружение

Функция `DateTime.to_string` имеет множество параметров, которые позволяют настраивать формат вывода даты. Например, можно указать часовой пояс или формат времени (24-часовой или 12-часовой). Кроме того, в языке Elixir есть еще одна функция, `DateTime.to_iso8601`, которая позволяет конвертировать дату в строку, соответствующую международному стандарту ISO 8601.
