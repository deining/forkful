##Чому

Якщо ви хочете завершити свій проект Elixir і зробити його більш користувацьких-дружнім, ви, можливо, захочете використовувати великі літери для деяких текстів. Наприклад, ви можете використовувати великі літери для им'я користувача, створеного на основі введених даних. Це може допомогти відокремити істотні елементи і зробити їх більш помітними для користувача. У цьому блозі ми покажемо вам, як легко отримати такий результат за допомогою Elixir.

##Як це зробити

Як і багато інших завдань в Elixir, заздалегідь заздодіжно, щоб отримувати великий тимчасовий аспект коду. Таким чином ми можемо зривати кожен елемент та виконувати деякі дії з ним, щоб отримати бажаний результат. Ми можемо використовувати метод ```String.upcase/1``` для зміни регістру всіх символів в строкі на великі. Ось невеликий код, який покаже це на прикладі введеного тексту:

```Elixir
input = "Вітаю у нашій програмі, користувач!"

IO.puts String.upcase(input)
```

Виходом буде ```"ВІТАЮ У НАШІЙ ПРОГРАМІ, КОРИСТУВАЧ!"```, як ми й очікували. Тепер ми можемо використовувати цю функцію для будь-якої строкової змінної, щоб отримати великі літери. Це просто, ефективно і дозволяє полегшити роботу зі строками.

##Глибокий пір

У глибину цієї проблеми полягає те, що функція ```String.upcase/1``` дійсно використовує патерн місяця для зміни регістру символів. Приймаючи вхідний аргумент з типом ```String.t```, ця функція повертає строку з використаними великими літерами. Вона також буде враховувати будь-які спеціальні символи чи числа, тому ви можете бути впевнені, що ваша інформація залишиться незмінною.
