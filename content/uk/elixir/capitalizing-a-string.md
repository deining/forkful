Що та чому?

Капіталізація рядка - це процес зміни першої літери в слові на велику. Програмісти зазвичай це роблять для полегшення сприйняття тексту та зручного форматування виводу.

Як це зробити?

Щоб капіталізувати рядок в Elixir, використовуйте функцію `String.capitalize/1`, передаючи в неї вхідний рядок як аргумент. Наприклад:

```Elixir
String.capitalize("ukraine")
```

Вивід:

```Elixir
"Ukraine"
```

Щось більш складне? Можете використати повну функцію `String.capitalize/2`, передавши другим аргументом потрібну мову. Наприклад:

```Elixir
String.capitalize("обережно!", :ukrainian)
```

Вивід:

```Elixir
"Обережно!"
```

Глибше копання

Історичний контекст: Капіталізація рядків була введена для того, щоб зробити текст більш зрозумілим та читабельним, особливо у великих текстових файлах.

Альтернативи: Іншим способом капіталізувати рядок є використання регулярних виразів для зміни першої літери на велику.

Деталі реалізації: Функція `String.capitalize/1` використовує вбудований алгоритм для зміни першої літери на велику в рядку.
