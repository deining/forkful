## Чому

Генерація випадкових чисел - це важлива складова у багатьох програмах, таких як ігри, криптографічні системи та багато іншого. За допомогою Elixir, ви можете легко створювати випадкові числа для використання у своїх проектах.

## Як

```elixir
# Генеруємо випадкове число від 1 до 10:
rand(1..10)

# Генеруємо випадкове число з плаваючою точкою від 0 до 1:
rand()
```

Ви також можете використовувати `Enum.random/1` для отримання випадкового елемента зі списку:

```elixir
names = ["Олексій", "Марія", "Андрій", "Софія"]
Enum.random(names)
```

Ви можете задавати свої власні параметри за допомогою `:seed` та `:pseudo`:

```elixir
rand(1..10, seed: 123)
rand(1..10, pseudo: :uniform)
```

## Глибоке поглиблення

Під капотом, `rand/1` використовує рандомизатор Erlang для генерації випадкових чисел. Це значить, що ви можете використовувати будь-які Erlang функції для генерації випадкових чисел, і вони будуть працювати так само, як і в Elixir.

Також, в Elixir є вбудована бібліотека `:random` для більш складної роботи з випадковими числами. Вона містить функції для генерації послідовностей та управління генератором випадкових чисел.
