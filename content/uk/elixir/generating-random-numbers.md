## Чому

Написання випадкових чисел може бути корисним для широкого кола завдань, таких як тестування програм, генерація унікальних номерів або створення випадкових паролів.

## Як це зробити

Почати роботу з генерацією випадкових чисел в Ерланг легко з допомогою модуля Random. Перш за все, необхідно імпортувати модуль:

```Elixir
import Random
```

Тепер ми можемо скористатися декількома функціями для генерації випадкових чисел. Функція `uniform/1` повертає випадкове число з діапазону між 0 та заданим числом, наприклад:

```Elixir
Random.uniform(10) # 8.137425318748536
```

Функція `uniform/2` дозволяє вказати діапазон, в якому буде генеруватися число:

```Elixir
Random.uniform(5, 10) # 7.436863002908513
```

Крім того, можна використовувати функцію `seed/1` для встановлення початкового значення генератора випадкових чисел:

```Elixir
Random.seed(42)
Random.uniform(10) # 3.987085855901226
```

## Глибші дослідження

Модуль Random заснований на алгоритмі Mersenne Twister і може бути налаштований для додаткового контролю над генерацією випадкових чисел. Можете дізнатися більше про цей алгоритм та його параметри в [документації](https://hexdocs.pm/elixir/Random.html#content).

Іншим способом генерації випадкових чисел є використання криптографічно безпечного генератора, який забезпечує більшу випадковість. Його можна використовувати з функціями `crypto_rand_uniform/1` та `crypto_rand_uniform/2`. Більше про них можна прочитати у [документації](https://hexdocs.pm/elixir/Crypto.html#content).
