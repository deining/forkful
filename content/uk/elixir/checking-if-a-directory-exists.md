## Що і Чому?
Перевірка існування директорії - це процес перевірки, чи існує вказана програмою директорія на комп'ютері. Це важлива задача для програмістів, оскільки це дозволяє перевірити наявність необхідних ресурсів та виконати потрібні дії в залежності від результату перевірки.

## Як це зробити:
```Elixir
def check_directory(directory) do    # оголошуємо функцію з параметром - директорією
  if File.exists?(directory) do      # перевіряємо, чи існує вказана директорія за допомогою функції "exists?"
    IO.puts("Директорія #{directory} існує на комп'ютері.") # виводимо повідомлення про наявність директорії
  else
    IO.puts("Директорії #{directory} не існує на комп'ютері.") # виводимо повідомлення про відсутність директорії
  end
end
```
приклад виклику функції та виведення результату:
```
check_directory("Documents") 
"Директорія Documents не існує на комп'ютері."
check_directory("Downloads") 
"Директорія Downloads існує на комп'ютері."
```

## Глибше в деталі:
Перевірка існування директорії є важливою частиною процесу програмування, оскільки дозволяє перевірити наявність необхідних ресурсів та уникнути помилок при роботі з ними. Варіанти перевірки можуть залежати від конкретної мови програмування, але загалом використання функції "exists?" є універсальним.
