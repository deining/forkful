## Чому
Розрахунок дати в майбутньому або минулому є важливим елементом для багатьох програм. Наприклад, це може бути потрібно для планування подій або автоматичного оновлення часових маркерів в додатку. Використання Elixir дозволяє легко обчислювати дати в майбутньому або минулому за допомогою вбудованих функцій та бібліотек.

## Як це зробити
Для обчислення дати в майбутньому або минулому можна використовувати функцію `Date.add/3`. Вона приймає 3 аргументи: початкову дату, кількість днів для додавання і напрямок, в якому додавати (1 для майбутнього, -1 для минулого).

```Elixir
start_date = Date.utc_today()
#=> ~D[2020-12-04]
future_date = Date.add(start_date, 7, 1)
#=> ~D[2020-12-11]
past_date = Date.add(start_date, 14, -1)
#=> ~D[2020-11-20]
```

Також можна використовувати бібліотеки, які надають зручний інтерфейс для обчислення дат. Наприклад, бібліотека Timex має функцію `Timex.shift/3`, що дозволяє додавати або віднімати будь-який часовий період з будь-якої дати.

```Elixir
start_date = ~D[2020-12-04]
future_date = Timex.shift(start_date, weeks: 2)
#=> ~D[2020-12-18]
past_date = Timex.shift(start_date, years: -1, months: -2)
#=> ~D[2019-10-04]
```

## Поглиблене дослідження
Для обчислення дати можна використовувати різні формати дати, включаючи Unix час та міжнародний формат ISO 8601. Варто звернути увагу на часові зони при обчисленні дат, оскільки вони можуть впливати на результат. Також, бібліотеки Timex і Calendar надають додаткові функції для роботи з часовими зонами та переходом на літній час.
