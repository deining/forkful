# Чому 

Регулярні вирази є могутнім інструментом для роботи з текстом в Elixir. Вони дозволяють здійснювати швидкий та ефективний пошук та заміну певних шаблонів у тексті за допомогою простого синтаксису.

# Як використовувати регулярні вирази в Elixir

Для використання регулярних виразів необхідно імпортувати модуль ```Regex``` та використовувати функцію ```match```. Наприклад, якщо ви хочете перевірити, чи містить рядок слово "Привіт", ви можете використовувати такий код:
```Elixir
import Regex

str = "Привіт, це блог про Elixir!"

match = Regex.match? ~r/Привіт/, str

IO.puts match
```
Це виведе ```true```, оскільки рядок містить слово "Привіт".

# Глибше поглибитися

Регулярні вирази дають можливість використовувати такі спеціальні символи, як ```+```, ```*```, ```?``` для визначення кількості повторень шаблону. Також можливе використання групування частин тексту за допомогою дужок та використання зворотніх посилань.

Детальну інформацію про використання регулярних виразів ви можете знайти в документації Elixir.
