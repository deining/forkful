## Що & Навіщо?
Використання регулярних виразів дозволяє програмістам шукати та витягувати певну інформацію з об'єктів у програмі, таких як рядки тексту. Це дуже корисний інструмент у розробці програмного забезпечення, оскільки допомагає автоматизувати процеси та зберегти час.

## Як використовувати:
У мові програмування Elixir для роботи з регулярними виразами існує вбудована бібліотека - Regex. Для того, щоб використати її, необхідно додати наступний код у ваш файл програми:

```Elixir
import Regex
```
Потім можна виконувати пошук та заміну з використанням регулярного виразу, наприклад:

```Elixir
Regex.run(~r/[aeiou]/, "Hello World", global: true) # Виведе список усіх голосних букв з рядка
Regex.replace(~r/[aeiou]/, "Hello World", "*") # Замінить всі голосні букви зірочками
```
Результат виконання буде виглядати так:
```
["e", "o", "o"]
"H*ll* W*rld"
```

## Глибинне підгляд:
Регулярні вирази почали використовуватися ще в 1956 році, але найчастіше були популярні в текстових процесорах та оперційних системах. Їх можна замінити на використання функцій для роботи з рядками в Elixir, таких як `String.split` та `String.replace`. Однак, регулярні вирази все ще корисні для більш складних операцій, наприклад, валідації даних.
