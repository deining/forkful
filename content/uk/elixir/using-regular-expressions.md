# Чому
Регулярні вирази є потужним інструментом для роботи з рядками в будь-якій мові програмування, включаючи Elixir. Вони дозволяють шукати, заміняти та перевіряти наявність певних патернів у рядках, що дозволяє зменшити зусилля та покращити ефективність кодування.

# Як
```Elixir
# Приклад використання регулярного виразу для шуку певного шаблону у рядку
"Hello, World!" |> Regex.scan(~r/lo/)
# Вивід: [["lo"]]
```

```Elixir
# Приклад заміни певного шаблону у рядку на інший патерн
"The cat sat on the mat" |> Regex.replace(~r/cat/, "dog")
# Вивід: "The dog sat on the mat"
```

```Elixir
# Перевірка наявності певного шаблону у рядку
"abcdef" |> Regex.match?(~r/abc/)
# Вивід: true
```

# Глибше вдивитися
У Elixir регулярні вирази пишуться за допомогою модуля `Regex` та спеціального синтаксису `~r/.../`. Вони підтримують різні метасимволи та опції для більш точного пошуку та заміни. Також, регулярні вирази можуть бути корисними для валідації введених даних та виконання складних перетворень рядків.
