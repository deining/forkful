## Чому

Існує багато ситуацій, коли потрібно створити тимчасовий файл під час виконання програми. Наприклад, для збереження даних, що зберігаються в оперативній пам'яті, або для обробки великих обсягів даних, які необхідно зберегти тимчасово.

## Як

Для створення тимчасового файлу у Elixir, ми можемо використовувати функцію `File.temp_dir/0`. Наприклад:

```Elixir
file_path = File.temp_dir()
```

Ця функція повертає шлях до тимчасової директорії, де ми можемо створити наш файл. Для цього в Elixir також є готова функція `File.temp_file/2`, яка створює тимчасовий файл у вказаній директорії. Наприклад, ми можемо створити файл з розширенням `.txt` у тимчасовій директорії:

```Elixir
file_path = File.temp_dir()
File.temp_file(file_path, ".txt")
```

Також, ми можемо використовувати модуль `Tempfile` для створення тимчасових файлів:

```Elixir
file = Tempfile.open("example")
```

Результатом цих операцій буде створення тимчасового файлу у вказаній директорії з унікальним іменем, яке можна використовувати для збереження, читання та редагування даних.

## Глибока підготовка

Створення тимчасових файлів може бути корисно для збереження файлів, які потрібно використовувати тимчасово під час виконання програми. Також, це дозволяє заощадити місце на диску, оскільки тимчасові файли будуть автоматично видалені після завершення програми.
