## Чому

Створення тимчасового файлу є важливим елементом багатьох програм. Вони дозволяють зберігати тимчасові дані, які потрібні для виконання певних дій. Тимчасові файли також допомагають уникнути конфліктів між назвами файлів та забезпечити більш безпечне виконання програм.

## Як

Щоб створити тимчасовий файл в мові Elixir, можна використати вбудовану функцію `File.temp_file/2`. Також необхідно імпортувати модуль `File` у вашому коді. Ось приклад використання цієї функції:

```elixir
import File

# створення тимчасового файлу та збереження шляху до нього в змінну
temp_file = File.temp_file("prefix", "ext")

# запис даних у тимчасовий файл
write_data = "Це тимчасовий файл, створений для прикладу."
File.write(temp_file, write_data)

# читання з тимчасового файлу та виведення вмісту на екран
read_data = File.read(temp_file)
IO.puts read_data
```

Вивід на екран буде виглядати наступним чином:

```
Це тимчасовий файл, створений для прикладу.
```

## Глибоке дослідження

Функція `File.temp_file/2` приймає два параметри - префікс та розширення, та повертає шлях до створеного тимчасового файлу. Це дозволяє точно визначити назву тимчасового файлу та його тип. Також, ця функція автоматично створює папку для збереження тимчасового файлу та повертає повний шлях до неї.

Додатково, функція `File.temp_file/2` дозволяє визначити час життя тимчасового файлу. Наприклад, можна вказати, що файл повинен автоматично видалятися через певний час після створення. Це дуже зручно для використання в тестуванні або у випадках, коли необхідно забезпечити швидке очищення тимчасових даних.
