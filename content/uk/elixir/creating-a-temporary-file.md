## Чому

Створення тимчасових файлів є важливою складовою програмування. Вони можуть бути використані для збереження даних, які потрібні тільки на певний час. Це також допомагає підтримувати чистоту та організованість вашого коду.

## Як

Для створення тимчасового файлу в Elixir, використовуйте функцію `Tempfile.new/2`. Перший аргумент цієї функції - це назва файлу, а другий - розширення. Наприклад:

```Elixir
file = Tempfile.new("temp", ".txt")
```

Це створить тимчасовий файл з назвою "temp" та розширенням ".txt". Ви можете використовувати будь-які інші назви та розширення залежно від вашої потреби. Також можете встановити шлях для збереження файлу за допомогою другого аргументу функції, якщо хочете зберігати його в певному місці.

Після створення тимчасового файлу, ви можете записувати у нього дані за допомогою функції `IO.write/2` та читати їх за допомогою функції `IO.read/2`. Приклад виведення даних до тимчасового файлу та їх подальше читання виглядатиме так:

```Elixir
IO.write(file.path, "Дивовижні дані")
contents = IO.read(file.path)
IO.puts contents
```

Результатом буде виведення "Дивовижні дані" у консоль.

## Детальний аналіз

У Elixir існує модуль `Tempfile`, який надає функції для роботи з тимчасовими файлами. Під час створення тимчасового файлу за допомогою функції `Tempfile.new/2`, ви можете передати третій аргумент - опції для розширеного керування файлом. Наприклад, ви можете вказати `{:sparse, true}` для створення файлу з розрідженими блоками.

Також є приклад функції `Tempfile.open/4`, яка дозволяє вам виконати операції з тимчасовим файлом всередині блока. Приклад виглядатиме так:

```Elixir
file = Tempfile.new("temp")
Tempfile.open(file, "w") do
  IO.puts "Цей текст буде записаний у тимчасовий файл"
end
```

Щоб видалити тимчасовий файл, використовуйте функцію `File.rm/1` та передайте в неї шлях до файлу:

```Elixir
File.rm(file.path)
```

Можливості роботи з тимчасовими файлами в Elixir є дуже широкими та детально описані в документації.
