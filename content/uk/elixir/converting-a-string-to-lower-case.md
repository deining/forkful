## Чому

Перетворення рядка на нижній регістр є часто використовуваною операцією в програмуванні. Це дозволяє зменшити кількість можливих варіантів рядка, що полегшує порівняння та обробку даних.

## Як

Перетворення рядка на нижній регістр в Elixir можна здійснити за допомогою функції `String.downcase/1`. Приклад використання:

```elixir
string = "ПрИкЛаД ТеКсТу"
lowercase = String.downcase(string)
IO.inspect(lowercase)

# Вивід: "приклад тексту"
```

Метод `downcase/1` приймає один аргумент - рядок, який потрібно перетворити. Результатом виклику функції є новий рядок, що містить вхідний рядок у нижньому регістрі.

А якщо ви хочете перетворити тільки першу літеру на нижній регістр, в Elixir є й інший корисний метод - `String.capitalize/1`. Приклад використання:

```elixir
string = "ПрИкЛаД ТеКсТу"
lowercase = String.capitalize(string)
IO.inspect(lowercase)

# Вивід: "Приклад тексту"
```

Будьте уважні, навіть якщо ви використовуєте метод `capitalize`, решта рядка не повинна містити літер у верхньому регістрі, тому що вони також будуть перетворені на нижній регістр.

## Глибокий занурення

В Elixir рядки є незмінними даними, що означає, що усі функції перетворення рядків повертають нові значення, а не модифікують вхідний рядок.

Також, для того, щоб визначити, чи є даний символ літерою, можна використовувати функцію `is_alpha/1` з модуля `String`. На приклад, бажаємо перевірити, чи перша літера рядка є буквою:

```elixir
string = "Рядок символів"
IO.puts String.is_alpha(String.first(string))

# Вивід: true
```

Для більш докладної інформації про функції для роботи з рядками в Elixir, рекомендуємо переглянути [документацію](https://hexdocs.pm/elixir/String.html).
