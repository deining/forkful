## Для чого

Переведення рядка у нижній регістр є важливою функцією у програмуванні. Це допомагає уникнути помилок через різницю у регістрах букв у рядках, а також уможливлює порівняння рядків без урахування регістру. Також, у багатьох випадках, переведення рядка у нижній регістр зробить його більш зрозумілим та зручним для обробки.

## Як це зробити

```Elixir
string = "HELLO WORLD"
String.downcase(string) # => "hello world"
```

У цьому прикладі ми використовуємо функцію `downcase` з модуля `String` для переведення рядка `HELLO WORLD` у нижній регістр. Це простий спосіб зробити перетворення для одного рядка, але ми також можемо використовувати цю функцію для списків рядків або навіть для любих даних, які можуть бути приведені до рядка.

```Elixir
list = ["ABC", "DEF", "GHI"]
Enum.map(list, &String.downcase/1) # => ["abc", "def", "ghi"]
```

Також, варто зазначити, що у функції `downcase` є параметр `:ascii`, який може бути використаний для ігнорування не-ASCII символів.

### Глибша погляд

У Elixir використовується Unicode для роботи з рядками, що означає, що у функції `downcase` будуть працювати всі символи з Юнікода. Це означає, що можна бути впевненими у коректності перетворень незалежно від мови або символу.

Також, для більшої ефективності, можна використовувати функцію `downcase/2`, яка приймає аргументом позначення мови (наприклад, `:bg` для болгарської). Це дозволяє більш точно визначити регістри букв для даної мови.
