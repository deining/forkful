## Для чого

Є багато ситуацій, коли може знадобитися аналізувати HTML-код, наприклад, веб-скрапінг, парсинг даних або створення веб-сервісів. Використовуючи Еліксир, ви можете швидко та ефективно зможете розбиратися з HTML-кодом.

## Як це зробити

Простий спосіб зробити парсинг HTML-коду в Еліксирі - використати бібліотеку Floki. Перш ніж почати, необхідно встановити цю бібліотеку за допомогою менеджера пакетів mix. Потім потрібно підключити Floki до вашого проекту та імпортувати його функції.

```Elixir
mix deps.get
```
Після встановлення бібліотеки, ви можете розпочати парсинг HTML-коду. Наприклад, якщо вам потрібно отримати заголовок сторінки, ви можете використати наступний код:
```Elixir
html = """
<html>
<head>
<title>Мій веб-сайт</title>
</head>
<body>
<h1>Ласкаво просимо на мій сайт!</h1>
<p>Цей сайт присвячений Еліксиру.</p>
</body>
</html>
"""

Floki.find(html, "h1") 
# виводить: Ласкаво просимо на мій сайт!
```

Ви також можете знайти більш складні елементи, використовуючи селектори з Floki. Наприклад, якщо ви хочете знайти всі посилання на сторінці, ви можете використати наступний код:
```Elixir
links = Floki.find(html, "a")
Enum.each(links, fn link -> IO.puts Floki.attribute(link, "href") end)
# виводить: 
# https://elixir-lang.org/
# https://github.com/elixir-lang/elixir/
# https://medium.com/elixir-mix
```

## Глибше в деталі

Floki використовує CSS-селектори для пошуку елементів у HTML-коді. Ви можете використовувати будь-які CSS-селектори, які ви бажаєте, для знаходження потрібних елементів. Також варто звернути увагу на те, що Floki повертає результат у вигляді нод, що дозволяє вам детальніше працювати з HTML-кодом, наприклад, отримувати атрибути елементів чи змінювати їх структуру.
