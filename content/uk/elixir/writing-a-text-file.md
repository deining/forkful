## Як

Написання текстового файлу в Elixir є простим завданням, яке використовує основні засоби мови програмування. Давайте розглянемо деякі приклади коду для створення текстового файлу та виведення його вмісту:

```Elixir
# Створення нового текстового файлу з назвою "hello.txt"
File.write("hello.txt", "Привіт, це перший текстовий файл, створений мною в Elixir!")

# Виведення вмісту текстового файлу "hello.txt"
File.read("hello.txt")
# Результат: "Привіт, це перший текстовий файл, створений мною в Elixir!"
```

Також можна додавати новий текст до вже існуючого файлу за допомогою функції `File.append`:

```Elixir
# Додавання нового рядка "Мені подобається вивчати Elixir!" до файлу "hello.txt"
File.append("hello.txt", "Мені подобається вивчати Elixir!")
# Результат: "Привіт, це перший текстовий файл, створений мною в Elixir! Мені подобається вивчати Elixir!"
```

## Глибоке занурення

Написання текстових файлів в Еlixir може бути корисним в багатьох ситуаціях. Ви можете використовувати цей метод для збереження данних, що змінюються в процесі виконання програми, для створення лог-файлів або для створення сховища для необхідних даних.

Також важливо знати, що можна працювати з різними типами файлів, наприклад, CSV або JSON, використовуючи спеціальні бібліотеки, такі як `:csv` та `:jason`.

Написання текстових файлів в Elixir - це лише початок вивчення можливостей цієї мови програмування. Рекомендуємо досліджувати більше можливостей для створення потужних програм.
