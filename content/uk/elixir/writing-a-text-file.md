## Чому

Запис текстового файлу є важливою частиною роботи в багатьох програмах. Наприклад, ви можете створювати звіти або зберігати дані для подальшого використання. Також це може бути необхідно для збереження налаштувань або конфігурацій. Наприклад, якщо ви розробляєте веб-додаток, то можливо захочете зберегти налаштування бази даних в окремому текстовому файлі.

## Як

```Elixir
файл = File.open("дані.txt", [:write])
File.write(file, "Це текст, який буде записаний у файл")
File.close(file)

файл = File.open("дані.txt", [:append])
File.write(file, "А цей текст буде доданий до кінця файлу")
File.close(file)

файл = File.open("дані.txt", [:read])
зміст = IO.read(file)
IO.puts(зміст)
File.close(file)
```

В цьому прикладі ми відкриваємо файл з правами на запис і записуємо в нього текст за допомогою функції `File.write`. Потім ми закриваємо файл. В наступному прикладі ми відкриваємо той самий файл з правами на додавання і додаємо до нього ще один рядок тексту. Наприкінці ми можемо відкрити файл з правами на читання і вивести його вміст за допомогою функцій `IO.read` і `IO.puts`.

## Deep Dive

У Elixir для роботи з текстовими файлами є багато корисних функцій, які дозволяють робити це простим і зручним. Наприклад, функція `File.read!` зчитує вміст файлу в змінну, але видасть помилку, якщо файл не існує. Також є можливість скористатися пакетом `File` зі стандартної бібліотеки, який має багато корисних функцій, таких як `File.rename` для перейменування файлів або `File.copy` для копіювання.
