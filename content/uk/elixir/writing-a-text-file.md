## Чому

Написання текстового файлу є важливим елементом програмування, оскільки він дозволяє зберігати інформацію у структурованому та доступному форматі. Будь-який програміст повинен знати, як управляти і зберігати дані за допомогою текстових файлів.

## Як

Нижче наведені приклади коду та його результатів в рамках блоків коду ```Elixir ... ```, які демонструють основні прийоми написання текстових файлів у мові Elixir.

Створення файлу:

```Elixir
File.write("file.txt", "Це текст, який буде збережений у файлі.")
```

Додавання даних до файлу:

```Elixir
File.append("file.txt", "Це доданий текст до вже існуючого файлу.")
```

Читання файлу:

```Elixir
File.read("file.txt") # Поверне "Це текст, який буде збережений у файлі. Це доданий текст до вже існуючого файлу."
```

Перезаписування файлу:

```Elixir
File.write("file.txt", "Це оновлені дані у файлі.")
```

Видалення файлу:

```Elixir
File.rm("file.txt")
```

## Глибше вдивимось

При роботі з текстовими файлами важливо пам'ятати, що вони можуть бути корисними для зберігання будь-якого типу даних, включаючи числа, рядки, списки та навіть власні типи даних. Також, текстові файли можуть бути використані для комунікації між різними програмами та платформами. У мові Elixir є багато різних функцій та методів для роботи з текстовими файлами, які можуть бути корисними при розробці програм.
