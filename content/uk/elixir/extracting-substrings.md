## Чому

Усім подобається використовувати функцію екстракції підрядкових ланок в Elixir. Це допомагає нам з легкістю витягувати певні частини рядка, щоб робити з ними операції або просто отримувати потрібну інформацію. Це робить програмування більш ефективним та допомагає зменшити кількість коду.

## Як використовувати екстракцію підрядкових ланок

```Elixir
# Створюємо рядок
str = "Привіт, це приклад рядка для екстракції підрядкових ланок в Elixir"

# Екстракція підрядкових ланок, починаючи з 7-го символу та довжиною 5 символів
substring = String.slice(str, 7, 5)

# Виведення результату
IO.puts(substring) # "це прик"

# Екстракція підрядкових ланок за допомогою регулярних виразів
# Виводимо всі підрядкові ланки, які починаються з символу "а"
IO.inspect Regex.scan(~r/a\w+/, str) # ["адка", "а", "абня"]

```

## Поглиблене вивчення екстракції підрядкових ланок

Існує кілька вбудованих функцій у Elixir, які допомагають виконувати екстракцію підрядкових ланок більш ефективно. Наприклад, `String.slice` може бути використана для отримання підрядкового рядка за його позицією та довжиною. Також, з `Regex.scan` можна витягувати підрядкові ланки за допомогою регулярних виразів, що дає більшу гнучкість при роботі з рядками.
