Що & Чому?

Читання текстового файлу в програмуванні означає отримання доступу до інформації, що зберігається у вигляді тексту. Програмісти часто читають файли, щоб отримати дані, що можуть бути використані в їх програмах або для відображення користувачам.

Як це зробити:

```Elixir
# Відкриття та читання файлу "sample.txt"
file = File.open("sample.txt")
content = IO.read(file)

# Виведення вмісту файлу на екран
IO.puts(content)

# Закриття файлу
File.close(file)
```

Вивід:

```
Привіт, це зразковий текстовий файл.
Вітаємо вас з використанням Elixir!
```

Детальніше:

Читання текстового файлу - це один із способів отримання даних для програм. Інші можливості включають в себе підключення до баз даних або користування інтернет-запитами. Раніше, читання файлу вимагало використання більше коду, але завдяки Elixir це тепер стало простішим та швидшим.
