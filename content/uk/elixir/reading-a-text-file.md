## Чому

Процес читання текстового файлу є важливою частиною в програмуванні. Вони містять інформацію, яка може бути дуже корисна для виконання різних завдань програми.

## Як

В Elixir, читання текстових файлів є простим завданням, що допоможе зберегти час та зусилля. Спочатку вам потрібно встановити наступну бібліотеку:

```Elixir
def deps do
  [{:exfile, "~> 0.0.2"}]
end
```

Після встановлення бібліотеки, потрібно використовувати наступний код для читання файлу у вашій програмі:

```Elixir
File.read("text_file.txt")
|> IO.puts
```

У цьому прикладі, ми використовуємо функцію `File.read`, щоб отримати вміст файлу і передаємо це до функції `IO.puts`, щоб роздрукувати його у консолі. Ви також можете зберегти вміст файлу в змінну для подальшої обробки.

## Поглиблення

У Elixir, бібліотека `exfile` також надає можливість читати кожен рядок файлу окремо з використанням функції `File.stream!`:

```Elixir
File.stream!("text_file.txt")
|> Enum.each(&IO.puts/1)
```

Цей код буде роздрукувати кожен рядок файлу в окремому рядку у консолі. Ви також можете використовувати функції з модуля `File` для отримання детальної інформації про сам файл.
