## Чому

Робота з текстовими файлами є важливою частиною програмування в Elixir. Це дозволяє зберігати і обробляти великі обсяги даних, що зробить ваш код більш ефективним та швидким. Читання текстових файлів також допоможе вам зрозуміти, як взаємодіють різні частини вашої програми, забезпечуючи надійність та якість вашого коду.

## Як

Перш за все, додайте завдання, щоб зчитати текстовий файл у вашому проекті Elixir. Ви можете зробити це за допомогою функції `File.read/1`, яка приймає шлях до файлу як параметр.

```Elixir
file_content = File.read("path/to/file.txt")
```

Якщо вам потрібно обробити кожен рядок окремо, ви можете розділити зчитаний текст за допомогою функції `String.split/2`.

```Elixir
file_lines = String.split(file_content, "\n")
```

Потім ви можете використовувати ці дані для подальшої обробки, наприклад, для аналізу тексту або збереження даних у базі даних.

## Глибока занурення

У Elixir також є багато інших корисних функцій для роботи з текстовими файлами. Наприклад, ви можете використовувати `File.stream!/1` для створення потоку даних з файлу, що дозволяє читати його по порціях. Ви також можете використовувати `File.write/2` для запису даних у файл або `File.ls/1` для отримання списку файлів у директорії.

Інше корисне рішення - використання бібліотеки CSV для роботи з CSV файлами, яка дозволяє зручно створювати та обробляти дані у цьому форматі.
