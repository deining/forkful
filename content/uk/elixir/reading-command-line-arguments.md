## Чому

З часом, як ми розвиваємося як програмісти, збираємося виростати або створювати свої власні програми зі своїми власними наборами параметрів командного рядка. Розуміння того, як зчитувати ці аргументи командного рядка, є важливим для покращення функціональності наших програм та забезпечення їх користувачам більшої зручності. В цій статті ми розглянемо, як використовувати Elixir для зчитування аргументів командного рядка.

## Як

Зчитування аргументів командного рядка з Elixir можна зробити дуже простим і ефективним способом. Все, що потрібно зробити - це використовувати модуль `OptionParser` і функцію `OptionParser.parse/2`, що дозволить нам зчитати та обробити аргументи командного рядка. Давайте подивимось на приклад коду:

```Elixir
defmodule MyApp do
  def main(argv) do
    {options, args, _} = OptionParser.parse(argv, switches: [help: :boolean], aliases: [h: :help])
    if options.help do
      IO.puts "Допомога:"
      IO.puts "  -h, --help  Вивести цю допомогу"
    else
      IO.puts "Аргументи командного рядка: #{inspect args}"
    end
  end
end

MyApp.main(System.argv)
```

Якщо запустити цей код з такими аргументами командного рядка, як `--help` або `-h`, то отримаємо такий вивід:

```bash
$ mix run my_app.exs --help
Допомога:
  -h, --help  Вивести цю допомогу
```

В іншому випадку, будь-які інші аргументи командного рядка будуть виведені на екран:

```bash
$ mix run my_app.exs arg1 arg2
Аргументи командного рядка: ["arg1", "arg2"]
```

Це дуже корисний метод для доступу до аргументів командного рядка в нашому коді.
