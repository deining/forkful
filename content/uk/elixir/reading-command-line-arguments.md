## Чому
Програмування на мові Elixir може бути цікавим, оскільки мова ця належить до функціональних мов та має потужну систему обробки аргументів командного рядка. Ця функція дозволяє передавати аргументи програмі при запуску з командного рядка, що дозволяє змінювати поведінку програми в залежності від потреб користувача.

## Як
Для того, щоб передати аргументи командного рядка програмі на Elixir, використовується функція `System.argv/1`. Наприклад, якщо ви хочете передати два аргумента, перший з яких є назвою програми, а другий - ім'я користувача, це можна зробити наступним чином:

```Elixir
args = System.argv(["my_program", "John"])
```

В результаті отримаємо список `["my_program", "John"]`. Також, ми можемо використовувати аргументи в більш зручному для нас форматі, використавши функцію `System.argv/2`:

```Elixir
args = System.argv(["my_program", "John"], ~w(-n --name))
```

Цей код верне список `[["-n", "John"], ["--name", "John"]]`, де перший елемент списку - рядок, що містить флаг аргумента, а другий - відповідне значення. Також, можливе використання хеш-таблиці з назвами аргументів:

```Elixir
args = System.argv(["my_program", "John"], %{"-n" => "--name"})
```

Цей код поверне хеш `[%{"--name" => "John"}]`. Будьте уважні, використовуючи текстові аргументи, оскільки вони можуть містити пробіли, тому краще використовувати список або хеш з назвами аргументів.

## Глибоке занурення
При обробці аргументів командного рядка, ми також можемо використовувати функцію `System.get_env/1`, яка дозволяє отримувати значення змінних середовища. Наприклад, якщо ми хочемо отримати значення змінної `PORT`, ми можемо використати наступний код:

```Elixir
port = System.get_env("PORT")
```

Для того, щоб бути впевненими, що аргументи командного рядка та значення змінних середовища передаються правильно, можна використовувати функцію `Mix.Utils.parse_arguments/1`, яка повертає хеш з розпарсеними аргументами та значеннями змінних середовища.
