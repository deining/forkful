## Чому

Існує багато причин, чому в програмуванні може знадобитися на маніпуляції з рядками, одна з них - перетворення першої літери рядка на велику. Це може стати необхідним для створення красивого вигляду тексту, для полегшення роботи з даними, або для виконання певної операції з даними.

## Як це зробити

Для того, щоб перетворити першу літеру рядка на велику, необхідно скористатися методом `toUpperCase()` та `slice()` на рядках в Javascript. Ось приклад коду і його вихід:

```Javascript
let str = "приклад рядка";
let newStr = str.charAt(0).toUpperCase() + str.slice(1);

console.log(newStr); // Виведе "Приклад рядка"
```

## Вдаваймося у деталі

Навіть якщо ви знаєте, як перетворити першу літеру рядка на велику, варто звернути увагу на деякі особливості цього процесу. Наприклад, якщо ви використовуєте метод `toUpperCase()` на рядку, який вже має велику першу літеру, то результат буде той самий, тобто велика літера не буде змінена. Також, якщо рядок має ще декілька слів, то для їх великих літер можна використовувати цей же метод в сполученні з `slice()`.
