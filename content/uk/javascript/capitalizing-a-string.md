# Чому

Капіталізація строк є важливим елементом в програмуванні, оскільки вона дозволяє користувачу працювати зі строковими даними у більш зручному форматі. Наприклад, капіталізовані строки є корисними при форматуванні тексту для заголовків або виправленні орфографічних помилок.

# Як це зробити

```Javascript
// Приклад використання вбудованої функції toUpperCase() для капіталізації першої літери строки

let str = "привіт світ!";
console.log(str.toUpperCase()); // результат: "ПРИВІТ СВІТ!"

// Приклад власної функції для капіталізації всієї строки

function capitalize(str) {
  let newStr = "";
  for (let i = 0; i < str.length; i++) { // проходимо циклом по кожному символу строки
    if (str[i] === " ") { // перевіряємо, чи символ є пробілом
      newStr += " "; // якщо так, то додаємо пробіл до нової строки
    } else {
      newStr += str[i].toUpperCase(); // якщо ні, то додаємо капіталізований символ до нової строки
    }
  }
  return newStr;
}

console.log(capitalize("привіт світ!")); // результат: "Привіт Світ!"

// Приклад використання методу join() для капіталізації кожного слова в реченні

function capitalize(str) {
  let arr = str.toLowerCase().split(" "); // перетворюємо строку в масив слів
  for (let i = 0; i < arr.length; i++) { // проходимо циклом по кожному слову в масиві
    arr[i] = arr[i].charAt(0).toUpperCase() + arr[i].substring(1); // капіталізуємо першу літеру кожного слова
  }
  return arr.join(" "); // повертаємо з'єднаний масив у вигляді строки
}

console.log(capitalize("привіт світ!")); // результат: "Привіт Світ!"
```

# Глибше дослідження

Капіталізація строк може бути корисна для автоматизації великих об'ємів тексту, зокрема при обробці даних веб-сайтів або роботі зі словниками та перекладом текстів. Також існують різні підходи до капіталізації, наприклад, можна використати функцію toLocaleUpperCase() для капіталізації з урахуванням мови та правопису. Важливо також звернути увагу на кодування символів, оскільки деякі літери у різних мовах можуть мати різні коди.
