## Чому

Порівняння двох дат є важливою завданням для багатьох програмістів, оскільки це дозволяє нам визначити, яка з дат більш нова або стара. Це особливо корисно для створення функцій, які працюють з часом і датами.

## Як це зробити

Існує кілька способів порівняти дві дати в JavaScript. Один з них - використання функції `getTime()`, яка повертає кількість мілісекунд, які пройшли з 1 січня 1970 року до вказаної дати. Давайте подивимося на приклад коду:

```javascript
let date1 = new Date("2020-01-01");
let date2 = new Date("2021-01-01");

if (date1.getTime() < date2.getTime()) {
  console.log("date1 є більш новою датою");
} else if (date1.getTime() === date2.getTime()) {
  console.log("дати ідентичні");
} else {
  console.log("date2 є більш новою датою");
}
```

У прикладі вище ми створюємо дві дати за допомогою конструктора `new Date()` і порівнюємо їх за допомогою функції `getTime()`. В результаті ми отримуємо відповідне повідомлення в консолі, яке вказує на те, яка з дат є більш новою.

Інший спосіб - використовувати оператори порівняння `<`, `>`, `===`. Приклад коду виглядає наступним чином:

```javascript
let date1 = new Date("2020-01-01");
let date2 = new Date("2021-01-01");

if (date1 < date2) {
  console.log("date1 є більш старою датою");
} else if (date1 === date2) {
  console.log("дати ідентичні");
} else {
  console.log("date2 є більш старою датою");
}
```

Обидва ці способи є ефективними для порівняння двох дат.

## Глибше дослідження

Важливо пам'ятати, що при порівнянні двох дат у JavaScript також береться до уваги часовий пояс. Якщо дати знаходяться в різних часових поясах, результат може бути непередбачуваним. Також варто враховувати, що при порівнняні різних форматів дат (наприклад, "2020-01-01" та "01-01-2020") результат може не бути однаковим.
