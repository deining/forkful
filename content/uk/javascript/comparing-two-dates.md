## Чому

Порівняння двох дат є важливою задачею у програмуванні, яка дозволяє визначити, яка з дат є більшою або меншою. Це корисно у випадках, коли необхідно порівняти дати для сортування, фільтрування чи виконання певних дій з даними залежно від дати.

## Як зробити

Найпростіший спосіб порівняти дві дати - використовувати оператори порівняння (`<`, `>`, `<=`, `>=`) між ними. Наприклад, для порівняння двох дат `date1` та `date2` можна написати наступний код:

```Javascript
if (date1 > date2) {
  console.log("Перша дата більша за другу");
} else if (date1 === date2) {
  console.log("Дати рівні");
} else {
  console.log("Перша дата менша за другу");
}
```

В результаті в консолі буде виведено відповідне повідомлення залежно від порівняння дат.

Окрім операторів порівняння, також можна використовувати методи об'єкта `Date`, який містить дату і час. Наприклад, метод `getTime()` повертає кількість мілісекунд, які пройшли з 1 січня 1970 року до заданої дати. Таким чином, для порівняння двох дат можна використовувати цей метод, як показано нижче:

```Javascript
if (date1.getTime() > date2.getTime()) {
  console.log("Перша дата більша за другу");
} else if (date1.getTime() === date2.getTime()) {
  console.log("Дати рівні");
} else {
  console.log("Перша дата менша за другу");
}
```

Якщо ви бажаєте порівняти дати з різницею меншою за день, використовуйте метод `toDateString()`, який повертає дату в рядковому форматі. Можна скористатися цим методом для порівняння двох дат у форматі "день, місяць рік". Приклад коду для такого порівняння:

```Javascript
const date1Str = date1.toDateString();
const date2Str = date2.toDateString();
if (date1Str > date2Str) {
  console.log("Перша дата більша за другу");
} else if (date1Str === date2Str) {
  console.log("Дати рівні");
} else {
  console.log("Перша дата менша за другу");
}
```
