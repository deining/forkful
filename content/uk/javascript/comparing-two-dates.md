## Чому

Програмування з датами - одне з найпоширеніших та багатофункціональних використань в Javascript. Порівняння дат є корисною функцією для визначення, яка з двох дат пізніша або раніша. Це може бути корисно для будь-якого проекту, де потрібно перевірити, чи спрацювала подія, до або після певної дати.

## Як це зробити

Для порівняння двох дат у Javascript, використовуйте вбудований оператор `>`, `<`, `==`, `>=`, `<=`. Оператори використовуються для порівняння наступних властивостей дат: рік, місяць, день, година, хвилина та секунда. Наприклад:

```Javascript
let date1 = new Date('2020-01-01');
let date2 = new Date('2021-01-01');
console.log(date1 < date2); // виведе true, бо дата 2020-01-01 раніше за 2021-01-01
```

## Глибоке занурення

Оператори порівняння використовують часовий пояс та мільсекунди для порівняння дат. Якщо дата визначена без вказівки часового поясу, вона використовує час та дату в системному часовому поясі користувача. Якщо дата визначена з вказівкою часового поясу, вона буде використовувати цей часовий пояс для порівняння. Крім того, мільсекунди також враховуються при порівнянні двох дат. Також, зверніть увагу, що при порівнянні двох дат тоді, коли в одній даті не вказаний час, вона використовує 00:00:00 як час. Наприклад:

```Javascript
let date1 = new Date('01-01-2021');
let date2 = new Date('01-01-2021T12:00:00+05:00');
console.log(date1 == date2); // виведе false, оскільки в одній даті немає вказівки часу, а в другій є
```
