## Чому

Можливо, ви задумуєтеся, чому вам потрібно отримувати поточну дату в програмуванні. Для початку, це допоможе вам зберегти дату час кожного разу, коли запускаєте програму, щоб дізнатися точну дату, коли код був виконаний. Також, це дозволить вам створювати різні функції, які швидко обчислюватимуть різні дати для різних потреб. Наприклад, чи вам потрібно дізнатися, скільки днів залишилось до вашого наступного дня народження? Отримання поточної дати допоможе вам розрахувати це без зайвих зусиль.

## Як отримати поточну дату

Починаючи з Javascript, є кілька шляхів, якими можна отримати поточну дату. Один з них - це використання вбудованого об'єкта Date. Подивіться на цей приклад:

```Javascript
// створюємо новий екземпляр об'єкта Date
let currentDate = new Date();

// відображаємо поточну дату у зрозумілому форматі
console.log(currentDate.toLocaleDateString()); // виводить дату у форматі "місяць/день/рік"
```

Ви також можете вказати певну дату, яку хочете отримати, використовуючи вбудовані методи об'єкта Date. Наприклад, щоб отримати поточний місяць, ви можете використати метод `getMonth()`.

```Javascript
// повертає номер поточного місяця, де 0 - січень, 11 - грудень
console.log(currentDate.getMonth()); // виводить "3" для квітня
```

Або, якщо вам потрібно отримати поточний рік, ви можете використати метод `getFullYear()`.

```Javascript
// повертає поточний рік
console.log(currentDate.getFullYear()); // виводить наприклад "2021"
```

Ще одним способом отримати поточну дату є використання бібліотек, наприклад Moment.js або Day.js. Вони мають багато корисних функцій для роботи з датами та часом.

## Детальніше про отримання поточної дати

Як уже було згадано, в Javascript є вбудований об'єкт Date, що дозволяє отримати дані про поточну дату та час. Цей об'єкт має також вбудовані методи, які дозволяють зробити різні операції над датою, наприклад обчислювати різницю між датами або перетворювати дату у різні формати.
