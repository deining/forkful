Що та чому?
Отримання поточної дати - це процес отримання поточної дати та часу у форматі, який може бути використаний в програмі. Програмісти часто використовують цю функцію для визначення часу виконання програми, або для отримання точної дати для відображення на веб-сторінці чи в додатку.

Як це зробити:
```Javascript
const currentDate = new Date(); 
console.log(currentDate); // Output: Mon Jun 21 2021 09:00:17 GMT+0300 (Eastern European Summer Time) 
```
Детальний аналіз:
Історичний контекст:
Отримання поточної дати було дуже важливою функцією для багатьох програмистів ще з часів появи Javascript. Початково, це було реалізовано за допомогою конструктора Date, який був ізольованим від основного об'єкту мови. Однак, з виходом ES5 у 2009 році, більш сучасні методи для отримання поточної дати були додані до мови. На сьогоднішній день, найбільш розповсюдженою і простою його реалізацією є використання конструктора Date та його методів.

Альтернативи:
Є кілька альтернативних способів отримання поточної дати в Javascript, таких як сторонні бібліотеки та фреймворки, наприклад Moment.js. Однак, за великої кількості документації та підтримки технології Javascript, використання конструктора Date є найпростішим та найшвидшим варіантом.

Деталі реалізації:
Конструктор Date приймає різні аргументи для отримання поточної дати в різних форматах. Наприклад, ```new Date('June 21, 2021 9:00:00')``` або ```new Date(2021, 5, 21, 9, 00, 00)``` повернуть поточну дату у встановленому форматі. Також, досягнути точності до мілісекунд можливо за допомогою методів хвилина, секунда та мілісекунда від об'єкту дати.
