## Що & Чому?
Створення тимчасового файлу - це техніка, яку програмісти використовують для тимчасового збереження даних чи додаткового простору для обробки інформації. Це особливо корисно, коли нам потрібно зберегти дані для подальшої роботи, а потім видалити їх, щоб не зашморгати пам'ять.

## Як це зробити:
Для створення тимчасового файлу у Javascript достатньо використати вбудовану функцію `fs.mkdtempSync` та передати їй шлях із префіксом `./temp`. Після виклику цієї функції, повертається шлях до створеного тимчасового файлу. Нижче приведений приклад коду з виведенням шляху до створеного тимчасового файлу:

```Javascript
const fs = require('fs');
const tempPath = fs.mkdtempSync('./temp'); // створення тимчасового файлу
console.log(tempPath); // виведеться шлях до створеного тимчасового файлу
```

## Глибока занурення:
Створення тимчасових файлів часто використовувалося в старших версіях Javascript, оскільки вони були потрібні для збереження даних при обробці файлів. Також існують альтернативні способи, наприклад, створення тимчасового файлу у папці системи `/tmp`, але використання вбудованої функції є найбільш безпечним і простим способом.
