## Чому

Створення тимчасових файлів є важливою частиною програмування, оскільки це дозволяє зберігати дані тимчасово і використовувати їх для подальшої обробки або виведення інформації. Також воно допомагає зменшити обсяг використовуваної пам'яті та покращити продуктивність програм.

## Як

Існує кілька способів створення тимчасового файлу в Javascript, але для початку давайте розберемося з базовим методом використання вбудованого модуля fs.

```Javascript
// підключаємо вбудований модуль fs
const fs = require('fs');

// визначаємо шлях та ім'я тимчасового файлу
const tempFile = './temp/data.txt';

// викликаємо метод writeFile для створення файлу та запису в нього данних
fs.writeFile(tempFile, 'Це тимчасовий файл.', (err) => {
  if (err) throw err;
  console.log('Файл створений!');
});
```

Вище наведений код створить тимчасовий файл з ім'ям "data.txt" в папці "temp" і запише в нього строку "Це тимчасовий файл.".

## Deep Dive

Щоб краще зрозуміти створення тимчасових файлів, давайте розглянемо деякі з їх важливих характеристик:

- Тимчасовий файл буде автоматично видалений після закриття програми або після виконання спеціальної функції для видалення файлу.

- Використовуючи метод writeFile, ми можемо записувати будь-які дані до тимчасового файлу, включаючи об'єкти та масиви, оскільки вони будуть перетворені в строку.

- Бачити вміст файлу можна, використовуючи метод readFile та вивід його змісту консольним командою console.log.
