# Чому

Програмування є захоплюючим хобі для багатьох людей. Запускаючи новий проект, ви можете висловити свою творчість та зробити щось корисне для інших користувачів.

# Як

Для того, щоб почати новий проект, вам потрібно знати мову програмування Javascript та мати базові знання про його синтаксис. Для початку створіть новий файл з розширенням ".js" та відкрийте його у своєму улюбленому редакторі коду.

Давайте спробуємо створити просту функцію, яка буде додавати два числа разом. Нижче наведений приклад коду та виведення його у консоль.

```Javascript
function addNumbers(a, b){
  return a + b;
}

console.log(addNumbers(2, 3)) // Output: 5
```

Якщо ви бажаєте дізнатися більше про мову Javascript та її можливості, перегляньте [офіційну документацію](https://developer.mozilla.org/uk/docs/Web/JavaScript).

# Глибоке дослідження

Перш ніж починати новий проект, варто ретельно продумати його ідею та цілі. Які вимоги має задовольняти ваш проект? Чи потребує він певної бази даних? Як ви будете організовувати свій код та його структуру?

Крім того, важливо також обрати найбільш підходящі інструменти для вашого проекту. Наприклад, для створення веб-додатків можна використовувати фреймворки, такі як React або Angular, а для мобільних додатків - фреймворки Cordova або React Native.

Не бійтеся експериментувати та вдосконалювати свої навички програмування.
