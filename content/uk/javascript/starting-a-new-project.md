## Що & Чому?
Розпочаття нового проекту - це процес створення нової програми або веб-сайту. Програмісти розпочинають нові проекти, щоб розвивати свої навички, реалізувати нові ідеї та вирішувати проблеми.

## Як:
Створення нового проекту в Javascript дуже просте. Для початку, створіть нову папку для свого проекту і відкрийте її в командному рядку. Введіть наступну команду, щоб ініціалізувати ваш проект з файлом package.json:
```Javascript
npm init
```
Далі, ви можете встановити будь-які необхідні залежності, використовуючи команду ```npm install```. Щоб почати писати код, створіть новий файл з розширенням .js і починайте програмувати!

## Поглиблене дослідження:
Історично, Javascript був створений в 1995 році з метою надання можливості робити веб-сайти більш динамічними. Сьогодні, він є одним з найпопулярніших мов програмування, який використовується для створення як веб-сайтів, так і мобільних додатків. Існують також альтернативи, такі як React та Angular, які використовуються для створення складних веб-додатків.
