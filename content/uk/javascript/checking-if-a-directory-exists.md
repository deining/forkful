## Чому

Перевірка існування директорії є важливим етапом у програмуванні на Javascript. Вона дозволяє перевірити наявність певної директорії перед виконанням певних дій, а також уникнути помилок і зберегти час при роботі з файловою системою.

## Як це зробити

Для перевірки наявності директорії використовується функція `fs.existsSync()` у компоненті `fs` (file system). Ось приклад коду, який перевіряє наявність директорії з назвою "images" у поточній робочій директорії та виводить відповідне повідомлення у консоль:

```Javascript
if(fs.existsSync('./images')) {
  console.log("Директорія images існує!");
} else {
  console.log("Директорії images не існує.");
}
```

Виконаний код надрукує "Директорія images існує!", якщо директорія існує, або "Директорії images не існує.", якщо директорії не існує.

## Глибші занурення

Функція `fs.existsSync()` повертає булеве значення `true`, якщо директорія існує, або `false`, якщо директорії не існує. Для більшого розуміння, також можна використовувати метод `fs.statSync()` для отримання додаткової інформації про директорію, наприклад, час створення, останньої зміни, розмір тощо.
