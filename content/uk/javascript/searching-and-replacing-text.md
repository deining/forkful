## Чому

Багато розробників та програмістів часто зіштовхуються з необхідністю замінити деякий текст у своєму коді. Це може бути через помилку в написанні коду або потребу внесення змін. У таких випадках, пошук та заміна тексту стає незамінною операцією для ефективної роботи з програмним кодом.

## Як

З появою сучасних програмних мов та інструментів, пошук та заміна тексту стало значно простішим та швидшим процесом. Для цього можуть бути використані вбудовані функції або сторонні бібліотеки. Нижче наведено приклад коду на мові Javascript для пошуку та заміни тексту.

```Javascript
// Створення змінної з початковим текстом
let text = "Привіт, як справи?";

// Використання функції replace для заміни тексту
let newText = text.replace("справи", "настрій");

// Виведення результату в консоль
console.log(newText); // "Привіт, як настрій?"
```

В даному прикладі ми використовуємо вбудовану функцію `replace` для заміни слова "справи" на "настрій" у змінній `text`. Цей приклад демонструє лише один зі способів пошуку та заміни тексту. В залежності від потреби, можна також використовувати регулярні вирази або інші методи для досягнення більш точної та ефективної заміни тексту.

## Deep Dive

У всесвітньо відомій мові Javascript, є декілька вбудованих методів для пошуку та заміни тексту, таких як `replace`, `indexOf`, `lastIndexOf`, `search` та інші. Кожен з цих методів має свою унікальну функціональність та синтаксис. Крім того, існують також сторонні бібліотеки, які пропонують більш розширені можливості пошуку та заміни тексту. Наприклад, бібліотека `lodash` має метод `replaceAll`, який дозволяє замінювати всі входження шуканого тексту одночасно.
