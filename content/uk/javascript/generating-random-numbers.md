## Чому

Генерація випадкових чисел є важливою частиною програмування в Javascript і може бути корисною в різних ситуаціях, наприклад, для створення унікальних ідентифікаторів або для випадкового вибору елементів з масиву.

## Як

Для генерації випадкових чисел використовується метод ```Math.random()```, який повертає випадкове число від 0 до 1, не включаючи саме число 1. Це значення потім можна множити на максимальне значення, яке вам потрібно, і додати мінімальне значення, якщо необхідно. Наприклад, якщо вам потрібно згенерувати випадкове число від 1 до 10, ви можете використовувати такий код:

```Javascript
let randomNum = Math.random() * 10 + 1;
console.log(randomNum); // приклад вихідного значення: 7.359186413524801
```

Для отримання цілого числа можна використовувати метод ```Math.floor()``` або ```Math.ceil()```, які округлять результат до меншого або більшого цілого числа відповідно. Наприклад, якщо ви хочете отримати випадкове ціле число від 1 до 10, то потрібно такий код:

```Javascript
let randomInt = Math.floor(Math.random() * 10) + 1;
console.log(randomInt); // приклад вихідного значення: 4
```

## Глибше вдивимося

Метод ```Math.random()``` насправді не повертає повноцінну випадкову послідовність чисел, але базується на *псевдовипадковій* послідовності, що означає, що при кожному запуску програми результат буде однаковий. Це може бути корисно для тестування і налагодження. Якщо ж вам потрібна більшість  випадковість, то можна використовувати зовнішні бібліотеки, такі як ```random.js```, яка надає більш складні методи генерації випадкових чисел, наприклад, з використанням шумових алгоритмів або генераторів.
