## Чому

У програмуванні іноді потрібно вилучати частини рядків зі стрічки даних. Наприклад, для отримання підстрічок для подальшої обробки або порівняння їх з іншими значеннями. У цьому пості ми розглянемо, як ви можете ефективно вилучати підстрічки в Javascript.

## Як

Існує декілька способів вилучення підстрічок в Javascript. Найбільш простий - використовувати метод `substring()` з об'єктом рядків. Наведений нижче приклад показує використання цього методу для вилучення підстрічки зі стрічки "Hello World!":

```Javascript
let str = "Hello World!";
let subStr = str.substring(6); // subStr = "World!"
console.log(subStr); // "World!"
```

Також можна використовувати методи `slice()` та `substr()` для вилучення підстрічок. Однак, слід пам'ятати, що ці методи можуть мати різні вимоги до аргументів та повертати різні результати, тому варто уважно ознайомитись з їх документацією.

Також варто відзначити, що можливо використати і метод `indexOf()` разом з іншими методами для вилучення підстрічки на основі індексу початку та кінця.

## Deep Dive

Як вже згадувалося, для вилучення підстрічок в Javascript можна використовувати методи `substring()`, `slice()` та `substr()`. Однак, їх поведінка може бути трішки складнішою, ніж здається на перший погляд.

Наприклад, якщо в аргументи цих методів передати від'ємні значення, то вони будуть трактуватися як значення відстані від кінця стрічки. Також, якщо передати лише один аргумент, то методи будуть використовувати це значення як початковий індекс та вилучати підстрічку до кінця стрічки.

Також варто пам'ятати, що метод `slice()` може приймати в якості аргументів від'ємні значення, але методи `substring()` та `substr()` - ні.

Детальніше про ці та інші особливості вилучення підстрічок в Javascript можна дізнатися з документації та інших джерел.
