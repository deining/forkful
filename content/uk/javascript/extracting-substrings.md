## Чому

Екстрагування підрядків є важливим аспектом програмування, оскільки це дозволяє обробляти та маніпулювати рядками з легкістю. Це особливо корисно при роботі з текстовою інформацією, такою як веб-ресурси або дані користувача.

## Як

Для початку нам потрібно зрозуміти, що таке підрядки та як їх виділяти з основного рядка. Для цього використовується метод `slice`, який приймає початковий та кінцевий індекси, щоб визначити діапазон символів, який необхідно виділити.

```Javascript
let string = "Привіт, світ!";
let substring = string.slice(0, 5);
console.log(substring); // виведе "Привіт"
```

Також можливо використовувати негативні індекси, щоб вказати кінець рядка та визначити діапазон зворотньо.

```Javascript
let string = "Привіт, світ!";
let substring = string.slice(-5);
console.log(substring); // виведе "світ!"
```

Крім методу `slice`, також існує метод `substr`, який приймає початковий індекс та кількість символів, яку необхідно виділити.

```Javascript
let string = "Привіт, світ!";
let substring = string.substr(7, 6);
console.log(substring); // виведе "єт, св"
```

## Глибоке поглиблення

Усе звучить просто, але є кілька важливих речей, які варто зауважити під час роботи з екстрагуванням підрядків.

По-перше, індекси починаються з 0 в JavaScript. Це означає, що перший символ має індекс 0, другий - 1 і так далі. Також індекс останнього символа завжди буде на 1 менший, ніж кількість символів в рядку.

По-друге, якщо вказане число символів при використанні методу `slice` або `substr` перевищує кількість символів в рядку, то будуть виділені всі символи від вказаного індексу до кінця рядка.

Наприклад:

```Javascript
let string = "Привіт, світ!";
let substring = string.slice(5, 20);
console.log(substring); // виведе ", світ!"
```
