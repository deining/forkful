## Що і чому?
Інтерполяція рядків - це метод JavaScript, який дозволяє об'єднувати різні рядки в один, вставляючи змінні або вирази у певні місця. Це дозволяє програмістам створювати динамічні рядки змінюючогося контенту, такого як користувацький ввод або калькуляції.

## Як це зробити?
Інтерполяція рядків може бути виконана за допомогою спеціального строкового літерала - оберненого апострофа (``) замість звичайної лапки або подвійної лапки. У цьому літералі можна вставляти змінні або вирази за допомогою синтаксису ${змінна/вираз}. Ось приклад:

```javascript
let name = 'Іван';
let language = 'JavaScript';

console.log(`Привіт, я - ${name} і я вивчаю мову програмування ${language}.`);
// Результат: Привіт, я - Іван і я вивчаю мову програмування JavaScript.
```

## Поглиблене дослідження
Інтерполяція рядків була введена у стандарті JavaScript ES6 (2015) для заміни конкатенації рядків за допомогою + оператора. Її можна розглянути як альтернативу методові join() для масивів рядків. Також, інтерполяція рядків покращує читабельність коду та зменшує кількість помилок при використанні багатострокових рядків.
