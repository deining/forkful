## Чому

Регулярні вирази є потужним інструментом для роботи з текстом у Javascript. Вони дозволяють здійснювати швидкий і ефективний пошук та заміну рядків у тексті за допомогою умов і шаблонів. Використання регулярних виразів може значно полегшити процес розробки та оптимізувати код.

## Як

Використання регулярних виразів в Javascript дуже просте. Спочатку ми повинні створити регулярний вираз за допомогою об'єкта ```RegExp```, де ми вказуємо шаблон і, за необхідності, флаги. Далі ми можемо застосовувати цей вираз до рядків за допомогою методу ```test()``` або ```exec()```, щоб перевірити, чи відповідає рядок заданому шаблону, або знайти всі співпадіння в рядку. Наприклад:

```Javascript
let text = "Цей текст містить слова, такі як JavaScript, розробка, та програмування";
let regex = /javascript/i; // шукати слово "javascript" без врахування регістру
let result = regex.test(text); // результат: true
```

У нашому прикладі, метод ```test()``` поверне ```true```, тому що в рядку є слово "JavaScript", яке відповідає заданому шаблону. Якщо ми хочемо знайти всі співпадіння, ми можемо використовувати метод ```exec()```:

```Javascript
let text = "Цей текст містить слова, такі як JavaScript, розробка, та програмування";
let regex = /а[а-я]+/;
let result = regex.exec(text); // результат: ['ак', index: 5, input: 'Цей текст містить слова, такі як JavaScript, розробка, та програмування']
```

В цьому прикладі, метод ```exec()``` повертає перше співпадіння з заданим шаблоном (слово, що починається на "а"), його позицію в тексті і весь рядок, який проходить перевірку.
