## Що & Чому?

Використання регулярних виразів у програмуванні - це спосіб шукати та маніпулювати текстом за певним шаблоном. Програмісти використовують їх для швидшого та ефективнішого обробки даних, так як це дозволяє замінити велику кількість коду коротким та зрозумілим виразом.

## Як використовувати:

Регулярні вирази в JavaScript можна створити за допомогою вбудованого об'єкту RegExp. Для обробки тексту потрібно використовувати методи цього об'єкту, такі як `test()` для перевірки входження шаблону або `exec()` для повернення співпадінь. Нижче наведені декілька прикладів використання регулярних виразів:

```Javascript
const pattern = /hello/; // створюємо регулярний вираз з шаблоном "hello"
const text = "Hello world!"; // задаємо текст для перевірки
const result = pattern.test(text); // поверне true, оскільки в тексті є слово "Hello"
```

Більш складні шаблони можна використовувати, наприклад, для перевірки формату електронної пошти:

```Javascript
const emailPattern = /^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$/; // шаблон для перевірки електронної пошти
const email = "example@example.com"; // електронна пошта для перевірки
const result = emailPattern.test(email); // поверне true, оскільки пошта відповідає формату
```

## Приглиблене вивчення:

Регулярні вирази були запроваджені у JavaScript у версії 1.2 та є стандартною функцією у більшості сучасних мов програмування. Є альтернативи використанню регулярних виразів, наприклад, String методи `indexOf()` та `replace()`, але це зазвичай потребує більше коду та є менш ефективним. Ознайомитися з більш докладною інформацією про використання та синтаксис регулярних виразів можна за посиланням [MDN документація](https://developer.mozilla.org/uk/docs/Web/JavaScript/Reference/Global_Objects/RegExp).
