## Чому

Часто під час програмування, ми зустрічаємося з великою кількістю текстової інформації, яку потрібно обробити. Використання регулярних виразів дозволяє нам легко та швидко шукати, замінювати та перевіряти текст з використанням певних шаблонів. Це зробить наш код більш ефективним та продуктивним.

## Як

Щоб використовувати регулярні вирази в Javascript, вам потрібно використовувати об’єкт `RegExp`. Ми можемо створити новий об’єкт RegExp, передавши у конструктор шаблон, за яким ми будемо шукати текст.

```Javascript
// Приклад шаблону для знаходження збігу зі словом "привіт"
let pattern = /привіт/;
// Створюємо новий об’єкт RegExp
let regex = new RegExp(pattern);

// Перевіряємо, чи містить рядок "Привіт, як справи?" шаблон
regex.test("Привіт, як справи?"); // поверне true
```

Окрім цього, ми можемо використовувати різні модифікатори, щоб зробити наші пошуки більш гнучкими. Наприклад, модифікатор `i` дозволяє ігнорувати регістр символів, а модифікатор `g` знайде всі входження шаблону у тексті.

```Javascript
// Приклад шаблону для знаходження збігу зі словом "hello" та букви "o" будь-яку кількість разів
let pattern = /hello+?/gi;
// Створюємо новий об’єкт RegExp
let regex = new RegExp(pattern);

let text = "Hello, how are you doing?";
// Пошук у тексті знайде слово "Hello", а пошук у шаблоні знайде слово "hello", оскільки використовується модифікатор "i"
regex.test(text); // поверне true

// Пошук у шаблоні знайде слова "hello" та "o", оскільки використовується модифікатор "g"
text.match(regex); // поверне ["Hello", "o"]
```
