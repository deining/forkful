## Чому

Програмування на Javascript є важливою та популярною навичкою, особливо для розробників веб-додатків та програм. Один із важливих аспектів програмування на Javascript - це робота з командними аргументами. Якщо ви ще не знайомі з цим концептом, то варто дізнатися більше про нього. Тому, читайте далі, щоб дізнатися, як зчитувати командні аргументи у програмах на Javascript.

## Як

Для чтения командных аргументов в програмах на Javascript, найкраще використовувати вбудований модуль `process` і його метод `argv`. Давайте зробимо невеликий приклад, який покаже, як це працює.

```Javascript
// index.js
const args = process.argv;
console.log(args);

//node index.js test1 test2 test3
```

У цьому прикладі ми перший аргумент вважаємо назвою файлу, який ми запускаємо у терміналі. Далі, будь-які інші аргументи, які ми вводимо у терміналі під час запуску програми, будуть виведені у вигляді масиву `args`. У цьому випадку, ми отримаємо такий результат:

```Javascript
[
  '/usr/local/bin/node',
  '/Users/User/Documents/index.js',
  'test1',
  'test2',
  'test3'
]
```

Таким чином, ми можемо отримати доступ до командних аргументів та їх значень у нашій програмі на Javascript.

## Deep Dive

При роботі з командними аргументами, важливо знати, що `process.argv` повертає значення у вигляді рядка. Це означає, що якщо ви хочете отримати значення у вигляді числа або булевого типу, ви повинні самостійно здійснити конвертацію. Також, важливо знати, що перший аргумент `process.argv` є шляхом до файлу, який ми запускаємо у терміналі.
