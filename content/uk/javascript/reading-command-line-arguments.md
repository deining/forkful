## Чому

Зчитування аргументів командного рядка є важливою частиною програмування на Javascript. Це дозволяє передавати користувачеві вхідні дані при запуску програми, що дозволяє програмі взаємодіяти з користувачем та зробити її більш гнучкою та корисною.

## Як це зробити

Для зчитування аргументів командного рядка, використовується об'єкт `process` та метод `argv`. Приклад коду нижче показує, як ми можемо отримати перші два аргументи командного рядка та вивести їх на екран у вигляді рядка.

```Javascript
const args = process.argv.slice(2); // збережемо аргументи в змінну args
console.log("Перший аргумент: " + args[0]); // виведемо перший аргумент
console.log("Другий аргумент: " + args[1]); // виведемо другий аргумент
```

При виконанні цього коду з наступними командами командного рядка:

`node index.js hello world`

Ми отримаємо наступний вихід:

```
Перший аргумент: hello
Другий аргумент: world
```

## Глибокий занурення

Крім щойно описаних основ, зчитування аргументів командного рядка також може бути використане для більш складних операцій, таких як валідація вхідних даних та визначення режимів роботи програми залежно від переданих аргументів. 

Наприклад, ми можемо перевірити, чи користувач передав достатню кількість аргументів для запуску нашої програми і повернути повідомлення про помилку, якщо це не так.

```Javascript
function validateArgs(args) {
  // перевіримо, чи передано достатню кількість аргументів
  if (args.length < 3) {
    console.log("Для запуску програми необхідно передати принаймні 3 аргументи.");
    return false; // повернемо значення false, щоб зупинити виконання коду
  }
  return true;
}

const args = process.argv.slice(2); // збережемо аргументи в змінну args
// перевіримо, чи передано достатню кількість аргументів перед виконанням інших дій
if (!validateArgs(args)) {
  process.exit(1); // завершимо виконання програми з кодом помилки
}
// код для виконання, якщо передано достатню кількість аргументів
```
