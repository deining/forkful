## Чому

Написання текстового файлу є важливою частиною програмування, оскільки це дозволяє зберігати дані для подальшого використання та обробки. Також це зручний спосіб редагування та збереження коду.

## Як

```Javascript
// Код для створення текстового файлу
const fs = require("fs"); // Підключення модуля для роботи з файловою системою

let data = "Це приклад тексту, який буде записаний у файл."; // Змінна з текстом, який буде записаний у файл
fs.writeFile("example.txt", data, (err) => { // Виклик функції writeFile для створення файлу з назвою "example.txt" і змістом змінної "data"
    if (err) {
        console.log(err); // Виведення помилки, якщо вона виникне
    } else {
        console.log("Текстовий файл успішно створений та заповнений!"); // Виведення повідомлення про успішне створення та заповнення файлу
    }
});
```

При виконанні цього коду, у вас буде створений текстовий файл з назвою "example.txt", який містить текст з змінної `data`. Також, якщо у файлі вже є якісь дані, цей код їх перезапише новим текстом.

## Глибоке погруження

Окрім створення текстового файлу, ви можете також читати та редагувати вже існуючі файли з допомогою Node.js. Для цього використовується методи `fs.readFile` та `fs.appendFile` з подібним синтаксисом до `fs.writeFile`. Більше інформації про ці методи та роботу з файловою системою можна знайти у документації Node.js.
