##Чому: 
Написання текстового файлу є важливим кроком у розробці програмного забезпечення, оскільки це дозволяє зберігати та обмінюватися даними між різними системами та платформами.

##Як:
Нижче наведений приклад коду на мові Javascript для створення та запису текстового файлу:

```Javascript
// Підключення модуля "fs" (система файлів)
const fs = require('fs');
// Створення текстового файлу з назвою "example.txt"
fs.writeFile('example.txt', 'Привіт це приклад тексту у файлі', (err) => {
    // Звернення до помилок
    if (err) throw err;
    // Завершення процесу
    console.log('Текстовий файл створено та успішно записано!');
});
```

В результаті виконання цього коду буде створено файл "example.txt" та в нього буде записаний текст "Привіт це приклад тексту у файлі".

##Глибока погрузка:
Щоб більш детально розібратися з написанням текстового файлу, рекомендуємо краще ознайомитися з документацією Node.js та модулем "fs". Тут ви знайдете більше інформації про різні методи запису та роботи з файлами.
