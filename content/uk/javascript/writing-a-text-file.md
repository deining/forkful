## Чому

Написання текстового файлу є важливим елементом програмування, оскільки він дозволяє зберігати та зчитувати дані у структурованому форматі. Така можливість дозволяє легше керувати даними та зменшує ризик втрати інформації.

## Як

Існує декілька способів написання текстових файлів у Javascript. Один з них - використання вбудованої функції `writeFile()` з модуля `fs`. Нижче показано приклад коду, який створює новий текстовий файл з даними та виводить його зміст:

```Javascript
const fs = require('fs');

// створення текстового файлу
fs.writeFile('data.txt', 'Це зміст нового файлу!', (err) => {
  if (err) throw err;
  console.log('Текстовий файл створено успішно.');

  // зчитування та виведення змісту текстового файлу
  fs.readFile('data.txt', 'utf8', (err, data) => {
    if (err) throw err;
    console.log(data);
  });
});
```

Вийшовший результат буде виглядати так:

```
Текстовий файл створено успішно.
Це зміст нового файлу!
```

Щоб додати дані до вже існуючого текстового файлу, можна використовувати функцію `appendFile()`. Вона додає нові дані до кінця файлу, не змінюючи його вмісту.

```Javascript
// додавання нових даних до існуючого текстового файлу
fs.appendFile('data.txt', '\nЦе нові дані!', (err) => {
  if (err) throw err;
  console.log('Дані додано успішно.');

  // зчитування та виведення змісту текстового файлу
  fs.readFile('data.txt', 'utf8', (err, data) => {
    if (err) throw err;
    console.log(data);
  });
});
```

Вийшовший результат буде виглядати так:

```
Дані додано успішно.
Це зміст нового файлу!
Це нові дані!
```

## Deep Dive

Існують інші способи написання текстових файлів у Javascript, наприклад, з використанням пакету `path` або за допомогою стандартної функції `FileWriter` у браузері. Також можна встановити різноманітні параметри, щоб змінювати форматування та кодування текстового файлу.
