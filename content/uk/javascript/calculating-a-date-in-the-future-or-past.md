## Чому

Розрахунок дати в майбутньому або минулому є важливою задачею для багатьох розробників, особливо в сфері веб-розробки. Наприклад, потрібно встановити термін проведення майбутнього заходу або замовлення, або ж дату створення файлу. Це може також бути корисно для роботи з датами в ігрових програмах або створення календарів.

## Як

Використання Javascript для розрахунку дати в майбутньому або минулому дуже просте і займає всього кілька кроків. Спочатку нам потрібно встановити поточну дату за допомогою об'єкту `Date()`. Далі, використовуючи різницю між поточною датою і бажаною датою, ми можемо встановити потрібну дату за допомогою вбудованих методів, наприклад `setDate()` або `setFullYear()`. Нижче наведено приклад коду та вихідні дані:

```Javascript
let currentDate = new Date();

// Розраховуємо дату за 5 днів вперед
currentDate.setDate(currentDate.getDate() + 5);
console.log(currentDate);

// Розраховуємо дату за 1 рік назад
currentDate.setFullYear(currentDate.getFullYear() - 1);
console.log(currentDate);

// Виводить:
// Date(2021, 9, 18) - для першого прикладу
// Date(2020, 9, 18) - для другого прикладу
```

Ці приклади показують, як ми можемо легко маніпулювати датами за допомогою вбудованих методів `setDate()` та `setFullYear()`. Можна також використовувати інші методи, такі як `setMonth()`, `setHours()`, `setMinutes()` для роботи зі специфічними частинами дати.
