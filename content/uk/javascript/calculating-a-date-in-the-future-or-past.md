## Чому

Час настільки важливий для всіх нас, що ми постійно хочемо знати, які дати на нас чекають в майбутньому або які минулі дати вже пройшли. Це робить обчислення дат в майбутньому або минулому дуже важливим для програмістів. Це дозволяє нам створювати корисні програми та інструменти для вирішення різноманітних задач, які стикаємося щодня.

## Як до цього підійти

Розрахунки дат у майбутньому або минулому можна зробити дуже просто за допомогою Javascript. Мова програмування JavaScript має спеціальну функцію Date, яка дозволяє нам отримувати та обробляти різні дати. Давайте подивимось на приклад коду, який показує, як отримати дату, що буде через 5 днів:

```javascript
let currentDate = new Date();
currentDate.setDate(currentDate.getDate() + 5);
console.log(currentDate);
```

В результаті виконання цього коду у нас з'явиться дата, яка буде через 5 днів від поточної дати. Ми також можемо додавати або віднімати різні значення від дати, щоб отримати потрібну нам дату.

## Глибші деталі

Можна також використовувати функцію Date, щоб обчислити різні дати у майбутньому або минулому з використанням різних параметрів, таких як місяці, роки, дні та інші. Наприклад, для отримання дати 7 липня 2022 року можна використовувати такий код:

```javascript
let futureDate = new Date(2022, 6, 7);
console.log(futureDate);
```

У цьому прикладі ми передаємо потрібні значення для створення дати - рік, місяць та день. Для місяців використовується нумерація від 0 до 11, тому для отримання липня ми передаємо значення 6. Це допомагає нам більш точно контролювати створювану дату.
