## Чому

Якщо ви працюєте з веб-розробкою або просто цікавитеся програмуванням, ви можете стикнутися з ситуацією, коли потрібно прочитати текстовий файл у своїй програмі. Наприклад, ви можете зберігати дані у текстовому форматі або отримати дані з сервера, що повертаються у вигляді текстового файлу. У цьому випадку потрібно знати, як прочитати цей файл за допомогою Javascript.

## Як це зробити

Існує кілька способів читати текстовий файл за допомогою Javascript. Розглянемо деякі з них в прикладах нижче.

```Javascript
const fs = require('fs'); // підключення модуля для роботи з файловою системою

// читання файлу за допомогою синхронної функції
const data = fs.readFileSync('file.txt', 'utf8');
console.log(data); // вивід зчитаного тексту

// читання файлу за допомогою асинхронної функції
fs.readFile('file.txt', 'utf8', (err, data) => {
  if(err) throw err; // обробка можливих помилок
  console.log(data); // вивід зчитаного тексту
});
```

### Приклад вхідного текстового файлу "file.txt":

```
Hello world!
This is a text file.
```

### Вихідний результат:

```
Hello world!
This is a text file.
```

Розглянемо кожен зі способів більш детально.

- Синхронне читання файлу - це коли весь код зупиняється до того, як файл буде прочитаний. Такий підхід підходить лише в тому випадку, якщо ви точно знаєте, що на виконання цієї операції буде витрачено небагато часу. У протилежному випадку, краще використовувати асинхронний підхід.

- Асинхронне читання файла - це коли програма продовжує виконання і в той час, коли файл зчитується. Цей підхід надає кращу професійну практику, а також дозволяє обробляти помилки, які можуть виникнути під час читання файлу.
