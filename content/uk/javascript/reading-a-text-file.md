## Чому

Програмісти часто потребують читати тексові файли як частину роботи з даними або для виконання певних завдань програм. Знання цього процесу допомагає вдосконалити уміння програмування та полегшує роботу з даними.

## Як

Для читання текстового файлу використовується функція `readFile` з модуля `fs` в мові Javascript. Спочатку потрібно підключити модуль за допомогою ключового слова `require`. Потім, можна викликати функцію `readFile` та передати їй шлях до файлу, що потрібно прочитати, та функцію зворотнього виклику, яка отримає дані з файлу у вигляді параметра. Ось приклад коду та виведення в консолі:

```Javascript
const fs = require('fs');
fs.readFile('text.txt', (err, data) => {
    if (err) throw err;
    console.log(data.toString());
});
```
В цьому прикладі, ми читаємо файл з назвою "text.txt" та виводимо його вміст у консоль. Застосовуючи метод `toString()`, ми перетворюємо дані з файлу в рядок перед виведенням.

## Глибока занурення

Читання текстового файлу може бути складнішим, якщо файл має більш сучасний формат, наприклад JSON або CSV. В такому випадку, необхідно використовувати додаткові методи, які допоможуть зчитати та розшифрувати дані з файлу. Також, можливе використання функції `writeFile` для створення нового файлу та запису даних у нього.
