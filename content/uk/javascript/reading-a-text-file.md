## Чому

Програмування на Javascript є одним з найбільш популярних напрямків в веб-розробці. І часто, коли ми створюємо веб-додатки, нам потрібно прочитати і обробити дані з текстових файлів. Тому, розуміти як працювати з ними є важливим для будь-якого програміста на Javascript.

## Як

У нашому прикладі ми будемо читати текстовий файл з назвою "data.txt". Для цього використовується вбудована функція "fs" в Node.js, яка надає доступ до функцій для роботи з файловою системою. Щоб прочитати файл, використовується метод "readFileSync", який зчитує файл і повертає його вміст у вигляді рядка.

```
```Javascript
const fs = require('fs'); 
// підключаємо модуль для роботи з файловою системою

let data = fs.readFileSync('data.txt', 'utf8'); 
// читаємо файл і зберігаємо вміст у змінну

console.log(data); 
// виводимо вміст файлу у консоль
```

Вищенаведений приклад демонструє простий спосіб прочитати текстовий файл на Javascript і вивести його вміст у консоль. Крім того, можна використовувати метод "writeFileSync" для створення нового текстового файлу або перезапису існуючого.

## Більш глибоке дослідження

Наш приклад демонструє лише базовий спосіб читання та запису текстових файлів на Javascript. Проте, існує багато інших функцій та методів, які можуть бути корисними при роботі з файловою системою, наприклад, визначення шляхів до файлів, перевірка наявності файлу та інше.

Також, варто зазначити, що для більших файлів може стати неефективним використання методу "readFileSync", оскільки він блокує виконання інших команд до завершення зчитування файлу. Тому, для обробки великих файлів доцільніше використовувати асинхронні методи, наприклад "readFile", які не блокують інші команди.
