Що і Чому?

Читання текстового файлу - це процес, який дозволяє програмістам отримати дані з файлу, що зберігається на комп'ютері або в мережі. Це дозволяє працювати з великими об'ємами даних або зберігати дані для подальшого використання.

Як це зробити?

```Javascript
const fs = require('fs'); // підключення модуля Node.js для роботи з файлами

// прочитання текстового файлу
fs.readFile('./myfile.txt', 'utf8', function(err, data) {
  if (err) {
    console.error(err); // якщо виникла помилка, виводимо її в консоль
    return; // вихід з функції
  }
  console.log(data); // виводимо дані з файлу в консоль
});
```

Результат виконання коду буде виглядати так:

```Javascript
Hello world!
This is a text file.
```

Глибоке занурення

Читання текстового файлу є стандартною операцією у більшості програмувальних мов. Раніше це вимагало використання бібліотек або низькорівневих методів, але зараз завдяки модулю fs у Node.js це стало сильно спрощеним. Існують також альтернативні способи читання файлів, наприклад, використання потоків у Node.js, але вони є складнішими у реалізації.
