## Чому

Ви могли б замислитися, чому люди видаляють символи, що співпадають з певним шаблоном. Це недоступно для звичайних користувачів, але для програмістів це може бути корисним для швидкого та ефективного оброблення даних в програмах.

## Як видалити символи за шаблоном

Є кілька шляхів додати видалення символів за шаблоном в ваш код на Javascript. Найпростішим способом є використання методу `replace()` з регулярним виразом. Давайте розглянемо приклад:

```Javascript
// створюємо рядок з даними
let text = "Це. є. приклад. рядка.";
// використовуємо метод replace() з регулярним виразом для видалення крапок
let newText = text.replace(/\./g, ""); 
// виводимо результат
console.log(newText); // виведе "Це є приклад рядка"
```

Через використання регулярного виразу `/\.g`, ми вказали програмі, що потрібно замінити всі входження крапок в рядку на порожні символи, що в результаті дає нам оновлений рядок без крапок.

Ще одним популярним способом є використання циклу `for` для перебору символів у рядку та видалення їх, якщо вони відповідають шаблону. Давайте розглянемо цей приклад:

```Javascript
// створюємо рядок з даними
let text = "Це. є. приклад. рядка.";
// перетворюємо рядок у масив з окремими символами
let letters = text.split("");
// створюємо порожній масив для збереження нового рядка
let newText = [];

// проходимо циклом через кожен символ
for(let i = 0; i < letters.length; i++) {
  // перевіряємо, чи символ відповідає шаблону крапки
  if(letters[i] !== ".") {
    // якщо символ не співпадає з крапкою, то додаємо його в новий рядок
    newText.push(letters[i]);
  }
}
// перетворюємо масив у рядок та виводимо результат
console.log(newText.join("")); // виведе "Це є приклад рядка"
```

Цей метод може бути корисним, якщо ви хочете замінити не тільки один символ, а кілька.
