## Чому

Знаходження довжини рядка в Javascript може бути дуже корисним у багатьох сценаріях програмування, наприклад, для валідації введеного користувачем тексту, знаходження максимальної довжини для обрізання рядка або для обчислення кількості символів у рядку.

## Як це зробити

Для початку, нам потрібно отримати сам рядок, для якого ми хочемо знайти довжину. Це може бути будь-який рядок, введений користувачем або збережений у змінній. У нашому прикладі ми використаємо текст "Привіт світ!".

```Javascript
let str = "Привіт світ!";
```

Щоб знайти довжину рядка, ми можемо використати вбудовану функцію `length`, яка поверне нам кількість символів у рядку.

```Javascript
let str = "Привіт світ!";
console.log(str.length);
```

Очікуваний вивід: `12`

Також, можна використовувати цей метод у поєднанні з методом `trim`, який видалить всі пробіли з початку і кінця рядка, і отримати більш точну довжину:

```Javascript
let str = " Привіт світ! ";
console.log(str.trim().length);
```

Очікуваний вивід: `12`

## Deep Dive

Для тих, хто більше цікавиться деталями, варто відмітити, що довжина рядка обраховується за допомогою властивості `length` у рядку. Це властивість, а не метод, оскільки ми не викликаємо його з допомогою дужок `()`. Замість цього ми просто звертаємося до нього, як до властивості об'єкта.

Також, слід зазначити, що метод `length` підраховує не тільки літери, але й будь-які символи, які є частинами рядка, включаючи пробіли, числа та спеціальні символи.
