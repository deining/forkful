# Чому

Чому важливо знати, як знаходити довжину рядка? В програмуванні дуже часто потрібно працювати з текстовими даними, і визначення довжини рядка є важливою операцією. Наприклад, для перевірки правильності введеного паролю або для обмеження введеного користувачем тексту.

# Як вивчити

Щоб знаходити довжину рядка у Javascript, ми можемо використовувати вбудовану функцію .length. Наприклад:

```Javascript
let text = "Привіт, світ!";
console.log(text.length); // виведе 13, оскільки є 13 символів у рядку
```

Також важливо пам'ятати, що довжина рядка обчислюється починаючи з нуля. Тобто, перший символ має індекс 0, другий - 1, і так далі. Наприклад:

```Javascript
let text = "Javascript";
console.log(text[0]); // виведе "J"
console.log(text[5]); // виведе "c"
```

# Глибоке поглиблення

Знаходження довжини рядка є не такою простою операцією, як може здатися на перший погляд. Варіантів підрахунку довжини дещо більше, залежно від типу даних, з якими ми працюємо. Наприклад, якщо у нас є рядок з кирилицею, то для коректного визначення довжини нам потрібно використовувати функцію .length замість властивості .length, оскільки кількість байтів у різних символах у кирилиці може бути різною.

Також важливо пам'ятати, що функція .length повертає число, а не реальну кількість символів. Це означає, що якщо в нашому рядку є спеціальні символи, такі як \n (переклад рядка) або \t (табуляція), то вони також підраховуються як символи.
