## Чому

Програмування - це постійна життєва навичка, і кожен розробник прагне вдосконалюватись. Одним із способів покращити свої навички є вивчення регулярних виразів. Це потужний інструмент, який дозволяє швидше та ефективніше обробляти текстові дані.

## Як це зробити

Щоб почати використовувати регулярні вирази в своїй програмі, вам буде потрібно підключити заголовочний файл `<regex>` і використовувати клас `std::regex`.
Наприклад, давайте створимо програму, яка перевіряє, чи введений користувачем текст містить лише цифри:

```C++
#include <iostream>
#include <regex>

int main() {
  std::string text;
  std::cout << "Введіть текст: ";
  getline(std::cin, text);
  std::regex digits_regex("[0-9]+");
  if (std::regex_match(text, digits_regex)) {
    std::cout << "Текст містить лише цифри.\n";
  }
  else {
    std::cout << "Текст не містить лише цифри.\n";
  }
  return 0;
}
```

Виходячи з введеного тексту, програма виведе повідомлення з відповідним результатом.
Наприклад, якщо користувач введе "123", програма виведе "Текст містить лише цифри.", а якщо "abc", то "Текст не містить лише цифри.".

## Глибше погрузження

Регулярні вирази дозволяють не тільки перевіряти текст на відповідність певному шаблону, але і виконувати пошук і заміну в тексті. Для цього існують різні методи класу `std::regex`, такі як `search()` і `replace()`.

Наприклад, давайте модифікуємо нашу програму, щоб вона заміняла всі цифри в тексті на зірочки:

```C++
#include <iostream>
#include <regex>

int main() {
  std::string text;
  std::cout << "Введіть текст: ";
  getline(std::cin, text);
  std::regex digits_regex("[0-9]+");
  std::cout << std::regex_replace(text, digits_regex, "*") << "\n";
  return 0;
}
```

Якщо користувач введе "ab1cd2ef3", програма виведе "ab*cd*ef*".
