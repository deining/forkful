## Чому

Навіть найбільш досвідчені програмісти час від часу стикаються з необхідністю обробки тексту в своїх програмах. І саме для цього були створені регулярні вирази - потужний інструмент для роботи з текстом у програмуванні. Використання регулярних виразів дозволяє швидко та ефективно здійснювати пошук, заміну та перевірку рядків у тексті. Це робить програмування більш продуктивним і зекономлює час.

## Як використовувати регулярні вирази в C++

Програмування з використанням регулярних виразів в C++ дуже просте. Для цього вам потрібно включити бібліотеку `<regex>` та використовувати потрібні класи та методи. Давайте розглянемо декілька прикладів, щоб краще зрозуміти цю тему.

```C++
// Приклад використання регулярного виразу для пошуку слова "hello"
#include <iostream>
#include <regex>

int main() {
    std::string text = "Hello World, hello C++";

    // Створюємо об'єкт регулярного виразу
    std::regex regex("hello");

    // Знаходимо входження слова "hello" в тексті
    std::smatch matches;
    std::regex_search(text, matches, regex);

    // Виводимо результат
    std::cout << "Знайдено: " << matches.size() << " входження" << std::endl;

    return 0;
}

// Output: Знайдено: 2 входження
```

```C++
// Приклад використання регулярного виразу для заміни всіх цифр у тексті на знак питання "?"
#include <iostream>
#include <regex>

int main() {
    std::string text = "abc123xyz!";

    // Створюємо об'єкт регулярного виразу для пошуку цифр
    std::regex regex("[0-9]");

    // Замінюємо всі цифри на знак питання
    std::string result = std::regex_replace(text, regex, "?");

    // Виводимо змінений текст
    std::cout << "Результат: " << result << std::endl;

    return 0;
}

// Output: Результат: abc???xyz!
```
