## Причина

Найпростіша відповідь - регулярні вирази роблять вашу роботу з обробки тексту більш простою і ефективною. Вони дозволяють знаходити, замінювати та видаляти певні рядки або символи з тексту за заданим шаблоном.

## Як використовувати регулярні вирази в C++

У першу чергу, необхідно включити бібліотеку `<regex>`, яка надає функції та класи для роботи з регулярними виразами. Наступним кроком є створення об'єкту `regex`, в якому буде зберігатися шаблон для пошуку. Наприклад, для знаходження всіх чисел у тексті, ми можемо використати шаблон `"[0-9]+"`. Далі, за допомогою методу `regex_search` ми можемо перевірити, чи підходить певний рядок до нашого шаблону. Якщо потрібно замінити або видалити знайдений рядок, можна використати метод `regex_replace`. Останнім кроком - виведення результату на екран. Ось приклад коду:

```C++
#include <iostream>
#include <regex>

using namespace std;

int main() {
  // створення об'єкту regex з шаблоном для пошуку чисел
  regex numberPattern("[0-9]+");
  string text = "Сьогодні на 203 п'єстах ви зможете побачити більше 1500 скульптур.";

  // пошук збігу шаблону у тексті
  smatch matches;
  regex_search(text, matches, numberPattern);

  // виведення знайденого числа на екран
  cout << "Знайдене число: " << matches[0] << endl;

  // заміна числа у тексті
  string newText = regex_replace(text, numberPattern, "100");
  cout << "Новий текст: " << newText << endl;

  return 0;
}
```

Результат виконання програми буде наступним:

```
Знайдене число: 203
Новий текст: Сьогодні на 100 п'єстах ви зможете побачити більше 100 скульптур.
```
