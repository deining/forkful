## Чому

У програмуванні час від часу виникає потреба видаляти певні символи, які відповідають певному шаблону. Це може бути корисно в таких ситуаціях, як заміна певних символів у тексті або очищення вхідних даних перед обробкою.

## Як

Для того, щоб видалити символи, які відповідають певному шаблону, необхідно використати стандартну бібліотеку C++ та функцію `erase()` для рядків. Приклад коду наведено нижче:

```C++
#include <iostream>
#include <string>

int main() {
  // вхідний рядок
  std::string str = "Це [тест] для видалення.";

  // видаляємо всі квадратні дужки та їх вміст
  size_t start_pos = str.find('[');
  size_t end_pos = str.find(']');
  str.erase(start_pos, end_pos - start_pos + 1);

  // виводимо результат
  std::cout << str << std::endl;

  return 0;
}

// Output: Це  для видалення.
```

Як бачимо, ми використовуємо функцію `find()` для знаходження позицій входження шаблону у рядок і функцію `erase()` для його видалення. Цей приклад можна адаптувати для видалення інших символів або паттернів.

## Глибока занурення

Якщо ви хочете детальніше дізнатись про видалення символів за певним шаблоном, варто детальніше ознайомитись з функцією `erase()` та її параметрами. Наприклад, є можливість вказати початкову позицію та кількість символів для видалення замість використання функції `find()`.
