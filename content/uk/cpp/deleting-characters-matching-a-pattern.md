## Що & Чому?

Ви вже бачили, як видаляти окремі символи зі строки. Але що, якщо вам потрібно видалити не один символ, а всі символи, які відповідають певному шаблону? Для цього існує функція `std::erase`, яка дозволяє видаляти символи зі строки, що відповідають певному шаблону. Це особливо зручно, якщо ми маємо те ж саме повторюване джерело символів, яке потрібно видалити зі строки, наприклад, лишні пробіли.

## Як це зробити:

```C++
#include <string>
int main(){
  std::string str = "Привіт,   світ!";
  std::erase(str, ' '); //видаляємо всі пробіли із строки
  std::cout << str << '\n'; // результат: "Привіт,світ!"
return 0;
}
```

Зверніть увагу, що для використання функції `std::erase` нам потрібно включити бібліотеку `string`. Ми також можемо видаляти не тільки один символ, але і всі символи, що відповідають певному шаблону. Наприклад:

```C++
#include <string>
int main(){
  std::string str = "Привіт,світ";
  std::erase(str, 'і'); // видаляємо всі символи 'і' зі строки
  std::cout << str << '\n'; // результат: "Првт,свт"
return 0;
}
```

## Глибоке погруження

Видалення символів, що відповідають певному шаблону, є частиною стандартної бібліотеки C++, і це працює приблизно так само для всіх комбінацій символів. Але для єдиного символу, який вам може здатися незначним, ви можете використовувати метод `find_and_replace` замість цього, або навіть просто змінити символ, який ви хочете видалити, на інший символ.

## Підказки

Є також інші способи видалення символів зі строки, наприклад, застосування регулярних виразів або використання інших методів зі строки, таких як `substr` або `replace`.
