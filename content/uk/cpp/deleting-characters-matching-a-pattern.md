## Чому

У програмуванні, іноді виникає необхідність у видаленні символів, які відповідають певному шаблону. Наприклад, при обробці текстових файлів або рядків, можуть зустрітися символи, які потрібно замінити або видалити. Такі ситуації можуть виникнути при роботі зі словниками, де необхідно вилучити певні слова або літери.

## Як це зробити

Для видалення символів, що відповідають певному шаблону, можна використовувати функцію `erase()` зі стандартної бібліотеки C++. Ця функція дозволяє видалити символи за певним інтервалом або за допомогою умов. Нижче наведено приклад коду, який видаляє усі голосні літери зі строкового рядка:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    string text = "Привіт, світе!";
    
    // Проходимося по кожному символу в рядку
    for (int i = 0; i < text.length(); i++) {
        
        // Якщо символ - голосна літера, видаляємо його
        if (text[i] == 'а' || text[i] == 'е' || text[i] == 'є' || text[i] == 'и' || text[i] == 'і' || text[i] == 'о' || text[i] == 'у' || text[i] == 'ю' || text[i] == 'я') {
            text.erase(i, 1);
        }
    }
    
    // Виводимо результат
    cout << text; // Првт, свт!
    
    return 0;
}
```

## Вглиб

Функція `erase()` також підтримує видалення символів за допомогою інтервалу. Наприклад, якщо у нас є строковий рядок `text = "Абракадабра"` і ми хочемо видалити 4 символи, що починаються з 4-го символу, ми можемо використати наступний код:

```C++
text.erase(4, 4); // Результата буде "Абрабра"
```

Можна також використовувати функцію `replace()`, яка замінює символи за певним інтервалом на новий символ або рядок. Для цього потрібно передати ще два аргументи - початкове значення і довжину інтервалу. Нижче наведено приклад коду, який замінює усі цифри у строковому рядку на символ `*`:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    string text = "Some123random45numbers78";
    
    // Замінюємо усі цифри на символ "*"
    text.replace(text.find_first_not_of("0123456789"), text.find_last_not_of("0123456789") + 1, text.length(), '*');
    
    // Виводимо результат
    cout << text; // Some***random**numbers**
