# Чому 

Видалення символів, що відповідають певному шаблону, може бути корисним для збільшення ефективності коду, особливо при обробці великих об'ємів даних. Також, це може бути необхідно при пошуку та заміні певних елементів у рядках.

## Як це зробити

Для видалення символів за певним шаблоном, можна використовувати функцію `erase` з стандартної бібліотеки C++. Ця функція приймає два параметри: індекс початку та кількість символів, які потрібно видалити. 

```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    string str = "Hello world";

    // Видаляємо перші 5 символів
    str.erase(0, 5);

    cout << str << endl; // виведе "world"
    
    return 0;
}
```

Наведений приклад використовує функцію `erase` для видалення перших 5 символів змінної `str`. Зверніть увагу, що індексуються символи починаючи з 0.

## Глибший аналіз

Крім видаляння певної кількості символів, функція `erase` також може приймати один параметр - індекс символу, з якого починається видалення, тоді кількість символів залишається за замовчуванням і видаляється до кінця рядка. Також, можна використовувати цю функцію для видалення підрядка. Для цього треба передати функції `find()` індекс початку та кінця підрядка, який потрібно видалити. 

```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    string str = "Hello world";

    // Видаляємо слово "world" з рядка
    str.erase(str.find("world"), 5);

    cout << str << endl; // виведе "Hello"
    
    return 0;
}
```

У даному випадку, функція `erase` приймає значення, яке повертає функція `find()`, яка знаходить індекс підрядка `world` у змінній `str`, а потім видаляє 5 символів починаючи з цього індексу.
