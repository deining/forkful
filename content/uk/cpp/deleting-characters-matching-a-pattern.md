## Why
Іноді при програмуванні може виникнути потреба видалити певні символи зі строки. Наприклад, забрати всі пробіли або спеціальні символи. У цих випадках, видалення символів з певним шаблоном може бути корисним для оптимізації або обробки даних.

## How To
У даній статті, ми покажемо як видалити символи з текстової строки використовуючи C++. Для цього ми будемо використовувати функцію `erase()` з бібліотеки `string`. Давайте подивимося на приклад:

```C++
// Приклад коду для видалення символів зі строки
#include <iostream>
#include <string>

using namespace std;

int main() {
  // Створюємо рядок з текстом для обробки
  string str = "Ця строки містить лишні пробіли та крапки.";

  // Використовуємо цикл for для проходження по кожному символу в рядку
  for (int i = 0; i < str.length(); i++) {
    // Перевіряємо чи символ є пробілом чи крапкою і видаляємо його за допомогою функції `erase()`
    if (str[i] == ' ' || str[i] == '.') {
      str.erase(i, 1);
      // Зменшуємо лічильник щоб уникнути пропускання наступного символу
      i--;
    }
  }

  // Виводимо результат
  cout << str << endl; // Результат: "Цястрокіміститьлишніпробілитакрапки"
  return 0;
}
```

## Deep Dive
Функція `erase()` видаляє певну кількість символів починаючи з вказаної позиції в рядку. У нашому прикладі, ми вказали позицію `i` (поточний символ) та кількість символів `1`, оскільки ми хочемо видаляти тільки один символ в кожній ітерації циклу.

Також, варто зазначити, що ми змінюємо оригінальну строку, оскільки функція `erase()` діє без повернення нового рядка. Якщо ви не хочете змінювати оригінальну строку, можна скористатися функцією `substr()` для повернення бажаної частини рядка без видалених символів.
