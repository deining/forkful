## Чому

В цьому блозі ми розглянемо, чому важливо перетворювати рядок на верхній регістр. Це може бути корисно при роботі з текстовими даними, наприклад, при пошуку або порівнянні рядків.

## Як це зробити

Програмування в мові C++ надає нам багато зручних інструментів для роботи з рядками. Для перетворення рядку на верхній регістр, ми можемо використати функцію `toupper()` від стандартної бібліотеки C. Перш за все, давайте створимо змінну `str` зі значенням "Hello, World". Потім ми створимо цикл, який буде проходитися по кожному символу рядка і перетворювати його у верхній регістр з допомогою функції `toupper()`. Зверніть увагу, що функція `toupper()` повертає ASCII код верхньої літери, тому ми також повинні привести результат до типу `char`. Останнім кроком буде виведення зміненої змінної `str` на екран. Ось як це виглядає в коді:

```C++
#include <iostream>
#include <cctype>

int main() {
    char str[] = "Hello, World";
    for (size_t i = 0; i < strlen(str); i++) {
        str[i] = char(toupper(str[i]));
    }
    std::cout << str;
    return 0;
}
```

Результат на екрані буде виглядати так:

`HELLO, WORLD`

## Детальний аналіз

Перетворення рядку на верхній регістр може бути корисно при виконанні різних операцій з текстом. Наприклад, якщо ми хочемо порівняти два рядки, то для більшої точності ми можемо спочатку перетворити їх обидва на верхній регістр, щоб уникнути помилок через різницю в написанні літер. Також, це може бути корисно при пошуку певної частини рядку або при обробці великих обсягів даних.
