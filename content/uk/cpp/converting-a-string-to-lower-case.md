## Чому 
Відомо, що часто у програмуванні потрібно конвертувати рядок у нижній регістр. Це може бути необхідно для подальшої обробки тексту, порівняння рядків або просто для зручності користувача. Хоча це може здатися простим завданням, але важливо знати правильний підхід до його вирішення у програмі.

## Як це зробити
Для конвертування рядка у нижній регістр у мові програмування C++, можна скористатися функцією `tolower` з бібліотеки `<cctype>`. Для цього спочатку потрібно підключити цю бібліотеку до програми. Далі, необхідно створити змінну типу `char` та привести до нижнього регістру кожен символ цього рядка за допомогою циклу `for`.

```C++
#include <iostream>
#include <cctype>

int main() {
    // Приклад рядка
    std::string str = "Приклад РЯДКА для конвертування";
    
    // Перетворення рядка у нижній регістр
    for (int i = 0; i < str.length(); i++) {
        str[i] = tolower(str[i]);
    }
    
    // Вивід результату у консоль
    std::cout << str << std::endl;
    
    return 0;
}
```

Вищенаведений код виведе на консоль рядок `приклад рядка для конвертування`.

## Глибокий аналіз
Цей підхід до конвертування рядка у нижній регістр використовує функцію `tolower`, яка переводить символ у нижній регістр лише у випадку, якщо це можливо. Тобто, якщо символ вже є у нижньому регістрі, він залишиться без змін.

Також, варто зазначити, що цей підхід не змінює початковий рядок, а створює новий рядок зі зміненими символами. Якщо необхідно змінити початковий рядок, можна скористатися функцією `strlwr` з бібліотеки `<cstring>`.
