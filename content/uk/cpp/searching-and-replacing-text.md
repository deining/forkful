## Чому

Пошук і замінювання тексту є важливою частиною програмування, оскільки дозволяє ефективніше редагувати та змінювати великі обсяги тексту. Це особливо корисно в програмуванні, де часто потрібно швидко змінювати кілька рядків коду.

## Як це зробити

Для пошуку і заміни тексту в C++, використовується функція `replace()`. Для використання цієї функції спочатку потрібно підключити бібліотеку `<algorithm>`. Потім можна використати цю функцію з наступними параметрами:

- Початковий ітератор: показує на початок ділянки тексту, в якій буде здійснюватися пошук і заміна.
- Кінцевий ітератор: показує на кінець ділянки тексту, в якій буде здійснюватися пошук і заміна.
- Об'єкт, що шукаємо: текст, який потрібно знайти.
- Об'єкт, на який буде замінений знайдений текст.

Наприклад, якщо ми маємо рядок зі словом "програмування" і хочемо замінити його на "кодування", то код буде виглядати наступним чином:

```C++
#include <iostream>
#include <algorithm>
#include <string>
using namespace std;

int main() {
  string text = "Я люблю програмування";
  replace(text.begin(), text.end(), "програмування", "кодування");
  cout << text << endl;
  // Виведе: "Я люблю кодування"
  return 0;
}
```

## Глибока зміна

Функція `replace()` використовує алгоритм пошуку "через простори", тому вона буде замінювати всі входження шуканого тексту, навіть якщо вони перетинаються. Також, вона може замінювати не тільки текст, але і будь-які об'єкти, які можна порівнювати за допомогою `==`.
