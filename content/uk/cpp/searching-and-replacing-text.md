## Для чого

Замінювання тексту - це невід'ємна частина програмування в C++. Ви можете використовувати цей метод для ефективного заміни заданих символів, слів або фраз у вашому коді. Це дозволяє швидко внести потрібні зміни та полегшує процес роботи з багатими текстовими файлами.

## Як

Для початку, давайте розглянемо простий приклад заміни символу у тексті за допомогою функції `replace`:

```
#include <iostream>
#include <string> // увімкнення бібліотеки для роботи зі строками

using namespace std;

int main() {
   string text = "Привіт світ!";
   text.replace(7, 4, "Всесвіт"); // заміна "світ" на "Всесвіт" у тексті
   
   cout << text << endl; // виведе "Привіт Всесвіт!"
   return 0;
}
```

У цьому прикладі ми спочатку підключаємо бібліотеку `string`, оскільки ми будемо працювати зі строками. Далі оголошуємо змінну `text`, яка містить наш початковий текст. При виклику функції `replace`, ми вказуємо позицію, на якій потрібно зробити заміну, та кількість символів, які мають бути замінені. У нашому випадку, `7` означає позицію символу "с", а `4` - кількість символів у слові "світ". На місці цих символів буде вставлено новий текст, який вказаний в третьому параметрі функції.

Також, можливо замінювати символи за допомогою індексації. Для прикладу, використаємо цикл `for` для проходження крізь кожен символ у тексті і заміни кожного пробілу на знак питання:

```
#include <iostream>
#include <string>
using namespace std;

int main() {
   string phrase = "Це нормальний текст";
   
   for (int i = 0; i < phrase.length(); i++) {
      if (phrase[i] == ' ') { // перевірка, чи символ є пробілом
         phrase[i] = '?'; // заміна на знак питання
      }
   }
   
   cout << phrase << endl; // виведе "Це?нормальний?текст"
   return 0;
}
```

Це демонструє, що замінування тексту може бути простим та ефективним способом змінювати текст у вашому коді з мінімальними зусиллями.
