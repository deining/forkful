## Чому

Програмування є не тільки про створення нових програм, але і про вдосконалення існуючих. Заміна тексту є однією з основних технік, яка допомагає виправляти помилки, змінювати інформацію та зберігати чистоту коду. У цій статті ми розглянемо, як шукати та замінювати текст у програмі на мові C++.

## Як це зробити

### Перш за все, давайте створимо змінну для зберігання тексту, який ми будемо заміняти:

```C++
std::string text = "Привіт, Україно! Слава Україні!";
```

Тепер давайте виведемо цей текст на екран, щоб переконатися, що маємо правильне значення:

```C++
std::cout << "Початковий текст: " << text << std::endl;
```

Результат виконання програми буде такий:

```
Початковий текст: Привіт, Україно! Слава Україні!
```

Тепер давайте скористаємося функцією ```find()```, щоб знайти позицію слова, яке ми хочемо замінити. У даному випадку, ми хочемо замінити слова "Україно" на "Світ". У функції ```find()``` ми передаємо два параметри: шуканий текст та позицію, з якої почнемо пошук. Якщо слово не знайдено, то функція поверне значення ```std::string::npos```:

```C++
size_t position = text.find("Україно", 0);
```

Тепер, використовуючи функцію ```replace()```, ми можемо замінити слово на інше. Для цього потрібно передати їй позицію слова, яке ми хочемо замінити, його довжину та нове значення:

```C++
text.replace(position, 7, "Світ");
```

І, нарешті, давайте виведемо оновлений текст на екран, щоб побачити результат:

```C++
std::cout << "Оновлений текст: " << text << std::endl;
```

Результат виконання програми буде такий:

```
Оновлений текст: Привіт, Світ! Слава Україні!
```

Тепер ми успішно замінили слово "Україно" на "Світ" у нашій програмі на мові C++.
