## Чому

Навіщо нам потрібно виводити відладочної інформації? Відладка - це важлива частина процесу програмування, яка допомагає нам знайти та виправити помилки у нашому коді. Виводячи відладочну інформацію, ми можемо зрозуміти як працює наш код та де відбуваються помилки, що дозволить нам покращити якість та ефективність нашого програмного продукту.

## Як це зробити

Для виведення відладочної інформації у C++, ми використовуємо функцію `cout` з бібліотеки `iostream`. Для цього нам потрібно оголосити простір імен `std` та підключити бібліотеку `iostream` до нашого файлу.

```C++
#include <iostream>

int main() {
  int x = 10;
  std::cout << "Змінна x має значення " << x << std::endl;
  return 0;
}
```

У цьому прикладі, ми оголосили змінну `x` та вивели її значення за допомогою функції `cout` разом зі строковими значеннями. Символ `<<` показує що передавати в `cout`, а `std::endl` вказує на кінець рядка. Запускаючи цей код, ми повинні отримати наступний вивід:

```
Змінна x має значення 10
```

Це стандартний шаблон, який ми можемо використовувати для виведення будь-якої інформації, яку нам потрібно перевірити під час виконання нашої програми.

## Глибоке занурення

Крім виведення змінних, ми також можемо використовувати функцію `cout` для виведення повідомлень про різні події у нашій програмі. Наприклад, ми можемо використовувати `cout` для виводу повідомлень під час виконання циклу:

```C++
#include <iostream>

int main() {
  for (int i = 0; i < 5; i++) {
    std::cout << "Цикл використовує ітерацію " << i << std::endl;
  }
  return 0;
}
```

Виводом буде наступне:

```
Цикл використовує ітерацію 0
Цикл використовує ітерацію 1
Цикл використовує ітерацію 2
Цикл використовує ітерацію 3
Цикл використовує ітерацію 4
```

Таким чином, ми можемо контролювати певні аспекти нашої програми та виводити інформацію про ці аспекти під час їх виконання,
