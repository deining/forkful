## Для чого

Цей пост призначений для українських програмістів, які хочуть дізнатися більше про виведення дебаг-інформації у своїх програмах. При відладці програм ми часто використовуємо виведення дебаг-інформації для знаходження і усунення помилок. Тому вміння користуватися цим інструментом є необхідним для ефективної роботи.

## Як

Для виведення дебаг-інформації у своїй програмі, спочатку необхідно підключити бібліотеку `iostream` у своєму програмному коді. Після цього можна використовувати оператор виведення `<<` для виведення значень змінних та іншої інформації у консоль.

Простий приклад коду з виведенням дебаг-інформації виглядає так:

```C++
#include <iostream>

int main()
{
    int num = 5;
    std::cout << "Значення змінної num: " << num << std::endl;

    return 0;
}
```

Результатом виконання цього коду буде виведення `Значення змінної num: 5` у консоль.

Але виведення дебаг-інформації не обмежується лише виведенням значень змінних. Можна також виводити різні повідомлення для зручності відладки, наприклад:

```C++
#include <iostream>

int main()
{
    int a = 5;
    int b = 10;

    // Виведення повідомлення для перевірки роботи if-else
    if (a > b)
    {
        std::cout << "a більше за b" << std::endl;
    }
    else
    {
        std::cout << "a менше або дорівнює b" << std::endl;
    }

    return 0;
}
```

В результаті виконання цього коду у консоль буде виведено повідомлення `a менше або дорівнює b`, означаюче, що умова у `if` не була виконана.
