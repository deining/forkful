## Чому

Займатися роботою з YAML корисно для програмістів, які працюють з конфігураційними файлами та даними у структурованому форматі. YAML - це зручний та зрозумілий спосіб зберігання та обміну інформацією між різними програмами.

## Як

Для роботи з YAML у C++ коді, спочатку потрібно інсталювати бібліотеку yaml-cpp. Тоді можна продовжувати до наступних кроків:

1. Підключіть заголовочний файл yaml-cpp до вашого проекту:

```C++
#include <yaml-cpp/yaml.h>
```

2. Створіть об'єкт типу YAML::Node, який буде використовуватися для зберігання та обробки даних у YAML форматі:

```C++
YAML::Node myNode;
```

3. Завантажте YAML файли за допомогою методу load_file та передайте йому шлях до файлу:

```C++
myNode = YAML::load_file("config.yaml");
```

4. Звертайтеся до значень по ключу за допомогою методу operator[], наприклад:

```C++
int port = myNode["server"]["port"].as<int>();
```

## Deep Dive

У YAML є декілька особливостей, які важливо знати, щоб ефективно працювати з цим форматом. Наприклад:

- Розділення ключів та значень в YAML здійснюється за допомогою двокрапки та відступів, тому важливо дотримуватися правильного форматування.
- Звертання до значень по ключу в більш вкладених структурах може бути достатньо складним. Треба враховувати точність та кількість відступів.
- Для створення масиву значень у YAML використовується простий формат з дужками та розділенням значень комою.

Більше інформації про роботу з YAML можна знайти у [документації yaml-cpp](https://github.com/jbeder/yaml-cpp/wiki).
