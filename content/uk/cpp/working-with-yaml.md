## Для чого 
ЯML - це формат збереження та передачі даних, який є одним з найпопулярніших у світі програмування. Використовувати його дуже зручно і просто, тому процес роботи з даним форматом є обов'язковим для кожного програміста.

## Як використовувати YML в C++
Для початку необхідно підключити бібліотеку розбору YML до свого проекту. Наприклад, якщо ви працюєте в середовищі Visual Studio, ви можете додати бібліотеку з використанням пакетного менеджера NuGet.
Після цього додайте необхідні директиви `#include` у свій код, щоб мати доступ до функцій і класів, що необхідні для роботи з YML форматом.

```C++
#include "yaml-cpp/yaml.h"
```

Тепер вам потрібно вказати шлях до файлу з даними YML, з яким ви будете працювати. Наприклад, якщо ваш файл знаходиться в тій же папці, що й код, просто вкажіть назву файлу:

```C++
YAML::Node data = YAML::LoadFile("my_data.yml");
```

Тепер, використовуючи функції бібліотеки YML, ви можете отримати дані з файлу і виконувати над ними необхідні дії. Наприклад, якщо ваш файл містить дані про користувачів, ви можете отримати їхні імена і вік:

```C++
std::string name = data["user"]["name"].as<std::string>();
int age = data["user"]["age"].as<int>();
```

Окрім зчитування даних, ви також можете створювати нові об'єкти і записувати їх у файл у форматі YML:

```C++
// Створення об'єкту user з іменем та віком
YAML::Emitter out;
out << YAML::BeginMap;
out << YAML::Key << "user";
out << YAML::BeginMap;
out << YAML::Key << "name";
out << YAML::Value << "John";
out << YAML::Key << "age";
out << YAML::Value << 28;
out << YAML::EndMap;
out << YAML::EndMap;

// Запис у файл
std::ofstream file("new_data.yml");
file << out.c_str();
```

## Глибше розуміння YML
На відміну від інших форматів збереження даних, YML не є просто текстовим файлом. Він має структуру, що складається з скалярних значень, масивів та об'єктів. Наприклад, дані у форматі YML можуть виглядати так:

```yaml
user:
  name: John
  age: 28
hobbies:
  - reading
  - hiking
```

В бібліотеці YML використовується об'єкт `Node`, який схожий на контейнер для даних. Він містить ключі та відповідні знач
