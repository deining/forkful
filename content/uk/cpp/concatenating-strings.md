# Чому
Якщо ви працюєте зі стрічками в C ++, то ви, напевно, чули про конкатенацію стрічок. Ця техніка корисна для злиття двох або більше стрічок в одну. Це може бути корисно, коли ви хочете створити повідомлення з декількох частин, або зберегти частини даних в одній стрічці для подальшого використання.

# Як
Для конкатенації стрічок у C ++ використовується оператор "+" або функція "append ()". Давайте подивимося на приклади:

```C++
// Злиття двох стрічок
string str1 = "Привіт";
string str2 = "Світ!";
string result = str1 + str2;
cout << result; // Виведе "ПривітСвіт!"

// Додавання стрічки до існуючої
string str = "Це";
str.append(" дуже корисно!");
cout << str; // Виведе "Це дуже корисно!"
```

# Глибша розробка
Конкатенація стрічок працює за допомогою поєднання пам'яті двох або більше стрічок і створення нової стрічки. Це може бути трохи більш складним під капотом, коли працюєте з великими стрічками або багатьма стрічками одночасно. Також важливо знати, що конкатенація стрічок є більш ефективною за допомогою функції "append ()", оскільки вона не створює додатковий об'єкт.
