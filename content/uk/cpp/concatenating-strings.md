## Почему

З'єднання рядків є важливим процесом у програмуванні, оскільки дозволяє об'єднувати різні рядки в один, щоб створити більш складні повідомлення або дані. Це корисно при створенні програм, які повинні виводити інформацію на екран, наприклад, у хвилі привітання чи повідомлень про помилки.

## Як

Є кілька способів з'єднати рядки в C++. Розглянемо декілька прикладів:

```C++
// конкатенація двох рядків
string s1 = "Привіт";
string s2 = " світ!";
string result = s1 + s2;
cout << result; // виведе "Привіт світ!"

// додавання числа до рядка
string num = "42";
string message = "Вітаю, ваше щастя: " + num;
cout << message; // виведе "Вітаю, ваше щастя: 42"
```

## Глибоке дослідження

У C++ є також функція ```std::to_string()```, яка дозволяє перетворити будь-який тип даних у строку. Це дуже зручно, коли потрібно вивести сполучені дані різних типів. Давайте розглянемо приклад:

```C++
// додавання числа до рядка
int num = 2021;
string message = "Вiтаю з Новим Роком " + std::to_string(num);
cout << message; // виведе "Вiтаю з Новим Роком 2021"
```
На практиці використання даних методів можуть бути різними, залежно від потреб програміста. Важливо пам'ятати, що при з'єднанні рядків, тип даних для результату буде строковим.
