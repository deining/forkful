## Чому:

 Об'єднання рядків - це важливий аспект програмування в мові C++, особливо якщо ви працюєте зі стрічками тексту. Використання цієї техніки допомагає зробити ваш код більш ефективним і зручним у використанні.

## Як:

Щоб об'єднати два рядки, вам потрібно використовувати оператор "+". Розглянемо приклад:

```C++
string hello = "Привіт";
string world = "світ";
string result = hello + ", " + world + "!";
cout << result;
```

В результаті виведеться "Привіт, світ!".

## Глибше розуміння:

Техніка об'єднання рядків застосовується для створення нових стрічок на основі існуючих. Це дозволяє здійснювати різноманітні маніпуляції з текстом, наприклад, включення змінних або констант у цільовий рядок. Також важливо зазначити, що при об'єднанні рядків, тип результату буде дорівнювати типу першого операнду.
