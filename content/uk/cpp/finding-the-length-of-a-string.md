Що і навіщо?

Отримання довжини рядка - це процес визначення кількості символів у даному рядку. Програмісти це роблять, щоби знати, скільки місця займає рядок у пам'яті, або для роботи з рядком у подальших обчисленнях.

Як?

```C++
#include <iostream>
using namespace std;

int main() {
  string myString = "Привіт світ!";
  int length = myString.length();
  cout << "Довжина рядка: " << length << endl;
  return 0;
}
```
**Вивід:** 
```
Довжина рядка: 12
```

## Глибше вдивимося

**Історичний контекст:** Розробники Сі-подібної мови програмування у 1960-х роках винайшли функцію для обчислення довжини рядка, щоби полегшити роботу з рядками.

**Альтернативи:** Є різні методи для отримання довжини рядка, такі як використання циклу для підрахунку символів або використання стандартних функцій у бібліотеці С.

**Деталі реалізації:** Вбудована функція .length() використовується для отримання довжини рядка в C++. Вона повертає значення типу int, яке відповідає кількості символів у рядку.
