## Як

Крок 1: Почнемо з підключення необхідних бібліотек за допомогою препроцесорної директиви `#include <iostream>`. Це дозволить нам використовувати функції вводу-виводу у консолі.

Крок 2: Створимо змінну `str` типу `string`, яка буде містити наш рядок. Після цього введемо дані у консоль за допомогою функції `getline(cin, str)`, яка дозволяє отримати введену користувачем строку та зберегти її у змінну `str`.

Крок 3: У рядку програмного коду напишемо функцію `findLength`, яка приймає змінну `str` та повертає довжину рядка за допомогою функції `length()`.

```C++
int findLength(string str){
    return str.length();
}
```

Крок 4: У функції `main` викличемо функцію `findLength` та передамо у неї змінну `str`. Результат буде збережено у змінну `length` і виведено на екран за допомогою функції `cout`.

```C++
int length = findLength(str);
cout << "Довжина рядка: " << length << endl;
```

При виконанні даного коду, у консоль буде виведено довжину введеного рядка. Наприклад, якщо користувач введе рядок "Привіт світ", то на екран виведеться "Довжина рядка: 10".

## Deep Dive

В мові програмування C++ довжина рядка визначається як кількість символів у рядку, враховуючи пробіли та спеціальні символи. Для отримання довжини рядка використовується функція `length`, яка належить класу `string`. Ця функція надається бібліотекою `<string>` та повертає значення типу `size_t` - це беззнаковий цілий тип даних, який використовується для збереження позицій у рядку. Також, зверніть увагу, що довжина рядка включає в себе термінальний символ `\0`, тому вона буде на один символ більшою за фактичну кількість символі
