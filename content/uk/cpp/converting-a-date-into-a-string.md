## Чому

 В даний час, коли комп'ютеризація стала необхідною частиною нашого життя, часто ми отримуємо дані в різних форматах і часто необхідно перевести ці дані у зручний для нас формат. Одним з таких завдань є перетворення дати в рядок, що може бути корисним для подальшої обробки та візуалізації даних.

## Як це зробити

Для того, щоб перетворити дату в рядок у мові програмування C++, ми можемо використати функцію `strftime`, яка дозволяє форматувати дату згідно з заданим шаблоном.

```C++
#include <iostream>
#include <ctime>
using namespace std;

int main() {
    // поточна дата та час (time_t)
    time_t now = time(0);

    // структура для зберігання дати та часу
    struct tm tstruct;
    char buffer[80];

    // форматування дати та часу
    // приклад: "день.місяць.рік година:хвилина:секунда"
    strftime(buffer, sizeof(buffer), "%d.%m.%Y %H:%M:%S", localtime(&now));

    // виведення отриманого рядка
    cout << buffer << endl;
    
    return 0;
}
```

При виконанні коду з виведенням ви отримаєте дату та час у вигляді рядка за заданим шаблоном: "день.місяць.рік година:хвилина:секунда". Наприклад, "05.04.2021 18:30:15".

## Глибока занурення

Функція `strftime` має багато варіантів форматування дати, які дозволяють отримати бажаний результат. Деякі з варіантів:

- `%d` - день місяця (01-31)
- `%m` - місяць (01-12)
- `%Y` - рік (позначення з 4 цифр)
- `%H` - година (00-23)
- `%M` - хвилина (00-59)
- `%S` - секунда (00-59)

Для усіх доступних варіантів форматування та інших корисних функцій дати у мові С++, рекомендується ознайомитися з документацією [по функції strftime](https://www.cplusplus.com/reference/ctime/strftime/).
