# Чому

У програмуванні, генерація випадкових чисел є корисною функцією, яка дозволяє забезпечити випадковість у різних алгоритмах та додатках. Ви можете використовувати випадкові числа для створення різноманітних елементів, таких як паролі, групи користувачів або вибір випадкового елементу зі списку. 

# Як

```C++
#include <iostream>
#include <cstdlib>
#include <ctime>

using namespace std;

int main() {
    // Встановлення загальних налаштувань для генерації випадкових чисел
    srand(time(0));

    // Генерування випадкового числа від 1 до 10
    int randomNumber = rand() % 10 + 1;

    // Виведення числа на екран
    cout << "Випадкове число: " << randomNumber << endl;

    return 0;
}
```

**Вихід:**
```
Випадкове число: 7
```

# Вдавайтеся глибше

Точність та ефективність генерації випадкових чисел залежить від використовуваного алгоритму та початкового зерна. У C++, для генерації випадкових чисел використовується функція `rand()`, яка генерує псевдовипадкові числа. Щоб забезпечити реальну випадковість, можна використовувати сторонні бібліотеки або власні алгоритми генерації.
