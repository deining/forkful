##Чому

Згенерувати випадкові числа допомагає програмі цікавіше та різноманітніше виконувати свої функції. Це особливо корисно для ігрових програм, але може бути також використано в багатьох інших випадках.

##Як це зробити

Для генерування випадкових чисел використовуються різні функції та бібліотеки в мові програмування C++. Нижче наведено приклади коду та вихідний результат для різних методів генерування випадкових чисел.

```C++
// Метод 1: Використання функції rand()
#include <iostream>
#include <cstdlib>

int main()
{
    // Встановлюємо наступне випадкове число від 1 до 100
    int random_num = rand() % 100 + 1;

    // Виводимо випадкове число
    std::cout << "Випадкове число: " << random_num << std::endl;

    return 0;
}
// Вихідний результат: Випадкове число: 79
```

```C++
// Метод 2: Використання бібліотеки <random>
#include <iostream>
#include <random>

int main()
{
    // Створюємо об'єкт генератора випадкових чисел
    std::random_device rd;
    std::mt19937 gen(rd());

    // Встановлюємо діапазон від 1 до 100
    std::uniform_int_distribution<> dist(1, 100);

    // Генеруємо випадкове число та виводимо його
    int random_num = dist(gen);
    std::cout << "Випадкове число: " << random_num << std::endl;

    return 0;
}
// Вихідний результат: Випадкове число: 46
```

##Глибше вдивлення

Функція `rand()` повертає випадкове число від 0 до `RAND_MAX`, яке визначається в бібліотеці `cstdlib`. Якщо потрібно генерувати більш великі числа, можна використовувати більш повну описану бібліотеку `<random>`, яка містить багато корисних функцій для генерування випадкових чисел.

Бажано не використовувати функцію `rand()` для критичних застосувань або безпосередньо у генеруванні паролів, оскільки результат його роботи може бути передбачуваним та не надійним. Для цих цілей краще використовувати спеціальні бібліотеки для генерування випадкових значень.
