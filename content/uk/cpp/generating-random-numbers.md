## Чому

Генерація випадкових чисел є важливою частиною багатьох програм та ігор. Вона дозволяє створювати різноманітні об’єкти, виконувати різношарову обробку та керувати поведінкою програми. 

## Як

```C++
#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {
  // встановлення насіння для генерації випадкових чисел
  srand(time(0));

  // генерація випадкового цілого числа в діапазоні від 1 до 100
  int random_number = rand() % 100 + 1;

  std::cout << "Випадкове число: " << random_number << std::endl;

  return 0;
}
```

**Вихідний код:**

```
Випадкове число: 79
```

```C++
#include <iostream>
#include <random>

int main() {
  // встановлення генератора випадкових чисел та діапазону від 1 до 100
  std::random_device rd;
  std::mt19937 gen(rd());
  std::uniform_int_distribution<int> distribution(1, 100);

  // генерація випадкового цілого числа
  int random_number = distribution(gen);

  std::cout << "Випадкове число: " << random_number << std::endl;

  return 0;
}
```

**Вихідний код:**

```
Випадкове число: 21
```

## Deep Dive

Генерація випадкових чисел в C++ базується на алгоритмах псевдовипадкових чисел. Це означає, що випадкові числа фактично не є повністю випадковими, а створюються за певними математичними правилами. У літературі існує багато різних алгоритмів для генерації випадкових чисел, і вони мають свої переваги та недоліки. Деякі із цих алгоритмів використовуються в стандартній бібліотеці C++, наприклад, `std::rand()`, тоді як інші можна підключити окремими бібліотеками, як наприклад `<random>`. Важливо ретельно вивчати алгоритми та обирати найкращий для конкретної ситуації.
