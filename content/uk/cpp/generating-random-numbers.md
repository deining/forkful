Для чого: Генерація випадкових чисел є важливим елементом багатьох програм вивчення, тестування та моделювання. Вона дозволяє створювати різноманітні сценарії та випадкові дані, що допомагає перевірити роботу програм та отримати більш реалістичні результати.

Як це зробити: Для генерації випадкових чисел використовуються функції зі стандартної бібліотеки C++ ```rand()``` та ```srand()```. Нижче наведений приклад коду, який генерує 10 цілих чисел в діапазоні від 1 до 100 та виводить їх на екран:

```C++
#include <iostream>
#include <cstdlib>
#include <ctime>

int main() {

  // встановлення початкового "насіння" для генерації випадкових чисел
  srand(time(0));

  // генеруємо та виводимо 10 випадкових чисел
  for(int i = 0; i < 10; i++) {
    int randomNum = rand() % 100 + 1;
    std::cout << randomNum << " ";
  }

  return 0;
}
```

В результаті ми отримаємо 10 випадкових чисел, наприклад: 78 2 43 53 17 91 35 81 66 28.

Важливо відзначити, що при кожному запуску програми, генератор випадкових чисел буде створювати нову послідовність чисел на основі початкового "насіння", тому вивід може змінюватися.

Глибоке занурення: Генерація випадкових чисел використовує псевдовипадковий алгоритм, який насправді не генерує повноцінні випадкові числа. Він виконує обрахунки на основі попереднього числа та початкового "насіння". Тому, якщо знати початкове "насіння", можна передбачити послідовність чисел, що буде згенерована. Існують спеціальні методи та алгоритми для підвищення надійності та запобігання "прогадуванню" послідовності чисел.
