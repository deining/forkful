## Чому
У програмуванні дуже важливо вміти правильно обробляти помилки, а одним з найпоширеніших методів є запис до стандартного потоку помилок. Такий підхід дозволяє виявляти та виправляти недоліки у програмному коді швидко та ефективно.

## Як це зробити
Щоб вивести повідомлення до стандартного потоку помилок, потрібно використати функцію `std::cerr`, яка приймає на вхід рядок або змінну типу `char` та виводить їх до спеціального потоку для помилок. Нижче наведено приклад коду та його виведення:

```C++
#include <iostream>

int main() {
  int number = 10;
  std::cerr << "Це виведеться до стандартного потоку помилок: " << number << "\n";
  return 0;
}
```

Виведення:
```
Це виведеться до стандартного потоку помилок: 10
```

## Глибше про запис до стандартного потоку помилок
Якщо у вашому програмі виникають помилки, то ви зможете їх легко виявити та виправити, якщо користуватиметесь записом до стандартного потоку помилок. Це стає особливо корисним при відладці коду та покроковому виконанні програми, адже ви можете бачити, який саме рядок спричинив помилку та яку значення мали змінні на той момент. Також, використання `std::cerr` дозволяє більш докладно виводити повідомлення про помилку, які допомагають зрозуміти, що саме потрібно змінити в коді.
