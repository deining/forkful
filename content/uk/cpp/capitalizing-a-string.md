## Що і чому?

Капіталізація рядків - це процес перетворення кожної першої літери в рядку на велику. Програмісти часто роблять це для того, щоб зробити рядок більш читаємим та зрозумілим для інших програмістів.

## Як?

```C++
// Приклад коду
std::string str = "це приклад рядка";
for (int i = 0; i < str.length(); i++) {
    str[i] = toupper(str[i]); // перетворення на великі літери
}
std::cout << str << std::endl;

// Вихід: ЦЕ ПРИКЛАД РЯДКА
```

## Поглиблене вивчення

**Історичний контекст:** Ідея капіталізації рядків виникла з друкувального машинопису, коли перша літера кожного рядка була автоматично великою. У мові C, ця функція була вперше реалізована в стандартній бібліотеці.

**Альтернативи:** Крім циклу for, що використовується в коді, існують інші методи для капіталізації рядка, такі як використання бібліотечних функцій або використання стандартного алгоритму transform().

**Деталі реалізації:** У багатьох мовах програмування, включаючи C++, існує функція to_upper, яка дозволяє перетворити символ на велику літеру. В цьому випадку, ми застосовуємо її до кожного символу в рядку за допомогою циклу.
