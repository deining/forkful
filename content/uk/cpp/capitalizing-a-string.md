## Чому

Привіт, друзі! Сьогодні ми будемо говорити про те, як використовувати функцію "capitalize" для капіталізації рядків у програмуванні на C++. Це корисний інструмент, який може бути використаний для полегшення роботи з рядками у вашому коді. 

## Як

Найпростіший спосіб використання функції "capitalize" в C++ - це створити новий рядок, що буде містити капіталізований варіант початкового рядка. Наприклад, якщо у нас є рядок "програмування", ми можемо застосувати функцію "capitalize" і отримати рядок "Програмування". 

```C++
#include <iostream>
#include <string>

int main() {
    std::string word = "програмування";
    std::string capitalized = capitalize(word);
    std::cout << capitalized; // виведе "Програмування"
}
```

У випадку, якщо у вас є більше одного слова у рядку і ви хочете капіталізувати кожне слово, ви можете використати цикл для проходження через рядок та застосувати функцію "capitalize" до кожного слова окремо. 

```C++
#include <iostream>
#include <string>

int main() {
    std::string sentence = "програмування на C++";
    for (int i = 0; i < sentence.length(); i++) {
        if (sentence[i] == ' ') {
            // перевіряємо чи поточний символ є пробілом
            sentence[i + 1] = toupper(sentence[i + 1]); // капіталізуємо наступний символ після пробілу
        }
    }
    std::cout << sentence; // виведе "Програмування На C++"
}
```

## Детальний огляд

Функція "capitalize" визначена у файлі заголовка <string> і вона приймає рядок як параметр та повертає новий рядок, що містить капіталізований варіант початкового рядка. Ця функція використовує стандартну бібліотеку С++, тому вам не потрібно додатково її підключати. Вона також ігнорує числа та символи, тому їх значення залишаються незмінними у капіталізованому рядку. 

Іншим варіантом капіталізації рядка є використання функцій "toupper" і "tolower", що приймають символ як параметр і повертають капіталізований або некапіталізований варіант цього символу. Вони також включені у бібліотеку <string>.
