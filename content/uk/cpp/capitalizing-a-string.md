## Як

З мовою програмування C++ ви можете реалізувати функцію capitalize за допомогою інструкції if та функцій рядків. Ця функція отримує рядок як вхідний параметр, перевіряє першу букву та, якщо вона мала, перетворює її у велику.

```C++
#include <iostream>
#include <string>

using namespace std;

//функція capitalize
string capitalize(string input) {
    //перевірка першої букви
    if (islower(input[0])) {
        //перетворення першої букви великою
        input[0] = toupper(input[0]);
    }
    return input;
}

int main() {
    //створення рядка для перетворення
    string word = "programming";
    //виклик функції
    string capitalizedWord = capitalize(word);
    //вивід результату
    cout << capitalizedWord << endl;
    return 0;
}

//ВИВОД: Programming
```

## Глибший аналіз

Функція capitalize є простою та корисною в будь-якій ситуації, коли потрібно потрібно змінити першу літеру рядка на велику. Однак, вона не враховує можливих випадків, коли перша літера є великою або є інші символи, крім букв. Тому, будьте обережні та ретельно перевіряйте вхідні дані під час застосування цієї функції.
