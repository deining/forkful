## Чому

Доброго дня, шановні читачі! Сьогодні ми розглянемо основи програмування мовою C++ та навчимося створювати програми для обробки тексту. Одною з найпоширеніших задач у роботі зі строки є її капіталізація. Чому це потрібно? Ви могли б замінювати маленькі літери на великі вручну, але це забирає багато часу і сил, особливо якщо у вас великий обсяг тексту. Тому використання програми, яка зробить це за вас, є найефективнішим варіантом.

## Як

Для початку, давайте створимо просту програму, яка буде капіталізувати будь-яку строку, введену користувачем. Ми використаємо функцію `toupper()` для перетворення маленьких літер рядка на великі. Перш за все, нам потрібно підключити бібліотеку `<string>` та використати директиву `using namespace std;` для спрощення доступу до функцій.

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    string input;
    
    cout << "Введіть строку: ";
    getline(cin, input); // зчитуємо введену строку та зберігаємо в змінну input
    
    for (int i = 0; i < input.length(); i++) {
        input[i] = toupper(input[i]); // застосовуємо toupper() до кожної літери у строкі
    }
    
    cout << "Капіталізована строка: " << input << endl;
    
    return 0;
}
```

Вхідні дані: `hello world`
Вихідні дані: `HELLO WORLD`

Можливо, ви помітили, що ми викликаємо функцію `toupper()` в циклі `for`. Це дуже важливо, оскільки функція повертає результат для однієї літери, але ми хочемо застосувати її до всієї строки, тому нам потрібно викликати її для кожної літери окремо.
