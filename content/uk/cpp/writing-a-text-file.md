## Чому

Написання текстового файлу є важливою частиною програмування в C++, оскільки це дозволяє зберігати дані у структурованому форматі для подальшого використання. Також це є потужним інструментом для обміну даними між різними програмами та системами. 

## Як це зробити

Для написання текстового файлу в C++, спочатку потрібно оголосити змінну для даних, які ви хочете записати у файл. Наступним кроком буде відкриття файлу за допомогою функції `ofstream`, яка дозволяє використовувати файл як потік даних. Потім використовуйте функцію `<<` для запису даних у файл за допомогою потоку. Наприклад, `myfile << data;` додасть значення змінної `data` у файл `myfile`. Коли ви закінчите записувати дані, не забудьте закрити файл за допомогою функції `close()`, щоб зберегти внесені зміни.

```C++
#include <iostream>
#include <fstream>

using namespace std;

int main() {
	// оголошення змінної для даних
	string name = "John";
	int age = 25;

	// відкриття файлу
	ofstream myfile("data.txt");

	// запис даних у файл
	myfile << "Name: " << name << endl;
	myfile << "Age: " << age << endl;

	// закриття файлу
	myfile.close();

	return 0;
}
```

Ви можете перевірити, чи дані були успішно записані, відкривши файл у текстовому редакторі. Ви повинні побачити наступне:

```
Name: John
Age: 25
```

## Глибше занурення

У C++, для запису даних у текстовий файл, ви можете використовувати не тільки функцію `<<`, але й функцію `ofstream::put()` для додавання окремих символів у файл. Також ви можете використовувати функцію `ofstream::write()`, яка дозволяє записувати більше одного символу за раз. Для унікального відмінення від функції `<<`, ви можете використовувати функцію `ofstream::write()` для запису бінарних даних, наприклад, чисел або рядків байтів.
