## Чому
Іноді програмістам потрібно зберегти певну інформацію у структурованому та легко доступному форматі. Це може бути список заміток, таблиця даних або ціле розрахункове завдання. Таким чином, найефективнішим інструментом для цього є створення текстового файлу.

## Як це зробити
Нижче наведено приклад коду на мові C++, який демонструє створення текстового файлу та запис даних в нього:

```C++
#include <iostream>
#include <fstream>
using namespace std;

int main() {
  // відкриття файлу для запису
  ofstream file("example.txt");

  // перевірка чи файл було успішно створено
  if (file){
  	// запис рядків у файл
  	file << "Це приклад того, як записати дані у текстовий файл." << endl;
  	file << "Один рядок." << endl;
  	file << "Інший рядок." << endl;
  	// закриття файлу
  	file.close();
  	cout << "Дані успішно записано у файл." << endl;
  } else {
  	// вивід повідомлення про помилку
  	cout << "Не вдалося створити файл." << endl;
  }
  return 0;
}
```

Після виконання цього коду, у вашій директорії повинен з'явитися файл під назвою "example.txt", який містить наші рядки.

## Глибинне дослідження
У створеному коді використано два основних класи - `ofstream` та `ifstream`, які дозволяють виконувати операції запису та читання з файлу відповідно. Для відкриття файлу використовується конструктор, який приймає назву файлу як аргумент. Далі, через оператор `<<` виконується запис даних у файл. Якщо в результаті цієї операції виникла помилка, то в інструкція `if` виводиться повідомлення про це. Після того, як ми використали файл, його потрібно закрити, щоб не залишити його відкритим.
