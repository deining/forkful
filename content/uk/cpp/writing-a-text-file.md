## Що це таке та чому програмісти це роблять?
Політіради, стукіт клавіш та повноцінні програми. Всі ці елементи лежать в основі комп'ютерної програми. Однак, іноді програмісти хочуть сформувати інформацію в інший спосіб - у вигляді текстового файлу. Текстовий файл - це простий спосіб зберігання інформації у вигляді тексту, що може бути прочитаним іншими програмами. Програмісти часто використовують текстові файли для зберігання налаштувань, баз даних або даних, які потрібні для інших програм.

## Як це зробити?
Для створення текстового файлу використовуються такі бібліотеки як ```<fstream>``` та ```<iostream>```. Давайте подивимось, як можна створити та записати інформацію до файлу:

```
#include <iostream>
#include <fstream>
using namespace std;

int main() {
   ofstream file;
   file.open ("myfile.txt");      // відкриття файлу для запису
   file << "Це зразок тексту, який буде записаний у файл";   // запис тексту до файлу
   file.close();        // закриття файлу
   return 0;
}

```

На виході ми отримаємо файл із назвою "myfile.txt", в якому буде записаний наш рядок. Також, за допомогою бібліотеки ```<fstream>``` ми можемо виконувати інші операції з файлом, такі як читання та оновлення існуючої інформації.

## Глибше копання
Створення текстового файлу було можливо з появою комп'ютерів. Люди користувалися цим зручним способом зберігання інформації для різних потреб. Сьогодні, існує багато форматів для зберігання даних, такі як XML, JSON та бази даних. Проте текстові файли є одним з найпростіших і швидших способів зберігання даних.

Існують і альтернативи для створення текстового файлу за допомогою бібліотеки ```<fstream>```. Наприклад, можна використовувати бінарний файл, який дозволяє зберігати більше типів даних, але його читання та запис може бути складніше.

Для створення більших та складніших текстових файлів можна використовувати інші бібліотеки, наприклад, ```<boost/filesystem>```. Ця бібліотека дозволяє працювати з файловою системою, в тому числі створювати та редагувати тексти та каталоги.
