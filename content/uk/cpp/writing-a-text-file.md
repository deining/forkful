## Чому

Написання текстового файлу - важлива складова частина розробки програмного забезпечення. Воно дозволяє зберігати та зчитувати дані для подальшого використання, а також забезпечує можливість легкого обміну даними між різними програмами. Також це може бути необхідно для створення звітів або збереження налаштувань користувача.

## Як це зробити

Для написання текстового файлу використовується стандартна бібліотека мови програмування C++. Нижче наведено приклад коду, який демонструє створення та запис даних до текстового файлу.

```C++
#include <iostream>
#include <fstream> // підключення необхідної бібліотеки

using namespace std;

int main()
{
    // створення об'єкта файлу та відкриття його для запису
    ofstream file("text.txt");

    // перевірка, чи файл вдало відкрито
    if (!file.is_open())
    {
        cout << "Помилка відкриття файлу!" << endl;
        return 1;
    }

    // запис даних до файлу
    file << "Це текстовий файл, створений за допомогою C++!" << endl;

    // закриття файлу
    file.close();

    return 0;
}
```

Після виконання цього коду, у папці з проєктом з'явиться новий файл з назвою "text.txt", в якому буде збережений наданий текст.

## Глибока занурення

Крім звичайного запису даних до текстового файлу, існують також різні методи для його написання. Наприклад, можна використовувати функцію `getline()` для зчитування даних з консолі та запису до файлу, або використовувати цикл для додавання даних до файлу. Також можна налаштувати різні режими відкриття файлу, наприклад, для запису у кінець файлу або для додавання даних до існуючого файлу.

Варто також звернути увагу на обробку помилок та винятків при роботі з файлами. Наприклад, необхідно перевірити, чи вдалося відкрити файл для запису, чи не переповнюємо зберігану інформацію або чи маємо необхідні права на редагування файлу.
