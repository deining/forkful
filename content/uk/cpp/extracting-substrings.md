## Чому: Причини витягувати підрядки

Програмування - це красиве мистецтво, що надає можливості розвивати свою творчість та розв'язувати цікаві задачі. Однією з таких задач є витягування підрядків з рядків у програмуванні. Це корисний навичка, яка допомагає працювати з текстом та знаходити потрібну інформацію.

## Як витягувати підрядки: Приклади коду та вихідні дані

Для початку, нам знадобиться зазначити, що таке підрядок. Підрядок - це частина рядка, яку ми витягуємо за допомогою певних параметрів. Наприклад, якщо ми маємо рядок "моя улюблена книга - Книга джунглів", то підрядок "Книга джунглів" буде витягуватись за допомогою параметрів "14" та "18".

Ось приклад коду на мові C++, який допоможе нам витягнути підрядок:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    string text = "моя улюблена книга - Книга джунглів";
    string substring = text.substr(14, 18);
    
    cout << substring << endl;
    return 0;
}
```

Вихідні дані будуть наступними:

```
Книга джунглів
```

Також, можна витягати підрядки за допомогою знаку "=" та необмеженою кількістю обмежувачів. Наприклад:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    string text = "Ласкаво просимо до Лас-Вегаса!";
    string substring = text.substr(10, text.find("=") - 10);
    
    cout << substring << endl;
    return 0;
}
```

Вихідні дані будуть такі:

```
Лас-Вегас
```

## Глибоке поглиблене дослідження: Витягування підрядків

Витягування підрядків - це більш складна тема, ніж здається на перший погляд. Важливо розуміти, які параметри використовувати для витягування підрядка та при яких обставинах вони дають нам потрібний результат. Також, можливо виникнення проблем із погано структурованим текстом або невірними параметрами, тому важливо бути уважними при використанні цієї функції.
