## Чому

Витягування підрядків є важливим інструментом у програмуванні, який дозволяє здійснювати операції з даними, що зберігаються у вигляді рядків (strings). Це може бути корисно, наприклад, для роботи з текстовими данними в базах даних або файлових системах.

## Як це зробити

Щоб витягти підрядок із рядка, необхідно використати метод substr(). Синтаксис цього методу виглядає наступним чином:

```C++
string substr (size_t pos = 0, size_t len = npos) const;
```

Де pos - це індекс елемента, з якого починається витягування підрядка, а len - кількість символів, які потрібно витягнути. Якщо len не вказано, то буде витягнуто все до кінця рядка. Наприклад:

```C++
string text = "Привіт усім!";
string subtext = text.substr(0, 6); // subtext = "Привіт"
```

## Глибокий занурення

Цей метод є досить простим, проте дуже корисним для обробки рядків. Його також можна використовувати для витягування підрядків із підрядків, а також для зміни вже існуючих рядків. Також варто відзначити, що індекси у рядках починаються з 0, а не з 1. Для більш детального ознайомлення з методом substr() можна звернутися до [офіційної документації](https://en.cppreference.com/w/cpp/string/basic_string/substr) з майже всіма можливими використаннями.
