## Чому

Якщо ви займаєтеся програмуванням на C++, це цікава навичка, яка дає можливість працювати з текстовими даними. Витягування підрядків - це одна з найбільш корисних вмінь у роботі з рядками в C++. Це дає змогу отримати конкретну частину даних зі строки, що дуже зручно у багатьох випадках.

## Як це зробити

Для витягування підрядків існує кілька способів, але ми розглянемо основний. У прикладі нижче ми будемо витягувати підрядок із рядка "Hello World" і виведемо результат у консоль.

```C++
#include <iostream>
#include <string>

int main() {
    std::string str = "Hello World";
    std::string substring = str.substr(6); // витягуємо підрядок починаючи з індексу 6
    std::cout << substring; // виводимо "World"
    
    return 0;
}
```

При запуску цього коду, у консолі ми побачимо вивід "World". Якщо потрібно витягувати підрядки по більш складному шаблону або з определеними умовами, можна використати інші функції бібліотеки `string`, такі як `find` та `substr`.

## Глибока занурення

Правила для витягування підрядків можуть здатися незрозумілими на перший погляд, але насправді вони досить прості. Функція `substr` приймає два аргументи: початковий індекс та довжину підрядка. Також важливо зазначити, що індекси в C++ починаються з 0, тому перший символ має індекс 0, другий - 1, і так далі.

Наприклад, якщо ми маємо рядок "Programming is fun" і хочемо витягнути підрядок "is fun", ми можемо вказати індекс початку як 11 (індекс "i" в слові "is"), а довжину 6 (кількість символів у "is fun"). Це буде виглядати так: `str.substr(11, 6)`. 

Також варто зазначити, що функція `substr` також працює зі строки, які містять кирилицю (українські літери).
