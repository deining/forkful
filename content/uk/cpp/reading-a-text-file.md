# Чому

Читання текстового файлу може бути корисним для отримання інформації збереженої в файловій системі. Наприклад, ви можете прочитати список студентів з вхідного файлу, щоб автоматично додати їх до бази даних.

# Як використовувати

Прочитання текстового файлу може бути зроблено за допомогою класу ```std::ifstream``` у C++. Ось приклад коду, який відкриває файл для читання та використовує цикл для виводу кожного рядка до екрану:

```C++
#include <iostream>
#include <fstream>

using namespace std;

int main(){
  ifstream inputFile;
  string line;
  inputFile.open("students.txt");

  if (!inputFile){
    cout << "Помилка відкриття файлу!" << endl;
    return 1;
  }

  while (getline(inputFile, line)){
    cout << line << endl;
  }

  inputFile.close();
  return 0;
}
```

Цей код відкриє файл "students.txt" та виведе кожен рядок до екрану, використовуючи функцію ```getline()```, яка повертає рядок до змінної ```line```. Потім ми закриваємо файл, щоб уникнути помилок при подальшій роботі з ним.

Вихідний файл з вхідними даними може виглядати так:

```
John Smith
Jane Doe
Mark Johnson
```

Тоді вивід програми буде виглядати так:

```
John Smith
Jane Doe
Mark Johnson
```

# Детальніше

При читанні текстового файлу, можна здійснити додаткову перевірку на наявність даних та обробку помилок. Також, необхідно враховувати форматування вхідних даних та використовувати різні функції для взаємодії з файлом, наприклад, ```seekg()``` для переміщення покажчика читання до деякого місця у файлі. Для більш детального ознайомлення з роботою з файлами у C++, можна прочитати документацію [тут](https://www.cplusplus.com/doc/tutorial/files/).
