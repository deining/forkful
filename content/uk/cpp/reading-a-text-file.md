## Чому
Перегляд текстового файлу є важливою навичкою для програмістів C++, оскільки це дозволяє їм зчитувати та обробляти дані з файлів, що містять велику кількість інформації.

## Як
```C++
#include <fstream>
#include <iostream>

int main(){
	// відкриваємо файл
	std::ifstream file("example.txt");
	
	// перевіряємо, чи файл є відкритим
	if (!file.is_open()) {
		std::cout << "Не вдалося відкрити файл!" << std::endl;
		return 0;
	}
	
	// створюємо змінну для збереження рядка з файлу
	std::string line;
	
	// зчитуємо рядок з файлу та виводимо його на екран
	while (getline(file, line)) {
		std::cout << line << std::endl;
	}
	
	// закриваємо файл
	file.close();
	
	return 0;
}
```
### Виходячи з прикладу:
Вміст файлу "example.txt":
```text
Привіт!
Це приклад зчитування текстового файлу.
Цей рядок виведеться на екран.
```
Вивід на екран:
```text
Привіт!
Це приклад зчитування текстового файлу.
Цей рядок виведеться на екран.
```

## Глибшого занурення
Існує кілька важливих функцій, які необхідно знати при зчитуванні текстових файлів на мові C++:
- `ifstream` - клас, що дозволяє зчитувати дані з файлу
- `getline()` - функція для зчитування одного рядка з файлу
- `open()` - метод для відкриття файлу
- `close()` - метод для закриття файлу
- `is_open()` - метод для перевірки, чи файл є відкритим

Ці функції дозволяють програмісту ефективно зчитувати та обробляти дані з текстового файлу у своїй програмі.
