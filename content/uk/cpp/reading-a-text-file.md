Що мотивує на читання текстового файлу
============================================

Читання текстових файлів є основою багатьох програм в C++. Вони дозволяють зчитувати та обробляти великі обсяги даних, які можуть бути збережені у файловій системі. Це також дає можливість створювати програми, які можуть працювати з введеними користувачем даними, зчитуючи їх зі збережених файлів.

Як це зробити
==========================

```C++
#include <iostream>
#include <fstream> // необхідна бібліотека для роботи з файлами

using namespace std;

int main()
{
    ifstream inputFile; // об'єкт для читання файлу
    string fileName = "example.txt"; // назва файлу для читання

    inputFile.open(fileName); // відкриття файлу

    if (!inputFile) // перевірка на успішне відкриття файлу
    {
        cout << "Помилка при відкритті файлу " << fileName << endl;
        return 1;
    }

    string line; // змінна для збереження зчитаного рядка

    // читання файлу до його кінця
    while (getline(inputFile, line)) {
        cout << line << endl; // виведення прочитаного рядка у консоль
    }

    inputFile.close(); // закриття файлу
    return 0;
}
```

Вивід програми:
```
Це приклад текстового файла.
Тут можуть бути будь-які дані.
Даний текст буде виведений у консоль.
```

Зробити детальний аналіз
=============================

У програмуванні часто потрібно прочитати великий обсяг даних з файлу. Для цього можна використовувати цикл `while` та функцію `getline()` з бібліотеки `<fstream>`. Функція `getline()` зчитує рядок, включаючи пробіли та перехід на новий рядок. У цьому прикладі ми використовуємо `while` для того, щоб читати файл до його кінця. У кожній ітерації циклу ми зчитуємо новий рядок і виводимо його у консоль.

Пам'ятайте, що перед початком роботи з файлом його необхідно відкрити, а після завершення - закрити.
