## Чому

Створення тимчасових файлів є важливою частиною програмування для збереження тимчасової інформації або для виконання певних операцій над даними.

## Як

Тимчасовий файл можна створити за допомогою функції ```C++ std::tmpfile() ``` з бібліотеки ```C++ <cstdio>```. Нижче наведений приклад коду, який створює тимчасовий файл та записує в нього деякий текст.

```C++
#include <cstdio>
#include <iostream>

int main() {
    // Створення тимчасового файлу
    std::FILE *temp_file = std::tmpfile();
    
    // Перевірка, чи файл був успішно створений
    if (temp_file == NULL) {
        std::cout << "Помилка при створенні тимчасового файлу!" << std::endl;
        return 1;
    }
    
    // Запис деякого тексту в файл
    std::fputs("Це тимчасовий файл, який був створений за допомогою C++!", temp_file);
    
    // Закриття файлу
    std::fclose(temp_file);
    
    return 0;
}
```

Вище наведений код створить тимчасовий файл та запише в нього текст "Це тимчасовий файл, який був створений за допомогою C++!". Ви можете перевірити це шляхом перегляду вмісту файлу за допомогою програми для перегляду текстових файлів.

## Огляд

Під час створення тимчасових файлів важливо враховувати, що вони будуть автоматично видалені при завершенні програми. Тому, якщо ви хочете зберегти дані на стійкому носії, то використовуйте звичайні файли. Тимчасові файли мають свої переваги, але також варто враховувати можливі проблеми з їх використанням у певних випадках.
