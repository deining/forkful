# Чому

Цю статтю присвячено тим, хто хоче вивчити як створювати тимчасові файли у своїх програмах на мові C++. Тимчасові файли можуть бути корисними для тимчасового зберігання даних або для роботи з файловою системою.

# Як

```C++
#include <iostream>
#include <fstream> // необхідна для роботи з файлами

int main() {
    // Використовуємо бібліотеку <fstream> для створення тимчасового файла
    std::ofstream temp_file("temp.txt");
    
    // Перевіряємо чи файл вдалося створити
    if (!temp_file) {
        std::cout << "Помилка при створенні тимчасового файла!" << std::endl;
        return 1;
    }
    
    // Записуємо дані у файл
    temp_file << "Ця інформація буде збережена у тимчасовому файлі.";
    
    // Закриваємо файл
    temp_file.close();
    
    // Відображаємо повідомлення про успішне створення файлу
    std::cout << "Успішно створено тимчасовий файл 'temp.txt'!" << std::endl;
    
    return 0;
}
```

Після виконання цього коду, у нашій програмі буде створено тимчасовий файл з назвою "temp.txt", який буде містити нашу інформацію.

# Дослідження

Створення тимчасових файлів може бути корисним при написанні програм, які потребують тимчасового збереження даних, наприклад, під час обробки великих обсягів інформації. Також, цей метод може бути використаний для створення тимчасового імені для новоствореного файлу, їхнього збереження та подальшого використання у програмі.
