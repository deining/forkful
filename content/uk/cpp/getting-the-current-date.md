## Чому

Програмування може бути цікавою та захоплюючою грою, але його основною метою є допомога людям вирішувати певні завдання. Одним з таких завдань є отримання поточної дати. Незалежно від того, чи ви працюєте над веб-додатком, десктоп-програмою або мобільним додатком, у багатьох випадках вам потрібно буде використовувати поточну дату, наприклад, для виведення на екран або для збереження у базу даних. Тому вміння отримати поточну дату є корисним для будь-якого програміста.

## Як це зробити

Для отримання поточної дати у C++ ми можемо використовувати стандартну бібліотеку `ctime` та функцію `time`. Вона повертає кількість секунд, що пройшли з початку 1970 року до теперішнього часу. Щоб отримати поточну дату, нам необхідно перетворити цю кількість секунд у структуру `tm`, яка містить різні поля, такі як день, місяць, рік тощо. Нижче наведено приклад коду та його виведення:

```C++
#include <iostream>
#include <ctime>

int main() {
    // Отримання кількості секунд
    time_t now = time(0);
    
    // Перетворення у структуру tm
    tm *ltm = localtime(&now); 
    
    // Виведення поточної дати
    std::cout << "Теперішня дата: ";
    std::cout << ltm->tm_mday << "." << ltm->tm_mon + 1 << "." << ltm->tm_year + 1900;
    return 0;
}
```

Виведення: Теперішня дата: 5.10.2021

## Глибоке погруження

Як уже було згадано вище, функція `time` повертає кількість секунд з 1970 року. Цей момент відомий як "Епоха Unix". Функція `localtime` перемінює цю кількість секунд у локальний час на вашому комп'ютері. Час може бути представлений у різних стандартах, таких як UTC (координований всесвітній час) або його локальна віріанта. Якщо ви хочете отримати час у різних часових зонах, вам необхідно буде додатково працювати з часовим полем в структурі `tm`.
