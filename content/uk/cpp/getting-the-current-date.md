## Чому

Якщо ви працюєте з даними, іноді вам може знадобитися вивести поточну дату у своїх програмах. Це може бути корисно для створення звітів, логування подій або для будь-якої іншої задачі, яка потребує знання поточної дати.

## Як це зробити

Використовуючи мову програмування C++, є кілька способів отримати поточну дату. Один з найпростіших - це використання функції `time()` з бібліотеки `<ctime>`, яка повертає кількість секунд, які пройшли з 1 січня 1970 року. Для відображення цієї кількості секунд у зрозумілому форматі дати, ми можемо використовувати функцію `localtime()`, яка повертає структуру `tm`, що містить детальну інформацію про поточну дату та час. Подивімся на код та його виведення нижче:

```C++
#include <iostream>
#include <ctime>

int main()
{
    // Отримуємо поточну дату та час
    time_t now = time(0);
    tm *ptm = localtime(&now);

    // Виводимо поточну дату у форматі "ДД-ММ-РРРР"
    std::cout << "Поточна дата: " << ptm->tm_mday << "-" << ptm->tm_mon + 1 << "-" << ptm->tm_year + 1900 << std::endl;
    
    // Виводимо поточний час у форматі "ГГ:ХВ:СС"
    std::cout << "Поточний час: " << ptm->tm_hour << ":" << ptm->tm_min << ":" << ptm->tm_sec << std::endl;

    return 0;
}
```

Виведення:

```
Поточна дата: 15-02-2021
Поточний час: 12:30:45
```

## Поглиблене дослідження

Якщо ви хочете отримати більше інформації про поточну дату та час, вам можуть бути корисні різні функції та структури, які надає бібліотека `<ctime>`. Наприклад, функція `asctime()` повертає рядок, що містить повну назву дня тижня, місяця та року. А структура `tm` містить багато полів, таких як `tm_wday` (день тижня), `tm_yday` (день року) та багато інших. Ви можете експериментувати з цими функціями та структурами, щоб отримати потрібну вам інформацію про поточну дату і час.
