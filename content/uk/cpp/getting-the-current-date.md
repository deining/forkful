## Що і чому?

Отримання поточної дати - це процес, коли програма отримує поточну дату та час у форматі, який можна використовувати для виконання певних дій. Програмісти зазвичай роблять це для відстеження часу виконання програми або для виведення поточної дати для користувача.

## Як це зробити:

```C++
#include <iostream>
#include <ctime>

int main() {
    // Отримання поточної дати та часу
    time_t current_time = time(0);

    // Конвертація поточного часу у локальну дату та час
    char* time_string = ctime(&current_time);

    // Виведення поточної дати та часу
    std::cout << "Поточна дата та час: " << time_string << std::endl;

    return 0;
}
```

Виведення:

```
Поточна дата та час: Mon Oct 18 09:30:00 2021
```

## Глибоке занурення:

Отримання поточної дати та часу є важливим завданням для багатьох програм. У минулому використовувалися різні бібліотеки для отримання поточного часу, але з появою бібліотеки <ctime> у стандарті C++ цей процес став набагато простішим. Існують також альтернативні способи отримання поточної дати, наприклад, використання сторонніх бібліотек або використання вбудованих функцій операційних систем. Однак, використання бібліотеки <ctime> є простим та широко підтримуваним.
