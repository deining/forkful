## Чому

У програмуванні часто виникає потреба отримати поточну дату. Це може бути необхідно для збереження полету замовлення на сайті, створення резервних копій або для інших операцій, пов'язаних з часом. В цій статті ми розглянемо, як отримати поточну дату у програмі на мові C++.

## Як

```C++
#include <iostream>
#include <ctime>
using namespace std;

int main() {
    // Отримання поточного часу
    time_t now = time(0);
    // Конвертація часу до локального часового поясу
    tm *ltm = localtime(&now);
    // Виведення поточної дати
    cout << "Поточна дата: ";
    cout << ltm->tm_mday << "-";
    cout << 1 + ltm->tm_mon << "-";
    cout << 1900 + ltm->tm_year << endl;
    
    return 0;
}
```

**Вихід:**

```
Поточна дата: 28-04-2021
```

У цьому прикладі ми використовуємо стандартний заголовок `<ctime>` для роботи з часом. Ми створюємо змінну `now` типу `time_t`, яка містить поточний час. Потім ми використовуємо функцію `localtime()`, щоб перевести цей час улокальний часовий пояс та зберегти його у змінну `ltm`. Нарешті, ми виводимо поточну дату, використовуючи різні поля структури `tm`, які зберігають день, місяць та рік.

## Глибокий занурення

Отримання поточної дати не є простою задачею, оскільки вона залежить від налаштувань системи та доступних функцій програмування. У деяких випадках може бути корисно встановити поточну дату у ручному режимі, а не отримувати її з системи.

Крім того, при роботі з датами необхідно уважно відноситися до локалізації, оскільки формат дати може відрізнятися залежно від мови та країни.

Наприклад, у Закарпатті дати можуть представлятися у форматі "день-місяць-рік", тоді як у Канаді - "місяць/день/рік". Також слід звернути увагу на якість та точність отриманої дати, оскільки у разі неправильної налаштування можуть виникнути проблеми з обробкою і збереженням даних.
