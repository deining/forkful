## Чому: Лише 1-2 речення щодо *чому* хтось би хотів отримати поточну дату.
Хтось може хотіти отримати поточну дату для різних цілей, наприклад для створення журналу або для користування у своїх програмах, де поточна дата є важливою інформацією.

## Як: Приклади коду та вихідні дані в блоці коду " ```C++ ... ``` "
Використання бібліотеки <ctime> дозволяє нам легко отримувати поточну дату у своїх програмах на мові C++. Ось декілька прикладів коду, як вивести поточну дату на екран:

```C++
// Приклад 1:
#include <iostream>
#include <ctime>

int main()
{
  time_t rawtime;
  struct tm * timeinfo;
  char buf[80];

  time (&rawtime);
  timeinfo = localtime(&rawtime);

  strftime(buf, 80, "Поточна дата: %d-%m-%Y", timeinfo);
  std::cout << buf;
  return 0;
}

// Вихідні дані:
// Поточна дата: 07-10-2021

// Приклад 2:
#include <iostream>
#include <ctime>

int main()
{
  time_t rawtime;
  struct tm * timeinfo;
  char buf[80];

  time (&rawtime);
  timeinfo = localtime(&rawtime);

  strftime(buf, 80, "Поточний час: %H:%M:%S", timeinfo);
  std::cout << buf;
  return 0;
}

// Вихідні дані:
// Поточний час: 08:00:00
```

## Глибоке погруження: Детальна інформація про отримання поточної дати
Отримання поточної дати у програмах на мові C++ є дуже простим завданням. Бібліотека <ctime> надає нам доступ до функцій для роботи з часом і датою. Функція time() повертає кількість секунд, що пройшли з 1 січня 1970 року. Для перетворення цього значення у зрозумілу форму використовується функція localtime(), яка повертає структуру з даними про час і дату. Ми можемо використовувати функцію strftime() для форматування даних у зручний для нас вигляд.
