## Чому

Перевірка наявності директорії є важливим етапом у програмуванні на мові C++. Це дозволяє переконатися, що необхідні файли і папки наявні перед початком виконання програми.

## Як

У мові програмування C++, перевірка наявності директорії виконується за допомогою функції `std::filesystem::exists()`. Давайте розглянемо приклад нижче:

```C++
#include <iostream>
#include <filesystem>

using namespace std;
namespace fs = std::filesystem;

int main() {
    // створюємо шлях до директорії
    fs::path dir_path = "приклад/директорії";
    
    // перевіряємо, чи існує директорія
    if (fs::exists(dir_path)) {
        cout << "Директорія існує!" << endl;
    }
    else {
        cout << "Директорія не існує." << endl;
    }
    
    return 0;
}
```

Приклад виводу:

```
Директорія існує!
```

У цьому коді ми використовуємо простір імен `std::filesystem` для доступу до функції перевірки наявності директорії `exists()`. Змінна `dir_path` містить шлях до перевіряємої директорії. Ми використовуємо конструкцію `if` для перевірки, чи повертає функція `exists()` значення `true` (яке буде означати, що директорія існує) чи `false` (яке буде означати, що директорія не існує).

## Глибинна аналітика

Перевірка наявності директорії зазвичай виконується перед виконанням інших операцій з файлами або директоріями, такими як створення, переміщення чи видалення. Це дозволяє уникнути помилок виконання програми, що може призвести до збою або втрати даних.

Крім того, важливо враховувати, що функція `exists()` перевіряє тільки наявність директорії і не перевіряє, чи є вона доступною для читання або запису. Для цього потрібно використовувати інші функції, наприклад `std::filesystem::is_readable()` чи `std::filesystem::is_writable()`.
