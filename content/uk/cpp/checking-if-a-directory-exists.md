## Чому
Перевірка існування директорії є важливою частиною програмування, оскільки це дозволяє перевірити наявність необхідних даних перед їх використанням. Це також допомагає уникнути помилок і забезпечує більш надійну роботу програми.

## Як виконати перевірку існування директорії
Нижче наведені приклади коду, які показують, як перевірити наявність директорії в C++. Вони демонструють використання різних методів та їх виведення у консоль.

```C++
#include <iostream>
#include <filesystem>

using namespace std;
namespace fs = std::filesystem;

int main() {
    // Перевірка існування директорії за допомогою exists()
    if (fs::exists("my_directory")) {
        cout << "Директорія існує!";
    } else {
        cout << "Директорія не існує.";
    }

    // Перевірка існування директорії за допомогою status()
    fs::file_status status = fs::status("my_directory");
    if (fs::is_directory(status)) {
        cout << "Директорія існує!";
    } else {
        cout << "Директорія не існує.";
    }

    // Перевірка існування директорії за допомогою is_directory()
    if (fs::is_directory("my_directory")) {
        cout << "Директорія існує!";
    } else {
        cout << "Директорія не існує.";
    }

    return 0;
}
```

Приклад виведення у консоль:
```
Директорія існує!
Директорія існує!
Директорія існує!
```

## Детальна інформація
Перевірка існування директорії полягає в тому, щоб перевірити наявність файлу, вказаного шляхом, за допомогою різних методів. exists() функція перевіряє, чи існує файл або директорія, і повертає true, якщо вона існує, та false - якщо ні. Також можна використовувати is_directory() для перевірки того, що шлях вказує на директорію, а не на файл. Ці методи дозволяють програмі більш зручно реагувати на відсутність потрібних даних або застосовувати необхідні дії залежно від результату перевірки.
