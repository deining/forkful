## Чому

Ви, напевно, часто чули про перевірку наявності каталогу в програмуванні. Але чому це так важливо? Чому нам потрібно перевіряти наявність каталогів в наших програмах? Ця стаття розкриє це питання та покаже вам, як це зробити.

## Як це зробити

Це дуже просто! Ми можемо використовувати функцію `std::filesystem::exists()` з бібліотеки `<filesystem>`, яка з'явилася у C++17. Давайте подивимось на цей приклад коду:

```C++
#include <iostream>
#include <filesystem>
namespace fs = std::filesystem;
int main()
{
    fs::path dir = "my_directory";
    if(fs::exists(dir))
    {
        std::cout << "Каталог існує";
    }
    else
    {
        std::cout << "Каталог не існує";
    }
}
```
В цьому прикладі ми оголосили об'єкт `dir` типу `fs::path`, який представляє шлях до каталогу, який ми хочемо перевірити. З допомогою умовного оператора `if`, ми перевіряємо, чи існує цей каталог за допомогою функції `std::filesystem::exists()`. У випадку, якщо каталог існує, ми виводимо "Каталог існує", а в іншому випадку виводимо "Каталог не існує".

## Поглиблене дослідження

Отже, ми дізналися, як зробити перевірку наявності каталогу в C++. Тепер давайте нескінченно глибоко поглянемо на цей процес. Функція `std::filesystem::exists()` повертає значення типу `bool`, тобто `true`, якщо каталог існує, і `false` у протилежному випадку. Також, якщо ви хочете перевірити наявність файлу замість каталогу, ви можете використовувати функцію `std::filesystem::is_regular_file()`.
