## Чому

У програмуванні іноді необхідно виконувати дії з певними директоріями. Однією з таких дій може бути перевірка наявності директорії. Це може бути корисно для виконання певних умов або для впевненості, що програма буде працювати з потрібними файлами.

## Як це зробити

Для перевірки існування директорії використовуємо функцію `std::filesystem::exists()`, яка повертає `true` або `false` в залежності від того, чи існує директорія. Послідовність дій буде наступною:

```C++
#include <filesystem>
namespace fs = std::filesystem; // для зручності користування

if (fs::exists("directory_path")) {
    // Якщо директорія існує
    std::cout << "Директорія існує!\n";
} else {
    // Якщо директорія не існує
    std::cout << "Директорія не існує!\n";
}
```

У вище наведеному прикладі функція `exists()` перевіряє існування директорії `"directory_path"`, після чого виводить відповідне повідомлення в залежності від результату.

## Глибокий занурення

Перевірка наявності директорії може бути корисною і в інших випадках. Наприклад, можна використовувати цю функцію для перевірки наявності конкретного файлу, перед тим як виконувати дії з ним.

Також важливо зазначити, що ця функція може приймати абсолютний або відносний шлях до директорії. Наприклад, `"C:\\Users\\User\\Documents"` або `"../../Documents"`.
