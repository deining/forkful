## Чому 

Іноді в програмуванні необхідно порівнювати дві дати, наприклад, для визначення часу між подіями або встановлення скільки днів пройшло з певної дати. У даній статті ми розглянемо як саме це можна зробити за допомогою мови програмування C++.

## Як це зробити

Для порівняння двох дат в C++, найпростіший спосіб - використання стандартної бібліотеки `ctime`. Для цього нам знадобляться дві змінні типу `struct tm`, які зберігатимуть наші дати. Приклад коду показаний нижче:

```C++
#include <iostream>
#include <ctime>

using namespace std;

int main() {
  // ініціалізуємо дві змінні типу struct tm
  struct tm date1 = { 0, 0, 0, 15, 3, 2000 }; // 15.03.2000
  struct tm date2 = { 0, 0, 0, 25, 12, 2021 }; // 25.12.2021

  // перетворюємо змінні в формат часу time_t за допомогою функції mktime
  time_t time1 = mktime(&date1);
  time_t time2 = mktime(&date2);

  // порівнюємо отримані значення за допомогою операторів порівняння
  if (time1 > time2) {
    cout << "Перша дата пізніше за другу" << endl;
  } else if (time1 < time2) {
    cout << "Перша дата раніше за другу" << endl;
  } else {
    cout << "Дати однакові" << endl;
  }

  return 0;
}
```

Результат виконання даного коду буде виглядати наступним чином:

```bash
Перша дата раніше за другу
```

## Глибока занурення

У цій статті ми розглянули лише простий спосіб порівняння дат в C++. Однак, в мові існує багато інших методів, наприклад, можна використовувати більш розширену бібліотеку `boost::date_time` або самостійно написати функцію порівняння, використовуючи арифметичні операції над датами. Також слід бути уважним при порівнянні дат, оскільки це може призвести до помилок через різні формати та часові зони.
