## Чому

Перед тим, як розпочати будь-який проект, важливо знати, як це зробити правильно. Це стосується й програмування. Один із важливих аспектів програмування - це зчитування аргументів з командного рядка. Це дає змогу користувачам взаємодіяти з програмою, вказуючи необхідні параметри і налаштування.

## Як це зробити

Приклад зчитування аргументів з командного рядка в C++ виглядає так:

```C++
#include <iostream>
using namespace std;

int main(int argc, char *argv[]) {

  // argc - кількість переданих аргументів
  // argv[] - масив з переданими аргументами

  cout << "Кількість аргументів: " << argc << endl;
  
  // проходимо по кожному аргументу
  for (int i = 0; i < argc; i++) {
    cout << "Аргумент " << i << ": " << argv[i] << endl;
  }
  
  return 0;
}
```

Приклад введеної команди та відповідного виводу:

```
./main.exe 10 test.txt

Кількість аргументів: 3
Аргумент 0: main.exe
Аргумент 1: 10
Аргумент 2: test.txt
```

## Що розбиратися детальніше

Зчитування аргументів з командного рядка може бути корисним для багатьох проектів. Наприклад, для створення програми, яка отримує ім'я файлу для обробки даних, або для програми, яка отримує режим роботи від користувача. Подальша обробка переданих аргументів може бути зроблена за допомогою умовних операторів та циклів.
