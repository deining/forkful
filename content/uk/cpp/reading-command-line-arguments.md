## Чому

Командний рядок є важливою частиною програмування на C++. Він дозволяє користувачам передавати аргументи програмі з командного рядка, що дозволяє змінювати поведінку програми без редагування вихідного коду.

## Як

Найпростіший спосіб отримати та обробити аргументи командного рядка в устройств як вбудованих, так і зовнішніх, є за допомогою визначення параметрів функції `main()`:

```C++
int main(int argc, char* argv[])
{
    // argc містить кількість аргументів командного рядка,
    // а argv - масив з назвами аргументів

    // Обробляємо аргументи командного рядка
    for (int i = 0; i < argc; ++i)
    {
        // Виконуємо потрібні дії з aргументами
        // Наприклад, виводимо їх на екран:
        std::cout << argv[i] << std::endl;
    }

    return 0;
}
```

Приклад виводу:

`./programName input.txt output.txt`

`output.txt` буде містити `input.txt`, оскільки ми передали його як другий аргумент командного рядка.

## Глибокий занурення

Якщо вам потрібно отримати конкретні типи даних з аргументів командного рядка, ви можете скористатися стандартною бібліотекою C++ і функцією `std::stoi` для отримання цілочисельного значення із рядка.

Приведений нижче код перетворює перший аргумент командного рядка в ціле число та виводить його на екран:

```C++
int main(int argc, char* argv[])
{
    // Перетворює перший аргумент в ціле число
    int num = std::stoi(argv[1]);

    // Виводимо результат на екран
    std::cout << num << std::endl;

    return 0;
}
```

Також, ви можете використовувати бібліотеку `boost::program_options` для створення більш складних програм, які приймають аргументи з командного рядка.
