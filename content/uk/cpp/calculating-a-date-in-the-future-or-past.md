## Чому

Розрахунок дат в майбутньому або в минулому може бути корисним при написанні програм для планування або організації подій. Також це може бути важливим для розрахунку терміну дії певної послуги або контракту.

## Як робити розрахунок дат

Найпростіший спосіб розрахувати дату в майбутньому або в минулому - використати функцію `time()` та структуру `tm` зі стандартної бібліотеки C++. Нижче наведений приклад коду, який показує як розрахувати дату за допомогою цих інструментів:

```C++
#include <iostream>
#include <ctime> 
using namespace std;

int main() {
  // Отримання поточної дати та часу
  time_t now = time(0);
   
  // Перетворення поточної дати та часу в структуру tm
  tm* ltm = localtime(&now);
   
  // Розрахунок дати в майбутньому або минулому (30 днів вперед/назад)
  ltm->tm_mday += 30;
  time_t futureDate = mktime(ltm);

  // Виведення розрахованої дати в форматі "мм/дд/рррр"
  cout << "Дата в майбутньому: " << ltm->tm_mon << "/" << ltm->tm_mday << "/" << ltm->tm_year + 1900 << endl;

  return 0;
}
```

Після запуску цього коду, ви отримаєте вивід, що показує дату, яка буде через 30 днів від поточної. Можна також змінити значення `ltm->tm_mday` для розрахунку дат в інших інтервалах часу.

## Глибше в дати

Розрахунок дат в майбутньому або в минулому також може бути корисним для вирішення більш складних задач, наприклад, розрахунку дати наступного платежу або дати виконання підроботки. Для цього можна використовувати допоміжні бібліотеки, наприклад, Boost.Date_Time, яка надає більш широкі можливості для роботи з датами та часом в C++.
