##Чому
Можливо, ви плануєте важливу подію і хочете знати, яка дата буде через певний час, або вам потрібно встановити деадлайн для проекту із використанням конкретної дати. Незалежно від причини, розрахунок дати в майбутньому або минулому може допомогти вам планувати час та події більш ефективно.

##Як
Цей процес є досить простим у розумінні та реалізації за допомогою мови програмування C++. Все, що потрібно, це використати вбудовану функцію `std::chrono::system_clock::now()` для отримання поточного часу, а потім використати функцію `std::chrono::duration` для додавання або віднімання певної кількості часу від поточної дати і вивести отриманий результат у зручному форматі.

Наприклад, якщо ви хочете додати 7 днів до поточної дати, код може виглядати так:

```C++
#include <iostream>
#include <chrono>

int main() {
    auto now = std::chrono::system_clock::now();
    auto future_date = now + std::chrono::duration<int>(7, std::chrono::days());
    std::cout << "Дата через 7 днів: " << future_date << std::endl;
    return 0;
}
```

Цей код виведе наступне:

`Дата через 7 днів: 2021-02-20 12:00:00`

До цього ж коду можна застосувати і інші вбудовані методи для розрахунку дати в минулому, наприклад, вибрати певний місяць або рік для розрахунку часу.
