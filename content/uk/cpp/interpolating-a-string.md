## Що це таке та чому це потрібно?

Інтерполяція рядків - це процес подальшого об'єднання кількох рядків в один, використовуючи показівні символи, такі як `%s` або `%d`. Програмісти використовують інтерполяцію рядків для зручності та ефективності, замість використання кількох операторів `+` для об'єднання рядків.

## Як це зробити?

```C++
#include <iostream>
#include <string>

int main() {
    std::string name = "John";
    int age = 25;

    // Наприклад, ми хочемо вивести рядок "Мене звати John і мені 25 років."

    // Використовуємо показівні символи для об'єднання рядків
    std::string message = "Мене звати %s і мені %d років.";
    std::cout << message << std::endl;

    // Використовуємо функцію printf для об'єднання рядків
    printf("Мене звати %s і мені %d років.", name.c_str(), age);

    return 0;
}
```

Вивід програми:
```
Мене звати John і мені 25 років.
Мене звати John і мені 25 років.
```

## Глибше погруження

Інтерполяція рядків була вперше реалізована в мові програмування BCPL у 1966 році. Також є альтернативними способами об'єднання рядків, такими як використання функції `strcat` або типу `stringstream`. Інтерполяція рядків може бути реалізована за допомогою функцій `sprintf` або `printf`, які дозволяють використовувати показівні символи для заміни значень у рядках.
