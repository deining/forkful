## Чому

Навіть у сучасній ері інтернету, таблиці даних досі є необхідною частиною багатьох проектів програмного забезпечення. Одним із найпопулярніших форматів для зберігання та обробки даних є CSV (Comma Separated Values). Робота з CSV файлами може бути складною і вимагати багато часу, але вона значно полегшує роботу з великим обсягом даних, зокрема в області фінансів, бухгалтерії та аналітики.

## Як

Найпростішим способом працювати з CSV файлами в C++ є використання бібліотеки CSV-parser. Для початку потрібно створити об'єкт цієї бібліотеки, який буде представляти CSV файл:

```C++
CSVReader dataset("data.csv");
```

Далі можна отримати доступ до даних, використовуючи індекси стовпців та рядків:

```C++
std::cout << dataset[0][2] << std::endl; // виведе елемент на перетині першого рядка та третього стовпця
```

Також можна витягнути всі дані з рядка або стовпця за допомогою циклу:

```C++
for(auto& row : dataset){
    std::cout << row[1] << std::endl; // виводимо значення з другого стовпця усіх рядків
}
```

Для зручності, бібліотека CSV-parser також надає можливість вибирати дані, використовуючи імена стовпців:

```C++
for(auto& row: dataset.getHeader()){
    std::cout << row["Name"] << std::endl; // виводимо значення всіх рядків у стовпці з іменем "Name"
}
```

## Поглиблене дослідження

Бібліотека CSV-parser має багато корисних функцій, таких як можливість додавати нові рядки чи стовпці, видаляти рядки та стовпці, а також робити фільтрацію та сортування даних. Також вона може обробляти різні типи даних, включаючи рядки, цілі числа та дати.

Щоб дізнатися більше про можливості цієї бібліотеки, рекомендуємо ознайомитися з [документацією](https://github.com/ben-strasser/fast-cpp-csv-parser), а також [прикладами](https://github.com/LukasBanana/CSV-parser/tree/master/examples) використання.
