## Чому
Перевірка наявності директорії є важливою частиною багатьох програм на Go. Це допомагає переконатися, що необхідні файли та директорії існують, перед тим як починати виконувати інші операції з ними.

## Як це зробити
Для перевірки існування директорії в Go, ми можемо використовувати функцію `os.Stat()`. Вона повертає інформацію про файл або директорію, яка допоможе нам перевірити їх наявність.

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    path := "/home/user/Documents"
    fileInfo, err := os.Stat(path)
    if err != nil {
        fmt.Printf("Директорія %s не існує.\n", path)
    } else {
        fmt.Printf("Директорія %s існує.\n", path)
        fmt.Printf("Розмір: %d байт. Права доступу: %s.\n", fileInfo.Size(), fileInfo.Mode())
    }
}
```

Вище ми створили змінну `path`, що містить шлях до директорії, яку ми хочемо перевірити. Потім викликали функцію `os.Stat()`, передаючи їй цей шлях. Якщо директорія існує, то ми отримаємо інформацію про неї у змінній `fileInfo`. Якщо ж директорії не існує, то ми отримаємо помилку, яку ми роздруковуємо.

Для перевірки існування директорії можна також використовувати функцію `os.IsExist()`. Вона повертає `true`, якщо файл або директорія існують, і `false` в іншому випадку.

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    path := "/home/user/Music"
    if os.IsExist(path) {
        fmt.Printf("Директорія %s існує.\n", path)
    } else {
        fmt.Printf("Директорія %s не існує.\n", path)
    }
}
```

## Deep Dive
Функції `os.Stat()` та `os.IsExist()` використовують функцію `syscall.Stat()` для отримання інформації про файл або директорію. Ця функція викликає системний виклик `stat()` для отримання цієї інформації.

Також варто відзначити, що функції `os.Stat()` та `os.IsExist()` перевіряють існування файлу або директорії на локальній файловій системі. Якщо ви хочете перевірити існування файлу або директорії на віддаленому сервері, то вам потрібно буде використовувати інші методи, наприклад, виконати
