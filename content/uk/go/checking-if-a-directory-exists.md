## Чому

Існує багато ситуацій, коли нам потрібно перевірити, чи існує певна директорія у нашій програмі. Наприклад, може бути необхідно створити нову директорію, якщо вона ще не існує, або виконати певні дії тільки в тому випадку, якщо директорія існує. Тому, в даній статті ми розглянемо як перевірити, чи існує директорія у нашій програмі за допомогою мови програмування Go.

## Як це зробити

Щоб перевірити, чи існує директорія, нам потрібно використати бібліотеку "os" та її метод `Stat()`. Цей метод повертає об'єкт `os.FileInfo`, який містить інформацію про файл або директорію, в тому числі і її ім'я та шлях. Ми можемо перевірити, чи директорія існує, перевіривши чи останній елемент шляху відповідає імені директорії. Нижче наведено приклад коду:

```Go
package main

import (
  "fmt"
  "os"
)

func main() {
  path := "./test_dir" // шлях до директорії
  if _, err := os.Stat(path); os.IsNotExist(err) { // перевіряємо чи існує директорія
    fmt.Println("Директорія не існує.")
  } else {
    fmt.Println("Директорія існує.")
  }
}
```

Якщо директорія не існує, програма виведе "Директорія не існує." інакше, буде виведено "Директорія існує.". Це дуже простий приклад, але ми можемо використати цей метод для виконання більш складних дій, наприклад, створення директорії за потребою, виконання дій з файлами в певній директорії і т.д.

## Поглиблене дослідження

Намагаючись отримати інформацію про файл або директорію за допомогою методу `Stat()`, може бути корисною наступна інформація:

- Якщо об'єкт `os.FileInfo` повертається без помилки, це означає, що файл або директорія існують.
- Якщо була повернена помилка, то це означає, що файл або директорія не існують.
- Якщо помилка не пов'язана з фактом, що файл або директорія не існують, то це може бути пов'язано з правами доступу або якимось іншими причинами.
