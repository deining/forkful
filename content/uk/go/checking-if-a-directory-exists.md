## Чому

Робота з файловою системою є необхідною складовою для багатьох програм, написаних на мові програмування Go. Перевірка наявності директорії є важливою задачею, яка допомагає забезпечити коректну роботу програми, а також захистити від помилок.

## Як перевірити наявність директорії

Для перевірки наявності директорії вам потрібно використовувати функцію `os.Stat()`, яка повертає об'єкт `os.FileInfo`. Також вам знадобиться імпортувати пакет `os`. Ось приклад коду, який перевіряє наявність директорії під назвою `test` у поточній директорії:

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	fileInfo, err := os.Stat("test")
	if os.IsNotExist(err) {
		fmt.Println("Директорія не існує.")
	} else {
		fmt.Println("Директорія існує.")
		fmt.Println("Ім'я директорії:", fileInfo.Name())
		fmt.Println("Розмір директорії:", fileInfo.Size())
		fmt.Println("Дозволи на запис:", fileInfo.Mode().Perm())
		fmt.Println("Дата і час останньої зміни:", fileInfo.ModTime())
	}
}
```

Приклад виводу програми:

```
Директорія існує.
Ім'я директорії: test
Розмір директорії: 4096
Дозволи на запис: 755
Дата і час останньої зміни: 2021-08-12 10:00:00 +0300 EEST
```

## Глибоке занурення

Функція `os.Stat()` перевіряє не тільки наявність директорії, але і повертає інформацію про неї, таку як ім'я, розмір, дозволи на запис та дату останньої зміни. Це дозволяє докладніше вивчити директорію, яку ви перевіряєте, і використовувати цю інформацію для подальшого функціонування вашої програми.
