## Чому

Під час програмування на Go, може виникнути ситуація, коли нам потрібно перевірити, чи існує директорія в нашій файловій системі. Це важливо, щоб уникнути помилок та зберегти час, перевіряючи наявність директорії перед обробкою файлів.

## Як це зробити

Ми можемо використовувати функцію `os.Stat()` для перевірки наявності директорії в файловій системі. Також ми можемо використовувати оператор `os.IsNotExist()` для перевірки, чи не виникла помилка під час перевірки.

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	// Шлях до директорії, яку ми перевіряємо
	path := "my_directory"

	// Використовуємо функцію `os.Stat()` для отримання інформації про директорію
	// Якщо директорія існує, то `err` буде рівним `nil`
	_, err := os.Stat(path)

	// Використовуємо оператор `os.IsNotExist()` для перевірки наявності помилки,
	// яка означає, що директорія не існує
	if os.IsNotExist(err) {
		fmt.Println("Директорія не існує")
	} else {
		fmt.Println("Директорія існує")
	}
}
```

Приклад виводу:

```
Директорія не існує
```

## Глибший аналіз

Отримана за допомогою `os.Stat()` інформація про директорію містить багато корисної інформації, такої як розмір, час створення та змінення, права доступу і т.д.

У разі, якщо ми не отримуємо помилку при перевірці і також не перевіряємо наявність файла, але хочемо переконатися, що директорія справді є директорією, ми можемо використовувати функцію `os.IsDir()`. Ця функція повертає `true` у разі, якщо шлях відноситься до директорії.
