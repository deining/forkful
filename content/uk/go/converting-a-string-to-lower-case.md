# Тому: Почему?

Перетворення строки на рядок меншого регістру є важливою операцією в багатьох програмах Go. Це допомагає полегшити пошук та порівняння різних рядків даних, забезпечуючи єдність в написанні програмного коду.

В цій блозі ми покажемо вам, як легко виконувати цю операцію в програмі Go.

## Як це зробити

Існує кілька способів конвертувати рядок до різних регістрів, але ми можемо використовувати функцію `strings.ToLower` для зручного перетворення рядка в менший регістр.

Подається код виконання цієї операції:

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	str := "Приклад Рядка Великих Літер"
	fmt.Println(strings.ToLower(str))
}
```

Результат виконання коду буде наступним:

```Go
приклад рядка великих літер
```

Надзвичайно просто! Вибір `strings.ToLower` значно полегшує перетворення рядка в менший регістр.

Ви також можете скористатися функцією `strings.ToLower` для перетворення лише першої літери рядка:

```Go
fmt.Println(strings.ToLower(str)[0:1] + strings.ToLower(str)[1:])
```

## Глибокий дайв

Щоб краще розібратися з процесом конвертації рядка в менший регістр, звернемося до документації Go про функцію `strings.ToLower`. Вона приймає рядок як аргумент і повертає його копію в меншому регістрі.

Ця функція також знає, як обробляти рядки із нелатинськими символами, надаючи правильне перетворення Юнікод-рядків в менший регістр.

Для більшої гнучкості, функція `strings.ToLower` приймає другий аргумент - структуру `locale`.

Розглянемо приклад використання структури `locale` в функції `strings.ToLower`:

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	str := "Пример Строки Больших Литер"
	fmt.Println(strings.ToLowerSpecial(strings.Locale, str))
}
```

Вивід коду буде наступним:

```Go
пример строки больших литер
```
