## Чому

Збереження рядків - важлива частина багатьох програм. Комбінування рядків разом може бути корисним для створення повідомлень, логів, запитів до баз даних та багатьох інших сценаріїв. У цій статті ми поговоримо про те, чому комбінування рядків у Go є корисним та як це можна зробити.

## Як це зробити

Почнімо зі створення 2 рядків і використання знака "+" для їх комбінування:

```Go
str1 := "Привіт"
str2 := "Україно!"
fmt.Println(str1 + " " + str2)
```

Ви отримаєте вивід:

```
Привіт Україно! 
```

Також можна використовувати `fmt.Sprintf` для форматування бажаного виводу. Наприклад:

```Go
name := "Олеся"
age := 25
fmt.Println(fmt.Sprintf("Мене звати %s і мені %v років", name, age))
```

Результат буде наступним:

```
Мене звати Олеся і мені 25 років
```

Також можна використовувати `strings.Join` для комбінування багатьох рядків разом. Наприклад:

```Go
strSlice := []string{"Гарного", "дня", "українці!"}
fmt.Println(strings.Join(strSlice, " "))
```

Ви отримаєте наступний вивід:

```
Гарного дня українці!
```

## Глибоке погруження

Крім звичайного комбінування рядків, в Go є також можливість форматування рядків за допомогою пакету `format`. Це дуже корисний інструмент для створення складних шаблонів. Також варто зазначити, що `strings.Join` може бути ефективнішим способом комбінування багатьох рядків, особливо коли маєте справу з великими об'ємами даних.
