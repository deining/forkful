##Чому

Написання текстового файлу є важливим елементом програмування, оскільки він дозволяє зберігати та передавати дані між програмами та користувачами. Також, це є простим способом для організації та збереження великих обсягів даних.

##Як

Нижче наведені приклади коду і результатовної виведення, які покажуть, як легко створювати та записувати текстові файли в Go:

```Go
// Створюємо файл з назвою "file.txt"
file, err := os.Create("file.txt")

if err!= nil {
  // обробка помилки
}

// Записуємо рядок в файл
_, err = file.WriteString("Привіт Go!")

if err != nil {
  // обробка помилки
}

// Закриваємо файл 
file.Close()
```

Результат:

Файл з назвою "file.txt" створений та в нього записаний рядок "Привіт Go!".

##Глибока навчальна програма

Написання текстових файлів також може бути більш складним та має додаткові функціональні можливості. Наприклад, можна задавати режим доступу до файлу, додавати нові рядки в кінець файлу та читати вже існуючі файли.

```Go
// Відкриваємо файл для зчитування
file, err := os.Open("file.txt")

if err != nil {
  // обробка помилки
}

// Читаємо вміст файлу та зберігаємо його в змінну content
content, err := ioutil.ReadAll(file)

if err != nil {
  // обробка помилки
}

// Додаємо новий рядок у кінець файлу
_, err = file.WriteString("\nЦе другий рядок.")

if err != nil {
  // обробка помилки
}

// Закриваємо файл
file.Close()

// Виводимо вміст файла
fmt.Println(string(content))
```

Результат:

Файл "file.txt" містить два рядки: "Привіт Go!" та "Це другий рядок.".
