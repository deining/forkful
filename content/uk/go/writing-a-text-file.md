## Чому

Запис текстового файлу може бути важливим етапом у написанні програм на Go. Інколи це необхідно для збереження деякої інформації або комунікації з іншими компонентами програми.

## Як створити текстовий файл у Go

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    // Відкрити файл для запису
    file, err := os.Create("file.txt")
    
    // Перевірити наявність помилок
    if err != nil {
        fmt.Println(err)
        return
    }
    
    // Записати текст у файл
    fmt.Fprintln(file, "Привіт світе!")
    
    // Закрити файл
    file.Close()
    fmt.Println("Файл успішно записано.")
}
```

Вище наведений приклад демонструє як створити текстовий файл у Go та записати у нього деякий текст. Запис відбувається за допомогою функції `Fprintln`, яка автоматично додає символ нового рядка після тексту.

## Глибше в деталі

У Go є кілька способів записувати текст у файл. Найпростіший спосіб - використовувати функцію `Fprintln` з пакету `fmt`, як показано в прикладі вище. Але при записі більших обсягів даних цей підхід може бути неефективним. У такому випадку варто розглянути інші пакети, такі як `bufio` або `io`, для більш оптимального використання ресурсів.
