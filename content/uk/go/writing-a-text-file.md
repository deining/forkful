## Чому

Найпростіший спосіб зберегти і відформатувати текстові дані - написати їх у текстовий файл. Це може бути корисно для збереження великої кількості інформації, такої як нотатки або таблиці даних.

## Як

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	// Створення нового файлу з назвою "example.txt"
	file, err := os.Create("example.txt")
	if err != nil {
		fmt.Println(err)
		return
	}
	defer file.Close()

	// Запис тексту у файл
	fmt.Fprintln(file, "Вітаємо у текстовому файлі!")
	fmt.Fprintln(file, "Це - приклад того, як його створити та записати дані.")
}
```

Вище наведений код використовує пакет `os` для створення і записування текстового файлу. Перший рядок цього коду створює новий файл з назвою "example.txt". Далі, за допомогою функції `fmt.Fprintln()`, ми можемо записати текст у файл по рядках. Наприклад, у даному випадку ми записуємо два рядки з привітанням та описом файлу. По завершенні роботи з файлом, ми закриваємо його за допомогою `file.Close()`.

```Go
package main

import (
	"fmt"
	"io/ioutil"
)

func main() {
	// Читання змісту файлу
	data, err := ioutil.ReadFile("example.txt")
	if err != nil {
		fmt.Println(err)
		return
	}

	// Виведення інформації з файлу
	fmt.Println(string(data))
}

// Output:
// Вітаємо у текстовому файлі!
// Це - приклад того, як його створити та записати дані.
```

Щоб прочитати дані з файлу, ми використовуємо функцію `ioutil.ReadFile()`, яка повертає байтовий масив. Зміст файла можна вивести за допомогою функції `fmt.Println()`, яка отримує аргумент типу `string`.

## Deep Dive

У цьому прикладі ми коротко познайомилися зі створенням і записом текстового файлу. Однак, у Go є багато інших інструментів та пакетів для більш складних операцій з файлами і даними. Наприклад, ви можете використовувати пакети `bufio` або `encoding/csv` для роботи з текстовими файлами, які містять структуровані дані. Також, є можливість працювати з бінарними файлами та використовувати різні методи читання та запису для отримання більш точного контролю над файлами.
