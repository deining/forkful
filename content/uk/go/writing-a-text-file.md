## Що і чому?

Записування текстового файлу - це процес створення файлу, який містить текстову інформацію. Це корисно для програмістів, які хочуть зберегти певну інформацію для подальшого використання у своїх програмах.

## Як:

Кодування та запис текстового файлу в Go дуже прості. Приведений нижче приклад показує, як створити новий файл, відкрити його для запису та додати деякий текст. Використовуйте ```os.Create ()``` для створення файлу, ```io.WriteString ()``` для запису та ```os.OpenFile ()``` для відкриття файлу для читання.

```
package main

import (
	"io"
	"os"
)

func main() {
	file, err := os.Create("newfile.txt") // Створіть новий файл
	if err != nil {
		log.Fatal(err) // Обробка помилки
	}
	defer file.Close() // Закрийте файл при завершенні функції

	// Запишіть текст в файл
	text := "Привіт, це текст, який буде записаний у файл!\n"
	io.WriteString(file, text)
}
```

В результаті ви отримаєте файл з назвою "newfile.txt", який буде містити текст, який ви записали у своєму коді.

## Глибокий погляд

Записування текстових файлів - це не новий процес і не ексклюзивно для мови програмування Go. Багато мов програмування постачають функції для роботи з файлами, включаючи запис та читання тексту. Альтернативою може бути використання бази даних, де дані зберігаються у вигляді таблиць з рядками та колонками. Реалізація запису до файлу включає в себе створення файлу та використання різних методів для запису даних у файл.
