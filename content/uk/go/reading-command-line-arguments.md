## Чому

На жаль, іноді нам потрібно декілька конкретних інструкцій для того, щоб запустити наші програми вірно і з бажаним результатом. Зчитування аргументів командного рядка є однією з таких інструкцій. Без цього навички, ми не зможемо передати додаткові параметри до нашої програми або взагалі не зможемо запустити її нашою власною умовою.

## Як

Найкращий спосіб навчитися читати аргументи командного рядка в Go - це за допомогою прикладів. Давайте розглянемо декілька прикладів коду, де ми зчитуємо аргументи командного рядка та демонструємо їх вивід.

```Go
// Приклад 1: Зчитування одного аргументу
package main

import (
    "fmt"
    "os"
)

func main() {
    // Зчитуємо перший аргумент командного рядка
    arg := os.Args[1]
    // Виводимо його на екран
    fmt.Println("Перший аргумент:", arg)
}

// Вивід:
// Перший аргумент: привіт
```

```Go
// Приклад 2: Зчитування декількох аргументів
package main

import (
    "fmt"
    "os"
)

func main() {
    // Зчитуємо всі аргументи командного рядка
    args := os.Args[1:]
    // Виводимо їх на екран
    fmt.Println("Аргументи:", args)
}

// Вивід:
// Аргументи: [привіт, світе]
```

```Go
// Приклад 3: Зчитування числового аргументу
package main

import (
    "fmt"
    "os"
    "strconv"
)

func main() {
    // Зчитуємо третій аргумент командного рядка
    arg := os.Args[3]
    // Конвертуємо його в числовий тип
    num, _ := strconv.Atoi(arg)
    // Виводимо результат
    fmt.Printf("%d + 5 = %d\n", num, num + 5)
}

// Вивід:
// 10 + 5 = 15
```

Як бачимо, зчитування аргументів командного рядка дуже просто і дозволяє нам зробити наші програми більш гнучкими та корисними.

## Глибоке занурення

Зчитування аргументів командного рядка може бути не тільки простою задачею, але й дуже корисною. Наприклад, ви можете передавати додаткові параметри до програми, щоб змінювати її поведінку відповідно до ваших потреб. Також, ви можете зчитувати аргументи з конфігураційних файлів або з використанням прапорців (-key=value).
