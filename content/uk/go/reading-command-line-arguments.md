## Чому 

Отримання аргументів командного рядка є важливою частиною програмування в Go. Все більше і більше програмистів використовують цю можливість для покращення ефективності та гнучкості своїх програм. Також це дає можливість користувачам налаштувати програму під свої потреби. У цій статті ми розглянемо, як саме отримати аргументи командного рядка в Go та як це може бути корисним для вас.

## Як це зробити

Для отримання аргументів командного рядка в Go використовується `os.Args` пакет. Цей пакет містить надзвичайно корисні функції для роботи з аргументами командного рядка. Для початку, потрібно ініціалізувати цей пакет у своєму коді, вказавши `import "os"`.

Далі, ми можемо використовувати функції `len()` та `os.Args` для отримання кількості аргументів та масиву зі всіма аргументами. Наприклад:

```Go
package main

import "fmt"
import "os"

func main() {
    // отримання кількості аргументів
    fmt.Println("Кількість аргументів:", len(os.Args))

    // отримання всіх аргументів
    fmt.Println("Аргументи:", os.Args)
}
```

Після цього, ми можемо використовувати ці аргументи для своїх потреб. Наприклад, показати повідомлення користувачу залежно від переданих аргументів, читати файл для подальшої обробки або виконати інші необхідні дії.
