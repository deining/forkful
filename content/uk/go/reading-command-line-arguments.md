## Чому

Читання аргументів командного рядка є важливим аспектом при програмуванні в Go. Використання аргументів командного рядка дозволяє передавати вхідні дані в програму від користувача з зручним і простим способом.

## Як це зробити

Для читання аргументів командного рядка в Go існує вбудований пакет `flag`, який дозволяє легко визначати необхідні аргументи та їх значення. Нижче приведений код прикладу використання цього пакету:

```Go
import "flag"

// Оголошення змінної, яка буде зберігати значення аргумента командного рядка
var name string

// Визначення аргументу командного рядка зі значенням за замовчуванням та описом
flag.StringVar(&name, "name", "World", "Enter your name")

// Парсинг аргументів командного рядка
flag.Parse()

// Виведення аргументу командного рядка на екран
fmt.Println("Hello,", name)
```

Використовуючи цей підхід, ми можемо передати значення аргументу командного рядка при запуску програми, наприклад: `go run main.go -name John`. Результат виконання програми буде `Hello, John`.

## Глибоке занурення

Пакет `flag` також має можливості для більш складних випадків читання аргументів командного рядка, таких як вказування типу даних та перевірки на валідність. Більш детальна інформація про можливості пакету може бути знайдена у [документації Go](https://golang.org/pkg/flag/).
