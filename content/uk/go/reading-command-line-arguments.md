## Чому

Чи вам коли-небудь знадоблялося передавати якісь параметри або налаштування власної програми під час її запуску? Якщо так, то ви, напевно, повинні були звертатися до командного рядка і передавати аргументи. В цій статті ми розглянемо, як саме можна читати аргументи з командного рядка у мові програмування Go.

## Як це зробити

Для зчитування аргументів командного рядка у Go використовується пакет `flag`. Давайте подивимося на приклад коду, який демонструє простий спосіб зчитування аргументів та їх значень:

````Go
package main

import (
	"flag"
	"fmt"
)

func main() {
	// визначаємо аргумент зі значенням за замовчуванням
	name := flag.String("name", "Анонімний", "ім'я користувача")

	// розбір аргументів командного рядка
	flag.Parse()

	// використання значення аргументу у програмі
	fmt.Println("Вітаємо,", *name, "!")
}
````

Тепер, якщо ми запустимо нашу програму з цим кодом і передамо аргумент `-name`, наприклад `go run main.go -name Ігор`, то у консоль виведеться `Вітаємо, Ігор!`. А якщо не передати жодного аргументу, то виведеться значення за замовчуванням - `Вітаємо, Анонімний!`.

Цей приклад демонструє лише основи роботи з аргументами командного рядка, але пакет `flag` має багато інших корисних функцій для більш складних випадків.

## Глибоке занурення

Крім пакету `flag`, є ще один популярний спосіб читання аргументів командного рядка у Go - це використання пакету `os`. Цей пакет дозволяє отримати доступ до усіх аргументів командного рядка у вигляді масиву рядків і також має багато інших корисних функцій для роботи з командним рядком.

Більш детальну інформацію про роботу з аргументами командного рядка у Go ви можете знайти у документації мови програмування або на різних веб-ресурсах, присвячених цій темі.
