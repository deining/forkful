# Чому

У програмуванні дата і час є важливими концепціями, оскільки вони дозволяють відстежувати час виконання програми та робити необхідні обчислення. Отримання поточної дати використовується для різних цілей, таких як логування подій, обчислення тривалості виконання програми та відображення поточного часу в інтерфейсі користувача.

# Як

В Go існує спеціальна стандартна бібліотека `time`, яка дозволяє отримувати поточну дату та час. Для цього необхідно використовувати функцію `Now()` з пакету `time`, яка повертає значення типу `Time` - структури, яка містить інформацію про дату і час.

```Go
import "time"

func main() {
    currentTime := time.Now()
    fmt.Println(currentTime)
}
```

Виходом цього коду буде поточна дата та час у форматі `2021-09-23 15:30:00 +0300 EEST`.

# Огляд

Якщо ви бажаєте отримати більш детальну інформацію про поточний час, в `Time` структурі є різні методи, що дозволяють отримати окремі компоненти дати та часу, наприклад, `Day()`, `Month()`, `Year()`, `Hour()`, `Minute()`, `Second()` та інші. Також існує метод `Format()`, який дозволяє вказати необхідний формат для виведення дати та часу.

```Go
import "time"

func main() {
    currentTime := time.Now()
    fmt.Println("Рік:", currentTime.Year())
    fmt.Println("Місяць:", currentTime.Month())
    fmt.Println("День:", currentTime.Day())
    fmt.Println("Година:", currentTime.Hour())
    fmt.Println("Хвилина:", currentTime.Minute())
    fmt.Println("Секунда:", currentTime.Second())
    fmt.Println("Дата та час:", currentTime.Format("02-01-2006 15:04:05"))
}
```

Виходом цього коду буде повідомлення з окремими компонентами дати та часу, а також повна дата та час у форматі `23-09-2021 15:30:00`.
