# Що і чому?

Отримання поточної дати - це процес отримання поточної дати та часу на комп'ютері. Програмісти часто використовують цю функцію для виконання різних завдань, наприклад, для збереження даних у відповідних часах.

# Як:

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	currentTime := time.Now()

	fmt.Println("Поточна дата та час: ", currentTime)
}
```

Вийде:

```Go
Поточна дата та час: 2021-09-16 21:31:25.514646629 +0300 EEST m=+0.001524001
```

# Глибше:

Для отримання поточної дати та часу у програмі є кілька альтернативних методів, наприклад, використання сторонніх бібліотек або звернення до внутрішніх системних засобів. У мові програмування Go є вбудована функція `time.Now()`, яка повертає поточну дату та час у вигляді значення типу `time.Time`.
