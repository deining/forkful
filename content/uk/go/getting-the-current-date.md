## Пояснення

Уяви, що ти пишеш свою першу програму на Go. При цьому, ти хочеш, щоб вона виводила поточну дату. Це дозволить тобі відслідковувати, коли в твоїй програмі відбуваються певні події або коли було зроблено певні зміни.

## Пошагова Інструкція

Не всі знають, як отримати поточну дату у своєму коді на Go. Тому, давайте розглянемо кілька простих прикладів.

```Go
package main 

import ( 
	"fmt" 
	"time" 
) 

func main() { 
	fmt.Println("Поточна дата:", time.Now().Format("01-02-2006")) 
} 
```

Цей код буде виводити поточну дату в форматі "місяць-день-рік". Також, ти можеш використовувати інші формати дати, за допомогою різних комбінацій чисел і літер у функції `Format()`.

```Go
package main 

import ( 
	"fmt" 
	"time" 
) 

func main() { 
	fmt.Println("Поточна дата та час:", time.Now().Format("01-02-2006 15:04:05")) 
} 
```

У цьому прикладі, ми використовуємо формат "місяць-день-рік години:хвилини:секунди" для виводу поточної дати та часу.

## Глибока поглядів

Тепер, коли ти знаєш, як отримати поточну дату у своєму коді на Go, давайте розглянемо те, як саме цей процес працює.

У Go, є вбудований пакет `time`, який дозволяє нам працювати з часом та датами. Функція `Now()` повертає поточний час, а `Format()` дозволяє нам вибрати потрібний формат для виведення дати.

Також, можна використовувати інші функції з пакету `time`, наприклад, `AddDate()`, щоб додати/відняти дні, місяці чи роки до/від поточної дати.

Надіємося, що тебе захопив цей аспект програмування на Go та ти зможеш використовувати отримані знання у своїх проектах.
