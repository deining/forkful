## Чому

Використання поточної дати є важливим аспектом програмування, оскільки це дозволяє робити динамічні обчислення та забезпечує актуальність даних у програмі. Також це корисно для відстеження часу виконання певних дій у програмі.

## Як

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	currentDate := time.Now()
	fmt.Println("Поточна дата:", currentDate.Format("02-01-2006"))
}
```

При запуску цього коду, ви отримаєте вихідну дату у форматі день-місяць-рік, наприклад: 20-12-2021. Ви можете змінювати форматування залежно від ваших потреб, використовуючи метод Format().

## Глибоке занурення

У Go існує багато різних методів отримання поточної дати та часу, таких як time.Now(), time.Date() або time.Parse(). Це дозволяє робити більш точні обчислення та маніпулювати різними аспектами дати та часу. Також варто згадати, що часові зони та локалі можуть впливати на отримані значення.
