## Чому

Отримання поточної дати є важливим елементом багатьох програм, особливо тих, які працюють зі справами часу або датами. Це дає можливість робити розрахунки часу, створювати звіти або просто відображати поточну дату для користувачів.

## Як це зробити

Використовуючи мову програмування Go, надзвичайно просто отримати поточну дату. Для цього ви можете використовувати стандартний пакет "time". Ось приклад коду:

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    now := time.Now()

    fmt.Println("Поточна дата:", now.Format("02.01.2006"))
    fmt.Println("Поточний час:", now.Format("15:04:05"))
}
```

Виходом буде щось на зразок:

```
Поточна дата: 03.10.2021
Поточний час: 15:32:49
```

Також ви можете використовувати методи "Year()", "Month()", "Day()" та інші, щоб отримати окремі елементи дати.

## Глибока занурення

Коли ви отримуєте поточну дату, важливо знати, що вона залежить від часової зони вашого сервера або комп'ютера. Тому, обов'язково встановіть правильну часову зону перед отриманням поточної дати. Це можна зробити за допомогою методу "LoadLocation()" та використовувати його в методі "Now()".

Також існує багато корисних методів для роботи з датами в пакеті "time". Наприклад, ви можете додавати або віднімати час до дати за допомогою методу "Add()", перевіряти на рівність або порівнювати дати за допомогою методів "Equal()" і "Before()"/"After()".
