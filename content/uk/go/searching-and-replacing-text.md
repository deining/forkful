## Чому

У програмуванні часто потрібно замінити певний текст на інший. Це може бути необхідно для виправлення помилок, оновлення даних або зміни форматування. У цій статті ми розглянемо, як це зробити важливу задачу у Go мові програмування.

## Як

Для заміни тексту в Go використовується функція `strings.Replace()`. Ця функція приймає три аргументи: вихідний рядок, рядок для заміни і рядок з новим текстом. Поміж рядками можна вказати ще один аргумент - кількість входжень, які потрібно замінити.

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	str := "Привіт, світ! Це старий текст."
	newStr := strings.Replace(str, "старий", "новий", 1)
	fmt.Println(newStr) // виведе "Привіт, світ! Це новий текст."
}
```

У цьому прикладі ми використали функцію `strings.Replace()` для заміни слова "старий" на "новий" у рядку. Зауважте, що ми вказали 1 як третій аргумент, тому заміна відбулася лише один раз. Якщо цей аргумент не вказувати, за замовчуванням будуть замінені всі входження.

## Детальний аналіз

У Go є також інші функції для пошуку і заміни тексту, такі як `strings.ReplaceAll()` або `strings.ReplaceFirst()`. Кожна з цих функцій має свої особливості і може бути корисна в певних ситуаціях. Для більшої гнучкості та контролю над процесом заміни, можна використовувати і регулярні вирази.
