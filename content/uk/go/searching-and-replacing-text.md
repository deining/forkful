## Чому
Цей стаття просто пояснює, як зробити пошук і заміну тексту у програмі Go. Така функціональність може бути корисною для виправлення орфографічних помилок або для автоматичної заміни певних фраз у великих текстових файлів.

## Як зробити
У програмі Go існує вбудована функція `strings.Replace`, яка дозволяє шукати та замінювати текст у рядках. Нижче наведений приклад використання цієї функції:

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	text := "Hello, world!"
	newText := strings.Replace(text, "world", "Ukraine", 1)

	fmt.Println(newText)
}
```
Output:
```
Hello, Ukraine!
```

## Глибока навчання
Щоб краще розібратися з пошуком і заміною тексту у програмі Go, ми можемо розглянути кожну змінну, що використовується в прикладі вище. `strings.Replace` приймає три аргументи: вихідний рядок, підстрічку для пошуку і підстрічку для заміни. Четвертим, необов'язковим аргументом є кількість входжень для заміни, за замовчуванням вона замінює всі входження. Також, при використанні функції `strings.Replace` важливо враховувати регістр символів у словах, оскільки вона буде шукати точний співпадіння.
