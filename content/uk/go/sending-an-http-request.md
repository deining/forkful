# Чому

Відправлення HTTP-запиту є важливою частиною програмування в сучасному інтернеті. Вона дозволяє взаємодіяти з веб-сайтами та отримувати необхідну для вашої програми інформацію. Наприклад, ви можете використовувати HTTP-запити для отримання даних зі своєї бази даних або для отримання останніх новин з веб-сайтів.

# Як

Для відправлення HTTP-запиту використовується пакет `net/http` в Go. Для цього нам потрібно визначити `Client` та `Request` об'єкти. Далі ми можемо викликати метод `Do()` на клієнті, щоб виконати запит та отримати відповідь. В кодовому блоку нижче показано приклад використання GET-запиту для отримання HTML-сторінки.

```Go
package main

import (
  "fmt"
  "io/ioutil"
  "net/http"
)

func main() {
  // Створення HTTP-клієнта
  client := &http.Client{}

  // Створення запиту
  req, err := http.NewRequest("GET", "https://example.com", nil)
  if err != nil {
    fmt.Println("Помилка створення запиту:", err)
  }

  // Виконання запиту та отримання відповіді
  resp, err := client.Do(req)
  if err != nil {
    fmt.Println("Помилка виконання запиту:", err)
  }
  defer resp.Body.Close()

  // Читання тіла відповіді
  body, err := ioutil.ReadAll(resp.Body)
  if err != nil {
    fmt.Println("Помилка читання відповіді:", err)
  }

  // Виведення вмісту сторінки
  fmt.Println(string(body))
}
```

Виконання цього прикладу дасть вам виведений HTML-код сторінки з `example.com`.

# Детальний опис

Для відправлення більш складних HTTP-запитів, ви можете використовувати більш широкі можливості пакета `net/http`, такі як налаштування заголовків, передача параметрів у форматі JSON та багато іншого. У цьому випадку ви можете створити об'єкт `Request` вручну та налаштувати його перед виконанням запиту. 

Якщо ви прагнете більшого контролю, ви можете використовувати пакет `net/http` разом з пакетом `net/http/httputil`, який надає функціонал для керування редиректами та відлагодження HTTP-запитів. Ви також можете використовувати пакет `net/url` для побудови URL-адрес з параметрами запиту.
