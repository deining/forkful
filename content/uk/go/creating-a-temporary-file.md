## Чому

Створення тимчасових файлів є важливою частиною багатьох програмних проєктів. Це може бути корисно для збереження тимчасових даних або запуску деяких процесів. У цій статті ми поговоримо про те, як створити тимчасовий файл в мові програмування Go.

## Як

```Go
import "io/ioutil"
import "log"
import "os"

func main() {
    // Створення тимчасового файлу за допомогою функції ioutil.TempFile
    tempFile, err := ioutil.TempFile("", "example")
    if err != nil {
        log.Fatal(err)
    }
    // Написання тексту у тимчасовий файл
    text := "Це тимчасовий файл!"
    _, err = tempFile.Write([]byte(text))
    if err != nil {
        log.Fatal(err)
    }
    // Закриття файлу
    err = tempFile.Close()
    if err != nil {
        log.Fatal(err)
    }
    // Виведення назви тимчасового файлу
    fmt.Println("Назва тимчасового файлу:", tempFile.Name())
    
    // Видалення тимчасового файлу
    err = os.Remove(tempFile.Name())
    if err != nil {
        log.Fatal(err)
    }
}
```
Виведення:
Назва тимчасового файлу: /var/folders/7w/2g6nvrl977z2wp_zp_jm9fhc0000gn/T/example971201388

## Глибокий аналіз

Зараз, коли ми знаємо як створювати тимчасові файли в мові Go, давайте розберемось детальніше про процес їх створення. Функція `ioutil.TempFile` приймає два аргументи - перший є префіксом для назви тимчасового файлу, а другий є розширенням. Якщо префікс не заданий, то він буде `tmp`. Потім, вона створює файл у директорії зазначеній у змінній `TMPDIR` або у директорії за замовчуванням.

Функція `Write` приймає масив байтів який потрібно записати у файл. У цьому прикладі ми передаємо строку яку необхідно перетворити до масиву байтів.

Не забудьте закрити створений тимчасовий файл за допомогою функції `Close`. А якщо ви більше не потребуєте цей файл, то його можна видалити за допомогою функції `os.Remove`.
