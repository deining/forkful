## Чому

Створення тимчасового файлу є важливою складовою частиною програмування в багатьох мовах, у тому числі і в Go. Це дозволяє нам зберігати тимчасові дані, які можуть бути необхідні для подальшої обробки або використання у програмі. Також це може бути корисним при тестуванні програми або використанні зовнішніх бібліотек.

## Як це зробити

Для створення тимчасового файлу в Go, нам потрібно використати пакет "os" та функцію "CreateTemp". Перш ніж почати використовувати цю функцію, потрібно імпортувати пакет:

```Go
import "os"
```

Далі, створюємо тимчасовий файл із префіксом "test" та розширенням ".txt":

```Go
file, err := os.CreateTemp("", "test*.txt")
```

Якщо використання функції пройшло успішно, то в змінній "file" буде збережений шлях до нашого тимчасового файлу. Помилку (якщо вона є) можна перевірити за допомогою змінної "err". Далі, ми можемо взаємодіяти з цим файлом, наприклад, записувати та читати дані.

```Go
// Запис даних в файл
file.WriteString("Hello world!")

// Читання даних з файлу
data, err := file.Read()

// Виведення змісту файлу
fmt.Println(data)

// Видалення тимчасового файлу
os.Remove(file.Name())
```

## Глибокий занурення

Функція "CreateTemp" також має другий аргумент, який вказує на шлях до тимчасової директорії, де буде створений файл. При відсутності цього аргументу, тимчасовий файл буде створений встановленою системою тимчасовою директорією.

Також, за допомогою функції "CreateTemp" можна вказувати бажану префікс та суфікс для імені тимчасового файлу, наприклад:

```Go
file, err := os.CreateTemp("", "data-*.bin")
```

Більше інформації про роботу з файлами та директоріями в Go можна знайти у [офіційній документації](https://golang.org/pkg/os/).
