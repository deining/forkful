## Чому
Перетворення дати в рядок - це важлива операція в програмуванні. Саме це дає нам можливість відображати дати у зрозумілому користувачу форматі.

## Як
Програма, написана на мові Go, має у своєму арсеналі вбудовану функцію `Format`, яка дозволяє конвертувати дату в рядок з використанням заданого шаблону. Давайте подивимося на приклад:

```Go
import "fmt"
import "time"

func main() {
    date := time.Date(2021, 1, 1, 0, 0, 0, 0, time.UTC)
    
    fmt.Println(date.Format("02 January 2006"))
}
```

Вивід програми буде наступним:

`01 January 2021`

Як бачите, ми використали шаблон `02 January 2006`, що відображає дату у форматі дня, місяця та року.

## Глибокий аналіз
Давайте розглянемо деякі з шаблонів для форматування дати:

- `2006/01/02`: відображає рік, місяць та день в числовому форматі з роздільниками;
- `02 Jan 2006`: відображає день, скорочену назву місяця та рік;
- `Monday, January 02, 2006`: відображає повну назву дня тижня, місяця та рік.

Більш детальна інформація про шаблони форматування дати доступна на офіційному сайті мови Go: https://golang.org/pkg/time/#Time.Format
