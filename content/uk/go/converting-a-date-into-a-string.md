## Чому: 

Найпопулярнішим завданням, з якими стикаються програмісти, є конвертація дати в рядок. Чи є в цьому якийсь особливий сенс? Так, це необхідно для відображення дати на екрані або в файлі, а також для передачі даних між різними сервісами.

## Як: 

Для початку нам знадобиться зрозуміти, як зберігається дата у програмах на Go. У Go використовується тип даних `time.Time`, щоб представити дату і час. Для перетворення дати в рядок нам знадобиться використати функцію `Format` з пакету `time`.

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	// Задаємо дату
	date := time.Date(2020, 3, 15, 10, 30, 0, 0, time.UTC)
	// Форматуємо дату у вигляді рядка
	strDate := date.Format("02-01-2006")
	// Виводимо результат
	fmt.Println(strDate)
}
```

Результат виконання цього коду буде `15-03-2020`. У такий спосіб, змінивши форматування, ми можемо отримувати дату в різних форматах, наприклад `15.03.2020`, `15-03-20`, `March 15, 2020` тощо.

## Глибше про конвертацію дати:

Крім вже згаданої функції `Format`, в пакеті `time` є ще кілька корисних функцій для роботи з датами. Наприклад, функція `Now` дозволяє отримати поточну дату і час, а функція `Parse` виконує зворотню операцію - перетворює рядок у тип `time.Time`.

```Go
// Отримуємо поточну дату і час
now := time.Now()
// Перетворюємо дату у рядок зі зручним форматуванням
str := now.Format("15-01-2006 3:04 PM")
// Виводимо результат
fmt.Println(str)

// Перетворюємо рядок у тип дати
input := "2020-06-10"
date, _ := time.Parse("2006-01-02", input)
// Форматуємо дату у рядок з необхідним форматом
str := date.Format("02-01-2006")
// Виводимо результат
fmt.Println(str)
```

У більш складних випадках, коли необхідно працювати з різними часовими зонами, використовується пакет `time/zone`, де є функції для конвертації часових зон та розрахунку різниці між ними.
