## Чому

Програмування з часом і датами є важливою частиною розробки програмного забезпечення. У Go є потужна бібліотека для роботи з датами та часом, і часто виникає потреба перетворити дату у рядок. В цій статті ми розглянемо, як це зробити у Go.

## Як це зробити

Перш за все, нам потрібно імпортувати пакет time в нашому коді. Тоді ми можемо використовувати функцію Format, яка приймає два аргументи: шаблон та часову мітку.

```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	date := time.Now()
	stringDate := date.Format("02.01.2006")
	fmt.Println(stringDate)
}
```

У цьому прикладі ми використовуємо шаблон "02.01.2006", що представляє день, місяць та рік у цифровому форматі. Вихідний рядок буде виглядати як "00.00.0000", де перші дві цифри позначають день, наступні дві - місяць, а останні чотири - рік.

Також, у функції Format є можливість вказати обрану локаль для форматування дати та часу. Наприклад, для української локалі це буде виглядати так:

```Go
stringDate := date.Format("02.01.2006", "uk_UA")
```

## Глибока знахідка

У Go є декілька вбудованих шаблонів для форматування дати та часу, які можна використовувати у функції Format. Наприклад, "RFC3339" представляє часовий шаблон з точністю до секунди та з повною інформацією про часову зону.

Також, деякі останні версії Go додали можливість форматувати дату та час у стилі Unix, за допомогою спеціального шаблону "UnixDate". Це дозволяє отримати часову мітку у форматі Unixtimestamp, що є стандартом для багатьох систем.

У документації Go є повна інформація про всі доступні шаблони для форматування дати та часу, а також приклади використання.
