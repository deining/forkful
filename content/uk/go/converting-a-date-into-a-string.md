## Чому

Конвертація дати у рядок - це важлива техніка, яку можна використовувати для роботи з датами у програмуванні. Вона дозволяє легко і зрозуміло представляти дати для користувачів.

## Як

```Go
func main() {
    // Визначення дати
    date := time.Date(2021, 6, 15, 12, 30, 0, 0, time.UTC)

    // Конвертація у рядок
    str := date.Format("2006-01-02 15:04:05 MST")

    // Виведення рядка
    fmt.Println(str)
}

// Виведення: 2021-06-15 12:30:00 UTC
```

За допомогою функції `Format()` з пакету `time`, ми можемо змінити формат дати у рядок за допомогою спеціальних символів. Наприклад, `2006-01-02` представляє рік, місяць та день, а `15:04:05` - години, хвилини та секунди. Додавання букви "MST" дає можливість встановити часовий пояс.

## Deep Dive

Функція `Format()` також дозволяє використовувати інші символи для представлення дати у різних форматах. Наприклад, `Mon Jan 2 15:04:05 -0700 MST 2006`. Додаткову інформацію про спеціальні символи можна знайти в документації Go.
