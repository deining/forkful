## Для чого

Генерація випадкових чисел є важливою частиною багатьох програм, таких як ігри, криптографічні системи та тестування випадковими даними. Це допомагає створити непередбачуване та різноманітне поведінку в програмах, що зробить їх більш захищеними та цікавими для користувачів.

## Як

Щоб згенерувати випадкові числа у мові програмування Go, використовуйте пакет "math/rand". Почніть з імпортування пакету і створення об'єкту типу "rand.Source", що буде слугувати посібником для створення випадкових чисел. Далі, напишіть кілька рядків коду, які використовують функцію "rand.Intn" для отримання випадкового числа з вказаного діапазону. Нижче наведений приклад коду та виходу програми:

```Go
package main

import (
	"fmt"
	"math/rand"
)

func main() {
	source := rand.NewSource(time.Now().UnixNano())
	random := rand.New(source)
	output := random.Intn(100)
	fmt.Println(output)
}
```

Вихід: 56

## Глибоке занурення

В мові програмування Go існує два різні підходи до генерації випадкових чисел: використання генератора псевдовипадкових чисел та використання криптографічної генерації випадкових чисел. Перший підхід використовує функції, які створюють послідовність чисел, що здаються випадковими, але насправді є регулярними та можуть бути передбаченими. Другий підхід використовує криптографічні алгоритми для створення випадкових чисел, що забезпечує вищий рівень непередбачуваності та безпеки.
