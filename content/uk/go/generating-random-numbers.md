## Для чого?

Згенерувати випадкові числа може бути корисною у багатьох ситуаціях, наприклад, під час тестування програм або створення випадкових паролів. Основним поглядом цієї статті буде створення випадкових чисел у Go.

## Як це зробити?

Для початку, потрібно імпортувати пакет `math/rand`, а потім використовувати функцію `Intn()` для отримання випадкового цілого числа з межею, яку потрібно встановити явно.

```Go
import "math/rand"

rand.Intn(100) // згенерує випадкове ціле число від 0 до 99
```

Можна також згенерувати випадкове число з плаваючою точкою у діапазоні від 0 до 1 використовуючи функцію `Float64()`.

```Go
rand.Float64() // згенерує випадкове число з плаваючою точкою від 0 до 1
```

Якщо потрібно згенерувати більш складні випадкові числа, наприклад, у встановленому діапазоні, можна використовувати функцію `Rand()` з потрібним генератором випадкових чисел.

```Go
source := rand.NewSource(time.Now().UnixNano()) // створюємо генератор випадкових чисел з основою від дати і часу
random := rand.New(source) // створюємо новий випадковий генератор

random.Intn(100) // згенерує випадкове ціле число від 0 до 99
```

## Глибоке погруження

Щоб зрозуміти, як саме відбувається генерація випадкових чисел, потрібно звернутися до Алгоритму Лемера. Цей алгоритм використовує константу `a`, початкове значення `x` і максимальне значення `m` для генерації випадкових чисел.

Алгоритм Лемера виглядає наступним чином:

`x = a*x (mod m)`

У Go, цей алгоритм застосовується у функції `Intn()` із зафіксованими значеннями `a` (береться за змовчуванням) і `m` (позначається як `1<<31-1`).
