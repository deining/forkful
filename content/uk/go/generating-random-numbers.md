# Чому

Генерування випадкових чисел - це важлива функція в багатьох програмах. Вона може бути використана для шифрування даних, створення випадкових паролів або для випадкового вибору елементів у грі. Також це може бути корисно для тестування програм, щоб побачити як вони поводяться у різних сценаріях.

# Як це зробити

Існує кілька способів генерування випадкових чисел в мові програмування Go. Найпростішим способом є використання пакету "math/rand". Ось приклад коду, який генерує випадкове число від 0 до 100:

```Go
package main

import (
	"fmt"
	"math/rand"
)

func main() {
	fmt.Println(rand.Intn(100))
}
```

Виходом цього коду буде випадкове число, наприклад 72.

# Глибоке погруження

Створення випадкових чисел - це складний процес, який включає в себе використання алгоритмів генерування псевдовипадкових чисел і використання внутрішніх генераторів таких як генератор Мерсенна Твіста. Детальніше про це можна дізнатись в офіційній документації мови Go.
