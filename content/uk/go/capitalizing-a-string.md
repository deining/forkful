## Чому

Правильне написання тексту є важливою частиною програмування, особливо коли приходиться працювати з користувачем. Українські читачі цього блогу ймовірно знають, як важливо, щоб текст був подано якісно і зрозуміло. Тому, сьогодні ми розглянемо, як можна зробити текст більш зрозумілим, використовуючи Go.

## Як це зробити

Для початку нам потрібно створити змінну з текстом, який ми хочемо змінити:

```Go
text := "прИкЛАД ТекстУ"
```

Наступним кроком буде використання функції `strings.Title()`, яка першу літеру кожного слова рядка робить великою, а всі інші - маленькими. Для цього необхідно імпортувати пакет `strings`:

```Go
import "strings"

output := strings.Title(text)
fmt.Println(output)

// Output: Пріклад Тексту
```

Як бачимо, за допомогою цієї функції ми легко виконали першу бажану задачу - зробили першу літеру кожного слова рядка великою.

## Глибока поглибка

Також варто зазначити, що функція `strings.Title()` має багато різних варіацій, залежно від нашої потреби. Наприклад, вона також може ігнорувати певні слова, які не потрібно змінювати, щоб не порушити правильне написання. Для цього можна використовувати другий аргумент - `stopWords`:

```Go
stopWords := make(map[string]bool)
stopWords["наприклад"] = true
output := strings.Title(text, stopWords)
fmt.Println(output)

// Output: наприклад Тексту
```

Це дозволяє нам зберегти слово "наприклад" без змін, а префікс "не" перед словом "приклад" змінений на велику літеру.
