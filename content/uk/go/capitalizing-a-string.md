Що і чому?

Капіталізація рядка - це процес, коли перша літера кожного слова у рядку стає великою. Це зроблено для полегшення читання та зрозуміння рядка для людей. Програмісти роблять це для досягнення кращої читабельності та стандартизації коду.

Як це зробити?

```
Go strings.Title("Hello, world!")
```
Результат:
```
"Hello, World!"
```
```
Go strings.ToTitle("Привіт, світ!")
```
Результат:
```
"Привіт, Світ!"
```

Глибше:

Капіталізація рядка має свої коріння у друкованій пресі та типографії. Використання цього правила полегшувало читання і розуміння тексту для людей. Існують інші методи капіталізації, такі як використання заголовків або зміни кольору шрифту. У Go, для капіталізації рядка також можна використовувати функції ```strings.ToUpper``` та ```strings.TrimSpace```.
