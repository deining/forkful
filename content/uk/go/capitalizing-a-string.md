## З чого почати

Програмування в Go є однією з найпопулярніших мов для розробки сучасних додатків та веб-сайтів. У цій статті ми розглянемо, як можна використовувати мову Go для роботи з текстом та конкретно - з капіталізації рядків. Ви дізнаєтесь, чому це корисно і як точно це зробити.

## Як зробити капіталізацію рядків в Go

Для початку, для тих, хто ще не має досвіду з програмуванням в Go, ми рекомендуємо ознайомитись з документацією та основними концепціями мови. А тепер перейдемо до самої капіталізації.

Стандартна бібліотека мови Go має вбудовану функцію `strings.ToUpper()`, яка приймає рядок і повертає його капіталізовану версію. Ось приклад коду:

```Go
package main

import (
    "fmt"
    "strings"
)

func main() {
    str := "привіт, україно"
    fmt.Println(strings.ToUpper(str))
}
```

Результатом виконання цього коду буде `ПРИВІТ, УКРАЇНО`. Зверніть увагу, що кирилиця також залишається вонамальовуванням.

## Глибоке вивчення капіталізації рядків в Go

Хоча вбудована функція `strings.ToUpper()` виконує достатньо просту роботу, ви можете знайти більше опцій для капіталізації рядків у сторонній бібліотеці `github.com/fatih/camelcase`. Ця бібліотека має багато корисних функцій, які допоможуть вам з управлінням текстом. 

Наприклад, вона має функції для переведення рядка до camelCase і snake_case, як і за європейській вимові (наприклад, Střih vlasů стане "StribVlasu"). Зверніть увагу, що ця бібліотека вимагає встановлення через `go get`, якщо ви хочете скористатися її функціями.
