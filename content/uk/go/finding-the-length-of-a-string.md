## Чому

Треба знати довжину рядка в програмуванні для здійснення різних операцій, таких як обробка тексту, валідація введених даних або встановлення обмежень.

## Як То

```Go
// Приклад коду для знаходження довжини рядка "Hello World"
str := "Hello World"
length := len(str)
fmt.Println(length) // Виведе 11
```

У цьому прикладі ми використовуємо вбудовану функцію `len()`, яка повертає довжину рядка у вигляді цілого числа. Для отримання довжини ми просто запускаємо функцію і передаємо їй рядок, про який хочемо взнати довжину.

Існує декілька кейсів, коли потрібно знати довжину не лише рядка, але й інших типів даних, наприклад, слайсів або мап. Щоб отримати довжину слайса, можна використовувати `len()` в тому ж самому форматі, що й для рядка. А для мап повертається кількість ключ-значення пар у мапі.

```Go
// Приклад отримання довжини слайсу та мапи
slice := []string {"apple", "banana", "orange"}
sliceLength := len(slice)
fmt.Println(sliceLength) // Виведе 3

m := map[string]int {"apple": 5, "banana": 3, "orange": 7}
mapLength := len(m)
fmt.Println(mapLength) // Виведе 3
```

## Глибша Заглиблення

До цього моменту ми розглядали лише вбудовану функцію `len()`, але у Go можна використовувати й інші способи для визначення довжини рядків. Наприклад, існує можливість перевірити кількість символів у рядку за допомогою пакету `unicode/utf8`. Це особливо корисно, якщо ми маємо справу з мультибайтовими символами, такими як кирилиця або китайські ієрогліфи. Для цього потрібно буде імпортувати пакет та використовувати функцію `utf8.RuneCountInString()`.

```Go
// Імпортуємо пакет та використовуємо функцію для підрахунку кількості символів у рядку
import "unicode/utf8"

str2 := "Привіт Світ"
characterCount := utf8.RuneCountInString(str2)
fmt.Println(characterCount) // Виведе 11, оскільки українські літери займають 2 байти кожна
```
