## Що & Чому?

Порівняння двох дат - це процес, у якому програмісти порівнюють дати між собою. Це може бути корисно для визначення порядку дат або перевірки, чи одна дата раніша чи пізніша за іншу. Програмісти використовують цей процес для розв'язання різних завдань у своїх програмах.

## Як це зробити?

```Go
// створюємо дату в форматі рррр-мм-дд
date1 := time.Date(2020, 5, 15, 0, 0, 0, 0, time.UTC)
// створюємо другу дату
date2 := time.Date(2019, 5, 15, 0, 0, 0, 0, time.UTC)

// порівнюємо дати
if date1.Equal(date2) {
    fmt.Println("Дати однакові")
} else if date1.After(date2) {
    fmt.Println("Перша дата пізніше за другу")
} else if date1.Before(date2) {
    fmt.Println("Перша дата раніше за другу")
}
```

Вивід: ```Перша дата пізніше за другу```

## Глибше розуміння

У світі програмування існують різні методи порівняння дат, але в Go для цього використовується метод ```Equal()```, який порівнює з точністю до мілісекунд. Також є методи ```After()``` і ```Before()```, які розраховують порядок дат з точністю до мікросекунд. Це дозволяє програмістам використовувати різні діапазони дат для розв'язання своїх завдань.
