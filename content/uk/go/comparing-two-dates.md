## Чому

Порівняння двох дат є необхідним у багатьох програмах, особливо тих, що працюють зі створенням та управлінням записів про події або транзакції. У програмуванні мовою Go, процес порівняння дат може бути дещо складніше, але все ж це можливо зробити з використанням правильних методів та функцій.

## Як це зробити

Для початку, нам потрібно створити два об'єкти типу `time.Time` і присвоїти їм значення дат, які ми хочемо порівняти. Для цього ми можемо використати функцію `time.Parse`, яка приймає у форматі строкового значення та повертає об'єкт типу `time.Time`.

```Go
date1, _ := time.Parse("2006-01-02", "2020-03-15")
date2, _ := time.Parse("2006-01-02", "2020-03-20")
```

Зверніть увагу, що перший параметр у функції `time.Parse` вказує на опис формату дати, як вона вказана у другому параметрі.

Тепер, ми можемо використати метод `Before` або `After`, які перевіряють, чи одна дата розташована перед або після іншої.

```Go
if date1.Before(date2) {
    fmt.Println("Дата 1 перед датою 2")
}
if date2.After(date1) {
    fmt.Println("Дата 2 після дати 1")
}
```

Так само, існують методи `Equal` і `Before`, які дозволяють порівняти дати на рівність та чи одна дата є раніше за іншу.

```Go
if date1.Equal(date2) {
    fmt.Println("Обидві дати рівні")
}
if date1.Before(date2) {
    fmt.Println("Дата 1 раніше за дату 2")
}
if date2.Before(date1) {
    fmt.Println("Дата 2 раніше за дату 1")
}
```

## Глибша погляд

Є ще кілька методів, які можуть бути корисними при порівнянні дат в Go. Наприклад, метод `Sub` дозволяє вираховувати різницю у часі між двома датами.

```Go
difference := date1.Sub(date2)
fmt.Println(difference.Hours()) // виведе кількість годин різниці
```

Також, можна використати методи `Add` і `Sub`, щоб додати або відняти певну кількість часу до або від дати.
