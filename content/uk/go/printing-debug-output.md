## Почему

Існує багато причин, чому ви могли б знадобитися вивід відлагодження під час програмування на Go. Наприклад, ви можете хотіти відстежувати значення змінних або перевіряти правильність виконання програми. 

## Як

Для виводу відлагодження в Go використовується функція `fmt.Println ()`. Для цього потрібно імпортувати пакет `fmt`. Наступним кроком є виклик функції змінної або значень, які ви хочете вивести. Нижче наведено приклад:

```Go
package main

import "fmt"

func main() {
    name := "John"
    age := 30
    
    fmt.Println("Name:", name)
    fmt.Println("Age:", age)
}
```

В результаті ви отримаєте вивід наступного вигляду:

```
Name: John
Age: 30
```

## Глибше розкладання

Більш детально про вивід відлагодження в Go. Функція `fmt.Println ()` приймає потокові параметри, що означає, що ви можете використовувати форматування рядків, як, наприклад, `%d` для цілих чисел та `%s` для рядків. Нижче наведено приклад з використанням форматування рядків:

```Go
package main

import "fmt"

func main() {
    name := "John"
    age := 30
    
    fmt.Printf("Name: %s\n", name)
    fmt.Printf("Age: %d", age)
}
```

Результат буде точно такий же, як і в попередньому прикладі. Однак, користувач може викличати функцію `fmt.Print ()`, яка не додає перенесення рядка в кінці виведення. Також існує функція `fmt.Sprintf ()`, яка повертає рядок замість виведення його на екран.
