## Чому

Розбір HTML-коду є важливою складовою частиною веб-розробки, оскільки він дозволяє отримати доступ до цілого ряду даних на веб-сторінках. Парсинг HTML є необхідним для створення скраперів, аналізу веб-сторінок та взаємодії зі сторонніми додатками.

## Як це зробити

Для парсингу HTML з використанням мови програмування Go, вам знадобляться наступні кроки:

1. Встановіть необхідні бібліотеки. Найпопулярнішою бібліотекою для парсингу HTML є "goquery", яку можна встановити з використанням команди `go get`.

2. Імпортуйте бібліотеку у свій проект.

3. Визначте вхідний URL, з якого буде проводитись парсинг даних.

4. Створіть новий об'єкт за допомогою функції `NewDocumentFromUrl`, передавши в неї вхідний URL.

5. Використовуйте методи бібліотеки для отримання необхідних даних зі сторінки. Наприклад, метод `Find` дозволяє знайти елементи за їх CSS-селекторами.

6. Обробіть отримані дані та виведіть результат.

Нижче наведено приклад реалізації парсингу HTML з використанням бібліотеки "goquery" та виведенням отриманої інформації.

```Go
package main

import (
    "fmt"
    "log"
    "net/http"

    "github.com/PuerkitoBio/goquery"
)

func main() {
    // Визначення URL
    url := "https://example.com"

    // Створення нового об'єкта goquery.Document
    doc, err := goquery.NewDocument(url)

    if err != nil {
        log.Fatal(err)
    }

    // Знаходження всіх елементів <a> з class="link"
    doc.Find("a.link").Each(func(i int, s *goquery.Selection) {
        // Виведення тексту елементу та його посилання
        linkText := s.Text()
        link, _ := s.Attr("href")
        fmt.Printf("[%d] %s - %s\n", i+1, linkText, link)
    })
}
```

В результаті виконання коду, ви отримаєте виведення подібне до такого:

```
[1] Link 1 - https://example.com/link-1
[2] Link 2 - https://example.com/link-2
[3] Link 3 - https://example.com/link-3
...
```
