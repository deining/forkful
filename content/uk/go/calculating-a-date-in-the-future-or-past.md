## Чому

Калькулювання дати в майбутньому або минулому може бути корисним для створення програм, які працюють зі шляхами файлів, планування подій або навіть для виконання тестів. Також це може бути корисно для зберігання та відстеження історії змін даних.

## Як виконати

Існує декілька способів виконати калькуляцію дати в Go. Один з них - використання пакету `time`, який має вбудовані функції для роботи з часом. Наприклад:

```Go
// Визначення поточної дати та часу
now := time.Now()

// Калькуляція дати в майбутньому (1 день після поточної дати)
future := now.AddDate(0, 0, 1)
fmt.Println(future) // 2021-10-15 20:00:00 +0000 UTC

// Калькуляція дати в минулому (1 день до поточної дати)
past := now.AddDate(0, 0, -1)
fmt.Println(past) // 2021-10-13 20:00:00 +0000 UTC
```

Іншим способом є використання бібліотеки `dateparse`, яка дозволяє парсити різні формати дати та часу. Наприклад:

```Go
// Визначення поточної дати та часу
now := time.Now()

// Розпарсити дату зі стрічки (формати 2021-10-14, 2021/10/14, 14.10.2021)
parsedDate, _ := dateparse.ParseAny("2021-10-14")
fmt.Println(parsedDate) // 2021-10-14 20:00:00 +0000 UTC

// Калькуляція дати в майбутньому (1 день після поточної дати)
future := now.AddDate(0, 0, 1)
fmt.Println(future) // 2021-10-15 20:00:00 +0000 UTC

// Калькуляція дати в минулому (1 день до поточної дати)
past := now.AddDate(0, 0, -1)
fmt.Println(past) // 2021-10-13 20:00:00 +0000 UTC
```
