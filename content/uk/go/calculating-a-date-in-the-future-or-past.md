## Чому

Програмування може бути захоплюючим і цікавим заняттям для людей, які люблять логічне мислення та вирішення складних задач. Розрахунок дати в майбутньому або в минулому може бути простим завданням, але водночас воно має важливе застосування в реальному житті. Це може бути корисно для планування подій, розрахунку термінів або просто для відшукання відповіді на питання "Який день буде через 100 днів?" Також, це може бути цікавою вправою для вивчення різних методів обробки дат в мові програмування Go.

## Як

Ми можемо використати вбудовані функції в Go для роботи з датами. Наприклад, функція "time.Now()" дозволяє отримати поточну дату та час, а функція "time.AddDate()" дає можливість додавати або віднімати дні, місяці або роки до вказаної дати. Для прикладу, нижче ми покажемо, як розрахувати дату через 100 днів в майбутньому, використовуючи функції time.Now() та time.AddDate():

```Go
currentTime := time.Now() // отримуємо поточну дату
futureDate := currentTime.AddDate(0, 0, 100) // додаємо 100 днів до поточної дати
fmt.Println(futureDate) // виводимо отриману дату у форматі "рік-місяць-день година:хвилина:секунда"
```

Результатом виконання цього коду буде дата через 100 днів від поточного моменту.

## Deep Dive

У загальному, обробка дати в Go є простою та зручною завдяки використанню вбудованих функцій. Проте, для більш складних завдань, може бути корисно ознайомитись з пакетом time в Go, який містить детальну документацію та багато корисних функцій для роботи з датами та часом. Наприклад, функція "time.Parse()" дозволяє перетворити рядок з датою у об'єкт типу time.Time, що дозволяє більш гнучко обробляти дати з різноманітними форматами.
