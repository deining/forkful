## Чому

За допомогою програмування можна легко розрахувати дату в майбутньому або минулому, що зробить життя більш організованим та планованим.

## Як це зробити

Використовуючи мову програмування Go, ми можемо легко виконати розрахунок дати в майбутньому або минулому за допомогою нашої програми. Для цього скопіюйте наступний код в свій редактор коду:

```Go
package main

import (
	"fmt"
	"time"
)

func main() {

	// Розрахунок дати в майбутньому
	futureDate := time.Now().AddDate(0, 0, 7)
	fmt.Println("Дата через тиждень буде:", futureDate)

	// Розрахунок дати в минулому
	pastDate := time.Now().AddDate(0, -1, 0)
	fmt.Println("Дата минулого місяця була:", pastDate)
}
```

В результаті ви отримаєте вивід наступного вигляду:

```Go
Дата через тиждень буде: 2021-05-13 17:22:43.6576888 +0300 EEST m=+604800.539194801
Дата минулого місяця була: 2021-03-15 17:22:43.6576888 +0200 EET m=+2678399.5391948
```

## Глибоке погруження

Щоб розрахувати дату в майбутньому, ми використовуємо функцію `.AddDate()` і передаємо в неї необхідну кількість років (перший аргумент), місяців (другий аргумент) та днів (третій аргумент). Аналогічно, для розрахунку дати в минулому, ми передаємо від'ємне значення для місяців або років. Більш детальну інформацію про цю функцію можна знайти в [документації Go](https://golang.org/pkg/time/#Time.AddDate).
