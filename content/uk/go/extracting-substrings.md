## З чого почати

В розробці програмного забезпечення іноді необхідно витягнути певну частину тексту зі строкового значення. Наприклад, можна витягнути номер телефону зі строкового значення або перевірити наявність певного слова в тексті. У цьому пості ми розглянемо, як витягати підрядки в мові програмування Go.

## Як це зробити

Існує кілька способів витягти підрядки в мові програмування Go. Давайте розглянемо один з них за допомогою кодового блоку ```Go ... ```.

```Go
package main

import "fmt"

func main() {
    s := "Це рядок для прикладу"
    fmt.Println(s[4:9])
}
```

В цьому прикладі ми створили змінну `s`, яка містить строкове значення. За допомогою виразу `s[4:9]` ми можемо витягнути підрядок зі строкового значення, починаючи з четвертого і закінчуючи дев'ятим символами. У цьому випадку виведено на екран слово "рядок". Можна також використовувати вираз `s[:5]`, щоб витягнути підрядок з першого символу до п'ятого, або `s[10:]`, щоб витягнути підрядок з десятого символу до кінця строки.

Існують також інші методи для витягування підрядків в мові програмування Go, такі як `strings.Index()`, `strings.LastIndex()` і `strings.Contains()`. Наприклад:

```Go
package main

import "fmt"
import "strings"

func main() {
    s := "Це рядок для прикладу"
    fmt.Println(strings.Index(s, "для"))
}
```

У цьому прикладі ми використовуємо функцію `strings.Index()` для пошуку індексу першого входження слова "для" у строковому значенні `s`. Результатом буде число 13, що означає, що слово "для" починається з тринадцятого символу.

## Глибоке занурення

Крім вищенаведених методів, в мові програмування Go є ще багато інших функцій та пакетів для роботи з підрядками. Наприклад, пакет `regexp` дозволяє виконувати регулярні вирази для пошуку підрядків у строковому значенні.

Також важливо пам'ятати, що символи в мові програмування Go кодуються за допомогою UTF-8, тому деякі підрядки можуть складатися з більш ніж одного байту. Це потрібно враховувати при витягуванні підрядків.
