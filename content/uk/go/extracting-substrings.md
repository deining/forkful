## Чому

Вилучення підрядків є важливою функцією в програмуванні, оскільки воно дозволяє отримати підрядки з рядків для подальшої обробки. Це особливо корисно, якщо вам потрібно розбити великий рядок на більш маленькі частини або виконати пошук і заміну певних символів.

## Як використовувати

В Go існує кілька способів вилучення підрядків з рядків. Один з них - використовувати метод `Substring()` для стандартного пакета `strings`. Розглянемо приклад:

```Go
package main

import "fmt"
import "strings"

func main() {
  str := "Привіт, світ!"
  sub := strings.Substring(str, 7, 12)
  fmt.Println(sub)
}
```

Результатом цього коду буде `світ!`. Тут ми застосовуємо метод `Substring()`, який приймає три аргументи - рядок, початковий і кінцевий індекси для вилучення підрядка. Також ви можете використовувати знак `-` для позначення останнього символу рядка. Наприклад, `substr := strings.Substring(str, 7, -1)` поверне `світ!`.

Також ви можете використовувати функцію `Slice()` для вилучення підрядків, використовуючи індекси зрізів. Розглянемо приклад:

```Go
package main

import "fmt"

func main() {
  str := "Привіт, світ!"
  sub := str[7:12]
  fmt.Println(sub)
}
```

Результатом буде той самий підрядок `світ!`. Тут ми застосовуємо індекси зрізів, щоб вилучити частину рядка з позицій 7 до 12.

Незалежно від того, який метод ви використовуєте, важливо пам'ятати, що позиції символів починаються з 0, а не з 1.

## Глибше поглянемо

Існує багато інших методів і функцій для вилучення підрядків в Go, які можуть бути корисними при роботі з рядками. Наприклад, функція `Trim()` дозволяє видалити пробільні символи в початку і в кінці рядка, а метод `Replace()` дозволяє замінити певні символи на інші. Також ви можете складати рядки за допомогою методу `Join()`, який поверне один рядок з розділювачем між елементами.
