## Чому

Часто під час роботи зі строками в програмуванні, може виникнути необхідність видалити певні символи, які відповідають певному шаблону. Це може бути потрібно для того, щоб позбутися зайвої інформації або відфільтрувати список даних за певною умовою. У цьому блозі ми розглянемо, як написати програму на мові Go, щоб видалити символи, які відповідають певному шаблону.

## Як

Найпростіший спосіб видалити символи, що відповідають певному шаблону, це використання методу ```strings.Replace()```. Цей метод приймає вхідну строку, патерн символів, які потрібно видалити, та новий патерн, який замінить старий. Давайте подивимося на приклад:

```Go
package main
 
import (
 	"fmt"
 	"strings"
)
 
func main() {
 	str := "HelloGoWorld"
 	str = strings.Replace(str, "Go", "", -1)
 	fmt.Println(str)
}

// Output: HelloWorld
```

У цьому прикладі, ми використовуємо метод ```strings.Replace()``` для видалення фрагмента "Go" з нашої строки. Зверніть увагу, що ми передали -1 останнім аргументом, що означає, що патерн буде видалений з усіх входжень у строку. Якщо ви хочете видалити тільки перше входження, тоді використовуйте значення 1.

Однак, цей метод не враховує регістр символів. Якщо вам потрібно видалити символи з врахуванням регістру, використовуйте метод ```strings.ReplaceAll()```.

```Go
package main

import (
 	"fmt"
 	"strings"
)

func main() {
 	str := "Happy birthday, John!"
 	str = strings.ReplaceAll(str, "h", "")
 	fmt.Println(str)
}

// Output: Happy birtday, John!
```

Цей метод також приймає питання як останній аргумент, вказуючи кількість входжень, які треба замінити. Якщо потрібно видалити всі входження, тоді використовуйте -1 або не вказуйте останній аргумент взагалі.
