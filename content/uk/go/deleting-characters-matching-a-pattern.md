## Чому

Вилучення символів, які відповідають заданому шаблону, може бути корисною функцією для роботи з текстовими даними, якщо ви хочете видалити певну частину тексту, яка відповідає певній умові.

## Як це зробити

```Go
// Приклад коду для вилучення символів "abc" з рядку "a1b2c3"
import "strings"

str := "a1b2c3"
newStr := strings.ReplaceAll(str, "abc", "")
fmt.Println(newStr) // Результат: "123"

```

```Go
// Інший приклад, використовуючи регулярні вирази, для видалення символів "a-z" з рядку "123abc456"
import "regexp"

str := "123abc456"
re := regexp.MustCompile("[a-z]")
newStr := re.ReplaceAllString(str, "")
fmt.Println(newStr) // Результат: "123456"
```

## Розглянути детальніше

Видалення символів за заданим шаблоном може бути корисною функцією при роботі зі складними текстовими даними, що містять даний шаблон. Вона також може бути використана для побудови більш складних функцій для маніпулювання рядками. Використання регулярних виразів дозволяє більш гнучко налаштовувати шаблон для вилучення символів, наприклад, ви можете використовувати спеціальні символи для пошуку кінця рядка, цифр або букв.
