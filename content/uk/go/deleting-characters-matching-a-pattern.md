## Для чого
Якщо ви використовуєте мову програмування Go, ви, напевно, знаєте про можливість видалення символів, що відповідають певному шаблону. Але для чого це може бути корисно і як це реалізувати? Давайте дізнаємося більше про цю функцію.

## Як це зробити
Для початку, давайте подивимося на приклад коду, який допоможе нам видалити усі цифри з рядка:

```Go
str := "hello123"
output := strings.Map(func(r rune) rune {
    if !unicode.IsDigit(r) {
        return r
    }
    return -1
}, str)
fmt.Println(output)
```
Цей код використовує функцію `strings.Map`, яка проходить по кожному символу у рядку і замінює його на `rune`, що повертається з функції. Якщо це не цифра, то повертається оригінальний символ, а в іншому випадку - `-1`. Таким чином, у нашому прикладі, всі цифри будуть замінені на `-1`.

Результат роботи цього коду буде: `hello`.

Ви можете змінити умову функції `if` для того, щоб відповідати різним шаблонам: буквам, прогалинам, знакам пунктуації тощо. Також, ви можете використовувати цю функцію для видалення кількох символів одночасно.

## Вглиб
Функція `strings.Map` використовує руни, оскільки вони є єдиним типом символів у Go, що відповідає усім потрібним кодуванням. Рунів використовуються для представлення символів Unicode, щоб забезпечити максимальну гнучкість та підтримку мов і писменностей.

Також, можна використати функцію `strings.Replace` для видалення символів. Вона приймає 3 аргументи: вхідний рядок, символи, які потрібно замінити, і новий символ, яким вони мають бути замінені. Ця функція може бути корисна, якщо ви хочете видалити конкретний символ з рядка, а не за певним шаблоном.
