## Чому

Нерідко у розробці програмного забезпечення необхідно прочитати цілу купу текстових файлів. Прикладом може бути імпортування даних з зовнішнього джерела або пошук інформації в конфігураційних файлах. Читання текстових файлів є важливою вмінням для розробника, тому ми описуємо процес цього процесу у цій статті.

## Як

Щоб прочитати текстовий файл у програмі Go, найпростіший спосіб - використати функцію `ioutil.ReadFile()`. Нижче ми покажемо приклад, як це можна зробити:

```Go
file, err := ioutil.ReadFile("example.txt")
if err != nil {
    fmt.Print("File reading error", err)
    return
}
fmt.Println(string(file))
```

У цьому прикладі ми спочатку отримуємо доступ до файлу `example.txt` за допомогою `ioutil.ReadFile()`, а потім виводимо його зміст в консоль за допомогою `fmt.Println()`. Якщо все пройшло успішно, ми повинні побачити вміст файлу у консолі.

## Глибші дослідження

У процесі читання текстового файлу, можливо, виникнуть питання про юнікод або обробку символів з помилками. У таких випадках, ви можете скористатися пакетом `bufio` або `encoding` залежно від вашої потреби. Пакет `bufio` дозволяє зчитувати файл по рядках, а пакет `encoding` дозволяє писати байти в зручний формат.
