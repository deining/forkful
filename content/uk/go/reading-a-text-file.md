## Що і чому?
Читання текстового файлу - це процес отримання інформації з файлу, який містить текстові дані. Програмісти часто використовують цей метод для отримання важливої інформації з файлів для подальшої обробки та використання її в програмах.

## Як це зробити:
```Go
package main

import "fmt"
import "io/ioutil"

func main() {
	data, err := ioutil.ReadFile("file.txt")
	if err != nil {
		fmt.Println("Помилка читання файлу:", err)
		return
	}
	fmt.Print(string(data))
}
```
В цьому прикладі ми використовуємо функцію `ReadFile` з пакету `io/ioutil` для отримання даних з файлу `file.txt`. Якщо при читанні файлу виникає помилка, то ми виводимо її на екран. Інакше, ми виводимо дані з файлу за допомогою функції `string`.

## Глибші деталі:
Історичний контекст: читання текстових файлів - це традиційний та широко використовуваний метод отримання інформації з диска. Існують альтернативи, такі як читання бінарних файлів або використання баз даних, але читання текстових файлів зазвичай є простим та ефективним способом отримати дані.

Деталі реалізації: у прикладі вище ми використовуємо функцію `ReadFile`, яка повертає масив байтів з вмістом файлу. Для отримання тексту з цього масиву використовується функція `string`. Також слід звернути увагу на використання структури `err` для перевірки помилок при читанні файлу.
