##Чому
Якщо ви шукаєте спосіб читати інформацію з текстового файлу за допомогою Go, цей стаття для вас. Читання текстового файлу може допомогти вам зрозуміти інформацію, яка зберігається у форматі тексту, і дозволить здійснювати подальшу обробку цих даних.

##Як
Для того, щоб прочитати текстовий файл у Go, спочатку потрібно відкрити його за допомогою функції `Open()` з пакету `os`. Потім, використовуючи буфер для зберігання даних, можна зчитувати інформацію з файлу використовуючи метод `Read()`. Нижче наведено приклад коду та відповідного виводу:

```Go
file, err := os.Open("textfile.txt")

if err != nil {
    panic(err)
}

// Створюємо буфер для зберігання даних
buf := make([]byte, 1024) 

// Читаємо дані з файлу у буфер
n, err := file.Read(buf)

// Виводимо отриману інформацію
// Якщо під час зчитування виникли помилки, вони будуть відображені тут
fmt.Printf("%s", string(buf[:n])) 
```

В результаті ви повинні отримати вивід, що відображає вміст вашого текстового файлу.

##Глибокий розгляд
Існує кілька різних способів читання текстового файлу у Go, також потрібно враховувати різні розділові знаки між рядками та байтами. Для подальшої обробки даних можна використовувати пакети `strings` та `bufio`. Також, корисно знати різні методи роботи з файлами у Go, такі як `Write()` та `Create()`.
