## Чому

Причини, чому люди читають текстові файли, можуть бути різними. Наприклад, можливо вони хочуть отримати інформацію про велику кількість даних, що зберігаються у текстовому форматі. Або може це просто улюблений спосіб збереження та надійного зберігання даних. Незалежно від причини, у програмуванні навчатися читати та обробляти текстові файли досить корисно.

## Як це зробити

Існує кілька шляхів читання текстових файлів у Go. Розглянемо найпростіший з них: використання функції `ReadFile` з пакету `ioutil`.

```Go
package main

import (
    "fmt"
    "io/ioutil"
)

func main() {
    // Читаємо текстовий файл "test.txt" та зберігаємо його в змінну data
    data, err := ioutil.ReadFile("test.txt")
    if err != nil {
        // Якщо виникла помилка, виводимо її на екран
        fmt.Println("Помилка читання файлу:", err)
        return
    }
    // Відображаємо зміст файлу на екрані
    fmt.Println(string(data))
}
```

Вищенаведений приклад дозволяє прочитати вміст текстового файлу та вивести його на екран. У разі, якщо виникне помилка під час читання файлу, програма виведе повідомлення про це на екрані.

У Go також існує можливість читання текстових файлів по рядках з використанням більш розширеної функції `ReadString` пакету `bufio`. Цей підхід дозволяє зчитувати та обробляти дані по одному рядку за раз, що може бути корисним у випадку великих текстових файлів.

    "os"
