## Чому

Видрізання підрядків є важливою технікою в програмуванні Clojure, оскільки дозволяє легко отримувати та обробляти конкретні частини рядків даних.

## Як

Для видрізання підрядків у Clojure використовуються функції `subs` та `substring`. Розглянемо декілька прикладів:

```Clojure
; Видрізання підрядка починаючи з позиції 3 до кінця рядка
(subs "Clojure є дуже потужною мовою програмування" 3)
; => "jure є дуже потужною мовою програмування"

; Видрізання підрядка починаючи з позиції 6 до 15 символу
(substring "Перші вступні слова" 6 15)
; => "вступні с"

; Видрізання підрядка починаючи з позиції 0 до 10 символу включно
(substring "0123456789" 0 (+ 10 1))
; => "0123456789"
```

Ще однією корисною функцією є `split`, яка дозволяє розділити рядок на підрядки за заданим роздільником:

```Clojure
; Розділення рядка на підрядки за пробілом
(split "Розділи цей рядок на окремі слова" #"\s+")
; => ["Розділи" "цей" "рядок" "на" "окремі" "слова"]

; Розділення рядка на підрядки за комою
(split "Обчисли ці числа, 1, 2, 3, 4" #",")
; => ["Обчисли ці числа" " 1" " 2" " 3" " 4"]
```

## Глибше дослідження

Для роботи з підрядками також можна використовувати функції `replace` та `replace-first`, які дозволяють замінювати певну підрядок у рядку на новий. Наприклад:

```Clojure
; Заміна слова "дуже" на "надзвичайно" у рядку
(replace "Ця мова дуже цікава" "дуже" "надзвичайно")
; => "Ця мова надзвичайно цікава"

; Заміна першої літери у слові на велику
(replace-first "привіт!" #"\w" #(str (clojure.string/upper-case %)))
; => "Привіт!"
```
