## Чому

У світі Clojure існує безліч корисних функцій, одна з яких - виокремлення підрядків. Ця функція може бути корисною в багатьох випадках, включаючи обробку тексту, роботу з рядками та створення власних алгоритмів.

## Як використовувати

Для виокремлення підрядків у Clojure використовується функція `subs`. Ця функція приймає два аргументи: рядок, з якого ми хочемо витягнути підрядок, та два індекси, які вказують початок та кінець підрядка (індекси починаються з 0). Приклад:

```Clojure
(subs "Привіт, світе" 0 5)
```

Вищенаведений код витягне перші п'ять символів із рядка "Привіт, світе" та поверне рядок "Привіт".

Щоб витягнути підрядок від певного індексу до кінця рядка, можна використовувати функцію `subs` з одним аргументом:

```Clojure
(subs "Привіт, світе" 7)
```

У цьому випадку буде витягнутий підрядок від сьомого символу до кінця рядка, тобто "світе".

## Глибші дослідження

Окрім базового використання, функція `subs` також підтримує витягнення підрядків за допомогою від'ємних індексів. Це означає, що можна витягти підрядок з кінця рядка, використовуючи від'ємні значення для індексів. Наприклад, `-1` буде означати останній символ рядка, `-2` - передостанній, і так далі.

```Clojure
(subs "Привіт, світе" -4) ;; поверне "світе"
```

Також можна використовувати `subs` для витягнення підрядків з довільних індексів. Для цього необхідно передати `-1` як другий індекс, що означає витягнення підрядка до кінця рядка. Наприклад:

```Clojure
(subs "Привіт, світе" 3 -1) ;; поверне "іт, світе"
```
