## Чому

Ви, можливо, хочете виконувати операції з підрядками для обробки текстових даних. Це може бути корисно для пошуку конкретної інформації або для створення нових текстових об'єктів зі складних рядків.

## Як

```Clojure
(str/split "Цей рядок має бути розділений" #" "). 
```

```
("Цей" "рядок" "має" "бути" "розділений")
```

```Clojure
(str/slice "Цей рядок має бути обрізаним" 4 10)
```

```
"рядок"
```

```Clojure
(str/replace "Цей рядок має бути заміненим" "має" "повинен")
```

```
"Цей рядок повинен бути заміненим"
```

## Глибший дивний

Використання підрядків є важливою частиною обробки текстових даних. Вони дозволяють ефективно маніпулювати рядками, щоб отримати потрібну інформацію або створити нові текстові об'єкти. Бібліотека Clojure "string" надає розширений набір функцій для роботи з підрядками, таких як вищеописані "split", "slice" та "replace". Детальніша інформація про використання цих функцій може бути знайдена в документації для "string" бібліотеки.
