# Чому
Чи ви коли-небудь потребували отримати поточну дату в своїй програмі? Наприклад, для створення логу чи для позначення часу події? Зараз ми поговоримо про те, як це можна зробити у Clojure.

## Як
```Clojure
(import 'java.util.Date) ; імпортуємо клас для роботи з датою
(def current-date (Date.)) ; створюємо змінну з поточною датою
```

Тепер my маємо змінну `current-date`, яка містить поточну дату. Латинські букви `my` перед назвою змінної - це лише конвенція, їх можна замінити на будь-які інші символи. Для отримання поточного часу можна також використовувати функцію `System/currentTimeMillis`, яка повертає час в мілісекундах з 1970 року.

```Clojure
(def current-time (System/currentTimeMillis))
```

Обидва ці підходи повертають об'єкти типу `Date`, тому якщо вам потрібна саме дата без часу, потрібно зробити додатковий крок.

```Clojure
(def current-date-only (.setTime (Date.) (.getTime (Date.))))
```

Просто змінюємо час у поточній даті на нульовий, тим самим видаляючи інформацію про час. Тепер ми можемо використовувати ці змінні для створення логів або для будь-яких інших потреб у ваших програмах.

## Deep Dive
Щоб краще розібратися з отриманням поточної дати, давайте поглянемо на кілька важливих деталей. По-перше, у Clojure існує багато різних способів працювати з датами, і вибір залежить від потреб вашої програми. Наприклад, для більш точного вимірювання часу можна використовувати тип `java.time.Instant`.

```Clojure
(def current-instant (java.time.Instant/now)) ; повертає поточний час у наносекундах
```

Також є більш зручні способи форматування дати, наприклад, функція `clj-time.format/strftime` дозволяє використовувати стандартні шаблони форматування дати.

```Clojure
(require '[clj-time.format :as time-format])
(def current-date-formatted (time-format/strftime "%d-%m-%Y" (Date.))) ; повертає поточну дату у форматі ДД-ММ-РРРР
```

Крім того, у Clojure є багато сторонніх бібліотек для роботи з датами, які можуть бути корисними у ваших проектах. Наприклад, `dateclj` або `clj-time`.
