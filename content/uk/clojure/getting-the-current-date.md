Що та Чому?
Отримання поточної дати - це процес отримання актуальної дати та часу на комп'ютері. Програмісти часто використовують цей засіб для відстеження часу виконання коду, порівнювання часу виконання між різними запусками програми та багатьох інших цілей.

Як це зробити:
```Clojure
; Підключення потрібної бібліотеки
(require ‘[clojure.java-time :as time])

; Отримання поточної дати та часу без часового поясу
(time/now) 

; Отримання поточної дати та часу з наступним перетворенням в рядок
(str (time/local-date) " " (time/local-time))

; Отримання поточної живої дати та часу з часовим поясом
(time/zoned-date-time)

```

Поглиблене дослідження:
1. Історичний контекст: у різних мовах програмування є різні способи отримання поточної дати та часу. Наприклад, у Java необхідно використовувати об'єкт типу "Date", у Python - модуль datetime. У Clojure цей процес значно спрощений та стандартизований завдяки бібліотеці clojure.java-time.

2. Альтернативи: окрім цієї бібліотеки, існує також стандартний інструмент Clojure для роботи з датою та часом - clojure.data-time. Це може бути зручним вибором для тих, хто працює з базовими типами даних у Clojure.

3. Деталі реалізації: бібліотека clojure.java-time використовує стандартні механізми Java для операцій з датою та часом, що дозволяє зручно та ефективно працювати з ними в програмах Clojure.
