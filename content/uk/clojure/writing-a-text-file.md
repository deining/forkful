# Чому

Написання текстового файлу є важливою частиною розробки програмного забезпечення на Clojure. Використання текстового файлу дозволяє зберігати і відстежувати код для подальшої роботи.

# Як написати текстовий файл на Clojure

Написання текстового файлу на Clojure є простим завданням. Для цього вам знадобиться використати функцію `spit`. Наприклад:

```Clojure
(spit "file.txt" "Hello, world!")
```

Цей код створить файл з назвою "file.txt" і запише у нього рядок "Hello, world!". Також, ви можете використати цю функцію для створення інших типів файлів, наприклад CSV або JSON.

# Глибоке погруження

Наочні приклади використання функції `spit` не закінчуються тут. Ви також можете використовувати цю функцію для динамічного створення файлів, використовуючи інформацію з вашої програми. Крім цього, ви можете використовувати функцію `slurp` для зчитування вмісту існуючих файлів та редагування їх напряму.
