## Чому

Написання текстового файлу є важливою частиною роботи програміста у Clojure. Це дозволяє зберігати дані, зберігати налаштування та робити багато інших речей. Також це є частиною доброї практики програмування, яка допомагає підтримувати організовану структуру у вашому коді.

## Як

У Clojure існує кілька способів створення та записування текстового файлу. Один з них - використання функції "spit", яка записує рядок у вказаний файл. Наприклад:

```Clojure
(spit "text.txt" "Привіт, світ!")
```

Це створить файл з назвою "text.txt" та запише у нього рядок "Привіт, світ!". Ми також можемо використати функцію "slurp" для читання вмісту файлу у змінну. Наприклад:

```Clojure
(def my-file (slurp "text.txt"))
```

Це створить змінну "my-file", яка буде містити вміст файлу "text.txt". Можна також використовувати функції "with-open" та "write" для створення та записування текстового файлу. Детальніше про ці методи можна дізнатися у розділі "Deep Dive".

## Deep Dive

У Clojure є багато інструментів для роботи з текстовими файлами, таких як "slurp", "spit", "write", "with-open" та багато інших. Використовуючи їх, ми можемо створювати, зчитувати та змінювати дані у файлі. Крім того, можна використовувати різні бібліотеки, які надають більш широкий функціонал для роботи з текстовими файлами. Це можуть бути бібліотеки для обробки CSV файлів, XML файлів чи для створення PDF файлів.

Крім вищезгаданих методів, важливо також вміти працювати зі строками і символами, оскільки багато функцій для роботи з текстовими файлами очікує аргументу типу "string" або "char". Також, варто знати про правила роботи з кодуванням, таким як UTF-8, яке визначає як символи зберігаються у файлі. Детальнішу інформацію про ці теми можна знайти у документації Clojure та у відкритих джерелах.
