## Чому

Розмістити виведення дебаг-інформації є важливою складовою частиною програмування, незалежно від мови, яка використовується. Це допомагає відстежувати помилки та зрозуміти, як працює програма. У даній статті ми розглянемо, як розміщувати дебаг-інформацію у Clojure та як це може полегшити роботу програмістів.

## Як робити

Використовуючи мову програмування Clojure, виведення дебаг-інформації можна робити за допомогою функції `println`, яка виводить передані їй аргументи у консоль. Наприклад:

```Clojure
(println "Значення x: " x)
```

У даному прикладі, значення змінної `x` буде виведено у консоль. Також, можливо використовувати `println` для виведення більш складної інформації, наприклад, вмісту списків чи мап. Наприклад:

```Clojure
(println "Список: " [1 2 3])
```

Результатом буде виведення наступного рядка у консолі:

```Clojure
Список: [1 2 3]
```

## Глибша розбірка

У деяких випадках, може бути корисно використовувати додаткові функції для виведення дебаг-інформації, наприклад, `prn` або `str`. Функція `prn` виконує те ж саме що і `println`, але додає додаткові символи нового рядка та пробілу для читабельності. Функція `str` об'єднує аргументи у рядок, який потім можна вивести у консоль за допомогою `println`. Наприклад:

```Clojure
(println "Змінні x та y: " (str x y))
```

Результатом буде виведення наступного рядка у консолі:

```Clojure
Змінні x та y:  11 22
```

Ці функції можуть бути корисними у випадках, коли необхідно вивести більш складну та чітку дебаг-інформацію.
