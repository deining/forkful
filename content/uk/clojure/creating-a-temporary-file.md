## Чому

Створення тимчасового файлу є важливою задачею для багатьох програмістів, оскільки воно дозволяє збереження тимчасової інформації, яка буде видалена після закінчення програми. Це може бути корисно для збереження проміжних результатів обчислень або для тимчасового збереження шифрованих даних.

## Як

Створення тимчасового файлу в Clojure дуже просте. Ось приклад коду, який демонструє цей процес:

```Clojure
(with-open [temp-file (java.io.File/createTempFile "prefix" "suffix")]
  (println (str "Ім'я тимчасового файлу: " (.getName temp-file)))
  (println (str "Шлях до тимчасового файлу: " (.getAbsolutePath temp-file))))
```

В цьому прикладі ми використовуємо функцію `createTempFile` з класу `java.io.File` для створення тимчасового файлу з префіксом "prefix" та суфіксом "suffix". Зверніть увагу, що це створить тимчасовий файл в тій самій директорії, де знаходиться ваша програма.

Після створення тимчасового файлу, ми використовуємо конструкцію `with-open`, яка автоматично закриє файл після того, як ми будемо готові. В цій конструкції ми також можемо виконувати операції з файлом, наприклад, отримати його назву та шлях.

Вивід виконання цього коду буде наступним:

```
Ім'я тимчасового файлу: prefix7541003050868702696suffix
Шлях до тимчасового файлу: C:\Users\user\AppData\Local\Temp\prefix7541003050868702696suffix
```

## Глибоке викладення

Окрім вказаних вище можливостей, Clojure також надає ряд додаткових функцій для створення тимчасових файлів. Наприклад, ви можете задати бажану директорію для файлу, або ж вказати, що файл повинен бути видалений після завершення програми.

Ось приклад:

```Clojure
(with-open [temp-file (java.io.File/createTempFile "customPrefix" "customSuffix" (java.io.File. "C:\\Users\\user\\Desktop"))]
  (println (str "Ім'я тимчасового файлу: " (.getName temp-file)))
  (println (str "Шлях до тимчасового файлу: " (.getAbsolutePath temp-file)))
  (.deleteOnExit temp-file))
```

В даному випадку, ми вказали бажану директорію для файлу за допомогою функції `File` з вказаним шляхом. Також ми використовуємо метод `deleteOnExit`, який вказує, що файл має бути видалений після закінчення програми.
