## Чому

Використання регулярних виразів дає можливість більш ефективно та зручно обробляти та шукати тексти за допомогою встановленого шаблону. Це дозволяє виконувати складні трансформації даних та пошук необхідної інформації швидше та простіше.

## Як

```Clojure
; Знайти номери телефонів в текстових повідомленнях
(re-seq #"[0-9]{3}-[0-9]{3}-[0-9]{4}" "Привіт, мій номер телефону 123-456-7890 та ще один 098-765-4321")
; => ("123-456-7890" "098-765-4321")

; Замінити всі входження символу "a" на "b" у рядку
(replace #"[a]" "abcdef")
; => "bbcdef"
```

## Глибокий занурення

Регулярні вирази використовуються у багатьох мовах програмування та редакторах тексту. Використовуючи їх, можна виконувати складні стрічкові операції, такі як розбиття тексту на підрядки, шукання та видалення певних патернів, заміна тексту та багато іншого. Навчитися використовувати регулярні вирази корисно для покращення продуктивності та ефективності під час обробки даних.
