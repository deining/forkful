## Чому
Один з найпоширеніших завдань програмування - перетворення дати в рядок. Це може бути необхідно для збереження дат у базі даних, передачі дати для веб-сайту або просто для зручного відображення дати для користувачів. У наш час це необхідна навичка для будь-якого програміста, тому добре знати, як це зробити у Clojure.

## Як це зробити
```Clojure
(require '[clojure.java-time :as time])

;; Приклад 1: Перетворення дати в рядок зі стандартним форматом
(time/format (time/local-date) "yyyy-MM-dd")
;; Вивід: "2020-12-03"

;; Приклад 2: Перетворення дати в рядок з власним форматом
(time/format (time/local-date) "dd MMMM yyyy")
;; Вивід: "03 грудня 2020"
```

У Clojure існує багато вбудованих функцій для роботи з датами та часом. Ви можете використовувати функції з простору імен `clojure.java-time` для перетворення дати в рядок з різними форматами. На цьому прикладі ми використовуємо функцію `format`, яка отримує два аргументи: дату і шаблон форматування.

## Глибша погляд
У Clojure дати та час зберігаються як об'єкти Java, тому для перетворення їх у рядок ми використовуємо функції з бібліотеки `java-time`. Функція `format` дозволяє використовувати різні формати дати, які можуть бути використані для виведення дати у зручному форматі для користувачів. Крім того, у бібліотеці `java-time` є також функція `parse`, яка дозволяє зчитувати рядки у датовий об'єкт. Це дуже зручно для обробки дат, які надходять від користувачів або з зовнішніх джерел.
