#Чому

Перевірка існування папки є важливим етапом в програмуванні для забезпечення належної організації і уникнення помилок.

#Як

```Clojure
;; Імпортуємо необхідну функцію для перевірки папки
(require '[clojure.java.io :as io])

;; Передаємо шлях до папки як аргумент функції
(io/file-exists? "/Users/user/Desktop/my_directory")

;; Виводимо значення перевірки - true або false
;; Якщо папка існує, буде виведено: true
;; Якщо папка не існує, буде виведено: false
```

# Глибокий розгляд

У Clojure для перевірки існування папки використовується функція `file-exists?` з модуля `clojure.java.io`. Ця функція приймає аргумент типу `File` і перевіряє, чи існує файл або папка за зазначеним шляхом. Якщо файл або папка існують, функція поверне значення `true`, інакше - `false`.

Також можна використовувати функцію `directory?`, яка приймає аргумент типу `File` і перевіряє, чи є це папкою. Якщо так, повертається значення `true`, якщо ні - `false`.
