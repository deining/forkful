# Чому

Коли ви задумуєтеся про створення нового проекту, може виникнути питання, чому вам це потрібно. Наприклад, ви можете бути зацікавлені в експериментах з новими технологіями або вирішити якусь проблему за допомогою програмування. Незалежно від причини, створення нового проекту може бути цікавим і захоплюючим досвідом.

# Як це зробити

Кодування у Clojure може бути простим і потужним одночасно. Ось декілька прикладів коду, які покажуть вам, як створити базовий проект у Clojure та вивести певні значення:

```
;; Оголошення функції
(defn hello-world []
  (println "Привіт світ!"))

;; Виклик функції
(hello-world)

;; Оголошення змінної
(def name "Іван")
(def age 25)

;; Виведення значень
(println "Привіт, " name)
(println "Тобі " age " років.")
```

Ви маєте отримати наступний вивід:

```
Привіт світ!
Привіт, Іван
Тобі 25 років.
```

Тепер ви можете використовувати ці основи для створення більш складних програм та проектів у Clojure.

# Глибше вдивимося

Якщо ви плануєте створити більший проект у Clojure, вам, напевно, знадобиться додаткова інформація та підтримка. Не соромтеся досліджувати бібліотеки, фреймворки та спільноту Clojure, які можуть допомогти вам у вашому проекті. Також можете роздивитися додаткові приклади коду та ресурси у мережі для поглиблення своїх знань.
