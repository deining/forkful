## Чому

Нерідко, під час програмування на Clojure, нам потрібно змінити або замінити текст у наших файлах. Це може бути потрібно для розширення функціональності програми, виправлення помилок або для кращого читання коду. Навіть найкращим програмістам іноді доводиться зіткнутися з несподіваними проблемами, які вимагають пошуку та заміни великої кількості тексту. Якщо ви не знаєте як це зробити, то цей пост стане для вас корисним.

## Як це зробити

Використання `clojure.string` простий спосіб для пошуку та заміни тексту. Для початку нам потрібно імпортувати цей модуль:

```Clojure
(ns my-project.core
  (:require [clojure.string :as str]))
```

Тепер ми можемо використати функцію `replace` для заміни тексту у рядках. Наприклад, якщо ми хочемо замінити всі пропуски на підкреслення, то ми можемо зробити так:

```Clojure
(str/replace "Це текст з пропусками" #" " "_")
```

Результат буде виглядати так: `"Це_текст_з_пропусками"`.

Якщо нам потрібно замінити лише перші три знаходження, можна використовувати флаг `:limit`:

```Clojure
(str/replace "Це текст з пропусками" #" " "_" {:limit 3})
```

Результат буде виглядати так: `"Це_текст_з пропусками"`.

## Глибші занурення

Якщо ви хочете більше контролю над процесом заміни, можна використати функцію `replace-first` або `replace-last`. Функція `replace-first` буде замінювати лише перше знаходження, а `replace-last` - останнє. Наприклад:

```Clojure
(str/replace-first "Це перше знаходження" #" " "@")
```

Результат буде виглядати так: `"Це@перше знаходження"`.

Також, можна використовувати функцію `replace-map` для заміни відповідно до переданої мапи. Наприклад, якщо ми хочемо замінити всі голосні літери на пропуски, то код може виглядати так:

```Clojure
(str/replace-map "Це слово містить голосні літери" {"а" "", "е" "", "и" "", "о" "", "у" "", "і" ""})
```

Результат буде виглядати так: `"Ц слов мстть глснн лтр"`.
