## Чому
Навіщо шукати довжину рядка в Clojure? Існує безліч ситуацій, коли корисно знати довжину рядка, таких як перевірка валідності введених користувачем даних, розміщення тексту на екрані чи обробка текстового вводу. Вивчаючи техніки знаходження довжини рядка, ви зможете зробити свій код більш ефективним та гнучким.

## Як
Для початку, імпортуємо функцію "count", яка дозволяє знайти довжину рядка в Clojure. Далі, створимо змінну, яку ми будемо використовувати для збереження довжини рядка.

```Clojure
(import '(clojure.string :as str)) ; імпортуємо функцію count для знаходження довжини рядка

(def text "Привіт, світ!") ; створюємо текстовий рядок

(def length (str/count text)) ; встановлюємо змінну length з довжиною рядка

(prn length) ; виводимо довжину рядка на екран: 12
```

## Глибоке занурення
Як ви, можливо, помітили, ми використовували функцію "count" з Clojure для знаходження довжини рядка. Але як саме вона працює? Ця функція приймає аргумент-рядок та повертає кількість символів у цьому рядку. Вона також може працювати зі списками та іншими типами даних.

Знайомство з функцією "count" допоможе вам краще розібратись із засобами знаходження довжини рядка та інших операцій над текстом в Clojure.
