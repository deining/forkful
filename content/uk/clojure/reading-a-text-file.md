Що та чому?

Читання текстового файлу - це процес, при якому програміст зчитує вміст файлу та використовує його для подальшої обробки у своїй програмі. Це може бути корисним, коли потрібно отримати дані з зовнішнього джерела або зберегти результат обробки у файл.

Як?

```Clojure
(with-open [rdr (clojure.java.io/reader "file.txt")] 
  (doseq [line (line-seq rdr)] 
  (println line)))
```

У цьому прикладі ми використовуємо функції ```with-open``` та ```clojure.java.io/reader```, щоб відкрити текстовий файл для читання. Потім ми використовуємо цикл ```doseq```, щоб прочитати рядки файлу та вивести їх на екран.

Занурення

Історичний контекст: з часом, коли комп'ютери стали змінювати людську працю, з'явилася потреба вчитися читати дані з різних джерел. Читання текстового файлу є одним з найбільш поширених способів отримати дані зовнішнього джерела.

Альтернативи: у Clojure є деякі інші методи для читання текстового файлу, наприклад, використання функції ```slurp```, яка повертає вміст файлу у вигляді рядка.

Деталі реалізації: у прикладі ми використовуємо функцію ```line-seq```, яка дозволяє читати по одному рядку з файлу. Також, ми використовуємо функцію ```println```, яка виводить рядок на екран.
