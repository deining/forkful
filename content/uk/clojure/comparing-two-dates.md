## Чому

Коли ви працюєте з датами у своїх програмах, іноді може знадобитися порівняти дві дати - чи одна дата раніше, пізніше чи рівна іншій. В цій статті ми розглянемо, як використовувати Clojure для порівняння двох дат.

## Як

Для порівняння дат використовується функція `compare`, яка повертає -1, 0 або 1 в залежності від того, яка з переданих дат є раніше:

```Clojure
(let [date1 (java.util.Date.)  ; створюємо першу дату
      date2 (java.util.Date. 2020 1 1)] ; створюємо другу дату
  (compare date1 date2)) ; повертає 1, оскільки date2 є ранішою за date1
```

Також використовуючи функцію `compare`, можна проводити рівність дат:

```Clojure
(let [date1 (java.util.Date. 2020 1 1) ; створюємо першу дату
      date2 (java.util.Date. 2020 1 1)] ; створюємо другу дату
  (compare date1 date2)) ; повертає 0, оскільки date1 і date2 рівні
```

Також можна використовувати оператор `>` для порівняння дат:

```Clojure
(let [date1 (java.util.Date.)  ; створюємо першу дату
      date2 (java.util.Date. 2020 1 1)] ; створюємо другу дату
  (> date1 date2)) ; повертає true, оскільки date2 є ранішою за date1
```

## Deep Dive

Функція `compare` порівнює дати за їх значеннями, тобто за кількістю мілісекунд, які пройшли з 1 січня 1970 року. Це означає, що ця функція може повертати не тільки -1, 0 і 1, але і будь-які числа, які показують різницю між датами.

Важливо також пам'ятати, що тип даних `java.util.Date` в Clojure не містить інформацію про часовий пояс, тому при порівнянні дат слід ураховувати цей момент і використовувати правильні функції для роботи з часовим поясом.
