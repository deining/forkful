## Що і для чого?

Використання регулярних виразів дозволяє нам шукати і заміняти текст за певним шаблоном. Програмісти застосовують їх, щоб швидко та ефективно працювати з текстом, будь то перевірка правильності введених даних чи обробка великих обсягів даних.

## Як використовувати:

Використовуючи регулярні вирази в Ruby, ми можемо створити шаблон для пошуку та заміни тексту. Наприклад, за допомогою функції `gsub` ми можемо замінити усі або лише перші збіги з шаблоном на вказане значення:

```ruby
"Привіт, світ!".gsub(/світ/, "земля")
# => "Привіт, земля!"
```

Також ми можемо використовувати регулярні вирази для перевірки правильності введених даних:

```ruby
"12345".match(/\d+/)
# => #<MatchData "12345">
"abcde".match(/[a-z]+/)
# => #<MatchData "abcde">
```

## Глибоке дослідження:

Регулярні вирази були винайдені в 1956 році користувачами форуму комп'ютерів SHARE і з тих пір знаходять своє застосування у багатьох програмувальних мовах. У Ruby ми також можемо використовувати методи об'єкта `String` для роботи з регулярними виразами. Альтернативами до регулярних виразів можуть бути методи для обробки тексту, такі як `split` та `slice`. Щоб дізнатися більше про регулярні вирази в Ruby, варто ознайомитися з документацією [Regexp](https://ruby-doc.org/core-2.7.1/Regexp.html) та [String](https://ruby-doc.org/core-2.7.1/String.html).
