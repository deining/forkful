## Завдяки чому

Регулярні вирази є потужним інструментом для роботи зі строками в програмуванні. Вони дозволяють шукати, замінювати та валідувати певні частини тексту за допомогою складних шаблонів. Використання регулярних виразів може суттєво полегшити завдання обробки тексту, що дозволяє ефективніше працювати з даними.

## Як користуватися регулярними виразами

Для початку, необхідно імпортувати бібліотеку `regexp` в Ruby. Далі, можна використовувати методи цієї бібліотеки для роботи з регулярними виразами.

```Ruby
# Відповідно до певного шаблону шукаємо текст в строкі
"Привіт, світ!" =~ /світ/ # Видасть індекс, на якому починається слово "світ"

# Замінимо певну частину тексту іншим
"Привіт, світ!" .sub(/світ/, "окружність") # Результат: "Привіт, окружність!"
```

Використання зворотніх слешів дозволяє створювати складніші шаблони. Також можна використовувати спеціальні символи, наприклад, `^` для початку строки, `$` для її кінця або `.` для позначення будь-якого символу.

## Глибшій занурення

Регулярні вирази мають багато функціоналу, який може допомогти при обробці даних. Наприклад, використання груп дозволяє виділяти певні частини тексту для подальшого використання.

```Ruby
# Виділимо з кожної стрічки тільки ті слова, які містять у собі більше 5 літер
names = ['Mike', 'John', 'Steven', 'Alex', 'Rachel']
names.select { |name| name =~ /\w{5,}/ } # Результат: ['Steven', 'Rachel']
```

Також, використання модифікаторів дозволяє змінювати поведінку регулярного виразу, наприклад, враховувати регістр символів або ігнорувати пробіли.
