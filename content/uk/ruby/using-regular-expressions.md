## Чому
У програмуванні часто виникає необхідність у використанні регулярних виразів для розбору тексту і здійснення певних операцій над ним. Вони дозволяють більш швидко та ефективно працювати зі строками, не витрачаючи багато часу на написання власної логіки.

## Як
Для початку, нам потрібно встановити модуль `regexp` за допомогою команди `gem install regexp`. Потім ми можемо почати використовувати регулярні вирази у нашому коді. Нижче наведено приклади використання регулярних виразів для пошуку і заміни тексту:

```Ruby
string = "Це рядок з якоюсь інформацією"
pattern = /якоюсь/

# Пошук за допомогою методу `match`:
match = string.match(pattern)
puts match[0] # виведе "якоюсь"

# Заміна за допомогою методу `gsub`:
replaced_string = string.gsub(pattern, "певною")
puts replaced_string # виведе "Це рядок з певною інформацією"
```

## Глибші дослідження
Регулярні вирази дозволяють виконувати широкий спектр операцій, таких як пошук і заміна, перевірка на відповідність певному формату, вилучення необхідної інформації зі строки і багато іншого. Також їх можна комбінувати з умовами, циклами та іншими конструкціями мови Ruby для більш складних операцій.

Можна ознайомитися з детальнішими прикладами та функціями регулярних виразів у документації [Ruby Regexp](https://ruby-doc.org/core-3.0.0/Regexp.html). Також варто звернути увагу на додаткові модулі, які розширюють можливості регулярних виразів, наприклад [Rubular](https://github.com/marcelovicentegc/rubular) або [Regexp::Common](https://github.com/ruby/ruby/blob/trunk/lib/regexp/common.rb).
