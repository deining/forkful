## Зачем

Часто наші програми мають велику кількість даних, з якими нам потрібно працювати. Використання регулярних виразів дозволяє нам швидко та впевнено знаходити, перевіряти та обробляти певні шаблони або відповіді. Це зберігає час та зусилля, які витрачаються на роботу з великими об'ємами даних вручну.

## Як використовувати

Для використання регулярних виразів у Ruby, нам потрібно використовувати методи класу `Regexp`. Давайте розглянемо кілька прикладів коду, які допоможуть нам краще зрозуміти цю тему.

```ruby
# Перевірка чи є мейл валідним
email = "example@email.com"
valid = /\A([\w+\-].?)+@[a-z\d\-]+(\.[a-z]+)*\.[a-z]+\z/i.match?(email)
puts valid
# Виведе: true
```

```ruby
# Знаходження усіх чисел у заданому тексті
text = "Today I bought 3 apples and 2 oranges."
numbers = text.scan(/\d+/)
puts numbers
# Виведе: ["3", "2"]
```

```ruby
# Заміна всіх посилань у HTML коді на [Посилання]
html = "<a href='https://example.com'>Link</a>"
new_html = html.gsub(/<a.+href='(.+)'>.+<\/a>/, "[Link]")
puts new_html
# Виведе: "[Link]"
```

## Занурення у глибини

Регулярні вирази можуть бути складними та заплутаними, але є кілька корисних інструментів, які допоможуть нам розібратися з ними. Клас `Regexp` має функції для створення, компіляції та маніпуляції з регулярними виразами. Також є багато онлайн-ресурсів, які допоможуть нам тестувати та перевіряти наші регулярні вирази перед використанням у програмі.
