## Чому

Виділення підрядків (або підстрічок) є важливим елементом програмування на Ruby. Випадки, коли користувачам потрібно отримати підрядок з довгого рядка або змінної, є досить поширеними. Це може бути корисним для обробки і дослідження даних, а також для забезпечення ефективності і пришвидшення розвитку програм. Давайте дізнаємося, як це зробити!

## Як це зробити

Існує кілька способів виділення підрядків у Ruby. Один з них - використання методу `[]` з індексом або діапазоном для вибору бажаного підрядка з рядка або змінної. Подивіться на приклад коду нижче:

```Ruby
str = "Привіт, Україно!"

# Виділяємо підрядок за індексом
puts str[1] # р
puts str[3..6] # івіт

# Вибираємо підрядок за допомогою умови
str2 = "РОЖЬ"
puts str2[/[А-Я]/] # Р
```

У цьому прикладі ми використовуємо метод `[]` для отримання підрядків за індексом або діапазоном. Крім того, ми можемо використовувати умови, які допоможуть нам вибрати підрядок з використанням регулярних виразів.

## Deep Dive

Окрім використання методу `[]`, ми також можемо використовувати методи `slice` та `sub` для виділення підрядків. Метод `slice` працює аналогічно методу `[]`, але дозволяє використовувати іменовані параметри для зручності. Метод `sub`, з свого боку, видаляє перший відповідний підрядок з рядка.

```Ruby
str = "Це просто тест"
puts str.slice(0..1) # Це
puts str.sub("тест", "програма") # Це просто програма
```

Також існує метод `gsub`, який дозволяє замінювати всі відповідні підрядки в рядку.

```Ruby
str = "Привіт, Ruby!"
puts str.gsub("Ruby", "Україна") # Привіт, Україна!
```

Як бачите, існує багато способів, які дозволяють виділяти підрядки у Ruby. Виберіть найзручніший для вас і використовуйте його залежно від потреб вашої програми.
