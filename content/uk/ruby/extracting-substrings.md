## Що & Чому?
Видобування підстрок - це процес візуального вилучення частини тексту зі строки або масиву. Це корисний інструмент для програмістів, оскільки дозволяє здійснювати більш точну та ефективну обробку даних.

## Як це зробити:
Існує кілька способів вилучення підстрок у мові програмування Ruby. Один із способів - використовувати метод `slice` на строкових чи масивних об'єктах. Для роботи з масивами можна використовувати індекси, а для строк - границі підстроки, яку потрібно вилучити. Нижче наведено приклади коду та виведеного результату:

```Ruby
str = "Привіт, світ!"
str.slice(0,4)  # виведе "Привіт"
str[7..10]  # виведе "світ"
```

## Глибоке занурення:
Історично слід зазначити, що вилучення підстроки було відомо ще з часів C-подібних мов, де використовувалися функції `strstr` та `strncpy`. Однак, з появою об'єктно-орієнтованого підходу до програмування, методи виклику функцій було замінено на більш зручні методи об'єктів. Імплементація вилучення підстроки на мові Ruby використовує ефективні алгоритми для забезпечення швидшої обробки даних.

Існують також альтернативні методи вилучення підстрок, такі як використання регулярних виразів чи метод `split` для розділення строки на масив з підстрок.
