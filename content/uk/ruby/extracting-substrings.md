# Чому

Одним з найпопулярніших методів роботи з рядками в Ruby є використання методу #slice. Виконуючи дії зі стрічками, часто може виникнути потреба витягнути певну частину з них - підрядок. Для цього істнує спеціальний метод - #slice.

# Як використовувати

```Ruby
str = "Привіт, це Ruby блог по програмуванню"

puts str.slice(7, 4)
# виводить "Ruby"
```

В першому рядку ми створили змінну, в якій зберегли рядок з нашим повідомленням. У другому рядку використовуємо метод #slice, передаючи йому два аргументи - початкову позицію символу, з якої починаємо виділяти підрядок (у цьому випадку це символ з індексом 7 - буква "R") і кількість символів для виділення (у цьому випадку 4 - "R" "u" "b" "y"). Як результат, ми отримали підрядок "Ruby" і вивели його на екран.

Цей метод також можна використовувати з використанням від'ємних індексів, що дозволяє виділяти підрядок з кінця рядка:

```Ruby
puts str.slice(-14, 11)
# виводить "Ruby блог"
```

Метод #slice також приймає рядок з індексами, як показано нижче:

```Ruby
puts str.slice("7,4")
# виводить "Ruby"
```

# Глибоке дослідження

Метод #slice можна використати не тільки з цілими числами аргументів, ай і з рядками, що містять діапазони:

```Ruby
puts str.slice(7..9)
# виводить "Ruby"
puts str.slice("7..9")
# виводить "програмувати"
```

Також, метод #slice можна використати зрізами:

```Ruby
puts str.slice(7...18)
# виводить "Ruby блог"
```

Замість #slice, можна використати аналогічний метод #[]:

```Ruby
puts str[7, 4]
# виводить "Ruby"
```
