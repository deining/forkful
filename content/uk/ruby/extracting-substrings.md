## Чому

Однією з важливих та корисних функцій мови програмування Ruby є можливість витягувати частини тексту за допомогою функції вилучення підрядків. Це дозволяє зробити програму більш гнучкою та ефективною для виконання різних завдань, особливо з рядками тексту.

## Як використовувати

Щоб вилучити підрядки з тексту у Ruby, використовуйте метод `slice` або `[]`. Вони приймають два параметри - початковий та кінцевий індекси. Наприклад:

```Ruby
text = "Це приклад рядка тексту."

# Вилучення підрядка розпочинаючи з 5-го символу
puts text.slice(5..-1)
# Виводить: приклад рядка тексту.

# Вилучення підрядка з 11-го по 16-й символ
puts text[11, 6]
# Виводить: рядка
```

## Глибше копаємо

Методи `slice` та `[]` можна також використовувати зі змінними або виразами, що дає більшу гнучкість. Крім того, можна використовувати регулярні вирази для пошуку та вилучення певних підрядків з тексту. Наприклад:

```Ruby
text = "Це приклад рядка тексту."

# Вилучення підрядків, які починаються з цифри
puts text[/\d.*/]
# Виводить: 1 приклад рядка тексту.

# Вилучення всіх цифр з тексту
puts text.gsub(/\D/, '')
# Виводить: 0

```
