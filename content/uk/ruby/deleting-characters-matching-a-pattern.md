# Що & Чому?
Видалення символів, які відповідають патерну, є процесом у програмуванні, який дозволяє програмістам знаходити і видаляти певні символи з рядка тексту або документу. Це може бути корисним для очищення даних або для пошуку конкретної інформації.

# Як це зробити:
Використовуючи метод ```gsub```, ми можемо замінити всі символи, що відповідають патерну, на порожній рядок. Наприклад, якщо ми хочемо видалити всі цифри з рядка, ми можемо написати ```str.gsub(/\d+/, '')```, де ```str``` - наш рядок. Результат цієї операції буде рядком, з якого були видалені всі цифри. Альтернативно, ми також можемо використовувати методи ```delete``` або ```tr```, щоб видалити окремі символи, які відповідають патерну.

# Глибше погляд:
Видалення символів за певним патерном використовувалося в більш старих мовах програмування, таких як C і Perl. У Ruby, крім методів ```gsub```, ```delete``` і ```tr```, також існує метод ```sub``` для заміни тільки першого входження патерна. Для видалення всіх символів, крім відповідних певному патерну, можна використовувати метод ```squeeze```.
