## Чому

Ви, як програміст, можливо захочете видалити символи, які відповідають певному шаблону, зі свого коду. Є багато причин, чому ви можете це зробити, однак, основною може бути виправлення помилок або оптимізація коду.

## Як

Використовуючи Ruby, ви можете видалити символи, що відповідають певному шаблону, зробивши це за допомогою методу `gsub`. Наприклад, якщо ви хочете видалити всі пропуски з рядка, ви можете виконати такий код:

```Ruby
string = "Це рядок з пробілами"
string.gsub!(" ", "")
puts string
```

Output:
```
Церядокльить
```

У цьому прикладі ми використовуємо метод `gsub`, щоб замінити всі пробіли (перший параметр) на порожній рядок (другий параметр). Параметр `!` змінює початковий рядок, а не повертає новий. Ви можете використовувати будь-який шаблон для видалення символів, наприклад, видалити всі цифри, використовувати шаблон `[0-9]` тощо.

## Детальний аналіз

Існує багато інших способів видалення символів з рядка, наприклад, за допомогою методу `delete` або `tr`. Однак, метод `gsub` є найбільш гнучким і може застосовуватися до будь-якого рядка, незалежно від його вмісту. Також варто зазначити, що метод `gsub` може приймати регулярні вирази як параметри для більш складних шаблонів видалення символів.
