# Чому

Нещодавно ви могли зіткнутися з необхідністю видалити певні символи зі своїх рядків у своїй програмі на Ruby. Наприклад, ви можете бажати видалити всі цифри з рядка або всі голосні літери. Рішенням цієї проблеми буде використання методу `gsub` для видалення символів, які відповідають певному шаблону.

# Як це зробити

Для початку, зазначимо, що метод `gsub` доступний для використання на об'єктах рядків (String) і дозволяє замінити всі входження певного шаблону на заданий замінник.

Наприклад, для видалення цифр з рядка ми можемо використати наступний код:

```Ruby
string = "Hello123World456"
new_string = string.gsub(/\d/, "")
puts new_string # => HelloWorld
```

У цьому прикладі, за допомогою регулярних виразів (`/\d/`), ми вказали, що хочемо замінити всі цифри на порожні символи, тобто видалили їх із рядка.

Також, ви можете використати метод `gsub` для видалення символів, які не відповідають певному шаблону. Наприклад, якщо ми хочемо лишити у рядку тільки цифри, ми можемо зробити так:

```Ruby
string = "Hello123World456"
new_string = string.gsub(/[^\d]/, "")
puts new_string # => 123456
```

Тут, ми вказали шаблон `[^\d]`, що означає "будь-який символ, крім цифр", і замінили їх на порожні символи.

# Vertdeep Вивчення

Щоб краще зрозуміти, як працює метод `gsub`, можна вивчити його сусідній метод `sub`. Вони працюють так само, за винятком того, що `gsub` замінює всі входження шаблону, а `sub` - тільки перше знайдене входження. Також, обидва методи приймають два параметри: шаблон та замінник.

Крім того, метод `delete` також може бути використаний для видалення символів з рядка. Він приймає один аргумент - рядок символів, які потрібно видалити.
