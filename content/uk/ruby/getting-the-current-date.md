## Чому 

Отримання поточної дати - це важлива функція в багатьох програмах. Незалежно від того, чи ви створюєте звичайний термінал, веб-сайт або мобільний додаток, іноді вам потрібно буде знати, яка зараз дата, щоб виконати певні операції. Наприклад, ви хочете друкувати поточну дату на листі, створювати унікальні ідентифікатори для користувачів або визначати вік людини. Затребування поточної дати може бути різним, але основна ідея залишається - це потрібно для багатьох програм.

## Як

Існує декілька способів отримання поточної дати в Ruby. Один з них - використовувати готовий метод `Time.now`. Перш ніж показати приклад коду, давайте спершу подивимося, як виглядає виведення поточної дати в терміналі:

```
$ ruby -e 'puts Time.now'
2020-12-03 21:51:43 +0200
```

Тепер давайте зробимо те ж саме за допомогою спільного коду Ruby:

```ruby
puts Time.now
```

Виведення буде аналогічним:

```
2020-12-03 21:51:43 +0200
```

Крім використання `Time.now`, ви також можете використовувати клас `Date`, який дозволяє працювати з датами більш детально. Наприклад, ви можете отримати поточну дату в форматі `YYYY-MM-DD`:

```ruby
require 'date'
puts Date.today.to_s
```

Виведення буде наступним:

```
2020-12-03
```

Існує багато інших можливостей отримання дати в Ruby, і вам варто провести додаткове дослідження, щоб знайти той, який найкраще підійде для вашої програми.
