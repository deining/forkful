## Почему

Замена текста - часто необходимая задача при программировании, особенно при работе с большими объемами данных. Это позволяет быстро и эффективно изменять большие блоки текста без необходимости вручную редактировать каждый отдельный символ.

## Как это сделать

Для выполнения поиска и замены текста в Ruby используется метод `gsub`. Ниже приведен пример кода, который заменяет все вхождения слова "привет" на слово "hello" в строке `text`:

```Ruby
text = "Привет, мир!"
text = text.gsub("привет", "hello")
puts text
```

Вывод:

```
hello, мир!
```

Метод `gsub` принимает два аргумента: строку, которую нужно заменить, и строку, на которую нужно заменить. Он ищет все вхождения первой строки в исходной строке и заменяет их на вторую строку.

## Глубокое погружение

Метод `gsub` также позволяет использовать регулярные выражения для поиска и замены текста. Например, следующий код заменяет все цифры в строке `text` на слово "num":

```Ruby
text = "123 hello 456"
text = text.gsub(/\d+/, "num")
puts text
```

Вывод:

```
num hello num
```

Здесь мы использовали регулярное выражение `\d+`, которое означает любое количество цифр подряд. Таким образом, все вхождения цифр в строке будут заменены на слово "num".
