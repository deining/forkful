# Що і чому?

Пошук і заміна тексту - це процес знаходження та заміни певних символів або рядків в текстовому файлі за допомогою програмного коду. Програмісти часто використовують цей процес для автоматизації великих обсягів роботи, зокрема для зміни даних або виправлення помилок у великих текстових файлах.

## Як це зробити:

```Ruby
# Приклад коду заміни "hello" на "hi"
string = "hello world"
new_string = string.gsub("hello", "hi")
puts new_string # виведе "hi world"
```

```Ruby
# Приклад коду заміни всіх чисел на їх квадрати
numbers = [1,2,3,4,5]
new_numbers = numbers.map { |num| num ** 2 }
puts new_numbers # виведе [1,4,9,16,25]
```

## Детальніше:

Історично, пошук і заміна тексту були широко використовуваними методами у редакторах тексту та текстових процесорах, але з часом стали популярними серед програмістів. Альтернативами доцільності пошуку та заміни тексту є використання регулярних виразів або використання готових бібліотек для обробки тексту.

У Ruby є кілька методів для заміни тексту, включаючи `gsub` та `sub`. `gsub` дозволяє замінити всі співпадіння, а `sub` замінює тільки перше співпадіння. Ці методи можуть бути корисними при роботі з великими даними чи докладанні зусиль до виправлення одного елементу.
