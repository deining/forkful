# Чому

Привіт, усі! Сьогодні ми поговоримо про те, які можливості надає нам мова програмування Ruby. Особливо, ми розберемося, як і для чого здійснюється капіталізація рядка.

Капіталізація рядка - це процес перетворення першої літери рядка на велику, а решту літер - на малі. Це може бути корисно, наприклад, при форматуванні даних, коли потрібно вивести слово з великої літери.

# Як

Щоб зробити капіталізацію рядка в Ruby, нам необхідно виконати наступні кроки:

1. Створити змінну, яка буде містити наш рядок: `str = "привіт, усі!"`

2. Використати метод `.capitalize` для цієї змінної: `str.capitalize`

3. Вивести результат на екран: `puts str`

В результаті у нас буде виведено `Привіт, усі!`.

Давайте подивимося на приклад коду:

```Ruby
str = "привіт, усі!"
str.capitalize
puts str
```

Результат:

```
Привіт, усі!
```

# Глибоке дослідження

Важливо знати, що метод `.capitalize` не тільки змінює першу літеру на велику, але і використовує локальні налаштування для коректного форматування рядка. Наприклад, якщо ваш рядок містить букви зі спеціальними символами, метод може неправильно виконати капіталізацію.

Також існує метод `.capitalize!`, який змінює початковий рядок без створення нового. Ви можете спробувати його використати в своєму коді.
