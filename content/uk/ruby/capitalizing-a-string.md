## Чому

Якщо ви знайшлися на цій сторінці, ви, напевно, цікавитесь програмуванням на мові Ruby. Можливо, ви вивчаєте її на курсах або самостійно, а можливо, вже маєте певний рівень досвіду. У будь-якому випадку, ви можете знати, що однією з основних операцій в будь-якій мові програмування є обробка рядків. І сьогодні ми поговоримо про одну з таких операцій - великі літери рядка.

## Як це зробити

Щоб зрозуміти, як працює операція капіталізації рядка в Ruby, давайте розглянемо декілька прикладів. Для початку, ми створимо просту змінну зі стрічкою і використаємо метод `capitalize` для перетворення першої букви великою:

```Ruby
name = "марія"
puts name.capitalize
```
Вивід: Марія

Якщо нам потрібно перетворити всі слова в рядку у великі літери, то для цього використовується метод `upcase`:

```Ruby
phrase = "привіт, україно!"
puts phrase.upcase
```
Вивід: ПРИВІТ, УКРАЇНО!

З іншого боку, якщо ми хочемо перетворити рядок у нижній регістр, то використовуємо метод `downcase`:

```Ruby
team = "Динамо"
puts team.downcase
```
Вивід: динамо

Іншим варіантом є метод `swapcase`, який замінює великі літери на маленькі і навпаки:

```Ruby
country = "україна"
puts country.swapcase
```
Вивід: УКРАЇНА

## Глибоке вивчення

Ми розглянули декілька методів, але що стоїть за ними? Як саме працює операція капіталізації рядка в Ruby?

Все дуже просто. Коли ми викликаємо методи `capitalize`, `upcase`, `downcase` або `swapcase`, вони працюють з ASCII (American Standard Code for Information Interchange) значеннями символів. Тобто, вони перетворюють одну літеру в іншу в залежності від її ASCII коду. Наприклад, літера "а" має значення 97 у ASCII таблиці, а літера "А" - 65. Тому, коли ми викликаємо метод `upcase` для рядка "марія", він знаходить ASCII код для кожної літери і, якщо це маленька літера, замінює її на відповідну велику.
