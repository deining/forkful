## Чому

Навіть коли ви ще тільки починаєте вивчати Ruby, ви, напевно, вже чули про рядкові дані. Це дуже корисний тип даних, який дозволяє зберігати текстову інформацію. Часто, наші програми вимагають роботи з рядками, навіть якщо це тільки виведення їх на екран.

Знання довжини рядка є важливим для багатьох задач програмування. Наприклад, можна використовувати його для обрізання рядків до потрібної довжини або для перевірки, чи не перевищує введена користувачем інформація максимальну довжину. Тож знати, як знаходити довжину рядка в Ruby допоможе вам створювати більш ефективні та корисні програми.

## Як це зробити

Для початку, нам потрібно завантажити наш сценарій Ruby та визначити змінну, яку ми хочемо перевірити:

```Ruby
string = "Привіт, усім!"
```

Тепер, використовуючи метод `length`, ми можемо знайти довжину рядка `string` і вивести результат на екран: 

```Ruby
puts string.length
```

Також, зверніть увагу, що метод `length` не враховує пробіли в кінці рядка. Якщо ви бажаєте врахувати і їх, використовуйте метод `size` замість `length`.

```Ruby
string_with_spaces = "Привіт, усім! "
puts string_with_spaces.length # Виведе 12
puts string_with_spaces.size # Виведе 13
```

Якщо потрібно знайти довжину рядка, який отримуємо від користувача, просто використайте метод `gets` для отримання даних та метод `length` для визначення їх довжини:

```Ruby
puts "Введіть своє ім'я:"
name = gets.chomp
puts "Привіт, #{name}! Ти маєш #{name.length} букв в своєму імені."
```

Цей код зчитує введені користувачем дані та виводить повідомлення про кількість букв у підставленому імені.

## Глибоке занурення

У Ruby довжина рядка обчислюється за допомогою методу `length`. Цей метод викликається змінною рядка і повертає її довжину у вигляді цілого числа. Також, варто відзначити, що у Ruby методи `length` та `size` є синонімами, тож вони виконують одну й ту саму функцію.
