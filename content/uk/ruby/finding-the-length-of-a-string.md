## Що & Навіщо?
Знаходження довжини рядка є важливим елементом для багатьох програмістів у Ruby. Це дозволяє вам дізнатися, скільки символів міститься у вашому рядку. Іноді це потрібно для виконання певних операцій або для форматування виведення. 

## Як це зробити:
Найпростішим способом знайти довжину рядка є використання методу `length` або `size` для вбудованих об'єктів типу String. Наприклад:

```Ruby
"Привіт!".length # виведе 6, оскільки цей рядок складається з 6 символів
```

Ми також можемо знайти довжину списку елементів за допомогою методу `count`:

```Ruby
[1, 2, 3, 4].count # виведе 4, оскільки список містить 4 елементи
```

## Глибше:
В Ruby метод `length` є аліасом методу `size`, тобто обидва ці методи мають однакове значення і можуть використовуватися взаємозамінно. Також існує аналогічний метод `bytesize`, який дозволяє знайти розмір у байтах. 

Наприклад, якщо ми використовуємо метод `length` для кириличного рядка, то отримаємо неочікуваний результат:

```Ruby
"Привіт!".length # виведе 12, оскільки кожен символ у кириличному алфавіті займає 2 байти
```

Існують також інші способи знаходження довжини рядка, наприклад, з використанням регулярних виразів або методу `Enumerable#count`. Ще одна цікава функція - `unicode_normalize`, яка дозволяє нормалізувати кириличний рядок до формату UTF-8 і отримати правильну довжину за допомогою методу `length`.
