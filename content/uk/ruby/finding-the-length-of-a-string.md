## Чому

Програмування - це захоплення, яке вимагає постійного навчання та пошуків нових викликів. Одним з навичок, які можуть покращити ваші програмістські вміння, є знаходження довжини строки в Ruby. Це основний елемент рядкових маніпуляцій та використовується в багатьох програмах, тому варто ознайомитися з цим поняттям.

## Як це зробити

Для початку слід розуміти, що таке рядок та як його можна зберегти в програмі. У Ruby строки огорнуті подвійними лапками, наприклад: `"Привіт, це рядок!"`. Також ви можете використовувати одинарні лапки для простих рядків, але подвійні лапки дають можливість використовувати різні спеціальні символи і змінні.

Один зі способів знайти довжину строки - використовувати метод `length`. Давайте подивимося на приклад:

```Ruby 
my_string = "Привіт, це рядок!"
puts my_string.length
```
Ви отримаєте виведення `18`, оскільки ця строка містить 18 символів, включаючи пробіли.

Інший спосіб - використовувати метод `size`. І це також працює на числах та масивах! Ось приклад:

```Ruby
array = [45, 25, 18, 9]
puts array.size
```
Ви отримаєте виведення `4`, оскільки цей масив містить чотири елементи.

## Глибокий пір

Хоча методи `length` та `size` зазвичай використовуються для знаходження довжини рядків, є й інші цікаві способи це зробити у Ruby.

Наприклад, ви можете використовувати метод `bytesize`, який повертає кількість байтів у рядку. Або ж метод `empty?` для перевірки, чи строка не містить жодних символів.

Навіть якщо ви вже знаєте довжину своєї строки, навчіться використовувати ці методи, щоб покращити свої програмістські навички та додати більше гнучкості до своїх кодів.
