## Чому

Робота з рядками є постійною частиною розробки програмного забезпечення. Визначення довжини рядка є важливою задачею, яка може бути використана для різних операцій, таких як обчислення, перевірка на коректність та розбиття рядків на окремі частини.

## Як

```ruby
str = "Привіт світ!"
puts str.length

# Output: 12
```

У цьому прикладі ми використовуємо метод `length` для обчислення довжини рядка `str`. Метод повертає кількість символів у рядку, включаючи пробіли та спеціальні символи.

```ruby
puts "Рядок".length
# Output: 5
```

Крім того, ми можемо використовувати метод `size` для досягнення того ж самого результату.

```ruby
puts "Рядок".size
# Output: 5
```

## Глибші дослідження

У Ruby рядки є мутабельними об'єктами, тому ми можемо змінювати їх довжину. Наприклад, ми можемо використовувати методи `insert` та `delete` для додавання або видалення символів з рядка, що вплине на його довжину.

```ruby
str = "Привіт світ!"
str.insert(6, "ближче до ") # додає текст перед позицією 6
puts str.length

# Output: 20
puts str # виводить "Привіт ближче до світ!"
```

Важливо зазначити, що методи `insert` та `delete` змінюють оригінальний рядок, а не створюють новий. Тому, якщо ми хочемо зберегти оригінальний рядок, необхідно зберігати його у змінну перед застосуванням цих методів.
