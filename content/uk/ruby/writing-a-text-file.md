## Чому

Існує багато використань для писання текстових файлів у програмуванні, таких як збереження даних та обробка великої кількості інформації. Також, писання текстових файлів може бути корисним при створенні звітів або імпортів даних.

## Як це зробити

Щоб створити текстовий файл у Ruby, використовуйте метод `File.write` та вказуйте назву файлу та його вміст. Наприклад, `File.write("hello.txt", "Привіт, світе!")` створить файл з назвою "hello.txt" із вмістом "Привіт, світе!". Можна також використовувати метод `puts` для запису вмісту у файл у новому рядку: `File.open("hello.txt", "w") { |file| file.puts "Привіт, ще раз!" }`.

Для читання текстового файлу використовуйте метод `File.read` та вказуйте назву файлу. Наприклад, якщо цей файл містить "Привіт, світе!", `File.read("hello.txt")` поверне цей рядок. Також можна використати інший метод, як `File.foreach`, щоб виконати певну дію для кожного рядка у файлі. Наприклад, `File.foreach("hello.txt") { |line| puts line }` виведе кожен рядок у файлі у новому рядку.

## Глибше

Крім простого створення та читання файлів, у Ruby також є багато інших методів та опцій для роботи з текстовими файлами. Наприклад, ви можете перевірити, чи існує файл за допомогою методу `File.exist?`, змінити назву файлу або видалити його. Також можна використовувати бібліотеки, які надають доступ до більш спеціалізованої роботи з текстовими файлами, наприклад для роботи з CSV-файлами.
