## Для чого

В рубі, створення текстового файлу є необхідною задачею для багатьох процесів програмування. Він дозволяє зберігати дані та надає доступ до них, а також дозволяє іншим скриптам читати та змінювати ці дані. Також створення текстового файлу є важливим кроком під час роботи з веб-застосунками, де дані зберігаються у файлах для подальшого використання.

## Як це зробити

Щоб створити текстовий файл в Ruby, спочатку потрібно відкрити файл з допомогою функції `File.open()`. В цій функції ми можемо вказати назву файлу, який ми хочемо створити, а також режим, у якому ми будемо працювати. Для створення файлу ми використаємо режим `'w'` (запис), який дозволяє нам записувати дані у файл. Далі, ми можемо використати метод `write()` для додавання тексту до файлу. Наприклад:

```Ruby
file = File.open("my_file.txt", 'w')
file.write("Цей текст буде збережено у файлі.")
file.close
```

Це створить файл з назвою `my_file.txt` та додасть до нього рядок `Цей текст буде збережено у файлі.` Також важливо пам'ятати закривати файл після завершення роботи з ним, щоб уникнути проблем з доступом до нього.

## Глибоке дослідження

Розглянутий вище приклад є дуже простим, але у Ruby є багато інших методів та можливостей для роботи з текстовими файлами. Наприклад, ви можете використовувати метод `puts()` для додавання нового рядка до файлу, а метод `read()` для читання даних з файлу. Також є можливість працювати з більш складними даними, такими як JSON або XML, використовуючи вбудовану бібліотеку `JSON` та `Nokogiri`.
