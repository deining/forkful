## Чому

Написання текстового файлу - це важлива навичка для будь-якого програміста, оскільки це є необхідним кроком для створення програм та збереження даних.

## Як

Для початку, ми можемо використати вбудований метод `File.open`, щоб створити чи відкрити існуючий файл у режимі запису, і передати йому назву файлу, у який ми хочемо записати наш текст. Потім, ми можемо використати метод `write` для додавання тексту до файлу, а метод `puts` для перехіду на новий рядок. Наприклад:

```Ruby
File.open("my_file.txt", "w") do |file|
  file.write("Привіт усім!\n")
  file.puts("Я написав свій перший текстовий файл.")
end
```

В цьому прикладі ми створюємо новий файл з назвою `my_file.txt` та додаємо до нього рядки "Привіт усім!" та "Я написав свій перший текстовий файл.", розділені переносом на новий рядок. Ми можемо перевірити цей файл та побачити його вміст нашою обраною програмою для роботи з текстовими файлами.

## Глибокий занурення

Крім простого додавання тексту до файлу, ми також можемо використовувати змінні, об'єкти та методи для більш складних операцій із текстовими файлами. Наприклад, ми можемо використати методи `gets` та `chomp` для отримання вводу від користувача та видалення символів нового рядка з кінця рядка. Для отримання детальнішої інформації щодо роботи з текстовими файлами, варто ознайомитися з документацією Ruby та переглянути різні приклади коду.
