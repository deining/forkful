## Що & Чому?
Порівняння двох дат - це процес визначення, чи є одна дата більшою, меншою або рівною іншій. Це важливо для програмістів, оскільки вони можуть використовувати цю інформацію, наприклад, для сортування даних або для перевірки справедливості введених користувачем дат.

## Як робити:
Щоб порівняти дві дати в Ruby, використовуйте метод ```.<```, ```.<=```, ```.<=>``` або ```.==``` в залежності від того, що ви хочете порівняти (наприклад, день, місяць, рік або просто дату в цілому).

```Ruby
date1 = Date.new(2020, 10, 15)
date2 = Date.new(2019, 10, 15)

puts date1 < date2 # виведе false
puts date1 >= date2 # виведе true
puts date1 <=> date2 # виведе 1 (date1 більше за date2)
puts date1 == date2 # виведе false
```

## Глибокий занурення:
Історичний контекст: порівняння дат було вишколене в специфікації Ruby 1.9, де метод ```.<=>``` був вперше доданий. Раніше для порівняння дат використовували оператори ```.==```, ```.<``` та ```.<=```.
Альтернативи: у Ruby є багато гемів для порівняння дат, наприклад, ```chronic``` та ```date_diff```, але для простих порівнянь вбудованого методу достатньо.
Деталі реалізації: методи порівняння використовують оператори ```<```, ```<=```, ```<<```, ```==``` для порівняння числових значень, які представляють різноманітні складові дати (рік, місяць, день). Це дозволяє зробити порівняння більш гнучким та точним.
