## Чому

Порівняння двох дат є важливою задачею у програмуванні, яка дозволяє визначити, яка з дат більш рання або пізня. Це допомагає управляти датами у програмах, які роблять обчислення часу.

## Як

```Ruby
# Приклад порівняння дат
today = Time.now # сьогоднішня дата
tomorrow = today + (24*60*60) # дата завтра

if tomorrow > today
  puts "Завтра буде пізніше, ніж сьогодні."
elsif tomorrow < today
  puts "Сьогодні буде пізніше, ніж завтра"
else
  puts "Обидві дати рівні."
end

# Результат виконання
# Завтра буде пізніше, ніж сьогодні.
```

Для порівняння дат використовується оператор порівняння `>`, `<` або `==`. Крім того, можна перетворити дати у `Integer` та порівняти їх як числа.

## Глибше

У Ruby дати представлені у вигляді `Time` об'єктів. Ці об'єкти містять інформацію про час, дату та час зоною. Вони базуються на кількості секунд від початку поданих дат, що дозволяє їх легко порівнювати за допомогою вищезгаданих операторів порівняння. У Ruby також є універсальний метод `Comparable`, який дозволяє використовувати оператори порівняння для будь-яких об'єктів.
