# Чому

У програмуванні часто доводиться порівнювати дві дати для виконання різних завдань. Наприклад, перевірити, чи приближається термін дії паспорта або перевірити чи дати бронювання перекриваються. Тому знання процесу порівняння дат є корисним для будь-якого розробника.

# Як

У Ruby порівняння дат виконується за допомогою умовних операторів, таких як `if`, `else` та `elsif`. Найпростіший спосіб - використати метод `==`, який повертає значення `true`, якщо дати рівні, і `false`, якщо нерівні.

```Ruby
today = Date.today
tomorrow = Date.tomorrow
puts today == tomorrow
# Виведе "false"
```

Можна також використовувати методи порівняння `<` (меньше), `>` (більше), `<=` (меньше або дорівнює) та `>=` (більше або дорівнює).

```Ruby
date1 = Date.new(2021, 1, 1)
date2 = Date.new(2020, 12, 31)
puts date1 > date2
# Виведе "true"
```

Крім того, у Ruby є спеціальний клас `DateTime`, який дозволяє порівнювати дати з часом. Для його використання потрібно встановити гем `date` та імпортувати його в свій код. Подивимося приклад, як порівняти дату з часом за допомогою `DateTime`:

```Ruby
require 'date'
datetime1 = DateTime.new(2021, 1, 1, 12, 0, 0)
datetime2 = DateTime.new(2021, 1, 1, 12, 15, 0)
puts datetime1 < datetime2
# Виведе "true"
```

# Deep Dive

Порівнювання дат у Ruby базується на їх порядковому номері. Це означає, що, наприклад, дата 1 січня 2021 року буде мати більший порядковий номер, ніж дата 31 грудня 2020 року. Також треба враховувати різницю між об'єктами класів `Date` та `DateTime`, щоб уникнути помилок при порівнянні дат з часом.

У Ruby є багато різних методів та операторів для порівняння дат, тому варто ознайомитися з офіційною документацією для детальнішого розуміння.
