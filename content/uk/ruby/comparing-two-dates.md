## Чому

Порівняння двох дат є важливою задачею для програмістів у багатьох проектах. Це дозволяє визначити, яка з дат є більшою або меншою, а також порахувати різницю між ними. Знання цього може бути корисним для створення розкладів, обчислення віку або для організації даних за часом.

## Як

Перш ніж порівнювати дві дати, потрібно переконатися, що вони в однаковому форматі. Для цього ми можемо скористатися методом `strftime` для перетворення дати у строкове значення. Нижче представлені два приклади з різними форматами дат:

```Ruby
date1 = Time.new(2021, 8, 10) # Дата у форматі року, місяця та дня
date2 = "10/08/2021" # Дата у форматі дня, місяця та року, записана як строкове значення

```

Ми можемо скористатися `strftime` для переведення другої дати в той же формат, що й перша:

```Ruby
date2 = Time.strptime(date2, "%d/%m/%Y") # Перетворення дати у формат року, місяця та дня
```
Тепер ми можемо порівняти обидві дати за допомогою операторів порівняння (`>`, `<`, `==`):

```Ruby
if date1 > date2
  puts "Перша дата більша"
elsif date1 < date2
  puts "Друга дата більша"
else
  puts "Обидві дати однакові"
end
```
Якщо нам потрібно порахувати різницю між датами, ми можемо скористатися різницею часу (`TimeDifference`) з gem `time_difference`. Наприклад:

```Ruby
require 'time_difference'

diff = TimeDifference.between(date1, date2).in_years # різниця у роках
```

## Глибока пірководження

В Ruby існує багато різних методів для порівняння дат. Наприклад, метод `asctime` повертає дату у вигляді рядка з більш детальною інформацією, включаючи час та часовий пояс.

```Ruby
date1 = Time.new(2021, 8, 10, 12, 30, 0, "+03:00")
puts date1.asctime # Виведе: Tue Aug 10 12:30:00 2021 UTC+03:00
```

Також важливо розуміти, що порівнювати дати потрібно за допомогою тих самих типів даних. Наприклад, не можна порівнювати об'єкт `Time` з об'єктом `Date`.
