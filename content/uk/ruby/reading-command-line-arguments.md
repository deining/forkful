## Що & Чому?
Зчитування аргументів командного рядка - це процес отримання даних від користувача шляхом введення параметрів під час виконання програми. Ця функція дозволяє робити програму більш динамічною та пристосованою до потреб користувача.

## Як:
```Ruby
# Приклад коду для зчитування аргументів командного рядка
input = ARGV # зберігає усі введені аргументи в масив input
input.each do |arg| # перебирає всі аргументи
  puts arg # виводить кожен аргумент на новому рядку
end
```

Приклад виконання програми зі зчитуванням аргументів командного рядка:
```
$ ruby arguments.rb hello world
hello
world
```

## Deep Dive:
(1) Зчитування аргументів командного рядка стало популярним у минулому, коли функціональність програм обмежувалась користувальницькими інтерфейсами, та користувачі не мали можливості вводити додаткові параметри для виконання програми. (2) Існують інші способи отримання даних від користувача, такі як читання з файлів або через веб-інтерфейси, але зчитування аргументів командного рядка є швидким та простим способом отримати додаткову інформацію від користувача. (3) Для зчитування аргументів командного рядка у Ruby використовується спеціальний об'єкт ARGV.
