## Для чого

Один із способів взаємодії з вашою Ruby програмою є через використання аргументів командного рядка. Завдяки цьому, ви можете передавати параметри в програму при виконанні з командного рядка, що дозволяє вам змінювати поведінку програми за потреби. У цій статті ми розберемося, як це зробити за допомогою Ruby.

## Як зробити

```ruby
# Приклад читання аргументу командного рядка по індексу
# Перший аргумент завжди є назвою програми і не враховуємо його
first_arg = ARGV[1]
puts "Перший переданий аргумент: #{first_arg}"

# Приклад читання аргументу командного рядка по ключу
# Використовуємо `Hash` для збереження ключів та значень
options = {}
# Використовуємо метод `shift` для видалення першого елементу з масиву ARGV,
# а оскільки він повертає масив, то ми можемо одразу використати
# деструктуризацію для отримання значення і індексу ключа
while (key, value = ARGV.shift(2))
    options[key] = value
end
# Тепер у нас є `Hash` з параметрами командного рядка
puts "Аргументи ключів та значень: #{options.inspect}"
```

Введення з командного рядка:
```
$ ruby command_line_arguments.rb first_arg_key first_arg_value second_arg_key second_arg_value
```

Виведення:
```
Перший переданий аргумент: first_arg_value
Аргументи ключів та значень: {"first_arg_key"=>"first_arg_value", "second_arg_key"=>"second_arg_value"}
```

## Глибша розробка

Ви можете читати аргументи командного рядка за допомогою `ARGV` масиву, який містить усі передані при виконанні програми аргументи. Зверніть увагу, що перший аргумент завжди є назвою програми і не враховується.

Також, ви можете використовувати метод `shift` для отримання і видалення першого елемента з масиву, що дозволяє зручно читати аргументи по ключу та значенню.
