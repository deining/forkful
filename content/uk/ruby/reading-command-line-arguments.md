# Чому

Перед вами стаття про роботу з аргументами командного рядка в Ruby. Якщо ви вже займаєтесь програмуванням чи хочете вивчити цю мову, то вам точно будуть потрібні знання про цю техніку. Ця функціональність дозволяє передавати дані до програми під час її виконання, що є дуже зручною для роботи з більшими об'ємами даних.

# Як

```ruby
# Приклад коду, що зчитує аргументи командного рядка
# Ви можете використовувати метод ARGV для цього
# Першим аргументом буде назва файла, другим і далі - введені дані
# Зверніть увагу на те, що дані зчитуються у вигляді стрічок

puts "Привіт, #{ARGV[1]}!" 
# Приклад виклику: ruby hello.rb Іван
# Виведе: Привіт, Іван!
```

Ви можете поєднувати цю функціональність з умовними операторами та циклами, щоб зробити свою програму більш потужною та динамічною.

# Глибші води

Зчитування аргументів командного рядка дозволяє вам не тільки отримувати дані для програми, але і змінювати її поведінку на базі цих даних. Ви можете імпортувати різні бібліотеки, щоби робити свої програми більш функціональними та зручними для користувачів.
