## Що і чому?
Генерування випадкових чисел - це процес створення чисел, які не мають певного шаблону або послідовності. Програмісти використовують цей метод для створення випадкових даних або для випадкового вибору елементів з існуючих даних.

## Як це зробити:
```Ruby
# Використовуйте метод rand для генерування випадкового числа в межах 0 і 10
puts rand(10) 
=> 4

# Щоб задати верхню межу чисел, використовуйте діапазон (..)
puts rand(1..100) 
=> 53

# Щоб генерувати випадкові булеві значення, використовуйте метод rand для об'єкта Boolean
puts rand > 0.5 
=> true
```

## Огляд методу:
Генерування випадкових чисел використовується в багатьох програмах, включаючи ігри, криптографію та статистику. Існують також альтернативні методи, які можуть бути використані для створення випадкових чисел, наприклад, використання потоків даних або спеціальних алгоритмів. У Ruby, метод rand використовує генератор псевдовипадкових чисел, який робить його практично незалежним від попередніх результатів.
