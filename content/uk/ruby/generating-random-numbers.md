## Чому?

Побудова випадкових чисел є важливим аспектом програмування, особливо для розробки ігор, створення тестів та шумового процесу у додатках. Генерація випадкових чисел дозволяє створювати непередбачувані результати при кожному запуску програми. Це дозволяє створювати більш цікаві та різноманітні програми.

## Як?

Існує кілька способів генерації випадкових чисел в Ruby, але найпростішим та найпоширенішим є використання вбудованого методу `rand`. Наприклад, для генерації випадкового цілого числа в діапазоні від 1 до 10, ми можемо написати:

```Ruby
rand(1..10)
```

Цей метод також може бути використаний для генерації випадкових дійсних чисел, вказуючи діапазон у десяткових числах:

```Ruby
rand(1.0..10.0)
```

Також існує можливість використовувати вбудований клас `Random` для більш складних алгоритмів генерації випадкових чисел, наприклад:

```Ruby
random = Random.new
random.rand(1..10)
```

Цей метод також може бути більш ефективним для генерації великої кількості випадкових чисел, оскільки не використовується глобальний генератор випадкових чисел.

## Вдаваймося у деталі

Важливо знати, що вбудований метод `rand` базується на псевдовипадковому генераторі, тобто послідовність чисел, яку він генерує, не є повністю випадковою. Вона базується на початковому числі, яке називається "зерном" (seed). Цей параметр можна вказати при виклику методу `rand`, щоб отримати відмінну послідовність чисел. Також, бажано перемішувати зерно псевдовипадкового генератора перед викликом методу `rand`, щоб запобігти повторенню послідовності чисел.

`Random` клас, який був згаданий раніше, також має метод `srand`, який дозволяє задати зерно псевдовипадковому генератору.
