## Для чого

Старт нового проєкту може бути викликом для багатьох програмістів, але це також дає можливість розвивати свої навички та створювати щось нове і цікаве. Запуск проєкту за допомогою Ruby може бути зручним та ефективним способом для досягнення своїх мет і задоволення творчого потенціалу.

## Як

Запуск нового проєкту за допомогою Ruby простий та зручний. Для початку, створіть папку для свого проєкту та відкрийте його в терміналі. Далі, введіть наступні команди:

```Ruby
gem install bundler # встановлює gem, який дозволяє керувати залежностями вашого проєкту
bundle init # створює файл Gemfile для керування залежностями
```

Далі, відредагуйте файл Gemfile та додайте геми, які вам потрібні для вашого проєкту. Наприклад:

```Ruby
source 'https://rubygems.org'
gem 'sinatra' # гем для створення веб-додатків
gem 'pry' # гем для відлагодження коду
```

Після цього, встановіть геми за допомогою команди `bundle install` та починайте писати свій код. Нижче наведено приклад створення простого веб-додатку з використанням гему Sinatra:

```Ruby
require 'sinatra'

get '/' do # опрацьовує GET запит на головну сторінку
  'Привіт, це мій перший веб-додаток на Ruby!'
end
```

Після того, як код написаний, запустіть свій проєкт за допомогою команди `ruby app.rb` та перейдіть за адресою `localhost:4567` у вашому браузері, щоб побачити результат.

## Глибинний погляд

При створенні нового проєкту за допомогою Ruby, деякі речі, на які варто звернути увагу, це організація залежностей вашого проєкту та написання чистого та ефективного коду. Також, важливо розуміти базові принципи ООП та знати, як використовувати різні геми та бібліотеки для досягнення своїх мет.
