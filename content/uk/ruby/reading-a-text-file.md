## Навіщо

Зчитування файлів є важливою частиною програмування в багатьох мовах, в тому числі і в Ruby. Знання як правильно читати файли дозволяє нам отримувати потрібну інформацію із зовнішніх джерел та використовувати її для розв'язання задач. Тож, якщо ви цікавитеся програмуванням або шукаєте нові знання, то дуже корисно ознайомитися з процесом читання текстових файлів у Ruby.

## Як це зробити

Інструкція по читанню текстових файлів у Ruby дуже проста. Спочатку потрібно відкрити файл, використовуючи метод `File.open` та передати йому шлях до файлу та режим відкриття. Наприклад, `File.open("file.txt", "r")` відкриє файл для читання. Потім ми можемо використати метод `read` для зчитування всього тексту з файлу, або `readline` для зчитування по одному рядку за раз. Нижче наведено приклад коду та вихідний текст файлу.

```Ruby
# Відкриваємо файл для читання
file = File.open("file.txt", "r")

# Зчитуємо весь текст з файлу та виводимо його
puts file.read

# Зчитуємо по одному рядку та виводимо їх
file = File.open("file.txt", "r")
puts file.readline
puts file.readline

# Вихідний текст файлу:
# Це перший рядок
# Це другий рядок
```

Існує також можливість зчитувати та обробляти файли по рядках за допомогою конструкції `File.foreach("file.txt")` або використати цикл `while` для зчитування файлу рядок за рядком.

## Глибоке поглиблене вивчення

Навчання читання файлів у Ruby не обмежується простим зчитуванням та виводом тексту. Існує багато методів та функцій для обробки файлів, таких як перевірка наявності чи створення нового файлу, запис в файл, редагування тексту та багато інших. Також можливо зчитувати інші типи файлів, такі як Excel або CSV, використовуючи відповідну бібліотеку.

Як бачимо, читання файлів у Ruby є важливою і необхідною навичкою для програмістів. Вивчіть цей процес більш детально та поєднайте з іншими навичками для створення потужних інструментів у ваших програмах.
