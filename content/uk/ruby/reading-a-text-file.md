# Чому

Велика частина програмного забезпечення використовує текстові файли для зберігання інформації, тому важливо знати, як читати ці файли для подальшої обробки даних. Це може бути корисно, якщо ви хочете відкрити та аналізувати загальну інформацію про ваші файли, які ви використовуєте у своїх програмах.

# Як це зробити

```Ruby
# Відкриття та читання текстового файлу
file = File.open("file.txt", "r")

# Зчитування рядків з файлу та виведення їх на екран
file.each do |line|
  puts line
end

# Закриття файлу після закінчення роботи з ним
file.close

# Виведення вмісту файлу до змінної
file_content = File.read("file.txt")
puts file_content
```

```Ruby
# Альтернативний метод для читання файлу і рядків з нього
file_lines = File.readlines("file.txt")
# Виведення першого рядка з файлу
puts file_lines[0]
# Виведення кількості рядків у файлі
puts file_lines.length
```

Ви також можете бачити, що рядки, зчитані з файлу, можна розділяти на підстроки за допомогою методу `split`:

```Ruby
# Розділення рядка на підстроки
line = "Hello, world"
substrings = line.split("o")
puts substrings # Виводить ['Hell', ', w', 'rld']
```

# Глибше

При читанні текстового файлу важливо враховувати кодування файлу, оскільки це може впливати на зчитування та обробку даних. Для цього в Ruby є спеціальні методи, такі як `File.open("file.txt", "r:UTF-8")` або `File.read("file.txt", encoding: "UTF-8")`, де ви можете вказати необхідне кодування файлу.

Також важливо пам'ятати про відкриття та закриття файлу, інакше ваша програма може залишати відкриті з'єднання з файлом, що може призвести до некоректної роботи або навіть помилки.
