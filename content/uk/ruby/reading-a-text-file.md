# Причина

Навіщо комусь займатися читанням текстового файлу? Існує багато причин, чому це може бути корисно для вас. Якщо ви програміст, це може бути необхідним для обробки великих обсягів даних або для створення звітів. Або, якщо ви просто цікавитесь програмуванням, це може бути цікавим викликом для покращення своїх навичок.

# Як це зробити

Існує кілька можливих способів, якими ви можете зчитувати текстовий файл в Ruby. Один з найпростіших - використання вбудованого методу `File.readlines()`. Перш ніж почати, необхідно відкрити файл, використовуючи метод `File.open()`. Тут ми використаємо метод `each` для ітерації через кожен рядок у файлі та виведення його вмісту на екран:

```Ruby
file = File.open("sample.txt")
file.readlines.each do |line|
  puts line
end
```

Уявіть, що у нас є текстовий файл з наступним вмістом:

```
Hello,
I am a Ruby programmer!
```

В результаті виконання цього коду, ми отримаємо наступний вивід:

```
Hello,
I am a Ruby programmer!
```

Також є інші методи для зчитування текстового файлу, такі як `File.read()` або `IO.foreach()`, які можуть бути корисні в різних ситуаціях. Не бійтеся експериментувати та шукати найбільш підходящий для вас варіант.

# Глибокий погляд

Як ви, можливо, вже помітили, у вихідному виводі були присутні не тільки текстові рядки, але й знаки нового рядка. Це сталося через те, що метод `File.readlines()` автоматично додає знак нового рядка після кожного прочитаного рядка у файлі.

Щоб уникнути цього, ми можемо використовувати метод `chomp`, щоб видалити знак нового рядка з кожного рядка у файлі:

```Ruby
file = File.open("sample.txt")
file.readlines.each do |line|
  puts line.chomp
end
```

В результаті ми отримаємо такий же вихідний текст, але без знаків нового рядка.
