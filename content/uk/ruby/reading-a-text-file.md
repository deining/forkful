## Що & Навіщо?
Читання текстового файлу - це процес, коли програма або скрипт зчитує текстовий файл і виводить його вміст. Програмісти читають текстові файли, щоб отримати доступ до даних, зробити обробку або виконати інші дії з цими даними.

## Як це зробити:
```Ruby
# Відкриття файла для читання
file = File.open("file.txt", "r")

# Зчитування вмісту файла
contents = file.read

# Виведення вмісту
puts contents

# Закриття файла після завершення
file.close
```

Вивід:
`Це вміст мого текстового файлу`

## Глибока занурення:
**Iсторичний контекст:**
Читання текстових файлів було одним з перших способів обробки даних в програмуванні. До них кожна дана мова програмування мала свої власні методи та функції для роботи з файлами.

**Альтернативи:**
Існує кілька альтернативних способів для читання текстових файлів в Ruby, таких як використання готових бібліотек або використання різних методів читання вмісту з файлу. 

**Деталі реалізації:**
У Ruby, для читання текстових файлів використовується метод `File.open` з параметром `"r"`, що позначає режим читання. Після цього використовується метод `read` для отримання вмісту, а `puts` використовується для виведення його на екран. Після завершення роботи з файлом, його необхідно закрити за допомогою методу `close`.
