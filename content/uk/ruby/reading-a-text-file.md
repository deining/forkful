## Чому
Цей блог пост призначений для тих, хто хоче навчитися читати текстовий файл за допомогою Ruby програмування. Якщо ви зацікавлені в програмуванні та хочете дізнатися більше про роботу з файлами, то цей пост для вас.

## Як
Використовуючи Ruby, ми можемо легко читати текстовий файл та отримувати доступ до його вмісту. Ось приклад коду:

```Ruby
# Відкриваємо текстовий файл
file = File.open("myfile.txt")

# Читаємо всі рядки та додаємо їх до масиву
lines = file.readlines

# Виводимо вміст файлу
puts lines

# Закриваємо файл
file.close
```

В результаті ми отримаємо вміст файлу, який міститься в змінній `lines`.

## Глибинний аналіз
Крім простого читання, ми також можемо використовувати методи для роботи з текстовим файлом. Наприклад, таким чином ми можемо зчитувати лише певну кількість символів:

```Ruby
# Відкриваємо текстовий файл
file = File.open("myfile.txt")

# Задаємо кількість символів, які хочемо прочитати
num_chars = 20

# Читаємо вказану кількість символів
content = file.read(num_chars)

# Виводимо результат
puts content

# Закриваємо файл
file.close
```

Також ми можемо використовувати метод `each_line` для обробки кожного рядку окремо:

```Ruby
# Відкриваємо текстовий файл
file = File.open("myfile.txt")

# Використовуємо метод each_line
file.each_line do |line|
  # Обробляємо кожен рядок окремо
  puts "Рядок: #{line}"
end

# Закриваємо файл
file.close
```
