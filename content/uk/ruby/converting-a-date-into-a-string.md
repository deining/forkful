# Чому
Конвертація дати у рядок є важливою частиною програмування, оскільки дати є неодмінною частиною багатьох програм та додатків. Це дає можливість користувачам відображати та зберігати дати у зручному для них форматі.

## Як це зробити
Приклади коду та його вивід з'являться у блоці ````Ruby ... ````, щоб продемонструвати, як легко можна конвертувати дату у рядок в Ruby.

````Ruby
# Об'єкт дати
date = Date.today

# Конвертація до рядкового представлення
date.to_s                    # виводить "2021-10-16"
date.strftime("%d/%m/%Y")    # виводить "16/10/2021"
date.strftime("%B %d, %Y")   # виводить "October 16, 2021"
````

Як бачите, це досить просто. Використовуючи метод `to_s` або метод `strftime` зі спеціальними форматами, ми можемо отримати рядкове значення дати у потрібному нам форматі.

## Глибока занурення
У Ruby немає окремого типу для дати, вона представлена як об'єкт класу `Date`. Для конвертації дати у рядок, Ruby використовує `to_s` метод, що повертає рядкове представлення дати у форматі `"yyyy-mm-dd"`. Також можна використовувати метод `strftime`, щоб вказати певний формат, за допомогою спеціальних літер, які відповідають даті та часу.

Крім того, у Ruby є багато сторонніх бібліотек для роботи з датами, таких як `Chronic` та `Timecop`, які надають більше можливостей та функціональності при роботі з датами.
