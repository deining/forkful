## Чому

В програмуванні часто потрібно обчислювати дату в майбутньому або в минулому. Це може знадобитися для створення звітів, розрахунку термінів, або для інших додаткових функціональностей.

## Як

Один з способів обчислення дати в майбутньому або в минулому - використання вбудованих методів в Ruby. Наприклад, для отримання дати завтра можна скористатися методом `Date.tomorrow`. Також, можна використовувати математичні операції для обчислення дати в майбутньому або в минулому. Наприклад, для отримання дати, яка відповідає 3 дням до поточної можна використати вираз `Date.today - 3`.

```
Пример 1: обчислення дати завтра
```ruby
require 'date'

puts Date.tomorrow
```

В результаті отримаємо:

```
2020-04-08
```

```
Пример 2: обчислення дати, що відповідає 3 дням до поточної
```ruby
require 'date'

puts Date.today - 3
```

В результаті отримаємо:

```
2020-04-05
```

## Deep Dive

Для обчислення дати в майбутньому або в минулому можна використовувати різні методи та підходи. Наприклад, можна використовувати бібліотеки, які дозволяють виконувати більш складні обчислення, або створювати власні методи для зручного обчислення дати.

Також, варто пам'ятати про формат дати та часу, який використовується в програмі. Навіть невеликий пошкоджений формат може призвести до помилки під час обчислення дати в майбутньому або в минулому.
