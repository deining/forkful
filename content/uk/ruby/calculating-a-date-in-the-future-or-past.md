## Чому

Розрахунок дати в майбутньому або минулому може бути корисним для планування подій, наприклад, визначення терміну дії договору або поділу ресурсів на певний час.

## Як це зробити

Найпростіший спосіб розрахувати дату в майбутньому або минулому - використати вбудовані методи `Date` в Ruby.

```Ruby
require 'date'

# Розрахунок дати через 2 тижні в майбутньому
future_date = Date.today + 14
puts "Дата за 2 тижні: #{future_date}"

# Розрахунок дати 1 рік тому
past_date = Date.today - 365
puts "Дата 1 рік тому: #{past_date}"
```

В результаті отримаємо відповідні дати з правильним форматуванням.

```
Дата за 2 тижні: 2021-04-07
Дата 1 рік тому: 2020-03-10
```

Можна також використовувати методи `Date.parse()` та `Date.new()` для створення дати зі строкового або числового значення.

```Ruby
# Розрахунок дати по строковому значенню
date_string = "2021/05/25"
future_date = Date.parse(date_string)
puts "Розрахунок дати зі строкового значення: #{future_date}"

# Розрахунок дати за допомогою числового значення
past_date = Date.new(2000, 1, 1)
puts "Розрахунок дати за допомогою числового значення: #{past_date}"
```

```
Розрахунок дати зі строкового значення: 2021-05-25
Розрахунок дати за допомогою числового значення: 2000-01-01
```

## Глибоке занурення

У Ruby є багато інших методів для роботи з датами, наприклад, методи для перевірки валідності дати, обчислення різниці між двома датами та інші. Також можна використовувати геми, які надають додаткові функції для роботи з датами.
