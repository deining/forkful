## Чому

Друкування відлагоджуючих повідомлень є важливою частиною процесу написання програм на Ruby. Воно дозволяє використовувати науковий метод для знаходження та виправлення помилок у коді.

## Як

```Ruby
# Приклад коду для друкування повідомлення зі змінною
age = 25
puts "Я думаю, що вам #{25} років."
```

Використовуючи `puts`, ми можемо перевірити значення змінної `age` та переконатися, що воно відповідає очікуваному.

```Ruby
# Приклад коду для друкування повідомлення з використанням прапорця функції print
name = "Олена"
print "Привіт, #{name}!" # з функцією print, повідомлення не буде переведено на новий рядок
```

Використання прапорця `print` дає нам змогу друкувати повідомлення покроково, що дозволяє нам слідкувати за значеннями змінних в реальному часі.

## Глибока занурення

Друкування відлагоджуючих повідомлень та використання наукового методу є важливими навичками для розробників на Ruby. Існує багато інших методів друкування, таких як `p` та `debug`, які також можуть допомогти при відлагодженні програм.
