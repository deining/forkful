## Why

У програмуванні дуже важливо знати, що відбувається під час виконання нашого коду. Часто ми зіштовхуємося з проблемами, які нам треба виправити, або просто хочемо побачити, як працює наш код. Для цього і призначене виведення відладочної інформації.

## How To

Для виведення відладочної інформації використовуються методи `puts`, `p` або `print` у Ruby. Давайте розглянемо кожен з них на прикладі простої програми.

```Ruby
# Приклад програми, що виводить відладочну інформацію
def multiply(a, b)
  # Виведемо наші вхідні дані
  puts "Вхідні дані: #{a}, #{b}"

  # Результат множення
  result = a * b

  # Виведемо результат
  puts "Результат: #{result}"

  # Повертаємо результат функції
  return result
end

# Виклик функції
multiply(5, 10)
```

У цьому прикладі ми використали метод `puts` для виведення повідомлень зі значеннями змінних. Можна також використати метод `p` для виведення значень зі змінними або метод `print` для виведення повідомлення без переходу на новий рядок.

## Deep Dive

Для більш розширеного аналізу нашого коду, можна використовувати відладчик у Ruby. Він дозволяє встановлювати точки зупинки на певних рядках коду і аналізувати значення змінних на кожному кроці виконання. Це дозволяє нам більш детально досліджувати проблеми у нашому коді і знайти їх рішення.

Навчатися використовувати відладчик можна, наприклад, у вступних курсах до Ruby або використовуючи різноманітні онлайн ресурси та книги.
