## Тому
Існує багато ситуацій, коли потрібно привести дані в однорідний формат. Конвертування рядка в нижній регістр - це один з способів цього зробити. Наприклад, для порівняння чи сортування символів.

## Як
```ruby
str = "Привіт"
puts str.downcase
#=> привіт
```

Або, якщо потрібно замінити всі великі букви на малі в рядку:

```ruby
str = "ТЕКСТ В ВЕЛИКИХ ЛІТЕРАХ"
puts str.downcase
#=> текст в великих літерах
```

## Глибше копання
Конвертування в нижній регістр використовує метод `downcase` для рядка. Це метод, який доступний для всіх об'єктів класу String. Він перетворює всі великі літери в рядку на відповідні нижні, і повертає змінений рядок як результат. Також важливо пам'ятати про мову, яку використовується, оскільки деякі літери можуть мати різний вид у нижньому регістрі.
