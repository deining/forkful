# Чому

Перетворення рядка на рядок з малих літер є важливою функцією у Ruby для ефективної обробки текстових даних. Воно також може полегшити порівняння рядків, так як регістр не впливає на результат.

## Як

```ruby
# Перетворити рядок на рядок з малих літер
string = "Привіт, Світ!"
lowercase = string.downcase

puts lowercase
# виведе "привіт, світ!" 
```

Просто використовуйте метод `.downcase` для об'єкта рядка, який містить вихідний текст. Цей метод створить новий рядок з малих літер і поверне його як результат.

## Глибоке занурення

Під час перетворення рядка на рядок з малих літер, Ruby використовує стандарт ASCII алфавіту для визначення великих і малих літер. Це означає, що будь-які символи, які не належать до ASCII алфавіту, залишаться без змін.

Також, метод `.downcase` не змінює оригінальний об'єкт рядка, а повертає новий об'єкт. Тому, якщо ви хочете зберегти зміни, ви повинні присвоїти результат перетворення змінній.
