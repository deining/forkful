## Чому

У програмуванні, часто доводиться з'єднувати різні рядки тексту, щоб створити більш складний варіант. Це може бути корисно для створення повідомлень, URL-адрес або будь-яких інших розділів тексту. Використання з'єднання рядків допомагає зберегти час і зусилля, а також робить програму більш читабельною.

## Як

З'єднання рядків в Ruby дуже просте. Існує кілька способів, якими можна це зручно зробити. Перший спосіб - використання оператора "+" для поєднання двох рядків. Ось приклад:

```Ruby
hello = "Привіт"
name = "Олег"
greeting = hello + " " + name
puts greeting # Виведе "Привіт Олег"
```

Інший спосіб - використання методу `concat` для з'єднання рядків. Цей метод приймає в себе інші рядки і додає їх до поточного рядка. Ось приклад:

```Ruby
message = "Сьогодні "
message.concat("погода ", "дуже ", "гарна")
puts message # Виведе "Сьогодні погода дуже гарна"
```

Також, в Ruby є метод `<<`, який приймає в себе будь-який об'єкт і додає його до кінця поточного рядка. Давайте переглянемо на прикладі:

```Ruby
sentence = "Я люблю "
sentence << "програмування"
puts sentence # Виведе "Я люблю програмування"
```

## Глибокий погляд

У Ruby рядки є об'єктами і вони мають багато корисних методів для роботи з ними. Наприклад, для з'єднання рядків відповідає метод `concat`, але, також є і метод `prepend`, який додає рядок на початок поточного. Також, вам може знадобитись метод `insert`, який додає рядок в певне місце поточного рядка.

```Ruby
text = "Привіт"
text.prepend("Привіт, я ")
text.insert(10, "Oлег ")
puts text # Виведе "Привіт, я Олег Привіт"
```

Також, в Ruby є можливість використовувати форматування рядків, щоб було більш легко з'єднувати їх зі значеннями змінних.

```Ruby
number = 15
message = "Число: #{number}"
puts message # Виведе "Число: 15"
```

Варто зазначити, що при використанні форматування, тип даних значення має бути сконвертований до рядка. Тому, якщо ви маєте цілочисельне значення, то перш за все потрібно використати метод `to_s
