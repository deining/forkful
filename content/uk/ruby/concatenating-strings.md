## Чому

Найчастіша причина для використання конкатенації строк в Ruby - це створення більш складних повідомлень або текстів з використанням змінних. Ви можете додати значення змінних до інших строк, щоб створити потрібне повідомлення або повідомлення з можливістю налаштування.

## Як

Для конкатенації строк використовуються знак плюс (+) або метод `concat`. Давайте подивимось на приклад:

```ruby
name = "Іван"
age = 24
puts "Привіт, мене звуть " + name + " і мені " + age.to_s + " роки."
```

В результаті ми отримаємо повідомлення: `Привіт, мене звуть Іван і мені 24 роки.`

Також, ви можете використовувати метод `<<`, щоб додавати значення до існуючої строки. Наприклад:

```ruby
lang = "Ruby"
lang << " is a great language!"
puts lang
```

Результат буде: `Ruby is a great language!`

## Занурення

При використанні методу `concat`, Ruby насправді створює новий об'єкт строк, тому використання `<<` може бути ефективнішим з точки зору продуктивності.

Також, важливо пам'ятати, що не всі типи даних можна додавати до строк. Якщо ви спробуєте додати ціле число до строки, ви отримаєте помилку. Щоб уникнути цього, ви можете використовувати метод `to_s` для перетворення значення у строку.

Тепер ви знаєте, як конкатенувати строки в Ruby та які існують нюанси. Використовуйте ці знання для створення більш гнучких повідомлень у ваших програмах!
