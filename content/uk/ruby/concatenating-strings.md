## Чому

Конкатенація рядків є важливим аспектом програмування, оскільки вона дозволяє з'єднувати різні рядки у більш складені і корисні рядки. Вона широко використовується для створення повідомлень, шаблонів електронної пошти та багатьох інших корисних функцій. Це допоможе покращити ефективність вашого коду та заощадити час під час програмування.

## Як

Для початку, нам потрібно створити змінні, які ми хочемо конкатенувати в один рядок. Наприклад:

```Ruby
first_name = "Марія"
last_name = "Петренко"
```

Тепер, ми можемо використовувати оператор `+`, щоб з'єднати ці змінні разом:

```Ruby
full_name = first_name + " " + last_name
puts full_name
```

Цей код виведе наступне:

```
Марія Петренко
```

Можна також використовувати метод `concat`, який дозволяє додавати рядки у кінець існуючого рядка:

```Ruby
greeting = "Привіт!"
greeting.concat(" Як справи?")
puts greeting
```

Результат буде наступним:

```Привіт! Як справи?```

Окрім цього, ми також можемо використовувати метод `<<`, який додає рядок у кінець існуючого рядка, або метод `insert`, який дозволяє додавати рядки у будь-яке місце існуючого рядка. Наприклад:

```Ruby
greeting = "Привіт!"
greeting << " Як справи?"
puts greeting
# Результат буде такий самий
greeting.insert(8, " Як справи?")
puts greeting
```

Важливо пам'ятати, що коли ми конкатенуємо рядки із числовими змінними, числа будуть автоматично перетворюватися у рядки. Наприклад:

```Ruby
age = 25
puts "Я " + age + " років"
```

З цим кодом буде помилка, оскільки числа не можуть бути з'єднані з рядками. Щоб цього уникнути, нам потрібно перетворити числові змінні у рядки за допомогою методу `to_s`:

```Ruby
age = 25
puts "Я " + age.to_s + " років"
```
