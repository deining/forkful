## Why
Довідкова стаття для українських користувачів про з'єднання рядків в Ruby.

Цей процес дозволяє об'єднати два рядки в один, що може бути корисним для створення більш складних рядків або відображення даних у зручному форматі.

## How To
Давайте подивимося на приклади коду, як з'єднати рядки в Ruby:

```Ruby
# приклад 1
first_name = "Іван"
last_name = "Петренко"
full_name = first_name + " " + last_name
puts full_name
# виведе "Іван Петренко"

# приклад 2
friend_list = ["Олег", "Наталія", "Ірина"]
message = "Мої друзі: " + friend_list.join(", ")
puts message
# виведе "Мої друзі: Олег, Наталія, Ірина"
```

У прикладі 1 ми з'єднали два рядки і призначили результат новій змінній `full_name`. У прикладі 2 ми використали метод `.join`, щоб об'єднати елементи масиву `friend_list` з вказаним роздільником ", ".

Зверніть увагу, що оператор `+` можна використовувати як для об'єднання рядків, так і для додавання чисел. Тому важливо переконатися, що всі операнди є типом даних String.

## Deep Dive
Надалі, якщо ви хочете глибше дослідити роботу з'єднання рядків в Ruby, варто звернути увагу на такі моменти:

- Використання методу `.concat`: цей метод з'єднує два рядки, але змінює вихідний рядок, тоді як метод `+` повертає новий з'єднаний рядок.
- Використання вбудованої змінної `#{}` для вставки значення змінної в середину рядка. Наприклад: `full_name = "Іван #{last_name}"` поверне рядок "Іван Петренко".
- Застосування методу `.strip` для видалення зайвих пробілів з початку і кінця рядка. Це допоможе уникнути непотрібних прогалин у з'єднаному рядку.

Не бійтеся експериментувати з різними методами та способами з'єднання рядків, щоб знайти найбільш ефективний для своєї потреби.
