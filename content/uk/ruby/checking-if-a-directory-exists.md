## Для чого

Перевірка існування директорії є важливим кроком у програмуванні на Ruby. Ця функція дозволяє перевірити, чи існує директорія за заданим шляхом, що є корисним при роботі з файлами та папками.

## Як

```Ruby
if File.directory?('/home/username/Documents')
  puts "Директорія існує"
else
  puts "Директорія не існує"
end
```

В цьому прикладі ми використовуємо метод `directory?` з класу `File` для перевірки існування директорії за шляхом `'/home/username/Documents'`. Якщо директорія існує, то на екран виведеться повідомлення "Директорія існує", в протилежному випадку - "Директорія не існує". 

## Глибшого пір'я

Для того, щоб розуміти, як працює перевірка існування директорії, корисно знати про клас `File` та його методи. Клас `File` є частиною стандартної бібліотеки Ruby і містить різноманітні методи для роботи з файлами та директоріями. Метод `directory?` перевіряє, чи існує відповідний запис у файловій системі і повертає значення `true` або `false`.
