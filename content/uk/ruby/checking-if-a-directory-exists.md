## Що і чому?

Перевірка існування каталогу - це процес, який дозволяє програмі отримати інформацію про існування певного каталогу на комп'ютері. Це дозволяє програмі робити різні операції, залежно від того, чи існує даний каталог чи ні. Програмісти використовують цю перевірку для забезпечення зручності та безпеки своїх програм.

## Як це зробити?

Перевірити існування каталогу в Ruby дуже просто. Наслідуючи цей приклад, ви можете швидко реалізувати цю функцію у своїй програмі.

```ruby
if Dir.exist?("/path/to/directory")
  puts "Даний каталог існує."
else
  puts "Даний каталог не існує."
end
```

Ви також можете використати метод `Dir.exists?`, який поверне булеве значення замість виведення повідомлення:

```ruby
if Dir.exists?("/path/to/directory")
  # Виконати потрібні операції.
end
```

## Глибокий занурення

Перевірка існування каталогу є досить старою функцією, яка була вперше додана в Ruby 1.8. Subdirectories та символьні посилання (symlinks) не враховуються при перевірці існування каталогу.

Існують також інші можливості для перевірки існування файлів та каталогів, такі як `File.exist?` та `File.directory?`. Ви можете використовувати їх замість `Dir.exist?`, якщо це відповідає потребам вашої програми.

Метод `Dir.exist?` перевіряє доступ до файлу або системного каталогу. Якщо у вас немає необхідності перевіряти файл або каталог, вам може знадобитися інший метод, наприклад, `File.readable?`, який перевіряє лише читання файлу.
