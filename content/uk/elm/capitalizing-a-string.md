## Чому

В цьому пості ми розглянемо, як можна застосовувати засоби мови Elm для автоматичного великого рядка. Якщо ви хочете поліпшити якість своїх програм, або просто дізнатися більше про Elm, ви зараз читаєте правильний пост!

## Як створити програму на Elm

Для початку нам необхідно розробити програму на мові Elm. Нижче наведений приклад коду, що демонструє, як створити програму на Elm.

```
Elm.program
    { init = init,
      update = update,
      view = view
    }
```

Якщо ви запустите цю програму, ви отримаєте вивід "Hello, world!" у веб-браузері. Далі нам необхідно розглянути, як можна застосувати мовні функції для автоматичного великих рядків.

## Глибокий розгляд

Для того, щоб автоматично великі рядки працювали в нашій програмі, нам необхідно використовувати функції, які надає мова Elm. Найпростішим способом це зробити є застосування функції `toUpper`, яка перетворює всі символи великими літерами. Нижче наведений приклад коду, який демонструє цей процес:

```
String.toUpper "hello" 
```

Цей приклад виведе у веб-браузері "HELLO". Також, мова Elm надає інші корисні функції, які можна використовувати для автоматично великого рядка. Необхідно бути уважними при використанні цих функцій, оскільки вони можуть впливати на продуктивність програми.
