## Чому

Програмування за допомогою мови Elm є ефективним та потужним інструментом для створення веб-додатків. Одним з важливих аспектів програмування є перетворення данних - зокрема, форматування тексту. Капіталізація рядка є одним з таких перетворень, яке може бути корисним при розробці додатків.

## Як

Для того, щоб капіталізувати рядок в Elm, необхідно використовувати вбудовану функцію `String.toUpper`. Наприклад, якщо ми маємо рядок `"привіт"`, то після застосування цієї функції до нього отримаємо `"ПРИВІТ"`.

```Elm
import String

String.toUpper "привіт" -- результат: "ПРИВІТ"
```

## Глибокий занурення

Крім вбудованої функції `String.toUpper`, також існує можливість капіталізувати лише перший символ рядка за допомогою функції `String.capitalize`. Результатом її застосування до рядка `"привіт"` буде `"Привіт"`. Також, варто зазначити, що Elm автоматично приводить рядки до їхнього правильного регістру при порівнянні, тому окреме форматування може бути необхідне лише в окремих випадках.
