## Чому

У сучасному світі написання програмного коду перш за все має бути ефективним та безпечним. Переконатися, що певна директорія існує, може допомогти уникнути падіння програми та небезпечних помилок. У цій статті ми розглянемо як перевірити наявність директорії за допомогою мови програмування Elm.

## Як

Для початку нам потрібно імпортувати модуль `File` який дозволяє отримувати інформацію про файлову систему.
```Elm
import File
```

Далі, нам потрібно визначити шлях до директорії, яку ми плануємо перевірити.
```Elm
directoryPath : String
directoryPath = "foo/bar"
```

Тепер за допомогою функції `directoryExists` з модуля `File` ми можемо перевірити чи існує ця директорія та отримати результат у вигляді `Bool` значення.
```Elm
doesExist : Bool
doesExist = File.directoryExists directoryPath
```

Якщо результат дорівнює `True`, це означає що директорія існує. Інакше, якщо результат дорівнює `False`, це означає що директорії не існує.

## Глибинний аналіз

Унаслідок використання функцій з модуля `File`, стає можливим отримання не лише інформації про наявність директорії, але і іншої корисної інформації, такої як розмір та дата створення. Для отримання цих даних можна використовувати наступні функції:

- `size` - отримання розміру директорії у байтах
- `dateCreated` - отримання дати створення директорії

Також можливо перевірити наявність вмісту в директорії за допомогою функції `listContents`, яка повертає список усіх файлів та піддиректорій, які знаходяться у заданій директорії.
