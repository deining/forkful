## Чому: Чому вам варто прийняти участь у конвертації дати в рядок.

Конвертація дати в рядок може бути корисною у випадках, коли ви хочете відображати дати в зрозумілному форматі для користувачів вашого сайту або додатка. Це може бути зручно для створення календарів, планування подій або просто для зручності презентації дат в зрозумілій формі.

## Як: Приклади коду та вихідні дані у розділах "``` Elm ... ```".

```Elm
import Html
import Time

-- Конвертація дати у рядок з використанням Time.format
formatDate : Time.Posix -> String
formatDate date =
    Time.format "%d.%m.%Y" date
```

У цьому прикладі ми використовуємо функцію `formatDate` для конвертації дати у рядок у форматі "день.місяць.рік". Коли ми викликаємо цю функцію з датою `Time.now`, ми отримуємо такий результат: `"14.03.2021"`. Ви можете змінити формат, змінюючи рядок у функції `format`.

```Elm
import Html
import Time

-- Конвертація дати у рядок з використанням Time.formatIso
formatDate : Time.Posix -> String
formatDate date =
    Time.formatIso date
```

У цьому прикладі ми використовуємо функцію `Time.formatIso`, яка конвертує дату в рядок у форматі ISO. Коли ми викликаємо цю функцію з датою `Time.now`, ми отримуємо такий результат: `"2021-03-14T00:00:00.000Z"`. Ви можете змінити формат, змінюючи параметри у функції.

## Глибоке занурення: Поглиблена інформація про конвертацію дати в рядок.

У Elm, дати представлені як значення типу `Time.Posix`, яке відповідає кількості мілісекунд, що пройшли з 1 січня 1970 року. Щоб конвертувати це значення у зрозумілий для користувачів рядок, нам потрібно використати функції пакету `Time`. В цьому пакеті є кілька функцій, які можуть бути корисними для конвертації дати у рядок, включаючи `format`, `formatIso` та `fromTime`.

`format` дозволяє нам вказати шаблон для форматування дати у рядок. Шаблон може містити символи, які будуть замінені даними з дати, наприклад `%Y` для року і `%d` для дня. `formatIso` генерує рядок у форматі ISO, який використовується для збереження дат у база
