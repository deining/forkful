## Чому
Написання програм на Elm є захопливим та цікавим процесом, але що ж такого можна зробити з текстовими файлами? Розберемося!

## Як
Перш за все, щоб прочитати текстовий файл за допомогою Elm, вам потрібно використати модуль `File`. Спочатку необхідно імпортувати цей модуль у вашому коді:

```Elm
import File exposing (readAsString)
```

Потім ви можете використати функцію `readAsString`, яка приймає шлях до файлу та повертає результат у вигляді `Task` об'єкта. Наприклад, якщо ви хочете прочитати файл з назвою "input.txt", ваш код буде виглядати так:

```Elm
Task.perform readHandler (readAsString "input.txt")
```

Де `readHandler` - це функція, яка викликається після завершення читання файлу та приймає на вхід результат.

```Elm
readHandler : Result File.Error String -> Cmd msg
readHandler result =
    case result of
        Ok text ->
            -- робота зі зчитаним текстом
            
        Err error ->
            -- обробка помилки
```

Отриманий текст з зчитаного файлу буде переданий у параметр `Ok` функції `readHandler`.

## Занурення у глибини
Тепер, коли ми знаємо, як прочитати текстовий файл, давайте трохи поглибимось у цю тему. Функція `readAsString` повертає `Task` об'єкт, а не просто змінну типу `String`. Це означає, що ця операція виконується асинхронно та може зайняти деякий час, тому ми повинні мати на увазі цей факт при роботі з цим результатом.

Також варто зауважити, що функція `readAsString` приймає аргумент типу `File` та не вплине на дійсний файл у вашій операційній системі. Вона просто читає його вміст та повертає результат.
