## Як це зробити

Щоб прочитати текстовий файл у програмі на Elm, вам знадобиться використати модуль `File` та його функцію `readFile`. Нижче наведений приклад коду для відкриття та зчитування файлу з назвою "example.txt" та виведення змісту на екран:

```Elm
module Main exposing (main)

import File exposing (readFile)

main =
  readFile "example.txt"
    |> doSomethingWithFile


doSomethingWithFile : Maybe String -> Cmd msg
doSomethingWithFile maybeString =
  case maybeString of
    Just content ->
      content |> Debug.log "File content:"

    Nothing ->
      Debug.log "File not found"
```

Відкривши цей код у браузері, ви повинні побачити виведений у `Debug` консоль зміст файлу, якщо він був успішно зчитаний. Якщо ж файл не знайдений, виведеться повідомлення про помилку.

## Під детальнішу лупу

Щоб розуміти, як саме відбувається зчитування текстового файлу у Elm, потрібно розібратися з функціями модулю `File`. Основною функцією є `readFile`, яка приймає шлях до файлу та повертає `Cmd msg`, що представляє собою команду, яку необхідно виконати для зчитування файлу.

Після того, як команда була виконана, результат буде доступний у функції-операторі `|>`, за допомогою якого ми можемо передати його у іншу функцію для обробки. У нашому прикладі, ми використовуємо функцію `Debug.log` для виведення результату в консоль. Зверніть увагу, що результатом зчитування файлу є `Maybe String`, тому необхідно обробити обидві можливі ситуації: коли файл успішно зчитується, а також коли немає такого файлу.
