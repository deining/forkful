## Що & Чому?
Створення тимчасового файлу - це процес створення файлу, який існуватиме тимчасово і буде використовуватися для певної задачі. Програмісти часто створюють тимчасові файли, щоб зберегти проміжні дані або виконати певну операцію.

## Як це зробити:
Щоб створити тимчасовий файл в Elm, вам потрібно використовувати модуль `File.System` та функцію `openTemp`, як показано нижче. Виберіть шлях для вашого тимчасового файлу та запустіть програму, щоб створити його.

```Elm
import File.System
import Basics exposing (..)

path = "/tmp/temp_file.txt"

main = 
    openTemp path
        |> andThen fileHandler
        |> Task.attempt SetMessage

fileHandler info =
    case info of
        Ok tempFile ->
            tempFile |> toString |> SetMessage
        Err error ->
            error |> toString |> SetMessage
```

Ви повинні побачити результати виконання програми у вигляді повідомлення з шляхом до створеного тимчасового файлу або повідомлення про помилку.

## Заглиблення:
Історичний контекст: створення тимчасових файлів використовується в програмуванні з давніх часів для збереження тимчасових даних або для виконання деяких операцій. Існують інші способи досягнення того ж результату, наприклад, використання пам'яті чи баз даних, але створення тимчасових файлів залишається популярним і часто використовується.

Альтернативи: окрім використання `File.System` модуля в Elm, можна створити тимчасовий файл за допомогою засобів операційної системи, наприклад, команди `mktemp` в UNIX системах або методу `createTempFile` в Java.

Деталі реалізації: функція `openTemp` потребує шляху для тимчасового файлу та повертає задачу (Task), яка може бути виконана за допомогою функції `andThen` для отримання об'єкта `TempFile` або повернення об'єкта `File.Error`. Об'єкт `TempFile` представляє сам тимчасовий файл та надає доступ до його властивостей та методів.
