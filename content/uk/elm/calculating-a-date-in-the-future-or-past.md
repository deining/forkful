# Чому

В програмуванні багато разів нам потрібно обчислювати дати в майбутньому або минулому. Це може бути корисно для створення розкладів, збереження журналів, відстеження терміну до платежів або для будь-яких інших додатків, що потребують роботи з часом. В цій статті я розповім про особливості обчислення дат в мові Elm, щоб допомогти вам зробити свій код більш ефективним та зрозумілим.

# Як

Перш за все, давайте подивимося, як обчислити дату в майбутньому в Elm. Для цього нам знадобиться вбудована функція `add`, яка дозволяє додавати певну кількість одиниць до дати. Ось приклад:

```Elm
add (Days 5) (Date.now) -- виведе дату через 5 днів
```

Також ми можемо використати функцію `subtract`, щоб відняти певну кількість одиниць від дати. Наприклад:

```Elm
subtract (Months 2) (Date.now) -- виведе дату 2 місяці тому
```

Для зручності можна створити свої функції, які будуть приймати не тільки дату, але й кількість одиниць, щоб не повторювати код кожен раз. Наприклад:

```Elm
addDays : Int -> Date -> Date
addDays days date =
  add (Days days) date
  
addDays 10 (Date.now) -- виведе дату через 10 днів
```

Для більш складних обчислень, наприклад, дати через рік та півмісяця, ми можемо використати функцію `advancedBy`, яка приймає список одиниць і значень, які потрібно додати. Ось приклад:

```Elm
add (advancedBy [ Years 1, HalfMonths 1 ] (Date.now)) -- виведе дату через рік та півмісяця
```
