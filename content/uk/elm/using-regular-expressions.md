## Чому

Регулярні вирази є потужним інструментом у програмуванні. Використання їх дозволяє швидко та ефективно виконувати пошук та заміну символів у тексті. Вони особливо корисні для обробки великих об'ємів даних, таких як логи або користувацькі вводи, а також для перевірки правильності введеного користувачем. Якщо ви ще не використовували регулярні вирази у своєму коді, то цей пост допоможе вам розібратися з їх основами та почати їх застосовувати в своїх проектах.

## Як

Існує багато форматів регулярних виразів, але ми будемо використовувати формат, який підтримується у бібліотеці Elm. Щоб почати, давайте створимо новий проект у Elm та додамо бібліотеку `elm/regex` до нашого `elm.json` файлу. Після цього ми можемо імпортувати бібліотеку за допомогою оператора `as Regex`, що дозволяє нам викликати її методи за допомогою префіксу `Regex.`.

Для створення регулярного виразу, ми використовуємо функцію `Regex.regex`, яка приймає два аргументи - рядок з регулярним виразом та регулярним флагом. Наприклад, якщо ми хочемо знайти всі голосні букви у рядку `Hello World`, ми можемо використати наступний регулярний вираз: `Regex.regex "[aeiou]"`.

```Elm
import Regex exposing (..)

findAllVowels : String -> List String
findAllVowels str =
    let
        regex = Regex.regex "[aeiou]"
        matches = Regex.findMatches regex str
    in
        List.map Regex.match matches |> List.map Tuple.first

findAllVowels "Hello World" -- ["e", "o", "o"]
```

## Глибокий занурення

Регулярні вирази можуть бути дуже потужним інструментом, але вони також вимагають знань та експертизи для ефективного використання. На додаток, вони можуть бути дуже заплутаними та складними для читання, особливо у випадку складних виразів. Якщо ви збираєтесь активно використовувати регулярні вирази у своїх проектах, рекомендую ознайомитися з документацією бібліотеки `elm/regex` та вивчати приклади використання на практиці.
