## Для чого

В програмуванні інколи необхідно створювати тестові файли, щоб перевірити правильність роботи програми. Також, це може бути корисно для створення звітів або баз даних.

## Як це зробити

```Elm
main =
  writeFile "test.txt" "Привіт Elm!"
```

У цьому прикладі ми використовуємо функцію `writeFile` для створення файлу з назвою "test.txt" і записуємо в нього рядок "Привіт Elm!".

Ви також можете використовувати цю функцію для запису даних змінної в файл. Наприклад, якщо ми маємо змінну `message` зі значенням "Це повідомлення з Elm", то ми можемо записати його в файл так:

```Elm
main =
  let
    message = "Це повідомлення з Elm"
  in
    writeFile "message.txt" message
```

Після запуску програми, ви зможете знайти створений файл `message.txt` зі змістом "Це повідомлення з Elm".

## Глибше копаємо

Щоб створювати більш складні текстові файли, ви можете використовувати бібліотеку `elm/file`. Вона надає більш широкі можливості для створення, читання і редагування файлів.

Також, за допомогою цієї бібліотеки ви можете працювати з різними форматами файлів, такими як CSV, JSON, XML та інші.
