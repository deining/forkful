## Чому

Часто під час програмування вам може знадобитись видалити певні символи зі свого коду. Наприклад, якщо ви працюєте зі строками, ви можете хотіти видалити всі пропуски або символи певного типу. У цьому випадку, видалення символів, які задовольняють певному шаблону, може бути дуже корисним і зменшити час та зусилля, потрібні для розробки програми.

## Як це зробити

Існує кілька способів видалення символів, які задовольняють певному шаблону, в Elm. Нам буде потрібна функція, яка приймає на вхід строку і повертає видалений результат. Один із способів - використання вбудованої функції `String.filter`, яка пройде по кожному символу строки та видалить ті, які задовольняють певному умові. Давайте подивимося на приклад:

```Elm
-- функція, яка видаляє пробіли зі строки
removeSpaces : String -> String
removeSpaces str =
    String.filter (\char -> char /= " ") str

-- приклад виклику функції
removeSpaces "Hello World"
-- вивід: "HelloWorld"
```

Зверніть увагу, що ми використовуємо анонімну функцію `\char -> char /= " "`, яка перевіряє кожен символ та видаляє лише ті, які не є пробілами.

## Глибоке поглиблення

Надалі, ви можете розширити цю функцію та додати параметри, які дозволяють видаляти символи будь-якого типу. Наприклад, ви можете створити функцію, яка видаляє всі числа зі строки:

```Elm
-- функція, яка видаляє числа зі строки
removeNumbers : String -> String
removeNumbers str =
    String.filter (\char -> not (String.isDigit char)) str

-- приклад виклику функції
removeNumbers "Hello 123 World"
-- вивід: "Hello World"
```

Просто змінивши умову, ми тепер можемо видаляти символи будь-якого типу. Також, можна створити функцію, яка приймає на вхід строку та шаблон, за яким слід видаляти символи. Це дає вам більший контроль над тим, які саме символи будуть видалені, щоб вони відповідали вашому конкретному випадку.
