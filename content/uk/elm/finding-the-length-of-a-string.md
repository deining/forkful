## Чому

Elm - це мова програмування, яка стала все більш популярною на останні роки, оскільки вона пропонує простий та ефективний підхід до веб-розробки. Одним із найважливіших аспектів програмування є здатність працювати з рядками, а однією з ключових задач у цьому контексті є знаходження довжини рядка. У цій статті ми розглянемо, як це можливо зробити в Elm.

## Як зробити

```Elm
stringLength : String -> Int
stringLength str = String.length str
```

Почнемо з найпростішого способу - використання вбудованої функції `length` для типу `String`. Ця функція повертає можливі довжини для будь-якого `String`, який їй передається. Зауважимо, що результат є цілим числом, оскільки функція повертає кількість символів у рядку.

```Elm
stringLength "Привіт, світе!" --> 13
```

Проте, ви можете помітити, що цей варіант не надає жодного контролю над тим, що в якості аргументу передається `String`. Тому ми можемо створити власну функцію `stringLength`, яка перевіряє тип аргументу та повертає оновлену кількість символів.

```Elm
import String exposing (length)
import Char exposing (toLower)

stringLength : String -> Int
stringLength str =
    case str of
        "" -> 0
        _ -> length (String.map toLower str)
```

У даному прикладі ми використовуємо функції `String.map` та `Char.toLower`, щоб замінити всі символи на їх нижній регістр перед обчисленням довжини. Також додано умову, що для пустого рядка результат буде 0.

## Глибока занурення

Важливо пам'ятати, що в Elm рядки - це не звичайні масиви символів, тож знаходження довжини рядка може вимагати додаткових операцій порівняно з іншими мовами програмування. Наприклад, ми можемо використати рекурсію для проходження кожного символу у рядку та підрахунку їх кількості.

```Elm
recursiveLength : String -> Int
recursiveLength str =
    case str of
        "" -> 0
        _ -> 1 + recursiveLength (String.dropLeft 1 str)
```

Функція `String.dropLeft` дозволяє нам видаляти перший символ з рядку, тож ми рекурсивно викликаємо `recursiveLength` до тих пір, поки не залишиться порожній рядок.
