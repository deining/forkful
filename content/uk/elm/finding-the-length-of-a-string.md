## Чому

Програмування - це провокація, завдання та шлях до незабутніх переживань. Завдання - це завжди нові виклики, які допомагають розширити свій кругозір та розвинути навики програмування. Наприклад, пошук довжини рядка може здатися простим завданням, але він може бути корисним при роботі з будь-якими масивами даних, що поширюється в програмуванні.

## Як

```Elm
strlength: String -> Int
strlength str =
    String.length str
```

Виведення:

```Elm
strlength "Hello World"
-- 11
```

Рядки є важливою частиною багатьох програм, тому необхідно знати, як обробляти їх у своїй роботі. У Elm для обчислення довжини рядка існує вбудована функція String.length, яка повертає кількість символів у рядку. Це зручна та ефективна функція, яка допомагає зробити обробку текстових даних легшою та швидшою.

## Глибоке занурення

Функція String.length використовується для обробки рядків будь-якої довжини, включаючи й пусті рядки. Вона також може бути використана для розробки інших корисних функцій, наприклад для перевірки максимально допустимої довжини рядка вводу користувача. Крім того, Elm має багато інших корисних функцій для роботи з рядками, таких як String.toUpper та String.toLower для перетворення рядків в верхній та нижній регістр відповідно.
