# Чому отримування поточної дати є важливим для програмістів Elm

Для багатьох програмістів, отримання поточної дати є необхідною задачею, особливо коли мова йде про роботу з часом або розкладами. У Elm, щоб отримати поточну дату, потрібно використовувати вбудовану функцію `Date.now`.

## Як отримати поточну дату в Elm

Для початку, нам потрібно імпортувати модуль `Date` в нашому коді.

```Elm
import Date exposing (now)
```

Потім, для отримання поточної дати, ми можемо застосувати функцію `now` та зберегти результат у змінну `currentDate`.

```Elm
currentDate = now
```

Щоб перевірити, чи правильно ми отримали дату, ми можемо вивести значення `currentDate` у консоль.

```Elm
main = 
    Html.text (toString currentDate)
```

Якщо запустити програму, то у консолі ми побачимо поточну дату у мілісекундах.

## Глибші занурення в отримання поточної дати

У Elm, дата представлена як тип `Time`. Це дозволяє легко працювати з датами та виконувати різні операції, такі як додавання або віднімання часу.

Крім того, можна змінити формат дати за допомогою функції `Format`. Наприклад, якщо ми хочемо відобразити дату у форматі "день.місяць.рік", то нам необхідно використати такий код:

```Elm
import Time
import Date.Format exposing (format, default)

currentDate = now

formattedDate = 
    format default "%d.%m.%Y" currentDate

main = 
    Html.text formattedDate
```

У вищезазначеному коді, ми використовуємо функцію `format` з модуля `Date.Format` та вказуємо бажаний формат, який відображатиме дату. Як результат, у консолі ми побачимо дату у форматі "день.місяць.рік".
