# Чому

У цій статті ми розглянемо важливу задачу - отримання поточної дати. Це дуже корисно для створення додатків, які пов'язані з організацією рапортів та звітів. Також це може бути використано для реалізації функції стеження за оновленнями або датами подій.

# Як

Давайте розглянемо кілька прикладів коду Elm, які допоможуть нам отримати поточну дату:

```Elm
import Time

-- Отримання поточної дати у форматі ISO 8601
Time.now |> Time.toIsoString -- приклад виводу: "2019-10-10T15:30:00.000Z"

-- Отримання поточного часу у мілісекундах з початку 1970 року
Time.now |> Time.toMillis -- приклад виводу: 1564428600000
```

# Глибоке дослідження

Все що ви бачите в коді - це використання функцій модуля `Time` для конвертації поточного часу у різні формати. Щоб краще зрозуміти, як це працює, давайте детальніше розглянемо можливості модуля `Time`.

Найважливіша функція цього модуля - це `now`, яка повертає поточний час у форматі `Posix` - це кількість секунд з початку 1970 року. Далі, всі інші функції просто конвертації цього часу у зрозуміліші форомати. Наприклад, `toMillis` конвертує у мілісекунди, а `toIsoString` - у стандартний формат дати ISO 8601.

Але не забувайте, що час залежить від часової зони, тому враховуйте це при використанні цих функцій.
