## Що & навіщо?
Отримання поточної дати є одним з базових завдань програміста. Це дозволяє використовувати актуальну інформацію для обрахунків, створення логічних перевірок та інших потрібних дій. Така функціональність особливо корисна при розробці веб-додатків, враховуючи те, що поточна дата залежить від регіону, налаштувань користувача та інших факторів.

## Як зробити:
Для отримання поточної дати в Elm потрібно використати вбудований модуль ```Time```, імпортувати його та використати функцію ```now``` з параметром ```utc```. Для цього нам необхідно створити об'єкт типу ```Cmd Msg```, щоб отримати дату у всесвітньому часі.

```
import Time exposing (utc, now)

getTime : Cmd Msg
getTime =
  now utc
```

Щоб отримати форматовану поточну дату, можна скористатися функцією ```toString``` та використати бажаний формат оформлення дати.

```
import Time exposing (utc, now)
import Time.Format exposing (format, pairs)

getTime : Cmd Msg
getTime =
  now utc

getDate : Time.Posix -> String
getDate time =
  format pairs "%d/%m/%Y" time
```

В результаті виклику функції ```getDate```, ми отримаємо поточну дату у форматі "день/місяць/рік", наприклад "24/06/2021".

## Глибше в деталі:
Історично, для отримання поточної дати використовувалися функції, що повертали час з системного годинника комп'ютера. Однак, з появою різноманітних систем та налаштувань, важко було отримати загальноприйняту поточну дату. Також, це відкривало двері до можливих помилок і проблем з точністю.

У сучасній архітектурі, для отримання поточної дати часто використовуються зовнішні сервіси, які спеціалізуються на цій функціональності, забезпечуючи більш точний і надійний результат. Також, використання функції ```format``` з декількома форматами оформлення дати допомагає зробити її більш придатною та зрозумілою для користувача.
