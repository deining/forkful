## Завдання

Написання веб-застосунків може бути складною та викликаючою задачею. Стає шляхом у світ програмування, ви можете зіткнутися з різноманіттю завдань, які необхідно вирішити. Один із них - конвертація рядка до нижнього регістру. У цій статті ми розглянемо, навіщо це потрібно та як це зробити в PHP.

## Чому

Конвертація рядка до нижнього регістру є корисною технікою у програмуванні. Це дозволяє наочно демонструвати вам різницю між великими та малими буквами. Також це дозволяє забезпечити правильну сортування та пошук даних у базі даних. Тепер давайте подивимося, як у PHP можна це зробити.

## Як це зробити

```
<?php
    $string = "ПРИВІТ ВСІМ!";

    // Конвертація до нижнього регістру
    echo strtolower($string);

    // Вивід: привіт всім!
?>
```

Ви можете використовувати вбудовану функцію PHP `strtolower()`, яка приймає рядок і повертає його у нижньому регістрі. Ця функція дуже проста і повертає новий рядок, тому вам не потрібно зберігати його у змінну.

## Deep Dive

Щоб краще зрозуміти, як працює конвертація до нижнього регістру, давайте поглянемо на реалізацію цієї функції.

```
function strtolower($string) {
    $new_string = "";
    for ($i = 0; $i < strlen($string); $i++) {
        $char = $string[$i];
        
        // Перевірка чи символ є великою літерою
        if (ord($char) >= 65 && ord($char) <= 90) {
            // Якщо так, то додаємо його у нижньому регістрі до нового рядка
            $new_string .= chr(ord($char) + 32);
        } else {
            // Якщо символ не є великою літерою, просто додаємо його до нового рядка без змін
            $new_string .= $char;
        }
    }
    return $new_string;
}
```

У цій функції ми перевіряємо кожен символ з вхідного рядка та перетворюємо його до нижнього регістру за допомогою функції `chr()`, яка приймає код символу та повертає його у вигляді символу. Таким чином, ми повертаємо новий рядок у нижньому регістрі.
