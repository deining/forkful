## Що & Чому?

Конвертування рядка до нижнього регістру означає перетворення всіх символів у рядку на нижній регістр. Це корисна операція для забезпечення стандартного форматування тексту та зручності порівняння рядків. Програмісти часто виконують цей процес для полегшення пошуку даних та уникнення помилок при порівнянні рядків.

## Як:

```PHP
$string = "ПРИВІТ СВІТ";
echo strtolower($string); // виведе "привіт світ"
```

Як показано в прикладі, для конвертування рядка до нижнього регістру використовується функція `strtolower()`. Вона повертає копію рядка, у якому всі символи були перетворені на нижній регістр. 

## Глибоке занурення:

Історичний контекст: Конвертування рядка до нижнього регістру було здійснене у багатьох мовах програмування ще з початку комп'ютерної ери. У PHP ця функція була вперше додана до версії 4.0.

Альтернативи: Існують інші функції для виконання подібної операції, наприклад `mb_strtolower()`, яка повертає результат для Unicode символів. Також можна використовувати оператор `mb_convert_case()`, який дозволяє конвертувати не лише до нижнього регістру, але і до верхнього.

Деталі реалізації: Приклад вище використовує функцію `strtolower()`, яка працює тільки з ASCII символами. Для роботи з Unicode символами рекомендується використовувати функцію `mb_strtolower()`. Якщо ж рядок містить лише ASCII символи, обидва методи будуть поводитися однаково.
