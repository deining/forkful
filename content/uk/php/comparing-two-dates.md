Що & Чому?
У програмуванні часто доводиться порівнювати дві дати - це означає перевірку того, або одна дата йде після іншої. Програмісти роблять це, щоб перевірити, чи спрацювала коректно програма, яка виконує дії в залежності від дат.

Як зробити:
```PHP
// Порівняння двох дат за допомогою оператора < (менше)
$date1 = '2020-01-06';
$date2 = '2020-01-10';
if ($date1 < $date2) {
echo "Перша дата перед другою";
}

// Порівняння двох дат за допомогою функції strtotime()
$date3 = '25 October 2020';
$date4 = '10 November 2020';
if (strtotime($date3) < strtotime($date4)) {
echo "Перша дата перед другою";
}

// Порівняння двох дат за допомогою функції DateTime()
$date5 = new DateTime('2020-12-25');
$date6 = new DateTime('2020-12-31');
if ($date5 < $date6) {
echo "Перша дата перед другою";
}
```

Глибше заходження:
Порівняння дат має довгу історію в програмуванні. У деяких старіших мовах програмування, таких як COBOL і FORTRAN, для порівняння дат використовується особливий тип даних. У PHP є кілька способів порівняння дат, які ви можете використати в залежності від вашого коду. Альтернативою для порівняння дат є використання об'єкту DateTime, який надає більше можливостей для роботи з датами, такі як додавання або віднімання певної кількості днів.
