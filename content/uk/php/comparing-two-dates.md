## Чому

У програмуванні дати є важливою частиною. Іноді нам потрібно порівняти дві дати, щоб перевірити, яка з них більша або менша. Це може бути корисно у багатьох сценаріях, наприклад, при сортуванні даних або при перевірці терміну дії документа. У цій статті ми розглянемо, як порівняти дві дати у PHP.

## Як

Для порівняння дат у PHP існує кілька способів. Розглянемо два найпоширеніших.

### Спосіб 1: Строкове порівняння

Можна порівняти дві дати, конвертувавши їх у строкове представлення та використовуючи оператори порівняння, такі як `>` та `<`. Наприклад, якщо нам потрібно порівняти дати "20.03.2021" та "25.03.2021", ми можемо написати наступний код у PHP:

```PHP
$date1 = "20.03.2021";
$date2 = "25.03.2021";

if ($date1 > $date2) {
  echo "$date1 більше за $date2";
} elseif ($date1 < $date2) {
  echo "$date1 менше за $date2";
} else {
  echo "Обидві дати рівні";
}
```

В результаті ви отримаєте повідомлення "20.03.2021 менше за 25.03.2021", оскільки строка "20.03.2021" буде вважатись меншою, ніж "25.03.2021".

### Спосіб 2: Використання функції strtotime()

Ще один спосіб порівняти дати у PHP - це використовувати вбудовану функцію `strtotime()`. Вона дозволяє конвертувати строку у дату та час у форматі Unix timestamp, який потім можна порівняти. Давайте перепишемо наш попередній приклад, використовуючи `strtotime()`:

```PHP
$date1 = strtotime("20.03.2021");
$date2 = strtotime("25.03.2021");

if ($date1 > $date2) {
  echo "$date1 більше за $date2";
} elseif ($date1 < $date2) {
  echo "$date1 менше за $date2";
} else {
  echo "Обидві дати рівні";
}
```

Отримаємо той самий результат: "20.03.2021 менше за 25.03.2021". Однак, цей спосіб може бути корисним, коли дати мають відмінний від стандарту формат, наприклад, "2021-03-20" або "03/20/2021".

## Глибокий занурення

У PHP дати можна порівнювати за допомогою інших функцій, таких як `date_diff()`, `DateTime` та `strtotime()`, які дозволяють більш гнучко працювати з датами та виконувати різні операції
