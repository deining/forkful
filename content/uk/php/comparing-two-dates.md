## Чому

В сучасному світі, де дані є надзвичайно важливими, порівняння двох дат може бути корисним завданням для багатьох програмістів. Наприклад, можна порівняти дату створення та оновлення файлу, дату оформлення замовлення тощо. Розуміння того, як правильно порівнювати дати, допоможе покращити функціональність та точність програми.

## Як це зробити

Для порівняння двох дат в PHP можна використовувати функції `strtotime()` і `date()`. Для цього необхідно конвертувати дати у формат, який можна порівняти. Наприклад, змінні `$date1` та `$date2` містять дати, які потрібно порівняти. Створимо змінну `$formatted_date1` та скористаємося функцією `strtotime()`, щоб конвертувати дату в Unix timestamp:

```PHP
$formatted_date1 = strtotime($date1);
```

Аналогічно, змінній `$formatted_date2` присвоїмо результат від функції `strtotime()` для другої дати:

```PHP
$formatted_date2 = strtotime($date2);
```

Тепер, коли дати представлені у форматі Unix timestamp, можна просто порівняти їх за допомогою операторів `>` (більше), `<` (менше) або `=` (рівно). Якщо ви хочете перевірити, чи дата більша за іншу, пишіть:

```PHP
if ($formatted_date1 > $formatted_date2) {
    // дата 1 більша за дату 2
}
```

## Глибоке занурення

Конвертація дат у формат Unix timestamp є досить простим і популярним методом порівняння дат в PHP. Проте, існують й інші способи, які можуть бути корисними в різних ситуаціях. Наприклад, можна використовувати клас `DateTime` для роботи з датами та використовувати його методи для порівняння.

Найголовніше, не забувайте про правильний формат дати. Це дуже важливо, оскільки неправильний формат може спричинити непередбачувані помилки в програмі. Ретельно перевіряйте дати та завжди використовуйте методи або функції, які знаходяться в стандартній бібліотеці PHP.
