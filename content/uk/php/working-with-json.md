# Чому

JSON (JavaScript Object Notation) є форматом для обміну даними, що використовується для передачі структурованих даних між різними системами. Його легкість використання та читабельність робить його популярним в середині розробників, які працюють з веб-програмуванням.

# Як

Щоб почати роботу з JSON у своїх PHP програмах, спочатку потрібно переконатися, що розширення JSON встановлене і включене в PHP. Для цього перегляньте свій файл php.ini та знайдіть наступну лінію:

```PHP
extension=json.so
```

Якщо вона закоментована, просто розкоментуйте її, а якщо відсутня, додайте її та перезапустіть веб-сервер.

Для створення та роботи з JSON даними використовуйте вбудовані функції `json_encode()` та `json_decode()`. Наприклад:

```PHP
// Створення асоціативного масиву даних
$data = array(
    'id' => 123,
    'title' => 'Привіт, світ!',
    'content' => 'Це текстове повідомлення'
);

// Перетворення в JSON
$json = json_encode($data);

// Виведення результату
echo $json;

// Вихід:
// {"id":123,"title":"Привіт, світ!","content":"Це текстове повідомлення"}
```

Як бачите, `json_encode()` перетворив наш асоціативний масив у рядок у форматі JSON. Інший приклад - перетворення JSON назад у масив PHP:

```PHP
// Рядок у форматі JSON
$json = '{"id":123,"title":"Привіт, світ!","content":"Це текстове повідомлення"}';

// Декодування рядка у масив
$data = json_decode($json, true);

// Виведення елементу масиву
echo $data['title'];

// Вихід:
// Привіт, світ!
```

# Глибоке занурення

Для більш детального дослідження роботи з JSON у PHP, можна ознайомитися з офіційною документацією PHP про роботу з JSON (https://www.php.net/manual/en/book.json.php). Тут ви знайдете повну інформацію про всі функції та можливості, що пропонуються для роботи з JSON.

Також, врахуйте, що JSON підтримує не тільки масиви та об'єкти, але й значення типу null, числа, булеві значення та рядки. Для цього використовуйте функцію `json_last_error()` для перевірки останньої помилки при перетворенні даних з JSON.
