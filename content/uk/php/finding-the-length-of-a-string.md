## Чому

Написання коду часто вимагає виконання різноманітних операцій з рядками даних. Знання довжини рядка є важливим для багатьох функцій і може спростити процес програмування.

## Як

Найпростішим інструментом для визначення довжини рядка в PHP є функція `strlen()`. Вона приймає рядок як аргумент і повертає його довжину у вигляді цілого числа.

```PHP
$string = "Привіт, світ!";
echo strlen($string); // Виведе 14
```

Також можна використовувати оператор `strlen` для отримання довжини рядка:

```PHP
$string = "Привіт, світ!";
echo $string; // Виведе 14
```

Якщо рядок містить символи, що займають більше одного байта (наприклад, кирилицю), то рекомендується використовувати функцію `mb_strlen()` для визначення правильної довжини.

```PHP
$string = "Привіт, світ!";
echo mb_strlen($string); // Виведе 11
```

## Глибше

Зверніть увагу, що функції `strlen()` і `mb_strlen()` обчислюють кількість символів в рядку, а не кількість байтів. Це означає, що для рядків, що містять символи, що займають більше одного байта, результат може бути відмінним від очікуваного. Наприклад, якщо рядок містить emoji, то він буде займати більше одного байта, і в результаті довжина рядка буде більша, ніж очікувана.

Також варто зазначити, що функції `strlen()` і `mb_strlen()` повертають кількість символів в рядку, а не підстрічки. Якщо потрібно отримати довжину підстрічки, то можна використовувати функцію `mb_substr()`.
