## Що і чому?

Пошук довжини рядка - це процес знаходження кількості символів у рядку. Це важлива функція для програмістів, оскільки рядки є одним із основних типів даних у багатьох мовах програмування, включаючи PHP. Знаючи довжину рядка, ми можемо ефективно обробляти дані та виконувати необхідні обчислення.

## Як:

У PHP для знаходження довжини рядка ми можемо використовувати функцію `strlen()`. Ця функція приймає рядок як аргумент і повертає його довжину у вигляді числа. Наприклад:

```PHP
$string = "Привіт світ!";
echo strlen($string);
// Виведе: 13
```

Якщо ж у рядку є кириличні символи, то потрібно використовувати функцію `mb_strlen()`, яка правильно підрахує кількість символів. Також, можливо, вам знадобиться використати функцію `mb_internal_encoding()` для встановлення правильного кодування рядка. Наприклад:

```PHP
$string = "Привіт світ!";
mb_internal_encoding("UTF-8");
echo mb_strlen($string);
// Виведе: 11
```

## Глибше вдивіться:

Процес визначення довжини рядка існує вже довго і використовується не тільки в PHP, але і в багатьох інших мовах програмування. Наприклад, у C ++ є функція `strlen()`, в Python - метод `len()`, а в JavaScript - властивість `length` у рядків.

Окрім того, існують і альтернативні способи знаходження довжини рядка у PHP, такі як використання функції `count_chars()` або методу `str_word_count()`, якщо нам потрібно підрахувати кількість слів у рядку.

У рядках можуть бути також приховані символи, які ми не бачимо, наприклад, символи нового рядка або табуляції. Тому, якщо потрібно врахувати всі символи у рядку при підрахунку його довжини, слід використовувати функцію `strlen()` з встановленням другого аргументу у `true` або використовувати функцію `mb_strlen()` з встановленням третього аргументу у `mb_strlen($string, "UTF-8")`.
