# Що і чому?

Інтерполяція рядків - це процес вставки змінних або виразів в середину строки для отримання динамічного виводу. Програмісти використовують це для створення більш гнучкого і зручного коду, який може адаптуватися до різних умов.

# Як використовувати:

```PHP
$name = "John";
echo "Привіт $name, як справи?";
```
Виведе: Привіт John, як справи?

```PHP
$number = 42;
echo "Ви обрали $number як своє улюблене число?";
```
Виведе: Ви обрали 42 як своє улюблене число?

# Глибше дослідження:

Ідея інтерполяції рядків з'явилася ще в перших версіях PHP, і зараз є багато альтернативних підходів до реалізації цієї функціональності. Однак, найпоширенішим є використання символу долара перед іменем змінної в рядку для підстановки її значення. Це дозволяє більш точно контролювати місце та спосіб вставки змінних у строку.
