## Чому

Найбільшою перевагою написання тестів є забезпечення надійності програмного коду. Це дозволяє переконатись у правильності роботи програми та зменшує кількість помилок, які можуть виникнути під час виконання коду.

## Як

```PHP
// Приклад функції, яка перевіряє чи є передане число парним
function isEven($number) {
  if ($number % 2 == 0) {
    return true;
  } else {
    return false;
  }
}

// Використання тесту для перевірки чи повертає функція правильний результат
echo isEven(6); // Виведе "true"
echo isEven(7); // Виведе "false"
```

## Глибока погруження

Написання тестів також допомагає виявити та виправити проблеми у програмному коді на ранніх етапах розробки. Крім того, вони допомагають збільшити швидкість розробки та спрощують впровадження нових функцій у вже існуючий код.
