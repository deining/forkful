## Why
Для чого: Іноді нам потрібно обчислити дату у майбутньому або минулому для різних цілей, таких як планування події або створення розкладу.

## How To
Як це зробити: Для початку, нам необхідно використовувати функцію `strtotime()` у PHP, яка дозволяє нам перетворити типу текстового виразу у дату. Далі, ми можемо використати цю функцію разом з операторами "+" та "-" для додавання або віднімання днів, тижнів, місяців або років. Давайте подивимося на приклад коду нижче:
```PHP
// Обраховуємо дату у майбутньому
echo strtotime("+1 day"); // результат: понеділок, 22.06.2020
echo strtotime("+1 week"); // результат: неділя, 28.06.2020

// Обраховуємо дату у минулому
echo strtotime("-2 months"); // результат: четвер, 23.04.2020
echo strtotime("-1 year"); // результат: вівторок, 23.06.2019
```

Ви також можете використовувати `strtotime()` разом з датами, які ви вводите вручну. Наприклад:
```PHP
echo strtotime("June 25 2020"); // результат: четвер, 25.06.2020
echo strtotime("2020-06-25 12:00:00"); // результат: четвер, 25.06.2020 12:00:00
```

Зверніть увагу, що результатом функції `strtotime()` є кількість секунд, що пройшли з 1 січня 1970 року до заданої дати. Якщо ви хочете отримати дату у зрозумілому форматі, ви можете використовувати функцію `date()` наступним чином:
```PHP
echo date("d.m.Y", strtotime("+1 day")); // результат: 22.06.2020
echo date("l, d F Y", strtotime("-1 year")); // результат: п'ятниця, 23 червня 2019
```

## Deep Dive
Глибше про обчислення дат: Функція `strtotime()` також має деякі корисні параметри, які дозволяють вам вказувати більш точну дату. Наприклад, ви можете використати параметр "last" для того, щоб отримати останній день тижня або місяця, наприклад:
```PHP
echo strtotime("last Monday"); // результат: понеділок, 15.06.2020
echo strtotime("last day of February 2020"); // результат: п'ятниця, 28.02.2020
```

Крім того, функція `strtotime()` може розпізнавати такі слова, як "next" або "previous", які дозволяють вам вибрати наступний або попередній день, тиждень, місяць або рік. Наприклад:
```PHP
echo strtotime("next Sunday"); // результат: неділя, 28
