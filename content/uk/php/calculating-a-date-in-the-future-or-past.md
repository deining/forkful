## Чому
Часом в житті нам потрібно знати точну дату в майбутньому або минулому - наприклад, для планування подорожей або розрахунку дати народження для юридичних документів. У таких випадках необхідно мати знання про те, як визначити дату в майбутньому або минулому за допомогою програмування.

## Як це зробити
```PHP
// Приклад коду для визначення дати у майбутньому
$date = new DateTime(); //створюємо новий об'єкт DateTime
$date->modify('+7 days'); //додаємо 7 днів до поточної дати
echo $date->format('Y-m-d'); //виводимо отриману дату у форматі рік-місяць-день

// Результат: 2021-05-22
```

```PHP
// Приклад коду для визначення дати у минулому
$date = new DateTime(); //створюємо новий об'єкт DateTime
$date->modify('-1 month'); //віднімаємо 1 місяць від поточної дати
echo $date->format('d-m-Y'); //виводимо отриману дату у форматі день-місяць-рік

// Результат: 23-03-2021
```

## Глибоке дослідження
У PHP існує безліч функцій для роботи з датами, таких як `date()`, `strtotime()`, `DateTime` і багато інших. Вони дозволяють виконувати різноманітні маніпуляції з датами, такі як додавання або віднімання днів, місяців або років, встановлення часових зон та форматування дати за власними потребами.

Також важливо пам'ятати про правильне використання часових зон, оскільки вони можуть впливати на результати обчислень дат. Рекомендується завжди встановлювати потрібну часову зону перед роботою з датами.
