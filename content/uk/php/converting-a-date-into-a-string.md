## Чому

Перетворення дати в рядок є важливим завданням для багатьох програм, особливо тих, які працюють з даними про події або звіти за період часу. У цій статті ми розглянемо, як це зробити за допомогою PHP.

## Як

```PHP
// Задаємо дату в форматі UNIX timestamp:
$timestamp = 1617982800;

// Використовуємо директиву date() для перетворення дати в рядок в форматі "Рік-Місяць-День":
$date = date('Y-m-d', $timestamp);

// Виводимо отриманий рядок:
echo $date;
```

Вивід: 2021-04-09

## Поглиблене вивчення

Перетворення дати в рядок може бути складним завданням, оскільки формати дати можуть відрізнятися в залежності від країни або культури, в якій ви працюєте. Тому, корисно знати, які формати доступні для використання у функції `date()` та який результат буде виведений при їх використанні. Також, варто звернути увагу на особливості локалізації дати під час перетворення.
