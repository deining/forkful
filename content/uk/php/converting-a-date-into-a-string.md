## Чому

Існує багато ситуацій, коли в програмуванні потрібно перетворити дату в рядок. Наприклад, для збереження дати у базі даних або відображення її на веб-сторінці. У цьому блозі ми розглянемо, як зробити це в PHP.

## Як

Почнемо з простого прикладу. Давайте вважати, що ми маємо дату у форматі ден / місяць / рік і нам потрібно перетворити її у рядок у форматі місяць / день / рік.

```PHP
$date = "14/02/2021";
$new_date = date("m/d/Y", strtotime($date));
echo $new_date;
```

Результат: 02/14/2021.

Тут ми використали функцію `date()` для форматування дати та функцію `strtotime()` для перетворення дати в числове значення.

Також, можна використовувати різні формати дати в функції `date()`. Наприклад:

```PHP
$today = date("F j, Y, g:i a"); // літера F відображає повний назву місяця (наприклад, грудень), а літера g - час у 12-годинному форматі.
echo $today;
```

Результат: Лютий 14, 2021, 3:45 pm.

## Глибоке поглиблення

У функції `date()` також можна використовувати різні параметри, щоб отримати потрібну нам інформацію. Наприклад, можна отримати день тижня, назву місяця або час у годинах.

```PHP
$day = date("l"); // l відображає повне назва дня тижня (наприклад, понеділок).
$month = date("M"); // M відображає скорочену назву місяця (наприклад, лютого).
$hour = date("h"); // h відображає час у 12-годинному форматі.
```

Також, можна використовувати більш складні функції, які дозволяють враховувати часовий пояс і міжнародний формат дати.
