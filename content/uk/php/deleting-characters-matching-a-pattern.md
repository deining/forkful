## Чому
Зазвичай, ви можете хотіти видалити символи з рядка, які відповідають певному шаблону, коли розробляєте програму на PHP. Це може бути необхідним для очищення тексту або форматування даних перед обробкою. 

## Як
Процес видалення символів в PHP може бути здійснений за допомогою функції `preg_replace()`, яка використовує регулярні вирази для знаходження шаблону символів, які потрібно видалити. Наприклад, якщо вам потрібно видалити всі цифри з рядка, ви можете використовувати такий код:

```PHP
$string = "Hello123, World!";
$new_string = preg_replace('/[0-9]/','', $string);
echo $new_string; // Виведе "Hello, World!"
```

У цьому прикладі, `[0-9]` - це регулярний вираз, який відповідає будь-якому символу від 0 до 9. Другий параметр `''` означає, що ми заміняємо знайдені символи на порожній рядок, тобто видаляємо їх.

## Deep Dive
Існує багато опцій для видалення символів за допомогою `preg_replace()`. Наприклад, ви можете також задати межі для вибору символів, наприклад `[a-z]`, щоб вибрати лише малі літери англійського алфавіту. Додаткові опції, такі як `i` для ігнорування регістру символів чи `u` для роботи з мультибайтовим текстом, також доступні для використання з регулярними виразами.
