## Що та чому?

В PHP, можливо видалення символів, які відповідають певному шаблону, використовуючи вбудовану функцію `preg_replace()`. Це корисний інструмент для виконання певних операцій над рядками, таких як видалення цифр або заміна спеціальних символів. Програмісти часто використовують цю функцію для очищення вхідних даних або форматування виводу.

## Як це зробити:

```PHP 
$str = "Hello, world!";
$pattern = "/[aeiou]/"; // видалити всі голосні з рядка
$result = preg_replace($pattern, "", $str);

echo $result; // Hll, wrld!
```

```PHP 
$str = "123-456-789";
$pattern = "/\d/"; // видалити всі цифри з рядка
$result = preg_replace($pattern, "", $str);

echo $result; // ---
```

## Занурення:

Ця функція була вперше введена в PHP версії 4.0 і основана на регулярних виразах Perl (PCRE). Існують інші альтернативи, які можна використовувати для видалення символів, такі як функції `strpos()` та `str_replace()`. Однак, `preg_replace()` безпечніше застосовувати при обробці спеціальних символів або символів, що використовуються в регулярних виразах.

Імплементаційна деталі: Ця функція приймає три аргументи - шаблон для пошуку, рядок для заміни та рядок для обробки. Вона повертає змінений рядок з видаленими символами, відповідними вказаному шаблону.
