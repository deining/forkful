## Чому

Часом в процесі програмування може знадобитися видалення символів, які відповідають певному шаблону. Це може бути необхідно для роботи зі строками або з даними, які необхідно обробити перед подальшим використанням. У цій статті я розкажу, як це можна зробити за допомогою PHP.

## Як

Для початку нам необхідно визначити шаблон, за яким будемо видаляти символи. Це може бути зроблено за допомогою функції `preg_match()`, яка приймає два параметри - шаблон і строку. Далі, за допомогою функції `preg_replace()`, ми можемо виконати видалення символів, використовуючи отриманий шаблон.

```PHP
<?php
// Визначаємо шаблон
$pattern = '/[a-z]/';

// Строка, з якої будемо видаляти символи
$string = "Hello World";

// Використовуємо функції preg_match і preg_replace
$new_string = preg_replace($pattern, "", $string);

echo $new_string;

// Вивід: Hllo Wlrd
?>
```

У прикладі вище ми використовуємо шаблон, який відповідає всім малим літерам англійського алфавіту. Таким чином, всі символи, які відповідають цьому шаблону, будуть видалені і ми отримаємо строку без цих символів.

## Глибоке дослідження

Функція `preg_replace()` приймає ще два параметри, які дозволяють змінювати регулярний вираз під час видалення символів. Перший з них - це параметр `$limit`, який вказує максимальну кількість замін, які можуть бути виконані. За замовчуванням, всі символи, що відповідають шаблону, будуть видалені, але якщо встановити значення `$limit` меншим, то будуть видалені лише перші елементи.

Другий параметр - це `$count`, який приймає змінну, в яку буде записано кількість замін, які були виконані. Це дозволяє відслідковувати кількість замін і виконувати додаткові дії з результатами.
