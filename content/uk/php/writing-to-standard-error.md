## Що і чому?

Запис до стандартного виведення помилок - це спосіб друкування повідомлень про помилки у вашому коді. Це корисно для відлагодження та знаходження помилок у вашій програмі.

## Як це зробити:

```
<?php
// Приклад виведення повідомлення про помилку до стандартного виведення помилок
$number = "не число";
if (!is_numeric($number)) {
    fwrite(STDERR, "Помилка: '$number' не є числом!");
}
?>
```
```
Помилка: 'не число' не є числом!
```

## Глибока занурення:

Запис до стандартного виведення помилок був запроваджений у PHP 4.3.0. Що стосується альтернатив, то ви можете використовувати функцію ```error_log()``` для записування повідомлень про помилки до файлу або бази даних. Щодо деталей реалізації, стандартний потік помилок доступний за допомогою змінної ```STDERR```.
