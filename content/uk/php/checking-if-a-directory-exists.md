## Чому

Перевірка існування директорії є важливою частиною програмування, оскільки дозволяє перевірити наявність певної структури даних і виконати певні дії в залежності від результату.

## Як

```PHP
if (is_dir('/home/user/directory')) {
    echo 'Директорія існує';
} else {
    echo 'Директорія не існує';
}
```

У цьому прикладі ми використовуємо вбудовану функцію `is_dir()` для перевірки існування директорії `/home/user/directory`. Якщо директорія існує, то виконається перший блок коду, в якому виводиться повідомлення про її наявність. Якщо директорія не існує, то виконається другий блок коду, в якому виводиться повідомлення про її відсутність.

## Глибше

Під час перевірки існування директорії можна використовувати різні функції для отримання додаткової інформації. Наприклад, функція `is_readable()` дозволяє перевірити, чи можливо читати дану директорію, а функція `is_writable()` - чи можна писати в неї. Також можна використовувати функцію `scandir()`, яка повертає масив зі списком файлів та директорій у вказаній директорії. Більше інформації про ці та інші функції можна знайти в [офіційній документації PHP](https://www.php.net/manual/en/book.dir.php).
