## Чому

Здрастуйте, друзі! Сьогодні ми розглянемо одну дуже важливу і потрібну техніку програмування - перевірку наявності каталогу. Чому це важливо? Існує багато причин, чому вам може знадобитися перевірка існування каталогу при розробці веб-додатків. Наприклад, ви можете хотіти переконатися, що перед використанням певного каталогу він існує, щоб уникнути помилок під час виконання коду. Або ж ви можете потребувати створити новий каталог, якщо поточний не існує. Тому вивчення техніки перевірки наявності каталогу є обов'язковим для кожного програміста.

## Як

Код на PHP для перевірки наявності каталогу дуже простий. Використовуючи функцію `is_dir()`, ми можемо легко визначити, чи існує каталог за запропонованим шляхом. Давайте подивимось на приклад:

```PHP
$path = "my_directory"; // шлях до каталогу, який ми перевіряємо

if (is_dir($path)) {
    echo "Каталог існує";
} else {
    echo "Каталог не існує";
}
```

Ви можете також використовувати функцію `file_exists()` для перевірки існування каталогу. Або ж використовуйте `scandir()` для переліку всіх каталогів у певній директорії.

## Глибоке занурення

Перевірка наявності каталогу - це частина обробки помилок в програмуванні. Якщо каталог не існує, ви можете обробити цю помилку, видаючи відповідне повідомлення або створюючи новий каталог. Також варто пам'ятати, що функція `is_dir()` може повертати `true`, якщо передано ім'я файлу, а не каталогу. Тому в кожному коді перевіряйте тип файлу за допомогою функції `is_file()`.
