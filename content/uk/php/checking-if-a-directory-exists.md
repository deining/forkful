## Чому
Перевірка наявності директорії є важливою для більш ефективної роботи програмістів, оскільки це дозволяє їм переконатися, що необхідна директорія існує перед тим, як продовжувати програмування. З цим можна уникнути появи помилок та зберегти час при пошуку відповідних файлів для подальшої обробки.

## Як це зробити
Для перевірки наявності директорії в PHP можна використовувати функцію `is_dir()`, яка повертає `true` або `false` в залежності від того, чи існує директорія. Наприклад:

```PHP
if(is_dir('шлях/до/директорії')) {
  echo "Ця директорія існує!";
} else {
  echo "Цієї директорії не існує!";
}
```

Вихідний код виведе повідомлення про наявність або відсутність директорії. Також можна використати функцію `file_exists()`, яка перевіряє, чи існує не тільки директорія, але і будь-який інший файл. Наприклад:

```PHP
if(file_exists('шлях/до/директорії/файлу.txt')) {
  echo "Файл існує!";
} else {
  echo "Цього файла не існує!";
}
```

В залежності від потреб можна використовувати ту чи іншу функцію для перевірки наявності директорії.

## Детальний аналіз
Функції `is_dir()` та `file_exists()` дозволяють перевірити наявність директорії або файлу відносно шляху, переданого в аргументі. Це значить, що можна перевірити існування директорії не тільки на локальному сервері, але і на віддаленому. Також можна використовувати ці функції для переходу до певної директорії або виконання певних дій, якщо цієї директорії немає.
