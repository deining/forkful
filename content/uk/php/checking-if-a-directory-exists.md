# Чому

Перевірка наявності директорії є важливою частиною програмування PHP, оскільки дозволяє переконатись у правильності шляху до файлів та забезпечує більш безпечну роботу з ними.

## Як це зробити

Існує декілька способів перевірити, чи існує директорія в PHP. Один з них - використання функції `file_exists()`:

```PHP
if (file_exists("/шлях/до/директорії")) {
  echo "Директорія існує";
} else {
  echo "Директорія не існує";
}
```

Якщо директорія існує, в цьому прикладі буде виведено повідомлення "Директорія існує". Якщо ж директорія не існує, буде виведено повідомлення "Директорія не існує".

Також можна використати функцію `is_dir()` для перевірки наявності директорії:

```PHP
if (is_dir("/шлях/до/директорії")) {
  echo "Це директорія";
} else {
  echo "Це не директорія";
}
```

Ця функція перевіряє, чи є вказаний шлях директорією і повертає `true` або `false` відповідно.

## Вглиб

При перевірці наявності директорії важливо знати, як правильно задати шлях до неї. Наприклад, якщо використовується сервер Apache, то шлях до кореневої директорії буде виглядати так: `/var/www/html`. Також слід пам'ятати про те, що шлях може бути відносним або абсолютним, тому слід правильно задати шлях у відповідності до потреб проекту.
