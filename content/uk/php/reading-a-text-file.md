## Чому

Програмування PHP - це захоплююча і корисна навичка, яка може принести велику користь в сучасному цифровому світі. Читання текстових файлів є однією із ключових вмінь, яку необхідно мати для роботи зі збереженими даними на комп'ютері. Ця стаття розкриває процес зчитування текстових файлів в PHP та надає корисні приклади для використання в вашому коді.

## Як це зробити

Читати текстові файли в PHP досить просто. Спочатку потрібно відкрити файл за допомогою функції `fopen()` і передати його ім'я та режим доступу. Наприклад, якщо ми хочемо прочитати файл `textfile.txt` з режимом доступу "тільки для читання", наш код буде виглядати наступним чином:

```PHP
$myfile = fopen("textfile.txt", "r");
```

Потім ми можемо використовувати функцію `fgets()` для зчитування рядків з файлу по одному. Ця функція повертає наступний рядок з файлу в рядкову змінну. Наприклад, ми можемо використати цикл для читання усіх рядків в файлі та вивести їх на екран:

```PHP
while(!feof($myfile)) {
  $line = fgets($myfile);
  echo $line . "<br>";
}
```

Цей код буде продовжувати читати рядки з файлу, поки не дійде до кінця файлу, використовуючи функцію `feof()` для перевірки. Наш вихідний код буде виглядати наступним чином:

```PHP
<?php
$myfile = fopen("textfile.txt", "r");
while(!feof($myfile)) {
  $line = fgets($myfile);
  echo $line . "<br>";
}
fclose($myfile);
?>
```

Результатом буде виведення всіх рядків з файлу на екран у вигляді списку.

## Глибоке занурення

Є багато інших методів для читання і обробки текстових файлів в PHP, таких як використання функцій `file()`, `file_get_contents()` або `fgetcsv()`. Кожен з цих методів має свої особливості та може бути корисним у різних ситуаціях. Також варто звернути увагу на різні режими доступу до файлу, такі як "тільки для читання", "тільки для запису" або "для читання та запису", які можуть впливати на роботу з файлом.
