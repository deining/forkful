## Причина

Чому б хтось займався читанням текстового файлу? Трохи здорового людського роздумування здатне відповісти на це питання. Можливо, вони хочуть отримати доступ до текстових даних, щоб обробити їх у своїй програмі, або перевірити вміст файлу перед надсиланням його на веб-сторінку. Незалежно від причини, дуже важливо знати, як читати текстовий файл за допомогою PHP, тому що це дуже корисний навичка для кожного програміста.

## Як

Найпростіший спосіб читати текстовий файл за допомогою PHP - використати функцію `file()`. Вона зчитує вміст файлу і повертає його у вигляді масиву, де кожен елемент - рядок з файлу.

```PHP
$file = file("myfile.txt");
print_r($file);
// Виведе:
// Array ([0] => Рядок 1, [1] => Рядок 2, [2] => Рядок 3)
```

Якщо ви хочете зчитати вміст файлу у вигляді одного рядка, то ви можете використати функцію `file_get_contents()`. Вона поверне вміст файлу у вигляді рядка.

```PHP
$file = file_get_contents("myfile.txt");
echo $file;
// Виведе:
// Рядок 1 Рядок 2 Рядок 3
```

Якщо у вас є більш складний текстовий файл, наприклад, CSV-файл з даними у таблиці, то ви можете використати функцію `fopen()` для відкриття файлу та цикл `fgets()` для построчного зчитування даних.

```PHP
$file = fopen("mycsvfile.csv", "r");
while (!feof($file)) {
  $line = fgets($file);
  // Обробка рядка даних тут
  echo $line;
}
fclose($file);
```

## Глибокий пір

Якщо ви працюєте з більш великими текстовими файлами, то ви можете стикнутися з проблемами швидкості та пам'яті. Для покращення продуктивності ви можете використати функцію `fopen()` з дескриптором `r+`, який дозволяє вам зчитувати та записувати дані у файл одночасно.

Також варто зазначити, що у більших текстових файлів можуть бути спеціальні символи, такі як переноси рядків, які можуть впливати на обробку та виведення даних. У таких випадках, ви можете використати функцію `mb_convert_encoding()` для перетворення рядків у бажаний кодування.
