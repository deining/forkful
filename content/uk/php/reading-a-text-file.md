## Чому

Читання файлу тексту - це важлива техніка, яку повинен вміти кожен програміст. Використовуючи PHP, ви можете легко читати дані з текстових файлів, що дозволяє вам працювати з інформацією збереженою в цій формі.

## Як це зробити

Для початку, вам потрібно відкрити файл за допомогою функції `fopen()`, яка приймає два аргументи - шлях до файлу і режим, у якому ви хочете його відкрити. Для зручності, приклад коду буде показаний на прикладі простого текстового файлу з наступним вмістом:

```PHP
Це перша рядок
Це другий рядок
Це третій рядок
```

Отже, для відкриття цього файлу у режимі читання, нам потрібно використати наступний код:

```PHP
$file = fopen("textfile.txt", "r"); // відкриваємо файл у режимі читання
```

Тепер, коли файл відкритий, зможемо скористатися функцією `fgets()`, щоб прочитати рядки з файлу. Ця функція повертає наступний рядок і пересуває вказівник на наступний рядок у файлі. Тому, ми можемо використати цю функцію у циклі, поки ми не дійдемо до кінця файлу:

```PHP
while(!feof($file)){ // поки не досягнуто кінця файлу
  echo fgets($file) . "<br>"; // вивести наступний рядок
}
```

В результаті отримаємо наступний вивід:

```
Це перший рядок
Це другий рядок
Це третій рядок
```

## Глибоке занурення

Крім функції `fgets()`, у PHP є багато інших функцій для читання файлів. Наприклад, `file_get_contents()` дозволяє вам прочитати весь файл у строку, а `fread()` - вказану кількість байтів з файла.

Також можна використовувати ці функції у поєднанні з іншими, наприклад, `explode()`, щоб розділити прочитаний текст на окремі рядки або `preg_match()` для пошуку певного тексту в файлі.
