## Чому

Начитання текстового файлу в PHP - це важлива та корисна навичка, яка допоможе вам зчитувати та обробляти великі обсяги даних, що є необхідним для багатьох веб-розробок. Навчитися читати текстові файли допоможе вам вдосконалити свої навички програмування та зробить вашу роботу більш ефективною.

## Як

```PHP
$file = fopen("textfile.txt", "r"); // відкрити файл у режимі читання
if ($file) { // перевірити, чи файл існує та відкритий
  while (($line = fgets($file)) !== false) { // зчитування файлу рядок за рядком
    echo $line; // вивести рядок
  }
  fclose($file); // закрити файл
}
```

Ви можете використовувати функцію `fopen()` для відкриття текстових файлів у режимі читання, а потім зчитувати їх рядок за рядком, використовуючи функцію `fgets()`. Пам'ятайте про закриття файлу після того, як ви закінчили його читати за допомогою функції `fclose()`.

## Глибокий занурення

Зчитування текстових файлів в PHP більш складне, ніж просте використання функцій `fopen()` та `fgets()`. Ви можете вживати інші функції, такі як `file_get_contents()` або `fread()`, для зчитування файлів за байтами та обробки їх у більш гнучкий спосіб. Навчання цих функцій буде корисним для вашої роботи з великими обсягами даних.
