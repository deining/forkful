## Чому

Виведення відладкового виводу є важливим інструментом для розробників, який допомагає знайти та виправити помилки у програмному коді. Він також може бути використаний для відстеження значень змінних і функцій у реальному часі, що є корисною функцією при розробці програм.

## Як

Для виведення відладкового виводу використовуйте функцію `echo` у комбінації з функцією `print_r` або `var_dump` для виведення значень змінних і функцій у вигляді рядка або масиву. Наприклад:

```PHP
$name = 'John';
echo $name; // виведе "John"
print_r($name); // виведе "John" у вигляді масиву
var_dump($name); // виведе детальну інформацію про змінну $name
```

Крім того, ви можете використовувати функцію `die` або `exit` для виведення відладкового виводу і зупинки виконання програми, якщо виникла помилка. Наприклад:

```PHP
echo 'Цей рядок буде виведений перед виконанням die';
die('Помилка! Виконання програми зупинено'); // виведе "Помилка! Виконання програми зупинено"
echo 'Цей рядок не буде виведений, оскільки виконання програми вже зупинено';
```
