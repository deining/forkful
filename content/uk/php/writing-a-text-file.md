## Чому

Написання текстового файлу є важливим процесом в програмуванні, оскільки це дозволяє легко зберігати і організовувати дані для подальшого використання.

## Як

Для написання текстового файлу у PHP потрібно використовувати функцію `fopen()` для створення файлового дескриптору, `fwrite()` для запису даних у файл та `fclose()` для закриття дескриптору.

Наприклад:
```
<?php
$file = fopen("textfile.txt", "w"); //створення файлового дескриптору
fwrite($file, "Це приклад тексту, який буде записаний у файл."); //запис даних у файл
fclose($file); //закриття дескриптору
```

Це призведе до створення файлу з назвою "textfile.txt" із зазначеним текстом в ньому.

## Глибше вдивлення

Крім звичайного запису даних у текстовий файл, у PHP є ще кілька варіантів, які можна застосувати при написанні файлу. Наприклад, `file_put_contents()` дозволяє записати дані у файл одним рядком коду, не використовуючи окремі функції для створення дескриптору та його закриття.

Також, є можливість додавання даних до вже існуючого файла за допомогою флагу `a` при відкритті дескриптору.

Наприклад:
```
<?php
file_put_contents("textfile.txt", "Це приклад тексту, який буде доданий до уже існуючого файлу.", FILE_APPEND); //запис даних у файл, додаючи до вже існуючого змісту
```

Більш докладну інформацію про можливості роботи з текстовими файлами у PHP можна знайти у [документації PHP](https://www.php.net/manual/ru/function.fwrite.php).
