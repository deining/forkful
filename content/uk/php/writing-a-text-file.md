## Що & Чому?
Запис текстового файлу є базовій задачею для більшості програмістів. Це означає створення нового файлу та запису даних до нього. Цей процес може бути корисним для збереження важливої інформації або створення нового вихідного коду для майбутнього використання.

## Як це зробити:
Нижче наведено приклади коду і вихідного результату, які демонструють як створити текстовий файл і записати дані у нього за допомогою PHP.

```PHP
<?php
// Відкриття файлу для запису
$file = fopen("example.txt", "w") or die("Неможливо створити файл!");

// Запис даних до файлу
$txt = "Цей текст буде записаний до файлу.";
fwrite($file, $txt);

// Закриття файлу
fclose($file);
?>
```

В результаті ви отримаєте файл з назвою "example.txt", який міститиме текст "Цей текст буде записаний до файлу."

## Глибокий погляд:
Історичний контекст: Запис файлів був однією з основних функцій першої версії PHP, випущеної у 1995 році. В цей час він використовувався для запису і виводу інформації на сервер веб-сторінки.

Альтернативи: У PHP є багато функцій для створення, зчитування і запису файлів, таких як "file_put_contents", "fopen", "fwrite" і "file_put_contents". Кожна з цих функцій має свої особливості, тому варто вивчити їх і вибрати оптимальну для своїх потреб.

Деталі реалізації: Для того, щоб записати дані до файлу в PHP, ми використовуємо функцію "fwrite", яка приймає два параметри - відкритий файл та дані, які ви хочете записати. Ми також можемо використати цю ж функцію для додавання нових даних до вже існуючого файлу, просто вказавши такий же параметр "a" (добавити) при відкритті файлу.
