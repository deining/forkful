## Чому

Написання текстового файлу є важливим елементом програмування. Це дозволяє зберігати та зчитувати дані з файлу, що є необхідним для багатьох програм.

## Як

Написання текстового файлу в PHP досить просте за допомогою вбудованої функції `file_put_contents()`. Спочатку потрібно відкрити файл для запису, а потім використати цю функцію для запису даних у вигляді рядка. Наприклад:

```PHP
$file = fopen("text.txt", "w"); // відкриття файлу для запису
$text = "Привіт, українці!"; // дані для запису
file_put_contents($file, $text); // записуємо дані у файл
fclose($file); // закриваємо файл
```

Також можна додавати дані до вже існуючого файлу, використовуючи режим "a" замість "w". Це дозволить дописувати дані до кінця файлу замість перезаписування його.

## Глибокий погляд

При написанні текстових файлів важливо пам'ятати про права доступу до файлів. Якщо файл не має прав на запис або читання, то функція `file_put_contents()` не буде працювати. Також слід пам'ятати про захист від SQL ін'єкцій, якщо дані для запису є користувацькими вводами. Можна використовувати функцію `filter_input()` для фільтрації цих даних перед записом у файл.
