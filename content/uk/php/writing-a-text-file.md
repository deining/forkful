##Чому
Написання текстового файлу є важливою частиною програмування, оскільки це дозволяє зберігати дані та передавати їх між різними програмами або сайтами.

##Як це зробити
Написання текстового файлу в PHP є простим завданням за допомогою функції `fwrite()`. Давайте розглянемо цей процес на прикладі:

```PHP
<?php
$file = fopen("myfile.txt", "w") or die("Не вдалося створити файл"); //створення нового файлу або відкриття існуючого
$txt = "Привіт світ!"; //текст, який ми хочемо зберегти в файлі
fwrite($file, $txt); //записуємо текст у файл за допомогою функції fwrite()
fclose($file); //закриваємо файл
echo "Текст успішно записаний у файл!";
?>
```

Результатом цього коду буде створення або перезаписання файлу `myfile.txt` з текстом "Привіт світ!".

##Глибоке занурення
Крім простого запису до файлу, в PHP є багато додаткових функцій, які дозволяють працювати з текстовими файлами. Наприклад, функції `fread()` та `fgets()` дозволяють зчитувати дані з файлу, а `feof()` перевіряє, чи досяг курсор кінця файлу. Крім того, PHP також має функцію `file()`, яка дозволяє зчитати весь файл у масив рядків для подальшої обробки.

Але саме створення та запис до текстового файлу є базовою інструкцією, яку потрібно знати для початку роботи з файлами у PHP.
