## Що & Чому
Використання регулярних виразів - це могутня інструмент для пошуку та заміни тексту в програмах PHP. Вони дозволяють легко знаходити та витягувати необхідну інформацію з тексту, що дозволяє програмістам зосередитися на самій логіці програми, а не на маніпуляції зі строками. Це збільшує швидкість написання програм та робить код більш читабельним.

## Як
Для використання регулярних виразів у програмах PHP потрібно використовувати функцію `preg_match()`, яка приймає два параметри: шаблон регулярного виразу та рядок, у якому потрібно здійснити пошук. Наприклад, якщо ви хочете знайти всі кількість зустрічень слова "привіт" у рядку, ви можете написати наступний код:
```PHP
<?php
$text = "Привіт, я PHP програміст!";
preg_match("/привіт/i", $text, $matches);
print_r($matches);
?>
```
Результатом буде масив, що містить знайдене співпадіння:
```
Array
(
    [0] => Привіт
)
```
Використання регулярних виразів дозволяє задавати більш складні шаблони для пошуку, наприклад, шукати всі слова, що починаються на "привіт". Також можна використовувати спеціальні символи для пошуку різних форматів даних, наприклад, електронних адрес.

## Глибокий занурення
Регулярні вирази були створені в 1950-х роках та стали популярними серед програмістів у 1970-1980 роках. Використання регулярних виразів є альтернативою для роботи зі строками через функції PHP, такі як `strpos()`, `substr()` та інші. Регулярні вирази можуть бути складні для зрозуміння та написання, тому багато мов програмування, включаючи PHP, надають більш простість у використанні функцій для роботи зі строками.

Регулярні вирази також мають спеціальні прапорці, що дозволяють задавати режими пошуку, наприклад, чи враховувати регістр символів чи ні. Використовуючи ці прапорці у правильних комбінаціях, можна здійснювати більш точний пошук у тексті.
