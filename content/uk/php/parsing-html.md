## Що і чому?

Розбор HTML - це процес отримання інформації з HTML-сторінки шляхом обробки її коду. Програмісти роблять це з різних причин, наприклад, для створення веб-скрапера, який автоматично витягує дані з сайту, або для аналізу та обробки інформації з веб-сторінок.

## Як це зробити:

```
<?php  

// Спочатку нам потрібно завантажити HTML-сторінку з URL
$url = "https://example.com";
$html = file_get_contents($url);

// Використовуємо функцію PHP "preg_match" для знаходження відповідного елементу у сторінці
preg_match('/<title>(.*?)<\/title>/s', $html, $matches);

// Виводимо результат
echo $matches[1];
```

У даному прикладі ми використовуємо функцію "file_get_contents" для завантаження HTML-сторінки, після чого використовуємо "preg_match", щоб знайти заголовок сторінки і вивести його. Це дуже простий приклад, але дозволяє зрозуміти, як працює розбір HTML.

## Глибокий захват:

Розбір HTML існує вже довгий час і є одним із способів отримання даних з веб-сторінок. Програмісти також можуть використовувати XML-парсери або бібліотеки PHP для отримання даних з HTML-сторінок. Крім того, існує також можливість використовувати JavaScript для отримання даних зі сторінок, але у цьому випадку потрібно мати додаткові знання про цю мову.

Ще одним важливим питанням при розборі HTML є те, як вибирати потрібні елементи зі сторінки. Зазвичай це виконується за допомогою селекторів CSS або XPath, які дозволяють точно вибрати потрібний елемент.
