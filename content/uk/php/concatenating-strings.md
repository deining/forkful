## Чому

У зв'язку зі стрімким ростом веб-додатків і написанням складних програм, рано чи пізно кожен програміст побачить на себе нагрузку по з'єднанню декількох стрічок тексту. Це корисний навик, який допоможе оптимізувати ваш код і зробить його більш читабельним.

## Як це зробити

```PHP
// З'єднання двох стрічок без заданої розділової символи
$str1 = "Привіт";
$str2 = " Україно!";
echo $str1 . $str2;
// Виведе "Привіт Україно!"

// З'єднання більше двох стрічок за допомогою функції implode ()
$arr = ["Я", "люблю", "PHP"];
$str = implode(" ", $arr);
echo $str;
// Виведе "Я люблю PHP"
```

## Глибше копання

При з'єднанні стрічок, PHP використовує оператор "точка" (".") або функцію implode (). Оператор "точка" є швидшим способом з'єднання двох стрічок, тоді як функція implode () дозволяє з'єднати більше двох стрічок і використовує роздільник, заданий користувачем.
