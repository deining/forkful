## Чому

У цій статті ми дізнаємося, як отримати поточну дату в PHP програмуванні і чому це може бути корисним для вас. Отримання поточної дати може бути корисним для ведення журналів або для створення динамічних функцій на вашому веб-сайті.

## Як це зробити

Існує кілька способів отримати поточну дату в PHP. Один з них - це використання вбудованої функції `date()`. Для цього просто передайте формат дати у перший параметр функції, наприклад:

```PHP
$date = date('d/m/Y');
echo $date; // Результат: 23/04/2021
```

Ви також можете відображати дату у вигляді тексту, використовуючи функцію `strtotime()` та параметр `now`, який вказує на поточну дату. Наприклад:

```PHP
$date = strtotime('now');
echo date('F d, Y', $date); // Результат: April 23, 2021
```

## Глибоке занурення

Крім вбудованих функцій, PHP також має деякі додаткові функції для роботи з датами. Наприклад, функція `mktime()` дозволяє створювати дати на підставі заданих параметрів, таких як рік, місяць та день. Розглянемо приклад:

```PHP
$date = mktime(0, 0, 0, 4, 23, 2021); // дата: 2021-04-23
echo date('l', $date); // Результат: Friday (день тижня)
```

Крім того, в PHP є можливість працювати з об'єктом `DateTime`, який надає більш гнучкі можливості для роботи з датами. Це особливо корисно, якщо вам потрібно проводити операції з датами, такі як додавання часу чи порівняння двох дат.
