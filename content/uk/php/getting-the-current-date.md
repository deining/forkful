## Чому

У програмуванні часто потрібно отримувати поточну дату. Це може бути корисно, наприклад, для ведення журналів або відстеження важливих подій в програмі.

## Як

```PHP
<?php
// Отримання поточної дати у форматі "День, Місяць Рік"
$currentDate = date("d, F Y");
echo $currentDate; // Виведе, наприклад, "01, Лютий 2021"

// Отримання поточного часу у форматі "Години:хвилини:секунди"
$currentTime = date("H:i:s");
echo $currentTime; // Виведе, наприклад, "18:30:00" 
?>
```

За допомогою вбудованої функції `date()` у PHP можна отримувати поточну дату та час у різних форматах. Доступні різні параметри для форматування дати та часу. Наприклад, `d` виводить день у числовому форматі, `F` - повний назва місяця українською мовою, `Y` - повний з 4 цифр рік, `H` - години у 24-годинному форматі, `i` - хвилини, `s` - секунди.

## Глибше вдивіться

Функція `date()` отримує 2 обов'язкових параметра: перший - формат дати, а другий (необов'язковий) - timestamp, який вказує, з якої дати буде отримана поточна дата. При використанні параметра timestamp, виведена дата буде відповідати заданому часу, а не поточному. Наприклад, `date("d, F Y", mktime(0, 0, 0, 5, 9, 2015));` виведе "09, Травень 2015", оскільки ми передали функції значення 9.05.2015.

Також в PHP є функція `time()`, яка повертає поточний timestamp. Це може бути корисно, якщо вам потрібно використовувати поточну дату та час у своїй програмі.
