## Чому

Отримання поточної дати є важливою задачею для багатьох програмістів. Незалежно від того, чи ви створюєте програму для підприємства або особистого використання, знання поточної дати може бути корисним для створення різних функцій та алгоритмів.

## Як

В PHP є кілька способів отримання поточної дати. Один з найпростіших і найпоширеніших - це використання функції `date()`. Ця функція приймає два аргументи: перший вказує формат дати, а другий - тільки опційний для вказівки поточного часу. Наприклад, якщо ми хочемо вивести поточну дату на екран у форматі "день.місяць.рік", ми можемо скористатися таким кодом:

```PHP
<?php
echo date("d.m.Y"); // 29.07.2021
?>
```

Іншим часто використовуваним прикладом є функція `getdate()`, яка повертає масив з інформацією про поточну дату та час в різних форматах. Наприклад:

```PHP
<?php
$date = getdate();
print_r($date); // Array ( [seconds] => 11, [minutes] => 30, [hours] => 17, [mday] => 29, [wday] => 4, [mon] => 7, [year] => 2021, [yday] => 209, [weekday] => Thursday, [month] => July, [0] => 1627571411 )
?>
```

## Глибше

Як вже згадувалося, функція `date()` приймає перший аргумент - формат дати. Цей аргумент може бути дуже різноманітним і дозволяє детально налаштувати виведення дати та часу. Наприклад, формат "l, d F Y, H:i:s" виведе на екран день тижня, дату, місяць, рік та час у 24-годинному форматі і форматі секунд. Більш детальну інформацію про доступні формати можна знайти в [офіційній документації PHP](https://www.php.net/manual/en/function.date.php).
