## Чому

Щоб зрозуміти, як працюють командні аргументи в PHP і як їх можна використовувати для полегшення роботи з програмами.

## Як це зробити

В PHP існує вбудована змінна `$argv`, яка містить масив аргументів виклику програми. Також існує `$argc`, яка містить кількість переданих аргументів. Завдяки цим змінним, можна отримати доступ до всіх переданих аргументів і їх значень.

```PHP
<?php
// Передали аргументи "John" та "Doe"
var_dump($argc); // 3
var_dump($argv); // ["script.php", "John", "Doe"]
```

Також можна використовувати функцію `getopt()`, яка приймає рядок з аргументами та шаблоном і повертає асоціативний масив, де ключами є передані флаги, а значеннями - їх значення.

```PHP
<?php
// Передали аргументи "-u John -p password"
$options = getopt("u:p:");
var_dump($options); // ["u" => "John", "p" => "password"]
```

## Розглянемо детальніше

Існує багато корисних функцій, які допомагають взаємодіяти з аргументами командного рядка. Наприклад, функція `array_slice()`, яка дозволяє обрізати масив `$argv` і отримати тільки необхідні аргументи.

```PHP
<?php
// Передали аргументи "a b c d"
$arguments = array_slice($argv, 1); // видаляємо перший елемент, який є назвою скрипта
var_dump($arguments); // ["a", "b", "c", "d"]
```

Також в PHP існує можливість передавати аргументи за допомогою GET-запиту. Для цього потрібно додати аргументи до URL-адреси, розділивши їх знаком `&`.

```PHP
<?php
// Передали аргументи "id=123&name=John"
parse_str($argv[1], $options); // перетворюємо рядок в асоціативний масив
var_dump($options); // ["id" => 123, "name" => "John"]
```
