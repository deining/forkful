## Чому

Програмування з використанням командного рядка є важливою навичкою для будь-якого PHP розробника. Це дозволяє працювати зі скриптами із зручнішим управлінням та покращує продуктивність розробки.

## Як

Існує кілька способів отримання даних з командного рядка в PHP. Один із них - використання змінної $argv, яка містить масив з переданими параметрами командної стрічки. Наприклад:

```PHP
$arguments = $argv;
var_dump($arguments);
```

При виклику скрипта з командного рядка з параметрами, ви отримаєте вивід, схожий на такий:

```
array(3) {
  [0]=>
  string(10) "script.php"
  [1]=>
  string(7) "param1"
  [2]=>
  string(7) "param2"
}
```

Також, ви можете використовувати функцію getopt() для отримання іменованих аргументів з командного рядка. Наприклад:

```PHP
$options = getopt("f:h");
$filename = $options['f']; // значення переданого параметра -f
$help = $options['h']; // значення переданого параметра -h
```

## Глибока занурення

Крім отримання переданих параметрів, ви також можете перевірити, чи наявні деякі прапорці у командній стрічці. Наприклад, для перевірки наявності прапорця "-v" можемо використати умову:

```PHP
if (in_array("-v", $argv)) {
    echo "Включений режим з високим рівнем деталізації."; // вивід для команди `php script.php -v`
}
```

Також зверніть увагу, що приклади, що подані вище, не описують обробку помилок та некоректних вхідних даних. При реальному використанні необхідно враховувати ці ситуації та гарно розроблювати логіку обробки винятків.
