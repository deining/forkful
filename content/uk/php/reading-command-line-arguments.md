## Чому

Нерідко у програмістів виникає потреба обробити аргументи командного рядка для своєї PHP програми. Це може бути корисно, коли вам потрібно передати деякі значення в процесі виконання програми або коли ви працюєте з розробкою веб-застосунків.

## Як

Для того, щоб прочитати аргументи командного рядка в PHP, вам потрібно використовувати глобальну змінну ```$argv```. Ця змінна містить масив із всіма переданими в програму аргументами командного рядка. Наприклад, якщо ви запустите свою програму з командою ```php index.php arg1 arg2```, то змінна ```$argv``` буде мати значення ```Array(0 => "index.php", 1 => "arg1", 2 => "arg2")```.

Окрім того, ви можете використовувати таку корисну функцію як ```getopt()```, яка дозволяє зчитати аргументи командного рядка в зручнішому форматі. Давайте подивимося на приклад:

```PHP
<?php

$options = getopt("u:p:h", ["username:", "password:", "help"]);

var_dump($options);
```

Якщо запустити цей код з командою ```php index.php -u username -p password -h```, то ви отримаєте такий результат:

```
Array (
    ["u"] => "username",
    ["p"] => "password",
    ["h"] => false,
    ["help"] => false
)
```

Ви можете використовувати ```getopt()``` для зчитування різних типів аргументів, таких як одинарні букви, подвійні букви або навіть аргументи зі значеннями.

## Вивчення глибоко

При роботі з аргументами командного рядка, важливо розуміти, що значення в ```$argv``` завжди будуть в першій функції, яка викликається в процесі виконання програми. Це означає, що якщо ви зчитуєте аргументи у внутрішній функції, то ви отримаєте пустий масив. Тому, рекомендується зчитувати аргументи відразу в головній функції вашої програми.

Також важливо пам'ятати, що аргументи командного рядка не обов'язково мусять бути передані в програму. Тому, перед тим як використовувати значення з ```$argv```, перевірте чи вони дійсно були передані.
