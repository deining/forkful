## Що і чому?
Читання аргументів командного рядка - це спосіб для програмістів отримати вхідні дані від користувачів через командну строку. Це особливо корисно для виклику PHP-скриптів з командного рядка та для розробки інструментів командного рядка. Завдяки цьому, програмісти можуть швидко і легко змінювати параметри своєї програми без необхідності перекомпіляції або змінювання початкових кодів.

## Як робити:
Нижче наведено приклад коду на PHP, який демонструє, як читати аргументи командного рядка та виводити їх:

```php
<?php
// Отримати всі аргументи командного рядка
$arguments = $_SERVER['argv'];

// Вивести кількість аргументів
echo "Кількість аргументів: " . count($arguments) . "\n";

// Вивести кожен аргумент окремо
foreach ($arguments as $arg) {
  echo "Аргумент: " . $arg . "\n";
}
```

При запуску цього коду з командного рядка з аргументами "Hello" та "world", ви отримаєте такий результат:

```
Кількість аргументів: 3
Аргумент: PHP
Аргумент: Hello
Аргумент: world 
```

## Занурення на глибину:
Читання аргументів командного рядка було додано до PHP у версії 4.3.0 в якості експериментальної можливості. У наступних версіях воно було стабілізовано та стало частиною основної функціональності. 

Альтернативою читанню аргументів командного рядка є використання змінних середовища, які також можна отримати через ```$_SERVER``` змінну. Проте, це може бути неефективним та нервовим методом, оскільки не всі змінні середовища будуть доступні.

Щоб детальніше ознайомитися з читанням аргументів командного рядка в PHP, рекомендуємо прочитати [документацію PHP](https://www.php.net/manual/en/features.commandline.php).
