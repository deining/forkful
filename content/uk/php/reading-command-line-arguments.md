## Чому

Програмісти часто стикаються зі ситуаціями, коли потрібно передати програмі певні параметри щодо її виконання. Це може бути шлях до файлу, налаштування чи будь-які інші дані. Командний рядок дозволяє вводити ці параметри безпосередньо при запуску програми, що робить процес налаштування більш зручним та швидким.

## Як

Для зчитування параметрів з командного рядка використовується змінна `$argv`, яка є масивом із введеними користувачем значеннями. Щоб отримати доступ до цих значень, потрібно вказати індекс елемента, аналогічно до доступу до елементів інших масивів.

```PHP
// Приклад використання командного рядка в PHP
<?php
// Перевіряємо, чи були введені параметри
if (isset($argv[1]) && isset($argv[2])) {
    echo "Ви ввели: " . $argv[1] . ", " . $argv[2];
}
else {
    echo "Будь ласка, введіть два параметри";
}
?>
```

Вихідні дані:

```bash
php script.php Hello World
Ви ввели: Hello, World
```

## Детальний огляд

Крім змінної `$argv`, також є змінна `$argc`, яка містить кількість переданих параметрів. Але варто зазначити, що вона завжди має значення, щонайменше 1, оскільки ввагається, що перший елемент масиву `$argv` є назвою виконуваного скрипту.

Крім того, існують різні функції, які допомагають зчитувати параметри з командного рядка, наприклад, `getopt()` та `argc()`.
