## Чому

Розширення YAML є дуже корисним засобом для роботи зі структурованими даними. Воно дозволяє легко зберігати та обробляти інформацію у зручному форматі, що робить його незамінним для програмістів.

## Як працювати з YAML

Почати працювати з YAML дуже просто. Це текстовий формат, який дуже схожий на звичайні конфігураційні файли. Давайте подивимося на приклад PHP коду, яким ми можемо зчитати дані з YAML файлу:

```PHP
<?php
// зчитуємо YAML файл у змінну $data
$data = yaml_parse_file('config.yaml');

// виводимо дані у зручному форматі
print_r($data);
```

Якщо у нас є файл "config.yaml" з таким змістом:

```yaml
name: John
age: 25
skills:
  - PHP
  - Python
  - JavaScript
```

Ми отримаємо наступний результат:

```
Array
(
    [name] => John
    [age] => 25
    [skills] => Array
        (
            [0] => PHP
            [1] => Python
            [2] => JavaScript
        )

)
```

Бачимо, що дані з YAML файлу були успішно зчитані та збережені у змінній $data. Далі ми можемо працювати з цими даними як з будь-яким масивом у PHP.

## Глибинний розгляд

Окрім зчитування даних з YAML файлів, ви також можете їх створювати та редагувати за допомогою PHP. Для цього ви можете використовувати функцію yaml_emit(), яка дозволяє створювати YAML строки з масивів данних.

Наприклад, якщо у вас є масив з даними про користувача, ви можете створити YAML строку наступним чином:

```PHP
<?php
// масив з даними про користувача
$user = [
  'name' => 'Kate',
  'age' => 30,
  'skills' => ['PHP', 'HTML', 'CSS']
];

// створюємо YAML строку з цього масиву
$yaml = yaml_emit($user);

// виводимо результат
echo $yaml;
```

Ми отримаємо наступний результат:

```yaml
name: Kate
age: 30
skills:
  - PHP
  - HTML
  - CSS
```

Також важливо знати, що дані у YAML форматі можуть бути багаторівневими. Наприклад, у нашому попередньому прикладі ми бачили, як можна створити масив з ключем "skills" та значенням, яке також є масивом. Таким чином, ми можемо створювати дуже складні структури даних, що дозволяє нам зручно зберігати та обробляти інформацію.
