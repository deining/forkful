## Чому?

Створення тимчасових файлів є необхідною частиною багатьох програм, особливо веб-додатків. Вони дозволяють тимчасово зберігати дані та ресурси, що необхідні для виконання певної операції, і очищати їх після завершення процесу. Це допомагає зменшити використання пам'яті та покращити продуктивність програми.

## Як?

Для створення тимчасового файлу використовуйте функцію `tmpfile()` в PHP. Наприклад:

```PHP
$file = tmpfile();
```

Ця функція повертає вказівник на створений тимчасовий файл, який можна використовувати для запису та читання даних. Крім того, функція автоматично видаляє файл після використання.

Нижче наведено приклад виводу файлу в браузері:

```PHP
$file = tmpfile();
fwrite($file, "Hello world!");
echo fread($file, 1024);
```

Результат:

Hello world!

## Глибоке погруження

Функція `tmpfile()` створює тимчасовий файл в системній тимчасовій папці. Це звичайно `/tmp` на Unix системах або `C:\Windows\Temp` на Windows. Але ви також можете використовувати іншу папку за допомогою функції `tmpfile()`, передаючи шлях до неї як аргумент:

```PHP
$file = tmpfile("/path/to/temp/folder/");
```

Крім того, ви можете використовувати функцію `tmpfile()`, щоб створити більш стабільний тимчасовий файл, застосовуючи перехоплення помилок та виклики `ftell()` та `fseek()`, як описано в [офіційній документації](https://www.php.net/manual/en/function.tmpfile.php).
