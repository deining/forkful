Що і чому?
Витягування підрядків - це процес отримання частини рядка зі строки. В програмуванні це важлива техніка, яку використовують для обробки даних та роботи зі змінними.

Як це зробити:
```PHP
// Приклад 1: Використання функції substr ()
$str = "Привіт світ!";
echo substr($str, 0, 6); // результат: Привіт

// Приклад 2: Використання оператора зрізу []
$str = "Hello world!";
echo $str[0]; // результат: H
echo $str[1]; // результат: e
echo $str[2]; // результат: l
echo $str[3]; // результат: l
echo $str[4]; // результат: o
```

Поглиблене дослідження:
Витягування підрядків було засновано в 1970-х роках і є одним із найбільш поширених методів обробки рядків у програмуванні. Є ще кілька альтернативних методів, таких як використання регулярних виразів або функцій заміни. Реалізація витягування підрядків може відрізнятися в залежності від мови програмування, але основна ідея залишається однаковою - отримати частину рядка зі строки.
