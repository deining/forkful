## Why
Доброго дня, шановні програмісти! Сьогодні ми говоритимемо про важливу тему у програмуванні - видобування підрядків (substrings). Це корисний інструмент, який допомагає нам працювати зі стрічками даних, що дозволяє зробити наш код більш гнучким і ефективним. Давайте з'ясуємо, чому варто вміти екстрагувати підрядки у PHP.

## How To
Для видобування підрядків у PHP ми будемо використовувати вбудовану функцію `substr()`. Ця функція приймає два обов'язкових параметри - початкову позицію і довжину підрядка. Давайте розглянемо декілька прикладів:

```
// Визначаємо вихідну стрічку
$string = "Привіт, світе!";

// Видобуваємо підрядок з позиції 7 до кінця
$newString = substr($string, 7);
// Результат: "світе!"

// Видобуваємо підрядок з позиції 0 довжиною 6 символів
$newString = substr($string, 0, 6);
// Результат: "Привіт,"
```

Додатково, ми можемо використовувати від'ємні значення для початкової позиції та довжини підрядка. Наприклад, `-5` вказуватиме на п'ять останніх символів стрічки, а `-1` - на останній символ.

```
// Визначаємо вихідну стрічку
$string = "Привіт, світе!";

// Видобуваємо останнє два символи
$newString = substr($string, -2);
// Результат: "те!"

// Видобуваємо п'ять останніх символів
$newString = substr($string, -5);
// Результат: "світе!"
```

## Deep Dive
У PHP є також багато інших корисних функцій для роботи зі стрічками, що доповнюють функцію `substr()`. Наприклад, `strpos()` для пошуку позиції певного символу у стрічці, `strlen()` для визначення довжини стрічки та `str_replace()` для заміни частини стрічки іншим текстом.

Також важливо зазначити, що підрядки можна екстрагувати не тільки зі звичайних стрічок, але й з масивів, асоціативних масивів та об'єктів. Це дозволяє нам працювати зі складнішими структурами даних та витягувати необхідну нам інформацію.
