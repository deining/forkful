## Чому

Виділення підрядків є важливою частиною програмування на PHP. Воно дозволяє знаходити та використовувати конкретну частину тексту, що спрощує обробку даних та створює зручність для користувачів.

## Як

Існує кілька способів витягування підрядків зі строки в PHP. Розглянемо основні з них за допомогою прикладів коду та виведення результатів.

### Використання `substr()`

Функція `substr()` дозволяє виділити підрядок зі строки з заданим початковим та кінцевим індексами. Приклад використання:

```PHP
<?php
$str = "Привіт, світ!";
$substring = substr($str, 7, 5); // Починаємо з індекса 7 та виділяємо 5 символів
echo $substring; // Виведе "світ"
?>
```

### Використання `strstr()`

Функція `strstr()` дозволяє знаходити рядки в підрядках та повертати все, що йде після знайденого рядка. Приклад:

```PHP
<?php
$str = "Це рядок, що містить слово слово";
$substring = strstr($str, "слово"); // Поверне "слово слово"
echo $substring; // Виведе "слово слово"
?>
```

### Використання `explode()`

Функція `explode()` дозволяє розділити строку на підрядки, використовуючи заданий роздільник та повернути масив із цих підрядків. Приклад:

```PHP
<?php
$str = "Розділіть цю строку!";
$substring = explode(" ", $str); // Розділяємо по пробілу
echo $substring[1]; // Виведе "цю"
?>
```

## Глибоке занурення

У PHP існують багато інших функцій та методів для виділення підрядків, таких як `substr_replace()`, `str_replace()`, `preg_replace()` та багато інших. Крім того, можна використовувати регулярні вирази для більш гнучкого пошуку та виділення підрядків.
