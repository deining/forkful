## Що & Чому?

Знаходження довжини рядка - це процес визначення кількості символів у рядку. Це дуже важлива операція для програмістів, оскільки вона дозволяє зрозуміти, скільки місця потрібно виділити для збереження рядка в пам'яті, а також для обробки даних за допомогою цього рядка.

## Як це зробити:

```PowerShell
$string = "Це мій перший рядок"
$string.length
```

**Вивід:** 21

## Поглиблене дослідження:

*Історичний контекст:* Знаходження довжини рядка є ключовою операцією у багатьох мовах програмування, проте спосіб її виконання може відрізнятися. Наприклад, у деяких мовах для цього використовується окрема функція, а у PowerShell достатньо викликати властивість `.length` для рядка.

*Альтернативи:* На практиці, для знаходження довжини рядка можна використовувати іншу операцію - порівняння з пустим рядком. Зазвичай, це працює швидше, але не завжди дає правильний результат, особливо якщо у рядку є спеціальні символи.

*Деталі реалізації:* Для знаходження довжини рядка PowerShell перебирає всі символи у рядку та підраховує їх кількість. Це робить операцію неефективною, особливо для довгих рядків.
