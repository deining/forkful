## Що і чому?

Використання обчислювальних виразів (regular expressions) є незамінною технікою для багатьох програмістів. Це потужне засіб пошуку та заміни тексту за допомогою певних шаблонів. Програмісти використовують регулярні вирази для ефективного та швидкого обробки текстових даних.

## Як?

{% PowerShell
$text = "Привіт, це мій номер телефону: 123-456-7890."
$text -match '\d{3}-\d{3}-\d{4}' # Виводить знайдені співпадіння ('123-456-7890')
$text -replace '\d{3}-\d{3}-(\d{4})', 'xxx-xxx-$1' # Замінює знайдене співпадіння на новий формат ('Привіт, це мій номер телефону: xxx-xxx-7890.')
%}

## Вдивіться глибше

Регулярні вирази були винайдені у 1950-х роках та стали стандартною технікою для обробки тексту у багатьох мовах програмування. Існують альтернативи, такі як використання підстрок та функцій для роботи з текстом, але вони не дають такої гнучкості та ефективності, як регулярні вирази.

Реалізація регулярних виразів у PowerShell базується на .NET Framework Regex класі. Це дозволяє використовувати повний набір можливостей та функцій для створення та роботи з виразами.
