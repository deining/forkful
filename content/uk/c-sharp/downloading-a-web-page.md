## Чому

Скачування веб-сторінки є важливою задачею для багатьох програмістів. Воно дає змогу отримати доступ до різних даних з Інтернету, наприклад, текстів, зображень або інших важливих ресурсів. Також, це дозволяє автоматизувати певні завдання та полегшує роботу з веб-додатками.

## Як це зробити

Для початку, нам необхідно встановити бібліотеку ```System.Net``` та імпортувати її у наш проект. Далі, необхідно створити об'єкт типу ```WebClient``` та застосувати до нього метод ```DownloadFile()```, який приймає два параметри - URL веб-сторінки та шлях для збереження файлу. Наприклад:

```C#
using System.Net;

WebClient client = new WebClient();
client.DownloadFile("https://example.com", "C:/Users/User/Desktop/example.html");
```

Цей код завантажить веб-сторінку з URL https://example.com та збереже її на робочий стіл користувача у файлі example.html.

## Глибоке дослідження

Подальшим розвитком даної функціональності може бути використання методу ```DownloadString()```, який дозволяє отримати текстовий вміст веб-сторінки. Також, можна додати обробку помилок та виводити відповідні повідомлення користувачеві.
