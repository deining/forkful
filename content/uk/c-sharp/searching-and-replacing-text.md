## Чому

Заміна тексту є важливим елементом програмування, який допомагає минути час та зусилля. Коли ви працюєте над великим проектом, вам може знадобитися змінити деякі елементи тексту у багатьох місцях. Використання пошуку та заміни тексту дозволить вам швидко та ефективно внести необхідні зміни.

## Як

Щоб замінити текст у своєму коді, ви можете скористатися методом `Replace()` в C#. Нижче наведено приклад коду, який замінює слово "hello" на "hi" у всіх рядках у змінній `text`:

```C#
string text = "hello world, hello Ukraine!";
string newText = text.Replace("hello", "hi");
Console.WriteLine(newText);
```
**Вихідний код:**

```
hi world, hi Ukraine!
```

Як бачимо з прикладу, метод `Replace()` замінює всі входження першого параметра на другий параметр у даному рядку. Крім того, ви можете задати третій параметр - кількість входжень, які потрібно замінити. Наприклад:

```C#
string text = "hello world, hello Ukraine!";
string newText = text.Replace("hello", "hi", 1);
Console.WriteLine(newText);
```
**Вихідний код:**

```
hi world, hello Ukraine!
```

Як бачимо, тепер метод `Replace()` замінив лише перше входження тексту "hello". Це може бути корисно, якщо ви не бажаєте заміняти всі входження.

## Глибоке занурення

Метод `Replace()` також може бути використаний для заміни окремих символів, наприклад, заміни символу "a" на "b":

```C#
string text = "abcde";
string newText = text.Replace('a', 'b');
Console.WriteLine(newText);
```
**Вихідний код:**

```
bbcde
```

Крім того, метод `Replace()` може бути використаний із рядками, які містять юнікод символи, що дозволяє замінювати текст у будь-якій мові.
