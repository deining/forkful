## Чому

У програмуванні іноді виникає потреба замінити певні частини тексту на інші. Це може бути необхідно для виправлення помилок або швидкої зміни форматування. В таких випадках корисно знати, як виконати пошук і заміну тексту у програмі на C#.

## Як це зробити

Існує кілька способів виконати пошук і заміну тексту у C#. Розглянемо приклади з використанням різних функцій та методів.

Найпростішим методом є використання методу `Replace()` для заміни всіх входжень певного рядка у тексті на інший рядок. Наприклад, використаємо текст "Привіт, світ!" та замінимо рядок "світ" на "Земля":

```C#
string text = "Привіт, світ!";
string newText = text.Replace("світ", "Земля");

Console.WriteLine(newText); // Виводиться "Привіт, Земля!"
```

Також можна використовувати регулярні вирази для більш складних пошуків і замін. Наприклад, замінимо у тексті всі цифри на зірочки:

```C#
string text = "Сьогодні 15 листопада";
string newText = Regex.Replace(text, @"\d", "*");

Console.WriteLine(newText); // Виводиться "Сьогодні ** листопада"
```

## Глибша розробка

При роботі з більшими текстовими файлами можна використовувати `StreamReader` та `StreamWriter`, щоб читати та записувати дані по частинам, а не усій файл цілком. Також варто звернути увагу на різноманітні методи виконання пошуку та заміни у тексті, такі як `IndexOf()`, `LastIndexOf()` та `Substring()`, що можуть бути дуже корисними в різних ситуаціях.
