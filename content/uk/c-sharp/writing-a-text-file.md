## Чому
Написання текстового файлу є необхідним елементом у багатьох програмах. Це дає змогу зберігати інформацію, з якою можна буде працювати пізніше, зберігаючи її в структурованому та доступному для зчитування форматі.

## Як
Написання текстового файлу у C# є простим і легким завданням. За допомогою класу `File` можливо використовувати метод `WriteAllText()` для створення та запису інформації у файл. Ось приклад коду для створення файлу з текстом "Привіт, світ!" та збереження його на робочому столі:

```C#
using System.IO;

string path = Environment.GetFolderPath(Environment.SpecialFolder.Desktop) + "\\hello.txt";
string text = "Привіт, світ!";
File.WriteAllText(path, text);
```

Як результат, у вас з'явиться файл з назвою "hello.txt" на вашому робочому столі, який буде містити текст "Привіт, світ!".

## Поглиблене дослідження
У C# також є багато інших функцій для роботи з текстовими файлами, таких як `ReadAllText()` для читання вмісту файлу, `AppendAllText()` для додавання тексту до вже існуючого файла та `Delete()` для видалення файлу. Також можливе використання класу `StreamWriter` для створення та записування інформації у файл. Для більш докладної інформації та прикладів, дивіться [документацію Microsoft](https://docs.microsoft.com/en-us/dotnet/standard/io/how-to-write-text-to-a-file).
