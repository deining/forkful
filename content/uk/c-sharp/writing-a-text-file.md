Що & Чому?
Написання текстового файлу - це процес створення файлу, який містить текстову інформацію. Програмісти виконують цю задачу, оскільки текстові файли використовуються для зберігання і передачі даних, а також для створення логів та налаштувань програм.

Як?
Нижче наведені приклади коду та виводу для написання текстового файлу у C#.

Написання нового текстового файлу:
```C#
using System;
using System.IO;

// Створення файлу за допомогою StreamWriter
StreamWriter file = new StreamWriter("newfile.txt"); 

// Запис даних у файл
file.WriteLine("Це є перший рядок у файлі");
file.WriteLine("Це є другий рядок у файлі");

// Закриття файлу
file.Close();

// Результат:
// Файл newfile.txt: 
// Це є перший рядок у файлі
// Це є другий рядок у файлі
```

Створення та запис до існуючого файлу:
```C#
using System;
using System.IO;

// Використання StreamWriter для дозапису до файлу
StreamWriter file = new StreamWriter("existingfile.txt", true); 

// Запис даних у файл
file.WriteLine("Це є третій рядок у файлі");

// Закриття файлу
file.Close();

// Результат:
// Файл existingfile.txt:
// Це є перший рядок у файлі
// Це є другий рядок у файлі
// Це є третій рядок у файлі
```

Глибший пошук
Написання текстових файлів існує з самого початку програмування. У минулому, для цього використовувалися інші формати, наприклад, бінарні файли. Зараз текстові файли є більш доступним та зрозумілим форматом збереження даних. Також існують інші способи запису даних, наприклад, робота з базами даних або XML форматом. Проте, написання текстових файлів залишається популярним, простим та надійним способом забезпечення даними.
