## Чому

Старт нового проекту може бути захопливим та викликаючим завданням для багатьох програмістів. Це дозволяє вам виявити свої навички та творчий потенціал у реальному середовищі, а також розширити свої знання та досвід у кодуванні. Незалежно від того, чи ви працюєте над особистим проектом або над комерційною програмою, це може бути захоплюючим викликом та можливістю для росту.

## Як створити проект за допомогою C#

Для початку нового проекту вам знадобиться деякий базовий набір інструментів. Ви можете вибрати IDE (інтегроване середовище розробки) за своїм вибором, таке як Visual Studio або Visual Studio Code. Надалі ми будемо використовувати Visual Studio для прикладів.

### Створення проекту

1. Відкрийте Visual Studio та оберіть "Create a new project" (Створити новий проект).
2. Оберіть мову C# та тип проекту, наприклад "Console App" (Консольний додаток).
3. Введіть назву проекту та оберіть розташування для вашого проекту.
4. Натисніть "Create" (Створити) та ваш новий проект буде створений.

### Кодування в C#

Надалі ми будемо створювати просту програму для обчислення середнього арифметичного введених чисел. Давайте спробуємо це!

1. Відкрийте `Program.cs` в своєму редакторі коду.
2. Введіть наступний код у відповідній функції:

```C#
double sum = 0; // створюємо змінну для зберігання суми чисел
int counter = 0; // створюємо змінну для зберігання кількості введених чисел
string input; // створюємо змінну для отримання введеного значення

Console.WriteLine("Введіть числа, розділити комою: "); // виводимо підказку користувачу
input = Console.ReadLine(); // зчитуємо введене значення в рядок

foreach (string num in input.Split(',')) // проходимо крізь кожне введене число
{
    double number = Convert.ToDouble(num); // перетворюємо рядок в число
    sum += number; // додаємо число до суми
    counter++; // збільшуємо лічильник на 1
}

double average = sum / counter; // обчислюємо середнє арифметичне
Console.WriteLine("Середнє арифметичне: " + average); // виводимо результат на екран
```
