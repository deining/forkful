## Що & Чому?

Читання текстового файлу - це процес, при якому програміст звертається до файлу, що містить текстову інформацію, і отримує доступ до цієї інформації. Це необхідно для зчитування вмісту файлу та подальшої обробки цієї інформації у програмі. Зазвичай це використовується для зчитування даних, що зберігаються у вигляді тексту, таких як налаштування програми або користувацький ввід.

## Як:

```
// Приклад читання текстового файлу і виведення його вмісту на консоль

using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Відкриття файлу для читання
        StreamReader reader = new StreamReader("file.txt");

        // Зчитування рядка з файлу
        string line = reader.ReadLine();

        // Виведення на консоль
        Console.WriteLine(line);

        // Закриття файлу
        reader.Close();

        // Забезпечення того, що консоль залишиться відкритою, доки користувач не натисне будь-яку клавішу
        Console.ReadLine();
    }
}
```

```
// Приклад читання текстового файлу і збереження його вмісту у масиві рядків

using System;
using System.IO;

class Program
{
    static void Main()
    {
        // Відкриття файлу для читання
        StreamReader reader = new StreamReader("file.txt");

        // Створення масиву рядків для збереження інформації з файлу
        string[] lines = new string[5];

        // Зчитування усіх рядків з файлу та збереження їх у масиві
        for (int i = 0; i < lines.Length; i++) 
        {
            lines[i] = reader.ReadLine();
        }

        // Виведення на консоль
        foreach (string line in lines)
        {
            Console.WriteLine(line);
        }

        // Закриття файлу
        reader.Close();

        // Забезпечення того, що консоль залишиться відкритою, доки користувач не натисне будь-яку клавішу
        Console.ReadLine();
    }
}
```

## Детальний аналіз:

### Історичний контекст:

Читання текстового файлу є однією з основних операцій, які виконуються при роботі з комп'ютерами. В минулому для цього використовувалися дуже прості методи, такі як відкриття файлу і читання його змісту по одному символу за раз. Однак, з появою високорівневих мов програмування, таких як C#, робота з файлами стала набагато зручнішою та ефективнішою.

### Альтернативи:

Літературне джерело, що представляють собою рядок тексту, можуть бути читані й іншими способами. Один з найпоширеніших способів - це використання об'єктів класу ```TextReader```. Клас ```StreamReader```, який був використаний у прикладах вище, є похідним від класу ```TextReader```. Клас ```TextReader``` надає більш розширені можливості для зчитування текстових файлів, такі як читання блоку даних замість одного символу.

### Деталі реалізації:

Для читання текстових файлів у C# використовуються два основних класи: ```TextReader``` та ```StreamReader```. Клас ```TextReader``` є базовим класом і не може бути використаний безпосередньо, але він надає ряд методів та властивостей, які полегшують роботу з текстовими файлами. Клас ```StreamReader``` є похідним від класу ```TextReader``` і надає більш розширені можливості для читання та обробки текстових файлів.
