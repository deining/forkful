## Чому

У цій статті ми розглянемо основи читання текстового файлу в програмуванні на C#. Вона буде корисна для початківців, які хочуть ознайомитися з цим процесом або розширити свої знання.

## Як

Читання текстового файлу є важливою частиною багатьох програм на C#. Це дозволяє програмістам отримати доступ до даних, збережених у файлі, і обробити їх для подальшого використання в програмі.

Найпростіший спосіб читати текстовий файл - за допомогою класу `StreamReader`. Цей клас містить метод `ReadLine()`, який дозволяє зчитувати файл по одному рядку за раз. Наприклад, якщо у нас є текстовий файл з наступним вмістом:

```
Привіт, світ!
Це перший рядок.
А це другий рядок.
```

Ми можемо зчитати цей файл за допомогою наступного коду:

```C#
using System.IO; // Підключення простору імен

// Створення змінної, яка зберігатиме шлях до файлу
string filePath = "textfile.txt";

// Створення нового екземпляра класу StreamReader, передавши шлях до файлу в якості аргументу
StreamReader reader = new StreamReader(filePath);

// Циклічно читаємо файл по одному рядку до тих пір, поки не дійдемо до кінця
string line;
while((line = reader.ReadLine()) != null)
{
    // Тут можна обробити рядок і виконати потрібні дії з ним
    Console.WriteLine(line); // Виводимо рядок на консоль
}

// Не забуваємо закрити ресурси
reader.Close();
```

Цей код виведе на консоль наступне:

```
Привіт, світ!
Це перший рядок.
А це другий рядок.
```

## Глибоке погруження

Окрім читання файлу по одному рядку за раз, ми також можемо зчитати весь його вміст у вигляді одного рядка за допомогою методу `ReadToEnd()` класу `StreamReader`. Це зручно, наприклад, якщо у нас є текстовий файл з великою кількістю даних.

```C#
// Читаємо весь файл у вигляді одного рядка
string content = reader.ReadToEnd();

// Виводимо цей рядок на консоль
Console.WriteLine(content);
```

Відмінністю між цими методами є те, що `ReadLine()` повертає один рядок, незалежно від того, скільки було всіх рядків у файлі, а `ReadToEnd()` повертає повністю весь вміст файлу.
