# Чому
Ви можете знайти себе у ситуації, коли потрібно видалити символи, які відповідають певному шаблону. Наприклад, ви можете мати рядок з числами, але вам потрібно знайти та видалити всі літери. У цьому випадку ви можете скористатися функцією видалення символів за шаблоном в C#.

# Як
Для початку, необхідно знайти функцію, яка відповідає вимогам. У нашому випадку, це була б функція "Trim" з параметрами "{'a', 'z'}", яка використовується для видалення літер від "a" до "z". Далі, потрібно викликати цю функцію на рядку, який містить символи, які потрібно видалити.

```C#
string str = "abc123";
str = str.Trim(new char[] {'a', 'z'});
Console.WriteLine(str); // Output: "123"
```

Якщо ви хочете видалити символи, які не підпадають під певний шаблон, наприклад, видалення всіх символів крім цифр, ви можете використати функцію "Replace" з параметром "{'a', 'z'}".

```C#
string str = "abc123";
str = str.Replace(new char[] {'a', 'z'}, "");
Console.WriteLine(str); // Output: "123"
```

# Deep Dive
Існує багато різних функцій для видалення символів за шаблоном, таких як "TrimStart", "TrimEnd", "Trim", "Replace", "Remove" та інші. Кожен з них має свої власні параметри та можливості видалення символів з рядка. Для ретельного ознайомлення з цими функціями, радимо ознайомитися з документацією Microsoft про роботу з рядками в C#.
