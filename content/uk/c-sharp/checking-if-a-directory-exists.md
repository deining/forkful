## Чому

Провірка існування директорії є важливою задачею у програмуванні, оскільки дозволяє впевнитися, що необхідна структура файли та папки існує і доступна для подальшої роботи.

## Як це зробити

Для перевірки існування директорії у C# ми можемо використати клас `Directory` з простору імен `System.IO`. Спочатку нам потрібно додати цей простір імен у наш проект за допомогою директиви `using`:

```C#
using System.IO;
```

Після цього ми можемо використати метод `Exists()` класу `Directory`, який приймає рядок з шляхом до директорії і повертає булеве значення, яке показує, чи існує ця директорія чи ні. Приклад коду може виглядати так:

```C#
string path = @"C:\Users\JohnDoe\Documents\ProjectFolder";

if (Directory.Exists(path))
{
    Console.WriteLine("Директорія існує.");
}
else
{
    Console.WriteLine("Директорія не існує.");
}
```

Для даного прикладу ми використовуємо змінну `path`, яка містить шлях до директорії, що буде перевірятися. Ми використовуємо оператор `@` перед рядком, щоб уникнути необхідності збільшувати кількість обернених слешів у шляху. У випадку, якщо директорія з вказаним шляхом існує, на консоль буде виведено повідомлення "Директорія існує." В іншому випадку, якщо директорія не існує, буде виведено повідомлення "Директорія не існує."

## Поглиблене дослідження

Метод `Exists()` також може бути виконаним для перевірки існування файлу. Це може бути корисно, якщо необхідно перевірити, чи існує конкретний файл у вказаній директорії. Для перевірки існування файлу ми можемо використати такий же метод `Exists()`, але вказати шлях до файлу замість шляху до директорії.

Наприклад, якщо нам потрібно перевірити існування файлу з назвою "data.txt" у директорії "C:\Users\JohnDoe\Documents\", ми можемо використати такий код:

```C#
string filePath = @"C:\Users\JohnDoe\Documents\data.txt";

if (File.Exists(filePath))
{
    Console.WriteLine("Файл існує.");
}
else
{
    Console.WriteLine("Файл не існує.");
}
```
