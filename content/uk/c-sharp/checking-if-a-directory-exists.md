## Чому

В програмуванні часто потрібно виконувати дії відносно файлів та папок. Перш ніж працювати з вмістом папки, потрібно встановити, що сама папка існує. Тому перевірка існування директорії є важливою частиною багатьох програм і дозволяє уникнути помилок при роботі з файлами і папками.

## Як

```C#
// Перевірка існування директорії за допомогою класу Directory з простору імен System.IO
if (Directory.Exists("C:/Users/User/Documents/")) 
{
    // Код, що виконується, якщо папка існує
    Console.WriteLine("Папка знаходиться в системі.");
}
else 
{
    // Код, що виконується, якщо папка НЕ існує
    Console.WriteLine("Папка не знайдена в системі.");
}

// Виведення результату на екран консолі
// У випадку наявності папки виведе: Папка знаходиться в системі.
// В іншому випадку виведе: Папка не знайдена в системі.
```

## Deep Dive

Перевірка існування директорії виконується за допомогою методу Exists класу Directory. Цей метод повертає булеве значення - true (якщо папка існує) або false (якщо папка не існує). Якщо папка існує, то ми можемо виконувати потрібні дії з її вмістом. У разі, якщо папка не існує, можна вивести повідомлення користувачу або створити нову папку, використовуючи метод CreateDirectory.
