## Чому

Якщо ви хочете обчислити дату у майбутньому або минулій, наприклад, для створення календаря або додатка для планування подій, то цей процес може бути складним і викликати багато помилок. У цьому матеріалі ми розглянемо, як можна використовувати C# для обчислення дати у майбутньому або минулому з мінімальними зусиллями.

## Як використовувати

Існує кілька способів обчислити дату у майбутньому або минулій за допомогою C#. Одним з найпростіших і найбільш часто використовуваних є використання вбудованих класів "DateTime", "TimeSpan" та "DateTimeOffset". Для початку нам потрібно підключити простір імен `System` та `System.Linq`, а потім визначити початкову дату та період в майбутньому або минулому, щоб обчислити кінцеву дату. Після цього ми можемо використати вбудовані методи, наприклад `Add()` або `Subtract()`, щоб отримати бажану дату.

```C#
using System;
using System.Linq;

// Встановлюємо початкову дату та період
DateTime startDate = new DateTime(2021, 1, 1);
TimeSpan period = new TimeSpan(10, 0, 0, 0);  // період в 10 днів

// Отримуємо кінцеву дату шляхом додавання або віднімання періоду
DateTime endDate = startDate.Add(period);  // обчислюємо дату у майбутньому
// DateTime endDate = startDate.Subtract(period);  // обчислюємо дату у минулому

Console.WriteLine(endDate);  // виводимо результат: 1/11/2021 12:00:00 AM

```

За допомогою інших методів, таких як `DateTime.Parse()` або `DateTime.TryParse()`, ми можемо також скористатися введеним користувачем форматом дати, щоб обчислити бажану дату в майбутньому або минулому.

```C#
using System;
using System.Linq;

// Введений користувачем формат дати
string userInput = "2021/12/31";

// Перетворюємо вхідний рядок у DateTime об'єкт
DateTime date = DateTime.Parse(userInput);

// Обчислюємо дату у майбутньому або минулому, додаючи або віднімаючи період
DateTime futureDate = date.AddDays(10);  //  обчислюємо дату у майбутньому
// DateTime pastDate = date.AddDays(-10);  // обчислюємо дату у минулому

Console.WriteLine(futureDate);  // виводимо результат: 1/10/2022 12:00:00 AM
```
