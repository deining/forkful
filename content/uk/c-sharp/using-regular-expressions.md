## Чому

У цій статті ми розглянемо регулярні вирази (regular expressions) в мові C#. Відтепер ви зможете швидше та легше виконувати пошук та заміну тексту в своїх програмах.

## Як використовувати регулярні вирази в C#

Щоб почати, потрібно імпортувати простір імен ```System.Text.RegularExpressions```. Далі, можна використовувати клас ```Regex``` для створення екземпляра регулярного виразу і використовувати його методи для пошуку та заміни тексту. Наприклад, заміна всіх пропусків у рядку на коми може бути здійснена наступним чином:

```C#
using System;
using System.Text.RegularExpressions;

string text = "Це приклад тексту з пропусками";
string pattern = @"\s+"; // шаблон для пошуку пропусків
string replacement = ","; // заміна на кому
string result = Regex.Replace(text, pattern, replacement);

Console.WriteLine(result); // виведе "Це,приклад,тексту,з,пропусками"
```

## Глибоке занурення

Регулярні вирази дозволяють виконувати дуже точний та гнучкий пошук та заміну тексту. Можна використовувати метасимволи, які означають певні шаблони, наприклад ```\d``` для пошуку цифр або ```\w``` для пошуку будь-якої літери або цифри. Також можна використовувати квантифікатори, які визначають кількість повторень певного шаблону. Наприклад, ```\d{3}``` буде шукати тільки три цифри. Для детальнішого ознайомлення з регулярними виразами, рекомендуємо переглянути офіційну документацію Microsoft.
