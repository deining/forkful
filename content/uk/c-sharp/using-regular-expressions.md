## З чого почати
Регулярні вирази - це потужний інструмент, який дозволяє здійснювати ефективний пошук та обробку текстової інформації. Використовуючи регулярні вирази, ви можете швидко та з легкістю здійснювати складні пошукові запити і заміни для різних застосувань, таких як парсинг, валідація даних та трансформація тексту.

## Як використовувати регулярні вирази в C#
Щоб почати використовувати регулярні вирази в C#, вам потрібно імпортувати простір імен System.Text.RegularExpressions. Потім ви можете використовувати клас Regex для створення об'єкта, який буде зберігати ваш регулярний вираз. Далі ви можете використовувати методи цього об'єкта для здійснення пошуку або заміни тексту за вашим шаблоном. Нижче наведений приклад коду для пошуку email адрес у тексті:
```C#
// Приклад створення регулярного виразу
Regex regex = new Regex(@"\w+@\w+\.\w+");
// Приклад пошуку в тексті
string text = "Email адреса: example@mail.com";
Match match = regex.Match(text);
// Виведення результату
Console.WriteLine("Знайдено: {0}", match.Value); // Виведе: example@mail.com
```
