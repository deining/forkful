## Чому

Існує багато причин, чому можна бути зацікавленим у використанні регулярних виразів у своїй С# програмуванні. Вони дозволяють ефективно шукати, змінювати та перевіряти рядки тексту, що дуже корисно у випадках, коли потрібно робити складні патерни пошуку.

## Як використовувати регулярні вирази в С#

Регулярні вирази можна використовувати в С# за допомогою класу Regex з простору імен `System.Text.RegularExpressions`. Давайте розглянемо декілька прикладів того, як з ними працювати.

```
// Пошук у тексті підстроки, що складається з 4 цифр
string text = "Число 1234 є дуже цікавим.";
string pattern = "\\d{4}"; // регулярний вираз для пошуку 4 цифр
Match match = Regex.Match(text, pattern); // шукаємо в тексті
Console.WriteLine(match.Value); // виводимо результат: 1234
```

```
// Заміна всіх входжень підстроки на іншу
string text = "Це речення містить слово кіт і кітеня.";
string pattern = "кіт"; // регулярний вираз для пошуку слова "кіт"
string replacement = "пес"; // слово, яке буде використовуватись для заміни
string result = Regex.Replace(text, pattern, replacement); // замінюємо усі входження
Console.WriteLine(result); // виводимо результат: Це речення містить слово пес і песеня.
```

## Глибоке дослідження

Регулярні вирази можна використовувати для розв'язання складних завдань. Вони мають багато різних можливостей, таких як:

- Використання метасимволів для пошуку певних шаблонів у тексті (наприклад, `\\d` для пошуку цифр або `\\w` для пошуку букв).
- Використання квантифікаторів для задання кількості повторень певного шаблону (наприклад, `\\d{3,5}` для пошуку чисел з 3 до 5 цифр).
- Використання групування для збереження певної частини тексту для подальшого використання (наприклад, `(\\w+)\\s(\\w+)` для збереження двох слів, розділених пробілом).

Більш детальну інформацію про використання регулярних виразів в С# можна знайти у [документації Microsoft](https://docs.microsoft.com/uk-ua/dotnet/standard/base-types/regular-expression-language-quick-reference).
