## Чому

Генерування випадкових чисел є важливою складовою будь-якої програми, яка потребує взаємодії з користувачем або вирішення випадкових задач. Такі числа можуть використовуватися для створення унікальних ідентифікаторів, перевірки даних, генерації випадкових подій або навіть для тестування програми.

## Як

Використовуючи мову програмування C#, ви можете згенерувати випадкове число за допомогою класу `Random`. Наприклад:

```C#
Random rnd = new Random(); // створити екземпляр класу Random
int randomNumber = rnd.Next(1, 100); // згенерувати випадкове число від 1 до 100
Console.WriteLine(randomNumber); // вивести згенероване число
```

В результаті ви побачите випадкове число, яке змінюється при кожному запуску програми. За проміжком від 1 до 100 ви можете вказувати будь-який діапазон чисел, який вам потрібен.

## Deep Dive

Клас `Random` використовується для генерування випадкових цілих чисел, але ви також можете використовувати його для генерування випадкових дробових чисел. Для цього ви можете використовувати метод `NextDouble()`, який повертає випадкове число від 0 до 1.

```C#
Random rnd = new Random();
double randomDecimal = rnd.NextDouble(); // згенерувати випадкове дробове число від 0 до 1
Console.WriteLine(randomDecimal); // вивести згенероване число
```

Також можна замість створення нового екземпляру класу `Random` використовувати один і той самий екземпляр для кожного випадкового числа. Це дозволить уникнути повторного генерування одного і того ж числа при близьких запусках програми.

```C#
Random rnd = new Random(); // створити екземпляр класу Random
int randomNumber1 = rnd.Next(1, 100); // згенерувати перше випадкове число
int randomNumber2 = rnd.Next(1, 100); // згенерувати друге випадкове число
int randomNumber3 = rnd.Next(1, 100); // згенерувати третє випадкове число
```

Якщо ж ви будете використовувати один і той самий екземпляр класу, то кожен наступний виклик методу `Next()` буде повертати нове випадкове число.

За допомогою класу `Random` ви можете також генерувати випадкову послідовність символів для створення випадкових рядків. Для цього ви можете використовувати вбудований метод `NextBytes()`, який заповнює масив байтів випадков
