## Чому
Згенерувати випадкові числа може бути корисно для багатьох програм, таких як ігри, статистичні аналізи та криптографічні алгоритми. Випадковість допомагає отримати непередбачувані результати, що робить програми більш реалістичними та безпечними.

## Як
Існує декілька способів згенерувати випадкові числа в C#, але ми розглянемо найбільш простий та часто використовуваний - клас `Random`. Цей клас має різні методи для генерації цілих чисел, дійсних чисел та байтів. Ось приклади коду та відповідний вихід:

```C#
// Імпортуємо простір імен для класу Random
using System;

// Створюємо об'єкт класу Random
Random rnd = new Random();

// Генеруємо випадкове ціле число в діапазоні від 0 до 10
int number = rnd.Next(0, 11);
Console.WriteLine(number); // Виводиться, наприклад, 7

// Генеруємо випадкове дійсне число від 0 до 1
double num = rnd.NextDouble();
Console.WriteLine(num); // Виводиться, наприклад, 0,264658492746

// Генеруємо випадковий байт
byte b = new byte[1];
rnd.NextBytes(b);
Console.WriteLine(b[0]); // Виводиться, наприклад, 137
```

Зверніть увагу, що для отримання різноманітних результатів, зазвичай необхідно викликати метод `Next()` кілька разів, наприклад, `rnd.Next(0, 11)`.

## Заглиблення
Клас `Random` використовує генератор псевдовипадкових чисел для отримання результатів. Це означає, що згенеровані числа насправді не є випадковими, але створюються за допомогою математичної формули. Також слід бути обережними при використанні методу `Random` для криптографічних цілей, оскільки результати можуть бути передбачуваними для зловмисників.
