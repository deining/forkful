У цій статті ми дізнаємося, як створювати випадкові числа в програмуванні C# та чому це корисно.

## Чому

Згенеровані випадкові числа можуть бути корисними для багатьох сценаріїв програмування, включаючи тестування, генерацію випадкових даних для симуляцій та створення унікальних ідентифікаторів для об'єктів.

## Як

Для генерації випадкових чисел використовується клас `Random`. Спочатку необхідно його ініціалізувати, передаючи в конструктор мінімальне та максимальне значення, які можуть бути згенеровані.

``` C#
Random random = new Random();
int randomNum = random.Next(1, 100); // згенерує випадкове число від 1 до 100
```

Щоб згенерувати випадкове дробове число, можна використовувати метод `NextDouble()`.

``` C#
double randomDecimal = random.NextDouble(); // згенерує випадкове дробове число від 0.0 до 1.0
```

Також можна використовувати метод `NextBytes()`, щоб заповнити масив байтів випадковими значеннями.

``` C#
byte[] buffer = new byte[4];
random.NextBytes(buffer); // заповнює масив байтів випадковими значеннями
```

## Deep Dive

У класі `Random` є декілька методів та властивостей, які дозволяють налаштовувати процес генерації випадкових чисел. Наприклад, метод `Next()` дозволяє вибрати максимальне значення, яке може бути згенероване.

``` C#
int randomMax = random.Next(100); // згенерує випадкове число від 0 до 100
```

Також можна використовувати метод `NextBytes()` для генерації більш складних випадкових об'єктів, наприклад, масивів цілих чисел.
