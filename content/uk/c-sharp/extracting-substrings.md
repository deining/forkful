##Чому
У програмуванні, часто потрібно працювати з текстовими даними, але іноді необхідно обробляти лише частину тексту. Для цього існує процес вилучення підрядків, який дозволяє отримати необхідну частину інформації з тексту для подальшої обробки.

##Як це зробити
Одним із способів вилучення підрядків є використання методу Substring() в C#. Для цього необхідно виконати наступні дії:

1. Створити рядок, з якого потрібно вилучити підрядок.
2. Використовуючи метод Substring(), вказати початковий і кінцевий індекси для вибору бажаної частини рядка.
3. Обробити знайдений підрядок за необхідною логікою.

Наприклад, якщо у вас є рядок "Привіт, як справи?", і ви хочете витягнути тільки слово "Привіт", то код буде виглядати наступним чином:

```C#
string message = "Привіт, як справи?";
string greeting = message.Substring(0, 6);
Console.WriteLine(greeting);
```

В результаті на екран виведеться слово "Привіт".

##Глибока занурення
Метод Substring() може бути використаний не тільки для вилучення частини рядка, але і для виконання різноманітних дій з текстом. Наприклад, ви можете використати цей метод для перевірки, чи містить рядок певне слово, або для заміни певних символів у тексті.

Також, метод Substring() дозволяє вибирати частину тексту не тільки по індексах, але і за допомогою ключового слова "length". Наприклад, якщо у нас є рядок "Hello World!", і нам потрібно вилучити слово "World", то можна використати наступний код:

```C#
string message = "Hello World!";
string world = message.Substring(6, 5);
Console.WriteLine(world);
```

В результаті на екран виведеться слово "World".
