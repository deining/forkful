## Чому

В цій статті ми розглянемо, як використовувати функцію виокремлення підрядків в мові програмування C#, а також подивимося на деталі цього процесу. Виокремлення підрядків дозволяє нам отримувати частину тексту зі строки, що використовується для подальшого аналізу або обробки. Це корисно для роботи зі словниками, рядками часу або будь-якими іншими текстовими даними.

## Як

Почнемо з декількох прикладів використання функції `Substring()` для виокремлення підрядків.

```C#
// Приклад 1: Виокремлення 5 символів з рядка
string myString = "Це прикладний текст";
string substring = myString.Substring(4, 5);

Console.WriteLine(substring);
// Виведе: прикла

// Приклад 2: Виокремлення підрядку за допомогою індексів
string myString = "Це прикладний текст";
string substring = myString.Substring(11);

Console.WriteLine(substring);
// Виведе: текст
```

Як бачимо, функція `Substring()` приймає два аргументи: початковий індекс та кількість символів, які необхідно виокремити. Також є можливість використовувати тільки початковий індекс, якщо потрібно виокремити підрядок до кінця строки.

## Глибоке погруження

Щоб краще розібратися з функцією `Substring()`, слід знати, як працюють індекси в мові програмування C#. Індексація починається з 0, тобто перший символ у рядку буде мати індекс 0, другий - 1 і так далі. Також важливо знати, що індекси можуть бути від'ємними, що означає початок рядка з кінця. Наприклад, індекс -1 буде означати останній символ у рядку.

Також варто згадати про те, як функція `Substring()` поводить себе у випадку, якщо вказаний індекс виходить за межі рядка. У такому випадку, вона поверне помилку. Тому потрібно бути обережними при використанні цієї функції і перевіряти правильність індексів.
