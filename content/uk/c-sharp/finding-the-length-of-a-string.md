## Навіщо

У цій статті ми поговоримо про пошук довжини рядка у програмуванні на мові C#. Вчення цього простого, але важливого концепту дозволить нам розуміти, які інші операції можуть бути виконані зі строками, а також ефективно працювати з ними.

## Як це зробити

Існує кілька шляхів отримати довжину рядка у мові C#. Найпростіший спосіб - використання властивості `Length` на об'єкті типу `string`. Наприклад:

```C#
string myString = "Привіт світ!";
Console.WriteLine(myString.Length);
// Outputs: 11
```

Існують також деякі методи, які дозволяють виконувати більш складні операції зі строками, такі як `Split()` для розділення рядка на частини або `Substring()` для отримання підрядка з основного рядка. Давайте розглянемо приклад коду з використанням цих методів:

```C#
string myString = "Привіт світ!";
string[] array = myString.Split(" "); // Розділяє рядок на два елементи, використовуючи пробіл як роздільник
Console.WriteLine(array[0]); // Outputs: Привіт
Console.WriteLine(array[1]); // Outputs: світ!
string subString = myString.Substring(3, 5); // Видає з основного рядка підрядок, який починається з третього символу і має 5 символів
Console.WriteLine(subString); // Outputs: віт с
```

## Глибока погруження

Щоб більше зрозуміти як саме працює пошук довжини рядка у мові C#, важливо знати, що це число символів, яке ми можемо побачити у консолі, включаючи пробіли та розділові знаки. Однак, багато з нас часто не враховували ці символи при обчисленні довжини рядка, тому це деталь, яку варто уточнити.

Існує також можливість використовувати різні кодування знаків при роботі з рядками у мові C#. Наприклад, Unicode зазвичай використовується при роботі з міжнародними символами та мовами. Це може впливати на кількість символів та довжину рядка.
