## Чому

Довжина рядка - це важлива операція в програмуванні, особливо в C#. Підрахувати довжину рядка дозволяє вам зробити різні маніпуляції з вхідними даними, наприклад, перевірити, чи не перевищує рядок максимально допустиму довжину чи знайти певну підрядок.

## Як це зробити

Для початку нам потрібно створити змінну, в якій будемо зберігати наш рядок. Наприклад:

```C#
string name = "Україна";
```

Тепер, для того, щоб дізнатись довжину рядка, ми можемо використати вбудовану функцію `Length`. Наприклад:

```C#
int length = name.Length; 
```

Давайте перевіримо, чи це працює, виведемо результат у консоль:

```C#
Console.WriteLine(length); 
```

Ви повинні побачити, що довжина рядка "Україна" дорівнює 7. Також важливо зазначити, що пропуски і символи пунктуації також враховуються в довжині рядка.

## Глибокий занурення

Наше рішення працює з використанням вбудованої функції `Length`, але давайте трохи розглянемо, як вона працює під капотом. Вона використовує властивість `Length` класу `String`, яка повертає значення типу `int` - довжину рядка. Цей метод використовується для обчислення кількості Unicode-символів у рядку, тому кириличні символи та символи інших алфавітів також будуть враховані.
