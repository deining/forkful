Що & Навіщо?

Знаходження довжини рядка - це процес вимірювання кількості символів у заданому рядку. Програмісти часто роблять це, щоб зрозуміти, наскільки великі дані їм потрібно обробляти або для перевірки коректності введених даних.

Як:

```C#
string myString = "Привіт, світ!";
int length = myString.Length;
Console.WriteLine(length); // Результат: 12
```

Глибинне занурення:

У минулому, для вимірювання довжини рядка потрібно було використовувати складні алгоритми або мати спеціальну функцію у своїй програмі. Однак, у сучасних мов програмування, таких як C#, можна просто використовувати вбудований метод Length, що значно спрощує роботу.

Також, у деяких інших мов програмування, довжина рядка може бути виміряна не тільки за кількістю символів, але і за байтами чи бітами.

Для отримання точної довжини рядка, слід уникати використання методу Length у поєднанні з іншими функціями, такими як Substring. Це може призвести до неправильного результату через наявність нелітерних символів.
