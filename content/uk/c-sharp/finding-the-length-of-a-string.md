## Чому

Найбільш поширеною операцією у багатьох програмах є знаходження довжини рядка. Це корисне для різних завдань, наприклад, для перевірки, чи введено користувачем достатньо символів у пароль або для обрізання зайвих символів з тексту. У цій статті ми розглянемо, як у програмі на C# можна знайти довжину рядка та навіщо це потрібно.

## Як

У C# існує метод `Length`, який можна використовувати для знаходження довжини рядка. Наведемо приклад використання цього методу:

```C#
string str = "Привіт, світ!";
int length = str.Length;
Console.WriteLine("Довжина рядка: " + length); // Виведе: Довжина рядка: 13
```

У цьому прикладі ми створили змінну `str`, яка містить рядок "Привіт, світ!". За допомогою методу `Length` ми знаходимо довжину цього рядка та виводимо її на екран.

Пам'ятайте, що довжиною рядка є кількість символів у рядку, включаючи пробіли та пунктуаційні знаки.

## Deep Dive

Щоб краще зрозуміти, як працює метод `Length`, потрібно зрозуміти, що таке рядок у програмуванні. У C# рядок є послідовністю символів, яку представляють за допомогою класу `string`.

Метод `Length` насправді є властивістю цього класу, яка повертає ціле число - довжину рядка. Унікальним для .NET є те, що довжина рядка обчислюється тільки один раз під час створення рядка і потім зберігається для швидшого доступу у майбутньому.
