## Чому
При працюванні зі строками в програмуванні часто виникає необхідність перевести рядок в нижній регістр. Це може бути необхідно для подальшої обробки та порівняння даних, або для забезпечення консистентності даних. У цій статті ми розглянемо, як це зробити в C#.

## Як виконати
Щоб перевести строку в нижній регістр в C#, використовуйте метод `ToLower()`. Наприклад:
```C#
string str = "HELLO WORLD";
string lowerCaseStr = str.ToLower();
Console.WriteLine(lowerCaseStr);
```
Виконавши цей код, ви побачите наступний результат:
```
hello world
```
Цей метод повертає копію вихідного рядка у нижньому регістрі, не змінюючи оригінальний рядок. Якщо вам потрібно змінити оригінальний рядок, то використовуйте метод `ToLowerInvariant()`, який також зменшує регістр, але використовує міжнародну стандартну нижню модифікацію регістру.

Іншим значущим варіантом є використання методу `ToLower(CultureInfo)`, який дозволяє вказати конкретну культуру для перетворення регістру. Наприклад:
```C#
string str = "ВІТАЮ СВІТ";
string lowerCaseStr = str.ToLower(new CultureInfo("ua-UA"));
Console.WriteLine(lowerCaseStr);
```
Виконавши цей код, ви побачите наступний результат:
```
вітаю світ
```

## Глибоке дослідження
Окрім вищезгаданих методів, в C# також є можливість використання логіки перетворення регістру за допомогою циклу та таблиці ASCII-кодів. Проте, використання методу `ToLower()` є більш зручним та ефективним способом для перетворення регістру строк.
