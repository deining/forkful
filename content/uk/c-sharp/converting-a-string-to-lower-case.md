## Чому

Існує багато ситуацій, коли конвертація рядка до нижнього регістру є необхідною. Наприклад, для порівняння різних рядків у рівних умовах або для забезпечення правильної роботи програми, яка вимагає рядків у нижньому регістрі.

## Як

```C#
string input = "HELLO WORLD";
string output = input.ToLower(); //output: hello world
Console.WriteLine(output);
```

У цьому прикладі ми використовуємо метод `ToLower()` для конвертації рядка `input` до нижнього регістру. Цей метод повертає новий рядок зі зміненим регістром.

## Детальний аналіз

У C# існує кілька способів конвертації рядка до нижнього регістру. Найбільш часто використовується метод `ToLower()`, який ми розглянули в попередньому прикладі. Однак, цей метод не підтримує всі символи, тому у деяких ситуаціях може знадобитися використання інших методів, наприклад `ToLowerInvariant()`. Також варто враховувати, що конвертація рядка до нижнього регістру може бути дещо повільнішою за конвертацію до верхнього регістру.
