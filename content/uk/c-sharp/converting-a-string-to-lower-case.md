## Чому
Перетворення рядка в нижній регістр є важливим кроком у багатьох програмах, оскільки це дозволяє легше здійснювати пошук та порівняння даних. Крім того, це також є прийнятною практикою при роботі з введеними користувачем даними, щоб уникнути помилок через різницю в регістрах.

## Як
```C#
string input = "Convert To Lower Case"; //створюємо рядок для конвертації
string output = input.ToLower(); //використовуємо вбудовану функцію для перетворення в нижній регістр
Console.WriteLine(output); //виводимо результат на консоль
```
Вихід: "convert to lower case"

Цей приклад демонструє швидкий і простий спосіб перетворити рядок в нижній регістр за допомогою вбудованої функції ToLower (). Також можна використовувати цю функцію при порівнянні двох рядків, щоб уникнути будь-яких проблем з регістром.

## Глибоке занурення
Щоб краще зрозуміти процес перетворення рядка в нижній регістр, варто детальніше розглянути роботу цієї функції. Коли ми викликаємо ToLower () на рядку, він повертає новий рядок, який складається зі всіх символів з вихідного рядка, перетворених в нижній регістр. Це не змінює оригінальний рядок, але повертає новий об'єкт рядка з конвертованим значенням.

Крім того, важливо зазначити, що ToLower () використовує культурні налаштування поточного об'єкта для здійснення конвертації. Це означає, що для різних мов можуть використовуватись різні правила перетворення регістра, тому варто бути обережним при використанні цієї функції у великих міжнародних проектах.
