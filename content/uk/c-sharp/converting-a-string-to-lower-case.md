# Що і навіщо?

Зменшення регістру рядків - це процес перетворення всіх літер у рядку на нижній регістр. Це корисна властивість для програмістів, оскільки дозволяє легше порівнювати дві рядки, не зважаючи на їх регістр. Це також може спростити деякі операції з рядками, такі як пошук та заміна.

# Як це зробити:

```C#
string myString = "Привіт Світ!";
Console.WriteLine(myString.ToLower());
```
Вивід: ```привіт світ!```

# Подробиці:

Зменшення регістру рядка є потужним інструментом, який використовується в багатьох мовах програмування, включаючи C#. Воно дозволяє додавати рядки у незаперечний регістр, що може бути корисно в деяких ситуаціях. Альтернативними методами зменшення регістру є використання функцій регулярних виразів або всіляких додаткових бібліотек.

Імплементація зменшення регістру в C# є досить простою, оскільки ця мова має вбудовану функцію ```ToLower()```, яка автоматично змінює всі літери в рядку на нижній регістр.
