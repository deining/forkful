## Чому

Програмування може бути захоплюючою та цікавою галуззю, але інколи з прогресом процесу виникають проблеми. Одна з таких задач може бути читання аргументів командного рядка. Це важлива частина процесу створення програм та підтримання їх працездатності. Тому, якщо ви хочете більш детально дізнатися про цей елемент програмування, продовжуйте читати цей пост.

## Як це зробити

Використовуючи мову програмування C#, є декілька способів читання аргументів командного рядка. Один з них - використання класу Environment, який надає доступ до змінних середовища та аргументів командного рядка. Ось приклад коду, який демонструє цей підхід:

```C#
static void Main(string[] args)
{
    string[] arguments = Environment.GetCommandLineArgs();
    // arguments[0] - назва програми
    // arguments[1] - перший аргумент командного рядка
    // arguments[2] - другий аргумент командного рядка і т.д.
}
```

Якщо ви хочете передати більш складні аргументи, як, наприклад, кілька назв файлів або шляхів, використовуйте клас System.Collections.Specialized.NameValueCollection. Ось як це виглядає у коді:

```C#
static void Main(string[] args)
{
    string[] arguments = Environment.GetCommandLineArgs();
    NameValueCollection parameters = new NameValueCollection();
    // arguments[0] - назва програми
    // arguments[1] - перший аргумент командного рядка
    // arguments[2] - другий аргумент командного рядка і т.д.
    for (int i = 1; i < arguments.Length; i++)
    {
        string[] parts = arguments[i].Split('=');
        parameters.Add(parts[0], parts[1]);
    }
}
```

Цікавим фактом є можливість читати аргументи, передані в звичайному форматі запиту URL. Наприклад, якщо ваше програмне забезпечення приймає URL `https://yoursite.com/yourapp?param1=value1&param2=value2`, то цей запит можна перетворити на аргументи командного рядка і прочитати їх за допомогою вищенаведеного коду.

## Глибока погляд

Іншим способом читання аргументів командного рядка є використання класу System.Diagnostics.ProcessStartInfo. Це дозволяє запустити інший процес та передати йому аргументи. Ось приклад коду, який демонструє цей підхід:

```C#
static void Main(string[] args)
{
    ProcessStartInfo startInfo = new ProcessStartInfo("program.exe");
    // Дозволяє передати аргументи командного рядка
    startInfo.Arguments = "/arg1=value1 /arg2=value2";
    Process.Start(startInfo);
}
```

Зверніть увагу, що цей підхід не дозволить вам читати аргументи, передані під час запус
