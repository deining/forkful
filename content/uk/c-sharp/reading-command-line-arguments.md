## Чому

Програмування може бути складною справою, але часто є важливою для розв'язання різних завдань. Одним зі способів полегшити собі життя є робота з аргументами командного рядка. Вони дозволяють передавати різноманітну інформацію до програми без необхідності введення її вручну. Це особливо корисно для розробки скриптів або автоматизації процесів.

## Як

Для зчитування аргументів командного рядка використовується клас `Environment`. Він містить змінну `CommandLine`, яку можна використовувати для доступу до переданих аргументів. Давайте подивимося на приклад:

```C#
using System;

class Program
{
    static void Main(string[] args)
    {
        string[] arguments = Environment.CommandLine.Split(' ');

        Console.WriteLine("Кількість аргументів: " + arguments.Length);
        Console.WriteLine("Перший аргумент: " + arguments[0]);
        Console.WriteLine("Другий аргумент: " + arguments[1]);

        // Приклад виклику програми:
        // myprogram.exe arg1 arg2
        // Виведе:
        // Кількість аргументів: 3
        // Перший аргумент: myprogram.exe
        // Другий аргумент: arg1
    }
}
```

Зверніть увагу, що аргументи передаються як рядок, тому потрібно приводити їх до бажаного типу.

## Поглиблене вивчення

Крім стандартного способу використання класу `Environment`, існує ще кілька цікавих можливостей. Наприклад, ви можете використовувати простір імен `System.CommandLine` для більш продвинутих функцій, таких як парсинг аргументів за допомогою атрибутів або створення власних опцій для аргументів. Також варто приділити увагу обробці неправильно заданих аргументів, щоб уникнути помилок в роботі програми.
