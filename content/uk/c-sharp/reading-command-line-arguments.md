## Для чого
Цей код призначений для тих, хто бажає навчитися читати аргументи командного рядка у своїх програмах на C#.

## Як це зробити
Перш за все, потрібно імпортувати простір імен System у свою програму:

```C#
using System;
```

Далі, ми можемо використовувати готову функцію `Environment.GetCommandLineArgs()` для отримання всіх аргументів командного рядка:

```C#
string[] args = Environment.GetCommandLineArgs();
```

Тепер ми можемо працювати з цим масивом `args`, наприклад, виводити кожен аргумент окремо:

```C#
for (int i = 0; i < args.Length; i++)
{
    Console.WriteLine("Аргумент №{0}: {1}", i, args[i]);
}
```

Якщо ми запустимо нашу програму з наступними аргументами: `dotnet run аргумент1 аргумент2`, то ми отримаємо такий результат:

```
Аргумент №0: "dotnet.exe"
Аргумент №1: "argument1"
Аргумент №2: "аргумент2"
```

## Глибокий занурення
У деяких випадках може бути потрібно обробляти аргументи командного рядка інакше в залежності від їх типу або значення. Для цього можна використовувати різні методи обробки рядків, такі як `int.Parse()` або `bool.TryParse()`. Також можна використовувати бібліотеку `CommandLineParser`, яка дозволяє легко і зручно обробляти аргументи командного рядка з використанням атрибутів.
