## Чому

Зчитування аргументів командного рядка є важливим аспектом програмування, оскільки воно дозволяє зберігати та передавати дані у вигляді аргументів при запуску програми. Це дозволяє зробити програми більш гнучкими та зручними у використанні.

## Як це зробити

Існує декілька способів зчитування аргументів командного рядка у C# програмах. Один із них - використання методу ```Main()``` та зчитування параметрів з масиву ```args```. Для цього ви можете скористатися циклом ```foreach``` та умовними операторами, щоб обробити передані аргументи. Нижче показано приклад коду та його виведення:

```C#
static void Main(string[] args)
{
    int sum = 0;
    foreach (string arg in args)
    {
        if (int.TryParse(arg, out int num))
        {
            sum += num;
        }
    }
    Console.WriteLine($"Сума переданих аргументів: {sum}");
}
```

При запуску програми з аргументами ```1 2 3```, результат буде виглядати наступним чином:

```bash
> програма.exe 1 2 3
Сума переданих аргументів: 6
```

## Поглиблене дослідження

Хоча приклад коду вище є простим та зрозумілим, у реальному житті може знадобитися більш складніша обробка аргументів командного рядка, наприклад, обробка ключових флагів або використання нестандартного формату передачі даних. Для цього можна використовувати спеціальні бібліотеки, наприклад, ```CommandLineParser```, які дозволяють ефективно та зручно читати аргументи з командного рядка.
