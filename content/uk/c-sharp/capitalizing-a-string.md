# Що & Чому?

У програмуванні існує поняття "капіталізація рядка", яке означає зміну першої літери рядка на велику. Програмісти зазвичай використовують цей метод для полегшення читабельності коду та для забезпечення однорідності стилю в програмі.

# Як це зробити:

```C#
string myString = "hello world";
string capitalizedString = myString.ToUpper();

Console.WriteLine(capitalizedString);
// Output: HELLO WORLD
```
У цьому прикладі ми використовуємо вбудовану функцію `ToUpper()` для зміни першої літери на велику. Існують також інші способи здійснити капіталізацію рядка, наприклад, підключення додаткових бібліотек або власноручний напис коду.

# Глибше в капіталізацію рядка:

Історично, зміна першої літери на велику була популярна в засобах обробки тексту та шрифтів. В програмуванні, цей метод здобув популярність завдяки його корисності та семантичному значенню. 

Існують також альтернативні шляхи капіталізації рядка, такі як зміна регістру використовуваних символів або використання різних кодувань. Використання капіталізації рядка є більш універсальним та простим підходом до полегшення роботи з текстом.
