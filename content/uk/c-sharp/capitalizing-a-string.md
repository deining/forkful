## Як

Завдання даної статті - показати вам, як використовувати метод "ToTitleCase" для великих літер в будь-якому рядку в C#. Цей метод змінює рядок таким чином, що перші символи кожного слова будуть великими літерами. Давайте розглянемо приклад коду:

```C#
string text = "привіт усім";
text = CultureInfo.CurrentCulture.TextInfo.ToTitleCase(text);
Console.WriteLine(text);
```
Вивід: "Привіт Усім"

Ви можете побачити, що метод "ToTitleCase" перетворив наш рядок деяким чином. Це корисна функція для роботи з даними, наприклад з назвами країн, міст чи книг.

## Глибоке занурення

Коли ми говоримо про використання методу "ToTitleCase", варто знати, що це не є ідеальним способом для перетворення рядка на великі літери. Наприклад, цей метод не правильно обробляє рядки з апострофами, дужками чи подвійними літерами. Також варто пам'ятати, що результати цього методу можуть різнитися в залежності від встановленої мови системи. Так, якщо ваша мова системи - українська, то метод "ToTitleCase" буде правильно перетворювати слова з української мови на великі літери.
