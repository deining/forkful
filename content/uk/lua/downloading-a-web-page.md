## Що і чому?
Завантаження веб-сторінки - це процес отримання контенту з інтернету і зберігання його на вашому комп'ютері. Програмісти часто роблять це, щоб отримати доступ до даних, які вони можуть використовувати для створення програм або для аналізу даних.

## Як?
```Lua
local http = require("socket.http") -- завантажуємо бібліотеку для роботи з мережевими запитами
local response, status, headers = http.request("https://www.example.com/") -- виконуємо запит до веб-сторінки
print(response) -- виводимо отриманий контент
```
Вище наведений код використовує бібліотеку "socket" для роботи з мережевими запитами і здійснює запит до веб-сторінки за допомогою функції "request". Результатом буде збережений контент з веб-сторінки. Можна також задати додаткові параметри, такі як заголовки і метод запиту.

## Глибинний погляд
Історично, завантаження веб-сторінок було важливим для розробки програм для роботи з інтернетом. Однак, існує багато інших способів отримати дані з інтернету, такі як API і веб-сервіси. Також, важливо знати, що не всі веб-сторінки дозволяють виконувати запити, тому під час роботи з ними необхідно дотримуватися їх правил.
