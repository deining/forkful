Що це таке та чому це важливо?
Перевірка наявності каталогу у програмуванні означає перевірку, чи існує певна директорія на комп'ютері або веб-сервері. Це важлива операція, яку розробники використовують для забезпечення правильної роботи програми та попередження можливих помилок.

Як це зробити?
Перевірку наявності каталогу можна здійснити за допомогою функції ```doesdirectoryexist()```, яка приймає шлях до директорії як аргумент. Якщо директорія існує, функція поверне значення ```true```, якщо ні - ```false```. Ось приклад коду:

```lua
if doesdirectoryexist("/home/user/images/") then
  print("Директорія існує!")
else
  print("Директорія не знайдена.")
end
```

## Поглиблене вивчення
Історичний контекст: перевірка наявності каталогу є важливою частиною програмування на багатьох мовах, у тому числі і на мові Lua. Раніше для цього використовувалися більш складні методи, але з появою функції ```doesdirectoryexist()``` цей процес став значно простішим.

Альтернативи: існують інші способи перевірки наявності каталогу, такі як використання модуля ```lfs``` (обговорювати його буде за межами цієї статті), але функція ```doesdirectoryexist()``` є найпростішим та найбільш зрозумілим варіантом.

Деталі реалізації: функція ```doesdirectoryexist()``` використовує системний виклик для перевірки наявності каталогу, тому її роботу можна залежно від операційної системи. Для цілей цієї статті ми припускаємо, що ми використовуємо Lua на Linux або Windows.
