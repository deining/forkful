		<h2>Що і для чого?</h2>
		<p>Робота з текстовими файлами - це процес зчитування та обробки текстових даних з файлу. Це важлива складова програмування, оскільки дає можливість зчитувати та використовувати збережені дані для подальшої обробки та відображення користувачеві.</p>
	</section>
	<section>
		<h2>Як це зробити?</h2>
		 ```Lua
		-- Приклад коду для зчитування текстового файлу
		local file = io.open("file.txt", "r") -- створення змінної, що зберігає відкритий файл
		local contents = file:read("*all") -- читання всього вмісту файлу та збереження його у змінну
		file:close() -- закриття файлу
		print(contents) -- виведення змісту файлу у консолі
		```
		
		<code>file.txt</code> - це шлях до текстового файлу, який ми хочемо зчитати.<br>
		<code>*all</code> - це параметр, який вказує, що необхідно зчитати всі дані з файлу. Інші параметри можуть вказувати на читання певної кількості символів або рядків.<br>
		<code>print(contents)</code> - виведе зміст файлу у консолі, але ви можете зробити з ним що завгодно, наприклад, обробити дані та вивести їх у вікні програми.
	</section>
	<section>
		<h2>Глибше: історичний контекст, альтернативи та деталі реалізації</h2>
		<p>Читання текстових файлів є однією з основних операцій в більшості мов програмування, оскільки це дає можливість роботи зі збереженими даними. У Lua є багато інших способів роботи з текстовими файлами, таких як функції <code>io.lines()</code> та <code>io.input()</code>, але <code>io.open()</code> є найпростішим та найчастіше використовуваним. При обробці тексту з файлу, важливо пам'ятати про читання та запис лише певної кількості символів або рядків, оскільки це може зменшити кількість використаної пам'яті та покращити продуктивність програми.</p>
