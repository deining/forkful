## Що і Чому?

Друкування відладочного виводу - це процес виведення додаткової інформації під час виконання програми для виявлення та виправлення помилок. Програмісти роблять це, щоб з'ясувати де саме виникають проблеми в їх коді та як їх виправити.

## Як використовувати:

```Lua
-- Використання вбудованої функції `print` для виведення повідомлення на екран
print("Привіт, світ!")

-- Використання функції `io.write` для виведення повідомлення без нового рядка
io.write("Привіт,") 
io.write("світ!")

-- Додавання додаткових даних в відладочний вивід
local age = 25
print("Мій вік: " .. age)
```

Вихід:

```
Привіт, світ!
Привіт, світ!
Мій вік: 25
```


## Глибше вирубки:

Друкування відладочного виводу було стандартним методом виявлення помилок в програмуванні вже десятки років. Однак, зростання складності програм та з'явлення інших методів відлагдки, таких як дебагери, призводить до того, що друкувати виводи стає менш популярним методом.

Альтернативою друкуванню виводу може бути використання логування, тобто збереження деякої інформації про виконання програми у файл для подальшого аналізу. Також, можна використати додатковий модуль для Lua, наприклад [DebugLib](https://github.com/debuglib).

Особливістю друкування виводу в Lua є те, що вони виводяться на екран, а не в файл як у більшості мов програмування. Це робить відлагдку більш зручною для використання під час розробки програм.
