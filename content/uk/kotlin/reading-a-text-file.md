## Що і Навіщо? 
Читання текстового файлу є процесом отримання інформації з вмісту файлу. Програмісти використовують цю функціональність, щоб зчитати дані з файлів, які містять велику кількість інформації, наприклад, бази даних або конфігураційні файли.

## Як використовувати: 
Нижче наведені приклади коду Kotlin для читання текстового файлу та виводу його вмісту на консоль.

```Kotlin
// Читання файлу за допомогою вбудованої функції readText()
val fileContent = File("file.txt").readText()
println(fileContent)

// Читання файлу за допомогою буферізованого читача
val reader = BufferedReader(FileReader("file.txt"))
var line = reader.readLine()
while (line != null) {
    println(line)
    line = reader.readLine()
}
```

Вихідний код буде виглядати наступним чином:

```
Це текстовий файл.
Він містить деяку інформацію.
Це приклад вмісту, який може бути прочитаний програмою.
```

## Глибше вглиб: 
Читання текстового файлу є однією з основних функцій в програмуванні. Раніше, коли комп'ютери використовувалися для обробки великих обсягів даних, важливим було знати, як працювати з текстовими файлами. Сьогодні це все ще важливо, оскільки багато програм обробляють текстову інформацію, і читання файлів є важливою частиною цього процесу. Альтернативними способами читання текстових файлів є використання вбудованої функції ```readLines()``` або да-класу ```inputStream()```. При цьому потрібно врахувати, що ці способи можуть бути менш ефективними за розміром пам'яті, і використання буферізованого читача може бути більш практичним.
