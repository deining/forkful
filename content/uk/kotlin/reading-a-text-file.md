## Чому

Коли ви працюєте з програмуванням, часто потрібно зчитати вміст файлу для подальшої обробки. Це може бути здійснене за допомогою Kotlin в декілька простих кроків.

## Як

Для зчитування текстового файлу у Kotlin, спочатку необхідно відкрити файл за допомогою `File` класу. Для цього скористайтеся таким кодом:

```Kotlin
val file = File("шлях/до/файлу.txt")
```

Після цього, ми можемо створити `Scanner` об'єкт для читання файлу. Це можна зробити наступним чином:

```Kotlin
val scanner = Scanner(file)
```

Наступним кроком є виконання читання файлу за допомогою`nextLine()` функції. Це дозволить нам отримати рядок з файлу. Наприклад:

```Kotlin
val firstLine = scanner.nextLine()
```

Також можна використовувати цей підхід для читання всього файлу циклом за допомогою `while` умови. Наприклад:

```Kotlin
while (scanner.hasNextLine()) {
    val line = scanner.nextLine()
    println(line)
}
```

Ці кодові блоки можна додавати в свій код для зчитування текстового файлу у Kotlin.

## Глибоке вивчення

Крім вищезазначених кроків, можна зробити ще кілька корисних речей під час читання текстового файлу у Kotlin. Наприклад, ви можете установити розділитель рядків за допомогою `scanner.useDelimiter()` функції. Це дозволить нам читати розділені рядки під час читання файлу.

Також, можна використовувати `Charsets` клас для задання кодування файлу під час читання. Наприклад:

```Kotlin
Scanner(file, Charsets.UTF_8)
```

Це дозволить нам читати файл з відповідним кодуванням, якщо він був створений у різних кодуваннях.
