## Чому 

Чи ви коли-небудь хотіли прочитати вміст текстового файлу, але не знали, як це зробити? Текстовий файл може містити різноманітну інформацію, таку як тексти, дані або налаштування. Іноді може бути необхідно прочитати і обробити цю інформацію в своїй програмі. Якщо ви хочете дізнатися, як вчити свою програму читати текстовий файл, ця стаття саме для вас.

## Як це зробити

Найпростішим способом прочитати текстовий файл у Kotlin є використання класу `File` і його методу `readText()`. Для цього нам потрібно створити екземпляр класу `File` з шляхом до нашого текстового файлу і викликати на ньому метод `readText()`.

```Kotlin
val file = File("path/to/your/file.txt")
println(file.readText())
```

Це виведе вміст вашого текстового файлу на екран. Якщо ваш файл дуже великий, ви можете розділити його на декілька рядків для зручності. Наприклад:

```Kotlin
val file = File("path/to/your/file.txt")
file.readLines().forEach { println(it) }
```

Цей код розділить файл на список рядків і виведе їх по одному на кожному рядку.

Якщо ви хочете зберегти вміст файлу у змінну для подальшої обробки, ви можете зробити це таким чином:

```Kotlin
val file = File("path/to/your/file.txt")
val content = file.readText()
// Далі ви можете працювати зі змінною content
```

Наведені приклади використовують шлях до файлу в форматі рядка. Але ви також можете використовувати об'єкт класу `URI`, якщо вам зручніше:

```Kotlin
val file = File("path/to/your/file.txt")
val uri = file.toURI()
println(uri.readText())
```

## Deep Dive

Крім методу `readText()`, клас `File` має також інші методи для читання текстових файлів. Один із них - `readLines()`, який був згаданий раніше. Він повертає список рядків з файлу. Ви також можете використовувати методи `inputStream()` і `reader()` для отримання вхідного потоку або читача для вашого файлу. Це дозволяє більш гнучко працювати з файлом та обробляти його частинами або під час читання.

Наприклад, ви можете використовувати `reader()` разом з `useLines()` для обробки кожного рядка поодинці, що може бути корисним при роботі з великими файлами:

```Kotlin
val file = File("path/to/your/file.txt")
file.reader().useLines { lines ->
    lines.forEach { println(it) }
}
```
