## Чому

Програмування в Kotlin - це захоплива та корисна навичка, яка може зробити ваше життя легшим та більш організованим. Читання текстового файлу є важливою частиною програмування, оскільки це дозволяє вам зберігати та обробляти великі обсяги даних. Тож, якщо ви хочете покращити свої навички в кодуванні, перейти до рівніше програмування або просто бачити конкретний приклад, то ця стаття для вас.

## Як

Для початку давайте створимо простий текстовий файл під назвою "data.txt", який містить такі рядки:

```
Привіт!
Це мій перший текстовий файл у Kotlin.
Я радий поділитися ним з вами.
```

Тепер, коли у нас є файл, ми можемо перейти до написання коду для його зчитування. Використовуючи Kotlin, ми можемо використовувати функцію `readText ()`, яка дозволяє нам читати текстові файли. Нижче показано приклад коду, який зчитує наш файл і виводить його вміст:

```Kotlin
fun main() {
    val file = FileReader("data.txt")
    val content = file.readText()
    println(content)
}
```

Вихідним результатом буде:

```
Hello!
This is my first text file in Kotlin.
I'm happy to share it with you.
```

Як бачимо, за допомогою функції `readText ()` ми зручно зчитали вміст файлу та вивели його на екран. Також варто зазначити, що для кращої практики навколо роботи з файлами, ми використали блок `try-catch` для обробки можливих помилок при читанні файлу.

## Глибоке закопання

Крім простого зчитування файлів, Kotlin також має багато інших інструментів для роботи з ними. Наприклад, ви можете використовувати клас `BufferedReader` для послідовного зчитування рядків з файлу, або використовувати функцію `readLines ()` для зчитування всіх рядків та збереження їх у змінну типу `List`.

Зверніть увагу, що при зчитуванні файлу важливо використовувати правильні шляхи до файлу. Наприклад, ви можете використовувати абсолютний шлях, якщо ваш файл знаходиться у конкретному місці на вашому комп'ютері, або використати відносний шлях, якщо файл знаходиться в тій же папці, що й ваш код.
