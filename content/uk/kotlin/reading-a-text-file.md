# Чому

Часто, у процесі програмування, виникає необхідність прочитати та обробити дані з текстового файлу. Це може бути потрібно для імпорту даних, обробки їх у програмі чи просто для перегляду вмісту файлу. Щоб мати змогу виконати ці дії, важливо мати знання про роботу з текстовими файлами у мові програмування Kotlin.

# Як

Зчитування текстового файлу у мові Kotlin є простою задачею. Спочатку необхідно відкрити файл за допомогою класу `File`, передавши йому шлях до файлу в якості параметру. Потім можна використати метод `readText()` для отримання текстового вмісту файлу у вигляді рядка.
```Kotlin
val file = File("path/to/file.txt")
val content = file.readText()
```
Для зчитування файлу по рядках, можна використати метод `readLines()`, який поверне список рядків. Також існує можливість вказати кодування файлу за допомогою параметру `charset`.

```Kotlin
val file = File("path/to/file.txt")
val lines = file.readLines(charset = Charsets.UTF_8)
```

Якщо файл дуже великий та його вміст необхідно обробити по частинам, можна використати метод `forEachLine()`. Цей метод дозволяє виконати певну операцію з кожним рядком у файлі.

```Kotlin
val file = File("path/to/file.txt")
file.forEachLine { line ->
    // операції з рядком
}
```

# Глибока зглибина

Крім вищезазначених методів, Kotlin має багато інших функцій для роботи з текстовими файлами. Наприклад, можна використовувати методи `bufferedReader()` та `bufferedWriter()` для зчитування та запису даних у файл. Такі методи дають можливість оптимізувати процес роботи з файлом та підвищити швидкість операцій.

Також важливо пам'ятати про обробку помилок, які можуть виникнути при роботі з файлами. Необхідно користуватися конструкцією `try/catch` для перехоплення винятків та виконання відповідних дій.
