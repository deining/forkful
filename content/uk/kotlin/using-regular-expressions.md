## Чому

Регулярні вирази є потужним інструментом для роботи з текстовими даними. Вони дозволяють швидко та ефективно виконувати пошук, заміну та перевірку рядків за певними правилами. Використання регулярних виразів може зменшити витрати часу та зусиль на обробку текстового вводу.

## Як

```Kotlin
val pattern = """[A-Z]+\d{2}-[A-Z]+\d{3}""" 
val string = "ABCDE19-XYZ123"
val match = Regex(pattern).matches(string) 
println(match) 
```

В цьому прикладі ми використовуємо регулярний вираз для перевірки, чи відповідає рядок певному шаблону. У нашому полірні ми перевіряємо, чи рядок має на початку п'ять великих літер, потім дві цифри, тире та знову три великих літери перед трьома цифрами. Вивід програми буде `true`, так як наш рядок відповідає шаблону.

## Глибоке дослідження

Існує багато різних методів та функцій, які можна використовувати з регулярними виразами у Kotlin. Наприклад, ви можете використовувати метод `split()` для розбиття рядка на підрядки за певним роздільником або `replace()` для заміни певних частин рядка на інші значення. Також є можливість створення незмінних регулярних виразів для більш швидкого та ефективного використання.
