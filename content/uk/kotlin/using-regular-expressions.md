## Чому

Застосування регулярних виразів дає можливість більш ефективно та точно обробляти текстові дані, шукати певні шаблони та замінювати відповідно до цих шаблонів. Це важливий інструмент для розробки програм, які мають великий обсяг текстової інформації.

## Як

Перш за все нам потрібно імпортувати бібліотеку для роботи з регулярними виразами. Для цього можна використовувати ключове слово "import" та вказати назву бібліотеки - "kotlin.text.Regex". Після цього можна починати писати регулярні вирази.

Наприклад, якщо нам потрібно знайти всі слова, які починаються з літери "а", то ми можемо скористатися наступним кодом:

```Kotlin
val text = "Абрикосовий сік - смакові коктейлі з латексних листків."
val pattern = Regex("^а\\w*")
val results = pattern.findAll(text)
results.forEach { println(it.value) }
```

В результаті виконання цього коду, ми отримаємо наступний вивід:

```
Абрикосовий
а
```

Цей приклад демонструє основні засоби для роботи з регулярними виразами, а саме: використання спеціальних символів та методів для пошуку в тексті.

## Глибока поглиблення

Існує багато спеціальних символів, які можна використовувати при створенні регулярних виразів в Котлін. Найбільш популярні з них це:

- "^" - символ, який позначає початок рядка;
- "$" - символ, який позначає кінець рядка;
- "." - символ, який відповідає будь-якому символу;
- "\d" - символ, який відповідає будь-якій цифрі;
- "\D" - символ, який відповідає будь-якому символу, що не є цифрою;
- "\s" - символ, який відповідає пробілу;
- "\S" - символ, який відповідає будь-якому символу, що не є пробілом;
- "\w" - символ, який відповідає букві або цифрі;
- "\W" - символ, який відповідає будь-якому символу, крім букви або цифри.
