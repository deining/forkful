## Чому

В прикладних програмах, часто виникає потреба змінити регістр в рядку тексту, подібно до переконвертування всіх літер великого вузла. У цьому блозі ми покажемо, як за допомогою Kotlin це можливо зробити з легкістю.

## Як

В Kotlin є вбудована функція capitalize(), яка приймає рядок тексту і повертає той самий рядок, з першою літерою великого регістру, а всі інші залишаються незмінними. Подивіться на приклад коду нижче:

```Kotlin
fun main() {
    val name = "котяча морда"
    print(name.capitalize()) // виведе "Котяча морда"
}
```

У цьому прикладі ми використовуємо функцію capitalize() на змінній name, яку ми створили з рядком "котяча морда". Зверніть увагу, що функція capitalize() не змінює змінну, але повертає новий рядок з великою першою літерою. Також, важливо пам'ятати, що функція враховує правила правильного написання, тому якщо наш початковий рядок буде містити велику літеру на початку, то функція поверне рядок без змін.

Також, є можливість визначити власну кастомну функцію capitalize(), яка буде виконувати таку саму операцію. Подивіться на приклад коду нижче:

```Kotlin
fun String.customCapitalize(): String {
    return this.substring(0, 1).toUpperCase() + this.substring(1)
}

fun main() {
    val name = "котяча морда"
    print(name.customCapitalize()) // виведе "Котяча морда"
}
```

У цьому прикладі ми визначаємо кастомну функцію capitalize() для класу String, яка буде виконувати таку саму операцію - першу літеру зробить великою, а решту лишить незмінними.

## Deep Dive

Функція capitalize() використовує метод replaceFirst(capitalize, "") для зміни першої літери в регістрі. Цей метод, як і назва вказує, замінює перше входження зазначеної підстроки (в нашому випадку capitalize) на передане значення (у нашому випадку пустий рядок). Це підтверджує, що функція capitalize() працює правильно як і для кирилиці, так і для латиниці.
