## Що & Чому?
Капіталізація рядків - це процес перетворення першої букви кожного слова в рядку на велику. Це зроблено для зручності читання та поліпшення естетичного вигляду тексту. Багато програмістів роблять це, щоб зробити свій код більш читабельним та професійним.

## Як:
```Kotlin
fun capitalizeString(str: String): String {
    val words = str.split(" ") // розбиваємо рядок на окремі слова
    val result = StringBuilder() // створюємо об'єкт, щоб зберігати новий рядок

    words.forEach {
        result.append(it.capitalize()).append(" ") // капіталізуємо кожне слово та додаємо до результуючого рядка
    }
    return result.toString().trim() // повертаємо результат та видаляємо зайві пробіли в кінці
}

fun main() {
    val str = "це цікаве речення для прикладу."
    println(capitalizeString(str)) // виведе "Це Цікаве Речення Для Прикладу."
}
```

## Глибоке Занурення:
Капіталізація рядків належить до сфери форматування тексту, яке виникло в друкувальній справі в середині XV століття. У сучасному програмуванні, особливо в ООП, капіталізація є важливим елементом, щоб зробити код читабельним і зрозумілим для інших програмістів. Є також інші способи форматування рядків, такі як верхній регістр або нижній регістр, але капіталізація є стандартом для читабельного коду. У Kotlin також є вбудована функція ```.capitalize()```, яка дозволяє капіталізувати тільки першу букву рядка без необхідності власноруч прописувати логіку.
