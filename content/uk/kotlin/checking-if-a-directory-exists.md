## Чому

Перевірка існування директорії є важливою для багатьох задач програмування, таких як перевірка наявності файлів, збереження даних у відповідних місцях і багато іншого. Ця стаття надає корисні вказівки щодо перевірки наявності директорій у мові Kotlin.

## Як

Існує декілька способів перевірити наявність директорії у мові Kotlin. Розглянемо декілька прикладів за допомогою ```Kotlin if``` умови і виведемо результат за допомогою ```println()``` функції.

```
if (File("c:/test").exists()) {
  println("Директорія існує")
} else {
  println("Директорія не існує")
}

// Вивід: Директорія не існує
```

Зверніть увагу, що умова ```File("c:/test").exists()``` використовує шлях до директорії, яку ми перевіряємо. Також, якщо ви працюєте на мобільному пристрої, наприклад, Android, ви можете використовувати ```Context.getDir()``` метод для отримання шляху до директорії і перевірити її наявність за допомогою того ж правила.

```
if (context.getDir("test", Context.MODE_PRIVATE).exists()) {
  println("Директорія існує")
} else {
  println("Директорія не існує")
}

// Вивід: Директорія не існує
```

## Глибока підтримка

Для більш точної перевірки існування директорії, можна використовувати метод ```File.isDirectory()```, який повертає значення ```true``` якщо зазначена директорія існує і є дійсною директорією. Також, ви можете додати перевірку на доступність директорії за допомогою методу ```File.canRead()```, який повертає значення ```true``` якщо директорія доступна для читання.

```
val directory = File("c:/test")

if (directory.isDirectory && directory.canRead()) {
  println("Директорія існує та доступна для читання")
}

// Вивід: Директорія існує та доступна для читання
```
