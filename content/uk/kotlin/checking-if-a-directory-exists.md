## Чому

Перевірка існування директорії є важливим елементом розробки програмного забезпечення. Завдяки цій функції можна переконатися, що потрібна директорія існує та виконувати певні дії з її вмістом.

## Як

Для перевірки існування директорії використовується метод `exists()` з класу `java.io.File`. Спочатку необхідно створити об'єкт `File`, передавши у конструктор шлях до директорії. Потім за допомогою методу `exists()` можна перевірити, чи існує ця директорія. Приклад коду:

```Kotlin
val directory = File("/path/to/directory")
if (directory.exists()) {
    println("Directory exists")
} else {
    println("Directory does not exist")
}
```
Результат виконання програми буде залежати від того, чи існує директорія за вказаним шляхом. Якщо директорія існує, на екран буде виведено повідомлення "Directory exists", інакше - "Directory does not exist".

## Глибоке погруження

Перевірка існування директорії може бути використана для багатьох різних сценаріїв, наприклад для перевірки наявності певного файлу або для перевірки доступу до директорії перед виконанням деяких дій з її вмістом. Також, ця функція може бути корисною при роботі зі зберіганням даних, де необхідно перевіряти наявність потрібної директорії перед записом або зчитуванням даних з неї.
