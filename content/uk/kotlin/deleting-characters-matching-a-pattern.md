## Чому
Ви, можливо, замітили, що іноді у вашому тексті з'являються непотрібні символи, які не потрібно вказувати при роботі з ним. Видалення цих символів може полегшити вашу роботу та зробити текст більш зрозумілим та легко читаємим.

## Як
```Kotlin
val text = "Привіт! Як справи?"
val pattern = Regex("[!?]")
val result = text.replace(pattern, "") // Результат: Привіт Як справи
println(result)
```

У цьому прикладі ми використовуємо функцію `replace` для заміни символів, які задовільняють патерн `"[!?]"`, на порожні стрічки. Таким чином, ми видаляємо всі символи питання та окличні знаки зі стрічки `text`.

Для більш складних виразів, ви можете використовувати більш гнучкий клас `Regex`, який дозволяє вам задавати складніші шаблони. Також ви можете комбінувати патерни для знаходження та видалення різних символів.

```Kotlin
val text = "Ця стрічка містить 123 числа, @ та $"
val pattern = Regex("[0-9@\$]")
val result = text.replace(pattern, "") // Результат: Ця стрічка містить числа, та
println(result)
```

Якщо ви бажаєте видалити всі символи окрім певних, ви можете використовувати інвертований патерн. Наприклад, якщо ви хочете залишити тільки букви та цифри, ви можете використовувати `[^\w]` в якості патерну.

```Kotlin
val text = "Займаймось програмуванням разом!"
val pattern = Regex("[^\\w]")
val result = text.replace(pattern, "") // Результат: Займаймосьпрограмуваннямразом
println(result)
```

## Глибоке дослідження
При видаленні символів за допомогою патернів, важливо розуміти, як саме вони працюють. Патерни це розширений механізм для пошуку та заміни певних рядків у тексті. Вони дозволяють використовувати метасимволи, які представляють певні класи символів, такі як цифри, букви, пропуски тощо.

Наприклад, для знаходження всіх цифр у стрічці, ми можемо використовувати патерн `"[0-9]"`, де `0` та `9` представляють межі класу цифр. Аналогічно для букв, ми можемо використовувати `"[a-zA-Z]"`, де `a` та `z` представляють межі класу літер, а `A` та `Z` - межі класу великих
