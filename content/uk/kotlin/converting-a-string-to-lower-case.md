## Чому

'Напевно, кожен із нас мав досвід роботи з рядками в програмуванні. І, ймовірно, було потрібно перетворити рядок на малий або великий регістр. Щоб зробити це, ми можемо скористатися функцією lowerCase(), яка перетворить всі великі літери на малі. У цій статті ми розглянемо, як саме це зробити в мові Kotlin.'

## Як це зробити

'Для початку, нам потрібно мати рядок, який ми хочемо перетворити на малий регістр. Наприклад, "Привіт, Kotlin!" За допомогою функції lowerCase(), ми можемо зменшити регістр цього рядка шляхом виклику його на змінну:

```Kotlin
val text = "Привіт, Kotlin!"
val lowerCaseText = text.lowerCase()
println(lowerCaseText)
```

Результатом виконання цього коду буде виведення на консоль "привіт, kotlin!" з малими літерами.'

## Вглиб

'У наведеному прикладі ми використали вже готову функцію lowerCase(), але ви можете також використовувати метод toLowerCase() на об'єкті типу String для досягнення того ж результату. Наприклад:

```Kotlin
val text = "Привіт, Kotlin!"
val lowerCaseText = text.toLowerCase()
println(lowerCaseText)
```

Але є ще один спосіб змінити регістр рядка в мові Kotlin - це використання функції replace() з регулярним виразом:

```Kotlin
val text = "Привіт, Kotlin!"
val lowerCaseText = text.replace(Regex("[A-Z]"), { it.value.toLowerCase() })
println(lowerCaseText)
```

Функція replace() виконує заміну всіх заголовних літер (вказаних у регулярному виразі) на їхні малі відповідники. Якщо необхідно, ви також можете використовувати регулярний вираз для заміни малих літер на заголовні.'
