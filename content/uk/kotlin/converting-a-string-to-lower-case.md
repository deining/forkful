## Що і чому?

Зниження регістру рядка - це коли рядок перетворюється на букви нижнього регістру. Це корисна функція для програмістів, оскільки допомагає впоратися з різними форматами даних і полегшує порівняння рядків.

## Як це зробити:

```Kotlin
val string = "HeLLo, WoRld!"
println(string.toLowerCase())
```

Output: hello, world!

Цей приклад показує, як просто використовувати функцію `toLowerCase()` для перетворення рядка `string` на літери нижнього регістру.

## Глибока погляд:

Історичний контекст:
Перетворення рядка на літери нижнього регістру було використано ще в перших мовах програмування для полегшення порівняння рядків. В деяких мовах, таких як C++, це було обов'язковою процедурою, оскільки вони не підтримували автоматичне порівняння рядків з великою і малою літерами.

Альтернативи:
Існують інші способи перетворення рядка на літери нижнього регістру, наприклад, використання циклу для заміни кожної букви на її нижній еквівалент або використання методу `toLowerCase(Locale.ROOT)`, який дозволяє вказати конкретну локаль для перетворення.

Деталі реалізації:
Всі літери в Юнікоді мають свої коди, які використовуються для перетворення їх на малі або великі літери. Також важливо враховувати локаль, оскільки в різних мовах можуть бути різні правила для перетворення регістру.
