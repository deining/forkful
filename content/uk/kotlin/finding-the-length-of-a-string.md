## Чому

Знаходження довжини рядка є однією з найбільш основних операцій у багатьох програмах. Іноді це може бути необхідно для обробки текстових даних або для перевірки правильності введених користувачем даних. У цій статті ми розглянемо, як це робити за допомогою мови програмування Kotlin.

## Як це зробити

Існує декілька способів знайти довжину рядка у Kotlin. Перший - використати функцію `length ()` для об'єкта String. Давайте подивимося на приклад коду:

```Kotlin
val str = "Привіт світ!"
println (str.length ()) // виводить 12, оскільки є 12 символів у рядку
```

Наступна можливість - використання властивості `length`. Давайте подивимося на приклад:

```Kotlin
val str = "Привіт світ!"
println (str.length) // виводить 12, так само як і попередній приклад 
```

Останній спосіб, який ми розглянемо, полягає в тому, щоб використовувати функцію `size` для отримання довжини рядка. Подивимося, як це працює:

```Kotlin
val str = "Привіт світ!"
println (str.size) // виводить 12, так само як і два попередніх приклади
```

Як бачите, існує кілька способів знайти довжину рядка у Kotlin. Виберіть той, який найбільш зручний для вашого проєкту.

## Глибина

Розглянемо деякі класи та методи, які використовуються для розрахунку довжини рядка у Kotlin. 

### Клас String

Клас `String` є основним для роботи з рядками у Kotlin. Він надає доступ до багатьох функцій та властивостей, включаючи `length ()`, `length`, та `size`, які ми розглянули раніше.

### Методы `length()` та `size`

Команда `length()` повертає кількість символів у рядку, тоді як метод `size` повертає розмір цього рядка в байтах.
