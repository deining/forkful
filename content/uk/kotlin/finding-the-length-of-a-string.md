## Що і чому?

Знаходження довжини рядка - це процес визначення кількості символів, які містяться у рядку тексту. Це корисна функція для програмістів, оскільки вона дозволяє визначити, наскільки довгий текст потребує обробки, і це може вплинути на швидкодію програми.

## Як:

Нижче наведені приклади коду із зразками виходу, які демонструють, як знаходити довжину рядка у мові програмування Kotlin.

```Kotlin
fun main() {
    val str = "Це приклад рядка"
    println("Довжина рядка: ${str.length}")
}
```

Вивід: `Довжина рядка: 17`

```Kotlin
fun main() {
    val str = "Рядок з українськими символами: привіт!"
    println("Довжина рядка: ${str.length}")
}
```

Вивід: `Довжина рядка: 30`

## Deep Dive:

Ідея знаходження довжини рядка виникла з необхідності вимірювання розміру тексту для його подальшої обробки. Іншою альтернативою є використання функцій `count()` або `size()`, але вони можуть бути менш ефективними у порівнянні із використанням властивості `length`. В Kotlin довжина рядка обчислюється за допомогою методу `length()`, який повертає ціле число, що відповідає кількості символів у рядку.
