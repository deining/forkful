## Для чого
Таке просте поняття, як довжина рядка може бути корисне в багатьох випадках. Наприклад, вам може знадобитись знати кількість символів у вашому паролі або імені, аби переконатись, що воно відповідає вимогам безпеки або не перевищує обмеження на довжину.

## Як це зробити
Для знаходження довжини рядка використовується вбудована функція `length()`. Давайте подивимося на приклад коду, щоб краще зрозуміти, як це працює:

```Kotlin
fun main() {
    val string = "Привіт, світ!"
    
    // Виведемо довжину рядка
    println(string.length()) // Виводить 12
}
```

В даному прикладі ми використовуємо змінну `string`, якій присвоєно значення `"Привіт, світ!"`. І за допомогою функції `length()`, яку ми викликаємо на цій змінній, отримуємо кількість символів у цьому рядку.

Існує також інший спосіб знаходження довжини рядка, за допомогою свойства `length`. Давайте подивимося на приклад:

```Kotlin
fun main() {
    val string = "Привіт, світ!"
    
    // Виведемо довжину рядка
    println(string.length) // Виводить 12
}
```

Обидва ці підходи дають однаковий результат, тому ви можете використовувати той, який комфортніше для вас.

## Поглиблене вивчення
За допомогою функції `length()` ми можемо отримати довжину будь-якого рядка, незалежно від його вмісту. Але варто пам'ятати, що символи многобайтових рядків, таких як кириличні, не обробляються правильно. У цьому випадку, для отримання правильної довжини рекомендується використовувати метод `codePointCount()`, який обробляє символи одного чи декількох байтів.
