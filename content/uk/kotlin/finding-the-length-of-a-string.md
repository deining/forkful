## Чому

Строки є необхідною частиною багатьох програм, тому важливо знати, як обробляти їх. Одним із найпоширеніших дій з рядками є знаходження їх довжини. Для цього і було створено метод length () у Kotlin.

## Як

```Kotlin
fun main() {
    val string = "Привіт, світ!"
    val length = string.length()

    println("Довжина рядка \"$string\" становить $length символів.")
}
```
Вивід: Довжина рядка "Привіт, світ!" становить 12 символів.

Припустимо, ви маєте змінну типу String зі значенням "Коти", і ви хочете дізнатися, скільки букв у цьому слові. Вам потрібно лише викликати метод length () та присвоїти його значення змінній. Вивід буде містити число 4.

Метод length () також можна використовувати на пустих рядках, тобто рядках без символів. У цьому випадку, результат буде 0.

## Глибокий погляд

За покроковий процес отримання довжини рядка в Kotlin відповідає вбудована функція length (). Вона повертає ціле число, яке відображає кількість символів у вказаному рядку. Усі символи, включаючи пробіли та спеціальні символи, зараховуються до довжини.

Також варто знати, що довжина рядка не залежить від кодування, тому результат буде однаковим незалежно від того, який символ символу використовується в рядку.
