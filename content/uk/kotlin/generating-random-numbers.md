## Чому

У галузі програмування ігор та створення комп'ютерних симуляцій, випадкові числа грають важливу роль. Вони дозволяють створювати різноманітні результати та насолоджуватися різноманітністю. Генерація випадкових чисел також корисна для тестування програм та тестування алгоритмів, де необхідно випардково підібрати дані. Давайте розглянемо, як генерувати випадкові числа в Котлін.

## Як

```Kotlin
// Генерується випадкове ціле число в діапазоні від 0 до 10
val randomNum: Int = (0..10).random()
println(randomNum)
```

Таким чином, ми використали функцію `random()` у межах відрізка від 0 до 10 та присвоїли результат змінній `randomNum`, щоб потім його вивести за допомогою функції `println()`.

```Kotlin
// Генерується випадкове дійсне число в діапазоні від 0.0 до 1.0
val randomNum: Double = Math.random()
println(randomNum)
```

Крім того, ми можемо використовувати функцію `Math.random()`, яка повертає дійсне число від 0.0 до 1.0 та присвоїти результат змінній `randomNum`.

 ```Kotlin
 // Генерується випадкове логічне значення
 val randomBoolean: Boolean = (0..1).random() == 1
 println(randomBoolean)
 ```

 Також ми можемо використати випадкове число для визначення логічного значення в діапазоні від 0 до 1.

## Deep Dive

Оскільки випадкові числа є важливою частиною багатьох програм, давайте поглянемо на деякі підводні камені, які можуть впливати на їх генерацію. По-перше, важливо використовувати вбудовані функції, такі як `random()` та `Math.random()`, які забезпечують високу якість псевдовипадкових чисел. Крім того, важливо використовувати правильний діапазон для випадкових чисел, щоб досягти необхідної точності та варіативності.

Для більш детальної інформації про генерацію випадкових чисел в Котлін, можна ознайомитися з [офіційною документацією](https://kotlinlang.org/docs/reference/random-numbers.html).
