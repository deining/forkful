## Чому?

Генерація випадкових чисел - це важлива частина багатьох програм на мові Kotlin. Вона допомагає розв'язувати різні задачі, такі як генерація унікальних ідентифікаторів або створення рандомного вмісту для тестування. Без генерації випадкових чисел, багато програми було б більш простими та менш функціональними.

## Як це зробити?

Ми можемо згенерувати випадкове число за допомогою методу `nextInt()` з класу `Random`.

```Kotlin
val random = Random()
val randomNumber = random.nextInt()

println(randomNumber)
```

Вищевказаний приклад згенерує випадкове ціле число та виведе його на екран. Можна обмежити максимальне значення, використовуючи метод `nextInt(max)`, де `max` - це максимальне значення, яке може бути згенероване.

```Kotlin
val random = Random()
val randomNumber = random.nextInt(100)

println(randomNumber)
```

У цьому випадку буде згенероване ціле число від 0 до 99. Можна також вибирати мінімальне значення, використовуючи метод `nextInt(min, max)`, де `min` - це мінімальне значення, а `max` - максимальне.

```Kotlin
val random = Random()
val randomNumber = random.nextInt(10, 20)

println(randomNumber)
```

Вищевказаний приклад згенерує ціле число від 10 до 19. Якщо потрібно згенерувати випадкове дробове число, можна використовувати метод `nextDouble()`, який поверне десяткове число від 0 до 1.

```Kotlin
val random = Random()
val randomDecimal = random.nextDouble()

println(randomDecimal)
```

Існує також багато інших методів для генерації випадкових чисел, таких як `nextBoolean()`, `nextLong()` та інші. Для детальнішої інформації про них, можна ознайомитися з офіційною документацією Java для класу `Random`.

## Глибші деталі

Клас `Random` використовує лінійний конгруентний генератор псевдовипадкових чисел для створення випадкових чисел. Цей генератор використовує формулу `X = (a * X + c) % m`, де `a`, `c` та `m` - це константи, що визначаються при створенні екземпляру `Random`. Всі екземпляри `Random` мають однакові значення для цих констант.
