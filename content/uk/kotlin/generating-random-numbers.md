## Чому 

Створення випадкових чисел може бути корисним для вас, якщо ви програміст, який працює з великими обсягами даних або робить багато різних тестів на програму.

## Як 

```Kotlin 
val random = Random() // Створення об’єкту Random
val randomNumber = random.nextInt(100) // Генерує випадкове число від 0 до 99
println("Випадкове число: $randomNumber") // Виведення результату
```

У цьому коді ми створюємо об’єкт типу Random, який дозволяє нам генерувати випадкові числа. За допомогою методу `nextInt()` ми можемо вказати максимальне значення, яке хочемо отримати. У вищезазначеному випадку це число буде від 0 до 99. Ми також використовуємо рядки-шаблони для виведення результату зі значенням змінної `randomNumber`.

## Глибокий вигляд 

Генерування випадкових чисел ґрунтується на алгоритмах псевдовипадкових чисел, які використовують унікальні початкові значення для кожного обчислення. Ці алгоритми є невід’ємною частиною багатьох криптографічних протоколів та комп’ютерних ігор. Важливим аспектом є використання одного початкового значення для отримання більшого числа випадкових значень, що забезпечує більшу непередбачуваність.
