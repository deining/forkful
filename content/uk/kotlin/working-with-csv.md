## Чому

Робота з CSV (Comma-Separated Values) є надзвичайно корисною та поширеною задачею в програмуванні. Вона дозволяє зчитувати та зберігати дані у зручному форматі, що зробить роботу з великими обсягами даних більш простою та ефективною.

## Як працювати з CSV у Kotlin

Для початку, нам потрібно імпортувати необхідну бібліотеку `kotlin.io`. Приклад імпорту та оголошення змінної, яка міститиме шлях до CSV файлу, можна побачити нижче:

```Kotlin
import kotlin.io.*
val filePath: String = "path/to/file.csv"
```

### Читання даних з CSV файлу

Для читання даних з CSV файлу, використовуємо функцію `readCSV()` та передаємо в неї шлях до файлу. Результатом буде список списків, де кожен внутрішній список представляє рядок з файлу. Наприклад:

```Kotlin
val fileData = readCSV(filePath) // читаємо дані з CSV файлу
println(fileData) // виводимо результат
```

Припустимо, у нашому CSV файлі є наступні дані:
```
Name, Age, City
John, 25, New York
Emily, 30, London
```
В цьому випадку результатом виконання буде `[[Name, Age, City], [John, 25, New York], [Emily, 30, London]]`.

### Запис даних до CSV файлу

Для запису даних до CSV файлу, використовуємо функцію `writeCSV()` та передаємо в неї список списків з даними та шлях до файлу. Наприклад, ми хочемо записати до нашого CSV файлу новий рядок з даними:

```Kotlin
val newData: List<String> = listOf("Adam", "20", "Paris") // новий рядок з даними
writeCSV(listOf(fileData, newData), filePath) // записуємо результуючий список у файл
println(readCSV(filePath)) // виводимо оновлений список
```

Після виконання коду, у нашому CSV файлі з'явиться новий рядок з даними та результат виведення буде `[[Name, Age, City], [John, 25, New York], [Emily, 30, London], [Adam, 20, Paris]]`.

## Глибока розширена інформація про роботу з CSV

У бібліотеці `kotlin.io` також є можливість налаштувати розділювач, який використовується у CSV файлі, за замовчуванням це кома. Щоб змінити розділювач, можна використовувати функцію `setCSVSeparator()`, передавши в неї бажаний розділювач.
