## Чому

Об'єднання рядків - це потужний і корисний інструмент в програмуванні. З його допомогою можна створити більш складні рядки, маніпулювати даними і полегшити роботу з текстом у програмах.

## Як використовувати

```Kotlin
fun main() {
    //Створення двох рядків
    val firstName = "Анна"
    val lastName = "Петренко"

    //Об'єднання рядків з додаванням проміжного пробілу
    val fullName = "$firstName $lastName"

    //Об'єднання рядків без проміжного пробілу
    val initials = "${firstName.first()}${lastName.first()}"
    
    // Виведення результатів
    println("Повне ім'я: $fullName")
    println("Ініціали: $initials")
}
```
Вивід:
```
Повне ім'я: Анна Петренко
Ініціали: АП
```

## Глибше вдивимося

У Kotlin є декілька способів об'єднання рядків. Один із них - використання оператора додавання "+". Однак, цей спосіб не є найкращим, оскільки застарів і може призвести до виконання зайвих операцій при роботі з багато рядків.

Натомість, у Kotlin є два додаткових способи об'єднання рядків - використання символу "$" та функції "format()".

Символ "$" дозволяє вставляти значення змінних безпосередньо у середину рядка. При цьому, якщо змінна є типом "String", то вона буде вставлена як є, а якщо іншого типу - автоматично буде перетворена у "String".

Функція "format()" приймає аргументи у вигляді позиційних або іменованих змінних і дозволяє створювати більш складні рядки.

Наприклад:
```Kotlin
fun main() {
    //Створення три рядким і функції
    var name = "Софія"
    var surname = "Коваленко"
    fun addInfo() = "вік: 22 роки"

    //Об'єднання рядків з використанням "$"
    println("$name $surname - ${addInfo()}")

    //Об'єднання рядків з використанням функції "format()"
    println("%s %s - %s".format(name, surname, addInfo()))
}
```
Вивід:
```
Софія Коваленко - вік: 22 роки
Софія Коваленко - вік: 22 роки
```
