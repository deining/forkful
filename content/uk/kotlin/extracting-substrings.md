## Чому

<Organization> Kotlin надає нам простий спосіб використовувати методи для витягування підрядків з рядків. Це може бути корисно в багатьох ситуаціях, наприклад, для розбиття дати на різні частини або витягування певної інформації з URL адреси.

## Як

Приклади коду та вихідні дані наведені нижче.

```Kotlin
val date = "2021-05-23" // приклад рядка з датою
val year = date.substring(0, 4) // вилучити перші чотири символи (рік)
val month = date.substring(5, 7) // вилучити п'ятний та шостий символи (місяць)
val day = date.substring(8, 10) // вилучити дев'ятий та десятий символи (день)
println("Рік: $year, Місяць: $month, День: $day") // вивести інформацію про рік, місяць та день
```
Вихідні дані будуть такими:
```
Рік: 2021, Місяць: 05, День: 23
```

```Kotlin
val url = "https://example.com/users/123" // приклад URL адреси
val id = url.substringAfterLast("/") // вилучити частину після останнього знаку "/" (ідентифікатор користувача)
println("Ідентифікатор користувача: $id") // вивести інформацію про ідентифікатор користувача
```
Вихідні дані будуть такими:
```
Ідентифікатор користувача: 123
```

## Глибокий занурення

При витягуванні підрядків з рядків в Kotlin, важливо пам'ятати, що нумерація символів починається з 0. Також, метод `substring()` приймає два параметри: позицію початку та позицію кінця, але не включає останній символ у вилучену підрядок. Це означає, що якщо ми використовуємо позицію кінця, яка відповідає довжині рядка, то останній символ не буде включений у підрядок.
