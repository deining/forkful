## Що і навіщо?

Вилучення підрядків - це процес вилучення шматків тексту з рядків у програмуванні. Програмісти використовують це для отримання частини тексту, який необхідний для виконання певних дій.

## Як це зробити?

```Kotlin 
val str = "Привіт, це Kotlin!"
val substr = str.substring(7,12)
println(substr)
```

Вивід: `це Kotl`


Для вилучення підрядків у Kotlin існує кілька методів, таких як `substring()`, `slice()` і `filter()`, які дозволяють задавати діапазони символів для вилучення.

## Завантаження у подробиці

У цьому процесі вилучення підрядків був розроблений для того, щоб полегшити маніпулювання текстом у програмуванні. Існують інші альтернативи, такі як використання регулярних виразів або використання вбудованих методів для роботи зі строками.

У Kotlin, метод `substring()` приймає початковий і кінцевий індекси для визначення діапазону символів, які необхідно вилучити з рядка. Можливо також використовувати негативні значення для позначення символів від кінця рядка.
