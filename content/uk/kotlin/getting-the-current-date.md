## Чому

Можливо, у вас є необхідність отримати поточну дату в своїй програмі, але ви не знаєте як це зробити? Чи можливо ви просто цікавитесь розвитком мови Kotlin і хочете дізнатися про нові можливості?

## Як це зробити

Програмування різноманітних застосунків часто пов'язане з використанням поточної дати. У мові Kotlin існує декілька способів, якими можна це зробити. Нижче наведені приклади коду та приклад виведення, який показує поточну дату.

```Kotlin
// Можна отримати поточну дату за допомогою класу Date
val currentDate = Date()
```
```
Sat Oct 16 15:31:14 UTC 2021
```

```Kotlin
// Або використовуючи статичний метод now() у класі LocalDateTime, який дозволяє вибрати формат дати.
val currentDate = LocalDateTime.now()
println(DateTimeFormatter.ofPattern("yyyy-MM-dd HH:mm:ss").format(currentDate))
```
```
2021-10-16 15:31:14
```

```Kotlin
// Для більш розширеного управління датою і часом є класы Calendar та GregorianCalendar.
// Наприклад, ми можемо вивести кількість днів у поточному місяці.
val calendar = Calendar.getInstance()
println(calendar.getActualMaximum(Calendar.DAY_OF_MONTH))
```
```
31
```

Як бачите, в Kotlin є декілька можливостей для отримання поточної дати. Виберіть той спосіб, який вам більше підходить для вашої програми.

## Глибше в деталі

Код, який був показаний вище, є лише деякими прикладами роботи з датою в Kotlin. Однак, для більш розширеного розуміння, давайте розглянемо більш детально деякі ключові концепції:

1. В Kotlin, так само як і в багатьох інших мовах програмування, існує поняття "приводимості" (castability). Це означає, що дані можуть бути перетворені з одного типу в інший. Наприклад, у нас є об'єкт Date, який може бути приведений до класу LocalDateTime за допомогою статичного методу from().

2. Для роботи з датами та часом, Kotlin використовує бібліотеку Java.time. Ця бібліотека дозволяє працювати з датами у різних форматах та перетворювати їх з одного в інший.

3. У багатьох випадках ви можете використовувати методи з попереднього прикладу безпосередньо. Наприклад, datetime.now () - це той самий метод, що і LocalDateTime.now().
