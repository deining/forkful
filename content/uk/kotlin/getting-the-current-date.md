## Чому

Одним з найпоширеніших завдань у програмуванні є отримання поточної дати. Це може бути корисно для багатьох різних задач, таких як відстеження часу виконання програми, робота з таймерами або відображення поточної дати на інтерфейсі користувача. У цій статті ми розглянемо, як виконати цю задачу за допомогою Kotlin.

## Як це зробити

```Kotlin
val currentDate = LocalDate.now()
println(currentDate)
```

В першому рядку ми використовуємо стандартну бібліотеку Java для отримання поточної дати. Тип даних `LocalDate` містить інформацію про день, місяць та рік. Потім ми виводимо отриману дату на консоль.

Також можна використовувати додаткові методи для отримання більш детальної інформації про дату, наприклад, визначити часовий пояс або номер дня в тижні.

```Kotlin
val currentDateTime = LocalDateTime.now()
println(currentDateTime.dayOfWeek) // виведе номер дня в тижні
```

## Розбір

Одним із способів отримання поточної дати є використання системного часу комп'ютера. Під час виконання програми, комп'ютер автоматично створює системний час, який потім можна використовувати для отримання поточної дати.

Також важливо знати, що отримана дата може бути залежна від часового поясу комп'ютера. Якщо потрібно отримати дату в певному часовому поясі, необхідно використовувати спеціальні методи для перетворення дати.
