## Що і чому?

Отримання поточної дати - це процес отримання поточної дати та часу у форматі, зрозумілому для комп'ютера. Програмісти часто використовують цей процес для виконання різних завдань, таких як збереження часу створення файлу, синхронізація даних або для показу поточної дати користувачам.

## Як це зробити?

Для отримання поточної дати використовуються функції, доступні у бібліотеці Java, яка підтримується Kotlin. Один з найпростіших способів - використання класу ```java.util.Date``` та його методу ```getTime()```, який повертає час у мілісекундах з 1 січня 1970 року. Наприклад:

```kotlin 
var currentDate = Date()
println("Поточна дата: " + currentDate.getTime())
```

Виконуючи цей код, ви отримаєте поточну дату і час у мілісекундах. Будьте впевнені, що імпортуєте необхідний пакет, використовуючи ключове слово ```import``` перед початком вашого коду:

```kotlin 
import java.util.*
```

## Поглиблене дослідження

Отримання поточної дати та часу є важливою частиною роботи програмістів. Раніше для цього використовувалися різні формати дат, але з винайденням стандарту ISO 8601 дати та часу стали стандартом в більшості систем. Крім отримання поточної дати та часу, програмісти також можуть використовувати функції для роботи з іншими часовими значеннями, наприклад, для отримання дати народження користувача та інших подібних завдань.

Існує також більш простий спосіб отримання поточної дати у Kotlin, який використовує функцію ```now()``` з класу ```kotlin.time.TimePoint```:

```kotlin 
var currentDate = TimePoint.now()
println("Поточна дата: " + currentDate)
```

Це стандартний спосіб отримання поточної дати у Kotlin, але ви також можете використовувати різні бібліотеки та фреймворки для отримання поточної дати іншими способами.
