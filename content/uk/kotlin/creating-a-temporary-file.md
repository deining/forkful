## Чому

Створення тимчасових файлів є важливою частиною багатьох програм, особливо якщо вони працюють зі збереженням даних або потребують зберігання тимчасової інформації. Тимчасові файли допомагають зберегти ресурси комп'ютера та покращують ефективність програми, оскільки вони автоматично видаляються після виконання операцій.

## Як

Для створення тимчасового файлу використовується функція `createTempFile()`. Ось приклад коду, який створює тимчасовий файл і записує в нього текстовий рядок:

```Kotlin
val tempFile = createTempFile()
tempFile.writeText("Це тимчасовий файл, який був створений для демонстрації.")
```

Після виконання цього коду, у вашій програмі з'явиться тимчасовий файл з унікальною назвою та розширенням `.tmp`. Для отримання доступу до змісту тимчасового файлу використовується функція `readText()`:

```Kotlin
val text = tempFile.readText()
```

На цьому прикладі ми прочитали зміст тимчасового файлу та зберегли його у змінну `text`. Після виконання цього коду, змінна `text` буде містити текстовий рядок "Це тимчасовий файл, який був створений для демонстрації.".

## Deep Dive

Тимчасові файли зазвичай зберігаються в системній тимчасовій папці, яка знаходиться на вашому комп'ютері. Якщо ви бажаєте, щоб тимчасовий файл зберігався у конкретному місці, можна передати цю папку як параметр у функції `createTempFile()`. Наприклад:

```Kotlin
val tempFile = createTempFile("my_temp_folder")
```

Це призведе до створення тимчасового файлу у папці з назвою "my_temp_folder".

Також варто пам'ятати, що тимчасовий файл буде автоматично видалено при закритті програми або при самостійному виконанні операцій видалення файлів у цій папці системою.
