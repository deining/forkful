## Тому, що треба

Створення тимчасових файлів є важливою частиною багатьох програм. Вони можуть бути використані для зберігання тимчасових даних або для обробки чутливої інформації, щоб уникнути її втрати чи пошкодження. Також тимчасові файли можуть допомогти у покращенні продуктивності програми шляхом обмеження використання пам'яті.

## Як

Створення тимчасових файлів у Kotlin є достатньо простим завданням. Нижче наведено приклад коду, який демонструє, як створити тимчасовий файл за допомогою вбудованої функції "createTempFile":

```Kotlin
val tempFile = createTempFile("prefix", "suffix")
println(tempFile.name) // виведе назву тимчасового файлу, наприклад: "prefix2564945352954suffix"

tempFile.writeText("Hello World") // запише дані у тимчасовий файл
println(tempFile.readText()) // виведе "Hello World", тільки виконані з іншими запусками скрипта виведіть той самий рядок з проблемами

tempFile.deleteOnExit() // видаляє тимчасовий файл після завершення програми
```

## Глибоке вивчення

У цьому прикладі ми використовуємо функцію "createTempFile" безпосередньо, але Kotlin також має багато інших варіантів створення тимчасових файлів, які можуть бути корисними в різних ситуаціях.

Наприклад, можна вказати папку для зберігання тимчасового файлу або задати ім'я та розширення файлу. Також існує можливість використовувати "File" для створення тимчасових файлів з більш специфічними функціями, такими як "createNewFile" або "createTempDirectory".
