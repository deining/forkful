## Чому

Порівняння двох дат дуже корисне у випадках, коли потрібно визначити, яка з них є більш ранньою чи пізньою, або ж визначити, яка дата настає раніше за іншу. Наприклад, це може бути корисно під час роботи з датами в програмах бронювання або планування подій.

## Як це зробити:

Для того, щоб порівняти дві дати у Kotlin, можна скористатись класом `LocalDate`, який зберігає інформацію про дату без часу та часової зони. Для цього необхідно імпортувати пакет `java.time`, який містить цей клас:

```Kotlin
import java.time.LocalDate
```

Наступним кроком буде створення двох змінних типу `LocalDate`, в які збережемо наші дати для порівняння:

```Kotlin
val firstDate = LocalDate.of(2020, 1, 1)
val secondDate = LocalDate.of(2020, 1, 15)
```

Останнім кроком буде використання методу `compareTo()` для порівняння дат. Цей метод повертає значення 0, якщо дати рівні, -1, якщо перша дата раніше за другу, і 1, якщо перша дата пізніше за другу.

```Kotlin
val result = firstDate.compareTo(secondDate)

if (result == 0) {
    println("Перша та друга дати рівні")
} else if (result == -1) {
    println("Перша дата раніше за другу")
} else {
    println("Перша дата пізніше за другу")
}
```

В нашому прикладі, в консоль виведеться наступне повідомлення: `Перша дата раніше за другу`.

## Глибший занурення:

При порівнянні дат у Kotlin, цікавим може бути також використання методу `isBefore()` та `isAfter()`, який повертають булеве значення, чи є перша дата раніше або пізніше за другу. Крім того, клас `LocalDate` містить багато інших методів, що дозволяють здійснювати операції з датами, такі як додавання або віднімання певної кількості днів, місяців або років.
