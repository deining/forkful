## Чому
Програмування з використанням мови Kotlin стає все більш популярним серед українських розробників. Однією з цікавих можливостей цієї мови є порівняння дат. Ця можливість може бути корисною для перевірки часових меж, розрахунку часового інтервалу або просто для порівняння двох дат і знаходження більш нової або старішої з них. У цій статті ми розглянемо, як порівнювати дві дати в Kotlin і як це може бути корисно для вашого коду.

## Як
Для порівняння двох дат в Kotlin ми будемо використовувати метод `compareTo()` з класу `DateTime`. Цей метод приймає на вхід іншу дату і повертає значення, яке показує, яка з двох дат є більшою. Наприклад, якщо ми хочемо дізнатися, чи є перша дата більшою за другу, ми можемо використовувати наступний код:

```Kotlin
val firstDate = DateTime(2021, 5, 15)
val secondDate = DateTime(2020, 12, 31)

if(firstDate.compareTo(secondDate) > 0) {
    println("Перша дата більша за другу")
} else {
    println("Друга дата більша або рівна першій")
}
```

Цей код виведе повідомлення "Перша дата більша за другу", оскільки 2021 рік є більшим за 2020 рік.

Також ми можемо використовувати метод `isEqual()` для перевірки, чи є дві дати рівними. Наприклад, якщо нам потрібно перевірити, чи мають дві дати однаковий день:

```Kotlin
val firstDate = DateTime(2021, 5, 15)
val secondDate = DateTime(2021, 5, 15)

if(firstDate.isEqual(secondDate)) {
    println("Дати однакові")
} else {
    println("Дати різні")
}
```

Цей код виведе повідомлення "Дати однакові", оскільки обидві дати мають той самий рік, місяць і день.
