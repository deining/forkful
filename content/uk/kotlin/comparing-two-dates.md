## Чому

Порівняння дат у програмуванні є важливою та корисною навичкою, особливо для роботи з даними та створення зручних інтерфейсів для користувачів. Використання правильних методів для порівняння дат допоможе уникнути помилок та забезпечити правильність обробки даних.

## Як це зробити

Для порівняння дат використовується клас `LocalDate`, доступний в Kotlin. Для цього потрібно використати метод `compareTo`, який буде порівнювати дві дати та повертати результат у вигляді числа. Позитивне значення означатиме, що перша дата більша за другу, негативне - навпаки. У прикладі нижче ми порівняємо дати 1 січня 2021 року та 1 березня 2021 року.

```Kotlin
val date1 = LocalDate.of(2021, 1, 1)
val date2 = LocalDate.of(2021, 3, 1)

println(date1.compateTo(date2)) // Виведе -1, оскільки date1 передує date2 в хронологічному порядку
```

## Глибоке дослідження

У класі `LocalDate` доступні й інші методи для порівняння дат, такі як `isBefore`, `isAfter` та `isEqual`. Також можна порівнювати дати за допомогою операторів `>` та `<`. Для більш складних порівнянь існує клас `LocalDateTime`, який дозволяє враховувати час при порівнянні дат.
