## Чому

У цій статті ми дізнаємося, як створювати текстові файли в Kotlin і для чого це потрібно.

## Як це зробити

Створення текстових файлів в Kotlin дуже просте. Спочатку нам потрібно обрати місце, де ми хочемо зберегти наш файл. Після цього ми можемо використати функцію `File` для створення об'єкту файлу. Нижче наведений приклад коду:

```Kotlin
val file = File("file.txt")
```
Цей код створить файл з назвою "file.txt" у поточній директорії, де ми знаходимося. Тепер ми можемо використати функцію `writeText` для запису тексту у наш файл. Наприклад:

```Kotlin
file.writeText("Привіт, світе!")
```
Цей код запише текст "Привіт, світе!" у наш файл. Також ми можемо додавати текст до вже існуючого файлу за допомогою функції `appendText`. Наприклад:

```Kotlin
file.appendText("Це ще один рядок тексту.")
```
Цей код додасть рядок "Це ще один рядок тексту." до нашого файлу.

## Глибоке занурення

Ви можете задавати більші обсяги тексту використовуючи функцію `writeBytes`. Ця функція приймає масив байтів як параметр. Також ви можете вказати кодування для вашого тексту, використовуючи параметр `charset`. Наприклад:

```Kotlin
val text = "Це приклад тексту, який буде записаний у файл."
file.writeBytes(text.toByteArray(Charsets.UTF_8))
```
Цей код сконвертує наш текст у масив байтів і запише його у файл в кодуванні UTF-8.
