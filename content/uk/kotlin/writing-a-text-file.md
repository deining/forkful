# Чому

Написання текстового файлу є важливою складовою частиною програмування в Kotlin. Він дозволяє зберігати та організовувати інформацію для подальшого використання.

## Як це зробити

Нижче подані приклади коду та вихідних даних для створення текстового файлу за допомогою мови програмування Kotlin.

```Kotlin
fun main() {
    val text = "Привіт, світ!" // створюємо рядок, який буде записаний у файл
 
    File("hello.txt").printWriter().use { out ->
        out.println(text) // записуємо рядок в файл
    }
}
```

Результатом цього коду буде створений файл з назвою "hello.txt" та з текстом "Привіт, світ!".

## Глибші деталі

При написанні текстового файлу варто враховувати кілька нюансів. Спочатку необхідно створити екземпляр класу `File`, передавши йому назву та шлях до файлу, який потрібно створити. Далі використовуємо метод `printWriter()` для створення об'єкта для запису даних у файл. Після завершення запису необхідно закрити зв'язок з файлом за допомогою `use` для того, щоб звільнити ресурси.
