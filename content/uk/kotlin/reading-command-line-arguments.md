## Чому

Командний рядок аргументів є важливою частиною знань, які потрібні кожному програмісту. Вивчення того, як правильно читати командний рядок аргументів, допоможе зробити ваші програми більш гнучкими та корисними для користувачів.

## Як

```Kotlin
fun main(args: Array<String>) {
    println("Кількість аргументів у командному рядку: ${args.size}")
    for (arg in args) {
        println("Аргумент: $arg")
    }
}
```
Вихідний код виведе кількість аргументів та їх значення у командному рядку. Наприклад, якщо запустити програму з командним рядком аргументів ```java MainKt hello world```, ви отримаєте наступний результат:

```
Кількість аргументів у командному рядку: 2
Аргумент: hello
Аргумент: world
```

Це дозволяє вам отримувати інформацію з командного рядка для подальшого виконання певних дій у програмі.

## Deep Dive

У командного рядку можна передати дуже багато аргументів, включаючи рядки, числа, булеві значення та навіть власні об'єкти. Якщо абсолютно кожен елемент командного рядку починається з двокрапки ```:```, ви можете отримати повну документацію про передані аргументи:

```Kotlin
fun main(args: Array<String>) {
    for (arg in args) {
        val parts = arg.split(":", limit = 2)
        val type = parts[0]
        val value = parts[1]
        when (type) {
            "string" -> println("Рядок: $value")
            "int" -> println("Число: $value")
            "bool" -> println("Булеве значення: $value")
            else -> println("Невідомий тип аргументу: $arg")
        }
    }
}
```

Для командного рядка аргументів ```java MainKt string:Hello int:10 bool:true```, ви отримаєте наступний результат:

```
Рядок: Hello
Число: 10
Булеве значення: true
```

Це показує, що ви можете передати різні типи даних через командний рядок та їх зручно обробляти у вашій програмі.

## Дивовижне поглиблення

Прочитавши командний рядок, ви можете дійти до висновку, що це дуже важлива частина при написанні програм. Для більш глибокого розуміння, варто дослідити документацію у Інтернеті та почитати більш складні приклади коду. Це допоможе вам використовувати командний рядок аргументів більш ефективно та перетворити ваші програми на справжні інструменти.
