## Чому

Зчитування аргументів командного рядка є важливою частиною процесу програмування на Kotlin. Це дозволяє вам передавати дані в консольну програму під час її запуску і це може бути корисно для розв'язання багатьох задач.

## Як

Для зчитування аргументів командного рядка у Kotlin, вам потрібно використовувати клас `args` у масиві строк. Нижче наведений приклад, який демонструє, як використовувати `args` для виведення аргументів на консоль:

```Kotlin
fun main(args: Array<String>) {
    for (arg in args) {
        println(arg)
    }
}
```

Приклад введення команд і виведення результату на консоль: 

```bash
$ kotlin Main.kt Hello World!
Hello
World!
```

## Deep Dive

Окрім простого зчитування аргументів, Kotlin також дозволяє передавати прапорці аргументів командного рядка. Прапорець - це команда з мінусом перед назвою і потрібна для передачі параметрів у програму. Наприклад, якщо вам потрібно передати файл, названий `example.txt`, у вашу програму, ви можете використати прапорець `-f` як `kotlin Main.kt -f example.txt`.

Крім того, Kotlin також дозволяє задавати значення за замовчуванням для аргументів командного рядка, використовуючи символ рівності (`=`). Наприклад, якщо ви хочете, щоб другий аргумент задавався за замовчуванням як `World`, можна написати `kotlin Main.kt Hello`.
