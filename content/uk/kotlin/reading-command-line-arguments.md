## Чому

Передані через командний рядок аргументи є важливими для будь-якої програми, адже вони дозволяють змінювати поведінку програми та взаємодіяти з користувачем без необхідності перекомпілювати код. Читайте далі, щоб дізнатися, як легко працювати з аргументами командного рядка у Kotlin.

## Як це зробити

Для того, щоб прочитати передані через командний рядок аргументи у Kotlin, необхідно використовувати змінну `args` типу `Array<String>`. Приклад коду і його результату наведено нижче.

```Kotlin
fun main(args: Array<String>) {
    // читання першого аргументу
    val firstArg = args[0]
    println("Перший аргумент: $firstArg")
    // читання другого аргументу
    val secondArg = args[1]
    println("Другий аргумент: $secondArg")
}

```

Вихідна стрічка командного рядка `kotlin example.kt argument1 argument2` дасть такий результат:

```
Перший аргумент: argument1
Другий аргумент: argument2
```

Також, можливо, вам знадобиться перетворити аргументи до цілих чисел або інших типів даних. Для цього можна використати метод `toInt()`, який вбудований у змінну типу `String`. Наприклад:

```Kotlin
fun main(args: Array<String>) {
    // читання першого аргументу та перетворення його до Int
    val firstArg = args[0].toInt()
    println("Перший аргумент у числовому форматі: $firstArg")
}

```

Запускаючи цей код з командного рядка `kotlin example.kt 10`, ви отримаєте такий результат:
```
Перший аргумент у числовому форматі: 10
```

## Поглиблене вивчення

Крім передачі простих аргументів, у командному рядку можна також передавати флаги (`flags`) або прапори (`flags`), які дозволяють додатково настроювати поведінку програми. Для цього необхідно використати ключове слово `--`, яке вказує на початок флагів. Приклад коду і результату показано нижче.

```Kotlin
fun main(args: Array<String>) {
    // перевіряємо наявність флага "-d" та друкуємо результат
    if (args.contains("-d")) {
        println("Флаг -d був переданий")
    } else {
        println("Флаг -d не був переданий")
    }
}

```

Запускаючи код з командного рядка `kotlin example.kt -d`, ви отримаєте такий результат:

```
Флаг -d був переданий
```
