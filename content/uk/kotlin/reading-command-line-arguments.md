## Чому 

Читання аргументів командного рядка є необхідною частиною будь-якої програми. Це дозволяє користувачам передавати додаткову інформацію під час запуску програми, щоб налаштувати її роботу або передати необхідні дані. Це дуже корисно при створенні інструментів командного рядка або скриптів.

## Як це зробити

Щоб прочитати аргументи командного рядка, нам потрібно використати функцію `args` в змінній `args`, яка містить масив рядків з усіма переданими аргументами.

```Kotlin
fun main(args: Array<String>) {
  val input = args[0]
  println(input)
}
```
В цьому прикладі ми передаємо перший аргумент в змінну `input` і виводимо його на екран. Щоб передати кілька аргументів, просто використовуйте відповідний індекс у масиві `args`.

Просто запустіть програму з додатковими аргументами командного рядка для перевірки:

```
$ kotlin MyProgram.kt hello world
hello
```

## Глибше про читання аргументів командного рядка

Приклад вище був дуже простим, але є багато інших можливостей для читання аргументів командного рядка. Наприклад, ми можемо використовувати функцію `startsWith()` для перевірки, чи починається вхідний рядок з певного символу або фрази:

```Kotlin
fun main(args: Array<String>) {
  val input = args[0]
  if (input.startsWith("-")) {
    println("Параметр: ${input.substring(1)}")
  } else {
    println("Невідомий аргумент")
  }
}
```

Тут ми перевіряємо, чи починається аргумент з дефісу і виводимо наступне слово після дефісу як параметр. Також можна використовувати цю техніку для передачі значень параметрів у програму.
