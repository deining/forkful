## Что и зачем?

В программировании получение текущей даты означает извлечение реальной даты "здесь и сейчас". Это делается для логирования событий, добавления временных меток к транзакциям или просто для планирования мероприятий.

## Как это сделать:

```Gleam
import gleam/calendar.{Date, now}
import gleam/io

pub fn main() {
  let today: Date = now
  io.println(today)
}
```

Пример вывода:

```
Date(year: 2023, month: 4, day: 14)
```

## Подробнее

Концепция получения текущей даты так же стара, как и само вычислительное дело. Она уходит корнями в необходимость связывать вычислительные задачи с реальными событиями во времени. В Gleam модуль `calendar` значительно упрощает работу с датами, предлагая типы и функции, такие как `Date` и `now`.

До появления таких модулей разработчики часто напрямую взаимодействовали с операционной системой для получения дат. Это могло быть кропотливо и подвержено ошибкам.

Тип `Date` в Gleam — это простая структура кортежа, которая дает вам составляющие год, месяц и день соответственно. За кулисами `now` обычно вызывает соответствующие системные API для получения даты за вас, абстрагируясь от различий между платформами.

Альтернативы для более сложной обработки даты и времени могут включать использование внешних пакетов, поскольку стандартная библиотека Gleam намеренно содержит минимум возможностей. Эти пакеты могут предоставлять дополнительную функциональность, такую как часовые пояса, форматирование и анализ.
