## Что и Почему?
Извлечение подстрок похоже на выбор вишенки на торте; вы выбираете конкретный кусочек из строки. Программисты делают это для того, чтобы изолировать, анализировать или манипулировать меньшими блоками данных из больших текстовых последовательностей.

## Как:
В Gleam есть несколько способов для извлечения подстрок. Ниже приведены некоторые примеры:

```gleam
import gleam/string

let story = "Learning Gleam is fun!"

// Извлекаем "Gleam" из строки
let gleam = string.slice(story, 9, 14)
assert gleam == Ok("Gleam")

// Вытаскиваем "fun!" используя отрицательный индекс
let fun = string.slice(story, -4, -1)
assert fun == Ok("fun!")

// Если индексы выходят за границы, получаем ошибку
let oops = string.slice(story, 30, 40)
assert oops == Error(Nil)
```
Функция `slice` является основным инструментом для получения подстрок. Предоставленные индексы включительны в начале и исключительны в конце. Отрицательные индексы считаются с конца.

## Глубокое погружение
Извлечение подстрок не является чем-то новым; это старо как мир в программировании. В давние времена, языки программирования вроде C заставляли прыгать через обручи с указателями для получения подстрок. Gleam и современные языки упрощают задачу с помощью встроенных функций, вроде `slice`.

Альтернативы? Конечно. Вы могли бы использовать сопоставление с образцом для разбиения строк на части, но `slice` более изящен для простых извлечений.

С точки зрения реализации, `slice` должен учитывать кодировку строки, например, UTF-8. Это гарантирует, что извлекаются именно символы, а не просто байты, без искажения многобайтных символов. В эпоху, когда был только ASCII, это не было просто.
