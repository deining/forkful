## Что и Почему?
Удаление символов, соответствующих шаблону, заключается в поиске конкретных последовательностей в тексте и их удалении. Программисты делают это для очистки данных, извлечения релевантной информации или санитарной обработки ввода.

## Как:
В Gleam для манипуляции с текстом обычно используется модуль `String`. Встроенной поддержки регулярных выражений нет, но вы можете удалять фиксированные шаблоны или использовать внешние библиотеки для более сложных задач. Давайте очистим текст, удалив восклицательные знаки из строки.

```gleam
import gleam/string

pub fn remove_exclamations(text: String) -> String {
  string.replace(text, "!", "")
}

// Использование
fn main() {
  let cleaned_text = remove_exclamations("Hello, World!!")
  assert cleaned_text == "Hello, World"
}
```
Этот код заменяет все восклицательные знаки на пустую строку, эффективно удаляя их.

## Подробнее
Gleam — это статически типизированный язык для виртуальной машины Erlang, приоритетом которого являются производительность и надежность. Его модуль String предоставляет базовые функции манипуляции, но не имеет продвинутого согласования шаблонов, найденного в библиотеках регулярных выражений.

В историческом контексте, регулярные выражения существуют с 1950-х годов, происходя из теории формальных языков и теории автоматов. Большинство языков программирования приняли некоторую форму реализации регулярных выражений для сопоставления с образцом.

В Gleam для работы с более сложными шаблонами удаления обычно используют библиотеку Erlang или модуль Elixir через интероп, поскольку экосистема Gleam все еще молода. Реализация опирается на надежность BEAM (виртуальная машина Erlang) и его давние библиотеки.

Альтернативы в Gleam могут включать написание собственных функций сопоставления с шаблоном для более предсказуемых шаблонов или обработку конкретных случаев с помощью функций `String`, таких как `slice`, `split` или `trim`.
