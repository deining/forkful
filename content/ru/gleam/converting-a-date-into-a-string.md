## Что и почему?

Преобразование даты в строку означает превращение объекта даты, который представляет конкретный момент времени, в формат текста, удобный для чтения человеком. Программисты делают это, чтобы отображать даты удобным для пользователя способом или для их сериализации с целью хранения и передачи.

## Как сделать:

В Gleam нет встроенного типа даты, но допустим, что мы используем собственный тип `Date` и хотим преобразовать его в строку. Сначала определите свой тип даты и функцию преобразования:

```gleam
type Date {
  Date(year: Int, month: Int, day: Int)
}

fn date_to_string(date: Date) -> String {
  let Date(year, month, day) = date
  int_to_string(year) ++ "-" ++ int_to_string(month) ++ "-" ++ int_to_string(day)
}

pub fn main() {
  let my_date = Date(2023, 4, 3)
  let date_string = date_to_string(my_date)
  io.println(date_string) // "2023-4-3"
}
```

## Подробнее

Исторически форматирование и разбор дат были сложными из-за разнообразия представлений дат и времени в различных локациях и стандартах. Большинство программных сред предлагают библиотеки, которые обрабатывают эти сложности. В Gleam, который стремится к строгой типизации и параллелизму, часто приходится работать с внешними библиотеками, такими как `chronotope`, для операций с датой и временем.

Альтернативой ручному преобразованию строки является использование стандартизированного формата, например ISO 8601 (`YYYY-MM-DD`), который может быть реализован с использованием функций для добавления нулей к однозначным месяцам и дням.

С точки зрения реализации преобразование даты в строку может включать в себя не только конкатенацию целых чисел; локальные предпочтения могут диктовать использование слэшей или точек вместо дефисов, а также встает вопрос о часовых поясах и о том, следует ли включать информацию о времени вместе с датой.
