## Что и почему?

Чтение аргументов командной строки позволяет программам действовать на основе данных, переданных при запуске. Программисты используют это для настройки поведения без изменения кода.

## Как это сделать:

Функция `main` в Gleam может получить доступ к аргументам командной строки через список строк. Итерируйте, используйте сопоставление с образцом и делайте то, что нужно.

```gleam
import gleam/io

fn main(args: List(String)) {
  let message = match args {
    [] -> 
      "Аргументы не найдены."
    [single] -> 
      single
    _ -> 
      "Слишком много аргументов!"
  }
  io.println(message)
}
```

Запуск программы:

```
$ my_program
Аргументы не найдены.

$ my_program "Привет, Gleam!"
Привет, Gleam!

$ my_program Слишком много аргументов передано
Слишком много аргументов!
```

## Подробнее

Чтение аргументов командной строки является основой программирования с самых первых дней. Утилиты UNIX здесь выдаются. Gleam, будучи основанным на Erlang VM, предлагает современный подход к этой функциональности. Среди альтернатив - библиотеки для разбора для сложных случаев, например, для флагов опций. Gleam делает это без многословности Erlang или неясности C.
