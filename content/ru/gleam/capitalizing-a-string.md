## Что и Почему?

Преобразование строки в заглавную форму означает изменение первой буквы на заглавную, в то время как остальные буквы остаются строчными. Программисты делают это для стандартизации ввода данных, в стилистических целях или для соответствия грамматическим правилам в пользовательских интерфейсах.

## Как это сделать:

В Gleam можно определить функцию для преобразования строки в заглавную форму. В настоящее время стандартная библиотека Gleam напрямую не предоставляет функцию преобразования в заглавную форму, поэтому мы создаем свою с помощью срезов строк:

```gleam
import gleam/string

pub fn capitalize(text: String) -> String {
  let head = string.slice(text, 0, 1)
  let tail = string.slice(text, 1, string.len(text))
  
  string.append(string.uppercase(head), string.lowercase(tail))
}

pub fn main() {
  assert capitalize("hello") == "Hello"
  assert capitalize("WORLD") == "World"
}
```

Пример вывода:

```plaintext
"Hello"
"World"
```

## Подробный анализ

Исторически функция преобразования строк в заглавные формы часто включалась в стандартные библиотеки многих языков. Однако Gleam, будучи молодым языком, может не иметь некоторых удобных функций, оставляя их реализацию на усмотрение разработчиков. Альтернативные методы преобразования строк в заглавные формы могут включать использование регулярных выражений или библиотек Unicode для более сложных случаев, которые учитывают специфические для языка правила. В нашем базовом примере детали реализации просты: мы разделяем строку на две части (голову и хвост), делаем первую букву заглавной и соединяем их.
