## Как это сделать:
```Python
import re

# Пример строки
text = "Hello, World! 1234"

# Удаление всех цифр
no_digits = re.sub(r'\d', '', text)
print(no_digits)  # Вывод: "Hello, World! "

# Удаление знаков пунктуации
no_punctuation = re.sub(r'[^\w\s]', '', text)
print(no_punctuation)  # Вывод: "Hello World 1234"

# Удаление гласных
no_vowels = re.sub(r'[aeiouAEIOU]', '', text)
print(no_vowels)  # Вывод: "Hll, Wrld! 1234"
```

### Пользовательская функция, которую я написал

Я делаю это достаточно часто, поэтому я рефакторил этот процесс в функцию `delete()`. Это также хороший пример использования [doctests](https://docs.python.org/3/library/doctest.html):

```python
def delete(string: str, regex: str) -> str:
    """
    >>> delete("Hello, world!", "l")
    'Heo, word!'

    >>> delete("Hello, world!", "[a-z]")
    'H, !'
    """
    return re.sub(regex, "", string)
```

## Глубокое погружение
Практика удаления символов, соответствующих шаблону в тексте, имеет глубокие корни в компьютерных науках, что восходит к ранним инструментам Unix, таким как `sed` и `grep`. В Python модуль `re` предоставляет эту возможность, используя регулярные выражения — мощный и универсальный инструмент для обработки текста.

Альтернативы модулю `re` включают:
- Методы строк, такие как `replace()`, для простых случаев.
- Сторонние библиотеки, такие как `regex`, для более сложных шаблонов и лучшей поддержки Unicode.

Под капотом, когда вы используете `re.sub()`, интерпретатор Python компилирует шаблон в серию байт-кодов, которые обрабатываются автоматом состояний, выполняющим сопоставление шаблонов непосредственно с входным текстом. Эта операция может быть ресурсоемкой для больших строк или сложных шаблонов, поэтому учет производительности критичен для обработки больших объемов данных.
