## কিভাবে:

```Python
import re

# উদাহরণ স্ট্রিং
text = "হ্যালো, ওয়ার্ল্ড! ১২৩৪"

# সব অঙ্ক মুছে দিন
no_digits = re.sub(r'\d', '', text)
print(no_digits)  # আউটপুট: "হ্যালো, ওয়ার্ল্ড! "

# বিরামচিহ্ন মুছে দিন
no_punctuation = re.sub(r'[^\w\s]', '', text)
print(no_punctuation)  # আউটপুট: "হ্যালো ওয়ার্ল্ড ১২৩৪"

# স্বরবর্ণ মুছে দিন
no_vowels = re.sub(r'[aeiouAEIOU]', '', text)
print(no_vowels)  # আউটপুট: "হ্যাল, ওয়ার্ল্ড! ১২৩৪"
```

### আমি যে কাস্টম ফাংশনটি লিখেছি

আমি এটি যথেষ্ট প্রায়ই করি যে, এটি আমি এই `delete()` ফাংশনটিতে রিফ্যাক্টর করেছি। এটি [doctests](https://docs.python.org/3/library/doctest.html) -এর একটি ভাল ডেমনস্ট্রেশনও বটে:

```python
def delete(string: str, regex: str) -> str:
    """
    >>> delete("হ্যালো, ওয়ার্ল্ড!", "l")
    'হ্যাও, ওয়ার্ড!'

    >>> delete("হ্যালো, ওয়ার্ল্ড!", "[a-z]")
    'হ, !'
    """
    return re.sub(regex, "", string)
```



## গভীর ডাইভ
টেক্সটে নির্দিষ্ট প্যাটার্নের সাথে মেলে এমন অক্ষরগুলো মুছে ফেলার অনুশীলনের কম্পিউটার বিজ্ঞানে গভীর শিকড় আছে, যা `sed` এবং `grep` এর মতো প্রাথমিক Unix টুলস পর্যন্ত ট্রেস করা যায়। পাইথনে, `re` মডিউলটি এই ক্ষমতা প্রদান করে, যা নিয়মিত এক্সপ্রেশনকে—টেক্সট প্রসেসিংয়ের জন্য একটি শক্তিশালী এবং বিবিধ টুল—ব্যবহার করে।

`re` মডিউলের বিকল্পগুলি অন্তর্ভুক্ত:
- সহজ কেসগুলির জন্য `replace()` এর মত স্ট্রিং মেথড।
- আরো জটিল প্যাটার্ন এবং ভালো ইউনিকোড সাপোর্টের জন্য `regex` এর মতো থার্ড-পার্টি লাইব্রেরি।

অন্তর্নিহিতভাবে, যখন আপনি `re.sub()` ব্যবহার করেন, তখন পাইথন ইন্টারপ্রিটারটি প্যাটার্নটিকে একাধিক বাইটকোডে কম্পাইল করে, যা ইনপুট টেক্সটে সরাসরি প্যাটার্ন-ম্যাচিং সঞ্চালন করে এমন একটি স্টেট মেশিন দ্বারা প্রসেস করা হয়। বড় স্ট্রিং বা জটিল প্যাটার্নের জন্য এই অপারেশনটি রিসোর্স-ইনটেনসিভ হতে পারে, তাই বড় ডেটা প্রসেসিংয়ের জন্য পারফরমেন্স বিবেচনা অত্যন্ত জরুরি।
