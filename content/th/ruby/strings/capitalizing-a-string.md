## อะไร & ทำไม?
การทำให้สตริงเป็นตัวพิมพ์ใหญ่ในการเขียนโปรแกรมมักหมายถึงการเปลี่ยนตัวอักษรแรกของสตริงให้เป็นตัวพิมพ์ใหญ่และที่เหลือเป็นตัวพิมพ์เล็ก นักพัฒนาทำเช่นนี้ด้วยเหตุผลหลายประการ เช่น การปฏิบัติตามมาตรฐานการตั้งชื่อ, ทำให้ผลลัพธ์ที่แสดงออกมาอ่านง่ายขึ้น, หรือเพื่อความสอดคล้องของข้อมูลสำหรับการเปรียบเทียบและการจัดเก็บ

## วิธีการ:
Ruby มีวิธีการใช้งานที่ตรงไปตรงมาสำหรับการจัดการสตริง รวมถึงการทำให้เป็นตัวพิมพ์ใหญ่ นี่คือวิธีที่คุณสามารถทำให้สตริงเป็นตัวพิมพ์ใหญ่ใน Ruby:

```ruby
# วิธีการภายในของ Ruby
string = "hello world"
capitalized_string = string.capitalize
puts capitalized_string # => "Hello world"
```

เมธอด `.capitalize` ของ Ruby สะดวกแต่มีผลเฉพาะตัวอักษรแรกเท่านั้น หากคุณต้องการควบคุมมากขึ้นหรือต้องการทำให้ทุกคำในสตริงเป็นตัวพิมพ์ใหญ่ (ที่รู้จักว่าเป็นกรณีของชื่อเรื่อง) คุณอาจต้องการใช้เมธอด `titleize` จากส่วนเสริมของ Rails ActiveSupport, หรือจัดการเอง:

```ruby
# การใช้ 'titleize' ของ ActiveSupport ใน Rails
require 'active_support/core_ext/string/inflections'
string = "hello world"
puts string.titleize # => "Hello World"
```

ถ้าคุณไม่ใช้ Rails หรือต้องการวิธีการล้วนๆ ใน Ruby นี่คือวิธีที่คุณอาจทำให้ทุกคำในสตริงเป็นตัวพิมพ์ใหญ่:

```ruby
string = "hello world"
capitalized_each_word = string.split.map(&:capitalize).join(' ')
puts capitalized_each_word # => "Hello World"
```

วิธีนี้แบ่งสตริงออกเป็นอาร์เรย์ของคำ ทำให้แต่ละคำเป็นตัวพิมพ์ใหญ่ แล้วรวมมันกลับเข้าด้วยกันด้วยช่องว่าง.
