## 왜
새로운 프로젝트를 시작하는 것에 대해 궁금해 한다면, 물음을 던지고 싶을 수 있습니다. 제작자 자신이면서 동시에 다른 이들에게 재료를 제공하기 위한 인간 노력에 참여하는 기회 입니다.

## 어떻게
어떠한 프로그래밍 언어를 쓰지 않고서는 새로운 프로젝트를 진행할 수 없습니다. 그래서 대표적인 C 언어를 소개할 수 있습니다. "직관적인 프로그래밍 언어로 새로운 것을 제작하는 것은 매우 쉽습니다" 라고 생각할지 모르겠지만 그것은 조금 낙관적으로 그리고 고민 없이 나온 말 입니다. 누구나 일반적으로 배우는 어떤 언어라도 최종 과정에는 거짓말이 아닙니다.

```
#include <stdio.h>

int main()
{
  printf("안녕하세요, 새로운 프로젝트에 오신 것을 환영합니다!");
  return 0;
}
```

위의 코드는 "Hello, welcome to a new project!" 라는 문자열을 출력합니다.

## 깊이 파고들기
같은 프로그래밍 언어를 사용하더라도 새로운 프로젝트를 시작하는 방법은 모두 다를 수 있습니다. 또한 프로젝트가 성공적으로 끝나지 않는 이유 중 하나는 부자연스러운 작업 프로세스 입니다. 이것을 방지하기 위해 몇 가지 가이드라인을 제공합니다.

1. 목표 설정하기: 새로운 프로젝트를 시작하기 전에 우리가 해결하고 싶은 문제 또는 얻고 싶은 결과물이 무엇인지를 명확하게 설정해야 합니다. 이것은 프로젝트를 진행하는 중요한 동기 부여제가 될 것 입니다.

2. 계획하기: 프로젝트를 시작하기 전에 일정 계획을 세워야 합니다. 어떤 작업을 할 것이며, 언제 할 것인지 등 세부 사항을 정리하여 주변 상황에 따라 일정을 조절할 수 있도록 하세요.

3. 코드 설계하기: 프로젝트를 진행할 때, 좋은 코드 설계는 반드시 필요합니다. 코드의 가독성을 높이고, 유지보수가 용이하도록 최대한 단순하고 효율적인 코드를 작성하세요.
