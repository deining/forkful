# 왜 무작위 숫자를 생성하는가?

무작위 숫자를 생성하는 것은 프로그래밍에서 매우 중요한 부분입니다. 우리는 종종 무작위 숫자를 사용하여 게임의 캐릭터나 아이템을 생성하거나 보안 기능을 구현하고, 다양한 알고리즘에서도 사용합니다. 무작위 숫자를 생성하는 것은 우리의 프로그램을 다양하고 흥미로운 것으로 만드는 데에도 큰 역할을 합니다. 이 블로그 포스트에서 우리는 C 프로그래밍을 통해 무작위 숫자를 생성하는 방법을 알아보겠습니다.

## 어떻게 하나요?

우리는 무작위 숫자를 생성하기 위해 표준 라이브러리인 `<stdlib.h>`의 `rand()` 함수를 사용합니다. `rand()` 함수는 0부터 `RAND_MAX` 값까지의 범위에서 무작위 숫자를 반환합니다. 이 함수를 사용하기 전에 우리는 우선 `srand()` 함수로 시드(seed)를 설정해주어야 합니다. 시드란 무작위 숫자를 생성할 때 사용되는 기준값으로, 이를 설정하지 않으면 `rand()` 함수는 항상 같은 숫자를 반환하게 됩니다. 일반적으로 시간을 시드 값으로 사용해주면 좋습니다.

다음 예시 코드를 살펴보면서 자세히 알아보겠습니다.

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main()
{
    // 시드 설정
    srand(time(NULL));

    // 1부터 100까지의 난수 생성
    int num = rand() % 100 + 1;

    // 결과 출력
    printf("생성된 난수: %d", num);

    return 0;
}
```

위 코드에서는 `srand()` 함수를 사용하여 시드 값으로 현재 시간을 사용하고, `rand()` 함수를 사용하여 1부터 100까지의 난수를 생성하여 그 값을 `num` 변수에 저장합니다. 이후 결과를 출력하면 다음과 같이 될 것입니다.

```
생성된 난수: 73
```

때때로 우리는 0부터 시작하는 난수를 생성하는 경우도 있습니다. 예를 들어 성별 같은 이진 표현을 위해 0과 1 중에서 무작위로 선택해야 하는 경우 등이 있습니다. 이런 경우에는 `rand() % 2`를 사용하면 됩니다.

## 깊게 파헤쳐보기

위 예시 코드에서 우리는 무작위 숫자의 범위를 `rand() % 100`의 형태로 제한했습니다. 이 값은 불규칙적으로 변하는 하한값과 상한값 사이에서 수를 반환합니다. 하지만 이 값은 우리가 사용한 범위를 실제로 생성하는 데에는 문제를 일으킬 수 있습니다. 예를 들어 `rand() % 100`을 사용하면 0부터 99까지의 값만 반환하므로, 우리가 원하는 대로 범위를 생성하지 못할 수 있습니다. 따라서 더 나은 방법은 `(int)((float) rand() / RAND_MAX * (max - min + 1)) + min`과 같은 방식으로 범위를 제한하는 것입니다. 이 방식에서는 `max`와 `min`이 사용자가 원하는 범위로 지정되어 `max`부터 `min` 값 사이에서 무작위 숫자를 반환합니다.
