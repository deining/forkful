## 왜

모든 프로그래밍 언어에서 문자열을 대문자로 바꾸는 것은 매우 일반적입니다. 이것은 대소문자 구분이 필요한 작업을 수행할 때 유용합니다. 예를 들어 사용자가 비밀번호를 입력할 때 대소문자 구분을 포함시켜 더 강력한 보안을 만듭니다. 또한 대문자와 소문자가 혼합된 문자열을 처리하거나 출력할 때 문장의 일관성을 유지할 수 있습니다.

## 방법

문자열을 대문자로 바꾸는 가장 간단한 방법은 각 문자를 하나씩 검사하여 대문자로 변환하는 것입니다. 이를 위해 C 언어에서 제공하는 내장 함수인 `toupper()` 함수를 사용할 수 있습니다. 이 함수는 아스키 코드를 기준으로 문자를 변환하므로 문자열을 다룰 때 유용합니다.

```C
#include <stdio.h>
#include <ctype.h>

int main() {
  char str[100];
  printf("문자열을 입력하세요: ");
  scanf("%s", str);
  
  // 문자열의 각 문자를 대문자로 변환
  for (int i = 0; str[i]!='\0'; i++) {
    str[i] = toupper(str[i]);
  }
  
  printf("변환된 문자열: %s\n", str);
  return 0;
}
```

위의 예제 코드를 입력하고 실행하면 사용자로부터 입력받은 문자열이 대문자로 변환되어 출력됩니다. 예를 들어 "Hello"라는 문자열을 입력하면 "HELLO"라는 결과가 출력됩니다.

## 깊은 이해

문자열을 대문자로 바꾸는 방법은 프로그래밍 언어마다 다소 차이가 있을 수 있지만 기본적인 과정은 비슷합니다. 우선 문자열은 배열 형태로 저장되기 때문에 각 문자에 접근하기 위해 반복문을 사용해야 합니다. 또한 문자열 안에는 문자뿐만 아니라 공백이나 숫자 등 다양한 문자가 포함될 수 있으므로 대문자로 변환되어야 할 문자만 선택적으로 변환해야 합니다.

실제로 `toupper()` 함수를 구현하는 방법은 각 언어마다 다르지만 대부분 아스키 코드를 기준으로 해당 문자가 소문자인지 대문자인지 판단하는 방식으로 동작합니다. 따라서 해당 언어의 아스키 코드 표를 참고하여 구현해야 합니다.
