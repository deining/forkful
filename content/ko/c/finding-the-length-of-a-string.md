## 왜
문자열의 길이를 찾는 작업에 왜 관심을 가져야 할까요? 이 작업은 주어진 문자열의 크기와 범위를 알기 위해 중요한 단계입니다. 예를 들어, 사용자에게 입력받은 문자열의 길이가 제한되어 있을 경우, 이 작업을 통해 입력값의 유효성을 검증할 수 있습니다.

## 방법
아래 코드 블록으로 간단한 예제와 그에 대한 출력 결과를 보여드리겠습니다. 코드 내부에는 주석으로 해당되는 코드의 설명을 적어놓았습니다.

```C
#include <stdio.h>
#include <string.h>

int main()
{
    // 문자열 변수를 선언하고 값을 할당해줍니다.
    char message[20] = "안녕하세요";

    // strlen() 함수를 이용해 문자열의 길이를 구합니다.
    int length = strlen(message);

    // 결과를 출력합니다.
    printf("입력받은 문자열의 길이는 %d입니다.", length);

    return 0;
}

// 출력 결과: 입력받은 문자열의 길이는 5입니다.
```

## 깊게 들어가기
위 예제에서 사용된 `strlen()` 함수는 문자열의 길이를 구하는 데에 가장 간단한 방법입니다. 이 함수는 `string.h` 라이브러리에 선언되어 있으며, 문자열의 끝을 나타내는 널 문자 (`'\0'`)를 만나기 전까지의 문자의 수를 셉니다. 이 외에도 다른 방법으로는 `for` 루프를 이용해 문자열을 순회하며 카운트하는 방식이 있습니다.
