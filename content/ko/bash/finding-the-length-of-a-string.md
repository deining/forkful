## 왜
문자열의 길이를 찾는 것이 중요한 이유는 많은 프로그래밍 언어에서 문자열을 다루는 경우가 많기 때문입니다. 즉, 문자열의 길이를 알아야만 다음에 어떤 작업을 할지를 판단할 수 있기 때문에 문자열의 길이를 찾는 것은 매우 유용합니다.

## 방법
문자열의 길이를 찾는 방법에는 여러 가지가 있지만 우선적으로 Bash 스크립트를 이용하는 방법을 알아보겠습니다. 아래의 코드에서는 `strlen`을 사용하여 문자열의 길이를 찾는 방법을 보여줍니다.

```Bash
# 문자열의 길이를 찾는 스크립트
str="안녕하세요, 이것은 문자열입니다."
echo "문자열의 길이: ${#str}"
```

위의 스크립트를 실행하면 다음과 같은 결과가 나타납니다.

```Bash
문자열의 길이: 20
```

또 다른 방법으로는 `wc -c` 를 사용하는 것입니다. 이 명령어는 입력받은 문자열의 바이트 수를 출력해줍니다. 하지만 이 경우, 입력받은 문자열에 공백이 들어있는 경우 공백도 포함하여 길이를 계산하기 때문에 주의해야 합니다.

```Bash
# 문자열의 길이를 찾는 스크립트
str="안녕하세요, 이것은 문자열입니다."
echo "문자열의 길이: $(echo -n "$str" | wc -c)"
```

위의 스크립트를 실행하면 다음과 같은 결과가 나타납니다.

```Bash
문자열의 길이: 18
```

## 심화 공부
Bash 스크립트를 사용하여 문자열의 길이를 찾는 것 외에도 다른 방법들이 있습니다. 그 중 하나는 `expr length` 를 사용하는 것입니다. 이 명령어는 입력받은 문자열의 길이를 출력해줍니다.

```Bash
# 문자열의 길이를 찾는 스크립트
str="안녕하세요, 이것은 문자열입니다."
echo "문자열의 길이: $(expr length "$str")"
```

위의 스크립트를 실행하면 다음과 같은 결과가 나타납니다.

```Bash
문자열의 길이: 20
```
