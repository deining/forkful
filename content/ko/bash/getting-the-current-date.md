# 왜 현재 날짜를 구하는 것이 중요할까요? 

많은 프로그래밍 언어에서는 현재 날짜 및 시간을 구하는 내장 함수를 제공합니다. 이는 파일 이름, 로그 파일 및 시스템 백업 등 다양한 목적으로 사용될 수 있기 때문에 매우 유용합니다. 또한 날짜를 사용하여 특정 이벤트의 발생 시간을 추적하거나 작업 일정을 설정하는 등 다양한 용도로 활용할 수 있습니다.

## 어떻게 현재 날짜를 구할 수 있을까요?

Bash에서 현재 날짜를 구하는 가장 간단한 방법은 `date` 명령어를 사용하는 것입니다. 예를 들어, 다음 코드를 실행하면 오늘 날짜와 시간이 출력됩니다. 

```Bash
date 
```

출력 예시: `Tue Jul 27 14:05:14 KST 2021`

`date` 명령어는 기본적으로 시스템 시간대에 따라 날짜와 시간을 출력합니다. 따라서 해당 시간대에 맞게 날짜와 시간을 변경하고 싶다면 다음과 같이 `-u` 옵션을 추가하여 실행하면 됩니다.

```Bash
date -u
```

출력 예시: `Tue Jul 27 05:05:14 UTC 2021`

원하는 출력 형식으로 날짜와 시간을 출력하려면 다음과 같이 `+` 옵션을 사용하고 싶은 형식을 지정하면 됩니다.

```Bash
date "+%Y-%m-%d %H:%M:%S" 
```

출력 예시: `2021-07-27 14:05:14`

또한 날짜와 시간을 변수에 저장할 수도 있습니다.

```Bash
today=$(date "+%Y-%m-%d")
echo "$today"
```

출력 예시: `2021-07-27`

## 깊게 파고들어

`date` 명령어는 매우 유연하며 다양한 옵션을 제공합니다. 몇 가지 유용한 옵션에 대해 더 알아보겠습니다.

- `%A`: 현재 요일을 출력합니다.
- `%B`: 현재 월을 출력합니다.
- `%j`: 현재 연 중 몇 번째 날인지 출력합니다.

예를 들어, 다음과 같은 코드를 실행하면 오늘의 요일과 월, 그리고 이번 연도의 몇 번째 날인지를 출력합니다.

```Bash
echo "오늘은 $(date "+%B의 %j일, %A 입니다")"
```

출력 예시: `오늘은 7월의 208일, 수요일 입니다.`

또한 `date` 명령어는 특정 날짜 또는 시간을 계산하여 출력할 수도 있습니다. 예를 들어, 다음과 같은 코드를 실행하면 2일 전의 날짜를 출력합니다.

```Bash
echo "2일 전의 날짜는 $(date -d "2 days ago" "+%Y-%m-%d") 입니다."
```

출력 예시: `2일 전의 날짜는 2021-07-25 입니다.`

더 많은 유용한 옵션을 알아보려면 `man date` 명령어를 사용하여 매뉴얼을 확인해보세요.
