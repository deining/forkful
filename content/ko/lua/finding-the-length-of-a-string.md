## 무엇이고 왜?
문자열의 길이를 찾는 것은 프로그래머들이 자주 하는 작업입니다. 문자열의 길이란 해당 문자열이 가진 문자의 개수를 의미합니다. 프로그래밍에서 문자열은 매우 중요한 자료 형식이며, 문자열의 길이를 알아야만 다양한 작업을 수행할 수 있습니다.

## 하는 방법:
이제 문자열의 길이를 찾는 방법을 알아보겠습니다. 간단한 코드 예제와 함께 살펴보겠습니다.

```Lua
-- 문자열의 길이를 찾는 함수
function findLength(str)
  return string.len(str)
end

-- 코드 예제
local testString = "안녕하세요"
local length = findLength(testString)

--결과 출력
print("문자열의 길이는 " .. length .. " 입니다.")
```

위 코드에서는 문자열의 길이를 찾는 함수를 정의하고, 예제로 "안녕하세요"라는 문자열을 가지고 테스트를 해보았습니다. 결과로는 "문자열의 길이는 5입니다."라는 메시지가 출력되었습니다. 따라서 우리는 이 함수가 잘 작동한다는 것을 알 수 있습니다.

## 깊이 파고들기:
이제 문자열의 길이를 찾는 과정에서 좀 더 깊이 있는 정보들을 알아보겠습니다.

1. 역사적 맥락: 문자열의 길이를 찾는 함수는 대부분의 프로그래밍 언어에서 공통적으로 사용되는 기능입니다. 이러한 함수는 프로그래머들의 편의성을 위해 자주 사용되고 있으며, 매우 중요한 기능이라 할 수 있습니다.

2. 대안: 문자열의 길이를 찾는 다른 방법으로는 루프를 이용하는 방법이 있습니다. 하지만 이 방법은 보다 복잡하고 속도가 느릴 수 있으므로 함수를 사용하는 것이 더 효율적입니다.

3. 구현 세부 사항: 위에서 살펴본 코드에는 string.len()이라는 함수를 사용하여 문자열의 길이를 찾았습니다. 이 함수는 루아의 내장 함수로써, 루아 엔진 내부에서 최적화된 방식으로 길이를 찾게 됩니다.
