# 왜

문자열의 길이를 찾는 것에 참여할 이유는 있을까요? 문자열의 길이를 알아봄으로써 우리는 문자열 처리를 더 간결하고 효율적으로 할 수 있게 됩니다.

## 어떻게

```Gleam
fn main() {
  let str = "안녕하세요!";
  let len = str.len();
  io.println("문자열의 길이는 " ++ to_string(len) ++ "입니다.");
}
```

위의 코드를 실행하면, `"문자열의 길이는 6입니다."`라는 결과를 얻을 수 있습니다. 위 예제처럼 `len()` 함수를 활용해 우리는 문자열의 길이를 구할 수 있습니다.

## 깊이 들어가기

문자열의 길이를 알아내기 위해서는 어떤 메모리 구조로 저장되는지 알아야 합니다. 보통 문자열은 배열로 저장되며, `len()` 함수는 해당 배열의 길이를 반환해줍니다. 만약, UTF-8 인코딩을 통해 다중 바이트를 한 문자로 표현하는 경우, 이 함수는 실제 문자 수를 반환하는 것이 아니라 바이트 수를 반환합니다.
