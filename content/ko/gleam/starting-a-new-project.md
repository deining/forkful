# 왜?

새로운 프로젝트를 시작하는 이유는 무엇일까요? 당신이 원하는 것을 만들기 위해 직접 제어하고 싶을 뿐만 아니라, 새로운 기술에 대해 배우기 위해서일 수도 있습니다.

## 어떻게?

```Gleam
pub fn main() {
  let message: String = "안녕하세요!";

  println(message);
}
```

```
안녕하세요!
```

위의 코드 예제에서 우리는 Gleam으로 "안녕하세요!"라는 메시지를 출력하는 간단한 프로그램을 만들었습니다. 이처럼 Gleam은 간결하고 명확한 문법을 가진 함수형 언어입니다.

## 딥 다이브

새로운 프로젝트를 시작하는 것은 기술적으로도 도전적인 일일 수 있습니다. 따라서 Gleam을 배우는 것은 새로운 도전으로 이어질 수 있습니다. Gleam은 Elixir와의 호환성도 높으며, Erlang VM 위에서 실행되기 때문에 안정성과 확장성도 뛰어납니다. 또한 Gleam 커뮤니티에서 많은 도움을 받을 수 있으며, 공식 문서와 각종 예제도 제공됩니다.
