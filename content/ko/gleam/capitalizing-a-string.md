# 왜
이 포스트에서 우리는 왜 문자열을 대문자로 변환하는 것이 필요한지에 대해 이야기할 것입니다.

## 어떻게
아래의 코드 블록에서 우리는 Gleam 언어로 문자열을 대문자로 변환하는 방법을 보여줍니다. 이를 실행하면 어떤 결과가 나오는지도 확인할 수 있습니다.

```Gleam
string.capitalize("gleam")
```

출력 결과:
```
"GLEAM"
```

## 깊게 파고들기
문자열을 대문자로 변환하는 과정에서 Gleam 언어에서 어떤 메소드를 사용하는지, 그리고 어떻게 이를 구현하는지 자세히 살펴봅시다.

예를 들어, 위의 예제에서 우리는 `string.capitalize()` 메소드를 사용했습니다. 이 메소드는 문자열을 인수로 받아 그 문자열을 대문자로 변환해주는 함수입니다.

보다 자세한 내용은 [Gleam 문서](https://gleam.run/)를 참고하시기 바랍니다.
