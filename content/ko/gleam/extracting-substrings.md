지금 나오는 새로운 Gleam 프로그래밍 언어는 많은 프로그래머들에게 인기를 끌고 있습니다. 이 언어는 간결하고 명확하며 기본적인 상속 메커니즘을 가지고 있어서 작성하기 쉬운 코드를 만들 수 있게 해줍니다. 하지만 가끔씩은 문자열에서 원하는 부분만 추출할 때가 있습니다. 그래서 오늘은 우리가 자주하는 일 중 하나인 부분 문자열 추출에 대해 알아보려고 합니다.

## 무엇이고 왜 하는지?

부분 문자열 추출이란 무엇일까요? 이것은 문자열에서 우리가 원하는 일부만 사용하는 것을 말합니다. 가령, "Hello World!"라는 문자열에서 "Hello"나 "World"만 따로 사용할 수 있도록 한다는 것이죠. 그렇게 하기 위해서는 문자열을 다루는 다양한 방법을 알아야 합니다. 그래서 프로그래머들은 부분 문자열 추출을 자주 사용합니다.

## 하는 방법

예제를 통해 자세히 알아보겠습니다. 우리가 "Hello World!" 라는 문자열을 선언하고 싶다면, 다음과 같이 하시면 됩니다.

```Gleam
let message = "Hello World!"
```
이제 이 문자열에서 "Hello"만 추출해보겠습니다. 아래 코드를 실행하면 "Hello"만 나오게 됩니다.

```Gleam
let hello = String.sub(message, 0,5)
```
출력은 다음과 같습니다.

```Gleam
"Hello"
```

## 깊은 더 들어가기

이제 부분 문자열 추출에 대해 더 깊게 알아보겠습니다. 우선, 이 기능은 단순한 문자열을 처리하는데에 사용할 수 있지만, 더 복잡한 처리를 위해서는 더 많은 함수들을 사용해야 할 수도 있습니다. 또한, 다른 프로그래밍 언어에서도 비슷한 기능을 제공하는데, 그중에서도 Rust 언어에서 제공하는 원시 배열(primitive array)과의 조합으로 매우 빠르게 문자열을 다룰 수 있습니다.

## 관련 정보

Gleam에서의 부분 문자열 추출에 대한 자세한 내용은 공식 문서를 참고하실 수 있습니다. 또한, 다른 프로그래밍 언어에서도 비슷한 기능을 제공하는지 확인하시기 위해 Rust 언어의 관련 사이트를 참고하시기 바랍니다.
