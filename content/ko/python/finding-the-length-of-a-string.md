## 무엇 & 왜?
문자열 길이를 찾는 것이 무엇인지, 그리고 프로그래머가 왜 이를 하는지에 대해 두 성문으로 설명합니다.

문자열의 길이는 해당 문자열이 얼마나 많은 문자로 이루어져 있는지를 나타냅니다. 프로그래머는 코드를 작성하는 과정에서 자주 문자열의 길이를 알아야 하는데, 예를 들어 문자열을 잘라서 다른 역할을 수행하거나, 지정된 길이를 가진 문자열을 생성하는 작업에서 자주 사용합니다.

## 하는 방법:
다음과 같이 ```Python ... ``` 코드 블록 내에 코딩 예제와 샘플 출력을 제공합니다.

```
# 예제 코드
sample_string = "Hello, world!"
# 문자열의 길이를 찾는다.
print(len(sample_string))
# 출력 결과: 13
```

## 깊게 보기:
(1) 역사적인 맥락, (2) 대체 방법, (3) 문자열의 길이를 찾는 과정에 대한 자세한 정보를 제공합니다.

(1) 문자열 길이를 찾는 방법은 컴퓨터 언어의 발전과 함께 발전해 왔습니다. 초기의 컴퓨터 언어는 문자열을 인식하지 못하고, 숫자로만 처리할 수 있었기 때문에 문자열의 길이를 구하기 위해 복잡한 계산이 필요했습니다. 하지만 현재의 컴퓨터 언어는 문자열을 표현하고 처리할 수 있기 때문에 길이를 찾는 과정은 더욱 간편해졌습니다.

(2) 다른 언어에서는 문자열 길이를 찾는 방법이 조금씩 다릅니다. 예를 들어 자바에서는 ```.length()``` 메소드를 사용하고, C언어에서는 널 문자가 포함된 문자열을 다루기 때문에 문자열의 끝까지 반복해서 길이를 구해야 합니다.

(3) 파이썬에서 ```len()``` 함수는 내장 함수로, 함수를 호출하는 방법을 제외하면 따로 별다른 구현 과정이 없습니다. 하지만 코드의 내부에서는 문자열을 반복하면서 길이를 구하므로 함수 호출보다는 직접 반복하는 것이 더 효율적입니다.
