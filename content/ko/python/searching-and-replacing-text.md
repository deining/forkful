# 왜
텍스트를 검색하고 바꾸는 것에 참여하는 이유는 무엇일까요? 많은 경우, 우리는 특정 텍스트를 대량으로 바꿔야할 때 사용합니다. 예를 들어, 우리가 자주 쓰는 단어를 다른 단어로 대체하거나, 오타를 고치거나, 언어를 번역할 때 등등 말입니다. 이러한 작업을 수작업으로 하면 매우 번거롭고 시간도 많이 걸립니다. 그렇기 때문에 검색과 바꾸기를 할 수 있는 프로그래밍 언어가 필요한 것입니다.

## 어떻게
파이썬에서 텍스트를 검색하고 바꾸는 방법은 간단합니다. 먼저, 우리는 're' 모듈을 import해야 합니다. 그런 다음, 're.sub()' 함수를 사용하여 바꿀 텍스트와 바꿀 내용을 입력하면 됩니다. 예를 들어, 우리가 '안녕하세요' 라는 문장에서 '안녕'을 'Hello' 로 바꾸고 싶다면, 다음과 같이 작성할 수 있습니다.

```Python
import re

text = '안녕하세요'
new_text = re.sub('안녕', 'Hello', text)
print(new_text)
```

출력 결과는 다음과 같습니다.

```Python
Hello하세요
```

위 예제에서 보듯이, 're.sub()' 함수는 첫 번째 인자로 검색할 텍스트, 두 번째 인자로 바꿀 내용, 세 번째 인자로 검색할 대상 텍스트를 받습니다. 이렇게 간단하게 텍스트를 바꿀 수 있다는 것이 파이썬의 강력한 기능 중 하나입니다.

## 깊이 들어가기
텍스트를 검색하고 바꾸는 방법을 좀 더 자세히 알아보겠습니다. 're.sub()' 함수는 정규 표현식을 사용하여 텍스트를 검색하게 됩니다. 정규 표현식은 문자열에서 특정 패턴을 검색하는 방식으로, 파이썬 외에도 다양한 프로그래밍 언어에서 사용됩니다. 정규 표현식은 복잡하고 다양한 표현 방식을 가지고 있기 때문에 처음에는 어려울 수 있지만 익숙해지면 매우 유용하게 쓰일 수 있습니다.

예를 들어, 우리가 파일 이름에서 확장자를 제거하고 싶다면 정규 표현식을 사용하여 간단하게 처리할 수 있습니다. 다음 예제를 보겠습니다.

```Python
import re

file_name = 'picture.jpg'
new_name = re.sub('\.jpg', '', file_name)
print(new_name)
```

출력 결과는 다음과 같습니다.

```Python
picture
```

위 예제에서 보듯이, 정규 표현식을 사용하면 텍스트를 더욱 다양하게 검색하고 바꿀 수 있습니다. 여러분도 정규 표현식을 공부하여 다양한 방식으로 텍스트를 바꿀 수 있도록 노력해보세요.
