## 왜

정규 표현식을 사용하는 이유는 무엇일까요? 이것은 프로그래밍에서 문자열을 검색하고 조작할 때 매우 유용합니다. 즉, 특정 패턴에 일치하는 문자열을 찾아내고 해당 패턴을 대체하거나 추출할 수 있습니다. 이는 매우 강력한 기능으로, 특히 문자열 다루기가 많은 프로젝트에서 유용하게 사용할 수 있습니다.

## 어떻게

이제 실제 코딩을 통해 정규 표현식을 사용해보겠습니다. Fish Shell에서 정규 표현식을 사용하는 방법은 매우 간단합니다. ```string match``` 또는 ```string replace```와 같은 Fish Shell의 기본 함수를 사용하고 해당 매개변수에 정규 표현식을 전달하면 됩니다. 예를 들어, 다음과 같이 사용할 수 있습니다.

```Fish Shell
string match "e([a-z]+)s" "expressions are awesome"
```

위의 코드는 정규 표현식 ```e([a-z]+)s```와 문자열 ```expressions are awesome```를 비교하여 일치하는 부분을 찾아냅니다. 이 경우에는 ```expressions```가 일치하는 부분이 됩니다.

## 깊숙한 곳에서

더 깊이 들어가보면, 정규 표현식은 더 강력한 패턴 매칭 기능을 제공합니다. 예를 들어, 특정 패턴을 가진 파일을 한 번에 찾아내는 것은 매우 어려운 일입니다. 그러나 정규 표현식을 사용하면 이를 매우 쉽게 처리할 수 있습니다. 이 외에도 정규 표현식은 문자열에서 특정 패턴을 추출하고 분리하는 등 다양한 용도로 활용할 수 있습니다.
