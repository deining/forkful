## 왜
날짜를 비교하는 것의 중요성에 대해 생각해보면, 다양한 분야에서 날짜 차이를 계산하는 경우가 많습니다. 예를 들어, 이전 날짜와 현재 날짜를 비교하여 얼마나 지났는지 알아보거나, 특정 이벤트가 몇 일 후에 발생하는지 계산하는 경우가 있을 수 있습니다. 또는 날짜 데이터를 정렬하거나 필터링하기 위해 날짜를 비교해야 할 때도 있습니다. 이러한 상황에서 날짜를 비교하는 방법을 알고 있다면 더욱 효율적으로 작업할 수 있습니다.

## 방법
날짜를 비교하는 방법은 간단합니다. Fish Shell의 `date` 명령어를 사용하면 됩니다. 먼저 비교할 날짜를 yyyy-mm-dd 형식으로 지정하고, 두 번째 인자로 비교할 날짜를 지정하면 됩니다. 
```
Fish Shell date 명령어 예시:
date -f '%Y-%m-%d' 2021-01-01 2020-12-01
```
이렇게 하면 첫 번째 날짜가 두 번째 날짜보다 이후인지 여부를 반환합니다. 만약 날짜가 같다면 0, 이후라면 양수, 이전이라면 음수를 반환합니다. 
```
Fish Shell date 명령어 출력 예시: 
34
```
따라서 날짜를 비교할 때는 이러한 값을 기준으로 조건문을 작성하면 됩니다.

## 깊은 곳
날짜를 비교하는 방법은 위에서 소개한 방법 이외에도 더욱 다양합니다. 예를 들어, 날짜의 정확한 시간까지 포함하여 비교하고 싶다면 Fish Shell의 `set` 명령어와 `Hour`, `Minute`, `Second` 등의 변수를 사용하여 시간까지 포함시킬 수 있습니다. 또는 날짜를 문자열로 저장하고 이를 숫자로 변환하여 비교하는 방법도 있습니다.
