# 왜
문자열의 길이를 찾는 것에 참여하는 이유는 프로그래밍에서 자주 사용되는 작업이며, 코드를 더 효율적으로 작성하는 데도 도움이 됩니다.

# 어떻게
```fish shell```에서 문자열의 길이를 찾는 방법은 매우 간단합니다. 다음 코드를 실행하면 됩니다.
```fish shell
set my_string "안녕하세요"
echo (count $my_string)
```
출력은 다음과 같이 표시됩니다.
```
5
```
위의 코드에서 ```count``` 함수는 문자열의 길이를 반환하므로, 변수에 할당된 문자열의 길이를 출력할 수 있습니다.

# 깊게 파보기
문자열의 길이를 찾는 데에는 여러 가지 방법이 있습니다. 예를 들어, ```string``` 모듈을 사용하여 다른 언어에서와 마찬가지로 ```length``` 함수를 사용할 수도 있습니다. 또는 ```split``` 함수를 사용하여 문자열을 나눈 후, 각 조각의 길이를 합산하는 방식도 있습니다. 이러한 깊은 정보를 파보면서 더 많은 방법을 배우고 실제 사용 상황에 맞게 선택할 수 있습니다.
