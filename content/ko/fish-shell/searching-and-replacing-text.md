## Why
왜 텍스트를 찾고 바꾸는 작업에 참여해야 할까요?

텍스트를 찾고 바꾸는 작업은 쉘 프로그래밍에서 매우 중요한 부분입니다. 텍스트를 검색하고 수정하면 작업을 더 효율적으로 처리할 수 있습니다. 또한 프로그래밍 작업에서 반복되는 코드를 일괄적으로 수정할 수 있어서 시간을 절약할 수 있습니다.

## How To
코드를 작성하기 전에 Fish Shell 환경을 설정해야 합니다. 해당 명령을 실행하여 Fish Shell 환경을 설정할 수 있습니다.
```
$ fish_config
```

텍스트를 찾고 바꾸기 위해서는 `sed`라는 유틸리티를 사용할 수 있습니다. 아래의 예제를 통해 더 자세히 알아보겠습니다.
```
$ sed -i 's/오렌지/사과/g' 과일.txt
```

위의 예제는 "과일.txt" 파일에서 "오렌지"를 "사과"로 바꾸는 작업을 수행합니다. `sed` 유틸리티는 파일에 대해 원래 값을 변경하여 작업하기 때문에 `-i` 옵션을 사용합니다.

## Deep Dive
`sed` 유틸리티는 탭, 캐리지 리턴, 개행 등과 같은 제어 문자를 사용하는 파일도 검색하고 수정할 수 있습니다. 또한 정규식을 사용하여 좀 더 복잡한 검색 및 수정 작업을 수행할 수도 있습니다.

예를 들어, 아래의 예제는 "과일.txt" 파일에서 "오렌지"를 "사과"로, "바나나"를 "딸기"로 동시에 바꾸는 작업을 수행합니다.
```
$ sed -i 's/오렌지/사과/g; s/바나나/딸기/g' 과일.txt
```

`sed` 유틸리티의 더 많은 사용법에 대해서는 공식 문서를 참조하시기 바랍니다.
