# Swift에서 문자열의 길이 찾기: 무엇이며 왜 필요한가?

"## 무엇 & 왜?": 문자열의 길이 찾기는 문자열이 총 몇 개의 문자로 이루어져 있는지를 알아내는 것을 의미합니다. 프로그래머들은 이를 자주 사용하는데, 이는 다양한 이유로 인해 문자열의 길이를 알아야 할 때가 많기 때문입니다.

"## 방법:": 다음은 문자열의 길이를 찾는 기본적인 코드 예제와 출력 결과입니다.

```Swift
let str = "Hello World"
print(str.count)
```

*출력 결과: 11*
	
"## 깊은 곳 탐험": 이제 문자열의 길이를 알아내는 방법에 대해 더 자세히 알아보겠습니다. 문자열의 길이를 찾는 메서드는 Swift 언어의 내장 함수이기 때문에 별도의 라이브러리나 패키지를 임포트할 필요 없이 간단하게 사용할 수 있습니다. 또한, Swift에서는 문자열을 다루는 다양한 기능과 함께 문자열의 길이를 쉽게 찾을 수 있도록 제공하고 있습니다.
