# 왜 정규 표현식을 사용해야 할까요?
정규 표현식은 문자열을 다룰 때 매우 유용한 도구입니다. 특히, 문자열에서 패턴을 찾거나 추출할 때 정규 표현식을 사용하면 더 쉽고 빠르게 작업할 수 있습니다.

## 어떻게 정규 표현식을 사용할 수 있을까요?
아래 예시는 주어진 문장에서 모든 이메일 주소를 찾아내는 코드입니다.
```Ruby
# 이메일 주소를 찾는 정규 표현식
emails = /[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,4}/i

# 테스트할 문장
sentence = "이메일 주소는 abc@def.com와 123@456.co.kr 뿐만 아니라, def@ghi.net도 있습니다."

# 정규 표현식을 사용하여 이메일 주소를 찾은 다음 출력
puts sentence.scan(emails)

# 출력 결과:
# ["abc@def.com", "123@456.co.kr", "def@ghi.net"]
```

위 코드에서 볼 수 있듯이 정규 표현식을 사용하면 한 번에 여러 개의 패턴을 찾을 수 있습니다. 이뿐만 아니라, 정규 표현식을 조금 더 복잡하게 작성하면 원하는 형식에 맞지 않는 문자열을 걸러낼 수도 있습니다.

## 딥 다이브
정규 표현식을 작성할 때는 주의해야 할 몇 가지 규칙이 있습니다. 예를 들어, 정규 표현식은 대소문자를 구분하므로 `[A-Z]`는 대문자만, `[a-z]`는 소문자만 찾는다는 것을 기억해야 합니다. 또한, `.`은 모든 문자를 의미하기 때문에 특수한 의미를 갖는 문자(예: `.`은 모든 문자, `*`은 0개 이상의 문자, `+`는 1개 이상의 문자를 의미)는 반드시 이스케이프(\)를 사용하여 일반 문자로 인식할 수 있도록 해야 합니다.
