## 무엇 & 왜?

문자열을 연결하는 것은 무엇인지 설명하고 프로그래머들이 그렇게 하는 이유를 두세 문장으로 설명합니다.

순서대로 읽어서 이해하도록 해보세요 : 

(1) 문자열 연결이란 무엇일까요? 문자열 연결이란, 단순히 두개의 문자열을 합치는 것을 말합니다. 예를 들어서, "안녕하세요"와 "저는 루비입니다"를 합치면 "안녕하세요저는루비입니다"가 됩니다.

(2) 프로그래머들은 왜 문자열을 연결할까요? 문자열 연결은 텍스트를 다루는 작업을 할 때 매우 유용합니다. 만약에 여러분이 월드와이드 웹을 이용하고 있다면, 웹사이트의 제목과 내용을 합쳐서 보여주는 것이 간단하고 빠르게 사이트를 업데이트 할 수 있게 해줍니다. 즉, 문자열 연결은 우리 일상에서 보는 웹사이트, 애플리케이션 등 다양한 곳에서 사용되고 있어 필수적인 기능이라고 할 수 있습니다.

 
## 어떻게:

```ruby
# 기본적인 방법

"Hello, " + "world!" # "Hello, world!"

# 다른 문자열 연결 방법들

"Hello, " << "world!" # "Hello, world!"
"Hello, ".concat("world!") # "Hello, world!"

# 변수를 사용하는 경우

first_name = "Jane"
last_name = "Doe"
full_name = first_name + " " + last_name # "Jane Doe"

# 여러 개의 변수를 합치는 경우

numbers = [1, 2, 3, 4]
numbers.join(", ") # "1, 2, 3, 4"
```

위 예제들은 가장 기본적인 문자열 연결 방법들을 보여줍니다. << 연산자와 concat 메소드는 문자열을 더할 때 사용할 수 있고, join 메소드는 배열 안의 원소들을 구분자로 연결하여 하나의 문자열로 만들어 줍니다. 변수를 사용할 때는 변수들 사이에 문자열을 넣어주는 것을 잊지 말아주세요!

## 깊이 파고들기:

컴퓨터 과학에서, 문자열 연결은 매우 기본적인 개념으로 매우 오래 전부터 사용되고 있습니다. 예전에는 더해주기(+) 연산자를 사용했지만, 루비에서는 더하기 연산자와 << 연산자를 모두 사용할 수 있도록 해줍니다. 또한 자바스크립트의 문자열 템플릿과 유사한 기능을 가진 문자열 보간법(interpolation)을 이용하여 더 간단하게 문자열을 조합할 수 있습니다.

또한 문자열 연결에는 여러 가지 대안이 있습니다. 가령, 문자열을 배열 형태로 저장하고 join 메소드를 이용하여 합치는 방법 외에도, << 연산자를 이용하여 여러 개의 문자열을 쉼표로 구분하지 않고도 한번에 덧붙일 수 있는 방법, 그리고 여러 개의 문자열 보간법을 사용하여 더 스마트하게 문자열을 연결하는 방법 등이 있습니다.

루비 엔진은 C 언어로 작성되었기 때문에 문자열 연결 구현이 매우 빠릅니다. 이전 버전의 루비에서는 concat 메소드가 << 연산자보다 더 느렸습니다. 하지만 최근 JIT 컴파일러와 최적화 기술이 적용되면서 큰 차이는 없어졌습니다.
