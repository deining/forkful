## 왜

랜덤 숫자를 생성하는 것에 참여하는 *왜*에 대해 설명합니다.

컴퓨터 프로그래밍에서 랜덤 숫자를 생성하는 것은 영감을 주는 새로운 아이디어를 찾거나 지루한 코딩 작업을 더 재미있게 만드는 데 도움이 될 수 있습니다. 또한 테스트 데이터를 생성하거나 무작위로 선별해야 하는 경우에도 유용할 수 있습니다.

## 방법

랜덤 숫자를 생성하는 여러 가지 방법이 있지만, 우선 Ruby의 기본 라이브러리에 있는 `rand()` 메서드를 살펴보겠습니다. 이 메서드는 0보다 크거나 같고 1보다 작은 부동 소수점 숫자를 반환합니다. 기본적으로 천만 개의 경우의 수를 가지고 있기 때문에 매번 실행할 때 마다 다른 숫자를 반환합니다.

```Ruby
puts rand()
# 0.746025369562695
puts rand()
# 0.28652315078567
```

같은 결과를 얻고 싶다면 `srand()` 메서드를 사용하여 시드 값을 설정할 수 있습니다. 이 메서드는 랜덤 숫자를 생성하기 전에 호출해야 합니다. `srand()` 메서드에는 임의의 정수 값을 매개변수로 전달하는 것이 좋습니다.

```Ruby
srand(5)
puts rand()
# 0.3858115397578417
srand(5)
puts rand()
# 0.3858115397578417
```

이 외에도 `Random` 클래스를 사용하여 더 정교한 랜덤 숫자를 생성할 수 있습니다. 이 클래스에는 `Random.new()` 메서드를 사용하여 새로운 인스턴스를 만들고, `rand()` 메서드를 호출하여 랜덤 숫자를 생성할 수 있습니다. 이 경우 시드 값을 설정할 필요가 없으며, 다양한 옵션을 사용하여 난수의 범위나 타입을 지정할 수 있습니다.

```Ruby
rand_num = Random.new()
puts rand_num.rand(100..500)
# 381
puts rand_num.rand(-10..10)
# -4
```

## 깊이 파고들기

랜덤 숫자를 생성하는 알고리즘은 매우 복잡하기 때문에 더 자세한 설명은 이 글의 범위를 벗어납니다. 그러나 랜덤 숫자를 생성하는 방식에는 몇 가지 일반적인 구조가 있습니다. 가장 일반적인 것은 선형 합동 발생기(linear congruential generator)입니다. 이 방식은 이전 숫자를 사용하여 새로운 숫자를 생성하는 수학적인 공식을 사용합니다. 해당 공식에는 최소값, 최대값, 시드 값 등의 매개변수가 필요합니다.

또 다른 방식은 난수 생성기(random generator)를 사용하는 것입니다. 이 방식은 큰 수를 생성하기 위해 이전에 생성된 숫자와 시드 값을 조합합니다. 일반적으로 이 방식은 선형 합동 방식보다 더 정교하고 무작위로 보이는 숫자를 생성합니다.

하지만 어떤 방식을 사용하더라도 랜덤 숫자는 완전히 예측할 수 없는 결과를 제공하지는 않습니다. 따라서 랜덤 숫자를 사용하는 데 있어서는 항상 암호화 같은 보안 시스템이나 신뢰성이 중요한 경우는 다른 방법을 사용해야 합니다.
