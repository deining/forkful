# 왜
코드 패턴과 일치하는 문자를 삭제하는 방법을 익히는 이유는 자바스크립트에서 문자열을 조작하는 기술을 배우기 위해서입니다.

## 어떻게 하나요
자바스크립트에서 문자열을 조작하는 방법은 다양하지만, 우리는 여기서 정규표현식을 사용하여 코드 패턴과 일치하는 문자를 삭제하는 방법을 알아보겠습니다. 정규표현식은 일정한 패턴의 문자열을 찾고 조작하는 강력한 도구입니다. 아래의 예제 코드를 통해 정규표현식을 사용하여 코드 패턴과 일치하는 문자를 삭제하는 방법을 살펴보겠습니다.

```TypeScript
const phrase:string = "Hello, TypeScript!";
const result = phrase.replace(/[a-zA-Z]/g, '');
console.log(result); // ", !"
```

위 코드에서는 `replace()` 메서드를 사용하여 `[a-zA-Z]`라는 정규표현식을 이용하여 알파벳 문자를 모두 찾고 빈 문자열로 대체하고 있습니다. 따라서 출력 결과는 `", !"`가 됩니다. 이와 같이 정규표현식을 사용하여 코드 패턴과 일치하는 문자를 삭제할 수 있습니다.

## 깊이 들어가기
정규표현식은 매우 강력한 도구이지만, 표현식을 작성하는 것이 어려울 수도 있습니다. 따라서 정규표현식에 익숙해지는 것이 중요합니다. 간단한 예제를 통해 정규표현식에 대해 더 알아보겠습니다.

- `[a-z]` : 소문자 알파벳을 찾습니다.
- `[A-Z]` : 대문자 알파벳을 찾습니다.
- `[0-9]` : 숫자를 찾습니다.
- `[^a-zA-Z0-9]` : 알파벳과 숫자를 제외한 문자를 찾습니다.

정규표현식을 사용할 때는 이러한 기호들을 조합하여 다양한 패턴에 맞는 문자열을 찾을 수 있습니다. 정규표현식에 대해 더 자세히 알고 싶다면 아래의 링크를 참고하세요.
