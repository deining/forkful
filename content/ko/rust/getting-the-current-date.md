# 왜 날짜를 구하는 것이 중요할까요?
컴퓨터 프로그래밍에서 현재 날짜를 구하는 것은 필수적입니다. 예를 들어, 특정 애플리케이션이나 웹사이트에서 사용자가 언제 계정을 생성했는지를 알아야 할 때가 있습니다. 또는 파일을 생성한 날짜 역시 중요한 정보일 수 있습니다. 이러한 이유로 날짜를 구하는 기능은 매우 유용하고 필수적입니다.

## 날짜를 구하는 방법
Rust는 날짜를 구하는 데에 있어서 효율적이고 직관적인 방법을 제공합니다. 먼저, `chrono`라는 라이브러리를 사용해야 합니다. 다음과 같이 코드를 작성해보겠습니다.

```rust
use chrono::{Local, Date};
// 현재 시간을 구하는 함수
let now: Date<Local> = Local::today();
println!("현재 날짜: {}", now);
```
위 코드를 실행하면 다음과 같은 결과가 출력됩니다.

```
현재 날짜: 2021-06-17
```

`Local::today()` 함수를 호출하면 오늘의 날짜를 가져옵니다. 또한 `Date` 객체를 생성할 때는 타임존 정보를 함께 전달해야 합니다. 위 예제에서는 로컬 타임존을 사용했습니다.

## 더 깊게 들어가기
Rust의 `chrono` 라이브러리는 날짜와 관련된 다양한 기능을 제공합니다. 예를 들어, 특정 날짜와 비교하거나 시간 간격을 계산하는 기능도 있습니다. `chrono`의 공식 문서를 참고하면 더 많은 기능을 확인할 수 있습니다.
