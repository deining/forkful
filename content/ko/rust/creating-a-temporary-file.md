## 왜 

일시적인 파일을 만들기 위해 어떤 사람이 참여하는지에 대해 설명하는 1-2 문장입니다.

Temporary file을 생성하는 이유는 무엇일까요? 컴퓨터 프로그래밍에서 우리는 종종 파일을 사용합니다. 이 파일들은 데이터를 저장하거나 사용자와 상호작용하는 데 사용됩니다. 여러분은 대부분 파일을 사용한 경험이 있을 것입니다. 그렇다면 왜 우리는 일시적인 파일을 만들어야 할까요? 우선, 일시적인 파일은 우리가 사용하는 일반 파일과 달리 더욱 특수한 용도로 사용됩니다. 예를 들어, 프로그램이 실행 중에 임시적인 데이터를 저장하기 위해 일시적인 파일을 만들 수 있습니다. 일시적인 파일은 프로그램이 더 많은 작업을 처리하도록 도와줍니다.

## 하루 

일시적인 파일을 생성하는 방법에 대한 코딩 예시와 ```Rust ... ``` 코드 블록에서의 샘플 출력을 제시합니다.

Rust를 사용하면 파일을 다루는 것이 매우 쉽습니다. 우선, 우리는 Rust의 기본패키지인 std를 사용해야 합니다. 이 패키지는 우리가 필요로하는 기본적인 기능들을 제공합니다. 따라서 이 문제에서도 우리는 std를 이용해야 합니다.

우리가 생성할 일시적인 파일의 이름을 알아보고, 대부분의 운영체제에서는 일시적인 파일을 생성하는 데 환경변수를 사용해야 합니다. 이 환경변수는 ```TMP``` 또는 ```TEMP```로 정의되어 있을 것이며, 이 환경변수의 값을 사용하여 우리는 일시적인 파일을 생성할 위치를 찾을 수 있습니다. 이 값을 디폴트로 사용하면, 우리는 특별한 설정 없이 일시적인 파일을 생성할 수 있습니다.

이제 우리가 생성할 파일의 이름과 경로를 정의해야 합니다. 우리는 파일의 경로, 생성할 파일의 이름, 파일의 확장자, 그리고 접두어를 사용하여 파일의 이름과 경로를 정의할 수 있습니다. 또한 우리는 일시적인 파일이 필요한 드라이브를 지정할 수 있습니다.

다음으로 우리는 일시적인 파일을 생성하는 작업을 수행할 수 있습니다. 이 작업은 ```File::create()``` 함수를 사용하여 파일을 생성할 수 있습니다. 이 함수는 우리가 전달한 이름과 경로에 새로운 파일을 만듭니다.만약 파일을 성공적으로 생성하면, 우리는 파일 핸들을 가지게 될 것입니다. 만약 파일 생성에 실패하면 오류를 반환할 것입니다.

이제 우리는 생성한 파일에 정보를 쓰는 작업을 수행할 수 있습니다. 우리는 ```File::write()``` 함수를 사용하여 파일에 정보를 쓸 수 있습니다.이 함수는 파일 핸들을 통해 이용될 수 있으며, 이 함수는 지정한 데이터를 파일에 쓸 것입니다. 만약 쓰기가 성공적으로 이루어지면, ```File``` 객체는 닫힐 것입니다. 그렇지 않은 경우 ```File``` 객체는 계속해서 열릴 것입니다.
