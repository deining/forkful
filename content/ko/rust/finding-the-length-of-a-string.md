## 왜

본격적으로 Rust 프로그래밍을 시작하기 전에, 꼭 필요한 기본 개념 중 하나인 문자열의 길이를 구하는 방법에 대해 배워보도록 하겠습니다. 문자열의 길이를 구하는 것은 프로그래밍에서 매우 중요한 기능이며, 이를 활용해 다양한 작업을 할 수 있습니다. 따라서 우리는 Rust에서 문자열의 길이를 구하는 방법을 알아보며, 더욱 효율적인 코드를 작성할 수 있도록 준비해보겠습니다.

## 어떻게

먼저, 문자열의 길이를 구하는 가장 간단한 방법은 `.len()` 메소드를 사용하는 것입니다. 이 메소드는 문자열의 길이를 바이트 단위로 반환해줍니다. 예를 들어, `"안녕하세요".len()`은 15를 반환합니다. 또한, `str::len(&문자열)`와 같이 함수를 호출하는 것도 가능합니다.

아래는 `.len()` 메소드를 활용한 간단한 예제 코드입니다.
```Rust
let greeting = "안녕하세요";
let greeting_len = greeting.len();
println!("이 문자열의 길이는 {}입니다.", greeting_len);
```
출력 결과는 다음과 같습니다.
```
이 문자열의 길이는 15입니다.
```

만약 한글을 비롯한 유니코드 문자열의 문자 수를 구하고 싶다면, `.chars().count()` 메소드를 사용해야 합니다. 이 메소드는 유니코드 문자 갯수를 반환해줍니다. 다음 예제 코드를 확인해보세요.
```Rust
let hello = "안녕하세요";
let hello_count = hello.chars().count();
println!("이 문자열은 {}개의 문자로 이루어져 있습니다.", hello_count);
```
출력 결과는 다음과 같습니다.
```
이 문자열은 5개의 문자로 이루어져 있습니다.
```

## 깊게 파헤치기

두 가지 방법 모두 간단하게 문자열의 길이를 구할 수 있지만, 내부적으로는 다른 동작을 하고 있습니다. `.len()` 메소드는 바이트 수를 반환하는 반면, `.chars().count()` 메소드는 실제 유니코드 문자의 개수를 반환해줍니다. 따라서 한글과 같은 다중 바이트 문자를 다룰 때는 `.chars()` 메소드를 사용하는 것이 더 정확하고 안전합니다.

또한, Rust 컴파일러는 문자열 안에 있는 실제 문자의 개수를 알고 있는 경우에는 `.chars().count()` 방식으로 문자열의 길이를 구하도록 권장하고 있습니다. 이는 런타임에서 문자열의 길이를 구하는 것보다 더 효율적인 방법이기 때문입니다.
