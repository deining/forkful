## 과업 & 왜?: 
테스트 작성이란 무엇인지와 프로그래머들이 이를 하는 이유에 대해 1~2 문장으로 설명합니다.

## 작성 방법: 
```Javascript
// 테스트 함수 작성 예제
function add(num1, num2) {
  return num1 + num2;
}

// 위 함수를 테스트하는 예제
console.log(add(2, 3));
// 결과: 5
```
