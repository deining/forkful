# 왜: 난수 생성에 참여하는 이유

난수 생성은 많은 프로그래밍 언어에서 매우 중요한 요소입니다. 이를 활용하면 컴퓨터 프로그램을 더욱 다양하게 만들 수 있습니다. 아두이노에서도 동일한 개념이 적용됩니다. 난수 생성을 통해 다양한 기능을 가진 프로그램을 만들 수 있습니다.

# 이렇게 하세요: ```Arduino ... ``` 코드 블록을 이용한 코딩 예제 및 샘플 출력

아래 코드를 따라하고 새로운 기능을 실험해보세요!
 ```
void setup() {
  // 선언문
  int number;
 
  // 난수 생성
  number = random(10); // 0 부터 9까지의 난수 생성
  Serial.println(number); // 생성된 난수 출력
}
 
void loop() {
  // 추가적인 동작
}
```

위의 코드를 실행하면 0~9 사이의 난수가 생성되어 시리얼 모니터에 출력됩니다. 위의 코드에서 random() 함수에 인자 값을 변경하면 다른 범위의 난수를 생성할 수 있습니다. 또한, random() 함수 대신 randomSeed() 함수를 이용하여 시드 값을 설정하면 더욱 다양한 난수를 생성할 수 있습니다.

# 깊게 들어가보기: 난수 생성에 대한 더 깊은 정보

난수 생성은 컴퓨터의 수식 연산을 통해 이루어집니다. 아두이노에서 사용되는 random() 함수는 랜덤 시드 값을 기반으로 난수를 생성합니다. 시드 값이 같다면 항상 같은 난수를 생성하므로, randomSeed() 함수를 이용하여 시드 값을 변화시키는 것이 중요합니다.

아두이노에서 난수 생성은 난수 생성기를 사용합니다. 이는 시스템의 마이크로세컨드 단위의 시간을 기반으로 난수를 생성하는 방식입니다. 또한, 랜덤 값의 범위를 제한하기 위해 값을 나누거나 곱하는 등의 조작을 할 수 있습니다.
