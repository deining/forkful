# 왜 읽을까요?
이번 글에서는 여러분이 어떻게 텍스트 파일을 읽을 수 있는지에 대해 알려드릴 거에요. 텍스트 파일을 읽는 것은 아두이노 프로그래밍에서 중요한 개념이고, 여러분이 조금만 더 배우고 싶거나 더 깊이 파고들고 싶다면, 이 글을 읽는 게 도움이 될 거에요.

## 왜 읽어야 하나요?

텍스트 파일을 읽는 것은 아두이노 프로그래밍에서 기본적이고 중요한 부분이에요. 텍스트 파일은 우리가 사용하는 컴퓨터에서 매우 중요한 역할을 하고 있기 때문에, 이를 다루는 능력은 여러분에게 큰 도움이 될 거에요.

## 어떻게 하나요?

아래의 예시 코드를 살펴보세요. 이 코드는 "example_text.txt"라는 이름의 텍스트 파일을 읽고, 파일 내용을 시리얼 모니터에 출력하는 간단한 예시에요.

```Arduino
void setup() {
  // 시리얼 통신 초기화
  Serial.begin(9600);

  // 파일을 읽기 모드로 열기
  File file = SD.open("example_text.txt");

  // 파일이 존재하는지 확인하기
  if (file) {
    while (file.available()) {
      // 파일로부터 한 줄씩 읽어서 시리얼 모니터에 출력하기
      Serial.println(file.readStringUntil('\n'));
    }
    // 파일 닫기
    file.close();
  } else {
    // 파일이 존재하지 않을 때 출력할 메시지
    Serial.println("Could not open file.");
  }
}

void loop() {
  // 아무런 동작도 하지 않음
}
```

위 코드를 실행시키면, "example_text.txt" 파일의 내용이 시리얼 모니터에 한 줄씩 출력될 거에요. 텍스트 파일 내의 줄바꿈 문자(\n)를 기준으로 파일을 한 줄씩 읽어온 다음, 시리얼 모니터에 출력하고 있는 거죠.

코드를 실행하려면 필요한 다음 라이브러리들도 추가로 설치해야 해요:
- SD 라이브러리 (SD카드 모듈을 사용할 때)
- SPI 라이브러리 (SD카드 모듈을 사용할 때)
- Wire 라이브러리 (EEPROM이나 외부 메모리를 사용할 때)

## 깊게 파보세요

위 예시 코드에서 사용한 ```SD.open()``` 함수는 SD 카드 모듈이나 외부 메모리를 사용하려면 꼭 필요한 함수에요. 이 함수가 제대로 동작하려면 라이브러리를 설치해야 하고, SD 카드 모듈을 이용하기 위한 연결과 설정도 해야 해요.

또한, 텍스트 파일을 읽는 데 사용하는 다양한 함수들은 여러분이 원하는 방식에 따라 다르게 사용할 수 있어요. 파일 내용을 모두 읽어서 한 번에 처리할 수도 있고, 한 줄씩 읽어서 원하는 로직을 수행할 수도 있어요. 여러분이 읽는 텍스트 파일의 형식이나 로직에 따라 적절하게 함수를 선택하는 것이 중요해요.
