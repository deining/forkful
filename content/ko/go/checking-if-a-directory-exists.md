## 왜 디렉토리의 존재 여부를 확인할까요?

Go 프로그래밍에서 사용하는 디렉토리는 파일 시스템의 중요한 부분입니다. 때때로, 우리는 특정 디렉토리가 존재하는지 여부를 확인해야 할 때가 있습니다. 이를 통해 우리는 프로그램을 더욱 유연하고 안전하게 관리할 수 있을 뿐만 아니라, 예상치 못한 오류를 방지할 수 있습니다.

## 확인 방법

다음은 Go 언어를 사용하여 디렉토리 존재 여부를 확인하는 간단한 예시 코드입니다. 

```Go
package main

import (
  "fmt"
  "os"
)

func main() {
  path := "/Users/UserName/Documents"

  if _, err := os.Stat(path); err == nil {
    fmt.Printf("디렉토리가 존재합니다.")
  } else {
    fmt.Printf("디렉토리가 존재하지 않습니다.")
  }
}
```

위의 코드를 실행하면 해당 디렉토리가 존재하는 경우 `디렉토리가 존재합니다.`라는 메시지가 출력되고, 그렇지 않은 경우 `디렉토리가 존재하지 않습니다.`라는 메시지가 출력됩니다.

## 깊이 있는 정보

디렉토리 존재 여부를 확인하는 방법은 다양하지만, 위에서 제시한 예시는 가장 간단하면서도 일반적인 방법입니다. 만약 여러분이 디렉토리 경로를 입력받는 경우라면, 입력 값에 따라 존재 여부를 동적으로 확인할 수 있도록 코드를 구성하는 것이 좋습니다.

또한, `filepath` 패키지를 사용하면 디렉토리 존재 여부뿐만 아니라 다양한 파일 경로를 쉽게 다룰 수 있습니다. 해당 패키지는 Go 언어의 공식 패키지이므로, 적극적으로 활용하는 것이 좋습니다.
