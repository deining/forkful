## 이거 뭐고 왜 해야하나요?
여러분은 새로운 프로젝트를 시작하려면 무엇이 필요한지 아시나요? 프로그래머들이 새로운 프로젝트를 시작하는 이유는 무엇일까요? 새로운 프로젝트란 무엇인가요?

- 새로운 프로젝트란 새로운 소프트웨어나 앱, 웹사이트 등을 만드는 과정을 말합니다. 이는 새로운 아이디어를 실제로 구현하는 것을 의미하며, 실제로 이용자들에게 제공되는 결과물이 될 수 있습니다.
- 프로그래머들은 새로운 기능을 추가하거나, 기존의 문제를 해결하기 위해 새로운 프로젝트를 시작합니다. 또한, 새로운 언어나 기술을 학습하기 위해서도 새로운 프로젝트를 시작할 수 있습니다.

## 어떻게 하나요?
자바는 오랜 역사를 가진 인기 있는 프로그래밍 언어입니다. 새로운 프로젝트를 시작하는 가장 기본적인 예제는 "Hello World!"를 출력하는 것입니다.

```java
public class Main {
  public static void main(String[] args) {
    System.out.println("Hello World!");
  }
}
```

출력 결과는 다음과 같습니다.

```
Hello World!
```

## 깊이 파고들기
자바는 1995년에 제임스 고슬링과 그의 동료들이 개발한 객체지향 프로그래밍 언어입니다. 현재까지도 많은 개발자들이 사용하고 있으며, 다양한 플랫폼에서 실행되는 크로스 플랫폼 언어로 발전하고 있습니다.

새로운 프로젝트를 시작하는 다른 방법으로는 다양한 자바 프레임워크를 이용하는 것이 있습니다. 스프링 프레임워크나 하이버네이트 등은 자주 사용되는 자바 프레임워크 중 하나입니다. 이 외에도 안드로이드 앱 개발에 주로 사용되는 안드로이드 스튜디오가 있습니다.

새로운 프로젝트를 시작할 때, 우리는 보통 이전의 프로젝트에서 배운 것들을 활용하고 복제하는 경우가 많습니다. 이를테면, 같은 데이터베이스 구조나 기능들을 사용하여 새로운 앱을 만들 수 있습니다.
