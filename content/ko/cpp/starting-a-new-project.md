# 무엇 & 왜
새 프로젝트를 시작하는 것은 새로운 컴퓨터 프로그램이나 애플리케이션을 만드는 것을 의미합니다. 프로그래머들은 새로운 아이디어를 구현하고 새로운 기술을 배우며, 사용자에게 더 나은 서비스를 제공하기 위해 새 프로젝트를 시작합니다.

## 어떻게
새로운 C++ 프로젝트를 시작하는 방법은 매우 간단합니다. 먼저 새로운 폴더를 만들고 그 안에 ```main.cpp```라는 이름의 파일을 생성합니다. 그리고 아래의 예시 코드를 복사하여 붙여넣고 실행하면 됩니다.

```C++
#include <iostream>

// 주석: 이 부분은 코드에 설명을 추가하는 용도로 사용됩니다.

// main() 함수는 프로그램의 진입점(entry point)이며 모든 프로그램은 이 함수에서 시작됩니다.
int main() {
  std::cout << "새로운 프로젝트를 시작합니다!" << std::endl;  // 화면에 "새로운 프로젝트를 시작합니다!"를 출력합니다.
  return 0;  // main() 함수의 끝을 나타냅니다.
}
```

위 코드를 실행하면 콘솔창에 "새로운 프로젝트를 시작합니다!"가 출력될 것입니다.

## 깊이 들어가기
새 프로젝트를 시작하는 것은 매우 중요한 일입니다. 새로운 아이디어를 구현하고 앞서 사용하지 않은 새로운 기술을 배우는 과정에서 많은 도전과 성취감을 경험할 수 있습니다.

C++ 외에도 프로그래밍 언어로는 다른 옵션이 있습니다. C++는 많은 기능을 제공하고 성능이 우수하지만 다른 언어에 비해 어려울 수 있습니다. 프로젝트의 목적과 개인의 취향에 따라 다른 언어를 선택할 수 있습니다.

새 프로젝트를 시작할 때 주의할 점도 있습니다. 꼭 필요한 기능만 추가하여 코드를 깔끔하게 유지하고, 주석을 작성하여 나중에 코드를 이해하기 쉽도록 하고, 주기적으로 백업을 하여 데이터를 보호하는 것이 좋습니다.
