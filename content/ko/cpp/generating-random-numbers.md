# 왜

난수 생성을 도전해볼 이유는 무엇일까요? 잠깐만 기다려 주세요! 많은 이들이 난수 생성이 왜 중요한지 궁금하실텐데요, 이번 글에서는 그 이유를 자세히 알아보도록 하겠습니다.

# 어떻게

난수 생성은 프로그래밍에서 매우 유용합니다. 예를 들어, 게임에서 랜덤한 아이템을 생성하거나, 암호화 기법에서 초기화 벡터를 생성하는 데 사용될 수 있습니다. 난수 생성을 구현하는 방법을 살펴보겠습니다. 아래 코드 블록을 참고해주세요.

```C++
#include <iostream>
#include <cstdlib> // rand() 함수를 사용하기 위해 추가합니다.
#include <ctime> // 시드값을 설정하기 위해 추가합니다.

using namespace std;

int main()
{
    // 시드 값 설정
    srand(time(0));

    // 0부터 9까지의 난수 생성
    int num1 = rand() % 10;
    cout << num1 << endl;

    // 1부터 10까지의 난수 생성
    int num2 = rand() % 10 + 1;
    cout << num2 << endl;

    // 5부터 15까지의 난수 생성
    int num3 = rand() % 11 + 5;
    cout << num3 << endl;

    return 0;
}
```

위 코드는 `cstdlib` 라이브러리의 `rand()` 함수를 사용하여 난수를 생성하는 방법을 보여줍니다. 또한, `ctime` 라이브러리를 사용하여 시드 값을 설정하여 매번 다른 난수를 생성할 수 있도록 합니다. 매번 실행할 때마다 아래와 같은 결과가 출력됩니다.

```
7
10
5
```

이제 원하는 구간의 난수를 생성할 수 있는 방법을 알게 되었으니, 이제는 난수 생성의 원리에 대해 자세히 알아보도록 하겠습니다.

# 깊이 들어가기

`rand()` 함수는 `0`부터 `RAND_MAX` 사이의 값을 반환합니다. `RAND_MAX`는 표준 라이브러리에서 정해진 최대 난수 값으로, `cstdlib` 라이브러리에서는 `32767`로 정의되어 있습니다. 따라서 `0`부터 `32767`까지의 값을 반환하게 됩니다. 이 값은 시드 값을 기반으로 산출되는 일련의 숫자입니다. 따라서 시드 값이 같다면 매번 같은 순서로 난수가 생성되기 때문에, 매번 다른 난수가 생성되도록 시드 값을 변경해주는 것이 중요합니다.

난수 생성 알고리즘은 매우 복잡하지만, 다음의 공식을 사용하여 간단히 구현할 수 있습니다.

```
num = (rand() % (max - min + 1)) + min
```

여기서 `num`은 생성할 난수의 변수, `max`와 `min`은 원하는 구간의 최대값과 최소값을 의미합니다. 이를 통해 우리는 간단한 방법으로 다양한 구간의 난수를 생성할 수 있게 됩니다.
