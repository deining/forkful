## 왜
날짜를 두 개 비교하는 것에 참여하려는 이유는 무엇일까요? 날짜는 프로그래밍에서 중요한 요소이며, 데이터를 분석하고 통계를 작성할 때 자주 사용됩니다. 따라서 날짜를 정확하게 비교하는 방법을 알아두는 것은 매우 중요합니다.

## 어떻게
두 개의 날짜를 비교하는 방법은 간단합니다. 먼저 ```<iostream>``` 헤더 파일을 포함하고, 날짜를 나타낼 수 있는 변수 두 개를 선언합니다.

```C++
#include <iostream>

int main() {
  int date1 = 10252021; // YYYYMMDD 형식으로 나타냅니다.
  int date2 = 11012021;

  // 비교하는 코드를 작성합니다.
}
```

이제 비교하는 방법에는 여러 가지가 있을 수 있지만, 가장 간단한 방법은 두 날짜를 각각 ```date1```과 ```date2```라는 변수에 저장하고, ```if```문을 사용하여 두 변수가 같은지 비교하는 것입니다.

```C++
if (date1 == date2) {
  std::cout << "두 날짜는 같습니다.";
}
else {
  std::cout << "두 날짜는 다릅니다.";
}
```

위의 코드는 두 날짜가 같은 경우 "두 날짜는 같습니다."라는 메시지를 출력하고, 다른 경우 "두 날짜는 다릅니다."라는 메시지를 출력합니다. 따라서 프로그램을 실행하면, 다음과 같은 결과가 나타납니다.

```
두 날짜는 다릅니다.
```

## 깊게 들어가기
날짜를 비교하는 더 많은 방법들이 있지만, 위에서 보여드린 방법은 가장 기본적이고 간단한 방법입니다. 그러나 이 방법만으로는 날짜를 정확하게 비교할 수 없는 경우가 있습니다. 예를 들어, 2월이나 31일처럼 정확한 날짜인지 확인할 수 없는 경우에는 어떻게 해야 할까요? 이러한 경우에는 C++의 일련의 라이브러리 함수를 사용하여 날짜를 비교하는 것이 더 정확하고 효율적일 수 있습니다.

예를 들어, ```time.h``` 헤더 파일에는 시간과 관련된 여러 가지 함수들이 포함되어 있습니다. 이 중에서 ```difftime()``` 함수는 두 날짜를 비교하여 일수 차이를 구할 수 있습니다. 또는 ```ctime()``` 함수를 사용하여 날짜를 실제로 문자열로 나타내는 것도 가능합니다. 따라서 이러한 라이브러리 함수들을 통해 더 정확하고 유연하게 날짜를 비교할 수 있습니다.
