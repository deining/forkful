# Dlaczego warto zacząć nowy projekt?

Zastanawiasz się, dlaczego miałbyś zacząć nowy projekt w Fish Shell? Powody mogą być różne - może chcesz nauczyć się nowej umiejętności, stworzyć narzędzie ułatwiające Twoją pracę lub po prostu zaspokoić swoją ciekawość. Niezależnie od powodu, programowanie w Fish Shell może być nie tylko przyjemne, ale również bardzo użyteczne w codziennym życiu.

# Jak to zrobić?

Pierwszym krokiem w rozpoczęciu projektu w Fish Shell jest zainstalowanie tego narzędzia na swoim komputerze. W tym celu możesz skorzystać z menedżera pakietów, na przykład `brew` na macOS lub `apt` na systemach Linux. Następnie przejdź do tworzenia nowego katalogu i przystąp do kodowania!

```Fish Shell
mkdir nowy-projekt
cd nowy-projekt
```

Teraz możesz zacząć pisać swoje skrypty w Fish Shell. Pamiętaj, że Fish Shell oferuje wiele przydatnych funkcji, które mogą ułatwić Twoją pracę, na przykład autouzupełnianie czy kolorowanie składni. Sprawdź poniższy przykład:

```Fish Shell
set my_name Jan
echo Witaj, $my_name!  # wyświetla się: Witaj, Jan!
```

# Wnikliwe spojrzenie

Podczas tworzenia projektu w Fish Shell możesz również skorzystać z jego bibliotek i pluginów. Dzięki temu możesz znacznie rozszerzyć możliwości tego narzędzia. Na przykład, jeśli chcesz wyświetlić informacje o pogodzie w Twoim miejscu zamieszkania, możesz skorzystać z biblioteki `wttr.in`. Wystarczy, że wpiszesz w terminalu:

```Fish Shell
curl wttr.in  # wyświetla prognozę pogody w Twoim miejscu
```

To tylko jedno z wielu przykładów, jak możesz wykorzystać Fish Shell w swoim projekcie. Pamiętaj, że im głębiej wnikać będziesz w jego możliwości, tym więcej możesz osiągnąć.
