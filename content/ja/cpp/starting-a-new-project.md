## 何をするのか？ それともなぜ？
新しいプロジェクトを始めるとは、新しいソフトウェアやアプリケーションを開発することを指します。プログラマーたちは、新しいプロジェクトを開始することで、新しいアイデアを実現したり、既存のコードを改善したりすることができます。

また、新しいプロジェクトを始めることで、最新の技術やツールを使用したり、チームメンバーと協力してプロジェクトを進めたりすることができます。

## 方法：
新しいプロジェクトを始める際には、まずプログラムを作成する準備をしなければなりません。 ```C++ ... ```コードブロック内に、コーディングの例と出力のサンプルを示します。

例えば、新しいプロジェクトを始めるためには、最初にC++のプログラムファイルを作成します。次に、必要なライブラリをインクルードし、コーディングを開始します。

```
#include <iostream>

int main() {
    std::cout << "新しいプロジェクトを始めます！" << std::endl;
    return 0;
}
```

このコードをコンパイルして実行すると、ターミナルに「新しいプロジェクトを始めます！」というメッセージが表示されるはずです。
