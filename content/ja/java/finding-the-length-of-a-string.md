# なぜ
Javaプログラミングの世界では、文字列の長さを求めることは非常に一般的な作業です。文字列の長さを求めることにより、データの処理や操作が可能になり、プログラムの機能性が向上します。今回は、Javaで文字列の長さを求める方法をご紹介しましょう。

## 手順
まずは、文字列の長さを求めるための基本的なコードをご紹介します。

```Java
String str = "こんにちは！";
int length = str.length();
System.out.println(length);
```

上記のコードを実行すると、コンソールには「5」という文字列の長さが表示されます。このように、```length()```メソッドを使用することで文字列の長さを求めることができます。

また、文字列の中に含まれる特定の文字の数を求めることもできます。例えば、「こんにちは！」という文字列には「ん」が1つ含まれています。その数を求めるには、以下のようなコードを使用します。

```Java
String str = "こんにちは！";
int count = str.length() - str.replace("ん", "").length();
System.out.println(count);
```

このコードを実行すると、コンソールには「1」という数が表示されるので、この文字列には「ん」が1つ含まれていることがわかります。

## 深堀り
文字列の長さを求める方法はとても簡単ですが、実はこのメソッドにはいくつかのオプションが存在します。たとえば、```length()```メソッドの代わりに```codePointCount()```メソッドを使用することで、サロゲートペアと呼ばれる特殊な文字に対応することができます。

さらに、様々な言語や文字コードに対応するために、```codePointCount()```メソッドには追加のパラメーターを指定することができます。詳しくはJavaの公式ドキュメントや専門書を参考にしてみてください。
