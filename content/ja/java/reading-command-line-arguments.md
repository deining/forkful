## 何と なぜ?

コマンドライン引数を読み取ることとは、プログラマーがプログラムを実行する際に利用できるパラメーターを指定することを意味します。これを行う理由は、プログラムが複数の異なる状況で動作する必要がある場合に、異なるパラメーターを指定することで必要な動作を実現することができるためです。

## 方法:

```Java
public static void main(String[] args) {
  // コマンドライン引数を取得する
  String arg1 = args[0]; // 最初の引数を取得
  String arg2 = args[1]; // 2番目の引数を取得
  
  // コンソールに出力する
  System.out.println("最初の引数: " + arg1);
  System.out.println("2番目の引数: " + arg2);
}
```

**出力:**

```
最初の引数: HelloWorld
2番目の引数: Java
```

## 詳細:

- **歴史的背景:** コマンドライン引数は、古い時代から存在している機能です。コンピューターがまだコマンドラインでしか操作できなかった時代、プログラマーたちはコマンドライン引数を使用して複数のタスクを実行しました。現在でも、コマンドライン引数は様々なプログラムで使用されています。

- **代替手段:** コマンドライン引数の代わりに、プログラム内で直接パラメーターを指定することもできます。しかし、コマンドライン引数を使用することで、プログラムを実行する前に再コンパイルする必要がなくなるため、柔軟性が増します。

- **実装の詳細:** Javaでは、コマンドライン引数は```main```メソッドのパラメーターとして配列として渡されます。```args```配列内の各要素は、プログラム実行時に指定した引数の値を保持しています。配列の最初の要素は、```args[0]```というように指定することで取得することができます。
