## なぜ
ランダムな数字を生成することの魅力をお伝えします。

## 方法
```Fish Shell```を使用して簡単にランダムな数字を生成する方法をご紹介します。

まずはFish Shellを開き、以下のコードを入力してください。

```
set random_number (math --random %10)
echo "ランダムな数字は$random_numberです。"
```

上記のコードを実行すると、0から9までのランダムな数字が生成されます。もし、0から10までの数字を生成したい場合は「%10」の部分を「%11」に変更しましょう。

さらに、上記のコードを繰り返し実行すると、毎回異なるランダムな数字が生成されることがわかります。

```
set random_number (math --random %10)
echo "ランダムな数字は$random_numberです。"
set random_number (math --random %10)
echo "ランダムな数字は$random_numberです。"
```

出力結果は以下のようになります。

```
ランダムな数字は8です。
ランダムな数字は3です。
```

また、特定の範囲内のランダムな数字を生成したい場合は、以下のようにコードを記述します。

```
set start 20
set end 30
set random_number (math --random $end-$start+1+$start)
echo "ランダムな数字は$start以上$end以下の範囲内で、$random_numberです。"
```

出力結果は以下のようになります。

```
ランダムな数字は20以上30以下の範囲内で、25です。
```

## 深堀り
ランダムな数字を生成する際に使用する関数は```math --random```です。この関数は内部的には乱数ジェネレーターを使用しており、毎回異なる数字を生成することができます。

また、上記で使用したようにパラメーターを指定することで特定の範囲内の数字を生成することもできます。範囲は任意の数値や変数で指定することができます。

しかしながら、この乱数ジェネレーターは完全にランダムな数字を生成するわけではありません。コンピューター内部で使用する種の「種」と呼ばれる値によって決まるため、実際には何らかの規則性があります。そのため、完全にランダムな数字が必要な場合は、外部の乱数ジェネレーターを使用することが推奨されます。
