## ソースコードを活用：日付を文字列から取得する方法

あなたがプログラマーであるならば、カレンダーアプリ、日付指定の機能、または過去の日付を参照する機能を開発する可能性があります。こうした機能の実現には、日付形式を文字列から取得することが必要不可欠です。今回は、Fish Shellにおける日付の文字列パース方法についてご紹介します。

## 何をしようとしているの？

日付を文字列から取得するとは、以下を意味します：
1. プログラマーは、yyyy-MM-ddのような形式で日付情報を入力することで、過去または未来の日付を指定することができます。
2. ソフトウェアは、入力された日付に応じて適切な処理を行うことができます。

プログラマーが日付を文字列から取得する理由は、予め定められたフォーマットに沿った情報を取得することで、日付に関する処理を容易に行うためです。

## 方法：

日付を文字列から取得する方法を説明します。
```Fish Shell
set current_date (date +%Y-%m-%d)
echo $current_date
```
上記のコードでは、`date +%Y-%m-%d` というコマンドを使用し、現在の日付を `yyyy-MM-dd` 形式で取得しています。その後、 `current_date` 変数にこの日付が設定され、`echo` コマンドを使用して変数の中身を表示しています。実行結果は、 `2021-06-03` のようになります。

## 深く掘り下げてみよう

日付の文字列パースに関する歴史的な背景を調べてみると、C言語やUnix環境では、`localtime` という関数が使用されていました。これは、数値形式の日付を文字列形式に変換するために使用されており、現在でも一部の環境では使用されています。

しかし、Fish Shellでは、`date` コマンドを使用することでより簡単に日付を文字列形式で取得することができます。さらに、`set` コマンドを使用することで変数を定義し、取得した日付を格納することもできます。

日付の文字列パースには、他にもライブラリやモジュールを利用することもできますが、Fish Shellでは標準のコマンドを使用することで手軽に実現することができます。
