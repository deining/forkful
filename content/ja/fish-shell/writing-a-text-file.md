## なぜ

テキストファイルを書く理由はどのようなものがあるのでしょうか？テキストファイルを使うことで、情報を整理し、プログラミングのコードを保管することができます。また、複数の人で共有することも可能です。

## 作り方

テキストファイルを作成するには、Fish Shellを使用することができます。まずは「touch」というコマンドを使用して、空のテキストファイルを作成します。その後、テキストエディタを開き、内容を記述します。最後に、保存して閉じることで、テキストファイルが完成します。

```Fish Shell
touch filename.txt
nano filename.txt
```
ファイル名の部分は任意の名前をつけることができます。

## ノウハウを深める

テキストファイルを書く際の注意点として、ファイルの拡張子を忘れないことが挙げられます。拡張子をつけることで、他のプログラムで開いたり、コンピューターが認識しやすくなります。また、文法を正しく使用することも重要です。例えば、ファイル名や文章のスペースを含む部分は引用符で囲むようにしましょう。
