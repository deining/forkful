## なぜ

エルムプログラミングの初心者の皆さん、こんにちは！今日は、エルムでのサブ文字列の抽出についてお話ししたいと思います。サブ文字列の抽出はとても便利な機能であり、クリーンなコードを書くのに役立ちます。そのため、エルムのプログラミングにおいて重要なスキルの一つです。では、早速見ていきましょう！

## 方法

サブ文字列を抽出するには、2つの方法があります。1つは`String.slice`関数を使う方法です。この関数は、文字列の始まりと終わりの位置を指定することで、その範囲内のサブ文字列を抽出します。実際のコードと出力は以下のようになります。

```elm
name = "Elmはとても面白いです！"
slicedName = String.slice 0 3 name
```
出力：
```
"Elm"
```

もう1つの方法は、`String.left`や`String.right`関数を使用する方法です。これらの関数は、文字列の先頭や末尾から特定の文字数を抽出します。例えば、以下のコードでは、文字列の先頭から3文字を抽出しています。

```elm
name = "Elmはとても面白いです！"
leftName = String.left 3 name
```

出力：
```
"Elm"
```
