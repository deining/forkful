## なぜ 

パターンにマッチする文字を削除するのに、なぜプログラミングをするのかというと、例えば文章を編集するときに、不要な文字を削除したい場合があります。また、テキストデータを整理する際にも、パターンにマッチする文字を削除することで、データの整合性を保つことができます。

## 使い方 

パターンにマッチする文字を削除するには、Pythonで以下のコードを使うことができます。

```Python
import re # reモジュールをインポートする
sentence = "私の名前は東京です。" # 文字列の定義
pattern = r"の|は" # 削除したい文字のパターンを定義
result = re.sub(pattern, "", sentence) # patternにマッチする文字を削除
print(result) # 結果を出力する
```

出力結果：私私東京です。

## 深堀り 

文字の削除には、正規表現を使用することができます。正規表現とは、パターンにマッチする文字列を検索や置換するための表現方法です。上記の例では、`r"の|は"`というパターンを定義しています。ここで、`|`は「または」を意味し、`の`または`は`にマッチする文字を削除することができます。

また、`re.sub()`以外にも、`re.match()`や`re.search()`といった関数を使うことで、パターンにマッチする文字列を検索することができます。正規表現を使うことで、より柔軟に文字を操作することができます。
