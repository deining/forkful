# なぜ文字列の長さを求めるのか

プログラミングにおいて、文字列の長さを求めることは非常に重要です。文字列の長さを知ることで、テキストの解析や処理を行うことができます。例えば、ユーザーが入力したパスワードの長さをチェックしたり、文章の長さに応じてレイアウトを調整したりすることができます。Go言語で文字列の長さを求める方法を学びましょう。

## 手順：Go言語で文字列の長さを求める方法

Go言語では、len()関数を使って文字列の長さを求めることができます。以下のようなコードを書くことで、文字列の長さを求めることができます。

```
package main

import "fmt"

func main() {
    str := "こんにちは世界"
    fmt.Println(len(str))
}
```

上記のコードを実行すると、出力結果は12となります。Go言語では、文字列はユニコードで表現されるため、日本語の文字数も正しくカウントされます。また、スペースや記号も含まれるため、単純に文字数を数えるだけではなく、テキストの解析にも活用できます。

## 深堀り：Go言語で文字列の長さを求める仕組み

Go言語では、string型はバイトの配列として管理されます。そのため、len()関数は単純に配列の長さを返すだけではなく、バイト数を返すことになります。例えば、日本語の「こんにちは世界」はUTF-8で表現すると12バイトとなり、それぞれの文字は3バイトで表されます。

また、Go言語では文字列をイミュータブル（変更不可）なデータ型として扱います。そのため、文字列の長さを変更することはできません。しかし、文字列を変更する場合はバイト配列として扱うことができるため、それぞれの文字を個別に変更することができます。
