## なぜ

ディレクトリが存在するかどうかを確認することに対しては、Go言語プログラマーたちにとって非常に重要です。例えば、アプリケーションがファイルを処理する必要がある場合、事前にディレクトリが存在するかどうかを確認することで、ユーザーが望む結果を得ることができます。

## 方法

ディレクトリが存在するかどうかを確認するには、os パッケージの `Stat` 関数を使用します。以下のようにコードを書くことで、任意のファイルやディレクトリが存在するかどうかを確認することができます。

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    fileInfo, err := os.Stat("example")
    if os.IsNotExist(err) {
        fmt.Println("ディレクトリは存在しません。")
    } else {
        fmt.Println("ディレクトリは存在します。")
        fmt.Println(fileInfo)
    }
}
```

上記のサンプルコードでは、`os.Stat` 関数を使用してファイルまたはディレクトリの情報を取得し、`os.IsNotExist` 関数を使ってエラーが存在しないかどうかをチェックしています。

## ディープダイブ

`os.Stat`関数は、ファイルやディレクトリの情報を取得するためだけでなく、その他の情報も提供してくれます。例えば、ファイルのパーミッションやサイズ、更新日時などを取得することができます。また、ファイルがシンボリックリンクやデバイスファイルであるかどうかも確認することができます。
