## なぜ？
プログラミングをしているとき、一時的なファイルを使用することはよくあります。ファイルを作成し、編集した後に必要がなくなったら削除する、といったことをします。これらの一時的なファイルは、プログラムの実行中に必要な作業用の一時的なスペースとして有用です。

## 作り方
一時的なファイルをGoで作成する方法はとても簡単です。まずは`ioutill`パッケージをインポートします。次に、一時的なファイルを作成するための`TempFile()`関数を使います。以下のコードを参考にしてください。

```Go
package main

import (
	"fmt"
	"io/ioutil"
)

func main() {
	// 一時的なファイルを作成し、エラーを確認する
	file, err := ioutil.TempFile("", "example")
	if err != nil {
		panic(err)
	}
	defer file.Close()

	// 一時的なファイルの情報を出力する
	fmt.Println("一時的なファイルの名前:", file.Name())
	fmt.Println("作成されたディレクトリ:", file.Name())
}
```

上記のコードは、プログラムが実行されると一時的なファイルが作成され、そのファイルの名前と作成されたディレクトリの情報が出力されます。実際に実行して、出力結果を確認してみてください。

```
一時的なファイルの名前: /var/folders/jd/7t6tlclr42n4c799ss02t8000000gn/T/example358289662
作成されたディレクトリ: /var/folders/jd/7t6tlclr42n4c799ss02t8000000gn/T/example358289662
```

## 詳細を深める
一時的なファイルを作成する際に、指定したディレクトリに作成したいというときもあります。そのような場合には、`TempFile()`関数の第一引数にディレクトリのパスを指定することで実現できます。また、ファイルの内容を書き込む必要がある場合は、`WriteString()`や`Write()`関数を使ってファイルに書き込むことができます。さらに詳しい情報は、`ioutill`パッケージのドキュメントをご覧ください。
