# なぜ文字列を連結するのか？
文字列の連結は、Rustプログラミングでよく使用される基本的なタスクです。例えば、ユーザーから受け取った入力をフォーマットして表示する場合や、複数の文字列を結合して新しい文字列を作成する場合などに使用されます。文字列の連結は、プログラムの柔軟性と効率性を向上させるために重要なスキルです。

## 連結の方法
Rustで文字列を連結するには、`+`演算子を使用します。例えば、次のように記述することで文字列を連結することができます。
```Rust
let first_name = "太郎";
let last_name = "山田";
let full_name = first_name + last_name;
println!("名前: {}", full_name);
```

上記のコードは、`太郎山田`という文字列を作成し、`名前: 太郎山田`という出力を生成します。

また、複数の文字列を連結する場合は、`format!`マクロを使用することもできます。次の例では、`format!`マクロを使用して3つの文字列を連結しています。
```Rust
let food = "寿司";
let drink = "日本酒";
let dessert = "抹茶アイスクリーム";
let menu = format!("ご注文は {} と {}、そしてデザートは {} です。", food, drink, dessert);
println!("{}", menu);
```

上記のコードからは、`ご注文は寿司と日本酒、そしてデザートは抹茶アイスクリームです。`という出力が生成されることがわかります。

## 深く掘り下げる
文字列を連結する時には、`+`演算子による単純な結合方法だけでなく、`format!`マクロを使用することで、より複雑な文字列のフォーマットが可能になります。例えば、`format!`マクロでは、変数の型に応じてフォーマットを行うことができます。
```Rust
let age = 20;
let message = format!("今日は私の {} 歳の誕生日です。", age);
println!("{}", message);
```

上記のコードは、`今日は私の 20 歳の誕生日です。`という出力を生成します。このように、`format!`マクロを使用することで、文字列をより柔軟に操ることができます。
