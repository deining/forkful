# なぜ

日付を文字列に変換することについて考えたことはありますか？よく使われるデータ型の1つである日付を文字列に変換することは、Cプログラミングの世界で重要なことです。なぜなら、日付を文字列に変換することで、ユーザーにとって分かりやすい形式で日付を表示したり、日付を処理したりすることができるからです。この記事では、C言語で日付を文字列に変換する方法をご紹介します。

# 方法

日付を文字列に変換するには、`strftime()`という関数を使用します。この関数は、指定したフォーマットに従って、日付を文字列に変換してくれます。例として、以下のコードをご覧ください。


```C
#include <stdio.h>
#include <time.h>

int main()
{
    // 現在の日付を取得
    time_t current_time;
    time(&current_time);
    
    // 文字列に変換するためのバッファーを用意
    char buffer[25];
    
    // 日付をフォーマット指定して文字列に変換
    strftime(buffer, 25, "%Y-%m-%d", localtime(&current_time));
    
    // 変換した文字列を出力
    printf("今日の日付は %s です", buffer);
    
    return 0;
}
```

このコードを実行すると、以下のような結果が得られます。

```
今日の日付は 2021-12-31 です
```

このように、`strftime()`関数を使用することで、簡単に日付を文字列に変換することができます。ただし、フォーマット指定には`strftime`に用意されている特別な文字列を使用する必要があります。詳細は次のセクションで説明します。

# ディープダイブ

先ほど紹介した`strftime()`関数のフォーマット指定には、決まったルールがあります。以下に、よく使用されるフォーマットの一部をご紹介します。

- `%Y`: 年（4桁）
- `%m`: 月（2桁）
- `%d`: 日（2桁）
- `%H`: 時間（24時間表記）
- `%M`: 分
- `%S`: 秒
- `%A`: 曜日（英語）
- `%B`: 月（英語）

例えば、年-月-日というフォーマットで日付を表示するには、`%Y-%m-%d`と指定します。また、曜日や月を英語で表示するには、`%A`や`%B`を使用します。詳細なフォーマット指定については、参考リンクをご覧ください。
