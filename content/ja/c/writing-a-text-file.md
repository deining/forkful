## なぜ
テキストファイルを記述する理由はさまざまです。例えば、データのバックアップや共有、プログラムの設定ファイルの作成などが挙げられます。C言語を使ってテキストファイルを作成する方法を学ぶことで、より効率的に作業を行うことができます。

## 作り方
テキストファイルを作成するには、まずファイルを開きます。その後、テキストを書き込んでいきます。最後にファイルを閉じることで、変更を保存することができます。

```C
#include <stdio.h> //標準入力出力ライブラリをインクルード

int main(){
    FILE *fp; //ファイルポインタを定義

    fp = fopen("sample.txt", "w"); //ファイルを書き込みモードで開く

    if(fp == NULL){ //エラーチェック
        printf("ファイルを開けませんでした。");
        return 1;
    }
    fprintf(fp, "こんにちは、世界！"); //テキストを書き込む
    
    fclose(fp); //ファイルを閉じる

    return 0; 
}
```

## 詳細
テキストファイルを作る際には、使用するファイルポインタの種類を選ぶことが重要です。例えば、書き込みモードでファイルを開く場合は、既存のファイルが存在する場合は中身が上書きされます。また、読み込みモードでファイルを開く場合は、ファイルが存在しない場合エラーが発生します。
