なぜ： ランダムな数値を生成することに興味がある人のための簡単な説明です。

## Why

ランダムな数値を生成することは、さまざまなプログラムやアプリケーションを開発する際に非常に重要な役割を果たします。ランダムな数値は、ゲームデザインや乱数に基づくアルゴリズム、さまざまな統計データの収集など、多くの分野で利用されています。また、ランダムな数値を使用することで、プログラムの可読性やセキュリティを向上させることができます。

## How To

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    // 数値をランダムに生成するためのシードを取得
    srand(time(NULL));

    // 0~10までのランダムな整数を生成する
    int random_num = rand() % 11;
    printf("%d\n", random_num);

    // 0~1までのランダムな小数を生成する
    float random_float = (float) rand() / (float) RAND_MAX;
    printf("%f\n", random_float);

    return 0;
}
```

**出力：**

```
5
0.895730
```

上記のコードでは、`<stdio.h>`と`<stdlib.h>`ライブラリを使用して、標準出力にランダムな数値を表示する方法を示しています。まず、`rand()`関数を使用してランダムな数値を生成し、`srand()`関数に`time(NULL)`を引数として渡すことで、実行ごとに異なるシードを設定し、よりランダムな数値を生成します。次に、`%`演算子を使用して、取得したランダムな数値を特定の範囲内に制限することができます。上記の例では、0~10までの整数を生成するように設定しています。また、小数を生成する場合は`rand()`関数を`RAND_MAX`で割ることで実現できます。このように、ランダムな数値を生成する方法はさまざまな方法がありますので、実際にプログラミングする際には、自分の用途にあった最適な方法を選択しましょう。
