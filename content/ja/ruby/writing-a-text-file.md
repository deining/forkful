## なぜ

プログラマーの多くはテキストファイルを作成することが日常的な作業です。テキストファイルを作成することは、コードを整理し、データを保存し、プログラムを実行するために必要です。

## 作り方

Rubyを使ってテキストファイルを作成する方法は簡単です。最初に、ファイルを作成するディレクトリに移動します。次に、Rubyのコードを使って新しいファイルを作成し、その中にテキストを書き込むことができます。最後に、ファイルを保存して閉じます。

以下の例を参考にしてみてください。

```Ruby
# 新しいファイルを作成します
file = File.new("hello.txt", "w")

# テキストを書き込みます
file.puts "こんにちは、世界!"

# ファイルを閉じます
file.close
```

上記のコードでは、"hello.txt"というファイルが作成され、その中に"こんにちは、世界!"というテキストが書き込まれます。

## 深堀り

テキストファイルを作成する際には、テキストの書き方に注意する必要があります。Rubyでは、ファイルを開くときに使用するモードを指定することができます。上記の例では、"w"というモードを使用しています。これは、新しいファイルを作成して書き込むことを意味しています。もし既存のファイルを上書きせずに追記したい場合は、"a"というモードを使用する必要があります。

また、ファイルを閉じることも重要です。ファイルを閉じることで、メモリを解放し、不要なエラーを防ぐことができます。
