## なぜ

テキストファイルを書くことの仕事。

テキストファイルはプログラミングの世界では非常に重要な役割を果たします。コードを記述したり、データを保存したりするために使用されます。しかし、なぜ私たちはテキストファイルを書いているのでしょうか？それについて考えてみましょう。

一つの理由は、テキストファイルは非常にシンプルであるということです。プログラムを実行するためには複雑なコンパイラやインタプリタが必要ですが、テキストファイルを作成するために必要なものは単なるテキストエディタだけです。そのため、プログラミング初心者でも比較的簡単にコードを記述できるようになります。

また、テキストファイルは非常に汎用性が高いという特徴もあります。どのプログラミング言語でもテキストファイルを作成することができるため、学習コストも低くなります。さまざまなプログラミング言語に対応したアプリケーションやシステムを開発する際にも、テキストファイルは重要な役割を果たします。

## 方法

では、実際にテキストファイルを書くための方法を見てみましょう。ここではRubyプログラミング言語を使用して、テキストファイルを作成する方法を説明します。

まずは、テキストファイルを新規作成するためのRubyコードを書きます。以下のように、「```Ruby```」のコードブロックを使用して、コードを記述しましょう。

```Ruby
File.new("sample.txt", "w")
```

これは、「sample.txt」という名前の新しいテキストファイルを作成するコードです。その後の「w」は、ファイルを書き込みモードで開くことを意味します。

次に、作成したテキストファイルにデータを書き込みます。例えば、以下のコードを使うことで「Hello World!」というテキストをファイルに書き込むことができます。

```Ruby
file = File.new("sample.txt", "w")
file.puts "Hello World!"
file.close
```

最後に、作成したテキストファイルを閉じるために、「file.close」を使います。これで、テキストファイルの作成は完了です。
