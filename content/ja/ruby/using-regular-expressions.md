##なぜ

正規表現を使用する理由は何でしょうか？実は、文字列のパターンマッチングとデータの検索や置換など、多くのことが正規表現によって容易に行えるようになります。特にRubyでは、正規表現がスムーズに組み込まれており、データ処理において非常に役立つツールです。

##使い方

正規表現を使用するには、Rubyの正規表現オブジェクトを作成する必要があります。以下のように、文字列の中にパターンを指定することでオブジェクトを作成することができます。

```Ruby
pattern = /Ruby/
```

次に、作成したオブジェクトを使用して文字列の中からマッチする部分を検索することができます。例えば、以下のようにサンプルの文章から"Ruby"という単語を検索することができます。

```Ruby
text = "Rubyはとても便利なプログラミング言語です。"
result = pattern.match(text)
puts result[0]       # "Ruby"と出力される
```

また、正規表現はパターンに一致する部分を置換することもできます。例えば、"Ruby"という文字列を"Python"に置換するには、以下のようにします。

```Ruby
new_text = pattern.sub("Python", text)
puts new_text        # "Pythonはとても便利なプログラミング言語です。"と出力される
```

正規表現にはさまざまなオプションや機能がありますので、詳しくは公式ドキュメントを参照してください。
