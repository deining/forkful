## なぜ

日常生活では、私たちは時刻や日付を知ることが重要です。特定のイベントやタスクの期限を追跡するため、あるいは過去のデータを分析するために、現在の日付を取得する必要があります。Rubyを使えば、簡単に現在の日付を取得することができます。

## 方法

現在の日付を取得する最も簡単な方法は、Rubyの標準ライブラリのDateクラスを使用することです。以下のようなコードを使用して、現在の日付を取得できます。

```Ruby
require 'date' # Dateクラスを使うために必要なライブラリをロードする

today = Date.today # Dateクラスのtodayメソッドを使って、現在の日付を取得する

puts today # 現在の日付を出力する
```

このコードを実行すると、何日か前の出力が表示されます。

```
2020-09-30
```

今日の日付が表示されるでしょう。また、Dateクラスには他にも多くのメソッドがあり、例えば特定の日付の曜日や月の最終日を取得することもできます。

## ディープダイブ

Dateクラスは、日付情報のみを取得するために使用されることが多いですが、DateTimeクラスを使えば日時を取得することもできます。また、strftimeメソッドを使えば、日付や日時の表示フォーマットをカスタマイズすることもできます。

```Ruby
require 'date'

now = DateTime.now # 現在の日時を取得する

puts now.strftime('%Y年%m月%d日 %H時%M分%S秒') # 日時のフォーマットをカスタマイズして出力する
```

このコードを実行すると、例えば以下のような出力が表示されます。

```
2020年09月30日 12時30分23秒
```

また、DateクラスやDateTimeクラスのドキュメントを読むことで、さらに多くの機能を発見することもできます。
