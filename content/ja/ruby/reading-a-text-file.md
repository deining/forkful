## なぜ
ファイルを読み込むことの魅力について1〜2文で説明します。

ファイルを読むことは、プログラミングの世界で非常に重要なスキルです。テキストファイルを読み込むことで、プログラムの動作を制御したり、大量のデータを処理したりすることができます。

## 方法
ファイルを読み込む方法を実際のコード例と共に、「```Ruby ... ```」のコードブロックで説明します。

まず、ファイルを開くためには `File.open()` メソッドを使用します。次に、`.read` メソッドを使ってファイルを読み込みます。最後に、`.close` メソッドを使ってファイルを閉じます。以下に、実際のコード例を示します。

```ruby
file = File.open("sample.txt") # ファイルを開く
content = file.read # ファイルを読み込む
file.close # ファイルを閉じる

puts content # ファイルの内容を出力する
```

上記のコードを実行すると、`sample.txt` ファイルの内容が出力されます。

## 深堀り
ファイルを読み込む際に、より詳細な情報を得たい場合もあります。そのような場合に役立つのが、`File` クラスの多様なメソッドです。

例えば、`.readline` メソッドを使うと、ファイルを1行ずつ読み込むことができます。また、`.readlines` メソッドを使うと、ファイル全体を配列として取得することができます。

さらに、ファイルのデータを書き換えたい場合には、`.write` メソッドを使ってファイルに書き込むことができます。

さまざまなメソッドを使いこなすことで、より複雑なファイル処理を行うことができます。
