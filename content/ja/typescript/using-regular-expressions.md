## なぜ正規表現を使うのか

正規表現は、テキストのパターンマッチングや検索に非常に役立つツールです。あなたのコードが特定の文字列を含むかどうかを確認したり、特定のフォーマットに従うかどうかを検証したりする際に役立ちます。また、大量のテキストデータから特定の情報を抽出する際にも役立ちます。

## 使い方

正規表現の基本的な構文は、以下のようになります。```TypeScript
/パターン/フラグ
```
フラグは、正規表現の動作を制御するために使用されます。よく使用されるフラグには、i (大文字と小文字を区別しない)、g (マッチするテキスト全体を検索)、およびm (複数行を検索)があります。

さて、実際に正規表現を使用してみましょう。たとえば、あなたがあるテキストの中からURLを抽出したいとします。以下のようにコードを書くことで実現できます。```TypeScript
const text = "Visit my website at https://www.example.com";
const regex = /https:\/\/[a-z]+\.[a-z]+/g;
const result = text.match(regex);
console.log(result);
```
コンソールには、```[ 'https://www.example.com' ]```という出力が得られます。ここでは、"https://"に続く文字列と、ドット（.）で区切られた2つの文字列（ドメインと拡張子）を抽出するための正規表現を使用しました。

## 深堀り

正規表現には、さまざまなパターンや特殊な文字を組み合わせることで、さらに高度な検索が可能です。たとえば、電話番号やメールアドレスなどのよく使用されるパターンを簡単にマッチングすることができます。

また、正規表現を使用する際には、データのサニタイズ（不正な入力を防ぐ）やバリデーション（正しいフォーマットかどうかをチェックする）といったセキュリティ上の観点にも注意しましょう。
