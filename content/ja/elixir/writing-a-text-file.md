## なぜ

テキストファイルを作成することのメリットは多岐にわたります。テキストファイルはプログラミングの世界において不可欠なツールであり、データの保管や共有に役立つばかりでなく、ソースコードのバージョン管理にも重要な役割を果たします。Elixir言語を学ぶ上でも、テキストファイルの作成は欠かせないスキルです。

## 作り方

テキストファイルを作成するためには、まず "File"モジュールを使用する必要があります。"File"モジュールには、ファイルを作成するための便利な関数が用意されています。以下の例では、"hello.txt"というファイルを作成し、中に "Hello, Elixir!"という文を書き込んでいます。

```elixir
File.write("hello.txt", "Hello, Elixir!")
```

上記のコードを実行すると、現在のディレクトリに "hello.txt"が作成され、その内容が "Hello, Elixir!"という文になります。また、指定したファイルが既に存在している場合は、その内容が上書きされてしまいます。ファイルを新規作成するだけでなく、既存のファイルに追記することも可能です。その場合は、"append: true"オプションを指定する必要があります。

```elixir
File.write("hello.txt", " World!", [:append])
```

上記のコードを実行すると、既存の "hello.txt"ファイルに " World!"という文が追記されます。

## 深く掘り下げる

ここまでで、テキストファイルの作成方法をご紹介しましたが、実は "File"モジュールにはさらに便利な関数が用意されています。例えば、作成したファイルを読み込んだり、削除したりすることも可能です。また、ファイルを作成する際のオプション設定や、ファイルのパーミッションを変更することもできます。これらの機能については、Elixirの公式ドキュメントを参照することでさらに詳しく学ぶことができます。
