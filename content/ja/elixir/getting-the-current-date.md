# なぜ現在の日付を取得するのか

コンピュータプログラムを書く上で、現在の日付を取得することは非常に重要です。たとえば、ユーザーの登録日や作成日を記録する際には、現在の日付を取得してデータベースに保存する必要があります。また、期限や有効期限など、特定の期間に関連する処理を行う際にも日付情報が必要になります。

# どのようにして現在の日付を取得するか

Elixirでは、```Date.utc_today```関数を使用して現在の日付を取得することができます。また、```DateTime.utc_now```関数を使用することで現在の日時を取得することもできます。

例えば、以下のようにコードを書くことで現在の日付と日時を取得することができます。

```Elixir
date = Date.utc_today
#=> ~D[2021-07-25]

datetime = DateTime.utc_now
#=> ~U[2021-07-25 12:30:00.123456Z]
```

簡単ですね！

# 深堀り

Elixirでは、日付や時刻をさまざまなフォーマットで取得することができます。例えば、```~U[2021-07-25 12:30:00.123456Z]```のようにUTC時刻で取得することもできますし、```~D[2021-07-25]```のように日付のみを取得することもできます。

また、日付と日時の間には関連性がありますが、別々に扱うことができます。日付や日時を比較する際には、```Date.compare```や```Date.compare_time```関数を使用することができます。
