## なぜ
私たちが日常的にコンピュータープログラムを作成するとき、しばしば文字列を操作する必要があります。例えば、電子メールアドレスや電話番号など、特定の規則に従う必要があるデータを抽出したい場合があります。そのような場合、サブストリングを抽出することが役立ちます。これにより、必要な情報を簡単に取り出すことができます。

## 技術方法
サブストリングを抽出する最も簡単な方法の1つは、Stringモジュールの```slice/3```関数を使用することです。この関数は、開始インデックスと終了インデックスを指定して、文字列から指定した部分を切り取ります。

```elixir
text = "こんにちは！私はエリックスです。"
String.slice(text, 6, 10) #=> "私はエ"
```

切り取りたい部分が固定されている場合は、```split/2```関数を使用するのも良い方法です。この関数は、指定した文字で文字列を分割し、分割した部分のリストを返します。また、```split/3```関数を使用すると、最大分割数を指定することもできます。

```elixir
text = "apple,banana,orange"
String.split(text, ",") #=> ["apple", "banana", "orange"]

text = "1,2,3,4"
String.split(text, ",", 2) #=> ["1", "2,3,4"]
```

正規表現を使用することもできます。```Regex```モジュールの```scan/2```関数を使用すると、正規表現に一致する部分を抽出してリストに格納することができます。

```elixir
text = "こんにちは！私はエリックスです。"
Regex.scan(~r/(\p{Hiragana}+)！/, text) #=> [["こんにちは"]]
```

## ディープダイブ
サブストリングを抽出する方法はたくさんありますが、重要なのはどの方法を使用しても目的を達成できるということです。また、文字列の操作はパフォーマンスに影響を与えることがあるため、適切な方法を選択することも重要です。さらに、正規表現を使用するときは、パターンが文字列の中に複数回出現する可能性を考慮する必要があります。
