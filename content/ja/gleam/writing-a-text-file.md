## なぜテキストファイルを書くか

テキストファイルを書くことは、プログラムとコードの基本的な概念を理解するうえで非常に重要です。テキストファイルは、プログラムのデータを保存するための基本的な方法です。また、実際にコードを実行する前に、そのコードをプレーンなテキストフォーマットで確認することができます。

## 書き方のコツ

テキストファイルを書くには、最も一般的かつ簡単な方法は、テキストエディタを使用することです。テキストエディタは、プレーンテキスト形式でファイルを作成できる、シンプルで使いやすいツールです。Gleamでは、"```Gleam ... ```"のコードブロック内にコードを記述し、実行することができます。

例えば、以下のようにリストを作成し、ファイルにデータを書き込むことができます。

```Gleam
let list = [1, 2, 3]

File.write("output.txt", list)
```

これにより、output.txtという名前のファイルに、1から3までの数字が書き込まれます。

## 詳細を調べる

テキストファイルを作成する際、注意しなければならないのが改行コードです。改行コードには、Windowsの"CR LF"やUnixの"LF"など様々な種類があり、それぞれのOSや環境によって異なります。Gleamでは、OSに依存せずファイルモードを指定することで、自動的に適切な改行コードが使用されるようになっています。

また、ファイルの読み取りや書き込みには、様々なオプションがあります。例えば、ファイルをバイナリモードで開くことや、エンコーディングを指定することができます。詳細については、公式ドキュメントを参照してください。
