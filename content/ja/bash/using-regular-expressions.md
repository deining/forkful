## Why
正規表現を使う理由は多々ありますが、最も一般的な理由はデータのパターンを探すことです。例えば、特定のフォーマットを持つ電話番号やメールアドレスを抽出したり、特定のキーワードを含む文章を検索したりすることができます。

## How To
正規表現の基本的な使い方を紹介します。まず、```Bash```のターミナルで"grep"コマンドを使い、特定のパターンを含む行を検索します。例えば、電話番号のパターンを探す場合は以下のように入力します。
```Bash
grep "[0-9]{2}-[0-9]{4}-[0-9]{4}" file.txt
```
これは"file.txt"内の行から、"XX-XXXX-XXXX"のパターンにマッチする行を抽出します。

また、正規表現をワイルドカードと組み合わせて使うこともできます。例えば、"*.txt"と入力すると、.txt形式の全てのファイルを検索することができます。

さらに、正規表現を使ってマッチした部分を置換することもできます。例えば、以下のコマンドで電話番号の"-"を削除することができます。
```Bash
sed -i 's/-//g' file.txt
```
こうすることで、file.txt内の電話番号の"-"が全て削除されます。

## Deep Dive
正規表現にはさまざまな規則やコマンドがあり、様々なパターンを検索・置換することができます。例えば、メタキャラクターと呼ばれる特殊な文字を使うことで、数字やアルファベット、特定の文字列などを一括で表すことができます。また、クラスと呼ばれるグループを作ることで、特定の文字や数字のみを抽出することもできます。

さらに、正規表現を使ってパターンを指定することで、複雑な条件を含む検索や置換を行うことができます。例えば、複数のキーワードを含む文章を検索する場合や、特定の条件を満たす行を抽出する場合などです。
