## なぜ 

読者のみなさんこんにちは！今日はBashのコマンドライン引数についてお話しします。なぜなら、コマンドライン引数を読み取ることで、より効率的にプログラミングすることができるからです。それではさっそく、コマンドライン引数を読み取る方法を見ていきましょう！

## 方法 

まずはコマンドライン引数を解釈するための基本的な構文を学びましょう。以下のようにコマンドライン引数を読み取ります。

```
Bash [オプション] [引数...] 
```

これは、Bashのコマンドライン引数を読み取るための標準的なフォーマットです。では、コマンドライン引数を含む簡単なスクリプトを書いてみましょう。

```
#!/bin/bash
# ファイル名: arguments.sh

echo "こんにちは、$1さん！私の名前は$2です。"
```

このスクリプトでは、コマンドライン引数を使用して、最初の引数が「こんにちは、」の後ろに、2つ目の引数が「さん！」の前に挿入されます。これを実行すると、以下のような出力を得ることができます。

```
$ Bash arguments.sh Taro Bash
こんにちは、Taroさん！私の名前はBashです。
```

さて、もう少し複雑なスクリプトを作ってみましょう。

```
#!/bin/bash
# ファイル名: calculator.sh

if [ $# -eq 3 ]; then #引数の数をチェック
    case "$1" in #最初の引数によって処理を分岐
        "+" ) echo "$2 + $3 = $(($2 + $3))" ;;
        "-" ) echo "$2 - $3 = $(($2 - $3))" ;;
        "*" ) echo "$2 x $3 = $(($2 * $3))" ;;
        "/" ) echo "$2 ÷ $3 = $(($2 / $3))" ;;
        * ) echo "サポートされていない演算子です。" ;;
    esac
else 
    echo "引数を3つ入力してください。"
fi
```

このスクリプトでは、最初の引数が演算子であるかどうかをチェックし、対応する計算を行います。引数を3つ指定することで、計算結果を得ることができます。

```
$ Bash calculator.sh "+" 5 3
5 + 3 = 8
```

## ディープダイブ 

では、コマンドライン引数を読み取るための裏側の仕組みを学びましょう。Bashでは、特殊変数と呼ばれる、コマンドライン引数を参照する変数が用意されています。以下がよく使用される特殊変数の例です。

| 変数 | 内容 |
|---|---|
| `$#` | コマンドライン引数の数 |
| `$?` | 最後に実行されたコマンドの終了コード |
| `$0` | スクリプトの名前 |
| `$@` | 全ての引数を含む配列 |

また、`shift`コマンドを使用することで、特殊変数の内容を前にずらすことができます。これを利用することで、引数の一部を無視して処理を行うことができます。
