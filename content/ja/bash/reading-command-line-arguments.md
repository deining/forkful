## なぜ
みなさんこんにちは！このブログ記事では、Bashでコマンドライン引数を読む方法についてお話しします。コマンドライン引数を読むことで、スクリプトをより柔軟に、そして便利に使用することができます。コマンドライン引数を読むことは、Bashをより効率的に使うための重要なスキルです。さぁ、一緒にコマンドライン引数の読み方を学びましょう！

## 使い方
コマンドライン引数を読むためには、特定の構文を使用してスクリプト内でその値にアクセスする必要があります。例えば、`$1`は最初の引数を、`$2`は二番目の引数を表します。また、特定のフラグやオプションを指定するために`$#`を使用することもできます。

以下のコード例をご覧ください。

```Bash
#!/bin/bash

# スクリプトに引数を与える
# 使用例：./script.sh apple banana cherry
# 出力：引数１：apple, 引数２：banana, 引数３：cherry

# 引数を表示するためのループ処理
for arg in "$@"
do
    # 引数を表示する
    echo "引数： $arg"
done
```

このスクリプトでは、`$@`を使用して引数をループ処理しています。引数を`"$@"`とクォーテーションで囲むことで、引数内にスペースや別の構文が含まれていた場合でも正しく処理することができます。

## ディープダイブ
コマンドライン引数の読み方には、さまざまな応用があります。引数の数を検証したり、特定の引数の値に応じて処理を分岐させることもできます。また、`getopts`関数を使用することで、オプションを指定する際のより柔軟なオプション処理を実現することができます。

さらに、引数の内部に変数を埋め込んで使用することもできます。例えば、`$1`ではなく`$1.txt`のように指定することで、引数を動的に変更することができます。

コマンドライン引数の読み方については、まだまだ詳しく語り尽くせませんが、これらの基本的な知識を身につけることで、より高度なスクリプトの作成が可能になります。
