## なぜ

コマンドライン引数を読むことは、Bashプログラミングを向上させるために非常に重要です。引数を読み取ることで、ユーザーがプログラムをより柔軟に使うことができるようになります。

## 方法

引数を読むには、`$#`という変数を使います。この変数は、入力された引数の数を示します。以下は、引数を読むための例です。

```Bash
# 変数を使って、引数を読み取る
# 引数の数を確認
echo "引数の数は、 $#" 

# 引数が1つ以上ある場合にのみ実行
if [ $# -gt 0 ]; then
    # 引数の内容を表示
    echo "最初の引数は、 $1"
    echo "2番目の引数は、 $2"
    echo "3番目の引数は、 $3"
fi
```

このコードを実行すると、引数の数と内容を表示できます。例えば、`$ bash script.sh ファイル1 ファイル2`というコマンドを実行すると、以下のような出力が得られます。

```
引数の数は、 2
最初の引数は、 ファイル1
2番目の引数は、 ファイル2
```

## 深堀り

コマンドライン引数をより詳しく学ぶには、`shift`コマンドを使うことができます。このコマンドを使うと、引数を1つずつシフトし、変数`$1`には最新の引数が、変数`$2`には2番目に新しい引数が、というように値が入ります。これを使えば、引数の数が不明な場合でもすべての引数を処理することができます。

また、特定の引数を指定して削除することもできます。例えば、`shift 2`を実行すると、最初の2つの引数が削除され、残りの引数がシフトされます。
