## なぜ
今日の日付を取得することを学ぶ理由

現代の世界では、日付や時間に関する情報を簡単に取得できるようになりました。しかし、私たちの日々の生活で日付を取り扱うときに、プログラムを使って自動的に日付を取得することがとても便利です。例えば、タイムスタンプやログファイルの作成時刻を記録するために、現在の日付の情報が必要になるかもしれません。そのため、日付を取得することは非常に重要です。

## 方法
日付を取得する方法についてのコーディング例と出力サンプルを「```Bash ... ```」コードブロック内に記載します。

### 現在の日付を取得する
以下のコマンドを使用することで、現在の日付を取得することができます。出力されるフォーマットは、デフォルトでは「年-月-日 時:分:秒」ですが、コマンドのオプションを使用することで変更することができます。

```Bash
date
```

出力例:
```Bash
2021年8月27日 15:30:00
```

オプション「+%d/%m/%Y」を使用して、出力フォーマットを日付のみに変更することができます。

```Bash
date +%d/%m/%Y
```

出力例:
```Bash
27/08/2021
```

### 秒数単位まで取得する
デフォルトでは、現在の日付は秒数のみ表示されませんが、オプション「+%s」を使用することで秒数まで取得することができます。

```Bash
date +%s
```

出力例:
```Bash
1630075800
```

この秒数は、Unixエポックからの経過秒数を表しています。

## 深堀り
日付を取得する命令「date」は、システムクロックを基準にした時刻情報を返します。そのため、システムクロックを調整することでコマンドの出力も変わってきます。また、オプションを使うことで様々なフォーマットで日付を取得することができます。
