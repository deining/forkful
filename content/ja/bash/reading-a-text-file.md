## なぜ

テキストファイルを読み込むことは、Bashプログラミングを始める上で重要なスキルです。テキストファイルには、テキストデータやコマンドのリストなど、さまざまな情報が記録されています。この記事では、テキストファイルを読み込む方法を紹介します。

## どのように

Bashでテキストファイルを読み込むには、```cat```コマンドを使用します。例えば、以下のコードを実行すると、テキストファイルの内容を表示することができます。

```Bash
cat example.txt
```

また、ファイルの行数や単語数などの統計情報を取得することもできます。例えば、以下のコードを実行すると、テキストファイルの行数を表示することができます。

```Bash
wc -l example.txt
```

さらに、テキストファイル内の特定の文字列を検索することもできます。例えば、以下のコードを実行すると、テキストファイル内で"こんにちは"という文字列が含まれている行を表示することができます。

```Bash
grep "こんにちは" example.txt
```

## ディープダイブ

テキストファイルを読み込むには、さまざまな方法があります。例えば、```sed```や```awk```といったコマンドを使用することで、ファイル内の特定のパターンやパターンに一致した行のみを表示することができます。また、テキストファイル内のデータを処理して新しいファイルを作成することも可能です。

Bashでは、様々なオプションやコマンドを使用することで、より高度なテキストファイルの読み込みができます。是非、試してみてください。
