## למה

אנשים משתמשים בקוד Kotlin עבור המשימה למצוא את אורך המחרוזת, כי המחרוזות הן מבנה יסודי בכל שפת תכנות וישנם רבים מאתגרי תכנות שדורשים את ידע על אורך מחרוזת.

## איך לעשות

הקוד הבא הוא דוגמא של כיצד למצוא את אורך המחרוזת ב-Kotlin:

```Kotlin
fun main() {
    val name = "אהובי"
    val length = name.length
    println("אורך המחרוזת של $name הוא $length.")
}
```

פלט:

```
אורך המחרוזת של אהובי הוא 4.
```

בקוד זה, אנו משתמשים בפונקציה המובנת של Kotlin בשם "length" כדי למצוא את אורך המחרוזת. ניתן להשתמש בפונקציה זו על משתנה מחרוזת או ישירות על מחרוזת.

כדי למקסם את הביצועים של הקוד ולחסוך משאבים, ניתן להשתמש במאפייני המחרוזת של Kotlin כמו אורך (length) במקום ליצור פונקציה חדשה.

## כיול

כדי להבין יותר היטב את כיצד הקוד מוצא את אורך המחרוזת, כדאי לבצע כיול למנגנונית של Kotlin.

כאשר אנו מכניסים מחרוזת לתוך משתנה, Kotlin מכירה כי זו היא מחרוזת ויודעת לפענח את הנתונים. בנוסף, Kotlin מכירה את מבנה המחרוזת - משתנה שמכיל מחרוזת הוא אובייקט של מחלקה מסוג String.

כאשר אנו משתמשים בפונקציות כגון "length", הן נגשות למאפייני המחרוזת המצוינים בתוך מחלקת String ומספקות את התשובה הרצויה - אורך המחרוזת.
