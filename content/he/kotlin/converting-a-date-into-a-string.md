## למה

למעט תרגום תאריך למחרוזת שחשוף נועם את קוראי הקוד שלך? פשוט, כמה מקרים שבהם תאריך מופיע כחלק ממחרוזת, כגון הצגת תאריך בממשק משתמש או שמירת תאריך באופן מושלם לבסיס נתונים. לכן, חשוב לדעת כיצד לבצע תהליך זה בקוד בצורה יעילה וברורה.

## איך לעשות

נתחיל ככה: תאריך נמצא בשימוש נרחב ביותר בקוד. כשמשתמשים רוצים להציג תאריך בממשק, לרוב הם ירצו לראות את התאריך בפורמט מוכר כמו "DD/MM/YYYY" או "MM/DD/YYYY". בקוד הבא, נראה כיצד להמיר תאריך למחרוזת בפורמט זה באמצעות כלי התאריך המובנה בקוטלין:

```Kotlin
val date = LocalDate.now() // תאריך נוכחי
val formattedDate = date.format(DateTimeFormatter.ofPattern("MM/dd/yyyy")) // תאריך בפורמט מתאים
println(formattedDate) // פלט: 07/15/2021
```

בקוד זה, אנו יוצרים משתנה של תאריך נוכחי עם מחלקת התאריך המובנת LocalDate. אז, אנו משתמשים בפונקציה format כדי לבצע את ההמרה למחרוזת באמצעות כלי DateTimeFormatter. כדי לקבל את הפורמט הרצוי שלנו, אנו משתמשים במבנה "MM/dd/yyyy". ולבסוף, אנו מדפיסים את התוצאה הנמצאת במשתנה formattedDate למסך.

בנוסף למבנה המתאים לתאריך, כלי DateTimeFormatter מציע גם אפשרויות רבות נוספות להתאמה וניתוח של תאריכים. נוכל להציג למשתמש את התאריך בכל פורמט שנרצה, להוסיף יחידות זמן מקומיות או להפריד את התאריך לחלקים כגון יום, חודש ושנה.
