כתיבת קוד בקודות פלגות בקוד Kotlin

## למה

הצגת פלט מנקודת הנظהון היא כלי חשוב כאשר ניתן להתחיל לכתוב קוד בקודות פלגות. היא מאפשרת לנו לבדוק ולאמת את הערכים של המשתנים בכל מהלך הקוד, מה שיכול לעזור לנו לזהות בעיות ולפתור בעיות שיכולות להתחשב בכתיבת הקוד. כמו כן, הצגת פלט גם יכולה להיות מאוד מועילה בזמן קידום ובניית הרפתקה בכתיבת קוד.

## איך לעשות את זה

בקוד Kotlin ניתן להציג פלט בשני דרכים עיקריות: באמצעות פונקציות `println()` ו- `print()` או באמצעות אופרטור התאמה `%`. בשני המקרים, ניתן להפעיל את פעולות אלה בתוך אינטרפולציה של מחרוזת. לדוגמה:

```Kotlin
val name = "דוגמה"
val age = 25
println("שלום, אני $name ואני בן $age!")
print("שלום, ")
print(name)
print(" ואני בן ")
print(age)
print("!")
println("שלום, אני ${name.toUpperCase()} ואני בן ${age + 5}!")
```

פלט:

```
שלום, אני דוגמה ואני בן 25!
שלום, דוגמה ואני בן 25!
שלום, אני דוגמה ואני בן 30!
```

אופרטור התאמה `%` מאפשר לנו לייצר מחרוזת שמכילה משתנים רבים מבלי לצרוך יותר קוד. לדוגמה:

```Kotlin
val name = "דוגמה"
val age = 25
val height = 175.5
val weight = 70.2
val details = "שלום, אני %s ואני בן %d וגביי %.1f ומשקלי: %.1f קילו"
println(details.format(name, age, height, weight))
```

פלט: `שלום, אני דוגמה ואני בן 25 וגביי 175.5 ומשקלי: 70.2 קילו`
