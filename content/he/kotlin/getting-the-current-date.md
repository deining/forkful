**מדוע:**
רק כמה משפטים קצרים המסבירים למה מישהו ירצה להשתמש בתאריך הנוכחי.

מתחברים למחשב שלנו, פותחים את המסנג'ר שלנו או כניסה לאתר כלשהו ומגלים שם את התאריך הנוכחי. התאריך תמיד כל כך חשוב עבורנו, מתחילת היום ועד לסיוםו. אנחנו עוקבים אחריו בטיחותינו, מזינים אותו במסמכים ובתכנים שונים. לכן, זוהי משימה שלא ניתן להתמודד בלעדיה כאשר מדובר בתכנות. כעת, נלמד כיצד לקבל את התאריך הנוכחי בעזרת שפת התכנות Kotlin.

**כיצד:**
נרצה להשתמש בפונקציית `now()` של `LocalDate` לקבלת התאריך הנוכחי.

```kotlin
fun main() {
    val currentDate = LocalDate.now() // משתנה שמכיל את התאריך הנוכחי
    println(currentDate)
}
```

כעת, נראה כיצד הקוד זה יתמודד עם תאריך מסוים:

```kotlin
fun main() {
    val date = LocalDate.of(2021, 7, 18) // משתנה שמכיל את התאריך שאנחנו רוצים לבדוק
    val currentDate = LocalDate.now() // משתנה שמכיל את התאריך הנוכחי
    if (currentDate == date) { // בדיקה האם התאריך הנוכחי שווה לתאריך הנתון
        println("היום הוא ה-18 ביולי, 2021!") // פלט: היום הוא ה-18 ביולי, 2021!
    } else {
        println("היום לא ה-18 ביולי, 2021.") // פלט: היום לא ה-18 ביולי, 2021.
    }
}
```

נראה כעת דרך נוספת להשתמש בתאריך הנוכחי, הפעם עם פונקציית `format()` של `DateTimeFormatter`:

```kotlin
fun main() {
    val currentDate = LocalDate.now() // משתנה שמכיל את התאריך הנוכחי
    val dateFormat = DateTimeFormatter.ofPattern("dd/MM/yyyy") // מבנה התאריך לפי חוק הארץ
    val formattedDate = currentDate.format(dateFormat) // משתנה שמכיל את התאריך הנוכחי לפי המבנה
    println(formattedDate) // פלט: 18/07/
