## למה

לבדוק אם תיקייה קיימת בשפת קוד היו יכול להיות כלי חזק ושימושי לכל מתכנת. ייתכן שתצטרכו לבדוק אם תיקייה קיימת לפני שאתם תופיעו על תוכן בעת הפעלת תוכניות מרחוק. בצורה כזאת, אתם תוכלו להמנע ממצבים בלתי רצויים כמו נסיונות ליצור תיקייה שכבר קיימת או לפתוח קבצי בזבל.

## איך לעשות זאת

בתכנית הוסיפו את הידע הזה לצורת הקוד השבועה שלכם כדי לבדוק אם תיקייה קיימת:

```Kotlin
val directory = File("/path/to/directory")
if (directory.exists()) {
    println("התיקייה קיימת")
} else {
    println("תיקייה זו איננה קיימת")
}
```
כאן אנו משתמשים במשתנה "directory" לייצג את התיקייה ומשתמשים בפעולת exists() שבודקת אם התיקייה ממשימה ביצירת התיקייה שלנו או לא. אם התיקייה קיימת, הוא ידפיס "התיקייה קיימת" ואם לא, הוא ידפיס "תיקייה זו איננה קיימת."

## חפירה עמוקה

הבדיקה המתמטית של התיקייה היא עקרונית כדי לוודא שאנו לא נתקלים בבעיות כאשר אנו מנסים ליצור או לערוך קבצים קיימים. אנו יכולים גם להשתמש בפקודת "ליל" המאפשרת לנו לבדוק אם תיקייה או קובץ קיימים לפני שאנו מנסים לבצע כל פעולה הדורשת שם קובץ או תיקייה. כמו כן, פקודת "ליל" מקלה עלינו לבדוק את התיקייה בצורה אוטומטית, כך שניתן להגנת עצמינו מפני טעויות של אנשי מתונים.
