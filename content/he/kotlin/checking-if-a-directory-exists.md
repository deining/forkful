## מדוע

למה אנשים יעסוקים בבדיקה אם תיקייה קיימת?

לפני שאנחנו מתחילים לתכנת, חשוב לוודא שהתיקייה שנרצה לגשת אליה כבר קיימת. הכוונה היא שאם נרצה לקרוא או לכתוב לתיקייה והיא לא קיימת, התוכנית תיכשל כי היא לא תוכל למצוא את התיקייה הרצויה. זה יכול להפסיק אותנו מלגשת לתיקייה, מה שישפיע על תוכניות שלנו ויגרום לתקלות בתכנות.

## איך לא

כדי לבדוק אם תיקייה קיימת, נשתמש בפונקציה בשם `exists()` במחלקת `File`. למשל, נניח שיש לנו תקייה בשם "documents" ונרצה לבדוק אם היא קיימת:

```Kotlin
val directory = File("documents")
if (directory.exists()) {
    println("התיקייה קיימת!")
} else {
    println("התיקייה לא קיימת.")
}
```
אם התיקייה קיימת, התוכנית תדפיס "התיקייה קיימת!" אחרת היא תדפיס "התיקייה לא קיימת."

ניתן גם לבדוק אם תיקייה קיימת על ידי הכנסת נתיב מלא לתיקייה:

```Kotlin
val directory = File("/Users/username/documents")
if (directory.exists()) {
    println("התיקייה קיימת!")
} else {
    println("התיקייה לא קיימת.")
}
```

במקרה זה, אם התיקייה "documents" קיימת בתקיית המשתמש שעבדנו עליה, התוכנית תדפיס "התיקייה קיימת!".
