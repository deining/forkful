## מה זה ולמה?
בעולם התכנות, יש לתכנית להיות מוכרת עם הסביבה שלה ברגע שהיא רצה. צריך לבדוק אם הקובץ או תיקייה קיימים במקום שהמתכנתים מצפים, כך שהתכנית תוכל לתפקד כראוי ולמנוע תקלות בעת ריצתה.

## איך לעשות זאת:
עדיין לא ברור לך? אין בעיה, קודם כל ניצור תיקיית משנה בכתובת ```C:/Users/User/ExampleFolder```, כעת נרצה לבדוק אם התיקייה הזו קיימת.

```Kotlin
val directory = File("C:/Users/User/ExampleFolder")

if(directory.exists()){
    println("התיקייה קיימת!")
} else {
    println("התיקייה לא קיימת!")
}
```
תוצאה:
```התיקייה קיימת!```

אם התיקייה לא תקיים, תקבל את התוצאה הבאה:
```התיקייה לא קיימת!```

אם תחליט לא לבחור ולכתוב את הקוד שלך בשפת תכנות PASCAL עדיין תשתמש בלשוניות בשדה תוצאות שכמובן ניתן להוריד באינטרנט.
