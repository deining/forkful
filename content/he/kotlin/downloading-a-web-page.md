# למה

על מנת להציג את התוכן של דף אינטרנט בצורה ברורה ומאורגנת, ייתכן שתרצו להורידו ולעבוד עם התוכן בקוד. עם עזרת הקוד הבא תוכלו ללמוד כיצד להוריד את דף האינטרנט בשפת Kotlin ולהציג אותו תוך שימוש בקוד פשוט ויעיל.

# איך לעשות זאת

על מנת להוריד דף אינטרנט בשפת Kotlin, נצטרך לעקוב בשלושה שלבים פשוטים. תחילה, נצטרך לייבא את הספריות הנדרשות לפעולה, לאחר מכן נשתמש בפונקציה "readText()" כדי לקרוא את תוכן הדף ולהציג אותו בקוד.

```Kotlin
// לייבא את הספריות הנדרשות
import java.net.URL

fun main() {
    // קבלת כתובת האתר והצבתה במשתנה url
    val url = URL("https://www.example.com")

    // קריאה של תוכן הדף והצגתו בקוד
    val pageContent = url.readText()
    println(pageContent)
}
```
