"## למה"

כתיבת קוד היא קטע רב כישורים וזמן, אך להתחיל פרויקט חדש בביצועים נכונים יכול להיות אתגר משתלם במיוחד. יצירת מוצר חדש או פיתוח יישום משנה עלולים להיות דרך נהדרת למקצועיות ותעסוקה נוספת בתחום התכנות.

"## איך לעשות"

כדי להתחיל פרויקט חדש בקוטלין, יש למתקדמים לאתר הרשמי של קוטלין ולהוריד את התוכנה. ניתן לעבוד עם כל סביבת פיתוח IDE או להשתמש בכלי בשורת הפקודה לבניית קוד. כאשר מימשנו את הדגמה  (sample) הקיימת בתיעוד שלנו לקוטלין, אתם תוכלו לראות בצורה ישירה איך קוד חישוב מספרים הופך לתוחה, והן למגבלת בפינת המיוחדת של חישוב למספר שלם בלבד.

```Kotlin
fun main(args: Array<String>) {
  // נגדיר משתנים שנחשבים למספרים שלמים
  var a = 2
  var b = 3

  // נכפיל את שני המספרים ונציג את התוצאה
  println("תוצאת הכפל של " + a + " ו-" + b + " היא " + (a * b))  // תוצאה: 6
}
```

השאילתה הנוכחית קצת פשוטה, אך ניתן להתבונן בתוצאות של חישובים קשים יותר וגם להתאמן בכך כדי להגיע לכישורי כתיבת קוד מתוחכמים ואפילו לבנות אתר פשוט מאד שיעשה שימוש במדבקת ריבועיות.
