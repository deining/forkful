## מדוע:

כל פעם שאנחנו מתחילים פרויקט חדש, יש לנו הזדמנות ללמוד דברים חדשים, לשפר את הכישורים שלנו ולהיות יותר יצירתיים. הפתיחה של פרויקט חדש היא אתגר מרתק שמשמעותו החלטה עם הרבה אפשרויות.

## איך לעשות:

ברגע שאנחנו מחליטים לפתוח פרויקט חדש בשפת קוטלין, יש לנו את הידע והכלים בכדי להתחיל. ניתן להשתמש במספר דרכים שונות על מנת ליצור פרויקט, אך הבחירה הנכונה עלולה להשפיע על האיכות וההתקדמות של הפרויקט בעתיד. להלן כמה דוגמאות של קוד ב-Kotlin ופלטים מתאימים:

```Kotlin
fun main() {
    println("שלום עולם!")
}
```
פלט:
```
שלום עולם!
```

```Kotlin
fun main() {
    var num1 = 5
    var num2 = 7
    var sum = num1 + num2
    println("הסכום של $num1 ו-$num2 הוא $sum")
}
```
פלט:
```
הסכום של 5 ו-7 הוא 12
```

## חצי הים:

כאשר אנו מתחילים פרויקט חדש, חשוב להתמקד ולתכנן כיצד הפרויקט ייראה ואילו יחידות וקהילות יהיו בו. עלינו לשים לב להערכות זמן ולאתגרים שעלולים להשפיע על הפרויקט. נוכל לבחון מה הכי חשוב ולהתמקד בזה, כמו גם להיות פתיחים ללמידה ותובנות חדשות שניתן להשתמש בהם בפרויקטים עתידיים.
