# למה

כאשר אנחנו מתחילים פרויקט חדש בתחום התכנות, המטרה העיקרית שלנו היא ליצור משהו חדש וייחודי. הכתיבה ב-Kotlin היא אחת מהדרכים הטובות ביותר ליצור פרויקט חדש שיכול להיות מגוון ומקורי ויכול לספק לנו פתרונות ייחודיים לבעיות שונות. בכתיבה הזו, אנחנו נצא מהתיבה ונוכל להוסיף יכולות חדשות לפרויקטים הקיימים או ליצור חדשים לגמרי.

## איך לעשות זאת

כדי להתחיל פרויקט חדש ב-Kotlin, נצטרך להתקין את סביבת הפיתוח שלנו וכמה כלים נחוצים. ניתן להתקין את IntelliJ IDEA, כלי הפיתוח הראשי של JetBrains, היכן שאתם עדיין עובדים עם מערכת ההפעלה שלכם.

שפת תכנות Kotlin מבוססת על Java, לכן היא משתמשת בספריות דה-פקטו של Java. כתיבת פרויקט ב-Kotlin טובה ביותר נעשית על ידי כתיבת קוד Java טהור, שנמצא בכלי הפיתוח של IntelliJ IDEA. הנה דוגמא לקוד Kotlin:

```Kotlin
fun main() {
    val name = "יון"
    println("שלום $name!")
}
```

התנאי הזה מחזיר את שם המשתמש ומדפיס אותו על המסך בצורה יפה יותר.

```
שלום יון!
```
