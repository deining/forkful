מה ולמה?
קריאת ארגומנטים משורת הפקודה הוא תהליך שבו התוכנית שלנו קוראת פרמטרים שמועברים כחלק מפתרון הפקודה. הסיבה שמתכנתים מעשים את זה היא כי זה מאפשר להם לתקשר עם התוכנית ולהעביר לה מידע חיצוני לעיבוד.

איך לעשות?
לעיל דוגמאות לתסריטים עם לפלציני קוד ותוצאת פלט דוגמא כדי להדגים את הפעולה הזו.

	```kotlin
	// הגדרת משתנה שמכיל את הארגומנטים שורת הפקודה
	val args = commandLine.arguments
	
	// פתיחת לולאה המביאה את הארגומנטים המכילים את התפריט שלך
	for (i in 0 until args.size) {
	    println("ארגומנט מספר $i: ${args[i]}")
	}
	```

Deep Dive:
קריאת ארגומנטים משורת הפקודה נוצרה כדי לאפשר למשתמשים להשמע התפריטים של התוכניות שלהם בכל זמן כאשר הם מריצים את התכנית. אם תיקחו למשל תוכנית שנועדה לפתוח תמונות, תשימו לב שהיא צריכה את הנתיב של התמונה בגדול כדי לפתוח אותה בתוך התוכנית. במקצת נוסף לזה, קריאת ארגומנטים משורת הפקודה אפשרית כדי להשתמש במצבים משנים כדי להגדיר או להתאים את התכנית שלכם בצורה ייחודית על המערכת הפעלתית שלכם.
