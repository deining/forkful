מה ולמה?
כתיבת קובץ טקסט היא הפעולה של בניית קובץ טקסט פשוט, שכולל מידע ותוכן כתוב בפורמט טקסטואלי. פעולה זו נחשבת לחשובה וחיונית עבור מתכנתי תוכנה, מאחר שהיא מאפשרת יצירת, קריאה ועידכון של מידע בקובץ בצורה פשוטה ונוחה.

איך לעשות?
לפניכם ישנם שני דוגמאות קוד שיכולות להמחיש את פעולת הכתיבה של קובץ טקסט בשפת Kotlin, כאשר התוכן שיופק בתוך הקוד קיים תחת כותרת `Kotlin...`.

אופציה 1: כתיבת טקסט פשוט לקובץ חדש
```Kotlin
val content = "זהו טקסט פשוט שיחולק לקובץ חדש."

// יוצר קובץ טקסט חדש ושומר בתוך התיקיה הנוכחית.
File("newFile.txt").writeText(content) 
```

אופציה 2: הוספת תוכן לקובץ קיים
```Kotlin
val content = "זהו טקסט נוסף שישורה ישתמשה מתחת למה שהיה נכתב מוקדם יותר."

// פתיחת קובץ קיים שמוסיף תוכן לסופו.
File("existingFile.txt").appendText(content) 
```

עודכן קובץ: `זהו טקסט נוסף שישורה ישתמשה מתחת למה שהיה נכתב מוקדם יותר.`

הוספת תוכן של טקסט אחר: `זהו טקסט נוסף שיהיה למטה לתוכן שנוסף ממש שתחתיו.`

נכתוב לתוך `existingFile.txt`.

מעומק הנושא
פעולת הכתיבה של קובץ טקסט היא חשובה מאוד בתוספת נוכחיות, מאחר שראשונה של המידע היא ניתנת לכאן בקוד מהרבה מדינ אחרים. סוג המידע הנכתב בקובץ טקסט יכול לכלול מידע טקסטואלי, כתמונות, קובץ סאונד ועוד. הדרך המקובלת לכתוב קובץ טקסט היא בעזרת שְׂפָת תכנות כמו Java או Kotlin. אך ישנם גם תוכניות עזר שיכולות ליצור, לקרוא ולעדכן קבצי טקסט, כגון Microsoft Word או Google Docs.
