## למה

למה לכתוב קובץ טקסט בכללי? כתיבת קובץ טקסט הוא פעולה חשובה ושימושית בתהליך התכנות. אם אתם מתחילים בתחום התכנות ב-Kotlin או פשוט מתחילים להשתמש בשפת תכנות זו, לכתוב קובץ טקסט פשוט היא אחת הדרכים הכי טובות להתחיל להכיר את השפה ולהתחיל להתארגן כתכניתיסט. כתיבה של קובץ טקסט יכולה לעזור לכם להתארגן את הקוד שלכם, להתרשם ממבנים שונים שלה ולהבין מנגנונים עיקריים כמו משתנים, מחרוזות ותנאים.

## כיצד לכתוב קוד Kotlin

כדי לכתוב קובץ טקסט ב-Kotlin, נצטרך להשתמש בפקודת `FileWriter` ולהעביר לה את המחרוזת הרצויה. בכדי להשתמש בפקודה זו, נצטרך לייבא את החבילה `java.io`. אפשר לראות מספר דוגמאות של קוד כדי ללמוד יותר טוב כיצד לכתוב קובץ טקסט ב-Kotlin:

```Kotlin
import java.io.FileWriter

fun main() {
  val text = "כתיבת קובץ טקסט ב-Kotlin היא כידוע פעולה חשובה ושימושית."

  val fileWriter = FileWriter("output.txt") // נוצר קובץ טקסט ריק עם השם "output.txt"
  fileWriter.write(text) // הוסף את המחרוזת לקובץ
  fileWriter.close() // סגור את חיבור הקובץ

  println("קובץ טקסט נוצר בהצלחה!")
}
```

כאשר הקוד מתבצע ומוריד את הקובץ, נקבל תוצאה כזו:

```Kotlin
כתיבת קובץ טקסט ב-Kotlin היא כידוע פעולה חשובה ושימושית.
```
