## למה
כתיבת קובץ טקסט היא חלק חיוני מתהליך התכנות בכל שפת תכנות. קובץ הטקסט מכיל את כל הקוד שהתכנת, כך שהוא חיוני לבניית התוכנה כולה. לכן, זה חשוב לדעת כיצד לכתוב ולנהל קבצי טקסט בכתיבת קוד בכדי להיות מסוגלים ליצור תוכניות מעולות בכל פעם.

## איך לעשות
תחילה, ניצור קובץ חדש שמכיל את הקוד. לדוגמה, ננסה להדפיס את הטקסט "שלום עולם" בקובץ טקסט בשפת קוטלין. נתחיל על ידי יצירת קובץ חדש ונקרא לו "hello.kt". לאחר מכן, אנו נשתמש בפקודת ההדפסה הפשוטה של קוטלין כדי להדפיס את הטקסט "שלום עולם". הנה דוגמא של הקוד בקובץ הטקסט:

```kotlin
fun main() {
    println("שלום עולם")
}
```

נשמור את הקובץ ונפעיל את הקוד כדי לבדוק את הפלט. כדי לעשות זאת, נפתח את חלון הטרמינל ונהפעיל את הפקודה הבאה:

`kotlin hello.kt`

כמידי, מכאן אתה יכול לראות את הפלט "שלום עולם" נדפס על המסך.
