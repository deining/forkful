## למה

כשאנו מתכנתים ב-Kotlin, ייתכן שנתקל במצבים שבהם נדרש למחוק תווים שמתאימים לתבנית מסוימת מהמחרוזת שלנו. זה יכול להיות משהו שכתב משתמש או אפילו תווים לא רצויים. במקרים כאלה, ייתכן שנעדיף למחוק את התווים הללו על מנת להשיג את הפלט שאנו רוצים.

## איך לעשות זאת

ראשית, נתחיל עם נקודת התחלה - מה יותר טוב מלהדגים דרך קוד? בקוד הבא, נגיע למקום שבו נמחק את התווים המתאימים לתבנית תוך שימוש בפונקציית `replace(Regex, String)`.

```Kotlin
val str = "מחיקת תווים עם תבנית"
val pattern = Regex("[zםץןנסw]")
val output = str.replace(pattern, "")
println(output)
// מחיקת תווים עם תבנית
```

כפי שאתה רואה, התווים המתאימים לתבנית המוגדרת נמחקים והפלט הסופי הוא מחרוזת ארוכה יותר עם תווים מיותרים בתוכם.

עכשיו, נדון באופן עמוק יותר על הפונקציה `replace()` ומהי Regex בדיוק.

## מה נמצא במעמד

כאמור, בניגוד לשפות תכנות אחרות, ב-Kotlin אנו משתמשים בפונקציה `replace()` על מנת להחליף תווים מסוימים בתוך מחרוזת. אולם כדי לבצע דבר כזה, אנו צריכים להשתמש בתבנית מסוימת שתייצג את התווים שנרצה למחוק.

זו המטרה של Regex - להתאים תווים לפי תבניות מסוימות. בקוד הבא, נכתוב תבנית כדי להתאים לתווים לא רצויים במחרוזת.

```Kotlin
val str = "פטר צפצוף"
val pattern = Regex("[tצף]")
val output = str.replace(pattern, "")
println(output)
// פר ווא
```
