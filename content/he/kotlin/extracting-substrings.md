# למה:
אם אתה בונה אפליקציית Android ואתה נתקל בצורך להפוך מחרוזת לקטע קטן יותר, ייתכן שתצטרך לשים לב להפיכת תת-מחרוזות בקוד שלך. למשל, אם אתה צריך לשנות כתובת אימייל כדי להציג אותו בצורה מבוזרת, אתה תשתמש בתכונת ההפיכה של תת-מחרוזות כדי לקבל רק את חלק הנדרש של הכתובת.

# איך לעשות זאת:
כדי להפוך תת-מחרוזות בקוד שלך בכוונה, יש לעקוב אחר השלבים הבאים:
1. הגדר את המחרוזת שאתה רוצה להפוך לתת-מחרוזת.
2. השתמש בפונקציית `substring()` כדי לבחור את התת-מחרוזת הנדרשת. למשל, אם אתה רוצה לקבל רק את המספרים האחרונים בכתובת האימייל שלך, אתה יכול להשתמש בכך: 
```Kotlin
val email = "example@example.com"
val domain = email.substring(8, email.length)
```
כעת משתנה `domain` יכיל את התת-מחרוזת "example.com".

3. אתה יכול להשתמש גם בקטעי קוד משתנים כדי להשתמש בתת-מחרוזת כחלק ממחרוזת גדולה. למשל, אם יש לך מחרוזת של כתובת לאתר, תוכל להשתמש בזה:
```Kotlin
val domain = "https://www.example.com"
val page = "/blog"
val finalURL = domain + page
```
כעת `finalURL` יכיל את המחרוזת המלאה "https://www.example.com/blog".
