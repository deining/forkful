## למה

יצירת מספרים אקראיים היא יישום נפוץ בתחום התכנות, במיוחד כאשר נדרשים נתונים אקראיים לצורך סימולציות או יצירת משחקים. במאמר זה, נלמד איך ליצור מספרים אקראיים בקוד Kotlin בצורה קלה ומהירה.

## איך לעשות זאת

בכדי ליצור מספרים אקראיים בקוד Kotlin, נשתמש בפונקציית `Random()` המאפשרת לנו ליצור מספרים אקראיים בצורה פשוטה ויעילה. 

```Kotlin
// לייבא את המחלקה Random
import java.util.Random

// יצירת אובייקט מסוג Random
val random = Random()

// יצירת מספר אקראי בין 0 ל-10
val randomNumber = random.nextInt(10)
```

בקוד הנ"ל, אנחנו יוצרים אובייקט מסוג `Random` ומשתמשים בפונקציה `nextInt()` כדי ליצור מספר אקראי בין 0 ל-10. ניתן לשנות את הגבולות כפי שנרצה על פי הצורך.

ניתן גם ליצור מספרים אקראיים בין מספרים מסוימים באמצעות הפונקציה `nextInt(bound: Int)` כפי שנראה בדוגמה הבאה:

```Kotlin
// יצירת מספר אקראי בין 20 ל-50
val randomNumber = random.nextInt(50 - 20) + 20
```

כאשר הפרמטר `bound` מציין את הגבול העליון והמינימלי יחד עם ההזזה של המספר באמצעות הפרמטר השני.
