## למה

כתיבת קוד זה מרתיח את הלב שלנו בדרך ייחודית, ולכן לראות תוצאות הריצה שלו הוא דבר מרגש ומעניין. הנה הבהירות בהפעלתו מבוצעת כשזה הולך ויורד: לעולם אל תאמינו בכללות בנתוני מילוטית והיא נראית כאילו מסובבת את ראשך בכל זאת (למעשה היא מסית אותו) ואז כשזה הופך אפשרי למעלה, אתה מבין שאני יכול להשתכנע לעשות גם זה (או לשנות את כתבי המילוטים שכבר זה נמצא לי בכתב היד).

## איך לעשות את זה

הקוד הבא מציג דוגמאות לקריאת קובץ טקסט בשפת קוטלין וכמה דוגמאות לתוצאות הריצה שלהן. השתמש בקבצי טקסט משלך לפי הצורך:

 ```kotlin
fun main() {
    // קורא את כל התוכן של הקובץ ומדפיס אותו
    File("myfile.txt").forEachLine { println(it) }

    // יוצר רשימה של כל השמות בקובץ
    val names = File("names.txt").readLines()
    
    // מחזיר את התוכן של השורה הראשונה בקובץ
    val firstLine = File("mytextfile.txt").readLines().first()
    
    // קודות נוספים יכולים להיות מאתגרים יותר ולדרוש שימוש בלולאות ותנאים
    // כדי לסנן את המידע הנדרש מהקובץ
}
```

הנה כמה תוצאות אפשריות של הרצת הקוד הזה על ידי כניס לקובץ טקסט עם התוכן המתאים:

```
Hello
This is a sample text file
with multiple lines
```

```
[John, Sarah, David, Rachel]
```

```
Hello
```

עכשיו אתה מוזמן לנסות זאת בעצמך עם קבצי הטקסט שלך!
