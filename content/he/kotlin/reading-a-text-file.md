**מה ולמה?**
קריאת קובץ טקסט היא פעולה המאפשרת למתכנתים לקרוא ולעבד תוכן שנשמר בתוך קובץ טקסט במחשב. זהו דרך נוחה ויעילה לקבל נתונים לתוך תוכניות מחשב.

**איך לעשות?**
למטה תוכלו למצוא דוגמאות קוד ותוצאות דוגמה המדגימות איך לקרוא קובץ טקסט בשפת קוטלין. השתמשו בבלוקי קוד ```Kotlin ... ``` כדי לראות את הדוגמאות.

```kotlin
val file = File("example.txt") // יצירת אובייקט שמכיל את הקובץ

file.forEachLine { // איטרציה על כל שורה בקובץ
    println(it) // הדפסת תוכן השורה
}
```

ניתן גם לקרוא את הנתונים בצורה יותר ממוקדת תוכן, כך שנוכל לעבד ולשלוט על התוכן של הקובץ:

```kotlin
val file = File("example.txt") // יצירת אובייקט שמכיל את הקובץ
val lines = file.readLines() // אמור להכיל את כל השורות בקובץ

for(line in lines) { // איטרציה על כל שורה בקובץ
    println(line) // הדפסת תוכן השורה
}
```

**יצירת קשת**
כבר מאוחר בשנות ה-80 וה-90, קריאת קבצי טקסט הייתה אחת מהדריכים הרגילים בסדנאות בתחום המחשוב. היא מאפשרת למתכנתים לקבל נתונים ממקור חיצוני בצורה נוחה וממוקדת, והמבנה הפשוט של קבצי טקסט מאפשר נגישות גבוהה יותר לנתונים.

ישנם גם דרכים נוספות לקרוא נתונים מקבצים, כגון כתיבת נתונים למסדי נתונים, או קריאת נתונים מאתרי אינטרנט. אך קריאת קובץ טקסט נשמעת כמעט כמו תקן לפעולות קריאה וכתיבה, ולכן נותנת גם למתכנתים המתחילים להתחיל להתנסות עם עיבוד נתונים בשפה קומפקטית כמו קוטלין.

**ראו גם**
אם אתם מעוניינים לקרוא עוד על קריאת קבצים בשפת קוטלין, ניתן להתייעץ עם המסמכים הרשמיים של Google על ספקים בנושא זה, כמו תיעוד ה-API של וואצי.
