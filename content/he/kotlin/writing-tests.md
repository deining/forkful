כתיבת טסטים בהכרחים: למה?

כנראה שאתם מתעניינים בכתיבת קוד ופיתוח בכלל. ככל שהאתרים והאפליקציות שלנו משתמשים יותר בקוד, כך גם חשיבות כתיבת טסטים גדלה ביותר. בשביל להיות מקצוען אפשרי, חשוב מאוד לדעת כיצד להכין טסטים מתאימים ואימות שם עבור כל שורה קוד שביצענו.

איך לכתוב טסטים בקוד Kotlin:

למרבה המזל, הכתיבה של טסטים בקוד Kotlin היא מאוד פשוטה. נוכל לבדוק שהקוד שכתבנו נותן את התוצאה הצפויה על-ידי שימוש במילת המפתח "assert".
```Kotlin
fun sum(x: Int, y: Int): Int {
  return x + y
}

fun main() {
  val result = sum(3, 5)
  assert(result == 8)
}
```

דוגמא זו מראה כיצד ניתן לבדוק כי הפונקציה sum שכתבנו מחשבת נכון. אם התוכנית תרוץ בהצלחה, תחזיר את התוצאה הצפויה 8. אם לא, נקבל הודעת שגיאה מפורטת ביותר שתגיד לנו מה לא בסדר עם הקוד שלנו.

טיפ מעניין נוסף הוא שאנו יכולים להשתמש במילת המפתח "assert" על ידי מעבר על מערך של ערכים ואימות שהערכים התאימו לנו באופן זהה לדוגמא הקודמת.
