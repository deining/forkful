## למה

בעולם של היום, תחום התכנות הוא חשוב ומרתק במיוחד לאנשים רבים. מתכנתים צריכים לדעת את הדרכים השונות להתמודד עם בעיות שונות כדי ליצור תוכניות איכותיות ויעילות. אחת המשימות העיקריות שהתכנתים נתקלים בהן היא השוואת שתי תאריכים. השוואת תאריכים היא משימה חשובה למדי לכל מתכנת, והיא נדרשת בכמעט כל תוכנית שמטרתה לעבוד עם תאריכים. בכתבה זו, נדבר על כיצד מתאפשר לבצע שוואה בין שני תאריכים בשפת קוטלין.

## איך לעשות זאת

עבור הדוגמה הבאה, נשתמש בפונקצייה "compareDates" שמקבלת שני משתנים מטיפוס "LocalDate" ומציגה הודעה המראה את סוג השוואה בין שני התאריכים. לנוחיות הקוראים, ניצור משתנים שהם תאריכים קנה-נפש, וכבר נניח כי הפונקצייה "print" כבר קיימת בקוד.

```Kotlin
// יצירת משתנים מטיפוס "LocalDate"
val date1 = LocalDate.of(2021, 7, 10)
val date2 = LocalDate.of(2021, 5, 25)

// פונקציית השוואה
fun compareDates(date1: LocalDate, date2: LocalDate) {
    // בדיקה האם התאריכים שווים
    if (date1 == date2) {
        print("תאריכים שווים.")
    } else {
        // בדיקה האם התאריך הראשון אחרי התאריך השני
        if (date1.isAfter(date2)) {
            print("התאריך הראשון אחרי התאריך השני.")
        } else {
            // אם כל התנאים לא מתקיימים, התאריך הראשון בהכרח לפני התאריך השני
            print("התאריך הראשון לפני התאריך השני.")
        }
    }
}

// קריאה לפונקציית "compareDates"
compareDates(date1, date2)
```
