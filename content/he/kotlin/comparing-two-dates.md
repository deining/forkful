## למה

מדוע יש לנו צורך להשוות שני תאריכים בתוך כדי תכנות בשפת Kotlin? ייתכן ותציבו את השאלה הזאת בראשכם כשתתחילו לכתוב קוד בשפה זו. אבל הייתכן ולא תדעו כיצד לעטוף את כל התהליך בתוך "פונקציות" מסובכות ולתת לקוד שלכם לעבוד בצורה אפקטיבית. לא לדאוג, במאמר הזה נלמד אתכם לעשות את זה בקלות ובפשטות.

## כיצד לעשות את זה

בואו נתחיל עם אחד הדוגמאות הכי פשוטות להשוואת שני תאריכים. נצור פונקציה שתקבל שני פרמטרים תאריך ותחזיר את התאריך הגדול יותר. בשפת Kotlin נוכל לעשות זאת באמצעות המתודה `max()` של המחלקה `LocalDate`.

```Kotlin
fun compareDates(date1: LocalDate, date2: LocalDate): LocalDate{
    return date1.max(date2)
}
```

פונקציה זו תקבל שני תאריכים ותחזיר את התאריך הגדול יותר ביניהם. נציב עכשיו שתי תאריכים כלשהם עבור הפרמטרים ונדפיס את התאריך הגדול יותר.

```Kotlin
fun main(){
    val date1 = LocalDate.of(2021, 3, 15)
    val date2 = LocalDate.of(2021, 6, 30)
    println(compareDates(date1, date2))
}
```

הפלט של קוד זה יהיה התאריך 30/6/2021 - התאריך הגדול יותר מבין השניים. ניתן גם להשתמש במתודות נוספות כדי לבצע שאר סוגי השוואות בין שני תאריכים, כגון השוואת השנים או החודשים.
