## למה

אם אתה מתעניין בתכנות ורוצה ללמוד עוד על שפת תכנות חדשה ומתקדמת, ראסט יכולה להיות בדיוק מה שאתה מחפש. כתב הקוד המודרני של ראסט מציע מגוון רחב של כלים ותכונות מתקדמות, בהם יכולת לקשר מחרוזות.

## איך לעשות זאת

תחילה, נצור פונקציה שלוקחת שתי מחרוזות כארגומנטים ומחברת אותן יחד. בראש כל בלוק קוד, נגדיר את הפונקציה עם ההגדרת "fn", ונתת את שם הפונקציה ואת הארגומנטים בעזרת סוגי הנתונים שהם יקבלו:

```rust
fn concatenate_strings(string1: &str, string2: &str) {
   // פעולות התחברות מחרוזות כאן
}
```

יכולנו ללכת ישר לכתיבת הפעולות של התחברות, אבל כדי שתהיה לנו פעולת החברת המחרוזות, נצטרך תחתית של הסוגים הנתונים של הארגומנטים. בפונקציה של המחברת, לראשונה ניצור משתנים חדשים באמצעות התחברות של כל המחרוזות המצויות בתוך הפונקציה. לאחר מכן, נחבר את המחרוזות האלה ונוסיף אותו לתוך משתנה המחרוזת החדש של משתני המחרוזות:

```rust
let new_string = format!("{} {}", string1, string2);
```

לסיום, נחזיר את המשתנה החדש של המחרוזות הכבויות בתוך הפונקציה:

```rust
return new_string;
```

וכעת נריץ את הפונקציה כדי לראות את התוצאות של ההפעולה. נכתוב את הפונקציה של החיבורים ככה:

```rust
concatenate_strings("שלום", "עולם");
```

ותוצאת התכונה תהיה:

```
שלום עולם
```
