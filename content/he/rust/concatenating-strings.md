## למה

אני יכול לחשוב על הרבה סיבות שבהן תרצו לחבר מחרוזות בפרויקט ה־Rust שלכם. למשל, אתם רוצים לייצר מחרוזת תחביר עם משתנים או להציג מתאם, ליד הקטעים של קוד שכתבתם. לחיבור המחרוזות יש תפקיד מיוחד בכתיבת קוד הנקרא "כתיבת ארגומנטים".

## איך לעשות זאת

הנה דוגמא בקוד Rust:

```Rust
fn main() {
    let name = "ישראל";
    let greeting = "שלום"; // מחרוזת מתאימה לכל מיני שפות, בלוגים ותוכניות תחביר
    let full_greeting = greeting.to_string() + ", " + name;

    println!("{}", full_greeting); // הדפסת המחרוזת המלאה
}
```

### פלט
```
שלום, ישראל
```

ניתן להשתמש במספר שיטות שונות לחיבור מחרוזות בּ־Rust. במקרה שלנו, החלטנו להשתמש בפונקציית "to_string()" על המחרוזת "greeting" שתמיד תריץ את המחרוזת הנכונה עבור כל תווי שפה שנבחר. אל תשתמש בפונקציית "to_string()" כשאתה רוצה לחבר מחרוזת שמסוגלת לנהל את המידע המורעב כמה שניתן, לכן זה משמר את רטור מודפס.

## כרטיס עומק

אולי אינך משתמש בסגווניות הרגילה של שפת תכנות, אז הנה הבהרה נוספת שתסבך את־סוגייות חיבור המחרוזות. ניתן להשתמש בפעולות שונות כדי לקלוט את התוכן מהמכשיר לאחר ביצוע מבחן השוואת הנתונים. אתם תוכלו יד שנייה של ישע בדקות האחרונות מאחר ואני משתמש ברגישות השפה האמיתית תוכן מחרוזת, נתונים פריים והרעיון החידוד בדיקותית או פרגמנטי או מקורי.
