## למה

במאמר זה נפרט על חיבור מחרוזות בשפת תכנות Rust ולמה זה חשוב עבור תכנות בשפה זו. חיבור מחרוזות הוא תהליך פשוט במבנה של השפה שמאפשר לחבר מחרוזות שונות כדי ליצור מחרוזת אחת.

## כיצד לעשות זאת

כדי לחבר מחרוזות ב-Rust ישנם מספר שיטות שאנו יכולים להשתמש בהם. הראשונה היא פעולה פשוטה בעזרת האופרטור `+` כמו שאנו יכולים לעשות בשפות אחרות. ניתן לראות דוגמאות לכך בקטע הקוד הבא:

```rust
let str1 = "שלום";
let str2 = "עולם";

let str3 = str1 + str2;

println!("המחרוזת המחוברת היא: {}", str3);
```

פלט צפוי:

```
המחרוזת המחוברת היא: שלוםעולם
```

שיטה נוספת לחיבור מחרוזות היא עם עזרת `format!` המאפשרת לנו להשתמש בתבניות כדי להציג מחרוזות בצורה מפוררת. ניתן לראות הדגמה לכך בקטע הקוד הבא:

```rust
let name = "שלום";
let age = 30;

let properties = format!("שמי הוא {} וגילי הוא {}", name, age);

println!("{}", properties);
```

פלט צפוי:

```
שמי הוא שלום וגילי הוא 30
```

## חקירה מעמיקה

בתוך קוד Rust, מחרוזות מנוהלות כמשתנים בסוגה `str` או `String`. ישנו הבדל עיקרי בין השניים והוא שמחרוזות שנמנות כלבד נמנות בסוגה `str`, בעוד שמחרוזות שנמנות ביחד מנוהגות בסוגה `String` ומאפשרות שימוש בפעולות נוספות כגון חיבור וכן גם גישה לתווים ספציפיים באמצעות אינדקסים.
