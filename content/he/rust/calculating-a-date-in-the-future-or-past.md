## מדוע

אם אתה פועל עם תאריכים בתוכניות שלך, היכולת לחשב תאריך בעתיד או בעבר עשויה להיות אפשרות שימושית. אולי אתה צריך לתכנן את התאריך של יום ההולדת שלך בשנה הבאה, או שאולי אתה צריך לחשב כמה ימים עברו מאז אירוע מסוים. בראסט קיימת יכולת מובנית לחישוב תאריך בעתיד או בעבר, ובמאמר זה אנחנו נלמד כיצד לעשות זאת בקלות.

## איך לעשות זאת

תחילה, נצטרך ליצור משתנה מסוג `DateTime` שיכיל את התאריך הנוכחי שלנו. לדוגמה, אם אנחנו רוצים לחשב תאריך בעתיד, התאריך הנוכחי יהיה הנקודת ההתחלה שלנו. כדי לעשות זאת, נשתמש בפונקציה `Utc::now()` כדי לקבל את התאריך והשעה הנוכחיים באופן אוטומטי מהמחשב שלנו.

```rust
use chrono::prelude::*;

fn main() {
  let now = Utc::now();
}
```

עכשיו שיש לנו את התאריך הנוכחי, נוכל לחשב תאריך בעתיד או בעבר על ידי הוספת או החסרת מספר ימים מהתאריך הנוכחי. לדוגמה, כדי לחשב תאריך של 100 ימים מעכשיו, נשתמש בפונקציה `num_days()` ונוסיף את הערך 100 לתאריך הנוכחי. לאחר מכן, נוכל להדפיס את התאריך החדש באמצעות פונקציית `format()`.

```rust
use chrono::prelude::*;

fn main() {
  let now = Utc::now();
  let future_date = now.num_days(100);
  println!("{}", future_date.format("%Y-%m-%d"); // תוצאה: 2021-12-27
}
```

כדי לחשב תאריך בעבר, נוכל להשתמש באותו העקרון, אך הפעם נחסר מספר ימים מתאריך הנוכחי.

```rust
use chrono::prelude::*;

fn main() {
  let now = Utc::now();
  let past_date = now - Duration::days(50);
  println!("{}", past_date.format("%Y-%m-%d"); // תוצאה: 2020-11
