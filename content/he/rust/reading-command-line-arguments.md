## למה

כשמתחילים לתכנת ב־Rust, יתכן שתרצו לשים לב לפרטים קטנים כדי ליצור קוד איכותי ויעיל. קל לתעוף מעבר לאפליקציה שבה אנחנו משתמשים ולנדוד בקוד כשמגיע לפעולות עם הפרמטרים שהשתמשנו בהם להפעיל את התוכנית. למזלנו, קודם כל צריך לפענח פרמטרים מהפקודה הזו. כך נקבל הערות חשובות כדי לפענח את הפקודה הזו בכל בקשה שתביא כונן ה-USB לחשיפת הקבצים הקיימים שבו.

## איך לעשות זאת

```Rust
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();

    println!("{:?}", args);
}
```

תוכנת Rust תשתמש בספרייה של שיטות לתכנות לעבוד עם פרמטרים בשורת הפקודה. אנחנו אוספים את כל הקלטים מהפרמטרים שיתמוך בהם תוכנת Rust. במילים אחרות, את כל הסידור הלבוש מהפלאוור הנכון (כמו שאתם עושים עם כל הפקודה var).
