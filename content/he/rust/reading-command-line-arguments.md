## מדוע

כתיבת תוכניות בשפת Rust יכול להיות מאתגר ומעניין מאוד. כשמדובר בכתיבת תוכניות למערכת הפעלה, השתמשות בפרמטרים מקו הפקודה היא דבר חשוב ונפוץ במיוחד. קריאת כתוביות הפקודה היא דרך מועילה לקבלת מידע מהמשתמש והשתמשות בו כחלק מהתוכנית. אז למה לא ללמוד כיצד לקרוא ולעבד את כתוביות הפקודה בשפת Rust? המדריך הזה יעזור לך להתחיל.

## כיצד לעשות

השתמשו בקוד הבא כדי לקרוא ולעבד את כתוביות הפקודה בשפת Rust:

```rust
use std::env; // ייבוא החבילה הנחוצה
let args: Vec<String> = env::args().collect(); // קבלת הפרמטרים מקו הפקודה
let first_arg = &args[1]; // הצגת הפרמטר הראשון (אינדקס 0 מכיל את שם התוכנית עצמה)
println!("הפרמטר הראשון: {}", first_arg); // הדפסה של הפרמטר למסך
```
כאשר אתם מריצים את התוכנית שנכתבה בעזרת הקוד הנ"ל באמצעות כתובת הקובץ שלה, תוכלו להוסיף את הפרמטרים שברצונכם להעביר כחלק מכתוביות הפקודה. לדוגמה, אם הקובץ נקרא "פרטי_אישי.rs", תוכלו להריץ אותו עם הפרמטר "אימייל" כך:

```
$ ./פרטי_אישי אימייל
```

הפלט של התוכנית יהיה:

```
הפרמטר הראשון: אימייל
```

## חפירה עמוקה

הכירו את חבילת המידע "env" ששמה המלא הוא "environment". החבילה הזאת מאפשרת למתכנתים לגשת ולהתייחס למשתנים הסביבתיים הנמצאים כעת במערכת הפעלה בו הם מריצים את התוכנית.
