## מה ולמה?
גניזת מספרים אקראיים היא תהליך בו מחשב יוצר מספרים אקראיים עם ערך קבוע או לא ידוע, ומשמש ככלי מרכזי בפיתוח תוכנה. תכנתנים משתמשים בגניזת מספרים אקראיים כדי ליצור משחקים, יישומי לוח, ואפליקציות שונות.

## איך לעשות:
תחת נתון גניזת מספרים אקראיים בשפת Rust, תוכל לגרום למחשב לייצר מספרים אקראיים עם ערך מסוים באמצעות קוד הבא:

```
use rand::Rng;

fn main() {
  let mut rng = rand::thread_rng();
  // לייצר מספר אקראי בין 1 ל-100:
  let random_num: u32 = rng.gen_range(1, 101);
  println!("המספר האקראי הוא: {}", random_num);
}
```
הפלט של הקוד הנ"ל יהיה כמו כך:
```
המספר האקראי הוא: 74
```

## חקר עמוק:
בתחילת שנות ה-50 של המאה ה-20, היו המחשבים הראשונים מחשבי פאנצ'ר, אשר הייתה קבוצת אבני קליפה נפוצה. בשנות ה-60 הוחלה הפיתוח של מחשבים דיגיטליים, וכיום כל מחשב מכיל גניזת מספרים אקראיים. פתרון נפוץ נוסף לגניזת מספרים אקראיים הוא שימוש בשעון המחשב ליצירת מספרים אקראיים.

## ראה גם:
למידע נוסף על גניזת מספרים אקראיים ב-Rust, ניתן לעיין בדוקומנטציה הרשמית של השפה (https://doc.rust-lang.org/std/rand/). כמו כן, ניתן למצוא משאבים נוספים באתר מדען (https://scientistatwork.wordpress.com/2019/03/14/random-number-generation-in-rust/).
