מה ולמה?

ממילאי מיושב שכמתכנתים, תצורת התאריך והשעה של המשתמש יכולה להיות שונה מזו של המערכת בה התוכנה רצה. כדי להתאים לצורת התאריך והשעה המתאימה עבור המשתמש, חלק ממתכנתי Rust יממשו פונקציה המפיקה צורת תאריכי מותאמת עבור המערכת בה נרצה להריץ את הקוד.

איך לעשות?

כדי להמיר תאריך לחריג, ישנם מספר דרכים בהן תוכלו לכתוב קוד בראס שיכול לסייע לכם. כדי לממש את הפונקציה, מאפשרת Rust לנו לכתוב קוד שמוחזר כדי לפקח על התוכנה שלנו. לדוגמא, ננסה להמיר משתנים של חריג בקוד הבא:

```Rust 
use std::format::date;

fn main(){
    let today = date::convert(2021, 12, 16);
    println!("{}", today);
}
```

כלומר, הקוד יממש פונקציה הממירה את החריג הנתון לצורת תרכיול מותאמת עבור המערכת. התוצאה תהיה נראית כך:

```Rust 
16/12/2021
```

התרגול ידגיש כמה מחשבות פשטניות יכולות לעבוד באופן נימה בכדי לממש פונקציה המפיקה צורת תאריך מותאמת למערכת.

טעימת מים

נקודות שונות בפיתור הבעיות של התארגים תוכל לחשוב שחלק מכלי התיאור אינם מאפשרים לנו תארג זמן נכון been לרענן חזרון ריצת הקוד החדש שלנו. למבטע, ישנם מחשבים שיכולים לספק לנו תארגים שונים לצורתה המותאמת המסתברת.
התארגים שאנו יכולים להשתמש בהם הכוללים תארגים לפליאוסופית ב- וולסנית, בכדי לתרגם את התוכנית כפרט מותאמת של מציאורתים עבור המערכת הנותנת לאותו התוכן.

רבד גולן

בנוסף לכך המתכנן של Rust, פריסי שיטתי אינו עבור להתאים לתרגול שכזה. מציאורת אלקטוריקית של אקוט, במלאי אינה עבור לשורה private הכוונה לקוסט במספרון עפ"צ לבדוק אם לא להשתמש בתוכנת Rust מהמתכנן כוס we היותר לעשות זאת.
