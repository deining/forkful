## למה

מתי מדובר על תאריך, ייתכן שתרצה להציג אותו בפורמט של מחרוזת. זה יכול להיות שימושי במקרים רבים, כגון הצגת תאריך למשתמש או ייצוגו בבסיס נתונים.

## איך לעשות זאת

התחילו על ידי הגדרת משתנה לתאריך בפורמט של `chrono::NaiveDate`. לדוגמה, נגדיר תאריך עבור היום הנוכחי:

```Rust
let today = chrono::Local::now().date();
```

לאחר מכן, תוכלו להמיר את התאריך למחרוזת באמצעות פונקציות מובנות כמו `to_string()` או `format!()`:

```Rust
// מחרוזת בפורמט ISO 8601
let today_str = today.to_string(); // Output: "2021-11-03"

// מחרוזת בפורמט מותאם אישית
let custom_format = format!("{}/{}/{}", today.day(), today.month(), today.year()); // Output: "3/11/2021"
```

## טיול לתוך המעמקים

המרת תאריך למחרוזת היא פעולה פשוטה במראה הראשוני, אך ישנם סיטואציות בהן ייתכן שתרצו לעשות זאת באופן מותאם אישית יותר. כדי לקבל ייצוג מותאם אישית לתאריך, יש להשתמש בפונקציות כגון `format!()` או `strftime()` שמפרמטות את התאריך לפי קודי תבנית מסוימים. למען הדוגמה, אם ברצונכם להציג את התאריך בפורמט "יום-חודש-שנה", תוכלו להשתמש בקוד הבא:

```Rust
let custom_date = today
    .format("%d-%m-%Y")
    .to_string(); // Output: "03-11-2021"
```

בנוסף, אם ברצונכם להציג את התאריך בשפה שונה, תוכלו להשתמש בספריית `intl` שתאפשר לכם לתרגם את התאריך לשפות שונות.
