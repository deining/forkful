## למה

כשאנו פותחים תוכניות בעזרת ראסט, חשוב לוודא שהתיקייה שאנו מנסים לגשת אליה קיימת. זה יכול למנוע באגים ותקלות בתוכנית שלנו ולהבטיח שהיא תרוץ כראוי.

## איך לבדוק אם תיקייה קיימת בעזרת ראסט

הנה דוגמא של קוד ראסט שבודק אם התיקייה "example_directory" קיימת ומדפיס תוצאה בהתאם:

```Rust
use std::fs;

if fs::metadata("example_directory").is_ok() {
    println!("The directory exists!");
} else {
    println!("The directory does not exist.");
}
```

תוצאת הרצת התוכנית תהיה:

```Rust
The directory exists!
```

## Deep Dive

כדי לבדוק אם תיקייה קיימת, אנו משתמשים בפונקציית `metadata` של המודול `fs` שמאפשרת לנו לקבל את המאפיינים של קובץ או תיקייה. אם המונח המתאים קיים, הפונקציה תחזיר ערך של סוג `Result`, שבעזרתו אנו יכולים לבדוק אם התיקייה קיימת או לא. אנו משתמשים בפונקציית `is_ok()` כדי לבדוק את ערך ה-`Result` ולהחזיר `true` אם התיקייה קיימת ו-`false` אם לא.

בנוסף, כדי לבדוק אם קיים או לא, ניתן להשתמש גם בפונקציות כגון `exists()` או `is_dir()` שבודקות את המאפיינים של התיקייה. חשוב לתרגל ולבחון את התיעוד הרשמי של ראסט כדי להבין במה כדאי להשתמש.
