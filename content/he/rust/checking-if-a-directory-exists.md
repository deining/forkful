## למה

בעולם התכנות, חשיבה ישרה היא חלק מרכזי בכתיבת קוד איכותי. כשמתכנתים, אנו תמיד חייבים לוודא שהקוד שאנו כותבים יתפקד כאשר נמשיך לפניות של משתמש משתתף. אחד הדרישות הנפוצות ביותר הוא בדיקה האם תיקייה קיימת. זה חלק חשוב בבניית שורת הבטחה ומניעת תקלות לא צפויות בתוכניות שלנו.

## איך לעשות זאת

תיקיות הן חלק חשוב מאוד במערכת הקבצים של המחשב שלנו. הם מאפשרים לנו לארגן ולנהל את הקבצים שלנו בצורה מסודרת ומאורגנת. אם אנחנו מתעניינים לבדוק אם תיקייה מסוימת קיימת במערכת הקבצים שלנו, נוכל לעשות זאת באמצעות פונקציית ה "exists" הקיימת בשפת תכנות Rust. למשל, אם נרצה לבדוק אם התיקייה "Documents" קיימת במחשב שלנו, נוכל להשתמש בקטע הקוד הבא:

```Rust
use std::fs;

if fs::exists("Documents") {
    println!("The directory exists!");
} else {
    println!("The directory does not exist.");
}
```

כאשר נריץ את הקוד הזה, נקבל את הפלט הבא:

```
The directory exists!
```

אם התיקייה לא תקיימת, נקבל את ההודעה השנייה.

## חידוד

בקטע הקוד הזה, אנחנו משתמשים בפונקציית "exists" הנמצאת במודול "fs" של ספריית הסטנדרט של Rust. הפונקציה הזו מקבלת את שם התיקייה כפרמטר ומחזירה ערך בוליאני (true / false) המציין אם התיקייה קיימת או לא. אם אנחנו רוצים לבדוק אם קיימת נתיב מלא, נוכל להשתמש בפונקציית "exists" של המודול "path" כך:
