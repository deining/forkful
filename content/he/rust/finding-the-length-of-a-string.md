## למה
במקרים רבים בתכנות נדרש לנו להכיר את אורך המחרוזת של שם נתון. במאמר זה נלמד כיצד למצוא את אורך המחרוזת בשפת ראסט.

## איך לעשות זאת

המשימה למצוא את אורך המחרוזת נראית פשוטה, אך קיימים מספר דרכים לעשות זאת בראסט. בואו נתרגל עם כמה דוגמאות בקוד כדי ללמוד את הטכניקות שונות.

```Rust
// דרך 1: שימוש בפונקציית len()
let my_str = "שלום עולם";
let len = my_str.len();
println!("אורך המחרוזת הוא: {}", len); // פלט: אורך המחרוזת הוא: 10

// דרך 2: כתיבת פונקציה משלנו
fn string_length(my_str: &str) -> usize {
    let mut count = 0;
    for _ in my_str.chars() {
        count += 1;
    }
    return count;
}

let len = string_length("שלום עולם");
println!("אורך המחרוזת הוא: {}", len); // פלט: אורך המחרוזת הוא: 10
```

כפי שאתם רואים, אנו יכולים להשתמש בפונקציה המובנית len() או לכתוב פונקציה משלנו לחישוב אורך המחרוזת. נשתמש בקודות אלו להדגים את הצורך להחזיר תוצאה מטילה של סוג נתונים בטיפוס usize.

## Deep Dive

ברקורסיה הוא תכנית שבה הפעולה מופיעה באופן שוב על עצמה. ברשת החישובים שלנו ברסט, אנחנו רואים שהפונקציה π(n) כתובת כמפנה רקורסיבית של עומק (כמפנה של n) להטיל את אורך המחרוזת.
