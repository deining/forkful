## למה?
"Capitalizing" זה לקבל מחרוזת ולעבור עליה כדי לשנות את האות הראשונה של כל מילה לאות גדולה. מדוע אנשים ירצו לעשות זאת? יתכן שזו פעולה שמתבצעת לאיזור יצירתי, למשל במתנפחתי שיעורי ביצוע - או שיכול להיות שדוברים מחפשים להציג מידע בטקסט בכירז.

## איך לעשות זאת
```Rust
fn main() {
    let text = "שלום עולם";
    let capitalized = text.to_uppercase();
    println!("מחרוזת המקורית: {}", text);
    println!("מחרוזת מוכללת: {}", capitalized);
}
```

פלט:
```
מחרוזת המקורית: שלום עולם
מחרוזת מוכללת: שלום עולם
```

איך זה עובד?
כאמור, פעולת ה-"capitalizing" משנה את האות הראשונה של כל מילה במחרוזת לאות גדולה. זה יכול להיות מאוד שימושי כאשר מנסים להציג מידע בצורה מסודרת וברורה. לדוגמה, אם אנחנו רוצים להדגיש את המילה הראשונה במשפט, או לפרק את המחרוזת למילים נפרדות.

## צלילה עמוקה
בשפת ראסט, ישנם מספר דרכים לבצע פעולת "capitalizing". הפונקציה `to_uppercase()` שנראתה בדוגמה היא דרך נוחה ופשוטה לבצע זאת, אבל ישנן גם פעולות נוספות כמו `to_title_case()` ו `to_capital_case()` שמאפשרות לשנות את האות הראשונה של כל מילה לגדולה וגם לשנות גם את האות הראשונה של כל מילה לגדולה. כמו כן, ישנן גם אפשרויות נוספות כמו לשנות רק את האות הראשונה של המחרוזת עצמה או רק את האות הראשונה של המילים הראשונות במחרוזת.
