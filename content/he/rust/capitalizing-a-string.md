## למה

ראסט הוא שפת תכנות חדשה שמתבססת על תחביר המטרה ומתאימה ליישומים רבים כמו תכנות מערכות, מסדי נתונים ועוד. אחת היתרונות של השפה הינו שהיא מאפשרת למתכנתים לנהל כמה משימות באותו זמן תוך כדי שמירה על מקור קוד נקי ונכון. אחת מהמטרות הקלות לשימוש בראסט הינה לכתוב תוכניות פשוטות ויפות שתוכל לתפוס את תשומת לב הקוראים.

## כיצד לעשות זאת

תכנות שפה רנדרינג יחידה וקוד נקי:

```
רנדרינג
    שאר
```

כתוצאה מהקוד הנ"ל יתקבל הפלט הבא:
```
משתמש
```


## חקירה עמוקה

כדי ליישם את הפעולה הנ"ל, עלינו להשתמש במתודה שנקראת capitalize() על המחרוזת שמכילה את הטקסט שאנו רוצים לכתוב. כאשר מאחדים את השתי המחרוזות, אנו מקבלים את הטקסט השני ושלם.
