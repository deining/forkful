## למה

נכון להיום, קוד המתכנתים נתקבע בכיוון הכי יעיל שאפשר בשפת תכנות, ולכן רוב המתכנתים מעדיפים לעבוד עם שפות תכנות עם ביצועים מהירים ותכונות יעילות. ראסט הוא שפת תכנות חדשה ומתוחכמת שפותחה על ידי חברת Mozilla לצורכי תיכנות מערכות מדורגות בצורה גבולית. אחת התכונות המרכזיות של ראסט היא יושרת העקרון של "אתכסון מסיבי במידת הנצח", שמאפשר למתכנתים לכתוב קוד בצורה מודולרית ונקייה. קריאת קובץ טקסט בראסט הינה דוגמה נהדרת לשימוש ביושרת המצויה בשפה זו.

## איך לעשות

התנהגות קריאת קובץ טקסט זהה במאוד לפתיחה וסגירה של קובץ מאזן של נתונים. בתחילת התהליך, נצטרך לפתוח את הקובץ דרך כימורץ `File::open()` ולכתוב אותו למשתנה של `file`:

```Rust
let file = File::open("example.txt")?;
```

מה שמעביר `let mut file` לתוך משתנה זה הינו קישור למה שקורה בקובץ. 

אחרי שפתחנו את הקובץ, ניתן לקרוא את התוכן שלו באמצעות פעולת `read_to_string()`:

```Rust
let mut contents = String::new();
file.read_to_string(&mut contents)?;
println!("כל התוכן שבקובץ:\n\n{}", contents);
```

בקוד זה, אנחנו מכוונים את ה- `file` ומכוון אותו לתוך משתנה ריק בשם `contents`. לאחר מכן, אנו מדפיסים את כל התוכן שבקובץ.
