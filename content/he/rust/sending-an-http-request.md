## למה

יש המון סיבות שאנשים מעוניינים לשלוח בקשת HTTP. ייתכן שהם רוצים לגלוש לאתר אינטרנט מסוים, לקבל מידע מהשרת או פשוט לבצע בקשת נתונים.

## איך לעשות זאת

לפניכם תמצאו דוגמאות לקוד ולפלט דוגמאות תוך שימוש בבלוקי "```Rust ... ```" כדי להדגים את תהליך של שליחת בקשת HTTP באמצעות שפת ראסט.

```Rust
use reqwest::{Error, Response};//התאמת המודול
let response = reqwest::get("https://www.example.com") //שליחת בקשה לאתר הדוגמה
    .await?; //חץ ימין למידע נוסף
println!("התקבל קוד תגובה: {}", response.status());//הדפסת קוד תגובה
```

הקוד ישלח בקשה GET לכתובת אתר שלמעלה ויחזיר את הקוד התגובה בפלט. ייתכן גם שיהיה צורך בכניסת פרטי התחברות או פרמטרים נוספים כדי לקבל תוצאות מדויקות.

## נחישות עמוקות

התהליך של שליחת בקשת HTTP באמצעות ראסט כולל מספר צעדים. תחילה, יש להתאים את המודול המתאים. לאחר מכן, יש לשלוח את הבקשה לכתובת הנדרשת ולכן לקבל תגובה מהשרת. סוג הבקשה תלוי במטרת השליחה שלה וניתן לשלוח בקשות נוספות עם פרמטרים נוספים. השימוש באובייקטים מסוגים שונים ופעולות נוספות יכול להיות שימושי על מנת להתאים את הבקשה לצרכים המיוחדים שלכם.
