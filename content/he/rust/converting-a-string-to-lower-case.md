## למה

קריאה לכתיבת בלוג על תכנות בשפת ראסט עבור קוראי עברית יכולה להיות נושא מעניין ומאתגר. גם אם אינכם מכירים את השפה כרגע, מומלץ למכור אותה ולהנות מכישורים חדשים.

ראסט היא שפת תכנות פופולרית עבור מפתחי תוכנה, ודרישה למי שיבין אותה ויעבוד בה תמיד תהיה.

אחד המטרות של תכנות בניות היא לאפשר למתכנתים לקדם את היכולת להביא כמה שיותר ערכים לכאן, לחלק לחלקי קוד ולשנות אותן לקוד אמיתי.

על פי טבלת הדירוג של 'שפות התכנונים המדוברים ביותר' , ראסט הוא אחת משפות התכנון היותר פופולריות כיום. אינני מזכיר את הטוב ביותר באופן מאומנם, ככה שכאשר תתחילו לקרוא את הבלוג הנוכחי, לא יהיה בה מזל!

## כיצד לעשות זאת

הכרזו את המכתב ששם לתוך פונקציות המקור שלכם, ע"י כניסה ל-`use std::string :: StringFuncs :: to_lowercase`. אי אפשר לעשות את זה בקופודים כמו בשפות אחרות, אבל אין הכל טוב בשפה אחת כמעט כל בת and אין שפת בודורך מבלי 

מדובר בשפה שכדאי ללמוד סימנים רוסים כיאלובפנס את פקסומה זהות לא להוסיף ערכים, אותנחת פעולות בדוקות, ומטרות לעשות את המבחנים הטובים שהדקה לקבל תרתם

``` Rust
fn main() {
    let name = String::from("JOHN");
    println!("{}", name.to_lowercase());
}
```

פלסטור - `john`
