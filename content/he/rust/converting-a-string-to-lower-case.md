## מה ולמה?
ממילה אחת למילה מייצרת התכנה שלכם, לעיתים קרובות תצטרכו להשתמש באותיות קטנות במקום אותיות גדולות. לכן, תכניות מעצבות החליטו ליצור פונקציה שתוכל להמיר מחרוזת לאותיות קטנות. זאת נקראת "ממיר מחרוזת לאותיות קטנות". דבר זה ככל הנראה לא יזיז את עולם התכנות שלכם, אבל אנחנו מבטיחים לכם שתשתמשו בו כל הזמן!

## איך לעשות:
ראשית, ניצור מחרוזת באותיות גדולות:

```Rust
let my_string = "HELLO WORLD";
```

עכשיו נשתמש בפונקציה המובנית "to_lowercase" כדי להמיר את המחרוזת לאותיות קטנות:

```Rust
let lowercase_string = my_string.to_lowercase();
```

ועכשיו נדפיס את התוצאה:

```Rust
println!("{}", lowercase_string);
```

תוצאה:

```bash
hello world
```

כעת ניתן לשחזר את התהליך וליצור מחרוזת באותיות קטנות ולהמירה לאותיות גדולות באמצעות הפונקציה "to_uppercase".

## טיול עמוק:
פונקציות להמרת מחרוזות לאותיות קטנות/גדולות קיימות כבר מאז ומעולם. במקור, הן נוספו לשפות תכנות על מנת לקשט את הקוד ולהפוך אותו לקריא יותר. אם אתם מחפשים אלטרנטיבות לפונקציה "to_lowercase", תוכלו להשתמש במגוון של ספריות חיצוניות כמו "rust-string-case", "unicode-normalization", ועוד.

הרעיון העיקרי של "to_lowercase" הוא להחזיר מחרוזת חדשה עם אותיות קטנות, במקום לשנות את המחרוזת המקורית. כך ניתן לשמור על מחרוזת המקור ולממש בצורה יעילה יותר.
