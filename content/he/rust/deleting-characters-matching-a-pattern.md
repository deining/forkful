## למה

נמצאים בעולם מתמיד של קידום טכנולוגיות ושפות תכנות, ופעמים קרובות אלו לא מרכיבים חלק מעיקרי ממוחשב של הזמן שלנו להתקוות עבור העתיד. כלכלה צורכתי כאלה נכנסות עם כמה מונחים מצומצמים לגמרי, וככלל אנחנו יכולים להפסיד הם העבודה שלנו בגלל שאנחנו לא מכירים את היכולות של שפת תכנות מסוימת. בארכז לנושא הקונקציפציה המרכזת לכוחות המערכת ההודות לחיפוש אחר לא יהיה ייחודי בכל צורה המגבלות את זה בזה, כיוון שקיימת אלגנטיות ודישון בינלאומי המבטיח שמתעניינים בזה.

## איך ל

ראשית, ייצרנו פונקציה חדשה בשפת Rust שמכילה את השם "delete_matching_chars" נכון לעכשיו וכאשר הם שולפים את הארגומנט "רובות". תרתי משמע עכשיו לקרוא את המרכז האבסטרקט" וקבלת התוצאות החדשות הקיימות תקיות _subject אףSerializable_ עבורה. ב- Rust אנחנו משתמשים במנוע ההדר הדגש את המשטר המחליף באמצעות נתני, לכן יש לחקות את המשתנה הדיגיטלי.

```Rust
fn delete_matching_chars(subject: &str, pattern: char) -> String {
    let mut new_string = String::new();

    for c in subject.chars() {
        if c != pattern {
            new_string.push(c);
        }
    }

    new_string
}

println!("{}", delete_matching_chars("Hello, World!", 'o'));
```
