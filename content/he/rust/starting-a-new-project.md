# למה

אם אתה מאתגר את עצמך כמתכנת ובטוח ביכולת שלך להשתלב בספריית שָׂפָה שעדיין לא ניסית, Rust היא הבחירה המושלמת עבורך. השפה מספקת יכולות מתקדמות של חסינות מפני שגיאות וניהול זיכרון יעיל, וכן יכולת לעשות שימוש בקוד שכתוב בִּמְסִפר ספריות ושפות שונות.

## איך לעשות זאת

כדי להתחיל פרויקט חדש בִּּשְׂפַת Rust, ישנם כמה דרכים שניתן לעשות זאת. אחת הדרכים היא להתקין את החבילה Rust ואז להשתמש בכלי פיתוח Rust, כגון Cargo, כדי ליצור פרויקט חדש ולהתחיל לכתוב קוד. ניתן להשתמש בפרקטיס (templates) כדי לקבל תבנית פרויקט שמוכנה מראש או ליצור משלך מאפס. להלן דוגמאות לכיצד ניתן ליצור פרויקט חדש ולהדפיס "היי עולם" למסך בעזרת Rust:

```Rust
fn main() {
    println!("Hello world!");
}
```

פלט:

```bash
Hello world!
```

עבור פרויקטים שיכולים להיות גדולים ומורכבים יותר, חשוב לעקוב אחר תרגילי בית ומתקדמים של Rust כדי להבין כיצד לנהל ולארגן את הקוד בצורה היעילה ביותר.
