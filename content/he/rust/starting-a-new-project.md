## למה

למה להתעסק עם פרויקט חדש בכדי ללמוד פרוגרמות ב־Rust? כי Rust הוא שפת תכנות מתקדמת שמאפשרת למפתחים ליצור קוד בצורה בטוחה ויעילה. עם Rust ניתן לכתוב קוד שמשלב בין מהירות לשמירה על הטמעת באגים כלכליים כך שפרויקטים חדשים נכתבים ברמה גבוהה יותר.

## כיצד להתחיל

בכדי להתחיל בפרויקט חדש ב־Rust יש להתקין את הגרסה האחרונה של Rust מהאתר הרשמי. כאשר התקנת התכנה מושלמת, ניתן להשתמש בכלי הקוד Rust כדי ליצור קוד חדש. הנה דוגמא פשוטה:

```Rust
fn main() {
   println!("שלום עולם!");
}
```

התוצאה של הקוד הזה יהיה "שלום עולם!" המודפס בטרמינל. מכאן ניתן להתחיל להתעמק בלמידת המילות המפתח והסכמטיקות של Rust על מנת לכתוב פרויקטים יותר מתקדמים.

## חפירה עמוקה

כאשר מתחילים פרויקט חדש ב־Rust, חשוב להבין את המבנה הבסיסי של השפה. Rust משתמשת בעקרון של "בעל המקור הקצר" כאשר ישנם רק כמה כללים יסודיים שניתן להפעיל כדי ליצור קוד אפקטיבי. בנוסף, ישנם מספר כלים נהדרים שניתן להשתמש בהם עבור ניתוח קוד, כמו Rust Compiler או Cargo.
