## למה:

הדפסת פלט תיקונים ותוכנית בעבודת קוד יכולה לעזור למפתחים לזהות בעיות ולפתורן בקלות. תוכניות שמדפיסות הודעות תיקונים יכולות לשמש ככלי מצויין לאיתור באגים ולאיתור מקומות בקוד שצריך תיקונים.

## איך לעשות:

כדי להדפיס תיקונים בראסט, מומלץ להשתמש בפונקציות `println!` או `eprintln!`. לדוגמה:

```Rust
let name = "ישראל";
let age = 73;

println!("היי, שלום {}! אתה נראה כמו אדם בן {}.", name, age);
```

פלט:

```
היי, שלום ישראל! אתה נראה כמו אדם בן 73.
```

כדי להדפיס תיקונים בקובץ, ניתן להשתמש בפונקציית `dbg!`. פונקציה זו תדפיס את הערך ואת שם המשתנה. לדוגמה:

```Rust
let x = 5;
let y = 6;

dbg!(x);
dbg!(y);
```

פלט:

```
[src/main.rs:3] x = 5
[src/main.rs:4] y = 6
```

## חקירה מתקדמת:

כדי להדפיס תיקונים מתקדמים בראסט, ניתן להשתמש במאפייני תיקונים. זהו מנגנון שמאפשר למפתחים לתאר מאפיינים לשדות במבני נתונים כגון מערכים ומפתחות של טבלאות השתחררות. ניתן להשתמש במאפיינים כדי לקבל מידע על המשתנים שלכם בזמן ריצה. לדוגמה:

```Rust
#[derive(Debug)]
struct Person {
    name: String,
    age: u8,
}

fn main(){
    let p = Person{name: "ישראל".to_string(), age: 73};
    println!("היי, שלום {:#?}!", p)
}
```

פלט:

```
היי, שלום Person {
    name: "ישראל",
    age: 73,
}!
```

בנוסף, ניתן להשתמש בפונקציית `dbg!` יחד עם מאפייני תיקונים כדי להדפיס מידע מפורט על המשתנים שלכם בזמן
