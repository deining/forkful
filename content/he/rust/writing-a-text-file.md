## למה

למה לכתוב קובץ טקסט? נתחיל עם הדבר הכי חשוב - אלגוריתם ומבנה הנתונים. כתיבת קובץ טקסט היא דרך המאפשרת לנו לשמור ולאחסן מידע בצורה מסודרת וקריאה.

## כיצד לכתוב קובץ טקסט בעזרת Rust

כתיבת קובץ טקסט בעזרת Rust היא פשוטה ויש לנו כמה אפשרויות לבחירה. נציג לכם שתי דרכים לכתוב קובץ טקסט, אחת בעזרת חבילת `std::fs` והשניה בעזרת `std::io`. תוכלו לבחור את הדרך הנוחה יותר עבורכם.

```rust
use std::fs::File;
use std::io::prelude::*;
 
fn main() {
    // דוגמה בשימוש בחבילת `std::fs` 
    let mut file = File::create("sample.txt").expect("לא הצלחנו ליצור את הקובץ");
    file.write_all(b"כתוב מילות נתון לקובץ").expect("לא הצלחנו לכתוב לקובץ");
    
    // דוגמה בשימוש בחבילת `std::io`
    let mut file2 = File::create("sample2.txt").expect("לא הצלחנו ליצור את הקובץ");
    file2.write_all(b"כתוב מילות נתון לקובץ").expect("לא הצלחנו לכתוב לקובץ");
}
```

### פלט:

- קובץ `sample.txt` יכיל את הטקסט "`כתוב מילות נתון לקובץ`".
- קובץ `sample2.txt` יכיל את הטקסט "`כתוב מילות נתון לקובץ`".

## Deep Dive

כתיבת קובץ טקסט בעזרת Rust היא בעצם כתיבת נתונים שונים לתוך קובץ מסוים. ישנם כמה פעולות שניתן לבצע על קבצים טקסט, כגון כתיבה, קריאה, מחיקה ועדכון. לדוגמא, אם נרצה לקרוא מידע מקובץ, נשתמש ב `File::open()` וב `read_to_string()`.
