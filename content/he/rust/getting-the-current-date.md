## למה

כתיבת קודים בראסט יכולה להיות אתגר ומאתגרת, אך גם מאוד משתלמת. רודינה קודה, אתם יכולים לייחס פתרונות יותר איכותיים ומהירים לכל משומשה. אחד מהתוצרים הכי נפוצים שנדרשים בכל יום להישרד ולפצות לכל צורכים שלך, כאשר אתם מפתחים קוד בראסט, הוא לקבלת התאריך הנוכחי.

## איך לעשות את זה

האם אתה יכול להשתמש במודול נשלטויי פתרונות שנעים שלא נגעו? אם כן, תוכל להשתמש במודול `chrono`. אני אגיע לπנ [deep dive] (/ # deep-dive) על `chrono` בהמשך. לעת עתה, אני אוגד את הקוד בראסט הבא:

```rust
שימוש מהיר(Bracket במודול) chrono;
שימוש מהיר(S מודול) S;
שימוש מהיר(S מודול _) S;
fn מעשיו() {
    // קבלת מידע לגבי התאריך הנוכחי
    הזמן_נוכחי() .format (סוג: "% d.% M.% y") .נכתב (& מודוקומה3שום);

    // קבלת מידע לגבי שעת הקיץ/החורף
    הזמן_החורף(דגני) .format (סוג: "% I:% M:% S% P") .נכתב (& מודוקומה3יותים);
}
```

*לֵאם הפלט שלֵאלֵו נהיה:*
