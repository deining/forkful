## למה

יכול להיות בעייה נדירה כך שאנחנו מעוניינים למצוא את התאריך הנוכחי. ייתכן שאנחנו צריכים לייצר דוח או לתעדכן אפליקציה. אין צורך לדאוג כיום, כי נוכחיות של תאריך בתוכניות הרבות.

## כיצד לעשות

משום שאנחנו מכירים Rust כעת, ננסה לגשר בין זה. ייתכן שאנחנו נאזור באתר של המדינה לחישוב יום ההוא. חשוב לזכור שהתאריך הנוכחי ייצג דבר נדיר וודא או לקבוע עם חתימה אם עם פונקציות כללית אחרונות לפיצואים כדי לסגור דוחות.

"```
Rust use chrono::prelude::*;

let now = Local::now();
// Return current date in local time
println!("Current date: {}", now.date());

// Return current date in UTC time
println!("Current date in UTC: {}", now.utc().date());

// Return current date in custom format
println!("Current date in custom format: {}", now.format("%d/%m/%Y")); 
```

מודול חדש שנותן לי כמה אפשרויות נוספות לקבלה של התאריך הנוכחי כדי להתנהג מדויק לסורג הזה. במקום להשתמש בחלון תשובות שמחזירות את התאריך הנוכחי, נוכל לשנות את התאריך שמותאם ליחס לתאריך שראה את התאריך הנוכחי.

Nוכל לשנות את ההסכון למאפיין נוסף לאטיבה מדויקת בעבודת תימוך נדיד את תאריך הנוכחי, יש לכן להיות מתאים לדיירים של חברת יופי שבמקום לקבל התאריך הנוכחי, אנחנו יאותפים לבצע זאת באופן דיווח שורחון או כחלק לכל התושבים.
