## מה ולמה?:
קבלת התאריך הנוכחי היא פעולה רגילה שתכולתה לקבלת התאריך העכשווי במחשב ולהחזירו כתוצאה. תהליך זה חשוב למתכנתים כדי להשתמש בתאריך הנוכחי בתוכניותיהם, לדוגמה ליצירת לוגים או לבצע פעולות של תאריך ושעה בתוכנית.

## כיצד לעשות זאת:
הנה דוגמאות של קוד בראשקר משתמשות בפונקציה לקבלת התאריך הנוכחי ותוצאת הריצה:

```rust
// הכנסת הספריה לניהול תאריכים
use std::time::SystemTime;

fn main() {
    // קבל את התאריך הנוכחי
    let current_date = SystemTime::now();

    // הדפס את התאריך הנוכחי
    println!("{:?}", current_date);
}
```

פלט התוכנית יהיה תאריך ושעה בפורמט של seconds: nanoseconds.

## חקירה מעמקית:
בעבר, המתכנתים השתמשו בספריה יממה לקבלת התאריך הנוכחי, אך בשפת ראשקר, ישנם מנגנונים נוספים כדי לקבל את התאריך הנוכחי. לדוגמה, ניתן להשתמש בפונקציות like `SystemTime::now()`.

כאשר משתמשים בפונקציות של תאריך ושעה, חשוב להיות נעילים מפני שינויים בזמן כדי לאבטח את התוצאות. כמו כן, כדי לקבל תוצאות מדויקות, עלינו לוודא כי הזמן הנוכחי נכון ומדויק במחשב שלנו.
