# למה
כל אחד מאיתנו נתקל במצבים כאשר נרצה להשתמש בחלק מתוך מחרוזת גדולה יותר. למשל, קל לדמיין מתי תרצו להציג את שם המשתמש של המשתמש הנוכחי המחובר באפליקציה, במקום לחשוף את המשתמש הנוכחי בכל האופן. מימוש פונקציונאליות כזו די נפוץ במתכנתי Rust ותמיד ידידותי למשתמש.

# כיצד לעשות זאת
כדי לחלץ מחרוזת החלק שאנו מעוניינים לתצוגה, נשתמש בפונקציית `slice()` בסטנדרט המובנה של Rust. הסינטקס של הפונקציה הוא `slice(start, end)` והיא מחזירה כתבת מחרוזת חדשה עם החלק הרלוונטי שנמצא בין האינדקסים `start` ו- `end`. כדי להציג את הכתבת החדשה, נפעיל את הפונקצייה `println!()` ונעביר לה את התוצאה של הפונקציה `slice()` כפרמטר.

```Rust
let original_string = "מחרוזת גדולה";
let substring = original_string.slice(2, 6);
println!("החלק המחזיק של המחרוזת הוא {}", substring);
```

כתוצאה מהקוד הנ"ל, נקבל את התוצאה הבאה:

```
הכתובת המחזקת של המחרוזת הוא רוזה
```

ניתן גם להשתמש במכניקה דומה כדי לחלץ מספר תווים מהסוף של המחרוזת, על ידי כתיבת ערכי אינדקס שליליים:

```Rust
let original_string = "מחרוזת גדולה";
let end_substring = original_string.slice(7, -1);
println!("התווים האחרונים במחרוזת הם {}", end_substring);
```

כדי לקבל את המחרוזת המלאה, נרצה להשתמש בערכי האינדקס המקבילים לגודל המחרוזת:

```Rust
let original_string = "מחרוזת גדולה";
let full_substring = original_string.slice(0, original_string.len());
println!("המחרוזת המלאה היא {}", full_substring);
```
