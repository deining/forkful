## למה

איך אתם מצפים לתת לטקסט שלכם עמודים ארוכים במייבשיים או מסמכים ארוכים שמציגים מידע ממורכז במקום אחד? אולי אתם רוצים לחלץ מידע מתוך טקסט רב עמודים בכדי להפוך אותו לנוח יותר לקריאה ולניווט. במקרה כזה, החלצת תת-מחרוזת מטקסט גדול יכולה להיות שימושית ומועילה.

## איך לעשות זאת

המחרוזת בשפת Rust כוללת כמה פעולות נוחות שמאפשרות לנו לחלץ תת-מחרוזות מהמחרוזת המקורית בצורה פשוטה וקלה. באמצעות הפונקציה `slice()` ניתן לחלץ תת-מחרוזת ממקום כלשהו בתוך המחרוזת המקורית. לדוגמה:

```Rust
let text = "טקסט לדוגמה";
let substring = &text[0..5];
println!("התת-מחרוזת היא: {}", substring);
```

פלט:

```
התת-מחרוזת היא: טקסט 
```

אם אתם רוצים לחלץ תת-מחרוזת מאמצע המחרוזת, ניתן להשתמש בנקודתיים כפל כדי לציין את המיקום. לדוגמה:

```Rust
let text = "טקסט לדוגמה";
let substring = &text[7..];
println!("התת-מחרוזת היא: {}", substring);
```

פלט:

```
התת-מחרוזת היא: לדוגמה
```

כעת, אם אנו רוצים לחלץ תת-מחרוזת עם אורך מסוים, ניתן להשתמש בפרמטר הנוסף של הפונקציה `slice()`. לדוגמה, אם אנו רוצים לחלץ תת-מחרוזת באורך 4 תווים מתוך המחרוזת המקורית, ניתן לעשות כך:

```Rust
let text = "טקסט לדוגמה";
let substring = &text[3..7];
println!("התת-מחרוזת היא: {}", substring);
```

פלט:

```
התת-מחרוזת היא: לדו
```
