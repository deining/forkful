## למה

אז למה אנו בכלל צריכים להתעסק בביטויים רגילים? בעצם, בעזרת ביטויים רגילים אנו יכולים לחפש ולקבל מידע מטקסטים שונים על פי תבניות שונות. זה מאפשר לנו לבצע פעולות כמו תיקון עיתונים וחילוץ מידע מרחבי רשת האינטרנט.

## איך לעשות

כדי להתחיל לעבוד עם ביטויים רגילים ב-Rust, נצטרך להשתמש בחבילת ה-Build-in Regex של השפה. הנה דוגמא של כיצד לכתוב ולהשתמש בביטוי רגיל בתוך קוד Rust:

```
use regex::Regex; // מייבאים את החבילה

let expression = Regex::new(r"(\d{3})-(\d{3})-(\d{4})").unwrap(); // משרשרים את הביטוי על התבנית שלו

let text = "מס' הטלפון שלי הוא 555-0123"; // מידע של טקסט עליו אנו רוצים לחפש

if expression.is_match(text) { // בודקים אם המידע מתאים לביטוי רגיל
    println!("נמצא מס' טלפון!"); // אם כן, מדפיסים הודעה
}
```

כפי שאתם רואים, אנו נתחיל בהיכרות עם הפונקציות הבסיסיות של החבילה וכיצד להשתמש בהן בתוך קוד Rust. למידע נוסף על הפונקציות והאפשרויות השונות, ניתן לעיין בתיעוד של החבילת Build-in Regex.
