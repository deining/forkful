## מדוע

ישנם כמה סיבות למה יכול להיות משתלם להשתמש בביטויים רגולריים בתכנות ב־Rust. בין היתר, כלי זה מאפשר לנו למצוא טקסטים מסוימים בתוך מחרוזות ולהחליפם, וגם ללכוד ולטפל בכניסות משתמש מסובכות.

## איך להשתמש בביטויים רגולריים ב־Rust

תחילה, נטפל בהיסטוריה: ביטויים רגולריים הם כלים מאוד נפוצים בתכנות. הם משמשים למציאת טקסטים מסוימים בתוך מחרוזות ולטיפול במחרוזות מורכבות כמו כתובות אימייל או מספרי טלפון. בהינתן הכלים העוצמתיים ב־Rust, להשתמש בביטויים רגולריים הוא פשוט ויעיל.

ב־Rust ישנם מספר דרכים ליצור ביטויים רגולריים, אך הדרך המומלצת היא להשתמש בספריית regex. הנה דוגמה לשימוש בספריית regex כדי למצוא מספרי טלפון בתוך מחרוזת:

```Rust
use regex::Regex;

fn main() {
    let phone_numbers = "המספרים שלי הם: 050-123-4567, 054-987-6543, 052-567-8901";
    let re = Regex::new(r"\d{3}-\d{3}-\d{4}").unwrap(); // פטרון למציאת מספרי טלפון

    for phone in re.captures_iter(phone_numbers) { // פונקציה שלאחר מצא מספר טלפון עושה איטרציה בסטרינג המכיל את כל המספרים
        println!("{}", phone.get(0).map_or("", |m| m.as_str())); // הדפסת המספר

        // Output:
        // 050-123-4567
        // 054-987-6543
        // 052-567-8901
    }
}
```
