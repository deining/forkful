מה ולמה?
קריאת פרמטרים של שורת פקודה היא תהליך שבו מתכנתים קוראים ומטפלים בארגומנטים שמועברים לתוכנית באמצעות הפקודה המורכבת. פקודות שורת הפקודה הן פחות נוחות לדיבוג מתוך תוכניות עם ממשק גרפי, אך הן שיטה מקובלת לדרך תכנות ושני מרכיבים חשובים יותר בחיי המתכנתים, ובמיוחד במתכנתים המתחילים, שמשתמשים בהם תדיר כדי לצפות ולנתח את פרטי התכנית.

כיצד ל:
איך לקרוא ארגומנטים של שורת פקודה ב TypeScript? נותן לכם את היכולת לקרוא ולטפל בפרמטרים האמורים לתוכנית לפני שהיא נפתחת. אתם יכולים גם להשתמש בשורת הפקודה כדי להעביר ארגומנטים לתוכנית שלכם בזמן ריצה. 

TypeScript מציע שתי דרכים לקרוא ארגומנטים של שורת הפקודה:
1. Process.argv:
זהו האובייקט הבנוי עם אתרוגן בשורת הפקודה, שמכיל מערך של ארגומנטים שהתוכנית קיבלה משורת הפקודה. ניתן להשתמש בשורת הפקודה מתוך כל קבצי TypeScript.

כדי לקרוא את ארגומנטים של שורת הפקודה ב TypeScript, ניתן להשתמש בפקודה `process.argv.slice(2);` שמחזירה מערך עם כל הארגומנטים של שורת הפקודה, ללא הפרמטרים הראשונים ששייכים לפונקציונליות של המערכת והפעלת הפקודה.

לדוגמה:
```TypeScript
const arguments = process.argv.slice(2);
console.log(arguments); // שייצן מערך של ארגומנטים במסיים הדף הראשון של שורת הפקודה
```

2. Yargs:
Yargs היא ספריית קוד פתוח שנועדה לעזור למתכנתים בקריאת ומטרת מסוימת באמצעות שורת הפקודה ב TypeScript. היא מאפשרת גם להעביר ארגומנטים משורת הפקודה באמצעות שמות הפרמטרים, ומייצרת תיעוד למערכת הפקודה לפני שהתוכנית נפתחת.

כדי להשתמש ב Yargs, צריך להתקין אותה דרך npm ולרשום את המודול בתוך הקוד.

לדוגמה:
```TypeScript
const yargs = require("yargs");
console.log(yargs.argv); // יחזיר את הארגומנטים הנתונים בתוך מערכת הפקודה
```
