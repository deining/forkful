## מדוע
קריאה של ארגומנטים מפקודת השורה יכולה להיות כלי חשוב במסגרת פיתוח תוכניות בשפת TypeScript. קוד שמאוורר מאפשר למשתמש להגדיר פרמטרים בזמן הפעלת התוכנית ולהשתמש בהם כדי להשתנות את התפקיד שלהם בהתאם לצורך.

## איך לעשות זאת
כדי לקרוא את הארגומנטים מפקודת השורה, ניתן להשתמש במודול המובנה בשם `process` בתוך Node.js. בקוד הבא ננסה להדגים איך לכתוב תכנית ב-TypeScript שקוראת את הארגומנטים הנמסרים מפקודת השורה:
```TypeScript
// מובילי השורה הבאים יתבצעו בתוך פקודת השורה
// נניח שיש לנו תכנית ראשונית המקבלת את שם המשתמש כארגומנט
const username = process.argv[2];
// נדפיס למסך את המשתמש שתועבר אלינו מפקודת השורה
console.log("שלום " + username);
```
כעת בעת ריצת התכנית נניח שנקבל את הקלט הבא בפקודת השורה:
```
node greet.ts John
```
הפלט שיחולם במסך יהיה:
```
שלום John
```

## חקירה עמוקה
קריאה של ארגומנטים מפקודת השורה יכולה להיות אפילו יותר מורכבת מאשר הדוגמא הפשוטה שצירפנו כאן. ניתן להשתמש בספרייה יעילה כמו `yargs` כדי לנתח ולנהל ארגומנטים מסובכים ביותר. בנוסף, ניתן להשתמש בספרייה `dotenv` כדי לאחסן את הארגומנטים במשתנים סביבתיים במקובל לצורך שימוש בהם בכל חלקי התוכנית.
