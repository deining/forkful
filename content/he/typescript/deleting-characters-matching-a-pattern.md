## למה:

כדי לשפר את פונקציונאליות התוכנית ולייצר קוד יותר נקי ויעיל, ייתכן שנרצה למחוק תווים כלשהם המתאימים לתבנית מסוימת. זה יכול להיות יעיל במיוחד כאשר מעורבים מחרוזות גדולות עם תווים מיותרים או תווים שגויים.

## איך לעשות זאת:

כדי למחוק תווים מתאימים לתבנית, נשתמש בפונקציות המובנות של TypeScript כדי למצוא ולמחוק את התווים הרצויים. למשל, נשתמש בפונקציית `replace()` שתחליף תווים מתאימים לתבנית עם תווים ריקים. הנה דוגמה פשוטה:

```TypeScript
const str = "Hello there! 123";
console.log(str.replace(/[^\w\s]/g, ""));
// Output: "Hello there 123"
```

כאן, אנו משתמשים בתבנית מסוימת כדי למצוא תווים שאינם אותיות או מספרים ולמחוק אותם מהמחרוזת. ניתן להתאים את התבנית כרצוננו כדי למחוק תווים נוספים או כדי להשאיר תווים מסוימים בשמירה.

## לעבור לעומק:

עם TypeScript, ניתן להשתמש גם בפונקציות נוספות כדי למחוק תווים מתאימים לתבנית מסוימת. למשל, ניתן להשתמש במתודת `split()` כדי לפצל את המחרוזת למערך, למחוק את התווים הרצויים מכל איבר במערך, ולאחר מכן למזג את האיברים חזרה למחרוזת כשתווים נוספים הוסרו. נה דוגמה עם `split()`:

```TypeScript
const str = "Hello there! 123";
console.log(str.split("").filter(c => c.charCodeAt(0) < 97).join(""));
// Output: "123"
```
