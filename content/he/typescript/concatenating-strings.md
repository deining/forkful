## למה:

כתיבת קוד בטיפוסקריפט היא דרך נהדרת ליצור אפליקציות דינמיות ולייצר תכנים מרתקים ופונקציונליים. לחיבור מחרוזות הוא כלי חשוב שניתן להשתמש בו כדי להכניס מידע ותוכן נוסף לאפליקציה שלך. לכן, זה חיוני ללמוד כיצד לחבר מחרוזות בפונקציות לתמיכה נוספת.

## כיצד לעשות זאת:

תחיבור מחרוזות בפונקציות בטיפוסקריפט באמצעות מנגנון שנקרא "חיבור מחרוזות". עם החיבור מחרוזות, אתה יכול לרכיב מחרוזות שונות יחד כדי ליצור מחרוזת חדשה. כאשר אתה חובר מחרוזות כך, הן יכולות להיות מוצגות כמחרוזות או כמספרים.

תוכל להשתמש בחיבור מחרוזות בפונקציות כדי להציג מידע דינמי, לבנות כותרות מעניינות ולשלב מידע מתוך קובץ נתונים. הנה כמה דוגמאות לכיצד לחבר מחרוזות בטיפוסקריפט:

```TypeScript
// חיבור מחרוזות עם פונקציות
function fullName(firstName: string, lastName: string): string {
  return firstName + " " + lastName;
}

console.log(fullName("דני", "אברם")); // פלוני פיוטר

// חיבור מחרוזות עם מספרים
let num1: number = 5;
let num2: number = 10;
console.log("סכום המספרים הוא: " + (num1 + num2)); // סכום המספרים הוא: 15

// חיבור מחרוזות עם מחרוזת אחת
let message: string = "שלום לך";
console.log("הורדתי 5 שקל בך: " + message); // הורדתי 5 שקל בך: שלום לך
```
