## למה
מדוע יש צורך לחשוב על החצנים בקוד? התשובה פשוטה - החצנים הם אחד הערכים החשובים ביותר של כל מחרוזת טקסט. הם מאפשרים לנו לגשת לחלקים נפרדים בתוך מחרוזת ולעבד אותם באופן נפרד. במאמר זה, אנחנו נדבר על אופן הפעולה של אקסטרקטינג (חילוץ) של חצנים בקוד TypeScript ונראה שימושים נפוצים ומיומנים למטרה זו.

## איך
הנה דוגמא לפעולת החילוץ באמצעות קוד TypeScript:

```TypeScript
// ייבוא ספריית החילוץ ב TypeScript
import { extract } from 'substring';

// תיעוד מחרוזת הראשונית
const originalString: string = "שלום לך חברים!";

// הוצאת חצן מצירוף של המחרוזת הראשונית
const extractedSubstring: string = extract(originalString, 6, 2);

// פלט: "לך"
console.log(extractedSubstring);
```

בדוגמא זו, אנחנו משתמשים בספרייה כדי להחליף את מחרוזת הראשונית ולסמן את החצן שאנחנו רוצים לחצוץ. במקרה זה, אנחנו מחליצים חצן בתחילת המחרוזת וחיילנו על החצן שלוש תווים.
