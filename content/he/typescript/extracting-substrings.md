## למה

למה לעסוק בחילוץ חתיכות מחרוזות? ניתן להשתמש בפעולה זו לעיבוד וטיפול במחרוזות בצורה מדויקת יותר, כך שניתן לפענח ולהשתמש במידע שמופיע בתוכן המחרוזת.

## איך לעשות זאת

תחילה, עלינו להגדיר מחרוזת מתאימה ולאחר מכן להשתמש בפעולת ה-slice לחילוץ חלק מהמחרוזת. לדוגמה:

```typescript
let str = "ברוכים הבאים לבלוג שלי!";
let substring = str.slice(9); // חילוץ התת-מחרוזת מהתו התשיעי והלאה

console.log(substring); // ידגלים לבלוג שלי!
```

ניתן גם לחלק את המחרוזת לחלקים על פי מיקום התו התחלתי והסופי, כך שניתן לחלץ רק את החלק הרצוי מהמחרוזת. לדוגמה:

```typescript
let str = "ברוכים הבאים לבלוג שלי!";
let substring = str.slice(0, 9); // חילוץ התת-מחרוזת מהתו הראשון ועד התו השמיני

console.log(substring); // ברוכים הב
```

בנוסף, ניתן להשתמש בפעולת ה-substring לחילוץ תת-מחרוזת בדיוק כמו ב-slice. ההבדל העיקרי בין הפעולות הוא באופן שבו הן מתייחסות לאינדקסים של התווים המקוריים. נסו לבדוק את הפלט של הקודים הבאים:

```typescript
let str = "ברוכים הבאים לבלוג שלי!";
let sliceSubstring = str.slice(0, 9); // חילוץ התת-מחרוזת

let substring = str.substring(9, 15); // חילוץ התת-מחרוזת

console.log(sliceSubstring); // ברוכים הב
console.log(substring); // ידגל
```
