## למה

כשמתחילים ללמוד תכנות בטווח רחב של שפות תכנות, אחת הדברים הראשונים שנלמד הוא איך לקבל את אורך המחרוזת (string) הנתונה. אורך המחרוזת הוא כמות התווים, מרכיבי הנתונים או תווים בשפת התכנות. מציאת האורך של מחרוזת הוא נחשב כבסיסי וחשוב להבנת השפה ולבניית קוד תקין.

## איך לעשות זאת

תחילה, ניצור משתנה חדש בשם "text" ונשמיט לו את המחרוזת המיועדת לבדיקה. לאחר מכן, נשתמש בפעולת "length" כדי לקבל את אורך המחרוזת הנתונה. לדוגמה:

```TypeScript
let text = "מחרוזת לבדיקה";
console.log(text.length); // הפלט יהיה 14
```

ניתן גם להשתמש בלולאת "for" לעבור על כל תו במחרוזת ולמנוע אורך כולל על ידי הוספת 1 לכל סיבוב. לדוגמה:

```TypeScript
let text = "מחרוזת לבדיקה";
let length = 0;

for (let i = 0; i < text.length; i++) {
  length++;
}

console.log(length); // הפלט יהיה 14
```

## התעמקות נוספת

כאשר משתמשים בפעולת "length" על מחרוזת, היא מחזירה את אורך התווים שבמחרוזת בצורה תלת פעמית, ומתחשבת גם בתווים כמו רווחים, סימני פרידה וכו'. בנוסף, הפעולה תחזיר את התוצאה שלא התשובה אחריות על כל נושא, אלא רק על הבדיקה של האורך. כמו כן, חשוב לזכור שביצירת מופעים חדשים של מחרוזות, אורך המחרוזת יגיע לאפס ולכן יהיה חשוב לקבוע את האורך מחדש במידת הצורך.
