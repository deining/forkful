## למה
הורדת עמודי אינטרנט היא כלי חשוב בתכנות ובפיתוח אתרים. באמצעות הורדת עמודים, ניתן ליצור אפליקציות שונות כגון ממירי מטבע, צפייה במבזקי חדשות מרחבי העולם ועוד.

## איך לעשות זאת
לאחרי שהתקנתם את חבילת הקוד TypeScript וכלי הניהול שלו, מומלץ לעקוב אחר החיבורים הבאים בכדי להתחיל לשלוט על הורדת עמודים באמצעות כלי הקוד הזה.
```TypeScript
import https from 'https';

const url = 'https://www.example.com';
https.get(url, (response) => {
  response.on('data', (chunk) => {
    console.log(chunk.toString());
  });
});
```

קוד זה משתמש במודול "https" שמלווה בקוד TypeScript ומתאים אותו באמצעות אופציות בדרך שלה החבירה הישירה. כאן יש הזמנת פונקציות האירועים של תגובה וקטע נתונים המוגדר כמחזור הדפדפן שמדמה נתונים בעת הטעינה.
