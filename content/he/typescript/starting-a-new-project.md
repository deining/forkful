# מה ולמה?
פתיחת פרויקט חדש היא תהליך בו מתבצעת תכנון וכתיבת קוד חדש ליישום חדש או פרויקט חדש של אותו יישום. מתכנתים מתחילים פרויקטים חדשים כדי לשפר את היצירתיות ולהתאים לשטח העבודה שלהם.

# איך לעשות?
תחת ניתוח קודי ותוצאות דוגמא בתוך בלוקי קוד ```TypeScript ... ```.

```TypeScript
// יצירת קובץ חדש
touch newfile.ts

// יצירת תיקיה חדשה
mkdir newfolder

// הוספת קבצי TypeScript לפרויקט קיים
tsc file1.ts file2.ts
```

# חצי הדרך
מומלץ להשתמש בכלים נוספים לניהול פרויקטים כמו TypeScript-CLI או יישומי ניהול קוד מקובלים כמו Git בכדי לשפר את ארגונו של פרויקט חדש.

## חלופות
ישנן כמה חלופות לפתיחת פרויקטים חדשים כמו להשתמש בתבניות פרויקט קיימות או לשתף את המידע עם ידידים וקבוצות מטרה עבור פרויקט משותף.

## פירוט נרחב
פתיחת פרויקט חדש עשויה לכלול תהליך של תכנון מסודר, התאמת כתיבת הקוד לתלות וכיוונון של ספקים חיצוניים וערכי מיקום שונים שלפרויקט יכול להיות כלול כשתוכניות קיימות או דפוס מחירים כלליים. בכדי להימנע מצעד זה לחלוטין וליצור פרויקטים חדשים בקלות ניתן להשתמש בתבניות פרויקט קיימות וכלים נוספים עבור יוצרי קוד.

# ראה גם
למידע נוסף על פתיחת פרויקטים חדשים, ניתן לקרוא את המדריך הרשמי של TypeScript: https://www.typescriptlang.org/docs/handbook/project-setup.html
