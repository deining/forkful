## למה

משך התכנות: ובינתיים המטרה היא להיות יעילים ולכן אנו רוצים לכתוב קוד בצורה נקייה ומובנת, שכן זה יקל על מפתחים נוספים לקרוא, לפתח ולתחזק את הקוד שלנו בעתיד. דוגמה לכך יכולה להיות משימה פשוטה מאוד - לכתוב קוד שבו מחלצים את התווים הראשונים מכל מילה במחרוזת ומכוונים את האותיות הראשונות שלהן. זה ישפר את הקריאות והבנה של הקוד עבורנו ועבור מפתחים אחרים שייעבדו עם הקוד בעתיד.

## כיצד לעשות זאת

תוכנית זאת הינה בסך הכל פעולה מצומצמת, אבל יעלה לנו כמה קווים עודף שיחוללו קוד לא נקי ומפולט. כדי לבצע אותה, ישנם מספר דרכים אפשריות, אבל אנו נבחר כאן להשתמש בפונקציית "map" הקיימת בשפת TypeScript.

הנה דוגמא פשוטה של משימה זו בצורה כללית, כאשר רוצים לחלץ את האותיות הראשונות של המילים במחרוזת:

```TypeScript
let inputString: string = "Hello World";
let result: string = inputString
  .split(' ')
  .map(word => word[0]) //מחלץ את האות הראשונה של כל מילה
  .join(''); //מכוון את האותיות הראשונות בסדר כלשהו

console.log(result); //תצוגת תוצאה: HW
```

שימו לב שהקוד הנחוץ לביצוע המשימה לא ארוך במיוחד: נחתוך את המחרוזת למערך מילים, וממירים אותו למערך חדש של אותיות הראשונות של המילים, ואת המערך החדש הופך למחרוזת. תוצאת הלוג שלנו היא המחרוזת "HW", עם האותיות הראשונות של כל המילים מחוברות לאות אחת.
