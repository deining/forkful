## למה
אחת המטרות העיקריות של להשתמש בתכנות TypeScript היא להפוך מחרוזת לתאריך. זה רלוונטי למגוון רחב של פרויקטים כמו מערכות לניהול אירועים, יישומי שליחת הודעות ועוד.

## איך לעשות זאת
עבור חלק זה נניח שיש לנו תאריך מסוים בקוד שלנו ואנו רוצים להפוך אותו למחרוזת מסוג מסוים. לדוגמה, נניח שיש לנו את התאריך הבא:
```TypeScript
const date = new Date("2021-10-31");
```

### המרת תאריך למחרוזת בפורמט שנרצה
נשתמש בפונקציית `toLocaleDateString()` כדי להמיר את התאריך למחרוזת בפורמט שנרצה. בפונקציה זו, נעביר שני מספרים למיון את השנה והחודש (יחד או נפרד) במחרוזת.

```TypeScript
const date = new Date("2021-10-31");
const dateString = date.toLocaleDateString("he-IL", {
  year: "numeric",
  month: "long",
  day: "numeric",
});
console.log(dateString); // 31 באוקטובר 2021
```

### המרת תאריך למחרוזת בפורמט מותאם אישית
אם רוצים להגדיר את הפורמט של המחרוזת לחלוטין, נשתמש בפונקציית `format()` ונעביר לה את הפורמט המבוקש כפרמטר. לדוגמה, אם רוצים להציג את התאריך במבנה "יום חודש שנה" נעביר לפונקציה את הפורמט הבא: "d MMMM yyyy".

```TypeScript
const date = new Date("2021-10-31");
const dateString = date.format("d MMMM yyyy");
console.log(dateString); // 31 October 2021
```
