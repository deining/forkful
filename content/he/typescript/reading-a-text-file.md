## Why

רבים מתכנתים מתעניינים בכתיבת קוד ויש להם משימה שדורשת קריאת קובץ טקסט. קריאת הקובץ הופכת לכתיבה באמצעות קוד ב-TypeScript ולכן, כדי להיות כמו קבלן תוכנה מנוסה, חשוב לדעת איך לקרוא ולעבוד עם קבצי טקסט.

## How To

כדי לקרוא קובץ טקסט ב-TypeScript, ניתן להשתמש בפונקציה `readFileSync()` מהמודול "fs" של נוד. הפונקציה מקבלת שני פרמטרים - שם הקובץ וקוד ההפעלה המציין את אופן הקריאה (למשל "utf8" עבור קריאה כקובץ טקסט). ניתן להשתמש במשתנה כדי לאחסן את תוכן הקובץ ולעבוד איתו.

```TypeScript
import * as fs from "fs";

let fileContent = fs.readFileSync("text.txt", "utf8");
console.log(fileContent);
```

כדי לקרוא קובץ טקסט בשיטה אסינכרונית, ניתן להשתמש בפונקציה `readFile()` שמקבלת את אותם פרמטרים כמו `readFileSync()` אבל עם פונקצית קריאה כמחזור חזרה.

```TypeScript
import * as fs from "fs";

fs.readFile("text.txt", "utf8", (err, data) => {
    if (err) throw err;
    console.log(data);
});
```

## Deep Dive

קביעת הקוד המציין את אופן הקריאה היא חשוב מאוד כי במידה והקובץ לא בפורמט סטנדרטי, הפלט שנקבל יהיה שונה בהתאם. לדוגמה, אם נקבל קובץ טקסט עם תווים עבריים כפולים, הפלט יופיע כתווים מוזרים. כדי לפתור את זה, ניתן להשתמש בפונקציה `iconv` ממודול "iconv-lite" שמאפשרת להמיר את הקודים לקוד נתמך בקובץ טקסט.
