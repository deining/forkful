## למה

תאר לעצמך מצב שבו אתה מבקש להציג למשתמש באתר או מערכת המתנה את התאריך הנוכחי. האם אתה יכול לעשות זאת בלי להשתמש בשעון או לסמן תאריך וזמן ידנית? אם אתה מקבל את התאריך של מחשב המשתמש המקומי, יתכן שהוא לא יהיה מדוייק או יתבצע טעות אם הוא ישנה את הזמן שלו. לכן, קבלת התאריך הנוכחי מהשרת הינה אספקט חשוב כדי לוודא שהתאריך והזמן המוצגים הם מדוייקים.

## איך לעשות זאת

נתחיל עם ההגדרה של המשתנה שמכיל את התאריך הנוכחי בצורה של משתנה תאריך עם ערך מתאים. לאחר מכן, נשתמש בפונקציית JavaScript המשולבת Date() כדי לקבל את התאריך הנוכחי מהשרת. האלמנט `new` יורה פעילות לתאריך נוכחי ובפורמט של JavaScript הדיווח על התאריך נמצא כאן. שימו לב כי התאריך שנמצא כאן הוא מתאים לעילה מכיוון ששנת הארציות שלנו לשאקארית הפועלת כעקרון ממש אסורות עברו כמה ימים מלפני שהשינויים נכנסו לתוקף.

```TypeScript
// להתאמת תאריך זה שנשתמש בו בקוד contrib שלנו (הדרום)
const currentDate = new Date().toLocaleDateString('he-IL');
// "אוקטובר 05, 2021"

// מבלי לקבל את התאריך מהשרת
const jsDate = new Date(Date.now());
console.log(jsDate.toISOString());
// "2021-10-05T17:32:03.084Z"
```

הפונקציה toLocaleDateString מקבלת המחרוזת של התאריך בצורה כזו ומציגה את התאריך בעברית בצורה מתאימה. בנוסף, ניתן להתאים את הפוקציה ללוקל שלך כדי לקבל את התאריך בפורמט המתאים לאזור שלך.
