## למה

כמו בכל פרויקט תכנותי, קבלת התאריך הנוכחי יכולה להיות בסיסית לחלקים רבים בקוד התוכנית. לדוגמה, אם אנו רוצים להציג את התאריך בכותרת של פוסט בבלוג או בתור חלק ממנגנוני צפייה במשתמש, נצטרך לקבל את התאריך הנוכחי. לכן, חשוב לדעת כיצד לקבל בצורה נכונה את התאריך הנוכחי בקוד TypeScript שלנו.

## איך לעשות זאת

השימוש בפונקציה המובנית `Date()` יאפשר לנו לקבל את התאריך הנוכחי בתור אובייקט של תאריך בעצמו. באמצעות הפונקציה `getSeconds()` נוכל לקבל את מספר השניות בתאריך הנוכחי, ועם הפונקציה `getMinutes()` נוכל לקבל את מספר הדקות. לדוגמה:

```TypeScript
let currentDate = new Date();
console.log(currentDate.getSeconds()); // פלט: ערך השניות הנוכחי
console.log(currentDate.getMinutes()); // פלט: ערך הדקות הנוכחי
```

חשוב לזכור שעלינו להשתמש בפונקציות אלו בתוך אובייקט התאריך שיצרנו, כדי לקבל את התאריך הנוכחי.

## חקירה מעמיקה

איך המחשב מבין את התאריך הנוכחי ומציג אותו לנו? כאן נכנסת לתמונה פונקציית הגריניץ', אשר היא אלגוריתם מובנה במחשב שמחשב את התאריך הנוכחי ביחס לתאריך הקבוע של המחשב. פונקציית הגריניץ' כוללת מספר נוסחאות ותנאים שתוכננו על ידי מתמטיקאי יוסף לואי גריניץ' בשנת 1986.
