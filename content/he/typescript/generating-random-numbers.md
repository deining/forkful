איך ליצור מספרים אקראיים בטיפוסקריפט

## למה

היית מתפעל ליצור מספרים אקראיים בקוד שלך בטיפוסקריפט? במילים פשוטות, ייצור מספרים אקראיים יכול להיות שימושי לחלוטין בתוכנה שלך. מספרים אקראיים יכולים לשמש ככאילו "מנוע" ליצירת נתונים שונים כל פעם שהתוכנית שלך מחלוקת. זה יכול להיות מועיל בתהליך הנקרא "בדיקת תומך" שבתוכנה, תוך כדי אימות חוקי, וזה אפילו יכול להיות מועיל בפיתוח משחקים שכשיש להם הרבה דברים למנה.

## איך לעשות זאת

תחילה, בבבני שמיעים תלת אותיות את היחידה המקבילה לשימוש במספר מקרים. תוסיפה תבנית לקוד שלך עשוי להסביר את זה:

```Typescript
// ייצור מספר אקראי בין 0 ל־1:
let randomNumber = Math.random();

// ייצור מספר אקראי בין 0 ל־10:
let randomNumberBetween0And10 = Math.floor(Math.random() * 10);

// ייצור מספר אקראי בין 10 ל־50:
let randomNumberBetween10And50 = Math.floor(Math.random() * 40) + 10;
```

אם אתה יודע שהתוכן, תוכל להשתמש בכל צופן אתה רוצה כדי לחשוב על הטווח של המספרים שנוצרים.

## צעד בצעד

אפשר להרחיב את החלק הזה לאורך חריצה אחת או שתיים לפני היצירה ועבור כל איפש המחשבת מחדש את עצמו.תוכל לכתוב יותר על סיפור הייצו ׽של מספרים תיקניים ואת מספר הרנדומליים, המקורות שלהם, סקירות רישיון (ex: licenses) שלהם, ועוד.
