## למה

בעולם התכנות ישנם כמה דרכים לשמור ולארגן את המידע המכיל בקובץ. כתיבת קבצי טקסט היא אחד מהדרכים הפשוטות והנפוצות ביותר להכין ולאחסן מידע. בכתיבת קבצי טקסט ב־TypeScript ניתן לתת צורה מסודרת למידע ולטפל בו בקלות.

## איך לעשות זאת

נוכל להשתמש בפונקציות ייצוא וייבוא של TypeScript כדי לכתוב ולקרוא מידע מתוך קובץ טקסט. לדוגמה, נוכל ליצור פונקציה שמדפיסה את הנתונים שלנו לקובץ טקסט, ולאזן בין יצירת קבצים חדשים ועדכון קבצים קיימים. הנה דוגמא לכיצד נפעל בפונקציה הזאת:

```TypeScript
export function printToFile(data: string) {
  fs.writeFileSync("data.txt", data); // קריאה לפונקציה שתכתוב לקובץ הטקסט את המידע שמועבר אליה
  console.log("המידע נכתב לקובץ בהצלחה!");
}

printToFile("היי אני קובץ טקסט חדש!"); // פונקציה שמוסיפה את השורה הזאת לקובץ המקומי
```

בתום הקריאה לפונקציה נוכל לראות בקובץ הטקסט שלנו את המידע החדש שנועד להיות בו.
