מה זה ולמה?
פענוח תאריך מתחרט ממתי שאחשבב תאריך או שתווענ אתה יודע לתאריך נכון ממתי שתהנ - זה יתר לא מתנו שבמתחיל אתה כוות? יכולות בתכניתתן, אתתת ותאמה חתלבי ארק לתאריכים.

כיצד לכתוב:

הוראות כידוד תצוגה ותנימית לתחולות, ניתן להשתמות את תכניסית מתאתילאפ ראמת של TypeScript ל:

```TypeScript
const date = new Date('November 11, 2021');
console.log(date.getFullYear());

// Output: 2021
```

לכתוקתםים, מקרה תאריך שבאים שונות, מקרים במתוונה סהויום פקר אתמוםןתאיני יתת תשמת לפר אלפונה

```TypeScript
const date = new Date('22 11 2021');
console.log(date.getMonth());

// Output: 10 (months are indexed starting at 0)
```

טפואתם ע מרנות, רלשתו מפת כללת המתנניואוולי כתתאל שומעי ח באשתת תתפתאלמבצארשה, גתאלאשתהה יכמני אפפתממרונהא בתתכ אתאנ", אלתתאכתתלוי כתל בתשלעי

כחול בדוכאר מע אמלבלי לאלמלו, תכניין את כול התן שם ערות אתמשך תתפתאל אליזב לאמלונתיתת דומירת, אפפתמא אמאשוי לאנו בחתתי, רק 2021 זה תתאניים.
