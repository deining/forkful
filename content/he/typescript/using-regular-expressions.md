## למה?

תכנות ב-TypeScript הוא אחד מהכלים המופעלים ביותר בעולם המחשוב המודרני. בזכות הריכיז שלו בכתיבת קוד מובנית, הוא מאפשר לנו ליצור אלגוריתמים יצירתיים ומתקדמים יותר מתמיד. בנוסף, כיוון שהוא קוד פתוח, ישנן כמויות עצומות של ספריות וכלים זמינים שיכולים להעשות בקלות הרבה ולהפעיל במהירות חיונית את הקוד שלנו.

## איך לעשות זאת?

כעת, נמצא בפנייה לשימוש בביטויים רגולריים, כמו כאשר עושים משפטיים בביטויים רגולריים חשובים במיוחד. תכנות ב- TypeScripts מאפשר לנו ליצור חיפושי טקסט מתוכננים בביטויים רגולריים בקלות. לדוגמה, אם אתם רוצים לקבל את כל המילים בטקסט הנתון עם אות גדולה או נחשבים למילים, אתה יכול להשתמש בצורה הבאה בריכוז הראשון:

```TypeScript
const text = "זהו דוגמה לטקסט בעברית עם אות גדולה." 
const regex = /[\u0591-\u05F4\uFB1D-\uFB4F]/ 
const output = text.match(regex) 
console.log(output)
```

׳זה תחרות דוגמה׳׳גם לט הסביבה עם אותית אתרם כאן. הפונקציה `match` מחזירה מערך של כל המתאימים המצאי־מצפים בביטוי הרגולרי. בדוגמה זו, המתאים היחיד הוא האות "כ״, אשר היא האות היחידה הראשונה בסדר של האותיות בעברית.
