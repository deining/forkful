## מדוע
בשפת TypeScript קיימת כמה פתרונות לעיבוד טקסט. אחת מהן היא יכולת להשתמש בביטויים רגולריים, היכן שתוכלו לחפש ולמצוא תבניות מסוימות של טקסט בקלות. ביטויים רגולריים נהנים מניסיון והם מאפשרים לכם לבצע מספר כדי להתאים את הכוונה של הטקסט שאתם מעוניינים לעקוף.

## איך להשתמש
```TypeScript
// הגדרת ביטוי רגולרי דרך הסימבול "^" לתחילת מחרוזת והסימבול "$" לסוף
let הביטוי_הרגולרי = /^ספרים?$/;

// מישהו כתב שלוש מילים בטקסט של UNIVERSAL ורוצים למצואו בטירוף
let הטקסט = "אני שונא ISBN של שרותם של ספר";

// צרוף את הכיוון הרגולרי לטקסט והדפס את התוצאה
console.log(הביטוי_הרגולרי.test(הטקסט)); // false
```
במקרה זה, התוכנית תדפיס את `false` מכיוון שהמילה `ספר` אינה מתאימה לביטוי הרגולרי שנצרף אליו.

## חקירה עמוקה
ביטויים רגולריים יכולים להיות קשים ללמוד ולהבין, אך כאשר מבינים את הלוגיקה מאחורי הם יכולים להיות מאוד חזקים בתכנות. ניתן להשתמש במתים או לולאות להשתמש יחד עם ביטויים רגולריים על מנת לממש תוכניות לוגיות מורכבות.
