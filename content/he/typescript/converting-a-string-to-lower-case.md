## למה
שפת תכנות TypeScript מציעה כמה כלים מובנים שמאפשרים למפתחים לנהל ולעבוד עם טקסט בצורה יעילה יותר. להמרה של מחרוזות לאותיות קטנות היא אחת מהן ויכולה להיות שימושית במספר מקרים שונים, כגון אימות קלט משתמש או עיבוד נתונים.

## איך לעשות 
קוד לביצוע הופך אותיות גדולות לאותיות קטנות בשפת TypeScript הוא פשוט וקל להבנה. באמצעות הפונקציה `toLowerCase()` אנו יכולים לשנות את המחרוזת לתוכנה קטנה, בדוגמה הבאה קוד להמרת מחרוזת לתוך קטנה והדפסת התוצאה המבוקשת:

```TypeScript
let myString = 'כותרת בתחילת מחרוזת'
console.log(myString.toLowerCase()) // כותרת בתחילת מחרוזת
```

בנוסף, ניתן להשתמש במתודה `toLocaleLowerCase()` שתשתמש באיתור הסביבוני של הלשון במחרוזת הנתונה ותמיר אפילו את האותיות הכי איכותיות. לדוגמה:

```TypeScript
let myString = 'מחרוזת עם תווים מיוחדים'
console.log(myString.toLocaleLowerCase()) // מחרוזת עם תווים מיוחדים
```

## חקירה מעמיקה
המרת מחרוזות לאותיות קטנות היא תהליך פשוט אבל מאפשר למשתמשים לעבוד ולנהל מחרוזות בצורה מאורגנת יותר. חשוב לזכור שהפונקציה `toLowerCase()` כוללת את כל האותיות באותו הצורה כפי שמופיעות בתוך המחרוזת הרגילה. זאת אומרת שאם נרצה להמיר מחרוזת בניווט לאותיות קטנות עם תווים מיוחדים, עלינו להשתמש במתודה `toLocaleLowerCase()` כדי לקבל את התוצאה הרצויה.
