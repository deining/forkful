## למה

קיימות כמה סיבות למה מתכנתים בדרך כלל משתמשים בפעולת המרה של מחרוזות לתווים קטנים כאשר אנו עובדים עם מחרוזות בקוד שלנו. הסיבות עשויות להיות איתור של מחרוזת אחת ספציפית באופן חופשי או כשאנו רוצים להשוות שני מחרוזות באמצעות אופרטור של השוואה.

## כיצד לעשות זאת

תחילה, נגדיר מחרוזת כלשהי בצורה רגילה כמו פרמטר בפונקציה כדי ליצור מחרוזת חדשה. לאחר מכן, נשתמש בפעולה המשווה .toLowerCase() כדי להמיר את המחרוזת לתווים קטנים. הנה דוגמה בקוד TypeScript:

```TypeScript
let name = "דניאל";
let lowercaseName = name.toLowerCase();
console.log(lowercaseName);
```

פלט:

```
דניאל
```

בדוגמה זו, אנו משווים את המחרוזת המקורית "דניאל" למחרוזת חדשה שמכילה תווים קטנים ("דניאל"). ניתן לראות את התוצאה באמצעות פקודת הדפסה שלנו.

## חקירה מעמיקה

כאשר אנו משתמשים בפונקציה .toLowerCase(), התוכנית שלנו מחפשת במערכת תווים קיימים במחרוזת וממירה אותם לתווים קטנים. יש לציין כי פונקציה זו יכולה להתמודד עם יוניקוד וקודי ANSI כאשר אנו משתמשים בחישבון 8 ביט במיוחד. בכל מקרה, פונקציה זו מוכיחה קלה ואלגנטית לשימוש כאשר מדובר בהמרת מחרוזות לתווים קטנים.
