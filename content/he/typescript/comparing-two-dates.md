## מדוע

בדיקת תאריכים היא כלי חשוב בתכנות המאפשר לנו לבצע פעולות כמו מיון תאריכים והשוואתם. זה גם יכול להיות מועיל למשתמשים באתרים או אפליקציות שמציגים תאריכים שונים, וכך לייצר ממשק משתמש נוח וידידותי.

## איך לעשות זאת

תחילה נצטרך לייבא את הספריה המתאימה של TypeScript, למשל `moment.js`, שמאפשרת לנו לעבד תאריכים בקלות ובמדויק. לאחר מכן, נוצר לדוגמה שני אובייקטים של תאריכים - אחד מהם נמצא בעבר והשני בהווה. לבסוף, נשתמש בפונקציות המתאימות כדי להשוות בין התאריכים ולקבל תוצאה בהתאם. במקרה זה, ניתן להשתמש בפונקציית `isBefore()` כדי לבדוק אם התאריך המבוקש נמצא לפני התאריך הנוכחי.

```TypeScript
import * as moment from 'moment'; //מייבאים את הספריה
let pastDate = moment('2000-01-10', 'YYYY-MM-DD'); // יצירת אובייקט תאריך
let presentDate = moment(); // יצירת אובייקט תאריך נוכחי
// השוואה בין התאריכים
if (pastDate.isBefore(presentDate)) {
    console.log('התאריך 10/01/2000 היה בעבר');
} else {
    console.log('התאריך 10/01/2000 טרם הגיע');
}
```
פלט:
`התאריך 10/01/2000 היה בעבר`
