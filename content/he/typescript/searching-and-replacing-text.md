## למה
תחילה, בואו נתאר בקצרה למה אנשים מתעניינים בחיפוש והחלפת טקסט בתוכניות שכתבתם. אחד הסיבות העיקריות לכך היא ליישמו תיקונים ומציאות שגיאות בטקסט בקלות ומהירות. ניתן לחילופי טקסט להיות בעייתיים יחסית, ולכן חייבים לכתוב קודים שיוכלו לעשות את העבודה במקום שלנו באופן מהיר ומדויק.

## איך לעשות זאת
כדי לעשות חיפוש והחלפת טקסט בתוכנית שלנו, נעזר בספריית ה-TS-PEGJS. נבין את התהליך תחילה ע"י כתיבת קוד העברת טקסט מתאים למתבקש תחתכים שלא מכילים רווחים. לדוגמה, השתמשו במשתנה הבא עם הטקסט שברצונכם להחליף:

    ```TypeScript
    let text = "זהו טקסט לדוגמה להחלפת מילים בתוך טקסט";
    ```

בשלב הבא, נוסיף את הטקסט שברצוננו להחליף כך כמו המשמש בתוכנות נוספות, יקח כמה מילים ויחליף את הטקסט הקיים עם המילים החדשות. נשתמש בפונקציה `replace()` כדי לעשות זאת. הנה דוגמה של קוד שמחליף את המילה "החלפת" במילה "חיפוש":

    ```TypeScript
    let newText = text.replace("החלפת", "חיפוש");
    console.log(newText);
    ```

וכעת הקוד מחזיר את התוצאה הבאה:

    זהו טקסט לדוגמה לחיפוש מילים בתוך טקסט
