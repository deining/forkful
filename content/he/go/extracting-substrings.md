## למה 

למה יש לנו צורך לספק קטעי טקסט מתוך מחרוזת?

החיתוך והסינוני הערכים מתוך תוך כדי עיבוד הטקסט הינו פעולה חשובה בתכנות. את הקטעים הללו ניתן להשתמש בהם בכדי לפענח מידע או להעבירו לפונקציות אחרות לטיפול נוסף. באמצעות נרחבת שפת התכנות, גולם היחודי של GO ניתן לפעולה רחבה זו ביעילות.

## איך לספק קטעי טקסט ב-GO

כדי לספק קטעי טקסט מתוך מחרוזת ב-GO, ניתן להשתמש בפונקצייה "Slice", המקבלת כפרמטרים את המחרוזת, המיקום התחילי והמיקום הסופי של הקטע. לדוגמה:

```Go
stringToSlice := "תכנות GO הוא כיפי למדי"
slicedText := stringToSlice[:18] // מחזיר את הפרק הראשון
fmt.print(slicedText) // פלט: "תכנות GO הוא כיפי"
```

כאשר המיקום הסופי לא מוגדר, הפונקציה תספק את כל הטקסט מהמיקום התחילי ועד הסוף של המחרוזת.

```Go
stringToSlice := "המחרוזת אורך של 20 תווים"
slicedText := stringToSlice[5:] // מחזיר את הפרק השני
fmt.print(slicedText) // פלט: "חרוזת אורך של 20 תווים"
```

ניתן גם להשתמש בפונקציה "Substring", המקבלת כפרמטרים את המחרוזת, המיקום התחילי ואת אורך הקטע.

```Go
stringToSub := "קטע טקסט ב-GO"
substring := stringToSub.Substring(4, 9) // מחזיר את הפרק השלישי
fmt.print(substring) // פלט: "טקסט ב-GO"
```
