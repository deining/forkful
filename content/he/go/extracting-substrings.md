# מדוע:

כשמפתחים בגו פתחנו, אנו מתקדמים לעתים קרובות למצב בו נאלצים לעסוק בשחרור מחרוזות מטקסט ארוכות ומורכבות הכילולניות מידע שאינו נחוץ לנו. החזרת תת מחרוזת מתאימה יכולה להקל על ניהול המידע הזה ולהפוך את הקוד שלנו לנקי יותר ומובן יותר.

## איך לעשות זאת:

תהליך החיתוך של מחרוזת לתת מחרוזת משתמש בפונקציות המובנות של גו כמו `substring` ו- `slice`. הנה דוגמא פשוטה לחיתוך תת מחרוזת ממחרוזת נתונה:

```Go
package main

import "fmt"

func main() {
	str := "קוד גו פשוט"
	substr := str[4:7]
	fmt.Println(substr)
}
```

פלט:
```
גו
```

בדוגמא זו, אנו משתמשים במיקום התווים לדוגמאים, המספקים לנו את המידע הנחוץ לחיתוך תת מחרוזת ממחרוזת נתונה.

## לטבוע עומק יותר:

החיתוך תת מחרוזת כולל גם את האפשרות לקבל מספרים שליליים, שערכם מחשב עיניים מימין לשמאל של המחרוזת. בנוסף, ניתן לשדרג את התהליך על ידי שימוש בפונקציית `split` שתאפשר לנו לחלק את המחרוזת לחתכים מסוימים ולחתוך תת מחרוזת מתוך חתך מסוים באמצעות מיקומי התווים.
