## למה

למה לעסוק בחילוץ תת מחרוזות? תהיה לנו לעשות עבודה עם מחרוזות גדולות יותר ולהפשיט את התנאים שלנו בתכנות.

## איך לעשות?

```Go
func main() {
	// נבדוק את המחרוזת המקורית
	str := "שלום עולם!"

	// נחתוך את המחרוזת למקטע שכולל את האות הראשונה עד האות הרביעית
	substr := str[0:4]

	// נדפיס את המקטע החדש
	fmt.Println(substr)
}
```

הפלט יהיה:

```
שלום
```

## צלילה עמוקה

כאשר אנו מתחילים לתכנת בשפת גו, חישובים על מחרוזות יכולים להיות קצת מעצבנים. ישנם כמה פעולות ומתודות הקשורות לחילוץ תת מחרוזות, וכמו כל דבר, כדאי ללמוד אותם בהעמקה כדי להיות מיומן יותר בשימושן. בדקו את קישורי הלמידה למטה לפני שמתחילים לעבוד עם מחרוזות בגו.
