## למה?

בעולם התכנות קיים מגוון רחב של שפות תכנות, אך בשנים האחרונות ניתן לראות צמיחה מאוד משמעותית של שפת התכנות Go. השאלה המתבקשת היא - למה כדאי להתחיל להתנסות ולפתח פרויקט חדש בשפה זו?

התשובה היא פשוטה - Go מציעה יתרונות רבים וחשובים למפתחים. היא מתבססת על מודל פתוח וקוד פתוח מאפשר הרחבה והתאמה לצרכים המשתנים של הפרויקטים השונים. כמו כן, היא נפוצה ונתמכת על ידי קהילת מפתחים גדולה ופעילה. מומלץ ללמוד את השפה ולהכיר את היתרונות שהיא מציעה.

## איך לעשות זאת?

ההתחלה היא פשוטה וקלה. יש להתקין את הסביבה של Go על המכשיר שלכם ולהגיע להמנעות על המשמעת המקומית. ניתן לבצע זאת באמצעות הפקודה `go build` כדי לבנות את הקוד, `go fmt` כדי לנהל את מבנה הנתונים ו`go test` כדי לבצע בדיקות יחידה. להלן דוגמאות של קוד Go ותוכלו לראות את תוצאות הפלט המשובצות בתוך בלוקי הקוד `Go`.

```Go
package main

import "fmt"

// פונקציה המדפיסה את "שלום עולם"
func main() {
    fmt.Println("שלום עולם")
}
```

הפלט יהיה `שלום עולם`.

```Go
package main

import "fmt"

// פונקציה המחשבת את הסכום של שני מספרים
func main() {
    var x = 3 // מה שזוהי הבחנה
    var y = 4
    var sum = x + y // הוספת שני מספרים
    fmt.Print("הסכום הוא: ", sum)
}
```

הפלט יהיה `הסכום הוא: 7`.
