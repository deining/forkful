## למה

ברוך הבא לבלוג הגיטהאב החדש שלנו על תכנות בגו! אם אתם מתחילים בתחום התכנות או רוצים לעבור ממקור קוד אחד לשפת גו, אנו ממליצים לכם להתחיל פרויקט חדש בשפה זו. גו היא שפת תכנות יעילה ופשוטה, ויכולה לתת לכם כלים מתקדמים לכתיבת קוד נקי וממוקד.

## איך

כדי להתחיל פרויקט חדש בגו, ישנם כמה צעדים פשוטים שיעזרו לכם להתחיל במהירות וביעילות. ראשית יש להתקין את סביבת הפיתוח של גו על המחשב שלכם. לאחר מכן, יש ליצור קובץ חדש בשם "main.go" ולכתוב קוד ראשון פשוט:

```Go
package main

import "fmt"

func main() {
  fmt.Println("שלום עולם")
}
```

כעת כדי להריץ את הקוד, יש להשתמש בפקודות "go build" ו-"./[שם התוכנית]". וואלה, עכשיו יש לכם תוכנית ראשונה בגו!

## חפירה מעמיקה

עם תחילת הפרויקט, כדאי להתחיל לחקור ולהתמקצע בגו עוד יותר. כדי לעזור לכם בדרך, אנו ממליצים לכם לבדוק את המדריכים הבאים:

- [מדריך התחלה מהירה בגו](https://golang.org/doc/tutorial/getting-started)
- [מדריך רשמי של גו](https://tour.golang.org/welcome/1)
- [מדריך "איך לכתוב קוד בגו"](https://golang.org/doc/effective_go.html)

בנוסף, ישנם מגוון רחב של ספריות, מתועדת ונעימה לשימוש, שיכולות להקל על התקנת הפרויקט. ניתן למצוא את המודולים הפופולריים ביותר במאגר הגוגל "GoDocs".
