## למה?

כתיבת קוד בשפת גו (Go) נהדרת בכדי ליצור פרויקט חדש! זו שפה מהירה, פשוטה ללמידה ועמידה בפני כמות גדולה של מיכשורים ופלטפורמות. אם אתה מעוניין להתחיל פרויקט חדש וללמוד עוד על גו, המאמר הזה יתאים לך.

## איך לעשות זאת?

כדי להתחיל פרויקט חדש בגו (Go), עליך להתחיל בכתיבת קוד בשפה החדשה של גו (Go). למרבה המזל, יש לך מדריכים רבים שזמינים ברשת, שיעזרו לך להתחיל במהירות.

אם אתה רוצה לצור פרויקט חדש, עליך להתחיל ב- "גו" (Go)! למשל, בואו נתחיל עם משימה פשוטה - עריכת קובץ טקסט. עם גו, אתה יכול להשתמש במודול newfile כדי ליצור קובץ חדש ולהכניס טקסט לתוכו. ננסה את זה בכך שניצור קובץ טקסט חדש בשם "hello.txt" ונכניס לתוכו את הטקסט "Hello, world!".

```Go
package main

import (
	"fmt"
	"io/ioutil"
)

func main() {
	fileName := "hello.txt"
	fileContent := "Hello, world!"

	err := ioutil.WriteFile(fileName, []byte(fileContent), 0644)
	if err != nil {
		panic(err)
	}

	fmt.Printf("Created file: %s\n", fileName)
}
```

הרץ את הקוד ותראה שנוצר קובץ טקסט חדש בשם hello.txt שמכיל את המילה "Hello, world!". אתה יכול להבין מהרץ של הקוד שתהליך כתיבת הקוד הוא פשוט וקל יותר מאשר בשפות תכנות אחרות.
