## מה & למה?
התחלת פרויקט חדש היא תהליך שבו מתחילים ליצור קוד חדש עבור פרויקט חדש. כרגע, גורם רבים מרוצים לפרושרים להתחיל פרויקט חדש כדי לשפר את החוויה של המשתמשים ולפתח טכנולוגיות חדשות.

## איך ל:
להראות את הדוגמאות הבאות של השלמת פרויקט חדש בשפת גו:

```Go
package main

import "fmt"

func main() {
  fmt.Println("היי עולם!")
}
```

פלט:
```
היי עולם!
```

בכדי להתחיל פרויקט חדש, יש ליצור קובץ חדש בשם "main.go" ולכתוב בתוכו קוד שיציג את התוצאה הרצויה. לאחר מכן, ניתן להריץ את הקובץ בכתובת השורה של הטרמינל עם פקודה זו:

```
go run main.go
```

סיימנו! עם זאת, ניתן לשנות את הקוד הדוגמא כדי ליישם פונקציונליות נוספת כגון קלט מהמשתמש או בדיקת תנאים.

## צלילות מקור:
שפת התכנות גו היא לשונית ויעילה, ונועדה במקור להיות שפת התכנות הבאה של פרויקט גוגל. אם אתה מעוניין להתחיל ללמוד גו, ישנם תוכנית של למידה לנצח זמינה באינטרנט ומחברת תקשורת ייחודית.

ישנן גם אלטרנטיבות לשפת גו כמו פייתון, קודי וגם גו רגיל בבתים כה ארסי לא למידה.

הטכנולוגיות הנוספות שמבקשות ממצבי גו בדוגמא: סטאן, החפש גו-קוד, מאים ו- סימי טז.
