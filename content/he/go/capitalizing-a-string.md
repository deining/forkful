## מדוע

למה שתרצו להתעסק בכתיבה תחתי. בדוגמאות אנו נראה כמה דרכים שבהן אנו יכולים להשתמש בפונקציית הכתיבה תחתי בשפת גו על מנת לשנות סטרינג.

## איך לעשות זאת

בשפת גו, אפשר להשתמש בפונקציית הכתיבה תחתי כדי להפוך את האותיות הראשונות במחרוזת לגדולות. זה יסייע לנו להגדיר מחרוזות בצורה נכונה לפי התקנון הדרוש במירוץ שלנו, לדוגמא: אם התקנים דורשים מחרוזת להיות כתובה באותיות גדולות בלבד, אפשר להשתמש בפונקציית הכתיבה תחתי כדי לוודא שהמחרוזת החדשה תתאים לדרישות.

לדוגמאות אנו נשתמש במחרוזת "hello world" כדי להראות איך הפונקצייה עובדת:

```Go
package main

import (
    "fmt"
    "strings"
)

func main() {
    str := "hello world"
    capital := strings.ToUpper(str)
    fmt.Println(capital)
}
```

פלט:

`HELLO WORLD`

בנוסף, אפשר להשתמש בפונקציית הכתיבה תחתי כדי להפוך את האותיות שאינן בראש המחרוזת לקטנות, באמצעות הפונקציה `ToLower` כפי שמופיע בדוגמא הבאה:

```Go
package main

import (
    "fmt"
    "strings"
)

func main() {
    str := "HeLlO WoRlD"
    lower := strings.ToLower(str)
    fmt.Println(lower)
}
```

פלט:

`hello world`

## חקירה מעמיקה

בשפת גו, הפונקציה `ToUpper` והפונקציה `ToLower` משתמשות בנתון לוגי פשוט שמאפשר להפעיל את הפונקציה על סטרינג מסוים. במקרה שלנו, הנתון הוא האות הראשונה במחרוזת, ולכן פונקציית הכתיבה תחתי יכולה להוסיף קפיטל לאותיות המחרוזת או להוריד אותן. מידע נוסף על הפונ
