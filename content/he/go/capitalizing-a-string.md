# למה:

למה כדאי לדבר על פונקציית ההגדלה בשפת תכנות גו? התשובה פשוטה - כי פונקציית ההגדלה היא כלי חשוב ושימושי בשפה שיכול לשמשנו במגוון דרכים ובתחומים שונים. 

# איך להגדיל טקסט בשפת גו:

אם ברצונך להגדיל טקסט בשפת גו, תוכל לעשות זאת בקלות מאוד באמצעות פונקציית ההגדלה - `strings.ToUpper()` . השתמש בדוגמאות הקוד הבאות כדי להבין איך זאת עובדת בפועל:

```Go
// קוד דוגמה עבור הגדלת טקסט
package main

import (
	"fmt"
	"strings"
)

func main() {
	str := "זאת הולכת להיות טקסט באותיות קטנות."
	fmt.Println("טקסט לפני ההגדלה:", str)

	// החזרת הטקסט באותיות גדולות באמצעות הפונקציה ToUpper()
	str = strings.ToUpper(str)
	fmt.Println("טקסט אחרי ההגדלה:", str)
}

// פלט:
// טקסט לפני ההגדלה: זאת הולכת להיות טקסט באותיות קטנות.
// טקסט אחרי ההגדלה: זאת הולכת להיות טקסט באותיות גדולות.
```

בדוגמה הנ"ל, אנו משתמשים במשתנה `str` המכיל את הטקסט שברצוננו להגדיל. נשתמש בפונקציית `strings.ToUpper()` על המשתנה שלנו כדי להחזיר את הטקסט באותיות גדולות. בשורה האחרונה, אנו מציגים את הפלט שלנו כדי לוודא שההגדלה עבדה.
