## מה ולמה?

הגרלת מספרים אקראית היא פעולה שמשתמשים בה פרסומותית בתחומי התכנות. בגדול, היא מאפשרת למפתחים ליצור מספרים אקראיים בתוך תוכניותיהם, מה שיועיל בהרבה מקרים כגון משחקים, אבטחת מידע ובדיקות.

## איך לעשות:

כדי ליצור מספרים אקראיים בשפת Go, ישנם שני דרכים עיקריות - באמצעות החלקה או באמצעות פונקציה מובנית.

### באמצעות החלקה:

ישנן כמה שיטות שונות להחליק מספרים אקראיים בשפת Go. בהמשך נראה קוד דוגמה לשיטה פשוטה ביותר, רק כדי להראות איך זה נעשה בגדול. הרחבה על הנושא תוכלו למצוא בסעיף "עומק מתקדם".

```Go
package main

import (
	"fmt"
	"math/rand"
	"time"
)

func main() {
	rand.Seed(time.Now().UnixNano()) // עדכון של הזרע של המחולחן עם ערכים אקראיים
	num := rand.Intn(10)              // תפוס את הערך האקראי שזורק החולץ
	fmt.Println(num)                  // הדפס את המספר האקראי
}
```

### באמצעות פונקציה מובנית:

שפת Go מכילה גם פונקציה מובנית ליצירת מספרים אקראיים. ניתן להשתמש בה בפשטות כדי לקבל מספר אקראי בתוך תוכנית.

```Go
package main

import (
	"fmt"
	"math/rand"
)

func main() {
	num := rand.Intn(100) // תפוס את הערך האקראי שזורק החולץ
	fmt.Println(num)      // הדפס את המספר האקראי
}
```

## עומק מתקדם:

ההחלקה היא שיטת הגרלת מספרים אקראיים שנהיה מוכרת בשפת Go. היא מתבצעת בעזרת החולץ המובנה שבחבילת `math/rand` ובאמצעות פונקציית הרנדום `Intn(n)`. החולץ מקבל כפרמטר את מספר הסטים שמתחילים מ-0 ועד `n`-1. כל שהערך הוא חיובי יותר, כך יהיה מעט אפשרויות יותר להחליק כל מספר אקראי בלי שיופיע שוב.
