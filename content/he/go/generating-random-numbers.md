## למה

לכתוב קוד גואה שמגריל מספרים הוא צורך נפוץ בפרוייקטים שונים. גירוד אקראי של מספרים יכול לשמש למגוון רחב של מטרות, כולל המחשבה על אירועים רנדומליים ובדיקת תנאים בתנאי לולאה.

## איך לעשות זאת

להגריל מספרים באמצעות גואה פשוט ואינטואיטיבי. גואה מספקת מספר פונקציות שניתן להשתמש בהן ליצירת מספרים אקראיים בצורה קלה ומהירה.

```Go
// ייבוא החבילה של גואה
import "math/rand"

// יש להשתמש בחבילה "time" ליצירת סדר למספרים שמתוותחים
import "time"

func main() {
	// נקבע זרע עבור הגירוד האקראי המתוותח
	rand.Seed(time.Now().UnixNano())

	// הגירוד האקראי של מספר בין 0 ל-10
	fmt.Println(rand.Intn(10))

	// הגירוד האקראי של מספר בין 0 ל-100
	fmt.Println(rand.Intn(100))

	// הגירוד האקראי של מספר בין 1 ל-6 (כמו קוביות)
	fmt.Println(rand.Intn(6) + 1)
}
```
**פלט:**
```
7
93
4
```

## מנועי הגנרציה של גואה

במקרים רבים ישנם צרכים מתקדמים כדי ליצור מספרים אקראיים באופן אמיתי. למשל, היכולת למגריל מספרים בתחום מסוים או ליצור חלקות לולאות עם ניצורים אקראיים מובנים. גואה מציגה מנועי גנרציה שונים אשר מתאימים לכל אחד מאלה הצרכים.
