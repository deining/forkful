# מה & למה?
השגת התאריך הנוכחי היא פעולת תכנות קריטית שמאפשרת למפתחים לקבל מידע עדכני ומדוייק לגבי תאריך ושעה בשביל היישומים שלהם. להגיע לתאריך הנוכחי מאפשר למפתחים לבצע פעולות זמן קריטיות ולהתאים את היישומים שלהם לזמנים בצורה יעילה יותר.

# איך לעשות?
קוד של Go לשלוח כמה שורות קוד פשוטות כדי לקבל את התאריך הנוכחי.

```Go
import "fmt"
import "time"

fmt.Println("התאריך הנוכחי: ", time.Now())
```

פלט:
```
התאריך הנוכחי: 2021-07-20 22:10:15.731599 -0500 CDT m=+0.000390109
```

# כניסה עמוקה
כבר מאז שהמחשבים התחילו לתפקד, הייתה צורך להיות מדויקים עם הזמן והתאריכים. בעבר, השגת התאריך הנוכחי היתה ענין מורכב יותר ודרשה יכולת בניית קוד מורכבת כדי לקבל את התאריך המדויק ביותר. במקום זה, עם שפת תכנות Go ניתן טיפול לנתוני תאריך בצורה פשוטה יותר ומדויקה.

אם אתה מחפש דרכים אחרות לשלוט על התאריך הנוכחי בפרויקט שלך, ישנם גם חבילות Go אחרות כמו Chronos וDate Utils שיכולות לסייע לך.

לדוגמה, כאן תמצא קוד לשימוש עם Chronos כדי להדפיס תאריך ושעה ספציפית של אירוע ספציפי:

```Go
import "github.com/davecgh/go-spew/spew"
import "github.com/gualtierl/chronos"

t, _ := chronos.Parse("2006-January-02 15:04:05", "2006-September-14 12:05:00")
spew.Dump("תאריך המבחן: ", t.Time)
```

פלט:
```
"תאריך המבחן: " ([]uint8) <[]uint8 | Len:0, Cap:16, Ptr:0x10fde0>
  <"2006-09-14 12:05:00">
```
