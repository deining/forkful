## למה

בכל יום אנו משתמשים בתאריך הנוכחי כאשר אנו פותחים אפליקציות, עורכים מסמכים או בונים אתרים. תאריך נכון הוא חשוב להצגת מידע ולהתאמה אישית למשתמשים. למרבה המזל, בשפת גו יש פשטות לקבלת התאריך הנוכחי, שנה, חודש ותאריך. במאמר הזה אנחנו נלמד כיצד לקבל את התאריך הנוכחי בשפת גו בצורה פשוטה ומהירה.

## איך לעשות זאת

כדי לקבל את התאריך הנוכחי בשפת גו, נשתמש בפונקציה `Now()` מחברת הספרייה `time`. לכן, נתחיל על ידי ייבוא הספרייה של `time`:

```Go
import "time"
```

כעת, נוכל להשתמש בפונקציה `Now()` כדי לקבל את התאריך הנוכחי בלי להכתב פעמים רבות. לדוגמה, אם אנחנו רוצים להדפיס את התאריך הנוכחי בתוך משפט, נוכל לעשות כך:

```Go
fmt.Println("התאריך הנוכחי הוא:", time.Now())
```

פקודה זו תחזיר את התאריך הנוכחי בפורמט התאריך של מחשבך. לדוגמה:

```Go
התאריך הנוכחי הוא: 2020-07-23 09:35:00.28494 +0000 UTC m=+0.002875601
```

כדי להציג את התאריך בפורמט אחר, נוכל להשתמש בפונקציה `Format()` ולהצהיר על פורמט רצוי כמו כך:

```Go
fmt.Println(time.Now().Format("02/01/2006, 03:04:05 PM"))
```

פקודה זו תחזיר את התאריך בתור `23/07/2020, 09:40:20 AM`. כדי ללמוד עוד על פורמטי תאריך בשפת גו, ניתן לקרוא את [תיעוד התאריך של חברת גוגל](https://pkg.go.dev/time?tab=doc).
