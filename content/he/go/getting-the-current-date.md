## למה

קבלת התאריך הנוכחי היא תכונה חשובה בתוכניות תוכנה רבות. יתכנו מספר סיבות שבזכותן נדרשת קבלת התאריך הנוכחי, כגון ניהול כניסות ויציאות, יצירת דוחות, או שימוש מול שירותים מקוריים. במאמר זה, נלמד איך להשיג את התאריך הנוכחי באמצעות שפת התכנות Go.

## איך לעשות זאת

כדי להשיג את התאריך הנוכחי בפורמט של שנה-חודש-יום המובנה בשפת Go, ניתן להשתמש בפונקציה time.Now():

```Go
import "fmt"
import "time"

func main() {
    currentTime := time.Now()
    fmt.Println(currentTime.Format("2006-01-02"))
}
```

בקוד הנ"ל, אנו משתמשים בפונקציה time.Now() כדי לקבל את התאריך הנוכחי בשפת Go. בפעם הראשונה שנכתוב את הפונקציה כזו, אנו נמצאים במצב "לימודי" כאשר נכתוב את הקוד. לאחר מכן, נשתמש בפונקציה currentTime.Format() כדי לקבל את התאריך בפורמט הרצוי. במקרה זה, אנו משתמשים בפורמט "2006-01-02", שכן בשפת Go המספרים המיוחדים אלו מיישמים את הפורמט של שנה-חודש-יום.

כעת, נריץ את הקוד ונקבל את התאריך הנוכחי בפורמט שנרצה:

```
2021-06-29
```

אם ברצונך לקבל את התאריך הנוכחי עם חלקי השעה והדקות, ניתן להשתמש בפורמט הבא:

```Go
import "fmt"
import "time"

func main() {
    currentTime := time.Now()
    fmt.Println(currentTime.Format("2006-01-02 15:04"))
}
```

כאן, אנו משתמשים בפורמט "2006-01-02 15:04", כאשר "15" מייצג את השעה ו"04" מייצג את הדקות. אם נריץ שוב את הקוד, נקבל את התאריך הנוכחי עם חלקי השעה והדק
