## למה

משהו מהכי שכשות בתחום התכנות הוא להשוות שתי תאריכים. ייתכן שאתה מחלק חברתי עם כמה חברים על איזה חופשת קיץ ואתה צריך למצוא את התאריך הטוב ביותר. או אתה יכול להיות מפתח תוכנה שמשתמש בתאריכים כדי להציג מידע מועיל למשתמשים. בכל מקרה, ידע כיצד להשוות שתי תאריכים בפרוגרמה שלך הוא חשוב ביותר.

## כיצד לעשות זאת

תחילה, נציג כמה דוגמאות לקוד בשפה של Go להשוואת שתי תאריכים:

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    // תאריך ראשון
    date1 := time.Date(2020, time.June, 1, 0, 0, 0, 0, time.UTC)
    
    // תאריך שני
    date2 := time.Date(2020, time.June, 15, 0, 0, 0, 0, time.UTC)
    
    // השוואת שתי תאריכים
    if date1.Before(date2) {
        fmt.Println("תאריך 1 מוקדם יותר מתאריך 2")
    } else if date2.Before(date1) {
        fmt.Println("תאריך 2 מוקדם יותר מתאריך 1")
    } else {
        fmt.Println("התאריכים זהים")
    }
}
```
פלט:
תאריך 1 מוקדם יותר מתאריך 2

בקוד זה, אנו משתמשים בפונקציות של חבילת "time" כדי ליצור שתי משתנים של תאריכים. לאחר מכן, אנו משתמשים בפונקציות "Before" ו-"After" כדי להשוות את התאריכים. אם תאריך יותר מוקדם לפני תאריך אחר, הפרינטר ידפיס הודעה מתאימה.

להלן עוד דוגמאות לפונקציות שתיאריך אפשר להשתמש עבור השוואה:

- בתאריך שנתי: פונקציות "Year", "Month" ו-"Day" ניתנות לשימוש כדי להשוות את שנה, חודש ויום בנפרד. לדוגמה: `date1.Year() < date2.Year()`
- בתאריך שעתי: פונקציות "Hour
