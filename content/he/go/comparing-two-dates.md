## למה

בעולם של קידמת טכנולוגיות כמו גו ומדעי מחשב, השוואת שני תאריכים היא כלי חיוני ליישום איכותי ויצירתי. עם תכונות מתקדמות כמו העבודה עם תאריכים במקום מחרוזות ושימוש באיזורי זמן שונים, ביצוע השוואה בין תאריכים נהיה מאתגר יותר ומדויק יותר.

## איך לבצע השוואה בין שני תאריכים בגו

המטרה העיקרית של גו היא להיות ספריית תכנותית פשוטה ויעילה. הוא מספק כמה פעולות מובנות שמאפשרות לנו לבצע השוואה בין שני תאריכים. 

תחילה, נצרף את היחידה "time" באמצעות השורה הבאה:

```Go
import "time"
```

לאחר מכן, ניצור תאריך ראשון על ידי שימוש בפונקציה time.Date ונוכיח אותו על ידי שימוש במתודה time.Equal:

```Go
date1 := time.Date(2021, 6, 15, 0, 0, 0, 0, time.UTC)
if time.Equal(date1, time.Now()) {
  fmt.Println("התאריך שווה לתאריך הנוכחי")
}
```

כדי לבצע השוואה בין שני תאריכים, ניצור תאריך שני על ידי שימוש בפונקציה time.Date ונשתמש במתודה time.Before או time.After כדי לבדוק אם התאריך הראשון מוקדם יותר או מאוחר יותר מהתאריך השני:

```Go
date2 := time.Date(2021, 6, 20, 0, 0, 0, 0, time.UTC)
if date1.Before(date2) {
  fmt.Println("תאריך 1 מוקדם יותר מתאריך 2")
}
```
