גו (גרסה הנוכחית) כתיבת מאמר לקוראים בשפה העברית, בסגנון לא פורמלי ובסגנון הדוק, כדי לשער ישר טעמים כדי לרווח על הנתונים של הימים ™ ומדוע 💻 גיבורים עושים את זה?

## מה זה ולמה?:
השוואת שני תאריכים היא תהליך שבו משווים בין שני תאריכים כדי לדעת מהו התאריך שנמצא בביניהם הכי מאוחר או מוקדם יותר. תהליך זה נחוץ למתכנתים כדי לעבוד עם תאריכים במכשירי המחשוב ובאפליקציות שלהם.

## איך לעשות:
ניתן להשוות שני תאריכים בשפת גו באמצעות הפונקציה `time.Equal()`. לדוגמא:

```Go
time1 := time.Date(2021, time.January, 1, 0, 0, 0, 0, time.UTC)
time2 := time.Date(2021, time.February, 1, 0, 0, 0, 0, time.UTC)

if time1.Equal(time2) {
    fmt.Println("שני התאריכים שווים.")
} else {
    fmt.Println("שני התאריכים לא שווים.")
}

// פלט:
// שני התאריכים לא שווים.
```

## מהמילים הראשונות:
השוואה בין שני תאריכים היא תהליך נפוץ בתחום המתכנות והמחשוב, ונעשה בעיקר כדי למצוא את התאריך המאוחר יותר או המוקדם יותר בין שני תאריכים. תהליך זה נחוץ גם במספר יישומים של אפליקציות ומכשירי חשיבה.

כלי אחרים שניתן להשתמש בהם להשוואה בין שני תאריכים כוללים את הפונקציה `Before()` ו- `After()` בשפת גו.

- למידע נוסף על שני תאריכים וכיצד לעבוד איתם, ניתן לעיין במדריך המפורט של גו על תאריכים: https://golang.org/doc/tutorial/go-tour#59
