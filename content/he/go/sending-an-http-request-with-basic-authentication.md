## מה ולמה?
שליחת בקשת HTTP עם אימות בסיסי היא תהליך שבו תכנתנים יוצרים קשר עם שרת באמצעות בקשה פרוטוקול HTTP עם משנה אימות בסיסי. פיתוחן של עיצוב דרך זה כדי להבטיח גישה מוגן כאשר משתמשים ניסיונייה כדי לגשת למידע נמצא על השרת.

## איך לעשות:
תוכנית דוגמה עבור שליחת בקשת HTTP עם אימות בסיסי בשפת גואנג. פתח את הקוד הבא במקרה שלך:

```Go
package main
import (
	"fmt"
	"net/http"
	"log"
	"io/ioutil"
)
func main() {
	url := "https://example.com/api/someendpoint"
	method := "GET"
	client := &http.Client {
	}
	req, err := http.NewRequest(method, url, nil)
	if err != nil {
		log.Fatal(err)
	}
	req.SetBasicAuth("username", "password")
	res, err := client.Do(req)
	if err != nil {
		log.Fatal(err)
	}
	defer res.Body.Close()
	body, err := ioutil.ReadAll(res.Body)
	if err != nil {
		log.Fatal(err)
	}
	fmt.Println(string(body))
}
```
