# מה ולמה?

פריסת תאריך ממחרוזת הוא תהליך בו מתרכזים ערכים מיד כדי ליצור תאריך תקף. מתכנתים מבצעים פעולה זו כדי להפוך את הנתונים לקלים יותר לניתוח והפעלה בתוכניות מיוחדות.

תהליך הפריסה נחשב לחשוב במיוחד בכדי לטפל בעבודה עם תאריכים בתכנית, ולכן יותר נוח למתכנת להשתמש במתודות שיכולות לבצע את התהליך באופן אוטומטי.

# איך ל: 

```Go
// משתנה בחלק העליון של הקוד לציון תאריך
str := "30/06/2021"

// אופציה אחת לפריסת התאריך היא ע"י פונקציה פנימית בשם Parse ושימוש בתבנית פריסה תקפה
t, _ := time.Parse("2/01/2006", str)

// תוכנן עבור פלט
output := "30th June, 2021"

// נדפיס את התוצאה 
fmt.Println(t.Format(output))
```

# מעומק לעומק

הפריסה של תאריך ממחרוזת התוכנה נבנתה כחלק מתוכניות המונחתות על סמך תאריכים ושעות. ישנם אלגוריתמים נוספים לפריסת תאריך ממחרוזת, אך פונקציות פנימיות כמו Parse נחשבות לאחת האפשרויות הנפוצות והפשוטות ביותר לבצע את התהליך.

פריסת תאריך ממחרוזת משתמשת בפורמט תאריך מבני קושרים, כמו יום/חודש/שנה, כדי לספק את המלל המדויק של התאריך. פונקציות אחרות בשפות תכנות נפוצות יכולות גם לבצע את התהליך באופן יישומי.
