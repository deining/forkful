## מדוע

string קטן מהאותיות הגדולות, אז ייתכן שתרצה להמיר אותם לכתבי סגל

איך 
קוד לדוגמה להמרת string לכתבי סגל ופלט לדוגמה: 

 ```
// קבלת string מהמשתמש
str := "GOLANG IS AWESOME"

// המרת ה string לכתבי סגל
str = strings.ToLower(str)

// פלט: "golang is awesome"
fmt.Println(str)
```

כאן אנו משתמשים בפונקציה strings.ToLower () שמקבלת את הכניסה המקורית ומחזירה את הכניסה בכתבי סגל. בכך ניתן לבצע המרה פשוטה ומהירה של string לכתבי סגל.

## חקירה עמוקה

תכונת הכניסה בכתבי סגל יכולה להיות שימושית לעשות חיפוש ומיון ב string, כיוון שתאפשר לנו להתעלם מהפעלות האותיות ולרקוד רק על התוכן האמיתי של המחרוזת.

כמו כן, כאשר מתחברים לשירותי טכנולוגיות כמו API, ייתכן שתרצה להשתמש בכתבי סגל כדי לפתוח אתחולים מטעות כגון מופע של השם המשתמש של המשתמש במקום של שם משתמש מדויק.
