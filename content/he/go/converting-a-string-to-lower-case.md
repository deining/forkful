# למה: 

מודעות היומיום שלנו נמצאות בכתיבה באותיות גדולות וקטנות. כאשר אנו פותחים קבצי טקסט או אתרים, אנו יכולים לראות כתבי הדפס גדולים וקטנים שונים. כדי לפתור את זה ולהפוך את הטקסט לברור יותר, נדרש להמיר את הכתב לאותיות קטנות. במאמר הזה נלמד כיצד להמיר מחרוזת לאותיות קטנות בשפת גו.

## איך לעשות זאת: 

כדי להמיר מחרוזת לאותיות קטנות בשפת גו, ישנם שני שיטות פשוטות. הראשונה היא להשתמש בפונקציה ToLower של חבילת strings. השיטה השנייה היא לבצע את ההמרה ידנית באמצעות לולאת חלק לולאת איטרציה על המחרוזת ולהחליף כל אות גדולה באות קטנה בעזרת הפונקציה ToLower של חבילת strings.

```Go
package main

import "fmt"
import "strings"

func main() {
	// השימוש בפונקציה ToLower
	hello := "Hello, עולם!"
	fmt.Println(strings.ToLower(hello))

	// המרה ידנית
	helloManual := "Hello, עולם!"
	newHello := ""

	for _, letter := range helloManual {
		if letter >= 'A' && letter <= 'Z' {
			newHello += strings.ToLower(string(letter))
		} else {
			newHello += string(letter)
		}
	}

	fmt.Println(newHello)
}
```

תוצאה: 

hello, עולם!

hello, עולם!
