# מה ולמה?

המרת מחרוזת לאותיות קטנות הינה פעולה נפוצה בתוך תחום התכנות בשפת Go. זהו פשוט מנגנון שמאפשר לשנות את כל האותיות במחרוזת לאותיות קטנות, ולהפוך אותה לנוחה יותר לשימוש והשוואה עם מחרוזות אחרות. תכנתפאי עושים את זה בכדי להתאים לתבניות סטנדרטיות או לבצע השוואות כדי לוודא ששתי מחרוזות הנראות זהות בעין האדם גם הן זהות למחשב.

## איך לעשות?

```
Go strings.ToLower("HELLO, WORLD!") 
```
פעולה פשוטה כזו תחזיר את התוצאה הבאה:

```
hello, world!
```

בפעם הראשונה, מחרוזת המוגדרת ככזו תתאמץ לכתוב באותיות גדולות. אם תחשוב עליהם כאנשים, הפעולה תפעל חיסור ותעשה אותם יותר שקטים ונעימים.

## צלילה לעומק

קשר היסטורי

 במקור, פעולת ה switch-קייס הייתה נקראת "פעולת טרנספורמציה" והיא הייתה נחשבת לאחת הפעולות הפשוטות והחשובות בתוך משפת Go. אולם, עם השנים, הפעולה הזו נחשבה לא כל כך חשובה והתחום של התכנות נקבע על ידי הכעס.

אלטרנטיבות

קיימות כמה אפשרויות לבצע המרת מחרוזת לאותיות קטנות בתוך התחום של Go. לדוגמה, אפשר להשתמש בפעולת strings.ToLower() שנתתי לעיל, או להשתמש בפעולה strings.Map() המאפשרת למדוד את המחרוזת ולהחליט מה לעשות עם כל אות בנפרד.

למרבה המזל, בגרסה הנוכחית של Go, פעולת הווסת בין המחרוזות התאימה לפרמטר הראשון, אך כמובן שאפשר להתייחס למצבים שלגרסה הישנה ולהשתמש בטכנולוגיות חדשות שנמצאות זמינות בשפת Go.

פירוט היישום

פעולת ההמרה לאותיות קטנות מיוחדת בשפת Go זהה לפעולת ההמרה לאותיות גדולות. מפני שמחרוזות נמצאים בתוך ערכי שמות, ניתן להשתמש בפעולה לא משתנה כדי למצוא את ההמרה המתאימה לאותיות קטנות. אולם, הפעולה הזו אינה חובה ולא תישמש כדי לעשות יותר מדי פערי.
