## למה

יכול להיות שיצור קובץ זמני יהיה כדאי במצבים שונים, כגון כאשר מתבצעת עיבוד נתונים וצריך לשמור את התוצאות בקובץ זמני לפני שמועברות לזיכרון הראשי.

## איך לעשות

ראשית, נזכיר כי יש להוסיף את החבילה "os" לרשימת החבילות המובאות, לפני הכתיבת הקוד. לצורך יצירת קובץ זמני נשתמש בפונקציית "TempFile" שבחבילת ה-OS.

```Go
import "os"

func main() {
    file, err := os.TempFile("", "example.*.txt")
    if err != nil {
        fmt.Println(err)
        return
    }
    // write to the temporary file
    _, err = file.WriteString("Hello, World!")
    if err != nil {
        fmt.Println(err)
    }
}
```

כאשר נריץ את הקוד הנ"ל, נקבל כתוצאה קובץ זמני עם שם מיוחד המתחיל במילת "example" ומסתיים ב"txt". בנוסף, בשורת הקוד "writeString" נכתוב טקסט לקובץ הזמני המיוצר.

## Deep Dive

כדי להבין עוד על יצירת קובץ זמני ננסה להבין כיצד הפונקציה "TempFile" פועלת. למעשה, הפונקציה מורכבת ממספר שלבים:

1. יצירת קובץ זמני חדש עם שם מיוחד.
2. יצירת הקובץ בכתובת ההימצאות הזמנית שנתונה ע"י הפרמטר הראשון בפונקציה (במקרה שלנו, פשוט נתיק את הוכחת הקובץ הזמני).
3. אם כל הפרמטרים נכונים, הפונקציה מחזירה את הקובץ שנוצר ואת השגיאה אם נתקלת.
