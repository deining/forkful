## למה 

יצירת קובץ זמני היא כלי חשוב בכתיבת קוד בשפת Go, המאפשר לנו לטפל במידע שמתעכב ולעשות שינויים בשקט ובקלות. מכיוון שהקבצים הזמניים נמחקים אוטומטית לאחר שימושם, הם מושלמים לשימושים קצרים כמו בדיקת עקביות הקוד, כתיבת קבצי לוג או בניית ממשקים כדי לבדוק את הפעולות של פונקציות אחרות.

## כיצד לבצע יצירת קובץ זמני בשפת Go

```Go
package main

import (
    "fmt"
    "io/ioutil"
    "os"
)

func main() {
    // יצירת קובץ זמני ריק
    file, err := ioutil.TempFile("", "example")
    if err != nil {
        panic(err)
    }
    // הדפסת שם הקובץ
    fmt.Println("שם הקובץ הזמני: ", file.Name())
    // כתיבה לקובץ
    data := []byte("תוכן לקובץ הזמני")
    if _, err := file.Write(data); err != nil {
        panic(err)
    }
    // סגירת הקובץ על מנת להשחיל את השינויים
    if err := file.Close(); err != nil {
        panic(err)
    }
    // ניקוי הקובץ הזמני אחרי השימוש
    defer os.Remove(file.Name())
}
```

פלט:

שם הקובץ הזמני: /tmp/example132130602

כאשר תפעול את הקוד תקבל את הקובץ הזמני עם התוכן שהוזן.
