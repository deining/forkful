## מה ולמה?
יצירת קובץ זמני היא פעולה שמאפשרת למתכנתים ליצור קובץ שימושי בזמן הרצת התוכנית. הקובץ זמני מתאים לשימושים כגון: שמירה של מידע זמני, יצירת קובץ בזמן הרצת התוכנית ועוד.

## כיצד ל:
כדי ליצור קובץ זמני בשפת Go, יש להשתמש בפקודת ```ioutil.TempFile()``` ולציין את התיקיה הזמנית בה יופץ הקובץ. לדוגמה, אם נרצה ליצור קובץ זמני בתוך התיקיה temp, נשתמש בפקודת הבאה:

```
f, err := ioutil.TempFile("temp", "example")
```

במקרה זה, הקובץ הזמני יקבל שם כמו "example123456", כאשר המספר האחרון יהיה רנדומלי.

אם נרצה לשנות את התיקיה הזמנית לתיקיה אחרת, נוכל להשתמש בפקודת ```ioutil.TempDir()```:

```
dir, err := ioutil.TempDir("", "example")
```

במקרה זה, התיקיה הזמנית תיקבע לתיקיה שנציינה בפקודה ויתווסף אליה גם קובץ זמני עם שם כמו "example123456".

## צלילה עמוקה:
יצירת קובץ זמני פועלת כדי לאפשר למתכנתים ליצור קבצים זמניים ולגשת אליהם בזמן הרצת התוכנית. זאת נחשבת לפעולה קלה יותר משימוש בקבצים קיימים, כיוון שהיא מאפשרת גם שימוש בקיצורי משתנים ובפקודות שלשום ינצפו פעם אחת וזהו.

אם אינך משתמש בשפת Go, תוכל ליצור קובץ זמני גם באמצעות שפות תכנות אחרות, כגון C וגם באמצעות פקודות של מערכות הפעלה בשביל ליצור קובץ זמני.

כאשר יוצרים קובץ זמני, יש לשים לב להסימן (safely), שזה מצוין בתיעוד הקוד כדי לוודא שהקובץ יימחק בסופו של דבר.
