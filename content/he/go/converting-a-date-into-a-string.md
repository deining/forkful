## למה 

לא משנה אם אתם מתחילים את הדרך שלכם בתכנות בשפת גו או שאתם מכירים בה היטב, תמיד תצטרכו להתמודד עם משימה של ייצוג תאריך כמחרוזת. זהו תהליך מרכזי בתכנות שמאפשר לנו להציג תאריך בצורה נוחה וקריאה למשתמשים. במאמר זה, אנו נלמד איך להמיר תאריך למחרוזת בשפת גו באמצעות דוגמאות מעשיות.

## איך לעשות זאת

נתחיל עם דוגמה פשוטה של כיצד להמיר תאריך למחרוזת באמצעות הפונקציה `Format` של חבילת `time` בשפת גו. נצור תאריך חדש, שיוצג כתאריך היום, ונשמור אותו במשתנה `today`:

```Go
today := time.Now()
```

כעת, נשתמש בפונקציה `Format` כדי להמיר את התאריך למחרוזת. נציין את התבנית הרצויה שבה נרצה להציג את התאריך, המופרדת באמצעות תוי תיבות והתאריך יוצג על פי הפורמט הנתון. לדוגמה, אם נרצה להציג את התאריך בפורמט "DD-MM-YYYY", נשתמש בפונקציה כדי ליצור את המחרוזת הרצויה:

```Go
formattedDate := today.Format("02-01-2006")
```

במשתנה `formattedDate` נמצאת כעת התאריך המרוצה בתבנית המבוקשת. נוכל להדפיס את התאריך באמצעות הפונקציה `Println` של חבילת `fmt`:

```Go
fmt.Println(formattedDate)
```

הפלט שלנו יהיה:

```bash
11-09-2021
```

כמו כן, ניתן להשתמש בפונקציה `Sprintf` כדי להמיר את התאריך למחרוזת ישירות בתוך הדפסה שלנו, באמצעות תבנית סטטית:

```Go
fmt.Sprintf("Today is %s", formattedDate)
```
