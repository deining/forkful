## למה

מתייחסים לתאריך בתוכנות רבות כדי לנהל נתונים של זמנים יצירה או שימוש ברמזורי זמנים, וגם כדי להציג תאריכים למשתמשים באופן מובן להם. להמיר תאריך למחרוזת הינה תהליך נפוץ בעבודה עם תאריכים, ובכתיבת קוד בשפת Go זה נעשה בקלות ובפשטות.

## איך לעשות זאת

 ```Go 
package main 

import (
	"fmt"
	"time"
)

func main() {
	// יצירת משתנה שמכיל תאריך ושעה נוכחיים
	currentTime := time.Now()

	// המרת התאריך למחרוזת בפורמט מסוים
	dateString := currentTime.Format("02 January 2006")

	// הדפסת התאריך כמחרוזת
	fmt.Println("תאריך נוכחי:", dateString)
}
```

תוצאה:  `תאריך נוכחי: 25 אוקטובר 2021`

על מנת להמיר תאריך למחרוזת בשפת Go, אנו משתמשים בפונקציה `Format()`, שמקבלת כארגומנט את פורמט המחרוזת הרצוי לתאריך. הפורמט של תאריך מגדיר את שימוש האותיות הקיבוציות על התאריך והזמן לייצוג המקובל.

כאן נשתמש בתאריך ושעה נוכחיים כדי להדפיס תאריך בפורמט `02 January 2006`, המציג את התאריך בתבנית "יום חודש שנה". כל אות מייצגת מרחק בין המשתנים. לדוגמה, `02` מייצג את יום החודש, `January` מייצג את החודש במלאו ו`2006` מייצג את השנה. חשוב לשים לב שלפני כל אותר יש רווח כדי להקפיד על התבנית במיקומה המתאים.
