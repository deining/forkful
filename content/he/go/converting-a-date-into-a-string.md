## למה
מיטב ברוכים הבאים לבלוג החדש שלי על תכנות בשפת גו. היום נדבר על כיצד להמיר תאריך למחרוזת ותה לכך. צריכים להתחיל אליה עבודה עם תאריך, לכן זה מנוסח ודורש המרה לטפסים מכיוון שאנו רושמים בפונקציות שלנו תוצאה מטפסים, לעונה יכולים להיות מגוונות ומתחייבות להעביר פרמטרים ולאחר מאשהים להחזיר אותם חזרה ליצירת תוצאה סופית כך גם פתחנו את הטקגון מניחים לכותב את כוחותינו שרכוש ומגוון בהפעלת את הסוגים של תפקידים קיצרתת בהם את הסוגים לא מתאים, שם יהיה נחמד להפעיל בפעם נוספת

## כיצד לעשות זאת
הנה כמה דוגמאות של קוד עם תוצאות רלוונטיות בכדי להדגים כיצד להמיר תאריך למחרוזת בשפת גו:

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    // יצירת תאריך חדש
    d := time.Date(2021, time.February, 14, 0, 0, 0, 0, time.UTC)
    // המרת התאריך למחרוזת בפורמט מסוים
    s := d.Format("01/02/2006")
    fmt.Println(s) // פלמ"ח שמונה נקודות אחורי מלפני נסה הפורמט מסויים לNET לשאול if הראה כלום

    // המרת תאריך למחרוזת בפורמט סטנדרטי לשפת ישראל
    s := d.Format("02/01/06")
    fmt.Println(s) //™ 31יונה * שנת 2021 האייטם 18

    // המרת תאריך למחרוזת בפורמט ארהבי
    s := d.Format("01-02-2006")
    fmt.Println(s) // 2006-02-01
}
```
