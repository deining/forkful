## למה
מיומנות בהמרת תאריך למחרוזת חשובה עבור מתכנתים שעוסקים בפיתוח אפליקציות ואתרי אינטרנט בשפת התכנות Go. כתיבת תאריך בפורמט של מחרוזת היא נדרשת כאשר מעוניינים להציג את התאריך בצורה נוחה לקריאה עבור המשתמשים או לשמירתו בקבצים או במסדי נתונים.

## כיצד לעשות זאת
למרבה המזל, התכנית הרצויה כבר כלולה בחבילות המובנות של שפת תכנות Go, מה שמקל על כתיבת הקוד והופך את התהליך לפשוט יותר. כדי להמיר תאריך למחרוזת, אנו צריכים להשתמש בפונקצייה  `Format` בחבילת `time` ולספק לה את הפורמט הרצוי. לדוגמה:

```Go
package main
import "fmt"
import "time"

func main() {
    // מקדמת את פונקציית זמן
    t := time.Now()

    // מייצג את תאריך היום בפורמט של מחרוזת
    dateString := t.Format("01/02/2006") 
    fmt.Println(dateString) // פלט: 08/23/2021
}
```

ישנם מספר אפשרויות לפורמט המחרוזת הנוצרת, בין אם לפי התאריך, החודש או השנה. בנוסף, ניתן להציג גם את האזור הזמני ואת השעה בפורמט של מחרוזת עם שימוש בפונקצית `Format` ובתבנית הנכונה.
