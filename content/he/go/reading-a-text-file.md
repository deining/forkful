## מה ולמה?
קריאת קובץ טקסט היא פעולה שמאפשרת לקרוא ולעבוד עם מידע שמאורגן בצורה של טקסט פשוט. תוכנתנים רבים, כולל מתכנתים בגו, כותבים ומשתמשים בטכניקה זו כדי לקרוא ולעבד מידע מקורה כמו קבצי טקסט.

## איך לעשות?
גו מספק פונקציות רבות לקריאת קבצי טקסט בשפה התכנותית הוותיקה והמוכרת וואס. נהדר, ניתן לקרוא קובץ טקסט באמצעות הפונקציה `bufio.NewScanner`, שמכילה את האלגוריתם שמאפשר קריאת כל שורת טקסט בקובץ. עבור קובץ גדול יותר למשל, ניתן להשתמש בפונקציה `ioutil.ReadFile` שתקרא את כל הקובץ לזכרון ותאפשר לנו לבצע את הפעולות הנדרשות על מידע מלא.

```go
// דוגמא קוראת קובץ טקסט בעזרת ביבליות בפיות
func main() {
	file, err := os.Open("file.txt")
	if err != nil {
		log.Fatal(err)
	}
	defer file.Close()

	scanner := bufio.NewScanner(file)
	for scanner.Scan() {
		fmt.Println(scanner.Text())
	}

	if err := scanner.Err(); err != nil {
		log.Fatal(err)
	}
}
```

## חפירה עמוקה
בהתחלה, כשהייתה קיימת רק שפת תכנות "קשה לגמור" הייתה תפורה כך שתתבצע בעזרת רכיבים / תכונות רבים שעוזרים להפעילה, כולנל אפשרו לחבור רכיב תכנתי להפעלת קבוצת שרפרפים ותוכנית אחרים ( למשל נפתח תערובת קולונוס תכנתי שונה) שכך מכיוון שכללים.

רוב כל תכנתניים רק כתבו על מנת להפעיל את זו במקרים מסויימים, כמו כל התכניות הופכנות להיות טיקי לחלקים הנתינים, או הנתונות המסוימים שהפעלו והנתונים העבירו כמשווקי.
