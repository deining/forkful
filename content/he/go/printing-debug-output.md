# למה

עדיין לא מרוצים עם התוצאות של הקוד שכתבתם? אולי ייצא לכם להוסיף כמה בדיקות ולקבל פרטים נוספים על הפעולות שמתבצעות בקוד שלכם. במאמר זה נלמד איך להדפיס פלט לניפוי בפורמט של Go.

# כיצד לבצע ניפוי בפורמט של Go

הדפסת פלט לניפוי בפורמט של Go מאפשרת לנו לקבל במערכת ויזואלית את התהליך של הקוד שלנו. הנה כמה פעולות שניתן לבצע על מנת להדפיס תוצאות לניפוי:

```Go
// פונקציית הדפסה לניפוי
func debugPrint(message string) {
  fmt.Println("DEBUG: " + message)
}

// ניתן להדפיס כל משתנה שבא לנו
debugPrint("תהליך הקוד הפעיל פעולה A והפסקתה")

// ניתן להדפיס ערך בעזרת סימן %
num := 10
debugPrint(fmt.Sprintf("הערך של num הוא %d", num))

// ניתן להדפיס תווים מתוך מערך עם הקוד הבא
arr := []string{"א", "ב", "ג"}
for _, letter := range arr {
  debugPrint("אותיות מהמערך: " + letter)
}
```

פלט של הניפוי יופיע בסדר הבא:

```
DEBUG: תהליך הקוד הפעיל פעולה A והפסקתה
DEBUG: הערך של num הוא 10
DEBUG: אותיות מהמערך: א
DEBUG: אותיות מהמערך: ב
DEBUG: אותיות מהמערך: ג
```

# העמקה

כאשר אנחנו מדפיסים נתוני ניפוי, חשוב לסמנם בצורה ברורה ומזהה את האזורים השונים בקוד שלנו. ניתן להשתמש בסוגי מידע שונים עם הפעולות הבאות:

- `fmt.Printf()` - מחליף את הסימלים עם ערכים מתאימים המופיעים אחריו. לדוגמה: `%d` - מתאים לסוג מספר שלם, `%f` - מתאים לסוג מספר עם נקודה צפהת
