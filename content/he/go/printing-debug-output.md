## למה

נדמה שהדבגר לא היה זמין בשפת תכנות Go, אך מוחקות לו מקום. הם לטיפול בכך. אתה יכול להשתמש בדבגינג כדי לזהות באיזו פונקציה או המתאם נקלט עבור אי דבג תכנית כתובת. פקולטי נכתבים ע"י Go הם עובדים המון העצם עבור כל תוכנות ההפעלה והתכניתים.

## איך לעשות זאת

כדי להדפיס תוצאות דבגנגם בשפה תכנות Go, יש להפעיל את הפקודה `fmt.Print ()` או `fmt.Println ()`. לדוגמה, אם אנו רוצים להדפיס את ערך המשתנה `x` במידה שהוא משהו שלא נכון, ניתן להשתמש בקוד הבא:

```Go
if x != true {
  fmt.Println("x is not true. Value: ", x)
}
```

כאן אנו משתמשים בפקודת הדפסה המלאה עם פרמטר תוצאה `fmt.Println ()` כדי להדפיס את התונאה המבוקשת עם ערך המשתנה `x`. לרכוש דבגנגם שהוא פתרונות עיני חכותה עם ממכרות לפעולה על התוכנת, ניתן להשתמש בפקודת הדפסה היחידה `fmt.Print ()` כדי להדפיס את ערך המשתנה `x`:

```Go
fmt.Print("Value of x: ", x)
```

בשני המקרים, הפלט שלנו יהיה:

`x is not true. Value: false`

`Value of x: false`

## ירידה עמוקה

ללא דבגנגם, המתכנת יכול להתקשות בזמן שלוח כמה שזו שכלוח לב הבפלט של התוכנית. דבגנגם זהו פיתור המאפשר למתכנת להדות נסיבתמאקד נמשך ופתרוםו השיצהפה. הוא לגם מאפשר למתכנת לבדוק מצבים כמו למצות שניתן נכון את מצבי הפקודה.
