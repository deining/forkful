# למה

כאשר אתה מפתח תוכניות ב-Go, ייתכן שתצטרך לקרוא פרמטרים שהועברו לתוכנית שלך מהשורת פקודה. קריאת פרמטרים של שורת פקודה היא חלק חשוב מתפקיד התכנות ומאפשרת לך להתאים את התוכנית שלך לצורך הספציפי של המשתמש.

# איך לעשות זאת

אתה יכול לקרוא פרמטרים מהשורת פקודה ב-Go באמצעות הפונקציה `os.Args`. מטרתה של הפונקציה היא להחזיר מערך של מחרוזות, כאשר כל מחרוזת מייצגת פרמטר אחד מהשורת פקודה. נסו את הדוגמה הבאה:

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	args := os.Args[1:]

	fmt.Println("מספר פרמטרים בשורת הפקודה:", len(args))
	fmt.Println("הפרמטרים המועברים הם:", args)
}
```

כאשר תריץ את הקוד, תקבל את הפלט הבא (לדוגמה, נניח שתריץ `go run main.go hello world`):

```
מספר פרמטרים בשורת הפקודה: 2
הפרמטרים המועברים הם: [hello world]
```

כעת תוכל לגשת לפרמטרים שהועברו על ידי התוכנית שלך על ידי גישה לאינדקס של המערך `args`. לדוגמא, נניח שהנתיב לתוכנית שלך הוא `go run main.go /Users/foo/Desktop`:

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	args := os.Args[1:]

	fmt.Println("נתיב התוכנית הוא:", args[0])
}
```

כרגע השתמשנו במיקום הראשון במערך (`args[0]`) כדי לגשת לפרמטר הראשון שהועבר לתוכנית. ניתן להשתמש בערכים נוספים במערך יחד עם פקודות תנאי ולבדוק את הפרמטרים שנמצאים בארגומנטים.
