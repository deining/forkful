## למה

בעולם התכנות, לפעמים נתקלים בצורך לספק טקסט, מספרים או פרמטרים נוספים לאפליקציית Go במהלך הרצתה. במאמר זה, נכיר את דרך הקבלה והקריאה של פרמטרים משורת הפקודה בגו, אשר יכולה להיות כלי חזק בזמן פיתוח האפליקצייה שלנו.

## איך לעשות זאת

כדי לקרוא פרמטרים משורת הפקודה בגו, נשתמש בפונקציית `os.Args`. ניתן לראות במספר פרמטרים במקום אחד בדרך פשטונית:

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    fmt.Println(os.Args)
}
```

אם נריץ קובץ זה, נוכל לראות את הפרמטרים שהזנו משורת הפקודה כך:

```bash
go run command_line_args.go
# Output: [command_line_args]
```

ניתן גם לעבוד עם פרמטרים יותר מפורטים, כגון הסדר שבו הם נכנסים או לשנות את ערכם. לדוגמה, אם נרצה להדפיס רק את הפרמטר השני משורת הפקודה, נשתמש באינדקס של הפרמטר הנוכחי ונכניס את הערך הרצוי:

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    fmt.Println("My favorite color is", os.Args[1])
}
```

אם נריץ את התכנית הזו כפי שהיא מוגדרת, עלינו לספק פרמטר נוסף ממש שנכנס בין שני גרשיים, כך:

```bash
go run command_line_args.go "blue"
# Output: My favorite color is blue
```

ניתן גם לשנות את ערכי הפרמטרים שהוזנו על ידי משתמש כאשר האפליקציה מריצה. למעשה, יתכן שזה יתן לנו אפשרות ייצוגית או דרך לשנות את התקנון של האפליקציה בטיפול בפרמטרים נוספים. לדוגמה, אם נר
