## למה 

מחשבות ראשונות! אני בטוח שישנם רבים מכם שמתעניינים בתכנות וכבר מכירים את השפת תכנות Go. אז מדוע אנחנו מתעניינים בחישוב אורך של מחרוזת? הפתרונות הפשוטים לשאלה זו שאנו מעוניינים לבדוק את היכולות הגעגועים לשפת תכנות המכילה משחק, או לכל מי שרוצה לדעת אורכו של מחרוזת כלשהו.

## כיצד לעשות זאת 

בואו נתחיל לעבוד ונראה איך נוכל למצוא את אורך המחרוזת בשפת תכנות Go. 

```Go
package main

import "fmt"

func main() {
	str := "זהו דוגמה"
	fmt.Println(len(str))
}
```

בקוד הנ"ל, אנו מחשבים את אורך המחרוזת "זהו דוגמה" ומדפיסים את הערך שלו למסך. כאן אנו משתמשים בפונקציה "len" המחשבת את אורך המחרוזת. ניתן לעשות כך עם כל מחרוזת אחרת בקוד. כעת, נבדוק תוצאה: 

```
9
```

הפתרון מאוד פשוט וברור. ניתן גם להשתמש במידע חיצוני כמו קריאת נתיב שווה או לאורכו של המחרוזת, אך זה נושא לפני עמומות. כעת עם הפתרון שלנו, אנו יכולים לקבל המון מידע על נסיבות השגם יחד עם הצלחה!
