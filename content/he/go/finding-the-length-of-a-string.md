## למה

למה להתעסק בחיפוש אורך של מחרוזת? בתכנות בשפת Go, הגישה לאורך של מחרוזת היא חיונית וחשובה במיוחד כאשר עובדים עם מחרוזות גדולות ומורכבות. היכולת למצוא את אורך המחרוזת מאפשרת לנו לעבוד ביעילות עם הנתונים וליישם אלגוריתמים שונים על המחרוזות שלנו.

## איך לעשות זאת

כדי למצוא את אורך המחרוזת בגולנג, נוכל להשתמש בפונקציה len המובנית. לדוגמה:

```Go
str := "שלום עולם"
length := len(str)
fmt.Println(length)
// פלט: 10
```

הפונקציה len מחזירה את מספר התווים במחרוזת כמספר שלם. ניתן להשתמש גם בערכים מוצפנים ובהם כוללת למספר התווים במחרוזת כולה.

```Go
encodedStr := "eJxLT8nVKMHostZWBSXRAgATAwA="
length := len(encodedStr)
fmt.Println(length)
// פלט: 27
```

ניתן גם להשתמש בלולאה כדי לעבור על כל תו במחרוזת ולספור אותם יחד.

```Go
str := "Hello World"
count := 0
for range str {
	count++
}
fmt.Println(count)
// פלט: 11
```

## העמוק

בכדי להבין טוב יותר את הגישה למציאת אורך מחרוזת בגולנג, נדון מעט בפרטים. העיקרון המאחורי הפונקציה len הוא לנווט על תגים של הכתיבה הכתובית של המחרוזת ולספור את התווים שמכילים ככלולים בתגים אלו. זה יכול להיות לא טריוויאלי כאשר מתעסקים עם מחרוזות שמכילות תווים מיוחדים, כמו תווי חריגה או תווי רווח.

כמו כן, חשוב לזכור שהפונקציה len מחזירה את מספר התווים במחרוזת ולא את כמות הבתים.
