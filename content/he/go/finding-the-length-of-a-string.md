## מדוע

למה לדעת אורך של מחרוזת בקוד? בשפת גו, כמו בשפות תכנות אחרות, מחרוזות הם חלק חשוב מאוד מהתכנית. ייתכן שתרצה לדעת את אורך המחרוזת כדי לבצע פעולות נוספות או להשוות בין מחרוזות שונות. בכתיבת תוכניות מורכבות יותר, הבנת אורך המחרוזת יכולה להיות מאוד שימושית.

## איך לעשות זאת

הראיה הבסיסית למציאת אורך של מחרוזת היא להשתמש בפונקציה `len()` ולקרוא לה על המחרוזת המבוקשת. לדוגמה:

```Go
str := "שלום עולם!"
length := len(str)
fmt.Println(length)
```

פלט:

```
12
```

ניתן גם להשתמש בלולאת `for` כדי לעבור על כל תו במחרוזת ולספור את כמות התווים ידנית. לדוגמה:

```Go
str := "Hello, World!"
length := 0
for range str {
	length++
}
fmt.Println(length)
```

פלט:

```
13
```

אולם, השימוש בפונקציה `len()` יותר יעיל ומנצח במקרים רבים, בכיוון שהיא מתחשבת בתווים ברווחים וכו'.

## ייעול נמען

בכתיבת תוכניות מורכבות יותר, ייתכן שתרצה להשתמש בפונקציה `len()` יותר מפעם אחת. כדי למנוע את חזרת `len()` כמה פעמים, ניתן לצבור את הערך שהוחזר ולשלוח אותו כפרמטר לפונקציה אחרת. כמו כן, ניתן ליצור פונקציה משלנו שתקבל מחרוזת כקלט ותחזיר את האורך שלה. לדוגמה:

```Go
func getLength(str string) int {
	return len(str)
}

length := getLength("שלום")
fmt.Println(length)
```

פלט:

```
4
```
