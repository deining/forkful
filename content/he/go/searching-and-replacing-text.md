## למה

כשמתכנתים בשפת תכנות כגון Go, קיימת תמיד צורך לחילופי טקסט והחלפות בקוד המפתח. הפעולה הזו מאפשרת לנו לעקוב ולתקן בקלות שגיאות או לשפר את הקוד שלנו באופן מהיר ויעיל.

## איך לעשות זאת

בכדי לבצע חילוף טקסט בקוד Go, ניתן להשתמש בפונקציות שנמצאות בחבילת "strings" של שפת תכנות זו. הנה דוגמאות של כמה פונקציות חשובות לחילוף טקסט:

```Go
// העתקת נתונים מטקסט קיים
text := "שלום עולם"
replaceText := strings.Replace(text, "שלום", "היי", 1) // החלפת "שלום" ב-"היי" רק פעם אחת
fmt.Println(replaceText) // הדפסת הטקסט שקיבלנו: "היי עולם"

// בדיקת פסקה בתקינות של טקסט
paragraph := "גנב המליח אמר קנת." 
contains := strings.Contains(paragraph, "גנב") // בדיקה אם ישנה מילה "גנב" בטקסט
fmt.Println(contains) // הדפסת התוצאה: true

// חילופי פונקציה בטקסט
original := "הדגים במים הם חיים"
funcText := strings.Map(func(r rune) rune {
	if r == 'ה' {
		return 'ו'
	}
	return r
}, original)
fmt.Println(funcText) // הדפסת התוצאה: "ודגים במים ום חיים"
```
