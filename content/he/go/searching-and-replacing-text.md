## למה
בתכנות, החיפוש וההחלפה של טקסט הם כלי חיוניים לעיבוד ועיבוד נתונים. בשפת גו, ישנם מספר מתווכי חיפוש והחלפה שיכולים לסייע בהפיכת עבודה עם טקסט לפעולה יעילה ופשוטה יותר. במאמר הזה נלמד כיצד לכתוב קוד בגו שיכול לעזור בחיפוש והחלפת טקסט באופן מהיר ויעיל.

## איך לעשות זאת
בכדי לבצע חיפוש והחלפה של טקסט בגו, נשתמש בפונקציות השיש והחל, כאשר הנפוצה ביותר היא strings.Replace. לדוגמה, ננסה להחליף את המילה "פס" במילה "כוח" בתוך הטקסט הבא:

```Go
str := "כדי להשיג כוח, עליך להעמיד פס תורים מתאימים."

newStr := strings.Replace(str, "פס", "כוח", -1)

fmt.Println(newStr)
```

תוצאה:
```
כדי להשיג כוח, עליך להעמיד כוח תורים מתאימים.
```

ניתן לראות שפונקציית Replace החליפה את המילה "פס" במילה "כוח" בכל המקומות בטקסט. ניתן גם להגדיר את האופציה של כמות ההחלפות, במקרה שלנו השתמשנו בערך של -1 שמסמל חילוף בכל המקומות בטקסט.

אם נרצה לחלופין לחפש מחרוזת מסוימת ולהחליפה בערך אחר, נוכל להשתמש בפונקציית strings.Contains כדי לבדוק אם המחרוזת מופיעה בטקסט ולקבוע מתי להחליף אותה עם פונקציית Replace. לדוגמה:

```Go
str := "השבוע הייתי נמצא בבית החולים כי חליתי עם נזלתים חמורים."

if strings.Contains(str, "נמצא") {
    newStr := strings.Replace(str, "נמצא", "הייתי ב", 1)
    fmt.Println(newStr)
}
