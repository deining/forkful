## מדוע

מחשבות המועברות במשך הרבה זמן? אולי תוכל להשתמש בפונקציות של Go לחזור על נתונים שאתה מתעניין לקראת העתיד או העבר.

## איך לעשות זאת?

כדי לחשב תאריך בעתיד או העבר ב-Go, ניתן להשתמש בחבילת "time" המובנית בשפת Go. הנה דוגמא פשוטה שמדגימה כיצד לחשב את התאריך של 5 ימים בעתיד:

```Go
// מייבאים את חבילת הזמן המובנית
import "time"

// הגדרת תאריך נוכחי
now := time.Now()

// קבלת תאריך 5 ימים בעתיד
future := now.AddDate(0, 0, 5)

// הדפסת התאריך החדש
fmt.Println(future)
```

פלט:

`2021-06-19 11:16:59.2887252 +0300 IDT m=+4286.014117599`

ראשית, אנו מייבאים את חבילת הזמן המובנית של Go. לאחר מכן, אנו משתמשים בפונקציה `AddDate()` כדי להוסיף 5 ימים לתאריך הנוכחי. סופסוף, אנו מדפיסים את התאריך החדש שחושבנו.

בנוסף למקרים בעתיד, ניתן גם לחשב תאריך בעבר על ידי שימוש בפונקציה `Sub()` כדי להחסיר ימים מהתאריך הנוכחי.

## מעמיקים יותר

חבילת הזמן המובנית של Go כוללת גם פונקציות נוספות כמו `Add()` לחיבור זמנים, `Sub()` לחיסור זמנים ו-`Before()` ו- `After()` לבדיקת התאריכים. כמו כן, ניתן להשתמש בפונקציות נוספות כדי לקבל את השנה, החודש או היום של התאריך הנוכחי.

למידע נוסף על חבילת הזמן המובנית של Go, ניתן לעיין בתיעוד המלא של Time package במקור הרשמי של Go או לחפש בגוגל עבור "Go Time Package".
