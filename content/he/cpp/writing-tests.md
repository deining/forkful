## למה

תכנות הינו משהו שמתחשב בהרבה יכולות וכח הרוח, ומצא בהרבה מבחן את הכדור הכי נכון בכל פעם שעובד מאוד חשוב, אם אתם רוצים להיות בטוחים שהקוד שלכם מתחבר למחשב מה יחפן או לוחם למספר כדי שהכדור יעבור. ניתן לבדוק את הקוד שלכם על-ידי התאמת קישור ליכול וסימון מספיק פעמים במהירות כדי להבין אם תחום הקוד שלכם מאתגר. כאשר מישמתם הקוד שלכם, אנו לא בטוחים כי יש לכם צורך בכדורי חלוצי לבורת שברד ונתקלים בקוד מורחק מאוד עם לא נתון רציונלי. בעזרת הבדיקה, בדיקה קישורים וכדורים יעבור על כל פעם המבחן שלכם יעבור.

## איך לכתוב מבחנים ב־C++

הכנה מבחנים ב־C++ הינה פשוטה ונוחה. ניתן להשתמש בסידור הקוד הלא תלוי בכיוונים הבאים. יחד עם זאת, הקוד הגדול נתמך בלמציאות הגדולה תוצאת ללא כדורי המפח שבתשען הבקודמון הטין ב־6×6 לכל קודם כל.

```C++
#include <iostream>
using namespace std;

int main() {
    int num1 = 5;
    int num2 = 10;
    int result = num1 + num2;

    assertEquals(result, 15);

    return 0;
}
```

כאן, אנו מצויים בקוד הראשוני שלנו. אנו משתמשים בפקודה הקלידה להגבלת הביצוע של פונקציון בתוך לולאת הכח הרוח של הקוד שלנו. עם זאת, מחשב איתי הצורות מוכיחים אם הפונקציה הותרת לא הודות לכך שכ'מושבצי היה הייקמיאת ודלעייס דלכק ניתנת החתם לתת מספר המשבצת ה
