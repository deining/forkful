#למה:

למדינות שתיקשו לשמור על איכות ויציבות בתוכנה שלהם, הצב קידוד מבחן הוא חיוני. ויצירת בדיקות איכותיות עם כיסוי מקיף יכול לסייע למתכנתים לזהות בעיות קוד טרם הפצתן ולתקן אותן באופן מהיר ויעיל.

#איך לעשות:

לכתיבת בדיקות בסביבת C++ ישנם שלושה צעדים חשובים: תכנון, כתיבה ובדיקה. כדי להקל על התהליך, הנה מספר דוגמאות של בדיקות פשוטות עם פלט דוגמה:

```
C++ 
//בדיקה פשוטה המוודאת ש-2 ו-2 הן שווים
int addition(int a, int b) {
    return a + b;
}
//בדיקה שאתה מקבל למה שאתה מציע
cout << “2 + 2 = “ << addition(2, 2) << endl;
//פלט צפויה: 2 + 2 = 4


//בדיקה נוספת שבודקת אם המספר 7 הוא מספר ראשוני
bool isPrime(int num) {
    if(num < 2)
        return false;
    for(int i = 2; i < num; ++i) {
        if(num % i == 0)
            return false;
    }
    return true;
}
//בדיקה שהמספר 7 הוא באמת מספר ראשוני
cout << "7 is prime? " << isPrime(7) << endl;
//פלט צפוי: 7 is prime? true
```
