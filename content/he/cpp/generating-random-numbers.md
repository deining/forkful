## למה
גנרטורים אקראיים הם כלי חשוב בתכנות שכוללים יכולות שונות כמו טסטרים, הגרלת ערכים רנדומליים ועוד. בכתיבת קוד לפעמים נדרש ליצור ערכים רנדומליים ואז גנרטורים אקראיים הם הטובים ביותר כדי לספק את הצורך הזה.

## איך
כדי ליצור ערכים רנדומליים בספריה הסטנדרטית של C++ נשתמש בפונקציית "rand()" וכדאי להתחיל עם זריקת זירה לפני שנתחיל להשתמש בזה. למדו על ההתחלה והגדרות נפוצות בקוד שלצד נושא הערכים הרנדומליים כדי לבדוק פעולת גננרטור זירות. כאשר נגדיר את הפונקציה "rand()" נחשב לשפן כנושא בשל חוסרו של קיבולת שיטתיות. תכנית זה יוצר ערכים רנדומליים שהופיעו בתחום מקוון מ- 0 עד ערכי reinterpret_cast<int>.

```
#include <iostream>
#include <cstdlib>
#include <ctime>

// גדירות ערך רנדומלי
const int START = 0;
const int END = 10;

int main()
{
    // כנס במתנד
    srand(time(0));

    // להדפיס ערכים רנדומליים
    for (int i = 0; i < 5; i++) {
        int randNum = START + (rand() % (END - START + 1));
        std::cout << randNum << std::endl;
    }

    return 0;
}
```

פלט:

```
3
8
2
0
5
```
