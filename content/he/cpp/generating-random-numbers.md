כמה טיפים לכתיבת קוד עם מספרים אקראיים ב - C++

## מדוע לכתוב קוד עם מספרים אקראיים?
מספרים אקראיים משמשים בתכנות בשפת C++ כדי לייצר נתונים אקראיים בצורה המייצג תוצאות שאינן תלויות בזמנים או אירועים חיצוניים. נתונים אלה ניתן להשתמש בהם למטרת ניסויים, מבחנים, או כל מקום שנדרשת שחרור נתונים אקראיים.

## הכנה
### טיפול בזיכרון עם srand()
כדי להשתמש במספרים אקראיים ב - C++ נצטרך להשתמש בזיכרון לקביעת נתונים. תחילת כלה תכנית, נשתמש בפונקצית srand() כדי לאתחל הגנרטור שלנו לפעול על הזיכרון הקבוע של המחשב שבו התכנית רצה. הפונקציה srand() מקבלת כקלט את הזמן הנוכחי של המחשב כך שלרחוץ נתונים חדשים אנו נוכל להשתמש בזמן כקלט של הפונקציה.
```C++
#include <iostream>
#include <cstdlib>
#include <ctime>

int main()
{
    srand(time(0));

    // יצירת מספר אקראי בין 0 ל - 100 והדפסתו
    int num = rand() % 101;
    std::cout << num << std::endl;

    return 0;
}
```
הפלט עבור קוד זה ייראה משהו כזה:
```
57
```
