## מדוע

כלי יצירת מספרים אקראיים הוא חשוב מאוד בתכנות כדי ליצור הפרדה וסדר אקראי בין הערכים השונים שאנחנו משתמשים בהם. בעקבות כך, הוא משמש כלי עזר עבור אלגוריתמים שונים ומאפשר לנו לעבוד בצורה יעילה יותר כאשר אנחנו עובדים עם נתונים אקראיים בתוכניות שלנו.

## איך לעשות זאת

כדי לייצר מספרים אקראיים בשפת C++ נשתמש בספריית הפונקציות המובנות <random>. נתחיל על ידי כתיבת שורת הכותרת הנדרשת:

```C++
#include <random>
```

לאחר מכן נשתמש בפונקציה rand() כדי לייצר מספרים אקראיים בתוך תכנית הC++ שלנו. כדי להגביל את טווח המספרים שניוצרים, נשתמש בפונקציות הפיתון srand() ו־rand() כאשר rand() מחזיר מספר אקראי בטווח של 0 עד הערך המרבי שנכנס ל־rand().

```C++
// מדפיס מספר אקראי בין 0 ל־9
cout << rand() % 10 << endl; 

// מייצר מספר אקראי בין 1 ל־6
cout << (rand() % 6) + 1 << endl; 
```
