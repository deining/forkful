## למה

למה אפשר להתעסק ביצירת מספרים אקראיים? כי יצירת מספרים אקראיים היא חלק חשוב מתהליך התכנות ויכולה לשמש במגוון רחב של יישומים, כגון משחקים, מודלים סטטיסטיים וטכנולוגיות למידת מכונה.

## איך לעשות זאת

ליצור מספרים אקראיים בשפת סי++ ניתן בעזרת פעולת הגרירה של ספריית הגנרטורים הכללית כמו `rand()` ו- `srand()` או על ידי שימוש במחלקות של ספריית ה-`<random>`. הנה דוגמא קצרה של כיצד ניתן לגרום לתוכנית להדפיס מספר אקראי בין 1 ל-100:

```C++
#include <iostream> 
#include <cstdlib> 
#include <ctime> 

using namespace std;

int main() 
{ 
    // גרירת שנות הזמן העכשוויות כגרעין לגנרטור 
    srand(time(0)); 

    // הדפסת מספר אקראי בין 1 ל-100 
    int randomNum = rand() % 100 + 1; 
    cout << "המספר האקראי הוא: " << randomNum << endl; 

    return 0; 
} 
```

הפלט של הקוד הנ"ל ייראה כמו זה:

```
המספר האקראי הוא: 42
```

ניתן גם לשנות את טווח המספרים האקראיים על ידי שימוש בפונקציות נוספות כמו `mt19937` או `uniform_int_distribution`.

## להעמיק
