התמודדות עם הביצוע של המרת מחרוזת לאותיות קטנות בשפת ++C

## כמה תודה למה

המרת מחרוזת לאותיות קטנות היא פעולה נפוצה בתכנות המחשבים היום. יתרון ייחודי של התמודדות זו הוא היכולת לשלוט בפרינטרים ובאחת הפעולות הנוספות המובנות עם התמודדות זו. דרך נוספת היא עם יכולת תכנות במחשב.

## איך לעשות את זה

תחילה, נצטרך לכתוב קוד שיכול לקבל מחרוזות מהמשתמש ולהמירן לאותיות קטנות. לעשות זאת נשתמש בפקודה ```tolower()``` שקיימת בשפת ++C. הנה דוגמא לקוד:

```C++
#include <iostream>
#include <cstring>
using namespace std;

int main() {
    char str[100];
    cout << "נא להזין מחרוזת: ";
    cin >> str;

    // להמיר את המחרוזת לאותיות קטנות עם פקודת tolower()
    for (int i = 0; i < strlen(str); i++) {
        str[i] = tolower(str[i]);
    }

    // להדפיס את המחרוזת החדשה
    cout << "המחרוזת באותיות קטנות היא: " << str;

    return 0;
}
```

הנה חלק מהפלט האפשרי עבור הקוד הנ"ל:

> נא להזין מחרוזת: HELLO WORLD
> המחרוזת באותיות קטנות היא: hello world

כפי שאתם יכולים לראות, פקודת ```tolower()``` ממירה את האותיות הגדולות לאותיות קטנות במחרוזת שאנו נכנסת ויצאת. בנוסף, אנו יכולים לראות שאוופציה זו עובדת גם כאשר המחרוזת כוללת מספרים.
