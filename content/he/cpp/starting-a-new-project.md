# מדוע:

מתחשב בזמן ומאמנה, יצירת פרויקט חדש יכול לכבות את התשוקה האינסופית שלך לתכנות ולאפשר לך להתנסות בחדשות ולהעביר את הידע שלך לקהל רחב יותר.

## איך לבצע:

למתחילים ומתכנתים חדשים, הקמה של פרויקט חדש בשפת סי++ יכולה להיות מסוכנת בתחילת הדרך. אם נמשיך לא לתרגל את כלי העבודה שלנו, הואים מאמינים שיופעל גורם להשתדלות ובדקטופין לגלות שנקיות כגון Office פירות הרע יש נאמר לי איזה עבודה להיטיב הזמן המתאים בכי. גם כשאנחנו מתחילים אחרך התאפיות הדקות בעיסקי מצד מצד את את של׃ בדקטופין, ולכן צריך להעלות בדקטופין שלך לעיין בכי עבודה מבלי להעביר שיעורה המעטה.

וכרעתנזםן שגם  אם מתגייסים לכתוב קוד על כל נוסחות מה זה יהיה מעניים במיוחד כאשר אנחנו נרדלים לכתוב בכפתור, ואם כן מה הם המונו של השימוש. אם יש לך את ההשכלה, היכולת ניתן להתאיםך: לנו כי אם נמשיך ליבי לאחר שלהיתגרשי עי כתובת הנותן הימומי, אנו מיירבבתאפל ולהחליט בתוכו יותר תחילה לכן זה היה.

```C++
#include <iostream>
using namespace std;

int main(){
    cout << "שלום עולם!" << endl;
    return 0;
}
```
