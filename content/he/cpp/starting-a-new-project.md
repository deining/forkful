# למה

למה כדאי להתחיל פרויקט חדש בסגנון שפת התכנות "C++"? כמה מציאותות: כדאי להבין את השפה הזו למרות המאמצים, מראה שלה הוא הכוח שלה. בהתחלה פשוט נתחיל עם ״היי עולם״ נעבוד עד שנוכל לבנות משהו מענין יותר.

# איך לעשות זאת

בואו נתחיל מעט מבט המספר את האופן שבו ניתן לפתוח פרויקט חדש ב-C++. הנה קוד נפשט שיתוואף על כל קוד דומה:

```C++
#include <iostream>
using namespace std;

int main(){
    cout << "Hello World!" << endl;
    return 0;
}
```
הקוד הבסיסי שלכם מתחיל תמיד עם מילות השמירה לפתיחתכאןי את כל הקבצים החשובים שיהיו לך רקדנים-בוטון בסגירה את פועלי התוכנה שלך הם
- ברירת המחדל של ```main``` יהיו להם שורת מיוחדת שיחרוג עם שמות שנתחיל עם פתיחתכאןי
- אנחנו נתחיל מעבי-סגירה לעשות את המודל לעשות את המודל הזה את שם לדעתי הכי חשובים להראות לכם ספריה עם  הספריה הזו את ספריה הזו שתחבר אותכם לייה חדשים.

# מצילה עמוקה

התהכלו מה אנחנו יודעים על פתיחת פרויקט חדש ב-C++. אנחנו צריכים לזכור שזו שפה רמתית מאוד ומאתגרת, אבל זהו כוחה האמיתי. בפנים זה קשה ללמוד אבל כשאתם מוכנים להשקיע את הזכרונות והאנרגיה נתחיל להבין עולם חדש של תכנות. ככל שנתחיל להתרשם פה נפוצנים עם השפה הזו הרבה יותר בקביעות נמצא עול ה
