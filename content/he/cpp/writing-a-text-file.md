## למה

ניתן להשתמש בכיתוב קבצי טקסט בכדי ליצור מבנה מסוים של מידע או לצורך הצגת נתונים בצורה נגישה ומידית למשתמשים. לכן, יצירת קבצי טקסט הינה חלק חשוב מתהליך התכנות ב-C++.

## כיצד לעשות זאת

כצעד ראשון, נצטרך ליצור אובייקט של סוג קובץ טקסט באמצעות הפקודה "ofstream", כדי לאפשר לנו לכתוב לקובץ טקסט. נשתמש בפקודת "open" כדי ליצור או לפתוח קובץ טקסט קיים. לאחר מכן, נוכל להשתמש בפקודת "write" כדי לכתוב נתונים לקובץ, ולסגור את הקובץ באמצעות פקודת "close". נמשיך לדוגמאות ולהעלות נתונים לקובץ באמצעות שימוש במספרים ומחרוזות.

```C++
#include <iostream>
#include <fstream>
using namespace std;

int main(){
    // יצירת אובייקט קובץ טקסט
    ofstream textfile;
    // פתיחת קובץ טקסט
    textfile.open("example.txt");
    
    // הכנסת נתונים לקובץ באמצעות פקודת "write"
    textfile << "זהו המסגרת הראשונה בטקסט\n";
    textfile << "מסגרת נוספת בטקסט שורה השנייה";
    
    // סגירת הקובץ
    textfile.close();
    
    return 0;
}
```
### פלט:
```
זהו המסגרת הראשונה בטקסט
מסגרת נוספת בטקסט שורה השנייה
```
