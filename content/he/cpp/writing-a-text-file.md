## למה

עבור רבים מאיתנו, כתיבת קבצי טקסט היא חלק בלתי נפרד מתהליך התפיסה וההבנה של התכנות. כתיבת קבצי טקסט מאפשרת לנו ליצור קוד נקי ומסודר וליצור מסמכים שמסבירים בצורה מפורטת כיצד הקוד פועל.

## איך לעשות זאת

```C++
#include <iostream>
#include <fstream>
using namespace std;

int main() {
  ofstream myFile ("example.txt"); // פתיחת קובץ טקסט חדש לכתיבה
  if (myFile.is_open()) { // בדיקה שהקובץ נפתח בהצלחה
    myFile << "זוהי תוכן דוגמה בקובץ שלנו!\n"; // הכנסת תוכן לקובץ
    myFile.close(); // סגירת הקובץ
  }
  else cout << "אירעה שגיאה בפתיחת הקובץ";
  return 0;
}
```

התוכנית לעיל מדגימה את השימוש בספריית "fstream" בכדי ליצור קובץ טקסט חדש ולהכניס לו תוכן. נוסף על כך, התוכנית בודקת אם הקובץ נפתח בהצלחה ומאפשרת טיפול בשגיאות אם הן מתרחשות.

כדי לכתוב תוכן נוסף לקובץ קיים, ניתן להשתמש בספרייה "ofstream" ובפעולת הפתיחה "ios::app" על מנת למנוע דריסת התוכן הקיים.
