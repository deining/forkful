## למה:
כתיבת קובץ טקסט היא חלק בלתי נפרד מתהליך התכנות בשפת C++, והיא מתבצעת לפעמים בכדי לשמור מידע מסוים או כחלק מקושחתת משלב האתר את הקובץ הטקסט לתוך תוכנה אחרת. כתיבת קובץ טקסט יכולה להיות כדאי עבור מטרות מגוונות ותאריכים, ולכן זה כדי לפתור בעיות מסוימות שהן חלק מהתהליך שלך.

## איך לכתוב קובץ טקסט בשפת C++:
כאשר נעיצו החלקים את הקוד, נוכל להתחיל לייצר את הקוד הפעילות שלנו על ידי שימוש בפונקציות ושבאתה אשר ידעתנו, לדוגמה `ofstream`, `open()` ו- `close()`. הנה דוגמאות לקוד ולפלט שיכולים להתקבל:

```C++
// פתיחת קובץ לכתיבה
ofstream file;
file.open("קובץ.txt");

// כתיבת מידע לקובץ
file << "כתיבה לתוך הקובץ כאן. " << endl;
file << "כתיבה חדשה לאחר שורת המעבר. " << endl;

// סגירת הקובץ
file.close();
```

פלט:

```
כתיבה לתוך הקובץ כאן.
כתיבה חדשה לאחר שורת המעבר.
```

כדי לכתוב לקובץ גם בדרכים נוספות, ניתן להשתמש בפונקציות נוספות כגון `write()` ו- `put()`.
