#למה#

כאשר אתם מפתחים תוכניות בשפת סי++, ייתכן שתצטרכו לקבל קלט מהמשתמש מתוך פקודת הפתיחה של התוכנית. כך נוצרו פרמטרים שנקראים "command line arguments". במאמר זה, אנו נתאר את התהליך שלקריאת פרמטרים זהים ונבין למה זה יכול להיות שימושי בתכנות.

#כיצד לעשות זאת#

הקריאה של פרמטרים מפקודת הפתיחה של התוכנית מתבצעת באמצעות הפונקציה main(). הפטיטייל של הפונקציה מקבל שני פרמטרים: argc ו-argv. הפרמטר argc מכיל את מספר הפרמטרים שהועברו לתוכנית, והפרמטר argv מכיל את רשימת הפרמטרים עצמם. כדי לקרוא את הארגומנטים, ניתן להשתמש בלולאת for כדי לעבור על רשימת הארגומנטים ולסרוק אותם.

```C++
int main(int argc, char* argv[])
{
   for(int i = 0; i < argc; i++)
      cout << argv[i] << endl;
   return 0;
}
```

בקוד זה, אנו משתמשים בלולאת for על מנת לעבור על רשימת הארגומנטים ולהדפיס אותם בעזרת הפונקציה cout של שפת C++.

מהפקודה בשורת הפקודה, ניתן לעבור על מספר אינסופי של ארגומנטים ולעבור אותם לתוכנית. לדוגמה, אם נרצה להוסיף את הארגומנטים "Hello World" ו- "2021", נכתוב את הפקודה הבאה:

```sh
./myprogram Hello World 2021
```

בתכנית, הארגומנטים יתווספו לרשימת argv ונוכל לעבור עליהם ולהדפיס אותם:

```
Hello
World
2021
```
