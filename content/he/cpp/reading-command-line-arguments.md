## למה

משתמשי תכנות יעשו שימוש בארגומנטים בשורת הפקודה כדי לתת לתוכנה שלהם לבצע פעולות שונות בהתאם לקלט שלהם.

## כיצד לעשות זאת

```C++
#include <iostream>

using namespace std;

int main(int argc, char *argv[])
{
    // מודפס אורך מספר הארגומנטים
    cout << "מספר הארגומנטים הוא: " << argc << endl;

    // מנתח את כל הארגומנטים ומדפיס את ערכם
    for (int i = 0; i < argc; i++) {
        cout << "ארגומנט מספר " << i << " הוא: " << argv[i] << endl;
    }

    return 0;
}
```

פלט:

    מספר הארגומנטים הוא: 3
    ארגומנט מספר 0 הוא: אתגרים.exe
    ארגומנט מספר 1 הוא: מילה
    ארגומנט מספר 2 הוא: מספר

השורה הראשונה מציגה את מספר הארגומנטים הכולל, כולל את השם של התוכנית. השורות הבאות מציגות את כל הארגומנטים שהוזנו על ידי המשתמש.

## צלילה עמוקה

כאשר אנו מדברים על "ארגומנטים בשורת הפקודה", אנו מתכוונים לפרמטרים שמועברים לתוכנית כארגומנטים בזמן פעולתה. ניתן להשתמש בפקודת הפקודה `argc` ובמערך הפוינטרים `argv` כדי לקרוא ולנתח את הארגומנטים שהוזנו על ידי המשתמש. ישנן גם ספריות ופונקציות נוספות שניתן להשתמש בהן כדי לטפל בארגומנטים בצורה מורכבת יותר.
