## למה

חיבור מחרוזות הוא תהליך נפוץ מאוד בתכנות בשפת סי++ ובשפות תכנות אחרות. זהו דרך יעילה ליצור מחרוזת חדשה על ידי הצבת מחרוזות נוספות ביחד. פעולה זו יכולה להיות מועילה ביותר כאשר צריך ליצור מחרוזת מורכבת ממספר מחרוזות קטנות יותר.

## איך לעשות את זה

לעיתים קרובות צריך ליצור מחרוזת חדשה המכילה מילות שונות, תאריכים או מספרים. לדוגמה, אם רוצים להציג את שמי בפני המשתמש, נוכל לחבר את מחרוזת "שלום" עם מחרוזת המכילה את שמי: "שלום פרסטוט". המחרוזת החדשה תיראה כך: "שלום פרסטוט".

```C++
#include <iostream>
using namespace std;

int main() {
  string message = "שלום ";
  string name = "פרסטוט";
  string greeting = message + name;

  cout << greeting;
}
```

פלט:

שלום פרסטוט

ובכן, נוכל לראות שהמחרוזת החדשה נוצרה על ידי חיבור של שתי מחרוזות קיימות. כמו כן, ניתן גם להשתמש במתודה אפשרית נוספת לחיבור מחרוזות בעזרת הסימן +=. כך ניתן לחבר גם מחרוזות שלמות וגם משתנים שמכילים מחרוזות.

```C++
string fullSentence = "השלום נמשך עד העתיד ומאחר כך";
string firstPart = "השלום נמשך עד העתיד";
string secondPart = "מאחר כך";
fullSentence += secondPart; // השלום נמשך עד העתיד מאחר כך

```
