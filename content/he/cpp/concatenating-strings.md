## מה ולמה?
שרשור מחרוזות הוא תהליך שישמש את מתכנתי C++ לחבר מחרוזות שונות ליחד וליצור מחרוזת חדשה שמציגה את המידע משני המחרוזות המקוריות. מתכנני תוכניות אלו משתמשים בשיטה זו כדי ליצור מחרוזות דינמיות שמשתנות בהתאם לצורך.

## איך לעשות?
כדי לשרשר מחרוזות בשפת C++, ניתן להשתמש במכשיר השרשור + או בפעולת השרשור המובנת של שפת C++, שניהם מאפשרים לנו לחבר שתי מחרוזות ליחד בפשטות. 

```C++
string firstString = "Hello";
string secondString = "world!";
string result = firstString + secondString;
cout << result << endl; // output: Hello world!
```
