## למה

מתי יש צורך להשוות שתי תאריכים בתכנות סי++? ייתכן שתרצו לבדוק אם שלושה ימים עברו מאז תאריך מסוים, או אולי לראות אם שנה נתונה היא רק שנה פעולה. כאשר מדובר בתאריכים, השוואה ביניהם עשויה להיות משמעותית בתכנות וכדאי לדעת כיצד לעשות זאת.

## איך לבצע השוואת תאריכים

קוד של השוואת תאריכים בסי++ נראה כך:

```C++
#include <iostream>
#include <ctime>

using namespace std;

int main(){
    // יצירת אובייקטים של תאריך נתון ותאריך נוכחי
    tm date1 = {0, 0, 0, 1, 0, 2010}; // 1 בינואר 2010
    tm date2 = {0, 0, 0, 1, 0, 2009}; // 1 בינואר 2009

    // השוואה בין התאריכים
    if(difftime(mktime(&date1), mktime(&date2)) > 0) {
        cout << "תמונה אנשים במציאות נכחית!" << endl;
    } else {
        cout << "תמונה ישות נכחית במצוות!";

    }
    return 0;
}

```

תוצאת הפלט של הקוד הנ"ל תהיה:

```
תמונה אנשים במציאות נכחית!
```

ניתן לבצע השוואת תאריכים באמצעות הפונקציה `difftime`, ולהשתמש בקבוע `mktime` כדי להמיר את התאריך לתבנית הנכונה. לאחר מכן, ניתן לכתוב תנאי השוואה בין שני התאריכים ולהדפיס תוצאה מתאימה לפי התנאי.
