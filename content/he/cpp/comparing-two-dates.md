# למה

כאשר אנו עובדים עם תאריכים בתוך תוכניות הפיתוח שלנו, יתכנו מקרים שבהם נידרש להשוות בין שני תאריכים כדי לבדוק אם הם שווים או קרובים. לפעמים זה עשוי להיות חלק מתהליך ביצוע אירועים או פעולות מסוימות בהתאמה לתאריכים מסוימים. כמו כן, השוואת תאריכים יכולה להיות שימושית לצורך סידור עריכת תאריך יומיומי או לתכנון לוח שנה.

# כיצד לעשות זאת

תהליך השוואת שני תאריכים בתוך תוכניות C++ יכול להיות בעייתי ומסובך. נדגים כאן כמה דרכים שונות לעשות זאת בקלות תוך שימוש בתכונות של שפת התכנות הזו. 

```C++
// משתני התאריך
int day1, month1, year1;
int day2, month2, year2;

// קביעת תאריכים לתאריך מסוים
day1 = 10;
month1 = 5;
year1 = 2021;

day2 = 20;
month2 = 5;
year2 = 2021;

// השוואת תאריכים עם הפעולות ==, != והחזרת תוצאה בתוך משתנה מסוג bool
bool result = (day1 == day2 && month1 == month2 && year1 == year2);

// הדפסת התוצאה
if(result)
{
  cout << "התאריכים זהים";
}
else
{
  cout << "התאריכים אינם זהים";
}
```

**פלט:**
התאריכים אינם זהים

# העומק שבהשוואת שני תאריכים

כאשר אנו משווים שני תאריכים, כדאי להתייחס גם למקרים מיוחדים כגון תאריכים חופשיים, שנה מעוברת או ימי המרה ביניהם. כמו כן, ניתן להשתמש בפונקציות מובנות כדי להציג מידע נוסף על תאריך מסוים כדי לעזור בהשוואה.
