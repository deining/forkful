## מה ולמה?

בתכנות, רגשיות רגילות הן כלי חזק שמאפשר לנו למצוא תבניות בטקסט בצורה מהירה ויעילה. הן משתמשות במילולאות וסימני פרידה כדי לתאר את התבניות שאנו מחפשים ולאפשר לנו לנתח את הטקסט בצורה אוטומטית. תכנות באמצעות רגשיות רגילות הוא כלי חשוב לתכנותנים, כך שתוכלו למצוא ולעבד מידע במהירות ובקלות.

## איך לעשות זאת?

דוגמאות קוד ופלט נלווים שיעזרו לכם להבין איך לעבוד עם רגשיות רגילות בשפת C++. נתחיל עם פונקציה פשוטה שמחפשת כתובת אימייל בטקסט נתון:

```C++
#include <iostream>
#include <regex>
using namespace std;

int main () {
  string text = "מיילים: user1@example.com, user2@example.com, user3@example.com";
  regex email_regex("([a-z0-9]+[_a-z0-9\\-.]*)@[a-z0-9\\-]+(\\.[a-z]+)+");
  smatch matches;
  while (regex_search(text, matches, email_regex)) {
    for (auto match : matches) {
      cout << match << endl;
    }
    text = matches.suffix().str();
  }
  return 0;
}
```

הפלט הוא רשימת כתובות האימייל בטקסט:

```
user1@example.com
user2@example.com
user3@example.com
```

עדיין לא ברור מה קורה בקוד? אז הנה תיבת כלים מעולה שתעזור לכם לבנות ולבדוק רגשיות רגילות: [regex101.com](https://regex101.com/).

## לעומק

רגשיות רגילות נמצאות כבר כמה עשורים והן נחשבות לכלי חשוב בתחום התכנות. בנוסף לשימוש בשפות תכנות רבות, רגשיות רגילות משמשות גם בתחומים נוספים כמו שפות התכנות בהוראת כתיבה ועיבוד טקסט. חשוב ללמוד רגשיות רגילות כדי לייעל את התהליך התכנותי ולאפשר לנו לעבוד עם טקסט בצורה נוחה ומתקדמת.

קיימים גם אלטרנטיבות לרגשיות רגילות, כגון ספריות טקסט נוספות שאפשר להשתמש בהם כדי לעבד טקסט. אך רגשיות רגילות נחשבות לתכונה נהדרת של שפת C++ והן מספקות לנו כלי חזק ויעיל לעבוד עם טקסט.
