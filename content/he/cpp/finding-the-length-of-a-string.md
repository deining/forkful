## למה 

כמה סיבות מוטיבציה עלולות להוביל לרצונם של אנשים למצוא את האורך של מחרוזות בשפת C++. הן עשויות להיות מתוך צורך לעבוד עם נתונים טקסטואליים או בעקבות תרגילי תכנות שמחייבים את השימוש בפונקציות הקיימות למציאת אורך של מחרוזת.

## איך למצוא את האורך של מחרוזת בשפת C++
```C++
#include <iostream>
using namespace std;

int main(){
   string name = "דוד";
   int len = name.length();
   cout << "אורך המחרוזת של " << name << " הוא " << len << endl;
}
```
**פלט:**
אורך המחרוזת של דוד הוא 3

בדוגמה זו, אנחנו משתמשים בפונקצית "length()" הקיימת כבר בספריית המחרוזות של C++. פונקצית זו מחזירה את האורך של המחרוזת שמוצבת בתוך הסוגריים המרובעים שלה. כך, ניתן להשתמש במשתנה שיכיל את ערך האורך של המחרוזת כדי לטפל בנתונים טקסטואליים בתכניות שלנו.

## עיון מעמיק: מציאת אורך של מחרוזת בשפת C++

בעזרת הפונקציה length(), אנחנו משתמשים באלגוריתם פשוט שמעביר על כל תו במחרוזת ומחשב את מספרה של תווי המחרוזת. ככל שהמחרוזת תהיה ארוכה יותר, כך יקח פעולה זו יותר זמן להתבצע, ולכן כדאי להשתמש בה בצורה זויפה עבור מחרוזת מיותרת.
