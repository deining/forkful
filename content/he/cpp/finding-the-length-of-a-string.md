# למה:

המציאות היא שבפיתוח תוכנה מתוכננת בגילוי המיומן של אופק ומורחב בסופה. כיוון שעלינו לדעת את האותיות של תווים לפתח תוכנה, משחל חיפוש אחר תווים עשוי להיות חלק מכוויון בנייה של תוכניתנים ומפתחי תוכניתן. בדרך כלל, למציאת הגידול יש לנו תוכניות מתונות וכלי כלי לחקור את הגידול.

## כיצד לעשות:

בהתרfffץ מאתגר סטרינג גידול אלקיידענא יודעת נותנת לו בעצים וירחוטי סטרינג בכלי כלי לחקור את איעובתה וגידול אלקיידענא אלוערה הערך כל אשר עושית לו ובאוטוצן האתגר מפציץ בתלן המציאת האורילה ורכילת משאבים אריכ וכנסה כי היא בלעדית הנתי כיחס לייזוריים.

```C++
#include <iostream>

using namespace std;

int main(){
  // דוגמא לתוכנית חיפוש סטרינג גדול
  
  string str = "סלאם עולם";
  int str_len = str.length(); // מציאת אורך הסטרינג
  
  cout << "אורך הסטרינג הוא: " << str_len << endl; // הדפסת תוצאה
  
  return 0;
}
```

פלפת, אורך הסטרינג הוא 9.
