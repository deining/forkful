## למה

אחת המשימות הנפוצות בתכנות בשפת C++ היא למצוא את אורך המחרוזת. המחרוזת היא סדרה של תווים, כגון מילים או מספרים, וכדי לעבוד איתם אנחנו צריכים לדעת את אורךם. למשל, כאשר אנחנו מקבלים קלט מהמשתמש, חשוב לדעת כמה תווים הקלט הוא כדי לבדוק אם הוא תקין או לעשות איתו חישובים מתאימים.

## כיצד

ראשית, נתחיל עם הקוד היסודי כדי לספק לכם רעיון כיצד הפונקציה לאורך עובדת. למשל, הנה קטע קוד פשוט שמדגים איך למצוא את אורך המחרוזת באמצעות לולאה:

```C++
#include <iostream>
using namespace std;

int main() {
    string str = "Hello World";
    int length = 0;

    // לרוץ על תווי המחרוזת ולהוסיף 1 לאורך כל תו
    for (int i = 0; str[i] != '\0'; i++) {
        length++;
    }

    cout << "אורך המחרוזת הוא: " << length << endl;
    
    return 0;
}
```

פתחו את הקוד ונסו להחליף את המחרוזת למשהו אחר ולראות את התוצאה. כאן אנחנו משתמשים בלולאת for כדי לרוץ על כל תו במחרוזת עד שנגיע לתו סוף המחרוזת, המסומן כמו '\0'. כל פעם שאנחנו מוצאים תו חדש נוסף לתווי המחרוזת, אנחנו מוסיפים 1 למשתנה אורך כדי לספור את התווים.

הנה עוד דוגמה קצרה עם השימוש בפונקציה אורך שקיימת בספריית התקנה הכללית:

```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    string str = "Hello World";
    int length = str.length(); //  קריאה ישירה לפונקציה אורך

    cout << "אורך המחרוזת הוא: " << length << endl;
    
    return 0;
}
```
