## למה

בעולם התכנות, תמיד נתקלנו בצורך לבצע טיפול במחרוזות. אחת הפעולות הנפוצות שנצטרך לבצע היא גיבוי מחרוזות, כלומר להפוך אותן לרשימת אותיות ראשונות גדולות. המטרה של הגיבוי הניצולי הזה מסלק את הצורך לבדוק ולהתעסק עם מספר בעיות נפוצות, כגון שמות עם רווחים או איחוד מחרוזות נפרדות.

## איך לעשות זאת

```C++
#include <iostream>
#include <string>
#include <algorithm>

using namespace std;

int main(){
    string str = "hello world";
    // עם הפונקציה transform, אנו משנים את האות הראשונה של כל מילה לאות גדולה
    transform(str.begin(), str.end(), str.begin(), ::toupper);
    cout << str << endl;

    return 0;
}
```

הפלט של התכנית הזו יהיה "HELLO WORLD". ניתן לראות בקוד את שימוש בפונקציה transform של ספריית ה-STL של C++ כדי לאתיים את המחרוזת לצורה מבוקשת.

## כיצד פועלת הגיבוי של מחרוזות

הגיבוי של מחרוזות משתמש בפונקציות שיכולות לשנות את מחרוזת המקור למחרוזת חדשה בתהליך שנקרא "כתיבת חריגים". בכתיבת חריגים, אנו שואפים לעבור על כל התווים במחרוזת, לזהות את האינדקס של התו הראשון של כל מילה ולשנות אותו לאות גדולה.

בתהליך הזה, נעבור על המחרוזת שנית בכדי לבצע שני סלים שונים - אחד לאותיות גדולות ואחד לאותיות קטנות. ע"י חיבור סלי התווים הנפרדים הללו נקבל מחרוזת חדשה שכל המילים בה תהיינה כתובות באותיות גדולות.
