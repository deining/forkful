## מה ולמה
אם אתה מתכנת ופעמים רבות נתקלת במשימות שדורשות ממך להפוך את האותיות הראשונות של מחרוזת לאותיות גדולות, אתה בטח מכיר את התהליך הזה שנקרא "כתיבה באותיות ראשונות גדולות". 
זה פשוט אומר שאתה משנה את האות הראשונה של מחרוזת לאות גדולה. ולמה כולנו עושים את זה? האמת היא שישנן מספר סיבות - אחת מהן היא שזה נראה יותר יפה ומסודר. בנוסף, יש תוכניות ואתרי אינטרנט שסביבת העבודה שלהם מציעה הכי מקובלת ומקלה לקרוא.

## איך לעשות זאת
האופציה הכי פשוטה ונפוצה היא לעשות שימוש במודפנקציית toupper המובנת של שפת התכנות C++. אנחנו רק צריכים להוסיף עוד אחד מהתווים באתר לתוך המודפעת כדי לייצר את האות הגדולה שמתחילה את המחרוזת. נבדוק זאת בקוד הבא:
```C++
#include <iostream>

using namespace std;

int main() 
{
    string name = "john";
    name[0] = toupper(name[0]);
    cout << name; // פלט: John
    return 0;
}
```
כמו שאתה רואה, אנחנו משתמשים במשתנה string בשם name כדי לאחסן את המחרוזת "john". אחר כך אנחנו משנים את התו הראשון במחרוזת לאיתו הוא באמת אות גדולה עם הפונקציה toupper. ולבסוף אנחנו מדפיסים את המחרוזת המעודכנת. 

אבל מה אם יש לנו מחרוזת שמתחילה עם כמה אותיות גדולות? כדי לא לשים בסכנה את האותיות הגדולות הקיימות, אנחנו יכולים להשתמש בפונקציה tolower ואז לעשות tolower על התו הראשון במחרוזת. נבדוק זאת בקוד הבא:
```C++
#include <iostream>

using namespace std;

int main() 
{
    string name = "JOhn";
    name[0] = tolower(name[0]);
    cout << name; // פלט: jOhn
    return 0;
}
```
כעת הנו מקבלים את הפלט המתאים למחרוזת שהיינו משתמשים בה.

## מציאה מרעננת
בעבר, היה לתכנות קושי רב יותר לבנות בתוכנית בה משתמשים לשינוי תווים במחרוזת שהוא אות אחרת. אבל עם הפונקציות toupper וגם tolower, היא עבודה פשוטה מאוד.
