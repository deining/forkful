##מדוע

מדוע לאנשים נהוג לכתוב באופן מנומס את המחרוזת (string) שהם משתמשים בה.

##כיצד

משלוחים כמה כוכביות נהדרים אליכם בזה שהופך את המחרוזת שלכם לכתב גדול. עקבו אחר הקוד הבא בדוגמה:

```C++
//פעולה זו מעבירה מחרוזת לאותיות גדולות ומחזירה את המחרוזת המעובדת.
string capitalizestring(string input)
{
     string output = to_upper(input);
     return output;
}

int main()
{
     //מחלקים את המחרוזת לשני מחרוזות
     string first_name = "שלום";
     string last_name = "כהן";

     //מחברים את המחרוזות לכדי מחרוזת אחת באמצעות רווח
     string full_name = first_name + " " + last_name;

     //מדפיסים את המחרוזה באותיות גדולות
     cout << capitalizestring(full_name) << endl;

     return 0;
}
```

פלט:
```
שלום כהן
```

כפי שאתם רואים, הקוד חייב להיות כתוב בכתב מנומס ובגודל המתאים לכן עשינו שימוש בפעולה שתעשה את העבודה בשבילנו.

##עומק

עכשיו, אם נשקיע קצת זמן לחקור יותר את הקוד, נמצא כמה נקודות מעניינות:

1. נשתמש בפעולה to_upper() כדי להמיר את המחרוזת לאותיות גדולות. הפעולה זו מוגדרת בתוך הספרייה <string> ומקבלת את המחרוזת כפרמטר.
2. ניתן לשנות את פעולת ההמרה שלנו לאילו זו תהרות את המחרוזת לאותיות קטנות על-ידי שימוש בפעולה to_lower().
3. ישנן דרכים נוספות לכתוב קוד באותו המטרה, כגון בשימוש בלולאות או בתנאים, אך פעולות ההרמת וההורדה עושות את העבודה בצורה יעילה יותר.
