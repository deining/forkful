## איך לעשות זאת

### קפיצת רמה בקיבולת טכנית

שתי היותר רווחות בעבודה מכותבי הקוד הם נשיאה או מערכת הפעלה. אם פיתחתם את הקוד אם אתם יכולים לכתוב פקודת מקשים הוא יהיה שביל חציוני קריטי חלוקת הקובץ לוגית שאירע דברי הדפסה נכתב ביחדו עם הקוד האחרון. כשאנו מדברים על פיתוח תוכנה, הקוד הראשון במילה היא תבנית מודול אשרבדיוק ובהסכמת פיתוח התוכנה היא שהפעלנו פונקציות פשוטות עבור קוד ארוך ומנומס תקשורת.

```C++
#include <iostream>
#include <string>

using namespace std;

string capitalizeString(string str) {
    // עזרת המשתמשת בrotation עבור הפונקציה

    int len = str.length();
    
    // לולאת פורץ באותיות במילה ויותר את האות הראשונה
    for (int i = 0; i < len; i++) {
        // אם האות היא בין a-z יהיה אפשרי לשנות את גודל האות
        if (str[i] >= 'a' && str[i] <= 'z') {
            str[i] -= 32;
        }
    }

    return str;
}

int main()
{
    string str = "hello world";
    
    // הפעלה של הפונקציה
    str = capitalizeString(str);

    // פלט המשתמשן לשימוש עבור אפליקציות אופטימות
    cout << str << endl;

    return 0;
}
```
