## למה

בכל פיתוח תוכנה, אנחנו נתקלים במצבים שבהם יש לנו להבין מה בדיוק קורה בקוד שלנו בכדי לפתור בעיות. טכניקה שיכולה לעזור לנו לזהות ולתקן תקלות היא הדפסת פלט עם נתונים נוספים המציגים את מצב התוכנית שלנו בנקודת התקלה. במאמר זה נציג לכם כיצד להדפיס פלט דיבאג בשפת סי++.

## איך לעשות זאת

בשפת סי++, נוכל להדפיס פלט דיבאג באמצעות פקודת `cout` והתבנית `<<`. למשל:

```C++
int sum = 0;

for (int i = 1; i <= 10; i++) {
    sum += i;
    cout << "The value of sum after " << i << " iterations is: " << sum << endl;
}
```

בתוך לולאת ה-`for` הנ"ל, אנחנו מדפיסים את ערך הסכום בכל פעם שהלולאה מתבצעת, על מנת שנוכל לראות את הערכים המתעדכנים. ניתן גם להדפיס משתנים בכללי באמצעות התבנית `<<`, על ידי הכנסת שם המשתנה ואת התו `<<` לפני הערך שלו.

בנוסף, ניתן גם להשתמש בפקודת `assert` כדי לבדוק אם משתנה מסוים מכיל את הערך הרצוי. למשל:

```C++
int num = 5;

// יתן מספר טעויות עד שהתנאי יתקיים
assert(num == 10);
```

אם התנאי לא מתקיים, התוכנית תפסיק לרוץ ותדפיס שגיאה המציינת את המשתנה שגרם לכך.

## הצצה עמוקה

ההדפסה של פלט דיבאג היא כלי מצוין ללמידה ותיקון שגיאות בקוד. זה יכול לעזור לנו להבין מה קורה בתוכנית שלנו ואיפה נמצאת הבעיה. ניתן להשתמש במגוון של פקודות דיבאג כגון `cout`, `assert`
