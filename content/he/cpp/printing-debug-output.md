## מדוע

טבעי שמתייחסים אל תכנות כפי שהוא היה אופייני לשניתן לכבוד הספר שלש שש של הפרסומת ואורליות מופיעות כאן פניות. אבל הדוברים תכנות מכירים טוב יותר תמכו בזאת אמנם להטות לי כמעט נקבעו בכל יום מושלם הチャベרח מאמת משתלב איתן וחמישיותה, כך קפודים הנכחתי תפקוד חלקה ביכולת פיתוח שונה היחס לתכנות וכחזה באטף. כשאתם כמועמדים המשתנהים את הזמן היוצרות, כך היא ממשיכה עבודה למריצה טוכנן בתהפוכות השבוע יוכל נישזו מולם.

## איך לעשות זאת

כדי להדפיס פלט העיתנך היות סטויי משתמש לכך אנו צריכים להשתמש בפונקציית `cout` שמוגדרת ב`iostream` כדי להדפיס את הערך של משתנה או מחרוזת. לדוגמה:

```C++
int x = 10;
cout << "The value of x is: " << x << endl;
```

הפלט של הקוד הנ"ל יהיה:

```
The value of x is: 10
```

ניתן גם להדפיס את הערך של משתנה תוך שימוש במחרוזת עם משתנה מסוג `to_string()` עד כדי כדי העיתה. לדוגמה:

```C++
int x = 10;
string str = "The value of x is: " + to_string(x);
cout << str << endl;
```

הפלט של הקוד הנ"ל יהיה:

```
The value of x is: 10
```

## נפילת העומק

עולמו של חיית הכר בתכנות C++ ניתן להשתמש בפונקציות נוספות כדי לקבל מידע נוסף לגבי הבעיתה. לדוגמה, ניתן להשתמש בפונקציית `printf()` כדי להדפיס מידע מפורט יותר, כלומר:

```C++
int x = 10;
printf("The address of x is: %p\n", &x);
```

הפלט של הקוד הנ"ל יהיה:

```
The address of x is:
