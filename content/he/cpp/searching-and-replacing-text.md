## מדוע

אם אתם מתכנתים בשפת פיתוח C++, ייתכן שתגידו לעצמכם - למה אני זקוק לחיפוש והחלפת טקסט? התשובה פשוטה - כדי לחסוך זמן ומאמץ בתהליך של עבודה עם קוד. החיפוש והחלפה של טקסט הם כלי חשוב במיוחד לעבודה יעילה יותר עם קודים גדולים ומיוזעים.

## איך לעשות זאת

לפניכם מספר דוגמאות קלות על כיצד לבצע חיפוש והחלפה טקסט בשפת C ++. כל אחת מהדוגמאות תיכלל בתוך בלוק קוד המבוצע בעזרת תוים "C ++ ```". אנא תקנו מיד לכתוב את הקוד בפורמט נכון כדי להימנע מבעיות קודמות.

1. חיפוש והחלפה עבור מחרוזת מסוימת בתוך מחרוזת נתונה.
```C++
#include <string>

// מחרוזת נתונה
std::string input = "ברוך הבא למדור הכתיבה שלנו, המרכז ללמידת תכנות בשפת C++";

// חיפוש והחלפה של המחרוזת "המרכז" ל-"סנטר"
input.replace(input.find("מרכז"), 6, "סנטר");

// הדפסת המחרוזת המעודכנת
std::cout << input << std::endl;

// פלט: ברוך הבא לסנטר הכתיבה שלנו, הסנטר ללמידת תכנות בשפת C++
```

2. חיפוש והחלפה של המחרוזת הכי אחרונה בתוך מחרוזת נתונה.
```C++
#include <string>

// מחרוזת נתונה
std::string input = "כי אנחנו אנשים רעבים, נחפש ונחליף אוכל חדש";

// חיפוש והחלפה של המחרוזת "חדש" ל-"טעים"
input.replace(input.rfind("חדש"), 4, "טעים");

// הדפסת המחרוזת המעודכנת
std::cout << input << std::endl;

// פלט: כי אנחנו אנשים רעבים, נחפש ונחליף אוכל טעים
```
