## למה:

יצירת קובץ זמני בתוך התוכנית שלך יכולה להיות מועילה במספר סיטואציות. למשל, כאשר יש צורך לכתוב נתונים לקובץ חדש ולא לפגוע בקובץ המקורי, או כאשר יש צורך לכתוב נתונים זמניים למטרת שיתוף עם תוכנית אחרת.

## כיצד:

כדי ליצור קובץ זמני בשפת C++, ניתן להשתמש בפונקציית "tmpfile()" כדי ליצור קובץ מלאכותי זמני. לדוגמה:

```C++
FILE* temp = tmpfile(); // יוצר קובץ זמני ומחזיר את הפוינטר שלו
fprintf(temp, "כתוב כאן את הנתונים שברצונך לכתוב לקובץ"); // כותב נתונים לקובץ הזמני
// לאחר הסיום, יש לסגור את הקובץ זמני
fclose(temp);
```

תוכלו גם להשתמש בתוכנה ליצירת קבצים זמניים בצורה נוחה יותר. לדוגמה, ב-Windows תוכלו להשתמש בפקודת "CreateFile". עבור לינוקס ומערכות מבוססות Unix, תוכלו להשתמש בפקודת "mkstemp". ולמערכות Mac OS X, תוכלו להשתמש בפקודת "mktemp". כמו כן, תוכלו גם להשתמש בספרייה של boost המכילה פונקציות נוספות ופשוטות יותר ליצירת קבצים זמניים.
