##
למה: מילים כמו "מחיקה" ו"תבנית" נשמעות נוחות כשמדובר בתכנות, אבל מה זה בעצם משמעות למחוק תווים על פי תבנית ומתי זה נחוץ?

איך לעשות: נסו להחזיר את המחרוזת "Hello World!" עם כל האותיות הקטנות מחוקות.

```
C++ 
#include <iostream>
#include <string>

using namespace std;

int main(){
    string str = "Hello World!";
    for (int i = 0; i < str.length(); i++) {
        if (str[i] >= 'a' && str[i] <= 'z') {
            str.erase(i, 1);
        }
    }
    cout << str << endl;
    return 0;
}
```

תוצאה: H W!

להשוק: כעת שניקיתם את הקובץ הזה בקלות, שווה לעקוב אחר ההסברים המפורטים בפוסט הבא להבין בעצמכם את השימוש וההיעד של מחיקת תווים על פי תבנית. אתם תגלו שחלק מהתכנים הכתובים הם ידועים לכם מאתרים אחרים, כיון שהם מתייחסים לספציפיים, ולמעשה אני מדבר על בסיס מידע - המעניק את הערך והחשפות הכי טובים לתלמידיים החדשים ולמשתמשים. אתם מוזמנים תמיד לבקר אותנו ולקבל מידע נוסף.
