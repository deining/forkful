ניתוח למה: רק 1-2 משפטים המסבירים *למה* מישהו יבחר למחוק תווים המתאימים לתבנית.

כיצד לעשות זאת: דוגמאות קידוד ותוצאות דוגמא בתוך בלוקים של "```C++...```". 

כשמדברים על תכנות בשפת C++, ישנם כמה פעולות שנעשות על בסיס קבוע. אחת מהן היא מחיקת תווים המתאימים לתבנית מטקסט כלשהו. למה לעשות זאת? יתר על כן, ישנם מצבים שבהם יש תווים כפולים או לא רלוונטיים שיוצרים רעש לא נחוץ בתוצאה הסופית. בכדי לשפר את הקוד ולהפחית את ארכיון הקוד, מתבצעת פעולת מחיקת התווים הללו.

```c++
//קטע קוד של פונקציית המחיקה הפשוטה להבנת העיקרון:

#include <iostream>
#include <string>
using namespace std;

//פונקציית מחיקת תווים עם פרמטרים המקבלת מחרוזת טקסט ותווים למחיקה
void deleteChars(string& text, char toDelete)
{
    int current = 0;
    //לולאה נגד כל התווים של המחרוזת
    while (current < text.length())
    {
        //אם התו הנוכחי במחרוזת זהה לתו למחיקה, נוציא אותו
        if (text[current] == toDelete)
            text.erase(current, 1);
        //שימוש במעטפת אינדקסים כדי להזיז על התווים הבאים
        else
            current++;
    }
}

int main()
{
    //טקסט לדוגמא
    string text = "שלום לך ליהלום!";
    //מחיקת הכתב ההפוך
    deleteChars(text, 'ל');
    cout << text << endl;
    return 0;
}
// תוצאה: חם ךניו!

```
