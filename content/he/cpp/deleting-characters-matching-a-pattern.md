## למה

כיצד: דוגמאות תכנות ופלט דוגמאות בתוך קטעי קוד " ```C++ ... ``` ". צולם: מידע עמוק על מחיקת תווים המתאימים לתבנית.

מחיקת תווים המתאימים לתבנית היא פעולה חשובה בתכנות המונחה עצמים, שמאפשרת לנו להעביר ולעבוד על מחרוזות בצורה יעילה ומהירה. כאשר אנו מבינים את המהות של הפעולה, נוכל להשתמש בה בצורה נכונה ולהפיק מתוכה את המרב.

## כיצד לבצע מחיקת תווים המתאימים לתבנית

למחיקת תווים המתאימים לתבנית, נשתמש בפונקציה erase של סטרינג בפלטפורמת C++. הפונקציה תקבל כארגומנטים את האינדקס הראשון של התו הראשון של התבנית, ואת האורך של התווים המתאימים לתבנית שברצוננו למחוק. ניתן להשתמש גם בפונקציות כמו find וreplace כדי לאתר את התווים המבוקשים ולהחליפם בהתאם.

```C++
// דוגמה למחיקת תווים המתאימים לתבנית
#include <iostream>
#include <string>

using namespace std;

int main()
{
    string str = "Hello, World!";
    string pattern = ", "; // התבנית שברצוננו למחוק

    // מציאת האינדקס של התבנית במחרוזת
    int index = str.find(pattern);

    // המחיקה של התבנית מהמחרוזת
    str.erase(index, pattern.length());

    // פלט: HelloWorld!
    cout << str << endl;
    
    return 0;
}

```
