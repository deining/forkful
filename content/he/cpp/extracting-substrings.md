## מה ולמה?

עיבוד מחרוזות הוא כלי חשוב בתכנות שהמאפשר לנו להתמודד עם מחרוזות ארוכות ומורכבות יותר מתוך הצורך להבציר נתונים מערכים מסובכים יותר. למשל, כאשר עובדים עם טקסט ממוכן כמחרוזת, ניתן להשתמש בפקודות הקיימות כדי להציג רק את החלק המעניין של המחרוזת.

## איך לעשות:

מאמר זה מראה כיצד להשתמש בפקודות עיבוד מחרוזות בשפת C++. בדוגמאות הבאות נראה כיצד ליצור מחרוזת חדשה מתוך מחרוזת קיימת, כיצד לגשת לתו ספציפי במחרוזת וכיצד להדפיס כמה תווים ממחרוזת מסוימת. תוצאות הקוד תהיינה מופיעות תחת פקודות ה-```C++```.

### יצירת מחרוזת חדשה:
```C++
#include <iostream>
using namespace std;

int main() {
    string str = "מחרוזת חדשה";
    cout << str << endl;
    return 0;
}
```

תוצאה:
```
מחרוזת חדשה
```

### גישה לתו ספציפי במחרוזת:
```C++
#include <iostream>
using namespace std;

int main() {
    string str = "תו ספציפי";
    cout << str[3] << endl;
    return 0;
}
```

תוצאה:
```
ת
```

### הדפסת כמה תווים ממחרוזת:
```C++
#include <iostream>
using namespace std;

int main() {
    string str = "כמה-תווים";
    cout << str.substr(0,5) << endl;
    return 0;
}
```

תוצאה:
```
כמה-
```

## לחקור עומקים:

השימוש בפקודות עיבוד מחרוזות נפוץ מאוד והתחום נמצא בתחילת האבולוציה שלו. במקור, פקודות אלו היו מאוד מורכבות וסובלנו לכתוב קטעי קוד ארוכים כדי להציג חלק מסוים של מחרוזת. אבל עם התפתחות הטכנולוגיות והשיפור של השפות התכנות, הפקודות התחילו להיות פשוטות יותר לשימוש.

כיום ישנן אפשרויות נוספות להתגבר על השימוש בפקודות עיבוד מחרוזות. תחת השם הכללי "מילוית מחרוזות", ניתן להשתמש בפעולות כגון חיפוש והחלפה של חלק ממחרוזת, יצירת מחרוזת חדשה עם קיצורי שמות, ועוד.

כאשר מדברים על פקודות עיבוד מחרוזות בשפת C++, מתכנתים מכירים את הפקודות הבסיסיות בלבד, אבל ישנם עוד פקודות מתקדמות שטוב להיות מודעים אליהן. כמו למשל, פקודת ```replace``` שמאפשרת להחליף חלק ממחרוזת עם חלק אחר, פקודת ```insert``` שמאפשרת להכניס חלק נוסף למחרוזת וכו'.
