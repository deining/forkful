## למה

חילוץ תת-מחרוזת הוא תהליך שנועד להשיג ערכים מסוימים בתוך מחרוזת גדולה יותר. זה נחשב לשימושי במיוחד כאשר אנחנו עובדים עם טקסטים ארוכים ונרצה לקבל מילות מפתח או מידע מסוים מתוך המחרוזת הגדולה.

## איך לעשות

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    // יצירת מחרוזת חדשה
    string sentence = "שלום לכולם! אני לומד את הפיתוח בשפת C++.";
    
    // חילוץ תת-מחרוזת מהמחרוזת כללת
    string word = sentence.substr(12, 11); // תחילת התת-מחרוזת מהתו ה-12 לאורך 11 תווים
    
    // הדפסת התוצאה
    cout << word << endl;
    
    return 0;
}
```

תוצאה: "לכולם!"

## נכנסים רחוק יותר

הופכים לשכיחים לעיתים קרובות ששפת C++ מציעה כמה פעולות שימושיות לעיבוד מחרוזות, כגון find, erase, replace ועוד. כדי לחלץ תת-מחרוזת, אנחנו משתמשים בשימוש בפעולת substr ובמשתנים של התחילה והאורך של התת-מחרוזת. כמו כן, חשוב לזכור כי אנחנו מתחילים לספור מתחילת המחרוזת כך שצריך להיות זהירים עם האינדקסים.
