## למה

משחקי מחשב, תוכניות ואפליקציות משתמשות בתת-מחרוזות בכדי לטפל במשימות שונות, כגון האיתור של מילים מסוימות או עיבוד טקסט. לכן, לאלו שמתעסקים בתכנות בשפת סי++ יהיה שימושי לדעת איך לחשב ולטפל בתת-מחרוזות.

## איך לעשות זאת

הנה דוגמאות של קוד מסומן בכדי להדגים איך לעשות את הפעולה המבוקשת:

```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
    string str = "Hello World";
    
    // משתנה מסוג string שמכיל תת-מחרוזת מהמיקום השני עד החמישי
    string sub1 = str.substr(2, 3);
    cout << sub1 << endl; // הדפסת התת-מחרוזת "llo"
    
    // משתנה מסוג string שמכיל תת-מחרוזת מהמיקום השישי עד סוף הטקסט
    string sub2 = str.substr(6);
    cout << sub2 << endl; // הדפסת התת-מחרוזת "World"
    
    return 0;
}
```

כאן אנו משתמשים בפונקציית substr(), שמקבלת שני מספרים: הראשון מציין את המיקום של התחלת התת-מחרוזת במחרוזת המקורית, והשני מציין את אורך התת-מחרוזת הרצוי. אם מספר השני הוא אופציונלי, כמו במקרה השני, אז התת-מחרוזת תתחיל מהמיקום שנציין ותסתיים בסוף המחרוזת.
