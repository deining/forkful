בכתבה זו נלמד איך לחלץ מחרוזת תת-מחרוזות בשפת סי++ בצורה פשוטה ובהתאמה רבה לקוראים מתחילים. נדגים שימוש של פקודה מובנית וגם נרחיב על הנושא כדי להבין אותו במימדים נרחבים יותר.

## מדוע

ישנם מספר סיבות שמשתמשים יכולים לרצות לחלץ תת-מחרוזות מתוך מחרוזת גדולה. כאשר מפיצים מחרוזות ארוכות עם מידע שונה ממנה, חשוב להפריד את המידע הרלוונטי בכדי לעבוד איתו בנפרד. ניתן להשתמש בתת-מחרוזות כדי לחלק את המידע לעמודות שונות בטבלת מידע או לעסוק במידע ספציפי במטרה לחפש פרטים מסוימים.

## איך

הדרך הפשוטה ביותר לחלץ תת-מחרוזות בשפת סי++ היא עם השימוש בפקודה המובנית `substr()`. הפקודה מקבלת שני פרמטרים - מיקום ההתחלה ואורך התת-מחרוזת המבוקשת. לדוגמה:

```C++
string str = "היי לכולם!";
string sub = str.substr(0, 2); // תת-מחרוזת ראשונה החל מהתו הראשון ובאורך של 2
cout << sub; // ידפיס: הי
```

ניתן גם לחלץ תת-מחרוזות עם עזרת מערכים. לדוגמה, אם יש לנו מילון של ערים ואנחנו רוצים להדפיס את שם העיר הראשונה, נוכל להשתמש בפקודת `substr()` כדי להכין את הטקסט כרגע:

```C++
string cities[3] = {"תל אביב", "אשדוד", "חיפה"};
cout << cities[0].substr(0, 3); // ידפיס: תל א
```
