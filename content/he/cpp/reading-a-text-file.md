## למה

קריאת קבצי טקסט היא חלק חשוב מתהליך התכנות בכל שפת תכנות. קבצי טקסט מכילים מידע חשוב כגון קוד ונתונים וכן נוכחים במגוון רחב של יישומים תוכניות. קריאת קבצי טקסט יכולה לעזור לנו להבין ולעבד את המידע הזה בצורה יעילה יותר. אם אתם מעוניינים ללמוד כיצד לקרוא קבצי טקסט בשפת סי++, המאמר הזה יכול לעזור לכם.

## איך לקרוא קובץ טקסט בC++

כדי לקרוא קובץ טקסט בשפת סי++, אנו צריכים לבצע את הצעדים הבאים:

1. לפתוח את הקובץ הרצוי באמצעות הפעולה `open()` ולהזין את הנתיב המדוייק של הקובץ בצורת מחרוזת.

2. לקרוא את התוכן של הקובץ באמצעות הפעולה `getline()` או `get()`.

3. לסגור את הקובץ באמצעות הפעולה `close()`.

להלן דוגמה של קוד בשפת סי++ שכולל את הצעדים המתוארים לעיל:

```C++
#include <iostream> // כוללים את הספרייה הנדרשת 
#include <fstream> 

using namespace std; 

int main() 
{ 
    ifstream myFile; // משתנה המציין את הקובץ הרצוי 
    string line; // לאחסן את השורות של הקובץ
    myFile.open("C:\\Example.txt"); // פתיחת הקובץ באמצעות הנתיב המדוייק שלו
    while (getline (myFile,line)) // קריאת שורה מהקובץ עד שסדרת התווים מסתיימת 
    { 
        cout << line << endl; // הדפסת השורה 
    } 
    myFile.close(); // סגירת קובץ
    return 0; 
} 
```
