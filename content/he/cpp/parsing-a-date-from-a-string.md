מה ולמה?
פירושו של ניתוח תאריך ממחרוזת הוא לתרגם תאריך מופיע בפורמט שונה למבנה של תאריך בפורמט סטנדרטי. תהליך זה נחוץ כאשר רוצים לקרוא נתונים תאריך מקוריים ממקורות שונים, כמו קבצי טקסט או בסיסי נתונים. מתכנתים מבצעים ניתוח תאריך כדי להבטיח שהנתונים המשמשים לתכנות מציגים תאריך נכונה ומדויק.

איך לעשות?
מטרת הדוגמאות הבאות היא להדגים כיצד לבצע ניתוח תאריך ממחרוזת בשפת C++. התוכן יהיה מצורף בבלוקי קוד ```C++ ... ``` עם פלט דוגמתי.

### דוגמה ראשונה: ניתוח תאריך מתאריך סטנדרטי

```C++
#include <iostream>
#include <iomanip>
#include <sstream>
#include <ctime>

int main() {
  //קביעת משתנים עבור התאריך המקורי והתאריך היעד
  std::string originalDate = "12/31/2021";
  tm destDate;
  
  //המרה של המחרוזת לערך זמן
  std::istringstream ss(originalDate);
  ss >> std::get_time(&destDate, "%m/%d/%Y");
  
  //בדיקת הצלחת הניתוח
  if (ss.fail()) {
    std::cout << "ניתוח תאריך נכשל";
  } else {
    //הדפסת התאריך בפורמט סטנדרטי
    std::cout << "תאריך מנותק: " << std::put_time(&destDate, "%m/%d/%Y");
  }
  
  return 0;
}
```

פלט דוגמתי:
```
תאריך מנותק: 12/31/2021
```

### דוגמה שנייה: ניתוח תאריך מתאריך לא סטנדרטי

```C++
#include <iostream>
#include <iomanip>
#include <sstream>
#include <ctime>

int main() {
  //קביעת משתנים עבור התאריך המקורי והתאריך היעד
  std::string originalDate = "31 DEC 2021";
  tm destDate;
  
  //המרה של המחרוזת לערך זמן
  std::istringstream ss(originalDate);
  ss >> std::get_time(&destDate, "%d %b %Y");
  
  //בדיקת הצלחת הניתוח
  if (ss.fail()) {
    std::cout << "ניתוח תאריך נכשל";
  } else {
    //הדפסת התאריך בפורמט סטנדרטי
    std::cout << "תאריך מנותק: " << std::put_time(&destDate, "%m/%d/%Y");
  }
  
  return 0;
}
```

פלט דוגמתי:
```
תאריך מנותק: 12/31/2021
```

צליל עמוק:
ניתוח תאריך ממחרוזת הוא תהליך שנמצא כבר שנים רבות בסביבת התכנות. מקור הצורך בתהליך זה נובע מכך שתאריך הוא סוג נתונים מכריע ביותר שמשמש בתכניות שונות. כמו כן, כמות הפורמטים השונים לתאריך גורמת לצורך בהתאמת הנתונים המקוריים לפורמט של השימוש המיועד.

2. המרת המחרוזת לערך זמן באמצעות הפונקציה ```std::get_time```.
