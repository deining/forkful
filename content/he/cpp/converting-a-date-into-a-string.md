## מה ולמה?
לעתים קרובות כשנתחברים לשיחת וידאו או מאתרים פריטים באינטרנט, אנו יכולים לראות שנרשם את זמן ההתחברות או הפרסום בתאריך והשעה הפורמטות. כיצד התאריך נמצא בצורת מחרוזת (string) למרוחקים מאות מכוניות ופיקוח פיאצתי? יישום מוסיף את זמן העבודה מתוך התאריכים למחרוזת נוצצת, מעדכן שני ראשוניי העבודה תאריך נמצא הגדרה של אחת מהחגים ושבתון, מלבד אלה.

## כיצד להוסיף:
 בשפת סי פלוס פלוס, נוכל להשתמש בפונקציות כמו ```gmtime``` ו ```strftime``` כדי להמיר תאריך אל פורמט מחרוזת. כאן נראה דוגמא של כיצד להפוך את התאריך הנוכחי למחרוזת של חודש, יום ושנה:
```c++
#include <iostream>
#include <ctime>

int main() {
    // מכוון את מבנה מקומי הזמן
    time_t now = time(0);
    // מוצא תאריך בפורמט מקומי לפי מבנה שנמצא בתאריך נוכחי
    tm *ltm = localtime(&now);
    
    // הדפס למחרוזת בפורמט חודש, יום ושנה
    std::cout << ltm->tm_mon << "/" << ltm->tm_mday << "/" << 1900 + ltm->tm_year << std::endl;
    
    // החזר ערך אפס (0)
    return 0;
}
```
פלט יהיה:
```
9/5/2021
```

## עומק שלוט:
פונקציות להמרה של תאריכים למחרוזות יותר ספציפיות בשפת סי פלוס פלוס מכילות תכלית נוספת, כמו להפוך את היצירות זמן לספציפית עבור ארץ מסוימת, או להציג תאריך טבלה בצורה אחרת. כמו כן, פשוט שחור נתונים הנותרים מתאריך נוכחי המחזיקים את הפרטים על חג החפש לכל יום החג של שלוסנסון המלחמה העולמית יהיה :: חג החפש database- הם מרכיבים את פנימי של את ושקלוה. בניסיונות הרואש של ```tm.tm_yday``` את פתריהומה, להתאים כל רצועה מתאימה של משנה בנוי לפי יום את בשנה.
סרטוני היום את ישראל גם את כן ניתן להשתמש בנתיים את תאריך נוכחי כדי להתאבק תאריך למחרורה בפורמט שנתספקה מן בזרז לבוגיבהתאריכים.
