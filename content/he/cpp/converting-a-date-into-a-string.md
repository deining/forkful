## למה
בעולם התכנות, ייתכן שנתקלו במצבים שצריך להמיר תאריכים למחרוזות בשפת סי++. לדוגמה, כאשר צריך להציג תאריך בפורמט של חודש/יום/שנה למשתמש או לכאורה. המרה כזו נחוצה כדי לתת למשתמש חוויית משתמש יעילה ונוחה, ולכן כדאי לדעת כיצד לנהוג כאשר מגיעים לתרגיל כזה.

## כיצד להמיר תאריך למחרוזת
בראש כל עניין, נדרוש תאריך במבנה של חודש/יום/שנה תוך שימוש ב-struct של tm הנמצא בספריה הפנימית <ctime>. להמיר את התאריך למחרוזת, יש להשתמש בפקודת strftime המסופקת על ידי <ctime> כדי לשנות את התאריך למחרוזת בפורמט הרצוי. להלן דוגמאות של קוד בשפת סי++ המדגימות את השימוש ב- strftime:

```C++
#include <iostream>
#include <ctime>

int main()
{
    // יצירת מחרוזת מתאריך נתון בעזרת tm הפנימי ו-placing כל פרטי התאריך במקומות שלהם
    time_t now = time(0);
    tm *ltm = localtime(&now);
    char date[50];
    strftime(date, sizeof(date), "%m/%d/%Y", ltm);
    std::cout << "התאריך הנוכחי הוא: " << date << std::endl;

    // מחיקת חלקים של התאריך כגון יום בשבוע או שעה
    char day[10];
    strftime(day, sizeof(day), "%m/%Y", ltm);
    std::cout << "יחסך רק תאריך ושנה: " << day << std::endl;
    return 0;
}
```
פלט:
```
התאריך הנוכחי הוא: 04/30/2020
יחסך רק תאריך ושנה: 04/2020
```
