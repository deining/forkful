## למה: רק 1-2 משפטים המסבירים *למה* איש יכול להתעסק בהמרת תאריך למחרוזת. 

ההמרה של תאריך למחרוזת היא תהליך חשוב ביישומי C++ של תאריכים וזמנים. זה מאפשר למשתמש לקבל מידע על תאריך מסוים בפורמט שנוח לו, ולייצר תצוגת תאריך מתאימה להצגה למשתמשים.

## איך לעשות זאת: דוגמאות קוד ופלט דוגמתי בבלוקי קוד "```C++ ...```" 

לכדי מימוש המרת תאריך למחרוזת ב- C++ ישנם כמה אפשרויות, אבל בדרך כלל השיטה הנפוצה ביותר היא להשתמש בספריה שנקראת "strftime". הספריה הזו מכילה פונקציה בשם "strftime" שמאפשרת לנו להתמודד עם כל סוגי התבניות של תאריך ושעה.

```C++
#include <iostream>
#include <ctime>

int main ()
{
	time_t now = time(0);
	char* dt = ctime(&now);
	std::cout << "The current date and time is: " << dt << std::endl;

	// Converting date to a string using strftime
	char str[26];
	tm *gmtm = gmtime(&now);
	strftime(str, 26, "%Y-%m-%d %H:%M:%S", gmtm);
	std::cout << "The current date and time in string format is: " << str << std::endl;

	return 0;
}
```

הפלט המתקבל מהקוד הנ"ל יהיה כזה:

```
The current date and time is: Tue Sep 21 11:51:20 2021
The current date and time in string format is: 2021-09-21 11:51:20
```

בקוד הנ"ל, אנחנו משתמשים בפונקציה "ctime" כדי לקבל את התאריך הנוכחי כמחרוזת. לאחר מכן, אנחנו משתמשים בפונקציה "strftime" כדי להמיר את התאריך למחרוזת בפורמט שלנו. ניתן להשתמש במגוון רחב של תבניות כדי ליצור את המרת תאריך מדויקת לפי הצורך.
