## למה

חישוב תאריך בעתיד או בעבר הוא יכול להיות מועיל לכל מי שעוסק בתכנות בשפה C++ ומעוניין לנהל את תאריכי הימים באופן יעיל ומדויק.

## כיצד לבצע

מצאנו תאריך מאוחר יותר, בהתאם למפגש הגנקוך בהתאם לשנה הנוכחית.
```C++
#include <iostream>
using namespace std;
int main () {
  int year = 2021;
  int month = 6;
  int day = 30;
  cout << "התאריך הנוכחי הוא: " << year << "/" << month << "/" << day << endl;
  day = day + 14;
  if (day > 30) {     // בדיקה אם התאריך חורג מהחודש שלו
    day = day - 30 ;  // מעבר ליום הראשון של החודש הבא
    month = month + 1 ;
  }
  cout << "בחישוב התאריך של 14 ימים מהיום נקבל: " << year << "/" << month << "/" << day << endl;
  return 0 ;
}
```
תוצאה:
```
התאריך הנוכחי הוא: 2021/6/30
בחישוב התאריך של 14 ימים מהיום נקבל: 2021/7/14
```

## חקירה מעמיקה

החישוב של תאריך בעתיד או בעבר נעשה באמצעות מתמטיקה פשוטה וכמה דרישות פשוטות. בכדי לחשב תאריך בעתיד, אנו משתמשים במשתנים של השנה, החודש והיום הנוכחיים ומוסיפים או מחסירים ימים כמספר שתנו לנו. בהתאם לדרישות של הכריית השנה והחודשים, אנו חייבים לבצע שינויים בתאריך שלנו כדי לקבל תוצאה נכונה. זהו רק דוגמה פשוטה של חישוב תאריך ואין צורך להכיר תשתית מתמטית מיוחדת כדי לבצע את הפעולה הזו.
