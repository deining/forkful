ראשית כי: למה לחשב תאריך בעתיד או בעבר

מחשבוני תאריך הם כלי חשוב לתכנות כי הם מאפשרים לנו לחשב ולהציג תאריכים בצורה נוחה ומדויקת. חישוב תאריך בעתיד או בעבר יכול להיות שימושי במגוון רחב של תכניות, כגון יישום לוחות שנה, ייצוג תאריכים בצורה מתקדמת, ועוד.

כיצד לעשות זאת:

לחישוב תאריך בעתיד או בעבר ב- C++ ישנם מספר שיטות שונות. כדי להציג את התאריך בפורמט מסוים, נוכל להשתמש בפונקציית סטוואק להצגת תאריך ומאתיים. ניתן לראות דוגמה לכך בקטע הקוד הבא:

```C++
#include <iostream>
#include <iomanip> 
#include <chrono>
using namespace std;

int main()
{
    // חישוב תאריך בעתיד 
    auto now = chrono::system_clock::now();
    auto future = now + chrono::hours(24); // הוספת 24 שעות לתאריך הנוכחי
    time_t futureTime = chrono::system_clock::to_time_t(future);

    // הצגת התאריך בפורמט מתאים 
    cout << "תאריך תוך 24 שעות: " << put_time(localtime(&futureTime), "%F") << endl;

    return 0;
}
```

פלט:

תאריך תוך 24 שעות: 2021-12-13

בכדי לחשב תאריך בעבר, ניתן להשתמש בתוכנית דומה להצגה של תאריך בעתיד, אך עם הוספת הפונקציה `chrono::hours(-24)` כדי להחליש את התאריך.

עמוד עמוק:

חישוב תאריך בעתיד או בעבר לא מקובל רק בשביל העין. תוכלו להשתמש בו בצורות רבות ומגוונות כדי להנות מתכנות יעיל וממוסגל. ניתן להשתמש בפונקציות מוכנות כגון `chrono::hours`, `chrono::minutes`, ו- `chrono::seconds`, כדי להגדיר מרווח זמן ולהוסיף אותו לתאריך הנוכחי. בנוסף, ישנם כלים נוס
