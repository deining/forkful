## למה

רוב הפעמים אנו מתכנסים לתאריך ספציפי בעתיד או בעבר בקוד שלנו. אולם חישובים אלה לא תמיד יכולים להתרחש ככה שייתכן שנצטרך להשתמש בפונקציות מיוחדות של אתרים או ספריות נוספות. לכן, בואו נלמד כיצד לחשב תאריך בעתיד או בעבר בעזרת קוד שכתבנו בעצמנו.

## כיצד לעשות זאת

אתחל את הקוד על ידי כתיבת התאריך הקיים במשתנה כמו זה:

```C++
string startingDate = "01/01/2021";
```

כעת, באמצעות הפונקציה ```time_t mktime (struct tm * timeptr)```, אנו יכולים להמיר את התאריך למבנה ציר זמן מהנייד מלא. להלן דוגמא לכך:

```C++
tm startingDate_tm = {};
strptime(startingDate.c_str(), "%d/%m/%Y", &startingDate_tm ); // פונקציה זו ממירה סדרת תוים של תאריך למבנה תאריך וזמן בפורמט מסוים
time_t startingDate_time = mktime (&startingDate_tm);
```

לבסוף, נוסיף את מספר היחידות המיועדות להתעכבות (בימים) לתאריך המקורי כדי להגיע לתאריך סופי בעתיד או בעבר כפי שנדרש.

```C++
int numOfDaysToDelay = 7; // נמנד בעזרת משתנה לדוגמא
time_t futureDate_time = startingDate_time + (numOfDaysToDelay * 60 * 60 * 24); // נעדיף להשתמש בקבועים לכתיבה קלה וקריאה
```

והוא זה! עכשיו אנו מציגים את התאריך הסופי לחשב את התאריך בעתיד או בעבר.
