בלוגינג על תכנות Elm לקוראי עברית: 

## Why

למה: רק 1-2 משפטים מסבירים למה מישהו יבחר ליצור קובץ זמני. 

התכנות בשפת Elm יכול להיות אתגר מרתק ומאתגר, אך כשמדובר ביצירת קבץ זמני, הדבר יכול להיות בדיוק מה שאתם צריכים כדי להפתיע את עצמכם ולתכנן מחדש. הקבצים הזמניים משמשים לכמה מטרות, כגון ניהול מצב עבודה ובדיקות תוכנה. 

## How To

כיצד ליצור קובץ זמני? באמצעות פקודת file.newTempFile תוך שימוש בספריית Elm/file על מנת ליצור קובץ זמני חדש. הקוד הבא יראה דוגמה: 

```Elm
import File exposing (newTempFile)

tempFile : Result String FilePath
tempFile =
  newTempFile "elm" -- שם הקובץ עם הסיומת 
```

מעתיק את הקוד הזה לסביבת הפיתוח שלכם ושורת הפקודה מופעלת. הקוד שלנו ייצור קובץ זמני חדש בשם "elm0002" והוא ישמר בתיקיית העבודה שלכם. כעת, תוכלו לנהל את הקובץ בצורה מתאימה שתושבת את מטרותיכם.
