## עבור מה

אז למה בעצם להשתמש בביטויים רגולריים? לעילות מספקות שפה רבת עוצמה וגמישה לחיפוש ועיבוד טקסט. במקום לחפש כל מחרוזת באופן ידני, ביטוי רגולרי יכול לעזור לך למצוא תבניות מסוימות ולעזור לך לעבוד עם תבניות אלה באופן מהיר ויעיל.

## איך להשתמש

תחילה, יש צורך ליצור מחרוזת רגילה באמצעות קוד הבא:

```Elm
regex : Regex
regex = RegExp.fromString "רשימה"
```

כעת, ניתן להשתמש במחרוזת רגולרית כדי לחפש ביטויים בטקסט, לדוגמה:

```Elm
matches : Int
matches =
  RegExp.findAll regex "זו הרשימה התואמת שנמצאת בטקסט שלנו"
```

יכול להפיק את התוצאה הבאה:

```
1
```

התוצאה מציגה את מספר המופעים בטקסט המכילים את הרשימה הנתונה.

## עיון מעמיק

ביטוי רגולרי יכול להיות מורכב ממספר תווים בשפה ולכן ייתכן שתצטרך ללמוד נושאים נוספים כדי להתמקצע בשימוש בהם בצורה מלאה. אתה יכול לבדוק את דף התיעוד הרשמי של ביטויים רגולריים בשפת Elm כדי ללמוד עוד על דרכים לבנות תבניות ולעבוד איתן.
