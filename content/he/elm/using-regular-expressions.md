## על מה למה
משתמשים בביטויים רגילים בכדי לחלץ מידע מסוגים שונים כמו טקסטים, טבלאות וכו'. באפשרותך להשתמש בביטויים רגילים כדי לחלץ פרטים מוובסייטים ולבצע מערכות חיפוש.

## איך לעשות זאת
בכדי להשתמש בביטויים רגילים באלם, עליך להתקין את החבילה Regex כך שתהיה לך גישה לפונקציות רלוונטיות. להלן דוגמא של כיצד להשתמש בביטוי רגיל באלם:

```elm
-- ייבוא של החבילה Regex
import Regex exposing (..)

-- חיפוש לתת-מחרוזת של מספר פלאפון בפורמט אמריקאי
findPhoneNumber : String -> Maybe String 
findPhoneNumber inputString
    -- נמצא רק פלאפונים במספרים של 10 ספרות 
    let pattern = regex "\\d{10}" 
    in
    case find pattern inputString of 
        Ok matches -> 
             -- החזרה מערך של כל הפלאפונים שנמצאו
            List.head matches
        Err msg -> 
            Nothing
```
### פלט:
```
findPhoneNumber "המספר שלי הוא 555-123-4567"
--> Just "5551234567"
```

שים לב שכדי לממש את הפונקציה, עליך לכתוב את הביטוי הרגולרי כמו מחרוזת ולאחר מכן להמיר אותו ל- Regex.

## דיב לילכוד
כעת שניצלנו דוגמא של כיצד להשתמש בביטוי רגיל באלם, נכנס לעומק ונתבונן יותר בשימוש נפוץ פונ ייחודיות במרחב של ביטויים רגילים באלם. נתרגש מהעובדה כי במרבית המקרים ניתן להשתמש בקוד פשוט כדי לחלץ את הנתונים שאנו צריכים מטקסטים מסוימים.
