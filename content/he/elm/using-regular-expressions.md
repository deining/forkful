## איך להשתמש
כאשר מדובר בתכנות ב־Elm, ישנן מספר מצבים בהם ביטויים רגילים יכולים לסייע בכתיבת קוד. לדוגמה, אם אתם מעוניינים לחשב את כמות המופעים של מחרוזת מסוימת ברשימה של מחרוזות, ניתן להשתמש בביטויים רגילים כדי למצוא את המופעים המתאימים. למטה יש לכם דוגמאות של קוד עם הסברים כיצד להשתמש בביטויים רגילים ב־Elm:

```Elm
-- פונקציה זו מקבלת רשימה של מחרוזות ומחזירה את המספר של המופעים של המחרוזת הנתונה
countOccurences : List String -> String -> Int
countOccurences strings search =
  List.length (Regex.match Regex.All (Regex.regex search) (String.join "" strings))
```

כפי שאתם רואים בקוד לעיל, יש לנו פונקציה שמקבלת רשימה של מחרוזות ומחפשת את המחרוזת המתאימה באמצעות ביטויים רגילים. פונקציה זו מחזירה את המספר של המופעים שנמצאו.
