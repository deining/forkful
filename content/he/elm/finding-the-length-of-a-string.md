## למה
בכתיבת קוד בשפת אלם, ייתכן שנתקלו במצבים בהם עלינו להציג את אורך המחרוזת שלנו. המטרה העיקרית שאנשים מחפשים כשמתעסקים בקוד היא לאפשר עבודה קלה ויעילה עם מחרוזות. לכן, מציאת אורך המחרוזת הוא נושא חשוב ושימושי בכתיבת קוד בשפת אלם.

## איך לעשות זאת
תחילה, נצטרך להגדיר משתנה שיכיל את המחרוזת שאנחנו רוצים לבדוק את אורכה. לדוגמה, נשתמש במחרוזת "שלום עולם" כדי להדגים את התהליך. לאחר מכן, נשתמש בפונקציה `String.length` כדי למצוא את אורך המחרוזת.

```Elm
import String exposing (length)

string = "שלום עולם"
lengthOfString = length string
```

התוצאה המצופה תהיה 9, מכיוון שיש תשע אותיות במחרוזת "שלום עולם". מה שיחזיר לנו את הערך של אורך המחרוזת הוא משתנה חדש בשם `lengthOfString` עם ערך של 9.
