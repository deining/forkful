## למה:
למה צריך להתעסק בפרויקט חדש? פרויקטים חדשים מאפשרים לנו להתחיל משהו חדש ומרגש, לחקור טכנולוגיות חדשות ולפתח כישורים חדשים. כמו כן, השתתפות בפרויקט משותף עם אנשים אחרים יכולה לסייע לנו לשרטט קהילה וליצור חומר לימודי חשוב לאנשים אחרים.

## איך להתחיל:
כתיבת קוד ב-Elm פשוטה וקלה ללימוד. אם אתם מתחילים פרויקט חדש, אתם יכולים להתחיל עם השתמשות ב[ Elm-REPL ](https://guide.elm-lang.org/install/elm.html). להלן דוגמאות לכתיבת קוד ותוצאות בתוך בלוקי קוד "```Elm ... ```":

```Elm
-- צור פונקציה בעזרת הפרמטרים name ו-age והחזר מחרוזת המברגת את שמם וגילם
sayHello : String -> Int -> String
sayHello name age =
    "היי " ++ name ++ ", אתה בן " ++ toString age ++ "!"

-- קריאה לפונקציה עם שמות וגילים שונים
sayHello "שרה" 27 --> היי שרה, אתה בן 27!
sayHello "יעקב" 35 --> היי יעקב, אתה בן 35!
```
כפי שאתם רואים, אפשר ליצור בקלות פונקציות ולקרוא להם עם ערכים שונים. בכדי להתאים את התוכן של האתר שלכם לצרכים ספציפיים, כדאי להתנסות עם כמה פונקציות ולחפש משאבים עשירים אחרים ללימוד כיצד להשתמש ב-Elm.
