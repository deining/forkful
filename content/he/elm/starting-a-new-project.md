## למה

אז למה להתחיל פרויקט חדש ב־Elm? התשובה פשוטה - בגלל שזה שפת תכנות מדהימה שמיועדת לפיתוח יישומים מורכבים ואלגנטיים עבור האינטרנט המודרני. אם אתה מחפש כלי חזק ונוח לשימוש, Elm הוא הווריאנט המושלם עבורך.

## איך לעשות זאת

אם אתה מעוניין להתחיל פרויקט חדש ב־Elm, הנה כמה טיפים כדי להתחיל:

```
Elm מודול Main exposing (..)
import Html exposing (text)

main =
  text "שלום עולם"
```

כאן אנו מייצגים את המודול הראשי של Elm ומתאימים אותו ל־"מייצגים" המאפשרים לנו להשתמש בכל המודולים של Html. בקוד זה אנו מדפיסים פשוט כתב "שלום עולם". הנה כמה נתונים הקלט שלנו ותוצאת הפלט שמקודדים אותו:
```
Input: שלום עולם
Output: שלום עולם
```

לכל פרויקט חדש יש שני חלקים חשובים: קוד וקובץ מחבר. כדי לפתוח פרויקט חדש, יש להוסיף את הפקודה "elm-translation" אל תקיית הפרויקט ולהשתמש בכלי הכתיבה בקובץ המטא. בנוסף, ניתן להצביע על הכיוון שמייצג את התקציב המבוקש כדי להשתמש בכלולי הקוד האופטימליים והתאימים ביותר.
