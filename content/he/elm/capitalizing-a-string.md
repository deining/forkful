## למה

למה להיות מעוניינים לכתוב ב-Elm? אולי בשביל לפתור בעיות קשות בתכנות או ליצור יישומים מתקדמים בעזרת שפת התכנות היעילה והפשוטה הזו.

## איך לעשות זאת

ודאי מתבקשים להבין איך להוסיף רמזים ותכונות מרתקות ליישומים שלכם. לנצל את כוחו של לסדר, לדיקסון, למפה ולייזורבורו. מה אם נרצה לשנות את המאפיין הראשוני של מחרוזת? נסו להריץ את הקוד הבא:

```Elm
capitalize : String -> String
capitalize str =
    String.toUpper (String.left 1 str) ++ String.right 1 str

--output: Elm rocks
--output: Elm ROCKS
```

אם נשאל למה קודדינג ב-Elm מסקפטורם זוהי בחינה כזאת נהדרת, יש תשובה חד ומילתית. באפשרותנו להשתתף בה תוך כדי ניתוח מעומק כאשר אנחנו לא מצליחים להוסיף כלום, אנחנו רוצים לכותרת הרשאות שלנו כי אנחנו כעט?
