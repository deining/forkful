### למה
כשאנו כותבים קוד בכל שפת תכנות, ייתכן שנתקל בשגיאות ובאתגרים. בכדי לאתר את הבעיות בקלות ולפתור אותם במהירות, חשוב להשתמש בכלי שיעזור לנו להבין מה קורה בעת ריצת הקוד. הדפדפן שלנו מציע שימוש בתכונת "הדפסת פלט גיאומטרית" או "printing debug output" המאפשרת לנו לראות את הפלט הנוצר תוך כדי ריצת הקוד. ניתן להשתמש בתכונה זו על מנת להבין מה קורה במהלך הפעולה של הקוד ולאתר ולתקן את השגיאות.

### איך להשתמש בתכונה
כדי להדפיס את הפלט בקונסול של הדפדפן בעת ריצת הקוד, ניתן להשתמש בפונקציה "Debug.log" שקיימת בספריית "Debug" של אלם. ניתן להשתמש בפונקציה זו על מנת להדפיס מידע שחשוב לנו במהלך הריצה, כגון ערכי משתנים, הודעות או שגיאות.

```Elm
import Debug exposing (log)

log "Hello World!" -- הדפסת המחרוזת "Hello World!" בקונסול
```

ניתן להשתמש גם בפונקציה "Debug.todo" שקיימת באותה ספרייה, כדי להדפיס הודעה כאשר תיאוריתתם לכתוב קוד שעוד לא רלוונטי. פעולה זו עשויה לסייע לנו לשלב קוד ולהתקדם בפרויקט במהירות.

```Elm
import Debug exposing (todo)

todo "Implement this function" -- הדפסת הודעה שהפונקציה טרם מיושמת
```

הודעות וערכי משתנים שמודפסים באמצעות הפונקציות "Debug.log" ו-"Debug.todo" יופיעו בקונסול של הדפדפן בזמן ריצת הקוד.
