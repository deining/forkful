## למה

רבים מתעניינים לכתוב קבצי טקסט בתוכניות מחשב, במיוחד מפעילי התכנות. כתיבת קבצי טקסט מאפשרת לנו ליצור מבנה סדרתי וברור של נתונים, או ליצור קוד תכנותי כדי לשמור על מידע שלנו. אם אתה נהיה אורגני וקפדן בכתיבת קבצי טקסט, זה יכול להקל על החיים שלך כתוכניתן.

## איך לעשות זאת

לכתיבת קוד תכנותי בעזרת Elm, אנחנו נדרשים ליצור קובץ טקסט עם קוד התכנות שלנו. לתחילת כל קבצי טקסט ב-Elm יש זרימת עבודה קבועה שכוללת דברים כמו יבוא של ארכיטקטורת קובץ טקסט, ייבוא של פונקציות ועוד. הנה דוגמאות חשובות שיעזרו לך להתאים את הקוד שלך לקובץ טקסט:

```Elm
module Main exposing (..)

import Architecture.Text
import Html.App exposing (beginnerProgram, program)
import Html exposing (div, text)
```

כאשר הקוד שלנו מוכן, נוכל ליצור את הקבצי טקסט שלנו על ידי ייבוא של "Browser.document.text" בתוך קובץ Main ולהקליד את הקוד הרלוונטי. לדוגמה:

```Elm
main : Program () Model Msg
main =
    beginnerProgram
        { model = initialModel
        , view = view
        , update = update
        }
```

לאחר מכן, נוכל לשנות את הקוד שלנו בהתאם לצרכים שלנו ולבנות את הקבצי טקסט בדיוק כמו שאנחנו רוצים.
