## למה

לכתוב קובץ טקסט בתכנות בשפת Elm יכול להיות כלי חשוב במחקר, פיתוח מוצר או אפילו בפשטות עבודה עם נתונים מורכבים. הוא מאפשר לנו לנהל ולאחסן מידע בצורה מהימנה ומובנת יותר, וכן לאפשר יעילות ויכולת חיפוש מתקדמות.

## כיצד לעשות את זה

לכתוב קובץ טקסט בשפת Elm הוא פשוט וקל, וניתן לעשות זאת בעזרת פקודת "writeFile" בעץ התוכנית. לדוגמה, נרצה ליצור קובץ טקסט חדש עם הטקסט שלנו ולאחר מכן לקרוא אותו ולהדפיס אותו למסך. הנה כמה קוד דוגמה:

``` Elm
import File

writeAndReadFile =
  let
    myText = "זוהי קבצי הטקסט"
  in
    File.writeFile "myFile.txt" myText
      |> andThen (\_ -> File.readFile "myFile.txt")
      |> andThen (\file ->
        case file of
          Err err ->
            -- הדפסת השגיאה אם יש כזו
            Debug.crash err

          Ok text ->
            -- הדפסת הטקסט מהקובץ למסך
            Debug.log "myText" text
      )
```

פקודת "writeFile" מקבלת שני פרמטרים - שם הקובץ והטקסט שנרצה לכתוב לתוכו. לאחר מכן, אנו משתמשים בפקודת "readFile" כדי לקרוא את הקובץ שנכתב ולהדפיס את הטקסט למסך. תמיד חשוב לבדוק לשגיאות ככל שנדרש, כמו שראינו בתוך ה-"case" statement.
