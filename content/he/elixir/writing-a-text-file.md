מדוע: תכתוב אסמכתא רגילה זו מהווה אמצעי יעיל לארגון ושמירה על קוד ומידע.

כיצד: בסעיפי זה תוכלו לראות דוגמאות קוד ופלטים לפי שפת איליקסיר תוך שימוש בבלוקי קוד "```Elixir ... ```".

## Why

כתיבת אסמכתא רגילה היא אחד הדרכים הטובות והיעילות ביותר לארגון ושמירה על קוד ומידע. על ידי כתיבת טקסט בקובץ, ניתן ליצור עץ תיקיות וכתביות, המאפשרים לך לסדר את הקוד שלך בצורה נקייה ומסודרת.

## How To

הנה מספר דוגמאות קוד פשוטות לכתיבת אסמכתא בשפת איליקסיר:

```Elixir
# יצירת קובץ חדש וכתיבת טקסט לתוכו
file = File.open("blog_post.md", [:write])
IO.write(file, "זוהי התחלה של כתיבת אסמכתא בשפת איליקסיר.")
File.close(file)

# הוספת טקסט לקובץ קיים
file = File.open("blog_post.md", [:append])
IO.write(file, "הנה עוד משפט שתוכלו להוסיף לאמצע הקובץ.")
File.close(file)

# קריאת תוכן קובץ
file = File.open("blog_post.md")
IO.read(file)

# הוספת תיקיות לארגון קוד
Dir.mkdir("lib")
Dir.mkdir("test")
```

טקסט הפלט של הדוגמאות הנ"ל יכול להיראות כך:

```text
זוהי התחלה של כתיבת אסמכתא בשפת איליקסיר.
הנה עוד משפט שתוכלו להוסיף לאמצע הקובץ.
```

ניתן לראות כי הטקסט הנוסף נוסף לסוף הקובץ שלנו.
