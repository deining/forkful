## למה
Escription כתיבת קובץ טקסט היא כלי חשוב בתחום התכנות. היא מאפשרת לנו ליצור קוד קריא יותר ולנהל אותו בצורה מסודרת. כתיבת קובץ טקסט באיקסיר מאפשרת לנו ליצור קוד שיכול להיות מרוכז בצורה טובה יותר וגם לשתף אותו עם אחרים בצורה קלה ומהירה.

## כיצד ל
```Elixir
IO.write("hello.txt", "שלום עולם")
```

מקור הקוד הנ"ל ייצור קובץ טקסט חדש בשם "hello.txt" ויכתוב בתוכו את המחרוזת "שלום עולם". כתיבת קובץ טקסט באיקסיר מאוד פשוטה ונוחה. כדי לקרוא מהקובץ ניתן להשתמש בפונקציות כמו `File.read/1` או `IO.inspect/1`.

## יצירה מקוונת
בנוסף לכתיבת קובץ טקסט למחשב האישי, ניתן גם ליצור קבצים בצורה מקוונת באמצעות אינטרנט. פונקציות כמו `HTTPoison.post/4` ו `HTTPoison.put/4` מאפשרות לנו לשלוח בקשות HTTP ישירות לשרת וליצור קבצים על המחסן הנמצא שם.

## צעדים נרחבים
כתיבת קובץ טקסט באיקסיר היא מעבר בקטעים כיף ושינות הקוד שלך. ניתן להשתמש בפונקציות כמו `File.write/3` ו`Path.join/2` כדי ליצור קבצים במקומות מסוימים בסביבת המחשב. ניתן גם להשתמש בספריות כמו "ExFile" כדי לקבל יתרונות נוספים והרחבה של היכן קבצים ניתנים ליזום.
