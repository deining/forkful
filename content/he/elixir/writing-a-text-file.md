## מה ולמה?
כתיבת קובץ טקסט היא הפעולה של יצירת קובץ עם תוכן מללי. תהליך זה נחוץ כאשר תרצו לשמור מידע מסוים ליצירת מאגרי נתונים, תיעוד קוד או כתיבת קובץ תצוגה שעוצב בפורמט טקסט. מתכנתים משתמשים בכתיבת קבצי טקסט כדי לאחסן ולארגן מידע בצורה קלה ונוחה.

## איך לעשות זאת:
```
Elixir File.write("file.txt", "כתיבת קוד שלוש מילים") 
"ok"
```

## יצירה בעומק: 
התוכן של הקובץ ניתן לפתיחה ועריכה בכל תוכנת טקסט רגילה או בעזרת שפות תכנות אחרות כמו בקובץ HTML בעזרת Visual Studio. ניתן לכתוב קבצי טקסט גם בשפות אחרות כגון JavaScript, Python, ועוד. אם אינכם נמצאים בסביבת פיתוח, ניתן ליצור קובץ טקסט במחברת טקסט כמו באפליקציות כגון Notepad או TextEdit שבתכונות המתקדמות שלהן ניתן לעשות כתיבת קוד עם סיומת .exs.

## ראו גם: 
מידע נוסף על כתיבת קבצי טקסט והשתמשות בהם במנוע החיפוש גוגל. ותוכלו למצוא מגוון רחב של שפות תכנות שניתן לכתוב קבצי טקסט בהן, תלוי בסביבת הפיתוח שלכם ובלשוניות התוכנית שאתם משתמשים בהם.
