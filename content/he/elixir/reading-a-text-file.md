## למה

נכתוב היום על קריאת קובץ טקסט ב- Elixir. קריאת קבצים היא פעולה חשובה בתכנות מחשבים וכדי להבין היטב את הסיבה לכך, אנו נעסוק בנושא במאמר הזה.

## איך לעשות זאת

ראשית, נציין כי קריאת קבצים ב- Elixir נעשית על ידי הפעולה `File.read/1` המקבלת כפרמטר את הנתיב לקובץ שנרצה לקרוא. ננסה זאת עם קובץ טקסט פשוט המכיל את המילים "שלום עולם".

```Elixir
content = File.read("hello_world.txt")

IO.puts(content)
```

כאן אנו קוראים את תוכן הקובץ ומדפיסים אותו למסך. התוצאה תהיה "שלום עולם".

כעת, ננסה לקרוא קובץ טקסט עם מספר שורות:

```Elixir
multi_line_content = File.read("sample.txt")

IO.puts(multi_line_content)
```

כדי לקרוא קובץ שמכיל מספר שורות, אנו נשתמש בפעולת `File.read_lines/1` כדי לקרוא את התוכן כשורות נפרדות.

```Elixir
file_lines = File.read_lines("sample.txt")

Enum.each(file_lines, fn line -> IO.puts(line) end)
```

כאן אנו משתמשים ב- `Enum.each` כדי לעבור על כל אחת משורות הקובץ ולהדפיס אותן למסך.

## כיול מעמיק 

בנוסף לפעולות `File.read/1` ו- `File.read_lines/1`, ישנם גם פעולות נוספות שניתן להשתמש בהן לקריאת קבצים. למשל, ניתן להשתמש בפעולת `File.stream!/1` המאפשרת יצירת מאפיין זרימה של תוכן הקובץ.

```Elixir
stream = File.stream!("sample.txt")

Enum.each(stream, fn line -> IO.puts(line) end)
```

פעולת `File.stream!/1` תופעל את הקובץ בתור "זרימה" כך שניתן יהיה לעבוד עם התוכן בצורה ידידותית לזכרון. כמו כן, ניתן להשתמש גם בפעולות נוס
