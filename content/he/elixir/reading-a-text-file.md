## למה
אנשים יעסוקו בקריאת קובץ טקסט כדי לקבל מידע ולעבד אותו באופן מבוקר ומהיר.

## איך לבצע זאת
נדון כאן באליקסיר גירסה נוכחית 1.12 גירסה איך לקרוא קבצי טקסט ולבצע עיבודים עליהם באמצעות קוד קומפטי וברור.

```Elixir
# לקרוא את הקובץ ולאחסן את התוכן שלו בתוך משתנה
file = File.read("file.txt")

# לחלץ את השורות של הקובץ כתוויות ולהציג אותן בתוך מערך
lines = String.split(file, "\n")

# לפרק את התוכן של התוויות לתתי תווים ולהציג אותם באמצעות פקודת IO
for line <- lines do
  chars = String.graphemes(line)
  IO.puts(chars)
end
```

### פרטי קריאת קובץ מתמטיקה
קבצים נדרשים עבור מספר תכנים, ואתם תוכלו לקרוא אותם בקלות תוך הכירות עם הפקודות הנדרסות. כל פקודה יכולה ליצור מצב חדש מתוך קובץ נתונים, אשר נשמרים כסדר כדי להעביר מידע נכון

## העמק נמוך
קריאת קובץ טקסט היא אחת מהמשימות היסודיות ביותר של מפתחי אליקסיר. למרבה המזל, במידתה כזו ניתן לעשות בקלות ולאחר מכן להעביר את המידע לפקודות נוספות כדי לעבד אותו. קריאת קובץ טקסט גם יכולה להיות מאתגרת כאשר התוכן מכיל תווים מיוחדים או פורמטים מיוחדים.
