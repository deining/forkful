# למה:

בעולם התכנות, לעתים קרובות נדרש ליצור אפליקציות המתיימרות להיות מבינות ולתאר את התוכן בצורה מובנת למשתמש. אחד הרכיבים החשובים ביותר של כך הוא להיות מסוגל למדוד ולנתח מחרוזות באופן יעיל. במאמר זה נתמקד בכיצד למצוא את אורך המחרוזת בשפת אליקסיר.

# כיצד לבצע:

האלכסיר מספק כמה דרכים לחישוב אורך מחרוזת. נציג שתי דרכים פופולריות כאן.

### אמצעי 1: השתמש בפונקציית len
הפונקציה "len" באלכסיר מחזירה את אורך המחרוזת הנתונה כפרמטר. לדוגמה:

```elixir
len("זהו מחרוזת")
```
הפלט שיחזיר יהיה 12, מכיוון שישנם 12 תווים במחרוזת הנתונה.

### אמצעי 2: השתמש בפונקציות באופן ידני
ניתן גם להשתמש בשיטה ידנית כדי לחשב את אורך המחרוזת באמצעות פונקציות ופעולות של שפת אליקסיר:

```elixir
def count(string) do
  result = 0
    string |> String.graphemes() |> Enum.reduce(result, fn _ -> acc + 1 end)
end
```

הפונקציה הזו מייצרת תצורת אורך למחרוזות על ידי ספירת כל מילה במחרוזת והחיבור שלהם. תוצאת הפונקציה הזו תהיה זהה לפלט של הפונקציה len שנתארה למעלה.
