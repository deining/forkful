## למה

בעולם התכנות, ייצור קבצים זמניים הוא טכניקה נפוצה עבור מתכנתים לשימוש בפעולות בלתי תקינות כמו חיבור לבסיס נתונים או כתיבה לדפדפן. בעזרת יצירת קבצים זמניים, ניתן לבדוק את הפעולות שלנו ללא שינוי על הקבצים המקוריים.

## איך לעשות זאת

תחילה, נשתמש בשיטת `File.open!/2` כדי לפתוח קובץ חדש בתיקייה המתאימה. למשל:

```
Elixir:
{:ok, file} = File.open!("temp_file.txt", [:write])
```

ניתן לראות שבשימוש באופן עצמאי, ניתן ליצור קובץ שנקרא `temp_file.txt` בתיקייה הנוכחית. נזיז לקצה של הקובץ כדי להוסיף תוכן, למשל `"This is a temporary file."` ולשמור אותו באמצעות גישה לפונקציות כמו `IO.write/2` או `IO.puts/1`. לדוגמא:

```
Elixir:
IO.write(file, "This is a temporary file.")
```

לסיום, זה חשוב להכניס את `File.close/1` בסוף הפעולה כדי להפעיל את השינויים. למשל:

```
Elixir:
IO.write(file, "This is a temporary file.")
File.close(file)
```

ניתן לבצע פעולות נוספות, כמו קריאה וכתיבה לקובץ זמני, ככל שנרצה עד שנשתמש בפונקציות שמתייחסות למנגנוני הקבצים הזמניים.
