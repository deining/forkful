## למה

יצירת קובץ זמני יכול להיות כלי מועיל בכתיבת קוד, ניתן להשתמש בו כדי לבדוק פרטים נוספים בתהליך ולאחר מכן לכתוב את התוכנית הסופית.

## כיצד

```Elixir
file = File.tempfile!
File.write(file.path, "שלום עולם!")
File.read(file.path)
```

הקוד לעיל יוצר קובץ זמני, כותב לו תוכן וקורא אותו. הנה הפלט המתקבל:

```
{:ok, #o(/tmp/plug-1587)  }
"שלום עולם!"
```

כאן ניתן לראות שהפונקציה `File.tempfile!` מחזירה משתנה המכיל את נתיב הקובץ הזמני שנוצר. באמצעות הפונקציה `File.write` מכתבים לקובץ את הטקסט "שלום עולם!" ובאמצעות הפונקציה `File.read` קוראים את הקובץ ומחזירים את התוכן שבו.

בנוסף לפונקציות המובנות של Elixir, ישנן גם ספריות נוספות כמו `Tempfile` שמספקות אפשרויות נוספות ליצירת קבצים זמניים בצורה יעילה יותר.

## צלילה מעמוקה

יצירת קובץ זמני באמצעות הפונקציה `File.tempfile!` משתמשת בתהליך שהקרנו "פתרון במלוא הפוטנציאל". כלומר, תהליך פרותות ועקביות המבצע מכן סימולציות ומתן הערכים הנכונים לפי הספקיפיקציות שהוגדרו מראש.

בכל מקרה, יצירת קובץ זמני יכולה לעזור בניסיונות של בדיקה והנבחנות של פרטים נוספים בתהליך והעברתם לפן אוטומטי וממונח.
