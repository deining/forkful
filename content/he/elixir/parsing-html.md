## מה ולמה? 
פירוק HTML הוא תהליך שבו מידע מקודד בשפת HTML נמסר ונעודד לתבנית לקריאה על ידי מחשבים. למה תרצו לעשות זאת? אנשי תכנות משתמשים בפירוק HTML כדי לקרוא נתונים מעמודי אינטרנט ולהשתמש בהם ביישומים שלהם.

## איך לעשות?
קוד לדוגמה ופלט ממוקזם תוך שימוש בבלוקים של קוד ```Elixir ... ```

כדי לפרוק HTML ב-Elixir, אתם צריכים להשתמש בספרייה המובנית הנקראת HTML, כאשר קל לייבא לכוונת השימוש. תחילה, פתחו את הקובץ המיועד למסמך והוסיפו את השורה הבאה לתחילתו:

```elixir
use HTML
```

אז, אתם יכולים להשתמש בפונקציות parse/1 כדי לקבל את נתוני התרחיש מהעמוד המקורי ב-HTML כמובן. אתם יכולים לראות דוגמאות נוספות לקוד באתר הרשמי של Elixir.

## חפירה עמוקה
HTML פותח בשנת 1993 על ידי טימ ברנרס-לי וולם-קון ראשונה. במאמרים נוספים, שילוב טכנולוגיות כמו סקריפטים ו-CSS פיתחו והוסיפו לשפת התכנות המקורית. מאז הפיסתי המקרוב לפירוק התאמה לשינויים החדשים.

כיוון שפירוק HTML כמו שלשה ברצונכם אתם להציג את הנתונים מעמודי האינטרנט לתוכניות שלכם, ישנם אלטרנטיבות למעבר ל-Elixir כדי לעשות זאת. דוגמאות בתוכנית זמינים כגון phpQuery, nokogiri בשפת רובי ו-Beautiful Soup ב-Python.

## ראה גם
לכבוד טכנולוגיות צפויות הניווט את האתר הרשמב האתר ותיז

כך, קיבלתם סקירה כללית של פירוק HTML ב-Elixir. ישנן הרבה שיקים שבהם אתם יכולים להשתמש כדי לקבל נתונים מעמודי אינטרנט ולאחר מכן לעשות איתם מה שתרצו.
