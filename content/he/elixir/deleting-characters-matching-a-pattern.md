# למה

למדים מה זה ראשית, למה משתמשים בהסרה של תווים המתאימים לתבנית?
מכיוון שזה כלי חשוב בתכנות עם אליקסיר, הוא מאפשר למתכנתים לטפל בתווים מיוחדים או להסיר כל תו חסר צורך מהמילה או המחרוזת.

# איך לעשות זאת

כדי לעסוק בהסרת תווים המתאימים לתבנית, ניתן להשתמש בפונקציית `Regex.replace/3` של אליקסיר. תחילה, נגדיר את התבנית או המילה שברצוננו למחוק, למשל, `regex = ~r/[aeiou]/` תמצא כל תו המתאים לתבנית הזאת ואז נשתמש בפונקציית `Regex.replace/3` לכלול את כל התווים מהמחרוזת הנתונה מראש.

כך ניתן להסיר את כל האותיות כאשר משתמשים בתבנית זו:

```Elixir
iex> str = "אליקסיר"
iex> regex = ~r/[aeiou]/
iex> Regex.replace(str, regex, "")
"לקסיר"
```

יש גם אפשרות להשתמש ב `String.replace/3` אם נרצה להשתמש בתווים פשוטים, למשל:

```Elixir
iex> str = "אליקסיר"
iex> String.replace(str, "א", "")
"ליקסיר"
```

## טיפים חשובים

- ניתן להשתמש גם בפונקציות נוספות כגון `Regex.replace_first/4` או `String.replace/4` כדי למחוק רק אות אחת או כדי להתאים למדוייק לתבנית.
- יש גם אפשרות להשתמש בתבניות רגולריות מתקדמות בעזרת הפונקציות בספריית `Regex`.
- כדי לקבל מידע נוסף על תבניות רגולריות והשימוש בהן באליקסיר, ניתן לראות את [תיעוד אליקסיר לתבניות רגולריות](https://hexdocs.pm/elixir/Regex.html).
