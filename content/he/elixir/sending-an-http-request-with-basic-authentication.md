## מדוע

במאמר זה, נלמד כיצד לשלוח בקשת HTTP עם אימות בסיסי באמצעות שפת תכנות אליקסיר. השימוש באימות בסיסי הוא חשוב כאשר אנו צריכים לאמת את המשתמש ששולח את הבקשה לשרת, כדי לקבל גישה למידע סודי או לבצע פעולות בשם המשתמש.

## כיצד לעשות זאת

בשפת תכנות אליקסיר, ישנן מספר דרכים לשלוח בקשת HTTP עם אימות בסיסי. הראשונה היא להשתמש בספריית HTTPoison, המאפשרת לנו ליצור בקשות HTTP ולקבל תוצאות מהשרת. נשתמש בפונקציה `HTTPoison.get/3` כדי לשלוח בקשת GET עם אימות בסיסי. בפרמטרים של הפונקציה, נעביר את כתובת האתר, את שם המשתמש ואת הסיסמה. נכתוב קוד בפורמט הבא:

```elixir
response = HTTPoison.get("https://www.example.com", [basic_auth: {"username", "password"}])
```

בתוצאה המצורפת לפניכם, ניתן לראות את התגובה מהשרת שמכילה את המידע שחזר מהבקשה.

```
%HTTPoison.Response{
  status_code: 200,
  body: "...",
  ...
}
```

ישנן עוד ספריות באליקסיר שניתן להשתמש בהן עבור בקשות HTTP עם אימות בסיסי, כגון Hackney ו- HTTPotion.
