# למה

תאר לעצמכם שאתם צריכים להשוות בין שתי תאריכים בתכנות ב-Elixir. השוואה בין שתי תאריכים יכולה להיות מועילה בעבור עיבוד נתונים, יישומי אתרים, אורכסטרציית מבני נתונים ועוד. מבינים את התאריכים הקודמים יכולים להיות מאתגר כיוון שהם מייצגים מידע מורכב. במאמר הזה נראה כיצד לבדוק את שיווי השתיים של שתי תאריכים שונים ב-Elixir ולמה זה משנה.

# איך לעשות

באמצעות יומרוטנים ב-Elixir, ניתן לבדוק את התאריכים ולהשוות ביניהם, באמצעות כמה דרכים שונות:

```
Elixir -- v1.10.0
19:39:49.674 [info] 2020-10-12

```

שימו לב שהתאריך מבוטא בפורמט המקובל של רבועים שמפרידים על ידי נקודות. ישנן כמה דרכים נפוצות לבדוק את שוויי התאריכים ב-Elixir, דוגמאות לנקודות מסוימות מוצגות להלן:

```
# בדיקת התאריך הקודם לתאריך עכשווי
iex> Date.before?(~D[2020-10-12], ~D[2020-10-15])
true
iex> Date.before?(~D[2020-10-12], ~D[2019-10-12])
false
```

```
# במידה ואנחנו רוצים לבדוק האם שתי תאריכים זהים
iex> Date.same?(~D[2020-10-12], ~D[2020-10-12])
true
iex> Date.same?(~D[2020-10-12], ~D[2019-10-12])
false
```

```
# בדיקת התאריך הבא לתאריך שנתן
iex> Date.after?(~D[2020-10-12], ~D[2020-10-15])
false
iex> Date.after?(~D[2020-10-12], ~D[2019-10-12])
true
```

שימו לב שבכמה מצבים יש להריץ את הפקודות שוב כדי לראות את התוצאות הרלוונטיות במכשיר שלכם.
