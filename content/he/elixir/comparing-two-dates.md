## למה

על דברי: אם אתה תקרא את המאמר הזה, סביר להניח שאתה מתעניין בקוד מכוכב Elixir. למה המתכוונים למצב שבו מתבצעים השוואות בין שתי תאריכים? בפוסט הזה, אני אלך על ידי כמה דרכים שבהן ניתן להחיל את השרשרת השלישית לשחזרות השוואת תאריך.

## איך לעשות

הכי פשוט זה להתחיל עם דוגמת בסיס של ההשוואה של שני תאריכים. נכתוב את הקוד הבא בתוך ```Elixir ``` בלוק:

```Elixir
d1 = ~D[2021-10-15]
d2 = ~D[2021-10-20]

IO.inspect d1 < d2 #output: true
IO.inspect d1 > d2 #output: false
IO.inspect d1 == d2 #output: false
```

אם אתה מטפל בטסקי לפני שאתה מעביר לחוק, טופל בזמן אחר מלפני את הגובה את גובה > Make

דוגמה נוספת מעט יותר מתקדמת שבה אנו משווים גם את השעות והדקות של התאריך. נרשום את זה בתוך הבלוק ```Elixir ``` כך שנוכל לראות את התוצאות של כל השוואה:

```Elixir
d1 = ~D[2021-10-15 15:30:00]
d2 = ~D[2021-10-20 12:00:00]

IO.inspect d1 < d2 #output: true
IO.inspect d1 > d2 #output: false
IO.inspect d1 == d2 #output: false
```

כמו שאתה רואה, כדי לבצע השוואת תאריך דרוש רק כמה שורות קוד פשוט, תוך שימוש בתכונות של השרשרת השלישית של Elixir. אפשר להשתמש גם בפונקציות נוספות כגון `Date.compare` או `Date.diff` עבור השוואות וחישובי הפרשים בין תאריכים.
