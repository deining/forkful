# למה

יתרונות של עבודה עם YAML כתובים כאן

למה לעבוד עם YAML בתוך קוד המכיל אליקסיר

התאמה מושלמת לעבודה של אליקסיר כפי שהכניסו זה בגורמי ה-Native Erlang. כמו כן, YAML מכיל עלויות נמוכות יחסית לתפעול וקורא כתוצא של קוד הנושא מוחשי

# איך

לעזור לעבוד עם YAML בדרך הבאה:

- פתחו קובץ קוד חדש באליקסיר ושמרו אותו כקובץ YAML.
- כבדו את הקוד שלכם תוך שימוש בתוספת `Yaml` ב `mix.exs` שלכם.
- כעת, יצרו משתנה חדש בשם `data` והקציתו לו ערכים בפורמט YAML של כולם.
- הוסיפו קוד כדי להציג את הנתונים הסרוו את אתה `Put` החדש שלכם, ושימוש בפונקציות `yaml.dump` להמרת הנתונים שלכם לפורמט YAML. 

הנה דוגמה של קוד באליקסיר עם פלט בפורמט YAML:

```Elixir
import Yaml

data = %{name: "Jane", age: 28}

yml = Yaml.dump(data)

IO.puts yml
```

פלט:

```YAML
---
name: Jane
age: 28
```

כעת, אתם יכולים לשנות או לקרוא את הנתונים שלכם בפורמט YAML כפי שתרצו.
