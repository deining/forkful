# למה

מדוע היה לך צורך לבדוק אם תיקייה קיימת? במאמר הזה נלמד כיצד לבדוק אם תיקייה קיימת באמצעות פונקציות פשוטות ב-Elixir.

## איך לבדוק אם תיקייה קיימת

בדיקת קיומה של תיקייה היא משימה פשוטה ב-Elixir. נבדוק זאת בעזרת הפונקציה File.exists? ונעבוד עם הנתונים תוך שימוש בתנאים if/else ותנאי הבדיקה המתאים לערך החזרה. הנה דוגמא של קוד באמצעות הפונקציה File.exists? המחזירה ערך בוליאני.

```Elixir
if File.exists?("תיקיית הקוד") do
  IO.puts "תיקייה זו קיימת"
else
  IO.puts "תיקייה זו לא קיימת"
end
```

כלומר, אם נרצה לבדוק אם התיקייה "תיקיית הקוד" קיימת, הפונקציה תחזיר את הערך true והבלוק if יודפס כך שתאתר את התיקייה זו קיימת. אם התיקייה לא קיימת, הערך שיוחזר יהיה false והבלוק else יודפס, מתריע שאין לתיקייה בסיס מקורי.
