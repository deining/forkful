## למה
כאשר אנחנו פותחים אפליקציות ותוכניות, ייתכן שנפגע בבעיה של איסוף מידע מקבצים או תיקיות. עלינו לבדוק אם התיקיות או הקבצים שאנחנו מחפשים קיימים במערכת. באמצעות השתמשות באיליקסיר, אנחנו יכולים לבדוק בקלות אם תיקיות קיימות, מה שיעזור לנו לטפל בבעיות ולשפר את ייעול האפליקציה שלנו.

## כיצד לבדוק אם תיקייה קיימת
הדרך הקלה ביותר לבדוק אם תיקייה קיימת היא בעזרת הפונקציה `File.dir?/1`. בשם התיקייה שאנחנו רוצים לבדוק אם היא קיימת, נשלח אותו כפרמטר לפונקציה והיא תחזיר `true` אם התיקייה קיימת ו-`false` אחרת.

```
Elixir
iex> File.dir?("my_directory")
true
```

ניתן גם לבדוק אם תיקייה קיימת באמצעות הפונקציה `File.exists?/1` ושם התיקייה כפרמטר כמו בדוגמה הבאה:

```
Elixir
iex> File.exists?("my_directory")
true
```

כדי לבדוק אם קיים קובץ בתיקייה מסוימת, ניתן להשתמש בפונקציה `File.ls/1` שתחזיר את כל הקבצים הקיימים בתיקייה כרשימה. בדיקה אם הרשימה ריקה או לא תגיד לנו אם התיקייה פנויה או לא.

```
Elixir
iex> File.ls("my_directory")
["file1.txt", "file2.txt"]
```

כבר אם ננסה לבצע את הפעולות לעיל על תיקייה שאינה קיימת, נקבל `false` כתוצאה. לדוגמה:

```
Elixir
iex> File.dir?("non_existing_directory")
false
```
