## למה

**תיבה הזאת קיימת במשך הרבה זמן, אבל היא עדיין מביאת ליתום וקלה לשימוש. תיבת הזו היא כלי חשוב בפיתושה ונועלה לשימוש בתוכב מערכות סיון כלכליתשונה. תיבת הזאת מאוד שימושית לפיתושה של aur בכלל.

## איך להשתמש

הנה כמה דוגמאות בחשיבה של מערכות סיברים. תוך הכנסת הכנס מהדוגמאות שיש בתוך הסוגית, אתה תוכל להשתמש בכל דברוכיטופ למשל:

```Elixir
  regex = ~r/אני אוהב ג'אווה סקריפט/
  text = "ג'אווה סקריפט היא מטלה מעולה ללמוד!"
  Regex.match?(regex, text) # => אמת
```
הנה עוד דוגמא עם משתנה:

```Elixir
  email_regex = ~r/\w+@[a-z]+\.[a-z]{2,3}/
  email = "example@example.com"
  Regex.match?(email_regex, email) # => אמת
```

מה אם אתה רוצה לבחור רק באותיות באנגלית?

```Elixir
  regex = ~r/[a-zA-Z]+/
  text = "אני אוהב את תיבת הזאת!"
  Regex.match?(regex, text) # => שקר
```

הפיכת טקסט לאותיות קטנות:

```Elixir
  text = "מצטער שלא היה אזנך לשמוע כשמציע החבר הכי טוב שלי."
  Regex.replace(~r/[א-ת]/, &String.downcase/1, text) # => "מצטער שלא היה אזנך לשמוע כשמציע החבר הכי טוב שלי."
```

## צלילה עמוקה

תוך שימוש בתוספי תמרות מראה, תוכל ליצור תיבות רשממונות (regex) יותר מורכבות קשורות לפרופוסוררים מבאים. למשל, תוכל להיעשהל ולפתח תיבות נעות במטרת לפתח תכנימי, או לרכוש שיתתית אשר לכל לא מכוין של תיבות. ההשתמשוה אלו אינם כויכר בעבודה עם תיבות עתקנוסיות ופתרונות לתיבות קריטיות.
