## למה

הקפיטליזציה של מחרוזת היא פעולה נפוצה בתכנות בכלל וב-Elixir בפרט. התרגיל המשוכלל של הכתיבה של מחרוזות באמצעות אותיות גדולות מאפשרת קריאה יותר נוחה של הקוד וגם עלות נמוכה יותר מהתחלופה של כל האותיות באופן ידני.

## כיצד לבצע

תחילה בואו ניצור נקודת התחלה עם מחרוזת ריקה:

```Elixir
string = ""
```

עכשיו נשתמש בפונקציה `String.capitalize/1` כדי להחזיר מחרוזת חדשה עם אות גדולה בתחילתה:

```Elixir
String.capitalize(string) # פלט: ""
```

במידה ונרצה להפוך את האות הראשונה של מחרוזת לגדולה ואת כל השאר לקטנות, נשתמש בפונקציה `String.capitalize/1` עם הפרמטר `:rest` מוגדר לערך `:lower`:

```Elixir
String.capitalize(string, :lower) # פלט: ""
```

אם נרצה להחזיר מחרוזת ריקה במקרה שהמחרוזת המקורית היא ריקה, נשתמש בפונקציה `String.capitalize/1` עם הפרמטר `:empty` מוגדר לערך `nil`:

```Elixir
String.capitalize(string, :empty) # פלט: nil
```

כדי לבדוק אם המחרוזת הוחזרה נכונה, נוכל להדפיס את הערך של המחרוזת החדשה:

```Elixir
IO.puts(String.capitalize(string)) # פלט: "STRING"
```
