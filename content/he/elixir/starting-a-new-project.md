## למה

במאמר זה נדבר על פרויקטים חדשים בשפת אליקסיר. נסביר למה כדאי להתעסק בפרויקט חדש וכיצד לבצע זאת.

## כיצד לעשות זאת

התחלת פרויקט חדש באליקסיר פשוטה וקלה. הרצת הפקודה `mix new [שם הפרויקט]` תיצור ספרייה חדשה עם קבצי הקוד הנחוצים. לדוגמא, לפתרון החידה הנתונה במאמר זה ניצור ספרייה בשם "puzzle" עם הפקודה הבאה:

```Elixir
mix new puzzle
```

בתוך הקבצים שנוצרו אנחנו נוכל לראות דוגמאות לכיצד להשתמש בכל פונקציה באליקסיר, למשל יצירת פונקציה חדשה וקריאתה, כמו בדוגמאות הבאות:

```Elixir
# יצירת פונקציה חדשה בשפת אליקסיר
def hello do
  IO.puts "Hello World!"
end

# קריאה לפונקציה
hello
```

כאשר רצים את הפקודה `elixir [שם הקובץ]` נקבל את הפלט הבא:

```
Hello World!
:ok
```

בחלק הזה ניתן לראות כיצד ליצור פתרונות באמצעות הכלים הנתונים לנו על-ידי אליקסיר.

## נבוא לעומק

התחלת פרויקט חדש היא חשובה כדי לעשות שינויים וכדי לשדרג את המיומנויות שלנו באליקסיר. אם נלמד כיצד ליצור פרויקטים חדשים ונתרגל כל פעם יותר ויותר, נוכל להפתיע את עצמנו עם הרבה יכולות חדשות ומעניינות במהלך עבודתנו.
