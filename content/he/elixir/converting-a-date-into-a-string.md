## למה

תוכנן כתיבה זו כדי להציג כיצד ניתן להמיר תאריך למחרוזת בשפת פונקציונלית כמו אליקסיר. יתרה מכך, יתוגדר גם למה יהיה נתון כדי להמיר את התאריך למחרוזת.

## כיצד

האלכסיר לא רק מאפשר לנו להתמודד עם טיפוסים פריימיטיביים כמו מחרוזת ומספר, אלא גם להמיר ביניהם בצורה מהירה ויעילה. כאן ניתן למצוא דוגמאות של כיצד ניתן להמיר תאריך למחרוזת בעזרת פונקציות של אליקסיר.

```elixir
defmodule DateToString do
  def convert(date) do
    IO.puts "תאריך לפני המרה: #{date}"
    date_string = to_string(date)
    IO.puts "תאריך אחרי המרה: #{date_string}"
  end
end

DateToString.convert({2019, 12, 21})
```

פלט:

```
תאריך לפני המרה: {2019, 12, 21}
תאריך אחרי המרה: "2019-12-21"
```

כאן ניתן לראות דוגמא נוספת עם מרחק זמן מסוים מתאריך נתון:

```elixir
defmodule DateToString do
  def convert(date) do
    IO.puts "תאריך לפני המרה: #{date}"
    today = Date.utc_today()
    days_ago = Date.to_days(today) - Date.to_days(date)
    date_string = "#{days_ago} ימים לפני"
    IO.puts "תאריך אחרי המרה: #{date_string}"
  end
end

DateToString.convert({2019, 12, 16})
```

פלט:

```
תאריך לפני המרה: {2019, 12, 16}
תאריך אחרי המרה: 5 ימים לפני
```

## חקירה עמוקה

בכדי להבין לעומק כיצד אליקסיר ממיר תאריך למחרוזת, נצפה על כמה פונקציות ומקרים שבהם המרה זו נעשתה.
