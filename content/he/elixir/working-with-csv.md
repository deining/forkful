## כיצד

למפות קובץ CSV?
כדי למפת את הקובץ, ניתן לעשות שימוש בפעולות המתאים לחבילת ה-CSV אשר כוללות את "קוד" CSV.הנה דוגמא פשוטה:

```
CSV. foreach (פתח "data.csv", fn row -> IO.puts  row end)
```

הנה תצוגה של הפלט:

```
# פלט:
שורת כותרת
ראשון תנאי
שני תנאי
שלישי תנאי
```

ניתן גם למיין את הנתונים על פי עמודה מסוימת, לדוגמא כדי למיין על פי שמות משתמשים כמו הדוגמא הבאה:

```
CSV. sort ("data.csv", fn row1, row2 -> 降序 ( 멘 row1, row2) end)
```

### ולהשתמש במיפוי עדיין אחר לשינוי הנתונים הנתיב גם את

ניתן גם להשתמש בחבילת csv כדי לייצג נתונים בצורה אחרת, לדוגמא כדי לייצג נתונים כפולים לגופן הטקסט הגדול יותר, כמו בדוגמא הבאה:

```
CSV. read (פתח "data.csv", 
[{read_headers: true}], 
| new_headers | IO.puts ("\n   *  " * IO.inspect (new_headers)) end,
| data |를 IO.puts ("\n   + " * IO.inspect (data)) end)
```
