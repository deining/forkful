## למה

כשמתחילים ללמוד את הפרוגרמינג ב-Elixir, נוכל לראות שקיים כמה דרכים לקבל נתונים מהמשתמש. אחת מהן היא קבלת ארגומנטים דרך שורת הפקודה. קריאת ארגומנטים משתמש מאפשרת לנו לספק לתוכנית שלנו קלט ייחודי בהתאם לצורך, וליצור תהליך שיפורט את הקוד כדי להשיג את היעדים הרצויים שלנו.

## איך לעשות זאת

כדי לקרוא ארגומנטים משתמש ב-Elixir, יש להשתמש בפונקציה `System.argv()`. ניתן למצוא את רשימת הארגומנטים שהועברו לתוכנית דרך הפקודה `mix run` או `elixir {file_name}`.

כאשר ניתן רשימת הארגומנטים, ניתן לעבוד איתם באופן דינמי באמצעות סיבוב עליהם עם `Enum.each` או על ידי משתמשות העברה במפענחים. לדוגמה:

```elixir
args = System.argv()

args |> Enum.each(fn arg -> IO.puts("הארגומנט הוא: #{arg}") end)

# יציג את הארגומנטים הבאים בצורה כזאת: "הארגומנט הוא: ארגומנט1"
```

ניתן להשתמש בזווית קודים משתמשת במשתנה המבוסס על פקודה לאחר פקודה כדי לקבל ארגומנטים משתמש נתונים ולתת פלט. לדוגמה:

```elixir
num1 = System.argv() |> Enum.at(0) |> String.to_integer()
num2 = System.argv() |> Enum.at(1) |> String.to_integer()
IO.puts("הסכום של #{num1} ושל #{num2} הוא: #{num1 + num2}")

# אם נפעיל את הפקודה mix run {file_name} 5 3, הפלט יהיה: "הסכום של 5 ושל 3 הוא: 8"
```
