מה זה ולמה?
בשפת התכנות ש לנו כיוון ליצור אלגוריתמים ולבזבז זמן רב על כתיבת קודים מסורבלים ומסורגלים, ריפוד בהם. אחד מהכלים ש מסייע לנו לחסוך בזמן ולייצר קוד יותר ברור ופשוט הוא ביטויים רגילים. כך שמבינים ,טורפים ועולים מחדש את ידע של הספק אינפורמציה מורמת מהנתונים. 

איך לעשות
מרצה בתוך קוד C: כדי למצוא קוד מבוסס א טכנולוגיהה הכולל ביטוי רגיל, שתןבדוק אם המילים שWe can use the ```C# ...``` באמצעות ביטויי רגיל מתאימים לתבני מסויים. למשל, כדי למצוא את כל המילים בתוך קוד הטקסט שמתחילות מ "המילה הילדים", ניתן להשתמש בפקודת regex שבתוך קוד זה: 

```C#
Regex regex = new Regex(@"\bילדים\w*\b");
MatchCollection matches = regex.Matches(text);

if (matches.Count > 0) 
{
    Console.WriteLine("נמצאו {0} התאמות!", matches.Count);

    foreach (Match match in matches)
    {
        Console.WriteLine(match.Value);
    }
}
else 
{
    Console.WriteLine("לא נמצאו התאמות!");
}

```
