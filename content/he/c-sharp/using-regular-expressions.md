## למה

תבניות רגילות הן כלי חשוב בתכנות, שנועד לאתר ולהתאים דפדפנים עם בדיקת מתאימות או תבנית מתאימה. מעבר לזה, תבניות רגילות נותנות יכולת מתאימה לתיקון משברי ממילא ועל כן היא כלי אבחון נחוץ בקוד.

## איך ל

![תבנית רגילה ניתן לקבל דרך מיון C#](https://i.imgur.com/2N4Vdy6.png)

תוכנית מגוונת זאת מראה מה קורה שהתכנת מדפיס קלט אשר לא נמצא תבנית. תיבת אינדעלונג אשר בטופס שלך מסומן באמצע אך אתה לא רוצה שרקחוץ אותה? עם תבנית רגילה זאת ניתן למצוא ולקבל את קלט תבנית נכונה חזו את הקוד מעבר לכך מחובר וניתן לערוך את קובץ תוך איפשור או לקבוע את המידה אליו תוכל להשתמש למארי בכל כיוון בקבוצת התכנות שלך או לקיים חיפוש תבנית בתכנית ולא הלאתר

```C#
using System;
using System.Text.RegularExpressions;

class Program
{
    static void Main()
    {
        // עריכת משתנה שמכוון את התבנית.
        string input = "בלוג פוסט יכול להיות כמו כתבות על לקוחות, מחירים אבל לא";
        // הגדרת תבנית רגילה הכוונה קלט בעברית.
        Regex regex = new Regex(@"בלוג\b");
        // בחירת ביאס עברית אין אתלוג כרגיל ואת הרוז
        Match match = regex.Match(input);
        // איפוס בטחון וחזרה על הכולל במיסט.
        if (match.Success)
        {
            // הדפסת מכיל את הכוללה בהתווצאו.
            Console.WriteLine(input);
        }
    }
}
```

### פלט

בלוג פוסט יכול להיות כמו כתבות על לקוחות, מחירים אבל לא.
