## למה
 ישנם כמה סיבות שעלולות להפוך את הקריאה על פרמטרי שורת הפקודה לשימושית. ראשית, כשאתה מפעיל תוכנית מתוך ממשק השורת הפקודה, אתה יכול לפענח נתונים יותר מהר ובקלות מאשר באמצעות ממשק משתמש גרפי. בנוסף, מספר רב של תוכניות וסקריפטים משתמשים בפרמטרי שורת הפקודה כדי לעבד מידע מתוך ייצוגי סמלי המערכת ולהריץ פונקציות אוטומטיות יותר.

## איך לעשות זאת
בקוד המצורף תוכלו למצוא שני דוגמאות של קריאה ועיבוד של פרמטרי שורת הפקודה בשפת C#:

```C#
using System;

namespace CommandLineArgs
{
    class Program
    {
        static void Main(string[] args)
        {
            // קריאת טקסט מהפרמטר השני
            string name = args[1];

            // קריאת מספר מתוך הפרמטר הראשון והמרתו ל- int
            int age = Convert.ToInt32(args[0]);

            // הדפסת הטקסט עם המידע שנקרא מהפרמטרים
            Console.WriteLine("שלום " + name + ", בן/בת " + age + "!");

            // המתנה להקשת מקש כדי שהחלון לא ייסגר מיד
            Console.ReadLine();
        }
    }
}
```

כאשר אתם מריצים את הקוד הזה דרך שורת הפקודה, תצטרכו להוסיף שני פרמטרים - גיל ושם. לאחר מכן, תוכלו לראות בפלט המסך של התוכנית את ההודעה "שלום [שם], בן/בת [גיל]!".

כעת, ישנם עוד המון אפשרויות ופעולות שניתן לבצע על פרמטרי שורת הפקודה בשפת C#. אתם יכולים לחפש בגוגל או לקרוא עוד על נושא זה במדריכים וספריות שונות.
