## מה ומדוע?

קריאת ארגומנטים מפקודת הקלט היא פעולה נפוצה בתכנות בשפת C# ובשפות תכנות נוספות. מדוע בעצם תכנתי מבצעים זאת? כאשר אתם מריצים תכנית מתוך מודעות השם של התוכנית, עשוי לשרת נייבים שונים, מחלק הסמכותיה: הנתונים שהוקלדו כקלט לתוכנית נקראו רצף ארגומנטים של התוכנית.

## איך ל
קריאת ארגומנטים נעשת פשוטה בקוד C#. הנה דוגמה של קוד פשוט שקורא את הארגומנט הראשון שהוזן על ידי המשתמש בשורת הפקודה ומדפיס אותו למסך:

```C#
class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine(args[0]);
    }
}
```

כאשר אתם מריצים את התוכנית ומזינים לדוגמה "Hello World" כארגומנט, התוכנית תדפיס את "Hello World" למסך.

## צלילה עמוקה
קריאת ארגומנטים מפקודת הקלט נעשית כבר ניחף. עם זאת, ביטוית הסטנדרט ANSI C ראשונית לא כוללת תמיכה בקריאת ארגומנטים מפקודת הקלט. שפות תכנות נוספות כמו Perl טרם נוצרו כדי לתמוך בקריאת ארגומנטים מפקודות הקלט, אך כיום זה נהפך לסטנדרט נפוץ ביותר בתעשיית התכנות.
