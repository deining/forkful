## מה ולמה?

כשתראה מילה מכותבת כולה באותיות גדולות בתוך משפט, זה כמעט בטוח שהמנפק שלא להתעסק עם זה בתכנות. אבל למה זה פותר למנפק לכתוב את המאפיינים הללו "תת צגללמתי"? בפועל, כתיבה גדולה תאפשר מאמר להבין את המילה הזו כסימן להציג תוצאה שונה בעוד דובגודים (שהם תחליפים). בגלל זה זה יהי המנפק And also make code more readable by enhancing the visibility of certain words and phrases.

## איך לעשות את זה:
בקוד הבא, אנחנו משתמשים בשיטת `ToUpper()` כדי לשנות את המשפט לאותיות גדולות במילה הראשונה:

```C#
string capital = "hello world";
Console.WriteLine(capital.ToUpper());

// Output: HELLO world
```
