## למה

הזדמנות גדולה

ניתן להשתמש בממשק תוכנה לתיחום והחלפת טקסט, או בבנאי תוכנית שמהווה את העזרה הטובה ביותר למרבית המתמשים. אין סיבה לבוא בקשר עם לפורט במצגת של 'איך לעשות זאת' בפרק 1.

## איך

כדי להחליף טקסט בתוך קובץ מבצעים, יש להשתמש בפעולת חיפוש והחלפה. לדוגמה, אם נרצה להחליף את המילה "קטן" במילה "גדול" בתוך קובץ, נוכל להשתמש בפקודות הבאות תוך שימוש בחיפוש והחלפה בתוך לולאה:

```C#
foreach (string line in file)
{
    if (line.Contains("קטן"))
    {
        string newLine = line.Replace("קטן", "גדול");
        // טיפול בשורות מחולקות
    }
    // שמירת נתונים לקובץ עם החלפה
    StreamWriter writer = new StreamWriter("newFile.txt");
    writer.WriteLine(newLine);
    writer.Close();
}
```

בתוך הלולאה, אנו בודקים כל שורה בקובץ ובמקרה שנמצאת בה מילה "קטן", אנו מחליפים אותה למילה "גדול". בסופו של דבר, אנו שומרים את הנתונים לקובץ חדש עם ההחלפות החדשות.
