## למה 

תארו לך שאתם מפתחים אפליקציית תאריכים לימים חגים או כראש השנה של העברים. גם לכם לא נותרו מעט ימים לעבוד ואתם צריכים לבדוק אם תאריך מסוים הוא חג מיוחד או לא. זו הסיבה למה אנחנו צריכים לעשות השוואה שבין תאריכים. במאמר זה נילמד כיצד לכתוב קוד ב-C# לכיווני ביצוע השוואה שבין תאריכים, וכיצד להשתמש בזה בכדי לקבל תוצאות מבוקשות.

## איך לעשות זאת

הנה קוד דגם ב־C# לבדיקת שוויון בין תאריכים:

```
DateTime firstDate = new DateTime(2021, 09, 18);
DateTime secondDate = new DateTime(2022, 09, 06);

if (firstDate == secondDate)
{
    Console.WriteLine("התאריכים שווים.");
}
else
{
    Console.WriteLine("התאריכים אינם שווים.");
}
```
### תוצאה:

התאריכים אינם שווים.

ניתן גם להשתמש בפעולות אחרות כדי לבצע השוואה בין תאריכים, כגון `Equals`, `Compare`, וכו'. בהמשך המאמר נלמד כיצד לבחון את התאריכים לפי חודשים או ימים, וכיצד להשתמש בכלים אלה על מנת ליצור פעולות מתקדמות בין תאריכים.
