## למה

אחת המשימות הנפוצות בפיתוח תוכנה היא להשוות בין תאריכים. זה יכול להיות לצורך תאריך הרצפה באתר או באפליקציה, לבדיקת מועדי תפוגה של מוצרים או פשוט כדי להשוות בין שתי אירועים. במאמר זה נלמד כיצד לבצע את המשימה הזו בקלות בעזרת שפת התכנות C#.

## כיצד לעשות זאת

ראשית נכיר את המחלקה "DateTime" שמחזיקה מידע על תאריך ושעה. נדרוס את התאריך הנוכחי על ידי השמה שלו למשתנה מסוג DateTime באמצעות הפונקציה "Now":

```C#
DateTime currentDate = DateTime.Now;
```

למשל, אם נרצה לדעת מהו התאריך והשעה בדיוק של הרגע הנוכחי, נוכל להדפיס את הערך של המשתנה "currentDate" כך:

```C#
Console.WriteLine(currentDate);
```

התוצאה תהיה משהו כזה:

```
11/15/2021 8:25:06 AM
```

עכשיו שאנו מכירים את השימוש במחלקת DateTime נוכל להשוות בין שני תאריכים בפשטות. ננסה לבדוק האם תאריך מסוים הינו בעבר או בעתיד. נניח שאנו מחפשים לדעת האם התאריך 12/31/2021 כבר עבר או שעוד לא הגיע. נוכל לכתוב קוד כזה:

```C#
DateTime compareDate = new DateTime(2021, 12, 31); // תאריך להשוואה
if (compareDate < currentDate) // אם התאריך שהשווה כבר עבר
{
    Console.WriteLine("התאריך שהשווה עבר");
}
else // אחרת הוא עוד לא הגיע
{
    Console.WriteLine("התאריך שהשווה עדיין לא הגיע");
}
```

כאן נהיה לב לשימוש בסימן קטן מיוחד ("<") להשוואת התאריכים. התוצאה שנדפיס תהיה:

```
התאריך שהשווה עדיין לא הגיע
```
