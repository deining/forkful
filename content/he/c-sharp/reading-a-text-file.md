##למה

מחקרים רבים מראים שקריאה תקצירים וקבצי טקסט יכולה להעשיר את מי שקוראם בידע ומובנות חדשות. בנוסף, קבצי טקסט הם חלק בלתי נפרד מהתכנות וחלק בסיסי ביותר בכל שפת תכנות. כך שיש חשיבות רבה בלימוד איך לקרוא קבצי טקסט בפלטפורמת C#.

##איך לעשות זאת
הקוד הבא מדגים איך לקרוא קובץ טקסט בשפה C# ואיך להדפיס את התוכן שלו למסך:

```C#
string path = @"C:\TextFile.txt"; //הגדרת נתיב לקובץ
string[] lines = File.ReadAllLines(path); //קריאת קובץ טקסט והכנסת התוכן למערך של מחרוזות
foreach (string line in lines) //לולאה שמדפיסה כל שורה מהמערך
{
    Console.WriteLine(line);
}
```

פלט התכנית יהיה:

```
שורה ראשונה בקובץ טקסט
שורה שנייה
זו שורה שלישית
```

כדי לשנות את הקובץ המקורי, ניתן להשתמש במתודה `File.WriteAllText()` ולהעביר אליה את הנתיב והתוכן החדש שברצוננו להכניס לקובץ.

##עומק הנחיקה
קריאת קבצי טקסט היא חלק חשוב מאוד בתכנות בשפת C#. למעשה, כמעט כל תכנית משתמשת בתוכן טקסט כמציאותי וכדי לעשות זאת, צריך לדעת איך לקרוא ולעבוד עם קבצי טקסט. כמו כן, ישנם דברים נוספים שניתן לעשות עם קובץ טקסט, כגון כתיבה לקובץ, עדכון תוכן ובקרת הצגתו למשתמש.
