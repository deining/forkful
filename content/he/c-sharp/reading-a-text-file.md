## למה

אם אתם מתעניינים בתכנות בשפת C# ורוצים ללמוד איך לקרוא קובץ טקסט, כנראה שאתם יודעים שהכישור הזה הוא אחד היסודיים ביותר. אלא שלא תדאגו, גם אם אתם רק מתחילים או לא בטוחים בנוגע לנושא, המדריך הזה יעזור לכם להסתדר.

## איך לעשות זאת

הכותרת של המדריך הזה כבר אומרת הכל - אנו אנונימים להעברת מידע. כדי לקרוא קובץ טקסט בשפת C#, ניתן להשתמש בפקודת `File.ReadAllLines()`. תחילה ניצור תיקייה חדשה ונכין קובץ טקסט ריק בשם "myfile.txt". לאחר מכן, נשתמש בשורת הקוד הבאה כדי לקרוא את הקובץ:

```C#
string[] lines = File.ReadAllLines("myfile.txt");
```

נדפיס את תוכן הקובץ למסך על ידי צורת הלולאה `foreach`:

```C#
foreach (string line in lines)
{
    Console.WriteLine(line);
}
```

בכל שורה של הלולאה, נדפיס את התוכן של השורה הנוכחית. למשל, אם תיכתבו מלל בתוך הקובץ כמו "שלום עולם", תקבלו את הפלט הבא:

```bash
שלום עולם
```

כמו כן, ניתן גם להשתמש בקוד HTML כדי להציג את הקובץ לפי סדר מתאים ולכיוון שנרצה.

## כיון מעמוק מזה

אם אתם מעוניינים לעמוד בתוכן מעלה ולהתחיל להשתמש בהם בצורה אפיקטיבית יותר, כדאי לכם לקרוא עוד על ספריות מתאימות שניתן להשתמש בהן בשילוב עם הפקודה `ReadAllLines()`. כמו כן, גם ניתן לממש פונקציות נוספות כדי לטפל בתהליך הקריאה של הקובץ והצגתו.
