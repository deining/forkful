## למה

למה אדם ירצה להשתמש בפעולת מחיקת תווים המתאימים לתבנית? ייתכן שמדובר בשגיאה במסמך או תווים שאינם רלוונטיים ורוצים להוריד אותם כדי לחסוך מקום ולהפחית את פוטנציאל התקלות.

## איך לעשות זאת

הנה דוגמא לקוד שבודק את התווים במחרוזת ומוחק את כל התווים המתאימים לתבנית שנכתבה על ידי המשתמש:

```c#
string str = "Hello, world!";
string pattern = "[aeiou]"; // תבנית למחיקת תווים שלא הם תווי צליל
string result = Regex.Replace(str, pattern, ""); // השימוש בפונקציה Regex.Replace כדי למחוק תווים שמתאימים לתבנית
Console.WriteLine(result); // פלט: Hll, wrld!
```

ניתן לראות שהתווים a, e, o ו-i נמחקו מהמחרוזת המקורית. את התבנית ניתן לשנות כמספר של מצרפים או תווים שמתאימים בכל זמן נתון.

## Deep Dive

כדי לבצע מחיקה כזו, עלינו להשתמש בפונקציה Regex.Replace שמאפשרת לנו לחפש תבניות במחרוזת ולהחליף אותן בתוכן שלנו. בכל עבור של לולאת הלולאה, נבדוק את התווים אם הם מתאימים לתבנית ואם כן, נחליפם ב-"", מתאים לתווים שלא לפחות אחד עומד בתבנית. ניתן לשנות את סדר התווים לפי הצורך הבנאלי. כדי למחוק תווים כפי שבהם, יש לשים לב שהתווים אינם משנים כלום אם הם לא מתאימים לתבנית ולכן כדאי להשתמש בתבנית שמתאים לתווים ספציפיים.
