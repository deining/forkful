# למה

טיפול דייט מתחשב בזמן העתיד או העבר הוא כלי חשוב עבור מתכנתי C# בטכנולוגיות נתונים ותקשורת. זה מאפשר למשתמשים לחשב חזרה או קדימה לתאריך מסוים ולהשתמש בנתונים במגוון דרכים שונות. בכתבה זו נדבר על כיצד לתכנן וליישם טיפול דייט בשפת התכנות C#.

# איך לעשות

הנה כמה דוגמאות של קוד C# עבור טיפול דייט.

```C#
// חישוב תאריך בעתיד - 14 ימים מהיום הנוכחי
DateTime futureDate = DateTime.Today.AddDays(14); 
Console.WriteLine(futureDate); // Output: 08/08/2021 00:00:00

// חישוב תאריך בעבר - 3 שנים מהיום הנוכחי
DateTime pastDate = DateTime.Today.AddYears(-3);
Console.WriteLine(pastDate); // Output: 08/08/2018 00:00:00
```

יכולת לחשב תאריך נתון בעתיד או בעבר היא מאוד חשובה ביישומים כמו תוכניות הת עבודה, וכן בכל מקרה בו נדרש לנתקאם טיפול דייט תכוניות, טרבלרי ומגוון אחר של נתונים.

# נכס

כאשר מחשבים תאריך בעתיד או בעבר, יתכן שיש צורך לדעת את היום, החודש, והשנה בנפרד. בשפת C# ניתן לעשות זאת באמצעות שימוש בתכונות DateTime השונות. לדוגמה:

```C#
// חישוב תאריך עתידי והדפסת היום בנפרד
DateTime futureDate = DateTime.Today.AddDays(14);
Console.WriteLine(futureDate.Day); // Output: 22
Console.WriteLine(futureDate.Month); // Output: 08
Console.WriteLine(futureDate.Year); // Output: 2021
```

ניתן לחשב גם את העתיד או העבר בהתאם לתאריך קיים על ידי שימוש בפעולות חשבון פשוטות, כמו חיבור וחיסור ימים, חודשים או שנים.
