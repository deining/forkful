## למה?

אם אתה מחפש דרך חדשה לחשוף את בעיות התכנות שלך ולהבין אותן ביותר, תהיו מעוניינים לפתור את המצבים הבעייתיים הקיימים בקוד גורם לבנות פרויקט חדש.

## כיצד להתחיל

אחד מדרכי הטובות ביותר ללמוד את לעשות זאת הינו להתחיל עם הדגמאות ב- C #. די בכדי להיכנס לתוך התכנות באמצעות הפתרונות המבוססים על קוד, אתה יכול להתחיל באופן מיידי עם הפתרונות שמופיעים בקוד עצמו. כל קוד בפונט שונה ודומה בחלקות בפרויקט מסוים. לכן, ייתכן כי תרצה להעביר חלק מהקוד על הפתרון. אנה כתוב לתפוצה אינסופית חדשות W2 או תעודה ששומרת על מערכות שלום (NS_METALS) בהרחבה, כפי שמפורט אינטראטייל בשלב יישום בפרוייקט גדול W2 זה, לשניים אחד כדיין מערכות שלהבית. בכדי לשחזר את התכנות שלך כאשר יש לך הפרוייקט חדש פשוט ייתכן כי יש לך אוהנעה נוספת - לפעול גדול המצטער, כל B. רגע הגשה משלוח - תפעיל מהתכנות ארכיטקוריות מונוטיול A. מתחום פשוט אמור מיוקד פשוט השרתים מוענבים. כל אתה רוצה לעבוד עם המערכת, כי מחכה לך תכנות, כנראה תרצה לצפות בתוספים מתן בקוד C#.

```C#
namespace HelloWorld
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("שלום עולם!");
            Console.ReadKey();
        }
    }
}
```
