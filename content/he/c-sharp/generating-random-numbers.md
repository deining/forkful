# למה

אנו כל כך רגילים לראות את המספרים האקראיים בחיינו, כדי לבחון משקיעים בבורסה, לאתגר חברים במשחקים או לזרוק אריח בבית חולים כדי להחליף את הפלפל. אפילו כאשר אנו משתמשים באופן מחשבותי במציאת מקומה של חבורת חברים במסיבת יום הולדת, אנו בעצם משתמשים במספרים אקראיים. ואם אנחנו מדברים על תחום התכנות, תמיד נתקל במצבים שבהם אנחנו זקוקים ליצירת מספרים אקראיים. אז למה אנחנו כל כך מעוניינים לייצר מספרים אקראיים? בפוסט הזה, אנו נחקר את הסיבות ונלמד כיצד ליצור מספרים אקראיים בשפת תכנות C#.

## איך לעשות זאת
כדי לייצר מספרים אקראיים בשפת תכנות C#, אנו צריכים להשתמש בפעולת Random. פעולת Random מאפשרת לנו לייצר מספרים אקראיים תוך כדי נתן התחלתי קבוע.

בדוגמאות הבאות תוכלו לראות כיצד להשתמש בפעולת Random כדי לייצר מספרים אקראיים ולהדפיס אותם למסך:

```C#
// יצירת אובייקט Random חדש
Random random = new Random();

// הדפסת מספר אקראי בין 0 ל־100
Console.WriteLine(random.Next(0, 100));

// הדפסת מספר אקראי מורכב בין 0 ל־100
Console.WriteLine(random.NextDouble() * 100);

// יצירת מערך של 10 מספרים אקראיים בין 0 ל־100
int[] numbers = new int[10];

// מעבר על המערך והדפסת כל מספר אקראי
for (int i = 0; i < numbers.Length; i++)
{
    numbers[i] = random.Next(0, 100);
    Console.Write(numbers[i] + " ");
}
```
