## למה

ישנם מספר סיבות למה אנשים מעורבים ביצירת מספרים אקראיים בתוכנות שלהם. כלי אלה נועדו לספק ערך שאינו תלוי בכניסה או תנאים קבועים, כך שניתן להשתמש בהם ליצירת מגוון של מידע אקראי כגון אותיות, מספרים, או תווים.

## איך לעשות זאת

כדי ליצור מספרים אקראיים בשפת C#, ניתן להשתמש בפקודת `Random` המספקת פונקציונליות ליצירת מספרים אקראיים. להלן נתונים כמה דוגמאות קוד ותוצאות.

```C#
// ניצור משתנה מסוג Random
Random rand = new Random();

// נציג את מספרים אקראיים בין 1 ל-100
int num1 = rand.Next(1,101);
Console.WriteLine("מספר אקראי ראשון: {0}", num1);
int num2 = rand.Next(1,101);
Console.WriteLine("מספר אקראי שני: {0}", num2);

// נציג אותיות אקראיות מאוסף של אלפבית
char letter1 = (char) rand.Next('a', 'z' + 1);
Console.WriteLine("אות אקראית ראשונה: {0}", letter1);
char letter2 = (char) rand.Next('a', 'z' + 1);
Console.WriteLine("אות אקראית שנייה: {0}", letter2);
```

תוצאה:

```
מספר אקראי ראשון: 63
מספר אקראי שני: 17
אות אקראית ראשונה: h
אות אקראית שנייה: n
```
