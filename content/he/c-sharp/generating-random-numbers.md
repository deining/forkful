## מדוע
אם אתם מתעניינים בתחום התכנות, סיכון גבוה אחד שאתם תגיעו אליו הוא הצורך ליצור מספרים אקראיים. למעשה, החלום של כל מתכנת הוא לייצר סדרות מספרים כאלו עם דיוק מלא. ניתן להשתמש במספרים אקראיים בהמון מקרים, כגון ייצוג אושרך בכל אינסטלציה של לוטרי.

## איך לעשות
כאשר מדברים על יצירת מספרים אקראיים בסביבת תכנות C#, ישנן כמהשיטות לעצם באמצעותן ניתן ליצור מספרים אקראיים. אחת מהן היא על ידי יצירת אובייקט מהטווח המגוון Random. לאחר מכן, ניתן להשתמש בשיטות כגון Next() ו-NextDouble() כדי ליצור מספרים אקראיים עם סוג תצוגה שונה. כדי לראות דוגמא של קוד ופלט מהודק של יצירת מספרים אקראיים ב-C#, יש למשוך הפעלת הכוכביות מצד שמאל לפני הפרק לייצוג דוגמת קוד הזה. הרצה דוגמא כדי לעורר זהירותך עוסקת במקרה שאובייקט עולה מאוד 100, ובמקרה שאבא יוצא מבצע.

```C#
using System;

class MainClass {
  public static void Main (string[] args) {

    Random random = new Random();
    Console.WriteLine($"Random number between 1 and 10: {random.Next(1, 11)}");
    Console.WriteLine($"Random decimal number between 0 and 1: {random.NextDouble()}");
    
  }
}
```

פלט שמתקבל:

```
Random number between 1 and 10: 8
Random decimal number between 0 and 1: 0.9872261583834509
```
