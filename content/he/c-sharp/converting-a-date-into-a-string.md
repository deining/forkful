## למה 

למה לכתוב תאריך כמחרוזת? כמו בכל שפת תכנות, כתיבת תאריכים כמחרוזות היא כלי נהדר להצגת מידע בצורה קריאה ופשוטה. המחרוזת מורכבת ממספרים ופסיקים, ולכן תוכלו לקרוא ולהתאים את התאריך לפי הצורך.

## איך לעשות זאת

 ```C#
// קוד להמרת תאריך למחרוזת בשפת C# 
DateTime date = DateTime.Now;  // לדוגמה, נתון תאריך נוכחי
string dateAsString = date.ToString(); // המרת התאריך למחרוזת
Console.WriteLine(dateAsString); // הדפסת התאריך הממורכז למחרוזת בתוך הקונסול
// פלט: 28/07/2021
 ```

ומה קורה אם אנחנו רוצים לעצב את המחרוזת של התאריך כך שהיום יוצג תחת השנה בשפה העברית? כדי לעשות זאת, נשתמש בפורמטינג ונגדיר את התבנית המבוקשת:

 ```C#
// קוד להמרת תאריך למחרוזת בשפת C# עם התאריך בפורמט בעברית
DateTime date = DateTime.Now; 
string dateAsString = date.ToString("dd/MM/yyyy"); // תבנית ההצגה של התאריך, בפורמט DD/MM/YY
Console.WriteLine(dateAsString); // פלט: 28/07/2021
 ```

ניתן גם להתאים את התבנית כדי להתאים לתצוגות שונות של התאריך, כגון תאריך עם שנה מלאה או שעה נוספת. נוסיף פנימה את התבנית הרצויה ונקבל את הפלט המתאים בהתאם.

 ## חקירה מעמיקה

כדי להמיר תאריך למחרוזת, כל שהצריך לעשות זה לבקש מהמחשב להמיר את התאריך לפורמט מסוים. בשפת C# ישנם כמה פורמטים נפוצים שניתן להשתמש בהם עבור התאריך, ולכן יש לבחור את התבנית המדוייקת עבור הצורך המיוחד שלנו.
