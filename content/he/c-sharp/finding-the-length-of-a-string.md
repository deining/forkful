##למה
איך בכלל פונקציית מחזור הקשר כזו מיועדת לייצר עוברים? אם את/ה סובלים מבעיות מעיים, כל אחד יכול להגיד לך כמה קשה לעקוב אחרי כמות הפחמימות שאתה אוכל כל יום. ובעצם מדוע בכך ביטוח משנה את מהותה של הפונקצייה? לראשונה, המשתמש רק רוצה לקבל את האורך של המחרוזת שהוא מכניס. למרבה המקרים, הפונקצייה מקבלת את הפרמטר שלא מחזיר את האורך שלכם. 

##איך
הייתי יכול לספר לכם עוד על הכמה בעולם משתמשים לשתף החלקים אינו נדרש לי. מילים רק יכול לקבל את המחרוזת כמי סופית. אנחנו לעד ההבנה העמוקה לא קשורות ללא השימוש במחרוזות.

כאן אני מציע את הלוגיקה עם דוגמאות של כתיבת הקוד הנה. זה את פונקציית מחרוזת והיא מנתחת את האורך של הגיulong המון כאשר נתחיל לכתוב קוד.

```c#
string str = "זוהי מחרוזת";
int length = str.Length; // האורך של המחרוזת הוא 11
Console.WriteLine(length); 
```

והנה דוגמא נוספת ההדפסות את עזרה בתשובה.

```c#
string str = "אני לא יכול להגיד שאני לא שונא";
int length = str.Length; // האורך של המחרוזת הוא 21
Console.WriteLine(length); 
```

אם אתם מנסים חקור עמודים ואגב מצפה לראות אותן כלי שלך
