## למה
משהו פשוט מאוד של C # הינו למצוא את אורך המחרוזת. כמעט כל יישום שנכתב בשפת תכנות צריך להתיימר לעסוק במחרוזות בזמן מסוים. לכן, אין ספק שהיכולת למצוא את אורך המחרוזת הינה כלי חיוני בתחום התכנות.

## איך לעשות זאת
ישנן מספר דרכים למצוא את אורך המחרוזת בשפת C#. אחת הדרכים הפשוטות הינה להשתמש בפונקציה "Length" שקיימת כבר במחלקת String. נעביר לפונקציה ערך של מחרוזת והיא תחזיר את אורך המחרוזת באמצעות מספר שלם.

```C#
string str = "זוהי מחרוזת לדוגמה";
Console.WriteLine(str.Length); // output: 15
```

עוד דרך למצוא את אורך המחרוזת היא להשתמש במתודה "Count" הקיימת עבור מחרוזת שמקבלת כפרמטר מחרוזת ומחזירה את מספר התווים שמתאימים למחרוזת זו.

```C#
string str = "זוהי מחרוזת לדוגמה";
Console.WriteLine(str.Count()); // output: 15
```

## כיוון עמוק
במרבית המקרים, התכונה "Length" חייבת לחשוב על התמחורה של המחרוזת כדי למצוא את אורךה. למשל, כאשר אנו עובדים עם טקסט בשפת C#, ייכולנו להשתמש בעזרתו בכמה חישובים - כמו להשמיע את כל אותיות הגירסה הפשוטות יותר. בנוסף, כשנתבול בחישוב תכונה זו, יש לכם להיות מוכווים לכך שבחינה אם נותנים למחרוזת לבד בכתובת המערכת החיצונית.
