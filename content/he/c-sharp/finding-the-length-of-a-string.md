## למה 

וודאי שראית את זה לפני - מצב שאתה צריך לדעת את אורך המחרוזת. אם אתה כותב תוכניות ב- C# (הגירסה הנוכחית), ייתכן ותצטרך להשתמש בפונקציה לאיתור אורך המחרוזת. אבל למה לעסוק בזה בכלל? באמת, באמצעות הפונקציה הזו אתה יכול לעבוד בצורה יעילה יותר עם מחרוזות ולשפר את הביצועים של התוכנית שלך.

## איך לעשות את זה

בכדי למצוא את אורך המחרוזת ב- C# באמצעות הפונקציה len, תצטרך קוד פשוט כמו זה:

```c#
string str = "שלום עולם";
int length = str.Length;
Console.WriteLine(length);
```

והפלט יהיה:

```sh
9
```

בכמה שורות אפשר לראות את כמה פשוט ואפקטיבי שזה. נסה לשחזר את הקוד ולשנות את המחרוזת למשהו אחר ותראה כי האורך משתנה בהתאם.

## צלילה עמוקה

כעת שאתה כבר יודע כיצד להשתמש בפונקציה len כדי למצוא את אורך המחרוזת, נצלק אל עמקי הנושא. ראשית, כדי להשתמש בפונקציה הזו, עליך להכיר את הטיפוס string בצורה הנכונה. אם אתה לא תמיד מצליח למצוא את אורך המחרוזת כמו שאתה רוצה, כנראה שיש בעיה עם הטיפוס string שלך. וטיפ מהיר - חשוב להכיר את התוכניות שבהן יש שימוש רב בפונקציה למציאת אורך המחרוזת כדי להציע למשתמשים נסיגה טובה יותר של התוכנית.
