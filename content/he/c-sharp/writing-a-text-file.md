## למה

כתיבת קבצי טקסט היא חלק חשוב בתהליך התכנות. זה מאפשר לנו לשמור ולטעון מידע בתוך קבצים נפרדים, מה שהופך אותו לקל יותר לארגן ולנווט בין הקוד שלנו. כמו כן, קבצי טקסט יכולים לשמש גם כדרך נוחה לשתף מידע עם אנשים אחרים בצוות.

## כיצד לכתוב קובץ טקסט ב-C#

הנה דוגמה לקוד שיכול להיות מועיל כאשר אתם רוצים לכתוב קובץ טקסט בשפת תכנות C#:

```
using System.IO;
//פתיחת קובץ טקסט חדש לכתיבה
StreamWriter file = new StreamWriter("testFile.txt");
//כתיבת מחרוזת לקובץ
file.WriteLine("זוהי מחרוזת היכן שהקובץ נמצא");
//סגירת הקובץ
file.Close();
```

כאשר הקוד הזה מוצג, הוא יוצר קובץ טקסט חדש בשם "testFile.txt" ומכניס את המחרוזת "זוהי מחרוזת היכן שהקובץ נמצא" לתוכו. כמובן שאתם יכולים לשנות את השם, המחרוזת והתוכן של הקובץ לפי הצורך שלכם.

כדי לקרוא קבצי טקסט, ניתן להשתמש בפקודה "File.ReadAllLines()" כדי לאחזר את כל השורות שבקובץ כמערך של מחרוזות.
