## למה

החיסור של פתיחת מחרוזת וסוף שלה יכול להיות פעולה שימושית בשפת תכנות סי שארפ, וזה נקרא חילוץ מחרוזת. יתכן שתרצו לחלץ חלק מתוך מחרוזת ארוכה יותר או לבדוק אם מחרוזת מסוימת מופיעה בתוך מחרוזת אחרת. בכתב הבלוג הזה, נתאר כיצד לחלץ חלקים מתוך מחרוזות בשפת תכנות סי שארפ ונספק דוגמאות לקוד ותוצאות.

## איך לעשות זאת

בכדי לחלץ חלק מתוך מחרוזת בסי שארפ, יש להשתמש בפקודת Substring (). הפקודה זו מקבלת שני מקרוים, המציינים את תחילת וסוף החיסור הרצוי. לדוגמה, אם נרצה לחלץ את המילה "עולם" מתוך המחרוזת "שלום עולם", נכתוב את הפקודה הבאה:

```C#
string word = "שלום עולם";
string substring = word.Substring(6, 4);
Console.WriteLine(substring);
```

בתוכנית זו, נגדיר משתנה בשם word ונשמור בתוכו את המחרוזת "שלום עולם". לאחר מכן, נשתמש בפקודת Substring () כדי לחלץ את המילה "עולם", ונשמור את החיסור במשתנה חדש בשם substring. סך הכול נדפיס את החיסור באמצעות פקודת Console.WriteLine ().

התוצאה של התוכנית הזו תהיה:

```
עולם
```

ניתן גם להשתמש בפקודה Substring () כדי לבדוק אם מחרוזת מסוימת מופיעה בתוך מחרוזת אחרת. נסתכל על הדוגמה הבאה:

```C#
string sentence = "אהבה ושלום בעולם";
string word = sentence.Substring(0, 4);
if (word == "אהבה")
{
    Console.WriteLine("המילה אהבה מופיעה בתוך המשפט");
}
else
{
    Console.WriteLine("המילה אהבה לא מופיעה בתוך המשפט");
}
```
