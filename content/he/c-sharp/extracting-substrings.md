## למה
מחרוזות הם חלק חשוב בתכנות בשפת סי-שארפ, וככל שהקוד שלנו מתקדם יותר, כך נדרשנו על פני לטפל במחרוזות רבות יותר ומורכבות יותר. אחת הפעולות החשובות ביותר שיכולים להיות לקחו על מחרוזות היא חילוץ חלקים שונים שלהן, או במונחים מסוימים מחרוזות משניות. זה נחוץ כאשר אנו רוצים לעבוד עם חלק מסוים של מחרוזת כמו מזהה מיוחד או תאריך, ולא רק כמו מחרוזת אחת שלמה.

## איך להתקדם
ניתן לחלץ מחרוזות בשימוש בפונקציות פנימיות כמו `SubString()`, `Split()` ו- `Remove()`. בפוסט הזה נשתמש בדוגמאות קוד כדי להראות כיצד להשתמש בפונקציות אלה כדי לחלץ מחרוזות לפי הצורך. נוסיף גם דוגמאות לפלט שיכפלו את התוצאות.

```C#
// דוגמא 1: חילוץ חלק מסוים של מחרוזת על ידי שימוש ב-SubString()
string fullName = "Sarah Smith";
string firstName = fullName.SubString(0, 5);

// פלט: "Sarah"


// דוגמא 2: חילוץ מילים נפרדות ממחרוזת על ידי שימוש ב-Split()
string sentence = "Hello, my name is John.";
string[] words = sentence.Split(' ');
foreach(string word in words)
{
    Console.WriteLine(word);
}

// פלט:
// "Hello,"
// "my"
// "name"
// "is"
// "John."


// דוגמא 3: מחיקת חלק משלם של מחרוזת על ידי שימוש ב-Remove()
string numbers = "123-456-789";
string newNumbers = numbers.Remove(3); // נמחק התוים הראשונים עד התו ה-3
// פלט: "456-789"
```
