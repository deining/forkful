רשימת-ארדוינו: בדיקת קיום תיקייה

## מה ולמה?
בדיקת קיום תיקייה היא פעולה שמאפשרת למתכנתים לבדוק אם תיקייה מסוימת קיימת במחשב. הפעולה משמשת כדי לבדוק אם קובץ או תיקייה נמצאים במסינג. למרבה המזל, ישנן כמה דרכים לבדוק את קיום תיקייה בארדוינו, וזה מאפשר למתכנתים להרתיח עם הנתונים וליישם את האלגוריתם הטוב ביותר עבורם.

## איך לבדוק את קיום התיקייה:

לבדוק אם תיקייה קיימת בארדוינו ניתן לכתוב את הפקודה הבאה:

```Arduino
if (SD.exists("myFolder")) {
  Serial.println("The folder exists.");
}
```

עם הפקודה המתאימה, ניתן לבדוק אם תיקייה בשם מסוים קיימת על כרטיס ה-SD של הארדוינו. אם התיקייה קיימת, יתווספו בהודעה המתאימה ל-Serial Monitor.

## לכיוון העומק:

בעבר, בדיקת קיום תיקייה הייתה תהליך מורכב יותר עם שימוש בפקודות נוספות ובדיקות מרובות כדי לוודא את קיום התיקייה. בזכות הארדוינו, העניין הפך יותר פשוט עם הפקודה היעילה שנתונת.

אם לכל אלטרנטיבה לבדיקת קיום תיקייה, ניתן להשתמש בפקודת [```SD.exists()``` ](https://www.arduino.cc/en/Reference/SDExists) המגיעה כחלק מספריית ה-SD המובנית של הארדוינו. זה יכול להפחית את מספר העשויים לבדוק מספר פקודות, ויכול לחסוך זמן ומשאבים.

כדי להגדיר את פקודת בדיקת קיום התיקייה, הוסף את התכונה הבאה לתוך קבצי הקוד שלך:

```Arduino
#include <SD.h>
```

## ראה גם:

- [SD library ](https://www.arduino.cc/en/Reference/SD) - מאגר מידע נרחב על ספריית ה-SD של הארדוינו
- [File System documentation](https://www.arduino.cc/en/Reference/FileSystem) - מידע נוסף על הפעולות השונות שניתן לבצע עם ספריית ה-SD של הארדוינו
