## למה
  לבדיקה אם תיקייה קיימת הרשתה אותנו שימושי בנסיבות מסוימות, למשל כאשר אנחנו צריכים לבדוק אם קובץ קיים או לבקר בנתונים נמצאים בתיקייה מסוימת.

## איך לבדוק אם תיקייה קיימת
  ניתן לבדוק אם תיקייה קיימת באמצעות פונקציית `exists()` שמייצגת את ביצוע הפעולה של קיום. כאשר הפונקציה מקבלת כפרמטר את הנתיב של התיקייה שאנחנו רוצים לבדוק, תחזיר `true` אם התיקייה קיימת ו `false` אם התיקייה לא קיימת. לדוגמה:

  ```Arduino
  if (exists("/example/directory")) {
    Serial.println("The directory exists");
  } else {
    Serial.println("The directory does not exist");
  }
  ```
  הפלט של הקוד הנ"ל יהיה "The directory exists" אם התיקייה קיימת ו"the directory does not exist" אם התיקייה לא קיימת.

## העלייה העמוקה
על פניו, בדיקה של ביצוע תיקייה נראית פשוטה ומובנת. אולם, ישנן מספר שיקולים שכדאי לקחת בחשבון כאשר מבצעים בדיקת תיקייה:

- חשיבות המיקום - כאשר מבצעים בדיקה של תיקייה, כדאי לקחת בחשבון את הנתיב המלא של התיקייה ולוודא שהוא תקין ונמצא במקום הנכון ברכיב החומרה.
- זמן ריצה - במחשבים קטנים כמו ארדוינו, הפעולות הביצועיות יכולות לצאת לפועל יחסית מהר. אם יש לבצע מספר בדיקות, כדאי לחשוב על אופציות שיחסכו זמן ריצה ויאפשרו פעולה יעילה יותר.
