## למה

למשתמשי ארדוינו הבאים מהתכנות, ייתכן שיהיה צורך למחוק תווים התואמים מבנה מסוים. זאת יכולה להיות כתוצאה מנסיונות תיקוני באגים או כחלק מתהליך לשיפור קוד. אין עתידן, צעד זה יכול להיות שימושי במגוון רחב של מצבים, אשר הבנה שלו עשויה לעזור למשתמשי ארדוינו לפתור בעיות מסוימות ולשפר את הקוד שלהם.

## איך לעשות זאת

המחיקה של תווים התואמים מבנה מסוים בארדוינו הנוכחי נעשה באמצעות פקודת `substring()`. השתמשו בדוגמאות הקוד הבאות כדי להבין איך למחוק תווים התואמים מבנה מסוים.

``` Arduino
// דוגמא מ-1: מחיקת אותיות "abc" מנתוני הקלט
String input = "Hello abc World";
input = input.substring(0, 5) + input.substring(8);
// תוצאה: "Hello World"

// דוגמא מ-2: מחיקת מספרים המתחילים בתו "0"
String input = "0123 4567 8901";
int index = input.indexOf('0');
while (index >= 0) {
  input = input.substring(0, index) + input.substring(index + 1);
  index = input.indexOf('0');
}
// תוצאה: "123 4567 91"
```

## שפוכו עומק

פקודת `substring()` מאפשרת למחוק תווים מבנה מסוים על ידי ציון האינדקסים של התווים הנמחקים. הוקפץ על פניו מבנה החלקים השונים של מחרוזת המקור, מאפשר למשתמשים לבחור באיזו חלק מהמחרוזת למחוק ובאיזו לא לפי הצורך. אתם יכולים להשתמש גם בפקודת `replace()` כדי למחוק אותיות מבנה מסוים, אך עליכם לזכור שהיא תחליף את כל המופעים של האות או המילה שתתאים לפטרן המצויין. כמו כן, ישנם יישומים מתקדמים יותר של פקודת
