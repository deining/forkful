## למה: 

מחיקת תווים התואמים תבנית ניתנת לשנייה כדי לקצר קוד ארדואינו ולהפוך אותו לקריא יותר ומובן.

## איך לעשות זאת:

כאשר לוחצים על כפתור מסוים ב-Arduino, רכיב זה ישלח סידרה של תווים לבקר מקושר. אבל בעתיד, כשתרצו להדפיס סדרה רק עם נתונים ספציפיים, תיהיה צורך להסיר את התווים הלא רצויים מסידרה. זה יכול להיות מאתגר, אך עם כמה שורות פשוטות של קוד, ניתן למחוק תווים התואמים תבנית נתונה ולקבל רק את הנתונים הרלוונטיים. ראשית, ניצור משתנה חדש שיכיל את הסדרה שרצינו למחוק ממנה את התווים. לדוגמה, אם ברצוננו להדפיס רק מידע של טמפרטורה מסידרה של נתונים, ניצור משתנה חדש ונכניס לו את הסדרה. לאחר מכן, נשתמש בפקודת "for" לעבור על הסדרה ולבדוק כל תו אם הוא תואם את ההתבנית שרצינו. אם כן, נשתמש בפקודת "remove" כדי למחוק את התו מהסדרה. לאחר מכן, נדפיס את הסדרה החדשה שנותרה עם רק הנתונים הרלוונטיים עבורנו. 

```arduino
String data = "Temp: 27C, Hum: 50%, Light: 60%";
String newData = data;
for(int i = 0; i < data.length(); i++){
  if(data[i] == "H" || data[i] == "L"){
    newData.remove(i);
  }
}
Serial.println(newData); // Output: Temp: 27C
```
