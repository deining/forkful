## למה

כתיבת בדיקות עבור פרויקטי ארדואינו היא חשובה מאוד עבור כל מתכנת. הבדיקות אומנם מוסיפות זמן נוסף לכתיבת הקוד, אך זהו הזמן הכמותי שלא יעלה כלום לידיו התוצאות הראשוניות בתקופת הפיתוח. בסופו של דבר, כתיבת בדיקות עוזרת למתכנתים ליצור תוכניות ארדואינו שמעבירות את כל השגיאות האפשריות כבר בתחילת התהליך ומקטינות אפשרויות תקלות בפעולתן.

## כיצד לבצע

כדי לכתוב בדיקות בפרויקטי ארדואינו, ראשית יש להתקין ספריית כתיבת הבדיקות בפרויקט. לאחר מכן, ניתן להשתמש בפקודות לכתיבת בדיקות ולהפעיל אותן בפתיחת הפרויקט. לדוגמה, כאן נדגים כיצד לכתוב בדיקה פשוטה שבודקת את פעולת הסוויטצ' המחובר לפיני כניסה 2 של הארדואינו:

```Arduino
#include <unity.h>
#include "Switch.h"

Switch mySwitch(2); // יצירת אובייקט מערך למחלצת עם קלט כניסה 2

void testSwitchState() {
    TEST_ASSERT_EQUAL(mySwitch.getState(), HIGH); // בדיקת המצב של הסוויטצ' אמור להיות גבוה
}

void setup() {
    delay(2000); // המתנה לרתיחת הסוגריים של הסריקה
    UNITY_BEGIN();
    RUN_TEST(testSwitchState); // הפעלת הבדיקה
    UNITY_END();
}

void loop() {
  
}
```

כאשר הפרויקט יתאפשר, יתבצע בדיקה ותוצג התוצאה מכילה את המצב הנכון של הסוויטצ' כפי שציפינו.
