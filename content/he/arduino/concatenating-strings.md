# למה:

נדמה שאחד הכישורים החשובים ביותר כאשר מדברים על תכנות ארדווינו הוא היכולת לחבר מחרוזות. המחשב הפיזי שלכם לא יכול להתקשר עםכם בלי מחרוזות. לכן, כדי לנהל משוב מהציוד או לשלוט על הממיר הדו-כיווני שמחובר לרמקולים, אתם צריכים לדעת איך לחבר מחרוזות.

# כיצד לעשות זאת:

היכנסו לקוד המצורף כדי לראות דוגמאות של כיצד לחבר מחרוזות ומקודד אותם לארדווינו. הנה דוגמאות פשוטות לחיבור מחרוזות:

```Arduino
// דוגמא #1: החיבור של שם לשלום!
String name = "עדי";
String greeting = "היי";
Serial.println(greeting + ", " + name);

// דוגמא #2: החיבור של קוד למוצר מוגמר
String productCode = "ARD-001";
String productOffer = "מוצר היום: ";
Serial.println(productOffer + productCode);
```

 פלט לדוגמא #1: היי, עדי
פלט לדוגמא #2: מוצר היום: ARD-001

# הטמעה עמוקה:

אם אתם רוצים לעשות הטמעה עמוקה יותר בנושא חיבור מחרוזות, ישנם תחומים נוספים ללמוד כמו סיבת השימוש ב"פונקציות מובנות" ממחרוזת. ישנם פונקציות כמו `concat()`, `charAt()`, ו- `substring()` שיכולות לסייע לכם בתהליך של עבודה עם מחרוזות בפרויקטים שלכם. חשוב לשים לב לנקודת החזרה הנכונה כאשר אתם משתמשים בפונקציות אלה כדי להבטיח שהתוכנית שלכם תרוץ בצורה תקינה.
