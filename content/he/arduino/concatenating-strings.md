## למה

במחשבים כיום אנו משתמשים לעתים קרובות בתופעת חיבור תווים כחלק מתהליך תכנות מתקדם. לאיזו דבר התרשמתי? חיבור תווים הוא תהליך שבו אנו בונים מחרוזת אחת מתוך שתי מחרוזות נפרדות כדי ליצור מחרוזת חדשה. אנו נהוגים לעשות זאת בקוד כשאנחנו מנסים להציג מסרים מותאמים אישית למשתמשים, ליצור אובייקטים דינמיים ועוד. האם אתה מוכן ללמוד כיצד לבצע חיבור תווים בחלקה הנשכח של עולם הארדוינו? 

## איך לעשות זאת

נחלק את התהליך לשני שלבים עיקריים כדי להסביר איך מבצעים חיבור תווים בארדוינו:

1. כתיבת הקוד: תחילה, צריך להגדיר את המחרוזות שנרצה לחבר. כדי לעשות זאת, נשתמש בפקודה `String` ונשים בתוכה את המחרוזות שנרצה לחבר בין פסיקים.
   ```Arduino
   String greeting = "שלום";
   String name = "דוד";
   ```
2. חיבור המחרוזות: לאחר כתיבת המחרוזות, נשתמש בפקודה `+` כדי לחבר את המחרוזות זו לזו. נשתמש גם בפקודת `println` כדי להדפיס את התוצאה הסופית לטרמינל.
   ```Arduino
   String result = greeting + ", " + name; // מחרוזת סופית שתתוצג בטרמינל: שלום, דוד
   Serial.println(result);
   ```
