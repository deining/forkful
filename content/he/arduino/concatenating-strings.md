## למה

אם אתה מפתח אפליקציות או מחשבים ניידים ואתה צריך לעבוד עם מחרוזות, חברם ביחד על ידי קישור בינהם קריטי עבור התקף חלקי קוד. גם כאשר ניתן לחבר ישראלי לארדואינו במחרוזות, התחביר ארדואינו עצמו הוא לא תמיד ברור, ולכן כתיבת התוכנה שלך על ידי חיבור מחרוזות באמצעות גירסאות נוספות יכולה לאבץץ את המשימה הכל כך דרוש למפתחי ישראל.

## כיצד לעשות

### הערות קוד

קישור מחרוזות באמצעות גירסאות נוספות ניתן לעשות בוואקו ידידותי לארדואינו ופשוט להתאים בין מחרוזות המפרד באמצעות `+` סימן. לדוגמה:

```Arduino
String name = "ישראל";
int age = 73;

String message = "שלום, שמי הוא " + name + " ואני בן " + age;

Serial.println(message);
```

### פלט

פלט המתקבל עבור הקוד בדוגמה העליונה יהיה:

```
שלום, שמי הוא ישראל ואני בן 73
```

## העמקה

כמו שניתן לראות בדוגמה, השימוש בגירסאות נוספות ניתן להרחיב את היכולות של הארדואינו ולהפעיל שלל פעולות המבוססות על מחרוזות, כגון חלוקת מחרוזת לחלקים, חיפוש מחרוזות מסוימות, ועוד.
