# למה?

*למה לזיכרון אטומי:*

יצירת קובץ זמני יכולה להיות שימושי במספר רב של מקרים כאשר אנחנו מבצעים תוכניות על לוח Arduino. תהליך זה מאפשר לנו לאחסן נתונים באופן זמני, ולכן ניתן להשתמש בהם להמשך התוכנית במהלך הריצה.

## איך ליצור קובץ זמני ב- Arduino

כדי ליצור קובץ זמני ב- Arduino, אנו נשתמש בפונקציה `File::createTemp()` שמגדירה בנפרד מחרוזת זמנית ומחזירה קובץ זמני שניתן לשימוש בתוכנית. הנה דוגמא של פונקציית ליצירת קובץ זמני עם שם "temp_file":

```Arduino
File temp_file = File::createTemp();
```

כעת נוכל להשתמש במטודות של האובייקט `temp_file` כדי לכתוב או לקרוא מנתונים מתוך הקובץ הזמני.

לדוגמא:

```Arduino
temp_file.println("שלום עולם!"); // כתיבת נתונים לקובץ זמני
temp_file.close(); // סגירת הקובץ
temp_file = File::openTemp(); // פתיחת הקובץ מחדש לקריאה
Serial.println(temp_file.readString()); // קריאת הנתונים מהקובץ והדפסתם למסך
temp_file.close(); // סגירת הקובץ שוב
```

כפי שאתם רואים, אנו יכולים לבצע פעולות כמו פתיחה, כתיבה וקריאה מהקובץ הזמני כמו כל קובץ אחר.

## חקירה עמוקה: יצירת קובץ זמני בפונקציות תחתית

אם תרצו לחקור את פעולת יצירת הקבצים הזמניים במתקדמים יותר, אתם יכולים לצפות בפונקציית התחתית `createTemp()` במקור הקוד של הספריה File עבור Arduino.
