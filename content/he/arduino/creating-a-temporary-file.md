## למה

כתבתי מאמר זה כדי לענות על שאלה נפוצה בקהילת הארדוינו - למה ליצור קובץ זמני? יתכן שתגיעו לנקודה בקוד שלכם שבה תרצו לאחסן מידע בצורה זמנית כדי לגשת אליו מאוחר יותר. במאמר זה, אני אראה לכם כיצד ליצור קובץ זמני באמצעות ארדוינו וכיצד זה יכול להיות שימושי בפרויקטים שלכם.

## איך לעשות

הקוד הבא מדגים כיצד ליצור קובץ זמני בארדוינו ולהצג את התוכן שלו בקונסולה:

```arduino
#include <SPI.h> // לטעינת ספריות
#include <SD.h> 

String tempData = "Hello, world!"; // המידע שנרצה לאחסן בקובץ זמני
File tempFile; // משתנה מסוג קובץ

void setup() {
  Serial.begin(9600); // פתיחת קשר עם המחשב
  SD.begin(4); // טעינת כרטיס SD
  tempFile = SD.open("temp.txt", FILE_WRITE); // פתיחת קובץ זמני לכתיבה
  tempFile.println(tempData); // כתיבת המידע לקובץ
  tempFile.close(); // סגירת הקובץ
  Serial.println("Temporary file created!"); // הדפסת הודעה בקונסולה
}

void loop() {
  // כאן ניתן להוסיף קוד נוסף לתהליך האחסון הזמני
}
```

כאשר הקוד מוכנה לשלב עם ארדוינו, ניתן לראות בקונסולה את ההודעה "Temporary file created!" ואז לבדוק את התוכן של הקובץ זמני באמצעות תוכנת מקור פתוח כמו Notepad++.
