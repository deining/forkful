## למה:

למה אתם כדאי שתזכו בתאריך הנוכחי בקשת הארדווינו? לדוגמה, ניתן להשתמש בתאריך נוכחי בטמפלייטים שונים, בבניית סדרת אירועים או בסיקור של נתונים בזמן אמת. בנוסף, לזיהוי התאריך נוכחי תוכלו ליצור עורך רפה לארדווינו.

## איך לעשות זאת:

```Arduino

#include <RTClib.h> //מינהלים את הספריה הנדרשת

RTC_DS1307 rtc; //קושרים את המכשיר המתאים

void setup () {
Serial.begin (9600); //הגדרת מהירות שולח

//בדיקה אם הקשר מתיישב צפוי
if (! rtc.begin ())
{
Serial.println ("לימוד תאריך נוכחי לא הצליח!");
Serial.
while (1);
  }
}

void loop (){
DateTime תאריך = rtc.now ();
//אחזרת תאריך נוכחי
Serial.println (תאריך);
}
```

פלט: 2021-07-15 16:07:00 

כעת, אם תוסיפו את הקוד הבא לשורת loop תוכלו להציג את התאריך בפורמט נוח יותר כגון "mm / dd / yyyy":

```Arduino
//פלט: 07/15/2021
Serial.println (תאריך.תאריך / 10,2), תאריך.חודש (), תאריך.שנה ());
```

## חקירה עמוקה:

בחלק זה נראה כיצד תאריך נוכחי מיוצג ומופעל על ידי ארדוינו. בכדי לקרוא את התאריך באופן מלא, הוא מיוצג כמצטבר של כמה יחידות זמן שונות כגון שנים, חודשים, ימים, שעות, דקות ושניות. בנוסף, פונקציית "now" מאפשרת לנו לקבל כל יחידה זמן הכוללת, כמו תאריך וזמן נוכחיים.

ניתן גם לשנות את הפורמט בו התאריך מוצג על ידי שימוש בפונקציות נוספות כמו "day()", "month()" ו-"year()". כמו כן, ניתן להשתמש בפונקציות נוספות כגון "weekday()", "hour
