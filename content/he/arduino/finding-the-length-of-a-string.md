## למה

מצאתם את עצמכם מנסים להתאים תווים או מילים בברכים ותרגילים בארדואינו? או שאתם רוצים להכיר את אורך המחרוזת שלכם לצורך חישובים ותנאים? למדו איך למצוא את אורך המחרוזת שלכם באמצעות ביצוע הקוד הנכון בארדואינו.

## איך לעשות זאת

```Arduino
int stringLength = strlen("שלום עולם!"); // האורך של המחרוזת שקיבלנו יהיה 13
Serial.println(stringLength); // יוצא: 13
```

ניתן למצוא את אורך המחרוזת באמצעות פונקציית `strlen()` הקיימת בספרייה הסטנדרטית של ארדואינו. הפונקציה מקבלת כפרמטר מחרוזת ומחזירה את אורךה ביחידות של תווים.

```Arduino
char myString[] = "ברכים לכולם";
int stringLength = strlen(myString); // המשתנה `stringLength` יכיל את אורך המחרוזת שהוגדרה
Serial.println(stringLength); // יוצא: 11
```

ניתן גם להשתמש בפונקציה על משתנה מטיפוס מחרוזת, כך שכל מחרוזת שתיכנס למשתנה תכניס את אורךה למשתנה האחר.

## חפירה עמוקה

אם אתם מעוניינים להבין איך הפונקציה `strlen()` מחשבת את אורך המחרוזת, כדאי לדעת שכל תו במחרוזת מוצג על ידי טבלת תווים ספציפית. לכל תו מספרי ערך שמייצג אותו והתו הראשון במספר 0. כאשר הפונקציה מקבלת מחרוזת כפרמטר, היא מצפין את כל התווים במחרוזת ומחשבת כמה תווים יש בה על ידי ספירת התווים עד לתו הנל תו המייצג סוף מחרוזת.
