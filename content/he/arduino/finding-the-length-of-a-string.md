## למה

להימנע מבעיות פוטנציאליות ולהבטיח שהקוד שלנו עובד כמו שרצינו, חשוב לדעת את אורך המחרוזת שאנו משתמשים בה. יכול להיות שהפונקציה שלנו מצפה למחרוזת מסוימת באורך מסוים והקוד לא יעבוד כמו שציפינו אם לא נדע את אורך המחרוזת במדויק. לכן, חישוב אורך המחרוזת הוא כלי חשוב בתכנות שלנו.

## איך לעשות זאת

כדי למצוא את אורך המחרוזת ב- Arduino, ישנם כמה דרכים לעשות זאת. נציג כאן דוגמאות לשתי פתרונות אחרים, כך שיהיה לכם ברירת מחדל בהתאם לצרכים שלכם.

```arduino
// דרך ראשונה: באמצעות הפונקציה `strlen`
char str1[] = "arduino";
int len1 = strlen(str1); // len1 תכיל את הערך 7
```
```arduino
// דרך שנייה: באמצעות לולאת `while` לחישוב הגודל של המחרוזת
char str2[] = "hello";
int len2 = 0; // נגדיר משתנה עבור אורך המחרוזת
while(str2[len2] != '\0') { // כל עוד לא נתקלנו באינדקס האחרון של המחרוזת
  len2++; // נוסיף 1 לאורך המחרוזת כל פעם שנרוץ על הלולאה
}
// len2 תכיל את הערך 5
```
