## מדוע
למה אפיון האורך של מחרוזת יכול להיות חשוב בתכנות? אחת הסיבות הנפוצות היא לאבחון ולתקן בעיות בלתי צפויות כמו ייעולי מעבדה או תגובה תקינה לקלט משתמש.

## איך לעשות זאת
כדי להשתמש בפונקציה לאפיון האורך של מחרוזת בארדוינו, תצטרכו לכתוב את המחרוזת שאתם מעוניינים למדוד בתוך סוגריים כפולים ("). לדוגמה:
```Arduino
String myString = "שלום עולם";
```
כאשר רצים את ארדוינו (Arduino), תוכלו להשתמש בפונקציה `length()` כדי להדפיס את אורך המחרוזת שלכם. לדוגמה:
```Arduino
int stringLength = myString.length();
Serial.println(stringLength);
```
הפלט של הקוד יהיה 10 משום שיש 10 תווים במחרוזת "שלום עולם".

## העומק של הנושא
אנו כבר יודעים כיצד לקבוע את האורך של מחרוזת בארדוינו באמצעות הפונקציה `length()`, אך מה אם אנו רוצים לעקוף תווים מסוימים כאשר אנו מגדירים את האורך? למשל, אם נרצה שאורך המחרוזת יתווסף על מנת לכלול את כולם פרט מסוים, נוכל להשתמש בפונקציה `substring()` כדי למשוך רק את החלק הרלוונטי של המחרוזת. לדוגמה, אם נרצה למצוא את האורך של המחרוזת "שלום עולם" ללא מרווחים, ניתן להשתמש בפקודה הבאה:
```Arduino
String myString = "שלום עולם";
myString.substring(0, 7);
```
הפלט של הקוד יהיה 7, מכיוון שאנו מושגים רק את החלק הרלוונטי של המחרוזת בין התווים 0 ו-7 (שהם האותיות "שלום עולמ").
