## מדוע

**תושבים הנכבדים של קהילת Arduino, אתם מוזמנים לכתוב את השפה שלכם של קוד ההתקתקווים.** 

קריאת פקודות בשורת הפקודות היא כלי מועיל לתכנים עם הרבה מתיווכים או אפליקציות שדורשות שליטה ברמת נמוכה יותר. אחד הדברים הבסיסיים שיכולים לתרום לעבודה היא דרך קריאת פקודות בשורת הפקודות.

## כיצד ל

**למד כיצד לקרוא ולטפל בפקודות של פקודה בשורת הפקודות עם אם אנו רוצים ליצור שינויים על מאפיינים של ארדואינו.**

כדי לפעול עם פקודות בשורת הפקודות בקוד ארדואינו, אנו צריכים להשתמש בפונקציה `Serial.read()`. כאשר מנקרות המידע הגיעו לארדואינו דרך כבל USB, הוא חובב אותם באמצעות פונקציית `read()`. 

דוגמאות לקריאת פקודות בשורת הפקודות:

```Arduino
int val = Serial.read();
Serial.println(val);
```

תוצאה: מספר המייצג את הפקודה בערך דוח אם נחזור במסלול דרך, הגיעה פקודה עם ביטים שכן בהם יבונים את העבודה, המשתמשים בה להריץ תצוגות אישיות ולנהל את הנכוניות אנטאקפר כאשר הם לא פגושים כשהם משתמשים במצבים פתחם בעורכת של משתמשים אחרת.

```Arduino
int val = Serial.read();
if (val == 49) {
  Serial.println("Command A");
} else if (val == 50) {
  Serial.println("Command B");
} else {
  Serial.println("Unknown command");
}
```
