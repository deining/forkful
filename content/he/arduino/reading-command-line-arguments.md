# למה:

קריאת ארגומנטים מכפתור הפקודה בארדוינו היא כלי מאוד שימושי לכתיבת קוד מתקדם ויישומים מתפתחים. היא מאפשרת לקבל קלטים מהמשתמש בזמן הריצה של התוכנית ולשנות את התנאים והנתונים של הקוד בהתאם, במקום לכתוב אותם ישירות בקוד המקור. קריאת ארגומנטים מעניקה גם גמישות ונוחות תוך שמירה על קוד נקי ומאורגן.

# איך לעשות:

כאשר נרצה לקרוא ארגומנט מכפתור הפקודה בארדוינו, נצטרך להשתמש בפונקציית "Serial.read()" לקרוא את הנתון הנמצא בכפתור. נשתמש בלולאת "while" כדי לאתר האם יש נתון קיים בכפתור ואם כן, נשתמש בתנאי "if" כדי לאתר את הפקודה המבוקשת ולטפל בה בהתאם. לדוגמה, אם נרצה לשנות את מהירות התאורה בשלב הבא של הקוד, נוכל להשתמש בפקודה "Serial.read()" כדי לקבל את הערך של המהירות מהמשתמש ולשנות את הערך בהתאם.

  Serial.begin(9600); // הגדרת קצב שידור סיריאלי
}

  while(Serial.available()) { // בדיקת קיום נתון בכפתור הפקודה
