## למה

כדי לחזק את קוד ה-ארדואינו ולהפוך אותו ליותר יעיל ונוח לשימוש, ניתן להשתמש בביטויים רגולריים. הם מאפשרים לנו לחפש ולהשתמש במילות מפתח או תבניות מסוימות בטקסט שאנו מקבלים מהחיישנים.

## כיצד להשתמש בביטויים רגולריים בארדואינו

בכדי להשתמש בביטויים רגולריים בארדואינו, נצטרך להתעלם מהמערכת המובנית ולהשתמש בספריה חיצונית שנקראת "Regex". ניתן להתקין את הספריה הנ"ל על ידי לחיצה על "כלים" בתפריט העליון של התוכנית ובחירת "הוספת ספרייה", ולאחר מכן לחיצה על "Regex". לאחר מכן נוכל להתחיל להשתמש בביטויים רגולריים בקוד שלנו בעזרת הפקודות הנלוות:

```Arduino
#include <regex.h> // כניסה לספריה

String input = "Hello, my name is John!"; // טקסט לבדיקה
regex matchName("my name is (\\w+)"); // ביטוי רגולרי לחיפוש של שם
match_results<String::iterator> results; // התאמת התבנית לטקסט
regex_search(input, results, matchName); // החפץ והתאמת התבנית לטקסט

if (results[1] == "John") { // בדיקת התאמה
    Serial.println("Found John!"); // פלט: Found John!
}
```
