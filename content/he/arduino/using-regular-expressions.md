# למה
כתיבת קוד היא כישור ייחודי ובעל חשיבות רבה בתחום ההנדסה האלקטרונית. אחד האתגרים הגדולים בכתיבת קוד הוא הבנה וניהול של מחרוזות מורכבות. כלומר, נחוץ למצוא דרך מסוימת לאתר ולטפל במילים או תבניות מסוימות בתוך טקסט מורכב. זאת הסיבה שישנם כלים מתקדמים כגון רגולריים שניתן להשתמש בהם עבור דיפי דייב וכתיבת קוד במכשירי ארדואינו.

# איך להשתמש ברגולריים בארדואינו
כשמדובר בכתיבת קוד בארדואינו, יש להתעסק עם מחרוזות מורכבות וניהולן. כדי לטפל בכך, ניתן להשתמש ברגולריים. הרגולריים הם דפסים תבניות או מילוליים שמאפשרים למצוא ולשנות את הטקסט בצורה פשוטה ומהירה. לכתוב רגולריים בארדואינו משתמשים בספריית Regex. הנה דוגמא של קוד עם שימוש ברגולריים לניהול מחרוזת:

```Arduino
#include <Regex.h>
String text = "Hello_world_123";
Regex regex("[a-z]+");
if (regex.find(text)) {
  Serial.println(regex.getString(0));
}
```

הקוד הנ"ל ימצא וידפיס את המילה "Hello" מתוך המחרוזת "Hello_world_123". ניתן להשתמש ברגולריים גם לצורך תיקון או החלפת סדר של אותיות ותווים במחרוזת.
