ארדואינו פוסט קלאסי בתכנות שיטת ארדואינומדוע: רק 1-2 משפטים להסביר * למה * מישהו ייתקע בשימוש בביטויים רגולריים.
איך ל: דוגמאות קוד ותמונת פלט בתוך תבניות קוד "```ארדואינו ... ```".
עמוק לכיוון: מידע עמוק יותר על שימוש בביטויים רגולריים.
גם ראה: רשימת קישורים להמשך למידה על שימוש בביטויים רגולריים.

המטרה של ביטויים רגולריים היא לאפשר לנו לחפש מאפיינים ספציפיים בטקסט, כגון מילים מסוימות או תבניות, ולעבד את המידע הזה. בארדואינו, ביטויים רגולריים משמשים להתאמת טקסט כדי לעבוד בדקות וביעילות. אם אתם מתכנתים בארדואינו ורוצים למחוק מילים ספציפיות מטקסט או לחפש ספרת נתונים מסוימת מתוך טקסט, כדאי ללמוד את יכולות הביטויים הרגולריים.

## איך

להשתמש בביטויים רגולריים בארדואינו, עליכם להתקין את הספרייה "Regexp". לפני שאתם מתחילים לכתוב את הקוד, צריך להגדיר את הביטוי הרגולרי שתרצו להשתמש בו באמצעות חיבור של נתוני סטרינג לביטוי רגולרי מסוים. הנה דוגמא לקוד עם ביטוי רגולרי שמחפש ספרה בטקסט המקור:

```ארדואינו
#include <Regexp.h>

void setup() {
  Serial.begin(9600);
  String str = "זאת ספרה 123 בטקסט המקור";
  String regex = "(\\d+)";
  Serial.println(str);
  if (str.match(regex)) {
    Serial.println("מצאנו ספרה!");
  }
  else {
    Serial.println("לא מצאנו ספרה :(");
  }
}

void loop() {

}
```
