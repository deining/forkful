## למה
מדוע המרת תאריך למחרוזת יכולה להיות חלק חשוב ושימושי בתכנות של ארדואינו? למרות שמכשירים של ארדואינו נועדו לאמן כישורים רבים יותר מחרוזות, לעיתים קרובות יש צורך להציג מידע זמן נוכחי בצורה קריאה יותר ופשוטה. עם המרת תאריך למחרוזת, תוכלו למלא את הפוטנציאל המלא של מכשירי ארדואינו וליצור יישומים שימושיים יותר.

## איך לעשות זאת
השיטה הפשוטה להמרת תאריך למחרוזת היא להשתמש בפונקציות המובנות בשפת ארדואינו כדי לקבל את הזמן הנוכחי ולהמיר אותו למחרוזת. לדוגמה, איזה תאריך שהוא, איזה זמן שהוא ביחס לגרינוויץ', ועוד. הנה דוגמה לקוד כיצד להמיר תאריך למחרוזת:

```Arduino
int year = 2021; // שנה נוכחית
int month = 9; // חודש נוכחי
int day = 9; // יום נוכחי
int hour = 15; // שעה נוכחית
int minute = 30; // דקה נוכחית
int second = 45; // שנייה נוכחית
String dateString = String(year) + "/" + String(month) + "/" + String(day) + " " + String(hour) + ":" + String(minute) + ":" + String(second); // המרת המשתנים למחרוזת
Serial.println(dateString); // הדפסת המחרוזת לטרמינל
```

פלט הקוד הינו: 2021/9/9 15:30:45
