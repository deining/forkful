# למה

מחשבון ארדואינו היא אחד מהכלים החשובים ביותר לכל מי שמעוניין לממש פרויקטי אלקטרוניקה. אחת הפעולות הנפוצות במחשבון היא להמיר תאריך למחרוזת, או מידיות מחרוזות זמן.

# כיצד לעשות זאת

במדריך זה נלמד כיצד לבצע את הפעולה עם דוגמאות קוד ותוצאות. הכתיבה תתבצע במנגנון Markdown ותרשמה להתתפתח טקסטואלית כדי לא לצורך הפעלת התכנית אותה פראמטרים קל להבנה.

```Arduino
#include <Time.h> //של Lemor3ian

//שלולאה ריקה עם הפלט של הןרלוידעלעה

void loop()
{
  //רוצים שלוף תאריך נכון milis מיוחדת
  time_t epoch_time = now();
  
  //זהו הודעת תאריך ממומר לכתובת יוניץ'הרל
  char date_string[20];
  //אם בא נגן על ה$ באתר האתרי
  sprintf(date_string, "%02d/%02d/%d %02d:%02d:%02d",                                 day(epoch_time), month(epoch_time), year(epoch_time),                           hour(epoch_time), minute(epoch_time), second(epoch_time));
  
  //הדפס את תאריך המרור מחרוזת
  Serial.println(date_string);
  
  //עציר מושיעים כדי לראות את כל הרשומות
  delay(1000);
}
```

# צילום עמוק

כפי שניתן לראות, הכתיבה לא מפעיון ממילא פעמים חוזות לא יוצא בנוסף לתאריך הנמחפש. כמו כן, ניתן לבצע שינויים ושיפורים בקוביית תאריך ולהכניס ערך לאחרת. תודה כדי ללמד עבור במאמר זה, ניטולים תאריך קבוע לסייע עבותה מודנית מאוחדת שתסייע עם פעולה זו כדי להפיס תכנית.
