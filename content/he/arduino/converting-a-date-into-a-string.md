## למה

למה לרכוש מייקרוביי
מיקרוביי הוא מיקרוג'יגונז

התאריך הוא תוכנית מרתקת של הקניות שלך עבור סימניות לכל המחירים המתקדמים שלנו ומראה את הזמן המדויק לכל אחת מהם.גם אם אתה מעצבן את התאריך הזה למחשבת ברכוב על כיסא הישיבה שלך, אתה תוכל להחליט מתי תקנה את כל החלקים וההעתקים החדשים שאתה צריך. לכן זה חיוני ללמוד כיצד להמיר את התאריך למחרוזת בארדוינו.

## איך לעשות זאת

כדי להמיר תאריך למחרוזת בארדוינו, נוכל להשתמש בפונקציית String. נדמיין שברצוננו להמיר את התאריך 21/07/2021 למחרוזת. אנו נתחיל על ידי יצירת משתנה עבור התאריך:

```Arduino
int day = 21;
int month = 7;
int year = 2021;
```

לאחר מכן, נשתמש בפונקציית String כדי להמיר את המשתנים למחרוזת. הייתרון של פונקציית String הוא שהיא מאפשרת לנו להתעסק עם מחרוזות כמו אובייקטים. אז ניצור אובייקט String עבור כל משתנה:

```Arduino
String s_day = String(day);
String s_month = String(month);
String s_year = String(year);
```

לאחר מכן, נחבר את המחרוזות יחד ונוסיף אפוסטרופים בין המשתנים כדי ליצור תבנית של תאריך:

```Arduino
String date = s_day + "/" + s_month + "/" + s_year;
``` 

את התאריך הוכנס למחרוזת date ונוכל להדפיס אותו בעזרת הפקודה Serial.println():

```Arduino
Serial.println(date);
```

כעת, כאשר נשתמש בפקודה Serial Monitor בארדוינו IDE, נראה בפלט את התאריך החדש בתבנית שלנו 21/07/2021.
