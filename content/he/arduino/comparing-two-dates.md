## למה

למה להשתתף בהשוואת שני תאריכים? ייתכן שתרצו לבדוק אם תאריך מסוים אפנה או יעיל יותר מתאריך אחר, או שתרצו ליצור תנאים מותאמים באופן דינאמי לתאריך נתון.

## איך לעשות זאת

תהליך השוואת שני תאריכים דורש שימוש בפקודות של מכשיר ארדואינו. כאן נמצא דוגמה לקוד פשוט שמדגים את התהליך בשפת C++:

```
# הגדר תאריך ראשון
ArduinoDateTime d1 = {2020, 1, 1, 0, 0, 0};

# הגדר תאריך שני
ArduinoDateTime d2 = {2019, 12, 31, 23, 59, 59};

# השווה את התאריכים והדפס את התוצאה (0 או 1)
Serial.println(d1 == d2); // יחזיר 0
```

בדוגמה זו, אנו משווים בין שני תאריכים ומדפיסים את התוצאה. ניתן להשתמש גם בפקודות נוספות כדי להשוות את התאריכים לפי תנאים שונים, כמו לדעת אם הם אותו יום בשבוע או האם התאריך הראשון קודם או מאוחר יותר מהתאריך השני.

## העומק של הנושא

לשוואת תאריכים יש כמה היבטים שניתן לעשות בהם שימוש. בעזרת הפקודות של מכשיר ארדואינו, ניתן להשוות את השנה, החודש, היום, שעה, דקה ושנייה של תאריך כלשהו. ניתן להשתמש גם בפקודות נוספות כדי להשוות בין תאריכים שונים, לבדוק תנאים כגון אם השנה הינה מעוברת או אם התאריך מתאים ליום שבועי מסוים.
