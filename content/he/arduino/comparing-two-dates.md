# למה

בעולם התכנות קיימות מספר רב של מצבים שבהם עלינו להשתמש בתאריכים. לדוגמה, כאשר יש לנו אירועים שצריך לתכנן או לחיות בדיוק כמו חתונות ויום הולדת, חשוב להיות מסוגלים להשוות בין שני תאריכים כדי להבין מהו הקטע הטוב יותר. במאמר הזה, אנו נלמד כיצד להשוות בין תאריכים בעזרת ערדוינו.

# איך לעשות זאת

הבסיס לעבודה עם תאריכים בערדוינו הוא זמן UNIX, אשר מורכב ממספר שניונים נפרדים: זמן נוכחי ותאריך. בכדי להשוות בין שני תאריכים בעזרת ערדוינו, נצטרך להמיר את התאריכים לזמן UNIX ולאחר מכן להשוות ביניהם.

בקוד הבא נראה דוגמאות להשוואת שני תאריכים עם פלט תוצאה:

```arduino
// הגדרת תאריכים
int date1 = 20210921; // תאריך ראשון בפורמט yyyymmdd
int date2 = 20210918; // תאריך שני בפורמט yyyymmdd

// המרת תאריכים לזמן UNIX
long unix1 = timeUtils.getTime(date1);
long unix2 = timeUtils.getTime(date2);

// השוואת תאריכים והדפסת תוצאה
if(unix1 > unix2) {
    Serial.println("תאריך 1 מאוחר יותר מתאריך 2");
} else if(unix1 < unix2) {
    Serial.println("תאריך 1 מוקדם יותר מתאריך 2");
} else {
    Serial.println("שני התאריכים זהים");
}
```

פלט תוצאה:
תאריך 1 מאוחר יותר מתאריך 2
