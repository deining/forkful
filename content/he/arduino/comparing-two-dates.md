# למה:

השוואת תאריכים היא כלי חשוב בתכנות הארדואינו שיכול לסייע לנו להבין מתי משהו קרה ולהשתמש במידע זה לצרכים שונים. אם אתה מעוניין לאתגר את עצמך ולהשתפר בתכנות הארדואינו, עליך ללמוד כיצד להשוות שני תאריכים ולהשתמש בהם בקוד שלך.

## כיצד לעשות את זה:

הנה כמה דוגמאות של קוד שיעזרו לך להבין טוב יותר כיצד להשוות שני תאריכים בארדואינו.

```Arduino
// ייבוא ספריות חשובות

#include <DateTime.h> 
#include <DS1307RTC.h>

// קביעת תאריך מופעל

void setup()
{
    // הגדרת חיבור הטבלה ומחיקת פרטים לפני עדכון
    setSyncProvider(RTC.get);
    RTC.setDate(15, 7, 2021); 
    RTC.setTime(12, 30, 0); 
}

// קורא ומדפיס את התאריך הנוכחי

void loop()
{
    // יצירת רכיבי זמן חדשים
    DateTime now = RTC now);
  
    // השווה את התאריך הנוכחי לתאריך מופלא

    if (now == DateTime(2021, 7, 15))
    {
        Serial.println("היום הוא יום הולדתי!");
    }
    else
    {
        Serial.println("היום אינו יום הולדתי.");
    }
    
    // המתן 1 שניות לפני הצגת תאריך חדש
    delay(1000);
}
```

### פלט:

המכשיר שלך ידפיס "היום הוא יום הולדתי!" אם התאריך הנוכחי הוא 15/07/2021. אחרת, הוא ידפיס "היום אינו יום הולדתי."

## העומק:

עכשיו שאתה יודע כיצד להשוות שני תאריכים בארדואינו, כדאי ללמוד כיצד להשתמש במידע זה לצורךים שונים. לדוגמה, ניתן להשתמש בהשוואת תאריכים כדי ליצור מערכות אירוח במכשיר שלך או ליצור פעולות יומיות בהתאם לתאריך הנוכחי.
