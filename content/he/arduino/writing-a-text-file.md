# למה

רק כמה משפטים לפרט *למה* מישהו ירצה לכתוב קובץ טקסט.

כתיבת קבצים טקסט נחשבת לחלק חשוב ומרכזי בתכנות של ארדואינו. על ידי יצירת קבצים טקסט, אנו מאפשרים לבצע פעולות ופונקציות מתוכננות מראש בלוח המיקרו-בקר הנמצא בתוך הארדואינו.

## איך לעשות זאת

לתוכנית זאת יש כמה צעדים בסיסיים כדי ליצור קובץ טקסט ולמלא אותו במידע. הנה מספר הדגמאות של קוד ופלט בתוך חלק הטקסט "```Arduino ... ```". 

### צור ופתח קובץ טקסט חדש

לפני כל דבר, נצטרך ליצור שני משתנים - אחד לשמור את הקובץ שנבחר ואחד להכיר מול הקוד עבור הערך שנרצה להכניס לקובץ. הנה דוגמא קוד ליצירת קובץ ופתיחתו עם ביצוע הפריקה אותו לחלקים עבור שני המשתנים: 

```
ArduinoString file_name = "my_file.txt"; 
ArduinoString text_message = "Hello, world!"; 
```

השתמשו בפקודות כדי לסיים את היצירה והפתיחה של הקובץ.

### כתוב מידע לקובץ

עכשיו שהקובץ פתוח, אנו יכולים להשתמש בפקודות כדי לכתוב את המידע שנרצה להוסיף. הנה כמה שורות קוד להוספת טקסט לקובץ:

```
File.write(text_message); 
File.flush();
File.close();
```
