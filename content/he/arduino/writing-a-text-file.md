## למה

כתיבת קובץ טקסט יכולה להיות כלי חשוב כאשר משתמשים במיקרוקוויק (Arduino) - היא דרך מעולה לשמור מידע ולעבד אותו בקוד בקלות ובנוחות.

## כיצד לעשות זאת

לפני שנתחיל, חשוב לזכור שכיתבת קוד למיקרוקוויק (Arduino), יש להשתמש בתכנית הבאה: שפת Arduino IDE.

היצירה של קובץ טקסט באמצעות מיקרוקוויק ייחודי מאוד קלה. הנה קוד לדוגמה שיצר קובץ טקסט חדש בשם "myFile.txt" ויכתוב את המחרוזת "היי עולם" בתוכו:

```Arduino
File myFile = SD.open("myFile.txt", FILE_WRITE);
myFile.println("היי עולם");
myFile.close();
```

הפקודות נותנות פקודה לפתוח קובץ, להוסיף נתונים ולסגור אותו בסופו.

אם נירצה לקרוא את הקובץ שנוצר, נוכל לעשות זאת באמצעות הפקודה הבאה:

```Arduino
File myFile = SD.open("myFile.txt");
while (myFile.available()) {
  Serial.write(myFile.read());
}
myFile.close();
```

כאן אנו פותחים את הקובץ וקוראים את המידע שנכלל בו, ומדפיסים אותו לטרמינל עם הפקודה `Serial.write`. נא לזכור לסגור את הקובץ לאחר כל פעולה.
