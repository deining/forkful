## מה ולמה?
כתיבת קובץ טקסט היא אחת הפעולות המרכזיות בתכנות המתבצעת ביחד עם כתיבת קוד. הכוונה היא ליצור קובץ המכיל טקסט מסוים, כגון מסרים, מידע או נתונים. תוכנית ארדואינו יכולה ליצור קובץ טקסט כך שניתן יהיה להשתמש בו בהמשך.

## איך לעשות זאת?
ניתן לכתוב קוד בארדואינו שיצור קובץ טקסט בעזרת הפונקציות המתאימות. למשל, הפונקציה "Serial.println()" מאפשרת להדפיס טקסט לטור הפלט הקיים בתכנית המחשב הראשונית של כבל ה- USB המחובר ללוח ארדואינו. ניתן להשתמש בפונקציות דומות כדי להדפיס טקסט לקובץ טקסט במקום לטור הפלט.

```arduino
Serial.println("כתיבה לקובץ טקסט!");
File myFile = SD.open("test.txt", FILE_WRITE);
if (myFile) {
  myFile.println("כאן תוכלו לכתוב כל מה שאתם רוצים אל הקובץ");
  myFile.close();
}
```

לאחר הרצת הקוד, ניתן למצוא את הקובץ בתיקיית הכרטיס ה- SD המחובר ללוח ארדואינו. ניתן לקרוא את הקובץ הזה באמצעות תכנות אחרות או להוסיף עוד קוד לתוכנית הארדואינו שיקרא ויעבד את המידע בקובץ.

## צלילה עמוקה
כתיבת קובץ טקסט היא חלק מרכזי בבניית תכניות ארדואינו וכמעט תמיד נדרשת כאשר מפתחים תוכניות שכוללות מידע או נתונים יוצאי דופן. אלטרנטיבות לכתיבת קובץ טקסט כוללות שמירה של נתונים בזיכרון פנימי או העברתם למחשב מחברת הפונקציות של ארדואינו.
