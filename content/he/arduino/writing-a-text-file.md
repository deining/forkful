## על מה?

כתיבת קבצי טקסט בארדוינו היא דרך מצוינת לאורך טווח לשמור על מידע ולקבל נתונים מהמשתמש. זה יכול להיות שימושי למטרות שונות, כגון ניהול מידע או ייצור נתונים לצורך שימוש עתידי.

## איך לעשות את זה?

תחילה, נצטרך להוסיף את הספרייה "SD.h" לקוד הארדוינו שלנו. זאת נדרשת כדי לאפשר לנו לתקשר עם כרטיסי הזיכרון הקשיח שמחוברים למיקרו-קונטרולר שלנו. לאחר מכן, נכתוב קוד לפתיחת וכתיבת קובץ טקסט חדש:

```Arduino
#include <SD.h>

File myFile;

void setup() {
  Serial.begin(9600);
  // בדיקה האם הכרטיס מחובר ופתיחת קובץ טקסט לכתיבה
  if (SD.begin(10)) {
    Serial.println("מזכיר כי כרטיס זיכרון הוחבר");
    myFile = SD.open("myTextFile.txt", FILE_WRITE);
    // פתיחת הקובץ להוספת נתונים
    if (myFile) {
      Serial.println("מזכיר כי הקובץ נפתח ומוכן לכתיבה");
      // כתיבת תוכן לקובץ
      myFile.println("זהו מסע תחליף קטן לקובץ טקסט");
      myFile.close();
      Serial.println("הקובץ נכתב בהצלחה");
    } else {
      Serial.println("אין אפשרות לפתוח את הקובץ");
    }
  }
}

void loop() {
  // לא נעשה כאן כלום
}
```
