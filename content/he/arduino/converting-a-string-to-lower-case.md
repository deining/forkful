# למה
קוד יותר מובן עבור קוראים כשהוא משתמש באותיות קטנות במקום אותיות גדולות. במאמר זה נלמד כיצד להמיר מחרוזת לאותיות קטנות בפיתוח ארדואינו.

# כיצד לעשות זאת
כדי להמיר מחרוזת לאותיות קטנות בארדואינו, יש להשתמש בפונקציה `toLowerCase()` כפי שמוצג בקוד הבא:

```Arduino
void setup() {
  Serial.begin(9600); //התחלת השרתות הסירליות
  String str = "HELLO";
  Serial.println(str.toLowerCase()); //הצגת "hello"
}

void loop() {}

```

על ידי שימוש בפונקציה `toLowerCase()` ניתן להמיר את כל האותיות במחרוזת לאותיות קטנות באמצעות הפקודה הבאה: `str.toLowerCase()`. בתוך הפקודה נשתמש במשתנה שמכיל את המחרוזת ונוסיף לו את הפונקציה `toLowerCase()`.

הפקודה תחזיר את המחרוזת המומרת לאותיות קטנות, שתוכלו לראות בתוצאה המוצגת בסריאל מוניטור.

# חקירה מעמיקה
על מנת להבין מדוע משתמשים בהמרת מחרוזת לאותיות קטנות, נצפה בקוד הבא:

```Arduino
void setup() {
  Serial.begin(9600); //התחלת השרתות הסירליות
  String str1 = "ArDuInO";
  String str2 = "aRdUiNo";
  if(str1.toLowerCase() == str2.toLowerCase()){
    Serial.println("המחרוזות זהות!"); //הצגת "המחרוזות זהות!"
  } else {
    Serial.println("המחרוזות שונות!"); //הצגת "המחרוזות שונות!"
  }
}

void loop() {}

```

בקוד זה נשתמש בפונקציה `toLowerCase()` בשני משתנים שמכילים את אותה מחרוזת, אך באותיות גדולות וקטנות בהתאמה. נשתמש בתנאי `if` כדי לבדוק האם המחרוזות זהות אחת לשנייה לא משנה מהגודל של האותיות.
