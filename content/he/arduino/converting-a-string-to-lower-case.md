## למה:

למה להתעסק בהמרת מחרוזת לאותיות קטנות?

השתמשתי בכלים רבים לאורך השנים כדי לכתוב קטעי תוכנה ופרויקטים ארדינו. אחד האתגרים הטכניים המרכזיים הוא לטפל במחרוזות. מחרוזות הן אוסף של אותיות ותווים המייצגים מידע טקסטואלי. במצבים מסוימים, ייתכנו צורך להמיר מחרוזת לאותיות קטנות כדי לבצע משוואות והשוואות תקפות. למשל, אם אנו צריכים לבדוק אם הקלט שהתקבל מהמשתמש תואם למילה מפתח מסוימת, ייתכן שנצטרך להמיר את הקלט לאותיות קטנות לפני שאנו משווים אותו למילה המפתח.

## איך להמיר מחרוזת לאותיות קטנות:

```arduino
void setup() {
  Serial.begin(9600);
  String myString = "HELLO WORLD";
  String lowerString = myString.toLowerCase();
  Serial.println(lowerString);
}
```

פלט:
```arduino
hello world
```

בקוד הזה, אנו משתמשים בפונקציה `toLowerCase()` כדי להמיר את המחרוזת לאותיות קטנות. הפונקציה מחזירה את המחרוזת המומרת לפרמטר הממיר שהיינו כותבים לפני כן. בפלט שלנו, המחרוזת "HELLO WORLD" מהתאריך myString מומרת לאותיות קטנות ומודפסת בכלי הקלט הסיריאלי.

## מהמחקרים העמוקים:
