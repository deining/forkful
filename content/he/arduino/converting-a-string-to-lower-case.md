## למה

שלום חברים! במאמר הזה אנחנו נלמד לכתוב תוכנית בארדואינו שתאפשר לנו להמיר מחרוזת לאותיות קטנות. אבל למה זה נחוץ? ישנם מצבים כאשר נרצה לבדוק את מחרוזת כשהאותיות בה מופיעות בכתב קטן, למשל בשימוש בתנאים או במחרוזות המתבצעות באופן דינמי בתוכנית. עם הפונקציה שנלמד לכתוב, זה יהיה פשוט וקל!

## איך לעשות זאת

תחילה, נצטרך ליצור משתנה שיכיל את המחרוזת שנרצה להמיר לאותיות קטנות. נקרא למשתנה ״str״ ונתחיל בכתיבת הקוד:

```Arduino
String str = "ARDUINO";
```

ניצור משתנה נוסף שיכיל את האורך של המחרוזת המקורית על מנת להשתמש בו בלולאה:

```Arduino
int len = str.length();
```

עכשיו נשתמש בלולאה for כדי לעבור על כל אות במחרוזת ולהמיר אותה לאות קטנה. תוך כדי כך, נשתמש בפונקציה charAt כדי לקבל את האות הנוכחית של הלולאה ולהמיר אותה לאות קטנה באמצעות פונקציה toLowerCase. לבסוף, נשתמש בפונקציה setCharAt כדי להחליף את האות הנוכחית במחרוזת לאות הקטנה ולשמור את העדכון במשתנה str. הקוד יראה כך:

```Arduino
for(int i = 0; i < len; i++) {
  char letter = str.charAt(i);
  char lowercase = toLowerCase(letter);
  str.setCharAt(i, lowercase);
}
```

כעת נוכיח שזה עובד על ידי הדפסת המחרוזת המעודכנת:

```Arduino
Serial.println(str);
```

כאשר נריץ את התוכנית ונבדוק את התוצאה במוניטור הסידורי, המחרוזת ״ARDUINO״ תהפוך ל״arduino״ - כל האותיות יהיו בכתב קטן!
