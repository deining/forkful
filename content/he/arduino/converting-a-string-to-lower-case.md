## מדוע
 למה להתעסק בהמרת מחרוזת לאותיות קטנות רק במכשירי ארדווינו?
התוכנית או הפרויקט שאנחנו מעסקים בו עלול לכלול אינפורמציה באותיות גדולות או קפיטלים בתוך הקוד, אבל זה חשוב להיות מנומקים ולהתחבר לשפה שכל המשתמשים יכולים להבין. לכן, ייתכן שאנו נתקלים בצורך להמיר מחרוזת לאותיות קטנות כדי להקל על הפעולות המכניות שלנו.

## איך לעשות זאת
```Arduino
// נתחיל על ידי הגדרת מחרוזת כלשהי
String text = "Hello World";
// נשתמש בפונקציה פנימית של ארדווינו להמרת החרוז לקטניות
text.toLowerCase();
// נדפיס את המחרוזת המומרת לאותיות קטנות
Serial.println(text);

// תוצאה מודפסת על המסך הסדיר:
// hello world
```

## טיול עמוק
כדי להבין טוב יותר כיצד התהליך של המרת המחרוזת לאותיות קטנות עובד, ניתן לראות כי פונקציית ה-`toLowerCase` משנה את כל התווים במחרוזת לאותיות בקטניות. כמו כן, חשוב לזכור כי פונקציית ה-`toLowerCase` משנה את המחרוזת המקורית ולא יוצרת מחרוזת חדשה. בנוסף, כדי להשתמש בפונקציה על ידי כתיבת השם שלה והסוגריים, עלינו לוודא שאנחנו נותנים לה גישה למחרוזת ממשתנה שהוגדר או מערך של אותיות.
