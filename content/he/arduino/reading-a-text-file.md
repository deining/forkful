## מדוע:

בכדי ליצור פרויקטים מתקדמים עם ארדוינו נדרשת היכולת לקרוא קבצי טקסט, כגון קבצי קוד או נתוני תצוגה. כתב המאמר הזה יסביר כיצד לקרוא קובץ טקסט בארדוינו באמצעות שיטות שונות ומה הימצאויות שלהן.

## כיצד לעשות:

קוד לקריאת קובץ טקסט בארדוינו נראה כך:

```arduino
File file = SD.open("file.txt");
if (file) {
  // קוד לטיפול בקובץ כאן
  file.close();
} else {
  // טיפול בשגיאה אם הקובץ לא נמצא
}
```

כאשר מפעילים את הקוד הזה, הארדוינו יפתח את הקובץ "file.txt" הממוקם בכרטיס ה SD ויתחיל לקרוא את התוכן שלו. אם הקובץ נמצא והקריאה נעשית בהצלחה, תוכלו לעבוד עם הנתונים שנקראו. אחרת, יופיעה הודעת שגיאה.

ניתן להשתמש בפקודות נוספות כדי לעבוד עם הקובץ, כגון:

- `file.read()` - לקרוא תו אחד בכל פעם מהקובץ ולהחזיר את הערך שלו בתור מספר בין 0 ל 255.
- `file.readString()` - לקרוא את התוכן של הקובץ כמחרוזת.
- `file.available()` - לבדוק אם עדיין נשאר תוכן שלא נקרא מהקובץ.
- `file.seek()` - לנווט לתו מסוים בקובץ ולהתחיל לקרוא משם.
