תכנות קזואלית עבור Arduino

##למה
כתובת קבצי טקסט מהווה כלי עיקרי בתכנות של מיקרובקטרוקונטרולרים כמו Arduino. זהו דרך נוחה לקריאת נתונים בקוד ולהשתמש בהם בתוך התוכנית באמצעות שפת תכנות כמו C. קריאת קובץ טקסט עם Arduino מאפשרת לנו להשתמש במידע חיצוני ולפעולות באמצעות קוד.

##איך לעשות זאת
כדי לקרוא קובץ טקסט באמצעות Arduino, ניתן להשתמש בפוֹנְקצִיית "File" בספריית SD. כאשר משתמשים בפוֹנְקצִייה זו, ניתן לפתוח ולקרוא תוכן של קובץ טקסט ולהשתמש במידע שבו בתוך הקוד. למשל:

```
Arduino .................

#include <SPI.h>
#include <SD.h>

File myFile; // יצירת אובייקט קובץ חדש

void setup() {
  // התחברות כרטיס SD ובדיקת מצב תקינות 
  if (!SD.begin(4)) {
    Serial.println("לא ניתן לטעון כרטיס SD");
    while (1);
  }
}
// פתיחת קובץ והדפסת תוכן שלו בקונסול
Serial.println("------קובץ הטקסט הנמצא בכרטיס SD-------");
myFile = SD.open("filename.txt"); 
while (myFile.available()) { // קריאת שורות בקובץ
    Serial.write(myFile.read());
  }
}
```

פנקציית `open()` מאפשרת לנו לבצע פעולות על קובץ טקסט מסוים מכרטיס ה-SD. בכדי לקרוא נתונים מתוך הקובץ, ניתן להשתמש בפונקציות כמו `read()` ו- `available()`.
