# מה ולמה?
קריאת קובץ טקסט היא תהליך שבו המחשב קורא ומציג את התוכן של קובץ טקסט מסוים. תהליך זה מועיל למתכנתים כדי לקרוא ולעבד מידע מקבצים טקסט ולהשתמש בו בתוכניות שלהם.

# איך לעשות זאת?
כדי לקרוא קובץ טקסט עם ארדוינו, יש להשתמש בפקודת ```file = SD.open("fileName")``` כדי לפתוח את הקובץ. ניתן להשתמש בפקודת ```if(file.available())``` כדי לוודא שישנו מידע קיים בקובץ. לאחר מכן, ניתן להשתמש בפקודה ```file.read()``` כדי לקרוא את התו הבא בקובץ ולהציב אותו במשתנה של התוכן שנקרא.

# כיול עמוק
קריאת קובץ טקסט אינה תהליך חדש ונמצא בשימוש מזה שנים. אפשרויות אלטרנטיביות כוללות שימוש בכרטיס זיכרון חיצוני או שליחת המידע למכשיר חיצוני. כאשר מתכנתים משתמשים בקריאת קבצים טקסט, חשוב לוודא שהקובץ נמצא במסלול נכון ושהמידע שנקרא תואם לפורמט המתאים לתוכנית.

# ראו גם
לכיתוב ארדוינו נוסף ולמידע על פקודות נוספות שימושיות, ניתן לבדוק את המדריכים הרשמיים של ארדוינו. כמו כן, ניתן למצוא מגוון של פרויקטים וקודים פתוחים אודות קריאת קבצים טקסט עם ארדוינו ברשת.
