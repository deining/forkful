## למה

קריאת קבצי טקסט היא כלי חשוב בארדואינו עבור מגוון רחב של יישומים, מאחסון נתונים והגדרות עד לתקשורת עם מכשירים חיצוניים. קריאת קובץ טקסט יכולה לסייע ביצירת מערכות חכמות יותר ולהקל על פיתוח תוכניות.

## כיצד לעשות זאת

לפנים, יש לי a קובץ טקסט קטן נקרא "data.txt". יש לו הערות והערכים שונים שנכתבו בצורה שונה. הנה דוגמה לקובץ זה:

```
# זהו קובץ טקסט פשוט עם הערות וערכים
מספר = 5
שם = "דניאל"
תאריך = 2021
```

על מנת לקרוא את הקובץ הזה בארדואינו, ניצור משתנה "File" ונפתח אותו כקובץ טקסט:

```
File dataFile = SD.open("data.txt"); // פתיחת הקובץ
```

לאחר מכן, נבדוק אם הקובץ נפתח בהצלחה על ידי בדיקת הפונקציה "isOpen()":

```
if (dataFile.isOpen()) {
  // קובץ נפתח בהצלחה
} 
```

כעת, נתבגר לקרוא את המידע מהקובץ באמצעות הפונקציה "readString()". נוכל לשמור את כל המידע במשתנה תוך ששומים את כל המידע על מסך הטרמינל על ידי פונקציית "print()":

```
String data = dataFile.readString(); // קריאת המידע מהקובץ
Serial.print(data); // שמירת המידע על מסך הטרמינל 
```

והנה הפלט של המידע המופיע על מסך הטרמינל:

```
# זהו קובץ טקסט פשוט עם הערות וערכים
מספר = 5
שם = "דניאל"
תאריך = 2021
```
