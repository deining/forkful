## למה
בכתיבת תוכניות עבור Arduino, לעיתים קרובות נצטרך לחלץ מחרוזת מתוך מחרוזת ארוכה יותר. לדוגמה, יתכן שנרצה לקרוא מתאמן ממחרוזת שמכילה כמה פרטים שונים כגון שם, גיל וכתובת. במידה כזו, נתקלים בצורך לחלץ תת-מחרוזת של הפרטים הרלוונטיים לכל אישיות. החלפת תאוריות לפעולה, אתרים עם קודי ארדואינו לפעילה עתידית, וכדורים אולפניים עתידיים.

## איך לעשות?
את תהליך החילוץ של התת-מחרוזת ניתן לבצע בכמה שלבים מתחשבים. נתחיל עם הגדרת המחרוזת המקורית ובחירת תוכנה אחת או יותר כדי לחלץ את התת-מחרוזת ממנה. הנה דוגמת קוד הארדואינו המדגים את התהליך:

```Arduino
String originalString = "שם: יום גיבור גיל: 25 ארץ: ישראל";
String name = originalString.substring(5, 15);
String age = originalString.substring(23, 25);
String country = originalString.substring(34, 38);

Serial.println("שם: " + name);
Serial.println("גיל: " + age);
Serial.println("ארץ: " + country);
```

התוצאה המודפסת תהיה:
```
שם: יום גיבור
גיל: 25
ארץ: ישראל
```

בקוד זה, אנחנו משתמשים בפונקציה "substring" של המחרוזת המקורית כדי לבחור את החלק הרלוונטי של המידע ולהכניס אותו למשתנים שונים. ניתן להתאים את המספרים בתוך הסוגריים לפי המיקום הרצוי של התת-מחרוזת במחרוזת המקורית.
