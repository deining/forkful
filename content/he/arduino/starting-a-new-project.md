## למה

אז החלטתם שהגיע הזמן להתחיל פרוייקט חדש עם ארדוינו הגירסה האחרונה? יכול להיות שאתם רוצים לבנות מכשיר אלקטרוני חדש, ללמוד לתכנת בלוקים, או פשוט להשתעשע עם חובבי ארדוינו. בכל מקרה, בכתיבת הפרוייקט הזה נלמד איך להתחיל פרוייקט חדש עם ארדוינו, ללא דיבורים מיותרים.

## איך לעשות זאת

קוד מקור של ארדוינו נראה כך:

```arduino
void setup() {
  // פונקציה זו מופעלת רק פעם אחת כאשר המחשב נדלק.
  // כאן ניתן לכתוב קוד שאותו אנחנו רוצים להריץ רק פעם אחת.

  // כאן ניתן להגדיר את הפורטים שאנחנו משתמשים בהם, כגון פורט כניסה או יציאה.
  pinMode(13, OUTPUT);
}

void loop() {
  // פונקציה זו מופעלת כל הזמן עד שהמעגל נכבה.
  // כאן ניתן לכתוב קוד שאותו אנחנו רוצים להריץ כל הזמן.

  // כאן ניתן לכתוב קוד כדי להפעיל פורט כלשהו.
  digitalWrite(13, HIGH);
  // נחכה כמה שניות.
  delay(1000);
  // נכבה את הפורט.
  digitalWrite(13, LOW);
  // נחכה כמה שניות נוספות.
  delay(1000);
}
```

כאן אנחנו רואים שיש לנו שתי פונקציות עיקריות: `setup` ו-`loop`. כל הקוד המצוי בפונקציית `setup` יופעל רק פעם אחת בזמן שהמחשב מתחיל ואין צורך לסמן שובם. הקוד שכתוב בפונקציית `loop` יופעל כל הזמן עד שהמחשב נכבה או עד שאנחנו עוצרים את התוכנית. כאן אנחנו רואים את הפונקצי
