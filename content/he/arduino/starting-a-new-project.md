## מדוע

למה אדם יחשוב להתחיל פרויקט חדש עם ארדואינו? פרויקט ארדואינו הוא דרך מעולה ללמוד תכנות, ליצור פרויקטים מעניינים ולהתחבר לקהילת תכנות גלובלית. הוא מציע מערכת קלה לשימוש ומשמש ככלי נהדר לפתרון בעיות הייחודיות שלך. אם אתה מחפש דרך חיה לשפר את כישורי התכנות שלך, פרויקט ארדואינו מהווה את הפתרון המושלם עבורך.

## איך להתחיל

בואו נלמד יחד איך להתחיל פרויקט חדש עם ארדואינו. השלבים הבאים יכילו הסברים פשוטים עם קוד דוגמא ותוצאות. *כדי להבטיח תוצאות דומות, נא השתמש במערכת ארדואינו בגרסת 1.8.10.*

1. אתחל את הפרויקט שלך על ידי יצירת קובץ ריק שנקרא "ארדואינו.ino".
2. נכנס לנתונים בקובץ וכתב ```Arduino  void setup() { }```  ליד שורת ההתחלה.
3. לאחר מכן, נוסיף פונקציה נוספת לקוד שנקראת ```void loop() { }```.זו פונקציה עם קוד שיופעל כל פעם שהפינג בפלטפורמה ארדואינו יקלוט ערך או תחום.
4. כעת המכשיר שלנו מוכן לביצוע פעולות! נראה דוגמא פשוטה של הדפסת ערך לקונסולה:

```
void setup() {
    Serial.begin(9600);
}

void loop() {
    Serial.print("שלום עולם!");
    delay(1000);
}
```

תוצאות:

```
שלום עולם!
שלום עולם!
שלום עולם!
...
```
