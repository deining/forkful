## למה

כשאתם מתחילים פרויקט חדש עם ארדוינו, יש לכם הזדמנות ללמוד ולפתח יכולות חדשות ומרתקות. בעזרת התכנות בארדוינו, תוכלו ליצור מכשירים חכמים ומתוכנתים שיכולים לשמש אתכם ואת הסביבה המסביב לכם בצורה מותאמת אישית.

## איך לפתח פרויקט חדש עם ארדוינו

תחילה, יש צורך להתקין את תוכנת הארדוינו על המחשב שלכם. לאחר מכן, תוכלו לכתוב קוד תכנותי בסביבת אנדארדוין ולטעון אותו ליישומון שמחובר לכרטיס המעגלים. הנה דוגמה לקוד בספריה של ארדוינו:

```Arduino
int led = 13;

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  digitalWrite(led, HIGH);
  delay(1000);
  digitalWrite(led, LOW);
  delay(1000);
}
```

בקטע הקוד הזה, אנו מגדירים את הגישה של הבקר כתקליטן ומקבעים את העדכון שלו לכיוון של עלי ולעכב אותו תחילה לשנייה ולאחר מכן ריחוף אותו המעגל בכיוון השני. העתיד תוכלו ליצור פרויקטים מתקדמים יותר ולשלב גם חיישנים וקלטים נוספים למכשיר שלכם.

## מעמקים

כאשר אתם מתחילים פרויקט חדש עם ארדוינו, כדאי להתחיל עם פרויקט פשוט כדי להיכוון ולהתחבר לסביבת התכנות החדשה. ישנם עשרות פרויקטים מקוריים ומגניבים על אתרים רבים כמו Instructables וHackster. שים את ארדוינו ותוכלו לצאת בשיגרה שלכם וליצור מכשירים חדשים ומרתקים.
