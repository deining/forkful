# למה

כיצד תוכלו ליצור מספרים אקראיים באמצעות ארדואינו?

## כיצד לעשות זאת

כדי ליצור מספרים אקראיים באמצעות ארדואינו, תצטרכו להשתמש בפונקציית #random# הזמינה במערכת הפעלה של ארדואינו. תחילה, עליכם להגדיר משתנה שיוצר את המספרים האקראיים על ידי השתמשות בפונקציית random() המקבלת פרמטר שמציין את הגבולות שבין המספר הנמוך ביותר לבין המספר הגבוה ביותר שרציתם.

```Arduino
int randomNumber = random(10, 20); // מספר אקראי בין 10 ל-20
```

לאחר מכן, תוכלו להשתמש במשתנה המכיל את המספר האקראי בכל מקום בקוד התכנית שלכם. לדוגמה, אם תרצו להדפיס את המספר האקראי במחברת של ארדואינו, תוכלו להשתמש בפונקציית #Serial.print# כדי להדפיס את הערך של המשתנה.

```Arduino
Serial.print(randomNumber); // הדפסת המספר האקראי
```

בתוך קוד התכנית שלכם, תוכלו גם להשתמש במספרים אקראיים כדי לגרום לתכנית שלכם לפתוח פעולות אקראיות כמו להפעיל את המנועים או להבהיר את המסך.
