## למה

יצירת מספרים אקראיים היא כלי חשוב בתכנות של ארדוינו. היא מאפשרת לנו ליצור אתגרים חדשים בכל פעם שנפעיל את הקוד והיא יכולה לשמש במגוון רחב של יישומים כמו משחקי מחשב, אוצרות סודיים ועוד. 

## כיצד לעשות זאת

בקוד הבא, אנו משתמשים בפקודות הפנימיות המסומנות בתווים ```random()``` ו```randomSeed()``` כדי ליצור מספרים אקראיים ולהציגם על גבי מסך הסדר הרגיל של ארדוינו. 

```Arduino
void setup() {
    randomSeed(analogRead(0));
    Serial.begin(9600);    
  }
  
void loop() {
    int randomNumber = random(100);
    Serial.println(randomNumber);
    delay(500);
}
```

פקודת ```randomSeed(analogRead(0))``` משתמשת בערך של כניסה אנלוגית כדי להבטיח שהמספרים שיופיעו יהיו אקראיים באמת. פקודת ```random(100)``` מגרילה מספרים אקראיים בין 0 ל-99 ומאפשרת לנו להגדיר טווח מסוים של מספרים. הפקודה ```Serial.println(randomNumber)``` מדפיסה את המספר האקראי על גבי מסך הסדר הרגיל של ארדוינו. תוך שימוש בפקודת ```delay(500)``` אנו מאפשרים פסק זמן של 500 מילישניות (חצי שנייה) בין כל הדפסה של מספר אקראי לבין הבא.

כעת, ברגע שתפעילו את הקוד הזה על ארדוינו, תראו שהמספרים משתנים בכל פעם שהקוד רץ ואין להם שום סדר כללי.
