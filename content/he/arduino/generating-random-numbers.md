## מדוע

כתבה זו תייעץ כיצד ליצור מספרים אקראיים באמצעות קוד Arduino. זה יהיה מועיל לבעלי ארדואינו שמחפשים דרך פשוטה לייצר מספרים אקראיים כדי להשתמש בדוגמאות, משחקים או לצרכי טסט.

## כיצד לעשות זאת

נסו את הקוד הבא על דפן Arduino IDE כדי ליצור מספרים אקראיים.

```arduino
// כותרת ראשית
void setup() {
  // אתחול של דיגיטאל פין 13 כפלט
  pinMode(13, OUTPUT);
  // אתחול של מחולל המספרים האקראיים
  randomSeed(analogRead(0));
}

// מעבר על קוד זה תחזיר מספר אקראי
int getRandomNumber() {
  // שמירת המספר האקראי במשתנה
  int randomNumber = random(1, 10);
  // החזרת המספר האקראי
  return randomNumber;
}

// לולאה ראשית
void loop() {
  // קריאה לפונקציה כדי לקבל מספר אקראי
  int number = getRandomNumber();
  // הדפסת המספר האקראי
  Serial.println(number);
  // שינוי האור הכבד על פין 13
  digitalWrite(13, HIGH);
  delay(100);
  // ביטול האור הכבד על פין 13
  digitalWrite(13, LOW);
  delay(number * 1000);
}
```

הפלט של הקוד הזה יהיה הדפסה של מספרים אקראיים בקצבים שונים וכיבוי והדלקה של אור על פין 13.
