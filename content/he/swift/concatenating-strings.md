# מה ולמה?
הצברת מחרוזות היא פעולה שבה מוחברות שתי מחרוזות יחד ליצירת מחרוזת אחת גדולה יותר. תהליך זה נקרא "הצברה" והוא נהוג בתכנות כדי ליצור מחרוזות מורכבות יותר ממחרוזות קטנות, כמו לדוגמה להציג טקסט מפורט במספר שורות שונות על ידי הצברה שלהם יחד.

# איך לעשות? 
לעיתים קרובות, ניתן לצבור מחרוזות בעזרת הסימן "+" כדי להוסיף מחרוזות אחת אל גובה מחרוזת אחרת, ככה: 

```Swift
var greeting = "שלום"
var name = "איילת"
var personalizedGreeting = greeting + " " + name // אחרי בצביעה נראה "שלום איילת"
```

# עיון מעמיק
אינכם חייבים להשתמש בהצברות כדי ליצור מחרוזות מורכבות. ישנן גם פיתרונות נוספים כגון השתמשות בפונקציות כמו `append()` או `join()`. אנחנו ממליצים להשתמש בהצברות כאשר יש צורך בצריכת מרחב של מקום זיכרון, במקרים אלו יש להימנע מהשימוש בהצברות.
