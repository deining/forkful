## למה

למה מישהו יחליט ליצור מחרוזות מצורפות?
כאשר אנו מפצלים את הטקסט למספר מחרוזות קטנות יותר, אנו מקלים על עיבוד הנתונים ויוצרים קוד יותר לגישה ולקריאה. הכל בשביל לפשט את הפעולות שלנו ולהסתמך על הטכנולוגיות המתקדמות של קוד Swift.

## איך לעשות זאת

בואו נתחיל עם דוגמא פשוטה ליצירת מחרוזות מצורפות בשפת סוויפט:

```Swift
let firstWord = "שלום"
let secondWord = "עולם"
let greeting = firstWord + secondWord
print(greeting) // ידפיס "שלוםעולם"
```

בדוגמה זו, אנו מפצלים את המחרוזת לשני חלקים קטנים ומצורפים אותם ליצירת מחרוזת חדשה. ניתן להשתמש גם באופרטור `+=` כדי להוסיף את התוכן של מחרוזת אחת לצד השני:

```Swift
var greeting = "שלום"
greeting += "עולם"
print(greeting) // ידפיס "שלוםעולם"
```

אנו יכולים גם להשתמש בתווים מיוחדים כדי לצרוך מחרוזות מצורפות ולהפריד ביניהם. נראה דוגמא נוספת:

```Swift
let firstSentence = "מה שלומך?"
let secondSentence = "אני מצטער, איני מבין."
let dialogue = firstSentence + "\\n" + secondSentence
print(dialogue)
// ידפיס:
// מה שלומך?
// אני מצטער, איני מבין.
```

השתמשנו באופרטור `\n` כדי לצורך ירידת שורה חדשה במחרוזת המצורפת שלנו.

## לעומק

מתחת למעט הדוגמאות הפשוטות שלנו, יש כמה דברים נוספים שצריך לדעת על מחרוזות מצורפות.
