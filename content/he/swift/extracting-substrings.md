# למה
מתוך הצורך לעבוד עם מחרוזות בצורה יעילה, אנחנו רוצים ללמוד כיצד לחלץ תתי מחרוזות. זה יכול להיות שימושי במגוון מקרים, כגון תיאום עם אינדקסים או חלוקה למרכזים תתיים.

## איך לעשות זאת
נתחיל עם מחרוזת פשוטה ונחלץ תת חלקים על ידי ציון נקודת התחלה וסוף. ישנם מספר דרכים לעשות זאת, לפי הצורך והנוחות שלכם. לדוגמה:

```Swift
let str = "היי חברים!"

// חלוץ תת מחרוזת מהאינדקס 2 ועד הסוף
let substring1 = str[str.index(str.startIndex, offsetBy: 2)...]

print(substring1) 
// תוכלו לראות שהפלט יהיה "חברים!"
```

כמו כן, אפשר לאחד מחרוזות כדי ליצור מחרוזת חדשה עם עזרת הפונקציה `joined()`:

```Swift
let str1 = "היי"
let str2 = "חברים!"

let newString = str1 + " " + str2

print(newString)
// הפלט יהיה "היי חברים!"
```

אם אתם מעוניינים לחלץ תת מחרוזת לפי אינדקס מסוים, ניתן לעשות זאת באמצעות ממשק ה- `StringProtocol`:

```Swift
let str = "שלום לכם"

// חלוץ תחילת המחרוזת עד לאינדקס מסוים
let substring2 = str.prefix(4)

print(substring2)
// הפלט יהיה "שלום"
```
