# למה

אנשים משתמשים בתאריך נוכחי בתכנות ב-Swift כדי לעזור להם ליצור יישומים יעילים ומדויקים. התאריך הנוכחי משמש כמדד חשוב ביותר לתאריך ושעה ויכול לשמש ליישום פעולות שונות בתוך היישום שלך.

# איך לעשות זאת

כדי לקבל את התאריך הנוכחי בשפת תכנות Swift, ניתן להשתמש במחלקת Date.

```
let date = Date()
print(date)
```

פלט:

```
2021-05-22 14:53:36 +0000
```

בדוגמה זו, אנו משתמשים במחלקת Date כדי ליצור משתנה אחד שמכיל את התאריך הנוכחי. לאחר מכן, אנו משתמשים בפקודת הדפסה כדי להדפיס את התאריך הנוכחי.

אתה יכול גם לציין את הפורמט של התאריך הנוכחי באמצעות המחלקה DateFormatter.

```
let dateFormatter = DateFormatter()
dateFormatter.dateFormat = "dd/MM/yyyy"
let date = Date()
let dateString = dateFormatter.string(from: date)
print(dateString)
```

פלט:

```
22/05/2021
```

בדוגמה זו, אנו משתמשים במחלקה DateFormatter כדי להגדיר פורמט חדש לתאריך הנוכחי. לאחר מכן, אנו משתמשים בפקודה string כדי להמיר את התאריך למחרוזת.
