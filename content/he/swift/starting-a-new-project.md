## למה

טרום ראשית לערוך כל פרויקט חדש של שפת Swift, עלינו לחשוב על מחשבה משותפת כיצד הפרויקט הזה יעזור וישפר את הקוד והפתרונות עבור שפת ברמה הבינלאומית הידועה במערכות הפעלה של אפל, iOS ו mac OS.

## איך לעשות זאת

כאשר אנו מתחילים לכתוב את הפרויקט החדש שלנו, חשוב להיות מתוחכמים ולהשתמש בפונקציות חזקות של שפת Swift. הנה כמה דוגמאות של קוד מסוג ```Swift```:

1. יצירת מערך חכם:

```Swift
let arr = [3, 10, 2, 6, 1, 8]
```

2. זימון של משתנה מהמערך לפי אינדקס:

```Swift
let num = arr[2] // num = 2
```

3. שימוש בלולאת for כדי לעבור על כל אלמנטים במערך:

```Swift
for i in arr {
    print(i)
}

// Output:
// 3
// 10
// 2
// 6
// 1
// 8
```

4. שימוש בפונקציות משולבות כגון ```map``` ו- ```filter``` כדי לעבוד עם מערכים בצורה מתקדמת יותר:

```Swift
let squareArr = arr.map({ $0 * $0 }) // squareArr = [9, 100, 4, 36, 1, 64]

let evenArr = arr.filter({ $0 % 2 == 0 }) // evenArr = [10, 2, 6, 8]
```

עוד דגמאות קוד ופתרונות מתקדמים ניתן למצוא באתר הרשמי של שפת Swift.

## יעודים עמוקים

בתחילת כתיבת פרויקט חדש, ניתן להיות קצת מתבלבלים ולא יודעים מאיפה להתחיל. נהוג להתחיל עם כמה קבצים בסיסיים כגון ```main.swift``` ו- ```ViewController.swift``` ולהתחיל לבנות עליהם. כמו כן, חשוב להיזהר מכך שמסומנים פרויקט יחדיו ע"י Nimble, כדי לקבל את מלאכותיות הראויה שנותן לנו Swift.
