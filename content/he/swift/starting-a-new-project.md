## למה
אולי את/ה שואל/ת את עצמך למה את/ה אמור/ה להתעסק בפתיחת פרויקט חדש? התשובה פשוטה - כי זה כיף! כתיבת קוד בשפת Swift היא חוויה מהנה ומאתגרת, ולבנות משהו מהפרויקט הראשון שלך יכול לתת לך גאווה עצומה. כמו כן, יצירת פרויקט חדש יכול להיות דרך נהדרת ליצור יכולות חדשות ולהתנסות בכדי לשפר את הכישורים שלך כמתכנת.

## איך לעשות זאת
השתמשו בדגמים הבאים כדי להתחיל וליצור פרויקט חדש בשפת Swift:

```Swift
// יצירת משתנה סטרינג
let hello = "שלום!"

// הדפסת המשתנה
print(hello)

// פעולת כפל מספרים
let num1 = 5
let num2 = 10
let result = num1 * num2
print(result)
```

פלט:

```
שלום!
50
```

בדוגמאות העליונות, אנו מדגימים איך ליצור משתנים ולהדפיס תוכן על המסך, וכן איך לבצע פעולות חשבוניות פשוטות. כעת, עם היצירה של פרויקט חדש, ניתן להתחיל להתנסות בכתיבת קוד נוסף על מנת ליישם יכולות נוספות ומתקדמות.

## העמקה
כאשר את/ה מתחיל/ה פרויקט חדש בשפת Swift, חשוב להיות ארגונים ומתוכננים. הנה כמה טיפים להתחיל בצורה נכונה:

- קבעו מטרות מדויקות ופרויקטים ספציפיים שאת/ה רוצה ליצור.
- תכננו את מבנה הקבצים עם פרויקט מקורי, מודעון טזט, ותיקיות.
- בחרו תצורת פרויקט אידאלית עבור חלונות ומערכות הפעלה האחרות שאת/ה מתכנן
