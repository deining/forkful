מה זה ולמה?

"מזדהה במחרוזת" הוא פעולה שמאפשרת למתכנתים להכניס משתנים או ביטויים מתמטיים לתוך מחרוזת באופן דינמי. זה מאפשר לנו ליצור מחרוזות מותאמות אישית ולהפוך את הקוד שלנו ליותר קריא ונקי. מתכנתים שואפים לבצע זאת כדי להפוך את הקוד שלהם למשתנה וקל לתחזוק.

איך לעשות:

לפניכם נמצא דוגמאות של קוד בשפת Swift עם החלקים הדינמיים ממחרוזת מזוהה ופלט מצורף עם התוצאה מהרצת הקוד. תוכלו להשתמש בקוד הבא כדי להעתיק ולהקליד את הדוגמה הרלוונטית.

```Swift
let name = "נעמה"
let age = 25
let message = "שלום, שמי הוא \(name) ואני בן \(age) שנים."
print(message)
```

פלט:
```
‫שלום, שמי הוא נעמה ואני בן 25 שנים.
```

לחלופין, תוכלו להשתמש בתו $ לפני פרמטרים במחרוזת במקום סוגריים. לדוגמה:

```Swift
let name = "נעמה"
let age = 25
let message = "שלום, שמי הוא $name ואני בן $age שנים."
print(message)
```

פלט:
```
‫שלום, שמי הוא נעמה ואני בן 25 שנים.
```

להמציא מחרוזת מזוהה, פשוט הוסיפו את הסימן \ לפני הסוגריים עם המשתנים. לדוגמה:

```Swift
let name = "נעמה"
let message = "שלום, שמי הוא \(name)."
print(message)
```

פלט:
```
‫שלום, שמי הוא נעמה.
```

לעומת זאת, אם תשתמשו בסימן " לפני הסוגריים, המשתנה לא יחולף. לדוגמה:

```Swift
let name = "נעמה"
let message = "שלום, שמי הוא \"\(name).\""
print(message)
```

פלט:
```
‫‫שלום, שמי הוא "נעמה."
```

עודכנות יותר, אם תשתמשו בסימן # לפני המשתנים, הם יוחלפו בקודים HTML. לדוגמה:

```Swift
let name = "נעמה"
let message = "שלום, שמי הוא #\(name)."
print(message)
```

פלט:
```
<p>שלום, שמי הוא נעמה.</p>
```

חפשו את קוד המחרוזת מזוהה בתוך קוד קשור? ניתן להתעלם מניקוד בחלק הכתיבה של מחרוזות. לדוגמה:

```Swift
let find = "חיפשתי"
let paragraph = "אני חיפשתי חיפשתי מחרוזת,  drove."
if paragraph.contains(find) {
    print("מחרוזת מזוהה נמצא.")
}
```

פלט:
```
מחרוזת מזוהה נמצא.
```
