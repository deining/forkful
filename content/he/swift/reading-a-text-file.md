## למה

קריאת קובץ טקסט נחשבת כפעולה בסיסית בתכנות בכל שפת תכנות, וקריאת קוד בשפת Swift במיוחד. כאשר ניתן לקרוא קבצי טקסט, ניתן לבצע פעולות רבות עם הנתונים המכילים בתוכם. לדוגמה, עיבוד נתונים והצגתם בצורה מובנת.

## איך לבצע קריאת קובץ טקסט בשפת Swift

הנה דוגמא קצרה בשפת Swift המדגימה כיצד לקרוא קובץ טקסט ולהציג את תוכנו על מסך הטרמינל:

```Swift
// מיקום הקובץ הנמצא באותו תיקיה כמו הקובץ הנוכחי
// אם הקובץ נמצא בתיקייה אחרת, ניתן לכתוב את הכתובת המלאה של הקובץ
let path = "textFile.txt"

do {
    // קריאת תוכן הקובץ באמצעות המתודה "String" המקבלת כפרמטר את הנתיב של הקובץ
    let content = try String(contentsOfFile: path)
    print(content) // הדפסת התוכן לטרמינל
} catch {
    print(error) // הדפסת שגיאת קריאת הקובץ במקרה של כישלון
}
```

### פלט התוכן מתוך הקובץ

```
זהו טקסט אחר. הכיתוב הזה יופיע כאשר נתבצע קריאה של הקובץ.
```
