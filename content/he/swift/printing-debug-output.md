## למה

תמיד נתקלים בבעיות ושגיאות בקוד שלנו כשאנחנו מפתחים אפליקציות בסוויפט. למרבה המזל, יש לנו כלי יעיל כדי לעזור לנו לתקן ולמצוא את השגיאות האלה - בהדפסת פלט לאיתור וניתוח השגיאות. במאמר זה נלמד כיצד להשתמש בהדפסת דיבאג כדי לשפר את תהליך הפיתוח שלנו.

## איך לעשות

השתמשו בפונקציית "print" בשפת סוויפט כדי להדפיס מידע לסיוע בניתוח ותיקון שגיאות בקוד שלכם. להלן מספר דוגמאות:

```Swift
// הדפסת משתנה מספרי
let number = 25
print("מספר: \(number)")

// הדפסת מחרוזת
let string = "זהו מחרוזת"
print(string)

// הדפסת נתונים ממערך
let array = ["אפל", "בננה", "תות"]
print("המילה האחרונה במערך היא: \(array.last!)")
```

פלט ההדפסות של קטע הקוד הנ"ל יהיה:

```
מספר: 25
זהו מחרוזת
המילה האחרונה במערך היא: תות
```

ניתן להשתמש גם בפונקציה "dump" כדי להדפיס מידע נוסף על אובייקטים ומערכים כדי לעזור לכם למעקב אחרי ערכי המשתנים בזמן ריצת התכנית שלכם.

```Swift
// הדפסת אובייקט מסוג מחרוזת
let string = "זהו מחרוזת"
dump(string)

// הדפסת אובייקט מסוג מערך
let array = ["אפל", "בננה", "תות"]
dump(array)

// הדפסת אובייקט מסוג מחרוזת ואובייקט מסוג מספרי
let string = "זהו מחרוזת"
let number = 25
dump(string, number)
```

פלט ההדפסות של קטע הקוד הנ"ל יהיה:

```
▿ "זהו מחרוזת"
  - ...
▿ 0 : "אפל"
▿ 1 : "בננה"
▿ 2 : "תות
