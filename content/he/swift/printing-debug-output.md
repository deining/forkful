## למה:

למה אנשים מדפיסים פלט דיבאג? כי זה כלי שימושי וחיוני בפיתוח בשפת סוויפט. זה מאפשר למפתחים לבדוק את ריצת הקוד ולזהות באילו נקודות וגם כיצד התכנים עובדים כאשר נתקלים בבעיות וטעויות.

## איך לעשות:

במסגרת החקר שלנו נלמד כיצד להדפיס פלט דיבאג בשפת סוויפט. כדי להדגיש את רלבנטיות הדיבאגינג, נראה תיאורים של הפלט שמשמשים לאספקות של הבעיה:

```Swift
// קוד פריימר קטן בתוך הפונקציה שלי
func addNumbers(_ a: Int, _ b: Int) -> Int {
    print("הערך של a הוא: \(a)")
    print("הערך של b הוא: \(b)")
    let result = a + b
    print("התוצאה היא: \(result)")
    return result
}

// הפלט של הקוד הנ"ל יהיה:
// הערך של a הוא: 2
// הערך של b הוא: 3
// התוצאה היא: 5
```

כאן, אנו מגיעים למבנה הדיבאג מושלם: הרכיב הראשון מדפיס את הערך והמשתנה "a", בזמן שהרכיב הזהה לב מדפיס את הערך שמוחזק בפרמטר השני "b". הנרא עד שכאן אנו יכולים לדון ברמות המשמעותיות של הערך המוחזק של כל משתנה ואנו מבינים שקיימים כמה מכוונים להדפסות הפלט כאופציות לאימות של הקוד שעובדיםההודפתים זה על ידי הכרעות קבועות בשקם נקודות הבעיה.
