## למה

מתי שמצטרכים להמיר מחרוזת לאותיות קטנות?

המרת מחרוזת לאותיות קטנות יכולה להיות שימושית במספר מקרים, בין היתר:

- כאשר מעוניינים להשוות מחרוזות, כמו בתנאי if
- כאשר מעוניינים שכל הטקסט יופיע באותיות קטנות, למשל כאשר מציגים טקסט למשתמש בכתב ידוע ונקראות באותיות קטנות
- כאשר מעוניינים לפרוס טקסט לחלקים יותר קלים לעבירה ועיבוד, כמו במצב של אינטרנט

## איך לבצע את ההמרה?

ההמרה פשוטה מאוד על ידי שימוש בפונקציה `lowercased()`. נסו את הקוד הבא בתוך `Swift ...` כדי לראות את התוצאות המדויקות:

```Swift
let string = "זהו טקסט עם אותיות גדולות"
print(string.lowercased())
// תוצאה: זהו טקסט עם אותיות קטנות
```

על כדי להשוות מחרוזות בלי התחשבות באותיות, ניתן להשתמש בפונקציה `caseInsensitiveCompare()`:

```Swift
let string1 = "טקסט"
let string2 = "טקסט"
if string1.caseInsensitiveCompare(string2) == .orderedSame {
    print("המחרוזות זהות!")
}
// תוצאה: המחרוזות זהות!
```

## מעמקים

בזמן שניתן להשתמש בשיטת חלופיים להמרה כמו שימוש במתודת `lowercased()`, ישנם מקרים שבהם נחשבת מהירות ואמינות האפשרות שלנו. למשל, אם מכריחים מחרוזת להכריח את כל האותיות להיות בקצה ארוכים כגון "ß", תוצאת ההמרה עשויה להיות לא מדויקת. כדי למנוע זאת, ניתן להשתמש במתודות יותר מפורטות כמו `folding(options:locale)`.
