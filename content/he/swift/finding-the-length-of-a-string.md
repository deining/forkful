# למה

למה עשויים חוברי קוד לחפש את אורך המחרוזת? המחרוזת היא סוג של משתנה שמכיל טקסט, ולא פעמים רבות נצטרך לדעת את אורכה. לדוגמה, אם אנחנו מנסים להציג טקסט על מסך המחשב, אנחנו צריכים לדעת את אורך הטקסט כדי להתאים את המסך למידות הטקסט. לכן, ידע על אורך המחרוזת הוא חשוב עבורם חוברי קוד שרוצים ליצור אפליקציות מתקדמות.

# איך לעשות

תחילה, נגדיר משתנה מסוג String ונשים בתוכו טקסט כלשהו. לדוגמה: 

```Swift
let myString = "שלום עולם"
```

כעת, נשתמש בפונקציה `count` כדי לקבל את אורך המחרוזת שבתוך המשתנה שלנו. השתמשו בשורת קוד הבאה:

```Swift
print(myString.count) 
```

הפלט שנקבל יהיה 9, כיוון שיש לנו 9 תווים במחרוזת "שלום עולם". 

ניתן לקחת צעד נוסף וליצור פונקציה עצמאית שתחשב את אורך המחרוזת עבורנו. לדוגמה:

```Swift
func stringLength(_ string: String) -> Int {
    var count = 0
    for _ in string {
        count += 1
    }
    return count
}

print(stringLength(myString))
```

פלא, נקבל שוב את הפלט 9. כמו הפונקציה `count`, פונקציה זו תכתוב את הערך של `count` שנמצא בתוך הלולאה עבור כל תו במחרוזת.
