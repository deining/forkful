## למה

בתכנות Swift נתקלים לעיתים קרובות במצבים שבהם נדרש למצוא את אורך המחרוזת המורכבת ממספר תווים. מציאת אורך המחרוזת הינו פעולה חיונית שמאפשרת לנו לעבוד במדויקות עם הנתונים וליישם פתרונות תכנותיים מדויקים.

## איך לעשות זאת

מציאת אורך המחרוזת בשפת Swift הינו פשוט וקל, וניתן לבצע זאת בעזרת פונקציה מובנית ראשית בשם `count`. כדי להשתמש בפונקציה זו, יש להעביר את המחרוזת כפרמטר ולקבל כתוצאה את אורך המחרוזת ביחידות תווים. לדוגמה:

```Swift
let myString = "זהו דוגמה למחרוזת"
let length = myString.count
print(length) // תוצאה ייצוג האורך של המחרוזת - 21
```

בדוגמה זו, באמצעות המשתנה `myString` אנו מגדירים מחרוזת עם ערך מסוים ובאמצעות הפונקציה `count` אנו מוצאים את אורך המחרוזת. לאחר מכן, בעזרת הפונקציה `print` אנו מדפיסים את הערך של משתנה `length` שמכיל את תוצאת הפעולה.

בנוסף, ניתן גם להשתמש בפונקציה `count` בתוך מחרוזת מקוננת כדי להגדיר את האורך של המחרוזת בצורה ישירה:

```Swift
let myNestedString = "המחרוזת הראשונה: \(myString), והאורך שלה הוא: \(myString.count)"
print(myNestedString) // תוצאה תיוגדת המחרוזת הנמצאת ביחידות תווים
```
