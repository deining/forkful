# מה ולמה?
מציאת אורך של מחרוזת היא התהליך של חישוב מספר התווים שיש במחרוזת. פעולה זו נחשבת לבסיסית ונפוצה בתכנות, כך שמומלץ להיות מכירים אתה ע"מ להשתמש בה בצורה מהירה ויעילה.

## איך לעשות זאת:
תהליך זה הוא פשוט וקל לביצוע בשפת סוויפט. כדי למצוא את האורך של מחרוזת, פשוט נקדות את המחרוזת ומשתמשים בפקודה `count()` לחישוב התווים. לדוגמה:

```Swift
let str = "מחרוזת לדוגמה"
print(str.count) // פלט: 12
```

ניתן גם להשתמש בפקודה `characters.count` כדי לקבל את כמות התווים במחרוזת, אבל פקודה זו נספרת כבר בגרסה החדשה של סוויפט ולכן נמליץ להשתמש ב-`count()` לצורך תאימות עם גירסאות קודמות.

## טיפול עמוק:
בעבר, בשפת C++, כדי למצוא את אורך המחרוזת ניתן היה להשתמש בפקודה `strlen()`. אם אתה מכיר את הפקודה הזו, ותרגיש עם הפקודה `count()`, תראה שהן דומות מאוד. אם תחשוב על כך, אורך של מחרוזת זהה למספר האיברים במערך, כאשר כל איבר מייצג תו אחד במחרוזת.

אם נרצה לקבל את אורך מחרוזת בשפות אחרות כגון Python או Java, ניתן להשתמש בפקודות שונות שיש תמיכה בתוכניות שפה אלו. אם אתה מכיר את השפה, תוכל למצוא מידע על הפקודות המתאימות במקומות אחרים.
