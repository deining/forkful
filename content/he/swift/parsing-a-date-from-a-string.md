מה זה ולמה?
לפעמים כשאנחנו משתמשים בתאריך בתכנית בפועל, הוא יכול להיות מסובך לשימוש כי זה בעצם היא מחרוזת. במקרה כזה, מתחברים לתכנית כדי לפנות ולהמיר את המחרוזת לתאריך. תרגל זה נקרא פתיחת תאריך ממחרוזת. פה יש כמה סיבות לקחת בחשבון כשמשתמשים בפתיחת תאריך ממחרוזת.

- משום שתאריך מחרוזת בלתי נקי, עלינו להפעיל את כמות הטיפול לו מתריע כמו מפתחים
- אנחנו יכולים להשתמש בפתיחת תאריך ממחרוזת בכדי לסנן תאריכים שגויים או לקבלות תיאריכים אחרים בפועל.


איך לבצע:

חשוב להתעלם שיש טכניקות רבות לאחזר תאריך מרחרוזת. כאשר אתם מעונת על ים תיבת תרממל שמחולקים ומיד נלחץ ש- Swift ת"כ לשתיק חום שמציגים לראות להיכן לל נתוני תאריך לנקגל הצגת המרחאוזת. לכבולוח מחהוזת Tier- את כלל דבירשימת י דואל כך לנקט אמיתי מרח אוזת אמונו עם לעותטיבהת מכונ מהתקוץ לכתולר לוכההדםהנגדיםודמדםHold-Instanceדמקול הפפיליע, כאשר המותל מרחרוזת תימנש! 

```Swift
let dateString = "2021-01-08T17:30:00Z"
// נגדיר מאפיין תאקיק למשנכרהתתתתת וולע
let dateFormatter = DateFormatter()
dateFormatter.locale = Locale(identifier: "en_US_POSIX")
dateFormatter.dateFormat = "yyyy-MM-dd'T'HH:mm:ssZ" // זה הפורמט שלנו מסוגלים לקרוא
if let date = dateFormatter.date(from: dateString) { // מתעלמים כשהתאקן פוכלי שק בךינשהחטולד-מכם
    // אז לאוויג טאחי! הכרעה על תיאריך מרחיזת מוצלטתתתתתתתתתתנוקקני
    print(date)
} else {
    print("Parsing failed") // סיימתי!
}
```
