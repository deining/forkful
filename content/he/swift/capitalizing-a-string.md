## למה 

בא מהפשוט ביותר שיצטרך לשנות טקסט במחרוזת ולהכניס אות הראשונה לאות גדולה. שינוי זה יכול להיות שימושי כאשר מדובר בכותרות או כשרוצים להדגיש מילה מסוימת.

## איך לעשות זאת

אחת הדרכים הנפוצות לכתוב קוד כדי לשנות מחרוזת ולשנות את האות הראשונה לאות גדולה היא להשתמש בפונקציה "capitalized" של לועזית. בישורת הקוד הבאה, נשתמש במילת מפתח "let" כדי ליצור מחרוזת חדשה ולהכניס אות הראשונה לאותיות גדולות:

```Swift
let string = "hello world"
let newString = string.capitalized
print(newString)
```

התוצאה בקובץ המחושב תהיה "Hello World".

כדי לשנות את האות הראשונה לאותיות גדולות רק של מילה מסוימת, נשתמש בפונקציה " replacingOccurrences" כדי להחליף את המחרוזת המקורית עם החלק הנדרש במחרוזת חדשה. למשל:

```Swift
let string = "swift programming"
let editedString = string.replacingOccurrences(of: "swift", with: "Swift")
print(editedString)
```

הפלט יהיה "Swift programming" כאשר רק המילה "swift" תוחלף לאות גדולה.

## העמקה

בתהליך של שינוי מחרוזת ושימוש בפונקציות כדי להכניס אות ראשונה לאותיות גדולות, חשוב לזכור כי מחרוזת הנתונה תישמר בתוך לועזית ללא שינויים. כמו כן, יש לציין שהפונקציות אלו נעימות בתוך מחרוזת ולא על כותרות שבורות. כדי להפעיל את הפונקציות על כותרות, עלינו לממש אחרת כמו להכניס אותיות פרטניות לאותיות גדולות במקום למשתמש בפונקציות.
