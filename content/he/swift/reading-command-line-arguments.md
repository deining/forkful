## למה
קריאה של ארגומנטי שורת הפקודה היא כלי חיוני בכתיבת קוד בשפת Swift ומאפשרת לנו לתאם את התוכנית שלנו למקרים מיוחדים או להעביר פרמטרים שונים בעת הרצת התוכנית. לכן, כדאי ללמוד כיצד לקרוא ארגומנטי שורת הפקודה כדי להיות מוכנים לכתוב קודים ברמה גבוהה יותר ולעבוד בצורה יעילה יותר.

## כיצד לעשות זאת
ניתן לקרוא את ארגומנטי שורת הפקודה באמצעות פונקציית `CommandLine.arguments` בתוך מחלקת `CommandLine`. כדי להדפיס את כל ארגומנטי הפקודה שהועברו לתוכנית, ניתן להשתמש בלולאת `for` ולהדפיס כל ארגומנט בנפרד. לדוגמה:

```Swift
for argument in CommandLine.arguments {
    print(argument)
}
```
כאשר מריצים את הקוד הזה, נקבל כפלט את רשימת הארגומנטים שהועברו לתוכנית, כולל את שם התוכנית עצמה.

## הנכנס לעומק
כאשר מדובר על ארגומנטי שורת הפקודה, כדאי לזכור שמדובר במחרוזת ולכן ייתכן שנדרש לבצע המרה לסוג נתונים שונים. כמו כן, כדאי לטפל במקרים קצה כמו כאשר לא מועברים ארגומנטים כלל או כאשר מתקבל ארגומנט שאינו תואם לפורמט הצפוי. בקיצור, כדאי לקרוא מידע מפורט יותר על הנושא כדי להיות מוכנים לכל התרחישים האפשריים.
