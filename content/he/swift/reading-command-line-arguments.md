# למה

כשמתחילים ללמוד לתכנת בשפת Swift, ייתכן שנתקלו במונח "ארגומנטי קו־פקודה". זהו נושא חשוב ללימוד, שיכול להקל על תהליך כתיבת קוד ולהוביל לרכישת כישורים נוספים. במאמר הזה נבין מדוע חשוב ללמוד איך לקרוא ארגומנטי קו־פקודה ונראה דוגמאות קוד בשפת Swift כדי ללמוד איך לעשות זאת.

# איך לקרוא ארגומנטי קו־פקודה

```Swift
// הגדרת משתנה מטיפוס String שמכיל את הארגומנט הראשון של קו־פקודה
let arg1 = CommandLine.arguments[1]
// הדפסת כמה ארגומנטים נמצאים בקו־פקודה
print("מס' הארגומנטים המועברים בקו־פקודה: \(CommandLine.arguments.count)")
// הדפסת כל הארגומנטים שנומצאים בקו־פקודה
for argument in CommandLine.arguments {
    print(argument)
}
```

הנה דוגמאות לכמה פלטים אפשריים עבור הקוד הנ"ל:

פלט:
```
מס' הארגומנטים המועברים בקו־פקודה: 4
/usr/bin/swift
/Users/username/Desktop/file.swift
arg1
arg2
```

### פלט: 
```
מס' הארגומנטים המועברים בקו־פקודה: 2
/Users/username/Desktop/file.swift
arg1
```

כפי שאתם יכולים לראות, הארגומנט הראשון בקו־פקודה הוא שם התוכנית עצמה, והארגומנטים הבאים הם הפרמטרים שמועברים לתוכנית.
