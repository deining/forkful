## למה

אנשים יכולים לקרוא קובץ טקסט מסיבות רבות, כולל עיבוד נתונים, כתיבת תוכניות ומציאת מידע חשוב תוך כדי פיתוח תוכניות בפייתון. למידע נוסף על כיצד לקרוא קבצי טקסט בפייתון, תיכנסו לטקסט הבא.

## כיצד לקרוא קובץ טקסט בפייתון

כדי לקרוא קובץ טקסט בפייתון, נוכל להשתמש בפונקציה `open()` ולציין את שם הקובץ ואת המצב הרצוי לפתיחה שלו. למשל, אם נרצה לקרוא קובץ טקסט במצב קריאה בלבד, נשתמש במצב `r`:

```Python
file = open("example.txt", "r")
```

כעת, כאשר הקובץ נפתח, נוכל לקרוא את התוכן שלו באמצעות פונקציה `read()`:

```Python
content = file.read()
print(content)
```

לאחר מכן, נסגור את הקובץ על ידי השימוש בפונקציה `close()`:

```Python
file.close()
```

עם זאת, אם נרצה לקרוא קובץ טקסט במצב כתיבה, נשתמש במצב `w`, ואם נרצה להוסיף תוכן לקובץ קיים נשתמש במצב `a`. הנה דוגמא לכתיבת תוכן חדש לקובץ טקסט:

```Python
file = open("new_file.txt", "w")
file.write("This is a new text file.")
file.close()
```

וכאן דוגמא להוספת תוכן לקובץ קיים:

```Python
file = open("existing_file.txt", "a")
file.write("This is an added text.")
file.close()
```
