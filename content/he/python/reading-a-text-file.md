## למה
ייתכן שמותר לכתוב משפט כאן שגורם לי לבחור בבלוג למידע אחר מסוג זה.
אנשים רבים מתחילים ללמוד פייתון משום שהוא שפת תכנות פופולרית מאוד, נוחה למתחילים ללמוד וכאשר מלמדים אותה, הם לומדים את היכולות הבסיסיות של שפת התכנות. אחת היכולות שהם לומדים היא איך לקרוא קבצי טקסט ולעבוד איתם. זה יכול להיות מועיל בכל סוגי הפרויקטים הקטנים והגדולים.

## איך לעשות זאת
בשביל לקרוא קובץ טקסט עם פייתון, אתם צריכים לנפח בפייתון את הפרויקט שלנו.
כדי לעשות זאת, אנו נשתמש בפלטפורמה פייתון 3, כי זו הפלטפורמה המומלצת כרגע. אנו נשתמש גם בפעולת "פתח את", כדי לפתוח קובץ טקסט. כאן יש קוד בסיסי כדי לספק לכם התחלה בסיסית עם קריאה.

```Python
# ייבוא הפונקציונליות הנדרשת
import codecs

# פתח את הקובץ עם קודקס UTF-8 UTF-8
with codecs.open("שם הקובץ", "r", "utf-8") as file:
    # קרא את הקובץ בעזרת פעולת הקריאה
    text = file.read()
    
    # תדפיס את הטקסט
    print(text)
```

פעולת הקריאה תכניס את כל הקובץ טקסט לתוך משתנה טקסט וכאן נדפיס אותו. כעת נזכיר כי פייתון מסוגל בקלות לקרוא קבצי טקסט ולשלמות עם הטקסטים או לעבוד עם מידע מסוים בקובץ.
