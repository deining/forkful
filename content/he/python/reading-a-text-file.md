## למה
ישנם המון דרכים לקרוא ולהתמודד עם קבצי טקסט בתוך פייתון. אתה יכול למצוא בקלות מידע על כיצד לעשות זאת באינטרנט, אך למה בכלל לקרוא קובץ טקסט עם פייתון? המטרה של הכתבה הזאת היא להראות לך כמה נושאים מעניינים ומגוונים שאתה יכול לעשות עם קבצי טקסט בפייתון.

## איך לעשות זאת
```Python
# פתיחת קובץ טקסט בפרוטוקול התרשימים
with open('file.txt', 'r') as f:
    # אינדקסציה של כל שורה בקובץ
    for line in f:
        # הדפסת כל שורה
        print(line)
```

```Python
# קריאת כל הקובץ כטקסט אחד
with open('file.txt', 'r') as f:
    # השמת כל התוכן במשתנה
    text = f.read()
    # הדפסת כל הטקסט
    print(text)
```

```Python
# בדיקה אם מספר מסויים נמצא בקובץ טקסט
number = 5
with open('file.txt', 'r') as f:
    # השמת כל התוכן ברשימה
    lines = f.readlines()
    # בדיקה אם המספר נמצא ברשימה
    if str(number) in lines:
        # הדפסת המספר ונוסף מאחריו
        print(str(number) + " נמצא בקובץ")
```

### הערות:
- ניתן להשתמש ב-"with" כדי לאפשר סגירת אוטומטית של הקובץ לאחר השימוש בו.
- ניתן להשתמש בפונקציית "readlines()" כדי לקרוא את התוכן של הקובץ כרשימה של שורות.
