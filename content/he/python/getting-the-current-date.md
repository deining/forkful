## למה

לדעת מה התאריך הנוכחי חשוב לכל מתכנת פייתון. זה יכול להיות שימושי ביצירת יומן פרטי, תכניות מתוזמנות או פשוט כדי לדעת את התאריך הנוכחי.

## איך לקבל את התאריך הנוכחי

```python
import datetime
today = datetime.date.today()
print(today)
```

הפלט יהיה התאריך הנוכחי בתבנית של חודש/יום/שנה. לדוגמה, אם היום הוא 29 באוקטובר 2021, הפלט יהיה:

```python
2021-10-29
```

ניתן גם לקבל תיאור קצר יותר של התאריך, בשימוש בפקודה `strftime`:

```python
print(today.strftime("%A, %B %d, %Y"))
```

הפלט יהיה התאריך הנוכחי עם היום באותיות, החודש בשם מלא והשנה בפורמט של ארבע ספרות. לפני הפקודת `strftime`, צריך להעביר פרמטר שמגדיר את הפורמט של התאריך הרצוי.

## יצירת תאריך או התאריך הנוכחי

בכדי ליצור תאריך ספציפי, צריך ליצור אובייקט ממחלקת `date` ולהעביר את השנה, החודש והיום כפרמטרים.

```python
import datetime
some_date = datetime.date(2021, 7, 15)
print(some_date)
```

הפלט יהיה:

```python
2021-07-15
```

כדי לקבל את התאריך הנוכחי בעת יצירת האובייקט, ניתן להשתמש בפעולה `today()` במקום להציב ערכים ספציפיים.

עוד פעולות נוחות הינן חיסור בין תאריכים והוספת ימים או חודשים לתאריך קיים. לכל פעולה זו יש מתודת עזר המעכבת את האובייקט בהתאם.

לדוגמה, ניתן לחשב כמה ימים עברו עד לתאריך הנוכחי:

```python
print((today - some_date).days)
```

הפלט יהיה:

```python
106
```

ניתן גם להוסיף 30 ימים לתאריך קיים
