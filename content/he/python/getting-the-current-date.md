# למה

הקבלת התאריך הנוכחי היא כלי חשוב בתכנות פייתון שעוזר לנו לעבוד בצורה מסודרת וליישם ניהול זמן יעיל. זה גם מאפשר לנו להציג תאריכים או לשנות אותם על פי הצורך. הקבלה של התאריך הנוכחי בפייתון היא פעולה פשוטה שיכולה להקל עלינו בעבודתנו ולשפר את הביצועים של התוכניות שלנו.

# כיצד לעשות זאת

בחישוב התאריך הנוכחי בפייתון ישנן כמה אפשרויות שונות. אנו נדגים כמה מהן בקודים המוצגים להלן עם תוצאות דוגמה:

```Python
# ייבוא הספריות המתאימות
import datetime
from dateutil import parser

# קבלת התאריך הנוכחי
current_date = datetime.datetime.now()

# הדפסת התאריך
print("התאריך הנוכחי הוא:", current_date)

# המרת תאריך ממחראות לתאריך מדוייק
date_string = "2021-06-13 18:00:00"
exact_date = parser.parse(date_string)

# הוספת ימים לתאריך קיים
new_date = current_date + datetime.timedelta(days=5)

# הדפסת התאריכים
print("התאריך המדוייק הוא:", exact_date)
print("התאריך החדש הוא:", new_date)
```

תוצאה:

```
התאריך הנוכחי הוא: 2021-06-13 18:35:00.027027
התאריך המדוייק הוא: 2021-06-13 18:00:00
התאריך החדש הוא: 2021-06-18 18:35:00.027027
```

כמו שניתן לראות, ישנן שתי דרכים נפוצות לקבלת התאריך הנוכחי - תוך שימוש בספריית `datetime` ובספריית `dateutil`. אנו גם רואים דוגמאות למרת תאריכים ממחראות לתאריכים מדוייקים ולהוספת ימים לתאריך קיים.

# חקירה עמוקה

כעת, נחקור מעט יותר על שתי המתודות הראשיות לקבלת התאריך הנוכחי בפייתון - `datetime.now()`
