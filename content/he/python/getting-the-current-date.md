# מה ולמה?

היום כולנו משתמשים במחשבים ומכשירים חכמים שמציגים את התאריך הנוכחי במסך. אבל מה זה בעצם אומר לקבל את התאריך הנוכחי? ולמה תכנתנים מתעסקים בכך?

קבלת התאריך הנוכחי היא תהליך שבו אנחנו מקבלים את התאריך של היום הנוכחי בדיוק כפי שהוא מוצג לנו במכשירים שלנו. כאשר יש לנו בדיוק תאריך ושעה, יכולים לתכנת פעולות נוספות על פי התאריך הספציפי והיום הנוכחי.

## איך לעשות זאת?

לצורך קבלת התאריך הנוכחי ב-Python, אנחנו נשתמש בפעולה המתארת תאריך נוכחי - ```datetime.now()```. הרצת הפעולה תחזיר את התאריך והשעה הנוכחית בפורמט מסודר ונוח למשתמש.

```Python
from datetime import datetime
print(datetime.now())
```
פלט:
```
2021-08-30 15:30:00.123456
```

ניתן גם להציג את התאריך או השעה בנפרד על ידי שימוש בפעולות נוספות כגון ```date()``` לתאריך, או ```time()``` לשעה.

## העומק

לפני שהיו תכני Python, נכתבו תכני הפעולה datetime בשפת C תחת התוכנית "time". אפשר למצוא עדיין את הקוד הזה בתוכנית הליבה במערכת ההפעלה שלכם.

אם אתם מעוניינים להתעמק יותר בתהליך קבלת התאריך הנוכחי ב-Python, ניתן להשתמש במודול datetime ולבצע פעולות נוספות כגון הוספת או חיסור ימים לתאריך, או הצגת התאריך בפורמטים שונים.

עוד אפשרות היא להשתמש במודול dateutil כדי לטפל בפורמטים שונים של תאריך ושעה ולהשתמש באותם פעולות להצגת התאריך הנוכחי.

תכלית התהליך של קבלת התאריך הנוכחי היא לאפשר למשתמשים להציג ולעבוד עם תאריך ושעה בפורמט שונה ונוח להם.
