שלום חברים!

אתם בטח מכירים את התחום המפתח המרתק והפועל תחת שם פייתון. פייתון היא שפת תכנות פופולארית המשמשת לפתרון בעיות ובניית יישומים רבים כמו אתרי אינטרנט, מכשירי חשיבה וכלים ניתוחיים. בפוסט הזה אני רוצה לחלוק איתכם כמה טיפים נפלאים על כיצד להשתמש בפייתון לקבלת התאריך הנוכחי. אז בואו נתחיל!

## למה

למה לקבל את התאריך הנוכחי? נראה שזו פעולה פשוטה ומי צריך לזכור תאריכים כשיש לנו טלפון חכם ושעון על כך? אבל השאלה המכרעת היא, כשאנחנו עובדים עם מערכת, איזו תאריך אנחנו משתמשים? התיאורטיים צריכים להיות מעודכנים לתאריך הנוכחי על מנת להיות מדוייקים ומתאימים. לכן, קבלת התאריך הנוכחי היא פעולה חשובה ביותר.

## איך לעשות זאת

לקבלת התאריך הנוכחי באמצעות פייתון פשוט מאוד. קוד שאנחנו צריכים להשתמש הוא:

```Python
import datetime

current_date = datetime.datetime.now()

print(current_date)
```

והפלט שלנו יהיה:

```
2021-07-15 11:30:00.940244
```

לתוכנית הזו אין מצב שתצטרך לזכור את התאריך כי זה ימידי מתוך עצמו בתוך התוכנית. אם תרצו את התאריך בפורמט אחר, תוכלו להשתמש בפונקציות תאריך שונות כמו: `.strftime()` לפורמט מותאם אישית ו `date()` לקבלת רק מידע על התאריך ולא השעה.
