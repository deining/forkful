# למה
עדכון התאריך הנוכחי הוא חלק חשוב בתכנות המחשב המודרני. בעצם, זה מאפשר לנו ליצור אפליקציות ואתרי אינטרנט נוחים שיכולים להתאים לזמן המקומי של המשתמשים.

# איך לעשות זאת
תאריך ושעה הם חלק חשוב מתכנות Python. כדי לקבל את התאריך הנוכחי, ניתן להשתמש בפונקציה פנימית בשם "datetime" ולשתף אותה עם הפרמטרים הנכונים. לדוגמה, אם נרצה להדפיס את התאריך הנוכחי בפורמט ישראלי, נוכל להשתמש בפונקציה "strftime" ולאתחל את התאריך לכך.

```Python
import datetime
now = datetime.datetime.now()
print(now.strftime("%d/%m/%Y"))
```

תוצאה:
03/10/2021

כמו כן, ניתן גם להשתמש בפקודת "time" כדי לקבל את השעה הנוכחית בעזרת ציון השעה, הדקות והשניות.

```Python
import time
current_time = time.strftime("%H:%M:%S")
print(current_time)
```

תוצאה:
13:42:15

# חקירה מעמיקה
כמו שראינו בדוגמאות הקוד, תאריך ושעה הם נתונים חשובים ונפוצים בתכנות. אם נרצה ללמוד עוד על פונקציות ומודולים המטפלים בתאריך ושעה, ניתן לקרוא עוד במדריכים ומאמרים על הנושא. כמו כן, כדי להתמיד ולהתאים את התאריך למדינה או לכל מידע גלובלי אחר, ניתן להשתמש בפונקציות שקיימות בפייתון או להתקין מודולים חיצוניים.
