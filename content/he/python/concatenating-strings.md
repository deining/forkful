## למה

תיחיב מחרוזות היא כלי חשוב בתכנות Python שמאפשר למתכנתים לשלב טקסט או מידע מכמה מחרוזות שונות באופן יעיל ופשוט. זה מאפשר לנו ליצור מחרוזות חדשות ודינמיות, לשנות טקסט קיים וליצור תבניות מופשטות לשימוש בקוד.

## איך לעשות זאת

קוד של תיחיב מחרוזות נראה כך:

```Python 

#יצירת שתי מחרוזות:
string1 = "שלום"
string2 = "עולם!"

#תיחיב שתי המחרוזות:
result = string1 + string2

#הדפסת התוצאה:
print(result)

```

הפלט של הקוד הינו:

שלוםעולם!

ניתן גם לתחב את מחרוזות באמצעות פונקציות כמו `format()` ו-`join()`. כמו כן, ניתן להשתמש בתווים מיוחדים כמו `\n` ליצירת שורות חדשות ו `\t` ליצירת טאבים.

## העומקים

תיחיב מחרוזות היא פעולה יסודית בתכנות והיא מאפשרת לנו לעבוד בצורה יעילה עם טקסט בקוד. בעזרת תיכון ופונקציות נוספות, אנו יכולים לעצב ולעבוד עם מחרוזות בתור ערכים משתנים בקוד שלנו.
