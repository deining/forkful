## מה ולמה?
קיצור סיפור, "concatenating" הוא פשוט תהליך של חיבור של שני סטרינגים (מחרוזות) ליחד בסדר איתו שתבחרו. תפעולה זו נקראת "concatenation" והיא מאפשרת למתכנתים ליצור מחרוזות יותר מורכבות ומגוונות בקלות ובמהירות.

## איך לעשות:
```Python
first_name = "דניאל"
last_name = "כהן"
full_name = first_name + " " + last_name
print(full_name)
```
הפלט:
```
דניאל כהן
```

## לעומק:
כמו שכבר למדנו, concatenating מאפשר לנו ליצור מתחרות יותר מורכבות ומגוונות מסתכל על יחד - זהו עושה את החיים של מתכנתים רבים אחדות. בנוסף, ישנם גם פיתרונות אחרים כמו f-strings שתומכים בחזרה לשימוש במחרוזות יותר חכם ויעיל.
בנוסף, לומדים שמשאבים אחרים כמו תווי פישה, אופרטורי מחרוזות ותווי עצמים יכולים לשלב בין מחרוזת fosכח הזקןות.
