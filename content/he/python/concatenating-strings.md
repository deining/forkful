# למה

ישנם כמה סיבות טובות לשימוש בפעולת חיבור מחרוזות בתכנות פייתון. החיבור מחרוזות מאפשר לקבל תוצאה שהיא תמיד מחרוזת חדשה, ולא קיימת אפשרות שתוצאת החיבור תהיה שונה מאשר תוכן המחרוזת הקיימת. כמו כן, החיבור מחרוזות מאפשר לנו לשלב בין כמה מחרוזות לעזור לנו ליצור מחרוזות מורכבות ומבוססות על הערכים המצפים מהמשתמש.

# איך לעשות זאת

כדי לחבר מחרוזות בפייתון, נוכל להשתמש בסימן פלוס (+) או להשתמש בפעולת חיבור מחרוזת (str.format). ניתן לראות בדוגמאות הבאות כיצד להשתמש בפעולת חיבור מחרוזות כדי ליצור מחרוזת חדשה עם ערכים מגוונים:

```Python
# פעולת חיבור מחרוזות באמצעות סימן פלוס (+)
name = "אליס"
age = 23
print("היי, שמי הוא " + name + " ואני בן " + str(age) + " שנים.")

# פעולת חיבור מחרוזות באמצעות str.format
food = "אוכלים"
qty = 5
print("אנחנו " + food + " עוגיות " + str(qty) + " במקרר.")
```

פלט:

```
היי, שמי הוא אליס ואני בן 23 שנים.
אנחנו אוכלים עוגיות 5 במקרר.
```
