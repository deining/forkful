## למה

למה כדאי להמיר תאריך למחרוזת? מידע ברור של תאריך עשוי להיות קשה לקריאה ובעיקר במידה והאתר אינו מציג את התאריך בצורה מתאימה לך. המרת התאריך למחרוזת יכולה לסייע לך להציג את התאריך בצורה נוחה וקריאה יותר, במיוחד כאשר מדובר בכתיבת קוד בפייתון. 

## כיצד להמיר תאריך למחרוזת

```python
import datetime

# הגדרת תאריך בתוך משתנה
date = datetime.datetime(2021, 5, 12)

# המרת התאריך למחרוזת בפורמט נתון
date_string = date.strftime("%Y-%m-%d")

# הדפסת התאריך במחרוזת
print(date_string)
```

תוצאה: `2021-05-12`

בקטע הקוד לעיל, אנו משתמשים במודול datetime כדי ליצור את התאריך. לאחר מכן, באמצעות הפונקציה `strftime()` משנים את פורמט התאריך לפי הצורך ואת התוצאה משמיעים באמצעות פונקצית ההדפסה `print()`. כך ניתן להמיר תאריך למחרוזת בפייתון תוך שמירה על פורמט מסוים.
