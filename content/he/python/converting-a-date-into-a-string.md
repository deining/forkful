## מדוע

המרת תאריך למחרוזת היא פעולה חשובה ושימושית בתכנות בפייתון. היא מאפשרת לנו להציג תאריך בפורמט של אנשים יכולים לקרוא ולהבין בקלות. בנוסף, היא עוזרת לנו לעבוד עם תאריך ולהתאים אותו למטרות ספציפיות כגון הצגתו בטבלאות או במסרים.

## איך לעשות זאת

הנה כמה דוגמאות לקוד פייתון המציגות איך להמיר תאריך למחרוזת:

```Python
# התכונה datetime מאפשרת לנו לעבוד עם תאריכים ושעות
import datetime

# המשתנה today מכיל את התאריך והשעה הנוכחיים
today = datetime.datetime.now()

# הפונקציה strftime מאפשרת להמיר תאריך למחרוזת בפורמט שנקבע על ידי המשתנה format
# בדוגמה זו אנחנו מחליטים שהפורמט המוצג יהיה חודש/יום/שנה
formatted_date = today.strftime("%m/%d/%Y")

# מציג את המחרוזת המיוצגת התאריך הנוכחי
print(formatted_date)
```

פלט:

```
08/04/2021
```

עיבוד נוסף שניתן לעשות הוא להציג את התאריך בפורמט משותף כגון ISO 8601. לדוגמה:

```Python
# המשתנה today מכיל את התאריך והשעה הנוכחיים
today = datetime.datetime.now()

# הפונקציה isoformat מחזירה את התאריך בפורמט ISO 8601
# המתודה replace מאפשרת לנו להחליף תווים במחרוזת לתווים אחרים. במקרה זה, אנחנו מחליפים את המקף (-) בפסיק (,)
# כדי להשתמש בפונקציה replace, אנחנו צריכים להמיר את התאריך למחרוזת
iso_date = str(today).replace("-", ",").isoformat()

# מציג את התאריך בפורמט ISO 8601
print(iso_date)
```

פלט:

```
2021-08-04T15:21:09.286695
```
