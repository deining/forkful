## למה

למה לדבר על חיפוש והחלפת טקסט בפייתון? כי חיפוש והחלפת טקסט הם כלים חשובים לכתיבת קוד יעיל ומהיר. הם מאפשרים לנו להשתמש בתנאים ולשנות חלקים של הטקסט שהוא נמצא בתוך משתנים או קבצים. כלומר, הם מאפשרים לנו לעבוד על כמות גדולה של טקסט בדרכים שונות ומותאמות לצרכים שלנו.

## איך לעשות זאת

כדי לבצע חיפוש והחלפת טקסט בפייתון, נשתמש בשני פונקציות חשובות: `re.search()` ו-`re.sub()`. במקרה הראשון, נחפש את המחרוזת המבוקשת בתוך הטקסט, ואם היא נמצאת נדפיס את המיקום שבו נמצאה. במקרה השני, נחליף את המחרוזת המתאימה בתוך הטקסט במחרוזת אחרת שנציין.

בקוד הבא, אנו מחפשים כל תאריכים בפורמט dd/mm/yyyy בתוך הטקסט ומחליפים אותם לתאריך בפורמט mm/dd/yyyy:

```python
import re

text = "התאריך המקורי הוא 21/11/2021 ותאריך החילוף יהיה 11/21/2021"

new_text = re.sub(r'(\d{2})/(\d{2})/(\d{4})', r'\2/\1/\3', text)

print(new_text)
```

פלט:

`התאריך המקורי הוא 11/21/2021 ותאריך החילוף יהיה 11/21/2021`

בקוד זה, אנו מחפשים את כל המספרים בטקסט ומחליפים את כל אחד מהם במספר הכפול שלו:

```python
import re

text = "המספרים המקוריים הינם 5, 18 ו-2021 והמספרים החדשים יהיו 10, 36 ו-4042"

new_text = re.sub(r'(\d+)', r'\1\1', text)

print(new_text)
```

`המספרים המקוריים הינם 55, 118 ו-2021 והמספרים הח
