# למה?

למה לכתוב קובץ טקסט בפייתון? קובץ טקסט הוא דרך נוחה ויעילה לאחסן ולשמור מידע מתמציתי בפרויקטי פייתון. יצירת קובץ טקסט תוכנתי יכולה לסייע בארגון וניהול פרויקטים, וכן לתקשורת והעברת מידע בין פיתונאים.

# איך לכתוב קובץ טקסט בפייתון?

קוד פייתון פשוט ליצור קובץ טקסט ולהוסיף מידע לתוכו. למשל, השתמשו בפקודת open() לפתיחת קובץ טקסט חדש ושמרו אותו במשתנה. לאחר מכן, ניתן להשתמש בפקודת write() כדי להוסיף מידע לקובץ ולסגור אותו באמצעות פקודת close(). כדי לקרוא ולהדפיס את התוכן של הקובץ, ניתן להשתמש בפקודת read() ולהדפיס את התוכן על ידי השתמשות בפקודת print().

```Python
# פתיחת קובץ טקסט חדש
file = open("my_file.txt", "w")

# הוספת מידע לקובץ
file.write("שלום עולם!")

# סגירת הקובץ
file.close()

# פתיחת הקובץ במצב קריאה והדפסת התוכן
file = open("my_file.txt", "r")
print(file.read())
```

שימו לב שכאשר תכתבו לקובץ טקסט חדש בפייתון, הוא יוצר קובץ חדש אם הקובץ הקודם לא קיים במיקום שאתם מציינים.
