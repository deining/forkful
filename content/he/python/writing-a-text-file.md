## למה

כתיבת קובץ טקסט היא חלק חשוב בתהליך תכנות פייתון שאינו ניתן לעבור במכונת תירוץ. כתיבת קובץ טקסט מאפשרת לנו ליצור קבצים בעבודת המחשב שלנו, כגון קבצי קוד, קבצי נתונים ומסמכי טקסט.

## איך לעשות זאת

```Python
# פתיחת קובץ טקסט לכתיבה, נוסף פרמטר לכתיבה בשורות אחת והוספת טקסט
with open("קובץ.txt", "w") as f:
    f.write("זהו טקסט שנכתב תוך שימוש בקוד פייתון.")

# פתיחת קובץ טקסט לקריאה והדפסת התוכן שלו
with open("קובץ.txt", "r") as f:
    print(f.read())
```

פלט:

```
זהו טקסט שנכתב תוך שימוש בקוד פייתון.
```

## חקירה מעמיקה

כדי לכתוב טקסט בקובץ בעזרת פייתון, אנחנו משתמשים בפקודה המובנית `open()`. פונקציה זו מאפשרת לנו לפתוח קובץ טקסט ולהפעיל עליו פעולות קריאה או כתיבה. בנוסף, אנחנו משתמשים במידע על הנתיב לקובץ כדי לפתוח אותו ובאפשרות להוסיף פרמטרים נוספים לפקודה.
