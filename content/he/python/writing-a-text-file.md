# למה

כתיבת קובץ טקסט היא כלי חיוני בתחום התכנות הקומפיוטרי, המאפשר לנו ליצור ולערוך קבצים המכילים טקסט פשוט. היא מאפשרת ממשק ידידותי שבו אנו יכולים לטפל במידע, לקרוא קבצים ולבצע התאמות מקובץ למשתמש.

# איך לכתוב קובץ טקסט בפייתון

אם אתם מעוניינים לכתוב קובץ טקסט בפייתון, תוכלו לעשות זאת באמצעות הפעולה "open" ולהשתמש במיתוג "w" כדי לכתוב וליצור קובץ טקסט חדש. למטה תמצאו דוגמה של קוד ופלט ליצירת קובץ טקסט חדש בשפת פייתון.

```python
# יצירת קובץ טקסט חדש
file = open("new_file.txt", "w")

# כתיבת תוכן לקובץ
file.write("זהו קובץ טקסט חדש שנכתב בעזרת פייתון!")

# סגירת הקובץ
file.close()

# קריאת הקובץ החדש והדפסת התוכן
new_file = open("new_file.txt", "r")
print(new_file.read())

# פלט: זהו קובץ טקסט חדש שנכתב בעזרת פייתון!
```
