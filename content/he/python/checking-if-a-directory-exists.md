## למה

כשאנו עובדים עם קבצים ותיקיות בתוך תוכניות הפייתון שלנו, ייתכן שנרצה לבדוק אם תיקייה קיימת כדי להתבטא עליה באופן מעשי. לדוגמה, אם תיקייה מכילה קבצים שאנו רוצים לגשת אליהם עבור עיבוד נתונים או העתקת קבצים, יהיה חשוב לבדוק שהתיקייה קיימת כדי שהתכנית שלנו תוכל להתמודד עם זה.

## איך לבדוק אם תיקייה קיימת

מגוון השיטות לבדיקת קיום תיקייה בפייתון, אבל באופן כללי ייתכן שנעשה שימוש בפונקציית "os.path.exists" כדי לבדוק האם תיקייה קיימת או לא. נראה דוגמה לשימוש בפונקציה זו בקוד הבא:

```Python
import os

path = "/home/user/Documents" # נתיב למיקום התיקייה

if os.path.exists(path): # בדיקה האם התיקייה קיימת
    print("התיקייה קיימת!")
else:
    print("התיקייה לא קיימת...")
```

כאשר נריץ את הקוד, אם התיקייה תקיימת, נקבל את התוצאה הבאה:

```
התיקייה קיימת!
```

אם התיקייה לא תקיימת, נקבל את התוצאה הבאה:

```
התיקייה לא קיימת...
```
