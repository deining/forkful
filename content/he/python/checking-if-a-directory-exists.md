## למה
ישנם מקרים רבים כאשר יש לנו תוכניות או תהליכים נוספים שבתוכן נדרש לבדוק האם תיקייה קיימת. לדוגמה, לפני שנוציא קבצים לספרייה, נעשה בדיקה כדי לוודא שהספרייה קיימת ונוכל להעביר אליה את הקבצים. במקרים כאלו, הוא חיוני לדעת איך לבדוק האם תיקייה קיימת ולבצע את זה בצורה יעילה ומדויקת.

## איך לבדוק האם תיקייה קיימת
האתגר הגדול בבדיקת קיומה של תיקייה הוא למצוא את המתודה המתאימה עבור הנוד שלנו. בשפת פייתון, את המתודה הזאת מתקראת "path.exists()" והיא נמצאת במודול os.

```Python
import os
# בדיקה אם התיקייה "documents" קיימת
if os.path.exists("documents"): 
    print("התיקייה קיימת")
else:
    print("התיקייה אינה קיימת")
```

בתוך התנאי, אנו משתמשים בפונקציה "exists()" כדי לבדוק אם הנתיב המוגדר קיים. אם התנאי שלנו מתקיים, אנו מדפיסים הודעה שהתיקייה קיימת. אחרת, אנו מדפיסים הודעה כי התיקייה אינה קיימת.
