## למה

למה להדפיס פלטי גימור מקושרים בתכנות פייתון? עלינו לדבר על נקודת האור שבגישה זו וכיצד יכולה להרחיב ולשפר את התהליך התכנותי שלכם.

## איך לעשות

עכשיו נתאר כיצד להדפיס פלטי גימור באמצעות קוד פייתון. נבצע זאת באמצעות ```print()``` פקודה, כפי שמוצג בדוגמאות הבאות:

```Python
x = 3
y = 5
print("המספרים שלנו הם:", x, y)
print("רק נסו להביע את כיוונם, זה ממהיר ונוח!")
```

תוצאות הפלט יהיו:

```
המספרים שלנו הם: 3 5
רק נסו להביע את כיוונם, זה ממהיר ונוח!
```

עוד דוגמא להדפסת פלט גימור בקוד הפייתון:

```Python
my_list = [1, 2, 3, 4, 5]
for num in my_list:
    print("המספר הנוכחי הוא:", num)
```

הפלט יהיה:

```
המספר הנוכחי הוא: 1
המספר הנוכחי הוא: 2
המספר הנוכחי הוא: 3
המספר הנוכחי הוא: 4
המספר הנוכחי הוא: 5
```

כפי שאתם יכולים לראות, המסרים שנדפסים יכולים להיות כל דבר שתרצו לכלול בתוך הקוד שלכם, כגון משתנים, מחרוזות, או כל תוכן אחר שתרצו. זה מאפשר לכם לבדוק ולבחון את ערכי המשתנים שלכם בזמן הריצה של התוכנית שלכם.
