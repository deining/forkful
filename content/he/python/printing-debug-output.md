## למה

למה לדאג להדפיס פלטים מפעולות הדגמה שלנו? הדפסת פלטים ניתנת להשתמש ככלי עוזר לנו בזמן פתחת ותיקון באגים ובאיתור של קוד שגוי.

## איך לעשות זאת

כדי להדפיס פלטים כדאי להשתמש בפונקציית `print()` בשפת פייתון. הנה דוגמה להדפסת הודעה פשוטה:

```python
print("כל הכבוד, נראה שהקוד עובד כמו שצריך!")
```

הפלט הצפוי הוא:

```
כל הכבוד, נראה שהקוד עובד כמו שצריך!
```

בנוסף, אפשר להשתמש בפונקציית `print()` להדפיס ערכים של משתנים בתוך הקוד שלנו כדי לעזור לנו לבדוק שהם מכילים את הערכים הנכונים. לדוגמה:

```python
num1 = 10
num2 = 5
print("המשתנה num1 מכיל את המספר:", num1)
print("המשתנה num2 מכיל את המספר:", num2)
```

הפלט הצפוי הוא:

```
המשתנה num1 מכיל את המספר: 10
המשתנה num2 מכיל את המספר: 5
```

בנוסף, ניתן להשתמש בפונקציית `print()` בשילוב עם מחרוזת פורמט כדי להדפיס ערכים יחד עם טקסט. לדוגמה:

```python
name = "דניאל"
age = 30
print("שמי הוא: {}, ואני בת {} שנים".format(name, age))
```

הפלט הצפוי הוא:

```
שמי הוא: דניאל, ואני בת 30 שנים
```
