# מה ולמה?

חילוץ תת מחרוזת הוא פעולה שמאפשרת למתכנתים לחלץ חלק מהמחרוזת המקורית לפי קריטריונים שונים. ככל שהמתכנתים משתמשים בתתי מחרוזת, הם יכולים לפענח מידע מורחב ולשפר את ביצוע התוכנית שלהם.

# איך לעשות?

קוד פייתון מציג שני דרכים פשוטות לחלץ תת מחרוזת: שימוש בווילדקארדים ובפעולת הרי-אוור. הנה כמה דוגמאות של קוד פייתון לחילוץ תת מחרוזת, עם תוצאות להוכיח את התוצאה:

```python
# שימוש בווילדקארדים
string = "מאמין שפייתון הוא מערכת פיתוח פנטסטית"
substring = string[8:14]
print(substring) # פייתון

# שימוש בפעולת הרי-אוור
string = "פייתון התחבר למחלקת הורים"
substring = string.split()[0]
print(substring) # פייתון
```

# לחקור עומק

חילוץ תת מחרוזת הוא טכניקה נפוצה ושימושית למתכנתים מכל הרמות ניסיון. הטכניקה הזו נולדה כתוצאה מהצורך לפענח ולתמוך בתהליך עיבוד שפת תכנות. ניתן למצוא טכניקות טל כמו חיתוך במילים מסוימות, טיפוס ומופעים, ארגומנטים נכונים ומשתנים עוד פחות טריוויאליים.
