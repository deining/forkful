## למה צריך לחלץ תת-מחרוזות?

ליתר נוחות ויעילות בתכנות, נדרש לפעמים לחלץ חלקים מסוימים של מחרוזות. כך ניתן לטפל בנתונים גדולים יותר וליישם תנאים ופעולות על חלקים מסוימים של המחרוזת בקלות ובמהירות.

## איך לחלץ תת-מחרוזות באמצעות פייתון?

```python
# דוגמה לחליצת תת-מחרוזת ממחרוזת נתונה
string = "זהו מחרוזת לדוגמה"
substring = string[2:6] # מחזיר את התת-מחרוזת "מחר"
print(substring) # ידפיס "מחר"

# דוגמה לחילוץ של תת-מחרוזות מקובץ
file = open("קובץ.txt", "r") # פתיחת הקובץ כקובץ קריאה
for line in file: # לולאה שעוברת על כל השורות בקובץ
    substring = line[4:9] # מחזיר את התת-מחרוזת בין התווים 4 ל-9 בשורה הנוכחית
    print(substring) # ידפיס את התת-מחרוזת בכל שורה
file.close() # סגירת הקובץ
```

## בעומק: נתונים נוספים על חילוץ תת-מחרוזות

כאשר מחלץ תת-מחרוזות ממחרוזת, התווים נבחרים לפי מספר המיקומים שלהם במחרוזת, שפורטו בצורה חיונית את התת-מחרוזת. כמו כן, ניתן להשתמש במגוון של פעולות כמו תנאים ולולאות כדי לטפל במחרוזות מסובכות יותר ולחלץ מהן תת-מחרוזות בדיוק לפי הצורך.
