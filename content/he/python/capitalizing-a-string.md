## מה ולמה?
כשמדברים על "להקפיטל את המחרוזת", מתכוונים לשנות את האות הראשונה במחרוזת לאות רישיות, ולאתר שמי היא האות הראשונה במחרוזת. המטרה העיקרית של כך היא לשפר את נראות הקוד ולהפוך אותו ליותר קריא עבור מתכנתים אחרים שעשויים לעבוד עם הקוד בעתיד.

## כיצד לעשות זאת:
ניתן להקפיטל מחרוזת באמצעות הפקודה upper(). לדוגמה, אם נקליד בקוד "hello" ונקרוא לה פקודת upper(), הפלט שנקבל יהיה "HELLO". ניתן להשתמש גם בפונקציה capitalize() כדי להקפיטל את המילה הראשונה במחרוזת. ניתן לראות דוגמה להוספת מחרוזת מקופלת לרשימה עם מילים אחרות להלן:

```Python
names = ["david", "rachel", "josh"]
capital_names = [name.capitalize() for name in names]
print(capital_names)
```

פלט:
```Python
["David", "Rachel", "Josh"]
```

## חקירה מעמיקה:
השימוש באופציות כמו upper() ו-capitalize() התחיל בשנות ה-70 של המאה הקודמת עם הפיתוח של שפת התכנות פייתון. ניתן להשתמש גם במתודות נוספות כמו title() ו swapcase() כדי להקפיטל את המחרוזת לפי כללים שונים. ישנן גם אפשרויות נוספות כמו שימוש בספריות חיצוניות, כגון PyTitleCase או inflect, שמאפשרות להקפיטל את המחרוזת לפי כללי הכתיב המקובלים בשפת העברית ובשפות אחרות.
