## למה

למעשה, יש לומר כי קריאת ארגומנטים משורת פקודה היא חלק חשוב מתכנות בשפת פייתון. ייתכן שתרצו לכתוב תוכניות שיאפשרו למשתמשים להשתמש בתכשיטים ולהעביר רשומים וארגומנטים בצורה נוחה. בואו נראה כיצד תוכלו לקרוא ארגומנטים משורת הפקודה בשפת פייתון בצורה פשוטה ומהירה!

## איך לעשות זאת

כדי לקרוא ארגומנטים משורת הפקודה בשפת פייתון, תצטרכו להשתמש במודול sys ולהפעיל את הפקודה sys.argv. הנה דוגמא פשוטה:

```Python
import sys

# מדפיס את כל הארגומנטים שהוכנסו משורת הפקודה
print(sys.argv)

# מדפיס את הארגומנט הראשון
print(sys.argv[1])

# מדפיס את כל הארגומנטים בלי שם התוכנית
print(sys.argv[1:])
```

כאשר תריצו את התוכנית ותכניסו ארגומנטים משורת הפקודה, תוכלו לראות איך התוכנית מדפיסה אותם. לדוגמא, אם נריץ את התוכנית עם הארגומנטים "python example.py test argument", הפלט יהיה:

```Python
['example.py', 'test', 'argument']
test
['test', 'argument']
```

ניתן להשתמש בארגומנטים שנקלטו משורת הפקודה כדי לבצע פעולות נוספות כמו כתיבת קבצים או שליחת הודעות דוא"ל. כמו כן, ניתן להשתמש בתנאי if לפי שם התוכנית הנקבע בארגומנטים על מנת לבצע פעולות שונות לפי הארגומנטים שנקבלו.
