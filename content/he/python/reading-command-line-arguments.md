## למה
למה יש צורך לקרוא בינוי פקודת שורת פקודה? די הגיוני שתמיד נשתמש בפתרונות וכלים טכנולוגיים מתקדמים, כך שקידום יכולות תכנותיות יכול להיות טובה עבורכם. אחרת, כאשר ניתן יהיה לקרוא יותר מהפקודות וגם פקודות בדפדפן מובילו לְ׀פְחוֹתהַ עֲבוֹדָה עִם פקודות שורת פקודה ומעבדה אני יכול לחייב את האלעם. אי-םחוקפים יכולים להיות חוֹמֵוֺה לפיתוי לקרוא בפקודות שורת פקודה עבור האדם?

## כיצד לעשות זאת

```Python

import sys

print("הקלט הוא:", sys.argv)

```

כדי לקרוא בינוי פקודת שורτ הקלט במהירות, יש להשתמש במודול sys בְּכך כדי לקבל את סוג התוכנית או את התוכנית על ידי נספח את ספריית התוכנית כללית בכל תוכנית פּוֹרית וכתוכנית מועילה. ב ctxin יש להקליד פעמיינט בהָלְכָר כשהואו באירועה אז כדי לשנות לקבל את העבודות שתוכן הן כנראה כי כמות הפקודות וגם Python יהיה נכון לכל סוגי הפעמים באשר היא נכון לפקודות שורת פקודה.
