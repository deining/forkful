## מה ולמה

קריאת ​​ארגומנטים ממסך הפקודה היא תהליך בו מתכנתים מקבלים קלט מהמשתמש דרך המסך האחורי של המחשב. המטרה העיקרית של קריאת ​​ארגומנטים היא לאפשר למשתמש לשנות את התנהגות התוכנית על ידי מתן ערך לאחת הפונקציות שלה.

## איך לעשות זאת

```Python
import sys # מייבאים את הספרייה sys
args = sys.argv # קוראים את הארגומנטים מהמסך האחורי ושמים אותם ברשימה

# מציגים כל ארגומנט נתון על ידי לולאה פשוטה
for arg in args:
  print(arg) 

# אפשר גם להשתמש בו על מנת לבדוק את כמות הארגומנטים שניתנו
print(len(args))
```

לדוגמה, אם נריץ את התוכנית כך: ```python program.py argument1 argument2```, הפלט המשוער אילו יהיו שני הארגומנטים הנתונים והכמות המתאימה שלהם תהיה:

```
argument1
argument2
2
```

## עיון עמוק

קריאת ​​ארגומנטים ממסך הפקודה היא כלי הרחבה וחיוני לכל מתכנת פייתון. בעקבות זאת, המון ספריות עשויות להתייחס אליה במקרה של צורך בתצורת ארגומנטים מתקדמת יותר. ייתכן כי תצטרכו לכתוב קוד מצומצם כדי לזהות כל כניסה מתאימה. ניתן גם להשתמש בספריות נוספות, כמו "click" או "argparse", שתסייעו להפוך את התהליך לפשוט יותר.
