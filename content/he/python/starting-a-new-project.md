## מה ולמה?

התחלת פרויקט חדש היא התהליך שבו מפתחים יוצרים פרויקט חדש עבור תוכנית מחשב או אפליקציה. פעולה זו נדרשת כאשר עולים לפעם הראשונה עם פרויקט חדש או מכשיר מחשב חדש, או כאשר רוצים להתחיל מקדמה חדשה לפרויקט קיים.

## איך לעשות?

להלן כמה דוגמאות לשימוש בפייתון כדי להתחיל פרויקט חדש.

```Python
# כדי להתחיל פרויקט חדש, ניתן לשים את הקוד הבא בקובץ חדש עם סיומת .py
print("Hello World!") 
# לאחר הריצה של הקוד, יודפס "Hello World!"
```

```Python
# לעתים יתרבו בפרויקט חדש, אנחנו נעדיף להשתמש בכמה קבצים ולא רק באחד.
# לדוגמה, בקובץ הראשי נוכל להשתמש בקוד הבא:
import helper # יבוא קובץ העזר helper.py

numbers = [1, 2, 3, 4]
total = helper.sum_numbers(numbers) # נפעיל פונקציית מחשבון בקובץ העזר
print(total) # יודפס 10
```

## מבולטות

- התחלת פרויקט חדש בפייתון נפוצה בעיקר כאשר מתחילים עם מכשיר מחשב חדש או רוצים ליצור פרויקט חדש לתוך מערכת הקוד המקור שלנו.
- ישנן אפשרויות נוספות כמו שימוש בתבניות או קובצי הגדרות כדי להיפטר מחריץ התחלה.
- בכותרת ההתחלה של הקובץ הראשי, נהיה עושים צוואר ונרשום את שם הפרויקט ושם המפתח.
