# למה:
כתיבת קטעי קוד בשפת פייתון היא תהליך מהנה ומאתגר כאחד. אחת מהפעולות הנפוצות ביותר שתיכף לימדו היא איך למצוא את אורך המחרוזת (string). היכן כאשר כתבתם כמה מחרוזות בשפת פייתון, חשבתם אי פעם מה הם האורך שלהן? זו הסיבה שממליצים לכם לדעת כיצד למצוא את אורך המחרוזת בפייתון.

# איך לעשות את זה:
למצוא את אורך המחרוזת בשפת פייתון פשוט מאוד. מצורף דוגמא עם הסברים המפורטים בתוך בלוקי קוד בשפת Markdown.

```Python
string = "שלום עולם"
print(len(string))
```

פלט:
```
9
```

הראשון שנעבור עליו הוא שתישתמשו בפונקציה המוגדרת מראש len(). נתחיל ע"י יצירת משתנה שמכיל את המחרוזת שברצונכם לבדוק אורכה. כאן השתמשנו במחרוזת "שלום עולם". לאחר מכן, נדפיס באמצעות הפקודה print את תוצאת הפונקציה len(string). תוצאת הפלט תהיה 9, מראה שאורך המחרוזת הוא 9 תווים. זהו, קל כמו מעוף!
