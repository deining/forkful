# למה:

בכתיבת תוכניות המחשב בשפת פייתון, נתקלים לעיתים קרובות בצורך לדעת את אורך מחרוזת מסוימת. לא חשוב אם מדובר במחרוזת קצרה או ארוכה, יכולות חישוב אורך המחרוזת יכולות להיות מאוד שימושיות ליישומי מחשב רבים. 

# כיצד לקבל את אורך המחרוזת בפייתון:

```Python
string = "שלום לכולם"
print(len(string))
```

פלט:

12

בדוגמא הנ"ל, נוצרת משתנה בשם "string" ומוגדרת לו מחרוזת עם התוכן "שלום לכולם". לאחר מכן, בעזרת הפונקציה len(), אנו יכולים להדפיס את אורך המחרוזת בתווים. כפי שאתם יכולים לראות, התוצאה היא 12, מספר התווים במחרוזת כולל הרווחים.

אפשר גם להשתמש בלולאת for כדי לעבור על התווים במחרוזת ולספור את מספרם, או לפתוח קובץ ולספור את מספר השורות במחרוזת. יכולות חישוב אורך מחרוזת יכולות להיות מזויפות, אבל הן תמיד מצליחות לתת לנו מידע חיוני על התמליל שאנחנו מתעסקים איתו.
