## למה

היי כולם! היום אני רוצה לדבר איתכם על כיצד למצוא את אורך המחרוזת בשפת פייתון. למה זה חשוב? בכדי להפוך למתכנתים מקצועיים, חשוב לדעת איך לטפל במחרוזות ולעבוד איתן ביעילות.

## כיצד

```python
string = "שלום עולם"
print(len(string))
```

ברגע שיש לנו מחרוזת, ניתן להשתמש בפונקציה len כדי למצוא את אורךה. כאן אנו מגדירים משתנה שמכיל את המחרוזת שאנו רוצים לבדוק ואז מדפיסים את התוצאה. במקרה הזה, התוצאה תהיה 11, מכיוון שיש לנו 11 תווים במחרוזת "שלום עולם". 

בכמה מקרים, ניתן להשתמש בפונקציה len עם פרמטרים נוספים, לדוגמה כאשר עובדים עם רשימות או מילונים של מחרוזות. אם נרצה לחשב את אורך המחרוזת הארוכה ביותר ברשימה, נוכל להשתמש בקוד הבא:

```python
list_of_strings = ["מחרוזת אחת", "מחרוזת ארוכה מאוד", "string"]
longest_string = 0
for string in list_of_strings:
    if len(string) > longest_string:
        longest_string = len(string)
print(longest_string)
```

כאן אנחנו עוברים על כל אחת מהמחרוזות ברשימה ובודקים את אורךה. אם האורך גדול יותר מהמשתנה longest_string, אנחנו מעדכנים אותו בעזרת הערך החדש. כעת, אנחנו מדפיסים את הערך הסופי של longest_string, שהוא 18 מכיוון שהמחרוזת הארוכה ביותר ברשימה היא "מחרוזת ארוכה מאוד".
