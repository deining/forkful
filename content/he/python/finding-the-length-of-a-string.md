## Why

למה לעסוק במציאת אורך של מחרוזת?

למרבה המקרים, כאשר אנחנו מתעסקים בתיכנות בשפת פייתון, ייתכן שנצטרך ליצור תוכניות המתייחסות לטקסטים ארוכים או לערוך מחרוזות קיימות. כדי לעשות זאת, חשוב מאוד לדעת את אורך המחרוזת כדי לבצע פעולות לוגיות ומבנה לולאות מתאימים. לכן, ידע על כיצד למצוא את האורך של מחרוזת הוא מיוחד לשפת פייתון ומאפשר לנו לכתוב קוד יעיל יותר.

## How To

למצוא את אורך המחרוזת בפייתון פשוט מאוד ונעשה על ידי שימוש בפונקציית "len ()". נתחיל עם הקוד הבא:

```Python
# דוגמה לבדיקת אורך של מחרוזת בפייתון
text = "שלום עולם"
length = len(text)

print("המחרוזת היא: ", text)
print("אורך המחרוזת הוא: ", length)
```

כמו שאתם רואים, באמצעות הפונקציה "len" אנו מקבלים את מספר התווים שבמחרוזת. היא יכולה לעבוד על כל סוג של מחרוזת ונותנת לנו את אורך התווים ללא קשיים.

ננסה עם מחרוזת נוספת כדי לראות את הפונקציה גם בפועל:

```Python
# דוגמה נוספת לבדיקת אורך של מחרוזת בפייתון
text = "למה לימוד פייתון כיף כל כך?"
length = len(text)

print("המחרוזת היא: ", text)
print("אורך המחרוזת הוא: ", length)
```

```
המחרוזת היא: למה לימוד פייתון כיף כל כך?

```
