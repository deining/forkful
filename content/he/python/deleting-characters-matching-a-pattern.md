## למה

מדוע למחוק תווים התואמים לתבנית? יתרה משמעותית, מחיקת תווים התואמים לתבנית יכולה לסייע בסירוב טקסטים, ניתוח מאפיינים, ועוד.

## איך לעשות

ישנם מספר אפשרויות למחיקת תווים התואמים לתבנית בפייתון. להלן מספר דוגמאות של קוד ופלט כדי להדגיש כיצד לעשות זאת בצורה קלה ויעילה.

```Python
# מחיקת תווים מתוך מחרוזת מסוימת
# התבנית להיות נמחקים היא "לו"
str = "שלום לך, לי להיות לו מחרוזת!"
nstr = str.replace("לו", "")
print(nstr)

# התנאים גם יכולים להיות מבוססים על התבנית
str = "אני קורא ל(not) אל כף השליש במשימה 3"
nstr = re.sub(r"\(not\) ", "", str)
print(nstr)
```

פלט:

```
שלום , לי מחרוזת!
אני קורא לאל כף השליש במשימה 3
```

## טיפול חמור

קוד פייתון מאפשר לנו לחלק מאוד מתקדמים של טיפול בתווים, כולל התאמת תבניות למחיקת כמו נקודת השנייה של מחרוזת; קודם נייר לפרסוד השייך למחיקת תווים בתבנית על פי יסודת לוגיה. עבור סקרפ, אנחנו חייבים להשתמש אבל את .search (), אשר תואמת מחרוזת כמו לטיפול בתווים.
