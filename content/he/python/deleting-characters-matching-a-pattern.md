## למה

כתיבת קוד תמיד כוללת גזירה ועיבוד נתונים. לפעמים, נתקלים במצבים בהם נרצה למחוק תווים ספציפיים מתוך מחרוזת בהתאם לתבנית מסוימת. למה נרצה לעשות זאת? כמה דוגמאות של תיקווה יהיו לכתחילה ניקוי נתונים לפני עיבודם, או שוליים של נתונים שלא מעניינים אותנו.

## כיצד

דוגמתון אנלוגי של מחרוזת תת מחרוזת מתאימה


```Python
# הגדרת תבנית
pattern = "py" 

# מחרוזת המקור
string = "python programming is fun!" 

# הדפסת הרווחים תחת התחביר החדש
print("חיפוש תבנית: " + pattern)
print ("מחרוזת מקור: " + string)
print("מחרוזת מיועדת: " + string.replace(pattern,""))
```


פלט:
חיפוש תבנית: py
מחרוזת מקור: python programming is fun!
מחרוזת מיועדת: thon rogaming is fun!

כאן, אנו משתמשים בפונקציה שמרכזת את התבנית ומפעילה אותה על המחרוזת המקור בכדי להסיר את כל האותיות "py" ממנה. כמו כן, ניתן להשתמש בתבנית כמחרוזת מחלקת בכדי למחוק תווים במיקומים ספציפיים במחרוזת.

## Deep Dive

מחיקת תווים מתוך מחרוזת היא תהליך חשוב בתוך תכנות פייתון. נוכל לשפר את היכולות שלנו בשימוש בדפי פייתון בכדי לנהל עבודה כלכלית נכונה ולנתח קוד צורה נכונה.

כמו כן, ניתן להשתמש בתבניות רגולריות כדי להגדיר תבניות מורכבות ומתקדמות לגזירה ועיבוד נתונים.
