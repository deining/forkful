כתיבת ופרסום אפליקציות ותוכניות מחייבת יכולות יצירתיות וכישורים טכניים, ולאחרונה נתקלתי בצורך לשלוח בקשת HTTP עם אימות בסיסי. בכתבה הזו, אני אספר על מהות הבקשות הללו וכיצד ניתן לממש אותן בקלות בשפת פייתון.

## Why

בעולם התכנות, ייתכן שתגיעו לפעם בה תצטרכו לאמת את זהותכם בקשורה לשרת או לאפליקציה מסוימת. הדרך היעילה והמועילה ביותר לעשות זאת היא דרך שליחת בקשת HTTP עם אימות בסיסי. כיצד זה עובד? בתצורה הקלאסית של אימות, המשתמש מעמיס שם משתמש וסיסמה בצורה מודפסת על האתר או על גבי שרת. כאשר הבקשה מגיעה לשרת, הוא מנסה לאמת זאת על ידי בדיקה האם שם המשתמש והסיסמה תואמים לזוג הנתונים המאוחסנים במסד הנתונים. אם האימות אומת, המשתמש יכול לגשת למשאבים המוגנים בהתאם.

## How To

תחילה, נצטרך ליצור חיבור עם השרת הרלוונטי באמצעות הפונקציה `requests.get ()` של ספריית ה- requests. יש להעביר לפונקציה את הכתובת URL של המשאב שאנחנו רוצים לגשת אליו, כך כמו גם שם משתמש וסיסמה. ניתן לעשות זאת באמצעות הפרמטרים `auth = ('username', 'password')` כפרמטר נוסף:

```Python
import requests

url = "http://www.example.com"

response = requests.get(url, auth=('username','password'))

print(response.text)
```
