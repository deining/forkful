# למה:
כתיבת קוד מורכב עשויה להיות מעט מבולגנת וקשה לקריאה. בעזרת תבניות רגולריות, ניתן לייעל את תהליך החיפוש והחילוץ של נתונים מתוך קוד סורס ולהפוך את הקוד לקריא ונוח יותר לעיבוד והבנה.

# כיצד להשתמש:
כדי להתחיל להשתמש בתבניות רגולריות בפייתון, יש לייבא את המודול "re" ולהשתמש בפונקציית search כדי לגשת לתבנית שלנו. לדוגמה:

```Python
import re

text = 'שלום, הייתי רוצה לקנות 3 תפוחים ו-5 בננות.'

pattern = r'\d+ תפוחים'
result = re.search(pattern, text)

print(result.group()) # 3 תפוחים
```
ניתן לראות שבעזרת הפונקצייה search והתבנית שיצרנו, הצלחנו לחפש ולחלץ את המספר 3 תפוחים מתוך הטקסט.

# חפירה עמוקה:
ישנם כמה תכונות מתקדמות יותר בתבניות רגולריות שניתן להשתמש בהן, כמו קווי מתח, תכונות של מניפולציה ותנאים. כמו כן, ניתן להשתמש בתבניות רגולריות בפייתון כדי לתקן ולנקות נתונים מסוימים. חשוב לשים לב לכתיבת התבניות בקוד כדי להבטיח שהחיפוש יתבצע בהצלחה.
