## למה

בפוסט זה נדבר על איך להשתמש בביטויים רגילים בפייתון ולמה כדאי ללמוד להשתמש בהם כחלק מהתהליך התכנותי שלכם.

## איך לעשות זאת

רגע, מה זה בכלל הכוונה של "ביטוי רגיל"? נתחיל בהסבר קצר: ביטוי רגיל הוא סט של תבניות אלפאנומריות המשמשות לאיתור מחרוזות בתוך טקסטים. הם יכולים להיות מאוד מועילים בתהליך של מיצוא או עיבוד טקסט בתכניות פייתון.

בואו נראה מספר דוגמאות פשוטות של כיצד להשתמש בביטויים רגילים בפייתון:

```Python
import re  # ייבא את המודול regex

# נבנה ביטוי רגיל שממצא את כל המחרוזות המתחילות באותיות בקטנות
regex1 = r"[a-z]+"

# נבדוק אם המחרוזת "hello" מתאימה לביטוי הרגיל
if re.match(regex1, "hello"):
    print("המחרוזת מתאימה לביטוי הרגיל")

# נפעיל את הביטוי על מחרוזת ונדפיס את התוצאה
regex2 = r"[0-9]+"  # הביטוי מוצא את כל המספרים במחרוזת
text = "123abc456def"
matches = re.findall(regex2, text)
print(matches)  # תוצאה: ['123', '456']
```

כמו שאתם יכולים לראות בדוגמאות, אנחנו משתמשים במודול "re" של פייתון כדי לבנות את כל הביטויים הרגילים ולבדוק אותם על מחרוזות.
