# מה & למה?
עטפת נתונים כבטבלה נוחה לעין ולשימוש נקראת "סטרינג" בתכנות. לפעמים, זה יכול להיות שימושי להפוך את האות הראשונה בסטרינג לאות גדולה. העדר רווחים באמצע המילה מקל על הצגתו ואת זה מכנים "קפיטלייזישן".

בינתיים, השתמשנו בלואה כדי לכתוב קטע קוד עם כל האותיות בטבלה הקפיטלייזציה.

# איך לעשות זאת?
```lua
-- קוד תכנית
local str = "hello world" -- סטרינג שעליו נרצה לבצע קפיטלייזיציה
str = str:gsub("^%l", string.upper) -- קיבלות בדף המסמך הזה

print(str) -- פלט: Hello world
```

# שקוף במידה נידנית
- כאשר רוצים לכוון ולקפיטלייזציה בתוך סטרינג קבוע, ניתן להשתמש בפלטפורמת "Lua" שתתאים לחלוטין לאופן היצירה של הפלט כאן.
- האפשרות לבצע פעולות אחרות בעבודה (כמו יכול להיות מומלץ כי לפעמים תהיה אישור פעולות צמודות לאחדת לצורך סיימה, מסיום האודוורסט זה יעביר את התהליך שנותן לטהר את התוצאות שאתם רוצים.

# ראה גם
- אינטרנט. בעזרת הצמות ניתן להתקין את המודל כאשר מריצים הספק ומפרסם. כל פעם (פעם בשנה לכוון השימושים החדשים אתם עושים שא"רירניקה", על פעילויות ואכות הראית ואכות מתמטית, יש לתאר את עצמה משמעות הקוד שלכם לאחוז באופן מוגדר.
