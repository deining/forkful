## למה
הטאסק כדי לבדוק אם הספרייה קיימת נדרש במקרים רבים כאשר עובדים עם קבצים ומידע שמתוחזק במחשב. זה יכול להיות חשוב במיוחד כאשר פועלים עם תהליכים אוטומטיים שצריכים לבדוק את קיומו של ספרייה מסוימת לפני שהם ממשיכים בפעילותם.

## איך לבדוק אם ספרייה קיימת ב-C
הקוד הבא מדגים איך לבדוק אם ספרייה קיימת ב-C. הוא משתמש בפונקציית `opendir()` שבודקת אם אפשר לפתוח את הספרייה, ואם כן, הוא מדפיס הודעת התראה שקיימת.

```C 
#include <stdio.h>
#include <dirent.h>

int main() {
    char* path = "path/to/directory";
    DIR* dir = opendir(path);

    if (dir) {
        printf("הספרייה קיימת!");
        closedir(dir);
    } else {
        printf("הספרייה לא קיימת!");
    }
    
    return 0;
}
```

#### פלט:
`הספרייה קיימת!`

שימו לב שלפני המדפסת יש פונקציית `closedir()` שמסגרת את הספרייה אחרי שסיימנו לבדוק אותה.
