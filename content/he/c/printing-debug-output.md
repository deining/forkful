## למה

כשמתכנתים מתחילים ללמוד את שפת התכנות C, תחילה הם לומדים כיצד להדפיס תוצאות, לבסוף הם משתמשים בזה גם בתהליך הפיתוח. כאשר משתמשים בדפוג כדי לבקש מידע נוסף ברגעי צריכת מפתחי מידע, זה יכול להיות תועלת להדפיס מידע נוסף בעת הפיתוח באמצעות תוצאות ההדפסה של הקוד שלך.

## כיצד להדפיס תוצאות

קוד הדגמה הבא יראה לכם איך להדפיס תוצאות בעזרת הפונקציה printf:

```
#include <stdio.h>

int main()
{
    int num1 = 10;
    int num2 = 20;
    
    // הדפסת המספר הראשון
    printf("המספר הראשון הוא: %d\n", num1);
    
    // הדפסת המספר השני
    printf("המספר השני הוא: %d\n", num2);
    
    // חיבור המספרים והדפסת התוצאה
    printf("התוצאה של חיבור המספרים הוא: %d\n", num1 + num2);
    
    return 0;
}
```

תוצאת הריצה של הקוד תהיה:

```
המספר הראשון הוא: 10
המספר השני הוא: 20
התוצאה של חיבור המספרים הוא: 30
```

בקוד הדגמה ניתן לראות דגמאות של כיצד להדפיס ערכים של משתנים שונים, וגם כיצד לחבר ביניהם ולהציג תוצאות.
