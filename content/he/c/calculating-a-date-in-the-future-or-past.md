כתיבה בסגנון כתב פולחן על תכנות בשפת C עבור קוראי עברית.

## למה

גם בתכנות בשפת C ניתן להשתמש במינייות בכדי לפענח קודקוד אישי שנכתב ולבצע חישובים שונים. אחת התוכליות השונות הוא חישוב תאריך בעתיד או בעבר. ניתן להשתמש בקוד כדי לחשב כמה ימים נמצאים בין תאריכים שונים, או לחשב תאריך אחד מסוים לפני או אחרי תאריך נתון.

## איך לעשות זאת

על מנת לחשב תאריך בעתיד או בעבר, ניתן להשתמש בפונקציות של מעבר תאריך וזמן בשפת C. לדוגמה, נוכל להשתמש בפונקציה `time()` כדי לקבל את הזמן הנוכחי בשניות מאז תאריך ספציפי, ולאחר מכן להוסיף או לחסור כמה ימים מהתאריך הנוכחי באמצעות פונקציות כגון `mktime()` ו- `gmtime()`.

תוכלו גם להשתמש בספריית המתמטית של C בשביל לבצע חישובים אריתמטיים מתקדמים כדי לחשב את התאריך הנחוץ.

להלן דוגמאות לקודים עם תוצאות:

```C
#include <stdio.h>
#include <time.h>

int main() {
    // קבלת הזמן הנוכחי
    time_t current_time = time(NULL);

    // חישוב תאריך מסוים בעל 10 ימים לפני התאריך הנוכחי
    time_t target_time = current_time - (10 * 24 * 60 * 60);

    // כיוון שמערכת C מחזירה את התאריך בשניות מאז ה- epoch (1 בינואר 1970), ניתן להציג אותו בפורמטי תאריך נוחים כדי לקרוא אותו בצורה נורמלית
    struct tm *info = localtime(&target_time);
    printf("10 ימים לפני היום הנוכחי היה בתאריך: %d/%d/%d\n",
           info->tm_mday, info->tm_mon + 1, info->tm_year + 1900);
