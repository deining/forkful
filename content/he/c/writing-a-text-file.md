# למה:

כתיבת קבצי טקסט היא חלק חשוב מאוד בתכנות בשפת C. קבצי הטקסט מאפשרים לנו לקבל קלט מהמשתמש ולשמור אותו כמו מידע על מחשב כדי להשתמש בו לצורך עיבוד נוסף. בקבצי טקסט ניתן לאחסן מידע מכל סוג - מספרים, מחרוזות, או אפילו מחרוזות תווים בין מספרים. כתיבה של קבצי טקסט נותנת לנו הרבה יכולות חיוניות כשמדובר בתכנות, לכן חשוב ללמוד כיצד לעשות זאת כמה שיותר טוב.

# כיצד לעשות זאת:

```C
#include <stdio.h>

int main()
{
	// פתיחת קובץ לקריאה ולכתיבה
	FILE *file = fopen("file.txt", "w+");

	// בדיקה שהקובץ נפתח בהצלחה
	if (file != NULL)
	{
		// כתיבת מחרוזת לקובץ
		fprintf(file, "זוהי המחרוזת שכתבתי לקובץ\n");

		// כתיבת מספר לקובץ
		int num = 42;
		fprintf(file, "מספר: %d\n", num);

		// סגירת הקובץ
		fclose(file);

		printf("קובץ נוצר בהצלחה!");
	}

	return 0;
}
```

הפונקציה הכי חשובה בכתיבת קבצי טקסט היא fprintf. עם עזרתה אנחנו יכולים לכתוב לקובץ מחרוזות, מספרים ואפילו מחרוזות תווים. לאחר כתיבת המידע, חשוב לסגור את הקובץ בעזרת הפונקציה fclose כדי למנוע אובדן מידע. חשוב לזכור גם להכניס את כתובת הקובץ בביקורתת כדי לוודא שהקובץ נפתח בהצלחה.
