אחת הסיבות העיקריות להשתמש ב-C כדי לכתוב קובץ טקסט היא כדי לייצר קבצים שאפשר לקרוא ולערוך באמצעות תוכנות טקסט בעבודת יום יום. זהו דרך נוחה לאחסן ולארגן מידע שונה, במיוחד כאשר מדובר במידע שישנה על בסיס קבוע.

## איך

לכתוב קובץ טקסט ב-C אינו קשה כל כך כמו שאתה יכול לחשוב. כל מה שאתה צריך לעשות הוא לכתוב קוד פשוט שיכתוב את התוכן של הקובץ לתוך קובץ חדש. כדי להבין זאת טוב יותר, ניצור דוגמה קטנה.

\n```C
#include <stdio.h>
 
int main()
{
    FILE *fp; // משתנה לאובייקט הקובץ
    char str[100]; // מערך לאחסון המחרוזת שתתבקשו מהמשתמש
    printf("Enter a value: \n");
    fgets(str, 100, stdin); // קבל את המחרוזת מהמשתמש
    fp = fopen("example.txt", "w"); // פתח את הקובץ לכתיבה
    fprintf(fp, "%s", str); // כתוב את המחרוזת לקובץ
    fclose(fp); // סגור את הקובץ
    printf("File written successfully.\n");
    return 0;
}
\n```

בדוגמה זו, אנו פותחים קובץ חדש עבור המשתמש, מקבלים ממנו את התוכן שאנו רוצים לכתוב ושולחים אותו לכתיבה לקובץ. כדי לקרוא ולערוך את הקובץ, ניתן להשתמש בפקודות נוספות כמו קריאה וכתיבה לקובץ.
