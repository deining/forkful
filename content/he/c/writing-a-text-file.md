## למה

כתיבת קובץ טקסט בשפת סי היא חלק חשוב מאוד בתהליך התכנות. בעזרת כתיבת קובץ טקסט, אנו יכולים ליצור קשר בין המשתמש והתוכנה ולשמור את הפלט של התוכנית כדי להשתמש בו בעתיד. במאמר זה, אנו נלמד כיצד לכתוב ולקרוא קבצי טקסט בשפת סי.

## כיצד לכתוב קוד

כיצד אנו כותבים קובץ טקסט בשפת סי? ניתן לעשות זאת בעזרת פונקצייה פנימית כגון `fopen ()` שתוכנתה לפתוח קובץ עם מצב קריאה או כתיבה. למשל:

```C
FILE *file = fopen("example.txt", "w");
```

בדוגמא זו, אנו פותחים קובץ בשם "example.txt" עם מצב כתיבה (`"w"`). לאחר הפתיחה, יש לנו את המצב לכתוב לקובץ בעזרת פונקציות כמו `fprintf ()` או `fputs ()`. לדוגמא:

```C
fprintf(file, "Hello, world!");
```

זאת תדפיס את המחרוזת "Hello, world!" לתוך הקובץ המקובל.

## חפירה עמוקה

בחלק זה, נלמד לעומק כיצד מתבצעת כתיבת קובץ טקסט בשפת סי. כאשר אנו פותחים קובץ עם מצב כתיבה, רשומת הקובץ פותח ונמצאת בנקודת ההתחלה של הקובץ. כל פעם שאנו משתמשים בפונקציות כמו `fprintf ()` או `fputs ()`, הם יכתבו לקובץ במיקום האחרון שבו הגענו. בסופו של דבר, ניתן לסגור את הקובץ בעזרת הפונקציה `fclose ()` כדי להפוך את הקובץ לקובץ טקסט רגיל.
