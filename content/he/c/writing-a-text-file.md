# מה ולמה?

כתיבת קבצי טקסט היא פעולה שוהה מאוד לתכנתנים בשפת C. בהתמחות בתכנות, התכנית המנוהלת על ידי התכנתן נכתבת כתסדרת קבצי טקסט, שזהו קובץ עם טקסט פשוט שמכיל את הקוד שנכתב על ידי התכנתן.

כתיבת קבצי טקסט היא למעשה הפעולה שמתקבלת כאשר התכנתן מנסה לאחסן את הקוד שכתב במחשב 
כך שיהיה אפשר לקרוא אותו במקום אחר, עבור עריכה, או למטרות אחרות. הפעולה השוהה מאוד זו נעשית על מנת לשמור על ארגוניות ובכדי להפוך את התכנות לנוח יותר ויעיל יותר.
 
# כיצד ל:

פונקציה שנקראת `fopen` משמשת כדי לפתוח את הקובץ ולפגוש את קבצי הטקסט שמעולם לא יקראו במצב פתוח.

```C
FILE *fp;
fp = fopen("myfile.txt", "w"); 
```

בשורת הקוד התחתונה, החלק החשוב ביותר הוא לאחר המילה הראשונה "fopen", המזהה מיקום של הקובץ במחשב. המשתנה "w" מתייחס למצב הכתיבה. לאחר מכן, על מערכת ההפעלה לפתוח את הקובץ, להציג את הקבצים הפתוחים ולאפשר לתכנתן לבצע שינויים בקובץ ללא כניסה לתוך הקובץ.

מודוס הכתיבה "w" יוצר קובץ חדש אם הקובץ אינו קיים ומחק את התוכן שלו אם הקובץ קיים כבר.

# כיוון עמוק:

הכתיבה לתכונת טקסט בתוכנה המתווך מונעת כשלון בערכה בכתיבת קוד יצירתי. למרבה המזל, בשפת סי הוגדר פונקציה עולמית המאפשרת למשתמש להתמרח באופן נוח על הטכנולוגיה שלו עבור עבודת יצירה.

נוסף על כך, יש כמה אפשרויות אחרות לכתיבת קובץ טקסט, כגון שימוש בתכונות טכנולוגיות אחרות, בתורנת כל כמה שבועות תוכנות ניקיטתיות מונעות מהתכנתן לבצע שלוש שניות. בנוסף לכך, ניתן להשתמש בתמיכה בעבודה עם קבצים נוספים, שכן לכתיבת קוד טקסט ביישומון, כדאי שכולם יכתבו קוד טקסט אותו תוכנת העבודה שלכם משתמשת בו.
