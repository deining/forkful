## למה

מה המטרה של כתיבת קובץ טקסט בשפת C? התשובה המובנית היא רשימות, כיצד ניתן להשתמש בכתיבת קובץ טקסט כדי לעזור עם איבוד נתונים או ליצור קוד בן-חיים שיכול להיות משתמש ידי בעיות ספציפיות נוספות.

## כיצד לכתוב קובץ טקסט ב-C

כדי לכתוב קובץ טקסט בשפת C, יש צורך לבצע כמה צעדים פשוטים:

```
#include <stdio.h>

int main()
{
    FILE *file;
    
    // פתיחת קובץ לכתיבה
    file = fopen("example.txt", "w");
    
    // כתיבת תוכן לקובץ
    fprintf(file, "זהו דוגמה לקובץ טקסט בשפת C.");
    
    // סגירת קובץ
    fclose(file);
    
    return 0;
}
```

כאשר הקובץ נכתב באמצעות הקוד הנתון לעיל, הוא יימצא באותו תיקייה שבה נמצאת התוכנית הקומפיילר. ניתן לקרוא את תוכן הקובץ באמצעות קוד דומה, עם שימוש בפקודת `fscanf` לקריאת מידע מהקובץ.

### תכונות נוספות בכתיבת קובץ טקסט בשפת C

בנוסף לכתיבה וקריאה של תוכן לקובץ טקסט, ישנן תכונות נוספות שניתן להשתמש בהן כאשר מתמודדים עם קבצי טקסט ב-C. למשל, ניתן להשתמש בפקודות להזיזת מיקום בקובץ, למחוק או לשנות מידע קיים, וכו'. כל אלו תלויים בסוג הקובץ ובמטרתו.
