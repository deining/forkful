## למה

למה לדעת את אורך המחרוזת? ישנם רבים מסיבות שבהם עלינו למצוא את אורך המחרוזת בתכנות בשפת C. ייתכן שאנחנו זקוקים לתת תאים מיוחדים עבור מחרוזת כדי לאפשר עבודה מסוימת עמה, או ייתכן שאנחנו צריכים לולאה לעבור על כל התווים במחרוזת ולבצע פעולה על כולם. בכל מקרה, לדעת את אורך המחרוזת יכול לחסוך לנו זמן וכוחות בכתיבת קוד.

## כיצד

ניתן למצוא את אורך המחרוזת בשתי שיטות שונות: שימוש בפונקציה `strlen` מספרט טקסטואל בשם `string.h` או בעזרת לולאה פשוטה שתעבור על כל התווים במחרוזת ותספור אותם. להלן דוגמאות בשני השיטות:

```C
#include <string.h>

int main()
{
    // מחרוזת עם אורך 10 תווים
    char str1[11] = "Hello World";

    // מציאת האורך באמצעות strlen
    int length = strlen(str1);

    // הדפסת התוצאה
    printf("אורך המחרוזת הוא: %d", length);

    return 0;
}

// הפלט: אורך המחרוזת הוא: 11
```

```C
#include <stdio.h>

int main()
{
    // מחרוזת עם אורך 10 תווים
    char str2[11] = "Hello World";

    // מציאת האורך באמצעות לולאה
    int length = 0;

    // לולאה עוברת על כל התווים ומספרת אותם
    while (str2[length] != '\0')
    {
        length++;
    }

    // הדפסת התוצאה
    printf("אורך המחרוזת הוא: %d", length);

    return 0;
}

// הפלט: אורך המחרוזת הוא: 11
```
