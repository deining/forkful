# מה ולמה?

מציאת אורך של מחרוזת היא תהליך בו מחשבים כמה תווים יש במחרוזת מסוימת. זה חשוב עבור מתכנתים כי כאשר עובדים עם מחרוזות, כמו טקסטים, עלינו לדעת את האורך שלהן כדי למנוע שגיאות.

# איך לעשות את זה:

כדי למצוא את אורך המחרוזת בשפת C, נוכל להשתמש בפונקציה strlen() המובנת. נתון לנו מחרוזת כלשהי, נכתוב את הפקודה הבאה:

```C
int len = strlen("זוהי מחרוזת לדוגמה");
```

והמספר len יצביע על אורך המחרוזת, שמבחינתנו הנוח לנו להכיל 22 תווים.

ניתן גם לקבל את אורך המחרוזת הקלט מהמשתמש באמצעות פקודת scanf ולהשתמש בפונקציה strlen כדי לחשב את התווים שהמשתמש הכניס.

```C
char str[20];
scanf("%s", str);
int len = strlen(str);
printf("אורך המחרוזת הוא: %d", len);
```

בפלט נקבל את הפלט הבא:

```
> הכנס מחרוזת: שלום
אורך המחרוזת הוא: 5
```

# חפירה עמוקה:

## הקשר ההיסטורי:

בעבר, חישוב אורך המחרוזת היה מסובך למכיוון שמחשבים לא היו מבינים מבנה של מחרוזות ועבדו עם קוד ASCII בלבד. כיום בשפות תכנות כמו C ישנן פונקציות מובנות שיעשו את התחשבות בנינוי של המחרוזת עבורנו.

## אלטרנטיבות:

ישנן גישות אחרות למציאת אורך המחרוזת כמו להשתמש במשתנה מסוג מחרוזת כאורך מותאם אישית או לעקוב אחרי אורך המחרוזת בעזרת משתנה נוסף המייצג את האורך.

## פירוט היישום:

פונקציית strlen עוברת על כל תו במחרוזת ומשווה אותו לתו שאינו NULL. כשהיא מגיעה לתו NULL, היא מסתיימת ומחזירה את הערך הנתון, שמייצג את אורך המחרוזת.
