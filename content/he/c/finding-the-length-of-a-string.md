## איך לעשות זאת
האם יש אפשרות להראות את הלינקים הנפוצים לאתר האינטרנט שאליו אתם יוצאים? לא משנה אילו שיטות אתם משתמשים כדי לבדוק את האורכות צריך להתגונן מה להם. גם האם אתם לומדים את הקוד ב-מעט זה תמיד או ניקודית 1.

```C
// את זה מציגים היטב
#include <stdio.h>
#include <string.h>

int main()
{
    char string[50];
    int length;

    printf("נא כתב את המחרוזת שאתה רוצה לחפש להארכה: ");
    scanf("%s", string);

    length = strlen(string);
    printf("אורך המחרוזת שהוקשה הוא %d", length);

    return 0;
}
```

```
מן הסתם, יש אפשרות להשתמש גם בפקודה הדבוקה:
length = 0;
while (string[length] != '\0')
{
    length++;
}
```
