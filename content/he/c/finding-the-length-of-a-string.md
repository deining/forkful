תמיד היה מעניין לדעת כמה אותיות יש במחרוזת כלשהי, אבל עם שפת תכנות כמו סי, אנו יכולים להכניס זאת לפעולה בקלות. במאמר זה נלמד כיצד למצוא את אורך המחרוזת בסי ונשקול גם כמה עומק נוסף על הנושא.

##עלילה

כאמור, החשיבה על מספר האותיות במחרוזת כלשהי יכולה להיות מעניינת בפני עצמה, איך אנו מגיעים להתשוממות, כמובן, היא תלויה במידת הטכנולוגיות כמו סי. בנוסף, המצאת אורך המחרוזת יכולה להיות שימושית כאשר אנו עובדים עם טקסטים בשפת תכנות כמו סי, כמו לדוגמה עיבוד מחרוזת כדי לבצע כל מיני פעולות אחרות עליה.

##כיצד לעשות זאת

למצוא את אורך המחרוזת בסי היא תרגיל פשוט מאוד, כאשר אתם מבינים את מבנה השפה. נדגים זאת באמצעות דוגמאות קוד פשוטות.

```C
#include <stdio.h>
#include <string.h>

int main()
{
    char str[] = "שלום עולם";

    // finding length using strlen function
    int length = strlen(str);
    printf("אורך המחרוזת הוא: %d \n", length);

    // finding length without strlen function
    int count = 0;
    while (str[count] != '\0')
        ++count;
    printf("אורך המחרוזת הוא: %d", count);

    return 0;
}
```

כפי שאתם רואים, אנו משתמשים בפונקציית "strlen" המובנית של סימני לכדי למצוא את אורך המחרוזת, או פשוט עוברים על המחרוזת עם לולאת while ומשתמשים במונה כדי לספור כמה פעמים יש לנו את התו הסיום המיוחד '\0'.
