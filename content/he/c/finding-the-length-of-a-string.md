# מדוע
 למה שהמילים מכילות כמות מסוימת של תוים, זה חיוני לדעת את האורך של מחרוזת בתכנות כי בלי זה, אין אפשרות לטפל בנתונים כהרצוננו. בנוסף, כשמילים נמצאות באורך משתנה, יהיה צורך לטפל בלולאות וביצירת ביטויים תנאיים שנכילבהמשך הפוסט.

# כיצד לעשות זאת
על מנת למצוא את אורך המחרוזת בשפת סי, נשתמש בפונקציית `strlen` הקיימת בספריית המערכת הסטנדרטית (stdio). נמצא דוגמא לשימוש בפונקציה זו להלן:
```c
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "שלום עולם"; // מחרוזת כלשהי באורך משתנה
    int length = strlen(str); // משתנה חדש בשם length שהוא התוצאה של הפונקציה
    printf("אורך המחרוזת הוא: %d", length); // הדפסת התוצאה למסך
    return 0;
}
```
כאן, אנו משתמשים במשתנה חדש בשם `length` שמכיל את תוצאת הפונקציה `strlen` על המחרוזת `str` שהוגדרה כמערך תווים. ניתן להשתמש במשתנה `length` כדי לבצע פעולות נוספות עם המחרוזת, כמו לדוגמה חיתוך או איחסון. הפלט של הקוד הנ"ל יהיה:
```
אורך המחרוזת הוא: 10
```

# עיון מעמיק: נסיגת משתנה מסוג `char *`
כאשר אנו משתמשים בפונקציה `strlen` כפי שיציג הדוגמא, אנו מקבלים תוצאה מסוג `int` ולא `char *` כמו מחרוזת עצמה. מחרוזת בשפת סי מיוצגת כאורך משוערך של תוים והתו `\0` המייצג את סיום המחרוזת. כשנגדיר מחרוזת כמערך תווים, ייוצר מקום זיכר
