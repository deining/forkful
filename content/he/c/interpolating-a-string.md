## מה ולמה?
אינטרפולציה של מחרוזת היא פעולה שמשתמשים בה כדי להחליף משתנים או ערכים נתונים בתוך מחרוזות. תהליך זה מאפשר לנו ליצור מחרוזות דינמיות ומאוחסנות בתוך קוד התכנית. תהליך זה נקרא "אינטרפולציה" מכיוון שהוא משתמש במחרוזת כבין, או ממסיל עבור המידע בקוד.

## כיצד:
דוגמאות קוד לאינטרפולציה של מחרוזות ופלט דוגמאות:

```C
#include <stdio.h>

int main() {
  int num = 5;
  char name[20] = "יוסף";
  char sentence[40];
  
  sprintf(sentence, "המספר שלי הוא %d ושמי הוא %s.", num, name);
  printf("%s", sentence);
  
  return 0;
}
```

פלט:
```
המספר שלי הוא 5 ושמי הוא יוסף.
```

בדוגמה הזו, אנחנו משתמשים בפונקציות ```sprintf``` ו-```printf``` כדי למלא ערכים נתונים בתוך מחרוזות.

## צריכת תכנית:
תהליך האינטרפולציה נולד כתוצאה מהצורך ליצירת מחרוזות דינמיות בשפת C. בעבר, תהליך זה היה קצת מסורבל ודורש המון קוד כדי ליצור מחרוזות יעילות. עם השנים, מומחי C פיתחו פונקציות כמו ```sprintf``` ו-```printf``` כדי לפשט ולקצר את תהליך האינטרפולציה.
