## למה

יצירת קובץ זמני בתכנות C נדרשת כשאנו רוצים לשמור נתונים במקום זמני ולא רציני, או לבצע פעולות מסוימות על קבצים קיימים. זה יכול להיות שימושי במקרים של בדיקות, מחקר או כל פעולה שמצריכה שימוש בקובץ זמני.

## איך לעשות זאת

קוד הדגמה הבא מראה כיצד ליצור קובץ זמני בתכנות C:

```C
#include <stdio.h>
#include <stdlib.h>

int main()
{
    // יצירת קובץ זמני בעזרת fopen והגדרת האופציות הנדרשות
    FILE *fp = fopen("temp.txt", "w+");
    
    if(fp == NULL) // בדיקה האם הקובץ נוצר בהצלחה
    {
        printf("לא ניתן ליצור קובץ זמני\n");
        exit(1);
    }
    
    // מבצע פעולות על קובץ הזמני...
    
    // סגירת הקובץ ומחיקתו מהמערכת
    fclose(fp);
    remove("temp.txt");
    
    return 0;
}
```

תוצאת הרצה של הקוד הנל תהיה:

```
לא ניתן ליצור קובץ זמני
```

הסיבה שהקובץ לא נוצר היא בגלל שתנאי ה- if לא נתקן. נוכל לתת שם או כתובת שונים לקובץ הזמני על מנת שהוא יתווסף למערכת.

## מידע עמוק

כשאנו משתמשים בפונקציה fopen כדי ליצור קובץ זמני, אנו יכולים להשתמש באופציות שונות כדי להגדיר את התנאים של הקובץ. בקוד הדגמה הנל נעשה שימוש באופציות "w+" שיצור את הקובץ גם אם הוא לא קיים ויאפשר לנו לבצע כתיבה וקריאה לקובץ. אופציות אחרות כוללות "a+" שתשמור את הנתונים בקבצים אחרים, "r+" שתאפשר גישה לקבצים קיימים וכו'.
