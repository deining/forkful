מה ולמה?

קריאת ארגומנטים מפקודת השורת הפקודה היא תהליך שבו מתכנתים קוראים ומשתמשים בפרמטרים שסופקו בזמן הפעלת התוכנית. מתכנתים עושים זאת כדי לקבל מידע מותאם אישית ולדאוג לתפיסת ההוראות בזות של קלות וינותן את התוכנית הקלה עד כמה שאפשר.

איך לעשות?

כדי לקרוא ארגומנטים מפקודת השורת הפקודה ב- C, יש להשתמש בפונקציית main עם משתנה argc המייצג את מספר הארגומנטים ומערך דו-ממדי של ארגומנטים המייצגים את הערכים האמיתיים. ניתן להדפיס את הארגומנטים באמצעות לולאת for ומשתנה i המתאים לאינדקס של המערך. כדי להדפיס תוצאה מותאמת אישית, ניתן להשתמש בפונקציית printf.

צפירה עמוקה

בעבר, קריאת ארגומנטים מפקודת השורת הפקודה נעשתה באמצעות פרמטרי הקלדת ערכים שהיו נמצאים בתחילת קובץ ההרצת. כיום, נעשית זאת באמצעות מזהים אחת למערך שהמשתמש יכול לקבל את ערכו על ידי שימוש במערכת אופציות פקודה. פרמטרי פקודה אלו יכולים להיות גם אופציונליים וגם רנדומליים וטסקים המשפיעים על מצב הרצתו של התוכנית.

תראו גם

לפרטים נוספים על קריאת ארגומנטים מפקודת השורת הפקודה ניתן לבדוק את המקורות הבאים:

- עמוד מדריך לקריאת ארגומנטים מפקודת השורת הפקודה בקורס C של Code Academy
- מדריך מעמיק בנושא קריאת ארגומנטים מפקודת השורת הפקודה באתר GeeksforGeeks
