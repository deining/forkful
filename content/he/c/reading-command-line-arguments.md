## מדוע

בתכנות בשפת C, כדי ליצור תוכניות התקנה או שימושיות נוספות, ייתכן שנצטרך להעביר פרמטרים לתוכנית מהשורת פקודה. נקרא לנסות את הפקודה עם המונחים שונים, כדי לראות איך הוא מתנהג עם פרמטרים שונים, ואם צריך לעשות שינויים לפי הצורך.

## איך לעשות זאת

בפקודת הטרמינל, הנחם כמה פרמטרים אתה רוצה להעביר לתוכנית. הם יתווספו מידי שנותן הפקודה מתקבלת על ידי הפתרון לשימוש כך: `ארגונית main (int argc, להבין הרשימה של ארגומנה השגרירה (ארגוני **)”גר ()בנפרד.

כדי ליצור את תוכנת הפקודה הזו:

```C
#include <stdio.h>

// הפונקציה הראשית
int main(int argc, char *argv[]) {
    // הדפסת ערכי הפרמטרים שהועברו לתוכנית
    for (int i = 1; i < argc; i++) {
        printf("הפרמטר ה-%d הוא: %s\n", i, argv[i]);
    }
    
    // משהו יםערה מושהה לקלוט פרמטר נוסף
    char input[50];
    printf("יש לך עוד פרמטר שרצית להעביר? ");
    scanf("%s", input);
    printf("הפרמטר העברת שהוא: %s\n", input);
    
    return 0;
}
```

כדי להריץ תכנית זו מהשורת הפקודה, יש להכנס הפרמטרים לאחר הפקודה:

```bash
./program param1 param2 param3
```

תוצאות התכנית יהיו:

```
הפרמטר ה-1 הוא: param1
הפרמטר ה-2 הוא: param2
הפרמטר ה-3 הוא: param3
יש לך עוד פרמטר שרצית להעביר? lastparam
הנך הפרמטר העברת שהוא: lastparam
```
