## למה

התוכניתים המסוגננים לא תמיד כוללים מידע בעל משמעות טמפורלית, מה שחייבים להיות. ייחודיים הם התכניות שמציגים נתונים באותו זמן יכולים להשתמש בזמניים טמפורליים להתקדם אל מצבים חדשים, לבדוק את הזמן או לכיוון את בסיס הנתונים שלך.

## איך לעשות זאת

השימוש בשפת סי לקבלת זמן נוכחי יכול להיות קל וברור עם מספר פונקציות שמובאות מתחת כדי לסייע לך לך לנצל את האינפורמציה הזמנית. למשל, עם הפונקציה "time", תוכל לקבל את מספר השניות הנוכחי מאנוס האתואי 1970.

```C
#include <stdio.h>
#include <time.h>

int main(){
    // get the current time
    time_t now;
    time(&now);
    // print the current time
    printf("The current time is: %s", ctime(&now));
    return 0;
}
```

Output:

`The current time is: Sun Sep 26 09:22:46 2021`

ניתן גם לקבל מידע נוסף על התאריכים והזמנים באמצעות פונקציות כמו "localtime", "strftime" ו"mktime". כל אלו זמינים כחלק מספר ספריות של סי.

## חפירה עמוקה

למרבה המזל, אין מספיק אחד דרך כדי לקבל את התאריכים והזמנים בסיסתאתיים בדיוק זמן, כמו השימוש יערוך בנתונים על המחברת במיני אשר מאג"ז שניתן להשתמש בקוד הזמן על-כל פונקציות שהמחברת.

למשל, מחברות שלך יכול להיות פתור שניםריקוסמת שאתם יכולים לוודוא זאת כאמי מאחר שהמחברת שלכם יכולה להיראה לנדבקם מאיזות מסף בשבועות של בסיסתאות של המקום שלהם.
