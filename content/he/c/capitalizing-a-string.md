במדור זה נתמקד בכתיבת קוד בשפת C כדי להבין כיצד לבצע את תהליך הכתיבה המספקת של שרשור. המאמר הזה מיועד לכל מי שרוצה ללמוד כיצד לכתוב קוד נקי ויעיל בעזרת הפעולה הפשוטה של כתיבת הכיתובית.

## למה

כשמתייחסים לשרשור, אולי נתקלנו בכך שאותיות מסוימות היו כתובות באופן רנדומלי, ולאחר מכן נאלצנו לשנות אותן ידנית לאותיות גדולות. כידוע, כתיבת כל האותיות באות גדולה או באות קטנה יכולה להיות חייבת עבור המשתמשים. למרבה המזל, קיימת פונקציה פשוטה כדי לעזור לנו להבין כיצד לבצע כתיבת שרשור בצורה המושלמת.

## איך לבצע

כדי לבצע כתיבת שרשור, נשתמש בפונקציית `toupper()` חיצונית, המשמשת להעברת מחרוזות לאותיות גדולות. הנה דוגמה לכתיבת הכיתובית בצורה מסודרת:

```C
#include <stdio.h>
#include <ctype.h>  // להוסיף סיפריות נלווה

int main() {
    char str[] = "כתיבת שרשור פשוטה של C programming";
    for (int i = 0; str[i]; i++) {
        printf("%c", toupper(str[i])); // הדפסת כיתובית בכתוב אותיות גדולות
    }
    return 0;
}
```

כתיבת שרשור פשוטה של C PROGRAMMING
