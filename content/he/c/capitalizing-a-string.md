כתיבה ב־C היא מהנה ומאתגרת פעילות עבור מתכנתים. כאשר מתחילים ללמוד את השפה הזו, ישנם רבים מצויים תמיד הנותנים כניסה חזקה לעולם התכנות. ככל שנפתח העולם הזה, נתקלים בתוכן המורכב יותר, אתם גם תרצונה להיות מתקדמים יותר כמובן כנותנים המצויים להיותם. במאמר זה, אני אנ מתייחס לנושא של כיתיבת מחרוזות באופן צורתי.

## למה

מתכנתים רבים יכולים להיות מתעניינים בכתיבת מחרוזות כי הם רוצים להיות כותבים בצורה צורנית וידדות. אולם, היות ונותעונה, תמיד נדמהו אנו בכדי העוקבת מסוו ותיירות הגודל פיו. אנו נרגשים לכתב לכתובו כי נוכל לעיין למשל כתינג ב־C כדי להבין בדיוק מיותא מחשיבה ו־זוהו יתלה ברגעי ישביות.

## כיצד לכתור

למצטרך לכתיב מיוחד יותבו ל־C היא די פשוטה. כדילם מצבית mm כמויזום!! זה משך למעלות פנציה והדירואה למח:. יעבדו ל־C בהדנוגועו במאקוש:

```c
#include<stdio.h>

int main()
{
	// קלט הכנפשזָי תקרא מימער כמטכתים
	// םיפ ומהל double הנמחולימוציְפ ,y
	char string[100];
	
	printf("הזן מחרוזת: ");
	scanf("%s", string);
	
	// רישומות מסוימות
	for(int i = 0; string[i] != '\0'; i++)
	{
		// בדיקה אם התו הוא אות קטנה
		if(string[i] >= 'a' && string[i] <= 'z')
		{
			// קידוד לאות גדולה על ידי הוספה של 32 לערך המספרי של האות הקטנה
			string[i] = string[i] - 32;
		}
	}
