## מה ולמה?
הגדלת רצף היא תהליך שבו משתנה טקסט נכתב באותיות קטנות לאותיות גדולות. הכוונה המרכזית של הגדלת רצף בתכנות היא להפוך לקל יותר לזהות, לקרוא ולעבוד עם טקסט שמכיל מילים באותיות גדולות.

## כיצד לבצע:
כדי להגדיל רצף בפורמט השקף, ניתן להשתמש בפונקציה "toupper" המקבלת כפרמטר את התו ומחזירה את התו באותיות גדולות. הפונקציה זמינה במאגר הסטנדרטי של השפת C.

```C
char my_string[] = "hello world";
for (int i = 0; i < strlen(my_string); i++) {
    my_string[i] = toupper(my_string[i]);
}
printf("%s", my_string);
```

התוצאה של הקוד היא:
HELLO WORLD

## חקירה מעמיקה:
המקורות המקובלים של השפה C מאפשרים למפתחים לכתוב קוד באותיות גדולות או קטנות. למרבה המזל, בתור בנל קוד, אין צורך בחקירה מיוחדת על התהליך של הגדלת רצף. למימושים נוספים ניתן להשתמש בפונקציות כגון "I8N" לתמיכה בתווים מיוחדים ולשמירה הסטנדרטית של הטקסט שלנו.

## ראו גם:
ללמוד עוד על השפה C ניתן לעיין במדריכים ובמקורות המקוונים המקוריים. ניתן למצוא רשימה של טקסטים לפני עצמך על גדלת רצף יהיו זמינים לפניך למשל דף האינטרנט שלנו רגל.
