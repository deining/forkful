## למה
כתיבת תווים בראש פעל הנק"ז המתאימים. בכתיבת התווים ניתן להפוך אותם לתווים ראשוניים בכדי להראות את הטקסט בצורה מודגשת יותר וקריאה יותר.

## איך לעשות
השתמשו בפונקציה הכתיבה ```toupper()``` על כל אות ראשונה של המחרוזת ועדכנו את האינדקס של האות באמצעות לולאת ```for```. לדוגמה:
```C
void capitalize_string(char string[]) {
    // השתמש בלולאת for כדי לעבור על כל האותיות
    for (int i = 0; string[i] != '\0'; i++) {
        // השתמש בפונקציה toupper כדי להמיר את האות לאות ראשונה ולעדכן את המחרוזת
        string[i] = toupper(string[i]);
    }
}
```
כעת תוכלו לקרוא לפונקציה זו ולהעביר אליה את המחרוזת שברצונכם לכתוב בכתיבה ראשונה.

פלט דוגמה:
```C
// הגדרת המחרוזת
char example[] = "this is a sample string!";

// קראו לפונקציה כדי לעדכן את המחרוזת לכתיבה ראשונה
capitalize_string(example);

// הדפיסו את המחרוזת המעודכנת
printf("%s", example);
```

פלט:
```
This is a sample string!
```

## עומק
ישנן פונקציות מובנות נוספות בשפת C שמאפשרות לנו לשנות את תווי המחרוזות בדרכים שונות. לדוגמה, הפונקציה ```tolower()``` מאפשרת להמיר תווים לכתיבה קטנה במקום לכתיבה ראשונה. כמו כן, ניתן להשתמש במנגנון "ביטוי של פונקציה" (function pointer) כדי להעביר את הפונקציה לפונקציה אחרת כדי לבצע פעולות מתקדמות יותר על תווי המחרוזת. מומלץ לחקור עוד על נושא זה כדי לשפר את יכולות הכתיבה בשפת C.
