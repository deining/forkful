## למה

למה לתפוס את המחרוזת? אנשים משתמשים בפעולה זו כדי לטפל במחרוזות שהופכות להיות מאוד גדולות וקשות לקריאה. על ידי כיווץ האותיות לתצוגה ראוייה, פעולת תפוס מאפשרת למשתמשים לקרוא או לטפל במחרוזות גדולות יותר בצורה נוחה.

## איך לפעול

כאשר נתנו מחרוזת כארגומנט לפונקציה המכתבת, נטפס את האות הכנס למחרוזת. נראה כאשר משתמשים בלולאה מחרידה תוצאה זה בשורה צורה קו ושלוח אותו למכתביות בית הכנסת :

```C
#include <stdio.h>

int main()
{
  // תפוס את המחרוזת
  char string[] = "שלום לעולם";
  
  // ללולאת את כל האותיות במחרוזת ולהשתמש בפונקציה להפוך לאותיות גדולות
  for(int i=0; i < strlen(string); i++)
  {
    string[i] = toupper(string[i]);
  }
  
  // עכשיו נדפיס את המחרוזת באותיות גדולות
  printf("המחרוזת כעת נרכסת הוא: %s", string);
  
  return 0;
}
```

הפלט הצפוי הוא: "המחרוזת כעת מאותיותוג'יס הוא: שלום לעולם"
