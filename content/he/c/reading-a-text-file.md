## מדוע

קריאת קבצי טקסט בתוכנית בשפת C נחשבת לשיטה חשובה וקריטית להתמודדות עם מידע טקסטואלי. באמצעות זה ניתן לממש פונקציונליות ייחודית וחזקה, כגון תפעול ועיבוד של מידע, כתיבה וקריאה של קבצים, חיפוש ותיעוד של נתונים ועוד. כמו כן, קריאת קובץ טקסט גם מאפשרת לנו לשמור נתונים שונים בקובץ חיצוני ולהשתמש בהם בתוכנית בצורה נוחה ויעילה.

## איך לעשות זאת

הנה מספר דוגמאות קלות של קוד C שמיועדות לקריאת קובץ טקסט ועיבוד הנתונים שבו:

```C
#include <stdio.h>

int main() {
    // פתיחת קובץ לקריאה
    FILE *fp = fopen("file.txt", "r");
    
    // הגדרת מספרים לעיבוד הנתונים הקרובים מהקובץ
    int num1, num2, sum;
    
    // קריאת המספרים מהקובץ והחשבת סכוםם
    fscanf(fp, "%d %d", &num1, &num2);
    sum = num1 + num2;
    
    // סגירת הקובץ
    fclose(fp);
    
    // הדפסת הסכום החדש שנמצא
    printf("הסכום הוא: %d", sum);
    return 0;
}
```

בתוכנית זו, אנו משתמשים בפונקציות המיועדות לפתיחת קובץ, לקריאת נתונים מתוכו ולסגירתו. בנוסף, אנו משתמשים במשתנים לאחסון הנתונים ובפעולות חשבון פשוטות כדי לעבד אותם. להלן תוצאת הרצת התוכנית מול קובץ הטקסט "file.txt" שהינו קובץ עם שני מספרים, אחד בכל שורה:

```
הסכום הוא: 20
```
