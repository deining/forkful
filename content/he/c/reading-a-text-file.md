## למה

קריאת קובץ טקסט היא פעולה נפוצה בתכנות בשפת C. קריאת קובץ מאפשרת לנו לקרוא מידע מקובץ ולהשתמש בו בתוכנית שלנו. בכתבה הזו נלמד כיצד לקרוא קובץ טקסט ב-C ולהציג את התוכן שלו.

## איך לעשות זאת

כדי לקרוא קובץ נשתמש בפונקציית fopen שתפתח את הקובץ ותחזיר לנו מצביע למבנה של FILE. הפונקציה fopen מקבלת שני פרמטרים - שם הקובץ ומצב קריאה או כתיבה. התפתחות הקובץ נעשית באמצעות הפונקציה fgets שתקרא את התוכן של הקובץ ותכניס אותו למערך של char. לאחר מכן נדפיס את המערך באמצעות הפונקציה printf.

```C
#include <stdio.h>
 
int main()
{
   FILE *file = fopen("myfile.txt", "r");
   if(file == NULL)
   {
      printf("לא ניתן לפתוח את הקובץ");
      return -1;
   }
 
   char line[255];
   while(fgets(line, sizeof(line), file))
   {
      printf("%s", line);
   }
 
   fclose(file);
   return 0;
}
```

הפלט של הקוד הנ"ל יהיה:

```
זה השורה הראשונה של הקובץ.
זה השורה השנייה של הקובץ.
זה השורה השלישית של הקובץ.
...
```

## חפירה עמוקה

כאשר אנחנו קוראים קובץ טקסט ב-C, המידע נכנס למערך של char. ניתן לקרוא את התוכן של המערך ולהשתמש בו בכל דרך שנרצה בתוכנית שלנו. כמו כן, ניתן להשתמש בפונקציית fscanf לקריאת נתונים ספציפיים מהקובץ במקום לקרוא את כל התוכן.
