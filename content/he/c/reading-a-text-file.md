## מה ולמה?
קריאת קובץ טקסט היא פעולה נפוצה בעולם התכנות. זהו תהליך שבו התוכנית שלנו קוראת ומקבלת מידע מתוך קובץ שנמצא במחשב. תהליך זה מאפשר לנו לעבור על מידע גדול בצורה מהירה ויעילה. תכניות המכילות הרבה נתונים כמו מילות מחפש או נתונים סטטיסטיים, יכולות להשתמש בקריאת קבצי טקסט על מנת להתחיל לעבוד עם המידע הרלוונטי.

## כיצד לבצע קריאה של קובץ טקסט בשפת C:
האופן הכי נפוץ לבצע קריאה של קבצי טקסט בשפת C הוא באמצעות הפונקציה fgets. פונקציה זו מאפשרת לנו לקרוא שורה אחת מתוך קובץ טקסט בכל פעם ולשמור אותה במשתנה בתוך התוכנית שלנו. הנה דוגמה לשימוש בפונקציה fgets:

```
#include <stdio.h>

int main(){
    char line[100]; // משתנה המכיל את השורה הנוכחית שנקראה

    FILE *fp = fopen("myfile.txt", "r"); // פתיחת הקובץ לקריאה
    if(fp == NULL){ // בדיקה אם לא ניתן לפתוח את הקובץ
        printf("Error opening file");
        return 1;
    }

    while(fgets(line, 100, fp) != NULL){ // קריאת שורה מהקובץ כל פעם עד לסוף הקובץ
        printf("%s", line); // הדפסת השורה שנקראה
    }

    fclose(fp); // סגירת הקובץ
    return 0;
}
```
פלט לדוגמה זו יהיה המלל שנמצא בקובץ המראה במסך.

בכדי לממש קריאת קובץ טקסט בשפת C, המחשב יצטרך לבצע המרת המידע מטקסט לפורמט שבו התוכנית יכולה לעבד את הנתונים. כמו כן, חשוב לבדוק שהקובץ נמצא בנתיב הנכון ושיש הרשאות לקרוא אותו.
