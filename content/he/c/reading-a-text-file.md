## למה
ישנם מספר סיבות שיכולות להביא מישהו לקרוא קובץ טקסט בתוכנית שנכתבת בשפת C. החשיבה הראשונה שעולה ברוח היא שהכלי הזה נצרך על מנת לעבד מידע מתוך הקובץ בצורה מבוקרת וקלה לנתיב המשתמש. עלולות להיות גם צורך להעתיק או לעיבד את המידע הכתוב בקובץ על מנת להוציא נתונים מועילים ולנהל דטהסט טוב יותר.

## איך לעשות זאת
לכתוב קוד המאפשר קריאה של קובץ טקסט בצורה תקינה ויעילה, ניתן לעקוב אחר השלבים הבאים:
```C
#include <stdio.h>
int main() {
    // פתיחת הקובץ לקריאה בעזרת הפקודה fopen()
    FILE *file_pointer = fopen("file.txt", "r");

    // בדיקה האם הקובץ נפתח בהצלחה
    if(file_pointer == NULL) {
        printf("לא ניתן לפתוח את הקובץ.");
        return 1;
    }

    // קריאה של כל שורת הטקסט בקובץ ע"י ריצת לולאה בעזרת הפקודה fgets()
    char text[100];
    while(fgets(text, 100, file_pointer) != NULL) {
        printf("%s", text);
    }

    // סגירת הקובץ בעזרת הפקודה fclose()
    fclose(file_pointer);

    return 0;
}
```

## חפירה עמוקה
כדי לקרוא קובץ בצורה מוקדמת יותר ומותאמת יותר לצרכי התוכנית שנכתבת, ניתן להשתמש בפונקציות נוספות כמו fscanf() או gets(). כמו כן, חשוב לתת תשומת לב לניהול שגיאות וטיפול בהן באמצעות הפקודה perror() ושימוש בתנאי if לבדיקת תקינות התהליך.
