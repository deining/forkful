## למה

משהו נפלא בתכנות הוא שקיימים מיליונים של דרכים לעשות את אותה עבודה. במילים אחרות, קיימים פתרונות מגוונים לבעיות תכנותיות. אחד מפתרונות אלו הכוונים לחלוטין הן "לחתוך" או להפריד מחרוזת למחרוזות קטנות יותר. זהו תהליך שבשפת תכנות C ידוע כ $מחרוזות קטנות$(substrings) והוא מאפשר שימושים רבים ושונים בתכניות שלנו.

## איך לעשות זאת

אם אינך מכיר פתרונות לחלוטין, תיתכן הסתברות גבוהה שלא השתמשת בדרך $חיתוך מחרוזת$(string slicing) לפני. אז כאן נמצא דוגמא לדרך לעשות זאת בשפת תכנות C:

```C
// קוד לחיתוך מחרוזת בשפת C
#include <stdio.h>

int main(void) {
    // הכנסת מחרוזת למשתנה
    char str[] = "שלום לעולם!";
    
    // ביצוע חיתוך והדפסה
    printf("חלק המחרוזת הראשונה: %s\n", &str[0]);
    printf("חלק המחרוזת השני: %s\n", &str[6]);
    
    return 0;
}

// פלט:
// חלק המחרוזת הראשונה: שלום
// חלק המחרוזת השני: לעולם!
```

מתוך הקוד הנ"ל אפשר לראות שהמחרוזת נמצאת במשתנה `str` ואנו משתמשים בקידום אינדקס כדי לחלץ את המחרוזת הרצויה. כמו כן, ניתן גם לשנות את האינדקס בתוך ה`printf` לפי המחרוזת הרצויה.
