# מה ולמה?

וואו, כמה כיף לשחק עם מחרוזות ב־C! אז מה אנחנו עושים כשאנחנו מדברים על "חילוץ מחרוזות"? בקיצור, חילוץ מחרוזות הוא פשוט לקבל חלק מהמחרוזת ולהשתמש בו בנפרד. אנחנו חולצים חלקים מייחודים מהמחרוזת המקורית ומשתמשים בהם כרצוננו. התוצאה היא שאנחנו יכולים לעבוד עם המחרוזת בצורה יותר מאורגנת ומתוקנת.

אז למה תוכניותיסטים נהנים מחילוץ מחרוזות? בעיקר כי הוא מאפשר לנו לשנות או לעבוד עם חלקים שונים של המחרוזת ולהתאים אותם לצרכים שלנו. ניתן להשתמש בחילוץ מחרוזות כדי לעבוד עם תווים מסוימים, לבצע חיפושים מתקדמים או אפילו לבנות תבניות למחרוזות.

# איך לעשות זאת:

הנה כמה דוגמאות לאיך לחצוץ מחרוזות בקוד C ואיך התוצאה תיראה:

&#96&#96&#96C
char str[20] = "Hello world";
char substr[6];
int i;

// חלוץ מחרוזת ממקומות 2 עד 6
for (i = 2; i < 7; i++) {
  substr[i-2] = str[i];
}

// מדפיס את החלק הנחלץ של המחרוזת
printf("The substring is: %s", substr);

// תוצאה יישורצבת היא "llo w"

&#96&#96&#96

אם כבר נדבר על איך לחצוץ מחרוזות, למה לא נבדוק גם איך לחלק מחרוזות עם פקודת "strtok":

&#96&#96&#96C
char str[20] = "Hello world";
char *token;

// חלץ את החלק הראשון מבין התווים הרווחים
token = strtok(str, " ");

while (token != NULL) {
  // print the extracted substring
  printf("%s\n", token);
  token = strtok(NULL, " ");
}

// תוצאה יישורצבת היא "Hello" ואז "world"

&#96&#96&#96

# טיפול מקיף:
