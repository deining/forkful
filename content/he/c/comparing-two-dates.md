## למה

אנשים משתמשים בהשוואת שתי תאריכים במטרה לבדוק היכן יש הבדל בין שתי תאריכים, לדוגמה כאשר משתמשים באלגוריתם לחישוב גיל.

## כיצד לבצע זאת

הגרסה הנוכחית של שפת תכנות C מספקת שיטות שונות להשוואת שתי תאריכים, בין אם מדובר בתאריכים שמיוצגים כמחרוזות או על ידי פעולות חשבוניות על הקוד המספרי של התאריך. להלן כמה דוגמאות לכתיבת קוד והפלט המשוער בכל דוגמה:

```C
// השוואת שני תאריכים מיוצגים כמחרוזות
char date1[] = "20/05/2020";
char date2[] = "19/05/2020";
int difference = strcmp(date1, date2); // יחזיר 1 כאשר התאריכים שונים ו-0 כאשר התאריכים זהים
if (difference > 0) {
    printf("התאריך %s יגדיל את התאריך %s\n", date1, date2);
} else if (difference < 0) {
    printf("התאריך %s יגדיל את התאריך %s\n", date2, date1);
} else {
    printf("שני התאריכים זהים\n");
}
// הפלט: התאריך 20/05/2020 פחות מהתאריך 19/05/2020
```

```C
// השוואת שני תאריכים על ידי פעולות חשבוניות על התאריך המספרי
int date1 = 20200520;
int date2 = 20200519;
if (date1 == date2) {
    printf("שני התאריכים זהים\n");
} else if (date1 > date2) {
    printf("התאריך הראשון יגדיל את התאריך השני\n");
} else {
    printf("התאריך השני יגדיל את התאריך הראשון\n");
}
// הפלט: התאריך הראשון יגדיל את התאריך השני
```
