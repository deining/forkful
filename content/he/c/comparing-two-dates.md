#למה

השוואת שתי תאריכים היא כלי חשוב בתכנות בשפת C כאשר נרצה לבצע שוואה בין תאריכים להחליט כמי גדול מבין שני התאריכים.

###איך לבצע שוואה בין שני תאריכים
ניתן להשוות בין שני תאריכים באמצעות הפונקציה strcmp המובנית בשפת C. ניתן להשתמש בפונקציה זו על ידי העברת שני התאריכים כפרמטרים והשאר תלוי בדרישות התוכנית.

אם התאריך הראשון גדול יותר מהתאריך השני, הפונקציה תחזיר מספר חיובי. אם התאריך השני גדול יותר מהתאריך הראשון, הפונקציה תחזיר מספר שלילי. ואם התאריכים שווים, הפונקציה תחזיר את התו 0.

```C
// דוגמה לשוואה בין שני תאריכים
#include <stdio.h>
#include <string.h>

int main() {
   char date1[10] = "01/01/2021";
   char date2[10] = "01/05/2021";

   int result = strcmp(date1, date2);
   if(result > 0) {
      printf("date1 גדול יותר");
   } else if(result < 0) {
      printf("date2 גדול יותר");
   } else {
      printf("התאריכים שווים");
   }

   return 0;
}
```
בפלט של הדוגמה נקבל:
```
date2 גדול יותר
```

###עומק ההשוואה בין שני תאריכים
השוואת תאריכים בשפת C היא לא פשוטה כמו להשוות מחרוזתים רגילות. כאשר משווים שתי תאריכים, נדרש להתחשב במבנה של התאריך ולוודא שהשוואה תתאים לצורך שלנו.

לדוגמה, נרצה להשוות שני תאריכים בפורמט dd/mm/yyyy. אם נעשה שוואה בפשטיות על פני התאריכים, יתכן שנקבל תוצאות לא נכונות כיוון שהפורמט שלהם אינו תואם את המבנה של התאריכים המבוקש
