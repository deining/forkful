## למה

למה למישהו למחוק תווים התואמים תבנית?

למחוק תווים התואמים תבנית יכול להיות נדרש כאשר אנחנו רוצים לנקות קלט מיותר או פשוט לפנות מקום בזיכרון בטיפול במחרוזות.

## איך לעשות

לדוגמה, אם אנו רוצים למחוק את כל התווים האי לטיניים ממחרוזת מסוימת, ניתן לעשות זאת באמצעות הפונקציה `isalpha()` ולסמן את התווים שאנחנו רוצים למחוק בעזרת תנאי `if`.

```C
// דוגמה למחיקת תווים האי לטיניים
#include <stdio.h>
#include <string.h>
#include <ctype.h>

int main()
{
    char str[] = "צאפני קוד";
    int i, j = 0;
    
    // מעבר על כל התווים במחרוזת ובדיקה אם הם הם האי לטיניים התואמים
    for (i = 0; str[i] != '\0'; i++)
    {
        if (isalpha(str[i]) == 0)
        {
            // אם התו אינו אי לטיני, נתון לפקודת מחיקה באמצעות התרחיש הזה
            j++;
        }
        // פקודת העתקה שתעביר את התווים הנותרים למיקומים המתאימים במערך
        str[i - j] = str[i];
    }
    // שינוי הפונקציות השניות של תיבת הטקסט למחרוזת הנקייה והדפסתה
    str[i - j] = '\0';
    printf("%s\n", str);
    
    return 0;
}

// פלט: צאפני
```

## חקירה מעמיקה

במאמר זה למדנו כיצד למחוק תווים התואמים תבנית באמצעות פקודת `isalpha()` ותנאי `if`. אבל ישנן עוד דרכים להתמודד עם מחרוזות ולמחוק את התווים השונים הנתונים. לדוגמה, ניתן להשתמש בפקודת `strtok()` כדי לפרק את המחרוזת למחרוזות קטנות יותר ולמחוק את התווים הלא רצויים מהן.
