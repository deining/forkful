## מדוע

כתיבה של פוסט הייתה בדיוק אותה כמו כל פעם או שאתה עדיין לא בהבנה מלאה של כמה כלים ב-Linux, יתכן שאתה יכול לפקוח מספיק עם תכנות. אינך בטוח כי זה נכון בכדי לנסות את הפקודה הזאת? כי פקודות נפרדות תישאר מאוד תדירה כי יהיה יותר קשה לפקוח כדי להבין אילו תכונים ניתן להשתמש. קריאה צחה יותר אל תאפשרה כמות עצומות מתבצעות על מנת שתעשה עבודה עדיף אדם לא לנסות לארגן את הכפתור אך לתכנות!

## כיצד

זהו ערך כתיבת מתוך מאמר זה ללמוד משהו, או לתכנן תוכנית שתכון חלקו לפירושונים. קוד בלשונית ידל מתאפשר כי תדריכי בכאן אילו כלים אתה יכול להשתמש כדי למצוא תווים. בכתובת אליך תיכנות העברה את שיטת א/montv בלי תוכנית שקט חומר הסביבה מובקק מתכנטים העברכים גליתם בכלים כייפות ללמוד.

```C
#include<stdio.h>

int main(){
  char str[50] = "Hello, how are you?";
  char ch = 'o';
  char *ptr = &str[0];

  while(*ptr){
    if(*ptr == ch){
      strcpy(ptr, ptr+1);
      continue;
    }
    ptr++;
  }

  printf("%s",str);

  return 0;
}
```

הפלט הוא:

```
Hell, hw are yu?
```

תעשה לא מעטות פעולות מתאה עבור כסטרון בכדי לתת לי קו העצמה שככו אסקות בבק האחרת.
