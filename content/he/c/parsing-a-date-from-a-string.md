## מה זה ולמה?

פרקים מסוימים בתוכנה מתקבלים כתאריכים בפורמט של מחרוזת טקסט. החלק הקשהותי ביותר בעיבוד תאריכים הוא להפוך אותם ממחרוזת לתאריך נתון. זהו תהליך נקרא "פיענח של תאריך ממחרוזת". מפתחי תוכנה משתמשים בפיענוח תאריכים ממחרוזת כדי לעבד תאריכים כקלט נתון ולבצע חישובים מתאימים עליהם.

## איך לעשות זאת?

### דוגמאות קוד:

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    char date_str[] = "09/30/2021"; // תאריך בתור מחרוזת
    struct tm date; // הגדרה של מבנה מסוג tm
    strptime(date_str, "%m/%d/%Y", &date); // פיענח תאריך ממחרוזת לתוך המבנה
    printf("Day: %d\n", date.tm_mday); // הדפסת היום מהתאריך
    printf("Month: %d\n", date.tm_mon); // הדפסת החודש מהתאריך
    printf("Year: %d\n", date.tm_year); // הדפסת השנה מהתאריך
    return 0;
}
```

### תוצאות לדוגמה הנ"ל:

Day: 30
Month: 8
Year: 121

## חקר עומק

### היסטוריה

המצב שבו תאריכים נעברים ממחרוזות לתוך מבנים נתונים קיים כבר מאז תחילת תוכנות המחשבים. רעיונות רבים נוספים חלפו במהלך השנים, אבל הפיתרון שנבחר הוא פיענוח תאריכים על ידי השתמשות במבנה נתונים מתאים.

### אלטרנטיבות

פיענוח תאריכים ממחרוזת לתאריך יכול להיעשות גם באמצעות תיבות אחרות כמו תיבות קלט, תיבות דיאלוג ופעולות מתים. כל אלטרנטיבה מגיעה עם יתרונות וחסרונות שונים, אבל פיענוח תאריכים ממחרוזת נשאל כיוון שהוא מספק יישום פשוט ויעיל המתאים לרוב המקרים.

### פרטי יישום

מעבר ממחרוזת לתאריך נתון באמצעות C נעשה באמצעות קריאה לפונקציה `strptime()` שנמצאת בספריית התקנות `time.h`. הפונקציה תקבל את המחרוזת לפיענוח, את התבנית לפיענוח (כמו "%m/%d/%Y" בדוגמה שלנו) ואת המבנה הנתונים שמקבל את התאריך המפוענח.
