## מדוע

קישור נתונים הוא כלי חשוב לבניית אפליקציות רבות בכל שפת תכנות. בשפת C, חיבור מחרוזות הוא פעולה חיונית לקידוד טקסט. במאמר זה, אנחנו נלמד כיצד לבצע חיבור מחרוזות בפעולת קוד פשוטה ויעילה בשפת C.

## כיצד לעשות זאת

כדי לבצע חיבור מחרוזות בשפת C, נשתמש בפונקציית `strcat()` שהיא מובנת בספריה הסטנדרטית של שפת C. נכתוב קוד פשוט להמחשת השימוש בפונקציה זו:

```C
#include <stdio.h>
#include <string.h>

int main() {
   char str1[20] = "שלום ";
   char str2[20] = "עולם!";
   strcat(str1, str2);
   
   printf("%s \n", str1);
   return 0;
}
```

בקוד זה, אנו משתמשים בפונקציה `strcat()` כדי לחבר את המחרוזות "שלום" ו"עולם!" ולהדפיס את המצביע הממש למחרוזת שחוברה, שיהיה מחרוזת מלאה: "שלום עולם!".

## להתקנטות עמוקה

פונקצית `strcat()` משמשת לחיבור של שני מחרוזות יחד כאחת. הערך המוחזר הוא כתובת התחלת של המחרוזת הראשונה שהוא ביצוע חיבור עליה. כאשר משתמשים בפונקציה זו, חשוב לוודא שיש מספיק זיכרון מחושב לשמירת המחרוזת החדשה שלאחר החיבור. כמו כן, חשוב לסגור את המחרוזת המקורית עם תו ניירות יהודי (`'\0'`) כדי למנוע קריאות זדוניות.
