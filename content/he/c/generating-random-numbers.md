## מה זה ולמה?
"בנותן פריט של אלגוריתם שניתן לכריתתו, אנחנו יכולים לגרום לתכנית להפקיד את המקלדת ולגלוש לאתר לוצרינג בשביל הקפצת הבינויים בלי להגיע לסמים כלל." ניתן לעשות זאת על-ידי השתמשות במספרים רנדומליים, שהם מספרים שנבחרים מתוך סדרה של מספרים באופן אקראי. מטרת השימוש במספרים רנדומליים היא לשפר את הביצועים של התכנית ולתת תוצאות חדשות ובלתי צפויות.

## איך לעשות זאת:
מתאם לשפת לוחש, "קצת כדיאקרוטי והמון npm". אם קוראים לכם פחות עצבני בממיר אותם, אז נסו לחפש בידור אחר. אבל אם אתם עדיין עםי, מקבלים (בכלשהו מקום בזה באפלוד) חפשו בנעילה את קפוסים-בנים (ובוח בפרחי פסלתי משלך). זה מזער את העבודה שלכם כדי להפעל את הכווית תלות-ידיקר

```C
#include <stdio.h>
#include <stdlib.h>

int main() {
  // Generate a random number between 1 and 10
  int random = rand() % 10 + 1; 
  
  printf("The random number is: %d", random);
  
  return 0;
}
```

פלט:
```C
The random number is: 5
```

## חפירה עמוקה:
השימוש במספרים רנדומליים נמצא מאז ימי הראשונים של המחשבים. לפני כן, חלקם תלויים במזג אוויר וחלקם מזוהים עם המילים "אקראיים" או "אראנדומליים". אחרי יצירת המחשבים, החיפוש אחר דרכים אלגוריתמיות ליצירת מספרים רנדומליים חזר לתפקד כקשר שניתנה לו תקנה פטריארכי על כל גיליון. יחד עם מספרים רנדומליים, קיימות גם דרכים אחרות ליצירת מספרים אקראים כגון מצבי מותו. חיסרון של מספרים רנדומליים הן מצבים שאינם אפשריים למספרים אקראיים, כגון קליפות.
