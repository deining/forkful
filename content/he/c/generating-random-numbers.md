##למה

למה בכלל להתעסק ביצירת מספרים אקראיים? אם מתעניינים בתכנות בשפת C, יש כמה סיבות מוצדקות להשתמש במספרים אקראיים בתוכניות שלנו. למשל, הם יכולים לשמש כדי ליצור מצביעים לאובייקטים אקראיים, ליצור מצביעים כמו מערכות תכנון, וליצור מצביעים למבני נתונים כמו רשימות ומילונים.

##כיצד לעשות זאת

לקודם כל, נצטרך לכתוב את הכותרת הנכונה במערכת ההפעלה שלנו. לדוגמה, בווינדוס יש להוסיף את ההגדרה הבאה לתוכנית העיקרית שלנו:

 ```
#define _CRT_RAND_S 
 ```
 
לאחר מכן, נוכל להשתמש בפונקציות מובנות כמו rand() כדי ליצור מספרים אקראיים. נלקח לדוגמה את הקוד הבא:

 ```
#include<stdio.h> 
#include<stdlib.h> 

int main() { 
	int i, random_number; 

	printf("Enter the number of random numbers you want to generate: "); 
	scanf("%d", &i); 

	printf("Random numbers: \n"); 
	for (int c = 1; c <= i; c++) { 
		random_number = rand(); 
		printf("%d\n", random_number); 
	} 

	return 0; 
} 
 ```

כאן אנחנו כותבים תוכנית שמבקשת מהמשתמש להזין את מספר המספרים האקראיים שהוא רוצה ליצור, ולאחר מכן מדפיס את המספרים האקראיים שנוצרו. מומלץ ליצור גם פונקציות נוספות כדי לטפל בתוך מגבלות כמו שינוי הזריקה או גבלות הפלט.
