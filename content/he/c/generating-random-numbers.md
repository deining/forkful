## למה

כדי להגדיר גורמים אקראיים בתוכנה שלנו וליצור את האפשרות לקבלת תוצאות שונות כל פעם שנפעיל את התוכנית.

## איפה ניסוי

```C
#include <stdio.h> 
#include <stdlib.h> 

int main() 
{ 
    // השורה הבאה מגדירה את התחום שבו ניתן להגריל מספרים
    int min = 1; 
    int max = 10; 
    // השורה הבאה יוצרת מספר אקראי בין 1 ל-10
    int random_number = (rand() % (max - min + 1)) + min; 
    // השורה הבאה מדפיסה את המספר האקראי שנוצר
    printf("המספר האקראי שנוצר הוא: %d\n", random_number); 
    return 0; 
}
```

פלט:

```
המספר האקראי שנוצר הוא: 7
```

### חקירה עמוקה

יצירת מספרים אקראיים נחשבת לחלק חשוב בתכנות מחשבים. מתוך המספרים האקראיים ניתן ליצור מצבים כמו משחקי קלפים או סדרות אקראיות שימושיות כמו בבדיקה ובפיתוח תוכניות. 

ליצירת מספרים אקראיים בשפת C, אנחנו משתמשים בפונקציה rand() המייצרת מספרים אקראיים בתחום של מספרים שלמים. על מנת להבטיח שכל אפשרות תמיד תיהיה בעלת אותו סיכוי להיות נבחרת, אנחנו משתמשים בנוסחא פשוטה כדי לסדר את החפיפה. 

בנוסף, אנחנו משתמשים במשתני min ו-max כדי לגבול את התחום שבו ייווצרו המספרים האקראיים. אנחנו גם נותנים למשתמש את האפשרות לקבוע את התחום בעצמו תוך שימוש במשתני min ו-max.
