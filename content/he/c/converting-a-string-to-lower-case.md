##למה
למה כדאי להמיר מחרוזת לאותיות קטנות? המרת מחרוזת לאותיות קטנות מאפשרת לנו לבצע פעולות על המחרוזת בקלות ובאופן יעיל יותר, כגון השוואת מחרוזות או חיפוש מחרוזת מסוימת בתוך מחרוזת אחרת.

##איך לבצע?
לאחר שלמדנו לפתרון בעיות בשפת סי, הגיע הזמן ללמוד קצת על פעולות מחרוזות. כאשר נרצה להמיר מחרוזת לאותיות קטנות, נצטרך להשתמש בפונקציית "tolower". נבצע את הפעולה הבאה בשלבים:

1. נגדיר מחרוזת שאנחנו רוצים להמיר לאותיות קטנות.
2. נגדיר משתנה חדש שישמור את התוצאה של ההמרה.
3. נשתמש בלולאה "for" כדי לעבור על כל אות במחרוזת ולהמיר אותה לאות קטנה באמצעות הפונקצייה "tolower".
4. נדפיס את התוצאה החדשה של המחרוזת.

בהמשך תוכלו למצוא קוד מלא ותוצאות של פעולת ההמרה ב־C:

```C
#include <stdio.h>
#include <ctype.h>

int main()
{
    // כאן נגדיר את המחרוזת שנרצה להמיר
    char str[] = "HELLO WORLD";
    // נגדיר משתנה חדש עבור התוצאה
    char new_str[sizeof(str)];
    // נשתמש בלולאה כדי לעבור על כל אות ולהמיר אות לאות קטנה
    for(int i = 0; i < sizeof(str); i++){
        new_str[i] = tolower(str[i]);
    }
    // נדפיס את התוצאה
    printf("%s\n", new_str);
    return 0;
}
```
תוצאת הריצה: `hello world`
