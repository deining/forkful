ניתוח C ל-Markdown: המרת מחרוזת לאותיות קטנות

## למה

להמרה של מחרוזת לאותיות קטנות הוא תהליך חשוב ותכליתי שיכול להיות שימושי למספר סיבות, כגון:

- ולידציה: לצורך וידוא שנתונים המוזנים תואמים לניסוח מסוים, לעיתים קרובות צריך להתחשב בהבדל בין אותיות קטנות לאותיות גדולות. המרת מחרוזת לאותיות קטנות יכולה לסייע בוידוא נכון של הנתונים.
- ישירות להצגה: כאשר מדובר בהצגת נתונים למשתמש, לעיתים קרובות רצוי להטעין את המחרוזת באופן קולח ומתוקשר. למשל, בניסוח שפות התכנות, כתיב מחרוזת שנמתחת ללא צורך, נוח לקריאה.
- בנייה של תכניות: גם כאשר מדובר בסקריפטים שכתובים באמצעות שפת התכנות של C, נכתבים בקוד רחבי מכל מיני נסיבות. השתמשויות של מחרוזות עוזרת כאשר מימוש נעשה במשנתדים, עם פעולות לביקורת נכונה על תוצאת המחרוזת.

## איך להמיר מחרוזת לאותיות קטנות

BH ```c
  // הגדרת מחרוזת
  char string[] = "מחרוזת לאותיות קטנות";
  // שימוש בפונקציית לעיל בכדי להמיר את המחרוזת לאותיות קטנות
  strlwr(string);
  // הודפס תוצאה
  printf("מחרוזת לאותיות קטנות: %s", string);
```

התוצאה המודפסת תהיה:

```
מחרוזת לאותיות קטנות: מחרוזת לאותיות קטנות
```
