## למה:
כתיבת תוכניות עם שפת תכנות כמו C דורשת המון עבודה ידנית כדי לגרום לתכנים לעבוד כדי לספק את התוצאה הרצויה. בין המשימות הידניות שנדרשות היא חיפוש והחלפה של טקסט בקוד, ולכן העבודה הזאת יכולה להיות יעילה בהרבה בדרך כלל אם תבוצע באמצעות כלי תוכנה או טכניקה שמסייעים לבצע את הפעולה הזאת בצורה יכולה יותר.

## איך לבצע חיפוש והחלפה בשפת C:
כדי לבצע חיפוש והחלפה של טקסט בקוד בשפת C, ניתן להשתמש בפונקציות כמו "strstr" ו"strcpy" הקיימות בספרייה הסטנדרטית של C. לדוגמה, אם נרצה להחליף את המילה "פיתוח" במילה "תכנות" בשלושה מספרי מחרוזות ביחד, נוכל להשתמש בקוד הבא:

```c
#include <stdio.h>
#include <string.h>

int main()
{
    char str1[50] = "פיתוח תוכנה הוא תהליך מעניין.";
    char str2[50] = "אני מעוניין לחפש את המילה 'פיתוח'.";
    char str3[50] = "פיתוח הינו חלק חשוב מתהליך התכנות.";

    char *result;

    // החלפת המילה "פיתוח" במילה "תכנות" בכל אחת מהמחרוזות:
    
    result = strstr(str1, "פיתוח");
    strcpy(result, "תכנות");
    
    result = strstr(str2, "פיתוח");
    strcpy(result, "תכנות");
    
    result = strstr(str3, "פיתוח");
    strcpy(result, "תכנות");
    
    // הדפסת המחרוזות המעודכנות:
    
    printf("%s\n", str1);
    printf("%s\n", str2);
    printf("%s\n", str3);
    
    return 0;
}
```

**פלט:**

תכנות תוכנה הוא תהליך מעניין.
אני מעוניין לחפש את המילה 'תכנות'.
פיתוח הינו חלק חשוב מתהליך התכנות.
