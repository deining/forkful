## למה

למה לפתוח את הדף הזה? ממש חשוב שלא נהרוג בעבודה שיחרוג מבנים. הפוסט הזה ילמד אותך איך לעשות חיפוש והחלפה של טקסט בשפת סי.

## איך לעשות

לפני שנתחיל, אם אתם לא מכירים את שפת התכנות C, אנחנו ממליצים להתחיל עם הערך [הזה](https://he.wikipedia.org/wiki/C), כדי להכיר את יסודותיה. אחרי שהתעכבתם כמה דקות על פניו, בואו נתחיל להתחמק!

כדי לבצע חיפוש והחלפה של טקסט בשפת סי, נשתמש בפונקציה המובנית `str_replace`. הערך הראשון שלנו הוא המחרוזת (string) שאנחנו רוצים לעבוד איתה, והערך השני הוא המחרוזת החדשה שנשתמש בה כדי להחליף את התווים במחרוזת המקורית. הנה דוגמא קוד:

```C
#include <stdio.h>
#include <string.h>

int main()
{
    char str[] = "Hello, world!";
    char replace[] = "Shalom, olam!";
    char new_str[100];

    strcpy(new_str, str_replace(str, replace));

    printf("%s", new_str);

    return 0;
}
```

תוצאה:

```
Shalom, olam!
```

כמו שאתם רואים, הפונקציה `str_replace` מחזירה מחרוזת חדשה שמכילה את ההחלפה שביקשנו.
