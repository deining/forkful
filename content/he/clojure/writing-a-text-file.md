## מדוע

לכתוב קובץ טקסט בשפת קלוז'ר הינו דרך מעניינת ליישם מיומנויות תכנות וליצור תוכניות מתוחכמות. זהו דרך מצוינת ללמוד את יסודות השפה ולחקור את יכולותיה.

## איך לעשות

כדי לכתוב קובץ טקסט בשפת קלוז'ר, ניתן להשתמש בפונקציה `spit`, המקבלת שני פרמטרים - נתיב לקובץ ותוכן הקובץ. לדוגמה, נרצה ליצור קובץ טקסט פשוט בשם "hello.txt", אשר יכיל את המילה "שלום" בשורה אחת:

```Clojure
(spit "hello.txt" "שלום")
```

כעת נרוץ את הקוד ונתבקש לצפות את הקובץ הנוצר. לאחר מכן, נוכל לראות בהצלחה את הקובץ hello.txt המכיל את המילה "שלום"!

## כיוון עמוק

כאשר אנו כותבים קוד בשפת קלוז'ר, אנו משתמשים בפונקציות מובנות ועובדים עם נתונים בצורה אבסטרקטית. בנוסף, כתיבת קובץ טקסט מאפשרת לנו ליצור תוכניות ייחודיות ומתקדמות, כגון יצירת מסמכי טקסט למדיניות וניהול קבצים.
