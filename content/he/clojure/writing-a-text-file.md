## מדוע
למה אנשים כותבים קוד בשפת Clojure? קוד הטקסט הוא דרך נוחה לשלוט על מידע ולייצר תוכניות תוכנה מורכבות בקלות ונוחות.

## איך לעשות זאת
קוד בשפת Clojure כתוב בתוך קובץ טקסט פשוט, ומכיל ביטויים ופקודות שנקראים "פורמה". נציג כאן דוגמאות של קוד ופלט בתוך בלוקי קוד "```clojure ... ```".

כאן יש דוגמה של קוד שמדפיס את המילה "שלום":

```clojure
(print "שלום")
```

והפלט של הקוד הזה יהיה:

```
שלום
```

זהו קוד פשוט וקצר, אבל הנוכחות של קוד בתוך קובץ טקסט מאפשרת יצירת תוכניות תוכנה מורכבות יותר, עם יכולות רבות ומגוונות.

## חקירה מעמיקה
שפת Clojure היא שפת תכנות פונקציונלית, מה שאומר שבמקום להשתמש בפקודות וביטויים מוכנים, קוד הוא מבנוי ממספר פונקציות שיופעלו על ידי פרמטרים.

למשל, ניתן לכתוב פונקציה פשוטה שמחשבת את הזיכרון המחושב של הקלט המספרי שנמצא ברשימה:

```clojure
(defn compute-memory [numbers]
  (apply + numbers))
```

הפונקציה משתמשת במילון "defn" כדי להגדיר את שם הפונקציה ואת הפרמטרים שהיא מקבלת. בדוגמה זו, הפונקציה מקבלת רשימת מספרים ומשתמשת בפונקציית השרשור "+" כדי לחשב את הסכום של כל המספרים ברשימה. ניתן לשלוח פעם אחת או יותר את הקלטים שלכם לפונקציה בשימוש בפקודת "apply".
