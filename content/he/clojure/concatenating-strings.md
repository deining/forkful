## למה

Concatenate (לשרשור) הוא תהליך של חיבור שני מחרוזות ליחד. זה מסייע לנו ליצור מחרוזות מורכבות יותר ולשלב מחרוזות עם משתנים או ערכים אחרים. פעולת השרשור משמשת בעיקר בתחום התכנות כדי ליצור מחרוזות דינמיות ומשתנות.

## איך לעשות זאת

הנה דוגמאות לשימוש בפונקציות השרשור ב-Clojure:

```
(def string1 "שלום")
(def string2 "עולם")

(str string1 string2)
;; output: שלוםעולם

(str string1 "אהבה" string2)
;; output: שלוםאהבהעולם

(str "המספר " 5 " הוא מספר מפתיע")
;; output: המספר 5 הוא מספר מפתיע

```

בפונקציות של השרשור התחבורה את המחרוזות בסדר שלהם ומשלימות אותן ליחד. אם אתה רוצה להוסיף רווח או תווים אחרים בין המחרוזתים, אתה יכול להשתמש בפונקציית str עם תווים בודדים:

```
(str string1 " " string2)
;; output: שלום עולם

(str string1 ", " string2 "!")
;; output: שלום, עולם!
```

בנוסף, אם אתה רוצה להשתמש בפונקציית שרשור עם מספר גדול של מחרוזות, אתה יכול להשתמש בפונקציית str-join עם מחרוזת המפרידה שאתה מעדיף. לדוגמא:

```
(def strings ["ייחודי" "אהוב" "מושלם"])

(str-join " " strings)
;; output: ייחודי אהוב מושלם

(str-join ", " strings)
;; output: ייחודי, אהוב, מושלם
```
