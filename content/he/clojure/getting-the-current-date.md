# למה

איך לעבוד עם תאריכים בקלוג'ר

למה: רק כמה משפטים המסבירים *למה* מישהו ירצה לעסוק בקבלת תאריך נוכחי.

## מהיכן

קבלת תאריך נוכחי היא משימה נפוצה בתכנות ובקלוג'ר יש כמה דרכים לעשות זאת.

תחילה נוכל להשתמש בפונקציה `java.util.Date` כדי ליצור את התאריך הנוכחי. הנה דוגמא של קוד כיצד לבצע זאת:

```Clojure
(require '[java.util.Date :as date])

;; פונקציה פשוטה שמחזירה את התאריך הנוכחי
(defn get-current-date []
  (date.)
```

הפלט של פונקציה זו תהיה משהו כמו זה:

```Clojure
#inst"2022-01-17T16:18:39.114-00:00"
```

עם זאת, ישנן דרכים נוספות כדי לקבל תאריך נוכחי בקלוג'ר. ניתן להשתמש בפונקציות מובנות שקיימות בספריית `clojure.java-time`  או להשתמש בסיימבול `#inst` שמאפשר לנו ליצור תאריך בפורמט נוח לקריאה.

## העמקה

העמקת תעודה זו מטרתה לאפשר מגוון רחב של אפשרויות לקבלת תאריך נוכחי בקוד קלוג'ר.

ניתן להשתמש בפונקציות `now` ו- `today` בספריית `clojure.java-time` כדי ליצור תאריך נוכחי בפורמטים שונים, כגון פורמט ISO ופורמט Unix.

בנוסף, שימוש בסיימבול `#inst` יכול להיות שימושי במקרים שבהם נרצה ליצור תאריך מותאם אישית כגון תאריכים מיוחדים או תאריכים עם זמן מסוים.
