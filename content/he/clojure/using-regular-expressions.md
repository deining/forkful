# למה

משתמשים בפעם ראשונה יכולים להתקשות לראות את הטוב שנוצר על ידי ביטויים רגולריים, אבל הם בטוח יגלו שהם יכולים לבצע פעולות מורכבות במהירות וביעילות רבה. למומחים, ביטויי רגולריים הם כלי עזר חיוני בכתיבת קוד וטכניקה חשובה למיטב איכות הקוד.

## איך לבצע

התחבר למחברת Clojure ותחילה נכתוב קטע קוד המשתמש בטכניקת ביטוי רגולרי פשוטה. נשתמש בפונקציה `re-find` למציאת תאים במחרוזת שמכילים מילים שמתחילות עם האות "hello".

```Clojure
(import '[clojure.string :as str])

(let [str "Hello, my name is John"
      pattern #"hello\w*"]
  (re-find pattern str))
```

כתוצאה מהקוד נקבל את התוצאה הבאה:

```
"Hello"
```

באמצעות ביטויים רגולריים ניתן ליצור הרבה יותר מכניס חזקים לטיפול במחרוזות. לדוגמה, אפשר לחפש כל תווים שהם לא אותיות באמצעות הטכניקה \W כדי למצוא תווים מיוחדים כגון רווחים וסימנים מיוחדים.

## מכטת ראיונות

ביטויים רגולריים משמשים להתאים מופע חסר של אותיות את החסרות, או לבנות את תנאי גמישים. המישור למילים מבוסס "כמהם" את התבניות.

 אם אנחנו נרצה לבנות ביטוי רגולרי שמכיל מספרים בטווח ספציפי, אפשר לעשות זאת בעזרת תחום פתיחה וסגירה.

לדוגמה, אם נרצה לחפש מספרים שנמצאים בין 100 ל- 999, נצטרך להשתמש בביטוי הרגולרי:

```Clojure
#"1[0-9]{2}|2[0-9]{2}|3[0-9]{2}"
```
