## למה

במאמר זה נלמד כיצד להשתמש בפונקציות של Clojure כדי למחוק תווים התואמים תבנית מסוימת. לעתים יש לנו לטפל בטקסטים גדולים ולמחוק או להחליף חלקים של הטקסט הלא רלוונטיים או לא מתאימים לצורך שלנו. באמצעות מנגנון זה, נוכל לנקות את הטקסט ולשפר את עיבוד הנתונים שלנו.

## איך לעשות זאת

מתחילים קוד:

```Clojure
; תאריך קודם
(def birthdays ["10/5/1998" "12/8/2000" "5/11/2003" "7/2/1995" "9/10/1999"])

; כדי למחוק רק את השנה האחרונה של כל תאריך, נשתמש בפונקציית שבת
(map #(replaced-with % #"\d{4}$" "") birthdays)
```

אנו מבצעים תהליך דומה לכך לריכוז טקסט עם נתונים כמו כתובת אייפי. נתחיל על ידי לבדוק את כל התווים בטקסט ונמחק את כולם שאינם נחוצים לנו.

```Clojure
(def ip "192.168.1.1")
(clojure.string/replace ip #"\d{3}" "") ; אנו רוצים למחוק את כל הספרות האלו כי הן לא רלוונטיות לנו
```

### נתונים נוספים

כדי לאפשר מייפות ביבליות על טקסט אחד, נוסיף תכונה נוספת לפונקציית replace-in-string. תכונה זאת פותחה לרכבים שונים ויישומים אחרים.

```Clojure
(def items ["Apple 1" "Orange 2" "Banana 3" "Pineapple 4"])

; נמחק מלך מכל תוכן
(clojure.string/replace items #"^King\s+[IV]\s+\s+" "")
; = ["Apple" "Orange" "Banana" "Pineapple"]
```

כדי לעשות זאת, נשתמש בפונקציית sub Gothic כדי למחוק את הבלשנות הכללית והסיפרות של מלך או נשיא.
