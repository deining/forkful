## למה

תמיד ניתן לזמן תאריך ומחרוזת מפעלים קלים להשתמש במחשב עם צוות ה-Clojure. כאשר אנו צריכים להציג את התאריך בפורמט מסוים או להשמיט מידע מיותר, כמו חלקים של פורמט תאריך (כמו שנת, חודש, יום), ניתן לשימוש בשיטה להמיר תאריך למחרוזת כדי לקבל את הפלט הרצוי.

## לאיך לעשות

הנה דוגמאות להמרת תאריך למחרוזת עפ"י פונקציה `format`:

```Clojure
(format "%1$tY" (date "2019-12-31"))
=> "2019"

(format "%1$td.%1$tm.%1$tY" (date "2019-12-31"))
=> "31.12.2019"

(format "%1$tA, %1$td %1$tB %1$tY" (date "2019-12-31"))
=> "Tuesday, 31 December 2019"
```

המחרוזת הראשונה בכל פונקציה מייצגת את התאריך בפורמט מסוים והתאריך השני המופיע בסוגריים המרובעים המתאימים מייצג את התאריך שאנו רוצים להמיר. ניתן להשתמש במספר של האתרוג(אי)ס כדי לציין את החלקים השונים של התאריך, כמו שנת, חודש ויום כאשר הם מכניסים את המחרוזת הסופית.

## צבירה עמוקה

אם אתה מעוניין לבחון מפורט יותר כיצד מתנהגת פונקציית ה- `format` כאשר ממירים תאריך למחרוזת, ניתן ללמוד עוד על התבניות השונות של מחרוזות התאריך בלינקים המצורפים כאן למטה.
