# מדוע
למה לכתוב את הקוד שלנו בצורה שמייצרת מספרים אקראיים? השתמשתי כבר פעמים רבות במספרים אקראיים בקוד שלי לאלגוריתמים שמנסים לפתור בעיות בעולם אמיתי, משחקים או פשוט מנסות לנסות מהדול קומפי. להשתמש במספרים אקראיים יכול לפתור בעיות במהירות רבה יותר מאשר עיוורונים או מתמטאים מבולבלים.

# איך לעשות זאת
ליצור מספרים אקראיים בקוד שלכם בצורה קלה ופשוטה באמצעות Clojure. הראה אנחנו את השימוש בפונקציות `rand` ו- `rand-int` כדי ליצור מספרים אקראיים בין מספרים כלשהם. לדוגמה:

```Clojure
(rand) ; יוצר מספר אקראי בטווח של [0, 1)
(rand 10) ; יוצר מספר אקראי בטווח של [0, 10)
(rand-int 100) ; יוצר מספר אקראי שלם בטווח של [0, 100)
```

שימו לב שהטווח באינדקס הראשון תמיד הוא כולל והאינדקס האחרון הוא בלתי כולל. תוכלו לשנות את הטווח כרצונכם כדי ליצור מספרים אקראיים בטווח של ערכים כלשהם.
