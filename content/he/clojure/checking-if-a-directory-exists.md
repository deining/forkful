# מדוע

למה לבדוק אם תיקייה קיימת? בדיקת האם תיקייה קיימת יכולה להיות שימושית במגוון מקרים, כגון ביצוע קוד מסוים אם התיקייה קיימת או להתאים פעולות ספציפיות לכל סוג של תיקייה.

# איך לבדוק אם תיקייה קיימת

לאחרונה, יצרתי תכנית לאתר העברית שקוראת ומדפיסה את רשימת התיקיות הקיימות במערכת הקבצים הנוכחית. כחלק מכך, היה חייב לוודא שהתיקייה שאני מנסה לגשת אליה קיימת. אם כן, התוכנית תדפיס את שם התיקייה כחלק מהרשימה.

```Clojure
(require '[clojure.java.io :as io])
(let [directory (io/file ".")
      existing? (.exists directory)]
  (when existing?
    (println "התיקייה" (.getName directory) "קיימת")))
```

שורת הקוד הראשונה מייבאת את הספריה `clojure.java.io`, המכילה כלים לעבודה עם קבצים ותיקיות בקבצי פעולה של ג'אווה. בשורת הקוד השנייה, אני משתמש בפונקציה `file` כדי ליצור משתנה שמייצג את התיקייה הנוכחית במערכת הקבצים. כדי לבדוק אם התיקייה קיימת אם כן או אינה, אני משתמש בפונקציה `exists` המחזירה `true` אם התיקייה קיימת ו-`false` אם לא. אם התיקייה קיימת, הקוד מדפיס את שם התיקייה בשורת הפלט.
