## למה: 
נדמה כאילו למצוא את אורך המחרוזת אין משמעות מיוחדת, אבל אם אנחנו קודדים פונקציה כזאת בצורה נכונה, אנו יכולים לעזור לעבודה שלנו ליהיות יותר יעילה ומובנת. כשאנו יכולים למצוא את אורך המחרוזת, אנו יכולים לטפל במתווהים, לבצע אינדקסים ולבצע פעולות אחרות חשובות כדי לכתוב קוד בצורה טובה יותר.

## איך לעשות זאת:

פונקציה למציאת אורך המחרוזת קיימת בפונקציות המובנות של Clojure. כדי להשתמש בה, ניתן להשתמש בפונקציה "count" ולכנות אותה עם המחרוזת אליו אנחנו רוצים לחשב את האורך שלו. נהפוך את המילים "Hello world" למחרוזת ונחשב את אורכה:

```Clojure
(defn length-of-string [string]
  (count string))

(length-of-string "Hello world")
```

Output: 11

הפונקציה "count" מחזירה את מספר האלמנטים ברשימה או במחרוזת.

## מעמקים:

אנו יכולים לעשות עוד כמה דברים מעניינים עם פונקציה זו שניתן לשפץ את הקוד שלנו. לדוגמה, אם נכניס מחרוזת ריקה לפונקציה שלנו, היא תחזיר אפס, מה שאינו מייצג את אורך המחרוזת הריקה באמת. כדי להתמודד עם זה, ניתן להשתמש בפונקציה "nil?" כדי לבדוק אם המחרוזת היא ריקה ולהחזיר אפס במקום זאת:

```Clojure
(defn length-of-string [string]
  (if (nil? string) 0 (count string)))

(length-of-string "") ; output: 0
```
