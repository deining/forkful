## למה

מדוע כדאי להפעיל בדיקת מנשק (debug output) בקוד שלכם?

## איך לבצע

מספר דוגמאות של קוד ופלט בתוך חלקקי קוד "```Clojure ...```"

```
Clojure (println "הודעת המנשק") 
```

פלט:
```
הודעת המנשק
```

## כיוון נמוך

אינו משנה אם אתם משתמשים בכלי לתכנות כמו Eclipse או IntelliJ, או אם אתם אוהבים להשתמש באחד מעורקי הפעלת קוד Clojure, אתם תצטרכו בפיצ'ר הדפסת המנשק בהתחלה. בהמשך תוכלו למצוא כמה טיפים שיכולים לעזור לכם לנהול את הדפסת המנשק שלכם בצורה יעילה יותר ולמצוא באילו מצבים זה יכול להיות מועיל.
