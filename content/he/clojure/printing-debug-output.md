ראשיד סקריפט להדגמת ביצועים בקוד שלכם באמצעות פלט מסוג DEBUG:

## מה זה ולמה?

פלט מסוג DEBUG הוא פלט נוסף שנוצר בקוד לצורך פיתוח ועדכונים. הכוונה היא להדגים את שלבי התפתחות הקוד ולבדוק שמובטחים התנאים שמצוינים עבורו.

## כיצד לבצע?

אם תרצו להציג את נתוני הפלט שלכם בקוד שלכם בעזרת פלט מסוג DEBUG, כך תעשו:

```Clojure
(debug "הודעת פלט: " data)
```

פקודה זו תשתמש בפונקציה debug כדי להציג את נתוני הפלט שלכם. תוכלו לשנות את הודעת הפלט כך שתתאים לצורכים שלכם.

## בינתיים רק מעמקים

בעבר, לפני שתוותרו לדבר עם הודעות פלט, הייתה עליכם למצוא דרך אחרת של ניתוח התכנית שלכם. אם תמשיכו עדיין עם פלט מסוג DEBUG, ישנן אלטרנטיבות בעיה עם מספר תכנית בלבד. בנוסף, ישנן גם הכוונים על פיתוח כלים נוספים כדי להציג תכנים ולהעביר נתיבים חסרו לכם עבור פרוייקט ספציפי לאחר פיתוח הזה.
