## למה

בכדי לפתור בעיות בקוד הנגרם ולמצוא את האיזורים שגורמים לתקלות, יש מקרי שבו כדי להבין את הפעולה של התוכנית תהיה עלינו להדפיס מידע לאיזור מסוים. הדפסת הפלט המכיל מידע נוסף ומפורט תוכל לסייע לך למצוא את הבעיה ולתקן אותה.

## כיצד לעשות זאת

השימוש בפונקציה `println` תאפשר לך להדפיס מסרים לפלט של התוכנית שלך. לדוגמא:

```Clojure
(defn add [x y]
  (println "x:" x)
  (println "y:" y)
  (+ x y))
```

במקרה זה, המסרים "x: " ו- "y: " ידפיסו את ערכי המשתנים x ו-y בהתאמה, ואז התוכנית תמשיך לרצות ולהחזיר את התוצאה של החיבור.

## כיון עמוק

כאשר מדובר בהדפסת פלט לכוונת דיבאג, ישנן כמה ייעודיות נוספות בשפת קלוז'ור שתוכלו להשתמש בהן. לדוגמא, הפונקציה `prn` תדפיס את הנתונים בקוד שלכם בצורה של Clojure, כך שתוכלו להשתמש בם כמו קוד כאשר מנסים לבנות תנאים או לבדוק ערכים. גם הפונקציה `pprint` תדפיס את הנתונים בצורה יפה יותר ובסדר מבואר יותר.
