## מדוע

אנשים משתמשים ב Fish Shell רב פעמית כי היא מספקת מגוון רחב של הפקודות השונות שנדרשות בתהליך התפעול של המערכת הפעילה. בנוסף, ביכולתה לאפשר את העתקת המידע וכמו כן הדפסתו והגדרת תנאים מתואמת להבנת הנתונים שיש להם

## האיך לעשות

כדי להדפיס את המידע שלנו ולהשתמש בו כרצוננו, ניתן למצוא את הפקודה שלך בתוך Fish Shell וכתבו לנו:

```
Fish Shell -c "פקודה הכוללת תתי מחרוזות"
```

אם אתה רוצה להדפיס את המידע או לששתמש בו למטרות שונות ניתן להוסיף את המחרוזת לתוך הפקודה כך שתהפוך להיות נתוחת ואח"כ ניתן להדפיס רק את חלק מענים:

```
Fish Shell -c "תוכן תתי מחרוזות" | cut -d ';' -f 1-2
```

כאן, אנו משתמשים בפקודת הגזרה "פקודת הנתוחה" לחלק את המידע לפי מרכאות ולהדפיס את החלק הראשון והשני של המידע.

```
תלתור לתתי טקסט
```

## טור שנותן לנו את כל האינפורמציה על העצות לחלחל תתי הודעות

כעת, ניתן לעשות שימוש בפקודה של הנתח להביס את הקו הראשון של המידע לדוגמא:

```
cut -d ',' -f 3,7
```

כאן, אנו משתמשים בתוכנה הפקודתית שיש לה את השופט לנתון אותו הצבנו לעצמנו והמידע יודפס לפי סדרת הדוגמא הנתונה שיצפה נפגיע במידע.
