# Fish Shell - כתיבת תוכניות בשפת פיש
## What & Why?
רצתם לכתוב תוכנית חדשה אבל לא יודעים איך להתחיל? כמו כל פרויקט חדש, זה יכול להיות מעט מעיד ומבלבל. כפי שכל תוכנית גדולה מתחילה באינטרפייס הראשון, כך גם פיתוח תוכניות מתחיל עם צורך ליצור פרויקט חדש. מתייחסים לתהליך הזה כ"התחלת פרויקט". כשמתחילים פרויקט חדש, אנחנו ניגשים לבניית התוכנית שלנו בראש פתוח עם כאלה או אחרים שיטות.

## How to:
כדי לפתוח פרויקט חדש בשפת פיש, ראשית צריך ליצור תיקייה חדשה לפרויקט. יש לך כמה אפשרויות לבחור במיקום ושם לתיקייה החדשה של הפרויקט. לדוגמה: 
```Fish Shell
mkdir my_project
```
כעת ניתן להכנס לתיקייה החדשה וליצור קבצים מתאימים לפרויקט כפי שנדרש. ניתן גם להשתמש בפקודות כמו ```touch``` ו ```nano``` כדי ליצור קבצי קוד בתיקייה החדשה. 
כדי להפעיל את הפרויקט שלנו, ניתן להשתמש בפקודת ```cd``` כדי לעבור לתיקייה החדשה. 
כעת אפשר לכתוב קוד בשפת פיש ולהריץ אותו באמצעות פקודת ```fish <file_name>.fish```, כאשר <file_name> הוא שם הקובץ שברצונכם להריץ. הפקודה תפעיל את התוכנית ותציג את התוצאות בטרמינל.

## Deep Dive:
באופן כללי, ישנן כמה דרכים לכתוב תוכניות בשפת פיש ולפתוח פרויקטים חדשים. אחת השיטות הפופולריות היא קיצורי המקלדת. קיצורי המקלדת הם קוד שנכתב מראש וניתן להפעילו בטרמינל לצורך תוכנית מסוגים שונים. יכולת זו מאפשרת ישימות שונות ומאפשרת למשתמש להקל על התכנון וביצוע התוכניות. בנוסף לכך, ניתן גם להשתמש באתרים כמו GitLab, GitHub ו-Bitbucket לבניית פרויקטים ושתפם עם אחרים.
