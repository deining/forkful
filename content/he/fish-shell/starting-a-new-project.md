## למה

דווקא פיש של הוא קונסולת הפקודות שמביאה לכם את כל היתרונות של מספר קוד ופעולות שאפשר לבצע בעזרתה, כולל ניהול יעיל של פרוייקטים חדשים.

פיש של הוא חזקה, מהירה ויעילה ביותר והיא מציעה גם אפשרויות לבניית מנהלי פאקטת שמאפשרים יעילות רבה יותר עם פרוייקטים גדולים.

## איך לעשות

תחילה, צריך להוריד את פיש של מהאתר הרשמי למערכות ההפעלה שלנו. לאחר ההורדה וההתקנה, ניתן לפתוח את פיש של ולהתחיל לכתוב בו את הפקודות שאנחנו רוצים.

לדוגמה, ליצור פרוייקט חדש, ניתן לכתוב:

```Fish Shell
mkdir new_project
```

לאחר מכן, יש לנו להתקין חבילות נוספות לפני שנוכל להתחיל לעבוד על הפרוייקט החדש. לדוגמה, ניתן להתקין חבילת פתרון קוד כמו Git עם הפקודה:

```Fish Shell
pacman -S git
```

לאחר ההתקנה, ניתן להתחיל לכתוב קוד ולנהל את הפרוייקט באמצעות פיש של.

## Deep Dive

להלן כמה נקודות שיכולות לעזור לנו בניהול פרוייקטים בפיש של:

- השימוש ב- "הטיפול" בינאריים  גם בנסין בפיקט פיש של, ניתן גם להוסיף ספריות בינאריות לפרוייקטינץף באופן תוכנן לבניה עם חומת לשתילים.

- הנסיך הו יחזר לשימוש בחפש מביאה עניין אי שם בכתיבה של פרוייקטים חדשים.

- שירותים יש לך ליוש
