# למה
 מדוע אחד ירצה לבצע המרת מחרוזת לאותיות קטנות בשפת פיש ייתכן מאוד שמחרוזות בפיש הן ארגומנטים משמעותיים לפקודות ותוכניות מגוונות, ולכן יכול להיות רלוונטי והועיל להמיר אותן לאותיות קטנות.

## איך לעשות זאת
כדי לבצע המרת מחרוזת לאותיות קטנות בשפת פיש, נוכל להשתמש בפקודה `string tolower` ולהעביר אליה את המחרוזת שברצוננו להמיר. לדוגמה:

```Fish Shell 
string tolower "HELLO WORLD"
```

פלט:
```Fish Shell 
hello world
```

ניתן גם להשתמש בפקודת `set` כדי להשתמש במחרוזת הממוקמת במשתנה ולהעביר את התוצאה למשתנה חדש, כך:

```Fish Shell 
set message "HELLO WORLD"
set lower_message (string tolower $message)
echo $lower_message
```

פלט:
```Fish Shell 
hello world
```

## עומק ניתוח
כדי להבין איך פקודת ההמרה לאותיות קטנות עובדת, נדרוש מספר כלים ויכולות בשפת פיש. כשאנו משתמשים בפקודה `string tolower`, השפה מזהה את סוג התווים שבתוך המחרוזת ומפעילה את הפונקציה הרלוונטית עבור כל תו על בפני עצמו. כמו כן, זאת אומרת שהתוצאה תהיה תמיד מחרוזת חדשה בתבנית זהה למחרוזת המקורית, אך עם האותיות המומרטות לאותיות קטנות.
