##למה
 בעולם התכנות ישנם מצבים רבים שבהם צריך להמיר את המחרוזת שנמצאת באופן גדול לאותיות קטנות. לדוגמה, אם המשתמש מכניס טקסט באותיות גדולות או ריבועיות, רצוי להמיר אותם לאותיות קטנות לצורך עבודה נוחה ויעילה. כתבת הבלוג הזו תסביר איך לבצע את ההמרה הזאת ב- Fish Shell ומדוע כדאי לעשות זאת.

##כיצד לעשות את זה
להמיר מחרוזת לאותיות קטנות ב- Fish Shell פשוט מאוד. כדי לבצע את המשימה, יש להשתמש בפקודת `string tolower` ולציין את המחרוזת שברצונכם להמיר כפרמטר. למשל:

```Fish Shell
string tolower "טקסט באותיות גדולות"
```

פלט של הפקודה הזו יהיה:

```Fish Shell
טקסט באותיות קטנות
```

בנוסף, ניתן גם להשתמש בפקודת `set` כדי לשמור את המחרוזת המומרת לאותיות קטנות כמשתנה ולהשתמש בו בהמשך בקוד. למשל:

```Fish Shell
set string "טקסט באותיות ריבועיות"
set string (string tolower $string)
echo $string
```

פלט של הפקודות הללו יהיה:

```Fish Shell
טקסט באותיות ריבועיות
```

ככל שתשתמשו יותר ב-Fish Shell תגלו שישנם המון פקודות שיסייעו לכם בעבודה עם מחרוזות ועוד.
