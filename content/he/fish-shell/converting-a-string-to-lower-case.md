## מה זה ולמה?
המרת מחרוזת לאותיות קטנות היא תהליך שבו משנים את כל האותיות במחרוזת לאותיות קטנות. פעולה זו נדרשת כאשר רוצים להשוות מחרוזות ולהיות בטוחים שהן תואמות, לפני שמשתמשים בהן בתנאי מסוים או כשרוצים להציג את המחרוזת בפורמט יציב. מתכנתים מבצעים את הפעולה הזו כדי להבטיח תיאום בין מחרוזות, וכדי לעבוד עם נתונים נכונים ומדויקים.

## איך לעשות זאת:
משתמשים בשפת התכנות Fish Shell כדי להפעיל את הפקודה להמרת מחרוזת לאותיות קטנות. הנה כמה דוגמאות של קוד שמפעיל את הפקודה ומראה את התוצאה:

```Fish Shell
echo "Hello World" | tr '[:upper:]' '[:lower:]'
# תוצאה: hello world

set str "This iS a TeSt"
echo $str | tr '[:upper:]' '[:lower:]'
# תוצאה: this is a test
```

## העמקה:
מרת מחרוזת לאותיות קטנות היא פעולה הנמצאת בשימוש מאז תחילת ההתפתחות של שפות התכנות. אחת הפעולות המוכרות אחרות להמרת מחרוזת לאותיות קטנות היא פקודת LOWERCASE בשפת Bash. פקודה זו משתמשת בספריות חיצוניות כדי להבציע את הפעולה ולא תמיד אפשר לסמוך עליה בכל המערכות. בנוסף, פעולה זו נדרשת במצבים רבים כאשר רוצים לבצע השוואה של מחרוזות או להציג תוצאות בפורמט קבוע.
