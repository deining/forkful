## למה

כשאנחנו עובדים עם אתרים אינטרנט, לעיתים קרובות אנחנו צריכים להשתמש בנתונים מתוך קוד HTML. פירושו של לפרק את האתר ולקבל רק את מה שאנחנו רוצים, כמו לדוגמה מידע על מוצרים או מחירים. בכתבה הזו נלמד כיצד להשתמש בשפת תכנות Fish Shell כדי לפרק קוד HTML ולקבל את המידע שנחוץ לנו.

## איך לבצע פעולת פירוק קוד HTML בעזרת Fish Shell

בכדי לפענח את הקוד ה-HTML ולקבל נתונים ממנו, נצטרך להשתמש בכמה דברים שכדאי לקחת בחשבון מראש:

1. שפת התכנות Fish Shell - היא מספקת את כל הפיצ'רים שאנחנו נזדקק
2. הספרייה (library) libxml2 - היא תעזור לנו לעבוד עם קוד XML ולמצוא את הנתונים שאנחנו מחפשים

כדי להניח את הבסיס לעבודה עם פירוק קוד HTML, נצטרך להתקין מראש את הספרייה libxml2 באמצעות הפקודה הבאה:

```
sudo apt-get install libxml2
```

לאחר מכן, ניצור קובץ ריק כדי לכתוב את הקוד שלנו ונפתח אותו עם התכנית fish. בכדי לקרוא את קוד HTML ולשמור את הנתונים שנמצאים בו, נשתמש בפקודה הבאה:

```
set -l html (curl -s "https://www.example.com" | xml2json)
```

בתחילת הפקודה, אנחנו משתמשים בפונקציה curl כדי לקרוא את הקוד ה-HTML של האתר. לאחר מכן, אנחנו משתמשים בפקודת xml2json כדי להמיר את הקוד לפורמט JSON, שקל יותר לפרוק ולעבוד איתו. בסוף, אנחנו שומרים את התוצאה במשתנה בשם html.
