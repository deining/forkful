כתיבת של מאמר על פקודת Fish של בעברית

## מה ולמה?
פרקה של תאריך מחרוזת היא התהליך של קילבה תאריך ממחרוזת לפורמט שניתן לעבוד איתו בסביבת תכנות. תהליכז משמעו שהתאריך המופיע כמחרוזת, כמו "20 יוני 1996" נוכל להמיר לפורמט עם חלקים נפרדים, כמו יום או חודש, וכן לבצע פעולות חישוב נוספות עם התאריך. בתור מתכנתים, אנחנו עושים זאת כדי לעבוד עם תאריכים בצורה קלה ונוחה.

## כיצד לעשות?
כאשר אנחנו רוצים לפרק תאריך ממחרוזת בפקודת Fish, ישנן שתי אפשרויות: השתמש בתכנית מוכנה או לייצר תכנית משלנו. אם נבחר באפשרות הראשונה, נוכל להשתמש בפקודה `date` עם האופציה `-f` כדי לציין את פורמט התאריך שאנחנו רוצים. לדוגמה, נרשום בפקודת השלב הבא:
```
date -f "%H:%M %d/%m/%Y" "14:36 20/06/2021"
```
ובתוצאה נקבל:
```
14:36 20/06/2021
```

אם נבחר באפשרות השנייה, נוכל להשתמש בפקודת `string match` כדי לקחת את החלקים הרלוונטיים של המחרוזת ולבצע פעולות חישוב עם התאריך. נראה את הדוגמה הבאה:
```
set date_string 14:36 20/06/2021
string match -r "(\d{2}:\d{2}) (\d{2}/\d{2}/\d{4})" $date_string --date $hour $date
echo $hour $date
```
והתוצאה תהיה:
```
14:36 20/06/2021
```

## מעמד עמוק
פריסת תאריך ממחרוזת קיימת כבר מזמן רב, וככל הידוע לי היא משמשת בפקודות ותכניות רבות של שפות תכנות שונות. לפיכך, אנחנו יכולים למצוא דוגמאות וכלים רבים באינטרנט לעזרה בפריסת תאריך ממחרוזת המתאימה לצורת העבודה שלנו. כמו כן, ניתן גם לכתוב פקודות משלנו שיישמעו את סגנונות המבנה של תאריך בתוך המחרוזת ויפעלו לפי הצורך.
