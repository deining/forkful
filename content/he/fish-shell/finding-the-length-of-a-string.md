## למה

מה הסיבה שאנשים מתעסקים בחיפוש אורך המחרוזת?

אין צורך להסביר שלאחרונה פקודת גבול הפכה לפופולרית יותר ויותר בעולם התכנות. כל מי שמתעסק בכתיבת סקריפטים או פיתוח אפליקציות מודע לחשיבות של פקודות הסיסמא בשפת התכנות שלו. כאשר מדובר בפקודת גבול קטנה כגון מציאת אורך המחרוזת, ניתן לחסוך זמן וכוחות רבים, ובעזרת הפקודה הנכונה ניתן לכתוב קוד יעיל יותר.

## כיצד לעשות

תחילה נכנס לשפת השאול ואחר כך נבצע פונקציה פשוטה למציאת אורך המחרוזת. ניתן לעשות זאת על ידי הקלדה של הפקודה הבאה:

```Fish Shell
string length "hello world"
```

ימצא את אורך המחרוזת ויחזיר את התוצאה המתאימה:

```Fish Shell
11
```

כמו כן, ניתן גם לבצע את הפונקציה עם משתנה בעזרת סימן הנקודה (.):

```Fish Shell
set str "Hello, World!"
echo (str length)
```

הפקודה תחזיר את התוצאה הבאה:

```Fish Shell
14
```
