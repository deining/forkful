# למה
יצירת קובץ זמני נחשבת לחלק חשוב בתכנות בקונסול בשפת Fish Shell. זה מאפשר לנו ליצור קובץ שימש כפתרון זמני לצורך הקוד שלנו, ולאחר מכן למחוק אותו כשסיימנו את העבודה שלנו. זה יכול להיות מועיל במקרים של צורך לעבוד עם קבצים זמניים או לשמור מידע זמני בתוך המחשב.

# כיצד ליצור קובץ זמני בשפת Fish Shell
בשפת Fish Shell קיימים שני אפשרויות ליצירת קובץ זמני:

1. להשתמש בפקודת `mktemp` כדי ליצור קובץ זמני ריק. לפני זה, נוכל להגדיר את הכתובת של הקובץ הזמני בפרמטרים של `mktemp`, ואת התבנית של הקובץ באמצעות `--tmpdir` ו -`--template`. לדוגמה:
 
```Fish Shell
mktemp --tmpdir=/tmp --template="tempfile_XXXXX.txt"`
```
פקודה זו תוציא מספר רנדומלי של 5 מספרים בסוף השם של הקובץ כדי שייתאים לתבנית ותיצור קובץ זמני בכתובת `/tmp/tempfile_12345.txt`.

2. להשתמש בחסימת '{ }' כדי ליצור פקודה מרובה שתיצור את הקובץ ואת התוכן שלו בו יחד כאחד. לדוגמה:

```Fish Shell
>{ /tmp/tempfile.txt; echo "This is a temporary file" }
```
פקודה זו תיצור קובץ זמני בכתובת `/tmp/tempfile.txt` ותכתוב את המחרוזת "זהו קובץ זמני" בתוכו.

# חקירה מעמיקה
הנה מספר דברים שכדאי לדעת על יצירת קובץ זמני בשפת Fish Shell:

- קבצים זמניים נמחקים באופן אוטומטי כאשר פקודת השירשור (command substitution) הסתיימה. זה אומר שאין צורך למחוק אותם ידנית.
