## למה?

ממש כמעט בכל פרויקט של מתכנת, נדרשת התמרה של תאריך לתוך מחרוזת. הסיבה לכך היא שתכניות נמדדות בעיקר בפעם האחרון בהן התמרת התאריך המקורי לתוך מחרוזת. כמו כן, תמרת התאריך למחרוזת מאפשרת למשתמש לצפות בתאריך בפורמט שהוא נוח מבלי להשתמש בפעולות מתמטיות.

## איך לעשות את זה

```Fish Shell
set my_date (date -u "+%Y-%m-%d")
echo $my_date
```

תמונה: 2021-05-18

הכל מתחיל מפקודת התאריך, והפרמטר "-u" מסמל שנרצה לקבל את התאריך המעודכן באופן מעניין רק לרבע שנה. לאחר מכן, בתוך הפקודה נשתמש בפרמט המבוקש באמצעות הסימון "+%Y-%m-%d". הפורמט הזה מייצג את השנה (4 ספרות), החודש (2 ספרות) והיום (2 ספרות). כמו שאתה רואה, עבור תאריך כלשהו אתה תכול את המספרים הנדרשים באמצעות מקאתזים. לאחר התאריך נציג את המחרוזת שנוצרה בעזרת "echo" ונקבל תוצאה דומה למאתר לערוץ.

לאחר מכן, נמיר את התאריך בזמן האמיתי. למשל, נוכל לקבל את התאריך המעודכן בתור סוג נתונים אחוזים. כמו שאתה רואה, התאריך יכול להחזיר ערך אחריך ביצירת המחרוזת.

```Fish Shell
set my_date (date -u "+%Y-%m-%d")
set formatted_date (date -f "%Y-%m-%d" $my_date)
echo $formatted_date
```

תאריך שמים למשתמש כפתור החלטת השניות

```Fish Shell
set my_date (date -u "+%Y-%m-%d")
set formatted_date (date -f "%H:%M:%S" $my_date)
echo $formatted_date
```

מגיע בתוך טבלה בתוך התמונה

17:02:53
