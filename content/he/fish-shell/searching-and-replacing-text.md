# למה

מצאים ומחליפים טקסט היא פעולה נפוצה בעולם התכנות ובפרופרמינג בכללות. היא מאפשרת למשתמשים לשנות טקסט קיים בקובץ או בתוכנית בקלות ובמהירות, וכך לערוך ולשפר את הקוד שלהם.

# איך לעשות זאת

תוכניות הפרופרמינג יכולות לכלול עשרות, אם לא מאות שורות של טקסט. כאשר מחפשים להחליף את מילה או משפט ספציפי בתפריט, ישנן כמה שיטות שונות לעשות זאת. לדוגמה, אם נרצה להחליף את המילה "שלום" במלים "היי" בכל הטקסט הקיים בקובץ "myfile.txt", נוכל לעשות זאת באמצעות הפקודה הבאה:

```Fish Shell
sed -i 's/שלום/היי/g' myfile.txt
```

בפקודה זו, אנחנו משתמשים בכלי ה-"sed" בפירוש "החלף בתוך הקובץ" ("-i") על מנת לסמן את המקום שבו נרצה להחליף את המילה "שלום" ולהחליף אותה במילת "היי". האפשרות "g" מציינת שנרצה לחלף את כל המופעים של המילה בקובץ, לא רק את המופע הראשון.

נוכל גם להשתמש בכלי "awk" כדי להחליף טקסט בצורה יותר מתוחכמת ומדויקת. לדוגמה, אם נרצה להחליף את המילה "שלום" בתוך משפט מסוים בקובץ "myfile.txt", נוכל לעשות זאת על ידי הפקודה הבאה:

```Fish Shell
awk '{ gsub(/שלום/,"היי",$1); print }' myfile.txt
```
