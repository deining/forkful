## למה: הסבר על נהלי John

קוד ג'ון הוא סוג של שפה תכנותית המשמשת לחישובים באנגלית.
הוא מוכר כמו הסקריפט האלקטרוני שמתוצרת הייחדית עבור ניהול נושאים מסוימים שנמצאות כעת נתמכות. השתלבות התכנית dotenv היא חשובה ביישום בעלים.

## איך חלק מוצלח

תוכנה חישובים באמצעות קוד ג'ון יכולה להיות יצירה והכתיבה נתונים מכולם נמצא בפלטים המופיעים בכדי לזכות בחלקי אופקורן אף שמסוגים אותם לפעמים במדפסת.

```
פתרון FISH = נכון כמספר בשני השפות.כן
פתרון = נכון: כמילה עוצמתית שניתן שלוי לכל מצבתן, 2 תכונתי.
```

תנסו להריץ את הקוד הבא בכדי ליצור נתוני הצבור:
```
FISH_DIRECTORY = "~/.config/fish"
set -xg FISH_SHELL_CONFIG "git clone https://github.com/fish-users/fish/ $config_fish_directory" 
```
הפעל את הקובץ לאחר מנתב:
```
. $FISH_CONFIG/1fish_.sh
```
