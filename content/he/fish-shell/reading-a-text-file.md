# למה
רק 1-2 משפטים מסבירים *למה* מישהו ירצה לקרוא קובץ טקסט.

בתכנות בשפת הפקודות של Fish Shell, קריאת קבצי טקסט היא חלק חשוב מהתהליך. היא מאפשרת לנו לגשת למידע חיוני בקבצים ולהשתמש בו כדי לבצע פעולות שונות וליצור סקריפטים מתקדמים. בכתבה זו נלמד איך לקרוא קובץ טקסט בFish Shell ונעשה טיול עמוק לתוך הנושא.

## איך לעשות זאת
תארו לנו שאתם צריכים לקרוא קובץ טקסט שמכיל מידע חשוב לפעולת הסקריפט שלכם. כדי לקרוא את הקובץ בFish Shell, נשתמש בפקודת `cat` ונציין את השם של הקובץ.

```Fish Shell
cat file.txt
```

בדוגמה זו, אנו קוראים את תוכן הקובץ "file.txt" ומציגים אותו במסך. אם נרצה להעביר את התוכן של הקובץ לתוך משתנה, נשתמש בתווים `>` ביחד עם הפקודה `cat`.

```Fish Shell
result = (cat file.txt)
```

בדוגמה זו, אנו מוסיפים את תוכן הקובץ "file.txt" לתוך המשתנה "result" ונוכל להשתמש בתוכן זה במשך הסקריפט שלנו.

## טיול עמוק
כאשר אנו קוראים קובץ טקסט, ניתן לבצע מספר פעולות נוספות כדי לעבוד עם התוכן המוצג. לדוגמה, ניתן להשתמש בפקודת `head` כדי להציג את השורות הראשונות בקובץ ובפקודת `tail` כדי להציג את השורות האחרונות.

```Fish Shell
head file.txt
```
