## מה ולמה?

קריאת קובץ טקסט היא תהליך בו מקבלים מידע ממקור מסוים בתוך קובץ טקסט. כמו שכבר מתאר השם, קבצי טקסט הם פשוט קבצים המכילים טקסט בפורמט מסוים. אנו קוראים קבצי טקסט כדי לקבל מידע חשוב ונתונים מגוונים שנמצאים בתוך קובץ זה. תהליך הקריאה של קובץ טקסט הוא כלי חשוב שמאפשר לנו, כמתכנתים, לעבוד עם מידע ולהשתמש בו כדי לבנות תוכניות מורכבות יותר.

## איך לעשות זאת?

כדי לקרוא קובץ טקסט בשפת התכנות Fish Shell, ניתן להשתמש בפקודה `cat` ולציין את הנתיב של הקובץ. לדוגמה, אם יש לנו קובץ טקסט בשם "data.txt" שמכיל מידע על עובדים בחברה שונים, נוכל להציג את התוכן שלו באמצעות הפקודה הבאה:

```
cat data.txt
```

כאשר נריץ את הפקודה זו, נוכל לראות את המידע המיצג את העובדים ואת החברות בהן הם עובדים.

## ירידה לעומק

כשקרואים קובץ טקסט, אנו עלולים להשתמש בפקודה שנבחרה כדי לקרוא את הקובץ ולהציג את המידע. אך ישנן אפשרויות נוספות לקרוא קובץ טקסט בשפת Fish Shell, כמו למשל להשתמש בפקודת `head` על מנת להציג רק את השורות הראשונות בקובץ, או להשתמש בפקודת `tail` כדי להציג את השורות האחרונות. כמו כן, ישנן אפשרויות נוספות לעבוד עם קבצי טקסט בשפת Fish Shell, ואתה מוזמן לחקור את האפשרויות השונות הקיימות.
