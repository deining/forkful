# למה:

ניתן להיות מושך לכתיבת קובץ טקסט בשפת פיש כדי ליצור סקריפטים פשוטים שיעזרו בביצוע פעולות אוטומטיות וקלים יותר במכשיר המחשב שלכם. זה יכול להיות שימושי במיוחד אם אתם משתמשים בפיש כשפת סקריפטים עבור מערכת ההפעלה שלכם.

## איך לעשות:

```Fish Shell
echo "שלום, עולם!"
```

פקודה זו תדפיס את המחרוזת "שלום, עולם!" בטרמינל שלכם. כמו כן, ניתן להשתמש בקובץ טקסט עבור סקריפטים יותר מורכבים ומתוחכמים.

```Fish Shell
#!/usr/bin/fish
for i in (seq 1 10)
    echo $i ". שלום, עולם!"
end
```

פקודה זו תדפיס את המחרוזת "שלום, עולם!" עם מספרים מ-1 עד 10 בצדק מהחלון הטרמינל שלכם. כדי לנסות את הסקריפט הזה, נתקינו את הפיש שלנו תחילה, ואז נכתוב את הפקודות הבאות בצד מסומן ה"prompt" של הטרמינל שלנו:

```Fish Shell
chmod +x שם הקובץ.sh
./שם הקובץ.sh
```

## בידור נעמך:

עכשיו שאתם יודעים איך לכתוב קובץ טקסט פשוט בפיש, נוכל לעבור לעומק יותר וללמוד איך לתכנן סקריפטים מתוחכמים יותר ומתואמים עם הפקודות שלנו. אתם יכולים ללמוד עוד על המודל של פיש, פיש פונקציות, פיש גלובליות, וכו '.
