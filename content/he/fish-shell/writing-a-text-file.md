## מדוע

כתיבת קובץ טקסט יכולה להיות כלי חשוב עבור מתכנתים ושימוש לצורך ניהול פרויקטים ותיעוד. הקוד שלכם עשוי להיות מורכב ולכן כתיבת קובץ טקסט יכולה לאפשר לכם לאחסן חלקים שונים של הקוד בקובץ אחד נגיש ובכך לייעל את התהליך שלכם.

## כיצד לעשות זאת

```fish shell
echo "כתבתי קובץ טקסט במכשיר שלי" > my_text_file.txt
cat my_text_file.txt
```

```
כתבתי קובץ טקסט במכשיר שלי
```

כדי לכתוב קובץ טקסט בשפת Fish Shell יש להשתמש בפקודת `echo` ולציין את הנתיב ושם הקובץ שאליו ברצונכם ליצור או לכתוב. בדוגמה הנ"ל הרצתי את הפקודה echo והוספתי את הטקסט "כתבתי קובץ טקסט במכשיר שלי" כתוכן לקובץ החדש שפיי שראת.

## לחקור עמוק יותר

כאשר אתם כותבים קובץ טקסט בשפת Fish Shell, אתם יכולים להשתמש בפקודות נוספות כמו `touch` ליצירת קובץ ו `nano` לפתיחת עורך טקסט מובנה שיעזור לכם לערוך את הקובץ ולהוסיף תוכן נוסף לפי הצורך.
