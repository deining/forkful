# למה

מחיקת תווים התואמים לתבנית היא כלים נפוץ בתכנות עם לשונית של Fish Shell. ישנם מספר סיבות למה ניתן יהיה לרצות למחוק תווים התואמים לתבנית: לצורך עיבוד טקסטים, כדי לבצע התאמות לתבנית ועוד.

# איך לעשות זאת

השימוש בפקודות של Fish Shell נעים וקל להשתמש במחיקת תווים התואמים לתבנית. תחילה יש להקיש על לשונית הפקודות ולמצוא את הפקודה של "grep". מתוך שם, ניתן להשתמש בתפקיד של הפקודה כדי למחוק את התווים הצמודים לתבנית מתוך הטקסט.

```Fish Shell
grep -v "תבנית התאמה" שם_קובץ
```

פקודת "grep" מחזירה את כל השורות שאינן מתאימות לתבנית שצוינה, וכך מתאפשר למחוק את כל התווים התואמים לתבנית מתוך הטקסט. 

# לפנות עמוק יותר

כדי לפנות עמוק יותר במחיקת תווים התואמים לתבנית, ניתן להשתמש בפקודת "sed" בתוך לשונית הפקודות של Fish Shell.

```Fish Shell
sed -e "s/תבנית התאמה/תבנית מחיקה/g" שם_קובץ
```

פקודה זו תחליף כל מופע של התבנית הראשונה בתבנית השנייה, וכך תמחק את כל התווים התואמים לתבנית מתוך הטקסט שבשם הקובץ המצוי.
