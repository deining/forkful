**## למה**
מחיקת תווים המתאימים לתבנית היא כלי חזק שניתן להשתמש בו כאשר מתעסקים עם שפת תסריטים כמו פיש של. זה יכול לחסום קוד מיותר ולאפשר פילוח יעיל ומיוחד של הנתונים.

**## כיצד לעשות**
למחיקת תווים המתאימים לתבנית בפיש של, אתה יכול להשתמש בפקודת ```transpose```. למשל, אם אתה רוצה למחוק את כל האותיות הגדולות מתוך תווים מסוימים, תוכל לעשות זאת עם הפקודה הבאה:

```
echo "Hello World" | tr -d 'A-Z'
```

פקודת ```tr -d``` מסירה את כל התווים שמוכנסים לתוכה, כך שבפקודה זו התווים הגדולים הינם התווים מהתא הראשון ועד התא האחרון של הפקודה. כדי לכתוב פקודת טרנספוז זו בצורה נוחה יותר, אתה יכול להשתמש בפקודת ```invert``` כך:

```
function invert
	transpose | tr '[A-Z]' '[a-z]'
end
```

הפקודה ```tr``` בשורת התגובה הראשונה מסירה כל תווים גדולים, ואז הזזת ה-S אל התא הכי תחתון תימחק את התו הראשון מהכתב, לאחר מכן התו השני ימחק כתובח וכו 'לכל הדרך לתחתון נתונים. אם אתה רוצה לחסום את התוי הגדולים המקושרים ליו תוצאות לאחר קיימים, אתה יכול להשתמש בהפקודה המיוחדת של ה-S במבנה מסוג list: הארב ספיך להפקודה הזו במטרה לקבל תא ראשון לחסום את כל הוא משתנה התווים הגדוליות על חסום של כל התווי הגדונים המתאימים.

```
echo "Hello World" | tr '[A-Z]' ' '
```
