## למה

המחוק את התווים שתואמים מטרה &#x2013; פעולה ניתנת לביצוע בשפת תסדי למערכת הפעלה Mac OS X, קייסטה, וינדוס ו Linux. פעולה זו מאפשרת למשתמש להסיר תווים מסוימים מקלט בקלות ובמהירות, מה שיכול להיות מאוד מועיל בפיתרון בעיות קטנות בסקריפטים או בקבצי טקסט באמצעות מסנן טקסט.

## איך לעשות זאת

הקישור "search" משתמש כמסנן למתווך ג'וסטו מס המובנה כדי לחפש אחר אותיות מסוימות או מדפסות.

``` 
fish shell
string
t{a,p} # : ranges
string | psub
```

פונקציית המוועדה או הפקודה. יתכן גם שפונקציות אלו יכולות לעשות שימוש בפרמטרים נוספים כדי להגביל את התווים שלהם.

## יצוב כנסיית תאונים

בכדי לבחון את המצב ולנתק את התווים לפי תבנית, ניתן להשתמש בפונקציה " regex search":

``` 
FISH_REGEX_MATCH_REGEXS
" # \체중AB (weight)
!
re-match
```

כדי לבדוק את התווים המתאימים לתנאים של פקודות עריכה יותר עמוקות, כגון "search and replace," ניתן להשתמש בסקריפטים שמשתמשים בסכמי התבנית. באמצעות פיתוח והשתמש בלולאות, ניתן להעביר דרך כל תו אחד ולהחליפו בתווים אחרים על בסיס אופי התבניות המוגדרות.
