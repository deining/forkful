## למה

חשוב לדעת את התאריך הנוכחי במערכת שלכם? אם אתם עובדים עם פקודות זמן או כאןכולשתכו, כדי לייצר תיעוד או כדי לעזור לכם לעקוב אחרי תהליכים שלכם, ייתכן כי תרצו להתעדכן בתאריך הנוכחי. בכתבה זו נלמד כיצד לקבל את תאריך היום הנוכחי באמצעות הפקודה של Shell.

## איך לעשות זאת

בשפת Fish Shell ישנם כמה אפשרויות לקבלת תאריך היום הנוכחי. נדגים כאן שתי אפשרויות עם דוגמאות קוד לממש ופלט תוצאה רלוונטית:

```Fish Shell
# אפשרות א': השתמשו בפקודת date
date +"%d/%m/%Y"

# אפשרות ב': השתמשו בפקודת strftime לעיצוב יותר מדויק של התאריך
printf "%(B %d, %Y)" (strftime '%b %d, %Y')
```

### פלט תוצאה:

```
# אפשרות א':
22/11/2021

# אפשרות ב':
נובמבר 22, 2021
```

ישנם עוד אפשרויות רבות לקבלת תאריך היום הנוכחי בשפת Fish Shell, אתם מוזמנים לבדוק את התיעוד של Fish Shell עבור פקודות נוספות ולהשתמש במה שמתאים לכם.

## נכנסים לפרטים

כדי להבין איך הפקודות date ו-strftime מזיגים ביחד כדי ליצור תאריך נוכחי, חשוב להבין כמה נקודות מרכזיות.

### נקודה מספר 1: הפקודה date

הפקודה date מקבלת פרמטרים נוספים על מנת לשנות את תעריך הפלט שלה. אחד הפרמטרים הנפוצים הוא +"format", שמאפשר לכם להגדיר את שְׂגִיב רוויה של התאריך. אחרים פרמטרים שניתן להשתמש בהם כוללים את -I (מודעת שעון חור
