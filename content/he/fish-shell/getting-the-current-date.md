## למה

כתיבת תוכניות תוכנות לעשות פעולות שונות על בסיס יומיום, ואתם זקוקים לקבלת התאריך הנוכחי על מנת להשתמש בו בתוך התוכנית שלכם. אם אתם משתמשים בפיש של, תוכלו בקלות לקבל את התאריך הנוכחי באמצעות כמה פקודות פשוטות. קראו הלאה כדי ללמוד איך.

## איך לעשות זאת

```Fish Shell
set date (date +'%Y-%m-%d')
echo $date
```

בדוגמה זו, אנו מגדירים משתנה נקרא "תאריך" ומתאריך אותו עם התבנית הנכונה. לאחר מכן, אנו מדפיסים את התאריך באמצעות הפקודה "echo". כעת, כאשר אתם מריצים את התוכנית ומבקשים את התאריך, תקבלו את התאריך הנוכחי בפורמט מובנה. ניתן גם להוסיף נוסחאות נוספות כמו שנה, חודש ויום באמצעות נוסף תבנית בפקודת התאריך.

## מעמקים

כעת שאנו יודעים איך לקבל את התאריך הנוכחי בפיש של, ניתן לצפות במספר נתונים נוספים הקשורים לתאריך. למשל, ניתן להשתמש בפקודה "date -u" כדי לקבל את התאריך הנוכחי בפיזור עולמי. אם אתם רוצים להחליף שנה או חודש, ניתן לעשות כך על ידי הוספת או הסרה של מספרים יחד עם התבנית המתאימה. כמו כן, אנו יכולים להשתמש בפקודה "date -d" כדי להציג תאריכים מסוימים בעתיד או בעבר.
