# למה

ניתן להשתמש בביטויים רגולריים כדי לחפש ולעבד טקסט בצורה מתוחכמת יותר. באמצעות ביטויים רגולריים, ניתן לחפש דפוסים מסוימים בטקסט ולבצע פעולות כגון החלפות ועיבוד תווים מיוחדים.

# איך לעשות

## הבנה ביטוי רגולרי

כדי להשתמש בביטויים רגולריים בפיש של, יש להבין את התחביר הבסיסי שלהם. לדוגמה, ביטוי רגולרי יכול להיות מכיל אותיות, מספרים, או תווים מיוחדים כגון נקודות וסוגריים. על מנת להתאים לביטוי רגולרי ספציפי, ניתן להשתמש בתווים מיוחדים כגון הסימן הכוכב, המצביע על כל תו אחד שחלק מהמילה.

לדוגמה, ניתן להשתמש בביטוי הרגולרי `^hello`, המתאים לכל מילה רגילה שמתחילה ב- "hello".

## שימוש בביטויים רגולריים בפיש של

כדי להשתמש בביטויים רגולריים בפיש של, יש להשתמש בפקודת `string match`, ולציין את הביטוי הרגולרי שברצונכם לחפש, וסוג הפעולה שברצונכם לבצע.

לדוגמה, ניתן להשתמש בפקודה הבאה כדי לחלץ את כל המספרים הסרוקים מתוך תוויות זוגיות:

```
string match "(\d+)" -- "This is 123 a 4567 sample"
```

פקודה זו תחזיר את התוצאה הבאה:

```
123 4567
```

## אימות מילים

ביטויים רגולריים גם יכולים לשמש לאימות מילים ספציפיות. לדוגמה, ניתן להשתמש בביטוי הרגולרי `^[A-Z]+$` כדי לאמת רק מילים שמורכבות מאותיות
