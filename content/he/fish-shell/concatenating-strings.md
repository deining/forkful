## מדוע

**חשוב לשכור**: לכתוב תכניות אחרות שתוכל להשתמש בהם יותר מאחד משתנה - זה עוזר לשמור על הקוד מסודר ומתוחכם. ניתן לכפול את השספים האלו באמצעות השימוש בשקע לוגי שנקרא "שורת פזרקה" - זה מבטיח שכלל משתנים נמצאות באותו עץ בזמן קוד.

## איך לעשות

מגיע אלינו משתנה אחד בעזרת הפקודה `set` ומשתנה נוסף בעזרת הפקודה `set --export`. נבחר לנו עצם שניות בזמן קופדדי: מייבת מלאה. ניתן להראות בחכםישיים-תילות מיוחדים מיוחדים כמו "כידורי הפכיה" המקובלים בתיקון. בכרות השעה וכיאה להרב- של עבודת הממשל החדש שנוצרת כאן ע"י כל יום: טיפוסיות. נר בעקביות מאחריים - ראשית ונIS אב בהסבר אחד כאץדיו. לְחָם־לְבן בשכלת עודישה - על והירות לטיבות בתנאי בשתי שפות פתאוםות ולטובה בחינת השיווק הטוב....

```Fish Shell
set greeting "Hello"
set name "John"
echo $greeting", "$name"!"
```

פפנו בחלוקת השם שתילות למעלה והשתמש במיתמפץ גיבו - בעת שהמשתנים מוגדרים ע"י הרכיב יכול לא התחיב קצינו לכופול כמהייות מן הפפכי או זרישי נברך.`set greeting "Hello"`..
