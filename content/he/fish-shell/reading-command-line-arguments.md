## מה ולמה?

קוראי פקודות מחשב הם פונקציות קטנות שיכולות לקרוא נתיבים לתוך הפקודה שאנו מריצים בסביבת השל כדי לקבל מידע נוסף או כדי לשנות את התנהגותה. קריאת פרמטרי שורת הפקודה היא פעולה חשובה למתכנתים בכדי לעבוד ביעילות על הפרוייקטים שלהם ולהפוך את קודם ליותר מנוהל ומודולרי.

## איך לעשות:

תוכניות קטנות שמאלו כדי לטפל בפקודות שורת הפקודה בסביבת השל שלהם צריכות לעשות שימוש בפונקציה שנקראת `argparse` כדי לקרוא את הנתיבים שהמשתמש הכניס בשורת הפקודה. להלן דוגמה של קוד בשפת יצירה הכי שימושית בפשוטות כדי לנתח נתיבים:

```
Fish Shell in foo.fish
---
#!/usr/bin/env fish

# Standard library function to handle command line arguments
function handle_arguments
  argparse -s ":"
end

handle_arguments $argv
echo $arg_path        # prints the value input for "-p" argument
echo $arg_file        # prints the value input for "-f" argument
echo $arg_output      # prints the value input for "-o" argument
```

## נכנס לעומק:

פתרונות נוספים עבור קריאת פרמטרי שורת הפקודה כוללים את השימוש בשפות תכנות שונות כמו שפות סקריפט ופייתון, אבל השימוש בפונקציות הכלליות הופך את קוד הבנייה לפשוט יותר ומנוהל בצורה נכונה. כדי להגביר את היעילות של הפונקציה שלך, ניתן לחפש תיעוד או סרטוני הדרכה על פתרון קריאת פרמטרי שורת הפקודה בשפה המיועדת לפרוייקט שלכם.

## ראו גם:

אם תרצו לדעת יותר על פקודת השל של דגים או על קריאת פרמטרי השורה הראשונה, ניתן לבקר באתר הרשמי של פקודת השל של דגים או להעביר את המשא ומתנה בפורום שלנו לקבלת עזרה על כל הצריכים. תרגם את מילים של הרשמה המדברות עבור אנשים שמבינים את הנושאים שאתה מנגן עם צבטיילה אדת של החושים לקראת הבדיקה שלכם איך שאנחנו מגיבים לכל היותר, בין אם לא כתב אחד ורוצה להיות בידי שלב או מנגיש של זוה בכדי שתוכל להביא אליך את אנשים הראלתיהים שכיל דבר שאיני הפומה ולכתב אותו לתוך המידע הצפוי.
