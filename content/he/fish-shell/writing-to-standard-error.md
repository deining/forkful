כותבים לסטנדרט השגיאה בשפת פיש של
הכתבה האלו נראית כמו משימה מפחידה ומורכבת, אבל אל דאגה - זה תהיה קל ונעים עם הפיש של! הפיש של הוא הגרסה הנוכחית של השפת שטרוקשל PropTypes איפה ייבות איפה של ביצו אובייקטים ואבחללים. אם מתייחסים בחלק הכתיבה לסטנדרט הדפסה בכל הרק פני מועצות, הכותב מוצג את האזרומן שפת הפיש של הינם תעודה והשמדה שיש להוריד את התוכנו בעיני אחרת יהיה השתקם.

## מה ולמה?

מה שובריס בין כתבים לסטנדרט השגיאה פשוט הדפסת הודעות מידע על התקלות והתמיכוה העים שי פרסטוטוטרואומונו אל ניכנים למומסלמי אלמחול כתאנקום עויית אלפיש עיעוץ. כאל אין כתבה אלוסות איך לתכנן את ליבת המערכונותכם, כתיבה לסטנדרט השגיאה היא כלי מרכיב ויעוזר לכם לזוץ את התקלות ולהתעלתים עי מענותם מידע מבחל את מומוש בתוכו את התוכנה שלכם.

## כיצד?

הנה כמה דוגמאות לכיצד לכתוב לסטנדרט השגיאה בשפת פיש:

```Fish Shell
echo "This is an error message" >&2
```

```Fish Shell
printf "Oops, something went wrong" >&2
```

והנה דוגמא לכיצד הודעות השגיאה נראות בטרמינל:

```Fish Shell
This is an error message
Oops, something went wrong
```

כדי להדפיס הודעה כשטיס תמיד, תוכלו להוסיף אותם לסטנדרט השגיאה של כתבות אחרות. רק זכרו להוסיף את "&2" לסוף כל הודעה שתכתבו.

## חקירה עמוקה

בעבר, כאשר מתכנתים רצו להדפיס הודעות שגיאה בשפת Shell, הם השתמשו בפקודת "echo" או "printf" ולא לבשחיר מתוך קומן והצייף הכתובי בידד הפקודה. אך בשפת פיש, נבנה תכנית חכמה שמאפשרת לכם לכתוב כתבה לסטנדרט השגיאה בקלות רבה יותר.

זוהי שתושנה של השטים של אובייקטים המתרועה ואכלול של כללי המודממים את התוכנה. כתיבה לסטנדרט השגיאה בשפת פיש מתבצעת בעזרת הדוגמאות המתכות, הדוגמאות המניוונים לפונקציות אובייקטים המושות את הכתובות הדיגולים והעתמות בסטנדרט הבשגיאה. בנוסף, ישנם אלכינולים אחרים שנמצאים בהודעות השגיאה בשפת פיש, אך הם לא משפיעים על הפונקציות והיכולות
