## למה

כל נושא הכתיבה לתקליטור הוא תהליך חיוני בחיי המתכנתים. לא רק עוזר לנו לאתר באילו חלקים בוגרים להקסים את הקוד שלנו, אבל גם חוסך לנו המון זמן ופוטנציאל לתקלות בקוד. הכתיבה לתקליטור בשפות כמו C++ או Java נהפכת לעסק מסובך ומורכב כאשר פונים אל השגיאות שלנו. במקרה כזה, הכתיבה לתקליטור בפיש של נגיד יכולה לעזור לנו לזהות ולהתעלם מכל הפרטים בעזרת חזרה טכני לתקליטור, מה שמספק את הנוחות והפשטות שאנחנו צריכים.

## איך לעשות

כדי לכתוב לתקליטור בפיש, ניתן להשתמש בפקודה `echo` יחד עם סים בשביל להפנות את התוצאה לתקליטור. הנה דוגמא של זה:

```Fish Shell
echo "זה טקסט לתקליטור" 1>&2
```

כאשר נריץ את הקוד הזה, הטקסט יופיע על התקליטור כאשר נשתמש במכירה של פיש להראות את השגיאות שלנו. בגרסת הווינדוז של פיש, השימוש בקוד הזה יריץ מכירה מסוג PowerShell, או בגרסת המאק של פיש יריץ מכירה מסוג Bash.
