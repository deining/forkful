## מדוע
במערכות תכנות מסוימות כמו Fish Shell, בדיקת תאריכים היא חשובה ליישומי המחשב שלנו. זה יכול להיות מועיל לשלטון על התאריכים המתאימים עבורנו, כמו כן גם לאפשר לנו לטפל בשלל אירועים המגיעים בבסיס יום יום.

## איך לעשות זאת
דהיינו שכבר קבלתי מבחן מספר איורים המספקים את התאריך המתאים לגיליון הפעולה. למשל, את תוכל להשתמש בתאריך לבדוק את התאריך שלך האחרון ושימו את עוד בתאריך הנוכחי.

```Fish Shell
set date1 (date "+%m/%d/%Y")
set date2 (date -d 'next Monday' "+%m/%d/%Y")
if test $date1 > $date2
echo "התאריך הראשון הוא התאריך החדש ביותר"
else
echo "התאריך השני הוא התאריך החדש ביותר"
end
```

Output:
התאריך השני הוא התאריך החדש ביותר
