# למה
למה כדאי לכתוב קוד שיגדיל את האותיות שבמחרוזת שנכתבה?

כתיבת קוד מעניינת ואתגרתית תמיד הייתה חלק אינטגרלי של התכנות. ייתכן שאת/ה מתעסק/ת עם פרויקט שמצריך שתנתח/י מחרוזות. אחת הפעולות הנפוצות ביותר בטיפול במחרוזות היא להגדיל את האותיות שלהן. זה יכול להועיל במספר מקרים, כגון מיון המחרוזות או עיצוב הפלט של הקוד.

# איך לכתוב קוד לגדול את האותיות שבמחרוזת
ניתן לעשות זאת באמצעות פונקציית פיש שנקראת `string capitalize`. בעזרתה, תוכל/י להגדיל את האות הראשונה של מחרוזת או את כל האותיות שבה. הנה דוגמה לשימוש בתוך קוד פיש:

```fish
set my_string "קוד לתכנות"
echo $my_string | string capitalize
```

פלט: `קוד לתכנות`

כדי להגדיל את כל האותיות שבמחרוזת, ניתן להשתמש בפונקציה `string lower` לפני הפקודה `string capitalize`, כך שהפלט יהיה באותיות גדולות.

לצורך נוחות, ניתן גם להגדיל את האותיות על כל המחרוזות באותו זמן באמצעות פקודת החד פעמית `string capitalize`, כך שהכתיבה תהיה קצרה יותר:

```fish
string capitalize "מחרוזת עם אותיות גדולות"
```

# העומק של הגדלת מחרוזות
הגדלת מחרוזות היא פעולה פעילה ונפוצה בתכנות. כדי להפוך למתכנת/ת יעיל/ה יותר, כדאי לשלב פעולות נוספות כמו בדיקה האם האותיות במחרוזת כולן גדולות או נמוכות, או החלפת מחרוזות עם מחרוזות אחרות. פונק
