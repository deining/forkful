אז מה זה הפרש כתיבה ולמה מתכנתים עושים את זה?

פרש כתיבה הוא פעולה שמשנה את רמת האותיות המיוחדות במחרוזת מתחת למילת המפתח, כך שהיא תהיה באותיות ראשונות גדולות. זה נעשה לפעמים כדי להיות עקביים עם סגנון הכתיבה, להדגיש מילים מסוימות או פשוט כדי להבחין בין מילים באופן ברור יותר.

איך מבצעים זאת?

```
Fish Shell מתאפיינת בכך שהיא מאפשרת לנו לבצע פרש כתיבה בפשטות ובמהירות. כדי לבצע זאת, נוסיף את התו השואף (<-) לפני המחרוזת שנרצה לשנות, ונשתמש בפקודה "capitalize" כדי לבצע את השינוי. לדוגמה:

```
set string "שלום לכולם"

capitalize <- $string
```

פלט:
```
שלום לכולם
```

פשוט וקל, נכון?

עוברים לעומק:

מבוא ראשון יכול להיות כי פרש כתיבה היא תכונה מאוד ידועה ומוכרת בכלים תכנותיים אחרים כמו Python ו-C. בעולם התכנות, ישנן גם אפשרויות נוספות לשנות את האותיות במחרוזת, כמו lowercasing ו-uppercasing, אך בפשטות שלנו ניתן להשתמש בפקודת capitalize כדי לבצע כך.

לפקודה יש גם את האופציה "capitalize-all" שתחזיר את המחרוזות שבהן כל המילים יהיו באותיות גדולות. ניתן לבצע זאת על ידי הוספת תווים אחרי התו השואף, כמו כך:

capitalize-all <- $string

פלט:
```
שלום לכולם
```

לכבוד הגנטלמן שייצר את פקודת capitalize עבור Bash, ישנו גם את הפקודה "ootstray" שלא פחות מבלסדך מתרגם את המחרוזת המתחילה באותיות גדולות. אפשר לחשב על זה כפקודת capitalize משולבת עם lowercasing שנותנת לנו גמישות נוספת כאשר מדובר באותיות באמצע המחרוזת. לדוגמה:

ootstray <- $string

פלט:
```
שלום להעדפה עבור
```

עוד דברים לראות:

מאמר זה כיסא בסיסי במידר כיצד לבצע פרש כתיבה בשפת פיש שלנו הנפלאה. אם ברצונכם לקרוא עוד על פקודות אחרות של הכיתה "string", או על השימוש במידר כדי לשפר את יכולותיו הצבעוניות, אפשר לעיין בתיעוד הרשמי של Fish Shell.
