## למה

יתרון גדול של שימוש בתכנות בסקריפטים הוא שניתן לבדוק באופן אוטומטי אם קיימת תיקייה במערכת הקבצים. הפעולה הזו יכולה לחסוך זמן ומאמץ במהלך פיתוח תוכניות ולמנוע אירועים לא צפויים במהלך ריצת התוכנית.

## איך לבדוק אם תיקייה קיימת

תכנות בשפת השקול שניתן לבדוק אם תיקייה קיימת בקלות. פקודת "test" מאפשרת זאת באמצעות האפשרות "-d" שמאפשרת לבדוק אם הארגומנט הנתון הוא תיקייה. תכנות בשפת Fish Shell מציעה דרך מהירה וקלה לבדוק אם תיקייה קיימת באמצעות הפקודה "status". כל מה שנדרש הוא להכניס את שם התיקייה או נתיבה לתוך הפקודה ולהריץ אותה.

```fish
# דוגמה לבדיקה אם תיקייה קיימת
test -d ~/Documents
# פלט: true

# דוגמה נוספת של בדיקת תיקייה בפקודת "status"
status ~/Pictures
# פלט: Path '/Users/username/Pictures' exists and is a directory.
```

## משימושים נוספים ופירוט נוסף

כדי לקבל מידע מפורט יותר על התבניות של הפקודות "test" ו-"status" ניתן להתייחס לתיעוד רשמי של Fish Shell. בנוסף, ניתן להשתמש בפקודת "test" גם בתוך לולאות כדי לעבור על מספר תיקיות בפעם אחת ולבדוק אם הן קיימות.
