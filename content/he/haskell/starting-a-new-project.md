# למה

מתחילים פרוייקט חדש בשפת התכנות הזו עשוי להיות מאתגר כאשר אתם מתחילים ללמוד את השפה, אבל יש המון סיבות משמעותיות לכתוב קוד ב־Haskell. השפה הזו מציעה תכונות מתקדמות כמו סוגיות ותיכנות פונקציונלית שעוזרות לייצור קוד נקי ויעיל.

# איך לעשות זאת

הנה כמה דוגמאות של קוד ופלט כדי להראות איך לכתוב ב־Haskell:

```Haskell
-- כדי להדפיס שלום בשפת התכנות Haskell נצטרך להשתמש בפונקציית putStrLn
main :: IO ()
main = putStrLn "שלום"
```
פלט: שלום

```Haskell
-- הגדרת פונקציה שמחברת שני מספרים
-- הפונקציה גם מקבלת פרמטרים מסוג אינטים
add :: Int -> Int -> Int
add x y = x + y

main :: IO ()
main = print (add 5 3)
```
פלט: 8

כמו שאתם רואים, ב־Haskell ניתן להשתמש בכמה סוגיות כגון פונקציות חכמות, פונקציות גביש וטיפוסים יכולים ליצור קוד מובנה וברור.

# צלילת משך

השכבה העמוקה של השפה הזו מתקשרת לתכנות בעזרת סימבולים ואין שימוש פעיל בלשוניות כמו בכמה שפות אחרות. אחד הדברים האהובים עליי ב־Haskell הוא היכולת ליצור טיפוסים מורחבים כמו רשימות וטיפוסים מורכבים.

רצוי להתחיל בעקיפת מקורות מידע על השפה כדי להבין טיפוסים, פונקציות והיררכיות. גם לכתוב קוד פשוט ולבדוק אותו באמצעות REPL (ר־אנטר פרינט לופפ).
