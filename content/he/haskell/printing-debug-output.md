## למה

להדפיס פלט מידע דיבאג יכול לעזור לנו לזהות באים בתכניות הייקל שלנו ולפתור אותם. זה גם יכול לעזור לנו לבדוק את ערכי המשתנים ולהבין את התהליכים שמתרחשים בשורת הכתרת במחשב.

## איך לעשות זאת

כדי להדפיס פלט מידע דיבאג בהסקל, אנו נשתמש בפונקציית הדפס טקסט `print`. בהמשך תוכלו לראות כמה דוגמאות של שימוש בפונקציה זו.

```Haskell
print "שלום עולם" -- פלט: שלום עולם
print [1,2,3]      -- פלט: [1,2,3]
```

בנוסף, אפשר להשתמש בפונקציה `show` כדי להדפיס את הערכים של משתנים או מבני נתונים מסוגים שונים כמו סטרינג, אינט, או רשימות.

```Haskell
let x = [1..5]
print (show x)    -- פלט: "[1,2,3,4,5]"
```

## נכנסים לעומק

בנוסף לפונקציות הדפס ו-showם, ישנם כמה דרכים נוספות להדפיס מידע דיבאג בהסקל. למשל, אפשר להשתמש בפונקציה `putStrLn` כדי להדפיס מחרוזות עם שורת חדש.

```Haskell
putStrLn "דוגמה" -- פלט: דוגמה
```

אם יש לנו קובץ טקסט עם מידע דיבאג שאנחנו רוצים להדפיס, אפשר לפתוח אותו בעזרת פונקציית `openFile` ולהשתמש בפונקצית `hGetContents` כדי לקרוא את התוכן שלו ולהדפיס אותו.
