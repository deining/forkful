## למה
למה לקרוא קובץ טקסט? ישנם מספר סיבות למה אדם ירצה לעשות זאת. לדוגמה, ייתכן שיהיה לכם קובץ טקסט עם נתונים שרציתם לטפל בהם באופן אוטומטי, או שתרצו להפוך את הטקסט לפורמט אחר. האפשרויות הן מגוונות וזה רק תלוי בדמיון שלכם!

## איך לעשות את זה?
בכדי לקרוא קובץ טקסט עם Haskell, נצטרך להשתמש בפונקציות אשר קשורות לטיפול בקבצים. במקרה כזה, נעשה שימוש בפונקציה הפנימית `readFile`. הנה דוגמא לקריאת קובץ טקסט והדפסת התוכן שלו:

```Haskell
main = do
  textFile <- readFile "myFile.txt"
  putStrLn textFile
```

כאן אנחנו משתמשים בפונקציה `readFile` על הקובץ `myFile.txt` ומשמיעים את תוצאת הפקודה באמצעות הפונקציה `putStrLn`. התוצאה של הקוד הנ"ל תהיה הדפסה של כל הנתונים שמצויים בקובץ.
