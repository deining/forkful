## למה

למה מתעניינים בחילוץ תת מחרוזות כגון חילוץ משפטים במחרוזת?

## כיצד לעשות זאת

מספר דוגמאות לכתיבת קוד ותוצאות תוך שימוש בבלוקי קוד "```Haskell ... ```":

```Haskell
-- קוד פשוט לחילוץ משפט מתוך מחרוזת
import Data.List (isPrefixOf, isSuffixOf)

sentence = "הקוד הזה שולחלחילוץמשפטמתוךמחרוזתכלשהי."

-- בדיקה אם המחרוזת מתחילה עם המילה "שולחל"
if "שולחל" `isPrefixOf` sentence
  then "שולחל מתחיל את המשפט"
  else "שולחל אינו מתחיל את המשפט"

-- בדיקה אם המחרוזת מסתיימת עם המילה "כלשהי."
if "כלשהי." `isSuffixOf` sentence
  then "כלשהי. מסיימת את המשפט"
  else "כלשהי. אינה מסיימת את המשפט"
```

תוצאות:

```
שולחל מתחיל את המשפט
כלשהי. מסיימת את המשפט
```

## ייעול

לגבי חילוץ מחרוזות, קיימות כמה אפשרויות אחרות שיכולות לעזור לנו להתקדם עם עבודתנו. אחת האפשרויות היא להשתמש בפונקציה `words` לחלוקת המחרוזת לתתודות של מילים, ואז לחפש תת-מחרוזת בקלות יותר בתוך התתודות הללו. נוסף על כך, ישנם פונקציות נוספות שמסייעות בחילוץ מחרוזות ועשויות להתאים יותר לצורכים שלנו.

## חקירה מעמיקה

לחילוץ מחרוזות ישנן כמה שיטות ואפשרויות מתקדמות יותר, כולל שימוש בפונקציות כמו `split`, `span`, ו-`group`. בנוסף, ישנם אלגוריתמים שיכולים לעזור לנו למצוא תת מחרוזות מכל מיקום נתון במחרוזת.
