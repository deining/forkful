## מה ולמה?

חיפוש והחלפה של טקסט הוא תהליך שבו מחפשים בטקסט מסוים עבור מחרוזת נתונה וממירים אותה למחרוזת אחרת. תהליך זה נחשב חשוב למתכנתים מכיוון שטקסט הוא אחד היסודות של התכנות ולכן צריך להיות באפשרות להתאים אותו לצורכי הקוד.

## איך לעשות:

```Haskell
replace :: String -> String -> String -> String
replace _ _ "" = ""
replace old new str@(x:xs) =
  if old `isPrefixOf` str
     then new ++ replace old new (drop (length old) str)
     else x:replace old new xs
```

קוד זה מדגים פונקציה בשם replace שמקבלת שלושה מחרוזות: מחרוזת ישנה, מחרוזת חדשה וטקסט עליו יש לבצע את ההחלפה. הפונקציה בודקת האם המחרוזת הישנה נמצאת בתחילת הטקסט ומחליפה אותה במחרוזת החדשה אם כן.

כאשר קוראים לפונקציה ניתן להשתמש בה כדלקמן:
```Haskell
replace "red" "blue" "The sky is red"
```

פלט הפונקציה יהיה "The sky is blue".

## טיפול מתקדם:

### היסטוריה

חיפוש והחלפה של טקסט היה חלק חשוב מהמתכנות כבר מאז התחילו לכתוב תוכניות מחשב. בעבר, זה נעשה ידנית על ידי כתיבה של קוד לכל מקרה אפשרי של החלפה. עם התפתחות הטכנולוגיות והשתלבותן בשפות תכנות, כיום ניתן להשתמש בפונקציות כמו replace לטיפול בחיפוש והחלפה של טקסט.

### אלטרנטיבות

אחת האלטרנטיבות לפונקציה replace בשפת Haskell היא להשתמש בפונקציות של ספריית Text, המציעה אפשרויות נוספות כמו replaceAll ומתן דפולטים לקריאה אחת.

פונקציות נוספות ניתן למצוא בפרוייקט Regex-tdfa שמציע פתרונות לחיפוש והחלפה עם תמיכה בתבניות רגולריות.

### פיתוח פנימי

הפונקציה replace נמצאת בתוך הספרייה Data.List וממומשת באמצעות רקורסיה. בחיפוש והחלפה של טקסט יש לקחת בחשבון את פורמט המילות והקישורים ביניהם כדי להבטיח תוצאות נכונות. ישנם גם פתרונות מתקדמים אחרים לחיפוש והחלפה כגון אלגוריתמי Boyer-Moore ו-Knuth-Morris-Pratt.
