המחיקה של תווים מתאימים לתבנית בפסקול היא כלי חזק ושימושי ברכיב הHaskell. בצורה כללית, הפעולה הזו מתאימה לבעיות של ניהול טקסט וטיפול בנתונים. במאמר זה, אנחנו נספק את הגיונן של הפעולה הזו ונראה כיצד ניתן להשתמש בה בקלות וביעילות.

## למה
למה לעשות מחיקה של תווים מתאימים לתבנית? מתי צריך להשתמש בפעולה זו? אחד הדוגמאות הטובות היא כאשר אתה צריך להסיר תווים מיוחדים מתוך טקסט או נתוני קלט. למשל, במקרה שאתה עובד עם טבלת נתונים, ייתכן שתרצה להסיר תווים מיוחדים כמו רווחים או תווים לא רצויים. הפעולה הזו נוחה לשימוש כאשר יש שפע של תנאים שיש לצמצם או לסנן מתוך הטקסט או הנתונים שמעורבים.

## כיצד להשתמש בפעולת המחיקה
נניח שאנחנו רוצים להסיר מהטקסט את כל האותיות הקטנות. בHaskell, יש לנו כמה משימות שקשורות לפעולת המחיקה. הנה מספר דוגמאות שיסבירו איך להשתמש בהן:

``` haskell
delete_lowercase :: String -> String
delete_lowercase input = filter isUpper input
```
