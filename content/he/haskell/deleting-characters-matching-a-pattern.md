## למה

שימוש במחיקת תווים התואמים תבנית יכול להיות שימושי במגוון רחב של מצבים, כגון ניקוי נתונים מיותרים או תיקון שגיאות בקוד.

## איך לעשות זאת

למחיקת תווים התואמים תבנית בשפת Haskell ניתן להשתמש בפונקציות כמו `filter` ו- `delete`, שמאפשרות לנו למצוא ולמחוק תווים בהתאם לתבנית שניתנת לנו. לדוגמה, אם נרצה למחוק את כל התווים הקטנים מתוך רשימת מספרים, נוכל להשתמש בפונקציה הבאה:

```Haskell
deleteSmall :: [Int] -> [Int]
deleteSmall list = filter (>5) list
```

כאן, אנו משתמשים בפונקציה `filter` כדי למצוא את כל המספרים שגדולים מ-5 ולוותר על כל התווים הקטנים מכך. לאחר מכן, ניתן להשתמש בפונקציה `delete` כדי למחוק את כל התווים התואמים לתבנית זו.

```Haskell
deleteSmall [1, 6, 9, 3, 2] -- Output: [6, 9]
```

## מעמקים בתוך הנושא

מחיקת תווים התואמים תבנית היא רק חלק מהפונקציונאליות הרבה יותר מורכבת של הפונקציות של Haskell. ניתן להשתמש באופרטורים כמו `==` כדי לבחור תווים ספציפיים בתוך תבנית, או להשתמש בפונקציות נוספות כדי לבצע פעולות מתקדמות כמו חיבור וחיסור תווים.

אם תרצו ללמוד עוד על הפונקציונאליות הרבה שיש ל-Haskell להציע, כדאי לכם לקרוא עוד על נושאים כמו השוואת תבניות ליצירת תוכניות גיזרה יותר משוכללות.
