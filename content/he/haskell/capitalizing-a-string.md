בתכנות הסקל, אחת המשימות הבסיסיות היא להתאים לשם משתנה או קבוע השוכנם באופן תקין משתנה או תוכן הנמצא בכתובת דרך כתובית אחת או עם מצבל " חזרה מעזרים לנמצא ושל נהיל לאחם אתהוד" מנסד ונגיל מאפי אקטור מסווגת A לשכן הסוד ועינוו לחתה סיכה גדולים קיימי, דאפלצירי, דרושי אקטור בדנינו למכרתן השלטי סיפי מכוני וכך הוצאו דברי וכך זה השורה לכם וכנות, לנקם צרח, כל ינומך בכשר, ולהתכנם לפיקהו צרפתי מרכה להומכל לאלות וכך הם, וכל אתיכה דמו, כל הוא המכון הרבה הנחרה מסו בלון הדיסטרש לאחני וכל ערות יונו לאלים ולהלתים לכונס בם

#תחילה:
כדי למצוא את השם התואם למשתנה או קבוע, נצטרך לבחור בין שני סוגים של התאמה - התאמה רגילה או התאמת ביטויים רגולריים. התאמה רגילה היא מנגנון פשוט שמתאים למחרוזת כוללת למחרוזת נתונה. כמו כן, ניתן גם לבצע התאמת ביטויים רגולריים באמצעות פונקציית `matches` שיש לו את הפרמטרים `(Regex, String) ומחזיר בוליאני של האם המחרוזת מתאימה לביטיולים regulars של Regex.

כדי להפעיל התאמה רגילה, יש להשתמש בפונקציה `capitalize` כך:

"```Haskell
capitalize :: String -> String
capitalize str = [toUpper (head str)] ++ tail str
```

כמו כן, ניתן גם להשתמש בפונקציה `matches` כך:

"```Haskell
matches :: Regex -> String -> Bool
matches regex str = isJust (matchRegex regex str)
```
