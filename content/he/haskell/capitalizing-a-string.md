## למה

עשרות פעמים בכתיבת קוד, נתקלים בצורך לכתחילה להגדיר מחרוזת באותיות גדולות. כך, ניתן להבהיר נקודה מסוימת או להפוך את המחרוזת למובן במקרה של שימוש בנתונים ממקור חיצוני. כתיבה ידנית של מחרוזות באותיות גדולות עשויה להיות מסורבלת ואף לישנות טכנית במקרים רבים, ולכן כדאי להשתמש בפונקציות כמו capitalize בכדי לשמור על יכולת מניפולציה ועריכה קלה של המחרוזת המיועדת.

## איך לעשות

בשפת התכנות Haskell, ניתן לכתוב פונקציה פשוטה המכילה פקודות פרוגרמטיות בכדי להפוך את המחרוזת לכתיב אותיות גדולות. הפונקציה הבאה מקבלת את המחרוזת הנתונה ומחזירה אותה בכתיב אותיות גדולות:

```Haskell
capitalize :: String -> String
capitalize str = [toUpper char | char <- str]
```

כאן, אנו משתמשים בתנאי אחד "toUpper" אשר מפעיל קיפים על כל תו במחרוזת שהתקבלה כפרמטר.

לדוגמא, אם נרצה להפוך את המחרוזת "hello world" למחרוזת "HELLO WORLD", נוכיח את הפונקציה כדי לקבל את התוצאה הרצויה:

```Haskell
capitalize "hello world"
```

את התוצאה "HELLO WORLD" נוכל לראות בניתוח הפשוט והמהיר שלנו.
