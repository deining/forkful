## למה 

מַהֵר להתחל את הפרויקט החדש שלך כאשר יש לך את הכלים הנכונים כדי להשיג את היעדים שלך. \n

## כיצד לעשות את זה 

כאשר נעשה שימוש בעיקר בסביבת התכנות Bash, יש לנו את כל היכולות הדרושות כדי להפעיל את הפרויקט החדש שלנו. תחת הנחיות והזרזורים שלנו, ניתן להיות רעות בקקוד ולייצר תוצאות מדוייקות.

### דוגמאות 

```Bash
#!/bin/bash
# פקודה להדפס טקסט "שלום עולם"
echo "שלום עולם"
```

התוצאה מהפקודה הנ"ל תהיה: `שלום עולם`

### פלט מתורגם

הבא יהיה הפלט נתון מתורגם 

```Bash
#!/bin/bash
# פקודה להדפס טקסט "שלום עולם"
echo "שלום עולם"
```

התוצאה מהפקודה הנ"ל תהיה: `Hello World` לפי התרגום. 

## מכה לתוך הכונן העמוק 

בכדי להתחיל פרויקט חדש, יש לנו מספר דברים לתת את כולם לאותו צבירת של אבני יסוד. נתחיל עם מאמר זה על Bash ואת הכלים שנכונים עבור הקוד. נממש את הקוד הנכון לפי הצורך, בחלשר יתר מאלפים של חברי הצוות השונים. יש לנו את כל זה כדי להיות הכי מוכנים אנו לקו להציג את הפתרונות המוכן זאת המצא. דרך ייעודית  היא בשביל הקוד הנכון והנמצא.
