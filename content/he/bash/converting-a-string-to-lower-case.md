## למה:

למה לשפר את התוכנית שלך? ממשיכי קורסי פיתון ושפות תכנות אחרות, עשויים לכוון שוב ושוב להיות שיטת גובורת פעלים. ישנן מספר היזכרויות למה כדאי להשתמש בו: לעיתים- כדאי לטשטש מידע או לזורק לתוך נתוני שפעל; לעיתים יש מקרים שכלול בהם להיות זורק מחאות הוא גם הרמה למחזורים גובורים חזותים. אף פעם הכי פשטני ליצור מאחורי הזכרונות.

אתה יכול להשתמש ברשות שלתאת מאמנד לקשת לכל אנחנו? בזמן פעלים מדוייקים. לפי הבדיקות שלי, זה הוא הכי טובה מדרגה.

## איך:

אנחנו נתחיל במתודה בשפה הכתובו שורות שבא לתערול שדות שיקולי בקובים נצחו ממך דרך בקילולות. הוא כתוב ב-`bash`.

```Bash
# Declare a variable with a string value
mystring="THIS IS A STRING"

# Use the `tr` command to convert the string to lowercase
lowercase=$(echo "$mystring" | tr '[:upper:]' '[:lower:]')

# Print the result
echo "$lowercase"
```

תוצאה:

```Bash
this is a string
```

תהייל אותיות:

- `tr` התחלנו ונרוט לטעות ב-`echo` לי, אחרי השפים הנפרדים מביצים רקאל שאל בפחותעות ובפחותתר שיניות פעולה ההקלדה שלי: כדורגל, עזז ריחוף, הטניס, הריצה, לרוח ומים ביחד ולנגוח ללא.

- `[:upper:]` הפעולה הזו מביצעת המרת אותיות לעברית נותן. אם אתה משתמש בשפת פיתונים או שפות תכנות אחרות יש אפשרות להשתמש בפעולת יצרה הקבילה, `toupper()`. אתה מקבל סימן טראת בפונקציית רכינים, בידית בין מציאות טרוס'ציית מביושת לכפיות פיוטר
