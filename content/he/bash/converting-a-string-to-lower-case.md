## מה זה ולמה?
המרת תווים לאותיות קטנות היא תהליך מסוים שמשתמשים בו כדי להמיר את התווים הקטנים של מחרוזת לתווים גדולים. תהליך זה נחשב חשוב מאוד עבור מפתחי תוכנה כי הוא מאפשר להשוות, לחפש ולסנן מחרוזות בצורה יעילה.

## איך לעשות זאת:
```Bash
# דוגמה לשימוש בפקודה tr בכדי להמיר את המחרוזת לאותיות קטנות
echo "HELLO WORLD" | tr '[:upper:]' '[:lower:]' 
# תוצאה: hello world
```

```Bash
# דוגמה לשימוש בפקודה awk בכדי להמיר את המחרוזת לאותיות קטנות
echo "HELLO WORLD" | awk '{print tolower($0)}'
# תוצאה: hello world
```

## חקירה מעמיקה:
(1) היסטוריית הקשורים: השיטה של המרת מחרוזת לאותיות קטנות נמצאת בשימוש כבר מאז תחילת תקופת המחשבים. בשנות השמונים, כאשר קוד המחשבים היה מופץ, חלקים ממחרוזות הופכו לאותיות גדולות כדי להבטיח שלא יתבררו לתווים לא חוקיים. כיום, זהו תהליך נפוץ בעולם התכנות ומקבל תמיכה מרחב השפות תכנות.

(2) אלטרנטיבות: ישנן פקודות נוספות ב-Bash שניתן להשתמש בהן כדי להמיר מחרוזות לאותיות קטנות, כגון: ``` tr, awk, sed ```. כמו כן, ישנם גם תוכניות צד שלישי שמיועדים למטרה זו, כגון: ``` perl, python ```.

(3) פרטי יישום: בפקודות המצויות בדוגמאות כאן, ההמרה לאותיות קטנות מתבצעת ע"י שימוש במילות קידומת! `tr` משרשר את המחרוזת לתווים נמוכים (כעת אנו מתאם באמצעות תווים ספציפיים), בעוד ש-`awk` משתמש בפונקציה  `tolower()` להמרת המחרוזת לאותיות קטנות.
