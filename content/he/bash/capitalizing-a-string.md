## למה
כתיבת תוכניות בש גיבורות לילדים קשה מרגש והמון כיף! אבל לפעמים, יכול להיות שנדרש לכתוב תוכנית שתדרוג את הטקסט שלנו. כגון, להפוך אותו לכתב רקע או לכתב גדול. אם אתם מתכנתים חדשים בש ורוצים ללמוד איך לכתוב קוד שיכול לעשות זאת, קראו הלאה!

## איך לעשות זאת
ראשית ונעימה, נצור תוכנית פשוטה שתכתוב ותציג את הטקסט שלנו בפקודה באש. לדוגמה:

```
Bash
string="טקסט רגיל"
echo $string
```

כעת, נפעיל את התוכנית ונראה את הפלט בפקודה הבאה:

```
Bash
טקסט רגיל
```

נכתוב בעזרת הגדרה רקע את הטקסט שלנו.

```
Bash
string="טקסט רגיל"
echo ${string^}
```

כעת התוכנית תציג את הפלט הבא:

```
Bash
טקסט רגיל
```

ניתן לראות שהתו הראשון של המחרוזת הופך מאות קטנה לאות גדולה. אם אנחנו רוצים להפוך את כל המחרוזת לאותיות גדולות, נשתמש בפקודה הבאה:

```
Bash
string="טקסט רגיל"
echo ${string^^}
```

והפלט שלנו יהיה:

```
Bash
טקסט רגיל
```

## מכטב ורחב
כך שהגענו לשימוש בפקודת הופך המחרוזת, נראה מה הולך ומתבצע מאחורי הקלעים. הפקודה עובדת באמצעות יחסים בש, הם מעיקרת בוויקיי של בָש ולשם ניתוח ליה את כמה סוגי כתיבות אחרים.
