## למה

כתיבת תוכניות ב-Bash היא כלי חשוב ומרתק לפיתוח תוכנה ותכנות מערכות. השימוש בשפת תכנות זו מאפשר כתיבת תוכניות פשוטות ומהירות להתאים לצרכי המשתמש. בכתיבת תוכניות ב-Bash, כמו כל שפת תכנות אחרת, אנו צריכים לברר כיצד לטפל בכל מחרוזת כדי להקפיץ אותה. 

## איך לעשות זאת

ישנם מספר דרכים לכתוב פקודות פשוטות וחכמות לקיפוץ מחרוזות בשפת Bash. דרך אחת לעשות זאת היא להשתמש בפקודת ```tr``` ולהשתמש בפרמטר - ```[A-Z]``` כדי לייצר אותיות גדולות במיקומים זרים במחרוזת. לדוגמה, אם נרצה לקפוץ את המחרוזת "hello world" נלמד כיצד להשתמש בפקודה הבאה:

```
echo "hello world" | tr '[a-z]' '[A-Z]' 
```

פקודה זו תיצור את הפלט הבא:

```
HELLO WORLD
```

כמו כן, נוכל גם להשתמש בפקודת ```sed``` עם האופציה "```s/.\+/.&/g```" כדי להוסיף נקודה מול כל אות. לדוגמה, אם נרצה לקפוץ את המחרוזת "hello world" נשתמש בפקודה הבאה:

```
echo "hello world" | sed 's/.\+/.&/g' 
```

פקודה זו תייצר את הפלט הבא:

```
.H.E.L.L.O. .W.O.R.L.D.
```
