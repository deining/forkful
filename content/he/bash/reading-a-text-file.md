## למה?

קריאת קובץ טקסט היא כלי חשוב בתכנות ב-Bash. קריאת קובץ טקסט מאפשרת לנו לעבוד עם מידע ולבצע פעולות עליו, והיא משמשת כבסיס לפעולות נרחבות יותר בשפת Bash.

## איך לעשות זאת?

```Bash
# קריאת קובץ טקסט באמצעות הפקודה cat 
cat textfile.txt

# קריאת קובץ טקסט ותצוגתו בצורה מושכתת באמצעות הפקודה less 
less textfile.txt

# קריאת השורה הראשונה בקובץ טקסט עם הפקודה head 
head -n 1 textfile.txt 

# קריאת השורה האחרונה בקובץ טקסט עם הפקודה tail
tail -n 1 textfile.txt
```

הפקודות הנ"ל הן רק חלק מהאפשרויות של קריאת קובץ טקסט בשפת Bash. בנוסף, ניתן ליצור סקריפטים מתאימים שיאפשרו עבודה מתוחכמת יותר עם קבצי טקסט. להלן כמה דוגמאות פשוטות לסקריפטים שמשתמשים בפקודת cat לקריאת קבצי טקסט ובהמשך לבצע פעולות נוספות, כגון חיפוש או סידור הנתונים בקובץ.

```Bash
#!/bin/bash
# סקריפט קורא קובץ טקסט ומכין רשימת נמענים משם השורה הראשונה

cat file.txt | cut -d ',' -f2 | sort | uniq
# הפקודה הנ"ל משתמשת בסימן המפריד בין הנתונים בקובץ הטקסט (במקרה זה נפריד בפסיק) ומציגה רק את השדה השני מכל שורה, כלומר את שמות הנמענים. לאחר מכן, היא מסדרת את הנתונים לפי הא"ב (sort) ומסננת רק את הערכים היחודיים (uniq) ומציגה רשימה של הנמ
