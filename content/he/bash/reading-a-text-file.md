## על מה לכתובת זו

קריאת קבצי טקסט נחשבת ליסוד בשפת התכנות Bash והיא אופן אפקטיבי לעבוד עם נתונים שמאוחסנים בקבצים. במאמר זה נלמד איך לקרוא קובץ טקסט ונראה דוגמאות קוד כדי להבין טוב יותר את התהליך.

## איך לקרוא קובץ טקסט

קריאת קובץ טקסט באמצעות Bash נעשית באמצעות הפקודה "cat" בשילוב עם שם הקובץ הרצוי. לדוגמא, אם נרצה לקרוא את קובץ הטקסט "file.txt", נכתוב בטרמינל:

```Bash
cat file.txt
```

כאשר נכתוב את פקודת ה"cat" ישים לב שנכתוב את שם הקובץ המלא עם הסיומת שלו. השורה התחתונה שתופיע בטרמינל תהיה התוכן של הקובץ המתבקש.

כמו כן, ניתן לקרוא קבצי טקסט באמצעות הפקודה "less" אשר מאפשרת לנו לראות את התוכן של הקובץ בצורה מאורגנת יותר ולגלול בו בעזרת מקשי החצים. ניתן להשתמש בפקודת ה"less" בצורה דומה ל-"cat":

```Bash
less file.txt
```

ניתן להפעיל את הפקודות "cat" ו-"less" באופן דינמי בתוך סקריפטים של Bash, כדי לקרוא קבצי טקסט שמצוים בתיקייה מסוימת או לפי תנאים מסוימים.
