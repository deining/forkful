# למה

לפעמים אנו נתקלים בקבצי טקסט שהם חלק חשוב מהתוכניות שלנו, וכדי לעשות זאת, אנו צריכים להיות במצב לקרוא את התוכן שבתוכם. בכתבה זו נלמד איך לקרוא ולהתמודד עם קבצי טקסט בשפת Bash.

# איך לעשות זאת

קוד קריא לקובץ טקסט ניתן לכתוב בכמה שורות קוד פשוטות בשפת Bash. כדי לקרוא קובץ טקסט, נשתמש בפקודת `cat`:

```Bash
cat file.txt
```

כל מה שכתוב בתוך הקובץ יודפס בטרמינל. אם נרצה להדפיס רק חלק מהתוכן של הקובץ, נוכל להשתמש בפקודת `head` ו-`tail` עם פרמטר של כמה שורות להדפסה:

```Bash
head -n 10 file.txt
tail -n 5 file.txt
```

לכאורה קריאה אחת של קובץ טקסט נראית חסכונית מדי, אבל כאשר מדובר בתוכן גדול יותר, יהיה לנו צורך לבצע פעולות על התוכן של הקובץ. כדי לבצע פעולות כאלה, נשתמש בפקודת `grep` כדי לחפש ולסנן מידע מתוך הקובץ, ובפקודת `sed` כדי לערוך ולשנות את התוכן של הקובץ.

לדוגמה, אם נרצה למצוא את כל השורות בקובץ שמכילות את המילה "Bash", נשתמש בפקודת `grep` עם הפרמטר `-i` שיחפש ללא תלות באותיות גדולות וקטנות:

```Bash
grep -i "Bash" file.txt
```

אם נרצה להחליף את המילה "Bash" ל-"Git" בכל המופעים שלה בקובץ, נשתמש בפקודת `sed` עם הפרמטר `-i` שיערוך את הקובץ במקום:

```Bash
sed -i 's/Bash/Git/g' file.txt
```
