## למה

תוכנת Bash היא כלי יעיל ופשוט לשימוש בכתיבת תוכניות וסקריפטים במערכת הפעלה לינוקס או בסביבת Unix. קריאת קובץ טקסט בעזרת Bash נחשבת למשימה מתחילה שחשובה ללמוד כשמתחילים לתכנת בשפה זו.

## כיצד לבצע זאת

הקוד של Bash מופעל על ידי מנוע ביצוע הקוד שנקרא תוכנת Bash shell. זהו שפה של יצירת שכבת קשר בין המשתמש והמערכת הפעלה. כשאתם מנסים לקרוא קובץ טקסט באמצעות Bash, הפקודה העיקרית בשורת הפקודה היא `cat`. ניתן להשתמש בפרמטרים נוספים עם פקודת `cat` כדי להתאים את הפלט בהתאם לצורך.

```Bash
cat file.txt # קריאת קובץ טקסט עם פקודת cat
cat -n file.txt # קריאת קובץ טקסט עם מספור השורות
cat -s file.txt # קריאת קובץ טקסט עם רווח בין שורות ריקות
```

כאשר אתם מריצים את הפקודה בשורת הפקודה, הפלט הוא התוכן המלא של הקובץ הטקסטואלי שנקרא. אתם יכולים גם להשתמש בפקודת `echo` כדי לראות את התוכן של קובץ טקסט בכל הזמנים.

```Bash
echo "Hello World" > file.txt # כתיבת המערך hello World לקובץ file.txt
cat file.txt # קריאה של קובץ טקסט שכתבנו
```
