##מדוע##

למה לנו לעסוק בחיפוש והחלפת טקסט בכתיבת תוכניות בשפת Bash?

##איך##

בדוגמאות קוד ופלט דוגמאות בתוך חסרי שורה עליהם הפעלנו את הכתיבת Bash. 

```Bash
# חיפוש והחלפת טקסט בקובץ ספציפי
sed -i 's/מילת_מפתח/מילת_מטרה/g' file.txt

# חיפוש והחלפת טקסט ביותר מקובץ אחד
sed -i 's/מילת_מפתח/מילת_מטרה/g' *.txt

# חיפוש והחלפת טקסט בכל תת הספריות והקבצים בתיקייה נוכחית
sed -i 's/מילת_מפתח/מילת_מטרה/g' ./*

# שמירת התוצאה לקובץ חדש במקום לערוך את הקובץ המקורי
sed 's/מילת_מפתח/מילת_מטרה/g' file.txt > new_file.txt
```

##חפירה עמוקה##

כאשר מדברים על חיפוש והחלפת טקסט בכתיבת תוכניות בשפת Bash, כוונתנו בדרך כלל על כלי הפקודה "sed". זהו כלי עוצמתי שמשמש לעריכת טקסט בקבצים. הוא מאפשר לנו לחפש מילים מפתח וטווחים של תווים ולהחליפם במילים אחרות, באופן מהיר ויעיל. הכלי "sed" נמצא בכמעט כל מערכת Linux וזמין גם עבור מכשירי MacOS. כמו כן, קיימים גם גירסאות ניידות שזמינות להורדה ברשת.
