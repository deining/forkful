## למה

השוואת שתי תאריכים היא כלי חשוב בתכנות בש למטרות מגוונות, כגון בדיקת תקינות, תאריך יותר גדול או קטן ועוד. כתבה זו תלמד אותך כיצד לשווא תאריכים בבש בצורה פשוטה ובלי להשתמט.

## איך לעשות

תחילה, צריך להכיר את הפקודות שמשמשות לשוואת תאריכים בבש. הפקודה הראשונה היא "date", המאפשרת להציג תאריך נוכחי. לדוגמה, אם נרצה להדפיס את התאריך הנוכחי, נכתוב את השורה הבאה:

```Bash
date
```

הפקודה הבאה היא "diff", המאפשרת להשוות בין שני תאריכים. לדוגמה, נרצה להשוות בין התאריך 01/01/2020 ל-01/01/2021, נכתוב את השורה הבאה:

```Bash
diff -d 01/01/2020 01/01/2021
```

לעיתים ייתכן שנרצה להשוות גם את השעה של התאריך. במקרה כזה, נצטרך להשתמש בפקודת "stat" ולשים אותה בתוך הפקודה "diff". לדוגמה, נרצה להשוות בין התאריך 01/01/2020 בשעה 10:00 לבין התאריך 01/01/2020 בשעה 12:00, נכתוב את השורה הבאה:

```Bash
diff -d "$(stat -c %y 01/01/2020 10:00)" "$(stat -c %y 01/01/2020 12:00)"
```

כמו כן, אם נרצה להשוות גם את הזמן החלקי, נשתמש בתיקיית "/etc" המכילה מידע על התאריכים והזמנים של המערכת. לדוגמה, נרצה להשוות בין התאריך 01/01/2020 בשעה 10:30:25 לבין התאריך 01/01/2020 בשעה 10:30:30, נכתוב את השורה הבאה:

```Bash
diff -d "$(stat -c %y /etc/ 01/01/2020 10:30:25)" "$(stat -c %y /etc/ 01/01/2020 10:30:30)"
```
