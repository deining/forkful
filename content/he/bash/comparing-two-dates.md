## למה
השוואת שתי תאריכים נחשבת לפעולה חשובה בתכנות בש ומבוצעת בכדי לבדוק אם תאריך אחד הוא קודם או אווירוע מארכיות ישנים מתאריך אחר. פעולה זו נחשבת לשכיחה בתכנות ויכולה להיות מועילה במגוון רחב של מקרים.

## איך לבצע
כדי לבצע השוואת שתי תאריכים בתוך סקריפט של בש, ניתן להשתמש בפקודה "date" עם הפרמטר "-d" כדי להפעיל תאריך מסוים. לדוגמה, אם נרצה להשוות בין תאריך היום לתאריך 1 בינואר 2020, יש להריץ את הפקודה הבאה:

```Bash
date -d "20200101"
```

לאחר מכן, ניתן להשתמש בתנאי "if" עם הפקודה "[$year1 -lt $year2]" כדי לבדוק אם השנה הקטנה יותר מהשנה הגדולה. כדי לקבל את השנה הקודמת או הלאחרונה, ניתן להשתמש בפקודה "cut" לכלול את השנה בפלט של התאריך. לדוגמה, ניתן להשתמש בפקודה הבאה עבור תאריך היום:

```Bash
cut -d " " -f5
```

הפלט יהיה השנה הנוכחית. לאחר מכן, ניתן להשוות את השנת הנוכחית לשנה של התאריך השני ולפעול בהתאם לתוצאה.

## עיון מעמיק
בכדי להשוות שני תאריכים מדויקים, ניתן להשתמש בפקודה "date" עם הפרמטר "-d" בכדי לקבל את התאריך המדויק של השנה הנוכחית. כדי להשתמש בפקודה זו, יש להתקין את החבילה "sysstat". לאחר התקנת החבילה, ניתן להשתמש בפקודה כדי לקבל את התאריך המדויק של תאריך מסוים עם הפקודה הבאה:

```Bash
date "+%Y-%m-%d"
```
