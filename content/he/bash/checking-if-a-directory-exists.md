## למה?

בדיקת האם תיקייה קיימת היא פעולה חשובה בכתיבת קוד בשפת Bash. ביצוע בדיקה כזו מאפשר לנו לוודא שהתיקייה שאנו מנסים לגשת אליה קיימת במערכת הקבצים שלנו. כך, אנו יכולים לקבל מענה תקין במקרה שהתיקייה אינה קיימת ולהמשיך לפעול על פני מערכת הקבצים שלנו. לכן, כדאי לדעת כיצד לבצע בדיקת קיומה של תיקייה ולהשתמש בזה בתכנות שלנו.

## איך לבצע בדיקת קיומה של תיקייה?

כדי לבדוק אם תיקייה קיימת במערכת הקבצים שלנו נוכל להשתמש בפקודת `[ -d ]` בשילוב עם שלבים נוספים לבדיקת התיקייה המבוקשת. הנה מספר דוגמאות קוד לכך:

```Bash
# דוגמה 1 - בדיקה אם תיקייה קיימת והדפסת הודעה בהתאם
if [ -d "/home/user/Documents" ]; then
    echo "תיקיית המסמכים קיימת"
fi
```

```Bash
# דוגמה 2 - בדיקה אם תיקייה לא קיימת ויצירתה במידת הצורך
if [ ! -d "/home/user/Documents" ]; then
    mkdir "/home/user/Documents"
fi
```

בדוגמאות אלו נשתמש בפקודה `[ -d ]` כדי לבדוק אם התיקייה `/home/user/Documents` קיימת או לא. אם התיקייה קיימת, נעביר לכתובת `echo` כדי להדפיס הודעה מתאימה. אם התיקייה אינה קיימת, נבצע פעולת יצירה של התיקייה באמצעות `mkdir`.
