# מה זה ולמה?
בשוורצ'ים, אנו משתמשים בתנאי `if` כדי לבדוק אם תיקייה קיימת או לא. הפעולה הזו נחשבת חשובה כאשר אנו רוצים להחליט מה לעשות לפי התנאי שנתבקש.

# כיצד לבדוק אם תיקייה קיימת:
ניתן לבדוק אם תיקייה קיימת בעזרת הפקודת `test` והפרמטר `-d` המציין "תיקייה". לדוגמה:

```Bash
if test -d /path/to/directory; then
  echo "התיקייה קיימת."
else
  echo "התיקייה לא קיימת."
fi
```
תוצאה:

```
התיקייה קיימת.
```

במקרה של הפקודה `bash`, ניתן לההשתמש במשתנה שמציין את התיקייה ולבדוק אם הוא קיים. לדוגמה:

```Bash
directory="/path/to/directory"

if [[ -d "$directory" ]]; then
  echo "התיקייה קיימת."
else
  echo "התיקייה לא קיימת."
fi
```

תוצאה:

```
התיקייה קיימת.
```
