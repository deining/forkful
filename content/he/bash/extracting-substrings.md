## למה

למה ישנם עלילות בשפת Bash ונדרש להשתמש בחיבורי תת מחרוזות? ניתן להשתמש בתת מחרוזות כדי לסנן ולטפל במידע מורכב מתוך מחרוזת גדולה יותר, כגון קבצי טקסט או מידע מכשירים חיצוניים.

## איך לעשות

כדי לחלץ תת מחרוזת בשפת Bash, ניתן להשתמש בפקודת "```cut```". הנה דוגמא לשימוש בפקודה זו על קובץ טקסט:
```
#!/bin/bash
# קוד שנמצא בקובץ example.sh
while read line; do
  echo "$line" | cut -d " " -f 2
done < example.txt
```

Input (example.txt):
```
שלום עולם
בואו נכיר
```

Output:
```
עולם
נכיר
```

## חפירה עמוקה

פקודת "```cut```" מאפשרת לנו להתאים את הגבלות החיתוך על ידי שימוש באופציות שונות. לדוגמה, ניתן להשתמש באפשרות "-f" כדי לציין את השדה שאנו רוצים לחלץ, ובאפשרות "-d" כדי לציין כיצד להפריד בין השדות. ניתן גם להשתמש בדגמאות של תבניות כדי לחלץ תבניות מסוימות של מחרוזות.
