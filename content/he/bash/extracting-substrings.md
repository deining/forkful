## מדוע 

החיפוש והחילוץ של תת-מחרוזות הוא כלי חשוב בתכנות ב־Bash. זהו דרך נוחה ויעילה לעבוד עם טקסט ולהוציא את המידע הנחוץ מבחינתנו.

## איך לעשות זאת 

הנה דוגמא לקוד שמציג את כיצד לחפש ולחלץ תת-מחרוזות ב־Bash:

```Bash 
# הגדרת משתנים 
string="זו המחרוזת שאנחנו רוצים לחבר תת-מחרוזת ממנה" 
substring1="מחרוזת" 
substring2="רוצים"

# חיפוש תת-מחרוזות באמצעות הוראת בנייה [string:] 
echo "${string:${#substring1}}" # נדפיס "המחרוזת שאנחנו רוצים לחבר תת-מחרוזת ממנה" 
echo "${string:${#substring2}}" # נדפיס "לחבר תת-מחרוזת ממנה"
```

תוצאות הפלט:

```Bash
המחרוזת שאנחנו רוצים לחבר תת-מחרוזת ממנה 
לחבר תת-מחרוזת ממנה
```

## חקירה עמוקה 

בנתיב המחרוזת ב-Bash, התו "#" מייצג את האורך של המחרוזת. לכן, כאשר אנו רושמים "${string:${#substring}}", אנו מבקשים שהתת-מחרוזת תתחיל מהאינדקס של אורך התת-מחרוזת המבוקשת.

אם נרצה לחפש ולחלץ תת-מחרוזות עם מרווחים, נוכל להשתמש בהוראות של "cut". למשל:

```Bash
cut -d " " -f 2 # יחזיר את המילה השנייה במחרוזת, מופרדת על-ידי מרווחים
```

זו הפעולה התכונה של "cut" והיא יכולה לקלוט מספר ארגומנטים נוספים כדי להשתמש בפטרונים נוספים לחיתוך.
