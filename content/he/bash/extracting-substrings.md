# למה

תחילת עבודתנו עם שפת התכנות Bash יכולה להיות מאתגרת, אך אם נצלות את הכלים הנכונים, נוכל לכתוב קודים מיישרים ויעילים. מאתגר נוסף הוא להתמודד עם מחרוזות שיש לנו בקוד ולקלוט מהן חלקים מסוימים בלבד. אבל אל תדאגו, ישנן כמה דרכים לחתוך מחרוזות ולהשתמש בתת מחרוזות בקוד שלכם. במאמר זה נראה איך לממש את כלי חיתוך המחרוזות הזה בשפת התכנות Bash.

## איך לעשות זאת

ניתן לחתוך מחרוזות בשפת Bash בזוגות עם שתי פקודות חשובות: `cut` ו- `sed`. אם נרצה לחתוך חלק מהמחרוזת, נוכל להשתמש בפקודת `cut` עם הפרמטר `d` כדי לציין את המדדים של החיתוך. לדוגמה, אם נרצה לחתוך את האותיות המיותרות מהמחרוזת "Hello, World!", נוכל להשתמש בפקודה הבאה:

```Bash
echo "Hello, World!" | cut -d',' -f1
```

תוצאה: "Hello"

אם נרצה לחתוך מחרוזת מסוימת באמצעות תבנית, נוכל להשתמש בפקודת `sed` עם הפרמטר `s` ותבנית שנרצה לחלץ. לדוגמה, נרצה להחליף את "Hello" ב-"Hi" במחרוזת "Hello, World!", נוכל להשתמש בפקודה הבאה:

```Bash
echo "Hello, World!" | sed 's/Hello/Hi/'
```

תוצאה: "Hi, World!"
