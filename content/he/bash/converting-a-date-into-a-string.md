## למה
כשמתחילים לתכנת בבאש, ישנו צורך בכוח מוותר להמיר תאריך נתון למחרוזת. זה יכול לשמש להצגת תאריך בפורמט שהמשתמש יודע לקרוא או כדי לבדוק אם תאריך מסוים נמצא בין תאריכים אחרים. 

## איך לעשות זאת
להמיר תאריך למחרוזת בבאש ניתן באמצעות הפקודה "date". יש להשתמש בפקודה עם התווים "+%s" להצגת תאריך בפורמט של מספרים של שניות מתאריך המקור. לדוגמה:

```Bash
date "+%s" 2020-01-01
```

פלט: 1577833200

כדי להציג את המספר החוזר בפורמט של תאריך, ניתן להשתמש בפקודת "date" עם התווים "+%Y-%m-%d" ולתת לה כפרמטר את המספר שחזר מהפקודה הקודמת:

```Bash
date "+%Y-%m-%d" -d @1577833200
```

פלט: 2020-01-01

כעת, כשיש לנו את התאריך בפורמט של תאריך מספרי, אפשר להשתמש בפקודת "date" עם הקווים " ' '" כדי להמיר אותו למחרוזת:

```Bash
date "+%Y-%m-%d" -d @1577833200
```

פלט: 2020-01-01

כדי להציג את התאריך בפרמט של יום מלא, ניתן להשתמש בפקודת "date" עם התווים "+%A" ולתת לה כפרמטר את המחרוזת שהמירנו מתאריך בפורמט מספרי:

```Bash
date "+%A" -d '2020-01-01'
```

פלט: Wednesday
