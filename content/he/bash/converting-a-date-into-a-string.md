## למה

להמציא תאריך מפורמט למחרוזת בשפת Bash יכול להיות חלק חשוב מתהליך כתיבת תוכניות או סקריפטים בסביבה הפתוחה הזו. לפעמים יש צורך לייצא את התאריך בתבנית שמתאימה לשימושים שונים ולכן ייתכן שתרצו להמיר אותו למחרוזת.

## איך לעשות זאת

עבור המימוש של ההמרה ניתן להשתמש בכמה דרכים שונות, אבל כאן נדגים שיטה פשוטה ויעילה. נתחיל בהתקנת התוכניות הנדרשות על ידי הרצת הפקודה `sudo apt-get install coreutils`. לפני שנדגים את הקוד, נגדיר משתנה המכיל את התאריך הנוכחי: `date=$(date +%Y-%m-%d)`. עם זאת, עם השמירה ניתן להשתמש במשתנה `date` לרשומת התאריך שהוגדרה בפקודה המנוונת. נשתמש בפקודה המנוונת `echo` על מנת להציג את התאריך: ```Bash echo $date ```
לקבלת פלט דומה לעיל. כעת נוכל לראות איך התאריך הומר למחרוזת מתאימה לשימוש בתצוגת התוכניות שנתנו. לשם כך נשתמש בפקודה המעצבת `printf` בפעולה השרשור תנאים עם אופציות כך: ```Bash
output=$(printf "%s" "$date");
printf "תאריך:% s\n" "$output"```
דוגמא זו נועדה להמתין לתשובה נוספת לפני החזרת הדפדפן. ניתן גם להשתמש באופציה `$(echo $output)` בהשמעת הפקודות המרובות לביצוע ולשימוש נוח יותר.
