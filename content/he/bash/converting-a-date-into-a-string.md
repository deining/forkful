## למה

תאר לפניכם אם נתקלתם במקרה שלכם בצורך להמיר תאריך למחרוזת, אולי עבור הצורך של דיווחים או עבודת ניתוח נתונים. בכתבה הזאת נלמד איך לכתוב קוד שיעשה את העבודה בקלות ובמהירות, באמצעות תכנות ניהול של חבילות תאריך-מחרוזת ב-Bash.

## איך לעשות

תחילה, נצטרך להתקין את החבילה "dateutils", שתווסיף לנו כלים קלים לתאריכיים ולימים. הפקודה להתקנה היא:

```Bash
apt-get install dateutils
```

עכשיו, נראה כיצד להמיר תאריך למחרוזת. ראשית, יש להכניס את התאריך למשתנה אחד (אנו נשתמש כאן בתאריך היום הנוכחי):

```Bash
today=$(date +'%d-%m-%Y')
```

האופציה "+'%d-%m-%Y'" תגדיר את תבנית התאריך שאנו מעוניינים בה, במקרה זה יום-חודש-שנה. ניתן להתאים את התבנית לפי צורת התאריך שברשותנו.

כעת, אם נדפיס את המשתנה החדש שיצרנו, נקבל את התאריך הנוכחי בתבנית המבוקשת:

```Bash
echo $today
```

שימו לב שהתאריך מתפרש לפי המיקום הגיאוגרפי של המחשב שלכם. אם תרצו לקבל את התאריך בזמן אחר, למשל תבנית אמריקאית, תוכלו להוסיף את הפקודה הבאה לקוד:

```Bash
TZ=America/New_York date
```

כך תעדיפו לקבל תאריך בתבנית אנגלית.

## העומק של תאריך-מחרוזת

כמו שראינו, באמצעות החבילה "dateutils" ניתן להמיר בקלות תאריך למחרוזת. אך השילוב בין תאריך ומחרוזת יכול להיות חייבת במידת מה בעיקר לצורך טיפול בנתונים.
