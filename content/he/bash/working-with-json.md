## למה

למה לדעתי כדאי לעבוד עם JSON ב-Bash? ה-JSON הינו פורמט נתונים נפוץ ומושך מאוד בעולם התכנות. העבודה עם JSON ב-Bash מאפשרת לנו לנתח ולעבד נתונים מורכבים ללא הצורך בתוספות חיצוניות ובכך להקל על פיתוח תוכניות.

## כיצד לעשות זאת

כדי לעבוד עם JSON ב-Bash נצטרך להשתמש בפקודות מיוחדות המאפשרות קריאה, כתיבה וניתוח של נתוני JSON. הנה דוגמא לקוד שבו אנו יוצרים קובץ JSON חדש וקוראים אותו באמצעות פקודות באש:

```Bash
# יצירת קובץ JSON חדש
echo '{"name": "John", "age": 30, "city": "New York"}' > person.json

# קריאה של נתונים מהקובץ
cat person.json

# ניתוח נתונים בקובץ לעזרת jq
jq '.name' person.json
```

הפלט המשובץ יהיה כך:

```Bash
{"name": "John", "age": 30, "city": "New York"}
John
```

בדוגמה זו אנו יוצרים קובץ JSON חדש וקוראים אותו על ידי פקודות בשלב הראשון. לאחר מכן, באמצעות הפקודה jq אנו משתמשים בתוספת חיצונית שמאפשרת לנו לנתח ולדפדף בנתוני ה-JSON בקלות. בשימוש בפקודות אלו אנו יכולים ליצור, לערוך ולקרוא נתוני JSON ב-Bash ובכך לשפר את תהליך הפיתוח שלנו.
