## למה
תיבה של פקודת כתיבה (Command Line Arguments) היא דרך חזקה ויעילה כדי להשתמש בתכונות של שפת התכנות באש (Bash), וכדי לבצע פעולות מתכונים. למשל, זה מאפשר לנו להעביר פרמטרים לתוכניות ב-bash ולהתאים אותן לתנאים שונים וליצור ערך מאיחור עבור piped commands.

## איך לעשות
##### הקוד:
```
#!/bin/bash

echo "מספר הפרמטרים: $#"

for arg in "$@"
do
    echo "פרמטר $arg"
done
```
##### פלט דוגמא:
```
מספר הפרמטרים: 3
פרמטר hello
פרמטר world
פרמטר אומנות
```

## מה בעומק
כאשר אנחנו משתמשים בפקודת כתיבה בכתובת הבאה, יש לנו אפשרות להגדיר פרמטרים מעל ידי הוספת המילה "$" לפני שם הפרמטר. למשל, כאשר אנחנו מגדירים את הפקודה "פרמטר $arg" המופיעה מעל, בתוך הפקודה אנחנו משתמשים בערך הפרמטר שנקבע מראש ע"י המשתמש כאשר הוא מכניס את הפרמטרים בעת הרצת התוכנית הזו.
