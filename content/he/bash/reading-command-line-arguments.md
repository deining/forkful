## מדוע

כאשר אנחנו רוצים להריץ תכניות ביש אפ ולתפעל פקודות בש ללא ממשק משתמש גרפי, עשויים להיות מצבים שבהם אנחנו צריכים לקרוא פרמטרים מהשורת הפקודה. למשל, כאשר אנחנו מריצים סקריפטים באופן אוטומטי, או כאשר אנחנו מריצים כמה פעולות בש ביחד. בכתבה זו נלמד איך לקרוא פרמטרים מהשורת הפקודה בעזרת שפת התכנות בש.

## איך לעשות זאת 

לקרוא פרמטרים מהשורת הפקודה בבש מתבצע בעזרת המשתנה $1 לועזי, המכיל את הפרמטר הראשון המועבר לתוכנית. ניתן לשרשר את המשתנה $ למספר כולליים נוספים כדי לקרוא פרמטרים נוספים מהשורת הפקודה. לדוגמה, אם אנחנו רוצים להדפיס את תוכן הפרמטר הראשון והשני, נשתמש בפקודה הבאה:

```Bash
echo $1 $2
```

כאשר במקומות המספריים ייכנסו הפרמטרים הראשון והשני שנועברו לתוכנית.

עוד דרך לקרוא פרמטרים מהשורת הפקודה היא בעזרת המשתנה $@ לועזי, שמכיל את כל הפרמטרים שנועברו לתוכנית בצורה של מערך. ניתן לעקוב אחרי אורך המערך ולקרוא את כל הפרמטרים שנועברו בעזרת לולאת for. לדוגמה:

```Bash
for param in "$@"
do
	echo $param
done
```

התוכנה שלנו תקרא כעת את כל הפרמטרים שנועברו לה מהשורת הפקודה ותדפיס אותם אחד אחרי השני.
