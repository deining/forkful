# למה:

קריאת ארגומנטים בשפת Bash היא כלי חיוני בתהליך התכנות. היא מאפשרת לנו לעבד עם פרמטרים שמועברים לתוכנית בעת הרצתה, מאפשרת גישה מהירה ויעילה למידע חיצוני, ומקלה על התאמת התכנית לצרכי המשתמש. במאמר זה נלמד איך לקרוא ארגומנטים בשפת Bash וניתן דוגמאות של כיצד להשתמש בכך בקוד.

# איך לעשות זאת:

נתחיל על ידי יצירת קובץ חדש בשם "sample.sh":

```Bash
#!/bin/bash

# יש לציין לשפת Bash כיצד לקרוא ולהשתמש בארגומנטים:
echo "הארגומנט הראשון הוא: $1"
echo "הארגומנט השני הוא: $2"

# כעת נראה דוגמאות של כיצד להשתמש בארגומנטים בקוד:
# לאחר התקנה תחילה של התוכנית, נכנס לתוכת ומפעילים אותה עם הארגומנטים:
./sample.sh Hello World

# תוצאות הרצת התוכנית:
הארגומנט הראשון הוא: Hello
הארגומנט השני הוא: World
```

בדוגמא הנ"ל, אנו משתמשים במפתחות $1 ו-$2 כדי לקרוא את הארגומנטים הראשון והשני שמועברים לתוכנית בסדר.

נוכל לעשות שימוש במפתח $@ כדי לקרוא את כל הארגומנטים שמועברים לתוכנית:

```Bash
#!/bin/bash

# הדפסת כל הארגומנטים באמצעות המפתח $@:
echo "הארגומנטים הם: $@"

# תוצאות הרצת התוכנית:
./sample.sh Hello World
הארגומנטים הם: Hello World
```
