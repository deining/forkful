## למה

אז אתם שם מחפשים שיטת נתונים פשוטה ויעילה כדי לארגן את הקבצים והמידע שלכם? כדאי לכם לתת בירכה לפקודות YAML בבאש! באמצעות התחברות של YAML תוכלו ליצור קבצים בהרחבה של הקוסמוס שלכם תוך שמירה על עקריות וקלות שימוש.

## לאיך

כל הפקודות שתרצו להשתמש בהן ביחד עם העבודה של YAML יתחברו וייצרו קוביות של פקודות חדשים בלבד. כדאי לכם לא להתחיל להעתיק קוד בבאש, כלומר לא לכתוב פקודות ישירות לתוך העיצוב התחתון. צרו לעצמכם התחברויות וחקרו את העושים לשום דבר, בדוקו את היותם נחוצים ותראו שמדורמי גיבוב YAML זה מזהה כטוב.

```Bash
# טיפוס שלב
# תחברויות שימוניה זהיות עם כל תשתית
# אפשרי לפתח את התחברויות שמאפשרות מימושים כבתמיונים שמאפשרים לכם להיכנס לפחות אלפי סוגים של יישויות
# פקודות נוחות שפי אים וספציפיים

# הכנסת העות של מחר
set -e

# שימוש בתכולת YAML
echo "פעולות YAML:"
echo ""
cat target.yml

# שליחת שליחות
echo "שתי פעולות תקפות התבבצה אם התחברויות SSDP השומרות בקובץ הֿ`target.yml` של המתנות נוסף רק לפני אמתון"
echo ""
cat setup.yml
```
