## מה ולמה?
עבודה עם YAML היא תהליך שבו תוכנת מחשב משתמשת בקוד כדי להכתיב תחבירים ולאחסן מידע. תוכנת YAML מאפשרת למתכנתים ליצור מבנים נתונים מורכבים ולשמור על מידע בצורה שהוא יהיה קריא ונוח לשימוש בתוכניות.

## איך לעבוד עם קובץ YAML?
קובץ YAML משתמש בתחביר פשוט וקריא שנקרא "מפתחי תמלול". בשביל להזין מידע לקובץ YAML, ניתן להשתמש במפתחות כדי לתאר נתונים, ערכים ותת-מפתחות. כדי להדפיס נתונים שנמצאים בקובץ YAML, ניתן להשתמש בפקודת "cat" בתוך פקודת bash. לדוגמה:

```
בשביל להשתמש בקובץ YAML בתוך פקודת באש, נכתוב את הפקודה הבאה:
cat document.yaml
```

כדי לשנות נתונים בקובץ YAML, ניתן להשתמש בתוכנת עריכה כגון Nano או Vim.

## חקירת עומק
YAML הוא פורמט תמלול שפותח בשנת 2001 כחלק מפרויקט תומך הגיבוק של ימל. פורמט זה נוצר כדי להפעיל את התחביר מתוך כלי ממשק נתונים שלטי. ישנן אלטרנטיבות לUBML כגון XML וJSON, אשר מתאימים למטרות דומות. בשל תחביר פשוט של YAML, ניתן ליצור קבצי YAML קריאים וקלים לתחזוקה.
