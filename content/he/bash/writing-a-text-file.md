## למה

לכתוב קובץ טקסט באש הוא דרך נוחה ומהירה ליצור קבצים טקסט פשוטים מתוך הטרמינל. זה יכול לשמש לשלוח תרגילים או הוראות למשתמשים אחרים, ליצור קבצי נתונים או להעביר מידע בין תוכניות שונות.

## איך לעשות זאת

ניתן לכתוב קובץ טקסט באש באמצעות כתיבת הפקודה `echo` וביטוי הטקסט הרצוי בין גרשיים כפולות. למשל, על מנת ליצור קובץ הכולל את המילה "שלום", נכתוב:

```Bash
echo "שלום" > hello.txt
```

תוכלו לוודא כי הקובץ נוצר באמצעות הפקודה `ls` כאשר רץ מתוך אותו תיקייה. יש לציין שהפקודה `>` מבצעת פעולת הכוונה (redirecting) של פלט הפקודה אל הקובץ המתאים.

אם אתם מעדיפים לשלוח פלט של פקודה קודמת לקובץ טקסט, ניתן להשתמש בפקודה `history` כדי למצוא את מספר הפקודה הרלוונטית ולהשתמש בפקודת `!` כדי להפעיל אותה. למשל:

```Bash
!100 > output.txt # אתם יכולים להשתמש במספר פקודה המתאים במקום 100
```
