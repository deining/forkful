## למה

כתיבת קובץ טקסט היא חלק חשוב מתכנות ב-Bash. היא מאפשרת לנו ליצור קובץ טקסט שיכיל מידע שנוכל להשתמש בו במגוון רחב של תכניות ותהליכים.

## איך לעשות זאת

קוד בפיתוח ב-Bash עשוי להיראות מסורבל או מסובך לפעמים, אבל ליצור קובץ טקסט אינו כזה קשה. הנה כמה דוגמאות לכיצד ניתן לצור קובץ טקסט ב-Bash:

```Bash
# יצירת קובץ טקסט חדש ריק
touch new_file.txt

# הוספת תוכן לקובץ
echo "שלום עולם!" >> new_file.txt

# קריאה והדפסת תוכן מקובץ
cat new_file.txt
```

תוצאה:

```
שלום עולם!
```

כמו כן, ניתן להשתמש בתוכניות נוספות כמו nano או vim כדי לערוך או ליצור קבצי טקסט בפני עצמם.

## טקסט מעמיק

כעת שאנו יודעים כיצד ליצור קובץ טקסט פשוט, נעבור לכמה פקטים עמוקים יותר. כאשר אנו משתמשים בכתיבת קבצי טקסט ב-Bash, ישנם כמה תווים ספציפיים שצריך להיות עריכים:

- הפקודה `>` משמשת כדי למחוק את התוכן הקיים בקובץ ולהאפס אותו לטקסט חדש.
- הפקודה `>>` משמשת כדי להוסיף תוכן לקובץ קיים, במקום למחוק את תוכן קיים.
- כאשר אנו מעבירים את תוצאת פקודה לקובץ באמצעות `>` או `>>`, התוצאה תהיה כאילו אנו משתמשים ב-stdout כדי להדפיס תוצאה.
