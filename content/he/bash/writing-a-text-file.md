# למה

כתיבת קובץ טקסט היא חלק חיוני מתכנות בש. היא מאפשרת לנו לאחסן ולארגן מידע ונתונים בקוד שלנו. בנוסף, יצירת טקסט טוב עוזרת לנו להתאים את הקוד שלנו לכללי תחביר.

# כיצד לעשות זאת

لכתוב קובץ טקסט בבש לא יהיה קשה יותר מדי. נוכל לעשות זאת באמצעות פקודת הבש "echo" ולהוסיף את התוכן לקובץ עם הפקודה ">>". לדוגמה, כדי ליצור קובץ טקסט חדש בשם "mytext.txt" ולהוסיף לו שורת טקסט פשוטה, נריץ את הפקודות הבאות:

```Bash
echo "שלום, עולם!" >> mytext.txt
```

בכדי לקרוא את התוכן שהוספנו לקובץ, נוכל להשתמש בפקודת "cat" כך:

```Bash
cat mytext.txt
```

הפלט שלנו יהיה:

```Bash
שלום, עולם!
```

עם כל זה הסבר, כעת יש לכרזת לבני, לכתוב את הקוד שלנו בכיתב - תוכלו להוסיף כמה תוויות ולנסות לעבוד אם פתרון כלשהו שלכם.

# יצירת קבצי טקסט בעומק

כשאנו מדברים על כתיבת קבצי טקסט בבש, ישנם עולם עמוק נוסף שאנו יכולים לחקור. אחת האפשרויות הכי מוכרות היא להשתמש בפקודת "printf" במקום "echo". הפקודה תאפשר לנו להתאים את התוכן שלנו מבחינת התבניות שאנו רוצים. לדוגמה, נריץ את הפקודה הבאה:

```Bash
printf "שלום, עולם\nמה המצב?" >> mytext.txt
```

שורות טקסט הלכו לצאת הפעם אחרה לקובץ שלנו:

```Bash
שלום, עולם
מה המצב?
```
