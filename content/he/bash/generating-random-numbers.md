## למה

בתכנות Bash יתכן כי תצטרך ליצור מספרים אקראיים כחלק מתהליך כלשהו. לדוגמה, לבדוק אלגוריתם או ליצור מספרים אקראיים שייכנסו לתוך קבצי נתונים. השתמשות בכלי שיבצע פעולת "גרירה" עבור כל המספרים בעליל יכול לקחת המון זמן ומאמץ, ולכן יעיל יותר ליצור אותם באופן אקראי.

## כיצד לכתוב קוד

בשביל ליצור מספרים אקראיים אנו נשתמש בפקודת rand של Bash. פקודה זו תחזיר לנו מספר מעורבב בין 0 למספר שנרשמן לפחות. הנה דוגמאות לכיצד ניתן ליצור מספרים אקראיים ב-Bash:

```bash
# מספר אקראי בין 0 ל-10
rand=$[$RANDOM % 10]
# מספר אקראי בין 20 ל-50
rand=$[$RANDOM % 50 + 20]

# יצירת רשימת מספרים אקראיים בקבץ מסויים
for ((i=1; i <= 10; i++))
do
  rand=$[$RANDOM % 100]
  echo $rand >> numbers.txt
done
```

התכנית הזו תיצור קובץ בשם "numbers.txt" המכיל 10 מספרים אקראיים בין 0 ל-100. ניתן להתאים את התנאים לפי הצורך, וכמה בלוקי rand כמו בדוגמאות ניתן לשים בתוך תוך if או while אם יש צורך ליצור מספרים אקראיים בהבנה תנאית בין הטקסטים להעביר רק על טקסט ספציפי.
