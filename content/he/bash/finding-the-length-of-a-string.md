# מדוע

כשמדברים על תכנות בשפת Bash, מדי פעם נתקלים בצורך לדעת את אורך המחרוזת שנמצאת על ידינו. זה עשוי להתחול מאבחון של מספר התווים במחרוזת או מרצון לבצע תנאי מסוים בהתאם לאורך המחרוזת. בכתבה הזו נלמד איך למצוא את אורך המחרוזת בBash.

# איך לעשות זאת

תחילה, ננסה להשים לב להבדיל בין שני טיפוסי תווים בBash: מחרוזת וטקסט. מחרוזת היא סדרה של תווים שנכתבים בתוך גרשיים, כמו ״מחרוזת״ בדוגמה הזו. טקסט, מצד שני, הוא סדרה של תווים שאינם נכתבים בתוך גרשיים, כמו בדוגמה של המספר 5.

למצוא את אורך המחרוזת ניתן להשתמש בפונקציה המתאימה שנקראת "כמות התווים". הנה דוגמה של שימוש בפונקציה כדי להדפיס את אורך המחרוזת "מחרוזת":

```Bash
#!/bin/bash

STR="מחרוזת"
echo "אורך המחרוזת הוא ${#STR}"
```

פלט של הקוד הזה יהיה:
```
אורך המחרוזת הוא 7
```

עוד דרך למצוא את אורך המחרוזת בBash היא להשתמש במפתח תוים "אורך" כך:

```Bash
#!/bin/bash

STR="מחרוזת"
echo "אורך המחרוזת הוא ${#STR}"
```

פלט של הקוד הזה יהיה:
```
אורך המחרוזת הוא 7
```

# תהליך מעמיק

ישנן עוד כמה דרכים למצוא את אורך המחרוזת בBash, כגון להשתמש בשאלה בצורה הבאה:

```Bash
STR="מחרוזת"
echo ${#STR}
```
פלט של הקוד הזה יהיה:
```
7
```
