# מדוע

למה לנו לרצות לדעת את אורך המחרוזת? זה נראה כמו משימה פשוטה, אבל איך נעשה את זה בפועל? בפוסט הזה נציג לכם כיצד למצוא את אורך המחרוזת בשפת התכנות באש.

## כיצד לעשות זאת

ניתן למצוא את אורך המחרוזת באמצעות שתי דרכים שונות. הראשונה היא באמצעות הפקודה `expr length`, לדוגמה: 
```Bash
length=$(expr length "Hello World")
echo $length
```
הפקודה `expr length` מחזירה את אורך המחרוזת כתוצאה, וכך ניתן לעבוד עם התוצאה במשך התוכנית.

הדרך השנייה היא באמצעות התכונה הטבעית של איוונט בשפת באש, הנקראת `$ {#string}`. לדוגמה: 
```Bash
string="Hello World"
echo ${#string}
```
במקרה זה, התוצאה תהיה 11, מכיוון שהשם "Hello World" כולל 11 תווים.

## צלילה עמוקה

למדנו על שתי הדרכים המקובלות למצוא את אורך המחרוזת, אך יש גם דרכים נוספות לעשות זאת. לדוגמה, ניתן להשתמש בפקודה `wc` עם הפרמטר `-m`, שמחזירה את מספר התווים במחרוזת. ניתן גם להשתמש בלולאות או בפקודת `sed` כדי לחשב את אורך המחרוזת של כל שורה בקובץ.

אנו גם יכולים לעשות שימוש באותיות האנגליות בשפת באש על מנת למצוא את אורך המחרוזת, על ידי המרת כל אות למספר באמצעות הפקודה `ord`.

מצאנו שהמציאות של שפת באש מאפשרת לנו למצוא את אורך המחרוזת בצורה מותאמת אישית לכל מקרה שאנו צריכים להשתמש בה.
