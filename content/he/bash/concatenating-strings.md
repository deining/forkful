## מה ולמה?
כדאי להכיר בתהליך של חיבור מחרוזות בתכנות כי זה חלק חשוב ביותר בכתיבת קוד. במקרה שלנו, חיבור מחרוזות הוא פשוט התמצאות מחדש של שתי מחרוזות כדי ליצור מחרוזת אחת ארוכה יותר. זה יכול להיות שימושי מאוד כאשר יש צורך לבנות קבצי טקסט או להדפיס משתנים במטלות מסוימות.

## איך לעשות זאת?
העובדה הכי מדהימה על חיבור מחרוזות זה שזה מאוד קל לבצע ב-Bash. אתה פשוט צריך להשתמש בסימן פיסוק `+` כדי לרצות מחרוזת למחרוזת אחרת. ננסה לדוגמה:

```Bash
name="גיל"
greeting="היי"
echo $greeting$name
```
רק השתמשנו בסימן פיסוק `+` כדי לחבר את שתי המחרוזות ולקבל את הפלט הבא:

```
הייגיל
```

ואם אתה רוצה להוסיף גם מעגליים בין המחרוזות, אתה יכול לעשות זאת כך:

```Bash
name="גיל"
greeting="היי"
echo ${greeting}${name}, אתה אהוב עליי!
```
ומה שיהיה אם תרצה להוסיף מספרים או תווים נוספים? זה גם אפשרי!

```Bash
first_number=10
second_number=5
result=$(($first_number + $second_number))
echo המספר הראשון הוא $first_number והשני הוא $second_number, הסכום שלהם הוא $result
```
תשובת הפלט המשוערת תהיה:

```
המספר הראשון הוא 10 והשני הוא 5, הסכום שלהם הוא 15
```
