## לטובת מה

תארו לכם שאתם רוצים ליצור כתבת בלוג ואתם רוצים שהתאריך הנוכחי יופיע בפוסט שלכם. או שתהיו צריכים ליצור קובץ עם שם התאריך היום. איך אתם יודעים את התאריך הנוכחי של מהשיור המערכת שלכם? נכון, אנחנו מדברים על מדף של באש.

## איך לעשות

האפשרות הכי פשוטה להשתמש בפקודה הפנימית DATE. הנה דוגמא של קוד פשוט המדפיס את התאריך הנוכחי:

```Bash
  date
```

תוצאה:

```
כיוון שכתבתי את הקוד הזה ב -23 במרץ 2021, התוצאה הייתה:

Wed Mar 24 02:08:58 EEST 2021
```

ניתן גם להוסיף את הפקודה --rfc-3339 כדי לקבל את התאריך בפורמט של תרגום זמן אינטרנטי:

```Bash
  date --rfc-3339=date
```

תוצאה:

```
2021-03-24
```

ישנם גם אפשרויות נוספות כמו לקבל את התאריך בפורמט מותאם אישית או להוסיף תאריכים נוספים לתוצאה. לפרטים נוספים, יש לעיין בדרך השימוש בפקודה DATE במניין העזר:

```Bash
  date --help 
```

## צלילה מתקדמת

כאמור, DATE הוא פקודה פנימית של באש וקיימות עוד המון אפשרויות לקבלת התאריך הנוכחי ולצבור מידע נוסף על השרת שלכם. אפשר להשתמש ב CRON JOB כדי ליצור סקריפט שיפעל באופן יסודי בתור רקע וניתן גם להשתמש ב VARIANCE לשמור את התאריך בקובץ או בבסיס נתונים.
