## למה

באיזו נסיבות בדיוק יכולים להיות סיבות לפנות כלכלית כדי לקבל תאריך נוכחי?

הרבה פעמים בשימוש ב- Bash, נצטרך להציג למשתמש את התאריך הנוכחי. זה יכול להיות משהו כמו להציג את תאריך היום בתוך תוצאת פעולת כלים, ליצור קבצים עם שם התאריך הנוכחי או אפילו רק כדי לדעת מתי ניתן לדעת בשביל להתמודד עם הקוד שכתבנו.

## איך לעשות זאת

ישנם מספר דרכים להשיג את התאריך הנוכחי בתוך Bash. הדרך הכי פשוטה ונפוצה היא להשתמש בפקודת `date` ולבחור את הסיגנצית הנכונה לתאריך שאנו רוצים לקבל.

לדוגמה, אם אנו רוצים לקבל את התאריך הנוכחי בפורמט של יום/חודש/שנה, נוכל להשתמש בפקודת הבאה:

```Bash
date +"%d/%m/%Y"
```

הפקודה תחזיר משהו כמו `04/04/2021` תלוי בתאריך הנוכחי.

ניתן גם להשתמש בפקודת `date` כדי ליצור קובץ עם שם התאריך הנוכחי כפי שאתה רוצה.

```Bash
touch "$(date +"%d-%m-%Y").txt"
```

זה יצור קובץ בשם "04-04-2021.txt" אם אתה מריץ את הפקודה כמו שכתוב עכשיו.

מידע נוסף על פקודת `date` והפרמטרים האפשריים ניתן למצוא ב[המדריך הרישמי של GNU `date`](https://www.gnu.org/software/coreutils/manual/html_node/date-invocation.html).
