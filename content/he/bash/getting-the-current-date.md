# למה
תארו לרגע שאתם מעצבים אתר או יישום חשמלי ואתם רוצים להציג למשתמש את התאריך הנוכחי. או שאתם רוצים ליצור קובץ עם שם היום כחלק מהפעולה שלכם. אלו הם רק כמה מהמקרים בהם ייתכן שתחשובו על מקבל התאריך הנוכחי. בחירה טובה של כל פקודת באש מוודאת שתקבלו את התאריך הנוכחי בפורמט שבו אתם צריכים.

# איך לעשות זאת
ניתן לקבל את התאריך הנוכחי בעזרת הפקודה "date" בפקודות הבסיסיות של באש. אתם יכולים להשתמש בפרמטרים מגוונים לפקודה כדי להגדיר את הפורמט שבו אתם רוצים שהתאריך יוצג. לדוגמה, ניתן להשתמש בפקודה הבאה כדי לקבל את התאריך הנוכחי בפורמט של MM/DD/YYYY:

```Bash
date +"%m/%d/%Y"
```

אם אתם רוצים להוסיף את שם היום לתאריך, ניתן לעשות זאת עם הפרמטר "%A" כך:

```Bash
date +"%A, %m/%d/%Y"
```

כלומר, אם אתם מריצים את הפקודה הנ"ל היום, התאריך שיתקבל יהיה:

```Bash
Wednesday, 08/11/2021
```

ניתן להשתמש גם בפקודה "cal" כדי לקבל את התאריך הנוכחי בפורמט של לוח שנה. ניתן לראות דוגמאות נוספות וללמוד עוד על הפקודות הללו על ידי הרצת פקודת "man" עליהם.

# חפירה עמוקה
