## מדוע
ביצוע פעולות על קבצים זמניים הוא בעייתי ויתרון ליישום של תוכניות שונות. כאשר תוכנית משתמשת בקובץ זמני, יכולה להיות בטוחה שהפעולות שנעשו עליו לא ישפיעו על קבצים אחרים במערכת. כמו כן, קבצים זמניים נמחקים בסופו של דבר, ולכן לא מצריך לדאוג לניקוים במיוחד.

## כיצד ליצור קובץ זמני בבאש
כדי ליצור קובץ זמני בבאש, אנו נשתמש בפקודת "mktemp", אשר יוצרת קובץ זמני ריק ומחזירה לנו את הנתיב שלו. ניתן להשתמש באופציות מגוונות כדי להתאים את הפעולה לצרכים שלנו. לדוגמה:
```Bash
new_file=$(mktemp) # יוצר קובץ זמני בתיקייה הנוכחית
echo "Welcome to my Bash blog post!" >> $new_file # מכניס טקסט לקובץ הזמני
cat $new_file # מדפיס את התוכן של הקובץ
```

פקודת "mktemp" מאפשרת גם ליצור קבצים זמניים בתיקיות ספציפיות ולהגדיר את השם של הקובץ. לדוגמה:
```Bash
new_folder=$(mktemp -d ./temp_folder_XXX) # יוצר תיקייה זמנית בתוך התיקייה הנוכחית ומחזיר את הנתיב שלה
new_temp_file=$(mktemp -p ./temp_folder_XXX new_file_XXX) # יוצר קובץ זמני בתיקיית התיקייה הזמנית שנוצרה ומכניס לו שם שמתאים לכל התיקיות הזמניות שיוצרות
```

כמו כן, ניתן גם להשתמש בפקודת "mktemp" בתוכניות כדי לשמור על יצירת קבצים זמניים עם שמות ייחודיים, כדי למנוע חומר גלישה בין תוכניות שונות.
