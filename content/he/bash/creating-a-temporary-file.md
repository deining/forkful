# למה

יצירת קובץ זמני בולטת בדרך כלל כאמצעי עבור תוכניות שמצרפות קבצים או מידע שנדרש בזמן הריצה. הקובץ הזמני מאפשר למשתמש לשמור ולדאוג שהמידע יישמר וימחק בסיום התוכנית.

## איך לעשות זאת

כדי ליצור קובץ זמני ב-Bash, ישנם שני אפשרויות עיקריות: באמצעות הפקודה "mktemp" או על ידי שימוש ב-Vim ופקודת "tempname()". נבחר להשתמש בפקודת "mktemp" בפוסט זה, מכיוון שהיא יותר פשוטה לביצוע.

כדי ליצור קובץ זמני ב-Bash, ניתן להשתמש בפקודה הבאה:

```Bash
temp=$(mktemp)
```
הפקודה הזו יוצרת קובץ זמני בתיקיית הפעלה הנוכחית ומשמשת את השם שנוצר בתור המשתנה "temp". ניתן להגדיר גם תבנית לשם הקובץ כדי להגדיר את סוג הקובץ ולהגביל את הגישה לקובץ למשתמשים מסוימים. לדוגמה, הפקודה הבאה תייצר קובץ זמני בפורמט "text" בתיקיית הפעלה הנוכחית:

```Bash
temp=$(mktemp --suffix=.txt) 
```
ניתן להוסיף גם את הפרמטר "p" כדי להפנות לתיקייה ספציפית ליצירת הקובץ בתוכה. לדוגמה, "mktemp -p /home/user/temp" ייצור קובץ זמני בתיקיית "/home/user/temp".

לאחר שיצרנו את הקובץ הזמני, ניתן להשתמש בו כך:

```Bash
echo "Hello World!" > $temp 
```
הפקודה הזו תכתוב את המידע "Hello World!" לקובץ הזמני שיצרנו. ניתן להוסיף גם את הפרמטר "-n" כדי להימנע מהוספת שורת תווים חדשה לסיום הכתיבה לקובץ.
