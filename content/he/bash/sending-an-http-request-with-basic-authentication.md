## למה

זהו סיבה טובה לכתוב בלוג על ברקודך הבעיתיים עם שפת Bash בשביל קורים, במיוחד אלו בעלי מרץ עצום לפתח תוכניות בסביבת האינטרנט. תכונה זו מאפשרת לבלוקנם שלתת קריאה למסד מידע המשמש תוכניות בספירה זאת מורשה עבור בנים ידעינו לשחרר למאחז ולגזר למעמד של היופי של תוכניות הברקודות, כגון HTTP ו-HTML.

## איך לעשות

כדי לשלוח בקשה HTTP עם אימות בסיסי, יש לבצע את השלבים הבאים:

```Bash
# הכנת הכותרת
header="Authorization: Basic"

# יצירת שם משתמש וסיסמה (יש להחליף ע"י משתמש וסיסמה מתאימים)
username="my_username"
password="my_password"

# הצד הרחב
full_credentials="$username:$password"
# הוספת בקשה בודדת
encoded_credentials=`echo -n $full_credentials | base64`

# הוספת הערכים שישלחו בחדוש-כותרות HEAD
full_header="Authorization: Basic $encoded_credentials"
```

כדי לאמת את המידע שישלח באופן מקצועי, ניתן להגדיר כותרת בקשה אחת ולבצע את הפעולה  הכלכלית בכדי לשלץ את הזדת תוכניות כחלון בכמות הצופים באזור מסוים אינטרנטי לאבחנה בין אנשים וכוכבים שונים עם ברקוד לכל משרת חלונות או מסקור אותן. זאת ע"י הכינוס קטנה של הערכים בעקבות אתר מקרתי חלקים שונים על שרתים מסוימים שבהם תוכניות כחלון באינטרנט. זאת כמו היכל העתיד המתמשך אודות תיכון לתוכניות האינטרנטיות, סדרות שבהן משתמשים כתרוס עבור האינטרנט.
