## למה

CSV הוא פורמט נתונים פשוט ופופולרי המשמש כדי לאחסן נתונים בטבלאות ולשתף אותם בין יישומים שונים. ניתן לתת פקודות למחשב בשפת Bash כדי לעבוד עם קבצי CSV ולצרוך וליצור מידע מסודר ונוח לשימוש.

## איך לעשות זאת

כדי לעבוד עם CSV ב-Bash, ניתן להשתמש בפקודות פשוטות כמו `sed`, `awk` ו-`grep`. להלן דוגמא פשוטה לפיתוח קובץ CSV ב-Bash:

```Bash
# צור קובץ CSV חדש
echo 'שם,גיל,עיר' > נתונים.csv

# הוסף שורות עם מידע על אנשים
echo 'שולה,25,תל אביב' >> נתונים.csv
echo 'דניאל,33,ירושלים' >> נתונים.csv

# הדפס את נתוני הקובץ
cat נתונים.csv

# סנן אנשים מגיל 30 ומעלה
cat נתונים.csv | awk -F',' '$2 >= 30' 

# חפש אנשים ששמם מתחיל באות 'ש'
cat נתונים.csv | grep '^ש'
```

הנה הפלט של הסינון והחיפוש:

```
שם,גיל,עיר
דניאל,33,ירושלים
שם,גיל,עיר
שולה,25,תל אביב
שם,גיל,עיר
שולה,25,תל אביב
```

## חקירה מעמיקה

כלי ה-Bash מספקים מגוון רחב של אפשרויות ופקודות לעבודה עם קבצי CSV. ניתן לעבוד עם נתונים מסוגים שונים ולבצע פעולות מגוונות כגון סינון, חיפוש, עריכה וסידור. בנוסף, ניתן להשתמש בעיבוד מתקדם כגון תנאים, לולאות ופונקציות כדי להתאים את העבודה לצרכים המיוחדים של המפעיל.
