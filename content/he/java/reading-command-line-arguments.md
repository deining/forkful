# למה
רבים מתבוננים מדוע יש לקרוא פרמטרים בשורת הפקודה כאשר מתכנתים בשפת ג'אווה. הסיבה העיקרית היא שזהו כלי חשוב שמאפשר לנו להפעיל את תוכניותינו עם פרמטרים שונים בכל פעם, מבלי לצטרך לשנות את הקוד עצמו. זה מאפשר לנו ליצור תוכניות גמישות יותר ולהתאים אותן לצרכים שונים.

# איך לעשות זאת
כדי לקרוא פרמטרים משורת הפקודה בשפת ג'אווה, יש להשתמש במחלקת "Args" שמכילה את כל הפרמטרים המועברים לתוכנית בזמן הרצה. הנה דוגמא של קוד פשוט ופלט של תוכנית המדגימה כיצד לקרוא פרמטרים משורת הפקודה:

```Java
public class Args {
    public static void main(String[] args) {
        for(int i=0; i< args.length; i++){
            System.out.println("פרמטר מספר " + (i+1) + ": " + args[i]);
        }
    }
}
```

כאשר נריץ את התוכנית עם שני פרמטרים משורת הפקודה "java Args hello world", הפלט יהיה:

פרמטר מספר 1: hello
פרמטר מספר 2: world
