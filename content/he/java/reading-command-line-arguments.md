## מה ולמה?
קריאת פרמטרים מהשורה הפקודה היא תהליך שמאפשר למתכנתים לקבל קלט מהמשתמש בזמן ריצת התוכנית. זה מאפשר לכם ליצור תוכניות גמישות יותר שיכולות לעבוד עם קלטים שונים בכל פעם שהתוכנית מתבצעת.

## איך לעשות זאת:
כדי לקרוא פרמטרים מהשורה הפקודה בשפת ג'אווה, יש להשתמש במחלקת "String[] args". הנה דוגמה פשוטה שמדגימה כיצד להדפיס את כל הפרמטרים שהוזנו מהשורה הפקודה:
```Java
public class CommandLine {
    public static void main(String[] args) {
        for (int i = 0; i < args.length; i++) {
            System.out.println(args[i]);
        }
    }
}
```
להמשך הדוגמה, אם נקרא את התוכנית באמצעות הפרמטרים "hello" ו"world", נקבל את הפלט הבא:
```
hello
world
```

## חקירה מעמיקה:
בהמשך לדוגמאות הקוד והפלט שהוצגו, כדאי לציין שמאז יצירת השפה בשנת 1995, דרך זו של קליטה של פרמטרים מעולם לא השתנתה. עם זאת, קיימות גם דרכים אחרות לקבל קלט מהמשתמש, כגון קריאה מקובץ או שימוש בממשק המשתמש הגרפי (GUI). ככל שהתוכנית שלכם מתקדמת, כה חשובה מאוד תהיה יכולת לקבל קלט מהמשתמש, ולכן קשה מאוד לכמות את חשיבותה של קריאת פרמטרים מהשורה הפקודה.

## ראה גם:
לפרטים נוספים על קריאת פרמטרים מהשורה הפקודה בשפת ג'אווה, אתם מוזמנים להיכנס לאתר הרשמי של Oracle Java: https://docs.oracle.com/javase/10/docs/api/java/lang/String.html
