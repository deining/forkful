## מדוע

קריאת ארגומנטי שורת הפקודה ב-Java היא כלי חשוב לכל מפתח. היא מאפשרת לנו לתקשר עם התוכנית שלנו בזמן ריצה ולקבל קלט מהמשתמש בצורה יעילה וקלה.

## איך לבצע

תחילה, נגדיר במחלקה שלנו את הפונקציה המקבלת את הארגומנטים משורת הפקודה:

```Java
public static void main(String[] args) {
    // כאן נוכל לעבוד עם הארגומנטים שהתקבלו
}
```

כדי לקבל את הארגומנטים בזמן הרצה, עלינו להשתמש במשתנה args שנמצא בתוך מחלקת ה-Main. לדוגמה, אם נרצה להדפיס את כל הארגומנטים שהתקבלו בשורת הפקודה הזו נוכל לעשות כך:

```Java
System.out.println(args);
```

ואם נרצה להדפיס כל ארגומנט בנפרד, נוכל לעשות כך:

```Java
for (String arg : args) {
    System.out.println(arg);
}
```

עכשיו, כאשר מפעילים את התוכנית ומעבירים את הארגומנטים דרך שורת הפקודה, נקבל את הפלט הבא:

```
[תוכן הארגומנטים שהועברו]
```

או אם אנחנו משתמשים בלולאה, את הארגומנטים הודפסו אחד אחד:

```
תוכן ארגומנט 1
תוכן ארגומנט 2
תוכן ארגומנט 3
. . .
```

## חקירה מעמיקה

כאשר אנחנו משתמשים בפונקציה "main" לקבלת הארגומנטים, אנחנו משתמשים במערך של מחרוזות כך שבניגוד למתודות אחרות, כאן לא ניתן להכניס כוונה מיוחדת כגון "." כדי לשלוט על הסדר של התוכן שהתקבל.
