## למה

עשרות פעמים בתהליך התכנות אנחנו נתקלים בצורך ליצור קבצים זמניים בקוד הג'אווה שלנו. יכול להיות שאנחנו רוצים לשמור נתונים בצורה מזמנית מבלי ליצור קובץ קבוע על המחשב, או אולי נדרש ליצור קובץ זמני על מנת לעבוד עם ממשקי יישום אחרים. למטרות כאלה ועוד, ייצור קבצים זמניים הוא כלי חיוני במחברת של כל מתכנת ג'אווה.

## כיצד ליצור קובץ זמני

כדי ליצור קובץ זמני בג'אווה, אנו נשתמש במחלקת File ויש לנו מספר אפשרויות לבחירה. אחת הדרכים היא להשתמש בפעולות createTempFile שמקבלת שם תיקייה ושם קובץ כפרמטרים ומחזירה אובייקט File המייצג את הקובץ הזמני הנוצר. לדוגמא:

```
File tempFile = File.createTempFile("myTempFile", ".txt");
```

בפעולה זו, אנו יוצרים קובץ זמני בשם "myTempFile" שיהיה בתיקייה המוגדרת הנמצאת במערכת הפעולה שלנו, כמו גם סיומת הקובץ המוגדרת כ ".txt".

נשים לב כי קובץ זמני נוצר במקום רנדומלי במערכת הפעולה שלנו. כדי לקבוע מה התיקייה ושם הקובץ המדויקים, נוכל להשתמש בפעולת createTempFile עם פרמטר additionalPrefixes המאפשר לנו לספק תיקייה ושם קובץ מדויקים כמקור ליצירת הקובץ הזמני.
