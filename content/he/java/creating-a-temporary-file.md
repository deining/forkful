## למה

יצירת קובץ זמני היא פעולה חשובה בתכנות בשפת ג'אווה. היא מאפשרת לנו ליצור ולעבוד עם קבצים שצריכים להיות זמינים רק לזמן קצוב ואחרי זה יימחקו. זה יכול להיות שימושי במקרים שבהם נרצה לשמור נתונים זמניים שאינם דרושים לנו לצמיתות, כגון לבצע קריאות בקבצי מידע מקוונים.

## כיצד לבצע

לצורך יצירת קובץ זמני בשפת ג'אווה, ניתן להשתמש בפונקציות הקיימות בחבילת `java.io.File`. אחת מהן היא `createTempFile()` שמקבלת כפרמטרים את הקידומת והסיומת של הקובץ הזמני. לדוגמה:

```Java
File tempFile = File.createTempFile("data", ".txt");
```

הקובץ הזמני ייווצר במיקום המחשב המדויק של המשתמש וישמור בו את הנתונים שרצינו לשמור. ניתן לקרוא את הנתונים שנשמרו בקובץ על ידי שימוש בפונקציות קריאה של ה-`File` כדוגמת `read()` ו- `readLine()`.

## חקירה עמוקה

כאשר נקודת התחלה של יצירת הקובץ הזמני היא מיקום המחשב של המשתמש, ניתן לשנות זאת על ידי שימוש בפרמטר האופציונלי השני של הפונקציה `createTempFile()`. ניתן להגדיר מיקום מסוים על-ידי שימוש בפונקציה `getPath()` על פרמטר הקובץ הזמני.
