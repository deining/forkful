# מדוע

כתיבת קוד בכל שפה מדויקת וספציפית תלויה בשימוש מתמדת של העכבר ותאריכים נכון. מכיוון שקבלת תאריך אינה משימה פשוטה, יכול להיות מועיל לדעת כיצד להשתמש בתקן של תאריך נתון בקוד הפיתון. בכתבה זו, נלמד כיצד לקבל את התאריך הנוכחי בשפת ג'אווה.

## איך לעשות זאת

```java
import java.time.LocalDate; // לייבוא מחלקת תאריך בשביל להשתמש

// קוד לקבלת תאריך נוכחי
LocalDate today = LocalDate.now();

// קבלת תאריך מלא בפורמט מסוג 
System.out.println("תאריך נוכחי: " + today);
```

פלט:

```java
תאריך נוכחי: 2021-09-23
```

הקוד הנ"ל מדגים איך להשתמש במחלקת התאריך של ג'אווה כדי לקבל את התאריך הנוכחי בפורמט מסוג.

## ירידה מעמוק

כיצד פועלת המחלקה LocalDate שתמונגה במקרה שלנו?

בגלל שהתאריך נצפה כמחרוזת (String) לפני המחלקה להפוך את התאריך למחרוזת. למחרוזות קיימים טיפים מיוחדים המאפשרים לנו להציג תאריכים בפורמט ידידותי יותר, כך שהם יהיו נוחים לקרוא.

לדוגמה, התאריך הנוכחי שמתקבל במחלקה LocalDate מוכנס למחרוזת בפורמט מסוג נתון בסביבת המחשב והנתונים נשמרים בתפריט התאריך של הסביבה. ישנם מספר ספציפציה המשמשות למיפוש או נידון בתאריך מסוים לפי שהמחלקה LocalDate שלך תשתמש במיפוי הנתונים של סביבת פיתוח להצגת זמן אמת (מקומי) כמו תאריך מסוים.
