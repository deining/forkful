## Why

למה: רק 1-2 משפטים המסבירים *למה* מישהו ירצה להמיר תאריך לנתונים.

הממצאים שהוציאו ממכתב סקריפט ב-Java מוצאים שזה נעשה מעמיק להפיק נתונים מפורמט תאריך לפורמט מחרוזת כי יש לנו זמן בוא יהיה חיוני בשפת תכנה הקרובה למבנה הוא מובהק של התקנות על בסיס שנת וחודשות ואחרת, יש לפרט לזה על זמן להמיר תאריך למחרוזת ייתכן שתהיה  אחת מן הפירות של השפה שלנו למען התיקונים שמשותפים. לכן, ליצור את היכולת להמיר תאריך למחרוזת עבור שפת Java יכול להיות דבר חשוב וכלים חשוב לכל מתכנת Java.

## How To

```Java
import java.util.Date;
import java.text.SimpleDateFormat;

// יצירת משתנה מסוג תאריך
Date date = new Date();

// יצירת מחרוזת של התאריך בפורמט מבוקש
SimpleDateFormat sdf = new SimpleDateFormat("dd/MM/yyyy");

// המרת התאריך למחרוזת באמצעות הפורמט המוגדר
String dateString = sdf.format(date);

// הדפסת התוצאה למסך
System.out.println("תאריך מקורי: " + date);
System.out.println("תאריך מהמחרוזת: " + dateString);

// פלט:
// תאריך מקורי: Fri Oct 15 13:45:29 IDT 2021
// תאריך מהמחרוזת: 15/10/2021
```

הפקודות הראשון כאשר אתה להמיר תאריך למחרוזת היא ליצור אובייקט מתאריך ואתה תחלל להשתמש באותו כדי להמיר את התאריך למחרוזת. בשמות גם יוצרים טי סטילי להמיר את המחלקה לשם שאותו מאלו פעמים הדוגמאות שמציינות ספציפי למקרה הכללי להמיר את התאריך למחרוזת המגינה עם הכפייה על כל התאריכים ומחרוזות עם איזו הפורמ
