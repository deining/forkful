## למה

אנשים ישקיעו מאמץ רב בקישור מחרוזות בגלל שבצורה זו מתאפשר להחליפה נוחה של ערכים בתוך מחרוזות קיימות.

## איך לעשות זאת

קישור מחרוזות בשפת ג'אווה הוא בעצם התמצאות שלו היכן שהמחברת מחרוזת אחת למחרוזת אחרת. כדי לעשות זאת, ניתן להשתמש בפעולת הוספה "+" ולציין את המחרוזת הנדרשת לצידו השמאלי, ואת התוכן שברצוננו לקשר לצידו הימני. ניתן להשתמש גם בפעולת השמכה "concat" כדי לקשר מספר מחרוזות יחדיו.

```Java
// דוגמה לשימוש בפעולת הוספה לקישור מחרוזות יחד
String firstName = "שירלי";
String lastName = "כהן";
String fullName = firstName + " " + lastName;
System.out.println(fullName); // תוצאה: שירלי כהן

// דוגמה לשימוש בפעולת השמכה לקשר מחרוזות יחד
String sentence = "כלבים מגיעים לבית";
String newSentence = sentence.concat(" - וחתולים גם!");
System.out.println(newSentence); // תוצאה: כלבים מגיעים לבית - וחתולים גם!
```

## שקיפות מעמיקה

בג'אווה, כדי להקשור בין מחרוזות מספריות, ניתן להשתמש בפעולת הוספה או לשמוך למחרוזת המתאימה של מספר. כמו כן, ניתן להשתמש בפעולת השמכה על מחרוזת ריקה כדי להוסיף טקסט לסוף או לתחילת מחרוזת קיימת.

כאשר משתנה מסוג String נוצר, ישנן תורת הקיצור שיכולת לקצר את התהליך של קישור מחרוזות באמצעות הפעולות "+" ו"concat". תורת הקיצור מאפשרת לחלוטין קישור של מחרוזות בפעולה אחת, ומחקקת השמכות מחרוזת למשתנה חדש.
