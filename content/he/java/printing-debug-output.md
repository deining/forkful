## למה
כשאנחנו מפתחים בג'אווה, חשוב להכיר כלים שיכולים לעזור לנו באתרוג פוגים בקוד שלנו. הדפסת פלטי דיבאג יכולה להיות כלי מועיל כאשר אנחנו מנסים להבין מה קורה בקוד שלנו ולמצוא באיזו נקודה הפגיעה התרחשה. 

## איך להדפיס פלטי דיבאג
נחביא כל קוד תחת בלוק קוד "```Java ... ```", ולהדר עליו כדי להדגים. לדוגמה, בואו ננסה להדפיס את הערך של משתנה באמצעות המתודה `System.out.println()`:

```Java
int num = 5;
System.out.println("The value of num is: " + num);
```

פלט:
```
The value of num is: 5
```

ניתן גם להשתמש במתודה `System.out.printf()` כדי לקבל פלט מעוצב יותר עם תבניות. לדוגמה:

```Java
String name = "יונתן";
int age = 30;
System.out.printf("%s הוא בן %d שנים.", name, age);
```

פלט:
```
יונתן הוא בן 30 שנים.
```

כדי להדפיס את ערך משתנה בלי נקודות עשרוניות, ניתן להשתמש במתודה `System.out.print()` ולאחריו להדר על המשתנה הרלוונטי. לדוגמה:

```Java
double pi = 3.14159;
System.out.print("The value of pi is: ");
System.out.println(pi);
```

פלט:
```
The value of pi is: 3.14159
```

```Java
int num = 10;
if(num > 5){
    System.out.println("The value is greater than 5.");
}
