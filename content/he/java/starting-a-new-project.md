## מה ולמה? 
פתיחת פרוייקט חדש היא פעולה כדי להתחיל את התהליך של כתיבת קוד חדש עבור פרוייקט מסוים. מתכנתים עושים זאת כדי ליישם רעיונות חדשים, לתקן באגים או לפתח תכונות נוספות.

## איך ל: 
Java לפתח פרויקט חדש בשפת ראשונם צריך ליצור קובץ לווא חדש, להוסיף את היבואים הדרושים, ולקבוע קוד את תהליך המערכת. להלן דוגמאות לכתיבת קוד בגישה נכונה:

```Java 
public class NewProject {
  public static void main(String[] args){
    System.out.println("פרוייקט חדש הוא הרצה מערכתית של קוד חדש.");
  }
}
```

בפעם הראשונה שנריץ את הקוד, נקבל כתוצאה "פרוייקט חדש הוא הרצה מערכתית של קוד חדש" בעורך הקוד.

## גילוי עומק: 
לפני התקן השרתים כלל בשנת 1995, מתכנתים היו צריכים לכתוב קוד בסדרת כלים שונים להיכנס לעסק התחתון של פרוייקט חדש. עם זאת, היום הכל נהיה קל יותר עם יישום Java הקיים. ישנם גם אלטרנטיבות כגון Kotlin ו-Groovy המשתמש בתחביר יותר אנושי וקל לקריאה. לפתח פרוייקט חדש כדי לבקש מפעלי תאגידי להשקיע כאפשרות אחרת נפוצה.
