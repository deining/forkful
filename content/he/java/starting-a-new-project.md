# למה

בעולם התכנות, יצירת פרויקט חדש היא דבר רגיל ועקבי. קיימים מספר סיבות שיאלצו אדם להתחיל פרויקט חדש, כגון: צורך לפתח יישומון חדש, חיפוש אחר חברים לשיתוף פעולה על פרויקט משותף או סתירה עם פרויקט קיים.

# איך לעשות זאת

כדי להתחיל פרויקט חדש בשפת ג'אווה, יש לעבוד בשלבים הבאים:

1. יצירת תיקייה חדשה - מומלץ להשתמש בניהול גרסאות כגון Git כדי לשמור על קוד יצירתי ולאפשר שיתופי פעולה קל בין מפתחים.
2. סיווג תיקיות - נוסיף תיקיות עבור קוד מלכתחילה (source code) ופקודות (commands) עבור בנייה והרצה.
3. יצירת קבצי Java - ניתן להשתמש בעורך טקסט כמו Notepad++ או IDE כגון Eclipse ליצירת קבצי Java חדשים עבור פרויקט.
4. כתיבת קוד - נתחיל לכתוב קוד עבור הפרויקט ולהריץ נסיונות כדי לבדוק את הפעולה שלו.
5. בניית והרצת - נשתמש באחד מכלי הבנייה כגון Maven או Gradle כדי להניע את הפרויקט ולהריצו.

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("שלום עולם!");
    }
}

// הפלט יהיה: שלום עולם!
```

# העימוק העמוק

כאשר מתחילים פרויקט חדש, יש לקחת בחשבון את המסלול המתאים עבור הפרויקט ואת הכלים המתאימים עבור הצורך. כמו כן, יש ליצור תיעוד זמין לפרויקט כדי לסייע למפתחים אחרים להצטרף לפרויקט ולתרום לו.
