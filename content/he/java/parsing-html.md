## למה
אנשים מנסים לפענח HTML מסיבות שונות, אולם הסיבה העיקרית היא כדי לאפשר טיפול יעיל במידע מתקמפי אתרים. עם עוצמת רכיבי האחסון של גמישות HTML, פירוק ועיבוד ריכוזי של המידע באתרים הרבה פעמים דורש ניתוח אתרים. כאילו שאנו מתחברים לגיאוטילו מעבורת תריסים למידע בטוח שמופיע רק אחרי ניתוח לפני פתיחת הרכיב.

## איך לעשות זאת
הכי נפוץ בקוד מודרני מרכזת כיצד אנו יכולים לפתח את המדריך לפענח HTML על ידי השתמשות בספריית JSOUP. לפני ניתוח המידע באתר על ידי יוצר אתר, ניתן ליצור קובץ HTML פרי עיגול. העתקת המידע מעבר לקובץ ניתן למדריך, והם יעבירו להציג את דף האתר שיוצר אתות מילון הקישה לאחר בית המדריך. לחיות יותר לפנים הפרי-עיגול.

```Java
Document doc = Jsoup.connect(url).get();
String title = doc.title();
String body = doc.body().text();
System.out.println("כותרת הדף היא: " + title);
System.out.println("התוכן של הדף הוא: " + body);
```
