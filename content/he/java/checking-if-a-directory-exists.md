## למה

בעולם התכנות, המידע והנתונים מאוחסנים במבני תיקיות שונים. לפעמים נדרש לבדוק אם תיקייה מסוימת קיימת לפני שאנו ממשיכים בקוד שלנו. ייתכן שנרצה להבין למה זה חשוב ואיך נוכל לבצע את הבדיקה באמצעות קוד ג'אווה. 

## איך לבדוק אם תיקייה קיימת

עבור חלק ממשימות התכנות, נדרש לבדוק אם תיקייה קיימת במערכת הקבצים. לדוגמה, אם אנו מעבירים קובץ ממיקום אחד למיקום אחר או מחלקת תיקיות ישנה לתיקייה חדשה. כאשר אנו משתמשים בשפת תכנות ג'אווה, נוכל לבדוק אם תיקייה קיימת באמצעות הפעולה File.exist (). בדוגמה הבאה, נבדוק אם תיקייה בשם "images" קיימת בתוך התיקייה הנוכחית:

```java
File directory = new File("images");
if (directory.exists()) {
    System.out.println("The directory exists!");
} else {
    System.out.println("The directory does not exist.");
}
```

תוצאה:

```
The directory exists!
```

כפי שניתן לראות, התכנית חיפשה תיקייה בשם "images" והדפיסה הודעה מתאימה, תלוית בתוצאת הבדיקה. אנו יכולים גם להשתמש בפונקציה זו עבור תיקייה בכתובת מלאה כפי שמוצג בדוגמה הבאה:

```java
File directory = new File("/Users/username/images");
```
