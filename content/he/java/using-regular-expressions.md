## למה

למה להשתמש בביטויים רגילים בתכנות? ביטויים רגילים הם כלי חזק וחשוב בתכנות, המאפשר לנו לחפש ולמצוא תבניות וטקסטים במהירות ובנוחות. הם מאפשרים לנו גם לבצע פעולות חילופיות והחלפות בטקסט בקלות רבה.

בנוסף, ביטויים רגילים מאפשרים לנו לאסוף ולסנן מידע מפורמטים שונים כמו כתובות דוא"ל, מספרי טלפון ותאריכים. הם מועילים ביישומים רבים כמו ניתוח נתונים, בדיקות אוטומטיות ויישומים ווב.

## איך להשתמש

כדי להשתמש בביטויים רגילים בקוד ג'אווה, נצטרך להשתמש במחלקה קיימת שנקראת "Pattern". אפשר ליצור אינסטנס חדש של מחלקה זו על ידי שימוש בפונקציה "compile" ולהעביר אליה את הביטוי הרגיל שאנו רוצים לחפש. לדוגמה:

```Java
Pattern pattern = Pattern.compile("[a-z]+");

//במקרה זה, הביטוי הרגיל מוצא כל אותיות קטנות בלבד בטקסט שמיועד להיווצא
```

אחרי שיצרנו את האינסטנס של "Pattern", נוכל להשתמש בפונקציות נוספות כדי לבדוק אם טקסט מסוים מכיל או לא מכיל את הביטוי הרגיל, לחלץ מידע מתוך הטקסט או לבצע פעולות חלופיות. לדוגמה:

```Java
//פונקציה שבודקת אם טקסט מסוים מכיל את הביטוי הרגיל
boolean containsRegex = pattern.matcher("Hello world").find();

//פונקציה שמחזירה את כל המתאים לביטוי הרגיל בטקסט
Matcher matcher = pattern.matcher("This is some text");
while(matcher.find()){
    System.out.println(matcher.group());
}
//פלט: "This", "is", "some", "text
