מה ולמה?
המרת מחרוזת לאותיות קטנות היא פעולה די נפוצה בתכנות המתבצעת כאשר מחרוזת מתאימה נדרשת להיות באותיות קטנות במקום האותיות הגדולות המקוריות. תכניתנים מבצעים פעולה זו כדי להקל על בדיקת מחרוזות והשוואה בין תווים.

איך לעשות זאת:
כדי להמיר מחרוזת לאותיות קטנות בשפת ג'אווה, ישנם שניים מתודות פשוטות לעשות זאת - "toLowerCase()" ו- "valueOf()". הנה כמה דוגמאות של הכתיבה של קוד והפלט התוצאה שלהן:

```java
//מחרוזת לאותיות קטנות באמצעות השיטה toLowerCase()
String str1 = "הגאלקטיקה";
String lower1 = str1.toLowerCase();
System.out.println(lower1); // הגאלקטיקה

//מחרוזת לאותיות קטנות באמצעות השיטה valueOf()
String str2 = "JAVA";
String lower2 = String.valueOf(str2).toLowerCase();
System.out.println(lower2); // java
```

תילקוט מעמיק:
המרת מחרוזת לאותיות קטנות היא פעולה שנמצאת מאז ימי תחילת התכנות ונמצאת בשפות רבות כמו פיית'ון, C++ וג'אווה. בכל אופן, יחד עם כמות השפות המובנות שתומכות בפעולה זו, ישנן גם כמה אלטרנטיבות שיכולות לבצע טריקים דומים עם מחרוזות, כגון השימוש במתודת "toUpperCase()" הממירה את התווים לאותיות גדולות במקום קטנות.

ראה גם:
כדי לשפר את היכולת שלך להשתמש במחרוזות ופעולות כמו המרת מחרוזת לאותיות קטנות, מומלץ לעיין במידע נוסף על פעולות מחרוזת בג'אווה, כגון המתודות "charAt()" ו- "replace()". אתה יכול למצוא מידע רב ייעודי יותר בקריאת עזרה מהמדריכים הרשמיים של ג'אווה או באתרים של מתכנתים שונים שנותנים מידע נוסף על יישומים ספציפיים.
