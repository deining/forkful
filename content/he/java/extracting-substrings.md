## למה

הוא לא ידידותי כשמנסים לאתר מחרוזת מסוימת בתוך מחרוזת ארוכה יותר. במקום לאפשר למשתמש לחפש בכל המחרוזת, חציצה (substring) מאפשרת לנו למצוא רק את המחרוזת המדויקת שאנו מחפשים ולכן מפחיתה את זמן החיפוש וגם מאפשרת לנו לעבוד בצורה יעילה יותר.

## איך לעשות זאת

תחילה, נצטרך לייבא את הספרייה המתאימה בשם ```java.lang.String```. לאחר מכן, נוכל להשתמש בפונקציה ```substring(int startIndex, int endIndex)``` כדי לחלץ חציצה מתוך המחרוזת המקורית. לדוגמה:

```Java
String str = "Hello World!";
String substr = str.substring(6, 11);
System.out.println(substr);
```

פלט: 

World

כפי שאתם רואים, אנחנו משתמשים בשני פרמטרים - המיקום הראשון מציין את האינדקס של התו הראשון בחציצה והשני מציין את האינדקס של התו אחרון בחציצה. חשוב לציין כי האינדקסים מתחילים מ-0 וכי התו האחרון של החציצה לא יכלל.

ישנן גם פונקציות נוספות שיכולות לעזור לנו לחלץ חציות ממחרוזות, כגון ```substring(int startIndex)``` המחזירה חציצה מהמיקום הנתון ועד סוף המחרוזת, או ```substring(int startIndex, int endIndex)``` המחזירה חציצה מהמיקום הנתון ועד סוף המחרוזת.
