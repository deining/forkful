## למה
פעם בפעם, אנו נדרשים לעבוד עם מספרים אקראיים בקוד שלנו. הם יכולים לשמש בתור מפתחים, או למטרות חישוביות ומשחקים. כתיבת קוד שיכול ליצור מספרים אקראיים בצורה מדויקת היא כישור חשוב שצריך לעשות כדי להצליח כמפתח תוכנה.

## כיצד לעשות זאת
נדמה שכתיבת קוד ליצירת מספרים אקראיים הוא משהו ממש מורכב. אבל למזלנו, יש לפתרון פשוט ויעיל - את הפקודה "Random" בשפת ג'אווה. אנחנו פשוט נצרף את הפקודה לחבילת הקוד שלנו ונגדיר את המספרים המינימלי והמקסימאלי שאנו רוצים ליצור. הנה דוגמה בשפת ג'אווה:

```Java
// ייבוא מהפקודה Random
import java.util.Random;

// הגדרת אובייקט מסוג Random עם מספרים מיוחדים
Random rand = new Random();

// הגדרת טווח ליצירת מספרים אקראיים מ-0 עד 100
int min = 0;
int max = 100;

// הדפסת מספר אקראי בין 0 ל-100
System.out.println(rand.nextInt(max - min + 1) + min);
```

פלט התוכנית:

```
50
```

אנחנו גם יכולים להשתמש בפקודה זו עם מספרים עשרוניים או עם מספרים שליליים באופן דומה.
