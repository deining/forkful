## למה

אתה יכול לזרוק קוביות, לפצצות ולמזג בישבץ ללא הכרח בחיים האמיתיים, אבל לתכנת באמצעות גיליונות ולוחות הבקרה היא לא כמו שתי הבעיות ביחד. ייצור מספרים אקראיים הוא תהליך חשוב עבור תכנים שבנויים על ערכים אקראיים, כגון משחקי מחשב, מבחר, ותשכחם. כתבות זה ילמד אותך כיצד ליצור מספרים אקראיים באמצעות השפה Java.

## איך לעשות

איך יצרנו מספרים אקראיים בשפת Java באמת? נוצר את הברי דקדוקיות ```Math.random ()``` בשימוש בפעל שזה יחזיר מספר אקראי בין 0 ל-1. נוכל להשתמש בפעולת כפולם ובשינוי של טווח כדי ליצור מספרים אקראיים בטווח נתון.

```Java
int min = 1;
int max = 10;
double random = Math.random() * (max - min + 1) + min;
System.out.println(random);
```
