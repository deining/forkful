## למה

כשמתחילים ללמוד לתכנת בשפת ג'אווה, אחת הפעולות הראשונות שאנו לומדים היא למצוא את אורך התווים במחרוזת. ייתכן שתתבאסו עם כמות הזמן והמאמץ שדרושים בכדי ללמוד יכולות בסיסיות כמו זו, אך כאשר תתחילו להשתמש בזה באופן רציף תבינו כמה מעולה זה לדעת את אורך התווים במחרוזת כדי להשתמש בו להמשך הקוד שלכם.

## איך לעשות זאת

הנה דרך פשוטה למצוא את אורך התווים במחרוזת בעזרת שפת ג'אווה:

```java
String myString = "זהו הטקסט שאני מנסה למצוא את אורךו בשפת ג'אווה.";

// קוד זה יחזיר את אורך התווים במחרוזת, שהכוונה הוא 67
int length = myString.length();

System.out.println(length); // פלט: 67
```

מה שעושים פה הוא שאנו משתמשים בפונקציה `length()` המאפשרת לנו למצוא את אורך התווים במחרוזת שלנו. היא מחזירה מספר שלם המייצג את כמות התווים במחרוזת שנמצאת בין הסוגריים. כפי שניתן לראות בקוד את התוצאה שלנו.

ננסה עוד דוגמא על מנת לצמצם את המושג:

```java
String myName = "משפחתינו";
int length = myName.length();

System.out.println(length); // פלט: 8
```

כאן נשים לב שהמחרוזת שלנו מכילה 8 תווים ולכן התוצאה שלנו היא 8.

## צעד עמוק

כאשר מדברים על אורך התווים במחרוזת, חשוב להבין שהתווים הנמצאים במחרוזת לאו דווקא צריכים להיות אותיות. התווים יכולים להיות פסיקים, רווחים, מספרים ולא רק. אם נעשה שימוש בפונקציה `length()`
