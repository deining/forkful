## מדוע

שלום לכולם! בכתבה הזאת, אני רוצה לדבר איתכם על כמה נישומים של איך אפשר למצוא אורך של מחרוזת (string) בשפת ג'אווה. המטרה העיקרית של התרגיל הזה היא להציג דרך נוספת וייצוג יותר מובן על איך אפשר לקבל את הפתרון הנכון לשאלה הזו. לצורך כך, אני אשתמש בדוגמאות קומפקטיות ומסודרות כדי לביא בידי כל אחד מהסקרנים את הפתרונות האפשריים לשאלתנו. כמובן, לא צפוי ממני שישמשו את התוכן הזה כדי לפתור מתן חכמה - צריך לזכור שכל תרגיל יהיה בודד, מה שאני רוצה לומר היא שעוד שטיח ארכו תלוי ממש במקום ומשתנות שהוא מכיל. 

## כיצד 

אז כפי שאתם אמורים לדעת כבר, כדי לקבל את האורך של מחרוזת בשפת ג'אווה, יש לנו מספר באפשרויות כבר ספרנו קצת על אחת מהם למשל, כמו שעשינו פה בבלוג שלנו, יש לשים לב שגם אם גרוע בזלזול, יש, כרגע בפונקצית כפרו בסיסית רק שמנתחת מחרוזת (String). לכן אנו מתחילים ממניות שבורות

```java
public class StringLenghtExample {

    public static void main(String[] args) {
    	String str = "Hello, world!";
    	int length = str.length();
    	System.out.println("The length of the string is: " + length);
    }
}
```

כמו שאתם רואים בדוגמה שלא, אני משתמש בפונקצית .length() כדי לקבל את אורך המחרוזת. איך הפונקציה הזו עובדת? טוב, למרבה המזל, זה לא משהו תוסכל מידה המשמעותי לצרכים שלנו, ומחזירה לנו את האורך של המחרוזת הזו. לפחות את זה רצו
