## מה ולמה?

מציאת אורך של מחרוזת היא תכונה חשובה בתוכנית מחשב שהיא עוזרת לנו לטפל במחרוזות שונות ולבצע פעולות עליהן, כגון חיתוך או חיפוש. כמו כן, כאשר אנו מבצעים פעולות על מחרוזות בתכניות שלנו, חשוב לדעת את האורך שלהן כדי להיות מידעים פנימיים על המחרוזות.

## איך לעשות זאת:

```Java
String myString = "זו המחרוזת שלי";
int length = myString.length();
System.out.println("המחרוזת שלי יש אורך של " + length + " תווים.");
```

פלט:

```
המחרוזת שלי יש אורך של 14 תווים.
```

אנו משתמשים בפונקציה length () הקיימת בחבילת String של Java כדי למצוא את האורך של מחרוזת ספציפית. הפונקציה מחזירה מספר שלם שמייצג את מספר התווים במחרוזת. במקרה שלנו, המחרוזת שלי מכילה 14 תווים ולכן יש לנו גם ערך של 14 כאשר אנו משתמשים בפונקציה.

אם אנו רוצים לגלות את אורך של מחרוזת מסוימת לפני שהיא מכילה ערך, אנו יכולים להשתמש בפונקציה length () כדי להתאים את התנאי הנדרש בתכנית שלנו.

## מערכת שקופה:

החישוב של אורך מחרוזת הוא מסורת בתכנות וניתן למצוא כמעט בכל שפת תכנות שקיימת. ב-Java, ניתן למצוא את האורך של מחרוזת באופן מובנה עם הפונקציה length (). ישנן גם ערכות כלים נוספות שמציעות פעולות נוספות בתחום המחרוזות, כגון הצגת מחרוזות בפורמטים שונים או חיתוך והרכבה של מחרוזות.
