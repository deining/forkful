## מדוע

כאשר אנו כותבים קוד בג'אווה, ייתכן שנתקל בבעיות בתרגום הקוד לתוך פיתוח התוכנה הסופי. כתיבת טסטים יכולה לעזור לנו לסדר ולשפר את הקוד, ולוודא שהתוכנה תפעל באופן תקין כאשר נגישה למשתמשים. זה יכול לחסוך לנו הרבה זמן ולמנוע תקלות בעתיד.

## כיצד לכתוב טסטים

כאשר אנו כותבים טסטים עבור קוד בג'אווה, חשוב לדעת כיצד להשתמש במספר כלים ופונקציות כדי לשפר את קוד הטסט ולהבטיח שהוא יעבוד בצורה תקינה. נהלים ושיטות שונות יכולים לעזור לנו ליצור טסטים איכותיים ויעילים, ולספק לנו תוצאות מדויקות שיעזרו לשפר את התוכנה שלנו.

```Java
// דוגמה לשימוש ב-JUnit לכתיבת טסטים
import static org.junit.Assert.assertEquals;

public class ExampleTest {
    
    // בדיקת התוצאה הראשונה מתאימה לצפוי
    @Test
    public void testResult1() {
        int result = someMethod(2, 3);
        assertEquals(5, result);
    }

    // בדיקת התוצאה השנייה מתאימה לצפוי
    @Test
    public void testResult2() {
        int result = someMethod(5, 10);
        assertEquals(15, result);
    }
}
```
