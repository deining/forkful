## למה

למה לבצע חיפוש והחלפה של טקסט בקוד? כי כאשר מפתחים ב-Java, לעיתים קרובות נתקלים בצורך לשנות מחרוזות או טקסטים בקוד. יכול להיות שאנחנו רוצים להחליף את המילה על מנת לשפר את קוד התוכנית, לנקות את הקוד מתווי טיופים, או אחרות. לכן, היכולת למצוא ולהחליף טקסט היא חשובה ושימושית כאשר מפתחים ב-Java.

## כיצד לעשות זאת

כדי לבצע חיפוש והחלפה של טקסט בקוד שלנו ב-Java, נשתמש בשיטה `replace()` שמקבלת שני מחרוזות כארגומנטים. המחרוזת הראשונה היא המחרוזת שנרצה לשנות את מופעיה בקוד, והמחרוזת השנייה היא המחרוזת החדשה שנרצה להחליף בה. לדוגמה:

```Java
String str = "Hello world!";
String newStr = str.replace("world", "Java");
System.out.println(newStr);
```

בתוצאה האחרונה, התוכנית תדפיס "Hello Java!" כפי שהצלחנו להחליף את המילה "world" במילה "Java". אם ישנן מספר מופעים של המחרוזת שנרצה לשנות, נוכל להשתמש בשיטה `replaceAll()` שתחליף את כל המופעים במחרוזת החדשה.
