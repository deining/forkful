## למה

כאשר אנו מתכנתים ב-PHP בגרסתו הנוכחית, ייתכן שמדי פעם נתקל בבעיות בקוד שלנו המדגישות את חשיפת השגיאות. הדבר הכי נכון לעשות במצב כזה הוא להדפיס פלט של השגיאה או בעיה כדי לעזור לנו לנתח ולפתור את הבעיה. זה נקרא "הדפסת פלט פיתוח" והוא הכלי שלנו לתקן את הקוד ולוודא שהוא פועל כראוי.

## איך לעשות זאת

הנה כמה דגמים של קוד PHP עם הדפסת פלט פיתוח:

```PHP
<?php
// דוגמה ראשונה עם הדפסת פלט של משתנה
$name = "ישראל";
echo "השם שלי הוא: " . $name;
```

```PHP
<?php
// דוגמה שנייה עם הודעת שגיאה פשוטה
echo "כאן נמצאת הודעת שגיאה בעקבות חסרון הנתיב לקובץ";
```

```PHP
<?php
// דוגמה שלישית עם הדפסת מערך עם תוכן לא תקין
$array = array("אופניים", "פטריות", "משקפי שמש");
echo "הפריט הראשון במערך הוא: " . $array[0];
```

## לעמוק יותר

ה-Hypertext Preprocessor (PHP) הוא שפת תכנות דנמיית דפים מונחים ברשת המשמשת בשביל לבצע חישובים מתקנים ועצמיים מול שרת האינטרנט. חלק מהתכווונים שמתחילים בקובץ קידוד של PHP ירדו או הידרדו עם הפלט שדפים הן נותנים לקוד HTML שאוסף מעכב איזשהו. מדוע לא לכתוב קוד שדורש הדפסת הודעות שגיאה פיתוח? השגיאות מתווספות לפעם שעונות בשנה כאשר מפתח הפעולה בעצם נתקלים בתקלה שצריכה להתייחס אלימים לתיעוד הפלט הפיתות במר
