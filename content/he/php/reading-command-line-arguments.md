# למה?

אם אתה מתחיל ללמוד PHP או אם אתה כבר מומחה, קריאת ארגומנטים מסובכים מאוד יכולה לשפר את הקוד שלך ולהפוך את התכנות לקל יותר. במאמר זה נתחיל להתמקד בשלוש מילים שמתחילות ב "-" (מסימי טיפול בקלט בשורת הפקודה) ולדון באופן מיוחד בקריאת ועיבוד שלהם בקוד PHP.

# איך לעשות זאת

ברשותכם כמה דרכים לקרוא ולעבד ארגומנטים מתוך שורת הפקודה בPHP, אנו נתחיל עם שני מדגמים שיסבירו איך לעשות זאת בעזרת הפקודה המובנית "getopt()". הפקודה הזו יכולה לקרוא ולטפל בסוגים שונים של קלט כמו אופציות ופרמטרים.

```PHP
<?php
// קריאת הארגומנטים שבשורת הפקודה
$options = getopt("t:h", ["title:", "help"]);
// בדיקת האופציות שנבחרו
if (isset($options['t'])) {
    $title = $options['t'];
    echo "הכותרת שבחרת היא: " . $title . PHP_EOL;
}
if (isset($options['h'])) {
    // מידע על האפשרויות שיש לנו
    echo "אופציות זמינים: -t <title>, -h" . PHP_EOL;
}
?>
```

הכנס פלט נוסף כאשר נריץ את הקובץ כדלקמן:

```
$ php examples/01-options.php -t "ברוכים הבאים למאמר ההדרכה"
```

כעת הוספנו כותרת למאמר והיא מודפסת בלוגו (תודה לאפשרות "-t"). אופציית ביצוע בדיקת תקיות שונות במאמר.

ישנם גם דרכים נוספות לקבל ולטפל בארגומנטים של שורת הפקודה כמו באמצעות הפקודה "$_SERVER['argv']". כדי לראות דוגמאות נוספות ומפורטות, אנא בקרו בקישורים שבסוף המאמר.
