## למה

מדוע משתמשים בכלל בארגומנטים משורת הפקודה? התשובה היא פשוטה - כי זה הדרך הכי מהירה ונוחה לתקשורת עם המחשב. במאמר זה, נלמד כיצד לקרוא ארגומנטים משורת הפקודה ונראה דוגמאות פעילות מתוך קוד PHP.

## כיצד לקרוא ארגומנטים משורת הפקודה

לפני שנתחיל, נדרוש קובץ קובץ PHP חדש כדי לראות את הארגומנטים המבוקשים. עבור זה, נשתמש בפונקציה המתאימה - `getopt()`. המתפקד של הפונקציה הזו הוא לקרוא את הארגומנטים משורת הפקודה ולהחזיר אותם כמערך או אובייקט.

נראה דוגמא של מה קורה כאשר אנו מכניסים ארגומנטים לתוכן הקוד:

```PHP
<?php
//קבל ארגומנטים משורת הפקודה והצג אותם עם הסכומים שהוספנו
$options = getopt("a:b:c:");
print_r($options);
```

כעת, נקרא את הקובץ PHP הזה על ידי כתיבה של הפקודה הבאה בשורת הפקודה:

```bash
php myscript.php -a Mango -b Apple -c Banana
```

הפלט של הקובץ הזה יכיל את המערך הבא:

```
Array
(
    [a] => Mango
    [b] => Apple
    [c] => Banana
)
```

כמו שאתה יכול לראות, הפונקציה `getopt()` יצרה מערך אשר כולל את הארגומנטים שכנסנו לתוכניות.

## לקוח עמוק

למדנו איך לקרוא ארגומנטים משורת הפקודה באמצעות הפונקציה `getopt()`. עכשיו, נקבל תובנות על כיצד לנתח את הארגומנטים הללו ולעשות את הטיפול מאחורי הקלעים.
