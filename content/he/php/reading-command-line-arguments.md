## למה

וודאי ישנם רבים שמתעניינים בלימוד פיתוח בשפת PHP, ותכונת ארגומנטי פקודת הקלדה במסך מספקת דרך נוחה להיות מדוקדק יותר עם ביצוע תכניות. מאמר זה חשוב לכל מתכנת תוכניות פקודת מסך שמעוניי בשימוש יעיל בהן.

איך להשתמש בקלט מסך כארגומנטים בשפת PHP

```PHP
<?php
// בקלט מסך נכתוב את השורה הבאה
print_r($argv);
?>
```

התוכנית הזאת תדפיס את הארגומנטים שניתן להוסיף לקבצים בפקודת הקלדה במסך. בדוגמה זו, התוכנית תוציא כאלה:

```PHP 
Array
(
    [0] => script.php
    [1] => arg1
    [2] => arg2
)
```

כל אחד מהארגומנטים יחשוב כמחרוזה ויסובב בתוכנית באמצעות הטפל המרכיבים מחרוזת. אפשר לבחור כל מחרוזת כארגומנט בביצוע הפקודה הזאת במסך. המחרוזות האלה יכולות להיות מדוייקות, והן יכולות לכלול תווים של מחרוזות אחרות. 

## חפירה מעמוק
