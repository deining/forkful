## למה
בכדי להתחיל לכתוב תוכניות עם PHP, חשוב להבין את הפעולות הבסיסיות של השפה. אחת מהן היא להמיר מחרוזת לאותיות קטנות. זה מאפשר לנו להתמודד באופן נוח עם טקסטים, לדוגמה, לבדוק אם מחרוזת מכילה את המילה הרצויה בלי להתעסק עם האותיות הכיווניות שלה.
## איך לעשות זאת
כדי להמיר מחרוזת לאותיות קטנות ב-PHP, עלינו להשתמש בפונקציה strtolower(). נחשב למשל שאנו מעוניינים להדפיס את המילה "הפעולה" באותיות קטנות. ניצור משתנה חדש שמכיל את המילה המקורית, ונחזיר אותו כאשר נחשוב לקריאתו:

```PHP
$word = "הפעולה";
$lowercase = strtolower($word);
echo $lowercase;
```

הפלט של קוד זה יהיה "הפעולה". ניתן לראות שהמילה כעת כתובה באותיות קטנות. יותר מזה, אם נבצע בדיקה אם המילה כוללת את האותיות "ח" ו"ל", נוכל להשתמש בפונקציה בפורמט הבא:

```PHP
$word = "הפעולה";
$lowercase = strtolower($word);
if (strpos($lowercase, "חל") !== false) {
    echo "המילה מכילה את האותיות ח ול";
}
```

הפלט יהיה "המילה מכילה את האותיות ח ול", מראה שהפונקציה strtolower() מאפשרת לנו לתמוך בדרישות מסוימות על הטקסט בצורה נוחה.
