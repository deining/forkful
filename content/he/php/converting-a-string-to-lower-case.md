## איך ל:
להמיס את המילה לאותיות קטנות עשוי להיראות כמו תירטים, אבל בעצם הוא פשוט מאוד. בשפת תכנות PHP, תוכלו להשתמש בפונקציית strtolower () כדי להמיר מחרוזת לאותיות קטנות. למשל:

```PHP
<?php
$str = "HELLO WORLD";
echo strtolower($str);
?>
```
פלט:
```hello world```

אתם יכולים גם להשתמש בפונקציית mb_strtolower () להמרת מחרוזת מתבנה מצויין לאותיות קטנות במקום בקוד פשוט. לדוגמה:

```PHP
<?php
$str = "מחרוזת באותיות גדולות";
echo mb_strtolower($str);
?>
```
פלט:
```מחרוזת באותיות קטנות```

הנה עוד דוגמה עם תווים לאנגלית ועברית:

```PHP
<?php
$str = "Hello שלום";
echo mb_strtolower($str);
?>
```
פלט:
```hello שלום```
