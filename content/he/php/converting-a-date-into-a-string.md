##למה
בעולם התכנות ייתכן להתעניין בהמרת תאריך למחרוזת מסוימת בגלל עבודה עם מסדי נתונים או יישומי אינטרנט שמציגים תאריכים למשתמשים. במדריך הבא נלמד כיצד לבצע המרה זו בשפת פי.אץ'.

##כיצד לעשות זאת
נתחיל עם משתנה שמכיל תאריך ונשתמש בפונקציה date() להמרתו למחרוזת. ניתן להשתמש במספר פורמטים שונים כדי לקבל את התאריך בצורה מותאמת לצרכיים שלנו.

```PHP
$date = date('d/m/Y'); // יוצר מחרוזת עם התאריך הנוכחי בפורמט יום/חודש/שנה
echo $date; // ידפיס: 18/12/2020
```

בנוסף, ניתן להגדיר תבנית משלנו עם פונקציית strftime() ולהשתמש בפורמטים כמו %d ליום, %m לחודש וכו' כדי לעצב את המחרוזת בדיוק כפי שנרצה.

```PHP
$template = '%A, %d %B %Y'; // תבנית מותאמת אישית
$date = strftime($template); // ממיר את התאריך למחרוזת עם התבנית הנתונה
echo $date; // ידפיס: יום שישי, 18 דצמבר 2020
```

ניתן גם להשתמש בפונקציות נוספות כמו strtotime() כדי להמיר מחרוזת שמכילה תאריך לתאריך בפורמט מסוים ולאפשר להחליף בין פורמטים שונים בקלות.

- המרת מחרוזת לתאריך והפוך: https://www.php.net/manual/en/function.strtotime.php
