## מה ולמה?

קבלת תאריך נוכחי היא תהליך בו מכניסים לתוך קובץ תוכנה את התאריך הנוכחי של האווירה של המחשב. תהליך זה נעשה על מנת לדעת מתי התוכנה נכנסת לעבודה וכאשר עובדים על הקוד, אנחנו רוצים לדעת את התאריך והשעה בה הבעיה או תכנית הופעלה.

## איך לעשות?

כדי לקבל את התאריך הנוכחי ב-PHP ישנם כמה אפשרויות.

### דוגמאות:

```
// תאריך מקומי
echo date('d/m/Y'); // פלט: 02/09/2021

// תאריך עם שעה
echo date('d/m/Y H:i'); // פלט: 02/09/2021 12:30

// תאריך ושעה עם איזור זמן מסוים
date_default_timezone_set('Asia/Jerusalem'); // קבלת איזור הזמן המקומי
echo date('d/m/Y H:i'); // פלט: 02/09/2021 05:30
```
### פלט מוצא:

הפלט ישתנה בהתאם לתאריך ושעה הנוכחיים על המחשב שלכם.

## ירידה מקיפה:

1. שקפי היסטוריתיים- בעבר, מלקחת תאריך הנוכחי היה תהליך יותר מורכב ודרש שימוש בספריות חיצוניות.

2. אלטרנטיבות- כיום ישנן מספר אלטרנטיבות לקבלת תאריך הנוכחי, כמו שימוש ב-API של גוגל או שימוש בניידות שלי כדי לקבל את התאריך הנוכחי.

3. פירוטי יישום- תהליך הקבלת תאריך הנוכחי מנוצל היטב במדעי מחשב ומתנגש בתורי היישומים הטכנולוגיים המתקדמים ביותר.
