# במהןגרסת PHP: למה לקבל את תאריך היום

בעולם התכנות, תאריך היום הוא נתון מבוקש מאוד. ייתכן שתרצה להשתמש בתאריך היום בתוך קוד שלך כדי ליישם תנאים שונים או להציג נתונים מעודכנים בממשק המשתמש. בכתבה זו נכיר את כיצד לקבל תאריך היום בשפת PHP.

## איך לקבל תאריך היום ב-PHP

 לקבלת תאריך היום ב-PHP נשתמש בפונקציית `date()`. פונקציה זו מחזירה נתונים של תאריך ושעה מעוצבים לפי הפורמט שנציין. לדוגמה, ניתן להשתמש בפורמט `d/m/Y` כדי לקבל את תאריך היום בתבנית של יום/חודש/שנה. הנה דוגמה של קוד PHP עם הפונקציה `date()`:

```PHP
<?php 
$date = date('d/m/Y');
echo $date;
```

מכיל את התוצאה:

```
08/06/2021
```

הנה עוד דוגמאות לפי הפורמטים האפשריים שניתן להשתמש בהם בפונקציה `date()`:

- `d/m/Y`: 08/06/2021
- `l, F jS, Y`: Tuesday, June 8th, 2021
- `d M Y`: 08 Jun 2021

ניתן לקרוא עוד על הפונקציה `date()` ועל הפורמטים האפשריים ב[תיעוד רשמי של PHP](https://www.php.net/manual/en/function.date.php).

## תיחקור עמוק

בנוסף לפונקציה `date()`, ניתן להשתמש גם בפונקציה `time()`. פונקציה זו מחזירה את הזמן הנוכחי בשניות מאז התחלת השנה הגרגוריאנית (1 בינואר 1970). ניתן להשתמש בפונקציה זו בצורה חדשנית כדי למצוא תאריך מסוים על ידי הוספה או הפחתה של שניות מתאריך מסוים.
