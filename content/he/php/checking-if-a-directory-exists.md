## מה ולמה?

בתכניתות הפיתוח קודם שתתחיל לי, ייתכן שתרצה לבדוק אם תיקייה מסוימת כבר קיימת במערכת הקוד הנוכחית. בדיקה זו נחשבת לחשובה מאוד וחשובה כאשר אתה מפתח בפHP.

בדיקת התיקייה הקיימת מתבצעת כדי לוודא שהקוד שלך יכול להמשיך לבצע את פעולותיו נכון ולא להתבטל במהלך הרצת התכנית. זה עוזר למנוע באגים והפסקות באמצעות שמירה על היישומון שלך עם עמידות ואמינות גבוהה יותר.

## איך לעשות?

כדי לבדוק אם תיקייה כבר קיימת במערכת הקוד שלך בפHP, יש להשתמש בפונקציה `is_dir()` כדי לבדוק אם התיקייה נמצאת תחת `$_SERVER['DOCUMENT_ROOT']` או באמצעות כתובת מבוקשת. תוצאת הפונקציה תחזיר `true` אם התיקייה קיימת ו`false` אם התיקייה לא קיימת.

לדוגמה:

```PHP
if(is_dir($_SERVER['DOCUMENT_ROOT'] . '/example_dir')) {
    echo 'The directory exists!';
} else {
    echo 'The directory does not exist!';
}
```

פלט:

```
The directory exists!
```

## מערכת יםע淋

הבדיקה הלא דדיםהדמתינה שבירק

בנוסף לפונקציה `is_dir()` קיימות גם פונקציות אחרות שמאפשרות לבדוק אם תיקייה קיימת במערכת הקוד בPHP. לדוגמה, פונקציות כמו `file_exists()` ו- `stat()` יכולות לספק מידע נוסף על התיקייה כמו זמן היצירה והתאריך האחרון שמאומת. מומלץ לבדוק את התיקייה באמצעות מספר פונקציות כדי לוודא שהיא נמצאת וכדי לשמור על עמידות היישומון שלך.

כדי לבצע הרחבה נוספת על הנושא של בדיקת תיקייה קיימת בPHP, ניתן להתעניין בפונקציות המספקות פרטים נוספים כמו `is_readable()` ו- `is_writable()`. כמו כן, ניתן להשתמש במודול Phar המאפשר ליצור ארכיוני תיקייות ולבצע בדיקה עליהם.

למידע נוסף על הנושא, ניתן לעיין בקוד פורותל הרשמי של PHP ולצפות במסמכי ההתאמה הראשיים.
