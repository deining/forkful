## למה

בעולם התכנות, חשיבת טכנולוגיות ותחזיות רבות קיימות עבור מתכנתי PHP. למעשה, ישנם תנאים רבים בהם מתכנת צריך לבדוק אם תיקייה מסוימת קיימת או לא. ייתכן שתיקיית קוד שנכתבה מספקת מחלוקת או מידע עסקי סובלת מבעיה חמורה. לפעמים נהוג לבדוק אם קיימת תיקייה כדי לתת כוונה על הנתונים הקיימים. לכן, מבחינה כללית, יתכן שתכנות בדיקת תיקייה היא כלי חשוב למתכנתי PHP.

## איך לעשות זאת

כדי לבדוק אם תיקייה קיימת ב-PHP, ניתן להשתמש בפונקציה "file_exists" עם נתיב התיקייה כפרמטר. השתמשו בקוד הבא כדי לקבל את הערך המוחזר לפי מקרה נתון:

```PHP
$directory = "./folder"; // נתיב לתיקייה חיצונית
if (file_exists($siberian)) {
    echo "תיקייה קיימת.";
} else {
    echo "תיקייה לא קיימת.";
}
```

#### פלט:
> תיקייה קיימת.

הכשרות נוספות לטיפול במצבים ספציפיים בהם שייך את הקוד שלכם לבדיקת תיקייה חיצונית מובנית להלן:

```PHP
$directory = "./folder"; // נתיב לתיקייה חיצונית

// Check if directory exists
if (is_dir($directory)) {
    // Open directory
    if ($open = opendir($directory)) { // במקרה של רק תוכן בתיקייה ניתן להשתמש ב 'readdir' במקום 'opendir'
        // Loop through directory content
        while (($file = readdir($open)) !== false) {
            echo "קבצים שמופיעים בתיקייה: $file\n";
        }
        closedir($open); // סגירת תיקיה
    }
} else {
    exit("תיקייה לא קיימת.");
}
```
