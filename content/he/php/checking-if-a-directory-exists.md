## למה

תרגישו מתחילים ללמוד פיתוח אינטרנט ומעוניינים להבין את הקרקעות של PHP? אחת הסיבות המרכזיות לכך היא לבדוק אם תיקיית מסווג שניתנת לקבלת משתנים קיימת. הבדיקה הזו יכולה להיות מאפשרת לנו ליצור קוד מותאם אישית שיותאם למצב ספציפי וכך לענות על הצרכים הייחודיים שלנו.

## איך לעשות זאת

בשפת PHP, ישנם שני דרכים נפוצות לבדוק אם תיקייה קיימת או לא. הראשונה היא להשתמש בפונקציה המובנית `is_dir()` שמקבלת מחרוזת המייצגת את שם התיקייה ומחזירה את הערך `true` אם התיקייה קיימת ו`false` אם לא. ניתן גם להשתמש בפונקציה המובנית `file_exists()` שמקבלת את השם המשולש של התיקייה ומחזירה `true` אם התיקייה קיימת ו`false` אם לא.

לדוגמה, אם נרצה לבדוק אם תיקיית התמונות שלנו קיימת, נשתמש בקוד הבא:

```PHP
if (is_dir('תמונות')) {
  echo "התמונות שלי קיימות!";
} else {
  echo "אוי, אין לי תמונות :(";
}
```

עבור תיקייה שאנחנו יודעים מראש שהיא לא תקייבה (למשל, תיקייה בשם "מוזיקה" שאנחנו יודעים שאינה קיימת), נשתמש בקוד הבא:

```PHP
if (file_exists('מוזיקה/שירים')) {
  echo "אני אוהב לשמוע שירים!";
} else {
  echo "כן נכון, אין לי מוזיקה...";
}
```

יתרון נוסף כשמשתמשים ב-`file_exists()` הוא שאנחנו יכולים לציין את הכתובת המלאה של התיקייה, לא רק את שם התיקייה הנמצא בספרית הפרויקט שלנו.
