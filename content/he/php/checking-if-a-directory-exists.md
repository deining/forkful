## למה

בעולם של הפרמיטיביות והורסיות כמו דירקטוריות וקבצים, זה חשוב לוודא לפני שממשיכים בקוד שהדירקטוריה שניתן הכתובת קיימת וזמינה במערכת. פעולה פשוטה כמו בדיקת קיום של דירקטוריה או קובץ יכולה להציל מהמון בעיות כאשר נתקלים בקבצים חסרים או דירקטוריות שאינן קיימות.

## איך לבדוק אם דירקטוריה קיימת ב-PHP

בדיקת קיום של דירקטוריה ב-PHP נעשית באמצעות הפונקציה `file_exists()`. הפונקציה יכולה לקבל כפרמטר את הנתיב של הדירקטוריה או הקובץ שברצוננו לבדוק. הפונקציה תחזיר `true` אם הדירקטוריה או הקובץ קיים ו-`false` אם הוא לא קיים. כדי להיות בטוחים שהיא מחזירה את הערך המתאים, נשתמש בפקודת `echo` כדי להדפיס את הפלט.

```PHP
if (file_exists('path/to/directory')) {
  echo "הדירקטוריה קיימת";
} else {
  echo "הדירקטוריה לא קיימת";
}
```

אם הדירקטוריה קיימת, הפלט יהיה "הדירקטוריה קיימת". אם הדירקטוריה לא קיימת, הפלט יהיה "הדירקטוריה לא קיימת".
