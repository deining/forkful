## מדוע

למה שותפים רבים בעולם התכנות מעדיפים לקרוא טקסטים במקום לצפות בתמונות ווידאו? כי קריאת טקסט מאפשרת לך לתאר במדויק מה בדיוק אתה מבקש לעשות ולהתאים את הקוד בהתאם לצרכים שלך. כמו כן, זה מאפשר לך להכיר יותר טוב את השפה ולהפוך אותך למתכנת עדיף.

## איך לעשות זאת?

בהמשך הפוסט תלמדו כיצד לקרוא טקסט מקובץ באמצעות שפת PHP. נתחיל עם פקודת הלולאה `while` וניצור משתנה שיכיל את השורה הנוכחית של הטקסט. נרוץ על כל השורות בקובץ תוך כדי שנמיר את השורה למערך. לבסוף, נדפיס את כל הסטרינגים במערך בכדי להציג את כל הטקסט שנמצא בקובץ.

```PHP
$file = fopen("example.txt", "r"); // פתיחת קובץ לקריאה
$textArray = []; // יצירת מערך ריק לאיחסון הטקסט

while(!feof($file)) { // לרוץ בין השורות עד סוף הקובץ
  $line = fgets($file); // קורא שורה אחת
  array_push($textArray, $line); // מוסיף את השורה למערך
}

fclose($file); // סגירת הקובץ

echo implode("\n", $textArray); // מציג את הטקסט
```

כאשר תפעילו את הקוד הנ"ל, תקבלו את הטקסט שנמצא בקובץ "example.txt". במידה ואין לכם קובץ כזה, תוכלו ליצור אחד עם הטקסט המתאים לכם.
