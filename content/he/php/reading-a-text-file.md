# למה
 כמדוע לקרוא קובץ טקסט? הסיבה העיקרית היא כי בעזרת פונקציונליות זו, ניתן לקרוא נתונים מקובץ טקסט ולעבד אותם ישירות בקוד התוכנית שלנו. זה יכול להיות שימושי כאשר רוצים ליצור אפליקציות שמתבססות על נתונים שנקבלים מקובץ טקסט חיצוני.

# איך לעשות זאת
לפניכם דוגמה לקוד PHP שמציג את השימוש בפונקציית fopen לקריאת קובץ טקסט והדפסת הנתונים המצויים בו.

```PHP
<?php
$file = fopen("example.txt", "r"); // פתיחת קובץ טקסט במצב "קריאה"
while(!feof($file)){ // בודק אם הוגעת לסוף הקובץ
  echo fgets($file) . "<br>"; // קריאת שורה אחת בכל איטרציה והדפסתה
}
fclose($file); // סגירת הקובץ
?>
```

תוצאת הקוד תהיה:

```
כל הכבוד! אתם הצלחתם לקרוא קובץ טקסט בעזרת פונקציית fopen.
מצורפת תוכנית דוגמה נוספת שהיא לקריאה של קובץ טקסט צריכתי קוד לעבודד עם כל איטרציה של שורה בקובץ יש דברים רבים שאפשר לעשות עם הנתונים הללו, כגון להציגם באופן מובן למשתמש או לשמורם במשתנים להעברה נוספת בקוד שלנו.
```
