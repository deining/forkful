## מה ולמה?

קריאת קובץ טקסט היא פעולה נפוצה בתכנות בשפת PHP, שמאפשרת למתכנת לגשת ולקרוא את תוכן הקובץ המצוי במחשב או באינטרנט. מתכנתים משתמשים בפעולה זו כדי לקרוא ולעבד נתונים כמו טקסט, נתונים ממופשטים או נתונים קוביים.

## איך לקרוא קובץ טקסט ב-PHP?

לפניכם כמה דוגמאות של קוד פשוט ב-PHP שמציגות איך לקרוא קובץ טקסט ולהדפיס את התוכן שלו:

```
<?php 

// פתיחת הקובץ
$file = fopen("myfile.txt", "r");

// קריאת תוכן הקובץ והדפסתו
echo fread($file, filesize("myfile.txt"));

// סגירת הקובץ
fclose($file);

?>
```

אם יש צורך בקריאת קובץ טקסט בשפה אחרת מהנתמכת בברירת המחדל, ניתן להשתמש בפונקציה mb_convert_encoding כדי להתאים את קידוד הקובץ לקידוד הרצוי. לדוגמה:

```
<?php 

// פתיחת הקובץ עם הקידוד הרצוי
$file = fopen("myfile.txt", "r", null, "UTF-8");

// קריאת תוכן הקובץ והדפסתו
echo fread($file, filesize("myfile.txt"));

// סגירת הקובץ
fclose($file);

?>
```

## נכנסים לפרטים

בעבר, לקרוא קובץ טקסט ב-PHP היה נחשב מבנה דרך שמוכר למתכנתים חדשים. עם התפתחות השפה, הוספתה פונקציה נוחה יותר כדי לקשט את הקוד ולאפשר למתכנת לקרוא קובץ טקסט עם קלות.

כאלו שמעדיפים להשתמש בספרייה חיצונית, יכולים להשתמש בפונקציית file_get_contents שמציעה פתרון פשוט יותר לקריאת קובץ טקסט.

במקום לרשום את הקוד ידנית כמו בדוגמאות בעמוד זה, ניתן להשתמש בלולאת foreach כדי לעבור על כל התוכן של הקובץ ולעבד את הנתונים כראוי. לדוגמה:

```
<?php 

// פתיחת הקובץ
$file = fopen("myfile.txt", "r");

// לולאה על כל התוכן של הקובץ
foreach (file("myfile.txt") as $line) {

// הדפסת כל שורה בנפרד
echo $line;

}

// סגירת הקובץ
fclose($file);

?>
```
