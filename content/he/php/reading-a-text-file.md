## לפני למה
באמצעות תוכנית PHP ניתן לקרוא קבצי טקסט ולהשתמש בתוכן שלהם, זו כלי חיוני בכתיבת תוכניות ויישומים.

## איך לעשות זאת
```PHP
$file = fopen("textfile.txt", "r") or die("לא ניתן לפתוח את הקובץ.");
// קריאת קובץ בשורות יחידות
while(!feof($file)) {
  echo fgets($file) . "<br>";
}
// קריאת קובץ באופן מלא
echo fread($file,filesize("textfile.txt"));
```

### שימוש בפונקציית fopen
בפונקציית fopen של PHP ישנם שני פרמטרים חשובים. הפרמטר הראשון הוא נתיב הקובץ שבו נרצה לקרוא והפרמטר השני הוא אופן הפתיחה. במקרה שלנו, השתמשנו ב־"r" שמציינת פתיחה לקריאה בלבד. בנוסף, אם לא ניתן לפתוח את הקובץ, התוכנית תחזיר הודעת שגיאה עם עזרת ההורידה.

### שימוש בפונקציות fgets ו־fread
פונקציות אלו מאפשרות קריאת קובץ בשורות יחידות או באופן מלא. בכדי להשתמש בפונקציות אלו, יש תחילה לפתח את הקובץ עם fopen.

## חפירה עמוקה
קריאת קובץ טקסט היא חלק חשוב מתהליך התכנות ב־PHP. כדי להיות מקצוען אמיתי, כדאי ללמוד כיצד להשתמש בפונקציות קריאת קבצים המתקדמות יותר כמו fseek, fseek, rewind ו־fgets.
