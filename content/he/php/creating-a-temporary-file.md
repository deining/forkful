## למה

יצירת קובץ זמני בפחות חשוב בעולם התכנות, אך מדובר בכלי חיוני עבור מתכנתים רבים. קבצים זמניים משמשים כדי לאחסן מידע זמני, לבדוק פונקציונליות של קוד ולשמור עדכונים זמניים לפני הם יועברו לקוד ראשי.

## איך לבצע

```PHP
// נוצרת קובץ זמני ריק בתיקייה הנוכחית
$tmpFile = tmpfile();
echo gettype($tmpFile); // resource

// לכידת שגיאות במקרה ולא ניתן ליצור קובץ זמני
if (!$tmpFile) {
    die("לא ניתן ליצור קובץ זמני") 
}

// כל הפעולות עם הקובץ המהעתק יתבצעו כמתאים
fwrite($tmpFile, "תוכן קובץ זמני\n");
ftell($tmpFile); // 19

// כדי לקרוא את התוכן של הקובץ המקורי ניתן להשתמש בפונקציית rewind
rewind($tmpFile);
echo fgets($tmpFile); // תוכן קובץ זמני

// כדי למחוק את הקובץ המקורי כשהוא כבר לא נמצא בשימוש, ניתן להשתמש בפונקציית unlink
unlink($tmpFile);
```

### שגיאות נפוצות

במידה וניסית לפתוח קובץ זמני וקיבלת הודעת שגיאה כי יש קובץ בשם זהה, זה אומר שקיים כבר קובץ זמני עם השם הזה בתיקיית המערכת הזמנית שמכוון אליה. לכן, כדי לפתור את הבעיה ניתן להשתמש בפונקציית tempnam עם שם ייחודי כדי ליצור קובץ זמני חדש בתיקייה המערכתית הזמנית.

במקרה של שגיאות אחרות, כדאי לבדוק האם התיקייה המקורית שבה ניסינו ליצור את הקובץ זמני קיימת ונגישה.
