## למה

ברוב המקרים, יצירת קובץ זמני היא נחוצה במסגרת תהליכי תיכנות עם שפת PHP. הקובץ הזמני נועד לשמש כספריה שמאחסנת מידע זמני ומסייעת בביצוע טיפולים מסוימים כגון איתות תרשים או שמירת נתונים.

## איך ליצור קובץ זמני ב-PHP

כדי ליצור קובץ זמני בשפת PHP, ניתן להשתמש בפונקציה המובנית "tempnam()". בקוד הבא נדגים דוגמא ליצירת קובץ זמני בשם "temp_file" בתיקיית הקוד הנוכחית:

```PHP
$temp_file = tempnam(".", "temp_file");
// תצוגת שם הקובץ הזמני
echo $temp_file; // תעקיף ל- http://www.example.com/temp_file
```

הראשון, אנחנו מייצרים קובץ עם שם רנדומלי ויזהה לשם הקובץ הזמני. כמו כן, אפשרי להגדיר שם מזותר כדי לשפר את הזיהוי של הקובץ. ניתן להשתמש גם בספרייה שנמצאת מחוץ לתיקיית הקוד כדי ליצור קובץ זמני במקום אחר.

בנוסף, ניתן להתאים את הפונקציה "tempnam()" כדי ליצור קובץ זמני בדרך אחרת. לדוגמא, אם ברצונך להראות עם השתמשת בתיקייה אחת להפעלת התוכנה, אפשר להשתמש בקוד הבא:

```PHP
$temp_file = tempnam("/var/www/html/", "temp_file_");
// תצוגת שם הקובץ הזמני
echo $temp_file; // תעקיף ל- /var/www/html/temp_file_voyi
```
