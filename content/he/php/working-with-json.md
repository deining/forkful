## למה

JSON הוא פורמט נתונים נפוץ בתכנות והוא משמש להעברת נתונים בין שפות תכנות שונות ובין שרתים ולצורכים אחרים רבים. לעולם לא היה קל יותר לעבוד עם נתונים מבניים ולעבוד עם מודלי נתונים שונים ביחד. 

## איך לעשות זאת 

כדי לעבוד עם JSON בפיתוח ובתכנות יש לנו את פונקציות ה-JSON המוכנות לשימוש בכל שפת תכנות ומאפשרות לנו לקרוא, ליצור ולהתאים פךטים בין השפות. ניתן לעבור על כל המידע הקטן של JSON ולעבוד איתו בקלות. 

```php
// מתחילים ביצירת מערך
$users = [
    [
        'name' => 'יוחנן',
        'age' => 26,
        'city' => 'תל אביב'
    ],
    [
        'name' => 'מרים',
        'age' => 30,
        'city' => 'ירושלים'
    ],
    [
        'name' => 'נתן',
        'age' => 23,
        'city' => 'חיפה'
    ]
];

// ממירים את המערך לפורמט JSON
$json_users = json_encode($users);

// מדפיסים את התוצאה
echo $json_users;
```

הפלט המתקבל יהיה:
```json
[
    {
        "name": "יוחנן",
        "age": 26,
        "city": "תל אביב"
    },
    {
        "name": "מרים",
        "age": 30,
        "city": "ירושלים"
    },
    {
        "name": "נתן",
        "age": 23,
        "city": "חיפה"
    }
]
```

## נכנסים לעומק

JSON מכיל מבנה פשוט ומצוין לשימוש בייצוג נתונים. לכל משתנה יש סוג נתונים מסוים שמגדיר כיצד יוצג הנתון. לדוגמה, משתנה מספרי יוצג כמספר, משתנה מחרוזת יוצג כמחרוזת וכן הלאה.
