## למה
 זה חשוב לדעת כיצד לחשב תאריך בעתיד או בעבר כאשר מתכנתים ב-PHP. כלי זה מאפשר לנו לנהל תאריכים ולבנות יישומי זמן לפי הצורך. 

## כיצד לעשות זאת
לחילופין, נציג שתי דרכים שונות לחישוב תאריך בעתיד או בעבר באמצעות פונקציות מובנות ב-PHP. ניתן להתאים את הקוד לצרכים המדוייקים שלכם.

### פונקציית time()
הפונקציה time() מחזירה את הזמן הנוכחי בזמן שעבר מהשנה ה-1970. כדי לחשב תאריך בעתיד או בעבר, נוכל להוסיף או להוריד כמות מסוימת של שניות מתאריך הנוכחי, כך שנקבל תאריך אחר. לדוגמה:

```PHP
<?php
// חישוב תאריך חזור ימים אחד
$date = date('Y-m-d', strtotime('-1 day'));
echo $date; // יחזיר "2021-12-05"
```

```PHP
<?php
// חישוב תאריך קדימה עונה אחת
$date = date('Y-m-d', strtotime('+1 month'));
echo $date; // יחזיר "2022-02-06"
```

הפונקציה strtotime() מאפשרת לנו להחזיר תאריכים בשורה אחת נוחה, ובאופן כללי תאריכים לפי תבניות מוגדרות.

### פונקציית mktime()
הפונקציה mktime() מחשבת תאריך על בסיס משתנים מפורטים כמו שעה, דקות, יום, חודש ושנה. ניתן להשתמש בפונקציה כדי לחשב תאריך בעתיד או בעבר, בדיוק כפי שאנחנו צורכים. לדוגמה:

```PHP
<?php
// חישוב תאריך חזור שבוע אחד
$date = date('Y-m-d', mktime(0, 0, 0, date('m'), date('d')-7, date('Y')));
echo $date; // יחזיר "2021-12-04"
```

```PHP
<?php
// חישוב תאריך קדימה שנייה אחת
$date = date('Y-m-d', mktime(0, 0, 0, date('m')+1, date('d'), date('Y')));
echo $date; // יח
