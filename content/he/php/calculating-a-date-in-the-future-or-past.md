## למה

למה ייתכן שמישהו ירצה לחשב תאריך בעתיד או בעבר? אחד הדוגמאות הנפוצות היא בקשר לבניית אתר או אפליקציה שמשתמשת בתאריך כחלק מהתכנון או התצוגה שלה. ניתן להשתמש ב פי אץ פי הנוכחי כדי להציג את תאריך היום, אבל כדי להציג תאריך מפורט יותר מונגד, בדרך כלל נדרשת חישוב מתאים.

## איך לעשות זאת

כדי לחשב תאריך בעתיד או בעבר בפי אץ פי, ניתן להשתמש בפונקציות כמו `strtotime() ` ו- `date()` . הנה דוגמה פשוטה לחישוב תאריך של יומיים מאוחר יותר:

```PHP
$future_date = strtotime("+2 days"); // מחשבת תאריך של יומיים מאוחר יותר
echo date("Y-m-d", $future_date); // מדפיס את התאריך בפורמט שנבחר
```

תוצאה:

```
2020-09-20
```

ניתן להשתמש גם בפונקציה `mktime()` כדי לחשב תאריך מדויק יותר. לדוגמה, נחשב תאריך של שבועיים אחרי 1 ינואר 2021:

```PHP
$future_date = mktime(0, 0, 0, 1, 1, 2021) + (14 * 24 * 60 * 60);
echo date("Y-m-d", $future_date);
```

תוצאה:

```
2021-01-15
```

כפי שניתן לראות, מה נראה שסובלים מחיסרון מתאריכים בפי אץ פי. למרבה המזל, ישנן פונקציות נוספות שניתן להשתמש בהן כדי לחשב תאריך בעתיד או בעבר יותר בצורה קלה יותר.
