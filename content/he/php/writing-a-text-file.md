## למה

למה לכתוב קובץ טקסט בפי.פי? קיימים מספר סיבות למה תרצו לכתוב קובץ טקסט כחלק מתהליך התכנות. לדוגמה, כדי לשמור נתונים משמעותיים או ליצור קובץ HTML שיציג את המידע שתכתבו.

## איך

באמצעות פי.פי ניתן לכתוב קובץ טקסט באופן פשוט ויעיל. לדוגמה, ניתן להשתמש בפונקציה "fwrite" כדי לכתוב לקובץ, ולאחר מכן לסגור את הקובץ עם הפונקציה "fclose". הנה דוגמא לקוד פי.פי פשוט שיכתוב טקסט לקובץ בשם "example.txt":

```PHP
<?php
$myfile = fopen("example.txt", "w") or die("Unable to open file!");
$txt = "This is some sample text.";
fwrite($myfile, $txt);
fclose($myfile);
echo "The text has been written to the file.";
?>
```

פלט הקובץ ייראה כך:

```
This is some sample text.
```

כדי להוסיף טקסט נוסף לקובץ קיים, ניתן לשנות את הפרמטר הראשון של הפונקציה "fopen" למצב "a", שעובר מצב התווספות לקובץ במקום החלפתו.
