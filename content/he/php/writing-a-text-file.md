## למה לכתוב קובץ טקסט

כתיבת קבצי טקסט היא חלק חשוב מתהליך התכנות בשפת פי אצ' פי. קבצי טקסט מאפשרים למתכנת לאחסן ולקרוא מידע בקלות ולעבד אותו בקוד. במאמר זה, נדבר על הסיבות לכתוב קובץ טקסט ואיך לעשות זאת בפירוט.

## איך לכתוב קובץ טקסט

כדי לכתוב קובץ טקסט בפי אצ' פי, יש להשתמש בפונקציה "fopen" ליצירת קובץ חדש או לפתיחת קובץ קיים. לאחר מכן, ניתן להשתמש בפונקציות "fwrite" ו-"fclose" לכתיבה וסגירה של הקובץ. לדוגמה, ניתן להשתמש בקוד הבא כדי ליצור קובץ שלום עולם ולקרוא אותו:

```PHP
$file = fopen("hello.txt", "w"); // פתיחת הקובץ במצב כתיבה
$txt = "שלום עולם"; // טקסט לכתיבה לקובץ
fwrite($file, $txt); // כתיבת הטקסט לקובץ
fclose($file); // סגירת הקובץ

// קריאה מהקובץ שנוצר
$file = fopen("hello.txt", "r"); // פתיחת הקובץ במצב קריאה
$txt = fread($file, filesize("hello.txt")); // קריאת הטקסט מהקובץ
fclose($file); // סגירת הקובץ

echo $txt; // מדפיס את הטקסט שנמצא בקובץ
```

כמו כן, ניתן להשתמש בפונקציות נוספות כדי לעבד קבצי טקסט, כגון "fgets" לקריאת שורה אחת מהקובץ ו-"fgetcsv" לקריאת נתונים מקובץ בפורמט CSV.
