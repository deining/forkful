## למה

כתיבת קובץ טקסט ב-PHP היא חלק חשוב מהתרגום של קובץ טקסט לקוד, ועשויה להיות שימושית עבור מגוון רחב של מטרות. כתיבת קובץ טקסט ב-PHP נעשית הרבה יותר קל ומסודרת באמצעות כלי שפה זה, מה שהופך את כתיבת הקוד והניווט בו לפעולה נוחה ומהירה.

## כיצד לעשות זאת

קוד PHP נכתב באמצעות פינוק ייחודי המשתמש במשתנים של שפת התכנות. ישנם שני אופנים יסודיים ליצור פעולות ב-PHP עם קבצי טקסט. הראשון הוא על ידי כתיבת הקוד ישירות לתוך הקובץ ושמירה באמצעות פלט HTTP באמצעות הפקודה "fwrite()" המביאה באמצעות PHP בצורה אסינכרונית נתונים מלאים לתוך הקובץ הנתון לאורח HTTP נתון.

### דוגמאות קוד

**כתיבת נתונים לקובץ:**

```PHP
<?php 
$mytext = "כל הטקסט כאן"; 
$file = fopen("data.txt","w"); 
fwrite($file,$mytext); 
fclose($file); 
?>
```
**קריאת נתונים מקובץ:**

```PHP
<?php 
$file = fopen("data.txt","r"); 
echo fread($file,filesize("data.txt")); 
fclose($file); 
?>
```

### תוצאות פלט

התוכנית הראשונה תכתוב "כל הטקסט כאן" לקובץ data.txt. התוכנית השנייה תקרא את קובץ data.txt ותדפיס את תוכנו.
