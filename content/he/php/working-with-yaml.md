## למה

YAML הוא שפת תיבנות פשוטה וקלה לקריאה, וחשובה לכל בעלי קווצ'באק בקבצי תצורה. זה מאפשר לנו לקרוא ולכתוב מידע מתצורות בקלות יותר מאשר בשפות אחרות, ועבור מפתחים רבים זה הופך אותה לכייפית יותר לנהל תצורות מורכבות.

## איך לעבוד עם YAML

המאמר הדין ידני ומנוסח למטה מציג לכם תוך כדי ניסוח בית למשליי דוגמאות ב-PHP שתראו כיצד לגשת עם YAML לא רק כדי לקרוא אותו , אלא גם לתצורה של מידע בדרך הכי יעילה ומתודולוגית.

לפניכם יובא שני דוגמאות, אחת לקריאה ואחת לכתיבה:

```PHP
<?php
// טעינת הקובץ
$yaml = file_get_contents('config.yaml');
// המרה למערך אוביקטים
$config = yaml_parse($yaml);
// לולאה על המערך והדפסת המידע
foreach ($config as $key => $value) {
	echo "{$key}: {$value}";
}
?>
```

תצורה מתחילה בפונקציה של yaml_emit אבל בכדי לשמור אותה תצטרכו להתגבר על המערכיות ולערוך אותו באוביקט כי שכח ולהתעלמ מזה זה בעלות בלתי נתנינות.
