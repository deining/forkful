# מדוע

ישנם המון סיבות לתת קשיח בשטח אחד למלל. ייתכן שתרצו להדגיש משהו במיוחד בטקסט שלכם, ליצור אתגר לקוראים, או פשוט כי זה נראה יפה מעין.

# כיצד לעשות זאת

בקוד הבא אנו מוצאים את הפתרון הפשטני ביותר לקשת את הראש. אתז תצטרכו לשנות את שם המשתנה `$string` כדי לקשט את המחרוזת שלכם.

```PHP
$string = "שלום עולם";
echo strtoupper($string); // החזרה: שלום עולם
```

בתוכנית הבאה אנחנו מציגים כיצד אפשר להשתמש בפונקציות `mb_strtoupper()` ו- `mb_convert_case()` על מנת לקשט רצף מחרוזות.

```PHP
$string = "שלום עם יד ימנה";
echo mb_convert_case($string,MB_CASE_TITLE,'UTF-8'); // החזרה: שלום עם יד ימנה
```

## חקירה עמוקה

זה לא מספיק פשוט לקשט רצף מחרוזות ולעשות הכללת נקודות. כשמדובר בשפות שלכמו העברית והערבית, יישור נכון של אותיות ידכה רצף תווים שונים. בקיצור, זה מציג טעיה של אינדיבידואליות כאנשים משתמשים בתווי רישום.

אך איך ניתן לשפר אפיי תינגים כדי לאפשר קשיח של רצף מחרוזות משונות? אחד הדרכים הטובות ביותר לעזור בזה הוא להשתמש בפונקציות `recapitalize()` ו `normalizer_normalize()` המיועדות בקשיח מהיר לתשחזב את הנתונה של השפה שלכם כדי להפוך את המחרוזת לאחידות. למפרט את המאמר זה נוסיף מספר עוגיות כפולות בקוד:

```PHP
setlocale(LC_CTYPE, "en_US.utf8"); // הגדרת תרגום ל- UTF-8
$string = "שלום שלום שלום";
echo mb_convert_case($string, MB_CASE_TITLE); // החזר
