## למה

אף אחד לא אוהב לכתוב קוד מתקופ טקסט ארוך ומצומצם. אבל לעתים קרובות אנו נתקלים במצבים בהם אנו צריכים להפעיל פעולות מסוימות על מחרוזות בצורה מתוחכמת. למשל, נרצה לגשת אל מילה מסוימת במחרוזת, ליצור חלק מחדש או לרמז במחרוזת. במקרים אלו, הפעולה השכיחה היא למשוך תת-מחרוזת מהמחרוזת המקורית. אז למה אנחנו ממשיכים תת-מחרוזות? נקרא כעת ונריץ מספר דוגמאות כדי למצוא את התשובה.

## איך לעשות

תת-מחרוזת ניתן למשוך בצורה קלה ע"י שימוש בפונקציה substr () ב-PHP. הנה דוגמאות של כמה פעולות שניתן לבצע על מחרוזות באמצעות פונקציה זו:

```PHP
$text = "אני אוהב ללמוד PHP";
echo substr($text, 0, 4); // תת-מחרוזת תחילית ממיקום 0 עד 4 - אני
echo substr($text, -3); // תת-מחרוזת אחרונה מהמחרוזת המקורית - PHP
echo substr($text, 8); // תת-מחרוזת ממיקום 8 עד סוף - פHP
```

ניתן גם לשים לב שאם מציינים מיקום למעלה מאורך המחרוזת, התוצאה תהיה המחרוזת המלאה מהמיקום המבוקש עד סוף המחרוזת.

## ייצוג ציור

כמו שראינו, פונקצית substrate תומכת בכמה דרכים להחלץ תת-מחרוזות ממחרוזת מקורית. כעת נוכל לעבוד עם התת-מחרוזת בצורה שמתארת מצב תואם, נוכל לעבוד עם תת-מחרוזות ולהשתמש בהן לפעמים כדי לעזור לנו לשפר את קוד.
