## למה

הוצאת תת סטרינגים היא כלי חשוב בתכנות PHP שמאפשר לנו לגשת ולעבוד על חלקים ספציפיים של מחרוזות. זה יכול לעזור לנו להפוך את העבודה עם טקסט לקלה יותר ולחסוך זמן ומאמץ.

## כיצד לעשות זאת

```php
<?php
$string = "אני אוהב לתכנת ב-PHP";
 
// הוצאת תת הסטרינג מהתחלה עד לאורך 3 תווים
$substring = substr($string, 0, 3); 
 
// הדפסת התוצאה של הוצאת התת סטרינג
echo $substring; // ימתק
 
// הוצאת תת הסטרינג מהתחלה עד התווים שלפני המילה "ב-PHP"
$substring = substr($string, 0, strpos($string, "ב-PHP")); 
 
// הדפסת התוצאה של הוצאת התת סטרינג
echo $substring; // אני אוהב לתכנת
 
// הוצאת תת הסטרינג מאחורה לאורך 3 תווים
$substring = substr($string, -3); 
 
// הדפסת התוצאה של הוצאת התת סטרינג
echo $substring; // PHP
```

## Deep Dive

הוצאת תת סטרינגים ב-PHP נעשתה אפשרית בעזרת פונקציית substr(). הפונקציה מקבלת שלושה פרמטרים: מחרוזת הבסיס, נקודת התחלה ואורך התת סטרינג שרוצים להוציא. ניתן לסמן נקודת תחילת התת סטרינג על ידי ספרה חיובית בטווח האורך של המחרוזת הבסיס. כמו כן, ניתן לסמן נקודת תחילת התת סטרינג על ידי ספרה שלילית, כאשר -1 מסמל את התו האחרון במחרוזת.

הפונקציה strpos() משתמשת גם היא בשלושה פרמטרים כדי לאתר את מיקום התווים שנמצאים לפני המחרוזת שצוינה. זה מאפשר לנו להיכנס לתוך התחרות במחרוזות ולהוציא
