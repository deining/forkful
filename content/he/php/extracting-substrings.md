## Why
למה איש דואג להפוך את המחרוזת שלו/שלה לבלתי נגיש ולכן חייבים להחליק אותו? מתברר שישנם הרבה סיבות למה תרצה להשתמש בפונקציה של PHP שתאפשר לך להחליק תת מחרוזת.

לעתים, ייתכן שתרצה לקבל מידע ממחרוזת ארוכה יותר, כגון כתובת אימייל או מספר טלפון, אבל תרצה רק להשתמש במספר האחרון בתוך המחרוזת. או אולי תרצה להפוך את התאריך או הזמן לתת מחרוזות נפרדות כדי לארגן את המידע בצורה נכונה. בכל מקרה, ישנם הרבה סיטואציות שזו הפונקציה המתאימה ביותר.

## How To
אם אתה מעוניין להחליק תת מחרוזת ב-PHP, קל למדי לבצע. הנה כמה דוגמאות עם הסברים קצרים:

```PHP
// טקסט מלא
$text = "שלום חברים! אני מחכה למסיבת הלילה הקרובה";
// החלק הראשון של המחרוזת
echo substr($text, 0, 10); // יציאה: "שלום חברים"
```

בדוגמה הזאת, אנו מעבירים את המשתנה $ text לפונקציה substr ומציינים את התחילה (0) והסוף (10) של הטקסט שאנו רוצים להחליק. הפלט שלנו מציג תת מחרוזת של המילים הראשונות בטקסט המקורי.
