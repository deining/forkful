# מה זה ולמה?
התחשבנו במצב כזה: יש לנו מחרוזת ארוכה ואנו רוצים לקבל חלק מסוים שלה, או לפרטים יותר, נרצה לחלץ מחרוזת חלק מסוים. בעזרת פעולת חיתוך מחרוזת, המכונה גם חיתוך תת-מחרוזת, אנו יכולים לעשות זאת בקלות בקוד PHP. כך, אנו יכולים לקבל את המידע הרלוונטי לנו ולעבוד איתו בצורה נוחה ויעילה.

## איך לעשות?
כדי לחלץ תת-מחרוזת ממחרוזת נתונה, נשתמש בפונקציה substr() שמקבלת שני פרמטרים - את המחרוזת עצמה ואת התחלת האינדקס לחיתוך. לדוגמה, הנה קוד ב-PHP שמחצה תת-מחרוזת ממחרוזת נתונה:
```PHP
$string = "ספר הגיבורים";
$substring = substr($string, 3); // תת-מחרוזת תתחיל מהאות הרביעית
echo $substring; // ידפיס: הגיבורים
```

אם נרצה לחלק תת-מחרוזת של מחרוזת מסוימת עם מספר מוגדר של תווים, נוכל להשתמש בפרמטר של האורך כציון מתחילת החיתוך. לדוגמה, נרצה לחלץ את המילה "גיבורים" מהמחרוזת הנ"ל:
```PHP
$string = "ספר הגיבורים";
$substring = substr($string, 7, 5); // תת-מחרוזת תתחיל מהאות השבעית ותופיע ממנה חמישה תווים
echo $substring; // ידפיס: גיבורים
```

## טיול עמוק
בעצם, אנחנו רואים כי חיתוך מחרוזת הוא פעולה נפוצה ומבוססת שאנו משתמשים בה כמעט כל הזמן בקוד PHP. בעבר, לפני קיומן של הפונקציות כמו substr(), לקיחת תת-מחרוזת הייתה מתבצעת באמצעות פונקציות מובנות כמו strspn(), strpos() ו-strlen(). עם זאת, חיתוך מחרוזת באמצעות substr() מאפשר קריאה נקייה ופשוטה של הפונקציה, מה שהופך אותה לפתרון הנמצא כיום בשימוש נרחב.

ניתן לבצע חיתוך תת-מחרוזת גם באמצעות מוסכמות הזמנות (regular expressions) או באמצעות פונקציות נוספות כגון mb_substr() המתאימה למחרוזות בלתי תלויות בלשון. כמו כן, ניתן להשתמש בחיתוך מחרוזת לעיבוד מידע שנמצא בפורמט קבוע, כגון טקסט פורמט SD (Structured Data).
