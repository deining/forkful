## למה

תחילה, בואו נתנדב לחשוב רגע על תאריכים. הם מופיעים בכמעט כל מערכת שלנו - בין אם זה תאריך הלידה שלנו, תאריך האירוע החשוב שאנחנו אנחנו מתכננים או את תאריך העסקה האחרונה שלנו. אבל מה קורה כאשר אנחנו רוצים להשוות שני תאריכים זמנים? ומה אם אנחנו רוצים לדעת אם התאריך הראשון מגיע לפני או אחרי התאריך השני? כאן מתחיל צורך בהשוואת שני תאריכים בתוכנת PHP.

## כיצד לעשות זאת

כדי להשוות שני תאריכים ב-PHP, ניתן להשתמש בפונקציית `strtotime ()` כדי להמיר את התאריך לפורמט unix ולאחר מכן להשתמש בפונקציית `date ()` כדי להשוות את התאריך הממיר לתאריך אחר. לדוגמה:

```
<?php

// הגדרת שני תאריכים להשוואה
$date1 = "2020-07-01";
$date2 = "2020-08-01";

// המרת התאריך לפורמט unix
$date1_unix = strtotime($date1);
$date2_unix = strtotime($date2);

// השוואה בין שני התאריכים
if ($date1_unix < $date2_unix) {
    echo "$date1 קודם מ-$date2";
} elseif ($date1_unix > $date2_unix) {
    echo "$date1 אחרי מ-$date2";
} else {
    echo "שני התאריכים שווים";
}
?>
```

תוצאה:

```
2020-07-01 קודם מ-2020-08-01
```

לנוחיותכם, אפשר להשתמש גם בפונקציית `strtotime ()` לשימוש ישיר בתאריך בפורמט טקסט כדי להשוות בין שני תאריכים כפתרון יעיל ומהיר.
