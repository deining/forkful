## למה
השוואת שתי תאריכים היא כלי חשוב בתכנות PHP שיוכל לעזור לכם לבצע פעולות מגוונות, כגון יצירת דוחות או ניהול תאריכים במסדי נתונים. בכתבה הזאת נלמד כיצד לבצע השוואה בין שני תאריכים בקלות ובמהירות בעזרת PHP.

## איך לעשות זאת
בשביל לבצע השוואת תאריכים ב-PHP, נצטרך להשתמש בפונקציות המובנות של השפה, כגון `strtotime()` ו-`date()`. הנה דוגמא פשוטה של קוד שמשווה בין שני תאריכים ומדפיס את התאריך המאוחר יותר:

```PHP
$date1 = strtotime("9 September 2020");
$date2 = strtotime("21 September 2020");

if ($date1 > $date2) {
    echo date("d F Y", $date1);
} else {
    echo date("d F Y", $date2);
}
```

פלט:

```
21 September 2020
```

בדוגמא הזאת, אנחנו משווים בין שני תאריכים שנכתבים בתבנית סטרינג לידענו, ואז אנחנו משתמשים בפונקציות המובנות כדי להמיר אותם לפורמט של תאריך לאתרת ולבצע את השוואה ביניהם.

## נכנסים לעומק
התאריכים נמצאים בגבולות של תאריכי Unix, שהם מספרים שלמים המסמלים את מספר השניים מחודש 1970. בשביל להשוות בין שני תאריכים המצויים בפורמט זה, משתמשים בפונקציות המובנות של השפה כדי להמיר אותם למספרים כגון `strtotime` או `mktime`.

עוד דוגמא:

```PHP
$date1 = mktime(0, 0, 0, 9, 9, 2020);
$date2 = mktime(0, 0, 0, 9, 21, 2020);

if ($date1 > $date2) {
    echo date("d F Y", $date1);
} else {
    echo date("d F Y", $date2);
}
```

פלט:

```
21 September 2020
```
