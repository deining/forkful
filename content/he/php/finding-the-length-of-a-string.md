## מדוע

אחד הדברים הבסיסיים בתכנות בשפת PHP הוא מציאת אורך של מחרוזת (string). מדוע לנו לבחור לעסוק בנושא הזה? התשובה פשוטה - מחרוזת היא חלק חשוב מעצבת חלק מתכניות PHP רבות ולכן מציאת אורך שלה היא חיונית לכל מתכנת.

## כיצד לעשות זאת

הצעד הראשון במציאת אורך של מחרוזת הוא להשתמש בפונקציה המובנית `strlen()`. ניתן להעלות את המחרוזת שהנו הארגומנט שלה ותחזיר את האורך שלה כתוצאה. לדוגמה:

```PHP
$string = "Hello World";
echo strlen($string); // Output: 11
```

המרכיב המעניין בפונקציה זו הוא שהיא יודעת להתייחס לכל מיני תווים בכל השפות, לכן היא אחידה ומדויקת.

בנוסף, בחלק מהמקרים יתכן שנחפש את האורך של מחרוזת כדוגמת משתנה או ערך מערך משתנים בלולאות. בכאילו ניתן להשתמש בפונקציה `mb_strlen()` שתתייחס לתווים מקודדים בצורה נכונה. לדוגמה:

```PHP
$strings = array("זאב", "אריה", "נמר");

foreach ($strings as $string) {
	echo mb_strlen($string); // Output: 3, 3, 2
}
```

## לעזוב לעומק

כמו בכל פעם שעוסקים בחלק בסיסי בתכנות, תיכנות מתנדב יותר מיחשב מידע על הנושא. לכן, המשתמש בפונקציות `strlen()` ו-`mb_strlen()` צריך להבין את מהותם התחתונה.

בסופו של דבר, הם משתמשים בתחזית גודל המערך של התווים כדי למצוא את האורך של מחרוזת. פונקציית `strlen()` מסתמכת על רשת מערכת ההפעלה לחישוב הגודל, בעוד אותה מבצעת מידע במרקדה פנימית.
