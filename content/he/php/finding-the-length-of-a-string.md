## למה

אם אתה מפתח אתר או אפליקציה ב-PHP, ייתכן שתצטרך להכיר את הכלים הבסיסיים של שפת התכנות הזו. אחד הכלים החשובים הוא למצוא את אורך התווים במחרוזת. יכול להיות שיש לך צורך לבדוק את האורך של הקלט שמקבל הגולשים באתר, או לעבוד עם מידע מיועד לחיפוש לפי גודל.

## איך לעשות זאת

על מנת למצוא את האורך של מחרוזת ב-PHP ישנם כמה אפשרויות מנתחות. האפשרות הראשונה היא להשתמש בפונקציה המובנית `strlen`, שמחזירה את אורך התווים במחרוזת. לדוגמה:

```PHP
$string = "מחרוזת לדוגמה";
echo strlen($string);
```

פלט:

```
14
```

ניתן גם להשתמש בלולאת `for` כדי לעבור על כל תו במחרוזת ולספור את התווים בעזרת משתנה ספירה:

```PHP
$string = "מחרוזת לדוגמה";
$count = 0;

for ($i=0; $i < strlen($string); $i++) { 
    $count++;
}

echo $count;
```

פלט:

```
14
```
