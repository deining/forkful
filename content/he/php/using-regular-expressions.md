##למה:

רשימות רגילות הן כלי חשוב בתכנות בשפת PHP. באמצעותן, ניתן לחפש ולמצוא מילים, רצפים או תבניות בטקסט של לקוח ולבצע פעולות כמו חילופים, כיוונים ועוד. ניתן להשתמש בהם בתכנות אתרים, יישומי ווב ואפליקציות מורכבות נוספות. רשימות יכולות לכפות תנאים ולסדר נתונים, ולכן הן כלי חשוב אם ברצונכם לנהל את הטקסט במסווה מתאים.

##איך להשתמש:

רשימות ניתן לנסות באמצעות הפונקציה preg_match כדי לחפש אחד או יותר מרחבים בתוך מחרוזת. כדי למצוא מילים מסוימות, ניתן להשתמש בכמה תווים יחד עם כמה תווים יחד כל מקרה מילת match של הפונקציה. באמצעות כמה תווים, ניתן לבצע פעולת התאמה רגילה ולכן לאפשרות לציון האילו מילות מסוימות אתרים מתאימים לנתונים המבוקשים.

```PHP
//קוד דוגמה להשתמש ברגיש רגילות עבור ששתן אחת
$shesh = "GBFJ7850SJK";
if(preg_match("/GB[A-Z]{6}\d{4}[A-Z]{3}/",$shesh)){
  print "קידום השימוש ברשומה חדשה.";
} else {
  print "טוב מאוד.";
}
```

##Dive עמוק:

יתרונות שימוש מוצרים לפני  עיצוביים נמצאים באמצעות הרחבים המבקשים לחפש אתאריך שלו במקומות מובילים. השיטה הזו יעילה יותר מסינוני טמפלייטים עם מילים מסוימות בניגוד לדרך בה נתון העינון נרחיב מערכת עבודה הניהול שטרם הוספנו לתכנים שלנו.
