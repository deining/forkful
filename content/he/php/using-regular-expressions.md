## מה זה ולמה?
משתמשים בביטויים רגילים בתוכנות כדי לבדוק את התאימות של מחרוזות טקסט לתבניות מסוימות. פיתוחנים יכולים להשתמש בהם כדי לבצע פעולות כמו: בדיקת תוצאה מתאימה לתבנית, איתור תחביר מסוים, וחילוץ נתונים מהמחרוזת.

## איך לעשות?
כדי להשתמש בביטויים רגילים ב-PHP, יש להשתמש בפונקציית "preg_match" ולציין את התבנית שבה אתם מעוניינים לבדוק את התאימות לפני המחרוזת. ניתן להוסיף את הנומסים "/i" ו"/s" כדי להוריד את התאמת הרישיות ולכלול ירידה לשורה חדשה במחרוזת, בהתאמה.

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*לדוגמה:*

```PHP
$text = "היי, שלום לכולם!";
if(preg_match("/שלום/i", $text)) {
  echo "מילת 'שלום' נמצאת במחרוזת!";
} else {
  echo "מילת 'שלום' לא נמצאת במחרוזת.";
}
```
&nbsp;*&lt;output&gt;: "מילת 'שלום' נמצאת במחרוזת!"*

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;*לדוגמה מתקדמת:*

```PHP
$full_name = "John Smith";
if(preg_match("/([A-Z][a-z]+)\s([A-Z][a-z]+)/", $full_name, $matches)) {
  echo "$full_name כולל שם פרטי ושם משפחה!";
  echo "שם פרטי: " . $matches[1] . "<br/>";
  echo "שם משפחה: " . $matches[2];
} else {
  echo "לא נמצא שם מלא תקין!";
}
```
&nbsp;*&lt;output&gt;: "John Smith כולל שם פרטי ושם משפחה!<br/>שם פרטי: John<br/>שם משפחה: Smith"*

## משובץ
הביטויים הרגילים נוצרו בשנת 1951 על ידי פנחס זלרן כדי לטפל במחרוזות טקסט בפתרונות מתמטיים. עם הפיתוח של תכניות לשפות תכנות בתחילת שנות ה-60, הביטויים הרגילים הפכו למכשיר חשוב למטרות בדיקת תבנית, ומאז הם משמשים ככלי חיוני למתכנתים בכל שפת תכנות. בנוסף, ישנם מספר כלים וספריות חיצוניים שיכולים לסייע בתכנות עם הביטויים הרגילים, כמו סיימון קרוצ'ט ועבור PEAR.
