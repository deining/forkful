##למה: למה שמישהו יתעסק בהמרת תאריך למחרוזת?

לפני שניכנס לפרטים טכניים, עשוי להיות שאתם שואלים למה צריך להמיר תאריך למחרוזת בכלל. התשובה פשוטה - לטובת התצוגה. כשאנו מציגים תאריך באתר או באפליקציה, הוא נראה בדרך כלל כמחרוזת ולא כתאריך ממשי. בכדי לעשות זאת, אנחנו צריכים להמיר את התאריך למחרוזת.

##איך לעשות זאת:

המרת תאריך למחרוזת ב-Javascript פשוטה מאוד. אנחנו יכולים להשתמש בפונקציות המגיעות עם התאריך שורה לשורה או להשתמש בספריית תאריכים כמו Moment.js. לדוגמה, ננסה להמיר את התאריך הנוכחי למחרוזת בפורמט "DD/MM/YYYY":

```Javascript
var today = new Date(); // יוצר תאריך נוכחי
var stringDate = today.getDate() + "/" + today.getMonth() + "/" + today.getFullYear(); // ממיר את התאריך למחרוזת
console.log(stringDate); // פלט: "05/12/2020"
```

במקרה של מספרים קטנים מעשירים (כמו חודשים ויום החודש), נצטרך להוסיף 0 לפני המספרים בכדי לשמור על הפורמט. לדוגמה, במקום להשתמש ב- today.getMonth() נשתמש ב- (today.getMonth() + 1). כמו כן, אם נכתוב את התוכנית שלנו בספריית Moment.js, נוכל לקבל פלט בתצורה שלנו בצורה יותר נוחה.
