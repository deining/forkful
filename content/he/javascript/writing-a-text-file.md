## למה

הכתיבה של קובץ טקסט היא חשובה בייחוד בעולם התכנות, ובמיוחד בשפת ג'אווהסקריפט. באמצעות חלון הטקסט שלכם, תוכלו ליצור קוד ברור וקריא עבור כל מרכיב בפרויקט שלכם. כתיבת קבצי טקסט גם עוזרת לכתוב תיעוד ובדיקות במהירות וביעילות.

## איך לכתוב קובץ טקסט בג'אווהסקריפט

`` `javascript
// השתמשו בפקודת writeFile של ייבוא קובץ המשתמש fs
const fs = require ('fs');

// הגדירו את הנתיב לקובץ החדש שתרצו לכתוב
const filePath = "./newFile.txt";

// מידע לאכסנה בקובץ החדש
const fileData = "כתוב כאן את המחרוזת או המידע שברצונכם לכתוב בקובץ החדש";

// השתמשו בפקודת writeFile של מודול ה- fs כדי ליצור או לעדכן את הקובץ החדש
fs.writeFile (filePath, fileData, (err) => {
  if (err) throw err;
  
  // אם הפעולה הצליחה, תדפיס הודעת הצלחה
  console.log ('הקובץ החדש נוצר בהצלחה!');
});
`` `

כעת נציג דוגמאות לפלט של כתיבת קובץ טקסט בג'אווהסקריפט:

```
// קובץ טקסט חדש יוצר כאשר אתם מריצים את הקוד הנ"ל
// אם קיים כבר קובץ טקסט עם השם שנבחר, הוא יוחלף עם הנתונים החדשים שצוינו

כתוב כאן את המחרוזת או המידע שברצונכם לכתוב בקובץ החדש
```

```
// ייצוא קובץ טקסט עם תוכן חדש יוצר קובץ טקסט חדש
// אם כבר קיים קובץ טקסט עם אותו השם, הנתונים החדשים יתווספו לקובץ הקיים
