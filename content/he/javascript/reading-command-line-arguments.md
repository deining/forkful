## למה

קריאת ארגומנטים של שורת פקודה היא כלי עזר מאוד חשוב בפיתוח תוכניות ב-Javascript. היא אמורה לספק למפתחים יכולות ניתוח וטיפול בפרמטרים שיועברו לתוכנית באמצעות שורת הפקודה. קריאת ארגומנטים יכולה לקלקל את התוכנית שלך ולחסוך זמן יקר באיתור באגים.

## כיצד לעשות זאת

קריאת ארגומנטים של שורת הפקודה ב-Javascript נעשית באמצעות האובייקט המובנה `process.argv`. כדי להשתמש באובייקט זה, עלינו לכתוב את השורה הבאה בתחילת הקוד שלנו:

```Javascript
const args = process.argv.slice(2);
```

השורה הזו תיגרום לתוכנית שלנו לקרוא ולאחסן את כל הארגומנטים שהועברו לתוכנית במערך בשם `args`, ללא כולל את הארגומנט הראשון שהוא המפיצך של האובייקט `process.argv`. ניתן לגשת לארגומנטים השונים באמצעות אינדקסים כרגיל, כדי לעשות זאת נוכל להשתמש בפקודות אסימון לולאות כמו `for` או `forEach`. ניתן לשים לב שהארגומנט הראשון במערך תמיד הוא המפיצך של האובייקט ולכן עלינו להתחיל לאחריו את האינדקס הראשון הזמין כך שכל ארגומנט יהיה במקום הנכון.

כדי להדפיס את הארגומנטים למסך, נוכל לעשות זאת בעזרת לולאת `forEach` ולהשתמש בפונקציה `console.log` כך:

```Javascript
args.forEach((arg, index) => {
  console.log(`Argument ${index}: ${arg}`);
});
```

פלט ישיר מתוכנית זו תמיד ייכתב כך:

```
$ node program.js foo bar baz
Argument 0: foo
Argument 1: bar
Argument 2: baz
```
