# למה

למה אנחנו צריכים לקרוא ארגומנטים ממסך הפקודה ב-Javascript? בפוסט זה נכיר כמה יתרונות שיש לקרוא ארגומנטים ממסך הפקודה ונלמד כיצד לעשות זאת בקלות ב-Javascript.

# איך לעשות

כאשר אנו מריצים קוד ג'אווהסקריפט במסך הפקודה, יש לנו את האפשרות לקרוא ולהשתמש בארגומנטים שנמסרו לנו. נתחיל על ידי יצירת טקסט מחרוזת אשר מכילה את הארגומנטים המועברים למסך הפקודה:

```Javascript
let args = process.argv.slice(2);
```

בשורת הקוד הנ"ל, אנו משתמשים בפונקציית `process.argv` כדי לקרוא את כל הארגומנטים המועברים לקוד. נשתמש בפונקציית `slice()` כדי לסנן את הארגומנטים שאנחנו רוצים להשתמש בהם. לדוגמה, אם נרצה לקרוא את הפרמטר השני, נשתמש בפונקציית `slice(2)` שתחזיר לנו רק את הארגומנט השני ועד הסוף.

כעת נדפיס את התוצאה של הארגומנט המבוקש עם `console.log()`:

```Javascript
console.log(args[1]);
```

הפלט יהיה: `Hello`

למעשה, ניתן להשתמש בפונקציה `forEach()` כדי לעבור על כל הארגומנטים שנמסרו לנו ולהדפיס אותם באופן נקי יותר:

```Javascript
args.forEach((arg) => {
  console.log(arg);
});

```

הפלט יהיה:

```
Hello
World
```
