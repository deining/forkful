## למה
פקודת השורת פקודה היא כלי עוצמתי בעבודת התכנות. הכוח הגדול ביותר שלה הינו יכולתה לקבל פרמטרים מפעולת הקריאה שלה. קריאת פרמטרים מפעולת הקריאה יכולה להרחיב את אפשרויות התכנות ולהקל על הפיתוח של יישומים בעזרת פקודת השורת פקודה.

## איך לעשות זאת
האם ידעת שניתן לקרוא פרמטרים כמו תאוריתמן מפעולת הקריאה בפקודת השורת פקודה בג'אווהסקריפט? באמצעות פקודת הקריאה בג'אווהסקריפט, תוכל לקבל את הפרמטרים המועברים מתוך הפקודה המורכבת שנכתבת על ידי המשתמש. נתחיל בדוגמה תיאורטית ונראה איך לקרוא פרמטרים בפקודת השורת פקודה בג'אווהסקריפט!

```Javascript
// דוגמה תיאורטית של פקודת הקריאה
// node הריצו עם פקודת הקריאה בפקודת השורת פקודה בעזרת פקודת לנק ואת הקוד הבא

process.argv.forEach((val, index) => {
  console.log(`${index}: ${val}`);
});
```

זה מה שאתה תקבל כתוצאה:

```bash
node ./app.js hello world
```

```bash
0: /usr/local/bin/node
1: /Users/username/path/to/your/app.js
2: hello
3: world
```

כפי שאתה רואה, הירשמו את פרמטרי הפקודה והתאים להם באופן נמען למערך של process.argv. כעת נשים לב כיצד הריצה של פקודת השורת פקודה משביעה את הצורך שלנו לקרוא את פרמטרי הפקודה שלה.
