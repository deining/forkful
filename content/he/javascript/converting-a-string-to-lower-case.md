# למה

מיטב הפרקטיקות בתכנות המודרני היא לעדכן ולשמור על תוויות באותיות קטנות (lower case). כדי לעדכן שורות בלולאות, או להשוות ערכים התווים במהלך התכנות, חשוב להפעיל פעולות טקסט על מחרוזות בתוך רצף הקוד. אחת הפעולות הקלות לביצוע היא גירוד מחרוזת (converting) למצב תווים קטנים ע"י שימוש בפונקציות שבשפה כמו *toLowerCase()*.

# איך

השתמשו בקוד הבא כדי להמיר מחרוזת לאותיות קטנות בשפת ג'אווהסקריפט:

```
let string = "HELLO WORLD";
console.log(string.toLowerCase());
```

פלט: `hello world`

בדוגמה הנ"ל, השתמשנו בפונקציה *toLowerCase()* שבתוך הנתון שלנו *string*. הפונקציה ממירה את כל התווים במחרוזת לאותיות קטנות.

המרה של מחרוזת לאותיות קטנות מאפשרת לנו לבצע השוואות או מחרוזות שונות תוך התייחסות לתווים באותיות קטנות, וכן להתאים לקבלת הבטיחות כסטנדרט מוכר שכזה כמו לולאות. פייתון (Python) ודארט (Dart) הם כמה שפות בעת הימים כמו את ג'אווהסקריפט, ואף שזה לא הנתון, הפיכת אותיות לקטנות מהווה בסיס של התייחסות למחרוזות בהם נפתחים משתנים שיכולים לבעוט לנו בת'רפים.
