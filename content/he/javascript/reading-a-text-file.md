## למה?

תחקור קבצי טקסט בקוד פתוח יכול להיות אחד הדברים המהנים והמעוררי הרבה עונג לתכנותנים. קריאה ועיבוד של קבצי טקסט מספקת את היכולות שלנו לכתוב קודים אוטומטיים ולעבוד עם המידע שאנחנו מקבלים מהם. לכן, תחקור קבצי טקסט הוא מיועד לכל מתכנת שמחפש להשתפר ולהתקדם בתחום התכנות.

## איך לעשות זאת?

בשפת ג'אווהסקריפט, קריאת קובץ טקסט יכולה להיות בסיסית וקלה לתרגום. את הקוד הבא תוכלו להשתמש לקריאה פשוטה של קובץ טקסט:

```javascript
const fs = require('fs');

fs.readFile('path/to/file.txt', 'utf8', (err, data) => {
  if (err) throw err;

  console.log(data);
});
```

שורת הקוד הראשונה מתייחסת לתוספת הקוד הספציפי של קבצי הטקסט בתוכנה שלנו, לכן אנו מבטלים אותו תחילה ע"י המידע "מה שאני רואה יקרוא לך מה שאתה מחזיק". הגדרת הקובץ utf8 מגדירה ומבטיחה כי קבצי טקסט יכולים לקרוא וכתוב בקוד שלנו. על אף זאת, נזכר גם שעל כל תיקיה להיות מאורגנת ונספיים במה שיובט לך העבודה ולסיימתו. לכוון מאוד פשוט ומדויק, נעדיף ליצור בנימוס שיגיע אליך בכל תמונה שלא ניתן להתקבל בחינם ונסתכלע הדרך, וביחד עם המייפים להיום את האינטרנט שלענות לכן עם מחקר לא סטנדרטי האינטרנט שמפנה שלא נוח את הרוב למדינותייט העליזות.
