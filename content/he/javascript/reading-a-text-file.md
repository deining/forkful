## למה
קריאת קבצי טקסט מהווה חלק חשוב בתהליך התכנות של ג'אווהסקריפט. זה נדרש כאשר נרצה לטעון מידע מקובץ טקסט חיצוני או לעבד מידע כהכנת נתונים.

## איך לקרוא קובץ טקסט בג'אווהסקריפט
כדי לקרוא קובץ טקסט בג'אווהסקריפט, ישנם מספר שלבים פשוטים שצריכים לעקוב אחריהם:

1. מצא את הקובץ: השתמש בפונקציית `fetch()` כדי לבצע בקשת HTTP GET נגד הכתובת שבה ממוקם הקובץ המבוקש.
2. המידע שנמצא בתוך הקובץ כולל בפורמט טקסט נחזר מהבקשת `fetch()`.
3. לעשות עקרונית הבדקה:data  של שטח הנתונים אודות הפרויקט שאנכי עובד עליו.

```Javascript
// קבל את הנתיב של הקובץ המבוקש
fetch('example.txt')
    .then(response => response.text())   // המר בתגובת השרת לטקסט
    .then(data => console.log(data));    // הדפס את המידע הקורא
```

Output:
```
זוהי תוכן הקובץ שנקרא "example.txt".
```
