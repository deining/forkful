## למה

רק 1-2 משפטים המסבירים *למה* מישהו יחבר לקרוא קובץ טקסט.

קריאת קובץ טקסט היא פעולה שימושית למתכנתים שרוצים לעבוד עם מידע מתוקדם בפורמט טקסט. במאמר זה, אני אחלוק איתכם כיצד לקרוא קובץ טקסט באמצעות קוד JavaScript ואסביר מה הדברים העמוקים יותר לגבי הפעולה הזו.

## איך לעשות את זה

קוד JavaScript זה יכול לקרוא קבצים טקסט באמצעות שימוש בפונקציות כמו "fs.readFileSync" או "fs.readFile". הנה דוגמא של קוד שמאפשר לנו לקרוא קובץ טקסט ולהדפיס את התוכן שלו בקונסול הפלט:

```javascript
// קרא קובץ טקסט והדפס את התוכן שלו בקונסול הפלט
const fs = require('fs');

// הפעולה הנוחה יותר היא להשתמש בפונקציית "fs.readFileSync"
// שתחזיר את תוכן הקובץ כמחרוזת
const content = fs.readFileSync('myfile.txt', 'utf8');

// הדפס את התוכן של הקובץ
console.log(content);
```

התוכן של הקובץ "myfile.txt" יידפס בקונסול הפלט.

כמו כן, ניתן לעבוד עם קבצים טקסט באמצעות פונקציות נוספות כמו "fs.readFile" אשר מחזירה את התוכן של הקובץ כאובייקט BUFFER המאפשר עבודה עם נתונים מסוגים שונים, או "fs.createReadStream" אשר מאפשרת קריאה שורה-שורה של הקובץ.
