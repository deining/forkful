## מה ולמה?
קריאת קובץ טקסט היא פעולה שמאפשרת למתכנתים לקרוא ולעבוד עם תוכן שנמצא בתוך קובץ טקסט מסוים. מתכנתים משתמשים בפעולה זו כדי לקרוא נתונים מקבצי מידע, כגון קבצי טקסט, קובצי נתונים או קבצי קוד. קריאת קובץ טקסט מאפשרת למתכנתים להבין ולעבוד עם המידע בצורה יעילה וקלה.

## כיצד לעשות זאת:
```Javascript
// כדי לקרוא קובץ טקסט, נשתמש בפונקציה fs.readFile
const fs = require('fs');

// הפונקציה מקבלת שני פרמטרים - שם הקובץ ופונקציה לטיפול בנתונים
// לאחר שהקובץ נטען, הנתונים יימסרו לפונקציה בתור פרמטר ראשון
fs.readFile('myfile.txt', (err, data) => {
    if (err) throw err;
    console.log(data); 
    // יוחזרו לנו הנתונים שנמצאים בקובץ בצורת בתים של Buffer
});
```
## מעמקים:
קריאת קובץ טקסט היא פעולה נפוצה וחשובה למתכנתים. בעבר, לפני קבלת תקני ה-ECMAScript, השפת ג'אווהסקריפט אינה תומכת ישירות בפעולה זו. במקום זאת, מתכנתים היו נאלצים להשתמש במכשירים כמו XMLHttpRequest או ActiveXObject על מנת לקרוא ולעבד נתונים מקובץ טקסט. אך עם התקני ה-ECMAScript החדשים, מתכנתים יכולים להשתמש בפעולות המובנות לקריאה וכתיבה של קבצי טקסט, על ידי שימוש בספריית ה- fs בג'אווהסקריפט.
