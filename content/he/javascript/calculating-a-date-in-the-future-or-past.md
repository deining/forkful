## למה

ניתן להשתמש בתכנות JavaScript כדי לחשב תאריך בעתיד או בעבר. זה יכול להיות שימושי למגוון רחב של דברים, כגון יצירת לוח זמנים, התאריך הצפוי של יום ההולדת של מישהו או פשוט כדי לדעת מתי יהיה החג הבא. במאמר זה, נלמד כיצד ליצור פונקציה פשוטה ב-JavaScript שתאפשר לנו לחשב תאריך בעתיד או בעבר.

## איך לעשות זאת

כדי לחשב תאריך בעתיד או בעבר באמצעות תכנות JavaScript, נצטרך להשתמש באובייקט המובנה "תאריך" ובפעולות שלו. הנה דוגמה לכיצד ניתן לחשב תאריך של 100 ימים בעתיד מהיום הנוכחי:

```javascript
let today = new Date(); // יצירת אובייקט התאריך של היום הנוכחי
let futureDate = new Date(); // יצירת אובייקט התאריך של התאריך בעתיד
futureDate.setDate(today.getDate() + 100); // נוסיף 100 ימים לתאריך הנוכחי
console.log(futureDate); // תאריך של 100 ימים בעתיד מהיום הנוכחי
```

אם נרצה לחשב תאריך בעבר באותו האופן, נוסיף פשוט את מספר הימים הנדרשים לתאריך מעבר לתאריך הנוכחי:

```javascript
let pastDate = new Date(); // יצירת אובייקט התאריך של היום הנוכחי
pastDate.setDate(today.getDate() - 30); // מחיקת 30 ימים מהיום הנוכחי
console.log(pastDate); // תאריך של יום לפני 30 ימים מהיום הנוכחי
```

ניתן לראות שהשימוש בפעולות של אובייקט התאריך מאפשר לנו להוסיף או לחסר ימים מתאריך נתון.
