## למה

חישוב תאריך בעתיד או בעבר נחשב לכלי שימושי כאשר מתעסקים במספר רב של פרויקטים תכנותיים. למשל, כאשר עובדים עם מרכיב זמן באינטרנט או כאשר מתכנתים אפליקציות לתאריכים ספציפיים. חישוב תאריך מעניין מתכנתים מכיוון שניתן להפליא באמצעות קוד להתאים תאריכים לצרכים מגוונים.

## איך לעשות זאת

תחילה, נצטרך ליצור אובייקט של התאריך הנוכחי ע"י קריאה לפונקציית Date המובנית. לדוגמה:

```Javascript
let today = new Date();
```

כדי לחשב תאריך בעתיד או בעבר, נוכל להשתמש בפונקציות המובנות המאפשרות להוסיף או לחסור יחידות זמן מהתאריך הנוכחי. לדוגמה, אם נרצה לחשב את התאריך של 7 ימים לפני היום, נוכל להשתמש בפונקציה המובנית setDate כדי להגדיר את היום החדש בעזרת מספר הימים הרצוי:

```Javascript
let oneWeekAgo = new Date();
oneWeekAgo.setDate(oneWeekAgo.getDate() - 7);
```

בנוסף, ניתן להשתמש בפונקציות המובנות המאפשרות לקבל את התאריך לפי יחידות זמן ספציפיות, כגון ימים, שעות או דקות. לדוגמה, אם נרצה לחשב את התאריך של 3 חודשים לאחור, נוכל להשתמש בפונקציה המובנית setMonth כדי להגדיר את החודש החדש בעזרת מספר החודשים הרצוי:

```Javascript
let threeMonthsAgo = new Date();
threeMonthsAgo.setMonth(threeMonthsAgo.getMonth() - 3);
```

בכדי להציג את התאריך בפורמט מסודר יותר, ניתן להשתמש בפונקציות המובנות המאפשרות לקבל את התאריך על פי יחיד
