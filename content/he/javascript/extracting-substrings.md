## למה

כתיבת תכניות ב-Javascript יכול להיות כיף ומאתגר בו זמנית. לרוב, אנו כותבים תכניות שמתייחסות למחרוזות טקסט. אחת התכונות החשובות ביותר של מחרוזת טקסט היא היכולת להפעיל תמחור חלקי על הטקסט על מנת ליצור מחרוזות חדשות מהטקסט המקורי. בשפת תכנות כמו Javascript, ניתן להשתמש בפעולת תת מחרוזת כדי ליצור מחרוזת חדשה המתאימה לצורך המיוחד שלנו.

## איך לעשות זאת

יצירת תת מחרוזת ב-Javascript היא פעולה פשוטה יחסית. פעולת תת המחרוזת נעשית באמצעות שימוש בתכונת ה-`substring` של אובייקט המחרוזת. הנה דוגמאות קוד ופלט של כמה אפשרויות ליצירת תת מחרוזת ב-Javascript:

```Javascript 
// יצירת מחרוזת ריקה עם אורך אפס
var emptyString = "";
var newSubstring = emptyString.substring(0, 2);
console.log(newSubstring); // פלט: r
```

```Javascript 
// יצירת מחרוזת עם טקסט 
var string = "כתיבת תכניות ב-Javascript";
var newSubstring = string.substring(5, 15);
console.log(newSubstring); // פלט: תכניות
```

הפעולה `substring` מקבלת שני משתנים: את המיקום הראשון של התת מחרוזת ואת מספר התווים שהיא צריכה להסתיים. בדוגמאות הקוד שנתנו מעל, אנחנו מחלצים תת מחרוזת של שני תווים ושל עשרה תווים לפי ההמקום הראשון והמספר שנמסרו לפעולה `substring`.
