# למה

למה לעסוק בחילוץ מחרוזות מתוך קוד Javascript? מחרוזות הן חלק חיוני מכל קוד, ולכן ייתכן שתצטרכו לחלץ מחרוזות מתוך ערכים אחרים בכדי לבנות קובץ טקסט, לחשב ערך ממוצע או לבדוק אם המחרוזת מתאימה לתבנית מסוימת.

# איך לעשות זאת

הנה שתי דרכים פשוטות לחלץ מחרוזות מתוך קוד Javascript:

<details>
  <summary>שיטה א': שימוש במתודת substring()</summary>

  ```Javascript
  let string = "שלום עולם!";
  let substring = string.substring(6); // Output: עולם!
  ```
  
  בשיטה זו, אתם משתמשים במתודה substring() כדי לחלץ מחרוזת מהמחרוזת המקורית. המתודה מקבלת שתי ארגומנטים: האינדקס של התו הראשון שבו אתם רוצים לחלץ את התת-מחרוזת והאורך של התת-מחרוזת שאתם רוצים לחלץ. בדוגמה שלנו, אנחנו מחלצים מחרוזת מהאינדקס השישי (כן, אנחנו מתחילים לספור מ-0 ב-Javascript) ועד סוף המחרוזת.

  אם אתם רוצים לחלץ מחרוזת מתוך מקור טקסט שונה או לחשב את האורך של התמליל המחולץ, אתם יכולים להשתמש במצב המתאים של המתודה substring() שמקבלת שני ארגומנטים.
</details>

<details>
  <summary>שיטה ב': שימוש במתודת split()</summary>

  ```Javascript
  let string = "אלופים, אלופים, אהה אלופים";
  let substring = string.split(", "); // Output: ["אלופים", "אלופים", "אהה אלופים"]
  ```

  בשיטה זו, אתם משתמשים במתודה split() כדי לפצל את המחרוזת המקורית לתת-מחרוזות עפ"י אותיות מסוימות או תבנית מסוימת. במקרה שלנו, אנחנו מפצלים את המחרוזת לתת-מחרוזות כאשר המחרוזת תחל
