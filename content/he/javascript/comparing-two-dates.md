# למה

בתחום התכנות, עיבוד תאריכים הוא חלק חשוב ביותר של כל יישום. לכן, לדעת כיצד להשתמש בקוד כדי להשוות בין שני תאריכים הוא מאוד חשוב כדי לבנות יישומים איכותיים ויציבים.

# כיצד לעשות זאת

לפניכם דוגמאות של כיצד להשוות בין שני תאריכים בקוד Javascript:

```Javascript
// יצירת שני אובייקטי תאריך
var date1 = new Date(2020, 1, 10);
var date2 = new Date(2021, 2, 15);

// השוואת תאריכים על פי מה אחד קודם
if (date1 < date2) {
  console.log("תאריך 1 מוקדם יותר מתאריך 2");
} else if (date1 > date2) {
  console.log("תאריך 2 מוקדם יותר מתאריך 1");
} else {
  console.log("שני התאריכים שווים");
}
// פלט: תאריך 1 מוקדם יותר מתאריך 2
```

אם אנחנו רוצים להשוות את התאריכים על פי חודש, יום או שנה ניתן להשתמש בפינוקציות כגון `getMonth()`, `getDate()` ו- `getFullYear()` כדי להשוות שני אובייקטי התאריך.

```Javascript
// השוואת חודשים
var month1 = date1.getMonth();
var month2 = date2.getMonth();

if (month1 === month2) {
  console.log("שני התאריכים באותו חודש");
} else {
  console.log("שני התאריכים אינם באותו חודש");
}
// פלט: שני התאריכים באותו חודש
```

# Deep Dive

כאשר אנחנו משווים בין שני תאריכים, אנחנו משווים בין מספרים המייצגים את התאריך בפועל. כל אובייקט תאריך ב-Javascript מייצגת את התאריך בפורמט של מספר שניות מאז ה-epoch date (1 בינואר 1970). לכן, כאשר אנחנו משווים בין שני אובייקטים תאריך, פועלת החישבון הבא: `(זמן בשניות מאז ה-epoch date) / 1000`.
