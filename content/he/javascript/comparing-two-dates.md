# למה

כשמפתחים אפליקציות או אתרים ווב, ייתכן שתצטרכו להשתמש בתאריכים כחלק מתהליך התכנות. כתרגום ללשון תכנות, תאריך הוא סוג משתנה מיוחד, שמכיל מידע על תאריך ושעה. אם אתם מעוניינים להשוות שני תאריכים ולבדוק אם הם זהים או אם אחד מהם מגיע לפני השני, אתם מחפשים אופציות להשוואת תאריכים ב-Javascript. בכתבה הזו, נלמד כיצד לבצע השוואה בין שני תאריכים באמצעות קוד Javascript פשוט ונחמד. 

## איך לעשות את זה?

בדוגמאות הבאות, נשתמש בפונקציה המובנית בשם 'Date' של Javascript כדי ליצור את התאריכים המושווים. נניח שנרצה לבדוק אם שני תאריכים זהים. הקוד הבא מראה איך להשתמש בפונקציות 'Date' ו-'getTime', כדי להשוות בין שני תאריכים: 

```Javascript
let date1 = new Date('June 17, 2021'); // תאריך ראשון
let date2 = new Date('June 17, 2021'); // תאריך שני
 
if(date1.getTime() === date2.getTime()){ // לבדוק אם שני התאריכים זהים
    console.log('התאריכים שווים.'); 
}else{
    console.log('התאריכים אינם שווים.'); 
}
```

התוצאה שתידפיס הקוד הזה היא 'התאריכים שווים.' כיוון שנתארך מספרי המייצג את שני התאריכים זהה. אם נשנה את התאריך הראשון בקוד ל-'June 18, 2021' למשל, התוצאה תהיה 'התאריכים אינם שווים.'
