## מדוע

כתיבת תוכניות ב-JavaScript היא כיף ומאתגר כאחד, ולפעמים ייתכן שתרצו לשנות את כתיבת המחרוזת שלכם כדי להראות יותר מושפעים או למנוע באגים פוטנציאליים. במאמר זה נלמד איך להפוך את האות הראשונה של כל מחרוזת במסך לאות גדולה באמצעות JavaScript.

## איך לעשות זאת

באמצעות מתודת `toUpperCase()` נוכל להפוך את האות הראשונה של כל מחרוזת לאות גדולה. בקוד הבא, `name` הוא משתנה שמכיל את המחרוזת שנרצה לשנות את הכתיב שלה.

```Javascript
let name = "john";
console.log(name[0].toUpperCase() + name.slice(1));
// Output: John
```

הפעולה הראשונה בקוד היא להשתמש במתודת `toUpperCase()` כדי להפוך את האות הראשונה של המחרוזת לאות גדולה. לאחר מכן, אנו משתמשים במתודת `slice()` כדי להחליף את התו הראשון מהמחרוזת המקורית עם התו הגדול. כך, במקרה שלנו, התוצאה הסופית תהיה "John".

כדי לחבר את התוצאה למחרוזת המקורית, אנו משתמשים בסימן "+" בין התוצאה החדשה והמחרוזת המקורית. התוצאה השלמה היא מחרוזת חדשה עם האות הראשונה בכתיב גדול.

## חקר מעמיק

ניתן להשתמש במתודות נוספות עם הפונקציה הזו כדי לשנות את הכתיב של מחרוזות. לדוגמה, נוכל להשתמש במתודה `toLowerCase()` כדי להפוך את המחרוזת לכתיב נמוך. ניתן גם להשתמש במתודה `split()` כדי להפריד את המחרוזת למערך, לעבוד על האיברים בכל אחד לבד ולחבר אותם חזרה למחרוזת חדשה.
