כתיבת תחביר צר עבור קוראים עבריים:

## מה ולמה?

תחביר צר הוא תהליך שבו משתארת מחרוזת כדי להיות ראויה הרבה יותר. כשממלאים את המחרוזת שלנו עם תווים גדולים, הוא מכפיל את המחרוזת עם ערכים המיוחדים בתווים גדולים במקום של תווים קטנים. תכף תתברבר לאך כי המראה שייך לקיימי תחבירות תפקיד. צריך ואולם למחשבות כי הוא כללו פעולות להחייאה.

## איך?

```Javascript
console.log("hello world".toUpperCase()); // שם חבר אנו רואים המתן /p p>
// נשים לב שמחרוזת המחרוזת האנגלית הופכת להיות מחרוזת גדולה המוסיפה כניטת למחרוזת הגדולה.
```
