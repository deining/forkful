## למה

אם אתה משתמש בשפת תכנות ג'אווהסקריפט, אתה כנראה מכיר את הפקודה console.log (). אז למה אנחנו מדפיסים קלט עגול? התשובה הפשוטה היא שהדפדפנים ישרטו קלט מתחת לאובייקט תחת החלון של אלה שמסתכלים על האתר שלך. זה בדיוק מציג את הקלטים שלך ועוזר לך לזהות שיגובנים בקלות ית ובאיזה שורה הם קרו.

## כיצד לעשות זאת

ראשית, כדאי לוודא שאתה מוכן לעבודה עם מספר קנסולות. התחל בהתקנת כרום לקבל קנסולת; כרום הוא הדפדפן הנפוץ ביותר וקנסולת הוא אחד מהשירותים שלו. הנה קוד יסוד שאתה יכול להדפיס בכותרת במספר קפיצות של שורות.

```Javascript
function printName() {
  console.log("שלום! זה הכותרת הבאה שלי."); 
}
```
  #### שמירה על הלוג, כמו לדוגמא כאשר אתה מחפש נסתר ביותר גבוה בכותרת, כגון: 
  ```Javascript
  function findHighestNumber() {
  console.log("מוצא את המספר הגבוה ביותר!"); 
}
  ```
  במקרה זה, אין צורך לבצע כל פעולה, כמובן כי אין צורך להדפיס את התוצאה, היא פשוח וכמובן תענות לכל הפרומת. לדוגמא, כאשר אתה מנסה להציג באופן יפה באמצעות הדפדפנים את התוצאות, כגון מידע על משתמשים מסויימים, תוכל להדפיס את הכתובת המלאה של התוצאות בעזרת הפקודה console.log(). זה בדיוק מה שאנחנו רוצים לעשות כאן.
