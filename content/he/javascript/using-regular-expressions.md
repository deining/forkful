## מדוע

כדי לכתוב קוד בצורה יעילה וקלה לתחזוקה, יש להשתמש בביטויים רגולריים בתכנות ג'אווהסקריפט. הביטויים הרגולריים מאפשרים לנו למצוא ולשנות טקסט בקלות, ללא צורך בכתיבה של קוד רב ומסורבל.

## כיצד

בכדי להתחיל לעבוד עם ביטויים רגולריים בג'אווהסקריפט, יש להשתמש בפקודת `RegExp` ולהגדיר את המבנה של הביטוי הרגולרי שנרצה למצוא. למשל, אם ברצוננו למצוא את כל הכתבות שמכילות את המילה "ג'אווהסקריפט", נשתמש בביטוי הרגולרי `/ג'אווהסקריפט/g` ונפעיל אותו על המחרוזת שנרצה לבדוק.

```Javascript
let text = "קוד לפיתוח אינטרנט באמצעות ג'אווהסקריפט מסורבל? רצי לקרוא על כך בכתבה הבאה.";
let regex = /ג'אווהסקריפט/g;
console.log(text.match(regex));
```

Output: `["ג'אווהסקריפט", "ג'אווהסקריפט"]`

בנוסף, ניתן להשתמש בפקודת `replace` כדי להחליף חלקים מהטקסט באמצעות ביטויים רגולריים. למשל, אם נרצה להחליף את כל הרווחים במחרוזת למקף תחתון, נוכל לעשות זאת כך:

```Javascript
let text = "מתי תתחילו לפתח אתר חדש בג'אווהסקריפט?";
let regex = / /g;
console.log(text.replace(regex, "_"));
```

Output: `מתי_תתחילו_לפתח_אתר_חדש_בג'אווהסקריפט?`
