# למה

למה אחד יבחר לחפש ולהחליף טקסט? התשובה פשוטה - כיוון שפעמים רבות נצטרך לערוך ולעדכן טקסט, ועם כלי חיפוש והחלפה יש לנו את היכולת לעבוד בקלות ובמהירות על כמויות גדולות של טקסט.

# איך לעשות זאת

ישנם כמה דרכים לחפש ולהחליף טקסט ב-Javascript. הנה כמה דוגמאות:

```Javascript
let str = "הייתי רוצה ללמוד עברית";
let newStr = str.replace("כמה", "מלא");
console.log(newStr);
```
*פלט: "הייתי רוצה ללמוד מלא עברית"*

```Javascript
let str = "פתוח לעסקים חדשים";
let newStr = str.replace(/לעסקים/g, "של בית הספר");
console.log(newStr);
```
*פלט: "פתוח לשל בית הספר חדשים"*

מתוך הדוגמאות שמוצגות, ניתן לראות שהפונקציה "replace" מאפשרת לנו להחליף חלק או הכול של טקסט בתוך מחרוזת מסוימת לאחר חיפוש של חריגיות או מחרוזת כללית (regex). ניתן גם להשתמש במאפיין "global" בשילוב עם Regex כדי לבצע החלפות בכל החריגיות המתאימות במחרוזת.
