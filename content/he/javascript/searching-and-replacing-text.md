## למה

ממשיכים לעסוק בתכנות ב-Javascript? אחת הסיבות הבולטות לכך היא היכולות הרבות של שפת התכנות הזו, בהן אפשרות לחיפוש והחלפת טקסטים בצורה מהירה ויעילה. כתיבת פונקציות שמאפשרות למשתמש לערוך טקסטים באופן דינאמי ולהחליף חלקים של הטקסט בהתאם לצורכיו היא חלק בלתי נפרד מתהליך התכנות.

## כיצד לבצע חיפוש והחלפה של טקסט ב-Javascript

לדוגמה, ניתן להשתמש בפונקציה המובנית `replace()` כדי לחלוף על כל המופעים של מחרוזת מסוימת בתוך מחרוזת אחרת ולהחליפה במחרוזת חדשה. נדגים את זה באמצעות הקוד הבא:

```Javascript
let sentence = "שלום לכולם, היום נלמד כיצד להחליף תווים ב-Javascript";
let newSentence = sentence.replace("שלום", "הי");

console.log(newSentence);
```

פלט:

```Javascript
הי לכולם, היום נלמד כיצד להחליף תווים ב-Javascript
```

ניתן גם להשתמש בפונקציה `replaceAll()` כדי להחליף את כל המופעים של מחרוזת מסוימת בתוך מחרוזת עם מחרוזת אחרת. נדגים את זה באמצעות הקוד הבא:

```Javascript
let sentence = "שלום לכולם, היום נלמד כיצד להחליף תווים ב-Javascript";
let newSentence = sentence.replaceAll("היום", "מחר");

console.log(newSentence);
```

פלט:

```Javascript
שלום לכולם, מחר נלמד כיצד להחליף תווים ב-Javascript
```
