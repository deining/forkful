## למה 

קישור מחרוזות הוא אחת היכולות הבסיסיות של פיתוח תוכנה ב-Javascript. באמצעות הוספת מחרוזות, אנו יכולים לשדרג את כפתורי הדיבוג, ליצור הודעות חלונית או להציג את ערכי המשתנים שלנו במנגנונים מיוחדים.

## איך לעשות זאת?

בפוסט הזה, אנו נלמד כיצד לחבר מחרוזות באמצעות הפונקציה `concat ()`. ראשית, אנו ניצור מקראים לנתונים שלנו באמצעות כמה משתנים:

```Javascript
// יצירת משתנים עם נתונים של מחרוזת
let firstName = "אבי";
let lastName = "כהן";
```

כעת, נשתמש בפונקציה `concat ()` כדי לקשר את שני המשתנים ליצירת שם מלא:

```Javascript
// הצגת שם מלא באמצעות פונקציית concat ()
let fullName = firstName.concat(lastName);
console.log(fullName);
// החזרה: "אבי כהן"
```

ניתן גם לחבר יותר ממחרוזת אחת בפונקציה `concat ()` כדי ליצור מחרוזת ארוכה:

```Javascript
// הוספת מחרוזת נוספת לפונקציית concat ()
let school = "בית ספר למדעי המחשב";
fullName = firstName.concat(lastName, "לומד ב", school);
console.log(fullName);
// החזרה: "אבי כהן לומד בבית ספר למדעי המחשב"
```

ניתן גם לחבר מקומיות באמצעות פונקציות מחרוזת נוספות כמו `join ()` או `slice ()` כדי ליצור מחרוזות מורכבות יותר.
