# למה 

 אם אתה מתחיל ללמוד תכנות, ייתכן שתמצא עצמך משתמש במספרים אקראיים בקוד שלך. למה ניצור מספרים אקראיים בכלל? אחת הסיבות העיקריות היא ליצירת שימושים שונים בפרויקטים שלנו. ניתן להשתמש במספרים אקראיים ליצירת משחקים, מחלקים את המשחק לרמות שונות, או אפילו ליצירת נתונים סטטיסטיים מכל הקלטים האקראיים.

# כיצד לעשות זאת

תחילה, נצטרך ליצור משתנה שיוכיח את המספר האקראי שהוא שווה והערך שלו יהיה מה שניחזה. לדוגמה, אם אנו רוצים ליצור מספר אקראי בין 1 ל10, נשתמש בפונקציית "Math.random()" כדי ליצור את המספר האקראי שלנו.

```Javascript
let randomNumber = Math.floor(Math.random() * 10) + 1;
```
 
השתמשנו בפונקציות של Math כדי ליצור מספרים אקראיים בטווח מסוים. בהמשך, נוכל להשתמש בפונקציות שונות שיעניקו לנו תוצאות שונות כגון ליצור מספרים עשרוניים או ליצור מספרים בטווח מסוים בין שני מספרים נתונים.

# מסילת נשק – מספרים אקראיים 

כמו שהצגנו, ניתן ליצור מספרים אקראיים בעזרת פונקציות שונות של Math. אבל מה קורה מאחורי הקלעים? כדי ליצור מספר אקראי, מחשבים משתמשים במסדר למספרים בסדר קבוע כמו עשרות מליונים שנקראים "זרימה". שימוש במסדר זה מסייע למחשב ליצור מספרים שונים בכל פעם שהפונקציה מופעלת, כך שתמיד יהיה
