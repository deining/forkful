## למה:

ישנן מצבים רבים בתכנות שבהם נעדיף להמיר מחרוזת לרישיות קטנות, כגון כאשר אנו רוצים למיין ולהשוות מחרוזות. המרת מחרוזת לרישיות קטנות גם יכול לגרום לקריאת הקוד להיות יותר קלה ומבונה.

## איך לעשות:

```Ruby
# כאן אנו משתנים את המחרוזת לרישיות קטנות עם השיטה .downcase
string = "זהו טקסט באותיות גדולות"
string.downcase # תוצאה: "זהו טקסט באותיות קטנות"
```

```Ruby
# פונקציית קוד שמקבלת מחרוזת וממירה אותה לרישיות קטנות
def to_lower_case(str)
  str.downcase
end

# קריאת הפונקציה עם מחרוזת כנקודת התחלה
to_lower_case("אין צורך לפחד ממרתונים") # תוצאה: "אין צורך לפחד ממרתונים"
```

## מע深ע חקל דיבע:

השיטה .downcase נמצאת בכל מערכת הפעולה של Ruby ומיועדת כדי להמיר את המחרוזת לרישיות קטנות. היא מחזירה תוצאה חדשה ואינה משנה את המחרוזת המקורית.

כאשר מחרוזת מתחילה עם רישיות גדולות או מכילה את האותיות "איו", הפתרון הפשוט יותר הוא להשתמש בשיטת .capitalize:

```Ruby
"myText".capitalize # תוצאה: "MyText"
```

לעומת זאת, אם אנו רוצים להמיר את המחרוזת לכתיב בשם טבעי כולל האותיות הקטנות, ניתן להשתמש בשיטת .titleize:

```Ruby
"my name is john".titleize # תוצאה: "My Name Is John"
```
