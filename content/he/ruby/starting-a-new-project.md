## למה

כתיבת קוד בשפת רובי היא חוויה ייחודית ומאתגרת. השפה מאפשרת לנו ליצור קוד קריא ויעיל תוך השתמשות בתחביר פשוט ואלגנטי. תחילת פרויקט חדש בשפת רובי יכולה להיות מאתגרת, אך כשנדע את הדרך הנכונה לגשת אליה, התוצאות יהיו יפות ופונקציונליות.

## איך מתחילים

נתחיל עם הגדרה של משתנים בשפת רובי. כדי ליצור משתנה צריך לקודד את שם המשתנה ואחריו להכניס את הערך שלו. לדוגמה:

```Ruby
name = "איתי"
age = 30
```

כעת נוכל להדפיס את המשתנים הללו באמצעות הפקודה puts מתוך גוף הקוד:

```Ruby
puts "שלום, שמי #{name}"
puts "אני בן #{age} ואני מתחיל בקוד"
```

פלט הקוד יהיה:

```
שלום, שמי איתי
אני בן 30 ואני מתחיל בקוד
```

זהו כמה קוד ראשוני ובסיסי שיכול להיות שימושי בתחילת פרויקט חדש.

## חפירה עמוקה

בשלב זה של כתיבת הקוד כדאי להתאים את השפה היטב לצורך הפרויקט המיועד. נדבר לדוגמה על פונקציות. בשפת רובי ניתן ליצור פונקציות בעזרת המילה השמורה def, ואז לשם הפונקציה נוסיף פרמטרים, כך:

```Ruby
def greet(name)
  puts "שלום, שמי #{name}"
end
```

כעת, נוכל לקרוא לפונקציה כדי להדפיס את שלום עם שם משתנה שעברנו כפרמטר:

```Ruby
greet("איתי")
```

פלט הקוד יהיה:

```
שלום, שמי איתי
```
