## למה

למה שחלק ניכר מפיתוח התוכנה היום משתמש ברכיב חשוב ופופולארי בשם JSON. נצטרך מדי פעם להתעסק עם סקירת נתונים ממשקי API חיצי-ממסד-נתונים או כשמתכנתים צריכים להתאים תבניות לשם תעבודה עם פריטים מטריים או אחרים בפורמטים תרגום כמו XML ורבים יותר. JSON מאפשר לנו לעשות זאת בדיוק וללא כל דווקאי, פשוט חוסך זמן וכסף.

## כיצד לעשות את זה

```Ruby
require 'json'

# ליצור אובייקט JSON ממחרוזת
json_string = '{"name": "John", "age": 31, "city": "New York"}'

# לקרוא את הנתונים מהאובייקט
data = JSON.parse(json_string)

puts "שם: " + data['name']
puts "גיל: " + data['age'].to_s
puts "עיר: " + data['city']
```

בתוך הקובץ שקיבלנו, אנו יכולים לראות שיש לנו מחרוזת עם נתונים בפורמט JSON. אנו משתמשים במודול `json` כדי להמיר את המחרוזת לאובייקט יוצא דופן שניתן לקרוא מתוכו את הנתונים שאותו נרצה להשתמש איתם. במחרוזת נראה גם את האיטנרציות למילים או לנתונים המרוצים; זה כי אנו רוצים לפתוח בקובץ זהם אמורה להיות קריאת הנתונים.
