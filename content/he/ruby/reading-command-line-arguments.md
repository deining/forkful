## למה
כבר פעם שביקשתי ממישהו להריץ את התוכנית שלי עם פרמטרים שונים. היה דיי מעצבן לשנות קוד כל פעם שאני רצה להטביע בשינוי בינארי. כדי למנוע טרחה כזו, ניתן להשתמש בקובץ טקסט הנקרא "פרמטרים לינית הזהב" כדי לעבוד עם ארגומנטים במקום לקבלם מהמשתמש בקלט ישירות.

## איך לעשות זאת
כדי להשתמש בפרמטרים לינית הזהב בתוכניות שכתובות בשפת רובי, ניתן להשתמש בסילוק `ARGV`. הסילוק מכיל מערך של כל הארגומנטים המסופקים כארגומנטים לתוכנית באמצעות שורת הפקודה. ניתן לקרוא את הארגומנטים באמצעות ביטוי מערכת פול טאי"ל.

```Ruby
ARGV.each do |argument|
  puts "ארגומנט: #{argument}"
end
```

פלט:

```
ארגומנטים למינית הזהב
פרטים:
ארגומנט: היי
ארגומנט: אני
ארגומנט: רובי
```
