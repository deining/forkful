## למה

להתייחס לתאריך הנוכחי בתכנות בשפת רובי יכול להיות חיוני ביישומים שונים, לדוגמה ייתכן שנרצה להציג את התאריך הנוכחי למשתמש או להשתמש בו כחלק מתנאי בתוכנית. ייתכן גם שנרצה להשתמש בתאריך בכדי לבצע מחשבונים מתימטיים או לנתח נתונים באופן יותר יעיל. כל אלה הם דרכים שונות בהן תאריך נוכחי יכול לתרום לחווית התכנות ברובי.

## כיצד לעשות זאת

הנה כמה דוגמאות לקוד רובי שיעזרו לך להשיג את התאריך הנוכחי על המסך:

```Ruby
# השתמשו במטודה המובנית DateTime.now כדי לקבל תאריך מלא עם יום, חודש ושנה:
DateTime.now #=> 2022-01-01 11:23:45

# ניתן גם להשתמש במתודה המובנית Time.now עבור תאריך נוכחי בפורמט פשוט יותר:
Time.now #=> 2022-01-01 11:23:45 +0200

# כדי להפוך את התאריך לסטרינג לשימוש נוח יותר, ניתן להשתמש במתודת strftime שמאפשרת לנו לבחור את הפורמט שבו נרצה להציג את התאריך:
DateTime.now.strftime('%d/%m/%Y') #=> 01/01/2022
```

הנה תוצאות מהקוד האמור מעלה:

```Ruby
DateTime.now #=> 2022-01-01 11:23:45
Time.now #=> 2022-01-01 11:23:45 +0200
DateTime.now.strftime('%d/%m/%Y') #=> 01/01/2022
```

## חקירה מעמיקה

לקבלת תאריך נוכחי ברובי אנו משתמשים במנגנון שנקרא "Time object". התאריך נוכחי מחזיר מעין "פריט" שבו מכיל את התאריך ואת הזמן באופן מופרד.

כדי להבין טוב יותר את המנגנון הזה, נוכל לנסות ולהדפיס את התאריך הנוכחי בתוך מערך כדי לחקור את התנה
