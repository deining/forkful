## מה ולמה?
הקבלת התאריך הנוכחי היא פעולת מתיחסת לאחד ההידורים העיקריים בתכנות - הזמן. בעזרת פעולה זו, מפתחי תוכנה יכולים להשתמש בתאריך הנוכחי בהתאם לצורכים השונים של יישום התוכנה שלהם.

## איך ל?
התאריך הנוכחי ניתן לקבל בקלות בעזרת הפקודה "Time.now" בשפת Ruby. תחתית תאריך נתון יכול להיות מאוד מועיל לצורך תעדוף, הזמנה של אירועים וכן הלאה. נשתמש בדוגמה כדי להדגים את השיטה להשיג את התאריך הנוכחי:

```Ruby
puts Time.now
```
כאשר תריץ את הקוד זה, אתה תקבל את התאריך והזמן הנוכחי בתור פלט:
``` 
2021-10-19 12:00:00 +0300
```

## הכנס כמות רבה
הפעולה "Time.now" היא חלק מספר רב של פעולות שניתן להשתמש במטרה להשיג את התאריך הנוכחי. פתרונות אחרים כוללים שימוש בספריות חיצוניות או כתיבת פונקציות משלך לחישוב התאריך הנוכחי. אתה יכול למצוא את הפעולה "DateTime.now" בתוך הספרייה המובנית "DateTime" או להשתמש בפונקציות מתוך ספריות חיצוניות כמו "Date" או "Timezone".
