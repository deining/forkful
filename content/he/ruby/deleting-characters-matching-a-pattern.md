## למה

חיפוש ומחיקת תווים המתאימים לתבנית בתוך מחרוזות טקסט הוא פעולה קלה ואפקטיבית כשאנחנו צריכים לעבוד עם מחרוזות מורכבות. פעולה זו יכולה להיות מועילה בעיבוד טקסטים ונתונים מסוגים שונים, כגון פתרונות לבעיות ניסוי טקסט או הסרת תווים מיוחדים.

## איך לעשות זאת

עבור רוובי, ישנן מספר דרכים בסיסיות למחוק תווים מתאימים לתבנית:

קוד ייצוגי:
```
str = "היי לכולם! שלום לכולם!"
str.gsub!(/[!כ]/, "")
puts str #=> "הי לולם שלום לולם"
```

בדוגמה זאת, אנו משתמשים בפונקציית gsub כדי למחוק את התווים מתאימים לתבנית, "\\ /" כאשר משתמשים בפקודת gsub! יחד עם תווים מיוחדים כמו \\,\,\ או / כדי למחוק אותם מהמחרוזת. בתור פרמטר נוסף, אנו מציינים את התווים שאנו רוצים להחליף אותם עם כלום – כלומר מחרוזת ריקה.

עוד דרך למחוק תווים מתאימים היא להשתמש בפונקציה delete:

קוד ייצוגי:
```
str = "אלפי המת לידה!"
str.delete!("א")
puts str #=> "לפי המת לידה!"
```

הפונקציה הזו מחקה את כל התווים המתאימים לתבנית מהמחרוזת.
