## למה

כתיבת קובץ טקסט היא כלי חשוב בתחום תכנות המחשבים. הוא מאפשר לנו ליצור מידע שמאורגן בתוך קובץ ולעשות בו שימוש בצורה חכמה ויעילה. כתיבת קובץ טקסט בשפת רובי היא דרך יעילה לכתוב קוד ולאחסן מידע חשוב במאגר הנתונים שלנו.

## איך לעשות זאת

לכתוב קובץ טקסט בשפת רובי פשוט וקל. תחילה, נצטרך לפתוח קובץ עם השם והנתיב הרצוי. לדוגמה, אם נרצה לכתוב קובץ טקסט בשם "example.txt" בתיקייה הנוכחית שלנו, נצטרך להריץ את הפקודה הבאה:

```Ruby
file = File.new("example.txt", "w")
```

בפקודה זו, אנחנו מנתחים שם ומצביעים על האופציה "w" שתאפשר לנו לכתוב לקובץ. לאחר מכן, נוכל להשתמש בפקודת "puts" כדי להוסיף תכנים לקובץ בצורה פשוטה ומהירה:

```Ruby
file.puts "כתיבת טקסט לקובץ"
file.puts "לשפת רובי פשוט וקל!"
```

ולסיום, אנחנו נצטרך לסגור את הקובץ כדי להימנע מאיות שגויים ואיבוד מידע. נוכל לעשות זאת עם הפקודה "close":

```Ruby
file.close
```

והנה כבר יש לנו קובץ טקסט עם הכיתובים שרצינו להוסיף!
