## למה

למה כדאי לכתוב קובץ טקסט? כי כתיבת קבצי טקסט היא חלק בלתי נפרד מתהליך התכנות והיא מאפשרת לנו לאחסן מידע בצורה פשוטה וקריאה. כתיבת קבצי טקסט גם חיונית ליצירת תכניות וסקריפטים תכנותיים שישמשו לנו בעתיד.

## איך לעשות זאת

לפניך דוגמאות קוד כיצד לכתוב קובץ טקסט בשפת רובי ואת התוצאות שאיתן תקבלו. ניתן להשתמש בקוד המופיע כאן כדגם עבור פרויקטים משלכם. כדי להציג את הקוד, ניתן להשתמש בסימון "```Ruby ... ```".

```Ruby
# פתיחת קובץ טקסט חדש עם שם "example.txt"
file = File.new("example.txt", "w")

# כתיבת טקסט מחרוזת לקובץ עם .puts
file.puts("Hello World!")

# סגירת קובץ לאחר כתיבת הטקסט
file.close

# פתיחת הקובץ החדש לקריאה והצגת התוכן
file = File.open("example.txt", "r")
puts file.read
```

תוצאה:
```
Hello World!
```

## חפירה עמוקה

עכשיו ניכנסים לחלק העמוק יותר של כתיבת קבצי טקסט בשפת רובי. הנה כמה יתרונות ואפשרויות שיש לנו לשחק עם:

- מאפשר לנו להגדיר את סוג הקודינג (utf-8, ASCII וכו') לכתיבת הקובץ ולקריאתו.
- ניתן להשתמש בפקודות כמו `.each` ו `.map` כדי לעבוד עם חלוקת המידע בקובץ להפניות שונות.
- ניתן ליצור מסמכי טקסט דינמיים עם שימוש במשתנים ופונקציות.
