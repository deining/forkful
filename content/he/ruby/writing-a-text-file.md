##למה:

כתיבת קובץ טקסט היא חלק חשוב מאוד בתהליך של תכנות בשפת רובי. באמצעות כתיבת קובץ טקסט, ניתן ליצור מידע נשמר שיכול לשמש כמקור תמציתי ומידעים מאורגנים לצורך תכנות פעולות מסוימות. 

##כיצד לעשות זאת:

כאשר מדובר בכתיבת קובץ טקסט ברובי, ישנן מספר דרכים לבצע זאת. אחת הדרכים הפשוטות היא להשתמש בפקודת הפתיחה "File.open" כדי ליצור את הקובץ הרלוונטי ולשמור אותו כגוף חי בתוך הקוד שלנו. לדוגמה:

```Ruby
File.open("example.txt", "w") do |file|
  file.puts "זהו דוגמה של כיצד לכתוב טקסט בקובץ"
end
```

כאשר נכתוב קוד כזה, יוצרים קובץ טקסט חדש בשם "example.txt" ומכניסים אליו את המחרוזת "זהו דוגמה של כיצד לכתוב טקסט בקובץ". אם נפעיל את התכנית הזו, נוכל לראות את המחרוזת הזו נכתבת בקובץ כמו שאנחנו רוצים.
