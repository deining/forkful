# למה
תצוגת הדפדפание אווטפוט (debug output) היא כלי חשוב בתהליך התכנות של רובי. היא מאפשרת למתכנתים לראות מידע חיוני בזמן ריצת הקוד כגון ערכי משתנים ושגיאות בקוד. במאמר זה, אני אסביר לכם מדוע כדאי להשתמש בו וכיצד לעשות זאת ברובי.

# כיצד להשתמש בקבצי טקסט להדפסת טקסט נתונים ומידע חיוני
ברובי, תצוגת דבג אוטפוט נעשית בעזרת פקודת הדפסה ל-stdout (פלט סטנדרטי). ניתן להדפיס מידע מתחת לקוד באמצעות פקודת `puts` או `p`. כמו כן, אפשר גם ליצור קבצי טקסט נפרדים ולהשתמש בפקודה `File.write` כדי להדפיס מידע לקובץ זה.

```Ruby
# פקודת הדפסה ל-stdout:
puts "זהו הדפסה רגילה"

# פקודת p ל-stdout:
p "זהו הדפסת טקסט בפורמט מעוצב"

# יצירת קובץ טקסט והדפסת נתונים לקובץ:
File.write("output.txt", "טקסט שישמר לקובץ")
```

הפלט של קוד זה יהיה:

```
זהו הדפסה רגילה
"זהו הדפסת טקסט בפורמט מעוצב"
```

ובקובץ output.txt תהיה המחרוזת "טקסט שישמר לקובץ".
