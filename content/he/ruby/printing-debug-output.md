## למה
שאלה חשובה לכל מתכנת היא - למה אנו מדפיסים פלט לאיתור באגים? התשובה היא פשוטה - כי המחשב אינו תמיד עושה מה שצריך. כשאנו מדפיסים פלט של קוד בשלבים שונים, אנו יכולים לראות במה טעות התכנית נוצרה ולתקן אותה באופן יעיל.

## כיצד לעשות זאת
נבינה כיצד להדפיס פלט כדי לאתר את הבאגים בעזרת דוגמאות קוד ופלט דוגמא בתוך בלוקי קוד "```Ruby ... ```":

```Ruby
# דוגמא של מחרוזת שנוספה לתוך נתיב קובץ
file_path = "C:/Documents/test.txt"

# דוגמא של יוזרניים שנקראו מאגר נתונים
users = Database.get_users()

# דוגמא של הדפסה של הגדרת משתנים
name = "יעל"
age = 30
puts "שלום, קוראים לי #{name} ואני בן #{age}!"

# דוגמא של הדפסה טכסט פורמטים ספציפיים
amount = 500
puts "הכמות הנוכחית שלך בחשבון היא #{amount} ש"ח."

# דוגמא של הדפסה של טופס תצוגה
form = "First Name: #{name} | Age: #{age}"
puts form
```
הפלט של הקוד הנ"ל יהיה:
```
שלום, קוראים לי יעל ואני בן 30!
הכמות הנוכחית שלך בחשבון היא 500 ש"ח.
First Name: יעל | Age: 30
```

## הכי עמוק
כאשר אנו מדפיסים פלט לאיתור באגים, חשוב לדעת כיצד לבחור את המידע הנכון להדפסה. נוכל להשתמש בכל מה שנמצא בתוך נתיב הקוד כפתרון, כך שנוכל לראות במה התכנית נתקלה בבעיה ואיך אנו יכולים לתקן אותה. כדאי להגדיר משתנים חשובים כמו שם, גיל, כתובת ועוד, כדי להיות יותר מדויקים באיתור הבעיה.
