## Why

למה לקרוא קובץ טקסט בכלל? קריאת קבצי טקסט היא כלי חשוב בתכנות בשפת רובי, המאפשר לנו לקרוא ולעבד מידע מקובץ טקסט בקלות ומהירות.

## How To

הנה דוגמאות לכיצד לקרוא קובץ טקסט בשפת רובי ולהדפיס את התוכן שלו:

```Ruby
# פתיחת הקובץ לקריאה והגדרת משתנה שמכיל את כל התוכן
file = File.open("text.txt", "r")
contents = file.read

# הדפסת התוכן של הקובץ
puts contents

# סגירת הקובץ
file.close
```

הנה עוד דוגמה עם שימוש בפקודת ציון מיקום (cursor) כדי להדפיס את התוכן מפסקה מסוימת:

```Ruby
# פתיחת הקובץ לקריאה והדפסת התוכן מפסקה מסוימת
file = File.open("text.txt", "r")
file.seek(10) # פקודה זו מציינת מקום בקובץ לפי מספר התווים מההתחלה
puts file.read(20) # הדפסת 20 תווים מהמקום שהצבנו

# סגירת הקובץ
file.close
```

הנה דוגמה נוספת עם שימוש בלולאות כדי לעבור על כל התוכן של הקובץ ולהדפיס כל שורה בנפרד:

```Ruby
# פתיחת הקובץ לקריאה והדפסת כל שורה בנפרד
File.foreach("text.txt") do |line|
    puts line
end

# סגירת הקובץ
file.close
```
