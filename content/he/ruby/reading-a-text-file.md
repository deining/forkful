## מדוע

קריאת קובץ טקסט היא כלי חשוב בתכנות בשפת רובי. אתה יכול לקרוא קובץ טקסט כדי לקבל מידע ממקור חיצוני ולעבוד איתו בתוכניות שלך. זה יכול לשמש לדוגמה לטעינת נתונים או ליצירת תוכניות שתעבוד על נתונים מסוימים.

## כיצד לעשות זאת

כדי לקרוא קובץ טקסט בשפת רובי, ניתן להשתמש בפעולות קובץ פנימיות שקיימות במכשיר עבודה. לדוגמה, ניתן לפתוח קובץ עם הפעולה `File.open` ולקרוא את התוכן שלו עם הפעולה `File.read`. ניתן לשנות את הדרך שבה הקובץ נפתח ונקרא, כגון להציג רק את חלק מתוך התוכן או לבצע פעולות נוספות עליו כמו לחפש מילים מסוימות בתוך הקובץ.

````ruby
# פתיחת קובץ בשם example.txt וקריאת התוכן שלו
File.open("example.txt") do |file|
  puts file.read
end

# פתיחת קובץ והצגת רק את חלק מתוכו
File.open("example.txt") do |file|
  puts file.read(10) # הצג שורה ראשונה בלבד
end

# פתיחת קובץ וחיפוש מילה מסוימת בתוכו
File.open("example.txt") do |file|
  puts file.read.include?("hello") # האם המילה "hello" קיימת בקובץ?
end
````

## חפירה עמוקה

כאשר נדבר על קריאת קובץ טקסט, ישנן עוד המון אפשרויות ופעולות שניתן לבצע כדי לעבוד עם התוכן שנמצא בתוך הקובץ. ניתן להשתמש בפעולות כמו `each_line` ו-`each_char` כדי לעבור על כל השורות והתווים בקובץ בנפרד, וניתן גם להשתמש בפעולות כמו `write` כדי לרשום
