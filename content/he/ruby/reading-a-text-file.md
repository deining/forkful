המאמר הזה יתמקד בכתיבת קוד בשפת רובי לקריאה של קבצי טקסט בסיסיים. אם אתם מתעניינים בכתיבת תכניות מחשב באמצעות רובי, קריאת קבצי טקסט היא כלי חיוני שיעזור לכם לעבוד בצורה יעילה ומהירה. כמו כן, היא יכולה להיות מאתגרת מבחינת קוד, ולכן כדאי ללמוד כיצד לקרוא ולעבוד עם קבצי טקסט.

## Why

למה כדאי לקרוא קבצי טקסט בשפת רובי? הקריאה של קבצי טקסט מאפשרת לנו לקרוא ולעבוד עם מידע מן הקבצים הנמצאים במחשבנו. היכולת לקרוא ולעבוד עם קבצי טקסט היא חשובה עבור כל מתכנת רובי, מאחר ומצוין לנו רשימת המידע הנמצאת בקבצי טקסט כפי שנרצה.

## How To

כתיבת תכניות מחשב באמצעות רובי כוללת גם עבודה עם קבצי טקסט. בכדי לפתור את התרגילים שלנו, נפעל באמצעות דפוס כתיבה ע"י כתיבת קוד, קלטות ופלטים. אתם יכולים לכתוב יותר מתכנת הרובי אם אתם יודעים כיצד לקרוא ולעבוד עם קבצי טקסט. כאן נמצא דוגמא לכתיבת קוד עבור קריאת קובץ טקסט בשפת רובי:

```Ruby
file = File.open("filename.txt", "r") # פתיחת הקובץ כדי לקרוא אותו

file_data = file.readlines # קריאת הקובץ ושמירה של הנתונים במשתנה

file.close # סגירת הקובץ

puts file_data # הדפסת הנתונים שנקראו
```
