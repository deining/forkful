## למה

כתיבת קוד היא מטלה עצומה, ולפעמים זה נראה שאלת חיים לקרוא מסמך טקסט בתוך הקוד. כל הקוד שאנחנו כותבים דורש טיפול ועיבוד של מידע ממקורות חיצוניים כמו קבצי טקסט, ואיך לקרוא אותם הינם חלק מהיסודות הבסיסיים ביותר של תכנות בשפת רובי.

## איך לעשות זאת

הדרך הכי פשוטה לקרוא קובץ טקסט ברובי היא דרך פתיחה של קובץ ועיבוד המידע שלו משורה לשורה. הנה דוגמה לקוד ולפלט שמציגים איך זה נעשה:

```ruby
file = File.open("sample.txt")
file.each_line do |line|
  puts line
end
```

פלט:

```
שלום עולם
מה שלומך?
```

בקוד הזה, אנו פותחים את הקובץ "sample.txt" ולאחר מכן משתמשים בשיטת `each_line` כדי לעבור על כל שורות הטקסט בקובץ. בכל שורה, אנו מדפיסים את התוכן שלה. כך, אנו יכולים לקרוא מסמך טקסט כולו תחת היישום של `each_line`.

## מסע לעומק

אם תרצו להשתמש בקובץ גדול יותר עם טקסט מבוצע שורה לשורה, ניתן לעבוד איתו כמו רכיב מעגל ולבצע פעולות נוספות על כל שורה בנפרד. כדי לעשות זאת, נתחיל עם קוד כמו זה:

```ruby
file = File.open("sample.txt")
file.each_line do |line|
  line_array = line.split(" ")
  puts "המשפט הראשון בשורה הוא: #{line_array[0]}"
end
```

פלט:

```
המשפט הראשון בשורה הוא: שלום
המשפט הראשון בשורה הוא: מה
```

בשורת הקוד הראשונה, אנו ממציאים מערך חדש של כל מילים בשורה על ידי חלוקתן לפי מרחב לב
