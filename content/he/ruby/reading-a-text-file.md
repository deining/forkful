## מה ולמה?
קריאת קובץ טקסט היא פעולת קוד נפוצה בתוך עולם התכנות. היא מאפשרת לנו לקרוא ולעבד מידע מתוך קבצים טקסט כגון קבצי טקסט רגילים, קובצי קוד ועוד. תהליך זה חשוב בגלל שהוא מאפשר לנו לטפל במידע רב כמו מסמכים, נתונים וקוד בקלות ויעילות.

## איך לעשות?
```Ruby
# קוד לקריאת קובץ טקסט
File.read("שם הקובץ")

# קוד לעיבוד מידע מתוך קובץ טקסט
text = File.read("שם הקובץ")
puts text

# פלט סופי
התוכן שנמצא בקובץ 
```

## עיון עמוק
העיבוד של קבצי טקסט הוא מושג בסיסי בתכנות והוא משמש ברוב השפות המתקדמות. לפני קודם ממנו קבצי טקסט היו מיועדים לקריאה בצורה כללית ומסודרת יותר, תהליך זה נקרא זכרון כללי. פעולת קריאת קובץ טקסט נחשבת כיום לפעולה רגילה בתכנות ויוצרת את היסוד לעיבוד מידע מורכב ומגוון יותר.
