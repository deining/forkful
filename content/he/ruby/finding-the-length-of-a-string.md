# מה ולמה?

מציאת אורך של מחרוזת היא תרגיל פשוט, שתירת הידיים מול שפת תכנות רב אפשרויות כמו ​​רובי. זהו תהליך שמאפשר למתכנתים לדעת כמה תווים נמצאים במחרוזת מסוימת. המשימה הזו יכולה להיות מאוד שימושית כאשר אנו צריכים לבדוק את אורך של לוחם או לאתר מילים מסוימות במחרוזת גדולה.

# איך לעשות זאת:

תנו לנו להציג לכם כמה קוד קצר לקבלת אורך של מחרוזת ברובי:

```ruby
"Hello World".length
# => 11

"שלום עולם".length
# => 9

"".length
# => 0
```

# כיול עמוק:

בארגז הכלים של כל מתכנת רובי ניתן למצוא הרבה פעולות שונות לעבוד עם מחרוזות. אחת מהן היא `length` ובעזרתה התוכנית שלנו תוכל לקבל גישה ישירה לאורך של מחרוזת מסוימת. מאחר והיא פעולה כל כך פשוטה ונגישה, לא בנויק כדי לגשת לפעולה כזו.

אם אתם מחפשים דרך אחרת לקבל את אורך של מחרוזת ברובי, תוכלו לנסות להשתמש בפעולת `size` או `bytesize`. הפעולות הללו יחזירו את אותו התוצאות כמו `length`, אבל נכתבות בקוד שונה.

אם מעוניינים ללמוד יותר על מחרוזות ברובי, ניתן לבחון פעולות נוספות כמו `capitalize`, `downcase`, `reverse` וכו '.
