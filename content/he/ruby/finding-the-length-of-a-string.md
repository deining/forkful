## למה

במאמר זה נלמד כיצד למצוא את אורך המחרוזת בשפת רובי. למה לתת את עצמכם למטרה זו? בפתרונות לבעיות תכניתה שונות, חל אילו מקרים בהם נדרש להכיר את אורך המחרוזת כדי להתמודד עם השאלה. לדוגמה, קוד זה יכול לשמש כדי לבדוק את אורך תווים שהוספו כחלק מהזנת משתנים או כדי לפצח סיסמאות.

## כיצד ל

כדי למצוא את אורך המחרוזת בשפת רובי, נשתמש בפעולת `.length`. נוכל גם להשתמש בפעולת `.size` כיוון שהן אותו הדבר. נשתמש בשתי הפעולות בדוגמאות שלאחר מכן.

```Ruby
phrase = "שלום רובי!" # קודים זו ההודעה שנמסרת
puts phrase.length # => 10
puts phrase.size # => 10
```

בדוגמה נוספת, נשתמש בשימוש של משתנה כיוון שזה יכול להיות שימושי כאשר מתקבלת הזנת משתמש ואנחנו רוצים לבדוק את אורך הקלט.

```Ruby
user_input = gets.chomp # קאסטינג של קלט המשתמש למחרוזת
puts user_input.length # => 5 (משתמשים חשבתם על המפתח הנכון?)
puts user_input.size # => 5
```
