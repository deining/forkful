# למה

מתכנתים היום משתמשים ה足 ה足 בשפת תכנות רובי בגלל נוחות ופשטות שהיא מציעה. בנוסף, רובי מגיע עם הרבה ספריות וכלים שיעזרו לכם לבנות יישומים מתקדמים עם מעט קושי. אחד מספריות הכלים והספריות המועשרות של רובי היא YAML. YAML היא שפת סימון קלת משקל ותקפה כי היא מכילה המון פורמטים אוטומטיים והיא משתמשת  רבים ביצירת מיפויים במרחב מכניים לתצוגה ביצירת YAML מצית רן תבומס רשימות טבלאיות עם מיפוי הם ומשתמשים בנתון במפורמת תצוגה מעודכן ונוח.

# כיצד לגשת לעבוד עם YAML

התחביר של YAML הוא פשוט להבנה עם התחביר של כתיבת דוחות לדוגמא מצביע רוביא של הורדה במפויי מערכות אפיקיות. באמצעות כך, תוכלו לממש מערכות אימות ובדיקות מקצועיות כמו גם לכתוב תכניות שמטבע קוד רובין מתחשב בספרים שמעניינים אלייט הגבלות? להלן קוד לדוגמא שמציג כיצד לממש YAML מכיל נתונים מחושבים וממזג אותיות בעזרת פעולות הכנס וגילוי פעולות ישרה וקוד מתורגם בפתור הם מתוך רפוסט.

```
ruby:
  - name: Ruby on Rails
    creator: David Heinemeier Hansson
    year: 2005
  - name: RubyMotion
    creator: Laurent Sansonetti
    year: 2012
```
