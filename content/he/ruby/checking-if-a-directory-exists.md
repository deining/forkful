## למה

לבדוק אם תיקייה קיימת היא חלק חשוב מתהליך התכנות ב-Ruby. אנשים ישתמשו בכתיבת תנאים כדי לוודא שהתיקייה קיימת לפני שהם מבצעים פעולות עליה, כגון כתיבה לתוך התיקייה או יצירת קבצים בתוכה.

## איך לעשות זאת

ב-Ruby, ניתן לבדוק אם תיקייה קיימת באמצעות הפעולה `Dir.exist?()` ולהעביר לה את הנתיב של התיקייה כארגומנט. לדוגמה:

```Ruby
if Dir.exist?("/path/to/directory")
    puts "The directory exists"
else
    puts "The directory does not exist"
end
```

הפעולה תחזיר את הערך `true` אם התיקייה קיימת ואת הערך `false` אם היא לא קיימת. ניתן גם להשתמש בפעולת השוואה `==` כדי לבדוק אם הערך הוא `true`, לדוגמה:

```Ruby
if Dir.exist?("/path/to/directory") == true
    puts "The directory exists"
else
    puts "The directory does not exist"
end
```

בנוסף, ניתן להשתמש בפעולת `Dir.exist?()` כחלק מתנאי `if` כדי לבדוק אם התיקייה קיימת בתוך בלוק קוד מסוים, כך שניתן למנוע הרצת קוד שלא יצליח אם התיקייה לא קיימת.
