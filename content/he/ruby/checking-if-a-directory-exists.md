## למה
בדיקה אם תיקייה קיימת היא חלק חיוני מתהליך פיתוח תוכנה. ביצוע בדיקה זו מאפשר את ניהול טוב יותר של קבצים ותיקיות ומונע תקלות פוטנציאליות באפליקציות.

## איך לעשות זאת
כדי לבדוק אם תיקייה קיימת באמצעות קוד רובי, ניתן להשתמש במתודת `Dir.exist?` אשר מקבלת כארגומנט את הנתיב לתיקייה ומחזירה ערך בוליאני (true/false) בהתאם לקיום התיקייה. לדוגמה:

```Ruby
if Dir.exist?("/Users/username/Documents")
  puts "תיקיית המסמכים קיימת"
else
  puts "תיקיית המסמכים לא קיימת"
end
```
פלט:

```
תיקיית המסמכים קיימת
```

על מנת לבדוק אם תיקייה קיימת בתיקיית העבודה הנוכחית, ניתן להשתמש במתודה המאפשרת לנו לשלב את הנתיב הנוכחי - `Dir.pwd` ולהוסיף לו את השם של התיקייה באמצעות סימן `/`. לדוגמה:

```Ruby
if Dir.exist?(Dir.pwd + "/pictures")
  puts "תיקיית התמונות קיימת בתיקיית העבודה הנוכחית"
else
  puts "תיקיית התמונות אינה קיימת"
end
```
פלט:

```
תיקיית התמונות אינה קיימת
```
