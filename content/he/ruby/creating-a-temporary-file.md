## למה:
יצירת קובץ זמני ברובי היא פתרון בעייתי ופשוט עבור כתיבת קוד עם הבנת נתונים חדשה או עיבוד קבצים.

## איך לעשות:
```
Ruby require 'tempfile'

# יצירת קובץ זמני בשם temp עם סיומת txt
temp = Tempfile.new(["temp", ".txt"])

# כתיבת מחרוזת לקובץ
temp.write("זוהי מחרוזת נכתבת לקובץ זמני")

# קריאה מהקובץ
puts temp.read

# סגירת הקובץ ומחיקתו
temp.close
temp.unlink

```
פלט:
```
זוהי מחרוזת נכתבת לקובץ זמני
```

## כיוון מעמיק:
יצירת קובץ זמני ברובי מאפשרת לנו לעבוד עם קבצים בצורה קלה יותר, ובמקרים כמו תחילת תכנות, יכולה לשמש כדי לתרגל וללמוד על כתיבת קבצים.
