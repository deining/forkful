# מה ולמה?
תאימויות רגילות הן כלי חשוב לכל מתכנת רובי. הן מאפשרות לנו לחפש ולתבון בטקסט על פי תבניות, המאפשרות לנו למצוא מידע מבוקש בצורה מהירה ויעילה בהרבה. לכן, מתכנתים משתמשים בתאימויות רגילות כדי לעבוד על קבצי טקסט ותבניות שונות בצורה פחות מתפשרת ומכוונת יותר.

# איך להשתמש:
הנה דוגמא פשוטה של תאימויות רגילות ברובי כדי לחפש מילים וספרות בתוך טקסט:

```ruby
text = "שלום לכולם! זהו טקסט פשוט וקל"
puts text =~ /שלום/ # הדפס את המיקום של המילה "שלום" בטקסט
puts text =~ /\d+/ # הדפס את המספר הראשון שנמצא בטקסט
```
הפלט המשובץ יהיה:

```
0
16
```

אנו מחפשים את המילה "שלום" בטקסט באמצעות כוכבית (/ /), המשמשת כגל הביטויים על פני הטקסט שאנו רוצים לחפש בו. לאחר מכן, אנו מחפסים אחר התבנית הבאה - ספרות (\d+) ומדפיסים את התוצאה הראשונה שנמצאה. במקרה זה, המספר הראשון המצוי בטקסט הוא 16 ולכן זה מה שיחזור.

# מעמקים עמוקים:
תאימויות רגילות נמצאות בשימוש כבר מאז תחילת שנות ה70 של המאה ה-20, והן כיום נמצאות בשימוש תכוף ברבות שפות תכנות שונות. לכמה שפות יש גם ליבות תאימויות רגילות מובנות. בתור מתכנתים רובי, אנו מבינים שתאימויות רגילות הן חלק חשוב מהשפה ולכן אנו צריכים להכיר תחומי שימוש רבים כגון חיפוש, עיבוד ותחלופים.
