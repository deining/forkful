## מה ולמה?

אנחנו כותבים קוד מחשב עבורכם ושולטים בשפת תכנות רובי, אנחנו הפעם מדברים על מושג שמקובל לפעמים להיפגש איתו בתוך הקוד שלנו - פריסת תאריך ממחרוזת. אז כמה מילים על מה זה אומר ולמה מתכנתים עושים את זה.
פריסת תאריך ממחרוזת פשוטה היא תהליך קילוט תאריך ממחרוזת טקסטואלית והמרתו לפורמט תאריך שניתן לעבוד איתו בקוד. מתכנתים מבצעים זאת בכדי לכוון את התאריך לשימושים שונים בקוד כגון מילוי שדות טפסים או חישוב של ימים בין תאריכים.

## איך לעשות:

לפניכם שלושה דוגמאות של כתיבת קוד רובי כדי לפרוס תאריך ממחרוזת ופלט חוזר:

```Ruby
# סוג מחרוזת טקסט המכיל תאריך
date_str = "10/15/2020"
# פירוק התאריך לפי חלקיו
month, day, year = date_str.split("/")
# המרת המחרוזת למספרים שימושיים לפעולות נוספות, כגון הכנסת התאריך לבסיסי נתונים
year = year.to_i
month = month.to_i
day = day.to_i
# יצירת תאריך עם המספרים שהפירוק נתן לנו
date = Date.new(year, month, day)
```
```Ruby
# פירוק תאריך מהמחרוזת לתווים בלבד
date_str = "March 20, 2021"
month = date_str[0..2]
day = date_str[4..5]
year = date_str[-4..-1]
# יצירת מערך של כל האלמנטים שפורסו
date_arr = [month, day, year]
# הצגת המערך בפורמט מתאים לתאריך
puts date_arr.join("/")
```
```Ruby
# פירוק תאריך מהמחרוזת באמצעות הפונקציונליות המצויה בתוך Ruby
require "date"
date_str = "01/05/2021"
# יצירת תאריך מהמחרוזת באמצעות הפונקציה תאריך
date = Date.strptime(date_str, "%m/%d/%Y")
# פלט תאריך בפורמט של תאריך מלא מתאים ללוח השנה הסטנדרטי
puts date.strftime("%B %d, %Y")
```

## כניסה מעמיקה:

פריסת תאריך ממחרוזת היא תהליך שמשתמשים בו כבר עשרות שנים ונמצא בשימוש נרחב כי חלק מהתוכניות או הכלים הפותחים באופן כללי אינם יכולים לעכל את כל הפורמטים שאנשים משתמשים בהם בתאריך. בנוסף, ישנן אפשרויות אחרות כמו פירוק תאריך מתוך תוכניות הפועלות בתשתיות מסוימות אך עבור הרבה מתכנתים פריסת תאריך ממחרוזת היא פתרון מספק ומהיר.
