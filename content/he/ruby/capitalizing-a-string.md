## למה
בעולם התכנות, ישנם מספר טכניקות שמאפשרות למפתחים להתאים קוד לצרכים שונים. אחת מהן היא הפונקציה להפוך את האות הראשונה של מחרוזת לאות גדולה. מדוע כדאי להשתמש בטכניקה זו? בכדי להימנע מטעויות או רווחים לא רצויים בקוד ולהבטיח שהמחרוזת תופיע בצורה נכונה בכל מקרה.

## איך לעשות זאת
כדי לפנות את האות הראשונה של מחרוזת לאות גדולה ב-Ruby, ניתן להשתמש במתודה .capitalize. נכתוב כמה דוגמאות כיצד לשימוש בפונקציה זו:

```Ruby
# דוגמא 1
string = "ruby programming is fun!"
puts string.capitalize
# פלט יהיה: "Ruby programming is fun!"

# דוגמא 2
string = "hello world"
puts string.capitalize
# פלט יהיה: "Hello world"

# דוגמא 3
string = "טקסט"
puts string.capitalize
# פלט יהיה: "טקסט"
```

כמו שניתן לראות בדוגמאות, המתודה תחזיר את המחרוזת המקורית עם האות הראשונה בצורה גדולה, ואת האותיות הנותרות באותיות קטנות.

## טיפול עמוק
על מנת להפיק את המרב מהפונקציה ולהבין טוב יותר את עיקרי התכנות ב-Ruby, חשוב לשים לב למספר נקודות:

- המתודה תפעל רק על האות הראשונה של המחרוזת ולא על האותיות האחרות במחרוזת.
- אם תנסו להחיל את המתודה על מחרוזת שלא מתחילה באות היאהודית, המתודה תחזיר את אותה מחרוזת בלי שינוי.
- ניתן להשתמש במתודה במחרוזות בעברית, אך יש לזכור שהאות הראשונה תהיה בגודל גדול לפי סדר האותיות האנגלי.
