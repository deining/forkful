# למה
בעולם התכנות ישנם מספר תפקידים לפי התנאים השונים של הקוד המתקבל. כאחד מתפקידי הקודר ייתכן שיהיה צורך לטפל בטקסט שיופיע במספר מקומות וצורות שונות. כדי לשלוט בטקסט זה כאן נכנסת לתמונה ההתעסקות עם פעולת Capitalize של שפת התכנות Ruby.

## איך לעשות זאת
בשפת תכנות Ruby ישנם מספר דרכים לבצע פעולת קפיטלייז על טקסט. הקפיטלייז משמש להמרת תווים בטקסט לצורה ספציפית, כך שתווים אכן יופיעו כאמור בראי הנתונים ומוצגו בצורה נכונה. 

```Ruby
"hello world".capitalize
=> "Hello world"
```

במקרים בהם יש צורך לקפיטלייז רק את המילה הראשונה בטקסט, ניתן להשתמש בפעולה של "capitalize!" המאפשרת לשנות את המחרוזת המקורית עצמה.

```Ruby
"hello world".capitalize!
=> "Hello world"
```

עוד דרך לשלוט בפעולת הקפיטלייז נוצרת כאשר נשתמש בפעולת "split", שמאפשרת לנו לפצל את הטקסט למערך ולאפשר לנו לטפל בכל מילה בנפרד על ידי שינוי ראשית האות לאות רישא ושאר האותיות לאותיות קטנות.

```Ruby
"hello world".split.map(&:capitalize).join(' ')
=> "Hello World"
```
