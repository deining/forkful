## למה

התקשרות באמצעות בקשת HTTP היא חלק בלתי נפרד מפיתוח אפליקציות ואתרים ברשת. באמצעות שליחת בקשת HTTP ניתן לקבל נתונים משרתים חיצוניים ולאפשר למשתמשים לבצע פעולות כמו לשלוח מידע או להתחבר לחשבון.

## איך לבצע?

כדי לשלוח בקשת HTTP בעזרת פרוטוקול ה-HTTP, ניתן להשתמש בסירטוני RJoll או כל ספרייה אחרת שתאפשר לנו לפתח בקשות ולקבל תגובות מהשרת. לדוגמה, בשימוש בספריית Net::HTTP בשפת Ruby, אנו יכולים להנות מפונקציות נוחות כדי ליצור ולשלוח בקשות, כגון:

```Ruby
require 'net/http'

# יצירת אובייקט חיבור לכתובת יעד HTTP
uri = URI('http://samplewebsite.com')

# שליחת בקשת GET לכתובת הנתונה
response = Net::HTTP.get(uri)

# הדפסת תוצאת הבקשה
puts response
```

ניתן לראות בדוגמה איך אנו בונים אובייקט חיבור לכתובת הנתונה ושולחים בקשת GET על-ידי קריאה לפונקציה המתאימה. כמו כן, אם ניכנס לשרת עם שם משתמש וסיסמא, ניתן להשתמש בפונקציה שליחת בקשה POST כדי לספק אותם.

## צלילה עמוקה

אנו יכולים להשתמש במגוון של פרוטוקולים שונים כדי לשלוח בקשות HTTP לשרתים חיצוניים, לדוגמה HTTP, HTTPS או FTP. בנוסף, ניתן לכתוב תנאי בוליאני בפונקציה המאפשרת לנו לטפל בתגובות שונות מהשרת - לדוגמה, במקרה של קבלת קובץ HTML משרת, ניתן לבצע פעולות מתאימות כדי לעקוב אחרי הרצת התוכנית.
