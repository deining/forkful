## למה

מחברת מחרוזות היא כלי חשוב וקריטי בתכנות בשפת רובי. השימוש במחרוזות מאפשר לנו לשלב טקסט מגוון וליצור מחרוזות ארוכות ומיושנות. בכתבה זו נלמד כיצד להשתמש במחברת מחרוזות בצורה מהימנה ויעילה.

## איך

נתחיל עם כמה דוגמאות פשוטות של חיבור מחרוזות באמצעות הפעולה `+` והפעולה `concat`:

```Ruby
str1 = "שלום"
str2 = "עולם"
puts str1 + str2
# פלט: שלוםעולם
puts str1.concat(str2)
# פלט: שלוםעולם
```

ניתן גם להשתמש בתווי מקרו (`#{}`) כדי לכלול בתוך מחרוזת ערך של משתנה או פעולה:

```Ruby
x = 5
puts "המספר #{x} הוא חמישה"
# פלט: המספר 5 הוא חמישה
puts "הסכום של 2 ו-3 הוא #{2+3}"
# פלט: הסכום של 2 ו-3 הוא 5
```

אם נרצה לחבר מספר מצביעים או אובייקטים, נוכל להשתמש בתורת השמש (string interpolation) של רובי על ידי שימוש בתווי `%{}`:

```Ruby
arr = ["שלום", "עולם"]
puts "הסידרה הראשונה של המערך היא %{arr[0]}, והסידירה השנייה היא %{arr[1]}" % {arr[0]: arr[1]}
# פלט: הסידרה הראשונה של המערך היא שלום, והסידירה השנייה היא עולם
```

ניתן לחבר מספר מחרוזות במקום במיקום באמצעות הפעולה `insert`:

```Ruby
str = "שלוםעולם"
puts str.insert(5, " ")
# פלט: שלום עולם
```

בנוסף, אם נרצה לחבר מחרוזת ריקה למחרוזת קיימת, ניתן להשתמש בפעולה `<<` באופן הבא:

```Ruby
str = "שלום"
puts str << ""
# פלט: שלום
```

לבסוף, ניתן להשתמש בפעולת הפלטוי (shovel operator)
