# למה
ניתן להשתמש בחיבור מחרוזות בשפת רובי על מנת לשרשר מספר מחרוזות לחכמה אחת, מה שיכניס סדר ותבונה לקוד שלנו.

## איך לעשות זאת
כדי לבצע חיבור מחרוזות ברובי, אנו נעזרים בפעולה "+" אשר תמתן לנו את האפשרות לחבר מספר מחרוזות יחד. נפעיל את הפעולה על המחרוזת הראשונה, ונציב בין הסימנים ואת המחרוזת השניה. לדוגמה:

```ruby
string1 = "שלום"
string2 = "עולם"
puts string1 + string2
```

פלט:
```
שלוםעולם
```

בנוסף, אפשר להשתמש בפעולת "+" בין מחרוזות למשתנים אחרים על מנת לשרשר אותם יחד וליצור מסר דינמי. לדוגמה, אם אנחנו רוצים להדפיס מסר תבררות לאחר מילות היום, נוכל להשתמש בפעולה הבאה:

```ruby
word = "יום"
puts "מה נעשה עכשיו? כן, זהו, היום הוא " + word + " טוב!"
```

פלט:
```
מה נעשה עכשיו? כן, זהו, היום הוא טוב!
```

## עיון עמוק
בנוסף לשימושים הבסיסיים של החיבור של מחרוזות, ניתן להגיע לרמה נמוכה יותר של עבודה עם מחרוזות על ידי חלוקת המחרוזת לתווים יחידים והשתמש בפעולות נוספות כמו מעבר לתווים מיוחדים או פיצול המחרוזת למערך.

לדוגמה, אם אנחנו רוצים להחליף את כל המופעים של המילה "מחר" במחרוזת, נוכל לעשות זאת באמצעות הפעולה "gsub". הפעולה תחפש את המחרוזת הנתונת ותחליף אותו במחרוזת המבוקשת. לדוגמה:

```ruby
string = "התור לארוחת ערב במסעדה נמוך מאוד. מחר נאכל שם!"
