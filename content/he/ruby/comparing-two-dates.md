## למה 

על מה דבר אנחנו רוצים לשווא מתאריכים? אנחנו רוצים להשוות שני תאריכים כדי לבדוק אם הם מופיעים באותו יום או חודש או שנה. זה עשוי להיות שימושי בעבודת תכנות כאשר נרצה לבדוק אם אירועים מתרחשים באותו תאריך או שנה.

## כיצד לעשות את זה 

בואו נלמד כיצד להשוואת שתי תאריכים בשפת Ruby. נתחיל עם פונקציה פשוטה שתכניס שני תאריכים ותחזיר תוצאה בוליאנית אם התאריכים זהים או לא. נשתמש בפונקציה `==` שממש אומרת "שווה ל". נבדוק את המתנהגות הבאה: 

```Ruby 
def compare_dates(date1, date2) 
  date1 == date2 
end 

puts compare_dates("03/23/2020", "03/23/2020") # פלט: true 
puts compare_dates("03/23/2020", "04/23/2020") # פלט: false
``` 

הפלט של הפונקציה יהיה `true` אם התאריכים שווים ו`false` אם הם לא. כעת נראה שניהם זהים אבל עם פורמטים שונים ונשתמש בפונקציה `==` כדי להשוות אותם:

```Ruby 
puts compare_dates("23/03/2020", "03/23/2020") # פלט: false
```

אז למרבה המזל, הפונקציה כבר מסוגלת להתמודד עם פורמטים שונים של תאריכים. כעת נרצה להשוות את התאריכים לפי נתונים נוספים כמו יום בשבוע וחודש: 

```Ruby 
date1 = Date.new(2020,3,23) 
date2 = Date.new(2020,4,23) 

puts date1.day == date2.day # פלט: false 
puts date1.month == date2.month # פלט: false 
puts date1.year == date2.year # פלט: true
```

במקרה זה, נוכל להשוות בין התאריכים לפי ערכים שונים כגון היום בשבוע וחודש, ולא רק בין התאריכים עצמם.
