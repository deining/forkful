# מה זה ולמה?
להשוות שני תאריכים הוא פעולה שמשווה בין שני תאריכים כדי לבדוק אם הם זהים או אם הם מתקדמים או מאחורה זה מהשני. תהליך זה מאוד חשוב בתכנות ונעשה באמצעות כלים פנימיים של השפה או ספריות חיצוניות.

# איך לעשות זאת:
להלן מספר דוגמאות של קודים ותוצאות: 
``` Ruby
date1 = Date.new(2021, 12, 31)
date2 = Date.new(2022, 1, 1)
date1 == date2 #=> false 
date1 < date2 #=> true 
```
אתם יכולים לראות שקיבלנו תוצאות שונות כאשר השווינו בין שני תאריכים לעומת הערכים האמיתיים שלהם. כמו כן, ניתן להשתמש בפונקציות נוספות כמו `before?` ו-`after?` כדי לבדוק את הזמן המושתם בין שני תאריכים.

# חקירה עמוקה:
בקשר להיסטוריה, השוואת תאריכים התפתחה כדי לנהל ולטפל בתאריכים בתצורה מסודרת ויעילה. כיום, מומלץ להשתמש בספריות חיצוניות כגון `Date` או `Time` שמאפשרות לנו לבצע פעולות כמו השוואה, חישובי תאריכים ועוד.

ישנם גם אלגוריתמים אחרים להשוואת תאריכים, כגון חישוב מרחק הזמן בין שני תאריכים, שמשתמשים בחישובי זמן ליחידת המידה המתאימה כגון שניות או דקות.

למי שמתעמק בנושא, ניתן למצוא מידע נוסף על דרכים להשוות תאריכים באמצעות אלגוריתמים ושימוש בספריות חיצוניות.
