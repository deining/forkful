## למה
בעולם התכנות, יש לנו כמעט תמיד לעשות עבודה עם תאריכים. כשצריך להשוות שני תאריכים על מנת לבדוק אם הם זהים או אם האחד מגיע לפני השני, יכול להיות לנו כיף בלתי נתפס למדי. אם אתם עובדים עם השפת תכנות רובי, אל תדאגו - ישנן כמה דרכים פשוטות לבצע השוואה בין תאריכים ותוכלו ללמוד אותם במהדורה הנוכחית.

## איך לעשות זאת
אם אתם רק מתחילים עם רובי, הנה כמה דוגמאות קוד כדי להבין איך לבצע השוואה בין תאריכים:

```Ruby
# יצירת תאריך 
date1 = Date.new(2021, 5, 20) # תאריך ראשון: 20/05/2021
date2 = Date.new(2021, 5, 25) # תאריך שני: 25/05/2021

# השוואת תאריכים בתוך תנאי אם
if date1 == date2 
  puts "התאריכים זהים"
else 
  puts "התאריכים שונים"
end

# השוואת תאריך עם תאריך נוכחי
today = Date.today # תאריך נוכחי
if date1 > today 
  puts "התאריך 1 יותר מוקדם מהתאריך הנוכחי"
else 
  puts "התאריך 1 פחות מוקדם מהתאריך הנוכחי"
end
```

הפלט של הדוגמאות הנ"ל יהיה:

```
התאריכים שונים
התאריך 1 פחות מוקדם מהתאריך הנוכחי
```

## טיול עמוק
ברוב המקרים, כשאנחנו משווים בין שני תאריכים, אנחנו רוצים לדעת אם התאריך האחד קודם או לאחר התאריך השני. למטרות כאלה, ניתן להשתמש בשיטות הפעולה "before?" ו-"after?":

# יצירת תאריכים עבור השוואה

date2 = Date.new(2021, 5, 25) # תאריך שנ
