## מדוע

מתרגמים תאריך למחרוזת בפייתון נדרש כדי להציג את התאריך בפורמט הרצוי עבור המשתמש. על פי הצורך, ניתן להציג את התאריך בפייתון בפורמט של חודש/יום/שנה או כל פורמט אחר שנדרש.

## כיצד

כדי להמיר תאריך למחרוזת בפייתון, יש להשתמש בפונקציה `strftime` שמחזירה את התאריך בפורמט מבוקש. לדוגמה, ניתן להשתמש בקוד הבא:

```Ruby
require 'date'

today = Date.today
puts today.strftime("%m/%d/%Y")
```

פלט:

```
05/13/2019
```

כפי שניתן לראות בקוד המתואר, ניתן להגדיר את הפורמט של המחרוזת המבוקשת כארגומנט לפונקציה `strftime`. מיקום התווים בפורמט מציין את תחילת האותיות הרלוונטיות - `m` מציין את החודש, `d` מציין את היום ו`Y` מציין את השנה.

## מעמיקים

הפונקציה `strftime` מאפשרת גם לייצג יום בשלמות שלו, על ידי השתמשות בפורמט `%A` או `%a`. בנוסף, ניתן להציג את שם החודש במקום מספר החודש באמצעות `%B` או `%b`.

נוסף לכך, ניתן להשתמש גם בפורמט `%H` כדי לייצג את השעה בתבנית 24-שעות ובפורמט `%I` כדי לייצג את השעה בתבנית 12-שעות.

למידע נוסף על פונקציית `strftime`, ניתן להתייחס למסמך הרשמי של פייתון.
