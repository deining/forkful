## מדוע

כתיבת קובץ טקסט היא חלק חשוב וקריטי בתהליך התכנות בכל שפת תכנות. באמצעות כתיבת קובץ טקסט, ניתן ליצור ולאחסן טקסט, קבצים, מידע ומידע מתוקשר לשימוש בתוך התוכנית שלנו. בכתיבת המאמר הזה, אנחנו נעסוק בכתיבת קובץ טקסט בשפת התכנות Gleam.

## איך לעשות זאת

כתיבת קובץ טקסט בשפת Gleam היא פשוטה ומהירה. להלן דוגמא קוד ליצירת קובץ טקסט בשפת Gleam:

```Gleam
file := File.create("my_text_file.txt")
File.write(file, "Hello, world!")
```

בקוד הזה, אנו מגדירים משתנה חדש בשם "file" ומשתמשים בפונקציה create של המודול File כדי ליצור קובץ טקסט חדש בשם "my_text_file.txt". לאחר מכן, אנו משתמשים בפונקציה write כדי לכתוב את הטקסט "Hello, world!" לקובץ הטקסט שיצרנו.

הנה הפלט של התוכנית בצעדים: 
1. יש ליצור קובץ טקסט חדש בשם my_text_file.txt
2. לכתוב את הטקסט "Hello, world!" לקובץ הטקסט שנוצר בשלב הראשון.

כעת, אם נבצע את הקוד הזה ונרוץ אותו בתוך תוכנית Gleam, ניתן לראות שהטקסט "Hello, world!" יופיע בתוך הקובץ הטקסט שיצרנו.
