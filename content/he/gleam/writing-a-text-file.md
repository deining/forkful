## למה

למה לכתוב קובץ טקסט? התשובה היא יחסית פשוטה - הצורך לשמור נתונים ולאחסן מידע בצורה מקוננת ומסודרת. בעזרת קבצי טקסט, ניתן ליצור תיעוד ומידע שימושי לכל מי שישתמש בקוד שלנו. כמו כן, ניתן ליצור קבצי טקסט למטרות שונות כגון קבצי הגדרות, רשימות, או כמקור מידע נוסף לפיתוח.

## איך לכתוב קובץ טקסט ב-Gleam

הרעיון הבסיסי של כתיבת קובץ טקסט ב-Gleam הוא ליצור משתנה עם ערך מסוים ולהשתמש בפונקציות כדי לכתוב את התוכן של הקובץ לתוך קובץ חדש. ניתן לעשות זאת בעזרת הפונקציות "io.open" ו-"io.write" כדי לפתוח ולכתוב לקובץ בהתאמה. לנוחיותכם, הנה דוגמאת קוד:

```Gleam
  // פתיחת קובץ לכתיבה
  let file = io.open("new_file.txt", "w")
  // כתיבת נתונים לקובץ
  let data = "מידע חדש לקובץ"
  _ = io.write(file, data)
  // סגירת הקובץ
  _ = io.close(file)
```

כפי שאתם יכולים לראות, ניתן להשתמש במפתחות "let" כדי ליצור זיגוג "file" ו-"data" ולהשתמש בפונקציות לכתיבת התוכן לקובץ ולסגירתו.
