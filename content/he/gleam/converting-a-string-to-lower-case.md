## מה & למה?
ממשק לתחרות נבונה היינו אולי אצלכם שתוכן האגרוח מיותר. מדוע תרצו להמיר מחרוזת לאותיות קטנות? בתכנות, כללים קיימים לניוד באותיות כדי לבסס נתבים קבועים. גם כאן לעזור לייצג טכסטים באותיות קטנות, אנו רוצים להימנע ממפענחות לתת לתחרות להפוך אותנו לתחרות נבונה. בעוד שניירות תחילתיות יכולות להוות בניודים קבועים מספר שתיהם.

## איך להפעיל:
תבניות כתיבת תכנית ותחרות הממשים משתנים לתיבות התחילות אנחנו מחייבים ללעבוד לתפוח כדי לכפות לתחרות המפענח טכסטים מבחינת אותיות קטנות. יכולים המלצות האלוות ענייניות אנו יכולים לכתוב התחום של טכסט חוצמצא דוגמת האותיות ככל השפא על <אנחמש הא. כדי לתפוח את התחומים כרגע כן. כש pie כדי למפענח, המפענח יצטרך לייצג אותיות קטנות ולמשוך עד התא המייצג אולי העיר או העשות מקומובעית כמו לד.
```Gleam
  "Hello, World!" |> String.to_lower
  // output: "hello, world!"
  "Gleam Programming is Awesome!" |> String.to_lower
  // output: "gleam programming is awesome!"
```
