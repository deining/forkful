## למה

למה להעסיק בהמרת מחרוזת לאותיות קטנות? קודם כל, ישנם מצבים רבים שבהם ייתכן שנקבע כי התחביר הנכון יהיה אך ורק באותיות קטנות. לרבות, אתרי אינטרנט יכולים לטבל את הנתונים שלכם על פי אותיות קטנות בלבד.

## איך לעשות זאת

בכדי להמיר מחרוזת לאותיות קטנות ב-Gleam, תוכלו להשתמש בפונקציה הפשוטה String.to_lower המאפשרת לנו להעביר מחרוזת ולקבל את אותה מחרוזת באותיות קטנות חזרה. נהיה מסוגלים להשתמש בפונקציה זו על מחרוזות בכל הפורמטים האפשריים ב-Gleam כמו UTF-8, ASCII ועוד. ראשית, נצרך להתחבר לתחילת מחרוזת.

```Gleam
let input = "Hello, World!"
```

כעת נשתמש בפונקציה String.to_lower כדי להמיר את המחרוזת לאותיות קטנות.

```Gleam
let output = String.to_lower(input)
```

התוצאה של אישור שימוש בפונקציה String.to_lower תהיה:

```Gleam
"hello, world!"
```
