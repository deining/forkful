## למה

אם אתם מתחילים לכתוב קוד בשפת גלים, חשוב לדעת שבמקרהים רבים תתקלו בהצורך להמיר את האותיות במחרוזת לאותיות קטנות. זה נדרש למספר סיבות שונות, כגון תיקון טעויות או להשוות מחרוזות באופן יעיל. בכתיבת המאמר הזה, נסביר לכם איך לבצע פעולה זו בשפת גלים.

## איך לעשות

המרת מחרוזת לאותיות קטנות בשפת גלים פשוטה ונוחה. הנה דוגמאות של קוד ופלט שיכולים לעזור לכם להבין את הפעולה בצורה טובה יותר:

```Gleam

string.to_lower("Hello world") // פלט: "hello world"

string.to_lower("ABCD") // פלט: "abcd"

string.to_lower("4098") // פלט: "4098"

```

כפי שאתם רואים, פעולת ההמרה מכילה את המחרוזת המקורית ומייצרת מחרוזת חדשה עם אותיות קטנות בלבד.

## הפעולה בעומק

אם אתם מעוניינים להבין קצת יותר על פעולת ההמרה בעומק, כאן יש מספר נקודות שיכולות לעזור לכם:

- בשפת גלים, מחרוזת היא סוג משתנה שמייצגים טקסט. מחרוזת היא לא סוג משתנה אם היא לא נכתבת בצורה נכונה (לדוגמה: מחרוזת נוצרת כאשר סותרים גרשיים או תווים כפולים).

- פעולת ההמרה משנה את אותיות המחרוזת המקורית לאותיות קטנות בלבד. התווים האחרים במחרוזת - מספרים, רווחים, סימנים - נשארים כפי שהם.

- אם אתם מעוניינים להמיר את האותיות לאותיות גדולות, תוכלו להשתמש בפעולת `string.to_upper` במקום `string.to_lower`.
