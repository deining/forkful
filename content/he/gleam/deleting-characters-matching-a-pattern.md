# למה
כשמתקשים לאתר תווים מסוימים בתוך מחרוזת, ניתן להשתמש בפונקציה המובנית למחיקת תווים ב-Gleam עם תבנית נתונה. זה יכול לסייע לך לייעל את תהליך הקוד במקרים שבהם יש צורך להסיר תווים מיוחדים או להחליף אותם.

# איך לעשות את זה
אם תרצו למחוק תווים מתוך מחרוזת עם תבנית מסוימת ב-Gleam, תוכלו להשתמש בפונקציה המובנית `String.replace()` ולהכניס אליה את התבנית והארגומנטים הרלוונטיים. כדי להציג את התוצאה, ניתן להשתמש בפונקציה `IO.inspect()` ולהדפיס את המחרוזת המעודכנת.

 ```Gleam
 let original_string = "ברוכים הבאים למאמר החדש שלנו!" 
 let pattern = /[!]/ 
 let updated_string = String.replace(original_string, pattern, "")

 IO.inspect(updated_string) 
 ```

 תוצאה:
 ```למאמר החדש שלנו```

 בדוגמה זו, אנו משתמשים בתבנית שמתאימה לתו הסופי של המחרוזת כדי למחוק אותו מהמחרוזת. תוכלו גם להשתמש בתבניות אחרות כגון תווים מיוחדים או תבנית מולטיליינית.

# עומק נוסף
כאשר משתמשים בפונקציה `String.replace()` עם תבנית מסוימת, יש לזכור שהתבנית תחלוף את כל התווים שתואמים לה במחרוזת. לכן, חשוב לבדוק את התבנית שנבחרה כדי לוודא שהמחרוזת המעודכנת תהיה בדיוק מה שציפיתם.
