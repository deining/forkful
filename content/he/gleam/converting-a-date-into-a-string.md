## למה

המרת תאריך למחרוזת היא משימה נפוצה בתכנות ועשויה להיות חיונית אם אנחנו רוצים להציג תאריכים בפורמטים שונים. בכתיבת פוסט זה, אנחנו נכיר איך ניתן לבצע את המשימה הזאת בשפת פיתוח Gleam.

## איך לעשות את זה

נתחיל עם דוגמא פשוטה של מרת תאריך למחרוזת ב-Gleam:

```Gleam
import gleam/time

time.format(gleam/time.Date(2021, 11, 11), "%d/%m/%Y")
```

בקטע הקוד הזה, אנחנו משתמשים בפונקציית `format` של ערכת הכלים "gleam/time" כדי להמיר את התאריך שנמצא בתוך רשימת הפרמטרים למחרוזת בפורמט שנקבע על ידי פרמטר השני. הפורמט בגוף הפונקציה מורכב מתווי ההנחייה של התאריך (יום, חודש ושנה) ביחד עם תווי נקודה שמופרדים על ידי סלש (/).

הנה פלט הקוד הזה:

```
"11/11/2021"
```

כמובן שניתן להשתמש בפורמטים שונים בתוך פונקציית `format` על ידי שינוי תווי ההנחייה והתווים המפרידים. לדוגמא:

```Gleam
time.format(gleam/time.Date(2021, 11, 11), "%Y-%m-%d")
```

תפוקת הקוד הזה תהיה:

```
"2021-11-11"
```

רעיון נוסף הוא להשתמש בפונקציות נוספות של ערכת הכלים "gleam/time" כדי לקבל תאריכים משונים, לא רק את התאריך הנוכחי. לדוגמא:

```Gleam
time.format_now("%d/%m/%Y")
```

הקוד הזה יחזיר את התאריך הנוכחי בפורמט המבוקש.
