## למה

רק 1-2 משפטים המסבירים *למה* מישהו ירצה למצוא את אורך מחרוזת.

המציאות היא שקוראי ולימקינים לא ממש אוהבים לעבוד עם מחרוזות. לא משנה אם אתם מפתחים בשפת גלים או בשפות אחרות, העבודה עם מחרוזות נחשבת למטרף ולא נעימה. כשמדובר באורך של מחרוזת, לפעמים זה יכול להיות רכישה קשה כדי להבין מספר רב של תווים ולהחזיר את ערכם האמתי. אם אתם מחפשים איך למצוא את אורך המחרוזת בשפת גלים, אתם במקום הנכון.

## כיצד לעשות זאת

לפניכם מגוון דוגמאות שונות המדגימות כיצד למצוא את אורך מחרוזת באמצעות קוד גלים.

```Gleam
string_length :: String -> Int
string_length(str) {
  let len = 0
  for _ in str {
    len = len + 1
  }
  len
}

string1 = "Hello"
string2 = "שלום"

string_length(string1) // Outputs 5
string_length(string2) // Outputs 4
```

בדוגמה זו, אנו משתמשים בפונקציה "string_length" המקבלת את המחרוזת כארגומנט ומחזירה את אורך המחרוזת כמספר שלם. בפנים של הפונקציה, אנו משתמשים בבקרת זרימה "for" כדי לעבור דרך כל תו במחרוזת ולהוסיף 1 למשתנה המכיל את האורך. לבסוף, אנו מחזירים את ערך האורך הכולל.
