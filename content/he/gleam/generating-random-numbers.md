# למה:

לוחםי התכנות מכירים היטב את חשיבות המספרים האקראיים בשפות תכנות. הם משמשים ליצירת קלט מקרי, ניסויים ותרגילים מתמטיים כמו גם ליצירת משחקים ואתגרים מגוונים. ב-Gleam, יצירת מספרים אקראיים נעשית בצורה פשוטה ויעילה, כך שתוכלו להוסיף אותם לכל פרויקט שתבחרו.

# איך לעשות זאת: 

ב-Gleam ישנן שתי פונקציות שמאפשרות ליצור מספרים אקראיים: `random.float` ו-`random.int`. לפניכם דוגמאות לשימוש בשתי הפונקציות ליצירת מספרים אקראיים בטווח מסוים:

```Gleam
import random

random.float(0.0, 1.0)
// מספר אקראי בין 0.0 ל-1.0 (לא כולל).

random.int(1, 10)
// מספר אקראי בין 1 ל-10 (כולל).
```

בנוסף, אפשר להשתמש בפונקציה `random.seed` כדי לשנות את הזירע של המספרים האקראיים וליצור תוצאות שונות בכל פעם.

# נכנסים לפרטים: 

פונקציית `random.float` מחזירה מספרים עשרוניים אקראיים בין 0.0 ו-1.0, לא כולל. ניתן להגדיר טווחים שונים על ידי מתן פרמטרים שונים לפונקציה.

פונקציית `random.int` מחזירה מספרים שלמים אקראיים בין שני ערכים מסוימים, כוללים את הערך הראשון והשני. גם כאן, ניתן להגדיר טווחים שונים על ידי העברת פרמטרים לפונקציה.

כדי להשתמש בפונקציות האלו, יש לייבא את המודול random ולהוסיף אותו כפונקציה ווירטואלית בפורמט
