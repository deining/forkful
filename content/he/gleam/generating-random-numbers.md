## למה: 
רק 1-2 משפטים המסבירים *למה* מישהו ירצה לבצע יצירת מספרים אקראיים.

בתור מתכנתים, אנו תמיד מתנסים בבניית מערכות ותוכניות שמשתמשות במספרים אקראיים. לעיתים, אנו צריכים ליצור פרמטרים רנדומליים למטרות כמו בדיקות יחידות או ביצועיים. באמצעות לימוד איך ליצור מספרים אקראיים בגלים, יהיה ברור לנו את הנושא הזה ואיך ליצור פשוט בניית תוכניות.

## כיצד לעשות:
באמצעות גרסת גלים 0.16 ומעלה, ניתן ליצור מספרים אקראיים בקלות על ידי שימוש במודול Rand. הנה דוגמא פשוטה של איך ליצור מספרים אקראיים בגלים:

```Gleam
import gleam/rand

fn main() {
  let random_number = rand.int(1, 10)
  IO.print("Your random number is: #{random_number}")
}
```

בקוד הנ"ל אנו משתמשים בפונקציות Rand כדי ליצור מספר אקראי בין 1 ו-10 ולהדפיס אותו למסך. כל פעם שפועלים את הקוד, נקבל תוצאה אחרת. הנה מספר דוגמאות נוספות ליצירת מספרים אקראיים בגלים:

- ליצור מספר אקראי בין 5 אלף ל-10 אלף: 
`rand.int(5000, 10000)`

- ליצור מספר אקראי בין 0 ל-1: 
`rand.float(0.0, 1.0)`

- ליצור תו אקראי בין a ל-z: 
`rand.char(97, 122)`
