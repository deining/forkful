## למה

למתכנתים רבים, הפעולה של יצירת מספרים אקראיים עשויה להתרחש כמשימה פשוטה וללא תועלת מיוחדת. אך בעולם התכנות, יצירת מספרים אקראיים היא כלי מאוד חשוב לגידול אפליקציות ושיפור פירוריות ברמת הקורדינציה וביצור נתונים משמעותיים לצורך בדיקה.

## איך לעשות זאת

### ניתוח ודאי

דרך פשוטה ויצירתית ליצור מספרים אקראיים היא לפנות לפונקציות "random" הסטנדרטיות בשפת ג'ליס. לדוגמה, אם נרצה ליצור מספר אקראי בין 1 ל-100, נוכל להשתמש בפונקציה "int" ולכתוב:

```Gleam
import random

random.int(1, 100)
```

כעת, אם נרצה ליצור מספר ממוצע בין 2 ל-10, נוכל להשתמש בפונקציה "float" ולכתוב:

```Gleam
import random

random.float(2.0, 10.0)
```

את המספרים אקראיים אפשר להכניס למשתנים כדי להשתמש בהם בכל נקודה אחרת בקוד.

### שיטת קונצנטציה ריא לופשלהל הדד

דרך נוספת ליצירת מספרים אקראיים היא להשתמש בפונקציות שמשתמשות בשיטת קונצנטציה ריא לופשלהל הדד ("Xorshift" בשם הקוד). פונקציות אלו מבצעות פעולות ספציפיות על מספרים קבועים שמשתנים בכל פעם שמפעילים אותן. לדוגמה, ניתוח לולאת שלש צעדים כדי לייצר מספר קבוע המשתנה בכל פעם יוצא אותו מספר.

```Gleam
import random

random.seed()
random.next_float()
```
