## למה

למגוון סיבות, מפתחי תוכנה עשויים להשתמש בהדפסת פלט של מידע דיבאג על מנת לנתח ולפתור בעיות בקוד שלהם. זהו כלי חשוב שיכול לסייע במציאת טעויות ובהבנת הפעולות של התוכנה שלנו. בפוסט הזה, נלמד כיצד להדפיס מידע דיבאג בשפת גלים.

## איך לעשות את זה

המתכנתים שמשתמשים בשפת גלים יכולים להדפיס פלט של מידע דיבאג בקלות באמצעות פונקציית `gleam.debug`. להמחשה, הנה דוגמה של כיצד להדפיס את ערכי המשתנים שלנו בתוך קוד שלנו:

```Gleam
fn print_debug_info() {
  let name = "John"
  let age = 30
  gleam.debug(name)
  gleam.debug(age)
}
```

כשנריץ את הפונקציה `print_debug_info`, נקבל את הפלט הבא:

```
John
30
```

הפונקציה `gleam.debug` מקבלת את הערך של המשתנה או התוצאה שאנחנו רוצים להדפיס ומדפיסה אותו לפלט. אפשר גם להדפיס מסרים ידנית על ידי פסיקים בין המחרוזת לבין המשתנה, כך:

```Gleam
gleam.debug("The value of the name variable is:", name)
```
