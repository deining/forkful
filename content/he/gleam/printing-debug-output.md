## איך לעשות:
עם גלים, פלט לאתריודים ניתן להדפיס באמצעות הפקודה `רייסתמועה`. ראשית, בשביל להשתמש בפקודה זו, עליך לייבא את הספריה המתאימה להדפסת פלט כמו כובש `רייסתמועה`:

```
רימועת אוין גלים

gr_ego `rsco מוד_section;

fn לחזור (ver: פכל) -> פכל {
  רוכל (מוצאמוד, נו גל);
  // צוות הפקודה `רייסתמועה` יודע להדפיס
  רייסתמעות!("מוצאמוד: {}", מוצאמוד);
  נו פכל;
}
```
