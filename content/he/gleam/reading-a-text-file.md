## למה

קריאת קובץ היא סיסמה חשובה בתוך התכנות המודרני ותוכנות המחשב בכלל. אנשים קוראים קובץ בעיקר כדי להבין מה שכתוב בו, לחפש מידע או למצוא חפצים מסוימים. אין ספק שקריאת קובץ היא חלק חשוב בתהליך התכנות ובאפשרותינו לייצר תוכניות קוד איכותיות ויעילות.

## איך לעשות זאת

קריאת קובץ ב-Gleam היא משימה פשוטה ונמצאת בין התכונות החשובות ביותר של השפה. הכי פשוט להיעזר בקוד לדוגמה על מנת להבין איך זה עובד בפועל:

```Gleam
let file = File.read("example.txt")
```

הקוד הזה יקרא קובץ טקסט בשם "example.txt" ויבחן את התוכן שלו. התוצאה תוכנן כנתוני מחרוזת ותוכלו לעבוד איתם בצורה פשוטה יותר.

לדוגמה, אם נרצה לפרק את התוכן לשורות בקובץ ולהדפיסם, נוכל לעשות זאת באמצעות הפונקציה `String.split()` וללולא על כל השורות באמצעות הפונקציה `List.each()`:

```Gleam
let lines = String.split(file, "\n")

List.each(lines) { line ->
  IO.print(line)
}
```

כעת נוכל להציג את התוכן של הקובץ במערך של שורות בקונסול על ידי השתמשות בפונקציות מובנות בשפת Gleam.
