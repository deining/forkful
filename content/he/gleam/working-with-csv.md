## למה

CSV הינו פורמט פופולארי לאחסון נתונים רבים בצורה נוחה וקריאה. לעשות שימוש ב-CSV נחשב כמוויתור אווראי בעולם התכנות וניתן להשתמש בו בכמעט כל שפת תכנות. ללמוד כיצד לעבוד עם CSV יכול להיות מועיל לכל מפתחי התוכנה.

## איך לעבוד עם CSV

בחלק זה נדגים כיצד לעבוד עם CSV בעזרת שפת התכנות Gleam. ישנם שני נסיונות שונים לקריאה של CSV וכיתובה לתוך קובץ טקסט פשוט וקל להבנה. נשתמש בחבילת `gleam/csv` כדי לקרוא ולכתוב נתונים לקובץ CSV.

```Gleam
import gleam/csv

// יצירה של קובץ CSV חדש
let file = csv.encode([["שם", "גיל"], ["יוסי", "30"], ["רחל", "25"]])

// קריאה מקובץ CSV קיים
let data = csv.decode(file)

// כתיבה לקובץ CSV
let updated_data = [["שם", "גיל"], ["יוני", "40"]] ++ data
let updated_file = csv.encode(updated_data)
```

פלט הקובץ יראה כך:

```text
שם,גיל
רחל,25
יוסי,30
יוני,40
```

## העברה עמוקה

CSV הינו פורמט פשוט אך חזק שימושי לניהול נתונים. ניתן להתאים אותו לקובץ טקסט רגיל בלי שימוש בתוספות מיוחדות כמו ניתוח פתיחות, ולכן הוא נחשב לפורמט מועיל עבור מידע גדול ומורכב.

ניתן להתאים CSV גם למטרות מורכבות יותר כגון עריכת נתונים במסדי נתונים או ליצור דווחים ותעודות מפורטים. העובדה שניתן לכתוב ולקרוא CSV בכמעט כל שפת תכנות גוררת שימוש רחב ועשיר בפורמט הזה.
