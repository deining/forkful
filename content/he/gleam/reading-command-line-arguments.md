## למה

אתה יכול לכתוב אפליקציות מדהימות ב-Gleam, אבל כדי להשתמש בהן, עליך להשתמש בפרמטרים שהמשתמש ימסיק לתת לאחר הפעלת התוכנית. למשתמשי Gleam ידוע שישנם שפורמטים נהדרים שבאפשרותך להשתמש בהם על מנת לכתוב אפליקציות, ואם אתה רוצה להימנע משימוש בממשק כמו גלית, אתה צריך לקרוא פרמטרים מפקודת קו הפקודה כדי לגשת אליהם. במאמר זה אנו נתאר את הצורה הטובה ביותר לקרוא את הפרמטרים.

## איך לעשות זאת

כאשר אתה משתמש בגלית כמשנה ל- Gleam, עליך להשתמש בפונקציות כמו `cmd_line.args()` כדי לקרוא את הפרמטרים שניתן הרבה לתוכנית שלך.

```Gleam
args = cmd_line.args()
io.format("הפרמטר הראשון הוא: {}", [args[0]])
```

הקוד הנ"ל ישמור את הפרמטר הראשון שהמשתמש הכניס לתוכנית וידפיס אותו. כעת כאשר אתה מפעיל את התוכנית שלך, תוכל להעזר בפרמטרים נוספים כדי לעשות משהו נוסף. לדוגמה, אם אתה רוצה ליצור קובץ עם השמות של משתמשים, אתה יכול להשתמש בפרמטר כדי לקבל את שם המשתמש מהמשתמש. הנה דוגמה לקבלת שם משתמש כפרמטר ולהשתמש בו על מנת ליצור קובץ עם השם ככותרת.

```Gleam
args = cmd_line.args()

file_name = args[0]
file = file.open(file_name)
file.write_line("User: {}", [args[1]])
file.close()
```
