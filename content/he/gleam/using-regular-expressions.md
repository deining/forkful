# למה

רק 1-2 משפטים להסביר *למה* מישהו יהיה מעוניין להשתמש בביטויים רגולריים.

לרוב, תכנתנים משתמשים בregular expressions כדי לסדר ולחפש מילים מסוימות בקבצים או טקסטים מאוד גדולים. הם נמצאים בשימוש באופן תדיר בעזרת פונקציות חיפוש כגון פונקציות גוגל והצלחה, אבל יכול להיות גם שימוש משמעותי עבור פונקציות זו.

# איך

## דוגמאות של קוד

נסו להתחבר לכתובת דוא"ל כאשר תשתמשו בכתובת חיפוש של השרת. השורות הבאות מדגימות כיצד ניתן להשתמש בוהם כדי לחפש כתובת אימייל חוקית:

``` Gleam
ه٥٥םםםם::מתאימים::דואר_אכברכת.כווני-??? *@gmail.com
```

מכיוון שכל כתובת דוא"ל מכילה את הסמל "@" ואח"כ סיומת חוקית, הביטוי הרגולרי מתאים לכל כתובת דואר אלקטרוני של Gmail.
