אינטרנט מאפשר לנו לעשות מגוון רחב של פעולות דיגיטליות, ואחת מהן היא יצירת קבצים זמניים. במאמר הזה נלמד ליצור קובץ זמני בשפת גלים, ונוודא שהתהליך הוא קל וברור.

## למה
יצירת קבצים זמניים הוא פעולה חשובה שניתן לעשות בכדי לנהל ולהעביר מידע בין כמה תוכניות או דפי אינטרנט. לרוב המטרה היא לשמור נתונים שאותם תוכניות או דפי אינטרנט זקוקים לעבוד איתם, ואז למחוק אותם כדי לחסוך מקום ולשמור על פרטיות.

## איך לעשות זאת
בשפת גלים, ניתן ליצור קבצים זמניים באמצעות הפונקציה "tempfile.create". אם נזין בתוכה את השם של התיקייה שבה נרצה ליצור את הקובץ הזמני, היא תצור באופן אוטומטי קובץ זמני בתוך התיקייה הזו. אם נשאיר את הפרמטר ריק, הקובץ ייווצר בתיקייה הזמנית של המערכת הפעילה (בדרך כלל “/tmp” ב-Linux).

ניתן לכתוב את הקוד בשתי שורות פשוטות בכדי ליצור קובץ זמני ולהדפיס את נתוניו:

```Gleam
import tempfile
file = tempfile.create()
io.print("The temporary file is located at:", file)
```

### פלט:
```
The temporary file is located at: /tmp/GLP2498.tmp
```
