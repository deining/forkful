כיצד ליצור קובץ זמני ב־Gleam

## Why

יצירת קבצים זמניים היא חלק חשוב מולצת בתהליך כתיבת קוד. אתה יכול להשתמש בהם כדי לבדוק ולבנות קוד חדש ולאחר מכן למחוקם. כמו כן, קבצים זמניים מתאימים במיוחד לטיפול בנתונים רבי–מדידה.

## How To

כדי ליצור קובץ זמני ב־Gleam, יש להשתמש בפונקציה IO.TempFile. קח, לדוגמה, את הקוד הבא:

```Gleam
file, err := IO.TempFile("my_dir", "prefix", "ext")

IO.puts("קובץ נוצר במיקום " ++ file) // דוגמה להדפסה

IO.inspect(err) // דוגמה לטיפול בשגיאות
```

פונקצית TempFile מקבלת שלושה פרמטרים: מיקום, תחילית וסיומת. בדוגמה שלנו, אנחנו מנסים ליצור קובץ בתיקייה my_dir עם השם prefix וסיומת ext. הפונקציה תחזיר את המיקום של הקובץ החדש ואם יש בעיה ביצירתו, היא תחזיר גם פרמטר של שגיאה.

בכדי לבדוק מה התוכן של הקובץ שנוצר, נציג את הקוד הבא בתוך הוואקציה שלנו:

```Gleam
data := File.read(file)

// data תתסביר לנו כעת את תוכן הקובץ שנוצר זמנית
```
