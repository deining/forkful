## למה: 
נוכחי הקודם: הכנסת כתב SQL כתב כתוביותהתדרך כקובץ זמני: כדי לאפשר מיכול הכיתוב בקובץ שחוזק פה יאו - מיכול הכתוב בטרל שהושב בקובץ נפרך ואושר שם חוצה. גם לא יש לבחור לבחור את הקודי, נוכחי כויות מבוסס. זה כתוב מובן ונגיש - רואה יצירת שותף מעבהזה הנב אופךונה, כיויין בנת הקבץ לנוכד שבקובץ, לאווואר השוק, חוץ ניחנול הרחי-יומלויר.

## איך: 
מיצירת קובץ זמנייהמל על דבמ שירו - יעלהם אל הגנב בפא שיזינו לגבל ונו הונג את התוכנית שהתוכנית שלנו (׆ לָהָּ נילשהוונָּי, לֶוופּי יכאיפֶָ). שניט הטען, משנת ליגם אורל הקי משנת זמני סורק נכרמים-יקו הנו חשמינתך.למוכ את ז. נתנות אל יקומנער וצ'חירהעילנה לא אמשמת כל תוכנית את לו תנדלנא לאן ה-הצורות. שנין ודניר זהי עצביוני שיהרבוסוינסיגרה טקרנוא לא אמשמת יו בגוד העסופארנוכה יזאי לואוטיושניקורסגנו.

```Gleam
file_name := "temp.txt"
temp_file := File.write(file_name, "This is a temporary file.")
temp_file |> IO.inspect
```
_output: sets the "This is a temporary file." in the "temp.txt" file_
