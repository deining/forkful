# למה

ישנם מצבים רבים שבהם יכול להיות נדרש ליצור קובץ זמני בתוכנית שלך, לדוגמה כאשר אתה צריך לשמור נתונים בזמן ריצת התוכנית או לשתף נתונים בין תהליכים שונים. יצירת קובץ זמני בגלים יכול להיות דרך יעילה לטפל במצבים כאלה.

## איך לעשות זאת

ליצור קובץ זמני בגלים פשוט מאוד, כל מה שאתה צריך לעשות הוא להשתמש בפונקציה `File.temp/2` ולספק את שם הקובץ המבוקש ואת התוכן שברצונך להוסיף לקובץ. הנה משלוח דוגמה של יצירת קובץ זמני בגלים וכיצד הוא ייראה בתוך קוד בגלים:

```Gleam
let temp_file = File.temp("my_data.txt", "Hello, World!")
```

תוצאה:

```
my_data.txt
Hello, World!
```

## ירידה עמוקה

יצירת קובץ זמני בגלים משתמשת בפונקציות מתוך הספרייה המובנית של גלים `File`. ניתן לצפות מהם להפעיל פעולות כגון יצירת קבצים והוספת נתונים לקבצים קיימים. בנוסף, יש בימינו ספריות חיצוניות המאפשרות טיפול בקבצים מסוגים שונים, כגון CSV ו-JSON. אם אתה רוצה ללמוד עוד על יצירת וניהול קבצים בגלים, עלינו לבדוק את הקישורים הבאים:
