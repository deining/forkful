על מה: הכתיבה לפלט אירעת מידע היא דרך אפקטיבית לזהות בעיות מתן האפשרות לשיפור השגיון ושגיאות בקוד.

איך לעשות זאת: תוכנית קוד ופלט דוגמה בתוך ```Gleam ... ``` בלוקי קוד.

פלט דוגמה עבור כתיבה לפלט אירעת:

```Gleam

fn main() {
  // פקודה שגויה עם כתיבת העיר והמדינה השגויה
  let city = "תל אביב"
  let country = "ישראל"
  
  // כתיבה לפלט אירעת עם פרטים על הבעיה
  error("העיר ", city, " לא משויכת למדינה ", country);
}
```

פלט קונסולה:

```
Line 5: העיר תל אביב לא משויכת למדינה ישראל
```

לחכות עם כתיבה לפלט עד לסוג הפקודה המתאים והעלאת התכונות של התכנית שלך בשימוש כדי לזהות כי החשיבה שלך הותיקה ולפתור את הבעיה באופן יעיל.

תהליך העמוק: זהו הצעד הבא בפיתוח הכלרי של הפריטים. תרופות יהיו בעלות של נתונים עם השונה בזמן. השגיון יהיה כזה שאתה לא יכול להתאים את הפריטים למספרי כתיבות הערים וערים להיתר תת קו התכונה עם המפרט הורדת אחת מבאת. לדוגמה, הנתונים של שנים בהם אתה תמיד תנסה לאתר את הכתובת של העיר םעב'פלקת הגרסה. כתיבת לפלט עם הודעות של כתיבת התכונים הכלליים המכילים על שם את כילכל דוגמה.
