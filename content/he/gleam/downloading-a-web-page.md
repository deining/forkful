## למה

הורדת דף אינטרנט היא תהליך חשוב עבור כל מתכנת אינטרנט. בעזרת התהליך הזה, ניתן לקבל מידע ונתונים מאתרים ולהשתמש בהם לצרכים שונים, כגון ניתוח מידע, כתיבת בוטים ועוד. חשוב לדעת כיצד להוריד דף אינטרנט בצורה יעילה ובקלות, וזה בדיוק מה שנלמד במאמר הזה.

## כיצד לבצע הורדת דף אינטרנט בעזרת Gleam

כדי לבצע הורדת דף אינטרנט בעזרת Gleam, יש לכתוב קוד פשוט המבצע את הפעולות הבאות:

```Gleam
// טעינת המודולים הנדרשים
import gleam_http
import gleam_url

// הגדרת כתובת דף האינטרנט להורדה
let url = "https://www.example.com"

// ביצוע בקשת GET לכתובת דף האינטרנט
let response = url |> gleam_url.parse |> gleam_http.get

// כתיבת תוכן הדף לקובץ חדש
"new_page.html" |> gleam_file.write(response.body)
```

בשורה הראשונה, אנו מטעין את המודולים gleam_http ו-gleam_url הנדרשים עבור התהליך. לאחר מכן, אנו מגדירים כתובת דף האינטרנט שנרצה להוריד ומבצעים בקשת GET לכתובת זו בעזרת הפונקציה gleam_http.get. לבסיס התשובה שמתקבלת, אנו יכולים לבצע פעולות נוספות, כגון כתיבת תוכן הדף לקובץ חדש.
