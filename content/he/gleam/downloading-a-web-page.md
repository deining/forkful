# למה

הורדת דף אינטרנט היא חלק חשוב מתהליך התפתחות כל אתר אינטרנט. בעזרת גלים, ניתן לכתוב קוד בקלות המאפשר להוריד תוכן מייצג של דף אינטרנט ולעבוד איתו. בפוסט הזה נלמד איך להשתמש בגלים כדי להוריד דף אינטרנט ולבצע פעולות על התוכן שלו.

## איך לעשות זאת

לפני שנתחיל, נצטרך להתקין את גלים ואת חבילת האינטרנט. ישנם כמה צעדים שיש לבצע כדי לקבל גישה לכלי הפיתוח של גלים. לאחר מכן נוכל להתחיל לכתוב קוד שיאפשר לנו להוריד דף אינטרנט.

```Gleam
import gleam/http
import gleam/io

url = "https://example.com" // כתובת האתר שנרצה להוריד

// בתוך פונקצית main נתקשר לאתר על ידי התחברות חד-פעמית (One-Time Password) ונשמור את תוכן הדף בתוך משתנה response
fn main() {
  response = http.get(url)
  .body(response, &io.from_bytes)
}

// בשלב הבא נוכל להתחיל לעבוד עם התוכן של הדף, למשל להדפיס אותו למסך או לבצע תיקונים עליו לפני שנשמור אותו.
```

כעת, נוכל להריץ את הקוד ולראות את תוצאת ההורדה של הדף האינטרנטי במסך.

## טיול עמוק

גלים מאפשר לנו להשתמש במגוון כלים להורדת דפים אינטרנטיים, כולל תוכניות כתיבת קוד באמצעות שפות תכנות נוספות כמו Erlang ו-Elixir. ניתן להשתמש בכלי הפיתוח של גלים כדי לבנות אינטרפייסים חזקים להורדת דפים אינטרנטיים ולפענח תוכן מורכב של אתרים.
