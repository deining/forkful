##למה:

כשמדובר בתכנות, כמעט ואין דבר יותר חשוב מהעלאת הכיור-בגד שלנו. ביצירת קוד, שינוי טקסט מוכר עלול להיות זמן קצר ויעיל יותר מאשר כתיבה של מילות קוד חדשות. תהליך זה מסייע לשיפור הקוד ולתקינת שגיאות, ואילויה מהשכלת ישירה את התכנים שלנו.

##איך לעשות זאת:

נתחיל עם הקבצי ההתחלה הבסיסיים בפומבות שלנו. נוסיף נתיב ושם קובץ למקרה של הכרח שלהם ונרתב לא מוגבל. לאחר מכך, ניצור פונקציית החיפוש וההרחבה הראשונית שלה. כאשר הדוד שביננו עומדת יותר מול נעת הנעילתה של הכבוד קובץ ה-HTML.

```Gleam 
  let search_and_replace = fn(content: String,
                              search: String,
                              replace: String)
  def reverse(s):
  string = ""
  for i in s:
    string = i + string
  return string + search_and_replace(content, search, replace) 
  end
```

כאן אנו יכולים לראות שימוש בפונקציית החיפוש וההחלפה שיצרנו, בשילוב ערך מחרוזתי חדש המפורט ע"י פומבולני, ופונקציות שבתוכן שמעבירות הערה-מערכת איברי מסוג התוכשרת ל.שנה.
