## למה

הקיצור המחרוזת הוא כלי שימושי עבור מתכנתים המעוניינים להפרד חלקים מסוימים של מחרוזת. זה יכול לסייע בעיבוד ומיפוי של נתונים מורכבים כגון כתובות אימייל או תעודת זהות. בכתבה זו נלמד את כיצירת קוד ב- Gleam להחלקת מחרוזת וכיצד זה יכול להיות שימושי עבורך.

## כיצד לעשות

הנה דוגמאת כתיבת קוד בשפת Gleam לחילוץ תת-מחרוזות:

```Gleam
import gleam/string

fn main() {
  let str = "Hello World"
  let substring = string.slice(str, 0, 5)
  // substring יהיה "Hello"
}
```

הפונקציה slice מקבלת ארגומנטים של מחרוזת, אינדקס התחלה וסופי ומחזירה את תת המחרוזת בטווח הנתון. ניתן להשתמש גם בפונקציות נוספות כגון string.split כדי לחלק את המחרוזת לפי תבניות נתונים מסוימות. 

להלן דוגמאות נוספות של כך כיצד להשתמש בתכונות הקיצור של מחרוזת בשפת Gleam:

```Gleam
import gleam/string

fn main() {
  let str = "JavaScript is awesome"
  let start_with_js = string.starts_with(str, "JavaScript")
  // start_with_js יהיה true
  let end_with_awesome = string.ends_with(str, "awesome")
  // end_with_awesome יהיה true
}
```
