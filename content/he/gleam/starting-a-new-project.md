# על מה למה

יצירת פרויקט חדש בג'לים יכולה להיות אתגר מרתק ומאתגר עבור מתכנתים שמעוניינים להיות בראש התעדיפים בעולם התכנות. באמצעות ג'לים, ניתן להפיק את המקסימום מכמה שעוד עיקרון הפתרונות המופשט של השפה.

## איך לייצר

מתחילה, ניתן להתחיל על ידי יצירת סקיפולדרספרייט בג'לים. לדוגמה:<br>
```Gleam
struct Person(name, age) {
  fn say_hello() {
    IO.print("שלום, שלום! אני " ++ name ++ " ואני בן " ++ age);
  }
}

let person = Person("דני", 25);
person.say_hello(); // שלום, שלום! אני דני ואני בן 25
```

בנוסף, ניתן להוסיף כל מיני פונקציות נוספות לסקיפולדרספרייט כדי להתאים אותו יותר לצרכים שלנו. למשל, נוכל להוסיף פונקציה המדפיסה צילום פונה של הודעת הברכה האישית של האדם בעזרת `fn print_greeting() { ... } `.

## שקיעת ראש בעמקים

ליצירת סקיפולדרספרייט ייחודי, יש להוסיף פונקציות של פונקציות בתוך הפונקציות של הסקיפולדרספרייט דרך `impl` באמצעות המילה השמורה `impl`, כך שהסקיפולדרספרייט יהיה מתאים יותר לכל מיני תכניות שונות.

ניתן לקרוא לפונקציות אלה בקוד מחוץ לסקיפולדרספרייט באמצעות המשתנה של הסקיפולדרספרייט, למשל: `person.print_greeting()`.

לפרסם את הסקיפולדרספרייט ניתן להפעיל את הפקודה `mix compile` ולהשתמש בקובץ היצא שנוצר כדי להחליט את הפרויקט שלנו.
