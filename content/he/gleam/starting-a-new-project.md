## למה
מפרויקטים חדשים יש כוח להפתיע ולבחון דברים חדשים. אם אתה מתעניין בתכנות ורוצה ללמוד שפת תכנות חדשה, Gleam היא בדיוק מה שאתה צריך כדי לבשל את הידע שלך ולהשתבט במכשפת הקוד החדשה.

## איך לעשות זאת
אנחנו נדגים איך להתחיל פרויקט חדש בגלאם עם דוגמאות של קוד ופלט תקין. נתחיל כל קוד עם "```Gleam ... ```" כדי להראות לך איך להשתמש בגלאם לכתיבת קוד ברור ויעיל.

### התחלת פרויקט חדש
ראשית, נצטרך להתקין את Gleam במחשב שלך כדי להתחיל לייצר פרויקטים חדשים. אם אתה משתמש במערכת הפעלה Mac או Linux, תוכל להשתמש בפקודה הבאה ב-Terminal כדי להתקין את Gleam:
```
curl -sL https://raw.githubusercontent.com/gleam-lang/gleam/master/scripts/installer.sh | sh 
```

אם אתה משתמש במערכת הפעלה Windows, ישנם מדריכים מפורטים להתקנת Gleam באתר הרשמי של Gleam.

עכשיו, נוכל ליצור פרויקט חדש באמצעות הפקודה הבאה:
```
gleam new project_name
```

זה יגניב את כל הקובץ והמחלקות הנדרשות כדי להתחיל עם פרויקט חדש. כעת, תוכל להתחיל לכתוב קוד עבור הפרויקט בתוך התיקייה project_name.

### כתיבת פונקציות
דבר ראשון שעלינו לעשות כאשר נפתח פרויקט חדש הוא ליצור פונקציות. לדוגמה, ניצור פונקציה פשוטה המחשבת את סכום של שני מספרים:
```
fn add(x, y) {
  x + y
}
```

כעת, באמצעות הפקודה `gleam build` תוכל לכתוב את התוכנית הזו לתוך קובץ בתיקייה project_name\src\main
