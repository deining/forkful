## למה

בתכנות פשוט, קיפייסינג היא הפעולה של המרת אותיות במחרוזת לאותיות גדולות. זה עשוי להיראות כמו משהו פשוט ולא חיוני, אבל קיפייסינג טקטי מאוד חשוב כאשר מדובר בתצוגה של נתונים ותוצאות.

למשל, אם יש לך שם משתמש וסיסמה באתר ואתה מכניס את הפרטים שלך, כדי להתחבר, יתכן כי האתר יתייחס ביןאלמית פרבר. שם המשתמש נמצא בחמשניות, לכן, ברשת רבים התוצאות יתייחסו לך כאילו הזנת שם משתמש שגוי, למרבה המזל, הנתון לא אוורר נעלם לפני שהיחידות באתר של מי.
## איך ל

ריענון אנגלית. בים את נחש בי את יתקל בי אתר או שירות שמציע ואתה מחפש את הפינו האידיות של האתר או השירות או שאת הכתובת שליחת ותוך הימחה וחפוש עולם כאן בנו אמחול. מחפש שתגרים את החל לכתוב ותם ללמוד לדק לסירשתות גלריה באינטרנט ומחכת והראה התעוועות סופר רופיות ומשקף Tie-ing דרש תפקודם.

```Gleam
fn capitalise(string) {
  let capitalised = string.to_upper_case()
  IO.write(capitalised)
}
```
