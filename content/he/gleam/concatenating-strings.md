## למה
ברור שבתכנות יש פעמים שאתה צריך לחבר יחד כמה טקסטים כדי ליצור משהו מתחדש ומעניין. בגלים, תהליך החיבור של טקסטים נקרא "תחביר" או concatenation ומשמש כלי חשוב למרבה המטרות כך שניתן ליצור קוד יעיל ומובן במיוחד בשילוב עם פונקציות אחרות. 

## איך לעשות זאת
כדי לחבר טקסטים נשתמש בפעולת התחביר שקיימת בגלים. ניתן להשתמש בפעולת התחביר באמצעות האופרטור המיוחד ++. ניתן להניח שיש לנו שני משתנים מסוג string ונרצה לחבר אותם לפי הסדר שאנחנו רוצים:
```Gleam
let first_name = "ישראל"
let last_name = "כהן"

let full_name = first_name ++ " " ++ last_name
```
כאן אנחנו משתמשים ב++ לחבר את שני המחרוזות יחד ולשמור את התוצאה במשתנה חדש בשם full_name. כעת אם נדפיס את המשתנה הזה:
```Gleam
IO.println(full_name)
```
נקבל את התוצאה:
```
ישראל כהן
```

## חקר עמוק
עכשיו שאנחנו יודעים איך לחבר טקסטים בגלים, נוכל לחקור כיצד התחביר עובד למעשה. הפעולה ++ בגלים אינה פעולה במשמעות המילה, אלא היא אופרטור המשמש לחיבור של שני מחרוזות. יתרה מכך, התחביר גם עובד עם משתנים מכל הסוגים, כך שאפשר לחבר משתנים אחד לשנייה בכל זמן נתון. כמו כן, התחביר יכול להיות שימושי לפירוש המחרוזת המכילה מספרים, בעזרת המתודה ``` to_string``` ניתן להמיר את המספר למחרוזת ולחבר אותה למחרוזת אחרת על מנת לייצר מילון מבולגן שמ
