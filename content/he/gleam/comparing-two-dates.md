טקסט כותרת "למה":

בכדי לנתח נתונים תאריכים ולהשוותם באופן יעיל, ניתן להשתמש בשפת תכנות Gleam. היא מאפשרת לנו לכתוב קוד קריא ואינטואיטיבי כדי לבצע השוואה מדויקת של שתי תאריכים. 

טקסט כותרת "איך לעשות":

תחילה, נציין את הספריות הנחוצות להשוואת תאריכים בשפת Gleam: `gleam/datetime` ו- `gleam/calendar`. כאשר משתמשים בשתי הספריות הללו, ניתן להשוות את השעות, הימים, החודשים והשנים בקלות. 

לדוגמה, ננסה להשוות בין שתי תאריכים בשימוש בקוד הבא:

```
Gleam import gleam/datetime 
import gleam/calendar 

let now = Datetime.now(UTC) // תאריך נוכחי 
let birthday = Datetime.from_calendar(1990, April, 30) // תאריך הלידה שלי 

let result = Calendar.compare_dates_now(now, birthday)

match result {
  Less -> Some("תאריך הלידה שלי נמצא לפני התאריך הנוכחי") 
  Equal -> Some("לידתי נפלה בתאריך נוכחי") 
  More -> Some("תאריך הלידה שלי נמצא אחרי התאריך הנוכחי") 
}
```

כאשר נריץ את הקוד הזה, נקבל כתוצאה את המשפט הבא: "תאריך הלידה שלי נמצא לפני התאריך הנוכחי". כך ניתן להשוות בין שתי תאריכים בשימוש בשפת Gleam.

טקסט כותרת "מעומקים":
