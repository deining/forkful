# क्यों

किसी को ये समझना के लिए आवश्यक है कि ये लोग यादृच्छिक संख्याओं की उत्पत्ति क्यों करते हैं।

यादृच्छिक संख्याओं की उत्पत्ति विशेषतः सरल, स्वतंत्र, और निष्पक्ष है। ये अनुसंधान, संग्रहीत डेटा एवं गणना दस्तावेजों को सिमुलेट करने और अनुकूलित करने के लिए उपयोगी हैं। इसलिए, ये उत्पन्न विशिष्ट मापदंडों तय करने में बहुत उपयोगी हैं।

# कैसे करें

```` Rust
use rand::Rng; // यादृच्छिक संख्या उत्पत्ति के लिए डायनामिक जीन।

fn main() {
    let num = rand::thread_rng().gen_range(1, 101); // 1 से 100 के बीच संख्या उत्पन्न करें
    println!("आपका यादृच्छिक संख्या है: {}", num); // आउटपुट: आपका यादृच्छिक संख्या है: 42
}
````

आपको डायनामिक जीन को अपने कोड में शामिल करना होगा जो `rand::thread_rng().gen_range()` द्वारा आपको एक मनचाही सीमा के बीच एक यादृच्छिक संख्या देगा। आप मनचाहे सीमा नंबर को बदल सकते हैं या आप अपने कोड में अन्य यादृच्छिक संख्या के उत्पादन के लिए अन्य फ़ंक्शन का भी उपयोग कर सकते हैं।
