# क्यों
कोई व्यक्ति दिए गए पैटर्न से मेल खाने वाले अक्षरों को हटाने में रुचि रखता होगा, उन्हें दो या अधिक स्तंभों में ठीक से संयोजित करने की आवश्यकता हो सकती है। इसके अलावा, कोड को साफ़ और उपयोगकर्ता के आसान और तेजी से समझने में मदद मिल सकती है।

## कैसे करें
```Rust
fn main() {
    let input_str = "H,a,ppy Hol,idays";
    let pattern = ',';

    let output: String = input_str.chars().filter(|&c| c != pattern).collect();

    println!("{}", output);
}

// आउटपुट: Happy Holidays
```
ऊपर दिए गए कोड में, `input_str` में दिए गए स्ट्रिंग को हम अपने विशेष पैटर्न, यानी कमा के आधार पर फिल्टर कर रहे हैं। इसके बाद, `collect` फ़ंक्शन की मदद से अलग-अलग शब्दों को फिल्टर करने और नई स्ट्रिंग बनाने का काम किया जा रहा है। फिर हम `println` फ़ंक्शन की मदद से नया शब्द दाखिल करते हैं। हमारा फाइनल आउटपुट "Happy Holidays" होगा। यहां, हमारा पैटर्न अपने स्तंभ को हटा देता है, जिससे आउटपुट में केवल शब्द बचता है।

## गहराई में खोज
इस कार्य में गहराई में खोज करने के लिए, हमें `for` लूप का उपयोग करना हो सकता है, जो एक-एक करके हर एक के साथ चलता है और हर एक करके उन्हें फिल्टर कर सकता है। यदि हम इस तरह के अलावा, `replace` फ़ंक्शन का भी उपयोग करते हैं, तो हम अपने पैटर्न को दिया गया शब्द से बदल सकते हैं।
