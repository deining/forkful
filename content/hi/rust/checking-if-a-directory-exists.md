## क्यों

आज के दूर में निर्माताओं को फाइल और फोल्डर पर अधिक नियंत्रण चाहिए होता है। इसलिए, रब्बरा में फोल्डर की उपस्थिति की जांच एक महत्वपूर्ण कार्य है। यह आपको निश्चित बनाता है कि फोल्डर वास्तव में है या आप इसे अब बना सकते हैं।

## कैसे करें

फोल्डर की उपस्थिति की जांच करने के लिए आपको रब्बरा में लाइब्रेरी केंद्र को इंस्टॉल करने की आवश्यकता होगी। इसके अलावा, आप निम्नलिखित कोड की मदद से फोल्डर की उपस्थिति को जांच सकते हैं।

```Rust
use std::fs;

let dir_name = "my_folder";

if fs::metadata(dir_name).is_ok() {
    println!("The folder exists.");
} else {
    println!("The folder does not exist");
}
```
इस कोड में, हमने सामान्य लाइब्रेरी का उपयोग किया है जो फोल्डर के मेटाडेटा को जांचता है और उसे उपलब्ध होने पर "The folder exists" और न होने पर "The folder does not exist" कहता है।

## गहराई में जाएँ

फोल्डर की उपस्थिति की जांच के लिए रब्बरा आपको अन्य तरीकों से इस्तेमाल करने की अनुमति देता है। आप एक फाइल या फोल्डर के उपस्थिति को भी जांच सकते हैं। इसके अलावा, आप इस आर्टिकल के नीचे दिए गए सामग्री को भी देख सकते हैं जो आपको अधिक जानकारी देगी।
