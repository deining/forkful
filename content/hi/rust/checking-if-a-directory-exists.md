## क्यों

क्या आपने कभी सोचा है कि आपको किसी डायरेक्टरी के अस्तित्व को जांचने की आवश्यकता हो सकती है? अगर आप रस्ट प्रोग्रामिंग में काम कर रहे हैं तो यह समस्या आपके सामने आ सकती है। इस ब्लॉग पोस्ट में, हम आपको यह बताएंगे कि डायरेक्टरी के अस्तित्व को कैसे जांचा जा सकता है और इसमें काम करने के लिए आपको क्या करना चाहिए। 

## कैसे करें

सबसे पहले, हम अपने कोड में `std::fs` मॉड्यूल को इम्पोर्ट करेंगे। यह मॉड्यूल फाइल और डायरेक्टरी से संबंधित ऑपरेशन को प्रदान करता है। उसके बाद, हम फाइल पथ को स्ट्रिंग के रूप में दर्ज करेंगे जो हमारी जांच करने के लिए है। फिर, हम `PathBuf` से एक डायरेक्टरी बनाएंगे और उससे `exists()` फंक्शन को कॉल करेंगे। `exists()` फंक्शन डायरेक्टरी के अस्तित्व को जांचता है और एक `bool` वापस देता है। 

```Rust 
use std::fs;

fn main() {
    let file_path = "test_directory";
    let directory = fs::PathBuf::from(file_path);
    if directory.exists() {
        println!("यह डायरेक्टरी मौजूद है!");
    } else {
        println!("यह डायरेक्टरी मौजूद नहीं है।");
    }
}
```

यदि आपका दर्ज किया गया फ़ाइल पथ मौजूद होता है, तो कोरलोग "यह डायरेक्टरी मौजूद है!" प्रिंट होगा और अन्यथा कोरलोग "यह डायरेक्टरी मौजूद नहीं है।" प्रिंट होगा।
