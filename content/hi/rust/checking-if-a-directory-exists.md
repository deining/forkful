# क्यों

क्या आपने कभी सोचा है कि अपने कंप्यूटर पर किसी निश्चित संदर्भ में एक निर्दिष्ट डायरेक्टरी की जांच करने का अवसर बन सकता है? 

जब हम एक कार्यक्रम लिखते हैं, तो हमें आमतौर पर उन कार्यक्रमों को डायरेक्टरी में सहेजने की आवश्यकता होती है. इसलिए, हमें अपने कार्यक्रम में एक निश्चित डायरेक्टरी का अस्तित्व निर्धारित करने के लिए एक चयनित संदर्भ को जांचने की आवश्यकता हो सकती है। 

# कैसे करें

```Rust
use std::fs;

let directory = "my_directory";

// जांचें कि क्या डायरेक्टरी मौजूद है
if fs::metadata(directory).is_ok() {
    println!("{} मौजूद है", directory);
} else {
    println!("{} मौजूद नहीं है", directory);
}
```

उपरोक्त कोड विधि में, हमने fs::metadata नामक एक आँकड़ा विधि का उपयोग किया है जो उपयोगकर्ताओं को जनित मूल्य को जज्ञात करने में मदद करती है। यदि डायरेक्टरी अस्तित्व में है, तो कोड लूप। अन्यथा, एक संदेश दर्शाया जाएगा। 

# गहराई तक

जब हम पहली बार अपने कोड का लोग करते हैं, हम अक्सर डायरेक्टरी के अस्तित्व की जांच के लिए शामिल विधियों का उपयोग करते हैं। इससे हमें अपने कोड में अतिरिक्त गलतियाँ और bugs को पता चलने का मौका मिलता है जो हमारा काम आसान बनाता है।
