## इसका कारण

डायरेक्टरी उपस्थित है या नहीं, इसके बारे में जानने का कारण है क्योंकि हम अपने रास्ते को सुनिश्चित करना चाहते हैं कि उसमें स्वीकृत फ़ाइलें मौजूद हैं। 

## कैसे करें

```rust
use std::path::Path;

let path = Path::new("example_directory");

if path.exists() {
    println!("डायरेक्टरी मौजूद है!");
} else {
    println!("डायरेक्टरी मौजूद नहीं है।");
}
```

उपरोक्त कोड उदाहरण के रूप में दिया गया है जो देखने की क्षमता सभी संभावित डायरेक्टरी नामों को जांचेगा। यदि डायरेक्टरी मौजूद है, तो "डायरेक्टरी मौजूद है!" का संदेश प्रिंट होगा। अन्यथा, "डायरेक्टरी मौजूद नहीं है।" का संदेश प्रिंट होगा।

## गहराई में जाएं

अब हम इस विषय में थोड़ी गहराई तक चलें। अक्सर हमें यह प्रश्न हो सकता है कि डायरेक्टरी के स्वीकृत फ़ाइलों की संख्या कैसे जाने। इसके लिए हम `read_dir` फ़ंक्शन का उपयोग कर सकते हैं जो डायरेक्टरी में स्वीकृत फ़ाइलों की एक इटरेटर वापस करेगा। प्रत्येक स्वीकृत फ़ाइल के लिए, हम इसके नाम और पूर्ण पथ को प्रिंट कर सकते हैं।

```rust
use std::fs;

let path = "example_directory";

if let Ok(files) = fs::read_dir(path) {
    for file in files {
        if let Ok(file) = file {
            println!("{}", file.file_name().to_string_lossy());
            println!("{}", file.path().display());
        }
    }
}
```

ऊपरोक्त कोड को एक और `for` लूप के साथ बढ़ा कर आप स्वीकृत फ़ाइलों को संख्या भी पायेंगे।
