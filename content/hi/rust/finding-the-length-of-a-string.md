## क्यों

कॉम्प्यूटिंग में, हमें क्‍या करने के लिए चाहिए कि पता हो कि स्ट्रिंग की लंबाई क्या है। यह हमें प्रोग्राम को अनुकूलित करने और स्ट्रिंग को सही से उपयोग करने में मदद करता है।

## कैसे करें

आप Rust में अपनी स्ट्रिंग की लंबाई पाने के लिए निम्नलिखित कदमों का पालन कर सकते हैं:

```Rust
// स्ट्रिंग बनाएं
let my_string = "हेलो दुनिया";

// लंबाई जांचें और प्रिंट करें
println!("स्ट्रिंग की लंबाई है: {}", my_string.len());
```

आउटपुट:

```
स्ट्रिंग की लंबाई है: 11
```

## गहराई में जाएं

हमारे द्वारा बनाई गई स्ट्रिंग को रखने के लिए, Rust एक चोखी बाइट वर्णमाला का उपयोग करता है। अतः, हमारे द्वारा दिखाई गई स्‍ट्रिंग की लंबाई बाइटों में होगी। यदि हम एक वर्ग का प्रयोग करते हैं, तो हम अपनी स्‍ट्रिंग के साथ काम कर सकते हैं और लंबाई को अनिश्चित (unknown) बनाएं:

```Rust
// स्ट्रिंग बनाएं
let my_string = "Hello Rust!";

// बाइट वर्णमाला में लंबाई
println!("स्ट्रिंग की लंबाई है: {}", my_string.as_bytes().len());
```

आउटपुट:

```
स्ट्रिंग की लंबाई है: 11
```
