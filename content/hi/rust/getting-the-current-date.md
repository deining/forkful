## उसका कारण

क्या आप कभी सोचा है कि हम अपने कंप्यूटर पर दिनांक और समय को कैसे पता करते हैं? आजकल हम सब अपने डिजिटल जीवन के साथ जुड़े हुए हैं और दिनांक और समय को अपने दैनिक कामों के लिए जानना बहुत महत्वपूर्ण है। इस ब्लॉग पोस्ट में हम आपको बताएंगे कि आप अपने रस्ट प्रोग्राम में दिनांक को कैसे प्राप्त कर सकते हैं।

## कैसे करें

दिनांक को प्राप्त करने के लिए आपको रस्ट के इनबिल्ट मोड्यूल `std::time` का उपयोग करना होगा। इसके लिए आप `chrono` का उपयोग कर सकते हैं, जो एक बहुत ही उपयोगी थर्ड पार्टी लाइब्रेरी है। `chrono` में आपको अनेक तरीकों से दिनांक को प्राप्त करने का विकल्प मिलेगा। नीचे एक सरल उदाहरण दिया गया है जो आपको समझने में मदद करेगा।

```rust
use chrono::prelude::*; // chrono लाइब्रेरी को इम्पोर्ट करें

fn main() {
    let local: DateTime<Local> = Local::now(); // वर्तमान स्थानीय समय प्राप्त करें
    println!("{}", local); // उपयोगकर्ता को दिनांक और समय को मुद्रित करें
}
```

आउटपुट:
```
2021-12-07 10:30:24.123456+05:30
```

आप इसी तरह से `Utc::now()` या किसी अन्य तरीके से भी दिनांक को प्राप्त कर सकते हैं। `chrono` की और गहराई तक जानकारी के लिए, आप `DateTime`, `TimeZone` और `Duration` वैल्यूज़ को समझ सकते हैं।
