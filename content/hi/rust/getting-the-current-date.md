## क्यों

वर्तमान तारीख को प्राप्त करना क्यों महत्वपूर्ण है? धन्यवाद, हिंदी भाषाओं के पाठकों! उस बनाए रखने के लिए मौका मिलता है। शुरू करते हैं!

## कैसे

मॉड्यूल `chrono` モジュール के साथ आप तारीखों को उपयोग कर सकते हैं। प्रोग्राम को शुरू करने से पहले, `chrono` को `Cargo.toml` के माध्यम से लोड करें।

```rust
# उदाहरण
[dependencies]
chrono = "0.4"
```

तारीख प्राप्त करने के लिए `Local` और `now()` को हटाएं।

```rust
# उदाहरण
use chrono::{Local, now};

let वर्तमान_तारीख = Local::now();
println!("वर्तमान दिनांक: {}", वर्तमान_तारीख);
```

फिर से श्रृंखला जोड़ें, `strftime` जो मूल्य को.

```rust
# उदाहरण
println!("वर्तमान मिनट नाम: {}",
        वर्तमान_तारीख.strftime("%M"));
```

आउटपुट:

```
वर्तमान दिनांक: 2021-10-12 10:24:30.956253600 +05:30
वर्तमान दिनांक: 24
```
