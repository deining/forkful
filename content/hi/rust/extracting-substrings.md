## क्यों
क्या आपको कभी लगा है कि आपके पास एक लम्बा और असमंजस में डैटा स्ट्रिंग है और आपको उसमें से कुछ खट्टे से खाने का मन हो रहा है? या शायद आपको उस स्ट्रिंग के कुछ हिस्से पर काम करना है लेकिन पूरी स्ट्रिंग को एक साथ लेने में दिक्कत हो रही है? यदि हाँ, तो आपको उस स्ट्रिंग से सबसे सटीक तरीके से कटबद्ध करने की आवश्यकता है जिसे substring extraction कहा जाता है। इस आलेख में, हम आपको Rust में substrings को extract करना सिखाएंगे और आपको एक अच्छी शुरुआत करने में मदद करेंगे।

## कैसे
```rust
let string = String::from("एक लंबा और असमंजस में डैटा स्ट्रिंग");
let substring = &string[3..6];

println!("{}", substring);
```

उपरोक्त कोड इस वाक्यांश को प्रिंट करेगा: "लंबा।", जो कि स्ट्रिंग के तीसरे से छठे अक्षरों का एक हिस्सा है। हम नीचे दिए गए उदाहरण भी देख सकते हैं जो आपको और अधिक समझ में मदद करेंगे।

```rust
// Rust में slicing
let string = String::from("Rust एक बेहतरीन प्रोग्रामिंग भाषा है।");
let substring = &string[5..];

println!("{}", substring);
```

उपरोक्त कोड इस वाक्यांश को प्रिंट करेगा: "एक बेहतरीन प्रोग्रामिंग भाषा है।", जो कि स्ट्रिंग के पाँचवें से अंत तक का हिस्सा है। अतः, आप देख सकते हैं कि हम substring को क्या समझते हैं और यह कैसे काम करता है।
