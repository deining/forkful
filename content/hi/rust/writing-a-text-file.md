## क्यों

रस्ट सबसे नयी प्रोग्रामिंग भाषा है जो की तेज़ी से उभर रही है और आपने कभी अपने रोजमर्रा के प्रोग्रामिंग या सॉफ्टवेयर डेवलपमेंट के काम में इसे सुना होगा। यदि आप अपने कॉडिंग स्किल्स को समझाना या दूसरों को समझाना चाहते हैं तो आप एक गहराई तक जाने के लिए एक सादा टेक्स्ट फाइल बनाने में रस्ट का उपयोग कर सकते हैं।

## कैसे करें

अगर आपने पहले से कभी रस्ट में कोई प्रोग्राम लिखा है, तो आप जानते होंगे कि हमेशा कुछ नया सीखना होता है। लेकिन टेक्स्ट फाइल बनाने के लिए आपको सबसे पहले एक फाइल को ओपन करना होगा और फिर उसमें हमारे कोड को लिखना होगा। 
```Rust
use std::fs::File;
use std::io::Write;

fn main() {
    let filename = "my_file.txt";
    let mut file = File::create(filename).expect("could not create file");

    file.write_all(b"Hello, world!").expect("could not write to file");
}
```

इस कोड से आप एक टेक्स्ट फाइल बना सकते हैं और "Hello, world!" टेक्स्ट भी उसमें डाल सकते हैं। अधिक जानकारी के लिए, आप रस्ट की ऑफिसियल डॉक्यूमेंटेशन को देख सकते हैं। 

## गहराई में जाएं

टेक्स्ट फाइल बनाना शायद समझने में आसान लगे, लेकिन इसके पीछे बहुत सारे तार के पार भी चलना हो सकता है। रस्ट में फाइल्स ऑपरेशन के लिए अलग-अलग मॉड्यूल्स जैसे ```std::fs``` और ```std::io``` उपलब्ध हैं जो की फाइल हैंडलिंग, लेखन, पढ़ने और अन्य ऑपरेशन के लिए
