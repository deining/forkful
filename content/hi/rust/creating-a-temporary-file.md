## क्यों

अगर आप किसी ऑनलाइन सेवा का निर्देशन देने के लिए टेम्पररी फाइल का उपयोग करते हैं, तो आप जानते ही होंगे कि यह बहुत जरूरी हो सकता है। एक अन्य कारण यह हो सकता है कि आप अपने कोड को टेस्ट करने के लिए टेम्पररी फाइल का उपयोग करते हैं।

## कैसे करें

टेम्पररी फाइल बनाने के लिए, हम रस्ट के बारे में कुछ माउन्टेनेयर्स साहित्य का उपयोग कर सकते हैं। निम्नलिखित उदाहरण में, हम एक टेम्पररी फाइल बनाने का उपयोग विशिष्ट अनुक्रमिकता से कैसे करते हैं और उसे संबंधित उत्पादनों के साथ जोड़ सकते हैं।

```Rust
use std::fs::File;
use std::io::prelude::*;

fn main() {
    // टेम्पररी फाइल बनाएं
    let mut temp_file = File::create("my_temp_file.txt").expect("फाइल बनाने में असमर्थ!");

    // फाइल में कुछ डेटा लिखें
    let data = "हैलो, यह एक टेम्पररी फाइल है!";
    temp_file.write_all(data.as_bytes()).expect("डेटा लिखने में असमर्थ!");

    // फाइल बंद करें क्योंकि हमें अभी उसे जोड़ने की आवश्यकता नहीं है
    drop(temp_file);

    // फाइल उठाएं और प्रतिलिपि बनाएं
    let mut temp_file_copy = File::open("my_temp_file.txt").expect("फाइल खोलने में असमर्थ!");
    let mut copy = String::new();
    temp_file_copy.read_to_string(&mut copy).expect("डेटा पढ़ने में असमर्थ!");
    println!("{}", copy);

    // असमय फाइल डिलीट करें
    std::fs::remove_file("my_temp_file.txt").expect("फाइल को डिलीट करने में असमर्थ!");
}
```

```
हैलो, यह एक टेम्पररी फाइल है!
```
