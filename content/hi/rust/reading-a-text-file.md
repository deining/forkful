## क्यों

क्या आपका पास भी कभी एक पाठ फ़ाइल को पढ़ने के लिए अपनी रोज़मर्रा की प्रोग्रामिंग की ज़रूरत पड़ि है? अगर हां, तो रूस्त प्रोग्रामिंग भाषा से आपको अपनी तस्वीर से बाहर आने की ज़रूरत नहीं है। रूस्त आपको फ़ाइलों को आसानी से पढ़ने और फ़ाइल में चेंज करने के लिए कई विकल्प प्रदान करता है। इस ब्लॉग पोस्ट में हम आपको दिखाएंगे कि रूस्त में तीनों साधनों को कैसे उपयोग करके आप आसानी से .txt फ़ाइलों को पढ़ सकते हैं।

## कैसे करें

पाठ फ़ाइलों को पढ़ने के लिए, हमें उन्हें पढ़ने के लिए एक फ़ंक्शन की आवश्यकता होती है। हम रूस्त के भाषा में इसे एक फ़ंक्शन ```file_read()``` के रूप में बनाएंगे। यह फ़ंक्शन हमारी फ़ाइल को खोलने, उसमें से पढ़ने और फाइल को बंद करने के लिए कई कदम लेगा। हमारे पास एक साधारण .txt फ़ाइल है जिसे हम ```hello_world.txt``` नाम से सहेजते हैं। तो चलिए इसे पढ़ना सीखें।

```
fn file_read() {
    let mut file = File::open("hello_world.txt").expect("फ़ाइल खोलने में समस्या हुई!");
    let mut s = String::new();
    file.read_to_string(&mut s).expect("फ़ाइल पढ़ने में समस्या हुई!");
    println!("{}", s);
}
```

आप ```main()``` फ़ंक्शन में नीचे दिए गए लाइनों को जोड़ कर ```file_read()``` फ़ंक्शन को कॉल कर सकते हैं।

```
fn main() {
    file_read();
}
```
