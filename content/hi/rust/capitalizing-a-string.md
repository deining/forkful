## क्यों

जेसे कि हम सब जानते हैं, रस्ट एक ताकतवर और पॉवरफुल प्रोग्रामिंग भाषा है। इसके साथ हम क्या कर सकते हैं? वेल, एक आम समस्या है कि हमें किसी स्ट्रिंग को कैपिटलाइज करना होता है। यह कोई असामान्य नहीं है, लेकिन किसी भी भाषा में स्ट्रिंग कैपिटलाइज करना थोड़ा समय लेता है। इस वेबलॉग पोस्ट में हम रस्ट की मदद से एक स्ट्रिंग को कैपिटलाइज करने के बारे में बात करेंगे।

## कैसे करें

पहला काम है `string` प्रोपर्टी `chars` का सेट. आप में रस्ट use करने से पहले उसको लोड कर सकते हैं। कोड कुछ दिखाता है जैसे कि:

```rust
fn capitalize(string: &str) -> String {
    let chars = string.chars();

    // Code to capitalize the string
    unimplemented!();
}

fn main() {
    let string = "hello";
    println!("Capitalized string: {}", capitalize(string));
}
```

उसको चलाने के लिए आप उसको अभी कॉमांड लाइन में चलाने से पहले कलेप्रोग्राम कर सकते हैं। यह कारक उपयोगकर्ता को स्ट्रिंग को कैपिटलाइज करने को देख सकता है।

```bash
$ cargo run
Capitalized string: Hello
```

पहले लाइन के बारे में कुछ सोचिए। हम कॉन्स्त्रक्टर को `main()` में डेक्लरेशन से string कॉन्वर्ट रेफिंस करते हैं। इसमें हम ड्राइव में करे और उसको खोल सकते हैं।
