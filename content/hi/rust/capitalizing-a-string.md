## क्यों

कोई भी प्रोग्रामिंग भाषा में, स्ट्रिंग (string) एक अत्यंत महत्वपूर्ण डेटा टाइप होता है। यह टेक्स्ट को अभिव्यक्ति करने के लिए उपयोग किया जाता है और इसे बहुत से काम करने के लिए उपयोग किया जाता है। इसलिए, जब हमें एक स्ट्रिंग में सभी शब्दों को कैपिटलाइज करने की आवश्यकता होती है, तो हम उसे आसानी से कर सकते हैं।

## कैसे करें

इसके लिए हम रस्ट (Rust) का उपयोग कर सकते हैं। रस्ट एक मानव-सहज भाषा है जो प्रोग्रामर्स को लाइब्रेरी, फंक्शन और मॉड्यूल के साथ एक भाषा में संग्रहित करती है। हम इसे सभी विशोधकताओं से मुक्त हैं और हमारे स्ट्रिंगों को कैपिटलाइज करने के लिए इसका उपयोग कर सकते हैं।

```rust
fn main() {
    let s = "hello world"; // दिया गया स्ट्रिंग
    let capitalized = s.to_uppercase(); // स्क्रिंग कैपिटलाइज करने की फंक्शन
    println!("{}", capitalized); // उत्पादन: HELLO WORLD
}
```

## गहराई में जाएं

जब हम एक स्ट्रिंग को कैपिटलाइज करते हैं, तो हम उसमें प्रत्येक अक्षर को अपने शब्द शुरुआति के रूप में बदलते हैं। यदि हमें उपरोक्त उदाहरण से अगले उत्पादन को प्राप्त करना हो, तो हमें इसके लिए कुछ गहराई में जाने की आवश्यकता होती है।
