## क्यों 
एक्सप्लेन करें कि बहुवर्णीकरण से लाभ क्या है और यह प्रोग्रामिंग में कैसे मदद कर सकता है।

बहुवर्णीकरण एक आम प्रोग्रामिंग कार्य है जो हमें दिए गए स्ट्रिंग को प्रविष्ट करने पर प्रथम अक्षर को बड़ा लेता है। यह आमतौर पर कहां प्रयोग किया जाता है- शब्दों को संवर्धित करते समय या उपयोगकर्ताओं को एक पदार्थ को पहचानने में मदद करते समय। प्रोग्रामिंग में सीखने के लिए यह सीखना आवश्यक है कि हम कैसे एक स्ट्रिंग को प्रोसेस करते हैं और उसे बदलते हैं। इस ब्लॉग पोस्ट के माध्यम से, हम आपको बताएंगे कि हम रस्ट में स्ट्रिंग को कैसे बहुवर्णीकरण कर सकते हैं।

## कैसे 
```रस्ट
fn main() {
  let word = "बहुवर्णीकरण";
  let capitalized = word.to_uppercase();
  println!("{}", capitalized);
}
```
आउटपुट:
```
बहुवर्णीकरण
```

ऊपर दिए गए कोड ब्लॉक में हमने दो मुख्य कार्य किया है। पहले, हमने `to_uppercase()` फ़ंक्शन का उपयोग करके वर्ड को बदलकर एक बड़े अक्षर वाली स्ट्रिंग में बदला है। और फिर हमने `println!()` फ़ंक्शन का उपयोग करके स्क्रीन पर बीच में कुछ दिखाया है। इस तरीके से, हमने बहुवर्णीकरण का खास एक सभी बड़ा अक्षर वाला आउटपुट बनाया है।
