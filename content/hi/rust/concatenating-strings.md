# क्यों

कभी कभी हमें डेटा को एक साथ जोड़ने की आवश्यकता होती है, जैसे कि एक मेसेज या पोस्ट के लिए हैशटैग बनाना। इससे हमारे कोड को और अधिक शक्तिशाली बनाने के लिए, हमें आरस्ट भाषा में स्ट्रिंग्स को जोड़ना सीखना होगा।

## कैसे करें

आरस्ट में स्ट्रिंग्स को जोड़ने के लिए, हम `+` ऑपरेटर का उपयोग कर सकते हैं। जैसे कि, यदि हमारे पास दो स्ट्रिंग हैं `नमस्ते` और `दुनिया`, तो हम इस तरह से उन्हें जोड़ सकते हैं:

```Rust
let name = "नमस्ते";
let message = "दुनिया";
let full_message = name + message;
```

यहां हमारा `full_message` वैल्यू `"नमस्ते दुनिया"` होगा।

## गहराई में

जब हम स्ट्रिंग्स को जोड़ते हैं, तो आरस्ट आगे जा कर स्ट्रिंग्स को `String` टाइप में कनवर्ट करता है। हम इसका प्रतिक्रियाशीलता कथन त्यागकर कह सकते हैं कि यह वास्तव में लगभग दो स्ट्रिंग्स को मिलाकर दिखाता है। जब हमारे पास बहुत सारे स्ट्रिंग्स होते हैं, तो यह बहुत आसान बनाता है कि हम उन्हें एक ही स्थान पर मिलाएं और एक साथ उपयोग करें।
