## क्यों

आपने अपने प्रोग्राम की परिकल्पना की है और आप अपने प्रोग्राम में कई अलग-अलग मूल्यों को जोड़ना चाहते हैं। इस मामले में, आपको स्ट्रिंग्स को योग द्वारा जोड़ने के लिए कोई अच्छा तरीका ढूंढना होगा। इससे आपको अधिक कोडिंग कंप्लिकेशन और लंबे कोड स्लो करने की जरूरत नहीं होगी। 

## कैसे करें

आप रस्ट में स्ट्रिंग्स को जोड़ने के लिए `format!` मैक्रो का उपयोग कर सकते हैं। इसमें आपको दो स्ट्रिंग्स और मूल्यों को दोनों के बीच अलग-अलग टोकन के साथ पास करने की आवश्यकता होगी। नीचे दिए गए उदाहरण में, हम दो स्ट्रिंग्स को `format!` मैक्रो का उपयोग करके जोड़ रहे हैं और उसका आउटपुट प्रिंट कर रहे हैं।

```rust
fn main() {
    let string1 = "Hello";
    let string2 = "world!";
    
    let result = format!("{} {}", string1, string2);
    println!("{}", result); // Output: Hello world!
}
```

इस तरह से आप स्ट्रिंग्स को आसानी से जोड़ सकते हैं और अपने प्रोग्राम को और भी सुंदर बना सकते हैं।
