## आखिर क्या है और क्यों करें?
डेट स्ट्रिंग से तारीख को पार्स करना अर्थात् उससे तारीख निकालना क्या है और क्यों प्रोग्रामर्स ये काम करते है।

इस प्रक्रिया का उपयोग डेटा में उपलब्ध तारीख को प्रोग्रामर्स को आसानी से उपयोग करने की अनुमति देता है। इससे तारीख अनुमानित तौर पर स्ट्रिंग से निकाला जा सकता है और इसे विभिन्न रूपों में प्रदर्शित किया जा सकता है।

## कैसे करें?
```Rust
use chrono::NaiveDate;

fn parse_date(date_string: &str) -> Result<NaiveDate, ParseError> {
    // मूल्यांकन की गई तारीख पढ़ें
    return NaiveDate::parse_from_str(date_string, "%Y-%m-%d").map_err(|e| e.to_string());
}

fn main() {
    // डेट स्ट्रिंग से तारीख पार्स करें
    let date_string = "2021-10-05";
    let parsed_date = parse_date(date_string);

    // पार्स की गई तारीख का प्रिंट करें
    println!("{:?}", parsed_date);
}
```

आउटपुट:
```Rust
Ok(2021-10-05)
```

## गहराई में फियादे
डेट स्ट्रिंग से तारीख को पार्स करने की इतिहासिक संदर्भ, विकल्प और इसके अंतर्गत विवरण के बारे में अधिक जानकारी।

यह प्रक्रिया पहले से ही बहुत समझदार और फायदेमंद है। पहले से ही कई आधुनिक प्रोग्रामिंग भाषाओं में इस तरह की सुविधा है, जो इसे अनुकूलित और सुविधाजनक बनाती है। तारीख स्ट्रिंग पार्सिंग के लिए रस का उपयोग करना और उसका अध्ययन करना प्रोग्रामिंग जगत में आपके लिए बहुत फायदेमंद हो सकता है।
