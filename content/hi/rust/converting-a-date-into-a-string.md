## आपकोई 

क्या आप जानते हैं कि रश भाषा में तारीख को स्ट्रिंग में परिवर्तित करना कैसे किया जाता है? यदि नहीं, तो आप सही जगह पर हैं! आइए हम इस ब्लॉग पोस्ट के माध्यम से हिंदी में रश की अंतर्दृष्टि सीखें।

## कैसे करें

हालांकि, रश भाषा में तारीखों को स्ट्रिंग में परिवर्तित करने की प्रक्रिया थोड़ी समझदारी और समय लेती है, लेकिन आप खुद को यह सीखने के लिए सक्षम बना सकते हैं। यहां हमने आपको "```Rust```" कोड ब्लॉक के साथ कुछ उदाहरण और परिणाम प्रदर्शित किए हैं जो आपको इस प्रसंग में अधिक समझने में मदद करेंगे।

```Rust
use chrono::{DateTime, NaiveDate, Utc, Local, Timelike, Datelike};

fn date_to_string(date: NaiveDate) -> String {
    // तारीख को रेखांकित पाठ में परिवर्तित करें
    let formatted_date = date.format("%d %B, %Y").to_string();
    formatted_date
}

fn main() {
    // एक इंस्टेंस बनाएं
    let date = NaiveDate::from_ymd(2021, 10, 15);

    // फ़ंक्शन को कॉल करें
    let formatted_date = date_to_string(date);
    println!("{}", formatted_date); // आउटपुट: 15 अक्तूबर 2021
}
```

इस उदाहरण में, हमने प्रथम फ़ंक्शन "```date_to_string()```" बनाकर तारीख को स्ट्रिंग में परिवर्तित किया है। हमने इस फ़ंक्शन को तारीख की फ़ॉर्मेटिंग भी किया है ताकि हमें आउटपुट में दिन, महीना और वर्ष की स्पष्ट जानकारी मिले। फिर हमने उस फ़ंक्शन को हमारे "```main()```" फ़ंक्शन में कॉल करके आउटपुट प्रदर्शित किया।
