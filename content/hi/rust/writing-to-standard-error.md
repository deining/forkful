## क्या है और क्यों? 

लेखन क्या स्टैंडर्ड त्रुटि में है, इसे व्यावसायिक क्यों करते है यह सिखाता है। इसका इस्तेमाल त्रुटियां स्पष्ट रूप से ग्रहण और संरक्षण करने के लिए किया जाता है।

## कैसे करें: 

```Rust
fn main() {
    eprintln!("यह स्टैंडर्ड त्रुटि में लिखा है।");
}
```

उपरोक्त कोड के नतीजे नेत है:
```
यह स्टैंडर्ड त्रुटि में लिखा है।
```

## गहराई की जांच: 

लेखन के लिए दूसरे विकल्प शामिल हैं, जिसमें स्पष्ट त्रुटियों को लिखने के लिए println! उपयोग किया जाता है। सिस्टम त्रुटि लेखन को निरोधित करने के लिए दूसरे विकल्पों वाला विकल्प भी मौजूद है।

## अधिक जानकारी: 

लेखन के इतिहास पर मूल्यांकन और इसके प्रयोग के प्रति आपकी सेहत के लिए अत्यधिक अहमियत है। आप ऑनलाइन स्रोत जैसे कि Rust लेखन स्टैंडर्ड त्रुटि के उपलब्ध विकल्पों की जांच कर सकते हैं।
