# क्यों

कोई भी प्रोग्रामिंग भाषा यही साबित करती है कि वे किसी भी प्रोग्राम के दौरान या उसके बाद समस्याओं को सुलझाने में बहुत मददगार हो सकती हैं। इसलिए, हम आज बात करेंगे कि रस्ट प्रोग्रामिंग में अलग-अलग तरीकों से स्टैण्डर्ड एरर में लिखने का एक अच्छा और सही तरीका क्या है।

## कैसे

स्टैण्डर्ड एरर होता है जहां प्रोग्राम ने किसी भी त्रुटि को देखा या उसमें कुछ गलतियां हो गई हैं। इसे फ़ेल करके आप अपनी कोड को इसे प्रिंट कर सकते हैं ताकि आपको पता चले कि कहाँ त्रुटि हो रही है और उसे कैसे ठीक किया जा सकता है। निचे दिए गए कोड ब्लॉक में हम रस्ट में स्टैण्डर्ड एरर में लिखने के लिए एक उदाहरण दीखाएंगे:

```Rust
fn main() {
    let mut input = String::new();
    std::io::stdin().read_line(&mut input).expect("Failed to read input!");
    eprintln!("Input: {}", input); // Write to standard error
}
```

इस कोड में, हमने कोड को बदलकर `println!` को `eprintln!` में बदला है, जो स्टैण्डर्ड एरर में लिखने को दर्शाता है। इसके बाद, जब हम कोड को रन करते हैं, तो अगर कोई त्रुटि होती है, तो वह हमें स्टैण्डर्ड एरर में संदेश दिखाएगा।
