## Rust में स्ट्रिंग इंटरपोलेशन क्या है और क्यों इसे प्रोग्रामर्स करते हैं?

स्ट्रिंग इंटरपोलेशन एक ऐसी तकनीक है जिसमें हम एक स्ट्रिंग में अन्य वैरिएबल्स या अंकों को विलय करते हैं। इससे हम बिना किसी अतिरिक्त कोड परिवर्तित किए बिना स्ट्रिंग को बनाने में सक्षम होते हैं। यह बहुत उपयोगी होता है क्योंकि हम अपनी स्ट्रिंग को अपडेट करने के लिए अपने दस्तावेजों या प्रोग्रामों में उपयोग कर सकते हैं और इससे हमारे कोड को स्पष्ट और साफ बनाने में भी मदद मिलती है।

## Rust में स्ट्रिंग इंटरपोलेशन कैसे किया जाता है?

Rust में स्ट्रिंग इंटरपोलेशन प्रत्येक स्ट्रिंग में ```{}``` द्वारा विशिष्ट इंटरपोलेशन अनुभाग के साथ किया जा सकता है। निम्नलिखित उदाहरण में, हमारे पास दो वैरिएबल्स हैं - ```name``` और ```age``` और हम इन दोनों को इंटरपोलेट करेंगे एक स्ट्रिंग में:

```Rust
let name = "Jane";
let age = 30;
let message = "My name is {} and I am {} years old";
println!(message, name, age);

/* Output:
My name is Jane and I am 30 years old
*/
```

इसके अलावा, हम स्ट्रिंगों का उपयोग अंकों या अन्य व्यापारिक लोजिक के साथ भी कर सकते हैं। निम्न उदाहरण में, हम एक उपयोगकर्ता से उनकी उम्र पूछेंगे और उन्हें उसकी प्रतिक्रिया दे सकते हैं:

```Rust
use std::io;
let mut age = String::new();
println!("Please enter your age:");
io::stdin().read_line(&mut age).expect("Failed to read line");
let age: u32 = age.trim().parse().expect("Please enter a valid age");
if age >= 18 {
    println!("You are an adult!");
} else {
    println!("You are not yet an adult");
}

/* Output:
Please enter your age:
25
You are an adult!
*/
```

## स्ट्रिंग इंटरपोलेशन के बारे में गहराई से

स्ट्रिंग इंटरपोलेशन कोड को साफ और स्पष्ट बनाने, स्ट्रिंगों के उदाहरणों को सुधारने और कोड को बहुत स्वरूपण आसान बनाने के लिए अत्यंत उपयोगी है। इसके अतिरिक्त, अन्य भाषाओं में इंटरपोलेशन के लिए इनकार्य भंडारण विकल्प हो सकते हैं जो कि अस्पष्ट या असंगत भाषा समस्याओं को पैदा कर सकते हैं। रस्ट में स्ट्रिंग इंटरपोलेशन हाल ही में जोड़ा गया है लेकिन यह पहले से ही अन्य भाषाओं में प्रचलित है। इसलिए, इस तकनीक को सीखने से आप कोडिंग को और भी सरल और आसान बना सकत
