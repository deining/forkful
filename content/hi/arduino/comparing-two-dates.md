## क्यों
आर्दुइनो में दो दिनांकों को तुलना करने के कारण आप किसी भी अन्य विशेषता को समझने के लिए उन्हें संपर्क में ले सकते हैं। एक निश्चित तारीख अन्य तारीख से प्राप्त किए गए डेटा को त्रुटि से मिलाने या संदर्भित करने के लिए उपयोगी हो सकता है।

## कैसे करें
सही तारीखों को तुलना करने के लिए आपको `if` स्टेटमेंट का उपयोग करना होगा। निम्न उदाहरण में, हम सभी कोड उदाहरण में दिए गए तिथियों को तुलना करते हैं, जो आपको उदाहरण के तौर पर दिखाते हैं। आप अपने आर्दुइनो में इसे आसानी से उपयोग कर सकते हैं।

```Arduino
int date1 = 22; //पहली तारीख
int month1 = 10; //पहले महीने
int year1 = 2021; //पहला साल

int date2 = 20; //दूसरी तारीख
int month2 = 10; //दूसरा महीना
int year2 = 2021; //दूसरा साल

if (year1 == year2 && month1 == month2 && date1 == date2) // दोनों तारीखों को तुलना करते हुए 
{
  Serial.println("दोनों तारीख एक दूसरे से मेल खाते हैं"); //अगर दोनों तारीखों में समानता है तो यह लाइन मुद्रण करेगा
}
else 
{
  Serial.println("दोनों तारीख एक दूसरे से मेल नहीं खाते हैं"); //अगर दोनों तारीखों में कोई भी समानता नहीं है तो यह लाइन मुद्रण करेगा
}
```

### आउटपुट
