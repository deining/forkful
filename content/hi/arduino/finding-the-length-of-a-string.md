# क्यों

क्या आपने कभी अपने पास कोई दो पाठ के बीच का अंतर ढूंढने की कोशिश की है? शायद आपको किसी दोस्त को अपनी सहायता के लिए एक सच चाहिए या आप अपने प्रोजेक्ट में दो पाठों को संयोजित करना चाहते हैं। इस साधन के लिए, आपको तकनीकी तौर पर जानना होगा कि पाठ की लंबाई कैसे पता लगाया जाए। इस हिंदी ब्लॉग पोस्ट में, हम आपको बताएंगे कि आप किस तरह से Arduino को प्रोग्रामिंग करके पाठ की लंबाई को कैसे पता लगा सकते हैं।

## कैसे करें

पाठ की लंबाई को पता लगाने के लिए, हम आपको एक सरल कोड उदाहरण प्रदान करेंगे जिसके माध्यम से आप आसानी से इस तकनीक को समझ सकेंगे।

```
एक पाठ को सहेजें
string myString = "हिंदी ब्लॉग पोस्ट";

// लंबाई को प्रिंट करें
Arduino.println(myString.length());
```
जैसा कि आप देख सकते हैं, हमने `myString` नामक चर पर एक पाठ सहेजा है और फिर `length()` फ़ंक्शन को इसमें लागू किया है। हम इसे `Arduino.println()` के साथ प्रिंट करते हैं जो हमें पाठ की लंबाई को दिखाएगा।

यदि आप यह कोड अपने Arduino बोर्ड पर अपनाएंगे, तो सीरियल मॉनिटर पर आपको "14" देखने को मिलेगा। यह पाठ के वर्णों की संख्या है जो `myString` में हैं।

## गहराई में जाएं

पाठ की लंबाई का पता लगाने के लिए, Arduino में `length()` फ़ंक्शन का उपयोग किया जाता है जो एक string वेरिएबल के अंतिम वर्ण का आईडी के साथ उपलब
