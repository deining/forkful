# हम क्यों जानें चरण की लंबाई

कभी-कभी हमें प्रोग्रामिंग में चरण की लंबाई की जानकारी की आवश्यकता हो सकती है। इसके जरिए हम अपनी स्ट्रिंग को एक्सेस कर सकते हैं और उसे अपनी आवश्यकताओं के अनुसार उपयोग कर सकते हैं।

## कैसे करें

Arduino में चरण की लंबाई पाने के लिए हम निम्नलिखित कोड का उपयोग कर सकते हैं:

```Arduino
// स्ट्रिंग बनाएँ
String myString = "नमस्ते";

// स्ट्रिंग की लंबाई पाएँ
int stringLength = myString.length();

// स्ट्रिंग की लंबाई को सीरियल मॉनिटर पर प्रिंट करें
Serial.println(stringLength);
```

इस उदाहरण में, हमने `length()` फंक्शन का उपयोग किया है जो हमें चरण की लंबाई को रिक्ति की संख्या के रूप में लौटाता है। इसके बाद हमने `Serial.println()` का उपयोग करके चरण की लंबाई को सीरियल मॉनिटर पर प्रिंट किया है। सीरियल मॉनिटर पर हमें `5` का आउटपुट मिलेगा क्योंकि हमारी स्ट्रिंग में `5` चरण हैं।

## गहराई में
