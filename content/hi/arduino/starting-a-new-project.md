# क्यों

क्या आपने हमेशा सपना देखा है कि आप एक नया प्रोजेक्ट शुरू करें और कूल गैजेट्स बनाएं? लेकिन कंप्यूटर साइंस किताबों से डार्क डेटा करके आपको डर लगता है? अभी आपका मौका है! अर्दुइनो प्रोग्रामिंग शुरू करने के लिए कोई भरोसा है क्योंकि यह बहुत सरल है और आपको कठिन काम नहीं होगा।

# कैसे

अपने प्रोजेक्ट को स्टार्ट करने के लिए, सर्वप्रथम जरूरत होती है की आपके पास अर्दुइनो हार्डवेयर हो। इसके बाद, आपको अर्दुइनो सॉफ्टवेयर को डाउनलोड और स्थापित करना होगा। फिर, आप बस नीचे दिए गए कोड ब्लॉक्स को आपने सोचा प्रोजेक्ट को देखने के लिए और बहुत जल्द आपका प्रोजेक्ट शुरू हो जाएगा!

```Arduino
// उदाहरण उपकरण चलाएं
int LED = 13;

void setup() {

pinMode(LED, OUTPUT); // लाइट इंडिकेटर को आउटपुट मोड में सेट करें

}

void loop() {

digitalWrite(LED, HIGH); // लाइट इंडिकेटर को ऑन करें
delay(1000); // 1 सेकंड के लिए वेट करें
digitalWrite(LED, LOW); // लाइट इंडिकेटर को ऑफ करें
delay(1000); // 1 सेकंड के लिए वेट करें

}
```
