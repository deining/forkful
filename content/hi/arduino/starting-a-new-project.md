## क्यों

क्या आपने अपने नए आर्डुइनो प्रोजेक्ट की शुरुआत करने का विचार किया है? आर्डुइनो माइक्रोकंट्रोलर एक उपकरण है जो आपको दुनिया में अनेक काम करने की सुविधा देता है। इससे आप अपने अविस्मरणीय और उपयोगी प्रोजेक्ट बना सकते हैं।

## कैसे करें

आपको स्थिर प्लेटफॉर्म संचालन योग्य आर्डुइनो सॉफ्टवेयर की आवश्यकता होगी। आप पहले बोर्ड जोड़ सकते हैं और उसके बाद आप एक CP2102 USB TTL सीरियल कनवर्टर का उपयोग करके आर्डुइनो को आपके कंप्यूटर से जोड़ सकते हैं। आपको विभिन्न सेंसर और अन्य संयुक्त साधनों का उपयोग करना होगा। एक आर्डुइनो प्रोग्राम को आप कई तरह से शुरू कर सकते हैं।

```Arduino
void setup(){
   // यहाँ आप बोर्ड पर पुन: प्रारम्भिक सेटअप कोड लिख सकते हैं
}

void loop(){
   // यहाँ आप बोर्ड पर निरंतर चल रहे कोड लिख सकते हैं
}
```

यह आवश्यक हो सकता है कि आपको आर्डुइनो लाइब्रेरी भी इंस्टॉल करनी पड़े। इन लाइब्रेरी आपको निवेश, जल प्रबंधन, वक्रवान आईसी, उल्लेखनीय शक्ति, एमओएसई सेंसर और बहुत कुछ शामिल हो सकते हैं। आप स्केच नामक निर्यात जोड़ सकते हैं।
