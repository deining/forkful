## क्यों
अगर आप एक Arduino प्रोग्रामर हैं तो आपने शायद सोचा होगा कि कैसे हम Arduino में डायरेक्ट्री की जांच कर सकते हैं। इस आर्टिकल में हम आपको इसके बारे में बताएंगे।

## कैसे करें
यह जांच करने के लिए हम पहले एक 'डायरेक्ट्री' ऑब्जेक्ट बनाएंगे। फिर हम `exists()` फ़ंक्शन का उपयोग करके उस डायरेक्ट्री की मौजूदगी जांचेंगे। नीचे दिया गया कोड सेगमेंट यह सब दिखाता है:

```Arduino
// डायरेक्ट्री बनाएं
Directory myDirectory = SD.open("myDirectory");

// जांचें कि क्या डायरेक्ट्री मौजूद है
if (myDirectory.exists()){
  Serial.println("यह डायरेक्ट्री मौजूद है!");
}
```

जैसा कि ऊपर के कोड सेगमेंट में दिखाया गया है, हम पहले से ही एक डायरेक्ट्री के नाम की उत्पत्ति कर रहे हैं और फिर उसे `exists()` फ़ंक्शन से जांचेंगे। यदि वह मौजूद होगा तो सीरियल मॉनिटर पर हमें उस की खबर मिलेगी।

## गहराई से जांच
जब हम `exists()` फ़ंक्शन का उपयोग करते हैं तो हम वास्तव में डायरेक्ट्री की मौजूदगी की जांच नहीं कर रहे हैं। बल्कि, हम दिए गए डायरेक्ट्री पथ के निर्देशों के साथ संबद्धवय कर रहे हैं। यदि उस पथ पर एक डायरेक्ट्री मौजूद होगी तो हमें सफल स्टेटस लौटेगा। इसलिए, हमें सुनिश्चित करना चाहिए कि हम उस पथ को सही ढंग से निर्देशित कर रहे हैं।
