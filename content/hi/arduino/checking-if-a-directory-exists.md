## क्यों

हम अर्दुइनो प्रोग्रामिंग कैसे करते हैं और कैसे इसका उपयोग करते हैं हम जानना चाहते हैं। ये पोस्ट हमें एक उदाहरण के साथ एक डायरेक्ट्री की उपस्थिति की जांच कैसे करे, इसके बारे में सीखाएगी।

## कैसे करे

अगर आप एक अर्दुइनो स्केच बना रहे हैं और एक डायरेक्ट्री की उपस्थिति की जांच करना चाहते हैं, तो आप निम्न उदाहरण के माध्यम से कोड को लिख सकते हैं।

```arduino
#include <SPI.h>
#include <SD.h>

const int chipSelect = 4;

void setup(){
    // SD कार्ड को शुरू करने के लिए पहले पिन 4 को बंद करें
    pinMode(4, OUTPUT);
    
    // रूट डायरेक्ट्री की उपस्थिति का जांच करें
    if(SD.exists("/")){
        // यदि डायरेक्ट्री मौजूद है, तो नीचे दिए गए कोड को भी काम में लें
        // अन्यथा, संदेश दिखाएं
        Serial.println("यह डायरेक्ट्री मौजूद है।");
    } else {
        Serial.println("यह डायरेक्ट्री मौजूद नहीं है।");
    }
}

void loop(){
    // कुछ काम या प्रक्रिया यहां करें
}

```

आप कंपाइल और अपने अर्दुइनो बोर्ड पर अपलोड करने के बाद, सीरियल मॉनिटर पर मैसेज देख सकते हैं कि क्या आपकी डायरेक्ट्री स्केच में मौजूद है या नहीं।
