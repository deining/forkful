## क्यों

कभी-कभी हमारे प्रोग्रामिंग प्रोजेक्ट में चुनौतियों का सामना करना पड़ता है। मौजूदा कोड में गलती तलाशना और सुधार करना आसान नहीं हो सकता है। इसलिए, debug output को प्रिंट करना एक महत्वपूर्ण तकनीक है जो हमें अपने कोड की दुरुपयोग से रक्षा करता है।

## कैसे

"```Arduino
Serial.begin(9600); // debug output के लिए serial पोर्ट को शुरू करता है
...
Serial.println("Debug output"); // कोई भी debug जानकारी प्रिंट करता है
...
Serial.print("Variable value: "); // debug के साथ वेरिएबल का मूल्य प्रिंट करता है
Serial.println(variable); // वेरिएबल के मूल्य को प्रिंट करता है
```"

ऊपर दिए गए आर्डुइनो कोड ब्लॉक में, हमने debug output को प्रिंट करने के लिए कुछ उदाहरण दिए हैं। सीरियल मॉनिटर के माध्यम से हम इस output को देख सकते हैं। यदि आपको अधिक जानकारी चाहिए तो आप एक कुछ tutorials भी देख सकते हैं जो debug output के बारे में विस्तार से बताते हैं।

## गहराई में जाएं

debug output का उपयोग केवल कोड में त्रुटियों को दिखाने के लिए ही नहीं किया जाता है। इसका उपयोग किसी विशेष प्रोटोकॉल में संदेश भेजने के लिए भी किया जाता है। एक उदाहरण के लिए, हम सिंसतिव संदेश भेजते हैं जो हमारे आसपास कुछ भ्रमण ड्रोनों द्वारा स्वचालित रूप से चलाए जाने से बचाते हैं। यदि हमारे ड्रोनों में से कुछ इस संदेश को प्राप्त नहीं करते हैं, तो हम उन्हें inspect कर सकते हैं और समस्या
