## हमारे द्वारा डिबग आउटपुट प्रिंट करने का कारण

डिबग आउटपुट प्रिंट करना बहुत उपयोगी हो सकता है जब हम अपने आर्डुइनो प्रोग्राम को चला रहे हैं और हमें उसमे कोई समस्या होती है। यह हमें अपने कोड में त्रुटियां ढूँढने में मदद कर सकता है और उन्हें सुधारने में हमारी सहायता कर सकता है।

## कैसे करें (Coding Examples)

इस माध्यम से आपको अपने आर्डुइनो कोड में डिबग आउटपुट प्रिंट करने का तरीका सीखाऊंगा।

```
Arduino Serial.begin (9600); // सीरियल पोर्ट सेटअप

int val = 5; // कोड की लाइन 1
Serial.println (val); // कोड की लाइन 2
```

ऊपर दिए गए कोड उदाहरण में, हमने `Serial.println()` का उपयोग करके वेरिएबल `val` को प्रिंट किया है। यह प्रिंट स्टेटमेंट हमेशा लाइन का अंत आने से पहले कुछ भी प्रिंट करेगी। आप अपनी मर्जी के अनुसार `Serial.print()` भी प्रयोग कर सकते हैं। आप कोड को कंपाइल करें और सीरियल मॉनिटर खोलें, आपको `5` का आउटपुट दिखाई देगा। यदि कोड की लाइन 1 को कमेंट्स में बदल दिया जाता है, तो कोड की लाइन 2 को ही प्रिंट करेगा। इस तरह से हम अपने कोड में किसी भी स्थान पर डिबग आउटपुट प्रिंट कर सकते हैं और कोड की गति और त्रुटियां जांच सकते हैं।
