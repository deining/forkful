## क्यों

तो आपने अर्दुइनो प्रोग्रामिंग में लोगों के बारे में सुना होगा, लेकिन क्या आप जानते हैं कि आप अपने कोड लिखने से पहले टेस्ट केस लिखने में फायदे हो सकते हैं? इस ब्लॉग पोस्ट के माध्यम से हम आपको बताएंगे कि अर्दुइनो प्रोग्रामिंग में टेस्ट लिखने की जरूरत क्यों होती है। 

## कैसे करें

अर्दुइनो प्रोग्रामिंग में टेस्ट लिखने के लिए हमें जावा (Java) या सी (C) की जानकारी की आवश्यकता नहीं होती है। हम आपको आसान और सरल उदाहरणों के साथ बताएंगे कि कैसे हम लिखें और चलाएं टेस्ट केस। यह हमारे कोड की गुणवत्ता को सुनिश्चित करने के लिए बहुत महत्वपूर्ण है। नीचे दी गई स्निपेट्स में हम आपको साधारण एक्साम्पल और उसका आउटपुट दिखाएंगे। 

```arduino
// सरल ग्रीटिंग प्रोग्राम
void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.println("नमस्ते, दुनिया!");
  delay(1000);
}
```
आउटपुट:
```
नमस्ते, दुनिया!
नमस्ते, दुनिया!
नमस्ते, दुनिया!
नमस्ते, दुनिया!
...

```

```arduino
// नंबर जोड़ने का प्रोग्राम
int a = 5;
int b = 10;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int result = a + b;
  Serial.println("योग: " + String(result));
  delay(1000);
}
```
आउटपुट:
```
योग: 15
योग: 15
योग: 15
योग: 15
...

```
