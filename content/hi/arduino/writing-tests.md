## क्यों

अर्डुइनो प्रोग्रामिंग में टेस्ट लिखना आपके स्केच में निश्चितता का साथ आपको और आपके प्रोजेक्ट को हित करेगा। अपने प्रोजेक्ट को धीमा नहीं करते हुए और सुनिश्चितता को सुनिश्चित करते हुए, आप अपने आपको समय, संसाधन और स्ट्रेस को बचाएंगे।

## कैसे करें

अपने आर्डुइनो स्केच में जांचें लिखने के लिए, आप ये स्टेप्स फॉलो कर सकते हैं: 

```Arduino
int sensorPin = A0; // सेंसर पिन गाइडनस करें
int sensorValue = 0; // सेंसर परिणाम गाइडनस करें
int threshold = 500; // सेंसर की लोड गाइडनस करें
int led = 13; // एलईडी पिन गाइडनस करें

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  sensorValue = analogRead(sensorPin); // सेंसर से डेटा पढ़ें
  if (sensorValue > threshold) { // एजेर old-wise ह्यु
    digitalWrite(led, HIGH); // जेमालनित्र की रोशनी चाल शुरू करें
  } else {
    digitalWrite(led, LOW); // जेमालनित्र बंद करें
  }
  delay(100); // इन्टर्वल दर की मिलिसेकंड्स से फंक्शन के काम करें
}

```

जैसा कि आप देख सकते हो, हमने अपने स्केच में जांचा लिखा है जो सेंसर परिणामों को पढ़ता है और उसे उत्साहित होने के निर्देश देती है। आपके स्केच में आपको अपने स्केच के साथ कई टेस्ट विश्लेषण जोड़ने की सलाह दी गई है।
