## क्यों

क्या आप हमेशा से अपने Arduino बोर्ड पर लिखे गए कोड को बाद में फिर से देखना चाहते हैं? या शायद आप अपने कोड को अपने कंप्यूटर पर संग्रहीत करना चाहते हैं? इन सभी कामों के लिए, आपको एक पाठ फ़ाइल में अपने कोड को लिखने की आवश्यकता होगी। इस ब्लॉग पोस्ट में, हम आपको Arduino प्रोग्रामिंग में पाठ फ़ाइल लिखने का तरीका समझाएंगे।

## कैसे करें

सबसे पहले, हमेशा की तरह, हम अपने Arduino बोर्ड को सेटअप करेंगे। इसके लिए, हम `SD` लाइब्रेरी का उपयोग करेंगे। निम्नलिखित कोड ब्लॉक में आप देख सकते हैं कि हमने कैसे `SD.begin()` फ़ंक्शन का उपयोग किया है। इससे सभी SD कार्ड्स को निजी मोड में सेटअप किया जाएगा।

```Arduino
#include <SPI.h>
#include <SD.h>

void setup() {
  SD.begin();
}
```

अब, हमें फ़ाइल लिखने के लिए एक `File` ऑब्जेक्ट बनाना होगा। निम्नलिखित कोड ब्लॉक में आप देख सकते हैं कि हमने कैसे `SD.open()` फ़ंक्शन का उपयोग किया है। यह फ़ंक्शन एक `File` ऑब्जेक्ट लौटाएगा जिसके माध्यम से हम फ़ाइल में डेटा लिख सकते हैं।

```Arduino
File dataFile = SD.open("data.txt", FILE_WRITE);
```
