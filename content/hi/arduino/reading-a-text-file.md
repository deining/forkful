## क्यों

क्या आपने कभी अपने आर्दूइनो को टेक्स्ट फाइल से जानकारी पढ़ाने की कोशिश की है? अगर हाँ, तो यह आर्टिकल आपके लिए है। इसमें हम आपको आर्दूइनो के साथ टेक्स्ट फाइलों को पढ़ने का तरीका सिखाएंगे। यह आपको अपने प्रोजेक्ट में बहुत सारे सही सूचनाओं और सेटिंग्स को आसानी से रीड करने में मदद कर सकता है।

## कैसे करें

अपनी आर्दूइनो बोर्ड को कॉम्प्यूटर से connect करें और Arduino IDE ओपन करें। सबसे पहले हमें एक टेक्स्ट फाइल बनानी होगी, जिसमें हम अपनी सभी सूचनाएं संरचित रखेंगे। इसके बाद हमें ऑथरिजेशन को set करना होगा की Arduino फाइलों को रीड कर सके। आप कोड ब्लॉक में दिए गए कोड को विभिन्न सेक्शन में डालें।

```Arduino
// टेक्स्ट फ़ाइल बनाएं
File myFile;
// आपका फाइल कॉम्प्यूटर के साधनों में स्टोर होता है
String fileName = "myFile.txt";
// आपके और मुझे पाठ संग्रह में नए लाइन सम्बन्धी नियम।
// आर्दुइनो?
String newLineRule = "\r\n";

void setup() {
    // आपको शुरुआत में आपका फ़ाइल कॉम्प्यूटर पर लोड होना होगा
    while (!Serial) {
        ;
    }

    myFile = SD.open(fileName, FILE_READ);
    if (myFile) {
        while (myFile.available()) {
            Serial.write(myFile.read());
        }
    }
}
```

जब आप अपना कोड upload करेंगे, आपको सीरियल मॉनिटर में अपनी पूर्ण टेक्स्ट फाइल को देख सकते हैं। आप इसमें अपनी जानकारी बदल सकते हैं और हर बार अपना कोड अपलोड करने के बाद आपको न
