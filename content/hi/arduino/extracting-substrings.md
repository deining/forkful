## क्यों

क्या आपको अपने एक शब्द या वाक्य के बीच से सबस्ट्रिंग निकालने की ज़रूरत है? यह Arduino अनुप्रयोग आर्टिकल आपको समस्या को हल करने में मदद करेगा। इस आर्टिकल में, हम सबस्ट्रिंग को कैसे निकाला जाता है इंजिनियरिंग और आर्डुइनो कोडिंग की दृढ़ता के साथ जानते हैं। 

## कैसे करे

 Arduino कोड का अध्ययन करना किसी कहानी से काफी बेरसख्ती से भरपूर हो जाता है। हालांकि, कुछ कोड स्निपेट्स आपको अपने काम को कायम करने में मदद कर सकते हैं। नीचे दिए गए Arduino कोड अधिगमकोश को अधुरे संख्याओं या अक्षरों और स्ट्रिंग्स तक हैं, इसलिए, अतिरिक्त सुविधा के साथ, सबस्ट्रिंग निकाल करते हैं। यह "coding_example.ino" नाम के नये Arduino संगठन से जारी होता है। 

```Arduino

//संज्ञानात्मक चिन्हों का प्रयोग सबस्ट्रिंग को निकालने के लिए
void setup(){
String full_string = "प्रिंस एक महान राजकुमार है।";
//स्थान से सबस्ट्रिंग को निकालते हैं
String substring = full_string.substring(13);
Serial.println(substring); //यह उत्पादन को "राजकुमार है।" गंभीर कर सकता है

//शुरुआती और अंतिम स्थान से सबस्ट्रिंग को निकालते हैं
String substring = full_string.substring(0, 5);
Serial.println(substring); //यह उत्पादन को "प्रिंस" गंभीर कर सकता है
}

void loop(){}; // Adruinp को जारी रखें या जांचें 
```

जैसा कि आप देख सकते हैं, "substring" फ़ंक्शन "स्रोत स्ट्रिंग" से सबस्ट्रिंग को निकालने के लिए प्रयुक
