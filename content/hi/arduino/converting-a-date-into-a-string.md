"## क्यों"

अक्सर हमारे आसपास आर्डुइनो बोर्ड का उपयोग उन्नत तकनीकी प्रोजेक्टों में किया जाता है। और प्रोजेक्ट बनाने के लिए हमें दिनांक के स्ट्रिंग में बदलनी की आवश्यकता होती है। इस तकनीक के साथ आप अपने प्रोजेक्ट में उपयोग किया जा सकता है जो आपको अपने काम को अधिक सुविधाजनक बनाता है।

"## कैसे करें"

किसी भी तकनीकी प्रोजेक्ट में दिनांक को स्ट्रिंग में कन्वर्ट करना बहुत ज़रूरी होता है। यह काम आसानी से आपके आर्डुइनो बोर्ड पर कर सकते हैं। आइए देखते हैं कि कैसे आप इस काम को क्रम बनाते हुए कर सकते हैं।

```Arduino
#include <Time.h> // Time library इन्क्लूड करें
Time now; // Time object डिफाइन करें
String timeString; // String जहां दिनांक स्टोर होगी, बनाएं
now = now.unixTime(); // सिस्टम की अपडेटेड दिनांक को Time ऑब्जेक्ट से ग्रहण करें
timeString = timeString + String(year(now)) + "/"; // वर्ष को string में कन्वर्ट करें और String object में जोड़ें
timeString = timeString + String(month(now)) + "/"; // माह को string में कन्वर्ट करें और String object में जोड़ें
timeString = timeString + String(day(now)); // दिनांक को string में कन्वर्ट करें और String object में जोड़ें
Serial.println(timeString); // Serial मॉनिटर पर देखें
```

इस कोड का आउटपुट `2021/9/16` होगा। आप भी अपनी ज़रूरतों के हिसाब से दिनांक को स्ट्रिंग में बदल सकते हैं।
