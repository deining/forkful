## क्यों

क्या आप अपने आर्डुइनो पर तारीख को स्ट्रिंग में बदलने के बारे में सोच रहे हैं? एक स्ट्रिंग को आर्डुइनो में उपयोगी बनाने के लिए इसका उपयोग कर सकते हैं, जैसे कि अपनी स्क्रीन पर या सीरियल मॉनिटर पर प्रिंट करके स्तंभों को अलग कर सकते हैं। इसके अलावा, आप अपने प्रोजेक्ट को अन्य डिवाइसों या वेब साइटों से भी संचालित कर सकते हैं। तो अभी हम सीखेंगे कि कैसे आर्डुइनो पर तारीख को स्ट्रिंग में बदला जा सकता है।

## कैसे करें

```arduino
#include <DateTime.h>

void setup() {
  DateTime now = DateTime();

  // Convert date to string
  String dateString = String(now.year()) + "-" + String(now.month()) + "-" + String(now.day());

  // Print to serial monitor
  Serial.println(dateString);
}

void loop() {
  // Your code here
}
```

जैसा कि आप देख सकते हैं, हमने `DateTime.h` लायब्रेरी को लोड किया है और तारीख को `DateTime` क्लास का स्ट्रिंग में बदला है। इसके बाद हमने सीरियल मॉनिटर पर स्ट्रिंग को प्रिंट किया है। आप अपनी आर्डुइनो बोर्ड को सीरियल मॉनिटर से कनेक्ट करके नम्बरों को देख सकते हैं। आप शायद ध्यान दे पाएंगे कि हमने `String` वर्ग का उपयोग भी किया है। यह आपको स्ट्रिंग को संयोजित करने में मदद करेगा। अगर आप अन्य प्रकार के तारीख फॉर्मेट चाहते हैं तो आप `DateTime` क्लास का उपयोग करके अपने स्वयं के साथ तारीख प्रिंट कर सकते हैं।
