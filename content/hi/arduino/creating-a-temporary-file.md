## क्यों 

क्या आपको कभी कोई अस्थायी फाइल बनाने की आवश्यकता हुई है? यह अस्थायी फाइल साधारणतया किसी टेम्पलेट या प्रोग्राम के लिए अस्थायी डेटा या जानकारी स्टोर करने के लिए उपयोगी होती है। अस्थायी फाइल बनाना आसान और उपयोगी है जब आपको किसी अस्थायी स्थिति के लिए डेटा को स्टोर करने की आवश्यकता होती है। 

## कैसे करें 

तो, आपको आर्डुइनो में अस्थायी फाइल बनाना है। नीचे दिए गए उदाहरण आपको समझाएंगे कि आप ऐसा कैसे कर सकते हैं। साथ ही आपको कोड ब्लॉक भी दिखाएंगे जो आपको कैसे आर्डुइनो में अस्थायी फाइल बनाने में मदद करेंगे। 

```Arduino 
// नई अस्थायी फाइल को खोलने के लिए नाम तय करें
String file_name = "sample.txt";
// एसडी कार्ड के लिए संचरण बनाएं
File file = SD.open(file_name, FILE_WRITE);
// अस्थायी फाइल में डेटा लिखें
file.println("यह एक अस्थायी फाइल है।");
// फाइल को बंद करें
file.close();
```

प्रयोगशाला का उत्पादन नीचे दिए गए स्क्रीनशॉट में दिखाया गया है।

## गहराई में जाएं 

अस्थायी फाइलों के साथ काम करने का एक आदर्श तरीका है SD लाइब्रेरी का उपयोग करना। इस लाइब्रेरी में आपको अस्थायी फाइलों के साथ काम करने के लिए कई फ़ंक्शन मिलेंगे, जो आसानी से समझ में आते हैं। आप इन फ़ंक्शन में से कुछ का उपयोग
