## क्यों

क्या आपने कभी सोचा है कि आर्डुइनो में अस्थायी फ़ाइल क्यों बनाने की आवश्यकता हो सकती है? अस्थायी फ़ाइल का उपयोग अपने कोड को ऑटोमेटिकली बनाए गए संग्रहण के लिए किया जाता है और ऐसे समयों में वे स्थानें जो आपके डिवाइस पर कम स्थान के साथ बनाया गया हो सकते हैं।

## कैसे

अस्थायी फ़ाइल बनाने के लिए, आपको `File` लाइब्रेरी का उपयोग करना होगा। नीचे दिए गए कोड स्निपेट में, हम एक अस्थायी फ़ाइल `temp.txt` बनाने के लिए कॉड देखेंगे।

```Arduino
#include <SPI.h>
#include <SD.h>

File myFile;

void setup() {
  // सीडी कार्ड सक्रिय करें
  // टेम्प|| नई फ़ाइल बनाएं
  myFile = SD.open("temp.txt", FILE_WRITE);
  // सफ़लतापूर्वक बनाया गया है तो सीडी कार्ड को बंद करें
  myFile.close();
  // सीडी कार्ड सक्रिय करें
  // सफ़लतापूर्वक बनाए गए फ़ाइल को हटाएं
  SD.remove("temp.txt");
  // सफ़लतापूर्वक बनाए गए फ़ाइल को हटाने के बाद सीडी कार्ड को बंद करें
  SD.end();
}

void loop() {
  // कुछ और कोड
}
```

जैसा कि आप कोड स्निपेट में देख सकते हैं, हमने सीडी कार्ड को सक्रिय किया, एक अस्थायी फ़ाइल बनाई और उसको हटाया। अस्थायी फ़ाइल को हटाने के बाद, सीडी कार्ड बंद करना जरूरी है।
