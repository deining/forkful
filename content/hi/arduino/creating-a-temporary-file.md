## आप क्या और क्यों करते हो?

ताजगीअर्डूइनो कर्नेल फाइल सिस्टम के साथ आता है जो फाइलों को आपकी डिवाइस में पढ़ने और लिखने की अनुमति देता है। एक कार्यकारी प्रोग्रामर के रूप में, हम अक्सर अस्थायी फाइलों को बनाते हैं जो कि ऑपरेशन के दौरान या डेटा स्टोरेज करने के लिए प्रयोग किए जाते हैं।

## कैसे करें:

अपने आर्डुइनो दीपा से एक अस्थायी फाइल बनाने के लिए, हम निम्न कोड का उपयोग कर सकते हैं:

```Arduino

// अस्थायी फाइल नाम
char *filename = "temp.txt";
// फाइल बनाओ
File file = SD.open(filename, FILE_WRITE);
// फाइल में डेटा लिखो
file.println("यह एक अस्थायी फाइल है!");
// फाइल बंद करो
file.close();
// सफलतापूर्वक बनायी गई फाइल को साफ़ करो
SD.remove(filename);

```

जब हम कोड को अपने डिवाइस पर अपलोड करेंगे, तो हमें इस तरह का आउटपुट मिलेगा:

```
यह एक अस्थायी फाइल है!
```

## गहराई में जाएं:

अस्थायी फाइलों को बनाने का अवसर आया जब बड़े एवं संगठित फाइल सिस्टम को अपनाने का टाइम दिखाई देने लगा। इन फाइल सिस्टम से बड़े होने के कारण उन्हें अपने डिवाइस पर ले जाना थोड़ा मुश्किल था, और इसलिए प्रोग्रामरों ने अस्थायी फाइलों को आपरेशन के दौरान ही उपयोग करना शुरू किया।

चुनाव के रूप में, मानो स्वचालित रूप से साफ़ नहीं किए गए अस्थायी फाइलें खुद बाद में साफ़ हो जाएंगी। इससे आपका डिवाइस मेमोरी स्पेस से बचता है और आपको अधिक फाइल स्टोर करने की अनुमति देता है।

अस्थायी फाइलों का उपयोग करते समय, ध्यान रखने योग्य बात यह है कि आपको सफलतापूर्वक फाइल को साफ़ करना होगा, अन्यथा एक्सेस समस्याएं हो सकती हैं। कई बार, यह सुनिश्चित करने के लिए कि सभी अस्थायी फ़ाइलें साफ़ हैं, आप स्केचबुक के आकार को छोटा कर सकते हैं या स्केचबुक को पुराना बना सकते हैं।
