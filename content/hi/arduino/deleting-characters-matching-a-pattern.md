## क्यों

कभी-कभी हमारे कोड में कुछ गलतियां हो जाती हैं जो एक ही पैटर्न को मैच करती हैं, और इससे इंपल्सिव कोडिंग के कारण उन्हें सिर्फ़ रद्द कर देने का फ़ैसला कर सकते हैं। इस आर्टिकल में हम आपको बताएंगे कि अर्दुइनो में किस तरह से आप पैटर्न से मैच करने वाले चरित्रों को हटा सकते हो। 

## कैसे करें।

अर्दुइनो में कुछ चरित्रों को हटाने के लिए आपको `String` को पैटर्न से मैच करने के लिए `replace()` फ़ंक्शन का उपयोग करना होगा। नीचे दिए गए कोड ब्लॉक में आपको दिखाया गया है कि कैसे यह काम करता है।

```arduino
String message = "Hello World!";
message.replace("o", ""); // इससे "Hello Wrld!" स्ट्रिंग दिखाई देगी। 
```

आप अपनी स्थिति के अनुसार `replace()` फ़ंक्शन का उपयोग कर सकते हैं। आप अधिक जानकारी के लिए आगे दी गई डीप डाइव सेक्शन को देख सकते हैं। 

## डीप डाइव

अर्दुइनो में `String` को पैटर्न से मैच करने के लिए `replace()` फ़ंक्शन का उपयोग करने से आप सिर्फ़ एक ही चरित्र को ही हटा सकते हैं। अगर आप दो चरित्रों को हटाना चाहते हैं, तो आपको `replace()` फ़ंक्शन को दो बार लिखना होगा। आप अपने अनुसार इसका उपयोग कर सकते हैं। इससे समझने के लिए नीचे दिए गए कोड ब्लॉक को देखें।

```arduino
String message = "Hello World!";
message.replace("l", "").replace("o", ""); // इससे "He Wrd!" स्ट्रिंग दिखाई देगी।
```
