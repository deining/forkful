## इसे क्यों करें

अर्डुइनो प्रोग्रामिंग के माध्यम से हम अपनी आवश्यकताओं को पूरा करने के लिए अलग-अलग तरीकों को लिख सकते हैं। ऐसे में, हमें कभी-कभी कुछ अलग करने की आवश्यकता पड़ती है। इस लेख में हम आपको बताएंगे कि अर्डुइनो प्रोग्रामिंग में किस तरह से किसी पैटर्न से मेल खाते हुए चरित्रों को हटाया जा सकता है।

## कैसे करें

यदि आप अपने स्केच पर विशेष पैटर्न के साथ मेल खाने वाले चरित्रों को हटाना चाहते हैं, तो आपको इस तरह से कोडिंग करने की आवश्यकता होगी:

```Arduino
String str = "Arduino";
str.replace("r", ""); // यहां आप जिन चरित्रों को हटाना चाहते हैं, उन्हें डाल सकते हैं।
Serial.println(str); // यहां आपको अंतिम परिणाम मिलेगा जिसमें से विशेष चरित्र हटा दिए गए होंगे।
```

इस से आपको नीचे दिए गए आउटपुट मिलेगा:

```
Aduino
```

इसी तरह से, आप अपनी आवश्यकताओं के अनुसार अपने कोड में और भी बहुत से पैटर्न और चरित्रों को हटा सकते हैं।
