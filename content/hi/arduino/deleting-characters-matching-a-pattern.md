## क्यों

कई बार हमारे पास एक स्ट्रिंग में कुछ खास पैटर्न के अनुसार कुछ अवांछित और अनावश्यक वर्णों के साथ सामना करना पड़ता है। इससे स्ट्रिंग का प्रिंटिंग और उसका उपयोग करना कठिन हो सकता है। इसलिए इस विषय पर एक विस्तृत लेख लिखने का मुख्य कारण है कि हम इस समस्या को कैसे हल कर सकते हैं और अपने अर्डुइनो प्रोजेक्ट में इसका उपयोग कर सकते हैं।

## कैसे करें

स्ट्रिंग में खाली या अनावश्यक वर्णों को हटाने के लिए हमें दो तरीके हैं - एक आसान तरीका और एक थोड़ा जटिल तरीका।

आसान तरीका:

```arduino
String str = "Arduino आसान है।";
str.replace(" ", "");
Serial.println(str);
```

थोड़ा जटिल तरीका:

```arduino
String str = "Arduino आसान है।";
String resultStr = "";
for (int i=0; i<str.length(); i++) {
  if (str.charAt(i) != ' ') {
    resultStr += str.charAt(i);
  }
}
Serial.println(resultStr);
```

उपरोक्त कोड का आउटपुट होगा:

```arduino
Arduinoआसानहै।
```

आप अपनी स्ट्रिंग को चाहे जितना भी बड़ा या जटिल बना सकते हैं, उपरोक्त तरीकों से स्ट्रिंग में से अनावश्यक वर्णों को हटाना संभव होगा। आप स्ट्रिंग में न केवल स्पेस बल्कि अन्य पैटर्न भी हटा सकते हैं।
