## क्यों

आजकल, हमारे सामान्य दिनचर्या में हम अक्सर अपने मोबाइल या कम्प्यूटर पर टाइपिंग करते हैं। अपनी पोस्टिंग, प्रोफ़ाइल या ऑनलाइन संदेशों में हम एक शब्द या समूह को बड़े अक्षरों में लिखने का उपयोग करते हैं। यह आसानी से पढ़ाई और बेहतर समझने में मदद करता है। ऐसे में, अपने आर्डुइनो प्रोग्राम में भी यदि आप किसी शब्द या समूह को बड़े अक्षरों में प्रिंट करना चाहते हैं तो आपको वो कैसे करना है इसे सीखना उचित होगा।

## कैसे करें - How To

जैसा कि मैंने पहले भी बताया है, आर्डुइनो में किसी शब्द या समूह को बड़े अक्षरों में प्रिंट करने के लिए हम capitalization या शीर्ष अक्षर चिह्न (`Uppercase`) का प्रयोग करते हैं। नीचे दिए गए उदाहरण में, हम एक शब्द और एक समूह को बड़े अक्षरों में प्रिंट करेंगे।

```Arduino
void setup() {
  Serial.begin(9600); // सीरियल मॉनिटर को शुरू करता है
}

void loop() {
  String word = "hindi";
  String sentence = "welcome to arduino";

  // String को capitalization के साथ प्रिंट करें
  Serial.println(word.toUpperCase()); 
  Serial.println(sentence.toUpperCase());

  delay(1000); // 1 सेकंड की रोक लगाता है
}
```

आपको ऊपर दिए गए कोड ब्लॉक को अपने आर्डुइनो में लाइब्रेरी के समान तरीके से पेस्ट करना होगा। फिर सीरियल मॉनिटर को खोलें और देखें कि कैसे शब्द और समूह अपने अक्षरों को बड़ा करते हैं।
