## क्यों
स्ट्रिंग को कैपिटलाइज़ करने के लिए कोई कारण है? यह एक आसान तरीका है जिसका इस्तेमाल आप अपने Arduino बोर्ड पर स्ट्रिंग के साथ कर सकते हैं।

## कैसे करें
जैसा कि हम जानते हैं, स्ट्रिंग एक परिणाम है जो कि अक्षरों का समूह है जो हमारे द्वारा उपयोगकर्ता प्रदान किया गया है। अक्षरों को कैपिटलाइज़ करने के लिए, हमें प्रत्येक अक्षर को उसकी ओर ट्रांसफॉर्म करने की आवश्यकता होती है। स्ट्रिंग को कैपिटलाइज़ करने के लिए, हम निम्नलिखित कोड का उपयोग कर सकते हैं:

```Arduino

String str = "hello world";  // आप इसे अपनी स्ट्रिंग के साथ बदल सकते हैं

str.toUpperCase();  // स्ट्रिंग को कैपिटल बनाने के लिए यह फंक्शन उपयोग करें

Serial.println(str);  // स्ट्रिंग की उच्च अक्षरों का प्रिंट करें, यह "HELLO WORLD" के रूप में प्रदर्शित होगा

```

इसके अलावा, Arduino कोड में `toUppercase()` फंक्शन का उपयोग करके आप अपनी स्ट्रिंग को आसानी से कैपिटल बना सकते हैं।

## गहराई में
अगर हम गहराई से देखें, तो स्ट्रिंग को कैपिटलाइज़ करने के लिए, हम एक लुप का उपयोग भी कर सकते हैं जो प्रत्येक अक्षर को अलग-अलग कैपिटल अक्षर बनाने में मदद करेगा। यह स्केच नीचे दिए गए लिंक से डाउनलोड किया जा सकता है। आप अपने बोर्ड पर इसे अपलोड करके इसका उपयोग कर सकते हैं।
