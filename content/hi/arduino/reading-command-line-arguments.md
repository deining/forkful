## क्यों

यदि आप Arduino से परिचित हैं, तो आपने कभी भी कमांड लाइन आर्ग्यूमेंट्स (command line arguments) के बारे में सुना होगा। आपको यह जानने में मदद मिल सकता है कि यह क्या है और आप इसका उपयोग कैसे कर सकते हैं।

## कैसे

```Arduino
void setup(){
  // यहां सिर्फ आपके सेटअप कोड होगा
}

void loop(){
  // आपका मेन कोड यहां होगा

  // संभवतः आपको विभिन्न कमांड लाइन आर्ग्यूमेंट्स को पढ़ना होगा
  int arg1 = atoi(argv[1]); // पहला आर्ग्यूमेंट के लिए int टाइप का मान बनाएं
  Serial.println(arg1); // आपकी सीरियल मॉनिटर पर आर्ग्यूमेंट दिखाएं
}
```

आ

र्ग्यूमेंट्स को पढ़ने के लिए, आपको `main()` फंक्शन में से `argc` और `argv` पैरामीटर्स को मिलाना होगा। यह पैरामीटर आपको आर्ग्यूमेंट्स की संख्या और उनके मानों को उपलब्ध कराता है। `argv` एक स्ट्रिंग एरे है जो आर्ग्यूमेंट्स को संग्रहीत करता है और हर आर्ग्यूमेंट को अलग स्ट्रिंग में संग्रहीत करता है। `atoi()` फंक्शन स्ट्रिंग को इंटीजर मान में रूपांतरित करता है।

आर्ग्यूमेंट पार्सिंग में एक और लाभ यह है कि आप अपने स्केच में प्रयोग करने के लिए इस्तेमाल कर सकते हैं। यदि आपको किसी भी सेंसर की सेटिंग या अन्य स्केच विशेषताओं को प्रवाहित करने के लिए बदलाव करने की जरूरत है, तो आप आर्ग्यूमेंट्स का उपयोग करके इसे काफी सरल
