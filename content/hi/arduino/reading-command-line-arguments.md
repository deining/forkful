## क्यों

आजकल हमारे आसपास बहुत सारे तकनीकी उपकरण हैं जो हमारे दैनिक जीवन को सरल बनाने में मदद करते हैं। इन उपकरणों में से एक भी आर्डुइनो है। आर्डुइनो एक बहुत ही प्रभावी माइक्रोकंट्रोलर है जो एक साथ कई एलेक्ट्रॉनिक कॉम्पोनेंट्स को नियंत्रित कर सकता है। आर्डुइनो प्रोग्रामिंग के इस लेख में हम आपको बताएंगे कि आर्डुइनो में command line arguments को कैसे पढ़ा जाता है।

## कैसे

अगर आप एक Arduino प्रोग्रामर हैं तो आपने अपने स्केच को दीवानगी में उससे बचाने के लिए कमांड लाइन आर्ग्यूमेंट का प्रयोग अवश्य ही किया होगा। हांफि, सी-लाइब्रेरी के साथ ही यह बेहतर बनाया गया है। अब, आपको बस उपयोगकर्ता को दिए गए स्केच में कमेंट करना है।

```Arduino
int main (int argc, char *argv[]) {
  // आप स्पष्ट कर सकते हैं कि कौन सा कमांड लाइन आर्ग्यूमेंट स्पष्ट रूप से निरीक्षित करता है जो आपको प्रक्रिया में निर्देशित या दिखाई देता है
  for (int i=0; i < argc; i++) {
    Serial.println(argv[i]);
  }
}
```

जब आप यहां तक पहुंच जाते हैं, तो आपको command line arguments का पढ़ाई करने का एक सरल तरीका है। आर्डुइनो स्केच में औपचारिक रूप से स्कान और कमेंट लेना चाहिए। जब आप लाइब्रेरी का उपयोग करते हैं तब आप अपनी एप्लिकेशन को अच्छी तरह से संभालें और यह सुनिश्चित करें कि अपने
