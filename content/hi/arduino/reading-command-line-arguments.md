## क्यों

कमांड लाइन आर्ग्यूमेंट पढ़ने का यह आर्टिकल आपके कोडिंग को आगे बढ़ाने और अपनी आर्डुइनो प्रोग्रामिंग छूटने के लिए आपको मदद करेगा। यह आर्ग्यूमेंट पूरे ड्रामिसचरियों को शुरू करने से पहले ही बहुत महत्वपूर्ण है।

## कैसे

आर्ग्यूमेंट पढ़ने के लिए सबसे पहले हमें ```Arduino``` लाइब्रेरी को इंपोर्ट करना होगा। इसके बाद हम एक स्ट्रिंग वेरिएबल में आर्ग्यूमेंट को स्टोर करेंगे। उसके बाद हम उस वेरिएबल को प्रिंट कर सकते हैं।

```Arduino
#include <Arduino.h>

void setup() {
  Serial.begin(9600);
}

void loop() {
  String argument = Serial.readString();
  Serial.println(argument);
}
```

जब आप अपने सीरियल मॉनिटर पर कोई आर्ग्यूमेंट्स टाइप करेंगे, तो वे आपके सीरियल मॉनिटर पर प्रिंट होंगे। यह वर्गीकरण एक मूल कथन मंगाता है: सीरियल मॉनिटर को सेटअप सीरिज़ में सेटअप करें।

## डीप डाउन

कमांड लाइन आर्ग्यूमेंट पढ़ने के अलावा, आप अपने एर्डुइनो को सक्रियता, भावनात्मकता, डेटा स्टोरेज और अन्य ग्राहक के मित्रल उपकरण बनाकर भी सेटअप कर सकते हैं। वे आगे बढ़ सकते हैं। कुछ उदाहरण उसमें समावेश हैं: जहां मेनेनांग में कुछ भी सिस्टम है और एशरिकराशियों के माध्यम से हम जो कि कमांड लाइन आर्ग्यूमेंट हैं हमें "शुधा" में चालाएं।
