## क्या और क्यों?
कमांड लाइन आर्ग्यूमेंट्स पढ़ना क्या है, और क्यों प्रोग्रामर इसे करते हैं। यह आर्डुइनो प्रोग्रामिंग में आपको क्षमता देता है प्रयोगकर्ता से डेटा को प्राप्त करने की।

## कैसे करें:
यहां आर्डुइनो में कमांड लाइन आर्ग्यूमेंट्स कैसे पढ़ें, उसका कोड दिया गया है। प्रत्येक उदाहरण के साथ आउटपुट दर्शाया गया है।

```arduino
// कमांड लाइन आर्ग्यूमेंट्स को पढ़ें
void setup(){
   // क्षेत्र या सेंसर का नाम कमांड लाइन आर्ग्यूमेंट्स से प्राप्त करें
   String area = arduino.getArgument(0);
   // कमांड लाइन आर्ग्यूमेंट्स से आकार की मात्रा प्राप्त करें
   int size = arduino.getArgument(1).toInt();

   // कमांड लाइन आर्ग्यूमेंट्स की मोटाई मापें
   Serial.print("क्षेत्र का नाम: ");
   Serial.println(area);
   Serial.print("क्षेत्र का आकार: " + String(size) + " cm");
}

void loop(){
    // कुछ कोड यहां लिखें
}
```
### आउटपुट:
```
क्षेत्र का नाम: मेज़नीन
क्षेत्र का आकार: 50 cm
```

## गहराई की जांच:
कमांड लाइन आर्ग्यूमेंट्स को पढ़ने के पीछे एक कुछ और इतिहास है। यह प्रोग्रामिंग के दूसरे तरीकों के बारे में भी ज्ञानवर्धक हो सकता है। आर्डुइनो कुछ इम्प्लीमेंटेशन डिटेल्स भी देता है कूच तर्क की प्रक्रिया दिया गया है इसलिए आप इसे प्रायोगिक तौर पर स्पष्ट विवरण में पा सकते हैं।
