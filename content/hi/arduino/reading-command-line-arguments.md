## क्यों

अर्दुइनो प्रोग्रामिंग में, एक बहुत अधिक उपयोग होने वाला तरीका है कमांड लाइन अर्गुमेंट को अपने स्केच में पढ़ने और उसका उपयोग करने का। इससे आप अपने प्रोग्राम को उपयोगकर्ता के जांच के लिए अधिक लोकप्रिय और उपयोगी बना सकते हैं।

## कैसे करें

कमांड लाइन अर्गुमेंट स्केच पढ़ने के लिए, आपको अपने स्केच में निम्न संरचना का उपयोग करना होगा:

```arduino
int argCount = CommandLine.getArgc(); 
String argValue = CommandLine.getArgv(N);
```

N को आपलोड करने के समय एक्सट्रा अर्गुमेंट के रूप में आपके स्केच उपलब्ध होंगे। उदाहरण के लिए, आप अपने उपयोगकर्ता को स्केच को अपलोड करते समय पासवर्ड भी दे सकते हैं और उसे अपने स्केच में सहेजा जा सकता है।

तो अपने स्केच में अर्गुमेंट पढ़ने का पूरा कोड निम्न हो सकता है:

```arduino
String password = ""; 

int argCount = CommandLine.getArgc(); 

if (argCount == 2) {
  password = CommandLine.getArgv(1); 
}

// अब आप पासवर्ड का उपयोग कर सकते हैं जैसे आप चाहें 
```

इससे आप प्रोग्राम प्रारंभ करने के बाद तुरंत अपने पासवर्ड को सॉफ्टवेयर पर लॉगिन कर सकते हैं।

## गहराई में

कमांड लाइन अर्गुमेंट्स अक्सर मोजूदा प्रोग्राम प्रासंग के साथ जोड़ने के लिए बहुत उपयोगी होते हैं। आप अपने प्रोग्राम में लंबे से लंबे सेटिंग्स को सुनिश्चित कर सकते हैं और उपयोगकर्ता को उसे बदलन
