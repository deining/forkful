## क्यूँ:

क्या आप कभी अपनी आर्डुइनो प्रोजेक्ट में रैन्डम नंबर्स की जरूरत महसूस करते हैं? अगर हाँ, तो आप सही जगह पर हैं! इस ब्लॉग पोस्ट में, हम आपको बताएँगे कि आप बिना किसी दिक्कत के कैसे आर्डुइनो में रैन्डम नंबर्स जनरेट कर सकते हैं।

## कैसे करें:

```Arduino
void setup() {
  Serial.begin(9600);  // सीरियल कम्यूनिकेशन शुरू करें
  randomSeed(analogRead(A0));  // रैन्डम सीड जनरेट करें
}

void loop() {
  int randomNumber = random(1, 10);  // 1 और 10 के बीच एक रैन्डम नंबर जनरेट करें 
  Serial.print("रैन्डम नंबर: ");  // सीरियल मॉनिटर पर प्रिंट करें 
  Serial.println(randomNumber);  
  delay(1000);  // 1 सेकंड का रुकावट करें 
}
```

आपने देखा होगा कि हमने `randomSeed()` और `random()` फंक्शन का उपयोग किया है। `randomSeed()` फंक्शन आर्डुइनो को बताता है कि वह कोनसे वायरल नंबर का उपयोग करके रैन्डम नंबर्स जनरेट करे। आप इसे आर्डुइनो के एनालॉग पिन से कनेक्टेड सेंसर के थ्रू रिडिंग का उपयोग करके कर सकते हैं। `random()` फंक्शन आर्ग्यूमेंट्स के बीच दिए गए रेंज में एक रैन्डम नंबर जनरेट करता है।

## गहराई में जाएं:

अब जब आप आर्डुइनो में रैन्डम नंबर्स जनरेट करना सीख गए हैं, आइए थोड़ी गहराई में जाते हैं। जब आप `randomSeed()` को `analogRead()` से सीधे जनरेटेड वैल्यू के साथ कॉल करते हैं, तो आपको सुनिश्चित करना होगा कि आपका सें
