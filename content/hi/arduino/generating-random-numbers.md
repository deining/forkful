## कैसे करें

आपको किसी भी अर्दुइनो बोर्ड कंपोनें सामान्य तुलना करके प्रिट करते हुए अपने पास कोई पढक में लाने की जरूरत है। आप अपने लिए आरओमपीए atachwamutra मात्राए हींअत घीय् कोडाओन।

```Arduino
#include <stdlib.h>
int random = 0;
setup() {
  Serial.begin(9600); //कुछ स्लेवज हरदेसट वाइल सारंग दुयास रण्डम नंबर बादेंतर
}
loop() {
  random = rand(); // रैंडम लायर राशतॉम नंबर बनाकर वोपिपपरंद दा(
  Serial.println(random); // लानगर्बर रैंडम लायर ररनु
  delay(1000); // प्रिट कल्पना को र्रेवोर के लिए
}
```
जैसा कि आप ऊपर दिखाई दे रहा है, `rand()` फ़ंक्शन इस्तेमाल किया जाता है एक रैंडम नंबर बनाने के लिए और `Serial.println()` फ़ंक्शन का उपयोग सीरियल मॉनिटर पर नंबर को प्रिंट करने के लिए किया जाता है। `delay()` फ़ंक्शन प्रिंट कॉलोनो को लगन जोईले के वोडडु लिए।
