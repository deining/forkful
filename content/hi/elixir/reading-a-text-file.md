## क्यों

क्या आपने कभी टेक्स्ट फ़ाइल में उपलब्ध डेटा को देखने के बारे में सोचा है? यदि हाँ, तो आपको यह लेख पढ़ने से फ़ायदा हो सकता है। इसमें हम ऐसे तीन तरीके सीखेंगे जिनसे आप एक टेक्स्ट फ़ाइल से डेटा पढ़ सकते हैं और उसे अपने आवश्यकताओं के अनुसार संशोधित भी कर सकते हैं। तो आइए शुरू करते हैं!

## कैसे करें

```elixir
# फ़ाइल को खोलें और डेटा पढ़ें
file = File.open("data.txt")
data = IO.read(file)

# डेटा को डिक्शनरी में परिवर्तित करें
dict = data |> String.split("\n") |> Enum.reduce(%{}, fn line, acc ->
  [key, value] = String.split(line, ":")
  Map.put(acc, key, value)
end)

# अनुचित डेटा को निकालें और संशोधित डेटा लिखें
correct_data = dict |> Map.delete("address") |> Map.put("phone", "555-555-5555")
File.write("correct_data.txt", Map.to_string(correct_data))
```

यहां दिए गए कोड उदाहरणों के संग्रह में आप देख सकते हैं कि कैसे एक टेक्स्ट फ़ाइल को खोला जाता है, उसमें से डेटा पढ़ा जाता है और उसे एक डिक्शनरी में परिवर्तित किया जाता है। फिर, हम डेटा में कुछ परिवर्तन करते हैं और उसे अन्य फ़ाइल में लिखते हैं। आप इस कोड प्रोसेस को अपनी आवश्यकताओं के अनुसार संशोधित कर सकते हैं और अपने डेटा को सहेज सकते हैं।
