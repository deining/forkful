## क्यों

एक टेक्स्ट फ़ाइल पढ़ना वह प्रक्रिया है जो हर एक विकासक को अपने कोड में किया गया बदलाव को दर्शाने के लिए महत्वपूर्ण है। यदि आप एलिक्सियर प्रोग्रामिंग के लिए उदाहरण देखना चाहते हैं या अपने काम को स्थानांतरित करना चाहते हैं, तो टेक्स्ट फ़ाइल पढ़ना क्रिया आपके लिए ही है।

## कैसे

```Elixir
path = "example.txt"
file = File.read(path)

IO.puts(file)

# output:
# This is an example text file.
# It contains some sample text for demonstration purposes.
```

ऊपर कोड ब्लॉक में हमने एक उदाहरण टेक्स्ट फ़ाइल को पढ़ा और इसकी पहली पंक्ति को मानचित्रित किया है। हमने टेक्स्ट फ़ाइल वास्तविक नाम द्वारा पढ़ा है जो उसके जहाँ पर स्थित है उसे निर्देशित करता है। इसके बाद, हमने `IO.puts` उपयोग कर उस टेक्स्ट को प्रिंट किया है।

## गहराई में

टेक्स्ट फ़ाइल को पढ़ने के लिए एलिक्सियर इनजन आपको एक `File` मॉड्यूल प्रदान करता है जो इस प्रक्रिया को बहुत ही सरल बनाता है। आप उसे समूहित फ़ॉर्मैट में भी पढ़ सकते हैं, जो आपको बहुत स्पष्ट और स्थानांतरित काम के लिए मदद कर सकता है।
