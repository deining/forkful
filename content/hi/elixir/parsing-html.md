# क्यों

HTML टैग को पार्स करना आवश्यक हो सकता है, क्योंकि ये वेब डेवलपमेंट में बहुत ही महत्वपूर्ण है। यह उपयोगकर्ता द्वारा बनाए गए कंटेंट को स्ट्रक्चर और स्टाइल दोनों को सुनिश्चित करने के लिए इस्तेमाल किया जाता है। इलिक्सिर में HTML पार्सिंग करने के लिए कई लाइब्रेरीज उपलब्ध हैं, जिनका इस्तेमाल करके आप इस काम को आसानी से कर सकते हैं।

## कैसे करें

इलिक्सिर में हम मॉक्स नामक लाइब्रेरी का इस्तेमाल कर सकते हैं HTML पार्सिंग के लिए। इसके लिए हम आगे दिए गए कोड का उपयोग कर सकते हैं:

```
Elixir Code Block:

require Mox
html = "<h1>Hello World</h1>"
Mox.parse(html)
```

इस कोड ब्लॉक को रन करने के बाद हमें निम्न आउटपुट मिलेगा:

```
Output:

{:ok, [{:tag, "h1", [], ["Hello World"]}]}
```

लहज़ा है! अब आप इस आउटपुट को अन्य अपनी आवश्यकताओं के अनुसार प्रयोग कर सकते हैं।

## गहराई में जाएं

HTML पार्सिंग के और अनेक तरीके हैं जो इलिक्सिर में उपलब्ध हैं। कुछ लोकप्रिय लाइब्रेरीज के नाम हैं Mox, Floki और Ace। हालांकि, Mox इनमे से सबसे प्रभावी है क्योंकि इसमें प्रत्येक टैग के साथ एक विग्यप्ति मिलती है जो आपको उस टैग के विषय में कुछ जानकारी देती है। आप Mox के डॉक्युमेंटेशन से और इसके उपयोग के अधिक उदाहरण भी देख सकते हैं।
