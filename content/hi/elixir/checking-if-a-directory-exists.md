# क्यों

जब हम फ़ाइलों को सहेजने और प्रबंधित करने के लिए एलिक्सीर प्रोग्रामिंग लैंग्वेज का उपयोग करते हैं, तो हम अक्सर यह प्रश्न पूछते हैं कि क्या यह फ़ोल्डर/डायरेक्टरी मौजूद है? इसलिए, फ़ोल्डरों की मौजूदगी की जांच अहम और उपयोगी होती है।

# कैसे करें

```
Elixir फ़ील्ड नाम = "documents"
यदि System.dir? (नाम) == true do
  IO.puts "फ़ोल्डर $ {नाम} मौजूद है।"
अन्यथा
  IO.puts "फ़ोल्डर $ {नाम} मौजूद नहीं है।"
पुन: प्रयास करें
कहीं
  IO.puts "कोई त्रुटि हुई।"
कहावत का अंत
```

उपरोक्त कोड अपने संदेश को तरह तरह से उदाहरण के साथ प्रदर्शित करता है। यदि फ़ोल्डर मौजूद है, तो हमें संदेश मिलेगा "फ़ोल्डर डॉक्यूमेंट्स मौजूद है" और यदि नहीं, तो हमें संदेश मिलेगा "फ़ोल्डर डॉक्यूमेंट्स मौजूद नहीं है।" यदि कोई त्रुटि होती है, तो हमें संदेश मिलेगा "कोई त्रुटि हुई।"

# गहराई में उतरें

अभी तक हमने System.dir? के उपयोग से अपने दस्तावेज़ फ़ोल्डर की मौजूदगी की जांच की है, लेकिन अधिक संशोधन के लिए, हमें इस फ़ंक्शन के भीतर क्या होता है इसे समझना होगा। System.dir? सच में फ़ोल्डर या डायरेक्टरी के लिए एक एक्सिस्टेंस चेक करता है। यदि फ़ोल्डर या डायरेक्टरी मौजू
