## क्यों:

कमांड लाइन आर्ग्यूमेंट पढ़ने के साथ साथ, एलिक्सिर प्रोग्रामिंग बहुत सारे लाभ प्रदान करता है। इससे, किसी भी प्रोग्राम को इंटरैक्टिव बनाने में और यूजर को अपने प्रोग्राम को अनुकूलित करने में मदद मिलती है।

## कैसे करे:

यदि आपने पहले से ही अपने पास एक एलिक्सिर प्रोजेक्ट बनाया है, तो आप अपने प्रोजेक्ट फोल्डर में नेविगेट करें और निम्न उदाहरण को अपनी एलिक्सिर फाइल में दर्ज करें:

```Elixir
# यादृच्छिक सांख्यिकी
defmodule RandomNumber do
  def generate do
    :rand.uniform(10)
  end
end

# मुख्य फ़ंक्शन
defmodule Main do
  def main(args) do
    IO.puts "आपका यादृच्छिक संख्या:"
    IO.inspect RandomNumber.generate()

    # अगर कोई एरर होता है तो इसका उपयोग करें
    case args do
      [] ->
        IO.puts "कोई सांख्यिक आर्ग्यूमेंट नहीं मिला।"

      [number] ->
        IO.puts "आपके सांख्यिक आर्ग्यूमेंट है: #{number}"
    end
  end
end

# चलाएं
Main.main(System.argv)
```

जब आप इस लाइन आर्ग्यूमेंट वाली एलिक्सिर फाइल को चलाते हैं, तो आपको मान्य संख्या देने का विकल्प मिलेगा। यदि आप कोई संख्या नहीं देते हैं, तो आपको एक संदेश मिलेगा जो आपको याद दिलाएगा कि आपको एक संख्यात्मक आर्ग्यूमेंट प्रदान करने की आवश्यकता है।
