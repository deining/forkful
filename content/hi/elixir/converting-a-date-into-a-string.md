## क्यों

अगर आप Elixir या अन्य किसी भी प्रोग्रामिंग भाषा के साथ काम करते हैं तो आपने शायद कभी किसी दिनांक को स्ट्रिंग में बदलने के बारे में सोचा होगा। डेटा को स्ट्रिंग में बदलने की यह महत्वपूर्ण कार्यप्रणाली बहुत समय बचाती है और ऐसे किसी भी सॉफ्टवेयर में उपयोगी हो सकती है।

## कैसे करें

जब आप Elixir में दिनांक को स्ट्रिंग में बदलते हैं, तो आप सबसे पहले डेटा टाइप को बदलने की जरूरत होती है। आप इस तरह से दिनांक वेहवानियता को उद्दीपित कर सकते हैं:

```Elixir
date = ~D[2020-01-01]
IO.puts DateTime.to_iso8601(date)
```

आपको इस तरह का आउटपुट मिलेगा:

```
2020-01-01T00:00:00
```

अब आप स्ट्रिंग में डेटा को बदल सकते हैं। आप इस तरह से कर सकते हैं:

```Elixir
date = ~D[2020-01-01]
IO.puts Date.to_string(date, [{:format, "dd/MM/yyyy"}])
```

और आपका आउटपुट यह हो सकता है:

```
01/01/2020
```

## गहराई में जायें

जेवा कि हमने ऊपर देखा है, Elixir में दिनांक को स्ट्रिंग में बदलना बहुत ही आसान है। यदि आप चाहे तो आप अपनी वेहवानियता को अनुकूलित करने के लिए अनुकूल प्रोग्राम भी बना सकते हैं। आप अलग-अलग अपेक्षित फॉर्मेट को चुन सकते हैं और अपने वेहवानियता को इस तरह से कन्फ़िगर कर सकते हैं।
