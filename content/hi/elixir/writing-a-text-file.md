## क्यों

टेक्स्ट फ़ाइल लिखने के क्या फायदे हो सकते हैं? वह कल्पना करें कि आपको एक लंबे कोड का सामना करना हो जो किसी भी समय बिगड़ सकता है। एक टेक्स्ट फ़ाइल के माध्यम से आप अपने कोड को संगठित ढंग से बचा सकते हैं और उसे सही तरीके से दोबारा उपयोग कर सकते हैं।

## कैसे करें

```elixir
# टेक्स्ट फ़ाइल बनाएं
File.write!("example.txt", "यह टेक्स्ट फ़ाइल है")

# फ़ाइल से डेटा पढ़ें
data = File.read!("example.txt")

# पढ़ा हुआ डेटा प्रिंट करें
IO.puts(data)
```

उपरोक्त कोड ब्लॉक में हम दिखाए गए तरीके से ईलिक्सियर में टेक्स्ट फ़ाइल को बनाने और उससे डेटा पढ़ने का तरीका दिखाया गया है। पहले हमने `File.write!` फ़ंक्शन का उपयोग करके "example.txt" नाम की फ़ाइल बनाई है और उसमें दिया गया डेटा लिखा है। फिर हमने `File.read!` फ़ंक्शन का उपयोग करके उसी फ़ाइल से डेटा पढ़ा और `IO.puts` फ़ंक्शन का उपयोग करके उस डेटा को प्रिंट किया है।
