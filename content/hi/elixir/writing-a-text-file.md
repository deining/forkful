## क्यों

क्या आप एलिक्सिर प्रोग्रामिंग से परिचित हैं? यदि ऐसा है, तो आपने शायद पाठ फ़ाइलें लिखने के बारे में सुना होगा। पर अगर आप खुद ही एक टेक्स्ट भंडार लिखना चाहते हैं, तो आपको क्या करना होगा? इस आर्टिकल में हम आपको बताएंगे कि आप किस तरह से ये कर सकते हैं।

## कैसे करें

सबसे पहले, हम फाइल हैंडलिंग फंक्शन `File.open/2` का उपयोग करके एक फ़ाइल खोलेंगे। उदाहरण के लिए, हम एक नयी टेक्स्ट फ़ाइल `hello.txt` खोलेंगे।

```Elixir
{:ok, file} = File.open("hello.txt", [:write])
```

अगर फ़ाइल सफ़लतापूर्वक खोल जाती है, तो हम `:ok` के साथ उसे प्राप्त कर सकते हैं। उसके बाद, हम फ़ाइल फ़न्क्शन `IO.write/2` का उपयोग कर सकते हैं जो हमें फ़ाइल में टेक्स्ट लिखने की अनुमति देता है।

```Elixir
IO.write(file, "नमस्ते दुनिया!")
```

अब हम एक आर्ग्युमेंट जोड़ेंगे `[:utf8]` जो हमें फ़ाइल में unicode लिखने की अनुमति देगा। बाद में, हम `IO.puts/2` का उपयोग करेंगे जो हमें फ़ाइल बन्द करने की भी अनुमति देगा।

```Elixir
IO.close(file)
```

और फाइनल आउटपुट एक्सेपेक्ट करें:

```
नमस्ते दुनिया!
```
