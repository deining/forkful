## क्या और क्यों?
किसी भी प्रोग्रामिंग भाषा में टेक्स्ट फाइल लिखना इस बात का प्रमाण है कि हम कंप्यूटर को कुछ निर्देश दे रहे हैं जो स्क्रिप्ट या अन्य प्रोग्राम को इस फाइल के माध्यम से समझने के लिए होते हैं। प्रोग्रामर अपने कोड को स्टोर और साझा करने के लिए भी टेक्स्ट फाइल लिखते हैं।

## कैसे करें:
```Elixir
File.write("file.txt", "यह टेक्स्ट फाइल में लिखा गया है!")
```
इस कोड से हमने "file.txt" फाइल में दी गई पाठ को लिखा है। हम भी एक नया फाइल बना सकते हैं, जैसे:
```Elixir
File.new("new_file.txt")
```
यदि आप के पास पहले से ही एक फाइल हो जो आप दोबारा लिखना चाहते हैं, तो आप इस तरह कर सकते हैं:
```Elixir
File.append("existing_file.txt", "नया पाठ!")
```

## गहराई में जाएं:
टेक्स्ट फाइलों को स्टोर और साझा करने के लिए पहले से ही बहुत सारे विकल्प हैं। Elixir में File का आप्शन टू है, जो कि फाइल लिखने के लिए बहुत आसान करता है। आप भी फाइल ओपन करने, वाइनडिंग करने, और फाइल विशेषताओं को जांचने के लिए File आप्शन का उपयोग कर सकते हैं।

## और भी देखें:
File आप्शन के अलावा, आप और भी विकल्प हैं जो आपको परफेक्ट टेक्स्ट फाइल लिखने में मदद कर सकते हैं। प्याथॉन की तुलना में ईलिक्सिर का उपयोग भी कर सकते हैं। आप चाहे तो File आप्शन को गहराई से जानकारियां के लिए [डॉक्यूमेंटेशन](https://hexdocs.pm/elixir/File.html) पढ़ सकते हैं।
