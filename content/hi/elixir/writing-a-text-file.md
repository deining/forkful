## क्यों
टेक्स्ट फ़ाइल लिखने का कार्य क्यों किया जाता है, यह जानना बहुत ज़रूरी है।

लोग टेक्स्ट फ़ाइल में विभिन्न प्रकार के डेटा को स्टोर करने और उसे स्पर्श एडिटर और अन्य ऐप्स में असानी से एक्सेस करने के लिए इस्तेमाल करते हैं। एक्सेलियर प्रोग्रामिंग में भी टेक्स्ट फ़ाइलें लिखने का उपयोग किया जाता है।

## कैसे करें
टेक्स्ट फ़ाइल लिखने के लिए, पहले एक नया फ़ाइल बनाएं और उसको तीनों कोनों पर से एक संकेतक या विभाजक रहने वाले टेक्स्ट फ़ाइल के लिए User Datagram Protocol (UDP) पोर्ट नंबर दर्ज करें। उदाहरण के लिए, आप इस प्रकार से एक टेक्स्ट फ़ाइल बना सकते हैं:

```
Elixir defmodule TextFile do
  def create(file_name) do
    file = File.open(file_name, [:utf8, :write])
    File.write(file, "This is a sample text file")
    File.close(file)
  end
end
```

पुस्तकालय संरक्षक बटन को दबाकर आप अपने अपने फ़ाैजित मोशौरफ के पूर्णानुत्तम वोकेले दस्तक आरो। ध्यान रखें कि यदि आप एक नया फ़ाैजित या फ़ाैजित मोशौरफ संचित करना चाहते हैं तो आपको दूसरा प्रोग्राम के बारे में विषय और दृष्टते प्रोग्राम पर संचितिपर्णे हैं।
