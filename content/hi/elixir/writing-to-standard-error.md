# मैं क्यों छापने के लिए कोई त्रुटि स्टैंडर्ड करने के लिए एलिक्सियर का उपयोग करूँगा? 

एलिक्सियर एक बहुत ही शक्तिशाली और सुरक्षित प्रोग्रामिंग भाषा है जिसका उपयोग करने से आप अपने कोड को सुरक्षित तरीके से चला सकते हैं। इसके साथ ही, एलिक्सियर में डीबगिंग के लिए उपयोगी फंक्शन हैं जिनका उपयोग करके आप आसानी से अपने कोड में त्रुटियों को पता लगा सकते हैं।

## कैसे क्योडिंग करें?

इस ब्लॉग पोस्ट में, हम स्टैंडर्ड त्रुटि में विवरण छापने के बारे में बात करेंगे। यदि आपको अपने कोड में कोई त्रुटि मिलती है तो आप चाहेंगे कि वह त्रुटि किसी तरह से आपको पता चले ताकि आप उसे ठीक कर सकें। आप इसके लिए स्टैंडर्ड त्रुटि में डिफाल्ट निर्देशों के साथ कोड ब्लॉक को छापने के लिए ```:stderr.print(मल्टिएक्स {मुद्रण, लाइन})``` फंक्शन का उपयोग कर सकते हैं। नीचे दिए गए उदाहरण में, हम एक संख्या को 2 से गुणा करने के लिए कोड दिया है जिसमें एक त्रुटि है। हम इस त्रुटि को छाप देंगे ताकि हम इसे ठीक कर सकें।

```elixir
def multiply_by_two(number) do
  if is_number(number) do
    number * 2
  else
    :stderr.print("एक संख्या पास करें।")
  end
end

# उदाहरण
IO.puts multiply_by_two("दो") 
# त्रुटि: एक संख्या पास करें।
```
