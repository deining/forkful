इलिक्सियर (Elixir) प्रोग्रामिंग भाषा अत्यंत शक्तिशाली और जाने माने है। इस भाषा का उपयोग स्कालेबल वेब एप्लिकेशन्स बनाने में किया जाता है जो मॉडर्न और बेहतर हों। लेकिन, कभी-कभी हम ऐसी स्थिति में आते हैं जब हमें कोई त्रुटि पता चलती है और हमें उसे सही करने के लिए सही स्थान चाहिए। इस तरह की समस्याओं को हल करने के लिए हम standard error पर लेखन का उपयोग कर सकते हैं। 

## क्यों

अक्सर हमारे कोड में त्रुटियां हो सकती हैं और उन्हें पाने के लिए हमें लंबे समय तक दुगनी कोशिश करनी पड़ सकती है। लेकिन, इलिक्सियर में हमें standard error को दर्ज करके उसे आसानी से पकड़ सकते हैं और उसे सही कर सकते हैं। यह हमारे कोड को सुधारने में हमारी मदद करता है और हमारे कोड और उसमें होने वाली त्रुटियों को स्पष्ट दिखाता है। 

## कैसे करें

यहाँ हम इलिक्सियर में standard error को कैसे दर्ज कर सकते हैं, उसके उदाहरण और उससे प्राप्त होने वाले आउटपुट को देखेंगे। 

```
Elixir.put_stderr("यह एक उदाहरण है।")
```
आउटपुट:
```
यह एक उदाहरण है।
```

```
Elixir.put_stderr("इलिक्सियर में कोडिंग करना आसान है।")
```
आउटपुट:
```
इलिक्सियर में कोडिंग करना आसान है।
```

हम आसानी से देख सकते हैं कि `put_stderr` फंक्शन के माध्यम से हम standard error पर टेक्स्ट दर्ज कर सकते हैं।
