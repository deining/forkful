## क्यों

 आपने कभी भी इंटरनेट पर साइट्स या वेब एप्लिकेशन्स का इस्तेमाल किया है तो आपने HTTP (हाइपरटेक्स्ट ट्रांसफर प्रोटोकॉल) के बारे में सुना होगा। इसमें से कुछ साइट्स और ऐप्स अपनी सुरक्षा सुनिश्चित करने के लिए HTTP रिक्वेस्ट में बेसिक ऑथेंटिकेशन (प्रमाणीकरण) का इस्तेमाल करते हैं। इस तरह के रिक्वेस्ट को भेजने के लिए ग्लीम प्रोग्रामिंग लैंग्वेज का इस्तेमाल किया जा सकता है, यह आर्टिकल आपको इसके बारे में विस्तार से बताएगा। 

## कैसे करें

अगर आप ग्लीम में एक HTTP रिक्वेस्ट भेजना चाहते हैं जो बेसिक ऑथेंटिकेशन के साथ हो, तो सबसे पहले आपको `httpc` मॉड्यूल को इम्पोर्ट करना होगा। इसके बाद, आपको `basic_auth` फंक्शन को यूज करके अपने क्रेडेंशियल्स (उपयोगर्ता नाम और पासवर्ड) को पारामीटर के रूप में पास करना होगा। यदि सभी सेटिंग्स सही हैं तो आपको सफलतापूर्वक एक HTTP रिक्वेस्ट भेजने का प्रोसेस आसानी से पूरा हो जाएगा। नीचे दिए गए कोड ब्लॉक आपको इसके बारे में और समझाएंगे।

```
import httpc

// प्रमाणीकरण के साथ HTTP रिक्वेस्ट भेजना
basic_auth("उपयोगर्ता नाम", "पासवर्ड")
```
