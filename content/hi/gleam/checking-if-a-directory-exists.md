## क्यों

एक निर्देशिका का अस्तित्व जांचने में आप क्यों लगे हो सकते हैं? यह अस्तित्व काफी महत्वपूर्ण हो सकता है, क्योंकि यह आपको अपने कोड में सही गतिविधियों को सुनिश्चित करने में मदद कर सकता है।

## कैसे करें

इसी काम के लिए हम आपको Gleam में निर्देशिका का अस्तित्व जांचने के लिए कुछ सरल कोड दिखाने जा रहे हैं।

```Gleam
import gleam/stdio
import gleam/file

pub fn main() {
  // निर्देशिका का नाम दूरभाष से प्राप्त करते हैं।
  let dir_name = stdio.read_line("~ Enter directory name: ")

  // अब निर्देशिका की जाँच करते हैं और सभी मान को फ़ोन करते हैं।
  let dir_exists = file.dir_exists(dir_name)

  // नापास को छापे दें।
  case dir_exists {
    E.Err(f) ->  
      stdio.format("Directory does not exist: {:s}", [dir_name])
    _ ->
      stdio.print("Directory exists!")
  }
}
```

## गहराई गोत

निर्देशिकाओं के लिए अस्तित्व जांच करने के लिए आपको ध्यान से Gleam के उपयोग की आवश्यकता हो सकती है। ज्यादातर लोग ध्यान नहीं देते हैं, लेकिन जब तक आप एक ठीक स्थान नहीं हैं और उस तक आपका कोड सही तरीके से काम नहीं करता है। हमेशा ध्यान रखें कि आप कैसे और क्यों कार्यशील तरीकों का उपयोग कर सकते हैं।
