## क्यों

क्या आप जानते हैं कि ग्लीम भाषा में जोड़ने तिथि को स्ट्रिंग में बदलना अत्यंत उपयोगी हो सकता है? इस ब्लॉग पोस्ट में, हम आपको कुछ आसान कोडिंग उदाहरण देंगे जो आपको इस प्रकार की कार्य भूमिका में मदद कर सकते हैं।

## कैसे करें

फ़रवरी 20, 2021 को स्ट्रिंग में तारीख को कनवर्ट करने के लिए, आपको निम्नलिखित कोड को अपने ग्लीम फ़ाइल में जोड़ना होगा:

```Gleam
let date = #date(2021, 02, 20)
let string = date |> Date.to_string
// Output: "2021-02-20"
```

यदि आप 1-9 के लिए एक अंक का महिना या दिन दर्ज करते हैं, तो ग्लीम उन्हें स्वचालित रूप से 2 दिगित तक बढ़ा देगा (01, 02, 03, आदि)। इसी तरह, यदि तारीख 13 से कम है, तो ग्लीम उसे स्वचालित रूप से "0" के साथ शुरू करेगा।

```Gleam
let date = #date(2021, 2, 7)
let string = date |> Date.to_string
// Output: "2021-02-07"
```

इसके अलावा, आप अपनी स्ट्रिंग के रूप में इस तारीख को भी प्रदर्शित कर सकते हैं:

```Gleam
let date = #date(2021, 02, 20)
let string = date
    |> Date.format("%A, %B %d, %Y")
// Output: "Saturday, February 20, 2021"
```
