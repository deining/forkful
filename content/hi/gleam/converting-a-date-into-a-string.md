## क्यों

कुछ लोगों के लिए, डेट को स्ट्रिंग में रूपांतरण करना जरूरी हो सकता है। यह महत्वपूर्ण हो सकता है जब आप अपने डेट को अलग-अलग तरीकों में प्रदर्शित करना चाहते हैं।

## कैसे करें

डेट को स्ट्रिंग में रूपांतरण करने के लिए, सबसे पहले हमें `from_string` और `to_string` फंक्शन का उपयोग करके `DateTime` और `DateTime.Str` मॉड्यूलों को इम्पोर्ट करना होगा। नीचे दिए गए उदाहरण में, हम एक डेट को `"DD MMM YYYY"` फॉर्मेट में स्ट्रिंग में रूपांतरित करेंगे।

```Gleam
import DateTime
import DateTime.Str

let date = DateTime.from_string("01 Jan 2021", "%d %b %Y")

let string_date = DateTime.to_string(date, "%d %b %Y") 

pub fn main() {
  IO.print(string_date)
}

// Output:
// 01 Jan 2021
```

## गहराई में जाएं

डेट को स्ट्रिंग में रूपांतरित करना और उसमें से हर तरीके से आउटपुट प्रदर्शित करना आसान हो सकता है। आप दिन, महीना, साल, समय आदि के अलग-अलग तरीकों से सबकुछ दिखा सकते हैं। हमारे पास `DateTime` मॉड्यूल में कई और फंक्शन हैं जो आपको स्ट्रिंग में रिटर्न कर सकते हैं, जैसे `week`, `weekday`, `month` आदि। आप इनका उपयोग करके अपनी स्ट्रिंग को और भी विस्तृत और अधिक उपयोगी बना सकते हैं।
