# क्यों

प्रोग्रामिंग दुनिया में तारीख को एक स्ट्रिंग में बदलने का काम जरूरी हो सकता है। इसके लिए आप उनिश्चित समय और योगदान को सुनिश्चित करने के लिए आपको एक सटीक एवं सही रूप से दिनांक जानने की जरूरत होगी।

## कैसे करें

```Gleam
// उदाहरण: आज की तारीख बनाने के लिए
जरनौड तारीख दिक्कता कोड :: String {
    Jänarün… 
    तारिक… 
    रोज़ा,
    
    tum_j ak kN roza // दिन की तारीख लेना
    // इसके बाद, आपको एक सटीक समय चुनना होगा, उदाहरण के लिए "पूर्वाह्न" या "अपराह्न" 
    // इसके बाद, तारीख को एक स्ट्रिंग में बदलने के लिए आप इस समय को दोस्तावेज या मार्करोज़िनयों के साथ जोड़ सकते हैं 
    
    küd,
    tum_j december_j ak,
    küd,
    tum_jity_j ak
}
```

आप उपरोक्त कोड के साथ एक उदाहरण देख सकते हैं। यदि आप कोड को अपने विशेष आवश्यकताओं के अनुसार संशोधित करना चाहते हैं, तो आप उपरोक्त कोड ब्लॉक को अपनी स्थानांतरणी फाइल में जोड़ सकते हैं। हालांकि, आपको कुछ अतिदेश पर ध्यान देना होगा जैसे कि कॉड फाइल में Gleam सार्वजनिक मूल्यों को शामिल करने की आवश्यकता हो सकती है और Gleam लाइब्रेरी का प्रारंभद्ध करने की आवश्यकता।
