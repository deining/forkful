## क्यों

एक स्ट्रिंग को लोअर केस में बदलने में रुचि रखने के लिए *क्यों* किसी को अपना समय निवेश करना चाहिए।

## कैसे करें

अगर आप एक स्ट्रिंग को लोअर केस में बदलना चाहते हैं, तो आप ग्लीम में इसके लिए बिल्ट-इन फंक्शन `String.to_lower` का उपयोग कर सकते हैं। यह फंक्शन दिए गए स्ट्रिंग को लोअर केस में रूपांतरित करता है।

```Gleam
let my_string = "Gleam Programming";
let lower_case_string = String.to_lower(my_string);

// Output: "gleam programming"
```

आप चाहें तो अपनी स्ट्रिंग के लिए कस्टम फंक्शन भी बना सकते हैं जो उसे लोअर केस में बदलता है। यह फंक्शन सरल तरीके से ग्लीम में बनाया जा सकता है और आप अपनी जरूरत के अनुसार उसे बदल सकते हैं।

```Gleam
fn to_lower_case(string) {
    String.to_list(string)
    |> List.map(fn (char) -> 
        if char >= "A" && char <= "Z" {
            Char.to_lower(char)
        } else {
            char
        }
    )
    |> String.from_list
}

let my_string = "Gleam Programming";
let lower_case_string = to_lower_case(my_string);

// Output: "gleam programming"
```

## गहराई में चलें

स्ट्रिंग को लोअर केस में बदलना एक आम फंक्शन है जो शब्दों को समलैंगिक बनाने के लिए उपयोगी हो सकता है। यह अक्सर यूजरनेम या संक्षिप्त संदेशों को संशोधित करने के लिए भी इस्तेमाल किया जाता है। आप यह फंक्शन अलग-अलग तरीकों से आगे बढ़ा सकते हैं जैसे कि बड़ी टेक्स्ट फाइल्स में एक स्ट्रिंग को स्विच केस में कॉन्वर्ट करना या अनुवाद सेवाओं में उपयोग करना।
