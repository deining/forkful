## क्यों

अगर आप डेटा को बाद में खोजने के लिए एक व्यावसायिक उपयोगकर्ता हैं, तो आपको आपके डेटा को ध्यानपूर्वक संगरोधित रखने की जरूरत होती है। इसके लिए, आप substrings को निकालने की आवश्यकता हो सकती है। यह आपको अपने डेटा के स्ट्रिंग के अंशों पर काम करने की अनुमति देता है, जो आपको अपने उद्धरण सीधे और मजबूत बनाता है।

## कैसे

```Gleam
// एक स्ट्रिंग से सभी सबस्ट्रिंग्स निकालना
let str = "विसर्जन";
let subs = substring.all(str);

// एक स्ट्रिंग से सभी सबस्ट्रिंग्स और उनके लंबाई
let str = "आसानी से कोडिंग";
let subs = substring.all_with_length(str);

// एक स्ट्रिंग से एक निर्दिष्ट सबस्ट्रिंग निकालना
let str = "हिंदी सीखना है";
let subs = substring.at(str, 3, 5);

// सांख्यिकीय उदाहरण
// स्ट्रिंग: "भाषा पढ़ने से सीखो"
// सबस्ट्रिंग: "भाषा"
let str = "भाषा पढ़ने से सीखो";
let subs = substring.before(str, " पढ़ने");

// सांख्यिकीय उदाहरण
// स्ट्रिंग: "भाषा को खासियत से सीखो"
// सबस्ट्रिंग: "सीखो"
let str = "भाषा को खासियत से सीखो";
let subs = substring.after(str, " खासियत से");
```

उपरोक्त उदाहरणों में, हमने स्ट्रिंग अलग करने के लिए कुछ साधारण फ़ंक्शन दिखाए हैं। आप आसानी से इन उदाहरणों को अपने उद्धरण उपयोग कर सकते हैं और अपने कोड को और अधिक संगठित बना सकते हैं।
