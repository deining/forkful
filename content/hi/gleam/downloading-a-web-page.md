## यो क्यों

अगर आप एक वेब पेज को डाउनलोड करना चाहते हैं, तो आप उसकी सामग्री को अपने डिवाइस पर देख सकते हैं, समायोजित कर सकते हैं, और बिना इंटरनेट की आवश्यकता के उस पेज को बार-बार पढ़ सकते हैं।

## कैसे करें

```
Gleam डाउनलोड अनुरोध (Web पता) (रिक्तक) {
  केएचटीटीपी.उपकारकीस.किराएमेंदी (अनुरोध) .मालिकता_आपूर्ति (|देखो डाउनलोड_निर्देशक, थूक)
    .खेलीय (|अमान्य, सफलता):
      कामकाज.बोल्ड 'आमान्य निर्देशक {}', अमान्य
      छोड़ना (अमान्य)
    उच्चोच्तिका.{}खेलीय_निर्देशक (सफलता)
}
```

ध्यान दें: यदि आपका वेब पेज हस्ताक्षर या प्रवेश दक्षता को आवश्यकता रखता है, तो आपको सही अनुरोध प्रस्तुत करना होगा। अन्यथा, आपको खाली उपयोग अन्न दे सकता है। आप अपने निर्देशक में अन्य विकल्प भी जोड़ सकते हैं, जैसे कि कुकीज़, रेफरर, आदि।

## डीप डाइव

Web पेज को डाउनलोड करने के लिए, Gleam में आपको HTTP कलस उपयोग करना होगा। आप अनुरोध की URL और उपयोगकर्ता जानकारी (जैसे किसी प्रकार का सत्यापन) को सही ढंग से प्रस्तुत कर सकते हैं। इसके अलावा, आप अपने डाउनलोड फ़ाइल को किसी अन्य निर्देशक में भी सुरक्षित कर सकते हैं, जिससे आप उसे आसानी से एक बार और डाउनलोड कर सकते हैं।
