## क्यों

कंडमेंट लाइन एर्ग्यूमेंट पढ़ना सीखना इसलिए जरूरी है क्योंकि यह आपको अपनी कोडिंग को विशेष बनाता है। यह आपको अपने प्रोजेक्ट को अधिक व्यावहारिक बनाने में मदद करता है और उत्पादकता को बढ़ाता है।

## कैसे करें

कंडमेंट लाइन एर्ग्यूमेंट स्वचालित रूप से प्रोग्राम को अपने उपयोगकर्ता से संपादित करने की अनुमति देता है। इसका उपयोग करके, आप उपयोगकर्ताओं से विवरण जान सकते हैं और उसी के अनुसार अपने कोड को बदल सकते हैं। यह आपको भी लिखता में अनुकूलता और सार्वजनिक विन्यास का भी समर्थन करता है।

```Gleam
fn main(args) {
  println("आपके द्वारा पास किए गए आर्ग्यूमेंट्स");
  for arg in args {
    println(arg);
  }
}
```

आप उपयोगकर्ता द्वारा पास किए गए सभी आर्ग्यूमेंट्स को प्रिंट कर सकते हैं। `args` एक सीवी है जिसमें सभी आर्ग्यूमेंट्स को स्ट्रिंग के रूप में संग्रहीत किया गया है।

उदाहरण के लिए, अगर आप कोड को निम्नलिखित आर्ग्यूमेंट्स के साथ चलाते हैं:

```sh
gleam run my_code.gleam hello
```

तो आपको निम्न आउटपुट मिलेगा:

```
आपके द्वारा पास किए गए आर्ग्यूमेंट्स
hello
```
