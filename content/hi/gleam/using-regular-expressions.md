## क्यों

Regular Expressions क्या हैं और उन्हें क्यों सीखना चाहिए? हमारे इस ब्लॉग पोस्ट में हम आपको बताएंगे कि यह प्रोग्रामिंग टूल आपके लिए कितना उपयोगी हो सकता है।

## कैसे

यदि आप ग्लीम प्रोग्रामिंग में नए हैं, तो आपको सबसे पहले रेगुलर एक्सप्रेशन्स की परिभाषा समझनी होगी। उसके बाद, आप निम्न उदाहरणों के साथ इस्तेमाल करके रेगुलर एक्सप्रेशन्स को देख सकते हैं:

```Gleam
let text = "यह बिल्कुल सही है।"
let regex = ~r/सही/

if Regex.match(regex, text) == Nil {
  Ok("बिल्कुल नहीं सही है।")
} else {
  Ok("बिल्कुल सही है।")
}
```

उपरोक्त कोड उन लोगों के लिए उदाहरण है जो रेगुलर एक्सप्रेशन्स को समझना चाहते हैं। आप अपने समझने के लिए इसे अपने ग्लीम प्रोजेक्ट में भी इस्तेमाल कर सकते हैं।

## गहरी तकनीक

रेगुलर एक्सप्रेशन्स में कई विशिष्ट लक्षण होते हैं जो उन्हें और अधिक शक्तिशाली बनाते हैं। आप पूरी तकनीक को समझने के लिए ग्लीम के अधिक विशेषताओं को समझ सकते हैं जैसे हेडोक्स और जनेरेटर। साथ ही आप इस लेख को भी संदर्भ में ले सकते हैं जो रेगुलर एक्सप्रेशन्स के विस्तारित रूप में बताता है।
