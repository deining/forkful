## क्यों

आज के दौर में हर एक कंप्यूटर कार्य या एप्लिकेशन आपको टेक्स्ट से संबंधित जानकारी को दर्शाता है। इसलिए, टेक्स्ट का लंबाई या उसमें कितने शब्द हैं का पता होना बहुत महत्वपूर्ण है। इसलिए, Gleam में एक श्रृंखला को बाहरी स्रोतों से प्रकाशित करने के लिए एक रूम प्रफरेंस का उपयोग करके स्ट्रिंग लंबाई जानना बहुत महत्वपूर्ण है।

## कैसे करें

इस निर्देशांक में, हम आपको Gleam में स्ट्रिंग लंबाई कैसे प्राप्त कर सकते हैं इसके बारे में बताएंगे। मूल ढांचे का उपयोग करते हुए, हम कुछ सरल उदाहरणों के साथ संपूर्ण प्रक्रिया दिखाएंगे।

```Gleam
let string = "Hello, world!"

// बाहरी स्रोतों से प्रकाशित करने के लिए रूम प्रफ़रेंस का इस्तेमाल
let length = string.byte_size()

// प्रदर्शित करें
io.println("String length is: ", length)

// आउटपुट:
// String length is: 13
```

इस उदाहरण में, हमने एक स्ट्रिंग दर्ज किया है और उसकी लंबाई Gleam कोड द्वारा प्राप्त की है। यह बहुत आसान है और आप इसे अपने खुद के कोड में भी लागू कर सकते हैं।

## गहराई में

स्ट्रिंग लंबाई का पता लगाना बहुत सरल है, लेकिन जब आप गहराई में जाते हैं तो इसके कई उपयोग भी हो सकते हैं। इसमें से कुछ महत्वपूर्ण उपयोग हैं:

- अधिकतम स्ट्रिंग लंबाई का पता लगाना: यदि आप एक वेब एप्लिकेशन बनाने की योजना
