## क्यों

डीबग आउटपुट प्रिंट करने के लिए क्यों किसी को संलग्न किया?

## कैसे

जब हम अपना कोड डीबग कर रहे हैं, तो इसमें कुछ गड़बड़ियों की संभावना होती है। इसको पता करने के लिए हम debug लाइब्रेरी का उपयोग करते हैं जिसका प्रिंट आउटपुट हमारे लिए बहुत महत्वपूर्ण हो सकता है। नीचे दिए गए कोड ब्लॉक में हम इसके उपयोग का तरीका देखेंगे।

```Gleam
import gleam/debug

let foo = 42

// अपने कोड को डीबग करने के लिए हम फॉलोइंग कोड प्रिंट कर सकते हैं
debug.print("foo बारे में जानकारी", foo)
```

जब हम ये कोड रन करेंगे, तो हमारी कंसोल पर निम्नलिखित आउटपुट प्रिंट होगा:

```
foo बारे में जानकारी 42
```

## गहराई

डीबग आउटपुट प्रिंट करने के लिए कुछ अतिरिक्त तरीके भी हैं। हमारी डीबग फंक्शन में हम अन्य प्राथमिक डेटा टाइप्स को भी प्रिंट कर सकते हैं जैसे डीबग पहुंच कोई काले बॉक्स में रहने की जगह। इससे हमें अपने कोड को डीबग करने में अधिक मदद मिल सकती है।
