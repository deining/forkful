## क्यों?

अगर आप अपने ग्लीम प्रोग्राम में किसी निश्चित पैटर्न से मेल खाते हुए अक्षरों को हटाना चाहते हैं, तो यह ब्लॉग पोस्ट आपके लिए है।

## कैसे करें?

```Gleam
let pattern = "[a-z]"
let str = "Gleam programming is fun!"

let updated_str = gstring.replace(str, pattern, "")

io.println(updated_str)
```

आपको सिर्फ कुछ लाइन बदलने की जरूरत होगी, जैसे कि पैटर्न और स्ट्रिंग का अपडेटेड नाम। उपरोक्त कोड आपको उदाहरण के रूप में दिखाता है कि कैसे आप अपने ग्लीम प्रोग्राम में charactersMatchingPattern फ़ंक्शन का उपयोग कर सकते हैं।

```
Gleam programming  fun!
```

## डीप डाइव

जिन अक्षरों को हम पैटर्न की वजह से हटाते हैं, उन्हें हम आमतौर पर "स्पेशियल कैरेक्टर्स" कहते हैं। जैसे कि ऊपर के उदाहरण में हमने सामान्य अल्फ़ाबेट के साथ "-" भी हटा दिया है। आप अपने पैटर्न को अनुकूलित कर सकते हैं और इस फ़ंक्शन का उपयोग करके अन्य स्पेशियल कैरेक्टर्स को भी हटा सकते हैं।
