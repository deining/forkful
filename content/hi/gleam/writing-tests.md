## क्यों

प्रोग्रामिंग में अद्यतन बनाओ इससे आपको असंभव से संभव तक का सफर मिलता है। टेस्ट लिखने में कमी करने से यह पहल अभिनंदन की बात है।

## कैसे करें

टेस्टिंग कोड नामवंत और स्पष्ट बनाने में मदद करता है। इससे आपको अनुप्रयोग में गड़बड़ी कम करने में मदद मिलता है। आप टेस्ट लिखने के लिए `initialise_test()` फंक्शन का उपयोग कर सकते हैं, जो आपके हर टेस्ट के आधार पर एक नया `test` के समापन में चयनित होगा। नीचे दिए गए उदाहरण देखें:

```Gleam
import gleam/test
import gleam/io

initialise_test()

test "गणित" {
  assert.equal(2+2, 4)
}

test "कार्यात्मक संख्या" {
  assert.equal(10*5, 50)
}

run_tests()
```

आपको यह आउटपुट मिलेगा:

```
गणित पास।
कार्यात्मक संख्या पास।
```

## गहराई में जाएं

टेस्ट लिखना स्वास्थ्यपूर्ण और कारगर कूटबन्धता शानदार ऐसे आपका प्रोजेक्ट पारित करता है। आप अपनी वेबसाइट पर प्रोग्रामिंग सीख सकते हैं।

## देखें ऐसा

यदि आप और गहराई जानना चाहते हैं या प्रोग्रामिंग को सीखना चाहते हैं, तो नीचे दिए गए लिंक देखें। और अपने उदाहरण शेयर करें जो आपने टेस्ट लिखने में पाया है!
