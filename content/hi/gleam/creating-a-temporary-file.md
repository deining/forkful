# क्यों

दुनिया में कोई भी भविष्य को नहीं जानता। इसलिए अभी कुछ भी स्थायी है नहीं। यदि आप काम कर रहे हैं, तो आपको अपने पास पिक्टी फाइल बनाने की जरूरत हो सकती है। इसलिए आपको अस्थायी फ़ाइल बनानी चाहिए।

# कैसे करे

अब हम प्रोग्रामिंग के बारे में बात करेंगे। आप जानते हैं कि गोलियों की रचना जख्मों के लिए है। उनमें कुछ भी भविष्य नहीं है। हम गोलियाँ बना सकते हैं! ग्लीम में अस्थायी फ़ाइल कैसे बनाते हैं यह देखते हैं।

```Gleam
import gleam/file

temp_file = file.create_temp()

file.write(temp_file, "हेलो, यह अस्थायी फ़ाइल है!")

read_file = file.read(temp_file)

print("यह फाइल पढ़ रहा है: " ++ read_file)

file.delete(temp_file)
```

आपको देखने को मिलेगा:

```
यह फाइल पढ़ रहा है: हेलो, यह अस्थायी फ़ाइल है!
```

# गहराई

अस्थायी फ़ाइले उपयोगकर्ताओं के द्वारा अनुभव करने वाली नई तकनीक है। यह आसानी से बनाया और उसे हटाया जा सकता है। इसलिए आपको अस्थायी फ़ाइलें बनाने की आवश्यकता है जब आपको एक स्थायी फ़ाइल की जगह अस्थायी डेटा की जगह का उपयोग करना हो। इससे आपके कंप्यूटर स्पीड में भी सुधार होगा।

# अधिक जानकारी

यदि आप ग्लीम के बारे में अधिक जानना चाहते हैं तो आप हमारे के निम्नलिखित स्रोतों से मदद ले सकते हैं।
