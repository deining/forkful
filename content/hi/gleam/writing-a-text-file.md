Gleam विश्व के प्रगतिशील और प्रभावशाली प्रोग्रामिंग भाषाओं में से एक है। यह हाई- परफॉर्मेंस, सुरक्षित, और स्केलेबल प्रोग्रामिंग को समर्थन करता है। इसके द्वारा, आप सरलता से उत्कृष्ट गोरों तैयार कर सकते हैं और भाषाओं को जु‍ड़ने के लिए संरक्षित सर्विसों के साथ डेटा का प्रबंधन कर सकते हैं।

## क्यों
कोई भी टेक्स्ट फ़ाइल लिखने का मुख्य कारण हो सकता है कि आप अपने कोड को संरक्षित रखना चाहते हैं या फिर किसी दूसरे व्यक्ति को आपके कोड को देखने के लिए दे सकते हैं। इससे आपका कोड सुरक्षित रहता है और आप और अन्य व्यक्तियों को उस पर बार बार काम करने की ज़रूरत नहीं पड़ती।

## कैसे करें
टेक्स्ट फ़ाइल लिखने के लिए, आपको सबसे पहले गोरों को इनिशियलाइज करना होगा। इसके बाद, आप फ़ाइल हैंडलर बनाकर उससे दस्तावेज़ात लिख सकते हैं। नीचे दिए गए उदाहरण में, हम "hello.txt" नाम की एक नई फ़ाइल बना रहे हैं और उसमें "Hello, World!" टेक्स्ट लिख रहे हैं।

```Gleam
fn main() {
    let file = File.create("hello.txt").unwrap()

    let output = "Hello, World!"
    File.write(file, output)
}
```

इस सम्पूर्ण प्रोग्राम को चलाने के बाद, आप "hello.txt" फ़ाइल में अपना कोड देख सकते हैं। अगर आप फ़ाइल को नए नाम से सेव करना चाहते हैं तो आप `File.rename` स्टेटमेंट का उपयोग कर सकते हैं।
