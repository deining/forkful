## लेकिन क्यों?

क्या आप कभी सोचा है कि दो तारीखों को तुलना करने का क्या महत्व हो सकता है? इस समस्या को हल करने के लिए, हमें दो तारीखों को तुलना करने की आवश्यकता होती है। इसलिए, इस ब्लॉग पोस्ट के माध्यम से हम तारीखों को तुलना करने के बारे में बात करेंगे और क्या करने के लिए उपलब्ध है।

## कैसे करें?

यदि हमें दो तारीखों को तुलना करना है, तो हम कई तरीकों का उपयोग कर सकते हैं। लेकिन इस पोस्ट में, हम TypeScript में "Date" ऑब्जेक्ट के साथ काम करने के लिए सबसे सरल तरीकों को देखेंगे। निम्न उदाहरण में, हम दो तारीखों को तुलना करना सीखेंगे और कैसे उसका आउटपुट प्रिंट करेंगे।

```TypeScript
let date1 = new Date('05/30/2021');
let date2 = new Date('06/15/2021');

if(date1 > date2) {
    console.log("Date 1 बड़ा है Date 2 से");
} else if (date1 < date2) {
    console.log("Date 1 छोटा है Date 2 से");
} else {
    console.log("दोनों तारीखें बराबर हैं");
}
```

आउटपुट:
"Date 1 छोटा है Date 2 से"

जैसा कि आप देख सकते हैं, हमने "if-else" शर्तों का उपयोग किया है जो हमें यह बताने में मदद करता है कि कौन सा तारीख दूसरे तारीख से बड़ा या छोटा है। आप इसे संपादित करके एक अलग आउटपुट भी प्राप्त कर सकते हैं।
