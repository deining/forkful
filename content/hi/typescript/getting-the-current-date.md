## क्यों

वर्तमान तिथि प्राप्त करने का काम करने में हमारे लिए क्या फायदे हो सकते हैं।

वर्तमान तिथि को प्राप्त करना आमतौर पर एक उपयोगी क्रिया है, जो कि अपनी वेबसाइट पर अपडेट की जानकारी को प्रस्तुत करने, या अपनी ऐप को नवीनीकृत करने के लिए उपयोग किया जाता है। यह भी एक उपयोगी टूल है जब आप किसी अन्य कार्यों को समय पर शुरू करना चाहते हैं, जैसे कि साइट पर एक विशेष ऑफर के लिए एक निश्चित दिनांक निर्धारित करना आदि।

## कैसे

TypeScript में वर्तमान तिथि प्राप्त करना बहुत आसान है। आप `new Date()` का उपयोग करके इसे प्राप्त कर सकते हैं। इसके बाद आप दिनांक के अलग-अलग अंशों जैसे दिन, महीना, वर्ष आदि को इस्तेमाल करके उन्हें प्रिंट कर सकते हैं।

```TypeScript
const currentDate = new Date(); // दिनांक को प्राप्त करें
const day = currentDate.getDate(); // दिन को प्राप्त करें
const month = currentDate.getMonth() + 1; // महीने को प्राप्त करें
const year = currentDate.getFullYear(); // वर्ष को प्राप्त करें

console.log(`आज का दिनांक ${day}/${month}/${year} है।`);
```

आप यहां देख सकते हैं कि वर्तमान तिथि को प्राप्त करना कितना आसान है और आप उसके साथ क्या कर सकते हैं।
