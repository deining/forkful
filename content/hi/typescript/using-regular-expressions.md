## क्यों

आपको पता है कि प्रोग्रामिंग में समस्याओं का सामना करते समय कई बार आपको टेक्स्ट डाटा को खोजने, विश्लेषण करने और प्रोसेस करने की जरूरत होती है। यहाँ आते हैं रेगुलर एक्सप्रेशन्स। ये एक पावरफुल टूल हैं जो आपको टेक्स्ट स्ट्रिंग्स पर काम करने में मदद करते हैं। इसकी मदद से आप आसानी से अपनी डेटा से जुड़े समस्याओं को हल कर सकते हैं और समझ सकते हैं कि कैसे आपकी कोडिंग अधिक उत्पादक हो सकती है।

## कैसे करें

टाइपस्क्रिप्ट में रेगुलर एक्सप्रेशन्स को समझना बहुत आसान है। ये एक RegExp ऑब्जेक्ट होता है जो आपको आपके द्वारा निर्दिष्ट नियमों के अनुसार टेक्स्ट स्ट्रिंग्स से मैचिंग करने और उनसे तुलना करने की सुविधा देता है। इससे आप अपने दिए गए नियमों में ठीक मिलने वाले सभी टेक्स्ट स्ट्रिंग्स को मैच कर सकते हैं और उनसे डेटा निकाल सकते हैं। यहाँ एक उदाहरण है:

```TypeScript
const pattern = /apples/g; // एक साधारण RegExp निर्माण करें
const text = "I like apples, apples are my favorite fruit."; // एक टेक्स्ट स्ट्रिंग निर्माण करें
const matches = text.match(pattern); // मैचिंग के लिए match() मेथड का उपयोग करें
console.log(matches); // ["apples", "apples"] कॉन्सोल पर आउटपुट होगा
```
