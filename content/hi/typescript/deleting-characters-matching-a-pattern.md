## क्यों
कभी-कभी हमें अपनी कोड में से एक विशेष पैटर्न के मात्राओं को हटाने की आवश्यकता होती है। उदाहरण के लिए, हमें एक स्ट्रिंग से सभी स्पेशल चरित्रों को हटाना हो सकता है ताकि हम उसे एक उचित प्रकार से प्रस्तुत कर सकें। इसलिए, इस ब्लॉग पोस्ट में हम आपको यह बताने जा रहे हैं कि कैसे आप एक पैटर्न के अनुसार अक्षरों को हटा सकते हैं ताकि आप अपने कोड में उचित विश्वसनीयता प्रदान कर सकें।

## कैसे करे
यदि आप किसी भी टाइपस्क्रिप्ट एप्लिकेशन में अक्षरों को हटाना चाहते हैं जो एक विशेष पैटर्न के मात्राओं से मिलते हैं, तो आपको सबसे पहले स्ट्रिंग को एक नया स्ट्रिंग में कॉन्वर्ट करना होगा। फिर आपको स्ट्रिंग में से विशेष पैटर्न के अनुसार अक्षरों को हटाना होगा। आप निम्नलिखित कोड ब्लॉक के माध्यम से यह कार्य कर सकते हैं:

```typescript
// एक साधारण स्ट्रिंग का उदाहरण दिया गया है
let str: string = "इस एप्लिकेशन से स्पेशल चरित्रों को हटाएं";
// स्ट्रिंग को एक नया स्ट्रिंग में कॉन्वर्ट करें
let newStr: string = "";
// स्ट्रिंग में से विशेष पैटर्न के अनुसार अक्षर हटाएं
newStr = str.replace(/[^\w\s]/gi, "");
// पाठ का आउटपुट प्रिंट करें
console.log(newStr);
```
