## क्यों

HTTP अनुरोध भेजने या प्राप्त करने के लिए आपको अपने प्रोग्रामिंग एप्लिकेशन में इस्तेमाल कर सकते हैं ताकि आप अंतर तक डेटा भेज सकें और अपने एप्लिकेशन को इंटरनेट के साथ संवारना सिख सकें। यह एक बहुत ही महत्वपूर्ण कौशल है जो हर एकीकरण और वेब डेवलपमेंट में उपयोगी है।

## कैसे करें

अगर आप TypeScript में HTTP अनुरोध कैसे भेज सकते हैं जानना चाहते हैं, तो आप सही जगह पर हैं! नीचे दिए गए कुछ कोड उदाहरण आपको सरलता से समझाएंगे कि कैसे आप अपने प्रोग्राम में HTTP अनुरोध भेज सकते हैं: 

```TypeScript
import { HttpClient } from '@angular/common/http'; 

const url = 'https://your-api.com/data'; 

HttpClient.get(url).subscribe((res) => {
    console.log(res);
}, (err) => {
    console.log(err); 
}); 
```

इस कोड से, आप स्पष्ट रूप से देख सकते हैं कि हम हमारे कोड में `HttpClient` लाइब्रेरी का इस्तेमाल कर रहे हैं जो हमें एक `get()` फंक्शन देता है जिससे हम अपने अनुरोध को भेज सकते हैं। हम एक URL प्रदान करते हैं जिससे हम डेटा प्राप्त करना चाहते हैं और फिर हम विभिन्न प्रतिक्रियाओं को हैंडल करने के लिए `subscribe()` फंक्शन को देते हैं।
