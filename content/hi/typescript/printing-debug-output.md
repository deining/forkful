# क्या और क्यों?

डिबग आउटपुट प्रिंट करना क्या है, और क्यों प्रोग्रामर्स यह करते हैं? डिबग आउटपुट प्रिंट करने से हम समझ सकते हैं कि हमारे कोड में कहां गलती हो रही है और उसे ठीक करने के लिए आवश्यक कार्रवाई ले सकते हैं। प्रोग्रामिंग में डिबग आउटपुट एक बहुत ही महत्वपूर्ण टूल है जो हमें अपने कोड को सुधारने में मदद करता है।

## कैसे करें?

टाइपस्क्रिप्ट में डिबग आउटपुट प्रिंट करने के लिए हम आम तौर पर `console.log()` फंक्शन का उपयोग करते हैं। नीचे दिए गए उदाहरण में हम एक मैसेज प्रिंट करने के लिए `console.log()` का उपयोग करते हैं:

```TypeScript
console.log("Hello World!"); 
```

इसका आउटपुट निम्नलिखित होगा:

```
Hello World!
```

मिले हुए मैसेज को प्रिंट करने के अलावा, हम `console.log()` के साथ वेरियबल्स को भी पास कर सकते हैं:

```TypeScript
let name = "John"; 
console.log("Hello " + name);
```

इसका आउटपुट निम्नलिखित होगा:

```
Hello John
```

## गहराई तक जाएं

डिबग आउटपुट की अधिक जानकारी के लिए, आप इसके इतिहास, वैकल्पिक विकल्प और इसे कैसे अमल में लाना है की जानकारी प्राप्त कर सकते हैं। प्रोग्रामिंग में डिबग आउटपुट का यह एक मुख्य उपयोग है क्योंकि यह हमें अपने कोड को सुधारने के लिए अपने लिए एक महत्वपूर्ण टूल प्रदान करता है।
