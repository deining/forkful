## क्यों

डिबग आउटपुट को प्रिंट करने में लोगों को क्यों इंगेज करना चाहिए, उसे समझने के लिए आवश्यक है कि हमारे कोड में क्या गलती हो रही है और कैसे उसे सही किया जा सकता है। यह हमें अपने कोड को बेहतर करने में मदद करता है और समस्याओं को निपटाने के लिए समय और प्रयास बचाता है।

## कैसे करें

```TypeScript
// दो संख्याओं को जोड़ने का उदाहरण
function addNumbers(num1: number, num2: number) {
    let sum = num1 + num2;

    // डिबग आउटपुट प्रिंट करें
    console.log(`दो संख्याओं का योग ${num1} + ${num2} है: ${sum}`);

    // योग को रिटर्न करें
    return sum;
}

// आउटपुट: दो संख्याओं का योग 5 + 10 है: 15
addNumbers(5, 10);
```

ऊपर दिए गए कोड ब्लॉक में आप देख सकते हैं कि हमने `console.log()` का उपयोग करके डिबग आउटपुट प्रिंट किया है। हमने अपनी फ़ंक्शन के अंदर एक मैसेज भी डाला है जो दो संख्याओं का योग क्या है वह दिखाता है। यह आपको पूरी तरह से समझने में मदद करेगा कि कैसे आप अपने कोड को डिबग कर सकते हैं।

## डीप डाइव

डिबग आउटपुट प्रिंट करना एक प्रोफेशनल कोडर के लिए एक बहुत ही महत्वपूर्ण टूल है। यह हमें अपने कोड में समस्याओं को ढूंढने में मदद करता है और उन्हें झोड़ने में भी। यह आमतौर पर विकास और उन्नति के समय बहुत उपयोगी साबित होता है और हमें सुनिश्चित करता है
