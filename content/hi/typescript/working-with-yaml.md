## क्यों

जब हम वैदिक दिग्गजों के लिए कोड करते हैं, तो वे अक्सर यामल को कॉन्फिगरेशन से अधिक समझते हैं। इसलिए, एक यामल कॉन्फिगरेशन काम को प्रभावी ढंग से स्थापित करने के लिए, यह प्राथमिकता होनी चाहिए कि हम भी यामल पर अपने कोड को लिखने की क्षमता विकसित करें।

## कैसे

```TypeScript 
import * as YAML from "yaml";

// YAML से JSON में निरंतरता बनाएं
const yamlString = `
- name: John Doe
- age: 25
  hobbies:
    - cooking
    - hiking
`;

// YAML से JSON बनाएं
const jsonData = YAML.parse(yamlString);
```

आप ऊपर दिए गए कोड ब्लॉक में देख सकते हैं कि हम इस XML पेशकश को आसानी से पारस्परिक रूप से प्रस्तुत कर सकते हैं और इसे हमेशा सामान्य डेटा की तरह इस्तेमाल कर सकते हैं। YAML से JSON में निरंतरता बनाने के लिए "yaml" पैकेज का उपयोग करें और परिणामांश आपको स्पष्ट JSON ऑब्जेक्ट देगा।

## गहराईयों में जाएं

YAML कोड को समझने से पहले आपको कुछ महत्वपूर्ण बातें जाननी होंगी। यामल एक कॉन्फिगरेशन फाइल है जो स्थानीय फाइल सिस्टम के प्राथमिक साथ उपयोग किया जाता है जो गैर संरचित डाटा को संरचित रूप से प्रदर्शित करने के लिए फर्मेट है। YAML सिंटैक्स इनडेंटेशन और नाम स्पेस द्वारा संरचित है। इसलिए, कोड लिखने के समय ध्यान देना होता है कि आप अंगुलियों का उपयोग करते हुए स्पष्ट ढंग से अपने यामल कोड को संरचित करते हैं।
