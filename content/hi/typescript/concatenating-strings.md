## क्यों

लगभग हर आधुनिक प्रोग्रामिंग भाषा में, स्ट्रिंग को अन्य स्ट्रिंग के साथ जोड़ने की क्षमता होती है। यह एक महत्वपूर्ण समस्या है क्योंकि हमें विभिन्न पाठों को संपर्क में रखने की जरूरत हो सकती है या एक दूसरे से डेटा को समन्वयित करना हो सकता है। इसलिए, स्ट्रिंग कनकटनेशन जोड़ने का एक अहम तरीका है, जो आपको अपने कोड को सुधारने में मदद कर सकता है।

## कैसे करें

तो, स्ट्रिंग को अन्य स्ट्रिंग के साथ जोड़ने के लिए हम टाइपस्क्रिप्ट में `+` ऑपरेटर का उपयोग कर सकते हैं। यह इस प्रकार काम करता है:

```TypeScript
let str1: string = "नमस्ते";
let str2: string = "दुनिया";

let result: string = str1 + " " + str2; // नमस्ते दुनिया

console.log(result); // नमस्ते दुनिया
```

हम यहां दो स्ट्रिंग परिभाषित करते हैं, उन्हें यूनिकोड मानों के साथ जोड़ते हैं और फिर इसे एक नए रिजल्ट स्ट्रिंग में स्टोर करते हैं। इसके बावजूद, अन्य तरीके भी हैं जो स्ट्रिंग्स को जोड़ सकते हैं। इनमें से कुछ कोड अधिक समझदारी तक चलते हैं।

```TypeScript
let str1: string = "नमस्ते";
let str2: string = "दुनिया";

let result: string = `${str1} ${str2}`; // नमस्ते दुनिया

console.log(result); // नमस्ते दुनिया
```

यहां हम बैकटिक्स इस्तेमाल करते हैं और दो स्ट्रिंग को उनके अपने आप जोड़ते हैं। `+` ऑपरेटर से मुकाबले, यह एक अधिक व्यावहारिक तरीका है ज
