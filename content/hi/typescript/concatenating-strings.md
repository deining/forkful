## क्यों

कोडिंग करते समय, हमें अक्सर स्ट्रिंग्स को जोड़ने की जरूरत होती है। TypeScript में स्ट्रिंग्स को जोड़ना बहुत आसान है और इससे हमारे कोड को अधिक स्पष्टता और स्वास्थ्य मिलता है।

## कैसे करें

```TypeScript
let first = "मेरा";
let last = "नाम";
console.log(first + last);

// आउटपुट: मेरा नाम
```

यहां हमने `+` ऑपरेटर का उपयोग करके दो स्ट्रिंग्स को जोड़कर एक नयी स्ट्रिंग बनाई है। आप अपनी मनचाही साइज़ और आकृति के लिए स्ट्रिंग्स को कंटीनेट भी कर सकते हैं।

```TypeScript
let sentence = "मैं";
sentence += " एक ";
sentence += "डेवलपर हूँ";
console.log(sentence);

// आउटपुट: मैं एक डेवलपर हूँ
```

यहां `+=` ऑपरेटर का उपयोग करके हमने स्ट्रिंग `sentence` को एक-दूसरे से जोड़कर नयी स्ट्रिंग बनाई है।

## डीप डाइव

स्ट्रिंग्स को जोड़ना वास्तव में पावरफुल है। एक साथ कई स्ट्रिंग्स को जोड़ने के लिए `+` ऑपरेटर के अलावा हम `concat()` फ़ंक्शन भी इस्तेमाल कर सकते हैं। इससे हमें अपनी स्ट्रिंग्स को फाइनल ऑब्जेक्ट में जोड़ने की सुविधा मिलती है।

```TypeScript
let first = "आपका नाम";
let last = "क्या है?";
let fullName = first.concat(" ", last);
console.log(fullName);

// आउटपुट: आपका नाम क्या है?
```

इसके अलावा, स्ट्रिंग इंटरपोलेशन (`${variable}`) का भी इस्तेमाल किया जा सकता है। यह कोड को और आसान बनाता है और स्ट्रिंग्स को जोड़ने में और फ्लेक्सिबिलिटी देता है।
