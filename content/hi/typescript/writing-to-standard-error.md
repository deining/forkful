## क्यों

एक प्रायोगिक दृष्टिकोण से देखें तो, टाइपस्क्रिप्ट में लिखा गया कोड आपकी गलतियों को सुधारने में मदद करता है। अगर आप कोड लिखने में नए हैं या कठिनाई से जूझ रहे हैं, तो स्टैंडर्ड एरर को समझने और उससे सीखने से आपको समस्याओं को सुलझाने में मदद मिल सकती है।

## कैसे करें

```TypeScript
// स्टैंडर्ड एरर को लिखने के लिए console.error () फ़ंक्शन का उपयोग करें
console.error("यहां एरर का संदेश होगा");

// विस्तृत स्टैंडर्ड एरर को लिखने के लिए, आप दो तत्व के साथ console.error का उपयोग कर सकते हैं - संदेश और एरर ऑब्जेक्ट
console.error("अपनी संदेश यहां लिखें", { errorCode: 404, errorType: "not found" });
```

आप अपने स्टैंडर्ड एरर के साथ कोई भी संदेश और एरर ऑब्जेक्ट दे सकते हैं। आप यहां एक सामान्य एरर संदेश और उसके साथ उदाहरण देख सकते हैं:

```
ERROR: Unable to connect to database
{
  errorCode: 500,
  errorType: "connection error"
}
```

## गहराई से समझें

स्टैंडर्ड एरर लिखना किसी भी प्रोग्रामर के लिए एक उपयोगी कौशल हो सकता है। यह आपको अपने कोड में गलतियों को सुधारने और उनसे सीखने में मदद करता है। यह आपको अपने एप्लिकेशन में निर्दिष्ट समस्याओं के साथ निपटने में भी मदद कर सकता है।
