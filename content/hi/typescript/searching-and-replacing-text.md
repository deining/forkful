## क्यों

आपको बहुत बार हो सकता है कि आपको एक प्रोग्राम में से कुछ टेक्स्ट को दूसरे टेक्स्ट से नया टेक्स्ट बदलना हो। उदाहरण के लिए, आप अपने कोड में सारे भाषाओं के शब्दकोश को हिंदी से अंग्रेजी में रूपांतरित करना चाहते हो। इस समस्या का समाधान करने के लिए, आपको विशेष प्रोग्रामिंग टूल जैसे TypeScript का उपयोग करना होगा।

## कैसे करें

टेक्स्ट को टेक्स्ट से बदलने के लिए, हम `replace()` फ़ंक्शन का उपयोग कर सकते हैं। यह फ़ंक्शन दो आर्ग्यूमेंट्स लेता है - पहला आर्ग्यूमेंट वह टेक्स्ट है जिसे हम बदलना चाहते हैं और दूसरा आर्ग्यूमेंट वह टेक्स्ट है जिसे हम इस से बदलना चाहते हैं। इसके अलावा, हम `replace()` फ़ंक्शन के दूसरे आर्ग्यूमेंट में एक विकल्प फ़्लैग भी दे सकते हैं। यह फ़्लैग हमें टेक्स्ट को बदलने के अनुसार अन्य विशेषताओं की तरह बदलने की अनुमति देता है।

चलो, एक उदाहरण कोड देखते हैं।

```TypeScript
// शुरुआती टेक्स्ट
let text = "मेरा नाम अनिता है। मैं भारत से हूँ।"

// टेक्स्ट को बदलने के लिए आर्ग्यूमेंट
let oldText = "भारत"
let newText = "India"

// टेक्स्ट को बदलने की प्रक्रिया
let newText = text.replace(oldText, newText)

// परिणाम निकालना
console.log(newText)
// Output: मेरा नाम अनिता है। मैं इंडिया से हूँ।
```
