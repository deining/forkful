## क्यों

कभी-कभी हमे अपने आभासी या सत्यापित डेटा के साथ सामान्य टेक्स्ट डेटा को बदलने की आवश्यकता होती है, जैसे जब हम कुछ त्रुटियां या अप्रचलित वर्तनी को सही करने की जगह होती है। इससे हमारे वर्कफ़्लो को बेहतर बनाने के साथ साथ समय भी बचता है।

## कैसे करें

```TypeScript
// टेक्स्ट रीप्लेस कोड जानकारी
let text = "मैं कुछ गलत लिखा हैं";
let newText = text.replace("गलत", "सही");
console.log(newText);
// आउटपुट: "मैं कुछ सही लिखा हैं"
```

ऊपर दिए गए कोड ब्लॉक में हमने `replace()` फ़ंक्शन का उपयोग करके ये काम किया है। इस फ़ंक्शन में हम पहले उस शब्द को देते हैं जो हमे बदलना है, और उसके बाद उस शब्द को जिस से बदलना है। क्या आसानी से आप इस कोड को समझ गए? अगर नहीं तो मत परेशान हो, नीचे भी हम और साधारण उदाहरण बताएंगे।

```TypeScript
// नीचे दिए गए कोड में हैंडल वर्णमाला को बदलने का कोड है
let handle = "@myhandle";
let newHandle = handle.replace("@", "#");
console.log(newHandle);
// आउटपुट: "#myhandle"
```

इस तरह आप अपने टेक्स्ट में संशोधन कर सकते हैं। अब आपके पास `replace()` फ़ंक्शन का गहराई तक जानकारी हो गयी है, चलिए आगे चलते हैं और कुछ और उदाहरण देखते हैं।
