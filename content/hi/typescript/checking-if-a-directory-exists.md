## क्यों

यदि आप अपनी TypeScript अनुप्रयोगों में स्थानीय फ़ाइल सिस्टम को पहचानना चाहते हैं, तो आपको एक डायरेक्टरी के मौजूदा होने की जाँच करनी होगी। इससे आप प्रोग्राम को दिशा-निर्देश देंगे कि कहां से फाइलें खोजी जानी हैं या कहीं से नई फ़ाइलें बनाई जानी हैं।

## कैसे करें

आपको सिस्टम को पहचानने के लिए `fs.existsSync()` फ़ंक्शन का उपयोग करना होगा। यह फ़ंक्शन एक पैरामीटर लेता है - डायरेक्टरी का पथ, और उसके आधार पर कीमत `true` या `false` वापस करता है। यदि दिए गए पथ पर कोई डायरेक्टरी मौजूद है, तो उत्पाद `true` होगा, जबकि अन्यथा उत्पाद `false` होगा। नीचे दिए गए उदाहरण में, हम `documents` डायरेक्टरी के मौजूद होने को जांच रहे हैं:

```TypeScript
if(fs.existsSync("documents")) {
    console.log("यह फाइल सिस्टम में मौजूद है!");
} else {
    console.log("यह फाइल सिस्टम में मौजूद नहीं है।");
}

// Output: यह फाइल सिस्टम में मौजूद है!
```

अगर आपको सिस्टम पर दिए गए पथ पर वास्तव में कोई फ़ाइल है, तो उसको बाहर क्रेंच देने के लिए आप `console.log()` के बदले `console.error()` का उपयोग कर सकते हैं।

## गहराई में जाइए

आप `fs.existsSync()` फ़ंक्शन को सबसे पहले अपने अनुप्रयोग में `import` करना होगा। इसके अलावा, आप अन्य फ़ंक्शनों के साथ `fs` मॉड्यूल का भी उपयोग कर सकते हैं, जैसे `fs.mkdirSync()`
