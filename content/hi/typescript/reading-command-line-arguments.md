## क्यों
क्या आप नए प्रोग्रामिंग भाषा TypeScript के बारे में जानना चाहते हैं? अगर हां, तो आप आज ही इस ब्लॉग पोस्ट को पढ़ें क्योंकि यह आपको कमांड लाइन आर्ग्यूमेंट्स को पढ़ने के बारे में बताएगा।

## कैसे करें
यदि आप अपने बनाए गए प्रोग्राम में कुछ बदलाव करना चाहते हैं और उसके लिए कमांड लाइन से आर्ग्यूमेंट्स पढ़ने की जरूरत है, तो आप बिना किसी टेंशन के बस कुछ ही आसान स्टेप्स को फॉलो करके आर्ग्यूमेंट्स को पढ़ सकते हैं। नीचे दिए गए कोड ब्लॉक में हमने कुछ कॉडिंग उदाहरण और आउटपुट दिया है, जो आपको समझ में आसानी से आर्ग्यूमेंट्स बदलने के लिए मदद करेगा।

```TypeScript
// इस उदाहरण में, हमने "name" और "age" आर्ग्यूमेंट्स को पढ़ा है और इन्हें प्रिंट किया है।
console.log("मेरा नाम है: " + process.argv[2]); // तिसरा पैरामीटर हमारा नाम है
console.log("मेरी उम्र है: " + process.argv[3] + " साल"); // चौथा पैरामीटर हमारी उम्र है
```
आउटपुट:
```
मेरा नाम है: John
मेरी उम्र है: 25 साल
```
