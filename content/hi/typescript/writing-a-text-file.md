## क्यों
टेक्स्ट फाइल लिखने की प्रक्रिया के बारे में जानने का मुख्य कारण सादगी है। कई पूर्वावलोकनों और एप्लिकेशन्स में टेक्स्ट फाइल को लिखा गया है, जिससे एक आम उपयोगकर्ता भी इसे पढ़, संपादित और साझा कर सकता है।

## कैसे करें
टाइपस्क्रिप्ट में टेक्स्ट फाइल लिखने की प्रक्रिया बहुत ही आसान है। नीचे दिए गए कोड ब्लॉक के माध्यम से आप नए टेक्स्ट फ़ाइल को बना सकते हैं, उसमें टेक्स्ट जोड़ सकते हैं और उसे सेव कर सकते हैं।

```TypeScript
// फाइल को लिखने के लिए नया स्ट्रीम तैयार करें
const fs = require('fs');
const file = fs.createWriteStream('myFile.txt'); 

// टेक्स्ट लिखें
file.write('हैलो, यह मेरा पहला टेक्स्ट फ़ाइल है!');

// फाइल बंद करें
file.end();

// टेक्स्ट फ़ाइल सेव करें
file.on('finish', () => {
 console.log('टेक्स्ट फ़ाइल सेव की गई है!');
});
```

यहां टेक्स्ट फ़ाइल को लिखने के लिए कुछ उदाहरण हैं:

```TypeScript
// 'myFile.txt' नाम की टेक्स्ट फ़ाइल बनाएं
// और उसमें "यह एक उदाहरण है" लिखें
const fs = require('fs');
const file = fs.createWriteStream('myFile.txt'); 
file.write('यह एक उदाहरण है');
file.end();
```

जब आप इस कोड को रन करेंगे, तो नई टेक्स्ट फ़ाइल 'myFile.txt' बन जाएगी और उसमें "यह एक उदाहरण है" लिखा होगा।
