## क्यों
क्या आप अपने कोड को संग्रहीत करने के लिए एक टेक्स्ट फ़ाइल लिखना चाहते हैं? एक टेक्स्ट फ़ाइल दर्शकों को आपके कोड के बारे में समझने और उसे बचाने के लिए अनुमति देता है। 

## कैसे करें
टाइपस्क्रिप्ट में एक टेक्स्ट फ़ाइल कैसे लिखें? नीचे कुछ कोड स्निपेट्स दिए गए हैं, जिनमें आपको यह दिखाया जाएगा कि कैसे आप एक नया टेक्स्ट फ़ाइल बना सकते हैं, कैसे उसमें टेक्स्ट लिख सकते हैं और उसे कैसे सेव कर सकते हैं। 

```TypeScript
// नया टेक्स्ट फ़ाइल बनाएं
let myFile = new File(["Hello, world!"], "myFile.txt");

// टेक्स्ट लिखें
myFile.text = "This is some text.";

// फ़ाइल को सेव करें
myFile.save();
```

इसके अलावा, आप अपने कोड में भी ```fs``` लाइब्रेरी का उपयोग कर सकते हैं जो आपको अधिक विस्तृत और शक्तिशाली तरीकों से टेक्स्ट फ़ाइलें लिखने और पढ़ने की अनुमति देगी। 

```TypeScript
// fs लाइब्रेरी इम्पोर्ट करें
import * as fs from "fs";

// टेक्स्ट लिखें
fs.writeFile("myFile.txt", "This is some text.", (err) => {
  if (err) {
    console.log(err);
  } else {
    console.log("File saved successfully!");
  }
});
```

## गहरी खोज
टेक्स्ट फ़ाइलें लिखने के लिए, आपको आमतौर पर एक डिफ़ॉल्ट एन्कोडिंग दी जाएगी जैसे UTF-8। लेकिन, आप भी एक अलग एन्कोडिंग का चयन कर सकते हैं या अपने फ़ाइल की भाषा को प्रभावित करने के लिए अलग एन्कोडिंग का उपयोग कर सकते हैं। एक
