## Why ("क्यों")

एक करगर डेवलपर होने के लिए, आपको अपने कोड को समझने से भी आवश्यकता होती है। तेकिंग हिसाब से, जब हम एक टेक्स्ट फ़ाइल पढ़ते हैं, हम अपने कोड को समझने से भी अधिक ज्ञानवर्धक ढंग से सीखते हैं। इसलिए, टेक्स्ट फ़ाइल पढ़ना एक महत्वपूर्ण और प्रासंगिक कौशल है जो हर कोडर को सीखना चाहिए।

## How To ("कैसे करें")

हम एक टेक्स्ट फ़ाइल सीखने को कोई कठिन काम नहीं है। हम साधारण तरीके से अपने कोड में एक टेक्स्ट फ़ाइल खोल सकते हैं और स्ट्रिंग या लाइन के रूप में आवृत्ति कर सकते हैं। निम्नलिखित कोड स्निपेट यह समझाता है कि टाइपस्क्रिप्ट में टेक्स्ट फ़ाइल कैसे पढ़ा जाए:

```TypeScript
import fs from 'fs';

fs.readFile('myfile.txt', 'utf-8', (err, data) => {
    if (err) throw err;
    console.log(data);
});
```

आप ऊपर दिए गए स्निपेट में देख सकते हैं कि हम `fs` मॉड्यूल को इम्पोर्ट करते हैं और उसे उपयोग करके फ़ाइल पढ़ते हैं। मॉड्यूल के साथ `readFile` फ़ंक्शन को उपयोग किया जाता है, जो हमें फ़ाइल के साथ आवृत्ति करने की अनुमति देता है। हमें फ़ाइल के इसी तरह से सामग्री को प्रिंट करने की अनुमति देता है।
