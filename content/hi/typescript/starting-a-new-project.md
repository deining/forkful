## क्यों

एक नया प्रोजेक्ट शुरू करने का कब और क्यों पहल करें? सवाल का सटीक जवाब हो सकते हैं कई. आप एक टेक्नोलॉजी में expertise बढ़ाना चाहते हो, आपका किसी को कुछ नया सिखाना होना हो, या हो सकता है कि आप सिर्फ एक रो़टिन के लिए बोर हो गए हैं. चाहे आपको लोगों के लिए कुछ महत्त्वपूर्ण और उपयोगी बनाना हो या आपने सिर्फ नयापन और चुनाव खोजना हो, एक नया प्रोजेक्ट शुरू करना एक महत्वपूर्ण कदम हो सकता है जो आपके लिए बहुत से फायदे लाएगा.

## कैसे करें

टाइपस्क्रिप्ट का उपयोग करके एक नया प्रोजेक्ट शुरू करने के लिए, आपको एक एडिटर या इंटरप्रेटर की जरूरत होगी. आपको सबसे पहले निम्न लाइन या एक रिपोजिटरी निर्मित करने के लिए गिटहब का उपयोग करना होगा:

```TypeScript
$ git init
```

उसके बाद, आप अपने कोड के साथ एक प्रोजेक्ट फोल्डर शुरू कर सकते हैं। आप 'index.ts' या 'main.ts' कहीं से भी नाम दे सकते हैं, लेकिन सर्वोत्तम तीसरी पार्टी की लाइब्रेरी से परिचित नामों का ही चुनाव करें। उदाहरण के लिए, आप अपने प्रोजेक्ट फोल्डर में निम्नानुसार एक 'package.json' फाइल बना सकते हैं:

```TypeScript
{
  "name": "my-project",
  "version": "1.0.0",
  "description": "My TypeScript project",
  "main": "index.js",
  "author": "",
  "license": "MIT",
  "dependencies": {
    "typescript": "^4.2.4"
  }
}
```

इसके बाद, आप अपने टाइपस्क्रिप्ट कोड फाइल में इंपोर्ट कर सकते
