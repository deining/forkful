# क्यों

आज की दुनिया में टाइपस्क्रिप्ट हमारे लिए अत्यंत उपयोगी हो गया है। इसमें संकल्पित प्रोग्रामिंग की शक्ति होती है जो धीरे-धीरे हमारे कोड को बेहतर बनाती जाती है। ये जानना भी महत्वपूर्ण है कि हमारे विकास के लिए नए प्रोजेक्ट शुरू करने का मार्ग और इससे हमें क्या फायदे हो सकते हैं।

## कैसे

टाइपस्क्रिप्ट नए प्रोजेक्ट शुरू करने के लिए आसान और सरल होता है। सबसे पहले, आपको टाइपस्क्रिप्ट कोड कॉंपाइल करने के लिए नोड इंस्टॉल करना होगा। फिर, आप अपनी पसंद के एडिटर का उपयोग करके अपना कोड लिख सकते हैं। नए प्रोजेक्ट शुरू करने के लिए, आप निम्नलिखित चरणों का पालन कर सकते हैं:

```TypeScript
npm init
npm install -g typescript
tsc --init
```

इससे आपका प्रोजेक्ट तैयार हो जाएगा और आप शुरू करके अपनी पसंद का कोड लिख सकते हैं। इसके बाद, आप अपने कोड को कॉंपाइल कर सकते हैं और अपनी राख को सूची में देख सकते हैं।
