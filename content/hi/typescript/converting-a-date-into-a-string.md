## क्यों

एक तारीख को स्ट्रिंग में बदलने के पिछले पोस्ट में, हमने TypeScript में आसानी से यूजर फ्रेंडली डेट फॉर्मेटिंग किस तरह से किया जाता है उसके बारे में बात करा था। आज हम आगे बढ़कर जानेंगे कि तारीख को स्ट्रिंग में बदलना क्यों जरूरी है और उससे हमें क्या फायदे हो सकते हैं।

## कैसे करें

हम तारीख को स्ट्रिंग में बदलने के लिए निम्नलिखित चरणों का पालन करेंगे:

```TypeScript
// तारीख को JavaScript ऑब्जेक्ट में इनिटिअलाइज़ करें
let date = new Date("2021/06/30");

// JavaScript की inbuilt फ़ंक्शन `toLocaleDateString()` से उदाहरण को स्ट्रिंग में कनवर्ट करें
let dateString = date.toLocaleDateString();

// उदाहरण की स्ट्रिंग को दिखाएं
console.log(dateString); // 06/30/2021
```

इसी तरह, हम अलग-अलग तारीख फॉर्मेट का भी उपयोग कर सकते हैं, जैसे कि `YYYY-MM-DD` या `DD/MM/YYYY`। यह सभी `toLocaleDateString()` फ़ंक्शन के माध्यम से इस सुविधा का लाभ उठाते हैं।

## गहराई में डूबना

तारीख को स्ट्रिंग में कनवर्ट करने के फायदे के अलावा, हमें इस काम को करते समय एक गहराई में भी डूबने का मौका मिलता है।

जैसा कि हम जानते हैं, JavaScript में तारीख को ऑब्जेक्ट में निर्देशित करने के लिए कई तरीके हैं। इन सभी में से, `toLocaleDateString()` में सबसे अधिक विकसित और यूजर-फ्रेंडली तरीका है।
