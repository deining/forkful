## क्यों

स्ट्रिंग की लंबाई पता करने के पीछे का कारण अधिकतर यह है कि हमें उस स्ट्रिंग की मानक लंबाई पता करनी होती है जो हमारी प्रोग्रामिंग में उपयोग की जाती है। यह जानने से हमें स्ट्रिंग को सही तरीके से प्रोसेस करने में मदद मिलती है।

## कैसे करें 

```TypeScript
// स्ट्रिंग का उपयोग करके लंबाई पता करें
const str = "यह है स्ट्रिंग";
console.log("स्ट्रिंग की लंबाई: " + str.length);

// डाइनामिक स्ट्रिंग का उपयोग करके लंबाई पता करें
const str2 = prompt("कोई स्ट्रिंग दर्ज करें:");
console.log("स्ट्रिंग की लंबाई: " + str2.length);
```
आउटपुट:
```
स्ट्रिंग की लंबाई: 14
```

## गहराई तक

स्ट्रिंग की लंबाई पता करना एक सरल कार्य हो सकता है, लेकिन असल में यह काफी गहराई का विषय है। स्ट्रिंग लंबाई को समझने के लिए, हमें सभी अक्षरों की गणना करनी पड़ती है, जिसमें हर अक्षर के लिए दीर्घताएं लिखी जाती हैं। इसमें कोई गौण समय नहीं होता है, लेकिन यह दृढ़ितता और कई तरह से उपयोगी है। इसके साथ ही, आप स्ट्रिंग का उपयोग करके अनेक दूसरे गणनाओं को भी कर सकते हैं, जैसे कि अक्षरों या शब्दों की संख्या में विशिष्ट शब्दों का पता लगाना।
