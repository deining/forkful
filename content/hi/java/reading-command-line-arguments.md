## क्यों

क्या आपने कभी सोचा है कि कमांड लाइन आर्ग्यूमेंट क्या होते हैं और आप उन्हें कैसे पढ़ सकते हैं? आज के ब्लॉग पोस्ट में हम इस विषय पर चर्चा करेंगे और देखेंगे कि कमांड लाइन आर्ग्यूमेंट को पढ़ने का महत्व क्या है। तो आइए आरम्भ करते हैं!

## कैसे

```java
public class CommandLineArguments {

    public static void main(String[] args) {

        // पहला आर्ग्यूमेंट प्रिंट करें
        System.out.println("प्रथम आर्ग्यूमेंट: " + args[0]);

        // दूसरा आर्ग्यूमेंट प्रिंट करें
        System.out.println("दूसरा आर्ग्यूमेंट: " + args[1]);
    }
}
```

जब आप ऊपर दिए गए कोड स्निपेट को कोंपाइल और रन करेंगे तो आपको दो स्ट्रिंग वैल्यू मिलेंगे, जो कि आपके द्वारा कमांड लाइन में दिए गए आर्ग्यूमेंट के बराबर होंगे। आप इस तरह से अपने कोड में आर्ग्यूमेंट्स को पढ़ सकते हैं और उनका प्रयोग कर सकते हैं।

## गहराई में जाएं

कमांड लाइन आर्ग्यूमेंट पढ़ने का महत्व क्या है? यह आपके कोड को और उपयोगी और अनुकूलित बनाता है। जब आप कोड में आर्ग्यूमेंट्स का प्रयोग करते हैं, तो आप अपने प्रोग्राम को अलग-अलग आर्ग्यूमेंट के अनुसार बदल सकते हैं। आप इस तरह से अपने प्रोग्राम को एक निश्चित सामान्य उपयोग के लिए और भी उपयोगी बना सकते हैं। इसके अलावा, कमांड लाइन आर्ग्यूमेंट उपयोग
