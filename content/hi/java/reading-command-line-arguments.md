## क्यों

कमांड लाइन आर्ग्यूमेंट्स पढ़ना एक बहुत ही उपयोगी कौशल है जो जावा प्रोग्रामिंग में बेहद उपयोगी है। यह आपको अपने प्रोग्राम को इनपुट प्रदान करने के लिए संभावित त्रुटियों से बचने में सहायता करता है।

## कैसे करें

कमांड लाइन आर्ग्यूमेंट्स पढ़ना अत्यंत सरल है। आप तुरंत ```String[] args``` को मेंटेन करके प्रोग्राम के विभिन्न हिस्सों में इसका उपयोग कर सकते हैं। नीचे एक उदाहरण है:

```Java
public static void main(String[] args) {
  System.out.println("आपने " + args.length + " कमांड लाइन आर्ग्यूमेंट्स प्रदान किए हैं!");
  
  for (int i = 0; i < args.length; i++) {
    System.out.println("आर्ग्यूमेंट " + (i+1) + ": " + args[i]);
  }
}
```
उपरोक्त कोड का आउटपुट निम्न रूप में होगा:

```
आपने 3 कमांड लाइन आर्ग्यूमेंट्स प्रदान किए हैं!
आर्ग्यूमेंट 1: a
आर्ग्यूमेंट 2: b
आर्ग्यूमेंट 3: c
```
