## क्यों

हमारे समय में, प्रोग्रामिंग एक बहुत ही महत्वपूर्ण और प्रभावशाली कौशल है। इसलिए, अधिकांश लोगों को ये जानना आवश्यक है कि हम अपने कंप्यूटर पर चल रहे प्रोग्राम्स को कैसे बनाते हैं और कैसे उन्हें बेहतर बनाते हैं। इसके लिए, हमें जानने की आवश्यकता होती है कि कैसे हम Java में कमांड लाइन आर्ग्यूमेंट्स पढ़ते हैं। इससे हम अपने प्रोग्राम को युक्तियुक्त बना सकते हैं और उसे और भी अधिक उपयोगी बना सकते हैं।

## कैसे करें

यदि आप Java में कमांड लाइन आर्ग्यूमेंट्स पढ़ना सीखना चाहते हैं, तो आपको सबसे पहले ```String[] args``` नामक मैथड का उपयोग करना होगा। इस मैथड में, हम कमांड लाइन आर्ग्यूमेंट्स को एक String आरेय में संग्रहीत कर सकते हैं। नीचे दिए गए उदाहरण में, हम एक सामान्य मैथड को बदलकर उसे कमांड लाइन आर्ग्यूमेंट्स पढ़ने के लिए बनाएँगे:

```Java
public static void main (String[] args) {
    for(int i=0;i<args.length;i++) {
        System.out.println("कॉमांड लाइन आर्ग्यूमेंट " + (i+1) + ": " + args[i]);
    }
}
```

जब आप यह कोड अपने IDE में चलाएंगे, तो आप इस तरह का आउटपुट देखेंगे:

```
कॉमांड लाइन आर्ग्यूमेंट 1: arg1
कॉमांड लाइन आर्ग्यूमेंट 2: arg2
कॉमांड लाइन आर्ग्यूमेंट 3: arg3
```
