## क्यों

क्या आप एक जावा प्रोग्रामिंग शिखार पर हैं और संख्याओं को कमांड लाइन आर्ग्यूमेंट्स के माध्यम से कैसे पढ़ा जाता है इस बारे में जानना चाहते हैं? या शायद आप एक कंप्यूटर साइंस में रुचि रखते हैं और शेल स्क्रिप्टिंग के माध्यम से कूछ नया सीखना चाहते हैं? या शायद आप निर्देशिका के साथ काम करते हैं और उसके दोनों चारों सीखना चाहते हैं? चाहे आप कोई भी कारण हो, लेकिन आप महत्त्वपूर्ण है कि आप कमांड लाइन आर्ग्यूमेंट्स को कैसे इस्तेमाल कर सकते हैं। इस लेख में, हम आपको इस प्रश्न का उत्तर देंगे और आपको बताएंगे कि यहां क्यों जावा ने बनाया गया है।

## कैसे

क्या आपको पता है कि आप किस तरह से कमांड लाइन आर्ग्यूमेंट्स को जावा में पढ़ सकते हैं? अगर नहीं तो, आपको चिंता करने की आवश्यकता नहीं है, क्योंकि हम आपको इसके लिए एक आसान से तरीका बताएंगे। आप सिर्फ कुछ छोटे से चरणों को ध्यान से पढ़ें और फिर कुछ ही मिनटों में आप कमांड लाइन आर्ग्यूमेंट्स को पढ़ सकेंगे।

```Java
public static void main(String[] args) {

//Code to read command line arguments in Java
for (int i = 0; i < args.length; i++) {
System.out.println("Argument " + i + ": " + args[i]);
}
}
```

यहां, हमने सबसे पहले ```main()``` मेथड को define किया है जो कि हमारे प्रोग्राम का आरंभिक बिंदु है। इसके बाद, हमने एक ```for``` लूप का उपयो
