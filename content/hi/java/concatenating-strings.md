## क्यों?

क्या आपने कभी सोचा है कि अपने जावा प्रोग्राम में एक स्ट्रिंग को दूसरे स्ट्रिंग से जोड़ने का समय क्यों आता है? स्ट्रिंग कॉनकैटनेशन एक बहुत ही उपयोगी कॉन्सेप्ट है जो अनेक तरह के प्रोग्रामिंग कार्यों के लिए इस्तेमाल किया जाता है। इस लेख में हम आपको स्ट्रिंग कॉनकैटनेशन के बारे में अधिक जानकारी देंगे और आपको बताएंगे कि यह कैसे काम करता है। 

## कैसे करें?

```Java
// स्ट्रिंग कंकटेनेशन का उदाहरण
String str1 = "मेरा ";
String str2 = "नाम ";
String str3 = "हिन्दी ";
String str4 = "है।";

String result = str1 + str2 + str3 + str4;
System.out.println(result);

// आउटपुट: मेरा नाम हिन्दी है।
```

ऊपर दिए गए उदाहरण में हमने चार अलग-अलग स्ट्रिंग्स को जोड़कर एक ही स्ट्रिंग बनाई है और उसे प्रिंट किया है। आप देख सकते हैं कि जोड़ने के लिए हमने `+` ऑपरेटर का इस्तेमाल किया है। ध्यान दें कि स्ट्रिंग कॉनकैटेनेशन करने के बाद भी नतीजे का डेटा टाइप `String` ही होता है।

इसके अलावा, आप `StringBuilder` का भी इस्तेमाल करके स्ट्रिंग कॉनकैटेनेशन कर सकते हैं। स्ट्रिंग बिल्डर को इस्तेमाल करने से स्ट्रिंग का प्रोसेसिंग स्पीड में भी इजाफा हो सकता है।

```Java
// स्ट्रिंग कंकटेनेशन को स्ट्रिंग बिल्डर का उदाहरण
StringBuilder sb = new StringBuilder();
sb.append("मेरा ");
sb.append("नाम ");
sb.append("हिन्दी ");
sb.append("है।");

String result = sb.toString();
System.out.println(result);

// आउटपुट: मेरा
