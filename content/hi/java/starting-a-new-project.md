## क्या है और क्यों?

नए प्रोजेक्ट शुरू करना क्या है, और प्रोग्रामर्स इसे क्यों करते हैं? एक नया प्रोजेक्ट शुरू करना एक नया कोडिंग प्रोजेक्ट होता है जो कुछ नया सीखने और जानने का एक अच्छा तरीका है। यह आपको अपने कोडिंग समृद्धी को बढ़ाने का मौका भी देता है। 

## कैसे करे:

```java 
public class Main {
  public static void main(String[] args) {
    System.out.println("नए प्रोजेक्ट शुरू करना कैसे करें?");
  }
}
```

उपरोक्त कोड में, हमने एक Main क्लास बनाई है जो एक public static void main मेथड जहा हमने एक प्रिंटल्न मैसेज प्रिंट किया है। यह नए प्रोजेक्ट शुरू करने के लिए आवश्यक जानकारी प्रदान करता है। 

## गहराई में जाएं:

प्रोजेक्ट शुरू करना एक बहुत अहम प्रक्रिया है, और यह आपके कोडिंग के लिए बहुत सारे अनुकूलनों को जानकारी प्रदान करता है। इस प्रक्रिया के हिस्से के रूप में, आपको सबसे पहले एक प्लेटफॉर्म का चयन करना होगा जो Java सपोर्ट करता है। कुछ अधिक अल्टर्नेटिव है जैसे कि Eclipse, Netbeans, या IntelliJ IDEA जो एक आसान GUI इंटरफेस के साथ आते हैं। अगर आप शुरूवाती हैं, तो सर्वश्रेष्ठ रूप से आप Notepad या Texteditor का उपयोग कर सकते हैं।
