**क्यों**: इस ब्लॉग पोस्ट में हम जानेंगे कि किस तरह से हम एक पैटर्न से मेल खाते हुए वर्णों को हटा सकते हैं।

## कैसे करें

यह कार्य करने के लिए हम एक String और एक पैटर्न दोनों के लिए एक क्षेत्रिमचिह्न स्थापित करना होगा। फिर हम स्ट्रिंग के हर एक अक्षर को चेक करेंगे और यदि वह पैटर्न के साथ मेल खाता है तो उसे हटा देंगे। नीचे दिए गए उदाहरण में हमें "Java programming is fun" स्ट्रिंग को "a" और "i" जैसे अक्षरों के साथ मेल खाते हुए हटाना है।

```Java
String str = "Java programming is fun";
String pattern = "ai";

for (int i = 0; i < str.length(); i++) {
    if (pattern.contains(String.valueOf(str.charAt(i)))) {
        str = str.replace(String.valueOf(str.charAt(i)), "");
    }
}

System.out.println(str); // Output: Jv progrmmng s fun
```

यहां हमने String के अंतर्गत ही एक दूसरी String को उपयोग करके replace() मेथड का उपयोग किया है। साथ ही आप इस काम को फाइलें, डेटाबेस या आपके आवेदनों में भी कर सकते हैं। आप इसे अपने आवेदन प्रकरणों में भी उपयोग कर सकते हैं।

## डीप डाइव
