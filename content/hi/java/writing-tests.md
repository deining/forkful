## क्यों

टेस्ट लिखने में शामिल होने के लिए व्यक्ति को बताने के लिए सिर्फ 1-2 वाक्य हों।

अगर आप अपने जावा कोड को चालू रखना चाहते हैं और सुनिश्चित करना चाहते हैं कि आपका कोड ठीक से काम कर रहा है, तो टेस्ट लिखना आवश्यक है। यह आपको बग्स को पकड़ने और अपने कोड को और सुधारने का अवसर देता है।

## कैसे करें

टेस्ट लिखने के लिए, आपको सबसे पहले अपने कोड के लिए उपयुक्त लाइब्रेरी चुननी होगी। फिर, एक नया जावा फ़ाइल बनाएं और उसे अपने फोल्डर में सेव करें। इसमें, आप अपने टेस्ट केस और कोड के साथ हार्ड कोडेड चरणों को दर्ज कर सकते हैं। आप निम्नलिखित कोड ब्लॉक का उपयोग करके अपने विशिष्ट कोड और आउटपुट को दिखा सकते हैं।

```Java
// तेज़ कोड का उदाहरण
int a = 5; // अंतरिक्ष समझे
int b = 10;
int c = a + b;
System.out.println(c); // प्रिंट: 15

// चरणों का उदाहरण
@Test
public void testSum() {
  int a = 5;
  int b = 10;
  int c = a + b;
  assertEquals(15, c);
}
```
