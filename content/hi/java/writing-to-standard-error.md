## इसका कारण

कभी-कभी हम अपने जावा प्रोग्राम में गलतियां करते हैं और हमें उन पर कई बार ध्यान नहीं होता है। इन गलतियों का पता चलाना और सुधारना आसान नहीं हो सकता है लेकिन अगर हम अपने जावा प्रोग्राम में स्टैंडर्ड त्रुटि में लिखते हैं तो हम उन गलतियों का पता लगा सकते हैं। इसलिए यह एक महत्वपूर्ण तकनीक है जो जावा प्रोग्रामर्स को अपने कोड में गलतियों का पता लगाने में मदद कर सकती है।

## इसे कैसे करें

यदि आप अपने जावा प्रोग्राम में स्टैंडर्ड त्रुटि में अनुरोध करना चाहते हैं तो आप स्टैंडर्ड त्रुटि (System.err) ऑब्जेक्ट का उपयोग कर सकते हैं। नीचे एक उदाहरण दिया गया है जिसमें आप स्टैंडर्ड त्रुटि में संदेश छाप सकते हैं:

```Java
public class StandardErrorExample {
    public static void main(String[] args) {
        System.err.println("यह स्टैंडर्ड त्रुटि में संदेश है।");
    }
}
```

जब आप इस कोड को चलाएंगे तो आपको निम्नलिखित आउटपुट मिलेगा:

```
यह स्टैंडर्ड त्रुटि में संदेश है।
```
