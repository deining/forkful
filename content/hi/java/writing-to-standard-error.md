# क्यों

स्टैंडर्ड एरर में लिखने में कारणों के बारे में कुछ दृष्टिकोण जानने के लिए, चूंकि हम प्रोग्रामिंग के दौरान उचित त्रुटियों का पता लगाना चाहते हैं। 

# कैसे करें

यदि हमारे पास एक नंबर सामग्री हो, तो हम कैसे स्टैंडर्ड एरर में उसे लिख सकते हैं? चलिए एक उदाहरण देखें:

```Java
public class WriteToError {

   public static void main(String[] args) {
      
      int number = 10;
      System.err.println("यह एक त्रुटि है: " + number);
   }
}
```

आउटपुट:

`यह एक त्रुटि है: 10`

# गहराई में जाएं

स्टैंडर्ड एरर लिखने के बारे में कुछ गहराई में जानने में आपको क्या मदद मिल सकती है? स्टैंडर्ड एरर को कैसे सेट किया जाता है और यह क्यों उपयोगी हो सकता है? हम स्टैंडर्ड एरर लिखने के फायदे और उनके उपयोग के बारे में और अधिक जानेंगे।

## सीएचएआर के साथ स्टैंडर्ड एरर

यदि हमारे पास सीएचएआर (स्टैंडर्ड एरर के साथ) हो, तो हम कैसे उसे सेट कर सकते हैं? हम इस उदाहरण के माध्यम से यह जान सकते हैं:

```Java
public class WriteToError {

   public static void main(String[] args) {
      
      int number = 20;
      System.setErr(new PrintStream("output.txt"));
      System.err.println("यह एक त्रुटि है: " + number);
   }
}
```

इस कोड के बाद, हमारे निकाल फ़ाइल "output.txt" में यह लाइन होगा:

`यह एक त्रुटि है: 20`
