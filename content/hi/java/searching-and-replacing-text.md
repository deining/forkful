## क्यों

कभी-कभी जब हम बड़े टेक्स्ट फ़ाइलों में टेक्स्ट खोजने या बदलने के साथ साथ सही शब्द या वाक्यों को ढूंढने की ज़रूरत होती है। यह काम हाथ से करने में बहुत देर से हो सकता है और अक्सर गलतियां हो सकती हैं। ऐसे में, टेक्स्ट का खोज-बदल करने के लिए प्रोग्रामिंग का उपयोग एक बेहतर विकल्प हो सकता है। यह वेबसाइट, ब्लॉग या अन्य प्रोजेक्ट में सटीकता और दक्षता को बढ़ावा देने के लिए आवश्यक होता है।

## कैसे करें

जावा में टेक्स्ट का खोज-बदल करना बहुत आसान है। निम्न उदाहरण में, हम एक वाक्य खोजेंगे और उसे दूसरे वाक्य से बदलेंगे। फ़ाइल नाम या स्ट्रिंग का नाम भी आसानी से बदला जा सकता है। दी गई जावा कोड ब्लॉक में दर्शाया गया है।

```Java
// वाक्य को खोजकर बदलने का लिए
String sentence = "मैं बहुत खुश हूँ।";
sentence = sentence.replace("बहुत खुश हूँ", "भी बहुत यादव हूँ");

// फ़ाइल को खोजकर बदलने का लिए
String fileName = "टेस्ट.जावा";
fileName = fileName.replace(".जावा", ".पावरबुटन");
```

उपरोक्त जावा कोड का उत्पाद निम्नानुसार होगा:

```java
मैं भी बहुत यादव हूँ।
टेस्ट.पावरबुटन
```
