## क्यों
कभी-कभी हमें अपनी जानकारी या कोड को संशोधित करने के दौरान एक अस्थाई फ़ाइल की आवश्यकता होती है। अस्थाई फ़ाइलें का उपयोग उन कार्यों के लिए किया जाता है जो अस्थाई होते हैं और स्थायी फ़ाइलें नहीं होती हैं। उदाहरण के लिए, कोड लिखते समय प्रोग्राम का निर्वाचन करने के लिए अस्थाई फ़ाइलों का उपयोग किया जाता है।

## कैसे करें
जावा में अस्थाई फ़ाइलें प्रबंधित करने के लिए "java.io.File" इंटरफ़ेस का उपयोग किया जाता है। नीचे दिए गए कोड ब्लॉक में एक अस्थाई फ़ाइल कैसे बनाई जाती है, उसे कैसे लिखा जाता है और उसका नाम कैसे बदला जाता है, उसका उदाहरण दिया गया है।

```Java
// अस्थाई फ़ाइल बनाएं
File tempFile = File.createTempFile("temp", ".txt");

// फ़ाइल में कुछ लिखें
FileWriter writer = new FileWriter(tempFile);
writer.write("यह अस्थाई फ़ाइल का उदाहरण है!");
writer.close();

// नाम बदलें
tempFile.renameTo(new File("my_temp_file.txt"));
```

उपरोक्त कोड ब्लॉक को चलाने के बाद, आपके कुछ इस तरह का फ़ाइल होना चाहिए:

```txt
my_temp_file.txt
```
