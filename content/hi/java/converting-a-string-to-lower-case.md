## क्यों

कभी-कभी हमारे पास किसी भी प्रोग्रामिंग समस्या का समाधान करने के लिए स्ट्रिंग को लोअर केस में बदलने की जरूरत होती है। यह बड़ी संख्या में ऑनलाइन डेटा प्रोसेसिंग और टेक्स्ट माइनिंग के साथ-साथ कम्प्यूटर सॉफ्टवेयर में भी उपयोगी होता है। 

## कैसे करें

जावा में स्ट्रिंग वस्तु को लोअर केस में बदलना बहुत सरल है। आप ```toLowerCase()``` मेथड का उपयोग करके उस रूप में कर सकते हैं। नीचे दिए गए उदाहरण में हमें स्ट्रिंग "Hello World!" को लोअर केस में बदलना है।

```Java
String str = "Hello World!";
String strLower = str.toLowerCase();
System.out.println(strLower);
```

आउटपुट:
```
hello world!
```
