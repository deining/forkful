# क्यों

जावा प्रोग्रामिंग के उदाहरण देखने के दौरान आपने शायद स्ट्रिंग की लंबाई को जानने के लिए कोड देखा होगा। लेकिन आपको ये समझने की जरूरत है कि स्ट्रिंग की लंबाई पर हमेशा काम करने के लिए क्यों होता है।

# कैसे

```Java
// एक स्ट्रिंग का उदाहरण दें
String example = "हेलो वर्ल्ड";
// कंपाइलर को स्ट्रिंग की लंबाई पता करने के लिए निर्देश दो
int length = example.length();
System.out.println("स्ट्रिंग की लंबाई है: " + length);
```
**आउटपुट: स्ट्रिंग की लंबाई है: 10**

ऊपर कोड द्वारा हमने स्ट्रिंग की लंबाई प्राप्त की है और उसे प्रिंट किया है। स्ट्रिंग की लंबाई `length()` फंक्शन कोल करके प्राप्त की जाती है।

# गहराई से जानें

स्ट्रिंग की लंबाई प्राप्त करना जोड़ के दो चरक्तरों का समान होता है। यानी हर चरक्तर को 1 कराया जाता है। हम `length()` फंक्शन कोल करके इसलिए स्ट्रिंग की लंबाई पर हमेशा काम करते हैं। दूसरे तरीके से स्ट्रिंग की लंबाई प्राप्त कर सकते हैं जैसे कि `charAt()` और `substring()` फंक्शन का इस्तेमाल करके गहराई से स्ट्रिंग की लंबाई पता कर सकते हैं।
