## क्यों

कोई भी व्यक्ति स्ट्रिंग को कैपिटलाइज करने में क्यों लग जाएगा, यह प्रश्न बहुत ही आम है। यह एक आम लोगों का सामान्य प्रोग्रामिंग कार्य है जो उनके दैनिक जीवन में बहुत से उपयोगी है। इसके अलावा, कपिटलाइजेशन स्ट्रिंग प्रोसेसिंग का एक अंग माना जाता है और विभिन्न वेब एप्लिकेशन और डेस्कटॉप उपयोगकर्ताओं के लिए आवश्यक होता है। 

## कैसे

यदि आप Java में स्ट्रिंग को कैपिटलाइज करना चाहते हैं, तो निम्नलिखित स्टेप्स का पालन करें:

```Java
String str = "hello world"; // दिए गए स्ट्रिंग को पसंद कीजिए
String capitalizedStr = str.toUpperCase(); //दोनों के बीच स्ट्रिंग तो कैपिटलाइज़ करें
System.out.println(capitalizedStr); // "HELLO WORLD" आउटपुट
```

यहां, `toUpperCase()` फ़ंक्शन दिए गए स्ट्रिंग को ऑपरेशन से पास करता है जो स्ट्रिंग को कैपिटलाइज करता है। आप भी `Character.toUpperCase()` फ़ंक्शन का उपयोग कर सकते हैं जो कि एक अलग से char या यूनिकोड स्ट्रिंग को परिवर्तित करता है।
