## क्यों

कुछ समय पहले अगर आपने Java में कोई प्रोग्रामिंग की हो तो आपने सायद `String` डेटा टाइप से जरूर काम किया होगा। इसमें हम टेक्स्ट को स्टोर कर सकते हैं और मैनिप्यूलेट कर सकते हैं। एक ऐसा मैशनिज्म है जो कई कठिनाइयों को आसान बना सकता है वो है स्ट्रिंग को कैपिटलाइज करना, यानि सभी शब्दों के पहले अक्षर को बड़ा करना। इस ब्लॉग पोस्ट के माध्यम से, हम सीखेंगे कि अपने स्ट्रिंग को कैसे कैपिटलाइज कर सकते हैं और इसका प्रयोग क्यों किया जाता है। 

## हैं कैसे 

अपनी स्ट्रिंग को कैपिटलाइज करने के लिए हम `toUpperCase()` फ़ंक्शन का प्रयोग कर सकते हैं जो हमें स्ट्रिंग को अपर केस में बदलने में मदद करता है। यदि हम न चाहें कि स्ट्रिंग के कुछ अक्षरों को अपर केस में बदला जाए तो हम `substring()` फ़ंक्शन का उपयोग कर सकते हैं। यहां हम `toUpperCase()` और `substring()` दोनों फ़ंक्शन का उदाहरण देखेंगे जिससे आपको स्पष्ट रूप से समझ आ जाएगा। 

```Java
// निम्नलिखित स्ट्रिंग बनाएं 
String str = "hello world"; 
  
// स्ट्रिंग को अपर केस में बदलने के लिए उपयोग किया जाए 
String newStr = str.toUpperCase(); 
  
// स्ट्रिंग का पहला अक्षर कैपिटलाइज करने के लिए उपयोग किया जाए 
String firstLetter = newStr.substring(0, 1) + str.substring(1); 
  
// आउटपुट को मुद्रित करें 
System.out.println(firstLetter); 
```

आउटपुट:

```
Hello world
```

ऊपर दिए गए उदाहरण में, `toUpperCase()` और `substring()` फ़ं
