# क्यों

प्रोग्रामिंग में रैंडम नंबर उत्पन्न करना क्यों महत्वपूर्ण है? यह हमारे कोड को अनुप्रयोगी और रुचिकर बनाता है। इससे हम समस्याओं का समाधान कर सकते हैं और नए और नए क्षेत्रों के साथ खेल सकते हैं।

# कैसे

यदि आप रैंडम नंबर उत्पन्न करना सीखना चाहते हैं, तो हमारे पास कुछ उदाहरण हैं जो आपको यह समझने में मदद करेंगे। नीचे दिए गए उदाहरण कोड ब्लॉक में आप रैंडम नंबर उत्पन्न करने का प्रोसेस देख सकते हैं:

```Java
// रैंडम नंबर उत्पन्न करने के लिए आपको डेमो लाइब्रेरी का इस्तेमाल करना होता है
import java.util.Random; 

public class RandomNumberGenerator {
    public static void main(String[] args) {
        // एक डेमो नंबर जेनरेट करने के लिए फंक्शन को कॉल करें
        // यहां हम 1 से 100 के बीच में रैंडम नंबर जेनरेट कर रहे हैं
        int randomNumber = getRandomNumber(1, 100);
        
        // रैंडम नंबर को प्रिंट करें
        System.out.println("रैंडम नंबर यह है: " + randomNumber);
    }

    // एक फंक्शन बनाएं जो रैंडम नंबर उत्पन्न कर सके
    public static int getRandomNumber(int min, int max) {
        // फार्मूले के अनुसार रैंडम नंबर उत्पन्न करें
        // max-min+1 के साथ सबसे मिनिमल नंबर से सबसे मैक्सिमम नंबर के बीच का नंबर उत्पन्न करने के लिए 
        int randomNumber = (int)(Math.random() * (max-min+1) + min);
        // रैंडम नंबर लौटाएं
        return randomNumber;
    } 
}
```

जब आप उपरोक्त कोड को कंपाइल और रन करेंगे, तो आपको हर बार एक नया रैं
