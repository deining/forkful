# क्यों

क्या आपने कभी सोचा है कि किसी डिरेक्टरी की उपस्थिति को जांचने का उद्देश्य क्या हो सकता है? जब हम किसी प्रोग्राम को लिखते हैं, तो हमारे पास बहुत से सिलोस हो सकते हैं और हमें सुनिश्चित करना होता है कि हमारे प्रोग्राम को सही तरीके से चलाने के लिए हमारी डिरेक्टरी उपलब्ध है। इसलिए, डायरेक्टरी की उपस्थिति की जांच बहुत महत्वपूर्ण होती है और इसलिए हमें इससे परिचित होने की आवश्यकता होती है।

# कैसे करें

डायरेक्टरी की उपस्थिति को जांचने के लिए, हम `exists()` फंक्शन का उपयोग कर सकते हैं। यह फंक्शन डायरेक्टरी की मौजूदा उपस्थिति को चेक करता है और उसकी स्थिति बूलियन में वापस देता है। नीचे एक उदाहरण है जो आपको समझने में मदद करेगा:

```Java
// एक नया फ़ाइल ऑब्जेक्ट बनाएं
File myDirectory = new File("C:\\my_directory");

// अगर डायरेक्टरी मौजूद है तो true वापस दें।
if(myDirectory.exists()){
  System.out.println("यह डायरेक्टरी मौजूद है।");
} else{
  System.out.println("यह डायरेक्टरी मौजूद नहीं है।");
}
```

यहां `C:\\my_directory` आपकी डिरेक्टरी की पथ और नाम है। आप इसे अपनी आवश्यकतानुसार संशोधित कर सकते हैं। उपरोक्त कोड आपको डायरेक्टरी की उपस्थिति की स्थिति का संदर्भ देगा और उसके अनुसार आप उचित संशोधन कर सकते हैं।
