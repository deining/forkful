## क्यों

क्या आपको कभी अपने ऑपरेटिंग सिस्टम पर किसी निर्देशित फ़ोल्डर को जांचने की आवश्यकता हुई है? या फिर आपको अपने एप्प्लिकेशन में एक निर्देशित फ़ोल्डर को चेक करने की आवश्यकता है? अगर हाँ, तो आपको इस जाँच को कंप्यूटर भाषा में "पथ निर्देशिका मौजूद है क्या" कहा जाता है। यह आपको अपने एप्लिकेशन या कोड में गलतियों को बचाने में मदद कर सकता है। 

## कैसे करें

जावा में फ़ोल्डर की मौजूदगी को जांचने के लिए, हम अपने कोड में निम्नलिखित तरीके का इस्तेमाल कर सकते हैं:

```java
import java.io.File;

public class DirectoryExistDemo {
    public static void main(String[] args) {
        String path = "C://Users//Documents//Files";
        File directory = new File(path);
        if(directory.exists()){
            System.out.println("फोल्डर मौजूद है");
        } else {
            System.out.println("फोल्डर नहीं मौजूद है");
        }
    }
}
```

आप अपने फोल्डर का पथ `String` आइटम के रूप में `File` ऑब्जेक्ट में पास कर सकते हैं। प्रदर्शित शर्त के अनुसार, हम कॉम्प्यूटर पर फोल्डर की मौजूदगी की जांच कर सकते हैं। यदि फोल्डर मौजूद है, तो हमें "फोल्डर मौजूद है" का सन्देश प्रदान किया जाएगा, वर्ना "फोल्डर नहीं मौजूद है"।
