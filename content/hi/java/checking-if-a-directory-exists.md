## क्यों

एक निर्देशिका मौजूद है या नहीं जांचने में लोगों को इंजीनियरिंग और सॉफ्टवेयर विकास के क्षेत्र में कई तरह की गतिविधियों को संभव बनाने की आवश्यकता होती है। इसलिए अपने प्रोग्राम में इसको अभिव्यक्त करना अत्यंत महत्वपूर्ण है।

## कैसे करें

```Java
import java.io.File;

public class DirectoryExistsExample {
  public static void main(String[] args) {
    // दिए गए पथ पर निर्देशिका बनाएँ
    File directory = new File("C:/Users/Username/Documents");
    // दिए गए पथ पर निर्देशिका मौजूद है या नहीं जांचें
    if(directory.exists()){
      System.out.println("निर्देशिका मौजूद है");
    } else{
      System.out.println("निर्देशिका मौजूद नहीं है");
    }
  }
}
```

प्रोग्राम की डाले गई यात्रा (output) हो सकती है:
निर्देशिका मौजूद है

## गहराई में जाएं

निर्देशिका की मौजूदगी को जांचने के लिए File वर्ग का उपयोग किया जाता है। यह तेज़ी से और कम स्टोरेज स्पेस के साथ निर्देशिका की मौजूदगी को भी जांच सकता है। जब एक निर्देशिका बनाई जाती है, तो File वर्ग के उदाहरण के साथ मौजूदा सहायक वर्गों में इसका प्रयोग किया जाता है। अतः, File वर्ग द्वारा उपलब्ध विधियों का प्रयोग करके, आप अपने प्रोग्राम में निर्देशिकाओं को जांच सकते हैं।
