## क्यों

क्या आपने कभी किसी बड़े वॉर्ड स्ट्रिंग से कुछ अलग अंश निकालने की कोशिश की है? अगर हाँ, तो आप ने सबसे सुअर्थिय तरीके में से एक को प्रयोग किया हो सकता है - उप-स्ट्रिंगों को निकालना। यह कम हथियारों में से एक है जो जावा में स्ट्रिंग मानचित्रण को सुविधाजनक बनाता है। हम इस कमांड के द्वारा कोडिंग का सार्थक उपयोग करके अपनी स्ट्रिंग में से दूसरी स्ट्रिंग के अंशों को प्राप्त कर सकते हैं, जिससे हमारे कोड को स्वस्थ और स्वचालित बनाए।

## कैसे करें

```
Java String subString(int beginIndex, int endIndex)
```

प्रारंभिक और अंतिम इंडेक्स को उपस्थित स्ट्रिंग कोड में प्रदान किए जाने पर, यह विशेष तरीके से दूसरी स्ट्रिंग के उन अंशों का निष्कर्षण करता है जो आपने प्रदान किए हुए इंडेक्स के बीच स्थित हैं। आइए इसे एक उदाहरण के साथ समझें:

```
String str = "यह एक बड़ी स्ट्रिंग है";
System.out.println(str.substring(4, 9));
```

उपरोक्त कोड का आउटपुट "बड़ी" होगा, क्योंकि हमने उप-स्ट्रिंग के निकासी की शुरुआत इंडेक्स 4 और अंत इंडेक्स 9 को प्रदान किया है, जो सुझाव है। स्पष्ट है कि यह तरीका हमारे द्वारा प्रदान किए गए अंश की सरलता और दृढ़ता सुनिश्चित करता है।
