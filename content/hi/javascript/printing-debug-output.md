# क्यों:

डिबग आउटपुट प्रिंट करने का प्रयोजन आपको अपनी कोड में सहीगलतियाँ ढूंढने और उन्हें सुधारने में मदद करता है। यह आपको अपने कोड को और सुथरा बनाने में मदद कर सकता है और आपको समय भी बचाएगा।

# कैसे करें:

```Javascript
console.log("यह एक डिबग प्रिंट है");
```

ऊपर दिए गए कोड को आपको अपने कोड में जोड़ना होगा। यह आपको अपने कोड का निर्देशांक और मानों को दिखाएगा और आपको पता चलेगा कि कौन से भीदआ आपके कोड में त्रुटि कर रही है।

# गहराई में जाएं:

डिबग आउटपुट प्रिंट करने के लिए, आप अपने कोड में `console.log()` उपयोग कर सकते हैं। आपको बताया जाएगा कि कौन से भीम हमारे कोड के कहीं गलत है, और आप अपने कोड को ठीक कर सकते हैं। आप भी अन्य डिबग उपकरण जैसे कि ब्राउनिंगर पर भी ब्राउज़र के बय्टर या अन्य डिबगर उपकरण जैसे कि एवलुटर जैसे उपकरण का उपयोग कर सकते हैं।
