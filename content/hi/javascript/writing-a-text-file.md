## इसलिए 
क्या आप पाठ फ़ाइल लिखने में शामिल होने के बारे में सोच रहे हैं? पाठ फ़ाइलों को लिखने से आप अपने कोड को सामान्य टेक्स्ट फ़ाइल में उपयोग कर सकते हैं जो आसानी से साझा किया जा सकता है और समूह काम करने को सुगम बनाता है। यह गाइड हिंदी में आपको जावास्क्रिप्ट प्रोग्रामिंग में पाठ फ़ाइल लिखने की प्रक्रिया के बारे में बताएगा।

## कैसे करें 
जावास्क्रिप्ट के सबसे सरल तरीके में से एक यह है कि हम एक नई फ़ाइल को सिर्फ़ जावास्क्रिप्ट के माध्यम से बना सकते हैं और उसे लिख सकते हैं। नीचे दिए गए कोड ब्लॉक में आप यह देख सकते हैं कि कैसे हम node.js लाइब्रेरी का उपयोग करके फ़ाइल बना सकते हैं और उसमें "नमस्ते हिंदी!" लिख सकते हैं।

```Javascript
const fs = require('fs');

fs.writeFile('hello.txt', 'नमस्ते हिंदी!', function (err) {
  if (err) throw err;
  console.log('फ़ाइल बना दी गई है!');
});
```
इसका आउटपुट होगा नई फ़ाइल "hello.txt" जिसमें "नमस्ते हिंदी!" लिखा होगा।
