## क्यों

जब हमारा कोड कुछ डाटा को सेव करने के लिए आवश्यक हो, तो हम एक पाठ फाइल में उस डाटा को लिख सकते हैं। इससे हमारे कोड को सुव्यवस्थित और उपयोगी बनाने में मदद मिलती है।

## कैसे करें

```Javascript
// मॉड्यूल importing करें
const fs = require('fs');

// डाटा को string में स्टोर करें
// ध्यान दें, stringify की जगह जो भी डाटा हमें लिखना हो, उसे इसके अंदर कीजिए
let data = JSON.stringify({
    name: "अनन्या",
    age: 26,
    profession: "वेब डेवलपर"
});

// टेक्स्ट फाइल में डाटा लिखें
fs.writeFile('विवरण.txt', data, (err) => {
    if (err) throw err;
    console.log('फाइल सफलतापूर्वक बनाई गई!');
});

```

आपको केवल एक्सपोर्ट किए गए `fs` मॉड्यूल को इम्पोर्ट करने की जरूरत होगी। अपने डेटा को string में स्टोर करें, और फिर `fs.writeFile()` फंक्शन का उपयोग करके उस string को एक नए फाइल में लिखें। हम फाइल का नाम और डाटा दोनों को क्यूँकि एक साथ इसका उपयोग कर रहे हैं, स्ट्रिंगिफाई कार्यक्रम को दोनों साथ भेजने की आवश्यकता है। अब जब आप अपने कोड को सिग्न अप करेंगे, तो आपको स्ट्रिंगिफाई कार्यक्रम को अपनी `विवरण.txt` फाइल के अंदर जाने की आवश्यकता होगी।

## गहराईगमन

जब आप अपने कोड को सिग्न अप करते हैं, तो इसमें कुछ महत्वपूर्ण चीजें हैं जिन्हें ध्यान में रखना चाहिए। पहले, आपको अपने कोड में string का उपयोग करने की आवश्यकता होगी। यद
