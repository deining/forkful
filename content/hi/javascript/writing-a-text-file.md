## क्यों

प्रोग्रामिंग की दुनिया मे आपने शायद पहले सुना होगा की टेक्स्ट फाइल क्यों लिखे जाते हैं? आज हम इस प्रश्न के उत्तर को जानेंगे।

## कैसे करें

यदि आप एक वेब डेवलपर हैं तो आपने शायद सुना होगा की टेक्स्ट फाइल या टेक्स्ट डॉक्यूमेंट लिखना एक आसान कार्य होता है। यह फाइल कोडिंग से सम्बंधित जानकारी युक्त कर सकता है जो आपको कोड को संगठित और स्पष्ट रूप से समझने में मदद कर सकता है। नीचे एक उदाहरण दिया गया है जो आपको समझ में आ सकता है की टेक्स्ट फाइल कैसे लिखी जाती है।

```Javascript
var fs = require('fs');
fs.writeFileSync('example.txt', 'यह एक टेक्स्ट फाइल हैं। यह फाइल आपके कोड के लिए उपयोगी जानकारी प्रदान करती हैं।');
console.log(fs.readFileSync('example.txt', 'utf-8'));
```

उपरोक्त कोड को रन करने पर आपको नीचे वाला आउटपुट मिलेगा:

यह एक टेक्स्ट फाइल हैं। यह फाइल आपके कोड के लिए उपयोगी जानकारी प्रदान करती हैं।
