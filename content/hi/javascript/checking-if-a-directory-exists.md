## वजह
कोई भी व्यक्ति जो जावास्क्रिप्ट प्रोग्रामिंग सीख रहा हो उसे अपने कोड में उपयोगी फ़ाइलों को प्रबंधित करने की आवश्यकता होती है, जैसे कि अपनी वेबसाइट पर फ़ोटो गैलरी या उपयोगकर्ता का अवतार आदि। इन फाइलों को सही स्थान पर स्टोर करने के लिए उन्हें एक निर्दिष्ट डायरेक्टरी में सहेजना आवश्यक होता है। इसलिए, हमें जांचना आवश्यक होता है कि क्या वह डायरेक्टरी पहले से मौजूद है या नहीं।

## कैसे करें
आप जावास्क्रिप्ट में ```fs.existsSync()``` फ़ंक्शन का उपयोग करके डायरेक्टरी की मौजूदगी जांच सकते हैं। इसका उपयोग करने के लिए, आपको आवश्यक फ़ाइल ```fs``` को import करना होगा। नीचे दिए गए कोड ब्लॉक में आप देख सकते हैं कि कैसे फ़ंक्शन का उपयोग और उसका आउटपुट होता है।

```Javascript
const fs = require('fs');

if (fs.existsSync("./gallery")) {
  console.log("गैलरी डायरेक्टरी मौजूद है।");
} else {
  console.log("गैलरी डायरेक्टरी मौजूद नहीं है।");
}
```
