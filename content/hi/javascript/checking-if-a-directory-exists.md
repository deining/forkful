## आखिर हम क्यों देखें कि क्या डिरेक्टरी मौजूद है?

डिरेक्टरी के मौजूद होने की जांच करने का उद्देश्य यह है कि हम उसे ऑपरेशन के दौरान उपयोग कर सकें, जैसे फ़ाइलें लिखें या पढ़ें।

## कैसे करें

नीचे दिए गए कोड ब्लॉक में आपको डिरेक्टरी मौजूद है या नहीं यह देखने के लिए `fs.existsSync()` फ़ंक्शन का उपयोग करना होगा। इसके बाद हम `if` के साथ संरचित ब्लॉक में मौजूद होने को चेक कर सकते हैं। इस सम्बन्ध में कुछ उदाहरण नीचे दिए गए हैं:

```Javascript
const fs = require('fs');

// यहां आपको अपने डायरेक्ट्री का पथ टाइप करना होगा
let directoryPath = '/Users/Username/Desktop/test';

// डायरेक्ट्री मौजूद होने की जांच
if (fs.existsSync(directoryPath)) {
  console.log('हाँ, यह डायरेक्ट्री मौजूद है!');
} else {
  console.log('नहीं, यह डायरेक्ट्री मौजूद नहीं है।');
}
```

आपको यहां `test` फ़ोल्डर को अपने डेस्कटॉप पर बनाना होगा जिससे आप उसका पथ सही ढंग से टाइप कर सकें। अब यदि आपके डेस्कटॉप पर `test` फ़ोल्डर है, तो आपका कंसोल पर निम्न लाइन दिखेगा:

```
हाँ, यह डायरेक्ट्री मौजूद है!
```

और अगर आपके डेस्कटॉप पर ऐसा फ़ोल्डर नहीं है, तो यह लाइन दिखेगा:

```
नहीं, यह डायरेक्ट्री मौजूद नहीं है।
```
