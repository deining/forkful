### क्यों

यदि आप एक वेब डेवलपर हैं और आपको एक निशान या फ़ोल्डर की उपस्थिति की जांच करने की आवश्यकता है, तो यह जानना काफी महत्वपूर्ण हो सकता है। इससे आप अपने स्क्रिप्ट में त्रुटियों को पकड़ सकते हैं और उन्हें सुधार सकते हैं। 

### कैसे करें

```Javascript
if(fs.existsSync('/path/to/directory')) {
    console.log('डायरेक्टरी मौजूद है।');
} else {
    console.log('डायरेक्टरी मौजूद नहीं है।');
}
```
इस कोड ब्लॉक में, हमने सिस्टम किताबख़ाने (fs) के existsSync() फ़ंक्शन का उपयोग करके डायरेक्टरी की उपस्थिति की जांच की है। यदि डायरेक्टरी मौजूद है तो "डायरेक्टरी मौजूद है।" कोंसोल में प्रिंट होगा, और यदि डायरेक्टरी मौजूद नहीं है तो "डायरेक्टरी मौजूद नहीं है।" प्रिंट होगा। 

### गहन अध्ययन

निशान या फ़ोल्डर की उपस्थिति की जांच करना थोड़ा सामान्य लग सकता है, लेकिन इसमें कुछ गहन अध्ययन भी है। अगर आप इससे नहीं बच सकते हैं तो आप त्रुटि को लेकर आत्मविश्वास खो सकते हैं। जब आप अपने कोड में इस चेक को लिखते हैं, आपको सुनिश्चित करना होगा कि आप सही पथ दे रहे हैं और आपके स्क्रिप्ट में वे स्थान या फ़ोल्डर मौजूद हैं जिन्हें आप खोज रहे हैं। यदि आपको सही पथ नहीं मिलता है तो आपको त्रुटि से सामना करना हो
