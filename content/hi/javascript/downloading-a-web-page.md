## क्यों
अगर आपने कभी सोचा है कि आप इंटरनेट पर देखने के लिए कोई भी वेब पेज डाउनलोड करते हैं तो आपने शायद सोचा हो कि यह एक चीज हो सकती है। लेकिन, जब भी हम एक वेब पेज खोलते हैं, तो हम उसे डाउनलोड करते हैं। इसलिए, हमारे समय से बचने के लिए, या यदि हमारे पास लेकिन नेटवर्क कनेक्शन है, तो हम इंटरनेट से डाउनलोड जारी रख सकते हैं।

## कैसे करें
वेब पेज डाउनलोड करने के लिए, हम सरल Javascript कोड का उपयोग कर सकते हैं। हम आमतौर पर `fetch()` फ़ंक्शन का उपयोग करते हैं, जो डाउनलोडिंग के लिए एक URL प्रदाता के साथ भेजा जाता है। `fetch()` फ़ंक्शन एक वास्तविक प्रॉमिस ऑब्जेक्ट वापस करता है, जो हमें वेब पेज का डेटा देता है।

उदाहरण:
```Javascript
fetch('https://www.example.com')
  .then(response => response.text())
  .then(data => console.log(data));
```

इसका परिणाम निम्न तरह से हो सकता है:
```
<!DOCTYPE html>
<html>
  <head>
    <title>Example Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>
```

## गहराई में जाएँ
वेब पेज डाउनलोड करने के लिए, आपको वेब पैज के स्रोत को पार्स करने की आवश्यकता होती है। प्रमुख तीसर पक्ष होंगे:
- URL निर्देशिका (एक URL प्रदाता जो वेब पेज को खोलने और उसे डाउनलोड करने के लिए उपयोग किया जाता है।)
- हेडर (वेब पेज के ऊपरी हिस्से में)
- बॉडी (वेब पेज के आंतरिक भाग में)
