## क्यों

प्रोग्रामिंग में एक अहम प्रोसेस है हस्तांतरण, जिसमें हम दिए गए स्ट्रिंग से एक छोटा भाग (सबस्ट्रिंग) निकालते हैं और उसे अन्य भागों में इस्तेमाल करते हैं। यह अधिकतर डेटा प्रोसेसिंग और वेब डेवलपमेंट में उपयोगी होता है। 

## कैसे करें

```Javascript
const str = "हमारा टेक्स्ट यहां है";
const subStr = str.substring(4); // सबस्ट्रिंग 4 वां अक्षर से शुरू होगी
console.log(subStr); // टेक्स्ट यहां है
```

सबस्ट्रिंग से डेटा निकालने के लिए, आप `substring()` फ़ंक्शन का इस्तेमाल कर सकते हैं। इसमें आप दो पैरामीटर दें सकते हैं - सबस्ट्रिंग की शुरुआत इंडेक्स और अंत इंडेक्स। अगर आप बस शुरूआत इंडेक्स देते हैं, तो वह उससे अंत तक का सबस्ट्रिंग वापस करेगा। 

```Javascript
const str = "यह टेक्स्ट हमारे पास है";
const subStr = str.substring(3, 9); // शुरूआत इंडेक्स 3 और अंत इंडेक्स 9 हैं
console.log(subStr); // टेक्स्  (चौथा अक्षर से नौवां अक्षर तक उठाएंगे)
```

इसके अलावा, `slice()` और `substr()` भी सबस्ट्रिंग निकालने के लिए इस्तेमाल किए जा सकते हैं। यहां भी पहला पैरामीटर सबस्ट्रिंग की शुरुआत इंडेक्स है, लेकिन अन्त इंडेक्स नहीं होता है बल्कि आपको कितने अक्षरों का सबस्ट्रिंग निकालना हो वह देना होता है।
