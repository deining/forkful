## क्यों

यहाँ आप जानेंगे कि कॉमांड लाइन आर्ग्यूमेंट पढ़ना क्यों जरुरी होता है और इसके क्या फ़ायदे हो सकते हैं।

## कैसे करें

क्या आपको पता है कि आप JavaScript में कॉमांड लाइन आर्ग्यूमेंट कैसे पढ़ सकते हैं? अगर नहीं तो आप इस ब्लॉग पोस्ट को पढ़ कर आसानी से सीख सकते हैं। नीचे दिए गए डेमो कोड से आप इसका प्रयोग करके समझ सकते हैं।

```JavaScript
// अपने फाइल के साथ दो संख्याओं को पास करें
node index.js 5 10

// पास की गई संख्याओं को प्रिंट करें
console.log(process.argv[2]); // अंतिम आर्ग्यूमेंट को प्रिंट करेगा
console.log(process.argv[3]); // दूसरा आर्ग्यूमेंट को प्रिंट करेगा

// संख्याओं को जोड़कर प्रिंट करें
let num1 = parseInt(process.argv[2]);
let num2 = parseInt(process.argv[3]);
console.log(`दोनों संख्याओं का योग ${num1 + num2} है।`);
```

आउटपुट:

```
5
10
दोनों संख्याओं का योग 15 है।
```

## गहराई में जाएं

क्या आप जानते हैं कि कैसे कॉमांड लाइन आर्ग्यूमेंट प्रोसेस करता है और कैसे उसे अपनी प्रोग्राम में प्रयोग कर सकते हैं? इस सेक्शन में हम इसके बारे में थोड़ी गहराई से बात करेंगे।

सबसे पहले, `process.argv` एक array है जिसमें प्रोग्राम की पथ, आर्ग्यूमेंट और अन्य फाइल से सम्बंधित जानकारी होती है। आप `process.argv` आर्ग्यूमेंट को `for` लूप के द्वारा इंडेक्स के अनुसार एक्सेस कर सकते हैं।
