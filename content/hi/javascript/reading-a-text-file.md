## क्यों
अगर आप एक वेब डेवलपर हैं तो आपने शायद टेक्स्ट फाइल की जरूरत को महसूस किया होगा। यह फाइलें आपको अपने कोड में कंफिगरेशन सेटिंग्स या अन्य स्टैटिक डेटा को संग्रहीत करने की अनुमति देती हैं। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि टेक्स्ट फाइल पढ़ने का क्या महत्व है और इसका सही तरीका क्या है।

## कैसे करें
टेक्स्ट फाइल पढ़ने के लिए, हमें जावास्क्रिप्ट फ़ंक्शन `fs.readFile()` का उपयोग करना होगा। इस फ़ंक्शन की मदद से हम टेक्स्ट फाइल में संग्रहीत डेटा को पढ़ सकते हैं। नीचे दिए गए कोड ब्लॉक में आप देख सकते हैं कि कैसे हम इस फ़ंक्शन को इस्तेमाल कर सकते हैं।

```Javascript
const fs = require('fs');

fs.readFile('textFile.txt', (err, data) => {
  if (err) throw err;
  console.log(data);
})
```

इस कोड में, हमने `fs` मॉड्यूल को इम्पोर्ट किया है और `fs.readFile()` फ़ंक्शन का एक पैरामीटर में टेक्स्ट फाइल का नाम दिया है। यदि कोई त्रुटि होती है तो हम `err` सहित परिणाम वापस करते हैं। अन्यथा, हम डेटा को कॉन्सोल पर प्रिंट करते हैं।

## गहराई में जाएं
जब हम टेक्स्ट फाइल में संग्रहीत डेटा को पढ़ते हैं, हमें `data` के साथ बस रॉ स्ट्रिंग मिलती है। अगर हम इसे डेटा तर्क काम के लिए इस्तेमाल करना चाहते हैं तो हमें उसे स्ट्रिंग या बाइनरी फाइल में एक्सपोर्ट करना होगा।
