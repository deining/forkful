## क्यों
क्या आपने कभी सोचा है कि जब आप किसी वेबसाइट पर जाते हैं, तो वहां की सारी जानकारी कैसे संग्रहित होती है? यह आपके द्वारा देखी गई तस्वीरें, वीडियो और अन्य सामग्री को कैसे प्रदर्शित करता है? सबकुछ प्रत्येक वेबसाइट से संबंधित जानकारी अलग-अलग फ़ाइलों में संग्रहित होती है। और आप कभी-कभी ऐसे तस्वीरों या नए सामग्री को वेबसाइट पर देखने के साथ-साथ, उन फ़ाइलों में ही आई जानकारी बदलती है। यह सब होने के पीछे सबसे महत्वपूर्ण चीज़ है - टेक्स्ट फ़ाइल को पढ़ना।

## कैसे करें
```Javascript
const fs = require('fs'); // फ़ाइल श्रृंखला लोड करें
fs.readFile('फ़ाइल का पथ', 'utf8', (err, data) => { // फ़ाइल को खोलें और उसमें संग्रहित डेटा को दिखाएं
  if (err) throw err; // अगर कोई त्रुटि हुई तो उसे छोड़ दें
  console.log(data); // संग्रहित डेटा को दिखाएं
});
```
जैसा कि आप देख सकते हैं, हमने `fs` मॉड्यूल को `require` करके फ़ाइल श्रृंखला को लोड किया है। यह हमें फ़ाइल पथ को प्रदान करने और उस फ़ाइल को खोलने में मदद करता है। `readFile` फ़ंक्शन को हम प्रदान किया है जो दो पैरामीटर्स - फ़ाइल का पथ और यूनिकोड फॉर्मेट में फ़ाइल की जानकारी प्रदान करता है। और अंत में एक फ़ंक्शन है जो कोड को चलाने के बाद `data` को कॉन्सोल में प्र
