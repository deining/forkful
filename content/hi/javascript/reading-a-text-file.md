## क्यों

पाठ फाइल को पढ़ना क्यों महत्वपूर्ण है? गणनाओं, प्रोग्रामिंग या अन्य सेटिंग्स को अनुकूलित करने के लिए सामान्य रूप से सुविधाजनक तरीके से डेटा को इम्पोर्ट करने के लिए फाइलें इस्तेमाल की जाती हैं।

## कैसे करें

पाठ फाइलों को जावास्क्रिप्ट में पढ़ने के लिए आप निम्नलिखित कदमों का पालन कर सकते हैं:

```
जावास्क्रिप्ट फ़ाइल को इंपोर्ट करें
const फाइल = इम्पोर्ट करने के लिए ('फाइल पथ / file.txt');
```

```
कॉनोल लॉग (फ़ाइल);
```

इस स्क्रीप्ट के माध्यम से, आप अपने फ़ाइल को लोड कर सकते हैं और इसमें रखा गया डेटा को डिस्प्ले कर सकते हैं। आप चाहे तो डेटा को प्रोसेस करने या पढ़ने के लिए अन्य जावास्क्रिप्ट कोड भी लिख सकते हैं।

## गहराई में जाएं

पाठ फाइलों को पढ़ने के लिए, जावास्क्रिप्ट में "fs" मॉड्यूल का उपयोग किया जाता है। इस मॉड्यूल के अंतर्गत आपको "fs.readFile()" फ़ंक्शन का उपयोग करना होगा। यह फ़ंक्शन फ़ाइल को असिंक्रोनस तरीके से पढ़ता है और जावास्क्रिप्ट को पढ़ने के लिए डेटा को वापस लौटाता है। आप चाहे तो अन्य जावास्क्रिप्ट मॉड्यूल्स भी इस्तेमाल कर सकते हैं, जो फ़ाइलों को पढ़ने और प्रोसेस करने के लिए और सुविधाजनक तरीके से सामग्री को व्यवस्थित करते हैं।
