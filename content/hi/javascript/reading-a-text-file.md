"## क्या और क्यों?"

एक टेक्स्ट फाइल को पढ़ना होता है क्या, और वह क्यों प्रोग्रामर्स इसको करते हैं? एक टेक्स्ट फाइल एक साधारण टेक्स्ट डेटा का संग्रह होता है जो एक कंप्यूटर प्रोग्राम के द्वारा प्रोसेस किया जा सकता है। प्रोग्रामर्स इसे अपने कोड में इस डेटा को इंपोर्ट करने या प्रोसेस करने के लिए करते हैं।

"## कैसे करें:"

```Javascript
const fs = require('fs');

// फाइल को ओपन करें
fs.readFile('myFile.txt', (err, data) => {
  if (err) throw err;
  
  // टेक्स्ट फाइल से डेटा पढ़ें
  console.log(data);
});
```

उपरोक्त कोड स्निपेट में, हमने ```fs```  मॉड्यूल को इंपोर्ट किया है और अपनी फाइल को ओपन करने के लिए ```readFile``` फंक्शन का इस्तेमाल किया है। फंक्शन को दो पैरामीटर के साथ पास किया गया है - एक फाइल का नाम और एक कॉलबैक फंक्शन। फाइल को ओपन करने के बाद, हम इसे पढ़ते हैं और कॉन्सोल में डेटा को प्रिंट करते हैं।

"## गहराई में जा यहां देखें:"

टेक्स्ट फाइलों को पढ़ने कोई नयी चीज नहीं है। पहले से ही कई विभिन्न तरीकों से इसका इस्तेमाल किया जाता है। आप इसे सूचीबद्ध स्‍थानों से पढ़ सकते हैं, जैसे कि डेटाबेस, वेब एप्लिकेशन या प्रोग्राम में डाउनलोड किए गए फाइलों को प्रोसेस करने के लिए।
