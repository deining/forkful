## क्यों

जावास्क्रिप्ट में एक तारीख को स्ट्रिंग में रूपांतरित करने का एक आसान तरीका है जो उपयोगकर्ताओं को अपनी दिनचर्या को सुव्यवस्थित और समझने में मदद करता है।

## कैसे करें

जब आप अपने वेब ब्राउज़र में एक तारीख को स्ट्रिंग में प्रदर्शित करते हैं, तो आप उसे मुख्यपृष्ठ पर अनुकूलित कर सकते हैं और स्थानीय समय के अनुसार उसे अपनी भाषा में प्रदर्शित कर सकते हैं।

```Javascript
let date = new Date(); // वर्तमान तारीख और समय को प्राप्त करें
let options = { weekday: 'long', year: 'numeric', month: 'long', day: 'numeric' }; // दिन का नाम, साल, महीना और दिन को स्पष्ट करने के लिए विकल्प निर्धारित करें
console.log(date.toLocaleDateString('hi-IN', options)); // तारीख को स्थानीयसंयुक्तअरबः भारतीय समय और भाषा में प्रदर्शित करें
```

कोड के बाएँ में स्थानीय समय, सफलता को ध्यान में रखकर अपनिर्दिष्ट करें।

```Javascript
let date = new Date('2021-12-25'); // उपयुक्त तारीख संसोधित करें
let options = { weekday: 'long', year: 'numeric', month: 'long', day: 'numeric' }; // दिन का नाम, साल, महीना और दिन को स्पष्ट करने के लिए विकल्प निर्धारित करें
console.log(date.toLocaleDateString('hi-IN', options)); // तारीख को स्थानीयअंग्रेजी समय और भाषा में प्रदर्शित करें
```

आमतौर पर, यह मूल्य स्थानीय रूप से परिभाषित होता है जो कि स्पष्ट नहीं है।
