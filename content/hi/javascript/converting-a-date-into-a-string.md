## क्यों
जावास्क्रिप्ट में तारीख को स्ट्रिंग में परिवर्तित करने का काम क्यों किया जाता है? जावास्क्रिप्ट में तारीख और समय को प्रदर्शित करने के लिए हमें स्ट्रिंग का प्रयोग करना पड़ता है। तारीख को स्ट्रिंग में परिवर्तित करके, हम स्ट्रिंग का उपयोग किसी विशेष कार्य को करने के लिए कर सकते हैं, जैसे कि डेटाबेस कम्यूनिकेशन या फॉर्मेटिंग परिवर्तन।

## कैसे करें
यहां हम जावास्क्रिप्ट में तारीख को स्ट्रिंग में परिवर्तित करने का एक आसान तरीका देखेंगे:

```Javascript
// जावास्क्रिप्ट में तारीख को स्ट्रिंग में अब कोनसा मापदंड का प्रयोग किया जाने चाहिए
const date = new Date(); // वर्तमान तारीख और समय के लिए नया ऑब्जेक्ट बनाएं
const dateString = date.toString(); // तारीख ऑब्जेक्ट को स्ट्रिंग में परिवर्तित करें
console.log(dateString); // उदाहरण तारीख स्ट्रिंग में प्रिंट करें: "Tue Sep 28 2021 14:33:07 GMT+0530 (India Standard Time)"
```
