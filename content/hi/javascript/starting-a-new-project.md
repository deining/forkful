## क्यों?

क्या आपको कभी एक नया प्रोजेक्ट शुरू करने के बारे में सोचने का मौका मिला है? यदि हां, तो आप जानते होंगे कि यह कठिन हो सकता है और ज्यादातर लोग इससे डरते हैं। लेकिन एक प्रोजेक्ट शुरू करने का काम बहुत मजेदार और रोमांचक हो सकता है और यह आपको नए सिक्के की तरह काम करने का मौका देता है। इसीलिए, आप नए प्रोजेक्ट को शुरू करने के बारे में सोच सकते हैं।

## कैसे करें?

एक नया प्रोजेक्ट शुरू करना आसान हो सकता है अगर आप सही तरीके से करें। इस लेख में, हम आपको बताएँगे कि आप अपना पहला प्रोजेक्ट कैसे शुरू कर सकते हैं। हम जावास्क्रिप्ट में कुछ कोड उदाहरण देंगे और साथ ही उनके आउटपुट भी दिखाएँगे। तो चलिए शुरू करते हैं!

```Javascript
// एक नया वैरिएबल डेक्लेयर करें
var myName = "नीला";

// एक नया वैरिएबल को परिवर्तित करें
myName = "लाल";

// कंसोल में इसका आउटपुट प्रिंट करें
console.log(myName); // आपको लाल मिलेगा
```

इस उदाहरण में, हमने एक वैरिएबल बनाया और उसे बाद में परिवर्तित कर दिया। फिर हमने उसका आउटपुट कंसोल में प्रिंट करवाया। आप अपने प्रोजेक्ट में भी ऐसे ही कोड का उपयोग कर सकते हैं।
