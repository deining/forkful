## क्यों

आज हम बात करेंगे जावास्क्रिप्ट प्रोग्रामिंग के बारे में। अगर आप नया प्रोजेक्ट शुरू करने का विचार बना रहे हैं तो आप बिल्कुल सही स्थान पर हैं। जावास्क्रिप्ट एक बहुत ही लोकप्रिय प्रोग्रामिंग भाषा है जो आपको अपने कैरियर में आगे बढ़ने में मदद कर सकती है।

## कैसे करें

जावास्क्रिप्ट शुरू करने के लिए, हमें सबसे पहले एक HTML डॉक्यूमेंट बनाना होगा जिसमें हम अपने कोड को लिखेंगे। आप नीचे दिए गए कोड ब्लॉक का उपयोग करके उदाहरण देख सकते हैं:

```Javascript
<html>
  <head>
    <title>मेरा प्रोग्राम</title>
  </head>
  <body>
    <script>
      // यहां हम अपना जावास्क्रिप्ट कोड लिखेंगे
      console.log("नमस्ते दुनिया!");
    </script>
  </body>
</html>
```

जैसे ही आप इस कोड को चलाएंगे, आपको कंसोल में "नमस्ते दुनिया!" दिखाई देगा। यह आपके लिए एक सरल उदाहरण है जो आपको शुरुआत करने में मदद कर सकता है। इसके अलावा, आप अपने ब्राउज़र के डेवलपमेंट टूल का भी प्रयोग कर सकते हैं जिससे आप अपने कोड को और भी बेहतर तरीके से समझ सकते हैं।
