## क्यों?

अगर आप जावास्क्रिप्ट प्रोग्रामिंग में किसी स्ट्रिंग को लोअर केस में बदलने से अधिक बेहतर परिणाम पाना चाहते हैं, तो इस ब्लॉग पोस्ट को लिखा गया है।

## कैसे करें?

```
उदाहरण १:
var string = "हेलो वर्ल्ड";
console.log(string.toLowerCase());
// आउटपुट: हेलो वर्ल्ड

उदाहरण २:
var string = "Hello World";
console.log(string.toLowerCase());
// आउटपुट: hello world
```

## गहराई में भविष्य्क

स्ट्रिंग को लोअर केस में कनवर्ट करने के लिए आप विभिन्न तरीकों का उपयोग कर सकते हैं, जैसे कि `toLowerCase()` फ़ंक्शन या अलग-अलग केस मापने वाले लिब्रेरी। इसके अलावा, आप इस टॉपिक को और अधिक गहराई में जान सकते हैं जैसे कि कैसे स्ट्रिंग्स के लिए इंटरनेशनलाइज़ेशन के स्टैंडर्ड लोअर केस को उपयोग किया जा सकता है।

आप किसी डेटा को सेंसिटिवाइज़ कर रहे हैं या नहीं, इसका पता लगाने के लिए आपको स्ट्रिंग का केस फ़ंक्शन का उपयोग करने की आवश्यकता होगी। यह आपको अधिक उदारीकृत लिखने में मदद करे गा और आपके कोड को बार-बार बदलने की आवश्यकता नहीं होगी।
