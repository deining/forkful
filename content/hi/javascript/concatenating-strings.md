## क्यों? 

जैसा कि हम सभी जानते हैं, जावास्क्रिप्ट को वेब डेवलपमेंट में एक महत्वपूर्ण भूमिका मिली है। स्ट्रिंग्स को क्रमबद्ध रूप से जोड़ना (Concatenation) जावास्क्रिप्ट में एक उपयोगी कौशल है जो आपको अपने वेबसाइट या एप्लिकेशन में डाटा को प्रबंधित करने में मदद कर सकता है। इसके अलावा, स्ट्रिंग्स को संभालना (Manipulation) भी आसान हो जाता है। 

## कैसे करें? 

जावास्क्रिप्ट में स्ट्रिंग्स को आसानी से जोड़ा जा सकता है। आइए एक उदाहरण देखते हैं: 

 ```Javascript
 let firstName = "अमित";
 let lastName = "शर्मा";
 let fullName = firstName + lastName;
 console.log(fullName); //अमितशर्मा 
 ``` 

यहां, हमने `+` ऑपरेटर का उपयोग करके दो स्ट्रिंग्स को जोड़ दिया है। आप अपनी आवश्यकता के अनुसार इसे आगे बढ़ा सकते हैं, जैसे कि: 

 ```Javascript
 let firstName = "अमित";
 let lastName = "शर्मा";
 let fullName = firstName + " " + lastName;  //अमित शर्मा
 fullName += " सर ";
 console.log(fullName); //अमित शर्मा सर 
 ``` 

यहां हमने स्ट्रिंग्स में एक स्पेस भी जोड़ दी है और आगे और भी स्ट्रिंग्स को जोड़ने के लिए `+=` ऑपरेटर का  उपयोग किया है।
