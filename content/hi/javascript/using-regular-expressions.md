## क्यों

क्या आपने कभी सोचा है कि अपने जावास्क्रिप्ट कोड को आसानी से संशोधित करने का एक तरीका हो सकता है? रेगुलर एक्सप्रेशन्स (regular expressions) को सीखने से आप अपने कोड को आसानी से पाठनीय और संशोधनीय बना सकते हैं। साथ ही यह आपको प्रोग्रामिंग की दुनिया में और भी आगे बढ़ने में मदद करेगा।

## कैसे

```Javascript
// साधारण एक्सप्रेशन के साथ मैचिंग का उदाहरण
const str = "मेरा नाम किम है";
const pattern = /मेरा नाम (.*) है/;

// मैचिंग फंक्शन का इस्तेमाल
const match = str.match(pattern);

// परिणाम की डिस्प्ले करें
console.log(match[1]); // किम
```

रेगुलर एक्सप्रेशन्स का सबसे महत्वपूर्ण उपयोग मैचिंग (matching) है। आप इसका उपयोग अधिकांश स्थितियों में स्ट्रिंग्स के साथ कर सकते हैं। यह स्ट्रिंगों का दौरा करता है और उनमें मौजूद स्पष्ट पैटर्न को खोजता है। आप मैचिंग के साथ स्‍ट्रिंगों को संशोधित भी कर सकते हैं और उनमें गलतियों को धुंध सकते हैं। आप विभिन्न मेथड्स का उपयोग करके रेगुलर एक्सप्रेशन्स का हिस्सा भी बना सकते हैं।

## डीप डाइव

रेगुलर एक्सप्रेशन्स छोटे से छोटे और बड़े स्केल परिदृश्यों को समझने में मदद करते हैं। यह प्रोग्रामिंग की दुनिया में बहुत उपयोगी है और आपको अपने कोड को अधिक संशोधनीय बनाने में सहायता प्रदान करता है। र
