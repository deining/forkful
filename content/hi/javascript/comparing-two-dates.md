## क्यों?

दो तारीखों को तुलना करना एक आम कार्य है जो समय की गणना और तारीखों की प्रबंधन के लिए उपयोगी है। उदाहरण के लिए, आप एक ई-कॉमर्स वेबसाइट बना रहे हैं और उपभोक्ता को ऑर्डर मिलने तक कितने दिन हुए हैं यह जानना चाहते हैं, तो आपको दो तारीखों को तुलना करने की आवश्यकता होगी।

## कैसे करें?

```Javascript
let date1 = new Date("2021-01-10");
let date2 = new Date("2021-01-25");

// दो तारीखों की तुलना करें
if (date1 < date2) {
  console.log("पहली तारीख दूसरी तारीख से पहले आती है।")
} else if (date1 > date2) {
  console.log("पहली तारीख दूसरी तारीख से बाद में आती है।")
} else {
  console.log("दोनों तारीखों में कोई अंतर नहीं है।")
}
```

आप दो तारीखों को `new Date()` का उपयोग करके बना सकते हैं और उन्हें `if` या `else if` की मदद से तुलना कर सकते हैं। आप `>` या `<` और `==` ओपरेटर का भी उपयोग कर सकते हैं।

## गहराई में जाएं

दो तारीखों की तुलना करने के लिए, आपको दो `Date` ऑब्जेक्ट में अंतराल को नंबर के रूप में प्राप्त करना होगा। आप इसके लिए `getTime()` मैथड का उपयोग कर सकते हैं। इससे आप नंबर को मिलीसेकेंड में प्राप्त कर सकते हैं जो आप फिर सूखने के लिए प्रयोग कर सकते हैं। आप इस अंतराल को दूसरी दिनांक से घटा कर सकते हैं तो आपको दो तारीखों के बीच का अंतर नंबर के रूप में प्राप्त होगा
