## क्यों

समय के साथ हमारे पास आधुनिक तकनीक से भरी हुई दुनिया में कम्प्यूटर और इलेक्ट्रॉनिक उपकरणों का उपयोग बढ़ता जा रहा है। जैसे-जैसे हमारी तकनीक आगे बढ़ती है, हमें लिखित कार्यों को प्रक्रियात्मक रूप से ठीक से सम्पादित और स्टोर करने की आवश्यकता होती है। इसके लिए अक्सर हम अस्थायी फाइलें बनाते हैं जो पास कारणों से क्योंकि हमें अस्थायी उपयोग के लिए फाइलों की आवश्यकता होती है।

## कैसे करें

विभिन्न प्रोग्रामिंग भाषाओं में अस्थायी फाइल क्रिएट करने का तरीका भिन्न होता है। आइए बात करते हैं जैसे कि जावास्क्रिप्ट में अस्थायी फाइल कैसे बनाया जाता है। सबसे पहले, हम ग्लोबल ```fs``` ऑब्जेक्ट को इम्पोर्ट करते हैं। फिर, हम ```fs. writeFileSync()``` फ़ंक्शन का उपयोग करके अस्थायी फाइल बनाते हैं।

```Javascript
const fs = सहित से योजना("fs");

fs.writeFile('temp.txt', 'यह एक अस्थायी फाइल है।', त्रुटि => {
    if (त्रुटि) {
        त्रुटि.छापो('अस्थायी फाइल बनाने में त्रुटि हुई। ');
    }
    अन्यथा {
        खातीर छापो('अस्थायी फाइल बन गई है। ');
    }
});
```

यहां, ```writeFileSync()``` कार्य सिम्पल है। हम पहले स्थानीय फ़ाइल नाम ```temp.txt``` को देते हैं, फिर स्ट्रिंग ```'यह एक अस्थायी फाइल है।'``` को लिखते हैं, और अंत में एक त्रुटि हैंडलर देते हैं।
