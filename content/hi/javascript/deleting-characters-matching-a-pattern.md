# क्यों

अगर आप डेटा स्ट्रिंग में से एक निशान निकालने की जरूरत हो तो आप जावास्क्रिप्ट का उपयोग करके बहुत सरल तरीके से एक पैटर्न के साथ मेल खाती वर्ण निकाल सकते हैं।

## कैसे करें

```javascript
const str = "Hello123";
const pattern = /\d/g;
const newStr = str.replace(pattern, "");
console.log(newStr); // Output: Hello
```

यहां हमने स्ट्रिंग `Hello123` से नंबर को हटाने के लिए पैटर्न `/\d/g` का उपयोग किया है। `replace()` फंक्शन स्ट्रिंग में से पैटर्न के साथ मिलते हुए वर्णों को खाली स्ट्रिंग से बदल देता है। इस तरह से हम आसानी से स्ट्रिंग से निशान निकाल सकते हैं।

## गहराई में जाएं

अब हम इस दिशा में कुछ और विस्तृत जानकारी प्राप्त करेंगे। एक पैटर्न क्या होता है और कैसे एक स्ट्रिंग से उसका उपयोग किया जाता है, इसके साथ हम डेटा स्ट्रिंग में से विशिष्ट पैटर्न के साथ मेल खाते वर्ण निकालने की तकनीक समझेंगे।

एक पैटर्न, एक निशानाधिकारी होता है जो एक या एक से अधिक वर्णों को संदर्भित करता है। पैटर्न्स को दो हिस्सों में जोड़ा जाता है। पहला हिस्सा, जिसे "बूलियन" हिस्सा कहा जाता है, स्ट्रिंग में अनुकूलन को जांचने के लिए पैटर्न की शर्तों को सभी मिलते वर्णों को निर्धारित करता है। दूसरा हिस्सा, जो "ग्लोबल" हिस्सा कहलाता है, स्ट्रिंग में सभी मिलते वर्णों को खाली स्ट्रिंग से बद
