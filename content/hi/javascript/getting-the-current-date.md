## क्यों
क्या आपने कभी सोचा है कि क्यों हमें कर्रेंट डेट प्राप्त करने की ज़रूरत होती है? आज जब हम डिजिटल दुनिया में जी रहे हैं, तो हमारे लिए जाना कि तारीख क्या है एक बहुत ही महत्वपूर्ण कौशल बन गया है। तारीख का उपयोग भी कई तरह के कामों में किया जाता है, जैसे कि खुद को अपडेट करना, फाइलों को नामित करना या डेटा को सॉर्ट करना।

## कैसे करें
अब जब हम जानते हैं कि तारीख क्यों महत्वपूर्ण है, तो काम का सबब हमें कैसे कर्रेंट डेट प्राप्त करना है। आपको चिंता करने की जरूरत नहीं क्योंकि जावास्क्रिप्ट में हमें तारीख प्राप्त करने के लिए एक बहुत ही सरल मेथड होता है। तो चलिए, एक साम्पल कोड के माध्यम से यह देखते हैं:

```Javascript
// वर्तमान तारीख का आधार बनाओ
let today = new Date();

// आधार पर तारीख को प्राप्त करें
let date = today.getDate();
let month = today.getMonth() + 1; // महीनों को 0 से शुरू किया जाता है, इसलिए हमें 1 जोड़ना होगा
let year = today.getFullYear();

// प्राप्त किया गया तारीख को फार्मेट करने के लिए
let formattedDate = `${date}/${month}/${year}`;

console.log(formattedDate); // output: 20/5/2021
```

जैसा कि आप देख सकते हैं, हमने सबसे पहले `new Date()` का उपयोग करके एक विशेष तारीख के आधार पर तारीख को प्राप्त किया और फिर इसे अलग-अलग भागों में विभांतरित करने के लिए उपयोग किया। आप अपनी आवश्यकतानुसार तारीख को फॉर्म
