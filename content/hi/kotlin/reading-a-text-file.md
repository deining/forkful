Kotlin में एक गहरी डाइव करो - टेक्स्ट फाइल पढ़ना

## क्यों
क्या आपको कभी अपने प्रोग्राम में सेमिकोलॉन या अन्य विशेष वाक्यांशों को बदलने की जरूरत पड़ी है? या क्या आपको एक बड़ी फ़ाइल में आवेदन संसाधित करने की आवश्यकता होती है? तो Kotlin में एक टेक्स्ट फाइल से डेटा पढ़ने की जरूरत पड़ सकती है। यह आसान हो सकता है और यह आपके प्रोग्राम में बहुत सारे समय बचा सकता है।

## कैसे करे
क्या आपको परेशानी हो रही है कि अपने प्रोग्राम में एक टेक्स्ट फाइल से डेटा कैसे पढ़े? यहां हम आपको एक उदाहरण देंगे जो ```readText``` फ़ंक्शन के माध्यम से कैसे काम करता है।

```
val fileName = "example.txt"
val fileText = File(fileName).readText()
println(fileText)
```
यहां, हम ```example.txt``` नाम के फ़ाइल से डेटा पढ़ रहे हैं और उसे ```fileText``` वेरिएबल में स्टोर कर रहे हैं। उसके बाद हम डेटा को प्रिंट कर रहे हैं। आप अपनी फ़ाइल का नाम अपने नीचे के द्वारा बदल सकते हैं।

```
val fileText = File("example.txt").readText()
```
