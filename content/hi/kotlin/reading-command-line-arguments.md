## क्यों

कमांड लाइन एर्ग्यूमेंट को पढ़ने में रुचि रखने के लिए क्यों हो सकता है, इसके कुछ कारण हो सकते हैं। यह आपको अपनी कोड को और उपयोगी बनाने में मदद कर सकता है, जैसे कार्यकारी पैरामीटर दर्शाना या अनुप्रयोग को कार्यकारी लाइन से नियंत्रित करना। 

## कैसे करें

आइसोलेबल कोरोटिनी कोड ब्लॉक के साथ "```Kotlin ... ```" के भीतर नमूना भाग कॉड और सैंपल आउटपुट संकलित करें। इसमें में `args` नाम का एक स्ट्रिंग एरे शामिल हो सकता है जो आपको कमाण्ड लाइन एर्ग्यूमेंट पढ़ने की अनुमति देंगे। 

```Kotlin
fun main(args: Array<String>) {
    for (arg in args) {
        println(arg)
    }
}
```

यदि हम इस कोड को `hello world` चलाएं तो निम्नलिखित आउटपुट मिलेगा: 

```bash
$ kotlin Example.kt hello world
hello
world
```

## गहराई से जानें

कमांड लाइन एर्ग्यूमेंट को पढ़ने से सम्बंधित अधिक जानकारी के लिए, आप `args` वेरियबल के अलावा `main()` फंक्शन को भी पढ़ सकते हैं। यह आपको कमांड लाइन पैरामीटर के बारे में अधिक जानकारी और उनके साथ किस तरह से आपको काम करना होगा उसे लंबा करेगा।
