## क्यों

किसी भी प्रोग्रामर के लिए उंगलियां उसके असली ताकत होती है। वे हमेशा अलग-अलग तरीकों से एक बड़े टेक्स्ट स्ट्रिंग से भिन्न-भिन्न सबस्ट्रिंग्स निकालने के लिए प्रयास करते हैं। यह उन्हें उस स्ट्रिंग के साथ काम करने में आसानी प्रदान करता है और कोड को अधिक स्पष्ट और धातुहित बनाता है। अगर आप एक नए कोडर हैं या कोटलिन में नए हैं, तो सबस्ट्रिंग का एकमात्र समाधान है जो आपको कोडिंग करने में जरूरत होगी।

## कैसे

यहां हम कोटलिन में स्ट्रिंग से सबस्ट्रिंग निकालने के कुछ प्रमुख तरीके देखेंगे।

```kotlin
// स्ट्रिंग से सबस्ट्रिंग निकालने का प्रथम तरीका
val str = "यह एक सादे से स्ट्रिंग है।"
val subStr = str.substring(3..5) // एक

// स्ट्रिंग के आधार पर सबस्ट्रिंग निकालने का द्वितीय तरीका
val name = "संजय कुमार"
val lastName = name.substringAfter(" ") // कुमार

// स्ट्रिंग में से अंतिम एक्स करें
val str = "कोटलिन"
val lastChar = str.last() // न

// स्ट्रिंग से पूर्व और उसके बाद से सबस्ट्रिंग निकालें
val str = "हिंदुस्तानी भारत"
val preSubStr = str.substringBefore(" ") // हिंदुस्तानी
val postSubStr = str.substringAfter(" ") // भारत
```

उपरोक्त उदाहरण आपको स्ट्रिंग से सबस्ट्रिंग निकालने के कई विभिन्न मेथड दिखाते हैं। आप भावी में अपने प्रोग्रामिंग प्रोजेक्ट में उन्हें उपयोग कर सकते हैं।
