कोटलिन (वर्तमान संस्करण) काला इस्तेमाल करके हिंदी पाठकों के लिए एक बातचीत लिखें। एक अनैतिक टोन और एक न साफ-सुथरा शैली में लिखें। अनावश्यक शब्दों और वाक्यों से बचें।
"## क्या और क्यों?" : दो से तीन वाक्यों में समझाएं (1) वर्तमान तिथि हासिल करना क्या है और (2) क्यों कार्यकारी इसे करते हैं।
"## कैसे करें :" कोडिंग उदाहरण और ```कोटलिन ...``` कोड ब्लॉक के भीतर नमूना आउटपुट।
"## डीप डाइव": उत्तराधिकार हिंस्तावलय, वैकल्पिक, और (3) वर्तमान तिथि पाने के इम्प्लीमेंटेशन की विस्तार से जानकारी है।
"## भी देखें": संबंधित स्रोतों के लिंक।
"कोटलिन में वर्तमान तिथि हासिल करना"
"## क्या और क्यों?"
किसी भी प्रोग्रामिंग भाषा में, वर्तमान तिथि हासिल करना बहुत महत्वपूर्ण है। इससे हम अपने प्रोग्राम में अवश्यक तिथि को परख सकते हैं और उसके आधार पर आवश्यक गति से काम कर सकते हैं। वर्तमान तिथि हासिल करने के लिए टेक्निकल कूद से प्रोग्रामर को हमेशा बेहतर छोटे से छोटा भी कर सकते हैं।

"## कैसे करें:"
कोटलिन में, हम वर्तमान तिथि लेने के लिए ```LocalDate.now()``` का उपयोग कर सकते हैं। यह मेथड अपने आप में ही वर्तमान तिथि को लेकर आता है। इसके अलावा, हम आपको एक Sample कोड डाल रहे हैं जिससे आप अपनी स्क्रीन पर वर्तमान तिथि को देख सकते हैं। 

```kotlin
// Sample code to get current date
fun main() {
    val currentDate = LocalDate.now()
    println("Current Date : $currentDate")
}
```

आउटपुट:
Current Date: 2021-07-03

"## डीप डाइव:"
कोटलिन एक modern और शक्तिशाली भाषा है जो JVM पर चलती है। इसकी शक्ति और लोकप्रियता को लेकर, कोटलिन काफी विकासशील और बढ़ती हुई भाषा है। वर्तमान तिथि को पाने के लिए, शुरुआत में यह एक कठिन काम था, लेकिन कोटलिन प्रोग्रामर्स को इसे आसान बनाने के लिए बिल्ट-इन वर्तमान तिथि को पाने के लिए विशेष मेथड प्रदान करता है। इसके अलावा, आप Java के ```currentTimeMillis()``` या ```LocalDateTime.now()``` जैसे अन्य विकल्पों का भी उपयोग कर सकते हैं।
