## क्यों

यदि आप हिंदी भाषा के पाठक हैं और कोटलिन प्रोग्रामिंग जगत में नए हों, तो आपने शायद सुना होगा कि टेस्ट कोड का लिखना एक अच्छा अभ्यास है। लेकिन क्या आपको पता है कि आप इससे क्यों लाभान्वित हो सकते हैं? इस आर्टिकल में हम आपको बताएंगे कि क्यों आपको ब्लॉगिंग के माध्यम से कोटलिन में टेस्ट कोड क्यों लिखना चाहिए।

## कैसे करें

टेस्ट कोड को लिखना शुरू करने से पहले, हमें इसकी आवश्यकता पर ध्यान देना होगा। टेस्ट कोड क्या होता है? यह क्यों आवश्यक है? इसके बाद हम आपको कोटलिन में इसे लिखने का तरीका सिखाएंगे। आप निम्न उदाहरण कोड ब्लॉक में देख सकते हैं और साथ ही साथ उनका आउटपुट भी देख सकते हैं।

```Kotlin
// क्लास बनाना
class Person(val name: String, var age: Int, var occupation: String)

// टेस्ट के लिए फंक्शन बनाना
fun testPersonInfo() {
    // टेस्ट करें कि क्या नाम सही है
    val person = Person("जॉन डो", 25, "डेवलपर")
    if (person.name == "जॉन डो") {
        println("नाम सही है")
    } else {
        println("नाम गलत है")
    }
    
    // टेस्ट करें कि क्या उम्र सही है
    if (person.age == 25) {
        println("उम्र सही है")
    } else {
        println("उम्र गलत है")
    }
    
    // टेस्ट करें कि क्या पेशे सही है
    if (person.occupation == "डेवलपर") {
        println("पेशे सही है")
    } else {
        println("पेशे गलत है")
    }
}

// फंक्शन को कॉल करें
testPersonInfo()
```

आउटपुट:

```
नाम सही है
उम्र सही है
पेशे सही है
```
