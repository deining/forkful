## क्यों
टेस्ट लिखने की क्या जरूरत है? यह एक महत्वपूर्ण सवाल है और हमारे पास एक उत्तर है - क्योंकि टेस्टिंग आपकी प्रोग्रामिंग कौशलता को सुनिश्चित करता है और आपको सुनिश्चित करता है कि आपने दिए गए प्रश्नों का सही उत्तर दिया है।

## कैसे करें
आईये अब हम देखें कि टेस्ट कैसे लिखे जाते हैं। आपको पहले से ही कोटलिन का ज्ञान होना चाहिए। यदि आपको अभी तक कोटलिन नहीं आता है, तो आप रेड हैट पढ़ सकते हैं। इससे आपको कोटलिन की बुनियादी जानकारी मिल जाएगी।

**उदाहरण:**

```Kotlin
fun addNumbers(x: Int, y: Int): Int {
    return x + y
}

fun main() {
    val result = addNumbers(5, 3)
    println(result)
}
```

**आउटपुट:**

8

इस उदाहरण में, हमने दो संख्याओं को जोड़कर उत्तर प्राप्त किया है। यदि हम प्राथमिकताओं को ठीक से पहचानते हैं और सही तरीके से लिंक करते हैं, तो हम टेस्ट कोड को भी लिख सकते हैं।

## गहराई में
टेस्ट कैसे लिखा जाए, इसके बारे में गहराई से जानने के लिए, हमें कुछ अतिरिक्त जानकारी की आवश्यकता होती है। इससे हमें उस समस्या का समाधान करने के लिए आसानी से तरीका पता चल जाता है और हमें सुनिश्चित कर देता है कि हमारा कोड सही है। आप अपने कोड में नेमस्पेस के साथ काम कर सकते हैं, जो कि आपको अलग-अलग कोड में अधिक आसानी से एक वैयविक होने में मद
