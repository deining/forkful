# क्यों

यदि आप किसी टेक्निकल ब्लॉग या व्यवसायिक क्षेत्र में काम कर रहे हैं, तो आपने शायद अपने कोड में कभी-कभी "स्ट्रिंग" को कैपिटलाइज़ करने की आवश्यकता महसूस की होगी। स्ट्रिंग कैपिटलाइज़ करने से आपका कोड पढ़ने में और समझने में आसान हो जाता है और यह आपके कोड को एक अभिनव दिखावट देता है। 

# कैसे करें

```Kotlin 
fun capitalize(str: String): String{
    return str.capitalize()
}

fun main(){
    val str = "hello world"
    println(capitalize(str))
}

```
आप स्ट्रिंग को कैपिटलाइज़ करने के लिए एक सिंपल फ़ंक्शन बना सकते हैं जो `capitalize()` फ़ंक्शन को कॉल करता है। आप यह फ़ंक्शन किसी भी स्ट्रिंग पर कॉल कर सकते हैं जैसे "hello world" और यह आपको "Hello world" लौटाएगा। 

# गहरा डाइव

स्ट्रिंग कैपिटलाइज़ करना अधिक समझने के लिए, आपको स्ट्रिंग टाइप के बारे में थोड़ी अधिक जानकारी होनी चाहिए। स्ट्रिंग को एक सिर की तरह समझा जा सकता है जो ग्राहकों को यूनिकोड या यूनिकोड सीरीज से मिलकर बनाई गई है। स्ट्रिंग `capitalize()` उपयोगकर्ता हार्डकोडेड कैसे सूचित करता है? यह यूनिकोड प्रतीकों को पढ़ता है जो अंतिम स्थान पर हैं और उसके बाद उसे कैपिटलाइज़ करता है। 

इसलिए यह अपने प्रयासों को दोनों तरीकों से और आसान बनाता है। प्रथम तरीके में, आपको किसी भी स्ट्रिंग के पहले एकर्स या उसके साथ बात
