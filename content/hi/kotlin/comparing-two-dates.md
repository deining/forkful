## क्यों

एक सामान्य प्रोग्रामिंग कार्य में, हमारे पास दो तारीखों का पोर्ट्फोलियो है। इन तारीखों को तुलना करना एक आमजन थोड़ा आसान काम हो सकता है, लेकिन इसमें थोड़ा कम्प्लेक्सिटी भी हो सकती है। इसलिए, हम आपको बताएंगे कि किस तरह आप कोटलिन का उपयोग करके दो तारीखों की तुलना कैसे कर सकते हैं।

## कैसे

यदि हमारे पास दो तारीखों का पोर्टफोलियो है, तो हम इन तारीखों को आसानी से तुलना कर सकते हैं और समय के रूप में उपयोग कर सकते हैं। आप ```Kotlinif (date1 < date2) { println("Date 1 comes before Date 2") } else if (date2 < date1) { println("Date 2 comes before Date 1") } else { println("Both dates are the same") } ``` कोड ब्लॉक में समानार्थक समानार्थक समानार्थक समानार्थक समानार्थक समानार्थक समानार्थक समानार्थक समानं कर सकते हैं। यह उदाहरण ```Kotlin2021-06-01 < 2021-06-15``` के लायक है। इसकी आउटपुट "Date 1 comes before Date 2" होगी।

## गहराई में जाएं

दो तारीखों की तुलना अक्सर ऑपरेशनल प्रोग्रामिंग जैसे कि समय सत्र या ईवेंट प्रकार जैसे "लंच", "दौड़" और "सोना" से जुड़ा होता है। हमारे पास बहुत से समय सत्र हो सकते हैं और हम उनमें से एक को दूसरे से तुलना करना चाहते हैं। ```LocalDate``` एंड कोटलिन कोड ऑपरेशन में सबसे आसान तरीका है। आप ```java.time.LocalDate``` और ```java.time.LocalTime``` का भी उपयोग कर सकते हैं।
