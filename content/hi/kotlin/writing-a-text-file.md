# क्यों

कोटलिन प्रोग्रामिंग भाषा पूर्णता के साथ आसानी से सीखी जा सकती है और यह आमतौर पर एक प्रोग्रामर के लिए पसंदीदा होती है। इसके साथ-साथ, टेक्स्ट फ़ाइलों को योग्य रूप से लिखने और पढ़ने के लिए कोटलिन का उपयोग करना सरल और सुलभ है। इस ब्लॉग पोस्ट में, मैं यह बताऊंगा कि टेक्स्ट फ़ाइल कैसे लिखने के लिए कोटलिन का उपयोग कर सकते हैं। 

# कैसे करें

यदि आपने पहले से ही कोटलिन प्रोग्रामिंग भाषा का इस्तेमाल नहीं किया है, तो पहले आपको इसे सिखना होगा। इसके बाद, आप टेक्स्ट फ़ाइल को लिखने के लिए कोटलिन की  `java.io.File`  कक्षा का उपयोग कर सकते हैं। 

उदाहरण के लिए, अगर आपने एक `sample.txt`  नामक टेक्स्ट फ़ाइल बनाई है, तो आप इसे निम्नलिखित कोड का उपयोग करके लिख सकते हैं: 

```Kotlin
import java.io.File

fun main() {
    val file = File("sample.txt")
    file.writeText("यह एक सैम्पल टेक्स्ट है।")
}
```

इस कोड में, `File` आपके फ़ाइल को संदर्भित करने के लिए उपयोग किया गया है, वहीं `writeText()`  आपकी फ़ाइल में दी गई सामग्री को लिखता है। इस उदाहरण में, `sample.txt`  फ़ाइल में "यह एक सैम्पल टेक्स्ट है।" यह लिख दिया जाएगा।
