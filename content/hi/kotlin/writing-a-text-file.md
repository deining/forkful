## क्यों
कोटलिन प्रोग्रामिंग लैंग्वेज में पाठ फाइल लिखना काफी उपयोगी हो सकता है। इससे, आप अपने कोड को एक स्थान पर संग्रहीत कर सकते हैं और दिए गए कोड ब्लॉक को बार-बार इस्तेमाल करने से बच सकते हैं।

## कैसे करें
कोटलिन में एक टेक्स्ट फाइल कैसे लिखें -
```kotlin
import java.io.File

fun main(){
    val textFile = File("myfile.txt")
    textFile.writeText("This is a sample text file.")
}
```

इस उदाहरण में, हमने `java.io.File` लाइब्रेरी का उपयोग करके `myfile.txt` नाम की टेक्स्ट फाइल बनाई है और उसमें "This is a sample text file." लिखा है। आप अपनी तरह का मात्राळ फाइल बना सकते हैं और उसमें अपने खुद के एडिटोरियल कॉंटेंट डाल सकते हैं।

## गहराई तक बहस करें
टेक्स्ट फाइल लिखने के लिए और भी कई तरीके हैं, जैसे फ़ाइल खोलकर लिखना या `FileWriter` का उपयोग करना। आप अपनी पसंद के अनुसार जो भी तरीका चुन सकते हैं और अपनी कोड बेस्ट प्रैक्टिस को ध्यान में रखते हुए टेक्स्ट फाइल लिख सकते हैं। इसके अलावा, आप टेक्स्ट फाइल की पढ़ाई और संपादन से संबंधित अधिक जानकारी भी खोज सकते हैं।
