## इसका क्यों (Why)

यैमल (YAML) एक पॉपुलर डेटा फॉर्मेट है जो पाठ, ऑब्जेक्ट और आरएसएस से डेटा को बचाने और प्रसंस्करण करने के लिए उपयोग किया जाता है। कोटलिन में यैमल फाइल के साथ काम करना एक आसान, सुरक्षित और उपयोगी तरीका है।

## कैसे करें (How To)

यैमल फाइल बनाने के लिए, आपको सबसे पहले `SnakeYaml` डिपेंडेंसी को अपने ग्रेडल फाइल में जोड़ना होगा। फिर स्नेक यैमल क्लास को इनिशलाइज़ करें और यैमल फाइल को खोलने के लिए `load()` यैमल क्लास का उपयोग करें। नीचे दिए गए उदाहरण में हम एक यैमल फाइल से डेटा को रीड करेंगे:

```Kotlin
// `snakeyaml` डिपेंडेंसी जोड़ें
import org.yaml.snakeyaml.Yaml

fun main(){
    // स्नेक यैमल क्लास इनिशलाइज़ करें
    val yaml = Yaml()

    // यैमल फाइल खोलें
    val data = yaml.load(FileReader("sample.yaml"))

    // डेटा फाइल से की गई थोकों प्रिंट करें 
    println(data)
}
```

उपरोक्त उदाहरण में, हम `sample.yaml` फाइल से डेटा पढ़ेंगे। यैमल फाइल में डालने के लिए, हम `dump()` यैमल क्लास का उपयोग  कर सकते हैं।
