## क्यों

समय-बचाव, सरलता और ताकत - ये सिर्फ कुछ वे लाभ हैं जो आपको नियमित अभिव्यक्तियों का उपयोग करने से मिलते हैं। वे आपकी कोडिंग प्रक्रिया को भी आसान और स्पष्ट बनाते हैं।

## कैसे करें

कोटलिन में नियमित अभिव्यक्तियों का उपयोग करना बहुत आसान है। आप बस `regex` को विकी सदस्य क्षमता के साथ एक कोटलिन श्रृंखला में पारंगत करने के लिए जोड़ सकते हैं। इसके बाद, आप अपने स्ट्रिंग को `matches()` या `find()` में पास करके एक पैटर्न को खोज सकते हैं। निम्न उदाहरण आपको इस बारे में और अधिक स्पष्ट रूप से समझाएगा:

```Kotlin
val str = "वर्ष = 2020"

val pattern = """(वर्ष)\s=\s(\d\d\d\d)""".toRegex()

val result = pattern.find(str) // पैटर्न से मिलता हुआ पहला समान हमारे रिजल्ट में होगा

if (result != null) {
  val (key, value) = result.destructured // रिजल्ट से अनुरोध करने के लिए दोनों आईटम को अलग करता है
  println("$key = $value") // आउटपुट : वर्ष = 2020
}
```

## गहराई में जाएँ

नियमित अभिव्यक्तियों को समझना महत्वपूर्ण है और इसके लिए समय लग सकता है। लेकिन जब आप इसे एक बार सीख लेते हैं, तो यह आपके कोडिंग को सरल बना सकता है। अधिक जानकारी के लिए, आप [कोटलिन डॉक्यूमेंटेशन](https://kotlinlang.org/docs/regular-expressions.html) का उपयोग कर सकते हैं और जान सकते हैं कि कैसे आप अलग-अलग तरीकों से नियमित अभिव्यक्तियों का उपयोग
