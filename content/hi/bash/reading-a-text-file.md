## क्यों

एक टेक्स्ट फाइल पढ़ने के लिए कुछ कारण हो सकते हैं। शायद आपको एक स्क्रिप्ट चलाना है जो टेक्स्ट फाइल से डेटा पढ़ कर उसे प्रोसेस करता है या फिर आप में से कोई एक टेक्स्ट फाइल को अपने सिस्टम पर स्टोर करना चाहता हो सकता है। इसलिए टेक्स्ट फाइल पढ़ना बहुत ही उपयोगी हो सकता है जिससे आप अपने काम को आसानी से पूरा कर सकते हैं।

## कैसे करें

टेक्स्ट फाइल को पढ़ने के लिए आपको `cat` कमांड का उपयोग करना होता है। `cat` कमांड फाइल जोड़ने और पढ़ने के लिए इस्तेमाल किया जाता है। नीचे दिए गए कोड ब्लॉक में हम देखेंगे कि कैसे आप एक टेक्स्ट फाइल को पढ़ सकते हैं और उससे डेटा प्रिंट कर सकते हैं।

```Bash
# टेक्स्ट फाइल बनाएं
touch sample.txt

# फाइल में कुछ डेटा लिखें
echo "नमस्ते दुनिया!" > sample.txt

# फाइल को पढ़ें
cat sample.txt
```

इस कोड का आउटपुट होगा:

```Bash Output
नमस्ते दुनिया!
```
