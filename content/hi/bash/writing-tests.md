## क्यों

ताजर्बा आपकी कोडिंग क्षमताओं को सबल बनाता है और सुनिश्चित करता है कि आपके कोड में खामियां नहीं हैं। यह आपको आपके उत्पाद की गुणवत्ता को बढ़ाने में महत्वपूर्ण भूमिका निभाता है।

## कैसे करें

कोड टेस्टिंग एक एक्सलरेटर हो सकता है। नीचे एक उदाहरण दिया गया है जो आपको समझने में मदद करेगा कि कैसे आप अपने खुद के बैश स्क्रिप्ट लिख सकते हैं और उन्हें परीक्षण कर सकते हैं।

```Bash
#!/bin/bash

# संचार स्क्रिप्ट दिखाएँ
echo "नमस्ते, कैसे हो?"

# मान सुनिश्चित करो
OUTPUT="नमस्ते, मैं ठीक हूँ"

# यदि आपको अपने संदेश को दोहराने के लिए खुद को पार्थिव करना होता है
if [ "$OUTPUT" == "नमस्ते, मैं ठीक हूँ" ]; then
  echo "परीक्षण पारित हुआ!"
fi

# नुक्तियों को शामिल करें
exit 0
```

उपरोक्त भाग में, हम संचार स्क्रिप्ट दर्शाते हैं। फिर हम अपने संदेश को सुनिश्चित करते हैं और यदि संदेश ठीक है तो हम परीक्षण को पारित करते हैं। अंत में, हम नुक्तियों को उपयोग करते हुए कोड से बाहर निकलते हैं।
