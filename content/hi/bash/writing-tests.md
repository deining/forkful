## क्यों

टेस्ट कोड लिखना क्यों महत्वपूर्ण है? बड़ से बड़ परियोजनाओं को भोले से छोटी समस्याओं से दूर रखने और कोड बग्स की जांच करने के लिए टेस्ट कोड लिखना आवश्यक होता है। यह प्रोग्रामिंग प्रक्रिया को आसान और सुरक्षित बनाने में मदद करता है।

## कैसे लिखें

टेस्ट कोड लिखने के लिए आपको अपने बैश शेल स्क्रिप्ट के साथ नए स्क्रिप्ट फाइल को खोलना होगा। आप बैश कमांड्स का उपयोग करके अपने स्क्रिप्ट में विभिन्न प्रकार की परिवर्तन लागू कर सकते हैं। यहां हम शामिल हैं एक coding example कोड ब्लॉक के साथ जो आपको इस प्रक्रिया को बताएगा:

```Bash
#!/bin/bash
# स्वागत करते हैं! यह हमारा टेस्ट स्क्रिप्ट है।
echo "हेलो वर्ल्ड"
echo "यह टेस्ट स्क्रिप्ट कंपलेशन सफल हुआ!"
exit 0
```

इस स्क्रिप्ट में, हमने अपने टेस्ट स्क्रिप्ट में विभिन्न बैश कमांड्स जोड़े हैं जो धीरे-धीरे आपको कोड की सहीता दिखाएगा। आप अपने स्क्रिप्ट को एक बार कुछ बदलाव करने के बाद सेव करके उसे एक टेस्टिंग वायल्डवेस्ट में चला सकते हैं।
