## क्यों
यदि आप एक वेब डेवलपर हो या आपको वेब क्षेत्र में काम करना पसंद है, तो आपने HTTP अनुरोध (HTTP request) के बारे में जरूर सुना होगा। HTTP अनुरोध भेजकर हम सर्वर से डेटा को प्राप्त कर सकते हैं और वेब अप्लिकेशन विकसित कर सकते हैं। 

## कैसे करें
यदि आपने कभी भी बैश प्रोग्रामिंग का अनुभव नहीं किया है, तो आपको सबसे पहले बैश इंस्टॉल करना होगा। आप इस आर्टिकल को पढ़कर बैश को कैसे इंस्टॉल करे सीख सकते है। 

अब, बैश के साथ HTTP अनुरोध भेजने के लिए हमें `curl` कमांड का उपयोग करना होगा। `curl` कमांड एक URL पर HTTP अनुरोध भेजने के लिए उपयोग में लाया गया है। नीचे दिए गए उदाहरण में, हम `curl` कमांड का उपयोग करके गूगल के होमपेज पर एक साधारण HTTP अनुरोध भेजेंगे।

```Bash
curl https://www.google.com
```

इससे हमें गूगल के होमपेज का HTML कोड प्राप्त होगा। यदि आप `curl` कमांड के बीच गूगल के लिए कोई अन्य URL देंगे, तो आप उस साइट का HTML कोड प्राप्त कर सकते हैं। इस तरह से, हम गूगल या किसी भी उद्देश्य के लिए किसी भी साइट से डेटा को प्राप्त कर सकते हैं।
