## क्यों

टेक्स्ट फ़ाइल लिखना शायद आपको एक समस्या को हल करने के लिए या अपने कंप्यूटर पर आपकी प्रोग्रामों या स्क्रिप्ट के साथ डेटा को स्टोर करने के लिए किया जाता है।

## कैसे करें

यदि आप अपने टेक्स्ट फ़ाइल को ब्राउजर पर देखना चाहते हैं, तो निम्नलिखित संकलन प्रकृति का उपयोग करके यह कार्य किया जा सकता है:

```Bash
echo "यह एक सरल टेक्स्ट फ़ाइल है" > textfile.txt
cat textfile.txt
```
आप भी अपने टेक्स्ट फ़ाइल को संपादित कर सकते हैं, उदाहरण के लिए:

```Bash
nano textfile.txt
```

आप अपनी टेक्स्ट फ़ाइल में अपनी विभिन्न लाइनों में सूचनाओं को जोड़ सकते हैं। संपादन पूरा होने के बाद, आप <code>Ctrl + X</code> दबा कर बाहर आ सकते हैं।

## गहराई तक

टेक्स्ट फ़ाइलों उचित ढंग से और आसानी से डेटा को स्टोर करने का एक तरीका है। आप प्रोग्रामिंग में उनका उपयोग कर सकते हैं ताकि आप पुनर्प्रयोग के लिए अपने काम को सुरक्षित रूप से संग्रह कर सकें। वे संचित डेटा को प्रपत्र के साथ अद्यतन और संशोधित करने की भी अनुमति देते हैं।
