## क्यों

बश (Bash) हमें YAML का उपयोग जैसे डेटा को संगठित और कॉन्फ़िगर करने के लिए काम में आता है। इससे हमारे काम को आसान बनाने और समय बचाने में मदद मिलती है।

## कैसे करें

अगर आपको YAML फ़ाइल से काम करना है, तो सबसे पहले आपको बश में YAML लाइब्रेरी को इंस्टॉल करना होगा। इसके लिए आप `yum install libyaml` कमांड का इस्तेमाल कर सकते हैं। इंस्टॉलेशन के बाद, आप नीचे दिए गए कुछ उदाहरणों का उपयोग करके YAML फ़ाइल से डेटा पढ़ सकते हैं और उसे डिक्शनरी या सूची में रूपांतरित कर सकते हैं:

```
#!/usr/bin/env bash

# डेटा पढ़ना
data=$(cat file.yaml)

# डेटा को डिक्शनरी में रूपांतरित करना
dict=$(yq read "$data" -)

# डेटा को सूची में रूपांतरित करना
list=$(yq read "$data" -- -)

echo "डिक्शनरी: $dict"
echo "सूची: $list"
```

यदि आप किसी YAML फ़ाइल को अपडेट करना चाहते हैं, तो आप `yq write` कमांड का इस्तेमाल कर सकते हैं। यह आपको अपनी फ़ाइल में उपयोग कर रही डेटा को अपडेट करने की अनुमति देता है।
