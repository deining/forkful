## क्या और क्यों? 
पार्सिंग एचटीएमएल क्या है और क्यों प्रोग्रामर्स इसे करते हैं? पार्सिंग एचटीएमएल से अपना कोड इंटरनेट से समझता है और उससे डेटा निकालता है। प्रोग्रामर्स इसे डेटा संग्रहित करने, विश्लेषण करने और साइटों को स्क्रैप करने के लिए करते हैं।

## कैसे करें: 
एक बेहद सरल उदाहरण है: दिए गए यूआरएल से डेटा पार्स करें और उसे दिखाएं।
```Bash
curl https://www.example.com | grep "data"
```

यहां एचटीएमएल डॉक्युमेंट से सिर्फ "डेटा" के लिए उपयुक्त पाठ धुंधला रहेगा।

```Bash
data
data
userdata
```

## गहराई से जानें: 
पार्सिंग एचटीएमएल को अपने रूप में बदलने के लिए कई विकल्प हैं, जैसे डोम के साथ खिलबाड़ी विश्लेषण, पाइसन, सोपा। इन विकल्पों में से सबसे स्वाभाविक एक है रेगुलर एक्सप्रेशन है। एचटीएमएल को पार्स करने के लिए दो नैविक आधार जोड़े गए हैं जो रेग्युलर एक्सप्रेशन में हैं। पार्सिंग टेक्स्ट ग्राहकों (जैसे वेब ब्राउज़र) को संग्रहित डेटा में प्रतिसाद देता है।
