## क्यों

बैश प्रोग्रामिंग एक बहुत ही उपयोगी कौशल है जो कंप्यूटर प्रोग्रामर्स को अपनी अनुपस्थिति के समय शक्तिशाली उपकरणों का उपयोग करने देता है। इसका एक अहम भाव ऑपरेटिंग सिस्टम से कमांड लाइन स्वरूप में दिया जाना है जिससे आप आसानी से अपनी कमांड लाइन आपरेशन को प्रोग्रामिंग भाषा के माध्यम से निर्मित कर सकते हैं।

## कैसे करें

आप अपने बैश स्क्रिप्ट में पास किये गए कमांड लाइन आर्ग्यूमेंट्स को कैसे पढ़ सकते हैं यह जानने के लिए निम्नलिखित उदाहरण का सुझाव दिया गया है:

```Bash
# यह एक डाउनलोड स्क्रिप्ट है जो आपको कोई भी वेब पेज (या छवि या वीडियो) को डाउनलोड करने के लिए अनुमति देता है।
#!/bin/bash

# पास किये गए आर्ग्यूमेंट्स के साथ बैश स्क्रिप्ट को कॉल करें
sh download_page.sh "https://www.example.com" "website"

# कॉल किए गए आर्ग्यूमेंट्स को प्रिंट करें
echo "वेबसाइट का नाम: $2, साइट का URL: $1"

# $1 को दुबारा प्रिंट करने के लिए `echo` का उपयोग करें
echo "$1 से डाउनलोड हो रहा है..."

# डाउनलोड शुरू करने के लिए `wget` का उपयोग करें
wget -m -E -k "$1"

# प्रिंट सफलतापूर्वक पूर्ण हो गया मैसेज
echo "$1 सफलतापूर्वक पूर्ण हो गया!"

# लॉग फ़ाइल में आउटपुट साहित्य को रखने के लिए `tee` कमांड का उपयोग करें
exec > >(tee "download_log.txt")

```
