## क्यों
क्या आप जानना चाहते हैं कि आपको आज किस दिन है? आपका दिन कैसे शुरू हो रहा है? एक संभावित काम और अवकाश की जरूरतों को मालूम करने के लिए, आप अपने स्क्रिप्ट में वर्तमान दिनांक का प्राप्त कर सकते हैं। 

## कैसे करें
मामूली आसान तरीके से, आप अपने बैश स्क्रिप्ट में निम्न लाइन जोड़ कर वर्तमान दिनांक को प्राप्त कर सकते हैं:
```Bash
date
```
यदि आप सक्रिप्ट को रन करते हैं, तो आपको कुछ इस तरह का प्रदर्शन मिलेगा:
```
शुक्रवार, 28 जून 2019 09:27:12 +0530
```

आप भी विभिन्न स्विच का उपयोग करके विभिन्न तारीख और समय प्रस्तुत कर सकते हैं। जैसे, आप सिर्फ दिनांक को प्रदर्शित करने के लिए `+%D` स्विच का उपयोग कर सकते हैं:
```Bash
date +%D
```
इस हालत में, आपको निम्नलिखित प्रदर्शित होगा:
```
06/28/19
```
