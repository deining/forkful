## क्या और क्यों?

वर्तमान तारीख का पता लगाने से तारीख कोडिंग में उपयोगी बनाया जाता है। प्रोग्रामर्स यह करते हैं ताकि अपने स्क्रिप्ट्स, एप्लिकेशन या कमांड लाइन में वर्तमान तारीख का उपयोग कर सकें।

## कैसे:

```Bash
# अगर आपको स्क्रिप्ट में वर्तमान तारीख का उपयोग करना है, तो यह कोड आपके काम आएगा:
current_date=$(date +"%m/%d/%Y")
echo "वर्तमान तारीख है: $current_date"

# यदि आप एक एप्लिकेशन बना रहे हैं तो आपको date आर्ग्यूमेंट के साथ date मेथड का उपयोग करना होगा:
date +"%m/%d/%Y"

# वर्तमान तारीख को अन्य फॉर्मेट में प्राप्त करने के लिए, आप date मेथड में अन्य फॉर्मेट ऑप्शन का उपयोग कर सकते हैं। जैसे की:
# वर्तमान तारीख: 24 मई
date +"%d %B"

# समय को प्राप्त करने के लिए, टाइम स्टाम्प का उपयोग करें:
date +"%s"
```

आउटपुट:
```
वर्तमान तारीख है: 05/24/2021
```

## गहराई में:

इस फीचर को bash में 1989 में एरिक जे रेमिंग्टन ने फर्स्ट से इंट्रोड्यूस किया था। अन्य प्रोग्रामिंग भाषाओं में भी वर्तमान तारीख को प्राप्त करने के बहुत से तरीके हैं जैसे python में datetime लाइब्रेरी और java में Date और Calendar क्लास।

वर्तमान तारीख को प्राप्त करने के लिए और भी तरीकों में से एक date मेथड का उपयोग करना सबसे सरल और ज्यादा उपयोगी है। इसमें सुनिश्चित किया गया है कि उपयोगकर्ता की स्थानिक समय क्षेत्र के अनुसार तारीख को प्राप्त किया जाता है।
