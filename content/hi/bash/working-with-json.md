## क्यों

यदि आप किसी भी एप्लिकेशन या वेबसाइट को डेटा को समूहों में संरचित करने और उसे समूहों में भेजने के लिए उपयोगकर्ता को देखें तो JSON (जावास्क्रिप्ट ऑब्जेक्ट नोटेशन) एक अत्यंत महत्वपूर्ण डेटा प्रारूप है। यह एक सरल एवं लोकप्रिय प्रारूप है, जिसे डेवलपर्स नेटवर्क के माध्यम से भेजते हैं और जिसे वापस प्राप्त करते हैं।

## कैसे करें

कैसे करें अब आप कुछ उदाहरण देखेंगे कि कैसे आप ब्राउज़र से JSON डेटा प्राप्त कर सकते हैं और कैसे उसे Bash में संपादित कर सकते हैं।

curl https://example.com/api/users # यह कमांड ब्राउज़र से JSON डेटा प्राप्त करता है
