## क्यों

JSON, जो जेसन नामक मानचित्रिक संरक्षण प्रारूप को दर्शाता है, आजकल प्रोग्रामिंग में बहुत उपयोगी है। यह एक सरल, संक्षिप्त और स्थायी संरचना है जो डेटा को संरचित करने और संग्रहीत करने के लिए उपयोग किया जाता है। इसके अतिरिक्त, यह विभिन्न प्रोग्रामिंग भाषाओं में समरूप है।

## कैसे करें

अगर आपको अभी तक JSON के साथ काम नहीं किया है, तो चिंता न करें। आपको इसकी थुथुकी होने से बचाने के लिए में कुछ आसान चरणों को दिखाऊंगा जो आपको इसे सीखने में मदद करेंगे। पहले, आपको `bash` टर्मिनल में `curl` कमांड का उपयोग करके सेवा का एक रिक्वेस्ट भेजना होगा। उदाहरण के लिए:

```Bash
curl https://jsonplaceholder.typicode.com/todos/1
```

यह आपको एक JSON रिस्पांस देगा, जैसा कि नीचे दिखाया गया है:

```Bash
{
  "userId": 1,
  "id": 1,
  "title": "delectus aut autem",
  "completed": false
}
```

अगर आप अपनी योग्यता के अनुसार खेताब और प्रति सक्षमता से कमांड को पहचानते हैं, तो आप अपने सर्वर पर JSON अपवित्रित कर सकते हैं।

## गहराई में उतरें

आपको जानना-कुछ चीजों पर ध्यान देना चाहिए जब आप JSON के साथ काम कर रहे हैं। पहले से ही बताया गया है कि यह मानचित्रिक संरक्षण प्रारूप बाहर बैठ सकता है और आप सामान्य JSON डोके पाएंगे।
