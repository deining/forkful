## क्यों

यदि आप बैश प्रोग्रामिंग में शुरुआती हैं तो मालूम होगा की टेक्स्ट को uppercase में लिखने के लिए `tr` कमांड आपको पाठ में कैसे जोड़ता है, लेकिन अगर आपका काम केवल कपिटलाइज करने पर ही सीमित होता है तो आपको इस तरह के एक्स्टरनल कमांड की आवश्यकता नहीं होगी। प्रोसेस में `tr` कमांड का उपयोग आसान दिखे रहता है, लेकिन जब बात केवल टेक्स्ट को कैपिटलाइज करने की होती है तो `awk` कमांड का उपयोग करना बेहतर हो सकता है।

## कैसे करें
कैपिटलाइज कोde परमेटर के रूप में की आज सीखें :

```
Bash

#!/bin/bash

function capitalize(){

# प्रत्यक शब्द को upper case में बदल देगा |
    awk 'BEGIN { FS="" ; OFS=""} { $1 = toupper($1) } 1' <<<"$1"
# यहाँ पर आपका parameter 1 शब्द को upper case में करनेके लिए होगा |
}
capitalize "hello world"
# यह output दिखेगा "Hello world"
```

अपने प्रोग्राम के लिए bash फंक्शन बनाना ताकि उसे अन्य प्रोग्रामों में भी इस्तेमाल किया जा सके।

## गहराई में

जब आप `awk` कमांड का उपयोग करते हैं।आपको उसकी विभिन्न प्रकार में परिचय होना चाहिए। जैसे :

- आप awk के औपचारिक और GNU परिभाषित परिचय कर सकते हैं
- आपको समान क्रिया के साथ कई बिल्ट-इन विधियां और स्क्रिप्टिङ्ग कपैबिलिटियां प्रदान करता है

यह तरीके आपको `awk` कमांड को बेहतर ढंग से समझने में मदद करेंगा|
