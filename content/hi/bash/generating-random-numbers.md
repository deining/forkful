## क्यों?
यदि आप कंप्यूटर प्रोग्रामिंग में दिलचस्पी रखते हैं, तो आपने कभी न कभी ‘यादृच्छिक संख्याएं’ शब्द को सुना होगा। यह कंप्यूटर कार्य करने का एक रोचक तरीका है। कम से कम एक बार जरूर उन्हें जानने लायक होना चाहिए। 

## कैसे करें?
यहां हम बैश भाषा का इस्तेमाल करके रांडम संख्याओं को उत्पन्न करने के तरीके पर चर्चा करेंगे। साथ ही हम उन परीक्षण तकनीकों को समझेंगे जो आपको अधिक स्पष्टता देंगे।

```Bash
# उच्च से उच्च संख्या (1-100) तक की यादृच्छिक संख्या को उत्पन्न करें
echo $RANDOM

# एक से वरुण (100 या दिए गए सीमा के बीच) में उत्पन्न रांडम संख्या को प्रिंट करें
echo $((RANDOM % 100 + 1))

# दिए गए सीमा के बीच की एक से अधिक रांडम संख्याओं को उत्पन्न करें और उन्हें रिकार्डिंग करें
for i in {1..10}
do
    echo $((RANDOM % 100 + 1))
done

# दिए गए सीमा के बीच की एक से अधिक रांडम संख्याओं को प्रिंट करने के लिए फंक्शन का उपयोग करें
generate_random_numbers() {
    for i in {1..10}
    do
        echo $((RANDOM % 100 + 1))
    done
}

generate_random_numbers
```

उपरोक्त कोड सन्मानिताएँ और अपने संख्या की परीक्षण करें।
