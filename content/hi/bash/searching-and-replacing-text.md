## क्यों

टेक्स्ट को ढूंढ़ने और बदलने के लिए *क्यों* कुछ लोग करते हैं यह जानना अधिकतर कंप्यूटर प्रोग्रामर्स के लिए आवश्यक है। यह युग टेक्नोलॉजी और डिजिटल सांस्कृतिकरण का होते हुए किसी भी ऑपरेटिंग सिस्टम को संपादित करने के लिए उपयोगी है।

## कैसे करें

इस ब्लॉग पोस्ट में, हम टेक्स्ट को खोजने और बदलने के लिए बैश प्रोग्रामिंग का उपयोग करना सीखेंगे। हम बैश के अनुसार ```sed``` और ```awk``` जैसी टूल्स का भी इस्तेमाल करेंगे। इन उपकरणों के द्वारा हम अपने टेक्स्ट फ़ाइल्स में कंटेंट को खोज और उसे बदल सकते हैं। हम अलग-अलग मेथड्स के साथ कोडिंग संग्रह और उसके साथ प्रदर्शित आउटपुट को भी समझेंगे। चलिए शुरू करते हैं!

```Bash
# आवश्यक लाइब्रेरी सामग्री को इम्पोर्ट करें
import re

# उपयोगकर्ता से स्ट्रिंग इनपुट लें
string = input("कृपया अपना टेक्स्ट इनपुट करें: ")

# आप अपने आवश्यकतानुसार यूजर के डाटा आउटपुट को अलग-अलग मान पर देख सकते हैं
# यूजर से जो चर्चा लिखा है, उसे अलग अलग मानों पर दिखाने की अनुमति है।
new_string = re.sub("चर्चा", "कल्पना", string)

# नया स्ट्रिंग आउटपुट करें
print("नया स्ट्रिंग:", new_string)
```

आपका टेक्स्ट फ़ाइल अब चर्चा की जगह कल्पना देखेगा।
