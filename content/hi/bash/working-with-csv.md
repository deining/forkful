## क्यों

आपने कभी शायद CSV (Comma Separated Values) फाइल्स के साथ काम किया होगा। वे इंटरनेट से हमेशा से एक लोकप्रिय तरीका रहे हैं, टेक्स्ट डेटा को संगठित और इंटरप्रेट करने के लिए। Bash आपको CSV फाइलों को आसानी से प्रोसेस करने में मदद कर सकता है, जिससे आप अपने डेटा को अधिक सुव्यवस्थित और प्रभावी तरीके से उपयोग कर सकते हैं। 

## कैसे करें

आप Bash उपयोग करके CSV फाइलों को कैसे प्रोसेस कर सकते हैं, उसके एक उदाहरण के साथ आसानी से समझाया जाएगा। इसके लिए, आपको आगे दिए गए कोड ब्लॉक को ```Bash``` कोड ब्लॉक्स में डालना होगा। 

```Bash
# CSV फाइल बनाएं
touch data.csv
# CSV फाइल में डेटा बनाएं
echo "नाम, उम्र, शहर" >> data.csv
echo "रोहन, 22, दिल्ली" >> data.csv
echo "अनिता, 25, मुंबई" >> data.csv
echo "मोहन, 20, चेन्नई" >> data.csv
# CSV फाइल को प्रिंट करें
cat data.csv
```

जब आप ऊपर दिए गए कोड ब्लॉक को रन करेंगे, तो आपको नया CSV फाइल बनाई गई होगी जिसमें नाम, उम्र और शहर स्तंभ होंगे। इसके बाद ```cat``` कमांड का उपयोग करके आप फाइल को प्रिंट कर सकते हैं।
