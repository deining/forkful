## क्यों
बश प्रोग्रामिंग अधिकांश बार अस्थाई फाइलें बनाने के लिए उपयोग किया जाता है। ये अस्थाई फाइलें बहुत सारे प्रोग्रामिंग टास्कों को सरल बनाने के लिए होती हैं, जैसे कि डेटा संसाधित करना या प्रतिस्थापित करना। 

## कैसे करें
आप बश में 'mktemp' कमांड का उपयोग करके आसानी से अस्थाई फाइलें बना सकते हैं। यह फाइलें हीरार्कियल नाम मिलते है और आप अपनी जरूरतों के अनुसार उन्हें बाहरी लोकेशन पर सहेज सकते हैं। निम्नलिखित उदाहरण में हम एक अस्थाई फाइल बना रहे हैं जो '/tmp' फोल्डर में सहेजी जाएगी। 

```Bash
#!/bin/bash
temp_file=$(mktemp /tmp/my_temp_file.XXXXXX)
echo "यह फाइल हीरार्कियल का नाम है: $temp_file"
```

जब आप रन करेंगे तो आपको कुछ इस तरह का आउटपुट मिलेगा: 

```
यह फाइल हीरार्कियल का नाम है: /tmp/my_temp_file.abcdef
```

## डीप डाइव
यह अभी तक जो उदाहरण हमने दिखाया है वह बस अवश्यक जरूरतों को पूरा करने के लिए है। यदि आप जानना चाहते हैं कि अस्थाई फाइल कैसे काम करती है और आपके सिस्टम पर कैसे उपयोग की जा सकती है, तो आप माइक्रोसॉफ्टवर्ड की यह चर्चा पढ़ सकते हैं: <https://www.microsoft.com/en-us/research/wp-content/uploads/2016/02/tr-2008-184.pdf>
