## क्यों

क्या आपने कभी अपनी बैश स्क्रिप्ट में इंटरएक्टिव टेक्स्ट खोजने की कोशिश की है? यदि हां, तो आपने शायद रेगुलर एक्सप्रेशन का उपयोग किया होगा। यह शक्तिशाली टेक्नोलॉजी आपको प्रतिक्रियाओं को फ़िल्टर करने, स्ट्रिंग्स को संशोधित करने और डेटा को खोजने के लिए अनुमति देती है। इसलिए, यदि आप अपने स्क्रिप्टिंग कौशल में सुमाधुर बनना चाहते हैं, तो आपको रेगुलर एक्सप्रेशन का उपयोग करना सीखना होगा।

## कैसे करें

यहाँ हम रेगुलर एक्सप्रेशन के बेसिक सिंटैक्स और उसके अलग-अलग प्रयोग देखेंगे। सबसे पहले, हमें एक नया बैश स्क्रिप्ट फ़ाइल बनाने की जरूरत होगी। हम `nano` कमांड का उपयोग कर सकते हैं या फिर अपने पसंद के कोई भी टेक्स्ट एडिटर का इस्तेमाल कर सकते हैं।

```Bash
nano regexp.sh
```

अब हम एक साधारण पाठ को मैच करने के लिए अपना प्रथम रेगुलर एक्सप्रेशन लिख सकते हैं। यहाँ हमें एक सदस्य की स्थिति को संभालने और उसको हमारे स्क्रिप्ट के अनुसार स्पष्ट करने की आवश्यकता होती है।

```Bash
#!/bin/bash

echo "Enter your membership status: "
read status

if [[ $status =~ ^(Active|Suspended|Expired)$ ]]; then
  echo "Your membership status is $status."
else
  echo "Invalid status."
fi
```

इस उदाहरण में, हमने `read` कमांड का उपयोग करके उपभोक्ता से स्थिति जानने के लिए कहा है और उसके बाद हम निर्धारित रेगुलर एक्सप्र
