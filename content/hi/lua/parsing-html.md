## परसिंग HTML यानी क्या और आखिर क्यों?

परसिंग HTML एक प्रोग्रामिंग कार्य है जहां प्रोग्रामर्स एक HTML कोड को विश्लेषित करते हैं और उसमें दिए गए टैग और अन्य तत्वों को पढ़कर मशीन संवाद में ट्रांसफर करते हैं। यह काम कुछ भी, जैसे डेटा स्रोत से जानकारी प्राप्त करना, वेब स्क्रैपिंग या वेब पेज के संरक्षण के लिए किया जाता है। 

## कैसे करे?

`lapis.html`गोधळिकासरगील ट्रांसफर करण्यासाठी, नेहमी एक आपल्या कोडमध्ये अगदी अधिसूचित मार्गाने प्रायोग करत जावे. HTML के कोड को दिखाने के लिए, आपको एक विशिष्ट प्रकार का ढंग (pattern) ठीक से स्थापित करना होगा:

```Lua
local var = string.find(html_code, "pattern")
```

इसके अलावा संबंधित लास (class) के मद्देने चेक करणे की आवश्यकता हो सकती है:

```Lua
if var then
    -- code goes here
end
```

आनुवंयन करण्यासाठी, आप `lapis.html.sanitize()` चोककळाने प्राप्त कर सकता हेतू आपला वापर करू शकता होताः

```Lua
local sanitized = lapis.html.sanitize(html_code)
```

इसके अलावा, आप `lapis.html.parse()` चोककळाने विशिष्ट रूप से व्याख्यान काम कर सकता हेतू आपला वापर करू शकता होताः

```Lua
local parsed = lapis.html.parse(html_code)
```

## गहरा उन्हाळा 

इतिहासिक संदर्भ, वैकल्पिक विधियाँ और परसिंग HTML के प्रायोगिक विधियों के बारे में गहरा जानकारी है:

- HTML का प्राकृतिक स्रोत कोड का उपयोग कर का वेब विश्वास है। इस काम को करने के लिए, आपको प्रायोगिक जैसे संबंधित निर्णायक कोड तैनात करने के लिए विशेष ध्यान देने की आवश्यकता न पड़े। 
- आप अपने वेबिसाइट पर सामाजिक साइटों का उपयोग करने के लिए एक Android ऐप के अपना वापर कर सकता हेतू आपला वापर करू शकता होताः जैसे [https://play.google.com/store/apps/details?id=com.qqnapex.translate_htmlscreenshot] 
- स्पेशल एक्सेंट को उपयोग करके अपने वेब साइट में कोड को संसाधित कर सकता हेतू आपला वापर करू शकता होताः जैसे MAC की आज़माइश.
