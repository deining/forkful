# क्या और क्यों?

नए प्रोजेक्ट की शुरुआत करना एक प्रोग्रामर के लिए उसके द्वारा एक नया उत्पाद बनाने का प्रयास है। यह उनके काम को और उनके क्षेत्र को आगे बढ़ाने के लिए होता है।

# कैसे करें?

```php 
<?php
echo "नया प्रोजेक्ट शुरू करना";
// आगे बढ़ने के लिए कुछ कोड यहां दर्ज करें 
?>
```

```php 
<?php
echo "नतीजा: नया प्रोजेक्ट सफलतापूर्वक शुरू हुआ";
?>
```

# गहराई में जाएं

(1) नए प्रोजेक्ट की शुरुआत की ऐतिहासिक संदर्भ, (2) वैकल्पिक विकल्पों की चर्चा, और (3) नए प्रोजेक्ट को कैसे शुरू करना है, उनकी गहराई में जाने की जरूरत हो सकती है। हालांकि, नए प्रोजेक्ट शुरू करने के लिए आपको सबसे पहले एक एमआलटी संस्करण का चयन करना हो सकता है।
