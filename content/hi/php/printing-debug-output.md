## क्यों
कभी-कभी हमारे कोड में दिक्कत आती है और हमें उसे ठीक करने का तरीका ढूंढना पड़ता है। ऐसे समय में डिबग आउटपुट का प्रयोग बहुत उपयोगी साबित होता है क्योंकि यह हमें कोड की गतिविधि को समझने और समस्या के स्थान का पता लगाने में मदद करता है।

## कैसे करें
डिबग आउटपुट को प्रिंट करने के लिए हम `echo` फंक्शन का उपयोग कर सकते हैं। यह फंक्शन स्ट्रिंग को प्रिंट करता है जो हमें कोड में डिबग करने के लिए उपयोगी सूचना प्रदान करता है। आइए एक उदाहरण के साथ इसे समझें:

```PHP
$name = "John";
echo "The value of name is: " . $name;
```

उपरोक्त कोड का आउटपुट निम्न रूप में होगा:

```
The value of name is: John
```

इस तरह से हम `echo` का उपयोग करके किसी भी वेरिएबल की मदद से उसका मान प्रिंट कर सकते हैं और डिबग कर सकते हैं। इसके अलावा, हम `print_r` फंक्शन का भी उपयोग कर सकते हैं जो रिकर्सिव रूप से वेरिएबल की जानकारी प्रदान करता है। आइए इसका भी एक उदाहरण देखें:

```PHP
$cities = ["Mumbai", "Delhi", "Kolkata"];

print_r($cities);
```

उपरोक्त कोड का आउटपुट निम्न रूप में होगा:

```
Array
(
    [0] => Mumbai
    [1] => Delhi
    [2] => Kolkata
)
```

इस तरह से हम `echo` और `print_r` फंक्शन का उपयोग करके आसानी से डिबग आउटपुट प्रिंट कर सकते हैं।
