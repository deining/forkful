## क्यों
  टेक्स्ट फ़ाइल लिखना कर नकशें, डेटा संग्रह और अन्य उद्देश्यों के लिए एक आवश्यक काम है। यह आसानी से पढ़ा और संपादित किया जा सकता है और लगभग सभी प्रोग्रामिंग भाषाओं में उपलब्ध है।

## कैसे करें
  टेक्स्ट फ़ाइल लिखने के लिए, हम `fopen()` और `fwrite()` फ़ंक्शन का उपयोग कर सकते हैं। नीचे दिए गए कोड ब्लॉक में एक आसान उदाहरण है।

```PHP
<?php
$file = fopen("example.txt", "w"); // नया फ़ाइल बनाएं या मौजूदा को खोलें
fwrite($file, "यह एक उदाहरण टेक्स्ट है।"); // फ़ाइल में डेटा लिखें
fclose($file); // फ़ाइल को बंद करें
?>
```

उपरोक्त कोड में, हम `fopen()` फ़ंक्शन के पहले पैरामीटर में नई फ़ाइल का नाम और दूसरे पैरामीटर में `w` जो कि लिखने का मोड है, पास करते हैं। फ़ाइल को बंद करने के लिए हम `fclose()` फ़ंक्शन का उपयोग करते हैं।

## गहराई में जाएं
  टेक्स्ट फ़ाइल लिखने के लिए अन्य भाषाओं की तुलना में, PHP में इसके कई सुविधाएं हैं। हम `fwrite()` फ़ंक्शन के अलावा `fputs()` और `file_put_contents()` भी उपयोग कर सकते हैं। इनमें से `file_put_contents()` फ़ंक्शन सबसे आसान है और इसका उपयोग करते हुए हमेशा फ़ाइल को बंद करने की आवश्यकता नहीं है। इसका निम्नलिखित उदाहरण देखें।

<?php
$file_contents = "यह दूसरा उदाहरण है।";
