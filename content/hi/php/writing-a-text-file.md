## क्यों

आप जब भी कोई टेक्स्ट फाइल लिखते हैं, तो ये आपके कोड या डेटा को सुरक्षित रखने का एक अच्छा तरीका है। इससे आप अपने डेटा की प्रत्येक मूल्यवान बिट को सुरक्षित रख सकते हैं और इसे आसानी से पढ़ा जा सकता है।

## कैसे करें

```PHP
<?php
$file = fopen("example.txt", "w"); // टेक्स्ट फाइल खोलिए और लिखने के लिए तैयारियाँ करें
fwrite($file, "यह सुपर सरल तरीके से लिखा गया है।"); // फाइल में कुछ लिखे
fclose($file); // खोला गया फाइल बन्द करें
echo "फाइल सफलतापूर्वक लिखी गयी!";
?>
``` 

फाइल में लिखना बहुत ही आसान है। पहले आपको ```fopen()``` फंक्शन का उपयोग करके फाइल खोलनी होगी और आप उस फाइल को कैसे लिखते हो यह बताना होगा। इसके बाद आप ```fwrite()``` फंक्शन का उपयोग करके अपने लिखें शब्दों को फाइल में लिख सकते हैं। आखिर में, आपको ```fclose()``` फंक्शन का उपयोग करके फाइल को बंद करना होगा।

## गहराई तक जाएं

टेक्स्ट फाइल लिखने के बारे में और अधिक जानने के लिए आप इस बारे में गहराई से जान सकते हैं। यह एक सुरक्षित तरीका है अपने कोड या डेटा को स्टोर करने का और उसे पढ़ने या लिखने के लिए आपको आसान से इस्तेमाल कर सकते हैं। आप दिनभर इसका उपयोग करते हैं जब आप कोई प्रोग्राम या स्क्रिप्ट लिखते हैं जो अपने डेटा को स्टोर करने की ज़रूरत
