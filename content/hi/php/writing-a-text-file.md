## क्यों

क्या आपने कभी सोचा है कि आप अपने कोड को टेक्स्ट फ़ाइल में कैसे स्टोर कर सकते हैं? यह संभव है यदि आप PHP प्रोग्रामिंग लैंग्वेज में काम करते हैं। टेक्स्ट फाइल आपके कोड को संग्रहीत रखने का एक बेहतरीन तरीका हो सकता है, जिससे आप अपनी कोड लेखन में आसानी से आग्रह कर सकते हैं। इस ब्लॉग पोस्ट में, हम टेक्स्ट फ़ाइल लिखने के तरीकों को जानेंगे और आपको इसके उपयोग के फायदे भी बताएँगे। आप अपने प्रोजेक्ट को और अधिक आसान बनाने के लिए यह लेख पढ़ सकते हैं।

## कैसे किया जाता है

तो चलिए देखते हैं कि कैसे आप एक टेक्स्ट फ़ाइल लिख सकते हैं। यह आपको आसान से समझने के लिए एक उदाहरण देता है: 

```PHP
$file = fopen("textfile.txt", "w"); // नया फ़ाइल क्रिएट करें या प्राचल से ओपन करें
$txt = "यह एक नयी लाइन द्वारा लिखा गया है।\n"; // यह यूनिकोड टेक्स्ट फ़ाइल में लिखने के लिए है
fwrite($file, $txt); // टेक्स्ट फ़ाइल में लिखें
$txt = "और यह दूसरी लाइन है।"; // एक और लाइन लिखें
fwrite($file, $txt); // टेक्स्ट फ़ाइल में लिखें
fclose($file); // फ़ाइल को बंद करें
```

अब आप `textfile.txt` फ़ाइल को ओपन करके देख सकते हैं और आपको दो नई लाइनें दिखाई देंगी।
