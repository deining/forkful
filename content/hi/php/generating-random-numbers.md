# यहां मुझे नंबर जेनरेट करने से कई लोगों को लोगों को जाना होगा कि अंकों को जेनरेट करने की विशेषता क्या है। इस काम का मुख्य उद्देश्य है कि इससे आप एल्गोरिथ्म को उपयोग करके अधिक ध्यानवर्धन प्रतिभागिता से जोड़ जा सकते हैं।

## कैसे करें?

```
PHP में यह हो सकता है संप्रधान एअर में यह एक तरिके से आता है कि यह आपको एल्गोरिथ्म का उपयोग करता है, क्योंकि यह अपनी प्राचीन तकनीक को अपवर्तक तैनात करेगा। एक उदाहरण

```

```PHP
// एक जग बनाइये
function randomNumber($min, $max) {
    return mt_rand($min, $max);
}

// फिर नंबर को कॉल करें
echo randomNumber(1, 100);

// नतीजा: 64
```

```
अतिरिक्त तौर पर, आप एक संपर्क मामला याद रखने के लिए सुमार वह को एक संप्रचार नंबर से अलग है। जैसे कि, आप थोड़ा सा कोड के जारी रख सकते हैं जिसमें आप अपने संपर्क को सूचीबद्ध रूप से छोड़ सकते हैं ताकि आपके लक्ष्य को सीमित किया जा सके।

## गहराई में पढ़ रहे हैं

दैनिक उपकरण IN PHP होल्डर डोम के साथ काम करने में अपने संपर्क होने से नंबर को बढ़ाने के लिए अतिरिक्त फीचर या व्यवस्थित तरीकों को प्रबंधित रूप से व्यवस्थापित रूप में पेश किया जाता है। उदाहरण के लिए, "mt_rand" फंक्शंन आपको प्राछली होगी। इसके अलावा, आप गंभीर सामान्य अनुभव धनादेश होना चाहिए।
