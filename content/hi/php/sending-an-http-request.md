## क्यों 

क्या आपने कभी सोचा है कि आप जब वेबसाइट पर किसी भी जानकारी देखते हैं तो उसमें कैसे निर्मित होता है? यदि हम इस विषय पर गहराई से जाएं, तो हम जानेंगे कि उन जानकारियों को लेकर हमें HTTP अनुरोध भेजने की जरूरत होती है। आप भी अपने कोड में HTTP अनुरोध जांचना चाहते होंगे या फिर यह सीखना चाहते होंगे कि कैसे आप HTTP अनुरोध भेज सकते हैं। इस लेख में, हम आपको बताएंगे कि PHP में कैसे HTTP अनुरोध भेजा जाता है। 

## कैसे 
आप कई प्रकार के HTTP अनुरोधों को भेज सकते हैं, जैसे GET, POST, PUT, DELETE आदि। ये सभी अनुरोध अलग-अलग प्रकार के डेटा को सर्वर पर भेज सकते हैं। नीचे दिए गए कोड ब्लॉक में हमने एक GET अनुरोध का उदाहरण दिया है, जो आपको समझने में मदद करेगा। 

```PHP 
<?php 
if(isset($_GET['name'])){
    $name = $_GET['name'];
    echo "नमस्ते, $name!";
} else {
    echo "कृपया नाम दर्ज करें।";
}
?> 
```

जब आप ब्राउज़र में `http://localhost/?name=अमित` चलाएंगे, तो आपको निम्नानुसार कुछ ऐसा दिखाई देना चाहिए। 

```
नमस्ते, अमित!
```

अगर आप इसमें `&age=30` भी जोड़ें तो आप नाम के साथ उम्र भी प्राप्त कर सकते हैं। 

```
नमस्ते, अमित! आपकी उम्र 30 साल है।
```

इसी तरह से आप POST अनुरोध के लिए भी कोड बना सकते हैं, जहां आप `$_POST` अर्गुमेंट का उपयोग कर सकते हैं।
