## क्यों
यदि आप PHP प्रोग्रामिंग करते हैं या सीखना चाहते हैं, तो आपको कमांड लाइन आर्ग्यूमेंट्स पढ़ने की जरूरत हो सकती है। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि कमांड लाइन आर्ग्यूमेंट्स क्यों महत्वपूर्ण हैं।

## कैसे करें
आइए अब हम PHP में कमांड लाइन आर्ग्यूमेंट्स पढ़ने का तरीका सीखें। नीचे दिए गए उदाहरण और सैम्पल आउटपुट के साथ "```PHP ... ```" कोड ब्लॉक द्वारा यह तरीका स्पष्ट रूप से समझाया गया है।

```PHP

// एक सादा कमांड लाइन आर्ग्यूमेंट सेट करें
$argument = $argv[1]; 

// यदि कमांड लाइन आर्ग्यूमेंट 10 है, तो सफलतापूर्वक प्रिंट करें
if($argument == 10) {
  echo "प्रवेश सफल";
} 

// अन्यथा, गलती संदेश प्रदर्शित करें
else {
  echo "अमान्य प्रवेश";
}
```

सैम्पल आउटपुट:
```
php commandline.php 10
प्रवेश सफल
```
```
php commandline.php 5
अमान्य प्रवेश
```

## डीप डाइव
कमांड लाइन आर्ग्यूमेंट्स को पढ़ने की यह एक अत्यंत महत्वपूर्ण कृत्य है। इसके विषय में अधिक समझने के लिए, आपको अंदरूनी काम के साथ कमांड लाइन आर्ग्यूमेंट्स को पढ़ना आवश्यक हो सकता है। इससे आपको अपने प्रोग्राम को और अधिक योग्य बनाने में मदद मिल सकती है।
