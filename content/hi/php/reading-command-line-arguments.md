## क्यों

कमांड लाइन आर्ग्यूमेंट पढ़ने का विषय थोड़ा अजीब लग सकता है, लेकिन यह एक बहुत ही महत्वपूर्ण कौशल है जो पिएचपी डेवलपर को सीखना चाहिए। इसका एक सबसे बड़ा फायदा है कि यह आपको अपने प्रोग्रामों को इंटरक्टिव बनाता है और उपयोगकर्ताओं से दिए गए डेटा का अनुरोध करने की सुविधा प्रदान करता है।

## कैसे करे

कमांड लाइन आर्ग्यूमेंट से जुड़े प्रश्नों का समाधान करने के लिए निम्नलिखित कोड ब्लॉक का उपयोग किया जा सकता है:

```PHP
<?php
// सामान्य आर्ग्यूमेंट 
echo "आपने दिए गए आर्ग्यूमेंट की संख्या: ". $argc ."\n";

// स्क्रिप्ट का नाम प्रिंट करें
echo "आपने दिए गए स्क्रिप्ट का नाम: ". $argv[0] ."\n";

// प्रथम आर्ग्यूमेंट का प्रिंट करें
echo "प्रथम आर्ग्यूमेंट: ". $argv[1] ."\n";

// दूसरा आर्ग्यूमेंट का प्रिंट करें
echo "दूसरा आर्ग्यूमेंट: ". $argv[2] ."\n";

// सभी आर्ग्यूमेंट्स को प्रिंट करें
echo "आपने दिए गए सभी आर्ग्यूमेंट: ";
print_r($argv);
?>
```

आउटपुट:

```
आपने दिए गए आर्ग्यूमेंट की संख्या: 3
आपने दिए गए स्क्रिप्ट का नाम: commandline.php
प्रथम आर्ग्यूमेंट: Hello
दूसरा आर्ग्यूमेंट: World
आपने दिए गए सभी आर्ग्यूमेंट: Array
(
    [0] => commandline.php
    [1] => Hello
    [2] => World
)
```
