## क्यों

अगर आप फिल होस्टेड या स्थायी फ़ाइल उपयोग करने के बजाय अस्थायी फ़ाइल उपयोग करते हैं तो यह आपके लिए कई लाभ हो सकते हैं। अस्थायी फ़ाइल बनाने के लिए आपको स्थायी फ़ाइल से ज़्यादा समय नहीं लगता है और यह आपके सिस्टम के स्थायी फ़ाइल तक पहुंच को कम करता है।

## कैसे करें

यदि आप फ़ाइल सिस्टम को PHP के माध्यम से एक्सेस करना चाहते हैं तो आपको अस्थायी फ़ाइल बनाने की आवश्यकता होगी। आप निम्नलिखित कोड को अपने प्रोग्राम में शामिल करके अस्थायी फ़ाइल तैयार कर सकते हैं:

```PHP 
<?php
$fileName = "myTempFile.txt";
$tempFile = fopen($fileName, "w");
if($tempFile){
    fwrite($tempFile, "Hello World!");
    fclose($tempFile);
    echo "सफलतापूर्वक अस्थायी फ़ाइल बनाई गई!";
}
else{
    echo "फ़ाइल बनाने में असफल!";
}
?>
```

जब आप यह कोड चलाएंगे, तो आपको "सफलतापूर्वक अस्थायी फ़ाइल बनाई गई!" के साथ अपनी अस्थायी फ़ाइल का संदेश मिलेगा। आप तब अपनी फ़ाइल सिस्टम में जाकर अस्थायी फ़ाइल को देख सकते हैं।
