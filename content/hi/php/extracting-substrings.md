## क्यों

आपने कभी सोचा है कि मांगलिक - समस्याओं जैसे कि टेक्स्ट स्ट्रिंग से आवश्यक जानकारियाँ निकालना हर PC काम करता है। बिना हिचक इस काम में व्यवहार करना काफी कठिन हो सकता है। 

## कैसे करे

```PHP
<?php
$testString = "हाय! में फिर से हूँ।";

// पूरे स्ट्रिंग को छोटे शब्दों में टुकड़ों में विभाजित करना
$splitString = explode(" ", $testString);
print_r($splitString);
/* इस कोड का आउटपुट होगा:
Array
(
    [0] => हाय!
    [1] => में
    [2] => फिर
    [3] => से
    [4] => हूँ।
)
*/
 
// स्ट्रिंग को एक निश्चित सीमा तक काटना
$substring = mb_substr($testString, 0, 4, "UTF-8");
echo $substring; // इसका आउटपुट होगा "हाय!"
?>
```

उपरोक्त कोड हमें एक स्ट्रिंग का छोटा-सा टुकड़ा मिला है जो हमें अपने काम के लिए आमतौर पर अधिक उपयोगी होता है। इससे आप किसी भी टेक्स्ट स्ट्रिंग से ज़रूरी डेटा निकाल सकते हैं। 

## डीप डाइव

अब आप प्रोग्रामिंग की थोड़ी गहराई में जा गए हैं, हम आपको बताएंगे कि आपको क्या-क्या फंक्शन्स का उपयोग करना चाहिए जो आप अलग-अलग स्थितियों के लिए सबसे अच्छे हैं। इससे आपको टेक्स्ट स्ट्रिंग को बेहतर ढंग से संशोधित करने में मदद मिलेगी।
