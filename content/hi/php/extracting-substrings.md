## क्यों

कभी-कभी हमारे पास एक बड़ा स्ट्रिंग होता है जिसमें हमें कुछ विशिष्ट जानकारी को खोजना होता है। ऐसे मामलों में हम स्ट्रिंग से उप-स्ट्रिंग निकालकर इस जानकारी को प्राप्त कर सकते हैं। PHP में उप-स्ट्रिंग निकालना भी बहुत आसान है और हम इसे अलग-अलग तरीकों से कर सकते हैं।

## कैसे करें

```PHP
$string = "मेरा नाम जॉन डो अपसन्ना है।";
// स्ट्रिंग से उप-स्ट्रिंग निकालने के लिए substr() फ़ंक्शन का उपयोग करें
$firstName = substr($string, 8, 4); // उप-स्ट्रिंग में से "जॉन" निकालें
$lastName = substr($string, 12, 3); // उप-स्ट्रिंग में से "डो" निकालें
echo $firstName; // जॉन
echo $lastName; // डो
```

यहां हमने स्ट्रिंग से उप-स्ट्रिंग निकालने के लिए `substr()` फ़ंक्शन का उपयोग किया है। इसमें तीन पैरामीटर होते हैं - स्ट्रिंग, इंडेक्स और लेंग्थ। स्ट्रिंग पैरामीटर में हमारी वांछित स्ट्रिंग दी जाती है, इंडेक्स पैरामीटर में हम उप-स्ट्रिंग की शुरुआत का इंडेक्स देते हैं और लेंग्थ पैरामीटर में हम उप-स्ट्रिंग की लंबाई देते हैं।
