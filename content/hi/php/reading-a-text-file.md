## क्या और क्यों?

टेक्स्ट फाइल पढ़ना क्या है और क्यों प्रोग्रामर इसे करते हैं? टेक्स्ट फाइल एक साधारण टेक्स्ट फाइल है जिसमें टेक्स्ट संपादित और संग्रहीत होता है। प्रोग्रामर इसे उन डेटा को संकलित करने के लिए पढ़ते हैं जो प्रोग्रामों को प्रोसेस करने या डेटा को लोड करने के लिए उपयोगी होता है। 

## कैसे करें:

```PHP
// टेक्स्ट फ़ाइल ओपन करें
$file = fopen("sample.txt", "r");

// पंक्तियों को पढ़ें
while(!feof($file)) {
  $line = fgets($file);
  echo $line . "<br>";
}

// फ़ाइल बंद करें
fclose($file);

// स्कीम पढ़ें
$file = file_get_contents("sample.txt");
echo $file;
```

आउटपुट:

```PHP
This is a sample text file.
It contains some lines.
And this is the last line.
This is the whole file in a variable.
```

## गहराई में जाएँ:

इस कार्य के इतिहास, विकल्प और टेक्स्ट फ़ाइल पढ़ने के विवरण जैसे मुद्दे भी हो सकते हैं। इससे पहले क्योंकि आईटी का संकल्पन नहीं था, उन डेटा को संकलित करने के लिए ट्रोली पंक्तियाँ छपाई की गईं। लेकिन आजकल कंप्यूटर अलग अलग प्रकार के डेटा का उत्पादन करते हैं, और टेक्स्ट फाइलें एक आसान और साधारण ढंग से उन डेटा को पढ़ने और लिखने के लिए हमेशा हाजिर होती हैं।

अन्य विकल्प शामिल हैं:
- ```fread()``` और ```fwrite()``` फ़ंक्शंस जो भीषणवादी काम करते हैं, लेकिन स्ट्रीमों को लिखने और पढ़ने के लिए एक बहुत व्यक्तिगत तरीका प्रदान करते हैं।
- ```spl_filesystem``` क्लास आप साधनों को विश्वसनीयता पाइप दिने विकल्प प्रदान करता है। इन अपेक्षाकृत समञ्जस्त सारणियाँ समस्या से सिद्ध हो सकते हैं।
- ```fgets()``` को पंक्ति द्वारा पढ़ता है। परंतु, ```fgetc()``` एक विशिष्ट चरित्र द्वारा अंश पढ़ता है।

अगर आप किसी अन्य भाषण या प्रोग्रामिंग भाषा का उपयोग करते हो, तो प्रणाली के लिए सूची में चयन जाए।
