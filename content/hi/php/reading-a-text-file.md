## क्यों

टेक्स्ट फ़ाइल पढ़ने का उद्देश्य हो सकता है कि आप किसी अन्य फ़ाइल से डेटा को पढ़कर अपने प्रोग्राम या वेबसाइट में उपयोग करना चाहते हैं।

## कैसे करें

टेक्स्ट फ़ाइल पढ़ने के लिए PHP कोड का उपयोग किया जा सकता है। नीचे दिए गए कोड ब्लॉक्स में आप यह देख सकते हैं कि कैसे टेक्स्ट फ़ाइल को खोलकर डेटा को पढ़ा जा सकता है और उसका उपयोग किया जा सकता है।

```PHP
// फ़ाइल खोलें
$file = fopen("data.txt", "r");

// फ़ाइल से डेटा पढ़ें
$data = fread($file, filesize("data.txt"));

// डेटा प्रिंट करें
echo $data;

// फ़ाइल बंद करें
fclose($file);
```

उपरोक्त उदाहरण में, हमने `fopen()` फ़ंक्शन का उपयोग करके फ़ाइल को खोला और `fread()` फ़ंक्शन का उपयोग करके उससे डेटा पढ़ा। फ़ाइल को काम के बाद हमने `fclose()` फ़ंक्शन से बंद किया। आप अपनी जगही फ़ाइल का नाम और उचित हैंडलर उपयोग कर सकते हैं।

## गहराई में

टेक्स्ट फ़ाइल पढ़ने का सबसे सामान्य और उपयोगी उपयोग अन्य फ़ाइलों से डेटा को पढ़ना है। यह आपको एक अधिक मिलजुल डेटा संग्रह अपने प्रोग्राम या वेबसाइट को उपयोग करने में मदद कर सकता है। आप टेक्स्ट फ़ाइल को पढ़कर डेटा को संरक्षित भी कर सकते हैं, जिससे आप इसे बाद में भी प्रयोग कर सकते हैं।
