## क्यों

लोग PHP प्रोग्रामिंग सीखने के साथ-साथ अपने कोड को बेहतर बनाने के लिए टेक्स्ट फ़ाइल पढ़ना चाहते हैं। इससे वे अपने कोड और डेटा को और सुविधाजनक तरीके से प्रबंधित कर सकते हैं। 

## कैसे करें

टेक्स्ट फाइल रीड करने के लिए PHP में fopen() फंक्शन का उपयोग किया जाता है। साथ ही, fgets() फंक्शन भी इसके लिए उपयोगी है जो लाइन वाइज़ रीड करता है। कोड ब्लॉक में नीचे दिए गए उदाहरण का पालन करें।

```PHP
$file = fopen("sample.txt", "r") or die("Unable to open file!");
// read file line by line
while(!feof($file)) {
  echo fgets($file) . "<br>";
}
fclose($file);
```

उपरोक्त कोड द्वारा टेक्स्ट फाइल का प्रत्येक लाइन पढ़ा जा सकता है। अगर आप चाहें तो लाइन वाइज़ रीडिंग के स्थान पर फाइल को पूरे आकार में रीड कर सकते हैं। साथ ही, fopen() फंक्शन के द्वारा फाइल के साथ काम करने के बाद उसे बंद भी कर दें। उपरोक्त कोड में दिए गए कंडीशनों को अपनी आवश्यकतानुसार बदल सकते हैं। 

## डीप डाइव

टेक्स्ट फ़ाइल पढ़ना PHP में बहुत ही आसान है। इसमें आप संख्यावाची और स्रोत को समायोजित कर सकते हैं ताकि आपका कोड और संग्रह हमेशा स्पष्ट और सुविधाजनक रहे। यह आपके कोड को साफ-सुथरा बनाए रखने में मदद करेगा और समस्याओं को जल्दी से समाधान करने में भी सहायता करेगा।
