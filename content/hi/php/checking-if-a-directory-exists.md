## क्या है और क्यों?  

डायरेक्टरी यदि मौजूद होती है या नहीं को जांचने का मतलब है कि एक निश्चित स्थान पर किसी विशेष नाम की डायरेक्टरी कहाँ से मिलेगी या नहीं। इसके लिए प्रोग्रामर जाँच करते हैं क्योंकि यह सुनिश्चित करने के लिए आवश्यक होता है कि एक फ़ाइल या संसाधन सही स्थान पर स्थित है या नहीं।

## कैसे करें:  

डायरेक्टरी की मौजूदगी जांचने के लिए, आपको ```file_exists``` फ़ंक्शन को उपयोग करना होगा। यह फ़ंक्शन एक फ़ाइल या डायरेक्टरी के नाम को लेता है और यदि वह मौजूद होता है, तो ```true``` को वापस देता है। निम्नलिखित है एक सरल उदाहरण: 

```PHP
<?php
$file = 'example.txt';

if(file_exists($file)) {
    echo "The file " . $file . " exists.";
} else {
    echo "The file " . $file . " does not exist.";
}
?>
```
उपरोक्त कोड का नतीजा निम्न हो सकता है:  
```
The file example.txt exists.
```

## गहराई में जाएं:  

प्राचीन लिनक्स सिस्टमों पर, डायरेक्टरी की मौजूदगी को जांचने के लिए ```dir``` या ```opendir``` को उपयोग किया जाता था। हालांकि, आजकल ```file_exists``` फ़ंक्शन को सर्वोत्तम माना जाता है क्योंकि यह सुनिश्चित करने में अधिक आसान है कि किसी फ़ाइल या डायरेक्टरी का नाम सही है या नहीं। अन्य विकल्प शामिल हमें ```is_dir``` और ```scandir``` फ़ंक्शन हैं।
