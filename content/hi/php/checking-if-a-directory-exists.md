## क्यों

किसी डायरेक्टरी का अस्तित्व जांचने में लोगों को क्यों रुचि हो सकती है, उसका कारण यह है कि वे सुनिश्चित करना चाहते हैं कि उनके कोड में वे निर्दिष्ट डायरेक्टरी के साथ संबंधित फ़ाइलों को आसानी से एकत्र कर सकें।

## कैसे करें

```PHP
if (is_dir($directory)) { // अगर डायरेक्टरी मौजूद है
  echo "यह डायरेक्टरी मौजूद है।";
} else { // अगर डायरेक्टरी मौजूद नहीं है
  echo "यह डायरेक्टरी मौजूद नहीं है।";
}
```

उपरोक्त कोड स्निपेट में, ```is_dir()``` फ़ंक्शन डायरेक्टरी का अस्तित्व जांचने के लिए प्रयोग किया गया है। यदि डायरेक्टरी मौजूद है, तो पहला कोड ब्लॉक चलेगा और संदेश "यह डायरेक्टरी मौजूद है।" प्रिंट किया जाएगा। और यदि डायरेक्टरी मौजूद नहीं है, तो दूसरा कोड ब्लॉक चलेगा और संदेश "यह डायरेक्टरी मौजूद नहीं है।" प्रिंट किया जाएगा।
