Hindi में PHP कार्यक्रमनवेश ब्लॉग पोस्ट

# क्यों

आमतौर पर हम सभी अपने के प्रोग्रामिंग कोड में सुरक्षित और स्थानिक फ़ाइल्स को संग्रहीत करना चाहते हैं। इसलिए एक अहम सवाल होता है, क्या एक डायरेक्टरी होने के बावजूद हम इसके मौजूदा होने की जांच कर सकते हैं? निःसंदेह हां, हम PHP का उपयोग करके इसके मौजूदा होने की जांच कर सकते हैं।

# कैसे 

``` PHP
<?php
if (is_dir('directory_name')) {
  echo 'डायरेक्टरी मौजूद है';
} else {
  echo 'डायरेक्टरी मौजूद नहीं है';
}
?>
```

जोड़ने से पहले, हम `is_dir()` फ़ंक्शन उपयोग करके कॉल अपने डायरेक्टरी मौजूदा होने की जांच कर रहे हैं। यदि वह मौजूद है, तो हम उसे बताते हैं कि और इससे उसे बताते हैं कि डायरेक्टरी मौजूद नहीं है।

# गहरी जांच

अब हमारी PHP कोड के साथ सबसे ज्यादा उपयोग किए जाने वाले फ़ंक्शन `is_dir()` है। यह फ़ंक्शन डायरेक्टरी मौजूद होने की जांच करता है और केवल डायरेक्टरी के लिए ही काम करता है। लेकिन यदि आपको फ़ाइल के लिए भी मौजूद होने की जांच करनी होती है, तो आप `file_exists()` फ़ंक्शन का उपयोग कर सकते हैं। यह फ़ंक्शन फ़ाइल और डायरेक्टरी सभी के लिए काम करता है।
