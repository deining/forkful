हेलो दोस्तों! आज हम प्रोग्रामिंग की दुनिया में बहुत ही जरूरी चीजों में से एक पर चर्चा करेंगे - टेस्टिंग। क्या आप जानते हैं की क्यों टेस्टिंग सभी के लिए जरूरी है? आइए हम इस पोस्ट के माध्यम से पता करें की क्यों और कैसे हम टेस्टिंग को लिख सकते हैं।

## क्यों

टेस्टिंग एक वह चरण है जो सभी प्रोग्रामर्स को उनके कोड को चेक करने के लिए करना चाहिए। टेस्टिंग के लिए टेस्ट केस बनाकर हम अपने कोड में बग्स और गड़बड़ियों को पहचान सकते हैं ताकि हमारा कोड सही और सुरक्षित हो सके। इससे हमारे कोड की quality होती है और दूसरों को भी हमारा कोड इस्तेमाल करने में आसानी होती है।

## कैसे

टेस्ट केस बनाने के लिए हमें  अपने कोड के सभी अहम हिस्सों को निश्चित करना होता है। एक बार जब हमारे पास ये अहम हिस्से हो जाते हैं तो हम अपने कोड में टेस्ट फंक्शन बना सकते हैं और उन टेस्ट फंक्शन को क्रम में चला कर देख सकते हैं। नीचे दिए गए उदाहरण में मैंने टेस्ट फंक्शन को प्रोग्रामिंग करके उसका आउटपुट दिखाया है।

```PHP
// टेस्ट फंक्शन कोड
function multiply($a, $b){
  return $a * $b;
}

// टेस्ट केस बनाना
echo multiply(2,3);

// आउटपुट: 6 
```

ऊपर कोड ब्लॉक में हमने एक multiply नाम का टेस्ट फंक्शन बनाया है जो की दो अंकों को multiply करता है और उसका आउ
