## क्यों

जब हम आराम से अपनी प्रोग्रामिंग स्किल्स को मजबूत और सुरक्षित बनाना चाहते हैं तो हमें टेस्ट लिखना अत्यंत महत्वपूर्ण बन जाता है। टेस्ट लिखने से हमारी कोड में आधारभूत गतिविधियों का पता चलता है और इससे हमारा कोड परिणामस्वरूप ठीक से काम करता है।

## कैसे करें

आइए हम एक छोटे से PHP कोड के माध्यम से समझते हैं कि कैसे हम टेस्ट लिख सकता हैं।

```PHP
<?php
    
    $num1 = 10;
    $num2 = 20;
    
    // नंबर जोड़े
    $sum = $num1 + $num2;
    
    // $sum को 30 से तुलना करें
    if($sum == 30){
        echo "समय हो गया है कि आप अपने नंबर आकड़ों को जोड़ें हुए चेक करें!";
    } else {
        echo "माफ़ कीजिये। आपके कोड में कुछ ग़लती है।";
    }
```

इस उदाहरण में, हमने दो नंबर जुड़ाए और उनकी गणना की। फिर हमने उस गणना का पता लगाने के लिए इकाइयों को दूसरे नंबर के साथ मिलाया। आप देख सकते हैं कि हमने `if` शर्त का उपयोग किया है जो हमें यह बताता है कि क्या हमारी गणना सही है या नहीं। इस उदाहरण में, हमारा कोड ठीक से काम कर रहा है इसलिए हमें ऊपर दिए गए संदेश मिलेगा। आप स्वयं इस कोड को चला कर टेस्ट कर सकते हैं।
