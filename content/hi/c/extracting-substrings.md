## क्यों
सबसे पहले हम अपने पढ़ दालों को ऐसी तरह से नहीं रख पाते हैं कि हम उसे पूरी तरह से समझ सकें। इसलिए, हमें अपने डेटा को छोटे तुकड़ों में विभाजित करने की आवश्यकता होती है, जो कि उसे पढ़ना और समझने को आसान बनाते हैं। इसी कारण से, हम substring निकालने की क्रिया में लगे हैं।

## कैसे करें
यह एक तरह से स्ट्रिंग्स से डेटा को विभाजित करने का सबसे सरल तरीका है। हमारे कॉड के विभिन्न हिस्सों में हम substring निकाल सकते हैं और इसमें से कुछ मुख्य कारक हैं:
आगे बढ़ते हुए हम एक सरल उदाहरण देंगे:

```C
char str[] = "हिंदी भाषा में प्रोग्रामिंग"
char substring[10];
int start = 7, length = 4;

// substring निकालने के लिए लूप चलाएं
for (int i = start; i < start + length; i++){
    substring[i - start] = str[i]; // substring में दिए गए इंडेक्स पर अक्षरों को कॉपी करें
}

// substring को प्रिंट करें
printf("substring: %s", substring);

```

उपरोक्त कोड का आउटपुट निम्नानुसार होगा:

```C
substring: भाषा
```
