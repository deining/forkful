## क्यों

क्या आपने कभी प्रोग्रामिंग में यह समस्या देखी है कि आपके प्रोग्राम के डेटा को आप हमेशा तक स्टोर नहीं कर सकते हैं? या यह डेटा आपको नए फॉर्मेट में इम्पोर्ट करना होता हैं? इस समस्या को सुलझाने के लिए, हमें टेक्स्ट फाइल लिखने की जरूरत होती है. टेक्स्ट फाइल एक वस्तु का संग्रह है जो नए लाइनों में टेक्स्ट को स्टोर करता है, जो प्रोग्रामिंग में आसानी से एक्सेस किया जा सकता है.

## कैसे करे

टेक्स्ट फाइल लिखने के लिए, हमें C का उपयोग करना होगा. सबसे पहले, हमें एक फाइल को ओपन करना होगा जिसमें हम अपना टेक्स्ट लिखेंगे. उसके बाद, हम "fopen" फंक्शन का उपयोग करके फाइल को ओपन करेंगे. तब हम "fprintf" फंक्शन का उपयोग करके अपने टेक्स्ट को फाइल में लिखेंगे. अंत में, हम फाइल को "fclose" फंक्शन का उपयोग करके क्लोज करेंगे. नीचे दिए गए कोड ब्लॉक में आप एक सरल उदाहरण देख सकते हैं:

```C
#include <stdio.h>
int main(void) {
    FILE *fp = fopen("sample.txt", "w");
    
    if (fp == NULL) {
        printf("Error opening file!");
        return 1;
    }
    
    fprintf(fp, "Hello, world!\n");
    fprintf(fp, "This is a sample text file.");
    
    fclose(fp);
    return 0;
}
```

यह उदाहरण प्रोग्राम "sample.txt" फाइल में दो लाइनों को लिखेगा. पहले लाइन में "Hello, world!" और दूसरे में "This is a sample text file." अब आप sample.txt फाइल को ओपन करके इन दोनों लाइनों को देख सकते हैं.
