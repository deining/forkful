# क्यों

टेक्स्ट फ़ाइल लिखना आजकल कंप्यूटर साइंस में एक महत्वपूर्ण कौशल है। यह आपको प्रोग्रामिंग और डेटा प्रोसेसिंग के लिए शामिल किया जाता है और आपको अपने सिस्टम पर यात्रा करने और डेटा को संगठित रूप से रखने की अनुमति देता है।

## कैसे करें

यदि आप C प्रोग्रामिंग में नए हैं तो आपको पूर्व ज्ञान का होना जरूरी है। यदि आपने पहले से ही किसी अन्य प्रोग्रामिंग भाषा का उपयोग किया है, तो यह आसान हो सकता है। हालांकि, अगर आप शुरुआत करने के लिए तैयार हैं, तो यह आपके लिए थोड़ा मुश्किल हो सकता है। यहां एक उदाहरण है जो आपको कैसे एक टेक्स्ट फ़ाइल लिखने में मदद करेगा:

```C
#include <stdio.h>

int main() {
    // file pointer
    FILE *fp;

    // open file in write mode
    fp = fopen("my_file.txt", "w");

    // write to the file
    fprintf(fp, "Hello World!");

    // close the file
    fclose(fp);

    return 0;
}
```

इस उदाहरण में, हमने स्टैंडर्ड इनपुट और आउटपुट पुस्तकालय का इस्तेमाल करके फ़ाइल को खोला और उसमें एक सदस्य लेखा जाता है। आप फ़ाइल के नाम और लेखन मोड का उपयोग करके आसानी से अपनी आवश्यकतानुसार फ़ाइल को खोल सकते हैं।
