## क्यों

अक्सर हमें अपने कोड में स्ट्रिंग को कैपिटलाइज करने की आवश्यकता होती है। यह स्ट्रिंग को उसके अस्पष्ट रूप से बढ़ा सकता है और इससे पढ़ना और लिखना आसान हो जाता है।

## कैसे करें

```C
#include <stdio.h>

// फ़ंक्शन जो स्ट्रिंग को कैपिटलाइज करती है
void capitalize(char str[]) {
    int i = 0;

    // जब तक कि स्ट्रिंग के अंत तक न पहुंच जाए
    while (str[i] != '\0') {
        // अगर वर्तमान चर सामान्य अक्षर है
        if (str[i] >= 'a' && str[i] <= 'z') {
            // स्ट्रिंग को कैपिटल में परिवर्तित कीजिए
            str[i] = str[i] - 32;
        }
        i++;
    }
}

int main() {
    // स्ट्रिंग जो कैपिटलाइज होना है
    char str[] = "hello world";

    // स्ट्रिंग को कैपिटलाइज करें
    capitalize(str);

    // परिणाम प्रिंट करें
    printf("%s", str); // "HELLO WORLD"

    return 0;
}
```

## गहराईगिरी

स्ट्रिंग को कैपिटलाइज करने के लिए हमें प्रत्येक चर को निर्दिष्ट अंतराल पर ऊपरी मामले में असीमित मामले में बदलना होता है। यह कार्य आवर्त को पूरा करने के लिए जोड़ित गणित ऑपरेटर और ऐडर ऑपरेशन को शामिल करता है। उदाहरण के लिए, सामान्य अक्षर 'a' का ASCII मान 97 है और यदि हमें उसे ऊपरी मामले में बदलना होता है, हमें उससे 32 घटाना होगा, जो उसे उसके ASCII मान 65 बनाएगा, जो हमारे ऊपरी मामले की सीमा है। इसी तरह, हम सभी सामान्य अक्षरों को कैपिटल में परिवर्तित कर सकते हैं।
