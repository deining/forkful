## क्यों

कोड को साफ और उपयोगी बनाने के लिए, जिसमें स्ट्रिंग के प्रथम अक्षर को बड़ा लेटर में बदला जाए।

## कैसे

```C
#include<stdio.h>
#include<string.h>

int main(){
    // एक स्ट्रिंग जो बड़ा किया जाना है
    char str[100];
    printf("कृपया स्ट्रिंग दर्ज करें: ");
    // स्ट्रिंग प्राप्त करें
    gets(str);
    // स्ट्रिंग में सभी अक्षरों को बड़ा लेटर में बदलें
    for(int i=0;i<strlen(str);i++){
        // प्रत्येक अक्षर को बड़ा लेटर में बदलने के लिए -32 करें
        str[i] = str[i] - 32;
    }
    // परिणाम प्रदर्शित करें
    printf("परिणाम: %s", str);
    return 0;
}
```

उदाहरण के लिए, यदि हम प्रोग्राम में स्ट्रिंग "hello world" दर्ज करते हैं, तो आउटपुट "HELLO WORLD" होगा। सभी अक्षरों को कैपिटल में बदलने के लिए हमने एक लूप का उपयोग किया है जो स्ट्रिंग के हर अक्षर को बड़ा लेटर में बदलता है।

## गहराई में जाएं

एक स्ट्रिंग को बड़ा करने के लिए, हम प्रोग्राम में अक्षरों को कैपिटल में बदलते हैं। सबसे पहले, हम स्ट्रिंग को प्राप्त करते हैं और फिर हर अक्षर को बड़ा लेटर में बदलने के लिए एक लूप का उपयोग करते हैं। स्ट्रिंग में हर अक्षर को एक्सेस करने के लिए हम इंडेक्स का उपयोग करते हैं। स्ट्रिंग की लंबाई को जानने के लिए हम `strlen()` फंक्शन का उपयोग करते हैं।
