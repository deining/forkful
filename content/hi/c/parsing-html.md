## क्या और क्यों?
प्रोग्रामर्स क्या होते हैं? यह एक प्रश्न है जो जवाब ढूंढने की कोशिश करते हैं। वे आपकी पसंद की कृत्रिम भाषा में लिखी गई विशाल साइटों से तरह तरह के डेटा को पढ़कर उसे मानचित्र बनाने में मदद करते हैं। यह प्रक्रिया कुछ लोग वेब साइटों का डेटा प्राप्त करने के लिए इस्तेमाल करते हैं जो लाइब्रेरी द्वारा उपलब्ध हो सकता है।

## कैसे करे?
हम प्रोग्राम बनाना शुरू करने से पहले सभी आवश्यक टूल्स और साधनों को तैयार कर लेते हैं। एक आसान तरीका है C प्रोग्रामिंग भाषा का उपयोग करना। पहले हम एक रचना या कार्य तैयार करते हैं जो उस ऑब्जेक्ट एड्रेस को विशाल साइटों से ढूंढता है और उसे उदाहरण के लिए हम यह दिखाएंगे कि कैसे यह काम करता है।

```C
// सभी आवश्यक लाइब्रेरी फाइल को शामिल करें
#include <stdio.h>
#include <stdlib.h>

void main() {
  // एक बचाव करता है कि उस ऑब्जेक्ट को अर्जित करें जो आप खोज रहे हैं
  char *object = "<div id='site-content'>This is a sample website</div>";
  char *address;

  // उदाहरण को खोजें
  address = strstr(object, "<div id='site-content'>");
  // उस िसे लाखों का जोड़ा करें
  if (address != NULL) {
    // प्रिंट करें
    printf("%s", address);
  }

  return 0;
}
```

आउटपुट: ```<div id='site-content'>This is a sample website</div>```

## गहराई में जाएं
HTML पार्सिंग के लिए पहले यह HTML पीढ़ी मे आना हमें उन सभी लेखों के मूल्यों को ढूंढ़ने या उदाहरणात्मक वाक्यांशों को खोजने की आवश्यकता है जिन्हें आमतौर पर एल्गोरिदमों द्वारा उपयोग किया जाता है। आप C प्रोग्रामिंग का उपयोग कर सकते हैं क्योंकि इसका कोड अत्यंत संक्षिप्त और सरल होता है। यह शीर्ष समस्याओं में से एक नहीं है जिसका समाधान करने के लिए हम C प्रोग्रामिंग का उपयोग कर सकते हैं। पार्सिंग की एक अन्य प्रक्रिया है DOM (Document Object Model) को उत्पादित करना। इसमें डेटा को हीरोन, कार्यवाही आदि विशिष्ट संरचना में संरचित किया गया होता है।
