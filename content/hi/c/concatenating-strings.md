## क्यों?

आपने कभी सोचा है कि हम प्रोग्रामिंग में दो या अधिक स्ट्रिंग्स को कैसे जोड़ सकते हैं? या क्यों जरूरत पड़ती है स्ट्रिंग्स को जोड़ने की? इस ब्लॉग पोस्ट में हम आपको स्ट्रिंग्स को जोड़ने के बारे में विस्तार से बताएंगे ताकि आप यह काम आसानी से कर सकें।

## कैसे?

```C
#include<stdio.h>
#include<string.h>

int main()
{
   // प्रथम स्ट्रिंग स्टोर करें
   char str1[50] = "मेरा ";
   // दूसरी स्ट्रिंग स्टोर करें
   char str2[50] = "नाम ";
   // दूसरी स्ट्रिंग को प्रथम स्ट्रिंग में जोड़ें
   strcat(str1, str2);
   // जोड़ी हुई स्ट्रिंग प्रिंट करें
   printf("%s", str1);
   return 0;
}
```

आपको दो स्ट्रिंग्स को `strcat()` फ़ंक्शन के माध्यम से जोड़ना होगा। यह फ़ंक्शन `<string.h>` हैडर फ़ाइल में प्रदर्शित होता है और यह दो स्ट्रिंग्स को प्रथम स्ट्रिंग में जोड़ता है। इस उदाहरण में हमने "मेरा" और "नाम" को जोड़कर "मेरा नाम" नई स्ट्रिंग बनाई है।

आप भी `strcat()` के साथ `strncat()` का उपयोग कर सकते हैं जो निर्दिष्ट नंबर की अक्षरों को ही जोड़ता है।

## गहराईगंजनी

स्ट्रिंग्स को जोड़ना अकेले में स्वाभाविक नहीं है। लेकिन यह कई स्थितियों में उपयोगी होता है। उदाहरण के लिए, आप दो स्ट्रिंग्स को जोड़कर नई स्ट्रिंग बना सकते हैं जो बाद में अन्य स्ट्रिंग्स के साथ इस्तेमाल किया जा सकता है।
