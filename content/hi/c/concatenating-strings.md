## क्यों
कुछ लोगों के लिए स्ट्रिंग को जोड़ना एक साधन है जो प्रोग्रामिंग काम को सरल बनाता है।

## कैसे करें
जब हमारे पास अलग-अलग प्रकार की स्ट्रिंग हो, तो हम उन्हें जोड़ने के लिए ```strcat()``` फ़ंक्शन का उपयोग कर सकते हैं। इसे निम्नलिखित प्रकार से उपयोग किया जा सकता है:

```C
// स्ट्रिंग जोड़ने का उदाहरण
#include <stdio.h>
#include <string.h>

int main()
{
    char string[100] = "हल्लो ";
    char name[50] = "दोस्त";
    
    strcat(string, name);
    
    printf("%s", string);
    
    return 0;
}

// आउटपुट: हल्लो दोस्त
```

## गहराई में जाएं
```strcat()``` फ़ंक्शन का उपयोग करने से पहले, हमें सुनिश्चित करना होगा कि पहली स्ट्रिंग का पर्याप्त स्थान हो जिसमें हम दूसरी स्ट्रिंग जोड़ना चाहते हैं। इसके अलावा, ```strcat()``` हमें वास्तव में यदि पहली स्ट्रिंग का अंतिम भाग निर्दिष्ट नहीं किया जाता है, तो स्ट्रिंग के अंत में एक एकल भराने वाले कैरेक्टर के साथ स्ट्रिंग जोड़ जाता है। इसलिए, हमें सुनिश्चित करना होगा कि पहली स्ट्रिंग का अंतिम भाग निर्दिष्ट नहीं है।
