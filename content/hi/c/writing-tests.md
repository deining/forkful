## इसलिए

प्रोग्रामिंग दुनिया में, टेस्टिंग एक अत्यंत महत्वपूर्ण कार्य है। यह कम समय तो बचाता है ही, बल्कि समस्याओं को पहचानने और दूर करने में भी मदद करता है। एक अच्छी टेस्ट सुनिश्चित करता है कि आपका कोड काम कर रहा है या नहीं, जिससे आप सुरक्षित और विश्वसनीय प्रोग्राम बना सकते हैं।

## कैसे करे

टेस्टिंग कोड को लिखना आसान है। सबसे पहले, हमें एक फ़ंक्शन बनाना होगा जो टेस्ट करेगा कि आपका कोड ठीक काम कर रहा है या नहीं। यहां हम एक उदाहरण देखेंगे जहां हम दो संख्याओं को जोड़ते हैं:

```C
#include <stdio.h>

// दो संख्याओं को जोड़ने वाला फंक्शन
int add(int num1, int num2) {
    return num1 + num2;
}

// टेस्ट करने के लिए हम आउटपुट मॉनिटर पर चाहते हैं कि 10 + 15 = 25 हो।
int main() {
    int sum = add(10, 15);

    printf("10 + 15 = %d\n", sum);

    return 0;
}
```

जैसा कि हमें उम्दा है, आउटपुट मॉनिटर पर 10 + 15 = 25 प्रिंट होता है। हमारा कोड काम कर रहा है!
