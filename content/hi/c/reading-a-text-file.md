## क्या और क्यों?

एक टेक्स्ट फ़ाइल को पढ़ना क्या है, और क्यों प्रोग्रामर्स इसे करते हैं? एक टेक्स्ट फ़ाइल एक प्रकार की फ़ाइल होती है जो साधारण टेक्स्ट का संग्रह होता है, जैसे कि पाठ, नोट्स या कोड। प्रोग्रामर्स विभिन्न डेटा प्रसंस्करण या लेखन कार्यक्रमों में इस्तेमाल करते हैं, जिससे वे डेटा को संरचित कर सकें और अन्य टेक्स्ट फ़ाइलों से डेटा को पढ़ सकें।

## कैसे करें?

```C
#include <stdio.h>
int main() 
{
    char str[100];
    FILE *fp;
    fp = fopen("test.txt", "r"); // "test.txt" को आपकी फ़ाइल का नाम सुनिश्चित करे
    while (fgets(str, 100, fp) != NULL) 
        printf("%s", str);
    fclose(fp);
    return 0;
}
```

जब आप ऊपर दिए गए कोड को कंपाइल करेंगे, तो आपकी टेक्स्ट फ़ाइल (`test.txt` को आपके द्वारा उपयोग किया जा सकता है) से पाठ को एक स्ट्रिंग में पढ़ाई जाएगी और उसे प्रिंट किया जाएगा। आप किसी भी अन्य तरीके से भी अपनी फ़ाइल को पढ़ सकते हैं, जैसे `getc()` अस्थित्व में है।

## गहराई में जाएं

इतिहास के संदर्भ में, पहले संगणकों में प्रोग्राम लिखने के लिए पूर्वावलोकन के लिए टेक्स्ट फ़ाइलों का उपयोग किया जाता था। आज भी, बहुत से रचनात्मक प्रोग्राम पत्रों और प्रोग्रामों में टेक्स्ट फ़ाइलों के उपयोग का इस्तेमाल किया जाता है। अतिरिक्त विकल्पों के रूप में, आप बाइनरी फ़ाइलें भी पढ़ सकते हैं जिन्हें `fread()` के आवेदन के साथ पढ़ा जाता है। आप अपने पूर्वावलोकन में, अपनी फाइल संरचना और श्रृंखला मानचित्रण भी देख सकते हैं।
