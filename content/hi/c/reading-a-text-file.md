## क्यों

क्या आपने कभी सोचा है कि आपके कंप्यूटर पर संग्रहीत टेक्स्ट फाइलों को कैसे पढ़ा जाता है? C प्रोग्रामिंग भाषा एक उपयोगी तरीका है टेक्स्ट फ़ाइलों को पढ़ने के लिए और उनमें से डेटा को प्रोसेस करने के लिए। इस लेख में, हम आपको सीखाएंगे कि C में टेक्स्ट फ़ाइल को कैसे पढ़ा जाता है।

## कैसे करें

```C
#include <stdio.h>
int main()
{
   FILE *fp;
   char c;

   fp = fopen("sample.txt", "r"); // फ़ाइल खोलें

   if (fp == NULL)
   {
      printf("फ़ाइल खोलने में ग़लती हुई।");
      return 1;
   }

   // फ़ाइल के अंत तक चलें
   while ((c = getc(fp)) != EOF) 
      printf("%c", c);

   // फ़ाइल बन्द करें
   fclose(fp);

   return 0;
}
```

जैसा कि आप देख सकते हैं, हमने स्टैंडर्ड लाइब्रेरी का ```stdio.h``` इस्तेमाल किया है जो हमें स्क्रीन पर डेटा प्रिंट करने और फाइल्स को खोलने आदि के लिए एक बहुत ही उपयोगी ```printf``` फ़ंक्शन प्रदान करता है। अपने कोड में, हमने एक फ़ाइल पॉइंटर ```fp``` बनाया है जो हमें फ़ाइल को खोलने और डेटा को पढ़ने की अनुमति देता है। फ़ाइल को खोलने के लिए हम उसकी नाम और मोड देते हैं, जहां ```"r"``` रीड मोड का कोड है। अगर फ़ाइल सफलतापूर्वक खोलती है, तो हम फ़ाइल के अंत तक चलते हैं, हर एक चरित्र को प्रिंट करते हैं और फिर उसे बंद करते हैं।
