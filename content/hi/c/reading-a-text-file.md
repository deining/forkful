## क्यों

अधिकांश सॉफ्टवेयर अन्य फ़ाइल फॉर्मेटों के साथ तुलना में, पाठ फाइलें सबसे साधारण और समझ में आसान होती हैं। इसलिए, सी प्रोग्रामर्स को किसी भी प्रकार के डेटा को पढ़ने और प्रसंस्करण करने के लिए पाठ फाइलों का उपयोग करना आसान होता है। इस ब्लॉग पोस्ट में, हम आपको क्यों सी प्रोग्रामिंग में पाठ फाइलों को पढ़ना महत्वपूर्ण है इसके बारे में बताएंगे। 

## कैसे

आमतौर पर, पाठ फाइलों को सी प्रोग्रामिंग में पढ़ने के लिए श्रेणी के पुराने फ़ंक्शन `fopen ()` और `fgets ()` का उपयोग किया जाता है। `fopen ()` फ़ंक्शन, जो फाइल उठाने के लिए उपयोग किया जाता है, एक `FILE` नक़ल को उत्पन्न करती है। इसके बाद, `fgets ()` फ़ंक्शन को फ़ाइल नक़ल और एक बड़े आकार के बफ़र को साथ में दिया जाता है ताकि वह फ़ाइल से पाठ पढ़ सके। 

```C
FILE *fp;
char buffer[1000];
fp = fopen("file.txt", "r");

if(fp != NULL) {
    while(fgets(buffer, 1000, fp) != NULL) {
        // code to process each line of the file
    }
    
    fclose(fp);
}
```

यहां `file.txt` आपके पाठ फ़ाइल का नाम होगा, जिसे आप पढ़ना चाहते हैं। फ़ंक्शन `fgets ()` से पाठ पढ़ने के बाद, आप चाहे तो हर पंक्ति को प्रसंस्करण करने के लिए `buffer` से उत्पादित स्ट्रिंग का उपयोग कर सकते हैं।
