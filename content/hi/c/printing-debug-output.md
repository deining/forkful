## क्यों
Debug आउटपुट प्रिंट करने में कोई सहायता क्यों लेगा, इसके पीछे का कारण बताया जाएगा।

Debug आउटपुट एक बहुत ही महत्वपूर्ण टूल है जो कोडिंग में समस्याओं को खोजने में मदद करता है। प्रोग्राम को चलाते हुए, हम समस्या के स्थान को पता लगाएं और उसे ठीक कर के बारे में पता लगाएंगे। इससे हमें यह समझने में मदद मिलती है कि कैसे हमारा कोड चल रहा है और कहीं कोई गलती तो नहीं हो रही है।

## कैसे करें
चाहे हम नए हों या एक्सपर्ट, हमें अपने कोड में बगों को खोजने के लिए debug आउटपुट का इस्तेमाल करना चाहिए। इस लेख में, हम आपको बताएंगे कि कैसे आप ```printf()``` और ```fprintf()``` फंक्शंन का प्रयोग करके अपने कोड में आउटपुट प्रिंट कर सकते हैं।

यहा पर एक साधारण उदाहरण है:

```C
#include <stdio.h>

int main()
{
    int num1 = 10;
    int num2 = 20;

    // debug स्टेटमेंट प्रिंट करें
    printf("num1 का मान है: %d\n", num1);
    printf("num2 का मान है: %d\n", num2);

    return 0;
}
```

आप इस उदाहरण को कोड के अपने हिस्से के साथ और अपने कोड में जहां आपको डिबग आउटपुट चाहिए उस स्थान पर इस्तेमाल कर सकते हैं। उपरोक्त उदाहरण में, हमने दो मानों के बीच अन्तर को देखने के लिए debug स्टेटमेंट को छोड़ा है। आप अपने आवश्यकतानुसार इसे अपने कोड में उपयोग कर सकते हैं।
