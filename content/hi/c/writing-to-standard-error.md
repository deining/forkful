## क्यों

यह प्रश्न उठता है कि हम को प्रोग्रामिंग में इरर स्टैण्डर्ड लेखन करने का क्या मिलता है? आपको यह जानने दिया जाए कि यह आपको क्यों चाहिए।

## कैसे करें

कोडिंग उदाहरण और नमूना आउटपुट के साथ "```C ... ```" कोड ब्लॉक्स में दिखाया गया है।

```C
#include <stdio.h>

int main()
{
    fprintf(stderr, "यह इरर स्टैण्डर्ड में लिखा हुआ है!\n");
    return 0;
}
```

उपरोक्त कोड ब्लॉक में हमने लॉग फंक्शन का उपयोग करके इरर स्टैण्डर्ड में पाठ लिखा है। प्रोग्राम को कम्पाइल और चलाने पर हम निम्न आउटपुट प्राप्त करेंगे:

```
यह अभेद्ड छपा दिया है!
```

उपरोक्त उदाहरण में, हमने `fprintf` फंक्शन का उपयोग किया है जो लॉग करने के लिए बनाया गया है। यह फंक्शन दो पैरामीटर लेता है - पहला पैरामीटर के रूप में हम इरर स्टैण्डर्ड का प्रक्षेपण और दूसरा पैरामीटर में हम आउटपुट स्ट्रिंग पास करते हैं। आप इस उदाहरण में अपनी शुभाषा में स्ट्रिंग बदल सकते हैं।
