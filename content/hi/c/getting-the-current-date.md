## क्यों
प्रोग्रामिंग के साथ खेलने का अधिकार हमें साथ जोड़ता है लेकिन कभी-कभी हमें तारीख को हासिल करने के लिए कोई आवश्यकता होती है। तारीख उपयोगकर्ता को प्रोग्राम के बारे में अधिक जानकारी देती है जैसे कि कब प्रोग्राम को चलाया जा रहा है और कितने दिन या प्रतिलिपि खोली गई है। तो इसलिए हमें तारीख हासिल करने की आवश्यकता हो सकती है।

## कैसे करें
अब हम देखेंगे कि हम किस तरह से C कोड की मदद से वर्तमान तारीख को हासिल कर सकते हैं। यह करने के लिए हम C की शामिलिकर्त designs and algorithms पर आधारित एक फ़ंक्शन या library का उपयोग कर सकते हैं। निम्नलिखित उदाहरण में, हम सी स्टैंडर्ड लाइब्रेरी का ही उपयोग करेंगे।

```C
#include <stdio.h>
#include <time.h>
int main()
{
    // current system time
    time_t now = time(NULL);

    // structure to store local time and date
    struct tm *t = localtime(&now);

    // Format time and date
    printf("Current Date is: %d/%d/%d \n", t->tm_mday, t->tm_mon + 1, t->tm_year + 1900);

    return 0;
}
```

आउटपुट:
Current Date is: 2/7/2021

अपने स्क्रीन पर आने वाली छोटी छोटी बदलावों से उपयोगकर्ता को तारीख और समय में अंतर देखने के लिए उनके विश्वास में बढ़ोतरी होती है।

## गहराई से खोजना
लेकिन इस C प्रोग्रामिंग के माध्यम से तारीख का अन्धाधुंध प्राप्त करने से पहले इसे समझना महत्वपूर्ण है। "time.h" वाली बातों को निरंतर उपयोगकर्ता को 1900 से 1970 तक के समय की एक परिस्थिति में मोट
