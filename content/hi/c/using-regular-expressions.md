## क्यों

आपने regular expressions के बारे में सुना है लेकिन आपको यह समझने में कठिन लगता है? या शायद आप ने पहले कभी यह सुना ही नहीं है? इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि आपको क्यों regular expressions का उपयोग करना चाहिए और क्या फायदे हैं इससे।

## कैसे करें 

Regular expressions के उपयोग का एक उत्कृष्ट उदाहरण C भाषा में कैसे करें। नीचे दिए गए कोड ब्लॉक के माध्यम से हम आपको दिखाएंगे कि आप कैसे regular expressions को अपनी कोड में शामिल कर सकते हैं।

```C 
#include <stdio.h>
#include <regex.h>

int main() {

    // रेगुलर एक्सप्रेशन बनाएं
    regex_t regex;
    int ret;

    // अपडेट करें जिस स्ट्रिंग के साथ आपका मिलान होगा
    char *string = "प्रोग्रामिंग भाषाओं में C सबसे लोकप्रिय है। हम इसके जानकारियों का प्रयोग करके अपनी कोड लिख सकते हैं।";

    // आपकी रेगुलर एक्सप्रेशन बनाएं
    ret = regcomp(&regex, "C", 0);

    // स्ट्रिंग के साथ मिलान के लिए रेगुलर एक्सप्रेशन पर regexec कॉल करें
    ret = regexec(&regex, string, 0, NULL, 0);

    // सफल बनाने के लिए 0 वापस के साथ regexec कॉल करें
    if (!ret) {
        printf("मिलान सफल।");
    } else if (ret == REG_NOMATCH) {
        printf("मिलान नहीं हुआ।");
    }
    
    // रेगुलर एक्सप्रेशन मुक्त करें
    regfree(&regex);
    
    return 0;
}
```

उपरोक्त उदाहरण में, हमने एक स्ट्रिंग में "C" शब्द को ढूंढा। अगर स्ट्रिंग में "C" शब्द पाया जाता है, तो हमें "मिलान सफल।" का प्रिंट आउट मिलेगा। अगर स्ट्रिंग में "C" शब्द न
