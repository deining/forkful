## क्यों

आपने संभवतः सुना होगा कि C भाषा में हम रैग्यूलर एक्सप्रेशन (regular expressions) का प्रयोग करते हैं। आजकल यह एक बहुत ही उपयोगी और प्रभावी तरीका है डेटा को प्रोसेस करने का। चलिए जानते हैं इसकी वजह क्या है।

## कैसे

अगर आपने C भाषा में प्रोग्रामिंग की है तो आप जानते हैं कि रैग्यूलर एक्सप्रेशन को `regex.h` हेडर फ़ाइल से इम्पोर्ट किया जाता है। इसके बाद हम `regex_t` डेटाटाइप का वेरिएबल बनाते हैं जो हमारे रैग्यूलर एक्सप्रेशन को कॉम्पाइल करता है। इसके बाद हम `regcomp()` फ़ंक्शन को कॉल करके रैग्यूलर एक्सप्रेशन को कंपाइल करते हैं। अगर कोई मैचिंग स्ट्रिंग मिलती है तो हम `regexec()` फ़ंक्शन को कॉल करके उससे मैचिंग डाटा निकाल सकते हैं। आप खुद भी कोडिंग द्वारा इसके उदाहरण को समझ सकते हैं।

```C
#include <stdio.h>
#include <regex.h>

int main()
{
    regex_t regex;
    int match;
    char *string = "Hello, World!";

    match = regcomp(&regex, "Hello", 0);
    if(match == 0)
    {
        match = regexec(&regex, string, 0, NULL, 0);
        if(match == 0)
        {
            printf("Match found!\n");
        }
    }

    regfree(&regex);
    return 0;
}
```
Output:
```
Match found!
```
