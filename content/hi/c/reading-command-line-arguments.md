## क्यों

क्या आपने कभी सोचा है कि हमारे पास यदि कोई प्रोग्राम हो और हमारे पास ज्यादा से ज्यादा स्टोरेज न हो, तो कैसे हमें प्रोग्राम को उससे अपनी मंगतव्य के हिसाब से खरीदेलने जाना हो तो कैसे हम कम स्टोरेज में ही इसे खरीद सकते हैं। कमांड लाइन आर्ग्यूमेंट्स यह सुविधाएं हमें प्रदान करते हैं और इसलिए यह जानना आवश्यक है कि इस प्रकार की छोटी बात के लिए आप मात्र कुछ ही स्टोरेज से ही के साथ इससे लाखों तक का सामान खरीद सकते हैं।

## कैसे करें

```C
#include <stdio.h>

int main(int argc, char *argv[]) {
    int i;

    for (i = 0; i <= argc; i++) {
        printf("%s \n", argv[i]);
    }

    return 0;
}
```

ऊंटपुलारोड पे यह प्रोग्राम कैसे वर्क्स करता है? इसमे हम फ़ोर लूप को फिक्स एक्सी करते हैं, i को 0 से लेकर argc तक थंब करते हुए और इससे तो उसमें प्रिंटएफ फंक्शन को अपरोम्यने थाकते हुए, हमें printf हाथों पर मिलते हुए उसी अंeko ith तक क्या होगे, ये सुप्रिमेन्सिंग. खासदासुचिक भी यहीँ हम ने डेबग करनेकी कोशिश की है, क्ेसे इसे कमैंड नन्दं सेफ़ की संस्थापना किया होता है।

### साम्प्ले जेनेरिकोडमन्ड्स
1. `./a.out hello world`: प्रोग्राम को "hello" और "world" जैसे आर्ग्यूमेंट्स पास करके चलाना।
2. `./a.out`: आर्ग्यूमेंट्स नहीं होने की स्थिति में प्रोग्राम कैसे चलता है।
3. `./a.out 1 2 three 4`: नंबर और स्ट्रिंग समान समान सक्षम
