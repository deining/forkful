# क्यों

क्या आप जानते हैं कि कंमांड लाइन आर्ग्यूमेंट्स क्या हैं और इन्हें क्यों पढ़ना जरुरी है? आपको अपने संगठन में काम करते हुए इन बहुत सकारात्मक और उपयोगी फीचर्स के बारे में पता होना चाहिए, जो C प्रोग्रामिंग में कमांड लाइन से संबंधित हो सकते हैं। इसलिए आपको कमांड लाइन आर्ग्यूमेंट्स को पढ़ना जरुरी है ताकि आप अपने प्रोग्राम में इस्तेमाल कर सकें और सही रूप से काम कर सकें।

# कैसे करें

जैसा कि हम सभी जानते हैं, C प्रोग्रामिंग में मुख्य दिशा निर्देश या मुख्य() फंक्शन होता है जो प्रोग्राम के मुख्य लॉजिक को चलाता है। लेकिन कई बार आपको अपने प्रोग्राम को किसी स्थिर मानदंड के साथ चलाना होता है, जैसे कि उपयोगकर्ता के द्वारा प्रदान किए गए आर्ग्यूमेंट्स के आधार पर। ऐसे में, प्रोग्राम कोमांड लाइन से आर्ग्यूमेंट्स पढ़ने के लिए argc और argv variables का उपयोग करता है। नीचे दिए गए उदाहरण में, हम एक प्रोग्राम बनाएंगे जो दो संख्याओं को आर्ग्यूमेंट्स के रूप में ले कर आकार को कैलकुलेट करेगा।

```C
#include<stdio.h>

int main(int argc, char *argv[]) {
	int length = atoi(argv[1]);
	int width = atoi(argv[2]);
	int area = length * width;

	printf("आकार: %d इकाई चौड़ाई / लंबाई: %d इकाई", area, width, length);

	return 0;
}
```
