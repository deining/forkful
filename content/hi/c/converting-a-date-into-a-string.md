## क्यों

कभी-कभी कंप्यूटर प्रोग्रामिंग में, हमें दिनांक को एक स्ट्रिंग में बदलने की आवश्यकता होती है। ये दिनांक, महीना और साल के रूप में नंबरों की श्रृंखला है और हमारी आंकड़ों को प्रदर्शित करने के लिए हमें इसे स्ट्रिंग में बदलना होता है। अगर आप भी इसके बारे में जानना चाहते हैं तो आप यह लेख ज़रूर पढ़ें। 

## कैसे करें

यदि आपको दिनांक को स्ट्रिंग में बदलना है, तो सबसे पहले हमारे पास सुनिश्चित करने के लिए एक `struct tm` वेरिएबल होनी चाहिए जो दिनांक को साल, महीना और दिन के रूप में संग्रहीत करती है। फिर हम `strftime()` फ़ंक्शन का उपयोग करके इस स्ट्रक्टर को स्ट्रिंग में बदल सकते हैं। यहां `strftime()` की फ़ोर्मेटिंग पाठ्यक्रम है: 

```
%[flag][width][.precision]specifier 
```

इसमें से दो पद ज़रूरी होते हैं। पहला specifier जो हमें दिनांक को स्ट्रिंग में बदलना है जैसे `%d` (दिन), `%m` (महीना), `%Y` (साल) आदि। और दूसरा जो width होता है जिसे हम fractions के साथ उपयोग कर सकते हैं। इसके उपयोग का एक उदाहरण निम्नलिखित रूप से हो सकता है। 

```
// स्ट्रिंग में दिनांक का बदलाव
void convertDateToString() {
    struct tm *today;
    char buffer[20];
    
    // अपनी तारीख को एक स्ट्रिंग में बदलें
    today = localtime(&t);
    strftime(buffer, sizeof(buffer), "%d %m %Y", today);
    
    printf("तारीख: %s\n", buffer);
}
```
