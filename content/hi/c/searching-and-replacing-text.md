## Why

क्यों: किसी भी टेक्स्ट को ढूंढने और बदलने के लिए एक सुविधाजनक समाधान जानने की आवश्यकता हो सकती है।

## How To

कैसे करे: टेक्स्ट खोज और बदलाव का एक सरल तरीका है। यह कोड खोजने और बदलने के लिए निम्न तरह से काम करता है:

```C
#include <stdio.h>
#include <string.h>

int main() {
    // संदर्भ स्ट्रिंग दर्ज करें
    char str[] = "यह एक उदाहरण है।";

    // खोजने के लिए पैटर्न स्ट्रिंग
    char pattern[] = "उदाहरण";

    // बदलने के लिए प्रतिस्थापन स्ट्रिंग
    char replacement[] = "नमूना";

    // पैटर्न स्ट्रिंग के साथ संदर्भ स्ट्रिंग की तारीखी स्थिति का पता लगाएं
    char *found = strstr(str, pattern);

    // पता लगाएं कि पैटर्न स्ट्रिंग कहां में मिला
    int start = found - str;

    // पाठ को बदलने के लिए प्रतिस्थापन स्ट्रिंग का उपयोग करें
    if (found) strncpy(found, replacement, strlen(replacement));

    // प्रतिदर्शित करें
    printf("प्रतिस्थापन के बाद: %s", str);

    return 0;
}
```

आउटपुट: प्रतिस्थापन के बाद: यह एक नमूना है।

## Deep Dive

गहराई में: टेक्स्ट खोज और बदलाव एक फ़ंक्शन है जो प्रत्येक स्ट्रिंग को खोजने या कहीं भी मिलने पर उसे एक दूसरे स्ट्रिंग के साथ बदलता है। यह C में एक साधारण लाइब्रेरी फ़ंक्शन है जिसका उपयोग स्ट्रिंग प्रसंस्करण के लिए किया जाता है। वैसे तो अनेक अल्ग अल्ग आउटपुट का उत्पन्न हो सकता है, लेकिन सामान्य रूप से यह फ़ंक्शन original string में देखें जो स्ट्रिं
