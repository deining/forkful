## क्यों

यदि आपको किसी भी तरह का टेस्टिंग या डेटा वैधीकरण करना हो, तो आपको कुछ प्रायोजनीय संख्याओं की आवश्यकता हो सकती है। इसलिए, हम आज जानेंगे कि कैसे एक C प्रोग्राम में रैंडम नंबर्स जेनरेट करना है।

## कैसे करें

कृपया ध्यान दें कि रैंडम नंबर्स का उपयोग कंप्यूटर पर दूसरों के लिए प्रवेश को शामिल नहीं है। चलिए शुरू करें।

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    // समय को बेसिक्स पर सेट करें
    srand(time(0));

    // 1 से 100 तक के बीच रैंडम नंबर्स जेनरेट करें
    int num1 = rand() % 100 + 1;
    int num2 = rand() % 100 + 1;

    // 2 नंबरों को मिलाएं और परिणाम प्रिंट करें
    printf("%d + %d = %d", num1, num2, num1 + num2);

    return 0;
}
```

उपरोक्त कोड के लिए, आप निम्नलिखित आउटपुट प्राप्त करेंगे:

```
65 + 41 = 106
```

अन्य तरीके से भी आप रैंडम नंबर्स जेनरेट कर सकते हैं, जैसे:

- `rand() % n`: 0 से n-1 तक के बीच रैंडम नंबर्स जेनरेट करने के लिए
- `rand() % (max - min + 1) + min`: min से max तक के बीच रैंडम नंबर्स जेनरेट करने के लिए

इसके अलावा, आप `srand()` को किसी भी अंक से शुरू कर सकते हैं, जिससे कि आप हमेशा वही रैंडम नंबर्स प्राप्त करें।
