##

सी प्रोग्रामिंग भाषा में रैंडम नंबर देने का यह फीचर बहुत ही उपयोगी है। यह बहुत सी अलग-अलग एप्लीकेशंस के लिए उपयोगी हो सकता है, जैसे कि गेम डेवलपमेंट, सक्रियता शुद्धिकरण के लिए टेस्टिंग और कंप्यूटर विज्ञान में विभिन्न अनुक्रम बनाने के लिए।

## कैसे करें

अंकों को रैंडम तरीके से उत्पन्न करना बहुत आसान है, सी प्रोग्रामिंग भाषा में इसके लिए `rand()` फ़ंक्शन का उपयोग किया जाता है। नीचे दिए गए सामान्य संकेत सहित हम आपको एक उदाहरण देंगे कि कैसे इस फ़ंक्शन का उपयोग करके रैंडम नंबर्स को कैसे प्रोग्राम में शामिल किया जा सकता है।

```C
// एक उदाहरण कोड
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int random_num = rand(); // random_num की मान को 0 से लगभग 32767 तक का एक रैंडम नंबर सेट करेगा।
    printf("रैंडम नंबर हैः %d", random_num);
    return 0;
}

// आउटपुट:
// रैंडम नंबर है: 20315
```

आप यहां देख सकते हैं कि प्रत्येक बार कोड चलाने पर `random_num` की मान अलग-अलग हो रही है। आप चाहे तो `srand()` फ़ंक्शन का भी उपयोग करके प्रारम्भिक सीड वैल्यू को सेट कर सकते हैं जिससे कि हर बार समान रैंडम नंबर सेट न हो।
