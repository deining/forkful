## क्या और क्यों?


क्या है और क्यों करते हैं यह काम जब हम कहते हैं "रैंडम नंबर उत्पन्न करना" 
तो यह मतलब होता है कि हम एक अनियमित संख्या को प्रदर्शित करने के लिए कंप्यूटर को उपयोग कर रहे हैं। 
हमारे पास बहुत से प्रायोगिक एप्लिकेशन हैं जो रॉमब्लिंग को चाहती हैं और हमारी जांच कितने भाग्यशाली हैं। 

## कैसे करें:

```C
/*
यहाँ हम रैंडम नम्बर उत्पन्न करते हैं 
और उसे प्रिंट करते हैं
*/
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(0));
    int random_num = rand(); 
    printf("रैंडम नंबर है: %d\n", random_num); 
    return 0;
}
```
आप उपरोक्त कोड को कॉपी करके कंपाइल करें और चलाएं। 
आप बार-बार चलाएं और देख सकते हैं कि आपको हर बार एक अलग रैंडम नंबर मिलता है। 

## गहराई में जाएं:

प्यारे पाठक, रॉमब्लिंग के इतिहास में, हम रॉमब्लिंग की इच्छा के लिए आपणे लगभग लगा है से लगा गई थी। 
लेकिन आजकल कुछ अन्य भी तरीके हैं जिनका हम उपयोग कर सकते हैं जैसे कि कंप्यूटर का समय और तारीख और इनपुट के प्रमाणों को भी उपयोग करना। 
पुराने समय में, इसके लिए हम याददाश्त भी करते थे, लेकिन मॉडर्न कंप्यूटर आपको बहुत सारा काम करने देता है। 
अत: यह कंप्यूटर तरीका भी उपयोगी हो सकता है जब हमें संयोजना के लिए एक अनियमित संख्या की आवश्यकता हो। 

## इससे और भी पढ़ें:

आप लगातार और भी तरीके जान सकते हैं जो रॉमब्लिंग के लिए उपयोगी होते हैं।
