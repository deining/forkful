## क्यों

क्या आपने कभी सोचा है कि हम यदि किसी भी भाषा प्रोग्रामिंग लैंग्वेज में रैंडम नंबर्स को कैसे बना सकते हैं? या हमारे क्या सबसे बड़े लक्ष्य है एक हुमेशा से अलग तरह के रैंडम नंबर्स प्राप्त करने का? इस लेख में, हम आपको बताएंगे कि रैंडम नंबर्स क्यों जरूरी हैं और उन्हें कैसे बनाया जा सकता है।

## कैसे करे

रैंडम नंबर्स को बनाने के लिए, हम स्टैंडर्ड लाइब्रेरी में उपलब्ध "rand()" फंक्शन का उपयोग कर सकते हैं। यह फंक्शन सिस्टम के समय से समय पर तारीख्यों को आधार बनाकर रैंडम नंबर उत्पन्न करता है। निम्नलिखित कोड ब्लॉक में, हम नंबर 0 से 99 तक के रैंडम नंबर्स को प्रिंट करेंगे।

```C
#include <stdio.h>
#include <stdlib.h>

int main() {
    for (int i = 0; i < 10; i++) {
        int rand_num = rand() % 100;
        printf("रैंडम नंबर: %d\n", rand_num);
    }
    return 0;
}
```

```C

रैंडम नंबर: 62
रैंडम नंबर: 39
रैंडम नंबर: 90
रैंडम नंबर: 52
```
