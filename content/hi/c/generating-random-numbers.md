## क्यों

क्या आपने कभी सोचा है कि आपके कंप्यूटर पर दिखाई दे रहे सभी नंबर कैसे उत्पन्न होते हैं? क्या आपको पता है कि कंप्यूटर प्रोग्रामिंग में रैन्डम नंबर्स बहुत महत्वपूर्ण होते हैं? इस ब्लॉग पोस्ट में हम आपको बताएंगे कि रैन्डम नंबर्स को कैसे तैयार किया जाता है और यह क्यों महत्वपूर्ण है।

## कैसे

यदि आप एक प्रोग्रामर हैं तो आपको जानना चाहिए कि रैन्डम नंबर्स को कैसे तैयार किया जाता है। आइए हम एक छोटे से कोड द्वारा इसका उदहारण देखें:

```C
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main()
{
    // सूचक चिन्हों की राशि
    int min = 1, max = 100;
    // वास्तविक समय को आधार बनाकर बीच में से एक रैन्डम नंबर प्राप्त करें
    srand(time(0)); 
    // रैन्डम नंबर तैयार करें
    int num = (rand() % (max - min + 1)) + min; 
    // प्रिंट करें
    printf("%d\n", num); 

    return 0;
}
```

उपरोक्त कोड को कंपाइल करने के बाद आपको एक रैन्डम नंबर दिखेगा जो 1 से 100 के बीच हो सकता है। आप इस उदाहरण को अपनी आवश्यकता अनुसार बदल सकते हैं।
