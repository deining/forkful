## क्यों
चारों ओर हम अपने आस-पास खासतौर से उस जगह जहां हमारा काम हमसे अधिक अनुभवी व्यक्ति से आशुभास्क हो गई हो, उदाहरण के लिए सेवक। और उसका बेबसी से स्वागत करने के लिए हमें उसकी माती के साथ चलना पड़ेगा। तो कैसे हम उसकी ओर सेवगया समर्थ रहेंतिरह चाहो, हमारी आनन्द अपादित रहेगा, आमतौर से विकस्त उपयुक्त हां।

## कैसे
```

#include <stdio.h>

int main() {
    char str[] = "Hello World";
    int length = 0;

    // using a loop to count the number of characters in the string
    while (str[length] != '\0') {
        length++;
    }

    // printing the result
    printf("The length of the string is: %d", length);

    return 0;
}
```

जैसा की आप देख सकते हैं, हमने ```while``` लूप का उपयोग करके स्ट्रिंग में कितने अक्षर होते हैं को गणना किया है और रिजल्ट प्रिंट किया है। आप किसी भी स्ट्रिंग के लिए यही प्रोग्राम का उपयोग कर सकते हैं।
