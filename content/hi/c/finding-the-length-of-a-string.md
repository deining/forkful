# क्यों

शब्द की लंबाई पाना, C कार्यक्रमन सीखने के पीछे के कारणों में से एक है। यह एक बनावटी सामान्य टास्क है जो बहुत से सीखने वाले प्रोग्रामर को एक प्रारंभिक आधार में शून्य से शुरू करने के लिए उनके पास किसी भी प्रोग्रामिंग भाषा के साथ समझने के लिए मिलता है। 

# कैसे करें

```C
#include <stdio.h>

int stringLength(char str[]);

int main()
{
    char str[50];
    int length;
    printf("Enter a string: ");
    scanf("%s", str);

    length = stringLength(str);

    printf("Length of string %s is %d", str, length);
    return 0;
}
```

उपरोक्त कोड में हमने सीखा है कि कैसे स्ट्रिंग की लंबाई कैसे निकाली जाती है। हमने पहले स्ट्रिंग को स्कैन किया और उसकी लंबाई को "stringLength" फंक्शन की मदद से निकाला। फंक्शन में हमने लूप चलाकर एक एक करके हर चरित्र को गिना लिया और इन काउंट में समायोजित किया। 

आप इस आम से स्ट्रिंग लंबाई को विभिन्न तरीकों से निकल सकते हैं। आप कुछ और मैथमैटिकल ऑपरेशन्स जोड़ सकते हैं, या आप एक भिन्न विधि भी उपयोग कर सकते हैं जो आपको अभी तक नहीं समझी होगी।
