## इससे क्यों
 
जब हम प्रोग्रामिंग करते हैं, तो हमें कभी-कभी एक वर्ग में दिए गए स्ट्रिंग (लेख) की लंबाई को जानने की जरूरत होती है। यह आवश्यक है क्योंकि हम उस वर्ग के अन्य वस्तुओं को स्ट्रिंग को कॉन्केट करने और उसमें डेटा को अद्यतन करने के लिए स्ट्रिंग की लंबाई को जानने की आवश्यकता होती है। इसलिए, हमें स्ट्रिंग की लंबाई को जानने का यह तरीका जानना जरूरी है।

## कैसे करें

अब आप जानते हैं कि स्ट्रिंग की लंबाई को जानने का कारण क्या है, तो आइए इसे कैसे करें। स्ट्रिंग की लंबाई को प्राप्त करने के लिए, हम `strlen()` फ़ंक्शन का उपयोग कर सकते हैं। यह फ़ंक्शन बिल्ट-इन है और C भाषा में उपलब्ध है। आप बिना कोई पैरामीटर या स्ट्रिंग के साथ `strlen()` को कॉल कर सकते हैं और यह वापस स्ट्रिंग की लंबाई देगा। यहां एक उदाहरण है:

```C
#include <stdio.h>
#include <string.h>

int main() {
    // एक स्ट्रिंग डेक्लेर करें
    char str[20] = "मेरा नाम राहुल है";
    
    // स्ट्रिंग की लंबाई प्राप्त करें
    int length = strlen(str);
    
    // प्राप्त किया गया लंबाई प्रिंट करें
    printf("स्ट्रिंग की लंबाई: %d", length);
    
    return 0;
}
```

```
स्ट्रिंग की लंबाई: 15
```
