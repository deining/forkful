## क्यों

कई बार हमें कोई प्रोग्राम बनाने के दौरान या किसी विशेष काम करते समय एक स्ट्रिंग की लम्बाई को जानने की आवश्यकता होती है। इसलिए, इस ब्लॉग पोस्ट में हम देखेंगे कि स्ट्रिंग की लम्बाई निकालने का क्या महत्व है।

## कैसे करें 

यदि हमें किसी भी स्ट्रिंग की लम्बाई निकालनी होती है, तो हम इसे "strlen" फंक्शन का इस्तेमाल करके आसानी से कर सकते हैं। ये फंक्शन "string.h" हैडर फाइल में उपलब्ध है। नीचे दिए गए कोड ब्लॉक में हम एक स्ट्रिंग की लम्बाई को निकालने का एक सरल उदाहरण देखेंगे।

```C
#include <stdio.h>
#include <string.h>

int main() {
    char str[50];
    printf("Enter a string: ");
    gets(str);

    // using strlen function to find length of string
    int len = strlen(str);

    printf("Length of string is: %d", len);
    return 0;
}
```

आउपुट: 

``` 
Enter a string: Hello World
Length of string is: 11
```

## गहराई में खोज

जब हम "strlen" फंक्शन का इस्तेमाल करते हैं, तो इसका समय की खोज करने का एक्सपोनेंशल ऑर्डर होता है। इसकी समय की खोज लगभग O(n) होती है, जहां n स्ट्रिंग की लम्बाई है। और यदि हमारे पास स्ट्रिंग के साथ विशेष चर भी होते हैं तो यह इसे विफल कर सकते हैं। इसलिए, हमेशा "strlen" फंक्शन का दायरा और समय की खोज को ध्यान में रखें।
