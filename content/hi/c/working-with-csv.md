## क्या और क्यों?
सीएसवी (CSV) काम क्या है और क्यों प्रोग्रामर इसे करते हैं, यही आपको इस आलेख में बताया जाएगा। CSV फाइलें डेटा को एक टेक्स्ट फाइल में स्टोर करने का एक प्रासंगिक तरीका है। प्रोग्रामर CSV को उन डेटा सेट के साथ काम करने के लिए उपयोग करते हैं जिन्हें स्प्रेडशीट ऐप्स द्वारा सौंपा जाता है।

## कैसे करें:
निम्न उदाहरणों में हम आपको CSV को पढ़ने और लिखने के साथ काम करने के तरीके दिखाएंगे। हमने इसके लिए कई C मेथड शामिल किए हैं, आप उन्हें अपने आवश्यकतानुसार बदल सकते हैं। सभी उदाहरणों के लिए हमने एक CSV फाइल "sample.csv" का उपयोग किया है।

```C
#include<stdio.h>
#include<stdlib.h>
#include<string.h>

// CSV फाइल से डेटा पढ़ें
void read_csv() {
    FILE *fp = fopen("sample.csv", "r"); // पढ़ने के लिए फ़ाइल खोलें
    char line[100]; // स्टोर करने के लिए लाइन
    char* tok; // टोकन के लिए बफ़र
    while (fgets(line, 100, fp) != NULL) { // प्रत्येक लाइन के लिए
        tok = strtok(line, ","); // "," के आधार पर शब्द आयात करें
        while(tok != NULL) { // सभी शब्दों के लिए
            printf("%s ", tok); // प्रिंट करें
            tok = strtok(NULL, ",");
        }
        printf("\n");
    }
    fclose(fp); // फ़ाइल पढ़ने के लिए बंद करें
}

// CSV फाइल में डेटा लिखें
void write_csv() {
    FILE *fp = fopen("sample.csv", "w"); // लिखने के लिए फ़ाइल खोलें
    char* name = "John";
    int age = 25;
    fprintf(fp, "%s,%d\n", name, age); // लिखें
    fclose(fp); // फ़ाइल लिखने के लिए बंद करें
}

int main() {
    read_csv(); // "sample.csv" फाइल से डेटा पढ़ें
    write_csv(); // "sample.csv" में डेटा लिखें
    return 0;
}
```
आउटपुट:
```
John 25
```

## गहराई में जाएं:
CSV फाइलें 1972 में IBM द्वारा विकसित की गई थीं। पूरे नाम डेटा कोमा सीपरेटेड वैल्यूज (CSCV) के रूप में जाना जाता है, और वे स्प्रेडशीट सॉफ़्टवेयर के बढ़ते उपयोग के कारण उभरे हुए थे। CSV फॉर्मेट का उपयोग डेटा को इलेक्ट्रॉनिक रूप में स्टोर करने और भंडारण करने के लिए किया जाता है और इसलिए वे विभिन्न प्रकार के सॉफ़्टवेयर में आसानी से पढ़े जा सकते हैं। CSV के अलावा, प्रोग्रामर अन्य फॉर्मेट जैसे TSV और XML को भी क्रमशः पसंद कर सकते हैं।
