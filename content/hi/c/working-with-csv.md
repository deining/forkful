## क्यों

क्या आपने CSV से कभी काम किया है? अगर हाँ, तो आप जानते होंगे कि ये डेटा संरचित करने और संशोधित करने के लिए महत्वपूर्ण है। आप किसी भी तरह के डेटा को CSV फॉर्मेट में सहेज सकते हैं और आसानी से प्रसंस्करण और विश्लेषण के लिए इसका उपयोग कर सकते हैं।

## कैसे करें

CSV फॉर्मेट की सरलता के कारण, इसे C में इंपोर्ट करना हमेशा से ही आसान रहा है। जब आप अपने कोड में CSV को पढ़ने और संशोधित करने के लिए fopen() और fgets() फ़ंक्शन का उपयोग करते हैं, तो आपको केवल संबंधित स्ट्रिंग और इंटीजर प्रकार के परिवर्तन के साथ काम करना होगा। निम्नलिखित कोड ब्लॉक एक CSV फ़ाइल से डेटा पढ़ता है और उसे आराम से प्रिंट करता है।

```C
#include <stdio.h>

int main() {
  FILE *file = fopen("data.csv", "r");
  char line[256];

  if (file == NULL) {
    printf("फ़ाइल खोलने में त्रुटि!\n");
    return 1;
  } else {
    while (fgets(line, sizeof(line), file)) {
      printf("%s", line);
    }
  }

  fclose(file);
  return 0;
}
```
