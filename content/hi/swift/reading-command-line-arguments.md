## क्यों
कमांड लाइन आर्ग्यूमेंट्स को पढ़ने से आपको Swift प्रोग्रामिंग में बहुत सारे फायदे हैं। यह आपको स्क्रिप्टिंग का काम आसान बनाता है, अधिक एफिकसी से कोडिंग करने का मौका देता है और उपयोगकर्ताओं से भागीदारी करने के लिए सुविधाजनक है।

## कैसे करें
```Swift
// प्रोग्राम्मिंग लैंग्वेजिंग गाइड के संदर्भ में, आईओस के प्रशिक्षण संसाधनों की प्रारंभिक पृष्ठ के साथ - [एप्पल, इंक।] (https://developer.apple.com/library/archive/documentation/General/Conceptual/DevPedia-CocoaCore/CommandLineArgs.html)

let arguments = CommandLine.arguments
for argument in arguments {
    print(argument)
}
```
जैसा कि आप देख सकते हैं, हमने प्रोग्राममा CommandLine.arguments का उपयोग किया है जो हम आर्ग्यूमेंट्स को स्ट्रिंग के लिए प्राप्त करते हैं और फिर हमने उन्हें प्रिंट किया है। आप इसे अपने स्कोपल में जोड़ सकते हैं और इसे अपने कोड के अनुसार अपनी आवश्यकताओं के अनुसार फ़िल्टर कर सकते हैं।
