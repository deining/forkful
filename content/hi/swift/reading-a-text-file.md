## क्यों

टेक्स्ट फाइलों को पढ़ने का सबसे आसान और अत्यधिक प्रभावी तरीका है स्विफ्ट प्रोग्रामिंग का इस्तेमाल करना। अगर आप स्विफ्ट बनाने के बारे में सीखना चाहते हैं तो यह एक अत्यधिक महत्वपूर्ण कदम है।

## कैसे करें

आप अपने स्विफ्ट कोड में निम्न संकेतों का अनुसरण करके टेक्स्ट फाइल को पढ़ सकते हैं:

```
let filePath = "sample.txt" // यहां आपको अपनी फाइल का पथ डालना होगा

do{
    let text = try String(contentsOf: URL(fileURLWithPath: filePath))
    print(text)
}catch{
    print("फाइल पढ़ने में समस्या हुई।")
}
```

यहां हम ``contentsOf`` में फाइल का पथ और ``fileURLWithPath`` में औसत फाइल का एक URL प्रदान करते हैं जिसे आप स्ट्रिंग में कन्वर्ट कर सकते हैं। यदि कोई त्रुटि होती है, हम त्रुटि संदेश प्रिंट करते हैं।

## गहराई में जाएं

टेक्स्ट फाइल को पढ़ने के लिए हम यह ``String`` क्लास को उपयोग करते हैं जिसमें हम स्ट्रिंग डेटा के साथ काम कर सकते हैं। इसके अलावा, हम ``Data`` क्लास का भी इस्तेमाल कर सकते हैं जो डेटा एक्सपोर्ट करने और इम्पोर्ट करने के लिए बनाया गया है। आप अपनी आवश्यकतानुसार, अलग-अलग तरीकों से फाइल को पढ़ सकते हैं।
