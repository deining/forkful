# आप एक स्विफ्ट प्रोग्रामिंग ब्लॉग कैसुअल तरीके से नृत्य पाठ पर क्यों लिखना चाहते हो.

आमतौर पर एक text फाइल उन व्यापक सामग्री को संकलित करता है जिसके आपके पास programming बनाने के लिए आवश्यक होते हैं. यह मरम्मत व अन्य साधनों से भी आपको फायदे प्रदान करता है.

## कैसे

अब आपको text file बनाना जरूरी है. एक स्विफ्ट फ़ंक्शन में या हाईराकी कोड को switch करने के लिए इस्तिमाल करना चाहिए. स्विफ्ट कोड के बारे में कुछ उदाहरण दें:

````Swift
func readTextFile() {
  if let path = Bundle.main.path(forResource: "sample text file", ofType: "txt") {
    do {
      let text = try String(contentsOfFile: path, encoding: .ascii)
      print(text)
    } catch {
      print(error)
    }
  }
}
````

यहां आप सीधे "sample text file" की शर्त का पालन कर रहे हैं जो आपके एप्लिकेशन बंदल इसके साथ साथ आपके शामिल किए गए होंगे. ये आप और आपके प्रोग्राम के बारे में कुछ सबक सीखने के लिए म्यानिंग्फुल विकल्प हो सकता है.

## गहराई तक

अब आपके पास text file है, आप उसे कितना स्पष्ट है इसको समझने के लिए. फाइल के साथ-साथ आप उसे कैसे बनाते होंगे, जिसके एक बड़े भाग एक अनिर्वचनीय और लाइन बात हो गई हो. जैसा इस हिसाब से सटीक वर्णन साफ़ हुए:

- यदिन कुछ problems हो तो आपको decoding वर्णन करने की कोशिश करनी चाहिये.
- यदि decoding पेशेवर होना हो तो महत्वपूर्ण बात यह हो जब आपकी फाइल खालास हो जाएगी, तो हो रहा है.
