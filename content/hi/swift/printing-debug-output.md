## क्यों
डिबग आउटपुट प्रिंट करने में इंजीनियरिंग में लगाने के लिए क्यों?

डिबगिंग एक सोचने का और समस्याओं का समाधान करने का प्रक्रिया है जो प्रोग्रामिंग और सॉफ्टवेयर निर्माण का अपूर्ण हिस्सा है। डिबग आउटपुट प्रिंट करना एक उपयुक्त और आसान तरीका है ताकि हम अपने कोड में किसी भी त्रुटि को खोज सकें और उसे सुधारें। यह हमारी समस्याओं को समझने में अधिक मदद करता है और हमारे कोड को बेहतर बनाने में मदद करता है। 

## कैसे
डिबग आउटपुट प्रिंट करने के लिए कूड़ा उदाहरण और सैंपल आउटपुट कदम संशोधन में उपयोग करने के लिए निम्नलिखित स्थानों पर "```Swift ... ```" कोड ब्लॉक्स के साथ दिए गए हैं।

"```Swift
// कूड़े कोड उदाहरण
let name = "रोहित"
print("नमस्ते, मेरा नाम है\(नाम)")
```
आउटपुट: "नमस्ते, मेरा नाम है रोहित"

"```Swift
// संशोधन कॉड उदाहरण
let numbers = [3, 6, 9, 12]
for number in numbers {
    print("3 से मिलकर \(number)")
}
```
आउटपुट:
"3 से मिलकर 3"
"3 से मिलकर 6"
"3 से मिलकर 9"
"3 से मिलकर 12"
