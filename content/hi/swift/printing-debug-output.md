## इससे क्यों?

डीबग आउटपुट प्रिंट करना एक महत्वपूर्ण कौशल है जो सिस्टम क्षतिग्रस्त या त्रुटि को खोजने और सुधार करने में मदद करता है। यह आपको अपने कोड को जांचने में भी मदद कर सकता है और आपके एप्लिकेशन को सुधारने में समय और श्रम बचाएगा।

## कैसे करें?

```Swift
// उदाहरण कोड ब्लॉक
let name = "आकाश"
let age = 28
print("नाम: \(name)")
print("उम्र: \(age)")
```
```
नाम: आकाश
उम्र: 28
```

यदि आपके पास बहुत से डेटा है, तो आप NSLog फंक्शन को उपयोग कर सकते हैं जो अधिक व्यापक और मेमोरी दोनों लेगी। आप इसके साथ विभिन्न डेटा टाइप्स को भी प्रिंट कर सकते हैं जैसे कि स्ट्रिंग, इंटेजर, फ्लोट, और बूलियन।

## गहरी खोज

डीबग आउटपुट प्रिंट करने के लिए आपको अधिक वैकल्पिक तरीकों का भी पता होना चाहिए। आप इसे अलग-अलग कोलम में प्रिंट कर सकते हैं, या उपयोगकर्ता इंपुट को पढ़ने के लिए स्केनर का उपयोग कर सकते हैं। आप भी गहराई स्तर पर डीबग आउटपुट प्रिंट करने के लिए डेबगिंग समय योचना का उपयोग कर सकते हैं।
