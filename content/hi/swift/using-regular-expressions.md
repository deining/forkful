## क्यों

अगर आप Swift प्रोग्रामिंग के प्रशंसक हो, तो आपने शायद कभी रेग्युलर एक्सप्रेशन के बारे में सुना होगा। रेग्युलर एक्सप्रेशन एक बहुत ही उपयोगी और शक्तिशाली टूल है जिसका उपयोग डेटा में खोजने और प्रोसेस करने में किया जाता है। इस लेख में हम सीखेंगे कि क्यों रेग्युलर एक्सप्रेशन एक भाषा स्वच्छता के लिए एक महत्वपूर्ण टूल है।

## कैसे करें

रेग्युलर एक्सप्रेशन का प्रयोग करना बहुत ही आसान है। आपको सिर्फ इसका प्रयोग अपनी कोडिंग में करना होगा और फिर आप अक्सर देखेंगे कि आर्ग्युलर एक्सप्रेशन आपकी कई समस्याओं को आसानी से हल कर सकता है। नीचे दिए गए कोड ब्लॉक में हम एक साधारण उदाहरण देखेंगे।

```Swift
// एक रेग्युलर एक्सप्रेशन बनाएं जो अंकों को पकड़ सकता है 
let regex = try! NSRegularExpression(pattern: "\\d+")

// स्ट्रिंग जाँचें कि क्या इसमें अंक हैं 
var str = "Hello 123 World"
if let match = regex.firstMatch(in: str, range: NSRange(str.startIndex..., in: str)) {
    print("पाया गया अंक: \(str[Range(match.range, in: str)!])")
}
```

आउटपुट:
```
पाया गया अंक: 123
```

## गहराई में जाइए

रेग्युलर एक्सप्रेशन का उपयोग मुश्किल और भी पूरी तरह से प्रोग्रामिंग की अधिकांश भाषाओं में और विभिन्न प्लेटफॉर्मों पर समर्थित है। यह रूल आधारित शब्दों की खोज को समर्थन करता है, जो बहुत ही उपयोगी होता है। आप भाषा क्ष
