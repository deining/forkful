## क्यों
अगर आप Swift प्रोग्रामिंग में नए हैं, तो आपने शायद उपलब्ध डेटा प्रकारों को स्ट्रिंग में बदलने के बारे में सुना होगा। आपको पता नहीं हो सकता कि यह क्यों जरूरी है, लेकिन हमारे इस ब्लॉग पोस्ट में हम आपको बताएंगे कि इसकी ज़रूरत क्यों होती है।

## कैसे करें
सबसे पहले, आपको Swift कोडिंग में तारीख को स्ट्रिंग में बदलने के लिए `DateFormatter` क्लास का उपयोग करना होगा। नीचे उदाहरण दिए गए हैं जो आपको समझने में मदद करेंगे:

```
// तारीखों को स्ट्रिंग में बदलने के लिए DateFormatter क्लास का उपयोग करें
let formatter = DateFormatter()

// इस तारीख के लिए प्रारूप चुनें
formatter.dateFormat = "dd-MM-yyyy"

// तारीख को स्ट्रिंग में बदलें
let date = Date() // वर्तमान तारीख
let stringDate = formatter.string(from: date)
print(stringDate) // प्रिंट: 03-06-2021
```

## गहराई में जाएं
जब हम तारीख को स्ट्रिंग में बदलते हैं, तो हमें दूसरे तारीखों के साथ बहुत सारे विकल्प भी मिलते हैं, जिन्हें हम अपनी ज़रूरतों के हिसाब से उपयोग कर सकते हैं। इसके अलावा, हमें समझना होगा कि तारीख को स्ट्रिंग में बदलने का माध्यम भी कई तरीकों से हो सकता है। आप इन विकल्पों को अपनी आवश्यकताओं के हिसाब से चुन सकते हैं और अपने कोड को और अधिक बेहतर बना सकते हैं। आप इस [गाइड](https://docs.swift.org/swift-book/LanguageGuide/StringsAndCharacters.html) को देख सकते हैं जो अधिक जानकारी प्रदान
