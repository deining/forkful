# क्यों

किसी पैटर्न से मिलते हुए अक्षरों को हटाने के लिए कोडिंग ब्लॉक का उपयोग कई स्थितियों में उपयोगी हो सकता है। इस समस्या के साथ काम करने के लिए बहुत सारे समय और श्रम की आवश्यकता हो सकती है जो आप इस तरह की प्रोग्रामिंग में कम कर सकते हैं। 

## कैसे

```Swift
// एक उदाहरण कोड

let text = "स्विफ्ट प्रोग्रामिंग में कैसे अणुनीत संख्याओं को हटाएं"

// पैटर्न मिलाना

let pattern = "[कैसे]"

// अक्षरों को हटाना

let filteredText = text.replacingOccurrences(of: pattern, with: "")

// नतीजा प्रिंट करें

print(filteredText)

// आउटपुट: स्विफ्ट प्रोग्रामिंग में अणुनीत संख्याओं को हटाएं 
```

यहां अक्षरों को हटाने के लिए `replacingOccurrences` फ़ंक्शन का उपयोग किया गया है जो पूरे टेक्स्ट में दिए गए पैटर्न को ढूंढकर हटा देता है। इसके अलावा, आप अलग-अलग प्रकार के पैटर्न के लिए अलग-अलग फ़ंक्शन का भी उपयोग कर सकते हैं। 

## गहराई से जाने

पूर्व में दिए गए उदाहरण में, हमने सामान्य पैटर्न `[कैसे]` का उपयोग किया था जिससे केवल टेक्स्ट में दिए गए अक्षरों को हटाया गया। लेकिन आप अपने आवश्यकताओं के अनुसार किसी भी तरह के पैटर्न का उपयोग कर सकते हैं। आइए जानते हैं कुछ अन्य उदाहरण जो आपके दिमाग में इस अंश को स्पष्ट कर सकते हैं।
