# क्या और क्यों?
एक चरित्र साफ़ करना एक एसई प्रोग्रामर द्वारा एक निश्चित पैटर्न का पालन करने वाले अक्षरों को हटाना है। प्रोग्रामर अक्षरों को हटाने के माध्यम से डेटा को संशोधित और आवश्यक और उपयोगी बनाने का प्रयास करते हैं।

# कैसे करें:
```Swift
// स्ट्रिंग से निकालें "आदि" संदेश
var str = "यह होना है नहीं आदि क्या आदि है।"
str = str.replacingOccurrences(of: "आदि", with: "")
print(str) // यह होना है नहीं ।
```

```Swift
// एरे दोहराएँ
var arr = [2, 4, 6, 8, 10]
for (index, element) in arr.enumerated() {
    if element % 4 == 0 {
        arr.remove(at: index)
    }
}
print(arr) // [2, 6, 10]
```

# गहराई जाँच:
इस फलक में, हम चरित्र साफ़ करने की विभिन्न परिकल्पनाएं देखेंगे और इसके पीछे की और विशेषताओं को समझेंगे। इसके साथ हम इसके स्थान पर उपयोग किया जाने वाला तत्कालिक समय और इम्प्लीमेंटेशन बात करेंगे।

## इतिहास संदर्भ:
चरित्र साफ़ करने को दूसरे प्रोग्रामिंग भाषाओं में भी जाना जाता है, जैसे कि पाइथन में इसे "शब्दों के साथ साफ़ करना" और जावा में "स्ट्रिंग से स्ट्रिंगों के साथ समान्य किसी अक्षर हटाना" कहते हैं।

## विकल्प:
उन प्रोग्रामिंग भाषाओं के अतिरिक्त, स्ट्रिंग का एक समान्य उपयोग है चरित्र अनुकूलन करने का और वेरिएबल स्ट्रिंग अनुसार द्वितीय द्वारा प्राप्त प्रति चरित्र रखने का डुबार।

## इम्प्लीमेशन विवरण:
स्ट्रिंग्स में एक वरियिंट्स में कैसे दृढ़ / अस्थिर इस बात देखी गई और इस कैसे स्थान पर कानालें करने के लिए संभव होता है। अगर आप इसे PHP जैसी टेक्स्ट मात्र औ नहीं रखेंगे, तो स्पष्टीकरण / संकुलन नीचे दी गई होगी।
