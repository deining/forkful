## क्यों

कभी-कभी हमें दो तारीखों को तुलना करने की जरूरत होती है, जैसे कि अपने प्रोग्राम में डेटा को फ़िल्टर करने के लिए। इस ब्लॉग पोस्ट में, हम यह जानेंगे कि स्विफ्ट में दो तारीखों को कैसे तुलना किया जाता है और यह हमारे प्रोग्रामिंग को कहाँ मदद कर सकता है।

## कैसे

दो तारीखों को तुलना करने का सबसे आसान तरीका है `compare` मेथोड का उपयोग करना। बस दो तारीखों को `Date` ऑब्जेक्ट में कनवर्ट करें और उन्हें `compare` मेथोड के दो विभिन्न पैरामीटरों के रूप में भेजें। नीचे दिए गए कोड ब्लॉक में आपको एक सादा उदाहरण मिलेगा:

```
// तारीख के साथ `compare` करने के लिए दो `Date` ऑब्जेक्टों के रूप में कनवर्ट करें
let date1 = Date(timeIntervalSince1970: 1577836800)  // 1 जनवरी 2020
let date2 = Date(timeIntervalSince1970: 1580515200)  // 1 फ़रवरी 2020

// तारीखों को तुलना करने के लिए `compare` मेथोड का उपयोग करें
if date1.compare(date2) == .orderedAscending {
    print("date1 तारीख से कम है date2 तारीख")
} else if date1.compare(date2) == .orderedSame {
    print("दोनों तारीखें बराबर हैं")
} else {
    print("date1 तारीख date2 तारीख से बड़ा है")
}

// कोड का आउटपुट: `date1 तारीख date2 तारीख से कम है`
```
