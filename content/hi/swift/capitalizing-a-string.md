## क्यों

कोई भी डेवलपर एक स्ट्रिंग को कैपिटलाइज करने का काम क्यों करेगा? स्विफ्ट अब तक किए गए सभी संस्करण में, स्ट्रिंग में वर्णों के मामले में स्विच का उपयोग होता है। यदि आप अपने कोड में स्विच का उपयोग करते हैं, तो आप बहुत सारी विशेषताओं को जोड़ सकते हैं जैसे कि स्विच का शामिल होना, स्ट्रिंग में से एक टाइप "string" या "racbat" आदि का टाइपसेफ़ भाग और अधिक। हालांकि, कभी-कभी हमें स्ट्रिंग को कैपिटलाइज करने की आवश्यकता होती है जैसे कि उपयोगकर्ता कम से कम प्रतिनिधि के लिए।

## कैसे करें

स्ट्रिंग को कैपिटलाइज करने के लिए, हम `capitalized` वस्तु चूक से स्ट्रिंग को उपयोग कर सकते हैं। 

```Swift
let string = "hello world"

print(string.capitalized)
```

आउटपुट:

```
Hello World
```

यहां हम स्विच का उपयोग करके एक उदाहरण देखते हैं, जो कशमीरी ग्राहकों को उनके द्वितीय निम्न वर्ण निम्न करने की अनुमति देगा। 

```Swift
let string = "कशमीरी पत्रकार"

switch string {
    case String.capitalized(string: "कशमीरी"):
        print("कशमीरी पत्रकार")
    default:
        print("कशमीर पत्रकार")
}
```

आउटपुट:

```
कशमीरी पत्रकार
```

इस तरह से, हमें एप्लिकेशन के लिए वैश्विक साबित हो सकता है कि उनके प्रतिनिधि या लोगों के अन्य नामों को अलगीबंदी कोडिंग द्वारा प्रविष्ट करके प्रदर्शित किया जा सकता है।
