## क्यों

नमस्ते स्विफ्ट प्रोग्रामरों! आज हम आपको बताएंगे कि स्विफ्ट में वर्तमान तारीख कैसे प्राप्त कर सकते हैं। आपने शायद यह सबसे आसान तरीका सोचा होगा - ख़ूब सोचो! क्योंकि वर्तमान तारीख की ज़रूरत हर एप्प या गेम में होती है। चाहे आप इसे अपने यूजर्स के लिए दिखाना चाहिए, या फिर आपको अपने एप्प के इंटरनल काम के लिए इसे प्रयोग करना हो। इसलिए कोडिंग का यह जादू आपके लिये बहुत काम आएगा। तो चलिए जानते हैं कि इसके लिए शुरुआती तरीके क्या हैं।

## हैं कैसे

मैंने स्विफ्ट के Date और DateFormatter क्लास का प्रयोग किया है और भाग्य से, इन दोनों को कंट्रोल करना बहुत ही आसान है! आपको सिर्फ एक नया Date इंस्टेंस बनाना होगा और उसे फॉर्मेट करना होगा। चलिए नीचे दिए गए कोड ब्लॉक्स में आपको दिशा-निर्देश दिए गए हैं।

```Swift
// अपना नया Date इंस्टेंस बनाएं
let currentDate = Date()

// अपने इंस्टेंस को फॉर्मेट करें
let dateFormatter = DateFormatter()
dateFormatter.dateFormat = "dd/MM/yyyy"

// अब हमें केवल Date इंस्टेंस के पास से स्ट्रिंग वर्जन प्राप्त करना होगा
let formattedDate = dateFormatter.string(from: currentDate)

// अब हमारे पास फॉर्मेटेड तारीख है! नीचे दिए गए प्रिंट कमांड से आप इसे प्रिंट कर सकते हैं
print(formattedDate)
```

और अगर आप इसका आसानी से इस्तेमाल करना चाहते हैं, तो इसे एक एकस्टेंशन के रूप में भी उपयोग
