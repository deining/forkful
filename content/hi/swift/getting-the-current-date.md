## कैसे करे

अब जब आप जानते हैं कि आपको अपना प्रोग्राम में तारीख को कैसे प्राप्त करनी है, तो आपको आवश्यकता होगी कुछ कोड दिखाने के लिए। फॉर्मेट में Swift में कोई भी आसानी से वर्तमान दिनांक को पाने का तरीका हो सकता है:

टोएगतेवः " अब हम कुछ कोड तैयार करते हैं जो तारीख को प्राप्त करेंगे

```
var date = Date() // वर्तमान तिथि और समय प्राप्त करें
print(date) // 2020-01-24 23:59:59 +0000

var formatter = DateFormatter()
formatter.dateFormat = "dd-MM-yyy" // दिन - माह - साल का तारीख प्रारूप
var formattedDate = formatter.string(from: date) // फॉर्मेट की गई तारीख को प्राप्त करें
print(formattedDate) // 24-01-2020
```
