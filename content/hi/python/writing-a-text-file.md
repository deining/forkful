शायद आपने हमेशा से पाठ फाइल (text file) बनाने के बारे में सुना होगा और आप यह जानना चाहते होंगे कि इसे लिखने का क्या महत्व है। पाठ फाइल बनाने का महत्व सही समय पर सही जगह पर सही छोड़ने के लिए होता है, जैसे कि कोड को लिखने के बाद आप उसे सेव करने के लिए पाठ फाइल का उपयोग कर सकते हैं।

## कैसे करें

आप आदेश डाल कर पाठ फ़ाइल बना सकते हैं। आपको अपने नये पाठ फ़ाइल के नाम और पथ को निर्देशित करना होगा। उदाहरण के लिए, हम एक "hello.txt" नामक पाठ फ़ाइल बनाने के लिए कोड लिखेंगे और उसमें "नमस्ते दुनिया!" टेक्स्ट टाइप करेंगे।

इसका कोड निम्न है:

```Python
f = open("hello.txt", "w")
f.write("नमस्ते दुनिया!")
f.close()
```

इसका आउटपुट हमारे नये पाठ फ़ाइल में से दिखाएगा:

नमस्ते दुनिया!

## गहराईगामी अध्ययन

 अब आप एक पाठ फ़ाइल क्यों लिखेंगे, किस प्रकार से पाठ फ़ाइल के साथ काम करेंगे और ज्यादा क्या कर सकते हैं। पाठ फ़ाइल एक साधारण टेक्स्ट फाइल होती है जिसमें प्लेन टेक्स्ट शामिल होता है। आप इसका उपयोग अपने काम के लिए नोट्स बनाने, कोड याद रखने या अन्य सामान्य कामों के लिए कर सकते हैं। आप इसका उपयोग स्पष्ट तौर पर अन्य Python कोड के साथ कर सकते हैं, जिससे आपको इसे पढ़ने या लिखने की आवश्यकता न हो।
