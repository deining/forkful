## क्यों

कभी-कभी हमारे पास एक टेक्स्ट फाइल बनाने की आवश्यकता होती है, चाहे वह एक डेटा फ़ाइल हो जिसमें हम अपने डेटा को संग्रहीत कर सकते हैं या फिर एक प्रोग्राम की कन्फ़िगरेशन फ़ाइल हो जिससे हम अपने प्रोग्राम को सेट अप कर सकते हैं। पाइथन में फाइल लिखने के लिए उपलब्ध कार्य हमें अपने उद्देश्य को पूरा करने में मदद करते हैं।

## कैसे करें

यदि आप पाइथन में टेक्स्ट फाइल लिखना सीखना चाहते हैं, तो पहले से बनने हुए एक टेक्स्ट फाइल को ओपन करना सबसे आसान तरीका हो सकता है। आप नीचे दिए गए कोड ब्लॉक को उपयोग करके एक फाइल बना सकते हैं।

```python
# फाइल ओपन करें
file = open("sample.txt", "w")

# टेक्स्ट लिखें
file.write("हैलो दोस्तों, यह एक पायथन ब्लॉग पोस्ट है!")

# फाइल बन्द करें
file.close()
```

ऊपर दिए गए कोड ब्लॉक को रन करने के बाद, आपके सिस्टम में "sample.txt" नाम का एक फाइल बन जाएगा जिसमें आपका टेक्स्ट लिखा होगा। आप इसको नोटपैड या किसी दूसरे टेक्स्ट एडिटर में खोलकर देख सकते हैं।

## गहराई में जाएं

फाइल को खोलने के लिए "open" कार्य के कुछ अन्य पैरामीटर भी होते हैं, जैसे "w" या "r" जिससे हम फाइल को लिखने या पढ़ने का मोड चुन सकते हैं। अधिक गहराई में जाने के लिए, आप [यहां](https://docs.python.org/3/tutorial/inputoutput.html#reading-and-writing-files) दिए
