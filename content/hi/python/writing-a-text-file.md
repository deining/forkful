## क्या और क्यों?
एक पाठ फ़ाइल लिखना क्या है और क्यों प्रोग्रामर्ज़ इसे करते हैं? एक पाठ फ़ाइल एक टेक्स्ट दस्तावेज़ होता है जो टेक्स्ट संपादक में दर्ज किए गए पाठ को सहेजने के लिए उपयोग होता है। इससे प्रोग्रामर्ज़ अपने कोड को सुरक्षित रख सकते हैं और अन्य डेवलपर्ज़ के साथ अपने कोड को साझा कर सकते हैं। 

## कैसे करें:
```Python
# एक पाठ फ़ाइल बनाएं
f = open("myfile.txt", "w")
# फ़ाइल पर टेक्स्ट लिखें
f.write("यह टेक्स्ट फ़ाइल सेव हुई है!")
# फ़ाइल बंद करें
f.close()
```

```Python
# एक मौजूदा फ़ाइल में टेक्स्ट लिखें
f = open("myfile.txt", "a")
f.write("अपना टेक्स्ट जोड़ें!")
f.close()
```

```Python
# फ़ाइल से मौजूदा टेक्स्ट देखें
f = open("myfile.txt", "r")
print(f.read())
# फ़ाइल से एक कतार को सामने लाएं
print(f.readline())
f.close()
```

उपरोक्त कोड द्वारा एक पाठ फ़ाइल बनाई जा सकती है और इसमें टेक्स्ट लिखा जाएगा। हम "w" या "a" मोड के साथ फ़ाइल खोल सकते हैं, जहां "w" नई फ़ाइल बनाएगा और "a" मौजूदा फ़ाइल में टेक्स्ट जोड़ेगा। फ़ाइल को बनाने के बाद, हम "write()" कहेंगे जो फ़ाइल पर टेक्स्ट लिखेगा। फ़ाइल का उपयोग करके हम "read()" और "readline()" भी कर सकते हैं जो फ़ाइल से टेक्स्ट को पढ़ेंगे।

## गहराई में जाएं:
एक पाठ फ़ाइल लिखने की परंपरा प्रारम्भिक कम्प्यूटिंग के दौरान हुई। उस समय, पाठ फ़ाइलें डिस्क पर स्टोर होती थीं और डिस्क स्पेस कम था, इसलिए परंपरागत फ़ाइलों भर सक्रिय मेमोरी में लोड होते थे। आज की दिनचर्या में, हम नए तरीके से डेटा को स्टोर कर सकते हैं जैसे डेटाबेस या वेब सर्विस। हालांकि, पाठ फ़ाइल का उपयोग अभी भी डेटा को सुरक्षित रखने और साझा करने का एक सरल तरीका है। प्रोग्रामर्ज़ उन्हें अक्सर प्रोग्राम के साथ एक्सपोर्ट करते हैं जो टेक्स्ट फाइल को अन्य डेवलपर्ज़ को साझा करने की अनुमति देता है।
