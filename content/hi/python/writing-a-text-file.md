# क्यों
कई लोगों को अपने काम को अस्थाई तौर पर सेव करने के लिए टेक्स्ट फ़ाइलों की जरूरत होती है। इसलिए, हिंदी में इस ब्लॉग पोस्ट में हम आपको पांच सरल चरणों के माध्यम से पांच सरल चरणों के माध्यम से बताएंगे कि आप टेक्स्ट फ़ाइल कैसे बना सकते हैं।

# कैसे करें
टेक्स्ट फ़ाइल बनाने के लिए, हमें सिर्फ कुछ ही लाइन कोड के साथ काम करना होगा। नीचे दिए गए उदाहरण में, हमने एक पाठ को टेक्स्ट फ़ाइल में लिखने के बारे में दिखाया है। आप अपना पाठ बदल सकते हैं और उदाहरण को अपनी आवश्यकताओं के अनुसार संशोधित भी कर सकते हैं।

```python
# टेक्स्ट फ़ाइल बनाएं
f = open("स्वागत.txt", "w")

# लिखें
f.write("हमारे ब्लॉग पोस्ट में आपका स्वागत है!")

# बंद करें
f.close()
```
जब आप इस कोड रन करेंगे, तो आपको नए टेक्स्ट फ़ाइल "स्वागत.txt" में यह पाठ दिखेगा:

```
हमारे ब्लॉग पोस्ट में आपका स्वागत है!
```
