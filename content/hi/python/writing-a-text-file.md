## क्यों

 टेक्स्ट फाइल लिखने का एक निर्देश होने के साथ-साथ, उन लोगों के लिए भी महत्वपूर्ण है जो पाइथन में प्रोग्रामिंग सीख रहे हैं और अपने कोड को सहेजने के लिए एक स्थान की जरूरत होती है।

## कैसे करें

टेक्स्ट फाइल को पाइथन में लिखने के लिए, आपको एक नया फ़ाइल बनाना होगा और उसे `.txt` या `.py` की फाइल एक्सटेंशन के साथ सेव करना होगा। फिर आप `open()` फ़ंक्शन का उपयोग करके उस फाइल को खोल सकते हैं और `write()` फ़ंक्शन के साथ अपना कोड लिख सकते हैं। अंत में, आपको फाइल को `close()` करना न भूलें।

```Python
# टेक्स्ट फाइल बनाना
file = open("file.txt", "w")

# कोड लिखना
file.write("यह है एक टेक्स्ट फाइल!")

# फाइल को क्लोज़ करना
file.close()
```

आपके द्वारा लिखी गई पाइथन कोड का संग्रह टेक्स्ट फाइल में सहेज लिया गया है। अब आप इसे पुनः खोलकर पढ़ सकते हैं।

## गहराई में जाएं

टेक्स्ट फाइल लिखने के लिए `open()` और `write()` फ़ंक्शन के अलावा भी आप `read()` और `append()` फ़ंक्शन का उपयोग कर सकते हैं। `read()` फ़ंक्शन आपको फाइल में से डेटा पढ़ने की अनुमति देता है और `append()` फ़ंक्शन आपको फाइल में नए डेटा को जोड़ने की अनुमति देता है।

```Python
# फाइल को खोलना
file = open("file.txt", "r+")

# डेटा पढ़ना
print(file.read())

# नया डेटा जोड़ना
file.append("\nयह एक और लाइन है।")
