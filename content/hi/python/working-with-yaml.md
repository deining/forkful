## आर्ली एक घटना

आपने संभवतः पाइथन प्रोग्रामिंग के बारे में सुना होगा, लेकिन क्या आपने याडीएम तक का पाइथन आवेदन
बारे में सुना है? याडीएम पाइथन का एक इंस्क्रिप्टिंग लैंग्वेज है जो संरचनात्मक डेटा को संगठित करने के लिए उपयोग किया जाता है। इस ब्लॉग पोस्ट में हम आपको याडीएम के साथ पाइथन प्रोग्रामिंग का एक पूर्ण अवलोकन प्रदान करेंगे।

## क्यों

याडीएम का उपयोग डेटाबेस या कॉन्फ़िगरेशन फाइल्स को संरचित करने के लिए किया जाता है। यह आसानी से पढ़ने, लिखने और संशोधित किया जा सकता है, जिससे वह अन्य फाइल फार्मेटों की तुलना में बेहतर है। इसलिए, याडीएम को सीखने के लिए पाइथन एक अच्छा विकल्प है।

## कैसे करें

तो आइए हम पाइथन में याडीएम का उपयोग करना सीखते हैं। पहले हम `pyyaml` पैकेज को इंस्टॉल करेंगे। यह ऑपरेशन निम्न लाइन द्वारा किया जा सकता है:

```
pip install pyyaml
```

अब हम `import` का उपयोग करके `yaml` पैकेज को इंस्टॉल करेंगे:

```
import yaml
```

अब हम एक सिंपल याडीएम फ़ाइल बनाएंगे `example.yml` नाम से जो निम्न जैसा होगा:

```
name: John
age: 25
fav_food:
  - pizza
  - ice cream
```

यह अब हमारे याडीएम फ़ाइल `example.yml` को पार्स करके उसका डाटा स्ट्रक्चर रिटर्न करेगा। निम्न जैसा:

```
{'name': 'John', 'age': 25, 'fav_food': ['pizza', 'ice cream']}
``
