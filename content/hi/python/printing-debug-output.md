## क्यों

डीबगआउटपुश्टि छपाई क्यों आवश्यक होती है? यह एक आम समस्या है जो न केवल नए लेखकों को अनुभव करना पड़ता है, बल्कि पुराने लेखकों को भी साधनों और कोड में बदलाव को समझने में मदद करती है।

## कैसे करे

```
कोड निम्नलिखित हे:
कोड = अगर गलती असफ़ल हुई:
         डिबगआउटपुश्टि ("गलती हुई!")
अन्यथा:
         डिबगआउटपुश्टि ("सही काम हुआ!")
```

आप अपने कोड में ```डिबगआउटपुश्टि ()``` स्टेटमेंट को जोड़कर अपने प्रोग्राम के विभिन्न चरणों पर छपाई कर सकते हैं। यह आपको गलतियों को पकड़ने और संशोधित करने में मदद करता है। आप भी अपने कोड में अपनी खुद की अनुप्रयोगों को शामिल करके डिबग कर सकते हैं।

उदाहरण के लिए, यदि आपका उद्देश्य यह है कि आपका कोड किसी संख्या के वर्ग को गणना करना है, तो आप अपने कोड में निम्नलिखित स्निपेट को शामिल करके डिबग कर सकते हैं:

```
कोड = int (इनपुट ("एक संख्या दर्ज करें।"))
यदि कोड > 0:
    डिबगआउटपुश्टि (कोड * कोड)
अन्यथा:
    डिबगआउटपुश्टि ("गलत दर्ज किया गया।")
```

यदि आप कोड को चलाने पर फिर भी समस्या हो, तो आप आसानी से पता लगा सकते हैं कि आपका द्वारा दी गई उपयोगी संख्या के साथ कुछ गलत है।
