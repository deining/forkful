## क्यों

हालाँकि हम प्रोग्रामिंग में बहुत सारे टूल्स और फंक्शन्स के माध्यम से हमारे कोड को अद्यतन और बग्स से मुक्त रख सकते हैं, लेकिन कभी-कभी हमें विशेष प्रकार के डेटा या यूजर के प्रवेश से सम्बंधित असामान्य या त्रुटि स्थितियों को हैंडल करना होता है। इस जानकारी को हमे प्रोग्राम के दौरान दिखाना होता है ताकि रणनाजी के दौरान हमारे कोड के विशेष लिप्तंर करने में हमें मदद मिल सके।

## कैसे करे

जब हम अपने कोड में उपयोगकर्ता से कुछ डेटा प्राप्त करते हैं या असामान्य स्थितियों को हैंडल करने की जरूरत होती है, तो हम उस जानकारी को `stderr` स्ट्रीम के माध्यम से डिरेक्टली लिख सकते हैं।

```Python
import sys

name = input("कृपया अपना नाम डालें: ") # यहा प्रयोगकर्ता से डेटा प्राप्त होगा

if name == "John": # कुछ सामान्य यूजर के प्रवेश से संबंधित कोड
    print("आपको स्वागत है, John!")
else:
    sys.stderr.write("त्रुटि: अवैध नाम हैं।") # त्रुटि स्थिति को stderr स्ट्रीम पर लिखा जाएगा
```

आपको ध्यान देने की जरूरत है कि `sys.stderr.write` फंक्शन से हम सदंताओं का उपयोग करते हैं, जो `stdout` स्ट्रीम के लिए उपलब्ध नहीं होते हैं।
