## क्यों

एक आम Python प्रोग्रामर के रूप में, आपने यह शब्द सुना होगा, "standard error"। आपको शायद यह समझने में मदद कर सकता है कि जब हम Python में कोड करते हैं, तो हमारे सामने कुछ गलतियां (errors) आ सकती हैं। यदि हमारा कोड संभवतः सही नहीं है, तो हमारे स्क्रिप्ट द्वारा छपी गई सभी गलतियों को पूर्णतापूर्वक दखाने के लिए हम standard error का उपयोग करते हैं।

## कैसे

अब जब आप जानते हैं कि standard error क्या है, आपको संज्ञेय लाइन से बेहतर कैसे लिखा जाता है, इसका पता चलना बहुत महत्वपूर्ण है। हम आपको एक साधारण उदाहरण दिखाने के लिए शॉर्टकट का उपयोग करेंगे। तो, हम अपने कोड ब्लॉक को "```Python" और "```" के बीच में चिह्नित करते हैं।

```Python
# आपका कोड इस स्थान पर होना चाहिए
print("नमस्ते, यह कोड सही नहीं है!")
```

जैसा कि आप देख सकते हैं, हमने `print` स्टेटमेंट का उपयोग करके स्क्रिप्ट में एक गलती प्रदर्शित की है और इसे standard error के माध्यम से छापा है। यह मामल थोड़ी समस्या है, लेकिन कुछ अधिक जटिल स्क्रिप्टों के साथ काम करने के दौरान, standard error आपके लिए महत्वपूर्ण साबित हो सकता है।

## गहराई में जाएं

हमारे पहले उदाहरण में, हमने Python `print` स्टेटमेंट का उपयोग किया जिससे हम standard error को छाप सके। लेकिन, यदि हम सीखने की योजना बनाते हैं, तो हमें `sys` मॉड्यू
