## क्यों

कमांड लाइन आर्ग्यूमेंट पढ़ने का कार्य निष्पादित करने के लिए कुछ लोगों को लग सकता है कि यह काफी जटिल हो सकता है। हालाँकि, यह आपको एक प्रोग्राम को बहुत सुविधाजनक ढंग से निष्पादित करने देता है। इस ब्लॉग पोस्ट में हम बताएंगे कि आप इस कौशल को कैसे सीख सकते हैं और इसके क्या फायदे हैं।

## कैसे करें

आप कमांड लाइन आर्ग्यूमेंट्स को पढ़ने के लिए `argparse` मॉड्यूल का उपयोग कर सकते हैं। इसके लिए सबसे पहले आपको `argparse` मॉड्यूल को आयात करना होगा। फिर, `argparse` के `ArgumentParser` क्लास का एक ऑब्जेक्ट बनाएं। इसके बाद, आप अपने प्रोग्राम के लिए आवश्यक आर्ग्यूमेंट्स को डिफाइन कर सकते हैं। आप भी अपने आर्ग्यूमेंट्स के लिए नाम, डेस्क्रिप्शन और डिफॉल्ट वैल्यू को सेट कर सकते हैं। अंत में, आप `parse_args()` फंक्शन को उपयोग करके आर्ग्यूमेंट्स को प्रोसेस कर सकते हैं। नीचे दिए गए कोड ब्लॉक में आप `argparse` का उपयोग करके सरल प्रोग्राम बना सकते हैं:

```Python
import argparse

# ArgumentParser का एक ऑब्जेक्ट बनाएं
parser = argparse.ArgumentParser()

# आपके प्रोग्राम के लिए आवश्यक आर्ग्यूमेंट्स डिफाइन करें
parser.add_argument("name", help="आपका नाम दर्ज करें")

# अपने आर्ग्यूमेंट्स को प्रोसेस करें
args = parser.parse_args()

# डिफॉल्ट वैल्यू का उपयोग करके डेस्क्रिप्शन प्रिंट करें
