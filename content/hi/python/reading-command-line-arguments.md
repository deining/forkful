## क्यों
क्या आपने कभी सोचा है कि आप Python में कमांड लाइन आर्ग्यूमेंट्स का उपयोग कर सकते हैं? यदि नहीं, तो यह आपको कंप्यूटर पर काम करने का नया तरीका सीखने का एक शानदार मौका हो सकता है।

## कैसे
जब किसी भी Python स्क्रिप्ट को कॉम्पाइल किया जाता है, तो उसमें से कमांड लाइन आर्ग्यूमेंट्स को भी पास किया जा सकता है। यह आपको स्क्रिप्ट को निर्धारित डेटा या मापदंडों के साथ चलाने की अनुमति देता है। नीचे दिए गए कोड ब्लॉक की मदद से आप इसका उपयोग सीख सकते हैं:

```Python
import sys

# आपका स्क्रिप्ट किसी भी कमांड लाइन आर्ग्यूमेंट्स को प्रिंट करेगा
print("कमांड लाइन आर्ग्यूमेंट्स:", sys.argv)
```

जब आप उपरोक्त कोड को चलाएंगे, तो आपको स्क्रिप्ट के नाम और कमांड लाइन आर्ग्यूमेंट्स की एक सूची मिलेगी। आप इसे अपनी मर्जिंग और डेटा एनालिसिस टास्कों में उपयोग कर सकते हैं।

## डीप डाइव
Python कमांड लाइन आर्ग्यूमेंट्स का उपयोग करने के लिए, दो वैश्विक चरणों में अपनी स्क्रिप्ट फ़ाइल को ़कमांड लाइन आर्ग्यूमेंट्स के साथ चलाना होगा। पहले, आपको स्क्रिप्ट को सीधे Python इंटरप्रीटर में पास करना होगा (`python script.py`)। और दूसरा, आपको स्क्रिप्ट फ़ाइल को `chmod +x script.py` कमांड की मदद से एक्सक्यूटेबल बना देना होग
