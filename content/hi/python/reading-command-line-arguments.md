# इसलिए
यदि आप पायथन प्रोग्रामिंग में एक नया शुरुआती प्रोग्रामर हैं, तो आपने शायद अपने साथ कमांड लाइन आर्ग्यूमेंट्स का सामना किया होगा। यह आपके प्रोग्राम को चलाने के लिए प्रयुक्त प्रविष्टियों का समूह होता है जो आपके कोड को स्विच और योग्यताओं को बाहरी स्रोत से डालने की अनुमति देते हैं। इस ब्लॉग पोस्ट में मैं आपको बताऊंगा कि कमांड लाइन आर्ग्यूमेंट्स क्यों महत्वपूर्ण हैं और आपको इसका उपयोग कैसे करना चाहिए।

# कैसे
कमांड लाइन आर्ग्यूमेंट्स का उपयोग पायथन में काफी आसान है। सबसे पहले, आपको `sys` मॉड्यूल को इम्पोर्ट करना होगा जो `argv` नामक एक लिस्ट प्रदान करता है जो आपके प्रोग्राम के साथ दिया गये सारे कमांड लाइन आर्ग्यूमेंट्स को संग्रहीत करता है। यह बहुत स्पष्ट और अधिकतम सुविधा प्रदान करने के लिए इस्तेमाल किया जा सकता है। नीचे एक उदाहरण दिया गया है जो आपको यह स्पष्ट करने में मदद करेगा कि कमांड लाइन आर्ग्यूमेंट्स कैसे प्राप्त किए जाते हैं।

```Python
import sys

args = sys.argv
print(args)
```
