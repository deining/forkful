## क्यों

कई बार प्रोग्रामिंग में, हमे किसी निश्चित डायरेक्ट्री में फाइलें क्रिएट और ऑपरेशन करने की आवश्यकता होती है। लेकिन कभी-कभी हमें उस डायरेक्ट्री की मौजूदगी की जाँच करने की आवश्यकता भी होती है। इसलिए हम पाएंगे कि Python में डायरेक्ट्री की मौजूदगी की जाँच क्यों करने की जरूरत होती है।

## कैसे करें

जब हमारे पास एक फ़ोल्डर नाम होता है तब हम उसकी मौजूदगी की जाँच करने के लिए `os.path.exists()` फ़ंक्शन का उपयोग करते हैं। यह फ़ंक्शन दिए गए फ़ोल्डर के नाम को मानक पथ में लेता है और उसकी मौजूदगी को चेक करता है। यदि फ़ोल्डर मौजूद है तो यह `True` लौटाएगा और अगर फ़ोल्डर मौजूद नहीं है तो यह `False` लौटाएगा।

```Python
import os

# फ़ोल्डर का नाम
folder_name = "फ़ोल्डर का नाम"

# फ़ोल्डर की मौजूदगी चेक करना
if os.path.exists(folder_name):
    print("फ़ोल्डर मौजूद है।")
else:
    print("फ़ोल्डर मौजूद नहीं है।")
```

आउटपुट:
```
फ़ोल्डर मौजूद है।
```
