# क्यों
कई बार हमारे पास अपने कंप्यूटर में दस्तावेज़ या फाइलें सहेजने के लिए एक आम स्थान होता है, जिसे हम directory के रूप में जानते हैं। लेकिन कभी-कभी हमें जानना भी पड़ता है कि क्या यह डायरेक्टरी हमारे सिस्टम में मौजूद है या नहीं। ऐसे में हमें जांचने के लिए एक आसान तरीका चाहिए होता है जो हमें बताए कि क्या यह डायरेक्टरी मौजूद है या नहीं।

## कैसे करें
डायरेक्टरी की जांच करने के लिए हम `os` मॉड्यूल का उपयोग कर सकते हैं। निम्नलिखित उदाहरण में, हम एक फाइल से उसकी पूरी पथ बनाकर उसे `os.path.exists()` फ़ंक्शन में पास करते हैं। अगर यह डायरेक्टरी मौजूद होती है, तो यह `True` रिटर्न करेगा, नहीं तो `False`।
```Python
import os

file_path = "/Users/username/Documents/sample.txt" # अपने सिस्टम पर उपयोगकर्ता नाम को बदलें
dir_exists = os.path.exists(file_path)

print("क्या डायरेक्टरी मौजूद है? ", dir_exists)
```

अगर आपकी डायरेक्टरी मौजूद होती है, तो निम्न आउटपुट मिलेगा:
```
क्या डायरेक्टरी मौजूद है? True
```
