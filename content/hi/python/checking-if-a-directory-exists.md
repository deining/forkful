## क्यों

कई बार हमारे पास अपने कंप्यूटर पर अपनी काम को सुचारू रूप से संगठित रखने के लिए कई फोल्डर या डायरेक्ट्री होती हैं। कभी-कभी हमें आवश्यकता होती है कि हम अपने कोड में किसी निश्चित डायरेक्ट्री को एक्सेस करें, और उनमें कुछ ऑपरेशन करें। यदि हम पहले से ही उस डायरेक्ट्री को बनाएंगे, तो हमें कोड के दौरान किसी समस्या का सामना नहीं करना पड़ेगा। इसलिए, इस लेख में हम जानेंगे कि अपने कोड में दर्जित डायरेक्ट्री का अस्तित्व कैसे जांचें।

## कैसे

पाइथन में, हम `os` मॉड्यूल का उपयोग करके आसानी से डायरेक्ट्री का अस्तित्व जांच सकते हैं। पहले हम `os` मॉड्यूल को इम्पोर्ट करते हैं।

```Python
import os
```

अब हम `os.path.exists()` फ़ंक्शन का उपयोग करके दर्जित डायरेक्ट्री का अस्तित्व जांच सकते हैं। यह फ़ंक्शन दिए गए पथ पर एक बूलियन वापस करेगी, जो बताता है कि दिए गए पथ पर डायरेक्ट्री है या नहीं। यदि दिए गए पथ पर डायरेक्ट्री मौजूद है, तो फ़ंक्शन `True` वापस करेगा, अन्यथा `False`। नीचे दिए गए उदाहरण में, हम `Documents` फ़ोल्डर के अस्तित्व को जाँच रहे हैं।

```Python
if os.path.exists("./Documents"):
    print("दर्जित डायरेक्ट्री मौजूद है।")
else:
    print("दर्जित डायरेक्ट्री मौजूद नहीं है।")
```

यदि `Documents` फ़ोल्डर मौजू
