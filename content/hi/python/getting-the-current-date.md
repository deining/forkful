Python में वर्तमान दिनांक प्राप्त करना क्यों जरूरी है?

दुनिया में हर एक काम करने के लिए हमें अपनी कार्यप्रणाली में अनुकूलता और तारीखों का पालन जरूरी होता है। कुछ समय पूर्व, हमारे पास किसी भी काम को करने के लिए हमें एक अछुते दिनांक की आवश्यकता होती थी। लेकिन अब जब हम Python का सुविधा सही तारीख प्राप्त करने के साथ हमारे काम को पूरा कर सकते हैं तो हमारे लिए समस्या का समाधान हो गया है।

कैसे करें: 

```
Python में वर्तमान दिनांक प्राप्त करने के लिए हम datetime और date लाइब्रेरी का इस्तेमाल कर सकते हैं। नीचे दिए गए कोड ब्लॉक में हम इसका उपयोग कैसे कर सकते हैं।

```
### Python कोड ब्लॉक:

```Python
# import करें datetime और date लाइब्रेरी
from datetime import datetime, date

# वर्तमान दिन मान प्राप्त करें
today = date.today()
print("आज का दिनांक है:", today)

# वर्तमान दिन और समय मान प्राप्त करें
now = datetime.now()
print("आज का दिनांक और समय है:", now)
```

### स्क्रीन आउटपुट:

```
आज का दिनांक है: 2021-09-01
आज का दिनांक और समय है: 2021-09-01 14:30:00.100500
```

गहराई में जाएं:

Python में वर्तमान दिनांक प्राप्त करने के लिए उपलब्ध लाइब्रेरी काफी हैं। हमने उपरोक्त कोड ब्लॉक में सिर्फ एक उदाहरण दिया है, लेकिन यदि आपको गहराई में जाना है तो आप इस लाइब्रेरी के अन्य विशेषताओं को भी जान सकते हैं। Python में datetime और date लाइब्रेरी में विभिन्न
