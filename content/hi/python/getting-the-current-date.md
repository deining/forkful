# क्यों

आजकल, कम्प्यूटर प्रोग्रामिंग में ज्यादातर काम हमारे आस-पास के समय को आसान बनाने के लिए किए जाते हैं। तारीख और समय के आधार पर विभिन्न काम करना बहुत आसान हो जाता है। यही कारण है कि वर्तमान तारीख जानना एक बहुत उपयोगी प्रैक्टिस है।

# कैसे करें

मैं आपको Python इंटरप्रीटर का उपयोग करके वर्तमान तारीख प्राप्त करने के लिए दो विधियां बताने जा रहा हूं।

## विधि 1:

```
import datetime
now = datetime.datetime.now()
print("वर्तमान तारीख और समय:", now)
```

आपको इस कोड को अपने कंप्यूटर पर रन करने के लिए Python इंटरप्रीटर की आवश्यकता होगी। कोड को रन करने के बाद, आपको आपके कंप्यूटर का वर्तमान तारीख और समय प्रदर्शित किया जाएगा।

## विधि 2:

```
from datetime import date
today = date.today()
print("वर्तमान तारीख:", today)
```

इस कोड को दूसरी विधि की तरह अपने कंप्यूटर पर रन करने से भी आपको वर्तमान तारीख प्रदर्शित किया जाएगा। आप किसी भी तारीख और समय फॉर्मेट को अपनी आवश्यकताओं के अनुसार बदल सकते हैं। यदि आप अधिक जानकारी चाहते हैं, तो आप Python डॉक्यूमेंटेशन का इस्तेमाल कर सकते हैं।
