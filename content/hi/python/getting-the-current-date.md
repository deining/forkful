## इसकी महत्व क्या है?

अगर आप भी एक पाइथन प्रोग्रामर हैं, तो आपने अक्सर अपने कोड में वर्तमान तिथि को प्रिंट करते हुए देखा होगा। वर्तमान तिथि अक्सर किसी प्रोग्राम को चलाने की दृष्टि से महत्वपूर्ण होती है क्योंकि यह हमेशा बदलती रहती है और अपने कोड को अपडेट करने का संकेत देती है। इसलिए, अगर आप भी इस विषय में अपने ज्ञान को बढ़ाना चाहते हैं, तो आगे पढ़िए हमारा पोस्ट।

## कैसे करें?

अगर आप भी पाइथन में वर्तमान तिथि प्रिंट करना चाहते हैं, तो आपको सबसे पहले `datetime` मॉड्यूल को इम्पोर्ट करना होगा। नीचे दिए गए कोड ब्लॉक में हमने इसका उदाहरण दिया है जिससे आप आसानी से समझ सकते हैं।

```python
import datetime

current_date = datetime.datetime.now()
print("वर्तमान दिनांक है:", current_date)
```

इस कोड का आउटपुट निम्नलिखित तरह का होगा:

```
वर्तमान दिनांक है: 2020-12-01 15:10:20.739315
```

## गहराईगामी जानकारी

पाइथन में वर्तमान तिथि प्राप्त करने के लिए हम `datetime` मॉड्यूल का `now()` फंक्शन का उपयोग करते हैं। जैसा कि आपने ऊपर दिखाया है, यह फंक्शन दोनों तारीख और समय को एक साथ दिखाता है। लेकिन अगर आप चाहें तो आप सिर्फ तारीख या समय को भी सेपरेटली प्रिंट कर सकते हैं। इसके लिए आप `date()` और `time()` फंक्शन का उपयोग कर सकते हैं।
