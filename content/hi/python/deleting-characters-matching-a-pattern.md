## क्यों

कई बार हमारे पास एक स्ट्रिंग होता है जिसमें कुछ एक्स्ट्रा चरित्र हो सकते हैं जो हमें नहीं चाहिए। उदाहरण के लिए, एक ईमेल पते में '@' का प्रयोग करना हो सकता है लेकिन हमें उसे हटाना चाहिए जब हम उसे अपने कोड में प्रयोग करते हैं। ऐसे मामलों में, हम पाठकों को बताएंगे कि वे किस प्रकार से एक दिया गया पैटर्न मिलाने वाले चरित्रों को हटाने का उपयोग कर सकते हैं।

## कैसे करें

मैं पाइथन कोड का उदाहरण देशकता हूँ। सबसे पहले हम एक स्ट्रिंग बनाएंगे जिसमें हमारे द्वारा हटाने के लिए प्रयोग करने के पैटर्न के साथ कुछ एक्स्ट्रा चरित्र होंगे। उसके बाद हम उन चरित्रों को हटाएंगे जिन्हें हम नहीं चाहते।

```Python
# स्ट्रिंग बनाना
string = "P@y#th%on!"

# लूप चलाकर चरित्रों को हटाना
new_string = ""
for char in string:
  if char not in "@#$%":
    new_string += char

# नया स्ट्रिंग प्रिंट करना
print(new_string)

# आउटपुट: Python!
```

ऊपर दिए गए कोड में, `"@"`, `"#"`, `"%"`, और `"$"` चरित्रों को हटाकर, एक नया स्ट्रिंग बनाया गया है। आप भी दिए गए कोड को अपने पैटर्न के अनुसार बदल सकते हैं और अनचाहे चरित्रों को हटा सकते हैं।
