## क्यों

अगर आपने Python कोडिंग में किसी पैटर्न के मिलते जुलते चरित्रों को डिलीट करने की कोशिश की है, तो यह आपके कोड को छोटा और साफ करने में मदद कर सकता है।

## कैसे करें

```Python
text = "हिन्दी दिवस की ढेर सारी शुभकामनाएं"
pattern = " सारी "
clean_text = text.replace(pattern, " ")
print(clean_text)
```

आपको पहले `replace()` फंक्शन का उपयोग करके चरित्रों को जोड़ने पर ध्यान देना होगा। तो यह आपके परिणाम को निर्दिष्ट पैटर्न के हिसाब से बदल देगा। इससे आपको चरित्रों को हटाने की आवश्यकता नहीं होगी।

```Python
import re
text = "हिन्दी दिवस की ढेर सारी शुभकामनाएं"
pattern = re.compile(r"सारी")
clean_text = pattern.sub(" ", text)
print(clean_text)
```

यदि आपके पास बहुत सारे टेक्स्ट है और आप इसे बदलना चाहते हैं, तो आप `re` मॉड्यूल का उपयोग कर सकते हैं। इससे आपको बहुत सारे पैटर्न और उनके साथ संबंधित कार्य करने की अनुमति होती है।

## गहराई में जाएं

चरित्रों को पैटर्न के आधार पर डिलीट करना काफी सामान्य और उपयोगी काम है। इससे आपको उन चरित्रों से छुटकारा मिलता है जो आपके कोड के लिए उपयोगी नहीं हैं। इसके अलावा, यह आपके कोड को छोटा और साफ रखने में मदद कर सकता है।
