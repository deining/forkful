## क्यों

कहाँ भी हो, हमेशा चुनौतियों का सामना करना पड़ता है। जब हम कोडिंग करते हैं, तो कभी-कभी हमें एक ऐसी स्थिति से गुज़रना पड़ता है जहाँ हमें शब्दों को छोड़ने या हटाने की आवश्यकता होती है। इस समस्या को हल करने के लिए, हम एक वर्ण समूह का इस्तेमाल करके उसे हटा सकते हैं, जिसे हम 'पैटर्न' कहते हैं। इस स्तर पर हमारे पास ऐसे अनेक मौके हैं जहाँ हमें इस तरह के कोड का इस्तेमाल करने की जरूरत हो सकती है।

## कैसे करें

```python
# चरित्र समूह खोजना और हटाना
import re # समूह खोजने के लिए 're' मॉड्यूल इम्पोर्ट करें

my_string = “Hello, world! How are you?” # एक आधारभूत स्ट्रिंग बनाएं
pattern = “[oe]” # हटाना है जो चरित्रों का एक समूह है
new_string = re.sub(pattern, “”, my_string) # स्ट्रिंग में से समूह को हटाएं
print(new_string) # स्ट्रिंग के नए रूप को मुद्रित करें
```

उपरोक्त कोड का आउटपुट निम्न होगा:

```
Hll, wrld! Hw ar yu?
```

जैसा कि आप देख सकते हैं, `re.sub()` फ़ंक्शन हमारे लिए `oe` समूह के सभी चरित्रों का हटाव कर देता है। इस तरह से, हम किसी भी पैटर्न को जल्दी से हटा सकते हैं और अपने कोड को सरल और कामयाब बना सकते हैं।
