## क्यों?

एक वेब पेज को डाउनलोड करने का काम आपको उस पेज पर दी गई जानकारी को ऑफलाइन उपयोग करने की अनुमति देता है और नेटवर्क कनेक्शन की उपस्थिति के बिना भी परिणामों को देखने की सुविधा प्रदान करता है।

## कैसे करें?

```Python
import requests

# URL अनुरोध जमा करें
url = "https://www.example.com/"

# अनुरोध भेजें और प्रतिक्रिया प्राप्त करें
response = requests.get(url)

# प्रतिक्रिया के साथ कंटेंट प्राप्त करें
content = response.content

# कंटेंट के लिए एक फ़ाइल बनाएँ
with open("example.html", "wb") as f:
    f.write(content)

# निर्धारित फ़ाइल माप की समीक्षा करें
print("फ़ाइल का आकार:", len(content), "बाइट")
print("डाउनलोड सफलतापूर्वक पूरा हुआ।")
```

Output:

```
फ़ाइल का आकार: 3856 बाइट
डाउनलोड सफलतापूर्वक पूरा हुआ।
```

## गहराई से खोजें

अब आपको जानकारी हो जाएगी कि वेब पेज को डाउनलोड करने के लिए क्या होता है। यह "requests" नामक एक थर्ड पार्टी लाइब्रेरी का उपयोग करके किया जाता है और यह सरल सिंटैक्स की पेशकश करता है। आप अपनी आवश्यकताओं के अनुसार इसे समायोजित कर सकते हैं और विस्तृत परिमाण की फ़ाइलों को भी डाउनलोड कर सकते हैं।
