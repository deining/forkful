## क्यों

वेब पेज को डाउनलोड करना क्यों महत्वपूर्ण है? वेब के अधिकांश सामग्री केवल ऑनलाइन उपलब्ध होती है और कई बार हम उपलब्ध सामग्री का दौरा बार-बार करना नहीं चाहते हैं। वेब पेज को डाउनलोड करने से हम उसे ऑफलाइन उपयोग कर सकते हैं और उसमें से हमेशा उपलब्ध सामग्री का इस्तेमाल कर सकते हैं।

## कैसे करें

```Python
import requests

response = requests.get("https://www.example.com")
print(response.text)
```

इस कोड ब्लॉक में, हमने requests की Python लाइब्रेरी का उपयोग करते हुए example.com वेब पेज को डाउनलोड किया है और उसकी सामग्री को प्रिंट किया है। इस तरह से हम अपने जॉब कोड में भी वेब पेज को डाउनलोड करके उसकी सामग्री का इस्तेमाल कर सकते हैं।

## गहराई से जाएं

बहुत से उपयोगकर्ताओं के लिए एक अच्छा गहराई से जाना जावित है कि वेब पेज के विभिन्न अनुभागों को कैसे अलग-अलग करें। कई बार, हमें सिर्फ एक विशेष भाग की सामग्री की आवश्यकता होती है जैसे कि शीर्षक, सामग्री का अंश या एक तालिका। इसलिए हमने BeautifulSoup के माध्यम से वेब पेज को पार्स किया है और उससे आवश्यक डेटा को उद्धरण किया है।
