वजह: बेसिक प्रमाणीकरण के साथ एचटीटीपी अनुरोध भेजने में क्यों किसी को दिलचस्पी हो सकती है, इसके बारे में कुछ एक-दो संज्ञानवान वाक्यों में बताएं।

कैसे: "```Python
import requests

# यूआरएल को निर्दिष्ट करें
url = "https://www.example.com"

# उपयोगकर्ता नाम और पासवर्ड दर्ज करें
user = "username"
password = "password"

# अनुरोध भेजने के लिए बेसिक प्रमाणीकरण जोड़ें
response = requests.get(url, auth=(user, password))

# प्रिंट करें आउटपुट स्टेटस कोड
print(response.status_code)
```"

प्रारंभ में, हमने `requests` मॉड्यूल को इंपोर्ट किया है। यह हमें अनुरोध भेजने और प्राप्त करने की सुविधा देता है। अगला, हमें अपनी यूआरएल को निर्दिष्ट करना होगा। फिर, हम उपयोगकर्ता नाम और पासवर्ड दर्ज करेंगे जो हमें एचटीटीपी अनुरोध भेजने के लिए आवश्यक होते हैं। अंत में, हम `auth` पैरामीटर का उपयोग करके अनुरोध में बेसिक प्रमाणीकरण जोड़ते हैं। इसके बाद, हम `response` नामक वेरिएबल में अपने अनुरोध का प्रतिक्रिया प्राप्त करते हैं और स्टेटस कोड को प्रिंट करते हैं। जब हम इस कोड को चलाएंगे, हमें उपयोगकर्ता को सफलतापूर्वक प्रमाणित किया जाएगा और स्टेटस कोड 200 देखने को मिलेगा।
