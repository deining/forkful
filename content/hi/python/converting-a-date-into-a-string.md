##क्यों

क्या आपने कभी सोचा है कि मंगलवार, 20 अगस्त 2021 को 2021-08-20 या "20 अगस्त 2021" कैसे बनाया जाता है? यह कार्य जब दिनांक को स्ट्रिंग में बदलने के और भी कई उद्धरणों के लिए महत्वपूर्ण होता है। अगर आप Python में प्रोग्रामिंग सीख रहे हैं, तो आपको यह जानना अनिवार्य होगा कि यह कैसे होता है।

## कैसे करें

``` Python
# आवश्यक पैकेज इम्पोर्ट करें
import datetime

# दिनांक तथा समय को दर्शाने के लिए फॉर्मेट चुनें
date = datetime.datetime(2021, 8, 20)

# स्ट्रिंग में बदलें
print(date.strftime("%Y-%m-%d"))
print(date.strftime("%d %B %Y"))

# आउटपुट:
# 2021-08-20
# 20 अगस्त 2021

```

यहां हमने datetime लाइब्रेरी का उपयोग करके दिनांक तथा समय को स्ट्रिंग में बदला है। स्ट्रिंग में बदलने के लिए हमने strftime () फंक्शन का उपयोग किया है जिसमें हम फॉर्मेट को दर्शाने के लिए चुन सकते हैं। इसमें कई फॉर्मेट ऑप्शन होते हैं जैसे "% Y" जो साल को दर्शाता है, "% m" जो महीने को दर्शाता है और इत्यादि।

## गहराई में जाएं

जब हम datetime लाइब्रेरी का उपयोग करते हैं, तो हमें यह भी समझना जरूरी है कि यह दिनांक और समय को डेटा टाइप में ले आता है। और जब हम इसे स्ट्रिंग में बदलते हैं, तो यह फिर से वापस दिनांक और समय को डेटा टाइप में लाता है। इसलिए, यह एक शक्तिशाली और महत्वपूर्ण टूल है
