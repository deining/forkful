अभी दुनिया भर में इंटरनेट से जुड़े लाखों साइट हैं, जो उतने ही बहुत सारे सर्वर भी उपयोग में होते हैं। इससे आपको जानना आवश्यक है कि आप किस तरह से वेब उपयोगकर्ताओं को दिये गये संसाधनों तक पहुँच सकते हैं। यहाँ हम एक साधारण प्रश्न से आगे जाएंगे, कि हमारा उदेश्य यूएरएल, HTTP, और यह कैसे काम करता है।

## क्यों

अगर आप एक वेब डेवलपर हैं, तो संभावत: आपने इन शब्दों को सुना होगा - HTTP या उदराता स्रोत प्रोटोकाल। यह वेबसाइट, एप, या साइट ट्रैफिक को प्राप्त करने के लिए सर्वर से अनुरोध भेजता है। उदाहरण के लिए, जब आप इंटरनेट ब्राउज़र में वेबसाइट खोलते हैं, आपका ब्राउज़र उस वेबसाइट का एक HTTP अनुरोध भेजता है। यह अनुरोध साइट स्थापक द्वारा सम्पत्ति के संदेश, या वेबपेज को देखने के लिए जोड़ देता है।

## कैसे करें

अब हम आपको प्रोग्रामिंग करके, HTTP अनुरोध कैसे भेज सकते हैं दिखाएँगे। नीचे दिए हुए कोड ब्लॉक में आप एक साधारण HTTP अनुरोध भेजने के तरीका देख सकते हैं:

```Python
import request

url = 'https://www.example.com'
response = requests.get(url)

print(response.text)
```
यहाँ हमने `request` लाइब्रेरी को इंपोर्ट किया है जो हमें HTTP अनुरोध भेजने और जवाब प्राप्त करने के लिए शेष API प्रदान करती है। हमने एक `url` बनाया है और इसे `get()` फ़ं
