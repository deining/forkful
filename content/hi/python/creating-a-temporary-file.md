## क्यों

कभी-कभी हमें अपने कोड या अन्य एप्लिकेशन के लिए अस्थायी फाइल की आवश्यकता होती है। अस्थायी फाइल उस समय बनायी जाती है जब हमें स्थायी फाइल की जरूरत नहीं होती है, लेकिन हमें डेटा को कुछ समय तक स्टोर करना होता है। Python में, हम अस्थायी फाइल "tempfile" मॉड्यूल का उपयोग करके बना सकते हैं। 

## कैसे करें

अस्थायी फाइल बनाने के लिए, हम "tempfile" मॉड्यूल के फंक्शन `TemporaryFile()` का उपयोग कर सकते हैं। यह फंक्शन डिफॉल्ट रूप से अस्थायी फाइल को सिस्टम के टेम्पोरेरी फोल्डर में बनाता है। नीचे दिए गए उदाहरण में, हमें फाइल को रेड और लिखे करने के लिए "wb+" मोड में ओपन करना होगा।

```Python
import tempfile

# अस्थायी फाइल बनाएं
with tempfile.TemporaryFile(mode='wb+') as temp_file:
    # डेटा टाइप स्ट्रिंग में होना चाहिए
    data = "यह एक उदाहरण डेटा है।"
    # फाइल में लिखें
    temp_file.write(data.encode())
    # फाइल से डेटा पढ़ें
    temp_file.seek(0)
    temp_data = temp_file.read()
    # प्रिंट करें
    print(temp_data.decode())
```

उपरोक्त कोड का आउटपुट निम्नप्रकार होगा:

```
यह एक उदाहरण डेटा है।
```
