आज हम बात करने वाले हैं की पाइथन में अस्थायी फ़ाइल कैसे बनाया जाता है। दोस्तों, अस्थायी फ़ाइल सिर्फ कुछ समय के लिए होती है और बाद में खुद बखुद हट जाती है। ज्यादातर लोग इस्तेमाल तब करते हैं जब कुछ काम जल्दी से आया जाना हो और उन्हें फ़ाइल को manually हटाने की ज़रूरत हो। अब आप अस्थायी फ़ाइल कैसे बनाएंगे, आइए उसे जानते हैं। 

## कैसे करें 
```python 
# पाइथन में अस्थायी फ़ाइल बनाने के लिए आप निम्न रिक्ति के फंक्शन का इस्तेमाल कर सकते हैं 
tempfile.NamedTemporaryFile()

# यह आपको एक टेम्प फ़ाइल की object देगा जिसका इस्तेमाल आप contents डालने या फ़ाइलों को लिखने के लिए कर सकते हैं 
temp_file = tempfile.NamedTemporaryFile()

# अब आप फ़ाइल में कुछ लिख सकते हैं। यहाँ मैंने "Hello World!" लिखा है 
temp_file.write("Hello World!")

# इसके बाद आप फ़ाइल को ज़रूर हटा दें (हो सकता है कि आपसे भूल जाए) 
temp_file.close()

# अब आप देख सकते हैं कि फाइल हट गई होगी 
``` 

## विशेष 
बात अस्थायी फ़ाइलों की आई है तो आपको ध्यान देना चाहिए कि यह system-dependent हो सकती हैं। अर्थात आपके कंप्यूटर पर बनाई गई अस्थायी फाइल दूसरे कंप्यूटर पर नहीं चलेगी। और यह भी आशंका हो सकती है कि कोई आवश्यक फ़ाइल जो आपने Save किया था अस्थायी फाइल को हटाने से हो गई। इसलिए जब भी आप इस्तेमाल कर
