# क्यों?

क्या आपने कभी पाया है कि आपको कभी-न-कभी निगरानी के लिए जारी रखने के लिए टेम्पोररी फ़ाइल की जरूरत होती है? प्रोग्रामिंग में, हम अक्सर स्थायी फ़ाइलों के साथ काम करते हैं जो हमारे सिस्टम पर स्थाई रूप से स्थापित होती हैं। लेकिन कभी-कभी हमें साथ काम करने के लिए फ़ाइल का निर्माण करने की जरूरत हो सकती है, और यहां टेम्पोररी फ़ाइलों की उपयोगिता आती है।

# कैसे करें?

```python
# टेम्पोररी फ़ाइल बनाने के लिए, हम "tempfile" मॉड्यूल का उपयोग करते हैं
import tempfile

# एक ग्लोबल टेम्प फ़ोल्डर निर्दिष्ट करें, या छोड़ें ताकि सिस्टम अपने अपने टेम्पोररी फ़ोल्डर का उपयोग करें
tempfile.tempdir = "टेम्प फ़ोल्डर का पथ"

# नया टेम्पोररी फ़ाइल बनाएं और "temp_file" नामक चलचित्र संदर्भ जोड़ें
temp_file = tempfile.NamedTemporaryFile()

# टेम्प फ़ाइल का नाम प्रिंट करें
print(temp_file.name)

# टेम्प फ़ाइल को स्ट्रिंग के साथ लिखें
temp_file.write("यह एक टेम्प फ़ाइल है")

# टेम्प फ़ाइल को पढ़ें और उसे प्रिंट करें
temp_file.seek(0)
print(temp_file.read())

# टेम्प फ़ाइल को साफ़ करें
temp_file.close()
```

आउटपुट:
```
टेम्प फ़ोल्डर का पथ/tmp/randomstr
यह एक टेम्प फ़ाइल है
```

यहां से, हम देख सकते हैं कि टेम्पोररी फ़ाइल को बनाने, लिखने, पढ़ने और साफ़ करने के लिए हमें सिर्फ 2-3 लाइनों कोड की आव
