## क्यों

क्या आपने कभी पाठ्यक्रम या सॉफ्टवेयर शिक्षण में पढ़ा है कि अस्थायी फ़ाइलों का उपयोग क्यों किया जाता है? अस्थायी फ़ाइलें जीवित प्रोसेस को स्थायित्व देने के लिए उपयोगी हो सकती हैं, इसलिए इस मजेदार ब्लॉग पोस्ट में हम आज इस विषय पर बात करेंगे।

## कैसे करें

Python में अस्थायी फ़ाइलें बनाने के लिए, हम `tempfile` मॉड्यूल का उपयोग कर सकते हैं। नीचे दिए गए कोड ब्लॉक में, हम `tempfile.NamedTemporaryFile()` का उपयोग करके अस्थायी फ़ाइल बना सकते हैं।

```python
import tempfile

with tempfile.NamedTemporaryFile() as temp:
    temp.write(b"Hello world!") # temporary file content
    print(temp.name) # prints the temporary file path
```

इस उदाहरण में, हमने `temp` नामक संगठन का उपयोग करके फाइल में लिखा गया सामग्री और फिर फाइल का नाम प्रिंट किया है।

## गहराई में जाएं

निर्देशिका `tempfile` मॉड्यूल में आप और भी अनेक सुविधाएं पाएंगे जो अस्थायी फ़ाइलों का प्रबंधन करने में आपकी मदद कर सकती हैं। कुछ उदाहरण हैं:

- `tempfile.TemporaryFile()` - यह अस्थायी बाइनरी फ़ाइल के लिए है जिसे `with` ब्लॉक के अंत में स्वचालित रूप से डिलीट किया जाता है।
- `tempfile.mkstemp()` - यह अस्थायी फ़ाइल निर्दिष्ट नाम के साथ बनाता है। आप एक नाम भी दे सकते हैं, लेकिन यदि आप कोई नाम नहीं देते हैं तो यह एक युनिक नाम खुद ही बनाएगा।
