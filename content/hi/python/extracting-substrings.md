## क्या है और क्यों?
पाइथन में सबस्ट्रिंग निकालना क्या है, और क्यों प्रोग्रामर्स इसे करते हैं।

पाइथन में सबसे आसान तरीका सुनिश्चित करने के लिए है कि दिए गए स्ट्रिंग से केवल एक से अधिक अक्षरों को कैसे पकड़ा जाए। यह उपयोगकर्ताओं को दिए गए बड़े और जटिल स्ट्रिंग को कम, संगठित और पढ़ने में सुगम बनाता है।

## कैसे करें:
```python
# एक स्ट्रिंग से सबस्ट्रिंग निकालना
example_string = "पाइथन प्रोग्रामिंग"
print(example_string[2:6])
# output: थन 
```

```python
# सबस्ट्रिंग को अलग एक वेरिएबल में सेव करना
example_string = "पाइथन प्रोग्रामिंग"
new_string = example_string[0:6]
print(new_string)
# output: पाइथन
```

```python
# सबस्ट्रिंग को विभाजित करने एवं अन्य स्ट्रिंग से सम्मिलित करना
string1 = "मैंने "
string2 = "पाइथन सीखी है।"
new_string = string1 + string2
print(new_string)
# output: मैंने पाइथन सीखी है।
```

## गहराई में जाएं:
पाइथन में सबस्ट्रिंग निकालने की विस्तृत जानकारी जैसे (1) ऐतिहासिक पृष्ठभूमि, (2) प्रतिकारों, और (3) समापन विवरण।

कई अन्य प्रोग्रामिंग भाषाओं में भी स्ट्रिंग स्लाइसिंग की तकनीक ऊपर वर्णित तरीके से होती है। लेकिन, पाइथन में सबस्ट्रिंग निकालने की यह धाराप्रवाहित गणना है जो सीखने और लिखने में आसान है। सबस्ट्रिंग निकालने में, स्ट्रिंग के प्रत्येक अक्षर को उनके स्थान और आकार के हिसाब से निकाला जाता है।
