## क्यों
अगर आप पाइथन में काम करते हैं या इसे सीखना चाहते हैं, तो आपने स्ट्रिंग्स के बारे में सुना होगा। स्ट्रिंग्स, यानि अल्फाबेटिकल वर्णों की जोड़ी होती हैं जो हमारे पास पाठ या शब्दों को संगठित रूप में रखने की अनुमति देती हैं। अक्सर हमें एक ही चीज़ को विभिन्न तरीकों से दिखाना होता है, जैसे कि पीएसएल अभिव्यक्ति या उपयोगकर्ता का नाम या अन्य जानकारियां। इसलिए, हमें एक के बाद एक स्ट्रिंग दोनों को एक साथ जोड़ने की जरूरत होती है।

## कैसे करें
```Python
# सीधा उपयोग
string_1 = "नमस्ते" 
string_2 = "दुनिया"
merged_string = string_1 + string_2
print(merged_string) # नमस्ते दुनिया 

# एक वेरिएबल में जोड़ें
first_name = "अन्वेश"
last_name = "गुप्ता"
full_name = first_name + " " + last_name
print(full_name) # अन्वेश गुप्ता

# स्ट्रिंग + स्ट्रिंग
word = "पाइथन"
repeated_word = word + word + word
print(repeated_word) # पाइथनपाइथनपाइथन 
```

आप एक स्ट्रिंग को दूसरी स्ट्रिंग के साथ जोड़ने के लिए `+` ऑपरेटर का उपयोग कर सकते हैं। आप विभिन्न तरीकों से स्ट्रिंग जोड़ने के लिए भी निर्देश दे सकते हैं, जैसे स्ट्रिंग में अतिरिक्त रिक्त स्थानों को जोड़कर स्पष्टता को बढ़ावा देना।
