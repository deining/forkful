## क्यों
क्या आपने कभी अपने कोड में हजारों लाइन का टेक्स्ट ढूँढ़ने और बदलने का काम किया है? अगर हाँ, तो आपको ये पता होगा कि यह काम थोड़ा समय लेता है और इससे हमारी प्रोग्रामिंग गतिविधियों को बाधित कर सकता है। पाइथन में टेक्स्ट को ढूँढ़ने और बदलने का सबसे आसान तरीका है "खोज और बदल" फंक्शन का उपयोग करना। इससे आप आसानी से टेक्स्ट को खोजकर उसे बदल सकते हैं और अपनी कोडिंग सेसन को अधिक फोकस कर सकते हैं।

## कैसे करें
यदि आप पाइथन में टेक्स्ट को खोज और बदलने के बारे में जानना चाहते हैं, तो आपको सबसे पहले "re" मॉड्यूल को इम्पोर्ट करना होगा। इसके बाद आप निर्दिष्ट टेक्स्ट या रेक्सप्रेशन के साथ "re.sub()" फंक्शन का उपयोग करके उस टेक्स्ट को आसानी से बदल सकते हैं। नीचे दिए गए कोड ब्लॉक में आपको इसका उदाहरण दिया गया है। आप अपने विशेष स्थानिक टेक्स्ट या रेक्सप्रेशन को अपनी जरूरत के अनुसार बदल सकते हैं।

```Python
import re

# टेक्स्ट खोज और बदल
text = "मेरा नाम है जॉन"
new_text = re.sub("जॉन", "जेन", text)
print(new_text)

# रेक्सप्रेशन के साथ टेक्स्ट खोज और बदल
text = "हमारी वेबसाइट का URL यह है: www.example.com"
new_url = re.sub("www\.\w+\.com", "www.newurl.com", text)
print(new_url)
```
