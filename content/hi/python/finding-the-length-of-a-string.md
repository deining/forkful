## क्यों

वह उम्मीद से मूल निष्कर्ष लेगा कि स्ट्रिंग की लंबाई क्यों प्राप्त करना उपयोगी है।

## कैसे करें

Python में स्ट्रिंग की लंबाई प्राप्त करने के लिए निम्नलिखित की तरह कोड का उपयोग किया जा सकता है।

```Python
# स्ट्रिंग शामिल करें
string = "हम पाइथन प्रोग्रामिंग सीखने में मदद कर रहे हैं।"

# स्ट्रिंग की लंबाई प्राप्त करें
length = len(string)

# प्रिंट करें
print("लंबाई:", length)

# परिणाम:
# लंबाई: 35
```

## गहराई में जाएं

स्ट्रिंग की लंबाई प्राप्त करना अधिक उद्देश्य हो सकता है जब आपको जहाँ भी प्रोग्रामिंग में स्ट्रिंग्स का उपयोग करने की आवश्यकता हो। आप स्ट्रिंग की लंबाई को उपयोग करके स्ट्रिंग्स को सॉर्ट और तुलना कर सकते हैं। इसके अलावा, आप C के साथ साझा भागदौड़ में स्ट्रिंग की लंबाई जानने के लिए भी उपयोग कर सकते हैं।
