Python Programming के बारे में ब्लॉग पोस्ट के लिए लेख

## क्यों

क्‍या आप कभी स्‍ट्रिंग डेटा को एक फाइल में सेव करने या उसे पढ़ने के लिए किसी कोड पर विश्‍वास करते हैं? आप आजकल प्रोग्रामिंग का इस्‍तेमाल ज्‍यादातर डेटा संसार को अपनी इच्छानुसार संवारने के लिए करते होंगे। इस ब्लॉग पोस्ट में हम एक महत्‍वपूर्ण अवसर के बारे में चर्चा करेंगे - टेक्‍स्‍ट फाइल पढ़ने का।

## कैसे करें

Python में टेक्‍स्‍ट फाइल पढ़ने के लिए, हम `open()` फंक्‍शन का इस्‍तेमाल करते हैं जो हमें फाइल को खोलने और उसमे कुछ अद्‍भत स्‍ट्रिंग फंक्‍शन के साथ उसे पढ़ने की अनुमति देता है। नीचे दिए गए कोड ब्‍लॉक में हम दिखा सकते हैं कि कैसे हम एक आसानी से टेक्‍स्‍ट फाइल के कन्‍टेन्‍ट को पढ़ सकते हैं:

``` Python
# फाइल को खोलने के लिए 'open()' का इस्तेमाल करें
file = open("फाइल.एक्सटेंशन", "r")

# उस फाइल के कन्‍टेन्‍ट को पढ़ने के लिए 'read()' फंक्‍शन को पुनर्वर्तित करें
content = file.read()

# फाइल को बंद करें
file.close()

# पढ़ा हुआ कन्‍टेन्‍ट प्रिंट करें
print(content)
```

ऊपर दिए गए कोड ब्‍लॉक में हमने `open()` फंक्‍शन का इस्‍तेमाल करके फाइल को खोला, `read()` फंक्‍शन को पुनर्वर्तित करके उसको पढ़ा और फाइल को बंद कर दिया। अगर आप यह चाह
