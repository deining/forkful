## क्यों

क्या आपको Python में टेक्स्ट फाइल पढ़ने का तरीका पता है? टेक्स्ट फाइलों को पढ़ना अपने ऊपर एक काम की चिंता है। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि टेक्स्ट फाइल को कैसे पढ़ा जाए। 

## कैसे

```Python
with open('file.txt', 'r') as file:
    data = file.read()
    print(data)
```

टेक्स्ट फाइल को पढ़ने के लिए, आपको `open()` फ़ंक्शन का उपयोग करना होगा और फाइल का नाम और यह कि आप उसे कैसे खोलना चाहते हैं, का जिक्र करना होगा। यदि आप सिर्फ फाइल को पढ़ना चाहते हैं, तो आप `r` विकल्प का उपयोग कर सकते हैं। फ़ंक्शन के बाद, आप `with` स्टेटमेंट का उपयोग करके फ़ाइल को खोल सकते हैं और हमेशा अन्त में `file.close()` को डालने की चिन्ता न करें। आप `read()` फ़ंक्शन का उपयोग करके फ़ाइल के सभी डेटा को पढ़ सकते हैं। आप फ़ाइल में से एक ही लाइन पढ़ना चाहते हैं, तो आप `readline()` फ़ंक्शन का उपयोग कर सकते हैं।

```Python
with open('file.txt', 'r') as file:
    for line in file:
        print(line)
```

आप `for` लूप का उपयोग कर लाइन एक-एक करके प्रिंट कर सकते हैं। आप भी कई लाइन दोरान एक साथ पढ़ना चाहते हैं, तो आप `readlines()` फ़ंक्शन का उपयोग कर सकते हैं। फ़ाइल को खोलने के बाद, आप उसे बन्द करने के लिए `close()` फ़ंक्शन का उपयोग कर सकते हैं।
