## क्यों

अधिकांश लोग पाइथन को केवल डेटा स्ट्रक्चर्स और एल्गोरिदम्स को एक्सेस करने के लिए ही जानते हैं, लेकिन आपको न जाने क्यों, पाइथन कोडिंग आपके साथ काम करने का एक और तरीका प्रदान करता है। पाइथन में रैंडम नंबर जेनरेट करना तकनीक है जो आपको निर्धारित संख्या के बजाए यादृच्छिक नंबर देता है। इसके साथ, आप खेल खेलने, कंप्यूटर सिमुलेशन चलाने या डेटा के साथ तर्क करने का अनुभव कर सकते हैं। 

## कैसे करें

अगर आप रन्डम नंबर जेनरेट करने की प्रक्रिया में जाना चाहते हैं, तो निम्न उदाहरण को देखें।

```Python
# पाइथन रैंडम नंबर जेनरेट करें
import random

# 1 से 10 के बीच रैंडम नंबर जेनरेट करें
num = random.randint(1, 10)
print(num) # उदाहरण आउटपुट: 7 

# 0 से 1 के बीच रैंडम डेसिमल नंबर जेनरेट करें
dec_num = random.random()
print(dec_num) # उदाहरण आउटपुट: 0.532789 

# सूची से छोटे और बड़े रैंडम नंबर चुनें
my_list = [3, 8, 1, 6, 9, 2]
print(random.choice(my_list)) # उदाहरण आउटपुट: 6
```
