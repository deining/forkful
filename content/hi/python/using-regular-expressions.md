## क्यों
रेगुलर एक्सप्रेशन का उपयोग करें? यह प्रश्न हो सकता है कि क्यों हमें रेगुलर एक्सप्रेशन सीखने में दिलचस्पी होनी चाहिए। पहले समझना होगा कि रेगुलर एक्सप्रेशन क्या है और उसे किस तरह से सबसे अच्छी तरह से समझा जा सकता है। रेगुलर एक्सप्रेशन आमतौर पर पाठों को ढूढ़ने, एक टेक्स्ट स्ट्रिंग में प्राप्त किए गए डेटा को ट्रांसफॉर्म करने और खोज के लिए सिफ़ारिशें देने के लिए उपयोग किया जाता है। यह किसी भी डेटा वैज्ञानिक या उस्के साथ संबंधित कोड में उच्च गुणवत्ता के साथ मिलता है।

## कैसे करें
अधिकांश स्क्रिप्टिंग भाषाओं की तरह, पाइथन में भी रेगुलर एक्सप्रेशनें उपलब्ध हैं। यदि आप पाइथन में अपनी रेगुलर एक्सप्रेशनों का उपयोग करना चाहते हैं, तो आपको `re` मॉड्यूल का उपयोग करना होगा। इस मॉड्यूल में स्वयं को `compile()` फ़ंक्शन के माध्यम से अपनी रेगुलर एक्सप्रेशन समन्वित करना होगा। इसके बाद आप `findall()` अथवा `search()` फंक्शन का उपयोग करके अपने पैटर्न को ढूढ़ सकते हैं। नीचे पाइथन में कुछ उदाहरण दिए गए हैं:

```Python
import re

# रेगुलर एक्सप्रेशन संचित करें
pattern = re.compile(r"\w+")

# स्ट्रिंग से मेल खोजें
result = pattern.search("यह कुछ अनुच्छेद है।")
print(result.group())  # अनुच्छेद
