आमतौर पर हमारे सामने कई तरह की प्रॉग्रामिंग चुनौतियों का सामना करना पड़ता है। और कुछ मामलों में हम नामिक व्यंजनों की सहायता लेते हैं। वास्तव में, टेक्स्ट या स्ट्रिंग मानिए, उसमें जो भी शब्द या वाक्य हों, उन्हें आसानी से शोधने या परिवर्तित करने के लिए हम नामिक व्यंजनों का उपयोग करते हैं जिसे हम रेगुलर एक्सप्रेशन भी कहते हैं। रेगुलर एक्सप्रेशन का उपयोग करने से हम अपने कोड को सरल बनाने और त्रुटियों को ध्यान से पता लगाने का समय कम कर सकते हैं।

## कैसे करें:

```python
import re

# re.findall() का उपयोग इस तरह से किया जा सकता है:
re.findall(r"[a-zA-Z]+","Hello World") # यह ['Hello', 'World'] लौटाएगा। 

# re.sub() उपयोग करने का एक और उदाहरण:
text = "मेरा नाम जॉन डो, और मैं पायथन प्रोग्रामर हूँ।"
re.sub(r"[a-zA-Z]+", "Robin", text) # यह "मेरा नाम Robin, और मैं Robin Robin Robin हूँ।" लौटाएगा।
```

## गहराई में जाएं:

आइए जानते हैं कि रेगुलर एक्सप्रेशन को शुरू में 1956 में कैनेडियन आइवी के लैंग्वेज डेवलपर के द्वारा विकसित किया गया था। यह पहले से ही काफी प्रचलित था और तब से बहुत आगे बढ़ गया है। हालांकि, रेगुलर एक्सप्रेशन के अलावा भी कुछ विकल्प हैं जो पायथन में उपलब्ध हैं, जैसे कि string methods और f-strings।

अगर आप रेगुलर एक्सप्रेशन का उपयोग करना चाहते हैं तो आपको `import re` कोड शीर्षक इस्तेमाल करना होगा। स्ट्रिंग पैटर्न बनाने के लिए हम पाइथन में नामिक व्यंजन को स्पेशल कैरेक्टर के साथ जोड़कर उपयोग कर सकते हैं जैसे जैसे `+`, `.`, `*` आदि। यदि आपके स्ट्रिंग में उपस्थित है तो आप चाहें तो `+` के बजाय `\+` का उपयोग कर सकते हैं।
