## क्यों
रेगुलर एक्सप्रेशन का उपयोग करने के पीछे का कारण है कि यह एक शक्तिशाली और सुगम तरीका है टेक्स्ट में खोज, निकाल और बदलाव करने का।

## कैसे करें
वैसे तो रेगुलर एक्सप्रेशन्स का इस्तेमाल करना थोड़ा चुनौतीपूर्ण हो सकता है, लेकिन यदि आप इस प्रोसेस को सीख लेंगे तो वह भी आपके कोड में काफी बदलाव ला सकता है। नीचे कुछ सरल उदाहरण हैं जो आपको इस तकनीक का इस्तेमाल करने का दिखाएँगे।

```Python
# टेक्स्ट से नंबर निकालना
import re
text = "मेरा नंबर 123456789 है"
result = re.search('\d+', text)
print(result.group(0))  # output: 123456789

# ईमेल सत्यापन करना
import re
email = "abc@xyz.com"
if re.match("^[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$", email):
    print("यह ईमेल वैध है।")  # output: यह ईमेल वैध है।

# शब्दों में से स्पेस हटाना
import re
sentence = "मैं हूँ      एक टेक्स्ट   उदाहरण।"
result = re.sub('\s+', ' ', sentence)
print(result)  # output: मैं हूँ एक टेक्स्ट उदाहरण।
```
