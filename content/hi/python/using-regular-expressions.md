## क्यों

आजकल प्रोग्रामिंग जगत में रेगुलर एक्सप्रेशन एक बहुत ही उपयोगी टूल है, जो टेक्स्ट पैटर्न को खोज करने और संशोधित करने में सक्षम होता है। इससे आपको टेक्स्ट से काम करने में काफी समय और प्रयास बचाने में मदद मिलती है।

## कैसे करें

रेगुलर एक्सप्रेशन्स को इस्तेमाल करने के लिए, सबसे पहले आपको `re` मॉड्यूल को इम्पोर्ट करना होगा। फिर, आप `re.compile()` के माध्यम से अपना पैटर्न बना सकते हैं और `re.search()` के साथ उसे टेक्स्ट में से खोज सकते हैं। नीचे एक उदाहरण दिया गया है:

```Python
import re

pattern = re.compile(r'\d+')

text = "Today is 10th June, 2021"

match = re.search(pattern, text)

print(match.group())
```
आउटपुट:
```
10
```

इसके अलावा, रेगुलर एक्सप्रेशन्स के साथ आप चार्टर सेट और मेटा कैरेक्टर जैसी अन्य विशेषताओं का भी उपयोग कर सकते हैं। आप `.+` का उपयोग करके कोई भी शब्द दूंगा, `.*` का उपयोग करके किसी भी टेक्स्ट को खोजे गा या `[]` का उपयोग करके किसी भी कैरेक्टर सेट को खोजे गा। आप और भी समझने के लिए नीचे दिए गए विडिओ को देख सकते हैं:
