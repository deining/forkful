## क्या करना है?

एक साधारण सा जीवन स्थिति है: जब हम अपने कोड को लिखते हैं, हम हमेशा सुनिश्चित करने का प्रयास करते हैं कि वह सही तरीके से काम कर रहा है और कोई समस्या नहीं है। इस कारण से, आपने शायद अपने कोड में टेस्टिंग का दौरना देखा होगा। इस ब्लॉग पोस्ट में, हम टेस्टिंग क्यों लिखने का प्रत्येक हिस्सा को समझने का प्रयास करेंगे।

## कैसे करें?

कोड को टेस्ट करने का सबसे सरल तरीका है एक उदाहरण के साथ सीधे सामान्य कोड ब्लॉक का उपयोग करना। आप आश्चर्यचकित होंगे कि कितना आसान है कोड को बदलना और टेस्ट करना। यहाँ एक साधारण उदाहरण है:

```Python
# चयनित नंबरों की गणना करें

def add(x, y):
    return x + y

assert add(3, 4) == 7
assert add(-1, 2) == 1
```

आप अपने कोड में बहुत सारी टेस्ट लिख सकते हैं और यह उन सभी समय व उलझन को बचाता है जो नए फीचर्स या बग्स के प्रकार में आ सकते हैं।

## गहराई में जाओ
