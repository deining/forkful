## क्यों

प्रोग्रामिंग में टेस्ट लिखना क्यों जरूरी है? यह एक आम सवाल है जो शुरुआती स्तर पर आने वाले पाठकों के दिमाग में उठ सकता है। सबसे अच्छी बात है कि डेवलपर के लिए टेस्ट लिखना जरूरी है क्योंकि यह उनके कोड के गुणवत्ता और स्केलेबिलिटी को बढ़ा कर उनके काम को आसान बनाता है।

## कैसे करें

आप कैसे अपने पायथन कोड और टेस्ट के साथ अभ्यास कर सकते हैं? यह दो बातें हैं: सीखने और अभ्यास करना। अगली सेक्शन में, हम सीखेंगे कि टेस्ट कोड कैसे लिखा जाता है, और उसे समझने के लिए कैसे कोड ब्लॉक्स और नमूना आउटपुट का उपयोग किया जाता है।

```Python
# एक साधारण संयोजन फ़ंक्शन का उदाहरण
def add(num1, num2):
    return num1 + num2

# टेस्ट केस: 
assert add(2, 3) == 5  # सफल उदाहरण।
assert add(8, 10) == 18  # सफल उदाहरण।
assert add(5, 7) == 12  # असफल उदाहरण।
```

## गहराई से जाओ

टेस्टिंग के लिए कोड लिखना सरल है, लेकिन गलतियों को ठीक करने के लिए समय और धैर्य की आवश्यकता होती है। टेस्ट लिखते समय, हमें सुनिश्चित करना होता है कि उनके द्वारा क्यूटर की सामग्री को सहेजने के लिए सिक्योरिटी, स्मृति और अन्य प्रकार की जांच की जाती है। इसके अलावा, हमें सुनिश्चित करना होता है कि टेस्ट लिखने के लिए उपयुक्त फ़ंक्शन
