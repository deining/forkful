##क्यों

एक स्ट्रिंग को लोअर केस में बदलने में रुचि रखने के लिए क्यों?

रख सकता है:

1. स्ट्रिंग मानपदों को लाइब्रेरी या अन्य प्रोग्रामों के साथ एक्सचेंज करने की आवश्यकता हो सकती है, जहां केस सेंसिटिव मामलों को हल करने के लिए लोअर केस की आवश्यकता हो सकती है।

2. स्ट्रिंग में अंतर न बताने के लिए समानता प्रकार के तुलनात्मक ऑपरेशन पर हाशि का उपयोग करने के लिए, जिससे शब्दों को आसानी से तुलनात्मक रूप से जांचा जा सकता है।

##कैसे करें

```
Python

# मूल रूप से दिए गए स्ट्रिंग को लोअर केस में कनवर्ट करें
string = "यह एक स्ट्रिंग है"
print(string.lower())

#आउटपुट: यह एक स्ट्रिंग है
```

```
Python

# यूजर इनपुट से स्ट्रिंग को लोअर केस में कनवर्ट करें
string = input("कृपया एक स्ट्रिंग दर्ज करें: ")
print(string.lower())

#इनपुट: यह एक स्ट्रिंग है
#आउटपुट: यह एक स्ट्रिंग है
```

##गहराई में जाएं

स्ट्रिंग को लोअर केस में कनवर्ट करने के लिए, पायथन में डिफॉल्ट फ़ंक्शन `lower()` का उपयोग किया जाता है। यह फ़ंक्शन जोड़ी गई स्ट्रिंग ऑब्जेक्ट को मूल रूप से दिए गए स्ट्रिंग को लोअर केस में कनवर्ट करता है। एक और तरीका है `casefold()` फ़ंक्शन का उपयोग करना, जो स्ट्रिंग को उसकी अक्षरमाला और सामान्य बराबर समानता स्टाइल के साथ दोनों को मूल्यांतर
