## क्यों

आपने कभी सोचा है कि आपको अपने स्ट्रिंग को लोअर केस में कन्वर्ट करने की क्या ज़रूरत है? इस आर्टिकल में हम इस प्रश्न का उत्तर देंगे और स्ट्रिंग को लोअर केस में कैसे कन्वर्ट किया जाता है का एक आसान सा तरीका बताएंगे।

## कैसे करें

जिस भी स्ट्रिंग को आप लोअर केस में कन्वर्ट करना चाहते हैं, उसके लिए आपको सिर्फ नीचे दिए गए सिंपल कोड को पाइथन में लिखना होगा।

```Python
# स्ट्रिंग लोअर केस में कन्वर्ट करें
string = "Python Program"
print(string.lower())
```

आउटपुट:

```Python
python program
```

आप देख सकते हैं कि स्ट्रिंग "Python Program" को लोअर केस में कन्वर्ट करने से वह "python program" हो गई। इस चरण में, हम `lower()` फ़ंक्शन का उपयोग करते हैं जो स्ट्रिंग को केस सेंसिटिविटी को मध्य करके उसे लोअर केस में बदल देता है। यदि आप अपने कोड में स्ट्रिंग के साथ कोई भी ऑपरेटर या मैथमेटिकल ऑपरेशन का उपयोग करना चाहते हैं, तो आपको स्ट्रिंग को लोअर केस में बदलने से पहले इस फ़ंक्शन का उपयोग करना ज़रूरी होगा।
