# क्यों

दो तारीखों के तुलना करने का काम कैसे हमें मदद कर सकता है?

यदि आप एक पायथन प्रोग्रामर हैं और आपको दो तारीखों को तुलना करने की जरूरत है, तो आप सही जगह पर हैं। इस लेख में, हम आपको बताएंगे कि आपको क्यों तारीखों को तुलना करने की आवश्यकता है और इसे कैसे कर सकते हैं। यह आपके लिए डेटा मैनिपुलेशन और तारीख गणना में अधिक सुविधा प्रदान करेगा।

## कैसे करें

यदि आपको किसी भी प्रोग्रामिंग भाषा का ज्ञान होता है, तो पायथन बहुत ही आसान और आकर्षक है। तारीखों को तुलना करने के लिए आपको सिर्फ दो समय दर्शकों को तुलना करने के लिए `==`, `!=`, `>`, `<`, `>=` और `<=` जैसे गणकों का उपयोग करना होगा। इस लेख में हम आपको एक सरल उदाहरण के साथ दिखाएंगे कि कैसे आप ये गणकों का उपयोग कर सकते हैं।

```Python
# दो तारीखों को तुलना करें
first_date = "2020-01-01"
second_date = "2020-01-02"

# तारीखों को तुलना करें
print(first_date == second_date) # आपको False मिलेगा
print(first_date < second_date) # आपको True मिलेगा
print(first_date > second_date) # आपको False मिलेगा
```

जैसा कि आप ऊपर देख सकते हैं, आप गणकों का उपयोग करके आसानी से तारीखों को तुलना कर सकते हैं। आप अपनी आवश्यकतानुसार अधिक गणकों का उपयोग कर सकते हैं और तारीखों को अपनी पसंद अनुसार संयोजित कर सकते हैं।
