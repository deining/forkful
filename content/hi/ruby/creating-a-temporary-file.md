अभी भी बहुत सारे यूजर्स यह नहीं जानते कि टेम्पोररी फ़ाइल्स क्यों बनाई जाती हैं। टेम्पोररी फ़ाइल्स को बनाने के लिए बहुत से उदाहरण हैं, जैसे कि डेटा को स्टोर करने के लिए या अन्य काम को पूरा करने के लिए। टेम्पोररी फ़ाइल्स बनाने से, हम स्थान खाली कर सकते हैं और कंप्यूटर और सिस्टम का समय भी बचा सकते हैं।

## कैसे करें?

यहाँ हम ```Ruby Tempfile``` क्लास का उपयोग करके टेम्पोररी फ़ाइल्स बनाने सेखेंगे। 

```
require 'tempfile' 

# टेम्पोररी फ़ाइल बनाएं
temp_file = Tempfile.new("tempfile")
 
# फ़ाइल में डेटा लिखें
temp_file.write("यह टेम्पोररी फ़ाइल है!")
 
# फ़ाइल से डेटा पढ़ें
temp_file.rewind
puts temp_file.read
 
# फ़ाइल बंद करें
temp_file.close
```

आउटपुट:
```
यह टेम्पोररी फ़ाइल है!
```

## गहराई में जाएं

टेम्पोररी फ़ाइल्स बनाने से पहले, वे रद्द करने के लिए ```#unlink``` फंक्शन के साथ सेट होती हैं। यह कन्ट्रोल करता है कि जब हम उस फ़ाइल को बंद करते हैं, तो वह ठीक से हट जाए। इसके अलावा, हम ```Tempfile``` को इस तरीके से भी बना सकते हैं:

```
Tempfile.new("tempfile", "/path/to/folder")
```

यहाँ "tempfile" फ़ाइल नाम है जो स्वचालित रूप से बनाई जाती है। संभवतः, आप "tempfile" की जगह कोई अन्य नाम भी दे सकते हैं। हम साथ ही डिफ़ॉल्ट डिरेक्टरी के रूप में ```Dir.tmpdir``` को भी पास कर सकते हैं। दूसरे संकेत द्वार
