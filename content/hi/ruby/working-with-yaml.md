## क्या और क्यों?
यैमल काम क्या है और इसका क्या उपयोग होता है ये समझने के लिए हमें रूबी की मदद लेनी होगी। यैमल एक इंटरएक्टिव डेटा से व्यवस्थापन प्रारूप है जो डेटा को रूबी ऑब्जेक्ट्स में स्टोर करने का एक आसान और सुविधाजनक तरीका है। रूबी प्रोग्रामर्स यैमल का उपयोग अपनी डेटा को स्टोर करने, लोड करने और ओपन करने के लिए करते हैं।

## कैसे:
```Ruby
require 'yaml'

# डेटा को रूबी ऑब्जेक्ट में स्टोर करना
data = {
  'नाम' => 'राहुल',
  'उम्र' => 25,
  'शहर' => 'मुंबई'
}

# फाइल में डेटा स्टोर करना
File.write('data.yml', data.to_yaml)

# डेटा को फाइल से रीड करना
loaded_data = YAML.load_file('data.yml')

# रीड किए गए डेटा को प्रिंट करना
puts loaded_data
```

आउटपुट:
```Ruby
{'नाम' => 'राहुल', 'उम्र' => 25, 'शहर' => 'मुंबई'}
```

## डीप डाइव:
रूबी में यैमल लाइब्रेरी 2001 में रिलीज की गई थी और यह पाइथन की एक कॉन्ट्रांटार्न पर आधारित है। आज, यह काफी लोकप्रिय प्रारूप है जो डेटा को स्टोर, व्यवस्थापित और साझा करने में इन्हें फाइलों और एप्लिकेशन्स के बीच बदलने का एक माध्यम है। अन्य प्रारूप जैसे XML, JSON और CSV भी इसी उद्देश्य के लिए उपयोग किए जा सकते हैं, लेकिन यैमल का लायटवेट और सरल सिंटैक्स होने के कारण, यह प्रोग्रामर्स द्वारा पसंद किया जाता है। यैमल पूरी तरह से कस्टमाइज किया जा सकता है और इसमें समर्थित एपिस का विस्तार भी होता है।
