## क्या और क्यों?
एक स्ट्रिंग की लंबाई ढूंढना यह है कि स्ट्रिंग में कितने अक्षर हैं। कई प्रोग्रामर्स इसे इस्तेमाल करते हैं क्योंकि वे अपने कोड में अक्षरों की संख्या को जानने के लिए जानकारी को बढ़ावा देना चाहते हैं।

## कैसे करें:
एक प्रायोजित स्ट्रिंग की लंबाई को ज्ञात करने के लिए, हम सबसे पहले उस स्ट्रिंग का `length` मेथड का उपयोग कर सकते हैं। यह मेथड स्ट्रिंग में अक्षरों की संख्या को गिनती करता है। उदाहरण के लिए:

```Ruby
puts "Hello World".length
# Output: 11
```

इसके अलावा हम एक `for` लूप या `each_char` मेथड का उपयोग करके भी स्ट्रिंग की लंबाई को निर्धारित कर सकते हैं। यहां भी हम उदाहरण देते हैं:

```Ruby
# Using for loop
str = "Hello World"

count = 0
for char in str
    count += 1
end

puts count
# Output: 11

# Using each_char method
str = "Hello World"
puts str.each_char.count
# Output: 11
```

## गहराई में जाएँ:
इतिहास में, स्ट्रिंग की लंबाई प्रोग्रामिंग में बहुत ही उपयोगी हुआ है। स्ट्रिंग की लंबाई न केवल एक स्ट्रिंग में अक्षरों की संख्या निर्दिष्ट करने में मदद करती है, बल्कि इससे हमारे कोड को एक्सेसिबल और पढ़ने में आसान बनाता है।

एक अन्य विकल्प है स्ट्रिंग की लंबाई को जानना है `size` मेथड का उपयोग करना। यह स्ट्रिंग की लंबाई और उसी में दिए गए उपादानों की संख्या को भी प्रदान करता है।

स्ट्रिंग की लंबाई को निर्दिष्ट करने के लिए, रूबी में हम ये भी कर सकते हैं:

```Ruby
str = "Hello World"

puts str.scan(/./).count
# Output: 11
```
