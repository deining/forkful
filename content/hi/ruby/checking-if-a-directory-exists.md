## क्यों  

क्या आपने कभी सोचा है कि अपने कंप्यूटर पर एक नया फाइल सहेजने से पहले कैसे पता करते हैं कि यह फाइल उस डायरेक्टरी में पहले से ही मौजूद है? अगर हाँ, तो आपको जानने की आवश्यकता है कि एक डायरेक्टरी मौजूद होने पर क्या करना है।

## कैसे करें

जानने के लिए कि एक डायरेक्टरी मौजूद है या नहीं, हम उपयोगकर्ताओं को निम्न रुप में निर्देशित करते हैं:

```Ruby
directory_name = "documents"

if File.directory?(directory_name)
  puts "डायरेक्टरी रहती है।"
else
  puts "डायरेक्टरी मौजूद नहीं है।"
end
```

जैसा कि आप ऊपर देख सकते हैं, हम `File.directory?` मेथड का उपयोग करके निर्देशित डायरेक्टरी में `documents` पोर्ट का नामांकन करते हैं। हम तुम्हें यहां बताना चाहते हैं क्योंकि यह एक उपयोगी और आसान तरीका है जो डायरेक्टरी मौजूद होने पर हमें सूचित करता है।
