## क्यों

कभी-कभी हमें ऐसे स्थिति से गुजरना पड़ता है जहां हमारे प्रोग्राम को कुछ काम करने के लिए एक विशेष डायरेक्ट्री का मौजूद होना आवश्यक होता है। इसलिए, हमें पहले से ही सुनिश्चित करना आवश्यक होता है कि उस डायरेक्ट्री के लिए प्रोग्राम में कोडिंग करने की आवश्यकता है या नहीं। ऐसे मामलों में, डायरेक्ट्री की जाँच तरीका काफी महत्वपूर्ण हो सकता है।

## कैसे

डायरेक्ट्री की जाँच करने के लिए, हम रेल स्टेट्स् (filestats) कक्षा का उपयोग कर सकते हैं। नीचे दिए गए उदाहरण में, हमने 'my_directory' नाम की एक फोल्डर बनाई है और उसकी जाँच करने के लिए कोड दिया है। यदि डायरेक्ट्री मौजूद नहीं होती है, तो कोड में दिये गए स्टेटस कोड (status code) का मूल्य false हो जाएगा।

```Ruby
require 'fileutils'

file_stats = File::Stat.new("my_directory")
puts file_stats.directory?
```
आउटपुट:
```
true
```

यदि आपको डायरेक्ट्री के अस्तित्व से कोई फायदा नहीं होता हो, तो आप file? अलगोरिथ्म का भी इस्तेमाल कर सकते हैं। यह आपको स्ट्रिंग गणना करेगा, जिसमें डायरेक्ट्री का नाम दिया जाएगा। इसे आप नीचे दिए गए उदाहरण से देख सकते हैं।

```Ruby
puts File.directory?("my_directory")
```
आउटपुट:
```
true
```
