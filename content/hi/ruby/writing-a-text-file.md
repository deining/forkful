## क्यों
टेक्स्ट फ़ाइल लिखने का काम क्यों किया जाता है, उसके पीछे की वजह जानने के लिए हमें समझना है कि यह काम क्या है। आपने कभी-न-कभी अपने कंप्यूटर पर टेक्स्ट फाइलें बनाते या संपादित करते हुए देखी होंगी। ये फाइलें हमारे आसपास की समस्याओं को हल करने या अपनी विचाराधीन शक्ति को फलस्वरूप बाबते की मदद करती हैं।

## कैसे करें
यदि आप एक प्रोग्रामिंग नया हैं और टेक्स्ट फाइल लिखने का तरीका जानना चाहते हैं, तो आप यहां सही जगह हैं। पहले, हम एक टेक्स्ट फाइल कैसे बनाते हैं इसके बारे में बात करेंगे। 

```Ruby
# एक नया टेक्स्ट फ़ाइल निर्मित करें
file = File.new("filename.txt", "w")

# फाइल में सामग्री लिखें
file.puts "यह एक नया टेक्स्ट फ़ाइल है!"
file.puts "हाय, विश्व!"
file.print "में साइड "
file.print "बाय साइड "
file.puts "लिख रहा हूँ!"

# फाइल बंद करें
file.close
```

इस मौजूदा कोड के बारे में कुछ बातें बताना चाहते हैं। पहले लाइन में, हम `File.new` फंक्शन का उपयोग करके एक नई टेक्स्ट फाइल बनाते हैं। दूसरे लाइन में, हम उस फाइल में लिखने के लिए `puts` और `print` मेथड का उपयोग करते हैं। और अंत में, हम `file.close` के द्वारा फाइल को बंद करते हैं। फाइल को बंद करने का यह आयोजन अनिवार्य है क्योंकि वह आपकी फाइल में लिखे गए सामग्री को हायवाए
