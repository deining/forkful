## क्यों

नए प्रोजेक्ट शुरू करने में शामिल होने के लिए *क्यों* कोई रुचि रखने वाला व्यक्ति होता है? संभवतः उनके पास नए और रोमांटिक परियोजनाओं पर संकल्पना है, या यह उनके कैरियर में आगे बढ़ने के लिए आवश्यक हो सकता है।

## कैसे करें

कोडिंग उदाहरण और संग्रहित आउटपुट के साथ "```Ruby ... ```" कोड ब्लॉक्स के भीतर।

उदाहरण के लिए, यदि हम एक सरल फ़ंक्शन बनाना चाहते हैं जो दो संख्याओं को जोड़ती है, तो हम इस प्रकार का कोड लिख सकते हैं:

  ```Ruby
  def add_numbers(x, y)
    return x + y
  end
  
  puts add_numbers(2, 5)
  ```
  
  जब हम यह कोड चलाएँगे, हमारे द्वारा दिए गए दो खाली संख्याओं को जोड़ा जाएगा। इस प्रकार का आउटपुट मिलेगा:
  
  ```
  7
  ```
  
  यह एक उदाहरण है कि कैसे हम रूबी में सरल फ़ंक्शन लिख सकते हैं। ऐसे ही आप दूसरे फ़ंक्शन भी लिख सकते हैं और अपने प्रोजेक्ट को आगे बढ़ा सकते हैं।
