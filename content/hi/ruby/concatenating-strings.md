## क्यों

जब हम प्रोग्रामिंग करते हैं, हम समय-समय पर स्ट्रिंग्स को जोड़ कर नए स्ट्रिंग बनाने की आवश्यकता महसूस करते हैं। यह लंबे स्ट्रिंग्स को अलग-अलग टेक्स्ट में विभाजित करने और संप्रयोग को सरल बनाने का एक आसान तरीका है।

## कैसे करें

जब हम दो स्ट्रिंग्स को `+` ऑपरेटर के साथ जोड़ते हैं, तो वे नए स्ट्रिंग के रूप में एक साथ मिलते हैं। इसका उदाहरण निम्नलिखित है:

```Ruby
puts "मैं " + "रूबी " + "सीख रहा हूँ।"

# Output:
मैं रूबी सीख रहा हूँ।
```

अगर हम `<<` ऑपरेटर का उपयोग करते हैं, तो यह पहली स्ट्रिंग पर दूसरी स्ट्रिंग को जोड़ता है। उदाहरण के लिए:

```Ruby
hello = "नमस्ते, "
name = "आपका नाम क्या है?"

puts hello << name

# Output:
नमस्ते, आपका नाम क्या है?
```

## गहराई में जाएँ

रूबी में स्ट्रिंग्स को जोड़ने के लिए एक और ऑपरेटर `+=` है। इसमें `+=` अपने आप में स्थायी रूप से नई स्ट्रिंग के साथ पहली स्ट्रिंग को जोड़ता है।

```Ruby
name = "बंदीत "

name += "वह दरवाज़े को दबाता है।"

puts name

# Output:
बंदीत वह दरवाज़े को दबाता है।
```
