## क्यों

Substring को प्राप्त करने के कारणों में से एक है कि आप एक द्वंद्व को प्रोग्राम में संसाधित करना चाहते हैं, जो आमतौर पर गणितीय ऑपरेशन में उपयोगी होता है, जैसे कि पंक्तिबद्ध और अव्यवधान करना।

## कैसे करें

आप substring को प्राप्त करने के लिए Ruby का `slice` या `slice!` मेथड का उपयोग कर सकते हैं। ये मेथड आपको एक शुरुआती स्थान और अंत स्थान के बीच एक निश्चित अंश को लेकर स्ट्रिंग को काटता है।

```Ruby
text = "मैं एक भाषा रुबी हूँ"
puts text.slice(5, 2)
# भा
```

यदि आप substring को एक चर प्रत्यावर्ती रूप में चाहते हैं, तो आप `[]` द्वारा भी लेकर सकते हैं।

```Ruby
text = "मैं एक भाषा रुबी हूँ"
puts text[6]
# ष
```

## गहराई में उतरें

substring का प्राप्त करना आपकी प्रोग्रामिंग कौशल और बेहतर अनुभव के प्रति आपकी क्षमता को बढ़ा सकता है। आप `each_char` या `chars` मेथड का इस्तेमाल करके स्ट्रिंग के हर अक्षर पर गहराई समझ सकते हैं। इसके लिए आप निम्नलिखित कोड का उपयोग कर सकते हैं:

```Ruby
text = "मैं एक भाषा रुबी हूँ"
text.each_char do |char|
  puts char
end
# म
# ै
# ं
# ए
# क
# (और इसी प्रकार)
```

इस प्रकार से आप सभी चर्स को एक समूह में कर सकते हैं और निश्चित संख्या के substring तैयार कर सकते हैं।
