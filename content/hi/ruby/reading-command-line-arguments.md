# क्यों
आपने कभी सोचा है कि कंमांड लाइन अर्गुमेंट पढ़ना क्यों ज़रूरी है? आज हम आपको बताएंगे कि आप कंमांड लाइन अर्गुमेंट्स कैसे पढ़ सकते हैं और इसके उपयोग क्या हो सकते हैं।

## कैसे करें
हम रूबी में कंमांड लाइन अर्गुमेंट्स पढ़ने के लिए `ARGV` युक्ति का उपयोग कर सकते हैं। यह एक विशिष्ट Array है जो कंमांड लाइन में दर्ज किए गए अर्गुमेंट्स को स्टोर करता है। नीचे दिए गए उदाहरण में हम `ARGV` युक्ति का उपयोग कर रहे हैं और इसका आउटपुट आपको दिखा रहे हैं:

```Ruby
puts ARGV[0] # दूसरा अर्गुमेंट प्रिंट करें
puts ARGV[1] # तीसरा अर्गुमेंट प्रिंट करें
```

यदि आप उपरोक्त कोड को एक्सेक्यूट करें और उससे पहले दो अर्गुमेंट्स (उदाहरण के लिए फ़ाइल का नाम) पास करें, तो आपको निम्न आउटपुट मिलेगा:

```
ruby arguments.rb file1 file2
file1
file2
```

यही तरीका आप `each` लूप के साथ भी उपयोग कर सकते हैं जो अर्गुमेंट्स के एक दूसरे के बाद से चलता है और उनपर काम करता है। नीचे दिए गए उदाहरण में, हम `each` लूप का उपयोग करके कंमांड लाइन अर्गुमेंट्स को प्रिंट कर रहे हैं:

```Ruby
ARGV.each do |arg|
  puts arg
end
```

यदि आप  उपरोक्त कोड को एक्सेक्यूट करें और तीन अर्गुमेंट्स पास करें, तो निम्न आउटपुट मिलेगा:

```
ruby arguments.rb apple banana orange
apple
