## क्यों

क्या आपने कभी सोचा है कि Ruby में टेक्स्ट फ़ाइल को कैसे पढ़ा जाए? शायद आपको लगता हो कि यह अधिकांश लोगों के लिए सरल होता है, लेकिन जब आप अपने कोड में अपनी तकनीकी पाठ को पढ़ रहे होते हैं तो तमाशा होता है। इस ब्लॉग पोस्ट के माध्यम से हम Ruby में टेक्स्ट फ़ाइल पढ़ने के लिए कुछ उदाहरण देखेंगे और साथ ही यह समझेंगे कि ऐसा करने के पीछे की तकनीक क्या है।

## कैसे करें

टेक्स्ट फ़ाइल को पढ़ने के लिए सबसे पहले हम उस फ़ाइल को खोलना होता है। इसके लिए हम `File` क्लास का इस्तेमाल कर सकते हैं। इसके बाद हम `read` मेथड का इस्तेमाल करके उस फ़ाइल के सभी डेटा को पढ़ सकते हैं। नीचे उदाहरण दिए गए हैं:

```ruby
File.open("data.txt") do |file|
    puts file.read
end

#Output:
#This is a text file.
#It contains some data for testing.
```

आप देख सकते हैं कि हमने सिर्फ दो स्टेप्स में उस फ़ाइल को पढ़ लिया है। समझाने के लिए हमने सारे कोड को `File.open` और `read` के अंदर रखा है। हम `puts` का इस्तेमाल इसलिए कर रहे हैं क्योंकि हम आपको सिर्फ उस फ़ाइल के डेटा को दिखाना चाहते हैं। यदि आप अधिक जानकारी के लिए [इस लिंक](https://ruby-doc.org/core-3.0.0/File.html#method-c-read) पर जा सकते हैं।
