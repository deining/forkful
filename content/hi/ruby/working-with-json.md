Why: क्यों करें JSON के साथ काम करना?

JSON (JavaScript Object Notation) एक लोकप्रिय डेटा प्रपत्र है जो विभिन्न एप्लिकेशन और लाइब्रेरियों के बीच डेटा को अनुरूप और संगठित रूप में तस्वीर देने के लिए इस्तेमाल किया जाता है। JSON आसानी से समझने और उपयोग करने में अत्यंत सरल होने के कारण यह बहुत लोकप्रिय हुआ है।

How To: जेसन का उपयोग कैसे करें?

तो चलिए देखें कि हम जेसन का उपयोग कैसे कर सकते हैं। पहले, हमें एक फ़ाइल बनानी होगी जिसमें हम डेटा को संग्रहीत करेंगे। फिर हम उस फ़ाइल को पढ़ने और डेटा को पार्स करने के लिए जेसन की लाइब्रेरी का उपयोग कर सकते हैं। नीचे दिए गए उदाहरण में, हम जेसन डेटा को एक सरल तालिका में दिखा रहे हैं:

```Ruby
require 'json'

# डेटा को संग्रहीत करने के लिए फ़ाइल बनाएं
file = File.read('data.json')

# डेटा को पार्स करें
data = JSON.parse(file)

# डेटा को तालिका में दिखाएं
puts data
```

आउटपुट:
```
{"name"=>"John Doe", "age"=>30, "occupation"=>"Software Engineer"}
```
