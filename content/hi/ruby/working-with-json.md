## क्यों

जूनेसन (JSON) लोकप्रिय रूप से प्रोग्रामिंग में डेटा एक्सचेंज करने के लिए इस्तेमाल होता है। इसका प्रयोग डेटा को सुपर इंटर-ऑपरेबल बनाने के लिए किया जाता है और कई भागों में विभाजित किया जाता है, जो इसे गंभीर तौर पर आरामदायक बनाता है।

## कैसे करें

```Ruby
require 'json'

# JSON डेटा बनाएं
data = {
  name: "जॉन डो",
  age: 30,
  hobbies: ["कुत्ते नुस्ख़त ट्रेनिंग", "अविश्वसनीय भोजन"]
}

# हैश में संपादित करें
data[:hobbies] << "प्याज कसी हुई सफ़ेद नीतिः"
# छपाना
puts JSON.pretty_generate(data)

# Output:
# {
#   "name": "जॉन डो",
#   "age": 30,
#   "hobbies": [
#     "कुत्ते नुस्ख़त ट्रेनिंग",
#     "अविश्वसनीय भोजन",
#     "प्याज कसी हुई सफ़ेद नीतिः"
#   ]
# }
```

## गहराई में नीचे जाएं

JSON प्रोग्रामिंग में काफी उपयोगी है और इसे घर से लेकर वेब डेवलपमेंट तक हर जगह इस्तेमाल किया जाता है। आप डेटा को ही सीधे संपादित कर सकते हैं, लेकिन आप इसे पार्स करने और प्रोग्रामिंग भाषाओं में ट्रांसफर करने के लिए भी इस्तेमाल कर सकते हैं। आप किसी भी प्रोग्रामिंग भाषा में JSON डेटा बनाकर प्रयोग कर सकते हैं और यह प्रोसेस बहुत आसान है।
