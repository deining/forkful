# क्यों

कमांड लाइन आर्ग्यूमेंट पढ़ना खोजने में मदद कर सकता है ताकि आप पारमीटरों को अनुकूलित कर सकें और अपने कोड में फ्लेक्सिबिलिटी जोड़ सकें। यह बढ़ीया टूल है जो आपको अपने प्रोग्राम को पर्याप्त बनाने में मदद करता है।

# कैसे करें

```Clojure
(defn main
  [& args]
  (println "आपने पास किए गए आर्ग्यूमेंट:" args))
```

जब आप अपने कोड में आर्ग्यूमेंट मान पाते हैं, आप इसका उपयोग अपने प्रोग्राम के लिए कर सकते हैं। परम के रूप में, आप अपनी स्क्रिप्ट में विभिन्न संख्या के आर्ग्यूमेंट पारित कर सकते हैं। आप इस साइंटेक्स को अपने स्क्रिप्ट के शीर्ष में उपयोग कर सकते हैं।

```Clojure
lein exec फ़ाइलनाम "प्रथम पैरामीटर" "दूसरा पैरामीटर"
```

यह स्क्रिप्ट परिणाम देगा:

```Clojure
आपने पास किए गए आर्ग्यूमेंट: ["प्रथम पैरामीटर" "दूसरा पैरामीटर"]
```

# गहराई की झलक

कमांड लाइन आर्ग्यूमेंट को पढ़ने के लिए आपको वाइल्डकार्ड्स (*) का उपयोग करने की आवश्यकता नहीं है, जो अन्य प्रोग्रामिंग भाषाओं में आमतौर पर देखा जाता है। आप आसानी से आर्ग्यूमेंट को साफ़ करने के लिए एक फॉर लूप का उपयोग कर सकते हैं और विभिन्न आर्ग्यूमेंट मान प्राप्त कर सकते हैं।
