## क्या और क्यों?

टिक्स्ट फ़ाइल लिखना क्या हैं, और क्यों हम इसे प्रोग्रामर्स्स करते हैं? अगर आप कभी अपनी कोड को प्रिंट आउट करने की कोशिश की है, या दूसरे लोगों को अपने कोड को देखने देना चाहते हैं, तो आपको जाना चाहिए कि कैसे टिक्स्ट फ़ाइलें लिखी जाती हैं। प्रोग्राम बनाने के दौरान, टिक्स्ट फ़ाइलें अपने कोड को सुगम समझने और साझा करने का एक अच्छा तरीका हैं।

## कैसे?

यहां आपको कुछ सीखने को होगा! लेकिन चिंता न करें, हम आपको सब कुछ बताएंगे। आपको बस किसी भी टेक्स्ट एडिटर में पुर्ण फ़ाइल नाम के साथ एक "।txt" एक्सटेंशन के साथ फ़ाइल बनानी होगी। नीचे दिए गए कोड के साथ आप स्टेप बाय स्टेप देख सकते हैं कि कैसे उस फ़ाइल में टेक्स्ट लिखा जाता है:

टेक्स्ट फ़ाइल बनाने का कोड:

```Clojure
(def file-name "my_text_file.txt")
```

टेक्स्ट फ़ाइल में लिखा हुआ टेक्स्ट:

```Clojure
(def text "Hello World!")
```
