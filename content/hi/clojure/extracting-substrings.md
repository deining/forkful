## क्यों

अगर आपको कोई बड़ी स्ट्रिंग है और आपको उसमें से छोटे टुकड़ों को निकालना है, तो आपको स्ट्रिंग से उप-स्ट्रिंग निकालने का काम करना हो सकता है। यह सबसे आम उपयोग अलग अलग डेटा को विभाजित करने के लिए आता है।

## कैसे

```Clojure
; सबसे पहले, आपको clojure.string/library को इंपोर्ट करना होगा
(require '[clojure.string :as str])

; निर्दिष्ट स्ट्रिंग से उप-स्ट्रिंग निकालना
(str/substring "Clojure Programming" 0 7)
; प्रकाशण आउटपुट: "Clojure"

; आरम्भिक स्ट्रिंग से अंतिम अक्षर तक उप-स्ट्रिंग निकालना
(str/substring "Apple" 0 -1)
; प्रकाशण आउटपुट: "Appl"

; आरम्भिक और अंतिम स्थानों के साथ उप-स्ट्रिंग निकालना
(str/substring "Orange" 1 -2)
; प्रकाशण आउटपुट: "rang"

; निर्दिष्ट स्ट्रिंग से वांछित अक्षरों को निकालकर उप-स्ट्रिंग निकालना
(str/substring "Mango" 0 3)
; प्रकाशण आउटपुट: "Man"

; अगर स्ट्रिंग से अधिक अक्षर निकालने हो, तो न्यूनतम और अधिकतम स्थान निर्दिष्ट करें
(str/substring "Java" 1 10)
; प्रकाशण आउटपुट: "ava"
```

## गहराई में जाएं

स्ट्रिंग से उप-स्ट्रिंग निकालना शायद आपको आसान लगेगा, लेकिन यह काम खासकर एल्गोरिथ्मिक और स्ट्रिंग हैंडलिंग में मदद कर सकता है। यदि आप एक बड़े डेटाबेस से डेटा निकालने के लिए क्वेरी चला रहे हैं और आपको कुछ विशेष डेटा ही चाहिए हो, तो स्ट्रिंग से उप-स्ट्र
