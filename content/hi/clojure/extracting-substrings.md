## क्यों

आपने हो सकता है कि वहाँ एक स्ट्रिंग के भीतर से उत्पन्न उप-स्ट्रिंगों को निकालने की आवश्यकता हो। यह उप-स्ट्रिंगों को अलग करने और डेटा को बदलने का एक आसान तरीका हो सकता है।

## कैसे करें

आइए पहले हम इस समस्या को हल करने के लिए विभिन्न तरीकों पर एक नज़र डालें।

```Clojure
; string के अंतिम तीन अक्षर लें
(substring string -3)

; 3वें अक्षर से शुरू होने वाला string ही लेगा
(substring string 2)
```

आपको दिए गए उदाहरण में, `substring` फ़ंक्शन स्ट्रिंग के भीतर उप-स्ट्रिंगों को निकालने के लिए उपयोगी हो सकता है। हम स्ट्रिंग के एक या अधिक अक्षरों को लेने के लिए शुरुआती और अंतिम स्थानों की स्थिति को निर्धारित कर सकते हैं।

अब आप `subs` फ़ंक्शन का उपयोग करके भी उप-स्ट्रिंगों को निकाल सकते हैं।

```Clojure
; 2वें यूनिकोड करैक्टर से शुरू होने वाला string ही लेगा
(subs string 1)

; 2वें यूनिकोड करैक्टर से शुरू करते हुए, 4 अक्षर लें
(subs string 1 5)
```

यहां मैंने `subs` फ़ंक्शन का उपयोग अलग स्ट्रिंग को निकालने के लिए उपयोगी होने वाले प्रदर्शन के साथ किया है।

## डीप डाइव

जब आप बात स्ट्रिंग से उप-स्ट्रिंगों तक जाते हैं, तो आपको समझने की आवश्यकता है कि कैसे `substring` और `subs` फ़ंक्शन वास्तव में काम करते हैं।

`substring` फ़ंक्शन `start` और `end`
