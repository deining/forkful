# क्यों

अपने कंप्यूटर पर फ़ाइलों और फ़ोल्डरों को संगठित रखने के लिए आपने कई डायरेक्ट्री बनाए होंगे। अपनी सभी कंप्यूटर ऐप्लिकेशन्स में आप शायद देखेंगे कि कभी-कभी कोड में फ़ोल्डरों की उपस्थिति के बारे में सवाल उठते हैं। चाहे यह कोड को किसी फ़ोल्डर में फ़ाइलें बनाने के लिए या फिर दी गई फ़ाइल में पढ़ने या लिखने के लिए हो। इसलिए, फोल्डरों की उपस्थिति को जाँचना एक अहम कार्य है।

## कैसे करें

```C++

#include <iostream>
#include <filesystem>  //संदर्भपुस्तिका जो फोल्डरों की उपस्थिति की जाँच करने के लिए इस्तेमाल करते हैं

using namespace std;

int main() {
    string directory = "C:/Users/HindiTester/Desktop";
    
    //जाँचने के लिए संदर्भपुस्तिका उपयोग करें
    if (filesystem::exists(directory)) {
        cout << "फ़ोल्डर मौजूद है!" <<endl;
    }
    else {
        cout << "यह फ़ोल्डर मौजूद नहीं है।" << endl;
    }
    
    return 0;
}

```

उस संदर्भपुस्तिका का उपयोग करके आप फ़ोल्डर की उपस्थिति को आसानी से जाँच सकते हैं। आपको यह सुनिश्चित करने की आवश्यकता हो सकती है कि एक फ़ोल्डर, जिसे आप अभी तक इस्तेमाल नहीं किया है, मौजूद है या एक नया फ़ोल्डर बनाने की आवश्यकता हो। इस उदाहरण में, हमने दी गई स्थान पर फ़ोल्डर की उपस्थिति की जाँच कररे, लेकिन आपने अपने स्थान के अनुसार उसका उपयोग कर सकते हैं।
