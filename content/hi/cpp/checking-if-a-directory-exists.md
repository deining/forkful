# क्या और क्यों?

जब हम किसी ऑनलाइन ऐप्लिकेशन या कंप्यूटर प्रोग्राम का उपयोग करते हैं, तो कई बार हमें फ़ाइल या फ़ोल्डरों को देखने और उनका उपयोग करने की जरूरत होती है। लेकिन क्या हम यह जानते हैं कि वे फाइल या फ़ोल्डर हमारे सिस्टम में उपस्थित हैं या नहीं? प्रोग्रामर्स इसलिए फ़ाइल या फ़ोल्डरों की उपस्थिति की जांच करते हैं ताकि वे सही तरीके से काम कर सकें।

# कैसे करें?

```C++
#include <iostream>
#include <sys/stat.h> // इस लाइब्रेरी का उपयोग पथ की उपस्थिति की जांच के लिए किया जाता है।
using namespace std;

int main() {
    string path = "/home/user/example.txt"; // जो भी पथ आप जांचना चाहते हैं उसको यहां डालें।
    struct stat buffer;
    if (stat(path.c_str(), &buffer) == 0) { // stat() फ़ंक्शन के द्वारा पथ की उपस्थिति की जांच की जाएगी।
        cout << "फ़ाइल या फ़ोल्डर मौजूद है।" << endl;
    }
    else {
        cout << "फ़ाइल या फ़ोल्डर मौजूद नहीं है।" << endl;
    }
    return 0;
}
```

अगर आपको जानना है कि एक डायरेक्टरी के साथ बाकि फ़ाइलों और डायरेक्टरीज़ की संख्या की पता या सूची करना चाहते हैं, तो आप `opendir()` और `readdir()` फ़ंक्शन का भी उपयोग कर सकते हैं।

# गहराई में जाएं

इस तरह की जांच प्रोग्रामिंग से संबंधित एक महत्वपूर्ण गतिविधि है। पहले, कंप्यूटर सिस्टम में पथों की उपस्थिति जांचने के लिए उपयोग की जाती थी। आजकल, इसकी आवश्यकता बढ़ गई है क्योंकि बहुत सारे ऑनलाइन ऐप्लिकेशन और सॉफ्टवेयर फ़ाइलों या फ़ोल्डरों का उपयोग करते हैं। इसके अलावा, `C++` के साथ संभावित त्रुटियों को भी दूर करने के लिए भी फ़ाइल या फ़ोल्डरों की उपस्थिति जांचना जरूरी होता है। अधिक गहराई के लिए, आप `man stat` कमांड का उपयोग कर सकते हैं।
