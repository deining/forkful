# क्यों

क्या आपने अपने प्रोग्राम में कभी ऐसा हुआ है कि आपने एक निर्दिष्ट डायरेक्टरी में फ़ाइलों को खोजने के लिए कोड लिखा है, लेकिन फिर भी आपको पता नहीं चल पाता है कि वह डायरेक्टरी क्या मौजूद है? अगर हाँ, तो आप सही जगह पर हैं! इस आलेख में, हम बताएंगे कि आप अपने प्रोग्राम में एक डायरेक्टरी का अस्तित्व कैसे जांच सकते हैं। 
डायरेक्टरी का अस्तित्व जांचना अपने कोड को सुरक्षित बनाने के लिए बहुत आवश्यक है, क्योंकि यह आपको अनुमति देता है कि आप जब आप फ़ाइल खोज रहे हैं, तो आप सही डायरेक्टरी का उपयोग कर रहे हैं।

## कैसे करें

जेस्ट कोड से बेचेडएर स्टे दएरेमोस्त्रेड:

```C++
#include <iostream>
#include <filesystem>

int main()
{
    // जेस्ट कोड
    std::string directoryPath = "C:/Users/Username/Documents/";
    if (std::filesystem::exists(directoryPath))
    {
        std::cout << "डायरेक्टरी मौजूद है!" << std::endl;
    }
    else
    {
        std::cout << "डायरेक्टरी मौजूद नहीं है।" << std::endl;
    }
    return 0;
}
```

उपरोक्त कोड में, हमने filesystem लायब्रेरी का उपयोग करके फ़ोल्डर का अस्तित्व जांचा है। आपको पहले filesystem लायब्रेरी को अपने प्रोग्राम में शामिल करना होगा। इसके बाद, आप कोड के समान पथ को डायरेक्टरी पाठ के साथ बदल सकते हैं और दी गई उदाहरण में, हमने "C:/Users/Username/Documents/" का उपयोग किया है।

उदाहरण की आउटपुट:

```
डायरेक्टरी मौजूद है!
```
