## क्यों

अगर आप C++ प्रोग्रामिंग के बारे में कुछ जानते हैं, तो आपने शायद `डायरेक्ट्री` (directory) शब्द को सुना होगा। अगर आप एक प्रोग्रामर हैं या C++ प्रोग्रामिंग सीखना चाहते हैं, तो आपको यह जानना जरूरी है कि डायरेक्ट्री क्या होता है, यह क्यों महत्वपूर्ण है, और कैसे हम अपने प्रोग्राम में इसका उपयोग कर सकते हैं। इस ब्लॉग पोस्ट के माध्यम से हम `डायरेक्ट्री` के बारे में जानेंगे और यह कैसे हमारे प्रोग्राम में प्रभावी ढंग से उपयोग किया जा सकता है।

## कैसे करें

अगर आप पहले से ही C++ प्रोग्रामिंग का ज्ञान रखते हैं, तो आपने `#include <filesystem>` से शुरू होने वाली `फाइल-सिस्टम-लाइब्रेरी` (filesystem library) सुनी होगी। यह लाइब्रेरी प्रोग्रामीज्मिंग में फाइल और फोल्डर्स को मैनेज करने के लिए बहुत उपयोगी है। आप `फ़ाइल-सिस्टम-लाइब्रेरी` का उपयोग करके डायरेक्ट्री के अस्तित्व को चेक कर सकते हैं। नीचे दिए गए कोड ब्लॉक में हम एक सरल उदाहरण देखेंगे:

```C++
#include <iostream>
#include <filesystem>

using namespace std;
namespace fs = filesystem;

int main() {
    string path = "/Users/User/Documents/Example";

    // डायरेक्ट्री का अस्तित्व चेक करें
    if (fs::is_directory(path)) {
        cout << "यह एक वैध डायरेक्ट्री है।" << endl;
    } else {
        cout << "इस फ़ोल्डर का अस्तित्व नहीं है।" << endl;
    }

    return 0;
}
```
