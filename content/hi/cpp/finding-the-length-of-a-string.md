## क्यों

आपने अपने कोड में कभी भी स्ट्रिंग की लंबाई को जानने की जरूरत महसूस की है? स्ट्रिंग के लंबाई को जानना आपको अपने कोड को समझने में मदद करता है और यह समस्याओं को खोजने और ठीक करने में भी सहायता करता है। इसलिए, स्ट्रिंग की लंबाई को जानना बहुत अहम हो सकता है। 

## कैसे करें

```C++
#include <iostream>
#include <string>
using namespace std;
int main(){
    string str = "यह स्ट्रिंग कोडिंग है";
    int len = str.length();
    cout << "स्ट्रिंग की लंबाई = " << len << endl;
    return 0;
}
```

उपरोक्त कोड स्ट्रिंग "यह स्ट्रिंग कोडिंग है" की लंबाई को 18 के रूप में प्रिंट करेगा। आपको सिर्फ एक स्ट्रिंग के लंबाई को ध्यान से गिनने की जरूरत है और आप कोड में एक बार ही लिख सकते हैं। 

## गहराई में खुदाई

स्ट्रिंग की लंबाई को जानने के लिए, कोड में विशेष लेकिन सरल सुधार करने के लिए स्ट्रिंग क्लास में length() फंक्शन होती है। इस फंक्शन का उपयोग करके, हम किसी भी स्ट्रिंग की लंबाई को आसानी से प्राप्त कर सकते हैं। यह फंक्शन स्ट्रिंग के अंतिम अक्षर के बाद का एक न्यूलाइन भी जोड़ता है। इसलिए, अगर आप अंतिम अक्षर को न जोड़ना चाहते हैं, तो आपको अंतिम अक्षर को छोड़ना होगा। हालांकि, यह समस्या ज्यादातर प्रोग्रामिंग भाषाओं में दिखाई देती है और C++ में भी विस्तृत जानकारी के साथ तो इस्तेमाल की जा सकती है।
