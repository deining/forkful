## क्यों

हेलो दोस्तों! आज हम C++ में स्ट्रिंग की लंबाई को निकालने के बारे में बात करेंगे। यह विषय आपको शायद थोड़ा सा जटिल लगें, लेकिन दर्शकों को चिंता करने की जरूरत नहीं है। यह विषय C++ की बेसिक टॉपिक में से एक है और आपको इसे समझना बहुत ज़रूरी है। स्ट्रिंग की लंबाई को निकालना C++ में काफी उपयोगी होता है और इसे सीखना आपको आगे की पढ़ाई में बेहतर बना सकता है।

## कैसे करें

```C++
// स्ट्रिंग के लिए विभिन्न लॉगों को प्रिंट करें
#include <iostream>
#include <cstring>
using namespace std;

int main()
{
  char str[] = "हैलो दुनिया!";
  int len = strlen(str);

  // प्रिंट करें स्ट्रिंग का आकार
  cout << "स्ट्रिंग का आकार है: " << len << " अक्षरों में।" << endl;

  return 0;
}
```

आउटपुट:

```
स्ट्रिंग का आकार है: 13 अक्षरों में।
```

जैसा कि आप देख सकते हैं, हमने `strlen()` फंक्शन का उपयोग किया है जो स्ट्रिंग की लंबाई को निकालता है। यह फंक्शन `<cstring>` हैडर फ़ाइल से आता है और `char` स्ट्रिंग्स के लिए काम करता है। हमने फिर वर्ग `cout` का उपयोग किया है ताकि हम आकार को प्रिंट कर सकें। आप चाहे तो फंक्शन `strlen()` की जगह `length()` भी उपयोग कर सकते हैं।
