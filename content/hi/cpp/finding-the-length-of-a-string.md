## क्यों

श्रृंखला की लंबाई निकालने में लोग क्यों दिलचस्पी लेंगे, इसके बारे में कुछ सरल विवरण है।

## कैसे करें

जैसा कि आप सब जानते हैं, विभिन्न मापने की इकाइयां होती हैं, जिनमें से एक है श्रृंखला की लंबाई। इसको प्रोग्रामिंग में भी इसी तरह से नंबरों के साथ इस्तेमाल किया जाता है। यदि आपको किसी श्रृंखला की लंबाई को निकालना है, तो निम्नलिखित कोड को अपने प्रोग्राम में शामिल करके आसानी से कर सकते हैं:

```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
  string str = "हाय दोस्तों!";
  int len = str.length();
  
  cout << "श्रृंखला की लंबाई है: " << len << " अक्षर।" << endl;
  
return 0;
}
```

आउटपुट:
```
श्रृंखला की लंबाई है: 13 अक्षर।
```

## गहराई से जानिए

श्रृंखला की लंबाई निकालना बहुत आसान है। यह कई प्रोग्रामिंग भाषाओं में उपलब्ध है। हालांकि, यहां हमने सी++ का उपयोग किया है। यहां हम `string` लाइब्रेरी का उपयोग कर रहे हैं जो C++ में सदियों से मौजूद है। `length()` फ़ंक्शन हमें श्रृंखला की लंबाई को प्राप्त करने में सक्षम बनाता है।
