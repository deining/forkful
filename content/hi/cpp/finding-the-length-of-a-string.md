## क्यों 
अपने स्‍ट्रिंग की लंबाई पता करने में शामिल होने का कारण एक प्रोग्रामिंग कार्यक्षमता बढ़ाना हो सकता है, जहां आपको बार-बार स्‍ट्रिंग की लंबाई को जांचने की आवश्यकता होती है।

## कैसे करें
```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
   string str = "हैलो दुनिया";

   // लंबाई को जनने का तरीका # 1
   int length = str.length();
   cout << length << endl;

   // लंबाई को जानने का तरीका # 2
   int count = 0;
   for (int i = 0; str[i] != '\0'; i++) {
      count++;
   }
   cout << count << endl;
   return 0;
}
```

आप ऊपर दिए गए कोड ब्लॉक में दो तरीकों से स्‍ट्रिंग की लंबाई को जान सकते हैं। पहला तरीका `str.length()` फ़ंक्शन का उपयोग करके संभव है, जबकि दूसरा तरीका एक लूप का उपयोग करके स्ट्रिंग में अक्षरों की संख्या को गिनता है।

## गहराई से समझें
जब हम `str.length()` फ़ंक्शन का उपयोग करते हैं, तो वास्तव में यह स्ट्रिंग के साथ एक `size_t` प्रकार का विशिष्टांक वापस करता है, जिसमें स्ट्रिंग की लंबाई होती है। यदि हम दूसरा तरीका उपयोग करते हैं, तो हमें खाली स्थानों को गिनने के लिए भींव लगाना हो सकता है।
