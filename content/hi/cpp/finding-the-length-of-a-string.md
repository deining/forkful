## क्या और क्यों?

स्ट्रिंग की लंबाई को खोजने का अर्थ है, उस स्ट्रिंग में कितने अक्षर हैं। यह शायद सरल लगता हो, लेकिन इस जानकारी का अहमियत है क्योंकि प्रोग्रामर्स को चुनौतियों का सामना करना पड़ सकता है जब उन्हें स्ट्रिंग की लंबाई का पता होना हो।

## कैसे:

```C++
#include<iostream> 
using namespace std; 

int main() { 
    string str = "हैलो दोस्तों"; 
    int length = str.length(); 
    cout << "स्ट्रिंग की लंबाई है: " << length << endl; 
    
    return 0; 
}
```

आउटपुट: स्ट्रिंग की लंबाई है: 12

## गहराई में जानकारी:

### इतिहासिक परिस्थिति:

स्ट्रिंग की लंबाई को खोजने का आग इन-स्ट्रक्शनल प्रोग्रामिंग में था, जब 1960 के दशक में पहली डिजिटल कंप्यूटर आया था। तब स्ट्रिंग की लंबाई को खोजने के लिए एक विशेष फंक्शन का उपयोग किया जाता था।

### वैकल्पिक मार्ग:

आप इस समस्या को कई तरीकों से हल कर सकते हैं। एक अन्य तरीका एक स्ट्रिंगबफर का इस्तेमाल है, जो आपको स्ट्रिंग के अंत में खाली स्थान की गुणवत्ता को ढूंढने की अनुमति देता है। आप भी स्ट्रिंग क्लास के size फंक्शन का उपयोग कर सकते हैं, जो आपको स्ट्रिंग के बाइट संख्या देता है।

### अमलदारी विवरण:

स्ट्रिंग की लंबाई को खोजने के लिए, यहां हम string डेटा टाइप का उपयोग करते हैं जिसमें स्ट्रिंग के char मान पंक्तिबद्ध हैं, जबकि स्ट्रिंगबफर एक बाइटरों में संग्रहीत है। हालांकि, कई प्रोग्रामिंग भाषाओं में दोनों का उपयोग किया जाता है।
