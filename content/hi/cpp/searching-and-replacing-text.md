# क्यों
क्या आप भी कभी टेक्स्ट को सर्च और रिप्लेस करने में खो चुके हैं? अक्सर हम लंबे टेक्स्ट में भी कुछ नया वाक्य या शब्द जोड़ने के लिए समय खर्च करते हैं। इससे बचने के लिए, हमें इस प्रक्रिया को सुगम बनाने के लिए C++ प्रोग्रामिंग के आगे बढ़ना होगा।

## कैसे करें
टेक्स्ट को सर्च और रिप्लेस करने के लिए, हमें सबसे पहले स्ट्रिंग को स्पेसिफाइड तरीके से खोजना होगा। इसके लिए, हम `find()` फंक्शन का उपयोग कर सकते हैं जिससे स्ट्रिंग के अंदर वह अल्फाबेट की पहली अक्षर की पोजिशन मिल जाएगी। इसके बाद, हम `replace()` फंक्शन का उपयोग करके उसी पोजिशन पर नए शब्द को डाल सकते हैं। नीचे एक उदाहरण देखते हैं:

```C++
#include <iostream>
#include <string>
using namespace std;

int main(){
    string sentence = "मैं आमूल हूं";
    string searchWord = "आमूल";
    string replaceWord = "दूध";
    size_t position = sentence.find(searchWord);
    sentence.replace(position, searchWord.length(), replaceWord);
    cout << sentence << endl;
}
```

उपरोक्त कोड की output में, `sentence` का मान अब "मैं दूध हूं" होगा। इस तरह से हम यूंही बहुत सारे संबंधित शब्दों को एक साथ बदल सकते हैं।
