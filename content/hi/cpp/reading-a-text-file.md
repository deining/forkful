# क्यों

क्या आप एक पाठ फ़ाइल से डेटा को पढ़कर उसे अपने एप्लिकेशन में उपयोग करना चाहते हैं? अगर हाँ, तो आप सही जगह पर हैं। इस लेख में, हम आपको सी++ में पाठ फाइल से डेटा पढ़ने के तरीके के बारे में बताएंगे। इससे आपको अपने प्रोजेक्ट को ज्यादा उपयोगी बनाने में मदद मिलेगी।

# कैसे करें

सी++ में, हम "fstream" नामक एक बिल्ड-इन लाइब्रेरी का उपयोग करके पाठ फाइलों को पढ़ सकते हैं। निम्नलिखित कोड उदाहरण आपको इस प्रक्रिया को समझने में मदद करेगा।

```C++
#include <iostream>
#include <fstream>
using namespace std;

int main() {

  // सही फ़ाइल को खोलें
  ifstream myfile("example.txt");

  // फ़ाइल का स्ट्रीम की जांच करें
  if (myfile.is_open()) {

    // फ़ाइल से डेटा पढ़ें
    string data;
    while (getline(myfile, data)) {
      cout << data << endl;
    }

    // स्ट्रीम बंद करें
    myfile.close();
  }
  else {
    // फ़ाइल नहीं मिली, इसका मतलब आपके द्वारा दी गई फ़ाइल का पाठ सही नहीं है
    cout << "Unable to open file";
  }

  return 0;
}
```

आपको इस कोड को अपने सिस्टम में कॉपी करके अपनी खुद की फ़ाइल का पाठ बदलकर, "example.txt" फ़ाइल को खोलने और उसमें से डेटा पढ़ने का प्रयास करना चाहिए। अगर आपकी फ़ाइल सही है, तो आपको फ़ाइल के दस्तावेज प्रिंट होने लगेंगे।

# गहराई में

सी++ में, "fstream" लाइब्रेरी "ifstream" (input file stream) और "ofstream" (output file stream) दोनों को समर्थित करती है। आप फ़ाइल से डेटा पढ़ने के साथ-साथ उसे अपने प्रोग
