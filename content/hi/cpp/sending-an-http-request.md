## क्यों

आज के समय में इंटरनेट के जरिए जानकारी अत्यंत महत्वपूर्ण है। एक्सेस नदी की तरह आसानी से जानकारी प्राप्त करने के लिए हमें HTTP रिक्वेस्ट का उपयोग करना पड़ता है। इससे हम डेटा को दूसरे सर्वर से लेकर अपने डीवाइस पर प्राप्त कर सकते हैं। इसलिए, HTTP रिक्वेस्ट एक आवश्यक टूल है जो हमें जल्दी से अपनी जरूरत की जानकारी प्राप्त करने में मदद करता है।

## कैसे करें

``` 
#include <iostream>
#include <curl/curl.h>

int main(){
    CURL *curl;
    CURLcode res;
    curl = curl_easy_init();
    if(curl){
        curl_easy_setopt(curl, CURLOPT_URL, "http://www.example.com/");
        res = curl_easy_perform(curl);
        std::cout << res;
        // सत्यापन के लिए 0 को साफ़ कर दें
        curl_easy_cleanup(curl);
    }
    return 0;
}
```

यहां हमने सी++ में cURL लाइब्रेरी को उपयोग करके एक बेसिक HTTP रिक्वेस्ट भेजना सीखा है। कोड ब्लॉक में हमने cURL लाइब्रेरी और सूचनाएं शामिल किए हैं जो भेजी गई रिक्वेस्ट को सत्यापित करती हैं। कोड को कंपाइल करने और एक्सेक्यूट करने के बाद, हमें प्रत्यक्ष रूप से रिक्वेस्ट की सफलता का हल्का पता मिलता है।
