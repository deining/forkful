## क्यों

अभी की तारीख को प्राप्त करने में लोगों का क्या उपयोग है, क्या आपने कभी सोचा है? यह एक साधारण लग सकता है लेकिन उसके पीछे अनेक उपयोग हो सकते हैं। अधिक जानने के लिए पढ़िए!

## कैसे

```C++
#include <iostream>  //स्तंभ को सहेजने के लिए योगदान

int main() {
  //एक स्ट्रिंग को स्थानांतरित करें
  std::string currentDate = __DATE__;
  
  std::cout << "आज की तारीख है: " << currentDate << std::endl;
  
  return 0;
}
```

### उत्पाद

```
आज की तारीख है: Jan 17 2021
```

ऊपर दिए गए कोड अभी की तारीख को स्थानांतरित करने का एक उदाहरण है। हालांकि आप `ctime` और `localtime` जैसे पुस्तकालयों का भी उपयोग कर सकते हैं जो दिनांक और समय के विस्तृत डेटा प्रदान करते हैं।

## गहराई में घोर विवेचना

अभी की तारीख को प्राप्त करना आसान हो सकता है, लेकिन यह सिस्टम के समयगणित के साथ सुधार और साथी उत्पादों का उपयोग करने के बिना असंभव होता है। यह विशिष्ट समय और समय सूचित करने के लिए एक उपयोगी उपकरण हो सकता है जो प्रोग्रामर्स को अपनी आवश्यकताओं के अनुसार तारीख को संशोधित करने की अनुमति देता है।
