# क्यों

आज हम बात करेंगे की वर्तमान तारीख को प्राप्त करना क्यों महत्वपूर्ण है। अपने कोड में आप वर्तमान तारीख को जोड़कर अपने प्रोग्राम को और समझदार बना सकते हैं।

## कैसे करें

हमारे पास बहुत सारे तरीके हैं जिनसे आप वर्तमान तारीख को प्राप्त कर सकते हैं, लेकिन हम यहां सबसे सरल तरीके को बताएंगे जो सी++ में समर्थित है। पहले हम <code>ctime</code> के लिए लाइब्रेरी को इंक्लूड करते हैं। फिर हम वर्तमान तारीख को प्राप्त करने के लिए <code>time()</code> फ़ंक्शन का उपयोग कर सकते हैं। नीचे चलोड का कोड देखें:

```C++
# include <ctime>
int main() {
  time_t now = time(0);
  char* date = ctime(&now);
  cout << "The current date and time is: " << date << endl;
  return 0;
}
```

और यह होगा आपका आउटपुट:

```
The current date and time is: Mon Nov 23 15:10:42 2020
```
