## क्या और क्यों?

करेंट डेट प्राप्त करना ऐसा काम है जिसमें हम वर्तमान दिनांक और समय को प्राप्त करते हैं। प्रोग्रामर्स इसे उनके प्रोग्राम में वर्तमान दिनांक को स्थापित करने के लिए उपयोग करते हैं।

## कैसे करें?

जैसा कि नाम से पता चलता है, हम वर्तमान दिनांक और समय को स्थापित करने के लिए C++ उपयोग कर सकते हैं। नीचे दिए गए कोड ब्लॉक में हमारे एक साधारण उदाहरण है।

```C++
#include <iostream>
#include <ctime>

int main() {
    // Getting the current date using the time function
    std::time_t now = std::time(0);

    // Converting it to string format
    std::string current_date = std::ctime(&now);

    // Printing the current date
    std::cout << "Current date is: " << current_date << std::endl;

    return 0;
}
```

इस कोड को चलाने के लिए, आपको अपने टर्मिनल में निम्नलिखित कमांड को टाइप करना होगा:

```bash
g++ current_date.cpp
./a.out
```

इस उदाहरण के तहत, आप इस तरह का आउटपुट देखेंगे:

```bash
Current date is: Sat Mar 30 16:06:47 2019
```

## गहराई में जाएं

अब आप बता पा चुके हैं कि हम कैसे C++ में वर्तमान दिनांक और समय को स्थापित कर सकते हैं, लेकिन अगर आपको कुछ अतिरिक्त जानकारी चाहिए तो नीचे दी गई जानकारी आपके काम आ सकती है:

1. इस काम को क्रम में पूरा करने के लिए हमने सिस्टम की time.h हेडर फ़ाइल को उपयोग किया है।
2. अगर आपको C++ जानकारी नहीं है या आपके पास C++ का कोड कैसे लिखते हैं के बारे में कोई भी विशेष ज्ञान नहीं है, तो आपको इस काम को करने के लिए अन्य भाषाओं जैसे Java, Python आदि का भी उपयोग कर सकते हैं।
