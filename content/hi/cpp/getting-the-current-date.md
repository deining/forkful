## क्यों

क्या आपके मन में कभी सोचा है कि क्यों हम वर्तमान तिथि प्राप्त करने की कोशिश करते हैं? शायद आपको इसके पीछे का पूरा तर्क नहीं पता हो, लेकिन हमारे इस ब्लॉग पोस्ट में हम आपको उसके फ़ायदे बताएंगे और सी++ में वर्तमान तिथि कैसे प्राप्त कर सकते हैं।

## कैसे करें

```C++ 
#include <iostream>
#include <ctime>

using namespace std;

int main() {
    // current date/time based on current system
    time_t now = time(0);
    // convert now to string form
    char* dt = ctime(&now);
    
    cout << "अभी की तारीख और समय है: " << dt << endl;
    
    return 0;
}
```

उपरोक्त कोड ब्लॉक में, हमने जिस वस्तु तथा फ़ंक्शन का उपयोग किया है वह सी++ की रूपरेखा में पहले से ही उपलब्ध होता है। जहां समय को एक "कई" int संख्या के स्थान पर भेजा गया है, वह उस समय को स्ट्रिंग के साथ समायोजित कर देता है, जो वर्तमान तिथि और समय को प्रदर्शित करता है।

आप इस तरह से अपने कोड में वर्तमान तिथि और समय को प्राप्त कर सकते हैं। प्रयोग में, आपको आउटपुट के साथ वर्तमान तिथि और समय को प्रदर्शित करने के साथ-साथ अन्य और बहुत कुछ करना हो सकता है।
