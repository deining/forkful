## क्यों

आज के बदलते समय में, प्रोग्रामिंग दुनिया बहुत तेजी से बढ़ती जा रही है। इसलिए, आपको जानना आवश्यक है कि आप आज के दिन का तारीख कैसे प्राप्त कर सकते हैं। इस जानकारी के बिना, आप अपने प्रोग्राम को समय और तिथि से सम्बधित काम करने में कठिनाई झेल सकते हैं।

## कैसे करें

इस ब्लॉग पोस्ट में हम C++ प्रोग्रामिंग में आज की तारीख कैसे प्राप्त करें इसके बारे में बात करेंगे। यह प्रक्रिया बहुत ही आसान है और हम आपको कुछ स्टेप्स के माध्यम से इसकी जानकारी देंगे। प्रोग्रामिंग कोड संग्रह हमारी गाइड में दिए गए हैं:

```C++
#include <iostream>
#include <ctime>

using namespace std;

int main() {
  // वर्तमान समय और तारीख की प्राप्ति के लिए ऑब्जेक्ट बनाएं
  time_t current_time = time(0);
  
  // उसे स्थानीय समय के साथ विश्लेषण के लिए struct में रखें
  tm *local_time = localtime(&current_time);
  
  // तारीख और समय को अलग-अलग चरों में प्रिंट करें
  cout << "आज की तारीख: " << local_time->tm_mday << "." << local_time->tm_mon + 1 << "." << local_time->tm_year + 1900 << endl;
  cout << "वर्तमान समय: " << local_time->tm_hour << ":" << local_time->tm_min << ":" << local_time->tm_sec << endl;
  
  return 0;
}
```

उपरोक्त कोड का आउटपुट निम्न रूप में होगा:

```
आज की तारीख: 12.10.2021
वर्तमान समय: 10:00:00
```
