## क्यों
अपना नया प्रोजेक्ट शुरू करने में आपको क्या मदद मिल सकती है? लेकिन, अभी तक यह बात नहीं बताई गई है कि अपने नये प्रोजेक्ट का सबसे अच्छा तरीका क्या है। इस लेख में हम आपको एक तरीका बताएंगे जो आपको अपने नये प्रोजेक्ट को शुरू करने के लिए मदद मिलेगा।

## यह कैसे करें
```C++
#include <iostream>

// सबसे पहले, हम एक नया C++ फाइल बनाएंगे
// इसे मेरा_प्रोजेक्ट.cpp के नाम से सहेजेंगे
int main() {
    // स्टैंडर्ड आउटपुट के लिए एक मैसेज दिखाएंगे
    std::cout << "नमस्ते दुनिया!" << std::endl;
    // कन्सोल में "नमस्ते दुनिया!" प्रिंट होगा
    return 0;
}
```

ऊपर दिए गए कोड प्रयोग करके, हम एक नया C++ प्रोग्राम बना सकते हैं और उसमें "नमस्ते दुनिया!" का संदेश प्रिंट कर सकते हैं। आप इसके अलावा भी किसी भी आवश्यक लाइब्रेरी का प्रयोग करके अपने नए प्रोजेक्ट को शुरू कर सकते हैं।

## अंतर्निरीक्षण
नए प्रोजेक्ट के शुरू करने में एक बार आप C++ को समझ लेंगे, तो आप हर तरह के प्रोजेक्ट बनाने के लिए इस्तेमाल कर सकते हैं। आप अपने कोड को अलग-अलग फाइलों में भी बाँट सकते हैं और घातक भूलों से बच सकते हैं। इसके अलावा, आप ऑब्जेक्ट ओरिएंटेड प्रोग्रामिंग के साथ समझ भी पाएंगे जो आपको किसी भी प्रोग्राम को बेहतर बनाने में मदद करेगा
