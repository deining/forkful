## क्यों
एक नया प्रोजेक्ट शुरू करने में एक व्यक्ति को क्यों रुचि होगी, इसका संक्षिप्त समाधान है।

## कैसे करें
इस अनुच्छेद में हम प्रोग्रामिंग के संबंधित उदाहरण एवं सेल्फ एग्जम्पल के साथ सी++ को सुरू करने की प्रक्रिया के बारे में विस्तृत जानकारी देंगे। आपसे निवेदन है कि आप अनुष्ठान को देखें और समझें।

```C++
// सी++ के साथ अनुष्ठान कैसे शुरू करें

#include <iostream>

int main() {
  std::cout << "नमस्ते, दुनिया!" << std::endl;
  return 0;
}
```

**आउटपुट:**
```
नमस्ते, दुनिया!
```
 इस उदाहरण में, हमने `iostream` हीडर फ़ाइल को इनक्लूड किया है जो `iostream` लाइब्रेरी को जोड़ता है जिससे हम `std::cout` और `std::endl` का उपयोग कर सकें। `std::cout` हमारे स्क्रीन पर टेक्स्ट को दिखाता है और `std::endl` हमारे द्वारा प्रिंट किए गए स्ट्रिंग को नए लाइन में लाने के लिए उपयोग किया गया है।

## गहराई में प्रवेश करें
अब हम एक नए प्रोजेक्ट को शुरू करने की गहरा जाएँगे। प्रोग्रामिंग की दुनिया में एक नया प्रोजेक्ट शुरू करना थोड़ा डरावना हो सकता है, लेकिन नीचे दिए गए लिंक्स की मदद से आप आसानी से इसे शुरू कर सकते हैं। आपको एक एडिशनल ऑनलाइन कोर्स लेने की भी सिफारिश की जाती है।
