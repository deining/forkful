## क्या और क्यों?
कीपोजिटिंग स्ट्रिंग किसी भी स्ट्रिंग के पहले अक्षर को बड़े अक्षर में बदलने की प्रक्रिया है। प्रोग्रामर इसे पढ़ने और लेखने को सरल और स्पष्ट बनाने के लिए करते हैं।

## कैसे करें?
स्ट्रिंग के पहले अक्षर को बड़े अक्षर में बदलने के लिए, आपको दो तरीके हैं। एक आप स् कोड के जरिए कर सकते हैं और दूसरा बाइलिब के द्वारा।
```C++ 
#include <iostream> 
#include <cstring> 
using namespace std; 
  
int main() 
{ 
    // सभी शब्दों को कैपिटल में बदलें 
    string word = "hello world!"; 
  
    // पहले अक्षर की कैसे बदलें
    word[0] = toupper(word[0]); 
    cout << word << endl; 
  
    return 0; 
}

```
आउटपुट:
Hello world!

## गहराई हो जाइये
इतिहास के कोने में, किस तरह के स्ट्रिंग प्रोसेसिंग को कैसे जन्म दिया गया है और ये संभव कैरिजें दिखाता है। दूसरे विकल्पों जैसे कि सूचना खूद के इस्तमाल भी किया जा सकता है।
पूर्वानुमानित संगतत और ये है कि कि C++ सभी जानकारी जैसे कि स्थान को स्थानान्त करने के कोड पर निर्भर है। स्ट्रिंग की तकनीक CSP सबसे ज्यादा प्रभावशाली रूप से HAL पुरे होता है।
