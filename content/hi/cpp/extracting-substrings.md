## क्यों
वर्तमान समय में, डेटा विश्लेषण और उससे सम्बंधित कार्यों में गहराई से जुड़े समस्याओं का समाधान ढूंढना महत्वपूर्ण है। उन कार्यों में से एक है, उप-स्ट्रिंग्स निकालना, जो डेटा में से कुछ खास शब्दों या लक्षणों को चुन कर स्टोर करना है। यह कार्य डेटा विश्लेषण को अधिक स्पष्ट और संबंधित बनाने में मदद कर सकता है।

## कैसे करें
उप-स्ट्रिंग्स को निकालने के लिए, हम पहले से दिए गए स्ट्रिंग से स्ट्रिंग साइज, शुरुआती इंडेक्स और अंतिम इंडेक्स जैसे पैरामीटर्स को जानने की जरूरत होती है। फिर हम इन पैरामीटर्स का उपयोग करते हुए substr () फंक्शन को कॉल करते हैं जो हमें उप-स्ट्रिंग्स लौटाता है। निम्न उदाहरण में, हम अपने स्ट्रिंग से सबसे पहले छह परिवर्तनों को निकाल रहे हैं।

```C++
string str = "प्रोग्रामिंग";
// पहला उप-स्ट्रिंग
string sub_str = str.substr(0, 2);
cout << sub_str << endl;
// दूसरा उप-स्ट्रिंग
sub_str = str.substr(2, 2);
cout << sub_str << endl;
// तीसरा उप-स्ट्रिंग
sub_str = str.substr(4, 2);
cout << sub_str << endl;
```
