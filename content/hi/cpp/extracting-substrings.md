## क्यों

क्या आपने कभी भी एक बड़े टेक्स्ट स्ट्रिंग से कुछ छोटे से स्ट्रिंग्स निकालने की कोशिश की है? सायद आपको कोई फॉर्मेट या प्रकार का पता नहीं है जिसमें छोटी स्ट्रिंग शामिल होती है। इसलिए, हम सभी को कभी न कभी स्ट्रिंग प्रोग्रामिंग की जरूरत होती है। धन्यवाद, C++ में स्ट्रिंग से स्ट्रिंग छोड़ना बहुत ही आसान और मज़े का काम है।

## कैसे करें

आप 'substr' फंक्शन का उपयोग करके स्ट्रिंग से स्ट्रिंग उत्पन्न कर सकते हैं। यहां मैं आपको एक उदाहरण दिखाऊंगा:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    string bigString = "Hello, World!";
    string smallString = bigString.substr(0, 5);
    cout << smallString; // Output: Hello
}
```

ऊपर दिए गए कोड में, हमने 'substr' फंक्शन का उपयोग करके 'Hello, World!' स्ट्रिंग से पहले 5 अक्षरों को निकाला है। आप जहां भी चाहें, वहां से किसी भी स्ट्रिंग से स्ट्रिंग छोटी स्ट्रिंग हासिल कर सकते हैं।

## गहराई में जाएं

स्ट्रिंग से स्ट्रिंग छोड़ना बहुत ही महत्वपूर्ण है। आपको इसे जानने की आवश्यकता हो सकती है, जिससे आप अपने कोड को अधिक स्वास्थ्यपूर्ण और उत्पादक बना सकें। स्ट्रिंग से स्ट्रिंग छोड़ना सिर्फ C++ में ही नहीं, बल्कि अन्य प्रोग्रामिंग भाषाओं में भी इस्तेमाल किया जाता है।
