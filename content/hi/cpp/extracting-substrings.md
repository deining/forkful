## क्यों

जब आप एक विशिष्ट तारीख को प्रस्तुत करने या उसमें से कुछ हिस्से को प्राप्त करने की आवश्यकता होती है, तो आप स्ट्रिंग्स से उप-स्ट्रिंग्स को निकालने के लिए ईंधन को उपयोग कर सकते हैं।

## कैसे करें

```C++
// प्रारंभिक स्ट्रिंग बनाएं
string str = "मैं एक ऊंचाई से उस पर्वत पर ऊपर ऊंचाई से नीचे जाता हूँ।";

// सभी उप-स्ट्रिंग्स को प्रत्येक समूह के रूप में निकालकर स्टोर करें
vector<string> substrings;

// सबसे पहले उप-स्ट्रिंग की स्थानिक मान स्थापित करें
int start = 0;

// उप-स्ट्रिंग होने का संभावित समाप्ति अनुक्रम का पता लगाएं
while(start < str.length()) {
  // समूह का अंतिम स्थानिक मान निकालें
  int end = str.find(" ", start); 
  
  // उप-स्ट्रिंग को स्टोर करें
  substrings.push_back(str.substr(start, end - start));
  
  // अगली समूह का प्रारंभिक स्थानिक मान सेट करें
  start = end + 1; 
}

// स्टोर किए गए सभी उप-स्ट्रिंग्स को प्रिंट करें
for(int i = 0; i < substrings.size(); i++) {
  cout << substrings[i] << endl;
}

// प्रकाशवर्ष मीटर में उप-स्ट्रिंग निकालें
double meters = stod(substrings[6]);

cout << meters * 9.461e+15 << " मीटर" << endl;
```
