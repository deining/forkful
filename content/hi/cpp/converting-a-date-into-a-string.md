## क्यों

जब हम किसी प्रोग्राम या एप्लिकेशन को बनाते हैं तो जरुरत पड़ती है कि हम अपने डेटा को बेहतर तरीके से प्रदर्शित कर सकें। डेटा को स्ट्रिंग में बदलना इस मामले में बहुत उपयोगी हो सकता है और इस ब्लॉग पोस्ट में हम आपको बताएंगे कि आप अपने कोड में डेटा को स्ट्रिंग में कैसे बदल सकते हैं।

## कैसे करें

डेटा को स्ट्रिंग में बदलने के लिए हम `std::to_string()` फंक्शन का उपयोग कर सकते हैं। इस फंक्शन का उपयोग करके हम अपने दिए गए डेटा को आसानी से स्ट्रिंग में बदल सकते हैं। इससे आपको कोई नया वेरिएबल बनाने की जरुरत नहीं पड़ती है, बल्कि आप चाहे तो अपने भविष्यवाणियों में यही स्ट्रिंग वेरिएबल को भी उपयोग कर सकते हैं। यह आपको केवल `#include <string>` कोड फाइल में जोड़ने के लिए काफी है। नीचे दिए गए कोड ब्लॉक में आप इसका उदाहरण देख सकते हैं।

```C++
#include <iostream>
#include <string>

int main() {
    double num = 5.5;
    std::cout << "Double value: " << num << std::endl;

    // convert to string
    std::string num_str = std::to_string(num);
    std::cout << "String value: " << num_str << std::endl;

    return 0;
}
```

आउटपुट:
Double value: 5.5
String value: 5.500000

अगर आपके पास यहाँ दिखाए गए समय का प्रकार का डेटा है तो आप `std::put_time()` फंक्शन का भी उपयोग कर सकते हैं। यह फंक्शन `#include <iomanip>` कोड फाइल में जोड़कर उपलब्ध होता है। इससे आप अपने डेटा को अपनी मनपसंद फॉर्मेट में प्रदर्शित कर सकते
