## क्यों:

काम को ट्रैक करने के लिए हमें अक्सर तारीख को स्ट्रिंग में बदलने की जरूरत होती है। यह एक सामान्य कार्य है जो कई एप्लिकेशन्स और वेबसाइट्स में किया जाता है। तारीख को स्ट्रिंग में बदलना हमें तारीख को प्रिंट करने, ऑनलाइन आर्काइव करने या दूसरे तारीख फॉर्मैट के साथ तुलना करने की सुविधा प्रदान करता है। 

## कैसे:

```C++
// दिनांक को वर्णमाला में बदलें
#include <iostream>
#include <string>
#include <ctime>
 
int main()
{
    
    // वर्तमान समय लायें
    time_t currentTime = time(0);
    
    // समय संरचना बनाएं
    tm *currentDate = localtime(&currentTime);
    
    // तारीख को वर्णमाला में बदलें
    char dateStr[11];
    strftime(dateStr, 11, "%d/%m/%Y", currentDate);
    
    // वर्णमाला में तारीख प्रिंट करें
    std::cout << "वर्तमान तारीख: " << dateStr << std::endl;
    
    return 0;
}
```

आउटपुट: वर्तमान तारीख: 19/07/2021

## गहराई में जाएं:

तारीख को स्ट्रिंग में बदलने के लिए, हम  `<ctime>`  हेडर फ़ाइल का उपयोग करते हैं जो ग्रीगोरियन क्यूलेंडर के माध्यम से तारीख को प्रबंधित करता है। हम `time()` फ़ंक्शन का उपयोग वर्तमान समय को प्राप्त करने के लिए कर सकते हैं और `tm` संरचना का उपयोग करके समय को तारीख में वर्णमाला रूप में परिवर्तित कर सकते हैं। तारीख को विभिन्न तारीख मापदंडों के साथ वर्णमाला में परिवर्तित करने के लिए, हम `strftime()` फ़ंक्शन का उपयोग कर सकते हैं। यह फ़ंक
