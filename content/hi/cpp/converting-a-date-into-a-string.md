## क्यों

कभी-कभी हमें अपने ऐप्लिकेशन या प्रोग्राम में तारीखों के साथ काम करने की आवश्यकता होती है, जो हमें अपने विभिन्न स्टाइल या यूजर इंटरफेस के साथ एक ही रूप में दिखाना होता है। ऐसे में, हम तारीख को एक स्ट्रिंग में बदलने के लिए आसान तरीके को जानने की जरूरत होती है।

## कैसे करे

```C++
#include <iostream>
#include <string>
#include <sstream>

using namespace std;

int main() {
    // हम यहां एक साधारण तारीख को स्ट्रिंग में बदलने का एक उदाहरण दिखाएंगे
    // इसकी मदद से हम स्ट्रिंग से तारीख को प्राप्त कर सकते हैं
    
    // यहां हम तारीख को स्ट्रिंग में अपडेट करने के लिए stringstream इस्तेमाल करेंगे
    stringstream date_string;
    
    // हम बी.सी. से प्राप्त तारीख को वर्तमान स्ट्रिंग में सेट कर सकते हैं
    int day = 18;
    int month = 4;
    int year = 2021;
    
    // यहां हम स्ट्रिंग को फॉर्मेट से तारीख में अपडेट करेंगे
    date_string << month << "/" << day << "/" << year;
    
    // यहां हम स्ट्रिंग को कनवर्ट करते हैं और उसे प्रिंट करते हैं
    string date = date_string.str();
    cout << date << endl; // आउटपुट: 4/18/2021
    
    return 0;
}
```
