## क्यों

क्या आपने कभी सोचा है कि आप टेक्स्ट फ़ाइल्स के साथ जुड़िये क्योंकि वे आपके लिए काफी उपयोगी हो सकते हैं? इस ब्लॉग पोस्ट में आपको यह समझेंगे कि टेक्स्ट फ़ाइल्स क्योंीकः आपके प्रोग्रामिंग में आपको क्या मदद कर सकती हैं।

## कैसे करें

अगर आप एक प्रोग्रामर हैं और आपको काम से फ़रार होने का ऑप्शन नहीं है तो आपका काम आसानी से शारीरिक उपलब्धियों पर संगत बन सकता है। इसके साथ साथ, आप टेक्स्ट फ़ाइल को अपने कंप्यूटर की यादिली समझने में मदद कर सकते हैं।

```C++
#include<iostream>
#include<fstream>

int main()
{
    // फ़ाइल खोलना
    std::ofstream file("example.txt");

    // टेक्स्ट लिखना
    file << "हैलो दुनिया!" << std::endl;

    // फ़ाइल बंद करना
    file.close();
    return 0;
}
```

आपको अपने कंप्यूटर पर example.txt नाम की एक नई टेक्स्ट फ़ाइल देखनी चाहिए जो "हैलो दुनिया!" लिखा हुआ होगा।
