## क्यों

आजकल हम सभी स्थानों पर कम्प्यूटर या अन्य इलेक्ट्रॉनिक डिवाइसों का उपयोग करते हैं। और कई बार हमें अपनी प्रासंगिक जानकारी को साझा करने के लिए टेक्स्ट फाइल की आवश्यकता होती है। आजकल हम आमतौर पर विभिन्न प्रोग्रामिंग भाषाओं में कोडिंग का उपयोग करके टेक्स्ट फाइल बनाते हैं। इस लेख में हम आपको सी++ भाषा में प्रोग्रामिंग करके टेक्स्ट फाइल बनाने का तरीका बताएंगे।

## कैसे करें

सबसे पहले, हम एक `ofstream` ऑब्जेक्ट बनाते हैं जिसका उपयोग फाइल में डेटा लिखने के लिए किया जाता है। इसके बाद हम `open()` फंक्शन की मदद से फाइल को खोलते हैं जिसमें हम लिखना चाहते हैं। `close()` फंक्शन की मदद से हम फाइल को सेव करते हैं और बंद करते हैं। इसके बाद हम `<<` ऑपरेटर का इस्तेमाल करके फाइल में डेटा लिख सकते हैं। यहां हम इसका उपयोग करके "Hello World!" लिखेंगे।

```C++
#include <iostream>
#include <fstream>

using namespace std;

int main() {
  ofstream myFile;
  myFile.open("myFile.txt");
  myFile << "Hello World!";
  myFile.close();
  
  return 0;
}
```

इस उदाहरण के बाद, हमारी `myFile.txt` फाइल में "Hello World!" उत्पन्न होगा। आप इस तकनीक का उपयोग करके टेक्स्ट फाइल में अपनी पसंदीदा जानकारी या डेटा को लिख सकते हैं।
