वर्तमान में, कंप्यूटर प्रोग्रामिंग हमारे समय का एक अनिवार्य हिस्सा बन गया है। सभी तरह के आधुनिक उपकरण और सेवाओं के लिए हमें कंप्यूटर प्रोग्रामिंग की आवश्यकता होती है। स्प्रेडशीट, ग्राफिक्स डिजाइन, गेम विकास और अन्य कई क्षेत्रों में कंप्यूटर प्रोग्रामिंग एक महत्वपूर्ण कौशल है। 

## क्यों

कंप्यूटर प्रोग्रामिंग में उपयोग किए जाने वाले अनुभव को सहेजने और साझा करने के लिए टेक्स्ट फाइल आपके लिए बहुत उपयोगी हो सकता है। एक टेक्स्ट फाइल बनाने के लिए सबसे आसान तरीका है C++ का उपयोग करना है। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि C++ में एक टेक्स्ट फाइल कैसे बनाया जाता है और उसमें क्या काम किया जा सकता है। 

## कैसे

टेक्स्ट फाइल बनाने के लिए सबसे पहले हमें एक फाइल पहचान करनी होगी। फाइल पहचान के लिए हम फ़ंक्शन "ofstream" का उपयोग करेंगे। एक डबल रिकार्ड जैसे "output.txt" फाइल नाम आपके पसंद को भी मिल सकता है। नीचे दिए गए उदाहरण में हम एक नए फाइल बनाते हैं, टेक्स्ट लिखें और फिर उसे बंद करें।

```C++
#include <iostream>
#include <fstream>

using namespace std;

int main() {
    // फाइल बनाएं
    ofstream myFile("output.txt");
    // टेक्स्ट लिखें
    myFile << "यह एक उदाहरण है।" << endl;
    // फाइल बंद करें
    myFile.close();
    
    return 0;
}
```
