## इसका मकसद

कोई भी पाठ फाइल लिखने का मकसद है कि हम जानकारी या डेटा को संग्रहीत कर सकें और उसे बाद में अपनी जरूरत के अनुसार प्रयोग कर सकें। यह भी हमें अपनी जानकारी को सीमित स्थान पर संचित रखने की सुविधा भी प्रदान करती है।

## कैसे करें

यदि आप किसी पाठ फाइल को संग्रहीत करना चाहते हैं तो सबसे पहले आपको उसे लिखना होगा। आपको C++ में ```ofstream``` का उपयोग करके फाइल खोलनी होगी और फिर आप उसमें अपनी जानकारी लिख सकते हैं। आइए नीचे दिए गए सामान्य उदाहरण के माध्यम से इस कार्य से जुड़ी कुछ जानकारी प्राप्त करें:

```C++
#include <iostream>
#include <fstream>

using namespace std;

int main(){
    ofstream myfile; // फाइल का ऑब्जेक्ट बनाएं
    myfile.open("data.txt"); // फाइल खोलें, आप अपनी मर्ज़ी से नाम दे सकते हैं
    myfile << "मैं इस फाइल में कुछ जानकारी लिख रहा हूँ।"; // जानकारी भेजें
    myfile.close(); // फाइल को बंद करें

    return 0;
}
```

जैसा कि आप देख सकते हैं, हमने ```ofstream``` का उपयोग करके एक फाइल खोली और फिर उसमें अपनी जानकारी भेजी। आप स्वयं अपने आवश्यकताओं के अनुसार फाइल नाम और जानकारी बदल सकते हैं।
