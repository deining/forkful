## क्यों
क्या आपने कभी एक स्ट्रिंग को लोअर केस में बदलने की कोशिश की है? यह एक आम समस्या है जो हर देवलपर को समान है। स्ट्रिंग को लोअर केस में बदलने से प्रोग्रामिंग अनेक तरह से आसान हो जाता है और आप स्ट्रिंग को संबोधित करने के लिए बहुत सारे उपकरण बन सकते हैं। इस ब्लॉग पोस्ट में, हम यह जानेंगे कि स्ट्रिंग को लोअर केस में कैसे बदला जा सकता है।

## कैसे करें
हिंदी में स्ट्रिंग को लोअर केस में बदलने के लिए, हमें सिस्टम सामग्री को छोड़कर स्ट्रिंग के अंत में `tolower()` फंगशन का उपयोग करना होगा। यह फंगशन स्ट्रिंग को एक लोअर केस स्ट्रिंग में रूपांतरित करता है। नीचे दिए गए कोड ब्लॉक में हमने इसका एक उदाहरण दिखाया है।

```C++
#include <iostream>
#include <string>
using namespace std;

int main() {
	string str = "HELLO WORLD";
	for (int i = 0; i < str.length(); i++) {
		str[i] = tolower(str[i]);
	}
	cout << str; // Output: hello world
	return 0;
}
```

## गहराई में जाएं
स्ट्रिंग को लोअर केस में बदलने की तकनीक है सिस्टम सामग्री को छोड़ने के लिए स्ट्रिंग के अंत में फंगशन का उपयोग करना। इसके अलावा, आप अपने कोड में स्ट्रिंग के अंत में `tolower()` फंगशन के साथ में दोबारा स्ट्रिंग की सामग्री को लोअर कर सकते हैं। इससे आपको दोबारा स्ट्रिंग को आपको प्रिंट करने की ज़रूरत नहीं पड़ेगी।
