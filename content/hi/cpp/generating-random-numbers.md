## क्यों

क्या आपको पता है कि सम्पूर्ण प्रोग्रामिंग दुनिया में दर्शायी गयी चीज़ों में से बहुत सारे रैंडम नंबर्स प्रयोग किए जाते हैं? रैंडम नंबर्स का उपयोग गेम डेवलपमेंट, सुरक्षा एगेंट्स, और अन्य एप्लीकेशन में भी किया जाता है। यदि आप कोई प्रोग्रामर हैं तो रैंडम नंबर्स की उत्पत्ति सीखना सबसे आवश्यक है।

## कैसे करें

```C++
#include <iostream>
#include <cstdlib>
#include <ctime>

int main()
{
    // जर्जर अपने समय निर्धारित करें
    srand(time(0));

    // 1 से 100 तक के बीच एक रैंडम नंबर बनाएँ
    int random_number = rand() % 100 + 1;

    // रैंडम नंबर प्रिंट करें
    std::cout << "रैंडम नंबर: " << random_number << std::endl;

    return 0;
}
```
 
आपको उपरोक्त कोड से रैंडम नंबर का प्रदर्शन मिलेगा। यहाँ हमने समय को यूटिलाइज करके जर्जर को सेट किया है इससे हर बार अलग रैंडम नंबर बनाया जाएगा। आप ये कोड अपने आवश्यकतानुसार बदल सकते हैं।

## गहराई तक

यदि आप गहराई तक जानना चाहते हैं कि कैसे रैंडम नंबर्स बनाए जाते हैं और उनके पीछे की विज्ञान है तो यह आर्टिकल आपके लिए है। रैंडम नंबर्स का उपयोग कुछ कामों को आसान बनाने के लिए किया जाता है जैसे कि गेमिंग और सुरक्षा उद्देश्यों के लिए। इसमें विभिन्न एल्गोरिदम, जैसे कि लिनियल कॉन्ग्रूयेंशियल जनरेट
