## क्यों

आजकल कॉम्प्यूटर प्रोग्रामिंग ज्ञान का होना बहुत जरूरी है। हैकर्स भी यहां अपने कौशलों का प्रयोग नहीं करते हैं बल्कि अपने उत्पादों को चुनौतीपूर्ण और सुरक्षित बनाने के लिए भी उनका उपयोग करते हैं। इसलिए, टेम्पररी फ़ाइलें तैयार करना उन्हें प्रोग्रामिंग में थोड़ी और मदद कर सकता है।

## कैसे करें

```C++
#include <iostream>
#include <fstream>
#include <cstdlib>

int main() {
	// टेम्पररी फ़ाइल नाम
	char temp_name[L_tmpnam];
	// टेम्पररी फ़ाइल नाम बनाएं
	std::tmpnam(temp_name);
	// टेम्पररी फ़ाइल खोलें
	std::ofstream temp_file(temp_name);
	// टेम्पररी फ़ाइल में कुछ डेटा लिखें
	temp_file << "इस फ़ाइल में टेम्प डेटा है";
	temp_file.close();
	// फ़ाइल हटाएं
	std::remove(temp_name);

	return 0;
}
```

आप ऊपर दिए गए कोड ब्लॉक में देख सकते हैं कि टेम्पररी फ़ाइल कैसे बनती है और उसमें डेटा कैसे लिखते हैं। नीचे दिए गए आउटपुट में आप देख सकते हैं कि टेम्पररी फ़ाइल एक ग़लत नाम से बनी है और फिर ग़लत नाम से हटा दी गई है।

```
इस फ़ाइल में टेम्प डेटा है
```
