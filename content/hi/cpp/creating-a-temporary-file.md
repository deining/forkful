## क्यों

टेम्पोरेरी फाइल बनाने का काम क्यों किया जाता है? यह फाइल उस समय बनाई जाती है जब आपको अपने प्रोग्राम में मिडियरी स्थान की जरूरत होती है, ठीक उसी तरह जब आप किसी फाइल को स्टोर करने के लिए एक गोपनीय तरीके से वर्क कर रहे हैं।

### कैसे

यहाँ, मैं आपको C ++ में टेम्पोरेरी फाइल बनाने के प्रभावी तरीके बताउँगा। आपको स्पष्ट रूप से दिखाया जाएगा कि कैसे आप कोड के माध्यम से अपने प्रोग्राम में टेम्पोरेरी फाइल बना सकते हैं।

```C++
#include <iostream> 
#include <fstream> 
using namespace std; 
int main() { 
	// Create a temporary file using ofstream 
	ofstream tempFile("temp.txt"); 
	// Write some text to the file 
	tempFile << "This is a temporary file."; 
	// Close the file 
	tempFile.close(); 
	// Delete the file 
	remove("temp.txt"); 
	return 0; 
} 
```

उपरोक्त कोड को चलाने के बाद, आपको एक नई ग्राहक फ़ोल्डर में `temp.txt` फाइल बनाई गई होगी जो आपके फ़ाइल सिस्टम पर स्थायी रूप से उपलब्ध नहीं होगी।

### गहरी खोज

टेम्पोररी फाइल्स बनाने के पीछे क्या समानों हैं? आप कभी भी किसी अन्य फ़ाइल को मंदिर कर सकते हैं खासकर अगर आप यहाँ पहले ही एक पुराना ठिकाना बना चुके हैं, अगर आप अपने स्वयं के तरीके से उपयोग करना चाहें तो आपको पता होना चाहिए कि कैसे आप उसे नष्ट करें। टेम्पोररी फाइल्स के साथ, आपको चिंता करने की ज़रुरत नहीं है क्योंकि अपने आप उन्हें सिस्टम से हटा दिया जाता है।
