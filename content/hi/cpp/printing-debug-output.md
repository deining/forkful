## क्यों

कभी-कभी, हम प्रोग्रामिंग के दौरान अपने कोड में कुछ त्रुटियां ठीक करने के लिए डिबग आउटपुट का सहारा लेते हैं। यह हमें अपने कोड को समझने में मदद करता है और सही समाधान खोजने में भी सहायता प्रदान करता है।

## कैसे करें

एक साधारण तरीका डिबग आउटपुट को प्रिंट करने का है लेकिन इससे कोड को अधिक स्पष्ट बनाने की आवश्यकता है। सुनिश्चित करें कि आप अपने कोड में "cout" या "printf" कमांड का प्रयोग करते हैं, और कोड के अनुकूलन के लिए आवश्यकतानुसार वेरिएबल्स को प्रिंट करें। इस विशेष उदाहरण में, हम आउटपुट में "Hello World" को प्रिंट करेंगे।

```C++
#include <iostream>
using namespace std;

int main() {
    int num = 10;
    cout << "The value of num is: " << num << endl;
    cout << "Hello World" << endl;
    return 0;
}
```

इस उदाहरण में, पहली "cout" स्टेटमेंट ने वेरिएबल "num" को प्रिंट किया है और दूसरी "cout" स्टेटमेंट ने "Hello World" शब्द प्रिंट किया है। वेरिएबल "num" की मान 10 है, इसलिए पहली आउटपुट लाइन "The value of num is: 10" होगी। आपने देखा होगा कि हमने "cout" के बाद एक रेखा ब्रैकेट शामिल नहीं किया है, यह संकेत प्रिंट लाइन को नए लाइन पर रखेगा।
