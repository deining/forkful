## क्यों:

कभी-कभी हमारी प्रोग्राम में कुछ समस्याओं का सामना करना पड़ता है जो हमारे द्वारा बनाई गई कोड में हैं। ऐसे में डीबगिंग आउटपुट को प्रिंट करके इन समस्याओं का पता लगाना आसान हो जाता है। 

## कैसे करे:

 "```C++
 #include <iostream>
 
 using namespace std;
 
 int main() {
    int num1 = 10;
    int num2 = 20;
    int result = num1 + num2;
    
    //यहां डीबगिंग आउटपुट प्रिंट करें 
    cout << "गणना का परिणाम: " << result << endl;
    
    return 0;
 }
 ```"

उपरोक्त कोड में, हमने दो संख्याओं को जोड़ा है और इसे डीबगिंग आउटपुट के रूप में प्रिंट किया है। इस तरह से हम अपने कोड में गलती को ढूंढ सकते हैं और उसे सुधार सकते हैं। 

## गहराई से जानें:

डीबगिंग आउटपुट प्रिंट करने का एक और तरीका है `cerr` स्ट्रीम का उपयोग करना। इससे हम अपने कोड में स्पष्टिकरण के लिए भी इस्तेमाल कर सकते हैं और समस्याओं को नॉन-इंटरेक्टिव मोड में सुलझा सकते हैं। साथ ही, हम यह भी सुनिश्चित कर सकते हैं कि क्यों हमारी कोड में कुछ गलती है।
