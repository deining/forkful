## क्यों
कमांड लाइन आर्ग्यूमेंट पढ़ना आपके कोड में एक अन्य बात को अनुमति देता है और आपको सुरक्षा और उपयोगिता के लिए विभिन्न विकल्पों को सक्षम करता है।

## कैसे करें
```C++
int main(int argc, char* argv[]) {
    // मूल्य को आर्ग्यूमेंट में परखें
    if (argc > 1) {
        // आर्ग्यूमेंट से मूल्य प्राप्त करें
        int value = atoi(argv[1]);
        // मूल्य निर्धारित करें
        std::cout << "मूल्य: " << value << "\n";
    }
    
    return 0;
}
```

उपरोक्त कोड उदाहरण देता है कि आप कैसे आर्ग्यूमेंट लाइन से मूल्य पढ़ सकते हैं और इसे कैसे अपने कोड में उपयोग कर सकते हैं। आर्ग्यूमेंट स्ट्रिंग से संख्याओं को पढ़ने के लिए आप atoi() या std::stoi() जैसे फ़ंक्शन का उपयोग कर सकते हैं। आप भी आर्ग्यूमेंट पढ़ने के लिए दुसरे तरीकों को अपना सकते हैं, जैसे स्विच केस नियंत्रण या आर्ग्यूमेंट पर text देखना।

## गहराई
आर्ग्यूमेंट्स को पढ़ने से पहले, सुनिश्चित करें कि आपका कोड जाँचता है कि वे कीबोर्ड से या अन्य स्रोत से नहीं आ रहे हैं। आप भी निश्चित कर सकते हैं कि स्ट्रिंग प्रवीण तार्किक और सक्रिय बचाया गया है। आप आर्ग्यूमेंट स्क्रिप्टंंग के साथ उपयोगी उपकरण प्रदान कर सकते हैं, जो आपको स्वचालित रूप से आर्ग्यूमेंट पढ़ते हैं और इसे अपने फाइलों में सभी उपयोगिता को
