## क्यों

आज कल हर दिन नए-नए सॉफ्टवेयर और ऐप्स आ रहे हैं। ये सॉफ्टवेयर और ऐप्स उपयोगकर्ताओं को बेहतर अनुभव प्रदान करने के लिए बनाए जाते हैं और उनके काम को सरल बनाने के लिए कई तरीकों का उपयोग किया जाता है। इस में से एक तरीका है कमांड लाइन आर्ग्यूमेंट पढ़ना। इस लेख में हम बात करेंगे कि आप कैसे अपने सॉफ्टवेयर या ऐप के लिए कमांड लाइन आर्ग्यूमेंट पढ़ सकते हैं और इससे आपको क्या लाभ हो सकता है।

## कैसे करें

इस कार्य को पूरा करने के लिए हमें उन भागों की आवश्यकता होती है जो सी++ में सीखे जाने पर ही बनते हैं, जैसे स्ट्रिंग, मत्रिका और फ़ोर लूप। यदि आप ये सब कहीं भी सीख चुके हैं तो आप आसानी से कमांड लाइन आर्ग्यूमेंट पढ़ सकते हैं। यदि आपने अभी तक इन बातों को नहीं सीखा है तो चिंता न करें, ये सीखना आसान है।

```C++
#include <iostream>
int main(int argc, char *argv[]) {
    std::cout << "कमांड लाइन आर्ग्यूमेंट पढ़ें" << std::endl;
    std::cout << "कुल आर्ग्यूमेंट्स: " << argc << std::endl;
    for (int i = 0; i < argc; ++i) {
        std::cout << argv[i] << std::endl;
    }
    return 0;
}
```

उपरोक्त कोड में हम आर्ग्यूमेंट की संख्या को प्रिंट किया है और फ़ोर लूप का उपयोग करके सभी आर्ग्यूमेंट्स को प्रिंट किया है। यदि आप इस को अपने कंप्यूटर पर सेव करके यदि आपके
