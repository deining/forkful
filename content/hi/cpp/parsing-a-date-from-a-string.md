## क्या और क्यों?

पार्सिंग एक तारीख को एक स्ट्रिंग से अलग करना है, यह एक तारीख को प्रोग्रामरों के लिए एक अतिरिक्त डाटा टाइप के रूप में उपयोगी बनाता है। इससे प्रोग्रामर आसानी से डेटा को विभिन्न मानों में प्रस्तुत कर सकते हैं।

## कैसे करें:

```C++
#include <iostream>
#include <string>

using namespace std;

int main() {
    // नमूना डेटा स्ट्रिंग
    string date_string = "16/05/2021";

    // स्ट्रिंग को स्प्लिट करने के लिए स्ट्रिंग स्ट्रीम्स का उपयोग करें
    stringstream ss(date_string);

    // स्ट्रिंग से तारीख भागों में अलग करें
    string date;
    getline(ss, date, '/'); // पहला भाग - दिन
    string month;
    getline(ss, month, '/'); // दूसरा भाग - महीना
    string year;
    getline(ss, year); // तीसरा भाग - साल

    // तारीख, महीना और साल कि स्ट्रिंग को अलग वेरिएबल्स को स्टोर करें
    int day = stoi(date); // day = 16
    int mth = stoi(month); // mth = 05
    int yr = stoi(year); // yr = 2021

    // स्ट्रिंग से स्क्रीन पर स्प्लिट किया गया तारीख प्रिंट करें
    cout << "Day: " << day << endl;
    cout << "Month: " << mth << endl;
    cout << "Year: " << yr << endl;

    return 0;
}
```

**आउटपुट:**
```
Day: 16
Month: 05
Year: 2021
```

## गहराई से जानें:

पार्सिंग एक तारीख को स्ट्रिंग से अलग करना एक प्राचीन कार्य है। पहले, लोगों द्वारा उपयोग किए जाने वाले कुचलते हुए और गैर-प्रमाणिक तारीख फॉर्मेट को उम्पन करने के लिए सर्वरल सॉर्टवर्क से तारीख को पार्स किया जाता था। हालांकि, आजकल अधिकांश भाषाओं में तारीख की प्रमाणिक डेटा प्रवाह को स्ट्रिंग से पार्सिंग करने के लिए उपयोग किया जाता है। प्रोग्रामर्स को स्ट्रिंग से तारीख को पार्स करने की ज़रूरत पड़ती है, ताकि वे अपने ऐप्लिकेशन में तारीख को सही रूप से प्रकट कर सकें।
