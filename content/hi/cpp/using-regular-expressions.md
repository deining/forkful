## क्यों

क्या आपने कभी सोचा है कि अपने कोड में दिए गए टेक्स्ट को सरल और आसान बनाने का तरीका क्या हो सकता है? रिग्युलर एक्सप्रेशन्स उन चमत्कारिक टूल्स में से एक है जो इस समस्या को हल कर सकता है। इस ब्लॉग पोस्ट में, हम इन रिग्युलर एक्सप्रेशन्स के बारे में विस्तार से बात करेंगे और आपको कोडिंग में उनका उपयोग करने के तरीके बताएंगे।

## कैसे

पहले, हमें रिग्युलर एक्सप्रेशन कैसे दिखते हैं, वह भी सामान्य भाशा में लिखी जाएंगी। सोचिए कि आपको एक वाक्य दिया गया है, "मैं खाना खाऊंगा।" अब यदि आपको इसमें से "खाना" शब्द निकालना है, तो क्या आप यह सोच सकते हैं कि आपने कैसे यह करेंगे? अब हम रिग्युलर एक्सप्रेशन्स के नियम के तहत इसे कैसे करेंगे। पहले, हम "/खाना/ जैसे टेक्स्ट लेंगे।" अब आपको इसमें "खाना" शब्द की जगह "/खां/ लिखना होगा।" अब यह पूरा वाक्य यूनिकोड के रिग्युलर एक्सप्रेशन में दिखेगा। हम इसे एक फंक्शन में लिखेंगे और इसको प्रिंट करेंगे। नीचे दिए गए कोड ब्लॉक को देखें:

```C++
#include <regex>
#include <iostream>
using namespace std;

void regexExample() {
    string sentence = "मैं खाना खाऊंगा।";
    regex regExpression("खां");
    smatch match;
    if (regex_search(sentence, match, regExpression)) {
        cout << match.str() << endl;
    }
}

int main() {
    regexExample();
    return 0;
}

// Output: खां
```

आप देख सकते हैं कि हमने सबसे पहले "sentence" नाम की चय
