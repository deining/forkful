## क्यों

क्या आपने कभी अपने प्रोग्राम में टेक्स्ट का सार्वजनिक खोज किया है और इसे स्वस्थ तरीके से व्यवस्थित करने के लिए कॉडिंग का समय खर्च किया है? इस समस्या का स्पष्ट विकल्प है - नियमित अभिव्यक्तियां (regular expressions)। नियमित अभिव्यक्तियां (regular expressions) को अपने प्रोग्राम में उपयोग करके, आप अपने दस्तावेजों, डेटा फ़ाइलों, और अन्य टेक्स्ट स्रोतों को त्वरित और आसानी से खोज सकते हैं।

## कैसे

नियमित अभिव्यक्तियां (regular expressions) का उपयोग करके टेक्स्ट के भीतर खोजने के लिए, आपको स्पष्ट अभिव्यक्ति पैटर्न के बारे में सोचना होगा। पैटर्न में, आप गुप्त मेटाचार या "wildcards" जोड़ सकते हैं जो आपको टेक्स्ट के भीतर कुछ भी खोजने में मदद करते हैं। नियमित अभिव्यक्तियां (regular expressions) में कुछ प्रमुख अभिव्यक्ति पत्तियां शामिल हैं, जैसे कि `[a-z]` (सभी लोहे) और `.` (कोई भी वर्ण)। स्पष्ट आकलन के साथ, आप इसे स्ट्रिंग क्लास के रूप में `c++` में बनाया पा सकते हैं। नीचे एक उदाहरण है कि कैसे आप टेक्स्ट में संख्याओं को ढूंढ सकते हैं:

```
#include <iostream>
#include <regex>
 
using namespace std;
 
int main() {
    string text = "आज 25 सितंबर है";
    regex pattern("[0-9]+");
    smatch matches;
    if (regex_search(text, matches, pattern)) {
        cout << "आपको यहां " << matches[0] << " संख्या मिली" << endl;
    }
    return 0;
}
```
