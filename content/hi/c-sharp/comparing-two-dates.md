# क्यों

दो तारीखों को तुलना करने के कारण जानने के लिए हमारे पास कई कारण हो सकते हैं। ये कारण प्रत्येक व्यक्ति के लिए अलग-अलग हो सकते हैं, लेकिन अधिकतर लोगों के लिए ये एक महत्वपूर्ण कौशल है। जैसे कि, अगर आप किसी बैंक या वित्तीय संस्था में काम करते हैं तो आपको तारीखों के बीच अंतर को कंट्रोल करना हो सकता है। 

# कैसे

दो तारीखों को तुलना करने के लिए, हम तीन मुख्य काम करेंगे: दो तारीखों को समान बनाएं, दो तारीखों को तुलना करें और फिर तुलना का परिणाम प्रिंट करें। हम इसे निम्नांकित तरीके से कोड कर सकते हैं। 

```C#
// समान तारीखों को बनाएं
DateTime firstDate = new DateTime(2020, 1, 1);
DateTime secondDate = new DateTime(2020, 1, 1);

// तारीखों को तुलना करें
int comparison = firstDate.CompareTo(secondDate);

if (comparison == 0)
{
    Console.WriteLine("दोनों तारीखों में अंतर नहीं है।");
}
else if (comparison < 0)
{
    Console.WriteLine("पहली तारीख दूसरी तारीख से पहले है।");
}
else
{
    Console.WriteLine("पहली तारीख दूसरी तारीख के बाद है।");
}

// तुलना का परिणाम प्रिंट करें
Console.WriteLine("तुलना का परिणाम: " + comparison);

```

परिणाम:

```
दोनों तारीखों में अंतर नहीं है।
तुलना का परिणाम: 0
```
