# आखिर क्यों?

जब हम प्रोग्रामिंग करते हैं, तो कभी-कभी हमें दो तारीखों को तुलना करने की जरूरत पड़ती है। यह कुछ वजहों से हो सकता है, जैसे कि समय के अनुकूल क्रियाकलापों को निर्धारित करना या हमारे प्रोग्राम में तारीखों के साथ कुछ काम करना। इसलिए, हम दो तारीखों को तुलना करने के तरीके पर ध्यान देना चाहते हैं।

# कैसे करें?

मैं आपको दो तारीखों को तुलना करने के लिए कुछ C# कोड दिखाऊंगा। आपको समझने में आसान बनाने के लिए, मैंने हर उदाहरण के साथ साथ मूल्यांकन किया है। आप यहां दी गई कोड ब्लॉक का सहारा ले सकते हैं और उन्हें अपने ऐप्लिकेशन में स्वयं प्रयोग कर सकते हैं।

```C#
// दो तारीखों को तुलना करने के लिए कोड हमेशा पहले और दूसरे तारीख को हमेशा से पहले होना चाहिए
DateTime firstDate = new DateTime(2021, 01, 01);
DateTime secondDate = new DateTime(2021, 01, 10);

// दो तारीखों को तुलना करने के लिए लेखिका का उपयोग करें
bool result = firstDate.CompareTo(secondDate);

if (result < 0)
{
    Console.WriteLine("पहली तारीख दूसरी तारीख से पहले है।");
}
else if (result > 0)
{
    Console.WriteLine("पहली तारीख दूसरी तारीख से बाद है।");
}
else
{
    Console.WriteLine("दोनों तारीखें बराबर हैं।");
}

// मूल्यांकन की उदाहरण देखें: पहली तारीख, दूसरी तारीख के दिन के मूल्यों से पहले है
// तो पहली तारीख दूसरी तारीख से पहले होने का मतलब है
```
