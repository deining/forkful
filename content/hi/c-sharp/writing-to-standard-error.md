आपने कभी सोचा है कि अपने प्रोग्राम को ऊर्जावर्धक तरीके से लिखने से क्या फ़ायदे हो सकते हैं? अगर हाँ, तो आप बिल्कुल सही स्थान पर हैं! इस ब्लॉग पोस्ट में, हम आपको स्टैंडर्ड त्रुटि पर लिखने के बारे में बताएंगे, जो एक कैजुअल उदार तरीका है अपने प्रोग्राम को लिखने का।

## क्यों
जब हम प्रोग्राम लिखते हैं, तो हमें कभी-कभी कोड में त्रुटियां मिल सकती हैं। ये त्रुटियां अक्सर अनुप्रयोग के स्थाई खामियों का पता लगाने में मदद करती हैं और हमारे प्रोग्राम को और बेहतर बनाने में योगदान देती हैं। इसलिए, हमें इन त्रुटियों को बचाने का प्रयास करना चाहिए और लिखने के दौरान हमें उन्हें सही से दिखाना भी चाहिए।

## कैसे
अब हम बात करते हैं कि हम कैसे स्टैंडर्ड त्रुटि पर लिखते हैं। स्टैंडर्ड त्रुटि को C# में *Console.Error* फ़ंक्शन हेतु उपयोग किया जाता है। निम्नलिखित उदाहरण में, हमने एक संदेश *"This is a standard error"* को स्टैंडर्ड त्रुटि पर लिखा है और उसका उत्पादन कांसोल में दिखाया है।

```C#
using System;
namespace StandardErrorExample
{
    class Program
    {
        static void Main(string[] args)
        {
         Console.Error.WriteLine("This is a standard error");  
        }
    }
}
```

उत्पादन:
```
This is a standard error
```
