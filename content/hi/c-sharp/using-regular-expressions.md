# क्यों

कभी-कभी हमें संदेशों को खोजने या विभिन्न डेटा को अभिव्यक्त करने के लिए विशिष्ट पैटर्न देखने की जरूरत होती है। इसके लिए हम डेटा के सटीक पैटर्न का पता लगाने के लिए कई पैटर्न मिलते हैं लेकिन ये सुलभ और समझने में आसान नहीं होते हैं। इसलिए, C# में नियमित अभिव्यक्तियों का उपयोग एक अच्छा विकल्प है।

# कैसे करें

```
C# string input = "मेरा नाम XYZ है। मैं भारत से हूं।";
Match match = Regex.Match(input, "ानाम [A-Z]{3} [^अ-ब-ह]{2}। मैं [^अ-ब-ह]{3}।");
if (match.Success)
{
    Console.WriteLine("प्रोफाइल मिला!");
}
```

इस उदाहरण में, हमने अभिव्यक्ति के द्वारा भारत से एक व्यक्ति की प्रोफ़ाइल खोजने के लिए एक नियमित अभिव्यक्ति का उपयोग किया है। हमने प्रॉफ़ाइल की जानकारी को मिलाने के लिए राइटन अर्डर और अक्षरों की संख्या पर ध्यान दिया है। नियमित अभिव्यक्तियों का उपयोग बहुत से ऐसे केस में भी किया जा सकता है जहां रसोई की व्यवस्ता, डाटा स्क्रबिंग और अन्य उदाहरणों में ये उपयोगी हो सकते हैं।
