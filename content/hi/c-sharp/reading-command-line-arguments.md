## क्यों

क्या आप जानते हैं कि सी# में कमांड लाइन आर्ग्यूमेंट्स पफॉर्मेंस तो बढ़ाते हैं ही, बल्कि आपको कोडिंग को भी सिर्फ एक ही स्टेटमेंट से समाप्त करने की अनुमति देते हैं? अधिक जानने के लिए यह ब्लॉग पोस्ट पढ़ें!

## कैसे करें

अपने सी# प्रोग्राम में कमांड लाइन आर्ग्यूमेंट्स पढ़ने के लिए, आपको `Main` फंक्शन की `string[] args` पैरामीटर का उपयोग करना होगा। यहां आपको एक उदाहरण दिया गया है:

```C#
using System;

class Program
{
    static void Main(string[] args)
    {
        Console.WriteLine($"Number of arguments: {args.Length}");
        foreach (string arg in args)
        {
            Console.WriteLine(arg);
        }
    }
}
```

यदि आप यह कोड कंपाइल और उन्हें निम्नलिखित आउटपुट के साथ चलाते हैं:

```
program.exe arg1 arg2 arg3
```

तो आपको निम्न आउटपुट मिलेगा:

```
Number of arguments: 3
arg1
arg2
arg3
```

आप देख सकते हैं कि `args` अर्ग्यूमेंट्स को `string[]` के रूप में काम करता है जो आप चलाने के समय दिए गए हैं। आप अपनी कोड में इन आर्ग्यूमेंट्स का उपयोग किसी भी तरह से कर सकते हैं।
