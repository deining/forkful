## क्यों

आज के समय में, कम्प्यूटर प्रोग्रामिंग बहुत जरूरी है और उसमें बेहतर और आसानी से चलने वाला तरीका है कमांड लाइन आर्ग्यूमेंट पढ़ना। कमांड लाइन आर्ग्यूमेंट्स को अपने कोड में जोड़ने से, आप अपने प्रोग्राम को बहुत से विशेषताओं के साथ बना सकते हैं जो आपके उपयोगकर्ताओं को लाभदायक हो सकते हैं।

## कैसे करें

चलिए अब तकनीकी बातों पर ध्यान देते हैं। आमतौर पर, क्या आपने अपने कोड में कभी प्रोग्राम के साथ कमांड लाइन आर्ग्यूमेंट्स को जोड़ने का प्रयास किया है? अगर नहीं, तो आप नीचे दिए गए उदाहरण को देख कर कमांड लाइन आर्ग्यूमेंट्स पढ़ना सीख सकते हैं।

```C#
using System;

public class Program
{
    public static void Main(string[] args)
    {
        Console.WriteLine("आपके द्वारा दर्ज किए गए अधिकार संख्या: {0}", args.Length);
		
        for (int i = 0; i < args.Length; i++)
        {
            Console.WriteLine("{0}. {1}", i + 1, args[i]);
        }
    }
}
```

ऊपर दिए गए कोड में, हमने `Main` मेथड में `string[] args` को एक प्राथमिक संदर्भ के रूप में दिया है। इससे हम उपयोगकर्ता द्वारा दर्ज की गई अनेक आर्ग्यूमेंट्स को पढ़ सकते हैं और उन्हें प्रोग्राम में उपयोग कर सकते हैं। ऊपर दिए गए कोड को रन करने के लिए, आपको आपके कम्प्यूटर पर `सीएलआर` कमांड लाइन में `कंडोलक्स्` खोलना होगा और उस डायरेक्टरी में जाना होगा जहां आपने अपना स
