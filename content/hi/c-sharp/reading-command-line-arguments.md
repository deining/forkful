## क्यों

क्या आपको पता है कि कमांड लाइन एर्ग्युमेंट्स काफी उपयोगी हो सकते हैं? यह प्रोग्रामिंग दुनिया में बहुत ही महत्वपूर्ण है क्योंकि यह आपको अपनी एप्लिकेशन को डायनामिक बनाने की अनुमति देता है। इसके माध्यम से, आप अपनी एप्लिकेशन को आसानी से परिवर्तित कर सकते हैं और इसे अपने उपभोक्ताओं के लिए और उपयोगकर्ता अनुरोधों के अनुसार अनुकूलित कर सकते हैं।

## कैसे करें

आपने कभी अपने एप्लिकेशन को कमांड लाइन एर्ग्युमेंट्स के साथ चलाया है? इसको समझना और लिखने के लिए काफी आसान हो सकता है। स्वयं को मजोदार करने के लिए, हम एक उदाहरण सही जीवन में दिखाएंगे। हम एक तकनीक द्वारा अपनी C# एप्लिकेशन में कमांड लाइन एर्ग्युमेंट कैसे पढ़ सकते हैं।

```C#
using System;

class Program
{
    static void Main(string[] args)
    {
        // कमांड लाइन एर्ग्युमेंट्स पढ़ें
        // args एक string array है जो कॉमांड लाइन एर्ग्युमेंट्स को होल्ड करता है
        // आप एक foreach लूप का इस्तेमाल करके इन्हें पढ़ सकते हैं

        foreach (string arg in args)
        {
            Console.WriteLine(arg);
        }
    }
}
```

उपरोक्त कोड को कंपाइल करते हुए, आप निम्नलिखित उपभोक्ताओं को इनपुट कर सकते हैं और आउटपुट को देख सकते हैं:

```
$ dotnet run one two three

one
two
three
```
