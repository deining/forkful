## क्यों

कई ऑनलाइन गेम खेलने और सोशल मीडिया खातों पर पोस्ट साझा करने जैसे कामों में, हमें अक्सर दो या अधिक टेक्स्ट स्ट्रिंग्स को एक साथ जोड़ने की जरूरत होती है। इससे हमें दोनों स्ट्रिंग का एक ही स्ट्रिंग में एकत्र करने की आवश्यकता होती है, जो हमें प्रोग्राम के अन्य हिस्सों में इस्तेमाल करने में आसानी प्रदान करता है।

## कैसे करें

हम C# में स्ट्रिंग्स को जोड़ने के लिए "+" और String.Concat() फ़ंक्शन का इस्तेमाल कर सकते हैं। यहां, हम एक सरल उदाहरण देखेंगे जिसमें हम दो स्ट्रिंग्स को जोड़ के नया स्ट्रिंग बनाएंगे:

```C#
string firstName = "जॉन";
string lastName = "डो";
string fullName = firstName + " " + lastName;
Console.WriteLine(fullName);
```
आउटपुट:
जॉन डो
 
स्ट्रिंग्स को जोड़ने के लिए फंक्शन का उपयोग करते हुए उदाहरण:

```C#
string firstName = "जॉन";
string lastName = "डो";
string fullName = String.Concat(firstName, " ", lastName);
Console.WriteLine(fullName);
```
आउटपुट:
जॉन डो
