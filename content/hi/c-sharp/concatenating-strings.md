# क्यों

कई बार हमें अपने कोड में विभिन्न स्ट्रिंग को जोड़ने की आवश्यकता होती है, जैसे उपयोगकर्ता से इनपुट लेना या प्रिंट स्टेटमेंट बनाना। इसके लिए हमारे पास स्ट्रिंग को जोड़ने का एक आसान तरीका है जो हमें अधिक प्रोग्रामिंग प्रक्रियाओं को ठीक से करने में मदद करता है।

## कैसे करें

```C#
string str1 = "Hello";
string str2 = "World";
string str3 = str1 + ", " + str2;
Console.WriteLine(str3);

// Output: Hello, World
```

स्ट्रिंग को जोड़ने के लिए हम सिर्फ दो स्ट्रिंग को "+" ऑपरेटर से जोड़ते हैं। आप एक से अधिक स्ट्रिंग जोड़ने के लिए इसी तरह का उपयोग कर सकते हैं। इसके अलावा आपको विभिन्न बिल्ट-इन फंक्शन जैसे "string.Join()" भी उपलब्ध है जो दो स्ट्रिंग को जोड़कर एक नया स्ट्रिंग बनाता है।

## गहराई में

स्ट्रिंग को जोड़ने का एक महत्वपूर्ण पहलू यह है कि यह इसका कार्य अनुमानित समय में करता है। अपेक्षित समय से स्ट्रिंग को जोड़ने के लिए कई अलग-अलग तरीके हैं, लेकिन एक असाधारण समस्या यह है कि सबसे अधिक समय जोड़ों का गणन और ज्वाइनिंग होता है। स्ट्रिंग को जोड़ने के लिए "+" ऑपरेटर का उपयोग एक उत्तम विकल्प है, क्योंकि यह मुख्यतः क्रमित पठन और ज्वाइनिंग को समान समय में करता है।
