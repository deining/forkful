## क्यों

स्रिंग्स को जोड़ना मूलतः यह एक प्रोग्रामिंग दक्षता है जो हमें विभिन्न डेटा प्रकारों को एक साथ अनुकूलित करने में मदद करती है। इससे हम अपने कोड को सुव्यवस्थित और अधिक पढ़ने योग्य बना सकते हैं।

## कैसे करें

```C#
string firstName = "जॉन";
string lastName = "डो";
string fullName = firstName + lastName;
Console.WriteLine(fullName);
```

आउटपुट:
जॉनडो

ऊपर दिए गए कोड द्वारा हमने दो विभिन्न स्रिंग वेरिएबल्स `firstName` और `lastName` को `+` ऑपरेटर की मदद से जोड़ा और उन्हें मिलाकर `fullName` नाम की एक नई स्ट्रिंग बनाई। उसके बाद `Console.WriteLine()` का उपयोग करके हमने `fullName` को आउटपुट किया। इस तरह से हम स्रिंग्स को जोड़कर उन्हें एक साथ दिखा सकते हैं।

## गहराई में विज्ञान

स्रिंग्स को जोड़ने के लिए C# में `+` ऑपरेटर के अलावा दो और तरीके हैं - `string.Concat()` और `string.Format()`। इनमें से `string.Concat()` को आप एक से अधिक स्ट्रिंग्स को जोड़ने के लिए इस्तेमाल कर सकते हैं। यह ऑपरेटर स्ट्रिंग के जोड़ने के बजाय उन्हें मिलाता है। दूसरे तरीके के आधार पर, `string.Format()` स्ट्रिंग को एक फॉर्मेटिंग नाम पद्धति में बदलता है। आप अपने डेटा को ऑप्टिमल तरीके से दिखाने के लिए इस्तेमाल कर सकते हैं। आप चाहे तो एक से अधिक स्ट्रिंग्स को भी दोनों तरीकों से मिला सकते हैं।
