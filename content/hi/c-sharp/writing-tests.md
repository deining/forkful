## क्यों

"टेस्ट कैसे लिखते हैं" यह आम सवाल है जो कई और मेरे जैसे डेवलपर्स को अक्सर परेशान करता है। जब हम अपने कोड को भविष्य में बदलना चाहते हैं, तो टेस्ट केस से हमारे प्रोग्राम की मापदंड और संरचना को सुनिश्चित करने में मदद मिलती है। इसलिए, टेस्ट लिखना बेहद महत्वपूर्ण है और इस लेख में हम यह जानेंगे कि आप अपने C# कोड के लिए टेस्ट कैसे लिख सकते हैं।

## कैसे करें

मैं आपको सबसे पहले एक उदाहरण द्वारा टेस्ट केस का उपयोग करने के तरीके को बताऊंगा, जिसमें हम एक सामान्य फ़ंक्शन की टेस्टिंग करेंगे जो दो संख्याओं को जोड़ता है। इसके लिए हम निम्नलिखित कोड ब्लॉक का उपयोग करेंगे:

```C#
// फ़ंक्शन बनाना
public int Add(int a, int b) 
{
  return a + b;
}

// टेस्ट केस लिखना
[TestMethod]
public void TestAdd()
{
  int result = Add(2, 3);
  
  // अपेक्षित परिणाम की जांच करें
  Assert.AreEqual(5, result);
}
```

यहां हमने एक सामान्य फ़ंक्शन बनाया और उसे टेस्ट केस के साथ जोड़ने के लिए Assert.AreEqual() का उपयोग किया है। जब आप इस टेस्ट केस को चलाएंगे तो आपको परेशानी से बचने के लिए यह हमेशा सुनिश्चित होगा कि आपकी फ़ंक्शन सही रूप से काम कर रही है।
