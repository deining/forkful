## वजह

टेक्स्ट को खोजने और बदलने में शामिल होने की वजह सिर्फ 1-2 हिंसे है। अधिकांश प्रोग्रामर्स कोडिंग के दौरान टेक्स्ट को खोजने और उसे बदलने की जरूरत पड़ सकती है, जो अपने आप में बहुत ही उपयोगी है।

## कैसे करें

टेक्स्ट को खोजने और उसे बदलने का एक सरल तरीका है `Regex.Replace()` फंक्शन का उपयोग करना। इस फंक्शन द्वारा हम दो पैरामीटर्स पास करते हैं: स्ट्रिंग और एक डेलिगेट जो हमारे स्ट्रिंग को बदलता है। हमारे पास एक स्ट्रिंग परिवर्तित हो जाएगा और उसे एक नया स्ट्रिंग से आवृत्ति कर दिया जाएगा। नीचे दिए गए उदाहरण में हम "Hello world" स्ट्रिंग को "Hola world" में बदल रहे हैं:

```c#
string originalText = "Hello world";
string newText = Regex.Replace(originalText, "Hello", "Hola");
Console.WriteLine(newText);
```

आउटपुट:
```c#
Hola world
```

इसके अलावा, `Regex.Replace()` फंक्शन के पास पैरामीटर के रूप में दो स्ट्रिंग पास किए जा सकते हैं जो हमारे स्ट्रिंग के भीतर से खोजने और उसे बदलने के लिए मदद करते हैं। नीचे दिए गए उदाहरण में, हम "123456" स्ट्रिंग के सभी अंकों को "X" में बदल रहे हैं:

```c#
string number = "123456";
string pattern = "[0-9]";
string replaceWith = "X";
string regexResult = Regex.Replace(number, pattern, replaceWith);
Console.WriteLine(regexResult);
```
आउटपुट:
```c#
XXXXXX
```
