# क्यों

टेक्स्ट को खोजना और बदलना कोई आसान कार्य नहीं है, लेकिन कभी-कभी हमें अपने कोड में बहुत सारी टेक्स्ट लाइन को बदलने की आवश्यकता होती है। इससे बहुत समय और मेहनत बचाया जा सकता है।

# कैसे करें

टेक्स्ट को खोजने और बदलने के लिए, हम C# में `Replace()` फंक्शन का प्रयोग कर सकते हैं। इस फंक्शन को `string` पैरामीटर के साथ कॉल करना होता है जिसमें हम टेक्स्ट को वहां बदलना चाहते हैं। यहां एक उदाहरण है:

```C#
string originalText = "मेरा नाम जॉन है।";
string replacedText = originalText.Replace("जॉन", "रॉबर्ट");
Console.WriteLine(replacedText);
```

इसका आउटपुट होगा:

```
मेरा नाम रॉबर्ट है।
```

हम इस तरह से किसी भी टेक्स्ट को आसानी से बदल सकते हैं।

# गहराई में जाएं

C# में `Replace()` फंक्शन के अलावा भी कई और तरीके हैं जिनसे हम टेक्स्ट को खोजकर बदल सकते हैं। कुछ और उदाहरण:

- `Replace()` फंक्शन का दूसरा पैरामीटर देखें और चाहें तो `string` के प्लेसहोल्डर (`{0}` या `{1}` आदि) की जगह पर इस्तेमाल कर सकते हैं।
- टेक्स्ट के आगे या पीछे और भी टेक्स्ट को स्पेसिफाइ कर सकते हैं।
- `Regex.Replace()` फंक्शन का प्रयोग करके रेगुलर एक्सप्रेशन (Regular Expression) के माध्यम से भी टेक्स्ट को बदला जा सकता है।
