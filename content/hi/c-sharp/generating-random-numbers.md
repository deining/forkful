## क्यों

क्या आपने कभी सोचा है कि डेवलपर्स क्यों यादृच्छिक नंबर उत्पन्न करने में इंगेज होते हैं? असल में, यह नंबर सेक्युरिटी सिस्टम, क्रिप्टोग्राफी, गेम्स, और मशीन लर्निंग जैसे कोई भी कॉम्प्यूटिंग सीन के लिए अत्यावश्यक है। इस ब्लॉग पोस्ट में, हम आपको यादृच्छिक नंबर उत्पन्न करने के तरीके के बारे में बताएंगे जो कि सीशार्प (C#) में काफी सरल है। 

## कैसे करें

यादृच्छिक (random) नंबर्स उत्पन्न करने के लिए, हम सीशार्प के `Random` लाइब्रेरी का इस्तेमाल करते हैं। यह लाइब्रेरी इनपुट के आधार पर एक यादृच्छिक नंबर उत्पन्न करती है। निचे दिए गए उदाहरण में, हमने `Random.Next()` मेथड का उपयोग किया है जो कि एक पूर्णांक उत्पन्न करता है जो 0 से 9 के बीच का होता है। 

```C#
using System;

public class Program
{
  public static void Main()
  {
    // Random number between 0 and 9
    Random random = new Random();
    int randomNumber = random.Next(10);

    Console.WriteLine("Random number: " + randomNumber);
  }
}

// Output: Random Number: 5
```

यदि हम 10 लिखते हैं `random.Next(10)` के बजाय, तो हम 0 से 9 के बीच का यादृच्छिक नंबर उत्पन्न करेंगे। इसी तरह, हम 100 लिखकर `random.Next(100)` का उपयोग करेंगे तो हम 0 से 99 के बीच एक यादृच्छिक नंबर उत्पन्न करेंगे।
