## और क्या हैं और क्यों?
अकसर हमें ऐसे प्रोग्राम बनाने होते हैं जिसमें हमें याद होते हैं कि अगले बार जब हम उस प्रोग्राम को चलाएंगे तो उसमें वह सभी चीजें अलग हों। ऐसे प्रोग्राम बनाने के लिए हम रैंडम नंबर्स उपयोग करते हैं। रैंडम नंबर्स का अर्थ होता है हम उपयोगकर्ताओं के अनुरोध पर हमें विभिन्न संख्याओं का एक अनुचित तरीके से आयात इकाई जेनरेट करते हुए संख्याएं उपयोगकर्ताओं को देते हैं।

## कैसे करें:
```C#
int num1 = new Random().Next(); // एक दूसरे और दूसरे नंबर 
double num2 = new Random().NextDouble(); //  एक दूसरे और एक दूसरे संख्याएं देता है जो अक्सर 0.0 से 1.0 के बीच होता है।
int num3 = new Random().Next(1, 10); // पहले दूसरे संख्याओं में किसी भी एक संख्या देता है 
double num4 = new Random().NextDouble(); // संख्याएं भिन्नरूप में देता है।
```

## गहराई में जाएँ:
हम हमेशा से बिचौलिया प्रोग्रामिंग भाषाओं को उपयोग करते आए हैं, जो हमें उपयोगकर्ताओं को विभिन्न संख्याओं का आपूर्ति करने में मदद करते हैं। रैंडम नंबर्स के अलावा, हम इस विषय में डिग्री को उपयोग कर सकते हैं, जो एक प्रकार की है जो हमारी संख्याओं के साथ मौजूद ताल में पड़ती है। रैंडम नंबर्स को छुपाने के लिए भी आप प्रोग्रामिंग कोड को बदल सकते हैं।
