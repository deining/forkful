## क्यों

नई परियोजना शुरू करने के पीछे का कारण क्या है? कोई इसलिए शर्त नहीं की जब वे शेष रहते हैं, या जो वे अपने विकास को बढ़ावा देना चाहते हैं, उससे उन्हें इससे साफ हो सकता है कि आप कंप्यूटर उपयोग के बारे में अधिक जानकारी जुटाएं. तस्वीर में निर्देश के लिए आप "कैसे" अनुभाग में जा सकते हैं.

## कैसे

आप एक नई C# परियोजना कैसे शुरू करें, इसके बारे में जानने के लिए आपको नमस्ते करने से पहले कुछ कोडों को बीचने की जरूरत होगी. कोड देखना इससे भी ज्यादा बना है, तो आप ठीक हैं, तो कृपया नोट करें त्योहार कळ सकते हैं. पहली बात से व्याख्यक, आप नए C# परियोजना कैसे शुरू करते हैं, क्योंकि शुरू करने के लिए शर्त है कि आप इसे अपने स्थापना के बार में सो रहे हो.

यह कोड देखकर आप अधिक सीख सकते हैं:

```C#
using System;

namespace NewProject
{
  class Program
  {
    static void Main(string[] args)
    {
      Console.WriteLine("नमस्ते, यहा से अपने C# परियोजना शुरू करें!");
    }
  }
}
```

इसका परिणाम है:

```
नमस्ते, यहा से अपने C# परियोजना शुरू करें!
```
