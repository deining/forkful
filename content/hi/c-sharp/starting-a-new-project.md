## क्या और क्यों?

प्रोग्रामर्स अपनी नई प्रोजेक्ट्स को शुरू करने के लिए किसी दिया गया काम या कार्य के साथ नए संबंधित संसाधित तरीके को अपनाते हैं। यह उनके वर्तमान कार्य या प्रोजेक्ट्स से अलग होता है और उन्हें नए चुनौतियों और नये दिशाओं को देखने और समझने का मौका देता है।

## कैसे करें?

आप अपनी प्रोजेक्ट्स को शुरू करने के लिए निम्न उदाहरण में दिए गए `C# ...` कोड ब्लॉक्स की मदद से कोडिंग का प्रयास कर सकते हैं:

```
// कोन्सोल एप्लीकेशन बनाएं
using System;

namespace प्रोजेक्ट_नाम
{
    class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("नमस्ते, दुनिया!");
        }
    }
}
```

यह कार्य सदियों से चला आ रहा है और डॉनेल्ड कन्निंघम और ब्रायन कर्निहैन ने १९६९ से १९७१ के बीच बर्तामान प्रोग्रामिंग की शुरुआत के समय बनाया गया था। दूसरे संबंधित स्त्रोतों में शुरुआत के प्रतिबिम्ब के रूप में स्क्रीनशॉट, कोड दस्तावेज़ीकरण और और ग़ल्तियों की सूचना शामिल है।

## गहराई में जाएं

प्रोजेक्ट की शुरुआत के विभिन्न दृष्टिकोण हैं, जैसे कि सामान्य प्रोग्रामिंग और एप्लीकेशन विकास के साथ स्इंगुलरिटी और DevOps दृष्टिकोण। Any 이 와이ज़ क्या आप अपनी नई प्रोजेक्ट को शुरुआत करने के लिए सरलता से किसी भी सुविधा का उपयोग कर सकते हैं, जैसे- ASP.NET या .NET Core के रूप में। आप अपने प्रोजेक्ट को एक अनुकरणीय प्रोग्राम भाषा में, जैसे कि C++ या जावा में भी आरंभ कर सकते हैं।
