# क्यों

एक स्ट्रिंग की लंबाई को जानने से पहले हमें यह पता होना चाहिए कि क्यों हम ऐसा करना चाहते हैं। एक स्ट्रिंग उपयोगकर्ता से स्ट्रिंग के विभिन्न धागों के साथ इंटरैक्ट करने और उसके अनुपात को समझने के लिए उपयोगी होती है। उदाहरण के लिए, यदि हमारे पास एक उपयोगकर्ता के नाम की एक स्ट्रिंग है, तो हम बाद में अपने कोड में उसके स्वागत का स्ट्रिंग जोड़ सकते हैं। इसलिए, स्ट्रिंग की लंबाई जानना एक महत्वपूर्ण प्रक्रिया है जो हमें स्ट्रिंग प्रोग्रामिंग में अधिक समझदार बनाती है।

# कैसे करें

एक स्ट्रिंग की लंबाई निकालने के लिए, हम C # का उपयोग कर सकते हैं। सबसे पहले, हम एक स्ट्रिंग वैरिएबल बनाएंगे और उसमे जानकारी दर्ज करेंगे। उदाहरण के लिए, ```C# string name = "John Doe";``` अब हम ``name`` वैरिएबल का उपयोग करके उसकी लंबाई को निकालेंगे। इसके लिए, हम ```C# name.Length``` का उपयोग करेंगे। इसके बाद, हम ``` Console.WriteLine(name.Length); ``` लेखक ब्लॉक का उपयोग करके उत्पाद को दर्शाएंगे। अब जब हम अपने कोड को कंपाइल और रन करेंगे, तो हमें स्ट्रिंग की लंबाई मिलेगी, जो हमें "8" दिखाएगी।
