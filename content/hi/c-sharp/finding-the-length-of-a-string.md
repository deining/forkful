## क्यों 

अक्सर हमारे पास स्ट्रिंग्स (पाठ) का लेंग्थ (लंबाई/आकार) पता करने की ज़रूरत पड़ती है। इसका उदाहरण, स्ट्रिंग्स को दिखाने के लिए एक लाइन में स्पेसिंग का सही तरीका ढूंढना या पाठ में कुछ विशेष शब्दों का समानांतर ढूंढना हो सकता है। इसलिए आज हम यह सीखेंगे कि कैसे हम C# में स्ट्रिंग की लंबाई को प्राप्त कर सकते हैं।

## कैसे करें

अपने लेखक कूदो को नॉर्मल, अंकुल, कैपिटल, बेरनिंग आदि मदों से मायनेित्म् करने के लिए कल मेमो से इस्तमाल करें।

```C#
// उदाहरण १: स्वाभाविक अर्थ
string exampleStr = "मेरा प्रिय भारत!";
Console.WriteLine(exampleStr.Length);
// output: 16 (स्पेस को शामिल नहीं)

// उदाहरण २: मद के साथ स्पेस
string exampleStr2 = "हमारे प्रिय देश में यही होना चाहिए!";
Console.WriteLine(exampleStr2.Length);
// output: 31 (स्पेस भी शामिल हैं)
```

## गहराईदेखो

स्ट्रिंग्स की लंबाई प्राप्त करना बहुत ही सरल है। यह लंबाई को स्पेस और पाठ के मदों का पता लगाने के लिए उपयोगी है। इस प्रकार से, हम अपने कोड को बेहतर और सुव्यवस्थित बना सकते हैं। इसके अलावा, हम अन्य प्रोग्रामिंग भाषाओं जैसे Java या Python में भी स्ट्रिंग की लंबाई को लेकर समान फंक्शन का उपयोग कर सकते हैं।
