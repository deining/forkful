# क्यों

क्या आपने कभी सोचा है कि आप कंप्यूटर पर कुछ सामग्री प्राप्त करना चाहते हैं और उसे बाद में दूसरे लोगों को भेज सकें? या शायद आपने एक प्रोग्राम को दोहराया जिसे बाद में आपको फिर से प्रयोग करना हो? बेसिकली, आपको दस्तावेज़ लिखने की जरूरत होगी। ऐसी स्थिति में, आपको पाठ फ़ाइल लिखने की जरूरत पड़ती है। 

# कैसे करें

यदि आपको कंप्यूटर पर नई पाठ फ़ाइल लिखनी है, तो आपको उसे सरलता से कर सकते हैं। पहले, आपको Visual Studio या किसी अन्य एडिटर में अपना प्रोग्राम लिखना होगा। फिर, `File` मेनू में जाएं और `New` चुनें, फिर से `File` और अगली पाठ फ़ाइल सेटिंग का चयन करें। तत्पश्चात, संग्रहण में `Save As` चुनें, फ़ाइल का नाम दें और `.txt` कोई अंतिम नाम में जोड़ें। अब, आपकी टेक्स्ट फ़ाइल बन गई है! हम आगे जाकर कुछ कोडिंग करेंगे जो आपके लिखा हुआ पाठ संग्रहण में आपके फ़ाइल में जोड़ देगा।
