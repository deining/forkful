## क्या और क्यों?
किसी भी प्रोग्रामर के लिए टेक्स्ट फ़ाइल पढ़ना एक बहुत ही महत्वपूर्ण कौशल है। यह एक आसान तरीका है जिससे हम डेटा को संग्रहीत कर सकते हैं और अपने प्रोग्राम में उसे उपयोग कर सकते हैं। 

## कैसे करें?
टेक्स्ट फ़ाइल को पढ़ने के लिए हमें सिर्फ दो चीजें करनी होती हैं - फ़ाइल का पथ निर्दिष्ट करें और पढ़ने के लिए कोड बनाएं। यहां हम आपको कुछ उदाहरण देंगे जो प्रोग्रामिंग भाषा C# में टेक्स्ट फ़ाइल पढ़ने के लिए हों सकते हैं।

```C#
// फ़ाइल का पथ निर्दिष्ट करना
string filePath = @"C:\Users\UserName\Documents\sample.txt";

// फ़ाइल पढ़ने का कोड
string content = File.ReadAllText(filePath);
Console.WriteLine(content);

// पंक्तियों को अलग-अलग लाइनों पर प्रिंट करने के लिए
string[] lines = File.ReadAllLines(filePath);
foreach (string line in lines)
{
    Console.WriteLine(line);
}
```

अगर आप केवल एक लाइन को पढ़ना चाहते हैं, तो आप निम्नलिखित कोड का उपयोग कर सकते हैं:

```C#
// फ़ाइल पढ़ने के लिए कोड
string line = File.Readline(filePath);
Console.WriteLine(line);
```

कोड के अंतिम उदाहरण में, हम फ़ाइल से डेटा को संग्रहीत भी कर सकते हैं:

```C#
// फ़ाइल से डेटा संग्रहीत करने के लिए कोड
string[] lines = File.ReadAllLines(filePath);

// फ़ाइल में पूरे डेटा की संख्या को प्रिंट करने के लिए
Console.WriteLine("कुल पंक्तियाँ: " + lines.Length.ToString());

// अंतिम पंक्ति प्रिंट करने के लिए
Console.WriteLine("अंतिम पंक्ति: " + lines[lines.Length - 1]);
```

## गहराई में जाएं
टेक्स्ट फ़ाइल पढ़ने का इतिहास बहुत पुराना है। पहले से ही लोग फ़ाइलों से डेटा को पढ़ने के लिए इस्तेमाल होती आ रही हैं। आज भी बहुत सारे प्रोग्रामिंग भाषाओं में टेक्स्ट फ़ाइल पढ़ने का समर्थन होता है। 

लेकिन आज भी कुछ भाषाओं में प्रोग्रामर्स फ़ाइल पढ़ने के लिए दूसरे तरीके भी इस्तेमाल करते हैं। इसमें से कुछ हैं - सीएसवी (CSV) फ़ाइल, डेटाबेस, वेब साइट से डेटा डाउनलोड करना आदि। 

फ़ाइल पढ़ने का कोड सिर्फ टेक्स्ट फ़ाइलों के लिए होता है। अगर आप अन्य प्रकार की फ़ाइलों को पढ़ना चाहते हैं, तो आपको अलग-अलग कोड का उपयोग करना होगा।
