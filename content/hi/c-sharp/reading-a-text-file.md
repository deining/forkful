## क्यों
एक टेक्स्ट फ़ाइल पढ़ना बहुत उपयोगी हो सकता है जब आपको समूह में ढीले-धारे डेटा को संगठित रूप से देखना हो या अधिक विस्तृत जानकारी को सजावट करने की आवश्यकता हो।

## कैसे करें
```C#
// टेक्स्ट फ़ाइल से सभी पंक्तियों को पढ़ें
string[] lines = File.ReadAllLines("file.txt");

// प्रत्येक पंक्ति को एक-एक करके देखें और काम करें
foreach (string line in lines)
{
    // पंक्ति में से शब्दों को अलग करें
    string[] words = line.Split(' ');

    // शब्दों को एक के बाद एक प्रिंट करें
    foreach (string word in words)
    {
        Console.WriteLine(word);
    }
}
```
इस उदाहरण में, हमने एक टेक्स्ट फ़ाइल से दो पंक्तियों को पढ़ा और प्रत्येक पंक्ति को शब्दों में अलग किया है। यह सुविधा शब्दों को अलग करने के लिए उपयोगी हो सकती है, जैसे कि एक दस्तावेज़ में शब्द समूहों को आसानी से बनाने के लिए। मुद्रित परिणाम को उपयोगकर्ता के द्वारा निकाले गए सेवा इवेंट से अलग किया जा सकता है।
