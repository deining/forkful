## क्यों 
किसी भी प्रोग्रामिंग परियोजना में सफलता पाने के लिए, एक डेवलपर को अक्सर टेक्स्ट फ़ाइल से डेटा को पढ़ने की आवश्यकता होती है। यह फ़ाइलों को काम करने का प्रमुख तरीका है और बहुत सारे प्रोग्रामिंग भाषाओं में उपलब्ध है। इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि C# में टेक्स्ट फ़ाइल को कैसे पढ़ा जा सकता है और इससे कैसे डेटा को अपनी आवश्यकताओं के अनुसार प्रसंस्करण किया जा सकता है। 

## कैसे करें 
पहले हमें संबंधित नेमस्पेस को इम्पोर्ट करने की आवश्यकता होती है। यह प्रोग्राम को टेक्स्ट फ़ाइल को पढ़ने और लिखने करने की सुविधा प्रदान करता है। उदाहरण के लिए, हम `System.IO` नेमस्पेस को इम्पोर्ट कर सकते हैं। अब हम `File.ReadAllLines()` मेथड का उपयोग करके अपनी फ़ाइल को पढ़ सकते हैं। यह मेथड फ़ाइल के सभी लाइनों को रिटर्न करता है और हम उन्हें अलग-अलग चरणों में डेटा संग्रहीत करके उनसे आगे काम कर सकते हैं। 

```
C#
using System.IO;

string[] lines = File.ReadAllLines("MyFile.txt");

foreach(string line in lines)
{
    // यहां आप दूसरे मापदंडों के साथ अपने डेटा को प्रसंस्करण कर सकते हैं
    Console.WriteLine(line);
}
```
