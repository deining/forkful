## क्या और क्यों?

जब हम एक डायरेक्टरी के अस्तित्व को देखते हैं, तो हम जांचते हैं कि वह कंप्यूटर में मौजूद है या नहीं। प्रोग्रामर इस जांच को करते हैं ताकि वे सुनिश्चित कर सकें कि डायरेक्टरी से जुड़े कोई भी एक्शन कोड को सही से चलाया जा सके।

## कैसे करें:

इस आर्टिकल में हम आपको एल्म में डायरेक्टरी का अस्तित्व कैसे जांचें उसका एक अभ्यास दिखाएंगे।

```Elm
import Basics exposing (..)
import File

main =
  case File.exists "data" of
    Ok _ ->
      text "यह डायरेक्टरी मौजूद है!"
    
    Err _ ->
      text "यह डायरेक्टरी मौजूद नहीं है!"
```

आउटपुट:
```
यह डायरेक्टरी मौजूद है!
```

```Elm
import Basics exposing (..)
import File

main =
  case File.exists "documents" of
    Ok _ ->
      text "यह डायरेक्टरी मौजूद है!"
    
    Err _ ->
      text "यह डायरेक्टरी मौजूद नहीं है!"
```

आउटपुट:
```
यह डायरेक्टरी मौजूद नहीं है!
```

## गहराई में जाएं:

डायरेक्टरी की जांच एक बहुत ही आम कार्य है। इसे हम प्रोग्राम में अपनी फिल्में, फोटोग्राफ, दस्तावेज़े आदि संग्रहित करने के लिए स्थान खोजने के लिए भी करते हैं। एल्म के अलावा इस काम को करने के लिए अन्य प्रोग्रामिंग भाषाएं भी हैं, जैसे JavaScript या Python। एल्म भाषा में डायरेक्टरी का अस्तित्व जांचने के लिए,

शुरू में आपको Basics और File मॉड्यूल्स को इम्पोर्ट करना होगा। फिर आपको File.exists फ़ंक्शन को उस डायरेक्टरी के साथ पहुँचना होगा जिसका आप का अस्तित्व जांच करना चाहते हैं। इस फ़ंक्शन का आउटपुट Ok या Err मौजूद हो सकता है। हमने यहां एक सॉलिड उदाहरण दिया है जो आपको डायरेक्टरी का अस्तित्व दिखा सकता है।

## अन्य स्रोत देखें:

अगर आपने अभी तक अपने पास एल्म में डायरेक्टरी का अस्तित्व जांचने का कोई अन्य तरीका देखा नहीं है, तो आप [ऑफिशियल डॉक्यूमेंटेशन](https://elm-lang.org/docs) देख सकते हैं। वहां आपको भिन्न प्रश्नों के जवाब मिल सकते हैं और अपनी जानकारी को बढ़ा सकते हैं। आप [Elm Weekly Newsletter](https://www.elmweekly.nl) के माध्यम से अधिक जानकारी प्राप्त कर सकते हैं जो
