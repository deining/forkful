## क्यों (Why)

क्या आपको कभी एल्म प्रोग्रामिंग में स्ट्रिंग को कैपटलाइज करने की जरूरत पड़ी है? स्ट्रिंग कैपटलाइजिंग एक उपयोगी काम है जो आपको एल्म में स्ट्रिंग को अपने आवश्यकतानुसार प्रभावी ढंग से संपादित करने में मदद कर सकता है।

## कैसे करें (How To)

यदि आपको एल्म में स्ट्रिंग को कैपटलाइज करना है, तो निम्नलिखित कोड उदाहरण देखें:

```elm
-- स्ट्रिंग को कैपटलाइज करने के लिए कोड फंक्शन
capitalizeString : String -> String
capitalizeString str =
  String.toUpper (String.left 1 str) ++ String.dropLeft 1 str 

-- स्ट्रिंग को कैपटलाइज करने के लिए कोड का उपयोग करें
capitalizedStr : String 
capitalizedStr = capitalizeString "welcome to elm programming!" 
-- छठेर ! आउटपुट -> "Welcome to elm programming!"
```

ऊपर दिए गए कोड में, हमने स्ट्रिंग को कैपटलाइज करने के लिए एक फंक्शन बनाई है जिसमें हमने `String` मॉड्यूल की दो फंक्शनों का उपयोग किया है। पहले हमने `String.left` फंक्शन का उपयोग करके स्ट्रिंग का पहला अक्षर लिया और उसके बाद हमने `String.toUpper` फंक्शन से उस अक्षर को कैपटलिज किया है। फिर हमने `String.dropLeft` फंक्शन का उपयोग करके स्ट्रिंग के पहले अक्षर को काट दिया है। आखिर में हमने दो स्ट्रिंग को जोड़ कर संपूर्ण स्ट्रिंग को कैपटलाइज कर दिया है।
