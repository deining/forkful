# क्यों

शायद आपने एल्म प्रोग्रामिंग के दौरान कभी विचार किया हो कि अपने स्ट्रिंग के मध्य से एक छोटा सा टुकड़ा निकालने का कोई तरीका हो सकता है। आपको शायद अपनी एप्लिकेशन के लिए बहुत से छोटे टेक्स्ट फ़ील्ड बनाने हों, जहां उपयोगकर्ता अपना नाम, ईमेल आईडी या कोई अन्य विवरण भर सकते हैं। इस वेब पोस्ट में हम आपको बताएंगे कि आप एल्म में स्ट्रिंग को कैसे काट सकते हैं। 

## कैसे करें

```Elm
-- एल्म ०.१९
import String

-- String.dropLeft n str : String
-- यह फ़ंक्शन दिए गए पैरामीटर्स के अनूसार दिए गए स्ट्रिंग से प्रत्येक अक्षरों को हटा देगा। यहां हमने प्रारम्भिक स्ट्रिंग से पांच अक्षरों को हटाया है।
String.dropLeft 5 "Hello World!"  --> " World!"
```

```Elm
-- String.left n str : String
-- यह फ़ंक्शन दिए गए पैरामीटर्स के अनुसार दिए गए स्ट्रिंग से निर्धारित संख्या के अक्षर लेकर आगे जाएगा। यहां हमने प्रारम्भिक स्ट्रिंग से पांच अक्षर ली हैं।
String.left 5 "Hello World!"  --> "Hello"
```

```Elm
-- String.slice from to str: String
-- यह फ़ंक्शन दिए गए पैरामीटर्स के अनुसार शुरुआत स्थान और अंत स्थान के बीच के अक्षर लेता है। यहां हमने प्रारंभिक स्ट्रिंग से दो अक्षरों ले रखे हैं।
String.slice 1 3 "Hello World!"  --> "el"
```
