## क्यों

वर्तमान समय में, धेर सारे विभिन्न प्रोग्रामिंग भाषाओं में स्ट्रिंग्स (पाठ की ड्राफ्ट) के साथ काम किया जाता है। इन शब्दों का लोगों के माध्यम से संवाद या डेटा की प्राप्ति होती है। कभी कभी हमें इन स्ट्रिंग्स से कुछ विशिष्ट भागों को निकालना पड़ता है जैसे कि एक नाम का प्रथम ग्राम या एक फाइल या फोल्डर का नाम। इससे साथ ही कुछ अन्य उदाहरण भी हो सकते हैं। इस ब्लॉग पोस्ट में हम इल्म प्रोग्रामिंग भाषा में स्ट्रिंग से उस का उपयुक्त भाग निकालने के बारे में जानेंगे।

## कैसे

इसके लिए, हम `String.substring` फंक्शन का उपयोग करेंगे। निम्न उदाहरण में, हमने एक स्ट्रिंग `myString` बनाया है और उससे पहले 5 अक्षर `Hello` को निकाला है।

```Elm
module Main exposing (..)

import Html exposing (Html, div, text)


myString : String
myString =
    "Hello World!"

subString : String
subString =
    String.substring 0 5 myString


main =
    div [] [
        text subString
    ]
```

आउटपुट:

`Hello`

यहाँ हमने `String.substring` फंक्शन के तीन पैरामीटरों का उपयोग किया है - पहला `startingIndex` जहाँ से हम अपनी सबस्ट्रिंग निकालना चाहते हैं, दूसरा `endingIndex` जहाँ से हम अपनी सबस्ट्रिंग को समाप्त करना चाहते हैं और आखिरी `string` जिससे हम हमारी सबस्ट्रिंग को निकालना चाहते हैं। आप अपने ओबरेशंग से मर्ज फंक्शन का भी उपयोग कर सकते हैं।
