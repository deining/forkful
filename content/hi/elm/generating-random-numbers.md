## क्यों

अक्सर, प्रोग्रामिंग में हमे तस्वीर-समझ और नंबर-समझ कैसे बनते है, इसे जानना तो जरूरी होता है। लेकिन कभी-कभी, हमें कुछ अनियंत्रित भाग बनाना पडता है जैसे कि रेंडम नम्बर जेनरेट करना। हम अक्सर एसेकल का इस्तेमाल नहीं करते हैं, जो इसकी वजह से हमें कम नंबर्स दिखते है या नहीं दिखते है। इसलिए, इस ब्लॉग पोस्ट में हम रेंडम नंबर्स को जनरेट करने के विभिन्न तरीकों के बारे में बात करेंगे। 

## कैसे

```Elm
import Random exposing (..)
import String
import Char

type alias Size =
    Int

randomChar : Size -> String
randomChar size =
    let
        character =
            Random.step (Random.chance (1 / Char.fromCode 90))
    in
    String.fromCharList (List.repeat size character)

main =
    randomChar 8
```

यहां, हमने Elm में रेंडम नंबर्स को जनरेट करना सीखा है। इसके लिए, हमने `Random` मोड्यूल को इम्पोर्ट किया है। हम एक `Size` नेतिव नाम्बर्स डिफाइन किया है, जो हमे योग्य साइज़ को पास करने मदार करता है। आखिर में, हम एक मेन मोड्यूल बनाते हैं जो रेंडम नंबर दरेश और कर सुचित करता है। जैसा कि हम देखते हैं, हम एक नाई सर्यत को पास कर सकते हैं जो एक बहुत ही कम संभावना को शह्र करती है। आप अपने साथ-सात खेल सकते हैं और एक बहुत ही अनियंत्रित नंबर जेनरेट कर सकते हो।
