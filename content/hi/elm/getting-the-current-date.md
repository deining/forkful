# क्यों

कॉडिंग करना एक बहुत ही शानदार अनुभव है, लेकिन कभी-कभी हमें वास्तविक दुनिया के तारीख और समय की जानकारी की आवश्यकता होती है। यदि आप इलम कार्यक्रमन को कभी भी सीखते हैं, तो आपको इससे आपके आवश्यकताओं को पूरा करने के लिए स्वतंत्रता मिलती है। तारीख और समय को जानने की क्षमता से, आप ठीक तारीख और समय प्रबंधन के बारे में सीखने का मौका प्राप्त करते हैं, जो आपको आपके प्रोजेक्ट्स में अधिक उपयोगी बनाता है।

# कैसे करें

जब आप ईल्म में वर्तमान तारीख कैसे प्राप्त कर सकते हैं इसके लिए, आपको उपरोक्त दिए गए मूल्यांकन चरणों का पालन करने की आवश्यकता होती है। इसलिए, सर्वप्रथम, आपको `Date` मॉड्यूल को आवश्यकता से आगे रखना होगा। 

``` Elm 
import Date
```

आगे आपको तारीख और समय के लिए एक नया `Date` वस्तु बनाने की आवश्यकता होगी। आप `Date.fromTime` फंक्शन का उपयोग करके ऐसा कर सकते हैं। यह आपको स्वतंत्र तारीख और समय क्षेत्र से तारीख को दिखाएगा। 

``` Elm 
Date.fromTime 1549651200
```

आखिर में, आप `Date` वस्तु के कुछ मौलिक गुणों का पता लगाएंगे, जैसे कि `year`, `month` और `day`. आप उन्हें निम्न प्रकार से प्राप्त कर सकते हैं: 

``` Elm 
date.year
date.month
date.day
```

हमारे नमूना `Date` वस्तु के उदाहरण में, हमारे फलन निम्नानुसार होंग
