## क्यों

तो आपने हमारे चुनाव किया है कि आप एक वेब पेज को डाउनलोड करने में शामिल होंगे। यह उन लोगों के लिए उपयोगी हो सकता है जो मानवीय प्रयोग करके अपने पाठकों को पेंटरर के माध्यम से एक वेब पेज को डाउनलोड करना चाहते हैं।

## कैसे करें

इसे करने के लिए, हम एल्म प्रोग्रामिंग भाषा का उपयोग करेंगे, जो कि वेब डिजाइन से संबंधित समस्याओं को सुलझाने के लिए एक लोकप्रिय विकल्प है। नीचे एक उदाहरण दिया गया है जो कि वेब पेज को डाउनलोड करते समय यह कैसे काम करता है।

```Elm
import Graphics.Element exposing (show)

-- वेब पेज को डाउनलोड करने के लिए फ़ंक्शन
downloadWebPage : String -> Task Error String
downloadWebPage url =
  Http.getString url

-- फ़ंक्शन को कॉल करें
downloadWebPage "http://example.com" 
  |> Task.perform 
    (\err -> show err) 
    (\html -> show html)
```

उपरोक्त कोड से जब हम "http://example.com" को डाउनलोड करने की कोशिश करेंगे, तो हमारे कोड हमें उस वेब पेज के HTML को बाहर निकालने के लिए फ़ंक्शन को कॉल करेगा। उत्पाद स्क्रिप्ट दिखाएगा "http://example.com" की जगह पर HTML को।
