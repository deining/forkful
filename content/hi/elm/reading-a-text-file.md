## क्यों

एक कंप्यूटर फ़ाइल में संख्याओं और तकनीकी शब्दों से भरी हो सकती है। ये फ़ाइलें दस्तावेज़ और डेटा संग्रहीत करने के लिए काम आती हैं। लेकिन कभी-कभी हमें इन फ़ाइलों में से डेटा को पढ़ने की आवश्यकता होती है, जैसे कि एक अनुक्रम को पढ़कर एक व्यावसायिक रूप से जानकारी हासिल करने के लिए। इस ब्लॉग पोस्ट के माध्यम से हम जानेंगे कि एल्म में फाइल को कैसे पढ़ा जाता है।

## कैसे

एल्म में टेक्स्ट फाइलों को पढ़ने के लिए, हमें `File` और `Text` मॉड्यूल के साथ काम करना होता है। हम समझेंगे कि कैसे हम इन नेमस्पेस से उपयुक्त फ़ंक्शन और टाइप निकाल सकते हैं। फ़ाइल को खोलने के बाद, हम `readLines` फ़ंक्शन का उपयोग करके टेक्स्ट को स्ट्रिंग के रूप में पढ़ सकते हैं। नीचे दिए गए कोड ब्लॉक में हम सॉम्पल प्रोग्राम को देखेंगे जो एक संख्या फ़ाइल पढ़ रहा है और संख्या दर्शाता है।

```elm
import File exposing (readLines)
import Text exposing (toInt)

main =
    readLines "numbers.txt"
        |> andThen handleResult

handleResult result =
    case result of
        Ok lines ->
            lines
                |> toInt
                |> List.map (\n -> "यह संख्या है: " ++ (toString n))
                |> String.join ","
                |> text
                |> element

        Err message ->
            text ("यह संख्या फाइल पढ़ने में असमर्थ है: " ++ message)
                |> element
```

सॉम्पल आउटपुट:

```
यह संख्या है: 1,यह संख्या है: 2,यह संख्या है: 3
```
