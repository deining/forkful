# Elm Programming: Text Files क्यों?

एल्म भाषा एक रोचक फ़ीचर है जो टेक्स्ट फाइलों को पढ़ने और प्रोसेस करने की क्षमता प्रदान करती है। यह आपको अनेक उपयोगी फ़ाइल फ़ॉर्मेट को पढ़ने और उनका उपयोग करने में मदद कर सकता है।

## कैसे करें

टेक्स्ट फाइल पढ़ने के लिए एल्म में `Text` और `File` मॉड्यूल का उपयोग कर सकते हैं। कोड ब्लॉकों के भीतर एल्म कोड उदाहरण और सैम्पल आउटपुट दिए गए हैं।

```Elm
import Text exposing (decodeUtf8)
import File

filePath = "textfile.txt"

fileContent = File.read filePath

case fileContent of
    Ok bytes ->
        let
            decodedText = decodeUtf8 bytes
        in
            -- कोड जारी रखें

    Err error ->
        -- त्रुटि संदेश दिखाएं
```

इस दृष्टिकोण से, हम `Text` मॉड्यूल से `decodeUtf8` फ़ंक्शन का उपयोग करके बाईट्स को टेक्स्ट में बदल सकते हैं। और फिर हम `File` मॉड्यूल से `read` फ़ंक्शन का उपयोग करके फ़ाइल को खोल सकते हैं और उसकी सामग्री को बाईट्स के रूप में प्राप्त कर सकते हैं। अगर सब कुछ सफल होता है, तो हम बाइट्स को `decodeUtf8` के माध्यम से टेक्स्ट में बदलकर उसे `decodedText` नाम की चर पर स्थानांतरित कर सकते हैं।

## गहराई में जाएं

टेक्स्ट फाइल पढ़ने में कुछ गहराई में जाने के लिए, आप `Text.Decoder` मॉड्यूल का भी उपयोग कर सकते हैं। यह आपको कस्टम डिकोडर तैयार करने की अनुमति देता है जो आपको अपनी फ़ाइल के स्ट्रक्चर पर समझौते पर लाने
