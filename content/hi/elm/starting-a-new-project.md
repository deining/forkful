Elm क्यों

अब दिलचस्प हो सकता है कि आपको नए प्रोजेक्ट शुरू करने में क्यों रुचि होगी। Elm एक फंक्शनल और टाइप सेफ प्रोग्रामिंग भाषा है जो एक बहुत ही सरल और बहुत शक्तिशाली है। यह एक बहुत प्रभावशाली कंपाइलर है जो कोड के लिए एक आसान प्रक्रिया प्रदान करता है। 

कैसे करें

```Elm
main =
  text "नए प्रोजेक्ट कैसे शुरू करें? यहाँ एक उदाहरण है:"```

```Elm
module Main exposing (main)

import Html exposing (text)

main =
  text "नए प्रोजेक्ट में भाग लें!"```

डीप डाइव

नए प्रोजेक्ट शुरू करने से पहले, आपको Elm को स्थापित करना होगा। आप अपने सिस्टम के लिए प्रयोग की जाने वाली संस्करण को डाउनलोड कर सकते हैं। यदि आपको वहां कोई समस्या होती है, तो आप दूसरे संस्करण हेतु https://guide.elm-lang.org/install.html देख सकते हैं। एक बार जब आपके पास Elm स्टैंडर्ड टूल सेटअप हो, आप सीधे आगे बढ़ सकते हैं। आप अपने संगठन या डिजाइन के अनुसार, या अपनी स्किल के अनुसार एक बहुत ही व्यक्तिगत प्रोजेक्ट को बनाने का आरंभ कर सकते हैं।
