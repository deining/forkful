## क्यों

एक अस्थायी फाइल बनाने का क्रियात्मकता प्रोग्रामिंग में अहम भूमिका निभाती है। वह समय और अक्सर आवश्यकता होती है जब आप एक अस्थायी डेटा को सुरक्षित रूप से संग्रहीत करना चाहते हैं लेकिन उसे स्थायी रूप से बनाने की सामर्थ्य नहीं है।

## कैसे बनाएं

अस्थायी फाइल बनाने के लिए आप इन चरणों का पालन कर सकते हैं:

### 1. मॉड्यूल इंपोर्ट करें
```
Elm.File
    exposing
        ( File
        , tempFile
        )
```

### 2. अस्थायी फाइल बनाएं
```
temp : Result String File
temp =
    tempFile "myTemporaryFile" "txt"
```

### 3. परिणाम प्रिंट करें
```
case temp of
    Ok file ->
        case file of
            ... -> ... -- अस्थायी फाइल से जारी जानकारी हासिल करें
    Err err ->
        ... -- अस्थायी फाइल बनाने के दौरान समस्या का सामना करें
```

## गहराई में

अस्थायी फाइल बनाने के लिए `tempFile` फ़ंक्शन `elm/file` मॉड्यूल में उपलब्ध है। यह दो पैरामीटर लेता है: फाइल का नाम और फाइल का प्रकार। आप मौजूदा फाइल नाम पर भी खाली पैरामीटर का उपयोग कर सकते हैं। इस फ़ंक्शन का परिणाम `Result` डेटा प्रकार होता है, जो अस्थायी फाइल को सफलतापूर्वक बनाने का मार्ग बताता है।
