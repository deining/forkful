नमस्ते हिंदी पाठकों,

जब हम एक नए प्रोग्रामिंग भाषा में शुरुआत करते हैं, तो अधिकतर हमें उस भाषा के फंक्शन और उनका सही उपयोग सिखने की आवश्यकता होती है। इसीलिए आज हम बात करेंगे की कैसे एक string की लंबाई को निकाला जाता है Elm में। यह प्रवेश के लिए जानना आवश्यक है कि यह काम हमारे लिए क्यों महत्वपूर्ण है।

## क्यों

एक string की लंबाई निकालना हमें उसमें कितने अक्षर हैं यह पता करने में मदद करता है। यह हमारे लिए किसी विशेष स्ट्रिंग के साथ काम करने में आसानी देता है और हमें उसे प्रोग्राम में इस्तेमाल करने देता है।

## कैसे

```Elm
-- डेमो:

len : String -> Int
len str =
    String.length str

main =
    "Hello, world!" |> len |> Debug.log
```


इस उदाहरण में, हमने ```String.length``` फंक्शन का उपयोग करके एक string की लंबाई निकाली है। यह ```len``` फंक्शन दो पैरामीटर जोड़कर एक Int को वापस देता है। आप आपस में मिलाने के लिए परिणाम पहले से ही जानते हो, हालांकि इस स्थिति में यह एक सटीक जवाब होगा। यह मानक Elm अपना प्रतिनिधित्व करता है।

## गहराई में जाइए

एल्म की एक और मजेदार बात इस फंक्शन के उपयोग से हम एक string के हर अक्षर को पल्लेट (palette) के रूप में भी प्रकट कर सकते हैं। इस विशेषता को हम मानचित्र कह सकते हैं। हम इसे निम्न उदाहरण में देखेंगे।

```Elm
-- डेमो:

palette : String -> String
palette str =
    String.map String.fromChar str

main =
    "Hello, world!" |> palette |> String
