## क्यों

क्या आपने कभी कोई टेक्स्ट फ़ाइल पर जाना कोशिश की है? फ़ाइलों में या ऐप्स में स्टोर होने वाले डेटा को कहा गया होता है। हिंदी अधिकांश लोग काम करने के लिए टेक्स्ट फ़ाइलों का उपयोग करते हैं। इस लेख में, हम एल्म को प्रयोग करके एक टेक्स्ट फ़ाइल कैसे लिख सकते हैं, इस पर बात करेंगे।

## कैसे

एल्म में एक टेक्स्ट फ़ाइल लिखना बहुत आसान है। आप एक `text` फ़ंक्शन का उपयोग करके एक पाठ स्ट्रिंग बना सकते हैं और फिर उसे एक `file` फ़ंक्शन के साथ संपर्क कर सकते हैं। नीचे दिए गए उदाहरण में, हम एक `myFile.txt` नामक फ़ाइल में "नमस्ते दुनिया" लिखते हैं।

```Elm
import File
import Text

main =
    let
        string = "नमस्ते दुनिया"
        fileContent = Text.text string
        fileName = "myFile.txt"
    in
    File.write fileName fileContent
```

इस स्क्रिप्ट को `elm repl` में चलाएं और देखें कि आपकी फ़ाइल के अंतर्गत क्या आ रहा है।

## डीप डाइव

टेक्स्ट फ़ाइलों को पढ़ने और लिखने के लिए, हम `File` और `Text` लायब्ररीज़ का उपयोग करते हैं। इन लायब्ररीज़ का विस्तृत जानने से पहले, आपको आपके प्रोजेक्ट में उन्हें जोड़ना होगा। आप `elm-package` के साथ `install elm/file` और `install elm/html` कमांड का उपयोग करके इन लायब्ररीज़ को अपने पेजकर्शन में शामिल कर सकते हैं।

इसके अलावा, आप `elm-lang.org` पर अधिक जानकारी पा सकते हैं।
