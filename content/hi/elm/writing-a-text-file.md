## क्यों

टेक्स्ट फ़ाइल लिखना एक प्रतिस्थापित कूदने का प्रयास है। यह दृश्य क्रियाओं और स्थानों को तस्कर्लिप्स प्रदान करने के लिए बहुत उपयोगी हो सकता है। आइए एल्म में एक टेक्स्ट फ़ाइल लिखने का तरीका देखते हैं।

## हाउ टू

टेक्स्ट फ़ाइल बनाने के लिए, हम `File.Writer` प्रक्रिया का उपयोग करेंगे। पहले, हम वर्णनात्मक नाम के साथ एक नया फ़ाइल सिद्धांत को ले लेंगे, फिर हम उस सिद्धांत का उपयोग करके एक नया फ़ाइल लिखेंगे।

```Elm
import File.Writer as Writer

-- एक नया फ़ाइल सिद्धांत बनाएं
sampleFile : Writer.File
sampleFile =
  "output.txt"
    |> Writer.file

-- फ़ाइल लिखें
Writer.writeFile sampleFile "यह टेक्स्ट फ़ाइल से लिखा गया है।"
```

ऊपर दिए गए उदाहरण में, हमने स्ट्रिंग `"यह टेक्स्ट फ़ाइल से लिखा गया है।"` एक नया फ़ाइल `output.txt` में लिखा है। यदि आप `output.txt` फ़ाइल को खोलते हैं, तो आप निम्नलिखित पंक्तियों को देखेंगे:

```
यह टेक्स्ट फ़ाइल से लिखा गया है।
```

## डीप डाइव

इस उपकरण के अलावा, हम `Writer.append` का भी उपयोग कर सकते हैं, जो फ़ाइल में टेक्स्ट जोड़ने की अनुमति देता है। इसका उपयोग करने का तरीका उपर दिए गए उदाहरण से बहुत समान होगा। हम बस `Writer.writeFile` की जगह `Writer.append` का उपयोग करके टेक्स्ट के साथ एक फ़ाइल लिखकर उसे बढ़ा सकते हैं
