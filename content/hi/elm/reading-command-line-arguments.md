## इसलिए

क्या आप एक जादुई कमांड लाइन संदर्भों के साथ एलम में काम कर रहे हैं? या शायद आप नए प्रोग्रामिंग भाषा में देख रहे हैं? या शायद आप अपने एप्लिकेशन को फ्लेक्सिबिलिटी देने की तलाश में हैं? चाहे आप किसी भी वजह से हो, एलम में कमांड लाइन एर्ग्यूमेंट्स सुपर उपयोगी हैं और उन्हें समझना आसान है। चलिए, हम आपको इस बारे में बताते हैं कि आप कैसे एलम में कमांड लाइन एर्ग्यूमेंट्स को पढ़ सकते हैं। 

## कैसे करें

एलम में कमांड लाइन एर्ग्यूमेंट्स पढ़ने के लिए, हमें `Platform` और `Cmd` मॉड्यूल के साथ काम करना होगा। नीचे दिए गए कोड ब्लॉक में हमने एक दूसरे मॉड्यूल से एलम में एर्ग्यूमेंट्स को पढ़ने का उदाहरण दिया है। 

```
import Platform 
import Cmd
```

अब हम `Cmd.able` फ़ंक्शन का उपयोग करके, कमांड लाइन स्ट्रिंग से `args` को पढ़ते हैं। 

```
(·) = 
    Cmd.able
    \{ program, args } ->
        ...
```

अगर आपके प्रोग्राम में अधिक एर्ग्यूमेंट्स हैं, तो आप उन्हें प्रत्येक अलग स्ट्रिंग के साथ एक भंडारण सूची में रख सकते हैं। आप `List.map` फ़ंक्शन का उपयोग करके `args` को परस्तुत सूची में बदल सकते हैं। 

```
(·) = 
    Cmd.able
    \{ program, args } ->
        ...
        let
            argList = 
                List.map (\arg -> arg) args 
        ...
```
