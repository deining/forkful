##क्यों

ऐसे हजारों प्रोग्रामिंग भाषाओं में से हम एल्म को क्यों चुनें? एल्म एक प्रोग्रामिंग भाषा है जो अन्य भाषाओं से अलग है। इसके लिए सबसे महत्वपूर्ण कारण है कि यह प्रोग्रामिंग को बहुत ही आसान बनाता है, साथ ही सुरक्षित भी है। इसका उपयोग आप प्रोजेक्ट्स में दुर्भाग्यपूर्ण त्रुटियों को ढूंढ़ और सुधार करने में कर सकते हैं।

##कैसे करें

```Elm
deletePattern : String -> String -> String
deletePattern pattern string =
  String.replace pattern "" string

main =
  deletePattern "abc" "aabbccddeeff" -- Output: "ddeeff"
```

इस उदाहरण में, हमने `deletePattern` नाम की एक फ़ंक्शन बनाई है जो एक स्ट्रिंग और पैटर्न ले लेती है और उसमे से स्ट्रिंग से सभी पैटर्न को हटा देती है। इसके बाद, हमने `main` फ़ंक्शन में इसका इस्तेमाल किया और उसे सम्पूर्ण `aabbccddeeff` स्ट्रिंग में से `abc` को हटाने में उपयोग किया। आप अपने प्रोजेक्ट में हमारे दिए गए कोड को सही से समझकर लगा सकते हैं।
