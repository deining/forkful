## क्यों

क्या आपको पिछले कुछ समय से हैस्केल प्रोग्रामिंग सीखने का इच्छुक हुआ है? यदि हाँ, तो एक टेक्स्ट फाइल लिखना सीखना आपको एक अच्छा शुरुआती प्रोजेक्ट हो सकता है। टेक्स्ट फाइल एक बहुत मूलभूत प्रोग्रामिंग कौशल है और यह आपको गिनती, खोज और अन्य फाइल प्रबंधन कार्यों में मदद कर सकता है।

## कैसे करें

मैंने नीचे कुछ हैस्केल कोड दिया है जो आपको टेक्स्ट फाइल बनाने का तरीका दिखाता है। पहले हम इस फंक्शन को बनाएंगे:

```Haskell
createTextFile :: String -> String -> IO()
createTextFile fileName contents = writeFile fileName contents
```

इस फंक्शन के दो आर्ग्यूमेंट हैं - फ़ाइल का नाम और उसमें लिखा गया सामग्री। इसके बाद, हम फ़ाइल का नाम लिखकर फ़ाइल को बनाते हैं:

```Haskell
main :: IO()
main = do
  let fileName = "example.txt"
  let contents = "यह टेक्स्ट फाइल है।"
  createTextFile fileName contents
  putStr "फाइल बना दी गई है"
```

इस समय, जब आप अपना प्रोग्राम चलाएंगे, यह फाइल `example.txt` नाम से आपके कंप्यूटर के साथी फ़ोल्डर में बनाई जाएगी। अब आप इस फ़ाइल को खोलकर उसमें देख सकते हैं कि आपने अपने प्रोग्राम में क्या लिखा है। यदि आपके पास सही आधारभूत प्रोग्रामिंग कौशल है, तो आप इस फ़ाइल में मदद के लिए कुछ सामग्री भी जोड़ सकते हैं।
