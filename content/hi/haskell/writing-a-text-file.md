## क्यों

क्या आपने कभी सोचा है कि आप एक पाठ फ़ाइल लिखने के प्रोसेस के बारे में जानना चाहते हैं? शायद आपको प्रोग्रामिंग शिखने के लिए एक नया भाषा सीखने की आवश्यकता हो सकती है या यदि आप एक व्यवसायिक प्रोग्रामर हैं तो आपने यह स्किल अपनाने में मदद को ढूंढा हो सकता है। हास्केल एक फंक्शनल प्रोग्रामिंग भाषा है जो आपको टेक्स्ट फ़ाइल लिखने के लिए अद्भुत फ़ांक्शन्स प्रदान करती है।

## कैसे करें

बस एक खाना वाला प्रोग्राम लिखने से पहले हास्केल में एक पाठ फ़ाइल लिखना सीखने के लिए, आपको निम्न चरणों का पालन करना होगा:

```Haskell
-- फाइल खोलने के लिए हेंडल सेट करें
main = do
  handle <- openFile "sample.txt" WriteMode
  -- फ़ाइल में पाठ लिखें
  hPutStrLn handle "मेरा पहला पाठ फाइल है!"
  -- हेंडल बंद करें
  hClose handle
```

जब आप यह कोड चलाएंगे, तो आपको "sample.txt" नाम की एक नई फ़ाइल में "मेरा पहला पाठ फाइल है!" लिखा मिलेगा।
