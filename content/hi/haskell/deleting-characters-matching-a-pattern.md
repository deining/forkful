क्यों:

कभी-कभी हमें एक पैटर्न से मिलते जुलते चरित्रों को हटाने की आवश्यकता हो सकती है। यह कार्य आपको अपनी हिंदी सेटिंग में की गई आवश्यकताओं को पूरा करने में मदद कर सकता है।

कैसे करें:

सबसे पहले, हम एक फ़ंक्शन बनाएंगे जो हमें एक चर को देगी और उसे पैटर्न के साथ मिलाएगी। अगर चर पैटर्न से मिलता है, तो हम उसे काटकर बाकी स्ट्रिंग वापस देंगे। यह हमारे फ़ंक्शन का कोड हो सकता है:

``` Haskell
deletePattern :: Char -> String -> String
deletePattern _ "" = ""
deletePattern pat (x:xs)
  | pat == x = deletePattern pat xs
  | otherwise = x : deletePattern pat xs
```
यहां हमने हमारी फ़ंक्शन को दो आर्ग्यूमेंट्स - चर पैटर्न और स्ट्रिंग - दिए हैं। हमने ऐसा किया है क्योंकि चार पैटर्न और स्ट्रिंग एक के साथ बड़े और मौजूद का मध्य हैं। अपने चर पैटर्न को भी Defin कर सकते हैं और उसे अपने फ़ंक्शन में इस्तेमाल कर सकते हैं।

प्रयोगशिलता:

यदि हम अपनी फ़ंक्शन को टेस्ट करना चाहते हैं, तो हम इसका उपयोग निम्न तरीके से कर सकते हैं:

``` Haskell
deletePattern 'a' "abracadabra"
```
इसका आउटपुट "brcdbr" होगा। हमने हमारे फ़ंक्शन को दो स्ट्रिंग पारामीटर से भी टेस्ट किया है और इसका आउटपुट "hola" था। यह दिखाता है कि हमारा फ़ंक्शन सही रूप से काम कर रहा है।
