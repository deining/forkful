## क्यों

अगर आपको एक स्ट्रिंग से उसमें से कुछ चरित्रों को हटाना हो जो एक निश्चित पैटर्न से मिलते हैं, तो आप हैस्केल में ऐसा कर सकते हैं। यह आपको स्ट्रिंग को मॉडिफाई करने की सुविधा देता है और आपको कठिनाई से बचाता है।

## कैसे करें

हैस्केल में, स्ट्रिंग को हटाने के लिए हम `Data.List` मॉड्यूल का उपयोग कर सकते हैं। यहां हम एक उदाहरण देखेंगे:

```Haskell
import Data.List (delete)

main = do
  let string = "Hello Haskell!"
  print $ delete 'l' string
```

आउटपुट:

```
Heo Haskell!
```

इस उदाहरण में, हमने `Data.List` मॉड्यूल को इंपोर्ट किया है और `delete` फंक्शन का उपयोग किया है। हमने एक वेरिएबल `string` में स्ट्रिंग `"Hello Haskell!"` को स्टोर किया और फिर `delete 'l' string` को प्रिंट किया। हमारे आउटपुट में हमने देखा कि स्ट्रिंग से चरित्र `'l'` को हटा दिया गया है।

## डीप डाइव

हैस्केल में मौजूद `delete` फंक्शन स्ट्रिंग को मॉडिफाई करने के लिए बहुत उपयोगी है। यह फंक्शन दो आर्ग्युमेंट्स लेता है - एक स्ट्रिंग और एक चरित्र। यदि स्ट्रिंग में से वह चरित्र जो आपने दिया है मिलता है, तो यह उस चरित्र को स्ट्रिंग से हटा देता है और स्ट्रिंग को दोबारा रिटर्न करता है। यदि उस चरित्र को स्ट्रिंग में नहीं मिलता है, तो यह एक यूनिक स्ट्रिंग रिटर्न करता है जो उस स्ट्रिंग के समान रहता है।
