## क्यों

टेक्स्ट फ़ाइल पढ़ने के लिए होने वाली कामनियों के बारे में जानना क्यों अधिक महत्वपूर्ण हैतोह पढ़ना काफी सरल लग सकता है, लेकिन हिंदी रीडर्स के लिए हैस्केल प्रोग्रामिंग की शुरुआत करना थोड़ा चुनौतीपूर्ण हो सकता है। यह आर्टिकल आपको टेक्स्ट फ़ाइल पढ़ने के लिए हैस्केल का इस्तेमाल करना सिखाएगा और आपको इस प्रोग्रामिंग भाषा में अधिक से अधिक शिक्षा दे सकता है।

## कैसे करें

टेक्स्ट फ़ाइल पढ़ने के लिए हैस्केल में निम्नलिखित कोड स्निपेट दिए गए हैं। प्रत्येक उदाहरण के साथ, हम एक टेक्स्ट फ़ाइल से संबंधित जानकारी पढ़ेंगे।

```Haskell
import System.IO

main = do
    file <- openFile "example.txt" ReadMode
    contents <- hGetContents file
    putStr contents
    hClose file
```

इस कोड में, हमने "example.txt" नाम की एक टेक्स्ट फ़ाइल को खोला है और उस फ़ाइल से अपने कंटेंट्स को पढ़ने के लिए `hGetContents` फ़ंक्शन का इस्तेमाल किया है। इसके बाद, हमने `putStr` फ़ंक्शन का इस्तेमाल करके फ़ाइल को प्रिंट किया है और अंत में `hClose` फ़ंक्शन का इस्तेमाल करके फ़ाइल को बंद किया है।

अगर आप चाहे तो, आप अपनी उपयोगकर्ता को भी टेक्स्ट फ़ाइल से अलग-अलग जानकारियां पढ़ा सकते हैं। नीचे दिए गए कोड स्निपेट में, हमने टेक्स्ट फ़ाइल से लाइनें पढ़ीं हैं और उन
