## क्यों

क्या आपने कभी सोचा है कि कंप्यूटर पर संग्रहीत डेटा में से डेटा को कैसे नवीनीकरण किया जाए? आप कभी-कभी किसी टेक्स्ट फ़ाइल को पढने के लिए टेक्स्ट एडिटर का उपयोग कर सकते हैं लेकिन क्या आप जानते हैं कि हैस्केल भाषा में भी आप एक टेक्स्ट फ़ाइल को पढ़ सकते हैं और इसमें संग्रहीत डेटा को नवीनीकरण कर सकते हैं। इस ब्लॉग पोस्ट में हम आपको बताएंगे कि हैस्केल में टेक्स्ट फ़ाइल पढ़ना कैसे संभव है और इससे कैसे लाभान्वित हो सकते हैं।

## कैसे करें

```Haskell
-- फ़ाइल खोलें
handle <- openFile "myFile.txt" ReadMode

-- कन्टेंट पढ़ें
contents <- hGetContents handle

-- कन्टेंट हस्ताक्षर को कहीं भी पार्स करें
let fileContents = lines contents

-- कन्टेंट के विशिष्ट भाग लें
let firstLine = head fileContents

-- फ़ाइल बंद करें
hClose handle

-- उत्पादन
putStrLn firstLine
```

उपरोक्त कोड ब्लॉक में हमने टेक्स्ट फ़ाइल को खोलने, कन्टेंट्स पढ़ने, हस्ताक्षर को पार्स करने और उत्पादन करने की प्रक्रिया को दर्शाया है। हमने ```openFile``` और ```hGetContents``` फ़ंक्शन का उपयोग किया है जो कि संग्रहीत डेटा को खोलने और पढ़ने में मदद करते हैं। उत्पादन में हमने पहली पंक्ति को प्रिंट किया है लेकिन आप अपनी सुविधा के अनुसार किसी भी कन्टेंट को प्रिंट कर सकते हैं।
