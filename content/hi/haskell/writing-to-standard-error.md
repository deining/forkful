## क्यों

क्या आप कभी हेस्केल प्रोग्रामिंग में ```standard error``` पर लिखने का अनुभव किया है? यह आमतौर पर एक छोटे लोग ही जानते हैं इसलिए इस ब्लॉग पोस्ट के माध्यम से मैं आपको बताऊंगा कि इसे क्यों और कैसे किया जाता है।

##  कैसे करें

बहुत सारे अनुभवी हेस्केल प्रोग्रामर्स (programmers) ```standard error``` पर लिखने का गंभीरता से उपयोग करते हैं। इस लेख में, मैं आपको कुछ कोडिंग उदाहरण भी दिखाऊंगा जो आपको समझने में मदद करेंगे।

```Haskell
main = do
    let num1 = 10
    let num2 = 0
    putStrLn "Enter a number: "
    num <- getLine
    let inputNum = read num :: Int
    putStrLn $ "Result: " ++ show (num1 `div` num2)
```

उपरोक्त कोड के लिए, यदि आप 0 को दरवाजे के माध्यम से दर्ज करते हैं, तो आप ```divideByZero``` त्रुटि (error) को प्राप्त करेंगे। यह उदाहरण आपको दिखाता है कि ```standard error``` कैसे आपको गलतियों को दिखाने कर लेता है जो कोई इस्तेमालकर्ता (user) को लगता है।
