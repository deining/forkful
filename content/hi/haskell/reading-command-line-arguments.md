Haskell में कमांड लाइन आर्ग्यूमेंट पढ़ना एक बहुत ही महत्वपूर्ण कौशल है। यह कैसे हमारे प्रोग्राम को अनुकूलित बनाता है जो कि अपनी स्वतंत्रता को बढ़ाता है, साथ ही, उपयोगकर्ता द्वारा निर्दिष्ट विषयों को प्रस्तुत करने का भी एक सरल तरीका है। तो चलिए जानते हैं हम हैस्केल में कैसे कमांड लाइन आर्ग्यूमेंट पढ़ सकते हैं।

## क्यों

कमांड लाइन आर्ग्यूमेंट हमारे विभिन्न प्रोग्राम्स को अपने अनुकूल करने के लिए उपयोगी होते हैं। ये हमारे प्रोग्राम को अधिक सामान्य और अनुकूलित बनाते हैं, साथ ही उपयोगकर्ताओं को भी एक आसान तरीके से अपने निर्दिष्ट विषयों को ज्ञात करने का विकल्प प्रदान करते हैं।

## कैसे करें

कमांड लाइन आर्ग्यूमेंट पढ़ने का सबसे सरल तरीका `getArgs` फ़ंक्शन का उपयोग करना है। हम इसे `System.Environment` मॉड्यूल से आयात कर सकते हैं। यह फ़ंक्शन कमांड लाइन के सभी आर्ग्यूमेंट्स को एक लिस्ट में वापस करता है।

चलिए एक उदाहरण देखें:
```Haskell
import System.Environment

main = do
  args <- getArgs
  putStrLn ("Number of arguments: " ++ show (length args))
  putStrLn ("Arguments: " ++ show args)
```
प्रोग्राम को कंपाइल करें और परिणाम देखें:
```bash
$ ghc command-line-args.hs       # प्रोग्राम कॉम्पाइल करें
$ ./command-line-args.exe test   # परिणाम देखें
Number of arguments: 1
Arguments: ["test"]
```
