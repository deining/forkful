## सबसे पहले: क्यों

क्या आपको कभी पता चला है कि हास्केल में टेम्पररी फाइल्स कैसे बनाएं? हाँ, इसे बनाना थोड़ा असान हो सकता है, लेकिन आपको शायद इससे आगे क्यों जानने की जरूरत हो सकती है। इस ब्लॉग पोस्ट में हम आपको बताएंगे कि आप टेम्पररी फाइल्स क्यों और कैसे बनाएं। 

## कैसे करें

टेम्पररी फाइल्स का उपयोग डेटा या अन्य जानकारी को संबंधित सत्रों और सेटिंग्स के बीच संचार करने के लिए किया जाता है। ये फाइलें कुछ समय के लिए ही होती हैं और फिर स्वचालित रूप से हट जाती हैं। इसका उपयोग किसी एप्लिकेशन के प्रक्रियाओं को सुविधाजनक बनाने के लिए किया जाता है। चलिए हम इसका उदाहरण देखें।

```
मैंने हास्केल में निम्नलिखित कोड लिखा है:

```
{-
  टेम्पररी फाइल बनाने के लिए आप निम्न कोड को लिख सकते हैं:
-}
import System.IO
import System.Directory
import System.IO.Temp

main = do
  tempFile <- openTempFile "/tmp" "haskeltmp"
  let fileName = fst tempFile
  putStrLn $ "आपने नया फाइल बनाया है: " ++ fileName

```

और यह है आपको उत्पादित फाइल का नाम:

```
आपने नया फाइल बनाया है: /tmp/haskeltmpআइ।

```
देखा गया है कि अपनी फाइल को "haskeltmpআइ।" नाम दिया गया है, यह उपयोगकर्ता से स्वचालित रूप से हटाई जाएगी। अब जब आप चाहें तो इस फाइल को बनाने के लिए अतिरिक्त कोड को अपने हास्केल एप्लिकेशन में श
