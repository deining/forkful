# क्यों

क्या आप जानते हैं कि गो भाषा में आज की तारीख को प्राप्त करना बहुत आसान है? आइए इस ब्लॉग पोस्ट में हम देखेंगे कि गो में आज की तारीख को कैसे प्राप्त किया जा सकता है और इसके क्या फायदे हैं।

# कैसे

कोड की मदद से आज की तारीख को प्राप्त करना गो में बहुत ही आसान है। हम `time.Now()` फंक्शन का उपयोग कर सकते हैं। नीचे दिए गए कोड ब्लॉक में दिखाए गए कोड इसका उदाहरण है:

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    today := time.Now()
    fmt.Println("आज की तारीख है:", today)
}
```

जैसा कि आप देख सकते हैं, हमने `time.Now()` फंक्शन को `today` नाम के मेमोरी उपकरण में स्थानांतरित कर दिया है और फिर इसे प्रिंट किया। इससे हमें आज की तारीख मिल गई। आप अपने कोड में इसका उपयोग अलग-अलग तरीकों से कर सकते हैं।
