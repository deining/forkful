## क्या और क्यों?
"Golang" में वर्तमान तिथि प्राप्त करना क्या है, और प्रोग्रामर्स इसे क्यों करते हैं।

वर्तमान तिथि प्राप्त करना अपनी वेब ऐप्लिकेशन, सिस्टम समय, या अन्य कार्यों को स्थानांतरित करने के लिए उपयोगी है। इसके अलावा, विश्व समय को समायोजित करने और अनुकूलित करने के लिए भी प्रोग्रामर्स वर्तमान तिथि को प्राप्त करना पसंद करते हैं। 

## कैसे करें?
```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    now := time.Now()
    fmt.Println(now.Format("Monday, January 2, 2006"))
}
```
```मंगलवार, जनवरी 2, 2006```

ऊपर दिए गए उदाहरण में, हम गोलांग के "time" पैकेज का उपयोग करके वर्तमान तिथि को प्राप्त कर रहे हैं। प्राप्त हुई तिथि को हम अपनी जरूरत के अनुसार फॉर्मेट कर सकते हैं। ऊपर दिए गए उदाहरण में, हमने वर्तमान तिथि को "Monday, January 2, 2006" फॉर्मेट में दिखाया है। 

## गहराई से जानें
गोलांग को 2009 में शुरू किया गया था। वर्तमान तिथि प्राप्तिकरण के लिए अन्य भाषाओं में भी विकल्प हों सकते हैं, जैसे "C" में "time.h" और "Java" में "java.util.Date"। यदि आपके पास ज्यादा समय नहीं है तो आप डिफॉल्ट तरीके से वर्तमान तिथि प्राप्त कर सकते हैं।
