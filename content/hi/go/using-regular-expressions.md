## क्यों

क्या आप Go प्रोग्रामिंग करते हो? यदि हां, तो आपने शायद नाम सुना होगा - ये कहां से भी मिल सकते हैं। आम गोरों की तरह, मुझे भी यह पहले से ही पता था कि रेगुलर एक्सप्रेशन क्या होती है? लेकिन जब मैंने उन्हें सीखें और उनका उपयोग करना शुरू किया, तो मुझे काफी आकर्षित किया। इसलिए आइए हम इस टूल के उपयोग की कुछ बातें सीखते हैं।

## कैसे करें

यदि आप Go का उपयोग करते हैं, तो संभवतः आपको समझ में आया होगा कि एक शब्द या पाठ की खोज करने के लिए आपको क्या करना होता है। वह आपकी खोज क्रिया के लिए आपके पास क्या पैटर्न है। रेगुलर एक्सप्रेशन को Go में कैसे उपयोग करने के बारे में जानने के लिए नीचे दिए गए कोड ब्लॉक को ध्यान से पढ़ें। यह आपको अच्छी ओर से समझाएगा कि कैसे आप इस टूल का उपयोग कर सकते हैं।

```Go
package main

import (
	"fmt"
	"regexp"
)

func main() {
	// पैटर्न बनाएँ जिसमें ध्यान से दिए गए पाठ की खोज करें
	pattern := "Hello"

	// सामान्य एक्सप्रेशन का उपयोग करें
	regex := regexp.MustCompile(pattern)

	// उक्त पाठ से मिलता-जुलता पाठ खोजें
	text := "Hello World!"

	// क्या खोज में सफलता हुई?
	match := regex.MatchString(text)

	// हाँ, हमें मेल गया है!
	if match {
		fmt.Println("हाँ, पाठ में खोज सफल हुई!")
	} else {
		fmt.Println("पाठ में खोज सफल नहीं हुई।")
	}
}
```

आप देख सकते हैं कि हमने `regexp` लाइब्रेरी का उपयोग कर
