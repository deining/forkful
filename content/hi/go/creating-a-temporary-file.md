## क्यों
क्या आप कभी एक टेम्पररी फ़ाइल बनाने की आवश्यकता महसूस करते हैं? आप अपने कोड के दौरान अस्थायी फ़ाइलों का उपयोग क्यों कर सकते हैं? इस ब्लॉग पोस्ट में, हम आपको बताएंगे कि Go बोलते हुए और लिखते हुए आयाम कैसे बनाएंगे और अस्थायी फ़ाइल बनाने में आपको कैसे मदद मिल सकती है।

## कैसे करें
अस्थायी फ़ाइल बनाना Go में बहुत आसान है। आप सिर्फ निम्न दिए गए कोड स्निपेट को फॉरमेट करें और उपयोग करें:

```Go
// निर्भरता लायिब्रेरी ऑफ्फि
import "io/ioutil"
import "os"

// अस्थायी फ़ाइल बनाने का प्रक्रिया
अस्थायी, err: = ioutil.TempFile ("", "उदाहरण_")
यदि err! = निल {पैनिक (err)}
डिफ़र अस्थायी.Close ()

fmt.Println (tempFile.Name ()) // उदाहरण_294134677
```

आप ऊपर दिए गए कोड से देख सकते हैं कि हमने `ioutil` और `os` लायिब्रेरी का उपयोग किया है और हमने एक अस्थायी फ़ाइल बनाने के लिए `ioutil.TempFile()` का उपयोग किया है। हमने फ़ाइल का नाम भी निर्धारित किया है, `उदाहरण_` और यह एक रैंडम संख्याओं से बनी हुई एक हस्ताक्षर होगी। आप `tempFile.Name()` का उपयोग करके इस नाम को देख सकते हैं।

एक और तरीका अस्थायी फ़ाइल बनाने का है `os.CreateTemp()` जो आपको दो विभिन्न रिटर्न वैल्यू देता है, `फाइल * os.File` और `err error`। आप निम्न दिए गए कोड से इसका उपयोग कर सकते हैं
