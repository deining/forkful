फ़्रांस कम्प्यूटर भाषा में, आप अस्थायी फ़ाइल (temporary file) को एक फाइल की तरह इस्तेमाल कर सकते हैं। यह सामान्यतया आपको कुछ काम करने के लिए कुछ समय के लिए संभावित डेटा या संसाधन को स्टोर करने की अनुमति देती है।

## क्यों

लोगों को अस्थायी फ़ाइल (temporary file) बनाने के लिए कई कारण हो सकते हैं। जैसे कि, एक सुरक्षात्मक उद्देश्य के लिए, यह आपके प्रोग्राम को सामान्य उपयोगकर्ताओं से कुछ गैर-आवश्यक डेटा को छिपाने में मदद कर सकती है। आपको लंबी समय तक उस डेटा को स्टोर करने की आवश्यकता नहीं होगी, तो यह आपके प्रोग्राम को सुरक्षित रखता है।

## कैसे

अस्थायी फ़ाइल (temporary file) को बनाने के लिए, हम `os` और `io/ioutils` पैकेज का उपयोग कर सकते हैं। हम नीचे एक उदाहरण युक्त `createTempFile` फ़ंक्शन दे रहे हैं, जो आधारभूत रूप से एक अस्थायी फ़ाइल (temporary file) बनाता है।

```Go
package main

import (
	"io/ioutil"
	"log"
	"os"
)

func createTempFile() {
	f, err := ioutil.TempFile("", "example")
	if err != nil {
		log.Fatal(err)
	}
	defer os.Remove(f.Name())

	// use the temp file
}
```

इस उदाहरण में, हम `ioutil.TempFile` फ़ंक्शन का उपयोग करते हैं जो एक अस्थायी फ़ाइल (temporary file) बनाता है और उसका नाम लौटाता है। हम `os.Remove` फ़ंक्शन का उपयोग बाद में करते हैं जो फ़ाइल को हटाता है जब हम उसको नहीं इस्तेमाल कर रहे होते हैं।
