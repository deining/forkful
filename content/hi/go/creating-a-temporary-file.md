# क्यों

क्या आपने कभी किसी एप्प्लिकेशन को काम करते वक्त ताजगी का अनुभव किया है? क्या आपको फाइलें बनाने और उन्हें हटाने का काम बहुत ही बोरिंग लगता है? अगर हाँ, तो यह आर्टिकल आपके लिए है। हम बात करेंगे कि गो प्रोग्रामिंग में टेम्परेरी फाइल का उपयोग क्यों और कैसे किया जाता है और इससे आपको कितना फायदा हो सकता है।

# कैसे करें

गो लैंग्वेज में टेम्परेरी फाइल बनाने की कुछ आसान स्टेप्स हैं।

```Go
package main

import "io/ioutil"

func main() {
    tempFile, err := ioutil.TempFile("", "example")
    if err != nil {
        panic(err)
    }
}
```

इस कोड में हमने `ioutil` का `TempFile` फंक्शन कोल किया है जो दो पैरामीटर्स लेता है - पहला *फ़ोल्डर* (जिसमे आपने टेम्प फाइल बनानी है) और दूसरा *नाम* (टेम्प फाइल का नाम)। यह फंक्शन हमें `*os.File` और `error` टाइप की रिटर्न करता है। यदि कोई एरर होता है, तो हम `panic` करते हैं और अपनी एप्प्लिकेशन को क्रैश करते हैं।

यहाँ हमने `ioutil` के अलावा अन्य पैकेज भी इम्पोर्ट किए हैं जैसे कि `io/ioutil` और `os`। ये पैकेज हमें टेम्परेरी फाइल को रीड और डिलीट करने के लिए आवश्यक फंक्शन प्रदान करते हैं।

```Go
tempFile.WriteString("Hello world!")
tempFile.Close()
```

इन कमांड्स से हम नई टेम्प फाइल में "Hello world!" लिखेंगे और फिर उसे क्लोज करेंगे। हमें अपनी फाइल को मेमोरी से फ्री करना भी बहुत आवश्यक होता है।
