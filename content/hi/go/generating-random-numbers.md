## क्यों

गू जावा स्क्रिप्टिंग भाषा का उपयोग करके रैंडम नंबर्स बनाने के क्या फायदे हैं? रैंडम नंबर्स सिस्टम का उपयोग बहुत सारे कम्प्यूटर प्रोग्रामिंग में किया जाता है, जैसे कि गेम डेवलपमेंट, समास्या हल करने, और साइट संदर्भन जैसे। एक तरह से, रैंडम नंबर्स जीवन में ऊर्जा का स्रोत हैं।

## कैसे करें

यदि आप गू का उपयोग करके रैंडम नंबर्स बनाना चाहते हैं, तो आपको सबसे पहले `math/rand` पैकेज को इंपोर्ट करना होगा। इसके बाद, आप `Intn()` फ़ंक्शन का उपयोग करके बहुत से चाहिए नंबर्स बताते हैं। उदाहरण के लिए:

```Go
package main

import (
    "fmt"
    "math/rand"
)

func main() {
    // रैंडम नंबर प्रिंट करें
    fmt.Println("रैंडम नंबर:", rand.Intn(100))
}
```  

इस दृष्टांत में, हमने 0 से 99 तक के बीच एक रैंडम नंबर की ताक लगाई है। आप यह नंबर अपने स्क्रिप्ट में जितने चाहों, उससे भी बड़ा या छोटा बदल सकते हैं। 

## गहराई से जाने

जब आप `Intn()` फ़ंक्शन का उपयोग करते हैं, तो वह आपके द्वारा दी गई सीमा में एक यादृच्छिक संख्या बनाता है। लेकिन असल में, गू में रैंडम नंबर कैसे बनाए जाते हैं? वास्तव में, गू एक पीएसीजी (प्रूडेंस मैरसौड मोशन) अल्गोरिदम का उपयोग करके रैंडम नंबर बनाता है। यह
