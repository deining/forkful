## क्यों

गैर-स्थायी प्रायोजन के लिए नंबर उत्पन्न करने का हमेशा से ही प्रयास प्रोग्रामिंग के लिए अच्छा तरीका रहा है। यह दुनियाभर के डेवलपर्स के लिए एक मज़ेदार और चुनौतीपूर्ण विषय है। अक्सर रैंडम नंबर्स की एक्सेस की आवश्यकता होती है, जैसे कि गेम्स और सिक्योरिटी टेस्टिंग में। गो प्रोग्रामिंग में यह भी बहुत आसान है और इस काम के लिए अनेक तरीके हैं।

## कैसे करें

```Go
package main

import (
	"fmt"
	"math/rand"
)

func main() {
	// रैंडम नंबर को सेट करने के लिए सीड वैल्यू सेट करें
	rand.Seed(time.Now().UnixNano())

	// 0 से 9 तक के बीच एक संख्या लौटाएँ
	fmt.Println(rand.Intn(10))

	// 1 से 100 तक के बीच एक संख्या लौटाएँ
	fmt.Println(rand.Intn(100) + 1)
}
```

बहुत आसान, ना? हम प्रोग्राम के लिए दोनों में से किसी भी संख्या को बदल सकते हैं। यहां हम तीन अलग-अलग तरीकों को देखेंगे जिनसे आप रैंडम नंबर्स को उत्पन्न कर सकते हैं। आओ चलते हैं।

```Go
// 0 से 9 तक के बीच एक संख्या लौटाएँ
fmt.Println(rand.Intn(10))
```

इस उदाहरण में, हमने `rand.Intn()` फ़ंक्शन का उपयोग किया है जो स्थिर अंकों के बीच एक संख्या लौटाता है। हमने इसे 0 से 9 तक सेट किया है। आप 0 से शुरू होने वाले किसी भी संख्या का उपयोग कर सकते हैं।
