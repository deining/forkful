अभी, तकनीक उन्नति के साथ, अधिक समाधान और उच्च-अस्थायी समस्याओं की ओर जा रही है। इसलिए, हिंदी अधिकांश पाठक अपने पुस्तकालय की संपत्ति में द्विस्तम्भ और उसके उपयोगों से वाकिफ होना चाहते हैं। इसके लिए हमे go भाषा सीखनी चाहिए ताकि हम सीख सकें कि आखिरकार go भाषा को design क्यों की गई थी। कोडिंग सिखने के लिए कई स्थानों में हमने प्रवेश किया भी है और उसके एक तरीके के रूप में standard error से कैसे लिखने।

# क्यों

सीखने के लिए नई भाषा हो सकती है, लेकिन standard error के साथ कोडिंग करना एक अनुभव से भरा है। यह आपको खोज के रास्ते में मदद करता है और अन्य प्रोग्रामिंग की भाषाओं से भिन्न बनता है। इससे आपके कोडिंग स्किल्स भी सुधारते हैं। इसलिए, go भाषा में standard error को सीखना बहुत महत्वपूर्ण है। 

# कैसे करें

```Go 
package main 

import (
	"fmt"
	"os"
) 

func main() { 
	fmt.Fprintln(os.Stderr, "यह एक टेक्स्ट स्टैण्डर्ड error में लिखा गया है") 
}
```

इस संकलन से, हमने main पैकेज में fmt पैकेज और os पैकेज को आयात किया है। नेत्रकाल से अभिन्नरूप से, हमने अपने कोड में fmt.Fprintln फंक्शन का उपयोग किया है जो एक string को टेक्स्ट स्टैण्डर्ड error में लिखता है। 

### आउटपुट:
यह एक टेक्स्ट स्टैण्डर्ड error में लिखा गया है
