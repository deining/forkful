## क्यों

क्या आपको कभी लगा है कि आपका कोड सही तरीके से काम नहीं कर रहा है और आपको कोई गलती नहीं दिख रही है? यह समस्या हर प्रोग्रामर के साथ होती है। इसका समाधान है, आपको अपने कोड को स्टैंडर्ड एरर के माध्यम से लिखना होगा ताकि आप गलतियों को पकड़ सकें और अपने कोड को सही कर सकें। इस लेख में हम आपको बताएंगे कि Go प्रोग्रामिंग भाषा में स्टैंडर्ड एरर पर कैसे लिखें और इससे कैसे लाभ हासिल करें।

## कैसे करें

स्टैंडर्ड एरर (Standard Error) टर्मिनल में प्रिंट होने वाली गलतियों और संदेशों को दर्शाता है। Go में इसे स्टैंडर्ड लाइब्रेरी के साथ अधिकृत किया गया है। नीचे दिए गए उदाहरण में, हमें कार्यक्रम का नाम, दूसरा पैरामीटर और एक स्ट्रिंग (गलती या संदेश) पारित करना होगा जो आपको स्टैंडर्ड एरर पर प्रिंट करने के लिए उपयोगी होगा।

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    // स्टैंडर्ड एरर पर प्रिंट
    fmt.Fprintf(os.Stderr, "एक साधारण गलती हुई।\n")

    // स्टैंडर्ड एरर पर संदेश प्रिंट
    fmt.Fprintf(os.Stderr, "यहां एक और साधारण संदेश है।\n")

    // गलती के साथ प्रिंट
    fmt.Fprintf(os.Stderr, "गलती: %v\n", "गलत विधि")

    // अन्य उदाहरण
    fmt.Fprintf(os.Stderr, "संदेश: %v, %s\n", "संदेश", "पैरामीटर")
}
```
