## क्यों

कभी-कभी कोडिंग में हमें किसी भी डायरेक्टरी की उपस्थिति को जांचने की जरूरत पड़ती है। इससे हम यह सुनिश्चित कर सकते हैं कि कोड ठीक से काम कर रहा है और कोई त्रुटि नहीं है। इस ब्लॉग पोस्ट में, हम Go प्रोग्रामिंग के एक महत्वपूर्ण विषय पर ध्यान देंगे - कि कैसे हम जांच सकते हैं कि दिए गए डायरेक्टरी का अस्तित्व है या नहीं।

## कैसे करें

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	// डायरेक्टरी का नाम निर्धारित करें
	dirName := "testDirectory"

	// डायरेक्टरी का अस्तित्व जांचें
	_, err := os.Stat(dirName)
	if err != nil {
		// अगर डायरेक्टरी मौजूद नहीं है, तो एक नई डायरेक्टरी बनाएं
		err := os.Mkdir(dirName, 0777)
		if err != nil {
			fmt.Println(err)
		}
	} else {
		fmt.Println(dirName, "डायरेक्टरी पहले से मौजूद है")
	}
}
```
इस कोड से, हम पहले डायरेक्टरी का अस्तित्व जांचते हैं। अगर डायरेक्टरी मौजूद नहीं होती है तो हम एक नई डायरेक्टरी बनाते हैं। अगर डायरेक्टरी मौजूद होती है तो कोड वहां से आगे बढ़ता है।

यह हमारे स्क्रीन आउटपुट का एक उदाहरण है:

```
testDirectory डायरेक्टरी पहले से मौजूद है
```

## गहराई में जाएं

डायरेक्टरी का अस्तित्व जांचने के लिए os.Stat() फ़ंक्शन का उपयोग किया जाता है। यह एक *os.FileInfo अधिकार हस्ताक्षर* वापस करता है। इसमें हमें उपयोगी जानकारी मिलती है
