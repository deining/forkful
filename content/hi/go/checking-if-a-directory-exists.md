## क्या और इसका क्या कारण है?
डायरेक्टरी की मौजूदगी की जांच क्या है और क्यों प्रोग्रामर इसको करते हैं? डायरेक्टरी की मौजूदगी की जांच वह तरीका है जिससे हम जान सकते हैं कि किसी प्रतिष्ठित मानक फाइल जो सिस्टम पर संग्रहीत है, वह हमारे सुविधापूर्ण सूचना संरचना में है या नहीं। यह सुनिश्चित करने के लिए कि आवश्यक डायरेक्टरी उपलब्ध है या नहीं, जहां हमारे अनुप्रयोग की सूचनाएं रखी गई हैं।

## कैसे करें:
डायरेक्टरी की मौजूदगी की जांच दो तरीकों से की जा सकती है - सामान्य तरीका और उनलोकल तरीका। सामान्य तरीका में, हम `os.Stat()` फंक्शन का उपयोग करके जांच सकते हैं कि निर्दिष्ट डायरेक्टरी मौजूद है या नहीं। यह फंक्शन एक त्रुटि और एरर के उपयोग साथ साथ एक जांच भंडारण को वापस करता है। उनलोकल तरीका में, हम `os.IsNotExist()` फंक्शन का उपयोग करके जांच सकते हैं कि डायरेक्टरी मौजूद है या नहीं। नीचे दिए गए उदाहरण में, हम पहले डायरेक्टरी की मौजूदगी की जांच करते हैं, फिर अपनी सूचनाएं देखते हैं।

```Go
import (
  "fmt"
  "os"
)

func main() {
  // सामान्य तरीका
  if _, err := os.Stat("sample_directory"); os.IsNotExist(err) {
    fmt.Println("डायरेक्टरी मौजूद नहीं है।")
  } else {
    fmt.Println("डायरेक्टरी मौजूद है।")
  }

  // उनलोकल तरीका
  if _, err := os.Stat("sample_directory"); err != nil {
    fmt.Println("डायरेक्टरी मौजूद नहीं है।")
  } else {
    fmt.Println("डायरेक्टरी मौजूद है।")
  }
}
```

## गहराई से जानें:
डायरेक्टरी की मौजूदगी की जांच हमारी अनुप्रयोग में आवश्यक होती है जब हम किसी फाइल को सिस्टम पर संग्रहीत करते हैं। यह सुनिश्चित करने के लिए कि फाइल सामान्य अवधि के दौरान पहुंचने योग्य है या नहीं। अन्य विकल्पों में, हम डायरेक्टरी की मौजूदगी को जांचने के लिए निम्नलिखित तरीके का उपयोग भी कर सकते हैं:

- `filepath.Dir()` फंक्शन का उपयोग करके फाइल का नाम निकालें और फिर उसे आपकी पास की फाइल सार्वजनिक निर्देशिका के साथ तुलना करें।
