## क्यों

कई बार जब हम किसी प्रोजेक्ट को बनाते हैं, तो हमें ऑपरेशन के दौरान कुछ डायरेक्टरीज को चेक करने की जरूरत पड़ सकती है। यह जाँच करना कि एक डायरेक्टरी के अस्तित्व में कुछ समस्या है या नहीं, हमारे कोड में सुरक्षा को मजबूत बनाने के लिए बहुत महत्वपूर्ण हो सकता है।

## कैसे करें

चेक करने के लिए क्या डायरेक्टरी मौजूद है, हमें रेखांकित करने के लिए गो के `os` और `path` पैकेज का उपयोग करना होगा। नीचे दिए गए कोड ब्लॉक में हम एक डायरेक्टरी के अस्तित्व को जाँचने के लिए एक उदाहरण दिखाएंगे।

```Go
package main

import (
    "fmt"
    "os"
    "path"
)

func main() {
    // चेक करें कि क्या "test" नाम का डायरेक्टरी मौजूद है।
    if dirExists("test") {
        fmt.Println("यह डायरेक्टरी मौजूद है!")
    } else {
        fmt.Println("यह डायरेक्टरी मौजूद नहीं है।")
    }
}

// डायरेक्टरी के अस्तित्व को चेक करने के लिए एक फंक्शन।
func dirExists(dir string) bool {
    // डायरेक्टरी का पूरा पथ बनाएं
    path := path.Join(".", dir)

    // फाइल सिस्टम में डायरेक्टरी का अस्तित्व जाँचें
    if _, err := os.Stat(path); os.IsNotExist(err) {
        return false
    }
    return true
}
```

उपरोक्त कोड को चलाने के लिए, आपको इसे अपने प्रोजेक्ट में `main.go` नाम से सेव करके रन करना होगा। ध्यान दें कि यदि "test" नाम का डायरेक्टरी मौजूद होता है, तो प्रिंट आउट में "यह डायरेक्टरी मौजूद है!" आना चाहिए।
