## क्यों

किसी भी डायरेक्टरी की जांच करना आवश्यक हो सकता है, क्योंकि यह आपको किसी भी फ़ाइल या फ़ोल्डर को प्रोग्राम के साथ संबंधित करने की अनुमति देता है। यह आपको प्रोग्राम को संभालने और सुरक्षित रखने में मदद कर सकता है।

## कैसे करें

आप अपने गो कोड में निम्न उदाहरण का पालन करके एक डायरेक्टरी की जांच कर सकते हैं:

```Go
package main

import (
    "fmt"
    "os"
)

func main() {
    directoryName := "./myDirectory"

    if _, err := os.Stat(directoryName); os.IsNotExist(err) {
        fmt.Println("यह डायरेक्टरी मौजूद नहीं है।")
    } else {
        fmt.Println("यह डायरेक्टरी मौजूद है।")
    }
}
```

उपरोक्त कोड में, हम `os` पैकेज का उपयोग कर रहे हैं जो डायरेक्टरी की जांच करने के लिए आवश्यक फ़ंक्शन प्रदान करता है। हम उस फ़ंक्शन को `os.Stat()` के रूप में कॉल करते हैं और डायरेक्टरी का नाम देते हैं। फिर हम दिया गया त्रुटि को चेक करते हैं और `os.IsNotExist()` फ़ंक्शन का उपयोग करते हैं जो यदि त्रुटि मौजूद होती है तो `true` रिटर्न करता है। यदि त्रुटि मौजूद नहीं होती है, तो डायरेक्टरी मौजूद होने की जांच की जाती है।

## गहराई में जाएं

डायरेक्टरी की जांच करने के लिए `os` पैकेज में कई अन्य उपयोगी फ़ंक्शन हैं जैसे `os.Lstat()` और `os.ReadDir()`। कई बार हमें अपने कोड में से वर्तमान डायरेक्टरी के साथ संबंधित सभी फ़ाइलों और फ़
