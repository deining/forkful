## इसलिए

आप अपनी गो ऐप्स को बनाने के लिए कमांड लाइन अर्ग्यूमेंट्स को पढने चाहते हैं क्योंकि यह आपको अपनी एप्लिकेशन को कस्टमाइज़ करके प्रोग्राम को संचालित करने की अनुमति देता है। इसके साथ ही, यह आपके गो प्रोग्राम को और उपयोगी बनाता है ताकि आप उसे अपने उपयोगकर्ताओं के लिए बेहतर बना सकें।

## कैसे करें

आइए अब हम देखें कि कैसे आप गो में कमांड लाइन अर्ग्यूमेंट्स को पढ़ सकते हैं। सबसे पहले, आपको `os` पैकेज को इम्पोर्ट करना होगा। फिर आप `os.Args` का उपयोग करके उपलब्ध अर्ग्यूमेंट्स को प्राप्त कर सकते हैं। यहां हम उन अर्ग्यूमेंट्स को एक स्ट्रिंग स्लाइस में स्टोर करेंगे और उन्हें प्रिंट करेंगे।

```Go
package main

import "fmt"
import "os"

func main() {
    arguments := os.Args
    fmt.Println(arguments)
}
```

इसका आउटपुट कुछ इस तरह होगा:

```
$ go run main.go hello world
[main hello world]
```

इस तरह से आप कमांड लाइन अर्ग्यूमेंट्स को पढ़ सकते हैं और उन्हें अपनी गो ऐप्स में उपयोग कर सकते हैं।
