## क्यों

कमांड लाइन आर्ग्यूमेंट्स को पढ़ने के लिए क्यों लोग इसका अध्ययन करने में जुटते हैं, इसकी कुछ मुख्य वजह है कि ये गो प्रोग्रामिंग भाषा में एक महत्वपूर्ण हिस्सा हैं और इसकी जानकारी आपको अपने सॉफ्टवेयर को बेहतर बनाने में मदद कर सकती है।

## कैसे करे

```Go
package main

import (
	"fmt"
	"os"
)

func main() {
	args := os.Args
	for i, arg := range args {
		fmt.Printf("आर्गुमेंट %d: %s\n", i, arg)
	}
}
```

उपरोक्त कोड स्निपेट में हमने `os` पैकेज का उपयोग करके कमांड लाइन आर्ग्युमेंट्स को पढ़ा है और उनको अपने स्क्रिप्ट में छपा दिया है। उत्पादन या समान आर्ग्युमेंट्स को `os.Args` वेरिएबल में रखा जाता है और इसे रेंज में जोड़े जाने के लिए `range` का उपयोग किया गया है। 

### आउटपुट

गो कैस्स कमांड के साथ नीचे दिए गई आउटपुट मिलेगी:

```
आर्गुमेंट 0: गो
आर्गुमेंट 1: run
आर्गुमेंट 2: main.go
```

## गहरा अध्ययन

कमांड लाइन आर्गुमेंट्स के बारे में गहराई से जानकारी हासिल करने के लिए आप गो के `flag` पैकेज का भी उपयोग कर सकते हैं। यह आपको पावरफुल और अत्यधिक विशेष फ़्लैग विधियों को दर्शाने का विकल्प देता है जो आपको अपने सॉफ्टवेयर में उपयोग कर सकते हैं। इस पैकेज को सीखने के लिए आप इस [अध्ययन ट्यूटोरियल](https://gobyexample.com/command-line-flags) का उपय
