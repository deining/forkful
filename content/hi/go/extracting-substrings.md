जो कारण: 
इस ब्लॉग पोस्ट में हम आपको बताएंगे कि Go प्रोग्रामिंग में स्ट्रिंग्स से सबस्ट्रिंग्स कैसे निकालें। इससे आपको अपनी कोडिंग को और अधिक आसान और उन्नत बनाने में मदद मिलेगी।

कैसे करें: 
```Go 
package main 

import ( 
"fmt" 
"strings" 
) 

func main() { 
s := "यह एक उदाहरण है।" 
sub := strings.TrimPrefix(s, "एक") 
fmt.Println(sub) 
} 
```
आपको परिणाम में "उदाहरण है। "मिलेगा। इस उदाहरण में हमने strings.TrimPrefix () फंक्शन का उपयोग किया है जो हमारी सबस्ट्रिंग निकालने में मदद करता है। आप अपने द्वारा उपयोग करने और उसे अपनी आवश्यकताओं के अनुसार समायोजित कर सकते हैं।

गहराई में खुदाई करें:
सबस्ट्रिंग्स को निकालने का एक और तरीका है strings.Split () फंक्शन का उपयोग करना। यह फंक्शन स्ट्रिंग को विभाजित करके एक स्लाइस बनाता है, जो आप फिर से अपनी मर्ज करके सबस्ट्रिंग को प्राप्त कर सकते हैं। आप strings.Replace (), strings.Count () और अन्य स्ट्रिंग्स से सम्बंधित फंक्शनों का उपयोग करके अपने कोड में और भी मजेदार चीजें कर सकते हैं।
