## क्यों

चलिए हम आज Go प्रोग्रामिंग में शब्दों को जोड़ने के तरीके के बारे में बात करें। शब्दों को जोड़ने का आसान और आवश्यक तरीका है जो हर एक Go डेवलपर को जानना चाहिए।

## कैसे करें

जब हम अलग-अलग स्ट्रिंग्स को आपस में जोड़ते हैं, तो हम इसे शब्दों का धागा (string concatenation) कहते हैं। स्ट्रिंग का धागा करने के लिए Go में `+` ऑपरेटर का उपयोग किया जाता है। यह बहुत आसान है। चलिए एक उदाहरण देखते हैं।

```Go
package main

import "fmt"

func main() {
    str1 := "Hello"
    str2 := "World"

    result := str1 + " " + str2

    fmt.Println(result)
}
```

उपरोक्त कोड को रन करने पर हमारे स्क्रीन पर "Hello World" लिखा हुआ दिखेगा। जैसा कि आप देख सकते हैं, हमने `+` ऑपरेटर का उपयोग करके दो स्ट्रिंग्स को जोड़ा है। आप चाहे तो अधिक से अधिक स्ट्रिंग्स को भी जोड़ सकते हैं।

## गहराई में जाएं

अभी तक हमने बस दो स्ट्रिंग्स को जोड़ने की बात की है, लेकिन हम जानते हैं कि अक्सर एक स्ट्रिंग्स में विभिन्न तरह की डेटा होता है। Go में हम `fmt.Sprintf()` फ़ंक्शन का उपयोग करके स्पेशल स्ट्रिंग फॉर्मेटिंग कर सकते हैं। चलिए एक उदाहरण देखते हैं।

```Go
package main

import "fmt"

func main() {
    name := "John"
    age := 27

    result := fmt.Sprintf("%s is %d years old.", name, age)

    fmt.Println(result)
}
```

उपरोक्त कोड को रन करने पर हमारे स्क्रीन पर "John is 27 years old." लिखा हुआ दिखेगा। हमने `%s` उपयोग करके वेरिएबल `name`
