## क्यों

जब हम गो प्रोग्रामिंग में कोई स्ट्रिंग लोअर केस में बदलना चाहते हैं, तो इसका क्या उपयोग हो सकता है? इस ब्लॉग पोस्ट में हम इस सवाल का उत्तर देंगे और सिखाएंगे कि कैसे हम गो प्रोग्रामिंग में स्ट्रिंग लोअर केस में बदल सकते हैं। 

## कैसे करें

जानना कि स्ट्रिंग को लोअर केस में बदलने के लिए हमें कौन से गो फंक्शन का उपयोग करना चाहिए, उपयोगी हो सकता है। नीचे हमने कुछ उदाहरण सहित कोडिंग के द्वारा स्ट्रिंग लोअर केस में बदलने का तरीका दिखाया है। 

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	// स्ट्रिंग लोअर केस में बदलने के लिए strings.ToLower() फंक्शन का उपयोग करें
	str := "GO प्रोग्रामिंग"
	fmt.Println("पहली स्ट्रिंग:", str)

	lowerStr := strings.ToLower(str)
	fmt.Println("लोअर केस स्ट्रिंग:", lowerStr)

	// स्ट्रिंग के अक्षरों को runes में कन्वर्ट करके लोअर केस में बदल सकते हैं
	str2 := "अनुभवी प्रोग्रामर"
	fmt.Println("पहली स्ट्रिंग:", str2)

	runes := []rune(strings.ToLower(str2))
	fmt.Println("लोअर केस स्ट्रिंग:", string(runes))
}
```

```
लोअर केस स्ट्रिंग: अनुभवी प्रोग्रामर
```
