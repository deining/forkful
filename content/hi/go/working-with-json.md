## क्यों

जेएसओएन का उपयोग करके डेटा अनुरोधों को संगठित करने और एक सरल, हाइरार्कीय और क्रोस प्लेटफॉर्म डेटा एक्सचेंज के साथ आसानी से निकास संभव होता है। जिससे कि इससे आपके निर्देशांक और एप्लिकेशन में डेटा को आसानी से शेयर किया जा सके।

## कैसे

```

मानदंड , एरर := जेएसओएन . अनमार्शल (जोनबाइट्स)
अगर एरर != निल {
अनादि ( "अमार्शल त्रुटि:" , एरर )
} अन्य {
मुद्रण (मानदंड[ "नाम" ].(स्ट्रिंग), मानदंड[ "उम्र" ].(इंट))
}
```

आपको ऊपर दिए गए गो कोड ब्लॉक में से एक सामान्य जावा स्क्रिप्ट को देखना चाहिए जिसमें एक जेएसओएन बाइट रेखा उदाहरण दिया गया है जो "नाम" और "उम्र" क्षेत्रों को संरचित करता है जो फिर से मानदंड में अनमार्शल किए जाते हैं। यहां, हम जावा स्क्रिप्ट में निर्दिष्ट किया गए "नाम" और "उम्र" को मुद्रित करेंगे।
