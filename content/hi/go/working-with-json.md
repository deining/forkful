## क्यों

क्या आप जासन प्रसंगों के साथ काम करने में शामिल होने के बारे में सोच रहे हैं? जासन एक लोकप्रिय डेटा इंटरचेंज फॉर्मेट है जो डेवलपर्स को एपीआई कॉल करने के लिए उपयोग किया जाता है। यह जासन अत्यधिक प्रभावशाली और आसानी से समझने और उपयोग करने के लिए है। इसलिए, यदि आप डेटा इंटरचेंज प्रोग्रामिंग को सीखना चाहते हैं, तो जासन एक अच्छा शुरुआत हो सकता है।

## कैसे करें

कैसे कई भाषाओं में ही उपलब्ध जासन का उपयोग करके आसानी से डेटा विनिमय किया जा सकता है। गो भाषा में भी आप जासन का उपयोग कर सकते हैं। नीचे एक उदाहरण दिया गया है:

```Go
// जासन का उपयोग डेटा को अन्य संरचित डेटा में परिवर्तित करने के लिए।
package main

import (
    "encoding/json"
    "fmt"
)

type Person struct {
    Name string `json:"name"` // "name" फील्ड के लिए अलाइस नाम
    Age  int    `json:"age"`  // "age" फील्ड के लिए अलाइस नाम
}

func main() {
    // JSON से द्वारा स्ट्रिंग प्राप्त करें।
    jsonStr := `{"name":"John Doe","age":30}`

    // जासन डेकोडर बनाएं।
    var person Person
    json.Unmarshal([]byte(jsonStr), &person)

    // जासन से डेटा लें और कंसोल पर प्रिंट करें।
    fmt.Printf("नाम: %s आयु: %d", person.Name, person.Age)
}

// Output: नाम: John Doe आयु: 30
```
