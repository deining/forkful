## वजह
क्योंकि टेक्स्ट फ़ाइलों को पढ़ने और समझने कोई मामूली काम नहीं है। यह एक आवश्यक कौशल है जो गो प्रोग्रामिंग में सफलता प्राप्त करने के लिए आवश्यक है।

## कैसे करें
अगर आपने अभी तक गो प्रोग्रामिंग में टेक्स्ट फ़ाइलों को पढ़ना नहीं सीखा है, तो चिंता न करें। हम आपको इस महत्वपूर्ण कौशल को सीखने के लिए संबंधित कोड उदाहरणों के साथ मदद करेंगे। निम्नलिखित कोड ब्लॉक में आपको सांख्यिकी गणना करने के लिए एक टेक्स्ट फ़ाइल से डेटा पढ़ना सिखाया जाएगा।

```Go
package main

import (
	"fmt"
	"os"
	"io/ioutil"
	"strings"
	"strconv"
)

func main() {
	data, err := ioutil.ReadFile("data.txt")
	if err != nil {
		fmt.Println("माफ़ी चाहता हूँ, फ़ाइल पढ़ने में समस्या हुई।")
		os.Exit(1)
	}

	lines := strings.Split(string(data), "\n")

	total := 0
	for _, line := range lines {
		num, _ := strconv.Atoi(line)
		total += num
	}

	fmt.Println("टेक्स्ट फ़ाइल के डेटा से सम संख्या:", total)
}
```

जैसा कि आप उपरोक्त कोड में देख सकते हैं, हमने `ioutil` पैकेज का उपयोग करके टेक्स्ट फ़ाइल से डेटा पढ़ना सीखा है। हमने `strings` पैकेज का भी उपयोग करके एक लाइन से दूसरे लाइन में डेटा को बांटा है, और फिर उस डेटा को `strconv` पैकेज का उपयोग करके अंक में बदला है। आप भी उपरोक्त कोड में बदलाव करके टेक्स्ट फ़ाइल से अपनी पसंद अनुसार डेटा पढ़ सकते हैं।
