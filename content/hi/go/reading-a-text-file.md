# इसलिए

टेक्स्ट फ़ाइल पढ़ने का काम आसान हो सकता है, लेकिन गो प्रोग्रामिंग भाषा में क्यूँवक़ी इसे करना सीखने का अवसर है। इस ब्लॉग पोस्ट में, हम टेक्स्ट फ़ाइल को कैसे पढ़ा जाता है, और इसके क्या फायदे हो सकते हैं, उसके बारे में चर्चा करेंगे।

## कैसे करें

टेक्स्ट फ़ाइल को पढ़ने के लिए, हमें एक `os` पैकेज का उपयोग करना होगा। निम्नलिखित गो प्रोग्राम द्वारा, हम एक इसी `os` पैकेज के भीतर कोड को दर्शाते हैं:

```Go
package main

import (
	"fmt"
	"io/ioutil"
	"os"
)

func main() {
	file, err := os.Open("textfile.txt")
	if err != nil {
		panic(err)
	}

	defer file.Close()

	data, err := ioutil.ReadAll(file)
	if err != nil {
		panic(err)
	}

	fmt.Println(string(data))
}
```
इस कोड में, हम सबसे पहले एक `os` पैकेज को `import` करते हैं। फिर हम `os.Open()` फ़ंक्शन को उस टेक्स्ट फ़ाइल के साथ बुलंद करते हैं जिसे हम पढ़ना चाहते हैं। अगर कुछ गलतियाँ होती हैं, तो हम `panic` फ़ंक्शन का उपयोग करते हैं जो उन्हें हमेशा के लिए हमला करने के लिए बदल देता है। इसके बाद, हम `file.Close()` को बुलाते हैं जिससे फ़ाइल को बंद करना दरवाज़ा जी जाता है। और अंत में, हम `ioutil.ReadAll()` फ़ंक्शन को उस फ़ाइल की सारी जानकारी प्राप्त करने के लिए बुलाते हैं और उसे `string` मान के साथ प्रिंट करते हैं।

अगर हमारे पास एक `textfile.txt` नाम की एक टेक्स्ट फ़ाइल होती है, तो ऊपर दिए गए कोड का रिज़ल्ट निम्नल
