## क्यों

टेक्स्ट फ़ाइल को लिखने और पढ़ने के लिए अनेक कारण हो सकते हैं। यह एक व्यापक डेटा संग्रहक युक्ति है जो आपको आपकी प्रोजेक्ट के साथ सहयोग करती है और इससे आपको कोडिंग का समय भी बचता है। इस लेख के माध्यम से हम Go में टेक्स्ट फ़ाइल को कैसे लिख सकते हैं और उसमें डेटा को कैसे ऐड कर सकते हैं, इसके बारे में जानेंगे।

## कैसे करें

आप एक टेक्स्ट फ़ाइल को Go में आसानी से लिख सकते हैं। सबसे पहले, आपको `os` पैकेज को `import` करना होगा। फिर, हमें फ़ाइल को ओपन करना होगा और इसे लिखने के लिए उपयुक्त `WriteString` फ़ंक्शन का उपयोग करना होगा। नीचे दिए गए कोड स्निपेट में आप इस प्रक्रिया को देख सकते हैं।

```Go
package main

import (
  "fmt"
  "os"
)

func main() {
  file, err := os.Open("test.txt")
  if err != nil {
    fmt.Println("Error in opening file:", err)
  }
  defer file.Close()
  data := "This is a sample text."
  _, err = file.WriteString(data)
  if err != nil {
    fmt.Println("Error in writing to file:", err)
  }
}
```

अपने फ़ाइल में डेटा को डालने के लिए, आप `WriteString` फ़ंक्शन को उस डाटा के साथ ओपन की गई फ़ाइल के साथ कॉल कर सकते हैं।

## गहराई में जायें

टेक्स्ट फ़ाइल को लिखने के साथ-साथ, आप उसे पढ़ने और रीड और व्राइट मेथड को शामिल कर सकते हैं। आप अपनी फ़ाइल के साथ कई सारे ऑपरेशंस कर सकते हैं जैसे कि लाइनों को स्वत: बदलना, उसमें से डेटा पढ़ना या उसे अन्य फ़ाइल में कॉपी करना
