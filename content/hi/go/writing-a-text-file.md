## क्या और क्यों?

एक पाठ फ़ाइल लिखना क्या है? यह एक पाठ फ़ाइल है जो लेखन या लोड करने के लिए उपयोग की जाती है। क्या आपको पता है कि आप अपने प्रोग्रामिंग कोड को सुरक्षित रखने के लिए इसे कैसे उपयोग कर सकते हैं?

## कैसे करें:

```Go
file, err := os.Create("sample.txt") // सम्पल टेक्स्ट फ़ाइल बनाएं
if err != nil {
  fmt.Println(err)
  return
}
defer file.Close() // फाइल बंद करने के लिए डिफ़र स्टेटमेंट का उपयोग करें
text := "मेरा पहला पाठ फ़ाइल"
_, err = file.WriteString(text) // पाठ फ़ाइल में पाठ लिखें
if err != nil {
  fmt.Println(err)
  return
}
```

इसके बाद, आप उपयोगकर्ता से एक पाठ फ़ाइल के नाम लेकर उसे प्रिंट कर सकते हैं:

```Go
fmt.Println("रचित फ़ाइल का नाम:", file.Name())
```

## गहराई में जाएं:

पहली गो संस्करण में, उपयोगकर्ता को फाइल बनाने और संपादित करने के लिए इस्तेमाल करने को केवल फ़ाइल नाम की आवश्यकता थी। इसके पूर्व, उपयोगकर्ता को स्वयं एक नया पाठ फ़ाइल बनाने के लिए प्रोग्राम को लख करना होता था। दूसरे विकल्प के रूप में, अन्य पुस्तिकायों में आप पाठ फ़ाइलों तक पहुंच के लिए भी शैली को देख सकते हैं।
