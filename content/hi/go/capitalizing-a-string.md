## खोज: 

कभी भी हमें कोई विशेष तरीके से शब्दों को समित करने की आवश्यकता पड़ सकती है, और गो प्रोग्रामिंग में ये काम अपनी भाषा से किया जा सकता है। इस ब्लॉग पोस्ट में, हम शब्दों को समित करने के तरीके पर चर्चा करेंगे। 

## कैसे करें: 

लिखने या प्रिंट करने के दौरान शब्दों के प्रथम अक्षर को बड़ा करने के लिए, हम अपने गो कोड में `strings.ToUpper()` फंक्शन का प्रयोग कर सकते हैं। इसी तरह, `strings.ToLower()` फंक्शन का प्रयोग करके हम शब्दों के प्रथम अक्षर को छोटा कर सकते हैं। इसके अलावा, गो में `strings.Title()` फंक्शन का प्रयोग करके हम प्रत्येक शब्द के पहले अक्षर को बड़ा कर सकते हैं। 

इस तरह: 
```Go 
package main 

import ( 
	"fmt" 
	"strings" 
) 

func main() { 
	// कैपिटलाइज़िंग स्ट्रिंग 
	sampleString := "गो प्रोग्रामिंग" 
	fmt.Println("साम्पल स्ट्रिंग:", sampleString) 
	fmt.Println("अपरकेस स्ट्रिंग:", strings.ToUpper(sampleString)) 
	fmt.Println("लोअरकेस स्ट्रिंग:", strings.ToLower(sampleString)) 
	fmt.Println("टाइटल केस स्ट्रिंग:", strings.Title(sampleString)) 
} 
```
आउटपुट: 
```
साम्पल स्ट्रिंग: गो प्रोग्रामिंग 
अपरकेस स्ट्रिंग: गो प्रोग्रामिंग 
लोअरकेस स्ट्रिंग: गो प्रोग्रामिंग 
टाइटल केस स्ट्रिंग: गो प्रोग्रामिंग 
```

## गहराई में जाने:

शब्दों को समित करना एक आम और उपयोगी काम है जो गो प्रोग्रामिंग में आसानी से किया जा सकता है। शायद आपने इस्तेमाल करने के अलावा यह यकीनन नहीं जाना होगा कि `strings` पैकेज में
