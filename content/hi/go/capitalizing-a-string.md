हाय फ्रेंड्स! आज हम आपको गो प्रोग्रामिंग के बारे में बताएंगे। जो लोग प्रोग्रामिंग में नए हैं वे गो को जानना चाहते हैं कि कोडिंग में कैसे काम करना है। आज का हमारा टॉपिक है "string को capitalize करना।" इसमें हम आपको ये बताएंगे कि आप फिर क्यों लोग अपने स्ट्रिंग को capitalize करना चाहते हैं, और उसके बाद हम आपको इसे कैसे करेंगे। तो चलिए शुरू करते हैं।

## इसका कारण क्या है?

स्ट्रिंग को capitalize करने का कारण यह है कि कई बार हमेरे पास एक string होता है जो कि lowercase में है और हम उसे UPPERCASE में या फिर पहले capital letter के साथ चाहते हैं। यह आमतौर पर उपयोगकर्ता को दिखाने के लिए बेहतर होता है या फिर डेटा को सॉर्ट करने में आसानी होती है।

## इसे कैसे करें?

इसे करने के बहुत से तरीके हैं, लेकिन हम आपको एक सरल तरीका दिखाएंगे। चलिए नीचे दिए गए कोड को देखें।

```Go
package main

import (
	"fmt"
	"strings"
)

func main() {
	s := "hello world"
	fmt.Println(strings.Title(s))
}
```
Output:
Hello World

जैसा कि आप देख सकते हैं, हमने सिर्फ "strings" पैकेज का उपयोग किया है और "Title" फंक्शन को स्ट्रिंग के साथ बुलाया है। इससे आपको अपने स्ट्रिंग को uppercase में मिल जाएगा।
