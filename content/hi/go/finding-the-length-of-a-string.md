## क्यों

गो प्रोग्रामिंग में लिखे गए स्ट्रिंग की लंबाई निकालना एक अहम् होता है। इस समस्या का समाधान आपको अपने कोड को अधिक उपयोगी बना सकता है और आपको समस्याओं को हल करने में मदद कर सकता है। इस ब्लॉग पोस्ट में, हम गो प्रोग्रामिंग के माध्यम से स्ट्रिंग की लंबाई निकालने के बारे में विस्तार से जानेंगे।

## कैसे करें

कोडिंग के उदाहरण के साथ गो प्रोग्रामिंग में स्ट्रिंग की लंबाई निकालने का प्रस्ताव । आइए हम एक उदाहरण के साथ देखें कि हम कैसे इसे कोड कर सकते हैं।

```Go
package main

import "fmt"

func main() {
    str := "गो प्रोग्रामिंग"
    length := len(str)
    fmt.Println("स्ट्रिंग की लंबाई है:", length)
}
```

आउटपुट:
```
स्ट्रिंग की लंबाई है: 13
```

## गहराई से जाएं

स्ट्रिंग की लंबाई को निकालने के लिए, गो प्रोग्रामिंग में `len()` फ़ंक्शन का उपयोग किया जाता है। इसके अलावा, गो में अनुक्रमिक शब्दों की लंबाई को निकालने के लिए `unicode/utf8` पैकेज भी उपलब्ध है। इसके अलावा, आप अपने कोड में स्ट्रिंग प्रकार का उपयोग करके भी स्ट्रिंग की लंबाई प्राप्त कर सकते हैं।
