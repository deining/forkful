## क्यों
तारीख को स्ट्रिंग में रूपांतरित करने का कोई प्रमुख कारण है कि हम अपने डेटा को और सुविधाजनक और सुंदर ढंग से प्रदर्शित कर सके।

## कैसे करें
```Go
package main

import (
	"fmt"
	"time"
)

func main() {
	t := time.Now()
	date := t.Format("02 Jan 2006")
	fmt.Println("तारीख:", date)
}
```

आउटपुट:
> तारीख: १६ सितम्बर २०२०

## गहराई में जाएं
दिनांक को स्ट्रिंग में रूपांतरित करने का प्रक्रिया वास्तव में सरल है। यह हमारे डेटा को प्रदर्शित करने के साथ-साथ अन्य कई संगणकों के साथ संभाव्य भविष्यवाणियां भी बनाए रखता है। हम अपने कोड में अपनी आवश्यकताओं के अनुसार तारीख को स्ट्रिंग में रूपांतरित कर सकते हैं।
