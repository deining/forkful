## क्यों

गो प्रोग्रामिंग भाषा एक मॉडर्न और प्रभावी तरीके से स्केलेबल एप्लिकेशन बनाने का एक उपकारी उपकरण है। यह स्थानीय डेस्कटॉप, वेब और मोबाइल एप्लिकेशन बनाने के लिए उपयोगी है। गो का स्केलेबिलिटी, उच्च प्रदर्शन और सुरक्षा के कारण यह सभी प्रोफेशनल विकासकर्ताओं के लिए एक जरूरी भाषा है।

## कैसे

गो प्रोग्रामिंग भाषा में एक नया प्रोजेक्ट शुरू करने के लिए आपको पहले गो के साथ अपने सिस्टम में सुनिश्चित करना होगा। अगर आपने अभी तक गो इंस्टॉल नहीं किया है, तो आप [[https://golang.org/][यहां]] से गो को डाउनलोड और इंस्टॉल कर सकते हैं। फिर, आपको [[https://golang.org/doc/code.html][यहां]] से गो कोड एक्सपीरियंस को समझने के लिए रुचि रखनी चाहिए।

गो कोडिंग का एक उदाहरणः
```Go
package main

import "fmt"

func main() {
    fmt.Println("नमस्ते, दुनिया!")
}
```

गो कोम्पाइल करने और उसके बाद एक्जीक्यूट करने के लिए, आपको निम्न आदेश का पालन करना होगा।

```Shell
go build
./आपका_प्रोजेक्ट_नाम
```

गो कोड खोजने और समझने के लिए आप [[https://golang.org/pkg/][इस डॉक्यूमेंटेशन]] से मदद ले सकते हैं।
