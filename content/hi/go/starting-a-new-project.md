Hindi में Go प्रोग्रामिंग के लिए एक आम ब्लॉग पोस्ट

## क्यों

क्या आपने कभी सोचा है कि आपने एक नया प्रोजेक्ट शुरू करने की क्यों चुनौती स्वीकार की है? आमतौर पर, उन्हें आलोचना या सुझाव देने वाले लोग की सलाह के साथ आपको जवाब देने के लिए एक नया प्रोजेक्ट शुरू करने की जरूरत होती है। लेकिन यह एक बहुत ही बेहतरीन अनुभव हो सकता है जो आपको आपके कौशल को बढ़ाने की अनुमति देता है और साथ ही आपको नए तकनीकों को सीखने के लिए प्रोत्साहित करता है।

## कैसे

मुख्य Go प्रोग्रामिंग भाषा पराकाष्ठा को प्राप्त करने के लिए हमें ```Go ... ``` कोड ब्लॉक का उपयोग करना पड़ता है। इस समझने के लिए देखें कि एक साधारण उदाहरण कैसा हो सकता है।

```Go 
प्रमाणित करें कि सभी संख्याओं को 2 से गुणा करें  
func सभी 2 सामग्रीय संखाया([] int) [] int {
  साप्ताहिक:= 2
  var को = make([] int, len(साप्ताहिकओप़सकां))
  को i, v := रांग साप्ताहिक {
    को[i] = v * साप्ताहिक
  }
  वापस कर दो
}

func मुख्य() {
  साप्ताहिक := [] int {4, 8, 12, 16, 20}
  2 सामग्रीय संखाया(साप्ताहिक)
}
```

इस छोटे से उदाहरण से, आपको समझ में आना चाहिए कि कैसे आप ``` ```Go कोड ब्लॉक का उपयोग कर सकते हो और किस प्रकार अपने कोड को सुंदर और साफ बना सकते हो।
