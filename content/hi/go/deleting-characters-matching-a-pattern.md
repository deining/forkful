## क्यों

करोड़ों लोग अपने आपगार सॉफ्टवेयर बनाने के लिए गो भाषा का उपयोग करते हैं। गो भाषा उन्हें कम समय में स्केलेबल, सुरक्षित और दुरुस्त सॉफ्टवेयर बनाने की अनुमति देती है। इसलिए, उन्हें अपनी कोडिंग का ध्यान रखने के लिए इस भाषा का उपयोग करना बहुत लाभदायक होता है।

## कैसे करें

आप अपने कोड में एक पैटर्न के मिलते जुलते अक्षरों को हटाना चाहते हैं तो आप गो भाषा के strings पैकेज का उपयोग कर सकते हैं। इस पैकेज में ReplaceAllString () फंक्शन होता है जो आपको वहां से सभी मिलते जुलते अक्षरों को हटाने में मदद करता है। नीचे दिए गए कोड ब्लॉक में आप इसका उदाहरण देख सकते हैं:

```Go
package main

import (
	"fmt"
	"regexp"
)

func main() {
	// पैटर्न दर्ज करें
	pattern := "[aeiou]+"
	// स्ट्रिंग दर्ज करें
	str := "hello world"
	// नया स्ट्रिंग जो मिलते जुलते अक्षरों को हटाएगा
	newstr := regexp.MustCompile(pattern).ReplaceAllString(str, "")
	// बाहरवाले स्ट्रिंग को प्रिंट करें
	fmt.Println(newstr) // hll wrld
}
```
