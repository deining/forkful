## क्यों

कभी-कभी हमें अपनी तारीखों या अन्य तिथियों को परिवर्तन करने की आवश्यकता होती है। शायद हम किसी समारोह को निर्दिष्ट तारीख पर ठहराना चाहते हैं या फिर कोई अन्य उद्देश्य हो सकता है। इस ब्लॉग पोस्ट में, हम जानेंगे कि गो प्रोग्रामिंग में इस प्रकार की तारीख को कैसे निर्धारित किया जा सकता है।

## कैसे करें

गो प्रोग्रामिंग में, हम तारीखों और समय को `time` और `time.Time` पैकेज के माध्यम से प्रबंधित कर सकते हैं। यहां हम तारीख को आगे या पीछे जाने के लिए कुछ उदाहरण प्रस्तुत कर रहे हैं।

```Go
package main

import (
    "fmt"
    "time"
)

func main() {
    // वर्तमान तारीख को प्राप्त करें
    today := time.Now()

    // 10 दिन बाद का तारीख
    future := today.AddDate(0, 0, 10)
    fmt.Println("10 दिन बाद का तारीख:", future.Format("02-01-2006"))

    // 2 वर्ष पहले का तारीख
    past := today.AddDate(-2, 0, 0)
    fmt.Println("2 वर्ष पहले का तारीख:", past.Format("02-01-2006"))
}
```

इस कोड का आउटपुट है:

```
10 दिन बाद का तारीख: 29-11-2020
2 वर्ष पहले का तारीख: 01-12-2018
```

इस उदाहरण में, हमने `AddDate()` फ़ंक्शन का उपयोग किया है जो हमें नया तारीख देगा जो कि वर्तमान तारीख से निर्धारित संभावित अंतरों को शामिल करता है। हमने तारीख को `Format()` फ़ंक्शन के माध्यम से दिनांक के रूप में प्रारूपित भी किया है।
