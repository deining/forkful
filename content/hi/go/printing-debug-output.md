## क्यों

डीबग आउटपुट प्रिंट करने के पीछे की वजह अपने कोड में त्रुटियों को ठीक करने के लिए है। यह प्रक्रिया आपको अपने कोड को समझने और सुधारने में मदद करती है।

## कैसे करें

आप डीबग आउटपुट प्रिंट करने के लिए `fmt` पैकेज का उपयोग कर सकते हैं। नीचे दिए गए कुछ उदाहरण आपको समझने में मदद करेंगे।

```Go
fmt.Println("हैलो, विश्व!")
// आउटपुट: हैलो, विश्व!
```

```Go
name := "आकाश"
fmt.Printf("मेरा नाम %s है।", name)
// आउटपुट: मेरा नाम आकाश है।
```

## गहराई में जाएँ

डीबग आउटपुट को सही ढंग से प्रिंट करने के लिए आप फॉर्मेट के साथ स्ट्रिंग और वेरिएबल्स को पास कर सकते हैं। आप इसमें वैधानिक त्रुटियां भी देख सकते हैं जो कोड को सुधारने में मदद करती हैं। आप अपने कोड शुरू करने से पहले `go fmt` कमांड का उपयोग करके इसे स्थिर और स्विचित कर सकते हैं।
