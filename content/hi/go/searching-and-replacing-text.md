## आत्मविश्वास के साथ अपने टेक्स्ट को सर्च और रिप्लेस करना के लिए मिलता हैWhy: क्या आपको कभी भी एक डॉक्यूमेंट या कोड फ़ाइल में गलत शब्द ठीक नहीं दिखाई दिया है? अगर हाँ, तो सर्च और रिप्लेस आपकी समस्या को हल कर सकता है। इसके साथ ही, आप कई लाइनों को हाईलाइट और बदल सकते हैं जो टाइप करने या खोजने में थोड़ा समय लेते हैं।

## कैसे करें: उदाहरण और नमूना उत्पादन के साथ कोड ब्लॉक में "```Go ... ```" कोड के साथHow To: टेक्स्ट सर्च और रिप्लेसमेंट करने के लिए Go का `strings.Replace()` फंक्शन उपयोग किया जाता है। निम्नलिखित है एक उदाहरण जिसमें आप स्ट्रिंग `Hello World` के `World` को `Universe` से बदल सकते हैं:

```Go
package main

import (
  "fmt"
  "strings"
)

func main() {
  text := "Hello World"
  newText := strings.Replace(text, "World", "Universe", 1)
  fmt.Println(newText)
}
```

आउटपुट: `Hello Universe`

इस उदाहरण में हमने `strings.Replace()` के साथ स्ट्रिंग, खोजने वाला स्ट्रिंग और रिप्लेस करने वाली स्ट्रिंग का उपयोग किया है। हमने `Universe` को से केवल 1 बार बदला है, लेकिन अगर हम `strings.Replace()` के चौथे पैरामीटर को `-1` कर दें, तो वह सभी मैचिंग स्ट्रिंग्स को बदल देगा।
