## क्यों

क्या आपने कभी सोचा है कि आप अपने कंप्यूटर के साथ और अधिक इंटरैक्टिव हो सकते हैं? क्या आप अपने कंप्यूटर के लिए और ताकतवर टूल्स ढूंढ रहे हैं? अगर हाँ, तो आपको कमांड लाइन एर्ग्यूमेंट्स पढ़ना सीखना चाहिए! यह ब्लॉग पोस्ट वह सभी तरह के प्रोग्रामर्स के लिए है जो फिश शेल को सीखने और उसका उपयोग करने में रूचि रखते हैं।

## कैसे करें

जब आप किसी फाइल को कंपाइल या इंस्टॉल करने के लिए एक एक्जिक्यूशन फाइल निर्दिष्ट करते हैं, तो आपके द्वारा दिए गए या आर्गुमेंट्स पढी जाती हैं जिसमें प्रोग्राम के लिए क्या प्रविष्टियों की जानकारी पढ़ाई जाएगी। यहां हम आपको बताएंगे कि कैसे आपको अपने फिश शेल स्रोत कोड के साथ मिली जानकारी को पढ़ना है।

```
Fish Shell प्रोग्राम को पढ़ने के लिए आप चाहते हैं।
set -l option_count (count $argv)
echo "आपके पास $option_count आर्गुमेंट्स हैं।"
```

आपके कमांड लाइन आर्गुमेंट्स को पढ़ने में समस्या हो सकती है। लेकिन यह प्रोग्रामिंग भाषा में संभव है, यह मदद करेगा कि आपको दिए गए आर्गुमेंट्स को संगठित तरीके से संग्रहीत करें और उनका उपयोग करें।
