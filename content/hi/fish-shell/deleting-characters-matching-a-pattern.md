## लक्ष्य

आपने कभी-कभी समझा होगा कि आपको कुछ वर्ण हटाने की जरूरत होती है जो एक निश्चित पैटर्न से मिलते हैं। यह पैटर्न आपको एक बड़ी फ़ाइल में ढूंढने या एक वर्ण शब्द के सभी अवसरों को हटाने के लिए उपयोगी होता है। इस लेख में, हम आपको बताएंगे कि फ़िश शेल (वर्तमान संस्करण) में वर्ण हटाने के लिए कैसे कोड किया जाता है।

## कैसे करें

चंद उदाहरणों के साथ कैसे कोड करें: 

```Fish Shell
# पूरी तरह से का उपयोग करें
string replace "bad शब्द" "सादा शब्द" -- कोई फ़ाइल

# वर्ण हटाएं जो एक पैटर्न से मिलते हैं
string match --regex सादा "सादा""-- "सादा शब्द" | each --out "echo हटाता हूँ" "echo यहाँ वर्ण हटाता हूँ"
```

उपरोक्त उदाहरण में, हमने `string replace` कमांड का उपयोग करके वर्ण हटाएं जो हमारे पैटर्न से मिलते हैं और `string match` कमांड का उपयोग करके एक स्ट्रिंग के साथ `each` कमांड को मिलाने का तरीका बताया है। आप भी अपने कोड में अपने पसंदीदा जरीबू का उपयोग कर सकते हैं। 

## डीप डाइव

`string replace` और `string match` दोनों कमांड फ़िश शेल में उपलब्ध हैं और आपकी वर्ण हटाने के तरीकों को सुधारने में आपकी मदद कर सकते हैं। इन कमांडों को आप `man string` कमांड के माध्यम से और अधिक विस्तार से समझ सकते हैं।
