## क्यों

कभी-कभी हमें अपने प्रोग्रामिंग में डेटा को दूसरे सर्वर से प्राप्त करने की आवश्यकता होती है। इसके लिए हम एक HTTP अनुरोध भेज सकते हैं। यह एक सुरक्षित और आसान तरीका है अन्य सर्वर से डेटा हासिल करने का। 

## कैसे करें

फिश शेल में एक HTTP अनुरोध भेजने के लिए आपको `curl` कमांड का उपयोग करना होगा। निम्नलिखित कोड ब्लॉक में, आप HTTP अनुरोध के साथ कॉन्फ़िगरेशन पेज के साथ डेटा डाउनलोड कर सकते हैं।

```Fish Shell
curl https://www.samplewebsite.com/config.json -o config.json
```

यह कमांड डिफ़ॉल्ट रूप से डेटा को करेंट डायरेक्ट्री में डाउनलोड करेगा। आप `~` या `.` के साथ उपयोग करके फ़ाइल को दूसरी जगह भी सहेज सकते हैं। आप उपलब्ध विकल्पों को `man curl` कमांड के ज़रिए चेक कर सकते हैं।

## गहराई में दूसरे सेवा पेश करें

HTTP अनुरोध भेजना सिर्फ डाटा प्राप्त करने का एक तरीका है। लेकिन आप गहरे समांजस्यता के साथ अन्य सेवाओं को भी पेश कर सकते हैं। आप हेडर्स, पोस्ट डेटा, और अन्य विकल्पों का उपयोग कर सकते हैं। आप आप के वहाँ काम किया जाता है, शामिल होने के लिए अनुमति के लिए स्पष्ट अनुरोध कर सकते हैं। प्रयोक्ता दस्तावेज़ आपको चलाने के लिए स्वचालित उपलब्ध प्रमाण-पत्र।
