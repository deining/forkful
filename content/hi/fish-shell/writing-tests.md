# क्यों

टेस्ट लिखने का काम क्यों जरूरी है? हमारे पास इस सवाल का जवाब है। यदि आप कोडिंग का कंसेप्ट सिखते हैं, तो टेस्ट लिखना एक आवश्यक प्रक्रिया है। टेस्ट आपको सुनिश्चित करता है कि आपका कोड सही तरीके से काम करता है और उसमें कोई बग नहीं है।

## कैसे करें

फिश शेल में टेस्ट कैसे लिखा जाता है? हम आपको इसके बारे में कुछ कोडिंग उदाहरण बताएंगे जो आपको समझने में मदद करेंगे:

```Fish Shell
# फ़न्क्शन बनाएं जो संख्याओं की गणना करें
function add
    echo "1+1 is" (math 1+1)
end

# टेस्ट फ़ंक्शन को कॉल करें
add
```

आउटपुट:

```
1+1 is 2
```

ऊपर दिए गए कोड में, हमने एक फ़ंक्शन बनाया जो संख्याओं की गणना करता है। निर्दिष्ट फंक्शन को कॉल किया जाता है और हम अपेक्षित आउटपुट पाते हैं। यह उदाहरण आपको बताता है कि आप कैसे अपने कोड को टेस्ट कर सकते हैं हम शुरुआती स्तर पर कुछ कॉमन फंक्शन्स जाँच सकते हैं, जैसे कि `math` फंक्शन जो गणितीय गणनाओं को संभव बनाता है।

## गहराई में जाएं

टेस्ट लिखने के बारे में अधिक जानकारी प्राप्त करने के लिए, आप हमारे ब्लॉग पोस्ट को देख सकते हैं जो आपको डीप डाइव में ले जाता है। आप कॉमप्रेशन टेस्ट और रंग फिर नियमों के बारे में जानकारी प्राप्त कर सकते हैं।
