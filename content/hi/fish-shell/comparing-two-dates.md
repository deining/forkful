## क्यों

यदि आप विभिन्न तारीखों को तुलना करना चाहते हैं, तो आप ऐसा कर सकते हैं, जो आपको कुछ अतिरिक्त जानकारी देता है। इस प्रकार, आप अपने तारीखों को ध्यान से तुलना कर सकते हैं और अपने कोड को सुधार सकते हैं।

## कैसे करें

इस आलेख में, हम तीन आसान उदाहरणों का उपयोग करके दो तारीखों को तुलना करने की प्रक्रिया दिखाएंगे। यह हर तारीख को मिलाने और उनके बीच का अंतर निकालने में आपकी मदद करेगा। नीचे दिए गए उदाहरणों को पढ़ें और अपने Fish Shell में उन्हें देखें:

```Fish Shell
# उदाहरण 1: मिलाने की प्रक्रिया

set startDate (date -d "2020-01-01" +'%s') # प्रारंभ तारीख को इस रूप में संगठित करें: सेकंड्स के जितने संख्यांक हैं।
set endDate (date -d "2020-01-10" +'%s') # समाप्ति तारीख को उसी तरह संगठित करें।

# अंत में, दो तारीखों को मिला दें।
echo ($endDate - $startDate) # इससे हमें दस दिनों का अंतर मिलेगा।

```

```Fish Shell
# उदाहरण 2: तिथियाँ को संगठित करें

set date1 "10/05/2020"
set date2 "05/10/2020"

set formattedDate1 (date -d $date1 +'%s') # "10/05/2020" को संगठित करने के लिए सूत्र
set formattedDate2 (date -d $date2 +'%s') # "05/10/2020" को संगठित करने के लिए सूत्र

echo ($formattedDate2 - $formattedDate1) # अंतर निकालने से हमें 1209600 सेकंड्स (यानी 14 दिन) का अंतर मिलेगा।

```
