## क्यों

आजकल मौजूदा तिथि और समय को संचालित करना बहुत ही आसान हो गया है। इसके साथ-साथ, कई बार हमें दो तारीखों को तुलना करने की भी आवश्यकता होती है। यह तुलना हमारे कार्यों को समय पर पूरा करने में सहायता करने के लिए बहुत महत्वपूर्ण हो सकती है। इस ब्लॉग पोस्ट में हम दो तारीखों को तुलना करने के लिए कैसे फिश शेल का उपयोग कर सकते हैं, इसके बारे में बात करेंगे।

## कैसे करें

```fish shell
# दो तारीखों को तुलना करने के लिए, 'date' संग्रह का उपयोग करें
date -d "3 days ago" # तारीख को 3 दिन पहले निर्दिष्ट करें
date -d "tomorrow"  # कल की तारीख को निर्दिष्ट करें

# दोनों तारीखों को तुलना करने के लिए, 'diff' संग्रह का उपयोग करें
diff <(date -d "3 days ago") <(date -d "tomorrow")

# तारीखों के साथ समय को भी तुलना कर सकते हैं
diff <(date -d "3 days ago 1:00 PM") <(date -d "tomorrow 2:30 PM")
```

```क्षैतिज
Date : Sun Mar  1 13:00:00 IST 2020 # 'date -d "3 days ago 1:00 PM"' स्तंभ के लिए आउटपुट
Date : Tue Mar  3 14:30:00 IST 2020 # 'date -d "tomorrow 2:30 PM"' स्तंभ के लिए आउटपुट
```

## गहराई में जाएं

तारीखों को तुलना करने के बारे में और गहराई से जानने के लिए, हम 'diff' संग्रह का उपयोग कर सकते हैं। इसमें हम दो स्ट्रिंग तारीखों को हल करते हैं और उनके बीच का अंतर प्रदर्शित करते हैं। यह एक उपयोगी उपकरण है जो अलग-अलग अक्षरों, अं
