## क्यों

रेगुलर एक्सप्रेशंस का उपयोग करने में क्यों लोग एंगेज होते हैं - इसकी कुछ समझ देनें वाली बातें लिखें। एक या दो वाक्य।

रेगुलर एक्सप्रेशंस का उपयोग करना एक महत्वपूर्ण एसी कार्य है जो डेटा को फिल्टर या ढूंढने के लिए उपयोग किया जाता है। यह डेटा संबधित ढूँढने को आसान करते हैं और प्रोग्राम्स को भी और अधिक इफेक्टिव बनाते हैं।

## कैसे रेगुलर एक्सप्रेशंस का उपयोग करें

```Fish Shell
# डेटा फिल्टर करने के लिए शब्द खोजें

fish_prompt> grep "hello" file.txt

ये कोड "file.txt" फाइल में से "hello" शब्द खोजेगा और उसे डिस्प्ले करेगा।
```

```Fish Shell
# मैचिंग पैटर्न को उपयोग करके स्ट्रिंग से डेटा खोजें

fish_prompt> set str "programming is fun"

fish_prompt> echo $str =~ "fun"

"fun" स्ट्रिंग में मैच है, इसलिए यह न्यूमेरिक वैल्यू 1 देगा।
```

## गहराई में जाएं

रेगुलर एक्सप्रेशंस का ऐसे कई तरीके हैं कि काफी आसानी से इसे सीखा और उपयोग किया जा सकता है। पहले से ही तैयार लाइब्रेरी का उपयोग करके और इसे अपने स्क्रिप्ट्स में शामिल करके आप अपने काम को अधिक आसान बना सकते हैं। इसके अलावा, आप अपने स्क्रिप्ट्स में अपने अपने विशिष्ट वर्तमान स्थान पर उपयोग करने के लिए अपनी स्वयं की रेगुलर एक्सप्रेशंस बनाकर भी सकते हैं। इसलिए, रेगुलर एक्सप्रेशंस
