## क्यों

अगर आप Fish Shell प्रोग्रामिंग में हैं, तो आपने शायद अपने स्क्रिप्ट को सेटिंग या फ़ेल के साथ काम करते हुए स्ट्रिंग वस्तुओं को अलग करने की आवश्यकता महसूस की होगी। इस प्रयोग में, आप स्ट्रिंग के एक हिस्से को अलग करके अन्य फ़ंक्शन के साथ इसका उपयोग कर सकते हैं, जिससे आपको अधिक साधन मिलें।

## कैसे

आप Fish Shell में सबसे सरल तरीके से वस्तुओं को अलग कर सकते हैं, `string match` फ़ंक्शन के साथ। इस फ़ंक्शन का उपयोग करके, स्ट्रिंग से अपने विशिष्ट हिस्से को अलग कर सकते हैं और उसे एक नए वैरिएबल में स्टोर कर सकते हैं। यह `string match` फ़ंक्शन कई पैरामीटर्स लेता है, जो कि `--start` और `--end` हैं। इन पैरामीटर्स के माध्यम से आप अपने स्ट्रिंग के आरंभ और समाप्ति अंक को स्पष्ट रूप से निर्दिष्ट कर सकते हैं।

```
Fish Shell में उदाहरण का कोड:
set string "मेरा नाम है जॉन डो"
set name (string match --start 14 --end 17 $string)
echo $name
```

आपको आउटपुट में जॉन दिखाई देगा, जो कि 14वें स्थान से आरंभ होता है और 17वें स्थान पर समाप्त होता है। इस तरह से आप अपनी वस्तुएं अलग कर सकते हैं और अपने स्क्रिप्ट में उनका उपयोग कर सकते हैं।
