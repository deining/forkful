## क्यों

जब हम किसी प्रोग्राम या स्क्रिप्ट में अलग-अलग विभिन्न स्ट्रिंग्स को एक साथ लगाना चाहते हैं, तो हम कॉनकैटनेटिंग स्ट्रिंग का उपयोग करते हैं। यह हमारे प्रोग्राम को और उपयोगी बनाता है जब हम अधिक लोगों को स्क्रिप्ट देते हैं।

## कैसे करें

फिश शेल में स्ट्रिंग्स को कॉनकैटनेट करने के लिए आपको `string join` कमांड का उपयोग करना होगा। नीचे उदाहरण देखें:

```Fish Shell
set fruits apple orange banana
echo (string join ", " $fruits)
```

आउटपुट:

```Fish Shell
apple, orange, banana
```

फिश शेल में `string join` कमांड को दो तरीकों से उपयोग किया जा सकता है। पहले, आप सभी स्ट्रिंग्स को अलग-अलग पैरामीटर के रूप में पास कर सकते हैं जैसे कि उपरोक्त उदाहरण में किया गया है। दूसरा तरीका है, आप स्ट्रिंग्स के एक एरे में पैक करके इसे एकल कैसे करें सकते हैं। उदाहरण के लिए:

```Fish Shell
set fruits fish salmon tuna
echo (string join "-fav, " $fruits)
```

आउटपुट:

```Fish Shell
fish-fav, salmon-fav, tuna
```

## गहराई में जाएं

कॉनकैटनेटिंग स्ट्रिंग अभिकल्प के माध्यम से किए जाते हैं जो कि उच्च स्तर पर फिश शेल द्वारा प्रदान किए गए फ़ंक्शन में से एक है। यदि आप इसे दो या अधिक स्ट्रिंग्स का एक्सेस करने के लिए उपयोग कर रहे हैं, तो आपको सिर्फ पहले स्ट्रिंग का एक्सेस करना होता है और बाकी स्ट्रिंग्स को उपरोक्त तरीकों से दो या अध
