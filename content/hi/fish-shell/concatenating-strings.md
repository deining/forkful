# क्यों

क्या आपको कभी अपने कोड में एक स्ट्रिंग को एक साथ जोड़ने की आवश्यकता हुई है? आपको दो स्ट्रिंग को एक साथ जोड़ने की जरूरत हो सकती है, जैसे कि प्रयोगकर्ता को एक स्वागत संदेश दिखाने के लिए दो अलग-अलग स्ट्रिंग जोड़ना. इस ब्लॉग पोस्ट में मैं आपको स्ट्रिंग को जोड़ने के तरीके के बारे में बताऊंगा जो आपको सही रीति से काम करने में मदद कर सकते हैं।

## कैसे करें

एक स्ट्रिंग को दूरस्थ स्थानों पर स्थित श्रृंखलाओं को जोड़ने के लिए, आपको `string concat` फंक्शन का उपयोग करना होगा। नीचे दिए गए उदाहरण में, मैंने `name` और `message` अनुक्रमिक श्रृंखलाओं को जोड़ा है और फिर `echo` कमांड का उपयोग करके उसे प्रिंट किया है। आप `echo` के अलावा `printf` और `string append` भी उपयोग कर सकते हैं।

```Fish Shell

set name "मोहन"
set message "आपका स्वागत है"
set full_message (string concat $name " " $message)
echo $full_message

```

आउटपुट:
`मोहन आपका स्वागत है`

## गहराई में जाएं

जब आप `string concat` फंक्शन का उपयोग करते हैं, तो Fish Shell एक नई स्ट्रिंग बनाता है। इस प्रक्रिया को "स्पाइपाइंडरिंग" कहा जाता है। यदि आप `echo` के साथ कई स्ट्रिंग्स को जोड़ना चाहते हैं, तो आप एक साथ उन्हें भी जोड़ सकते हैं। आप अलग-अलग स्ट्रिंग और वेरिएबल्स को स्पाइपाइंडरिंग के साथ जोड़ सकते हैं।
