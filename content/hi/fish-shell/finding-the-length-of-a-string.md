## क्या और क्यों?

"अब हम फिश शैल नाम के एक प्रोग्रामिंग लैंग्वेज में स्ट्रिंग की लम्बाई कैसे पता करे, इस बारे में बात करेंगे। स्ट्रिंग की लम्बाई से हमारे काम में कई फायदे हो सकते हैं, जैसे डेटा को अंकीकृत करना या फर्मेटिंग के लिए इस्तेमाल करना। इसीलिए यह एक उपयोगी प्रोग्रामिंग स्किल है जो हर प्रोग्रामर को सिखनी चाहिए।"

## कैसे:

फिश शैल स्ट्रिंग की लम्बाई को पता करने के लिए इस आसान से कोड को फॉलो करें:
```
fish
set str "Hello World!"
echo (string length $str)
```

आउटपुट: 12


अगर आप अपने स्ट्रिंग की लम्बाई को विभिन्न इकाइयों में हासिल करना चाहते हैं, तो निम्नलिखित कोड इस्तेमाल करें:
```
set km_str "150 km"
# स्ट्रिंग की प्रथम शब्द "150" को पदेनुम्या हटाएं
set km (string replace -r [a-zA-Z] $km_str '')
echo $km # आउटपुट: 150
echo (string length (string replace -r [a-zA-Z] $km_str))
# आउटपुट: 3
```

## गहराई में जाएं:

इतिहास की बात करें, तो स्ट्रिंग की लम्बाई की गणना प्रोग्रामिंग का एक बहुत बुनियादी स्किल है। पहले कंप्यूटर कोड को इंग्लैश में प्रोग्राम करने के लिए उपयोग किया जाता था, जिसमें स्ट्रिंग को गिनने के लिए अलग-अलग विधियां निर्धारित थीं। आज हमारे लिए, यह स्ट्रिंग की लम्बाई पता करने के लिए बनाए गए फॉर्म्यूलेशन का उपयोग करता है।

अन्य विकल्पों की बात करें, अन्य प्रोग्रामिंग भाषाओं जैसे Python या C++ में भी स्ट्रिंग की लम्बाई पता करने के लिए अलग कोड स्निपेट होते हैं। हालांकि, फिश शैल एक हाइ-लेवल प्रोग्रामिंग लैंग्वेज है जो इस प्रकार की प्रक्रिया को आसान बनाता है।

स्ट्रिंग की लम्बाई पता करने के लिए फिश शैल इंटरनली ```string length``` मेथड का इस्तेमाल करता है जो उपयोगकर्ता को सीधे उत्पन्न तालिका में लम्बाई प्रदान करता है।
