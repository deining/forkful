## क्यों

कोई भी प्रोग्रामिंग भाषा सीखने का मुख्य कारण होता है कि उससे अपने को और बेहतर बनाने में मदद मिलेगी। तो इसलिए, हम आज जानेंगे कि Fish Shell में स्ट्रिंग की लंबाई कैसे निकाली जाती है।

## कैसे

***चेतावनी: निम्न दिए गए कोड ब्लॉक में दिए गए कोड को अपने टर्मिनल में लिखने से पहले आपको फिश शैल को इंस्टॉल करना हो सकता है।***

```Fish Shell में कोड ब्लॉक बनाने के लिए निम्न आदेश का उपयोग करें:```

```
```
```
echo -n "अपनी स्ट्रिंग लिखें: "; read string
echo "लंबाई: $strlen(string)"
```

***उपरोक्त कोड ब्लॉक में हमने `echo` कमांड का उपयोग करके प्रयोगकर्ता से एक स्ट्रिंग दर्ज करने को कहा है और उसकी लंबाई प्रिंट करने के लिए `strlen` फंक्शन का उपयोग किया है। आप अपने अनुसार उपयोगकर्ता से स्ट्रिंग का नाम भी ले सकते हैं। इसके अलावा `echo` कमांड के साथ `-n` अविलंब उपयोग करने से टर्मिनल पर कोई नया लाइन नहीं बनेगा जो `echo` कमांड की जगह स्लॉट करेगा।***

**उत्पाद:**
```
अपनी स्ट्रिंग लिखें: Hello world!
लंबाई: 12
```

## गहराई तक खुदाई करें

अब कि आप जानते हैं कि Fish Shell में स्ट्रिंग कैसे निकाली जाती है, आप गहराई तक खुदाई कर सकते हैं और इसे अपने प्रोग्राम्स में उपयोग कर सकते हैं। फिश शैल में `strlen` फंक्शन को देखने के लिए आप इन्हें चेक कर सकते हैं
