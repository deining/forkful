## क्यों

दोस्तों, क्या आपने कभी सोचा है कि प्रोग्रामिंग में आपके पास एक स्ट्रिंग को हासिल करने के लिए कैसे पत्र बनाने की आवश्यकता होती है? या अपने स्क्रिप्ट में स्ट्रिंग की लंबाई को पता करने के लिए आपके पास कोई तरीका है? अगर नहीं, तो आप इस लेख को जरूर पढ़िए! हम आपको Fish Shell में स्ट्रिंग की लंबाई कैसे ढूंढ सकते हैं इसके बारे में बताएंगे।

## कैसे करें

यदि आप Fish Shell का उपयोग करते हैं, तो आप स्ट्रिंग की लंबाई को जानने के लिए "string length" कमांड का उपयोग कर सकते हैं। नीचे दिए गए कोड ब्लॉक में आप इसका उपयोग देख सकते हैं।

```Fish Shell
string length "Hello World"

# Output: 11
```

यदि आपके पास एक वेरिएबल है जिसमें स्ट्रिंग है, तो आप उसकी लंबाई को भी जान सकते हैं।

```Fish Shell
set my_string "This is a string"
string length $my_string

# Output: 16
```

आप भी अपने स्क्रिप्ट में इस कमांड का उपयोग कर सकते हैं और स्ट्रिंग की लंबाई को बदल सकते हैं।
