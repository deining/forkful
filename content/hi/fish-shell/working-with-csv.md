# क्यों 

क्या आप अपने डेटा को सीएसवी फाइल में संग्रहीत करना चाहते हैं? अगर हाँ, तो आप न केवल अपने डेटा को अच्छी तरह से संग्रहीत कर सकते हैं बल्कि अपने कोड को सरल और आसान बना सकते हैं। एक सरल और उपयोगी भाषा में, हम आपको बताएंगे कि कैसे आप फिश शेल का उपयोग करके सीएसवी फाइलों को प्रसंस्करण कर सकते हैं।

## कैसे करें

फिश शेल में CSV फाइलों की प्रसंस्करण करने के लिए हमें `csv` लाइब्रेरी का उपयोग करना होगा। सबसे पहले, हम `fs/csv` को आवश्यक लाइब्रेरी के रूप में लोड कर सकते हैं। इसके बाद, हम सीएसवी फाइल को पढ़ सकते हैं और उसके रिकॉर्ड तक पहुंच सकते हैं।

```
Fish Shell में अपनी सीएसवी फाइल को लोड करने के लिए निम्नलिखित कोड का उपयोग करें:
```Fish Shell
set csvfile 'example.csv'

set records (csv from $csvfile)
for record in $records
    echo $record
end

```

इसके रिजल्ट को नीचे देखें:

```
["विंडोज", "फ़ोन", "15000"]
["मैकबुक", "लैपटॉप", "50000"]
["एंड्रॉयड", "टैबलेट", "10000"]
```
