## क्यों

टेक्स्ट फाइल पढ़ने का काम भले ही आसान लगे, लेकिन अधिक माहिर और अनुभवी प्रोग्रामर के लिए यह एक उपयोगी कौशल है। इससे आप किसी भी प्रकार के टेक्स्ट स्ट्रिंग्स को ऑटोमेटिक रूप से प्रोसेस कर सकते हैं और अपने स्क्रिप्ट को और समय और हार्डवेयर की बचत कर सकते हैं। तो चलिए जानते हैं कि टेक्स्ट फाइल को कैसे पढ़ा जाता है।

## कैसे करें

```Fish Shell``` कोड ब्लॉक्स के भीतर हम एक ```echo``` कमांड का उपयोग करके टेक्स्ट फाइल का जो भी कंटेंट हो, उसे प्रिंट कर सकते हैं। यह उदाहरण आसानी से समझने के लिए है:

```
echo "हैलो, ये एक टेक्स्ट फाइल है"
```

आउटपुट:
```
हैलो, ये एक टेक्स्ट फाइल है
```

इसके अलावा आप ```cat``` कमांड का उपयोग करके भी टेक्स्ट फाइल का कंटेंट देख सकते हैं। उदाहरण के लिए:

```
cat कोड.तकनीक
```

आउटपुट:
```
यह एक टेक्स्ट फाइल है जो कोड्स के बारे में है
```

## गहराई में जाएं

अगर हम गहराई में जाएं तो टेक्स्ट फाइल को पढ़ना बहुत ही रोचक बन सकता है। आप नहीं जानते होंगे, लेकिन टेक्स्ट फाइल को प्रोसेस करते समय आप प्रत्येक लाइन को एक नए प्रॉम्प्ट की तरह समझ सकते हों। यह उदाहरण आपको एक अंतरिक्ष टेक्स्ट फाइल पढ़ने के लिए बताता है:

```
while read line
echo $line
