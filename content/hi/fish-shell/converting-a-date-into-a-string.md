## क्यों

किसी भी प्रोग्रामिंग भाषा में, तारीख को स्ट्रिंग में बदलने का एक सामान्य काम है। जब हम किसी प्रोग्राम को बनाते हैं, तो हमें अक्सर तारीख को ग्राफिकल रूप से प्रदर्शित करने की आवश्यकता होती है। तारीख को स्ट्रिंग में बदलने से, हमें अपने प्रोग्राम या स्क्रिप्ट में अपनी जरूरतों के अनुसार तारीख को चुन सकते हैं और उसे अपनी पसंद के अनुसार फॉर्मेट कर सकते हैं। इसलिए, यह एक उपयोगी कौशल है जो किसी भी प्रोग्रामर को सीखने या नए प्रोग्राम या स्क्रिप्ट में इस्तेमाल करने में मदद कर सकता है।

## कैसे

```Fish Shell
set today (date)
set string (strftime "%d-%m-%Y" $today)
echo $string
```
उपरोक्त कोड का आउटपुट निम्नप्रकार होगा:

```Fish Shell
29-07-2020
```

उपरोक्त कोड से आप देख सकते हैं कि हमने ```date``` कमांड की सहायता से वर्तमान दिनांक को लेकर एक वेरिएबल बनाया है और उसे स्ट्रिंग में बदलने के लिए ```strftime``` कमांड का उपयोग किया है। आप इसका प्रयोग अपने प्रोग्राम या स्क्रिप्ट में अपनी जरूरतों के अनुसार बदल सकते हैं।
