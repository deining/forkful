## क्यों

आपने कभी भी अपने वेब ब्राउज़र के माध्यम से वेब साइटों पर कॉन्टेंट एक्सेस करते समय खुद को लॉगिन पेज पर देखा होगा। आप कुछ वेब साइटों के लिए अपने यूज़रनेम और पासवर्ड को मांगने वाले लॉगिन प्रोसेस को देखा होगा। इस तरह के सुरक्षा उपयोगकर्ताओं के लिए एक सुरक्षित तरीके से वेब साइट से कनेक्ट करने के लिए इस्तेमाल किया जाता है। इस तरह के एक प्रोसेस को HTTP अनुरोध के रूप में भेजा जाता है जो बेसिक ऑथेंटिकेशन का उपयोग करता है। इस लेख में, हम पाठकों को सीखेंगे कि अपनी वेब साइटों पर बेसिक ऑथेंटिकेशन के साथ HTTP अनुरोध कैसे भेजा जाता है।

## कैसे करें

```Fish Shell
curl -u username:password https://example.com
```

ऊपर दिए गए उदाहरण में, हमने फिश शेल का इस्तेमाल करके बेसिक ऑथेंटिकेशन के साथ एक HTTP अनुरोध भेजा है। यहां, "कोयले" और "स्वर्णिम" यूज़रनेम और पासवर्ड हैं और "उदाहरण.कॉम" वेबसाइट को प्रतिनिधित्व करता है। जब आप यह रन करेंगे, आपको सफलतापूर्वक वेबसाइट से कनेक्ट होना चाहिए और काम पूरा होने पर सक्सेस कोड दिखाई देगा।
