## 为什么要获取当前日期

获取当前日期在PHP编程中是一个很常见的需求。它可以用来跟踪事件发生的时间，比如网站注册、订单生成等。也可以用来显示网站上最新的更新时间，让访问者知道网站内容的最新情况。

## 如何获取当前日期

要获取当前日期，我们可以使用PHP内置的函数 `date() `。这个函数接收两个参数，第一个参数是日期的格式，第二个参数是可选的时间戳。下面是一个例子：

```PHP
<?php
    // 获取当前日期
    $currentDate = date('Y-m-d');

    // 打印输出结果
    echo $currentDate;
?>
```

输出结果应该为当前的年月日，比如 `2021-07-08`。可以根据自己的需要自定义日期的显示格式，比如 `date('d/m/Y')` 会把日期显示为 `08/07/2021`。

如果想要获取当前的时间，可以把格式参数改为 `H:i:s`，比如 `date('H:i:s')`，输出结果为当前的时分秒，比如 `15:30:21`。详细的日期格式参考可以在PHP官方文档中找到。

## 深入了解获取当前日期

PHP的 `date()` 函数实际上是对C语言的库函数 `strftime()` 的封装。它接收同样的参数，并返回格式化后的日期和时间字符串。

另外，如果我们只想获取当前的年份、月份或者日期，还可以使用 `date('Y')`、`date('m')`、`date('d')` 分别来获取。这样可以方便地与其他变量相结合，完成更复杂的操作。

在使用 `date()` 函数时，要注意服务器的时区设置。如果时区不匹配，可能会导致日期和时间不准确。
