#为什么

提取子字符串是在编程中常见的任务。它可以帮助我们从长字符串中提取出我们所需要的部分，使我们的代码更加高效和简洁。

#如何操作

为了在Gleam中提取子字符串，我们可以使用`substring/2`函数。该函数需要两个参数：字符串和提取的起始和结束索引。让我们来看一个实际的例子：

```Gleam
let str = "这是一个示例字符串"
let sub_str = substring(str, 5, 9)
```

在这个例子中，我们从索引5开始提取，直到索引9结束。所以最终的结果将是`"一个示例"`。让我们来看看如何输出提取出的子字符串：

```Gleam
io.format("提取的子字符串是：{}", [sub_str])
```

输出将是：

```Gleam
提取的子字符串是："一个示例"
```

这样我们就成功地在Gleam中提取出了想要的子字符串。

#深入探讨

除了`substring/2`函数外，Gleam还提供了许多其他函数来帮助我们操作子字符串，如`length/1`函数用来计算字符串的长度，`startsWith/2`函数用来检查字符串是否以某个特定的子字符串开始，`contains/2`函数用来检查字符串中是否包含某个子字符串，等等。通过熟悉这些函数，我们可以更灵活地操作子字符串，实现更多的功能。
