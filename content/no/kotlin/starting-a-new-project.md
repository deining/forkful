## Hva & Hvorfor?
Når du starter et nytt prosjekt, betyr det at du begynner å lage et nytt program eller kodebase fra bunnen av. Dette er noe programmerere gjør for å møte nye utfordringer og forbedre sine ferdigheter.

## Hvordan:
For å starte et nytt Kotlin-prosjekt, åpne din IDE (Integrated Development Environment) og følg disse trinnene:

1. Opprett et nytt prosjekt og velg Kotlin som programmeringsspråk.
2. Gi prosjektet et navn og velg en plassering.
3. Når prosjektet er opprettet, vil du få en standard konfigurasjonsfil med nødvendige avhengigheter. 
4. Start å skrive koden din!

En eksempelkode på hvordan du kan skrive ut "Hello World" ser slik ut:

```Kotlin
fun main() {
    println("Hello World")
}
```

Output:
```
Hello World
```

## Deep Dive:
Historisk sett ble Kotlin utviklet av JetBrains i 2011 og ble offisielt lansert som et åpen kildekode-prosjekt i 2016. Det er et moderne programmeringsspråk som kjører på Java Virtual Machine (JVM) og er kompatibelt med Java-kode. Alternativer til Kotlin inkluderer Java, Scala og Groovy. For å implementere et nytt prosjekt i Kotlin, kan du også bruke byggeverktøy som Gradle eller Maven.
