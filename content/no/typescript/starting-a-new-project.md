## Hvorfor

Å starte et nytt prosjekt kan være en spennende og givende opplevelse for utviklere. Det kan åpne for nye muligheter og hjelpe deg med å utvide dine kunnskaper og ferdigheter innen programmering. Med TypeScript kan du ta prosjektene dine til et helt nytt nivå ved å kombinere JavaScript og typet programmering.

## Hvordan

For å komme i gang med et nytt prosjekt i TypeScript, må du først sikre deg at du har installert Node.js og NPM på datamaskinen din. Deretter kan du følge disse trinnene for å sette opp prosjektet ditt:

1. Åpne kommandolinjen og naviger til ønsket mappe for prosjektet ditt.
2. Skriv inn `npm init` for å initialisere et nytt NPM-prosjekt. Dette vil opprette en `package.json`-fil som inneholder informasjon om prosjektet ditt.
3. Deretter må du installere TypeScript ved å skrive `npm install typescript --save-dev` i kommandolinjen. Dette vil installere TypeScript som et utviklerverktøy for prosjektet ditt.
4. Opprett en ny fil med navnet `app.ts`. Dette vil være hovedfilen for prosjektet ditt hvor du vil skrive koden din.
5. Inne i denne filen kan du begynne å skrive din TypeScript-kode ved å bruke `console.log()` for å skrive ut en beskjed.
```TypeScript
console.log('Velkommen til TypeScript!');
```
6. For å kjøre koden din, må du først kompilere den til vanlig JavaScript. Dette kan gjøres ved å skrive `tsc app.ts` i kommandolinjen. Dette vil opprette en ny fil kalt `app.js` som inneholder den kompilerte koden.
7. For å kjøre den kompilerte koden, skriv `node app.js` i kommandolinjen. Dette vil vise beskjeden du skrev ut i trinn 5.

## Deep Dive

Når prosjektet ditt er satt opp, kan du begynne å dykke dypere inn i TypeScript og utforske alle funksjonene og mulighetene som dette språket har å tilby. Ved å bruke typestøttet programmering, kan du sikre at koden din er mer robust og feilfri.

Etter hvert som du blir mer komfortabel med TypeScript, kan du begynne å bruke det til større og mer komplekse prosjekter. Du kan også dra nytte av samfunnet rundt TypeScript ved å lære av andres kode og bidra til open-source-prosjekter.
