## ¿Qué y por qué?
Empezar un nuevo proyecto en programación es el proceso de crear una aplicación o software desde cero. Los programadores lo hacen para desarrollar nuevas ideas, resolver problemas o mejorar proyectos existentes.

## Cómo hacerlo:
```Kotlin
fun main() {
    println("¡Hola, mundo!")
}
```
Output: ¡Hola, mundo!

```Kotlin
fun main() {
    val num1 = 5
    val num2 = 10
    val sum = num1 + num2
    println("La suma de $num1 y $num2 es $sum.")
}
```
Output: La suma de 5 y 10 es 15.

## Inmersión profunda:
Empezar un nuevo proyecto es una práctica común en el mundo de la programación. En lugar de modificar un proyecto ya existente, los programadores prefieren empezar de cero para tener un mayor control y mayor flexibilidad en el desarrollo. Alternativas a empezar un nuevo proyecto son utilizar plantillas o frameworks preexistentes para agilizar el proceso. En Kotlin, el código inicial se escribe en una función main() y se compila a bytecode para ser ejecutado en la JVM (Java Virtual Machine).
