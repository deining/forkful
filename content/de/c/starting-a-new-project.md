## Was & Warum?
Wenn du ein neues Programm oder Projekt auf den Weg bringen willst, startest du ein neues Projekt. Warum machen Programmierer das? Weil es der erste Schritt ist, um eine Idee in einen funktionsfähigen Code umzusetzen.

## Wie geht's? 
```C
#include <stdio.h>

int main(void) {
    printf("Hallo Welt!");
    return 0;
}
```
Ausgabe: Hallo Welt!

Um ein neues Projekt zu starten, musst du zuerst eine Datei anlegen und einen Anfangspunkt für dein Programm definieren. In diesem Beispiel verwenden wir `main()` als Anfangspunkt und die `printf()` Funktion, um einen einfachen Text auszugeben.

## Tiefere Einblicke
Ein neues Projekt zu starten ist ein grundlegender Teil jeder Programmierarbeit. Es gibt mehrere Alternativen wie zum Beispiel das Verwenden von schon vorhandenem Code oder das Herunterladen von Bibliotheken. Diese können dir Zeit ersparen und helfen, komplexe Probleme zu lösen. Im Endeffekt ist es jedoch immer noch notwendig, ein neues Projekt zu starten und deinen eigenen Code zu schreiben.

Wenn du ein großer Fan der Geschichte des Programmierens bist, könntest du vielleicht interessiert sein zu erfahren, dass das erste Programm der Welt 1948 von Konrad Zuse geschrieben wurde. Damit hat er den Grundstein für die moderne Programmierung gelegt.
